## ¿Qué es un archivo AVERY?

Un archivo AVERY es un documento imprimible que contiene una o más etiquetas para fines de impresión. Se crea con el software Avery Design & Print o Design & Print Online. Las etiquetas dentro de un archivo AVERY pueden constar de diferentes elementos, como texto, gráficos e información de diseño de página. Las empresas de impresión utilizan esta información para imprimir una variedad de elementos de diseño, como tarjetas de presentación, etiquetas de nombre, insertos de carpetas, etiquetas de direcciones, etiquetas de discos, plantillas de obituarios y tarjetas de felicitación.

## Formato de archivo AVERY - Más información

Los archivos AVERY se almacenan en formato de archivo binario. Estos se pueden abrir con el software Avery Design & Print o Design & Print Online. Los diseños creados con Design & Print se pueden usar para crear etiquetas AVERY que pueden servir para varios propósitos. Estos diseños se pueden guardar como archivos [PDF](/es/pdf/) que se pueden compartir con la empresa Prints para fines de impresión.

### Imprimir un archivo AVERY

Un archivo AVERY puede guardarse como PDF o puede imprimirse directamente en cualquier impresora estándar. Para imprimir el archivo, ábralo en el software Design & Print y use la opción Imprimir del menú Archivo para enviar el archivo a la impresora para su impresión.
