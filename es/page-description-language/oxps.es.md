## ¿Qué es un archivo OXPS? ##

El formato de archivo OXPS se conoce como Open XML Paper Specification. Es un lenguaje de descripción de página y formato de documento. Microsoft es el desarrollador de este formato. El formato de archivo OXPS es muy familiar para estos archivos PDF. La configuración del archivo OXPS es la misma que la del formato XPS. La única diferencia es la descripción de OXPS como formato abierto. La extensión de archivo .oxps es la versión más actual del formato de archivo XPS originado por Microsoft y Ecma International. Si hablamos de la especificación, los archivos OXPS y XPS se utilizan para publicar, compartir y asignar documentos de diseño fijo. Como sustituto de los PDF, el formato de archivo OXPS puede almacenar texto y datos multimedia, con funciones de organización y características de diseño de impresión.

## Breve historia ##

El equipo de Microsoft eligió Global Graphics para brindar consultoría sobre XPS en 2003, y el equipo de desarrollo trabajó en estrecha colaboración con los equipos de Windows para desarrollar la especificación y la arquitectura de referencia para el nuevo formato. Reconociendo las especificaciones XPS, en junio de 2007 Ecma International creó el Comité Técnico 46 (TC46) para desarrollar un estándar basado en la especificación OpenXPS. La especificación de papel Open XML aprobada por Ecma International es un estándar Ecma (ECMA-388) en la 97ª Asamblea General celebrada en Budapest el 16 de junio de 2009.
