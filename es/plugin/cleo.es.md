## ¿Qué es un archivo CLEO?

Un archivo .cleo es un archivo complementario que se utiliza como parte de la biblioteca CLEO para la serie de juegos GRand Theft Auto. Se utiliza para ejecutar modificaciones que cambian o amplían el juego. El archivo CLEO almacena una biblioteca en formato de archivo DLL y agrega comandos de secuencias de comandos útiles (códigos de operación) para trabajar con archivos externos para cambiar la memoria del juego. Existen diferentes versiones de CLEO creadas para GTA III, GTA Vice City y GTA San Andreas.

## Formato de archivo CLEO

CLEO 4 permite crear nuevos códigos de operación utilizando la fuente del CLEO SDK. Los tres ejemplos más comunes de este tipo de archivos de complemento incluyen:

 * IniFiles.cleo: códigos de operación para archivos INI;
 * FileSystemOperations.cleo: códigos de operación para archivos y carpetas.
 * IntOperations.cleo: códigos de operación para operaciones lógicas (Y, O, XOR, etc.).

Los complementos se instalan de manera similar a los scripts. Para instalar un complemento CLEO, debe copiar el archivo .CLEO a la carpeta CLEO.

## Acerca de CLEO

CLEO es un complemento extensible muy popular para la serie de juegos Grand Theft Auto de Rockstar Games, que permite el uso de miles de modificaciones únicas que cambian o amplían la jugabilidad.
