## ¿Qué es un archivo INX?

Un archivo INX es un archivo complementario utilizado por el editor de gráficos vectoriales de código abierto, **Inkscape**. Contiene información sobre el programa, incluido el nombre y la ubicación de la funcionalidad en el menú. El archivo INX también especifica un script para la extensión que incluye información como el tipo de script, el intérprete, la ubicación y los argumentos variables.

## Formato de archivo INX

Los archivos INX mejoran la funcionalidad de Inkscape al incorporar funciones como importadores de archivos y filtros de gráficos. Almacenados en formato XML, estos archivos generalmente se encuentran en el directorio \​extensions\​ dentro de una instalación de Inkscape.

## ¿Cómo abrir archivos INX?

Puede abrir archivos INX utilizando el programa **Inkscape**.
