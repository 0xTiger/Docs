## ¿Qué es un archivo TGP?

Un archivo TGP sirve como complemento utilizado en Terragen Classic, un programa diseñado para generar paisajes naturales fotorrealistas. Este archivo encapsula datos que mejoran las funcionalidades de Terragen tras la instalación. Los archivos TGP se emplean para introducir diversas funciones en Terragen, incluidos efectos de cámara y opciones de paisaje adicionales.

## Acerca de Terragen Classic

Terragen es un software de generación y renderizado de paisajes desarrollado por Planetside Software. La versión Clásica se refiere a una versión anterior del software, ya que también hay una versión más reciente llamada **Terragen 4**.

Terragen Classic era conocido por sus capacidades para generar paisajes y terrenos realistas. Permitió a los usuarios crear entornos naturales detallados y creíbles, incluidas montañas, valles, lagos y cielos. El software incluía funciones para esculpir el terreno, mapear texturas y renderizar.
