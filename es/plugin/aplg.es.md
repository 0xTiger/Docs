## ¿Qué es un archivo APLG?

Un archivo con extensión .aplg es un archivo complementario utilizado por el software de grabación de transmisiones por Internet Audials. El software carga el complemento para agregar nuevas funciones, como nuevas funciones de búsqueda y grabación, para capturar transmisiones de música y videos de Internet. Audials ofrece un reproductor de radio con muchas funciones que permite a los usuarios acceder y escuchar una gran cantidad de estaciones de radio por Internet.

## Formato de archivo APLG

Los archivos APLG vienen empaquetados en los paquetes de complementos .APLP, lo que significa que el software Audials puede cargar varios complementos del paquete al mismo tiempo. Los usuarios a menudo pueden grabar sus canciones o programas de radio favoritos para reproducirlos más tarde. Esto puede resultar útil para crear listas de reproducción personalizadas. Audials normalmente mantiene una gran base de datos de estaciones de radio, lo que facilita a los usuarios descubrir contenido nuevo.
