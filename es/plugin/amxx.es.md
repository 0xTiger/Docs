## ¿Qué es un archivo AMXX?

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. Es una plataforma de secuencias de comandos que permite a los administradores de servidores personalizar y ampliar la funcionalidad de sus servidores. AMXX se utiliza junto con el sistema de administración de complementos MetaMod que permite ejecutar múltiples complementos simultáneamente en un servidor sin conflictos. Las funciones que ofrece el complemento AMXX incluyen comandos de administración, modificaciones del juego y administración del servidor.

## Formato de archivo AMXX - Más información

Los archivos AMXX son complementos que se guardan en formatos de archivo propietarios. La versión anterior del complemento AMXX es AMX, que fue utilizada por AMX Mod, el predecesor de AMX Mod X.
