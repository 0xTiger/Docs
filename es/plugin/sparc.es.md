## ¿Qué es un archivo SPARC?

Un archivo .sparc, también conocido como Archivo de complementos de Skype, sirve como un paquete que incluye funciones o mejoras adicionales para la aplicación Skype. Estos complementos tienen la capacidad de introducir una amplia gama de capacidades, como grabación de llamadas, uso compartido de pantalla, traducción,
Juegos y Entretenimiento, Emoticones y temas personalizados.

## Formato de archivo SPARC - Más información

En determinadas versiones de Skype, el Administrador de extras es un componente opcional que se puede integrar en la instalación de Skype. Si tiene dificultades para instalar un archivo SPARC, es posible que haya omitido el Administrador de extras durante la instalación inicial. Para solucionar este problema, reinstale Skype y asegúrese de seleccionar la opción Administrador de extras.

## ¿Cómo instalar el complemento de Skype?

Pasos para instalar complementos de Skype:

1. Obtenga el archivo .sparc correspondiente al complemento deseado.
2. Cierra Skype si se está ejecutando actualmente.
3. Localice el archivo .sparc descargado y haga doble clic en él.
4. Siga las instrucciones en pantalla para completar la instalación del complemento.
5. Después de la instalación, reinicie Skype para activar el complemento.
