## ¿Qué es un archivo ZXP?

Un archivo ZXP sirve como extensión, aumentando las funcionalidades del software de Adobe como Adobe Photoshop o Adobe After Effects. En el pasado, Adobe Extension Manager facilitaba la instalación de archivos ZXP. Actualmente, el proceso de instalación requiere el uso de la herramienta de línea de comandos Unified Plugin Installer Agent (UPIA), que viene incluida con la aplicación de escritorio Adobe Creative Cloud.

## Formato de archivo ZXP

El término ZXP significa Paquete ZIP eXtension. Los archivos ZXP se utilizan para empaquetar y distribuir extensiones para aplicaciones de Adobe Creative Cloud como Adobe Photoshop, Illustrator, InDesign y otras. Un archivo ZXP se comprime con el formato [ZIP](/compression/zip/), lo que facilita la distribución e instalación de extensiones.

Dentro de un archivo ZXP, normalmente encontrará los archivos necesarios para la extensión, incluidos [HTML](/web/html/), [CSS](/web/css/), JavaScript, imágenes y cualquier otro recurso necesario para mejorar o agregar funciones al software Adobe Creative Cloud.

### ¿Cómo instalar el archivo ZXP?

Los archivos ZXP se instalan utilizando Adobe Extension Manager o la aplicación Creative Cloud Desktop. Los usuarios pueden simplemente hacer doble clic en un archivo ZXP y el administrador de extensiones se encargará del proceso de instalación.

Los archivos ZXP están diseñados para funcionar en diferentes sistemas operativos, incluidos Windows y macOS, lo que hace que sea conveniente para los desarrolladores crear extensiones que puedan ser utilizadas por una amplia audiencia.

### Seguridad del formato de archivo ZXP

Los desarrolladores pueden firmar digitalmente los archivos ZXP, lo que garantiza que la extensión no haya sido manipulada y provenga de una fuente confiable. Esto ayuda a mantener la seguridad y la integridad de las aplicaciones de Adobe Creative Cloud.

Si está trabajando con extensiones de Adobe Creative Cloud o desarrollando una usted mismo, puede encontrar archivos ZXP como medio para empaquetar y distribuir su extensión. Sin embargo, los detalles específicos del trabajo con archivos ZXP pueden estar sujetos a cambios o actualizaciones por parte de Adobe, por lo que es una buena idea consultar la documentación más reciente de Adobe para obtener la información más precisa y actualizada.
