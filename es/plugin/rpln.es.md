## ¿Qué es un archivo RPLN?

Un archivo RPLN es un archivo complementario utilizado por el programa de diseño de páginas, **InDesign**, que crea diseños de páginas y también proporciona publicaciones impresas. contiene funciones que se requieren para su ejecución y se encuentra en un directorio requerido donde está instalado el software InDesign.

## Formato de archivo RPLN

Los archivos RPLN se guardan en el formato de archivo propietario de InDesign y no hay más detalles sobre la estructura de archivos interna de los archivos RPLN.

## Acerca de Adobe InDesign

Adobe InDesign es una aplicación de software de autoedición profesional que se usa ampliamente para crear medios impresos y digitales. InDesign admite el uso de complementos, que son componentes de software adicionales que se pueden agregar para ampliar la funcionalidad del programa.

## ¿Cómo abrir el archivo RPLN?

Puede abrir archivos RPLN con Adobe InDesign 2023.
