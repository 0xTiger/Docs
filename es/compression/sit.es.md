## ¿Qué es un archivo SIT?

Un archivo SIT es un archivo comprimido creado con el software Micro StuffIt. Este es el formato de archivo estándar original que introdujo StuffIt. El formato de archivo SIT se usa comúnmente en los sistemas Macintosh, pero ahora es compatible con las plataformas Windows y Macintosh. StuffIt amplió el formato de archivo SIT a [SITX](/es/compression/sitx/) que ofrece más compresión en comparación con los archivos SIT. Los archivos SIT se pueden abrir con el software Smith Micro StuffIt Deluxe Mac 16.

## Formato de archivo SIT - Más información

Los archivos SIT se guardan en formato de archivo binario y los detalles de su formato de archivo interno no están disponibles públicamente. Fue introducido por Aladdin Systems y se introdujo para comprimir archivos en archivos.
