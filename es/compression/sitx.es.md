## ¿Qué es un archivo SITX?

Un archivo con extensión .sitx es un archivo comprimido creado con el software StuffIt. Este formato se introdujo como una extensión del formato de archivo Stuffit (.sit) estándar original y ofrece comparativamente más compresión que su predecesor. Los archivos SITX son más compactos y ocupan menos espacio junto con la compatibilidad con la compresión de 512 bits destinada a la seguridad. Los archivos SITX se reconocen por sus tipos de medios de Internet como aplicación/x-stuffitx y aplicación/x-sitx con identificador de tipo uniforme como com.stuffit.archive.sitx.

## Formato de archivo SITX - Más información

STIX se guardan en formato de archivo binario y evolucionaron después del uso del formato de archivo SIT. Fue introducido por Aladdin Systems en septiembre de 2002 para StuffIt Deluxe 7.0 para Macintosh. Este nuevo formato de archivo fue diseñado teniendo en cuenta los siguientes objetivos:

* Extensible
* Soporte para más métodos de compresión
* Soporte para nombres de archivo largos
* Admite atributos de archivos de Unix y Windows

El StuffIt X actualizado mejoró el formato StuffIt original al incorporar múltiples algoritmos de compresión como PPM y compresión de tipo BWT a LZW. El nuevo formato de archivo también estaba equipado con una opción de "modo de bloque" para proteger contra la pérdida de datos mediante la corrección de errores de las opciones de "redundancia".
