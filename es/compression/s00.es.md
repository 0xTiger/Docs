## ¿Qué es un archivo S00?

Un archivo con extensión .s00 es la primera parte de un archivo comprimido que se divide con el software ZipSplitter. Dividir los archivos en archivos más pequeños reduce el tamaño total del archivo comprimido y facilita el transporte a través de Internet. Las partes restantes del archivo dividido se nombran con extensiones .S01, .S02, etc. Cuando se supone que se debe extraer el archivo, se hace doble clic en el archivo .EXE incluido con el archivo dividido. El EXE busca el archivo .S00 y usa los otros archivos adjuntos para la extracción de todo el archivo.

## Formato de archivo S00

Los archivos S00 se guardan en el disco como un archivo comprimido binario. Sin embargo, los detalles del formato de archivo interno del formato de archivo S00 no están disponibles públicamente. Los archivos grandes como .[WMV](/es/video/wmv/) o [.MP4](/es/video/mp4/) se suelen dividir con el software ZipSplitter para reducir su tamaño.
