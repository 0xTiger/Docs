## ¿Qué es un archivo DMG?

Un archivo con una extensión .dmg es un archivo de imagen de disco de Apple y se conoce como archivo de imagen de disco de Mac OS X. El formato de archivo DMG es el archivo de imagen de disco más utilizado destinado a asignar aplicaciones y archivos entre las PC de Apple. Es muy conveniente montar archivos DMG en una unidad virtual o puede usar el software Apple Disk Utility para abrir y ver el contenido del formato de archivo DMG. La versión anterior de los archivos IMG utilizados en Mac OS Classic se sustituye por el formato de archivo DMG. Los archivos DMG se generan para usuarios de Mac y no son compatibles con el sistema operativo Windows, pero los usuarios pueden acceder al contenido DMG en el sistema operativo Windows mediante el uso de herramientas de compresión y descompresión como 7-Zip, Pea Zip, etc. La extensión de archivo DMG también es utilizada por la herramienta Oracle Export and Ingress para archivos chatarra guardados en el formato binario de la base de datos Oracle.

## Breve historia

La extensión de archivo DMG fue desarrollada por Apple y se usa principalmente en computadoras Macintosh. Principalmente, el archivo DMG se usa para brindar a los dispositivos Mac la capacidad de instalar diferentes tipos de archivos y software que se inician desde Internet. El archivo DMG es fundamentalmente una imagen de disco montable que se muestra en su escritorio cuando se abre. El archivo comprende datos sin procesar que generalmente están encriptados y comprimidos. Los sistemas Mac tratan DMG de la misma manera que tratarían un disco intercalado, e instantáneamente intentan abrir o "rastrear" el archivo.

## Especificaciones técnicas del formato de archivo DMG

* Especialmente generado para computadoras Macintosh
* Desarrollado por Apple
* El formato de archivo es binario
* La categoría de archivo es archivo de imagen de disco
* El programa compatible es MacOS
* Puede contener una tabla de particiones
* Puede contener cualquier tipo de sistema de archivos como HFS+
* Funcionalidad para comprimir las imágenes
