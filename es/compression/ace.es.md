## ¿Qué es un archivo ACE?

Un archivo con extensión .ace es un formato de archivo comprimido ACE desarrollado por Marcel Lemke. Es el formato de archivo principal utilizado por la aplicación de compresión WinAce que se desarrolló y está disponible solo para la plataforma Microsoft Windows. El formato se desarrolló como una alternativa al formato de archivo [ZIP](/es/compression/zip/) pero con la descontinuación de WinAce, el formato no se pudo llevar a cabo por mucho tiempo.

## Formato de archivo ACE

ACE es un formato de archivo patentado y sus detalles técnicos no están disponibles públicamente. Ofrecía una relación de compresión más alta que ZIP y [RAR](/es/compression/rar/), junto con una velocidad de compresión rápida. También es compatible con el cifrado robusto Blowfish de 160 bits.
