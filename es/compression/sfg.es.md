## ¿Qué es un archivo SFG?

Un archivo SFG es el formato de archivo utilizado por Synfig Studio para guardar archivos de proyecto. Synfig Studio es un software de animación 2D gratuito y de código abierto que permite a los usuarios crear animaciones de alta calidad utilizando mapas de bits y gráficos vectoriales. El archivo SFG contiene toda la información necesaria para recrear una animación, incluidas las capas, el tiempo y los fotogramas clave para cada capa. También incluye cualquier imagen o recurso utilizado en el proyecto, así como cualquier pista de audio u otros medios.

## Formato de archivo SFG

Los archivos SFG se guardan en un disco en formato de archivo ZIP comprimido. Otros formatos de archivo compatibles con [Synfig Studio](https://www.synfig.org/) Compressed Project incluyen [SIFZ](/es/compression/sifz/) y SIF.
