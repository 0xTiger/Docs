## ¿Qué es un archivo MBW? ##

Un archivo con la extensión .mbw se asocia principalmente con MBRWizard de Firesage Solutions. El Master Boot Record Wizard (MBRWizard) realiza una copia de seguridad del MBR de su computadora y lo guarda en un archivo MBW que se puede restaurar si es necesario. Además de las copias MBR y VBR (Volume Boot Record), estos archivos pueden contener GPT, sectores definidos por el usuario o imágenes de disco completo. El MBR es un registro vital de 512 bytes en su disco duro que permite que su sistema operativo arranque. También puede contener la tabla de particiones y una firma de 32 bits que identifique el medio de disco.

## Sistemas operativos compatibles ##

El formato de archivo MPW solo es compatible con el sistema operativo Windows.

## Posibles problemas al abrir el archivo ##

Si no puede abrir y ejecutar el archivo MBW, puede haber uno de los siguientes problemas posibles:

* Archivo MBW corrupto
* Enlaces incorrectos al archivo MBW en las entradas de registro
* Descripción eliminada del MBW del registro de Windows
* Un archivo infectado con un malware no deseado
* El ordenador no dispone de recursos de hardware suficientes para operar con el archivo MBW
* Los controladores utilizados por la computadora para abrir un archivo MBW están desactualizados
