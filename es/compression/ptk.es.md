## ¿Qué es un archivo PTK? ##

Los archivos PTK se consideran archivos de datos utilizados por Intuit Quicken. Es una aplicación de administración de finanzas personales que ayuda a que la extensión de archivo .ptk funcione correctamente. Tiene diferentes funciones, como la importación de datos directamente desde los bancos participantes, informes sobre todos los datos financieros y descarga diaria de acciones y precios de fondos. El formato de archivo PTK es el tipo de archivo anterior que Quicken utiliza para guardar información financiera que ahora se sustituye por el formato GIF. El objetivo principal de los archivos PTK era importar datos de recursos en línea, como bancos e instituciones financieras, a la base de datos de Quickens.
