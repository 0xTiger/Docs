## ¿Qué es un archivo XAPK?

Un archivo con la extensión .xapk es un archivo de paquete comprimido que se utiliza para instalar aplicaciones de Android en dispositivos móviles. Es un formato de archivo contenedor que incorpora APK y archivos adicionales asociados necesarios para la instalación. El otro archivo asociado es un archivo OBB que almacena archivos adicionales, como gráficos, archivos multimedia y datos de aplicaciones que se requieren para mantener la aplicación en funcionamiento. Los archivos XAPK no son compatibles con Google Play y solo se utilizan para distribuirlos en sitios web de descarga de aplicaciones de Android de terceros. Estos se pueden instalar en un dispositivo Android usando el instalador XAPK.

## Formato de archivo XAPK

Los archivos XAPK se comprimen con el formato de archivo estándar [ZIP](/es/compression/zip/). Estos se pueden extraer utilizando un software estándar de compresión/descompresión como WinZIP. Una vez que el archivo XAPK se extrae al disco, contiene los siguientes archivos en la carpeta.

* APK: archivo de instalación estándar para instalar la aplicación en dispositivos Android
* OBB: archivo adicional que contiene archivos de recursos relevantes
