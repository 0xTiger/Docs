## ¿Qué es un archivo ALZ? ##

Un archivo con extensión .alz es un archivo comprimido, principalmente relacionado con ALZip de ESTsoft, Corp. El formato de archivo ALZ como archivo comprimido también se define como archivos ALZip. Los desarrolladores han desarrollado la herramienta de compresión de archivos llamada ALZip y el formato de archivo ALZ utiliza la misma tecnología de codificación y compresión. Este software es creado por ESTsoft para su distribución privada bajo versiones ampliamente utilizadas de Microsoft Windows. Con el tiempo, la herramienta de compresión y descompresión de archivos ALZip extendió su provisión a más de 36 registros de módulos diferentes y programas de compresión y sustento para el algoritmo ZIP.

## Breve historia ##

El formato de archivo ALZ se lanzó en 1999. ALZip y el formato de archivo ALZ generalmente no imponen ningún límite en el tamaño del archivo y el límite de expansión. Este software permite que el sistema de Windows dicte las limitaciones de tamaño para sus archivos de salida.
