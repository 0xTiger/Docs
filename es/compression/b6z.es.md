## ¿Qué es un archivo B6Z?

Un archivo con la extensión .b6z es un archivo comprimido creado con la aplicación de software macOS [B6Zip](http://b6zip.com) que se utiliza para comprimir y descomprimir archivos. Es un formato de archivo comparativamente nuevo que permite una mayor tasa de compresión. B6Z tiene una arquitectura abierta y admite tamaños de archivo de hasta 900 000 PB. Es compatible con la compresión de datos, la recuperación de errores y la distribución de archivos. El software de utilidad B6Zip está disponible de forma gratuita en MacOS para abrir diferentes tipos de archivos comprimidos, incluido B6Z.

## Formato de archivo B6Z

El formato de archivo B6Z se basa en una arquitectura abierta y proporciona una compresión sólida con el algoritmo de cifrado AES-256. Utiliza LZMA2 como método de compresión predeterminado, pero también admite otros métodos de compresión como se indica a continuación.

|Método|Descripción|
---|---|
|LZMA |Versión optimizada del algoritmo LZ77|
|LZMA2| Versión mejorada de LZMA|
|Desinflar| Algoritmo LZ77 normal |
|BZip2| Algoritmo BWT estándar |
|PPMD| PPMdH de Dmitry Shkarin|

La utilidad B6Zip es compatible con todos estos estándares de compresión y está altamente optimizada, lo que da como resultado una alta velocidad. Admite trabajar con formatos de archivo [ZIP](/es/compression/zip/), [TAR](/es/compression/tar/) y [RAR](/es/compression/rar/). Los archivos B6Z tienen aplicación de tipo MIME/x-b6z-compressed.
