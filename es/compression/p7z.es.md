## ¿Qué es un archivo P7Z?

Un archivo P7Z es un archivo comprimido que contiene un mensaje de correo electrónico en el estándar de cifrado S/MIME. Fue presentado por Microsoft Corporation y se clasifica como archivo de datos relacionados con el correo electrónico. Los archivos P7Z tienen el tipo de medio application/pkcs7-mime. Contiene una o más partes de un mensaje que se comprime con el algoritmo de compresión zlib.

## Formato de archivo P7Z - Más información

No se conocen los detalles del formato de archivo interno del formato de archivo P7Z. Algunas API como [Aspose.Email](https://products.aspose.com/email/) le permiten leer la parte S/MIME de los mensajes. Si un cliente de correo electrónico no tiene la capacidad de manejar los datos comprimidos, los mensajes S/MIME comprimidos vienen como adjuntos de correo electrónico.
