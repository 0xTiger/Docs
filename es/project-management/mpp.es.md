## ¿Qué es un archivo MPP?

Un archivo con extensión .mpp es un archivo de datos de Microsoft Project que almacena información relacionada con la gestión de proyectos de manera integrada. Microsoft ha desarrollado este formato de archivo propietario para que sea compatible con Microsoft Project (MSP), su producto de software de gestión de proyectos. Además de MPP, MSP puede trabajar con el esquema XML del proyecto. Muchas aplicaciones y API permiten que los MPP se conviertan a otros formatos de archivo. Microsoft ahora tiene un servidor de proyectos en línea donde varios usuarios pueden cargar archivos de administración de proyectos para la colaboración.

## Formato de archivo MPP ##

Microsoft desarrolló y publicó MPP como formato de archivo binario y, hasta la fecha, Microsoft no ha hecho públicas sus especificaciones. La estructura interna del archivo y los detalles, por lo tanto, no están disponibles para los desarrolladores. Hay varias API disponibles en línea que admiten la lectura/escritura en archivos MPP, pero nunca se logra una precisión del 100 %. Microsoft ha introducido varias características en su aplicación de gestión de proyectos a lo largo del tiempo, al igual que las características del formato de archivo MPP.

El archivo MPP utiliza los siguientes tipos MIME:

* aplicación/vnd.ms-proyecto
* aplicación/msproj
* aplicación/msproyecto
* aplicación/proyecto x-ms
* aplicación/x-ms-proyecto
* aplicación/x-dos_ms_proyecto
* aplicación/mpp
* zz-aplicación/zz-winassoc-mpp

## ¿Problemas para abrir un archivo MPP? ##

Aquí está la lista de algunos problemas comunes que pueden surgir y causar un mal funcionamiento del formato MPP:

* Ausencia de software de soporte
* Archivo dañado
* Archivo infectado por virus
* Sin derecho de acceso en el sistema para abrir los archivos
* Unidad obsoleta en su sistema
* Se renombra la extensión del archivo
