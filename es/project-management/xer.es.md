## ¿Qué es un archivo XER?

El formato de archivo XER es un formato de archivo de proyecto propietario utilizado por la aplicación de gestión y planificación de proyectos Primavera P6. Al igual que el formato de archivo [MPP](/es/project-management/mpp/), este formato de archivo almacena datos relacionados con la planificación del proyecto, como tareas, recursos, cronogramas y materiales. Primavera es desarrollado por Oracle y proporciona una planificación completa de proyectos, y encuentra sus aplicaciones en varios campos prácticos, como petróleo y gas, construcción, fabricación, etc. Ayuda a los gerentes de proyectos a planificar, programar y controlar programas a gran escala y proyectos individuales al integrar con la gestión financiera y los sistemas de gestión del capital humano. Muchos software y API pueden leer y convertir archivos XER a otros formatos de archivos de proyectos populares, como XML y MPP.

## Formato de archivo XER ##

Un archivo XER es un archivo de texto y se puede abrir fácilmente con un editor de texto. Abrir datos relacionados con el proyecto en un editor de texto podría ser útil, pero no es fácil comprenderlo estableciendo enlaces manualmente. Este formato es el más utilizado para intercambiar datos con otras bases de datos P6. Tiene un nivel de control relativamente bajo sobre los datos que se han importado desde otro formato. Las ventajas de XER sobre el formato XML equivalente incluyen la capacidad de mover códigos y calendarios globales a nivel de proyecto y asignar códigos, UDF, recursos y calendarios a valores existentes.


## ¿Problemas para abrir un archivo XER? ##

Algunos problemas comunes que pueden surgir y causar un mal funcionamiento del formato XER son los siguientes:

* Ausencia de software de soporte
* Archivo dañado
* Archivo infectado por virus
* Descripción de XER eliminada accidentalmente del registro de Windows
* Sin derecho de acceso en el sistema para abrir los archivos
* Unidad obsoleta en su sistema
* Se renombra la extensión del archivo
