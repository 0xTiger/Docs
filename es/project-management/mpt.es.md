## ¿Qué es un archivo MPT? ##

Un archivo con la extensión .mpt es un archivo de plantilla de [Microsoft Project](https://products.office.com/en-us/project/project-and-portfolio-management-software). Contiene información básica y estructura y configuración de documentos para crear archivos [MPP](/es/project-management/mpp/). Estas plantillas proporcionan configuraciones predeterminadas que son esenciales para un proyecto en particular, como cronogramas o información presupuestaria. Aunque no puede guardar datos específicos del proyecto, como tareas, recursos o asignaciones. Una vez modificados, los archivos de plantilla se pueden guardar como archivos MPP estándar para seguir trabajando con ellos. Un archivo de plantilla maestra, titulado "Global.mpt" está disponible en Microsoft Project, abierto para crear una plantilla individual. Los archivos MPT usan *application/vnd.ms-project* y *application/x-project* como tipos MIME estándar. Windows, Linux y macOS pueden abrir archivos MPT, junto con Moos Project Viewer, Microsoft Project y MPXJ.

## ¿Problemas para abrir un archivo MPT? ##

Aquí está la lista de algunos problemas comunes que pueden surgir y causar un mal funcionamiento del formato MPT:

* Ausencia de software de soporte
* Archivo dañado
* Archivo infectado por virus
* Sin derecho de acceso en el sistema para abrir los archivos
* Unidad obsoleta en su sistema
 *   Extension of the file is renamed
