## ¿Qué es un archivo P6XML? ##

La extensión de archivo P6XML tiene una excelente legibilidad, adaptabilidad y amplio uso que lo convierten en el formato ideal. Primavera P6 Professional tiene la funcionalidad de exportar e importar tanto archivos [XER](/es/project-management/xer/) como archivos XML. XML significa **Lenguaje de marcado extensible**. Este formato general permite a los usuarios compartir información de proyectos entre las bases de datos de Primavera P6. Los datos del proyecto se pueden almacenar fácilmente en un archivo de texto a través del archivo XML; esto simplifica las cosas para que numerosos programas de software intercambien los datos. Dichos datos pueden ser fácilmente compartidos y administrados por el usuario de Primavera P6.

## Ventajas de un formato de archivo P6XML ##

* Es fácil para los usuarios de Primavera definir qué líneas de base se exportan e importan usando el archivo XML
* La exportación XML puede contener todos los planes de acción detallados del proyecto
* Cualquier usuario que use Primavera 6 que no pueda editar, agregar o eliminar ciertos elementos de datos globales, como códigos de actividad, calendarios y recursos, no podrá importarlos.
* Además de la configuración del perfil de seguridad general y del proyecto que utiliza Primavera, no se importarán los elementos de datos que afectarán o cambiarán los datos en la base de datos.

## Exportando el archivo XML desde Primavera P6 ##

Para exportar proyectos de P6 Professional y datos de proyectos a un archivo P6XML, use Oracle Primavera Cloud. Puede descargar el archivo cuando se está formando desde la aplicación de escritorio P6 Professional e importarlo a Primavera Cloud. Antes de operar ininterrumpidamente el proceso de exportación, debe regular si está vinculado a una base de datos P6 Professional o un P6 EPPM. Algunas características del proceso de exportación pueden diferir dependiendo de su base de datos. Al tener configuraciones especializadas P6 con una base de datos competente P6, el proceso de exportación opera localmente en su sistema. Con una base de datos de P6 Professional, puede exportar y abrir fácilmente el archivo XML de P6 desde una configuración de su sistema.

Estos son algunos pasos que describen el proceso de exportación:

* Deseos abiertos proyecto Primavera P6
* Después de eso, seleccione Exportar en la pestaña Archivo
* Seleccione "Primavera P6 – (XML)". Después de eso, haga clic en Siguiente
* Haga doble clic en el proyecto que exportará
* Seleccione los truncamientos de la ruta donde desea colocar el archivo exportado y seleccione "Finalizar".
* Seleccione la opción de cómo desea guardar el archivo
* Si no se necesitan diseños a nivel de proyecto con el archivo de exportación, es fundamental borrar la selección de "Exportar todos los diseños a nivel de proyecto".
* Seleccione finalizar y luego recibirá una confirmación de la exportación XML

### Consejos importantes ###

* Cada vez que se conecte a una base de datos EPPM de P6, el administrador debe ingresar la URL de la fuente de contenido en el campo URL de P6 en la página General de Configuración de la aplicación en P6
* Es conveniente exportar numerosos proyectos a un archivo XML
* Los datos globales que se asignan al proyecto se exportan
* Es conveniente exportar archivos XML sin tener acceso a todos los recursos



## ¿Problemas para abrir un archivo P6XML? ##

Algunos problemas comunes que pueden surgir y causar un mal funcionamiento del formato P6XML son los siguientes:

* Ausencia de software de soporte
* Archivo dañado
* Archivo infectado por virus
* Descripción de P6XML eliminada accidentalmente del registro de Windows
* Sin derecho de acceso en el sistema para abrir los archivos
* Unidad obsoleta en su sistema
* Se renombra la extensión del archivo
