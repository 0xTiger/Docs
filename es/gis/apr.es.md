## ¿Qué es un archivo APR?

Un archivo con extensión .apr es un archivo de proyecto de ESRI ArcView que contiene información de proyecto para un proyecto de ESRI. Estos fueron utilizados por ArcView 3.x y no son compatibles con las nuevas versiones (9.x y 10.x) de ArcGIS. Sin embargo, ArcView 10 puede importar archivos APR. ArcView es un Sistema de Información Geográfica (SIG) que se utiliza para crear, visualizar, analizar y administrar datos geográficos. Hay otros formatos de archivo que utiliza ESRI para almacenar datos, como el popular ESRI ShapeFile ([.shp](/es/gis/shp/)). Los archivos APR se pueden abrir con cualquier editor de texto, como Microsoft Notepad y Notepad ++.

## Formato de archivo APR - Más información

Los archivos APR se almacenan en el disco como archivos de texto sin formato que se pueden abrir con cualquier editor de texto para examinarlos y editarlos.
