## ¿Qué es un archivo NMC?

Un archivo NMC es un archivo de contenido de paquete de mapas creado con el software ESRI ArcGIS Explorer. Puede almacenar varios elementos de un mapa, como superposiciones de imágenes, capas de servicios GIS, capas [KML](/es/gis/kml/), capas ráster, características y otros paquetes. Los archivos NCM se usan para exportar datos de mapas que se pueden usar en otros mapas GIS que están abiertos en ArcGIS Explorer. Esto permite que varios mapas se superpongan entre sí y se muestren.

## Formato de archivo NMC

Los archivos NMC se guardan en formato de archivo binario, y la estructura interna de estos no está disponible. Estos se pueden compartir con otras personas que pueden cargar estos archivos NMC en ArcGIS Explorer en su extremo. Esto hace que los archivos NMC se puedan usar en las computadoras de otros usuarios al enviarlos por correo electrónico a otros.
