## ¿Qué es un archivo NM2?

Un archivo con extensión .nm2 es un archivo de mapa creado con el sistema de navegación Navitel. Almacena datos geoespaciales relacionados con diferentes entidades del mapa, como carreteras, calles, carriles, puntos en el mapa, puntos de referencia y otros datos de GPS. Los archivos NM2 se pueden abrir con el software Navitel en diferentes tipos de dispositivos, como teléfonos Android, dispositivos iOS, Windows Phone y dispositivos Symbian. Estas aplicaciones están disponibles gratuitamente y se pueden descargar e instalar en los dispositivos respectivos.

## Formato de archivo NM2 - Más información

Los archivos NM2 se guardan como archivos binarios y su formato de archivo interno/estructura de datos no está disponible públicamente. Las versiones recientes de Navitel guardan los datos como archivos .nm7. Hay varios archivos de mapas disponibles por Navitel en su [sitio web](https://www.navitel.cz/en/downloads/demo). La [documentación de Navitel](https://www.navitel.cz/en/downloads/documentation) proporciona guías de usuario completas sobre el uso de su sistema de navegación para diferentes tipos de dispositivos y vehículos.
