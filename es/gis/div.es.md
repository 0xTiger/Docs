## ¿Qué es un archivo DIV?

Un archivo con extensión .div es un archivo de proyecto creado con el software [DIVA-GIS](https://en.wikipedia.org/wiki/DIVA-GIS) cuando se crea y guarda un nuevo proyecto de mapeo. DIVA-GIS es un software de mapeo que se utiliza para crear mapas espaciales y aplicaciones de análisis de datos geográficos. El archivo de proyecto consta de una imagen de mapa y capas que se utilizan para mostrar los objetos y las propiedades en el mapa. Además de la información espacial, también almacena información sobre la escala del mapa y el centro.

## Formato de archivo DIV - Más información

Los archivos DIV se almacenan en el disco como formato de archivo propietario de DIVA-GIS y su formato de archivo interno no está disponible para referencia del desarrollador.
