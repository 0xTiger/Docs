## ¿Qué es un archivo MSD?

Un archivo MSD es un archivo de distribución de servicios de mapas creado con el software de mapas SIG ArcGIS Desktop. Contiene el mapa publicado exportado desde un archivo de mapa [.mxd](/es/gis/mxd/). Un archivo MSD contiene información sobre un servicio de mapas alojado en ArcGIS Server. Otros usuarios pueden acceder al mapa alojado a través de Internet. El archivo MSD tiene el documento de mapa, información de capas y propiedades de tabla.

## Formato de archivo MSD

Los archivos MSD se guardan en el disco en formato de archivo binario. Estos son utilizados por la aplicación del servidor ArcGIS para crear y administrar el servicio de mapas con fines de alojamiento. ArcMap también utiliza archivos MSD para conectarse al servicio de mapas y mostrar el contenido en el mapa.
