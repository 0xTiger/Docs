## ¿Qué es un archivo GLB?

Un archivo con extensión .glb es un archivo compartible globalmente generado y utilizado por el software AGI Systems Tool Kit (STK). Contiene información de globo 3D que se puede compartir con otros usuarios para usarla en sus proyectos. El software STK de AGI Systems se utiliza para simular misiones aeroespaciales, sistemas de defensa y sistemas de telecomunicaciones. GLB es el formato de archivo estándar para STK y muchos modelos 3D diferentes como [.gltf](/es/3d/gltf/) se pueden convertir a este formato. Los archivos GLB se pueden abrir con el software AGI Simulation Toolkit, así como con cualquier editor de texto.

## Formato de archivo GLB - Más información

Los archivos GLB almacenan información sobre texturas, imágenes y otros archivos que se utilizan en un globo para un proyecto. Las referencias a la ubicación de los archivos vinculados se almacenan en los archivos GLB y estos archivos adicionales también deben compartirse con otros usuarios mientras se comparte el archivo de globo GLB para una carga adecuada. Los archivos GLB almacenan toda esta información en archivos de texto sin formato que se pueden abrir con cualquier editor de texto.
