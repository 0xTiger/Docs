## ¿Qué es un archivo GST?

Un archivo GST es un formato de archivo de mapa GIS que contiene datos geográficos y un mapa visual de un área mapeada. Lo utiliza **MapX** (desarrollado por UNEP/GRID-Ginebra y UNEP para administrar datos geoespaciales sobre recursos naturales) y el software de mapeo **MapXtreme** (un conjunto de herramientas de desarrollo de software que se usa para crear aplicaciones GIS personalizadas) que son plataformas en línea para la gestión de datos geoespaciales sobre recursos naturales. Un archivo GST contiene información geográfica en forma numérica, así como imágenes de trama que se superponen en el mapa.

## Formato de archivo GST - Más información

Los archivos GST se guardan en un formato de archivo propietario y los detalles de su formato de archivo interno no están disponibles públicamente para referencia del desarrollador.
