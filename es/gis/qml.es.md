## ¿Qué es un archivo QML?

Un archivo con extensión .qml es un archivo XML que almacena información de estilo de capa para capas QGIS. QGIS es una aplicación SIG multiplataforma de código abierto que se utiliza para mostrar datos geoespaciales con la capacidad de organizar datos en forma de capas. Los archivos QML contienen información que utiliza QGIS para representar geometrías de características, incluidas definiciones de símbolos, tamaños y rotaciones, etiquetado, opacidad, modo de fusión y mucho más. A diferencia de los archivos QLR, los archivos QML contienen toda la información de estilo en sí mismos. Los archivos QML se pueden abrir en cualquier editor de texto como Notepad++.
