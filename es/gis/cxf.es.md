## ¿Qué es un archivo CXF?

Un archivo CXF es un archivo GIS guardado en formato de exportación de coordenadas (CXF), desarrollado por [HeightMap.org] (http://heightmap.org/). Contiene datos de imágenes vectoriales de forma organizada y puede incluir datos GIS o gráficos vectoriales, como dibujos.

## Formato de archivo CXF - Más información

El archivo CXF se guarda como archivo de texto sin formato y contiene coordenadas de mapas vectoriales (VMAP) para la representación de ubicaciones geográficas. Algunos de los datos de mapas vectoriales que un archivo CXF puede incluir datos de terreno, información topográfica, puntos de referencia y edificios. Los archivos CXF se pueden exportar a varios formatos de archivo GIS diferentes, como .AI, [.XAML](/es/web/xaml/), [.SVG](/es/page-description-language/svg/), .EMF, [.KML ](/es/gis/kml/) y [HTML5](/es/web/html/).
