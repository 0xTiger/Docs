## ¿Qué es un archivo SDF?

Un archivo de datos espaciales (SDF) es un formato de archivo de geodatabase desarrollado por Autodesk. Almacena datos geoespaciales y es utilizado por el programa MapGuide de Autodesk GIS y AutoCAD Map 3D. El formato de archivo SDF se basa en el formato de archivo de base de datos de un solo archivo SQLite3. Se considera un formato de archivo optimizado para grandes conjuntos de datos de información espacial.

Puede abrir un archivo SDF con Autodesk AutoCAD Map 3D 2022 y Autodesk AutoCAD Civil 3D 2023.

## Formato de archivo SDF - Más información

El formato de archivo SDF se almacena en el disco como archivos binarios. Se basa en la base de datos SQLite que utiliza componentes de almacenamiento de bajo nivel para la serialización binaria de datos. Un archivo SDF contiene varias clases de entidad por archivo y varias propiedades de geometría para cada clase de entidad. Se puede acceder a los datos en un archivo SDF a alta velocidad debido a la indexación de cada propiedad de geometría mediante un árbol R.
