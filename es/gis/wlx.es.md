## ¿Qué es un archivo WLX?

Un archivo WLX es un archivo GIS creado por el software MapSource que viene con los productos de mapas de Garmin. Contiene una ubicación geográfica (en forma de latitud-longitud) en formato XML y una ubicación mundial para un sitio web de mapas como Google Maps o Bing maps. El archivo WLX actúa como un enlace de acceso directo y abre la ubicación almacenada en sitios web de terceros en la posición de latitud-longitud registrada.

Otro archivo similar (aunque no relacionado con GIS) es el formato de archivo [URL](/es/web/url/) que almacena la dirección web y dirige a los usuarios a esta URL cuando se abre.

## Formato de archivo WLX - Más información

Los archivos WLX se almacenan en formato de archivo de texto que se puede abrir en cualquier editor de texto para ver y actualizar su contenido. MapSource es un software basado en GIS que le permite ver mapas, waypoints, rutas y pistas. También puede transferir estos datos espaciales hacia o desde un dispositivo GPS de Garmin.

MapSource ya no es compatible con Garmin y ha sido reemplazado por BaseCamp, que está disponible para descargar de forma gratuita.
