## ¿Qué es un archivo DT1?

Un archivo DT1 es un archivo de datos de elevación de terreno digital (DTED) que contiene datos de terreno espaciados uniformemente. Es utilizado principalmente por aplicaciones de software GIS militar para obtener datos de elevación, pendiente y contorno de superficie. El archivo DT1 es similar a [DT0](/es/gis/dt0/) y [DT2](/es/gis/dt2/) pero con una granularidad de información media.

Los archivos DT1 se pueden abrir con aplicaciones GIS como ESRI ArcGIS Pro, TatukGIS, Blue Marble Geographic Global Mapper y Pitney Bowes MapInfo.

### Formato de archivo DT1

Los archivos DT1 se guardan como archivos binarios en un disco y se pueden leer con muchas aplicaciones GIS. Estos se pueden fusionar en un catálogo de ráster de geodatabase de archivos que se puede usar para crear un gran mosaico.
