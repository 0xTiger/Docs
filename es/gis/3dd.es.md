## ¿Qué es un archivo 3DD?

Un archivo con extensión .3dd es un archivo de datos GIS creado con el software de aplicación ESRI ArcGlobe. Almacena información de rutas donde se ubican los datos 3D en forma de geodatabase. Cuando se abre un documento con el archivo .3dd, aparecerá en la misma forma en que se guardó. ArcGlobe es una aplicación de visualización 3D que le permite ver grandes cantidades de datos GIS en la superficie de un globo. ArcGlobe es una de las varias aplicaciones que se unen con el paquete de software ArcGIS.

## Formato de archivo 3DD: más información

Los archivos 3DD se almacenan en el disco en formato de archivo binario y el formato de archivo interno de estos archivos no está disponible públicamente.
