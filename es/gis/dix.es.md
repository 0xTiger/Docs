## ¿Qué es un archivo DIX?

Un archivo DIX es un formato de archivo propietario creado con el programa de análisis de datos geográficos DIVA-GIS. Consiste en un archivo de proyecto DIVA-GIS (.div) y otras capas de datos GIS a las que hace referencia el proyecto. Los archivos DIX se pueden exportar desde DIVA-GIS y se pueden compartir con otros usuarios que pueden importarlos a DIVA. También puede utilizar archivos DIX para almacenar todos los archivos relevantes relacionados con un proyecto en una ubicación.

## Formato de archivo DIX

Un archivo DIX se almacena en el disco como un archivo binario comprimido. Debido a la compresión, los archivos DIX no ocupan mucho espacio en el disco y pueden enviarse a otros usuarios por correo electrónico. Un archivo DIX puede contener datos ráster y vectoriales, y también puede almacenar tablas de atributos, metadatos y otros datos asociados con los datos espaciales.
