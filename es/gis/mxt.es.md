## ¿Qué es un archivo MXT?

Un archivo MXT es una plantilla de mapa geográfico creada con la aplicación de mapeo GIS, ArcMap, que viene como parte de ESRI ArcGIS Desktop. Se utiliza para almacenar capas de mapa base o diseños de página que se pueden reutilizar para crear más mapas. El uso de plantillas le ahorra tiempo ya que no tiene que reproducir las partes comunes de los mapas. Los archivos MXT se pueden compartir con otros usuarios/organizaciones para construir rápidamente otros mapas.

## Formato de archivo MXT

Los archivos MXT se guardan como archivos binarios, pero los detalles de su estructura interna de archivos no están disponibles públicamente para referencia del desarrollador. Contiene información como las propiedades básicas de un mapa, las posiciones de diseño y la visibilidad de las barras de herramientas de la aplicación. Los archivos MXT permiten a los usuarios crear múltiples mapas con configuraciones predefinidas ya disponibles para ellos.
