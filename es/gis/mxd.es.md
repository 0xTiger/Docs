## ¿Qué es un archivo MXD?

Un archivo MXD es un archivo de documento de mapa utilizado por el software ESRI ArcMap. Es un formato de archivo patentado que almacena diferentes tipos de información, como la distribución y el diseño de los mapas, los datos utilizados para generar mapas y la simbología aplicada a los datos. El archivo MXD es, por lo tanto, un formato de archivo compuesto que contiene varios tipos diferentes de elementos de mapa en un solo archivo. Los archivos MXD son utilizados por el archivo ArcGIS Map Package ([.mpk](/es/gis/mpk/)).

## Formato de archivo MXD

Los archivos MXD se guardan en formato de archivo propietario de ArcGIS y sus detalles internos no están disponibles para referencia del desarrollador. Los elementos incrustados de un archivo MXD se organizan en unidades, conocidas como almacenamientos y flujos.
