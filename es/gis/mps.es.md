## ¿Qué es un archivo MPS?

Un archivo MPS es un archivo de mapa que se utiliza en dispositivos de mano portátiles Pocket PC. Contiene información geográfica sobre un área determinada, como una ciudad o un estado. Se puede cargar en dispositivos Pocket PC para la navegación sin necesidad de obtener datos EN VIVO de ningún servidor de datos geoespaciales. El formato de archivo MPS fue desarrollado por Microsoft para dispositivos portátiles basados en Windows y se usó en su aplicación MapPoint GIS, que se suspendió hace mucho tiempo.

## Formato de archivo MPS

La estructura de archivo interna del formato de archivo MPS no se conoce, ya que se suspendió durante bastante tiempo con MapPoint. Algunos otros formatos populares de archivos de mapas GIS incluyen [ESRI Shape Files](/es/gis/shp/), [KML](/es/gis/kml/) y [KMZ](/es/gis/kmz/).
