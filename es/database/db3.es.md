## ¿Qué es un archivo DB3?
El archivo DB3 es un archivo de base de datos creado por el software SQLite, que es un programa de base de datos autónomo y liviano que crea bases de datos utilizando archivos sin formato; contiene la anatomía de la base de datos, así como los registros de datos; comúnmente utilizado para recuperar o almacenar datos estructurados usando SQL. Estos archivos se pueden usar en dispositivos inteligentes o donde se requiera el mantenimiento de registros u otra gestión de datos, pero en un entorno de poco espacio.


## Formato de archivo DB3
El formato de archivo DB3 está asociado con RDBMS (Sistema de gestión de bases de datos relacionales) SQLite, una opción popular para bases de datos integradas. No hay ninguna extensión de archivo específica definida en un SQLite_3 en su especificación, puede usar extensiones que incluyen [.dbf](/es/database/dbf/) y [.sqlite](/es/database/sqlite/).

### La seccion de la base de datos
Comúnmente SQLite, el formato de archivo db relacionado con la versión 3 se puede considerar como el formato de archivo DB3 utilizado como el formato nativo documentado públicamente para el motor de base de datos SQLite desde junio de 2004. El formato de archivo DB3 es multiplataforma, transferible entre big-endian y arquitecturas little-endian o sistemas de 32 y 64 bits. Estas características hacen de DB3 una opción popular como formato de archivo de aplicación. El archivo principal de la base de datos SQLite_3 (DB3) consta de una o más páginas. Todos los tamaños de todas las páginas son iguales en la misma base de datos. El tamaño de página en bytes es una potencia de dos entre 512 y 65536 inclusive.
