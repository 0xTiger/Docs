## ¿Qué es un archivo ASK?

Un archivo con extensión .ask es un archivo de base de datos creado por askSam Systems, una aplicación de administración de información personal (PIM). Este tipo de archivos se almacenan en carpetas dinámicas y contienen datos de usuario, como mensajes de correo electrónico, documentos, archivos de texto, hojas de cálculo, direcciones y páginas web. Esto permite que el software askSam los busque fácilmente y recupere los datos con fines de investigación. El software askSam, aunque era una aplicación de escritorio simple basada en formularios que almacenaba datos en una base de datos, era popular para organizar y recuperar dichos datos. Los datos del archivo ASK también se pueden exportar a [hojas de cálculo](/es/spreadsheet/).

## Formato de archivo ASK - Más información

Los archivos ASK se guardan en un disco en formato de archivo binario patentado cuyos detalles de formato de archivo interno no están disponibles para referencia del desarrollador.
