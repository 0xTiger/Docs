## ¿Qué es un archivo ACCDE?

Un archivo con la extensión .accde es un formato de archivo de Microsoft Access 2007 para crear archivos de base de datos de solo ejecución para proteger el código de base de datos personalizado. Compila todo el código VBA en ejecutable solo para que no se pueda ver ni editar. El formato de archivo ACCDE es un reemplazo del formato de archivo MDE que usaban las versiones anteriores de Microsoft Access. ACCDE son archivos [ACCDB](/es/database/accdb/) en modo solo ejecutable y más compactos debido a la ausencia de código fuente. El tamaño compacto de estos archivos mejora el uso de la memoria y mejora el rendimiento de la aplicación. Los archivos ACCDE se pueden abrir con Microsoft Access 365.

## Formato de archivo ACCDE

Todos los archivos de Microsoft Access 2007+ utilizan Jet ACE (motor de base de datos de Access) que utiliza la familia de formatos de archivo ACCDB. Sin embargo, Microsoft no proporciona una implementación de código abierto ni una especificación técnica detallada.
