## ¿Qué es un archivo ADP?

Un archivo con extensión .adp es un archivo de proyecto de Microsoft Access que utiliza la arquitectura de componentes OLE DB para proporcionar una conexión directa y eficiente a una base de datos de Microsoft SQL Server. Dichos archivos no contienen las tablas reales, los diagramas de la base de datos u otros elementos de la base de datos. Los archivos ADP se pueden crear con Access 2007 y 2010. Los archivos creados con versiones anteriores de Access también se pueden abrir con Access 2007 y 2010. Los archivos ADP se pueden abrir con Microsoft Access 365.

## Formato de archivo ADP: más información

Un proyecto de Access creado como ADP le permite realizar cambios de diseño en los objetos del servidor SQL, como tablas y vistas. También permite crear, editar y utilizar funciones de SQL, como diagramas de bases de datos, procedimientos almacenados y funciones definidas por el usuario. Esto le da una ventaja sobre la vinculación a una base de datos del servidor SQL donde no puede realizar cambios en el diseño y solo puede vincular a las tablas y vistas del servidor SQL.

### Tipos de datos y compatibilidad con diagramas

**Tipos de datos de fecha/hora**

Los siguientes nuevos tipos de datos de fecha/hora son compatibles con Access 2010.

* TIEMPO
* FECHA
* FECHAHORA2
* DESPLAZAMIENTO FECHAHORA

**Tipos de datos de longitud variable**

Los siguientes tipos de datos de longitud variable se pueden usar en proyectos de Access 2010:

* VARBINA (MÁX.)
* VARCHAR(MAX)
* NVARCHAR(MAX)
