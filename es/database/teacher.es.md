## ¿Qué es un archivo PROFESOR?

Un archivo .TEACHER es un archivo de configuración educativa creado por el software SMART Response. Se utiliza en entornos educativos para proporcionar un entorno completo de gestión de información del aula, listas de clases, horarios, días festivos, perfiles de estudiantes, rutina de exámenes y resultados de evaluaciones. El software de evaluación SMART Response™ está instalado en la computadora que se utilizará para realizar las evaluaciones SMART Response.

## Formato de archivo PROFESOR

Los archivos TEACHER se almacenan en el disco en formato de archivo binario. Sus especificaciones de formato de archivo interno no están disponibles públicamente para referencia del desarrollador. El software SMART Response se puede utilizar para generar informes de estudiantes. Estos luego se pueden compartir entre las computadoras de la escuela.
