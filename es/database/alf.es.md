## ¿Qué es un archivo ALF?

Un archivo con extensión .alf es un registro de transacciones SQL generado por [ACT! CRM](https://www.act.com/)(Software de gestión de relaciones con los clientes). Mantiene un registro de las actividades entre el ACT! y SQL, y se utiliza para restaurar el ACT! base de datos. ¡ACTUAR! CRM almacena la información del cliente, como el nombre, la dirección de correo electrónico, el contacto, la información comercial y el historial de relaciones. Almacena esta información en el [ACT! ADF](/es/database/adf/) archivo que almacena los datos en tablas para vincularlos con otros módulos, como la recuperación automática de información para una llamada entrante.
Los archivos ALF se pueden abrir con el software SwiftPage ACT en el sistema operativo Windows.

## Formato de archivo ALF - Más información

Los archivos ALF se guardan como archivos binarios y su formato de archivo interno no está disponible.
