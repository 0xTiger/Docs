## ¿Qué es un archivo LOG?
Un archivo con extensión .log contiene la lista de texto sin formato con marca de tiempo. Por lo general, los softwares o sistemas operativos registran ciertos detalles de la actividad para ayudar a los desarrolladores o usuarios a rastrear lo que estaba sucediendo en un período de tiempo determinado. Los usuarios pueden editar estos archivos muy fácilmente usando cualquier editor de texto. Por lo general, los sistemas operativos registran los informes de errores o las actividades de inicio de sesión, pero otros softwares o servidores web también generan archivos de registro para rastrear a los visitantes y monitorear el uso del ancho de banda.

## Formato de archivo de registro
El formato de archivo LOG registra los eventos típicos que ocurren ya sea en un sistema operativo, mensajes entre diferentes usuarios de un software de comunicación o cualquier otra ejecución de software. Simplemente podemos decir que ciertos mensajes en ciertas marcas de tiempo se registran en un archivo LOG. Algunos términos comúnmente utilizados para el registro son:
### Registros de eventos
Registra los eventos que ocurren en la ejecución de un sistema para rastrear y comprender la actividad del sistema y diagnosticar problemas. Son importantes para identificar las actividades de sistemas complejos, normalmente en el caso de aplicaciones con muy poca interacción del usuario.
### Registros de transacciones
Casi todos los sistemas de bases de datos mantienen el registro de transacciones, que generalmente no está diseñado como un registro de auditoría para un análisis posterior y no es legible por humanos. Estos registros solo conservan el registro de los cambios en los datos existentes para permitir que la base de datos se recupere de bloqueos u otros errores de datos y mantenga los datos en un estado coherente. Por lo tanto, los sistemas de bases de datos suelen tener registros de eventos generales y registros de transacciones.
### Registros de mensajes
La comunicación textual guardada por Internet Relay Chat (IRC), los programas de mensajería instantánea (IM), los clientes de intercambio de archivos punto a punto con funciones de chat y los juegos multijugador (especialmente los MMORPG) se denominan registros de mensajes. Estos registros generalmente se guardan en archivos de texto sin formato, pero los clientes de MI y VoIP (por ejemplo, Skype) pueden guardarlos en archivos HTML para facilitar la lectura o habilitar el cifrado.
### Registros del servidor
El registro del servidor es en realidad un archivo de registro que contiene una lista de actividades realizadas, creadas o mantenidas automáticamente por el propio servidor. Por lo general, estos archivos no son accesibles para los usuarios generales de Internet, solo para el webmaster u otra persona administrativa de un servicio de Internet.
