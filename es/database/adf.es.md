## ¿Qué es un archivo ADF?

Un archivo con extensión .adf es un archivo de base de datos creado con el software ACT CRM (Customer Relationship Management) para almacenar datos de relación con los clientes. Almacena información sobre el nombre del cliente, correo electrónico, contacto, información comercial e historial de relaciones. Los sistemas CRM se integran más a menudo con los centros de llamadas establecidos por las empresas para recuperar automáticamente la información del usuario para las llamadas entrantes. Los archivos ADF se pueden abrir con el software SwiftPage ACT en el sistema operativo Windows.

## Formato de archivo ADF - Más información

Los archivos ADF se guardan en el disco como archivos binarios y la información de su estructura interna no está disponible públicamente. Otro archivo con la extensión .alf se guarda junto con los archivos ADF que contienen los metadatos necesarios para navegar, respaldar y restaurar el ACT. base de datos.
