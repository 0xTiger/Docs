## ¿Qué es un archivo BAK?

Un archivo .BAK también sirve como archivo de copia de seguridad de la base de datos generado por Swiftpage Act!, que es un software de gestión de relaciones con el cliente (CRM). ¡Este archivo incluye un duplicado de una Ley! base de datos (normalmente un archivo .ADF), preservando eficazmente datos esenciales como contactos de clientes, detalles de la empresa, información del grupo, notas, registros históricos y datos de actividad almacenados en la base de datos.

## ¡Ley Swiftpage!

¡Ley Swiftpage! es una aplicación de software utilizada para la gestión de relaciones con los clientes (CRM). Está diseñado para ayudar a las empresas y organizaciones a gestionar las interacciones con sus clientes, realizar un seguimiento de las actividades de ventas y marketing y almacenar información de contacto esencial. ¡Ley Swiftpage! ofrece funciones como gestión de contactos, marketing por correo electrónico, gestión de calendarios y tareas, automatización de ventas e informes para agilizar y mejorar los procesos de gestión de relaciones con los clientes.

Dentro de Act!, todos los datos relacionados con el cliente se almacenan en un archivo de base de datos dedicado, del que se puede realizar una copia de seguridad a discreción de Act! directivos y administradores. Esta copia de seguridad es especialmente crucial cuando, por ejemplo, una empresa planea actualizar a una nueva versión de Act!. En tales casos, el administrador de TI responsable de Act! podría optar por crear una copia de seguridad de Act! base de datos.

¡Estos actúan! Las copias de seguridad de las bases de datos se guardan en forma de archivos BAK. Normalmente, durante el proceso de copia de seguridad, estos archivos BAK se agrupan en un archivo [.ZIP](/es/compression/zip/). ¡Este archivo ZIP no solo contiene el contenido principal de Act! copia de seguridad de la base de datos, pero también incluye Act! elementos de datos, como documentos adjuntos, plantillas de documentos, plantillas de informes y consultas de bases de datos guardadas. Este enfoque integral de respaldo garantiza que la información crítica del cliente y los recursos relacionados estén protegidos y fácilmente recuperables cuando sea necesario.

## ¿Cómo abrir un archivo BAK?

Para abrir un archivo BAK en Swiftpage Act! en un sistema Windows, puedes usar Act! Diagnóstico. Sigue estos pasos:

1. **¡Acto abierto! Diagnóstico**:
- Haga clic en el menú Inicio de Windows.
- En la barra de búsqueda, escribe `actdiag`.
- ¡Presiona Enter para abrir Act! Diagnóstico.

2. **Acceda a la herramienta Restaurar base de datos**:
- ¡Dentro de Acto! Diagnóstico, navegue hasta el menú "Herramientas".

3. **Elija Restaurar base de datos**:
- En el menú "Herramientas", seleccione "Restaurar base de datos".

4. **Seleccione Restaurar archivo BAK**:
- Se te presentarán dos opciones:
- Para sobrescribir una base de datos existente con los datos de su archivo BAK, elija "Restaurar archivo BAK".
- Para agregar su archivo BAK como una nueva base de datos junto con los datos existentes, seleccione "Restaurar archivo BAK como".

5. **Siga las indicaciones**:
- ¡Acto! Diagnostics lo guiará a través del proceso de restauración de la base de datos contenida en su archivo BAK.
- Es posible que deba especificar la ubicación del archivo BAK y proporcionar cualquier configuración adicional necesaria para la restauración.

¡Sigue estos pasos y actúa! ¡Los diagnósticos restaurarán la base de datos de su archivo BAK, permitiéndole acceder a los datos almacenados en él en Swiftpage Act!
