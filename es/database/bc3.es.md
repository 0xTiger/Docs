## ¿Qué es un archivo BC3?

Un archivo BC3 es un archivo de base de datos para almacenar información relacionada con el formato de intercambio de bases de datos de construcción. Fue desarrollado basándose en el formato de intercambio de bases de datos de construcción estándar español. Un archivo BC3 almacena información que tiene registros de datos que incluyen detalles de los materiales de construcción, el costo de los materiales y el precio de la mano de obra. Los datos se pueden intercambiar entre otras aplicaciones sin convertirlos a otros formatos.

## Formato de archivo BC3

Los archivos BC3 se almacenan como archivos binarios con detalles especificados según los estándares FIEBDC. La implementación de los archivos BC3 ha pasado por diferentes revisiones y existen las siguientes versiones del formato de archivo BC3:

*FIEBDC-3/2004
*FIEBDC-3/2007
*FIEBDC-3/2012
*FIEBDC-3/2016

FIEBDC es la abreviatura de Formato de Intercambio Estándar - Bases de Datos de Construcción en idioma español.
