## ¿Qué es un Archivo CHCK?

Se crea un archivo CHCK como resultado de las comprobaciones de supervisión del estado en Microsoft Exchange Server. Contiene información sobre el estado general y la operativa de la base de datos del servidor. Microsoft Exchange Server se utiliza principalmente como servidor de correo electrónico y le permite mantener sus contactos y calendarios comerciales. Los administradores del servidor Exchange utilizan su EseUtil para comprobar el estado de la base de datos.

## Formato de archivo CHCK

Se desconoce el formato de archivo interno del archivo CHCK y no hay material de referencia disponible para leer/escribir archivos CHCK.
