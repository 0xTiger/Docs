## ¿Qué es un archivo FMP12?

Un archivo FMP12 es un archivo de base de datos creado con el programa de desarrollo de bases de datos FileMaker Pro 12. Contiene la estructura general de la base de datos (también conocida como esquema de la base de datos) junto con los datos que contiene. La estructura de un archivo FMP12 se basa en tablas, donde cada tabla consta de varios campos de base de datos que tienen diferentes tipos de datos. Algunos tipos de datos compatibles con el formato de archivo FMP12 incluyen texto, formularios personalizados, informes, fecha y hora, blobs y otros objetos de base de datos. Los formatos de archivo anteriores de FileMaker Pro como [FP7](/es/database/fp7/) se pueden actualizar a FMP12.

## ¿Cómo convertir archivos antiguos de FileMaker Pro a FMP12?

Formatos de archivo anteriores de FileMaker Pro, como bases de datos 11, 10, 9, 8 y 7 (.fp7) a formatos de archivo FMP12. FileMaker tiene pautas extensas para [convertir la versión anterior de los formatos de archivo de FileMaker Pro a FMP12](https://fmhelp.filemaker.com/help/16/fmp/en/index.html#page/FMP_Help/converting-files.html) .
