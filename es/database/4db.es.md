## ¿Qué es un archivo 4DB?

La estructura de la base de datos creada con 4th Dimension, un programa de diseño de software que comprende código 4D, interfaces e información de recursos, se conoce como 4DB. Se utiliza en el marco de diseño de software integrador de 4D solo durante el proceso de desarrollo. Los archivos de 4DB se encuentran tanto en plataformas de escritorio como móviles y el sistema operativo de Windows también es una opción. La calificación de popularidad de los principales archivos 4DB es "Baja", lo que significa que los archivos 4DB generalmente están presentes en un dispositivo móvil o de escritorio típico.

El archivo de estructura 4DB tiene todas aquellas especificaciones que están relacionadas con la estructura de la base de datos, como propiedades de campo, campos y tablas, listas de opciones, contraseñas, menús, métodos y formularios. El archivo recibe el nombre que ingresa en el cuadro de diálogo de creación de la base de datos, seguido de ".4db".

Hay dos formas de conectar un documento con una nueva aplicación (por ejemplo, my-file.4DB). El primer y más sencillo método es hacer clic derecho en el archivo 4DB en cuestión. Seleccione "Elegir programa predeterminado" en el menú de otoño, luego haga clic en "Examinar" para ubicar el software que desea. Al hacer clic en Aceptar, debe confirmar toda la operación. El próximo y más complejo paso es usar el Registro de Windows para vincular el tipo de datos 4DB al software apropiado.
