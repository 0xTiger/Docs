## ¿Qué es un archivo ACCFT?

Un archivo con extensión .accft es un archivo de plantilla de Microsoft Access que contiene la definición de tipos de datos de campo para las tablas de una base de datos. Microsoft Access es un sistema de gestión de bases de datos relacionales (RDBMS) de escritorio que se utiliza para crear bases de datos, formularios, consultas y vistas basados en tablas. El archivo ACCFT se puede usar para definir los tipos de datos para los campos de la tabla y las tablas de la base de datos se pueden crear usando este archivo de plantilla como punto de partida. Se puede abrir con Microsoft Access 2016 para crear nuevas tablas. Los archivos ACCFT se reconocen por el tipo MIME **aplicación/msaccess**.

## Formato de archivo ACCFT

Los archivos ACCFT son archivos binarios cuyas especificaciones de formato de archivo no están disponibles públicamente. Sus detalles internos, como la información del encabezado, la estructura del archivo, los pies de página, etc., no están disponibles en este momento.
