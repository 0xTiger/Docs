Una base de datos es una colección de datos que se organiza en tablas y se mantiene en un dispositivo de almacenamiento de computadora y se accede a ella electrónicamente. Las bases de datos relacionales almacenan datos en tablas de bases de datos que están vinculadas entre sí a través de técnicas formales de diseño y modelado. Los datos de una base de datos se analizan mediante el Sistema de gestión de bases de datos (DBMS), que es la principal fuente de interacción con los usuarios finales, las aplicaciones y la propia base de datos.

Las extensiones de archivos de bases de datos populares y sus formatos de archivo incluyen **SQLite**, **DB**, **ACCDB** y **MDB**.
