## ¿Qué es un archivo ABS?

Un archivo con extensión .abs es una base de datos generada y utilizada por Absolute Database, que es un motor de base de datos para el IDE de desarrollo de software de Delphi. Reemplazó al Borland Database Engine (DBE) y es más compacto, de alta velocidad, robusto y fácil de usar. La base de datos ABS se compila como parte del archivo EXE de la aplicación, lo que hace que su aplicación sea más rápida y más pequeña. Los datos se almacenan en un archivo ABS en un formato relacional estructurado.

## Formato de archivo ABS - Más información

Los archivos ABS se almacenan como parte del [archivo EXE](/es/ejecutable/exe/) de la aplicación, que generalmente se almacena como un archivo binario. Los archivos binarios no se pueden abrir en ningún editor de texto y brindan acceso rápido al contenido del archivo de base de datos.

### Características clave del formato de archivo ABS

Los archivos ABS son bastante simples y se integran en el ejecutable de la aplicación. Tiene las siguientes características:

* Sin BDE; sin DLL
* Base de datos de un solo archivo
* Soporte SQL'92 (DDL y DML)
* Compatible con controles de bases de datos estándar y de terceros
* Modo monousuario y multiusuario (servidor de archivos)
* Funciona muy bien en todas las versiones de Windows: desde 98 hasta la última, no requiere actualizaciones ni paquetes de servicio.
* Tablas en memoria ultrarrápidas
* Facilidad de uso inigualable
* Cifrado fuerte
* Compresión de BLOB
* Gratis para uso personal
* Código fuente completo disponible
* Distribución libre de regalías
