## ¿Qué es un archivo de base de datos?

Un archivo con extensión db es un archivo de base de datos genérico para almacenar datos. No hay especificaciones oficiales a seguir para dicho archivo de base de datos. Los datos se organizan en un formato estructurado dentro del archivo en forma de tablas, campos, tipos de datos y valores de campo. Los archivos DB también se pueden crear leyendo archivos de hojas de datos que tienen datos en filas y columnas. El contenido de los archivos DB se puede exportar aún más a los formatos de archivo [CSV](/es/spreadsheet/csv/) y [PDF](/es/pdf/) con fines informativos. Los archivos DB se pueden abrir en aplicaciones como SQLite y LibreOffice.
