## ¿Qué es un archivo ADB?

Un archivo con extensión .adb es un archivo de base de datos creado con [Alpha Five](https://www.alphasoftware.com/) (ahora conocido como Alpha Anywhere). Contiene los nombres de archivo de todas las tablas y conjuntos que son miembros del espacio de trabajo del proyecto. Además, contiene los nombres de todas las bibliotecas que se incorporan en el espacio de trabajo. Alpha Anywhere es un IDE utilizado para desarrollar aplicaciones web. Además de los archivos ADB, el espacio de trabajo de Alpha Anywhere puede contener muchos archivos que comienzan con el nombre de la tabla o del conjunto, pero que tienen diferentes extensiones de archivo.
