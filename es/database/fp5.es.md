## ¿Qué es un archivo FP5?

Un archivo FP5 es un archivo de base de datos creado con FileMaker Pro versión 5 o 6. Almacena datos de usuario y de organización en forma de registros dentro de tablas. FileMaker Pro es un sistema de gestión de bases de datos compatible con los sistemas operativos Macintosh y Windows. FP5 es un formato de archivo más antiguo que era compatible con la versión FileMaker Pro disponible en ese momento. El último formato de base de datos de FileMaker Pro es [FMP12](/es/database/fmp12/) que se introdujo con el lanzamiento de FileMaker Pro v 12. Otro formato de archivo de base de datos muy utilizado fue [FP7](/es/database/fp7/) que fue reemplazado por FMP12.

## Formato de archivo FP5 - Más información

Los detalles del formato de archivo interno del formato de archivo FP5 no están disponibles públicamente para referencia del desarrollador. Los formatos de archivo de base de datos de FileMaker Pro más antiguos, como [FP5 y FP7 se pueden actualizar a la versión más reciente de FMP12](https://fmhelp.filemaker.com/help/16/fmp/en/index.html#page/FMP_Help/converting-files. html) siguiendo las instrucciones detalladas de FileMaker Pro.
