## ¿Qué es un archivo ITW?

Un archivo ITW es un formato de archivo de base de datos utilizado por la aplicación CRM [InTouch](https://www.intouchsoftware.co.uk/InTouch). Almacena contenido en el formato de archivo propietario de InTouch y los detalles de su formato de archivo interno no están disponibles. Los archivos ITW almacenan información personalizada para almacenar nombres, direcciones que luego se utilizan para imprimir sobres y etiquetas. Esta información también se utiliza para enviar correos electrónicos, mensajes de texto, boletines electrónicos atractivos, crear plantillas reutilizables, archivos adjuntos y enlaces. Los datos de CRM también se pueden exportar al formato ITW.
