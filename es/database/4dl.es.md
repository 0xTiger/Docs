## ¿Qué es un archivo 4DL?

Se utiliza un archivo de registro 4DL para registrar las modificaciones realizadas en un archivo de base de datos 4D (.4DD). Este archivo se almacena junto con el archivo de la base de datos y comparte un nombre de archivo similar. Su propósito es rastrear con precisión y mantener un registro completo de las actualizaciones implementadas dentro de la base de datos. Un [archivo 4DD](/es/database/4dd/), en comparación con el archivo 4DL, está asociado principalmente con 4th Dimensions por 4D Inc.

## Formato de archivo 4DL - Más información

Normalmente, se almacenan varios archivos 4DL junto con un archivo 4DD. Por lo tanto, la primera versión del archivo de registro puede denominarse 0001.4dl, pero las versiones laterales de los archivos de registro se guardarán como 0002.4dl, 0003.4dl, etc. Ahora, si el archivo de registro se guarda tres veces más, se etiquetará como "db1[0005-0003].4dl".
