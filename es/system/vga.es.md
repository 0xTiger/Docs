## ¿Qué es un archivo VGA?

Un archivo VGA es un archivo de unidad de Windows compatible con monitores VGA. Se utilizó en las primeras versiones de Microsoft Windows para el ajuste de monitores VGA a nivel del sistema operativo (SO). Un archivo VGA que se utiliza para permitir a los usuarios ajustar la resolución y la profundidad del color de la pantalla mediante el panel de control del monitor.

## Formato de archivo VGA

El formato de archivo interno del formato de archivo VGA no se conoce como referencia del desarrollador. El sistema operativo Windows hace referencia a él para configurar la apariencia visual del monitor VGA.
