Los archivos de configuración son archivos de configuración y también se conocen como archivos de preferencias; almacenan la configuración tanto para el sistema operativo como para las aplicaciones. Estos archivos no están destinados al acceso del usuario, sino que la aplicación correspondiente los modifica cuando se modifican las preferencias del programa.
