## ¿Qué es un archivo OBI?

El archivo OBI es un archivo de suscripción RSS de Outlook que permite a los usuarios suscribirse a una fuente RSS en Microsoft Outlook. RSS significa Really Simple Syndication, que es un formato de alimentación web que se utiliza para publicar contenido actualizado con frecuencia, como publicaciones de blogs, titulares de noticias o podcasts.

Para suscribirse a la fuente RSS en Outlook, los usuarios pueden hacer clic en el enlace a la fuente RSS en un sitio web o copiar la URL de la fuente RSS y pegarla en el cuadro de diálogo "Agregar nueva fuente RSS" en Outlook. Luego, Outlook comprobará periódicamente la fuente RSS en busca de contenido nuevo y lo mostrará en la carpeta de fuentes RSS del usuario.

Los archivos de suscripción RSS de Outlook tienen la extensión de archivo ".rss". Estos archivos contienen la URL de la fuente RSS y se pueden utilizar para compartir suscripciones a fuentes RSS con otros o para realizar copias de seguridad y restaurar suscripciones a fuentes RSS.

## Formato de archivo OBI - Más información

Los archivos OBI se conocen como archivos OPML (lenguaje de marcado del procesador de esquemas) y contienen una lista de URL de fuentes RSS a las que se han suscrito en Microsoft Outlook.

Cuando el usuario exporta sus fuentes RSS a un archivo OBI, contiene una lista de todas las fuentes RSS a las que se ha suscrito, incluido el título de cada fuente, la URL de la fuente y cualquier otra información relevante. Esto permite a los usuarios transferir fácilmente sus suscripciones a fuentes RSS a otro lector de RSS o hacer una copia de seguridad de sus suscripciones para su custodia.

Los archivos OBI también se pueden utilizar para importar suscripciones a fuentes RSS a Microsoft Outlook u otro lector de RSS. Para importar un archivo OBI a Outlook, los usuarios pueden hacer clic en "Archivo" > "Abrir y exportar" > "Importar/Exportar" > "Importar desde otro programa o archivo" > "Siguiente" > "Importar fuentes RSS desde un archivo OPML" y luego busque la ubicación del archivo OBI en su computadora.

## ¿Cuál es el formato del archivo OBI?

El archivo de suscripción RSS de Outlook, también conocido como archivo OBI, utiliza XML (lenguaje de marcado extensible) para definir la estructura y el contenido del archivo.

Los archivos OBI se componen de una serie de elementos anidados, cada uno con su propio conjunto de atributos y valores. El elemento principal del archivo OBI es el elemento "esquema" que contiene información sobre la fuente RSS, incluido el título de la fuente, la URL de la fuente y cualquier otra información relevante.

Cada elemento de "esquema" también puede contener elementos secundarios que representan subcarpetas o subcategorías dentro de la lista de fuentes RSS. La estructura del archivo OBI permite una fácil organización y gestión de las suscripciones a fuentes RSS y se puede utilizar para transferir suscripciones entre diferentes lectores RSS o para realizar copias de seguridad y restaurar suscripciones.
