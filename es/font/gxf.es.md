## ¿Qué es un archivo GXF?

Un archivo GXF es un archivo de fuente de la herramienta de diseño CAD para Windows, General CADD Pro. El software usa la fuente para insertar texto y otros símbolos en un dibujo CAD. Las fuentes GXF deben estar disponibles para el software General CADD Pro para que un dibujo CAD se muestre correctamente.

Puede abrir un archivo GXF usando [General CADD Pro](https://www.generalcadd.com/oldSite/101release.htm).

## Acerca de General CADD Pro

General CADD Pro ofrece una estructura de comando de dos letras que muchos profesionales del dibujo incorporan y utilizan. Es compatible con el formato de archivo ACAD 2013. También proporciona compatibilidad con los comandos de rumbo/distancia del topógrafo y el comando Campo a meta.
