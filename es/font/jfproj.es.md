## ¿Qué es un archivo JFPROJ?

Un archivo con la extensión .jfproj es una fuente personalizada basada en JSON creada con la aplicación Microsoft Font Maker en Windows 10. Se pueden crear usando un lápiz electrónico con dispositivos de pantalla táctil como tabletas, aunque también se pueden usar otros periféricos de computadora como el mouse en caso de computadoras Los archivos JFPROJ contienen los glifos que se utilizan para representar los caracteres en la pantalla definiendo el espacio entre caracteres y palabras, y el tamaño de la fuente. La aplicación Microsoft Font Maker está disponible para descargar en la tienda de Windows.
