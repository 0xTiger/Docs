## ¿Qué es un archivo VLW?

Un archivo VLW es un formato de fuente utilizado por el software de creación de imágenes y animaciones, **Procesamiento**. Almacena fuentes vectoriales de forma compacta que se puede escalar a cualquier tamaño sin perder calidad. Esto hace que su uso en gráficos y diseño visual. El procesamiento brinda la capacidad de crear imágenes y animaciones, donde se utilizan estos glifos de fuente VLW.

## Formato de archivo VLW - Más información

Los archivos VLW se guardan en el disco como archivos binarios. En Procesamiento, puede usar la función **createFont()** para crear un objeto de fuente a partir de un archivo VLW. Luego puede usar este objeto de fuente para dibujar texto en sus bocetos usando la función **text()**. El procesamiento puede leer archivos VLW mediante el comando **loadFont()**.
