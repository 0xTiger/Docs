## ¿Qué es un archivo TPZ? ##

La extensión de archivo .tpz también se conoce como formato de libro electrónico Kindle Topaz por Amazon. Linux, Mac OS y Windows son algunas de las plataformas en las que se pueden distribuir archivos con extensión de archivo .tpz. Como archivo de libro electrónico Kindle Topaz, el archivo TPZ está formateado para parecerse a los libros electrónicos tradicionales y contiene estándares de compilación de datos patentados de Amazon para el formato TPZ. Este archivo tiene compatibilidad avanzada con fuentes incrustadas, y también se pueden incluir detalles adicionales, como anotaciones y marcadores, en los archivos de libros electrónicos Kindle Topaz, aunque estos detalles de metadatos se almacenan por separado. El archivo TPZ también se puede clasificar como un archivo de datos que se puede descargar de la base de datos en línea de libros electrónicos Kindle de Amazon.

## ¿Problemas para abrir un archivo TPZ? ##

Hay algunos de los problemas comunes que se mencionan a continuación que pueden surgir:

* Archivo infectado. Esto se puede arreglar a través de un software antivirus adecuado para que esté limpio.
* Espacio en disco insuficiente. Para esto, debe cerrar todas las aplicaciones irrelevantes para un mejor funcionamiento
* Los enlaces rotos pueden ser una causa de problemas a veces
* A veces, el archivo no tiene derechos de acceso; en este caso, asegúrese de que los derechos de acceso abran fácilmente el archivo.
