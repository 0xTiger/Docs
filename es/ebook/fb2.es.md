## ¿Qué es un archivo FB2?

Los archivos con la extensión .fb2 son archivos de libros electrónicos de FictionBook 2.0 que contienen la estructura del libro electrónico. Está basado en formato XML y contiene etiquetas especiales para describir cada elemento del libro. Fue desarrollado principalmente para escritos ficticios y literatura, pero no se limita solo a estos. El formato acomoda todos los metadatos, así como el contenido en sí mismo y permite flexibilidad para una serie de operaciones, como el procesamiento automático, la indexación y la conversión a otros formatos. En definitiva, se centra en describir la estructura del archivo en lugar de especificar su apariencia. Varias aplicaciones, así como API, están disponibles para convertir FB2 a varios otros formatos en Windows, MacOS y Linux.

## Formato de archivo FB2

El formato de archivo FictionBook 2.0 fue desarrollado por Dmitry Gribov de FictionBook en Rusia. A diferencia de otros formatos de archivo de libros electrónicos populares como [EPUB](/es/ebook/epub/) que contienen varios archivos XML, se basa en un solo archivo XML que contiene información completa sobre el contenido. Si hay imágenes dentro del archivo como [PNG](/es/image/png/) o [JPG](/es/image/jpeg/), estas se convierten a Base64 y se incrustan dentro del documento. Las [especificaciones de formato de archivo](http://gribuser.ru/xml/fictionbook/index.html.en) para FB2 contienen información para la versión 2.0 del formato.
