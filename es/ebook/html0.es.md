## ¿Qué es un archivo HTML0? ##

HTML0 es el formato de libro electrónico nativo utilizado por **Book Designer** que tiene código HTML y CSS estándar que describe el contenido y el diseño de cada página. **Book Designer** es el programa más común que está funcionando con archivos HTML0 y para cambiar documentos de texto a numerosos formatos de libros electrónicos. El formato de archivo HTML0 es compatible con el software que se puede instalar en el sistema Windows, ya que pertenece a la categoría de archivos de texto. Los archivos HTML0 comprenden HTML normal y se pueden ver y modificar con un editor de texto. Los archivos HTML0 no se pueden vincular a archivos CSS discretos en comparación con los archivos HTML habituales.

## ¿Problema para abrir un archivo HTML0? ##

* El archivo puede estar infectado cuando le resulta difícil abrirlo
* Debe escanear el archivo en busca de virus mediante un escáner antivirus en línea
* Algunos archivos necesitan derechos de acceso para abrirlos
* Si los sistemas tienen recursos inadecuados para abrir archivos HTML0, cierre todas las aplicaciones y luego vuelva a abrirlas
* El archivo puede estar corrupto o tener algún error si lo has descargado o recibido de algún correo
* Use siempre el sistema y los controladores más recientes para que su computadora sea más segura y resuelva todos los problemas relacionados con las extensiones de archivos
