## ¿Qué es un archivo FKB? ##

FKB es una extensión de archivo de libro electrónico desarrollada por Flipkart.com para la aplicación de software Flipkart. Este tipo de extensión de archivo generalmente se encuentra en los usuarios de Windows 10. Hay una especificación en el archivo FBK que puede variar en tamaño desde un par de cientos de kilobytes hasta numerosos megabytes, según la longitud del libro y la cantidad de imágenes incluidas. Para descargar el archivo de la aplicación de Android Flipkart eBooks, primero debe comprar el libro electrónico, luego hacer clic en él en la Biblioteca y el libro comenzará a descargarse. Si no puede ver el documento comprado, actualice el sistema para un mejor funcionamiento.

## Sistemas operativos compatibles ##

Los siguientes sistemas operativos admiten archivos FKB:

* ventanas 7
* ventanas 8
* ventanas 10
* Servidor Windows 2003/2008/2012/2016
* Mac OS X
* FreeBSD
* iOS
* linux
* androide
* NetBSD
* OpenBSD

## Problemas para abrir un archivo FKB ##

* Instalación incorrecta de programas de apoyo. Asegúrese de que se descargue correctamente con una versión actualizada
* Poco espacio de hardware
* Archivos corruptos o infectados. Este problema se puede resolver fácilmente a través de un antivirus instalado en su sistema
