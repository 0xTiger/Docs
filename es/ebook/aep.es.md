## ¿Qué es un expediente AEP? ##

El archivo con extensión .aep (que significa **Activ E-Book Project**) se clasifica como un libro electrónico (eBook). Los archivos AEP se crean utilizando el mismo motor que el de Internet Explorer para garantizar una compatibilidad del 100 %. Contiene las páginas como una persona puede leer un libro físico. Las páginas de los archivos AEP generalmente incluyen texto, imágenes, animaciones e hipervínculos. Además, admite [HTML](/es/web/html/), [DHTML](/es/web/dhtml/), [JPEG](/es/image/jpeg/) y [PNG](/es/image/png/) gráficos, animaciones , [JavaScript](/es/web/js/), VBScript y muchos de los complementos de Internet Explorer. Un archivo AEP se puede guardar como un archivo ejecutable de Windows (EXE) para que se pueda abrir en Windows 95 o versiones posteriores de Microsoft Windows.
 


## Formato de archivo AEP ##

Los archivos de formato AEP se crean utilizando un software AEC (Active E-Book Compiler). Este software utiliza un motor similar al de IE para crear libros electrónicos. El usuario puede crear copias ilimitadas de libros electrónicos sin pagar ningún costo por el uso del programa. El proceso de distribución también es fácil, ya que cada libro electrónico se compone de un programa de Windows autónomo, que se puede distribuir en cualquier medio, como correos electrónicos, descargas de Internet, disquetes, CD-ROM, discos ZIP, etc.

## Posibles problemas al abrir el archivo

Si no puede abrir y ejecutar el archivo AEP; no significa que no tenga un software adecuado instalado en su dispositivo. Puede haber otros problemas que impidan que el archivo funcione correctamente. Los posibles problemas pueden ser uno de los siguientes:

- Corrupción de un archivo AEP
- Enlaces incorrectos al archivo AEP en las entradas de registro
- Eliminada descripción del AEP del registro de Windows
- Instalación dañada de una aplicación que abre el formato AEP
- Un archivo AEP infectado con un malware no deseado
- El ordenador no dispone de recursos de hardware suficientes para operar con el archivo AEP
- Los controladores utilizados por la computadora para abrir un archivo AEP están desactualizados
