## ¿Qué es un archivo TR3? ##

El archivo TR3 es un libro electrónico TomeRaider 3 que brinda soporte para búsqueda y compresión rápidas. Eso puede funcionar correctamente a través del programa relacionado con TomeRaider. Yadabyte es el desarrollador de este software, una empresa británica de software y desarrollo web con sede en el Reino Unido. La aplicación de lectura de libros electrónicos TomeRaider se puede utilizar para el correcto funcionamiento y la gestión del contenido de estos archivos TR3. Este lector de libros electrónicos relacionado se puede instalar en computadoras que se ejecutan en sistemas basados en Microsoft Windows y en muchos dispositivos transferibles. El archivo TR3 pertenece al grupo de archivos de libros electrónicos utilizados en sistemas operativos como Windows 10, Windows 7, Windows 8/8.1, Windows Vista, Windows XP.

Las **etiquetas HTML** específicas de TR3 son las siguientes:

|Etiqueta|Descripción|
---|---|
|\<new> |Para crear archivos TomeRaider a partir de archivos de texto, la etiqueta Nuevo es todo lo que se necesita.<new> etiqueta define el inicio de las páginas.|
|\<CATDEF> ... \</CATDEF> |define el nombre de la categoría|
|\<META> ... \</META> |es una etiqueta utilizada para definir todos los metadatos utilizados en el formato de archivo. Esta etiqueta incluye una serie de parámetros.|
|\<EXPMSG> |Esta etiqueta controla el mensaje que aparece cuando un archivo ha caducado. Cada vez que un usuario intenta ver una página después de que el archivo haya caducado, aparece el mensaje de caducidad en lugar del texto real de la página.|
|\<DIEMSG> |Esta etiqueta es similar a EXPMSG. Se utiliza para mostrar un mensaje cuando el archivo muere.|
|\<ENGMSG> |Esta etiqueta se utiliza para crear el mensaje que se muestra para las páginas cifradas. El mensaje aparece en lugar del texto de la página si el usuario intenta abrir una página cifrada antes de desbloquearla.|
|\<expmsg> ,\<diemsg> y \<encmsg> |Ignorado en el texto de la página. Deben colocarse en la sección de encabezado del archivo antes de la primera<new> etiqueta.|
|\<SELECTION> . \</ SELECTION> |Utilizado para admitir Consultas. Esta etiqueta debe estar antes de la primera<new> etiqueta. Recopila los datos de selección para la consulta del usuario.|
|\<catset> . \</catset> | Se utiliza para asignar nombres de categoría para cada tema.|


## Problemas para abrir un archivo TR3 ##

Aquí está la lista de algunos problemas comunes que pueden surgir y causar un mal funcionamiento del formato de archivo:

* Archivo dañado
* Archivo infectado por virus
* Enlaces incorrectos al archivo TR3 en los accesos al archivo
* Sin derecho de acceso en el sistema para abrir los archivos
* Unidad obsoleta en su sistema
* Eliminación del informe del TR3 del archivo de Windows
* Ponerse en contacto con el desarrollador puede ser una mejor opción para obtener un mejor resultado
