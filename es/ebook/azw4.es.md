## ¿Qué es un archivo AZW4?

El AZW4 es un libro electrónico Amazon Print Replica que Amazon desarrolló para el software de escritorio Kindle y los dispositivos Kindle eReader. El AZW4 no es similar a [MOBI](/es/ebook/mobi/), [AZW](/es/ebook/azw/) o [AZW3](/es/ebook/azw3/). Este formato es en realidad un envoltorio para [PDF](/es/pdf/) o libros electrónicos MOBI con DRM agregado.

## Formato de archivo AZW4

El **AZW4** se ha presentado como una nueva versión del software **Textbooks** que es similar a **PDF**. Un eBook AZW4 contiene un archivo PDF completo en su interior, o puede considerarse como un archivo PDF con un envoltorio llamado AZW4. Principalmente, Kindle eReader utiliza el AZW4. Sin embargo, otras aplicaciones también pueden usar la extensión de archivo .azw4. Incluso los programas de malware pueden crear archivos .azw4. Tenga cuidado con los archivos .azw4 que provienen de una fuente dudosa.
