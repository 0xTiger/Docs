## ¿Qué es un archivo HTXT? ##

El formato HTXT fue diseñado por primera vez por Hanvon y categorizado en el tipo de archivo de texto. Generalmente se usa para archivos de libros electrónicos de Hanvon. El formato de libro electrónico de Hanvon solo es compatible con Windows o es compatible con el software que se puede instalar fácilmente en Windows. El software más común que aprovisiona archivos HTXT es la transferencia al dispositivo a través de una conexión USB. El formato de libro electrónico utilizado por los dispositivos de lectura electrónica WISEreader de Hanvon a menudo incluye texto en chino y se usa para libros en chino, se guarda en un formato doble y puede integrar seguridad DRM, no se puede leer con programas de software de escritorio, pero se puede transferir al dispositivo a través de una conexión USB.


## ¿Problema para abrir un archivo HTXT? ##

Aquí está la lista de algunos problemas relevantes que pueden surgir y que resultaron difíciles para abrir este formato de archivo:

* Expediente vacío o incompleto. Esto propone cortar el contenido del archivo HTXT al descargarlo o copiarlo. La única posibilidad de restauración es descargar el archivo nuevamente
* El archivo puede estar dañado o tener un virus. La mejor opción es el uso de un programa antivirus actualizado para despejar cualquier duda.
* Una falla de vinculación distintiva. El programa no proporciona la versión desactualizada del archivo HTXT. La solución más fácil a este error es un programa de adaptación o una versión anterior del programa.
