## ¿Qué es un archivo HTMLZ? ##

Es un archivo de almacenamiento que comprende numerosos archivos que contienen un sitio web, como archivos HTML, archivos de imagen, código CSS, etc. También puede incluir archivos de metadatos OPF. Los lectores de libros electrónicos pueden abrir fácilmente registros HTMLZ sin necesidad de descargarlos. Los archivos con la extensión de archivo HTMLZ generalmente se pueden iniciar como documentos y publicaciones en un formato especial de archivo HTML comprimido. La extensión de archivo HTMLZ especifica al sistema qué aplicación puede abrir el archivo. Sin embargo, diferentes aplicaciones pueden usar extensiones de archivo similares para diferentes tipos de información. Significa que es posible que un abridor HTMLZ no pueda abrir todos los tipos de archivos HTMLZ. El principal desarrollador de archivos HTMLZ o cualquier otro archivo de libro electrónico fue el código abierto de la aplicación de software Calibre.

## Problemas para abrir un archivo HTMLZ ##

El problema común que surge cuando es difícil abrir y utilizar un archivo HTMLZ es la falta de software adecuado. Puede haber varios otros problemas que también restringen la capacidad de operar el archivo de formato HTMLZ. Aquí algunos de estos problemas son:

* Drivers obsoletos del equipo
* Eliminación involuntaria de la representación de HTMLZ del archivo de Windows
* El archivo puede estar corrupto si le resulta difícil abrirlo en lugar del software apropiado
* Enlaces inadecuados al archivo HTMLZ en los accesos al archivo
* Descarga imperfecta de una aplicación que abre el formato HTMLZ
* El archivo puede estar infectado. Para esto, debe instalar un software antivirus adecuado y auténtico.
* El procesador no tiene suficientes propiedades de hardware para ayudar a abrir el archivo HTMLZ
