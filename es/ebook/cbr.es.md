## ¿Qué es un archivo CBR?

Un archivo con una extensión .cbr es una colección de imágenes almacenadas como archivos binarios comprimidos utilizando el formato de archivo de compresión RAR. Contiene imágenes en formato [JPEG](/es/image/jpeg/), [PNG](/es/image/png/) o [GIF](/es/image/gif/) que representan imágenes en un cómic y se muestran al usuario cuando El archivo se utiliza en lectores de libros electrónicos como Calibre. CBR es similar a [CBZ](/es/ebook/cbz/), excepto que CBZ está comprimido y almacenado en el formato de archivo [ZIP](/es/compression/zip/). Estos archivos pueden estar contenidos dentro de archivos de colección de cómics como [CBC](/es/ebook/cbc/).

## Formato de archivo CBR

Un archivo CBR es un archivo [RAR](/es/compression/rar/) que se almacena como un archivo binario en el disco. Las imágenes dentro de un archivo CBR se almacenan y muestran a los usuarios de manera ordenada cuando las abre un lector de libros electrónicos. Los archivos CBR se pueden descomprimir con cualquier software de descompresión que admita el formato de archivo RAR. Sin embargo, las imágenes descomprimidas de esta manera no están en el mismo orden en que aparecen en la pantalla cuando se abren en el software del lector.
