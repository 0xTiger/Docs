## ¿Qué es un archivo LIT?

Los archivos LIT pertenecen al antiguo **Microsoft Reader** que se usaba para leer libros electrónicos. Los libros electrónicos que muestra Microsoft Reader tienen la extensión **.lit**, que se abrevia como **Literatura**. El formato de archivo LIT consta de un archivo Microsoft ITOL/ITLS HTML Help 2.0, que brinda todo tipo de características sorprendentes como compresión [LZX](/es/compression/lzx/), la representación binaria del material de marcado y datos auxiliares opcionales arbitrarios asociados con cada corriente. El texto del libro es un número arbitrario de flujos de marcado OEBPS 1.0 y un subconjunto de OEBPS 1.0 CSS. Las imágenes son [GIF](/es/image/gif/), [JPEG](/es/image/jpeg/) y [PNG](/es/image/png/).

## Cuánto tiempo mostró Microsoft Reader los archivos LIT

**Microsoft Reader** es una aplicación para leer libros electrónicos que ha sido descontinuada por Microsoft. La primera versión de Microsoft Reader se lanzó en agosto de 2000 y utilizaba su propio formato LIT. Estaba disponible para PDA y computadoras con Windows. El nombre de la aplicación también se usó más tarde para una aplicación (que se lanzó inicialmente con Windows 8) para leer archivos [PDF](/es/pdf/) y [XPS](/es/page-description-language/xps/). La aplicación Microsoft Reader se suspendió en 2018.
