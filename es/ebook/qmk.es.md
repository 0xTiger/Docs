## ¿Qué es un archivo QMK?

Un archivo con la extensión .qmk generalmente se conoce como el formato de archivo **YanCEyDesktop Quickmarks**. YanCEyWare introdujo el formato de archivo YanCEyDesktop Quickmarks. El archivo QMK está relacionado con el archivo [YBK](/es/ebook/ybk/) y es compatible con un programa que está disponible para las máquinas que ejecutan el sistema operativo Windows. Un archivo con una extensión QMK se clasifica como un archivo de tipo texto que permite a los usuarios navegar fácilmente a las ubicaciones guardadas dentro de los libros electrónicos. YanCEyDesktop es el software recomendado para administrar archivos QMK. El software **YanCEyDesktop** fue creado por **D Remington**.

## Problemas para abrir un archivo QMK

Si no puede abrir y ejecutar el archivo QMK; no significa que no tenga el software adecuado instalado en su dispositivo. Puede haber otros problemas que impidan que el archivo funcione correctamente. Los posibles problemas pueden ser uno de los siguientes:

- Corrupción de un archivo QMK
- Enlaces incorrectos al archivo QMK en las entradas de registro
- Descripción eliminada del QMK del registro de Windows
- Instalación dañada de una aplicación que abre el formato QMK
- Un archivo QMK infectado con malware no deseado
- El ordenador no dispone de recursos de hardware suficientes para operar con el archivo QMK
- Los controladores utilizados por la computadora para abrir un archivo QMK están desactualizados
