## ¿Qué es un archivo OPF? ##

El OPF significa formato de paquete abierto. Este formato de archivo es en realidad un archivo [XML](/es/web/xml/) pero con la extensión **.opf**. Este formato de archivo es un formato de paquete OEB (OPF) que puede contener una revista digital u otra publicación que se puede leer con **FlipViewer**. También incluye metadatos que describen el contenido y una lista de páginas, imágenes y texto a los que hace referencia la publicación.

## Detalles técnicos del formato de archivo OPF ##

La extensión de archivo .opf se puede clasificar en varios tipos de archivos. Puede ser un archivo formateado con el estándar establecido por Open Packaging. Los OPF son más que archivos de libros electrónicos, ya que en su mayoría contienen información sobre el libro electrónico, desde el título hasta el escritor, así como otra metainformación al respecto. Es por eso que estos archivos se pueden abrir con el software Adobe Reading. Además, se usa para archivos creados por el programa FlipAlbum. Este programa se utiliza para crear álbumes de fotos digitales y estos archivos OPF se refieren al álbum de fotos creado con el software. Estos archivos contienen configuraciones del álbum; su estructura y referencias de archivo de imagen, y también se puede compartir con otras personas. Los archivos OPF también están asociados con **FlipBook**. Un archivo OPF se puede editar con cualquier editor de texto (p. ej., Notepad o Notepad++) porque se pueden formatear con el formato XML.
