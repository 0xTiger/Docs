## ¿Qué es un archivo AZW1? ##

El AZW1 es un formato de archivo de libro electrónico que también se conoce como **Amazon Kindle Topaz eBook** que se lanzó a través de **Whispernet**. Este formato de archivo se puede usar con el software Kindle en PC y teléfonos inteligentes y es responsable de almacenar los libros electrónicos descargados de la red 3G Whispernet de Amazon.

## Formato de archivo AZW1 ##

Los dispositivos lectores de libros Kindle de Amazon y el software utilizan archivos de formato AZW1 con una extensión .azw1. Estos archivos también se conocen como archivos de libros electrónicos Amazon Topaz, estos archivos AZW1 se utilizan para ensamblar y administrar partes descargadas de archivos de libros electrónicos Amazon Kindle de la red **3G Whispernet** de Amazon. El software Amazon Kindle se puede instalar en un sistema basado en **Microsoft Windows** para ejecutar soporte para estos archivos AZW1 y las características relevantes que están integradas en la aplicación. Las versiones de este programa para entornos Mac también están disponibles. Este software generalmente se usa para comenzar a descargar y ver el contenido de estos libros Kindle de Amazon. Dado que la aplicación Amazon Kindle brinda soporte para archivos AZW1 y también se puede instalar en estos dos sistemas conocidos, los creadores pueden crear fácilmente y compartir correctamente libros Kindle.
