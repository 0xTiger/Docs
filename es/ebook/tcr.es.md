## ¿Qué es un archivo TCR?

A principios de los años 90, se introdujo el formato de archivo TCR para los antiguos dispositivos portátiles **Psion Series 3**. La empresa usó la compresión específica de su plataforma y este formato tiene un soporte muy limitado para el formato de texto. Es un formato de archivo basado en ZVR y proporciona una tasa de compresión comparativamente mejor que PalmDOC. También funciona muy bien con FontMachine. Aunque el formato de archivo TCR pertenece a un dispositivo descontinuado, este formato de archivo se puede leer con algunos lectores electrónicos. Este formato de archivo también se puede ver en un escritorio con Sumatra PDF y con la aplicación Calibre en Windows y Linux.

## Detalles técnicos

Dado que el formato de archivo TCR se basa en un compresor ZVR que es capaz de reducir muchos archivos en aproximadamente un 50% de su tamaño original. El esquema de compresión utilizado por over es un poco antiguo. El archivo comprimido comienza con un diccionario de los 256 símbolos posibles que pueden aparecer en el archivo comprimido. El diccionario contiene una cadena alternativa fija para cada uno de estos símbolos. La descompresión se ha vuelto muy rápida incluso en OPL y puede comenzar en cualquier punto del archivo comprimido.

## Problemas para abrir un archivo TCR ##

Si no puede abrir y ejecutar el archivo TCR; no significa que no tenga el software adecuado instalado en su dispositivo. Puede haber otros problemas que impidan que el archivo funcione correctamente. Los posibles problemas pueden ser uno de los siguientes:

- Corrupción de un archivo TCR.
- Enlaces incorrectos al archivo TCR en las entradas de registro.
- Descripción eliminada del TCR del registro de Windows
- Instalación dañada de una aplicación que abre el formato TCR
- Un archivo TCR infectado con malware no deseado.
- El ordenador no dispone de recursos de hardware suficientes para operar con el archivo TCR.
- Los controladores utilizados por la computadora para abrir un archivo TCR están desactualizados.
