## ¿Qué es un archivo BKK?

Los archivos con la extensión .bkk están siendo utilizados por el software **BookBuddi** y estos archivos se pueden volver a leer con Microsoft Text to Speech Agent. Estos archivos pueden incluir texto, imágenes y pueden constar de varias páginas. Cuando se lee el texto, la sección respectiva se resalta y las páginas se pasan automáticamente.

## Formato de archivo BKK

La aplicación BookBuddi hace referencia al contenido del formato de archivo BKK cuando un usuario ve o edita el archivo BKK asociado. Estos detalles de referencia también incluyen atributos de formato de texto e imagen entre otros elementos de diseño de página ingresados por el creador del libro electrónico BKK. El agente de texto a voz de Microsoft, una aplicación que pertenece a los sistemas, es útil para ayudar a los usuarios a acceder al contenido digital almacenado en estos libros electrónicos BKK, ya que esta aplicación admite una función que lee el contenido de texto de un libro electrónico en audio hablado.
