## ¿Qué es un archivo EBK?
El software **eBook Pro Viewer** utiliza una extensión EBK cuando guarda un archivo de libro electrónico. El eBook Pro es un software que puede crear libros electrónicos y proteger la información con derechos de autor. Un archivo EBK contiene la versión cifrada y comprimida de un libro electrónico que se puede leer con **eBook Pro Viewer**; después de proporcionar las credenciales apropiadas para la autenticación.
