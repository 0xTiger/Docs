## ¿Qué es un archivo YBK? ##

El formato de archivo de libro electrónico YanCEyWare y Microsoft Encarta usan extensiones de archivo YBK. YanCEyWare eBook fue formado por YanCEyWare. Todas las aplicaciones de software están disponibles para dispositivos que ejecutan Windows, admiten archivos YBK. La Enciclopedia Encarta se puede actualizar mediante archivos .ybk sin tener que actualizar todo el programa. YanCEyDesktop es un programa conocido y popular que puede administrar y manejar archivos YBK fácilmente. YanCEyWare Reader utiliza ampliamente este formato de archivo YBK, como una aplicación de lectura de libros electrónicos para Windows Mobile o Pocket PC. Este archivo fue desarrollado con grandes recursos hipervinculados.

## Problemas para abrir un archivo YBK ##

A veces, cuando es difícil abrir y utilizar el archivo YBK, aquí hay una lista de algunos problemas posibles:

* Abrir un archivo YBK corrupto
* Enlaces inapropiados e indebidos al archivo YBK en accesos de registro
* Eliminación involuntaria de la descripción de YBK del archivo de Windows
* Instalación parcial de una aplicación que aprovisiona el formato YBK
*	Infected YBK file with an undesirable malware
