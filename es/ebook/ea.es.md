## ¿Qué es un archivo EA? ##

La extensión .ea se usa para el archivo **Kindle End Action**, que es uno de los formatos de archivo de Kindle. Estos archivos son creados por el **dispositivo de lectura electrónica Kindle de Amazon** para mostrar algo que un autor desea mostrar al final de un libro electrónico. Un archivo EA contiene la información en formato [XML](/es/web/xml/). Entonces, este archivo se puede abrir o ver usando cualquier editor de texto. Este archivo se puede encontrar donde los dispositivos de lectura electrónica Kindle guardan los libros electrónicos. El contenido podría ser etiquetas XML vacías si el cliente no hubiera configurado la información para que se mostrara al final de un libro electrónico.

## Un poco más sobre el formato de archivo EA ##

Un archivo con extensión .ea es un archivo XML que contiene los datos de los clientes que compraron el libro y también compraron una función para mostrar más sobre el autor después de la última página del libro electrónico; incluyendo **ASIN** o **ISBN** para que pueda saltar a la página de la Tienda Kindle del título.

### ¿Qué es un ASIN/ISBN? ###

El ASIN significa **Número de identificación estándar de Amazon**, que en realidad es un código alfanumérico de diez dígitos para identificar de forma única los productos en Amazon. El ASIN es un identificador único para cada producto y se asigna cuando creas un producto de Amazon. Todos y cada uno de los productos de Amazon tienen un ASIN, a excepción de los libros. Amazon utiliza el ISBN (International Standard Book Number) en lugar de un ASIN. Es necesario obtener este identificador de Amazon para sus productos antes de venderlos en Amazon.

## ¿Problema para abrir un archivo EA?

Si no puede abrir y ejecutar el archivo EA; no significa que no tenga un software adecuado instalado en su dispositivo. Puede haber otros problemas que impidan que el archivo funcione correctamente. Los posibles problemas pueden ser uno de los siguientes:

- Corrupción de un archivo EA.
- Enlaces incorrectos al archivo EA en las entradas de registro.
- Descripción eliminada del EA del registro de Windows
- Instalación dañada de una aplicación que abre el formato EA
- Un archivo EA infectado con un malware no deseado.
- El ordenador no dispone de los recursos de hardware suficientes para operar con el archivo EA.
- Los controladores utilizados por la computadora para abrir un archivo EA están desactualizados.
