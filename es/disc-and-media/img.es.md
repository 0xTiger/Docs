## ¿Qué es un archivo IMG?

En el campo de la informática, los archivos con extensiones de archivo .img se ocupan del almacenamiento de imágenes de disco sin procesar de discos como disquetes, discos duros o discos ópticos, etc. Esta imagen sin procesar consiste en una imagen binaria de la fuente con especificaciones de el sector por sector.

El contenido del formato de archivo IMG depende del sistema de archivos del disco especificado para la creación de la imagen. Cuando se trata de CD ROM y DVD, las imágenes de estos archivos contienen datos de los sectores correspondientes junto con los encabezados de control y campos para la corrección de errores según cada sector.

Los archivos IMG tienen contenido específico para el disco, por lo que son compatibles con aquellos programas específicos que son adecuados para la detección de sus sistemas de archivos.

## Formato de archivo IMG ##

Durante los años en que se usaba un disquete con fines de almacenamiento y administración del almacenamiento de datos, estos archivos se desarrollaron para acceder a las imágenes sin procesar del disco. Muchos programas fueron referidos por los archivos IMG. Estos fueron creados por programas de imágenes de disco. En los programas del sistema operativo Windows como power, se utiliza una grabadora de software para manejar estos archivos.

## Especificación técnica ##

Hay otras extensiones de archivo como IMA e IMZ similares a este formato de archivo. El tamaño de estas imágenes siempre tiene la forma de múltiples tamaños de sector diferentes. Este tamaño es de 512 bytes para disquetes y discos duros. El tamaño bruto del sector es 2.352 para los CD Roms y DVD. El tamaño de la imagen de disco sin formato es múltiplo de este tamaño.

Este formato se usa principalmente con programas como RawWrite y WinImage. Estos programas usan este formato para leer y escribir las imágenes de los disquetes. Además, estos archivos se pueden convertir a otros formatos de archivo más populares utilizando algún software como PowerISO.

Como los archivos IMG no tienen datos agregados, excepto el contenido de los discos, estos archivos IMG pueden ser manejados automáticamente por los programas y detectar los sistemas de archivos. Por ejemplo, una imagen de disco habitual de un disquete sin formato comienza con el sector de arranque de FAT y se puede utilizar para identificar el sistema de archivos.

Las imágenes de disco de los medios ópticos suelen ir acompañadas del archivo descriptor. Explica el diseño del disco e involucra información como los límites de las pistas y estos no solo se almacenan en un archivo de imagen sin formato.
