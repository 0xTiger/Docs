## ¿Qué es un archivo CUE?

Un archivo **.CUE**, también conocido como **CDRWIN Cue Sheet**, es un archivo de texto sin formato que contiene información sobre las pistas y el diseño de un CD de audio o una imagen de disco, generalmente en formato BIN o ISO. Estos archivos se utilizan a menudo para describir la estructura y el contenido del disco, lo que permite que el software de grabación de CD o el software de unidad virtual reproduzcan con precisión el disco original.

## Ejemplo de archivo CUE

A continuación se muestra un ejemplo de cómo se vería el archivo ".cue":

```
PERFORMER "Artist Name"
TITLE "Album Title"
FILE "DiscImage.bin" BINARY
  TRACK 01 AUDIO
    TITLE "Track 1 Title"
    PERFORMER "Artist Name"
    INDEX 01 00:00:00
  TRACK 02 AUDIO
    TITLE "Track 2 Title"
    PERFORMER "Artist Name"
    INDEX 01 03:45:21
  TRACK 03 AUDIO
    TITLE "Track 3 Title"
    PERFORMER "Artist Name"
    INDEX 01 07:28:17
  TRACK 04 AUDIO
    TITLE "Track 4 Title"
    PERFORMER "Artist Name"
    INDEX 01 12:15:40
  (Additional tracks go here...)
```

## Hoja de referencia CDRWIN

CDRWIN Cue Sheet es una variación específica del formato de archivo ".cue" utilizado por el software CDRWIN. CDRWIN es un software de creación y grabación de CD/DVD y sus hojas ".cue" están diseñadas para usarse con este software. Estas hojas ".cue" contienen información necesaria para que CDRWIN pueda crear o grabar CD o DVD con precisión.

Aquí hay algunos detalles clave específicos de las hojas de referencia CDRWIN:

1. **Exclusivo de CDRWIN**: las hojas ".cue" de CDRWIN son un formato propietario específico del software CDRWIN. Si bien comparten similitudes con los archivos ".cue" estándar, están diseñados para funcionar con las funciones y configuraciones de CDRWIN.
    






2. **Interfaz fácil de usar**: CDRWIN proporciona una interfaz fácil de usar para crear y editar sus hojas ".cue". Los usuarios pueden agregar o modificar información sobre pistas, índices, espacios y otros parámetros de manera fácil de usar.
    






3. **Tipos de discos compatibles**: Las hojas de referencia CDRWIN se utilizan principalmente para crear varios tipos de CD y DVD, incluidos discos de datos, CD de audio, discos de modo mixto y más. El formato permite a los usuarios especificar el tipo y contenido del disco que desean crear.
    






4. **Control sobre el diseño del disco**: Las hojas de referencia CDRWIN brindan un control detallado sobre el diseño y la estructura del disco, incluido el orden de las pistas, la configuración de pausa/espacio y cualquier otra preferencia específica que pueda tener el usuario.
    






5. **Soporte ISO y BIN**: CDRWIN puede funcionar con formatos de imagen de disco ISO y BIN. La hoja ".cue" especifica qué archivo de imagen se utiliza para el disco, asegurando una sincronización adecuada entre las señales y la imagen.
    






6. **Grabar y extraer**: estas hojas ".cue" son cruciales al grabar discos o extraer pistas de discos usando CDRWIN. Ayudan a garantizar que el producto final coincida con el diseño y el contenido previstos.
    






7. **Copia de seguridad y restauración**: Los usuarios de CDRWIN suelen crear hojas ".cue" cuando realizan copias de seguridad de sus CD o DVD. Estas hojas se pueden utilizar posteriormente para restaurar el contenido del disco, incluido el diseño y la sincronización de las pistas.

## ¿Cómo abrir el archivo CUE?

Las hojas de referencia de CDRWIN están diseñadas específicamente para el software CDRWIN, por lo que normalmente están diseñadas para abrirse y usarse dentro de CDRWIN.

Programas que abren o hacen referencia a archivos CUE

-CDRWIN
- Proyectos inteligentes IsoBuster
- EZB Sistemas UltraISO
