## ¿Qué es un archivo MDS?

El MDS (Media Descriptor Sidecar File) es un archivo asociado con el software Alcohol 120% y Daemon Tools, los cuales se utilizan para crear y administrar unidades virtuales de CD y DVD. El archivo MDS contiene información sobre el diseño de los datos en el disco, incluida la ubicación de todos los archivos y la estructura del disco. Esto permite que la unidad virtual emule el comportamiento de un disco físico, de modo que el software pueda leer los datos del disco virtual como si fuera un disco real.

Cuando desea montar un archivo de imagen en una unidad virtual, el archivo MDS se usa junto con el archivo de imagen correspondiente en un formato como ISO, BIN o CUE. El archivo MDS contiene una descripción del diseño de los datos en el disco y el software de la unidad virtual utiliza esta información para crear un disco virtual. Esto le permite usar el software como si estuviera usando un disco físico, sin tener que insertar físicamente el disco en la unidad.

## Relación con Alcohol 120%

Alcohol 120% es un software popular para crear y administrar unidades virtuales de CD y DVD, y utiliza el formato de archivo MDS para guardar y acceder a imágenes de disco. El formato de archivo MDS es propiedad de Alcohol 120 % y solo puede ser abierto y utilizado por el software Alcohol 120 % u otro software compatible.

## ¿Cómo abrir el archivo MDS?

Para abrir un archivo MDS en Alcohol 120%, debe tener el software instalado en su computadora. Una vez que haya instalado Alcohol 120%, puede abrir el archivo MDS haciendo lo siguiente:

1. Lanzamiento Alcohol 120%.
2. Haga clic en el menú "Archivo" y seleccione "Abrir".
3. Navegue hasta la ubicación del archivo MDS en su computadora y selecciónelo.
4. El archivo MDS ahora se abrirá en Alcohol 120% y se le pedirá que seleccione el archivo de imagen correspondiente (como ISO, BIN o CUE).
5. Después de seleccionar el archivo de imagen, Alcohol 120% montará la imagen en una unidad virtual.

Alternativamente, también puede abrir un archivo MDS haciendo doble clic en él, si Alcohol 120% está configurado como el programa predeterminado para abrir archivos MDS. También puede usar otro software compatible con el formato MDS como Daemon Tools, Virtual CloneDrive, Virtual Drive Manager y MagicISO.
