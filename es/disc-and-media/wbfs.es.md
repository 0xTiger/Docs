## ¿Qué es un archivo WBFS?

Un archivo WBFS (Wii Backup File System) es una imagen de disco de videojuego creada para Nintendo Wii. Contiene un juego de Wii que se puede cargar como una imagen para jugar. Se genera utilizando la herramienta Wiimms WBPS o Wii Backup Manager y se puede jugar en una Nintendo Wii cargándolo desde un USB Loader GX o un emulador de Wii como Dolphin. El juego se almacena en estos dispositivos de almacenamiento externo utilizando el formato de archivo WBFS. La Wii es una plataforma de juegos distribuida por Nintendo. El sistema de archivos WBFS fue desarrollado por el codificador homebrew de Wii Waninkoko.

## Formato de archivo WBFS - Más información

WBFS son archivos binarios que tienen un formato de archivo propietario. Sus especificaciones de formato de archivo no están disponibles públicamente.
