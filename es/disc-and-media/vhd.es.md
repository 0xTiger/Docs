## ¿Qué es un archivo VHD?

VHD es una abreviatura de disco duro virtual. Estos son formatos de archivo que representan una unidad de disco duro virtual. Los datos que se encuentran en una unidad de disco duro física pueden estar involucrados al igual que las particiones del disco duro y el sistema de archivos que comprende las carpetas.

Para una máquina virtual, los archivos VHD se consideran discos duros. En las versiones modernas de Windows, estos están integrados. Este tipo de disco duro virtual específico proporciona la posibilidad de que diferentes sistemas operativos residan en la máquina host común. Brinda una gran oportunidad para que los desarrolladores prueben el software en varios sistemas operativos sin realizar el proceso de instalación de otro disco duro. Hay muchas ventajas de usar este formato de archivo de archivos VHD, ya que estos proporcionan muchas características.


## Breve historia ##

**Connectix** desarrolló este formato de archivo como un producto de PC virtual para su uso. En 2003 Microsoft adquirió Connectix y lanzó otras versiones de este formato como VHDx junto con las mejores características y funcionalidades que faltaban en la primera versión. En 2012, se agregó VHDx a Windows. Se agregó a Windows para agregar más capacidad de almacenamiento, protección contra la corrupción de datos, optimización para evitar la degradación del rendimiento. Estas características se agregaron para discos físicos con sectores grandes.

## Formato de archivo VHD ##

Se accede a un sistema de archivos host para residir los archivos VHD en él. Estos tienen características de la imagen fija del disco de protección que permite al usuario acceder a un archivo al que se le asigna el tamaño de un disco virtual específico. Los archivos pueden modificar el disco duro de una máquina virtual.

Estos archivos VHD se pueden intercambiar fácilmente desde el disco duro virtual al sistema de archivos del host. Este formato también tiene la capacidad de copia de seguridad y restauración y seguridad antivirus. Los archivos VHD pueden admitir formatos específicos, como una imagen de disco duro fija, una imagen de disco duro de diferenciación y una imagen de disco duro dinámica.
