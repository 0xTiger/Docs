## ¿Qué es un archivo DBM?

Un archivo DBM (Daz Brick Material) es un archivo de red Shader Mixer creado con la aplicación de modelado de personajes 3D, **DAZ Studio**. Describe el material del modo 3D, es decir, superficie/desplazamiento/volumen interior. DAZ te permite crear tu propio shader personalizado conectando una serie de ladrillos. Esto se combina colectivamente como un archivo DBM que forma una red de sombreado. Es el archivo DBM que especifica cómo se sombrea un material.

Las aplicaciones que pueden **abrir archivos DBM** incluyen DAZ 3D DAZ Studio 4.

## Formato de archivo DBM

Los archivos DBM se guardan en el disco en formato de archivo [XML](/es/web/xml/).
