## ¿Qué es un archivo 3DM?

Un archivo con extensión .3dm es un formato de archivo de código abierto que se utiliza para el software de gráficos 3D. Contiene modelos 3D junto con sus elementos dependientes, como información de superficie, puntos y curvas. El formato de archivo 3DM fue desarrollado por la iniciativa [openNURBS](https://github.com/mcneel/opennurbs) para transferir con precisión geometría 3D entre aplicaciones. Los archivos 3DM se pueden abrir y convertir con aplicaciones como Rhinoceros, SAP VEViewer, Moment of Inspiration y Right Hemisphere Deep View.

## Formato de archivo 3DM

[openNURBS](https://github.com/mcneel/opennurbs), un conjunto de herramientas de código abierto, contiene las especificaciones de formato de archivo 3DM, documentación, bibliotecas de código fuente C++ y ensamblados .NET 2.0 para leer y escribir el formato de archivo.

### Archivos de ejemplo 3DM

Se pueden descargar algunos archivos 3DM de ejemplo desde la sección openNURBS [examples](https://github.com/mcneel/opennurbs/tree/7.x/example_files). Estos se pueden cargar y mostrar utilizando el kit de herramientas openNURBS.

## ¿Cómo leer o escribir archivos 3DM?

Las bibliotecas [openNURBS](https://github.com/mcneel/opennurbs) permiten que cualquier persona lea y escriba el formato de archivo 3DM sin necesidad de Rhino. Proporciona código fuente C++ para una biblioteca que leerá y escribirá modelos 3D utilizando el formato de archivo openNURBS. Este kit de herramientas también proporciona herramientas de evaluación NURBS y herramientas elementales de manipulación de vistas geométricas y 3D, además de incluir código fuente para varios programas de ejemplo. Los foros de soporte [Rhinoceros](https://discourse.mcneel.com/c/opennurbs/6) son un lugar rico en contenido y discusión para compartir los problemas que enfrenta la comunidad 3DM y obtener las soluciones.
