## ¿Qué es el archivo RVM?

Los archivos de datos **RVM** están relacionados con AVEVA PDMS. El archivo **RVM** es un archivo de proyecto del modelo del sistema de gestión de diseño de plantas de AVEVA. El sistema de gestión de diseño de plantas (PDMS) de AVEVA es el sistema de diseño 3D más popular que utiliza tecnología centrada en datos para gestionar proyectos. Numerosas aplicaciones están disponibles para abrir y convertir archivos RVM a otros formatos como [3DS](/es/3d/3ds/).

## Formato de archivo RVM

No se pueden encontrar muchos detalles sobre el formato de archivo RVM, aparte de que se puede guardar en formato binario y ASCII. El archivo RVM admite entidades como:

* Toda la geometría
* Atributos almacenados en grupos
* Texturas (a través de archivo RVS)
* Cámaras y pistas de cámara (a través de archivo RVS)
* Planos de clip (a través de un archivo RVS)
* Rótulos (a través de archivo RVS)
* Etiquetas (a través de archivo RVS)
* Etiquetas (a través de archivo RVS)
* Translucidez (a través de archivo RVS)
* Puntos de origen PDMS
