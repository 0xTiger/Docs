## Acerca de FBX a STL

FBX, FilmBox, es un popular formato de archivo 3D desarrollado por Kaydara para MotionBuilder. Fue adquirido por Autodesk Inc en 2006 y ahora es uno de los principales formatos de archivos de intercambio de datos 3D utilizados por muchas herramientas 3D. [STL](/es/cad/stl/), abreviatura de estereolitografía, es un formato de archivo intercambiable para la representación de la geometría de superficies tridimensionales.

Es un requisito común convertir [FBX](/es/3d/fbx/) al formato de archivo STL y hay varias aplicaciones disponibles en línea que pueden convertir FBX a STL GRATIS. Si es un desarrollador de aplicaciones que está interesado en incorporar la funcionalidad FBX a STL en su aplicación .NET, Java o Python, puede considerar usar las API Aspose.3D para este propósito.
