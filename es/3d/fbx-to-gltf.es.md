## Acerca de FBX a GLTF

FilmBox, comúnmente conocido como FBX, es un formato de archivo 3D ampliamente adoptado inicialmente diseñado por Kaydara para complementar MotionBuilder. En 2006, Autodesk Inc. tomó el mando, impulsando a FBX a una posición destacada como formato principal de intercambio de datos 3D adoptado por una multitud de herramientas de software 3D.

glTF, abreviatura de GL Transmission Format, sirve como un formato de archivo 3D diseñado para encapsular datos de modelos 3D dentro de la notación JSON. Esta implementación con JSON sirve para reducir eficientemente las dimensiones de los activos 3D, así como también para disminuir la sobrecarga computacional requerida para su extracción y utilización durante el tiempo de ejecución.

Es un requisito común convertir [FBX](/es/3d/fbx/) al formato de archivo GLTF y hay varias aplicaciones disponibles en línea que pueden convertir FBX a GLTF GRATIS. Si es un desarrollador de aplicaciones y está interesado en incorporar la funcionalidad FBX a GLTF en su aplicación .NET, Java o Python, puede considerar utilizar las API Aspose.3D para este propósito.
