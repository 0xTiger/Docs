## ¿Qué es un archivo DRC?

Un archivo con extensión .drc es un formato de archivo 3D comprimido creado con la biblioteca Google Draco. Google ofrece Draco como biblioteca de código abierto para comprimir y descomprimir mallas geométricas 3D y nubes de puntos, y mejora el almacenamiento y la transmisión de gráficos 3D. Codifica los datos de entrada y los guarda como archivo .drc. En el extremo receptor, la API lee archivos .drc y puede generarlos como archivos [PLY](/es/3d/ply/) u [OBJ](/es/3d/obj/). El archivo de salida comprimido permite a los usuarios descargar aplicaciones más rápido y proporciona una carga rápida de gráficos 3D en los navegadores.

## Formato de archivo DRC

Los detalles del formato de archivo para DRC no están disponibles públicamente. Google ha alojado la biblioteca de Draco en [GitHub](https://github.com/google/draco/tree/master/docs) para referencia del desarrollador. Proporciona codificadores y decodificadores para crear y convertir archivos DRC.
