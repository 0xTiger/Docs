## ¿Qué es un archivo 8SVX? ##

Un 8SVX es el formato de archivo estándar de audio desarrollado por Electronic Arts. Se considera como un subtipo de datos del formato de archivo IFF. El archivo se utiliza para almacenar datos de audio de 8 bits en fragmentos. En fragmentos de descriptores separados, se incluyen metadatos sobre el flujo de datos 8SVX. Es un formato de archivo de audio digital y lo utilizan las PC Amiga. Los sistemas informáticos de Amiga ahora rara vez se usan, por lo que estos archivos 8SVX generalmente se utilizan para conservar y escuchar la música antigua.

## Formato de archivo 8SVX

El archivo 8SVX puede almacenar grabaciones de audio en calidad de 8 bits que se deriva de la especificación [IFF](/es/audio/aiff/). Estos son archivos que contienen grabaciones de audio como otros archivos que contienen audio. También incluía el formato contenedor de IFF. En este formato, se incrustaron múltiples formatos de datos en un archivo y se usaron para obtener más detalles de los flujos incluidos en él. Estos se pueden usar solos dentro de los contenedores de archivos de solo audio de los archivos IFF. Por otro lado, la multiplexación con otros subtipos de IFF puede generar flujos de video animados.

## Breve historia del formato 8SVX

Fue lanzado inicialmente en 1985 por Steve Hayes y Jerry Morrison. En 2009 se desarrolló otro formato de archivo. Este formato de archivo se utilizó en los sistemas informáticos Amiga para música antigua. También se conoce como archivo de sonido de 8 bits de Amiga. El software de audio hecho para PC Amiga admitía este formato de archivo. Fue utilizado inicialmente por muchos programas de edición de sonido. La mayoría de sus flujos de datos están codificados mediante flujos PCM. Una versión modificada de Amiga proporcionó un marco decodificador 8SVX. Los formatos RIFF y WAVE co-desarrollados por Microsoft e IBM están hechos bajo la influencia del formato contenedor 8SVX/IFF.

## Especificaciones del formato de archivo 8SVX

Muchos programas están asociados con estos archivos en Windows y Mac OS. Se pueden utilizar muchos programas para abordar estos archivos. Es un formato de audio codificado para Commodore Amiga. Estos archivos admiten funciones como ataque, repetición de sección y liberación. Estas funciones se utilizan para almacenar muestras de instrumentos musicales. La compresión de datos con pérdida de Fibonacci-delta está disponible y da como resultado una relación de compresión del 50 % y una disminución de la fidelidad.
