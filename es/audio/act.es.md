## ¿Qué es un archivo ACT? ##

Un formato ACT o Adobe Color Table (también conocido como archivo Color Lookup Table) es un tipo de base de datos de Adobe Photoshop que almacena un conjunto de colores predefinidos. Puede agregar o eliminar colores al guardar una imagen para publicarla en línea para favorecer una imagen de mayor calidad o un tamaño de archivo más pequeño. Es posible que tenga un formato de audio cifrado ADPCM si no se utiliza con Illustrator. La modulación de código de pulso diferencial adaptable es una técnica de compresión de audio utilizada por varios dispositivos de música y dispositivos de escucha.

El formato de archivo ACT también es utilizado por el sistema de archivos CAD/CAM de Alma. Estas son las pautas utilizadas por las máquinas herramienta 3D para descubrir cómo cortar cualquier cosa. Un reproductor Genesis3D, una acción de desarrollador de juegos de DS o un archivo de esquema de acción del asistente de detalles de FoxPro son ejemplos de ACT.

## Especificación de formato de archivo ACT

Muchas grabadoras que usan el sistema ACT solo lo hacen para la opción de grabación más pequeña; si se mejora la calidad, se usa un formato separado, generalmente WAV, lo que da como resultado documentos mucho más grandes.

Hay varias generaciones de ACT; a partir de junio de 2009, las pistas creadas por dispositivos posteriores simplemente no pueden ser reconocidas por todos los reproductores de música estándar gratuitos o aplicaciones de conversión, dejando solo las herramientas de MP3 incluidas.

La optimización de imágenes para la distribución web se realiza con archivos ACT. Los clientes pueden probar la calidad de la imagen con las bases de datos de imágenes generadas aumentando o disminuyendo la cantidad de colores en CLUT.

Al tocar la opción Tabla de colores y elegir Cargar datos de color en el cuadro de diálogo Exportar para Web, se pueden cargar las tablas de colores de Adobe. El directorio */Presets/en_US/Save* for Web Settings/Color Tables/ contiene varios archivos ACT preconfigurados que venían con el sistema Photoshop.

Los guiones de acto se pueden generar únicamente dentro de Genesis 3D o mediante la transformación de otros modelos 3D al estándar ACT usando Genesis3D Actor Designer
