## ¿Qué es un archivo STY?

El archivo STY normalmente se refiere a un formato de archivo utilizado por los teclados y sintetizadores electrónicos de Yamaha. Los archivos ".sty" de Yamaha contienen datos de estilo que definen el ritmo, los patrones de acompañamiento y otros elementos musicales utilizados por el instrumento y se utilizan para mejorar la reproducción de archivos MIDI proporcionando estilos y arreglos preprogramados.

Para utilizar un archivo .sty con un teclado o sintetizador Yamaha, normalmente transfiere el archivo al instrumento mediante una conexión USB o una tarjeta de memoria. Una vez cargado, el instrumento puede aplicar los datos de estilo para acompañar la reproducción MIDI o tocar en vivo con el estilo seleccionado.

## Archivos STY de Yamaha y Korg

Yamaha y Korg guardan archivos STY en diferentes formatos, lo que significa que no son multiplataforma. Yamaha y Korg son dos destacados fabricantes de teclados y sintetizadores electrónicos y utilizan sus propios formatos de archivo patentados para archivos de estilo. Yamaha proporciona software dedicado para editar y crear estilos, como Style Fixer o Yamaha Expansion Manager. Korg proporciona herramientas de software como GNIStyle o Style Works para editar y crear estilos para teclados Korg. Estos programas permiten a los usuarios modificar estilos existentes o crear otros nuevos según sus preferencias.

## ¿Cómo abrir el archivo STY?

Puede utilizar los siguientes softwares para abrir archivos STY.

- Fijador de estilo (estilos Yamaha)
- GNIStyle (estilos Korg)
- Xtractor de ondas
