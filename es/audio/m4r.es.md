## ¿Qué es un archivo M4R?

El formato de archivo .m4r es un tipo de formato de archivo de audio que se utiliza para los tonos de llamada del iPhone. El formato es esencialmente una versión renombrada del formato contenedor multimedia MPEG-4 Parte 14 (MP4), con la extensión de archivo .m4r que indica que el archivo es un tono de llamada. Los archivos M4R están codificados en el formato Advanced Audio Coding (AAC) y se pueden crear utilizando varias aplicaciones de software que admiten la creación de tonos de llamada. Una vez creados, se pueden transferir a un iPhone usando iTunes u otros métodos y luego configurarlos como tono de llamada del dispositivo. Aunque los archivos M4R están diseñados para tonos de llamada de iPhone, el formato no es exclusivo de los dispositivos Apple y se puede reproducir en otros dispositivos que admitan archivos de audio AAC.

## Formato de archivo M4R - Más información

Apple introdujo el formato .m4r en 2007 con el lanzamiento del iPhone. Antes de eso, los tonos de llamada del iPhone estaban en formato .m4a, que también es un formato de audio AAC pero no fue diseñado específicamente para tonos de llamada. Los archivos M4R tienen una duración máxima de 40 segundos, que es la duración máxima permitida para un tono de llamada en un iPhone. Los archivos M4R son el formato utilizado por el iPhone para los tonos de llamada.

Los archivos M4R pueden contener contenido de audio y video, aunque el iPhone generalmente ignora el contenido de video cuando el archivo se usa como tono de llamada. Los archivos M4R se crean cambiando el nombre de un archivo .m4a existente a .m4r. El contenido del archivo no cambia, pero la extensión .m4r le dice a iTunes que el archivo debe tratarse como un tono de llamada.

Los archivos M4R se pueden crear utilizando varias aplicaciones de software, incluido iTunes, que incluye un creador de tonos de llamada incorporado. También se pueden utilizar aplicaciones de software de terceros, como GarageBand y Ringtone Maker, para crear archivos M4R. Los archivos M4R se pueden transferir a un iPhone usando iTunes o sincronizando el archivo con el iPhone usando servicios de almacenamiento en la nube como iCloud o Dropbox.
