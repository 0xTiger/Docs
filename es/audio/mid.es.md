## ¿Qué es un archivo MID?

Un archivo que tiene la extensión de archivo .MID o .MIDI es el archivo de interfaz digital de instrumentos musicales (MID). Es la diferencia con los archivos de audio ordinarios como [WAV](/es/audio/wav/) o [MP3](/es/audio/mp3/) en que no lleva el contenido de audio real; por lo tanto, es bastante más pequeño en tamaño. El contenido incluye las notas tocadas, su tiempo, duración y volumen deseado para cada nota.

El archivo MID consiste fundamentalmente en información que explica cómo se debe escuchar el sonido cuando se vincula con el dispositivo de reproducción o se carga en un software específico que interpreta datos de audio. Esto lo convierte en una herramienta ideal para compartir datos musicales entre aplicaciones compatibles y a través de una conexión de bajo ancho de banda. El tamaño más pequeño permite un fácil almacenamiento para el que se pueden utilizar incluso disquetes.

### Breve historia del formato de archivo MID

El primer instrumento, el Roland Jupiter-6 y el Prophet 600, se introdujeron con MIDI en 1982. La primera caja de ritmos MIDI, la Roland TR-909, salió en 1983. Posteriormente, se lanzó el primer secuenciador MIDI, el Roland MSQ-700. liberado. NEC PC-88 y PC-98, lanzados en 1982, fueron las primeras computadoras compatibles con MIDI.

## Especificaciones de formato de archivo MID

* Tamaño pequeño: a menudo tan pequeño como 10 KB para una reproducción de un minuto
* Puede ajustar la nota, el tempo y el tono para el rendimiento deseado
* Consta de tres partes: el conector físico, el formato del mensaje y el formato de almacenamiento
* Se puede convertir a MP3, WAV, WMA, FLAC, OGG, [AAC](/es/audio/aac/), MPC en Windows a través de Total Audio Converter.
