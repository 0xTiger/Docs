## ¿Qué es un archivo AST?

Un archivo AST es un archivo de plantilla de hoja de cálculo creado con el software Ability Spreadsheet, un componente de hoja de cálculo de la suite Ability Office. Estos archivos de plantilla se utilizan para crear un conjunto inicial de plantillas de hojas de cálculo que se pueden utilizar para crear archivos [AWS](/es/spreadsheet/aws/). Los archivos AST se pueden abrir con el software de hoja de cálculo Ability.

## Formato de archivo AST: más información

Los archivos AST se guardan en el disco como archivos binarios y se pueden abrir en el software Ability Spreadsheet. Aunque es totalmente compatible con los formatos de archivo XLS y XLSX de Microsoft, las especificaciones del formato de archivo AST no están disponibles públicamente para referencia del desarrollador. Los archivos AST se pueden guardar en el disco en formatos de archivo [PDF](/es/pdf/) y [XPS](/es/page-description-language/xps/).
