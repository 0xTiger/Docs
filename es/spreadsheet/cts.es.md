## ¿Qué es un archivo CTS?

Un archivo CTS es un archivo de hoja de cálculo creado con el programa de organización y entrada de datos multiplataforma de código abierto, **TreeSheets**. Contiene datos jerárquicos ingresados en diferentes formatos dentro de la hoja de cálculo, como notas, listas, calendarios, listas de tareas, notas, mapas mentales y otros formatos. Los archivos CTS solo se pueden abrir con TreeSheets, que es liviano, rápido y funciona en Windows, Linux y macOS.

## Formato de archivo CTS - Más información

TreeSheets guarda archivos CTS en su propio formato de archivo propietario. Las especificaciones de formato de archivo del formato de archivo CTS no están disponibles públicamente y es por eso que los archivos CTS solo se pueden abrir en el software TreeSheets.

### Exportar archivos CTS de TreeSheets a otros formatos

Los usuarios de TreeSheets pueden exportar las hojas de cálculo CTS a varios formatos de archivo diferentes. Los archivos CTS se pueden exportar a los formatos de archivo [HTML](/es/web/html/), [JSON](/es/web/json/), [CSV](/es/spreadsheet/csv/) y [Image](/es/image/) .

## Acerca del software TreeSheets

TreeSheets es un programa de software de código abierto que está disponible para multiplataforma. Le permite crear hojas de cálculo jerárquicas y le da la libertad de poner todo tipo de datos en un solo archivo de hoja de cálculo. La ventaja de usar TreeSheets es que puede poner todas sus discusiones en forma de notas, mapas mentales, datos de texto, imágenes y otros tipos de datos en un solo archivo de hoja de cálculo.
