## ¿Qué es un archivo SXC? ##

El formato de archivo SXC (Sun XML Calc) pertenece a una suite ofimática llamada OpenOffice.org. Este formato generalmente se ocupa de las necesidades de hojas de cálculo de los usuarios, ya que es un formato de archivo de hoja de cálculo basado en XML. El formato SXC admite fórmulas, funciones, macros y gráficos junto con DataPilot, que es una característica increíble porque individualiza automáticamente y proporciona resúmenes de los datos importados sin procesar. Los archivos creados con este software se guardan con la extensión .sxc.


## Formato de archivo SXC ##

Los archivos, que tienen extensiones .sxc, se pueden abrir con Microsoft Excel, que es un software de hoja de cálculo de la suite de Microsoft Office. Hay otro software que puede abrir archivos con la extensión .sxc, es decir, Libre Office, StarOffice, etc.

El tipo de contenido para este formato es:

* aplicación/vnd.sun.xml.calc
* aplicación/vnd.sun.xml.calc.template

## Compatibilidad de archivos ##

Los archivos SXC son compatibles con Apache Open Office calc y los datos también se pueden exportar a archivos de Microsoft Excel y otros formatos.

## Manejo de archivos ##

Los comandos ImportMatrix y ExportMatrix pueden leer y escribir archivos SXC. Cuando el origen es un archivo SXC y el destino también es un archivo SXC, **f** se considera como un archivo de hoja de cálculo Sun XML Calc. Tanto Import como Export Matrix admiten este formato, que se detecta automáticamente cuando no se especifica el formato y el nombre de archivo termina en .sxc.

Para abrir el archivo SXC, se requiere que un programa apropiado esté asociado al archivo. Cada vez que se abra un archivo con la extensión .sxc, asegúrese de que no tenga errores y que no esté dañado. Cualquier error en el archivo puede ser manejado fácilmente por el usuario y esto no requiere soporte técnico la mayoría de las veces.
