## ¿Qué es un archivo DEF?

Un archivo DEF es un archivo de datos creado en [formato de archivo de hoja de cálculo](/es/spreadsheet/) por la aplicación de base de datos SmartWare II, desarrollada por Informix Software. El software II tenía cuatro módulos que incluían una hoja de cálculo con gráficos, un procesador de textos, comunicaciones y una base de datos. La base de datos SmartWare II es una base de datos independiente que también se puede integrar con otras aplicaciones. Las consultas se pueden ejecutar en datos que se pueden extraer y guardar en el formato de archivo DEF. Los archivos de hojas de cálculo DEF se pueden abrir con el software SmartWare, que ahora se ha descontinuado.
