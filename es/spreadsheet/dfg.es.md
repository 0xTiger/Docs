## ¿Qué es un archivo DFG?

Un archivo con extensión .dfg es un archivo de hoja de cálculo creado por Interscape Inc. para el software Interscape Data Flask. Los datos se rellenan en archivos DFG en filas y columnas. Data Flask es un software de hoja de cálculo, diseñado para retener tipos de datos, fuentes e información clave. Puede migrar tablas e índices cambiando solo la cadena de conexión. Ofrece funciones como la línea de comandos de VBScript, fórmulas de VBScript y macros junto con scripts SQL para actualizar y migrar datos. Los archivos DFG se guardan como archivos binarios en el disco y se pueden abrir con el software Interscape Data Flask.

## Formato de archivo DFG - Más información

Los archivos DFG guardan la información de datos de la hoja de cálculo como archivos binarios en el disco. Sin embargo, se desconoce la estructura de datos interna y los detalles de formato de estos archivos.
