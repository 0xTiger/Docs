## ¿Qué es un archivo PMVX?

Un archivo con extensión .pmvx es un archivo de plantilla de hoja de cálculo creado con el programa de hoja de cálculo [PlanMaker](https://www.softmaker.de/softmaker-office-planmaker), que forma parte de Softmaker Office. Los datos se almacenan en filas y columnas en archivos PMVX de la misma manera que se hace en Microsoft Excel. Los archivos PMVX almacenan configuraciones predefinidas como una plantilla que se puede usar para generar archivos PMD y PMDX después de completar los datos. [XLSX](/es/spreadsheet/xlsx/) es el formato de archivo predeterminado para PlanMaker y estos se pueden abrir directamente con Microsoft Excel. También puede editar archivos [XLS](/es/spreadsheet/xls/) y crear archivos [PDF](/es/pdf/) con PlanMaker.

## Formato de archivo PMVX

Los archivos PMVX se almacenan en el disco en el formato de archivo patentado de PlanMaker y se desconocen sus estructuras internas de archivos. En lo que respecta a guardar archivos PMVX, guarda los datos en formato Open XML como archivos XLSX.
