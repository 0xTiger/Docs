## ¿Qué es un archivo XLSMHTML?

Un archivo con la extensión .xlsmhtml es una versión [HTML](/es/web/html/) del archivo XLS original que se exporta desde Microsoft Excel para ver los datos en un navegador. El formato ha sido descontinuado por mucho tiempo por Excel y ya no es compatible. El archivo XLSMHTML exportado seguía el formato de archivo XLS y estos podían abrirse y verse en navegadores web. Excel permite exportar los datos de la hoja de cálculo a los formatos de archivo [MHT](/es/web/mht/) y [MHTML](/es/web/mhtml/), y ahora se usan ampliamente.

## Formato de archivo XLSMHTML - Más información

Los archivos XLSMHTML se generaron con código HTML para representar los datos de la hoja de cálculo en un formato similar a XLS. El código HTML dentro del archivo XLSMHTML está organizado de tal manera que el archivo de salida se parece al archivo original cuando se abre en un navegador web. Excel no hace públicas las especificaciones de formato de archivo del archivo XLSMHTML y no hay tales referencias disponibles para que los desarrolladores comprendan y desarrollen aplicaciones para él.
