## ¿Qué es un archivo XL?

Un archivo con extensión .xl es un archivo de hoja de cálculo de Excel, similar a [XLS](/es/spreadsheet/xls/), que almacena datos en filas y columnas. Era un formato de archivo anterior que ahora Excel ya no admite. Los datos almacenados en archivos XL se pueden exportar a otros formatos, como CSV. Estos también se pueden convertir a PDF utilizando el método Guardar como predeterminado de Excel. Los archivos XL solo se pueden abrir con Microsoft Excel.

## Formato de archivo XL

Los archivos XL se guardan como archivos binarios y sus especificaciones de formato de archivo interno no están documentadas en ninguna parte. Sin embargo, el formato de archivo binario general para [Especificaciones de formato de archivo XLS](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) también se puede considerar si alguien intenta escribir un fragmento de código para aplicar ingeniería inversa a estos archivos.
