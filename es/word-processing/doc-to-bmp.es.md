## Acerca de la conversión de DOC a BMP

Si es usuario de Microsoft Word en su versión anterior a 2007, debe estar familiarizado con el tipo de archivo DOC, que era el formato de archivo predeterminado utilizado por Microsoft Word para guardar sus documentos en ese entonces. Ha estado en uso desde el inicio de Microsoft Word y permaneció allí durante bastante tiempo después de que fue reemplazado por el último formato de archivo DOC. Es posible que los sistemas de gestión de contenidos aún tengan archivos DOC como respaldo que se pueden convertir a imágenes BMP. Las herramientas en línea como la aplicación de conversión Aspose.Word le permiten convertir archivos [DOC](/es/word-processing/doc/) a imágenes [BMP](/es/image/bmp/).

## Conversión de DOC a BMP - Demostración gratuita

La aplicación Aspose.Words es un conjunto de herramientas en línea desarrolladas en base a las populares API de programación de Aspose. Proporciona funciones como visualización de documentos, conversión, compresión, rotación, división, fusión y muchas otras. Puede convertir [DOC](/es/word-processing/doc/) a [BMP](/es/image/bmp/) usando la aplicación Aspose.Words directamente desde su navegador web. No requiere que instales ningún software para realizar la conversión.

### Pasos para convertir DOC a BMP en línea

Puede convertir el formato de archivo DOC a BMP utilizando la [aplicación Aspose.Words DOC a BMP] (https://products.aspose.app/words/conversion/doc-to-bmp) de la siguiente manera.

1. Vaya a la aplicación Aspose.Words para obtener [convertidor de DOC a BMP] (https://products.aspose.app/words/conversion/doc-to-bmp)
1. Cargue su archivo DOC haciendo clic en el botón cargar
1. Una vez hecho esto, presione el botón Convertir. Esto iniciará el proceso de conversión del archivo DOC a BMP en segundo plano.
1. Se le pedirá que descargue el archivo BMP convertido una vez que se complete el proceso de conversión de DOC a BMP.

## API de Aspose.Words

[![Aspose.Words](../try-aspose-words.png)](https://products.aspose.com/words/)

[Aspose.Words](https://products.aspose.com/words/) es un conjunto de API que permite a cualquier desarrollador convertir [DOC](/es/word-processing/doc/) a [BMP](/es/image/bmp/) formato usando líneas simples de código. Está disponible para muchos lenguajes de programación, incluidos [C#](/es/programming/cs/), [Java](/es/programming/java/), [Python](/es/programming/py/) y [C++](/es/programming/cpp/). Además de BMP, las API de Aspose.Words le permiten convertir archivos DOC a muchos otros formatos de archivo populares.
