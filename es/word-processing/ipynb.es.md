## ¿Qué es un archivo IPYNB?

Un archivo con la extensión .ipynb es un archivo de Jupyter Notebook que contiene todo el contenido de Notebook creado con la sesión de la aplicación web de Jupyter Notebook. [Jupyter Notebook](https://jupyter.org/) es una aplicación web interactiva que se utiliza para crear y compartir documentos computacionales para su análisis mediante Python.

Un archivo IPYNB sirve como un registro computacional completo de una sesión e incluye información como entradas y salidas computacionales, funciones matemáticas y una rica representación de los objetos resultantes, como imágenes y texto explicativo.

## Formato de archivo IPYNB

IPYNB se guardan internamente en formato de archivo [JSON](/es/web/json/), que es un formato de archivo estándar abierto para compartir datos. Por esta razón, los archivos IPYNB son legibles por humanos y fácilmente comprensibles. JSON también facilita compartir estos archivos con otros.

## ¿Cómo convertir archivos IPYNB?

Los archivos IPYNB se pueden exportar a varios formatos de archivo diferentes mediante el comando nbconvert de la interfaz de Jupyter. Esto incluye la conversión de IPYNB a:

* PDF
* HTML
* texto reestructurado
* látex

## ¿Cómo ver el archivo IPYNB?

Hay dos métodos para ver o abrir archivos IPYNB.

1. El usuario puede cargar el documento de cuaderno .ipynb en alguna URL pública y abrirlo desde Jupyter Notebook Viewer<nbviewer> que representa el cuaderno como una página web estática desde la URL.
1. Descargue los archivos y ábralos en su propia instancia de Jupyter Notebook
