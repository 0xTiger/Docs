## Acerca de la conversión de DOC a JPG

Si alguna vez trabajó con versiones anteriores de Microsoft Word (es decir, versiones anteriores a Microsoft Word 2007), debe estar familiarizado con el formato de archivo DOC. Era el formato de archivo predeterminado utilizado por Microsoft Word antes de la introducción del formato de archivo DOCX con Microsoft Word 2007. La versión DOC a JPG es un requisito muy común, especialmente cuando Microsoft no permite exportar documentos de Word a imágenes JPG. Además, las últimas versiones de Microsoft Word no son totalmente compatibles con archivos DOC más antiguos. Puede utilizar aplicaciones en línea como la aplicación Aspose.Words para convertir formatos de archivo [DOC](/es/word-processing/doc/) a [JPG](/es/image/jpeg/).

## Conversión de DOC a JPG - Demostración gratuita

La aplicación Aspose.Words es un conjunto de herramientas en línea que le permite convertir sus archivos DOC a varios formatos de archivo diferentes. Puede convertir archivos DOC a JPG con sólo unos pocos clics desde su navegador web. La ventaja de utilizar la aplicación Aspose.Words es que no requiere codificación, es rápida y precisa y, lo más importante, es completamente gratuita.

Durante el proceso de DOC a JPG, cada página del documento se convierte a una imagen JPG y está disponible para descargar. El resultado es un conjunto de imágenes JPG que se guardan en el disco.

### ¿Cómo convertir DOC a JPG?

Entonces, ¿cómo se convierte DOC a JPG? Puede convertir el formato de archivo DOC a JPG utilizando la [aplicación Aspose.Words DOC a JPG] (https://products.aspose.app/words/conversion/doc-to-jpg) de la siguiente manera.

1. Vaya a la aplicación Aspose.Words para obtener [convertidor de DOC a JPG] (https://products.aspose.app/words/conversion/doc-to-jpg)
1. Cargue su archivo DOC haciendo clic en el botón cargar
1. Una vez hecho esto, presione el botón Convertir. Esto iniciará el proceso de conversión del archivo DOC a JPG en segundo plano.
1. Se le pedirá que descargue el archivo JPG convertido una vez que se complete el proceso de conversión de DOC a JPG.
