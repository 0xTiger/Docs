## Acerca de la conversión de DOCX a DOC

El tipo de archivo DOC ha sido un formato de archivo de documento popular utilizado en 2007 y versiones anteriores de Microsoft Word. Con el lanzamiento de Office 2007, Microsoft introdujo el formato de archivo DOCX basado en las especificaciones Office Open XML. Los archivos DOC se guardaron como archivos binarios en comparación con DOCX, que se guarda en formato de archivo XML comprimido. Hay muchas herramientas disponibles en línea para convertir archivos [DOCX](/es/word-processing/docx/) a [DOC](/es/word-processing/doc/).

## Conversión de DOCX a DOC - Demostración gratuita

Las aplicaciones de Aspose funcionan y se ejecutan mediante API de Aspose en el backend que pueden realizar una serie de operaciones en diferentes tipos de archivos. ### Pasos para convertir DOCX a DOC en línea

Puede convertir el formato de archivo DOCX a DOC utilizando la [aplicación Aspose.Words DOCX a DOC] (https://products.aspose.app/words/conversion/docx-to-doc) de la siguiente manera.

1. Vaya a la aplicación Aspose.Words para obtener [convertidor de DOCX a DOC] (https://products.aspose.app/words/conversion/docx-to-doc)
1. Cargue su archivo DOCX haciendo clic en el botón cargar
1. Una vez hecho esto, presione el botón Convertir. Esto iniciará el proceso de conversión del archivo DOCX a DOC en segundo plano.
1. Se le pedirá que descargue el archivo RTF convertido una vez que se complete el proceso de conversión de DOCX a DOC.
