## ¿Qué es un archivo WPS?

Un archivo con extensión .wps es un documento de Microsoft Works que se usaba antes de 2006 como documento de Word. Para entonces, era similar al formato de archivo [DOC](/es/word-processing/doc/) recientemente introducido, pero no tenía tantas funciones como el nuevo formato de archivo. Por ejemplo, no admitía la funcionalidad de macros ni las opciones de formato avanzadas. Los archivos WPS aún se pueden abrir con procesadores de texto modernos, pero para Windows 10, debe instalar un convertidor para cargar estos archivos. Microsoft Works fue reemplazado por el procesador de documentos Microsoft Word en 2010.

## Especificaciones de formato de archivo

Las especificaciones de formato de archivo del documento WPS no están disponibles públicamente ya que Microsoft no abrió las especificaciones a los usuarios finales. WPS son archivos binarios y se pueden abrir con filtros de conversión de formato de archivo creados por Microsoft para el procesador de Word. LibreOffice, NeoOffice y OxygenOffice han incluido una biblioteca general de C++, libwps, que puede extraer texto de muchas versiones diferentes de Microsoft Works.
