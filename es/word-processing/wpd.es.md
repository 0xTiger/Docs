## ¿Qué es un archivo WPD?

Un archivo con una extensión .wpd denominado Archivos de documento de WordPerfect (WPD). Corel WordPerfect se utilizó para producir este documento de texto. Un programa de procesamiento de textos se conoce como **WordPerfect**. Se utiliza en una variedad de sistemas informáticos, incluido Windows. WordPerfect también se utiliza en entornos comerciales, académicos y domésticos para boletines, currículos, invitaciones y cartas junto con otros documentos.

El archivo WPD contiene datos estructurados y los documentos pueden almacenar varios objetos incrustados, como gráficos y gráficos desarrollados en otras aplicaciones. Los documentos WPD también incluyen elementos de combinación de correspondencia, que permiten insertar información de una entidad externa, como una hoja de cálculo o una base de datos, en un diseño. Puede tocar dos veces un archivo WPD y cualquier otro documento en su PC para abrirlo. Si sus asociaciones de documentos son precisas, la aplicación diseñada para abrir su documento .wpd lo hará.


## Breve historia ##

WordPerfect para Mac no se desarrolló en paralelo con ediciones para otras aplicaciones de software y, por lo tanto, utilizó números de versión que no estaban relacionados con las versiones actuales para OS y Windows. La versión 2 fue una renovación completa que se apegó más a los principios de la interfaz de usuario de Apple.

La versión 3 amplió esto, haciendo un uso sustancial de la tecnología introducida por Apple en los sistemas 7.0–7.5 mientras se mantiene rápido y capaz de trabajar de manera efectiva en máquinas anteriores. La versión 3.5 se lanzó en 1996, seguida de la versión mejorada 3.5e en 1997. Nunca se actualizó después de eso y el producto finalmente se eliminó. Corel ha dicho que no tiene intenciones de seguir desarrollando WordPerfect para Mac a partir de 2004.

## Especificaciones del formato de archivo WPD ##

Los documentos WPD se utilizaron principalmente para procesar texto preparado. Como resultado, esos archivos incluyen fragmentos de texto. Las etiquetas se utilizan para contener información de formato, que WordPerfect reconoce como código. Esto hace que encontrar problemas de formato sea mucho más fácil porque la aplicación resalta áreas del texto y expone el formato heredado. Los documentos WPD, al igual que los tipos de archivos de procesamiento de texto más populares, como [DOC](/es/word-processing/doc/), pueden incorporar datos de diferentes fuentes, como archivos, bases de datos o diseños.
