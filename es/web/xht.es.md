## ¿Qué es un archivo XHT?

Un archivo con la extensión .xht es un archivo web asociado con el tipo de archivo [XHTML](/es/web/xhtml/) (lenguaje de marcado de hipertexto extendido), que a su vez es un archivo de marcado basado en [XML](/es/web/xml/). Ambos archivos son similares a HTML pero tienen una sintaxis similar a XML más estricta. Los archivos XHT están diseñados para ser más estructurados, implican menos secuencias de comandos y son genéricos además de utilizar las funciones existentes de XML.

## Formato de archivo XHT

Un archivo XHT se crea y almacena como un archivo de texto sin formato con codificación UTF-8 de forma predeterminada. Contiene el código fuente completo de un documento XHTML que se puede abrir y editar con cualquier editor de texto compatible con la codificación Unicode. Además de los editores de texto, el formato de archivo XHT es comprensible para la mayoría de los navegadores web modernos y se puede abrir con estos.

## Ejemplo XHT

El siguiente ejemplo de un documento XHT define las declaraciones XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
