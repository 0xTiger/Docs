## ¿Qué es un archivo ECE?

Un archivo ECE es una página web generada dinámicamente por la plataforma de administración de contenido, **Escenic Content Engine (ECE)**. Contiene código del lado del servidor que es procesado por el servidor web y devuelto al navegador del usuario como página [HTML](/es/web/html/). El formato de archivo ECE no es muy popular en comparación con otros formatos de archivo del lado del servidor, como ASP y PHP. Estos archivos se procesan internamente y se entregan a los usuarios finales con fines de visualización.

## Formato de archivo CEPE

Los archivos ECE no se guardan en el disco, pero se entregan directamente al usuario cuando lo solicita. Estos se convierten a HTML y se envían de vuelta al navegador del usuario para su visualización. Escenic Content Engine se usa activamente para publicar páginas de artículos de noticias. Basado en la tecnología JEE y SQL, el servidor ECE proporciona un repositorio confiable y escalable para todo el contenido en línea.

Los navegadores de los clientes pueden acceder al contenido de los servidores ECE a través de la API HTTP/REST. Además de servir páginas [HTML](/es/web/html/), ECE proporciona un flujo de trabajo altamente eficiente para administrar y publicar contenido de video/audio.
