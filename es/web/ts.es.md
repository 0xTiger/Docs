## ¿Qué es un archivo TS?

Un archivo TS es un archivo de traducción generado con la utilidad de software lupdate que viene con el Qt SDK de Digia. El SDK está disponible para los sistemas operativos Windows, Mac y Linux. El archivo TS contiene frases en un idioma y sus traducciones a otro idioma. Las traducciones almacenadas dentro de un archivo TS son específicas para una aplicación en particular. Los archivos TS se pueden abrir con Digia Qt SDK.

## Formato de archivo TS

Los archivos Digia Qt TS se almacenan en el disco en formato de archivo XML y se pueden abrir en cualquier editor de texto como Notepad y Notepad++. Algunos otros formatos de archivo compatibles con Digia Qt SDK incluyen:

* .pri
* .qm
* .qml
* .qph
* .qrc
* .qs
* .qss
* .ts
* Archivo de interfaz de usuario .ui
