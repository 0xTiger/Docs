## ¿Qué es un archivo FCGI?

Un archivo FCGI es un archivo de código fuente que contiene código fuente escrito para FastCGI (FCGI). Se puede escribir en varios lenguajes de programación diferentes y contiene la extensión del estándar CGI que se utiliza para un manejo más sólido de las solicitudes del servidor web. Los lenguajes de soporte más comunes para FCGI incluyen **Perl** que brinda soporte tanto para el código Perl como para la sólida interfaz externa de la API FastCGI.

## Formato de archivo FCGI

¿Qué hace que un archivo FCGI sea importante? La interfaz FCGI admite computación distribuida para manejar solicitudes web y distribuirlas entre múltiples servidores. Esto hace que los archivos FCGI se utilicen principalmente para equilibrar la carga en varios servidores.
