## ¿Qué es un archivo CHA?

Un archivo CHA es un archivo de configuración de usuario creado por los clientes de IRC como mIRC. Contiene parámetros como el servidor, el puerto, el nombre del canal y, opcionalmente, una contraseña para abrir un canal IRC. Los clientes de IRC se utilizan para comunicarse con otros en las redes de IRC de todo el mundo. Se utilizan para debates privados uno a uno o en conferencias de varios grupos. Cuando se hace clic en un archivo CHA, el cliente IRC se abre automáticamente con los detalles mencionados en el archivo.

Las aplicaciones que pueden abrir archivos CHA incluyen mIRC y Visual IRC.

## Formato de archivo CHA

Los archivos CHA se almacenan junto con otros archivos de instalación del cliente IRC. Cuando un usuario quiere conectarse a un servidor y unirse a un canal, ingresa la información en el cliente. Esta información se almacena en el archivo CHA que se puede recuperar en cualquier momento.
