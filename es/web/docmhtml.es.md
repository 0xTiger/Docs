## ¿Qué es un archivo DOCMHTML?

Un archivo DOCMHTML es una variante antigua del archivo web guardado por las versiones anteriores de Microsoft Word. Ya no es compatible con la última versión de Microsoft Word. Un archivo DOCMHTML contiene una representación HTML del documento de Word .doc original. Se puede abrir en Microsoft Word, así como en el navegador web. Cuando se publica un DOCMHTML, se puede alojar en un servidor web al que pueden acceder los usuarios a través de una solicitud web. Otro formato similar es [DOCHTML](/es/web/dochtml/) que también ha sido descontinuado.

## Formato de archivo DOCMHTML - Más información

Los archivos DOCMHTML se usaban para guardarse en formato HTML. Se puede abrir en un editor de texto como Notepad y Notepad++. Después de ser marcado como obsoleto, Microsoft Word introdujo las opciones de guardar documentos de Word como archivos [MHTML](/es/web/mhtml/) y [MHT](/es/web/mht/). Estos nuevos formatos de archivo se pueden abrir con navegadores web.
