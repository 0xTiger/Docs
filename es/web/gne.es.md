## ¿Qué es un archivo GNE?

Un archivo GNE es una página web generada dinámicamente y utilizada por el sitio web para compartir fotos de Flickr. Las páginas se alojan en el servidor desde donde se sirven estas al navegador del usuario solicitante. Los archivos GNE son similares a las páginas PHP y ASP, pero están desarrollados en un lenguaje de programación patentado cuyos detalles no están disponibles para referencia del desarrollador.

## Formato de archivo GNE - Más información

Los archivos GNE fueron utilizados por el juego de computadora Game Never Ending, desarrollado por [Ludicorp](https://en.wikipedia.org/wiki/Ludicorp). Flicker fue adquirido posteriormente por yahoo! en marzo de 2005.
