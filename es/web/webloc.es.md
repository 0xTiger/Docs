## ¿Qué es WEBLOC?

Un archivo WEBLOC es un archivo de acceso directo en macOS que apunta a una URL web en Internet. Se genera arrastrando la URL de la web desde la barra de direcciones del navegador al escritorio o cualquier otra carpeta donde deba guardarse. La URL del sitio de destino se almacena dentro de este archivo de acceso directo y se puede ver abriendo el archivo en un editor de texto como Microsoft Notepad, Notepad++ o Apple TextEdit. Los archivos WEBLOC son similares a los archivos [URL](/es/web/url/).

## Formato de archivo WEBLOC - Más información

Los archivos WEBLOC se almacenan en formato de archivo de texto sin formato. Estos se pueden generar en macOS arrastrando URL desde navegadores web como Safari, Google Chrome o Firefox. Si se van a abrir varios archivos WEBLOC simultáneamente, se requerirá una solución más rápida y sencilla para abrirlos a través de aplicaciones externas o API. Estas API pueden leer y convertir WEBLOC a PDF y JPEG.
