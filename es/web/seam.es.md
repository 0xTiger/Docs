## ¿Qué es un archivo SEAM?

Un archivo SEAM es un archivo de configuración utilizado por el marco JBOS Seam. Contiene metadatos/referencia a un servlet del lado del servidor que procesa solicitudes a la aplicación web. También contiene metadatos sobre los componentes de Seam, como los nombres de las clases y métodos que implementan los componentes. Puede utilizar archivos SEAM junto con el marco JavaServer Faces (JSF) para crear aplicaciones web dinámicas e interactivas. Los archivos SEAM generalmente se almacenan en el directorio WEB-INF de una aplicación Seam.

## Formato de archivo SEAM

Un archivo SEAM se almacena como un archivo de texto, normalmente en formato [XML](/es/web/xml/), y contiene información de configuración para una aplicación Seam. Es una parte clave de la configuración de una aplicación Seam. Puede abrir un archivo SEAM con cualquier editor de texto o XML.
