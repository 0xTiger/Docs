## ¿Qué es un archivo ESPROJ?

Un archivo ESPROJ es un archivo de proyecto creado por el software de creación de páginas web, Espresso. Está disponible en Mac OS y contiene una lista de los archivos relacionados que forman parte del proyecto del sitio web. Cuando abre el archivo ESPROJ en Espresso, abre todos los archivos relacionados en el editor para editar o actualizar el código fuente. El archivo ESPROJ también contiene la configuración de publicación del servidor y otras preferencias de usuario relacionadas.

## Formato de archivo ESPROJ

Los archivos ESPROJ se guardan como archivos binarios y su contenido no es legible por humanos. El directorio de archivos ESPROJ también es el directorio raíz del sitio web.
