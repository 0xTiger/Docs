## ¿Qué es un archivo AEX?

Un archivo AEX (con extensión .aex) es un archivo [Xbasic](https://documentation.alphasoftware.com/documentation/pages/Ref/Xbasic/index.xml) compilado que contiene código de programa compilado para funciones globales. Se puede usar en aplicaciones web llamándolas desde el lado del servidor [.a5w](/es/web/a5w/) páginas. Los archivos AEX se cargaron y descargaron llamando a las funciones a5w_load_aex y a5w_unload_aex respectivamente desde archivos A5W. Sin embargo, en una nueva implementación, estos archivos se cargan en la memoria al incluirlos en el archivo a5_application.5i. El lenguaje de programación Xbasic es utilizado por el software Alpha Anywhere para el desarrollo de aplicaciones móviles y web.

## Formato de archivo AEX - Más información

Los archivos AEX se guardan en el disco en formato de archivo binario y contienen un código compilado de funciones escritas en Xbasic. Las declaraciones de comando de script Xbasic determinan la estructura y el flujo de ejecución, incluida la realización o detención de una acción repetida, o la elección entre dos o más pasos en función de las condiciones. [Referencia del lenguaje Xbasic](https://documentation.alphasoftware.com/documentation/pages/Ref/Xbasic/index.xml) se puede consultar para obtener información detallada al respecto.
