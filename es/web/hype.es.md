## ¿Qué es un archivo HYPE?

Un archivo HYPE es un archivo de proyecto guardado con el software Hype que se utiliza para la creación de contenido web con HTML5. Contiene elementos web como gráficos, texto, animación, información de escena y otros elementos que en conjunto conforman el contenido. El contenido web en un archivo HYPE se puede exportar a archivos web que se pueden publicar en una página web.

Un archivo HYPE se puede abrir con el software [Tumult Hype](https://tumult.com/hype/).

## Formato de archivo HYPE

Los archivos HYPE se almacenan en formato de archivo propietario y los detalles internos de estos no se conocen públicamente. El contenido dentro de un archivo HYPE solo se puede ver cuando se abre en el software Tumult Hype.

### Contenido de un archivo HYPE

Un archivo HYPE se crea cuando utiliza la aplicación Hype para crear contenido web para diferentes propósitos. Almacena información sobre animaciones basadas en fotogramas clave para crear contenidos como tarjetas electrónicas, libros electrónicos, presentaciones, anuncios, infografías y material de páginas web. El contenido de estos funciona con HTML5 que permite que el contenido se muestre en dispositivos desde computadoras de escritorio hasta iPhones y iPads. Tumult Hype genera estilos HTML5 de última generación, [CSS3](/es/web/css/) y [JavaScript](/es/web/js/).
