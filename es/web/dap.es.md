## ¿Qué es un archivo DAP?

Un archivo con extensión .dap es un archivo de página de datos de Microsoft Access que se crea desde Microsoft Access y tiene acceso a los datos almacenados en la base de datos. Se puede acceder a los datos almacenados en un archivo web de este tipo a través de la red (intranet e Internet), que es diferente a los archivos [MDB](/es/database/mdb/) de Access que se desarrollan específicamente para aplicaciones de escritorio. Estas bases de datos se pueden crear con Access 2010 y Access Services (un componente opcional de SharePoint). Estas bases de datos pueden ser utilizadas por personas que tienen cuentas de SharePoint utilizando la dirección web en un navegador web.
