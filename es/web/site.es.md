## ¿Qué es un archivo SITIO?

Un archivo SITE es un archivo de proyecto utilizado por el software de diseño de sitios web "Adobe GoLive". Almacena información del proyecto del sitio web, como archivos, carpetas, imágenes, hojas de estilo personalizadas y scripts. También contiene la estructura del sitio web y los enlaces utilizados en las páginas web del sitio. Como es un archivo de proyecto, contiene referencias a archivos de proyecto, pero no contiene los archivos reales.

Adobe GoLive ha sido descontinuado y ya no es compatible.

## Formato de archivo del SITIO

Los archivos del SITIO se almacenan en el disco en un formato de archivo propiedad de Adobe y los detalles internos de su formato de archivo no están disponibles públicamente para referencia del desarrollador.
