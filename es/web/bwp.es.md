## ¿Qué es un archivo BWP?

Un archivo BWP es un archivo de script de página web creado en el lenguaje de programación **BuRg3r**. Es específico de BuRg3r y puede contener código HTML con secciones del código fuente de BuRg3r delimitadas por<?b and ?> etiquetas BuRg3r es un lenguaje de programación de propósito general que también es un motor de juego desarrollado por Fluffy Smoke Entertainment.
