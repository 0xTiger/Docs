## ¿Qué es un archivo NAVEGADOR?

Las aplicaciones web ASP.NET utilizan un archivo con extensión .browser para determinar la capacidad y la configuración de los navegadores web. Contiene información sobre las capacidades, limitaciones y características del navegador. Estas características ayudan al marco ASP.NET a reconocer el navegador solicitante y presentar la página web a los usuarios en consecuencia.

## Formato de archivo del NAVEGADOR

La aplicación ASP.NET utiliza los archivos de definición de BROWSER para determinar las capacidades del navegador. Es la información contenida en este archivo la que utiliza el servidor para generar marcas y otros recursos que sean compatibles con el navegador solicitante.

Los archivos BROWSER se guardan en formato de archivo de texto sin formato. Puede abrir un archivo NAVEGADOR en editores de texto como Notepad, Notepad++, Apple TextEdit y Visual Studio IDE.

### Estructura de archivos del formato de archivo BROWSER

Los archivos de definición de BROWSER utilizan una estructura jerárquica. Cada archivo .browser contiene un marcado XML que incluye las capacidades y características de un navegador en particular o de un grupo de navegadores. Estos detalles incluyen características como cadena de agente, números de versión y compatibilidad con tecnologías como JavaScript, CSS u otros elementos HTML. Al utilizar estos archivos de definición de navegador, los desarrolladores de ASP.NET personalizan las experiencias de los usuarios en función de las capacidades de los diferentes navegadores.
