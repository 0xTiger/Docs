## ¿Qué es un archivo XHTM?

Un archivo con extensión .xhtm es un archivo de página web escrito en XHTML. Utiliza XML 1.0 como implementación estándar y se considera una reformulación de HTML 4. Al igual que XHTML, un archivo XHTM contiene etiquetas HTML estándar que siguen pautas XML más estrictas. Los archivos XHTM se utilizan para publicar contenido de sitios web y comúnmente usan la extensión [XHTML](/es/web/xhtml/).

Puede utilizar navegadores web como Google Chrome, Apple Safari y Mozilla Firefox para abrir archivos XHTM.

## Ejemplo XHTM

El siguiente ejemplo de un documento XHTM define las declaraciones XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
