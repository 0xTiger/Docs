## ¿Qué es un archivo CODASITE?

Un archivo CODASITE es un archivo de proyecto de sitio web creado con el software de creación y publicación de páginas web [Coda](https://coda.io/). Contiene una lista de todos los archivos que forman parte del proyecto. Esto incluye páginas web como archivos [HTML](/es/web/html/), archivos de hoja de estilo en cascada [CSS](/es/web/css/), archivos Javascript [JS](/es/web/js/) y cualquier otro archivo de soporte relacionado. Los archivos de proyecto CODASITE también contienen información para la publicación de sitios web en el servidor web.

## Formato de archivo CODASITE

El archivo del proyecto CODASITE se guarda en el disco como archivos binarios dentro del directorio ~/​Library/​Application Support/​Coda 2/​Sites/​. La interfaz de usuario de Coda proporciona la interfaz para administrar los sitios de Coda, ya que abrirlos manualmente puede generar configuraciones incorrectas del proyecto y crear problemas.
