## ¿Qué es un archivo XOJ?

Un archivo XOJ es un archivo de cuaderno que contiene notas tomadas con el software de toma de notas manual Xournal. Xournal es una aplicación multiplataforma gratuita que te permite crear notas con diferentes tipos de contenido, como texto, imágenes y garabatos escritos a mano. Los archivos XOJ se crean cuando se crea un nuevo archivo de cuaderno con Xournal. Estos archivos XOJ también se pueden compartir con otros usuarios para compartir datos. Una versión más reciente de Xournal es [Xournal++](https://github.com/xournalpp/xournalpp) que es una variante multiplataforma de Xournal.

## Formato de archivo XOJ

Los archivos XOJ están en formato de archivo XML y se guardan en el disco en forma comprimida utilizando la compresión [.gz](/es/compression/gz/). Debido al uso del formato XML universal, los archivos XOJ también se pueden abrir y editar con la variante multiplataforma Xournal++.

Cada archivo XOJ puede contener una página más de un cuaderno Xournal.

## XOJ - Plataformas compatibles

Xournal es multiplataforma y sus cuadernos se pueden guardar como archivo XOJ en los siguientes sistemas operativos.

* linux
* Mac OS
* Ventanas
* androide
* Compilación de iOS
