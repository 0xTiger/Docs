## ¿Qué es un archivo COMPRIMIDO?

Un archivo COMPRIMIDO es una página web comprimida de Wordpress de tamaño pequeño generada con el complemento **WordCompress**. Contiene una sola línea de código para representar la página web que se genera al comprimir el código HTML, CSS y JavaScript mediante el complemento. Estos archivos COMPRIMIDOS convertidos son de tamaño pequeño, lo que ayuda a cargar y transferir más rápido a través de Internet.

### Formato de archivo COMPRIMIDO - Más información

Los archivos COMPRIMIDOS se guardan como archivos de texto que se pueden abrir con cualquier editor de texto, como Microsoft Notepad, Notepad++ y Apple TextEdit. Además de los editores de texto, los archivos COMPRIMIDOS se pueden abrir con software de creación web como Adobe Dreamweaver, que está disponible para Windows y Mac OS.

## Acerca del complemento WordCompress

El complemento WordCompress se utilizó inicialmente para comprimir páginas web de WordPress en formato de archivo comprimido. El complemento ya no es compatible y está descontinuado. Escribiría el código fuente de la página web en un único archivo ejecutable eliminando todos los espacios en blanco del código fuente de la página. Los archivos COMPRIMIDOS se almacenan junto con el código fuente original de la página web para preservar los comentarios, sangrías y otros espacios en blanco originales incluidos en el código fuente original de la página.
