## ¿Qué es un archivo EWP?

Un archivo EWP es un archivo de proyecto creado con el software Website Builder de [EWISoft](http://www.ewisoft.com/). Está vinculado a un archivo de base de datos binario que contiene toda la información del proyecto. El archivo EWP se usa comúnmente para guardar el progreso del desarrollo web. Es un software de diseño web basado en plantillas que permite a los usuarios crear sitios web a partir de plantillas de sitios web.

## Formato de archivo EWP: más información

Los archivos EWP son archivos de proyecto que no necesitan compilarse ni alojarse en ningún servidor. Estos se guardan como archivos de texto sin formato y contienen referencias al progreso del desarrollo web. Los proyectos se pueden publicar en un sitio web una vez que se completa el desarrollo.
