## ¿Qué es un archivo FMP?

Un archivo con .fmp es un archivo de menú animado creado con la aplicación 123 Flash Menu. Contiene menús animados para sitios web que incluyen componentes como fuentes, estilo de menú, colores, dimensiones, sonidos y animaciones. La aplicación 123 Flash Menu no requiere conocimientos informáticos especiales ya que la propia aplicación incorpora toda la funcionalidad para desarrollar dichos menús. Los archivos del menú FMP se pueden exportar a los famosos archivos [.swf](/es/page-description-language/swf/) de Flash que se pueden incrustar dentro de una página web.
