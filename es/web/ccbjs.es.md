## ¿Qué es un archivo CCBJS?

Un archivo CCBJS es un archivo Javascript creado por el editor de aplicaciones, juegos y sitios web 3D, **CopperCube**. Contiene el código JavaScript que se genera como parte del [archivo .html](/es/web/html/) creado a partir del archivo .ccb. CopperCube proporciona una interfaz de secuencias de comandos fácil de usar para desarrollar interacciones y lógica de juego sin escribir ningún código. Otros formatos de archivo admitidos por CopperCube incluyen .ccscene, .ccmodel, .ccmaterial, .cctex, .ccjs, .ccp y .ccpak.

## Formato de archivo CCBJS

Los detalles del formato de archivo interno del formato de archivo CCBJS no se conocen como tales. Sin embargo, los archivos Javascript se guardan como archivos de texto sin formato en un disco y se pueden abrir con cualquier editor de texto como Microsoft Notepad, Notepad++ y Apple TextEdit.
