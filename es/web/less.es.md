## ¿Qué es un archivo LESS?

Un archivo LESS es una hoja de estilo utilizada para páginas web y creada con las hojas de estilo Leaner (LESS). Implementa adiciones al lenguaje CSS estándar, lo que lo hace fácil de aprender e implementar. Un archivo LESS amplía el [CSS](/es/web/css/) agregando funciones como Variables, Mixins, Nesting, Operations, Functions, Namespaces y Accessors.

## Formato de archivo de hojas de estilo más eficiente

Los archivos LESS se crean en el formato de CSS, ampliando sus capacidades con las características adicionales que ofrece. Estos se guardan como archivos de texto y se pueden abrir en el lenguaje de hoja de estilo LESS, CodeKit y Adobe Dreamweaver.
