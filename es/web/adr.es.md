## ¿Qué es un archivo ADR?

Un archivo ADR (guardado con la extensión .adr) es una lista de marcadores creados por el navegador web Opera. Esta lista se completa a medida que el usuario sigue marcando las URL web. Estos marcadores se almacenan en el archivo ADR que se puede respaldar y restaurar cuando sea necesario. Dado que la mayoría de los navegadores, como Chrome y Firefox, guardan los marcadores en formato [HTML](/es/web/html/), estos se pueden convertir al formato de archivo ADR para importarlos en el navegador Opera. Los archivos ADR se pueden abrir en el navegador Opera.

## Formato de archivo ADR - Más información

Se desconocen los detalles del formato de archivo interno de los archivos ADR. Sin embargo, existen diferencias entre los formatos de archivo ADR exportados con diferentes versiones de Opera. Esto dificulta [combinar archivos ADR](https://superuser.com/questions/471959/how-do-i-merge-several-opera-adr-bookmark-files-to-one-single-file) generar con diferentes versiones de Opera. Sin embargo, se pueden usar algunas herramientas externas para fusionarlas.
