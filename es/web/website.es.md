## ¿Qué es un archivo WEBSITE?

Un archivo WEBSITE es un archivo de acceso directo a un sitio web creado con Microsoft Internet Explorer (IE). Contiene una URL y un archivo de icono (ICO) que apunta al favicon del sitio web. El icono se utiliza para mostrar el acceso directo. El formato de archivo WEBSITE se introdujo con el lanzamiento de la versión 9 de IE. Las versiones anteriores de IE solían guardar accesos directos a páginas web como archivos [.url](/es/web/url/).

## Formato de archivo del SITIO WEB

Un archivo .website se almacena en el disco como archivos de texto sin formato, cuyo contenido se puede ver en cualquier editor de texto. Además de otros contenidos, contiene la URL de destino para la que es un acceso directo. Si desea abrir el sitio web de destino en un navegador que no sea IE, puede copiar esta URL y pegarla en otros navegadores como Google Chrome, Safari o Firefox para ver el sitio web.

## ¿Cómo crear un archivo .WEBSITE?

Se puede crear un archivo de SITIO WEB en IE haciendo clic en la URL en la barra de direcciones y arrastrándolo a la ubicación deseada en su computadora, como escritorio o documentos. Esto crea el acceso directo a esa ubicación de Internet y se puede abrir haciendo doble clic en el acceso directo. Si desea guardar el acceso directo como archivo `URL`, mantenga presionada la tecla MAYÚS antes de arrastrar el favicon del sitio de destino al escritorio.
