## ¿Qué es un archivo EPIBRW?

Un archivo EPIBRW es un archivo genérico alojado en un servidor web basado en Microsoft IIS. El propósito de este archivo no es específico para ninguna operación específica o ningún tipo de almacenamiento de datos, sino que es general. Se utiliza con mayor frecuencia en archivos [.pdf](/es/pdf/), pero también puede utilizarse en otros tipos de archivos.

Cuando se hace doble clic en un archivo EPIBRW, puede abrirse en el navegador mostrando el contenido. Sin embargo, si el archivo está vinculado para su descarga, el proceso de descarga del archivo se iniciará como resultado de este evento.
