## ¿Qué es un archivo URL?

Un archivo con extensión .url es un acceso directo a la URL de un sitio web que se puede guardar en la computadora con cualquier dirección de sitio web. Cuando se abre haciendo doble clic, la URL abrirá el sitio web en el navegador predeterminado y la página se mostrará al usuario. Los archivos de URL generalmente se conocen como archivos de acceso directo a sitios web y están representados por el ícono del navegador predeterminado. Los archivos URL se pueden abrir y editar en cualquier editor de texto como Microsoft Notepad, Notepad++ o Apple TextEdit.

## Formato de archivo URL: más información

Los archivos URL se guardan en formato de texto sin formato y se pueden abrir y ver en cualquier editor de texto. Estos se abren en el navegador web predeterminado donde el usuario realiza otra navegación.
