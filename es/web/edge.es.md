## ¿Qué es un archivo EDGE?

Un archivo con extensión .edge es un archivo de Adobe Edge Project que se crea cuando se crea y guarda un proyecto con Adobe Edge. Adobe Edge permite a los diseñadores web crear animadores [HTML](/es/web/html/) interactivos para la web, la publicación digital y la publicidad enriquecida para navegadores móviles y de escritorio. Esto se logra utilizando HTML/HTML5, [JavaScript](/es/web/js/) y [CSS3](/es/web/css/). Los archivos Edge se generan como archivos de texto sin formato y se pueden abrir con varias aplicaciones diferentes, como Microsoft Notepad, Apple TextEdit y Notepad++.

## Formato de archivo EDGE

Los archivos EDGE se guardan en formato de texto sin formato y utilizan jQuery para habilitar la animación. Una alternativa a la animación EDGE eran los archivos Flash que ya no son compatibles. El archivo EDGE se guarda junto con los archivos HTML, JavaScript y CSS. Al abrir el archivo EDGE, se abre la página web, es decir, el archivo HTML.
