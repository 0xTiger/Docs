## ¿Qué es un archivo PRO? ##

Un archivo PRO es una maqueta de un sitio web o una aplicación móvil creada por Adobe Proto, una aplicación Adobe Touch desaparecida. Incluye uno o más diseños de interfaz de usuario para una página web o aplicación móvil, cada uno de los cuales puede incluir elementos interactivos. Las tecnologías CSS, WebKit y jQuery son compatibles con los archivos PRO.

### Breve historia del formato de archivo PRO

Los archivos PRO fueron creados por Adobe Proto que fue reemplazado por Adobe XD. Se utilizó para crear aplicaciones de creación de prototipos para tabletas. Las aplicaciones comunes de diseño web como Adobe Dreamweaver utilizaron estos archivos PRO para crear aplicaciones web y móviles.

## Formato de archivo PRO - Más información

Un archivo PRO es el primer paso en el desarrollo de un sitio web o aplicación móvil. Los desarrolladores primero crean maquetas digitales que definen el aspecto de la aplicación. Estos también definen el flujo de trabajo general de la aplicación y la navegación a través de las pantallas de la aplicación. Estas maquetas también incluyen una serie de elementos de navegación, como botones, cuadros combinados, barras de desplazamiento, campos de entrada y muchos otros.
