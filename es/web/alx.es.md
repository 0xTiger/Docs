## ¿Qué es un archivo ALX?

Un archivo ALX es un archivo ActiveX que está incrustado en páginas web en forma de componentes [HTML](/es/web/html/). Los controles ActiveX se conocían formalmente como controles OLE u OCX. Los archivos ALX se pueden usar para insertar una variedad de funcionalidades en una página web, como agregar un control de temporizador para actualizar el contenido de la página periódicamente, o agregar un control de ventana emergente para mostrar ayuda de estilo de información sobre herramientas basada en la acción del usuario. Archivos ALX cuidados vinculados mediante<Param> etiqueta dentro del código HTML de una página web. Puede abrir archivos ALX en Microsoft Internet Explore, Microsoft ActiveX Control Pad y editores de texto como Notepad y Notepad++.

## Formato de archivo ALX - Más información

Los archivos ALX están en formato de archivo de texto sin formato y se pueden editar abriéndolos en un editor de texto. Los controles ActiveX se pueden incluir en una página web colocándolos en la página web. Sin embargo, para crear páginas web dinámicas, los controles se integran con la ayuda de [JScript](/es/web/js/). Los controles en una página web están asociados con los eventos y valores de los controles, lo que brinda un control completo para las interacciones del usuario.
