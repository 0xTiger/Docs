## ¿Qué es un archivo XD?

Un archivo con extensión .xd es una maqueta creada para sitios web o aplicaciones móviles utilizando el software Adobe XD. Los diseñadores, los equipos creativos y las organizaciones utilizan estos archivos para diseñar aplicaciones haciendo referencia a estos. Un único archivo XD contiene elementos de diseño, como imágenes, formas y texto, que proporcionan un conjunto completo de funciones para desarrollar prototipos de estructura alámbrica rápidos e intuitivos para que los desarrolladores comiencen a desarrollarlos como aplicaciones.

## Formato de archivo XD

Los archivos XD contienen elementos gráficos (imagen, formas y texto) en capas como una o más mesas de trabajo. Estos, junto con las guías y las capas, proporcionan los componentes básicos para las estructuras alámbricas.

* "Mesas de trabajo": representan las pantallas de una aplicación o las páginas de un sitio web y agilizan los procesos de diseño. Una mesa de trabajo crea diseños interactivos para múltiples tamaños de pantalla en un solo documento.
* `Guías`: se utilizan para alinear objetos o administrar el espacio entre elementos en una mesa de trabajo. Las guías utilizan cuadrículas que sirven como marco para organizar los elementos de diseño (imágenes, glifos, párrafos) y garantizar la simetría del diseño.
* `Capas`: los diseñadores de UX usan capas en XD de la misma manera que las usan en Adobe Illustrator o Photoshop. Estos se utilizan para organizar el trabajo en mesas de trabajo para evitar el desorden y se controlan desde el panel Capas.
