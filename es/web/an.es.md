## ¿Qué es un archivo AN?

Un archivo con la extensión .an es un archivo de proyecto de [Adobe Edge Animate](https://www.adobe.com/products/animate.html) que contiene información sobre la animación. Se crea un archivo .html cuando se guarda el proyecto que contiene las referencias a los archivos asociados [.css](/es/web/css/) y [.js](/es/web/js/) generados por Adobe Edge Animate. El archivo AN se guarda junto con estos archivos que se pueden usar para abrir el archivo del proyecto. Adobe edge animate se puede utilizar como una alternativa a la creación de páginas web animadas con Adobe Flash, que ya no es compatible con los navegadores.

## Formato de archivo AN

Los archivos AN se almacenan en el disco en formato de archivo de texto sin formato y se pueden abrir en editores de texto para ver su contenido.
