## ¿Qué es un archivo WHTT?

Un archivo WHTT es una versión sin conexión de un sitio web que se descarga mediante el software [HTTrack Website Copier](https://www.httrack.com/). Contiene todo el contenido de un sitio web, incluyendo [HTML](/es/web/html/) páginas web, imágenes, hojas de estilo, enlaces y directorios. Todas las páginas de un sitio web se descargan recursivamente y se guardan en un disco como un espejo fuera de línea. Cuando el software HTTrack descarga un sitio web, se puede ver sin conexión sin necesidad de estar conectado a Internet. Para ver el sitio web descargado, simplemente abra la página del sitio web reflejado en el navegador y cada imagen/enlace se cargará correctamente.

## Formato de archivo WHTT

Los archivos WHTT se guardan como una combinación de páginas web, imágenes, hojas de estilo y enlaces de referencia THML.
