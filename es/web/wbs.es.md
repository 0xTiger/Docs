## ¿Qué es un archivo WBS?

Un archivo WBS es un archivo de proyecto creado por WebBlender que es un programa de creación de presentaciones y sitios web para niños. El archivo WBS se crea cuando se inicia un nuevo proyecto en WebBlender. Todas las funciones del sitio agregadas al proyecto, como páginas, imágenes, texto, animaciones, etc., se mencionan en el archivo del proyecto WBS. Una vez que el proyecto está listo, publica el contenido del sitio como archivo [HTML](/es/web/html/). WebBlender es creado y mantenido por [Tech4Learning](https://www.tech4learning.com/).
