## ¿Qué es un archivo FWP?

Un archivo con extensión .fwp es un archivo comprimido creado con Microsoft Expression Web y contiene todos los archivos y carpetas de un diseño. Microsoft Expression Web es un producto de software de diseño web general de Microsoft y se basa en el editor [HTML](/es/web/html/). Ha sido descontinuado desde 2012 y Microsoft lo puede descargar gratis. Los archivos FWP se pueden exportar y guardar como formato .cab de Microsoft.

## Formato de archivo FWP - Más información

Las especificaciones de formato de archivo exactas de los archivos FWP no están disponibles para referencia del desarrollador. Sin embargo, estos son archivos binarios y se pueden abrir en Microsoft Expression Studio, que se suspendió hace mucho tiempo.
