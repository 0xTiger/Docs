## ¿Qué es un archivo HXS?

Un archivo HXS es un archivo de ayuda compilado y utilizado para el archivo Microsoft Help 2.x. Se compila y genera a partir de un conjunto de páginas de temas y se escribe en un subconjunto de [HTML](/es/web/html/) que es similar al formato de archivo de ayuda anterior [CHM](/es/web/chm/). Los archivos HXS se almacenan en formato comprimido y se crean a partir de un archivo de proyecto principal .HxC, un archivo de inclusión .HxF, una tabla de contenido .HxT, un archivo de definición de atributos .HxA y varios índices .HxK. Los metadatos de un archivo HXS se crean a partir de [XML](/es/web/xml/).

Los archivos HXS se pueden abrir con un navegador web.

## Formato de archivo HXS

Los archivos HXS se crean y guardan como archivos comprimidos a partir de contenido HTML y metadatos XML.
