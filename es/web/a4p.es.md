## ¿Qué es un archivo A4P?

Un archivo A4P es un archivo de proyecto creado por el programa de software A4Desk Pro que se utiliza para crear sitios web Flash. Se crea con las plantillas incluidas con el A4Desk Pro y no necesita una para tener un conocimiento profundo de Flash. Los archivos A4P creados con plantillas pueden incluir contenido de sitios web en forma de texto y animaciones Flash. En caso de que no conozca Flash, se utilizó para crear animaciones que podrían exportarse a [SWF](/es/page-description-language/swf/) así como a formatos de archivo [GIF](/es/image/gif/) para su inclusión en sitios web.

## Formato de archivo A4P

Los archivos A4P se crean a partir de archivos de plantilla y se guardan como archivos binarios en el disco. La estructura de formato de archivo interno de los archivos A4P no se conoce.

Los archivos A4P se pueden abrir con el software **Avanquest A4Desk Pro**.
