## ¿Qué es un archivo DOCHTML?

Un archivo DOCHTML es el formato HTML de un documento generado con Microsoft Word. Es la versión [HTML](/es/web/html/) del documento creado que se puede usar para publicar en el servidor web. Los archivos DOCHTML se pueden abrir en un navegador web para ver el contenido del documento de Word convertido. La perspectiva y la apariencia del archivo DOCHTML son las mismas que las del documento de Word original.

## Formato de archivo DOCHTML

Los archivos DOCHTML se guardan en un disco en formato HTML que está listo para publicarse en servidores web. Microsoft Word dejó de admitir la posibilidad de guardar documentos como archivos DOCHTML. Esto fue reemplazado por formatos de archivo [.mht](/es/web/mht/) y [.mhtml](/es/web/mhtml/) que se usan comúnmente ahora.
