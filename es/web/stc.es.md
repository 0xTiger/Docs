## ¿Qué es un archivo STC?

Un archivo con extensión .stc es un archivo de conexión utilizado por el software Adobe Contribute para almacenar la información de conexión. Contiene información de conexión, como nombre de usuario, URL de host, protocolos como FTP, SFTP, etc. para su reutilización desde el sitio web existente. Para aquellos que no lo saben, Adobe Contribute solía ser un editor [HTML](/es/web/html/) especializado que se usaba para enviar actualizaciones a sitios web existentes, incluidos artículos basados en contenido conocidos como blogs. Incluía complementos para navegadores web populares, como Internet Explorer y Firefox, que aportaban contenido a los sitios web desde las páginas. Ha sido descontinuado desde enero de 2017 y Adobe Dreamweaver ahora se usa para crear sitios web desde cero.

## Formato de archivo STC

Se desconoce el formato de datos internos de los archivos STC. Sin embargo, la clave de conexión está encriptada con una contraseña y cualquier información de conexión de inicio de sesión de red o FTP enviada en el archivo permanece segura y solo se puede acceder a ella a través de Contribute. El archivo de conexión clave puede enviarse por correo electrónico a los usuarios o guardarse en la computadora. Cuando se hace clic en él, se abre automáticamente Contribute, que importa la conexión.
