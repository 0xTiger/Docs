## ¿Qué es un archivo HTC?

Un archivo HTC es un archivo de formato de archivo web utilizado por Microsoft Internet Explorer. Contiene definiciones para componentes HTML que realizan pequeñas funciones para realizar diferentes tareas dentro del navegador web. La funcionalidad definida dentro de los archivos de HTC es reutilizable y se puede acceder a ella a través de varias páginas [HTML](/es/web/html/).

## Formato de archivo HTC - Más información

Los archivos de HTC se guardan en formato de archivo de texto sin formato. Esto significa que puede abrirlos en cualquier editor de texto y realizar cambios en la funcionalidad. Por lo general, estos se abren en Internet Explorer, pero de esa manera solo podrá ver estos archivos y no podrá editarlos.

Un archivo HTC es en realidad un archivo HTML que es una combinación de script y un conjunto de elementos específicos de HTC para definir los componentes.
