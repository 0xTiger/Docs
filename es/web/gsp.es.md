## ¿Qué es un archivo GSP?

Un archivo GSP es un archivo de mensaje de error que indica un problema con la entrega del correo electrónico. Se envía mediante IMail Server, un programa de servidor de correo electrónico que se utiliza para la comunicación por correo electrónico de numerosas pequeñas y medianas empresas. El error puede deberse a varios motivos, como una gran cantidad de datos, la pérdida de datos en el archivo [MSG](/es/email/msg/) o errores de falta de memoria.

## Formato de archivo GSP - Más información

Los archivos GSP se almacenan en formato de archivo web y se pueden ver fácilmente con clientes de correo electrónico como Microsoft Outlook. Se genera un archivo GSP como respuesta a un correo electrónico enviado que genera un error.
