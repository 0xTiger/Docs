## ¿Qué es un archivo AWM?

Un archivo AWM es un archivo de proyecto creado por el programa creador de menús web, **AllWebMenus**. Contiene datos relacionados con el proyecto para crear menús desplegables basados en [HTML](/es/web/html/). Un archivo AWM se compila en un paquete web finalizado que se integra en los sitios web para mostrar los menús. Contiene todos los datos relacionados con el proyecto, como configuraciones de apariencia, entradas de menú, estilo (vertical u horizontal), colores y otra información relacionada.

Los archivos AWM se pueden abrir con **Likno AllWebMenus**.

## Formato de archivo AWM: más información

Los archivos AWM se guardan como archivos binarios compilados y solo se pueden abrir con el software Likno AllWebMenus. Los detalles del formato de archivo interno de los archivos AWM no están disponibles públicamente.

AllWebMenus le permite diseñar menús [CSS](/es/web/css/) o menús basados en [JavaScript](/es/web/js/) visualmente sin utilizar ningún lenguaje de programación. Los menús pueden personalizarse por completo y diseñarse utilizando plantillas de apariencia predefinidas disponibles con el software.
