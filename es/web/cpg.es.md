## ¿Qué es un archivo CPG?

Un archivo CPG es un archivo de proyecto creado por el software de edición de páginas web, Cool Page. El software quedó obsoleto hace muchos años y ya no se utiliza. El archivo CPG contenía detalles sobre los objetos utilizados en la página web y las propiedades de estos objetos. Los archivos del proyecto CPG se pueden abrir en el software Cool Page y, una vez finalizados, la página web se puede exportar como página web [HTML](/es/web/html/).

## Formato de archivo CPG

Se desconocen los detalles del formato de archivo interno del formato de archivo CPG. Estos se guardaron como archivos binarios y sólo se abrirán en el software Cool Page.
