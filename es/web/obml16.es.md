## ¿Qué es un archivo OBML16?

Un archivo OBML16 es una página web guardada con el navegador web móvil Opera Mini. Se basa en Opera Binary Markup Language ([OBML](/es/web/obml/)) y es un formato compacto para brindar acceso rápido a páginas web guardadas. Reemplazó el formato de archivo [OBML15](/es/web/obml15/) cuando se lanzó Opera Mini versión 7. Todas estas variantes de las páginas web de Opera Mini se actualizaron desde el formato de archivo OBML base y cada actualización de Opera Mini es compatible con un solo formato OBML. Los archivos OBML16 se pueden convertir a [PDF](/es/pdf/) y [HTML](/es/web/html/) en línea.

## Formato de archivo OBML16 - Más información

OBML16 se basa en el formato de archivo OBML, que es un formato compacto para guardar páginas web. La ventaja de utilizar este formato compacto es que reduce el tamaño del contenido web, lo que se traduce en un aumento de la velocidad de descarga. Una vez descargada, los usuarios de Opera Mini pueden visualizar estas páginas sin necesidad de tener acceso a una conexión a Internet.
