## ¿Qué es un archivo CHAT?

Un archivo CHAT es un archivo de configuración utilizado por el IRC (Internet Relay Chat) para conectarse a un canal IRC. Contiene parámetros de conexión como servidor, puerto, nombre del canal IRC y, opcionalmente, una contraseña para abrir un canal IRC. El cliente utiliza la configuración para conectarse a otros canales. Los chats y discusiones en IRC son uno a uno y son privados.

Las aplicaciones que pueden **abrir** archivos CHAT incluyen mIRC y Visual IRC.

## Formato de archivo de chat

Los archivos CHAT se almacenan como archivos binarios en el disco. Cuando un usuario quiere conectarse a un canal en IRC, ingresa la información en el cliente de iRC, ingresa la información en el software del cliente. La información se almacena en el archivo CAHT para recuperación lateral.
