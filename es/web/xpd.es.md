## ¿Qué es un archivo XPD?

Un archivo XPD es un archivo de página web creado a partir de un documento XML. Más a menudo, esto se genera mediante una transformación XSLT. Las [especificaciones de formato de archivo XPD](https://www.w3.org/TR/xml-pipeline/) están disponibles en W3 y definen una forma de aumentar la interoperabilidad de multiproceso [XML](/es/web/xml/) aplicaciones. El lenguaje XML Pipeline define las relaciones de procesamiento entre recursos XML.
