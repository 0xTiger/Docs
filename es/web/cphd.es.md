## ¿Qué es un archivo CPHD?

Un archivo CPHD es un documento de hipertexto utilizado por el motor de preprocesador de CloudChan. Contiene HTML y scripts para crear documentos y páginas web dinámicas. Los scripts dentro de un archivo CPHD se parecen a la sintaxis utilizada por Java. La página web CPHD devuelve un documento HTML cuando se ejecuta en procesos de CloudChan. El formato CPHD puede funcionar con cualquier documento XML.

## Formato de archivo CPHD

Los archivos CPHD se almacenan en el disco como archivos de texto sin formato. Esto hace posible abrir estos archivos en cualquier editor de texto como Notepad o Notepad++.
