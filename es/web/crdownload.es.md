## ¿Qué es un archivo CRDOWNLOAD?

Un archivo CRDOWNLOAD es un archivo parcialmente descargado que se crea cuando descarga un archivo de Internet utilizando los navegadores Microsoft Edge, Google Chrome o Chromium. El contenido del archivo que se descarga se almacena en este archivo hasta que se completa la descarga. Una vez que se completa la descarga, se elimina la extensión .crdownload y se reemplaza con la extensión de archivo original del archivo que se está descargando.

## Formato de archivo CRDOWNLOAD

Un archivo CRDOWNLOAD puede estar en formato de texto o binario. Los navegadores crean un archivo .crdownload de manera predeterminada una vez que comienza la descarga. Si la descarga se interrumpe en el medio, el archivo CRDOWNLOAD permanece guardado en el disco. Esto a veces permite reanudar la descarga desde donde se interrumpió.

### CRDOWNLOAD Ejemplo

Soporte, está descargando un archivo fileformatslist.doc de Internet. Cuando comienza la descarga, su navegador guarda el archivo en la carpeta Descargas como archivo fileformatslist.crdownload. Su tamaño de archivo aumenta a medida que se descarga el archivo. Una vez que se complete la descarga, se cambiará el nombre del archivo a fileformatslist.doc.
