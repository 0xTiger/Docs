## ¿Qué es un archivo GED?

Un archivo **GEDCOM (Comunicación de datos genealógicos)**, generalmente con extensión .ged, es un formato de archivo estándar utilizado para intercambiar datos genealógicos entre diferentes programas de software de genealogía; Los archivos GEDCOM son archivos de texto sin formato que contienen información sobre personas, familias, eventos y otros datos genealógicos; El propósito de GEDCOM es proporcionar un formato común para el intercambio de información genealógica.

Los archivos GEDCOM suelen tener la extensión de archivo .ged; esta extensión se utiliza para indicar que el archivo sigue el **estándar GEDCOM** para el intercambio de datos genealógicos; Por eso, cuando guarda o exporta sus datos genealógicos desde un programa de software de genealogía, el archivo resultante suele recibir un nombre como sunombredearchivo.ged.

Por ejemplo:

- yourfamilytree.ged
- john_doe_family.ged
- ancestry_data.ged

## Puntos clave sobre los archivos GEDCOM

A continuación se detallan algunos puntos clave sobre los archivos GEDCOM:

1.  **Formato de texto sin formato:** Los archivos GEDCOM son archivos de texto sin formato; lo que significa que pueden abrirse y visualizarse con cualquier editor de texto; este formato facilita el intercambio y la compatibilidad entre diferentes aplicaciones de software de genealogía.
    
2.  **Estructura:** Los archivos GEDCOM tienen una estructura jerárquica; los datos se organizan en registros; cada registro tiene una etiqueta específica que indica el tipo de información que contiene; por ejemplo, el nombre de un individuo puede estar representado por la etiqueta NOMBRE.
    
3.  **Individuals and Families:** GEDCOM files typically include information about individuals such as names, dates of birth and places of birth; they also include details about family relationships, marriages and other events.
    
4.  **Interoperabilidad de software:** Dado que GEDCOM es un formato estandarizado; permite a los usuarios mover sus datos genealógicos entre diferentes programas de software de genealogía; Esto es particularmente útil para usuarios que pueden cambiar a diferentes aplicaciones de software o compartir su investigación con otros.
    
5.  **Versión:** GEDCOM ha pasado por varias versiones y cada versión introduce mejoras y nuevas características; la versión más utilizada fue GEDCOM 5.5. 

## ¿Cómo abrir el archivo GED?

Para abrir un archivo GED, generalmente necesita un software de genealogía que admita el formato GEDCOM. Algunas opciones populares de software de genealogía incluyen:

- **Creador de árboles genealógicos**
- **Ancestry.com**
- **Mi herencia**
- **Abuelo** (gratis y de código abierto)
- **Árbol genealógico heredado**

Una vez que haya elegido su software de genealogía, busque una opción como Importar, Abrir GEDCOM o Importar desde archivo en el menú o barra de herramientas.

Dado que los archivos GED se guardan en formato de texto, también puede abrir los archivos con un editor de texto sin formato, por ejemplo

- **Bloc de notas**
- **Bloc de notas++**
- **Editar texto de Apple**
- Cualquier editor de texto
