## ¿Qué es un archivo DXB?

DXB significa Drawing Exchange Binary, que es un formato de archivo utilizado en software de diseño asistido por computadora (CAD). Los archivos DXB contienen datos binarios que representan gráficos vectoriales 2D y se utilizan para intercambiar dibujos entre diferentes programas CAD. Por lo general, se crean exportando dibujos desde un programa CAD en formato DXB y luego importándolos a otro programa CAD.

El formato DXB se usa con menos frecuencia hoy en día que otros formatos de archivos CAD como DXF o DWG. Sin embargo, es posible que algunos programas CAD más antiguos aún utilicen DXB como formato de archivo para exportar o importar dibujos.

Si tiene un archivo DXB y necesita verlo o editarlo, necesitará un programa CAD que admita el formato DXB. Algunos ejemplos de programas CAD que admiten DXB incluyen AutoCAD, DraftSight y LibreCAD.

## Formato de archivo DXB - Más información

DXB (Drawing Exchange Binary) es una versión binaria del formato de archivo DXF (Drawing Exchange Format), que es un formato estándar para intercambiar dibujos CAD entre diferentes aplicaciones de software.

Los archivos DXB contienen datos binarios que representan gráficos vectoriales 2D al igual que los archivos DXF. Sin embargo, los archivos DXB son más compactos y más rápidos de cargar que los archivos DXF porque se almacenan en formato binario en lugar de texto sin formato.

Para crear un archivo DXB, un programa CAD puede exportar un dibujo en formato DXB, lo que genera un archivo binario que puede ser leído por otros programas CAD que admitan DXB. De manera similar, para importar un archivo DXB, un programa CAD puede leer los datos binarios y convertirlos a su propio formato interno para editarlos.

## ¿Cuál es el formato del archivo DXB?

El formato de archivo DXB (Drawing Exchange Binary) es un formato de archivo binario, lo que significa que consta de datos binarios que pueden interpretarse directamente por computadora.

El formato binario utilizado por DXB es más compacto y más rápido de cargar que el formato de archivo DXF (Drawing Exchange Format) basado en texto que también se utiliza para intercambiar dibujos CAD entre diferentes aplicaciones de software.

La estructura del archivo DXB consta de una serie de registros binarios, cada uno de los cuales contiene encabezados y datos que representan un solo objeto o entidad en el dibujo. El encabezado contiene información sobre el tipo de objeto o entidad representada por los datos, así como su tamaño y otras propiedades.

Los datos en el archivo DXB están organizados jerárquicamente, con registros de nivel superior que contienen referencias a registros de nivel inferior que representan subobjetos o componentes del dibujo. Por ejemplo, un registro que representa una polilínea puede contener referencias a registros que representan segmentos de línea individuales que conforman la polilínea.

## ¿Qué contiene el archivo DXB?

El archivo DXB (Drawing Exchange Binary) contiene datos binarios que representan gráficos vectoriales 2D como líneas, arcos, círculos, texto y otras formas geométricas que componen el dibujo CAD.

Cuando un programa CAD exporta un dibujo al formato DXB, crea un archivo binario que contiene toda la información necesaria para recrear el dibujo en otro programa CAD que admita DXB. Esto incluye información sobre la ubicación, tamaño, color y estilo de cada objeto en el dibujo, así como cualquier texto o anotaciones que puedan incluirse.

Debido a que los archivos DXB se almacenan en formato binario, normalmente son más pequeños y más rápidos de cargar que los archivos DXF (Drawing Exchange Format), que se almacenan en texto sin formato. Sin embargo, los archivos DXB sólo pueden leerse mediante programas CAD que admitan el formato DXB, mientras que los archivos DXF pueden leerse mediante una gama más amplia de programas.
