## ¿Qué es un archivo PSM?

Un archivo con extensión .psm es un archivo CAD que se crea y abre con el software de modelado CAD Solid Edge para diseñar piezas y ensamblajes en 3D. Contiene modelo 3D del diseño para la fabricación de piezas a partir de la chapa. El paquete de software Solid Edge proporciona todas las opciones necesarias para crear un modelo 3D que también se puede integrar con otros diseños mecánicos. Su 2D Nesting admite los formatos de archivo PSM y PAR, además de otros formatos de datos neutrales, incluidos DXF y DWG, mediante la extracción de datos de patrones planos.

### Formato de archivo PSM

PSM es un formato de archivo cerrado y sus detalles no están disponibles a partir de ahora. Estos tienen una aplicación de tipo MIME/octet-stream. Los archivos de modelos de metal se pueden importar en Solid Edge y convertir en piezas utilizables para que sean editables. El modelado síncrono proporciona una mayor facilidad de uso para trabajar con archivos PSM.
