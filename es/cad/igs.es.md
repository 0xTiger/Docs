## ¿Qué es un archivo IGS?

Un archivo con la extensión .igs (Initial Graphics Exchange) es un formato de archivo de intercambio de diseño 2D-3D que es independiente de las especificaciones de formato de archivo de origen o de destino utilizadas por las aplicaciones CAD. Se utiliza para intercambiar información de diseño sobre diagramas de circuitos, estructura alámbrica, superficie de forma libre entre dos sistemas independientes. Es similar a [IGES](/es/cad/iges/) y es el requisito principal de los fabricantes para fijar precios y diseñar moldes para su producto. IGS ha sido reemplazado recientemente por el formato de archivo STEP (.STP) más nuevo. Los archivos IGS se pueden abrir con aplicaciones como Autodesk, FreeCAD, CADEX CAD Exchanger y otras aplicaciones similares.

## Formato de archivo IGS

Los archivos IGS están en formato de archivo de texto ASCII y se pueden abrir con cualquier editor de texto. Cada registro en un archivo IGS tiene una longitud de 80 caracteres. Las aplicaciones CAD guardan y exportan datos vectoriales en formato de archivo IGS para transferir archivos de modelos 3D entre diferentes paquetes de software. Aunque los archivos IGS se pueden abrir en cualquier editor de texto, el modelo o diseño solo se puede ver mediante aplicaciones de visualización que brinden soporte para el formato de archivo IGS.
