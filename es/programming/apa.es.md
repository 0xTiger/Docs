## ¿Qué es un archivo APA?

Un archivo con extensión .apa es un archivo de proyecto de desarrollo de software creado por la aplicación RSView. Contiene información necesaria para crear aplicaciones de interfaz hombre-máquina (HMI). Los archivos APA se almacenan en formato de archivo de archivo. Su formato de archivo de archivo ayuda a transferirlos de una computadora a otra fácilmente debido al tamaño de archivo más pequeño. Las utilidades de descompresión estándar se pueden utilizar para extraer el contenido de los archivos APA. Los archivos APA se pueden abrir con el software RSView Studio o RSView ME de Rockwell Automation.

## Formato de archivo APA - Más información

El administrador de la aplicación RSView guarda los archivos .apa de manera predeterminada en la ubicación C:\Documents and Settings\All Users\Documents\RSView Enterprise\ME\Archives en Windows XP. Sin embargo, puede guardar el archivo en cualquier directorio según los requisitos.
