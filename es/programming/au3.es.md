## ¿Qué es un archivo AU3?

Un archivo AU3 es un archivo de secuencia de comandos automatizado creado con el programa gratuito de secuencias de comandos, [AutoIt v3] (https://www.autoitscript.com/site/). Contiene el lenguaje de secuencias de comandos AutoIt3 para automatizar comandos dentro de Windows. El lenguaje de programación AutoIt3 le permite simular una serie de eventos que se pueden utilizar para automatizar tareas. Por ejemplo, la secuencia de comandos dentro del archivo AU3 se puede usar para simular pulsaciones de teclas, movimientos del mouse, controlar la posición de las ventanas y otros propósitos de secuencias de comandos.

Las aplicaciones que pueden abrir archivos AU3 incluyen [SciTE4AutoIt3](https://www.autoitscript.com/site/autoit-script-editor/downloads/) y AutoIt.

## Formato de archivo AU3

Los archivos AU3 se guardan como archivos de texto y contienen lenguaje de secuencias de comandos AutoIt3.
