## ¿Qué es un archivo ACD?

Un archivo ACD es un archivo de proyecto que contiene información para programar plataformas Logix de [Rockwell Automation](https://www.rockwellautomation.com). Incluye instrucciones para controladores y procesadores. Se utiliza para plataformas Logix como los sistemas ControlLogix, CompactLogix, FlexLogix, SoftLogix5800 y DriveLogix.

Las aplicaciones que pueden abrir archivos ACD incluyen Rockwell Automation RSLogix 5000.
