## ¿Qué es un archivo RBXM?

Un archivo RBXM es un formato de archivo de modelo 3D utilizado en Roblox Studio IDE para almacenar objetos 3D del juego Roblox multijugador en línea. Contiene objetos 3D creados para el juego usando el IDE. Los ejemplos de objetos 3D en un archivo RBXM incluyen camisas, pantalones, sombreros, edificios, vehículos, animales, aviones, automóviles o plantas. Los archivos RBMX se pueden convertir al [formato de archivo OBJ 3D](/es/3d/obj/) utilizando el IDE de Roblox. Roblox IDE se puede usar para abrir archivos RBMX.

Otro formato de archivo popular de Roblox es [RBXL](/es/programming/rbxl/), que es un formato de archivo de ubicación y contiene la ubicación definida en palabras 3D dentro del juego de Roblox.

## Formato de archivo RBMX: más información

Los archivos RBXM se crean y guardan en formato de archivo binario y sus especificaciones de formato de archivo interno no están disponibles públicamente. Esto es diferente del formato de archivo RBXMX que se basa en el formato de archivo [XML](/es/web/xml/). Hay muchas herramientas/repositorios disponibles en Github que se pueden usar para [procesar archivos Roblox RBXM](https://github.com/search?q=rbxm).
