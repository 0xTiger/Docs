## ¿Qué es un archivo ML?

Un archivo ML es un archivo de texto creado con el lenguaje de programación funcional de propósito general ML (a menudo conocido como Metal Language). Contiene código fuente de ML que se usa con mayor frecuencia en la investigación y el diseño de lenguajes de programación utilizando una semántica completamente definida. Hay múltiples variaciones de ML con Caml como la más utilizada. Otros incluyen Objective Caml (OCaml), JoCaml, Extended ML y MacroML. El lenguaje ML se usa comúnmente para la escritura de compiladores, la demostración automatizada de teoremas y las verificaciones formales.
