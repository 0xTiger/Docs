## ¿Qué es un archivo GMD?

Un archivo GMD es un archivo fuente de desarrollo de juegos generado con la herramienta de desarrollo de software Game Maker que se utiliza para crear juegos 2D. El archivo GMD se almacena como un archivo de texto y es un archivo de proyecto para el juego que se desarrollará. Game Maker está disponible para Windows, Mac, Linux, Android, iOS, HTML5, Xbox, PlayStation y Nintendo Switch. La herramienta es ampliamente utilizada por desarrolladores independientes, estudios profesionales y educadores de todo el mundo para crear juegos.

## Formato de archivo GMD

Los archivos GMD se guardan en formato de archivo binario y los detalles del formato de archivo interno no están disponibles para referencia del desarrollador.
