## ¿Qué es un archivo CXX?

Un archivo CXX es un archivo de código fuente, similar a C++, para escribir aplicaciones. Puede contener la implementación completa de una aplicación completa o puede tener una implementación parcial del programa que se puede incluir en un proyecto con otros archivos C++. Los archivos CXX se pueden abrir en cualquier editor de texto e IDE de programación populares como Notepad ++, Turbo C ++ y Visual Studio,

## Formato de archivo CXX

Los archivos CXX se almacenan como archivos de texto sin formato y contienen código escrito en sintaxis C++. Los archivos CXX también se encuentran comúnmente con la extensión [.cpp](/es/programming/cpp/). Los archivos CXX se consideran una alternativa a .cpp utilizada por algunos compiladores.
