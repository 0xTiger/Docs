## ¿Qué es un archivo MM?

Un archivo MM es un archivo de código fuente que puede contener tanto código de programación Objective-C como Objective-C++. Se usa en MacOS y se diferencia de los archivos .M que solo contienen código Objective-C. Cuando el compilador ve la extensión .mm, puede compilar las clases de C++ presentes en el archivo. Dado que un solo archivo de código fuente puede contener una combinación de lenguajes [C](/es/programming/c/) y [C++](/es/programming/cpp/), cambiar el nombre de los archivos .m a la extensión .mm permite que las características de C++ se configuren. en. Las aplicaciones que pueden abrir archivos .MM incluyen Apple TextEdit, Microsoft Notepad, Notepad++ y Microsoft Wordpad.
