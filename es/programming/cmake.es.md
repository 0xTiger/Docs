## ¿Qué es un archivo CMAKE?

Un archivo CMAKE es un archivo de proyecto creado por la herramienta multiplataforma de código abierto **CMake**. Contiene script fuente en lenguaje CMake para todo el programa. La herramienta CMake utiliza secuencias de comandos de programación, conocidas como CMakeLists, para generar archivos de compilación para un entorno específico, como una máquina Unix de archivos MAKE. Los scripts y módulos utilizados por el proyecto CMake están organizados como script.cmake y module.cmake a los que se hace referencia en el proyecto.
