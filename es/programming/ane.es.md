## ¿Qué es un archivo ANE?

Un archivo ANE es un archivo de extensión nativa de Adobe Air que utilizan los desarrolladores para escribir funciones específicas que invocan el código [Adobe ActionScript](https://en.wikipedia.org/wiki/ActionScript). En la mayoría de los casos, los archivos ANE contienen la biblioteca de código nativo utilizada por la extensión. Sin embargo, el archivo en sí está instalado en el dispositivo. Permite funciones específicas del sistema, como la vibración del dispositivo. Las aplicaciones de Adobe AIR deben compilarse para usar enlaces dinámicos para utilizar el archivo ANE.

## Formato de archivo ANE - Más información

Los archivos ANE se guardan en el disco como archivos binarios cuyas especificaciones de formato de archivo no están disponibles públicamente. Un archivo ANE puede contener código fuente nativo de Android Java para acceder a la funcionalidad del dispositivo Android, como invocar funciones de volumen o vibración. Todo esto se hace envolviendo el sistema nativo en el lenguaje ActionScript de Adobe.
