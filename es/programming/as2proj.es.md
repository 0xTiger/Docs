## ¿Qué es un archivo AS2PROJ?

Un archivo AS2PROJ es un archivo de proyecto creado por el IDE de programación Flash de código abierto, FlashDevelop. Contiene información relacionada con el proyecto, como configuraciones de compilación, referencias a archivos de origen del proyecto y opciones para generar el archivo de salida [.swf](/es/page-description-language/swf/). Cada nuevo proyecto creado en FlashDevelop se guarda como un archivo AS2PROJ. Puede cambiar la configuración de su proyecto y todos estos cambios se guardan en el archivo AS2PROJ.

Los archivos AS2PROJ se pueden abrir con FlashDevelop.

## Formato de archivo AS2PROJ

Un archivo AS2PROJ se guarda en formato de archivo [XML](/es/web/xml/) que es un formato de archivo universal para el intercambio de información. Se puede abrir en cualquier editor de texto como Notepad, Notepad++ y Atom para examinar el contenido.
