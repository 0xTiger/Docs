## ¿Qué es un archivo LGO?

Un archivo LGO, un código de programa escrito con la aplicación Logo para dibujar imágenes. Contiene instrucciones programadas que guían a la aplicación de lectura para realizar dibujos de acuerdo con estas instrucciones. Por ejemplo, el programa "tortuga" que puede abrir archivos LGO, use estas instrucciones para dibujar líneas según el código programado. El dibujo no se limita solo a las líneas. Los archivos LGO se pueden usar para dibujar diseños simples o complejos. Muchos programas educativos utilizaron archivos LGO para enseñar cómo hacer dibujos mediante programación.

## Formato de archivo LGO

Los archivos LGO se guardan en un formato de archivo propietario y su estructura interna de archivos no se conoce públicamente. Otro formato de archivo muy conocido para logotipos son los archivos [.ico](/es/image/ico/).
