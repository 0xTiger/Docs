## ¿Qué es un archivo IPR?

El archivo IPR es un archivo de proyecto creado con Java IDE **IntelliJ IDEA**. Toda la información relacionada con la configuración del proyecto, como la estructura del directorio, los archivos de código fuente, las bibliotecas incluidas en el proyecto y la configuración del compilador, se almacena en los archivos IPR. Cuando se abre un proyecto desde IntelliJ IDEA, el IDE lee la información en el archivo IPR y carga los archivos y configuraciones correspondientes. Los archivos IPR han sido reemplazados por los archivos .IDEA que contienen toda la información como Directorio de configuración.

## Formato de archivo de derechos de propiedad intelectual

Los archivos IPR se guardan en el formato de archivo universalmente comprensible [XML](/es/web/xml/) que es legible por humanos. Contiene toda la información relacionada con el proyecto en etiquetas que se cargan para mostrar contenidos relacionados con el proyecto.

Dado que los archivos IPR se almacenan en formato de archivo XML, se pueden abrir en cualquier editor de texto, como Notepad, Notepad ++, Atom y Apple TextEdit.
