## ¿Qué es un archivo ASC?

Un archivo ASC es un archivo **script** creado en el lenguaje de programación ActionScript. Contiene código que se ejecuta en el software de alojamiento Flash Media Server del lado del servidor. El archivo ASC contiene secuencias de comandos que se utilizan para controlar los eventos de la aplicación, como la carga de aplicaciones Flash, el manejo de las acciones del usuario desde los eventos del mouse y el teclado, la reproducción de contenido multimedia y la salida de la aplicación. Adobe adquirió Macromedia Flash que luego se convirtió en parte del traje de Adobe Creativity como Adobe Flash. Solía guardar archivos de animación en [formato de archivo .swf](/es/page-description-language/swf/).

## Formato de archivo ASC - Más información

Los archivos ASC se guardan como archivos de texto que se pueden abrir y editar con [Adobe Media Encoder](https://www.adobe.com/products/media-encoder.html). Estos se crean y guardan como archivos de texto y se pueden abrir con cualquier editor de texto como Microsoft Notepad, Notepad++ y Apple TextEdit. El archivo ASC se ejecuta de forma predeterminada cuando se ejecuta Flash Media Server y la aplicación.
