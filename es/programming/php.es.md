## ¿Qué es un archivo PHP?
Un archivo con extensión .php se refiere a un lenguaje de programación de código abierto, que se utiliza para escribir scripts del lado del servidor, que se ejecutarán en un servidor web. Es el lenguaje de secuencias de comandos web más utilizado que normalmente se utiliza para el desarrollo de aplicaciones web a gran escala. El sistema de blogs más grande de la web, es decir, WordPress, y la red social más grande, Facebook, se basan en el lenguaje PHP. Es compatible con varias plataformas como Windows, Linux, Unix, Mac OS X, etc. También se ejecuta con éxito en casi todos los servidores que se utilizan hoy en día, es decir, Apache, IIS, etc. PHP admite una amplia gama de bases de datos, incluida MySQL.

## Formato de archivo PHP ##

Un solo PHP puede contener texto, CSS, [HTML](/es/web/html/), JavaScript y código PHP. El código PHP se ejecuta en el servidor y el resultado se devuelve al navegador como HTML simple. Los archivos PHP se pueden abrir con cualquier editor de texto y editar en el lugar, aunque aplicaciones como Adobe Dreamweaver, las herramientas de desarrollo PHP de Eclipse ofrecen una manera conveniente de escribir y modificar código PHP.

### Sintaxis ###

Se ejecuta un script PHP en el servidor y el resultado HTML sin formato se envía de vuelta al navegador. Un script PHP comienza con<?php and ends with ?>

### Ejemplo PHP ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
