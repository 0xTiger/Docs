## ¿Qué es un archivo AS3PROJ?

Un archivo AS3PROJ es un archivo de proyecto creado por FlashDevelop, un IDE de programación de código abierto gratuito para desarrollar aplicaciones Flash y web. Guarda toda la información relacionada con el proyecto y la configuración aplicada por el usuario mediante el IDE de FlashDevelop. La información almacenada en un archivo AS3PROJ incluye configuraciones de compilación, referencias a archivos de origen del proyecto y opciones para generar la salida [archivo .swf](/es/page-description-language/swf/). Cuando se crea un nuevo proyecto, se guarda como archivo AS3PROJ en la misma ubicación especificada por el usuario para guardar los datos relacionados con el proyecto.

## Formato de archivo AS3PROJ

Un archivo AS3PROJ se guarda en formato de archivo [XML](/es/web/xml/) que es un formato de archivo universal para el intercambio de información. Se puede abrir en cualquier editor de texto como Notepad, Notepad++ y Atom para examinar el contenido.
