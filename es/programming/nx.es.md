## ¿Qué es un archivo NX?

Un archivo NX es un archivo de videojuego creado para usarse con la consola de juegos virtual, **LowRes NX**. Está escrito en el lenguaje de programación BÁSICO y contiene el código fuente del juego que define cómo funciona el juego. Puede crear sus propios archivos NX para usarlos con la aplicación que se puede cargar y los usuarios pueden jugar.

Puede abrir un archivo NX en un editor de texto, LowRes NX o cualquier editor de código fuente como Microsoft Visual Studio Code.

## Formato de archivo NX

Un archivo NX se guarda en el disco como un archivo de texto sin formato que contiene las instrucciones del juego. Cada archivo NX tiene el código fuente del juego que consta de varias entradas ROM del cartucho. Estas entradas de ROM tienen información de los componentes del juego, como gráficos, mapas de niveles, audio y otros datos que no son del código fuente del juego. Las entradas ROM de un programa se pueden agregar o editar usando los programas de edición NX que vienen incluidos con LowRes NX.

## ¿Cómo crear archivos NX?

### Crear archivos NX en la PC

Puede crear archivos NX en la PC usando:

* Editores de texto, como Microsoft Notepad, Notepad++ o Github Atom
* Editores de código fuente, como Microsoft VIsual Studio Code

### Crear archivos NX en iOS

LowRes NX tiene una versión de iOS que los desarrolladores pueden utilizar para crear archivos NX desde esta aplicación.
