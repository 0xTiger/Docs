## ¿Qué es un archivo F95?

Un archivo con extensión .f95 normalmente indica que es un archivo de código fuente Fortran 95. Fortran es un lenguaje de programación imperativo de propósito general que es particularmente adecuado para la computación científica y numérica. Fortran 95 es una versión mejorada de Fortran 90, que presenta varias características y mejoras nuevas.

Los archivos de código fuente de Fortran suelen contener instrucciones de programa escritas en el lenguaje de programación Fortran; si tiene un archivo con extensión .f95, es probable que deba compilarse utilizando un compilador Fortran; La compilación es el proceso de traducir código fuente legible por humanos en código binario legible por máquina que puede ser ejecutado por una computadora.

## ¿Cómo abrir el archivo F95?

Si desea trabajar o ejecutar un programa en un archivo .f95, necesitará un compilador Fortran; Los compiladores populares de Fortran incluyen GNU Fortran (gfortran), Intel Fortran Compiler (ifort) y otros. Puede compilar y ejecutar programas Fortran utilizando los comandos apropiados proporcionados por el compilador.

Por ejemplo, si tiene instalado gfortran, puede usar los siguientes comandos en la terminal para compilar y ejecutar un programa Fortran:

```
# Compile the Fortran program
gfortran -o my_program my_program.f95

# Run the compiled program
./my_program
```

Si sólo desea ver el contenido del archivo F95, es un archivo de texto sin formato, por lo que puede abrirlo con cualquier editor de texto.
