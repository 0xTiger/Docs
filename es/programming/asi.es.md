## ¿Qué es un archivo ASI? ##

Un archivo ASI es un formato de archivo variable utilizado por el software Alpha Five que se utiliza para desarrollar aplicaciones web. Almacena variables utilizadas por la aplicación Web. Cuando crea un nuevo proyecto de desarrollador en Alpha Five, el archivo ASI también se crea y se guarda junto con el proyecto de desarrollador Alpha Five.

Los archivos ASI se pueden abrir con Alpha Software Alpha Anywhere.

## Formato de archivo ASI

Los archivos ASI se guardan en el formato de archivo propietario de Alpha Five. La aplicación los lee y escribe como parte del proyecto del desarrollador.
