## ¿Qué es un archivo Java?

Un archivo Jav es un archivo de código fuente Java que se ha creado en el lenguaje de programación Java. Es similar al archivo fuente de Java y se puede compilar con intérpretes de Java. Al igual que los archivos [Java](/es/programming/java/), el archivo Jav se utiliza en el desarrollo de aplicaciones Java. Puede ser uno de los muchos archivos de código fuente disponibles en un proyecto Java.

## Formato de archivo Java

Los archivos Jav se almacenan como archivos de texto y se pueden abrir con cualquier editor de texto. Los tipos de archivos Jav han sido comúnmente reemplazados por el tipo de archivo Java y se usan comúnmente para escribir aplicaciones Java. Los códigos Jav se pueden escribir con IDE como Eclipse e IntelliJ Idea, lo que facilita el trabajo con archivos Java.

### Ejemplo de formato de archivo Java

```java
/*
The example code prints
Hello World from Java to the console.
*/
    public class JavaApp {
        public static void main(String[] args) {
            System.out.println("Hello World."); // Prints the string to the console.
        }
    }
```
