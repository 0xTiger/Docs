## ¿Qué es un archivo RIS?

Un archivo RIS es un formato de archivo de etiquetas para el intercambio de datos bibliográficos mediante programas de citas. Contiene varias líneas de datos de citas, que consisten en un código de dos caracteres y el valor correspondiente delimitado por un guión. La información de citas incluida en un archivo RIS incluye datos como el autor, el título, la fecha de publicación, las palabras clave, el editor, el resumen, la dirección del autor, etc. Los archivos RIS son legibles y se pueden abrir con cualquier editor de texto, como Microsoft Notepad, Notepad ++, y Apple TextEdit.

## Formato de archivo RIS: más información

Los archivos RIS se almacenan como archivos de texto sin formato. Cada línea de un archivo RIS consta de un código de dos letras (también conocido como etiquetas), seguido de dos espacios y un guión, y el valor correspondiente. Debido a las etiquetas, los archivos RIS se conocen como formato etiquetado para la representación de citas bibliográficas. El registro de ejemplo en un archivo RIS se muestra a continuación.

## Ejemplo de archivo RIS

El siguiente ejemplo de un artículo de Wikipedia muestra el contenido del archivo RIS para un artículo: Claude E. Shannon. Una teoría matemática de la comunicación, Bell System Technical Journal, 27:379–423, julio de 1948.


|Ejemplo RIS|
---|
|TY - JOUR|
|AU - Shannon, Claude E.|
|PY - 1948|
|DA - julio|
|TI - Una teoría matemática de la comunicación|
|T2 - Boletín técnico del sistema Bell|
|ES - 379|
|EP-423|
|VL - 27|
|ER - |

En este ejemplo, `TY` es Tipo de Referencia y `ER` es Fin de Referencia.
