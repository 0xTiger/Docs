## ¿Qué es un archivo SSP?

Un archivo SSP es un archivo de proyecto creado por el software de creación de protectores de pantalla de Windows conocido como Screensaver Producer de Axialis Software. Puede contener varios tipos diferentes de elementos de salvapantallas, como sprites, películas Flash, presentaciones de fotos y videos. Un archivo SSP puede incluir texto, imágenes, diferentes formas y configuraciones de animación utilizadas para mostrar la animación dentro del archivo SSP. Los archivos SSP se pueden abrir con Axialis Screensaver Producer.

## Formato de archivo SSP: más información

Los archivos SSP se almacenan en el disco como archivos binarios sin información pública disponible sobre su estructura de formato de archivo interno. Axialis Screensaver Producer, la herramienta profesional de creación de protectores de pantalla para Windows que le permite crear fácilmente protectores de pantalla de alta calidad para todas las versiones de Windows hasta Windows 10. Permite crear paquetes de protectores de pantalla instalables que se pueden distribuir fácilmente en línea a los usuarios.

Los archivos SSP se pueden exportar al formato de archivo [SCR](/es/system/scr/).
