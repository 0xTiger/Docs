## ¿Qué es un archivo XMZX?

Un archivo XMZX es un archivo de prueba de examen para realizar pruebas y es utilizado por la aplicación de software ExamSoft SofTest. La aplicación se puede utilizar tanto en Windows como en macOS. Contiene preguntas de un examen que el examinador puede responder a los estudiantes en una computadora con el software SofTest. Los archivos XMZX se envían a los examinadores para realizar la prueba. Se carga en el software SofTest y el candidato o aspirante realiza la prueba. Al finalizar la prueba, el archivo XMDX finalizado se carga automáticamente en los servidores en la nube de ExamSoft.

## Formato de archivo XMZX - Más información

Los archivos XMZX son utilizados por la plataforma de evaluación digital ExamSoft, que utilizan diversas organizaciones para realizar pruebas de certificación. La plataforma cuenta con sus propios servidores en la nube en línea donde se cargan automáticamente todas las pruebas completadas. Las pruebas se llevan a cabo en un centro examinador o los examinadores también pueden realizar el formulario de prueba a los candidatos en sus propias computadoras. ExamSoft está disponible tanto en sistemas operativos Windows como macOS. [XSMX](/es/misc/xsmx/) es otro formato de archivo de prueba de examen de ExamSoft.

## ¿Cómo abrir un archivo XMZX?

Puede abrir un archivo XMZX en el software ExamSoft SofTest solo porque es un formato de archivo propietario. El software SofTest tiene la opción de realizar un examen que se utiliza para iniciar el examen seleccionando el archivo XMZX.
