## ¿Qué es un archivo XSMX?

Un archivo XSMX es un archivo de cuestionario creado con la aplicación ExamSoft SofTest que se utiliza para realizar exámenes. Se utiliza para enviar el examen a los estudiantes que toman en línea y envían el resultado como un archivo XMDX. ExamSoft proporciona soluciones de evaluación basadas en decisiones basadas en datos a través de tecnología educativa y la aplicación SofTest disponible para Windows y MacOS es una de ellas. El archivo XSMX contiene todas las preguntas y se abre en la aplicación SofTest en la computadora del estudiante.

## Formato de archivo XSMX

Los archivos XSMX se guardan y se envían a los examinados como archivos protegidos con contraseña. La protección con contraseña está destinada a garantizar la seguridad y el secreto de las preguntas del examen. La contraseña del examen se comparte con el usuario final para abrir el archivo y realizar el examen.
