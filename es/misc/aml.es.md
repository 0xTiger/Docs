## ¿Qué es un archivo AML?

Un archivo AML (Automation Markup Language) es un archivo de datos que se utiliza para almacenar e intercambiar información de ingeniería de planta. Se basa en el formato de archivo XML, haciendo que el intercambio de información se base en estándares abiertos. AutomationML brinda la posibilidad de interconectar herramientas de ingeniería modernas en los campos de ingeniería de plantas mecánicas, diseño eléctrico, controladores lógicos programables y control de robots. Los archivos AML se pueden ver y editar con la herramienta Editor de AutomationML.

## Formato de archivo AML

AML utiliza modelos de datos orientados a objetos basados en [XML](/es/web/xml/) para el almacenamiento e intercambio de modelos de ingeniería. Los componentes de la planta de ingeniería se representan como objetos que tienen diferentes aspectos, y cada objeto tiene la capacidad de actuar como subobjetos. Así, cada modelo más pequeño o más grande puede ser descrito. Las [Especificaciones de AutomationML](https://www.automationml.org/about-automationml/specifications/) brindan una breve guía para comenzar con los detalles de este lenguaje de modelado.
