## ¿Qué es un archivo H4?

Un H4 o HDF4 es uno de los formatos de datos jerárquicos (HDF) y se utiliza para almacenar una gran cantidad de datos. Es un formato de archivo de código abierto que admite el almacenamiento de datos heterogéneos complejos. HDF4 es un formato de datos más antiguo en comparación con [HDF5](/es/misc/h5/). Los formatos de archivo HDF son mantenidos y publicados por el grupo HDF, una organización sin fines de lucro. Los archivos HDF también admiten el almacenamiento de metadatos para representar completamente los datos almacenados.

## Breve historia del formato de archivo HDF

La formulación de formatos de datos jerárquicos fue iniciada en 1987 por el Grupo de trabajo de fundamentos gráficos (GFTF). Durante este tiempo, se propusieron varios formatos de archivo diferentes para la estandarización de formato único. La NASA seleccionó HDF como el método estándar para almacenar y servir datos científicos.

## Formato de archivo HDF4

Los archivos H4 se guardan en una estructura similar a un "directorio de archivos" según las especificaciones de formato de archivo HDF. Estas especificaciones definen la estructura general de los archivos HDF para el almacenamiento en disco, pero sin que los usuarios se preocupen por la estructura del archivo subyacente.

Las áreas de aplicación de los archivos H4 incluyen aeroespacial, física, ingeniería, finanzas, investigación académica, genómica, astronomía, instrumentos electrónicos y campos médicos.

### Modelo de datos HDF4

El archivo HDF4 o H4 es como un contenedor que consta de objetos de datos heterogéneos. El tipo de conjuntos de datos admitidos por los archivos H4 incluyen:

* Imágenes
* Mesas
* Elementos de texto
* Gráficos
* Documentos como [PDF](/es/pdf/) y [documentos de Excel](/es/spreadsheet/)

## Visor de archivos H4

El grupo HDF proporciona [software HDFView](https://www.hdfgroup.org/solutions/hdf5/) que consiste en la utilidad HDFView y el paquete de objetos Java HDF.
