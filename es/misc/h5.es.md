## ¿Qué es un archivo H5?

Un H5 es uno de los formatos de datos jerárquicos (HDF) que se utiliza para almacenar una gran cantidad de datos. Se utiliza para almacenar una gran cantidad de datos en forma de matrices multidimensionales. El formato se utiliza principalmente para almacenar datos científicos que están bien organizados para una rápida recuperación y análisis. H5 se introdujo como un formato de archivo mejorado para H4. Originalmente fue desarrollado por el Centro Nacional para Aplicaciones de Supercomputación y ahora cuenta con el respaldo de The HDF Group.

## Breve historia del formato de archivo HDF

El formato de archivo HDF fue seleccionado por la NASA como método estándar para almacenar datos científicos. Antes de esto, la formulación de dicho estándar comenzó en 1987 por el Grupo de Trabajo de Fundaciones de Gráficos (GFTF) y HDF fue aprobado formalmente después de la investigación de 15 formatos de archivo diferentes.

## Formato de archivo HDF5

Los archivos H5 están en formato de datos jerárquicos que se adhiere a las especificaciones de formato de archivo HDF5. Estas especificaciones establecen la estructura general del archivo HDF5 para el almacenamiento en disco, pero los usuarios finales no necesitan estos detalles subyacentes.

Los archivos H5 se usan comúnmente en la industria aeroespacial, la física, la ingeniería, las finanzas, la investigación académica, la genómica, la astronomía, los instrumentos electrónicos y los campos médicos.

### Modelo de datos HDF5

El archivo HDF5 es como un contenedor que se compone de objetos de datos heterogéneos. Por ejemplo, estos conjuntos de datos pueden ser imágenes, tablas, elementos de texto, gráficos e incluso documentos como [PDF](/es/pdf/) y [documentos de Excel](/es/hoja de cálculo/).

## Visor de archivos H5

El grupo HDF proporciona [software HDFView](https://www.hdfgroup.org/solutions/hdf5/) que consiste en la utilidad HDFView y el paquete de objetos Java HDF.
