## ¿Qué es un archivo VIMRC?

Un archivo VIMRC es un archivo de configuración utilizado por el editor de texto Vim. Almacena configuraciones para el editor que se cargan cuando se abre el editor. Los desarrolladores de aplicaciones y los administradores del sistema definen configuraciones relacionadas con la apariencia del texto, como el resaltado de sintaxis, que se guardan en el archivo VIMRC. Por lo tanto, el archivo VIMRC permite a los usuarios personalizar la configuración del editor.

## Formato de archivo VIMRC

Los archivos VIMRC se guardan como archivos de texto sin formato y se pueden abrir/editar en cualquier editor de texto como Notepad, Notepad++ y TextEdit. Contienen instrucciones que se pueden ejecutar en el editor de Vim comenzando con un ":", seguido del comando. Los archivos VIMRC contienen instrucciones para configuraciones como sangría de texto y colores de ventanas y fuentes. Un formato de archivo relevante es [VIM](/es/misc/vim/).
