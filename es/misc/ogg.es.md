## ¿Qué es un archivo OGG?

Un archivo OGG es un archivo gráfico creado con el software Origin. Proporciona la capacidad de almacenar diferentes tipos de gráficos para mostrar datos importados para el análisis de datos. Los tipos de gráficos admitidos por el tipo de archivo OGG incluyen columna, barra, pastel, cascada, 3D, polar y símbolo. El software Origin fue desarrollado por OriginLab y permite a los usuarios realizar análisis de datos y generar resultados como gráficos.

## Formato de archivo OGG

Los archivos OGG contienen gráficos para la representación de datos importados. Cuando se abre un archivo OGG, muestra el gráfico en una ventana separada en Origin u Origin Viewer, que está disponible gratuitamente en comparación con el software Origin. La información de metadatos, como el título y los colores de un gráfico, también se almacena dentro del archivo OGG.

## Acerca del software OriginLab Origin

El software Origin también ofrece la opción de ser utilizado como servidor de automatización. En tal caso, las aplicaciones cliente, por ejemplo, LabVIEW, Microsoft Excel, Python o aplicaciones desarrolladas a medida en C#/VB/VC pueden intercambiar datos con Origin. [Python_to_OriginLab](https://github.com/jsbangsund/python_to_originlab) es una API de Python para automatizar el envío de datos y funciones básicas de trazado en OriginLab.
