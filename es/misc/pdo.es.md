## ¿Qué es un archivo PDO?

Pepakura Designer es un programa que permite a los usuarios crear modelos artesanales en papel a partir de modelos 3D; Los archivos Pepakura suelen tener una extensión .pdo; Estos archivos contienen instrucciones e información de diseño necesaria para imprimir y ensamblar un modelo en papel.

Los archivos de Pepakura Designer están en un formato propietario específico del software Pepakura; El formato de archivo está diseñado para almacenar información sobre el modelo 3D, incluida su geometría, referencias de textura e instrucciones de ensamblaje.

Pepakura Designer trabaja con modelos 3D en formatos como [.obj](/3d/obj/), [.3ds](/3d/3ds/), [.dxf](/cad/dxf/) y otros; Luego, estos modelos se desdoblan en patrones 2D que se pueden imprimir y ensamblar en el modelo final en papel.

El archivo .pdo incluye instrucciones sobre cómo cortar, doblar y pegar piezas de papel impresas para recrear el modelo 3D; Cada pieza está etiquetada y numerada para guiar el proceso de ensamblaje.

## ¿Cómo abrir un archivo PDO?

Para abrir y ver archivos de Pepakura Designer, necesitará **Pepakura Viewer**, un programa complementario gratuito de **Pepakura Designer**. El Visor permite a otros ver e imprimir modelos sin necesidad de un software de diseño completo.

## Acerca del diseñador Pepakura

Pepakura Designer es una aplicación de software especializada diseñada para crear modelos artesanales en papel a partir de modelos digitales 3D. El software permite a los usuarios importar modelos 3D en varios formatos, como .obj o .3ds, y luego desplegar y aplanar estos modelos en patrones 2D que se pueden imprimir en papel. Estos patrones, guardados con una extensión de archivo .pdo, contienen instrucciones y marcas para cortar, doblar y ensamblar las piezas de papel en un modelo tridimensional. Pepakura Designer es ampliamente utilizado por aficionados y entusiastas interesados en crear complejos modelos en papel de diversos objetos, personajes y diseños.
