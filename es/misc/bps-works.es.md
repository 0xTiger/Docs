## ¿Qué es un archivo BPS?

Un archivo .BPS es esencialmente una copia de seguridad de un documento de procesamiento de textos creado con Microsoft Works, que normalmente usa la extensión de archivo .WPS para sus documentos. Estos archivos .BPS suelen ser generados automáticamente por Microsoft Works como precaución en caso de que el programa falle mientras trabaja en el documento. Es una forma de salvaguardar su trabajo y evitar la pérdida de datos.

En la siguiente sección, exploraremos las aplicaciones de software asociadas con los archivos BPS.

## Microsoft funciona

Microsoft Works era un paquete de software de productividad desarrollado por Microsoft. Se introdujo por primera vez en 1987 y ganó popularidad como una alternativa más asequible al paquete completo de Microsoft Office. Microsoft Works incluía varias aplicaciones ofimáticas básicas, como:

- **Procesador de textos:** Microsoft Works tenía una aplicación de procesamiento de textos que permitía a los usuarios crear y editar documentos. Utilizaba el formato de archivo [.WPS](/es/word-processing/wps/) para documentos de procesamiento de textos.

- **Hoja de cálculo:** Incluía aplicación de hoja de cálculo para crear y administrar hojas de cálculo. El formato de archivo de las hojas de cálculo era [.XLR](/es/spreadsheet/xlr/).

- **Base de datos:** Microsoft Works también tenía una aplicación de base de datos para organizar y administrar datos. El formato de archivo para los archivos de bases de datos era [.WDB](/es/database/wdb/).

- **Calendario y libreta de direcciones:** Además de las aplicaciones principales de Office, Microsoft Works a menudo incluía herramientas para administrar calendarios y libretas de direcciones.

- **Formato de procesador de textos de Works (.WPS):** El formato de archivo nativo del componente de procesamiento de textos de Microsoft Works. Los archivos .WPS contenían texto, formato y gráficos simples.

- **Descontinuación:** Microsoft descontinuó oficialmente Microsoft Works en 2007 y alentó a los usuarios a cambiar a Microsoft Office u otras alternativas.

## ¿Cómo abrir el archivo BPS?

Los programas que abren archivos BPS incluyen
