## ¿Qué es un archivo PUB?

Un archivo PUB es un formato de archivo de documento de Microsoft Publisher. Se utiliza para crear varios tipos de documentos de diseño de diseño, como boletines, volantes, folletos, postales, etc. Los archivos PUB pueden contener imágenes de texto, rasterizadas y vectoriales. Los documentos creados con archivos PUB se utilizan principalmente en sitios web y materiales de marketing, como correos electrónicos. Los archivos PUB se pueden abrir con la aplicación Microsoft Publisher Desktop, [Microsoft Publisher 365](https://www.microsoft.com/en/microsoft-365), [LibreOffice Draw](https://www.libreoffice.org/discover /draw/) y [Adobe InDesign](https://www.adobe.com/products/indesign.html).

## Formato de archivo PUB: más información

Los archivos PUB se almacenan como archivos binarios en disco. El contenido de un archivo PUB también se puede convertir al formato de archivo [DOCX](/es/word-processing/docx/) desde la aplicación Microsoft Publisher mediante la opción Guardar como.
