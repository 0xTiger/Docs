## ¿Qué es un archivo BTAPP?

Un archivo BTAPP es una aplicación que se ha desarrollado para el programa de descarga de TORRENT uTorrent. Es esencialmente una aplicación para uTorrent que puede acceder a todas las funciones del cliente de uTorrent. Estas aplicaciones se pueden agregar al cliente de uTorrent desde la aplicación. Esto instalará la aplicación en el cliente de uTorrent y proporcionará toda la funcionalidad de descargar archivos [TORRENT](/es/misc/torrent/).

## Formato de archivo BTAPP - Más información

Un archivo de aplicación BTAPP se guarda como archivo binario. Contiene código [HTML](/es/web/html/) y [JavaScript](/es/web/js/) que se puede abrir en un navegador integrado.
