## ¿Qué es un archivo 3GP?

3GP (formato de archivo 3GPP) es un formato de contenedor multimedia utilizado en teléfonos móviles 3G. También se puede jugar en algunos teléfonos 2G y 4G. 3GP está definido por el Proyecto de Asociación de Tercera Generación (3GPP) para servicios multimedia 3G UMTS (Sistema Universal de Telecomunicaciones Móviles). Los archivos 3GP se almacenan con la extensión .3gp. El formato de archivo 3GP se diseñó para teléfonos basados en GSM (Sistema global para comunicaciones móviles) y es el formato estándar para archivos multimedia enviados a través de MMS (Servicio de mensajería multimedia). A diferencia de [MP4](/es/video/mp4/), que utiliza el códec H.263, los archivos 3GP utilizan preferentemente el códec H.263.

## Detalles técnicos del formato de archivo 3GP

El formato de video 3GP se basa en el formato de archivo de medios base ISO definido en ISO/IEC 14496-12 – MPEG-4 Parte 12. Fue desarrollado para teléfonos móviles para reducir el almacenamiento, el ancho de banda y el uso de datos. El formato de archivo 3GP admite los siguientes códecs:

- Video
  - MPEG-4 Part 2
  - H.263
  - MPEG-4 Part 10 (AVC/H.264)
-Audio
  - AMR-WB
  - AMR-WB+
  - AMR-NB
  - AAC-LC
  - HE-AAC v1
  - HE-AAC v2 (Enhanced aacPlus)

El formato de archivo 3GP permitió el uso de códecs H.263 y AMR y definió nuevas casillas a las que se refieren los códecs. Estos puntos de código se registraron en formato de archivo de medios base ISO para el almacenamiento de información específica de medios MPEG-4.

## Dispositivos compatibles con archivos de video 3GP

- Teléfonos móviles con capacidades 3G
- Audio de PlayStation 3 importado desde un CD
- La Nintendo 3DS reprodujo videos de YouTube usando tecnología 3GP
- Los iDevices de Apple admiten la reproducción de archivos solo como archivos de transferencia
