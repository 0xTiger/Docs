## ¿Qué es un archivo ECW? ##
Un archivo ECW consta de una imagen diseñada para su uso con datos geoespaciales. La imagen contenedora es en realidad una imagen comprimida creada en el formato Wavelet de compresión mejorada. Los archivos ECW se utilizan generalmente para almacenar proyecciones de mapas satelitales e imágenes aéreas. Para admitir aplicaciones geoespaciales, la información de proyección de mapas se puede incrustar en los archivos ECW.

## formato de archivo ECW
ECW es un formato de imagen patentado basado en la compresión wavelet; optimizado para imágenes satelitales y fotografía aérea. Este formato ahora es propiedad de Intergraph, parte de Hexagon AB, pero básicamente fue desarrollado por Earth Resource Mapping. ECW es un formato de compresión con pérdida que comprime imágenes de gran tamaño con un fino contraste alterno mientras conserva su calidad visual.
 

### Propiedades
Aquí hay algunas propiedades típicas del formato ECW:
- La información de proyección de mapas se puede agregar al formato de archivo ECW para admitir aplicaciones geoespaciales.
- Los datos de imagen de hasta 65 535 bandas, incluidas las capas o los colores, se pueden comprimir en el formato de archivo ECW a una velocidad de más de 25 MB por segundo.
- El protocolo ECWP es un protocolo de transmisión eficiente que se utiliza para transmitir imágenes ECW y JPEG2000 a través de redes.
- Un SDK rápido compatible con ECW y JPEG2000 está disponible sin cargo para la implementación de escritorio para Linux, Windows y MacOSX
