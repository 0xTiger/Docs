## ¿Qué es un archivo CLIP?

Un archivo CLIP es un archivo de imagen rasterizado generado por la aplicación de dibujo 2D Clip Studio Paint. Es similar al archivo [PSD](/es/image/psd/) de Adobe y contiene múltiples capas para administrar diferentes elementos gráficos en forma de capas como texto, imágenes, bolígrafo, lápiz, pinceles y formas. El formato de archivo CLIP se utiliza para diseñar varios tipos de gráficos diferentes, como ilustraciones y animaciones. La aplicación Clip Studio Paint fue desarrollada por una empresa japonesa CELSYS, Inc. que se estableció en mayo de 1991.

## Formato de archivo CLIP - Más información

El formato de archivo CLIP fue introducido por CELSYS para reemplazar el formato de archivo LIP original. Almacena elementos gráficos en forma de capas que se pueden editar para cambios. También es el formato de archivo predeterminado para la aplicación [CLIP Studio Paint](https://www.celsys.co.jp/en/service/studio_paint/) que está disponible para las plataformas Windows, MacOS, Android e iOS.

## Tipos de dibujo admitidos

El formato de archivo CLIP puede almacenar animaciones e ilustraciones que se pueden usar en una variedad de diferentes tipos de trabajo.
