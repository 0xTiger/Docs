## ¿Qué es un archivo CSL?

Un archivo de biblioteca de símbolos de Corel (CSL) es una biblioteca de símbolos que se utiliza en el software CorelDRAW. Almacena símbolos que se pueden usar en otros archivos. Los archivos CSL se pueden almacenar localmente o en una red para que estén accesibles para una fácil implementación y administración. Estos se guardan con el archivo de CorelDRAW junto con todas las instancias de símbolos utilizadas en el documento.

## Formato de archivo CSL

Los archivos CSL se guardan en formato de archivo propietario de CorelDRAW. Al ser una colección de símbolos, los símbolos dentro de un archivo CSL se pueden importar y agregar a los dibujos de un proyecto.
