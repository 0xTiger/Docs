## ¿Qué es un archivo PICT?

Un archivo PICT es una imagen gráfica guardada en el formato de archivo de imagen de Apple Macintosh. Puede almacenar datos de imagen como formatos de archivo rasterizados y vectoriales. Era el formato de archivo de imagen predeterminado para el programa Apple QuickDraw. Los archivos PICT permiten el intercambio de datos ráster y vectoriales entre aplicaciones Mac. Hace tiempo que el formato se reemplazó por el formato de archivo más flexible [PDF](/es/pdf/). Los archivos PICT a veces se encuentran con la extensión **[.PCT](/es/image/pct/)**.

## Formato de archivo PICT - Más información

Los archivos PICT se guardan como archivos de imagen binarios que se pueden abrir con aplicaciones como XnView, GIMP, Corel PaintShop Pro y Apple Preview. Admite imágenes de un solo canal (con capacidad de almacenamiento de 8 colores) o imágenes RGB de canal de color que pueden almacenar miles de colores. Estos se conocen como formatos de archivo **PICT 1** y **PICT 2** respectivamente.
