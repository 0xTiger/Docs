## ¿Qué es un archivo DCX?

Un archivo DCX es un archivo de imagen que consta de varias páginas para generar archivos de fax digital. Cada archivo DCX comienza con un encabezado de 4 bytes que identifica el archivo DCX como firma, seguido de imágenes individuales, almacenadas como archivos [PCX](/es/image/pcx/) incrustados. La firma de 4 bytes identifica el archivo DCX y va seguida de una lista de hasta 1023 compensaciones de archivos de 4 bytes. El formato de archivo DCX fue desarrollado por Zsoft. Varias aplicaciones pueden abrir archivos DCX como Adobe Photoshop, Corel PaintShop Pro e IrfanView.
