## ¿Qué es un archivo JPF? ##

Un archivo con la extensión .jpf es una extensión del sistema de codificación de imágenes JPEG 2000 ISO/IEC 15444 y se denomina Parte 2 ISO/IEC 15444-2. Define y especifica un conjunto de métodos de compresión sin pérdida (preservación de bits) y con pérdida para codificar imágenes fijas digitales en color, de dos niveles, en escala de grises, de tonos continuos o imágenes de componentes múltiples. La primera parte de ISO/IEC 15444-1 se refiere a [JP2](/es/image/jp2/) que utiliza la tecnología wavelet para codificar contenido sin pérdidas y es la base para los formatos de archivo de imagen JPEG 2000. El formato de archivo JPF no recibió una buena acogida debido al amplio uso del formato JPEG. Los archivos JPF se pueden abrir con aplicaciones de imágenes populares como Adobe Photoshop 2020, Adobe Illustrator 2020 y CorelDraw Graphics Suite 2020.

## Breve historia

En 2000, el comité del Grupo Conjunto de Expertos Fotográficos diseñó JP2 con el objetivo de mejorar su propio estándar JPEG basado en la transformada de coseno discreta con este nuevo método basado en wavelet. El comité de JPEG tenía como objetivo proporcionar sus métodos de referencia libres de derechos de licencia. En la licencia JP2 ganando competencia entre 20 empresas, ganaron por un pelo. JPEG 2000 ha sido declarado estándar ISO, aunque la mayoría de los navegadores web no están preparados para darle una mano a JPEG 2000 desde 2017. En 2004, el formato ISO/IEC 15444-2 se aceptó públicamente como extensión del formato de archivo JP2.

## Formato de archivo JPF

El formato de archivo JPF define procesos de decodificación extendidos para la conversión de datos de imágenes comprimidas para su reconstrucción. Es un formato de archivo extendido que especifica una sintaxis de flujo de código extendido que contiene información para interpretar los datos de la imagen comprimida. Este estándar ampliado especifica un contenedor para almacenar metadatos de imágenes y proporciona orientación sobre procesos de codificación ampliados para convertir datos de imágenes de origen en datos de imágenes comprimidos.

### Organización de archivos

JPF es el formato de archivo de almacenamiento formal cuando los archivos JPX se almacenan en sistemas de archivos informáticos. Además, otras Recomendaciones/Estándares internacionales pueden definir otras casillas para usar dentro de los archivos JPX. Sin embargo, toda la información contenida en un archivo JPX estará en formato de cuadro; los flujos de bytes que no estén en el formato de cuadro no se encontrarán en el archivo. La estructura binaria de un cuadro en un archivo JPX es idéntica a la definida en el formato de archivo [JP2](/es/image/jp2/).
