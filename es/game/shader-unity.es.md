## ¿Qué es un archivo SHADER?

Un **"Unity Shader Asset"** se refiere al sombreador creado en el motor de desarrollo de juegos Unity. En Unity, los sombreadores se utilizan para controlar cómo se realiza la representación de gráficos, definiendo cómo aparecen los objetos y materiales en una escena 3D. Los sombreadores se pueden utilizar para manipular la iluminación, el mapeo de texturas y varios otros efectos visuales en el proyecto Unity.

## Activo de sombreador de Unity

Un activo de Unity Shader normalmente consta de un archivo Shader Graph o ShaderLab. Aquí hay una breve explicación de ambos:

1. **Shader Graph**: Unity introdujo Shader Graph como herramienta visual para crear sombreadores. Permite a los desarrolladores crear sombreadores sin escribir código. Puede conectar nodos visualmente para definir cómo deben comportarse los materiales. El archivo Shader Graph normalmente tiene la extensión ".shadergraph".
    







2. **ShaderLab**: ShaderLab es un lenguaje de marcado utilizado en Unity para escribir sombreadores. Permite a los desarrolladores definir propiedades y comportamientos del sombreador en formato basado en texto. Un archivo ShaderLab normalmente tiene la extensión ".shader".
    







## Trabajar con recursos SHADER

Para trabajar con Shader Assets en Unity, normalmente haría lo siguiente:

1. Cree un nuevo activo de sombreado utilizando Shader Graph de Unity o escribiendo código de ShaderLab.
    







2. Adjunte Shader Asset al material en Unity. Este material luego se puede aplicar a los objetos de tu juego o escena.
    







3. Personalice y modifique Shader Asset según sea necesario para lograr los efectos visuales o el comportamiento de renderizado deseados.
    







4. Utilice Shader Asset para controlar varios aspectos del renderizado, incluido cómo reaccionan los objetos a la iluminación, las sombras y los materiales.
    







5. También puede animar propiedades dentro del sombreador para obtener efectos visuales dinámicos.
    








Al utilizar Shader Assets en Unity, puedes crear gráficos visualmente impresionantes y únicos para tus juegos o aplicaciones.
