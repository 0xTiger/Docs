## ¿Qué es un archivo USX?

Un archivo USX contiene mallas de Unreal Engine que son objetos/modelos 3D prediseñados que se utilizan para generar el entorno. Los modelos del archivo USX tienen diferentes posiciones, orientaciones y tamaños, pero permanecen estáticos. Las mallas estáticas se pueden optimizar para el rendimiento y se pueden usar para crear varios niveles de detalle (LOD) para mejorar el rendimiento del juego. También se pueden usar para crear volúmenes de colisión y se pueden usar como parte de simulaciones físicas.

Los archivos USX se pueden crear con el editor de niveles Unreal "UnrealEd".

## Formato de archivo USX: más información

Los archivos USX se guardan en el disco como archivos binarios. Los detalles de la estructura de archivo interna del formato de archivo USX no están disponibles públicamente para referencia del desarrollador.
