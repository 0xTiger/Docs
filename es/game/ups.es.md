## ¿Qué es un archivo UPS?

Un archivo UPS, que significa **Universal Patching System**, es un tipo de archivo de parche que se utiliza para modificar o aplicar cambios a los datos de un videojuego u otro software. Los archivos UPS se suelen utilizar en la comunidad de emulación para aplicar parches o modificaciones a las ROM (memoria de solo lectura) de videojuegos antiguos.

## Archivo de parche de UPS

Así es como funciona un archivo de parche de UPS:

1. **Datos Originales:** Comienzas con el archivo ROM original de un videojuego o software que deseas modificar. Esta ROM contiene el código y los datos del juego.
    






2. **Archivo de parche:** Se crea un archivo de parche de UPS utilizando un software de parcheo especializado. Este archivo de parche contiene cambios que desea aplicar a los datos originales. Estos cambios pueden incluir correcciones de errores, parches de traducción u otras modificaciones.
    






3. **Proceso de aplicación de parches:** Para aplicar los cambios, utilice una herramienta de aplicación de parches de UPS que toma como entrada la ROM original y el archivo de parche de UPS. Luego, la herramienta de parcheo aplica los cambios especificados en el archivo de parche de UPS a la ROM original, creando una versión modificada del juego o software.
    






4. **Resultado:** El resultado es una ROM parcheada o modificada que incorpora cambios del archivo de parche de UPS. Esta ROM parcheada se puede reproducir en un emulador o utilizar de otras formas dependiendo de las modificaciones realizadas.
    







Los archivos de parche de UPS son populares en la comunidad de emulación porque permiten a los usuarios realizar mejoras o modificaciones en juegos antiguos sin distribuir la ROM completa del juego, que puede estar sujeta a derechos de autor. En cambio, los usuarios pueden compartir solo el archivo de parche de UPS que contiene cambios y los usuarios pueden aplicarlo a su ROM original de propiedad legal.

Los parches de UPS ofrecen varias ventajas sobre [IPS](/es/game/ips/):

1. **Sin limitación de tamaño:** A diferencia de los parches IPS que están restringidos a 16 MB, los parches UPS se pueden usar con archivos de cualquier tamaño. Esta flexibilidad hace que UPS sea particularmente útil para juegos y software más grandes.
    






2. **Parches bidireccionales:** UPS admite parches bidireccionales, lo que significa que un único parche de UPS puede aplicar y eliminar cambios en un juego o software. Esta característica simplifica el proceso de modificación o reversión de alteraciones, mejorando la experiencia del usuario.
    






3. **Sumas de comprobación CRC32:** UPS emplea sumas de comprobación CRC32 como mecanismo integrado para garantizar que se apliquen parches al juego o software correcto. Esta verificación de suma de comprobación ayuda a mantener la integridad de los archivos parcheados y evita la aplicación accidental o incorrecta de parches.

## ¿Cómo aplicar un parche de UPS?

Para aplicar un parche de UPS, necesitará una herramienta de parcheo de UPS, como Lunar IPS (para Windows), NUPS (para Windows) o MultiPatch (para múltiples plataformas) y la ROM original del juego o software que desea parchear.

## ¿Cómo abrir el archivo UPS?

Los programas que abren archivos UPS incluyen

- **IPS lunar** (para Windows)
- **NUPS** (para Windows)
- **MultiPatch** (para múltiples plataformas)
