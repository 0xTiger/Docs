## ¿Qué es un archivo PUP?

Un archivo PUP es un archivo de parche de actualización del sistema utilizado por el software del sistema PlayStation 4 (PS4) y PlayStation 5 (PS5). Actualiza el firmware del juego a una versión más nueva que puede contener mejoras de software en términos de corrección de errores, nuevas funciones del sistema operativo y mejoras en términos de rendimiento del software. El archivo PUP también puede contener parches para corregir errores que, de lo contrario, podrían generar problemas de seguridad. Los archivos PUP se pueden descargar al sistema conectándolo a Internet o descargando manualmente la actualización del firmware a una computadora y luego transfiriéndola al dispositivo usando una memoria USB u otro dispositivo de almacenamiento. La versión anterior de PlayStation, es decir, PS3, también usaba el archivo PUP para actualizar el firmware.

## Formato de archivo PUP: más información

Los archivos PUP se guardan en el disco en formato de archivo binario [comprimido](/es/compression/). Sin embargo, los detalles de la compresión no están disponibles públicamente. El contenido de un archivo PUP se puede extraer con el programa PS3 PUP Extractor en el sistema operativo Windows. Para actualizar el sistema de software de PS3, el archivo PUP se carga y se guarda en la carpeta /PS3/UPDATE de la máquina. Esto permite que la opción de actualización del sistema encuentre el archivo de actualización del firmware y lo ejecute.

### Convención de nomenclatura de formato de archivo PUP

* `PS3UPDAT.PUP`: el nombre de archivo estándar utilizado para las actualizaciones de PlayStation 3
* `PS4UPDATE.PUP`: el nombre de archivo estándar utilizado para las actualizaciones de PlayStation 4
* `PS5UPDATE.PUP`: el nombre de archivo estándar utilizado para las actualizaciones de PlayStation 5

## Uso del archivo PUP para restablecer la PlayStation de fábrica

Los archivos PUP se pueden usar para restablecer de fábrica su dispositivo PlayStation. Esto sobrescribirá todos los datos del dispositivo, eliminará el firmware existente y lo configurará como un dispositivo nuevo. El restablecimiento de fábrica de tu PlayStation también eliminará los datos guardados del juego.
