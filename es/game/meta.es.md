## ¿Qué es un archivo META?

El archivo META contiene datos de varios juegos que utilizan Exient XGS Engine, como Angry Birds Go. o Transformadores de Angry Birds. El archivo META captura y retiene información sobre aspectos particulares del juego, como registros de fallas o registros de elementos.

Los archivos META son generados automáticamente por el juego y no están destinados a la modificación ni al acceso directo del usuario. Si el archivo META está dedicado a registrar fallas del juego, normalmente se llamará "crashes.meta". De manera similar, si se utiliza el archivo META para registrar información relacionada con elementos, seguirá la convención de nomenclatura de "XXX.xml.meta", donde "XXX" representa un identificador descriptivo para el tipo específico de elementos que se registran.

## Motor XGS existente

Exient Entertainment creó Exient XGS Engine, un motor de juegos. Es un marco de software creado para facilitar el proceso de creación y desarrollo de videojuegos. Ofrece a los programadores una variedad de herramientas, bibliotecas y funciones para acelerar el proceso de creación de juegos.

Una variedad de funciones y capacidades proporcionadas por Exient XGS Engine facilitan la creación de juegos en una variedad de plataformas, incluidas PC, consolas y dispositivos móviles. Estas funciones frecuentemente consisten en producir imágenes, simular física, administrar audio, manejar entradas, tener capacidades de red y otras cosas.
