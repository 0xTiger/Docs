## ¿Qué es un archivo DEK?

Un archivo con extensión .dek es un archivo de mazo de cartas utilizado por el juego de autos coleccionables en línea, Magic: The Gathering Online (MTGO). Aunque en la mayoría de los casos, estas barajas de cartas se pueden jugar, en algunos casos, los archivos DEK enumeran una colección de cartas que no se pueden jugar. Estos no se pueden reproducir, pero actúan como un subconjunto de una colección. Los archivos DEK se pueden crear a partir de archivos [CSV](/es/spreadsheet/csv/) al exportarlos con Delver Lens y se pueden usar en Tabletop Simulator.

## Formato de archivo DEK

Los archivos DEK se almacenan como archivos de texto sin formato de varias líneas en formato [XML](/es/web/xml/) que son legibles por humanos. Estos pueden abrirse con cualquier editor de texto como Microsoft Notepad, Notepad++ y Wizards of the Coast Magic: The Gathering Online. Muchos jugadores de Magic también utilizan los archivos DEK para importar/exportar desde muchos sitios web y de creación de mazos como [CardHoarder](https://www.cardhoarder.com/). Estos sitios web permiten cargar archivos MTGO DEK como un solo archivo o una lista de archivos.
