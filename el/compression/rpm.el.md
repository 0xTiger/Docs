## Τι είναι ένα αρχείο RPM;

Ένα αρχείο με επέκταση .rpm είναι ένα πακέτο λειτουργικού συστήματος Red Hat Linux για εγκαταστάσεις προγραμμάτων σε συστήματα Linux. Το Red Hat Package Manager χρησιμοποιεί τη μορφή αρχείου RPM και είναι δωρεάν και ανοιχτού κώδικα σύστημα διαχείρισης πακέτων. Αν και τα αρχεία RPM μπορούν να χρησιμοποιηθούν ως έχουν για την εγκατάσταση προγραμμάτων, αυτά μπορούν να μετατραπούν σε άλλες μορφές πακέτων όπως [DEB](/el/compression/deb/) χρησιμοποιώντας το πρόγραμμα Debian που ονομάζεται Alien.

## Μορφή αρχείου RPM

Ένα αρχείο RPM είναι ένα δυαδικό αρχείο που μπορεί να περιέχει ένα σύνολο αρχείων. Τις περισσότερες φορές, τα αρχεία RPM είναι "δυαδικά RPM" που είναι τα μεταγλωττισμένα εκτελέσιμα του λογισμικού. Το αρχείο RPM μπορεί να περιέχει RPM προέλευσης (SRPM) που μπορούν να χρησιμοποιηθούν για τη δημιουργία του πακέτου από τον πηγαίο κώδικα. Η μορφή αρχείου RPM αποτελείται από τέσσερις ενότητες.

* Lead - Προσδιορίζει το αρχείο ως αρχείο RPM
* Υπογραφή - Μπορεί να χρησιμοποιηθεί για να διασφαλιστεί η ακεραιότητα ή/και η αυθεντικότητα
* Κεφαλίδα - Περιέχει μεταδεδομένα, όπως όνομα πακέτου, έκδοση, αρχιτεκτονική, λίστα αρχείων κ.λπ.
* Αρχείο Αρχείων - Γνωστό και ως ωφέλιμο φορτίο, το οποίο είναι συνήθως σε μορφή cpio, συμπιεσμένο με [GZIP](/el/compression/gz/)
