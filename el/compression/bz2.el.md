## Τι είναι ένα αρχείο BZ2;

Τα BZ2 είναι συμπιεσμένα αρχεία που δημιουργούνται χρησιμοποιώντας τη μέθοδο συμπίεσης ανοιχτού κώδικα [BZIP2](http://www.bzip.org/), κυρίως σε σύστημα UNIX ή Linux. Χρησιμοποιείται για τη συμπίεση ενός μόνο αρχείου και δεν προορίζεται για την αρχειοθέτηση πολλών αρχείων. Αυτό έρχεται σε αντίθεση με τη μορφή αρχείου TAR στις ίδιες πλατφόρμες που αρχειοθετεί πολλά αρχεία σε ένα μόνο αρχείο αλλά χωρίς συμπίεση. Τα αρχεία που συμπιέζονται ως BZ2 μπορούν να αποσυμπιεστούν με εφαρμογές όπως το [WinZip](https://www.winzip.com/win/en/). Το BZIP2 χρησιμοποιεί τον αλγόριθμο συμπίεσης Run-Length Encoding (RLE) ή [Burrows-Wheeler](https://en.wikipedia.org/wiki/Burrows%E2%80%93Wheeler_transform) για να επιτύχει υψηλά επίπεδα συμπίεσης.

## Μορφή αρχείου BZ2

Δεν υπάρχουν διαθέσιμες επίσημες προδιαγραφές για αυτήν τη μορφή αρχείου. Ωστόσο, μια ανεπίσημη [αντίστροφη μηχανική προδιαγραφές](https://github.com/dsnet/compress/blob/master/doc/bzip2-format.pdf) δείχνει ότι μια ροή .bz2 αποτελείται από μια κεφαλίδα 4 byte που ακολουθείται με μηδέν ή περισσότερα συμπιεσμένα μπλοκ. Ακολουθείται αμέσως από έναν δείκτη τέλους ροής που περιέχει ένα CRC 32 bit για ολόκληρη τη ροή απλού κειμένου που υποβάλλεται σε επεξεργασία. Δεν υπάρχει επένδυση μεταξύ των συμπιεσμένων μπλοκ και όλα τα μπλοκ είναι ευθυγραμμισμένα με bit.

## Αποσυμπιέστε/Εξαγωγή αρχείου BZ2

Μπορείτε να αποσυμπιέσετε ένα αρχείο BZ2 σε Windows και Mac OS χρησιμοποιώντας λογισμικό όπως το WinZip. Στο linux, η ακόλουθη εντολή στο τερματικό.

```
bzip2 -d filename.bz2
```
