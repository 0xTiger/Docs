## Τι είναι ένα αρχείο TR3; ##

Το αρχείο TR3 είναι ένα eBook TomeRaider 3 που παρέχει υποστήριξη για γρήγορη αναζήτηση και συμπίεση. Αυτό μπορεί να λειτουργήσει σωστά μέσω του σχετικού προγράμματος TomeRaider. Η Yadabyte είναι ο προγραμματιστής αυτού του λογισμικού, μια βρετανική εταιρεία λογισμικού και ανάπτυξης ιστού με έδρα το Ηνωμένο Βασίλειο. Η εφαρμογή TomeRaider eBook reader μπορεί να χρησιμοποιηθεί για τη σωστή λειτουργία και διαχείριση του περιεχομένου αυτών των αρχείων TR3. Αυτό το σχετικό πρόγραμμα ανάγνωσης eBook μπορεί να εγκατασταθεί σε υπολογιστές που εκτελούνται σε συστήματα που βασίζονται σε Microsoft Windows και σε πολλά μεταφερόμενα gadget. Το αρχείο TR3 ανήκει στην ομάδα αρχείων eBook που χρησιμοποιούνται σε λειτουργικά συστήματα όπως τα Windows 10, Windows 7, Windows 8 / 8.1, Windows Vista, Windows XP.

Οι συγκεκριμένες **Ετικέτες HTML** για το TR3 είναι οι εξής:

|Ετικέτα|Περιγραφή|
---|---|
|\<new> |Για να δημιουργήσετε αρχεία TomeRaider από αρχεία κειμένου, η νέα ετικέτα είναι το μόνο που χρειάζεται.<new> η ετικέτα ορίζει την αρχή των σελίδων.|
|\<CATDEF> ... \</CATDEF> |καθορίζει το όνομα της κατηγορίας|
|\<META> ... \</META> |είναι μια ετικέτα που χρησιμοποιείται για τον ορισμό όλων των μεταδεδομένων που χρησιμοποιούνται στη μορφή αρχείου. Αυτή η ετικέτα περιλαμβάνει έναν αριθμό παραμέτρων.|
|\<EXPMSG> |Αυτή η ετικέτα ελέγχει το μήνυμα που εμφανίζεται όταν ένα αρχείο έχει λήξει. Κάθε φορά που ένας χρήστης επιχειρεί να δει μια σελίδα μετά τη λήξη του αρχείου, εμφανίζεται το μήνυμα λήξης αντί για το πραγματικό κείμενο της σελίδας.|
|\<DIEMSG> |Αυτή η ετικέτα είναι παρόμοια με την EXPMSG. Χρησιμοποιείται για την εμφάνιση μηνύματος όταν το αρχείο πεθαίνει.|
|\<ENGMSG> |Αυτή η ετικέτα χρησιμοποιείται για τη δημιουργία του μηνύματος που εμφανίζεται για κρυπτογραφημένες σελίδες. Το μήνυμα εμφανίζεται αντί για το κείμενο της σελίδας εάν ο χρήστης επιχειρήσει να ανοίξει μια κρυπτογραφημένη σελίδα πριν την ξεκλειδώσει.|
|\<expmsg> ,\<diemsg> και \<encmsg> |Παράβλεψη εντός του κειμένου της σελίδας. Πρέπει να τοποθετηθούν στην ενότητα κεφαλίδας αρχείου πριν από την πρώτη<new> ετικέτα.|
|\<SELECTION> . \</ SELECTION> |Χρησιμοποιείται για την υποστήριξη ερωτημάτων. Αυτή η ετικέτα πρέπει να είναι πριν από την πρώτη<new> ετικέτα. Συγκεντρώνει τα δεδομένα επιλογής για το ερώτημα του χρήστη.|
|\<catset> . \</catset> | Χρησιμοποιείται για την ανάθεση ονομάτων κατηγοριών για κάθε θέμα.|


## Προβλήματα με το άνοιγμα ενός αρχείου TR3 ##

Ακολουθεί η λίστα με ορισμένα κοινά ζητήματα που μπορεί να προκύψουν και να προκαλέσουν δυσλειτουργία στη μορφή αρχείου:

* Κατεστραμμένο αρχείο
* Μολυσμένο αρχείο λόγω ιού
* Ακατάλληλοι σύνδεσμοι προς το αρχείο TR3 στις προσβάσεις αρχείων
* Δεν υπάρχει δικαίωμα πρόσβασης στο σύστημα για το άνοιγμα των αρχείων
* Ξεπερασμένη μονάδα δίσκου στο σύστημά σας
* Αφαίρεση της αναφοράς του TR3 από το αρχείο των Windows
* Η επικοινωνία με τον προγραμματιστή μπορεί να είναι καλύτερη επιλογή για καλύτερη απόδοση
