## Τι είναι ένα αρχείο HTML0; ##

Το HTML0 είναι η εγγενής μορφή eBook που χρησιμοποιείται από τον **Book Designer** με τυπικό κώδικα HTML και CSS που περιγράφει το περιεχόμενο και τη σχεδίαση κάθε σελίδας. Το **Book Designer** είναι το πιο κοινό πρόγραμμα που προορίζεται για λειτουργία με αρχεία HTML0 και για αλλαγή εγγράφων κειμένου σε πολλές μορφές eBook. Η μορφή αρχείου HTML0 υποστηρίζει λογισμικό που μπορεί να εγκατασταθεί σε σύστημα Windows καθώς ανήκει στην κατηγορία αρχείων κειμένου. Τα αρχεία HTML0 αποτελούνται από κανονικό HTML και μπορούν να προβληθούν και να τροποποιηθούν χρησιμοποιώντας ένα πρόγραμμα επεξεργασίας κειμένου. Τα αρχεία HTML0 δεν μπορούν να συνδεθούν με διακριτά αρχεία CSS σε σύγκριση με τα συνηθισμένα αρχεία HTML.

* Ορισμένα αρχεία χρειάζονται δικαιώματα πρόσβασης για να τα ανοίξουν
