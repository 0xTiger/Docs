## Τι είναι ένα αρχείο POT;

Τα αρχεία με επέκταση .POT αντιπροσωπεύουν αρχεία προτύπων του Microsoft PowerPoint που έχουν δημιουργηθεί από εκδόσεις PowerPoint 97-2003. Τα αρχεία που δημιουργούνται με αυτές τις εκδόσεις του Microsoft PowerPoint είναι σε δυαδική μορφή σε σύγκριση με αυτά που δημιουργούνται σε μορφές αρχείων του Office OpenXML που χρησιμοποιούν τις υψηλότερες εκδόσεις του PowerPoint. Τα αρχεία, ως εκ τούτου, που δημιουργούνται μπορούν να χρησιμοποιηθούν για τη δημιουργία παρουσιάσεων που έχουν την ίδια διάταξη και άλλες ρυθμίσεις που απαιτούνται για εφαρμογή σε νέα αρχεία. Αυτές οι ρυθμίσεις μπορούν να περιλαμβάνουν στυλ, φόντο, παλέτα χρωμάτων, γραμματοσειρές και προεπιλογές. Τέτοια αρχεία δημιουργούνται για τη δημιουργία έτοιμα προς χρήση αρχείων προτύπων για επίσημη χρήση.

## Προδιαγραφές μορφής αρχείου ##

Οι προδιαγραφές μορφής αρχείου για τη μορφή αρχείου POT δεν είναι διαθέσιμες μεμονωμένα από τη Microsoft. Ωστόσο, η μορφή προτύπου είναι η ίδια με αυτή της δυαδικής μορφής αρχείου [PPT](/el/presentation/ppt/) και μπορεί να ανοίξει στο MS PowerPoint για σκοπούς επεξεργασίας. Ένα αρχείο POT αναγνωρίζεται από το αναγνωριστικό HEX ως εξής:

```
D0 CF 11 E0 A1 B1 1A E1 00 00 00 00
```

Οι τύποι MIME που σχετίζονται με τη μορφή αρχείου POT είναι:

* application/vnd.ms-powerpoint [επίσημο]
* εφαρμογή/mpowerpoint
* application/x-mpowerpoint
* εφαρμογή/powerpoint
* application/x-powerpoint
* application/x-dos_ms_powerpnt
* εφαρμογή/γλάστρα
* εφαρμογή/x-soffic
