## Τι είναι ένα αρχείο AHK;

Ένα αρχείο AHK είναι ένα αρχείο σεναρίου που δημιουργείται με εφαρμογή λογισμικού AutoHotkey που χρησιμοποιείται για την αυτοματοποίηση εργασιών στα Microsoft Windows. Περιέχει οδηγίες για την αυτοματοποίηση εργασιών με χρήση πλήκτρων συντόμευσης που μπορούν να χρησιμοποιηθούν για την εκτέλεση άμεσων ενεργειών. Το αρχείο εκτελείται από το AutoHotkey και όλες οι ενέργειες εκτελούνται διαδοχικά. Τα αρχεία AHK είναι αρκετά ισχυρά για να εκτελούν σύνθετες εργασίες χρησιμοποιώντας τα πλήκτρα πρόσβασης που ορίζονται χρησιμοποιώντας τα πλήκτρα πρόσβασης. Τα αρχεία AHK μπορούν να ανοίξουν με προγράμματα επεξεργασίας κειμένου όπως το Microsoft Notepad και το Notepad++.

## Μορφή αρχείου AHK

Τα αρχεία AHK αποθηκεύονται στο δίσκο ως αρχεία απλού κειμένου και περιέχουν γραμμές κώδικα που μπορούν να εκτελεστούν από το AutoHotkey. Το AutoHotkey είναι μια δωρεάν, ανοιχτού κώδικα γλώσσα δέσμης ενεργειών και επιτρέπει στους χρήστες να δημιουργούν απλά έως πολύπλοκα σενάρια για να εκτελέσουν ενέργειες όπως πλήρωση φορμών, αυτόματο κλικ, εκτέλεση μακροεντολών, κ.λπ. Ένα αρχείο AHK στο ελάχιστο μπορεί να εκτελέσει μία ενέργεια και στη συνέχεια να βγει .

Υπάρχουν διαθέσιμα εργαλεία που μπορούν να χρησιμοποιηθούν για τη μετατροπή του AHK σε αρχεία [Exe](/el/executable/exe/).

### Παράδειγμα AHK

Το παρακάτω παράδειγμα χρησιμοποιεί το κλειδί Win+Z για να εκτελέσει το Microsoft Notepad ή να το εμφανίσει, εάν εκτελείται ήδη.

```
#z::Run https://www.autohotkey.com  ; Win+Z

^!n::  ; Ctrl+Alt+N
if WinExist("Untitled - Notepad")
    WinActivate
else
    Run Notepad
return
```

## Πώς μπορώ να δημιουργήσω ένα αρχείο AHK;

Τα παρακάτω βήματα μπορούν να χρησιμοποιηθούν για τη δημιουργία ενός αρχείου AHK. Αυτά προϋποθέτουν ότι το AutoHotkey είναι ήδη εγκατεστημένο στο μηχάνημα.

* Κάντε δεξί κλικ στην επιφάνεια εργασίας σας.
* Βρείτε "Νέο" στο μενού.
* Κάντε κλικ στο "AutoHotkey Script" μέσα στο μενού "New".
* Δώστε στο σενάριο ένα νέο όνομα.
* Βρείτε το αρχείο που δημιουργήθηκε πρόσφατα στην επιφάνεια εργασίας σας και κάντε δεξί κλικ σε αυτό.
* Κάντε κλικ στο "Επεξεργασία σεναρίου".
* Θα έπρεπε να έχει εμφανιστεί ένα παράθυρο, πιθανώς το Σημειωματάριο.
* Αποθηκεύστε το Αρχείο.
