## Οι μορφές αρχείων ΛΔΚ
Οι επεκτάσεις ".prc" χρησιμοποιούνται για τη μορφή αρχείου Compact 3D της Αναπαράστασης Προϊόντος και τη μορφή αρχείου ηλεκτρονικού βιβλίου από την MobiPocket.

## Τι είναι το αρχείο Compact Representation Product (PRC);

Το PRC (Product Representation Compact) είναι μια μορφή αρχείου 3D που έχει βελτιστοποιηθεί για αποθήκευση, φόρτωση και εμφάνιση τρισδιάστατων δεδομένων που προέρχονται ειδικά από συστήματα CAD (Computer-Aided Design). Είναι ένα διαδοχικό δυαδικό αρχείο, γραμμένο με φορητό τρόπο. Η PRC μπορεί να χρησιμοποιηθεί για την ενσωμάτωση τρισδιάστατων δεδομένων σε αρχείο PDF. Η ΛΔΚ υποστηρίζει τις περισσότερες από τις κύριες κατασκευές του CAD, όπως συγκροτήματα και εξαρτήματα, δέντρα τρισδιάστατων οντοτήτων, ακριβής αναπαράσταση γεωμετρίας, τριγωνική αναπαράσταση και σήμανση. Η ΛΔΚ χρησιμοποιεί την επέκταση .prc και ο τύπος μέσων διαδικτύου που χρησιμοποιείται είναι *model/prc*.

Η PRC είναι μια μορφή αρχείου πολλαπλών χρήσεων που με βάση τη χρήση της μπορεί είτε να αποθηκευτεί χρησιμοποιώντας κανονική συμπίεση για να αναπαραστήσει άμεσα δεδομένα CAD είτε να αποθηκευτεί χρησιμοποιώντας υψηλή συμπίεση για μείωση του μεγέθους του αρχείου. Τα τρισδιάστατα δεδομένα που είναι αποθηκευμένα σε PDF χρησιμοποιώντας μορφή PRC είναι διαλειτουργικά με εφαρμογές CAM και CAE.

### Προδιαγραφή μορφής αρχείου Compact (PRC) για Αναπαράσταση προϊόντος

Ένα αρχείο PRC έχει μια κύρια ενότητα κεφαλίδας ακολουθούμενη από μία ή περισσότερες δομές αρχείων που ακολουθούνται από ένα αρχείο μοντέλου στο τέλος. Μια δομή αρχείου έχει μια κεφαλίδα ακολουθούμενη από τις ακόλουθες ενότητες δεδομένων:

- **Globals**: Περιέχει τις αναφερόμενες δομές και χρώματα αρχείων, στυλ γραμμής και συστήματα συντεταγμένων για κάθε οντότητα δέντρου της δομής αρχείου.
- **Δέντρο**: Περιέχει μια περιγραφή του δέντρου στοιχείων όπως εμφανίσεις προϊόντων, ορισμούς τμημάτων, στοιχεία αναπαράστασης και σήμανση.
- **Tessellation**: Περιέχει όλα τα tesselated (τριγωνικά) δεδομένα στις οντότητες φύλλων του δέντρου (στοιχεία αναπαράστασης και σημάνσεις).
- **Γεωμετρία**: Περιέχει όλα τα ακριβή δεδομένα γεωμετρίας και τοπολογίας των οντοτήτων φύλλων του δέντρου (στοιχεία αναπαράστασης).
- **Επιπλέον γεωμετρία**: Περιέχει τα συνοπτικά δεδομένα της γεωμετρίας. Αυτό επιτρέπει τη μερική φόρτωση του αρχείου χωρίς τη φόρτωση ολόκληρης της γεωμετρίας.

Ακολουθεί η δομή ενός τυπικού αρχείου ΛΔΚ.

```console
PRC Header

File Structure #1
- Header for File Structure #1
- Globals for File Structure #1
- Tree for File Structure #1
- Tessellation for File Structure #1
- Geometry for File Structure #1
- Extra Geometry for File Structure #1

File Structure #2
...

File Structure #n
- Header for File Structure #n
- Globals for File Structure #n
- Tree for File Structure #n
- Tessellation for File Structure #n
- Geometry for File Structure #n
- Extra Geometry for File Structure #n

PRC model file data
```
## Τι είναι ένα αρχείο ηλεκτρονικού βιβλίου MobiPocket με επέκταση PRC
Μια μορφή αρχείου ηλεκτρονικού βιβλίου που εισάγεται από μια γαλλική εταιρεία που ονομάζεται **Mobipocket** χρησιμοποιεί επίσης την επέκταση ".prc". Αρχικά, το Mobipocket κυκλοφόρησε μια δωρεάν εφαρμογή για πολλές έξυπνες συσκευές όπως, tablet pc, PDA και smartphone. Η επέκταση ".prc" είναι στην πραγματικότητα πανομοιότυπη με το .mobi, αλλά χρησιμοποιείται ιδιαίτερα για PDA που υποστηρίζουν μόνο επεκτάσεις **.prc** ή **.pdb**.

### Προδιαγραφές μορφής αρχείου Mobipocket PRC
Η μορφή αρχείου MobiPocket PRC βασίζεται στο πρότυπο Open eBook χρησιμοποιώντας XHTML και μπορεί επίσης να περιλαμβάνει JavaScript και πλαίσια. Διατίθεται επίσης υποστήριξη για εγγενή ερωτήματα SQL που θα χρησιμοποιηθούν με ενσωματωμένες βάσεις δεδομένων.

Το Mobipocket Reader διαθέτει βιβλιοθήκη αρχικής σελίδας. Οι αναγνώστες μπορούν να εισάγουν κενές σελίδες σε οποιοδήποτε μέρος ενός βιβλίου και να προσθέτουν μεταβλητά σχέδια. Τα αντικείμενα όπως οι σχολιασμοί, οι σελιδοδείκτες, οι διορθώσεις, οι σημειώσεις και τα σχέδια είναι διαχειρίσιμα από μία μόνο τοποθεσία. Οι εικόνες μετατρέπονται σε μορφή GIF και έχουν μέγιστο μέγεθος 64K που είναι αρκετό για κινητά τηλέφωνα με μικρές οθόνες. Το Mobipocket Reader διαθέτει τους ηλεκτρονικούς σελιδοδείκτες και ένα ενσωματωμένο λεξικό.

Ο αναγνώστης διαθέτει λειτουργία πλήρους οθόνης για ανάγνωση και υποστήριξη για πολλά PDA, Communicators και Smartphones. Τα προϊόντα Mobipocket υποστηρίζουν τα περισσότερα λειτουργικά συστήματα Palm, Windows, Symbian, BlackBerry, αλλά όχι το Android. Ο αναγνώστης λειτουργεί σε Linux ή Mac OS X χρησιμοποιώντας WINE.
