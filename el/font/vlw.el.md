## Τι είναι ένα αρχείο VLW;

Ένα αρχείο VLW είναι μια μορφή γραμματοσειράς που χρησιμοποιείται από το λογισμικό δημιουργίας εικόνων και κινούμενων εικόνων, **Επεξεργασία**. Αποθηκεύει διανυσματικές γραμματοσειρές με συμπαγή τρόπο που μπορεί να κλιμακωθεί σε οποιοδήποτε μέγεθος χωρίς απώλεια ποιότητας. Αυτό κάνει τη χρήση του σε γραφικά και οπτικό σχεδιασμό. Η επεξεργασία παρέχει τη δυνατότητα δημιουργίας εικόνων και κινούμενων εικόνων, όπου χρησιμοποιούνται αυτά τα γλυφά γραμματοσειράς VLW.

## Μορφή αρχείου VLW - Περισσότερες πληροφορίες

Τα αρχεία VLW αποθηκεύονται στο δίσκο ως δυαδικά αρχεία. Στην Επεξεργασία, μπορείτε να χρησιμοποιήσετε τη συνάρτηση **createFont()** για να δημιουργήσετε ένα αντικείμενο γραμματοσειράς από ένα αρχείο VLW. Στη συνέχεια, μπορείτε να χρησιμοποιήσετε αυτό το αντικείμενο γραμματοσειράς για να σχεδιάσετε κείμενο στα σκίτσα σας χρησιμοποιώντας τη συνάρτηση **text()**. Η επεξεργασία μπορεί να διαβάσει αρχεία VLW χρησιμοποιώντας την εντολή **loadFont()**.
