## Τι είναι ένα αρχείο JFPROJ;

Ένα αρχείο με επέκταση .jfproj είναι μια προσαρμοσμένη γραμματοσειρά που βασίζεται σε JSON που δημιουργήθηκε με την εφαρμογή Microsoft Font Maker στα Windows 10. Αυτά μπορούν να δημιουργηθούν χρησιμοποιώντας ηλεκτρονικό στυλό με συσκευές οθόνης αφής, όπως tablet, αν και άλλα περιφερειακά υπολογιστών όπως το ποντίκι μπορούν επίσης να χρησιμοποιηθούν σε περίπτωση των υπολογιστών. Τα αρχεία JFPROJ περιέχουν τους γλυφούς που χρησιμοποιούνται για την απόδοση των χαρακτήρων στην οθόνη ορίζοντας το διάστημα μεταξύ χαρακτήρων και λέξεων και το μέγεθος της γραμματοσειράς. Η εφαρμογή Microsoft Font Maker είναι διαθέσιμη για λήψη στο κατάστημα των Windows.
