## Σχετικά με το DOCX σε XPS

Η μετατροπή σελίδων DOCX σε μορφή αρχείου XPS (XML Paper Specifications) είναι μία από τις κοινώς χρησιμοποιούμενες λειτουργίες. Όταν ένα αρχείο DOCX μετατρέπεται σε XPS, όλες οι σελίδες του αποδίδονται σε ένα μόνο αρχείο εξόδου XPS. Τα Microsoft Windows παρέχουν προεπιλεγμένο πρόγραμμα προβολής XPS που μπορεί να χρησιμοποιηθεί για το άνοιγμα και την προβολή αρχείων XPS. Το DOCX μπορεί να μετατραπεί σε XPS χρησιμοποιώντας πολλά διαφορετικά διαδικτυακά εργαλεία. Η εφαρμογή Aspose.Words είναι μια τέτοια διαδικτυακή εφαρμογή που μπορεί να μετατρέψει το [DOCX](/el/word-processing/docx/) σε [XPS](/el/page-description-language/xps/) δωρεάν στο διαδίκτυο.
