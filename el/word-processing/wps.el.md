## Τι είναι ένα αρχείο WPS;

Ένα αρχείο με επέκταση .wps είναι ένα έγγραφο του Microsoft Works που χρησιμοποιήθηκε πριν από το 2006 ως έγγραφο του Word. Ήταν παρόμοιο με τη μορφή αρχείου [DOC](/el/word-processing/doc/) που είχε πρόσφατα εισαχθεί μέχρι τότε, αλλά δεν ήταν τόσο μεγάλο χαρακτηριστικό όσο η νέα μορφή αρχείου. Για παράδειγμα, δεν υποστήριζε λειτουργίες μακροεντολών και σύνθετες επιλογές μορφοποίησης. Τα αρχεία WPS μπορούν ακόμα να ανοίξουν με σύγχρονους επεξεργαστές Word, αλλά για τα Windows 10, πρέπει να εγκαταστήσετε έναν μετατροπέα για να φορτώσετε αυτά τα αρχεία. Το Microsoft Works αντικαταστάθηκε από τον επεξεργαστή εγγράφων Microsoft Word το 2010.

## Προδιαγραφές μορφής αρχείου

Οι προδιαγραφές μορφής αρχείου του εγγράφου WPS δεν είναι διαθέσιμες δημόσια, καθώς η Microsoft δεν άνοιξε τις προδιαγραφές στους τελικούς χρήστες. Τα WPS είναι δυαδικά αρχεία και μπορούν να ανοίξουν με φίλτρα μετατροπής μορφής αρχείου που έχουν δημιουργηθεί από τη Microsoft για επεξεργαστή Word. Τα LibreOffice, NeoOffice και OxygenOffice έχουν συμπεριλάβει μια γενική βιβλιοθήκη C++, το libwps, που μπορεί να εξάγει κείμενο από πολλές διαφορετικές εκδόσεις του Microsoft Works.
