## Σχετικά με τη μετατροπή DOC σε DOCM

Το DOCM είναι η έκδοση με δυνατότητα μακροεντολής των εγγράφων του Microsoft Word που είναι σε θέση να εκτελούν μακροεντολές. Η διαφορά μεταξύ ενός κανονικού εγγράφου του Word και ενός εγγράφου με δυνατότητα μακροεντολής είναι ότι το DOCM παρέχει την πρόσθετη δυνατότητα για υποστήριξη μακροεντολών.

Τα αρχεία DOC μπορούν να μετατραπούν σε DOCM χρησιμοποιώντας δωρεάν διαδικτυακά εργαλεία και εφαρμογές. **[Aspose.Words App](https://products.aspose.app/words/family)** είναι ένα σύνολο διαδικτυακών εφαρμογών που σας επιτρέπουν να πραγματοποιείτε διαφορετικές λειτουργίες σε έγγραφα του Word απευθείας από το πρόγραμμα περιήγησής σας στον ιστό. Εκτός από τις δυνατότητες του [document viewing](/el/word-processing/docx-viewer/), διαχωρισμού, συγχώνευσης, σύγκρισης, περιστροφής και άλλων, σας επιτρέπει να μετατρέπετε έγγραφα Word από μια μορφή σε άλλη. Μπορείτε να μετατρέψετε το [DOC](/el/word-processing/doc/) σε [DOCM](/el/word-processing/docm/) χρησιμοποιώντας την εφαρμογή Aspose.Words.

## Μετατροπή DOC σε DOCM - Δωρεάν επίδειξη

Η εφαρμογή Aspose.Words σάς επιτρέπει να μετατρέψετε το DOC σε DOCM δωρεάν online. Μπορείτε να αποκτήσετε πρόσβαση σε αυτήν τη μετατροπή DOC σε DOCM από το πρόγραμμα περιήγησής σας. Είναι γρήγορο και ακριβές, δεν απαιτεί κωδικοποίηση και είναι εντελώς δωρεάν. Επιπλέον, δεν χρειάζεται να εγκαταστήσετε κανένα πρόσθετο λογισμικό ή εργαλείο στον υπολογιστή σας για να πραγματοποιήσετε τη μετατροπή του DOC σε DOCM.

### Πώς να μετατρέψετε το DOC σε DOCM;

Μπορείτε να μετατρέψετε τη μορφή αρχείου DOC σε DOCM χρησιμοποιώντας την εφαρμογή [Aspose.Words DOC σε DOCM](https://products.aspose.app/words/conversion/doc-to-docm) ως εξής.

1. Μεταβείτε στην εφαρμογή Aspose.Words για [μετατροπέας DOC σε DOCM](https://products.aspose.app/words/conversion/doc-to-docm)
1. Ανεβάστε το αρχείο DOC κάνοντας κλικ στο κουμπί αποστολής
1. Μόλις τελειώσετε, πατήστε το κουμπί Μετατροπή. Αυτό θα ξεκινήσει τη διαδικασία μετατροπής του αρχείου DOC σε DOCM στο παρασκήνιο
1. Θα σας ζητηθεί να κάνετε λήψη του αρχείου DOCM που έχει μετατραπεί μόλις ολοκληρωθεί η διαδικασία μετατροπής DOC σε DOCM
