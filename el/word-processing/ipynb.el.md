## Τι είναι ένα αρχείο IPYNB;

Ένα αρχείο με επέκταση .ipynb είναι ένα αρχείο Jupyter Notebook που περιέχει όλα τα περιεχόμενα του Notebook που δημιουργήθηκε με την περίοδο σύνδεσης της εφαρμογής web Jupyter Notebook. Το [Jupyter Notebook](https://jupyter.org/) είναι μια διαδραστική διαδικτυακή εφαρμογή που χρησιμοποιείται για τη δημιουργία και την κοινή χρήση υπολογιστικών εγγράφων για ανάλυση χρησιμοποιώντας Python.

Ένα αρχείο IPYNB χρησιμεύει ως πλήρης υπολογιστική εγγραφή μιας συνεδρίας και περιλαμβάνει πληροφορίες όπως υπολογιστικές εισόδους και εξόδους, μαθηματικές συναρτήσεις και πλούσια αναπαράσταση αντικειμένων που προκύπτουν, όπως εικόνες και επεξηγηματικό κείμενο.

## Μορφή αρχείου IPYNB

Τα IPYNB αποθηκεύονται εσωτερικά σε μορφή αρχείου [JSON](/el/web/json/), η οποία είναι μια ανοιχτή τυπική μορφή αρχείου για κοινή χρήση δεδομένων. Για το λόγο αυτό, τα αρχεία IPYNB είναι αναγνώσιμα από τον άνθρωπο και εύκολα κατανοητά. Το JSON διευκολύνει επίσης την κοινή χρήση αυτών των αρχείων με άλλους.

## Πώς να μετατρέψετε αρχεία IPYNB;

Τα αρχεία IPYNB μπορούν να εξαχθούν σε πολλές διαφορετικές μορφές αρχείων χρησιμοποιώντας την εντολή nbconvert από τη διεπαφή Jupyter. Αυτό περιλαμβάνει τη μετατροπή του IPYNB σε:

* PDF
* HTML
* ReStructuredText
* LaTex

## Πώς μπορώ να προβάλω το αρχείο IPYNB;

Υπάρχουν δύο μέθοδοι για να προβάλετε ή να ανοίξετε αρχεία IPYNB.

1. Ο χρήστης μπορεί να ανεβάσει το έγγραφο του σημειωματάριου .ipynb σε κάποια δημόσια διεύθυνση URL και να το ανοίξει από το Jupyter Notebook Viewer<nbviewer> που αποδίδει το σημειωματάριο ως στατική ιστοσελίδα από τη διεύθυνση URL.
1. Κατεβάστε τα αρχεία και ανοίξτε τα στη δική τους παρουσία του Jupyter Notebook
