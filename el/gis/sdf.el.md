## Τι είναι ένα αρχείο SDF;

A Spatial Data File (SDF) is a Geodatabase file format that was developed by Autodesk. It stores geospatial data and is used by Autodesk GIS program MapGuide and AutoCAD Map 3D. SDF file format is based on the single file database file format SQLite3. Θεωρείται ως μια βελτιστοποιημένη μορφή αρχείου για μεγάλα σύνολα δεδομένων χωρικών πληροφοριών.

Μπορείτε να ανοίξετε ένα αρχείο SDF χρησιμοποιώντας το Autodesk AutoCAD Map 3D 2022 και το Autodesk AutoCAD Civil 3D 2023.

## Μορφή αρχείου SDF - Περισσότερες πληροφορίες

Η μορφή αρχείου SDF αποθηκεύεται σε δίσκο ως δυαδικά αρχεία. Βασίζεται στη βάση δεδομένων SQLite που χρησιμοποιεί στοιχεία αποθήκευσης χαμηλού επιπέδου για δυαδική σειριοποίηση δεδομένων. Ένα αρχείο SDF περιέχει πολλές κατηγορίες χαρακτηριστικών ανά αρχείο και πολλαπλές ιδιότητες γεωμετρίας για κάθε κατηγορία χαρακτηριστικών. Τα δεδομένα σε ένα αρχείο SDF είναι προσβάσιμα με υψηλή ταχύτητα λόγω της ευρετηρίασης κάθε ιδιότητας γεωμετρίας χρησιμοποιώντας ένα δέντρο R.
