## Τι είναι ένα αρχείο IMG;

Ένα αρχείο με επέκταση .img είναι ένα αρχείο χάρτη Garmin που χρησιμοποιείται από συσκευές GPS Garmin. Περιέχει χάρτες/γεωχωρικά δεδομένα για την αναπαράσταση οντοτήτων χαρτών όπως ονόματα δρόμων, διαδρομές πλοήγησης και σημεία ενδιαφέροντος. Οι χρήστες μπορούν να δημιουργήσουν προσαρμοσμένα δεδομένα χάρτη, να τα αποθηκεύσουν ως αρχείο .img και να τα φορτώσουν σε συσκευές Garmin. Το Garmin alos παρέχει προεγκατεστημένα δεδομένα χάρτη που αποθηκεύονται σε συσκευές Garmin σε μορφή αρχείου IMG. Τα αρχεία IMG μπορούν να ανοίξουν χρησιμοποιώντας εφαρμογές όπως Garmin Mobile PC, cGPSmapper, MapUpload και GPS Map Manager.

## Μορφή αρχείου IMG - Περισσότερες πληροφορίες

Τα αρχεία IMG αποθηκεύονται ως δυαδικά αρχεία σε ιδιόκτητη μορφή αρχείου Garmin. Οι πληροφορίες που είναι διαθέσιμες σε ένα αρχείο IMG μπορούν να αποδοθούν στις μονάδες GPS Garmin. Ορισμένες μονάδες GPS που υποστηρίζουν αυτόματη δρομολόγηση μπορούν επίσης να χρησιμοποιήσουν αυτά τα αρχεία χάρτη για καθοδήγηση κατά την πλοήγηση, όπως στο αυτοκίνητο, στο ποδήλατο ή κατά τη διάρκεια της πεζοπορίας.

Τα αρχεία GMAP είναι παρόμοια με μια εικόνα σκληρού δίσκου και βασίζονται σε ένα σύστημα αρχείων που αποτελείται από υποαρχεία για την παροχή λειτουργιών για τις μονάδες GPS Garmin. Τα υποαρχεία .GMP παρέχουν μια δομή υποκαταλόγου παρόλο που η μορφή αρχείου .gmap είναι επίπεδη.
