## Τι είναι ένα αρχείο DT0;

Ένα αρχείο DT0 είναι ένα αρχείο Digital Train Elevation Data (DTED) που περιέχει πληροφορίες επιπέδου 0 για τη μελέτη δεδομένων υψομέτρου εδάφους. Τα δεδομένα υψομέτρου χωρίζονται ομοιόμορφα σε ένα αρχείο DT0 και μπορούν να διαβαστούν από δημοφιλή προγράμματα GIS όπως τα ESRI ArcGIS Pro, TatukGIS, Blue Marble Geographic Global Mapper και Pitney Bowes MapInfo. Τα αρχεία DT0 χρησιμοποιούνται κυρίως από επιστημονικές και τεχνικές κοινότητες για τη μελέτη της κλίσης, του υψομέτρου και της τραχύτητας της επιφάνειας των παγκόσμιων εδαφών.

Η μορφή αρχείου DT0 είναι παρόμοια με τους τύπους αρχείων [DT1](/gis/dt1/) και [DT2](/gis/dt2/), αλλά με λιγότερη ευαισθησία.

### Μορφή αρχείου DT0

Τα αρχεία DT0 αποθηκεύονται ως δυαδικά αρχεία στο δίσκο. Αυτά μπορούν να διαβαστούν χρησιμοποιώντας πολλές εφαρμογές GIS. Τα αρχεία DT0 μπορούν να συγχωνευθούν σε έναν κατάλογο ράστερ της βάσης δεδομένων αρχείων που μπορεί να χρησιμοποιηθεί για τη δημιουργία ενός μεγάλου μωσαϊκού.

## Πώς να μετατρέψετε αρχεία Raster σε DTED;

Υπάρχουν πολλά εργαλεία που μπορούν να μετατρέψουν αρχεία Raster σε DTED. Το [ArgGIS Pro](https://pro.arcgis.com/en/pro-app/latest/tool-reference/data-management/raster-to-dted.htm) είναι ένα τέτοιο εργαλείο που μπορεί να μετατρέψει το Raster σε DTED.
