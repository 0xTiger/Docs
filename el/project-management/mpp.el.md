## Τι είναι ένα αρχείο MPP;

Ένα αρχείο με επέκταση .mpp είναι ένα αρχείο δεδομένων του Microsoft Project που αποθηκεύει πληροφορίες που σχετίζονται με τη διαχείριση έργου με ολοκληρωμένο τρόπο. Η Microsoft έχει αναπτύξει αυτήν την ιδιόκτητη μορφή αρχείου για να την κάνει συμβατή με το Microsoft Project (MSP), το προϊόν λογισμικού διαχείρισης έργων της. Εκτός από το MPP, το MSP μπορεί να λειτουργήσει με σχήμα XML έργου. Πολλές εφαρμογές και API επιτρέπουν τη μετατροπή MPP σε άλλες μορφές αρχείων. Η Microsoft διαθέτει πλέον έναν διαδικτυακό διακομιστή έργου, όπου μπορούν να μεταφορτωθούν αρχεία διαχείρισης έργων για συνεργασία από πολλούς χρήστες.

## Μορφή αρχείου MPP ##

Η Microsoft ανέπτυξε και δημοσίευσε το MPP ως δυαδική μορφή αρχείου και οι προδιαγραφές του δεν έχουν δημοσιοποιηθεί από τη Microsoft μέχρι σήμερα. Η εσωτερική δομή και οι λεπτομέρειες του αρχείου, επομένως, δεν είναι διαθέσιμα για προγραμματιστές. Υπάρχουν πολλά API διαθέσιμα στο διαδίκτυο που υποστηρίζουν την ανάγνωση/εγγραφή σε αρχεία MPP, αλλά ποτέ δεν επιτυγχάνεται 100% ακρίβεια. Η Microsoft έχει εισαγάγει αρκετές δυνατότητες στην εφαρμογή διαχείρισης έργων της με την πάροδο του χρόνου, όπως και οι δυνατότητες της μορφής αρχείου MPP.

Το αρχείο MPP χρησιμοποιεί τους ακόλουθους τύπους MIME:

* application/vnd.ms-project
* εφαρμογή/msproj
* εφαρμογή/msproject
* application/x-msproject
* application/x-ms-project
* application/x-dos_ms_project
* εφαρμογή/mpp
* zz-application/zz-winassoc-mpp

## Προβλήματα στο άνοιγμα ενός αρχείου MPP; ##

Ακολουθεί η λίστα με ορισμένα κοινά ζητήματα που μπορεί να προκύψουν και να προκαλέσουν δυσλειτουργία της μορφής MPP:

* Απουσία υποστηρικτικού λογισμικού
* Κατεστραμμένο αρχείο
* Μολυσμένο αρχείο λόγω ιού
* Δεν υπάρχει δικαίωμα πρόσβασης στο σύστημα για το άνοιγμα των αρχείων
* Ξεπερασμένη μονάδα δίσκου στο σύστημά σας
* Η επέκταση του αρχείου μετονομάζεται
