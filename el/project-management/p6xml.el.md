## Τι είναι ένα αρχείο P6XML; ##

Η επέκταση αρχείου P6XML έχει εξαιρετική αναγνωσιμότητα, προσαρμοστικότητα και ευρεία χρήση που την καθιστούν την ιδανική μορφή. Το Primavera P6 Professional έχει τη δυνατότητα εξαγωγής και εισαγωγής αρχείων [XER](/el/project-management/xer) και αρχείων XML. Το XML σημαίνει **Extensible Markup Language**. Αυτή η γενική μορφή επιτρέπει στους χρήστες να μοιράζονται πληροφορίες έργου μεταξύ βάσεων δεδομένων Primavera P6. Τα δεδομένα του έργου μπορούν εύκολα να αποθηκευτούν σε ένα αρχείο κειμένου μέσω του αρχείου XML. Αυτό κάνει τα πράγματα πιο απλά για πολλά προγράμματα λογισμικού να ανταλλάσσουν τα δεδομένα. Τέτοια δεδομένα μπορούν εύκολα να κοινοποιηθούν και να διαχειρίζονται από τον χρήστη Primavera P6.

## Πλεονεκτήματα μιας μορφής αρχείου P6XML ##

* Είναι εύκολο για τους χρήστες του Primavera να ορίσουν ποιες γραμμές βάσης εξάγονται και εισάγονται χρησιμοποιώντας το αρχείο XML
* Η εξαγωγή XML μπορεί να περιέχει όλα τα λεπτομερή σχέδια δράσης του Έργου
* Κάθε χρήστης που χρησιμοποιεί το Primavera 6 που δεν μπορεί να επεξεργαστεί, να προσθέσει ή να διαγράψει ορισμένα καθολικά στοιχεία δεδομένων, όπως Κωδικοί δραστηριότητας, Ημερολόγια και Πόροι, δεν θα μπορεί να τα εισαγάγει
* Εκτός από τις ρυθμίσεις Γενικού και Προφίλ Ασφαλείας Έργου που χρησιμοποιούνται από την Primavera, τα στοιχεία δεδομένων που θα επηρεάσουν ή θα αλλάξουν τα δεδομένα στη βάση δεδομένων δεν θα εισαχθούν

## Εξαγωγή του αρχείου XML από το Primavera P6 ##

Για εξαγωγή έργων P6 Professional και δεδομένων έργου σε αρχείο P6XML, χρησιμοποιήστε το Oracle Primavera Cloud. Μπορείτε να κάνετε λήψη του αρχείου όταν διαμορφώνεται από την εφαρμογή επιτραπέζιου υπολογιστή P6 Professional και να το εισαγάγετε στο Primavera Cloud. Πριν από την αδιάλειπτη λειτουργία της διαδικασίας εξαγωγής, θα πρέπει να ρυθμίσετε εάν είστε συνδεδεμένοι με μια βάση δεδομένων P6 Professional ή μια P6 EPPM. Ορισμένες δυνατότητες της διαδικασίας εξαγωγής ενδέχεται να διαφέρουν ανάλογα με τη βάση δεδομένων σας. Έχοντας P6 Specialized ρυθμίσεις με βάση δεδομένων P6 Proficient, η διαδικασία εξαγωγής λειτουργεί τοπικά στο σύστημά σας. Έχοντας μια βάση δεδομένων P6 Professional, μπορείτε εύκολα να εξαγάγετε και να ανοίξετε το αρχείο P6 XML από μια ρύθμιση στο σύστημά σας.

Ακολουθούν ορισμένα βήματα που περιγράφουν τη διαδικασία εξαγωγής:

* Ανοίξτε τις επιθυμίες Primavera P6 project
* Μετά από αυτό, επιλέξτε Εξαγωγή στην καρτέλα Αρχείο
* Επιλέξτε "Primavera P6 – (XML)". Μετά από αυτό, κάντε κλικ στο Επόμενο
* Κάντε διπλό κλικ στο έργο που θα εξαγάγετε
* Επιλέξτε τις περικοπές για τη διαδρομή όπου θέλετε να τοποθετήσετε το εξαγόμενο αρχείο και επιλέξτε "Τέλος".
* Επιλέξτε πώς θέλετε να αποθηκεύσετε το αρχείο
* Εάν δεν χρειάζονται διατάξεις σε επίπεδο έργου με το αρχείο εξαγωγής, είναι απαραίτητο να διαγράψετε την επιλογή "Εξαγωγή όλων των διατάξεων σε επίπεδο έργου".
* Επιλέξτε finish και, στη συνέχεια, θα λάβετε μια επιβεβαίωση της εξαγωγής XML

### Σημαντικές συμβουλές ###

* Κάθε φορά που συνδέεστε σε μια βάση δεδομένων P6 EPPM, ο διαχειριστής πρέπει να μεταβιβάζει τη διεύθυνση URL της πηγής περιεχομένου στο πεδίο URL P6 στη Γενική σελίδα των ρυθμίσεων εφαρμογής στο P6
* Είναι βολικό να εξάγετε πολλά έργα σε ένα αρχείο XML
* Εξάγονται συνολικά δεδομένα που διατίθενται στο έργο
* Είναι βολικό να εξάγετε αρχεία XML χωρίς να έχετε πρόσβαση σε όλους τους πόρους
  


## Προβλήματα στο άνοιγμα ενός αρχείου P6XML; ##

Μερικά κοινά ζητήματα που μπορεί να προκύψουν και να προκαλέσουν δυσλειτουργία της μορφής P6XML είναι τα εξής:

* Απουσία υποστηρικτικού λογισμικού
* Κατεστραμμένο αρχείο
* Μολυσμένο αρχείο λόγω ιού
* Η περιγραφή P6XML διαγράφηκε κατά λάθος από το μητρώο των Windows
* Δεν υπάρχει δικαίωμα πρόσβασης στο σύστημα για το άνοιγμα των αρχείων
* Ξεπερασμένη μονάδα δίσκου στο σύστημά σας
* Η επέκταση του αρχείου μετονομάζεται
