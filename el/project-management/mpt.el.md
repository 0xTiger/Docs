## Τι είναι ένα αρχείο MPT; ##

Ένα αρχείο με επέκταση .mpt είναι τα αρχεία προτύπων [Microsoft Project](https://products.office.com/en-us/project/project-and-portfolio-management-software). Αυτό περιέχει βασικές πληροφορίες και ρυθμίσεις δομής και εγγράφου για τη δημιουργία αρχείων [MPP](/el/project-management/mpp/). Αυτά τα πρότυπα παρέχουν προεπιλεγμένες ρυθμίσεις που είναι απαραίτητες για ένα συγκεκριμένο έργο, όπως χρονοδιαγράμματα ή πληροφορίες προϋπολογισμού. Ωστόσο, δεν μπορεί να αποθηκεύσει δεδομένα για συγκεκριμένα έργα, όπως εργασίες, πόρους ή αναθέσεις. Μόλις τροποποιηθούν, τα αρχεία προτύπου μπορούν να αποθηκευτούν ως τυπικά αρχεία MPP για περαιτέρω εργασία με αυτά. Ένα κύριο αρχείο προτύπου, με τίτλο "Global.mpt" είναι διαθέσιμο στο Microsoft Project, το οποίο ανοίγει για τη δημιουργία ενός μεμονωμένου προτύπου. Τα αρχεία MPT χρησιμοποιούν *application/vnd.ms-project* και *application/x-project* ως τυπικούς τύπους MIME. Τα Windows, το Linux και το macOS μπορούν όλα να ανοίξουν αρχεία MPT, μαζί με το Moos Project Viewer, το Microsoft Project και το MPXJ.

## Προβλήματα στο άνοιγμα ενός αρχείου MPT; ##

Ακολουθεί η λίστα με ορισμένα κοινά ζητήματα που μπορεί να προκύψουν και να προκαλέσουν δυσλειτουργία της μορφής MPT:

* Απουσία υποστηρικτικού λογισμικού
* Κατεστραμμένο αρχείο
* Μολυσμένο αρχείο λόγω ιού
* Δεν υπάρχει δικαίωμα πρόσβασης στο σύστημα για το άνοιγμα των αρχείων
* Ξεπερασμένη μονάδα δίσκου στο σύστημά σας
