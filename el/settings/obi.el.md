## Τι είναι ένα αρχείο OBI;

Το αρχείο OBI είναι το Outlook RSS Subscription File που επιτρέπει στους χρήστες να εγγραφούν σε μια ροή RSS στο Microsoft Outlook. Το RSS σημαίνει Really Simple Syndication, το οποίο είναι μια μορφή ροής ιστού που χρησιμοποιείται για τη δημοσίευση περιεχομένου που ενημερώνεται συχνά, όπως αναρτήσεις ιστολογίου, τίτλους ειδήσεων ή podcast.

Για να εγγραφούν στη ροή RSS στο Outlook, οι χρήστες μπορούν να κάνουν κλικ στον σύνδεσμο προς τη ροή RSS σε έναν ιστότοπο ή να αντιγράψουν τη διεύθυνση URL της ροής RSS και να την επικολλήσουν στο παράθυρο διαλόγου "Προσθήκη νέας ροής RSS" στο Outlook. Στη συνέχεια, το Outlook θα ελέγχει περιοδικά τη ροή RSS για νέο περιεχόμενο και θα το εμφανίζει στο φάκελο RSS Feed του χρήστη.

Τα αρχεία συνδρομής RSS του Outlook έχουν την επέκταση αρχείου ".rss". Αυτά τα αρχεία περιέχουν τη διεύθυνση URL της ροής RSS και μπορούν να χρησιμοποιηθούν για κοινή χρήση συνδρομών ροής RSS με άλλους ή για δημιουργία αντιγράφων ασφαλείας και επαναφορά συνδρομών ροής RSS.

## Μορφή αρχείου OBI - Περισσότερες πληροφορίες

Τα αρχεία OBI είναι γνωστά ως αρχείο OPML (Outline Processor Markup Language) και περιέχουν λίστα διευθύνσεων URL ροής RSS που έχουν εγγραφεί στο Microsoft Outlook.

Όταν ο χρήστης εξάγει τις ροές RSS του στο αρχείο OBI, περιέχει λίστα με όλες τις ροές RSS στις οποίες έχουν εγγραφεί, συμπεριλαμβανομένου του τίτλου κάθε ροής, της διεύθυνσης URL της ροής και οποιασδήποτε άλλης σχετικής πληροφορίας. Αυτό επιτρέπει στους χρήστες να μεταφέρουν εύκολα τις συνδρομές τους στη ροή RSS σε άλλο πρόγραμμα ανάγνωσης RSS ή να δημιουργούν αντίγραφα ασφαλείας των συνδρομών τους για φύλαξη.

Τα αρχεία OBI μπορούν επίσης να χρησιμοποιηθούν για την εισαγωγή συνδρομών ροής RSS στο Microsoft Outlook ή σε άλλο πρόγραμμα ανάγνωσης RSS. Για να εισαγάγετε αρχείο OBI στο Outlook, οι χρήστες μπορούν να κάνουν κλικ στο «Αρχείο» > «Άνοιγμα και εξαγωγή» > «Εισαγωγή/Εξαγωγή» > «Εισαγωγή από άλλο πρόγραμμα ή αρχείο» > «Επόμενο» > «Εισαγωγή ροών RSS από αρχείο OPML» και Στη συνέχεια, περιηγηθείτε στη θέση του αρχείου OBI στον υπολογιστή τους.

## Ποια είναι η μορφή του αρχείου OBI;

Το αρχείο συνδρομής RSS του Outlook γνωστό και ως αρχείο OBI χρησιμοποιεί XML (eXtensible Markup Language) για να καθορίσει τη δομή και το περιεχόμενο του αρχείου.

Τα αρχεία OBI αποτελούνται από σειρές ένθετων στοιχείων το καθένα με το δικό του σύνολο χαρακτηριστικών και τιμών. Το κύριο στοιχείο στο αρχείο OBI είναι το στοιχείο "περίγραμμα" που περιέχει πληροφορίες σχετικά με τη ροή RSS, συμπεριλαμβανομένου του τίτλου της ροής, της διεύθυνσης URL της ροής και οποιασδήποτε άλλης σχετικής πληροφορίας.

Κάθε στοιχείο "διάρθρωσης" μπορεί επίσης να περιέχει θυγατρικά στοιχεία που αντιπροσωπεύουν υποφακέλους ή υποκατηγορίες στη λίστα ροών RSS. Η δομή του αρχείου OBI επιτρέπει την εύκολη οργάνωση και διαχείριση των συνδρομών ροών RSS και μπορεί να χρησιμοποιηθεί για τη μεταφορά συνδρομών μεταξύ διαφορετικών προγραμμάτων ανάγνωσης RSS ή για τη δημιουργία αντιγράφων ασφαλείας και την επαναφορά συνδρομών.
