## Τι είναι ένα αρχείο CFG;

Ένα αρχείο διαμόρφωσης Celestia είναι αρχείο απλού κειμένου που χρησιμοποιείται από το Celestia, πρόγραμμα προσομοίωσης 3D σύμπαντος. Αυτά τα αρχεία είναι ζωτικής σημασίας για την προσαρμογή του τρόπου συμπεριφοράς του Celestia, των ουράνιων αντικειμένων που εμφανίζονται και του τρόπου εμφάνισης του προγράμματος. Ωστόσο, η επεξεργασία αυτών των αρχείων απαιτεί ιδιαίτερη προσοχή, επειδή οι ακατάλληλες αλλαγές μπορούν να διαταράξουν τη διαδικασία φόρτωσης του Celestia, εμποδίζοντάς το να εκτελεστεί σωστά.

## Celestia

Το Celestia είναι δωρεάν λογισμικό προσομοίωσης τρισδιάστατης αστρονομίας ανοιχτού κώδικα που επιτρέπει στους χρήστες να εξερευνούν και να οπτικοποιούν το σύμπαν με εξαιρετικά ρεαλιστικό και διαδραστικό τρόπο. Αναπτύχθηκε από τον Chris Laurel και κυκλοφόρησε για πρώτη φορά στις αρχές της δεκαετίας του 2000. Το Celestia είναι διαθέσιμο για λειτουργικά συστήματα Windows, macOS και Linux.

Τα βασικά χαρακτηριστικά και πτυχές του Celestia περιλαμβάνουν:

- **Ρεαλιστική τρισδιάστατη απεικόνιση:** Η Celestia παρέχει λεπτομερή και ακριβή αναπαράσταση του ηλιακού μας συστήματος, καθώς και των αστεριών, των γαλαξιών και άλλων ουράνιων αντικειμένων. Χρησιμοποιεί υψηλής ποιότητας τρισδιάστατα μοντέλα και υφές για να δημιουργήσει οπτικά καθηλωτική εμπειρία.

- **Εκτεταμένη Ουράνια Βάση Δεδομένων:** Το λογισμικό συνοδεύεται από τεράστια ενσωματωμένη βάση δεδομένων γνωστών ουράνιων αντικειμένων, όπως αστέρια, πλανήτες, φεγγάρια, αστεροειδείς, κομήτες και διαστημόπλοια. Οι χρήστες μπορούν εύκολα να εντοπίσουν και να εξερευνήσουν αυτά τα αντικείμενα.

- **Επιστημονική Ακρίβεια:** Ενώ το Celestia είναι εργαλείο οπτικοποίησης, στοχεύει να αναπαραστήσει ουράνια αντικείμενα και φαινόμενα όσο το δυνατόν ακριβέστερα, με βάση τα διαθέσιμα επιστημονικά δεδομένα.

## Μορφές αρχείων που χρησιμοποιούνται από την Celestia

Η Celestia χρησιμοποιεί διάφορες μορφές αρχείων για διαφορετικές πτυχές της τρισδιάστατης προσομοίωσης αστρονομίας της. Ακολουθούν ορισμένες βασικές μορφές αρχείων που χρησιμοποιεί η Celestia:

1. **Αρχεία διαμόρφωσης (.cel)**
- *Περιγραφή*: Αρχεία απλού κειμένου που επιτρέπουν στους χρήστες να προσαρμόζουν τη συμπεριφορά, την εμφάνιση και το περιεχόμενο της Celestia.
- *Σκοπός*: Προσαρμογή ρυθμίσεων, καθορισμός τοποθεσιών και καθορισμός ουράνιων αντικειμένων.

2. **Τρισδιάστατα μοντέλα και πλέγματα**
- *Μορφές*: [.3ds](/el/3d/3ds/), [.obj](/el/3d/obj/), [.dae](/el/3d/dae/), .ac
- *Περιγραφή*: Υποστηριζόμενες μορφές αρχείων τρισδιάστατων μοντέλων που χρησιμοποιούνται για την απόδοση ουράνιων σωμάτων και διαστημόπλοιων.
- *Σκοπός*: Εμφάνιση τρισδιάστατων αντικειμένων εντός Celestia.

3. **Αρχεία υφής**
- *Μορφές*: [.jpg](/el/image/jpeg/), [.png](/el/image/png/), [.dds](/el/image/dds/)
- *Περιγραφή*: Αυτά τα αρχεία παρέχουν επιφανειακές υφές για ουράνια σώματα.
- *Σκοπός*: Χαρτογράφηση υφών σε τρισδιάστατα μοντέλα για ρεαλιστική εμφάνιση.

4. **Κατάλογοι αστεριών και αρχεία δεδομένων**
- *Μορφές*: Προσαρμοσμένες μορφές, [.csv](/el/spreadsheet/csv/), [.tsv](/el/spreadsheet/tsv/)
- *Περιγραφή*: Αρχεία δεδομένων που χρησιμοποιούνται για την αναπαράσταση αστεριών και άλλων ουράνιων αντικειμένων, εξασφαλίζοντας ακριβή προσομοίωση.
- *Σκοπός*: Ακριβής αναπαράσταση ουράνιων αντικειμένων.

5. **Χάρτες κύβων υφής**
- *Περιγραφή*: Οι χάρτες κύβων χρησιμοποιούνται για την προσομοίωση της εμφάνισης μακρινών ουράνιων αντικειμένων όπως οι γαλαξίες.
- *Σκοπός*: Απόδοση μακρινών αντικειμένων με ρεαλιστικές υφές.

6. **Αρχεία σεναρίου**
- *Περιγραφή*: Αυτά τα αρχεία περιέχουν προσαρμοσμένα σενάρια γραμμένα στη γλώσσα σεναρίων της Celestia.
- *Σκοπός*: Δυνατότητα στους χρήστες να δημιουργούν δυναμικά συμβάντα και κινούμενα σχέδια στο σύμπαν της Celestia.

## Αρχείο ρυθμίσεων Celestia

Ακολουθεί η βασική επισκόπηση του τι μπορείτε να κάνετε με το αρχείο διαμόρφωσης Celestia:

1. **Ρύθμιση της τοποθεσίας σας**: Μπορείτε να καθορίσετε την τοποθεσία σας στη Γη χρησιμοποιώντας παραμέτρους γεωγραφικού πλάτους, γεωγραφικού μήκους και υψομέτρου. Αυτό επιτρέπει στην Celestia να εμφανίζει με ακρίβεια τον νυχτερινό ουρανό από τη θέση σας.

```
Location "My Location"
{
    Latitude 40.7128
    Longitude -74.0060
    Altitude 0
}
```

2. **Προσαρμογή των επιλογών προβολής:** Μπορείτε να προσαρμόσετε διάφορες επιλογές προβολής, όπως το οπτικό πεδίο, τις ρυθμίσεις ώρας και τις προτιμήσεις απόδοσης.

```
Viewpoint
{
    Location "My Location"
    Follow "Earth"
    Eye [0.0 0.0 0.0]
    Center [0.0 0.0 0.0]
    Up [0.0 1.0 0.0]
    Fov 45
}

Time
{
    Date "2023-09-25T12:00:00.000Z"
    Clock "Now"
}

Rendering
{
    Atmosphere false
    Stars 7
    Planetshine 0.25
}

```

3. **Φόρτωση ουράνιων αντικειμένων:** Μπορείτε να προσθέσετε ουράνια αντικείμενα όπως αστέρια, πλανήτες, αστεροειδείς, διαστημόπλοια και άλλα στην προσομοίωση σας. Κάθε αντικείμενο ορίζεται σε αρχείο ρυθμίσεων με τις ιδιότητές του.

```
Star "Sun"
{
    RA 0
    Dec 0
    Distance 0
    AppMag -26.74
    SpectralType "G2V"
}

Planet "Earth"
{
    Parent "Sol"
    Texture "earth.jpg"
    Radius 6371
    EllipticalOrbit
    {
        Period 365.25
        SemiMajorAxis 149.6e6
        Eccentricity 0.017
        Inclination 0
        AscendingNode 0
        ArgOfPericenter 102.94
        MeanAnomaly 100.464
}
}
```

4. **Ορισμός διαστημόπλοιων:** Μπορείτε να δημιουργήσετε το δικό σας φανταστικό διαστημόπλοιο ή να χρησιμοποιήσετε αληθινά, προσδιορίζοντας τις παραμέτρους τους, όπως τη θέση, τον προσανατολισμό και τα τρισδιάστατα μοντέλα.

```
Spacecraft "Voyager 1"
{
    Parent "Sol"
    Class "spacecraft"
    Mesh "voyager.3ds"
    Radius 0.01
    EllipticalOrbit
    {
        Period 30700
        SemiMajorAxis 1.08e11
        Eccentricity 0.044
        Inclination 3.4
        AscendingNode 49.0
        ArgOfPericenter 44.0
        MeanAnomaly 35.0
}
}
```

5. **Δημιουργία σεναρίων:** Μπορείτε να γράψετε σενάρια στην προσαρμοσμένη γλώσσα σεναρίων της Celestia για να δημιουργήσετε δυναμικά συμβάντα και κινούμενα σχέδια.
