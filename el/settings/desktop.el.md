## Τι είναι ένα αρχείο DESKTOP;

Ένα αρχείο .desktop είναι ένα αρχείο διαμόρφωσης που χρησιμοποιείται από περιβάλλοντα επιτραπέζιου υπολογιστή Linux για τον καθορισμό συντομεύσεων και εκκινητήρων εφαρμογών. Παρέχει μεταδεδομένα σχετικά με μια εφαρμογή, όπως το όνομα, το εικονίδιο, την εντολή για εκτέλεση και άλλες ιδιότητες. Αυτά τα αρχεία χρησιμοποιούνται συνήθως για τη δημιουργία συντομεύσεων σε μενού εφαρμογών, εκκινητές επιτραπέζιου υπολογιστή ή πίνακες σε συστήματα που βασίζονται σε Linux.

## Τι περιέχει το αρχείο DESKTOP;

Ένα αρχείο .desktop ακολουθεί συγκεκριμένη μορφή και περιέχει πολλά βασικά πεδία:

- **[Εισαγωγή στην επιφάνεια εργασίας]:** Αυτή είναι η κύρια κεφαλίδα ενότητας για το αρχείο .desktop.
- **Όνομα:** Καθορίζει το όνομα της εφαρμογής.
- **Σχόλιο:** Παρέχει σύντομη περιγραφή ή σχόλιο σχετικά με την εφαρμογή.
- **Exec:** Καθορίζει την εντολή που θα εκτελεστεί κατά την εκκίνηση της εφαρμογής.
- **Εικονίδιο:** Καθορίζει τη διαδρομή προς το αρχείο εικονιδίων που σχετίζεται με την εφαρμογή.
- **Terminal:** Καθορίζει εάν η εφαρμογή πρέπει να εκτελείται σε παράθυρο τερματικού.
- **Τύπος:** Καθορίζει τον τύπο καταχώρισης όπως "Εφαρμογή" ή "Σύνδεσμος".
- **Κατηγορίες:** Καθορίζει κατηγορίες ή ομάδες κάτω από τις οποίες θα πρέπει να εμφανίζεται η εφαρμογή στο μενού.
- **StartupNotify:** Καθορίζει εάν το περιβάλλον επιφάνειας εργασίας πρέπει να εμφανίζει μια ειδοποίηση εκκίνησης για την εφαρμογή.
- **NoDisplay:** Καθορίζει εάν η εφαρμογή θα πρέπει να είναι κρυφή από τα μενού.
- **Ενέργειες:** Καθορίζει πρόσθετες ενέργειες που μπορούν να εκτελεστούν στην εφαρμογή, όπως το άνοιγμα ενός συγκεκριμένου αρχείου.

## Παράδειγμα αρχείου DESKTOP

Ακολουθεί ένα παράδειγμα αρχείου .desktop για ένα εικονικό πρόγραμμα επεξεργασίας κειμένου που ονομάζεται "MyTextEditor":

```
[Desktop Entry]
Name=MyTextEditor
Comment=A simple text editor
Exec=mytexteditor %F
Icon=/path/to/icon.png
Terminal=false
Type=Application
Categories=TextEditor;Utility;
StartupNotify=true
NoDisplay=false
Actions=OpenNewWindow;OpenExistingFile;

[Desktop Action OpenNewWindow]
Name=Open New Window
Exec=mytexteditor

[Desktop Action OpenExistingFile]
Name=Open Existing File
Exec=mytexteditor %U
```

Σε αυτό το παράδειγμα, το αρχείο .desktop ορίζει την εφαρμογή "MyTextEditor" με τις σχετικές ιδιότητες. Περιλαμβάνει επίσης δύο πρόσθετες ενέργειες, "Άνοιγμα νέου παραθύρου" και "Άνοιγμα υπάρχοντος αρχείου", στις οποίες μπορείτε να έχετε πρόσβαση από το μενού περιβάλλοντος της εφαρμογής εκκίνησης.

Τοποθετώντας ένα αρχείο .desktop σε συγκεκριμένους καταλόγους όπως "/usr/share/applications" ή "~/.local/share/applications", το περιβάλλον επιφάνειας εργασίας θα το αναγνωρίσει και θα εμφανίσει την εφαρμογή ανάλογα στα μενού ή θα επιτρέψει την εκκίνηση από επιφάνεια εργασίας.

## Πώς να ανοίξετε το αρχείο DESKTOP;

Πολλά προγράμματα λογισμικού μπορούν να ανοίγουν και να χειρίζονται αρχεία .desktop. Αυτά τα προγράμματα είναι συνήθως διαχειριστές αρχείων ή περιβάλλοντα επιφάνειας εργασίας σε συστήματα που βασίζονται σε Linux. Να μερικά παραδείγματα:

- **Nautilus (Files):** Ο προεπιλεγμένος διαχειριστής αρχείων για το περιβάλλον επιφάνειας εργασίας GNOME.
- **Nemo:** Ο διαχειριστής αρχείων για το περιβάλλον επιφάνειας εργασίας Cinnamon.
- **Dolphin:** Ο προεπιλεγμένος διαχειριστής αρχείων για το περιβάλλον επιφάνειας εργασίας KDE Plasma.
- **Thunar:** Ο προεπιλεγμένος διαχειριστής αρχείων για το περιβάλλον επιφάνειας εργασίας Xfce.
- **Επεξεργαστής μενού KDE:** Ένα εργαλείο ειδικό για το περιβάλλον επιφάνειας εργασίας του KDE Plasma που σας επιτρέπει να προβάλλετε και να επεξεργάζεστε αρχεία .desktop.

Αυτοί οι διαχειριστές αρχείων και τα περιβάλλοντα επιφάνειας εργασίας παρέχουν γραφική διεπαφή για τη διαχείριση αρχείων .desktop. Σας επιτρέπουν να προβάλλετε και να επεξεργάζεστε ιδιότητες αρχείων .desktop, να δημιουργείτε προγράμματα εκκίνησης εφαρμογών και να οργανώνετε συντομεύσεις στα μενού εφαρμογών ή στην επιφάνεια εργασίας.

Τα αρχεία .desktop είναι αρχεία απλού κειμένου, επομένως μπορείτε επίσης να τα ανοίξετε και να τα επεξεργαστείτε με ένα πρόγραμμα επεξεργασίας κειμένου της επιλογής σας. Απλώς κάντε δεξί κλικ στο αρχείο .desktop και επιλέξτε "Άνοιγμα με" ή "Άνοιγμα με άλλη εφαρμογή" για να επιλέξετε ένα πρόγραμμα επεξεργασίας κειμένου από τη λίστα των εγκατεστημένων προγραμμάτων.

## Ποια είναι η μορφή του αρχείου DESKTOP;

Η μορφή αρχείου .desktop ακολουθεί συγκεκριμένη δομή και μορφή. Είναι ένα αρχείο απλού κειμένου με ένα σύνολο ζευγών κλειδιών-τιμών οργανωμένων σε ενότητες. Ακολουθεί μια επισκόπηση της μορφής:

- **Κεφαλίδες Ενοτήτων:** Κάθε ενότητα ξεκινά με μια κεφαλίδα που περικλείεται σε αγκύλες ([]). Η κύρια ενότητα ονομάζεται τυπικά [Εισαγωγή επιφάνειας εργασίας], η οποία περιέχει τα κύρια μεταδεδομένα για εφαρμογή ή εκκίνηση.
- **Ζεύγη κλειδιών-τιμών:** Σε κάθε ενότητα, ορίζετε ιδιότητες χρησιμοποιώντας ζεύγη κλειδιών-τιμών. Η μορφή είναι "Key=Value". Το κλειδί προσδιορίζει την ιδιότητα και η τιμή παρέχει τα αντίστοιχα δεδομένα.
- **Σύνταξη ιδιοτήτων:** Οι τιμές των ιδιοτήτων μπορεί να είναι διαφορετικών τύπων, συμπεριλαμβανομένων συμβολοσειρών, δυαδικών τιμών, διαδρομών αρχείων ή λιστών. Η μορφή για κάθε τιμή ιδιότητας εξαρτάται από τον τύπο της.
- **Σχόλια:** Μπορείτε να συμπεριλάβετε σχόλια σε αρχείο .desktop χρησιμοποιώντας το σύμβολο '#'. Οτιδήποτε ακολουθεί το '#' on line θεωρείται σχόλιο και αγνοείται.
