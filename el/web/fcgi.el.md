## Τι είναι ένα αρχείο FCGI;

Ένα αρχείο FCGI είναι ένα αρχείο πηγαίου κώδικα που περιέχει πηγαίο κώδικα γραμμένο για το FastCGI (FCGI). Μπορεί να γραφτεί σε πολλές διαφορετικές γλώσσες προγραμματισμού και περιέχει την επέκταση του προτύπου CGI που χρησιμοποιείται για πιο αποτελεσματικό χειρισμό αιτημάτων διακομιστή Ιστού. Οι πιο κοινές γλώσσες υποστήριξης για το FCGI περιλαμβάνουν την **Perl** που παρέχει υποστήριξη τόσο για τον κώδικα Perl όσο και για την ισχυρή εξωτερική διεπαφή του FastCGI API.

## Μορφή αρχείου FCGI

Τι κάνει ένα αρχείο FCGI σημαντικό; Η διεπαφή FCGI υποστηρίζει κατανεμημένους υπολογισμούς για τη διαχείριση των αιτημάτων Ιστού και τη διανομή τους σε πολλούς διακομιστές. Αυτό κάνει τα αρχεία FCGI να χρησιμοποιούνται κυρίως για εξισορρόπηση φορτίου σε πολλούς διακομιστές.
