## Τι είναι ένα αρχείο CPHD;

Ένα αρχείο CPHD είναι ένα έγγραφο υπερκειμένου που χρησιμοποιείται από τη μηχανή προεπεξεργαστή CloudChan. Περιέχει HTML και σενάρια για τη δημιουργία δυναμικών ιστοσελίδων και εγγράφων. Τα σενάρια μέσα σε ένα αρχείο CPHD μοιάζουν με τη σύνταξη που χρησιμοποιείται από την Java. Η ιστοσελίδα CPHD επιστρέφει ένα έγγραφο HTML όταν εκτελείται σε διαδικασίες CloudChan. Η μορφή CPHD μπορεί να λειτουργήσει με οποιοδήποτε έγγραφο XML.

## Μορφή αρχείου CPHD

Τα αρχεία CPHD αποθηκεύονται στο δίσκο ως αρχεία απλού κειμένου. Αυτό καθιστά δυνατό το άνοιγμα αυτών των αρχείων σε οποιοδήποτε πρόγραμμα επεξεργασίας κειμένου όπως το Σημειωματάριο ή το Σημειωματάριο++.
