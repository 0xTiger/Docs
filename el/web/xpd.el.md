## Τι είναι ένα αρχείο XPD;

Ένα αρχείο XPD είναι ένα αρχείο ιστοσελίδας που δημιουργήθηκε από ένα έγγραφο XML. Τις περισσότερες φορές, αυτό δημιουργείται μέσω ενός μετασχηματισμού XSLT. Οι [προδιαγραφές μορφής αρχείου XPD](https://www.w3.org/TR/xml-pipeline/) είναι διαθέσιμες στο W3 που καθορίζουν έναν τρόπο αύξησης της διαλειτουργικότητας του [XML](/el/web/xml/) πολλαπλών διεργασιών εφαρμογές. Η XML Pipeline Language ορίζει τις σχέσεις επεξεργασίας μεταξύ των πόρων XML.
