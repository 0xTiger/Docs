## Τι είναι ένα αρχείο DAP;

Ένα αρχείο με επέκταση .dap είναι ένα αρχείο σελίδας δεδομένων της Microsoft Access που δημιουργείται από τη Microsoft Access και έχει πρόσβαση στα δεδομένα που είναι αποθηκευμένα στη βάση δεδομένων. Τα δεδομένα που είναι αποθηκευμένα σε ένα τέτοιο αρχείο Ιστού είναι προσβάσιμα μέσω του δικτύου (intranet και internet) το οποίο είναι διαφορετικό από τα αρχεία [MDB](/el/database/mdb/) της Access που έχουν αναπτυχθεί ειδικά για εφαρμογές επιτραπέζιου υπολογιστή. Τέτοιες βάσεις δεδομένων μπορούν να δημιουργηθούν χρησιμοποιώντας την Access 2010 και τις Υπηρεσίες Access (προαιρετικό στοιχείο του SharePoint). Αυτές οι βάσεις δεδομένων μπορούν να χρησιμοποιηθούν από άτομα που έχουν λογαριασμούς SharePoint χρησιμοποιώντας τη διεύθυνση web σε ένα πρόγραμμα περιήγησης ιστού.
