## Τι είναι ένα αρχείο HTC;

Ένα αρχείο HTC είναι ένα αρχείο μορφής αρχείου Ιστού που χρησιμοποιείται από τον Microsoft Internet Explorer. Περιέχει ορισμούς για στοιχεία HTML που εκτελούν μικρές λειτουργίες για την εκτέλεση διαφορετικών εργασιών μέσα στο πρόγραμμα περιήγησης Ιστού. Η λειτουργικότητα που ορίζεται στα αρχεία HTC είναι επαναχρησιμοποιήσιμη και είναι προσβάσιμη σε πολλές σελίδες [HTML](/el/web/html/).

## Μορφή αρχείου HTC - Περισσότερες πληροφορίες

Τα αρχεία HTC αποθηκεύονται σε μορφή αρχείου απλού κειμένου. Αυτό σημαίνει ότι μπορείτε να τα ανοίξετε σε οποιοδήποτε πρόγραμμα επεξεργασίας κειμένου και να κάνετε αλλαγές στη λειτουργικότητα. Αυτά συνήθως ανοίγουν στον Internet Explorer, αλλά με αυτόν τον τρόπο θα μπορείτε να προβάλετε μόνο αυτά τα αρχεία και δεν θα μπορείτε να τα επεξεργαστείτε.

Ένα αρχείο HTC είναι στην πραγματικότητα ένα αρχείο HTML που είναι ένας συνδυασμός σεναρίου και ενός συνόλου στοιχείων ειδικά για το HTC για τον καθορισμό των στοιχείων.
