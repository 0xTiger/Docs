## Τι είναι ένα αρχείο JSON;

Το JSON (JavaScript Object Notation) είναι μια ανοιχτή τυπική μορφή αρχείου για κοινή χρήση δεδομένων που χρησιμοποιεί κείμενο αναγνώσιμο από τον άνθρωπο για την αποθήκευση και τη μετάδοση δεδομένων. Τα αρχεία JSON αποθηκεύονται με την επέκταση .json. Το JSON απαιτεί λιγότερη μορφοποίηση και είναι μια καλή εναλλακτική για [XML](/el/web/xml/). Το JSON προέρχεται από JavaScript αλλά είναι μια μορφή δεδομένων ανεξάρτητη από τη γλώσσα. Η δημιουργία και η ανάλυση του JSON υποστηρίζεται από πολλές σύγχρονες γλώσσες προγραμματισμού. *application/json* είναι ο τύπος μέσου που χρησιμοποιείται για JSON.

## Μορφή αρχείου JSON - Σύντομο ιστορικό

Υπήρχε ανάγκη για επικοινωνία διακομιστή σε πελάτη σε πραγματικό χρόνο που οδηγούσε στη δημιουργία του JSON. Η μορφή JSON καθορίστηκε για πρώτη φορά από τον Douglas Crockford τον Μάρτιο του 2001. Το JSON βασίστηκε στο Standard ECMA-262 3rd Edition—December 1999, το οποίο είναι υποσύνολο της JavaScript.

Η πρώτη έκδοση του προτύπου JSON ECMA-404 δημοσιεύτηκε τον Οκτώβριο του 2013 από την Ecma International. Το RFC 7159 έγινε η κύρια αναφορά για τις χρήσεις του διαδικτύου της JSON το 2014. Τον Νοέμβριο του 2017, το ISO/IEC 21778:2017 δημοσιεύτηκε ως διεθνές πρότυπο. Το RFC 8259 δημοσιεύτηκε στις 13 Δεκεμβρίου 2017 από το The Internet Engineering Task Force, το οποίο είναι η τρέχουσα έκδοση του Internet Standard STD 90.

## Δομή αρχείου JSON

Τα δεδομένα JSON γράφονται σε ζεύγη **κλειδί/τιμή**. Το κλειδί και η τιμή χωρίζονται με άνω και κάτω τελεία(:) στη μέση με το κλειδί στα αριστερά και την τιμή στα δεξιά. Διαφορετικά ζεύγη κλειδιών/τιμών χωρίζονται με κόμμα(,). Το κλειδί είναι μια συμβολοσειρά που περιβάλλεται από διπλά εισαγωγικά για παράδειγμα "όνομα". Οι τιμές μπορεί να είναι των παρακάτω τύπων.

- «Αριθμός».
- `String`: Ακολουθία χαρακτήρων Unicode που περιβάλλεται από διπλά εισαγωγικά.
- `Boolean`: Σωστό ή Λάθος.
- «Πίνακας»: Μια λίστα τιμών που περιβάλλονται από αγκύλες, για παράδειγμα

```json
[ "Μήλο", "Μπανάνα", "Πορτοκάλι" ]
```

- "Αντικείμενο": Μια συλλογή ζευγών κλειδιών/τιμών που περιβάλλονται από σγουρά τιράντες, για παράδειγμα

```json
{"name": "Jack", "age": 30, "favoriteSport" : "Football"}
```

Τα αντικείμενα JSON μπορούν επίσης να είναι ένθετα για να αντιπροσωπεύουν τη δομή των δεδομένων. Παρακάτω δίνεται ένα παράδειγμα αντικειμένου JSON.

### Παράδειγμα μορφής JSON

```json
{
   "name":"Jack",
   "age":30,
   "contactNumbers":[
      {
         "type":"Home",
         "number":"123 123-123"
      },
      {
         "type":"Office",
         "number":"321 321-321"
      }
   ],
   "spouse":null,
   "favoriteSports":[
      "Football",
      "Cricket"
   ]
}
```

## Ποιο είναι το μέγιστο μέγεθος αρχείου JSON;

Πρακτικά δεν υπάρχει όριο στο μέγιστο μέγεθος ενός αρχείου JSON. Μπορεί να είναι όσο ο χώρος που απαιτείται από το περιεχόμενο που πρέπει να αποθηκευτεί.

Όσον αφορά τη χρήση της μορφής αρχείου JSON για τη μεταφορά δεδομένων μέσω Διαδικτύου, πρέπει να είστε προσεκτικοί σχετικά με τους διαθέσιμους πόρους του υπολογιστή. Εάν μεταφερθούν μεγάλα δεδομένα JSON, η μεταφορά θα επηρεαστεί εάν το πρόγραμμα περιήγησης-πελάτη έχει περιορισμένη μνήμη.


Δεν υπάρχει σκληρό όριο που ορίζεται από τις προδιαγραφές, αλλά πρέπει να προσέχετε να μην εξαντλήσετε τους πόρους στους υπολογιστές των χρηστών σας, καθώς θα υποβαθμίσει γρήγορα την εμπειρία χρήστη τους και θα είναι πιθανό να εγκαταλείψουν την εφαρμογή σας.

## JSON εναντίον XML

Το **XML** είναι μια άλλη κοινή και ευρέως χρησιμοποιούμενη μορφή αρχείου για την ανταλλαγή δεδομένων μέσω του Διαδικτύου. Όταν πρόκειται για ανταλλαγή δεδομένων μεταξύ εφαρμογών, οι προγραμματιστές έχουν τη δυνατότητα να χρησιμοποιούν μορφές αρχείων XML και JSON. Ωστόσο, το JSON υιοθετείται ως ο πιο βολικός τρόπος για την ανταλλαγή δεδομένων μεταξύ εφαρμογών μέσω Διαδικτύου για τους ακόλουθους λόγους.

1. Το JSON παρέχει μια σαφή και ευανάγνωστη προβολή δεδομένων σε σύγκριση με τις μορφές αρχείων XML
1. Το JSON μειώνει την επιβάρυνση της μεταφοράς δεδομένων μέσω του Διαδικτύου, καθώς έχει λιγότερο αριθμό χαρακτήρων για να ορίσει το ίδιο σύνολο δεδομένων σε σύγκριση με το XML
1. Οι σύγχρονες γλώσσες προγραμματισμού παρέχουν ενσωματωμένους αναλυτές για την ανάλυση της απόκρισης JSON μέσω του ιστού.
