## Τι είναι ένα αρχείο DOCHTML;

Ένα αρχείο DOCHTML είναι η μορφή HTML ενός εγγράφου που δημιουργείται με το Microsoft Word. Είναι η έκδοση [HTML](/el/web/html/) του εγγράφου που δημιουργήθηκε και μπορεί να χρησιμοποιηθεί για δημοσίευση σε διακομιστή ιστού. Τα αρχεία DOCHTML μπορούν να ανοίξουν στο πρόγραμμα περιήγησης ιστού για την προβολή των περιεχομένων του εγγράφου word που έχει μετατραπεί. Η προοπτική και η εμφάνιση του αρχείου DOCHTML είναι ίδια με εκείνη του αρχικού εγγράφου του Word.

## Μορφή αρχείου DOCHTML

Τα αρχεία DOCHTML αποθηκεύονται σε δίσκο σε μορφή HTML που είναι έτοιμο για δημοσίευση σε διακομιστές Ιστού. Το Microsoft Word διέκοψε την υποστήριξη για αποθήκευση εγγράφων ως αρχεία DOCHTML. Αυτό αντικαταστάθηκε με τις μορφές αρχείων [.mht](/el/web/mht/) και [.mhtml](/el/web/mhtml/) που χρησιμοποιούνται συνήθως τώρα.
