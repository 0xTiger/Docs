## Τι είναι ένα αρχείο AXD;

Ένα αρχείο AXD είναι ένα αρχείο ρυθμίσεων ιστού που χρησιμοποιείται για το χειρισμό και την ανάκτηση αιτημάτων ενσωματωμένων πόρων στο ASP.NET. Αποτελείται από οδηγίες για την ανάκτηση ενσωματωμένων πόρων όπως εικόνες, αρχεία JavaScript ([.JS](/el/web/js/)) και αρχεία [.CSS](/el/web/css/). Τα αρχεία AXD χρησιμοποιούνται για την εισαγωγή πόρων σε σελίδες πελάτη. Αυτό επιτρέπει στις σελίδες πελατών να έχουν πρόσβαση σε αυτούς τους πόρους στον διακομιστή με τυπικό τρόπο.

## Μορφή αρχείου AXD

Τα αρχεία AXD αποθηκεύονται ως δυαδικά αρχεία στην πλευρά του διακομιστή. Αναφέρεται σε ένα αρχείο Web Handler στο ASP.NET, τα οποία είναι στοιχεία που δημιουργούν απαντήσεις σε συγκεκριμένα αιτήματα σε έναν διακομιστή web. Τα αρχεία AXD εκτελούν προσαρμοσμένη επεξεργασία πριν από τη δημιουργία της απάντησης βάσει αιτημάτων σελίδας από την πλευρά του πελάτη. Αυτά συνήθως αποθηκεύονται ως αρχεία **WebResource.axd**.

### Τι είναι το αρχείο WebResource.axd;

Τα περισσότερα έργα ASP.NET αποθηκεύουν αρχεία AXD ως αρχεία WebResource.axd που επιτρέπουν στις σελίδες πελάτη να κάνουν λήψη πόρων που είναι ενσωματωμένοι σε μια συναρμολόγηση. Η εφαρμογή σας ενδέχεται να αντιμετωπίσει αργή απόδοση σε περίπτωση που την εκτελέσετε σε λειτουργία εντοπισμού σφαλμάτων, καθώς το πρόγραμμα χειρισμού WebResources.axd δεν είναι αποθηκευμένο στην προσωρινή μνήμη.
