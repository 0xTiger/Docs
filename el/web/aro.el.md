## Τι είναι ένα αρχείο ARO;

Ένα αρχείο ARO είναι ένα αρχείο ιστοσελίδας από την πλευρά του διακομιστή που δημιουργείται με την εφαρμογή Web SteelArrow. Περιέχει ετικέτες και σενάρια [HTML](/el/web/html/) και SteelArrow. Αυτά εκτελούνται στον διακομιστή για τη δημιουργία πλήρους σελίδας απόκρισης πριν από την αποστολή τους στο πρόγραμμα περιήγησης του χρήστη. Τα αρχεία ARO περιέχουν σχεδόν το 95% του περιεχομένου HTML ενώ το υπόλοιπο αποτελείται από κώδικα SteelArrow. Για να λειτουργούν τα αρχεία ARO για εσάς, ο διακομιστής εφαρμογών Web SteelArrow θα πρέπει να είναι εγκατεστημένος και να εκτελείται στο μηχάνημα διακομιστή ιστού.

## Μορφή αρχείου ARO

Τα αρχεία ARO είναι γραμμένα σε αρχείο γλώσσας σήμανσης HTML με ενσωματωμένο κώδικα JavaScript και μικροεφαρμογές Java. Οι [ετικέτες SteelArrow](http://www.steelarrow.com/docs/basics1.aro) είναι τα βασικά δομικά στοιχεία για την ανάπτυξη ιστοσελίδων. Αν και αυτά δεν αλλάζουν την εμφάνιση της σελίδας, χρησιμοποιούνται για να γεμίσουν τη σελίδα με δεδομένα. Επιπλέον, μπορούν επίσης να χρησιμοποιηθούν για τον έλεγχο της εξόδου με δηλώσεις υπό όρους.

### Παράδειγμα ετικέτας ARO

ο<SAOUTPUT> Η ετικέτα ARO επιτρέπει στους προγραμματιστές να εξάγουν τιμές δεδομένων σε οποιοδήποτε σημείο μέσα σε ένα σενάριο. Για παράδειγμα, μπορεί να χρησιμοποιηθεί για τη λήψη της εξόδου του πίνακα PARAM με<SAOUTPUT VALUE=PARAM[]> που μπορεί να εμφανιστεί στο HTML<BODY> ετικέτα.
