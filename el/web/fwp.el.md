## Τι είναι ένα αρχείο FWP;

Ένα αρχείο με αρχείο .fwp είναι ένα αρχείο αρχειοθέτησης που δημιουργήθηκε με το Microsoft Expression Web και περιέχει όλα τα αρχεία και τους φακέλους για ένα σχέδιο. Το Microsoft Expression Web είναι ένα γενικό προϊόν λογισμικού σχεδιασμού ιστοσελίδων από τη Microsoft και βασίζεται σε πρόγραμμα επεξεργασίας [HTML](/el/web/html/). Έχει διακοπεί από το 2012 και διατίθεται δωρεάν για λήψη από τη Microsoft. Τα αρχεία FWP μπορούν να εξαχθούν και να αποθηκευτούν ως μορφή .cab της Microsoft.

## Μορφή αρχείου FWP - Περισσότερες πληροφορίες

Οι ακριβείς προδιαγραφές μορφής αρχείου των αρχείων FWP δεν είναι διαθέσιμες για αναφορά του προγραμματιστή. Ωστόσο, αυτά είναι δυαδικά αρχεία και μπορούν να ανοίξουν στο Microsoft Expression Studio που έχει διακοπεί εδώ και πολύ καιρό.
