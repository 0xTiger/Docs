## Τι είναι ένα αρχείο DOCMHTML;

Ένα αρχείο DOCMHTML είναι μια παλιά παραλλαγή του αρχείου Ιστού που αποθηκεύτηκε από τις προηγούμενες εκδόσεις του Microsoft Word. Δεν υποστηρίζεται πλέον από την πιο πρόσφατη έκδοση του Microsoft Word. Ένα αρχείο DOCMHTML περιέχει αναπαράσταση HTML του αρχικού εγγράφου .doc word. Μπορεί να ανοίξει στο Microsoft Word καθώς και σε πρόγραμμα περιήγησης ιστού. Όταν δημοσιεύεται ένα DOCMHTML, μπορεί να φιλοξενηθεί σε διακομιστή web που είναι προσβάσιμος στους χρήστες μέσω αιτήματος Ιστού. Μια άλλη παρόμοια μορφή είναι η [DOCHTML](/el/web/dochtml/) η οποία επίσης έχει διακοπεί.

## Μορφή αρχείου DOCMHTML - Περισσότερες πληροφορίες

Τα αρχεία DOCMHTML χρησιμοποιήθηκαν για αποθήκευση σε μορφή HTML. Θα μπορούσε να ανοίξει σε ένα πρόγραμμα επεξεργασίας κειμένου όπως το Σημειωματάριο και το Σημειωματάριο ++. Αφού επισημάνθηκε ως απαρχαιωμένο, το Microsoft Word εισήγαγε τις επιλογές αποθήκευσης εγγράφου word ως αρχεία [MHTML](/el/web/mhtml/) και [MHT](/el/web/mht/). Αυτές οι νέες μορφές αρχείων μπορούν να ανοίξουν με προγράμματα περιήγησης ιστού.
