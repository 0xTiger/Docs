## Τι είναι ένα αρχείο VDISCO;

Ένα αρχείο VDISCO είναι ένα αρχείο εντοπισμού που χρησιμοποιείται από το Microsoft Visual Studio στις Υπηρεσίες Ιστού του. Παρέχει πληροφορίες σχετικά με τις διαθέσιμες Υπηρεσίες Ιστού, όπως τις διευθύνσεις URL, τους χώρους ονομάτων και τις μεθόδους τους. Το αρχείο VDISCO είναι παρόμοιο με τη μορφή αρχείου [DISCO](/el/web/disco/), αλλά δημιουργείται κατά το χρόνο εκτέλεσης. Αποθηκεύεται στην εφαρμογή υπηρεσίας Web και χρησιμοποιείται από το Visual Studio για δυναμική ανακάλυψη και χρήση Υπηρεσιών Web σε ένα έργο. Τα αρχεία VDISCO χρησιμοποιούνται σε συνδυασμό με το αρχείο [.asmx](/el/web/asmx/) που περιέχει τον πραγματικό κωδικό της υπηρεσίας Web.

Μπορείτε να ανοίξετε το αρχείο VDISCO σε οποιοδήποτε πρόγραμμα επεξεργασίας κειμένου, όπως το Microsoft Notepad, το Github Atom και το Apple TextEdit. Μπορεί επίσης να ανοίξει με το Microsoft Visual Studio 2012 και μεταγενέστερα για εργασία με αυτό.

## Μορφή αρχείου VDISCO

Ένα αρχείο VDISCO αποθηκεύεται και φιλοξενείται σε μορφή αρχείου [XML](/el/web/xml/), η οποία είναι μια καθολική μορφή για τη σύνθεση και τη δημοσίευση δεδομένων. Περιέχει τη διεύθυνση URL της υπηρεσίας, τους χώρους ονομάτων και τις μεθόδους σε τυπικές ετικέτες XML όπου κάθε ετικέτα περιέχει την αντίστοιχη τιμή των πληροφοριών της.
