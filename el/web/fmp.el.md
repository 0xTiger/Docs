## Τι είναι ένα αρχείο FMP;

Ένα αρχείο με αρχείο .fmp είναι ένα κινούμενο αρχείο μενού που δημιουργήθηκε με την εφαρμογή 123 Flash Menu. Περιέχει κινούμενα μενού για ιστότοπο, συμπεριλαμβανομένων στοιχείων όπως γραμματοσειρές, στυλ μενού, χρώματα, διαστάσεις, ήχους και κινούμενα σχέδια. Η εφαρμογή 123 Flash Menu δεν απαιτεί ειδικές γνώσεις υπολογιστή, καθώς η ίδια η εφαρμογή ενσωματώνει όλες τις λειτουργίες για την ανάπτυξη τέτοιων μενού. Τα αρχεία του μενού FMP μπορούν να εξαχθούν στα διάσημα αρχεία [.swf](/el/page-description-language/swf/) του Flash, που μπορούν να ενσωματωθούν σε μια ιστοσελίδα.
