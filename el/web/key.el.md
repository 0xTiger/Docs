## Τι είναι ένα αρχείο KEY;

Ένα αρχείο KEY είναι το ιδιωτικό μέρος ενός μηχανισμού ασφαλείας που αποθηκεύεται σε δίσκο σε μορφή αρχείου Mal (PEM) με Enhanced Privacy-Enhanced. Χρησιμοποιείται για την αποκρυπτογράφηση πληροφοριών που ανταλλάσσονται μεταξύ ενός προγράμματος περιήγησης και του διακομιστή ιστού που εξυπηρετεί τα αιτήματα περιήγησης. Ένα αρχείο KEY περιέχει κωδικοποιημένες συμβολοσειρές που είναι άχρηστες για το ανθρώπινο μάτι, αλλά αποτελούν την ουσία της κρυπτογράφησης/αποκρυπτογράφησης της ανταλλαγής πληροφοριών ως μέρος του πιστοποιητικού SSL σε διακομιστές Ιστού. Τα αρχεία KEY δημιουργούνται αυτόματα και εγκαθίστανται στο τέλος του πελάτη.

Μπορείτε να ανοίξετε αρχεία **KEY** με οποιοδήποτε πρόγραμμα επεξεργασίας κειμένου όπως το Microsoft Notepad (Windows), το Apple TextEdit (Mac) ή το GitHub Atom (cross-platform). Τα αρχεία KEY είναι παρόμοια με τις μορφές αρχείων [CRT](/el/web/crt/) και [CER](/el/web/cer/).

## Μορφή αρχείου KEY - Περισσότερες πληροφορίες

Τα αρχεία KEY αποθηκεύονται σε δίσκο ως αρχεία απλού κειμένου, αλλά είναι κωδικοποιημένες συμβολοσειρές που δεν είναι φιλικές προς τον άνθρωπο για ανάγνωση. Πρακτικά, οι χρήστες έχουν μηδενική κατανόηση των συμβολοσειρών όταν ανοίγουν σε ένα πρόγραμμα επεξεργασίας κειμένου. Το πιστοποιητικό ιδιωτικού κλειδιού είναι εμπιστευτικό και δεν πρέπει να κοινοποιείται σε κανέναν. Η πλήρης διαδικασία διασφάλισης της ανταλλαγής πληροφοριών μέσω του Διαδικτύου περιλαμβάνει:

* **Δημόσιο κλειδί** - χρησιμοποιείται για την κρυπτογράφηση των πληροφοριών του χρήστη για την ανταλλαγή δεδομένων μεταξύ του προγράμματος περιήγησης και των διακομιστών Ιστού
* **Ιδιωτικό κλειδί** - χρησιμοποιείται για την αποκρυπτογράφηση των πληροφοριών καθώς λαμβάνονται από τον διακομιστή web

Τα πιστοποιητικά SSL, γνωστά και ως πιστοποιητικά X.509, είναι μοναδικά για κάθε ιστότοπο. KEY αρχεία που χρησιμοποιούν την ακόλουθη σύνταξη:
