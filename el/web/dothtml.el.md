## Τι είναι ένα αρχείο DOTHTML;

Ένα αρχείο DOTHTML είναι ένα έγγραφο προτύπου του Microsoft Word που είναι αποθηκευμένο σε έκδοση HTML. Μπορεί να ανοίξει στο πρόγραμμα περιήγησης και τα περιεχόμενα εμφανίζονται όπως οποιαδήποτε άλλη ιστοσελίδα. Τα αρχεία DOTHTML δημιουργούνται από αρχεία Microsoft Word .DOT που περιέχουν προδιαμορφωμένες ρυθμίσεις για τη δημιουργία αυτών των αρχείων. Όταν ένα αρχείο [DOT](/el/word-processing/dot/) μετατρέπεται σε DOCHTML, διατηρούνται όλες αυτές οι ρυθμίσεις και το αρχείο DOTHTML που έχει μετατραπεί εμφανίζεται με παρόμοια μορφοποίηση όπως το αρχικό αρχείο DOT. Με αυτόν τον τρόπο, τα δημοσιευμένα πρότυπα μπορούν να προβληθούν στο πρόγραμμα περιήγησης ιστού.

## Μορφή αρχείου DOTHTML - Περισσότερες πληροφορίες

Τα αρχεία DOTHTML αποθηκεύονται ως αρχεία [HTML](/el/web/html/) και περιέχουν πηγαίο κώδικα HTML που μπορεί να ανοίξει με οποιοδήποτε πρόγραμμα επεξεργασίας HTML. Τις πρώτες μέρες, το Microsoft Word προσέφερε τις εκδόσεις HTML όλων των μορφών εγγράφων του, όπως [DOCHTML](/el/web/dochtml/), [DOCMHTML](/el/web/docmhtml/), αλλά αργότερα αυτές οι εκδόσεις βασισμένες σε HTML διακόπηκαν . Ως αποτέλεσμα, το DOTHMLT δεν υποστηρίζεται πλέον.
