## Τι είναι ένα αρχείο XOML;

Το XOML είναι το ακρωνύμιο του Extensible Object Markup Language και είναι μια μορφή σειριοποίησης για τα αντικείμενα ροής εργασίας του Windows Workflow Foundation. Βασισμένο στο **[XAML](/el/web/xaml/)**, χρησιμοποιείται κυρίως για τη δημιουργία διεπαφών χρήστη σε απλό **[XML](/el/web/xml/)**. Αυτά χρησιμοποιούνται για τη δήλωση της ροής εργασίας για τη διεπαφή χρήστη και μεταγλωττίζονται μαζί με το ξεχωριστό αρχείο που περιέχει τη λογική υλοποίησης. Περιέχει μια δομή βασισμένη σε δέντρο που ορίζει έναν κόμβο ροής εργασίας ρίζας, ένθετα υποστοιχεία και ενσωματωμένα τμήματα κώδικα. Όταν δημιουργείται μια νέα ροή εργασίας με το Visual Studio για .NET, έχει τη δυνατότητα να επιλέξετε εάν θα είναι σε μορφή κώδικα ή σε μορφή διαχωρισμένου κώδικα. Σε περίπτωση που επιλέξετε το μεταγενέστερο, το IDE θα δημιουργήσει δύο αρχεία για εσάς. το ένα σε μορφή XOML (αποτελούμενο από διάταξη ροής εργασίας και ρυθμίσεις) και άλλο σε μορφή [.CS](/el/programming/cs/)/[.VB](/el/programming/vb/) όπου βρίσκεται ο κώδικας προγραμματισμού.
