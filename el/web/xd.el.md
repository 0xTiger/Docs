## Τι είναι ένα αρχείο XD;

Ένα αρχείο με επέκταση .xd είναι μια μακέτα που δημιουργήθηκε για ιστότοπους ή εφαρμογές για κινητές συσκευές χρησιμοποιώντας το λογισμικό Adobe XD. Οι σχεδιαστές, οι δημιουργικές ομάδες και οι οργανισμοί χρησιμοποιούν αυτά τα αρχεία για να σχεδιάσουν εφαρμογές παραπέμποντας σε αυτά. Ένα μεμονωμένο αρχείο XD περιέχει στοιχεία σχεδίασης όπως εικόνες, σχήματα και κείμενο που παρέχουν πλήρες σύνολο χαρακτηριστικών για την ανάπτυξη γρήγορων και έξυπνων πρωτοτύπων wireframe ώστε οι προγραμματιστές να ξεκινήσουν να τα αναπτύσσουν ως εφαρμογές.

## Μορφή αρχείου XD

Τα αρχεία XD περιέχουν γραφικά στοιχεία (εικόνα, σχήματα και κείμενο) σε επίπεδα ως έναν ή περισσότερους πίνακες τέχνης. Αυτά μαζί με οδηγούς και στρώματα παρέχουν τα δομικά στοιχεία για τα καλώδια.

* «Artboards» - Αυτά αντιπροσωπεύουν τις οθόνες σε μια εφαρμογή ή σελίδες ενός ιστότοπου και βελτιστοποιούν τις διαδικασίες σχεδιασμού. Ένα artboard δημιουργεί διαδραστικά σχέδια για πολλαπλά μεγέθη οθόνης σε ένα μόνο έγγραφο.
* «Οδηγοί» - Αυτά χρησιμοποιούνται για την ευθυγράμμιση αντικειμένων ή τη διαχείριση του διαστήματος μεταξύ των στοιχείων σε έναν πίνακα τέχνης. Οι οδηγοί χρησιμοποιούν πλέγματα που χρησιμεύουν ως πλαίσιο για την οργάνωση στοιχείων σχεδίασης (εικόνες, γλυφές, παραγράφους) και διασφαλίζουν τη συμμετρία της διάταξης.
* «Επίπεδα» - Οι σχεδιαστές UX χρησιμοποιούν επίπεδα στο XD με τον ίδιο τρόπο που τα χρησιμοποιούν στο Adoble Illustrator ή στο Photoshop. Αυτά χρησιμοποιούνται για την οργάνωση της εργασίας σε πίνακες τέχνης, προκειμένου να αποφευχθεί η ακαταστασία και ελέγχονται από τον πίνακα επιπέδων.
