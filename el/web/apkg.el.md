## Τι είναι ένα αρχείο APKG;

Ένα αρχείο με επέκταση .apkg είναι μια γκάμα από κάρτες flash που δημιουργούνται για χρήση στην εφαρμογή λογισμικού [Anki](https://ankiweb.net/about), η οποία είναι ένα πρόγραμμα εκμάθησης που βασίζεται σε κάρτες flash. Περιέχει [HTML](/el/web/html/) κείμενο που θα φορτωθεί και θα εμφανιστεί στην εφαρμογή Anki και μπορεί επιπλέον να έχει εικόνες και ήχους για οπτική και ακουστική εκμάθηση. Το Anki επιτρέπει στους χρήστες να δημιουργήσουν τις δικές τους τράπουλες για κάρτες flash Anki καθώς και να εισάγουν τις τράπουλες καρτών flash άλλων χρηστών.

## Μορφή αρχείου APKG

Οι κάρτες Anki δημιουργούνται από πρότυπα που είναι γραμμένα σε HTML, μια διάσημη και κοινή γλώσσα για τη δημιουργία ιστοσελίδων. Το στυλ των καρτών τράπουλας γίνεται με χρήση CSS που είναι η γλώσσα που χρησιμοποιείται για το στυλ ιστοσελίδων. Το στυλ περιλαμβάνει τροποποιήσεις σε:

* font-family - Το όνομα της γραμματοσειράς που θα χρησιμοποιηθεί στην κάρτα.
* font-size - Το μέγεθος της γραμματοσειράς σε pixel.
* text-align - Καθορίζει εάν το κείμενο θα πρέπει να στοιχίζεται στο κέντρο, αριστερά ή δεξιά.
* χρώμα - Καθορίζει το χρώμα του κειμένου που μπορεί να είναι απλά ονόματα χρωμάτων όπως "μπλε", "κόκκινο" κ.λπ. ή χρωματικοί κώδικες HTML.
* background-color - Καθορίζει το χρώμα φόντου της κάρτας

Οι πληροφορίες στυλ μοιράζονται μεταξύ όλων των καρτών, επηρεάζοντας όλες τις κάρτες όταν γίνεται μια αλλαγή. Το παρακάτω παράδειγμα θα χρησιμοποιεί ένα κίτρινο φόντο σε όλα τα φύλλα εκτός από το πρώτο:

```CSS
.card {
  background-color: yellow;
}
.card1 {
  background-color: blue;
}
```

Η αλλαγή μεγέθους των εικόνων σε μια κάρτα Anki μπορεί να ελεγχθεί ως εξής.

```CS
img {
  max-width: none;
  max-height: none;
}
```

## Ενσωμάτωση Javascript σε κάρτες Anki

Είναι δυνατή η ενσωμάτωση του [Javascript](/el/web/js/) σε μια κάρτα Anki χρησιμοποιώντας τα πρότυπα καρτών. Ωστόσο, λόγω του προηγμένου επιπέδου Javascript, δεν παρέχεται η υποστήριξή του. Επιπλέον, οι συσκευές απόδοσης ενδέχεται να εμφανίζουν διαφορετικά τις επιδράσεις υλοποίησης του Javascript στην κάρτα, με αποτέλεσμα την ανάγκη δοκιμής της υλοποίησης σε όλες τις συσκευές. Ορισμένες λειτουργίες Javascript, όπως το window.alert, καθιστούν δύσκολο τον εντοπισμό σφαλμάτων του κώδικα Javascript που έχετε γράψει.
