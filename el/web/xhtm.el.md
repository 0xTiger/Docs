## Τι είναι ένα αρχείο XHTM;

Ένα αρχείο με επέκταση .xhtm είναι ένα αρχείο ιστοσελίδας που είναι γραμμένο σε XHTML. Χρησιμοποιεί το XML 1.0 ως τυπική του εφαρμογή και θεωρείται αναδιατύπωση της HTML 4. Όπως το XHTML, ένα αρχείο XHTM περιέχει τυπικές ετικέτες HTML που ακολουθούν πιο αυστηρές οδηγίες XML. Τα αρχεία XHTM χρησιμοποιούνται για τη δημοσίευση περιεχομένου ιστότοπου και συνήθως χρησιμοποιούν την επέκταση [XHTML](/el/web/xhtml/).

Μπορείτε να χρησιμοποιήσετε προγράμματα περιήγησης ιστού όπως το Google Chrome, το Apple Safari και το Mozilla Firefox για να ανοίξετε αρχεία XHTM.

## Παράδειγμα XHTM

Το ακόλουθο παράδειγμα ενός εγγράφου XHTM ορίζει τις δηλώσεις XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
