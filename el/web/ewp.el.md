## Τι είναι ένα αρχείο EWP;

Ένα αρχείο EWP είναι ένα αρχείο έργου που δημιουργήθηκε με το λογισμικό [EWISoft](http://www.ewisoft.com/) Website Builder. Συνδέεται με ένα δυαδικό αρχείο βάσης δεδομένων που περιέχει όλες τις πληροφορίες για το έργο. Το αρχείο EWP χρησιμοποιείται συνήθως για την αποθήκευση της προόδου ανάπτυξης ιστού. Είναι ένα λογισμικό σχεδιασμού ιστοσελίδων που βασίζεται σε πρότυπα και επιτρέπει στους χρήστες να δημιουργούν ιστοσελίδες από πρότυπα ιστοτόπων.

## Μορφή αρχείου EWP - Περισσότερες πληροφορίες

Τα αρχεία EWP είναι αρχεία έργου που δεν χρειάζεται να μεταγλωττιστούν ή να φιλοξενηθούν σε κανένα διακομιστή. Αυτά αποθηκεύονται ως αρχεία απλού κειμένου και περιέχουν αναφορές στην πρόοδο ανάπτυξης ιστού. Τα έργα μπορούν να δημοσιευτούν σε έναν ιστότοπο μόλις ολοκληρωθεί η ανάπτυξη.
