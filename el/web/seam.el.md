## Τι είναι ένα αρχείο SEAM;

Ένα αρχείο SEAM είναι ένα αρχείο διαμόρφωσης που χρησιμοποιείται από το πλαίσιο JBOS Seam. Περιέχει μεταδεδομένα/αναφορά σε ένα servlet από την πλευρά του διακομιστή που επεξεργάζεται αιτήματα προς την εφαρμογή Ιστού. Περιέχει επίσης μεταδεδομένα σχετικά με τα στοιχεία Seam, όπως τα ονόματα των κλάσεων και τις μεθόδους που υλοποιούν τα στοιχεία. Μπορείτε να χρησιμοποιήσετε αρχεία SEAM σε συνδυασμό με το πλαίσιο JavaServer Faces (JSF) για τη δημιουργία δυναμικών και διαδραστικών εφαρμογών web. Τα αρχεία SEAM συνήθως αποθηκεύονται στον κατάλογο WEB-INF μιας εφαρμογής Seam.

## Μορφή αρχείου SEAM

Ένα αρχείο SEAM αποθηκεύεται ως αρχείο κειμένου, συνήθως σε μορφή [XML](/el/web/xml/) και περιέχει πληροφορίες διαμόρφωσης για μια εφαρμογή Seam. Αποτελεί βασικό μέρος της διαμόρφωσης μιας εφαρμογής Seam. Μπορείτε να ανοίξετε ένα αρχείο SEAM με οποιοδήποτε πρόγραμμα επεξεργασίας κειμένου ή XML.
