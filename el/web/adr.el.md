## Τι είναι ένα αρχείο ADR;

Ένα αρχείο ADR (αποθηκευμένο με επέκταση .adr) είναι μια λίστα σελιδοδεικτών που δημιουργήθηκε από το πρόγραμμα περιήγησης Web Opera. Αυτή η λίστα συμπληρώνεται καθώς ο χρήστης διατηρεί σελιδοδείκτες διευθύνσεις URL ιστού. Αυτοί οι σελιδοδείκτες αποθηκεύονται στο αρχείο ADR για το οποίο μπορείτε να δημιουργήσετε αντίγραφα ασφαλείας και να επαναφέρετε όταν απαιτείται. Δεδομένου ότι τα περισσότερα προγράμματα περιήγησης όπως το Chrome και το Firefox αποθηκεύουν σελιδοδείκτες σε μορφή [HTML](/el/web/html/), αυτοί μπορούν να μετατραπούν σε μορφή αρχείου ADR για εισαγωγή στο πρόγραμμα περιήγησης Opera. Τα αρχεία ADR μπορούν να ανοίξουν στο πρόγραμμα περιήγησης Opera.
