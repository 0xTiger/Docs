## Τι είναι ένα αρχείο KIT;

Ένα αρχείο με επέκταση .kit είναι ένα αρχείο HTML που δημιουργείται με την εφαρμογή γλώσσας προγραμματισμού [CodeKit](https://codekitapp.com/). Περιέχει "εισαγωγές" και "μεταβλητές" εκτός από τα υπάρχοντα αρχεία [HTML](/el/web/html/), γεγονός που το καθιστά ιδανικό για στατικούς ιστότοπους. Το CodeKit μεταγλωττίζει αρχεία KIT σε HTML που μπορούν εύκολα να χρησιμοποιηθούν ως στατικά αρχεία ιστότοπου.

## Μορφή αρχείου KIT

Τα αρχεία KIT είναι αρχεία HTML που περιλαμβάνουν επιπλέον εισαγωγές και μεταβλητές. Αυτά αποθηκεύονται ως αρχεία απλού κειμένου και μπορούν να ανοίξουν με οποιοδήποτε πρόγραμμα επεξεργασίας κειμένου ή επεξεργαστή αρχείων web.

### Εισαγωγές αρχείων KIT

Σχεδόν κάθε τύπος αρχείου μπορεί να εισαχθεί σε ένα αρχείο Kit. Ακολουθεί η σύνταξη εισαγωγής που χρησιμοποιείται για την εισαγωγή αρχείων σε ένα αρχείο .kit.

```
<!-- @import "someFile.kit" -->
<!-- @import "file.html" -->
```

Όταν μια εισαγωγή προστίθεται σε ένα αρχείο KIT και αποθηκεύεται, αντικαθίσταται με το κείμενο του αρχείου που εισάγεται. Μπορείτε επίσης να χρησιμοποιήσετε το @include αντί για το @import.

### Πολλαπλές εισαγωγές σε αρχείο KIT

Μπορείτε επίσης να εισάγετε περισσότερα από ένα αρχεία κάθε φορά χρησιμοποιώντας μια λίστα διαχωρισμένη με κόμματα:

```
<!-- @import someFile, otherFile.html, ../thirdFile.kit -->
```
