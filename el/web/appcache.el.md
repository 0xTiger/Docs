## Τι είναι ένα αρχείο APPCACHE;

Ένα αρχείο APPCACHE είναι ένα αρχείο κειμένου που περιέχει τη λίστα των πόρων που πρέπει να αποθηκευτούν προσωρινά από τα προγράμματα περιήγησης για πρόσβαση εκτός σύνδεσης στις εφαρμογές Ιστού. Αυτό είναι χρήσιμο όταν δεν υπάρχει σύνδεση στο Διαδίκτυο. Σε μια τέτοια περίπτωση, το πρόγραμμα περιήγησης χρησιμοποιεί τους πόρους της προσωρινής μνήμης εκτός σύνδεσης για να παρέχει πρόσβαση στο περιεχόμενο Ιστού. Το αρχείο APPCACHE περιέχει πόρους ιστού όπως εικόνες (για παράδειγμα **[PNG](/el/image/png/)**, **[WEBP](/el/image/webp/)**, κ.λπ.), φύλλα στυλ **([ CSS])(/el/web/css/)** και αρχεία σεναρίων (όπως αρχεία Javascript **[JS](/el/web/js/)**).

Οι εφαρμογές που μπορούν **να ανοίξουν αρχεία APPCACHE** περιλαμβάνουν προγράμματα περιήγησης ιστού όπως το Google Chrome, το Safari και το Firefox.

## Μορφή αρχείου APPCACHE - Περισσότερες πληροφορίες

Τα αρχεία APPCACHE είναι απλά αρχεία κειμένου, που παρέχουν πρόσβαση εκτός σύνδεσης σε ιστοσελίδες για εκτέλεση χωρίς σύνδεση στο Διαδίκτυο. Η παρουσία του APPCACHE ορίζει μια σελίδα ως διαθέσιμη εκτός σύνδεσης.

### Πώς να αναφέρω ένα αρχείο APPCACHE;

Στη σελίδα HTML σας, ένα αρχείο APPCACHE αναφέρεται ως εξής.

```
<html manifest="example.appcache">
  ...
</html>
```

## Δομή ενός αρχείου Manifest APPCACHE

Ένα απλό αρχείο δήλωσης APPCACHE έχει την εξής μορφή.

```
CACHE MANIFEST
index.html
stylesheet.css
images/logo.png
scripts/main.js
http://cdn.example.com/scripts/main.js
```

Αυτό είναι ένα απλούστερο παράδειγμα και μπορεί να υπάρχουν και πιο περίπλοκες δομές.

## Πλεονεκτήματα του APPCACHE Manifest

Η χρήση της διασύνδεσης κρυφής μνήμης δίνει στις εφαρμογές Ιστού τα ακόλουθα πλεονεκτήματα.

1. Περιήγηση εκτός σύνδεσης - Η διεπαφή προσωρινής μνήμης δίνει στους τελικούς χρήστες να περιηγηθούν στον πλήρη ιστότοπό σας όταν είναι εκτός σύνδεσης
1. Ταχύτητα - η προσωρινή μνήμη επιτρέπει την πρόσβαση υψηλής ταχύτητας σε περιεχόμενο εκτός σύνδεσης απευθείας από το δίσκο
1. Προσβασιμότητα συνεχώς - Σε περίπτωση που ο ιστότοπός σας καταρρεύσει, οι χρήστες θα εξακολουθούν να έχουν πρόσβαση στο περιεχόμενο ιστού και θα μπορούν να έχουν την εμπειρία εκτός σύνδεσης
