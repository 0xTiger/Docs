## Τι είναι ένα αρχείο CON;

Ένα αρχείο CON είναι ένα αρχείο πηγαίου κώδικα για εφαρμογή που αναπτύχθηκε για τον **Concept Application Server**. Χρησιμοποιείται για τη σύνταξη εφαρμογών για ανταλλαγή δεδομένων μεταξύ διακομιστή και χρήστη. Τα αρχεία CON που φιλοξενούνται στον διακομιστή είναι προσβάσιμα με ένα πρόγραμμα περιήγησης Ιστού, υπό την προϋπόθεση ότι το Concept Client είναι εγκατεστημένο στο τέλος του χρήστη. Το Concept Application Server είναι ένας διακομιστής ιστού με γλώσσα προγραμματισμού μικρής κλίμακας που μπορεί να έχει μηχανή βάσης δεδομένων υποσυνόλου [SQL](/el/database/sql/) (TinDB).

Τα αρχεία CON μπορούν να ανοίγουν/εκτελούνται με το RadGs Concept Application Server.

## Μορφή αρχείου CON - Περισσότερες πληροφορίες

Τα αρχεία CON φιλοξενούνται στον διακομιστή και προσπελάζονται μέσω του προγράμματος περιήγησης ιστού στο μηχάνημα χρήστη. Η έννοια [URLs](/el/web/url/) διαφέρει από τα κανονικά url HTTP και ξεκινά με **concept://**.

### CON Παράδειγμα διεύθυνσης URL αρχείου

Μπορείτε να προσπελάσετε ένα αρχείο CON που φιλοξενείται στον Concept Application Server μέσω του προγράμματος περιήγησης ιστού χρησιμοποιώντας διευθύνσεις URL όπως:

```
concept://domain.server.com/web_application/main.con.
```
