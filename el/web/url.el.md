## Τι είναι ένα αρχείο URL;

Ένα αρχείο με επέκταση .url είναι μια συντόμευση διεύθυνσης URL ιστότοπου που μπορεί να αποθηκευτεί στον υπολογιστή με οποιαδήποτε διεύθυνση ιστότοπου. Όταν ανοίξει με διπλό κλικ, η διεύθυνση URL θα ανοίξει τον ιστότοπο στο προεπιλεγμένο πρόγραμμα περιήγησης και η σελίδα θα εμφανιστεί στον χρήστη. Τα αρχεία URL αναφέρονται συνήθως ως αρχεία συντομεύσεων σε ιστότοπους και αντιπροσωπεύονται από το εικονίδιο του προεπιλεγμένου προγράμματος περιήγησης. Τα αρχεία URL μπορούν να ανοίξουν και να επεξεργαστούν σε οποιοδήποτε πρόγραμμα επεξεργασίας κειμένου όπως το Microsoft Notepad, το Notepad++ ή το Apple TextEdit.

## Μορφή αρχείου URL - Περισσότερες πληροφορίες

Τα αρχεία URL αποθηκεύονται σε μορφή απλού κειμένου και μπορούν να ανοίξουν και να προβληθούν σε οποιοδήποτε πρόγραμμα επεξεργασίας κειμένου. Αυτά ανοίγουν στο προεπιλεγμένο πρόγραμμα περιήγησης ιστού όπου ο χρήστης εκτελεί άλλη περιήγηση.
