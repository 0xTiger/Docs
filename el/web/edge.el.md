## Τι είναι ένα αρχείο EDGE;

Ένα αρχείο με επέκταση .edge είναι το αρχείο Adobe Edge Project που δημιουργείται όταν δημιουργείται ένα έργο και αποθηκεύεται με το Adobe Edge. Το Adobe Edge επιτρέπει στους σχεδιαστές ιστού να δημιουργούν διαδραστικά [HTML](/el/web/html/) animators για web, ψηφιακές εκδόσεις και διαφημίσεις εμπλουτισμένων μέσων τόσο για προγράμματα περιήγησης για υπολογιστές όσο και για κινητά. Αυτό επιτυγχάνεται χρησιμοποιώντας HTML/HTML5, [JavaScript](/el/web/js/), και [CSS3](/el/web/css/). Τα αρχεία Edge δημιουργούνται ως αρχεία απλού κειμένου και μπορούν να ανοίξουν με πολλές διαφορετικές εφαρμογές όπως το Microsoft Notepad, το Apple TextEdit και το Notepad++.

## Μορφή αρχείου EDGE

Τα αρχεία EDGE αποθηκεύονται σε μορφή απλού κειμένου και χρησιμοποιούν jQuery για την ενεργοποίηση της κίνησης. Μια εναλλακτική της κινούμενης εικόνας EDGE ήταν τα αρχεία Flash που δεν υποστηρίζονται πλέον. Το αρχείο EDGE αποθηκεύεται μαζί με τα αρχεία HTML, JavaScript και CSS. Με το άνοιγμα του αρχείου EDGE, ανοίγει η ιστοσελίδα, δηλαδή το αρχείο HTML.
