## Τι είναι ένα αρχείο OLM;

Ένα αρχείο με επέκταση .olm είναι ένα αρχείο Microsoft Outlook για λειτουργικό σύστημα Mac. Ένα αρχείο OLM αποθηκεύει μηνύματα email, περιοδικά, δεδομένα ημερολογίου και άλλους τύπους δεδομένων εφαρμογών. Αυτά είναι παρόμοια με τα αρχεία [PST](/el/email/pst/) που χρησιμοποιούνται από το Outlook στο λειτουργικό σύστημα Windows. Ωστόσο, τα αρχεία OLM που δημιουργούνται από το Outlook για Mac δεν μπορούν να ανοίξουν στο Outlook για Windows. Η μορφή αρχείου OLM εισήχθη με το Outlook 2011 για την αποθήκευση τοπικών δεδομένων του Outlook. Ένα μόνο αρχείο OLM μπορεί να περιέχει πολλά γραμματοκιβώτια, οργανώνοντας τα δεδομένα με τη μορφή φακέλων και υποφακέλων. Τα αρχεία OLM μπορούν να ανοίξουν/φορτωθούν στο Microsoft Outlook 365 και στο Outlook για Mac.

## Μορφή αρχείου OLM

Τα OLM είναι δυαδικά αρχεία με τύπο MIME εφαρμογής/οκτάδας ροής. Οι τεχνικές προδιαγραφές της μορφής αρχείου OLM δεν είναι διαθέσιμες ανοιχτά από τη Microsoft και δεν υπάρχουν διαθέσιμες λεπτομέρειες μορφής για αναφορά του προγραμματιστή. Ωστόσο, υπάρχουν ορισμένα API που μπορούν να διαβάσουν τη μορφή αρχείου OLM και να τα εγγράψουν σε ένα αρχείο PST για χρήση σε λειτουργικό σύστημα Microsoft Windows, όπως το Aspose.Email.
