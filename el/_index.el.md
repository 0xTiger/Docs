Μια μορφή αρχείου περιγράφει τη δομή των δεδομένων σε ένα αρχείο που ακολουθούν οι εφαρμογές για να διαβάσουν ή να δημιουργήσουν τέτοια αρχεία.

Οι τύποι αρχείων μπορούν να ομαδοποιηθούν σε κατηγορίες όπως διανυσματικές μορφές αρχείων, μορφές αρχείων υπολογιστικών φύλλων, μορφές αρχείων βίντεο, μορφές αρχείων εικόνας, μορφές αρχείων 3D και μια σειρά από άλλες. Ένα αρχείο μπορεί επίσης να μετατραπεί σε άλλη μορφή αρχείου, όπως BMP σε PNG.

## Λίστα μορφών αρχείων: Επεκτάσεις και οι σχετικές μορφές τους
Ακολουθεί μια λίστα με τις μορφές αρχείων που χρησιμοποιούνται συνήθως μαζί με τις επεκτάσεις.
