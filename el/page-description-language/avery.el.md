## Τι είναι ένα αρχείο AVERY;

Ένα αρχείο AVERY είναι ένα εκτυπώσιμο έγγραφο που περιέχει μία ή περισσότερες ετικέτες για σκοπούς εκτύπωσης. Δημιουργείται με το λογισμικό Avery Design & Print ή Design & Print Online. Οι ετικέτες μέσα σε ένα αρχείο AVERY μπορεί να αποτελούνται από διαφορετικά στοιχεία όπως κείμενο, γραφικά και πληροφορίες διάταξης σελίδας. Αυτές οι πληροφορίες χρησιμοποιούνται από εταιρείες εκτύπωσης για την εκτύπωση ποικιλίας στοιχείων σχεδίασης, όπως επαγγελματικές κάρτες, ετικέτες ονομάτων, ένθετα βιβλιοδεσίας, ετικέτες διευθύνσεων, ετικέτες δίσκων, πρότυπα μοιρολογιών και ευχετήριες κάρτες.

## Μορφή αρχείου AVERY - Περισσότερες πληροφορίες

Τα αρχεία AVERY αποθηκεύονται σε δυαδική μορφή αρχείου. Αυτά μπορούν να ανοίξουν με το λογισμικό Avery Design & Print ή Design & Print Online. Τα σχέδια που δημιουργούνται με το Design & Print μπορούν να χρησιμοποιηθούν για τη δημιουργία ετικετών AVERY που μπορούν να εξυπηρετήσουν διάφορους σκοπούς. Αυτά τα σχέδια μπορούν να αποθηκευτούν ως αρχεία [PDF](/el/pdf/) τα οποία μπορούν να μοιραστούν με την εταιρεία Prints για σκοπούς εκτύπωσης.

### Εκτύπωση αρχείου AVERY

Ένα αρχείο AVERY μπορεί είτε να αποθηκευτεί ως PDF είτε να εκτυπωθεί απευθείας σε οποιονδήποτε τυπικό εκτυπωτή. Για να εκτυπώσετε το αρχείο, ανοίξτε το αρχείο στο λογισμικό Design & Print και χρησιμοποιήστε την επιλογή Print από το μενού File για να στείλετε το αρχείο στον εκτυπωτή για εκτύπωση.
