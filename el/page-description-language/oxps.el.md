## Τι είναι ένα αρχείο OXPS; ##

Η μορφή αρχείου OXPS είναι γνωστή ως Open XML Paper Specification. Είναι μια γλώσσα περιγραφής σελίδας και μια μορφή εγγράφου. Η Microsoft είναι ο προγραμματιστής αυτής της μορφής. Η μορφή αρχείου OXPS είναι πολύ γνωστή σε αυτά τα αρχεία PDF. Η ρύθμιση του αρχείου OXPS είναι ίδια με μια μορφή XPS. Η μόνη διαφορά είναι η περιγραφή του OXPS ως ανοιχτής μορφής. Η επέκταση αρχείου .oxps είναι η πιο πρόσφατη έκδοση της μορφής αρχείου XPS που δημιουργήθηκε από τη Microsoft και την Ecma International. Αν μιλάμε για τις προδιαγραφές, τα αρχεία OXPS και XPS χρησιμοποιούνται για τη δημοσίευση, την κοινή χρήση και την κατανομή εγγράφων σταθερής διάταξης. Ως υποκατάστατο των αρχείων PDF, η μορφή αρχείου OXPS μπορεί να αποθηκεύσει δεδομένα κειμένου και πολυμέσων, με χαρακτηριστικά διάταξης και χαρακτηριστικά διάταξης εκτύπωσης.

## Σύντομη Ιστορία ##

Η ομάδα της Microsoft επέλεξε την Global Graphics για την παροχή συμβουλών στο XPS το 2003 και η ομάδα ανάπτυξης συνεργάστηκε στενά με τις ομάδες των Windows για την ανάπτυξη των προδιαγραφών και της αρχιτεκτονικής αναφοράς για τη νέα μορφή. Αναγνωρίζοντας τις προδιαγραφές XPS, τον Ιούνιο του 2007 η Ecma International δημιούργησε την Τεχνική Επιτροπή 46 (TC46) για την ανάπτυξη ενός προτύπου με βάση την προδιαγραφή OpenXPS. Η Ecma International εγκεκριμένη Open XML Paper Specification είναι ένα πρότυπο Ecma (ECMA-388) στην 97η Γενική Συνέλευση που πραγματοποιήθηκε στη Βουδαπέστη στις 16 Ιουνίου 2009.
