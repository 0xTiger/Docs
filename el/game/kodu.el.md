## Τι είναι ένα αρχείο KODU;

Ένα αρχείο .kudo είναι ένα αρχείο 3D Παιχνιδιού που δημιουργήθηκε και αποθηκεύτηκε με το Microsoft [λογισμικό Kodu Game Lab](https://www.kodugamelab.com/) που είναι ένα περιβάλλον ανάπτυξης εκπαιδευτικών παιχνιδιών. Έχει σχεδιαστεί για να διδάσκει στα παιδιά βασικές αρχές προγραμματισμού και επιτρέπει στους δημιουργούς να προγραμματίζουν τις συμπεριφορές των στοιχείων του παιχνιδιού χρησιμοποιώντας οπτική γλώσσα προγραμματισμού. Τα αρχεία Kodu μπορούν να αποθηκευτούν και να μοιραστούν με άλλους χρήστες, αλλά μπορούν να ανοίξουν και να εκτελεστούν μόνο στο λογισμικό Kodu.

## Μορφή αρχείου KODU - Περισσότερες πληροφορίες

Τα αρχεία Kodu είναι δυαδικά αρχεία, αποθηκευμένα σε ιδιόκτητη μορφή αρχείου της Microsoft. Οι προδιαγραφές μορφής αρχείου του δεν είναι διαθέσιμες δημόσια για προγραμματιστές και γι' αυτό τα αρχεία Kodu μπορούν να εκτελεστούν μόνο μέσα στο λογισμικό Kodu Game Lab.
