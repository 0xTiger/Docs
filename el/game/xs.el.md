## Τι είναι ένα αρχείο XS;

Ένα αρχείο XS είναι ένα αρχείο αναφοράς που χρησιμοποιείται στο Age of Empires III (AOE3) για τη δημιουργία τεχνητής νοημοσύνης (AI). Χρησιμοποιείται κατά τη διαδικασία μεταγλώττισης όταν φορτώνεται ένα προσαρμοσμένο αρχείο χάρτη XML. Εάν έχετε παίξει AOE3, γνωρίζετε ήδη ότι όχι μόνο μπορείτε να επιλέξετε οποιονδήποτε χάρτη για αναπαραγωγή, αλλά μπορείτε επίσης να δημιουργήσετε τους δικούς σας χάρτες χρησιμοποιώντας το πρόγραμμα Random Map Script.

## Πώς να δημιουργήσετε χάρτη AI στο Age of Empires 3;

 * Ανοίξτε ένα πρόγραμμα επεξεργασίας κειμένου, όπως το Microsoft Notepad, το Notepad++ ή το TextEdit. Αυτό θα χρησιμοποιηθεί για τον προγραμματισμό του χάρτη σας.
 * Προγραμματίστε όλες τις δυνατότητες του χάρτη σας χρησιμοποιώντας τη γλώσσα Random Map Script.
 * Μπορείτε να ανατρέξετε σε σεμινάρια προγραμματισμού για να βεβαιωθείτε ότι τον προγραμματίζετε σωστά
 * Αποθηκεύστε το αρχείο σας ως MyAOE3.xs και μετακινήστε το στους φακέλους παιχνιδιών Age of Empires 3.

### Μορφή αρχείου XS - Περισσότερες πληροφορίες

Τα αρχεία XS αποθηκεύονται ως αρχεία απλού κειμένου που μπορούν να ανοίξουν και να επεξεργαστούν με οποιοδήποτε πρόγραμμα επεξεργασίας κειμένου.
