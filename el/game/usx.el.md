## Τι είναι ένα αρχείο USX;

Ένα αρχείο USX περιέχει τα πλέγματα του Unreal Engine που είναι προσχεδιασμένα τρισδιάστατα αντικείμενα/μοντέλα που χρησιμοποιούνται για τη δημιουργία του περιβάλλοντος. Τα μοντέλα από το αρχείο USX έχουν διαφορετική θέση, προσανατολισμό και μεγέθη, αλλά αυτά παραμένουν στατικά. Τα Static Meshes μπορούν να βελτιστοποιηθούν για απόδοση και μπορούν να χρησιμοποιηθούν για τη δημιουργία διαφόρων επιπέδων λεπτομέρειας (LOD) για τη βελτίωση της απόδοσης του παιχνιδιού. Μπορούν επίσης να χρησιμοποιηθούν για τη δημιουργία όγκων σύγκρουσης και μπορούν να χρησιμοποιηθούν ως μέρος προσομοιώσεων φυσικής.

Τα αρχεία USX μπορούν να δημιουργηθούν με το πρόγραμμα επεξεργασίας επιπέδου Unreal UnrealEd.

## Μορφή αρχείου USX - Περισσότερες πληροφορίες

Τα αρχεία USX αποθηκεύονται στο δίσκο ως δυαδικά αρχεία. Οι λεπτομέρειες της εσωτερικής δομής αρχείου της μορφής αρχείου USX δεν είναι διαθέσιμες δημόσια για αναφορά του προγραμματιστή.
