Οι ROM παιχνιδιών, που είναι αντίγραφα λογισμικού παιχνιδιών προχωρημένης ηλικίας που έχουν αναπτυχθεί για άλλα παλιά συστήματα, εμπίπτουν επίσης στην κατηγορία αρχείων παιχνιδιών.
