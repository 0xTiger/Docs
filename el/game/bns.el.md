## Τι είναι ένα αρχείο BNS;

Ένα αρχείο με επέκταση .bns είναι ένα αρχείο σεναρίου παιχνιδιού που δημιουργήθηκε με το παιχνίδι επίλυσης γρίφων Portal Map που αναπτύχθηκε από τη Valve. Περιέχει σενάριο κειμένου για τον ορισμό πληροφοριών σχετικά με έναν χάρτη πύλης που αποθηκεύεται ως αρχείο .bsp. Τα αρχεία BNS χρησιμοποιούνται ως αναφορά σε αυτό το πραγματικό αρχείο χάρτη και περιέχει μια λίστα προκλήσεων που πρέπει να ολοκληρωθούν. Επιπλέον, περιέχει όνομα χάρτη, σχόλια σχετικά με τον χάρτη και αναφορές αρχείου μικρογραφιών εικόνας. Ένα αρχείο BNS μπορεί να ανοίξει σε οποιοδήποτε πρόγραμμα επεξεργασίας κειμένου όπως το Notepad++, το textEdit και το Notepad.

## Μορφή αρχείου BNS

Τα αρχεία BNS αποθηκεύονται ως αρχεία απλού κειμένου που είναι αναγνώσιμα από τον άνθρωπο. Αυτά μπορούν να ανοίξουν σε προγράμματα επεξεργασίας κειμένου και να επεξεργαστούν επίσης. Ωστόσο, αυτά θα πρέπει να επεξεργάζονται προσεκτικά για να αποφευχθούν τυχόν σφάλματα στο σενάριο.

## Παράδειγμα μορφής αρχείου BNS

Ένα αρχείο BNS μπορεί να μοιάζει με το εξής όταν ανοίγει σε ένα πρόγραμμα επεξεργασίας κειμένου όπως το Notepad++.

```
"#Bonus_Map_Challenges"
{
	"map"		"testchmb_a_08"
	"chapter"	"chapter5.cfg"	[$X360]
	"image"		"bonusmaps/testchmb_a_08_challenges"
	"comment"	"#Bonus_Map_ChallengesComment"
	"lock"		"0"

	"challenges"
	{
		"#Bonus_Map_ChallengePortals"
		{
			"comment"	"#Bonus_Map_LeastPortalsComment"

			"bronze"	"9"
			"silver"	"5"
			"gold"		"4"
		}
		"#Bonus_Map_ChallengeSteps"
		{
			"comment"	"#Bonus_Map_LeastStepsComment"

			"bronze"	"30"
			"silver"	"20"
			"gold"		"10"
		}
		"#Bonus_Map_ChallengeTime"
		{
			"comment"	"#Bonus_Map_LeastTimeComment"

			"bronze"	"40"
			"silver"	"30"
			"gold"		"19"
		}
	}
}
```

## Μέρη ενός αρχείου BNS

Στο παραπάνω παράδειγμα,

* `#Bonus_Map_Challenges` - Αντιπροσωπεύει το όνομα του χάρτη.
* `χάρτης` - Αντιπροσωπεύει το όνομα του χάρτη που θα τοποθετηθεί στο φάκελο χαρτών του παιχνιδιού
* `κεφάλαιο` - Αντιπροσωπεύει το κεφάλαιο στο οποίο ανήκει ο χάρτης. Όλα τα αρχεία κεφαλαίων βρίσκονται στο αρχείο VPK προσθέτοντας το όνομα του χάρτη στη μορφή «map your_map_name»
* «Εικόνα» - Περιέχει τη μικρογραφία του χάρτη και αποθηκεύεται στη διαδρομή ρίζας steam\steamapps\common\portal\portal\materials\VGUI.
* `Σχόλιο` - Περιγραφικό κείμενο για τον χάρτη που δημιουργήθηκε.
