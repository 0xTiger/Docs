## Τι είναι ένα αρχείο RPX;

Ένα αρχείο RPX είναι ένα αρχείο παιχνιδιού που μπορεί να φορτωθεί ή να παιχτεί από την κονσόλα παιχνιδιών Wii U. Μπορεί επίσης να φορτωθεί και να παίξει στον εξομοιωτή βιντεοπαιχνιδιών Cemu που είναι δημοφιλής για την παροχή περιβάλλοντος εξομοιωτή για αρχεία Wii U. Τα αρχεία παιχνιδιών Wii U είναι μεγάλου μεγέθους και μπορούν επίσης να αποθηκευτούν ως αρχεία [WUD](/el/disc-and-media/wud/) ή [WUX](/el/compression/wux/) για φόρτωση από εξομοιωτές παιχνιδιών. Ένα αρχείο RPX περιέχει ολόκληρο το εκτελέσιμο παιχνίδι που αποτελείται από όλα τα στοιχεία του παιχνιδιού, όπως γραφικά, υφές, ακολουθίες και τη ROM του παιχνιδιού.

## Μορφή αρχείου RPX - Περισσότερες πληροφορίες

Τα αρχεία RPX αποθηκεύονται σε δίσκο σε ιδιόκτητη μορφή αρχείου. Ορισμένα API είναι διαθέσιμα για τη συμπίεση ή την αποσυμπίεση αρχείων RPX, όπως το [wiiurpxtool](https://github.com/0CBH0/wiiurpxtool). Επιπλέον, εφαρμογές όπως ο εξομοιωτής βιντεοπαιχνιδιών Cemu σάς επιτρέπουν να παίζετε παιχνίδια Wii U στα Windows σε υπολογιστή.
