## Τι είναι ένα αρχείο SCR;
Ένα αρχείο με επέκταση .scr είναι ένα αρχείο προφύλαξης οθόνης που χρησιμοποιείται από το λειτουργικό σύστημα Microsoft Windows. Περιλαμβάνει κινούμενα σχέδια, γραφικά, προβολή διαφανειών ή βίντεο που μπορούν να χρησιμοποιηθούν ως προφύλαξη οθόνης των Windows. Τα αρχεία SCR αποθηκεύονται συνήθως στον κύριο κατάλογο των Microsoft Windows. Οι προφυλάξεις οθόνης υποτίθεται ότι αποτρέπουν τις οθόνες υπολογιστών CRT ή plasma από μια πάθηση που εμφανίζεται όταν η οθόνη εμφανίζει την ίδια εικόνα για πολύ μεγάλο χρονικό διάστημα. Αν και οι πιο πρόσφατες οθόνες δεν υποφέρουν σε τέτοια κατάσταση, αλλά οι προφύλαξη οθόνης εξακολουθούν να χρησιμοποιούνται για την αποτροπή της οθόνης για λόγους ασφαλείας.

## Μορφή αρχείου SCR
Η προφύλαξη οθόνης είναι ένα πρόγραμμα υπολογιστή που το γεμίζει με κινούμενες εικόνες ή μοτίβα όταν δεν εκτελείται καμία δραστηριότητα σε έναν υπολογιστή για μεγάλο χρονικό διάστημα. Οι προφυλάξεις οθόνης εισήχθησαν για να αποτρέψουν την καύση φωσφόρου σε οθόνες υπολογιστών πλάσματος, καθοδικού σωλήνα (CRT) και OLED. Οι προφυλάξεις οθόνης συνήθως ρυθμίζονται για να εφαρμόζουν ένα βασικό επίπεδο ασφάλειας, απαιτώντας κωδικό πρόσβασης για να ανοίξετε ξανά τη συσκευή. Οι προφύλαξη οθόνης συνήθως αναπτύσσονται και κωδικοποιούνται χρησιμοποιώντας διάφορες γλώσσες προγραμματισμού καθώς και διεπαφές γραφικών. Κυρίως οι προγραμματιστές προφύλαξης οθόνης χρησιμοποιούν τις γλώσσες προγραμματισμού C ή C++, μαζί με γραφικές βιβλιοθήκες ή GDI, όπως το OpenGL, το οποίο λειτουργεί σε πολλές πλατφόρμες με δυνατότητα τρισδιάστατης απόδοσης. Η έξοδος της προφύλαξης οθόνης αποθηκεύεται ως φορητό εκτελέσιμο αρχείο.

### Χρήση αρχείου SCR
Στις παλιές οθόνες CRT ή με βάση το πλάσμα αναφέρθηκε η καύση της οθόνης επειδή η ίδια εικόνα εμφανιζόταν στην οθόνη για μεγάλο χρονικό διάστημα. Το έγκαυμα οθόνης είναι μια περίπτωση όταν οι ιδιότητες των εκτεθειμένων περιοχών της επικάλυψης φωσφόρου στο εσωτερικό της οθόνης αλλάζουν σταδιακά και τελικά οδηγούν σε μια σκοτεινή σκιώδη εικόνα στην οθόνη. Έτσι, οι προφυλάξεις οθόνης έπρεπε να αλλάζουν συνεχώς την εικόνα της οθόνης και συνήθως τα αρχεία .scr ήταν απαραίτητα για τις οθόνες των μηχανημάτων έκδοσης εισιτηρίων ΑΤΜ ή Σιδηροδρόμων. Αργότερα, οι οθόνες LCD και οι πιο προηγμένες εκδόσεις οθονών έλυσαν το πρόβλημα. Ως εκ τούτου, οι προφύλαξη οθόνης εξακολουθούν να χρησιμοποιούνται στη σύγχρονη εποχή για την προστασία των συσκευών αδράνειας από τη χρήση δεύτερου προσώπου. Απαιτεί τον κωδικό πρόσβασης ή το μοτίβο για την εκ νέου πρόσβαση στη συσκευή.

## Δημιουργία προφύλαξης οθόνης με χρήση C#
Παρόλο που μπορούμε να δημιουργήσουμε μια προφύλαξη οθόνης σε οποιαδήποτε από τις γλώσσες προγραμματισμού .NET, εδώ δίνεται η γλώσσα προγραμματισμού C#:

```
class MyCoolScreensaver : Screensaver
{
   public MyCoolScreensaver()
   {
      Initialize += new EventHandler(MyCoolScreensaver_Initialize);
      Update += new EventHandler(MyCoolScreensaver_Update);
      Exit += new EventHandler(MyCoolScreensaver_Exit);
   }

   void MyCoolScreensaver_Initialize(object sender, EventArgs e)
   {
   }

   void MyCoolScreensaver_Update(object sender, EventArgs e)
   {
      Graphics0.Clear(Color.Black);
      Graphics0.DrawString(
         DateTime.Now.ToString(),
         SystemFonts.DefaultFont, Brushes.Blue,
         0, 0);
   }

   void MyCoolScreensaver_Exit(object sender, EventArgs e)
   {
   }

   [STAThread]
   static void Main()
   {
      Screensaver ss = new MyCoolScreensaver();
      ss.Run();
   }
}
```
Αλλάξτε την επέκταση του εκτελέσιμου αρχείου από .exe σε .scr. Έτσι, το αρχείο SCR μπορεί να ονομαστεί ως ScreenSaver.scr.
