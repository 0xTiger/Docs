## Τι είναι ένα αρχείο ADMX;

Ένα αρχείο ADMX είναι ένα αρχείο διαμόρφωσης πολιτικής που χρησιμοποιείται από το λογισμικό Microsoft Windows Group Policy που διαχειρίζεται ομάδα υπολογιστών σε μια ομάδα. Είναι ένα αρχείο προτύπου διαχείρισης που βασίζεται σε XML και παρουσιάστηκε με το Windows Vista Service Pack 1. Τα αρχεία ADMX περιέχουν ρυθμίσεις διαμόρφωσης για λογαριασμούς χρηστών, διαμορφώσεις λειτουργικού συστήματος και εφαρμογές. Τα αρχεία ADMX χρησιμοποιούνται για την αποθήκευση και την ανάπτυξη των διαμορφώσεων για την κεντρική διαχείριση πολλών συστημάτων υπολογιστών.

Μπορείτε να δημιουργήσετε ή να επεξεργαστείτε αρχεία ADMX χρησιμοποιώντας οποιοδήποτε συμβατό με XML, οποιοδήποτε πρόγραμμα επεξεργασίας κειμένου ή πρόγραμμα επεξεργασίας αντικειμένων πολιτικής ομάδας της Microsoft.

## Μορφή αρχείου ADMX - Περισσότερες πληροφορίες

Τα αρχεία ADMX αποθηκεύονται στην καθολική μορφή αρχείου [XML](/el/web/xml/) που είναι αναγνώσιμη από τον άνθρωπο. Είναι πολύγλωσση μορφή αρχείου και επιτρέπει σε διαχειριστές συστήματος από διαφορετικές γλώσσες να εργάζονται με τα ίδια αρχεία ADMX. Τα αρχεία ADMX αντικατέστησαν την παλιά μορφή αρχείου [ADM](/el/system/adm/), η οποία είναι μορφή αρχείου κειμένου με μορφοποίηση unicode. Τα αρχεία ADMX καθορίζουν τα κλειδιά μητρώου που αλλάζουν όταν αλλάζει μια συγκεκριμένη ρύθμιση πολιτικής ομάδας.

### Τι μπορώ να κάνω με το αρχείο ADMX;

Τα αρχεία ADMX καθορίζουν ποιες ενέργειες πρέπει να επιτρέπονται σε υπολογιστές χρηστών που αποτελούν μέρος μιας ομάδας. Η πολιτική ομάδας επιβάλλει τους κανόνες που ορίζονται σε συνδυασμό με το λογισμικό Active Directory. Η διαχείριση των αρχείων ADMX γίνεται από το κεντρικό κατάστημα στο λειτουργικό σύστημα Windows που είναι μια κεντρική τοποθεσία στον τομέα.

Ένα αρχείο ADMX που αναπτύσσεται σε περιβάλλον εργασίας μπορεί να επιτρέψει στους διαχειριστές να εφαρμόσουν πολιτικές όπως:

* Ελέγξτε τη χρήση του διαδικτύου
* Λήψη ορισμένων τύπων αρχείων
* Χρήση αφαιρούμενων συσκευών στα συστήματα
