## Τι είναι το 4ο αρχείο;

Ένα αρχείο με επέκταση αρχείου .4 είναι ένα αρχείο προγραμματισμού που δημιουργήθηκε για την **Τέταρτη Γλώσσα Προγραμματισμού**. Περιέχει πηγαίο κώδικα για προγράμματα γραμμένα στη γλώσσα προγραμματισμού Forth και παράγει την έξοδο κατά την εκτέλεση του προγράμματος. Είναι απλώς ένα άλλο αρχείο γλώσσας προγραμματισμού παρόμοιο με το αρχείο προέλευσης [C](/el/programming/c/) και [C++](/el/programming/cpp/).

## 4η Μορφή αρχείου - Περισσότερες πληροφορίες


### Παράδειγμα κώδικα 4ης γλώσσας προγραμματισμού

Ακολουθεί ένα παράδειγμα απλού προγράμματος γραμμένου στο Forth που υπολογίζει το παραγοντικό ενός δεδομένου αριθμού:

```
: factorial ( n -- n! )
   dup 0=
   if
      drop 1
      exit
   then
   1
   swap
   begin
      dup 1-
      dup 0=
      if
         drop
         exit
      then
      swap
      *
   repeat
;

```

Σε αυτό το παράδειγμα, η παραγοντική λέξη παίρνει ένα μόνο όρισμα n και επιστρέφει το παραγοντικό της. Η λέξη dup αντιγράφει την τιμή πάνω από τη στοίβα, η drop απορρίπτει την τιμή πάνω από τη στοίβα και * πολλαπλασιάζει τις δύο τιμές πάνω από τη στοίβα. Οι κατασκευές if και start/until ελέγχουν τη ροή του προγράμματος.

Για να χρησιμοποιήσετε αυτό το πρόγραμμα, θα πρέπει να εισάγετε τον ορισμό στον διερμηνέα Forth και στη συνέχεια να καλέσετε την παραγοντική λέξη με τον αριθμό που θέλετε να βρείτε το παραγοντικό του:

```
10 factorial .
```
Αυτό θα είχε ως αποτέλεσμα την έξοδο 3628800, που είναι το παραγοντικό του 10.
