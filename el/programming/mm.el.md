## Τι είναι ένα αρχείο MM;

Ένα αρχείο MM είναι ένα αρχείο πηγαίου κώδικα που μπορεί να περιέχει κώδικα προγραμματισμού Objective-C και Objective-C++. Χρησιμοποιείται σε MacOS και διαφοροποιείται από τα αρχεία .M που περιέχουν μόνο κώδικα Objective-C. Όταν ο μεταγλωττιστής δει την επέκταση .mm, μπορεί να μεταγλωττίσει τις κλάσεις C++ που υπάρχουν στο αρχείο. Δεδομένου ότι ένα μεμονωμένο αρχείο πηγαίου κώδικα μπορεί να περιέχει έναν συνδυασμό γλωσσών [C](/el/programming/c/) και [C++](/el/programming/cpp/), η μετονομασία των αρχείων .m σε επέκταση .mm επιτρέπει τη ρύθμιση των χαρακτηριστικών της C++ in. Οι εφαρμογές που μπορούν να ανοίξουν αρχείο .MM περιλαμβάνουν Apple TextEdit, Microsoft Notepad, Notepad++ και Microsoft Wordpad.
