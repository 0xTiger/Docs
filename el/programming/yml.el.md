## Τι είναι ένα αρχείο YML;

Το YML είναι ένα αρχείο κειμένου σε γλώσσα σειριοποίησης δεδομένων, [YAML](/el/programming/yaml/) (Ακόμα μια άλλη γλώσσα σήμανσης), που χρησιμοποιείται συχνά για τη δημιουργία αρχείων διαμόρφωσης. Μπορεί να λειτουργήσει σε συνδυασμό με οποιαδήποτε γλώσσα προγραμματισμού, συμπεριλαμβανομένων των C/C++, Ruby, Python, Java, PHP, C# και άλλων. Όντας ανεξάρτητο από οποιαδήποτε συγκεκριμένη γλώσσα προγραμματισμού, είναι ανοιχτό, διαλειτουργικό και αναγνώσιμο από τον άνθρωπο. Τα αρχεία YML μπορούν να ανοίξουν και να επεξεργαστούν με κοινά προγράμματα επεξεργασίας κειμένου όπως το Microsoft Notepad, το Notepad++, το Github Atom, το Apple TextEdit και το Visual Studio IDE.

## Μορφή αρχείου YML - Περισσότερες πληροφορίες

Τα αρχεία YML αποθηκεύουν τις ρυθμίσεις της εφαρμογής ως διαμόρφωση. Η βασική του δομή είναι ένας χάρτης που μπορεί επίσης να ονομαστεί κατάλογος ή κατακερματισμός. Οι καταχωρήσεις σε ένα αρχείο YML αποτελούνται από ένα ζεύγος κλειδιών και τιμών. Κυρίως, τα αρχεία διαμόρφωσης .yml αποθηκεύονται ως "configure.yml", αν και ορισμένες γλώσσες προγραμματισμού, όπως η Ruby on Rails, αποθηκεύουν τη διαμόρφωση της συνδεσιμότητας βάσης δεδομένων σε αρχεία "database.yml". [Προδιαγραφές μορφής αρχείου YML/YAML](https://yaml.org/spec/1.2.2/) εξηγεί τις πλήρεις δομές γλώσσας και τις πληροφορίες σχετικά με τη σύνταξη.
