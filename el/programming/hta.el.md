## Τι είναι ένα αρχείο HTA;

Το HTMLA σημαίνει **Hypertext Markup Language Application** είναι ένα πρόγραμμα που είναι συμβατό με τα Microsoft Windows. Ο πηγαίος κώδικας αυτού του προγράμματος περιλαμβάνει περισσότερες από μία γλώσσες δέσμης ενεργειών όπως [HTML](/el/web/html/) και [JavaScript](/el/web/js/). Για τη διεπαφή χρήστη, προτιμάται μια εφαρμογή HTML ενώ για την εκπλήρωση της απαίτησης της λογικής του προγράμματος χρησιμοποιείται οποιαδήποτε άλλη γλώσσα δέσμης ενεργειών.

Μια εφαρμογή HTML είναι ανεξάρτητη από το μοντέλο ασφαλείας του προγράμματος περιήγησης στο Διαδίκτυο και εκτελείται ως μια πλήρως αξιόπιστη εφαρμογή. Η επέκταση που χρησιμοποιείται για αρχεία σχετικά με αυτές τις εφαρμογές είναι HTA. Αυτές οι εφαρμογές περιλαμβάνουν χαρακτηριστικά HTML μαζί με ιδιότητες άλλων γλωσσών δέσμης ενεργειών.


## Σύντομη Ιστορία ##

Το HTA παρουσιάστηκε για πρώτη φορά το 1999 από τη Microsoft μαζί με την κυκλοφορία του Internet Explorer 5. Ήταν συμβατό με τον Internet Explorer και έτσι μπορούσε να εκτελεστεί μόνο σε λειτουργικό σύστημα Windows. Αυτή η τεχνολογία κατοχυρώθηκε με δίπλωμα ευρεσιτεχνίας το 2003. Τα αρχεία HTA εκτελούνται όπως όλα τα άλλα αρχεία .exe. Τα αρχεία HTA είναι συμβατά και με τη σημερινή ενημερωμένη έκδοση των Windows 11.


## Τεχνική προδιαγραφή ##

Τα HTA έχουν την ίδια μορφή με οποιαδήποτε άλλη σελίδα HTML που αποτελείται, ενώ ορισμένα χαρακτηριστικά χρησιμοποιούνται για τον έλεγχο των στυλ περιγραμμάτων ή εικονιδίων προγραμμάτων. Επιπλέον, παρέχονται επιχειρήματα για την έναρξη της HTA. Αυτές οι εφαρμογές μπορούν να εκτελεστούν χρησιμοποιώντας ένα πρόγραμμα που ονομάζεται mshta.exe. Μπορείτε να προσπελάσετε απλά κάνοντας διπλό κλικ στο αρχείο. Αυτά τα προγράμματα εκτελούνται αυτόματα μαζί με τον Internet Explorer. Εκτός από άλλες προδιαγραφές, αυτές δεν είναι ανεξάρτητες από το πρόγραμμα περιήγησης του κινητήρα Trident αλλά είναι ανεξάρτητες από τον Internet Explorer. Σημαίνει ότι αυτά μπορούν να εκτελεστούν χωρίς τη χρήση του Internet Explorer.

Οι ετικέτες χρησιμοποιούνται για λόγους προσαρμογής της εμφάνισης αυτών των εφαρμογών. Η μετατροπή από την εφαρμογή Microsoft HTML σε μορφή HTA είναι πιο εύκολη, δηλαδή χρειάζεται μόνο να αλλάξετε την επέκταση. Καθώς γνωρίζουμε ότι αυτές οι εφαρμογές είναι πλήρως αξιόπιστες, επομένως αυτές περιλαμβάνουν περισσότερες δυνατότητες και πλεονεκτήματα σε σύγκριση με απλά αρχεία HTML. Οι επεξεργαστές κειμένου μπορούν να χρησιμοποιηθούν για τη δημιουργία HTA. Αυτά τα προγράμματα επεξεργασίας μπορούν να αποκτηθούν από τη Microsoft ή οποιαδήποτε άλλη αξιόπιστη πηγή.


## Παράδειγμα μορφής αρχείου HTA ##

```
<HTML>
<HEAD>
<HTA:APPLICATION ID="HelloExample" 
   BORDER="bold" 
   BORDERSTYLE="complex"/>
<TITLE>HTA - Hello World</TITLE>
</HEAD>
<BODY>
<H2>HTA - Hello World</H2>
</BODY>
</HTML>

```
