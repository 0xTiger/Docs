## Τι είναι ένα αρχείο LGO;

Ένα LGO αρχείο ενός κώδικα προγράμματος γραμμένο με την εφαρμογή Logo για τη σχεδίαση εικόνων. Περιέχει προγραμματισμένες οδηγίες που καθοδηγούν την εφαρμογή ανάγνωσης να κάνει σχέδια σύμφωνα με αυτές τις οδηγίες. Για παράδειγμα, το πρόγραμμα "χελώνα" που μπορεί να ανοίξει αρχεία LGO, χρησιμοποιήστε αυτές τις οδηγίες για να σχεδιάσετε γραμμές σύμφωνα με τον προγραμματισμένο κώδικα. Το σχέδιο δεν περιορίζεται μόνο σε γραμμές. Τα αρχεία LGO μπορούν να χρησιμοποιηθούν για τη σχεδίαση απλών έως πολύπλοκων σχεδίων. Πολλά εκπαιδευτικά προγράμματα χρησιμοποίησαν αρχεία LGO για τη διδασκαλία του τρόπου δημιουργίας σχεδίων μέσω προγραμματισμού.

## Μορφή αρχείου LGO

Τα αρχεία LGO αποθηκεύονται σε ιδιόκτητη μορφή αρχείου και η εσωτερική δομή των αρχείων τους δεν είναι γνωστή δημόσια. Μια άλλη πολύ γνωστή μορφή αρχείου για λογότυπα είναι τα αρχεία [.ico](/el/image/ico/).
