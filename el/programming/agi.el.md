## Τι είναι ένα αρχείο AGI;

Ένα αρχείο AGI είναι ένα αρχείο σεναρίου που χρησιμοποιείται από το σύστημα τηλεφωνίας ανοιχτού κώδικα, το Asterisk. Περιέχει πληροφορίες που μπορούν να χρησιμοποιηθούν για την αυτοματοποίηση των διαδικασιών και το σχέδιο κλήσης με Asterisk. Χρησιμοποιώντας αρχεία σεναρίων AGI, μπορούν να δημιουργηθούν συνδέσεις με σχεσιακές βάσεις δεδομένων όπως η PostgreSQL ή η MySQL. Επιπλέον, τα σενάρια AGI μπορούν να χρησιμοποιηθούν για πρόσβαση και σε άλλους εξωτερικούς πόρους. Τα σενάρια AGI μπορούν να μετατρέψουν τον έλεγχο του σχεδίου κλήσης σε μια εξωτερική δέσμη ενεργειών AGI, επιτρέποντας στο Asterisk να εκτελεί πολύπλοκες εργασίες.

## Μορφή αρχείου AGI - Περισσότερες πληροφορίες

Τα αρχεία σεναρίων AGI αποθηκεύονται ως αρχεία κειμένου και μπορούν να ανοίξουν σε οποιοδήποτε πρόγραμμα επεξεργασίας κειμένου για την πραγματοποίηση αλλαγών.

### Πρότυπο μοτίβο επικοινωνίας AGI

Η δέσμη ενεργειών AGI φορτώνει τις μεταβλητές και τις τιμές τους που αποστέλλονται σε αυτό από το Asterisk. Μια κοινή εμφάνιση αυτών των μεταβλητών είναι η εξής.

```
agi_request: test.py
agi_channel: Zap/1-1
agi_language: en
agi_callerid:
agi_context: default
agi_extension: 123
agi_priority: 2
```

Αυτές οι μεταβλητές ακολουθούνται από μια κενή γραμμή που αποστέλλεται από το Asterisk, που δείχνει ότι το σενάριο AGI μπορεί να αναλάβει τον έλεγχο του σχεδίου κλήσης τώρα.

### Γλώσσα προγραμματισμού για AGI Script Files

Τα σενάρια AGI μπορούν συνήθως να γραφτούν σε Perl, [PHP](/el/programming/php/), [C](/el/programming/c/), Pascal ή Bourne Shell.
