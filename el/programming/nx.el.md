## Τι είναι ένα αρχείο NX;

Ένα αρχείο NX είναι ένα αρχείο βιντεοπαιχνιδιού που δημιουργήθηκε για να χρησιμοποιηθεί με την εικονική κονσόλα παιχνιδιών, **LowRes NX**. Είναι γραμμένο στη γλώσσα προγραμματισμού BASIC και περιέχει τον πηγαίο κώδικα του παιχνιδιού που καθορίζει τον τρόπο λειτουργίας του παιχνιδιού. Μπορείτε να δημιουργήσετε τα δικά σας αρχεία NX για χρήση με την εφαρμογή που μπορούν να φορτωθούν και οι χρήστες να παίξουν το παιχνίδι.

Μπορείτε να ανοίξετε ένα αρχείο NX σε ένα πρόγραμμα επεξεργασίας κειμένου, το LowRes NX ή οποιοδήποτε πρόγραμμα επεξεργασίας πηγαίου κώδικα, όπως το Microsoft Visual Studio Code.

## Μορφή αρχείου NX

Ένα αρχείο NX αποθηκεύεται στο δίσκο ως αρχείο απλού κειμένου που περιέχει τις οδηγίες του παιχνιδιού. Κάθε αρχείο NX έχει τον πηγαίο κώδικα του παιχνιδιού που αποτελείται από πολλές καταχωρήσεις ROM κασετών. Αυτές οι καταχωρήσεις ROM έχουν πληροφορίες στοιχείων παιχνιδιού, όπως γραφικά, χάρτες επιπέδων, ήχο και άλλα δεδομένα μη πηγαίου κώδικα για το παιχνίδι. Οι καταχωρήσεις ROM ενός προγράμματος μπορούν να προστεθούν ή να επεξεργαστούν χρησιμοποιώντας προγράμματα επεξεργασίας NX που συνοδεύουν το LowRes NX.

## Πώς να δημιουργήσετε αρχεία NX;

### Δημιουργία αρχείων NX σε υπολογιστή

Μπορείτε να δημιουργήσετε αρχεία NX σε υπολογιστή χρησιμοποιώντας:

* Επεξεργαστές κειμένου - όπως το Microsoft Notepad, το Notepad++ ή το Github Atom
* Επεξεργαστές πηγαίου κώδικα - όπως το Microsoft VIsual Studio Code

### Δημιουργήστε αρχεία NX σε iOS

Το LowRes NX διαθέτει έκδοση iOS που μπορεί να χρησιμοποιηθεί από προγραμματιστές για τη δημιουργία αρχείων NX μέσα από αυτήν την εφαρμογή.
