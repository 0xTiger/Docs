## Τι είναι ένα αρχείο APA;

Ένα αρχείο με επέκταση .apa είναι ένα αρχείο έργου ανάπτυξης λογισμικού που δημιουργήθηκε από την εφαρμογή RSView. Περιέχει πληροφορίες που απαιτούνται για τη δημιουργία εφαρμογών Human Machine Interface (HMI). Τα αρχεία APA αποθηκεύονται σε μορφή αρχείου αρχειοθέτησης. Η μορφή αρχείου του αρχείου βοηθά στη μεταφορά τους από έναν υπολογιστή σε άλλον εύκολα λόγω του μικρότερου μεγέθους αρχείου. Τα τυπικά βοηθητικά προγράμματα αποσυμπίεσης μπορούν να χρησιμοποιηθούν για την εξαγωγή των περιεχομένων των αρχείων APA. Τα αρχεία APA μπορούν να ανοίξουν χρησιμοποιώντας το λογισμικό Rockwell Automation RSView Studio ή RSView ME.

## Μορφή αρχείου APA - Περισσότερες πληροφορίες

Η διαχείριση εφαρμογών RSView αποθηκεύει τα αρχεία .apa από προεπιλογή στη θέση C:\Documents and Settings\All Users\Documents\RSView Enterprise\ME\Archives στα Windows XP. Ωστόσο, μπορείτε να αποθηκεύσετε το αρχείο σε οποιονδήποτε κατάλογο σύμφωνα με τις απαιτήσεις.
