## Τι είναι ένα αρχείο PL;

Ένα αρχείο με επέκταση .pl είναι ένα αρχείο Perl Script που είναι γλώσσα δέσμης ενεργειών. Αυτά μεταγλωττίζονται και εκτελούνται χρησιμοποιώντας το λογισμικό Perl Interpreter. Ένα αρχείο σεναρίου PL περιέχει γραμμές κώδικα, μεταβλητές και σχόλια. Οι γλώσσες γραφής είναι σχετικά δύσκολες
κατανοήσει σε άλλες μορφές αρχείων προγραμματισμού όπως [PHP](/el/programming/php/). Μπορούν να δημιουργηθούν αρχεία PL και είναι συμβατά με Windows, macOS και Linux.

## Σύντομη ιστορία της γλώσσας δέσμης ενεργειών Perl

Αυτή η γλώσσα διατηρήθηκε για πρώτη φορά σε χρήση το 1987, έτσι αυτά τα αρχεία άρχισαν την αρχή τους εκείνη τη χρονιά. Το 1989 κυκλοφόρησε το Perl 2. Αργότερα, έχει ενημερωθεί και έχει τροποποιηθεί μέχρι την έκδοση 5.35 και η επόμενη έκδοση έχει στόχο να κυκλοφορήσει το επόμενο έτος.

Σε κάθε ενημέρωση, προστέθηκαν εργαλεία σχετικά με τη λειτουργικότητα, την απόδοση και την εμφάνιση της γλώσσας και των αρχείων. Έχουν γίνει πολλές αναθεωρήσεις σχετικά με τις εκδόσεις αυτά τα χρόνια. Αρχικά ήταν μια βασική γλώσσα σεναρίου, αλλά τώρα περιλαμβάνει και πολλές άλλες ενότητες. Αρχικά, ήταν μια πολύ απλή γλώσσα, αλλά τα σενάρια αυτής της γλώσσας ήταν αρκετά δυσνόητα καθώς ήταν συμπαγή.

## Προδιαγραφές επέκτασης μορφής αρχείου Perl

Υπάρχουν ορισμένες ιδιότητες ή προδιαγραφές αυτών των αρχείων προγραμματισμού, μερικές από αυτές είναι οι εξής:

* Ο κώδικας που περιλαμβάνεται σε αυτά τα αρχεία είναι σε μορφή απλού κειμένου και χρησιμοποιείται για την ανάπτυξη σεναρίων
* Το σενάριο του διακομιστή, η ανάλυση κειμένου και η διαχείριση διακομιστή είναι οι κύριες πτυχές που καλύπτει το σενάριο αυτής της γλώσσας
* Πολλά δημοφιλή προγράμματα που σχετίζονται με αυτήν τη γλώσσα είναι το Active State Active Perl και το Bare Bones BBEdit (συμβατό με Mac OS)
* Αυτή η γλώσσα θεωρείται γλώσσα υψηλού επιπέδου
* Για το Win32, ο χρήστης μπορεί να χρειαστεί να εγκαταστήσει την εγγενή δυαδική διανομή της γλώσσας
* Απαιτεί ορισμένα εργαλεία δέσμης ενεργειών για να γίνουν εκτελέσιμα σε διάφορα κιτ πόρων των Windows
* Το Visual Studio .NET είναι ένα διάσημο πλαίσιο για την ανάπτυξη γλωσσών προγραμματισμού. Το εργαλείο Active State γνωστό ως Visual Perl χρησιμοποιείται για την προσθήκη Perl στο Visual studio
* Στα αρχεία, η πρώτη γραμμή του πηγαίου κώδικα περιγράφει τις πληροφορίες του διερμηνέα που χρησιμοποιείται. Τα αρχεία PL συνήθως ξεκινούν με **#!/usr/bin/perl** γραμμή που λέει στον υπολογιστή σας να εκτελέσει αυτό το σενάριο χρησιμοποιώντας έναν διερμηνέα Perl εγκατεστημένο στον υπολογιστή


## Παραδείγματα σεναρίου PL

```
#!/usr/bin/perl
print "Hello, world\n";
```

Αυτό θα εκτυπώσει

```
Hello, World
```

### Σχόλιο μονής γραμμής ###

```
#!/usr/bin/perl
# This is a single line comment
print "Hello Perl\n";
```

### Σχόλιο πολλαπλών γραμμών ###

```
#!/usr/bin/perl
=begin comment
This is a multiline comment.
Line 1
Line 2
=cut
print "Hello Perl\n";
```

### Ανάθεση μεταβλητής ###

```
#!/usr/bin/perl
$a = 10;
print "Variable a = $a\n";
```

### Εκχώρηση βαθμωτών μεταβλητών ###

```
#!/usr/bin/perl
$age = 35; # Assigning an integer
$name = "Tony Stark"; # Assigning a string
$pi = 3.14; # Assigning a floating point
```

### Απλές βαθμωτές πράξεις ###

```
#!/usr/bin/perl
$constr = "hi" . "perl";# Concatenates two or more strings.
$add = 40 + 10; # addition of two numbers.
$prod = 4 * 51;# multiplication of two numbers.
$connumstr = $constr . $add;# concatenation of string and number.
```
