## Τι είναι ένα αρχείο PY; ##

Τα Αρχεία με την επέκταση .py περιέχουν τον πηγαίο κώδικα της Python. Η γλώσσα Python έχει γίνει πολύ διάσημη πλέον στις μέρες μας. Μπορεί να χρησιμοποιηθεί για δέσμες ενεργειών συστήματος, ανάπτυξη ιστού και λογισμικού και μαθηματικά. Η Python υποστηρίζει συμβατότητα μεταξύ πλατφορμών. σημαίνει ότι οι αναπτυγμένες εφαρμογές στην Python μπορούν να λειτουργήσουν σε διαφορετικές πλατφόρμες όπως Windows, MAC, Linux, Raspberry Pi, κ.λπ. Η Python παρέχει μια απλή και ευανάγνωστη σύνταξη παρόμοια με την αγγλική γλώσσα. Οι προγραμματιστές μπορούν να γράψουν μια λογική εφαρμογή λογισμικού γράφοντας μερικές γραμμές κώδικα Python. Δεδομένου ότι η Python εκτελείται σε ένα σύστημα διερμηνέα, ο κώδικας μπορεί να εκτελεστεί αμέσως μόλις γραφτεί, γεγονός που τον καθιστά πολύ καλό για τη δημιουργία πρωτοτύπων.

## Σύντομη Ιστορία ##

Η Python επινοήθηκε στα τέλη της δεκαετίας του 1980 από τον Guido van Rossum ως διάδοχος της γλώσσας προγραμματισμού ABC. Η εφαρμογή του ξεκίνησε τον Δεκέμβριο του 1989 με τον Van Rossum ως τον μοναδικό κύριο προγραμματιστή. Εργάστηκε στην python μέχρι τις 12 Ιουλίου 2018. Τον Ιανουάριο του 2019, οι ενεργοί προγραμματιστές του πυρήνα της Python εξέλεξαν ένα πενταμελές "Διευθύνον Συμβούλιο" αποτελούμενο από τους Nick Coghlan, Brett Cannon, Carol Willing, Barry Warsaw και Van Rossum για να ηγηθούν του έργου.

## Πώς να εκτελέσετε το αρχείο py ##

Για να ελέγξετε την εγκατεστημένη έκδοση της Python, μπορείτε να χρησιμοποιήσετε την ακόλουθη εντολή:

```cmd
python --version
```

Αυτό θα παράγει την έκδοση στην κονσόλα όπως

```cmd
Python 3.7.4
```

Εάν η Python δεν είναι εγκατεστημένη στον υπολογιστή σας, μπορείτε να μεταβείτε στη διεύθυνση [python.org](https://www.python.org/) και να πραγματοποιήσετε λήψη και εγκατάσταση της Python για το σχετικό λειτουργικό σας σύστημα.

Για να εκτελέσετε ένα σενάριο Python, μπορείτε να χρησιμοποιήσετε την ακόλουθη εντολή:

```cmd
python helloworld.py
```

Το *helloworld.py* είναι ένα αρχείο σεναρίου που περιέχει τον ακόλουθο κώδικα

```py
print("Hello World from Python")
```

Αυτό θα εκτυπώσει τα ακόλουθα στο παράθυρο της κονσόλας.

```cmd
Hello World from Python
```

Σημείωση: Εάν χρησιμοποιείτε IDE, παρέχουν κουμπιά στην οθόνη ή διαφορετικές συντομεύσεις πληκτρολογίου για την εκτέλεση της Python. Για παράδειγμα, εμφανίζεται ένα κουμπί αναπαραγωγής στο αυλάκι με τον επεξεργαστή στο PyCharm που σας επιτρέπει να εκτελέσετε το σενάριο Python.

## Μορφή αρχείου PY ##

Η Python σχεδιάστηκε για αναγνωσιμότητα και έχει ομοιότητες με τα Αγγλικά και τα Μαθηματικά. Η Python χρησιμοποιεί νέες γραμμές για να υποδείξει την πλήρη εντολή σε αντίθεση με ερωτηματικά ή παρενθέσεις που χρησιμοποιούνται σε άλλες γλώσσες. Για πεδία, βρόχους και συναρτήσεις, η Python βασίζεται σε εσοχές και κενά σε αντίθεση με τα σγουρά στηρίγματα που χρησιμοποιούνται σε άλλες γλώσσες.

### Σύνταξη ###

Το παρακάτω είναι ένα παράδειγμα σύνταξης Python.

```py
print("Hello World")

#Variables
name = "John"
age = 25
print(name)
print(age)

#While Loop
i = 1
while i < 3:
  print(i)
  i += 1
  
#For Loop
names = ["John", "Harry", "Tom"]
for name in names:
  print(name)
```
