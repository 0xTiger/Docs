## Τι είναι ένα αρχείο VB;

Ένα αρχείο VB είναι ένα αρχείο πηγαίου κώδικα που δημιουργήθηκε σε γλώσσα Visual Basic που δημιουργήθηκε από τη Microsoft για την ανάπτυξη εφαρμογών .NET. Μια άλλη παρόμοια γλώσσα με διαφορετική σύνταξη είναι η C# της οποίας τα αρχεία αποθηκεύονται με επέκταση αρχείου [.CS](/el/programming/cs/). Η μορφή αρχείου παρέχει τη γλώσσα προγραμματισμού χαμηλού επιπέδου για τη σύνταξη κώδικα που μεταγλωττίζεται για τη δημιουργία του τελικού αρχείου εξόδου με τη μορφή EXE ή DLL. Αυτά μπορούν να δημιουργηθούν και να μεταγλωττιστούν με το Microsoft Visual Studio. Το Microsoft Visual Studio Express μπορεί επίσης να χρησιμοποιηθεί για τη δημιουργία και ενημέρωση τέτοιων αρχείων που είναι δωρεάν IDE. Ένα απλό έργο Visual Studio [λύση](/el/programming/sln/) που δημιουργήθηκε με γλώσσα VB μπορεί να αποτελείται από ένα ή περισσότερα τέτοια αρχεία. Τα αρχεία που έχουν επισημανθεί για συμπερίληψη στη μεταγλώττιση παρατίθενται στο αρχείο [CSPROJ](/el/programming/csproj/) που αποτελεί μέρος του έργου και λέει στον μεταγλωττιστή να χρησιμοποιήσει τα επισημασμένα αρχεία.

## Μορφή αρχείου VB - Περισσότερες πληροφορίες

Τα αρχεία VB είναι μορφές αρχείων που βασίζονται σε κείμενο και μπορούν να ανοίξουν σε οποιοδήποτε πρόγραμμα επεξεργασίας κειμένου για επεξεργασία. Ωστόσο, όταν ανοίγεται σε ένα υποστηριζόμενο IDE με σωστή επισήμανση σύνταξης, ο κώδικας είναι εύκολο να διαβαστεί και να τακτοποιηθεί. Ένα απλό αρχείο VB περιέχει:

* Δήλωση χώρων ονομάτων - Για αναφορά σε μια συγκεκριμένη λειτουργία που ορίζεται από τον συγκεκριμένο χώρο ονομάτων
* Δήλωση μεταβλητών - Για να δηλώσετε μεταβλητές επιπέδου κλάσης για τη συγκεκριμένη υλοποίηση
* Δήλωση μεθόδων - Για να δηλώσετε δήλωση μεθόδων για τη συγκεκριμένη λειτουργικότητα

## Παράδειγμα μορφής αρχείου VB

```
Imports System
Module Module1
   'This program will display Hello World
   Sub Main()
      Console.WriteLine("Hello World")
      Console.ReadKey()
   End Sub
End Module
```
