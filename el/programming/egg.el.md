## Τι είναι ένα αρχείο EGG;

Ένα αρχείο EGG, γνωστό και ως Python eggs, είναι μια παλαιότερη μορφή διανομής διανομές Python. Είναι ένα συμπιεσμένο αρχείο [ZIP](/el/compression/zip/) με επέκταση .egg και περιέχει τα αρχεία πηγής της εφαρμογής Python μαζί με μετα-πληροφορίες για τη διανομή. Τα αρχεία EGG είναι εναλλακτικά σε ένα εκτελέσιμο αρχείο των Windows [EXE](/el/executable/exe/), αλλά είναι cross-platform. Αυτή η παλαιότερη μορφή για διανομές Python έχει αντικατασταθεί με τη νεότερη μορφή αρχείου Wheel (WHL) στις αρχές του 2010.

## Μορφή αρχείου EGG

Τα αρχεία EGG αποθηκεύονται ως συμπιεσμένα αρχεία ZIP. Σημαίνει ότι εάν αντικαταστήσετε την επέκταση .egg με .zip, θα μπορείτε να την ανοίξετε με τυπικά βοηθητικά προγράμματα αποσυμπίεσης όπως το Corel WinZIP, η Microsoft Explorer ή το RARLAB WinRAR.

Τα αρχεία EGG μπορούν να δημιουργηθούν χρησιμοποιώντας το πακέτο **distutils** που διατίθεται από την Python. Ένα άλλο εργαλείο που μπορεί να δημιουργήσει και να ανοίξει αρχεία EGG είναι το **SetupTools**. Τα αρχεία EGG μπορούν να εγκατασταθούν ως πακέτο χρησιμοποιώντας το **easy_install**.

`ΣΗΜΕΙΩΣΗ - Η μορφή αρχείου EGG έχει ξεπεραστεί υπέρ της νέας μορφής αρχείου WHL τροχού.`
