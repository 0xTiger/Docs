## Τι είναι ένα αρχείο ANE;

Ένα αρχείο ANE είναι ένα αρχείο επέκτασης Adobe Air Native που χρησιμοποιείται από προγραμματιστές για τη σύνταξη συγκεκριμένων συναρτήσεων που καλούν τον κώδικα [Adobe ActionScript](https://en.wikipedia.org/wiki/ActionScript). Στις περισσότερες περιπτώσεις, τα αρχεία ANE περιέχουν την εγγενή βιβλιοθήκη κώδικα που χρησιμοποιείται από την επέκταση. Ωστόσο, το ίδιο το αρχείο είναι εγκατεστημένο στη συσκευή. Επιτρέπει συγκεκριμένες λειτουργίες του συστήματος, όπως η δόνηση της συσκευής. Οι εφαρμογές Adobe AIR θα πρέπει να μεταγλωττίζονται για να χρησιμοποιούν δυναμική σύνδεση για τη χρήση του αρχείου ANE.

## Μορφή αρχείου ΑΝΕ - Περισσότερες πληροφορίες

Τα αρχεία ANE αποθηκεύονται στο δίσκο ως δυαδικά αρχεία των οποίων οι προδιαγραφές μορφής αρχείου δεν είναι διαθέσιμες δημόσια. Ένα αρχείο ANE μπορεί να περιέχει εγγενή πηγαίο κώδικα Android Java για πρόσβαση σε λειτουργίες της συσκευής Android, όπως λειτουργίες κλήσης έντασης ή δόνησης. Όλα αυτά γίνονται με την αναδίπλωση του εγγενούς συστήματος στη γλώσσα ActionScript της Adobe.
