## Τι είναι ένα αρχείο RBXM;

Ένα αρχείο RBXM είναι μια μορφή αρχείου μοντέλου 3D που χρησιμοποιείται στο Roblox Studio IDE για την αποθήκευση τρισδιάστατων αντικειμένων του online παιχνιδιού Roblox για πολλούς παίκτες. Περιέχει τρισδιάστατα αντικείμενα που δημιουργήθηκαν για το παιχνίδι χρησιμοποιώντας το IDE. Παραδείγματα τρισδιάστατων αντικειμένων σε ένα αρχείο RBXM περιλαμβάνουν πουκάμισα, παντελόνια, καπέλο, κτίρια, οχήματα, ζώα, αεροπλάνα, αυτοκίνητα ή φυτά. Τα αρχεία RBMX μπορούν να μετατραπούν στη [μορφή αρχείου OBJ 3D](/el/3d/obj/) χρησιμοποιώντας το Roblox IDE. Το Roblox IDE μπορεί να χρησιμοποιηθεί για το άνοιγμα αρχείων RBMX.

Μια άλλη δημοφιλής μορφή αρχείου Roblox είναι το [RBXL](/el/programming/rbxl/) που είναι μια μορφή αρχείου τοποθεσίας και περιέχει τοποθεσία που ορίζεται σε λέξη 3D μέσα στο παιχνίδι Roblox.

## Μορφή αρχείου RBMX - Περισσότερες πληροφορίες

Τα αρχεία RBXM δημιουργούνται και αποθηκεύονται σε δυαδική μορφή αρχείου και οι εσωτερικές προδιαγραφές μορφής αρχείου τους δεν είναι διαθέσιμες δημόσια. Αυτό είναι διαφορετικό από τη μορφή αρχείου RBXMX που βασίζεται στη μορφή αρχείου [XML](/el/web/xml/). Υπάρχουν πολλά εργαλεία/αποθήκες διαθέσιμα στο Github που μπορούν να χρησιμοποιηθούν για την [επεξεργασία Roblox RBXM](https://github.com/search?q=rbxm) αρχείων.
