## Τι είναι ένα αρχείο RB;

Μια επέκταση αρχείου ".rb" συνήθως σχετίζεται με αρχεία γλώσσας προγραμματισμού Ruby. Η Ruby είναι μια δυναμική, αντικειμενοστραφής γλώσσα προγραμματισμού γνωστή για την απλότητα και την αναγνωσιμότητά της.

Ένα αρχείο ".rb" περιέχει τον πηγαίο κώδικα Ruby, ο οποίος μπορεί να εκτελεστεί από έναν διερμηνέα Ruby ή ένα περιβάλλον ανάπτυξης Ruby. Αυτά τα αρχεία συχνά περιέχουν ορισμούς κλάσεων, μεθόδων, μεταβλητών και άλλων δομών κώδικα Ruby.

## Πώς να εκτελέσετε το σενάριο Ruby σε αρχείο RB;

Για να εκτελέσετε ένα σενάριο Ruby που είναι αποθηκευμένο σε αρχείο ".rb", θα χρειαστείτε εγκατεστημένο στο σύστημά σας τον διερμηνέα Ruby. Ακολουθεί ένα βασικό παράδειγμα σεναρίου Ruby που είναι αποθηκευμένο σε αρχείο με το όνομα "example.rb":

```
# example.rb

# Define a method to calculate the square of a number
def square(number)
  number * number
end

# Call the square method with an argument
result = square(5)

# Print the result
puts "The square of 5 is: #{result}"
```

Για να εκτελέσετε αυτό το σενάριο, μπορείτε να ανοίξετε τη γραμμή εντολών ή το τερματικό σας, να πλοηγηθείτε στον κατάλογο όπου βρίσκεται το αρχείο "example.rb" και στη συνέχεια να χρησιμοποιήσετε τον διερμηνέα Ruby:

```
ruby example.rb
```

Η εκτέλεση της παραπάνω εντολής θα εκτελέσει το σενάριο και η έξοδος θα εμφανιστεί στην κονσόλα:

```
The square of 5 is: 25
```

Αυτό είναι ένα απλό παράδειγμα, αλλά τα αρχεία ".rb" μπορεί να περιέχουν πιο περίπλοκο κώδικα, συμπεριλαμβανομένων κλάσεων, λειτουργικών μονάδων και δομών ελέγχου, επιτρέποντάς σας να δημιουργήσετε πλήρεις εφαρμογές Ruby.

## Τι περιέχει το αρχείο RB;

Τα συγκεκριμένα περιεχόμενα του αρχείου ".rb" μπορεί να διαφέρουν ανάλογα με τον σκοπό του και τον προγραμματιστή που το έγραψε. Ωστόσο, γενικά, το αρχείο ".rb" περιέχει τον πηγαίο κώδικα Ruby, ο οποίος αποτελείται από μια σειρά εντολών που μπορεί να κατανοήσει και να εκτελέσει ο διερμηνέας Ruby.

Ακολουθούν ορισμένα κοινά στοιχεία που μπορεί να βρείτε στο αρχείο Ruby:

- **Σχόλια:** Το Ruby υποστηρίζει σχόλια μονής και πολλών γραμμών. Τα σχόλια χρησιμοποιούνται για την προσθήκη επεξηγηματικών σημειώσεων ή την απενεργοποίηση συγκεκριμένων γραμμών κώδικα από την εκτέλεση. Συμβολίζονται με τον χαρακτήρα #.

```
# This is a single-line comment

=begin
This is a
multi-line comment
=end
```

- **Δηλώσεις μεταβλητών:** Η Ruby είναι μια δυναμικά πληκτρολογημένη γλώσσα, επομένως οι μεταβλητές δεν χρειάζονται ρητές δηλώσεις τύπου. Μπορείτε να εκχωρήσετε τιμές σε μεταβλητές χρησιμοποιώντας τον τελεστή εκχώρησης (=).

- **Μέθοδοι:** Η Ruby χρησιμοποιεί τη λέξη-κλειδί "def" για να ορίσει μεθόδους, οι οποίες είναι επαναχρησιμοποιήσιμα μπλοκ κώδικα που εκτελούν συγκεκριμένες εργασίες.

- **Κλάσεις και αντικείμενα:** Η Ruby είναι μια αντικειμενοστραφή γλώσσα και οι κλάσεις χρησιμοποιούνται για τον ορισμό σχεδιαγραμμάτων αντικειμένων. Τα αντικείμενα είναι στιγμιότυπα κλάσεων και μπορούν να έχουν ιδιότητες (μεταβλητές στιγμιότυπου) και συμπεριφορές (μέθοδοι παρουσίας).

- **Δομές ελέγχου:** Η Ruby παρέχει διάφορες δομές ελέγχου, όπως εντολές υπό όρους (if, else, elif, αν δεν), βρόχους (ενώ, μέχρι, για κάθε) και πολλά άλλα, για τον έλεγχο της ροής εκτέλεσης στο πρόγραμμα.

```
if age >= 18
  puts "You are an adult."
else
  puts "You are a minor."
end

# Output: You are an adult.
```

## Ποια είναι η μορφή του αρχείου RB;

Η μορφή του αρχείου ".rb" είναι απλό κείμενο, συνήθως κωδικοποιημένη με κωδικοποίηση UTF-8 ή ASCII. Ακολουθεί μια συγκεκριμένη σύνταξη και δομή που ορίζονται από τη γλώσσα προγραμματισμού Ruby.
