## Τι είναι ένα αρχείο Java;

Ένα αρχείο Jav είναι ένα αρχείο πηγαίου κώδικα Java που έχει δημιουργηθεί σε γλώσσα προγραμματισμού Java. Είναι παρόμοιο με το αρχείο πηγής Java και μπορεί να μεταγλωττιστεί με διερμηνείς Java. Όπως τα αρχεία [Java](/el/programming/java/), το αρχείο Jav χρησιμοποιείται στην ανάπτυξη εφαρμογών Java. Μπορεί να είναι ένα από τα πολλά αρχεία πηγαίου κώδικα που είναι διαθέσιμα σε ένα έργο Java.

## Μορφή αρχείου Java

Τα αρχεία Java αποθηκεύονται ως αρχεία κειμένου και μπορούν να ανοίξουν με οποιοδήποτε πρόγραμμα επεξεργασίας κειμένου. Οι τύποι αρχείων Java έχουν συνήθως αντικατασταθεί από τον τύπο αρχείου Java και χρησιμοποιούνται συνήθως για τη σύνταξη εφαρμογών Java. Οι κώδικες Java μπορούν να γραφτούν με IDE όπως το Eclipse και το IntelliJ Idea που διευκολύνει την εργασία με αρχεία Java.

### Παράδειγμα μορφής αρχείου Java

```java
/*
The example code prints
Hello World from Java to the console.
*/
    public class JavaApp {
        public static void main(String[] args) {
            System.out.println("Hello World."); // Prints the string to the console.
        }
    }
```
