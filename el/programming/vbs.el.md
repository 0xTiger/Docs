## Τι είναι ένα αρχείο VBS;

Το VBS ή το VBScript συνδέεται με την έκδοση σεναρίου της Microsoft Visual Basic. Στο υπολογιστικό περιβάλλον, επιτρέπεται στον χρήστη να έχει πρόσβαση και να ελέγχει πολλές πτυχές του. Το VBScript επιτρέπεται να χρησιμοποιεί ένα μοντέλο με το όνομα **Component Object Model** για να αξιοποιήσει τα στοιχεία και τα εργαλεία του περιβάλλοντος. Αυτό το περιβάλλον έχει καθοριστεί για εργασία και εκτέλεση VBScript.

Λειτουργεί ως παρόμοιο με το Javascript όταν έχει πρόσβαση ο πελάτης στον τομέα της ανάπτυξης Ιστού. Χρησιμοποιείται επίσης από τους διακομιστές για την επεξεργασία ιστοσελίδων. Μπορεί να ληφθεί υπόψη σε ορισμένους άλλους τύπους αρχείων δέσμης ενεργειών όπως εφαρμογές του [HTML](/el/web/html/).


## Σύντομη Ιστορία ##

Κυκλοφόρησε για πρώτη φορά το 1996, ως μέρος των τεχνολογιών που χρησιμοποιούνται από τη Microsoft που καθορίζονται για τα Windows Scripts. Αναπτύχθηκε ειδικά για τη βοήθεια των προγραμματιστών Ιστού αρχικά. Την ίδια χρονιά κυκλοφόρησε ο εξερευνητής των Microsoft Windows με το όνομα Internet Explorer μαζί με τις δυνατότητες του Visual Basic Script.

Με την πρόοδο στην τεχνολογία και την ανάπτυξη ιστού, κυκλοφόρησαν πολλές εκδόσεις του VBScript μαζί με πολλές προηγμένες δυνατότητες. Επιπλέον, το επόμενο έτος, αυτή η γλώσσα σεναρίου θα είναι μέρος των Microsoft Windows με νέες δυνατότητες.

## Τεχνική προδιαγραφή ##

Για τις ιστοσελίδες στην πλευρά του διακομιστή, χρησιμοποιούνται εργαλεία όπως **Ενεργές σελίδες διακομιστή** μαζί με το VBScript. Αυτή η γλώσσα δέσμης ενεργειών μπορεί επίσης να χρησιμοποιηθεί στο στοιχείο σεναρίου των Windows. Τα αρχεία αυτής της γλώσσας αποθηκεύονται με επέκταση .vbs στα Windows.

Υπάρχουν πολλές δομές ελέγχου, όπως βρόχοι που χρησιμοποιούνται στον κώδικα αυτής της γλώσσας. Περιλαμβάνει επίσης ορίσματα που είναι γραμμή εντολών και θα μπορούσαν να ονομαστούν ή να μην ονομαστούν. Τα αρχεία αυτής της γλώσσας μπορούν να αποθηκευτούν απλά στους φακέλους ή στην επιφάνεια εργασίας ενός λειτουργικού συστήματος Windows. Παρόλο που δεν υπάρχει συγκεκριμένο ενσωματωμένο περιβάλλον ανάπτυξης για προγράμματα VBScript όπως το Microsoft Script Editor παρέχουν τη δυνατότητα ανάπτυξης αυτής της γλώσσας.

Όταν το VBScript φιλοξενείται από τον κεντρικό υπολογιστή Script των Windows, παρέχει διάφορες δυνατότητες που είναι αρκετά κοινές στις γλώσσες δέσμης ενεργειών, αλλά δεν είναι διαθέσιμες στη Visual Basic 6.0. Οι δυνατότητες που περιλαμβάνουν εύκολη ή άμεση πρόσβαση περιλαμβάνουν Εκτυπωτές Δικτύου, ανώνυμα και επώνυμα ορίσματα γραμμής εντολών, stdout και stdin, Κοινή χρήση δικτύου, Όργανα διαχείρισης των Windows, πληροφορίες χρήστη δικτύων όπως η ιδιότητα μέλους ομάδας και πολλά άλλα.

## Παράδειγμα μορφής αρχείου VBS ##

```
<% Option Explicit %>
 <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
 <html>
 	<head>
 		<title>VBScript Example</title>
 	</head>
 	<body>
 		<div><% 
 			' Grab current time from Now() function.
 			' An '=' sign occurring after a context switch (<%) is shorthand 
 			' for a call to the Write() method of the Response object.
 			Dim timeValue : timeValue = Now %>
 			The time, in 24-hour format, is 
 			<%=Hour(timeValue)%>:<%=Minute(timeValue)%>:<%=Second(timeValue)%>.
 		</div>
 	</body>
 </html>
```
