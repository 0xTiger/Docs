## Τι είναι ένα αρχείο ACCFT;

Ένα αρχείο με επέκταση .accft είναι ένα αρχείο προτύπου της Microsoft Access που περιέχει τον ορισμό των τύπων δεδομένων πεδίου για τους πίνακες μιας βάσης δεδομένων. Η Microsoft Access είναι ένα επιτραπέζιο Σύστημα Διαχείρισης Σχεσιακών Βάσεων Δεδομένων (RDBMS) που χρησιμοποιείται για τη δημιουργία βάσεων δεδομένων, φορμών, ερωτημάτων και προβολών που βασίζονται σε πίνακες. Το αρχείο ACCFT μπορεί να χρησιμοποιηθεί για τον καθορισμό των τύπων δεδομένων για τα πεδία του πίνακα και οι πίνακες βάσης δεδομένων μπορούν να δημιουργηθούν χρησιμοποιώντας αυτό το αρχείο προτύπου ως σημείο εκκίνησης. Μπορεί να ανοίξει με τη Microsoft Access 2016 για τη δημιουργία νέων πινάκων. Τα αρχεία ACCFT αναγνωρίζονται από τον τύπο MIME **εφαρμογή/msaccess**.

## Μορφή αρχείου ACCFT

Τα αρχεία ACCFT είναι δυαδικά αρχεία των οποίων οι προδιαγραφές μορφής αρχείου δεν είναι διαθέσιμες δημόσια. Οι εσωτερικές του λεπτομέρειες όπως πληροφορίες κεφαλίδας, δομή αρχείου, υποσέλιδα κ.λπ. δεν είναι διαθέσιμες αυτήν τη στιγμή.
