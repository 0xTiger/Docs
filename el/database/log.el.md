## Τι είναι ένα αρχείο LOG;
Ένα αρχείο με επέκταση .log περιέχει τη λίστα απλού κειμένου με χρονική σήμανση. Συνήθως, ορισμένες λεπτομέρειες δραστηριότητας καταγράφονται από τα λογισμικά ή τα λειτουργικά συστήματα για να βοηθήσουν τους προγραμματιστές ή τους χρήστες να παρακολουθούν τι συνέβαινε σε μια συγκεκριμένη χρονική περίοδο. Οι χρήστες μπορούν να επεξεργαστούν αυτά τα αρχεία πολύ εύκολα χρησιμοποιώντας οποιοδήποτε πρόγραμμα επεξεργασίας κειμένου. Συνήθως οι αναφορές σφαλμάτων ή οι δραστηριότητες σύνδεσης καταγράφονται από τα λειτουργικά συστήματα, αλλά άλλα λογισμικά ή διακομιστές ιστού δημιουργούν επίσης αρχεία καταγραφής για την παρακολούθηση των επισκεπτών και την παρακολούθηση της χρήσης εύρους ζώνης.

## Μορφή αρχείου LOG
Η μορφή αρχείου LOG καταγράφει τα τυπικά συμβάντα που συμβαίνουν είτε σε ένα λειτουργικό σύστημα, μηνύματα μεταξύ διαφορετικών χρηστών ενός λογισμικού επικοινωνίας ή οποιουδήποτε άλλου λογισμικού εκτελείται. Απλώς μπορούμε να πούμε ότι ορισμένα μηνύματα σε συγκεκριμένες χρονικές σφραγίδες καταγράφονται σε ένα αρχείο LOG. Μερικοί όροι που χρησιμοποιούνται συνήθως για την καταγραφή είναι:
### Μητρώα συμβάντων
Καταγράφει συμβάντα που συμβαίνουν κατά την εκτέλεση ενός συστήματος προκειμένου να παρακολουθεί και να κατανοεί τη δραστηριότητα του συστήματος και να διαγνώσει προβλήματα. Είναι σημαντικά για τον προσδιορισμό των δραστηριοτήτων σύνθετων συστημάτων, συνήθως στην περίπτωση εφαρμογών με πολύ μικρή αλληλεπίδραση με τον χρήστη.
### Αρχεία καταγραφής συναλλαγών
Σχεδόν όλα τα συστήματα βάσεων δεδομένων διατηρούν το αρχείο καταγραφής συναλλαγών, το οποίο συνήθως δεν προορίζεται ως διαδρομή ελέγχου για μεταγενέστερη ανάλυση και δεν είναι αναγνώσιμα από τον άνθρωπο. Αυτά τα αρχεία καταγραφής διατηρούν μόνο το αρχείο των αλλαγών στα υπάρχοντα δεδομένα για να επιτρέψουν στη βάση δεδομένων να ανακτήσει από σφάλματα ή άλλα σφάλματα δεδομένων και να διατηρήσει τα δεδομένα σε συνεπή κατάσταση. Ως εκ τούτου, τα συστήματα βάσεων δεδομένων συνήθως έχουν τόσο γενικά αρχεία καταγραφής συμβάντων όσο και αρχεία καταγραφής συναλλαγών.
### Αρχεία καταγραφής μηνυμάτων
Η επικοινωνία κειμένου που αποθηκεύεται από το Internet Relay Chat (IRC), τα προγράμματα άμεσων μηνυμάτων (IM), τους πελάτες κοινής χρήσης αρχείων peer-to-peer με λειτουργίες συνομιλίας και τα παιχνίδια για πολλούς παίκτες (ειδικά τα MMORPG) ονομάζονται αρχεία καταγραφής μηνυμάτων. Αυτά τα αρχεία καταγραφής αποθηκεύονται γενικά σε αρχεία απλού κειμένου, αλλά οι πελάτες IM και VoIP (π.χ. Skype) ενδέχεται να τα αποθηκεύσουν σε αρχεία HTML για να διευκολύνουν την ανάγνωση ή να ενεργοποιήσουν την κρυπτογράφηση.
### Αρχεία καταγραφής διακομιστή
Το αρχείο καταγραφής διακομιστή είναι στην πραγματικότητα ένα αρχείο καταγραφής που περιέχει μια λίστα δραστηριοτήτων που εκτελούνται και δημιουργούνται ή διατηρούνται αυτόματα από τον ίδιο τον διακομιστή. Συνήθως αυτά τα αρχεία δεν είναι προσβάσιμα σε γενικούς χρήστες του Διαδικτύου, μόνο στον webmaster ή άλλο διαχειριστικό πρόσωπο μιας υπηρεσίας Διαδικτύου.
