## Τι είναι ένα αρχείο ACCDE;

Ένα αρχείο με επέκταση .accde είναι μια μορφή αρχείου της Microsoft Access 2007 για τη δημιουργία αρχείων βάσης δεδομένων Execute Only για την προστασία προσαρμοσμένου κώδικα βάσης δεδομένων. Μεταγλωττίζει όλο τον κώδικα VBA σε εκτελέσιμο μόνο έτσι ώστε να μην είναι δυνατή η προβολή ή η επεξεργασία του. Η μορφή αρχείου ACCDE αντικαθιστά τη μορφή αρχείου MDE που χρησιμοποιήθηκε από προηγούμενες εκδόσεις της Microsoft Access. Τα ACCDE είναι αρχεία [ACCDB](/el/database/accdb/) σε εκτελέσιμη λειτουργία μόνο και πιο συμπαγή λόγω απουσίας πηγαίου κώδικα. Το μικρό μέγεθος αυτών των αρχείων βελτιώνει τη χρήση της μνήμης και βελτιώνει την απόδοση της εφαρμογής. Τα αρχεία ACCDE μπορούν να ανοίξουν με τη Microsoft Access 365.

## Μορφή αρχείου ACCDE

Όλα τα αρχεία Microsoft Access 2007+ χρησιμοποιούν το Jet ACE (μηχανή βάσης δεδομένων πρόσβασης) που χρησιμοποιεί την οικογένεια μορφών αρχείων ACCDB. Ωστόσο, η Microsoft δεν παρέχει εφαρμογή ανοιχτού κώδικα ή λεπτομερείς τεχνικές προδιαγραφές.
