## Τι είναι ένα αρχείο ADP;

Ένα αρχείο με επέκταση .adp είναι ένα αρχείο έργου της Microsoft Access που χρησιμοποιεί αρχιτεκτονική στοιχείων OLE DB για να παρέχει μια άμεση και αποτελεσματική σύνδεση με μια βάση δεδομένων του Microsoft SQL Server. Τέτοια αρχεία δεν περιέχουν τους πραγματικούς πίνακες, διαγράμματα βάσης δεδομένων ή άλλα στοιχεία βάσης δεδομένων. Τα αρχεία ADP μπορούν να δημιουργηθούν με την Access 2007 και 2010. Τα αρχεία που δημιουργούνται με τις προηγούμενες εκδόσεις της Access μπορούν επίσης να ανοίξουν Access 2007 και 2010. Τα αρχεία ADP μπορούν να ανοίξουν χρησιμοποιώντας τη Microsoft Access 365.

## Μορφή αρχείου ADP - Περισσότερες πληροφορίες

Ένα έργο της Access που δημιουργήθηκε ως ADP σάς επιτρέπει να κάνετε αλλαγές σχεδίασης σε αντικείμενα διακομιστή SQL, όπως πίνακες και προβολές. Επιτρέπει επίσης τη δημιουργία, την επεξεργασία και τη χρήση χαρακτηριστικών SQL, όπως διαγράμματα βάσης δεδομένων, αποθηκευμένες διαδικασίες και λειτουργίες που καθορίζονται από το χρήστη. Αυτό δίνει πλεονέκτημα έναντι της σύνδεσης σε μια βάση δεδομένων διακομιστή SQL όπου δεν μπορείτε να κάνετε αλλαγές στη σχεδίαση και μπορείτε να συνδεθείτε μόνο σε πίνακες και προβολές διακομιστή SQL.

### Τύποι δεδομένων και υποστήριξη διαγραμμάτων

**Τύποι δεδομένων ημερομηνίας/ώρας**

Οι ακόλουθοι νέοι τύποι δεδομένων ημερομηνίας/ώρας υποστηρίζονται στην Access 2010.

* ΧΡΟΝΟΣ
* ΗΜΕΡΟΜΗΝΙΑ
* DATETIME2
* DATETIMEOFFSET

**Τύποι δεδομένων μεταβλητού μήκους**

Οι ακόλουθοι τύποι δεδομένων μεταβλητού μήκους μπορούν να χρησιμοποιηθούν σε έργα Access 2010:

* VARBIN (MAX)
* VARCHAR(MAX)
* NVARCHAR(MAX)
