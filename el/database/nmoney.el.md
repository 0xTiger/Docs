## Τι είναι ένα αρχείο NMONEY;

Ένα αρχείο NMONEY είναι ένα αρχείο βάσης δεδομένων αποθήκευσης οικονομικών δεδομένων που περιέχει ιστορικό οικονομικών συναλλαγών. Αναπτύχθηκε από τη Nickvision και χρησιμοποιήθηκε στο λογισμικό Nickvision Denaro, το οποίο είναι μια προσωπική εφαρμογή χρηματοοικονομικού λογισμικού. Τα δεδομένα που αποθηκεύονται σε ένα αρχείο NOMONEY περιλαμβάνουν πληροφορίες συναλλαγών πίστωσης και χρέωσης σε έναν λογαριασμό.

Τα αρχεία NOMONEY μπορούν να ανοίξουν με την εφαρμογή [Github](https://github.com/NickvisionApps/Denaro).

## Σχετικά με το λογισμικό Denaro

Το Denaro αναπτύχθηκε αρχικά από την [Nickvision](https://nickvision.org/) ως προϊόν που αργότερα μετατράπηκε σε εφαρμογή λογισμικού ανοιχτού κώδικα που φιλοξενήθηκε στο [Github](https://github.com/NickvisionApps/Denaro) . Από τότε η εφαρμογή έχει τροποποιηθεί και ενημερωθεί μόνο στο Github. Η εφαρμογή έχει αναπτυχθεί σε C# στο περιβάλλον εργασίας των Windows με το Windows App SDK (WinUI 3 όπως αυτή τη στιγμή).

### Χαρακτηριστικά που προσφέρονται από την Denaro

* Διαχειριστείτε πολλούς λογαριασμούς ταυτόχρονα με την πιο δημοφιλή και οικεία διεπαφή καρτελών
* Φιλτράρετε τις συναλλαγές κατά τύπο, ομάδα ή ημερομηνία
* Επαναλαμβανόμενες συναλλαγές όπως λογαριασμοί που πραγματοποιούνται κάθε μήνα
* Μεταφορά χρημάτων από έναν λογαριασμό σε άλλο
* Εξάγετε τα δεδομένα από έναν λογαριασμό ως αρχείο CSV και εισάγετε ένα αρχείο CSV, OFX ή QIF για να προσθέσετε συναλλαγές σε έναν λογαριασμό

## Μορφή αρχείου Denaro - Περισσότερες πληροφορίες

Τα αρχεία Denaro αποθηκεύονται σε δίσκο σε δυαδική μορφή αρχείου. Τα οικονομικά δεδομένα αποθηκεύονται ως αρχείο βάσης δεδομένων σε μορφή αρχείου **.SQLITE3**. Το SQLITE είναι ένα ελαφρύ αρχείο βάσης δεδομένων SQL που δημιουργήθηκε με το λογισμικό SQLite. Παρέχει αυτόνομη μηχανή βάσης δεδομένων που είναι ικανή να παρέχει πλήρως εξοπλισμένη και εξαιρετικά αξιόπιστη βάση δεδομένων. Τα αρχεία βάσης δεδομένων SQLite μπορούν να χρησιμοποιηθούν για την κοινή χρήση πλούσιου περιεχομένου μεταξύ συστημάτων, ανταλλάσσοντας απλά αυτά τα αρχεία μέσω του δικτύου. Οι δεσμεύσεις SQLite υπάρχουν για γλώσσες προγραμματισμού όπως C, [C#](/el/programming/cs/), [C++](/el/programming/cpp/), [Java](/el/programming/java/), [PHP](/el/programming/ php/), και πολλά άλλα.
