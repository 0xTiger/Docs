## Τι είναι ένα αρχείο ABCDDB;

Το αρχείο με επέκταση **.ABCDDB** σημαίνει Apple Address Book List Database. Ανήκει στην εφαρμογή **Βιβλίο Διευθύνσεων**, επίσης γνωστή ως **Επαφές**. Είναι μια ενσωματωμένη εφαρμογή και περιλαμβάνεται με λειτουργικά συστήματα iOS και macOS. Η εφαρμογή Επαφές επιτρέπει στους χρήστες να αποθηκεύουν και να οργανώνουν πληροφορίες που σχετίζονται με τις επαφές τους, συμπεριλαμβανομένων ατόμων, επιχειρήσεων και οργανισμών. Αυτή η εφαρμογή επιτρέπει στους χρήστες να παρακολουθούν λεπτομέρειες όπως ονόματα, διευθύνσεις, αριθμούς τηλεφώνου και διευθύνσεις email, καθώς και να κατηγοριοποιούν τις επαφές τους σε ομάδες.

## Σχέση με SQLite και Τυπική διαδρομή

Το Βιβλίο Διευθύνσεων της Apple (γνωστό και ως Επαφές) αποθηκεύει τις πληροφορίες επικοινωνίας ως vCard στο φάκελο μεταδεδομένων. **Το αρχείο _ABCDDB_ είναι στην πραγματικότητα _SQLite 3 datafile_**.

Το SQLite είναι ένα δημοφιλές σύστημα διαχείρισης βάσεων δεδομένων που έχει σχεδιαστεί για να είναι ελαφρύ και αυτόνομο. Χρησιμοποιείται σε πολλούς διαφορετικούς τύπους λογισμικού και συσκευών. Το SQLite είναι ένας τύπος RDBMS, που σημαίνει ότι αποθηκεύει δεδομένα σε πίνακες που σχετίζονται μεταξύ τους μέσω κλειδιών. Μπορεί να χειριστεί μια σειρά τύπων δεδομένων, συμπεριλαμβανομένων ακεραίων, αριθμών κινητής υποδιαστολής, συμβολοσειρές και δυαδικών δεδομένων. Επιπλέον, το SQLite υποστηρίζει συναλλαγές, οι οποίες επιτρέπουν στους χρήστες να εκτελούν πολλαπλές λειτουργίες βάσης δεδομένων μαζί ως μια ενιαία μονάδα εργασίας.

Το αρχείο με επέκταση ABCDDB συνήθως αποθηκεύεται εδώ

`~/Βιβλιοθήκη/Υποστήριξη εφαρμογών/AddressBook/AddressBook-v22.abcddb`

## Διόρθωση κατεστραμμένης βάσης δεδομένων Επαφών

Πάρτε πρώτα το αντίγραφο ασφαλείας πριν επιχειρήσετε να το κάνετε αυτό. Στη συνέχεια, μεταβείτε σε

`~/Βιβλιοθήκη/Υποστήριξη εφαρμογών/Βιβλίο διευθύνσεων`

Σε αυτόν τον κατάλογο, θα βρείτε τρία αρχεία, συμπεριλαμβανομένου αυτού με επέκταση .abcddb

- `addressbook-v22.abcddb`
- `addressbook-v22.abcddb-wal`
- `addressbook-v22.abcddb-shm`

Διαγράψτε όλα αυτά τα αρχεία και ανοίξτε ξανά τις Επαφές και θα αρχίσει να λειτουργεί ξανά.

## Επαναφορά της βάσης δεδομένων του Βιβλίου διευθύνσεων από το Backup

Ας υποθέσουμε ότι οι επαφές της βάσης δεδομένων του Βιβλίου Διευθύνσεων είναι αποθηκευμένες στο `addressbook-v22.abcddb`

- Πρώτα δημιουργήστε αντίγραφα ασφαλείας των δεδομένων του Βιβλίου διευθύνσεών σας και κλείστε όλες τις εφαρμογές.
- Μετακινήστε το αρχείο της βάσης δεδομένων σας στον υποκατάλογο `~/Βιβλιοθήκη/Υποστήριξη εφαρμογών/Βιβλίο διευθύνσεων`
- Εκκινήστε το **Address Book.app**.

## Εξαγωγή δεδομένων αρχείου ABCDDB στη Microsoft Access

Δεδομένου ότι το αρχείο είναι αρχείο δεδομένων SQLite 3, μπορείτε να εξαγάγετε τα δεδομένα μέσα στο αρχείο abcddb στη Microsoft Access χρησιμοποιώντας την υποδοχή ODBC.

Η υποδοχή ODBC SQLite είναι μια βιβλιοθήκη λογισμικού και ένα πρόγραμμα οδήγησης που επιτρέπει σε εφαρμογές που υποστηρίζουν τη διεπαφή ODBC να συνδέονται με μια βάση δεδομένων SQLite. Περιλαμβάνει μια βιβλιοθήκη που ορίζει τη διεπαφή ODBC και ένα πρόγραμμα οδήγησης που μεταφράζει αυτή τη διεπαφή σε κλήσεις προς το SQLite API. Για να χρησιμοποιήσετε την υποδοχή ODBC SQLite, πρέπει πρώτα να την εγκαταστήσετε και στη συνέχεια να ρυθμίσετε μια πηγή δεδομένων ODBC στον υπολογιστή σας. Μετά την ολοκλήρωση αυτών των βημάτων, κάθε εφαρμογή που είναι εξοπλισμένη με υποστήριξη ODBC θα μπορεί να συνδεθεί στην προέλευση δεδομένων και να ανακτήσει δεδομένα από τη βάση δεδομένων SQLite.
