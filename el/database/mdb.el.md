## Τι είναι ένα αρχείο MDB;

Ένα αρχείο με επέκταση .mdb είναι ένα αρχείο βάσης δεδομένων της Microsoft Access που είναι Σύστημα Διαχείρισης Σχεσιακών Βάσεων Δεδομένων (RDBMS). Αποθηκεύει δεδομένα σε πίνακες βάσεων δεδομένων που συνδέονται μεταξύ τους μέσω πρωτεύοντος και ξένου κλειδιού. Το αρχείο MDB περιέχει την πλήρη δομή των πινάκων της βάσης δεδομένων, των ερωτημάτων, των αποθηκευμένων διαδικασιών. Το MDB είναι η προεπιλεγμένη μορφή αρχείου για τη Microsoft Access 2003. Οι πλευρικές εκδόσεις της Microsoft Access χρησιμοποιούν τις μορφές αρχείων [ACCDB](/el/database/accdb/), που είναι η πιο πρόσφατη μορφή αρχείου μέχρι σήμερα. Τα αρχεία MDB μπορούν να ανοίξουν με εφαρμογές όπως η Microsoft Access, το MDB Viewer, το MDBOpener και μπορούν να μετατραπούν σε μορφές ACCDB, [CSV](/el/spreadsheet/csv/), Excel κ.λπ.

## Μορφή αρχείου MDB

Υπάρχουν διαθέσιμες δημόσιες προδιαγραφές για τη μορφή MDB και παραμένει η ιδιόκτητη μορφή αρχείου της Microsoft. Η Microsoft, ωστόσο, παρέχει πρόσβαση συνδεσιμότητας στο αρχείο MDB χρησιμοποιώντας το πρότυπο Open Database Connectivity (ODBC) και τη Visual Basic for Applications (VBA). Ο ανεπίσημος οδηγός MDB παρέχει σύντομη ανεπίσημη περιγραφή της μορφής MDB με βάση την αντίστροφη μηχανική και μπορείτε να συμβουλευτείτε για να μάθετε σχετικά με τις προδιαγραφές.

### Σελίδες

Σύμφωνα με τον ανεπίσημο οδηγό MDB, ένα αρχείο MDB αποτελείται από σελίδες σταθερού μεγέθους (2048 byte για το Jeb DB3 και 4096 byte για το Jet DB 4). Το πρώτο byte υποδεικνύει τον τύπο της σελίδας. Ακολουθούν οι βασικοί τύποι σελίδων:

`Πρώτη σελίδα:` Περιέχει πληροφορίες κεφαλίδας βάσης δεδομένων που περιλαμβάνουν επίσης την αναγνώριση της έκδοσης Jet DB με την οποία είναι συμβατό το αρχείο. Επιπλέον, περιλαμβάνει επίσης πληροφορίες ασφαλείας αρχείων και χάρτη χρήσης της σελίδας.

`Σελίδα ορισμού πίνακα:` Μια σελίδα ορισμού πίνακα καθορίζει στήλες, τύπους δεδομένων, ευρετήριο και άλλες πληροφορίες. Χρησιμοποιεί πρόσθετες σελίδες εάν απαιτείται και περιλαμβάνει έναν χάρτη σελίδων που περιέχει τα δεδομένα σειρών για αυτόν τον πίνακα.

`Σελίδες δεδομένων:` Οι σελίδες δεδομένων είναι τα πραγματικά δοχεία δεδομένων όπου τα δεδομένα αποθηκεύονται ανά σειρές. Χρησιμοποιεί δευτερεύουσες σελίδες για την αποθήκευση μεγάλων τιμών δεδομένων.

Μια ενιαία βάση δεδομένων της Microsoft Access μπορεί να αποτελείται από πολλά αρχεία που επιτρέπουν την υπέρβαση των περιορισμών μεγέθους αρχείου και πίνακα. Αυτό διευκολύνει την τοποθέτηση φορμών και κώδικα σε ένα αρχείο MDB front-end στην επιφάνεια εργασίας του χρήστη και δεδομένων σε άλλα αρχεία MDB υποστήριξης σε διακομιστές που είναι συνδεδεμένοι στο δίκτυο.
