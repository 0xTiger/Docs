## Τι είναι ένα αρχείο FIC;

Το αρχείο FIC στο WinDev είναι μια μεταγλωττισμένη έκδοση μιας βάσης δεδομένων HyperFileSQL. Το ακρωνύμιο FIC σημαίνει Fichier de Code στα γαλλικά, που μεταφράζεται σε Αρχείο κώδικα στα αγγλικά. Στο WINDEV, ένα αρχείο FIC είναι ένα μεταγλωττισμένο αρχείο που περιέχει τον εκτελέσιμο κώδικα ενός έργου. Το αρχείο έχει την επέκταση .fic και δημιουργείται κατά τη μεταγλώττιση ενός έργου WINDEV. Μόλις ένα έργο μεταγλωττιστεί σε ένα αρχείο FIC, μπορεί να διανεμηθεί στους χρήστες ως εκτελέσιμο πρόγραμμα ή ως πακέτο εγκατάστασης.

Το αρχείο FIC σχετίζεται με το WINDEV, ένα εργαλείο ανάπτυξης λογισμικού που δημιουργήθηκε από την PCSoft, μια γαλλική εταιρεία. Επιτρέπει στους προγραμματιστές να δημιουργούν εφαρμογές Windows, web και κινητών χρησιμοποιώντας μια ενιαία βάση κώδικα. Περιλαμβάνει ένα περιβάλλον οπτικής ανάπτυξης, μια γλώσσα προγραμματισμού (WLangage) και μια ποικιλία προκατασκευασμένων στοιχείων ελέγχου και λειτουργιών για κοινές εργασίες. Χρησιμοποιείται κυρίως για επιχειρηματικές εφαρμογές και έχει ισχυρή βάση χρηστών στην Ευρώπη.

## Σχέση με τη βάση δεδομένων HyperFileSQL Mobile

Στο WinDev, ένα αρχείο FIC που δημιουργείται για μια βάση δεδομένων HyperFileSQL Mobile περιέχει τον εκτελέσιμο κώδικα για τη βάση δεδομένων, καθώς και τα δεδομένα που περιέχονται στη βάση δεδομένων. Το HyperFileSQL Mobile είναι ένα σχεσιακό σύστημα διαχείρισης βάσεων δεδομένων που περιλαμβάνεται στο WINDEV και έχει σχεδιαστεί ειδικά για κινητές συσκευές και εφαρμογές εκτός σύνδεσης. Το αρχείο FIC μπορεί να διανεμηθεί ως αυτόνομο εκτελέσιμο αρχείο ή ως μέρος ενός πακέτου εγκατάστασης και μπορεί να εκτελεστεί στη συσκευή προορισμού για πρόσβαση και διαχείριση των δεδομένων στη βάση δεδομένων HyperFileSQL Mobile. Όταν εκτελείται το αρχείο FIC, θα ανοίξει τη βάση δεδομένων και θα κάνει τα δεδομένα διαθέσιμα για χρήση στην εφαρμογή για κινητά.
