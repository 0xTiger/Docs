## Τι είναι ένα αρχείο WAV;

Το WAV, γνωστό ως WAVE (Waveform Audio File Format), είναι ένα υποσύνολο της προδιαγραφής Resource Interchange File Format (RIFF) της Microsoft για την αποθήκευση ψηφιακών αρχείων ήχου. Η μορφή δεν εφαρμόζει καμία συμπίεση στο bitstream και αποθηκεύει τις ηχογραφήσεις με διαφορετικούς ρυθμούς δειγματοληψίας και bitrates. Ήταν και είναι μια από τις τυπικές μορφές για CD ήχου. Τα αρχεία Wave είναι μεγαλύτερα σε μέγεθος σε σύγκριση με νέες μορφές αρχείων ήχου, όπως το [MP3](/el/audio/mp3/) που χρησιμοποιεί συμπίεση με απώλειες για να μειώσει το μέγεθος του αρχείου διατηρώντας την ίδια ποιότητα ήχου. Ωστόσο, τα αρχεία WAV μπορούν να συμπιεστούν χρησιμοποιώντας κωδικοποιητές Audio Compression Manager (ACM). Υπάρχουν πολλά διαθέσιμα API και εφαρμογές που μπορούν να μετατρέψουν αρχεία WAV σε άλλες δημοφιλείς μορφές αρχείων ήχου.

## Μορφή αρχείου WAV ##

Η μορφή αρχείου WAVE, που είναι ένα υποσύνολο της προδιαγραφής RIFF της Microsoft, ξεκινά με μια κεφαλίδα αρχείου που ακολουθείται από μια ακολουθία τμημάτων δεδομένων. Ένα αρχείο WAVE έχει ένα μόνο κομμάτι "WAVE" που αποτελείται από δύο υποτμήματα:

* ένα κομμάτι "fmt" - καθορίζει τη μορφή δεδομένων
* ένα κομμάτι "δεδομένων" - περιέχει το πραγματικό δείγμα δεδομένων

### Κεφαλίδα αρχείου WAV ###

Η κεφαλίδα ενός αρχείου WAV (RIFF) έχει μήκος 44 byte και έχει την ακόλουθη μορφή:


|Θέσεις|Δείγμα τιμής|Περιγραφή
---|---|---|
|1 - 4|"RIFF"|Σημαίνει το αρχείο ως αρχείο riff. Οι χαρακτήρες έχουν μήκος 1 byte.
|5 - 8|Μέγεθος αρχείου (ακέραιος)|Μέγεθος συνολικού αρχείου - 8 byte, σε byte (ακέραιος αριθμός 32 bit). Συνήθως, θα το συμπληρώσετε μετά τη δημιουργία.
|9 -12|"WAVE"|Κεφαλίδα τύπου αρχείου. Για τους σκοπούς μας, ισούται πάντα με "WAVE".
|13-16|"fmt "|Μορφοποίηση μαρκαδόρου. Περιλαμβάνει μηδενικό μηδενικό αποτέλεσμα
|17-20|16|Μήκος δεδομένων μορφής όπως αναφέρεται παραπάνω
|21-22|1|Τύπος μορφής (1 είναι PCM) - ακέραιος αριθμός 2 byte
|23-24|2|Αριθμός καναλιών - ακέραιος αριθμός 2 byte
|25-28|44100|Ρυθμός δειγματοληψίας - ακέραιος αριθμός 32 byte. Οι συνήθεις τιμές είναι 44100 (CD), 48000 (DAT). Sample Rate = Αριθμός δειγμάτων ανά δευτερόλεπτο ή Hertz.
|29-32|176400|(Ρυθμός δειγμάτων * BitsPerSample * Κανάλια) / 8.
|33-34|4|(BitsPerSample * Κανάλια) / 8.1 - 8 bit mono2 - 8 bit stereo/16 bit mono4 - 16 bit stereo
|35-36|16|Bits ανά δείγμα
|37-40|"δεδομένα"|τμήμα κεφαλίδας"δεδομένων". Σηματοδοτεί την αρχή της ενότητας δεδομένων.
|41-44|Μέγεθος αρχείου (δεδομένα)|Μέγεθος της ενότητας δεδομένων.
|Δείγματα τιμών δίνονται παραπάνω για μια στερεοφωνική πηγή 16-bit.
