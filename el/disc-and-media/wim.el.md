## Τι είναι ένα αρχείο WIM;
Τα αρχεία με επεκτάσεις .wim εμφανίστηκαν για πρώτη φορά στα Windows Vista. Το WIM ανήκει σε μια μορφή απεικόνισης που βασίζεται σε αρχεία, η οποία συνήθως εισήχθη στα Microsoft Windows Vista. Επιτρέπει την ανάπτυξη μιας εικόνας δίσκου σε διάφορες πλατφόρμες υπολογιστών. Τα αρχεία WIM χρησιμοποιούνται για τη διαχείριση αρχείων όπως ενημερώσεις, προγράμματα οδήγησης και στοιχεία χωρίς επανεκκίνηση της εικόνας του λειτουργικού συστήματος. Το αρχείο AQ WIM περιέχει ένα σύνολο αρχείων και σχετικών μεταδεδομένων που είναι πολύ παρόμοια με άλλες μορφές αρχείων δίσκου.

## Μορφές αρχείων WIM
Η μορφή αρχείου WIM δεν μοιάζει με τις μορφές που βασίζονται σε τομείς όπως το VHD ή το IS, στην πραγματικότητα είναι μια μορφή αρχείου που σημαίνει ότι η θεμελιώδης μονάδα πληροφοριών σε ένα WIM είναι ένα αρχείο. Το βασικό πλεονέκτημα του να βασίζεται σε αρχεία είναι ότι η αποθήκευση ενός αρχείου σε μία περίπτωση που αναφέρεται πολλές φορές στο δέντρο του συστήματος αρχείων και η ανεξαρτησία του υλικού. Μειώνει την επιβάρυνση του ανοίγματος και κλεισίματος διαφόρων αρχείων ξεχωριστά, επειδή τα αρχεία αποθηκεύονται σε ένα μόνο WIM αρχείο. Τα αρχεία WIM μπορούν να αποθηκεύσουν πολλές εικόνες δίσκου, οι οποίες αναφέρονται είτε με το μοναδικό τους όνομα είτε από το αριθμητικό ευρετήριό τους.
### Υποστήριξη για αλγόριθμους συμπίεσης
Το WIM υποστηρίζει τις ακόλουθες οικογένειες αλγορίθμων συμπίεσης σε φθίνουσα ταχύτητα και αναλογία ανόδου:
- XPRESS
- LZX
- LZMS
- Συμπίεση στερεού.
Οι τρεις πρώτες οικογένειες βασίζονται στο LZ77, ενώ η συμπίεση στερεάς μορφής εισήχθη μαζί με το LZMS στα WIMGAPI Windows 8 και DISM Windows 8.1.
### Εργαλεία για μορφή αρχείου WIM
Τα ακόλουθα εργαλεία συνήθως υποστηρίζουν τη μορφή αρχείου WIM:

- **ImageX**: Ένα εργαλείο γραμμής εντολών που χρησιμοποιείται για την επεξεργασία, τη δημιουργία και την ανάπτυξη εικόνων δίσκου των Windows στη μορφή απεικόνισης των Windows. Μαζί με το σχετικό WIMGAPI, διανέμεται ως μέρος του δωρεάν κιτ αυτόματης εγκατάστασης των Windows. Ξεκινώντας με τα Windows Vista, το πρόγραμμα Εγκατάστασης των Windows χρησιμοποιεί το WAIK API για την εγκατάσταση των Windows.
- **DISM**: Ένα εργαλείο που εισήχθη στα Windows 7 και στον Windows Server 2008 R2 που μπορεί να εκτελέσει εργασίες που σχετίζονται με την υπηρεσία σε μια εικόνα εγκατάστασης των Windows, είτε πρόκειται για ηλεκτρονική εικόνα είτε για εικόνα εκτός σύνδεσης σε φάκελο ή αρχείο WIM. αυτό το εργαλείο ήταν ικανό να προσαρτά και να αποσυνδέει εικόνες, να προσθέτει ένα πρόγραμμα οδήγησης συσκευής σε μια εικόνα εκτός σύνδεσης και να αναζητά τα εγκατεστημένα προγράμματα οδήγησης συσκευών σε μια εικόνα εκτός σύνδεσης.
