## Τι είναι ένα αρχείο SDI;
Το αρχείο με επέκταση .sdi περιέχει μια φόρτιση blob, boot blob και part blob. χρησιμοποιείται συνήθως από το Windows Embedded Studio για τη δημιουργία δίσκων RAM ή δισκών εκκίνησης για φόρτωση και εγκατάσταση των Windows. Το SDI συντομεύεται ως System Deployment Image. Το Windows Embedded Studio είναι ένα πρόγραμμα που χρησιμοποιείται για τη δημιουργία εικόνων δίσκου για Windows. Στην πραγματικότητα αυτό το πρόγραμμα περιέχει πρόγραμμα SDI File Manager και ένα εργαλείο γραμμής εντολών που ονομάζεται **sdimgr.exe**, και τα δύο μπορούν να χρησιμοποιηθούν για την ανάπτυξη, τη δημιουργία και την επεξεργασία εικόνων SDI.

## Μορφή αρχείου SDI
Η μορφή αρχείου SDI χρησιμοποιείται ευρέως για να επιτρέψει τη χρήση ενός εικονικού δίσκου για την εκκίνηση ενός συστήματος. Ορισμένες εκδόσεις των Microsoft Windows ενεργοποιούν την **εκκίνηση RAM**, η οποία είναι σημαντική για τη φόρτωση ενός αρχείου SDI στη μνήμη και στη συνέχεια την εκκίνηση από αυτήν. Η μορφή αρχείου SDI επιτρέπει επίσης την εκκίνηση του δικτύου χρησιμοποιώντας το PXE (Περιβάλλον εκτέλεσης προεκκίνησης). Χρησιμοποιείται επίσης για απεικόνιση σκληρού δίσκου.
### Ενότητες αρχείου SDI
Το ίδιο το αρχείο SDI μπορεί να υποδιαιρεθεί στις ακόλουθες ενότητες:
- **Boot BLOB**: Περιέχει το πραγματικό πρόγραμμα εκκίνησης, STARTROM.COM. Αυτό είναι ανάλογο με τον τομέα εκκίνησης ενός σκληρού δίσκου.
- **Φόρτωση BLOB**: Συνήθως περιέχει NTLDR και εκκινείται από το BLOB εκκίνησης.
- **Part BLOB**: Περιέχει τον πραγματικό χρόνο εκκίνησης. περιλαμβάνει επίσης τα αρχεία boot.ini και ntdetect.com που θα πρέπει να βρίσκονται στον ριζικό κατάλογο του χρόνου εκτέλεσης.
- **Disk BLOB**: Αυτή είναι μια επίπεδη εικόνα σκληρού δίσκου που ξεκινά με ένα MBR. Χρησιμοποιείται για απεικόνιση σκληρού δίσκου αντί για εκκίνηση. Επίσης, μόνο τα Disk BLOBs μπορούν να τοποθετηθούν με τα βοηθητικά προγράμματα της Microsoft.
