## Τι είναι ένα αρχείο WBFS;

Ένα αρχείο WBFS (Wii Backup File System) είναι μια εικόνα δίσκου βιντεοπαιχνιδιών που δημιουργήθηκε για το Nintendo Wii. Περιέχει ένα παιχνίδι Wii που μπορεί να φορτωθεί ως εικόνα για παιχνίδι. Δημιουργείται χρησιμοποιώντας το εργαλείο Wiimms WBPS ή το Wii Backup Manager και μπορεί να αναπαραχθεί σε ένα Nintendo Wii με φόρτωση από έναν εξομοιωτή USB Loader GX ή Wii όπως το Dolphin. Το παιχνίδι αποθηκεύεται σε αυτές τις εξωτερικές συσκευές αποθήκευσης χρησιμοποιώντας τη μορφή αρχείου WBFS. Το Wii είναι μια πλατφόρμα παιχνιδιών που διανέμεται από τη Nintendo. Το σύστημα αρχείων WBFS αναπτύχθηκε από τον κωδικοποιητή Wii homebrew Waninkoko.

## Μορφή αρχείου WBFS - Περισσότερες πληροφορίες

Τα WBFS είναι δυαδικά αρχεία που έχουν ιδιόκτητη μορφή αρχείου. Οι προδιαγραφές μορφής αρχείου του δεν είναι διαθέσιμες δημόσια.
