## Τι είναι ένα αρχείο DEF;

Ένα αρχείο DEF είναι ένα αρχείο δεδομένων που δημιουργήθηκε σε [μορφή αρχείου υπολογιστικού φύλλου](/el/spreadsheet/) από την εφαρμογή βάσης δεδομένων SmartWare II, που αναπτύχθηκε από την Informix Software. Το λογισμικό II είχε τέσσερις ενότητες, συμπεριλαμβανομένων υπολογιστικών φύλλων με γραφικά, επεξεργαστή κειμένου, επικοινωνιών και βάσης δεδομένων. Η βάση δεδομένων SmartWare II είναι μια αυτόνομη βάση δεδομένων που μπορεί να ενσωματωθεί και με άλλες εφαρμογές. Τα ερωτήματα μπορούν να εκτελεστούν σε δεδομένα που μπορούν να εξαχθούν και να αποθηκευτούν στη μορφή αρχείου DEF. Τα αρχεία υπολογιστικών φύλλων DEF μπορούν να ανοίξουν με το λογισμικό SmartWare που έχει διακοπεί τώρα.
