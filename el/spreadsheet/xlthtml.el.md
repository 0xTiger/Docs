## Τι είναι ένα αρχείο XLTHTML;

Ένα αρχείο με επέκταση .xlthtml είναι μια έκδοση [HTML](/el/web/html/) των αρχείων προτύπων [XLT](/el/spreadsheet/xlt/) και δημιουργήθηκαν χρησιμοποιώντας το Microsoft Excel τις πρώτες μέρες. Αυτή η μορφή δεν υποστηρίζεται πλέον από το Excel και έχει αντικατασταθεί από την εξαγωγή δεδομένων σε μορφές αρχείων [MHT](/el/web/mht/) και [MHTML](/el/web/mhtml/). Η βασική ιδέα πίσω από το αρχείο XLTHTML ήταν να επιτραπεί στους χρήστες να προβάλλουν τα αρχεία προτύπων XLT σε ένα πρόγραμμα περιήγησης ιστού αντί να χρειάζεται να έχουν εγκαταστήσει το Excel στο σύστημα. Τα αρχεία που εξάγονται σε μορφή αρχείου XLTHTML μπορούν να ανοίξουν σε προγράμματα περιήγησης.

## Μορφή αρχείου XLTHTML - Περισσότερες πληροφορίες

Τα αρχεία XLTHTML χρησιμοποίησαν τη μορφή αρχείου HTML για να αποθηκεύσουν τα πρότυπα XLT για άνοιγμα σε ένα πρόγραμμα περιήγησης ιστού. Το ίδιο το XLT βασίζεται στη μορφή δυαδικού αρχείου Excel που εισήχθη με την έκδοση Excel 97-2003. Αυτά τα αρχεία μπορούν να χρησιμοποιηθούν ως αρχικά αρχεία με καθορισμένη μορφοποίηση για τη δημιουργία κανονικών εγγράφων για τον πληθυσμό δεδομένων υπολογιστικών φύλλων. Απλώς για αναφορά, το XLT βασίζεται στο δυαδικό [προδιαγραφές μορφής αρχείου XLS](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) που τεκμηριώνεται από τη Microsoft για αναφορά του προγραμματιστή.
