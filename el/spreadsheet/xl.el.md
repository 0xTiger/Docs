## Τι είναι ένα αρχείο XL;

Ένα αρχείο με επέκταση .xl είναι ένα αρχείο υπολογιστικού φύλλου Excel, παρόμοιο με το [XLS](/el/spreadsheet/xls/), το οποίο αποθηκεύει δεδομένα σε σειρές και στήλες. Ήταν μια παλαιότερη μορφή αρχείου που δεν υποστηρίζεται πλέον από το Excel τώρα. Τα δεδομένα που είναι αποθηκευμένα σε αρχεία XL θα μπορούσαν να εξαχθούν σε άλλες μορφές, όπως το CSV. Αυτά θα μπορούσαν επίσης να μετατραπούν σε PDF χρησιμοποιώντας την προεπιλεγμένη μέθοδο Αποθήκευση ως του Excel. Τα αρχεία XL μπορούσαν να ανοίξουν μόνο με το Microsoft Excel.

## Μορφή αρχείου XL

Τα αρχεία XL αποθηκεύονται ως δυαδικά αρχεία και οι εσωτερικές προδιαγραφές μορφής αρχείου τους δεν τεκμηριώνονται πουθενά. Ωστόσο, η γενική μορφή δυαδικού αρχείου για τη μορφή αρχείου [XLS File Format Specifications](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) μπορεί επίσης να ληφθεί υπόψη εάν κάποιος προσπαθεί να γράψει ένα κομμάτι κώδικα για την αντίστροφη μηχανική αυτών των αρχείων.
