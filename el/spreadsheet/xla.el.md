## Τι είναι ένα αρχείο XLA;

Ένα αρχείο με επέκταση .xla είναι μια μορφή αρχείου πρόσθετων Microsoft Excel που προσθέτει λειτουργικές μονάδες, επιπλέον λειτουργίες και άλλα εργαλεία στο Microsoft Excel. Τα Ad-In παρέχουν πρόσθετη λειτουργικότητα στα υπολογιστικά φύλλα του Excel και υποστηρίζονται στην έκδοση 2007 ή νεότερη. Τα αρχεία XLA έχουν εφαρμογή τύπου mime media/vnd.ms-excel. Αυτά μπορούν να ανοίξουν χρησιμοποιώντας το Microsoft Excel που είναι διαθέσιμο για Windows καθώς και για MacOS.

## Μορφή αρχείου XLA

Τα αρχεία XLA βασίζονται σε δυαδική μορφή δεδομένων. Ωστόσο, περισσότερες πληροφορίες σχετικά με τις προδιαγραφές μορφής αρχείου δεν είναι διαθέσιμες. Η γενική υπόθεση είναι ότι το XLA ακολουθεί τη [[MS-XLS] - Δομή μορφής δυαδικού αρχείου του Excel](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx).
