## Τι είναι ένα αρχείο CSL;

Ένα αρχείο Corel Symbol Library (CSL) είναι μια βιβλιοθήκη συμβόλων που χρησιμοποιείται στο λογισμικό CorelDRAW. Αποθηκεύει σύμβολα που μπορούν να χρησιμοποιηθούν σε άλλα αρχεία. Τα αρχεία CSL μπορούν να αποθηκευτούν τοπικά ή σε δίκτυο για να είναι προσβάσιμα για εύκολη ανάπτυξη και διαχείριση. Αυτά αποθηκεύονται με το αρχείο CorelDRAW μαζί με όλες τις περιπτώσεις συμβόλων που χρησιμοποιούνται στο έγγραφο.

## Μορφή αρχείου CSL

Τα αρχεία CSL αποθηκεύονται σε ιδιόκτητη μορφή αρχείου CorelDRAW. Όντας μια συλλογή συμβόλων, τα σύμβολα μέσα σε ένα αρχείο CSL μπορούν να εισαχθούν και να προστεθούν σε σχέδια σε ένα έργο.
