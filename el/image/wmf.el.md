## Τι είναι ένα αρχείο WMF;

Τα αρχεία με επέκταση WMF αντιπροσωπεύουν το Microsoft Windows Metafile (WMF) για την αποθήκευση δεδομένων διανυσμάτων καθώς και δεδομένων εικόνων σε μορφή bitmap. Για να είμαστε πιο ακριβείς, το WMF ανήκει στην κατηγορία διανυσματικών μορφών αρχείων των μορφών αρχείων γραφικών που είναι ανεξάρτητη από συσκευές. Το Windows Graphical Device Interface (GDI) χρησιμοποιεί τις λειτουργίες που είναι αποθηκευμένες σε ένα αρχείο WMF για να εμφανίσει μια εικόνα στην οθόνη. Μια πιο βελτιωμένη έκδοση του WMF, γνωστή ως Enhanced Meta Files (EMF), δημοσιεύτηκε αργότερα που κάνει τη μορφή πιο πλούσια σε δυνατότητες. Πρακτικά, τα WMF είναι παρόμοια με τα SVG.

## Προδιαγραφές μορφής αρχείου WMF ##

Ένα αρχείο WMF αναφερόταν σε μορφή αρχείου 16-bit τη στιγμή της εκκίνησής του με τα Windows 3.0. Η μορφή αρχείου αποτελείται από μια σειρά εγγραφών μεταβλητού μήκους που περιέχουν εντολές σχεδίασης γραφικών, ορισμούς αντικειμένων και ιδιότητες. Δεδομένου ότι τα αρχεία WMF βασίζονται σε εντολές που αποδίδονται στο GDI για τη σχεδίαση της εικόνας, είναι επίσης γνωστή ως ψηφιακή εγγραφή της εικόνας που μπορεί να αναπαραχθεί για την αναπαραγωγή αυτής της εικόνας. Οι πλήρεις προδιαγραφές μορφής αρχείου του WMF είναι διαθέσιμες στο διαδίκτυο και θα πρέπει να παραπέμπονται για ανάπτυξη εφαρμογών για εργασία με αρχεία WMF. Ένα αρχείο WMF είναι οργανωμένο σε:

* Εγγραφή κεφαλίδων WMF
* Εγγραφές WMF
* Εγγραφή τέλους αρχείου WMF

### Εγγραφή κεφαλίδας WMF ###

Η εγγραφή **META_HEADER** περιέχει πληροφορίες που καθορίζουν τα χαρακτηριστικά του μετα-αρχείου, όπως:

* Ο τύπος του μετα-αρχείου
* Η έκδοση του μετα-αρχείου
* Το μέγεθος του μετα-αρχείου
* Ο αριθμός των αντικειμένων που ορίζονται στο μετααρχείο
* Το μέγεθος της μεγαλύτερης μεμονωμένης εγγραφής στο μετααρχείο

### Εγγραφή κεφαλίδας με δυνατότητα τοποθέτησης WMF ###

Η εγγραφή **META_PLACEABLE** περιέχει εκτεταμένες πληροφορίες σχετικά με την εικόνα, όπως:

* Ένα οριοθετημένο ορθογώνιο
* Λογικό μέγεθος μονάδας, για κλιμάκωση
* Ένα άθροισμα ελέγχου, για επικύρωση

### Ρεκόρ WMF ###

Οι εγγραφές WMF έχουν μια γενική μορφή, η οποία καθορίζεται στο έγγραφο προδιαγραφών. Κάθε εγγραφή WMF περιέχει τις ακόλουθες πληροφορίες:

* Το μέγεθος του ρεκόρ
* Η λειτουργία εγγραφής
* Παράμετροι, εάν υπάρχουν, για τη συνάρτηση εγγραφής
