## Τι είναι ένα αρχείο AGIF;

Ένα αρχείο AGIF είναι ένα αρχείο GIF που δημιουργήθηκε με το Active GIF Creator. Είναι ένα αρχείο έργου που μπορεί να ανοίξει και να προβληθεί σε έργα Active GIF Creator. Όταν ανοίγει στο Active GIF Creator, εμφανίζει ακολουθία εικόνων που τελικά αναπληρώνουν το τελικό αρχείο GIF με επέκταση .gif.

## Μορφή αρχείου AGIF

Το AGIF είναι ένα αρχείο έργου που αποθηκεύεται ως μορφή αρχείου εικόνας ράστερ. Το οριστικοποιημένο αρχείο [GIF](/image/gif/) μπορεί να ανοίξει και να εμφανιστεί ως κινούμενη εικόνα. Υπάρχουν Python API που είναι διαθέσιμα στο διαδίκτυο που μπορούν να χρησιμοποιηθούν για τη δημιουργία κινούμενων αρχείων GIF.
