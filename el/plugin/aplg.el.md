## Τι είναι ένα αρχείο APLG;

Ένα αρχείο με επέκταση .aplg είναι ένα αρχείο προσθήκης που χρησιμοποιείται από το λογισμικό εγγραφής ροής Internet Audials. Το πρόσθετο φορτώνεται από το λογισμικό για την προσθήκη νέων χαρακτηριστικών, όπως νέες λειτουργίες αναζήτησης και εγγραφής, ώστε να καταγράφονται ροές μουσικής και βίντεο στο Διαδίκτυο. Η Audials προσφέρει ένα πλούσιο σε χαρακτηριστικά ραδιοφωνικό πρόγραμμα αναπαραγωγής που επιτρέπει στους χρήστες να έχουν πρόσβαση και να ακούν έναν τεράστιο αριθμό διαδικτυακών ραδιοφωνικών σταθμών.

## Μορφή αρχείου APLG

Τα αρχεία APLG διατίθενται ως συσκευασμένα στα πακέτα προσθηκών .APLP, πράγμα που σημαίνει ότι το λογισμικό Audials μπορεί να φορτώσει πολλά πρόσθετα από το πακέτο ταυτόχρονα. Οι χρήστες μπορούν συχνά να ηχογραφήσουν τα αγαπημένα τους τραγούδια ή ραδιοφωνικές εκπομπές για μελλοντική αναπαραγωγή. Αυτό μπορεί να είναι χρήσιμο για τη δημιουργία εξατομικευμένων λιστών αναπαραγωγής. Η Audials διατηρεί συνήθως μια μεγάλη βάση δεδομένων ραδιοφωνικών σταθμών, διευκολύνοντας τους χρήστες να ανακαλύψουν νέο περιεχόμενο.
