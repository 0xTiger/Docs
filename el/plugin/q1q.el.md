## Τι είναι ένα αρχείο Q1Q;

Ένα αρχείο .q1q είναι ένα αρχείο προσθήκης Photoshop που αναπτύχθηκε από την Flaming Pear Software για τη δημιουργία τρισδιάστατων πλανητών. Σας επιτρέπει να κατεβάζετε LIVE δεδομένα σύννεφων, αέρα, υφή, κλίμα και άλλους τόνους λεπτομερειών. Σας επιτρέπει επίσης να εξάγετε όλους τους χάρτες υφής σε αρχεία πολλαπλών επιπέδων και να δημιουργείτε χάρτες για τρισδιάστατα κινούμενα σχέδια και δημιουργία χαρτών.

## Μορφή αρχείου Q1Q

Τα αρχεία προσθήκης Q1Q αποθηκεύονται σε ιδιόκτητη μορφή αρχείου Flaming Pear και οι λεπτομέρειες της εσωτερικής μορφής αρχείου δεν είναι διαθέσιμες δημόσια. Η ακόλουθη εικόνα δείχνει τη διεπαφή χρήστη του plugin Q1Q όταν φορτώνεται στο Photoshop.

## Πώς να ανοίξετε το αρχείο Q1Q;

Μπορείτε να ανοίξετε αρχεία Q1Q χρησιμοποιώντας το **Adobe Photoshop 2023** και το **Corel PaintShop Pro 2023**.
