## Τι είναι ένα αρχείο AMXX;

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. Είναι μια πλατφόρμα δέσμης ενεργειών που επιτρέπει στους διαχειριστές διακομιστών να προσαρμόζουν και να επεκτείνουν τη λειτουργικότητα των διακομιστών τους. Το AMXX χρησιμοποιείται σε συνδυασμό με το σύστημα διαχείρισης προσθηκών MetaMod που επιτρέπει την ταυτόχρονη εκτέλεση πολλαπλών προσθηκών σε έναν διακομιστή χωρίς διενέξεις. Οι δυνατότητες που προσφέρει η προσθήκη AMXX περιλαμβάνουν εντολές διαχειριστή, τροποποιήσεις παιχνιδιού και διαχείριση διακομιστή.
