## Τι είναι ένα αρχείο SPARC;

Ένα αρχείο .sparc, που αναγνωρίζεται εναλλακτικά ως Αρχείο Προσθηκών Skype, χρησιμεύει ως πακέτο που περιλαμβάνει επιπλέον λειτουργίες ή βελτιώσεις για την εφαρμογή Skype. Αυτά τα πρόσθετα έχουν την ικανότητα να εισάγουν ένα ευρύ φάσμα δυνατοτήτων όπως εγγραφή κλήσεων, κοινή χρήση οθόνης, μετάφραση,
Παιχνίδια και ψυχαγωγία, προσαρμοσμένα emoticon και θέματα.

## Μορφή αρχείου SPARC - Περισσότερες πληροφορίες

Σε ορισμένες εκδόσεις του Skype, το Extras Manager είναι ένα προαιρετικό στοιχείο που μπορεί να ενσωματωθεί στην εγκατάσταση του Skype. Εάν αντιμετωπίσετε δυσκολίες κατά την εγκατάσταση ενός αρχείου SPARC, είναι πιθανό να παραλείψατε το Extras Manager κατά την αρχική εγκατάσταση. Για να αντιμετωπίσετε αυτό το ζήτημα, εγκαταστήστε ξανά το Skype και βεβαιωθείτε ότι έχετε επιλέξει την επιλογή Extras Manager.

## Πώς να εγκαταστήσετε το πρόσθετο Skype;

Βήματα για την εγκατάσταση προσθηκών Skype:

1. Αποκτήστε το αρχείο .sparc που αντιστοιχεί στο επιθυμητό πρόσθετο.
2. Κλείστε το Skype εάν εκτελείται αυτήν τη στιγμή.
3. Εντοπίστε το ληφθέν αρχείο .sparc και κάντε διπλό κλικ πάνω του.
4. Ακολουθήστε τις οδηγίες που εμφανίζονται στην οθόνη για να ολοκληρώσετε την εγκατάσταση της προσθήκης.
5. Μετά την εγκατάσταση, επανεκκινήστε το Skype για να ενεργοποιήσετε την προσθήκη.
