## Τι είναι ένα αρχείο RBZ;

Ένα αρχείο RBZ είναι ένα αρχείο προσθήκης που χρησιμοποιείται από το πρόγραμμα μοντελοποίησης, **Trimble SketchUp**, για σχεδιασμό με τη βοήθεια υπολογιστή (CAD). Είναι ένα συμπιεσμένο αρχείο ZIP που περιέχει ένα ή περισσότερα σενάρια Ruby (αρχεία .rb) μέσα του. Γι' αυτό ο τύπος αρχείου είναι RBZ (RB + Z). Τα αρχεία RBZ διανέμονται εύκολα και εγκαθίστανται μέσω του δικτύου λόγω του μικρού μεγέθους τους ως αποτέλεσμα της συμπίεσης.

## Μορφή αρχείου RBZ - Περισσότερες πληροφορίες

Τα αρχεία RBZ είναι συμπιεσμένα αρχεία ZIP που περιέχουν ένα αρχείο σεναρίου Ruby. Συμπιέζεται με την πιο συχνά χρησιμοποιούμενη μέθοδο συμπίεσης **DEFLATE**, η οποία περιγράφεται στο IETF **RFC 1951**.

## Πώς να ανοίξετε ένα αρχείο RBZ;

Μπορείτε να ανοίξετε αρχεία RBZ με το Trimble SketchUp στα Windows καθώς και στο macOS.

Εκτός από αυτά, καθώς τα αρχεία RBZ είναι συμπιεσμένα αρχεία ZIP, μπορείτε να τα εξαγάγετε με τυπικά βοηθητικά προγράμματα εξαγωγής ZIP μετονομάζοντας την επέκτασή του από .rbz σε .zip.
