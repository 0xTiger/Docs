## Che cos'è un file MCWORLD?
Un file MCWORLD è costituito da file di risorse e comportamenti per creare un mondo virtuale in Bedrock Edition di Minecraft, un famoso gioco di costruzione di blocchi. Ha compresso vari file, come file .DAT e .TXT con compressione ZIP. i file utilizzano l'estensione di file .mcworld e questi sono particolarmente utilizzati per eseguire il backup o trasferire un mondo a un altro utente di Minecraft. Questi file sono solitamente creati da un software Minecraft chiamato Minecraft Launcher.

## Formato file MCWORLD
I file MCWORLD sono creati da **MinecraftSP.exe**, chiamato anche Minecraft Launcher. Questo file può essere creato se abbiamo compresso o compresso i file world in un unico file e cambiamo l'estensione in .mcworld. Dall'altro lato possiamo espandere i file .mcworld utilizzando semplici programmi di utilità zip decompressor, perché i file .mcworld vengono compressi utilizzando la compressione zip.
