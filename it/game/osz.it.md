## Cos'è un file OSZ?

Un file OSZ è un formato di file compresso utilizzato dal gioco ritmico osu! per memorizzare i dati della beatmap. Le beatmap sono essenzialmente livelli per il gioco e includono informazioni come la canzone, il ritmo del ritmo e il posizionamento degli oggetti colpiti sullo schermo di gioco. I file OSZ consentono una facile distribuzione delle beatmap e vengono generalmente scaricati da Internet e importati nel gioco. OSU! ha anche il formato file [OSR](/game/osr/) che è un file di replay del gioco e può essere riprodotto dai giocatori.

**Tipo MIME del formato file OSR:** x-osu-replay

## Formato file OSZ

I dettagli sul formato di file interno dei tipi di file OSZ non sono disponibili pubblicamente. Contiene dati compressi [ZIP](/compression/zip/) che contengono informazioni per riprodurre la musica, mostrare grafica e mostrare ai giocatori gli eventi in cui devono interagire con il gioco.

## Come creare un file OSZ?

OSU! contiene istruzioni dettagliate per i file [creating OSZ](https://osu.ppy.sh/wiki/en/Client/File_formats#creating-.osz-and-.osk-files) e OSK. È possibile utilizzare i seguenti passaggi per creare un file OSZ utilizzando OSU!.

1. Apri una beatmap tramite l'editor.
1. Dal menu in alto, seleziona File > Esporta pacchetto....
1. L'archivio .osz verrà inserito nella cartella Esportazioni all'interno di osu! cartella.
