## Cos'è un file USX?

Un file USX contiene le mesh di Unreal Engine che sono oggetti/modelli 3D pre-progettati utilizzati per generare l'ambiente. I modelli del file USX hanno posizione, orientamento e dimensioni diverse, ma rimangono statici. Le mesh statiche possono essere ottimizzate per le prestazioni e possono essere utilizzate per creare vari livelli di dettaglio (LOD) per migliorare le prestazioni del gioco. Possono anche essere utilizzati per creare volumi di collisione e possono essere utilizzati come parte di simulazioni fisiche.

I file USX possono essere creati con l'editor di livelli Unreal UnrealEd.

## Formato file USX: ulteriori informazioni

I file USX vengono salvati su disco come file binari. I dettagli della struttura interna del file del formato file USX non sono disponibili pubblicamente come riferimento per gli sviluppatori.
