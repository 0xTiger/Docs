## Che cos'è un file UMX?

Un file UMX è un file di pacchetto musicale utilizzato dal gioco sparatutto in prima persona Unreal Tournament, sviluppato da Epic Games, e contiene file tracker di musica di sottofondo al suo interno. Ecco perché è spesso considerato solo un'altra estensione di file di pacchetto contenente file tracker come .s3m e .it. Un tracker musicale è una sorta di software di sequencer musicale per la creazione di brevi file musicali utilizzando note musicali discrete. Diverse varianti di file UMX contengono uscite audio come esplosioni o passi. I file UMX funzionano con le versioni precedenti al rilascio di Unreal Engline 4.

## Formato file UMX - Ulteriori informazioni

I file UMX vengono salvati su disco in formato file binario utilizzando il formato file proprietario di Epic e i dettagli del formato file interno non sono disponibili pubblicamente. I file UMX possono essere aperti con applicazioni di localizzazione musicale come XimpleMOD e utility Unreal come Epic Games Unreal Tournament.
