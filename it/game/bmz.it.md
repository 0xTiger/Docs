## Che cos'è un file BMZ?

Un file BMZ è un file mappa bonus utilizzato dal gioco di risoluzione di puzzle 3D "Portal" sviluppato da Valve. Contiene una mappa bonus che può essere creata anche da sviluppatori di comunità di terze parti e importata nel gioco utilizzando l'interfaccia utente integrata. BMZ viene fornito come file di archivio [ZIP](/it/compression/zip/), contenente file BSP, TGA e [BNS](/it/game/bns/). I file BMZ possono essere aperti utilizzando Valve Portal e Valve Portal 2.

## Formato file BMZ

BMZ vengono salvati come file ZIP e possono essere estratti con utilità di estrazione ZIP standard come WinZIP o WinRAR. Quando viene estratto un file BMZ, è possibile trovare i seguenti file:

* BSP - Il file della mappa
* TGA - File del logo
* BNS - File di script della mappa bonus del portale

## Come creare e installare file di mappe bonus

La comunità di sviluppatori di Valve Software ha stabilito linee guida per le persone che desiderano [creare e installare file Bonus Map](https://developer.valvesoftware.com/wiki/Bonus_Maps). Include i seguenti passaggi:

1. Aggiunta di mappe bonus BSP al mod
1. Aggiunta di un'immagine in miniatura
1. Creazione di file di descrizione del bonus (.BNS)
