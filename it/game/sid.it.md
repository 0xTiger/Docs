## Che cos'è un file SID?

Un file SID è un file di backup del gioco creato dall'applicazione Steam che viene utilizzato per scaricare e gestire **videogiochi Valve** su sistema operativo Windows. Memorizza i dati di gioco in formato compresso e può essere utilizzato come backup dei giochi su supporti di archiviazione come DVD e dispositivi di archiviazione USB. Il file SID viene creato con l'utilità di backup di Steam ed è simile al formato di file [CSD](/it/game/csd/) per la memorizzazione del backup del gioco.

## Formato file SID - Ulteriori informazioni

I file SID vengono archiviati su disco come file binari. È sempre accompagnato da un file .sis e .sim che vengono utilizzati per estrarre informazioni sui metadati per ripristinare i dati di gioco di backup.
