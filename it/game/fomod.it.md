## Che cos'è un file FOMOD?

Un file FOMOD è un file di gioco che contiene le modifiche da installare per un gioco. Questi file vengono installati da programmi mod come Fallout mod manager (FOMM) e Nexus Mod Manager (NMM). Le modifiche nei file .fomod possono essere correlate a qualsiasi aspetto del gioco e sono pubblicate dai giocatori o dai fan della community del gioco. Le mod sotto forma di FOMOD possono prolungare la vita di un gioco cambiando, includendo piccole modifiche o una revisione completa del gioco.

## Formato file FOMOD - Ulteriori informazioni

I file FOMOD vengono generalmente salvati come archivio compresso utilizzando i formati di compressione [.7z](/it/compression/7z/), [.zip](/it/compression/zip/) o [.rar](/it/compression/rar/). In FOMM, questo file viene caricato e installato nel sistema.

## Come installare un file .fomod?

Un file FOMOD può essere installato facendo doppio clic su di esso. Questo lo aprirà utilizzando il FOOM con il nome della modalità nell'elenco che può essere attivato selezionando e premendo l'opzione di attivazione.
