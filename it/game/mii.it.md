## Cos'è un file MII?

Un file MII è un formato di file avatar virtuale utilizzato per memorizzare avatar virtuali sulla console di gioco Nintendo Wii. Questi file contengono informazioni come l'aspetto, i movimenti e le animazioni dell'avatar. Possono essere utilizzati in giochi, applicazioni di social network e altri software sul Wii. Un file MII contiene anche altre caratteristiche dell'avatar come sesso, colore dei capelli, colore della pelle, caratteristiche del viso e tipo di occhi.

Puoi aprire un file MII utilizzando [My Avatar Editor](https://rc24.xyz/goodies/mii/).

## Formato file MII

Il formato del file MII è definito in dettaglio su [Wii Brew](https://wiibrew.org/wiki/Mii_data). Le stringhe in un file MII vengono archiviate in formato Unicode (UTF-16), big-endian.

### Blocco MI

I dati dei file MII vengono archiviati sul Wiimote in due blocchi. Ogni blocco ha una lunghezza di 750 byte, con CRC da 2 byte, per un totale di 752 byte. Ogni blocco inizia con un valore di 4 byte (RNCD) che sembra essere il numero magico per il formato file MII.

## Come creare file MII?

Esistono diversi modi per creare avatar per Nintendo Wii:

1. Utilizzo del canale Mii integrato nella Wii: questo canale ti consente di creare avatar personalizzati utilizzando una varietà di strumenti, inclusi i tratti del viso, la forma del corpo e l'abbigliamento. Una volta creato il tuo avatar, puoi salvarlo come file Mii e utilizzarlo nei giochi e in altri software sulla Wii.

1. Utilizzo dell'app Creazione Mii su Nintendo 3DS: L'app Creazione Mii ti consente di creare avatar personalizzati utilizzando il touch screen e la fotocamera del tuo Nintendo 3DS. Una volta creato il tuo avatar, puoi salvarlo come file Mii e trasferirlo sulla tua Wii tramite una connessione wireless.

1. Utilizzo di software di terze parti: alcuni sviluppatori hanno creato software che consente di creare avatar personalizzati per Wii. Questo software è generalmente progettato per l'uso su un computer e potrebbe richiedere passaggi aggiuntivi per trasferire l'avatar sulla tua Wii.

1. Utilizzo di avatar predefiniti: alcuni giochi e altri software sul Wii potrebbero essere dotati di avatar predefiniti che è possibile utilizzare.

In tutti i casi, devi avere un account Nintendo per creare e salvare il tuo avatar sulla Wii.
