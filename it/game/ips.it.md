## Cos'è un file IPS?

Un "file IPS" si riferisce in genere a un "file di patch del sistema di patching interno". IPS è l'abbreviazione di "International Patching System" ed è un formato di file utilizzato per creare e applicare patch ai file ROM (memoria di sola lettura), spesso nel contesto dell'emulazione di videogiochi e dell'hacking della ROM.

Ecco come funziona:

- **ROM originale:** inizi con un file ROM originale, che è essenzialmente una copia di un videogioco o di un software archiviato in un formato ROM. Questo file è in genere di sola lettura, ovvero non è possibile modificarlo direttamente.

- **File patch IPS:** un file patch IPS contiene le modifiche che desideri applicare alla ROM originale. Queste modifiche potrebbero essere correzioni di bug, traduzioni o altre modifiche.

- **Applicazione patch:** per applicare le modifiche, è necessaria un'utilità o un programma in grado di leggere il file patch IPS e applicarlo alla ROM originale. Questo processo essenzialmente "corregge" il file ROM originale con le modifiche specificate nel file IPS, creando un file ROM modificato.

- **ROM modificata:** Il risultato è una ROM modificata che incorpora le modifiche del file patch IPS. Questa ROM modificata può quindi essere utilizzata in un emulatore o su hardware compatibile per giocare con le modifiche desiderate.

Queste patch vengono generalmente utilizzate per apportare modifiche o miglioramenti ai giochi e possono comprendere varie modifiche, comprese modifiche alla grafica, ai modelli e ad altri dati di gioco. I file IPS sono particolarmente adatti per patch relativamente piccole, in genere di dimensioni inferiori a 16 megabyte.

Nella sezione seguente esploreremo le applicazioni software associate ai file IPS.

## IPS lunare

Lunar IPS è un'utilità popolare e intuitiva utilizzata per creare e applicare patch IPS (Internal Patching System) ai file ROM. Ecco alcune caratteristiche e informazioni chiave su Lunar IPS:

- **Creazione di patch:** Lunar IPS consente agli utenti di creare patch IPS specificando le modifiche che desiderano apportare a un file ROM originale. Puoi selezionare la ROM originale e la versione modificata e Lunar IPS genererà un file patch IPS in base alle differenze tra i due file.

- **Applicazione patch:** una volta ottenuto un file patch IPS, Lunar IPS ti consente di applicarlo a una ROM originale. Questo processo essenzialmente "patch" la ROM con le modifiche specificate nel file IPS, creando una ROM modificata.

- **Leggero:** Lunar IPS è un programma piccolo e leggero, il che significa che non consuma molte risorse di sistema ed è veloce da scaricare ed eseguire.

## IPSWin

IPSWin è un'utilità progettata principalmente per gli utenti Windows che facilita la creazione e l'applicazione di patch IPS (Internal Patching System) ai file ROM. È uno strumento semplice e facile da usare, adatto a coloro che desiderano applicare facilmente patch alle ROM. Ecco alcune informazioni su IPSWin:

- **Creazione di patch:** IPSWin consente agli utenti di creare patch IPS confrontando due file ROM: la ROM originale e la ROM modificata. Identifica le differenze tra i due file e genera un file patch IPS contenente tali modifiche.

- **Applicazione patch:** questa utility consente inoltre agli utenti di applicare file patch IPS esistenti alle ROM originali. Selezionando la patch IPS appropriata e la ROM originale, IPSWin unirà le modifiche in un nuovo file ROM, creando una versione con patch.
