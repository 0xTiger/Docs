## Cos'è un file OSB?

Un file OSB è un file storyboard creato dal gioco ritmico Osu! che consente ai giocatori di toccare gli elementi visivi sullo schermo al ritmo della musica. Lo storyboard all'interno del file OSB contiene uno sfondo animato personalizzato che viene utilizzato dietro il gioco durante il gioco. Lo sfondo dello storyboard è in genere un file JPG o PNG che viene mostrato con il file beatmap OSZ durante il gioco.

## Formato file OSB

I file OSB vengono archiviati come file di testo semplice sul disco nelle cartelle beatmap. Questo schema consente di creare diversi storyboard con diverse difficoltà all'interno di una beatmap.
