## Che cos'è un file KODU?

Un file .kudo è un file di gioco 3D creato e salvato con Microsoft [software Kodu Game Lab](https://www.kodugamelab.com/) che è un ambiente di sviluppo di giochi educativi. È progettato per insegnare ai bambini i principi di programmazione di base e consente ai creatori di programmare i comportamenti degli elementi di gioco utilizzando il linguaggio di programmazione visivo. I file Kodu possono essere salvati e condivisi con altri utenti, ma possono essere aperti ed eseguiti solo nel software Kodu.

## Formato file KODU - Ulteriori informazioni

I file Kodu sono file binari, salvati in un formato di file proprietario di Microsoft. Le sue specifiche sul formato file non sono disponibili pubblicamente per gli sviluppatori ed è per questo che i file Kodu possono essere eseguiti solo all'interno del software Kodu Game Lab.
