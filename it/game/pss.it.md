## Che cos'è un file PSS?

Un file con estensione .pss è un file video di gioco Sony PlayStation 2 e viene utilizzato per archiviare dati audio e video dai giochi per la console Playstation 2. Questi dati video e audio sono in realtà filmati che possono essere estratti dai file PSS utilizzando utility gratuite. Una di queste utility è [RipBot](http://forum.doom9.org/showthread.php?t=127611) che è stata sviluppata in Delphi per estrarre i contenuti dal file PSS. I dati dei file PSS possono essere suddivisi nei suoi componenti video [MPEG2](/it/video/mpg/) e [WAV](/it/audio/wav/) e convertiti in diversi formati di file audio e video. I file PSS possono essere aperti con applicazioni come VideoLAN VLC media player, PSS Plex e PSS Player.

## Formato file PSS - Ulteriori informazioni

PSS sono file binari i cui dettagli sul formato file interno non sono disponibili pubblicamente. Nonostante la non disponibilità delle informazioni sulla struttura del file PSS, alcune utilità sono in grado di suddividere il contenuto di un file PSS in formati di file WAV video MPEG2 e audio separati.
