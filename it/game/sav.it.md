## Che cos'è un file SAV?

Un file con estensione .sav è un file di gioco salvato per Nintendo DS che può essere caricato in emulatori di giochi per Nintendo DS come [DeSmuME](http://desmume.org/) e [NO$GBA](https://www. nogba.com/). Questi file salvano lo stato corrente del gioco che può quindi essere caricato negli emulatori Nintendo. Le informazioni memorizzate nei file SAV includono il nome del personaggio del gioco, i punti esperienza, i livelli completati e le risorse di gioco. I file SAV possono essere caricati negli emulatori, ma la differenza nelle versioni del file SAV può portare alla generazione di errori negli emulatori. I file SAV possono essere aperti con gli emulatori DeSmuME e NO$GBA.

## Formato file SAV - Ulteriori informazioni

I file Nintendo DS SAVE vengono salvati su disco in formato binario. Sebbene non siano leggibili dall'uomo, è possibile utilizzare un editor esadecimale come [HxD](https://mh-nexus.de/en/hxd/) per aprirli per l'analisi. Negli scenari in cui il file SAV non può essere caricato in una versione aggiornata di un emulatore, può essere aperto nell'editor esadecimale per la rimozione di eventuali dati riempiti indesiderati.
