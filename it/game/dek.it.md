## Che cos'è un file DEK?

Un file con estensione .dek è un file di mazzo di carte utilizzato dal gioco di auto da collezione online, Magic: The Gathering Online (MTGO). Sebbene in più casi questi mazzi di carte siano riproducibili, in alcuni casi i file DEK elencano una raccolta di carte non riproducibili. Questi non possono essere riprodotti ma agiscono come un sottoinsieme di una raccolta. I file DEK possono essere creati da file [CSV](/it/spreadsheet/csv/) esportandoli utilizzando Delver Lens e possono essere utilizzati in Tabletop Simulator.

## Formato file DEK

I file DEK vengono archiviati come file di testo semplice multiriga in formato [XML](/it/web/xml/) leggibili dall'uomo. Questi possono essere aperti con qualsiasi editor di testo come Microsoft Notepad, Notepad++ e Wizards of the Coast Magic: The Gathering Online. I file DEK sono utilizzati anche da molti giocatori di Magic per importare/esportare da molti siti web e di costruzione di mazzi come [CardHoarder](https://www.cardhoarder.com/). Questi siti Web consentono di caricare file MTGO DEK come un singolo o elenco di file.
