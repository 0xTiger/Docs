## Che cos'è un file W3N?

Un file con estensione .w3n è un file di mappe della campagna utilizzato da [Warcraft III: The Frozen Throne](https://en.wikipedia.org/wiki/Warcraft_III:_The_Frozen_Throne) che è un'estensione di Warcraft III, un videogioco famoso del tempo. Il file W3N contiene mappe della campagna che vengono utilizzate per creare campagne personalizzate utilizzando la serie di dati delle missioni disponibili in esso. Il formato è spesso frainteso poiché i file delle mappe e gli utenti non hanno idea del motivo per cui i loro file W3N non vengono caricati in Frozen Throne. I file W3N possono essere aperti con Blizzard Warcraft 3: The Frozen Throne.

## Formato file W3N - Ulteriori informazioni

I file W3N vengono archiviati su disco come file binari e il loro formato di file interno non è disponibile pubblicamente. Il componente Campaign Editor di Warcraft III World Editor consente però di creare file W3N.

## Come installare i file W3N?

Non sbagliare posizionando i file W3N nella cartella "mappe" del tuo computer. I file W3N non sono file mappa, ma campagne che possono essere utilizzate per creare missioni. Pertanto, per caricare questi file, questi devono essere inseriti nella cartella "campagne" e non nella cartella "mappe".

* Azeroth reinventato
* Giorno del Drago
* Signore dei Clan
