## Che cos'è un file CON?

Un file CON è un file di configurazione utilizzato dal videogioco sparatutto in prima persona **Battlefield 1942**. Contiene informazioni di configurazione che vengono caricate e utilizzate dal gioco. Esistono più file .con che definiscono diversi tipi di oggetti come azioni bot, danni alle armi, animazioni, livelli sonori o altre caratteristiche.

## Formato file CON

I file CON vengono salvati su disco come file di testo semplice che possono essere aperti in qualsiasi editor di testo. Le impostazioni vengono salvate in righe di testo che possono essere modificate e salvate per essere ricaricate dal gioco.
