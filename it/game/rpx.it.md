## Che cos'è un file RPX?

Un file RPX è un file di gioco che può essere caricato o riprodotto dalla console di gioco Wii U. Può anche essere caricato e riprodotto nell'emulatore di videogiochi Cemu, popolare per fornire un ambiente di emulazione per i file Wii U. I file di gioco Wii U sono di grandi dimensioni e possono anche essere salvati come file [WUD](/it/disc-and-media/wud/) o [WUX](/it/compression/wux/) per essere caricati dagli emulatori di giochi. Un file RPX contiene l'intero gioco eseguibile che comprende tutti i componenti del gioco come grafica, trame, sequenze e la ROM del gioco.

## Formato file RPX - Ulteriori informazioni

I file RPX vengono salvati su disco in un formato di file proprietario. Alcune API sono disponibili per comprimere o decomprimere file RPX come [wiiurpxtool](https://github.com/0CBH0/wiiurpxtool). Inoltre, applicazioni come l'emulatore di videogiochi Cemu ti consentono di giocare ai giochi Wii U in Windows su PC.
