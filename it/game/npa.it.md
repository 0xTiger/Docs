## Cos'è un file NPA?

Un file NPA è un formato di archivio utilizzato dal nuovo videogioco Steins;Gate, sviluppato da 5pb e Nitroplus. Contiene risorse di gioco a cui si fa riferimento per le voci dei personaggi o per la colonna sonora del gioco. Esempi di file di risorse inclusi in un file NPA includono immagini [JPG](/image/jpeg/) e [PNG](/image/png/) per file di sfondo o audio.

## Formato file NPA

I dettagli interni dei tipi di file NPA non sono disponibili pubblicamente.

## Cos'è il gioco Steins Gate?

Steins Gate è una serie di acclamati [visual-novel adventure games](https://www.giantbomb.com/steinsgate-series/3025-2128/games/) creati da 5 pb. È anche diventato la base di numerosi manga, un audiodramma e film d'animazione. Alcuni di questi includono Elite, Divergencies Assort e Gate 0.
