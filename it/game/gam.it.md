## Che cos'è un file GAM?
Un file con estensione .gam viene utilizzato da vari videogiochi per coinvolgere i giocatori e continuare da dove si erano fermati la prossima volta che il programma viene aperto. Quindi, significa che il file GAM salva le informazioni di ciò che un utente ha fatto a che ora. Questi file possono essere salvati automaticamente dai software di gioco o ai giocatori potrebbe essere chiesto di salvare se vogliono continuare lo stato del gioco da cui stanno partendo.
## Formato file GAM
Il formato file GAM viene fondamentalmente utilizzato per conservare i dati di gioco nei salvataggi. Il file GAM non memorizza informazioni su creature, oggetti o aree, ma salva i dati sui membri del gruppo e le variabili generali o globali che influenzano i membri del gruppo.
### Struttura del file GAM
Il file GAM ha la seguente struttura:
- Intestazione
- Membri del partito
- Dati del file CRE membro del partito
- NPC (che non sono nel gruppo)
- Statistiche uccisioni NPC (incorporate nella struttura NPC)
- Dati del file NPC CRE
- Variabili (nello spazio dei nomi GLOBAL)
- Voci del diario
- Informazioni familiari
- Posizioni memorizzate
- Posizioni dell'aereo tascabile
- Extra familiare
