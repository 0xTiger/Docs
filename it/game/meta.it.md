## Cos'è un file META?

Il file META contiene i dati di gioco di vari giochi che utilizzano Exient XGS Engine come Angry Birds Go! o i trasformatori di Angry Birds. Il file META acquisisce e conserva informazioni su aspetti particolari del gioco come i registri degli arresti anomali o i registri degli oggetti.

I file META vengono generati automaticamente dal gioco e non sono destinati alla modifica dell'utente o all'accesso diretto. Se il file META è dedicato alla registrazione degli arresti anomali del gioco, verrà generalmente denominato "crashes.meta". Allo stesso modo, se il file META viene utilizzato per registrare informazioni relative agli elementi, seguirà la convenzione di denominazione "XXX.xml.meta", dove "XXX" rappresenta un identificatore descrittivo per il tipo specifico di elementi registrati.

## Motore XGS esistente

Exient Entertainment ha creato Exient XGS Engine, un motore di gioco. È un framework software creato per semplificare il processo di creazione e sviluppo dei videogiochi. Offre ai programmatori una varietà di strumenti, librerie e funzioni per accelerare il processo di creazione del gioco.

Una varietà di funzionalità e capacità fornite da Exient XGS Engine semplificano la creazione di giochi su una varietà di piattaforme, inclusi PC, console e dispositivi mobili. Queste funzioni consistono spesso nel produrre immagini, simulare la fisica, gestire l'audio, gestire l'input, avere funzionalità di rete e altre cose.

## Come aprire il file META?

I seguenti programmi possono aprire file META relativi al gioco utilizzando Exient XGS Engine

- Trasformatori di Rovio Angry Birds
- Rovio Angry Birds Go!
