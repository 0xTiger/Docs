## Cos'è un file SIMS3PACK?

Un file SIMS3PACK è un pacchetto di gioco creato per il gioco di simulazione di vita a tempo indeterminato, **The Sims 3**. Nel caso in cui non hai mai giocato a questo gioco, Sims 3 rappresenta i giocatori partecipanti in un ambiente simulato come persone virtuali chiamate **Sims**. Il file SIMS3PACK contiene in realtà informazioni che rappresentano parti dell'ambiente virtuale come persone, vestiti ed edifici Sim.

## Formato file SIMS3PACK

I file SIMS3PACK vengono salvati in formato file binario, senza informazioni disponibili sulla struttura del file interno. Tuttavia, sono disponibili editor di pacchetti di Sims 3 di terze parti che possono essere utilizzati per la generazione di file SIMS3PACK. Questi file SIMS3PACK possono essere caricati nel gioco semplicemente facendo doppio clic su di essi, a condizione che il gioco sia installato sul sistema.

### Come installare il file del pacchetto SIMS3PACK?

I Sims3Pack hanno l'estensione .sims3pack e possono essere installati seguendo i seguenti passaggi.

 * Scarica ed estrai il tuo CC in Documenti\Electronic Arts\The Sims 3\Download\.
 * Per installare un solo Sims3Pack, fai doppio clic su di esso e il Launcher si aprirà e lo installerà da solo.
