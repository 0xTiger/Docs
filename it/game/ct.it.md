## Che cos'è un file CT?

Un file CT è un file cheat table creato con il software [Cheat Engine](https://www.cheatengine.org/) che viene utilizzato per creare modifiche ai giochi basati su Windows per barare. Il Cheat Engine, un motore di cheating open source, esamina i giochi in esecuzione e registra le posizioni dei suoi indirizzi. Queste informazioni e le relative sostituzioni vengono scritte nel file CT che viene quindi caricato dai giocatori per modificare le proprietà del gioco come il punteggio di salute, il punteggio più alto e le vite rimanenti.

## Formato file Cheat Engine CT

I file CT vengono salvati con le posizioni degli indirizzi e altre informazioni associate da hackerare all'interno di un gioco. Nella maggior parte dei casi, i file vengono salvati come archivi compressi [ZIP](/it/compression/zip/) che possono essere facilmente estratti utilizzando qualsiasi utilità di decompressione standard come WinZIP o 7-Zip.

## Come usare le tabelle cheat

1. Scarica la tabella e copiala nella cartella di Cheat Engine
1. Avvia Cheat Engine
1. Esegui il gioco;
1. Usa la combinazione di ALT+TAB e scegli il gioco nell'elenco dei processi tramite Cheat Engine
1. Se la tabella si trova in una cartella diversa, premi semplicemente Control+O e guida Cheat Engine in quella cartella. Quindi seleziona la tabella (solitamente process_name.ct);
1. Una volta caricata la tabella, se è presente uno script, basta controllarlo.
1. ALT+TAB torna al gioco e divertiti.
