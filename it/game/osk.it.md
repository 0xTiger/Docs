## Cos'è un file OSK?

Un file OSK è un file di layout o skin utilizzato dal gioco ritmico osu! dove i giocatori usano mouse e tastiera in sincronia con la musica. Contiene il layout dell'interfaccia dell'editor e può essere personalizzato dai giocatori in base alle loro preferenze. osu! memorizza i dati della beatmap nel formato file [OSZ](/game/osz/).

**Tipo MIME del formato file OSK:** x-osu-skin-archive

Puoi aprire il file OSK con [osu!](https://osu.ppy.sh/home).

## Formato file OSK

I file OSK vengono salvati sul disco come file compressi [ZIP](/compression/zip/). Puoi semplicemente mantenere un file .osk in .zip ed estrarre l'archivio per visualizzarne il contenuto. I contenuti estratti devono essere inseriti nella directory \Skins\ per essere caricati nel gioco.

## Cosa può essere personalizzato con il file OSK?

I seguenti elementi di gioco possono essere personalizzati con il file .osk.

 * Colpisci i cerchi
 * Visualizzazione del punteggio
 * Tipo di cursore
 * Cursore
 * Colpisci a raffica
