## Cos'è un file GCF?

Un file .gcf è un file di cache del gioco utilizzato dalla piattaforma di gioco Steam. Contiene dati di gioco come trame, modelli e altri file utilizzati dal gioco. Questi file vengono archiviati sul computer dell'utente e vengono utilizzati per ridurre la quantità di dati che devono essere scaricati durante l'aggiornamento o l'installazione di un gioco. I file .gcf possono essere utilizzati anche per creare backup dell'installazione di un gioco o per trasferire un gioco tra computer.

I file GCF possono essere letti e scritti dalla libreria di programmazione C++ Open Source, **HLLib**.

## Formato file GCF

I file GCF vengono salvati su disco come file binari. GCF era originariamente utilizzato come acronimo per Grid Cache File (il primo nome in codice di Steam era Grid), ma in seguito fu preso come Game Cache File. Un file GCF è un file di archivio che memorizza i giochi Steam. Tutti i contenuti ufficiali vengono scaricati come file GCF. I file GCF possono anche essere condivisi tra i giochi.

NOTA: GCF è il predecessore di [VPK file format](/game/vpk/).
