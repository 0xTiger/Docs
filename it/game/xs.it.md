## Cos'è un file XS?

Un file XS è un file di riferimento utilizzato in Age of Empires III (AOE3) per generare intelligenza artificiale (AI). Viene utilizzato durante il processo di compilazione quando viene caricato un file di mappa XML personalizzato. Se hai giocato ad AOE3, sai già che non solo puoi scegliere qualsiasi mappa con cui giocare, ma puoi anche creare le tue mappe utilizzando il programma Random Map Script.

## Come generare la mappa AI in Age of Empires 3?

 * Apri un programma di modifica del testo, come Microsoft Notepad, Notepad++ o TextEdit. Questo verrà utilizzato per programmare la tua mappa.
 * Programma tutte le funzionalità della tua mappa utilizzando il linguaggio Random Map Script.
 * Puoi fare riferimento ai tutorial di programmazione per assicurarti di programmarlo correttamente
 * Salva il file come MyAOE3.xs e spostalo nelle cartelle di gioco Age of Empires 3.

### Formato file XS: ulteriori informazioni

I file XS vengono salvati come file di testo semplice che possono essere aperti e modificati con qualsiasi editor di testo.
