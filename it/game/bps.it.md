## Cos'è un file BPS?

Un'estensione di file ".bps" si riferisce in genere a un file patch binario. Questi file vengono utilizzati per applicare patch o aggiornamenti a file o software esistenti. Ecco alcune ulteriori informazioni sui file BPS:

1. **File patch:** i file BPS contengono dati binari che rappresentano le modifiche necessarie per aggiornare o modificare un file o un programma esistente. Invece di sostituire l'intero file, i file patch includono solo le differenze tra il file originale e la versione aggiornata.

2. **Hacking ROM:** i file BPS sono comunemente utilizzati nel contesto dell'hacking ROM. Gli hacker ROM utilizzano file BPS per distribuire patch che modificano il codice del gioco o il contenuto delle ROM dei videogiochi classici. I giocatori possono applicare queste patch alle ROM dei loro giochi per sperimentare modifiche e traduzioni create dai fan.

3. **Applicazione della patch:** per applicare una patch BPS, avrai bisogno di uno strumento di patch. Utilità di patch popolari come "Floating IPS" o "Beat" ti consentono di applicare patch BPS al file ROM corrispondente, creando una versione modificata del gioco o del software.

Nelle sezioni seguenti esploreremo le applicazioni software associate ai file BPS.

## IPS fluttuante

Floating IPS (Flips) è un'utilità popolare e facile da usare per applicare patch ai file ROM nel contesto dell'hacking ROM. È comunemente usato nelle comunità di emulazione e di traduzione dei fan. Di seguito vengono fornite ulteriori informazioni ad esso relative.

- **Applicazione patch:** L'IPS mobile è progettato per applicare file patch, in genere in formato BPS, ai file ROM. Queste patch contengono dati binari che rappresentano modifiche alla ROM originale, come traduzioni, correzioni di bug o modifiche al gameplay.

- **Patch automatico:** Floating IPS rileva automaticamente l'intestazione della ROM di destinazione e applica la patch di conseguenza, semplificando il processo di patch.

- **Verifica del checksum:** lo strumento spesso include una funzionalità di verifica del checksum per garantire l'integrità della ROM patchata, riducendo il rischio di errori o corruzione.

- **Creazione di patch:** Sebbene il Floating IPS venga utilizzato principalmente per applicare le patch, può anche essere utilizzato per creare file di patch (formato BPS) dalle differenze tra due ROM, rendendolo utile per progetti di hacking e traduzione di ROM.

## MultiPatch

MultiPatch è un'altra utility comunemente utilizzata nella comunità di hacking ROM, specificatamente per applicare patch ai file ROM. È simile nella funzione a Floating IPS ma viene fornito con le proprie funzionalità e interfaccia. Di seguito vengono fornite ulteriori informazioni ad esso relative.

- **Applicazione patch:** MultiPatch è progettato per applicare patch ai file ROM, in genere in formati come IPS, UPS o BPS. Queste patch contengono dati binari che modificano la ROM originale, spesso per scopi come traduzioni, correzioni di bug o personalizzazioni.

- **Supporto per vari formati di patch:** Supporta una gamma di formati di file di patch, tra cui IPS (International Patching System), UPS (Universal Patch Format) e BPS (Binary Patch System), rendendolo versatile per diversi tipi di cerotti.

- **Patch Patching:** MultiPatch può essere utilizzato per applicare più patch a un singolo file ROM o a più file ROM contemporaneamente, il che è utile per hack ROM complessi o progetti che coinvolgono più patch.

- **Creazione di patch:** Simile a Floating IPS, MultiPatch può essere utilizzato anche per creare file di patch dalle differenze tra due ROM, utile per progetti di hacking e traduzione di ROM.
