## Cos'è un file CAT?

Il file CAT appartiene al software Advanced Disk Catalog, sviluppato da ElcomSoft Co. Ltd, società di software russa che sviluppa vari strumenti software forensi e di sicurezza informatica. Alcuni dei suoi prodotti includono software per il recupero della password, l'acquisizione di dispositivi mobili e l'estrazione dal cloud. L'azienda è stata fondata nel 1990 e ha uffici in Russia e negli Stati Uniti. Sono ben noti per il loro software forense e nel complesso il loro software viene utilizzato sia da aziende private che da agenzie governative per eseguire analisi e indagini forensi.

Advanced Disk Catalog is a cataloguing program for Windows XP/2003. La versatilità di ADC deriva dal fatto che non si basa su un motore di database, il che lo rende altamente efficiente e compatto. La sua interfaccia utente, simile a Windows Explorer, consente una facile organizzazione dei dati archiviati su vari tipi di supporti come dischi rigidi (incluse unità di rete), dischi floppy, dischi ottici, dischi ZIP e JAZ, ecc.

## Come aprire i file CAT?

Scarica il software Elcomsoft Advanced Disk Catalog. Su Windows, puoi fare clic con il pulsante destro del mouse su Esplora file e quindi selezionare Apri con menu contestuale, specificare il percorso del software Elcomsoft Advanced Disk Catalog per aprire il file cat.
