## Che cos'è un file DB?

Un file con estensione db è un file di database generico per memorizzare i dati. Non ci sono specifiche ufficiali da seguire per un tale file di database. I dati sono organizzati in un formato strutturato all'interno del file nelle tabelle dei moduli, nei campi, nei tipi di dati e nei valori dei campi. I file DB possono anche essere creati leggendo da file di foglio dati con dati in righe e colonne. Il contenuto dei file DB può essere ulteriormente esportato nei formati di file [CSV](/it/spreadsheet/csv/) e [PDF](/it/pdf/) a scopo di reporting. I file DB possono essere aperti in applicazioni come SQLite e LibreOffice.
