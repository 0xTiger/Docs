## Cos'è un file CHCK?

Un file CHCK viene creato come risultato dei controlli di monitoraggio dello stato su Microsoft Exchange Server. Contiene informazioni sullo stato generale e sull'operatività del database del server. Microsoft Exchange Server viene utilizzato principalmente come server di posta elettronica e consente di mantenere i contatti e i calendari aziendali. Gli amministratori del server Exchange utilizzano EseUtil per verificare l'integrità del database.

## Controlla il formato del file

Il formato file interno del file CHCK non è noto e non è disponibile materiale di riferimento per leggere/scrivere file CHCK.
