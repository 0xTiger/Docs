## Che cos'è un file ADF?

Un file con estensione .adf è un file di database creato con il software ACT CRM (Customer Relationship Management) per archiviare i dati sulle relazioni dei clienti. Memorizza informazioni sul nome del cliente, e-mail, contatto, informazioni commerciali e cronologia delle relazioni. I sistemi CRM sono più spesso integrati con i call center creati dalle aziende per recuperare automaticamente le informazioni sugli utenti per le chiamate in arrivo. I file ADF possono essere aperti con il software SwiftPage ACT su sistema operativo Windows.

## Formato file ADF - Ulteriori informazioni

I file ADF vengono salvati su disco come file binari e le informazioni sulla loro struttura interna non sono disponibili pubblicamente. Un altro file con estensione .alf viene salvato insieme ai file ADF che contengono i metadati necessari per navigare, eseguire il backup e ripristinare ACT! Banca dati.
