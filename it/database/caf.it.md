## Cos'è un file CAF?

Un file CAF è un file di catalogo creato con lo strumento software di organizzazione multimediale, **Cathy**. Contiene un elenco di cataloghi indicizzati che puntano a tutti i supporti disponibili su dispositivi esterni come dischi flash e CD. Questi file di catalogo ti aiutano a mantenere un elenco pulito di contenuti multimediali facilmente ricercabili e accessibili. I file CAF offrono la possibilità di cercare contenuti multimediali per nome, dimensione e data.

## Formato file CAF

I file CAF sono archiviati in formato file di testo. Ciò significa che puoi aprire file CAF in qualsiasi editor di testo come Blocco note, Notepad++ e Apple TextEdit. Cathy fornisce la funzionalità di esportare la struttura delle directory di ciascun catalogo in formato TXT, creando elenchi di cartelle e file.

### Apri il file CAF

Puoi aprire CAF con il software ufficiale Cathy. Ciò richiederà di posizionare il file CAF nella stessa directory in cui si trova il file Cathy.exe. Cathy mostrerà automaticamente il catalogo all'avvio.

Per fare ciò, posiziona il file CAF nella stessa directory del file Cathy.exe. Quindi apri Cathy. Il tuo catalogo CAF apparirà in Cathy.
