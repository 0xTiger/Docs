## Che cos'è un file AQ?

Un file con estensione .aq è un file di database creato con l'applicazione Ancestral Quest da Incline Software, LC. Memorizza le informazioni sull'albero genealogico utilizzando i dati genealogici inseriti dalle persone che utilizzano la versione gratuita e premium del software. Il database memorizza informazioni come i nomi, i luoghi di nascita e le relazioni dei membri della famiglia. Ancestral Quest utilizza queste informazioni per collegare e classificare informazioni come la visualizzazione Pedigree, Discendenti, Famiglia e Timeline. I file AQ possono essere aperti utilizzando [Incline Software Ancestral Quest](https://www.ancquest.com/index.htm).

## Formato file AQ - Ulteriori informazioni

I file AQ vengono salvati come file binari e il loro formato di file interno non è disponibile.
