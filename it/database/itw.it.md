## Che cos'è un file ITW?

Un file ITW è un formato di file di database utilizzato dall'applicazione CRM [InTouch](https://www.intouchsoftware.co.uk/InTouch). Memorizza i contenuti nel formato file proprietario di InTouch e i dettagli sul formato file interno non sono disponibili. I file ITW memorizzano informazioni personalizzate per la memorizzazione di nomi, indirizzi che vengono ulteriormente utilizzati per la stampa di buste ed etichette. Queste informazioni vengono utilizzate anche per inviare e-mail, messaggi di testo, newsletter elettroniche attraenti, creare modelli, allegati e collegamenti riutilizzabili. I dati da CRM possono anche essere esportati in formato ITW.
