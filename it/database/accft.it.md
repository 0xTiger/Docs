## Che cos'è un file ACCFT?

Un file con estensione .accft è un file modello di Microsoft Access che contiene la definizione dei tipi di dati di campo per le tabelle di un database. Microsoft Access è un RDBMS (Relational Database Management System) desktop utilizzato per creare database, moduli, query e viste basati su tabelle. Il file ACCFT può essere utilizzato per definire i tipi di dati per i campi della tabella e le tabelle del database possono essere create utilizzando questo file modello come punto di partenza. Può essere aperto con Microsoft Access 2016 per creare nuove tabelle. I file ACCFT sono riconosciuti dal tipo MIME **application/msaccess**.

## Formato file ACCFT

I file ACCFT sono file binari le cui specifiche del formato di file non sono disponibili pubblicamente. I suoi dettagli interni come le informazioni sull'intestazione, la struttura del file, i piè di pagina, ecc. non sono al momento disponibili.
