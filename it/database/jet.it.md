## Cos'è un file JET?

Un file di database JET, noto anche come file di database di Microsoft Access, è un formato di file utilizzato da Microsoft Access, un popolare sistema di gestione di database (DBMS). JET sta per Joint Engine Technology, che era il nome originale del motore di database utilizzato da Microsoft Access. Le versioni successive di Access hanno utilizzato motori di database diversi, ma il termine "database JET" è ancora comunemente utilizzato per fare riferimento ai database di Access.

Ecco alcuni punti chiave sui file di database JET:

- **Estensione file:** i file di database JET hanno in genere un'estensione file ".mdb" per le versioni precedenti di Access (Access 2003 e precedenti) e ".accdb" per le versioni più recenti (Access 2007 e successive). I file ".mdb" utilizzano il vecchio motore di database JET, mentre i file ".accdb" utilizzano la più recente tecnologia ACE (Access Database Engine).

- **Database relazionale:** i database JET sono database relazionali, il che significa che archiviano i dati in tabelle con relazioni definite tra loro. Gli utenti possono creare, modificare ed eseguire query sui dati utilizzando SQL (Structured Query Language) o tramite l'interfaccia grafica di Access.

- **Archiviazione dei dati:** i database JET possono archiviare un'ampia gamma di tipi di dati, inclusi testo, numeri, date, immagini e altro ancora. Possono anche gestire strutture dati complesse e relazioni tra tabelle.

- **Accesso multiutente:** i database JET supportano l'accesso multiutente, consentendo a più utenti di lavorare con il database contemporaneamente, sebbene possano esserci limitazioni in termini di scalabilità e prestazioni per applicazioni su larga scala.

- **Integrazione:** i database di Access possono essere integrati con altre applicazioni Microsoft Office come Excel e Outlook, rendendo conveniente per gli utenti lo scambio di dati tra diversi programmi.

## Informazioni sui database JET

**Storia del motore JET:**

Il motore di database JET (Joint Engine Technology) è stato originariamente sviluppato da Microsoft all'inizio degli anni '90 per essere utilizzato in Microsoft Access. Il motore è stato progettato per essere una soluzione di database leggera e orientata al desktop che potrebbe essere integrata in vari prodotti Microsoft.

**Componenti di un database JET**

- **Tabelle:** i database JET archiviano i dati in tabelle, ovvero raccolte di record organizzate in righe e colonne.
- **Query:** gli utenti possono creare query per estrarre, filtrare e manipolare i dati dalle tabelle utilizzando SQL (Structured Query Language).
- **Moduli:** Access fornisce un designer di moduli che consente agli utenti di creare moduli personalizzati per l'immissione e la visualizzazione di dati.
- **Rapporti:** gli utenti possono generare rapporti dall'aspetto professionale dai dati archiviati nel database.
- **Macro e VBA:** gli utenti possono automatizzare le attività e aggiungere funzionalità personalizzate utilizzando macro o scrivendo codice VBA.

**Tipi di dati**

I database JET supportano una varietà di tipi di dati, tra cui:

- **Testo:** Per memorizzare caratteri alfanumerici.
- **Numero:** per valori numerici.
- **Data/Ora:** per i valori di data e ora.
- **Promemoria:** Per testi o note più lunghi.
- **Collegamento ipertestuale:** per collegamenti Web e indirizzi e-mail.
- **Allegato:** per archiviare file e allegati.
