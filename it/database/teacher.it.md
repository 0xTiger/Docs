## Cos'è un file INSEGNANTE?

Un file .TEACHER è un file di impostazioni didattiche creato dal software SMART Response. Viene utilizzato in contesti didattici per fornire un ambiente completo di gestione delle informazioni sulla classe, elenchi di classi, orari, vacanze, profili degli studenti, routine degli esami e risultati delle valutazioni. Il software di valutazione SMART Response™ è installato sul computer che verrà utilizzato per fornire valutazioni SMART Response.

## Formato file INSEGNANTE

I file TEACHER vengono archiviati su disco in formato file binario. Le specifiche del formato di file interno non sono disponibili pubblicamente come riferimento per lo sviluppatore. Il software SMART Response può essere utilizzato per generare report sugli studenti. Questi possono poi essere condivisi tra i computer della scuola.
