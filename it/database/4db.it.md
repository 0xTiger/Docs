## Che cos'è un file 4DB?

La struttura del database costruita utilizzando 4th Dimension, un programma di progettazione software che comprende codice 4D, interfacce e informazioni sulle risorse è noto come 4DB. Viene utilizzato nel framework di progettazione del software integrativo di 4D solo durante il processo di sviluppo. I file di 4DB si trovano sia su piattaforme desktop che mobili e anche il sistema operativo di Windows è un'opzione. Il punteggio di popolarità dei principali file 4DB è "Basso", il che significa che i file 4DB sono solitamente presenti su un tipico dispositivo mobile o desktop.

Il file di struttura 4DB ha tutte quelle specifiche relative alla struttura del database come proprietà dei campi, campi e tabelle, elenchi di scelta, password, menu, metodi e moduli. Il file ottiene il nome che inserisci nella finestra di dialogo di creazione del database, seguito da ".4db".

Esistono due modi per connettere un documento con una nuova applicazione (ad esempio, my-file.4DB). Il primo e più semplice metodo è fare clic con il pulsante destro del mouse sul file 4DB in questione. Seleziona "Scegli il programma predefinito" dal menu autunnale, quindi fai clic su "Sfoglia" per individuare il software desiderato. Facendo clic su OK, è necessario confermare l'intera operazione. Il passaggio successivo e più complesso consiste nell'utilizzare il registro di Windows per collegare il tipo di dati 4DB al software appropriato.
