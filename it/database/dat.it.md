## Che cos'è un file DAT?
Un file con estensione .dat appartiene a un formato file generico. Di solito questi non sono né crittografati né binari, ma potrebbero essere quasi qualsiasi cosa. Varie applicazioni utilizzano i file DAT come riferimenti a determinate operazioni nel rispettivo programma. Quindi non possiamo dire che un software o un programma specifico sia responsabile della creazione di file DAT. Questi file sono particolarmente accessibili solo dall'applicazione che li ha creati.


## Formato file DAT
Il formato file DAT è semplicemente un formato generale che contiene qualsiasi tipo di dati in un file, solitamente salvato con estensione .dat. Può contenere dati in formato binario o di testo, di solito un dato generico è costituito dalle informazioni specifiche dell'applicazione a cui si riferisce. Molto spesso, potresti trovarli con altri file di configurazione come i file DLL. Qui possiamo arruolare alcuni tipi di file famosi, con estensione file .dat:

### File video VCD
I VCD erano ben noti video o sorgenti multimediali negli anni '90 per la distribuzione di film e home video prima dei DVD. Di solito i file video dal disco VCD sono stati trovati con estensione .dat. Quindi ha memorizzato i dati video effettivi riprodotti dal VCD.
### Scambia allegato e-mail
I file DAT relativi all'allegato di posta elettronica di Exchange contengono informazioni sulla formattazione per i messaggi RTF (RTF) inviati da Microsoft Exchange Server. Di solito questi file vengono aggiunti alla fine delle e-mail quando il client di posta elettronica di un destinatario non supporta i messaggi RTF. Questi file si trovano solitamente come **Winmail.dat**.
### File hive del registro di Windows
NTUSER.DAT è un file hive del registro comune. Il registro di Microsoft Windows utilizza i file DAT per memorizzare le impostazioni dell'utente. Questi file DAT vengono caricati quando un utente accede a Windows. Può includere impostazioni dell'account, password e associazioni di file.
### File di dati di Minecraft
Il gioco Minecraft utilizza i file DAT per salvare le informazioni sull'inventario del giocatore, la posizione del giocatore e le informazioni sul mondo. I file DAT vengono utilizzati anche per memorizzare i livelli, che sono piccoli pezzi di mappa che vengono caricati su richiesta mentre un giocatore naviga nel mondo.
### File DAT piriforme
Le applicazioni come CCleaner, Speccy, Defraggler e Recuva utilizzano file DAT per le informazioni sulle loro operazioni tipiche.
