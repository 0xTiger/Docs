## Che cos'è un file FMP12?

Un file FMP12 è un file di database creato con il programma di sviluppo di database FileMaker Pro 12. Contiene la struttura generale del database (noto anche come schema del database) insieme ai dati al suo interno. La struttura di un file FMP12 si basa su tabelle in cui ogni tabella è composta da un numero di campi di database con diversi tipi di dati. Alcuni tipi di dati supportati dal formato file FMP12 includono testo, moduli personalizzati, report, datetime, BLOB e altri oggetti di database. I formati di file FileMaker Pro precedenti come [FP7](/it/database/fp7/) possono essere aggiornati a FMP12.

## Come convertire file FileMaker Pro meno recenti in FMP12?

Formati di file FileMaker Pro precedenti come database 11, 10, 9, 8 e 7 (.fp7) in formati di file FMP12. FileMaker ha linee guida complete per [convertire la versione precedente dei formati di file FileMaker Pro in FMP12](https://fmhelp.filemaker.com/help/16/fmp/en/index.html#page/FMP_Help/converting-files.html) .
