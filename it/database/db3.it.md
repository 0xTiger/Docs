## Che cos'è un file DB3?
Il file DB3 è un file di database creato dal software SQLite che è un programma di database leggero e autonomo che crea database utilizzando file semplici; contiene l'anatomia del database e i record di dati; comunemente usato per recuperare o archiviare dati strutturati tramite SQL. Questi file possono essere utilizzati in dispositivi intelligenti o dove è richiesta la conservazione dei registri o altra gestione dei dati ma in un ambiente con poco spazio.


## Formato file DB3
Il formato file DB3 è associato a SQLite RDBMS (Relational Database Management System), una scelta popolare per i database incorporati. Non esiste alcuna estensione di file specifica definita in SQLite_3 nelle sue specifiche, può utilizzare estensioni tra cui [.dbf](/it/database/dbf/) e [.sqlite](/it/database/sqlite/).

### La Secrezione del Database
Comunemente SQLite, il formato di file db relativo alla versione 3 può essere considerato come il formato di file DB3 utilizzato come formato nativo documentato pubblicamente per il motore di database SQLite dal giugno 2004. Il formato di file DB3 è multipiattaforma, trasferibile tra big-endian e architetture little-endian o sistemi a 32 e 64 bit. Queste caratteristiche rendono DB3 una scelta popolare come formato di file dell'applicazione. Il file principale del database SQLite_3 (DB3) è costituito da una o più pagine. Tutte le dimensioni di tutte le pagine sono uguali nello stesso database. La dimensione della pagina in byte è una potenza di due tra 512 e 65536 inclusi.
