## Che cos'è un file ASK?

Un file con estensione .ask è un file di database creato da askSam Systems, un'applicazione PIM (Personal Information Manager). Questi tipi di file sono archiviati in cartelle dinamiche e contengono dati utente come messaggi di posta elettronica, documenti, file di testo, fogli di calcolo, indirizzi e pagine Web. Ciò consente al software askSam di ricercarli facilmente e recuperare i dati a scopo di ricerca. Il software askSam, sebbene una semplice applicazione desktop basata su moduli che memorizza i dati nel database, era popolare per l'organizzazione e il recupero di tali dati. I dati dal file ASK possono anche essere esportati in [fogli di calcolo](/it/spreadsheet/).

## ASK File Format - Ulteriori informazioni

I file ASK vengono salvati su disco in un formato di file binario proprietario i cui dettagli sul formato del file interno non sono disponibili come riferimento per gli sviluppatori.
