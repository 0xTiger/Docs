Un database è una raccolta di dati che è organizzata in tabelle e conservata su un dispositivo di archiviazione del computer e vi si accede elettronicamente. I database relazionali archiviano i dati in tabelle di database che sono collegate tra loro tramite tecniche di progettazione e modellazione formali. I dati in un database vengono analizzati utilizzando il Database Management System (DBMS), che è la principale fonte di interazione con gli utenti finali, le applicazioni e il database stesso.

Le estensioni di file di database più diffuse e i relativi formati di file includono **SQLite**, **DB**, **ACCDB** e **MDB**.
