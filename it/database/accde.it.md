## Che cos'è un file ACCDE?

Un file con estensione .accede è un formato di file di Microsoft Access 2007 per la creazione di file di database Esegui solo per proteggere il codice del database personalizzato. Compila tutto il codice VBA in eseguibile solo in modo che non possa essere visualizzato o modificato. Il formato di file ACCDE sostituisce il formato di file MDE utilizzato dalle versioni precedenti di Microsoft Access. ACCDE sono file [ACCDB](/it/database/accdb/) in modalità solo eseguibile e più compatti per l'assenza di qualsiasi codice sorgente. Le dimensioni compatte di questi file migliorano l'utilizzo della memoria e migliorano le prestazioni dell'applicazione. I file ACCDE possono essere aperti con Microsoft Access 365.

## Formato file ACCDE

Tutti i file di Microsoft Access 2007+ utilizzano Jet ACE (motore di database di Access) che utilizza la famiglia di formati di file ACCDB. Tuttavia, Microsoft non fornisce un'implementazione open source o una specifica tecnica dettagliata.
