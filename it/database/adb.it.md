## Che cos'è un file ADB?

Un file con estensione .adb è un file di database creato con [Alpha Five](https://www.alphasoftware.com/) (ora noto come Alpha Anywhere). Contiene i nomi di file di tutte le tabelle e gli insiemi che sono membri dell'area di lavoro del progetto. Contiene inoltre i nomi di tutte le librerie incorporate nell'area di lavoro. Alpha Anywhere è un IDE utilizzato per sviluppare applicazioni web. Oltre ai file ADB, un'area di lavoro di Alpha Anywhere può contenere molti file che iniziano il nome di una tabella o di un set, ma hanno estensioni di file diverse.

## Formato file ADB - Ulteriori informazioni

Le informazioni sul formato del file interno dei file .adb non sono disponibili come riferimento per gli sviluppatori.
