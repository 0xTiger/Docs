## Cos'è un file ABX?

Un file .abx è una rubrica esportata dal software WordPerfect Office sviluppato da **Corel**. WordPerfect era un software di elaborazione testi sviluppato da Corel Corporation ed è in uso da parecchio tempo. I file ABX contenevano informazioni sui contatti estratti dal client di posta elettronica WordPerfect. Le informazioni sui contatti esportati possono essere trasferite su altri dispositivi e utilizzate di conseguenza.

## Formato file ABX: ulteriori informazioni

I file ABX sono archiviati nel proprio formato file proprietario e i dettagli del formato file interno dei file ABX non sono noti. La posta WordPerfect è stata successivamente integrata nella WordPerfect Office Suite.
