## Che cos'è un file ADP?

Un file con estensione .adp è un file di progetto di Microsoft Access che utilizza l'architettura del componente OLE DB per fornire una connessione diretta ed efficiente a un database di Microsoft SQL Server. Tali file non contengono le tabelle effettive, i diagrammi del database o altri elementi del database. I file ADP possono essere creati con Access 2007 e 2010. I file creati con le versioni precedenti di Access possono essere aperti anche con Access 2007 e 2010. I file ADP possono essere aperti utilizzando Microsoft Access 365.

## Formato file ADP - Ulteriori informazioni

Un progetto Access creato come ADP consente di apportare modifiche di progettazione agli oggetti del server SQL come tabelle e viste. Consente inoltre di creare, modificare e utilizzare funzionalità SQL come diagrammi di database, procedure memorizzate e funzioni definite dall'utente. Ciò offre un vantaggio rispetto al collegamento a un database del server SQL in cui non è possibile apportare modifiche alla progettazione e si possono collegare solo tabelle e viste del server SQL.

### Tipi di dati e supporto per diagrammi

**Tipi di dati data/ora**

I seguenti nuovi tipi di dati di data/ora sono supportati in Access 2010.

* VOLTA
* DATA
* DATA ORA2
* DATETIMEOFFSET

**Tipi di dati a lunghezza variabile**

I seguenti tipi di dati a lunghezza variabile possono essere utilizzati nei progetti di Access 2010:

* VARBIN(MAX)
* VARCHAR(MAX)
* NVARCHAR(MAX)
