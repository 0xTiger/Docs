## Che cos'è un file ALF?

Un file con estensione .alf è un registro delle transazioni SQL generato da [ACT! Software CRM](https://www.act.com/)(Customer Relationship Management). Tiene traccia delle attività tra l'ACT! e SQL e viene utilizzato per ripristinare ACT! Banca dati. ATTO! CRM memorizza le informazioni del cliente come nome, indirizzo e-mail, contatto, informazioni commerciali e cronologia delle relazioni. Memorizza queste informazioni in [ACT! ADF](/it/database/adf/) che memorizza i dati in tabelle per il collegamento con altri moduli, ad esempio il recupero automatico delle informazioni per una chiamata in arrivo.
I file ALF possono essere aperti con il software SwiftPage ACT su Windows OS.

## Formato file ALF - Ulteriori informazioni

I file ALF vengono salvati come file binari e il loro formato di file interno non è disponibile.
