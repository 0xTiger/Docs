## Cos'è un file EPIM?

L'estensione del file .epim viene utilizzata da EssentialPIM, un software di gestione delle informazioni personali che consente agli utenti di organizzare e gestire contatti, appuntamenti, attività e note. Il file .epim è un file di database che memorizza tutti i dati immessi nel programma, incluse informazioni personali, eventi del calendario ed elenchi di attività. È possibile eseguire il backup di questo file, esportarlo e importarlo in un'altra installazione di EssentialPIM. Il software è disponibile per piattaforme Windows, Android e iOS.

I file .epim sono il formato proprietario utilizzato da EssentialPIM. È importante notare che i file .epim sono specifici del software EssentialPIM e non possono essere aperti o modificati con altri software.

## Come creare un file EPIM?

Per creare un file .epim in EssentialPIM, dovrai prima avere il software installato sul tuo computer.

1. Apri EssentialPIM.
2. Fare clic sul menu File e selezionare Nuovo database.
3. Apparirà una finestra di dialogo che ti chiederà di selezionare la posizione e il nome del nuovo file di database.
4. Inserisci un nome e scegli una posizione per il file, quindi fai clic su Salva.
5. Il nuovo file di database verrà creato con l'estensione del file .epim e potrai iniziare ad aggiungere dati come contatti, appuntamenti, attività e note.

In alternativa è possibile anche utilizzare File -> Importa per importare dati da vari formati e creare un nuovo file .epim. Inoltre, se hai già un file di database e vuoi crearne uno nuovo, puoi utilizzare l'opzione File -> Esporta per esportare tutti i dati dal file .epim corrente e importarli nel nuovo uno.

## Posizione dei file EPIM

La posizione dei file .epim in EssentialPIM dipende dal sistema operativo in uso.

- Su Windows, il percorso predefinito per i file .epim è nella cartella Documenti, ma puoi scegliere un percorso diverso quando crei un nuovo file di database.
- Su Android, il percorso predefinito per i file .epim è nella cartella Download, ma puoi scegliere un percorso diverso quando crei un nuovo file di database.
- Su iOS, la posizione predefinita per i file .epim è nella cartella EssentialPIM, ma puoi scegliere una posizione diversa quando crei un nuovo file di database.

Puoi anche utilizzare l'opzione File -> Apri database per aprire un file .epim esistente da una posizione diversa sul tuo computer o dispositivo.
