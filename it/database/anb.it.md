## Che cos'è un file ANB?

Un file con estensione .anb è un grafico IBM i2 Analyst's Notebook che contiene informazioni sulla relazione tra dati diversi. Come suggerisce il nome, il software Analyst's Notebook aiuta a collegare i dati, risultando in una forma più concisa di connettività tra informazioni diverse e fornendo output di intelligence per l'analisi. Questo aiuta a identificare e interrompere attività dannose come minacce criminali, informatiche e fraudolente. I file ANB possono essere aperti con il software Notebook di IBM i2Analyst.

## Formato file ANB - Ulteriori informazioni

Un file ANB comprende i seguenti elementi costitutivi:

* Entità - Rappresentazione di informazioni
* Collegamenti - Mostra la connettività tra Entità
* Proprietà - Descrizione di un'entità o collegamento in modo più dettagliato

I file ANB memorizzano tutte queste informazioni in base al feed delle informazioni di input nel taccuino dell'analista ANB. Sulla base dell'input dei dati, il software crea collegamenti tra ogni informazione che aiuta a mappare le relazioni e tracciare le prove in modo geospaziale e temporale.
