## Cos'è un file ABS?

Un file con estensione .abs è un database generato e utilizzato dal database Absolute che è un motore di database per l'IDE di sviluppo software Delphi. Ha sostituito il Borland Database Engine (DBE) ed è più compatto, veloce, robusto e facile da usare. Il database ABS viene compilato come parte del file EXE dell'applicazione, rendendo la tua applicazione più veloce e più piccola. I dati vengono archiviati in un file ABS in un formato strutturato e relazionale.

## Formato file ABS: ulteriori informazioni

I file ABS vengono archiviati come parte del [file EXE](/it/executable/exe/) dell'applicazione, che solitamente viene archiviato come file binario. I file binari non possono essere aperti in nessun editor di testo e forniscono un accesso rapido al contenuto del file di database.

### Caratteristiche principali del formato file ABS

I file ABS sono piuttosto semplici e incorporati nell'eseguibile dell'applicazione. Ha le seguenti caratteristiche:

* Nessun BDE; nessuna DLL
* Database a file singolo
* Supporto SQL'92 (DDL e DML).
* Compatibile con controlli di database standard e di terze parti
* Modalità utente singolo e multiutente (file server)
* Funziona perfettamente su tutte le versioni di Windows, dalla 98 alla più recente, non richiede aggiornamenti o service pack
* Tabelle in memoria ultraveloci
* Facilità d'uso senza pari
* Crittografia avanzata
* Compressione BLOB
* Gratuito per uso personale
* Disponibile il codice sorgente completo
*Distribuzione esente da royalty
