## Cos'è un file BAK?

Un file .BAK funge anche da file di backup del database generato da Swiftpage Act!, che è un software di gestione delle relazioni con i clienti (CRM). Questo file include un duplicato di un atto! database (solitamente un file .ADF), preservando in modo efficace i dati essenziali come contatti dei clienti, dettagli dell'azienda, informazioni sul gruppo, note, record storici e dati sulle attività archiviati nel database.

## Atto Swiftpage!

Atto Swiftpage! è un'applicazione software utilizzata per la gestione delle relazioni con i clienti (CRM). È progettato per aiutare le aziende e le organizzazioni a gestire le interazioni con i clienti, tenere traccia delle attività di vendita e marketing e archiviare le informazioni di contatto essenziali. Atto Swiftpage! offre funzionalità come gestione dei contatti, email marketing, gestione del calendario e delle attività, automazione delle vendite e reporting per semplificare e migliorare i processi di gestione delle relazioni con i clienti.

All'interno di Act!, tutti i dati relativi ai clienti vengono archiviati in un file di database dedicato, di cui è possibile eseguire il backup a discrezione di Act! manager e amministratori. Questo backup è particolarmente importante quando, ad esempio, un'azienda sta pianificando l'aggiornamento a una nuova versione di Act!. In questi casi, l'amministratore IT responsabile di Act! potrebbe scegliere di creare un backup della legge! Banca dati.

Questi agiscono! i backup del database vengono salvati sotto forma di file BAK. In genere, durante il processo di backup, questi file BAK vengono raggruppati in un file [.ZIP](/it/compression/zip/). Questo archivio ZIP non contiene solo il nucleo di Act! backup del database ma include anche Act associato! elementi di dati, come documenti allegati, modelli di documenti, modelli di report e query di database salvate. Questo approccio di backup completo garantisce che le informazioni critiche sui clienti e le risorse correlate siano protette e facilmente recuperabili quando necessario.

## Come aprire un file BAK?

Per aprire un file BAK in Swiftpage Act! su un sistema Windows, puoi utilizzare Act! Diagnostica. Segui questi passi:

1. **Atto aperto! Diagnostica**:
- Fare clic sul menu Start di Windows.
- Nella barra di ricerca, digita "actdiag".
- Premi Invio per aprire Act! Diagnostica.

2. **Accedi allo strumento Ripristina database**:
- Entro l'Atto! Diagnostica, vai al menu "Strumenti".

3. **Scegli Ripristina database**:
- Nel menu "Strumenti", seleziona "Ripristina database".

4. **Seleziona Ripristina file BAK**:
- Ti verranno presentate due opzioni:
- Per sovrascrivere un database esistente con i dati del tuo file BAK, seleziona "Ripristina file BAK".
- Per aggiungere il tuo file BAK come nuovo database insieme ai dati esistenti, seleziona "Ripristina file BAK come".

5. **Segui le istruzioni**:
- Atto! La diagnostica ti guiderà attraverso il processo di ripristino del database contenuto nel tuo file BAK.
- Potrebbe essere necessario specificare la posizione del file BAK e fornire eventuali impostazioni aggiuntive richieste per il ripristino.

Segui questi passaggi e agisci! La diagnostica ripristinerà il database dal tuo file BAK, consentendoti di accedere ai dati archiviati al suo interno in Swiftpage Act!
