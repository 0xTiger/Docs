## Che cos'è un file LOG?
Un file con estensione .log contiene l'elenco di testo normale con timestamp. Di solito, alcuni dettagli dell'attività vengono registrati dai software o dai sistemi operativi per aiutare gli sviluppatori o gli utenti a tenere traccia di ciò che stava accadendo in un determinato periodo di tempo. Gli utenti possono modificare questi file molto facilmente utilizzando qualsiasi editor di testo. Di solito i rapporti di errore o le attività di accesso vengono registrati dai sistemi operativi, ma anche altri software o server Web generano file di registro per tracciare i visitatori e monitorare l'utilizzo della larghezza di banda.

## Formato file LOG
Il formato del file LOG registra gli eventi tipici che si verificano in un sistema operativo, i messaggi tra diversi utenti di un software di comunicazione o qualsiasi altro software eseguito. Possiamo semplicemente dire che determinati messaggi in determinati timestamp vengono registrati in un file LOG. Alcuni termini comunemente usati per la registrazione sono:
### Registri eventi
Registra gli eventi che si verificano durante l'esecuzione di un sistema al fine di tracciare e comprendere l'attività del sistema e diagnosticare i problemi. Sono importanti per identificare le attività di sistemi complessi, tipicamente nel caso di applicazioni con pochissima interazione da parte dell'utente.
### Registri delle transazioni
Quasi tutti i sistemi di database mantengono il registro delle transazioni, che di solito non è inteso come traccia di controllo per analisi successive e non sono leggibili dall'uomo. Questi registri conservano solo il record delle modifiche ai dati esistenti per consentire al database di eseguire il ripristino da arresti anomali o altri errori di dati e mantenere i dati in uno stato coerente. Pertanto, i sistemi di database di solito dispongono sia di registri eventi generali che di registri transazionali.
### Registri dei messaggi
Le comunicazioni testuali salvate da Internet Relay Chat (IRC), programmi di messaggistica istantanea (IM), client di condivisione file peer-to-peer con funzioni di chat e giochi multiplayer (in particolare MMORPG) sono chiamate registri dei messaggi. Questi registri vengono generalmente salvati in file di testo normale, ma i client IM e VoIP (ad es. Skype) potrebbero salvarli in file HTML per facilitare la lettura o abilitare la crittografia.
### Registri del server
Il registro del server è in realtà un file di registro contenente un elenco di attività eseguite e create o mantenute automaticamente dal server stesso. Di solito questi file non sono accessibili agli utenti Internet generici, ma solo al webmaster o ad altra persona amministrativa di un servizio Internet.
