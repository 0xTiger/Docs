## Che cos'è un file FP5?

Un file FP5 è un file di database creato con FileMaker Pro versione 5 o 6. Memorizza i dati dell'utente e dell'organizzazione sotto forma di record all'interno di tabelle. FileMaker Pro è un sistema di gestione di database che supporta i sistemi operativi Macintosh e Windows. FP5 è un formato di file precedente supportato dalla versione di FileMaker Pro allora disponibile. L'ultimo formato di database di FileMaker Pro è [FMP12](/it/database/fmp12/), introdotto con il lancio di FileMaker Pro v 12. Un altro formato di file di database ben utilizzato è stato [FP7](/it/database/fp7/) che è stato sostituito da FMP12.

## Formato file FP5 - Ulteriori informazioni

I dettagli del formato file interno del formato file FP5 non sono disponibili pubblicamente per riferimento dello sviluppatore. I formati di file di database FileMaker Pro precedenti come [FP5 e FP7 possono essere aggiornati all'ultimo FMP12](https://fmhelp.filemaker.com/help/16/fmp/en/index.html#page/FMP_Help/converting-files. html) utilizzando le linee guida dettagliate di FileMaker Pro.
