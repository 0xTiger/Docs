## Cos'è un file BC3?

Un file BC3 è un file di database per la memorizzazione di informazioni relative al formato di scambio del database di costruzione. È stato sviluppato sulla base del formato di scambio del database edilizio standard spagnolo. Un file BC3 memorizza informazioni contenenti record di dati inclusi dettagli sui materiali da costruzione, costo dei materiali e prezzo della manodopera. I dati possono essere scambiati tra altre applicazioni senza convertire i dati in altri formati.

## Formato file BC3

I file BC3 vengono archiviati come file binari con i dettagli specificati secondo gli standard FIEBDC. L'implementazione dei file BC3 ha subito diverse revisioni ed esistono le seguenti versioni del formato file BC3:

*FIEBDC-3/2004
*FIEBDC-3/2007
*FIEBDC-3/2012
* FIEBDC-3/2016

FIEBDC è l'abbreviazione di Formato de Intercambio Estándar - Bases de Datos de Construcción in lingua spagnola.
