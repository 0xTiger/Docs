## Cos'è un file MABK?
Un file MABK è un file di backup Android di Motorolla creato da Lenovo Android Management
utilità, Lenovo Moto Smart Assistant. È un file di backup contenente una copia delle applicazioni,
contatti, messaggi di testo e file multimediali memorizzati sul telefono. Il file MABK non viene salvato
impostazioni dell'applicazione dell'utente ifnormation.

## Formato file MABK

I file MABK vengono generati utilizzando l'applicazione Lenovo Moto Smart Assistant (LMSA).
Telefono MOtorola, telefono Lenovo e tablet Lenovo. È un'applicazione Windows e memorizza i file
eseguire il backup dei dati sui dispositivi dell'utente. L'applicazione, una volta eseguita, si connette al dispositivo dell'utente e
takes crea il file MABK quando è selezionata l'opzione **Backup e ripristino**.
Gli utenti finali possono salvare i file MABK sul proprio PC per utilizzarli in seguito se necessitano di ripristinare il file
contenuto dal file di backup. Gli utenti, di solito, eseguono il backup di tutti i file disponibili sul dispositivo come
contatti, messaggi di testo, immagini, audio, video e app.

## Come aprire i file MABK?

Puoi utilizzare un file MABK per il ripristino o il trasferimento di dati sul tuo telefono Motorola,
Telefono Lenovo o tablet Lenovo che utilizza Lenovo Moto Smart Assistant per Windows. A
realizzare questo:
1. Collega il tuo dispositivo mobile al PC.
2. Avvia Lenovo Moto Smart Assistant.
3. Passare a Backup e ripristino, quindi scegliere Ripristina.
4. Importa il tuo file MABK.
5. Selezionare Ripristina e seguire le istruzioni visualizzate per completare il ripristino dei dati oppure
processo di trasferimento.
Inoltre, puoi accedere manualmente ai file archiviati in un file MABK. Ad esempio, puoi
estrai le immagini contenute nel file MABK e visualizzale sul tuo PC. Da cui estrarre i file
un file MABK:
1. Apri Lenovo Moto Smart Assistant.
2. Vai su Backup e ripristino.
3. Selezionare Ripristina.
4. Importa il tuo file MABK.
5. Scegli Estrai e segui le istruzioni visualizzate per estrarre i file dal tuo file MABK.
