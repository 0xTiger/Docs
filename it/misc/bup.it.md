## Che cos'è un file BUP?

Un BUP è il backup di un file IFO. Memorizza informazioni simili al file IFO in modo che i lettori DVD possano utilizzare questo file se il file IFO originale non può essere letto. I file BUP comunicano al lettore DVD il menu di avvio, la posizione delle tracce video e audio e l'inizio dei capitoli su DVD. I file BUP possono essere aperti con VLC Player.

## Formato file BUP - Ulteriori informazioni

I DVD possono danneggiarsi a causa dei possibili graffi sulle sue superfici. Ciò rende difficile la loro lettura da parte di un'unità DVD. Se l'[IFO](/it/video/ifo/) non può essere letto a causa di graffi, i lettori DVD non possono riprodurre un tale disco. In questi casi, i lettori multimediali ricorrono al file BUP che è un vero backup del file IFO.
