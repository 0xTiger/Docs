## Che cos'è un file CTG?

Un file CTG è un file di catalogo creato sulle fotocamere digitali Canon. Questi vengono creati quando le foto vengono scattate da una fotocamera digitale Canon. Memorizza le informazioni sul numero di fotografie/immagini in ciascuna cartella sul dispositivo di archiviazione della fotocamera, ad esempio la scheda di memoria. I file CTG sono archiviati nella cartella CANONMSC sulla scheda di memoria. Questi file consentono al tuo computer di indicizzare i file dopo averli inseriti nel computer.

## Formato file CTG - Ulteriori informazioni

I file CTG vengono salvati nel formato di file proprietario Canon. Questi sono generati automaticamente e non devono essere modificati manualmente. La fotocamera elimina questi file quando le fotografie corrispondenti vengono cancellate dall'utente.
