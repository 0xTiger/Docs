## Cos'è un file XMZX?

Un file XMZX è un file di test d'esame per condurre test e viene utilizzato dall'applicazione software ExamSoft SofTest. L'applicazione può essere utilizzata sia su Windows che su macOS. Contiene le domande di un esame che l'esaminatore può sostenere dagli studenti su un computer con lo software SofTest. I file XMZX vengono inviati agli esaminatori per condurre il test. Viene caricato nel software SofTest e il candidato o il candidato sostiene il test. Al termine del test, il file XMDX finalizzato viene automaticamente caricato sui server cloud di ExamSoft.

## Formato file XMZX: ulteriori informazioni

I file XMZX vengono utilizzati dalla piattaforma di valutazione digitale ExamSoft, utilizzata da diverse organizzazioni per condurre test per le certificazioni. La piattaforma dispone di propri server cloud online dove tutti i test completati vengono caricati automaticamente. I test vengono condotti presso un centro d'esame oppure gli esaminatori possono anche sostenere il test da parte dei candidati sui propri computer. ExamSoft è disponibile sia sui sistemi operativi Windows che macOS. XSMX è un altro formato di file di test d'esame di ExamSoft.

## Come aprire un file XMZX?

È possibile aprire un file XMZX nel software ExamSoft SofTest solo poiché è un formato di file proprietario. Il software SofTest ha la possibilità di **Sostenere un esame** che viene utilizzata per avviare l'esame selezionando il file XMZX.
