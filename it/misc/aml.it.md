## Che cos'è un file AML?

Un file AML (Automation Markup Language) è un file di dati che viene utilizzato per memorizzare e scambiare informazioni di impianto. Si basa sul formato di file XML, rendendo lo scambio di informazioni basato su standard aperti. AutomationML offre la possibilità di interconnettere moderni strumenti di ingegneria nei campi dell'impiantistica meccanica, della progettazione elettrica, dei controllori logici programmabili e del controllo dei robot. I file AML possono essere visualizzati e modificati utilizzando lo strumento AutomationML Editor.

## Formato file AML

AML utilizza la modellazione dei dati orientata agli oggetti basata su [XML](/it/web/xml/) per l'archiviazione e lo scambio di modelli ingegneristici. I componenti dell'impianto di ingegneria sono rappresentati come oggetti con aspetti diversi, con ogni oggetto che ha la capacità di agire come sotto-oggetti. Pertanto, ogni modello più piccolo o più grande può essere descritto. Le [Specifiche di AutomationML](https://www.automationml.org/about-automationml/specifications/) forniscono una breve guida per iniziare con i dettagli di questo linguaggio di modellazione.

## Guida introduttiva all'AML

L'uso dell'AML per lo scambio di informazioni tecniche richiede determinate conoscenze per un'applicazione di successo. L'implementazione di nuove applicazioni richiede diversi passaggi a tale scopo, come illustrato di seguito.

* Conosci l'architettura
* Acquisire conoscenze di base
* Esame del caso d'uso
* Mappatura e modellazione specifiche dell'applicazione
* Implementazione
