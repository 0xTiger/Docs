## Che cos'è un file VIMRC?

Un file VIMRC è un file di impostazioni utilizzato dall'editor di testo Vim. Memorizza le impostazioni per l'editor che vengono caricate all'apertura dell'editor. Gli sviluppatori di applicazioni e gli amministratori di sistema definiscono le impostazioni relative all'aspetto del testo, come l'evidenziazione della sintassi, che vengono salvate nel file VIMRC. Pertanto, il file VIMRC consente agli utenti di personalizzare le impostazioni per l'editor.

## Formato file VIMRC

I file VIMRC vengono salvati come file di testo normale e possono essere aperti/modificati in qualsiasi editor di testo come Notepad, Notepad++ e TextEdit. Contengono istruzioni che possono essere eseguite nell'editor Vim iniziando con un ":", seguito dal comando. I file VIMRC contengono istruzioni per impostazioni come il rientro del testo e i colori delle finestre e dei caratteri. Un formato di file rilevante è [VIM](/it/misc/vim/).
