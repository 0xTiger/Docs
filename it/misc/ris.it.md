## Che cos'è un file RIS?

Un file RIS è un formato di file di tag per lo scambio di dati bibliografici mediante programmi di citazione. Contiene più righe di dati di citazione, costituite da un codice di due caratteri e un valore corrispondente delimitato da un trattino. Le informazioni sulle citazioni incluse in un file RIS includono dati come autore, titolo, data di pubblicazione, parole chiave, editore, abstract, indirizzo dell'autore, ecc. I file RIS sono leggibili e possono essere aperti con qualsiasi editor di testo come Microsoft Notepad, Notepad++, e Apple TextEdit.

## Formato file RIS - Ulteriori informazioni

I file RIS vengono archiviati come file di testo normale. Ogni riga in un file RIS è costituita da un codice di due lettere (noto anche come tag), seguito da due spazi e un trattino, e il valore corrispondente. A causa dei tag, i file RIS sono noti come formato con tag per la rappresentazione di citazioni bibliografiche. Il record di esempio in un file RIS è mostrato di seguito.

## Esempio di file RIS

L'esempio seguente dall'articolo di Wikipedia mostra il contenuto del file RIS per un articolo: Claude E. Shannon. Una teoria matematica della comunicazione, Bell System Technical Journal, 27:379–423, luglio 1948.


|Esempio RIS|
---|
|TY - JOUR|
|AU - Shannon, Claude E.|
|PY - 1948|
|DA - luglio|
|TI - Una teoria matematica della comunicazione|
|T2 - Rivista tecnica Bell System|
|SP - 379|
|PE - 423|
|VL - 27|
|ER - |

In questo esempio, "TY" è il tipo di riferimento e "ER" è la fine del riferimento.
