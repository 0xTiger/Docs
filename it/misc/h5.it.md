## Che cos'è un file H5?

Un H5 è uno dei formati di dati gerarchici (HDF) utilizzati per archiviare grandi quantità di dati. Viene utilizzato per archiviare grandi quantità di dati sotto forma di array multidimensionali. Il formato viene utilizzato principalmente per archiviare dati scientifici ben organizzati per un rapido recupero e analisi. H5 è stato introdotto come formato di file più avanzato rispetto a H4. È stato originariamente sviluppato dal National Center for Supercomputing Applications ed è ora supportato da The HDF Group.

## Breve storia del formato di file HDF

Il formato di file HDF è stato selezionato dalla NASA come metodo standard per la memorizzazione dei dati scientifici. Prima di questo, la formulazione di tale standard iniziata nel 1987 dalla Graphics Foundations Task Force (GFTF) e HDF è stata formalmente approvata dopo l'analisi di 15 diversi formati di file.

## Formato file HDF5

I file H5 sono in formato dati gerarchico che aderisce alle specifiche del formato file HDF5. Queste specifiche stabiliscono la struttura generale del file HDF5 per l'archiviazione su disco, ma gli utenti finali non hanno bisogno di questi dettagli sottostanti.

I file H5 sono comunemente usati nei settori aerospaziale, fisica, ingegneria, finanza, ricerca accademica, genomica, astronomia, strumenti elettronici e medicina.

### Modello dati HDF5

Il file HDF5 è come un contenitore che comprende oggetti dati eterogenei. Ad esempio, questi set di dati possono essere immagini, tabelle, elementi di testo, grafici e persino documenti come [PDF](/it/pdf/) e [Documenti Excel](/it/foglio di calcolo/).
