## Cos'è un file BPS?

Un file .BPS è essenzialmente un backup di un documento di elaborazione testi creato con Microsoft Works, che in genere utilizza l'estensione di file .WPS per i suoi documenti. Questi file .BPS vengono spesso generati automaticamente da Microsoft Works come precauzione nel caso in cui il programma si blocchi mentre stai lavorando sul documento. È un modo per salvaguardare il tuo lavoro e prevenire la perdita di dati.

Nella sezione seguente, esploreremo le applicazioni software associate ai file BPS.

## Microsoft funziona

Microsoft Works era una suite di software per la produttività sviluppata da Microsoft. È stato introdotto per la prima volta nel 1987 e ha guadagnato popolarità come alternativa più economica alla suite completa di Microsoft Office. Microsoft Works includeva diverse applicazioni per ufficio di base, come:

- **Elaboratore di testi:** Microsoft Works disponeva di un'applicazione di elaborazione testi che consentiva agli utenti di creare e modificare documenti. Utilizzava il formato file [.WPS](/it/word-processing/wps/) per i documenti di elaborazione testi.

- **Foglio di calcolo:** includeva un'applicazione per fogli di calcolo per la creazione e la gestione dei fogli di calcolo. Il formato file per i fogli di calcolo era [.XLR](/it/spreadsheet/xlr/).

- **Database:** Microsoft Works disponeva anche di un'applicazione database per l'organizzazione e la gestione dei dati. Il formato file per i file di database era [.WDB](/it/database/wdb/).

- **Calendario e rubrica:** oltre alle principali applicazioni per ufficio, Microsoft Works spesso includeva strumenti per la gestione di calendari e rubriche.

- **Formato elaboratore di testi di Works (.WPS):** il formato file nativo del componente di elaborazione testi di Microsoft Works. I file .WPS contenevano testo, formattazione e grafica semplice.

- **Interruzione:** Microsoft ha ufficialmente interrotto Microsoft Works nel 2007 e ha incoraggiato gli utenti a passare a Microsoft Office o ad altre alternative.
