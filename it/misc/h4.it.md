## Che cos'è un file H4?

Un H4 o HDF4 è uno dei formati di dati gerarchici (HDF) e viene utilizzato per archiviare grandi quantità di dati. È un formato di file open source che supporta la memorizzazione di dati eterogenei complessi. HDF4 è un formato dati più vecchio rispetto a [HDF5](/it/misc/h5/). I formati di file HDF sono mantenuti e pubblicati dal gruppo HDF, un'organizzazione senza scopo di lucro. I file HDF supportano anche l'archiviazione di metadati per rappresentare completamente i dati archiviati.

## Breve storia del formato di file HDF

La formulazione dei formati di dati gerarchici è stata avviata nel 1987 dalla Graphics Foundations Task Force (GFTF). Durante questo periodo, sono stati proposti diversi formati di file per la standardizzazione del formato singolo. La NASA ha scelto l'HDF come metodo standard per archiviare e fornire dati scientifici.

## Formato file HDF4

I file H4 vengono salvati in una "directory di file" come una struttura secondo le specifiche del formato di file HDF. Queste specifiche definiscono la struttura complessiva dei file HDF per l'archiviazione su disco, ma senza che gli utenti si preoccupino della struttura del file sottostante.

Le aree di applicazione dei file H4 includono aerospaziale, fisica, ingegneria, finanza, ricerca accademica, genomica, astronomia, strumenti elettronici e campi medici.

### Modello dati HDF4

Il file HDF4 o H4 è come un contenitore costituito da oggetti dati eterogenei. Il tipo di set di dati supportati dai file H4 include:

* Immagini
* Tabelle
* Elementi di testo
* Grafici
* Documenti come [PDF](/it/pdf/) e [Documenti Excel](/it/spreadsheet/)

## Visualizzatore di file H4

Il gruppo HDF fornisce [software HDFView](https://www.hdfgroup.org/solutions/hdf5/) che consiste nell'utilità HDFView e nel pacchetto di oggetti Java HDF.
