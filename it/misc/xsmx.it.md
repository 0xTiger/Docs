## Che cos'è un file XSMX?

Un file XSMX è un file di questionario creato con l'applicazione ExamSoft SoftTest che viene utilizzato per condurre gli esami. Viene utilizzato per inviare il foglio d'esame agli studenti che prendono online e inviare il risultato come file XMDX. ExamSoft fornisce soluzioni di valutazione basate sulle decisioni dei driver di dati attraverso la tecnologia didattica e l'applicazione SofTest disponibile sia per Windows che per MacOS è una di queste. Il file XSMX contiene tutte le domande e viene aperto nell'applicazione SofTest sul computer dello studente.

## Formato file XSMX

I file XSMX vengono salvati e inviati ai candidati all'esame come file protetti da password. La protezione con password ha lo scopo di garantire la sicurezza e la segretezza delle domande d'esame. La password dell'esame è condivisa con l'utente finale per l'apertura del file e lo svolgimento del test.
