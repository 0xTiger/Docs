## Cos'è un file PDO?

Pepakura Designer è un programma che consente agli utenti di creare modelli artigianali in carta da modelli 3D; I file Pepakura in genere hanno un'estensione .pdo; questi file contengono istruzioni e informazioni sul layout necessarie per stampare e assemblare un modello cartaceo.

I file Pepakura Designer sono in un formato proprietario specifico del software Pepakura; il formato del file è progettato per memorizzare informazioni sul modello 3D, inclusa la sua geometria, i riferimenti alle texture e le istruzioni di assemblaggio.

Pepakura Designer funziona con modelli 3D in formati come [.obj](/3d/obj/), [.3ds](/3d/3ds/), [.dxf](/cad/dxf/) e altri; questi modelli vengono poi spiegati in modelli 2D che possono essere stampati e assemblati nel modello cartaceo finale.

Il file .pdo include istruzioni su come tagliare, piegare e incollare pezzi di carta stampata per ricreare il modello 3D; ogni pezzo è etichettato e numerato per guidare il processo di assemblaggio.

## Come aprire un file PDO?

Per aprire e visualizzare i file di Pepakura Designer, avrai bisogno di **Pepakura Viewer**, un programma complementare gratuito a **Pepakura Designer**. Il visualizzatore consente ad altri di visualizzare e stampare modelli senza bisogno di un software di progettazione completo.

## Informazioni su Pepakura Designer

Pepakura Designer è un'applicazione software specializzata progettata per la creazione di modelli artigianali in carta da modelli digitali 3D. Il software consente agli utenti di importare modelli 3D in vari formati, come .obj o .3ds, e quindi dispiegare e appiattire questi modelli in modelli 2D che possono essere stampati su carta. Questi modelli, salvati con estensione file .pdo, contengono istruzioni e segni per tagliare, piegare e assemblare i pezzi di carta in un modello tridimensionale. Pepakura Designer è ampiamente utilizzato da hobbisti e appassionati interessati a creare intricati modelli di carta di vari oggetti, personaggi e disegni.
