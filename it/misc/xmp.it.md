## Che cos'è un file XMP?

I file XMP sono file di metadati che contengono le modifiche apportate a un file raw da fotocamera, come [CR2](/it/image/cr2/) o [NEF](/it/image/nef/). Vengono automaticamente generati e salvati nella stessa directory insieme alle immagini JPEG corrispondenti. XMP è stato creato da Adobe Systems Inc. per la creazione, l'elaborazione e lo scambio di metadati standardizzati e personalizzati per documenti digitali e set di dati. È stato accettato come **standard ISO 16684-1:2019 parte 1 e ISO 16684-2:2014 parte 2**. XMP è un marchio registrato di Adobe Systems Incorporated.

## Formato file XMP - Ulteriori informazioni

XMP è un formato di serializzazione basato sull'approccio del modello di dati. Stabilisce i principi per la definizione e l'elaborazione dei metadati. Il formato può essere utilizzato per incorporare informazioni XMP in diversi tipi di file come immagini, video e documenti, ad esempio [JPEG](/it/image/jpeg/) e [PDF](/it/pdf/).

## Cosa c'è all'interno di un file XMP?

I file XMP contengono informazioni sui metadati su vari aspetti dell'immagine grezza. Supporta dati come:

* Informazioni sull'autore e sul copyright
* Informazioni sulle proprietà dell'aspetto dell'immagine come luminosità, esposizione, contrasto, angolo di ritaglio, ombra, saturazione, nitidezza, tinta e temperatura
