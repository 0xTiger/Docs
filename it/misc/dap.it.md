## Che cos'è un file DAP?

Un file DAP è un file in corso di download creato dal file Download Accelerator Plus (DAP). Viene creato quando gli utenti avviano un download da qualsiasi repository di file online utilizzando il software DAP. DAP offre la possibilità di sospendere e riprendere un download in corso. A tale scopo salva il file parzialmente scaricato con estensione .dap fino al completo scaricamento. Il file DAP può essere visualizzato in anteprima durante il processo di download.

I file DAP possono essere aperti con il software SpeedBit Download Accelerator Plus (DAP).
