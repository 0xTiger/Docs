## Che cos'è un file OGG?

Un file OGG è un file grafico creato con il software Origin. Fornisce la capacità di memorizzare diversi tipi di grafici per visualizzare i dati importati per l'analisi dei dati. I tipi di grafico supportati dal tipo di file OGG includono colonna, barra, torta, cascata, 3D, polare e simbolo. Il software Origin è stato sviluppato da OriginLab e consente agli utenti di eseguire analisi dei dati e generare output sotto forma di grafici.

## Formato file OGG

I file OGG contengono grafici per la rappresentazione dei dati importati. Quando un file OGG viene aperto, visualizza il grafico in una finestra separata in Origin o Origin Viewer, che è disponibile gratuitamente rispetto al software Origin. Anche le informazioni sui metadati come il titolo e i colori di un grafico sono archiviate all'interno del file OGG.

## Informazioni sul software OriginLab Origin

Il software Origin offre anche la possibilità di essere utilizzato come server di automazione. In tal caso, le applicazioni client come LabVIEW, Microsoft Excel, Python o le applicazioni sviluppate su misura in C#/VB/VC possono scambiare dati con Origin. [Python_to_OriginLab](https://github.com/jsbangsund/python_to_originlab) è un'API Python per automatizzare l'invio di dati e le funzioni di stampa di base in OriginLab.
