## Che cos'è un file SSP?

Un file SSP è un file di progetto creato dal software di creazione screensaver di Windows noto come Screensaver Producer di Axialis Software. Può contenere diversi tipi di elementi salvaschermo come sprite, filmati Flash, presentazioni di foto e video. Un file SSP può comprendere testo, immagini, forme diverse e impostazioni di animazione utilizzate per mostrare l'animazione all'interno del file SSP. I file SSP possono essere aperti con Axialis Screensaver Producer.

## Formato file SSP - Ulteriori informazioni

I file SSP vengono archiviati su disco come file binari senza informazioni pubbliche disponibili sulla struttura del formato file interno. Axialis Screensaver produttore professionale strumento di creazione di screensaver per Windows che consente di creare facilmente screensaver di alta qualità per tutte le versioni di Windows fino a Windows 10. Consente di creare pacchetti salvaschermo installabili che possono essere facilmente distribuiti online agli utenti.

I file SSP possono essere esportati nel formato file [SCR](/it/system/scr/).
