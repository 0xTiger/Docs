I file di sistema sono un tipo integrale di file che si trova nei sistemi operativi Microsoft Windows, Mac e Linux. I file di sistema sono uno dei file più importanti archiviati nel sistema operativo del dispositivo in quanto contengono tutte le informazioni relative al sistema del dispositivo e, senza di esso, un dispositivo è destinato a non funzionare correttamente o non funzionare affatto. Per questo motivo i file di sistema sono altamente protetti con meccanismi di autorizzazione grazie ai quali non possono essere eliminati accidentalmente o anche intenzionalmente dal computer facilmente. Per una facile specifica e identificazione, questi file sono sempre archiviati in cartelle di sistema specifiche.

I file di sistema si trovano in vari tipi di estensione di file e ogni sistema operativo ha il proprio tipo di estensione di file di sistema univoco per scopi diversi. Le estensioni dei file di sistema più comuni sono; [CAB](/it/system/cab/), SYS, [DLL](/it/system/dll/) e DRV.
