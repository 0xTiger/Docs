## Cos'è un file PAT?

Un file PAT è comunemente associato a **DiskStation Manager (DSM)**, il sistema operativo utilizzato dai dispositivi Synology Network-Attached Storage (NAS). Il file PAT è il file di installazione utilizzato per aggiornare o installare il DSM su un Synology NAS.

## Utilizzo del file PAT

Ecco come in genere si utilizza un file PAT per installare o aggiornare DSM su Synology NAS:

1. Scarica il file PAT: puoi ottenere il file PAT dal sito Web ufficiale di Synology o da altre fonti attendibili.
    







2. Accedi al NAS: accedi all'interfaccia utente basata sul web del tuo Synology NAS inserendo il suo indirizzo IP in un browser web. Avrai bisogno dei privilegi di amministratore per eseguire questa operazione.
    







3. Vai al Centro pacchetti: nell'interfaccia web, vai a "Centro pacchetti". Qui è dove gestisci e installi le applicazioni sul tuo NAS.
    







4. Installazione manuale: nel Centro pacchetti dovrebbe essere presente un'opzione per "Installazione manuale" o "Installa/Aggiorna" a seconda della versione di DSM in uso. Scegli questa opzione.
    







5. Cerca il file PAT: ti verrà richiesto di sfogliare i file locali e selezionare il file PAT scaricato.
    







6. Installa o aggiorna: dopo aver selezionato il file PAT, seguire le istruzioni visualizzate sullo schermo per installare DSM (se si tratta di una nuova installazione) o aggiornare DSM (se si sta aggiornando a una versione più recente).
    







7. Attendi il completamento: il processo di installazione o aggiornamento può richiedere del tempo e il NAS potrebbe riavviarsi come parte del processo. Sii paziente e aspetta che finisca.
    







8. Configurazione post-installazione: dopo l'installazione o l'aggiornamento, potrebbe essere necessario configurare le impostazioni del NAS in base alle proprie preferenze.

## Gestore della DiskStation

DiskStation Manager, spesso abbreviato in DSM, è un sistema operativo sviluppato da Synology per i dispositivi NAS (Network-Attached Storage). Serve come interfaccia di gestione e controllo per i server Synology NAS. DiskStation Manager fornisce un'interfaccia web intuitiva che consente agli utenti di configurare e gestire vari aspetti del proprio NAS come archiviazione dati, condivisione file, soluzioni di backup, servizi multimediali e altro ancora.

DSM è noto per la sua versatilità e l'ampio ecosistema di pacchetti che consente agli utenti di installare ed eseguire varie applicazioni e servizi sul proprio Synology NAS trasformandolo in un server multiuso sia per uso domestico che aziendale. Alcune delle funzionalità comuni di DiskStation Manager includono condivisione di file, backup e sincronizzazione dei dati, streaming multimediale, funzionalità di sicurezza e supporto di virtualizzazione.

Synology rilascia regolarmente aggiornamenti e nuove versioni di DSM per migliorare la sicurezza, le prestazioni e le funzionalità. Gli utenti possono aggiornare manualmente il DSM utilizzando i file PAT o configurare gli aggiornamenti automatici per garantire che il proprio NAS esegua la versione più recente e sicura di DiskStation Manager.

## Come aprire un file PAT

Per aggiornare manualmente DiskStation Manager, è possibile utilizzare un file PAT scaricato da Synology Download Center utilizzando i seguenti passaggi:

1. Accedere al Pannello di controllo di DiskStation Manager.
2. Passare alla sezione "Aggiorna e ripristina" e scegliere "Aggiornamento DSM".
3. Da lì, seleziona "Aggiornamento DSM manuale".
4. Fai clic sul pulsante "Sfoglia", quindi individua e scegli il file PAT che hai scaricato.
5. Per avviare l'aggiornamento di DiskStation Manager, fare clic sul pulsante "Applica".
