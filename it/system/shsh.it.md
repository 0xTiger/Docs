## Cos'è un file SHSH?

Un file SHSH, noto anche come Signature HaSH, è una firma digitale utilizzata da Apple per autenticare e verificare gli aggiornamenti del firmware per dispositivi iOS come iPhone, iPad e iPod.

## Differenza tra i formati di file SHSH e SHSH2

Come un file SHSH2, il file SHSH contiene un identificatore univoco per il dispositivo chiamato "ECID" (Exclusive Chip ID), nonché informazioni sulla versione del firmware installata sul dispositivo. Tuttavia, i file SHSH venivano utilizzati nelle versioni precedenti di iOS (precedenti a iOS 10) e da allora sono stati sostituiti dai file SHSH2.

## Formato file SHSH: ulteriori informazioni

I file SHSH vengono salvati su disco in formato file binario. I dettagli della struttura interna del file di questo formato file non sono disponibili pubblicamente.

I file SHSH sono importanti per gli utenti che desiderano eseguire il downgrade del firmware del proprio dispositivo a una versione precedente che non è più firmata da Apple. Salvando i file SHSH per una versione firmware specifica quando è ancora firmata da Apple, gli utenti possono successivamente utilizzarli per bypassare la verifica della firma di Apple e installare quella versione firmware sul proprio dispositivo. Questo processo è noto anche come "jailbreaking".
