Un formato di file descrive la struttura dei dati in un file che le applicazioni seguono per leggere o creare tali file.

I tipi di file possono essere raggruppati in categorie come formati di file vettoriali, formati di file di fogli di calcolo, formati di file video, formati di file immagine, formati di file 3D e molti altri. Un file può anche essere convertito in un altro formato di file come BMP in PNG.
