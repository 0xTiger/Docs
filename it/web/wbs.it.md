## Che cos'è un file WBS?

Un file WBS è un file di progetto creato da WebBlender che è un programma di presentazione e creazione di siti Web per bambini. Il file WBS viene creato quando viene avviato un nuovo progetto in WebBlender. Tutte le funzionalità del sito aggiunte al progetto, come pagine, immagini, testo, animazioni e così via, sono referenziate nel file di progetto WBS. Una volta che il progetto è pronto, pubblica i contenuti del sito come file [HTML](/it/web/html/). WebBlender è creato e gestito da [Tech4Learning](https://www.tech4learning.com/).
