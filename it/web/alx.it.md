## Che cos'è un file ALX?

Un file ALX è un file ActiveX incorporato nelle pagine Web sotto forma di componenti [HTML](/it/web/html/). I controlli ActiveX erano formalmente noti come controlli OLE o OCX. I file ALX possono essere utilizzati per inserire una varietà di funzionalità in una pagina Web, ad esempio l'aggiunta di un controllo timer per aggiornare periodicamente il contenuto della pagina o aggiungere un controllo della finestra popup per visualizzare la guida in stile descrizione comando basata sull'azione dell'utente. I file ALX vengono collegati utilizzando<Param> tag all'interno del codice HTML di una pagina web. Puoi aprire file ALX in Microsoft Internet Explore, Microsoft ActiveX Control Pad e editor di testo come Notepad e Notepad++.

## Formato file ALX - Ulteriori informazioni

I file ALX sono in formato di file di testo normale e possono essere modificati aprendoli in un editor di testo. I controlli ActiveX possono essere inclusi in una pagina Web trascinandoli nella pagina Web. Tuttavia, per creare pagine web dinamiche, i controlli sono integrati con l'aiuto di [JScript](/it/web/js/). I controlli su una pagina Web sono associati agli eventi e ai valori dei controlli, fornendo così il controllo completo per le interazioni dell'utente.
