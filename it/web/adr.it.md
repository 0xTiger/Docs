## Che cos'è un file ADR?

Un file ADR (salvato con estensione .adr) è un elenco di segnalibri creato dal browser Web Opera. Questo elenco viene popolato quando l'utente continua a memorizzare gli URL Web nei segnalibri. Questi segnalibri vengono archiviati nel file ADR di cui è possibile eseguire il backup e ripristinarlo quando necessario. Poiché la maggior parte dei browser come Chrome e Firefox salva i segnalibri in formato [HTML](/it/web/html/), questi possono essere convertiti in formato file ADR per l'importazione nel browser Opera. I file ADR possono essere aperti nel browser Opera.

## Formato file ADR - Ulteriori informazioni

I dettagli del formato file interno dei file ADR sono sconosciuti. Tuttavia, esistono differenze tra i formati di file ADR esportati con diverse versioni di Opera. Ciò rende difficile [unire i file ADR](https://superuser.com/questions/471959/how-do-i-merge-several-opera-adr-bookmark-files-to-one-single-file) generare con diverse versioni di Opera. Tuttavia, alcuni strumenti esterni possono essere utilizzati per unirli.
