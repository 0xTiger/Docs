## Che cos'è un file VRT?

Un file VRT è un formato di file 3D che contiene mondi tridimensionali che possono essere visualizzati nel browser Web con il plug-in SVR [Viscape](https://www.viscape360.com/). Viene utilizzato dal plug-in per caricare e visualizzare i dati del mondo 3D all'utente per la visualizzazione e l'analisi. Le informazioni memorizzate in un file VRT possono comprendere dati come trame, suoni, dimensioni e comportamenti. I file VRT più recenti vengono archiviati nel formato di file SVR che è una versione compressa del file VRT.

## Formato file VRT

I file VRT sono archiviati nel formato di file proprietario di Viscape e i dettagli sul formato del file interno non sono disponibili pubblicamente.
