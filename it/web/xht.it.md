## Che cos'è un file XHT?

Un file con estensione .xht è un file Web associato al tipo di file [XHTML](/it/web/xhtml/) (Extended Hypertext Markup Language), che a sua volta è un file di markup basato su [XML](/it/web/xml/). Entrambi questi file sono simili all'HTML ma hanno una sintassi simile a XML più rigorosa. I file XHT sono progettati in modo da essere più strutturati, implicano meno script e sono generici oltre all'utilizzo delle strutture esistenti di XML.

## Formato file XHT

Un file XHT viene creato e archiviato come file di testo normale con codifica UTF-8 per impostazione predefinita. Contiene il codice sorgente completo di un documento XHTML che può essere aperto e modificato con qualsiasi editor di testo che supporti la codifica Unicode. Oltre agli editor di testo, il formato di file XHT è comprensibile dalla maggior parte dei moderni browser Web e può essere aperto utilizzando questi.

## Esempio XHT

L'esempio seguente di un documento XHT definisce le dichiarazioni XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
