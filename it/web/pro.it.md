## Che cos'è un file PRO? ##

Un file PRO è un mockup di un sito Web o di un'applicazione mobile creata da Adobe Proto, un'app Adobe Touch defunta. Include uno o più layout dell'interfaccia utente per una pagina Web o un'app mobile, ognuno dei quali può includere elementi interattivi. Le tecnologie CSS, WebKit e jQuery sono supportate nei file PRO.

### Breve storia del formato file PRO

I file PRO sono stati creati da Adobe Proto che è stato sostituito da Adobe XD. È stato utilizzato per creare app di prototipazione per tablet. Le comuni applicazioni di web design come Adobe Dreamweaver utilizzavano questi file PRO per creare applicazioni web e mobili.

## Formato file PRO - Ulteriori informazioni

Un file PRO è il primo passo nello sviluppo di un sito Web o di un'applicazione mobile. Gli sviluppatori creano prima modelli digitali che definiscono l'aspetto dell'applicazione. Questi definiscono anche il flusso di lavoro generale dell'applicazione e la navigazione attraverso le schermate dell'applicazione. Questi modelli includono anche una serie di elementi di navigazione come pulsanti, caselle combinate, barre di scorrimento, campi di input e molti altri.
