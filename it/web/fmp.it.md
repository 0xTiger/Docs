## Che cos'è un file FMP?

Un file con file .fmp è un file di menu animato creato con l'applicazione 123 Flash Menu. Contiene menu animati per il sito Web, inclusi componenti come caratteri, stile del menu, colori, dimensioni, suoni e animazioni. L'applicazione 123 Flash Menu non richiede particolari competenze informatiche poiché l'applicazione stessa incorpora tutte le funzionalità per sviluppare tali menu. I file del menu FMP possono essere esportati nei famosi file Flash [.swf](/it/page-description-language/swf/) che possono essere incorporati all'interno di una pagina web.
