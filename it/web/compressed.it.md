## Cos'è un file COMPRESSO?

Un file COMPRESSO è una pagina Web compressa Wordpress di piccole dimensioni generata con il plug-in **WordCompress**. Contiene una singola riga di codice per rappresentare la pagina web generata comprimendo il codice HTML, CSS e JavaScript dal plugin. Questi file COMPRESSI convertiti sono di piccole dimensioni e consentono un caricamento e un trasferimento più rapidi su Internet.

### Formato file COMPRESSO - Ulteriori informazioni

I file COMPRESSI vengono salvati come file di testo che possono essere aperti con qualsiasi editor di testo come Microsoft Notepad, Notepad++ e Apple TextEdit. Oltre agli editor di testo, i file COMPRESSI possono essere aperti con software di creazione web come Adobe Dreamweaver, disponibile sia per Windows che per Mac OS.

## Informazioni sul plugin WordCompress

Il plug-in WordCompress è stato inizialmente utilizzato per comprimere le pagine Web di WordPress in un formato di file compresso. Il plugin non è più supportato ed è fuori produzione. Scriverebbe il codice sorgente della pagina web in un singolo file eseguibile rimuovendo tutti gli spazi bianchi dal codice sorgente della pagina. I file COMPRESSI vengono archiviati insieme al codice sorgente originale della pagina Web per preservare i commenti originali, i rientri e altri spazi bianchi inclusi nel codice sorgente originale della pagina.
