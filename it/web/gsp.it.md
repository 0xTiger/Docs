## Che cos'è un file SPG?

Un file GSP è un file di messaggi di errore che indica un problema con la consegna dell'e-mail. Viene inviato da IMail Server, un programma server di posta elettronica utilizzato per le comunicazioni e-mail da numerose piccole e medie imprese. L'errore potrebbe essere dovuto a una serie di motivi come un'enorme quantità di dati, la perdita di dati nel file [MSG](/it/email/msg/) o errori di memoria insufficiente.

## Formato file SPG - Ulteriori informazioni

I file GSP sono archiviati in formato file Web e sono facilmente visualizzabili con client di posta elettronica come Microsoft Outlook. Un file GSP viene generato come risposta a un'e-mail inviata che genera un errore.
