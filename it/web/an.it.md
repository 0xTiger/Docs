## Che cos'è un file AN?

Un file con estensione .an è un file di progetto [Adobe Edge Animate](https://www.adobe.com/products/animate.html) che contiene informazioni sull'animazione. Al salvataggio del progetto viene creato un file .html che contiene i riferimenti ai file [.css](/it/web/css/) e [.js](/it/web/js/) associati generati da Adobe Edge Animate. Il file AN viene salvato insieme a questi file che possono essere utilizzati per aprire il file di progetto. Adobe edge animate può essere utilizzato come alternativa alla creazione di pagine Web animate utilizzando Adobe Flash, che non è più supportato dai browser.

## Un formato di file

I file AN vengono archiviati su disco in formato di file di testo normale e possono essere aperti in editor di testo per visualizzarne il contenuto.
