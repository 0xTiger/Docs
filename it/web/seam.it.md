## Cos'è un file SEAM?

Un file SEAM è un file di configurazione utilizzato dal framework JBOS Seam. Contiene metadati/riferimenti a una servlet lato server che elabora le richieste all'applicazione web. Contiene anche metadati sui componenti Seam come i nomi delle classi e dei metodi che implementano i componenti. È possibile utilizzare i file SEAM insieme al framework JavaServer Faces (JSF) per creare applicazioni Web dinamiche e interattive. I file SEAM sono solitamente archiviati nella directory WEB-INF di un'applicazione Seam.

## Formato file SEAM

Un file SEAM viene memorizzato come file di testo, tipicamente in formato [XML](/it/web/xml/), e contiene informazioni di configurazione per un'applicazione Seam. È una parte fondamentale della configurazione di un'applicazione Seam. È possibile aprire un file SEAM con qualsiasi editor di testo o XML.
