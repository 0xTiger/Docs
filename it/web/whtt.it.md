## Che cos'è un file WHTT?

Un file WHTT è una versione offline di un sito Web che viene scaricata utilizzando il software [HTTrack Website Copier](https://www.httrack.com/). Contiene tutti i contenuti di un sito Web, comprese le pagine Web [HTML](/it/web/html/), le immagini, i fogli di stile, i collegamenti e le directory. Tutte le pagine di un sito Web vengono scaricate in modo ricorsivo e salvate su disco come mirror offline. Quando un sito Web viene scaricato dal software HTTrack, può essere visualizzato offline senza la necessità di essere connesso a Internet. Per visualizzare il sito Web scaricato, è sufficiente aprire la pagina del sito Web con mirroring nel browser e ogni immagine/collegamento verrà caricato correttamente.

## Formato file WHTT

I file WHTT vengono salvati come una combinazione di pagine Web THML, immagini, fogli di stile e collegamenti di riferimento.
