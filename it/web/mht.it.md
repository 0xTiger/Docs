## Che cos'è un file MHT?

Un file con estensione .mht è un formato di file di archiviazione abilitato MIME che contiene diversi tipi di dati in un unico file. Può memorizzare dati come testo, immagini, stili di pagina sotto forma di file [CSS](/it/web/css/), JavaScript e altre risorse come risorse incorporate in esso. I file MHT, con messaggio di tipo MIME/rfc822, incapsulano tutto il contenuto di un file HTML come un unico file di archivio per l'archiviazione su dispositivi di archiviazione. Le applicazioni software come Microsoft Word ti consentono di convertire i tuoi documenti WORD in MHT esportandoli come file MHT. I file MHT possono essere aperti utilizzando i browser più diffusi come Microsoft Internet Explore e Google Chrome.

## Formato file MHT

Come [MHTML](/it/web/mhtml/), anche MHT è un incapsulamento MIME di documenti HTML aggregati. I contenuti del documento all'interno di un documento MHT come immagini in linea, fogli di stile, applet, ecc. sono codificati MIME dove sono collegati a una risorsa radice tramite URI. I client di posta elettronica come Microsoft Outlook archiviano le e-mail (di solito [EML](/it/email/eml/)) in formato file MHT. Le specifiche complete del formato di file MHT sono disponibili in [RFC 822](https://tools.ietf.org/html/rfc822) e possono essere consultate per lo sviluppo di applicazioni software in grado di elaborare file MHT.

## Differenza tra MHT e MHTML

Durante il salvataggio di una pagina online, all'utente viene chiesto di definire il tipo di formato di file all'interno del quale la pagina online deve essere salvata sul sistema nativo. Nella maggior parte dei casi, l'utente viene confuso tra il linguaggio di markup e i formati di file MHTML. Notano che è problematico vedere che il formato del file è più sano tra questi.

Quando un utente decide di evitare di sprecare una pagina online come linguaggio di markup, viene formata una cartella, che contiene più file per i vari contenuti della pagina ovvero unità di area di file completamente diverse create per testo, grafica, applet ecc. D'altra parte, il salvataggio la pagina online come MHTML crea un file per la pagina online completa. Tutti gli elementi della pagina insieme a grafica, link e file alternativi sono integrati all'interno di un unico file.

Naturalmente, è semplice gestire file MHT o MHTML perché il file può essere protetto e condiviso semplicemente tramite e-mail. Tuttavia, l'unità dell'area dei file del linguaggio di markup al di sotto della possibilità di perdita di informazioni poiché la perdita o l'eliminazione anche di un file potrebbe produrre la cartella del linguaggio di markup completa inutile per l'utente.
