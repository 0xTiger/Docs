## Che cos'è un file DCR?

Un file DCR è un file multimediale che viene creato e salvato nel formato Adobe Shockwave. Viene utilizzato per la creazione di giochi Web e giochi interattivi simili a quelli progettati e salvati nei file [SWF](/it/page-description-language/swf/). I file DCR sono stati precedentemente creati con Adobe Director che è stato successivamente interrotto dopo l'acquisizione da parte dei sistemi Adobe. I file DCR sono simili ai file SWF creati da Adobe Flash.
