## Cos'è un file SITE?

Un file SITE è un file di progetto utilizzato dal software di progettazione di siti Web "Adobe GoLive". Memorizza informazioni sul progetto del sito Web come file, cartelle, immagini, fogli di stile personalizzati e script. Contiene inoltre la struttura del sito web e i collegamenti utilizzati nelle pagine web del sito. Poiché si tratta di un file di progetto, contiene riferimenti ai file di progetto ma non contiene i file veri e propri.

Adobe GoLive è stato interrotto e non è più supportato.

## Formato file SITO

I file SITE vengono archiviati su disco nel formato di file proprietario di Adobe e i dettagli interni del relativo formato di file non sono disponibili pubblicamente come riferimento dello sviluppatore.
