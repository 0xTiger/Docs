## Che cos'è un file DAP?

Un file con estensione .dap è un file di pagina dati di Microsoft Access creato da Microsoft Access e che ha accesso ai dati archiviati nel database. I dati archiviati in un tale file Web sono accessibili sulla rete (intranet e Internet) che è diversa dai file [MDB](/it/database/mdb/) di Access sviluppati specificamente per applicazioni desktop. Tali database possono essere creati utilizzando Access 2010 e Access Services (un componente facoltativo di SharePoint). Questi database possono essere usati da persone che dispongono di account SharePoint utilizzando l'indirizzo Web in un browser Web.
