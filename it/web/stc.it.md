## Che cos'è un file STC?

Un file con estensione .stc è un file di connessione utilizzato dal software Adobe Contribute per memorizzare le informazioni di connessione. Contiene informazioni di connessione come nome utente, URL host, protocolli come FTP, SFTP, ecc. per il riutilizzo dall'interno del sito Web esistente. Per chi non lo sapesse, Adobe Contribute era un editor [HTML](/it/web/html/) specializzato che veniva utilizzato per inviare aggiornamenti a siti Web esistenti, inclusi articoli basati sui contenuti noti come blog. Comprendeva plug-in per browser Web popolari come Internet Explorer e Firefox che contribuivano senza problemi ai contenuti dei siti Web dall'interno delle pagine. È stato interrotto da gennaio 2017 e Adobe Dreamweaver è ora utilizzato per creare siti Web da zero.

## Formato file STC

Il formato dei dati interni dei file STC è sconosciuto. Tuttavia, la chiave di connessione è crittografata con una password e tutte le informazioni di connessione alla rete o all'accesso FTP inviate nel file rimangono protette e sono accessibili solo tramite Contribute. Il file di connessione della chiave può essere inviato tramite posta elettronica agli utenti o salvato sul computer. Quando viene cliccato, si apre automaticamente Contribute che importa la connessione.
