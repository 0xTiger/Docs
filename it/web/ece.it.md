## Che cos'è un file ECE?

Un file ECE è una pagina Web generata dinamicamente dalla piattaforma di gestione dei contenuti, **Escenic Content Engine (ECE)**. Contiene codice lato server che viene elaborato dal server Web e restituito al browser dell'utente come pagina [HTML](/it/web/html/). Il formato di file ECE non è molto popolare rispetto ad altri formati di file lato server come ASP e PHP. Questi file vengono visualizzati internamente e serviti agli utenti finali a scopo di visualizzazione.

## Formato file ECE

I file ECE non vengono salvati su disco ma vengono forniti direttamente all'utente su richiesta. Questi vengono convertiti in HTML e rispediti al browser dell'utente per la visualizzazione. Escenic Content Engine viene utilizzato attivamente per la pubblicazione di pagine di articoli di notizie. Basato sulla tecnologia JEE e SQL, il server ECE fornisce un repository affidabile e scalabile per tutti i contenuti online.

I browser client possono accedere al contenuto sui server ECE tramite l'API HTTP/REST. Oltre a servire pagine [HTML](/it/web/html/), ECE fornisce un flusso di lavoro altamente efficiente per la gestione e la pubblicazione di contenuti video/audio.
