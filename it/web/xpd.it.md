## Cos'è un file XPD?

Un file XPD è un file di pagina Web creato da un documento XML. Più spesso, questo viene generato tramite una trasformazione XSLT. Le [specifiche del formato file XPD](https://www.w3.org/TR/xml-pipeline/) sono disponibili su W3 che definiscono un modo per aumentare l'interoperabilità di multiprocesso [XML](/it/web/xml/) applicazioni. L'XML Pipeline Language definisce le relazioni di elaborazione tra le risorse XML.
