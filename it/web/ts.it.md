## Che cos'è un file TS?

Un file TS è un file di traduzione generato con l'utilità software lupdate fornita in bundle con l'SDK Qt di Digia. L'SDK è disponibile per i sistemi operativi Windows, Mac e Linux. Il file TS contiene frasi in una lingua e le loro traduzioni in un'altra lingua. Le traduzioni memorizzate all'interno di un file TS sono specifiche per una particolare applicazione. I file TS possono essere aperti con Digia Qt SDK.

## Formato file TS

I file Digia Qt TS vengono archiviati su disco in formato file XML e possono essere aperti in qualsiasi editor di testo come Blocco note e Blocco note ++. Alcuni altri formati di file supportati da Digia Qt SDK includono:

* .pri
* .qm
* .qml
* .qph
* .qrc
* .qs
* .qss
* .ts
* File dell'interfaccia utente .ui
