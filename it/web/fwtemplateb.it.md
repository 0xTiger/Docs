## Che cos'è un file FWTEMPLATEB?

Un file FWTEMPLATEB è un pacchetto di documenti modello di sito Web utilizzato da Freeway Pro, un'app software per la creazione di siti Web senza scrivere alcun codice. Contiene più modelli di siti Web che possono essere condivisi anche con altri utenti di Freeway Pro. Ciascun sito Web modello nel pacchetto contiene informazioni sul layout e sull'aspetto del sito.

## Formato file FWTEMPLATEB

I file FWTEMPLATEB vengono salvati come una raccolta di modelli di siti Web e contengono informazioni relative all'aspetto del sito. Questi forniscono il modo più semplice per creare un sito Web poiché tutte le impostazioni di layout e aspetto sono già state eseguite nel file modello. Un altro vantaggio dei file FWTEMPLATEB è che ottieni un pacchetto di modelli per creare facilmente siti Web.
