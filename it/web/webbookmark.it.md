## Che cos'è un file WEBBOOKMARK?

Un file WEBBOOKMARK è un file di segnalibri salvato con il browser Web Safari di Apple su macOS. Contiene informazioni sulla pagina Web che l'utente ha salvato dal browser. Le informazioni memorizzate in un file WEBBOOKMARK includono il titolo della pagina, l'URL e il gruppo di segnalibri in cui è memorizzata questa pagina. Quando una pagina web viene inserita nei preferiti, il browser Safari chiede al gruppo di memorizzare la pagina nei preferiti. La maggior parte degli utenti salva i segnalibri nella cartella Segnalibri, che è la posizione predefinita offerta dai browser comuni.

## Formato file WEBBOOKMARK

I file WEBBOOKMARK vengono salvati su disco in un formato di file binario che non è leggibile dall'uomo. Gli utenti possono contrassegnare una pagina Web come segnalibro se è intenzionata a essere visitata in seguito o si riferisce agli interessi dell'utente. Quando richiesto, l'utente può navigare nella sezione dei segnalibri nel browser Safari e visitare nuovamente la pagina.
