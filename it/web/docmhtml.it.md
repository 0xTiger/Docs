## Che cos'è un file DOCMHTML?

Un file DOCMHTML è una vecchia variante dell'archivio web salvato dalle versioni precedenti di Microsoft Word. Non è più supportato dall'ultima versione di Microsoft Word. Un file DOCMHTML contiene una rappresentazione HTML del documento word .doc originale. Può essere aperto in Microsoft Word così come nel browser web. Quando un DOCMHTML viene pubblicato, può essere ospitato su un server Web accessibile agli utenti tramite richiesta Web. Un altro formato simile è [DOCHTML](/it/web/dochtml/), anch'esso interrotto.

## Formato file DOCMHTML - Ulteriori informazioni

I file DOCMHTML sono stati utilizzati per essere salvati in formato HTML. Potrebbe essere aperto in un editor di testo come Blocco note e Blocco note ++. Dopo essere stato contrassegnato come obsoleto, Microsoft Word ha introdotto le opzioni per salvare i documenti di Word come file [MHTML](/it/web/mhtml/) e [MHT](/it/web/mht/). Questi nuovi formati di file possono essere aperti con i browser web.
