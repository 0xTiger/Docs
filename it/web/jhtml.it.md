## Che cos'è un file JHTML?

Un file con estensione .jhtml è un file [HTML](/it/web/html/) con codice [Java](/it/programming/java/) che viene eseguito sul server quando un client richiede questa pagina in un browser. Il server elabora le richieste, esegue le funzioni Java contenute nella funzione e restituisce la pagina al browser del client. Gli oggetti Java incorporati nelle pagine JHTML vengono eseguiti sul server per gestire le richieste di pagine di questo tipo. I file JHTML possono anche accedere alle informazioni dal database utilizzando la connessione JDBC (Java [Database](/it/database/) Connectivity). I file JHTML possono essere aperti in qualsiasi editor di testo e visualizzati in browser Web come Google Chrome, Firefox e Safari.

## Formato file JHTML

JHTML è una tecnologia proprietaria di ATG e può essere creata utilizzando l'editor di documenti Dynamo ATG (Art Technology Group). I file JHTML vengono scritti in formato di file di testo normale utilizzando il codice HTML e Java standard. Questi contengono tag HTML standard oltre a tag proprietari che fanno riferimento a oggetti Java. Quando tale pagina viene richiesta dall'utente, il server HTTP inoltra la richiesta a un server di applicazioni Java. La pagina JHTML viene prima convertita in un file .java e compilata per generare un file [.class](/it/programming/class/) che viene eseguito come servlet. Questo genera un flusso di dati HTTP e HTML standard che vengono restituiti al browser che richiede la visualizzazione per l'utente. Ciò è utile per eseguire query relative al database sul server e restituire il risultato accumulato finalizzato al browser del client.
