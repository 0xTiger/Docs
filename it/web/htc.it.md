## Che cos'è un file HTC?

Un file HTC è un file in formato file Web utilizzato da Microsoft Internet Explorer. Contiene definizioni per componenti HTML che eseguono piccole funzioni per eseguire diverse attività all'interno del browser web. La funzionalità definita all'interno dei file HTC è riutilizzabile ed è accessibile su più pagine [HTML](/it/web/html/).

## Formato file HTC - Ulteriori informazioni

I file HTC vengono salvati in formato di file di testo normale. Ciò significa che puoi aprirli in qualsiasi editor di testo e apportare modifiche alla funzionalità. Questi vengono solitamente aperti in Internet Explorer, ma in questo modo potrai solo visualizzare questi file e non potrai modificarli.

Un file HTC è in realtà un file HTML che è una combinazione di script e un insieme di elementi specifici di HTC per definire i componenti.
