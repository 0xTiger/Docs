## Che cos'è un file AWM?

Un file AWM è un file di progetto creato dal programma di creazione di menu Web, **AllWebMenus**. Contiene dati relativi al progetto per la creazione di menu a discesa basati su [HTML](/it/web/html/). Un file AWM viene compilato in un pacchetto Web finalizzato che si integra nei siti Web per visualizzare i menu. Contiene tutti i dati relativi al progetto come impostazioni dell'aspetto, voci di menu, stile (verticale o orizzontale), colori e altre informazioni correlate.

I file AWM possono essere aperti con **Likno AllWebMenus**.

## Formato file AWM - Ulteriori informazioni

I file AWM vengono salvati come file binari compilati e possono essere aperti solo con il software Likno AllWebMenus. I dettagli del formato file interno dei file AWM non sono disponibili pubblicamente.

AllWebMenus ti consente di progettare visivamente menu [CSS](/it/web/css/) o menu basati su [JavaScript](/it/web/js/) senza utilizzare alcun linguaggio di programmazione. I menu possono essere completamente personalizzati e progettati utilizzando modelli di aspetto predefiniti disponibili con il software.
