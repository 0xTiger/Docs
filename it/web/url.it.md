## Che cos'è un file URL?

Un file con estensione .url è un collegamento all'URL di un sito Web che può essere salvato sul computer con qualsiasi indirizzo di sito Web. Quando viene aperto facendo doppio clic, l'URL aprirà il sito Web nel browser predefinito e la pagina verrà visualizzata all'utente. I file URL sono generalmente indicati come file di collegamento ai siti Web e sono rappresentati dall'icona del browser predefinito. I file URL possono essere aperti e modificati in qualsiasi editor di testo come Microsoft Notepad, Notepad++ o Apple TextEdit.

## Formato file URL - Ulteriori informazioni

I file URL vengono salvati in formato di testo normale e possono essere aperti e visualizzati in qualsiasi editor di testo. Questi si aprono nel browser Web predefinito in cui l'utente esegue altre esplorazioni.
