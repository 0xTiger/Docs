## Che cos'è un file EDGE?

Un file con estensione .edge è un file Adobe Edge Project che viene creato quando un progetto viene creato e salvato con Adobe Edge. Adobe Edge consente ai web designer di creare animatori [HTML](/it/web/html/) interattivi per il Web, l'editoria digitale e la pubblicità rich media per browser desktop e mobili. Ciò si ottiene utilizzando HTML/HTML5, [JavaScript](/it/web/js/) e [CSS3](/it/web/css/). I file Edge vengono generati come file di testo normale e possono essere aperti con diverse applicazioni come Microsoft Notepad, Apple TextEdit e Notepad++.

## Formato file EDGE

I file EDGE vengono salvati in un formato di testo normale e utilizza jQuery per abilitare l'animazione. Un'alternativa all'animazione EDGE erano i file Flash che non sono più supportati. Il file EDGE viene salvato insieme ai file HTML, JavaScript e CSS. All'apertura del file EDGE, viene aperta la pagina Web, ovvero il file HTML.
