## Che cos'è un file LESS?

Un file LESS è un foglio di stile utilizzato per le pagine Web e creato con i fogli di stile Leaner (LESS). Implementa aggiunte al linguaggio CSS standard, che lo rende facile da imparare e implementare. Un file LESS estende il [CSS](/it/web/css/) aggiungendo funzionalità come variabili, mixin, annidamento, operazioni, funzioni, spazi dei nomi e accessor.

## Formato file di fogli di stile più snello

I file LESS vengono creati nel formato CSS, estendendo le sue capacità alle funzionalità aggiuntive che offre. Questi vengono salvati come file di testo e possono essere aperti nel linguaggio dei fogli di stile LESS, CodeKit e Adobe Dreamweaver.
