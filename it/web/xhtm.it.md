## Cos'è un file XHTM?

Un file con estensione .xhtm è un file di pagina Web scritto in XHTML. Utilizza XML 1.0 come implementazione standard ed è considerato una riformulazione di HTML 4. Come XHTML, un file XHTM contiene tag HTML standard che seguono linee guida XML più severe. I file XHTM vengono utilizzati per pubblicare contenuti di siti Web e comunemente utilizzano l'estensione [XHTML](/it/web/xhtml/).

È possibile utilizzare browser Web come Google Chrome, Apple Safari e Mozilla Firefox per aprire file XHTM.

## Esempio XHTM

Il seguente esempio di documento XHTM definisce le dichiarazioni XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
