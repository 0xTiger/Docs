## Cos'è un file CPHD?

Un file CPHD è un documento ipertestuale utilizzato dal motore del preprocessore CloudChan. Contiene HTML e script per creare pagine Web e documenti dinamici. Gli script all'interno di un file CPHD assomigliano alla sintassi utilizzata da Java. La pagina Web CPHD restituisce un documento HTML quando eseguita su un processo CloudChan. Il formato CPHD può funzionare con qualsiasi documento XML.

## Formato file CPHD

I file CPHD vengono archiviati su disco come file di testo normale. Ciò rende possibile aprire questi file in qualsiasi editor di testo come Blocco note o Blocco note++.
