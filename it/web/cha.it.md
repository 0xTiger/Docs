## Che cos'è un file CHA?

Un file CHA è un file di impostazioni utente creato dai client IRC come mIRC. Contiene parametri come il server, la porta, il nome del canale e, facoltativamente, una password per l'apertura di un canale IRC. I client IRC vengono utilizzati per comunicare con altri sulle reti IRC di tutto il mondo. Questi sono usati per discussioni private uno-a-uno o in conferenze multigruppo. Quando si fa clic su un file CHA, il client IRC si apre automaticamente con i dettagli menzionati nel file.

Le applicazioni che possono aprire file CHA includono mIRC e Visual IRC.

## Formato file CHA

I file CHA vengono archiviati insieme ad altri file di installazione del client IRC. Quando un utente vuole connettersi a un server e unirsi a un canale, inserisce le informazioni nel client. Queste informazioni sono memorizzate nel file CHA che può essere recuperato in qualsiasi momento.
