## Che cos'è un file EWP?

Un file EWP è un file di progetto creato con il software Website Builder [EWISoft](http://www.ewisoft.com/). È collegato a un file di database binario che contiene tutte le informazioni per il progetto. Il file EWP è comunemente usato per salvare i progressi dello sviluppo web. È un software di progettazione Web basato su modelli che consente agli utenti di creare siti Web da modelli di siti Web.

## Formato file EWP - Ulteriori informazioni

I file EWP sono file di progetto che non devono essere compilati o ospitati su alcun server. Questi vengono salvati come file di testo normale e contengono riferimenti allo stato di avanzamento dello sviluppo web. I progetti possono essere pubblicati su un sito Web una volta completato lo sviluppo.
