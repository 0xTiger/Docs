## Che cos'è un file EPIBRW?

Un file EPIBRW è un file generico ospitato su un server Web basato su Microsoft IIS. Lo scopo di questo file non è specifico per operazioni specifiche o alcun tipo di archiviazione dei dati, ma è generale. Viene spesso utilizzato dai file [.pdf](/it/pdf/), ma può essere utilizzato anche da molti altri tipi di file.

Quando si fa doppio clic su un file EPIBRW, è possibile che venga aperto nel browser visualizzandone il contenuto. Tuttavia, se il file è collegato per il download, il processo di download del file verrà avviato come risultato di questo evento.
