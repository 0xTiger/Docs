## Che cos'è un file DOCHTML?

Un file DOCHTML è il formato HTML di un documento generato con Microsoft Word. È la versione [HTML](/it/web/html/) del documento creato che può essere utilizzata per la pubblicazione sul server web. I file DOCHTML possono essere aperti nel browser web per visualizzare il contenuto del documento word convertito. La prospettiva e l'aspetto del file DOCHTML sono gli stessi del documento Word originale.

## Formato file DOCHTML

I file DOCHTML vengono salvati su disco in formato HTML pronto per essere pubblicato sui server web. Microsoft Word ha interrotto il supporto per il salvataggio di documenti come file DOCHTML. Questo è stato sostituito con i formati di file [.mht](/it/web/mht/) e [.mhtml](/it/web/mhtml/) comunemente usati ora.
