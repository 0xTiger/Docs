## Che cos'è un file OBML16?

Un file OBML16 è una pagina Web salvata utilizzando il browser Web mobile Opera Mini. Si basa su Opera Binary Markup Language ([OBML](/it/web/obml/)) ed è un formato compatto per consentire un rapido accesso alle pagine web salvate. Ha sostituito il formato di file [OBML15](/it/web/obml15/) quando è stata lanciata Opera Mini versione 7. Tutte queste varianti delle pagine Web di Opera Mini sono state aggiornate dal formato di file OBML di base e ogni aggiornamento di Opera Mini è compatibile con un solo formato OBML. I file OBML16 possono essere convertiti in [PDF](/it/pdf/) e [HTML](/it/web/html/) online.

## Formato file OBML16 - Ulteriori informazioni

OBML16 si basa sul formato di file OBML che è un formato compatto per salvare le pagine web. Il vantaggio dell'utilizzo di questo formato compatto è che riduce le dimensioni del contenuto web, con conseguente aumento della velocità di download. Una volta scaricate, gli utenti di Opera Mini possono visualizzare queste pagine senza la necessità di avere accesso a una connessione Internet.
