## Cos'è un file CCBJS?

Un file CCBJS è un file Javascript creato dall'app 3D, dal gioco e dall'editor del sito Web, **CopperCube**. Contiene il codice JavaScript generato come parte del [file .html](/it/web/html/) creato dal file .ccb. CopperCube fornisce un'interfaccia di scripting intuitiva per sviluppare la logica e le interazioni del gioco senza scrivere alcun codice. Altri formati di file supportati da CopperCube includono .ccscene, .ccmodel, .ccmaterial, .cctex, .ccjs, .ccp e .ccpak.

## Formato file CCBJS

I dettagli del formato file interno del formato file CCBJS non sono noti come tali. I file Javascript, tuttavia, vengono salvati come file di testo semplice su disco e possono essere aperti con qualsiasi editor di testo come Microsoft Notepad, Notepad++ e Apple TextEdit.
