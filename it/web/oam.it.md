## Cos'è un file OAM?

Un file .oam un file widget animato esportato da Adobe Edge Animate Widget. Le animazioni esportate come formato file widget OAM possono essere inserite in altre applicazioni Adobe come Documenti InDesign ([file INDD](/it/page-description-language/indd/), Dreamweaver e Muse. I file OAM sono come pacchetti di distribuzione che possono essere facilmente inseriti in altri progetti Adobe per utilizzarli. I file OAM contengono informazioni sulle risorse, sui comportamenti e sul codice actionscript dell'animazione. Puoi creare un file OAM pubblicando un progetto Adobe Animate [file .AN](/it/web/an/) .
Gli utenti possono creare file OAM pubblicando da un progetto Edge Animate (file .AN).

## Formato file OAM

Un file OAM viene salvato su disco come archivio compresso [ZIP](/it/compression/zip/). Ciò implica che puoi rinominare l'estensione del file in .zip ed estrarlo con qualsiasi utilità di compressione come WinZIP o WinRAR. La dimensione ridotta del file semplifica il trasferimento e la condivisione dell'animazione con altri utenti su Internet.

### Struttura dei file OAM

La struttura interna di un file .oam è proprietaria e non è documentata pubblicamente da Adobe. Tuttavia, è noto che i file .oam contengono una raccolta di file e risorse (come grafica, audio e codice ActionScript) racchiusi in un unico file. Il contenuto di un file .oam può includere file [XML](/it/web/xml/), file SWF e altri file di risorse. La struttura esatta del file .oam dipende dalla versione specifica di Adobe Animate utilizzata per crearlo, nonché dal tipo di animazione e risorse incluse nel file.

Un file OAM contiene le seguenti informazioni.

"Risorse:" informazioni sulle risorse grafiche, audio e video utilizzate nell'animazione.

"Comportamenti:" descrizioni delle azioni e delle interazioni che si verificano nell'animazione.

"Codice ActionScript:" il codice utilizzato per controllare il comportamento e l'interattività dell'animazione.

"Impostazioni pubblicazione:" informazioni sulla piattaforma e sul formato in cui verrà pubblicata l'animazione, come HTML5 o AIR.

"Metadati:" informazioni aggiuntive come l'autore dell'animazione, la data di creazione e il numero di versione.

## Come aprire il file OAM?

È possibile utilizzare le seguenti applicazioni per aprire file OAM.

* Adobe Edge Animate CC (fuori produzione)
*Adobe Animate 2023
*Adobe InDesign 2023
*Adobe Dreamweaver 2021
