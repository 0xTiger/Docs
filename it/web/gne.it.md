## Che cos'è un file GNE?

Un file GNE è una pagina Web generata dinamicamente e utilizzata dal sito Web di condivisione di foto di Flickr. Le pagine sono ospitate sul server da dove vengono servite al browser dell'utente richiedente. I file GNE sono simili alle pagine PHP e ASP, ma sono sviluppati in un linguaggio di programmazione proprietario i cui dettagli non sono disponibili come riferimento per gli sviluppatori.

## Formato file GNE - Ulteriori informazioni

I file GNE sono stati utilizzati dal gioco per computer, Game Never Ending, sviluppato da [Ludicorp](https://en.wikipedia.org/wiki/Ludicorp). Flicker è stato successivamente acquisito da yahoo! nel marzo 2005.
