## Che cos'è un file HYPE?

Un file HYPE è un file di progetto salvato utilizzando il software Hype utilizzato per la creazione di contenuti Web con HTML5. Contiene elementi web come grafica, testo, animazione, informazioni sulla scena e altri elementi che costituiscono collettivamente il contenuto. Il contenuto Web in un file HYPE può essere esportato in file Web che possono essere pubblicati su una pagina Web.

Un file HYPE può essere aperto con il software [Tumult Hype](https://tumult.com/hype/).

## Formato file HYPE

I file HYPE sono archiviati in un formato di file proprietario e i dettagli interni di questi non sono noti pubblicamente. I contenuti all'interno di un file HYPE possono essere visualizzati solo quando aperti nel software Tumult Hype.

### Contenuto di un file HYPE

Un file HYPE viene creato quando si utilizza l'applicazione Hype per creare contenuto Web per scopi diversi. Memorizza informazioni sulle animazioni basate su fotogrammi chiave per la creazione di contenuti come eCard, eBook, presentazioni, pubblicità, infografica e materiale per pagine web. Il contenuto di questi è alimentato da HTML5 che consente di visualizzare il contenuto sui dispositivi dai desktop agli iPhone e iPad. Tumult Hype genera lo stato dell'arte degli stili HTML5, [CSS3](/it/web/css/) e [JavaScript](/it/web/js/).
