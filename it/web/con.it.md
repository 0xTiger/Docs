## Che cos'è un file CON?

Un file CON è un file di codice sorgente per l'applicazione sviluppata per **Concept Application Server**. Viene utilizzato per la scrittura di applicazioni per lo scambio di dati tra il server e l'utente. I file CON ospitati sul server sono accessibili con un browser Web a condizione che Concept Client sia installato sul lato utente. Concept Application Server è un server web con un linguaggio di programmazione su piccola scala che può avere un motore di database di sottoinsiemi [SQL](/it/database/sql/) (TinDB).

I file CON possono essere aperti/eseguiti con RadGs Concept Application Server.

## Formato file CON - Ulteriori informazioni

I file CON sono ospitati sul server e vi si accede utilizzando il browser Web sulla macchina dell'utente. Concept [URLs](/it/web/url/) sono diversi dai normali URL HTTP e iniziano con **concept://**.

### CON Esempio di URL del file

È possibile accedere a un file CON ospitato sul Concept Application Server tramite browser Web utilizzando URL quali:

```
concept://domain.server.com/web_application/main.con.
```
