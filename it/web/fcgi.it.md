## Che cos'è un file FCGI?

Un file FCGI è un file di codice sorgente che contiene il codice sorgente scritto per FastCGI (FCGI). Può essere scritto in diversi linguaggi di programmazione e contiene l'estensione dello standard CGI utilizzato per una gestione più robusta delle richieste del server Web. I linguaggi di supporto più comuni per FCGI includono **Perl** che fornisce supporto sia per il codice Perl che per la potente interfaccia esterna dell'API FastCGI.

## Formato file FCGI

Cosa rende importante un file FCGI? L'interfaccia FCGI supporta l'elaborazione distribuita per gestire le richieste Web e distribuirle su più server. Ciò rende i file FCGI utilizzati principalmente per il bilanciamento del carico su più server.
