## Che cos'è un file AEX?

Un file AEX (con estensione .aex) è un file [Xbasic](https://documentation.alphasoftware.com/documentation/pages/Ref/Xbasic/index.xml) compilato che contiene il codice del programma compilato per le funzioni globali. Può essere utilizzato nelle applicazioni web chiamandole dalle pagine lato server [.a5w](/it/web/a5w/). I file AEX sono stati caricati e scaricati chiamando rispettivamente le funzioni a5w_load_aex e a5w_unload_aex dai file A5W. Tuttavia, nella nuova implementazione, questi file vengono caricati in memoria includendoli nel file a5_application.5i. Il linguaggio di programmazione Xbasic è utilizzato dal software Alpha Anywhere per lo sviluppo di applicazioni Web e Mobile.

## Formato file AEX - Ulteriori informazioni

I file AEX vengono salvati su disco in formato binario e contengono il codice compilato delle funzioni scritte in Xbasic. Le istruzioni di comando dello script Xbasic determinano la struttura e il flusso di esecuzione, inclusa l'esecuzione o l'interruzione di un'azione ripetuta o la scelta tra due o più passaggi in base alle condizioni. [Xbasic Language Reference](https://documentation.alphasoftware.com/documentation/pages/Ref/Xbasic/index.xml) può essere consultato per informazioni dettagliate a riguardo.
