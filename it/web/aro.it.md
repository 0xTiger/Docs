## Che cos'è un file ARO?

Un file ARO è un file di pagina Web lato server generato con l'applicazione Web SteelArrow. Contiene tag e script [HTML](/it/web/html/) e SteelArrow. Questi vengono eseguiti sul server per generare una pagina di risposta completa prima di inviarli al browser dell'utente. I file ARO contengono quasi il 95% del contenuto HTML mentre il resto è costituito da codice SteelArrow. Affinché i file ARO funzionino per te, il server delle applicazioni Web SteelArrow deve essere installato e in esecuzione sul computer del server Web.

## Formato file ARO

I file ARO sono scritti in un file del linguaggio di markup HTML con codice JavaScript incorporato e applet Java. I [tag SteelArrow](http://www.steelarrow.com/docs/basics1.aro) sono gli elementi costitutivi di base per lo sviluppo di pagine web. Sebbene questi non modifichino la visualizzazione della pagina, vengono utilizzati per riempire la pagina di dati. Inoltre, possono essere utilizzati anche per controllare l'output con istruzioni condizionali.

### Esempio di tag ARO

Il<SAOUTPUT> Il tag ARO consente agli sviluppatori di generare valori di dati in qualsiasi punto all'interno di uno script. Ad esempio, può essere utilizzato per ottenere l'output della tabella PARAM con<SAOUTPUT VALUE=PARAM[]> che può essere visualizzato all'interno dell'HTML<BODY> etichetta.
