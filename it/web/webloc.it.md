## Cos'è WEBLOC?

Un file WEBLOC è un file di collegamento su macOS che punta a un URL Web su Internet. Viene generato trascinando l'URL web dalla barra degli indirizzi del browser sul desktop o su qualsiasi altra cartella in cui deve essere salvato. L'URL del sito di destinazione è memorizzato all'interno di questo file di collegamento e può essere visualizzato aprendo il file in un editor di testo come Microsoft Notepad, Notepad++ o Apple TextEdit. I file WEBLOC sono simili ai file [URL](/it/web/url/).

## Formato file WEBLOC - Ulteriori informazioni

I file WEBLOC sono archiviati in un formato di file di testo normale. Questi possono essere generati su macOS trascinando gli URL da browser Web come Safari, Google Chrome o Firefox. Se devono essere aperti più file WEBLOC contemporaneamente, sarebbe necessaria una soluzione più rapida e semplice per aprirli tramite applicazioni o API esterne. Queste API possono leggere e convertire WEBLOC in PDF e JPEG.
