## Cos'è un file SVR?

Un file SVR è un formato di file 3D utilizzato da SuperScape per creare mondi interattivi 3D. Questi mondi interattivi tridimensionali possono essere visualizzati tramite browser Web con il plug-in SVR. I file SVR sono simili al formato [file VRT](/it/web/vrt/) ma sono compressi, il che comporta dimensioni di file più piccole. Pertanto, i file SVR vengono caricati più velocemente e possono anche essere condivisi su Internet.

## Formato file SVR: ulteriori informazioni

I file SVR vengono salvati su disco in formati di file binari. È un formato di file proprietario e i dettagli della struttura del file interno non sono disponibili pubblicamente come riferimento per lo sviluppatore.
