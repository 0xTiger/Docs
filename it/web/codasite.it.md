## Che cos'è un file CODASITE?

Un file CODASITE è un file di progetto di un sito Web creato con il software di creazione e pubblicazione di pagine Web [Coda](https://coda.io/). Contiene un elenco di tutti i file che fanno parte del progetto. Ciò include pagine Web come file [HTML](/it/web/html/), fogli di stile a cascata [CSS](/it/web/css/), file Javascript [JS](/it/web/js/) e qualsiasi altro file di supporto correlato. I file di progetto CODASITE contengono anche informazioni per la pubblicazione di siti Web sul server Web.

## Formato file CODASITE

Il file di progetto CODASITE viene salvato su disco come file binari all'interno della directory ~/​Library/​Application Support/​Coda 2/​Sites/​. L'interfaccia utente di Coda fornisce l'interfaccia per la gestione dei siti Coda poiché l'apertura manuale di questi può portare a impostazioni di progetto non corrette e creare problemi.
