## Che cos'è un file FWP?

Un file con .fwp è un file di archivio creato con Microsoft Expression Web e contiene tutti i file e le cartelle per un progetto. Microsoft Expression Web è un prodotto software di progettazione Web generale di Microsoft e si basa sull'editor [HTML](/it/web/html/). È stato interrotto dal 2012 ed è disponibile gratuitamente per il download da Microsoft. I file FWP possono essere esportati e salvati nel formato .cab di Microsoft.

## Formato file FWP - Ulteriori informazioni

Le specifiche esatte del formato di file dei file FWP non sono disponibili come riferimento per gli sviluppatori. Tuttavia, questi sono file binari e possono essere aperti in Microsoft Expression Studio che è stato interrotto molto tempo fa.
