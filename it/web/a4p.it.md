## Che cos'è un file A4P?

Un file A4P è un file di progetto creato dal programma software A4Desk Pro che viene utilizzato per creare siti Web Flash. Viene creato con i modelli inclusi con A4Desk Pro e non necessita di una conoscenza approfondita di Flash. I file A4P creati con i modelli possono includere il contenuto del sito Web sotto forma di testo e animazioni Flash. Nel caso in cui non conosci Flash, è stato utilizzato per creare animazioni che potevano essere esportate in [SWF](/it/page-description-language/swf/) e in formati di file [GIF](/it/image/gif/) per l'inclusione nei siti web.

## Formato file A4P

I file A4P vengono creati da file modello e salvati come file binari su disco. La struttura del formato file interno dei file A4P non è nota.

I file A4P possono essere aperti con il software **Avanquest A4Desk Pro**.
