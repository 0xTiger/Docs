## Che cos'è un file P7C?

Un file P7C, come altri file di certificati di sicurezza, è un certificato digitale utilizzato per autenticare l'identità sulla rete. Le applicazioni che utilizzano certificati autenticano l'identità utilizzando la chiave pubblica inclusa in questi file di certificato. La crittografia a chiave pubblica, o crittografia asimmetrica, utilizza una coppia di chiavi di cui una è la chiave pubblica e l'altra è nota come chiave privata. La chiave privata è protetta per una sicurezza efficace, mentre la chiave pubblica può essere condivisa con altri. Altri formati di file di certificato comuni includono [CRT](/it/web/crt/), DER e PEM.

## Formato file P7C - Ulteriori informazioni

I file P7C vengono salvati su disco come file binari e includono le informazioni sulla chiave pubblica generate utilizzando algoritmi crittografici basati su problemi matematici. Questi possono essere aperti in qualsiasi editor di testo ma il loro contenuto non è in forma leggibile dall'uomo. Alcune delle applicazioni in grado di aprire i file P7C includono Apple Keychain Access, Adobe Acrobat Reader DC, Microsoft Certificate Manager e Contatti Microsoft Windows.
