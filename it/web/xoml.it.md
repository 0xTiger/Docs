## Che cos'è un file XOML?

XOML è l'acronimo di Extensible Object Markup Language ed è un formato di serializzazione per gli oggetti del flusso di lavoro di Windows Workflow Foundation. Basato su **[XAML](/it/web/xaml/)**, viene utilizzato principalmente per la creazione di interfacce utente in chiaro **[XML](/it/web/xml/)**. Questi vengono utilizzati per dichiarare il flusso di lavoro per l'interfaccia utente e vengono compilati insieme al file separato contenente la logica di implementazione. Contiene una struttura ad albero che definisce un nodo del flusso di lavoro radice, elementi secondari nidificati e segmenti di codice incorporati. Quando viene creato un nuovo flusso di lavoro con Visual Studio per .NET, è disponibile un'opzione per scegliere se deve essere in formato Codice o Separato da codice. Nel caso in cui selezioni quello successivo, l'IDE creerà due file per te; uno in formato XOML (composto da layout e impostazioni del flusso di lavoro) e l'altro è in formato [.CS](/it/programming/cs/)/[.VB](/it/programming/vb/) in cui risiede il codice di programmazione.
