## Che cos'è un file HXS?

Un file HXS è un file della Guida compilato e utilizzato per il file della Guida Microsoft 2.x. Viene compilato e generato da un insieme di pagine di argomenti e scritto in un sottoinsieme di [HTML](/it/web/html/) che è simile al precedente formato di file della guida [CHM](/it/web/chm/). I file HXS sono archiviati in formato compresso e vengono creati da un file di progetto principale .HxC, un file include .HxF, un sommario .HxT, un file di definizione dell'attributo .HxA e diversi indici .HxK. I metadati di un file HXS vengono creati da [XML](/it/web/xml/).

I file HXS possono essere aperti con un browser web.

## Formato file HXS

I file HXS vengono creati e salvati come file compressi da contenuto HTML e metadati XML.
