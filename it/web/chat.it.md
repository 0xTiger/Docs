## Che cos'è un file CHAT?

Un file CHAT è un file di configurazione utilizzato dall'IRC (Internet Relay Chat) per la connessione a un canale IRC. Contiene parametri di connessione come server, porta, nome del canale IRC e, facoltativamente, una password per l'apertura di un canale IRC. Le impostazioni vengono utilizzate dal client per connettersi ad altri canali. Le chat e le discussioni su IRC sono individuali e private.

Le applicazioni che possono **aprire** file CHAT includono mIRC e Visual IRC.

## Formato file CHAT

I file CHAT vengono archiviati come file binari su disco. Quando un utente vuole connettersi a un canale in IRC, immette le informazioni nel client iRC, inserisce le informazioni nel software client. Le informazioni vengono memorizzate nel file CAHT per il recupero laterale.
