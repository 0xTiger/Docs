## Che cos'è un file XOJ?

Un file XOJ è un file del taccuino che contiene gli appunti presi utilizzando il software per prendere appunti a mano Xournal. Xournal è un'applicazione multipiattaforma gratuita che ti consente di creare note con diversi tipi di contenuto come testo, immagini e scarabocchi scritti a mano. I file XOJ vengono creati quando viene creato un nuovo file notebook con Xournal. Questi file XOJ possono essere condivisi anche con altri utenti per la condivisione dei dati. Una versione più recente di Xournal è [Xournal++](https://github.com/xournalpp/xournalpp) che è una variante multipiattaforma di Xournal.

## Formato file XOJ

I file XOJ sono in formato file XML e vengono salvati su disco in forma compressa utilizzando la compressione [.gz](/it/compression/gz/). Grazie all'uso del formato XML universale, i file XOJ possono anche essere aperti e modificati con la variante multipiattaforma Xournal++.

Ogni file XOJ può contenere un'altra pagina di un taccuino Xournal.

## XOJ - Piattaforme supportate

Xournal è multipiattaforma e i suoi notebook possono essere salvati come file XOJ sui seguenti sistemi operativi.

*Linux
* Mac OS
* Finestre
* Androide
* Versione iOS
