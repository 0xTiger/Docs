## Che cos'è un file BWP?

Un file BWP è un file di script di una pagina Web creato nel linguaggio di programmazione **BuRg3r**. È specifico di BuRg3r e può contenere codice HTML con sezioni del codice sorgente di BuRg3r delimitate da<?b and ?> tag. BuRg3r è un linguaggio di programmazione generico che è anche un motore di gioco sviluppato da Fluffy Smoke Entertainment.
