## Che cos'è un file GDOCX?

Un file GDOCX è un tipo di file di collegamento creato che punta a documenti di elaborazione testi online archiviati in Google Drive. Viene creato con Google Docs e viene salvato nel Google Drive dell'utente per impostazione predefinita. I file GDOCX sono simili a [GDOC](/it/web/gdoc/) che sono più comunemente usati. Le informazioni sul file di destinazione sono archiviate in file GDOCX in testo normale che possono essere aperti con qualsiasi editor di testo.

## Formato file GDOCX - Ulteriori informazioni

I file GDOCX sono simili a GDOC e contengono l'URL di destinazione che deve essere aperto. Questi risiedono nella cartella del client di Google Drive sul sistema dell'utente e rimangono sincronizzati con la cartella di Google Drive online. Google ha fornito la versione client desktop di Google Drive fino al 2018, quando è stata finalmente interrotta. Tuttavia, gli utenti possono ancora creare e archiviare i propri file nello spazio di archiviazione online di Google Drive online.
