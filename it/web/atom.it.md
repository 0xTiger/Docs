## Che cos'è un file ATOM?

Un file ATOM è un formato Syndication per la struttura del feed Atom e dei documenti di ingresso. Simile ai file .RSS, il formato di file ATOM è un formato di syndication basato su XML che è uno standard per la pubblicazione di contenuti. Un file ATOM viene utilizzato per i feed Web che consentono ai programmi software di verificare la presenza di aggiornamenti pubblicati sul sito Web. Contiene voci che possono essere di diverso tipo come titoli, articoli full-text, estratti o collegamenti a contenuti su un sito web

Le applicazioni che possono **aprire file ATOM** includono **Apple Safari**.

## Formato file ATOM - Ulteriori informazioni

I file ATOM vengono archiviati e pubblicati nel popolare formato di file XML che è un formato universale per lo scambio di informazioni. È stato sviluppato come alternativa all'RSS tenendo in considerazione i limiti e i difetti del formato di file RSS.

### Tipi di documenti Atom

1. "Atom Entry Documents" - Si tratta di un documento XML che consiste in un singolo elemento di informazione, noto come entry, per il feed Atom. Ha un atom-entry elemento che contiene un numero di elementi figlio. Il contenuto di una voce Atom può essere di testo normale, HTML, XHTML o un altro tipo di supporto IANA (Internet Assigned Numbers Authority).
1. "Documenti Atom Feed" - È un documento XML che fornisce metadati su un feed Atom e una o più voci per il feed. Quando un client effettua una richiesta di informazioni dal feed, il documento del feed viene generato dal server che include un numero di voci Atom per soddisfare la richiesta.
1. "Collezione Atom" - È un tipo speciale di documento feed Atom che contiene gli URL delle voci Atom che sono disponibili per essere modificate.
