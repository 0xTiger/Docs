## Che cos'è un file DOWNLOAD?

Un file DOWNLOAD è un file parzialmente scaricato da Internet che è ancora in fase di download. Quando scarichi un file da Internet utilizzando il browser Apple Safari, il completamento del download potrebbe richiedere del tempo. Questo file parzialmente scaricato viene salvato nella cartella **Download**. Se il download viene interrotto per qualsiasi motivo, è possibile fare doppio clic sul file per provare a riprenderlo.

## SCARICA Formato file - Ulteriori informazioni

I file DOWNLOAD vengono salvati su disco in formato testo e file binario. Non contiene l'intero file in fase di download. Una volta completato il download, l'estensione del file .download viene sostituita con l'estensione originale del file che si sta scaricando. Ad esempio, se stai scaricando un video da Internet in formato file MP4, il file verrà scaricato e salvato con estensione .download. Una volta completato il download, l'estensione del file verrà modificata in .mp4.

I file DOWNLAOD sono simili ai file [.CRDOWNLOAD](/it/web/crdownload/) utilizzati da Google Chrome e Microsoft Edge.
