## Cos'è un file VDISCO?

Un file VDISCO è un file di rilevamento utilizzato da Microsoft Visual Studio nei suoi servizi Web. Fornisce informazioni sui servizi Web disponibili come URL, spazi dei nomi e metodi. Il file VDISCO è simile al formato file [DISCO](/it/web/disco/) ma viene generato in fase di runtime. Viene archiviato nell'applicazione del servizio Web e viene utilizzato da Visual Studio per individuare e utilizzare dinamicamente i servizi Web in un progetto. I file VDISCO vengono utilizzati in combinazione con il file [.asmx](/it/web/asmx/) che contiene il codice del servizio Web effettivo.

Puoi aprire il file VDISCO in qualsiasi editor di testo come Microsoft Notepad, Github Atom e Apple TextEdit. Può anche essere aperto con Microsoft Visual Studio 2012 e versioni successive per lavorarci.

## Formato file VDISCO

Un file VDISCO viene salvato e ospitato nel formato di file [XML](/it/web/xml/), che è un formato universale per la composizione e la pubblicazione dei dati. Contiene l'URL del servizio, gli spazi dei nomi e i metodi nei tag XML standard in cui ciascun tag contiene il rispettivo valore delle sue informazioni.
