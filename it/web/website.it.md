## Che cos'è un file SITO WEB?

Un file SITO WEB è un file di collegamento a un sito Web creato con Microsoft Internet Explorer (IE). Contiene un URL e un file di icone (ICO) che punta alla favicon del sito web. L'icona viene utilizzata per visualizzare il collegamento. Il formato del file SITO WEB è stato introdotto con il rilascio di IE versione 9. Le versioni precedenti di IE erano utilizzate per salvare i collegamenti alle pagine Web come file [.url](/it/web/url/).

## Formato file SITO WEB

Un file .website viene archiviato su disco come file di testo normale i cui contenuti possono essere visualizzati in qualsiasi editor di testo. Oltre ad altri contenuti, contiene l'URL di destinazione per il quale è una scorciatoia. Se desideri aprire il sito Web di destinazione in un browser diverso da IE, puoi copiare questo URL e incollarlo in altri browser come Google Chrome, Safari o Firefox per visualizzare il sito Web.

## Come si crea un file .SITO WEB?

È possibile creare un file SITO WEB in IE facendo clic sull'URL nella barra degli indirizzi e trascinandolo nella posizione desiderata sul computer, ad esempio desktop o documenti. Questo crea il collegamento a quella posizione Internet e può essere aperto facendo doppio clic sul collegamento. Se vuoi salvare il collegamento come file `URL`, tieni premuto il tasto MAIUSC prima di trascinare la favicon del sito di destinazione sul desktop.
