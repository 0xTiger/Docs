## Che cos'è un file CRDOWNLOAD?

Un file CRDOWNLOAD è un file scaricato parzialmente che viene creato quando scarichi un file da Internet utilizzando i browser Microsoft Edge, Google Chrome o Chromium. Il contenuto del file in fase di download viene archiviato in questo file fino al completamento del download. Una volta completato il download, l'estensione .crdownload viene rimossa e sostituita con l'estensione del file originale del file in fase di download.

## Formato file CRDOWNLOAD

Un file CRDOWNLOAD può essere sia in formato testo che binario. I browser creano il file .crdownload per impostazione predefinita una volta avviato il download. Se il download viene interrotto nel frattempo, il file CRDOWNLOAD rimane salvato su disco. Questo a volte consente di riprendere il download da dove è stato interrotto.

### CRDOWNLOAD Esempio

Supporto per il download di un file fileformatslist.doc da Internet. All'avvio del download, il browser salva il file nella cartella Download come file fileformatslist.crdownload. La sua dimensione del file aumenta man mano che il file viene scaricato. Una volta completato il download, il file verrà rinominato in fileformatslist.doc.
