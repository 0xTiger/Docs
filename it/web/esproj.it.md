## Che cos'è un file ESPROJ?

Un file ESPROJ è un file di progetto creato da pagine Web che creano software, Espresso. È disponibile su Mac OS e contiene un elenco dei file correlati che fanno parte del progetto del sito web. Quando si apre il file ESPROJ in Espresso, vengono aperti tutti i file correlati nell'editor per la modifica o l'aggiornamento del codice sorgente. Il file ESPROJ contiene anche le impostazioni di pubblicazione del server e altre preferenze utente correlate.

## Formato file ESPROJ

I file ESPROJ vengono salvati come file binari e il suo contenuto non è leggibile dall'uomo. La directory dei file ESPROJ è anche la directory principale del sito Web.
