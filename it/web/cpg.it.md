## Cos'è un file CPG?

Un file CPG è un file di progetto creato dal software di editor di pagine Web, Cool Page. Il software è diventato obsoleto molti anni fa e non viene più utilizzato. Il file CPG conteneva dettagli sugli oggetti utilizzati nella pagina Web e sulle proprietà di questi oggetti. I file di progetto CPG possono essere aperti nel software Cool Page e, dopo la finalizzazione, la pagina web può essere esportata come pagina web [HTML](/it/web/html/).

## Formato file CPG

I dettagli del formato file interno del formato file CPG non sono noti. Questi sono stati salvati come file binari e verranno aperti solo nel software Cool Page.
