## Che cos'è un file WUX?

Un file WUX è la versione compressa di un file Wii U Disk Image [WUD](/it/disc-and-media/wud/). Viene generato utilizzando lo strumento di compressione delle immagini Wii U per ridurre la dimensione del file dei file segmentati WUD che altrimenti hanno una dimensione di circa 20 GB. Questi file di informazioni di gioco di dimensioni molto grandi sono ridotti a circa 2 GB in file WUX compressi, il che semplifica il trasferimento dei file. I file WUX possono essere aperti e riprodotti con emulatori di videogiochi come [Cemu](https://cemu.info/).

## Formato file WUX

I file WUX vengono salvati come file compressi che possono essere decompressi per essere caricati negli emulatori utilizzando lo strumento di compressione delle immagini Wii U. L'emulatore di videogiochi Cemu può anche caricare i file WUX e caricare le informazioni complete sul videogioco.
