## Che cos'è un file SITX?

Un file con estensione .sitx è un file di archivio compresso creato con il software StuffIt. Questo formato è stato introdotto come estensione del formato di file Stuffit (.sit) standard originale e offre una compressione relativamente maggiore rispetto al suo predecessore. I file SITX sono più compatti e occupano meno spazio insieme al supporto per la compressione a 512 bit finalizzata alla sicurezza. I file SITX sono riconosciuti dai loro tipi di media Internet come application/x-stuffitx e application/x-sitx con Uniform Type Identifier come com.stuffit.archive.sitx.

## Formato file SITX - Ulteriori informazioni

Gli STIX vengono salvati in formato file binario e si sono evoluti dopo l'utilizzo del formato file SIT. È stato introdotto da Aladdin Systems nel settembre 2002 per StuffIt Deluxe 7.0 per Macintosh. Questo nuovo formato di file è stato progettato tenendo presenti i seguenti obiettivi:

* Estensibile
* Supporto per più metodi di compressione
* Supporto per nomi di file lunghi
* Supporta gli attributi di file Unix e Windows

StuffIt X aggiornato ha migliorato il formato StuffIt originale incorporando più algoritmi di compressione come PPM e compressione di tipo da BWT a LZW. Il nuovo formato di file era inoltre dotato di un'opzione "modalità blocco" per la protezione contro la perdita di dati correggendo gli errori delle opzioni di "ridondanza".
