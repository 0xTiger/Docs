## Che cos'è un file XAPK?

Un file con estensione .xapk è un file di pacchetto compresso utilizzato per installare app Android su dispositivi mobili. È un formato di file contenitore che incorpora APK e file aggiuntivi associati necessari per l'installazione. L'altro file associato è un file OBB che archivia file aggiuntivi come grafica, file multimediali e dati dell'app necessari per mantenere l'applicazione in esecuzione. I file XAPK non sono supportati da Google Play e vengono utilizzati solo per la distribuzione su siti Web di download di app Android di terze parti. Questi possono essere installati su un dispositivo Android utilizzando XAPK Installer.

## Formato file XAPK

I file XAPK sono compresi nel formato file standard [ZIP](/it/compression/zip/). Questi possono essere estratti utilizzando un software di compressione/decompressione standard come WinZIP. Una volta che il file XAPK è stato estratto sul disco, contiene i seguenti file nella cartella.

* APK - File di installazione standard per l'installazione dell'applicazione su dispositivi Android
* OBB - File aggiuntivo che contiene file di risorse rilevanti
