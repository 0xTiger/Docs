## Che cos'è un file PAGES?

Un file PAGES è un documento Word creato con l'applicazione Apple Pages che è un elaboratore di testi sviluppato da Apple Inc. Il documento salvato come file PAGES può contenere una varietà di diversi tipi di contenuto come report, poster, brochure, newsletter e libri. Apple PAGES fornisce anche una serie di modelli diversi per creare questi tipi di documenti. Il contenuto di un file PAGES può contenere testo, immagini, grafici, tabelle e grafici. Apple PAGES funziona su sistemi operativi macOS, iPadOS e iOS.

## PAGINE Formato file

I file PAGES vengono salvati come file [ZIP](/it/compression/zip/) che possono essere rinominati con estensione ZIP e aperti con un'utilità di archiviazione per visualizzarne il contenuto. Include un file JPG in bundle nell'archivio che può essere utilizzato per visualizzare in anteprima la prima pagina del documento. Contiene anche un file PDF che viene utilizzato per visualizzare l'intero documento.

## Come convertire i documenti PAGES in WORD

Per convertire il file di Apple Pages in PDF, è possibile utilizzare i seguenti passaggi.

1. Apri l'app Pagine
1. Apri il documento che deve essere convertito in un documento WORD
1. Utilizzare l'opzione Esporta e selezionare il formato Word
1. Salva il documento in formato Word

## Come convertire il file PAGES

I documenti Apple PAGES possono essere convertiti in diversi formati di file. Questo può essere fatto dal software Apple Pages utilizzando l'opzione **Esporta in**. I file PAGES possono essere convertiti in:

* Microsoft Word [DOC](/it/elaborazione testi/doc/) e [DOCX](/it/elaborazione testi/docx/)
* [PDF](/it/pdf/)
* [EPUB](/it/ebook/epub/)
* [RTF](/it/elaborazione testi/rtf/)
* [TXT](/it/elaborazione testi/txt/)
