## Che cos'è un file NPK?

Un file NPK è un file del pacchetto di aggiornamento software utilizzato dai router **MikroTik** per l'aggiornamento del sistema operativo del router. Si presenta come un archivio binario compresso che viene caricato nel router per l'installazione degli aggiornamenti software. Le informazioni contenute in un file NPK includono informazioni di rete come indirizzo IP e servizio IP, informazioni sui connettori (interfacce Ethernet e porta seriale), configurazione e-mail, configurazione bridge e gestione utente locale.

## Formato file NPK

I file NPK vengono archiviati come archivio binario compresso. È un formato di file chiuso utilizzato solo da MikroTik stessi. Non è destinato a creare componenti aggiuntivi RouterOS tramite terze parti. I file NPK sono gestiti da MikroTik stesso e le versioni aggiornate degli stessi possono essere scaricate dalle sezioni di download del sito Web [MikroTik](https://mikrotik.com/download).

Quando un file NPK viene caricato su un router, il sistema operativo del router non entra in vigore a meno che non venga riavviato. Pertanto, è necessario un riavvio per aggiornare il sistema operativo del router.
