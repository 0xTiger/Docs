## Che cos'è un file SIT?

Un file SIT è un file di archivio compresso creato con il software Micro StuffIt. Questo è il formato di file standard originale introdotto da StuffIt. Il formato file SIT è comunemente usato sui sistemi Macintosh, ma ora è supportato sia su piattaforme Windows che Macintosh. StuffIt ha esteso il formato di file SIT a [SITX](/it/compression/sitx/) che offre una compressione maggiore rispetto ai file SIT. I file SIT possono essere aperti con il software Smith Micro StuffIt Deluxe Mac 16.

## Formato file SIT - Ulteriori informazioni

I file SIT vengono salvati in formato file binario e i dettagli relativi al formato file interno non sono disponibili pubblicamente. È stato introdotto da Aladdin Systems ed è stato introdotto per comprimere i file negli archivi.
