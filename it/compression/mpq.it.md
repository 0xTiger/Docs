## Che cos'è un file MPQ? ##

Un file con estensione .mpq appartiene principalmente a Blizzard Game di Blizzard Entertainment. Le specifiche principali di qualsiasi file MPQ sono grafica compressa, suoni, dati di livello, ecc. Se parliamo di archivi MPQ, questi sono stati prima familiarizzati con il gioco Diablo e ora molto popolari in numerosi giochi. Il file MPQ è un grande contenitore di sorgenti binari per mappe di livello e modelli di oggetti, musica più video, ecc.

## Formato file MPQ ##

Il formato di file MPQ è stato sviluppato per la prima volta da Mike O'Brien per la memorizzazione dei dati di gioco. Da Diablo, MPQ è stato utilizzato in molti giochi pubblicati da Blizzard Entertainment come Diablo II, StarCraft, Warcraft II e III, World of WarCraft, StarCraft II.

## Algoritmi di compressione supportati ##

* PKZIP
* Huffman
* zlib
* bzip2
* LZMA
