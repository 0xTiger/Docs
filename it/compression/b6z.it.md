## Che cos'è un file B6Z?

Un file con estensione .b6z è un file di archivio compresso creato dall'applicazione software macOS [B6Zip](http://b6zip.com) che viene utilizzato per comprimere e decomprimere i file. È un formato di archivio relativamente nuovo che consente un rapporto di compressione più elevato. B6Z ha un'architettura aperta e supporta file di dimensioni fino a 900000 PB. Supporta la compressione dei dati, il ripristino degli errori e lo spanning dei file. Il software di utilità B6Zip è disponibile gratuitamente su MacOS per aprire diversi tipi di file compressi, incluso B6Z.

## Formato file B6Z

Il formato di file B6Z si basa su un'architettura aperta e fornisce una solida compressione con l'algoritmo di crittografia AES-256. Utilizza LZMA2 come metodo di compressione predefinito, ma supporta anche altri metodi di compressione come segue.

|Metodo|Descrizione|
---|---|
|LZMA |Versione ottimizzata dell'algoritmo LZ77|
|LZMA2| Versione migliorata di LZMA|
|Sgonfia| Algoritmo LZ77 regolare|
|BZip2| Algoritmo BWT standard|
|PPMD| PPMdH di Dmitry Shkarin|

L'utilità B6Zip supporta tutti questi standard di compressione ed è altamente ottimizzata con conseguente alta velocità. Supporta il lavoro con i formati di file [ZIP](/it/compression/zip/), [TAR](/it/compression/tar/) e [RAR](/it/compression/rar/). I file B6Z hanno applicazione di tipo MIME/x-b6z-compresso.
