## Che cos'è un file LQR? ##

Il tipo di file LQR è principalmente correlato a un file di archivio compresso che copre uno o più file che sono stati ridotti in termini di dimensioni del file per una facile distribuzione tramite Internet o tramite supporti trasferibili come dischi ottici come DVD e CD tra l'altro spazio di archiviazione esterno dei dati dispositivi. Qualsiasi file archiviato in formato file LQR può essere combinato con disposizioni di compressione eseguite da un determinato algoritmo di compressione. I file nel formato LQR sono bloccati con l'estensione .lqr. Ora, l'estensione LQR è stata sostituita da standard di compressione e funzionalità di codifica più ampiamente utilizzati, inclusi i formati [RAR](/it/compression/rar/) e [ZIP](/it/compression/zip/).

## Breve storia ##

All'inizio degli anni '80, il formato di file [LBR](/it/compression/lbr/) era un formato di file di archivio utilizzato sui file LBR dei sistemi operativi CP/M e MS-DOS, file tipicamente confinati gestiti da SQ, oppure l'archivio stesso era compresso con SQ. A un file di archivio LBR compresso è stata assegnata l'estensione LQR o LZR.
