## Che cos'è un file LZH? ##

Un file con estensione .lzh e .lha di solito si riferisce al formato del file di compressione dell'archivio. Questo formato di file è lo stesso di altri formati di compressione di file come [ZIP](/it/compression/zip/), [RAR](/it/compression/rar/), ecc. Lo scopo principale di questi formati di file è ridurre la dimensione del formato per inviarli facilmente e per tenerli insieme in forma compressa. AS confronta con altre società occidentali i file LZH sono molto popolari in Giappone e solitamente utilizzati per comprimere i dati dei videogiochi. Il componente aggiuntivo LZH per Windows 7 è integrato nella versione giapponese di Windows 7. Microsoft ha rilasciato per la prima volta il componente aggiuntivo cartella Microsoft Compressed (LZH) per la versione giapponese di Windows XP. Questo formato di file è implementato con disposizioni e funzionalità di compressione utilizzate dall'algoritmo Lempel-Ziv e Haruyasu

## Specifica LZH ##

Il metodo di compressione di un archivio LZH viene mostrato come una stringa di testo di cinque byte, come -lz1-. Questi sono dal terzo al settimo byte nel file compresso.

|Specifica|Descrizione|
---|---|
|Estensione | .lzh, .lha|
|Tipo di supporto| applicazione/x-lzh-compresso|
|Tipo codice| "LHA_" (LHA-SPACE)|
|UTI| public.archive.lha|
|Sviluppato da| Haruyasu Yoshizaki (Yoshi)|
|Tipo| Compressione|
|Prorogato da| LArc|

## Problemi per aprire un file LZH ##

Di seguito sono riportati i pochi problemi che potrebbero causare un cattivo funzionamento del formato di file LZH:
  

* Il file potrebbe essere danneggiato. Per risolvere questo problema, scarica nuovamente il file o chiedi al mittente di inviare nuovamente il file
* Il secondo motivo è il file infetto, in questo caso scansiona il file correttamente
* Collegamenti impropri al file LZH
* Rimozione della descrizione della LZH
* Risorse hardware insufficienti
* Driver obsoleti dell'attrezzatura
