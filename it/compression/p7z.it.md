## Cos'è un file P7Z?

Un file P7Z è un file compresso che contiene un messaggio di posta elettronica nello standard di crittografia S/MIME. È stato introdotto da Microsoft Corporation ed è classificato come file di dati relativo alla posta elettronica. I file P7Z hanno il tipo di supporto application/pkcs7-mime. Contiene una o più parti di un messaggio compresso con l'algoritmo di compressione zlib.

## Formato file P7Z: ulteriori informazioni

I dettagli del formato file interno del formato file P7Z non sono noti. Alcune API come [Aspose.Email](https://products.aspose.com/email/) ti consentono di leggere la parte S/MIME dei messaggi. Se un client di posta elettronica non ha la capacità di gestire i dati compressi, i messaggi S/MIME compressi arrivano come allegati di posta elettronica.
