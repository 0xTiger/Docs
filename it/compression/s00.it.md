## Che cos'è un file S00?

Un file con estensione .s00 è la prima parte di un file compresso che viene diviso utilizzando il software ZipSplitter. La suddivisione dei file in file così piccoli riduce la dimensione complessiva dell'archivio compresso e ne semplifica il trasporto su Internet. Le parti rimanenti dell'archivio diviso sono denominate con estensioni .S01, .S02 e così via. Quando si suppone che il file venga estratto, si fa doppio clic sul file .EXE incluso nel file diviso. L'EXE cerca il file .S00 e utilizza gli altri file accompagnati per l'estrazione dell'intero archivio.

## Formato file S00

I file S00 vengono salvati su disco come archivio compresso binario. Tuttavia, i dettagli del formato file interno del formato file S00 non sono disponibili pubblicamente. I file di grandi dimensioni come .[WMV](/it/video/wmv/) o [.MP4](/it/video/mp4/) vengono spesso divisi utilizzando il software ZipSplitter per ridurne le dimensioni.
