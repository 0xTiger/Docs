## Che cos'è un file ALZ? ##

Un file con estensione .alz è un file compresso, principalmente correlato ad ALZip di ESTsoft, Corp. Il formato di file ALZ come file compresso è anche definito come archivi ALZip. Gli sviluppatori hanno sviluppato lo strumento di compressione dei file chiamato ALZip e la stessa tecnologia di codifica e compressione è utilizzata dal formato di file ALZ. Questo software è stato creato da ESTsoft per la distribuzione privata in versioni ampiamente utilizzate di Microsoft Windows. Con il tempo, lo strumento di compressione e decompressione di file ALZip ha esteso la sua fornitura a più di 36 diversi record di moduli e programmi di compressione e sostentamento per l'algoritmo ZIP.

## Breve storia ##

Il formato di file ALZ è stato rilasciato nel 1999. ALZip e il formato di file ALZ generalmente non impongono alcun limite alla dimensione del file e al limite di espansione. Questo software consente al sistema Windows di dettare i limiti di dimensione per i suoi file di output.
