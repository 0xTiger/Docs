## Che cos'è un file DMG?

Un file con estensione .dmg è un file immagine disco di Apple ed è noto come file immagine disco di Mac OS X. Il formato file DMG è il file immagine disco più utilizzato destinato ad allocare applicazioni e file tra i PC Apple. È molto comodo montare i file DMG in un'unità virtuale o utilizzare il software Apple Disk Utility per aprire e visualizzare il contenuto del formato di file DMG. La vecchia versione dei file IMG utilizzati in Mac OS Classic è sostituita dal formato file DMG. I file DMG vengono generati per utenti Mac e non sono supportati dal sistema operativo Windows, ma gli utenti possono accedere al contenuto DMG nel sistema operativo Windows utilizzando strumenti di compressione e decompressione come 7-Zip, Pea Zip, ecc. L'estensione del file DMG viene utilizzata anche da lo strumento Oracle Export and Ingress per i file scrapheap mantenuti nel formato binario del database Oracle.

## Breve storia

L'estensione del file DMG è stata sviluppata da Apple ed è utilizzata principalmente sui computer Macintosh. Principalmente il file DMG viene utilizzato per dare ai dispositivi Mac la capacità di installare diversi tipi di file e software avviati da Internet. Il file DMG è fondamentalmente un'immagine disco montabile che viene visualizzata sul desktop quando viene aperta. Il file comprende dati grezzi generalmente crittografati e compressi. I sistemi Mac trattano DMG nello stesso modo in cui tratterebbero un disco interfogliato e tentano istantaneamente di aprire o "tracciare" il file.

## Specifiche tecniche del formato file DMG

* Appositamente generato per computer Macintosh
* Sviluppato da Apple
* Il formato del file è binario
* La categoria di file è File immagine disco
* Il programma supportato è MacOS
* Può contenere una tabella delle partizioni
* Può contenere qualsiasi tipo di filesystem come HFS+
* Funzionalità per comprimere le immagini
