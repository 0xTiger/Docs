## Che cos'è un file PUP di Linux?

Un file PUP Linux è un pacchetto di installazione utilizzato dal sistema operativo Puppy Linux. Viene eseguito come programma di installazione e avvia l'installazione dei file del pacchetto. Su Linux, viene spesso indicato come file DotPup ed è simile ai file di installazione [.exe](/it/executable/exe/) e [.msi](/it/executable/msi/) utilizzati sul sistema operativo Windows. Le versioni più recenti di Puppy Linux utilizzano il formato di file .pet invece del formato di file PUP.
