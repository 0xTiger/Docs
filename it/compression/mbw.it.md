## Che cos'è un file MBW? ##

Un file con estensione .mbw è principalmente associato a MBRWizard di Firesage Solutions. Il Master Boot Record Wizard (MBRWizard) esegue il backup dell'MBR del computer e lo salva in un file MBW che può essere ripristinato se necessario. Oltre alle copie MBR e VBR (Volume Boot Record), questi file possono contenere GPT, settori definiti dall'utente o immagini di interi dischi. L'MBR è un record vitale di 512 byte sul disco rigido che consente l'avvio del sistema operativo. Può anche contenere la tabella delle partizioni e una firma a 32 bit che identifica il supporto del disco.

## Sistemi operativi supportati ##

Il formato file MPW è supportato solo dal sistema operativo Windows.

## Possibili problemi durante l'apertura del file ##

Se non riesci ad aprire ed eseguire il file MBW, potrebbe esserci uno dei seguenti possibili problemi:

* File MBW danneggiato
* Connessioni sbagliate del file MBWnelle iscrizioni del registro
* Eliminata la descrizione del MBW dal registro di Windows
* Un file infetto con un malware indesiderato
* Il computer non dispone di risorse hardware sufficienti per utilizzare il file MBW
* I driver utilizzati dal computer per aprire un file MBW sono obsoleti
