## Che cos'è un file PTK? ##

I file PTK sono considerati file di dati utilizzati da Intuit Quicken. È un'applicazione di gestione delle finanze personali che aiuta l'estensione del file .ptk per il corretto funzionamento. Ha diverse funzionalità come l'importazione dei dati direttamente dalle banche partecipanti, i rapporti su tutti i dati finanziari e il download giornaliero di azioni e prezzi dei fondi. Il formato di file PTK è il tipo precedente di file utilizzato da Quicken per salvare le informazioni finanziarie che ora sono sostituite con il formato GIF. Lo scopo principale dei file PTK era importare dati da risorse online, come banche e istituti finanziari, nel database Quickens.
