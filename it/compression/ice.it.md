## Che cos'è un file ICE?

Un file con estensione .ice è un archivio compresso creato con l'utilità software [ICEOWS](http://www.iceows.com/index.html) (Interface de Compression Ergonomique pour windOWS) precedentemente nota come ArjFolder. Fornisce un'eccellente compressione per ridurre le dimensioni dei file di testo. ICEOWS è stato sviluppato originariamente con il supporto della lingua polacca, ma ora è disponibile anche in altre lingue tra cui inglese, francese, tedesco, polacco, portoghese, spagnolo, italiano, cinese, ebraico, ucraino e svedese. Ti consente anche di estrarre il contenuto dei file di archivio e supporta ICE, ARJ, [ZIP](/it/compression/zip/), [GunZip](/it/compression/gz/), [TAR](/it/compression/tar/), Microsoft CAB , [RAR](/it/compression/rar/), ACE, MIME, Mac HQX, UUEncode, XXEncode, Base64, [JAR](/it/programming/jar/), EAR, WAR, LHA, IMP, [BZ2](/it/compression /bz2/) formati di file.

## Formato file ICE

ICE sono file binari ma non sono disponibili specifiche tecniche sui dettagli del suo formato di file interno.
