## Cos'è un file CIT?

Un file CIT è un file di progetto generato dall'applicazione di scrittura di rapporti investigativi basati su citazioni In8 Cite. Contiene i dettagli dell'incidente sotto forma di rapporto e dispone anche di un catalogo che contiene la documentazione di supporto e le prove citate nel rapporto. Gli investigatori criminali utilizzano i file CIT per fare riferimento a casi penali durante le loro indagini.

## Formato file CIT

I file CIT vengono salvati su disco in formato file compresso. Utilizza l'algoritmo di compressione ZIP standard per comprimere i file. Un file CIT e i relativi documenti di supporto possono essere convertiti ed esportati in un file [PDF](/pdf/) che può essere condiviso con altri per la revisione.

### Informazioni su In8

In8 è una società di software specializzata nello sviluppo di applicazioni utilizzate per documentare e segnalare rapporti di indagini penali.
