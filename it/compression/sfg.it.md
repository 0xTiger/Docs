## Cos'è un file SFG?

Un file SFG è il formato file utilizzato da Synfig Studio per salvare i file di progetto. Synfig Studio è un software di animazione 2D gratuito e open source che consente agli utenti di creare animazioni di alta qualità utilizzando grafica bitmap e vettoriale. Il file SFG contiene tutte le informazioni necessarie per ricreare un'animazione, inclusi livelli, tempi e fotogrammi chiave per ciascun livello. Include anche eventuali immagini o risorse utilizzate nel progetto, nonché eventuali tracce audio o altri media.

## Formato file SFG

I file SFG vengono salvati su disco in formato file ZIP compresso. Altri formati di file supportati da [Synfig Studio](https://www.synfig.org/) Compressed Project includono [SIFZ](/compression/sifz/) e SIF.
