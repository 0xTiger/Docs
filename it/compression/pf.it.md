## Che cos'è un file PF?

Un file PF è un file compresso e crittografato creato con il software Stuffit Deluxe. È un file protetto da password e può essere decrittografato e decompresso solo applicando la password. Stuffit Deluxe è un noto compressore ed espansore di file per Mac OS e può gestire più di 30 formati di file inclusi [RAR](/it/compression/rar/), [7zip](/it/compression/7z/) e altri formati di file popolari. I file PF possono essere aperti con il software Stuffit Deluxe.

## Formato file PF

I file PF vengono salvati su disco come file binari e vengono salvati nel formato di file proprietario Stuffit Deluxe.
