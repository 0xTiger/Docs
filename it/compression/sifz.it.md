## Che cos'è un file SIFZ?

Un file SIFZ è un file SIF compresso con gzip creato dall'applicazione di creazione di animazioni 2D open source **Synfig Studio**. Contiene più elementi grafici che costituiscono l'animazione. Il contenuto generale dell'animazione viene creato utilizzando una combinazione di un'area di disegno riempita con forme, pennellate o tratti di matita e testo. Tutti questi sono disposti nelle proprie posizioni, maniglie di raggio, tangente, angolo, vertice e larghezza. I file SIFZ possono essere aperti con [Synfig Studio](https://www.synfig.org/).

## Formato file SIFZ

I file SIFZ sono file compressi [ZIP](/it/compression/zip/) che vengono compressi utilizzando il metodo di compressione gzip. Synfig viene utilizzato per creare animazioni di qualità cinematografica utilizzando una grafica vettoriale e bitmap. A differenza del vecchio metodo di creazione di animazioni fotogramma per fotogramma, consente di produrre animazioni 2D di qualità superiore con meno risorse.

I file SIFZ, essendo compressi, sono più piccoli dei file SIF non compressi. Ciò semplifica anche il trasferimento su connessioni Internet a bassa larghezza di banda.
