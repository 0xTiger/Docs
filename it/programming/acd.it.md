## Che cos'è un file ACD?

Un file ACD è un file di progetto che contiene informazioni per la programmazione delle piattaforme Logix [Rockwell Automation](https://www.rockwellautomation.com). Include istruzioni per titolari e responsabili del trattamento. Viene utilizzato per le piattaforme Logix come i sistemi ControlLogix, CompactLogix, FlexLogix, SoftLogix5800 e DriveLogix.

Le applicazioni che possono aprire file ACD includono Rockwell Automation RSLogix 5000.
