## Che cos'è un file APA?

Un file con estensione .apa è un file di progetto di sviluppo software creato dall'applicazione RSView. Contiene le informazioni necessarie per la creazione di applicazioni HMI (Human Machine Interface). I file APA sono archiviati in formato file di archivio. Il suo formato di file di archivio aiuta a trasferirli da un computer all'altro facilmente a causa delle dimensioni del file più piccole. Le utilità di decompressione standard possono essere utilizzate per estrarre il contenuto dei file APA. I file APA possono essere aperti utilizzando il software Rockwell Automation RSView Studio o RSView ME.

## Formato file APA - Ulteriori informazioni

Il gestore dell'applicazione RSView salva i file .apa per impostazione predefinita in C:\Documents and Settings\All Users\Documents\RSView Enterprise\ME\Archives su Windows XP. Tuttavia, puoi salvare il file in qualsiasi directory secondo i requisiti.
