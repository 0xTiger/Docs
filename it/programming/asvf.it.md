## Cos'è un file ASVF?

Un file ASVF è un file compresso creato dal software Asphyre Sphinx che è un framework 2D/3D per applicazioni industriali e aziendali. Memorizza dati come caratteri, immagini e altri media. Sphinx può essere utilizzato anche per sviluppare videogiochi 2D/3D. Un file ASVF compresso può contenere file comuni come file .JPEG e .XML.

I file ASVF possono essere aperti con il software Asphyre Sphinx che è stato interrotto da molto tempo.

## Formato file ASVF

I file ASVF vengono salvati come file compressi e possono avere più file all'interno dell'archivio.
