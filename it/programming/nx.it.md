## Cos'è un file NX?

Un file NX è un file di videogioco creato per essere utilizzato con la console di gioco virtuale, **LowRes NX**. È scritto nel linguaggio di programmazione BASIC e contiene il codice sorgente del gioco che ne definisce il funzionamento. Puoi creare i tuoi file NX da utilizzare con l'app che può essere caricata e gli utenti possono giocare.

Puoi aprire un file NX in un editor di testo, LowRes NX o qualsiasi editor di codice sorgente come Microsoft Visual Studio Code.

## Formato file NX

Un file NX viene salvato su disco come file di testo normale che contiene le istruzioni del gioco. Ogni file NX ha il codice sorgente del gioco che comprende più voci ROM della cartuccia. Queste voci ROM contengono informazioni sui componenti del gioco come grafica, mappe di livello, audio e altri dati sul codice non sorgente del gioco. Le voci ROM di un programma possono essere aggiunte o modificate utilizzando i programmi di editing NX forniti in bundle con LowRes NX.

## Come creare file NX?

### Crea file NX sul PC

Puoi creare file NX su PC utilizzando:

* Editor di testo, come Microsoft Notepad, Notepad++ o Github Atom
* Editor del codice sorgente, come Microsoft VIsual Studio Code

### Crea file NX su iOS

LowRes NX ha una versione iOS che può essere utilizzata dagli sviluppatori per creare file NX dall'interno di questa app.
