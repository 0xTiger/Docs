## Cos'è un file ASC?

Un file ASC è un file **script** creato nel linguaggio di programmazione ActionScript. Contiene il codice eseguito sul software di hosting Flash Media Server lato server. Il file ASC contiene script utilizzati per controllare gli eventi dell'applicazione come il caricamento di applicazioni Flash, la gestione delle azioni dell'utente da eventi del mouse e della tastiera, la riproduzione di contenuti multimediali e il rendering dell'output dell'applicazione. Adobe ha acquisito Macromedia Flash che in seguito è diventato parte della tuta Adobe Creativity come Adobe Flash. Utilizzato per salvare i file di animazione in [formato file .swf](/it/page-description-language/swf/).

## Formato file ASC: ulteriori informazioni

I file ASC vengono salvati come file di testo che possono essere aperti e modificati con [Adobe Media Encoder](https://www.adobe.com/products/media-encoder.html). Questi vengono creati e salvati come file di testo e possono essere aperti con qualsiasi editor di testo come Microsoft Notepad, Notepad++ e Apple TextEdit. Il file ASC viene eseguito per impostazione predefinita quando viene eseguito Flash Media Server e l'applicazione.
