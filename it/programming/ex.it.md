## Che cos'è un file EX?

Un file EX è un file di codice sorgente che contiene il codice sorgente per le applicazioni sviluppate utilizzando il linguaggio di programmazione Euphoria. Euphoria è un linguaggio di programmazione procedurale che crede nella semplicità della programmazione piuttosto che nei linguaggi di programmazione complessi. Un'applicazione sviluppata in Euphoria può comprendere più file EX, ma un singolo file EX può anche contenere l'origine completa del programma.

Le applicazioni che possono **aprire file EX** includono openEuphoria Euphoria, Microsoft Notepad, Notepad++ e Atom.

## Breve storia del formato file EX

Lo sviluppo e l'esistenza del formato di file EX è legato allo sviluppo del linguaggio di programmazione Euphoria, creato da Robert Craig. È stato rilasciato per la prima volta per MS-DOS. È diventato un software open-source nel 2006 con il rilascio della versione 3. Il suo sviluppo è stato rilevato dal **gruppo openEuphoria** che amministra e sviluppa il progetto. La sua versione 4 è stata rilasciata a dicembre 2010 ed è disponibile per Windows, Linux, macOS e tre versioni di BSD.

L'Euphoria è stato inizialmente scritto nel linguaggio di programmazione **[C](/it/programming/c/)**. Successivamente, l'interprete Euphoria è stato diviso in due parti:

* `Front-end Parser` - scritto in Euphoria e viene utilizzato con il traduttore Euphoria-to-C e il Raccoglitore.
* `Back-end` - libreria runtime scritta nel linguaggio di programmazione C.

## Formato file EX - Ulteriori informazioni

I file EX vengono archiviati come file di testo semplice e contengono il codice sorgente per i programmi scritti nel linguaggio di programmazione Euphoria. Questi file possono essere aperti in qualsiasi editor di testo.
