## Che cos'è un file CXX?

Un file CXX è un file di codice sorgente, simile a C++, per la scrittura di applicazioni. Può contenere l'implementazione completa di un'intera applicazione o può avere un'implementazione parziale del programma che può essere incluso in un progetto con altri file C++. I file CXX possono essere aperti in qualsiasi editor di testo e IDE di programmazione popolari come Notepad++, Turbo C++ e Visual Studio,

## Formato file CXX

I file CXX vengono archiviati come file di testo normale e contengono codice scritto nella sintassi C++. I file CXX si trovano comunemente anche con estensione [.cpp](/it/programming/cpp/). I file CXX sono considerati un'alternativa al .cpp utilizzato da alcuni compilatori.
