## Che cos'è un file MM?

Un file MM è un file di codice sorgente che può contenere codice di programmazione sia Objective-C che Objective-C++. Viene utilizzato su MacOS e si differenzia dai file .M che contengono solo codice Objective-C. Quando il compilatore vede l'estensione .mm, è in grado di compilare le classi C++ presenti nel file. Poiché un singolo file di codice sorgente può contenere una combinazione di entrambi i linguaggi [C](/it/programming/c/) e [C++](/it/programming/cpp/), rinominare i file .m con estensione .mm consente di impostare le funzionalità C++ in. Le applicazioni che possono aprire file .MM includono Apple TextEdit, Microsoft Notepad, Notepad++ e Microsoft Wordpad.
