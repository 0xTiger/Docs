## Cos'è un file GMD?

Un file GMD è un file sorgente di sviluppo del gioco generato con lo strumento di sviluppo software Game Maker utilizzato per creare giochi 2D. Il file GMD viene archiviato come file di testo ed è un file di progetto per il gioco da sviluppare. Game Maker è disponibile per Windows, Mac, Linux, Android, iOS, HTML5, Xbox, PlayStation e Nintendo Switch. Lo strumento è ampiamente utilizzato da sviluppatori indipendenti, studi professionali ed educatori in tutto il mondo per la creazione di giochi.

## Formato file GMD

I file GMD vengono salvati in formato file binario e i dettagli sul formato file interno non sono disponibili come riferimento per lo sviluppatore.
