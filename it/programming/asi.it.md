## Cos'è un file ASI? ##

Un file ASI è un formato di file variabile utilizzato dal software Alpha Five utilizzato per lo sviluppo di applicazioni Web. Memorizza le variabili utilizzate dall'applicazione Web. Quando crei un nuovo progetto per sviluppatori in Alpha Five, anche il file ASI viene creato e salvato insieme al progetto per sviluppatori Alpha Five.

I file ASI possono essere aperti con Alpha Software Alpha Anywhere.

## Formato file ASI

I file ASI vengono salvati nel formato file proprietario Alpha Five. Vengono letti e scritti dall'applicazione come parte del progetto dello sviluppatore.
