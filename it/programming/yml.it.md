## Che cos'è un file YML?

YML è un file di testo nel linguaggio di serializzazione dei dati, [YAML](/it/programming/yaml/) (Yet Another Markup Language), che viene spesso utilizzato per creare file di configurazione. Può funzionare in combinazione con qualsiasi linguaggio di programmazione inclusi C/C++, Ruby, Python, Java, PHP, C# e altri. Essendo indipendente da qualsiasi linguaggio di programmazione specifico, è aperto, interoperabile e leggibile dall'uomo. I file YML possono essere aperti e modificati con editor di testo comuni come Microsoft Notepad, Notepad++, Github Atom, Apple TextEdit e Visual Studio IDE.

## Formato file YML - Ulteriori informazioni

I file YML memorizzano le impostazioni dell'applicazione come configurazione. La sua struttura di base è una mappa che può anche essere chiamata directory o hash. Le voci in un file YML sono costituite da una coppia chiave e valore. Per lo più, i file di configurazione .yml vengono salvati come `configure.yml`, sebbene alcuni linguaggi di programmazione come Ruby on Rails salvino la configurazione della connettività del database nei file `database.yml`. [Specifiche del formato file YML/YAML](https://yaml.org/spec/1.2.2/) spiega le strutture linguistiche complete e le informazioni relative alla sintassi.
