## Che cos'è un file ALB?

Un file ALB è un file di libreria utilizzato dall'applicazione IDE **Alpha Five** utilizzata per lo sviluppo di applicazioni Web. Il suo scopo principale è memorizzare le informazioni per il corrispondente Alpha Five [file ADB](/it/database/adb/). Ciò include informazioni sul dizionario dei dati come impostazioni, puntatori di script, menu, funzioni definite dall'utente, bitmap, barre degli strumenti e definizioni di importazione.

I file ALB possono essere aperti con **Alpha Software Alpha Anywhere**.

## Formato file ALB - Ulteriori informazioni

I file ALB vengono salvati nel formato di file proprietario di Alpha File e la loro struttura interna del formato di file non è disponibile pubblicamente. Questi vengono salvati in aggiunta al database generale della libreria degli sviluppatori Alpha Five. Essendo librerie, i file ALB possono essere riutilizzati in altre applicazioni Alpha Five.
