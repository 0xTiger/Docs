## Che cos'è un file AIA?

Un file AIA è un file di progetto creato dall'app MIT Inventor che insegna ai principianti a creare applicazioni Android e iOS. Viene salvato come file compresso [ZIP](/it/compression/zip/) e può contenere molte cartelle e file. Questi includono file sorgente come BKY, SCM e YAIL che fanno parte del progetto dell'app. MIT App Inventor fornisce un emulatore Android che funziona allo stesso modo di un Android ma appare sullo schermo. Questo aiuta a testare le app sugli emulatori.

I programmi che possono aprire file AIA includono [MIT App Inventor](https://appinventor.mit.edu/).

## Formato file AIA - Ulteriori informazioni

I file AIA vengono salvati su disco in formato ZIP compresso. Per visualizzare il contenuto di questo archivio ZIP, rinomina semplicemente l'estensione del file in .zip e sarai in grado di visualizzare il contenuto di questo archivio.

### Esercitazione sull'app del MIT

App Inventor offre diversi tutorial per principianti per sviluppare app Android e iOS. Consente di trascinare e rilasciare elementi dell'interfaccia utente come pulsanti, immagini, elenchi, caselle di testo e altri per creare un'applicazione mobile e programmarne le funzionalità. Il progetto App Inventor .aia viene salvato online ma può anche essere scaricato per l'utilizzo su computer.
