## Che cos'è un file RBXL?

Un file RBXL è un file di posizione Roblox che contiene la posizione definita nel mondo 3D all'interno del [gioco Roblox](https://www.roblox.com/). Roblox stesso è un gioco di costruzione multiplayer che consente ai giocatori di creare giochi utilizzando gli strumenti di sviluppo forniti dalla piattaforma. I file RBXL utilizzano strumenti, script, squadre e parti per descrivere il campo di gioco come modelli 3D. Roblox Studio consente al giocatore di creare gli oggetti di gioco e può essere utilizzato per aprire file RBXL.

## Formato file RBXL

I file RBXL vengono sviluppati utilizzando Roblox Studio e salvati nel formato di file [XML](/it/web/xml/). Grazie a questo formato di file universale, ci sono molti strumenti e API che ti consentono di creare e aprire questi file. Una di queste API è [MaximumADHD](https://github.com/MaximumADHD/Roblox-File-Format), una libreria [C#](/it/programming/cs/) per creare e manipolare file nel formato di file modello/posto di Roblox .
