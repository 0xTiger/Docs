## Che cos'è un file PHP?
Un file con estensione .php si riferisce al linguaggio di programmazione open source, utilizzato per scrivere script lato server, da eseguire su un server web. È il linguaggio di scripting Web più utilizzato, generalmente utilizzato per lo sviluppo di applicazioni Web su larga scala. Il più grande sistema di blogging sul web cioè WordPress e il più grande social network Facebook si basano sul linguaggio PHP. È compatibile con varie piattaforme come Windows, Linux, Unix, Mac OS X, ecc. Inoltre funziona con successo su quasi tutti i server utilizzati oggi, ad esempio Apache, IIS, ecc. PHP supporta un'ampia gamma di database, incluso MySQL.

## Formato file PHP ##

Un singolo PHP può contenere testo, CSS, [HTML](/it/web/html/), JavaScript e codice PHP. Il codice PHP viene eseguito sul server e il risultato viene restituito al browser come semplice HTML. I file PHP possono essere aperti con qualsiasi editor di testo e modificati sul posto, anche se applicazioni come Adobe Dreamweaver e gli strumenti di sviluppo PHP di Eclipse offrono un modo conveniente per scrivere e modificare il codice PHP.

### Sintassi ###

Uno script PHP viene eseguito sul server e il semplice risultato HTML viene rispedito al browser. Uno script PHP inizia con<?php and ends with ?>

### Esempio PHP ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
