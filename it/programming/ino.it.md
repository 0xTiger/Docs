## Cos'è un file INO?

**File INO** è correlato ad **Arduino Sketch** che è un programma scritto per schede Arduino utilizzando il **linguaggio di programmazione Arduino**. Il linguaggio di programmazione Arduino è basato sul Wiring; ed è simile a C/C++. Gli schizzi di Arduino vengono salvati con estensione file .ino e vengono utilizzati per controllare i circuiti stampati Arduino.

## Formato file INO: ulteriori informazioni

La piattaforma Arduino, progettata per la prototipazione elettronica, consente agli sviluppatori di creare e distribuire programmi, denominati schizzi, su circuiti stampati specializzati; questi schizzi spaziano da compiti fondamentali come l'illuminazione di un LED su un circuito a operazioni complesse come la trasmissione di dati su Internet; la creazione degli schizzi di Arduino avviene all'interno dell'**Arduino Integrated Development Environment (IDE)** e questi schizzi vengono archiviati con l'estensione del file .ino.

Nell'era precedente Arduino 1.0, gli schizzi venivano salvati con un'estensione di file .PDE dall'IDE di Arduino. Sebbene l'IDE supporti ancora l'apertura di file .PDE, la pratica consigliata è quella di salvare tutti gli schizzi, inclusi quelli originariamente archiviati come file [.PDE](/programming/pde/), con l'estensione file .ino. Questo spostamento riflette l'evoluzione della piattaforma e garantisce la compatibilità con gli standard attuali.

## A proposito di Arduino

Arduino è una piattaforma elettronica open source che semplifica la prototipazione e lo sviluppo di progetti interattivi; comprendendo sia elementi hardware che software, Arduino si rivolge a una base di utenti diversificata, dai principianti ai professionisti. Il cuore della piattaforma sono le schede Arduino, dotate di pin digitali e analogici, connettività USB e vari modelli basati su processori come Atmel AVR o ARM.

L'IDE Arduino funge da interfaccia user-friendly per codificare, compilare e caricare programmi (schizzi) sulle schede; il linguaggio di programmazione, simile a C/C++, è progettato per l'accessibilità, astraendo le complessità per i principianti. Le librerie Arduino forniscono frammenti di codice precompilati per attività che vanno dall'integrazione dei sensori ai protocolli di comunicazione, facilitando uno sviluppo efficiente.

Shield Arduino, schede aggiuntive, espandono le funzionalità, offrendo funzionalità come la connettività Ethernet o il controllo del motore. La vivace comunità Arduino contribuisce a un'ampia base di conoscenze attraverso forum, tutorial e progetti collaborativi; con applicazioni che spaziano dalla robotica, alla domotica, alle installazioni artistiche e ai sistemi basati su sensori, la versatilità e la semplicità di Arduino lo rendono la scelta preferita per diversi progetti elettronici.

## Come aprire il file INO?

**Arduino Integrated Development Environment** può essere utilizzato per aprire il file INO. Ma poiché si tratta di file di semplice testo, è possibile utilizzare qualsiasi editor di testo per aprirne o modificarne il contenuto.
