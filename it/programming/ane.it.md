## Che cos'è un file ANE?

Un file ANE è un file di estensione Adobe Air Native utilizzato dagli sviluppatori per scrivere funzioni specifiche che richiamano il codice [Adobe ActionScript](https://en.wikipedia.org/wiki/ActionScript). Nella maggior parte dei casi, i file ANE contengono la libreria di codice nativa utilizzata dall'estensione. Tuttavia, il file stesso è installato sul dispositivo. Consente funzioni specifiche del sistema come la vibrazione del dispositivo. Le applicazioni Adobe AIR devono essere compilate per utilizzare il collegamento dinamico per l'utilizzo del file ANE.

## Formato file ANE - Ulteriori informazioni

I file ANE vengono salvati su disco come file binari le cui specifiche del formato file non sono disponibili pubblicamente. Un file ANE può contenere codice sorgente Java Android nativo per accedere alle funzionalità del dispositivo Android come il richiamo del volume o le funzioni di vibrazione. Tutto ciò viene eseguito avvolgendo il sistema nativo nel linguaggio ActionScript di Adobe.
