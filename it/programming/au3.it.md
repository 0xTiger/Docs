## Cos'è un file AU3?

Un file AU3 è un file di script automatizzato creato con il programma di scripting freeware, [AutoIt v3](https://www.autoitscript.com/site/). Contiene il linguaggio di scripting AutoIt3 per automatizzare i comandi all'interno di Windows. Il linguaggio di programmazione AutoIt3 consente di simulare una serie di eventi che possono essere utilizzati per automatizzare le attività. Ad esempio, lo script all'interno del file AU3 può essere utilizzato per simulare la sequenza di tasti, il movimento del mouse, il controllo della posizione delle finestre e altri scopi di scripting.

Le applicazioni che possono aprire file AU3 includono [SciTE4AutoIt3](https://www.autoitscript.com/site/autoit-script-editor/downloads/) e AutoIt.

## Formato file AU3

I file AU3 vengono salvati come file di testo e contengono il linguaggio di scripting AutoIt3.
