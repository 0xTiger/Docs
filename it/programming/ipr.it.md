## Che cos'è un file IPR?

Il file IPR è un file di progetto creato con l'IDE Java **IntelliJ IDEA**. Tutte le informazioni relative alle impostazioni del progetto come la struttura della directory, i file del codice sorgente, le librerie incluse nel progetto e le impostazioni del compilatore sono archiviate nei file IPR. Quando un progetto viene aperto da IntelliJ IDEA, l'IDE legge le informazioni nel file IPR e carica i file e le impostazioni corrispondenti. I file IPR sono stati sostituiti dai file .IDEA che contengono tutte le informazioni come Settings Directory.

## Formato file IPR

I file IPR vengono salvati nel formato di file [XML](/it/web/xml/) universalmente comprensibile e leggibile dall'uomo. Contiene tutte le informazioni relative al progetto nei tag che vengono caricati per visualizzare i contenuti relativi al progetto.

Poiché i file IPR sono archiviati in formato file XML, possono essere aperti in qualsiasi editor di testo come Notepad, Notepad++, Atom e Apple TextEdit.
