## Che cos'è un file RBXM?

Un file RBXM è un formato di file modello 3D utilizzato in Roblox Studio IDE per la memorizzazione di oggetti 3D del gioco Roblox multiplayer online. Contiene oggetti 3D creati per il gioco utilizzando l'IDE. Esempi di oggetti 3D in un file RBXM includono magliette, pantaloni, cappelli, edifici, veicoli, animali, aeroplani, automobili o piante. I file RBMX possono essere convertiti nel [formato file OBJ 3D](/it/3d/obj/) utilizzando Roblox IDE. Roblox IDE può essere utilizzato per aprire file RBMX.

Un altro popolare formato di file Roblox è [RBXL](/it/programming/rbxl/) che è un formato di file di posizione e contiene la posizione definita in parole 3D all'interno del gioco Roblox.

## Formato file RBMX - Ulteriori informazioni

I file RBXM vengono creati e salvati in formato file binario e le loro specifiche di formato file interne non sono disponibili pubblicamente. Questo è diverso dal formato file RBXMX che si basa sul formato file [XML](/it/web/xml/). Ci sono molti strumenti/repository disponibili su Github che possono essere utilizzati per [elaborare i file Roblox RBXM](https://github.com/search?q=rbxm).
