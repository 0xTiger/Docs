## Che cos'è un file GROOVY?

Un file GROOVY è un file di codice sorgente scritto nel linguaggio di programmazione Groovy. Il codice scritto all'interno di un file GROOVY è simile al linguaggio orientato agli oggetti [Java](/it/programming/java/), che semplifica la progettazione e lo sviluppo di applicazioni. Un file GROOVY viene compilato nel bytecode della macchina virtuale Java (JVM) ed è compatibile con altri codici e librerie Java.

## Formato file GROOVY - Ulteriori informazioni

I file GROOVY contengono codice sorgente scritto con la sintassi Groovy. Può contenere codice che può essere utilizzato sia come linguaggio di programmazione che come linguaggi di scripting per la piattaforma Java. Il linguaggio Groovy ha caratteristiche simili a quelle di Python, Ruby e Smalltalk.
