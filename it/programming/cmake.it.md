## Che cos'è un file CMAKE?

Un file CMAKE è un file di progetto creato dallo strumento open source multipiattaforma **CMake**. Contiene lo script sorgente nel linguaggio CMake per l'intero programma. Lo strumento CMake utilizza script di programmazione, noti come CMakeLists, per generare file di build per un ambiente specifico come la macchina Unix makefiles. Gli script ei moduli utilizzati dal progetto CMake sono organizzati come script.cmake e module.cmake a cui si fa riferimento nel progetto.
