## Cos'è un file CUE?

Un file **.CUE**, noto anche come **CDRWIN Cue Sheet**, è un file di testo semplice che contiene informazioni sulle tracce e sul layout di un CD audio o di un'immagine disco, generalmente in formato BIN o ISO. Questi file vengono spesso utilizzati per descrivere la struttura e il contenuto del disco, consentendo al software di masterizzazione di CD o al software di unità virtuale di riprodurre accuratamente il disco originale.

## Esempio di file CUE

Ecco un esempio di come potrebbe apparire il file ".cue":

```
PERFORMER "Artist Name"
TITLE "Album Title"
FILE "DiscImage.bin" BINARY
  TRACK 01 AUDIO
    TITLE "Track 1 Title"
    PERFORMER "Artist Name"
    INDEX 01 00:00:00
  TRACK 02 AUDIO
    TITLE "Track 2 Title"
    PERFORMER "Artist Name"
    INDEX 01 03:45:21
  TRACK 03 AUDIO
    TITLE "Track 3 Title"
    PERFORMER "Artist Name"
    INDEX 01 07:28:17
  TRACK 04 AUDIO
    TITLE "Track 4 Title"
    PERFORMER "Artist Name"
    INDEX 01 12:15:40
  (Additional tracks go here...)
```

## CDRWIN Cue Sheet

CDRWIN Cue Sheet è una variazione specifica del formato file ".cue" utilizzato dal software CDRWIN. CDRWIN è un software di masterizzazione e creazione di CD/DVD e i suoi fogli ".cue" sono progettati per l'uso con questo software. Questi fogli ".cue" contengono le informazioni necessarie a CDRWIN per creare o masterizzare accuratamente CD o DVD.

Ecco alcuni dettagli chiave specifici dei Cue Sheet CDRWIN:

1. **Unico per CDRWIN**: i fogli ".cue" di CDRWIN sono formati proprietari specifici del software CDRWIN. Sebbene condividano somiglianze con i file ".cue" standard, sono personalizzati per funzionare con le funzionalità e le impostazioni di CDRWIN.
    






2. **Interfaccia facile da usare**: CDRWIN fornisce un'interfaccia facile da usare per creare e modificare i suoi fogli ".cue". Gli utenti possono aggiungere o modificare informazioni su tracce, indici, lacune e altri parametri in modo intuitivo.
    






3. **Tipi di dischi compatibili**: i cue sheet CDRWIN vengono utilizzati principalmente per creare vari tipi di CD e DVD, inclusi dischi dati, CD audio, dischi in modalità mista e altro ancora. Il formato consente agli utenti di specificare il tipo e il contenuto del disco che desiderano creare.
    






4. **Controllo sul layout del disco**: i Cue Sheet CDRWIN forniscono un controllo dettagliato sul layout e sulla struttura del disco, inclusi l'ordine delle tracce, le impostazioni di pausa/intervallo e qualsiasi altra preferenza specifica che l'utente può avere.
    






5. **Supporto ISO e BIN**: CDRWIN può funzionare con formati immagine disco ISO e BIN. Il foglio ".cue" specifica quale file immagine viene utilizzato per il disco, garantendo la corretta sincronizzazione tra cue e immagine.
    






6. **Masterizzazione e ripping**: questi fogli ".cue" sono fondamentali quando si masterizzano dischi o si estraggono tracce da dischi utilizzando CDRWIN. Aiutano a garantire che il prodotto finale corrisponda al layout e al contenuto previsti.
    






7. **Backup e ripristino**: gli utenti CDRWIN spesso creano fogli ".cue" quando eseguono backup dei propri CD o DVD. Questi fogli possono essere successivamente utilizzati per ripristinare il contenuto del disco, inclusi il layout e i tempi delle tracce.

## Come aprire il file CUE?

I Cue Sheet CDRWIN sono progettati specificamente per il software CDRWIN, quindi sono generalmente pensati per essere aperti e utilizzati all'interno di CDRWIN.

Programmi che aprono o fanno riferimento a file CUE

-CDRWIN
- Progetti intelligenti IsoBuster
- Sistemi EZB UltraISO
