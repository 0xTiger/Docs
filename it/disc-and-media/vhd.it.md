## Che cos'è un file VHD?

VHD è l'abbreviazione di Virtual Hard Disk. Questi sono formati di file che rappresentano un'unità disco rigido virtuale. I dati trovati su un disco rigido fisico possono essere coinvolti in esso proprio come le partizioni del disco rigido e il file system che comprende le cartelle.

Per una macchina virtuale, i file VHD sono considerati dischi rigidi. Nelle moderne versioni di Windows, questi sono integrati. Questo specifico tipo di disco rigido virtuale fornisce la possibilità a diversi sistemi operativi di risiedere sulla macchina host comune. Offre agli sviluppatori una grande opportunità per testare il software su diversi sistemi operativi senza eseguire il processo di installazione di un altro disco rigido. Ci sono molti vantaggi nell'usare questo formato di file di file VHD in quanto forniscono molte funzionalità.


## Breve storia ##

**Connectix** ha sviluppato questo formato di file come prodotto per PC virtuale per il loro utilizzo. Nel 2003 Microsoft ha acquisito Connectix e ha lanciato altre versioni di questo formato come VHDx insieme alle migliori caratteristiche e funzionalità che erano mancate nella prima versione. Nel 2012 il VHDx è stato aggiunto a Windows. È stato aggiunto a Windows per aggiungere più capacità di archiviazione, protezione contro il danneggiamento dei dati, ottimizzazione per prevenire il degrado delle prestazioni. Queste funzionalità sono state aggiunte per i dischi fisici con settori di grandi dimensioni.

## Formato file VHD ##

È possibile accedere a un file system host per risiedervi i file VHD. Questi hanno le caratteristiche dell'immagine fissa del disco di guardia che consente all'utente di accedere a un file a cui è allocata la dimensione di un disco virtuale specifico. I file possono modificare il disco rigido di una macchina virtuale.

Questi file VHD possono essere scambiati facilmente dal disco rigido virtuale al file system dell'host. Questo formato ha anche la capacità di backup e ripristino e sicurezza antivirus. I file VHD possono supportare formati specifici come un'immagine del disco rigido fisso, un'immagine del disco rigido con differenze e un'immagine del disco rigido dinamico.
