## Che cos'è un file Toast?
Il software Roxio Toast crea un'immagine CD o DVD e salva il file con estensione .toast. Roxio Toast è una nota applicazione software per la creazione di dischi ottici e la conversione dei media per Macintosh introdotta per la prima volta da Miles Software GmbH, poi acquistata da Adaptec. Il file Toast può contenere qualsiasi tipo di disco come video, audio o disco dati. Potrebbe anche essere una copia e un'altra immagine del disco.

## Formato file toast
Il formato file Toast viene utilizzato dal software Roxio Toast per salvarne l'output in un file con estensione .toast. Sebbene i dischi di vario tipo possano essere masterizzati direttamente tramite i sistemi Mac OS, Toast fornisce un controllo aggiuntivo sul processo e offre anche alcune funzionalità extra, come la catalogazione, il monitoraggio dei file masterizzati su disco e il ripristino dei file per i dischi danneggiati. fornisce anche supporto per formati audio e video non supportati da Quicktime, come [FLAC](/it/audio/flac/) e [OGG](/it/audio/ogg/). Poiché si tratta di un formato di file proprietario, le informazioni sull'intera struttura del formato di file non sono disponibili.
### Tipi di dischi
Pertanto, Roxio Toast può creare file Toast, contenenti i dati di immagine del disco dei seguenti tipi:
- **Creazione di dischi video**: Toast può creare file Toast contenenti dischi video utilizzando i seguenti progetti:
- DVD-Video
- Video Blu-ray
- DVD ad alta definizione
- Cartelle VIDEO_TS
- Compilazione VIDEO_TS
- Cartella BDMV
- Archivio AVCHD
- **Creazione di dischi dati**: Toast può creare file Toast contenenti dischi dati utilizzando i seguenti progetti:
- Solo Mac
- Mac e PC
- DVD-ROM
- Ibrido personalizzato
- Volume Mac
- Disco fotografico
- **Creazione di dischi video**: Toast può creare file Toast contenenti dischi dati utilizzando i seguenti progetti:
- CD audio
- DVD musicale
- Disco MP3
- CD audio avanzato
- CD in modalità mista
- **Copia disco**: Toast supporta la copia nei seguenti modi:
- Copia disco
- Copia immagine
- Unisci immagine disco
