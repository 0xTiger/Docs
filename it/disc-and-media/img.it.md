## Che cos'è un file IMG?

Nel campo dell'informatica, i file con estensione .img si occupano della memorizzazione di immagini di dischi grezzi di dischi come floppy disk, dischi rigidi o dischi ottici, ecc. Questa immagine grezza consiste in un'immagine binaria della sorgente con le specifiche di il settore per settore.

Il contenuto del formato di file IMG dipende dal file system del disco specificato per la creazione dell'immagine. Quando si tratta di CD ROM e DVD, le immagini di questi file contengono i dati dei settori corrispondenti insieme alle intestazioni di controllo e ai campi per la correzione degli errori in base a ciascun settore.

I file IMG hanno contenuto specifico per il disco, quindi sono compatibili con quei programmi specifici che sono adatti per il rilevamento dei loro file system.

## Formato file IMG ##

Negli anni in cui un floppy disk veniva utilizzato per la memorizzazione e la gestione dell'archiviazione dei dati, questi file venivano sviluppati per accedere alle immagini grezze del disco. Molti programmi sono stati indicati dai file IMG. Questi sono stati creati da programmi di imaging del disco. Nei programmi del sistema operativo Windows come power, per gestire questi file viene utilizzato un masterizzatore software.

## Specifica tecnica ##

Esistono altre estensioni di file come IMA e IMZ simili a questo formato di file. La dimensione di queste immagini è sempre sotto forma di più dimensioni di settore diverse. Questa dimensione è di 512 byte per floppy disk e dischi rigidi. La dimensione grezza del settore è di 2.352 per CD Rom e DVD. La dimensione dell'immagine del disco non elaborato è multipla di questa dimensione.

Questo formato è usato principalmente con programmi come RawWrite e WinImage. Questi programmi usano questo formato per leggere e scrivere le immagini dei floppy disk. Inoltre, questi file possono essere convertiti in altri formati di file più popolari utilizzando alcuni software come PowerISO.

Poiché i file IMG non hanno dati aggiunti tranne il contenuto dei dischi, questi file IMG possono essere gestiti automaticamente dai programmi e rilevare i file system. Ad esempio, una normale immagine disco di un floppy disk grezzo inizia con il settore di avvio di FAT e può essere utilizzata per identificare il file system.

Le immagini del disco del supporto ottico sono per lo più accompagnate dal file descrittore. Spiega il layout del disco e coinvolge informazioni come i limiti di traccia e questi non vengono semplicemente archiviati in un file immagine in forma grezza.
