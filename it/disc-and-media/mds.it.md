## Cos'è un file MDS?

L'MDS (Media Descriptor Sidecar File) è un file associato al software Alcohol 120% e Daemon Tools, entrambi utilizzati per creare e gestire unità CD e DVD virtuali. Il file MDS contiene informazioni sulla disposizione dei dati sul disco, inclusa la posizione di tutti i file e la struttura del disco. Ciò consente all'unità virtuale di emulare il comportamento di un disco fisico, in modo che il software possa leggere i dati sul disco virtuale come se fosse un disco reale.

When you want to mount an image file to a virtual drive, the MDS file is used along with the corresponding image file in a format such as ISO, BIN or CUE. The MDS file contains a description of the layout of the data on the disc, and the virtual drive software uses this information to create a virtual disc. This allows you to use the software as if you were using a physical disc, without having to physically insert the disc into the drive.

## Rapporto con l'alcol 120%

Alcohol 120% è un software popolare per la creazione e la gestione di unità CD e DVD virtuali e utilizza il formato file MDS per salvare e accedere alle immagini del disco. Il formato file MDS è proprietario di Alcohol 120% e può essere aperto e utilizzato solo dal software Alcohol 120% o da altro software che lo supporta.

## Come aprire il file MDS?

Per aprire un file MDS in Alcohol 120%, è necessario che il software sia installato sul tuo computer. Una volta installato Alcohol 120%, puoi aprire il file MDS procedendo come segue:

1. Lancia l'alcol al 120%.
2. Fare clic sul menu File e selezionare Apri.
3. Passare alla posizione del file MDS sul computer e selezionarlo.
4. Il file MDS verrà ora aperto in Alcohol 120% e ti verrà richiesto di selezionare il file immagine corrispondente (come ISO, BIN o CUE).
5. Dopo aver selezionato il file immagine, Alcohol 120% monterà l'immagine su un'unità virtuale.

In alternativa, puoi anche aprire un file MDS facendo doppio clic su di esso, se Alcohol 120% è impostato come programma predefinito per l'apertura dei file MDS. Puoi anche utilizzare altri software che supportano il formato MDS come Daemon Tools, Virtual CloneDrive, Virtual Drive Manager e MagicISO.
