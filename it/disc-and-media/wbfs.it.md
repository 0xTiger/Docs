## Che cos'è un file WBFS?

Un file WBFS (Wii Backup File System) è un'immagine disco di un videogioco creata per Nintendo Wii. Contiene un gioco Wii che può essere caricato come immagine per giocare. Viene generato utilizzando lo strumento Wiimms WBPS o Wii Backup Manager e può essere riprodotto su un Nintendo Wii caricandolo da un caricatore USB GX o da un emulatore Wii come Dolphin. Il gioco viene archiviato su questi dispositivi di archiviazione esterni utilizzando il formato di file WBFS. Il Wii è una piattaforma di gioco distribuita da Nintendo. Il file system WBFS è stato sviluppato dal programmatore homebrew Wii Waninkoko.

## Formato file WBFS - Ulteriori informazioni

I WBFS sono file binari che hanno un formato di file proprietario. Le sue specifiche sul formato dei file non sono disponibili pubblicamente.
