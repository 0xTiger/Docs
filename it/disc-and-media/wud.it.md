## Che cos'è un file WUD?

Un file con estensione .wud è un'immagine disco creata da un disco di gioco Wii U. Il file immagine del disco risultante può essere caricato da emulatori di videogiochi come [Cemu](https://cemu.info/) per emulare il gioco. I file WUD sono di grandi dimensioni e vengono quindi creati come segmenti multipli per essere trasferiti facilmente. Le informazioni sul gioco vengono esportate come segmenti di file WUD di dimensioni file di circa 2 GB. L'applicazione di apertura, come Cemu, necessita che tutti questi file segmentati vengano uniti in un unico file WUD prima di aprirli.

## Formato file WUD

I file WUD vengono salvati su disco in un formato di file proprietario e possono essere caricati nell'emulatore Cemu per emulare le applicazioni Wii U su PC. A causa delle sue grandi dimensioni, i file WUD vengono [compressi](/it/compression/) per ridurne le dimensioni. Il file compresso risultante è noto come file WUX.
