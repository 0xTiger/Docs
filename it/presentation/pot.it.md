## Che cos'è un file POT?

I file con estensione .POT rappresentano i file modello di Microsoft PowerPoint creati dalle versioni PowerPoint 97-2003. I file creati con queste versioni di Microsoft PowerPoint sono in formato binario rispetto a quelli creati nei formati di file Office OpenXML utilizzando le versioni successive di PowerPoint. I file, quindi, generati possono essere utilizzati per creare presentazioni che hanno lo stesso layout e altre impostazioni necessarie per essere applicate ai nuovi file. Queste impostazioni possono includere stili, sfondi, tavolozza dei colori, caratteri e impostazioni predefinite. Tali file vengono generati per creare file modello pronti per l'uso per l'uso ufficiale.

## Specifiche del formato file ##

Le specifiche del formato file per il formato file POT non sono disponibili singolarmente da Microsoft. Tuttavia, il formato del modello è lo stesso del formato file binario [PPT](/it/presentation/ppt/) e può essere aperto in MS PowerPoint a scopo di modifica. Un file POT è identificato dall'identificatore HEX come segue:

```
D0 CF 11 E0 A1 B1 1A E1 00 00 00 00
```

I tipi MIME associati al formato di file POT sono:

* application/vnd.ms-powerpoint [ufficiale]
* applicazione/mspowerpoint
* applicazione/x-mspowerpoint
* applicazione/powerpoint
* applicazione/x-powerpoint
* applicazione/x-dos_ms_powerpnt
* applicazione/vaso
* applicazione/x-soffici
