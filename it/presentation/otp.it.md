## Che cos'è un file OTP?

I file con estensione .otp rappresentano file modello di presentazione creati da applicazioni in formato standard OASIS OpenDocument. Il contenuto di tale file include informazioni sulla presentazione sotto forma di diapositive con testo, immagini, forme, contenuto multimediale, effetti di transizione e altri elementi di diapositiva. Questi file modello vengono utilizzati per creare rapidamente nuove presentazioni in base alle informazioni di stile memorizzate nel modello stesso. I file OTP possono essere creati e salvati con diverse applicazioni come Impress fornito con la suite OpenOffice e Microsoft PowerPoint. Il formato del file OTP è simile ai file modello di Microsoft PowerPoint [.pot](/it/presentation/pot/) e [.potx](/it/presentation/potx/).

## Formato file OTP

Il formato di file OTP si basa sullo standard OpenDocument che supporta la rappresentazione del documento come un singolo documento XML e la raccolta di diversi documenti secondari in un unico pacchetto zippato nel formato [ZIP](/it/compression/zip/). Il contenuto del file viene distribuito in più file XML impacchettati insieme. Questi file multipli [XML](/it/web/xml/) contengono informazioni sullo stile, il contenuto, le metainformazioni e le impostazioni del documento, come indicato di seguito.

* `content.xml` – Contenuto del documento e stili automatici utilizzati nel contenuto.
* `styles.xml` – Stili usati nel contenuto del documento e stili automatici usati negli stili stessi.
* `meta.xml` – Le metainformazioni del documento, come l'autore o l'ora dell'ultima azione di salvataggio.
* `settings.xml` – Impostazioni specifiche dell'applicazione, come le dimensioni della finestra o le informazioni sulla stampante.
