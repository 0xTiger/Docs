## Che cos'è un file PPS?

PPS, presentazione di diapositive di PowerPoint, i file vengono creati utilizzando Microsoft PowerPoint per lo scopo di presentazione. La lettura e la creazione di file PPS è supportata da Microsoft PowerPoint 97-2003. La versione più recente di questo formato di file è [PPSX ](/it/presentation/ppsx/) che si basa sugli standard Office OpenXML. I file PPS possono ancora essere letti dalle ultime versioni di Microsoft PowerPoint, ma i file appena creati possono essere salvati solo nel formato file PPSX. Quando un file PPS viene condiviso con un altro utente e aperto, si avvia come PowerPoint show a differenza del file [PPT](/it/presentation/ppt/) che si apre in modalità modificabile.

## Formato del file ##

PPS è un formato di file binario che aderisce alle specifiche dettagliate in [MS-PPT](https://msdn.microsoft.com/en-us/library/office/cc313106(v#office.12).aspx) formato di file documento. Consiste in una serie di flussi per rappresentare la struttura e i dati complessivi del documento. Questi includono:

* Flusso utente corrente
* Flusso di documenti PowerPoint
* Flusso di immagini
* Informazioni di riepilogo e Informazioni di riepilogo del documento (opzionale)

I dettagli completi sul formato di file PPT sono disponibili nel documento delle specifiche MS-PPT che si applica ugualmente anche a PPS.
