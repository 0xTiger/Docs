## Che cos'è un file IGS?

Un file con estensione .igs (Initial Graphics Exchange) è un formato di file di scambio di progettazione 2D-3D indipendente dalle specifiche del formato del file di origine o di destinazione utilizzate dalle applicazioni CAD. Viene utilizzato per scambiare informazioni di progettazione su schemi circuitali, wireframe, superficie a forma libera tra due sistemi indipendenti. È simile a [IGES](/it/cad/iges/) ed è il requisito principale dei produttori per la determinazione del prezzo e la progettazione di stampi per il tuo prodotto. IGS è stato recentemente sostituito dal più recente formato di file STEP(.STP). I file IGS possono essere aperti da applicazioni come Autodesk, FreeCAD, CADEX CAD Exchanger e altre applicazioni simili.

## Formato file IGS

I file IGS sono in formato di file di testo ASCII e possono essere aperti con qualsiasi editor di testo. Ogni record in un file IGS è lungo 80 caratteri. Le applicazioni CAD salvano ed esportano dati vettoriali nel formato di file IGS per il trasferimento di file di modelli 3D tra diversi pacchetti software. Sebbene i file IGS possano essere aperti in qualsiasi editor di testo, il modello o il progetto possono essere visualizzati solo da applicazioni di visualizzazione che forniscono supporto per il formato di file IGS.
