## Che cos'è un file NWC?

Un file con estensione .nwc è un file di cache generato dal software Navisworks di Autodesk durante l'utilizzo di file CAD. Viene generato automaticamente all'apertura di un file CAD

## Formato file NWC

NWC è il formato di file proprietario di Autodesk ed è per questo che i suoi dettagli non sono disponibili. .
