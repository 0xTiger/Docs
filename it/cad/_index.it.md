CAD sta per Computer Aided Design. Viene utilizzato anche il termine CADD (per Computer Aided Design and Drafting). Viene utilizzato per un formato di file di grafica 3D e può contenere progetti 2D o 3D. Il file CAD è un formato di file digitale di un oggetto generato e utilizzato dal software CAD. Un file CAD contiene un disegno tecnico, un progetto, uno schema o un rendering 3D di un oggetto. Potrebbero essere disponibili altri strumenti CAD che possono essere utilizzati per creare, aprire, modificare ed esportare questi file .cad in formati di file di disegno CAD più diffusi. Di seguito discuteremo dei formati di file CAD più popolari e del software più consigliato in grado di aprirlo, modificarlo e convertirlo in altri formati popolari.

Hai domande relative ai formati di file CAD? Visita la nostra community [forum](https://forum.fileformat.com/c/cad/12) per beneficiare delle conoscenze condivise dagli esperti di formati di file.
