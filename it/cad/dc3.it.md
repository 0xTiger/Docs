## Che cos'è un file DC3?

Un file con estensione .DC3 è un disegno 3D creato con DesignCAD 3D MAX che è un software CAD. Viene utilizzato per creare design di alta qualità, rendering e animazioni semplici e supporta l'importazione/esportazione di [STL](/it/cad/stl/). Un file DC3 viene utilizzato come alternativa al formato di file .DCD. I file DC3 possono essere aperti con IMSI DesignCAD 3D Max che fornisce una gamma completa di potenti primitive di modellazione 3D tra cui scatola, sfera, emisfero, piramide, cilindro, cono e toro.

## Formato file DC3

Un file DC3 contiene informazioni di disegno 3D in formato di testo ASCII che possono essere aperte in qualsiasi editor di testo. Tuttavia, il significato del suo contenuto può essere compreso solo da applicazioni che comprendono il formato del file sottostante.
