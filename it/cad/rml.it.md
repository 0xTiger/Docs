## Che cos'è un file RML?

Un file RML è un file di annotazione creato e utilizzato dall'applicazione Volo View da [Autodesk](https://autodesk.com). Contiene informazioni sulle annotazioni e segni di testo da sovrapporre al disegno per prendere appunti. Queste annotazioni vengono visualizzate in testo rosso sul disegno ed è per questo che il nome file Redline Markup Language. Volo View è stato interrotto e non è più supportato da Autodesk. I contenuti dei file RML sono incorporati nei disegni CAD come note.

## Formato file RML

I file RML vengono creati e salvati su disco in formato file XML, che è un formato file universale per lo scambio di dati. È in formato leggibile dall'uomo e può essere aperto con qualsiasi editor di testo come Microsoft Notepad, Notepad ++ e Apple TextEdit.
