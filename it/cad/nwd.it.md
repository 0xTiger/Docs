## Che cos'è un file NWD?

Un file con estensione .nwd è un file CAD di Autodesk per i file Navisworks che è un software di revisione di modelli 3D per architettura, ingegneria e costruzione. I file NWD combinano i dati di progettazione e costruzione in un unico modello. Ciò consente di identificare e risolvere differenze e problemi di interferenza prima che la costruzione inizi utilizzando il software. I file NWD memorizzano le informazioni sul modello, sull'ambiente, sulla vista corrente e sui punti di vista salvati (includono linee rosse, commenti, ecc.). Questi possono anche contenere metadati specificati dall'utente in forma pubblicata. I file NWD possono essere aperti in qualsiasi prodotto Navisworks e nel visualizzatore Navisworkds Freedom.

## Formato file NWD

NWD è il formato di file proprietario di Autodesk ed è per questo che i suoi dettagli non sono disponibili. Un file NWD può essere considerato come lo stato corrente del modello. Contiene tutte le informazioni sulla geometria insieme ai markup di revisione per l'elaborazione dei punti di vista. I file NWD sono di dimensioni inferiori dell'80% a causa della compressione dei dati rispetto alle dimensioni originali.
