## Che cos'è un file NC?

Un file NC è un file di controllo numerico computerizzato (CNC) creato con [software Mastercam CNC](https://www.mastercam.com/). Comprende istruzioni di controllo numerico per dirigere il movimento meccanico di strumenti di lavorazione CNC come trapani, torni, frese e stampanti 3D. Le macchine CNC utilizzano le istruzioni dei file NC sotto forma di un programma sequenziale per l'esecuzione. Queste istruzioni dal file NC hanno lo scopo di guidare la macchina per la produzione del componente.

## Formato file NC

I file NC vengono archiviati su disco come file binari e le loro specifiche di formato file interne non sono disponibili in modo specifico. Il software Mastercam CNC consente agli utenti di creare percorsi utensile macchina 2D o 3D. Questi percorsi utensile, una volta salvati su disco, generano il file .nc che viene infine utilizzato dalla macchina utensile CNC per creare il componente desiderato.
