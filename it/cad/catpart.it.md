## Che cos'è un file CATPART?

Un file CATPART è il formato file nativo del software CATIA V5. Memorizza parti e assiemi creati in CATIA V5, un'applicazione di produzione CAD 3D. Questi file possono essere aperti e modificati in CATIA V5 o altre applicazioni CAD che supportano il formato di file CATPART come AutoCAD. Un file CATPART contiene la geometria 3D insieme ai dati della struttura dei modelli. Viene salvato su disco in formato file binario e ha il numero magico `56 35 5F 43 46 56 32`.
