## Che cos'è un file DWFX?

Un DWFx (Design Web Format XPS) è una rappresentazione formattata del disegno 2D/3D come documento XPS. Contiene grafica e testo come parte dei dati di progettazione. È la versione più recente del formato file DWF e può essere visualizzata e stampata con Microsoft XPS Viewer. La natura XPS di questi file consente di condividere i dati di progettazione con i revisori senza richiedere loro di installare Autodesk Design Review.
Simile a DWF, DWFx è sviluppato da Autodesk in formato compresso per rendere adatta la trasmissione su Internet. Un singolo file DWFx può contenere uno o più disegni e gruppi di fogli.
