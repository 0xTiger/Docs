## Che cos'è un file ASC?

I file ASC sono file di testo Autodesk che memorizzano informazioni sugli oggetti nel disegno, inclusa la loro posizione, dimensione e colore. Questo li rende una scelta popolare per lo scambio di disegni tra diversi programmi [CAD](/it/cad/). I file ASC possono essere aperti con i prodotti software di Autodesk. Autodesk offre la possibilità di importare file ASC dall'interno dell'applicazione. I file ASC possono essere aperti in qualsiasi editor di testo come Microsoft Notepad, Notepad++ e Apple TextEdit.

## Formato file ASC

I file ASC vengono memorizzati su disco in un semplice formato di file di testo ASCII. Questi sono leggibili dall'uomo.

### Esempio di file ASC

```
ncols 200
nrows 200
xllcorner 119000.00
yllcorner 158000.00
cellsize 5.00
nodata_value -999999
```
