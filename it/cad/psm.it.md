## Che cos'è un file PSM?

Un file con estensione .psm è un file CAD che viene creato e aperto con il software di modellazione CAD Solid Edge per progettare parti e assiemi 3D. Contiene il modello 3D del progetto per la produzione di parti dalla lamiera. La tuta software Solid Edge fornisce tutte le opzioni necessarie per creare un modello 3D che può essere integrato anche con altri progetti meccanici. La sua nidificazione 2D supporta sia i formati di file PSM che PAR oltre ad altri formati di dati neutri inclusi DXF e DWG estraendo dati di pattern piatti.

### Formato file PSM

PSM è un formato di file chiuso e i suoi dettagli non sono disponibili al momento. Questi hanno il tipo MIME application/octet-stream. I file del modello in metallo possono essere importati in Solid Edge e convertiti in parti utilizzabili per renderli modificabili. La modellazione sincrona offre una maggiore usabilità per lavorare con i file PSM.
