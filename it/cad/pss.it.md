## Che cos'Ã¨ un file PSS?

Un file con estensione .pss viene utilizzato da AutoCAD e contiene informazioni sul timbro di stampa. Contiene informazioni sul timbro di stampa come data, ora, scala di stampa, formato carta, nome del layout e informazioni specifiche sul plotter. Il comando Timbro stampa posiziona queste informazioni di stampa su un angolo specificato di ogni disegno e le registra in un file. I file PSS possono essere aperti con l'applicazione software Autodesk AutoCAD.
