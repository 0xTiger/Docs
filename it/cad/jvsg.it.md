## Che cos'è un file JVSG?

JVSG è un file CAD che può essere lanciato su Windows. È un file in formato binario creato da JVSG. I file del progetto di videosorveglianza utilizzano principalmente il suffisso del nome file JVSG. Questo file può memorizzare la larghezza di banda, le posizioni, gli angoli di visualizzazione e i calcoli del disco rigido di una telecamera con informazioni sul suo design CCTV. I file di progetto JVSG offrono analisi e documentazione avanzate eseguendo il rendering nelle viste 3D.

Le reti di telecamere di sicurezza di diverse aziende che necessitano di telecamere di sorveglianza possono anche essere progettate con i file di progetto JVSG. Il software "CCTV design tool" è adatto per la gestione di file JVSG.
