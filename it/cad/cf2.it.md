## Che cos'è un file CF2?

Un file con estensione .cf2 è un formato di file CAD che contiene progetti di pacchetti 3D o altri dati del modello per la fustellatura. La maggior parte delle macchine CAD/CAM è in grado di elaborare e tagliare questi file. È stato creato dal National Space Science Data Center (NSSDC) per fornire un formato di archiviazione e manipolazione dei dati autodescrittivo che corrisponda alla struttura dei dati scientifici e alle applicazioni come metodi statistici e numerici, visualizzazione e gestione. Con il passare del tempo sono stati sviluppati più dispositivi hardware e applicazioni software che riconoscono i file .cf2. Queste applicazioni forniscono supporto per i file CF2 su Mac OS e sistemi operativi Windows.
