## Che cos'è un file ASM?

Un file ASM è un file di dati utilizzato da [Solid Edge](https://solidedge.siemens.com/en/) software di modellazione, rendering e assemblaggio CAD 3D. Contiene oggetti 3D costituiti da un assieme di parti modellate. Un singolo file ASM può contenere più parti per costituire l'intero file di dati. I file ASM sono completamente diversi dai file [ASM in linguaggio assembly](/it/programming/asm/) e non hanno alcuna connessione con questi. Il software Siemens Solid Edge viene utilizzato per aprire questi file ASM.
