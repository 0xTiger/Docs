## Che cos'è un file NWF?

Un file con estensione .nwf è un formato di file CAD proprietario di Autodesk che contiene gli indici di tutti i file di modello utilizzati dal progetto. Contiene puntatori agli elementi aggiunti ai file del modello per fare riferimento. I file NWF contengono anche informazioni come punti di vista e redline. Il vantaggio dell'utilizzo del formato file NWF è il riflesso immediato delle modifiche apportate ai file CAD originali [es. DWG](/it/cad/dwg/) dovute ai puntatori associati. L'uso di un NWF è altamente raccomandato per i test di conflitto. Il file NWF salva i risultati dell'interferenza originale e dopo la modifica dei file CAD, i file CAD modificati verranno aperti alla riapertura del file NWF.

## Formato file NWF

I dettagli del formato file interno di NWF non sono disponibili in quanto si tratta di un formato file proprietario di Autodesk.
