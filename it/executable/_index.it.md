I file eseguibili vengono utilizzati per istruire un computer a eseguire una o più attività o operazioni specifiche come insieme codificato di istruzioni; scritto in quei file. Il termine "Istruzioni" di solito indica le istruzioni del **codice macchina** per una **CPU** fisica. I file eseguibili possono essere codificati manualmente in linguaggio macchina, sebbene sia molto conveniente sviluppare software come codice sorgente in un linguaggio di alto livello che può essere facilmente compreso dagli esseri umani. A volte, invece, il codice sorgente potrebbe essere scritto in linguaggio assembly, rimane strettamente associato alle istruzioni del codice macchina che sono facilmente leggibili dall'uomo.

Alcuni formati di file disco più diffusi sono [BAT](/it/executable/bat/), [CGI](/it/executable/cgi) e [COM](/it/executable/com/)
