## Che cos'è un file EDN? ##

Il tipo di file EDN è principalmente correlato ad Acrobat di Adobe Systems Incorporated. Vengono eseguite tutte le specifiche di EDN, inclusi i rudimenti contrassegnati da convenzioni. Di seguito vengono fornite alcune informazioni di base relative all'EDN.

* Implementazione EDIF Netlist Tipo di file Descrizione
* Xilinx, INC File Developer
* Software relativo a Xilinx ISE Webpack
* Xilinx, sviluppatore di software INC
 

## Compatibilità file ##

* Acrobat di Adobe Systems Incorporated. Questa è una categoria di programmi per computer creata da Adobe Systems. Lo scopo principale era visualizzare, generare, utilizzare e gestire i file nel Portable Document Format (PDF) di Adobe. Alcuni dei software in questa categoria sono gratuiti e alcuni sono commerciali
* EDIF Netlist File sviluppato da Microsoft Developer è anche noto come Electronic Design Interchange Format, viene utilizzato principalmente come formato neutro in cui è necessario archiviare netlist e schemi elettronici
