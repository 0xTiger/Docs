## Che cos'è un file LIT?

I file LIT appartengono all'antico **Microsoft Reader** che veniva utilizzato per leggere gli e-book. Gli eBook visualizzati da Microsoft Reader hanno l'estensione **.lit** che viene abbreviata in **Letteratura**. Il formato di file LIT è costituito da un file Microsoft ITOL/ITLS HTML Help 2.0, che offre tutti i tipi di straordinarie funzionalità come la compressione [LZX](/it/compression/lzx/), la rappresentazione binaria del materiale di markup e dati ausiliari opzionali arbitrari associati a ogni flusso. Il testo del libro è un numero arbitrario di flussi di markup OEBPS 1.0 e un sottoinsieme di CSS OEBPS 1.0. Le immagini sono [GIF](/it/image/gif/), [JPEG](/it/image/jpeg/) e [PNG](/it/image/png/).

## Per quanto tempo Microsoft Reader ha visualizzato i file LIT

**Microsoft Reader** è un'applicazione per la lettura di e-book la cui produzione è stata interrotta da Microsoft. La prima versione di Microsoft Reader è stata rilasciata nell'agosto 2000, che utilizzava il proprio formato LIT. Era disponibile per PDA e computer Windows. Il nome dell'app è stato utilizzato in seguito anche per un'applicazione (che inizialmente era stata rilasciata con Windows 8) per la lettura di file [PDF](/it/pdf/) e [XPS](/it/page-description-language/xps/). L'applicazione Microsoft Reader è stata interrotta nel 2018.
