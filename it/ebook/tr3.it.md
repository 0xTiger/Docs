## Che cos'è un file TR3? ##

Il file TR3 è un eBook TomeRaider 3 che fornisce supporto per la ricerca e la compressione rapide. Ciò può funzionare correttamente tramite il programma correlato a TomeRaider. Yadabyte è lo sviluppatore di questo software, una società britannica di software e sviluppo web con sede nel Regno Unito. L'applicazione TomeRaider eBook reader può essere utilizzata per il corretto funzionamento e la gestione del contenuto di questi file TR3. Questo lettore di eBook correlato può essere installato in computer che eseguono sistemi basati su Microsoft Windows e molti gadget trasferibili. Il file TR3 appartiene al gruppo di file eBook utilizzati in sistemi operativi come Windows 10, Windows 7, Windows 8 / 8.1, Windows Vista, Windows XP.

I **tag HTML** specifici per TR3 sono i seguenti:

|Etichetta|Descrizione|
---|---|
|\<new> |Per creare file TomeRaider da file di testo, il tag Nuovo è tutto ciò che serve.<new> tag definisce l'inizio delle pagine.|
|\<CATDEF> ... \</CATDEF> |definisce il nome della categoria|
|\<META> ... \</META> |è un tag utilizzato per definire tutti i metadati utilizzati nel formato file. Questo tag include una serie di parametri.|
|\<EXPMSG> |Questo tag controlla il messaggio che appare quando un file è scaduto. Ogni volta che un utente tenta di visualizzare una pagina dopo che il file è scaduto, viene visualizzato il messaggio di scadenza al posto del testo effettivo della pagina.|
|\<DIEMSG> |Questo tag è simile a EXPMSG. Viene utilizzato per visualizzare un messaggio quando il file termina.|
|\<ENGMSG> |Questo tag viene utilizzato per creare il messaggio visualizzato per le pagine crittografate. Il messaggio viene visualizzato al posto del testo della pagina se l'utente tenta di aprire una pagina crittografata prima di sbloccarla.|
|\<expmsg> ,\<diemsg> e \<encmsg> |Ignorato nel testo della pagina. Devono essere inseriti nella sezione dell'intestazione del file prima della prima<new> tag.|
|\<SELECTION> . \</ SELECTION> |Utilizzato per supportare le query. Questo tag deve essere prima del primo<new> etichetta. Compila i dati di selezione per la richiesta dell'utente.|
|\<catset> . \</catset> | Utilizzato per assegnare nomi di categoria a ciascun soggetto.|


## Problemi per aprire un file TR3 ##

Ecco l'elenco di alcuni problemi comuni che possono sorgere e causare il malfunzionamento del formato del file:

* File corrotto
* File infetto a causa di virus
* Collegamenti impropri al file TR3 negli accessi all'archivio
* Nessun diritto di accesso nel sistema per aprire i file
* Unità obsoleta nel tuo sistema
* Rimozione del report del TR3 dall'archivio di Windows
* Contattare lo sviluppatore potrebbe essere un'opzione migliore per ottenere un output migliore
