## Che cos'è un file TR? ##

Un file con estensione .tr è un file eBook TomeRaider creato con il software eReader TomeRaider. Questi file possono essere aperti su dispositivi Windows e palmari come Android, Palm OS ed EPOC e sono classificati come file di testo. Il software TomeRaider sviluppato dal sito Web [Yadabyte](https://www.yottabyte.ltd/) e dal sito Web ufficiale fornisce ulteriori informazioni sui file TR. Il formato file TR2 è stato sostituito dal formato TR3 (TomeRaider 3). I file TR offrono una ricerca rapida e livelli di compressione elevati.

## Breve storia del formato di file TR

TomeRaider è stato sviluppato da una società di sviluppo software e web con sede nel Regno Unito Yadabyte nel 1999. L'azienda è andata oltre a TomeRaider versione 3.

## Formato file TR

L'ultima versione del formato di file TR è la versione 3 che fornisce supporto per la ricerca e la compressione rapide. Comprime i file di testo al 45-60% della dimensione originale. Ciò consente ai lettori di ebook TomeRaider di eseguire ricerche, indicizzazione e compressione rapide. Il formato file TR è adatto per documenti di riferimento di grandi dimensioni e la maggior parte dei documenti richiede una chiave DRM per essere crittografata.

## Problemi per aprire un file TR

Potrebbero esserci alcuni problemi comuni a causa dei quali non è possibile aprire il file:

* Assenza di software di supporto
* Versione software obsoleta
* File TR danneggiato
* Collegamenti impropri al file TR negli accessi all'archivio
* Nessun diritto di accesso nel sistema per aprire i file
* Unità obsoleta nel tuo sistema
* Rimozione del report del TR dall'archivio di Windows
* Installazione parziale di un'applicazione che supporta il formato TR
* File TR infetto
* Spazio insufficiente per le risorse hardware per facilitare l'apertura del file TR
