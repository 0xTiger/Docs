## Che cos'è un file OEBZIP? ##

I file OEBZIP sono stati introdotti dall'International Digital Publishing Forum (**IDPF**). Allo stesso modo, [OEB](/it/ebook/oeb/), questi file sono specifiche basate su XML per la struttura, il contenuto e la presentazione dei libri elettronici, ma sono in formato compresso o zip.

## Dettagli tecnici del formato file OEBZIP ##

L'OEBZIP ha fornito più nuove funzionalità nell'area del controllo del rendering, inclusi, tra le altre cose, miglioramenti nel vocabolario del markup (ora un sottoinsieme di base di XHTML 1.1) e un supporto CSS ampiamente ampliato. I fattori di sostenibilità sono invariati rispetto alla versione precedente.

Il formato di file OEBZIP è costituito da un insieme di file, incluso un file di pacchetto obbligatorio, che include un manifest che elenca tutti gli altri file componenti, e una spina dorsale, che indica l'ordine di lettura logico. Inoltre, per inviare testo in [XML](/it/web/xml/), un lettore di file OEBZIP deve essere in grado di eseguire il rendering di immagini nei formati [JPEG](/it/image/jpeg/) e [PNG](/it/image/png/) . È possibile incorporare componenti di contenuto in altri formati, ma il contenuto di riserva deve essere fornito in uno dei formati OEB di base.
  

Il pacchetto di file OEBZIP è stato utilizzato principalmente come formato intermedio, con la pubblicazione per gli utenti finali gestita tramite editori o aggregatori che hanno fornito eBook in forme adatte a diversi visualizzatori di eBook. È più probabile che la sua versione successiva, [EPUB](/it/ebook/epub/) raggiunga gli utenti finali in un formato di stato finale, oltre a fungere da formato di stato intermedio.
