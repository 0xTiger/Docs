## Che cos'è un file AZW1? ##

L'AZW1 è un formato di file eBook noto anche come **eBook Amazon Kindle Topaz** lanciato tramite **Whispernet**. Questo formato di file può essere utilizzato con il software Kindle su PC e smartphone ed è responsabile della memorizzazione degli eBook scaricati dalla rete 3G Whispernet di Amazon.

## Formato file AZW1 ##

I dispositivi di lettura di libri Amazon Kindle e il software utilizzano file in formato AZW1 con estensione .azw1. Questi file sono anche noti come file eBook Amazon Topaz, questi file AZW1 vengono utilizzati per assemblare e gestire parti scaricate di file eBook Amazon Kindle dalla rete **3G Whispernet** di Amazon. Il software Amazon Kindle può essere installato su un sistema basato su **Microsoft Windows** per eseguire il supporto per questi file AZW1 e le relative funzionalità integrate nell'applicazione. Sono disponibili anche le versioni di questo programma per ambienti Mac. Questo software viene solitamente utilizzato per iniziare a scaricare e visualizzare il contenuto di questi libri Amazon Kindle. Poiché l'applicazione Amazon Kindle fornisce supporto per i file AZW1 e può essere installata anche in questi due noti sistemi, i creatori possono facilmente creare e condividere correttamente i libri Kindle.
