## Che cos'è un file BKK?

I file con estensione .bkk vengono utilizzati dal software **BookBuddi** e questi file possono essere riletti utilizzando Microsoft Text to Speech Agent. Questi file possono includere testo, immagini e possono essere costituiti da più pagine. Durante la lettura del testo la rispettiva sezione viene evidenziata e le pagine vengono girate automaticamente.

## Formato file BKK

Il contenuto del formato di file BKK è referenziato dall'app BookBuddi quando un utente visualizza o modifica il file BKK associato. Questi dettagli di riferimento includono anche attributi di formattazione di testo e immagine tra gli altri elementi di layout di pagina inseriti dal creatore dell'ebook BKK. L'agente di sintesi vocale Microsoft, un'applicazione che appartiene ai sistemi, è utile per aiutare gli utenti ad accedere al contenuto digitale archiviato in questi ebook BKK poiché questa applicazione supporta una funzione che rilegge il contenuto di testo di un eBook in audio parlato.
