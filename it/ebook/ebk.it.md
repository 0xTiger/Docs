## Che cos'è un file EBK?
Un'estensione EBK viene utilizzata dal software **eBook Pro Viewer** quando salva un file eBook. eBook Pro è un software in grado di creare eBook e proteggere le informazioni protette da copyright. Un file EBK contiene la versione crittografata e compressa di un e-book che può essere letto utilizzando **eBook Pro Viewer**; dopo aver fornito le credenziali appropriate per l'autenticazione.
