## Che cos'è un file MBP?

Un file MBP è noto come file **Mobipocket Notes** ma non contiene l'eBook stesso; contiene le note dell'e-book che vengono create dagli utenti in un e-book come correzioni, annotazioni, disegni e altri segni importanti. I file MBP si riferiscono semplicemente alle note che gli utenti hanno aggiunto durante la lettura di un determinato eBook. In altre parole, possiamo dire che gli eBook consentono agli utenti di aggiungere alcune note dove vogliono durante la lettura e quelle note contrassegnate vengono solitamente salvate utilizzando l'estensione **.mbp**.

## Breve spiegazione sul file Mobipocket Notes

Il paragrafo precedente mostra che Mobipocket Notes facilita agli utenti l'aggiunta di alcune note ove necessario e queste note salvate in un file MBP. Poiché questi file sono interamente associati a un file eBook in quanto sono contenuti all'interno di un eBook, di solito vengono salvati anche con l'e-book e si trovano in una posizione simile in cui viene salvato il file eBook. Questi file sono generalmente prodotti dal programma **MBP Reader** o **Mobipocket Reader Desktop**, un'applicazione utile per leggere libri in formato elettronico. I file MBP vengono salvati in formato binario e possono essere aperti utilizzando un'utilità di lettura MBP.
