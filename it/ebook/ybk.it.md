## Che cos'è un file YBK? ##

Il formato di file eBook YanCEyWare e Microsoft Encarta utilizzano entrambi estensioni di file YBK. YanCEyWare eBook è stato formato da YanCEyWare. Tutte le applicazioni software sono disponibili per dispositivi che eseguono Windows, supportano i file YBK. L'Encarta Encyclopedia può essere aggiornata da file .ybk senza dover aggiornare l'intero programma. YanCEyDesktop è un programma noto e popolare in grado di gestire e gestire facilmente i file YBK. YanCEyWare Reader utilizza ampiamente questo formato di file YBK, come applicazione per la lettura di eBook per Windows Mobile o Pocket PC. Questo file è stato sviluppato con grandi risorse con collegamenti ipertestuali.

## Problemi per aprire un file YBK ##

A volte, quando è difficile aprire e far funzionare il file YBK, ecco l'elenco di alcuni possibili problemi:

* Apertura di un file YBK corrotto
* Collegamenti inappropriati e impropri al file YBK negli accessi al registro
* Rimozione involontaria della descrizione dell'YBK dall'archivio di Windows
* Installazione parziale di un'applicazione che prevede il formato YBK
*	Infected YBK file with an undesirable malware
