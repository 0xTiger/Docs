## Che cos'è un file CBZ?

Un file con estensione .cbz è un file di archivio di fumetti [ZIP](/it/compression/zip/) che è una raccolta di immagini, che rappresentano le pagine di un fumetto. CBZ è diverso da altri file di ebook che non sono compressi. La maggior parte dei lettori di ebook e fumetti supporta la visualizzazione di questi file. [CBR](/it/ebook/cbr/) è un altro formato di file di archivio di fumetti che utilizza la compressione [RAR](/it/compression/rar/) invece di ZIP. I file CBZ e CBR possono essere contenuti all'interno di file di raccolte di fumetti come [CBC](/it/ebook/cbc/). Le applicazioni in grado di aprire e leggere file CBZ includono app software come Calibre e Comic Book Reader.

## Formato file CBZ

Un file CBZ è un archivio ZIP che contiene pagine di ebook come immagini nei formati di file [PNG](/it/image/png/), [JPEG](/it/image/jpeg/) o [GIF](/it/image/gif/). Può essere decompresso e il suo contenuto può essere estratto utilizzando qualsiasi software di decompressione che supporti il formato di file ZIP. Le immagini estratte utilizzando il software di decompressione non sono nello stesso ordine in cui appaiono nel file CBZ poiché le informazioni sull'ordine delle immagini si trovano all'interno del file CBZ.
