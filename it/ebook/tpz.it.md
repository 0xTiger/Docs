## Che cos'è un file TPZ? ##

L'estensione del file .tpz è anche conosciuta come Kindle Topaz eBook Format da Amazon. Linux, Mac OS e Windows sono alcune delle piattaforme in cui possono essere distribuiti file con estensione .tpz. Come file di eBook Kindle Topaz, il file TPZ è formattato per assomigliare agli ebook tradizionali e contiene standard di compilazione dei dati proprietari di Amazon per il formato TPZ. Questo file ha un supporto avanzato per i caratteri incorporati e dettagli aggiuntivi come annotazioni e segnalibri possono anche essere inclusi nei file eBook Kindle Topaz, sebbene questi dettagli sui metadati siano archiviati separatamente. Il file TPZ può anche essere classificato come un file di dati che può essere scaricato dal database online di Amazon di ebook Kindle.

## Problemi con l'apertura di un file TPZ? ##

Ci sono alcuni dei problemi comuni qui menzionati di seguito che possono sorgere:

* File infetto. Questo può essere risolto tramite un software antivirus adeguato per renderlo pulito
* Spazio su disco insufficiente. Per questo, deve chiudere tutte le applicazioni irrilevanti per un migliore funzionamento
* I collegamenti interrotti a volte possono essere causa di problemi
* A volte il file non ha diritti di accesso, quindi in questo caso assicurati che i diritti di accesso aprano facilmente il file
