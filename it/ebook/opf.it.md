## Che cos'è un file OPF? ##

L'OPF sta per Open Package Format. Questo formato di file è in realtà un file [XML](/it/web/xml/) ma con estensione **.opf**. Questo formato di file è un OEB Package Format (OPF) che potrebbe contenere una rivista digitale o un'altra pubblicazione che può essere letta con **FlipViewer**. Include anche metadati che descrivono il contenuto e un elenco di pagine, immagini e testo a cui fa riferimento la pubblicazione.

## Dettagli tecnici del formato file OPF ##

L'estensione del file .opf può essere classificata in vari tipi di file. Può essere un file formattato con lo standard impostato da Open Packaging. Gli OPF sono più che file di eBook in quanto contengono principalmente informazioni sull'eBook dal titolo allo scrittore, nonché altre meta-informazioni su di esso. Questo è il motivo per cui questi file possono essere aperti utilizzando il software Adobe Reading. Inoltre, viene utilizzato per i file creati dal programma FlipAlbum. Questo programma viene utilizzato per creare album fotografici digitali e questi file OPF si riferiscono all'album fotografico creato utilizzando il software. Questi file contengono le configurazioni dell'album; la sua struttura e i riferimenti ai file immagine e possono essere condivisi anche con altre persone. I file OPF sono anche associati a **FlipBook**. Un file OPF può essere modificato con qualsiasi editor di testo (ad es. Notepad o Notepad++) perché possono essere formattati utilizzando il formato XML.
