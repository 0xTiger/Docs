## Che cos'è un file AEP? ##

Il file con estensione .aep (che sta per **Activ E-Book Project**) è classificato come libro elettronico (eBook). I file AEP vengono creati utilizzando lo stesso motore di Internet Explorer per garantire la compatibilità al 100%. Contiene le pagine come una persona può leggere un libro fisico. Le pagine dei file AEP di solito includono testo, immagini, animazioni e collegamenti ipertestuali. Inoltre, supporta grafica e animazioni [HTML](/it/web/html/), [DHTML](/it/web/dhtml/), [JPEG](/it/image/jpeg/) e [PNG](/it/image/png/) , [JavaScript](/it/web/js/), VBScript e molti dei plug-in di Internet Explorer. Un file AEP può essere salvato come file eseguibile di Windows (EXE) in modo che possa essere aperto su Windows95 o versioni successive di Microsoft Windows.
 


## Formato file AEP ##

I file in formato AEP vengono creati utilizzando un software AEC (Active E-Book Compiler). Questo software utilizza un motore simile a quello di IE per creare eBook. L'utente può creare copie illimitate di eBook senza pagare alcun costo per l'utilizzo del programma. Anche il processo di distribuzione è semplice poiché ogni eBook è composto da un programma Windows autonomo, che può essere distribuito su qualsiasi supporto come e-mail, download da Internet, floppy disk, CD-ROM, dischi ZIP, ecc.

## Possibili problemi durante l'apertura del file

Se non riesci ad aprire ed eseguire il file AEP; non significa che non hai un software adatto installato sul tuo dispositivo. Potrebbero esserci altri problemi che impediscono al file di funzionare correttamente. I possibili problemi potrebbero essere uno dei seguenti:

- Corruzione di un file AEP
- Connessioni sbagliate del file AEPnelle iscrizioni del registro
- Eliminata la descrizione dell'AEP dal registro di Windows
- Installazione danneggiata di un'applicazione che supporta il formato AEP
- Un file AEP infetto con un malware indesiderato
- Il computer non dispone di risorse hardware sufficienti per utilizzare il file AEP
- I driver utilizzati dal computer per aprire un file AEP sono obsoleti
