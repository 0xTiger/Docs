## Che cos'è un file BNS?

Il file con estensione .snb; appartiene ai dispositivi mobili Samsung. Quando i telefoni Galaxy S Note creano una nota utilizzando un tipico software (applicazione avanzata per prendere appunti sviluppata da Samsung), quella nota viene effettivamente salvata come file con estensione .snb nella memoria degli smartphone (es. Galaxy S e Galaxy Note ). Un file di note può contenere schizzi, testo, video, audio e immagini. I file SNB sono stati sospesi da Samsung e questi file sono stati sostituiti da file [SPD](/it/ebook/snb/). Pertanto, potresti non trovare molte informazioni su questo tipo di file tipico su Internet.

## Problemi per aprire un file SNB

Se non riesci ad aprire ed eseguire il file SNB; non significa che non hai un software adatto installato sul tuo dispositivo. Potrebbero esserci altri problemi che impediscono al file di funzionare correttamente. I possibili problemi potrebbero essere uno dei seguenti:

- Corruzione di un fascicolo BNS.
- Connessioni sbagliate del file SNBnelle iscrizioni del registro.
- Eliminata la descrizione della BNS dal registro di Windows
- Installazione danneggiata di un'applicazione che supporta il formato SNB
- Un file SNB infetto da malware indesiderato.
- Il computer non dispone di risorse hardware sufficienti per gestire il file SNB.
- I driver utilizzati dal computer per aprire un file SNB sono obsoleti.
