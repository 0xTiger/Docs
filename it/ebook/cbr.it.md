## Che cos'è un file CBR?

Un file con estensione .cbr è una raccolta di immagini archiviate come file binari compressi utilizzando il formato di file di compressione RAR. Contiene immagini in [JPEG](/it/image/jpeg/), [PNG](/it/image/png/) o [GIF](/it/image/gif/) che rappresentano immagini in un fumetto e vengono mostrate all'utente quando il viene utilizzato nei lettori di ebook come Calibre. CBR è simile a [CBZ](/it/ebook/cbz/) a parte il fatto che CBZ è compresso e archiviato nel formato file [ZIP](/it/compression/zip/). Questi file possono essere contenuti all'interno di file di raccolta di fumetti come [CBC](/it/ebook/cbc/).

## Formato file CBR

Un file CBR è un archivio [RAR](/it/compression/rar/) memorizzato come file binario sul disco. Le immagini all'interno di un file CBR vengono archiviate e visualizzate agli utenti in modo ordinato quando vengono aperte da un lettore di ebook. I file CBR possono essere decompressi utilizzando qualsiasi software di decompressione che supporti il formato di file RAR. Tuttavia, le immagini decompresse in questo modo non sono nello stesso ordine in cui appaiono sul display visivo quando vengono aperte nel software del lettore.
