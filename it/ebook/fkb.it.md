## Che cos'è un file FKB? ##

FKB è un'estensione di file eBook che è stata sviluppata da Flipkart.com per l'applicazione software Flipkart. Questo tipo di estensione di file si trova solitamente sugli utenti di Windows 10. C'è una specifica nel file FBK che può variare in dimensioni da un paio di centinaia di kilobyte a numerosi megabyte a seconda della lunghezza del libro e del numero di immagini incluse. Per scaricare il file dall'applicazione Flipkart eBooks per Android, è necessario prima acquistare l'eBook, quindi fare clic su di esso nella Libreria e il libro inizierà a scaricarsi. Se non riesci a vedere il documento acquistato, aggiorna il sistema per un migliore funzionamento.

## Sistemi operativi supportati ##

I seguenti sistemi operativi supportano i file FKB:

* Windows 7
* Windows 8
* Windows 10
* Windows Server 2003/2008/2012/2016
*Mac OS X
* FreeBSD
* iOS
*Linux
* Androide
* NetBSD
* OpenBSD

## Problemi per aprire un file FKB ##

* Installazione impropria di programmi di supporto. Assicurati che sia stato scaricato correttamente con una versione aggiornata
* Spazio hardware ridotto
* File danneggiati o infetti. Questo problema può essere facilmente risolto tramite un antivirus installato nel tuo sistema
