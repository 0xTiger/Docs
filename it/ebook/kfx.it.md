## Che cos'è un file KFX?

Un file KFX è un eBook creato in Amazon Kindle File Format 10 (KF10) per dispositivi Kindle. Le versioni precedenti dei dispositivi KIndle utilizzano i formati di file [AZW](/it/ebook/azw/) e [AZW3](/it/ebook/azw3/) (KF8). KFX è un miglioramento rispetto alle versioni precedenti con funzionalità aggiuntive specifiche in termini di presentazione del layout dei contenuti. Introduce nuove funzionalità come il supporto per i trattini morbidi utilizzando un nuovo motore di composizione migliorato. Introduce anche un nuovo formato di file immagine chiamato JXR, che ha un rapporto di compressione più elevato, e il nuovo font Bookerly che è un font esclusivo per Kindle che offre una migliore leggibilità sui dispositivi digitali. Il formato viene fornito con funzionalità avanzate come miniature multipagina, regolazione automatica del contenuto attorno a immagini e video in posizioni fisse con conseguente miglioramento del layout della pagina e DRM. Per aprire e visualizzare tali file, puoi utilizzare l'app Amazon Kinde disponibile per Windows, macOS, Android e iOS. Calibre è un'altra applicazione software per la lettura di eBook che può essere utilizzata per leggere diversi formati di file di eBook.

## Formato file KFX ##

KFX è un formato complesso in quanto combina diverse funzionalità dei formati di file Kindle, ma i dettagli interni sulla struttura del file non sono noti. Amazon ha aggiunto il suo schema DRM per proteggere i diritti digitali e salvare il formato KFX dall'antipirateria.
