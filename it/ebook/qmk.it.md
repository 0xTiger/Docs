## Che cos'è un file QMK?

Un file con estensione .qmk è generalmente noto come formato **YanCEyDesktop Quickmarks**. YanCEyWare ha introdotto il formato di file YanCEyDesktop Quickmarks. Il file QMK è correlato al file [YBK](/it/ebook/ybk/) e supportato da un programma disponibile per le macchine che eseguono il sistema operativo Windows. Un file con estensione QMK è classificato come un file di tipo testo che consente agli utenti di navigare facilmente nelle posizioni salvate all'interno degli eBook. YanCEyDesktop è il software consigliato per la gestione dei file QMK. Il software **YanCEyDesktop** è stato realizzato da **D Remington**.

## Problemi per aprire un file QMK

Se non riesci ad aprire ed eseguire il file QMK; non significa che non hai un software adatto installato sul tuo dispositivo. Potrebbero esserci altri problemi che impediscono al file di funzionare correttamente. I possibili problemi potrebbero essere uno dei seguenti:

- Corruzione di un file QMK
- Connessioni sbagliate del file QMKnelle iscrizioni del registro
- Eliminata la descrizione del QMK dal registro di Windows
- Installazione danneggiata di un'applicazione che supporta il formato QMK
- Un file QMK infetto da malware indesiderato
- Il computer non dispone di risorse hardware sufficienti per utilizzare il file QMK
- I driver utilizzati dal computer per aprire un file QMK sono obsoleti
