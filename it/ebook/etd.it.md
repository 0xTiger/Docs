## Che cos'è un file ETD? ##

L'estensione del file ETD è correlata ai file Adobe Reader EBX Transfer Data Format. Questa estensione di file è generalmente ben abbinata con software che può essere scaricato su piattaforme di sistema Linux, Mac OS, Windows. È un formato di file molto conosciuto e utilizzato come formato di alcuni eBook scaricabili dalla Rete. Ma chiarisci che in questo formato è possibile salvare i file eBook e il loro contenuto. Questo può essere utilizzato solo durante il trasferimento o il processo di download. Il principale vantaggio di questo formato è che permette di scaricare eBook nel computer e di leggerli o accedervi online. Principalmente è associato all'eBook, quindi funziona efficacemente con applicazioni come Adobe Reader e Acrobat.

## Problemi ad aprire un file ETD? ##

Se riscontri problemi per aprire un file ETD, controlla i possibili errori relativi all'estensione del file, indicati di seguito.

* Estensione del file ETD danneggiato. Per risolvere questo problema, scarica un antivirus
* Collegamenti interrotti al file ETD nell'archivio. Controllare i collegamenti prima di aprire
* Rimozione della descrizione dell'ETD dall'archivio di Windows
* Installazione non corretta di un'applicazione che supporta il formato ETD
* File infetto
* Versione obsoleta del software. Installa la versione aggiornata compatibile con il corretto funzionamento dell'estensione del file ETD
