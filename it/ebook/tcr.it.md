## Che cos'è un file TCR?

All'inizio degli anni '90, il formato file TCR è stato introdotto per gli antichi dispositivi palmari **Psion Series 3**. L'azienda ha utilizzato la compressione specifica della piattaforma e questo formato ha un supporto molto limitato per la formattazione del testo. È un formato di file basato su ZVR e fornisce un tasso di compressione relativamente migliore rispetto a PalmDOC. Funziona molto bene anche con FontMachine. Sebbene il formato del file TCR appartenga a un dispositivo fuori produzione, è comunque possibile leggere questo formato di file utilizzando alcuni eReader. Questo formato di file può essere visualizzato anche su un desktop con Sumatra PDF e con l'applicazione Calibre in Windows e Linux.

## Dettagli tecnici

Poiché il formato file TCR si basa su un compressore ZVR che è in grado di ridurre molti file di circa il 50% della loro dimensione originale. Lo schema di compressione utilizzato da over è un po' vecchio. Il file compresso inizia con un dizionario dei 256 possibili simboli che potrebbero essere visualizzati nel file compresso. Il dizionario contiene una stringa alternativa fissa per ciascuno di questi simboli. La decompressione è diventata molto veloce anche in OPL e può iniziare in qualsiasi punto del file compresso.

## Problemi per aprire un file TCR ##

Se non riesci ad aprire ed eseguire il file TCR; non significa che non hai un software adatto installato sul tuo dispositivo. Potrebbero esserci altri problemi che impediscono al file di funzionare correttamente. I possibili problemi potrebbero essere uno dei seguenti:

- Corruzione di un file TCR.
- Collegamenti sbagliati al file TCRnelle iscrizioni del registro.
- Eliminata la descrizione del TCR dal registro di Windows
- Installazione danneggiata di un'applicazione che supporta il formato TCR
- Un file TCR infetto con malware indesiderato.
- Il computer non dispone di risorse hardware sufficienti per utilizzare il file TCR.
- I driver utilizzati dal computer per aprire un file TCR sono obsoleti.
