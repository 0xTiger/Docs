## Che cos'è un file ACSM?

I file con estensione .acsm sono chiamati file di messaggi di Adobe Content Server. Questi file vengono utilizzati da **Adobe Digital Editions** per attivare e scaricare contenuti protetti da Adobe DRM. In realtà, un file ACSM non è un file eBook; non può essere aperto e letto come altri eBook (es. PDF); Contiene solo i dati per attivare e scaricare un ebook. Significa che un file ACSM è costituito solo dalle informazioni che comunicano con i server di Adobe. Gli utenti delle edizioni digitali possono vedere il file ACSM se scaricheranno un eBook, ma il download si sarebbe interrotto per qualsiasi motivo. Gli utenti possono riprendere il download facendo doppio clic sul file .acsm.

## Come funziona un file ACSM?

Poiché Adobe Content Server è responsabile della protezione degli eBook e di altri contenuti digitali. Dopo l'acquisto, il contenuto viene automaticamente collegato all'Adobe ID dell'acquirente. L'ID è privato e non può essere trasferito ad altri. Un utente deve configurarlo prima di acquistare qualsiasi documento con protezione dalla copia di Adobe. I clienti non possono accedere ai propri documenti protetti da copia senza passare il proprio Adobe ID all'applicazione server Adobe in esecuzione in background.

Quando i clienti effettuano l'acquisto, l'ACSM è l'unico file che possono inizialmente scaricare. Il cliente deve aver installato il software Adobe Digital Editions sul proprio sistema e averlo collegato al proprio Adobe ID per aprire il file ACSM. Adobe Digital Editions verificherà l'ID per l'autorizzazione a convertire il file ACSM. Se autenticato, l'utente può scaricare il proprio eBook in formato [EPUB](/it/ebook/epub/) o [PDF](/it/pdf/). Adobe Digital Editions utilizza anche il file ACSM per riconoscere la directory di download.
