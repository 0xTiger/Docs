## Che cos'è un file POBI?

Un file POBI appartiene al tipo di file eBook; è in realtà un **Dati periodici Kindle Touch**. Questo formato di file è anche noto come periodico [MOBI](/it/ebook/mobi/); utilizzato nel dispositivo di lettura elettronica Amazon Kindle Touch. Sebbene il tipo di file dei file POBI sia un eBook, non possiamo dire che il file con estensione .pobi sia un eBook; perché non ha il contenuto come un eBook. Contiene solo i dati come una rivista e funziona come un file MOBI. Tuttavia, se l'estensione .pobi verrà modificata in .mobi, le sezioni e l'indice degli articoli della rivista non verranno visualizzati.

## Problemi per aprire un file POBI ##

Se non riesci ad aprire ed eseguire il file POBI; non significa che non hai un software adatto installato sul tuo dispositivo. Potrebbero esserci altri problemi che impediscono al file di funzionare correttamente. I possibili problemi potrebbero essere uno dei seguenti:

- Corruzione di un file POBI.
- Connessioni sbagliate del file POBInelle iscrizioni del registro.
- Eliminata la descrizione del POBI dal registro di Windows
- Installazione danneggiata di un'applicazione che supporta il formato POBI
- Un file POBI infetto con malware indesiderato.
- Il computer non dispone di risorse hardware sufficienti per utilizzare il file POBI.
- I driver utilizzati dal computer per aprire un file POBI sono obsoleti.
