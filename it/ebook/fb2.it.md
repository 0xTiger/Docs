## Che cos'è un file FB2?

I file con estensione .fb2 sono file eBook FictionBook 2.0 che contengono la struttura dell'eBook. Si basa sul formato XML e contiene tag speciali per descrivere ogni elemento del libro. È stato sviluppato principalmente per scritti e letteratura di fantasia, ma non si limita solo a questi. Il formato contiene tutti i metadati e il contenuto in sé e consente flessibilità per una serie di operazioni come l'elaborazione automatica, l'indicizzazione e la conversione in altri formati. In breve, si concentra sulla descrizione della struttura del file invece di specificarne l'aspetto. Sono disponibili diverse applicazioni e API per convertire FB2 in molti altri formati su Windows, MacOS e Linux.

## Formato file FB2

Il formato di file FictionBook 2.0 è stato sviluppato da Dmitry Gribov di FictionBook in Russia. A differenza di altri formati di file Ebook popolari come [EPUB](/it/ebook/epub/) che contengono più file XML, si basa su un unico file XML che contiene informazioni complete sui contenuti. Se sono presenti immagini all'interno del file come [PNG](/it/image/png/) o [JPG](/it/image/jpeg/), queste vengono convertite in Base64 e incorporate nel documento. Le [specifiche del formato file](http://gribuser.ru/xml/fictionbook/index.html.en) per FB2 contengono informazioni per la versione 2.0 del formato.
