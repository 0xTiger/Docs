## Che cos'è un file HTML0? ##

HTML0 è il formato eBook nativo utilizzato da **Book Designer** con codice HTML e CSS standard che descrive il contenuto e il design di ogni pagina. **Book Designer** è il programma più comune che funziona con file HTML0 e per modificare documenti di testo in numerosi formati di eBook. Il formato file HTML0 supporta il software che può essere installato sul sistema Windows in quanto appartiene alla categoria dei file di testo. I file HTML0 comprendono un normale HTML e possono essere visualizzati e modificati utilizzando un editor di testo. I file HTML0 non possono essere collegati a file CSS discreti rispetto ai normali file HTML.
