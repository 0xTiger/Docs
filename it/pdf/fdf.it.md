## Che cos'è un file FDF?

Un file FDF (**Forms Data Format**) è un documento di testo generato esportando i dati dai campi modulo di un file [PDF](/it/pdf/). Include solo i dati dei campi di testo estratti dai campi del modulo disponibili in un file PDF. Ciò si traduce in un file di dati relativamente piccolo poiché i dati esportati non contengono il modulo stesso. Adobe Acrobat fornisce la funzione di esportazione dei dati dei campi modulo utilizzando l'opzione "Esporta dati moduli" dal menu dell'applicazione.

## Formato file FDF - Ulteriori informazioni

FDF è in formato testo normale ed è incluso come parte dello [standard ISO 32000](https://www.iso.org/standard/51502.html) per Portable Document Format. È stato sviluppato da Adobe per consentire l'importazione e l'esportazione di dati da Acrobat Forms o AcroForms.

Esistono due tipi di file FDF:

• `Classic FDF` – Fornisce i dati per compilare un modulo statico esistente.

• `Modello FDF` – Costruisce un nuovo PDF basato su modelli da file PDF specificati. I moduli all'interno del nuovo documento vengono compilati fornendo i dati.

## Creazione di FDF con Adobe Acrobat

Il [toolkit FDF](https://opensource.adobe.com/dc-acrobat-sdk-docs/) di Adobe ti consente di creare file FDF da dati di testo.
