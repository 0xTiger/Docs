## Cos'è un file ENC?

Un **file ENC** è correlato alla **Carta elettronica di navigazione (ENC)** che è una carta vettoriale digitale utilizzata nei sistemi di cartografia elettronica per la navigazione marina; I file ENC contengono dati geospaziali, come batimetri, aiuti alla navigazione e altre informazioni necessarie per una navigazione sicura. ENC è un formato standardizzato definito dalla **Organizzazione Idrografica Internazionale (IHO)** nello **standard S-57**.

Lo standard S-57 specifica la struttura e il contenuto dei file ENC; questi file hanno in genere l'estensione .enc. I file ENC sono utilizzati da **Electronic Chart Display and Information Systems (ECDIS)** sulle navi per la navigazione; I sistemi ECDIS utilizzano i dati ENC per visualizzare carte elettroniche, fornendo ai navigatori informazioni in tempo reale per facilitare una navigazione sicura ed efficiente.

## Formato file ENC: ulteriori informazioni

I **file ENC** sono **file binari** che memorizzano le informazioni in modo strutturato, consentendo un'archiviazione e un recupero efficienti dei dati di navigazione; il contenuto dei file ENC include informazioni su linee costiere, batimetriche, boe, luci e altri elementi di navigazione; i dati sono organizzati in vari livelli dati e ciascuna caratteristica è rappresentata come un oggetto vettoriale con attributi associati.

I file ENC vengono regolarmente aggiornati dagli uffici idrografici per riflettere i cambiamenti nelle informazioni di navigazione, garantendo che i marinai abbiano accesso ai dati più aggiornati e accurati per le loro rotte.

## Informazioni sullo standard S-57

Lo **standard S-57** è una specifica internazionale per la codifica, l'archiviazione e lo scambio di carte nautiche elettroniche (ENC); è stato sviluppato dall'Organizzazione Idrografica Internazionale (IHO) per fornire un formato standardizzato per le carte digitali utilizzate nell'ECDIS (Electronic Chart Display and Information Systems) per la navigazione marittima.

## Come aprire il file ENC?

Per aprire file ENC (Electronic Navigation Chart), in genere è necessario un software di navigazione specializzato che supporti lo standard S-57 e il formato file ENC.

Alcuni popolari software di navigazione includono:

-OpenCPN
-MaxSea
- Mare limpido
-Transas iSailor
-TEMPO ZERO
