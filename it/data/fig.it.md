## Cos'è un file FIG?

Un file FIG è un file speciale che contiene un'immagine o un grafico realizzato utilizzando il programma MATLAB, che viene utilizzato per eseguire calcoli e analizzare dati. Questo file contiene informazioni su immagini o grafici che MATLAB disegna per aiutare le persone a vedere e comprendere meglio i dati. È come un contenitore che conserva tutti i dettagli sui grafici.

## Come creare file FIG con MATLAB?

Per creare un file FIG utilizzando MATLAB, attenersi alla seguente procedura:

1. **Seleziona i tuoi dati:** Scegli la variabile per cui desideri creare un grafico dal riquadro "Area di lavoro" in MATLAB.

2. **Scegli il tipo di trama:** Vai alla scheda "PLOTS" e fai clic sul tipo di trama che desideri.

3. **Salva il grafico come file FIG:**

     - Vai al menu "File".
     - Scegli "Salva" o "Salva con nome".
     - Fornisci un nome al tuo file e decidi dove vuoi salvarlo.
     - Fai clic su "Salva" per creare il tuo file FIG.

## Come aprire un file FIG?

Per aprire il file FIG in MATLAB, attenersi alla seguente procedura:

1. **Apri MATLAB:** avvia MATLAB sul tuo computer.

2. **Vai alla scheda "PLOTS":** Fai clic sulla trama nella scheda "PLOTS", selezionando il tipo di trama desiderata.

3. **Apri il file FIG:**

     - Vai al menu "File".
     - Scegli "Apri".
     - Passa alla posizione in cui è salvato il file FIG, selezionalo e fai clic su "Apri".
