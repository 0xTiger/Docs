## Cos'è un file GED?

Un **file GEDCOM (Genealogical Data Communication)**, solitamente con estensione .ged, è un formato di file standard utilizzato per lo scambio di dati genealogici tra diversi programmi software di genealogia; I file GEDCOM sono file di testo semplice che contengono informazioni su individui, famiglie, eventi e altri dati genealogici; lo scopo di GEDCOM è fornire un formato comune per lo scambio di informazioni genealogiche.

I file GEDCOM di solito hanno l'estensione file .ged; questa estensione viene utilizzata per indicare che il file segue lo **standard GEDCOM** per lo scambio di dati genealogici; quindi quando salvi o esporti i tuoi dati genealogici da un programma software di genealogia, al file risultante viene spesso assegnato un nome come tuonomefile.ged.

Per esempio:

- tuoalbero genealogico.ged
- john_doe_family.ged
- ancestry_data.ged

## Punti chiave sui file GEDCOM

Ecco alcuni punti chiave sui file GEDCOM:

1.  **Formato testo semplice:** i file GEDCOM sono file di testo semplice; il che significa che possono essere aperti e visualizzati con qualsiasi editor di testo; questo formato facilita la condivisione e la compatibilità tra diverse applicazioni software di genealogia.
    
2.  **Struttura:** i file GEDCOM hanno una struttura gerarchica; i dati sono organizzati in record; ogni record ha un tag specifico che indica il tipo di informazioni che contiene, ad esempio il nome di una persona potrebbe essere rappresentato dal tag NAME.
    
3.  **Individuals and Families:** GEDCOM files typically include information about individuals such as names, dates of birth and places of birth; they also include details about family relationships, marriages and other events.
    
4.  **Interoperabilità del software:** Poiché GEDCOM è un formato standardizzato; consente agli utenti di spostare i propri dati genealogici tra diversi programmi software di genealogia; ciò è particolarmente utile per gli utenti che possono passare a diverse applicazioni software o condividere le proprie ricerche con altri.
    
5.  **Versione:** GEDCOM ha attraversato varie versioni e ciascuna versione introduce miglioramenti e nuove funzionalità; la versione più comunemente usata era GEDCOM 5.5. 

## Come aprire il file GED?

Per aprire un file GED, in genere è necessario un software di genealogia che supporti il formato GEDCOM. Alcune popolari opzioni del software di genealogia includono:

- **Creatore di alberi genealogici**
- **Ancestry.com**
- **La mia eredita**
- **Gramps** (gratuito e open source)
- **Albero genealogico preesistente**

Una volta scelto il software di genealogia, cerca un'opzione come Importa, Apri GEDCOM o Importa da file nel menu o nella barra degli strumenti.

Poiché i file GED vengono salvati in formato testo, puoi anche aprire i file con un editor di testo semplice, ad es

- **Bloc notes**
- **Blocco note++**
- **Modifica testo Apple**
- Qualsiasi editor di testo
