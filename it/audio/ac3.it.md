## Che cos'è un file AC3?

Un file con estensione .ac3 è un file Audio Codec 3, introdotto da Dolby Laboratories. È un formato audio che può contenere fino a sei canali di uscita audio. Il formato era stato originariamente utilizzato per l'audio, ma ora è utilizzato anche per altre applicazioni come trasmissioni HDTV, DVD, dischi Blu-ray e console di gioco. Le applicazioni che possono aprire i file AC3 includono Apple QuickTime Player, Microsoft Windows Media Player, Winamp, MPlayer e altri simili.

## Formato file AC3

I file AC3 sono di natura binaria e si basano su Modified Discrete Cosine Transform (MDCT), che è un algoritmo di compressione con perdita di dati. Dolby Laboratories ha utilizzato l'algoritmo MDCT insieme ai principi di codifica percettiva per sviluppare il formato audio AC-3. Ciò ha portato al rilascio del formato AC-3 come standard Dolby Digital nel 1991. Il formato supporta la fornitura di cinque canali per altoparlanti a gamma normale (20 Hz - 20.000 Hz) e un canale (20 Hz - 120 Hz) per gli effetti a bassa frequenza pilotati dal subwoofer. AC3 supporta frequenze di campionamento fino a 48 kHz.

### Dettagli tecnici del formato file AC3

La tecnologia Dolby Digital utilizza un modo molto efficiente per comprimere le dimensioni dei file audio multicanale senza compromettere la qualità del suono. Questa compressione porta a una dimensione del file più piccola che semplifica la distribuzione dei file audio. Utilizzando Dolby Digital, è possibile includere un mix audio completo a 5.1 canali su una stampa di film o su un DVD.

#### Specifiche
Le specifiche Dolby Digital sono le seguenti.

|Campo|Specifica|
---|---|
|Canali |da 1.0 a 5.1, discreti|
|Velocità dati DVD, audio a 5.1 canali| 384 o 448 kbps|

|Supporta i metadati Dolby |Sì|

|Capacità di missaggio/streaming|Sì|
