## Che cos'è un file DSS?
Un file DSS (Digital Speech Standard) è un formato di file audio digitale presentato dalla **International Voice Association**, un'impresa combinata di Philips, Olympus e Grundig Business Systems. È un codec piuttosto scadente e vecchio. Il formato di file DSS salva i dati audio della voce in un formato compresso che consente funzionalità di registrazione di base, come riproduzione, registrazione, ecc. La sua qualità è mantenuta relativamente inferiore all'MP3 per ridurre al minimo le dimensioni del file.

## Formato file DSS

Il formato di file DSS è adatto per la dettatura grazie alla sua capacità di registrare in sovrascrittura o inserimento. Un file con estensione .dss può essere scritto e salvato con metadati, come la lunghezza dell'audio, la data e l'ora della dettatura e la priorità. Il formato DSS è progettato in genere per la voce, simile a [MP3](/it/audio/mp3/) per la musica. La sua qualità è mantenuta relativamente inferiore a quella di MP3 per ridurre al minimo le dimensioni del file.

Il formato di file DSS ha molti vantaggi grazie al suo formato altamente compresso come l'invio del file .dss ovunque tramite Internet in modo più rapido. La dimensione più piccola è utile quando si ha a che fare con enormi quantità di file che devono essere elaborati. Sebbene il formato di file DSS abbia una serie di vantaggi, ma in termini di qualità e compatibilità con vari software e dispositivi, molte persone desiderano convertire i propri file DSS in MP3.
