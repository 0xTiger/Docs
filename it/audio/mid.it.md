## Che cos'è un file MID?

Un file con estensione .MID o .MIDI è il file di interfaccia MID (Musical Instrument Digital). È la differenza dai normali file audio come [WAVs](/it/audio/wav/) o [MP3s](/it/audio/mp3/) in quanto non trasporta il contenuto audio effettivo; quindi, è di dimensioni piuttosto ridotte. Il contenuto include le note suonate, la loro sincronizzazione, durata e volume desiderato per ciascuna nota.

Il file MID consiste fondamentalmente in informazioni che spiegano come dovrebbe essere ascoltato il suono quando è collegato al dispositivo di riproduzione o caricato in un software specifico che interpreta i dati audio. Ciò lo rende uno strumento ideale per condividere dati musicali tra app compatibili e su una connessione a larghezza di banda ridotta. Le dimensioni ridotte consentono una facile memorizzazione per la quale possono essere utilizzati anche floppy disk.

### Breve storia del formato file MID

Il primo strumento, il Roland Jupiter-6 e il Prophet 600, furono introdotti con il MIDI nel 1982. La prima drum machine MIDI, la Roland TR-909 uscì nel 1983. Successivamente, il primo sequencer MIDI, il Roland MSQ-700 fu rilasciato. NEC PC-88 e PC-98, rilasciati nel 1982, furono i primi computer a supportare il MIDI.

## Specifiche del formato file MID

* Di piccole dimensioni: spesso fino a 10 KB per una riproduzione di un minuto
* Può regolare nota, tempo e intonazione per la performance desiderata
* Si compone di tre parti: il connettore fisico, il formato del messaggio e il formato di archiviazione
* Può essere convertito in MP3, WAV, WMA, FLAC, OGG, [AAC](/it/audio/aac/), MPC su Windows tramite Total Audio Converter.
