## Che cos'è un file WV? ##

I file WV sono file audio compressi utilizzando la compressione WavPack Hybrid. Sia le compressioni lossless che quelle lossy sono supportate da WavPack. Questo formato audio offre funzionalità senza precedenti. I file WV contengono audio digitali che vengono compressi da WavPack Audio.

La modalità lossless è perfetta per l'archiviazione di dati audio e la compressione dei dati senza alcun degrado e perdita di un po' sono le capacità di Wavpack. Ha un'interfaccia di libreria open source insieme a una licenza illimitata. Questi file sono per lo più relativi ai compressori Sonarc. Questi file audio compressi WavPack hanno dal 30 al 70 percento della dimensione del file originale.

WV supporta la compressione audio ibrida. Questi file compressi sono di dimensioni relativamente ridotte, di qualità superiore e possono essere utilizzati nello stesso formato (ovvero non è necessaria alcuna conversione per aprire e utilizzare ). I file WV originali sono il formato altamente compresso e con perdita di dati di una clip audio. L'aggiunta dei dati corretti dal file WVC può essere utilizzata per ripristinare completamente l'audio originale.

## Breve storia del file WV

A metà del 1998 la versione 1.0 di WavPack è stata sviluppata da David Bryant e si presentava come la migliore per efficienza e velocità. I file WV sono stati originati insieme alla prima versione. Nello stesso anno dopo la prima versione, è stata rilasciata un'altra versione modificata 2.0. Nel 1999 è stata sviluppata e rilasciata la versione 3.0. Conteneva la modalità veloce aggiornata e la possibilità di comprimere file audio grezzi.

## Specifiche del formato file WV

Il codice di WavPack è portatile e compatibile con Unix, Mac OS e Windows. Plugin per diverse applicazioni sono disponibili per i file WV. Può essere utilizzato su molte architetture come x86 e Motorola 68k. Questo tipo di compressione è compatibile con la maggior parte dei formati audio PCM.

Alcuni di questi sono registrazioni a 8,16 e 32 bit. A volte i file WVC sono inclusi come compagni con i file WV. Questi file audio sono compatibili per essere utilizzati su quasi tutti i lettori multimediali attualmente utilizzati insieme al requisito di un codec esterno di WavPack.

La modalità ibrida in WV risolve il problema di dover scegliere tra un file piccolo con bassa qualità e un file compresso con alta qualità. Invece di creare un singolo file, la modalità ibrida crea un file di alta qualità con una piccola dimensione che può essere facilmente utilizzato da solo e una combinazione di questo con un file con perdita, fornisce il file di "correzione". Questa combinazione offre la possibilità di eseguire un restauro completo senza perdite. Questa modalità ibrida funziona a circa 196 kbps nel caso di CD audio o 2,25 bit per campione.
