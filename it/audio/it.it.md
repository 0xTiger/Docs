## Cos'è un file informatico?

Il formato file IT è correlato al software Impulse Tracker, un popolare programma di tracciamento musicale sviluppato da Jeffrey Lim a metà degli anni '90. I file IT sono file di moduli che contengono pattern musicali, dati di esempio, impostazioni dello strumento e altre informazioni sulla sequenza.

Il formato file IT è un formato binario, noto per la sua flessibilità ed efficienza nel rappresentare composizioni musicali complesse. Supporta fino a 64 canali consentendo vari effetti come panning, diapositive del volume, vibrato e altro, supportando più formati di campione, sia campioni digitali che forme d'onda sintetizzate. I file IT sono stati ampiamente utilizzati nella scena demo e nella comunità musicale tracker, in particolare tra la fine degli anni '90 e l'inizio degli anni 2000.

## Come aprire il file IT?

Impulse Tracker (Windows) può essere utilizzato per aprire un file IT, tuttavia, poiché il software non è più sviluppato attivamente, esiste la possibilità che non sia compatibile con la versione del sistema operativo e potrebbe non funzionare come previsto.

Ecco alcune altre opzioni che possono aprire il file IT.

-OpenMPT per Windows
- Nullsoft Winamp per Windows
- Localizzatore di scismi
- Audacity con plugin di terze parti

## Come convertire un file IT?

OpenMPT e Schism Tracker sono esempi di programmi che offrono la capacità di convertire file IT in diversi formati di file. per esempio

- [MP3](/it/audio/mp3/) - File audio MP3
- [OPUS](/it/audio/opus/) - File audio Opus
- [MID](/it/audio/mid/) - File MIDI
- [WAV](/it/audio/wav/) - File audio WAVE
- [OGG](/it/audio/ogg/) - File audio Ogg Vorbis
- [FLAC](/it/audio/flac/) - File codec audio senza perdita di dati gratuito
