## Cos'è un file M4R?

Il formato file .m4r è un tipo di formato file audio utilizzato per le suonerie dell'iPhone. Il formato è essenzialmente una versione rinominata del formato contenitore multimediale MPEG-4 Parte 14 (MP4), con l'estensione del file .m4r che indica che il file è una suoneria. I file M4R sono codificati nel formato Advanced Audio Coding (AAC) e possono essere creati utilizzando varie applicazioni software che supportano la creazione di suonerie. Una volta creati, possono essere trasferiti su un iPhone utilizzando iTunes o altri metodi, e quindi impostati come suoneria del dispositivo. Sebbene i file M4R siano progettati per le suonerie dell'iPhone, il formato non è esclusivo dei dispositivi Apple e può essere riprodotto su altri dispositivi che supportano i file audio AAC.

## Formato file M4R: ulteriori informazioni

Il formato .m4r è stato introdotto da Apple nel 2007 con l'uscita dell'iPhone. Prima di ciò, le suonerie dell'iPhone erano nel formato .m4a, che è anche un formato audio AAC ma non è stato progettato specificamente per le suonerie. I file M4R hanno una durata massima di 40 secondi, che è la lunghezza massima consentita per una suoneria su un iPhone. I file M4R sono il formato utilizzato dall'iPhone per le suonerie.

I file M4R possono contenere sia contenuti audio che video, sebbene il contenuto video venga generalmente ignorato dall'iPhone quando il file viene utilizzato come suoneria. I file M4R vengono creati rinominando un file .m4a esistente in .m4r. Il contenuto del file non cambia, ma l'estensione .m4r dice a iTunes che il file deve essere trattato come una suoneria.

I file M4R possono essere creati utilizzando varie applicazioni software, incluso iTunes, che include un creatore di suonerie integrato. Per creare file M4R è possibile utilizzare anche applicazioni software di terze parti, come GarageBand e Ringtone Maker. I file M4R possono essere trasferiti su un iPhone utilizzando iTunes o sincronizzando il file sull'iPhone utilizzando servizi di archiviazione cloud come iCloud o Dropbox.
