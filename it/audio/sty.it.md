## Cos'è un file STY?

Il file STY si riferisce in genere a un formato di file utilizzato dalle tastiere elettroniche e dai sintetizzatori Yamaha. I file ".sty" Yamaha contengono dati di stile che definiscono il ritmo, i pattern di accompagnamento e altri elementi musicali utilizzati dallo strumento e vengono utilizzati per migliorare la riproduzione dei file MIDI fornendo stili e arrangiamenti preprogrammati.

Per utilizzare un file .sty con una tastiera o un sintetizzatore Yamaha, in genere trasferisci il file sullo strumento utilizzando una connessione USB o una scheda di memoria. Una volta caricato, lo strumento può quindi applicare i dati di stile per accompagnare la riproduzione MIDI o esibirsi dal vivo con lo stile selezionato.

## File STY Yamaha vs Korg

Yamaha e Korg salvano i file STY in formati diversi, il che significa che non sono multipiattaforma. Yamaha e Korg sono due importanti produttori di tastiere elettroniche e sintetizzatori e utilizzano i propri formati di file proprietari per i file di stile. Yamaha fornisce software dedicato per la modifica e la creazione di stili, come Style Fixer o Yamaha Expansion Manager. Korg fornisce strumenti software come GNIStyle o Style Works per modificare e creare stili per le tastiere Korg. Questi programmi consentono agli utenti di modificare gli stili esistenti o crearne di nuovi in base alle proprie preferenze.

## Come aprire il file STY?

È possibile utilizzare i seguenti software per aprire i file STY.

- Correzione dello stile (stili Yamaha)
- GNIStyle (stili Korg)
- WAVE Xtrattore
