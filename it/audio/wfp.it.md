## Che cos'è un file WFP?

Un file WFP era un file di patch utilizzato dalle schede del sintetizzatore WaveFront di Turtle Beach. Non è più utilizzato poiché Turtle Beach ha smesso di produrre schede audio e sintetizzatori. Il file WFP contiene un singolo programma o patch che può essere applicato al sintetizzatore. I file WFP possono anche essere convertiti nel formato file [MP4](/it/video/mp4/).

## Formato file WFP

I dettagli del formato file interno dei file WF non sono disponibili. È passato molto tempo da quando il lavoro sui file WFP è stato interrotto a causa della mancata ulteriore produzione di sintetizzatori da parte di Turtle Beach.
