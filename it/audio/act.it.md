## Che cos'è un file ACT? ##

Un formato ACT o Adobe Color Table (noto anche come file Color Lookup Table) è un tipo di database di Adobe Photoshop che memorizza un insieme di colori predefiniti. Puoi aggiungere o rimuovere colori quando salvi un'immagine per la pubblicazione online per favorire un'immagine di qualità superiore o una dimensione del file inferiore. Potresti avere un formato audio crittografato ADPCM piuttosto se non viene utilizzato con Illustrator. Adaptive Differential Pulse Code Modulation è una tecnica di compressione audio utilizzata da vari dispositivi musicali e dispositivi di ascolto.

Il formato file ACT è utilizzato anche dal sistema di file CAD/CAM Alma. Queste sono le linee guida utilizzate dalle macchine utensili 3D per capire come tagliare qualsiasi cosa. Un lettore Genesis3D, un'azione dello sviluppatore di giochi DS o un file schematico dell'azione della procedura guidata per i dettagli di FoxPro sono tutti esempi di ACT.

## Specifica del formato di file ACT

Molti registratori che utilizzano il sistema ACT lo fanno solo per l'opzione di registrazione più piccola; se la qualità viene aumentata, viene utilizzato un formato separato, solitamente WAV, che risulta in documenti enormemente più grandi.

Esistono diverse generazioni di ACT; a partire da giugno 2009, le tracce create dai dispositivi successivi non possono essere riconosciute da ogni lettore musicale standard gratuito o applicazione di conversione, lasciando solo gli strumenti MP3 in bundle.

L'ottimizzazione delle immagini per la distribuzione sul Web viene eseguita con i file ACT. I clienti possono testare la qualità dell'immagine rispetto ai database di immagini generati aumentando o diminuendo il numero di colori nel CLUT.

Toccando l'opzione Tabella colori e selezionando Carica dati colore dalla finestra di dialogo Esporta per Web, è possibile caricare le tabelle colori Adobe. La directory */Presets/en_US/Save* for Web Settings/Color Tables/ contiene diversi file ACT preconfigurati forniti con il sistema Photoshop.

Gli script Act possono essere generati esclusivamente all'interno di Genesis 3D o trasformando altri modelli 3D nello standard ACT utilizzando Genesis3D Actor Designer
