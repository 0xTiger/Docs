## Che cos'è un file SDT?

I file SDT vengono utilizzati come mezzo di archiviazione per i suoni di gioco, inclusa la musica di sottofondo e gli effetti sonori. Alcuni programmi, come Dragon UnPACKer e MAStudio 2002, possono aprire e copiare il contenuto audio dai file SDT. Il file SDT viene aperto solo da un software speciale. Esistono 3 tipi di file e ognuno di essi può essere aperto da un software diverso. Vecchi giochi rilasciati da Electronic Arts (ad esempio Dungeon Keeper 2 (DK2), Theme Park World, videogiochi della serie Formula One (F1)) file del pacchetto audio con una raccolta di file audio compressi che possono essere salvati utilizzando un MPEG-2 formato. Formati ed estensioni di file simili includono .sdsk, .sds, .sdr, .sdq, .sdv, .sdw, .sdx e .sdz.

## Formato file SDT

I file SDT sono in un formato di file proprietario e i dettagli del relativo formato di file non sono disponibili.
