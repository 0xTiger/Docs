## Che cos'è un file AA?

Viene specificato un file con estensione .aa per la memorizzazione del formato file di dati audio digitali che include il file dell'Audible AudioBook. Questi file vengono utilizzati per memorizzare registrazioni audio parlate di Audible. I file AA sono progettati per archiviare anche audiolibri. Questi file AA sono progettati per memorizzare la parola audio in forma parlata e supportare segnalibri, marcatori di capitoli, ecc. I dati in questi file includono grezzi ma vengono trasmessi nel formato della codifica audio. Insieme alla crittografia della gestione dei diritti digitali, questi file contengono clip audio.

## Formato file AA

Amazon trattiene l'udibile e questi file possono essere utilizzati su vari dispositivi. Per utilizzare questi file su Windows Media Player, è necessaria la conversione in un formato di file più efficiente come [MP3](/it/audio/mp3/). Il file AA è una categoria del file audio ma c'è una differenza tra un semplice file MP3 e un file udibile. Questi file possono essere protetti tramite DRM in modo che non possano essere convertiti in altri formati audio anche se ampiamente utilizzati.

### Breve storia del formato file AA

La società Amazon ha lanciato il suo prodotto noto come Audible player nel 1998. Audible è un servizio online americano che fornisce la struttura per audiolibri e podcast. Quindi questi file con estensione .aa hanno avuto origine insieme al lancio di questo servizio. Nel 2003 Audible ha stipulato un accordo con Apple. Ha la capacità di essere un fornitore esclusivo per i libri di iTunes. Nel 2008, Amazon ha affermato di acquistare Audible. Negli anni successivi è stato aggiornato e modificato in base alle moderne esigenze e ai servizi di compatibilità. Nel 2020, è stato modificato l'ultima volta quando Audible ha apportato alcune modifiche al pagamento.

## Specifiche del formato file AA

Questi file possono essere compatibili con vari dispositivi. I lettori per questi file sono disponibili su dispositivi Android e iOS. Alcune versioni di questi file non sono compatibili con Linux. I programmi associati a questi file sono disponibili online per PC Windows. Per Mac OS Apple iTunes è il programma associato per questi file. Questi file forniscono anche l'incapsulamento del suono codificato in MP3. Ci sono state diverse qualità disponibili da Audible, ma i download sono attualmente disponibili solo per i formati "Formato 4" e "Avanzato".
