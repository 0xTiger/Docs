## Cos'è un file ENC?

Un **file ENC** potrebbe anche rappresentare una composizione musicale realizzata utilizzando **Encore di Passport Music Software**, un'applicazione di scrittura di spartiti compatibile sia con le piattaforme Windows che macOS; all'interno di questo file scoprirai un ricco arazzo di notazione musicale, che comprende elementi come note, misure, indicazioni di tempo e indicazioni di tonalità. Inoltre, il file racchiude dettagli essenziali come il titolo della partitura e le informazioni sul compositore che ha dato vita al pezzo musicale.

## Informazioni sul software Encore

Encore, inizialmente sviluppato da Don Williams per Passport Designs Inc. nel 1984, è stato originariamente progettato per Apple Macintosh. Nel 1998, Lyrrus Inc., operante come GVOX, acquisì la proprietà intellettuale di Passport, portando al rilascio di Encore 5 un decennio dopo Encore 4.

Un aspetto pionieristico di Encore risiede nella sua distinzione come uno dei primi programmi di scrittura di partiture che consentiva agli utenti di aggiungere e modificare facilmente elementi all'interno della partitura musicale tramite l'uso del mouse; questa caratteristica ha segnato un progresso significativo nell'interfaccia utente del software di notazione musicale.

## Come aprire un file ENC?

Puoi aprire il file ENC relativo alla composizione musicale utilizzando questi software

- **Passport Music Software Encore** (a pagamento) per (Windows, Mac)
- **Myriad Harmony Assistant** (prova gratuita) per (Windows, Mac)

Utilizzando i software sopra indicati, puoi anche convertire i file ENC nel formato [.MXL](/audio/mxl/) (Compressed MusicXML).
