## Che cos'è un file 8SVX? ##

Un 8SVX è il formato di file audio standard sviluppato da Electronic Arts. È considerato un sottotipo di dati del formato di file IFF. Il file viene utilizzato per memorizzare dati audio a 8 bit all'interno di blocchi. In blocchi descrittivi separati i metadati sono contenuti sul flusso di dati 8SVX. È un formato di file audio digitale ed è utilizzato dai PC Amiga. I sistemi informatici di Amiga ora sono usati raramente, quindi questi file 8SVX sono solitamente utilizzati per preservare e ascoltare la vecchia musica.

## Formato file 8SVX

Il file 8SVX può memorizzare registrazioni audio con qualità a 8 bit derivata dalla specifica [IFF](/it/audio/aiff/). Questi sono file che contengono registrazioni audio come altri file contenenti audio. Comprendeva anche il formato contenitore di IFF. In questo formato, più formati di dati sono stati incorporati in un file e utilizzati per ulteriori dettagli sui flussi inclusi in esso. Questi possono essere utilizzati da soli all'interno dei contenitori di file solo audio dei file IFF. D'altra parte, il multiplexing con altri sottotipi di IFF può comportare flussi video animati.

## Breve storia del formato 8SVX

Inizialmente è stato pubblicato nel 1985 da Steve Hayes e Jerry Morrison. Un altro formato archiviato è stato sviluppato nel 2009. Questo formato di file è stato utilizzato sui sistemi informatici Amiga per la vecchia musica. È anche noto come file audio Amiga a 8 bit. Il software audio realizzato per PC Amiga supportava questo formato di file. Inizialmente è stato utilizzato da molti programmi di editing del suono. La maggior parte dei suoi flussi di dati sono codificati utilizzando flussi PCM. Una versione modificata di Amiga ha fornito un framework di decodifica 8SVX. I formati RIFF e WAVE co-sviluppati da Microsoft e IBM sono realizzati sotto l'influenza del formato contenitore 8SVX/IFF.

## Specifiche del formato file 8SVX

Molti programmi sono associati a questi file su Windows e Mac OS. Molti programmi possono essere utilizzati per affrontare questi file. È un formato audio codificato per Commodore Amiga. Le funzionalità come attacco, ripetizione della sezione e rilascio sono supportate da questi file. Queste funzioni vengono utilizzate per memorizzare campioni di strumenti musicali. È disponibile la compressione dei dati con perdita di Fibonacci-delta che si traduce in un rapporto di compressione del 50% e diminuisce la fedeltà.
