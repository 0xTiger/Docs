Il formato di file audio è una categoria di formati di file digitali per la rappresentazione di informazioni audio insieme ai relativi metadati. Esistono numerosi formati di file audio in base alla natura dei dati contenuti nel file audio. Tali file possono essere archiviati in formati di file audio compressi e non compressi. I formati di file audio più diffusi includono [MP3](/it/audio/mp3/), [WAV](/it/audio/wav/), PCM e WMA.
