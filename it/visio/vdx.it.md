## Che cos'è un file VDX?

Qualsiasi disegno o grafico creato in Microsoft Visio, ma salvato in formato XML ha estensione .VDX. Un file XML di disegno di Visio viene creato nel software Visio, sviluppato da Microsoft. Microsoft Visio ha la capacità di generare documenti visivi che possono essere utilizzati in presentazioni e documenti. Il file XML di disegno di Visio contiene gli oggetti visivi ei dettagli dei metadati degli elementi visivi. Il testo può anche essere aggiunto a questi elementi visivi File XML di disegno di visione. Questi file XML di disegno di Visio sono integrati con standard di formattazione basati su XML e specifiche di codifica dei dati di immagine che consentono il rendering e l'archiviazione del contenuto dal software Microsoft Visio nel formato di file VDX. Questi file XML di disegno di Visio vengono aggiunti con l'estensione .vdx. Di seguito sono elencate le applicazioni che possono aprire il formato di file VDX.

* Microsoft Visio
* CS Odessa ConceptDraw PROGETTO per MAC

## Formati file Visio ##

Dal 2013 visio ha introdotto un nuovo formato di file .vsdx che sostituisce altri due formati di file noti come .vsd (formato di file binario di Visio) e .vdx (formato di file di disegno XML di Visio). .vsdx ha aumentato l'interoperabilità dei file di Visio perché ora il software di terze parti può anche modificare i file di Visio a livello di formato di file. Il nuovo formato di file è supportato in Visio Services in Microsoft SharePoint Server e non richiede alcun formato di file "intermediario" per la pubblicazione su SharePoint Server. Esistono diversi tipi di file che compongono il visio con le seguenti estensioni:

* .vsdx (disegno di Visio)
* .vsdm (disegno abilitato alle macro di Visio)
* .vssx (stencil di Visio)
* .vssm (stencil con abilitazione macro di Visio)
* .vstx (modello di Visio)
* .vstm (modello abilitato per le macro di Visio)

## Caratteristiche ##

Visio contiene un'ampia gamma di modelli e forme che aiutano nella creazione di diagrammi di rete, diagrammi di flusso, planimetrie e progetti ingegneristici, ecc. Inoltre, possiamo anche ottenere aiuto nella creazione di modelli di processo e visualizzazione dei dati. Microsoft Visio è semplice e intuitivo in termini di utilizzo. Aiuta a creare diagrammi visivi efficaci che possono essere ulteriormente aggiunti in molte altre applicazioni come PowerPoint, Word e Publisher.

Microsoft Visio supporta anche Unified Modeling Language (UML) 2.4, Business Process Model e Notation 2.0. Inoltre, Visio supporta anche la pubblicazione di flussi di lavoro di SharePoint 2010 e 2013. Visio ha anche la capacità di rendere dinamici i diagrammi collegando forme con dati in tempo reale. Team di professionisti possono collaborare allo stesso diagramma contemporaneamente e possono anche visualizzare i commenti dei revisori in Visio e Visio Services. Inoltre, se qualcuno non ha installato Visio nel proprio sistema, può comunque visualizzare i tuoi diagrammi tramite Office 365 o SharePoint.

## Edizioni Microsoft Visio ##

1. **Microsoft Visio Standard**: l'edizione Standard è progettata per le persone che desiderano una piattaforma per la creazione di diagrammi robusta che disponga di un numeroso set di stencil o modelli integrati. Gli individui possono semplificare informazioni complesse in diagrammi facili e semplici. Microsoft Visio Standard contiene modelli per aziende, diagrammi di rete, diagrammi, diagrammi di flusso e vari altri diagrammi multiuso.
1. **Microsoft Visio Professional**: l'edizione Professional è progettata per individui e team che possono semplificare informazioni complesse attraverso diagrammi professionali e versatili. La funzionalità di base include Visio Standard 2013 ma con forme, modelli e stili aggiuntivi e aggiornati con un supporto maggiore per il lavoro di squadra che include il lavoro di più persone che lavorano sullo stesso diagramma contemporaneamente. Inoltre, Visio Professional 2016 ha ulteriormente aggiunto modelli aggiuntivi per diagrammi aziendali e ingegneristici, diagrammi di processo, mappe e planimetrie ecc.
1. **Visio Pro per Office 365**: questa è un'edizione speciale di Visio 2016 ed è accessibile tramite Office 365. Dispone delle stesse funzionalità di MS Visio 2016 Professional.
