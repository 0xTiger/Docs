## Che cos'è un file P6XML? ##

L'estensione del file P6XML ha un'eccellente leggibilità, adattabilità e ampio utilizzo che lo rendono il formato ideale. Primavera P6 Professional ha la funzionalità per esportare e importare sia file [XER](/it/project-management/xer) che file XML. XML sta per **Extensible Markup Language**. Questo formato generale consente agli utenti di condividere le informazioni sul progetto tra i database del Primavera P6. I dati del progetto possono essere facilmente archiviati in un file di testo tramite il file XML; questo rende le cose più semplici per lo scambio di dati di numerosi programmi software. Tali dati possono essere facilmente condivisi e gestiti dall'utente Primavera P6.

## Vantaggi di un formato file P6XML ##

* È facile per gli utenti di Primavera definire quali Baseline vengono esportate e importate utilizzando il file XML
* L'esportazione XML può contenere tutti i piani d'azione dettagliati del progetto
* Qualsiasi utente che utilizza Primavera 6 che non può modificare, aggiungere o eliminare determinati elementi di dati globali, come codici attività, calendari e risorse, non sarà in grado di importarli
* Oltre alle impostazioni del profilo di sicurezza generale e del progetto utilizzate da Primavera, gli elementi di dati che influenzeranno o modificheranno i dati nel database non verranno importati

## Esportazione del file XML da Primavera P6 ##

Per esportare progetti P6 Professional e dati di progetto in un file P6XML, utilizzare Oracle Primavera Cloud. Puoi scaricare il file quando viene formato dall'applicazione desktop P6 Professional e importarlo in Primavera Cloud. Prima di eseguire ininterrottamente il processo di esportazione, è necessario stabilire se si è collegati a un database P6 Professional o a un P6 EPPM. Alcune funzionalità del processo di esportazione potrebbero differire in base al database. Avendo impostazioni P6 Specialized con un database P6 Proficient, il processo di esportazione opera localmente sul tuo sistema. Avendo un database P6 Professional, puoi facilmente esportare e aprire il file XML P6 da un'impostazione sul tuo sistema.

Di seguito sono riportati alcuni passaggi che descrivono il processo di esportazione:

* Aprire i desideri Progetto Primavera P6
* Successivamente, seleziona Esporta nella scheda File
* Seleziona "Primavera P6 – (XML)". Successivamente, fai clic su Avanti
* Fai doppio clic sul progetto che esporterai
* Seleziona i troncamenti per il percorso in cui desideri inserire il file esportato e seleziona "Fine".
* Seleziona l'opzione come vuoi salvare il file
* Se i layout a livello di progetto non sono necessari con il file di esportazione, è essenziale deselezionare la selezione su "Esporta tutti i layout a livello di progetto".
* Seleziona fine, quindi riceverai una conferma dell'esportazione XML

### Suggerimenti importanti ###

* Ogni volta che si connette a un database EPPM P6, l'amministratore deve inserire l'URL della fonte di contenuto nel campo URL P6 nella pagina Generale delle Impostazioni dell'applicazione in P6
* È conveniente esportare numerosi progetti in un file XML
* Vengono esportati i dati globali allocati al progetto
* È conveniente esportare file XML senza dover accedere a tutte le risorse
  


## Problemi per aprire un file P6XML? ##

Alcuni problemi comuni che possono sorgere e causare il malfunzionamento del formato P6XML sono i seguenti:

* Assenza di software di supporto
* File corrotto
* File infetto a causa di virus
* Descrizione P6XML eliminata accidentalmente dal registro di Windows
* Nessun diritto di accesso nel sistema per aprire i file
* Unità obsoleta nel tuo sistema
* L'estensione del file viene rinominata
