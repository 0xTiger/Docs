## Che cos'è un file XER?

Il formato di file XER è un formato di file di progetto proprietario utilizzato dall'applicazione di pianificazione e gestione del progetto Primavera P6. Come il formato di file [MPP](/it/project-management/mpp/), questo formato di file memorizza i dati relativi alla pianificazione del progetto come attività, risorse, tempi e materiali. Primavera è sviluppato da Oracle e fornisce una pianificazione completa del progetto e trova le sue applicazioni in diversi campi pratici come petrolio e gas, edilizia, produzione, ecc. Aiuta i project manager a pianificare, programmare e controllare programmi su larga scala e progetti individuali integrando con sistemi di gestione finanziaria e di gestione del capitale umano. Molti software e API possono leggere e convertire file XER in altri formati di file di progetto popolari come XML e MPP.

## Formato file XER ##

Un file XER è un file di testo e può essere facilmente aperto con un editor di testo. Potrebbe essere utile aprire i dati relativi al progetto in un editor di testo, ma non è facile comprenderli stabilendo manualmente i collegamenti. Questo formato è il più utilizzato per lo scambio di dati con altri database P6. Ha un livello di controllo relativamente basso sui dati che sono stati importati da un altro formato. I vantaggi di XER rispetto al formato XML equivalente includono la possibilità di spostare codici e calendari globali a livello di progetto e mappare codici, UDF, risorse e calendari su valori esistenti.


## Problemi per aprire un file XER? ##

Alcuni problemi comuni che possono sorgere e causare il malfunzionamento del formato XER sono i seguenti:

* Assenza di software di supporto
* File corrotto
* File infetto a causa di virus
* Descrizione XER eliminata accidentalmente dal registro di Windows
* Nessun diritto di accesso nel sistema per aprire i file
* Unità obsoleta nel tuo sistema
* L'estensione del file viene rinominata
