## Che cos'è un file MPT? ##

Un file con estensione .mpt è un file modello [Microsoft Project](https://products.office.com/en-us/project/project-and-portfolio-management-software). Contiene informazioni di base, struttura e impostazioni del documento per la creazione di file [MPP](/it/project-management/mpp/). Questi modelli forniscono impostazioni predefinite essenziali per un particolare progetto, come pianificazioni o informazioni sul budget. Tuttavia, non può salvare dati specifici del progetto come attività, risorse o assegnazioni. Una volta modificati, i file modello possono essere salvati come file MPP standard per lavorare ulteriormente con esso. In Microsoft Project è disponibile un file modello master, intitolato "Global.mpt", aperto per creare un modello individuale. I file MPT utilizzano *application/vnd.ms-project* e *application/x-project* come tipi MIME standard. Windows, Linux e macOS possono tutti aprire file MPT, insieme a Moos Project Viewer, Microsoft Project e MPXJ.

## Problemi per aprire un file MPT? ##

Ecco l'elenco di alcuni problemi comuni che possono sorgere e causare il malfunzionamento del formato MPT:

* Assenza di software di supporto
* File corrotto
* File infetto a causa di virus
* Nessun diritto di accesso nel sistema per aprire i file
* Unità obsoleta nel tuo sistema
 *   Extension of the file is renamed
