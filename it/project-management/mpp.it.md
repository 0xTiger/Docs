## Che cos'è un file MPP?

Un file con estensione .mpp è un file di dati di Microsoft Project che memorizza le informazioni relative alla gestione dei progetti in modo integrato. Microsoft ha sviluppato questo formato di file proprietario per renderlo compatibile con Microsoft Project (MSP), il loro prodotto software di gestione dei progetti. Oltre a MPP, MSP può funzionare con lo schema XML del progetto. Molte applicazioni e API consentono di convertire gli MPP in altri formati di file. Microsoft ora dispone di un Project Server online in cui i file di gestione dei progetti possono essere caricati per la collaborazione da più utenti.

## Formato file MPP ##

Microsoft ha sviluppato e pubblicato MPP come formato di file binario e le sue specifiche non sono state rese pubbliche da Microsoft fino ad oggi. La struttura e i dettagli dei file interni non sono quindi disponibili per gli sviluppatori. Sono disponibili diverse API online che supportano la lettura/scrittura su file MPP, ma la precisione del 100% non viene mai raggiunta. Microsoft ha introdotto nel tempo diverse funzionalità nella sua applicazione di gestione dei progetti, così come le funzionalità del formato di file MPP.

Il file MPP utilizza i seguenti tipi MIME:

* applicazione/progetto vnd.ms
* applicazione/msproj
* applicazione/msprogetto
* applicazione/x-msprogetto
* applicazione/progetto x-ms
* applicazione/x-dos_ms_project
* domanda/mpp
* zz-applicazione/zz-winassoc-mpp

## Problemi per aprire un file MPP? ##

Ecco l'elenco di alcuni problemi comuni che possono sorgere e causare il malfunzionamento del formato MPP:

* Assenza di software di supporto
* File corrotto
* File infetto a causa di virus
* Nessun diritto di accesso nel sistema per aprire i file
* Unità obsoleta nel tuo sistema
* L'estensione del file viene rinominata
