## Informazioni sulla conversione da DOCX a DOC

Il tipo di file DOC è stato un formato di file di documenti popolare utilizzato dal 2007 e dalle versioni precedenti di Microsoft Word. Con il rilascio di Office 2007, Microsoft ha introdotto il formato file DOCX basato sulle specifiche Office Open XML. I file DOC sono stati salvati come file binari rispetto a DOCX che viene salvato in formato file XML compresso. Ci sono moltissimi strumenti disponibili online per convertire file [DOCX](/it/word-processing/docx/) in [DOC](/it/word-processing/doc/).
