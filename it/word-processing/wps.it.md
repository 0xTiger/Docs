## Che cos'è un file WPS?

Un file con estensione .wps è un documento di Microsoft Works utilizzato prima del 2006 come documento di Word. A quel tempo era simile al formato di file [DOC](/it/word-processing/doc/) appena introdotto, ma non era tanto ricco di funzionalità quanto il nuovo formato di file. Ad esempio, non supportava la funzionalità macro e le opzioni di formattazione avanzate. I file WPS possono ancora essere aperti con i moderni elaboratori di testi, ma per Windows 10 è necessario installare un convertitore per caricare questi file. Microsoft Works è stato sostituito dall'elaboratore di documenti Microsoft Word nel 2010.

## Specifiche del formato file

Le specifiche del formato file del documento WPS non sono disponibili pubblicamente poiché Microsoft non ha aperto le specifiche agli utenti finali. I WPS sono file binari e possono essere aperti con filtri di conversione del formato file creati da Microsoft per l'elaboratore di testi. LibreOffice, NeoOffice e OxygenOffice hanno incluso una libreria C++ generale, libwps, che può estrarre testo da molte versioni differenti di Microsoft Works.
