## Informazioni sulla conversione da DOC a JPG

Se hai mai lavorato con versioni precedenti di Microsoft Word (ovvero versioni precedenti a Microsoft Word 2007), devi avere familiarità con il formato file DOC. Era il formato file predefinito utilizzato da Microsoft Word prima dell'introduzione del formato file DOCX con Microsoft Word 2007. La versione da DOC a JPG è un requisito molto comune, soprattutto quando Microsoft non consente di esportare documenti Word in immagini JPG. Inoltre, le ultime versioni di Microsoft Word non sono completamente compatibili con i file DOC precedenti. Puoi utilizzare app online come l'app Aspose.Words per convertire i formati di file [DOC](/it/word-processing/doc/) in [JPG](/it/image/jpeg/).

## Conversione da DOC a JPG - Demo gratuita

L'app Aspose.Words è un insieme di strumenti online che ti consente di convertire i tuoi file DOC in diversi formati di file. Puoi convertire file DOC in JPG con pochi clic dal tuo browser web. Il vantaggio dell'utilizzo dell'app Aspose.Words è che non richiede codifica, è veloce e precisa e, soprattutto, è completamente gratuita.

Durante il processo di conversione da DOC a JPG, ogni pagina del documento viene convertita in immagine JPG ed è disponibile per il download. L'output è un insieme di immagini JPG salvate su disco.

### Come convertire DOC in JPG?

Quindi, come convertire DOC in JPG? È possibile convertire il formato file DOC in JPG utilizzando l'[app Aspose.Words DOC in JPG](https://products.aspose.app/words/conversion/doc-to-jpg) come segue.

1. Vai all'app Aspose.Words per [convertitore da DOC a JPG](https://products.aspose.app/words/conversion/doc-to-jpg)
1. Carica il tuo file DOC facendo clic sul pulsante di caricamento
1. Una volta terminato, premere il pulsante Converti. Questo avvierà il processo di conversione del file DOC in JPG in background
1. Ti verrà chiesto di scaricare il file JPG convertito una volta completato il processo di conversione da DOC a JPG
