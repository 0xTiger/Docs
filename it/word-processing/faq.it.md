## Che cos'è un file delle domande frequenti?

Un file FAQ (Domande frequenti) è un documento di testo che contiene le risposte alle domande frequenti poste dagli utenti per un particolare prodotto, servizio o richiesta. È incluso principalmente come guida o manuale per la risoluzione dei problemi con i prodotti per facilitare agli utenti le domande comuni che possono sorgere nella loro mente sul servizio. Le informazioni comuni contenute in un file delle domande frequenti possono includere domande relative alle funzionalità del prodotto, guide pratiche, domande per la risoluzione dei problemi e domande sull'installazione. Per i prodotti software, le FAQ sono state per lo più sostituite con i file [.chm](/it/web/chm/).

## Formato file delle domande frequenti - Ulteriori informazioni

I file delle domande frequenti vengono creati e salvati come file di testo semplice leggibili dall'uomo. Questi di solito possono essere aperti in qualsiasi editor di testo come Microsoft Notepad, Microsoft Word e Apple TextEdit.

## Come scrivere una buona pagina delle domande frequenti?

Ecco alcune buone linee guida per le domande frequenti che possono essere incluse con un software o un prodotto durante l'implementazione per i clienti.

1. Includi le parole chiave che i clienti stanno cercando.
1. Includi cosa c'è nella tua pagina concorrente.
1. Esprimi le tue domande in modo che i clienti le scrivano o le facciano.
1. Formatta correttamente le tue FAQ
