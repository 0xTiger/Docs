Un file di elaborazione testi contiene informazioni sull'utente in formato testo normale o RTF. Un formato di file **testo normale** contiene testo non formattato e non è possibile applicare caratteri o impostazioni di pagina, ecc. Al contrario, un formato di file **rich text** consente opzioni di formattazione come l'impostazione del tipo di carattere, gli stili (grassetto, corsivo, sottolineato, ecc.), i margini delle pagine, i titoli, i punti elenco e i numeri e molte altre funzioni di formattazione. L'uso di file di testo normale si è ridotto in modo significativo con il passare del tempo poiché sono disponibili computer e programmi più potenti per offrire l'elaborazione di file di testo RTF.

Le estensioni di file di testo normale comuni e i formati di file associati includono **[TXT](/it/word-processing/txt/)**, **[CSV](/it/spreadsheet/csv/)**, mentre le estensioni di file per i documenti RTF includono **[DOCX](/it/word-processing/docx/)**, **[DOC](/it/word-processing/doc/)** e **[RTF](/it/word-processing/rtf/)** .

Hai domande relative ai formati di file di elaborazione testi? Visita la nostra community [forum](https://forum.fileformat.com/c/word-processing/5) per beneficiare delle conoscenze condivise dagli esperti di formati di file.
