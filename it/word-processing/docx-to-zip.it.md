## Informazioni su DOCX in ZIP

Microsoft Word 2007 e versioni successive utilizzano DOCX come formato file predefinito per il salvataggio dei documenti. Può contenere migliaia di parole, immagini, elementi multimediali, tabelle e altri dati rilevanti. Ciò rende la dimensione del file DOCX molto grande e ne rende difficile la condivisione online tramite Internet. Ciò occupa molto spazio anche quando viene caricato su un supporto di archiviazione file online come Google Drive o Dropbox. Da DOCX a ZIP risolve questo problema poiché il file DOCX viene convertito in un file compresso con dimensioni di file ridotte. Esistono diversi strumenti e app online in grado di convertire file DOCX in formato ZIP. Aspose.Words è una di queste app per la conversione di [DOCX](/it/word-processing/docx/) in [ZIP](/it/compression/zip/).
