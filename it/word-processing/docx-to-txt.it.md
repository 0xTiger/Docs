## Cosa succede quando un DOCX viene convertito in un file TXT?

DOCX è un formato di file multimediale che può contenere testo e informazioni multimediali. Inoltre, può contenere dati in tabelle, immagini, oggetti incorporati, forme e molti altri disegni. In confronto, un file .txt può contenere solo dati di testo e non ha la capacità di contenere contenuti multimediali. Pertanto, quando un file DOCX viene convertito in TXT, accade quanto segue.

* Tutta la formattazione del testo viene persa
* Anche le immagini nel file DOCX verranno perse
* Anche le tabelle nel file DOCX vengono rimosse e solo il testo viene esportato nel file TXT di output
* Vengono rimosse anche tutte le forme presenti nel file DOCX

In short, any content other than text is removed during conversion from DOCX to TXT.
