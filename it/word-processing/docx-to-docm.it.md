## Informazioni sulla conversione da DOCX a DOCM

DOCM è il formato di file di documenti di Microsoft Word 2007 che ha la capacità di eseguire macro. Il formato file interno di DOCM è simile al formato file DOCX ma con la funzionalità aggiuntiva del supporto per le macro che non è disponibile in DOCX.
