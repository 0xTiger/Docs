## Che cos'è un file ODM?

Un file con estensione .odm è un file di documento OpenDocument Master che contiene uno o più file [.odt](/it/word-processing/odt/) collegati al suo interno. Questi documenti secondari sono tutti combinati in un unico documento master. Ciò semplifica la gestione di tutti i file collegati dal singolo documento master. I file dei documenti master possono essere creati con Apache OpenOffice Writer o OpenDocument Word Processor. I file ODM possono essere esportati in un singolo file ODT utilizzando l'opzione di esportazione dall'applicazione OpenOffice Writer.

## Formato file ODM - Ulteriori informazioni

I file ODM vengono salvati come file di testo che possono essere aperti in OpenOffice Writer. Un singolo file ODM è costituito da più sezioni collegate ai documenti secondari .odt. Ciò consente a più autori di modificare i propri documenti ODT che sono collegati nel file ODM principale. Pertanto, il file ODM principale viene aggiornato man mano che ogni scrittore aggiorna le rispettive sezioni.

I file ODM possono anche includere un sommario e un indice. Ogni volta che si apre un documento Master, lo scrittore del documento master chiede di aggiornare i collegamenti. È necessario dire di sì affinché il contenuto venga mostrato correttamente.
