## Che cos'è un file IPYNB?

Un file con estensione .ipynb è un file Jupyter Notebook che contiene l'intero contenuto di Notebook creato con la sessione dell'applicazione web Jupyter Notebook. [Jupyter Notebook](https://jupyter.org/) è un'applicazione web interattiva utilizzata per creare e condividere documenti computazionali per l'analisi utilizzando Python.

Un file IPYNB funge da registrazione computazionale completa di una sessione e include informazioni come input e output di calcolo, funzioni matematiche e una ricca rappresentazione degli oggetti risultanti come immagini e testo esplicativo.

## Formato file IPYNB

IPYNB vengono salvati internamente nel formato di file [JSON](/it/web/json/), che è un formato di file standard aperto per la condivisione dei dati. Per questo motivo, i file IPYNB sono leggibili e facilmente comprensibili. JSON semplifica anche la condivisione di questi file con altri.

## Come convertire i file IPYNB?

I file IPYNB possono essere esportati in diversi formati di file utilizzando il comando nbconvert dall'interfaccia di Jupyter. Ciò include la conversione di IPYNB in:

* PDF
* HTML
* testo strutturato
* Latex

## Come visualizzare il file IPYNB?

Esistono due metodi per visualizzare o aprire i file IPYNB.

1. L'utente può caricare il documento del notebook .ipynb su un URL pubblico e aprirlo da Jupyter Notebook Viewer<nbviewer> che rende il notebook come una pagina Web statica dall'URL.
1. Scarica i file e aprili nella loro istanza di Jupyter Notebook
