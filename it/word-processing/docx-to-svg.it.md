## Informazioni sulla conversione da DOCX a SVG

SVG (Scalar Vector Graphics) è un tipo di file immagine che utilizza il formato di testo basato su XML per rappresentare l'aspetto di un'immagine. La bellezza di questo formato file è che le immagini salvate come file SVG possono essere ridimensionate a dimensioni diverse senza perdere la qualità. Da DOCX (essendo uno dei formati di file di documenti maggiormente utilizzati per il salvataggio di file Word) a SVG è una delle operazioni comunemente utilizzate che gli utenti di Microsoft Word eseguono sui file DOCX. Esistono numerose app e strumenti disponibili online per eseguire la conversione da DOCX a SVG. L'app Aspose.Words è uno di questi strumenti online che ti consente di convertire [DOCX](/it/word-processing/docx/) in [SVG](/it/page-description-language/svg/) gratuitamente.
