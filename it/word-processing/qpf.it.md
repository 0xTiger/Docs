## Che cos'è un file QPF?

Un file con estensione .qpf è un file di testo creato con il software QuickPad che è un'applicazione gratuita offerta da Klumbu, come versione evoluta di Blocco note per sistema operativo Windows. I file QPF possono essere aperti e visualizzati solo in QuickPad e in questo modo forniscono una sicurezza minima dei dati sensibili. Il testo all'interno del documento non viene crittografato utilizzando alcuna password. Invece, è codificato dal software per renderlo in qualche modo sicuro.

## Formato file QPF

I file QPF sono file di testo che vengono creati e salvati in un formato leggibile dall'uomo. L'unica differenza è che per leggere questi file, questi devono essere aperti in QuickPad. QuickPad offre funzionalità come Word Wrap, formattazione del testo, salvataggio automatico, modalità di messa a fuoco e altro.
Inoltre, il testo può essere importato da INI così come il formato di file TXT.
