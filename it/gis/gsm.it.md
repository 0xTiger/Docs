## Cos'è un file GSM?

Un file GSM è un file di progetto creato con il software MapViewer che viene utilizzato per l'analisi spaziale e la creazione di mappe geografiche. Si tratta di un file di impostazioni della mappa che memorizza le informazioni di configurazione relative alle impostazioni di visualizzazione delle mappe come etichette di testo, colori, legenda e file di dati di riferimento.

## Formato file GSM

I file GSM vengono salvati in formato file binario e i dettagli della struttura del file interno non sono disponibili pubblicamente come riferimento per lo sviluppatore. Il software MapViewer viene fornito con file GSM di esempio e alcuni file [DAT](/database/dat/) utilizzati per creare ulteriori file GSM.

### Informazioni su MapViewer

MapViewer ti consente di lavorare con file di mappe sovrapponendo i tuoi dati su aree geografiche. Ciò aiuta a visualizzare dati spaziali come mappe di densità di popolazione, mappe climatiche e molti altri tipi di dati.
