## Cos'è un file MSD?

Un file MSD è un file Map Service Distribution creato con il software di mappatura GIS ArcGIS Desktop. Contiene la mappa pubblicata esportata da un file di mappa [.mxd](/it/gis/mxd/). Un file MSD contiene informazioni su un servizio di mappa ospitato su ArcGIS Server. La mappa ospitata è accessibile tramite Internet per altri. Il file MSD contiene il documento della mappa, le informazioni sui livelli e le proprietà della tabella.

## Formato file MSD

I file MSD vengono salvati su disco in formato file binario. Questi vengono utilizzati dall'applicazione server ArcGIS per creare e gestire il servizio di mappa a scopo di hosting. I file MSD vengono utilizzati anche da ArcMap per connettersi al servizio di mappa e visualizzare i contenuti sulla mappa.
