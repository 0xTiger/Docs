## Che cos'è un file GMAP?

Un file GMAP è un file GIS che contiene una combinazione di regioni specifiche del mondo. È stato utilizzato da alcuni dispositivi Garmin. Le combinazioni di mappe possono comprendere diverse regioni che non devono necessariamente essere contigue. I dispositivi Garmin utilizzati per memorizzare questi file di mappe nella memoria incorporata. Tuttavia, i dispositivi Garmin più recenti supportano la modalità di archiviazione di massa che sembra simile a un normale dispositivo di archiviazione USB. I file GMAP possono essere aperti con il software Garmin BaseCamp e MapInstall su PC e software Mac.

## Formato file GMAP - Ulteriori informazioni

I file GMAP vengono salvati su disco come file binari e i dettagli sul formato file interno non sono disponibili pubblicamente.

## Tipi di file comuni utilizzati sui dispositivi Garmin

* Formato di scambio GPS ([.gpx](/it/gis/gpx/))
* Formato file FITness ([.fit](/it/gis/fit/))
* Database GPS Garmin® (.gdb)
* Garmin Training Center ([.tcx](/it/gis/tcx/))
* Posizione GPS ([.loc](/it/gis/loc/))
* Mappa personalizzata Garmin ([.kml](/it/gis/kml/) o [.kmz](/it/gis/kmz/))
* Foto (.jpg o [.jpeg](/it/image/jpeg/))
