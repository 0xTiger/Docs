## Che cos'è un file NM2?

Un file con estensione .nm2 è un file mappa creato con il sistema di navigazione Navitel. Memorizza i dati geospaziali relativi a diverse entità della mappa come strade, strade, corsie stradali, punti sulla mappa, punti di riferimento e altri dati GPS. I file NM2 possono essere aperti utilizzando il software Navitel su diversi tipi di dispositivi come telefoni Android, dispositivi iOS, Windows Phone e dispositivi Symbian. Queste app sono disponibili gratuitamente e possono essere scaricate e installate sui rispettivi dispositivi.

## Formato file NM2 - Ulteriori informazioni

I file NM2 vengono salvati come file binari e il loro formato di file interno/struttura dei dati non è disponibile pubblicamente. Le versioni recenti di Navitel salvano i dati come file .nm7. Ci sono diversi file di mappe disponibili da Navitel sul loro [sito web](https://www.navitel.cz/en/downloads/demo). La [documentazione Navitel](https://www.navitel.cz/en/downloads/documentation) fornisce guide utente complete sull'utilizzo del suo sistema di navigazione per diversi tipi di dispositivi e veicoli.
