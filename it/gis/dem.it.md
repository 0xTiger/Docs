## Che cos'è un file DEM?

Un file con estensione .dem è un file di mappatura che rappresenta i dati di elevazione (altezza) come terreno nella computer grafica 3D. Un esempio comune di tali dati è il terreno terrestre che include montagne, pianura e profondità del mare. I file DEM sono comunemente usati dai Geographic Information Systems (GIS) per produrre mappe di elevazione digitali. Le informazioni archiviate nei file DEM includono anche informazioni sui rilievi topografici su larga scala che vengono utilizzate nelle scienze della terra, nella gestione delle risorse, nella pianificazione del territorio, nel rilevamento e nei progetti di ingegneria. Le informazioni sul terreno XYZ sono posizionate a intervalli regolari distanziati dalla griglia per la rappresentazione del rilievo del terreno.

## Formato file DEM - Ulteriori informazioni

I dettagli interni dei file DEM non sono disponibili pubblicamente per il riferimento dello sviluppatore. Tuttavia, i file DEM possono avere diversi formati di file come:

* Griglia attribuita batimetrica (BAG)
*DTED
* Base dati DIMAP Sentinel 1 ESA
* SDTS DEM
* USGS DEM

## Dove trovare i file DEM?

I file DEM sono comunemente disponibili su Internet ed è possibile scaricare campioni per l'uso. L'[United States Geological Survey](https://www.usgs.gov/) (USGS) fornisce molti file DEM sul proprio sito web.
