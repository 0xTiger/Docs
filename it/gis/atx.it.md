## Che cos'è un file ATX?

Un file con estensione .atx è un file ESRI ArcGIS che memorizza gli indici degli attributi di ArcGIS 8. Esiste un file ATX separato per ogni [shapefile](/it/gis/shp/) o indice di attributo dBase creato in ArcCatalog. Va notato che gli indici degli attributi di ArcView GIS 3.x per gli shapefile ei file dBase non vengono utilizzati da ArcMap o ArcCatalog. Tuttavia, questi indici di attributi di ArcView GIS 3.x possono essere convertiti in indici di attributi di ArcGIS 8 utilizzando uno strumento ArcObjects.

## Formato file ATX - Ulteriori informazioni

Non sono disponibili specifiche del formato di file per il formato di file ATX per facilitare gli sviluppatori.
