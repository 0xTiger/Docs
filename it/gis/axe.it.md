## Cos'è un file AX?

Un file AX è un formato di file GIS creato con l'applicazione di routing Microsoft AutoRoute. Contiene informazioni sulla posizione dei Punti di interesse (POI) sulla mappa e altri dati di navigazione. Questi POI sono facili da navigare e potrebbero essere aggiunti al viaggio come parte del percorso. AutoRoute è stato sviluppato per fornire informazioni sui percorsi per la navigazione su strade e autostrade attraverso l'Europa.

I file AX potrebbero essere aperti con Microsoft AutoRoute ma è stato interrotto molto tempo fa e i file AX non sono più supportati.

## Formato file AX: ulteriori informazioni

I file AX vengono salvati su disco in formato file binario. Viene salvato in un formato di file proprietario e i dettagli della struttura del file interno non sono disponibili pubblicamente come riferimento per lo sviluppatore.
