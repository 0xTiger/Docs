## Cos'è un file SDF?

A Spatial Data File (SDF) is a Geodatabase file format that was developed by Autodesk. It stores geospatial data and is used by Autodesk GIS program MapGuide and AutoCAD Map 3D. SDF file format is based on the single file database file format SQLite3. È considerato un formato di file ottimizzato per grandi set di dati di informazioni spaziali.

È possibile aprire un file SDF utilizzando Autodesk AutoCAD Map 3D 2022 e Autodesk AutoCAD Civil 3D 2023.

## Formato file SDF: ulteriori informazioni

Il formato file SDF viene memorizzato sul disco come file binari. Si basa sul database SQLite che utilizza componenti di archiviazione di basso livello per la serializzazione binaria dei dati. Un file SDF contiene più classi di entità geografiche per file e più proprietà della geometria per ciascuna classe di entità geografiche. I dati in un file SDF sono accessibili ad alta velocità grazie all'indicizzazione di ciascuna proprietà della geometria utilizzando un R-tree.
