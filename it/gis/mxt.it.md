## Cos'è un file MXT?

Un file MXT è un modello di mappa geografica creato con l'applicazione di mappatura GIS, ArcMap, fornita come parte di ESRI ArcGIS Desktop. Viene utilizzato per archiviare layer di mappe di base o layout di pagina che possono essere riutilizzati per creare più mappe. L'utilizzo dei modelli ti fa risparmiare tempo poiché non devi riprodurre le parti comuni delle mappe. I file MXT possono essere condivisi con altri utenti/organizzazioni per creare rapidamente altre mappe.

## Formato file MXT

I file MXT vengono salvati come file binari ma i dettagli della struttura dei file interni non sono disponibili pubblicamente come riferimento per lo sviluppatore. Contiene informazioni quali le proprietà di base di una mappa, le posizioni del layout e la visibilità delle barre degli strumenti dell'applicazione. I file MXT consentono agli utenti di creare più mappe con impostazioni predefinite già a loro disposizione.
