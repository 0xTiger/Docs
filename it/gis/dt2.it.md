## Cos'è un file DT2?

Un file DT2 è un file di elevazione che rappresenta il file DTED (Digital Terrain Elevation Data). Contiene dati di elevazione in schemi spaziati uniformemente. I dati di elevazione possono essere letti dai programmi software GIS e trovano le sue applicazioni in diverse aree come comunità militari, scientifiche e di altro tipo.

Le applicazioni GIS che possono aprire file DT2 includono ESRI ArcGIS Pro, TatukGIS, Blue Marble Geographic Global Mapper e Pitney Bowes MapInfo. Altre varianti dei tipi di file DTED sono [DT0](/gis/dt0/) e [DT1](/gis/dt1/).

### Formato file DT2

I file DT2 vengono salvati come file binari su disco e possono essere letti utilizzando molte applicazioni GIS. Questi possono essere uniti in un catalogo raster di file geodatabase che può essere utilizzato per creare un grande mosaico.
