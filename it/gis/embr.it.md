## Cos'è un file EMBR?

Un file EMBR è un file GIS sviluppato e utilizzato da GEOREID Pty Ltd. Contiene dati geospaziali come lunghezza e volumi calcolati in qualsiasi punto specifico. Il formato file EMBR memorizza questi dati statistici in formato testo semplice che può essere aperto in un editor di testo per la visualizzazione e la modifica. I sistemi GIS memorizzano informazioni geologiche con riferimento a qualsiasi punto lat-long che rappresenta la posizione assoltativa sulla terra.

Altri formati di file GIS popolari utilizzati per archiviare dati geospaziali includono [gpx](/gis/gpx/), [geojson](/gis/geojson/), [kml](/gis/kml/) e [kmz](/gis/kmz/).

## Formato file EMBR: ulteriori informazioni

I file EMBR sono file di testo che memorizzano dati statistici di un determinato punto nel tempo. I dati possono essere utilizzati per eseguire diversi calcoli come il calcolo delle proiezioni e il confronto dei dati. Se disponi di un file EMBR, puoi aprirlo in un editor di testo come Blocco note, Notepad++ o Apple TextEdit per visualizzarne il contenuto e aggiornare le informazioni.
