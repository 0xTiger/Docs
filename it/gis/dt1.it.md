## Cos'è un file DT1?

Un file DT1 è un file DTED (Digital Terrain Elevation Data) che contiene dati del terreno uniformemente distanziati. Viene utilizzato principalmente dalle applicazioni software GIS militari per ottenere dati su elevazione, pendenza e contorno della superficie. Il file DT1 è simile a [DT0](/gis/dt0/) e [DT2](/gis/dt2/) ma con una granularità media delle informazioni.

I file DT1 possono essere aperti con applicazioni GIS come ESRI ArcGIS Pro, TatukGIS, Blue Marble Geographic Global Mapper e Pitney Bowes MapInfo.

### Formato file DT1

I file DT1 vengono salvati come file binari su disco e possono essere letti utilizzando molte applicazioni GIS. Questi possono essere uniti in un catalogo raster di file geodatabase che può essere utilizzato per creare un grande mosaico.
