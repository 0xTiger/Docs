## Cos'è un file TFRD?

Un file TFRD è un formato di dati di immagine utilizzato dalla National Geospatial-Intelligence Agency (NGA) per archiviare dati geospaziali. NGA fornisce intelligenza geospaziale che offre vantaggi nel prendere decisioni. I file TFRD possono memorizzare sia dati di immagini che testuali e possono essere utilizzati dalle applicazioni GIS per eseguire il rendering di dati geospaziali.

## Formato file TFRD

I dettagli del formato file interno del formato file TFRD non sono noti. Vengono, tuttavia, salvati come file compressi su disco e il rapporto di compressione è maggiore rispetto ai file in formato file NITF equivalenti.
