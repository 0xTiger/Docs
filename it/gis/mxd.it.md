## Cos'è un file MXD?

Un file MXD è un file di documento di mappa utilizzato dal software ESRI ArcMap. È un formato di file proprietario che memorizza diversi tipi di informazioni come il layout e il design delle mappe, i dati utilizzati per generare le mappe e la simbologia applicata ai dati. Il file MXD è, quindi, un formato di file composto che contiene diversi tipi di elementi della mappa in un unico file. I file MXD vengono utilizzati dal file ArcGIS Map Package ([.mpk](/gis/mpk/)).

## Formato file MXD

I file MXD vengono salvati nel formato di file proprietario ArcGIS e i suoi dettagli interni non sono disponibili come riferimento per lo sviluppatore. Gli elementi incorporati di un file MXD sono organizzati in unità, note come archivi e flussi.
