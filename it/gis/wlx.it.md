## Cos'è un file WLX?

Un file WLX è un file GIS creato dal software MapSource fornito con i prodotti cartografici Garmin. Contiene una posizione geografica (sotto forma di latitudine-longitudine) in formato XML e una posizione mondiale su un sito Web di mappatura come Google Maps o Bing Maps. Il file WLX funge da collegamento di scelta rapida e apre la posizione memorizzata in siti Web di terze parti nella posizione di latitudine e longitudine registrata.

Un altro file simile (anche se non correlato a GIS) è il formato file [URL](/web/url/) che memorizza l'indirizzo web e indirizza gli utenti a questo URL quando viene aperto.

## Formato file WLX: ulteriori informazioni

I file WLX sono archiviati in formato file di testo che può essere aperto in qualsiasi editor di testo per visualizzarne e aggiornarne il contenuto. MapSource è un software basato su GIS che consente di visualizzare mappe, waypoint, percorsi e tracce. Puoi anche trasferire questi dati spaziali da o verso un dispositivo GPS Garmin.

MapSource non è più supportato da Garmin ed è stato sostituito da BaseCamp disponibile per il download gratuito.
