## Cos'è un file LAN?

Un file con estensione .lan è un file GIS raster utilizzato dalle versioni precedenti dell'applicazione software ERDAS. Viene utilizzato anche da altre applicazioni GIS e ha lo scopo di archiviare e visualizzare immagini satellitari e aeree. Inoltre, può memorizzare anche altri tipi di dati raster. Un file LAN ERDAS può anche contenere informazioni aggiuntive specifiche di ERDAS come georeferenziazione e informazioni sui livelli.

È possibile aprire file LAN con ERDAS IMAGINE e ArcGIS Desktop.

## Formato file LAN

I file LAN vengono archiviati su disco in formato file binario e non sono leggibili dall'uomo. Il formato file LAN viene utilizzato principalmente nelle applicazioni di telerilevamento e GIS. È stato, tuttavia, sostituito dal formato file ERDAS IMAGINE ([.IMG](/it/gis/img/)).
