## Cos'è un file 3DL?

Un file 3DL è un file iGO Landmark utilizzato dal software di navigazione iGO. Memorizza le informazioni sulla posizione per punti di interesse (POI) e punti di riferimento. Queste informazioni possono includere il nome, la posizione e altri dettagli sul PDI o sul punto di riferimento. Il formato file iGO Landmark è proprietario e viene utilizzato solo da [iGO navigation software](https://en.wikipedia.org/wiki/IGO_(software)). I file 3DL possono essere utilizzati insieme ai file di dati della mappa per fornire indicazioni di navigazione agli utenti.

## Formato file 3DL

Il formato file 3DL è proprietario e non è documentato pubblicamente. Il software di navigazione iGO, che utilizza il formato file 3DL, è sviluppato e distribuito da una società chiamata NNG (ex Nav N Go). Pertanto, potresti non riuscire a trovare informazioni tecniche dettagliate sul formato su [NNG website](https://www.nng.com/) o altrove su Internet.
