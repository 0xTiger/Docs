## Che cos'è un file DIV?

Un file con estensione .div è un file di progetto creato con il software [DIVA-GIS](https://en.wikipedia.org/wiki/DIVA-GIS) quando viene creato e salvato un nuovo progetto di mappatura. DIVA-GIS è un software di mappatura utilizzato per creare mappe spaziali e applicazioni di analisi di dati geografici. Il file di progetto è costituito da un'immagine mappa e livelli utilizzati per visualizzare gli oggetti e le proprietà sulla mappa. Oltre alle informazioni spaziali, memorizza anche informazioni sulla scala della mappa e sul centro.

## Formato file DIV - Ulteriori informazioni

I file DIV vengono archiviati su disco come formato di file proprietario DIVA-GIS e il loro formato di file interno non è disponibile come riferimento per gli sviluppatori.
