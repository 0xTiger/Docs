## Cos'è un file WOR?

Un file WOR è un file GIS utilizzato dal software di mappatura e analisi geografica MapInfo. Contiene un'area geografica che contiene una mappa visiva contenente dati su caratteristiche geospaziali come popolazione e informazioni demografiche. È possibile caricare un file WOR nel software MapInfo per visualizzarlo ed eseguire diverse operazioni su di esso come lo zoom e la panoramica.

## Formato file WOR

I file WOR sono costituiti da più livelli di cui viene eseguito il rendering in un ordine specifico quando caricati in MapInfo. I dettagli sul formato file WOR non sono disponibili pubblicamente come riferimento per lo sviluppatore. I file WOR possono essere convertiti in QGIS con [QGIS 3.20 Odense](https://www.qgis.org/en/site/forusers/visualchangelog320/index.html#feature-new-embedded-styling-renderer-with-ogr-feature-styles-support).
