## Che cos'è un file APR?

Un file con estensione .apr è un file di progetto ESRI ArcView che contiene informazioni sul progetto per un progetto ESRI. Questi sono stati utilizzati da ArcView 3.xe non sono supportati dalle nuove versioni (9.xe 10.x) di ArcGIS. ArcView 10 può, tuttavia, importare file APR. ArcView è un GIS (Geographical Information System) utilizzato per creare, visualizzare, analizzare e gestire i dati geografici. Esistono altri formati di file utilizzati da ESRI per la memorizzazione dei dati, come il popolare ESRI ShapeFile ([.shp](/it/gis/shp/)). I file APR possono essere aperti con qualsiasi editor di testo come Microsoft Notepad e Notepad ++.

## Formato file APR - Ulteriori informazioni

I file APR vengono archiviati su disco come file di testo semplice che possono essere aperti con qualsiasi editor di testo per l'esame e la modifica.
