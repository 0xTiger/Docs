## Che cos'è un file 3DD?

Un file con estensione .3dd è un file di dati GIS creato con il software applicativo ESRI ArcGlobe. Memorizza le informazioni sui percorsi in cui si trovano i dati 3D sotto forma di geodatabase. Quando un documento viene aperto utilizzando il file .3dd, apparirà nella stessa forma in cui è stato salvato. ArcGlobe è un'applicazione di visualizzazione 3D che consente di visualizzare grandi quantità di dati GIS sulla superficie di un globo. ArcGlobe è una delle numerose applicazioni che vengono associate al pacchetto software ArcGIS.

## Formato file 3DD - Ulteriori informazioni

I file 3DD vengono archiviati su disco in formato binario e il formato file interno di questi file non è disponibile pubblicamente.
