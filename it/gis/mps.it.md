## Che cos'è un file MPS?

Un file MPS è un file mappa che viene utilizzato su dispositivi palmari Pocket PC portatili. Contiene informazioni geografiche su una determinata area come una città o uno stato. Può essere caricato in dispositivi Pocket PC per la navigazione senza la necessità di recuperare dati LIVE da qualsiasi server di dati geospaziali. Il formato di file MPS è stato sviluppato da Microsoft per dispositivi palmari basati su Windows ed è stato utilizzato nella sua applicazione MapPoint GIS che è stata interrotta molto tempo fa.

## Formato file MPS

La struttura dei file interni del formato file MPS non è nota poiché è stata interrotta da un po' di tempo con MapPoint. Alcuni altri formati di file di mappe GIS popolari includono [ESRI Shape Files](/it/gis/shp/), [KML](/it/gis/kml/) e [KMZ](/it/gis/kmz/).

## Alternativa MapPoint
