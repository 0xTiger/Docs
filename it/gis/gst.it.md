## Cos'è un file GST?

Un file GST è un formato di file di mappa GIS che contiene dati geografici e mappa visiva di un'area mappata. Viene utilizzato dal software di mappatura **MapX** (sviluppato da UNEP/GRID-Ginevra e UNEP per la gestione dei dati geospaziali sulle risorse naturali) e da **MapXtreme** (un kit di strumenti di sviluppo software utilizzato per creare applicazioni GIS personalizzate) che sono piattaforme online per la gestione dei dati geospaziali sulle risorse naturali. Un file GST contiene informazioni geografiche in formato numerico e immagini raster sovrapposte alla mappa.

## Formato file GST: ulteriori informazioni

I file GST vengono salvati in un formato di file proprietario e i dettagli del formato di file interno non sono disponibili pubblicamente come riferimento per lo sviluppatore.
