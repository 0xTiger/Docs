## Cos'è un file NMC?

Un file NMC è un file di contenuto del pacchetto di mappe creato con il software ESRI ArcGIS Explorer. Può memorizzare vari elementi di una mappa come sovrapposizioni di immagini, livelli di servizi GIS, livelli [KML](/gis/kml/), livelli raster, funzionalità e altri pacchetti. I file NCM vengono utilizzati per esportare i dati delle mappe che possono essere utilizzati in altre mappe GIS aperte in ArcGIS Explorer. Ciò consente di sovrapporre e visualizzare più mappe una sull'altra.

## Formato file NMC

I file NMC vengono salvati in formato file binario, la cui struttura interna non è disponibile. Questi possono essere condivisi con altri che possono caricare questi file NMC in ArcGIS Explorer alla fine. Ciò rende i file NMC utilizzabili sui computer di altri utenti inviandoli via email ad altri.
