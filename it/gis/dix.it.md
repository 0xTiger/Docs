## Cos'è un file DIX?

Un file DIX è un formato di file proprietario creato con il programma di analisi dei dati geografici DIVA-GIS. Consiste in un file di progetto DIVA-GIS (.div) e altri livelli di dati GIS a cui fa riferimento il progetto. I file DIX possono essere esportati da DIVA-GIS e possono essere condivisi con altri utenti che possono importarli in DIVA. Puoi anche utilizzare i file DIX per archiviare tutti i file rilevanti relativi a un progetto in un'unica posizione.

## Formato file DIX

Un file DIX viene archiviato su disco come file binario compresso. A causa della compressione, i file DIX non occupano molto spazio su disco e possono essere inviati ad altri utenti tramite e-mail. Un file DIX può contenere sia dati raster che vettoriali e può anche memorizzare tabelle di attributi, metadati e altri dati associati ai dati spaziali.
