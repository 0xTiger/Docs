## Cos'è un file 3DC?

Un file 3DC è un file di dati GPS utilizzato dall'applicazione software di navigazione GPS, [iGO](https://en.wikipedia.org/wiki/IGO_(software)). Contiene informazioni sui dati dell'edificio 3D insieme ai metadati per la visualizzazione dell'applicazione. I file 3DC vengono salvati sulla scheda di memoria dell'unità di navigazione e sono per lo più memorizzati nella directory \​iGO\​content\​building\​.

## Informazioni sulla navigazione iGO

Il software iGO viene utilizzato per scopi di navigazione. Ottiene informazioni sulla posizione dell'utente utilizzando dispositivi GPS e aiuta nella navigazione utilizzando mappe e percorsi 3D. È disponibile per Android, iOS e altri dispositivi portatili popolari. Oltre alle funzionalità di navigazione, il software iGO fornisce anche l'archiviazione offline delle mappe nel caso in cui gli utenti abbiano difficoltà a connettersi a Internet per ottenere dati cartografici in tempo reale.
