## Che cos'è un file IMG?

Un file con estensione .img è un file mappa Garmin utilizzato dai dispositivi GPS Garmin. Contiene dati mappa/geospaziali per rappresentare entità mappa come nomi di strade, percorsi di navigazione e punti di interesse. Gli utenti possono creare dati mappa personalizzati, salvarli come file .img e caricarli sui dispositivi Garmin. Garmin alos fornisce dati cartografici preinstallati che vengono memorizzati sui dispositivi Garmin nel formato file IMG. I file IMG possono essere aperti utilizzando applicazioni come Garmin Mobile PC, cGPSmapper, MapUpload e GPS Map Manager.

## Formato file IMG - Ulteriori informazioni

I file IMG vengono salvati come file binari nel formato di file proprietario di Garmin. Le informazioni disponibili in un file IMG possono essere visualizzate su unità GPS Garmin. Alcune unità GPS che supportano l'autorouting possono anche utilizzare questi file mappa come guida durante la navigazione come in auto, in bicicletta o durante le escursioni.

I file GMAP sono simili a un'immagine del disco rigido e si basano su un filesystem composto da file secondari per fornire funzionalità per le unità GPS Garmin. I file secondari .GMP forniscono una sorta di struttura di sottodirectory nonostante il formato del file .gmap sia piatto.
