## Cos'è un file DT0?

Un file DT0 è un file DTED (Digital Terrain Elevation Data) che contiene informazioni di livello 0 per studiare i dati di elevazione del terreno. I dati di elevazione sono spaziati uniformemente in un file DT0 e possono essere letti dai programmi GIS più diffusi come ESRI ArcGIS Pro, TatukGIS, Blue Marble Geographic Global Mapper e Pitney Bowes MapInfo. I file DT0 vengono utilizzati principalmente dalle comunità scientifiche e tecniche per studiare la pendenza, l'elevazione e la rugosità superficiale dei terreni globali.

Il formato file DT0 è simile ai tipi di file [DT1](/gis/dt1/) e [DT2](/gis/dt2/) ma con meno granularità.

### Formato file DT0

I file DT0 vengono salvati come file binari su disco. Questi possono essere letti utilizzando molte applicazioni GIS. I file DT0 possono essere uniti in un catalogo raster di file geodatabase che può essere utilizzato per creare un grande mosaico.

## Come convertire file Raster in DTED?

Esistono più strumenti in grado di convertire file Raster in DTED. [ArgGIS Pro](https://pro.arcgis.com/en/pro-app/latest/tool-reference/data-management/raster-to-dted.htm) è uno di questi strumenti che può convertire Raster in DTED.
