## Cos'è un file PMF?

Un file PMF è un file di mappa di sola lettura generato con ArcMap. Si può dire che si tratta di una versione più breve del formato file MXD che contiene informazioni limitate da condividere con gli utenti che non dispongono di una versione a pagamento del software ArcGIS. Ciò consente la condivisione di informazioni geografiche con altri allo stesso tempo. I file PMF possono essere condivisi con altri utenti via e-mail a causa delle loro dimensioni ridotte a causa delle informazioni limitate che contengono.

I file PMF possono essere aperti con ESRI ArcReader e ESRI ArcGIS Pro.

## Formato file PMF

I file PMF limitano il livello di dati accessibili agli utenti in base al loro abbonamento. Se un utente utilizza una versione gratuita del software ArcGIS, i file PMF limiteranno la visualizzazione delle informazioni come visualizzazioni dati, layout, navigazione, stampa, query, panoramica e funzionalità di zoom della mappa. I dati potrebbero non essere effettivamente contenuti nel file dei limiti, ma accessibili da una posizione remota ospitata sul server ArcMap o disponibile su Internet.
