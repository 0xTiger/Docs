## Che cos'è un file ETA?

Un file con estensione .eta è un file segnaposto di Google Earth creato con il software Google Earth. Contiene informazioni su segnalibri/marcatori per una posizione nel mondo utilizzando parametri come latitudine, longitudine, rotazione, zoom e titolo. Ciò consente agli utenti di avere una visione a volo d'uccello della posizione. Con l'introduzione dei nuovi formati di file, ad esempio [.kml](/it/gis/kml/) e [.kmz](/it/gis/kmz/), il formato di file ETA è stato gradualmente eliminato ed è stato sostituito con questi nuovi formati di file. I file ETA possono essere aperti nel software [Google Earth](https://earth.google.com/web/) disponibile per il download gratuito.


## Formato file ETA - Ulteriori informazioni

I dettagli interni del formato del file ETA non sono disponibili pubblicamente e non sono disponibili software open source o proprietari per leggere i file ETA delle informazioni.
