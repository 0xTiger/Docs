## Cos'è un file AVL?

Un file AVL è un file di legenda che contiene una chiave per una mappa generata con il software ESRI ArcView GIS (Geographic Information System). Contiene informazioni sulla simbologia di riferimento utilizzata nella mappa corrispondente per l'accesso. Un file AVL può contenere più informazioni come simbologia, impostazioni di visualizzazione, come trasparenza, proprietà di visualizzazione dipendenti dalla scala, informazioni sulla tabella unita/tabella correlata, query di definizione, proprietà di etichettatura per i feature layer e proprietà di visualizzazione delle annotazioni per i layer di annotazione a seconda del tipo sorta di strato.

È possibile fare riferimento a un file AVL dal file del progetto ArcView [.APR](/gis/apr/).

## Formato file AVL: ulteriori informazioni

I file AVL vengono salvati in formato file di testo normale. Significa che puoi aprire file AVL in qualsiasi editor di testo come Blocco note, Notepad++ e Apple TextEdit. Una volta aperto, non solo potrai visualizzare il contenuto del file ma anche modificarlo.

### Differenza tra file .LYR e .AVL

 * **Differenza di formato file** - .lyr è un file binario mentre .avl è un file di testo
 * **Differenza di contenuto**: un file .lyr contiene più informazioni di un file .avl. Un file AVL memorizza solo le informazioni sulla simbologia, mentre un file LYR memorizza tutte le informazioni disponibili nella finestra di dialogo delle proprietà del layer in ArcMap.
