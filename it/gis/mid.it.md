## Cos'è un file MID?

Un file MID nella categoria File GIS è un file di dati GIS generato con il software di analisi e visualizzazione dei dati spaziali MapInfo. Di solito viene creato insieme al file MapInfo Interchange Format ([MIF](/gis/mif/)) corrispondente e ha lo stesso nome. I file MID memorizzano set di dati di grandi dimensioni che contengono informazioni sugli attributi per gli oggetti nella mappa e vengono importati negli attributi dell'oggetto.

## Formato file MID: ulteriori informazioni

I file MID vengono salvati su disco in formato file binario. Gli attributi nei file MID vengono salvati nello stesso ordine di quelli del file MIF corrispondente. Ad esempio, la terza entità geometrica nel file .mif ha gli attributi memorizzati nella terza riga del file .mid.
