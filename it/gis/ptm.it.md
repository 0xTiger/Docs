## Cos'è un file PTM?

Un file PTM è un file di mappa utilizzato dal software di mappatura MapPoint sviluppato da Microsoft che ormai è fuori produzione. Contiene dati cartografici estratti da Terra Server di Microsoft. Le informazioni sulle mappe contengono dati relativi ai percorsi e ai territori dei viaggi d'affari. MapPoint è stato successivamente sostituito dalle mappe Bing. Altri file utilizzati insieme ai file PTM includono MPP (MapPoint Map), WPT (MapPoint Waypoint) e RTE (MapPoint Route).

## Formato file PTM

Il formato file PTM è un formato file proprietario e la struttura file interna dei file .ptm non è documentata pubblicamente.
