## Che cos'è un file GLB?

Un file con estensione .glb è un file condivisibile a livello globale generato e utilizzato dal software AGI Systems Tool Kit (STK). Contiene informazioni sul globo 3D che possono essere condivise con altri utenti per essere utilizzate nei loro progetti. Il software STK dei sistemi AGI viene utilizzato per simulare missioni aerospaziali, sistemi di difesa e sistemi di telecomunicazione. GLB è il formato di file standard per STK e molti diversi modelli 3D come [.gltf](/it/3d/gltf/) possono essere convertiti in questo formato. I file GLB possono essere aperti con il software AGI Simulation Toolkit e con qualsiasi editor di testo.

## Formato file GLB - Ulteriori informazioni

I file GLB memorizzano informazioni su trame, immagini e altri file utilizzati in un globo per un progetto. I riferimenti alla posizione dei file collegati sono archiviati nei file GLB e questi file aggiuntivi devono essere condivisi anche con altri utenti durante la condivisione del file globo GLB per un caricamento corretto. I file GLB memorizzano tutte queste informazioni in file di testo semplice che possono essere aperti con qualsiasi editor di testo.
