Un file di mappa GIS (Geographic Information System) è costituito da informazioni spaziali o geografiche su determinate località o aree. Un software di mappatura GIS è un programma o un insieme di programmi che presentano i dati spaziali o geografici ai visualizzatori per la visualizzazione, l'analisi e la gestione. Un file GIS può essere creato manualmente, estratto da un dispositivo GPS o registrato tramite dispositivi e applicazioni di telerilevamento. GIS trova utilizzo in quasi tutti i campi della vita; dall'agricoltura all'urbanistica, all'analisi delle risorse, all'urbanistica e alla creazione di strategie per la risoluzione dei problemi.

Le estensioni di file GIS comuni e i relativi formati di file includono **[GPX](/it/gis/gpx/)** (GPS Exchange File Format), **[KML](/it/gis/kml/)** (Keyhole Markup Language File) e **[SHP](/it/gis/shp/)** (file di forma ESRI).

Hai domande relative ai formati di file GIS? Visita la nostra community [forum](https://forum.fileformat.com/c/gis/7) per beneficiare delle conoscenze condivise dagli esperti di formati di file.
