## Che cos'è un file AVERY?

Un file AVERY è un documento stampabile che contiene una o più etichette a scopo di stampa. Viene creato con il software Avery Design & Print o Design & Print Online. Le etichette all'interno di un file AVERY possono comprendere diversi elementi come testo, grafica e informazioni sul layout di pagina. Queste informazioni vengono utilizzate dalle aziende di stampa per la stampa di una varietà di elementi di design come biglietti da visita, targhette nominative, inserti per raccoglitori, etichette per indirizzi, etichette per dischi, modelli di necrologi e biglietti di auguri.

## Formato file AVERY - Ulteriori informazioni

I file AVERY sono archiviati in formato binario. Questi possono essere aperti con il software Avery Design & Print o Design & Print Online. I design creati con Design & Print possono essere utilizzati per creare etichette AVERY che possono servire a diversi scopi. Questi disegni possono essere salvati come file [PDF](/it/pdf/) che possono essere condivisi con la società di stampe per scopi di stampa.

### Stampa di un file AVERY

Un file AVERY può essere salvato come PDF o può essere stampato direttamente su qualsiasi stampante standard. Per stampare il file, apri il file nel software Design & Print e usa l'opzione Stampa dal menu File per inviare il file alla stampante per la stampa.
