## Che cos'è un file OXPS? ##

Il formato di file OXPS è noto come Open XML Paper Specification. È un linguaggio di descrizione della pagina e un formato del documento. Microsoft è lo sviluppatore di questo formato. Il formato di file OXPS è molto familiare a questi file PDF. La configurazione del file OXPS è la stessa di un formato XPS. L'unica differenza è la descrizione di OXPS come formato aperto. L'estensione del file .oxps è la versione più recente del formato di file XPS originato da Microsoft ed Ecma International. Se parliamo delle specifiche, i file OXPS e XPS vengono utilizzati per pubblicare, condividere e allocare documenti a layout fisso. In sostituzione dei PDF, il formato di file OXPS può memorizzare dati di testo e multimediali, con funzionalità di organizzazione e layout di stampa.

## Breve storia ##

Il team Microsoft ha scelto Global Graphics per fornire consulenza su XPS nel 2003 e il team di sviluppo ha lavorato a stretto contatto con i team Windows per sviluppare le specifiche e l'architettura di riferimento per il nuovo formato. Riconoscendo le specifiche XPS, nel giugno 2007 Ecma International ha istituito il Technical Committee 46 (TC46) per sviluppare uno standard basato sulla specifica OpenXPS. L'Open XML Paper Specification approvato da Ecma International è uno standard Ecma (ECMA-388) alla 97a Assemblea Generale tenutasi a Budapest il 16 giugno 2009.
