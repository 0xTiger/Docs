## Che cos'è un file INDD?

Un file con estensione .indd è un file di progetto Adobe InDesign che contiene le informazioni sul layout di pagina del progetto. INDD sta per InDesign Document. Questi file memorizzano comunemente informazioni sulla formattazione della pagina, libri, riviste, giornali, volantini, opuscoli o altri supporti di stampa come modelli di pagine Web e dispositivi mobili. Il tipo di file INDD è un formato di file proprietario di Adobe ed è il formato di file predefinito per InDesign. Designer grafici, giornalisti, esperti di marketing ed editori che lavorano con InDesign utilizzano questo tipo di file.

## Ulteriori informazioni sul formato file INDD

I file INDD sono file vettoriali che possono essere ridimensionati senza perdere la risoluzione. Questi sono basati su grafici complessi che possono essere espansi all'infinito. Impacchetta tutte le informazioni sui caratteri e sui layout da condividere con un team.

I file INDD sono perfetti per la progettazione di stampe accattivanti e contenuti online. Un file INDD può estendersi su più pagine e può fungere da modelli per la creazione di pagine, capitoli e pubblicazioni. Quando il tuo progetto è completo, può essere pubblicato su media online senza la necessità di conversione, zippare o inviare e-mail.

## Il file INDD può essere convertito in altri formati?

I file INDD possono essere convertiti in diversi altri formati di file. La conversione più comune è INDD in PDF. InDesign ti consente di salvare il file INDD in PDF per condividerlo con altri utenti. Ti consente anche di esportare file INDD in JPG, [EPS](/it/page-description-language/eps/), [EPUB](/it/ebook/epub/), [SWF](/it/page-description-language/swf/) , FLA, [HTML](/it/web/html/), [XML](/it/web/xml/) e IDML.

## Come viene creato il file INDD?

Un file Indd viene creato quando crei un nuovo progetto con [Adobe InDesign](https://www.adobe.com/products/indesign.html) e lo salvi su disco. È possibile aprire il file di progetto Indd, apportarvi modifiche e salvare il file aggiornato per le modifiche.
