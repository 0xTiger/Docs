## Che cos'è un file INDT?

Un file con estensione .indt è un file modello di layout Adobe InDesign che contiene impostazioni predefinite. Il design e il contenuto di questi file includono modelli di newsletter, libri, riviste, volantini e supporti di stampa. Al termine della modifica, il file INDT può essere salvato su disco come file [INDD](/it/page-description-language/indd/). L'utente può anche creare un nuovo file INDT da InDesign e convertirlo in altri formati come [PDF](/it/pdf/), [EPUB](/it/ebook/epub/) o [JPEG](/it/image/jpeg/) .

## Formato file INDT - Ulteriori informazioni

Adobe InDesign fornisce diversi modelli con il software e online. Questi modelli forniscono agli utenti un punto di partenza per progettare i propri media online o cartacei. I file INDT sono leggeri, hanno molte caratteristiche di progettazione (forme, testo, colori e filtri diversi) e hanno una risoluzione infinita.
