## Cos'è un file GXF?

Un file GXF è un file di caratteri dello strumento di progettazione CAD per Windows, General CADD Pro. Il software utilizza il carattere per inserire testo e altri simboli in un disegno CAD. I caratteri GXF devono essere disponibili per il software General CADD Pro affinché un disegno CAD venga visualizzato correttamente.

Puoi aprire un file GXF utilizzando [General CADD Pro](https://www.generalcadd.com/oldSite/101release.htm).

## Informazioni su General CADD Pro

Generale CADD Pro offre una struttura di comando a due lettere incorporata e utilizzata da molti professionisti del disegno. Supporta il formato file ACAD 2013. Fornisce inoltre supporto per i comandi di rilevamento/distanza del geometra e il comando Campo per finire.
