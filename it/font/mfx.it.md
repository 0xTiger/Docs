## Che cos'è un file MFX?

Un file MFX è un file di font utilizzato dal gioco di simulazione Maxis SimCity 4. Viene utilizzato per eseguire il rendering del testo del gioco sullo schermo da uno dei tipi di carattere selezionati, inclusi Arial, Courier o qualsiasi altro carattere disponibile nel file del carattere. I file MFX vengono installati con il gioco SimCity 4 e non devono essere creati manualmente.

SimCity 4 è un gioco di simulazione per computer per la creazione di simulazioni di costruzione di città. Maxis, lo sviluppatore di SimCity, è una sussidiaria di Electronic Arts. Il gioco è stato inizialmente lanciato nel gennaio 2003.
