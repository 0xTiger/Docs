## Che cos'è un file JFPROJ?

Un file con estensione .jfproj è un font personalizzato basato su JSON creato con l'applicazione Microsoft Font Maker su Windows 10. Questi possono essere creati utilizzando una penna elettronica con dispositivi touch screen come tablet, sebbene sia possibile utilizzare anche altre periferiche per computer come il mouse nel caso di computer. I file JFPROJ contengono i glifi utilizzati per visualizzare i caratteri sullo schermo definendo la spaziatura tra caratteri e parole e la dimensione del carattere. L'app Microsoft Font Maker è disponibile per il download su Windows Store.
