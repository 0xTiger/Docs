## Che cos'è un file PFM?

Un file con estensione .pfm è un file Printer Font Metrics che contiene le misurazioni dei caratteri incluse nel file del font. È necessario avere il file PFM insieme al file .pfb(/it/font/pfb/) associato nella stessa cartella durante l'installazione del successivo. I file PFM sono codificati in binario e non sono leggibili dall'uomo.
