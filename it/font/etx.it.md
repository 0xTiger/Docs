## Cos'è un file ETX?

Un file ETX è un file di caratteri utilizzato dai sistemi di composizione, TeX e LaTeX. Viene salvato come file di testo e definisce l'aspetto di un carattere tramite la sua codifica. I file dei caratteri ETX vengono caricati nel software che fornisce la possibilità di lavorare con documenti Tex. Una volta caricati, i file dei caratteri ETX possono essere utilizzati per la personalizzazione dei documenti TeX.
