## Cos'è un file VLW?

Un file VLW è un formato di carattere utilizzato dal software per la creazione di immagini e animazioni, **Processing**. Memorizza i caratteri vettoriali in modo compatto che può essere ridimensionato a qualsiasi dimensione senza perdere la qualità. Questo trova il suo utilizzo nella grafica e nel visual design. L'elaborazione offre la possibilità di creare immagini e animazioni in cui vengono utilizzati questi glifi dei caratteri VLW.

## Formato file VLW: ulteriori informazioni

I file VLW vengono salvati su disco come file binari. In Processing, puoi utilizzare la funzione **createFont()** per creare un oggetto font da un file VLW. Puoi quindi utilizzare questo oggetto font per disegnare testo nei tuoi schizzi utilizzando la funzione **text()**. L'elaborazione può leggere file VLW utilizzando il comando **loadFont()**.
