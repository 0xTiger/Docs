## Che cos'è un file PFA?

Un file con estensione .pfa è una versione Printer Font ASCII del font Type 1 di Adobe e contiene i dati del glifo del font. Il file è in formato leggibile dall'uomo e può essere aperto in qualsiasi editor di testo. L'interprete PostScript può utilizzare questo tipo di carattere nel suo insieme per il suo funzionamento. La versione Printer Font Binary (PFB) del font Type 1 di Adobe contiene le stesse informazioni. Tuttavia, a differenza di PFA, è in formato file binario e non è leggibile dall'uomo. I file PFA possono essere incorporati nel sistema copiandoli nella cartella Fonts del sistema.
