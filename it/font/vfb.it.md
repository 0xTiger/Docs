## Cos'è un file VFB?

Un file VFB è un file di font creato con FontLab Studio. Memorizza i dati dei caratteri come forme di lettere, numeri e simboli in formato digitale. I glifi di questi caratteri digitali compongono l'alfabeto e altri simboli. Il formato file VFB memorizza i caratteri in un formato file vettoriale. Ciò rende i glifi indipendenti dal ridimensionamento senza perdere qualità.

Puoi aprire il file VFB utilizzando FontLab Studio e FontLab TransType Pro.

## Formato file VFB: ulteriori informazioni

Il formato file VFB è proprietario e può essere letto e scritto solo da software che lo supporta specificamente, come alcuni software di modifica dei caratteri. I file VFB vengono generalmente utilizzati da tipografi e grafici per creare e modificare i propri caratteri personalizzati.

### Converti il formato file VFB

Puoi convertire:

 * Da VFB a [.TTF](/font/ttf/)
 * Da VFB a [.OTF](/font/otf/)
 * Da VFB a [.PFB](/font/pfb/)
 * Da VFB a [.PFA](/font/pfa/)
