## Che cos'è un file PFB?

Un file con estensione .pfb è una versione binaria del font della stampante del font Type 1 di Adobe. È simile ai file .pfa(/it/font/pfa/) ma sono di natura binaria. I file PFB memorizzano i caratteri Adobe Type 1 utilizzati per il sistema operativo Windows. Questi contengono i dati del glifo del carattere e memorizzano i dati in una forma compatta in formato binario. Oltre ad avere contenuto codificato binario, i file PFB sono crittografati. I file PFB richiedevano anche file .pfm insieme a quelli che includevano le misurazioni dei caratteri all'interno del font. Per installare i font PFB, entrambi i file .pfb e .pfm devono essere inseriti nel sistema.
