## Cos'è un file FOT?

Un file FOT è un file di caratteri utilizzato per i file di dati di struttura dei caratteri. Contiene informazioni sulle forme generali e sui contorni dei caratteri in un font. Altre informazioni contenute in un file .fot includono il nome, lo stile e la dimensione del carattere. I file FOT vengono utilizzati dalle applicazioni che supportano TrueType (file [.TTF ](/font/ttf/)). Viene utilizzato più spesso dagli installatori di applicazioni durante l'installazione dei caratteri.

## Maggiori informazioni sul formato file FOT

I file FOT vengono salvati su disco come file binari e la loro struttura di file interna non è nota come riferimento per lo sviluppatore.
