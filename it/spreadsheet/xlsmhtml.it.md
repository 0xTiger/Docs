## Che cos'è un file XLSMHTML?

Un file con estensione .xlsmhtml è una versione [HTML](/it/web/html/) del file XLS originale che viene esportato da Microsoft Excel per visualizzare i dati in un browser. Il formato è stato a lungo interrotto da Excel e non è più supportato. Il file XLSMHTML esportato seguiva il formato file XLS e questi potevano essere aperti e visualizzati nei browser web. Excel consente di esportare i dati del foglio di calcolo nei formati di file [MHT](/it/web/mht/) e [MHTML](/it/web/mhtml/) e ora sono ampiamente utilizzati.

## Formato file XLSMHTML - Ulteriori informazioni

I file XLSMHTML sono stati generati con codice HTML per rappresentare i dati del foglio di calcolo in un formato simile a XLS. Il codice HTML all'interno del file XLSMHTML è organizzato in modo tale che il file di output sia simile al file originale quando viene aperto in un browser web. Le specifiche del formato del file XLSMHTML non sono in quanto tali rese pubbliche da Excel e non sono disponibili riferimenti di questo tipo per consentire agli sviluppatori di comprendere e sviluppare applicazioni per esso.
