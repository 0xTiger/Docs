## Che cos'è un file PMDX?

Un file PMDX è un foglio di calcolo creato con l'ultima versione di [PlanMaker](https://www.softmaker.de/softmaker-office-planmaker), un software che fa parte di SoftMaker Office. Memorizza i dati nelle celle sotto forma di righe e colonne. Ogni cella può contenere stringhe, numeri, formule o altri dati formattati. I file PMDX hanno sostituito i file della versione precedente [PMD](/it/spreadsheet/pmd/) con l'aggiornamento di PlanMaker. XLSX è il formato di file standard supportato da PlanMaker e non necessita di alcuna conversione durante l'apertura di questi file con Microsoft Excel. PlanMaker supporta anche l'utilizzo di file [XLS](/it/spreadsheet/xls/) e la conversione in altri formati come [PDF](/it/pdf/).

## Formato file PMDX

I file PMDX vengono archiviati su disco come file binari in un formato di file proprietario e le sue specifiche non sono disponibili pubblicamente.
