## Che cos'è un file BKS? ##

Un file BKS è un file di backup [Microsoft Works](https://en.wikipedia.org/wiki/Microsoft_Works) che memorizza l'ultimo stato di lavoro di un foglio di calcolo. Ciò salva l'ultimo stato valido del foglio di calcolo da ricaricare nuovamente nel caso in cui il documento originale venga danneggiato o non possa essere aperto per qualsiasi motivo. BKS è simile al formato di file [WPS](/it/word-processing/wps/) che contiene contenuto del documento simile al documento Microsoft Word [DOC](/it/word-processing/doc/). Microsoft Works è stato interrotto ei file BKS non sono più supportati.

## Formato file BKS

I file BKS vengono archiviati su disco come file binari e le relative specifiche del formato file non sono disponibili pubblicamente.
