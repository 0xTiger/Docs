## Che cos'è un file IMP? ##

Un file IMP è un file di foglio di calcolo creato con il software [Lotus Improv](https://en.wikipedia.org/wiki/Lotus_Improv) per memorizzare i dati in righe e colonne. Il contenuto di un file IMP è separato in tre diversi segmenti; vale a dire dati, visualizzazione dei dati e formule. Sebbene lo scopo di questo approccio fosse quello di semplificare la modifica, ciò rendeva difficile il trasferimento dei dati da un file IMP ad altri tipi di file di applicazioni comuni come Excel. I file IMP sono stati contrassegnati come obsoleti e non sono più un formato di file supportato. Lotus Improv è stata successivamente acquisita da IBM nel 1996.

## Formato file IMP

I file IMP vengono archiviati su disco come file binari e il loro formato di file interno non è disponibile pubblicamente. Lotus Improv è stato sostituito da IBM Lotus 1-2-3 in seguito.
