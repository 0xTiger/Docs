## Che cos'è un file PMD?

Un file PMD è un file di foglio di calcolo creato con il software [PlanMaker](https://www.softmaker.de/softmaker-office-planmaker) fornito con SoftMaker Office. Memorizza i dati del foglio di calcolo in righe e colonne simili ai file Excel. [XLSX](/it/spreadsheet/xlsx/) è il formato di file standard di PlanMaker che semplifica l'utilizzo del software al posto di Microsoft Excel. Un file PMD può contenere formule e grafici oltre ai dati archiviati nelle celle. I file creati con PlanMaker possono essere aperti con Microsoft Excel senza alcuna conversione aggiuntiva. Può anche modificare file [XLS](/it/spreadsheet/xls/) e supporta formati di file aggiuntivi come [PDF](/it/pdf/).

## Formato file PMD

I file PMD vengono archiviati su disco come file binari in un formato di file proprietario e le sue specifiche non sono disponibili pubblicamente.
