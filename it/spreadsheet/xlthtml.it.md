## Che cos'è un file XLTHTML?

Un file con estensione .xlthtml è una versione [HTML](/it/web/html/) di file modello [XLT](/it/spreadsheet/xlt/) ed è stato generato utilizzando Microsoft Excel nei primi giorni. Questo formato non è più supportato da Excel ed è stato sostituito dall'esportazione dei dati nei formati di file [MHT](/it/web/mht/) e [MHTML](/it/web/mhtml/). L'idea di base alla base del file XLTHTML era quella di consentire agli utenti di visualizzare i file modello XLT in un browser Web invece della necessità di avere Excel installato sul sistema. I file esportati in formato file XLTHTML possono essere aperti nei browser.

## Formato file XLTHTML - Ulteriori informazioni

I file XLTHTML utilizzavano il formato di file HTML per salvare i modelli XLT per l'apertura in un browser web. XLT stesso si basa sul formato di file binario di Excel introdotto con la versione Excel 97-2003. Questi file possono essere utilizzati come file iniziali con formattazione definita per creare documenti normali per la compilazione dei dati del foglio di calcolo. Solo per riferimento, XLT si basa sul binario [Specifiche del formato file XLS](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) che è documentato da Microsoft per riferimento dello sviluppatore.
