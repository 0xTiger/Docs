## Che cos'è un file NB?

Un file con estensione .nb è un formato di file per notebook Wolfram che salva le istruzioni per le istruzioni matematiche in un file di testo. Può contenere molti tipi diversi di dati come calcolo in tempo reale, interfacce dinamiche arbitrarie, input di composizione completa, input di immagini, annotazione automatica del codice, un'interfaccia programmatica di alto livello completa e migliaia di funzioni e opzioni accuratamente organizzate. Le istruzioni testuali sono l'input e l'output di Mathematica che viene generato e aggiornato man mano che le istruzioni di input vengono inserite nel file.

## Formato file Wolfram Notebook NB - Ulteriori informazioni

I file Wolfram Notebook NB vengono salvati in un formato testuale semplice, che è un formato di file leggibile dall'uomo. Il contenuto di un taccuino è organizzato in sezioni come testo normale in cui ciascuna è rappresentata da gruppi di celle, simili a un foglio di calcolo. L'intervallo di questi gruppi è definito da una parentesi verso la fine di ogni cella. Lo stile assegnato a una cella determina il suo ruolo all'interno del taccuino come descritto di seguito.

### Notebook come lingua Wolfram

Quando si intende salvare il Notebook costituito da istruzioni matematiche per l'esecuzione da parte di Wolfram Language Kernal, alle celle nel documento viene assegnato automaticamente lo stile di testo 'Input'. Questo dice al kernel di considerarle come istruzioni che vengono eseguite quando l'utente emette una combinazione di tasti `Maiusc+Invio`. ### Quaderni come documenti

I taccuini Mathematica possono essere in formato documento per dare un'occhiata a ciò che vedi e ciò che ottieni (WYSIWYG). Questi documenti sono gli stessi visualizzati sullo schermo o su carta stampata e sono interattivi. Per questo, lo `Stile del testo` viene assegnato alla cella per visualizzare i contenuti senza eseguire le istruzioni dal kernel.

## Esporta taccuini Mathematica

I taccuini Mathematica possono essere esportati in molti formati di file diversi come PDF, grafica, GIS, compresso e fogli di calcolo.
