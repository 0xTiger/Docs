## Che cos'è un file FCS?

Un file FCS è un file di foglio di calcolo creato con il software PFS: First Choice che è una suite per ufficio composta da un elaboratore di testi, foglio di calcolo, grafica, database e telecomunicazioni. FCS sta per First Choice Spreadsheet e può essere aperto con la suite PFS:First Choice per DOS di [Software Publishing Corporation](https://en.wikipedia.org/wiki/Software_Publishing_Corporation).
