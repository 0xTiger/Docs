## Che cos'è un file XLT?

I file con estensione .xlt sono file modello creati con Microsoft Excel, un'applicazione per fogli di calcolo che fa parte della suite Microsoft Office. Microsoft Office 97-2003 supportava la creazione di nuovi file XLT e l'apertura di questi. L'ultima versione di Excel è ancora in grado di aprire i file modello di questo vecchio formato. Tale file modello viene utilizzato per creare rapidamente nuovi file Excel con dati e impostazioni predefiniti come formattazione della pagina, dimensione del carattere, margini, grafici, ecc. che possono essere ulteriormente salvati come nuovi file [.xls](/it/spreadsheet/xls/).

## Formato file XLT

Il formato di file XLT è basato sul formato di file binario di Excel inizialmente supportato con la versione di Excel 97-2003. È stato sostituito con il formato di file [XLTX](/it/spreadsheet/xltx/) con l'introduzione di Excel 2007, basato sul formato Office Open XML. Oltre a Microsoft Excel, esistono altre applicazioni che supportano l'utilizzo di questo formato di file come Lotus Translation Table, PlanMaker, LibreOffice Calc, ecc. Sono disponibili diverse applicazioni e [API](https://products.aspose.com/cells) per convertire XLT in altri formati di file come:

* XLT in PDF
* XLT a DOC
* XLT al testo
* XLT a Lotus
* XLT a ODT
* XLT a ODS
* XLT in HTML
* XLT in CSV

XLT si basa sul binario [Specifiche del formato file XLS](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) che è documentato da Microsoft come riferimento per gli sviluppatori.
