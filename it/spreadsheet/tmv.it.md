## Che cos'è un file TMV?

Un file TMV è un file di foglio di calcolo creato con il software [TimeMap](https://www.lexisnexis.com/en-us/products/casemap.page) da LexisNexis. TimeMap ti consente di creare linee temporali con la data e l'ora dei fatti specificate. Con l'immissione di nuovi fatti, il software crea dei factbox per questi e li posiziona al verificarsi della sequenza temporale. Il progetto TMV può comprendere caselle dei fatti, caselle di testo e il testo al loro interno può essere formattato utilizzando diversi tipi di carattere. I file TMV possono essere aperti con il software TimeMap. LexisNexis fornisce anche uno strumento di collaborazione online ora in un ambiente basato su cloud.

## Formato file TMV

I file TMV vengono archiviati su disco in un formato di file proprietario CaseMap le cui specifiche del formato file non sono disponibili online. CaseMap è stata acquisita da LixesNexis.
