## Che cos'è un file PMVX?

Un file con estensione .pmvx è un file modello di foglio di calcolo creato con il programma di foglio di calcolo [PlanMaker](https://www.softmaker.de/softmaker-office-planmaker), che fa parte di Softmaker Office. I dati vengono archiviati in righe e colonne nei file PMVX allo stesso modo di Microsoft Excel. I file PMVX memorizzano le impostazioni predefinite come un modello che può essere utilizzato per generare file PMD e PMDX dopo aver popolato i dati al suo interno. [XLSX](/it/spreadsheet/xlsx/) è il formato file predefinito per PlanMaker e può essere aperto direttamente con Microsoft Excel. Puoi anche modificare file [XLS](/it/spreadsheet/xls/) e creare file [PDF](/it/pdf/) usando PlanMaker.

## Formato file PMVX

I file PMVX vengono archiviati su disco nel formato di file proprietario di PlanMaker e le loro strutture di file interne sono sconosciute. Per quanto riguarda il salvataggio dei file PMVX, salva i dati nel formato Open XML come file XLSX.
