## Che cos'è un file TSV?

Un formato di file TSV (Tab-Separated Values) rappresenta i dati separati con tabulazioni in formato testo normale. Il formato del file, simile a [CSV](/it/spreadsheet/csv/), viene utilizzato per organizzare i dati in modo strutturato al fine di importare ed esportare tra diverse applicazioni. Il formato viene utilizzato principalmente per l'importazione/esportazione di dati e lo scambio in applicazioni di fogli di calcolo e database. Ogni record in un file TSV è contenuto in una singola riga di file di testo in cui ogni valore di campo è separato da un carattere di tabulazione. Il tipo di supporto per il formato file TSV è testo/valori separati da tabulazioni.
