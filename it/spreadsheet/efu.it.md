## Che cos'è un file EFU?

Un file EFU è un file di foglio di calcolo [CSV](/it/spreadsheet/csv/) creato con il software Everything da [voidtools](https://www.voidtools.com/). Contiene un elenco di file cercati utilizzando il motore di ricerca Tutto per Windows. I risultati in un file EFU sono delimitati da CSV e possono essere aperti con programmi di fogli di calcolo come Microsoft Excel. I file EFU sono leggibili dall'uomo e possono essere aperti con qualsiasi editor di testo come Microsoft Notepad, Notepad ++ e Apple TextEdit.

## Formato file EFU - Ulteriori informazioni

I file EFU vengono salvati su disco in un file con valori separati da virgole che contiene informazioni sui risultati ottenuti dall'applicazione del motore di ricerca Everything. Questi record contengono informazioni sui file trovati come il nome del file, la dimensione, la creazione e la data dell'ultima modifica.
