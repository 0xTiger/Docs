## Che cos'è un file XLR?

Un file con estensione .xlr è un formato di file di foglio di calcolo di Microsoft Works che archivia i dati in righe e colonne. Microsoft Works è stato sviluppato per essere utilizzato come suite di produttività iniziale ed è rimasto funzionante dal 1987 al 2009. Comprendeva un elaboratore di testi, un foglio di calcolo e un sistema di gestione del database. Successivamente è stato interrotto dopo l'introduzione di Microsoft Office con funzionalità più avanzate incluse.

## Formato file XLR - Ulteriori informazioni

I file XLR sono stati salvati come file binari senza dettagli sulle specifiche del formato file disponibili pubblicamente. Gli sviluppatori, quindi, non hanno un'opzione per analizzare la struttura dei file e scrivere applicazioni per la lettura/scrittura su questi file a livello di codice. Il foglio di calcolo di Works utilizza lo stesso formato di file Excel per archiviare i dati in file .xlr.
