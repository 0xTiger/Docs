## Che cos'è un file XLA?

Un file con estensione .xla è un formato di file dei componenti aggiuntivi di Microsoft Excel che aggiunge moduli, funzioni extra e altri strumenti a Microsoft Excel. Gli annunci aggiuntivi forniscono funzionalità aggiuntive ai fogli di calcolo Excel e sono supportati nella versione 2007 o successiva. I file XLA hanno il tipo di supporto mime application/vnd.ms-excel. Questi possono essere aperti utilizzando Microsoft Excel disponibile per Windows e MacOS.

## Formato file XLA

I file XLA sono basati su un formato di dati binari. Tuttavia, non sono disponibili ulteriori informazioni sulle specifiche del suo formato di file. Il presupposto generale è che XLA segua la [[MS-XLS] - Struttura del formato file binario di Excel](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx).
