## Che cos'è un file NUMBERS-TEF?
I file NUMBERS-TEF vengono creati quando il software per fogli di calcolo iWork Numbers di Apple salva i fogli di calcolo su iCloud. Mentre i fogli di calcolo di Numbers salvati sul computer locale utilizzano l'estensione **.numbers**.


## Differenza tra NUMERI e NUMERI-TEF
I file NUMBERS-TEF sono fogli di calcolo simili come .numbers. Entrambi i file possono essere creati utilizzando il software per fogli di calcolo iWork Numbers di Apple. Tieni presente che gli Apple iWork Numbers sono inclusi in iWork Productivity Suite. NUMBERS-TEF è solo un suffisso **-TEF** aggiunto ai documenti di Numbers durante il salvataggio su iCloud per l'uso su un dispositivo iOS o un altro Mac.
Le versioni Mac OS o iOS di Numbers aprono **.numbers-tef** senza alcun problema. Il foglio di calcolo verrà caricato correttamente proprio come il file **.numbers**.
