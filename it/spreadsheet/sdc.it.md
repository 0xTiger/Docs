## Che cos'è un file SDC? ##

Un file SDC è un file di foglio di calcolo creato con [Apache OpenOffice Calc](https://en.wikipedia.org/wiki/LibreOffice_Calc) che fa anche parte di StarOffice. Come altre applicazioni per fogli di calcolo, i dati vengono archiviati in un file SDC in una griglia di celle in righe e colonne. I file SDC hanno tipo MIME application/vnd.stardivision.calc ed è stato progettato per essere utilizzato per i fogli di calcolo StarCalc 5.x. Calc è un programma di fogli di calcolo gratuito ed è utilizzato da nuovi arrivati e professionisti. Può esportare dati da file SDC a Microsoft Excel XLS e formati di file PDF. Oltre a Calc, [LibreOffice](https://www.libreoffice.org/) e [Planamesa NeoOffice](https://www.neooffice.org/neojava/en/index.php).

## Formato file DSC

I file SDC vengono salvati in un formato di file proprietario e il loro formato di file interno non è noto.
