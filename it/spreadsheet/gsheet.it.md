## Cos'è un file GSHEET?

L'estensione di un file di Fogli Google è ".gsheet". Tuttavia, questa estensione di file non è visibile all'utente quando lavora con il file in Google Drive o nell'app web Fogli Google. Invece, i file di Fogli Google vengono generalmente salvati con un nome file che include il nome del file e la parola "Foglio" o "Fogli" (ad esempio "Foglio budget" o "Fogli vendite"). Quando scarichi un file di Fogli Google da Google Drive, in genere viene salvato come file Microsoft Excel con estensione ".xlsx".

## Maggiori informazioni

Fogli Google è un programma di fogli di calcolo basato sul Web offerto da Google come parte della suite di produttività di Google Drive. Consente agli utenti di creare, modificare e collaborare su fogli di calcolo online, senza la necessità di software o hardware aggiuntivi.

Fogli Google offre un'ampia gamma di funzionalità e strumenti per organizzare e analizzare i dati, tra cui formule, diagrammi, grafici, tabelle pivot e formattazione condizionale. Si integra anche con altri prodotti Google, come Google Forms e Google Docs, nonché con applicazioni di terze parti, tramite le sue API e componenti aggiuntivi.

Uno dei principali vantaggi di Fogli Google sono le sue funzionalità collaborative, che consentono a più utenti di lavorare sullo stesso documento contemporaneamente, in tempo reale. Ciò lo rende ideale per progetti di gruppo, lavoro remoto e collaborazione online.

Fogli Google può essere utilizzato gratuitamente da singoli e piccoli team, con un limite di 15 GB di spazio di archiviazione per account. Le organizzazioni più grandi possono abbonarsi a Google Workspace (precedentemente noto come G Suite) per accedere a funzionalità aggiuntive di archiviazione, sicurezza e amministrazione.

I file di Fogli Google vengono salvati su Google Drive, un servizio di archiviazione e sincronizzazione di file basato su cloud fornito da Google. È possibile accedere ai file da qualsiasi dispositivo dotato di connessione Internet e browser Web e possono essere condivisi con altri per la visualizzazione o la modifica.

## Come aprire il file GHSEET?

L'estensione file ".gsheet" viene utilizzata da Fogli Google per salvare i file nel suo formato nativo. Per aprire un file .gsheet, dovrai disporre di un account Google e accedere a Fogli Google. Ecco i passaggi per aprire un file .gsheet:

1. Apri il browser web e vai al sito web di Google Drive (drive.google.com).
2. Accedi al tuo account Google se non l'hai già fatto.
3. Individua il file .gsheet che desideri aprire nel tuo Google Drive. Puoi accedere al file manualmente oppure utilizzare la barra di ricerca per trovarlo.
4. Fai doppio clic sul file .gsheet per aprirlo in Fogli Google. Se hai configurato un programma predefinito diverso per aprire i file .gsheet sul tuo computer, potrebbe essere necessario fare clic con il pulsante destro del mouse sul file e selezionare "Apri con" > "Fogli Google".

Una volta aperto il file .gsheet in Fogli Google, puoi visualizzarne e modificarne i contenuti come faresti con qualsiasi altro file di foglio di calcolo. Puoi anche salvare il file in un formato diverso, come Microsoft Excel o CSV, selezionando "File" > "Download" e scegliendo il formato desiderato.
