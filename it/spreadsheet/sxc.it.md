## Che cos'è un file SXC? ##

Il formato di file SXC(Sun XML Calc) appartiene a una suite per ufficio chiamata OpenOffice.org. Questo formato generalmente soddisfa le esigenze del foglio di calcolo degli utenti in quanto è un formato di file di foglio di calcolo basato su XML. Il formato SXC supporta formule, funzioni, macro e grafici insieme a DataPilot, che è una caratteristica incredibile perché individua automaticamente e fornisce riepiloghi dei dati grezzi importati. I file creati con questo software vengono salvati con estensione .sxc.


## Formato file SXC ##

I file, con estensione .sxc, possono essere aperti con Microsoft Excel, che è un software per fogli di calcolo della suite Microsoft Office. Esistono altri software in grado di aprire file con estensione .sxc, ad esempio Libre Office, StarOffice ecc.

Il tipo di contenuto per questo formato è:

* applicazione/vnd.sun.xml.calc
* application/vnd.sun.xml.calc.template

## Compatibilità file ##

I file SXC sono compatibili con Apache Open Office calc e i dati possono essere esportati in file Microsoft Excel e anche in altri formati.

## Gestione dei file ##

I comandi ImportMatrix ed ExportMatrix possono leggere e scrivere file SXC. Quando l'origine è un file SXC e anche la destinazione è un file SXC, **f** viene considerato come un file di foglio di calcolo Sun XML Calc. Sia l'importazione che l'esportazione Matrix supportano questo formato, che viene rilevato automaticamente quando il formato non è specificato e il nome del file termina con .sxc.

Per aprire il file SXC, è necessario associare un programma appropriato al file. Ogni volta che un file con estensione .sxc deve essere aperto, assicurati che sia privo di errori e che non sia danneggiato. Qualsiasi errore nel file può essere facilmente gestito dall'utente e questo non richiede un supporto tecnico il più delle volte.
