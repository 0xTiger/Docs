## Che cos'è un file CTS?

Un file CTS è un file di foglio di calcolo creato con il programma di immissione dati e organizzazione multipiattaforma open source **TreeSheets**. Contiene dati gerarchici inseriti in diversi formati all'interno del foglio di calcolo come note, elenchi, calendari, elenchi di cose da fare, note, mappe mentali e una serie di altri formati. I file CTS possono essere aperti solo con TreeSheets che è leggero, veloce e funziona su Windows, Linux e macOS.

## Formato file CTS - Ulteriori informazioni

TreeSheets salva i file CTS nel proprio formato di file proprietario. Le specifiche del formato file del formato file CTS non sono disponibili pubblicamente ed è per questo che i file CTS possono essere aperti solo nel software TreeSheets.

### Esporta i file CTS di TreeSheets in altri formati

Gli utenti di TreeSheets possono esportare i fogli di calcolo CTS in diversi formati di file. I file CTS possono essere esportati nei formati di file [HTML](/it/web/html/), [JSON](/it/web/json/), [CSV](/it/spreadsheet/csv/) e [Image](/it/image/) .

## Informazioni sul software TreeSheets

TreeSheets è un programma software open source disponibile per multipiattaforma. Ti consente di creare fogli di calcolo gerarchici e ti dà la libertà di inserire tutti i tipi di dati in un unico foglio di calcolo. Il vantaggio dell'utilizzo di TreeSheets è che puoi inserire tutte le tue discussioni sotto forma di note, mappe mentali, dati di testo, immagini e altri tipi di dati in un unico foglio di calcolo.
