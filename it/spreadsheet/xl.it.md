## Che cos'è un file XL?

Un file con estensione .xl è un foglio di calcolo di Excel, simile a [XLS](/it/spreadsheet/xls/), che memorizza i dati in righe e colonne. Era un formato di file precedente che ora non è più supportato da Excel. I dati archiviati nei file XL possono essere esportati in altri formati come CSV. Questi potrebbero anche essere convertiti in PDF utilizzando il metodo predefinito Salva con nome di Excel. I file XL possono essere aperti solo con Microsoft Excel.

## Formato file XL

I file XL vengono salvati come file binari e le loro specifiche di formato file interne non sono documentate da nessuna parte. Tuttavia, il formato file binario generale per [Specifiche del formato file XLS](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) può essere preso in considerazione anche se qualcuno tenta di scrivere un pezzo di codice per decodificare questi file.
