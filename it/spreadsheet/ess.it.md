## Che cos'è un file ESS?

Un file ESS è un file di foglio di calcolo creato e supportato dal software applicativo per fogli di calcolo EasySpreadsheet che fa parte di [EasyOffice](https://en.wikipedia.org/wiki/EasyOffice). EasyOffice è stato lanciato come alternativa a Microsoft Office ma non è più disponibile per l'uso ed è stato sostituito da un'altra suite, chiamata One. Era disponibile in due varianti: EasyOffice Freeware e EasyOffice Premium con il successivo dotato di filtro PDF e EasyAntiVirus. Il file ESS è compatibile con i file Microsoft [XLS](/it/spreadsheet/xls/).
