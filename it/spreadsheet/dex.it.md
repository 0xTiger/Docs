## Che cos'è un file DEX?

Un file DEX è un file di foglio di calcolo creato con Microsoft Excel. È simile ai file [XLSX](/it/spreadsheet/xlsx/) ma è in formato binario. I dati vengono archiviati in righe e colonne in un file DEX. Sebbene i file DEX siano anche Microsoft Excel, questi sono meno popolari. In confronto, il formato file XLSX è la scelta predefinita per il salvataggio delle cartelle di lavoro di Excel. I file DEX sono stati contrassegnati come obsoleti e non sono più supportati da Microsoft Excel.
