## Che cos'è un file \_XLS?

Un file con estensione .\_XLS è in realtà un file XLS che è stato rinominato in \_XLS da alcune altre applicazioni. Si basa sulla struttura del formato file XLS e, se rinominato in .xls, può essere aperto con tutte le applicazioni che supportano il formato file XLS. Nella maggior parte dei casi, i file \_XLS vengono generati con un file XLS allegato a un'e-mail che lo converte in \_XLS all'estremità ricevente. All'interno di questi, usano le stesse [Specifiche della formazione di file XLS](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) per contenere diversi tipi di informazioni come i grafici , grafici e dati di fogli di calcolo.

## \_Formato file XLS - Ulteriori informazioni

I file \_XLS vengono archiviati su disco in formato binario e possono essere aperti in Microsoft Word modificandone l'estensione in XLS. Il formato completo [specifiche del formato file di \_XLS file](https://msdn.microsoft.com/en-us/library/dd922181(v#office.12).aspx) è disponibile pubblicamente da Microsoft come riferimento per gli sviluppatori.
