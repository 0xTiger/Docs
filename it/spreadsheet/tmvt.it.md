## Che cos'è un file TMVT?

Un file TMVT è un file modello di foglio di calcolo timeline TimeMap che contiene impostazioni predefinite per la creazione di [file TMV](/it/spreadsheet/tmv/). A differenza dei file TMV, TMVT contiene impostazioni come stili di flag dei fatti, sfondi, stili di titolo e stili di scala. I file TMVT possono essere condivisi con gli utenti e da loro riutilizzati per mantenere le stesse impostazioni e creare i propri file di sequenza temporale per uniformità. Questi possono essere aperti con il software TimeMap da [LexisNexis](https://www.lexisnexis.com/en-us/products/casemap.page).

## Formato file TMVT

I file TMVT vengono creati e salvati su disco in formato binario.
