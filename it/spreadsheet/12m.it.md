## Che cos'è un file Lotus 12M?

Un file 12M è un file di foglio di calcolo Lotus 1-2-3 che contiene formati di foglio di calcolo predefiniti. È stato sviluppato da Lotus Development Corporation e faceva parte di IBM Lotus SmartSuite. Il foglio di calcolo predefinito nel file 12M può essere utilizzato per la raccolta di dati utilizzati per creare vari tipi di file di dati come rendiconti finanziari, budget e previsioni. Essendo come qualsiasi altro foglio di calcolo, il formato di file 12M supporta formule, formattazione e intervalli di stampa. [Lotus 123](/it/spreadsheet/123/) è un altro popolare formato di file di Lotus Spreadsheet.

## Formato file Lotus 12M - Ulteriori informazioni

Quando IBM ha acquisito Lotus Development Corporation nel 1995, tutti i formati di file proprietari supportati da Lotus 1-2-3 sono diventati parte di IBM Lotus SmartSuite. A quel punto, era il popolare programma per fogli di calcolo utilizzato principalmente. Le funzionalità popolari che supportava erano funzioni di foglio di calcolo, creazione di grafici e operazioni di archiviazione e gestione dei dati.

Con il lancio di Microsoft Excel, la popolarità di Lotus 1-2-3 è diminuita in modo significativo e alla fine è stata superata da Microsoft Excel. È stato ufficialmente interrotto nel 2013 e anche i suoi formati di file non sono più supportati.
