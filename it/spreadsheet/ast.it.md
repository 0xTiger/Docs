## Che cos'è un file AST?

Un file AST è un file modello di foglio di calcolo creato con il software Ability Spreadsheet, un componente del foglio di calcolo della suite Ability Office. Questi file modello vengono utilizzati per creare una serie iniziale di modelli di fogli di calcolo che possono essere utilizzati per creare file [AWS](/it/spreadsheet/aws/). I file AST possono essere aperti con il software per fogli di calcolo Ability.

## Formato file AST - Ulteriori informazioni

I file AST vengono salvati su disco come file binari e possono essere aperti nel software Ability Spreadsheet. Sebbene completamente compatibili con i formati di file XLS e XLSX di Microsoft, le specifiche del formato di file AST non sono disponibili pubblicamente come riferimento per gli sviluppatori. I file AST possono essere salvati su disco nei formati di file [PDF](/it/pdf/) e [XPS](/it/page-description-language/xps/).
