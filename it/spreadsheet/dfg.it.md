## Che cos'è un file DFG?

Un file con estensione .dfg è un file di foglio di calcolo creato da Interscape Inc. per il software Interscape Data Flask. I dati vengono inseriti in file DFG in righe e colonne. Data Flask è un software per fogli di calcolo, progettato per conservare i tipi di dati, l'origine e le informazioni chiave. Può migrare tabelle e indici modificando solo la stringa di connessione. Offre funzionalità come riga di comando VBScript, formule VBScript e macro insieme a script SQL per l'aggiornamento e la migrazione dei dati. I file DFG vengono salvati come file binari su disco e possono essere aperti con il software Interscape Data Flask.

## Formato file DFG - Ulteriori informazioni

I file DFG salvano le informazioni sui dati del foglio di calcolo come file binari su disco. La struttura dei dati interna e i dettagli del formato di questi file non sono tuttavia noti.
