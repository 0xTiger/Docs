## Che cos'è un file PUB?

Un file PUB è un formato di file di documento di Microsoft Publisher. Viene utilizzato per creare diversi tipi di documenti di layout di progettazione come newsletter, volantini, brochure, cartoline, ecc. I file PUB possono contenere testo, immagini raster e vettoriali. I documenti creati con i file PUB vengono utilizzati principalmente nei siti Web e nei materiali di marketing come le e-mail. I file PUB possono essere aperti con l'applicazione Microsoft Publisher Desktop, [Microsoft Publisher 365](https://www.microsoft.com/en/microsoft-365), [LibreOffice Draw](https://www.libreoffice.org/discover /disegna/) e [Adobe InDesign](https://www.adobe.com/products/indesign.html).

## Formato file PUB - Ulteriori informazioni

I file PUB vengono archiviati come file binari su disco. Il contenuto di un file PUB può anche essere convertito nel formato di file [DOCX](/it/word-processing/docx/) dall'applicazione Microsoft Publisher utilizzando l'opzione Salva con nome.

## Come utilizzare Microsoft Publisher?

Microsoft ha stabilito diverse linee guida per aiutare gli utenti a iniziare con Microsoft Publisher. Questa guida [Attività di base in Publisher](https://support.microsoft.com/en-us/office/basic-tasks-in-publisher-0e5ed249-1927-433f-a35c-63beb8216fcf) ti consente di iniziare mostrando come per creare una pubblicazione, salvare una pubblicazione, aggiungere testo e blocchi predefiniti e stampare la pubblicazione.
