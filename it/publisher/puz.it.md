## Che cos'è un file PUZ?

Un file con estensione .puz è un archivio compresso che contiene un documento Microsoft Publisher [.pub](/it/publisher/pub/) standard. Come risultato della compressione, è di dimensioni inferiori e facile da trasferire tramite Internet o altri dispositivi. I file PUZ possono essere decompressi/aperti utilizzando l'utilità Microsoft Unpack.

## Formato file PUZ

Un file PUZ è un archivio compresso che contiene il file PUB standard insieme ad altri file. Contiene tutti i file necessari per decomprimere, visualizzare e stampare la pubblicazione.

Un file PUZ contiene quanto segue.

* `Unpack.exe` - un'utilità software per decomprimere i file compressi
* `ReadMe.txt` - un file di descrizione
* `File .puz compresso` - Contiene la pubblicazione e può includere caratteri incorporati e grafica collegata.

### Disimballaggio del file PUZ

I file di Publisher compressi possono essere decompressi utilizzando l'utilità Unpack.exe. Estrae il file PUZ compresso nella scelta della directory selezionata dall'utente.
