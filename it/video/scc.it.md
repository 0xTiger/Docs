## Cos'è un file SCC?

I file SCC sono file video creati da ScreenFlow, un'applicazione di registrazione dello schermo desktop di Mac OS X. Memorizza il video catturato in un formato proprietario e può includere anche l'audio catturato dal microfono di un computer o dalla scheda audio del computer.

I file SCC vengono utilizzati da ScreenFlow come file temporanei per archiviare dati video non renderizzati prima che vengano esportati nel formato video finale, durante il processo di modifica in ScreenFlow, quando apporti modifiche al video, applichi effetti o esegui altre modifiche, il software salva i progressi del progetto in File SCC che contengono dati video grezzi e istruzioni di modifica.

Una volta completato il processo di modifica e sei pronto per esportare il tuo video, ScreenFlow esegue il rendering dei file SCC nel formato di output desiderato, come MP4 o MOV. Il processo di rendering prevede l'applicazione di tutte le modifiche e gli effetti per generare il file video finale.

Una volta completato il processo di rendering, i file SCC temporanei vengono generalmente eliminati e il file video finale viene salvato con il formato di output scelto che può essere condiviso, caricato o utilizzato secondo necessità.

## Cos'è ScreenFlow?

ScreenFlow di Telestream, Inc. è un software di screencasting e editing video per il sistema operativo macOS. Può acquisire audio e video dal computer, modificare il video catturato, aggiungere evidenziazioni o annotazioni e generare diversi tipi di file come AIFF, GIF, M4V, MOV e MP4.

Dopo aver catturato lo schermo, ScreenFlow fornisce un'interfaccia di editing video completa, puoi tagliare e dividere clip video, aggiungere transizioni e animazioni, inserire testo e didascalie e incorporare file multimediali aggiuntivi come immagini, musica o altri video, offre anche funzioni integrate strumenti di editing audio per regolare i livelli di volume, rimuovere il rumore di fondo e applicare effetti.
