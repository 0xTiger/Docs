## Che cos'è un file MJPEG? ##

Un file con estensione .mjpeg (Motion JPEG) è un file video creato comprimendo un singolo fotogramma video come immagine JPEG. È diverso dai formati di file MPEG-1 e MPEG-2 standard in cui la compressione è interframe e viene decisa in base alla previsione dei contenuti nei frame successivi. MJPEG è supportato da un'ampia gamma di applicazioni che includono browser Web, lettori multimediali, fotocamere digitali, webcam, server di streaming. I plug-in sono disponibili per le applicazioni che desiderano utilizzare il formato di file MJPEG.

## Formato file MJPEG ##

MJPEG si basa sullo standard di compressione maturo, ad esempio [JPEG](/it/image/jpeg/) che ha librerie ben definite. Non può essere standardizzato in modo simile a standard come MPEG-2 e non è disponibile documentazione che possa essere definita come specifiche complete per "Motion JPEG". In assenza di tale standard, le uscite di diversi produttori possono sollevare problemi di compatibilità. Tuttavia, aziende come Microsoft e Apple hanno documentato come i file M-JPEG vengono archiviati nei loro formati nativi come [AVI](/it/video/avi/) e [QT](/it/video/qt/). [rfc2435](https://tools.ietf.org/html/rfc2435) descrive il formato RTP Payload per video compresso JPEG e può essere consultato come materiale di riferimento.
