## Che cos'è un file MPV?

Un file MPV è un file video MPEG elementare costituito da dati video grezzi non compressi. I contenuti video all'interno di un file MPV comprendono specifiche di compressione MPEG-2 integrate. I file MPV non contengono dati audio. Il formato file MPV viene utilizzato anche come input per applicazioni di elaborazione video poiché il flusso elementare [MPEG](/it/video/mpg/) utilizza principalmente gli standard di codifica MPEG-2. I file MPV possono essere aperti e riprodotti con applicazioni software che supportano i formati di file video MPEG-2. Il software del lettore multimediale VideoLAN LVC è uno di questi software in grado di aprire e riprodurre file MPV.

## Formato file MPV

I file MPV vengono salvati su disco come semplici file binari, seguendo la struttura del formato file MPEG. Tuttavia, il loro formato di file interno non è disponibile pubblicamente per riferimento dello sviluppatore.
