## Che cos'è un file PZ?

Un file PZ è un file di progetto creato con l'applicazione per la creazione di video clip online **Panzoid**. Contiene informazioni sul progetto video e sui suoi contenuti come scene, oggetti, effetti, luci, effetti, audio e musica. Panzoid ti consente di creare video online utilizzando la loro funzione di editing video **ClipMaker**.

I file PZ possono essere aperti con Panzoid ClipMaker e Panzoid Backgrounder.

## Formato file PZ

I file PZ vengono salvati come file binari su disco. Quando un utente crea un nuovo progetto con [Panzoid](https://panzoid.com/), viene creato un file PZ che mantiene i dettagli degli oggetti definiti dall'utente salvati. Le stesse informazioni vengono sincronizzate con altri utenti se il progetto è di altri contributori. I file PZ possono quindi essere scaricati sul computer.

Quando si desidera aprire il file PZ, è possibile caricarlo nuovamente in **Panzoid** utilizzando l'opzione Carica dall'interfaccia online. Il file viene caricato online e il suo contenuto viene caricato per le modifiche.

### Clip video supportate da file PZ

Panzoid ha i programmi ClipMaker e Backgrounder che vengono utilizzati insieme per la creazione di video. Con Panzoid è possibile creare diversi tipi di video, inclusi sfondi personalizzati, grafica del canale YouTube, video dei social media e miniature.
