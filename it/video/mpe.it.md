## Che cos'è un file MPE?

Un file MPE è un file video codificato e salvato nel formato file [MPEG](/it/video/mpg/). È stato esteso con bit-rate aggiuntivi per essere supportato come la prossima versione della compressione MPEG-2. MPEG-2, tuttavia, è il tipo di compressione MPEG più comunemente utilizzato. L'estensione .mpe è stata inizialmente utilizzata come abbreviazione del formato di file .mpeg.
