## Che cos'è un file TS?

Un file con estensione .ts è un flusso video che memorizza i dati su DVD. Utilizza l'algoritmo di compressione MPEG-2 ([.mpeg](/it/video/mpg/)) per ottenere la massima efficienza e compatibilità tra diversi tipi di media come lo streaming e la trasmissione Internet. Il formato file TS è stato creato in modo che possa essere riprodotto su dispositivi con connessioni Internet scarse, come le smart TV.

## Formato file TS - Ulteriori informazioni

I dati in un file TS sono simili al file [MP4](/it/video/mp4/), ma sono divisi in piccoli blocchi. Ogni pezzo è costituito da un minuscolo pezzo di video, seguito da un po' di audio e una didascalia opzionale. Un singolo file TS è costituito da molti di questi blocchi. Oltre a video, audio e didascalia, ogni blocco comprende anche alcuni dati aggiuntivi per rilevare gli errori nel blocco. Per questo motivo, i file TS hanno dimensioni leggermente maggiori.

### Perché usare il formato file TS?

Quindi, se i file TS sono di dimensioni piuttosto grandi, quale vantaggio offre l'utilizzo di questi file invece di altri formati di file? Ebbene, nella trasmissione, i piccoli frammenti di video e audio possono essere inviati tramite i mezzi di comunicazione (cablati o radio) in tempo reale. I dati extra nei blocchi vengono utilizzati dal ricevitore per saltare i blocchi soggetti a errori.

Inoltre, i file TS vengono utilizzati perché il sistema di trasmissione non ha bisogno dell'intero flusso per riprodurlo. La trasmissione può essere captata e utilizzata in tempo reale assemblando audio e video.

## Come riprodurre file TS?

Bene, i file TS possono essere aperti e riprodotti nel popolare [VideoLAN VLC media player](https://www.videolan.org/vlc/features.html) che è disponibile gratuitamente per il download.
