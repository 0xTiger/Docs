## Che cos'è un file AMV? ##

L'acronimo AMV come Actions Media Video è classificato come un formato di file video prodotto per lettori multimediali portatili [MP4](/it/video/mp4/) cinesi e lettori MP3 S1 con riproduzione video. Il video AMV ha sostanzialmente due formati diversi: quello precedente per il chip Actions e l'ultimo per il chip M5661 di ALi; il formato per ALi uno era ALIAVI. Il formato video è una variante di Motion JPEG, con tabelle di divisione fisse anziché variabili. Il formato audio è una variante di IMA ADPCM. I primi 8 byte di ogni frame sono origine, indice e diversi campioni codificati a 16 bit; di solito, i file AMV riproducono il suono a 22050 campioni/secondo. Il video AMV è un video compresso ma a bassa risoluzione (da 96x96 a 320x240 pixel), che può essere riprodotto sul desktop utilizzando qualsiasi software di lettore multimediale.

## Formato file AMV ##

Come variante dei file [AVI](/it/video/avi/), alcuni lettori multimediali cinesi utilizzano comunemente file AMV. Sorprendentemente, i file AMV contengono il tipico testo AMV e amvh nelle loro intestazioni; mentre i file AVI contengono AMI e testo avih.

Il formato AMV è stato progettato per consentire ai lettori multimediali cinesi a basso costo e facilmente disponibili di archiviare e riprodurre molti video a bassa risoluzione. I video AMV di solito non sono molto compressi, ma la loro dimensione del file è piccola a causa della loro bassa risoluzione e frame rate. Questa cosa aiuta i lettori multimediali economici a eseguire il formato di file AMV senza alcun problema.
