## Cos'è un file AEP?

Un file AEP è un file di progetto creato da Adobe After Effects quando salvi un progetto After Effect. È il tipo di file principale utilizzato da After Effects per salvare tutte le impostazioni e i dati relativi al progetto come riprese video, immagini fisse, file audio e qualsiasi altro effetto. Il file AEP contiene informazioni sulla composizione, i livelli, i fotogrammi chiave, gli effetti e altri elementi che fanno parte del progetto After Effects.

Puoi aprire file AEP con il software Adobe After Effects.

## Formato file AEP: ulteriori informazioni

Adobe After Effect memorizza il file AEP in formato file binario. La struttura interna del file AEP non è disponibile pubblicamente come riferimento per lo sviluppatore ed è per questo motivo che può essere aperta solo in Adobe After Effect. Il file AEP memorizza i riferimenti agli elementi che fanno parte del progetto ma in realtà non contiene queste raccolte ed elementi. Se spostate i file di riferimento dopo la creazione del progetto, After Effects non sarà in grado di individuare i file e non potrete utilizzare questi file multimediali.

### Tipi di file di progetto After Effects

After Effects salva i file di progetto in:

* "File AEP" - È archiviato in formato binario e ha dimensioni ridotte. I file binari si caricano più velocemente e possono essere condivisi facilmente con altri utenti.
* "File AEPX": si tratta di un formato di file di progetto After Effects basato su XML che memorizza i dati in formato testo. I file AEPX sono generalmente più grandi e più lenti da caricare rispetto al formato di file AEP.

## Come convertire file AEP

I file AEP possono essere convertiti in altri formati di file popolari come:

* [.MOV](/it/video/mov/) - Filmato Apple QuickTime
* [.AVI](/it/video/avi/) - Interfoglio audio video
* [.JPG](/it/image/jpeg/) - Immagine JPEG
* [.PNG](/it/image/png/) - Grafica di rete portatile
* .DPX - Scambio di immagini digitali
* .EXR - Immagine OpenEXR
* .M4A - Audio MPEG-4
* [.MP3](/it/audio/mp3/) - Audio MP3
* [.WAV](/it/audio/wav/) - Audio ONDA
* .AEPX - Progetto XML di After Effects
* .AET - Modello After Effects
* .PRPROJ - Adobe Premiere Pro
