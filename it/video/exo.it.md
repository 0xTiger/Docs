## Che cos'è un file EXO?

Un file EXO è un file di blocco video, scaricato dalla versione Android di YouTube nel formato EXO. Quando un utente desidera scaricare un video per la riproduzione offline, il video viene diviso in piccoli blocchi e ogni blocco viene scaricato e salvato sul dispositivo dell'utente come file EXO. Questi più file EXO sono uniti per la riproduzione offline dall'app YouTube.

## Formato file EXO

I file EXO vengono salvati nel formato di file proprietario di YouTube. Vengono salvati come file crittografati compressi e possono essere aperti solo con l'applicazione YouTube per Android. Tutti i blocchi multipli scaricati vengono assemblati e decifrati dall'app per essere riprodotti offline. Essendo crittografati, i file EXO non possono essere convertiti in altri formati di file come [MP4](/it/video/mp4/).
