## Che cos'è un file ARF?

Un file ARF (Advanced Recording File) è un file Network-Based Recording (NBR) creato da una riunione online condotta tramite WebEx. Contiene video e audio di qualità superiore ed è il formato di file ampiamente utilizzato nelle riunioni WebEx. Le informazioni contenute in un file ARF includono dati video, sommario, elenco dei partecipanti, registrazione desktop, condivisione di applicazioni e altro contenuto della riunione. I file ARF non possono essere modificati direttamente, ma l'inizio e la fine della riproduzione possono essere troncati. ARF è il formato file predefinito utilizzato dalle riunioni WebEx.

## Formato file ARF

I file ARF vengono salvati nel sito Webex come file binari nella sezione **Le mie registrazioni** dell'account host. Questi possono essere effettuati solo da un organizzatore di una riunione dall'interno di una riunione online utilizzando l'opzione Registra sul server. I file ARF possono essere riprodotti utilizzando il lettore di registrazione di rete.

## Opzioni di conversione file ARF

I file ARF possono essere convertiti nei formati [WMV](/it/video/wmv/), [SWF](/it/page-description-language/swf/) o [MP4](/it/video/mp4/).
