## Che cos'è un file 3GP?

3GP (3GPP File Format) è un formato contenitore multimediale utilizzato sui telefoni cellulari 3G. Può anche essere riprodotto su alcuni telefoni 2G e 4G. 3GP è definito dal Third Generation Partnership Project (3GPP) per i servizi multimediali 3G UMTS (Universal Mobile Telecommunications System). I file 3GP vengono archiviati con l'estensione .3gp. Il formato file 3GP è stato progettato per telefoni basati su GSM (Global System for Mobile Communications) ed è il formato standard per i file multimediali inviati tramite MMS (Multimedia Messaging Service). A differenza di [MP4](/it/video/mp4/), che utilizza il codec H.263, i file 3GP utilizzavano preferibilmente il codec H.263.

## Dettagli tecnici del formato file 3GP

Il formato video 3GP si basa sul formato file multimediale di base ISO definito in ISO/IEC 14496-12 – MPEG-4 Parte 12. È stato sviluppato per i telefoni cellulari per ridurre lo spazio di archiviazione, la larghezza di banda e l'utilizzo dei dati. Il formato file 3GP supporta i seguenti codec:

- Video
  - MPEG-4 Part 2
  - H.263
  - MPEG-4 Part 10 (AVC/H.264)
- Audio
  - AMR-WB
  - AMR-WB+
  - AMR-NB
  - AAC-LC
  - HE-AAC v1
  - HE-AAC v2 (Enhanced aacPlus)

Il formato di file 3GP ha consentito l'utilizzo di codec H.263 e AMR e ha definito nuove caselle a cui fanno riferimento i codec. Questi punti di codice sono stati registrati nel formato di file multimediale di base ISO per l'archiviazione di informazioni specifiche del supporto MPEG-4.

## Dispositivi di supporto per file video 3GP

- Telefoni cellulari con funzionalità 3G
- Audio PlayStation 3 importato da un CD
- Il Nintendo 3DS ha riprodotto i video di YouTube utilizzando la tecnologia 3GP
- Gli iDevice di Apple supportano la riproduzione di file solo come file passthrough
