## Cos'è un file STR?

Un file STR è un file di registrazione creato da YouTube per un live streaming ospitato dall'evento di streaming dell'utente. Il file viene salvato automaticamente da Youtube nel suo backend una volta terminato lo streaming live. Un video in live streaming può essere trasmesso su più canali in modo che il pubblico possa interagire con l'editore tramite commenti. Questi file SRT sono accessibili utilizzando l'API livestream di YouTube che fornisce l'accesso ai server di YouTube.

## Formato file STR

YouTube salva i file STR sul server in formato file binario. Tuttavia, i dettagli della struttura interna dei file STR non sono disponibili pubblicamente.
