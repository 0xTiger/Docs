## Che cos'è un file 3DM?

Un file con estensione .3dm è un formato di file open source utilizzato per il software di grafica 3D. Contiene modelli 3D insieme ai relativi elementi dipendenti come informazioni su superfici, punti e curve. Il formato di file 3DM è stato sviluppato dall'iniziativa [openNURBS](https://github.com/mcneel/opennurbs) per trasferire accuratamente la geometria 3D tra le applicazioni. I file 3DM possono essere aperti e convertiti con applicazioni come Rhinoceros, SAP VEViewer, Moment of Inspiration e Right Hemisphere Deep View.

## Formato file 3DM

[openNURBS](https://github.com/mcneel/opennurbs), un toolkit open source, contiene le specifiche del formato di file 3DM, la documentazione, le librerie di codice sorgente C++ e gli assembly .NET 2.0 per leggere e scrivere il formato del file.

### File di esempio 3DM

Alcuni file 3DM di esempio possono essere scaricati dalla sezione openNURBS [examples](https://github.com/mcneel/opennurbs/tree/7.x/example_files). Questi possono essere caricati e visualizzati utilizzando il toolkit openNURBS.

## Come leggere o scrivere file 3DM?

Le librerie [openNURBS](https://github.com/mcneel/opennurbs) consentono a chiunque di leggere e scrivere il formato file 3DM senza bisogno di Rhino. Fornisce codice sorgente C++ per una libreria che leggerà e scriverà modelli 3D utilizzando il formato di file openNURBS. Questo toolkit fornisce anche strumenti di valutazione NURBS e strumenti elementari di manipolazione delle viste geometriche e 3D, oltre a includere il codice sorgente per diversi programmi di esempio. Il forum di supporto di [Rhinoceros](https://discourse.mcneel.com/c/opennurbs/6) è un luogo ricco di contenuti e discussioni per condividere i problemi affrontati dalla comunità 3DM e ottenere le soluzioni.
