## Cos'è il file RVM?

**I file di dati RVM** sono correlati ad AVEVA PDMS. **Il file RVM** è un file di progetto AVEVA Plant Design Management System Model. Il Plant Design Management System (PDMS) di AVEVA è il più popolare sistema di progettazione 3D che utilizza la tecnologia data-centric per la gestione dei progetti. Sono disponibili numerose applicazioni per aprire e convertire file RVM in altri formati come [3DS](/it/3d/3ds/).

## Formato file RVM

Non è possibile trovare molti dettagli sul formato di file RVM a parte il fatto che può essere salvato sia in formato binario che ASCII. Il file RVM supporta entità come:

* Tutta la geometria
* Attributi memorizzati sui gruppi
* Texture (tramite file RVS)
* Telecamere e tracce della telecamera (tramite file RVS)
* Piani di clip (tramite file RVS)
* Segni (tramite file RVS)
* Tag (tramite file RVS)
* Etichette (tramite file RVS)
* Traslucenza (tramite file RVS)
* Punti di origine PDMS
