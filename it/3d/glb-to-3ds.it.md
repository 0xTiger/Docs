## Informazioni su GLB su 3DS

L'uso di file GLB nei formati di file 3D non è una novità. GLB è un formato di file binario 3D comunemente utilizzato che rappresenta modelli 3D nel formato GL Transmission, ovvero ([glTF](/it/3d/gltf/)). [3DS](/it/3d/3ds/), d'altra parte, rappresenta il formato file mesh 3D Sudio (DOS) utilizzato da Autodesk 3D Studio.
