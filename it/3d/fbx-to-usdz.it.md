## Informazioni su FBX in USDZ

FBX, abbreviazione di FilmBox, è un formato di file 3D ampiamente utilizzato inizialmente realizzato da Kaydara per MotionBuilder. Autodesk Inc. ne ha assunto la proprietà nel 2006, consolidando il proprio status di formato primario per lo scambio di dati 3D utilizzato da una moltitudine di strumenti software 3D.

Un file con estensione .usdz rappresenta un archivio ZIP non compresso e non crittografato su misura per il formato file [USD](/it/3d/usd/) (Universal Scene Description). Ospita proxy per vari tipi di file, incluse trame e animazioni, tutti comodamente incapsulati all'interno dell'archivio.
