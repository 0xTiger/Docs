## Che cos'è un file IPT?

Un file con estensione .ipt è il formato di file di parti di Inventor di Autodesk nativo per le parti.
![IPT File](../ipt2.jpg "IPT File")
Viene utilizzato in combinazione con i file di assieme (.iam) di Autodesk. I file IPT possono essere importati in 3DS Max come Body Objects in cui la geometria nel formato dei solidi ACIS rimane nello stesso formato. 3DS Max conserva la denominazione degli oggetti assegnata in Autodesk Inventor e i modelli dei componenti possono essere modificati in modo simile ad altri oggetti, inclusa l'applicazione di modificatori, alterazione dei materiali, aggiunta di illuminazione e telecamere, creazione di animazioni e così via. È possibile aprire file IPT con applicazioni come Autodesk Inventor e Autodesk Fusion.

## Formato file IPT

Un **estensione file IPT** di Autodesk utilizzata per salvare un file che comprende schizzi, funzioni e corpi che rendono la parte utilizzabile negli assiemi. Uno schizzo è il componente più utilizzato e la maggior parte delle parti inizia con uno schizzo. È il profilo di una funzione e qualsiasi geometria necessaria per creare la funzione. Funzioni multiple costituiscono un "modello di parte" e una parte multicorpo può essere funzioni condivise. Le relazioni geometriche come parallelo e perpendicolare sono controllate da vincoli di schizzo. L'effetto delle modifiche può essere osservato regolando i vincoli oi parametri dimensionali per controllare la dimensione e la forma di un modello.
![IPT File Format](../ipt.jpg "IPT File Format")

**Nota:** le specifiche del formato di file IPT non sono disponibili pubblicamente.
