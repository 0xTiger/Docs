## Che cos'è un file DBM?

Un file DBM (Daz Brick Material) è un file di rete Shader Mixer creato con l'applicazione di modellazione dei personaggi 3D, **DAZ Studio**. Descrive il materiale della modalità 3D, ovvero superficie/spostamento/volume interno. DAZ ti consente di creare il tuo shader personalizzato collegando insieme una serie di mattoncini. Questo si combina collettivamente come un file DBM che forma una rete di shader. È il file DBM che specifica come viene ombreggiato un materiale.

Le applicazioni che possono **aprire file DBM** includono DAZ 3D DAZ Studio 4.

## Formato file DBM

I file DBM vengono salvati su disco nel formato di file [XML](/it/web/xml/).
