## Che cos'è un file MP?

Un file MP è un file di progetto creato con la Personal Learning Edition (PLE) del programma di modellazione 3D, **Maya**. È supportato solo dalla versione PLE di Maya ed è puramente per usi non commerciali. A tale scopo, l'edizione Maya PLE include una filigrana sul file esportato. Maya PLE fornisce funzionalità limitate e può essere installato solo su personal computer. Il file MP contiene tutte le informazioni sul progetto creato con PLE e può essere aperto con Autodesk Maya PLE.
