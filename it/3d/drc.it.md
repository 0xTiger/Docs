## Che cos'è un file DRC?

Un file con estensione .drc è un formato di file 3D compresso creato con la libreria di Google Draco. Google offre Draco come libreria open source per comprimere e decomprimere mesh geometriche 3D e nuvole di punti e migliora l'archiviazione e la trasmissione di grafica 3D. Codifica i dati di input e li salva come file .drc. All'estremità ricevente, l'API legge i file .drc e può emetterli come file [PLY](/it/3d/ply/) o [OBJ](/it/3d/obj/). Il file di output compresso consente agli utenti di scaricare le app più velocemente e di caricare rapidamente la grafica 3D nei browser.

## Formato file DRC

I dettagli del formato file per DRC non sono disponibili pubblicamente. Google ha ospitato la libreria Draco su [GitHub](https://github.com/google/draco/tree/master/docs) come riferimento per gli sviluppatori. Fornisce codificatori e decodificatori per la creazione e la conversione di file DRC.
