## Che cos'è un file MB?

Un file con estensione .mb è un file di progetto binario creato con l'applicazione Autodesk Maya. A differenza del formato file [MA](/it/3d/ma/), che è in formato file ASCII, i file MB vengono archiviati in formato file binario. Questi non possono essere aperti negli editor di testo ma possono essere esaminati negli editor esadecimali. I file MB contengono informazioni come geometria, illuminazione, animazione e rendering per la definizione delle scene 3D.

Autodesk Maya 2020 può essere utilizzato per aprire file .mb.

## Formato file MB

I file MB sono in formato file binario, che è un formato di file compresso e possono essere aperti solo supportando l'applicazione. I dettagli interni del formato file MB non sono disponibili e possono essere elaborati solo da una sorta di script per decomprimerli.
