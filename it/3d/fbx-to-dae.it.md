## Informazioni su FBX su DAE

FBX, abbreviazione di FilmBox, è un formato di file 3D ampiamente utilizzato che Kaydara ha inizialmente creato per migliorare la compatibilità con MotionBuilder. Nel 2006, Autodesk Inc. ha rilevato questo formato, consolidando il suo status di standard fondamentale per lo scambio semplice di dati 3D tra varie applicazioni software 3D.

Un file DAE, abbreviazione di file Digital Asset Exchange, funge da formato per il trasferimento di dati tra applicazioni 3D interattive. Si basa sullo schema XML COLLADA (COLLAborative Design Activity), che rappresenta uno standard aperto per lo scambio di risorse digitali tra varie applicazioni software di grafica.

Molti utenti hanno spesso bisogno di convertire i file [FBX](/it/3d/fbx/) nel formato DAE. Fortunatamente, sono facilmente accessibili numerosi strumenti online gratuiti per questo processo di conversione senza alcun costo.
