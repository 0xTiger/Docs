## Informazioni su GLB su FBX

GLB è un popolare formato di file binario 3D che rappresenta modelli 3D nel formato GL Transmission, ovvero ([glTF](/it/3d/gltf/)). FBX, FilmBox, è un altro popolare formato di file 3D originariamente sviluppato da Kaydara per MotionBuilder. Da GLB a FBX è un'operazione ampiamente utilizzata per convertire i file GLB nel formato file FBX.
