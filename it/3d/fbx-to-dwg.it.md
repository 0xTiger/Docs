## Informazioni su FBX in DWG

FilmBox, spesso abbreviato in FBX, è un formato di file 3D ampiamente utilizzato inizialmente creato da Kaydara per supportare MotionBuilder. Autodesk Inc. ha acquisito questo formato nel 2006, consolidandolo come standard primario per lo scambio di dati 3D tra vari strumenti software 3D. D'altra parte, DWG, abbreviazione di stereolitografia, funge da formato file intercambiabile progettato per rappresentare geometrie di superficie tridimensionali.

La conversione di file [FBX](/it/3d/fbx/) nel formato DWG è una necessità frequente e numerose applicazioni online offrono strumenti di conversione gratuiti per questa attività.
