## Circa 3DS per USDZ

L'estensione file .3ds è un chiaro identificatore del formato file mesh 3D Studio (DOS), utilizzato dal software 3D Studio di Autodesk. Il software 3D Studio di Autodesk ha mantenuto un ruolo di primo piano nel dominio dei formati di file 3D sin dagli anni '90 e si è successivamente trasformato in 3D Studio MAX. [Ulteriori informazioni sul formato file 3DS](/it/3d/3ds/)

Un file .usdz è un archivio ZIP non compresso e non crittografato per il formato file Universal Scene Description (USD). Contiene file proxy di vari formati, come trame e animazioni, consentendo loro di essere eseguiti direttamente con il runtime USD senza decomprimere. [Ulteriori informazioni sul formato file USDZ](/it/3d/usdz/)

La conversione di file 3DS nel formato USDZ è una necessità frequente e molti strumenti online offrono opzioni di conversione gratuite.
