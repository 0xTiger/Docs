## Informazioni su FBX su PLY

FBX, comunemente noto come FilmBox, è un formato di file 3D ampiamente adottato originariamente sviluppato da Kaydara per migliorare la compatibilità con MotionBuilder. Nel 2006, Autodesk Inc. ha assunto il controllo di questo formato, consolidando il suo ruolo di standard cruciale per lo scambio continuo di dati 3D tra varie applicazioni software 3D.

PLY, abbreviazione di Polygon File Format, è un formato di file 3D progettato per memorizzare oggetti grafici definiti da una raccolta di poligoni. Il suo obiettivo principale era creare un formato di file semplice e versatile adatto a un ampio spettro di modelli. I file PLY sono disponibili sia in formato ASCII che binario, quest'ultimo ideale per un'archiviazione efficiente e un recupero rapido.

La conversione da [FBX](/it/3d/fbx/) a [PLY](/it/3d/ply/) è un requisito comune che consente di convertire file FBX in PLY. Il vantaggio di utilizzare questi strumenti online è che sono accessibili dal browser Web senza la necessità di installarli esplicitamente.
