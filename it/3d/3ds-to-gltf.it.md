## Informazioni su 3DS su GLTF

L'estensione file .3ds indica il formato file mesh 3D Studio (DOS) utilizzato dal software 3D Studio di Autodesk. Autodesk 3D Studio ha ricoperto una posizione significativa nel panorama dei formati di file 3D sin dagli anni '90 e successivamente si è evoluto in 3D Studio MAX.

glTF, o GL Transmission Format, è un formato di file 3D che utilizza la notazione JSON per incapsulare i dati del modello 3D. Questo approccio basato su JSON riduce in modo efficiente le dimensioni delle risorse 3D e minimizza le risorse computazionali necessarie per l'estrazione e l'utilizzo in tempo reale.

La conversione di [3DS](/it/3d/gltf/) nel [formato file GLTF](/it/3d/gltf/) è una necessità frequente e numerosi strumenti online offrono la conversione gratuita da 3DS a GLTF.
