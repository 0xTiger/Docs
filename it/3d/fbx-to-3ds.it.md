## Informazioni su FBX su 3DS

FilmBox, comunemente indicato come FBX, è un formato di file 3D ampiamente adottato originariamente sviluppato da Kaydara per facilitare la compatibilità con MotionBuilder. Autodesk Inc. ha assunto il controllo di questo formato nel 2006, stabilendolo come standard chiave per lo scambio continuo di dati 3D tra diverse applicazioni software 3D.

Un file con estensione [.3ds](/it/3d/3ds/) indica il formato di file mesh 3D Studio (DOS) utilizzato dal software 3D Studio di Autodesk. La presenza di Autodesk nell'arena dei formati di file 3D risale agli anni '90 e da allora si è evoluta in 3D Studio MAX, uno strumento completo per la modellazione, l'animazione e il rendering 3D. Il file .3ds incapsula i dati per la rappresentazione tridimensionale di scene e immagini, rendendolo uno dei formati di file preferiti per l'importazione e l'esportazione di dati 3D.

La conversione da [FBX](/it/3d/fbx/) a 3DS è un requisito frequente da parte degli utenti. Sono disponibili diversi strumenti gratuiti online per la conversione GRATUITA di FBX in 3DS.
