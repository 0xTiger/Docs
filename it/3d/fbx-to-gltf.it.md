## Informazioni su FBX su GLTF

FilmBox, comunemente noto come FBX, è un formato di file 3D ampiamente adottato inizialmente realizzato da Kaydara per integrare MotionBuilder. Nel 2006, Autodesk Inc. ha preso il timone, portando FBX in una posizione di rilievo come formato primario di scambio di dati 3D abbracciato da una moltitudine di strumenti software 3D.

glTF, abbreviazione di GL Transmission Format, funge da formato di file 3D progettato per incapsulare i dati del modello 3D all'interno della notazione JSON. Questa implementazione con JSON serve a ridurre in modo efficiente le dimensioni delle risorse 3D, nonché a diminuire il sovraccarico computazionale richiesto per la loro estrazione e utilizzo durante il runtime.

È un requisito comune convertire [FBX](/it/3d/fbx/) nel formato di file GLTF e online sono disponibili diverse app che possono convertire FBX in GLTF GRATUITAMENTE. Se sei uno sviluppatore di applicazioni interessato a incorporare funzionalità da FBX a GLTF nella tua applicazione .NET, Java o Python, puoi prendere in considerazione l'utilizzo delle API Aspose.3D per questo scopo.
