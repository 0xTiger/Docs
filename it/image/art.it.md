## Che cos'è un file ART?

Un file ART è un file immagine bitmap compresso creato e utilizzato dal servizio America Online (AOL) e dal software client. È stato creato per ridurre le dimensioni dell'immagine per il download rapido su Internet. L'algoritmo di compressione applicato all'immagine per generare file .art è simile al formato [JPEG](/it/image/jpeg/), ma a volte riduce la qualità dell'immagine favorendo la riduzione delle dimensioni del file. I file ART possono essere aperti utilizzando AOL File Viewer.

## Formato file ART

Il formato di file ART utilizza la compressione per un facile trasferimento su Internet e la condivisione tra gli utenti AOL. È stato utilizzato per la presentazione di immagini nel servizio online utilizzando il browser Web del client. Le dimensioni ridotte dei file ART facilitano download più veloci anche su connessioni Internet più lente nei primi giorni.
