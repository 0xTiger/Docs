## Che cos'è un file SVGZ?

Un file con estensione .svgz è una versione compressa del file Scalable Vector Graphics (.SVG). È compresso con compressione gzip e contiene dati in formato [XML](/it/web/xml/). I file SVGZ supportano trasparenza, gradienti, animazioni e filtri. I file SVGZ sono di dimensioni inferiori rispetto ai file SVG predefiniti e questa dimensione ridotta del file aiuta a trasferire i file grafici online. Il designer grafico crea file SVGZ utilizzando strumenti come Adobe Illustrator, Corel PaintShop Pro e altri. Tuttavia, i file SVGZ possono essere generati abilitando [GZip compression](/it/compression/gz/) nel server Apache mentre si inviano i dati dell'immagine.

## Formato file SVGZ

SVGZ, essendo compresso [SVG](/it/page-description-language/svg/), si basa sul formato gzip compresso aperto che riduce le dimensioni dei documenti SVG. Questo archivio compresso contiene al suo interno documenti SVG di testo normale che sono puramente basati sul formato di file XML (/it/web/xml/). I file SVGZ possono essere aperti da visualizzatori/applicazioni conformi alle specifiche SVG 1.1. File SVGZ di dimensioni inferiori del 20-50% rispetto ai file SVG originali.
