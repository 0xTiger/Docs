## Che cos'è un file PCT?

Un file PCT è un formato file Macintosh Picture Image che era il formato file predefinito per il programma Apple QuickDraw. È stato sviluppato da Apple ma dopo l'interruzione del programma QuickDraw non è più in uso. **[PDF](/it/pdf/)**, in generale, ha sostituito il PCT e formati di file simili. I file PCT possono anche essere trovati a volte con estensione di file .pict, sebbene sia PCT che PICT abbiano lo stesso formato di file.

## Formato file PCT - Ulteriori informazioni

I file PCT vengono salvati su disco come file binari in **formato file pict** e i dettagli relativi al formato file interno non sono disponibili. Le applicazioni che possono aprire file PCT includono XnView, GIMP, Corel PaintShop Pro e Apple Preview.

### Tipi di formato file PCT

I dati di immagine in un file PCT Macintosh possono essere salvati nel formato originale `PICT 1` o come formato `PICT 2`.

**PICT 1** - Questo tipo di formato di file PICT può memorizzare otto colori
**PICT 2** - Questo tipo di formato di file PICT supporta migliaia di colori con immagini a 24 e 32 bit
