## Che cos'è un file THM?

Un file THM è un'immagine di anteprima creata automaticamente dal primo fotogramma di un file video su videocamere digitali, fotocamere e smartphone. Viene generato quando un file video viene salvato dalla fotocamera del dispositivo. Le immagini di anteprima in miniatura di tutti i video vengono visualizzate all'utente per la scelta del video di interesse dalla raccolta di video. I file THM vengono salvati in formato binario compresso in memoria per ridurre l'utilizzo dello spazio. I file THM possono essere aperti con applicazioni come Apple Preview e [Adobe Photoshop 2022](https://www.adobe.com/products/photoshop.html).

## Formato file THM - Ulteriori informazioni

I file THM sono immagini raster salvate su disco in formato binario. Lo scopo principale dei file THM è l'identificazione del file video tramite la sua immagine in miniatura. Nella maggior parte dei casi, i file THM hanno lo stesso nome del nome del file video corrispondente. Ad esempio, un file video con nome MOV001.mov avrà MOV001.THM come nome del file di anteprima. L'estensione dei file THM può essere rinominata in [JPEG](/it/image/jpeg/) per aprirli se non si aprono direttamente facendo doppio clic.

## Va bene eliminare i file THM?

I file THM sono immagini in miniatura per i video registrati dalle videocamere. L'eliminazione di questi non avrà alcun effetto poiché il video originale rimane conservato.
