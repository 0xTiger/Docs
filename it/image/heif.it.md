## Che cos'è un file HEIF?

Un file HEIF è un formato di file immagine contenitore ad alta efficienza in grado di memorizzare una singola immagine o una sequenza di immagini in un unico file. Pertanto, è fondamentalmente un contenitore con più immagini memorizzate. Il formato file HEIF comprime le immagini utilizzando lo standard HEVC (High Efficiency Video Coding). È più efficiente e leggero rispetto ad altri formati di file immagine come [JPEG](/it/image/jpeg/). Un singolo file HEIF può memorizzare sequenze di immagini come animazioni [GIF](/it/image/gif/) o sequenze HDR, insieme a metadati che descrivono ciascuna immagine. Apple Inc. ha adottato il suo formato di file HEIC variante per iOS 11. Ciò ha portato altre società a supportarlo nel proprio software, incluso Adobe.

## Formato file HEIF

Il formato HEIF è stato sviluppato da Moving Picture Experts Group (MPEG) ed è stato registrato secondo lo standard ISO/IEC 23008-12 nel 2015. È stato adottato da numerosi produttori come Apple, Microsoft e Canon. I file HEIF vengono archiviati con estensione .heif (se contiene una singola immagine) o come .heifs (se contiene una sequenza di immagini).

### Archiviazione dati HEIF

Il formato HEIF può memorizzare i seguenti tipi di immagini e dati.

* Singole immagini e miniature delle immagini
* Sequenze di immagini
* Istruzioni di post-elaborazione
* Metadati dell'immagine, come i dati [EXIF](/it/image/exif/).

Il formato immagine HEIF offre la capacità di diversi formati immagine tra cui [JPEG](/it/image/jpeg/), [PNG](/it/image/png/) e [GIF](/it/image/gif/) senza compromessi sull'immagine qualità.

## Sistemi operativi supportati

I seguenti sistemi operativi supportano il contenuto HEIF:

* Android 9 Pie e versioni successive
* Apple iOS 11 e versioni successive
* Apple macOS High Sierra e versioni successive
* Aggiornamento di Microsoft Windows 10 aprile 2018 (1803) e versioni successive
