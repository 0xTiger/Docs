## Che cos'è un file PSP?

Un file PSP è un'immagine bitmap colorata creata con PaintShop Pro. È il formato file nativo di PaintShop Pro e supporta file compressi non compressi, compressi RLE, compressi LZ77 e [JPEG](/it/image/jpeg/) (incluso Lossless JPEG). Supporta anche livelli simili ai file [PSD](/it/image/psd/) e può contenere altre informazioni come guide, metadati e oggetti rilevanti all'interno del file di immagine. Ogni livello può essere modificato e integrato con l'immagine principale da solo. I file PSP possono essere aperti con il software PaintShop Pro.

## Formato file PSP

I file PSP vengono salvati su disco come file binari con il supporto di diversi algoritmi di compressione. Le informazioni sui bit per pixel possono essere archiviate in un file PSP con la seguente opzione.

* 1 (grigio)
* 4 (grigio)
* 8 (grigio o colorato)
* 24 (colore)

I file PSP possono essere salvati su entrambi i sistemi operativi x32 e x64. [PSPIMAGE](/it/image/pspimage/) è un altro formato di file utilizzato da PaintShop Pro.
