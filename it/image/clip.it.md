## Che cos'è un file CLIP?

Un file CLIP è un file di immagine raster generato dall'applicazione di disegno 2D Clip Studio Paint. È simile al file [PSD](/it/image/psd/) di Adobe e contiene più livelli per gestire diversi elementi grafici sotto forma di livelli come testo, immagini, penna, matita, pennelli e forme. Il formato file CLIP viene utilizzato per progettare diversi tipi di grafica come illustrazioni e animazioni. L'applicazione Clip Studio Paint è stata sviluppata da una società giapponese CELSYS, Inc. fondata nel maggio 1991.

## Formato file CLIP - Ulteriori informazioni

Il formato di file CLIP è stato introdotto da CELSYS per sostituire il formato di file LIP originale. Memorizza elementi grafici sotto forma di livelli che possono essere modificati per le modifiche. È anche il formato file predefinito per l'applicazione [CLIP Studio Paint](https://www.celsys.co.jp/en/service/studio_paint/) disponibile per piattaforme Windows, MacOS, Android e iOS.

## Tipi di disegno supportati

Il formato di file CLIP può memorizzare animazioni e illustrazioni che possono essere utilizzate in una varietà di diversi tipi di lavoro.
