## Cos'è un file AFX?

Un file AFX è un file immagine utilizzato dal software Auto FX PhotoGraphic Edges utilizzato per il miglioramento delle immagini digitali. Contiene l'immagine di sovrapposizione raster che viene utilizzata sopra un'altra immagine per ottenere l'effetto fotografico. Questo processo migliora l'immagine di output creando effetti digitali sulle immagini del client. Conferisce inoltre un aspetto digitale migliorato alle fotografie che sono molto migliori da presentare.

## Formato file AFX

I file AFX vengono salvati come file binari che possono essere aperti utilizzando il software Auto FX PhotoGraphic Edges.
