## Cos'è un file AGIF?

Un file AGIF è un file GIF creato con Active GIF Creator. È un file di progetto che può essere aperto e visualizzato nei progetti Active GIF Creator. Quando viene aperto in Active GIF Creator, mostra una sequenza di immagini che infine compongono il file GIF finalizzato con estensione .gif.

## Formato file AGIF

AGIF è un file di progetto che viene salvato come formato di file immagine raster. Il file [GIF](/image/gif/) finalizzato può essere aperto e visualizzato come animazione. Sono disponibili API Python online che possono essere utilizzate per creare file GIF animati.
