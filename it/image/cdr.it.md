## Che cos'è un file CDR?

Un file CDR è un file immagine di disegno vettoriale creato in modo nativo con [CorelDRAW](https://www.coreldraw.com/en/pages/open-cdr-file/) per la memorizzazione di immagini digitali codificate e compresse. Tale file di disegno contiene testo, linee, forme, immagini, colori ed effetti per la rappresentazione vettoriale del contenuto dell'immagine. I file CDR possono essere aperti con CorelDRAW come applicazione principale e possono anche essere convertiti in altri formati come [PDF](/it/pdf/), [JPG](/it/image/jpeg/), [PNG](/it/image/png/ ), [BMP](/it/image/bmp/) e AI. Può essere utilizzato per la rappresentazione di vari dati grafici come brochure, tabloid, buste e cartoline. Oltre a CorelDRAW, anche altri prodotti Corel come Corel Paintshop Pro e CorelDRAW Graphics Suite possono aprire i formati di file CDR.

## Formato file CDR

Il CDR, essendo un formato proprietario di Corel, non ha specifiche tecniche pubblicamente disponibili. Il formato è stato in continua evoluzione dopo il suo primo lancio nel 1989 e supporta funzionalità di base come testo, gestione del colore e tinte piatte.
