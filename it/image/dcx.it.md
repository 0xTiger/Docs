## Che cos'è un file DCX?

Un file DCX è un file immagine costituito da più pagine per la generazione di file fax digitali. Ciascun file DCX inizia con un'intestazione di 4 byte che identifica il file DCX come firma, seguita da singole immagini, memorizzate come file [PCX](/it/image/pcx/) incorporati. La firma a 4 byte identifica il file DCX ed è seguita da un elenco di fino a 1023 offset di file a 4 byte. Il formato file DCX è stato sviluppato da Zsoft. Diverse applicazioni possono aprire file DCX come Adobe Photoshop, Corel PaintShop Pro e IrfanView.
