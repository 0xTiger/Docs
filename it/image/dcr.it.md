## Che cos'è un file DCR? ##
Un file con estensione dcr contiene il contenuto di una fotografia digitale salvata nel formato Kodak Digital Camera RAW (DCR). Il DCR è l'abbreviazione di **Digital Camera Raw**; contiene la versione non compressa e senza perdita di dati delle immagini acquisite da una fotocamera digitale Kodak. Ai fotografi professionisti piace utilizzare questi file, perché memorizzano le immagini con una qualità superiore rispetto ai formati di immagine compressa di bassa qualità.

## Formato file DCR
Il DCR è un formato proprietario sviluppato da Eastman Kodak Company; indicato semplicemente come Kodak. Un file immagine Digital Camera Raw (DCR) contiene dati minimamente elaborati dal sensore di immagine di una fotocamera digitale. Questi file non sono ancora elaborati e quindi non sono pronti per essere stampati o modificati con un editor grafico bitmap.
Di solito, un convertitore RAW elabora l'immagine in uno spazio colore interno ad ampia gamma in cui è possibile apportare accuratezza e perfezionamento prima della conversione in un formato di file raster come TIFF o JPEG per l'archiviazione, la stampa o un'ulteriore manipolazione.
### Contenuto del file
La struttura dei file non elaborati segue spesso uno schema generico:
- Una breve intestazione di file che contiene un indicatore dell'ordinamento dei byte del file.
- Metadati del sensore della fotocamera necessari per interpretare i dati dell'immagine del sensore, inclusi gli attributi del CFA, la dimensione del sensore e il suo profilo colore.
- Metadati dell'immagine che possono essere utili per l'inclusione in qualsiasi ambiente o database CMS. Alcuni file non elaborati contengono una sezione di metadati standardizzata con dati in formato Exif.
- Una miniatura dell'immagine.
- Una conversione JPEG a dimensione intera dell'immagine, che viene utilizzata per visualizzare in anteprima il file sul pannello LCD della fotocamera.
- Nel caso di scansioni di pellicole cinematografiche, il codice temporale, il codice chiave o il numero di fotogramma nella sequenza di file che rappresenta la sequenza di fotogrammi in una bobina scansionata.
- I dati dell'immagine del sensore
### Dati dell'immagine del sensore
Il file raw svolge il ruolo nella fotografia digitale in modo simile a quello della pellicola fotografica nella fotografia cinematografica. I file Raw, quindi, contengono i dati a piena risoluzione letti da ciascuno dei pixel del sensore di immagine della fotocamera. Il sensore della fotocamera è quasi costantemente sovrapposto a un CFA (Color Filter Array. Può essere utilizzato nella conversione di immagini ad alta gamma dinamica quando sono disponibili dati in formato raw; come alternativa più semplice all'approccio HDI multi-esposizione per acquisire tre immagini separate.
