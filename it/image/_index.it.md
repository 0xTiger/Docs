Un formato di file immagine è un metodo standard per organizzare e archiviare le immagini su dispositivi come computer, tablet e smartphone. Le immagini digitali memorizzano i dati dell'immagine in una griglia di pixel bidimensionale in cui ogni pixel è una rappresentazione del colore in termini di numero di bit. I tipi di file immagine sono classificati in formati immagine vettoriali e formati immagine raster. Le immagini 3D sono un altro tipo di formato di file di immagine vettoriale utilizzato per la gestione delle immagini 3D.

## Formati raster ##

**Grafica raster** sono immagini digitali che comprendono dati in pixel per la rappresentazione dei colori. Questi sono i tipi di immagine più comuni per la grafica utilizzata per il Web e per le foto digitali. Alcune delle immagini raster possono essere compresse per ridurre le dimensioni del file immagine. Le estensioni di file di immagine raster comuni e i relativi formati di file includono **[BMP](/it/image/bmp/)** (file di immagine bitmap), **[PNG](/it/image/png/)** (Portable Network Graphics) e **[GIF](/it/image/gif/)** (File di scambio grafico).

## Formati vettoriali ##

**Le immagini vettoriali** sono definite da punti 2D, anziché da pixel, che sono collegati per dare una forma geometrica all'immagine. I punti hanno proprietà che definiscono la direzione dei tracciati, il colore, la forma, la curva, lo spessore e il riempimento. Le comuni estensioni dei file di immagini vettoriali e i relativi formati di file includono **[SVG](/it/page-description-language/svg/)** (Scalable Vector Graphics), **[EPS](/it/page-description-language/eps/) ** (linguaggio Encapsulated PostScript) e **[PDF](/it/pdf/)** (Portable Document Format).
