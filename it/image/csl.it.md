## Che cos'è un file CSL?

Un file Corel Symbol Library (CSL) è una libreria di simboli utilizzata all'interno del software CorelDRAW. Memorizza simboli che possono essere utilizzati in altri file. I file CSL possono essere archiviati localmente o su una rete per renderli accessibili per una facile distribuzione e gestione. Questi vengono salvati con il file CorelDRAW insieme a tutte le istanze di simboli utilizzate all'interno del documento.

## Formato file CSL

I file CSL vengono salvati nel formato di file proprietario di CorelDRAW. Essendo una raccolta di simboli, i simboli all'interno di un file CSL possono essere importati e aggiunti ai disegni in un progetto.
