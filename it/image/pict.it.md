## Che cos'è un file PICT?

Un file PICT è un'immagine grafica salvata nel formato di file immagine Apple Macintosh. Può memorizzare dati di immagine sia come formati di file raster che vettoriali. Era il formato di file immagine predefinito per il programma Apple QuickDraw. I file PICT consentono lo scambio di dati raster e vettoriali tra applicazioni Mac. Il formato è stato a lungo sostituito dal più flessibile formato di file [PDF](/it/pdf/). I file PICT si trovano a volte con estensione **[.PCT](/it/image/pct/)**.

## Formato file PICT - Ulteriori informazioni

I file PICT vengono salvati come file di immagine binari che possono essere aperti con applicazioni come XnView, GIMP, Corel PaintShop Pro e Apple Preview. Supporta immagini a canale singolo (con capacità di memorizzazione di 8 colori) o immagini RGB a canale colore in grado di memorizzare migliaia di colori. Questi sono noti rispettivamente come formati di file **PICT 1** e **PICT 2**.
