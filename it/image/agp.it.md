## Cos'è un file AGP?

Un file AGP è un file immagine a 32 bit con più livelli. Viene creato con il software ArtGem che fornisce le funzionalità per lavorare con le immagini ed è simile ad Adobe Photoshop nelle funzionalità di base. AGP è un file di progetto che contiene tutte le impostazioni per creare l'immagine finale.

I file AGP possono essere aperti con RL Vision ArtGem. Tuttavia, ArtGem è stato interrotto da tempo dopo essere stato venduto.

## Formato file AGP

I file AGP sono stati salvati nel formato file proprietario ArtGem. La struttura dei file interni dei file AGP non è disponibile pubblicamente come riferimento per gli sviluppatori.
