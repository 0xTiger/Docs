## Cos'è un file AFPHOTO?

Un file AFPHOTO è un file immagine creato dal software di creazione di immagini, Serif Affinity Photo. Contiene il contenuto complessivo dell'immagine sotto forma di più livelli di immagine, forme, testo, tratti di pennello, campioni, effetti di colorazione e illuminazione. Serif Affinity Photo è simile ad Adobe Photoshop, piuttosto è un'alternativa ad Adobe Photoshop con quasi le stesse caratteristiche e capacità. Adobe Photoshop salva i file come [.psd file](/image/psd/) mentre i file [Affinity Photo](https://affinity.serif.com/en-gb/photo/) vengono salvati con estensione .afphoto.

Serif Affinity Photo può aprire file AFPHOTO.

### Formato file AFPHOTO

Il file AFPHOTO viene salvato nel formato file binario proprietario Serif Affinity Photo. La struttura del formato file interno e le specifiche dettagliate non sono disponibili per riferimento pubblico.

## Come convertire un file AFPHOTO?

I file AFPHOTO possono essere convertiti in numerosi formati diversi utilizzando il software Serif Affinity Photo. Sono supportati i seguenti formati di file di esportazione.

 * [PSD](/image/psd/) - Documento di Adobe Photoshop
 * [PDF](/pdf/) - Formato documento portatile
 * [JPG](/image/jpeg/) - Immagine JPEG
 * [PNG](/image/png/) - Grafica di rete portatile
 * [GIF](/image/gif/) - Formato di interscambio grafico
 * [EPS](/page-description-language/eps/) - Encapsulated PostScript
 * [SVG](/page-description-language/svg/) - Scalable Vector Graphics
 * [TIFF](/image/tiff/) - Formato file immagine con tag
 * TGA - Grafica Targa
 * EXR - Immagine OpenEXR
 * [HDR](/image/hdr/) - Immagine ad alta gamma dinamica
