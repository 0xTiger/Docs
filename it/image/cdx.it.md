## Cos'è un file CDX?

I file CDX vengono creati da CorelDRAW, un programma di editing di grafica vettoriale. Questi file sono una versione compressa dei file di disegno .CDR, che sono il formato di file principale utilizzato da CorelDRAW. Lo scopo della creazione di file CDX è ridurre la quantità di spazio di archiviazione richiesto per un disegno.

## Informazioni su CorelDRAW

CorelDRAW è un popolare programma di editing di grafica vettoriale che fa parte del pacchetto software CorelDRAW Graphics Suite. Quando crei un disegno in CorelDRAW e lo salvi, il formato file predefinito è in genere .CDR (CorelDRAW Drawing). Tuttavia, per rendere questi file più gestibili in termini di archiviazione e più facili da condividere, CorelDRAW consente agli utenti di salvare i disegni in un formato compresso noto come CDX (CorelDRAW Compressed File).

Il formato file CDX funge da versione compressa del file .CDR originale. La compressione è una tecnica utilizzata per ridurre la dimensione del file codificando le informazioni in modo più efficiente. Ciò è particolarmente vantaggioso per i disegni con dettagli intricati e grafica complessa, poiché aiuta a ridurre al minimo le dimensioni del file senza compromettere la qualità del contenuto visivo.

Il vantaggio principale dei file CDX è la dimensione ridotta, che li rende ideali per la condivisione tramite e-mail o l'archiviazione su supporti di archiviazione esterni dove lo spazio può essere limitato. Quando devi distribuire disegni CorelDRAW ma vuoi mantenere le dimensioni dei file gestibili, optare per il formato CDX può essere una soluzione pratica. Consente agli utenti di mantenere l'integrità del proprio lavoro grafico facilitando al tempo stesso il trasferimento dei file e la gestione dell'archiviazione.

## Come aprire il file CDX?

I file CDX possono essere aperti dai seguenti programmi

- **CorelDRAW Graphics Suite 2023** (prova gratuita) per Windows
