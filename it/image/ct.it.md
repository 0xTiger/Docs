## Che cos'è un file CT?

Un file CT (Continuous Tone) è un file di immagine raster utilizzato dalle apparecchiature di elaborazione grafica Scitex. Diverse suite grafiche come Adobe Photoshop, Adobe InDesign e QuarkXPress ne supportano l'utilizzo per il rendering. L'immagine dei dati viene archiviata all'interno del file CT nei formati immagine CMYK (Cyan, Magenta, Yellow Key), RGB e in scala di grigi, ma non supportano i canali alfa. I file CT possono essere aperti con applicazioni come Adobe Photoshop 2022, Adobe InDesign 2022 e XnViewMP.

## Formato file CT

Un file Scitex CT è costituito da un blocco di controllo, un blocco di parametri e i dati dell'immagine come da questo [articolo di Wikipedia CT](https://en.wikipedia.org/wiki/Scitex_CT). Le specifiche del formato file del formato file CT non sono disponibili, ma alcune ricerche hanno dimostrato che un'immagine CT rappresenta 4 colori, con ogni pixel colorato di dimensioni fino a 128 bit. I file CT sono generalmente di grandi dimensioni e consentono immagini di alta qualità.
