## Che cos'è un file JPC?

Un file JPC è un formato di file immagine codificato utilizzando il flusso di codice [JPEG2000](/it/image/jp2/). Questi sono anche a volte indicati come file Japan Picture. I file JPC utilizzano la compressione wavelet avanzata JPEG 2000, utilizzata anche dai file codificati JPEG2000. Questi supportano profondità di colore di 8, 24 e 32 bit per pixel negli spazi colore scala di grigi, RGB, YCbCr, XYZ e Lab.

I file JPC possono essere aperti con Adobe Photoshop 2022, Adobe Photoshop Elements e Adobe Illustrator 2021.
