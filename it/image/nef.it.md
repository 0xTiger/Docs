## Che cos'è un file NEF?

NEF è il formato di file di immagine RAW di Nikon, acquisito con le fotocamere digitali Nikon. Memorizza i dati dell'immagine grezza acquisiti dal sensore della fotocamera, nonché i metadati sull'immagine come l'identificazione della fotocamera, le sue impostazioni e altre informazioni correlate. Viene memorizzato nella scheda di memoria della fotocamera come forma compressa non compressa o senza perdita di dati.

## Formato file NEF - Ulteriori informazioni

I file NEF vengono scritti sulla scheda di memoria della fotocamera come file binari nel formato di file proprietario di Nikon. Questi vengono scritti sulla scheda di memoria della fotocamera in forma grezza che non richiede alcuna regolazione dei dati caratteristici dell'immagine come bilanciamento del bianco, tonalità, tono e nitidezza. Le modifiche dell'utente vengono salvate come set di istruzioni accanto ai dati dell'immagine grezza. Ciò avvantaggia in termini di mantenimento intatto del contenuto originale dei dati dell'immagine e ripristino di eventuali set di istruzioni precedenti.

I file NEF possono conservare dati a 12 o 14 bit a seconda della fotocamera, fornendo così un'immagine con una gamma tonale molto più ampia rispetto a un [JPEG](/it/image/jpeg/) o [TIFF](/it/) a 8 bit immagine/tiff/) immagini.

## Software in grado di aprire Nikon NEF File

I file Nikon NEF possono essere aperti utilizzando il seguente software.

* Microsoft Foto - Sistema operativo Windows
* Anteprima Apple - MacOS
* Adobe Photoshop
* Corel PaintShop Pro
* Google Foto

## Converti file NEF

È possibile convertire i file Nikon NEF in altri formati di file immagine come PNG e JPEG. Nikon fornisce l'opzione [Elaborazione RAW NEF](https://onlinemanual.nikonimglib.com/z50/en/09_menu_guide_07_01.html) per convertire i file NEF in JPG dall'opzione del menu della fotocamera.

I file NEF possono anche essere convertiti in altri file di immagine utilizzando Adobe Photoshop, Corel PaintShop Pro, Apple Photos o Corel AfterShot Pro su sistemi operativi Windows e MacOS.
