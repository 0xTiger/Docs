## Che cos'è un file SKP?

Un file SKP è un file di modello tridimensionale proprietario creato con il programma di progettazione 3D SketchUp. Può comprendere diversi elementi o componenti 3D come sfumature, wireframe e trame. Un file SKP può contenere l'intera architettura o può essere costituito da design più piccoli (come una porta o una sedia) di un design più grande. Ciò consente di riutilizzare i modelli definiti in tali file inserendoli in altri documenti di SketchUp.

I file SKP possono essere identificati dalla loro estensione, ad esempio .skp, che possono essere facilmente cercati e trovati su un sistema utilizzando la funzione di ricerca del sistema operativo.

## Formato file SKP - Ulteriori informazioni

I file SKP vengono salvati su disco nel formato di file binario proprietario di SketchUp. Il formato di file SKP può memorizzare immagini 2D oltre alle immagini 3D. Questi documenti 2D possono anche mostrare piante, dettagli di costruzione e prospetti.
