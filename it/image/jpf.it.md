## Che cos'è un file JPF? ##

Un file con estensione .jpf è un'estensione del sistema di codifica delle immagini JPEG 2000 ISO/IEC 15444 ed è indicato come parte 2 ISO/IEC 15444-2. Definisce e specifica un insieme di metodi di compressione lossless (conservazione dei bit) e lossy per la codifica di immagini fisse digitali a colori, a toni continui, bi-livello, in scala di grigi, o immagini multicomponente. La prima parte di ISO/IEC 15444-1 è riferita a [JP2](/it/image/jp2/) che utilizza la tecnologia wavelet per codificare contenuti senza perdita di dati ed è la base per i formati di file immagine JPEG 2000. Il formato di file JPF non ha ricevuto un'accoglienza calorosa a causa dell'ampio utilizzo del formato JPEG. I file JPF possono essere aperti con le applicazioni di imaging più diffuse come Adobe Photoshop 2020, Adobe Illustrator 2020 e CorelDraw Graphics Suite 2020.

## Breve storia

Nel 2000, il comitato del Joint Photographic Experts Group ha progettato JP2 con l'obiettivo di migliorare il proprio standard JPEG basato sulla trasformata del coseno discreto con questo nuovo metodo basato su wavelet. Il comitato JPEG mirava a fornire i propri metodi di base senza costi di licenza. Nella licenza JP2 guadagnando concorrenza tra 20 aziende, hanno vinto per un soffio. JPEG 2000 è stato dichiarato standard ISO, sebbene la maggior parte dei browser Web non sia pronta a dare una mano a JPEG 2000 dal 2017. Nel 2004, il formato ISO/IEC 15444-2 è stato pubblicamente accettato come estensione del formato di file JP2.

## Formato file JPF

Il formato file JPF definisce processi di decodifica estesi per la conversione di dati immagine compressi per la ricostruzione. È un formato di file esteso che specifica la sintassi del flusso di codice estesa contenente informazioni per interpretare i dati dell'immagine compressa. Questo standard esteso specifica un contenitore per archiviare i metadati dell'immagine e fornisce una guida sui processi di codifica estesi per convertire i dati dell'immagine di origine in dati dell'immagine compressi.

### Organizzazione dei file

JPF è il formato di file di archiviazione formale quando i file JPX sono archiviati nei file system del computer. Inoltre, altre Raccomandazioni/Standard internazionali possono definire altre caselle da utilizzare all'interno dei file JPX. Tuttavia, tutte le informazioni contenute in un file JPX devono essere nel formato box; i flussi di byte non nel formato box non devono essere trovati nel file. La struttura binaria di un box in un file JPX è identica a quella definita nel formato file [JP2](/it/image/jp2/).
