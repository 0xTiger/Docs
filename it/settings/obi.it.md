## Cos'è un file OBI?

Il file OBI è un file di sottoscrizione RSS di Outlook che consente agli utenti di iscriversi a un feed RSS in Microsoft Outlook. RSS sta per Really Simple Syndication, un formato di feed Web utilizzato per pubblicare contenuti aggiornati di frequente come post di blog, titoli di notizie o podcast.

Per iscriversi al feed RSS in Outlook, gli utenti possono fare clic sul collegamento al feed RSS su un sito Web o copiare l'URL del feed RSS e incollarlo nella finestra di dialogo "Aggiungi nuovo feed RSS" in Outlook. Outlook controllerà quindi periodicamente la presenza di nuovi contenuti nel feed RSS e li visualizzerà nella cartella dei feed RSS dell'utente.

I file di sottoscrizione RSS di Outlook hanno l'estensione file ".rss". Questi file contengono l'URL del feed RSS e possono essere utilizzati per condividere le sottoscrizioni ai feed RSS con altri o per eseguire il backup e ripristinare le sottoscrizioni ai feed RSS.

## Formato file OBI: ulteriori informazioni

I file OBI sono noti come file OPML (Outline Processor Markup Language) e contengono un elenco di URL di feed RSS a cui è stato effettuato l'abbonamento in Microsoft Outlook.

Quando l'utente esporta i propri feed RSS nel file OBI, contiene l'elenco di tutti i feed RSS a cui si è iscritto, incluso il titolo di ciascun feed, l'URL del feed e qualsiasi altra informazione rilevante. Ciò consente agli utenti di trasferire facilmente le proprie iscrizioni ai feed RSS su un altro lettore RSS o di eseguire il backup delle proprie iscrizioni per motivi di sicurezza.

I file OBI possono essere utilizzati anche per importare abbonamenti a feed RSS in Microsoft Outlook o in un altro lettore RSS. Per importare il file OBI in Outlook, gli utenti possono fare clic su "File" > "Apri ed esporta" > "Importa/Esporta" > "Importa da un altro programma o file" > "Avanti" > "Importa feed RSS da un file OPML" e quindi individuare la posizione del file OBI sul proprio computer.

## Qual è il formato del file OBI?

Il file di sottoscrizione RSS di Outlook, noto anche come file OBI, utilizza XML (eXtensible Markup Language) per definire la struttura e il contenuto del file.

I file OBI sono costituiti da una serie di elementi nidificati ciascuno con il proprio set di attributi e valori. L'elemento principale nel file OBI è l'elemento "contorno" che contiene informazioni sul feed RSS tra cui il titolo del feed, l'URL del feed e qualsiasi altra informazione rilevante.

Ogni elemento "contorno" può anche contenere elementi figlio che rappresentano sottocartelle o sottocategorie all'interno dell'elenco dei feed RSS. La struttura del file OBI consente una facile organizzazione e gestione degli abbonamenti ai feed RSS e può essere utilizzata per trasferire abbonamenti tra diversi lettori RSS o per eseguire il backup e ripristinare gli abbonamenti.
