I file di impostazioni sono file di configurazione e sono anche noti come file di preferenze, memorizzano le impostazioni sia per il sistema operativo che per le applicazioni. Questi file non sono destinati all'accesso dell'utente, ma vengono invece alterati dall'applicazione corrispondente quando vengono modificate le preferenze del programma.
