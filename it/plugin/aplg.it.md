## Cos'è un file APLG?

Un file con estensione .aplg è un file plugin utilizzato dal software di registrazione dello streaming Internet Audials. Il plug-in viene caricato dal software per aggiungere nuove funzionalità come nuove funzioni di ricerca e registrazione in modo da acquisire musica e flussi video su Internet. Audials offre un lettore radio ricco di funzionalità che consente agli utenti di accedere e ascoltare un vasto numero di stazioni radio Internet.

## Formato file APLG

I file APLG vengono forniti inclusi nei pacchetti di plug-in .APLP, il che significa che il software Audials può caricare più plug-in dal pacchetto contemporaneamente. Gli utenti possono spesso registrare le loro canzoni o programmi radiofonici preferiti per riprodurli successivamente. Questo può essere utile per creare playlist personalizzate. Audials in genere mantiene un ampio database di stazioni radio, consentendo agli utenti di scoprire facilmente nuovi contenuti.
