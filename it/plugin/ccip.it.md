## Cos'è un file CCIP?

Un file CCIP è un file plugin creato dal programma Curse Client che viene utilizzato per gestire e installare componenti aggiuntivi per giochi popolari come World of Warcraft, Minecraft e altri. I componenti aggiuntivi sono modifiche o miglioramenti creati dall'utente all'interfaccia o alla funzionalità di un gioco e possono migliorare l'esperienza di gioco fornendo funzionalità aggiuntive o opzioni di personalizzazione. Il file CCIP contiene le informazioni necessarie per scaricare e installare il componente aggiuntivo per uno dei giochi gestiti.

## Formato file CCIP

Quando selezioni l'opzione Installa tramite Curse Client dal programma, i file vengono scaricati come file CCIP. Questi file CCIP possono essere scaricati tramite Curse Client. Il client Curse consentiva agli utenti di scoprire, scaricare e gestire facilmente i componenti aggiuntivi del gioco. Forniva agli utenti una posizione centralizzata per trovare e installare le ultime versioni dei loro componenti aggiuntivi preferiti.
