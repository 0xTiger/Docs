## Cos'è un file 8BI8?

Il file .8BI8 è un file plug-in che migliora il software Photoshop introducendo filtri o effetti aggiuntivi. Per funzionare correttamente, dovrebbe essere inserito nella cartella dei plug-in all'interno della directory di Photoshop.
