## Cos'è un file RPLN?

Un file RPLN è un file plug-in utilizzato dal programma di impaginazione, **InDesign**, che crea il design della pagina e fornisce anche la pubblicazione di stampe. contiene funzioni necessarie per l'esecuzione e si trova in una directory Richiesta in cui è installato il software InDesign.

## Formato file RPLN

I file RPLN vengono salvati nel formato file proprietario InDesign e non sono presenti ulteriori dettagli sulla struttura dei file interna dei file RPLN.

## Informazioni su Adobe InDesign

Adobe InDesign è un'applicazione software di desktop publishing professionale ampiamente utilizzata per la creazione di supporti stampati e digitali. InDesign supporta l'uso di plug-in, ovvero componenti software aggiuntivi che possono essere aggiunti per estendere le funzionalità del programma.

## Come aprire il file RPLN?

Puoi aprire file RPLN con Adobe InDesign 2023.
