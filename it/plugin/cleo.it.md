## Cos'è un file CLEO?

Un file .cleo è un file plugin utilizzato come parte della libreria CLEO per la serie di giochi GRand Theft Auto. Viene utilizzato per eseguire modifiche che cambiano o espandono il gameplay. Il file CLEO memorizza una libreria nel formato file DLL, aggiungendo utili comandi di scripting (codici operativi) per lavorare con file esterni per modificare la memoria del gioco. Esistono diverse versioni di CLEO realizzate per GTA III, GTA Vice City e GTA San Andreas.

## Formato file CLEO

CLEO 4 consente di creare nuovi codici operativi utilizzando la sorgente dell'SDK CLEO. I tre esempi più comuni di tali file plugin includono:

 * IniFiles.cleo - codici operativi per file INI;
 * FileSystemOperations.cleo: codici operativi per file e cartelle.
 * IntOperations.cleo: codici operativi per operazioni logiche (AND, OR, XOR, ecc.).

I plugin vengono installati in modo simile agli script. Per installare un plugin CLEO, è necessario copiare il file .CLEO nella cartella CLEO.

## Informazioni su CLEO

CLEO è un plugin estensibile estremamente popolare per la serie di giochi Grand Theft Auto di Rockstar Games, che consente l'uso di migliaia di mod uniche che cambiano o espandono il gameplay.
