## Cos'è un file AMX?

Un file AMX è un file plugin personalizzato per il plugin metamod AMX Mod per Valve Half-Life e giochi correlati. Aggiunge caratteristiche e funzionalità aggiuntive alle caratteristiche esistenti del gioco.

## Formato file AMX: ulteriori informazioni

AMX Mod utilizza l'estensione del file AMX. Tuttavia, la versione estesa del plugin utilizza l'estensione del file .AMXX.

## Come aprire il file AMXX?

Puoi aprire file AMX utilizzando [AMX Mod X](https://www.amxmodx.org/) sui sistemi operativi Windows e Linux.
