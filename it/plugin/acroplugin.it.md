## Cos'è un file ACROPLUGIN?

Un file con estensione .acroplugin è un file plugin utilizzato da Adobe Acrobat. Viene utilizzato per aggiungere funzionalità alle capacità esistenti dell'applicazione. Il plugin viene fornito in bundle con Adobe Acrobat, un layout di pagina e un programma di pubblicazione [PDF file](/pdf/). Come altri plugin, il file ACROPLUGIN estende le funzionalità esistenti offerte dall'applicazione.

## Formato file ACROPLUGIN: ulteriori informazioni

Il file ACROPLUGIN viene utilizzato esplicitamente da Adobe Acrobat per estendere le sue funzionalità esistenti. I dettagli sul formato del file interno non sono disponibili pubblicamente come riferimento per lo sviluppatore.

## Come aprire il file ACROPLUGIN?

Puoi aprire il file ACROPLUGIN utilizzando Adobe Acrobat.
