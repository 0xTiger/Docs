## Cos'è un file AMXX?

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. È una piattaforma di scripting che consente agli amministratori di server di personalizzare ed estendere la funzionalità dei propri server. AMXX viene utilizzato insieme al sistema di gestione dei plug-in MetaMod che consente l'esecuzione simultanea di più plug-in su un server senza conflitti. Le funzionalità offerte dal plug-in AMXX includono comandi di amministrazione, modifiche al gameplay e gestione del server.
