## Cos'è un file CRX?

Un file CRX è un'estensione utilizzata per estendere la funzionalità dei browser Internet Google Chrome, Microsoft Edge e Firefox. Aggiunge funzionalità specifiche per l'accesso direttamente dal browser per il quale è stato sviluppato il file plug-in. Le estensioni sono create per scopi diversi come analisi del traffico del sito Web, blocco degli annunci, gestione delle password, analizzatore di elenchi di e-commerce, traslatore di pagine Web e molti altri.

## Formato file CRX

I file CRX vengono archiviati come file binario compresso con i suoi file costituenti imballati all'interno dell'archivio. Può contenere file [.JS](/it/web/js/), [.JSON](/it/web/json/), immagini o anche file eseguibili. I browser come Google Chrome consentono anche di caricare le estensioni in modalità sviluppatore che consente un ulteriore debug della funzionalità dell'estensione.

## Come installare un'estensione Web?

I browser Web come Chrome dispongono di un'opzione di menu "Estensioni" che ti consente di caricare i file di estensione CRX. Quando un'estensione viene caricata dal negozio di estensioni online, scarica il contenuto del file dell'estensione, li decomprime e installa l'estensione per caricare la funzionalità desiderata. L'intero processo non richiede alcuna interazione da parte dell'utente per l'installazione del file CRX.

## Come caricare i file crx in Chrome?

Per aprire un file CRX:

  1. Vai al menu delle estensioni nel browser Google Chrome
  1. Trascina e rilascia il file CRX nella pagina dell'estensione del browser
  1. Fai clic su "Aggiungi estensione" quando richiesto
