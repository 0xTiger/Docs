Cos'è un file MDE?

Un file .MDE è una versione compilata di un database Access (.mdb o .accdb). Quando si compila un database di Access in un file .MDE, non è più modificabile utilizzando gli strumenti di progettazione di Access standard. Lo scopo principale della creazione di un file .MDE è distribuire un'applicazione di database senza esporre il codice sorgente originale.

## Formato file MDE

Un file MDE viene creato compilando il codice VBA, moduli, report e altri oggetti. Ciò si traduce nella creazione del formato di file binario MDE. Il file .MDE risultante può essere distribuito agli utenti che possono eseguire l'applicazione ma non possono modificare la progettazione o visualizzare il codice originale. Il file MDE è in realtà la versione compilata di un [file .MDA](/database/mda/). La distribuzione di file .MDE può aiutare a proteggere la proprietà intellettuale impedendo agli utenti di accedere o modificare il codice sorgente. Può anche migliorare le prestazioni man mano che il codice viene compilato e ottimizzato.
