## Cos'è un file Q1Q?

Un file .q1q è un file plugin di Photoshop sviluppato da Flaming Pear Software per la creazione di pianeti 3D. Ti consente di scaricare dati LIVE di nuvole, aria, struttura, clima e altri tonnellate di dettagli. Consente inoltre di esportare tutte le mappe di texture in file multistrato e generare mappe per l'animazione 3D e la creazione di mappe.

## Formato file Q1Q

I file del plug-in Q1Q vengono salvati nel formato file proprietario Flaming Pear e i dettagli del formato file interno non sono disponibili pubblicamente. L'immagine seguente mostra l'interfaccia utente del plug-in Q1Q quando viene caricato in Photoshop.

## Come aprire il file Q1Q?

Puoi aprire file Q1Q utilizzando **Adobe Photoshop 2023** e **Corel PaintShop Pro 2023**.
