## Cos'è un file RBZ?

Un file RBZ è un file plugin utilizzato dal programma di modellazione, **Trimble SketchUp**, per la progettazione assistita da computer (CAD). È un file ZIP compresso che contiene al suo interno uno o più script Ruby (file .rb). Ecco perché il tipo di file è RBZ (RB + Z). I file RBZ sono facili da distribuire e non si trovano in rete a causa delle dimensioni ridotte del file a causa della compressione.

## Formato file RBZ: ulteriori informazioni

I file RBZ sono file ZIP compressi che contengono un file di script Ruby. Viene compresso con il metodo di compressione più comunemente utilizzato **DEFLATE**, descritto in IETF **RFC 1951**.

## Come aprire un file RBZ?

Puoi aprire file RBZ con Trimble SketchUp su Windows e macOS.

Oltre a questi, poiché i file RBZ sono file ZIP compressi, puoi estrarli con le utility di estrazione ZIP standard rinominandone l'estensione da .rbz a .zip.

## Informazioni su Sketchup
