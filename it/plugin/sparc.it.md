## Cos'è un file SPARC?

Un file .sparc, riconosciuto in alternativa come archivio dei plug-in Skype, funge da pacchetto che comprende funzionalità extra o miglioramenti per l'applicazione Skype. Questi plugin hanno la capacità di introdurre una vasta gamma di funzionalità come la registrazione delle chiamate, la condivisione dello schermo, la traduzione,
Giochi e intrattenimento, emoticon e temi personalizzati.

## Formato file SPARC: ulteriori informazioni

In alcune versioni di Skype, Extras Manager è un componente facoltativo che può essere integrato nell'installazione di Skype. Se riscontri difficoltà nell'installazione di un file SPARC, è possibile che tu abbia omesso Extras Manager durante l'installazione iniziale. Per risolvere questo problema, reinstalla Skype e assicurati di selezionare l'opzione Gestione Extra.

## Come installare il plugin Skype?

Passaggi per installare i plugin Skype:

1. Ottieni il file .sparc corrispondente al plugin desiderato.
2. Chiudi Skype se è attualmente in esecuzione.
3. Individua il file .sparc scaricato e fai doppio clic su di esso.
4. Segui le istruzioni visualizzate sullo schermo per completare l'installazione del plugin.
5. Dopo l'installazione, riavvia Skype per attivare il plugin.
