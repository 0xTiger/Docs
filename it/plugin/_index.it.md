I file plug-in migliorano ed espandono le capacità dei programmi software esistenti. Trovano applicazione frequente negli strumenti di editing di immagini, video e audio, nonché nei browser web. Queste aggiunte sono spesso conosciute con termini alternativi come componenti aggiuntivi o estensioni.

Le estensioni comuni dei file plug-in includono .PLUGIN, .8BI e .VST.
