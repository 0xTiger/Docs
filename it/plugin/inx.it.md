## Cos'è un file INX?

Un file INX è un file plugin utilizzato dall'editor di grafica vettoriale open source, **Inkscape**. Contiene informazioni sul programma incluso il nome e la posizione della funzionalità nel menu. Il file INX specifica anche uno script per l'estensione che include informazioni come tipo di script, interprete, posizione e argomenti variabili.

## Formato file INX

I file INX migliorano le funzionalità di Inkscape incorporando funzionalità come importatori di file e filtri grafici. Archiviati in un formato XML, questi file si trovano generalmente nella directory \​extensions\​ all'interno di un'installazione di Inkscape.

## Come aprire i file INX?

Puoi aprire file INX utilizzando il programma **Inkscape**.
