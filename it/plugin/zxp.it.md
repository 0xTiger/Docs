## Cos'è un file ZXP?

Un file ZXP funge da estensione, aumentando le funzionalità del software Adobe come Adobe Photoshop o Adobe After Effects. In passato, Adobe Extension Manager facilitava l'installazione dei file ZXP. Attualmente, il processo di installazione richiede l'utilizzo dello strumento da riga di comando Unified Plugin Installer Agent (UPIA), fornito in bundle con l'applicazione desktop Adobe Creative Cloud.

## Formato file ZXP

Il termine ZXP sta per pacchetto eXtension ZIP. I file ZXP vengono utilizzati per creare pacchetti e distribuire estensioni per applicazioni Adobe Creative Cloud come Adobe Photoshop, Illustrator, InDesign e altre. Un file ZXP viene compresso utilizzando il formato [ZIP](/compression/zip/), che semplifica la distribuzione e l'installazione delle estensioni.

All'interno di un file ZXP, in genere troverai i file richiesti per l'estensione, inclusi [HTML](/web/html/), [CSS](/web/css/), JavaScript, immagini e qualsiasi altra risorsa necessaria per migliorare o aggiungere funzionalità al software Adobe Creative Cloud.

### Come installare il file ZXP?

I file ZXP vengono installati utilizzando Adobe Extension Manager o l'applicazione Creative Cloud Desktop. Gli utenti possono semplicemente fare doppio clic su un file ZXP e il gestore delle estensioni gestirà il processo di installazione.

I file ZXP sono progettati per funzionare su diversi sistemi operativi, inclusi Windows e macOS, rendendo conveniente per gli sviluppatori creare estensioni che possano essere utilizzate da un vasto pubblico.

### Sicurezza del formato file ZXP

I file ZXP possono essere firmati digitalmente dagli sviluppatori, garantendo che l'estensione non sia stata manomessa e provenga da una fonte attendibile. Ciò aiuta a mantenere la sicurezza e l'integrità delle applicazioni Adobe Creative Cloud.

Se lavori con estensioni Adobe Creative Cloud o ne sviluppi una tu stesso, potresti incontrare file ZXP come mezzo per confezionare e distribuire la tua estensione. Tuttavia, le specifiche relative all'utilizzo dei file ZXP potrebbero essere soggette a modifiche o aggiornamenti da parte di Adobe, quindi è una buona idea fare riferimento alla documentazione Adobe più recente per le informazioni più accurate e aggiornate.
