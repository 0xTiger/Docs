## Che cos'è un file OLM?

Un file con estensione .olm è un file di Microsoft Outlook per il sistema operativo Mac. Un file OLM memorizza messaggi di posta elettronica, diari, dati del calendario e altri tipi di dati dell'applicazione. Questi sono simili ai file [PST](/it/email/pst/) utilizzati da Outlook sul sistema operativo Windows. Tuttavia, i file OLM creati da Outlook per Mac non possono essere aperti in Outlook per Windows. Il formato di file OLM è stato introdotto con Outlook 2011 per archiviare i dati locali di Outlook. Un singolo file OLM può contenere più cassette postali, organizzando i dati sotto forma di cartelle e sottocartelle. I file OLM possono essere aperti/caricati in Microsoft Outlook 365 e Outlook per Mac.

## Formato file OLM

OLM sono file binari con tipo MIME Application/octet-stream. Le specifiche tecniche del formato di file OLM non sono disponibili apertamente da Microsoft e non sono disponibili dettagli sul formato per il riferimento dello sviluppatore. Esistono tuttavia alcune API in grado di leggere il formato di file OLM e di scriverli in un file PST per l'utilizzo su sistema operativo Microsoft Windows come Aspose.Email.
