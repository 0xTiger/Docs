## DOC から DOCX への変換について

Microsoft が Office 2007 を導入する前は、すべての Word 文書はデフォルトでバイナリ DOC ファイル形式で保存されていました。 Microsoft Word 2007 の導入により、Microsoft はすべてのバイナリ ファイル形式を Open Office XML 仕様のファイル形式に切り替えました。この移行後、各バイナリ拡張子には、新しいファイル形式を表す「X」が追加されました。新しい DOCX ファイル形式への変換が必要な古い DOC ファイルがまだアーカイブされている可能性があります。いくつかのオンラインツールを使用して、DOC を DOCX に無料で変換できます。 Aspose.Words アプリは、ブラウザ内から DOC を DOCX に変換できるオンライン ツールの 1 つです。

## DOC から DOCX への変換 - 無料デモ

DOC ファイルを操作するための非常に強力な API を備えた Aspose は、DOC を DOCX ファイルに変換するための無料のオンライン アプリを提供しています。オンライン Aspose アプリを使用する利点は次のとおりです。

1. 「コーディングは不要」 - ファイルをアップロードするだけで、変換されたドキュメントが出力として取得されます。
1. 「高速かつ正確」 - オンライン変換ツールを使用して、Word ファイルを迅速かつ正確に DOCX に変換します。
1. 「完全に無料」 - ドキュメントを DOCX やその他の一般的な形式に変換できる十分な無料変換機能

### DOC を DOCX に変換するにはどうすればよいですか?

次のように [Aspose.Words DOC to DOCX アプリ](https://products.aspose.app/words/conversion/doc-to-docx) を使用して、DOC を BMP ファイル形式に変換できます。

1. [DOC から DOCX コンバータ] の Aspose.Words アプリに移動します(https://products.aspose.app/words/conversion/doc-to-docx)
1.アップロードボタンをクリックしてDOCファイルをアップロードします。
1. 完了したら、「変換」ボタンを押します。これにより、DOC ファイルを DOCX に変換するプロセスがバックグラウンドで開始されます。
1. DOC から DOCX への変換プロセスが完了すると、変換された DOCX ファイルをダウンロードするように求められます。
