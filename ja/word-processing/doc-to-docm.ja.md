## DOC から DOCM への変換について

DOCM は、マクロを実行できる Microsoft Word ドキュメントのマクロ有効バージョンです。通常の Word 文書とマクロ有効文書の違いは、DOCM がマクロをサポートする追加機能を提供していることです。

DOC ファイルは、無料のオンライン ツールやアプリを使用して DOCM に変換できます。 **[Aspose.Words アプリ](https://products.aspose.app/words/family)** は、Web ブラウザーから直接 Word 文書に対してさまざまな操作を実行できるオンライン アプリのセットです。 [文書表示](/ja/word-processing/docx-viewer/)、分割、結合、比較、回転などの機能に加えて、Word 文書をある形式から別の形式に変換できます。 Aspose.Words アプリを使用して、[DOC](/ja/word-processing/doc/) を [DOCM](/ja/word-processing/docm/) に変換できます。

## DOC から DOCM への変換 - 無料デモ

Aspose.Words アプリを使用すると、オンラインで DOC を DOCM に無料で変換できます。この DOC から DOCM への変換には、Web ブラウザ内からアクセスできます。高速かつ正確で、コーディングは不要で、完全に無料です。それに加えて、DOC から DOCM への変換を実行するために PC に追加のソフトウェアやツールをインストールする必要はありません。

### DOC を DOCM に変換するにはどうすればよいですか?

次のように [Aspose.Words DOC to DOCM アプリ](https://products.aspose.app/words/conversion/doc-to-docm) を使用して、DOC を DOCM ファイル形式に変換できます。

1. [DOC から DOCM コンバーター] の Aspose.Words アプリに移動します(https://products.aspose.app/words/conversion/doc-to-docm)
1.アップロードボタンをクリックしてDOCファイルをアップロードします。
1. 完了したら、「変換」ボタンを押します。これにより、DOC ファイルを DOCM に変換するプロセスがバックグラウンドで開始されます。
1. DOC から DOCM への変換プロセスが完了すると、変換された DOCM ファイルをダウンロードするように求められます。

Aspose.Words API スイートを使用すると、開発者はわずか数行のコードで [DOC](/ja/word-processing/doc/) を [DOCM](/ja/word-processing/docm/) 形式に変換できます。 C#、Java、Python、C++ などの多くのプログラミング言語で利用できます。 DOCM に加えて、Aspose.Words API を使用すると、DOC ファイルを他の多くの一般的なファイル形式に変換できます。

## ドキュメントビューア

DOC ファイルを開いて表示するための Microsoft Word またはその他の互換ソフトウェアが PC にインストールされていない場合は、無料で利用できるアプリ [Aspose.Words Viewer](/ja/word-processing/docx-viewer/) を使用して開いたり表示したりできます。 DOC ファイルをオンラインで表示します。
