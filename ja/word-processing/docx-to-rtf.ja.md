## DOCX から RTF への変換について

Microsoft Windows の初期のユーザーであれば、RTF (リッチ テキスト フォーマット) ファイル形式を知っている必要があります。 Microsoft は Windows 3.0 で [RTF ファイル形式](/ja/word-processing/rtf/) を導入しました。 RTF ファイルは、アプリケーション内で使用するためにフォーマットされたテキストとグラフィックスをエンコードする方法を表します。 RTF ファイルには、太字、斜体、下線などのテキストの書式設定を保持するだけでなく、テキストと画像を含めることができます。最も興味深いのは、RTF が相互運用可能なファイル形式として採用されており、クロスプラットフォームであることです。このため、編集目的で異なる OS 間でドキュメントを共有する場合、DOCX から RTF への変換が一般的な選択肢となります。
