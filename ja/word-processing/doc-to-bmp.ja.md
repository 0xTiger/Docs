## DOC から BMP への変換について

2007 年より前のバージョンの Microsoft Word ユーザーであれば、それまでに Microsoft Word が文書を保存するために使用するデフォルトのファイル形式であった DOC ファイル タイプをよく知っている必要があります。これは Microsoft Word の開始以来使用されており、最新の DOC ファイル形式に置き換えられた後もかなり長い間使用され続けました。コンテンツ管理システムには、BMP イメージに変換できる DOC ファイルがバックアップとして残っている場合があります。 Aspose.Word アプリなどの変換用オンライン ツールを使用すると、[DOC](/ja/word-processing/doc/) ファイルを [BMP](/ja/image/bmp/) 画像に変換できます。

## DOC から BMP への変換 - 無料デモ

Aspose.Words アプリは、Aspose による人気のプログラミング API に基づいて開発されたオンライン ツールのセットです。ドキュメントの表示、変換、圧縮、回転、分割、結合などの機能を提供します。 Aspose.Words アプリを使用して Web ブラウザから直接 [DOC](/ja/word-processing/doc/) を [BMP](/ja/image/bmp/) に変換できます。変換を実行するためにソフトウェアをインストールする必要はありません。

### DOC をオンラインで BMP に変換する手順

次のように [Aspose.Words DOC to BMP アプリ](https://products.aspose.app/words/conversion/doc-to-bmp) を使用して、DOC を BMP ファイル形式に変換できます。

1. [DOC から BMP コンバータ] の Aspose.Words アプリに移動します(https://products.aspose.app/words/conversion/doc-to-bmp)
1.アップロードボタンをクリックしてDOCファイルをアップロードします。
1. 完了したら、「変換」ボタンを押します。これにより、DOC ファイルを BMP に変換するプロセスがバックグラウンドで開始されます。
1. DOC から BMP への変換プロセスが完了すると、変換された BMP ファイルをダウンロードするように求められます。
