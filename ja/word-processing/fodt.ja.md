## .FODT オプション番号

拡張子が .fodt のファイルは、OpenDocument フラット [XML](/web/xml/) ファイル形式で保存されたテキスト ドキュメントです。 [.zip](/compression/zip/) コンテナー ファイル形式で保存される .odt ファイルとは対照的に、FODT ファイルは人間が読み取れるファイルとしてディスクに保存され、テキスト エディターで開いてプレーンな XML ドキュメントとして表示できます。 . OpenDocument オフィス ドキュメント標準は、OASIS 業界コンソーシアムによって開発され、誰でも誰でも何の制限もなく実装できる道を開きました。その他の一般的な OpenDocument ファイル形式には、[.odt](/word-processing/odt/)、[.ott](/word-processing/ott/)、[.ods](/spreadsheet/ods/)、[.odp] などがあります。 (/presentation/odp/) なども同様です。

## FODT ファイル形式 - 詳細情報

FODT ファイルはプレーン XML ファイル形式 (フラット XML または非圧縮 XML とも呼ばれます) で保存されますが、OpenDocument で一般的に使用されるドキュメント形式ではありません。 OpenDocument ファイルは、通常、多数のファイルとディレクトリで構成される ZIP パッケージとして保存されます。 XML ベースの OpenDocument ファイルは、一部のソフトウェアではサポートされていない可能性があるため、ZIP パッケージ ベースのファイルに比べて使用が一般的ではありません。
