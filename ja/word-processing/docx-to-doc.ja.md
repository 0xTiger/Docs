## DOCX から DOC への変換について

DOC ファイル タイプは、2007 およびそれ以前のバージョンの Microsoft Word で使用されている一般的なドキュメント ファイル形式です。 Office 2007 のリリースに伴い、Microsoft は Office Open XML 仕様に基づいた DOCX ファイル形式を導入しました。圧縮 XML ファイル形式で保存される DOCX と比較して、DOC ファイルはバイナリ ファイルとして保存されます。 [DOCX](/ja/word-processing/docx/) を [DOC](/ja/word-processing/doc/) ファイルに変換するためにオンラインで利用できるツールが多数あります。

## DOCX から DOC への変換 - 無料デモ

無料のオンライン アプリ Aspose.app を使用して、DOCX を DOC ファイル形式に変換できます。これは、DOCX ファイルを読み込み、表示し、DOC、PDF、TXT、RTF などのファイル形式に変換できるアプリのセットです。 Aspose アプリは、さまざまなファイル タイプに対してさまざまな操作を実行できるバックエンドの Aspose API によって強化および実行されます。次のボタンをクリックして、Aspose.Words オンライン アプリを今すぐ試してください。

[![DOCX から DOC](../docx-to-doc.png?width=120px&height=60px)](https://products.aspose.app/words/conversion/docx-to-doc)

### オンラインでDOCXをDOCに変換する手順

次のように [Aspose.Words DOCX to DOC アプリ](https://products.aspose.app/words/conversion/docx-to-doc) を使用して、DOCX を DOC ファイル形式に変換できます。

1. [DOCX から DOC コンバータ] の Aspose.Words アプリに移動します(https://products.aspose.app/words/conversion/docx-to-doc)
1.アップロードボタンをクリックしてDOCXファイルをアップロードします。
1. 完了したら、「変換」ボタンを押します。これにより、DOCX ファイルを DOC に変換するプロセスがバックグラウンドで開始されます。
1. DOCX から DOC への変換プロセスが完了すると、変換された RTF ファイルをダウンロードするように求められます。

## Aspose.Words API

[Aspose.Words](https://products.aspose.com/words/) は、開発者が [DOCX](/ja/word-processing/docx/) を [DOC](/ja/word-processing/doc/) に変換できる API セットです。 形式で、単純なコード行を使用します。 [C#](/ja/programming/cs/)、[Java](/ja/programming/java/)、[Python](/ja/programming/py/)、[C++](/ja/programming/cpp/) など を含む多くのプログラミング言語で利用できます。 ）。 DOC に加えて、Aspose.Words API を使用すると、DOCX ファイルを他の多くの一般的なファイル形式に変換できます。
