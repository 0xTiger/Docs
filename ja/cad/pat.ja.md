## .PAT ファイルとは?

拡張子が .pat のファイルは、AutoCAD ソフトウェアで使用される CAD ファイルです。 PAT ファイルを開くことができるアプリケーションは、これらのファイルに保存されているハッチング パターンを使用して、領域のテクスチャ/塗りつぶしに関する情報を取得します。含まれているパターンは、描画されたオブジェクトのマテリアルの外観に関する情報を提供します。 PAT ファイルは、Autodesk AutoCAD、CorelDRAW Graphics Suite、Ketron Software などのアプリケーションで開くことができます。 PAT ファイルは、JPG、PNG、BMP などのさまざまな画像形式に変換できます。

## PAT ファイル形式

PAT ファイルはプレーン テキスト形式で記述され、任意のテキスト エディターで開くことができます。これらのファイルのハッチング パターンはベクトル ベースであり、AutoCAD のドキュメントで概説されている構文に従います。

すべてのパターンは点 0,0 から始まり、パターンはハッチング境界をカバーするように計算されます。

### パターン定義

すべてのパターン定義は、次のフィールドで構成されています。
* 先頭の '\*'
* 名前 - 名前には、スペース、句読点、括弧、またはスラッシュを含めることはできません。
* 説明

ハッチング パターンの標準形式は `<\*> です。<name> <、説明>`.名前と説明はコンマで区切られ、説明は 80 文字の行の長さを超えることはできません。ハッチング パターンは、この情報の後に定義されます。

### ハッチパターンの例
以下は正方形パターンの例です
```
* square, a square pattern
0,     0,0,   0,1,   1,-1
90,   0,1,   0,1,   1,-1
```
平行四辺形パターンを示す別の例は次のとおりです。

```
*pgram, parallelogram pattern
0,     0,0,    1,1,                     1,-1
45,   0,0,    0,1.4142,   1.4142,-1.4142
45,   0,1,    0,1.4142,   1.4142,-1.4142
```
