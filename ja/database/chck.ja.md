## .CHCK ファイルとは何ですか?

CHCK ファイルは、Microsoft Exchange Server の健全性監視チェックの結果として作成されます。これには、サーバーのデータベースの全体的なステータスと運用状況に関する情報が含まれています。 Microsoft Exchange Server は主に電子メール サーバーとして使用され、ビジネス上の連絡先やカレンダーを管理できます。 Exchange サーバー管理者は、EseUtil を使用してデータベースの正常性をチェックします。

## CHCK ファイル形式

CHCK ファイルの内部ファイル形式は不明であり、CHCK ファイルの読み取り/書き込みに利用できる参考資料はありません。
