## .DAT ファイルとは?
拡張子が .dat のファイルは、一般的なファイル形式に属します。通常、これらは暗号化もバイナリもされていませんが、ほとんど何でもかまいません。さまざまなアプリケーションが、それぞれのプログラムの特定の操作への参照として DAT ファイルを使用しています。そのため、特定のソフトウェアまたはプログラムが DAT ファイルの作成に関与しているとは言えません。これらのファイルは、それらを作成したアプリケーションによってのみ特にアクセスされます。


## DAT ファイル形式
DAT ファイル形式は、ファイル内のあらゆる種類のデータを含む一般的な形式であり、通常は .dat 拡張子で保存されます。バイナリ形式またはテキスト形式のデータが含まれる場合があり、通常、汎用データは、参照するアプリケーションに固有の情報で構成されます。ほとんどの場合、DLL ファイルなどの他の構成ファイルでそれらを見つけることができます。ここでは、.dat 拡張子を持ついくつかの有名なファイル タイプを登録できます。

### VCD ビデオ ファイル
VCD は、1990 年代に DVD より前に映画やホーム ビデオを配信するためのビデオまたはマルチメディア ソースとしてよく知られていました。通常、VCD ディスクからのビデオ ファイルは拡張子が .dat であることが判明したため、VCD から再生された実際のビデオ データが保存されていました。
### 交換メールの添付ファイル
Exchange 電子メールの添付ファイルに関連する DAT ファイルには、Microsoft Exchange Server から送信されたリッチ テキスト (RTF) メッセージの書式設定情報が含まれています。通常、受信者の電子メール クライアントが RTF メッセージをサポートしていない場合、これらのファイルは電子メールの末尾に追加されます。これらのファイルは通常、**Winmail.dat** のように見つかります。
### Windows レジストリ ハイブ ファイル
NTUSER.DAT は、一般的なレジストリ ハイブ ファイルです。 Microsoft Windows レジストリは、DAT ファイルを使用してユーザー設定を保存します。これらの DAT ファイルは、ユーザーが Windows にログインしたときに読み込まれます。これには、アカウント設定、パスワード、およびファイルの関連付けが含まれる場合があります。
### マインクラフトのデータファイル
Minecraft ゲームは、DAT ファイルを使用して、プレーヤーのインベントリ、プレーヤーの位置、およびワールド情報に関する情報を保存しています。 DAT ファイルはレベルの保存にも使用されます。これは、プレイヤーがワールドをナビゲートするときに必要に応じてロードされる小さなマップ ピースです。
### 梨状DATファイル
CCleaner、Speccy、Defraggler、Recuva などのアプリケーションは、通常の操作に関する情報を得るために DAT ファイルを使用しています。
