## .ACCDE オプション番号

拡張子が .accde のファイルは、カスタム データベース コードを保護するために実行専用データベース ファイルを作成するための Microsoft Access 2007 ファイル形式です。すべての VBA コードを実行可能ファイルのみにコンパイルして、表示または編集できないようにします。 ACCDE ファイル形式は、以前のバージョンの Microsoft Access で使用されていた MDE ファイル形式を置き換えるものです。 ACCDE は [ACCDB](/database/accdb/) ファイルであり、実行可能モードのみであり、ソース コードがないためよりコンパクトです。これらのファイルのコンパクトなサイズにより、メモリ使用量が改善されるだけでなく、アプリケーションのパフォーマンスも向上します。 ACCDE ファイルは、Microsoft Access 365 で開くことができます。

## ACCDE ファイル形式

すべての Microsoft Access 2007+ ファイルは、ACCDB ファイル形式ファミリを使用する Jet ACE (Access データベース エンジン) を利用します。ただし、Microsoft はオープン ソースの実装や詳細な技術仕様を提供していません。
