## GLBからUSDZへの変換について

GLB は、GL Transmission 形式 ([glTF](/ja/3d/gltf/)) で 3D モデルを表す一般的な 3D バイナリ ファイル形式です。 .usdz ファイルは、本質的には、[USD](/ja/3d/usd/) (ユニバーサル シーン ディスクリプション) ファイル形式用に設計された非圧縮かつ暗号化されていない ZIP アーカイブです。このアーカイブは、その中に必要とされる代替形式 (テクスチャやアニメーションなど) のさまざまなファイルを便利にカプセル化し、プロキシとして機能します。これらのファイルは USD ランタイムでシームレスに利用できるため、解凍プロセスが不要になります。

オンラインでは、[GLB](/ja/3d/glb/) から [USDZ](/ja/cad/usdz/) への変換に無料で使用できるアプリがたくさんあります。アプリケーション開発者が .NET または Java の USDZ に GLB の機能を追加したい場合は、Aspose.3D API を使用してこれを実現できます。

## GLB から USDZ への変換 - 無料デモ

Aspose.3D 変換アプリを無料で使用して、GLB を USDZ に変換できます。次のボタンをクリックして、GLB ファイルを USDZ に今すぐ変換します。

[![GLB から USDZ](../glb-to-usdz.png)](https://products.aspose.app/3d/conversion/glb-to-usdz)

### GLBをUSDZに変換する手順

GLB ファイルを USDZ に変換するために従う必要がある手順は次のとおりです。

1. [Aspose.3D変換アプリ](https://products.aspose.app/3d/conversion/glb-to-usdz)にアクセスします。
1. GLB ファイルをブラウザにアップロードします
1. 出力画像ファイル形式として USDZ を選択します
1.「変換」ボタンを押します。

これにより、GLB ファイルから USDZ への変換プロセスが開始され、ファイルを PC に保存するためのダウンロード オプションが表示されます。
