## 3DSからPLYについて

「.3ds」ファイル拡張子は、現在 3D Studio MAX として知られる、1990 年代以来 3D ファイル形式の中心的なソフトウェアである Autodesk の 3D Studio で使用されている 3D Studio (DOS) メッシュ ファイル形式を表します。

PLY は Polygon File Format の略で、3D オブジェクトをポリゴンとして保存します。これは多用途であり、ASCII 形式とバイナリ形式の両方に対応しており、後者は保存と検索に効率的です。

[3DS](/ja/3d/3ds/) から [PLY](/ja/3d/ply/) への変換は頻繁に必要となるため、3DS ファイルを PLY 形式に変更できます。この変換には多くの無料のオンライン ツールが利用可能であり、明示的なインストールを必要とせずに Web ブラウザを通じてアクセスできるという利点があります。 Aspose.3D は、無料で試せるツールの 1 つです。
