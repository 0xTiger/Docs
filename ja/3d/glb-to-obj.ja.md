## GLB→OBJについて

GLB は、GL Transmission 形式 ([glTF](/ja/3d/gltf/)) で 3D モデルを表す一般的な 3D バイナリ ファイル形式です。 OBJ ファイルは、Wavefront の Advanced Visualizer アプリケーションによって、幾何学オブジェクトを定義および保存するために使用されます。 GLB ファイルを [OBJ](/ja/3d/obj/) ファイル形式に変換するのは一般的な要件であり、GLB を OBJ に無料で変換できるいくつかのアプリがオンラインで利用可能です。 Aspose.3D API を使用して、.NET または Java アプリケーションで GLB を OBJ に変換できます。

## GLB から OBJ への変換 - 無料デモ

オンラインでGLBをOBJに変換する無料サービスをお探しの場合は、次のボタンをクリックしてAspose.3D変換アプリを使用してこれを実現できます。

[![GLB から OBJ](../glb-to-obj.png)](https://products.aspose.app/3d/conversion/glb-to-obj)

### GLB を OBJ に変換する手順

GLB ファイルを OBJ に変換するために従う必要がある手順は次のとおりです。

1. [Aspose.3D変換アプリ](https://products.aspose.app/3d/conversion/glb-to-obj)にアクセスします。
1. GLB ファイルをブラウザにアップロードします
1. 出力画像ファイル形式として OBJ を選択します
1.「変換」ボタンを押します。

これにより、GLB ファイルを OBJ に変換するプロセスが開始され、ファイルを PC に保存するためのダウンロード オプションが表示されます。
