## 3DSからSTLへの変換について

.3ds ファイル拡張子は、オートデスクの 3D Studio ソフトウェアで使用される 3D Studio (DOS) メッシュ ファイル形式を示します。 Autodesk 3D Studio は、1990 年代以来 3D ファイル形式の分野で重要な存在感を保っており、その後 3D Studio MAX に進化しました。 [STL](/ja/cad/stl/) はステレオリソグラフィーの略で、3 次元表面形状を描画するための多用途のファイル形式として機能します。

[3DS](/ja/3d/3ds/) を STL ファイル形式に変換するのは一般的な要件であり、3DS を STL に無料で変換できるいくつかのアプリがオンラインで入手できます。 3DS から STL への機能を .NET、Java、または Python アプリケーションに埋め込むことに興味のあるアプリケーション開発者は、この目的のために Aspose.3D API の使用を検討できます。

[![3DS から STL](../3ds-to-stl.png)](https://products.aspose.app/3d/conversion/3ds-to-stl)

### 3DS を STL に変換する手順

3DS ファイルを STL に変換するために従う必要がある手順は次のとおりです。

1. [Aspose.3D 変換アプリ](https://products.aspose.app/3d/conversion/3ds-to-stl) に移動します。
1. 3DS ファイルをブラウザにアップロードします
1. 出力ファイル形式として STL を選択します。
1.「変換」ボタンを押します。

変換プロセスが完了すると、変換された STL ファイルをディスクに保存するかどうかを選択できます。
