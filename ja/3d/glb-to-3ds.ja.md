## GLBから3DSについて

3D ファイル形式で GLB ファイルを使用することは新しいことではありません。 GLB は、GL Transmission 形式 ([glTF](/ja/3d/gltf/)) で 3D モデルを表す、一般的に使用される 3D バイナリ ファイル形式です。一方、[3DS](/ja/3d/3ds/) は、Autodesk 3D Studio で使用される 3D Sudio (DOS) メッシュ ファイル形式を表します。 [GLB](/ja/3d/glb/) を [3DS](/ja/3d/3ds/) に無料で変換できるいくつかのアプリがオンラインで入手できます。 .NET または Java で GLB の機能を 3DS に追加したいアプリケーション開発者は、Aspose.3D API を使用してこれを実現できます。

Aspose.3D変換アプリを無料で使用して、GLBを3DSに変換できます。次のボタンをクリックして、今すぐ GLB ファイルを 3DS に変換します。

[![GLB から 3DS](../glb-to-3ds.png)](https://products.aspose.app/3d/conversion/glb-to-3ds)

### GLB を 3DS に変換する手順

GLB ファイルを 3DS に変換するために従う必要がある手順は次のとおりです。

1. [Aspose.3D変換アプリ](https://products.aspose.app/3d/conversion/glb-to-3ds)にアクセスします。
1. GLB ファイルをブラウザにアップロードします
1. 出力画像ファイル形式として 3DS を選択します
1. 変換ボタンを押します。

これにより、GLB ファイルを 3DS に変換するプロセスが開始され、ファイルを PC に保存するためのダウンロード オプションが表示されます。
