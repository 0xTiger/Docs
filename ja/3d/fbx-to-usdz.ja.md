## FBXからUSDZへの変換について

FBX (FilmBox の略) は、広く受け入れられている 3D ファイル形式で、当初は MotionBuilder 用に Kaydara によって作成されました。 Autodesk Inc. は 2006 年に所有権を引き継ぎ、多数の 3D ソフトウェア ツールで利用される主要な 3D データ交換フォーマットとしての地位を確立しました。

.usdz 拡張子を持つファイルは、[USD](/ja/3d/usd/) (Universal Scene description) ファイル形式に合わせた非圧縮および非暗号化の ZIP アーカイブを表します。テクスチャやアニメーションなど、さまざまなファイル タイプのプロキシが格納されており、すべてアーカイブ内にカプセル化されています。

[FBX](/ja/3d/fbx/) の USDZ 機能を .NET、Java、または Python アプリケーションに組み込むことを目指しているアプリケーション開発者は、次のことを達成するための貴重なリソースとして Aspose.3D API を検討するとよいでしょう。この目標。

### FBX を USDZ に変換する手順

FBX ファイルを USDZ に変換するために従う必要がある手順は次のとおりです。

1. [Aspose.3D 変換アプリ](https://products.aspose.app/3d/conversion/fbx-to-usdz) にアクセスします。
1. FBX ファイルをブラウザにアップロードします
1. 出力ファイル形式として USDZ を選択します
1.「変換」ボタンを押します。

変換プロセスが完了すると、変換された USDZ ファイルをディスクに保存するかどうかを選択できます。
