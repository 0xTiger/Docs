## .AN オプション番号

拡張子が .an のファイルは、アニメーションに関する情報を含む [Adobe Edge Animate](https://www.adobe.com/products/animate.html) プロジェクト ファイルです。プロジェクトを保存すると、Adobe Edge Animate によって生成された関連する [.css](/web/css/) および [.js](/web/js/) ファイルへの参照を含む .html ファイルが作成されます。 AN ファイルは、プロジェクト ファイルを開くために使用できるこれらのファイルとともに保存されます。 Adobe Edge Animate は、ブラウザーでサポートされなくなった Adobe Flash を使用してアニメーション化された Web ページを作成する代わりに使用できます。

## ANファイル形式

ANファイルはプレーンテキストファイル形式でディスクに保存され、テキストエディターで開いて内容を表示できます。
