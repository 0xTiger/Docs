## .STC オプション番号

拡張子が .stc のファイルは、Adobe Contribute ソフトウェアが接続情報を保存するために使用する接続ファイルです。これには、ユーザー名、ホスト URL、FTP、SFTP などのプロトコルなどの接続情報が含まれており、既存の Web サイト内から再利用できます。ご存じない方のために説明すると、Adobe Contribute は、ブログと呼ばれるコンテンツベースの記事を含む既存の Web サイトに更新をプッシュするために使用される特殊な [HTML](/web/html/) エディターでした。これには、Internet Explorer や Firefox などの一般的な Web ブラウザー用のプラグインが含まれており、ページ内からコンテンツを Web サイトにシームレスに提供しました。 2017 年 1 月に廃止され、現在は Adobe Dreamweaver を使用してゼロから Web サイトを作成しています。

## STC ファイル形式

STC ファイルの内部データ形式は不明です。ただし、接続キーはパスワードで暗号化され、ファイルで送信されるネットワークまたは FTP ログイン接続情報は安全に保たれ、Contribute を介してのみアクセスできます。キー接続ファイルは、ユーザーに電子メールで送信したり、コンピューターに保存したりできます。クリックすると、接続をインポートする Contribute が自動的に開きます。
