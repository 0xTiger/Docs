## .ECE オプション番号

ECE ファイルは、コンテンツ管理プラットフォーム **Escenic Content Engine (ECE)** によって動的に生成される Web ページです。これには、Web サーバーによって処理され、[HTML](/web/html/) ページとしてユーザーのブラウザーに返されるサーバー側のコードが含まれています。 ECE ファイル形式は、ASP や PHP などの他のサーバー側ファイル形式と比較して、あまり一般的ではありません。これらのファイルは内部的にレンダリングされ、表示目的でエンド ユーザーに提供されます。

## ECE ファイル形式

ECE ファイルはディスクに保存されませんが、要求に応じてユーザーに直接提供されます。これらは HTML に変換され、表示のためにユーザーのブラウザに送り返されます。ニュース記事ページの公開には、Escenic Content Engine が積極的に使用されています。 JEE および SQL テクノロジに基づいて、ECE サーバーは、すべてのオンライン コンテンツに対して信頼性が高くスケーラブルなリポジトリを提供します。

クライアント ブラウザは、HTTP/REST API を介して ECE サーバー上のコンテンツにアクセスできます。 [HTML](/web/html/) ページを提供することに加えて、ECE はビデオ/オーディオ コンテンツを管理および公開するための非常に効率的なワークフローを提供します。
