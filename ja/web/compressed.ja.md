## 圧縮ファイルとは何ですか?

COMPRESSED ファイルは、プラグイン **WordCompress** を使用して生成された、小さいサイズの Wordpress 圧縮 Web ページです。これには、プラグインによって HTML、CSS、および JavaScript コードを圧縮することによって生成される Web ページを表す 1 行のコードが続きます。これらの変換された COMPRESSED ファイルはサイズが小さいため、インターネット経由での読み込みと転送が高速化されます。

### 圧縮ファイル形式 - 詳細情報

COMPRESSED ファイルは、Microsoft Notepad、Notepad++、Apple TextEdit などのテキスト エディタで開くことができるテキスト ファイルとして保存されます。 COMPRESSED ファイルは、テキスト エディタに加えて、Windows と Mac OS の両方で利用できる Adobe Dreamweaver などの Web 構築ソフトウェアで開くことができます。

## WordCompress プラグインについて

WordCompress プラグインは、当初、WordPress Web ページを圧縮ファイル形式に圧縮するために使用されました。このプラグインはサポートされなくなり、廃止されました。ページのソース コードからすべての空白を削除して、Web ページのソース コードを単一の実行可能ファイルに書き込みます。 COMPRESSED ファイルは、Web ページの元のソース コードと一緒に保存され、ページの元のソース コードに含まれる元のコメント、インデント、その他の空白が保存されます。
