## .EDGE ファイルとは何ですか?

拡張子が .edge のファイルは、Adobe Edge でプロジェクトを作成して保存したときに作成される Adobe Edge Project ファイルです。 Adobe Edge を使用すると、Web デザイナーは、Web、デジタル パブリッシング、リッチ メディア広告用のインタラクティブな [HTML](/web/html/) アニメーターを、デスクトップ ブラウザーとモバイル ブラウザーの両方で作成できます。これは、HTML/HTML5、[JavaScript](/web/js/)、および [CSS3](/web/css/) を使用して実現されます。 Edge ファイルはプレーン テキスト ファイルとして生成され、Microsoft Notepad、Apple TextEdit、Notepad++ などのさまざまなアプリケーションで開くことができます。

## EDGE ファイル形式

EDGE ファイルはプレーン テキスト形式で保存され、jQuery を使用してアニメーションを有効にします。 EDGE アニメーションの代替は、サポートされなくなった Flash ファイルでした。 EDGE ファイルは、HTML、JavaScript、および CSS ファイルと一緒に保存されます。 EDGE ファイルを開くと、Web ページ、つまり HTML ファイルが開きます。
