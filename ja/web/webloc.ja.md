## WEBLOCとは？

WEBLOC ファイルは、インターネット上の Web URL を指す macOS 上のショートカット ファイルです。 Web URL をブラウザーのアドレス バーからデスクトップまたは保存先の他のフォルダーにドラッグすることによって生成されます。リンク先サイトへの URL はこのショートカット ファイル内に保存され、Microsoft Notepad、Notepad++、または Apple TextEdit などのテキスト エディタでファイルを開くことで表示できます。 WEBLOC ファイルは [URL](/web/url/) ファイルに似ています。

## WEBLOC ファイル形式 - 詳細情報

WEBLOC ファイルは、プレーン テキスト ファイル形式で保存されます。これらは、Safari、Google Chrome、または Firefox などの Web ブラウザーから URL をドラッグすることにより、macOS で生成できます。複数の WEBLOC ファイルを同時に開く場合は、外部アプリケーションまたは API を介してこれらを開くためのより高速で簡単なソリューションが必要になります。これらの API は、WEBLOC を読み取り、PDF および JPEG に変換できます。
