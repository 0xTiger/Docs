## URL ファイルとは何ですか?

拡張子が .url のファイルは、任意の Web サイト アドレスでコンピューターに保存できる Web サイト URL ショートカットです。ダブルクリックして開くと、URL はデフォルトのブラウザーで Web サイトを開き、ページがユーザーに表示されます。 URL ファイルは通常、Web サイトへのショートカット ファイルと呼ばれ、既定のブラウザーのアイコンで表されます。 URL ファイルは、Microsoft Notepad、Notepad++、Apple TextEdit などの任意のテキスト エディタで開いて編集できます。

## URL ファイル形式 - 詳細情報

URL ファイルはプレーン テキスト形式で保存され、任意のテキスト エディターで開いて表示できます。これらは、ユーザーが他のブラウジングを実行するデフォルトの Web ブラウザーで開きます。
