## .GNE オプション番号

GNE ファイルは、Flickr の写真共有 Web サイトによって動的に生成および使用される Web ページです。ページはサーバー上でホストされ、そこから要求元のユーザーのブラウザーに提供されます。 GNE ファイルは PHP および ASP ページに似ていますが、独自のプログラミング言語で開発されており、その詳細は開発者が参照することはできません。

## GNE ファイル形式 - 詳細情報

GNE ファイルは、[Ludicorp](https://en.wikipedia.org/wiki/Ludicorp) によって開発されたコンピュータ ゲーム、ゲーム ネバー エンディングで使用されました。 Flicker は後に yahoo! に買収されました。 2005 年 3 月。
