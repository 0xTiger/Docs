## .XHTM ファイルとは何ですか?

拡張子が .xhtm のファイルは、XHTML で記述された Web ページ ファイルです。これは、標準実装として XML 1.0 を使用し、HTML 4 の再定式化であると考えられています。XHTML と同様、XHTM ファイルには、より厳格な XML ガイドラインに従う標準 HTML タグが含まれています。 XHTM ファイルは Web サイトのコンテンツを公開するために使用され、一般的に [XHTML](/ja/web/xhtml/) 拡張子を使用します。

Google Chrome、Apple Safari、Mozilla Firefox などの Web ブラウザを使用して XHTM ファイルを開くことができます。

## XHTM の例

次の XHTM ドキュメントの例では、XML 宣言を定義しています。

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
