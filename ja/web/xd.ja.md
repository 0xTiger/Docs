## .XD オプション番号

拡張子が .xd のファイルは、Adobe XD ソフトウェアを使用して Web サイトまたはモバイル アプリケーション用に作成されたモックアップです。デザイナー、クリエイティブ チーム、および組織は、これらのファイルを使用して、これらを参照してアプリケーションを設計します。単一の XD ファイルには、画像、形状、テキストなどのデザイン要素が含まれており、開発者がこれらをアプリケーションとして開発し始めるための高速で直感的なワイヤーフレーム プロトタイプを開発するための完全な機能セットを提供します。

## XD ファイル形式

XD ファイルには、1 つまたは複数のアートボードとしてレイヤーにグラフィック要素 (画像、図形、およびテキスト) が含まれています。これらは、ガイドとレイヤーとともに、ワイヤーフレームのビルディング ブロックを提供します。

* 「アートボード」 - アプリケーションの画面や Web サイトのページを表し、デザイン プロセスを合理化します。アートボードは、1 つのドキュメントで複数の画面サイズのインタラクティブなデザインを作成します。
* `ガイド` - オブジェクトを並べたり、アートボード上の要素間の間隔を管理したりするために使用されます。ガイドは、フレームワークとして機能するグリッドを使用して、デザイン要素 (画像、グリフ、段落) を整理し、レイアウトの対称性を確保します。
* 「レイヤー」 - UX デザイナーは、Adoble Illustrator や Photoshop でレイヤーを使用するのと同じように、XD でレイヤーを使用します。これらは、混乱を避けるためにアートボード上の作業を整理するために使用され、レイヤーパネルから制御されます。
