## .FWP オプション番号

.fwp ファイルを含むファイルは、Microsoft Expression Web で作成されたアーカイブ ファイルで、デザインのすべてのファイルとフォルダーが含まれています。 Microsoft Expression Web は、Microsoft の一般的な Web デザイン ソフトウェア製品で、[HTML](/web/html/) エディターに基づいています。 2012 年に廃止され、Microsoft から無料でダウンロードできます。 FWP ファイルは、Microsoft の .cab 形式でエクスポートおよび保存できます。

## FWP ファイル形式 - 詳細情報

FWP ファイルの正確なファイル形式仕様は、開発者の参考にはなりません。ただし、これらはバイナリ ファイルであり、ずっと前に廃止された Microsoft Expression Studio で開くことができます。
