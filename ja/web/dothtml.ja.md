## .DOTHTML ファイルとは?

DOTHTML ファイルは、HTML バージョンで保存された Microsoft Word テンプレート ドキュメントです。 Web ブラウザーで開くことができ、コンテンツは他の Web ページと同じように表示されます。 DOTHTML ファイルは、これらのファイルを生成するための事前にフォーマットされた設定を含む Microsoft Word .DOT ファイルから生成されます。 [DOT](/word-processing/dot/) ファイルが DOCHTML に変換されると、これらの設定はすべて保持され、変換された DOTHTML ファイルは元の DOT ファイルと同様のフォーマットで表示されます。このようにして、公開されたテンプレートを Web ブラウザーで表示できます。

## DOTHTML ファイル形式 - 詳細情報

DOTHTML ファイルは [HTML](/web/html/) ファイルとして保存され、任意の HTML エディターで開くことができる HTML ソース コードが含まれています。最初の数日間、Microsoft Word は [DOCHTML](/web/dochtml/)、[DOCMHTML](/web/docmhtml/) などのすべてのドキュメント形式の HTML バージョンを提供していましたが、後にこれらの HTML ベースのバージョンは廃止されました。 .その結果、DOTHMLT もサポートされなくなりました。
