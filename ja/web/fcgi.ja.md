## .FCGI オプション番号

FCGI ファイルは、FastCGI (FCGI) 用に記述されたソース コードを含むソース コード ファイルです。いくつかの異なるプログラミング言語で記述でき、Web サーバー要求のより堅牢な処理に使用される CGI 標準の拡張が含まれています。 FCGI で最も一般的にサポートされている言語には、Perl コードと FastCGI API の強力な外部インターフェイスの両方をサポートする **Perl** が含まれます。

## FCGI ファイル形式

FCGI ファイルが重要な理由FCGI インターフェイスは、分散コンピューティングをサポートして、Web リクエストを処理し、それらを複数のサーバーに分散します。これにより、FCGI ファイルは主に複数のサーバー間の負荷分散に使用されます。
