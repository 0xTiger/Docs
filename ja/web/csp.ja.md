## .CSP オプション番号

CSP ファイルは、RadGs Concept Application Server で実行するために記述されたコンテンツを含む動的な Web ページ ファイルです。サーバー側でコンパイルおよび実行された後、HTTP プロトコルを介してリクエスト元のユーザーのブラウザーに提供されます。 CON ファイル形式とは異なり、CSP ファイルは、これらのファイルを表示するためにユーザーが Concept クライアントをインストールする必要はありません。 [PHP](/programming/php/) や [ASP](/web/asp/) などの他のサーバー サイド ページと同様に、CSP ファイルを表示できます。
