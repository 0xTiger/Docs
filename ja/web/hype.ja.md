## .HYPE オプション番号

HYPE ファイルは、HTML5 を使用した Web コンテンツの作成に使用される Hype ソフトウェアを使用して保存されたプロジェクト ファイルです。グラフィック、テキスト、アニメーション、シーン情報、およびコンテンツを構成するその他の要素などの Web 要素が含まれています。 HYPE ファイルの Web コンテンツは、Web ページで公開できる Web ファイルにエクスポートできます。

HYPE ファイルは [Tumult Hype](https://tumult.com/hype/) ソフトウェアで開くことができます。

## HYPE ファイル形式

HYPE ファイルは独自のファイル形式で保存されており、これらの内部の詳細は公開されていません。 HYPE ファイル内のコンテンツは、Tumult Hype ソフトウェアで開いた場合にのみ表示できます。

### HYPE ファイルの内容

HYPE アプリケーションを使用してさまざまな目的で Web コンテンツを作成すると、HYPE ファイルが作成されます。 eCard、eBook、プレゼンテーション、広告、インフォグラフィック、Web ページ素材などのコンテンツを作成するためのキーフレーム ベースのアニメーションに関する情報が保存されます。これらのコンテンツは、デスクトップから iPhone や iPad までのデバイスにコンテンツを表示できるようにする HTML5 を利用しています。 Tumult Hype は、最先端の HTML5、[CSS3](/web/css/) スタイル、および [JavaScript](/web/js/) を出力します。
