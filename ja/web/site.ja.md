## SITEファイルとは何ですか?

SITEファイルは、Webサイト制作ソフト「Adobe GoLive」で使用されるプロジェクトファイルです。ファイル、フォルダー、画像、カスタム スタイル シート、スクリプトなどの Web サイト プロジェクト情報が保存されます。また、Web サイトの構造と、サイトの Web ページで使用されるリンクも含まれます。これはプロジェクト ファイルであるため、プロジェクト ファイルへの参照が含まれていますが、実際のファイル自体は含まれていません。

Adobe GoLive は廃止され、サポートされなくなりました。

## サイトファイル形式

SITE ファイルは Adobe 独自のファイル形式でディスクに保存され、そのファイル形式の内部詳細は開発者が参照できるように公開されていません。
