## CODASITEファイルとは何ですか?

CODASITE ファイルは、[Coda](https://coda.io/) Web ページ作成および公開ソフトウェアで作成された Web サイト プロジェクト ファイルです。プロジェクトの一部であるすべてのファイルのリストが含まれています。これには、[HTML](/web/html/) ファイル、カスケード スタイルシート [CSS](/web/css/) ファイル、Javascript [JS](/web/js/) ファイル、およびその他の関連するサポート ファイルとしての Web ページが含まれます。 CODASITE プロジェクト ファイルには、Web サーバーに Web サイトを公開するための情報も含まれています。

## CODASITE ファイル形式

CODASITE プロジェクト ファイルは、~/ Library/ Application Support/ Coda 2/ Sites/ ディレクトリ内のバイナリ ファイルとしてディスクに保存されます。 Coda ユーザー インターフェイスは Coda サイトを管理するためのインターフェイスを提供します。これは、これらを手動で開くと不適切なプロジェクト設定につながり、問題が発生する可能性があるためです。
