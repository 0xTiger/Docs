## .JHTML オプション番号

拡張子が .jhtml のファイルは、[Java](/programming/java/) コードを含む [HTML](/web/html/) ファイルであり、クライアントがブラウザーでこのページを要求したときにサーバーで実行されます。サーバーはリクエストを処理し、関数に含まれる Java 関数を実行して、クライアントのブラウザにページを返します。 JHTML ページに埋め込まれた Java オブジェクトは、サーバー上で実行され、このタイプのページの要求を処理します。 JHTML ファイルは、JDBC (Java [データベース](/database/) 接続) 接続を使用してデータベースから情報にアクセスすることもできます。 JHTML ファイルは、任意のテキスト エディターで開くことができ、Google Chrome、Firefox、Safari などの Web ブラウザーで表示できます。

## JHTML ファイル形式

JHTML は ATG の独自技術であり、ATG (Art Technology Group) Dynamo Document Editor を使用して作成できます。 JHTML ファイルは、標準の HTML および Java コードを使用してプレーン テキスト ファイル形式で記述されます。これらには、Java オブジェクトを参照する独自のタグに加えて、標準の HTML タグが含まれています。このようなページがユーザーによって要求されると、HTTP サーバーはその要求を Java アプリケーション サーバーに転送します。 JHTML ページは最初に .java ファイルに変換され、コンパイルされて、サーブレットとして実行される [.class](/programming/class/) ファイルが生成されます。これにより、標準の HTTP および HTML データのストリームが生成され、ユーザーに表示するために要求元のブラウザーに返されます。これは、サーバー上でデータベース関連のクエリを実行し、最終的な累積結果をクライアントのブラウザに返すのに役立ちます。
