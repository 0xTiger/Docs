## .EWP オプション番号

EWP ファイルは、[EWISoft](http://www.ewisoft.com/) Web サイト ビルダー ソフトウェアで作成されたプロジェクト ファイルです。これは、プロジェクトのすべての情報を含むバイナリ データベース ファイルにリンクされています。 EWP ファイルは、Web 開発の進行状況を保存するために一般的に使用されます。これは、ユーザーが Web サイト テンプレートから Web サイトを作成できる、テンプレート駆動型の Web デザイン ソフトウェアです。

## EWP ファイル形式 - 詳細情報

EWP ファイルは、コンパイルやサーバーでのホストが不要なプロジェクト ファイルです。これらはプレーン テキスト ファイルとして保存され、Web 開発の進行状況への参照が含まれています。開発が完了すると、プロジェクトを Web サイトに公開できます。
