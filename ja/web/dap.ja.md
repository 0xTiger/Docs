## .DAP ファイルとは?

拡張子が .dap のファイルは、Microsoft Access 内から作成され、データベースに保存されているデータにアクセスできる Microsoft Access データ ページ ファイルです。このような Web ファイルに格納されたデータは、ネットワーク (イントラネットおよびインターネット) 経由でアクセスできます。これは、特にデスクトップ アプリケーション用に開発された Access の [MDB](/database/mdb/) ファイルとは異なります。このようなデータベースは、Access 2010 および Access Services (SharePoint のオプション コンポーネント) を使用して作成できます。これらのデータベースは、SharePoint アカウントを持つユーザーが Web ブラウザーで Web アドレスを使用して使用できます。
