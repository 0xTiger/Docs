## .TS ファイルとは何ですか?

TS ファイルは、Digia の Qt SDK にバンドルされている lupdate ソフトウェア ユーティリティで生成された変換ファイルです。 SDK は、Windows、Mac、および Linux オペレーティング システムで利用できます。 TS ファイルには、ある言語の語句と別の言語への翻訳が含まれています。 TS ファイル内に保存されている翻訳は、特定のアプリケーションに固有のものです。 TS ファイルは、Digia Qt SDK で開くことができます。

## TS ファイル形式

Digia Qt TS ファイルは XML ファイル形式でディスクに保存され、Notepad や Notepad++ などの任意のテキスト エディタで開くことができます。 Digia Qt SDK でサポートされているその他のファイル形式には、次のものがあります。

* .プリ
* .qm
* .qml
* .qph
* .qrc
* .qs
* .qss
* .ts
* .ui ユーザー インターフェース ファイル
