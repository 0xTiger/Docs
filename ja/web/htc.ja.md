## .HTC オプション番号

HTC ファイルは、Microsoft Internet Explorer で使用される Web ファイル形式のファイルです。これには、Web ブラウザー内でさまざまなタスクを実行するための小さな機能を実行する HTML コンポーネントの定義が含まれています。 HTC ファイル内で定義された機能は再利用可能で、複数の [HTML](/web/html/) ページからアクセスできます。

## HTC ファイル形式 - 詳細情報

HTC ファイルは、プレーン テキスト ファイル形式で保存されます。これは、これらを任意のテキスト エディターで開いて、機能を変更できることを意味します。これらは通常 Internet Explorer で開かれますが、この方法ではこれらのファイルを表示することしかできず、編集することはできません。

HTC ファイルは実際には、コンポーネントを定義するためのスクリプトと一連の HTC 固有の要素を組み合わせた HTML ファイルです。
