## .XHT オプション番号

拡張子が .xht のファイルは、[XHTML](/web/xhtml/) (拡張ハイパーテキスト マークアップ言語) ファイル タイプに関連付けられた Web ファイルであり、それ自体が [XML](/web/xml/) ベースのマークアップ ファイルです。これらのファイルはどちらも HTML に似ていますが、より厳密な XML に似た構文を持っています。 XHT ファイルは、より構造化されるように設計されており、スクリプト作成が少なく、XML の既存の機能を使用することに加えて汎用的です。

## XHT ファイル形式

XHT ファイルは、デフォルトで UTF-8 エンコーディングのプレーン テキスト ファイルとして作成および保存されます。これには、Unicode エンコーディングをサポートする任意のテキスト エディターで開いて編集できる XHTML ドキュメントの完全なソース コードが含まれています。テキスト エディターに加えて、XHT ファイル形式は最新の Web ブラウザーのほとんどで理解でき、これらを使用して開くことができます。

## XHT の例

次の XHT ドキュメントの例では、XML 宣言を定義しています。

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
