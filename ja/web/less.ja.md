## LESSファイルとは何ですか?

LESS ファイルは Web ページに使用されるスタイル シートで、Leaner Style Sheets (LESS) で作成されます。標準の CSS 言語に追加機能を実装しているため、学習と実装が容易になります。 LESS ファイルは、[CSS](/web/css/) を拡張し、変数、ミックスイン、ネスティング、操作、関数、名前空間、アクセサーなどの機能を追加します。

## 無駄のないスタイル シート ファイル形式

LESS ファイルは CSS の形式で作成され、提供される追加機能によって機能が拡張されます。これらはテキスト ファイルとして保存され、LESS スタイル シート言語、CodeKit、および Adobe Dreamweaver で開くことができます。
