## .PRO ファイルとは何ですか? ##

PROファイルは、廃止されたAdobe TouchアプリであるAdobe Protoによって作成されたWebサイトまたはモバイルアプリケーションのモックアップです。これには、Web ページまたはモバイル アプリ用の 1 つ以上の UI レイアウトが含まれ、それぞれにインタラクティブな要素が含まれる場合があります。 PRO ファイルでは、CSS、WebKit、および jQuery テクノロジがサポートされています。

### PRO ファイル形式の簡単な歴史

PRO ファイルは、Adobe XD に置き換えられた Adobe Proto によって作成されました。タブレット用のプロトタイピング アプリの作成に使用されました。 Adobe Dreamweaver などの一般的な Web デザイン アプリケーションは、これらの PRO ファイルを使用して Web およびモバイル アプリケーションを作成しました。

## PRO ファイル形式 - 詳細情報

PRO ファイルは、Web サイトまたはモバイル アプリケーションの開発における最初のステップです。開発者はまず、アプリケーションの外観を定義するデジタル モックアップを作成します。これらは、アプリケーション画面全体のアプリケーション ワークフローとナビゲーションも定義します。これらのモックアップには、ボタン、コンボボックス、スクロールバー、入力フィールドなど、多数のナビゲーション要素も含まれています。
