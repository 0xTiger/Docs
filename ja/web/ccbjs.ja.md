## .CCBJS オプション番号は何ですか?

CCBJS ファイルは、3D アプリ、ゲーム、Web サイト エディターである **CopperCube** によって作成された Javascript ファイルです。これには、.ccb ファイルから作成された [.html ファイル](/ja/web/html/) の一部として生成される JavaScript コードが含まれています。 CopperCube は、コードを書かずにゲームプレイ ロジックとインタラクションを開発するためのユーザーフレンドリーなスクリプト インターフェイスを提供します。 CopperCube でサポートされるその他のファイル形式には、.ccscene、.ccmodel、.ccmaterial、.cctex、.ccjs、.ccp、.ccpak などがあります。

## CCBJS ファイル形式

CCBJS ファイル形式の内部ファイル形式の詳細は、それ自体は不明です。ただし、JavaScript ファイルはプレーン テキスト ファイルとしてディスクに保存され、Microsoft Notepad、Notepad++、Apple TextEdit などのテキスト エディタで開くことができます。
