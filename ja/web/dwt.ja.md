## .DWT オプション番号

拡張子が .dwt のファイルは、Dreamweaver ソフトウェアで作成された Web ページ テンプレート ファイルです。テンプレートですでに定義されている基本設定を持つ複数のページを開発するための出発点として使用されます。新しい [HTML](/web/html/) ページは、これらのテンプレートのコピーを使用して構築でき、必要に応じてコンテンツを変更できますが、共通セクションは変更されません。ヘッダー、フッター、メニューなど、これらのテンプレートの編集不可の領域は、すべての Web ページで同じままです。

## DWT ファイル形式 - 詳細情報

DWT ファイルは、プレーン テキスト ファイルとして作成および保存されます。これらは、Dreamweaver または任意のテキスト エディターで開くことができ、必要に応じて編集することができます。 Dreamweaver ソフトウェアは、変更が適用されると UI を更新し、ユーザーはアプリケーションのユーザー インターフェイスで変更の影響を視覚的に確認できます。

[HTML] ページ (/ web / html /) をテンプレート ファイルとして保存すると、Dreamweaver によって、ローカル ルート フォルダーのルート レベルにテンプレート フォルダーが作成されます。 .dwt ファイルを Templates フォルダから移動したり、名前を変更したりすると、リンクが壊れます。したがって、.dwt ファイルは、Dreamweaver によって作成される Templates フォルダーに保持することが重要です。画像ソース ファイルや HTML ドキュメントなど、サイトの他の要素は Templates フォルダー以外の場所に保管することをお勧めします。

## ドリームウィーバーについて

Dreamweaver ソフトウェアは、もともと Macromedia によって開発されました。マクロメディアの他の人気のあるソフトウェアには、ブラウザー用のアニメーション プレーヤーとして長い間利用可能な Flash プレーヤーが含まれていました。 Adobe が 2005 年に Dreamweaver を買収した後、Dreamweaver は Adobe ソフトウェア アプリケーションの不可欠な部分となり、現在は Adobe によって個別に配布されています。
