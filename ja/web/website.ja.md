## WEBSITE ファイルとは?

WEBSITE ファイルは、Microsoft Internet Explorer (IE) で作成された Web サイトのショートカット ファイルです。これには、URL と、Web サイトのファビコンを指すアイコン (ICO) ファイルが含まれています。アイコンは、ショートカットを表示するために使用されます。 WEBSITE ファイル形式は、IE バージョン 9 のリリースで導入されました。以前のバージョンの IE では、Web ページへのショートカットを [.url](/web/url/) ファイルとして保存していました。

## ウェブサイトのファイル形式

.website ファイルはプレーン テキスト ファイルとしてディスクに保存され、その内容は任意のテキスト エディターで表示できます。他のコンテンツに加えて、ショートカットであるターゲット URL が含まれています。 IE 以外のブラウザで対象の Web サイトを開きたい場合は、この URL をコピーして、Google Chrome、Safari、Firefox などの他のブラウザに貼り付けて Web サイトを表示できます。

## .WEBSITE ファイルの作成方法

WEBSITE ファイルは、アドレス バーの URL をクリックし、デスクトップやドキュメントなどのコンピューター上の目的の場所にドラッグすることで、IE で作成できます。これにより、そのインターネットの場所へのショートカットが作成され、ショートカットをダブルクリックして開くことができます。ショートカットを「URL」ファイルとして保存する場合は、SHIFT キーを押したまま、ターゲット サイトのファビコンをデスクトップにドラッグします。
