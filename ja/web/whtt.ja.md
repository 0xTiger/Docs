## .WHTT ファイルとは何ですか?

WHTT ファイルは、[HTTrack Website Copier](https://www.httrack.com/) ソフトウェアを使用してダウンロードされる Web サイトのオフライン バージョンです。 [HTML](/web/html/) ウェブページ、画像、スタイルシート、リンク、ディレクトリなど、ウェブサイトのすべてのコンテンツが含まれています。 Web サイトのすべてのページが再帰的にダウンロードされ、オフライン ミラーとしてディスクに保存されます。 Web サイトが HTTrack ソフトウェアによってダウンロードされると、インターネットに接続しなくてもオフラインで表示できます。ダウンロードした Web サイトを表示するには、ミラーリングされた Web サイトのページをブラウザーで開くだけで、各画像/リンクが正しく読み込まれます。

## WHTT ファイル形式

WHTT ファイルは、THML Web ページ、画像、スタイルシート、参照リンクの組み合わせとして保存されます。
