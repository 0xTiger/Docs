## .DOCHTML ファイルとは?

DOCHTML ファイルは、Microsoft Word で生成されたドキュメントの HTML 形式です。 Web サーバーへの公開に使用できる、作成されたドキュメントの [HTML](/web/html/) バージョンです。 DOCHTML ファイルを Web ブラウザーで開いて、変換された Word 文書の内容を表示できます。 DOCHTML ファイルの外観と外観は、元の Word 文書と同じです。

## DOCHTML ファイル形式

DOCHTML ファイルは、Web サーバーで公開できる HTML 形式でディスクに保存されます。 Microsoft Word は、ドキュメントを DOCHTML ファイルとして保存するためのサポートを停止しました。これは、現在一般的に使用されている [.mht](/web/mht/) および [.mhtml](/web/mhtml/) ファイル形式に置き換えられました。
