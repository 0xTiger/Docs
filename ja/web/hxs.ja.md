## .HXS オプション番号

HXS ファイルは、Microsoft Help 2.x ファイル用にコンパイルおよび使用されるヘルプ ファイルです。これは、一連のトピック ページからコンパイルおよび生成され、以前の [CHM](/web/chm/) ヘルプ ファイル形式に似た [HTML](/web/html/) のサブセットで記述されます。 HXS ファイルは圧縮形式で保存され、.HxC メイン プロジェクト ファイル、.HxF インクルード ファイル、.HxT 目次、.HxA 属性定義ファイル、およびいくつかの .HxK インデックスから作成されます。 HXS ファイルのメタデータは [XML](/web/xml/) から作成されます。

HXS ファイルは Web ブラウザで開くことができます。

## HXS ファイル形式

HXS ファイルは、HTML コンテンツと XML メタデータから圧縮ファイルとして作成および保存されます。
