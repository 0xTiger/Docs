## .CPHDファイルとは何ですか?

CPHD ファイルは、CloudChan プリプロセッサ エンジンによって使用されるハイパーテキスト ドキュメントです。これには、動的な Web ページやドキュメントを作成するための HTML とスクリプトが含まれています。 CPHD ファイル内のスクリプトは、Java で使用される構文に似ています。 CPHD Web ページは、CloudChan プロセスで実行されると HTML ドキュメントを返します。 CPHD 形式は、あらゆる XML ドキュメントで使用できます。

## CPHD ファイル形式

CPHD ファイルはプレーン テキスト ファイルとしてディスクに保存されます。これにより、これらのファイルを Notepad や Notepad++ などのテキスト エディタで開くことができるようになります。
