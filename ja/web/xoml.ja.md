## .XOML ファイルとは?

XOML は Extensible Object Markup Language の頭字語であり、Windows Workflow Foundation のワークフロー オブジェクトのシリアル化形式です。 **[XAML](/web/xaml/)** に基づいており、プレーンな **[XML](/web/xml/)** でユーザー インターフェイスを作成するために主に使用されます。これらは、ユーザー インターフェイスのワークフローを宣言するために使用され、実装ロジックを含む別のファイルと共にコンパイルされます。これには、ルート ワークフロー ノード、ネストされたサブ要素、コードの埋め込みセグメントを定義するツリー ベースの構造が含まれています。 Visual Studio for .NET を使用して新しいワークフローを作成する場合、コード形式にするかコード分離形式にするかを選択するオプションがあります。後者を選択した場合、IDE は 2 つのファイルを作成します。 1 つは XOML 形式 (ワークフローのレイアウトと設定で構成される) で、もう 1 つはプログラミング コードが存在する [.CS](/programming/cs/)/[.VB](/programming/vb/) 形式です。
