Web ページを作成するときは、さまざまな種類のファイルが必要です。 Web ファイル形式は、Web ページの開発の標準を定義し、これらが構築されるプラットフォームに関連しています。静的および動的 Web ページで構成される完全な Web サイトを構築できます。最新の Web サイトのほとんどは、Web サーバーに読み込まれて実行される Active Server Pages (ASP) などのサーバー側のテクノロジに基づいて構築されています。これらには、カスケード スタイリング シート (CSS) と、UI の全体的なルック アンド フィールのスタイリングに使用されるスクリプト ファイルも含まれます。

一般的な Web ファイル拡張子とそれに関連するファイル形式には、**[HTML](/web/html/)** (Hypertext Markup Language)、**[ASP](/web/asp/) (Active Server Pages)**、 **CSS** (カスケーディング スタイル シート)。
