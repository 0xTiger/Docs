## OBML16 ファイルとは?

OBML16 ファイルは、Opera Mini モバイル Web ブラウザーを使用して保存された Web ページです。これは Opera Binary Markup Language ([OBML](/web/obml/)) に基づいており、保存された Web ページにすばやくアクセスできるコンパクトな形式です。 [OBML15](/web/obml15/) ファイル形式は、Opera Mini バージョン 7 の起動時に置き換えられました。これらの Opera Mini Web ページのバリアントはすべて基本 OBML ファイル形式からアップグレードされており、Opera Mini の各アップグレードは 1 つの OBML 形式のみと互換性があります。 OBML16 ファイルは、オンラインで [PDF](/pdf/) および [HTML](/web/html/) に変換できます。

## OBML16 ファイル形式 - 詳細情報

OBML16 は、Web ページを保存するためのコンパクトな形式である OBML ファイル形式に基づいています。このコンパクトな形式を使用する利点は、Web コンテンツのサイズが小さくなり、ダウンロード速度が向上することです。ダウンロードが完了すると、Opera Mini ユーザーはインターネット接続にアクセスする必要なく、これらのページを表示できます。
