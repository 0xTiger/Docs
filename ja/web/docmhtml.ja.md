## .DOCMHTML ファイルとは?

DOCMHTML ファイルは、以前のバージョンの Microsoft Word で保存された Web アーカイブの古い亜種です。 Microsoft Word の最新バージョンではサポートされていません。 DOCMHTML ファイルには、元の .doc Word ドキュメントの HTML 表現が含まれています。 Microsoft Word や Web ブラウザで開くことができます。 DOCMHTML が公開されると、Web 要求を介してユーザーがアクセスできる Web サーバーでホストできます。もう 1 つの同様の形式は [DOCHTML](/web/dochtml/) で、これも廃止されました。

## DOCMHTML ファイル形式 - 詳細情報

DOCMHTML ファイルは、HTML 形式で保存されていました。 Notepad や Notepad++ などのテキスト エディターで開くことができます。廃止された後、Microsoft Word は Word 文書を [MHTML](/web/mhtml/) および [MHT](/web/mht/) ファイルとして保存するオプションを導入しました。これらの新しいファイル形式は、Web ブラウザーで開くことができます。
