## .ALX オプション番号

ALX ファイルは、[HTML](/web/html/) コンポーネントの形式で Web ページに埋め込まれた ActiveX ファイルです。 ActiveX コントロールは、正式には OLE コントロールまたは OCX と呼ばれていました。 ALXファイルを使用して、ページのコンテンツを定期的に更新するタイマーコントロールを追加したり、ユーザーアクションに基づいてツールチップスタイルのヘルプを表示するポップアップウィンドウコントロールを追加したりするなど、Webページにさまざまな機能を挿入できます。 ALXファイルケアリンクを使用<Param>Web ページの HTML コード内のタグ。 ALX ファイルは、Microsoft Internet Explorer、Microsoft ActiveX Control Pad、および Notepad や Notepad++ などのテキスト エディタで開くことができます。

## ALX ファイル形式 - 詳細情報

ALX ファイルはプレーン テキスト ファイル形式で、テキスト エディターで開いて編集できます。 ActiveX コントロールは、Web ページにドロップすることで Web ページに含めることができます。ただし、動的な Web ページを構築するために、コントロールは [JScript](/web/js/) の助けを借りて統合されています。 Web ページ上のコントロールは、コントロールのイベントと値に関連付けられているため、ユーザーの操作を完全に制御できます。
