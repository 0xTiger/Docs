## .AMXX ファイルとは何ですか?

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. これは、サーバー管理者がサーバーの機能をカスタマイズおよび拡張できるようにするスクリプト プラットフォームです。 AMXX は MetaMod プラグイン管理システムと組み合わせて使用され、複数のプラグインを競合することなくサーバー上で同時に実行できるようになります。 AMXX プラグインによって提供される機能には、管理コマンド、ゲームプレイの変更、サーバー管理が含まれます。

Windows および Linux オペレーティング システムでは、[AMX Mod X](https://www.amxmodx.org/) を使用して AMXX ファイルを開くことができます。
