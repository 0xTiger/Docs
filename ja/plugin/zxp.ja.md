## .ZXP オプション番号は何ですか?

ZXP ファイルは拡張機能として機能し、Adobe Photoshop や Adobe After Effects などの Adobe ソフトウェアの機能を強化します。以前は、Adobe Extension Manager によって ZXP ファイルのインストールが容易になりました。現在、インストール プロセスでは、Adobe Creative Cloud デスクトップ アプリケーションにバンドルされている Unified Plugin Installer Agent (UPIA) コマンドライン ツールを使用する必要があります。

## ZXP ファイル形式

「ZXP」という用語は「ZIP eXtension Package」の略です。 ZXP ファイルは、Adobe Photoshop、Illustrator、InDesign などの Adobe Creative Cloud アプリケーションの拡張機能をパッケージ化して配布するために使用されます。 ZXP ファイルは、[ZIP](/compression/zip/) 形式を使用して圧縮されているため、拡張機能の配布とインストールが簡単になります。

通常、ZXP ファイル内には、拡張機能に必要なファイル（[HTML](/web/html/)、[CSS](/web/css/)、JavaScript、画像、Adobe Creative Cloud ソフトウェアの機能を強化または追加するために必要なその他のアセットなど）が含まれています。

### ZXP ファイルをインストールするには?

ZXP ファイルは、Adobe Extension Manager または Creative Cloud デスクトップ アプリケーションを使用してインストールされます。ユーザーは ZXP ファイルをダブルクリックするだけで、拡張機能マネージャーがインストール プロセスを処理します。

ZXP ファイルは、Windows や macOS などのさまざまなオペレーティング システムで動作するように設計されているため、開発者が幅広いユーザーが使用できる拡張機能を作成するのに便利です。

### ZXP ファイル形式のセキュリティ

ZXP ファイルは開発者によってデジタル署名されるため、拡張子が改ざんされておらず、信頼できるソースからのものであることが保証されます。これは、Adobe Creative Cloud アプリケーションのセキュリティと整合性を維持するのに役立ちます。

Adobe Creative Cloud 拡張機能を使用している場合、または自分で拡張機能を開発している場合は、拡張機能をパッケージ化して配布する手段として ZXP ファイルに遭遇する可能性があります。ただし、ZXP ファイルの操作の詳細は Adobe によって変更または更新される可能性があるため、最も正確で最新の情報については、最新の Adobe ドキュメントを参照することをお勧めします。
