## .CFG オプション番号は何ですか?

CFG ファイルは、**「Wesnoth マークアップ言語」(WML)** としても知られています。これは、ターンベースの戦略ゲームである「Battle for Wesnoth」というゲームで主に使用されるカスタム マークアップ言語です。 WML は、シナリオ、キャンペーン、ユニットなど、ゲームのさまざまな側面を定義およびカスタマイズするために使用されます。これは、モッダーや開発者がゲームのコンテンツを作成するための方法です。

これは、XML と単純なスクリプトを組み合わせたような形式で記述されています。ここでは、WML ファイルに含まれる一般的な要素と構造の概要を示します。

1. **タグ:** WML はタグを使用して、ゲーム内のさまざまな要素を定義します。タグは山括弧で囲まれます。例えば：

```
[unit]
    type=Elvish Archer
    hitpoints=25
[/unit]
```
    










2. **属性:** タグ内で属性を定義して、要素に関連付けられたプロパティまたは値を指定できます。上の例では、「タイプ」と「ヒットポイント」が属性です。
    










3. **配列と配列の配列:** データの配列や配列の配列を作成して、ユニット、地形タイプ、またはその他のゲーム要素のリストを定義できます。
    










4. **条件ステートメント:** WML は、ゲームのフローを制御するための条件ステートメントをサポートしています。例えば：

```
[if]
    condition=have_unit
    variable=x,y
[/if]
```
    










5. **ループ:** ループを使用して、項目のリストを反復処理したり、アクションを繰り返し実行したりできます。
    










6. **内容:** メイン WML ファイル内に他の WML ファイルを含めて、コンテンツを整理およびモジュール化できます。
    










7. **イベント ハンドラー:** ゲーム内で特定のイベントが発生したときにアクションをトリガーするイベント ハンドラーを定義できます。
    











カスタム単位を定義する WML ファイルの簡略化された例を次に示します。

```
[unit_type]
    id=my_custom_unit
    name="Custom Unit"
    description="A unit created using WML."
    image="units/my_custom_unit.png"
    hitpoints=30
    movement_type=foot
[/unit_type]
```

## ウェスノスの戦い

「The Battle for Wesnoth」は人気のあるオープンソースのターンベース戦略ゲームです。 Mac、Windows、Linux などの複数のプラットフォームで利用できます。ボランティアの献身的なコミュニティによって開発されたこのゲームは、奥深く魅力的なゲームプレイと豊かなファンタジーの世界で知られています。

「The Battle for Wesnoth」の主な特徴は次のとおりです。

1. **ファンタジー設定:** このゲームは、人間、エルフ、ドワーフ、オークなどを含むさまざまな種族が存在するファンタジー世界を舞台としています。ゲームの伝承とストーリーテリングは、その魅力の不可欠な部分です。
    










2. **ターンベースの戦略:** ゲームプレイはターンベースであり、プレイヤーは時間をかけて六角形のグリッド上で動きを計画し、実行します。戦術的な戦闘と戦略的な意思決定を組み合わせたものです。
    










3. **キャンペーン:** このゲームでは、それぞれに独自のストーリー、キャラクター、課題を備えた幅広いシングルプレイヤー キャンペーンが提供されています。プレイヤーはさまざまな物語やシナリオを探索できます。
    










4. **マルチプレイヤー:** 「Wesnoth」はオンライン マルチプレイヤーをサポートしており、プレイヤーは戦略的な戦いで互いに競うことができます。マルチプレイヤー モードには、協力プレイと競争対戦が含まれます。

## CFGファイルを開くにはどうすればよいですか?

CFG ファイルは、一般に「The Battle for Wesnoth」ゲームで使用される Wesnoth Markup Language (WML) に関連付けられており、標準のテキスト エディタを使用して簡単に編集できます。これらのファイルには、シナリオ、ユニット、キャンペーンなどのゲームのさまざまな側面を定義する、WML で記述された人間が読めるコードが含まれています。

CFG ファイルの変更には任意のテキスト エディタを使用できますが、Emacs や Vi などの一部の高度なテキスト エディタには、WML 構文強調表示プラグインが用意されています。これらのプラグインは、ユーザーが WML コード内のさまざまな要素や構造を簡単に区別できるようにする便利な色分けと書式設定を提供します。

CFG ファイルを開いたり参照したりするプログラムには、次のものがあります。

- ウェスノスの戦い (無料) (Windows、MAC、Linux)
- Microsoft メモ帳
