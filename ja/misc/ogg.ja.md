## .OGG オプション番号

OGG ファイルは、Origin ソフトウェアで作成されたグラフ ファイルです。データ分析のためにインポートされたデータを表示するために、さまざまなタイプのグラフを保存する機能を提供します。 OGG ファイル タイプでサポートされているグラフ タイプには、縦棒グラフ、棒グラフ、円グラフ、滝グラフ、3D、極グラフ、記号があります。 Origin ソフトウェアは OriginLab によって開発され、ユーザーはデータ分析を実行し、グラフとして出力を生成できます。

## OGG ファイル形式

OGG ファイルには、インポートされたデータを表すグラフが含まれています。 OGG ファイルを開くと、Origin または Origin Viewer の別のウィンドウにグラフが表示されます。これは、Origin ソフトウェアと比較して自由に利用できます。グラフのタイトルや色などのメタデータ情報も OGG ファイル内に保存されます。

## OriginLabについて Originソフトウェア

Origin ソフトウェアは、オートメーション サーバーとして使用するオプションも提供します。このような場合、LabVIEW、Microsoft Excel、Python、または C#/VB/VC でカスタム開発されたアプリケーションなどのクライアント アプリケーションは、Origin とデータを交換できます。 [Python_to_OriginLab](https://github.com/jsbangsund/python_to_originlab) は、OriginLab でデータの送信と基本的なプロット機能を自動化する Python API です。
