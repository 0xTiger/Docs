## .GPG オプション番号

GPG ファイルは、GNU Privacy Guard (GnuPG) 暗号化プログラムで使用される暗号化/復号化キー ファイルです。 GnuPC プログラム自体は、RFC4880 で定義されている OpenPGP 標準に基づいており、PGP としても知られています。最新のオペレーティング システムで GPG をうまく使用するための鍵は、その汎用性の高いキー管理システムです。 GPG のコマンド ライン ユーティリティを使用すると、他のアプリケーションと簡単に統合できます。

## GPG ファイル形式

GPG ファイルは暗号化されたバイナリ ファイルとして保存され、もちろん人間が判読することはできません。暗号化された GPG ファイルを復号化するには、同じセキュア キーを使用する必要があります。そのため、これらのファイルの内部ファイル形式は不明です。

## Linux で GPG を使用してファイルを暗号化および復号化する

GPG コマンド ライン ユーティリティを使用して、Linux 上のファイルを暗号化および復号化できます。

### ファイルの暗号化

ファイルは、以下に示すように -c (作成) オプションを指定して gpg コマンドを使用して暗号化できます。

```
gpg -c file1.txt
```
このコマンドを実行すると、元のファイル「file1.txt」の内容を暗号化するためのキーフレーズが要求されます。これにより、暗号化されたファイル file1.txt.gpg が作成されます。

### ファイルの復号化と抽出

暗号化されたファイルを復号化して抽出するには、次のコマンドを使用できます。

```
gpg cfile.txt.gpg
```
