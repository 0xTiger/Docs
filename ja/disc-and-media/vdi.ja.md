## .VDI ファイルとは何ですか?
拡張子が .vdi のファイルは仮想ディスク イメージです。これは、VirtualBox と呼ばれる Oracle のオープン ソース デスクトップ仮想化プログラムに固有のものです。 VDI ファイルは、VirtualBox 仮想マシンを起動するために使用されます。 VM を使用すると、ユーザーは追加のオペレーティング システムまたはプログラムを 1 台のコンピューターで実行できます。したがって、VDI ファイルの利点は、ユーザーがこれらのディスク イメージ ファイルをハードディスクに保存し、必要なときにいつでもエミュレータを使用してこれらを実行できることです。

## VDI ファイル形式
仮想ディスク イメージ (VDI) は、エンタープライズ クラスの仮想化製品である VirtualBox として知られる、活発に開発されているオープン ソースの Oracle VM の主要なディスク ファイル形式です。 VDI ファイル形式は、他の仮想化プログラムを使用して簡単に実行できるポータブル ディスク イメージを作成するように設計されています。動的に割り当てられるストレージと固定サイズのストレージの両方をサポートします。既にデータが含まれている場合でも、作成後に画像ファイルを展開できます。

### ディスク仮想化
システムは、VDI ディスク イメージ形式でハードディスクをエミュレートします。 VirtualBox VM は、Microsoft Virtual PC または VMware で作成された以前のディスクと、独自のネイティブ フォーマットを使用できます。 VirtualBox は、iSCSI ターゲットに接続することもでき、ホスト上で raw パーティショニングを行うこともでき、いずれかを仮想ハードディスクとして使用できます。 VirtualBox は、IDE (PIIX4 および ICH6 コントローラー)、SATA (ICH8M コントローラー)、ハード ドライブを接続できる SAS コントローラー、および SCSI をエミュレートします。

このサポートは、VirtualBox のバージョン 2.2.0 以降、Open Virtualization Format (OVF) で利用できます。ホストに接続された物理デバイスと ISO イメージの両方を CD または DVD ドライブとしてマウントできます。
デフォルトでは、VirtualBox は VESA 互換のカスタム仮想グラフィックス カードを介してグラフィックスをサポートします。

### イーサネットの仮想化サポート
VirtualBox は、イーサネット ネットワーク アダプター用に次のネットワーク インターフェイス カードを仮想化します。
- AMD PCnet PCI II (Am79C970A)
- AMD PCnet-Fast III (Am79C973)
- インテル Pro/1000 MT デスクトップ (82540EM)
- インテル Pro/1000 MT サーバー (82545EM)
- インテル Pro/1000 T サーバー (82543GC)
- 準仮想化ネットワーク アダプター (virtio-net)

エミュレートされたネットワーク カードを使用すると、ゲスト OS の一部として使用できるため、ネットワーク ハードウェアのドライバーを検索してインストールする必要なく、ゲスト OS を実行できます。特殊な準仮想化ネットワーク アダプターは、特定のハードウェア インターフェイスに合わせる必要性を減らすことで、ネットワーク パフォーマンスを向上させます。したがって、ゲストでの特別なドライバー サポートが必要です。
