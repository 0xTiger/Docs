## .WIM オプション番号
拡張子が .wim のファイルは、Windows Vista で初めて見られました。 WIM は、通常 Microsoft Windows Vista で導入されたファイル ベースのイメージング形式に属します。これにより、単一のディスク イメージをさまざまなコンピューター プラットフォームに展開できます。 WIM ファイルは、オペレーティング システム イメージを再起動せずに、更新プログラム、ドライバー、コンポーネントなどのファイルを管理するために使用されます。 AQ WIM ファイルには、一連のファイルと関連するメタデータが含まれており、他のディスク ファイル形式と非常によく似ています。

## WIM ファイル形式
WIM ファイル形式は、VHD や IS などのセクターベースの形式とは異なり、実際にはファイルベースであり、WIM 内の情報の基本単位がファイルであることを意味します。ファイルベースであることの基本的な利点は、ファイルの単一インスタンス ストレージがファイル システム ツリーで複数回参照され、ハードウェアに依存しないことです。ファイルは単一の WIM 内に格納されるため、さまざまなファイルを個別に開いたり閉じたりするオーバーヘッドが削減されます。ファイル。 WIM ファイルには複数のディスク イメージを格納できます。これらのイメージは、一意の名前または数値インデックスによって参照されます。
### 圧縮アルゴリズムのサポート
WIM は、速度の降順と比率の昇順で、次の圧縮アルゴリズムのファミリをサポートしています。
- エクスプレス
-LZX
-LZMS
- 固体圧縮。
最初の 3 つのファミリは LZ77 ベースですが、WIMGAPI Windows 8 および DISM Windows 8.1 では、LZMS とともに Solid 圧縮が導入されました。
### WIM ファイル形式のツール
次のツールは通常、WIM ファイル形式をサポートしています。

- **ImageX**: Windows イメージング フォーマットで Windows ディスク イメージを編集、作成、および展開するために使用されるコマンド ライン ツール。関連する WIMGAPI とともに、無料の Windows 自動インストール キットの一部として配布されます。 Windows Vista 以降、Windows セットアップは WAIK API を使用して Windows をインストールします。
- **DISM**: Windows 7 および Windows Server 2008 R2 で導入されたツールで、Windows インストール イメージ (オンライン イメージ、またはフォルダーまたは WIM ファイル内のオフライン イメージ) に対してサービス関連のタスクを実行できます。このツールは、イメージのマウントとアンマウント、オフライン イメージへのデバイス ドライバーの追加、およびオフライン イメージにインストールされているデバイス ドライバーのクエリを行うことができました。
