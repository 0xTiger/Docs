## Type 1 フォントとは

Type 1 フォントは、PostScript を使用できるデスクトップ ベースのパブリッシング ソフトウェアおよびプリンターで広く使用されていた非推奨の Adobe テクノロジーです。 Type 1 フォントは、最新のプラットフォーム、Web ブラウザー、モバイル オペレーティング システムの多くではサポートされていませんが、一部のオペレーティング システムではサポートされています。 Type 1 フォントでの Unicode 情報のサポートが不十分なため、拡張言語文字セットをサポートする能力も制限されます。

## 簡単な歴史

Type 1 フォント技術は、Adobe の PostScript ページ記述言語で使用するために 1984 年に開始されました。 90 年代半ば、Adobe は Type 1 ではなく、より柔軟な OpenType フォントの使用に専念することを決定しました。

### Type 1 のサポートはいつ終了しますか?
2023 年 1 月以降、ユーザーは Type 1 フォントを使用してコンテンツを作成できなくなります。それまでは、変更は行われません。
Document Cloud アプリケーションなどの一部の製品は、これまでと同様に Type 1 フォントを引き続き使用します。


## Type 1 フォントの制限事項

タイプ 1 の既知の制限の一部を以下に示します。

- 1 つのフォントに 256 を超えるグリフを含めることはできません。 CID フォントはこれだけ多くの出力デバイスを実行できますが、CID フォントを適切に処理できない場合があります。
- Type 1 フォントはクロスプラットフォームではありません。あるプラットフォームのフォントを別のプラットフォームのフォントに変換する際には、複雑な変換が必要です。
- 古い DOS 時代の 8 文字の制限にフォント名が固執しているため、ファイルに保存されている書体を判別するのが難しくなっています。

## Windows マシンでのタイプ 1 フォント
タイプ 1 フォント データは、ウィンドウ内の 2 つの個別のファイルで構成されます。

1. 拡張子「.PFB」のファイルはアウトラインデータです。
2. 拡張子が「.PFM」のファイルには、メトリック データが含まれています。
