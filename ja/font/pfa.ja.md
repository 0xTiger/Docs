## .PFA オプション番号

拡張子が .pfa のファイルは、Adobe の Type 1 フォントの Printer Font ASCII バージョンであり、フォントのグリフ データが含まれています。このファイルは人間が読める形式であり、任意のテキスト エディターで開くことができます。 PostScript インタープリターは、このタイプのフォントを全体として使用して作業できます。 Adobe の Type 1 フォントの Printer Font Binary (PFB) バージョンには、同じ情報が含まれています。ただし、PFA をアンラインすると、バイナリ ファイル形式であり、人間が判読することはできません。 PFA ファイルは、システムの Fonts フォルダーにコピーすることにより、システムに組み込むことができます。
