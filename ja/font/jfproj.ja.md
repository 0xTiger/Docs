## .JFPROJ ファイルとは何ですか?

.jfproj 拡張子を持つファイルは、Windows 10 の Microsoft Font Maker アプリケーションで作成された JSON ベースのカスタム フォントです。これらは、タブレットなどのタッチ スクリーン デバイスで電子ペンを使用して作成できますが、場合によってはマウスなどの他のコンピューター周辺機器も使用できます。コンピュータの。 JFPROJ ファイルには、文字と単語の間の間隔、およびフォントのサイズを定義することにより、画面に文字をレンダリングするために使用されるグリフが含まれています。 Microsoft Font Maker アプリは、Windows ストアからダウンロードできます。
