## NX オプション番号は何ですか?

NX ファイルは、仮想ゲーム コンソール **LowRes NX** で使用するために作成されたビデオ ゲーム ファイルです。これは BASIC プログラミング言語で書かれており、ゲームの機能を定義するゲームのソース コードが含まれています。アプリで使用する独自の NX ファイルを作成し、それをロードしてユーザーがゲームをプレイできるようにすることができます。

NX ファイルは、テキスト エディタ、LowRes NX、または Microsoft Visual Studio Code などのソース コード エディタで開くことができます。

## NX ファイル形式

NX ファイルは、ゲームの説明を含むプレーン テキスト ファイルとしてディスクに保存されます。各 NX ファイルには、複数のカートリッジ ROM エントリで構成されるゲームのソース コードが含まれています。これらの ROM エントリには、グラフィックス、レベル マップ、オーディオ、およびゲームのその他の非ソース コード データなどのゲーム コンポーネント情報が含まれています。プログラムの ROM エントリは、LowRes NX にバンドルされている NX 編集プログラムを使用して追加または編集できます。

## NXファイルを作成するには?

### PC 上で NX ファイルを作成する

以下を使用して、PC 上で NX ファイルを作成できます。

* テキスト エディタ - Microsoft Notepad、Notepad++、Github Atom など
* ソース コード エディタ - Microsoft VIsual Studio Code など

### iOS で NX ファイルを作成する

LowRes NX には、開発者がこのアプリ内から NX ファイルを作成するために使用できる iOS バージョンがあります。
