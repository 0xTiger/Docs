## .RBXL オプション番号

RBXL ファイルは、[Roblox ゲーム](https://www.roblox.com/) 内の 3D ワールドで定義された場所を含む Roblox ロケーション ファイルです。 Roblox 自体はマルチプレイヤー ビルディング ゲームであり、プレイヤーはプラットフォームが提供する開発者ツールを使用してゲームを構築できます。 RBXL ファイルは、ツール、スクリプト、チーム、およびパーツを使用して、競技場を 3D モデルとして記述します。 Roblox Studio を使用すると、プレーヤーはゲーム アイテムを作成でき、RBXL ファイルを開くために使用できます。

## RBXL ファイル形式

RBXL ファイルは、Roblox Studio を使用して開発され、[XML](/web/xml/) ファイル形式で保存されます。この普遍的なファイル形式により、これらのファイルを作成して開くためのツールや API が多数あります。そのような API の 1 つは [MaximumADHD](https://github.com/MaximumADHD/Roblox-File-Format) で、Roblox のモデル/場所ファイル形式でファイルを作成および操作するための [C#](/programming/cs/) ライブラリです。 .
