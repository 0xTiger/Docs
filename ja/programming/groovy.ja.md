## .GROOVY ファイルとは?

GROOVY ファイルは、Groovy プログラミング言語で記述されたソース コード ファイルです。 GROOVY ファイル内に記述されたコードは、オブジェクト指向言語 [Java](/programming/java/) に似ているため、アプリケーションの設計と開発が容易になります。 GROOVY ファイルは Java 仮想マシン (JVM) バイトコードにコンパイルされ、他の Java コードおよびライブラリと互換性があります。

## GROOVY ファイル形式 - 詳細情報

GROOVY ファイルには、Groovy 構文で記述されたソース コードが含まれています。 Java プラットフォームのプログラミング言語とスクリプト言語の両方として使用できるコードを含めることができます。 Groovy 言語には、Python、Ruby、および Smalltalk と同様の機能があります。
