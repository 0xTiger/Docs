## .SCRIPT ファイルとは?

SCRIPT ファイルは、任意のスクリプト言語のスクリプトを含むソース ファイルです。これにより、変数と関数を含む一般化されたスクリプト ファイルになります。スクリプト言語はそれぞれ異なる場合があるため、SCRIPT ファイル内に記述されたスクリプトは、適切なスクリプト エンジンを使用して実行する必要があります。

## SCRIPT ファイル形式 - 詳細情報

SCRIPT ファイルは、プレーン テキスト ファイルでディスクに保存されます。これらは、Microsoft Notepad、Notepad++、Apple TextEdit などの任意のテキスト エディタで開いて編集できます。スクリプト言語は異なる場合があるため、ファイル拡張子をそれぞれのスクリプト言語のデフォルトの拡張子に変更することで実行できます。たとえば、JavaScript ([.JS](/web/js/))、VBScript ([.VBS](/programming/vbs/))、AppleScript (.SCPT)、PHP ([ .PHP](/programming/php/))、および ASP ([.ASP](/web/asp/))。
