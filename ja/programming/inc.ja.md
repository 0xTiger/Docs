## .INC ファイル拡張子

INC ファイルは、ヘッダー情報を参照するためにソフトウェア プログラムのソース コードで使用されるインクルード ファイルです。 [.h ファイル形式](/programming/h/) に似ており、C++ などのソース コード ファイルで再利用できる宣言、関数、ヘッダー、またはマクロが含まれています。 INC ファイルは、C、[C++](/programming/cpp/)、Pascal、[PHP](/programming/php/)、および [Java](/programming/java/) などのさまざまなプログラミング言語で使用されます。 Microsoft Notepad、Notepad++、TextEdit などのテキスト エディタを使用して INC ファイルを開くことができます。

## INC ファイル形式 - 詳細情報

INC ファイルは、宣言構文に従ってすべての宣言が含まれたプレーン テキスト ファイルとして保存されます。 1 つの INC ファイルは、他の INC ファイルも参照できます。作成された INC ファイルはプロジェクトの一部となり、C++ などのソース ファイルから参照できます。重複を避けるために、複数のソース ファイルから参照できます。

## INC ファイルの内容

INC ファイルには、次の情報を含めることができます。

**`Variables`** - オブジェクト指向プログラミング (OOP) の場合、クラス ヘッダー ファイルには、実装ソース コード ファイル全体でアクセス可能なすべてのクラス レベル変数の定義が含まれます。

**`メソッド宣言`** - すべてのメソッド宣言は、複数の実装ファイルからアクセスできるように .h ヘッダー ファイルに含まれています。

**`非インライン関数定義`** - ヘッダー ファイルには、非インライン メソッドの定義を含めることもできます。

## PHPでINCファイルを使用するデメリット

PHP は、サーバー上で実行され、呼び出し元の Web ページに結果を返すサーバー側スクリプトです。 PHP ファイルが INC ファイルを参照しており、サーバーが .inc ファイルを解析するように構成されていない場合 (これは非常に一般的です)、ユーザーは URL ディレクトリにアクセスして .inc ファイル内の php ソース コードを表示できます。そのため、INC ファイルを PHP のインクルード ファイルとして使用する場合は注意が必要です。
