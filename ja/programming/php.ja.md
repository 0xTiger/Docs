## .PHP ファイルとは?
拡張子が .php のファイルは、Web サーバー上で実行されるサーバー側スクリプトを記述するために使用されるオープン ソース プログラミング言語を指します。これは、大規模な Web アプリケーションの開発に通常使用される、最も広く使用されている Web スクリプト言語です。 Web で最大のブログ システム、つまり WordPress と最大のソーシャル ネットワーク Facebook は、PHP 言語に基づいています。 Windows、Linux、Unix、Mac OS X などのさまざまなプラットフォームと互換性があります。また、Apache、IIS など、現在使用されているほぼすべてのサーバーで正常に動作します。PHP は、MySQL を含む幅広いデータベースをサポートしています。

## PHP ファイル形式 ##

単一の PHP には、テキスト、CSS、[HTML](/web/html/)、JavaScript、および PHP コードを含めることができます。 PHP コードはサーバー上で実行され、結果はプレーンな HTML としてブラウザーに返されます。 PHP ファイルは任意のテキスト エディターで開いてその場で編集できますが、Adobe Dreamweaver、Eclipse PHP 開発ツールなどのアプリケーションを使用すると、PHP コードを記述および変更するための便利な方法が提供されます。

### 構文 ###

サーバー上で PHP スクリプトが実行され、プレーンな HTML の結果がブラウザーに返されます。 PHPスクリプトは次で始まります<?php and ends with ?>

### PHP の例 ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
