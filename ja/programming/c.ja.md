## C ファイルとは何ですか?

cファイル拡張子で保存されたファイルは、Cプログラミング言語で書かれたソースコードファイルです。 **C ファイル**には、アプリケーションの機能のすべての実装がソース コードの形式で含まれています。ソース コードの宣言は、[.h](/programming/h/) 拡張子で保存されるヘッダー ファイルに記述されます。 C++ は C 言語の最新の形式であり、現在ほとんどのソフトウェアの開発に使用されています。

## 簡単な歴史

C 言語は、UNIX オペレーティング システム用のさまざまなユーティリティを作成した結果として誕生しました。このプログラミング言語の作成者であるデニス・リッチーは、1960 年代から 1970 年代初頭にかけてこの作業を開始しました。

## C ファイル形式

C ファイルは、プログラミング言語の構文に従ってプレーン テキスト ファイル形式で記述されます。典型的な C ファイルには次のものがあります。

* 任意のヘッダー ファイルをインポートするためのファイルの先頭にある import ステートメント
* 必要な機能を実装するための 1 つまたは複数の方法

### ヘッダーのインポート

拡張子が .h のヘッダー ファイルには、プロジェクトに他の機能ファイルを含めるために必要な include ステートメントが含まれています。さらに、これらには、実装ファイルで定義されたメソッドの宣言が含まれています。以下に示すように、include ステートメントを使用してヘッダー ファイルがインクルードされます。

```
#include <filename.h>
```

### ソースコードの実装

機能要件の実際の実装は、C ファイルのメソッドとしてコーディングされています。 C ファイル内の各メソッドには、戻り値の型、メソッド名があり、いくつかの入力パラメーターがある場合があります。戻り値の型が void でない場合、メソッドは何らかの値を返している可能性があります。

### C コードの例
以下は ac のサンプル プログラムです。

```
long some_function();
/* int */ other_function();

/* int */ calling_function()
{
    long test1;
    register /* int */ test2;

    test1 = some_function();
    if (test1 > 0)
          test2 = 0;
    else
          test2 = other_function();
    return test2;
}
```
