## .CS オプション番号

拡張子が .cs のファイルは、C# プログラミング言語のソース コード ファイルです。 .NET Framework で使用するために Microsoft によって導入されたファイル形式は、EXE または DLL の形式で最終的な出力ファイルを生成するためにコンパイルされるコードを記述するための低レベルのプログラミング言語を提供します。これらは、Microsoft Visual Studio で作成およびコンパイルできます。無料の IDE である Microsoft Visual Studio Express を使用して、このようなファイルを作成および更新することもできます。 CS ファイルは、単純なデスクトップ アプリケーションからより複雑なプログラムまで、さまざまなアプリケーション開発に使用されます。 C# 言語で作成された単純な Visual Studio プロジェクト [ソリューション](/programming/sln/) は、1 つ以上のそのようなファイルで構成できます。コンパイルに含めるようにマークされたファイルは、プロジェクトの一部である [CSPROJ](/programming/csproj/) ファイルに一覧表示され、マークされたファイルを使用するようにコンパイラに指示します。

## CS ファイル形式 ##

CS ファイルは、任意のテキスト エディタで開いて編集できるテキスト ベースのファイル形式です。ただし、サポートされている IDE で適切な構文の強調表示を使用して開くと、コードが読みやすく、整理しやすくなります。シンプルな CS ファイルには以下が含まれます。

* 名前空間の宣言 - 特定の名前空間によって定義された特定の機能を参照するため
* 変数宣言 - 特定の実装のクラス レベル変数を宣言する
* メソッド宣言 - 特定の機能のメソッド宣言を宣言する

### 構文 ###

※セミコロンは、文の終わりを示すために使用されます。
* 中括弧は、ステートメントをグループ化するために使用されます。一般に、ステートメントはメソッド (関数) に、メソッドはクラスに、クラスは名前空間にグループ化されます。
* 変数は等号を使用して割り当てられますが、2 つの連続した等号を使用して比較されます。
* 角括弧は、配列を宣言するためと、それらのいずれかの特定のインデックスで値を取得するために、配列で使用されます。

＃＃＃ 例 ＃＃＃

```
using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Hello, world!");
}
}
```
