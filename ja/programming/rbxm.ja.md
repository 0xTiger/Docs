## .RBXM オプション番号

RBXM ファイルは、Roblox Studio IDE でオンライン マルチプレイヤー Roblox ゲームの 3D オブジェクトを保存するために使用される 3D モデル ファイル形式です。 IDE を使用してゲーム用に作成された 3D オブジェクトが含まれています。 RBXM ファイル内の 3D オブジェクトの例には、シャツ、ズボン、帽子、建物、乗り物、動物、飛行機、車、植物などがあります。 RBMX ファイルは、Roblox IDE を使用して [OBJ 3D ファイル形式](/3d/obj/) に変換できます。 Roblox IDE を使用して RBMX ファイルを開くことができます。

もう 1 つの一般的な Roblox ファイル形式は [RBXL](/programming/rbxl/) です。これは場所ファイル形式であり、Roblox ゲーム内の 3D ワードで定義された場所が含まれています。

## RBMX ファイル形式 - 詳細情報

RBXM ファイルはバイナリ ファイル形式で作成および保存され、内部ファイル形式の仕様は公開されていません。これは、[XML](/web/xml/) ファイル形式に基づく RBXMX ファイル形式とは異なります。 [Roblox RBXM の処理](https://github.com/search?q=rbxm) ファイルに使用できるツール/リポジトリが Github で利用可能です。
