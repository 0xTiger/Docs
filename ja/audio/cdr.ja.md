## .CDR ファイルとは何ですか?

**「Raw CD Audio Data」** を含む **.CDR** ファイルは通常、音楽トラックなど、オーディオ CD から直接抽出されたオーディオ情報を含むファイルを指します。 「Raw CD オーディオ データ」とは、CD からの未処理のデジタル オーディオ データを指し、多くの場合バイナリ形式で保存されます。この生データには、オーディオとサブコード情報の両方を含む、CD 上のオーディオ サンプルがまったくそのまま含まれています。

これらのファイルは通常、直接再生することを目的としたものではなく、CD マスタリング、オーディオ編集、CD 書き込みなどのさまざまな目的に使用されます。オーディオ編集や CD オーサリング用のソフトウェア アプリケーションは、多くの場合、この生データを使用してオーディオ CD を作成したり、WAV や MP3 などの他のオーディオ ファイル形式に変換したりします。

「Raw CD オーディオ データ」を含む .cdr ファイルを扱う必要がある場合、それらのファイルはほとんどのメディア プレーヤーで再生できる標準のオーディオ ファイルではないため、通常は専用のオーディオ ソフトウェアを使用して処理または変換します。

## CDRファイルを変換するにはどうすればよいですか?

「Raw CD オーディオ データ」を含む .cdr ファイルを WAV や MP3 などのより一般的なオーディオ形式に変換するには、専用のソフトウェアが必要です。 VideoLAN VLC メディア プレーヤーで .cdr ファイルを再生できる場合は、VLC を使用してファイルを別のオーディオ形式に変換することもできます。

- [.MP3](/ja/audio/mp3/) - MP3 オーディオ
- [.FLAC](/ja/audio/flac/) - 無料のロスレスオーディオコーデック
- [.WAV](/ja/audio/wav/) - WAVEオーディオ
- [.OGG](/ja/audio/ogg/) - Ogg Vorbis オーディオ

## CDRファイルを開くには?

.cdr ファイル、特に「Raw CD オーディオ データ」を含むファイルを開くのは、標準のオーディオ ファイル形式ではないため、少し難しい場合があります。このようなファイルを開いて操作するには、生の CD オーディオ データを処理できる特殊なソフトウェアを使用できます。その方法といくつかのソフトウェアオプションは次のとおりです。

**.cdr ファイルを開いて操作するソフトウェア:**

1. **正確なオーディオ コピー (EAC)**:
    





- Exact Audio Copy をダウンロードしてインストールします。これは、オーディオ CD を操作し、「Raw CD オーディオ データ」を抽出するための人気のある強力なツールです。
- .cdr ファイルを含む CD をコンピュータの CD/DVD ドライブに挿入します。
- EAC を開き、出力形式 (WAV または MP3 など) を含む好みに合わせて構成します。
- EAC を使用して CD からオーディオ トラックを抽出し、希望の形式で保存します。
2. **大胆さ**:
    





- Audacity は、「Raw CD オーディオ データ」でも動作する、無料のオープンソース オーディオ編集ソフトウェアです。
- CD を挿入します。
- Audacity を開き、CD ドライブからオーディオを録音するように設定します。
- CD からオーディオを録音し、より一般的なオーディオ形式で保存します。
3. **VLC メディア プレーヤー**:
    





- VLC は、「Raw CD オーディオ データ」を含む CD から直接オーディオを再生できます。
- CD をコンピュータに挿入します。
- VLCを開き、「メディア」メニューに移動し、「ディスクを開く」を選択します。
- CD ドライブを選択し、オーディオ トラックを直接再生します。
4. **CD リッピング ソフトウェア**:
    





- Windows Media Player (Windows 用)、iTunes (Mac 用) など、さまざまな CD リッパー ソフトウェア オプションが利用可能です。
- CD をコンピュータに挿入します。
- 好みの CD リッパー ソフトウェアを開き、それを使用してオーディオ トラックを抽出し、一般的なオーディオ形式で保存します。
5. **DAEMON Tools** (.cdr イメージ ファイル用):
    





- .cdr ファイルが DAEMON Tools などのソフトウェアで作成されたイメージ ファイルの場合、DAEMON Tools または同様のソフトウェアを使用してイメージをマウントできます。
- イメージがマウントされると、物理 CD と同様にそのコンテンツにアクセスし、必要に応じてオーディオを再生または抽出できます。
