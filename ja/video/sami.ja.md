## SAMI ファイルとは何ですか?

拡張子「.sami」を持つファイルは通常、Subtitles And Metadata Interchange (SAMI) ファイルを指します。 SAMI は、ビデオに字幕またはクローズド キャプションを表示するために使用されるキャプション形式です。これはもともと Microsoft によって Windows Media Player 用に開発されました。

SAMI ファイルには、字幕またはクローズド キャプションのタイミング情報とテキスト コンテンツが含まれており、ビデオ再生と同期できるようになります。この形式は、フォント スタイル、色、画面上の字幕の位置などの基本的な書式設定オプションをサポートしています。

SAMI ファイルは通常、プレーン テキスト ファイルであり、テキスト エディタを使用して開いて編集できます。 SAMI ファイルのコンテンツには通常、ファイルに関する情報を提供するヘッダー セクションと、その後にそれぞれのタイミングとテキストを含む個々のサブタイトル エントリが含まれます。

SAMI ファイルの例を次に示します。

```
<SAMI>
<HEAD>
<TITLE>Example Subtitles</TITLE>
</HEAD>
<BODY>
<SYNC Start=100><P Class=ENCC>Subtitle 1</P></SYNC>
<SYNC Start=500><P Class=ENCC>Subtitle 2</P></SYNC>
<SYNC Start=1000><P Class=ENCC>Subtitle 3</P></SYNC>
</BODY>
</SAMI>
```

SAMI ファイルは通常、Windows Media Player や VLC Media Player などの、字幕表示をサポートするビデオ プレーヤーまたはメディア プレーヤーと組み合わせて使用されます。プレーヤーは SAMI ファイルを読み取り、字幕をビデオ コンテンツと同期させるため、視聴者はビデオを見ながらキャプションを読むことができます。

## サポートされているHTMLタグ

SAMI (Subtitles And Metadata Interchange) ファイルは、字幕のフォーマットとスタイルを設定するための HTML のようなタグの限定されたセットをサポートします。 SAMI でサポートされている一般的に使用されるタグの一部を次に示します。

-「<SAMI> :`` SAMI ファイル全体をカプセル化するルート要素。
-「<HEAD> :`` SAMI ファイルのヘッダー情報が含まれます。
<html>-「<TITLE> :`` SAMI ファイルのタイトルを指定します。 </html>
-「<BODY> :`` 字幕エントリとそのタイミング情報を囲みます。
-「<SYNC> :`` 字幕エントリの同期ポイントを表します。字幕を表示するタイミングを指定します。
-「<P> :`` 字幕の実際のテキスト内容を囲みます。通常、以下の範囲内で使用されます。<SYNC>ブロック。
<html>- `` <FONT>:`` 囲まれたテキストのフォントプロパティを定義します。 Color、Face、Size、Style などの属性を使用して、フォントの外観を変更できます。</font>
-「<BR> :`` サブタイトル内に改行を挿入します。
<html>- `` <B>:`` 囲まれたテキストを太字で表示します。</b>
<html>- `` <I>:`` 囲まれたテキストを斜体で表示します。</i>
<html>- `` <U>:`` 囲まれたテキストに下線を付けます。</u>
-「<C> :`` 画面上の字幕テキストの位置または配置を指定します。 Center、Middle、Left、Right、Top、Bottom などの属性とそれらの組み合わせがサポートされています。
-「<LANG> :`` 字幕テキストの言語コードを指定します。字幕の言語を識別するのに役立ちます。

これらは、SAMI ファイルでサポートされる基本タグの一部です。 SAMI は HTML タグと属性のすべてをサポートしているわけではないことに注意することが重要です。サポートされているタグは、広範なドキュメントの構造化や対話性を提供するのではなく、主に字幕のスタイル設定と位置決めに重点を置いています。
