## .MPT オプション番号##

拡張子が .mpt のファイルは、[Microsoft Project](https://products.office.com/en-us/project/project-and-portfolio-management-software) テンプレート ファイルです。これには、[MPP](/project-management/mpp/) ファイルを作成するための基本的な情報と構造およびドキュメント設定が含まれています。これらのテンプレートは、スケジュールや予算情報など、特定のプロジェクトに不可欠な既定の設定を提供します。ただし、タスク、リソース、割り当てなどのプロジェクト固有のデータは保存できません。テンプレート ファイルを変更したら、標準の MPP ファイルとして保存して、さらに作業を進めることができます。 「Global.mpt」というタイトルのマスター テンプレート ファイルは、Microsoft Project で入手でき、個々のテンプレートを作成するために開かれます。 MPT ファイルは、*application/vnd.ms-project* と *application/x-project* を標準の MIME タイプとして使用します。 Windows、Linux、および macOS はすべて、Moos Project Viewer、Microsoft Project、および MPXJ とともに、MPT ファイルを開くことができます。

## MPT ファイルを開く際の問題? ##

MPT 形式の誤動作の原因となる一般的な問題のリストを次に示します。

*サポートソフトウェアの不在
* 破損したファイル
* ウイルスによる感染ファイル
* システムにファイルを開くためのアクセス権がありません
* システムの古いドライブ
 *   Extension of the file is renamed
