## .MPP オプション番号

拡張子が .mpp のファイルは、プロジェクト管理に関する情報を統合的に格納する Microsoft Project のデータ ファイルです。 Microsoft は、この独自のファイル形式を開発して、同社のプロジェクト管理ソフトウェア製品である Microsoft Project (MSP) と互換性を持たせました。 MPP 以外に、MSP はプロジェクトの XML スキーマを操作できます。多くのアプリケーションと API では、MPP を他のファイル形式に変換できます。 Microsoft は現在、複数のユーザーによるコラボレーションのためにプロジェクト管理ファイルをアップロードできるオンライン Project Server を提供しています。

## MPP ファイル形式 ##

Microsoft は、MPP をバイナリ ファイル形式として開発および公開しましたが、その仕様は現在まで Microsoft によって公開されていません。したがって、開発者は内部ファイル構造と詳細を利用できません。 MPP ファイルの読み取り/書き込みをサポートするオンラインで利用可能な API がいくつかありますが、100% の精度が達成されることはありません。 Microsoft は時間をかけてプロジェクト管理アプリケーションにいくつかの機能を導入しており、MPP ファイル形式の機能も導入しています。

MPP ファイルは、次の MIME タイプを使用します。

* アプリケーション/vnd.ms-プロジェクト
* アプリケーション/msproj
* アプリケーション/msproject
* アプリケーション/x-msproject
* アプリケーション/x-ms-プロジェクト
* アプリケーション/x-dos_ms_project
* アプリケーション/mpp
* zz-アプリケーション/zz-winassoc-mpp

## MPP ファイルを開く際の問題? ##

MPP 形式の誤動作の原因となる一般的な問題のリストを次に示します。

*サポートソフトウェアの不在
* 破損したファイル
* ウイルスによる感染ファイル
* システムにファイルを開くためのアクセス権がありません
* システムの古いドライブ
※ファイルの拡張子はリネーム
