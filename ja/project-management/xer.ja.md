## .XER オプション番号

XER ファイル形式は、Primavera P6 プロジェクト計画および管理アプリケーションで使用される独自のプロジェクト ファイル形式です。 [MPP](/project-management/mpp/) ファイル形式と同様に、このファイル形式には、タスク、リソース、タイムライン、資料などのプロジェクト計画に関連するデータが保存されます。 Primavera は Oracle によって開発され、完全なプロジェクト計画を提供し、石油とガス、建設、製造などのいくつかの実用的な分野でそのアプリケーションを見つけます。統合することにより、プロジェクト マネージャーが大規模なプログラムと個々のプロジェクトを計画、スケジュール、および制御するのに役立ちます。財務管理と人的資本管理システムを備えています。多くのソフトウェアと API は、XER ファイルを読み取って、XML や MPP などの他の一般的なプロジェクト ファイル形式に変換できます。

## XER ファイル形式 ##

XER ファイルはテキスト ファイルであり、テキスト エディタで簡単に開くことができます。プロジェクト関連のデータをテキスト エディターで開くと便利ですが、リンクを手動で作成して理解するのは簡単ではありません。この形式は、他の P6 データベースとのデータ交換に最もよく使用される形式です。別の形式からインポートされたデータに対する制御レベルは比較的低くなります。同等の XML 形式に対する XER の利点には、グローバル コードとカレンダーをプロジェクト レベルに移動し、コード、UDF、リソース、およびカレンダーを既存の値にマップする機能が含まれます。


## XER ファイルを開く際の問題? ##

XER 形式の誤動作の原因となる一般的な問題は次のとおりです。

*サポートソフトウェアの不在
* 破損したファイル
* ウイルスによる感染ファイル
* XER の説明が誤って Windows レジストリから削除された
* システムにファイルを開くためのアクセス権がありません
* システムの古いドライブ
※ファイルの拡張子はリネーム
