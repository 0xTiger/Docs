## FB2 ファイルとは?

拡張子が .fb2 のファイルは、eBook の構造を含む FictionBook 2.0 eBook ファイルです。これは XML 形式に基づいており、本の各要素を説明するための特別なタグが含まれています。主に架空の文章や文学のために開発されましたが、これらだけに限定されるものではありません。この形式は、すべてのメタデータとコンテンツ自体を収容し、自動処理、インデックス作成、他の形式への変換など、さまざまな操作を柔軟に行うことができます。つまり、ファイルの外観を指定するのではなく、ファイルの構造を記述することに重点を置いています。 Windows、MacOS、および Linux で FB2 を他のいくつかの形式に変換するために、いくつかのアプリケーションと API を使用できます。

## FB2 ファイル形式

FictionBook 2.0 ファイル形式は、ロシアの FictionBook の Dmitry Gribov によって開発されました。複数の XML ファイルを含む [EPUB](/ebook/epub/) などの他の一般的な電子ブック ファイル形式とは異なり、コンテンツに関する完全な情報を含む単一の XML ファイルに基づいています。ファイル内に[PNG](/image/png/)や[JPG](/image/jpeg/)などの画像がある場合、これらはBase64に変換されて文書内に埋め込まれます。 FB2 の [ファイル形式の仕様](http://gribuser.ru/xml/fictionbook/index.html.en) には、バージョン 2.0 の形式に関する情報が含まれています。
