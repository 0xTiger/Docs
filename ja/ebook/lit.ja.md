## .LIT オプション番号

LIT ファイルは、電子書籍を読むために使用された古代の **Microsoft Reader** に属しています。 Microsoft Reader で表示される電子ブックには、**Literature** に短縮された **.lit** という拡張子が付いています。 LIT ファイル形式は、Microsoft ITOL/ITLS HTML Help 2.0 ファイルで構成され、[LZX](/compression/lzx/) 圧縮、マークアップ マテリアルのバイナリ表現、および関連する任意のオプションの補助データなど、あらゆる種類の素晴らしい機能を提供します。各ストリーム。ブック テキストは、任意の数の OEBPS 1.0 マークアップ ストリームであり、OEBPS 1.0 CSS のサブセットです。画像は[GIF](/image/gif/)、[JPEG](/image/jpeg/)、[PNG](/image/png/)です。

## Microsoft Reader が LIT ファイルを表示した時間

**Microsoft Reader** は、マイクロソフトが提供を終了した電子書籍を読むためのアプリケーションです。 Microsoft Reader の最初のバージョンは 2000 年 8 月にリリースされ、独自の LIT 形式を使用していました。 PDA と Windows コンピューターで使用できました。アプリの名前は、後に [PDF](/pdf/) および [XPS](/page-description-language/xps/) ファイルを読み取るためのアプリケーション (Windows 8 で最初にリリースされた) にも使用されました。 Microsoft Reader アプリケーションは 2018 年に廃止されました。
