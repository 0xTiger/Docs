## HTML0 ファイルとは? ##

HTML0 は **Book Designer** で使用されるネイティブの eBook 形式で、各ページのコンテンツとデザインを記述する標準の HTML および CSS コードが含まれています。 **Book Designer** は、HTML0 ファイルで機能し、テキスト ドキュメントを多数の eBook 形式に変換するための最も一般的なプログラムです。 HTML0ファイル形式は、テキストファイルのカテゴリに属しているため、Windowsシステムにインストールできるソフトウェアをサポートしています. HTML0 ファイルは通常の HTML で構成され、テキスト エディタを使用して表示および修正できます。 HTML0 ファイルは、通常の HTML ファイルと比較して、個別の CSS ファイルにリンクすることはできません。

## HTML0 ファイルを開くのに問題がありますか? ##

※ファイルが開けにくい場合は感染している可能性があります
* オンラインのウイルス対策スキャナーでファイルのウイルスをスキャンする必要があります
* 一部のファイルを開くにはアクセス権が必要です
* システムに HTML0 ファイルを開くのに十分なリソースがない場合は、すべてのアプリケーションを閉じてから再度開きます
* ファイルをダウンロードした場合、または電子メールから受信した場合、ファイルが破損しているか、エラーが発生している可能性があります
*常に最新のシステムとドライバーを使用して、コンピューターをより安全にし、ファイル拡張子に関連するすべての問題を解決します
