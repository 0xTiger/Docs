## .AEP オプション番号##

拡張子が .aep のファイル (**Activ E-Book Project** の略) は、電子書籍 (eBook) に分類されます。 AEP ファイルは、Internet Explorer と同じエンジンを使用して作成され、100% の互換性を保証します。人が物理的な本を読むことができるようなページが含まれています。 AEP ファイルのページには、通常、テキスト、画像、アニメーション、およびハイパーリンクが含まれています。また、[HTML](/web/html/)、[DHTML](/web/dhtml/)、[JPEG](/image/jpeg/)、[PNG](/image/png/)のグラフィック、アニメーションをサポートしています。 、[JavaScript](/web/js/)、VBScript、および多くの Internet Explorer プラグイン。 AEP ファイルは Windows 実行可能ファイル (EXE) として保存できるため、Windows95 以降のバージョンの Microsoft Windows で開くことができます。
 


## AEP ファイル形式 ##

AEP 形式のファイルは、AEC (Active E-Book Compiler) ソフトウェアを使用して作成されます。このソフトウェアは、IE と同様のエンジンを使用して電子書籍を作成しています。ユーザーは、プログラムの使用に費用を支払うことなく、電子ブックの無制限のコピーを作成できます。すべての eBook は自己完結型の Windows プログラムで構成されているため、配布プロセスも簡単で、電子メール、インターネット ダウンロード、フロッピー ディスク、CD-ROM、ZIP ディスクなどの任意のメディアで配布できます。

## ファイルを開く際に発生する可能性のある問題

AEP ファイルを開いて実行できない場合。デバイスに適切なソフトウェアがインストールされていないという意味ではありません。ファイルが正常に動作するのを妨げる他の問題がいくつかある可能性があります。考えられる問題は、次のいずれかです。

- AEP ファイルの破損
- レジストリ エントリ内の AEP ファイルへの間違ったリンク
- Windows レジストリから AEP の説明を削除
- AEP形式をサポートするアプリケーションのインストールの破損
- 望ましくないマルウェアを含む感染した AEP ファイル
- AEPファイルを操作するのに十分なハードウェアリソースがコンピュータにありません
- コンピュータが AEP ファイルを開くために使用するドライバが古い
