## .USR ファイルとは何ですか?

「.usr」ファイル拡張子も Lowrance GPS デバイスに関連しています。特に、Lowrance GPS デバイスで使用される「ユーザー データ フォーマット」(USR フォーマット) として知られるフォーマットで GPS データを保存するために使用されます。

Lowrance GPS ユニットを使用する場合、ウェイポイント、トラック、ルートを「.usr」ファイルとして保存できます。これらのファイルには通常、緯度、経度、高度、タイムスタンプ、GPS アクティビティに関連するその他のデータなどの情報が含まれています。

Lowrance GPS デバイスでの .usr ファイルの一般的な使用法をいくつか示します。

- **ウェイポイント:** ウェイポイントは、ランドマーク、お気に入りの釣り場、興味のある場所など、GPS 上にマークされた特定の場所です。これらの場所を .usr ファイルとして保存し、インポート、エクスポート、または他の Lowrance GPS ユーザーと共有することができます。

- **トラック:** トラックは、GPS の移動の記録された経路を表します。トラック ログを .usr ファイルとして保存すると、後でルートを確認して分析したり、他のユーザーと共有したりできます。

- **ルート:** ルートは、GPS デバイスに作成して保存できる事前定義されたパスです。これらのルートは、将来の使用または共有のために .usr ファイルとして保存することもできます。

Lowrance GPS デバイス上の .usr ファイルを管理するには、通常、Lowrance の「Insight Planner」や「Lowrance GPS Utility」などのソフトウェアを使用して、GPS データをインポート、エクスポート、操作します。

## USR ファイル形式 - 詳細情報

Lowrance iFinder GPS デバイスでは、.usr ファイルが作成され、デバイスに挿入されている MultiMediaCard (MMC) メモリ カードに保存されます。これらのファイルには、ウェイポイント、トラック、ルートなどのユーザー データが含まれています。

.usr ファイルを MMC からコンピュータに転送するには、次の手順に従います。

- **MMC を取り外します:** MultiMediaCard (MMC) を Lowrance iFinder GPS デバイスから慎重に取り外します。

- **MMC をコンピュータに挿入します:** 適切な MMC カード リーダーを使用して、メモリ カードをコンピュータのカード リーダー スロットに挿入します。

- **.usr ファイルの検索:** MMC がコンピュータに認識されると、MMC に保存されているファイルにアクセスできるようになります。 GPS データが含まれている .usr ファイルを探します。

- **GPSBabel による変換:** .usr ファイルを別の GPS 形式に変換するには、さまざまな GPS ファイル形式を処理するための無料のオープンソース ツールである GPSBabel を使用できます。 GPSBabel は幅広い入出力形式をサポートしているため、.usr ファイルを他の GPS ソフトウェアまたはデバイスと互換性のある形式に変換できます。

GPSBabel は公式 Web サイト (http://www.gpsbabel.org/) からダウンロードして、コンピューターにインストールできます。インストールすると、ソフトウェアのコマンドライン インターフェイスまたはグラフィカル ユーザー インターフェイス (利用可能な場合) を使用して変換を実行できます。

たとえば、.usr ファイルを GPX 形式に変換する場合は、次のようなコマンドを使用できます。

```
gpsbabel -i lowranceusr -f input.usr -o gpx -F output.gpx
```

上記のコマンドは、入力ファイル「input.usr」を Lowrance USR 形式で読み取り、出力ファイル「output.gpx」を GPX 形式で書き込むように GPSBabel に指示します。

- **変換されたファイルのインポート/使用:** 変換後、GPS データは新しい形式 (GPX など) になり、その形式をサポートする他の GPS ソフトウェアまたはデバイスで使用できます。
