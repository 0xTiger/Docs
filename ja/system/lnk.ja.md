## .LNK オプション番号##

LNK ファイルは、Mac システムの ID に似ており、元の画像ドキュメント フォルダーまたはプログラムへの接続として機能する Windows の代替または「リンク」です。これには、ショートカット先のタイプ、位置、ファイル名、およびターゲット ドキュメントを開くアプリケーションと追加のショートカット キーが含まれます。

Windows で、ファイル、フォルダー、または実行可能プログラムをまっすぐにして、[ショートカットの作成] を選択します。ファイル形式の場所と「Beginning」ディレクトリは、LNK ファイルの 2 つの基本的な機能です。 LNK ファイルのファイル形式はマスクされており、曲線の矢印はそれらがショートカットであることを示しています。

## LNK ファイル形式 ##

通常、LNK ファイル形式には、宛先ファイルと同じアイコンがありますが、ファイルが別の場所を指していることを示す小さな丸まった矢印が追加されています。ショートカットをダブルクリックすると、ユーザーが実際のファイルをダブルクリックしたかのように動作します。

アプリケーションのショートカットを含む LNK ファイルには、プログラムの実行方法に影響するプロパティが含まれている場合があります。属性を変更するには、ショートカット ファイルを右クリックして [**プロパティ**] を選択し、[ターゲット] フィールドを変更します。

ファイル拡張子ではなく、LNK ファイルは Windows エクスプローラーの拡張子です。端末の拡張機能として、.lnk ドキュメントは Windows エクスプローラーでファイルを置き換えるためにのみ使用でき、Windows エクスプローラーでは、ローカル ドキュメントへのショートカットとして機能する以外の目的があります。これらのファイルも文字「L」で始まります。

ショートカットは作成時に特定のファイルやディレクトリにリンクしますが、ターゲットが別の場所に変更されると操作できなくなる場合があります。エクスプローラーは、開いたときにデッド ターゲットを指すショートカット フォルダーの修復を開始します。


## 技術仕様 ##

[認識されたファイルの種類の拡張子を非表示にする] フォルダー表示設定がオフになっている場合のみ、Windows はドキュメント ショートカットの .lnk ドキュメント拡張子を表示しません。お勧めしませんが、HKEY_CLASSES_ROOT\lnk ファイルの Windows レジストリ項目から「NeverShowExt」プロパティを削除することで、ファイル拡張子を表示できるようにすることができます。

これを行うには、次の手順を実行します。

*タスクバーの検索フィールドに「Regedit」と入力し、プログラムを選択して、「登録エディター」を開きます
* アプリケーションで、Computer\HKEY HKEY_CLASSES_ROOT\lnkfile の場所に移動します。
*アイテムを右クリックして[エクスポート]を選択して、アイテムのバックアップを作成します
* 「NeverShowExt」属性を選択して削除
* Windows を再起動する必要があります
