## REG ファイルとは何ですか?

REG ファイルは、Windows レジストリ データのインポートまたはエクスポートに使用されるファイル形式です。 Windows レジストリは、Windows オペレーティング システムおよびインストールされているアプリケーションの構成設定とオプションを保存する階層データベースです。レジストリには、ユーザー設定、アプリケーション設定、ハードウェアおよびソフトウェア構成データなどの情報が含まれています。

REG ファイルは通常、「.reg」ファイル拡張子を持ち、特定の形式の一連のレジストリ エントリと値を含むプレーン テキスト ファイルです。この形式は、ファイルをレジストリ ファイルとして識別するヘッダー セクションと、それに続くレジストリ エントリを表す一連のキーと値のペアで構成されます。

## REG ファイル形式 - 詳細情報

reg ファイル形式の例を次に示します。

```
[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Run]
"Example"="C:\\Example.exe"

[HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced]
"Hidden"=dword:00000001
```

ファイルの最初の行は、レジストリ エディタのバージョンを指定します。次の行には、キー パスの後に値の名前と値のデータが続く形式のレジストリ エントリが含まれています。この例では、reg ファイルには 2 つのエントリが含まれています。1 つは「Example.exe」というプログラムを Windows スタートアップ リストに追加するエントリ、もう 1 つは Windows エクスプローラの詳細設定の「Hidden」値を「true」に設定するエントリです。

Reg ファイルは、メモ帳などのテキスト エディタを使用して作成および編集できます。これらは、バックアップと復元の目的だけでなく、複数のコンピュータを同じレジストリ設定で構成するためによく使用されます。

## Windows レジストリのインポートまたはエクスポート

REG ファイルは、Windows レジストリからデータをインポートまたはエクスポートするために使用されるファイルの種類です。 Windows レジストリは、Windows オペレーティング システムおよびインストールされているアプリケーションの構成設定とオプションを保存する階層データベースです。レジストリには、ユーザー設定、アプリケーション設定、ハードウェアおよびソフトウェア構成データなどの情報が含まれています。

Reg ファイルは、レジストリ エントリの作成または変更に使用でき、多くの場合、バックアップと復元の目的だけでなく、複数のコンピュータを同じレジストリ設定で構成するために使用されます。 reg ファイルを使用して、Windows レジストリに新しいレジストリ エントリを追加する方法は次のとおりです。

1. メモ帳などのテキスト エディタを使用して新しいテキスト ファイルを作成します。
2. レジストリ エントリを正しい形式で入力します。この形式は、ファイルをレジストリ ファイルとして識別するヘッダー セクションと、それに続くレジストリ エントリを表す一連のキーと値のペアで構成されます。以下に例を示します。

```
Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\Software\Example]
"Setting1"="Value1"
```

これにより、現在のユーザーのレジストリ ハイブの "Software" キーの下に "Example" という新しいキーが作成され、"Setting1" の値が "Value1" に設定されます。

3. ファイル拡張子 .reg を付けてファイルを保存します。
4. .reg ファイルをダブルクリックして、新しいレジストリ エントリを Windows レジストリに追加します。

レジストリにエントリを追加するかどうかを確認するメッセージが表示されます。確認すると、新しいエントリがレジストリに追加され、レジストリ エディター ツールを使用して確認できます。
