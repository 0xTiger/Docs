## .ICNS オプション番号は何ですか?

ICNS ファイルは、macOS でアプリケーション、フォルダー、その他の項目のアイコンを保存するために使用されるファイル形式です。これは、同じアイコンの複数の画像サイズと解像度を保持できるコンテナ ファイル形式です。

通常、ICNS ファイルには、macOS デバイス上のさまざまな表示サイズと解像度に対応するために、16x16 ピクセルから 1024x1024 ピクセルまでの複数の解像度の画像が含まれています。このファイル形式では、圧縮アルゴリズムを使用して画像のファイル サイズを削減します。

ICNS ファイルは、Icon Composer や Sketch などのさまざまなソフトウェア ツールを使用して作成でき、アプリケーション、フォルダー、ドキュメントなど、macOS システム上のさまざまなアイテムに割り当てることができます。 macOS システムで項目が選択されると、オペレーティング システムはアイコンの適切なサイズと解像度を ICNS ファイルから取得し、画面に表示します。

## ICNS ファイル形式 - 詳細情報

ICNS ファイルは、「アイコン ファミリ」と呼ばれる技術を使用して、同じアイコンの複数の画像サイズと解像度を保持できるコンテナ ファイルです。アイコン ファミリは画像リソースのグループで構成され、それぞれが同じアイコンを表しますが、サイズや解像度が異なります。アイコン ファミリ内の各画像リソースには、画像のサイズと解像度を識別する一意のタイプと ID があります。異なる画像サイズと解像度は、異なるタイプと ID を使用して同じ ICNS ファイル内に保存されます。

macOS システムで項目が選択されると、オペレーティング システムは、アイコンの種類と ID を使用して ICNS ファイルからアイコンの適切なサイズと解像度を取得し、画面に表示します。これにより、macOS システムは、ファイル サイズを小さくしながら、Retina ディスプレイなどのさまざまな解像度の画面に高品質のアイコンを表示できるようになります。

macOS の ICNS ファイルは、どちらもアイコンを保存するためのファイル形式として機能するという点で、Windows の .ICO ファイルに似ています。 ICNS ファイルと同様に、ICO ファイルには同じアイコンの複数の画像サイズと解像度を含めることができ、Windows デバイス上の異なる画面サイズと解像度でアイコンを表示するために使用されます。

ただし、ICNS ファイルと ICO ファイルの間にはいくつかの違いがあります。たとえば、ICNS ファイルは ICO ファイルよりも高解像度の画像を保存できるため、macOS デバイスの Retina ディスプレイなどの高解像度ディスプレイでの使用に適しています。さらに、ICNS ファイルにはアルファ チャネルを含む画像を含めることができ、ICO ファイルではサポートされていない透明な背景が可能になります。
