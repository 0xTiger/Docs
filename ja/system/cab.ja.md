## .CAB オプション番号##

拡張子が .cab のファイルは、システム ファイルのカテゴリに属する Windows キャビネット ファイルに属します。 [LZX](/compression/lzx/)、Quantum、[ZIP](/compression/zip/) などの圧縮データ アルゴリズムをサポートするバージョンの Microsoft Windows でアーカイブ ファイル形式で保存されるファイルです。 ）。このファイルは、ユーザーまたは開発者がソフトウェアのインストール データとファイルを格納して共有する場合に非常に役立ちます。これらのファイルに含まれる可逆データ圧縮とデジタル証明書の機能により、このファイルはそのようなファイルの保存と共有に最適です。デバイス インストーラー、SetUp API、AdvPak などのさまざまな Microsoft インストーラーをサポートしています。

## 簡単な歴史 ##

CAB ファイルは、Microsoft によって開発されたデータ圧縮ファイル タイプです。最初は「Diamond」と呼ばれていましたが、その後、「Cabinet」という単語の略で、CAB ファイルとして広く知られるようになりました。

## 技術仕様 ##

通常、CAB ファイルには最大 65535 個のフォルダーを含めることができ、さらに各フォルダーには最大 65536 個のファイルを含めることができます。 CAB ファイル ストレージ メカニズムは、各ファイルを個別に圧縮して保存するのではなく、各フォルダーを圧縮ブロックとして保存するため、時間とスペースの効率が高くなります。空のフォルダーを CAB アーカイブ フォルダーに格納することはできません。 CAB ファイルは Microsoft によって最初に開発され、InstallShield など、わずかに異なる形式のさまざまなインストーラーで使用されます。 CAB ファイルは、通常、自己解凍型プログラムに関連付けられています。 Microsoft CAB ファイルは、フォーマットの識別に役立つ独自のマーカーにより、簡単に認識できます。すべての Microsoft CAB ファイルの一意のマーカーは、4 語のプレフィックス MSCF です。このコードを見ると、ユーザーは Microsoft CAB ファイルを他のファイルと簡単に区別し、それに応じて圧縮プログラムまたはバージョンで使用できます。ファイルは、より多くのソフトウェア インストール データで圧縮するか、適切なソフトウェアを使用して現在のデータを解凍することができます。
