## .VDX オプション番号

Microsoft Visio で作成され、XML 形式で保存された図面またはチャートには、.VDX 拡張子が付きます。 Visio 図面 XML ファイルは、Microsoft が開発した Visio ソフトウェアで作成されます。 Microsoft Visio には、プレゼンテーションやドキュメントで使用できるビジュアル ドキュメントを生成する機能があります。 Visio 図面 XML ファイルには、ビジュアル オブジェクトとビジュアル要素のメタデータの詳細が含まれています。これらのビジュアル要素にテキストを追加することもできます。ビジョン ドローイング XML ファイル。これらの Visio 図面 XML ファイルは、XML ベースのフォーマット標準と画像データ エンコーディング仕様に統合されており、その内容を Microsoft Visio ソフトウェアによって VDX ファイル形式でレンダリングおよび保存できます。これらの Visio 図面 XML ファイルには、拡張子 .vdx が付加されます。 VDXファイル形式を開くことができるアプリケーションは次のとおりです。

* マイクロソフト ビジオ
* CS Odessa ConceptDraw PROJECT for MAC

## 簡単な歴史 ##

当初、Visio は、1992 年にバージョン 1.0 で Shapeware Corporation によってスタンドアロン製品として作成されました。 Shapeware Corporation は、1995 年に Visio Corporation に社名を変更し、市場での認知度を利用しました。 Microsoft は、2000 年に Vision を Microsoft Office アプリケーションとしてブランド変更しました。 Visio for Enterprise Architects は、2003 年と 2005 年に Visual Studio .NET と共に導入されました。Visio 2007 は 2006 年 11 月 30 日にリリースされました。リボンは、2010 年に Microsoft Visio によってユーザー インターフェイスに導入されました。

## Visio ファイル形式 ##

2013 年から、visio は新しいファイル形式 .vsdx を導入しました。これは、.vsd (Visio バイナリ ファイル形式) および .vdx (Visio XML 図面ファイル形式) として知られる他の 2 つのファイル形式を置き換えるものです。 .vsdx により、Visio ファイルの相互運用性が向上しました。これは、サードパーティ製ソフトウェアも Visio ファイルをファイル形式レベルで操作できるようになったためです。新しいファイル形式は、Microsoft SharePoint Server の Visio Services でサポートされており、SharePoint Server に発行するための "中間" ファイル形式は必要ありません。次の拡張子を持つ visio を構成するファイルの種類がいくつかあります。

* .vsdx (Visio 図面)
* .vsdm (Visio マクロ有効図面)
* .vssx (Visio ステンシル)
* .vssm (Visio マクロ有効ステンシル)
* .vstx (Visio テンプレート)
* .vstm (Visio マクロ有効テンプレート)

## 特徴 ＃＃

Visio には、ネットワーク ダイアグラム、フローチャート、フロア プラン、エンジニアリング デザインなどの作成に役立つさまざまなテンプレートと図形が含まれています。さらに、プロセス モデリングとデータ視覚化の作成にも役立ちます。 Microsoft Visio は、使い方がシンプルでユーザー フレンドリーです。 PowerPoint、Word、Publisher などの他の多くのアプリケーションにさらに追加できる、効果的なビジュアル ダイアグラムの作成に役立ちます。

Microsoft Visio は、Unified Modeling Language (UML) 2.4、Business Process Model、および Notation 2.0 もサポートしています。さらに、Visio は、SharePoint 2010 と 2013 の両方のワークフローの発行もサポートしています。 Visio には、図形をリアルタイム データにリンクすることで、ダイアグラムを動的にする機能もあります。専門家のチームは、同じ図面で同時に共同作業を行うことができ、Visio と Visio Services の両方でレビュー担当者のコメントを表示することもできます。さらに、システムに Visio がインストールされていない場合でも、Office 365 または SharePoint を介して図を表示できます。

## Microsoft Visio エディション ##

1. **Microsoft Visio Standard**: Standard エディションは、豊富な組み込みステンシルまたはテンプレートを備えた強力な作図プラットフォームを必要とする個人向けに設計されています。個人は、複雑な情報を簡単で単純な図に単純化できます。 Microsoft Visio Standard には、ビジネス、ネットワーク図、グラフ、フローチャート、およびその他のさまざまな多目的図のテンプレートが含まれています。
1. **Microsoft Visio Professional**: Professional エディションは、専門的で用途の広い図を使用して複雑な情報を簡素化できる個人およびチーム向けに構築されています。基本的な機能には Visio Standard 2013 が含まれますが、追加および更新された図形、テンプレート、およびスタイルが追加され、同じ図で同時に作業するさまざまな人々の作業を含むチーム作業をより高度にサポートします。さらに、Visio Professional 2016 には、ビジネスおよびエンジニアリング図、プロセス図、地図、フロア プランなどのテンプレートがさらに追加されています。
1. **Visio Pro for Office 365**: これは Visio 2016 の特別版で、Office 365 からアクセスできます。MS Visio 2016 Professional と同じ機能を備えています。

以下は、MS Visio のいくつかの基本的な機能/利点です。

* プロの作図ツールの助けを借りて、複雑さが簡素化されます

** クイック シェイプのミニ ツールバー

* Visio で SharePoint ワークフローをモデル化し、SharePoint Server 2010 で実行および監視する
