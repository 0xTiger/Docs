## .OLM オプション番号

拡張子が .olm のファイルは、Mac オペレーティング システム用の Microsoft Outlook ファイルです。 OLM ファイルには、電子メール メッセージ、ジャーナル、カレンダー データ、およびその他の種類のアプリケーション データが格納されます。これらは、Windows オペレーティング システムの Outlook で使用される [PST](/email/pst/) ファイルに似ています。ただし、Outlook for Mac で作成された OLM ファイルは、Outlook for Windows では開くことができません。 OLM ファイル形式は、Outlook のローカル データを保存するために Outlook 2011 で導入されました。 1 つの OLM ファイルに複数のメールボックスを含めることができ、フォルダーとサブフォルダーの形式でデータを整理します。 OLM ファイルは、Microsoft Outlook 365 および Outlook for Mac で開く/読み込むことができます。

## OLM ファイル形式

OLM は、Application/octet-stream MIME タイプのバイナリ ファイルです。 OLM ファイル形式の技術仕様は Microsoft から公開されておらず、開発者が参照できる形式の詳細はありません。ただし、OLM ファイル フォーマットを読み取って PST ファイルに書き込み、Aspose.Email などの Microsoft Windows オペレーティング システムで使用できる API がいくつかあります。
