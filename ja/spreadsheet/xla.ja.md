## .XLA オプション番号

拡張子が .xla のファイルは、モジュール、追加機能、およびその他のツールを Microsoft Excel に追加する Microsoft Excel アドイン ファイル形式です。アドインは、Excel スプレッドシートに追加機能を提供し、バージョン 2007 以降でサポートされています。 XLA ファイルの MIME メディア タイプは application/vnd.ms-excel です。これらは、Windows および MacOS で使用可能な Microsoft Excel を使用して開くことができます。

## XLAファイル形式

XLA ファイルは、バイナリ データ形式に基づいています。ただし、そのファイル形式の仕様に関する詳細情報は入手できません。一般的な前提として、XLA は [[MS-XLS] - Excel Binary File Format Structure](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) に従っています。
