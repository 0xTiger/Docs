## Wat is een PFA-bestand?

Een bestand met de extensie .pfa is een Printer Font ASCII-versie van Adobe's Type 1-lettertype en bevat glyph-gegevens van het lettertype. Het bestand heeft een door mensen leesbaar formaat en kan in elke teksteditor worden geopend. PostScript-interpreter kan dit type lettertype als geheel gebruiken voor zijn werking. Printer Font Binary (PFB)-versie van Adobe's Type 1-lettertype bevat dezelfde informatie. Unline PFA, het is echter in binaire bestandsindeling en is niet leesbaar voor mensen.
