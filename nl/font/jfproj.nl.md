## Wat is een JFPROJ-bestand?

Een bestand met de extensie .jfproj is een op JSON gebaseerd aangepast lettertype dat is gemaakt met de Microsoft Font Maker-toepassing op Windows 10. Deze kunnen worden gemaakt door een elektronische pen te gebruiken met apparaten met een aanraakscherm, zoals tablets, hoewel andere computerrandapparatuur zoals een muis ook kan worden gebruikt voor het geval dat van computers.
