## Wat is een MDS-bestand?

De MDS (Media Descriptor Sidecar File) is een bestand dat is gekoppeld aan de software Alcohol 120% en Daemon Tools, die beide worden gebruikt voor het maken en beheren van virtuele cd- en dvd-stations. Het MDS-bestand bevat informatie over de indeling van de gegevens op de schijf, inclusief de locatie van alle bestanden en de structuur van de schijf. Hierdoor kan de virtuele schijf het gedrag van een fysieke schijf emuleren, zodat de software de gegevens op de virtuele schijf kan lezen alsof het een echte schijf is.

When you want to mount an image file to a virtual drive, the MDS file is used along with the corresponding image file in a format such as ISO, BIN or CUE. The MDS file contains a description of the layout of the data on the disc, and the virtual drive software uses this information to create a virtual disc. This allows you to use the software as if you were using a physical disc, without having to physically insert the disc into the drive.

## Relatie met alcohol 120%

Alcohol 120% is een populaire software voor het maken en beheren van virtuele cd- en dvd-stations, en gebruikt het MDS-bestandsformaat om schijfimages op te slaan en te openen. Het MDS-bestandsformaat is eigendom van Alcohol 120% en kan alleen worden geopend en gebruikt door de Alcohol 120%-software of andere software die dit ondersteunt.

## Hoe open je een MDS-bestand?

Om een MDS-bestand in Alcohol 120% te openen, moet de software op uw computer zijn geïnstalleerd. Zodra je Alcohol 120% hebt geïnstalleerd, kun je het MDS-bestand openen door het volgende te doen:

1. Lancering Alcohol 120%.
2. Klik op het menu Bestand en selecteer Openen.
3. Navigeer naar de locatie van het MDS-bestand op uw computer en selecteer het.
4. Het MDS-bestand wordt nu geopend in Alcohol 120% en u wordt gevraagd het bijbehorende afbeeldingsbestand te selecteren (zoals ISO, BIN of CUE).
5. Nadat u het afbeeldingsbestand hebt geselecteerd, koppelt Alcohol 120% de afbeelding aan een virtuele schijf.

Als alternatief kunt u ook een MDS-bestand openen door erop te dubbelklikken, als Alcohol 120% is ingesteld als het standaardprogramma voor het openen van MDS-bestanden. U kunt ook andere software gebruiken die het MDS-formaat ondersteunt, zoals Daemon Tools, Virtual CloneDrive, Virtual Drive Manager en MagicISO.
