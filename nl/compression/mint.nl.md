## Wat is een MINT-bestand?

Een MINT-bestand is een installatiepakketbestand dat wordt gebruikt door de [Linux Mint](https://linuxmint.com/)-distributie. Het bevat gecomprimeerde broncodebestanden die worden uitgepakt en gebruikt om de installatie van Linux Mint OS te starten. Linux Mint OS heeft een gebruiksvriendelijke GUI (Graphical User Interface) en was de eerste met een elegant GUI-ontwerp. MINT-bestanden worden gedecomprimeerd en samen met het installatiepakket in een tijdelijke map opgeslagen. De installatie wordt uitgevoerd vanuit dezelfde directory voor de installatie van Linux Mint OS.
