## Wat is een STR-bestand?

Een STR-bestand is een opnamebestand gemaakt door YouTube voor een livestream die werd gehost door het streamingevenement van de gebruiker. Het bestand wordt automatisch door YouTube op de backend opgeslagen zodra de livestreaming voorbij is. Een livestreamvideo kan naar meerdere kanalen worden uitgezonden, zodat het publiek via reacties met de uitgever kan communiceren. Deze SRT-bestanden zijn toegankelijk via de YouTube livestream API die toegang biedt tot de servers van YouTube.

## STR-bestandsindeling

YouTube slaat STR-bestanden op de server op in binair bestandsformaat. De interne bestandsstructuurdetails van STR-bestanden zijn echter niet openbaar beschikbaar.
