## Wat is een CC-bestand?

Een CC-bestand is een type broncodebestand dat code bevat die is geschreven in de programmeertaal C++. Het kan alle code bevatten die nodig is voor een enkel programma of een deel van de code die deel uitmaakt van een groter programmeerproject dat uit meerdere CC-bestanden bestaat. Om de code in het CC-bestand te wijzigen of te bekijken, gebruiken ontwikkelaars vaak een broncode-editor. Na het bewerken compileren ze de code met behulp van een compiler om deze om te zetten in machinecode, die het besturingssysteem van de computer kan uitvoeren om het programma uit te voeren.

## CC-bestandsindeling - Meer informatie

De bestandsextensie .cc wordt al tientallen jaren gebruikt voor C++-broncodebestanden. De taal C++ werd eind jaren zeventig ontwikkeld door Bjarne Stroustrup, die op dat moment bij Bell Labs werkte. Stroustrup wilde de programmeertaal C uitbreiden met ondersteuning voor objectgeoriënteerde programmeerconcepten die volgens hem essentieel waren voor het bouwen van grote softwaresystemen.

De eerste versie van C++ werd uitgebracht in 1983 en werd snel populair onder ontwikkelaars die de krachtige functies en flexibiliteit ervan op prijs stelden. Naarmate C++ op grotere schaal werd gebruikt, werd het noodzakelijk om coderingsconventies en bestandsnaamgevingsconventies vast te stellen om grote codebases te helpen organiseren en beheren.

De C++ Style Guide van Google, waarin het gebruik van de .cc-bestandsextensie voor C++-broncodebestanden wordt aanbevolen, werd voor het eerst gepubliceerd in 2003. Sindsdien is de gids periodiek bijgewerkt om veranderingen in de C++-taal en ontwikkelingspraktijken van de bredere programmeergemeenschap weer te geven.

Tegenwoordig blijft de .cc-bestandsextensie een populaire keuze voor C++-broncodebestanden en wordt deze herkend door veel populaire ontwikkeltools en platforms.

## Hoe open je een CC-bestand?

Veel broncode-editors, waaronder multi-platform tools zoals Microsoft Visual Studio Code en Code Blocks, kunnen worden gebruikt om CC-bestanden te openen en te bewerken. Deze editors bieden handige functies zoals syntaxisaccentuering en automatisch aanvullen, waardoor ze een uitstekende keuze zijn voor het werken met CC-bestanden.

U kunt CC-bestanden ook openen met elke teksteditor, bijvoorbeeld Kladblok, Kladblok++ enz.

## CC compileren met C++ compilers

Het compileren van een CC-bestand met de C++-compiler omvat verschillende stappen. De eerste stap is het installeren van een compatibele C++-compiler voor uw besturingssysteem en ontwikkelomgeving. Er zijn veel verschillende compilers beschikbaar, waaronder GCC, Clang en Microsoft Visual C++.

Zodra u de C++-compiler hebt geïnstalleerd, is de volgende stap het openen van de terminal of opdrachtprompt en navigeren naar de map waar het CC-bestand zich bevindt. Vanaf hier kunt u de compiler uitvoeren door een opdrachtregelprogramma uit te voeren dat door de compiler wordt geleverd. De specifieke opdracht om het CC-bestand te compileren kan variëren, afhankelijk van de compiler die u gebruikt. Als u bijvoorbeeld een CC-bestand wilt compileren met GCC, voert u een opdracht uit die lijkt op:

```
g++ -o myprogram myfile.cc
```
