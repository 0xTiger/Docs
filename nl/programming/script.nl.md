## Wat is een SCRIPT-bestand?

Een SCRIPT-bestand is een bronbestand dat scripts in elke scripttaal bevat. Dit maakt het een gegeneraliseerd scriptbestand dat variabelen en functies bevat. Aangezien elke scripttaal anders kan zijn, moet het script dat in een SCRIPT-bestand is geschreven, worden uitgevoerd met de juiste scriptengine.

## SCRIPT-bestandsindeling - Meer informatie

SCRIPT-bestanden worden op schijf opgeslagen in platte tekstbestanden. Deze kunnen worden geopend en bewerkt met elke teksteditor, zoals Microsoft Notepad, Notepad++ en Apple TextEdit. Aangezien de scripttaal anders kan zijn, kan deze worden uitgevoerd door de bestandsextensie te hernoemen naar de standaardextensie van de respectieve scripttaal. Er zijn bijvoorbeeld verschillende veelgebruikte scripttalen zoals JavaScript ([.JS](/nl/web/js/)), VBScript ([.VBS](/nl/programming/vbs/)), AppleScript (.SCPT), PHP ([ .PHP](/nl/programming/php/)), en ASP ([.ASP](/nl/web/asp/)).
