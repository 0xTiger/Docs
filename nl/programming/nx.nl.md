## Wat is een NX-bestand?

Een NX-bestand is een videogamebestand dat is gemaakt voor gebruik met de virtuele gameconsole **LowRes NX**. Het is geschreven in de programmeertaal BASIC en bevat de broncode van het spel die definieert hoe het spel functioneert. U kunt uw eigen NX-bestanden maken voor gebruik met de app die kunnen worden geladen en gebruikers het spel kunnen spelen.

U kunt een NX-bestand openen in een teksteditor, LowRes NX of een broncode-editor zoals Microsoft Visual Studio Code.

## NX-bestandsformaat

Een NX-bestand wordt op schijf opgeslagen als gewoon tekstbestand dat de spelinstructies bevat. Elk NX-bestand heeft de broncode van het spel die bestaat uit meerdere cartridge-ROM-vermeldingen. Deze ROM-items bevatten informatie over spelcomponenten, zoals afbeeldingen, niveaukaarten, audio en andere niet-broncodegegevens voor het spel. De ROM-items van een programma kunnen worden toegevoegd of bewerkt met behulp van NX-bewerkingsprogramma's die bij de LowRes NX worden geleverd.

## Hoe NX-bestanden maken?

### Maak NX-bestanden op pc

U kunt NX-bestanden op pc maken met behulp van:

* Teksteditors - zoals Microsoft Notepad, Notepad++ of Github Atom
* Broncode-editors - zoals Microsoft VIsual Studio Code

### Maak NX-bestanden op iOS

LowRes NX heeft een iOS-versie die door ontwikkelaars kan worden gebruikt om vanuit deze app NX-bestanden te maken.
