## Wat is een PHP-bestand?
Een bestand met de extensie .php verwijst naar open source programmeertaal, die wordt gebruikt om server-side scripts te schrijven die op een webserver moeten worden uitgevoerd. Het is de meest gebruikte webscripttaal die doorgaans wordt gebruikt voor de ontwikkeling van grootschalige webapplicaties. Het grootste blogsysteem op het web, namelijk WordPress en het grootste sociale netwerk Facebook, zijn gebaseerd op PHP-taal. Het is compatibel met verschillende platforms zoals Windows, Linux, Unix, Mac OS X, enz. Het draait ook met succes op bijna alle servers die tegenwoordig worden gebruikt, zoals Apache, IIS, enz. PHP ondersteunt een breed scala aan databases, waaronder MySQL.

## PHP-bestandsindeling ##

Een enkele PHP kan tekst, CSS, [HTML](/nl/web/html/), JavaScript en PHP-code bevatten. PHP-code wordt uitgevoerd op de server en het resultaat wordt als gewone HTML naar de browser teruggestuurd. PHP-bestanden kunnen met elke teksteditor worden geopend en ter plekke worden bewerkt, hoewel toepassingen zoals Adobe Dreamweaver en Eclipse PHP-ontwikkelingstools een handige manier bieden om PHP-code te schrijven en aan te passen.

### Syntaxis ###

Een PHP-script wordt uitgevoerd op de server en het eenvoudige HTML-resultaat wordt teruggestuurd naar de browser. Een PHP-script begint met<?php and ends with ?>

### PHP-voorbeeld ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
