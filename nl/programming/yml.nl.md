## Wat is een YML-bestand?

YML is een tekstbestand in de taal voor gegevensserialisatie, [YAML](/nl/programming/yaml/) (Yet Another Markup Language), dat vaak wordt gebruikt om configuratiebestanden te maken. Het kan werken in combinatie met elke programmeertaal, waaronder C/C++, Ruby, Python, Java, PHP, C# en andere. Omdat het onafhankelijk is van een specifieke programmeertaal, is het open, interoperabel en door mensen leesbaar. YML-bestanden kunnen worden geopend en bewerkt met algemene teksteditors zoals Microsoft Notepad, Notepad++, Github Atom, Apple TextEdit en Visual Studio IDE.

## YML-bestandsindeling - Meer informatie

YML-bestanden slaan applicatie-instellingen op als configuratie. De basisstructuur is een map die ook een directory of hash kan worden genoemd. Vermeldingen in een YML-bestand bestaan uit een sleutel- en waardepaar. Meestal worden de .yml-configuratiebestanden opgeslagen als `configure.yml`, hoewel sommige programmeertalen, zoals Ruby on Rails, de configuratie van de databaseverbinding opslaan in `database.yml`-bestanden. [YML/YAML bestandsformaatspecificaties](https://yaml.org/spec/1.2.2/) leggen de volledige taalstructuren en syntaxisgerelateerde informatie uit.
