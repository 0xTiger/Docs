## Wat is een GROOVY-bestand?

Een GROOVY-bestand is een broncodebestand dat is geschreven in de programmeertaal Groovy. Code die in een GROOVY-bestand is geschreven, is vergelijkbaar met de objectgeoriÃ«nteerde taal [Java](/nl/programming/java/), die het ontwerpen en ontwikkelen van applicaties gemakkelijk maakt. Een GROOVY-bestand wordt gecompileerd naar Java virtual machine (JVM) bytecode en is compatibel met andere Java-code en bibliotheken.

## GROOVY-bestandsindeling - Meer informatie

GROOVY-bestanden bevatten broncode geschreven in Groovy-syntaxis. Het kan code bevatten die zowel als programmeertaal als als scripttaal voor het Java-platform kan worden gebruikt. Groovy-taal heeft de functies die vergelijkbaar zijn met die van Python, Ruby en Smalltalk.
