## Wat is een ALB-bestand?

Een ALB-bestand is een bibliotheekbestand dat wordt gebruikt door de **Alpha Five** IDE-toepassing die wordt gebruikt voor het ontwikkelen van webtoepassingen. Het primaire doel is om informatie op te slaan voor de corresponderende Alpha Five [ADB-bestand](/nl/database/adb/). Dit omvat gegevens uit een datadictionary, zoals instellingen, scriptaanwijzers, menu's, door de gebruiker gedefinieerde functies, bitmaps, werkbalken en importdefinities.

ALB-bestanden kunnen worden geopend met **Alpha Software Alpha Anywhere**.

## ALB-bestandsindeling - Meer informatie

ALB-bestanden worden opgeslagen in de eigen bestandsindeling van Alpha File en hun interne bestandsindelingsstructuur is niet openbaar beschikbaar. Deze worden opgeslagen naast de algemene Alpha Five-database met ontwikkelaarsbibliotheek. Omdat het bibliotheken zijn, kunnen ALB-bestanden worden hergebruikt in andere Alpha Five-toepassingen.
