## Wat is een C3-bestand?
Een C3-bestand bevat broncode geschreven in de programmeertaal C3. C3 is een experimentele programmeertaal die tot doel heeft de beste eigenschappen van C++ en Python te combineren.

C3-broncodebestanden kunnen worden gemaakt met behulp van een teksteditor, bijvoorbeeld Kladblok of Sublime Text. Ze kunnen worden bewerkt en gecompileerd met behulp van de C3-compiler, die de C3-broncode vertaalt naar uitvoerbare code die op de computer kan worden uitgevoerd. De C3-compiler genereert machinecode die rechtstreeks op de CPU kan draaien en goede prestaties levert.

C3-broncodebestanden kunnen een verscheidenheid aan programmeerconstructies bevatten, waaronder functies, klassen, variabelen, lussen en conditionals. C3 bevat ook een aantal ingebouwde gegevenstypen, zoals gehele getallen, getallen met drijvende komma en tekenreeksen.

Om het C3-broncodebestand te kunnen compileren, moet de C3-compiler op uw computer zijn geïnstalleerd. Zodra de compiler is geïnstalleerd, kunt u deze vanaf de opdrachtregel uitvoeren en de locatie opgeven van het broncodebestand dat u wilt compileren. De compiler genereert een uitvoerbaar bestand dat u vervolgens op uw computer kunt uitvoeren.

## Wat is het formaat van het C3-bestand?

C3-broncodebestanden zijn platte tekstbestanden; hun formaat volgt de regels van platte tekstbestanden, waarbij elke regel een reeks tekens bevat waaruit de code bestaat. C3-code volgt een syntaxis die vergelijkbaar is met C++ en Python. C3-broncodebestanden bevatten doorgaans functies, klassen, variabelen, lussen, conditionals en andere programmeerconstructies. De code in het C3-bestand is georganiseerd in een reeks instructies, waarbij elke instructie doorgaans één regel van het bestand beslaat.

## Wat zijn de kenmerken van de programmeertaal C3?

Enkele kenmerken van de programmeertaal C3 zijn dat wel

- Eenvoud
- Sterk typen
- Object georiënteerd programmeren
- Merkloos product
- Afvalinzameling
- Metaprogrammering
- Interoperabiliteit

## Hoe open je een C3-bestand?

U hebt een teksteditor of een geïntegreerde ontwikkelomgeving (IDE) nodig om het C3-bestand te openen. Enkele populaire opties zijn Visual Studio Code, Sublime Text, Atom en IntelliJ IDEA.
