## Wat is een F95-bestand?

Een bestand met de extensie .f95 geeft doorgaans aan dat het een Fortran 95-broncodebestand is. Fortran is een imperatieve programmeertaal voor algemeen gebruik die bijzonder geschikt is voor numeriek en wetenschappelijk computergebruik. Fortran 95 is een verbeterde versie van Fortran 90, waarin verschillende nieuwe functies en verbeteringen worden geïntroduceerd.

Fortran-broncodebestanden bevatten doorgaans programma-instructies geschreven in de Fortran-programmeertaal; als je een bestand hebt met de extensie .f95, is het waarschijnlijk bedoeld om te worden gecompileerd met behulp van een Fortran-compiler; Compilatie is het proces waarbij voor mensen leesbare broncode wordt vertaald naar machinaal leesbare binaire code die door een computer kan worden uitgevoerd.

## Hoe het F95-bestand openen?

Als u wilt werken met of een programma wilt uitvoeren in het .f95-bestand, heeft u een Fortran-compiler nodig; Populaire Fortran-compilers zijn onder meer GNU Fortran (gfortran), Intel Fortran Compiler (ifort) en andere. U kunt Fortran-programma's compileren en uitvoeren met behulp van de juiste opdrachten van de compiler.

Als u bijvoorbeeld gfortran hebt geïnstalleerd, kunt u de volgende opdrachten in de terminal gebruiken om een Fortran-programma te compileren en uit te voeren:

```
# Compile the Fortran program
gfortran -o my_program my_program.f95

# Run the compiled program
./my_program
```

Als u alleen de inhoud van het F95-bestand wilt bekijken: het is een tekstbestand, dus u kunt het met elke teksteditor openen.
