## Wat is een RIS-bestand?

Een RIS-bestand is een tagbestandsindeling voor de uitwisseling van bibliografische gegevens door citatieprogramma's. Het bevat meerdere regels citatiegegevens, bestaande uit een code van twee tekens en een bijbehorende waarde, gescheiden door een koppelteken. Citaatinformatie in een RIS-bestand bevat gegevens zoals auteur, titel, publicatiedatum, trefwoorden, uitgever, samenvatting, auteuradres, enz. RIS-bestanden zijn leesbaar voor mensen en kunnen worden geopend met elke teksteditor zoals Microsoft Notepad, Notepad++, en Apple Teksteditor.

## RIS-bestandsindeling - Meer informatie

RIS-bestanden worden opgeslagen als tekstbestand zonder opmaak. Elke regel in een RIS-bestand bestaat uit een code van twee letters (ook wel tags genoemd), gevolgd door twee spaties en een koppelteken, en de bijbehorende waarde. Vanwege tags staan RIS-bestanden bekend als het tagged-formaat voor de weergave van bibliografische citaten. Voorbeeldrecord in een RIS-bestand is zoals hieronder weergegeven.

## RIS-bestandsvoorbeeld

Het volgende voorbeeld uit een Wikipedia-artikel toont de inhoud van het RIS-bestand voor een artikel: Claude E. Shannon. Een wiskundige theorie van communicatie, Bell System Technical Journal, 27:379-423, juli 1948.


|RIS Voorbeeld|
---|
|TY - JOUR|
|AU - Shannon, Claude E.|
|PY - 1948|
|DA - juli|
|TI - Een wiskundige theorie van communicatie|
|T2 - Bell systeem technisch journaal|
|SP - 379|
|EP - 423|
|VL - 27|
|ER - |

In dit voorbeeld is 'TY' het type referentie en is 'ER' het einde van de referentie.
