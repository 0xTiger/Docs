## Wat is een XLA-bestand?

Een bestand met de extensie .xla is een bestandsindeling van Microsoft Excel Add-Ins die modules, extra functies en andere hulpmiddelen aan Microsoft Excel toevoegt. De Ad-Ins bieden extra functionaliteit aan Excel-spreadsheets en worden ondersteund in versie 2007 of later. XLA-bestanden hebben mime-mediatype application/vnd.ms-excel. Deze kunnen worden geopend met Microsoft Excel dat zowel voor Windows als voor MacOS beschikbaar is.

## XLA-bestandsindeling

XLA-bestanden zijn gebaseerd op binaire gegevensindeling. Er is echter geen verdere informatie over de specificaties van het bestandsformaat beschikbaar. Algemene aanname is dat XLA de [[MS-XLS] - Excel Binary File Format Structure](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) volgt.
