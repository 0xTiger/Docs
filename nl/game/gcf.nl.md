## Wat is een GCF-bestand?

Een .gcf-bestand is een gamecachebestand dat wordt gebruikt door het Steam-gamingplatform. Het bevat spelgegevens zoals texturen, modellen en andere bestanden die door het spel worden gebruikt. Deze bestanden worden op de computer van de gebruiker opgeslagen en worden gebruikt om de hoeveelheid gegevens te verminderen die moet worden gedownload bij het updaten of installeren van een game. De .gcf-bestanden kunnen ook worden gebruikt om back-ups te maken van de installatie van een game of om een game over te zetten tussen computers.

GCF-bestanden kunnen worden gelezen en geschreven door de Open Source C++-programmeerbibliotheek, **HLLib**.

## GCF-bestandsindeling

GCF-bestanden worden als binaire bestanden op schijf opgeslagen. GCF werd oorspronkelijk gebruikt als een acroniem voor Grid Cache File (de vroege codenaam van Steam was Grid), maar later werd het gebruikt als Game Cache File.
