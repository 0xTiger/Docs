## Wat is een AMXX-bestand?

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. Het is een scriptplatform waarmee serverbeheerders de functionaliteit van hun servers kunnen aanpassen en uitbreiden. AMXX wordt gebruikt in combinatie met het MetaMod-plug-inbeheersysteem waarmee meerdere plug-ins tegelijkertijd zonder conflicten op een server kunnen worden uitgevoerd. Functies aangeboden door de AMXX-plug-in omvatten beheerdersopdrachten, gameplay-aanpassingen en serverbeheer.

## Hoe een AMXX-bestand openen?

U kunt AMXX-bestanden openen met behulp van de [AMX Mod X](https://www.amxmodx.org/) op Windows- en Linux-besturingssystemen.
