## Wat is een MDE-bestand?

Een .MDE-bestand is een gecompileerde versie van een Access-database (.mdb of .accdb). Wanneer u een Access-database compileert in een .MDE-bestand, kan deze niet langer worden bewerkt met de standaard Access-ontwerptools. Het primaire doel van het maken van een .MDE-bestand is het distribueren van een databasetoepassing zonder de originele broncode bloot te leggen.

## MDE-bestandsindeling

Een MDE-bestand wordt gemaakt door de VBA-code, formulieren, rapporten en andere objecten te compileren. Dit resulteert in het creëren van een binair MDE-bestandsformaat. Het resulterende .MDE-bestand kan worden gedistribueerd naar gebruikers die de applicatie kunnen uitvoeren, maar het ontwerp niet kunnen wijzigen of de originele code kunnen bekijken. MDE-bestand is eigenlijk de gecompileerde versie van een [.MDA-bestand](/database/mda/). Het distribueren van .MDE-bestanden kan helpen bij de bescherming van intellectueel eigendom door te voorkomen dat gebruikers toegang krijgen tot de broncode of deze kunnen wijzigen. Het kan ook de prestaties verbeteren naarmate de code wordt gecompileerd en geoptimaliseerd.
