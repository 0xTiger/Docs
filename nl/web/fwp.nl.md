## Wat is een FWP-bestand?

Een bestand met .fwp-bestand is een archiefbestand gemaakt met Microsoft Expression Web en bevat alle bestanden en mappen voor een ontwerp. Microsoft Expression Web is een algemeen softwareproduct voor webdesign van Microsoft en is gebaseerd op de [HTML](/nl/web/html/)-editor.
