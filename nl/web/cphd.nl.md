## Wat is een CPHD-bestand?

Een CPHD-bestand is een hypertextdocument dat wordt gebruikt door de CloudChan-preprocessorengine. Het bevat HTML en scripts om dynamische webpagina's en documenten te maken. Scripts in een CPHD-bestand lijken op de syntaxis die door Java wordt gebruikt. De CPHD-webpagina retourneert een HTML-document wanneer het wordt uitgevoerd op een CloudChan-proces. Het CPHD-formaat kan met elk XML-document werken.

## CPHD-bestandsindeling

CPHD-bestanden worden als platte tekstbestanden op schijf opgeslagen. Dit maakt het mogelijk om deze bestanden in elke teksteditor zoals Notepad of Notepad++ te openen.
