## Wat is een DOCMHTML-bestand?

Een DOCMHTML-bestand is een oude variant van webarchief die is opgeslagen door eerdere versies van Microsoft Word. Het wordt niet meer ondersteund door de nieuwste versie van Microsoft Word. Een DOCMHTML-bestand bevat een HTML-weergave van het originele .doc Word-document. Het kan zowel in Microsoft Word als in een webbrowser worden geopend. Wanneer een DOCMHTML wordt gepubliceerd, kan deze worden gehost op een webserver die toegankelijk is voor gebruikers via een webverzoek. Een ander soortgelijk formaat is [DOCHTML](/nl/web/dochtml/) dat ook is stopgezet.

## DOCMHTML-bestandsindeling - Meer informatie

DOCMHTML-bestanden werden gebruikt om te worden opgeslagen in HTML-indeling. Het kan worden geopend in een teksteditor zoals Notepad en Notepad++. Nadat het als verouderd was gemarkeerd, introduceerde Microsoft Word de opties voor het opslaan van een Word-document als [MHTML](/nl/web/mhtml/)- en [MHT](/nl/web/mht/)-bestanden. Deze nieuwe bestandsformaten kunnen worden geopend met webbrowsers.
