## Wat is een XHTM-bestand?

Een bestand met de extensie .xhtm is een webpaginabestand dat is geschreven in XHTML. Het gebruikt XML 1.0 als standaardimplementatie en wordt beschouwd als een herformulering van HTML 4. Net als XHTML bevat een XHTM-bestand standaard HTML-tags die strengere XML-richtlijnen volgen. XHTM-bestanden worden gebruikt voor het publiceren van website-inhoud en gebruiken gewoonlijk de extensie [XHTML](/nl/web/xhtml/).

U kunt webbrowsers zoals Google Chrome, Apple Safari en Mozilla Firefox gebruiken om XHTM-bestanden te openen.

## XHTM-voorbeeld

Het volgende voorbeeld van een XHTM-document definieert de XML-declaraties.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
