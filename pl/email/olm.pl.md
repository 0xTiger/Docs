## Czym jest plik OLM?

Plik z rozszerzeniem .olm to plik Microsoft Outlook dla systemu operacyjnego Mac. Plik OLM przechowuje wiadomości e-mail, dzienniki, dane kalendarza i inne typy danych aplikacji. Są one podobne do plików [PST](/pl/email/pst/) używanych przez program Outlook w systemie operacyjnym Windows. Jednak plików OLM utworzonych przez program Outlook dla komputerów Mac nie można otwierać w programie Outlook dla systemu Windows. Format pliku OLM został wprowadzony w programie Outlook 2011 do przechowywania lokalnych danych programu Outlook. Pojedynczy plik OLM może zawierać wiele skrzynek pocztowych, organizując dane w postaci folderów i podfolderów. Pliki OLM można otwierać / ładować w programach Microsoft Outlook 365 i Outlook dla komputerów Mac.

## Format pliku OLM

OLM to pliki binarne typu MIME Application/octet-stream. Specyfikacje techniczne formatu plików OLM nie są jawnie udostępniane przez firmę Microsoft i nie są dostępne żadne szczegóły dotyczące formatu, które mogłyby być dostępne dla programistów. Istnieje jednak kilka interfejsów API, które mogą odczytywać format plików OLM i zapisywać je w pliku PST do użytku w systemie operacyjnym Microsoft Windows, takim jak Aspose.Email.
