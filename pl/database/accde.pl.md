## Czym jest plik ACDE?

Plik z rozszerzeniem .accde to format pliku programu Microsoft Access 2007 służący do tworzenia plików bazy danych tylko do wykonywania w celu ochrony niestandardowego kodu bazy danych. Kompiluje cały kod VBA do pliku wykonywalnego, aby nie można go było przeglądać ani edytować. Format pliku ACCDE zastępuje format pliku MDE, który był używany przez wcześniejsze wersje programu Microsoft Access. ACCDE to pliki [ACCDB](/pl/database/accdb/) tylko w trybie wykonywalnym i bardziej zwarte ze względu na brak kodu źródłowego. Kompaktowy rozmiar tych plików poprawia wykorzystanie pamięci, a także zwiększa wydajność aplikacji. Pliki ACCDE można otwierać za pomocą programu Microsoft Access 365.

## Format pliku ACCDE

Wszystkie pliki Microsoft Access 2007+ wykorzystują Jet ACE (silnik bazy danych Access), który korzysta z rodziny formatów plików ACCDB. Firma Microsoft nie zapewnia jednak implementacji typu open source ani szczegółowej specyfikacji technicznej.
