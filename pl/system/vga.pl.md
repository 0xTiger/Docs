## Co to jest plik VGA?

Plik VGA to plik na dysku systemu Windows obsługujący monitory VGA. Był używany we wczesnych wersjach systemu Microsoft Windows do regulacji monitorów VGA na poziomie systemu operacyjnego (OS). Plik VGA umożliwiający użytkownikom dostosowanie rozdzielczości i głębi kolorów wyświetlacza za pomocą panelu sterowania monitora.

## Format pliku VGA

Wewnętrzny format pliku VGA nie jest znany programistom. Odwołuje się do niego system operacyjny Windows w celu ustawienia wyglądu monitora VGA.
