## Czym jest plik MPP?

Plik z rozszerzeniem .mpp to plik danych programu Microsoft Project, który w zintegrowany sposób przechowuje informacje związane z zarządzaniem projektami. Firma Microsoft opracowała ten zastrzeżony format plików, aby był zgodny z programem Microsoft Project (MSP), oprogramowaniem do zarządzania projektami. Inne niż MPP, MSP może pracować ze schematem XML projektu. Wiele aplikacji i interfejsów API umożliwia konwersję MPP do innych formatów plików. Firma Microsoft udostępnia teraz internetowy serwer projektów, na którym wielu użytkowników może przesyłać pliki do zarządzania projektami w celu współpracy.

## Format pliku MPP ##

Firma Microsoft opracowała i opublikowała MPP w formacie pliku binarnego, a jego specyfikacje nie zostały dotychczas upublicznione przez firmę Microsoft. Wewnętrzna struktura plików i szczegóły są zatem niedostępne dla programistów. Istnieje kilka interfejsów API dostępnych online, które obsługują odczyt/zapis do plików MPP, ale nigdy nie osiąga się 100% dokładności. Z biegiem czasu firma Microsoft wprowadziła kilka funkcji do swojej aplikacji do zarządzania projektami, podobnie jak funkcje formatu plików MPP.

Plik MPP wykorzystuje następujące typy MIME:

* aplikacja/vnd.ms-projekt
* aplikacja/msproj
* aplikacja/msprojekt
* aplikacja/x-msproject
* aplikacja/projekt x-ms
* aplikacja/x-dos_ms_project
* aplikacja/mpp
* zz-application/zz-winassoc-mpp

## Masz problemy z otwarciem pliku MPP? ##

Oto lista typowych problemów, które mogą wystąpić i spowodować nieprawidłowe działanie formatu MPP:

* Brak oprogramowania pomocniczego
* Uszkodzony plik
* Zainfekowany plik z powodu wirusa
* Brak prawa dostępu w systemie do otwierania plików
* Przestarzały dysk w twoim systemie
* Zmieniono nazwę rozszerzenia pliku
