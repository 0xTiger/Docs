## Czym jest plik NX?

Plik NX to plik gry wideo stworzony do użytku z wirtualną konsolą do gier **LowRes NX**. Jest napisany w języku programowania BASIC i zawiera kod źródłowy gry, który definiuje sposób działania gry. Możesz tworzyć własne pliki NX do użytku z aplikacją, które można załadować, a użytkownicy będą mogli grać w grę.

Możesz otworzyć plik NX w edytorze tekstu, LowRes NX lub dowolnym edytorze kodu źródłowego, takim jak Microsoft Visual Studio Code.

## Format pliku NX

Plik NX jest zapisywany na dysku jako zwykły plik tekstowy zawierający instrukcje gry. Każdy plik NX zawiera kod źródłowy gry, który składa się z wielu wpisów ROM kasety. Te wpisy ROM zawierają informacje o komponentach gry, takie jak grafika, mapy poziomów, dźwięk i inne dane gry niebędące kodem źródłowym. Wpisy ROM programu można dodawać lub edytować za pomocą programów do edycji NX dostarczanych w pakiecie z LowRes NX.

## Jak tworzyć pliki NX?

### Twórz pliki NX na komputerze

Możesz tworzyć pliki NX na komputerze PC, korzystając z:

* Edytory tekstu - takie jak Microsoft Notepad, Notepad++ lub Github Atom
* Edytory kodu źródłowego - takie jak kod Microsoft VIsual Studio

### Utwórz pliki NX na iOS

LowRes NX ma wersję na iOS, której programiści mogą używać do tworzenia plików NX z poziomu tej aplikacji.
