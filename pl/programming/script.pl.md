## Czym jest plik SCRIPT?

Plik SCRIPT to plik źródłowy zawierający skrypt w dowolnym języku skryptowym. To sprawia, że jest to uogólniony plik skryptowy, który zawiera zmienne i funkcje. Ponieważ każdy język skryptowy może być inny, skrypt zapisany w pliku SCRIPT musi zostać wykonany przy użyciu odpowiedniego silnika skryptowego.

## Format pliku SCRIPT — więcej informacji

Pliki SCRIPT są zapisywane na dysku w postaci zwykłych plików tekstowych. Można je otwierać i edytować za pomocą dowolnego edytora tekstu, takiego jak Microsoft Notepad, Notepad++ i Apple TextEdit. Ponieważ język skryptowy może być inny, można go wykonać, zmieniając nazwę rozszerzenia pliku na domyślne rozszerzenie odpowiedniego języka skryptowego. Na przykład istnieje kilka popularnych języków skryptowych, takich jak JavaScript ([.JS](/pl/web/js/)), VBScript ([.VBS](/pl/programming/vbs/)), AppleScript (.SCPT), PHP ([ .PHP](/pl/programming/php/)) i ASP ([.ASP](/pl/web/asp/)).
