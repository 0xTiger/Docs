## Czym jest plik YML?

YML to plik tekstowy w języku serializacji danych, [YAML](/pl/programming/yaml/) (Yet Another Markup Language), który jest często używany do tworzenia plików konfiguracyjnych. Może działać w połączeniu z dowolnym językiem programowania, w tym C/C++, Ruby, Python, Java, PHP, C# i innymi. Będąc niezależnym od jakiegokolwiek konkretnego języka programowania, jest otwarty, interoperacyjny i czytelny dla człowieka. Pliki YML można otwierać i edytować za pomocą popularnych edytorów tekstu, takich jak Microsoft Notepad, Notepad ++, Github Atom, Apple TextEdit i Visual Studio IDE.

## Format pliku YML — więcej informacji

Pliki YML przechowują ustawienia aplikacji jako konfigurację. Jego podstawową strukturą jest mapa, którą można również nazwać katalogiem lub skrótem. Wpisy w pliku YML składają się z pary klucz-wartość. Przeważnie pliki konfiguracyjne .yml są zapisywane jako `configure.yml`, chociaż niektóre języki programowania, takie jak Ruby on Rails, zapisują konfigurację łączności z bazą danych w plikach `database.yml`. [Specyfikacje formatu plików YML/YAML](https://yaml.org/spec/1.2.2/) wyjaśniają pełne struktury językowe i informacje związane ze składnią.
