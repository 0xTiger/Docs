## .CC opcja № 1

Plik CC to typ pliku kodu źródłowego, który zawiera kod napisany w języku programowania C++. Może zawierać cały kod niezbędny dla pojedynczego programu lub część kodu będącą częścią większego projektu programistycznego składającego się z wielu plików CC. Aby zmodyfikować lub wyświetlić kod zawarty w pliku CC, programiści często korzystają z edytora kodu źródłowego. Po edycji kompilują kod za pomocą kompilatora w celu konwersji go na kod maszynowy, który system operacyjny komputera może wykonać w celu uruchomienia programu.

## Format pliku CC — więcej informacji

Rozszerzenie pliku .cc było używane w plikach kodu źródłowego C++ od kilkudziesięciu lat. Język C++ został opracowany pod koniec lat 70. XX wieku przez Bjarne'a Stroustrupa, który wówczas pracował w Bell Labs. Stroustrup chciał rozszerzyć język programowania C o obsługę koncepcji programowania obiektowego, które jego zdaniem były niezbędne do tworzenia dużych systemów oprogramowania.

Pierwsza wersja języka C++ została wydana w 1983 roku i szybko zyskała popularność wśród programistów, którzy docenili jej zaawansowane funkcje i elastyczność. W miarę jak język C++ stał się szerzej stosowany, konieczne stało się ustanowienie konwencji kodowania i konwencji nazewnictwa plików, aby pomóc w organizowaniu dużych baz kodu i zarządzaniu nimi.

Przewodnik po stylu C++ firmy Google, który zaleca używanie rozszerzenia pliku .cc dla plików kodu źródłowego C++, został po raz pierwszy opublikowany w 2003 roku. Od tego czasu przewodnik był okresowo aktualizowany, aby odzwierciedlić zmiany w języku C++ i praktykach programistycznych szerszej społeczności programistów.

Obecnie rozszerzenie pliku .cc pozostaje popularnym wyborem dla plików kodu źródłowego C++ i jest rozpoznawane przez wiele popularnych narzędzi i platform programistycznych.

## Jak otworzyć plik CC?

Do otwierania i edytowania plików CC można używać wielu edytorów kodu źródłowego, w tym narzędzi wieloplatformowych, takich jak Microsoft Visual Studio Code i Code Blocks. Edytory te oferują przydatne funkcje, takie jak podświetlanie składni i automatyczne uzupełnianie, co czyni je doskonałym wyborem do pracy z plikami CC.

Możesz także otwierać pliki CC za pomocą dowolnego edytora tekstu, np. Notatnika, Notepad++ itp.

## Kompilowanie CC za pomocą kompilatorów C++

Kompilacja pliku CC za pomocą kompilatora C++ obejmuje kilka kroków. Pierwszym krokiem jest zainstalowanie kompatybilnego kompilatora C++ dla Twojego systemu operacyjnego i środowiska programistycznego. Dostępnych jest wiele różnych kompilatorów, w tym GCC, Clang i Microsoft Visual C++.

Po zainstalowaniu kompilatora C++ następnym krokiem jest otwarcie terminala lub wiersza poleceń i przejście do katalogu, w którym znajduje się plik CC. Stąd możesz uruchomić kompilator, wykonując narzędzie wiersza poleceń dostarczone przez kompilator. Konkretne polecenie kompilacji pliku CC może się różnić w zależności od używanego kompilatora. Na przykład, aby skompilować plik CC przy użyciu GCC, uruchom polecenie podobne do:

```
g++ -o myprogram myfile.cc
```
