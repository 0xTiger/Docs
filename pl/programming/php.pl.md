## Co to jest plik PHP?
Plik z rozszerzeniem .php odnosi się do języka programowania open source, używanego do pisania skryptów po stronie serwera, które mają być wykonywane na serwerze WWW. Jest to najczęściej używany język skryptów internetowych, który jest zwykle używany do tworzenia aplikacji internetowych na dużą skalę. Największy system blogowy w sieci czyli WordPress i największy portal społecznościowy Facebook oparte są na języku PHP. Jest kompatybilny z różnymi platformami, takimi jak Windows, Linux, Unix, Mac OS X itp. Z powodzeniem działa również na prawie wszystkich używanych obecnie serwerach, tj. Apache, IIS itp. PHP obsługuje szeroką gamę baz danych, w tym MySQL.

## Format pliku PHP ##

Pojedynczy PHP może zawierać tekst, CSS, [HTML](/pl/web/html/), JavaScript i kod PHP. Kod PHP jest wykonywany na serwerze, a wynik jest zwracany do przeglądarki jako zwykły HTML. Pliki PHP można otwierać za pomocą dowolnego edytora tekstu i edytować na miejscu, chociaż aplikacje takie jak Adobe Dreamweaver, narzędzia Eclipse PHP Development oferują wygodny sposób pisania i modyfikowania kodu PHP.

### Składnia ###

Skrypt PHP jest wykonywany na serwerze, a zwykły wynik HTML jest wysyłany z powrotem do przeglądarki. Skrypt PHP zaczyna się od<?php and ends with ?>

### Przykład PHP ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
