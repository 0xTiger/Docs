## Czym jest plik HXT? ##

Format HTXT został po raz pierwszy zaprojektowany przez firmę Hanvon i sklasyfikowany jako typ pliku tekstowego. Jest powszechnie używany do plików e-booków Hanvon. Format Hanvon eBook jest obsługiwany tylko przez system Windows lub kompatybilny z oprogramowaniem, które można łatwo zainstalować w systemie Windows. Najpopularniejszym oprogramowaniem obsługującym pliki HTXT jest Przesyłanie do urządzenia przez połączenie USB. Format eBook używany przez urządzenia Hanvon WISEreader eReader często zawiera chiński tekst i jest używany do chińskich książek, zapisuje w podwójnym formacie i może integrować zabezpieczenia DRM, nie może być odczytywany przez programy komputerowe, ale można go przenieść do urządzenia przez połączenie USB.


## Masz problem z otwarciem pliku HTXT? ##

Oto lista niektórych istotnych problemów, które mogą wystąpić i mogą być trudne do otwarcia w tym formacie pliku:

* Pusty lub niekompletny plik. Proponuje to wycinanie zawartości pliku HTXT podczas pobierania lub kopiowania. Jedyną możliwością przywrócenia jest ponowne pobranie pliku
* Plik może być uszkodzony lub zawiera wirusy. Najlepszą opcją jest użycie aktualnego programu antywirusowego w celu wyjaśnienia wszelkich niejasności
* Charakterystyczny błąd połączenia. Program nie udostępnia nieaktualnej wersji pliku HXT. Najprostszym rozwiązaniem tego błędu jest program adaptacyjny lub starsza wersja programu
