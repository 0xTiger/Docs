## Czym jest plik FWP?

Plik z plikiem .fwp jest plikiem archiwum utworzonym za pomocą programu Microsoft Expression Web i zawiera wszystkie pliki i foldery projektu. Microsoft Expression Web to ogólne oprogramowanie firmy Microsoft do projektowania stron internetowych, oparte na edytorze [HTML](/pl/web/html/). Został wycofany od 2012 roku i jest dostępny bezpłatnie do pobrania przez firmę Microsoft. Pliki FWP można eksportować i zapisywać w formacie .cab firmy Microsoft.

## Format pliku FWP — więcej informacji

Dokładne specyfikacje formatu plików FWP nie są dostępne w celach informacyjnych dla programistów. Są to jednak pliki binarne i można je otworzyć w Microsoft Expression Studio, które zostało wycofane dawno temu.
