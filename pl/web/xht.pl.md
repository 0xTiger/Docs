## Czym jest plik XHT?

Plik z rozszerzeniem .xht to plik internetowy powiązany z typem pliku [XHTML](/pl/web/xhtml/) (Extended Hypertext Markup Language), który sam w sobie jest plikiem znaczników opartym na [XML](/pl/web/xml/). Oba te pliki są podobne do HTML, ale mają bardziej rygorystyczną składnię podobną do XML. Pliki XHT są zaprojektowane w taki sposób, aby były bardziej uporządkowane, wymagają mniej skryptów i są ogólne oprócz korzystania z istniejących funkcji XML.

## Format pliku XHT

Plik XHT jest tworzony i przechowywany jako zwykły plik tekstowy z domyślnym kodowaniem UTF-8. Zawiera pełny kod źródłowy dokumentu XHTML, który można otwierać i edytować za pomocą dowolnego edytora tekstu obsługującego kodowanie Unicode. Oprócz edytorów tekstu format pliku XHT jest zrozumiały dla większości nowoczesnych przeglądarek internetowych i można go otwierać za ich pomocą.

## Przykład XHT

Poniższy przykład dokumentu XHT definiuje deklaracje XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
