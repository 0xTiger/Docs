## Co to jest plik DOCMHTML?

Plik DOCMHTML to stary wariant archiwum internetowego zapisany przez wcześniejsze wersje programu Microsoft Word. Nie jest już obsługiwany przez najnowszą wersję programu Microsoft Word. Plik DOCMHTML zawiera reprezentację HTML oryginalnego dokumentu tekstowego .doc. Można go otworzyć zarówno w programie Microsoft Word, jak iw przeglądarce internetowej. Po opublikowaniu dokumentu DOCMHTML można go umieścić na serwerze WWW, który jest dostępny dla użytkowników za pośrednictwem żądania sieciowego. Innym podobnym formatem jest [DOCHTML](/pl/web/dochtml/), który również został wycofany.

## Format pliku DOCMHTML — więcej informacji

Pliki DOCMHTML były używane do zapisywania w formacie HTML. Można go otworzyć w edytorze tekstu, takim jak Notatnik i Notepad++. Po oznaczeniu jako przestarzały program Microsoft Word wprowadził opcje zapisywania dokumentów tekstowych jako pliki [MHTML](/pl/web/mhtml/) i [MHT](/pl/web/mht/). Te nowe formaty plików można otwierać w przeglądarkach internetowych.
