## Czym jest plik CCBJS?

Plik CCBJS to plik JavaScript utworzony przez edytor aplikacji 3D, gier i stron internetowych **CopperCube**. Zawiera kod JavaScript, który jest generowany jako część [pliku .html](/pl/web/html/) utworzonego z pliku .ccb. CopperCube zapewnia przyjazny dla użytkownika interfejs skryptowy do rozwijania logiki rozgrywki i interakcji bez pisania żadnego kodu. Inne formaty plików obsługiwane przez CopperCube to .ccscene, .ccmodel, .ccmaterial, .cctex, .ccjs, .ccp i .ccpak.

## Format pliku CCBJS

Szczegóły wewnętrznego formatu pliku CCBJS nie są znane jako takie. Pliki JavaScript są jednak zapisywane na dysku jako zwykłe pliki tekstowe i można je otwierać za pomocą dowolnego edytora tekstu, takiego jak Microsoft Notepad, Notepad ++ i Apple TextEdit.
