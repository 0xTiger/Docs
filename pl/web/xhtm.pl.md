## Czym jest plik XHTM?

Plik z rozszerzeniem .xhtm to plik strony internetowej napisany w języku XHTML. Wykorzystuje XML 1.0 jako standardową implementację i jest uważany za przeformułowanie HTML 4. Podobnie jak XHTML, plik XHTM zawiera standardowe znaczniki HTML, które są zgodne z bardziej rygorystycznymi wytycznymi XML. Pliki XHTM służą do publikowania treści stron internetowych i powszechnie korzystają z rozszerzenia [XHTML](/pl/web/xhtml/).

Do otwierania plików XHTM można używać przeglądarek internetowych, takich jak Google Chrome, Apple Safari i Mozilla Firefox.

## Przykład XHTM

Poniższy przykład dokumentu XHTM definiuje deklaracje XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
