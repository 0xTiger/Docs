## Czym jest plik MDS?

MDS (Media Descriptor Sidecar File) to plik powiązany z oprogramowaniem Alcohol 120% i Daemon Tools, które służą do tworzenia i zarządzania wirtualnymi napędami CD i DVD. Plik MDS zawiera informacje o układzie danych na dysku, w tym o lokalizacji wszystkich plików i strukturze dysku. Dzięki temu napęd wirtualny emuluje zachowanie dysku fizycznego, dzięki czemu oprogramowanie może odczytywać dane z dysku wirtualnego tak, jakby był to prawdziwy dysk.

When you want to mount an image file to a virtual drive, the MDS file is used along with the corresponding image file in a format such as ISO, BIN or CUE. The MDS file contains a description of the layout of the data on the disc, and the virtual drive software uses this information to create a virtual disc. This allows you to use the software as if you were using a physical disc, without having to physically insert the disc into the drive.

## Jak otworzyć plik MDS?

Aby otworzyć plik MDS w Alcohol 120%, musisz mieć zainstalowane oprogramowanie na swoim komputerze. Po zainstalowaniu Alcohol 120% możesz otworzyć plik MDS, wykonując następujące czynności:

1. Uruchom Alkohol 120%.
2. Kliknij menu Plik” i wybierz Otwórz”.
3. Przejdź do lokalizacji pliku MDS na swoim komputerze i wybierz go.
4. Plik MDS zostanie teraz otwarty w Alcohol 120% i zostaniesz poproszony o wybranie odpowiedniego pliku obrazu (takiego jak ISO, BIN lub CUE).
5. Po wybraniu pliku obrazu, Alcohol 120% zamontuje obraz na dysku wirtualnym.

Alternatywnie możesz także otworzyć plik MDS, klikając go dwukrotnie, jeśli Alcohol 120% jest ustawiony jako domyślny program do otwierania plików MDS. Możesz także użyć innego oprogramowania obsługującego format MDS, takiego jak Daemon Tools, Virtual CloneDrive, Virtual Drive Manager i MagicISO.
