## Czym jest plik WBFS?

Plik WBFS (Wii Backup File System) to obraz dysku gry wideo stworzony dla Nintendo Wii. Zawiera grę Wii, którą można załadować jako obraz do grania. Jest generowany za pomocą narzędzia Wiimms WBPS lub Wii Backup Manager i można go odtwarzać na Nintendo Wii, ładując z USB Loader GX lub emulatora Wii, takiego jak Dolphin. Gra jest przechowywana na tych zewnętrznych urządzeniach pamięci masowej w formacie pliku WBFS. Wii to platforma do gier dystrybuowana przez Nintendo. System plików WBFS został opracowany przez kodera homebrew Wii Waninkoko.

## Format pliku WBFS — więcej informacji

WBFS to pliki binarne, które mają zastrzeżony format pliku. Jego specyfikacje formatu plików nie są dostępne publicznie.
