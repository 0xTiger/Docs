## Czym jest plik AMXX?

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. Jest to platforma skryptowa, która umożliwia administratorom serwerów dostosowywanie i rozszerzanie funkcjonalności ich serwerów. AMXX jest używany w połączeniu z systemem zarządzania wtyczkami MetaMod, który umożliwia jednoczesne uruchamianie wielu wtyczek na serwerze bez konfliktów. Funkcje oferowane przez wtyczkę AMXX obejmują polecenia administratora, modyfikacje rozgrywki i zarządzanie serwerem.
