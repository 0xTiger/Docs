## O FBX do USDZ

FBX, skrót od FilmBox, to szeroko stosowany format plików 3D, pierwotnie stworzony przez Kaydarę dla MotionBuilder. Autodesk Inc. przejął własność w 2006 roku, umacniając swój status podstawowego formatu wymiany danych 3D wykorzystywanego przez wiele narzędzi programowych 3D.

Plik z rozszerzeniem .usdz reprezentuje nieskompresowane i niezaszyfrowane archiwum ZIP dostosowane do formatu pliku [USD](/pl/3d/usd/) (Universal Scene Opis). Zawiera proxy dla różnych typów plików, w tym tekstur i animacji, a wszystko to wygodnie zamknięte w archiwum.

Jeśli jesteś twórcą aplikacji i chcesz włączyć funkcjonalność [FBX](/pl/3d/fbx/) do USDZ do swojej aplikacji .NET, Java lub Python, możesz zapoznać się z interfejsami API Aspose.3D jako cennym zasobem do osiągnięcia ten cel.
