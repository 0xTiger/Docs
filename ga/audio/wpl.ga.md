## Cad is comhad WPL ann?

Tá seinmliosta na n-amhrán le rith ar an Microsoft Windows Media Player i gcomhad le síneadh .wpl. De ghnáth cruthaíonn úsáideoirí na comhaid seo dá mbailiúcháin físe agus fuaime. Níl san ábhar scríofa i gcomhad WPL ach cosáin eolaire nó láithreacha do na comhaid ilmheán roghnaithe ag cruthaitheoir an chomhaid .wpl, ionas gur féidir leis an bhfeidhmchlár seinnteoir meáin an t-ábhar ilmheán a aimsiú agus a imirt go pras óna láithreacha eolaire.

## Formáid Comhaid WPL

Is formáid comhaid dílsithe é WPL (Windows Media Player Playlist) a úsáidtear i leaganacha 9 nó níos mó de Microsoft Windows Media Player. Is formáid comhaid ríomhaire é a stórálann seinmliostaí ilmheán. De réir réamhshocraithe, déantar an seinmliosta a shábháil le síneadh .wpl (Seinnliosta Windows Media Player). Is féidir leat an iarmhír [.m3u](/audio/m3u/) a úsáid ina ionad sin, Más mian leat do dhioscaí sonraí a sheinm ar ghléas nach dtacaíonn leis an eisínteacht seo. Léirítear na heilimintí de chomhad WPL i bhformáid XML.

Sonraíonn an eilimint barrleibhéil aoibh gháire go leanann gnéithe an chomhaid struchtúr na dTeangacha Comhtháthaithe Ilmheán Sioncrónaithe (SMIL). Cruthaítear agus sábháltar an comhad leis an síneadh .wpl agus is é an cineál MIME atá leis ná application/vnd.ms-wpl.

### sampla WPL

Seo sampla de chomhad wpl:
```
<?wpl version="1.0"?>
<smil>
    <head>
        <meta name="Generator" content="Microsoft Windows Media Player -- 11.0.5721.5145"/>
        <meta name="AverageRating" content="33"/>
        <meta name="TotalDuration" content="1102"/>
        <meta name="ItemCount" content="3"/>
        <author/>
        <title>Bach Organ Works</title>
    </head>
    <body>
        <seq>
            <media src="\\server\vol\music\Classical\Bach\OrganWorks\cd03\audio01.mp3"/>
            <media src="\\server\vol\music\Classical\Bach\OrganWorks\cd03\audio02.mp3"/>
            <media src="SR15.mp3" tid="{35B39D45-94D8-40E1-8FC2-9F6714191E47}"/>
        </seq>
    </body>
</smil>
```
