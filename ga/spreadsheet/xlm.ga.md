## Cad is comhad XLM ann?

Is cineál comhaid Scarbhileog é XLM, le haghaidh Macra Excel, a úsáidtear chun Macraí a stóráil. Ó thaobh an fheidhmchláir de, is é atá i Macra ná sraith treoracha a úsáidtear le haghaidh próisis uathoibrithe. Úsáidtear macra chun na céimeanna a dhéantar arís agus arís eile i bhformáid comhaid [XLS](/spreadsheet/xls/) a thaifeadadh agus éascaíonn sé na gníomhartha a dhéanamh tríd an macra a rith arís. Déantar na Macraí a ríomhchlárú le Visual Basic for Applications (VBA) de chuid Microsoft ón taobh istigh de Excel Workbook ag baint úsáide as an Eagarthóir Visual Basic agus is féidir iad a rith / dífhabhtú go díreach as sin.

## Stair Ghearr ##

Microsoft Excel supported programming of macros since its first public launch. The features of macros remained the same through subsequent versions fo Excel with extension as per new features. XLM was the default macro language for Excel through Excel 4.0. Taifeadadh Macraí Excel 5.0 i VBA de réir réamhshocraithe ach le leagan 5.0 ceadaíodh taifeadadh XLM mar rogha fós. Tar éis leagan 5.0 cuireadh deireadh leis an rogha sin. Tá gach leagan de Excel, lena n-áirítear Excel 2010 in ann macra XLM a rith, cé go gcuireann Microsoft bac ar a n-úsáid.

## Macra á thaifeadadh in XLM ##

Soláthraíonn Excel céimeanna éasca le húsáid chun macra a thaifeadadh. Éilíonn sé go bhfuil uirlisí Forbróra suiteáilte agat chun oibriú le Macraí. Nuair a bhíonn macra-thaifeadadh i bhfeidhm, déanann sé taifead ar gach gníomh úsáideora a bheidh le himirt níos déanaí. Is éard atá i gceist le taifeadadh macra i ndáiríre na céimeanna go léir a dhéanann úsáideoir tar éis don taifeadadh tosú. Mar sin, má dhéanann tú ábhar cille trom, iodálach agus má shocraíonn tú a fírinniú téacs tar éis do thaifeadadh macra a bheith tosaithe, déanfar na horduithe seo go léir a thaifeadadh. Is féidir aicearra a shannadh do gach macra taifeadta freisin le haghaidh athsheinm tapa níos déanaí. Gineann taifeadadh Macra cód VBA i bhfoirm macra is féidir a chur in eagar ag baint úsáide as an Eagarthóir Amhairc Bunúsach (VBE).

## Múnla Oibiachta Excel ##

Úsáideann Macraí gnáthaimh VBA ar a gcúl agus leanann siad [Excel Object Model](https://learn.microsoft.com/en-us/office/vba/api/overview/excel/object-model) chun na críche seo. Aithníonn an tsamhail seo cuspóirí scarbhileog le haghaidh idirghníomhú leis an scarbhileog trí bharraí uirlisí ordaithe, barraí orduithe nó boscaí teachtaireachtaí atá sainithe ag an úsáideoir. Mar shampla, tugtar rochtain ar airíonna an leabhair oibre leis an réad Leabhar Oibre. Mar an gcéanna, tá réad Bileog Oibre sa mhúnla chun oibriú le bileoga oibre an leabhair oibre go ríomhchláraithe.

## Macraí agus Slándáil ##

Ceadaíonn VBA rochtain ar gach réimse den fheidhmchlár chomh maith le córas comhaid agus is féidir leis a bheith guaiseach freisin. Ardaíonn sé seo imní agus leabhar oibre á roinnt le duine atá in ann an comhad a rith ag an deireadh. Is é sin Microsoft Excel warns faoi oscailt comhad den sórt sin. Is féidir Macraí a dheimhniú le síniú digiteach ionas gur féidir le húsáideoirí eile a fhíorú go bhfuil siad iontaofa. Mar sin, is féidir macraí a chumasú tar éis a bhfoinse a fhíorú.
