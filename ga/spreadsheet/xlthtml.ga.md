## Cad is comhad XLTHTML ann?

Is leagan [HTML](/web/html/) de chomhaid teimpléid [XLT](/spreadsheet/xlt/) é comhad a bhfuil iarmhír .xlthtml air agus gineadh é le Microsoft Excel sna laethanta tosaigh. Ní thacaítear leis an bhformáid seo níos mó ag Excel agus cuireadh sonraí á n-easpórtáil go formáidí comhaid [MHT](/web/mht/) agus [MHTML](/web/mhtml/) ina hionad. Ba é an bunsmaoineamh a bhí taobh thiar den chomhad XLTHTML ná cur ar chumas úsáideoirí na comhaid teimpléid XLT a fheiceáil i mbrabhsálaí gréasáin seachas an gá le Excel a bheith suiteáilte ar an gcóras. D'fhéadfaí comhaid a easpórtáladh i bhformáid comhaid XLTHTML a oscailt i mbrabhsálaithe.

## Formáid Chomhaid XLTHTML - Tuilleadh Eolais

Bhain comhaid XLTHTML úsáid as formáid an chomhaid HTML chun na teimpléid XLT a shábháil le hoscailt i mbrabhsálaí gréasáin. Tá XLT féin bunaithe ar fhormáid comhaid dhénártha Excel a tugadh isteach le leagan Excel 97-2003. Is féidir na comhaid seo a úsáid mar chomhaid tosaigh le formáidiú sainithe chun gnáthdhoiciméid a chruthú le haghaidh daonra sonraí scarbhileog. Le haghaidh tagartha amháin, tá XLT bunaithe ar an dénártha [XLS file format specifications](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) atá doiciméadaithe ag Microsoft le haghaidh tagartha an fhorbróra.
