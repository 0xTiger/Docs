## Cad is comhad DIS ann?

A DIS file is a workbook file created with [Oracle Discoverer](https://docs.oracle.com/cd/E28389_01/bi.1111/b40107/overview.htm) software that comprises of several tools for querying, reporting, and data analysis for the Oracle Database. It contains multiple worksheets when data is exported from Oracle Discoverer. Oracle markets Discoverer as a Business Intelligence (BI) product used for for reporting and analysis that works in part of the [Oracle Fusion Middleware](https://docs.oracle.com/cd/E28280_01/bi.1111/b40108/toc.htm) suite. Oracle Discoverer doesn't save the reporting data in DIS file format by default.

## Conas Leabhair Oibre Discoverer a íoslódáil i bhFormáid DIS?

Stóráiltear Oracle Discoverer Workbooks sa bhunachar sonraí i bhformáid dhénártha agus ní féidir iad a íoslódáil mar [DIS files](https://forums.oracle.com/ords/apexds/post/can-i-download-all-discoverer-workbooks-to-my-computer-4127) go díreach. Mar sin féin, is féidir sonraí a onnmhairiú mar chomhaid EEX, é a oscailt i Discoverer Desktop, agus úsáid an rogha Save As chun na comhaid EEX a shábháil mar chomhaid DIS.

## Formáid Chomhaid DIS - Tuilleadh Eolais

Déantar comhaid DIS a shábháil ar diosca mar chomhaid dhénártha ach níl a sonraí formáid comhaid inmheánacha ar fáil mar thagairt don fhorbróir.
