## Cad is comhad AST ann?

Is éard atá i gcomhad AST ná comhad teimpléid scarbhileoige a cruthaíodh leis na bogearraí Ability Scarsheet, comhpháirt scarbhileog de shraith Ability Office. Úsáidtear na comhaid teimpléid seo chun sraith tosaigh teimpléid scarbhileog a chruthú is féidir a úsáid chun comhaid [AWS](/spreadsheet/aws/) a chruthú. Is féidir comhaid AST a oscailt le bogearraí scarbhileog Cumais.

## Formáid Chomhaid AST - Tuilleadh Eolais

AST files are saved to disc as binary files and can be opened in Ability Spreadsheet software. Though fully compatible with Microsoft XLS and XLSX file formats, AST file format specifications are not available publicly for developer's reference. AST files can be saved to disc in [PDF](/pdf/) as well as [XPS](/page-description-language/xps/) file formats.
