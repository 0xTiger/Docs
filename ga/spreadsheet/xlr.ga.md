## Cad is comhad XLR ann?

A file with .xlr extension is a Microsoft Works Spreadsheet file format that stored data in rows and columns. Microsoft Works was developed for use as an initial productivity suite and remained functional from 1987 to 2009. Áiríodh ann próiseálaí focal, scarbhileog, agus córas bainistíochta bunachar sonraí. Cuireadh deireadh leis níos déanaí tar éis don Microsoft Office a bheith tugtha isteach agus gnéithe níos saibhre san áireamh.

## Formáid Chomhaid XLR - Tuilleadh Eolais

Sábháladh comhaid XLR mar chomhaid dhénártha gan aon sonraí faoina sonraíochtaí formáid comhaid ar fáil go poiblí. Mar sin, níl an rogha ag forbróirí anailís a dhéanamh ar a struchtúr comhad agus iarratais a scríobh chun na comhaid seo a léamh/a scríobh go ríomhchláraithe. Úsáideann an Scarbhileog Oibreacha an fhormáid comhaid Excel chéanna chun sonraí a stóráil chuig comhaid .xlr.
