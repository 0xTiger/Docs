## Cad is comhad FM ann?

Is comhad scarbhileog é comhad FM a cruthaíodh le FileMaker nó FileMaker Pro ar feidhmchlár bunachar sonraí coibhneasta tras-ardán é. Tá formáid comhaid sách níos sine agus chun é seo a luchtú i FileMaker Pro, ní mór é a thiontú go formáid comhaid .fmp12 nua. Is é an .fmp12 nua seo an t-aon fhormáid comhaid ar féidir a óstáil ar FileMaker Server agus FileMaker Server Advanced. Is féidir comhaid FM a oscailt le FileMaker Pro Advanced.

## Formáid Chomhaid FM - Tuilleadh Eolais

Níl mórán eolais ar fáil faoi fhormáid inmheánach comhaid agus struchtúr comhaid FM. D'óstáil an Chartlann Náisiúnta an [PRONOM](https://www.nationalarchives.gov.uk/PRONOM/fmt/1059) faoi Formáid Chomhaid FM le haghaidh tagartha áfach.

## Conas Comhaid Old FileMaker Pro a Thiontú go Formáid Chomhaid .fmp12?

Is féidir le FileMaker Pro FM a thiontú go formáid comhaid .fmp12 go díreach. Is féidir treoir céim ar chéim le haghaidh [converting FM files to .fmp12](https://support.claris.com/s/article/Converting-older-FileMaker-Pro-files-to-the-fmp12-file-format-1503693002275?language=en_US) a tharchur chun na críche seo.
