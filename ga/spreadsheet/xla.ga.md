## Cad is comhad XLA ann?

Is formáid comhaid Breiseáin Microsoft Excel é comhad le síneadh .xla a chuireann modúil, feidhmeanna breise agus uirlisí eile le Microsoft Excel. Soláthraíonn na Ad-Ins feidhmiúlacht bhreise do scarbhileoga Excel agus tacaítear leo i leagan 2007 nó níos déanaí. Tá feidhmchlár cineál mime media/vnd.ms-excel ag comhaid XLA. Is féidir iad seo a oscailt le Microsoft Excel atá ar fáil do Windows chomh maith le MacOS.

## Formáid Chomhaid XLA

Tá comhaid XLA bunaithe ar fhormáid sonraí dénártha. Mar sin féin, níl a thuilleadh faisnéise ar fáil faoina sonraíochtaí formáid comhaid. Glactar leis go ginearálta go leanann XLA an [[MS-XLS] - Excel Binary File Format Structure](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx).
