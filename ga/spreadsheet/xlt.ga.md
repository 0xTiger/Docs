## Cad is comhad XLT ann?

Is comhaid teimpléid iad comhaid le síneadh .xlt a cruthaíodh le Microsoft Excel ar feidhmchlár scarbhileog é a thagann mar chuid de shraith Microsoft Office. Thacaigh Microsoft Office 97-2003 le comhaid XLT nua a chruthú chomh maith lena n-oscailt. Tá an leagan is déanaí de Excel fós in ann na comhaid teimpléid seanfhormáide seo a oscailt. Úsáidtear comhad teimpléid mar seo chun comhaid Excel nua a chruthú go tapa le sonraí réamhshocraithe agus socruithe mar fhormáidiú leathanaigh, clómhéid, imeall, cairteacha, srl ar féidir a shábháil níos mó mar chomhaid [.xls](/spreadsheet/xls/) nua.

## Formáid Chomhaid XLT

Tá formáid comhaid XLT bunaithe ar fhormáid comhaid dhénártha Excel a fuair tacaíocht ar dtús le leagan Excel 97-2003. Cuireadh formáid comhaid [XLTX](/spreadsheet/xltx/) ina áit le tabhairt isteach Excel 2007 atá bunaithe ar fhormáid Office Open XML. Seachas Microsoft Excel, tá feidhmchláir eile ann a thacaíonn le hoibriú leis an bhformáid comhaid seo mar Lotus Translation Table, PlanMaker, LibreOffice Calc, etc. Tá roinnt feidhmchlár agus [APIs](https://products.aspose.com/cells) ar fáil chun XLT a thiontú go formáidí comhaid eile mar:

* XLT go PDF

* XLT go DOC

* XLT chuig Téacs

* XLT go Lotus

* XLT go ODT

* XLT go ODS

* XLT go HTML

* XLT go CSV


Tá XLT bunaithe ar an dénártha [XLS file format specifications](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) atá doiciméadaithe ag Microsoft le haghaidh tagartha an fhorbróra.
