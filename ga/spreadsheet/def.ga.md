## Cad is comhad DEF ann?

Is comhad sonraí é comhad DEF a cruthaíodh in [spreadsheet file format](/spreadsheet/) ag feidhmchlár bunachar sonraí SmartWare II, arna fhorbairt ag Informix Software. Bhí ceithre mhodúl ag Bogearraí II lena n-áirítear scarbhileog le grafaicí, próiseálaí focal, cumarsáid, agus bunachar sonraí. Is bunachar sonraí neamhspleách é bunachar sonraí SmartWare II ar féidir é a chomhtháthú le feidhmchláir eile freisin. Is féidir fiosruithe a dhéanamh ar shonraí is féidir a bhaint amach agus a shábháil i bhformáid comhaid DEF. Is féidir comhaid scarbhileog DEF a oscailt le bogearraí SmartWare atá scortha anois.
