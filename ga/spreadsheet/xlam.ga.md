## Cad is comhad XLAM ann? ##

Is comhad Breiseáin Macra-Cumasaithe é XLAM a úsáidtear chun feidhmeanna nua a chur le scarbhileoga. Is clár forlíontach é Breiseán a ritheann cód breise agus a sholáthraíonn feidhmiúlacht bhreise do scarbhileoga. Stóráiltear comhaid XLAM leis an síneadh .xlam. Comhaid atá bunaithe ar XML is ea comhaid XLAM cosúil le formáidí comhaid [XLSM](/spreadsheet/xlsm/) agus [XLSX](/spreadsheet/xlsx/) agus déantar iad a shábháil le comhbhrú ZIP chun méid iomlán an chomhaid a laghdú.

### Sampla ###

```cmd
Public Function Add(num1 As Double, num2 As Double)
Add = num1 + num2
End Function
```

Tar éis seo, sábháil an comhad leis an síneadh .xlam.
