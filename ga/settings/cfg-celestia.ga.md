## Cad is comhad CFG ann?

Is comhad gnáth-théacs é comhad cumraíochta Celestia a úsáideann Celestia, clár insamhalta cruinne 3D. Tá na comhaid seo ríthábhachtach chun an chaoi a n-iompraíonn Celestia a shaincheapadh, cad iad na rudaí neamhaí a thaispeántar, agus an chuma ar an gclár. Mar sin féin, ní mór aird chúramach a thabhairt ar eagarthóireacht na gcomhad seo mar is féidir le hathruithe míchuí cur isteach ar phróiseas luchtaithe Celestia, rud a chosc ó rith i gceart.

## Celestia

Is bogearraí ionsamhlúcháin réalteolaíochta 3D foinse oscailte saor in aisce é Celestia a ligeann d’úsáideoirí cruinne a iniúchadh agus a shamhlú ar bhealach fíor-réadúil agus idirghníomhach. D'fhorbair Chris Laurel é agus scaoileadh den chéad uair go luath sna 2000í. Tá Celestia ar fáil do chórais oibriúcháin Windows, macOS, agus Linux.

I measc na bpríomhghnéithe agus gnéithe de Celestia tá:

- **Amharcléiriú Réalaíoch 3D:** Soláthraíonn Celestia léiriú mionsonraithe cruinn ar ár gcóras gréine, chomh maith le réaltaí, réaltraí, agus réada neamhaí eile. Úsáideann sé samhlacha agus uigeachtaí 3D ardcháilíochta chun taithí tumtha ó thaobh amhairc a chruthú.

- **Bunachar Sonraí Neamhfhairsing Neamhaí:** Tagann na bogearraí le bunachar sonraí ollmhór ionsuite de réada neamhaí aithnidiúla, lena n-áirítear réaltaí, pláinéid, gealacha, asteroids, Cóiméid agus spásárthaí. Is féidir le húsáideoirí na rudaí seo a aimsiú agus a iniúchadh go héasca.

- **Cruinneas Eolaíoch:** Cé gur uirlis léirshamhlaithe é Celestia, tá sé mar aidhm aici réada agus feiniméin neamhaí a léiriú chomh cruinn agus is féidir, bunaithe ar shonraí eolaíocha atá ar fáil.

## Formáidí Comhaid Úsáidte ag Celestia

Úsáideann Celestia formáidí éagsúla comhaid le haghaidh gnéithe éagsúla dá insamhalta réalteolaíochta 3D. Seo cuid de na formáidí tábhachtacha comhaid a úsáideann Celestia:

1. **Comhaid Chumraíochta (.cel)**
- *Cur síos*: Comhaid gnáth-théacs a ligeann d’úsáideoirí iompar, cuma agus ábhar Celestia a shaincheapadh.
- * Cuspóir *: Socruithe a shaincheapadh, suíomhanna a shainiú, agus rudaí neamhaí a shonrú.

2. ** Múnlaí 3D agus Mogaill**
- *Formáidí*: {{ HYPERLINK1}}, [.obj](/3d/obj/), [.dae](/3d/dae/), .ac
- *Cur síos*: Formáidí comhaid múnla 3D tacaithe a úsáidtear chun coirp neamhaí agus spásárthaí a rindreáil.
- *Cuspóir*: Ag taispeáint réada 3D laistigh de Celestia.

3. **Comhaid Uigeachta**
- *Formáidí*: [.jpg](/image/jpeg/), [.png](/image/png/), [.dds](/image/dds/)
- *Cur síos*: Soláthraíonn na comhaid seo uigeachtaí dromchla do choirp neamhaí.
- *Cuspóir*: Uigeachtaí a mhapáil ar shamhlacha 3D le haghaidh cuma réalaíoch.

4. **Catalóga Réalta agus Comhaid Sonraí**
- *Formáidí*: Formáidí saincheaptha, [.csv](/spreadsheet/csv/), [.tsv](/spreadsheet/tsv/)
- *Cur síos*: Comhaid sonraí a úsáidtear chun réaltaí agus réada neamhaí eile a léiriú, ag cinntiú insamhalta cruinn.
- *Cuspóir*: Léiriú cruinn ar réada neamhaí.

5. **Léarscáileanna Ciúb Uigeachta**
- *Cur síos*: Úsáidtear léarscáileanna ciúb chun cuma rudaí neamhaí i bhfad i gcéin cosúil le réaltraí a shamhlú.
- *Cuspóir*: Rindreáil réada i bhfad i gcéin le uigeachtaí réalaíocha.

6. **Comhaid Script**
- *Cur síos*: Tá scripteanna saincheaptha sna comhaid seo scríofa i dteanga scriptithe Celestia.
- *Cuspóir*: Cur ar chumas úsáideoirí imeachtaí agus beochan dinimiciúla a chruthú laistigh de chruinne Celestia.

## Comhad cumraíochta Celestia

Seo forbhreathnú bunúsach ar cad is féidir leat a dhéanamh le comhad cumraíochta Celestia:

1. **Do Shuíomh a Shocrú**: Is féidir leat do shuíomh ar an Domhan a shonrú ag baint úsáide as paraiméadair domhanleithead, domhanfhad agus airde. Ligeann sé seo do Celestia spéir na hoíche a thaispeáint go cruinn ó do shuíomh.

```
Location "My Location"
{
    Latitude 40.7128
    Longitude -74.0060
    Altitude 0
}
```

2. **Do Roghanna Amhairc a Chustaim:** Is féidir leat roghanna féachana éagsúla a choigeartú, mar réimse radhairc, socruithe ama, agus sainroghanna rindreála.

```
Viewpoint
{
    Location "My Location"
    Follow "Earth"
    Eye [0.0 0.0 0.0]
    Center [0.0 0.0 0.0]
    Up [0.0 1.0 0.0]
    Fov 45
}

Time
{
    Date "2023-09-25T12:00:00.000Z"
    Clock "Now"
}

Rendering
{
    Atmosphere false
    Stars 7
    Planetshine 0.25
}

```

3. **Réada Neamhaí a Luchtú:** Is féidir leat rudaí neamhaí cosúil le réaltaí, pláinéid, asteroids, spásárthaí agus níos mó a chur le d'ionsamhlú. Sainmhínítear gach réad i gcomhad cumraíochta lena n-airíonna.

```
Star "Sun"
{
    RA 0
    Dec 0
    Distance 0
    AppMag -26.74
    SpectralType "G2V"
}

Planet "Earth"
{
    Parent "Sol"
    Texture "earth.jpg"
    Radius 6371
    EllipticalOrbit
    {
        Period 365.25
        SemiMajorAxis 149.6e6
        Eccentricity 0.017
        Inclination 0
        AscendingNode 0
        ArgOfPericenter 102.94
        MeanAnomaly 100.464
}
}
```

4. **Spáis a Shainmhíniú:** Is féidir leat do spásárthach ficseanúil féin a chruthú nó fíorchinn a úsáid trína bparaiméadar a shonrú amhail suíomh, treoshuíomh agus samhlacha 3D.

```
Spacecraft "Voyager 1"
{
    Parent "Sol"
    Class "spacecraft"
    Mesh "voyager.3ds"
    Radius 0.01
    EllipticalOrbit
    {
        Period 30700
        SemiMajorAxis 1.08e11
        Eccentricity 0.044
        Inclination 3.4
        AscendingNode 49.0
        ArgOfPericenter 44.0
        MeanAnomaly 35.0
}
}
```

5. **Scripteanna a Chruthú:** Is féidir leat scripteanna a scríobh i dteanga scriptithe shaincheaptha Celestia chun imeachtaí agus beochan dinimiciúla a chruthú.
