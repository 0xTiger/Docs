## Cad is comhad RDP ann?

Úsáidtear an síneadh comhad .rdp le haghaidh comhaid Phrótacail Deisce Cianda (RDP). Is prótacal dílseánaigh é RDP arna fhorbairt ag Microsoft a ligeann d’úsáideoirí ceangal le ríomhaire cianda thar nasc líonra. Tá socruithe cumraíochta i gcomhaid RDP a úsáidtear chun seisiún deisce cianda a bhunú le ríomhaire ar leith.

Nuair a chliceálann tú faoi dhó ar chomhad RDP, seolann sé go hiondúil an clár Ceangal Deisce Cianda ar do ríomhaire agus cuireann sé tús le nasc le cianríomhaire atá sonraithe sa chomhad. Is féidir comhaid RDP a chruthú de láimh trí úsáid a bhaint as eagarthóir téacs nó a ghintear go huathoibríoch trí Chiancheangal Deisce nó bogearraí tríú páirtí eile.

Is féidir roghanna cumraíochta éagsúla a áireamh i gcomhaid RDP, mar shampla óstainm nó seoladh IP an ríomhaire iargúlta, an t-ainm úsáideora agus an focal faire don seisiún iargúlta, socruithe taispeána agus go leor eile. Trí chomhad RDP a úsáid, is féidir le húsáideoirí nascadh go tapa agus go héasca le ríomhairí cianda gan a bheith orthu na socruithe seo go léir a chumrú de láimh gach uair is mian leo nascadh.

## Formáid Chomhaid RDP - Tuilleadh Eolais

RDP file (Remote Desktop Protocol file) contains configuration settings that are used to connect to remote computer using the Remote Desktop Connection program or third-party RDP client. The configuration settings in RDP file may include:

- Óstainm nó seoladh IP an chianríomhaire ar mhaith leat ceangal leis.
- An t-ainm úsáideora agus do phasfhocal a úsáidfidh tú chun logáil isteach ar an gcianríomhaire.
- Fearann nó grúpa oibre an chianríomhaire más infheidhme.
- Taifeach taispeána agus socruithe doimhneacht datha le haghaidh seisiún cianda deisce.
- Na socruithe athsheinm fuaime agus físe le haghaidh seisiún cianda deisce.
- Na socruithe atreoraithe printéir agus gearrthaisce le haghaidh seisiún cianda deisce.
- Méid an chiandeasc agus suíomh ar scáileán an ríomhaire áitiúil.
- An crios ama agus socruithe teanga le haghaidh seisiún cianda deisce.
- Aon roghanna nó socruithe cumraíochta saincheaptha le haghaidh seisiún cianda deisce.

Nuair a osclaíonn tú comhad RDP, léann clár Ceangal Deisce Cianda socruithe cumraíochta agus úsáideann iad chun nasc le ríomhaire cianda a bhunú. Ligeann sé seo duit cianríomhaire a rochtain agus a rialú ó do ríomhaire áitiúil amhail is dá mbeifeá i do shuí os a chomhair.

## Cad é formáid an chomhaid RDP?

Is comhad téacs é comhad RDP (comhad Prótacail Deisce Chian) ina bhfuil socruithe cumraíochta a úsáidtear chun seisiún ciandeisce a bhunú le ríomhaire ar leith. Is comhad téacs simplí é is féidir a chruthú nó a chur in eagar ag baint úsáide as eagarthóir téacs cosúil le Notepad nó aon fheidhmchlár eile d’eagarthóirí téacs.

Is éard atá i bhformáid comhaid RDP ná sraith de phéirí eochairluacha, áit a léiríonn gach eochair socrú cumraíochta sonrach agus sonraíonn an luach comhfhreagrach an luach don socrú sin. Tá an fhormáid cosúil leis an bhformáid atá i gcomhad INI ina bhfuil ailt teorannaithe ag lúibíní cearnacha agus péirí aonair eochairluacha scartha le comhartha comhionann.
