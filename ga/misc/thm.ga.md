## Cad is comhad THM ann?

Is comhad téama é comhad THM a cruthaíodh d’fhóin phóca Sony Ericsson. Úsáidtear é chun an comhéadan grafach a athrú don úsáideoir deiridh le roinnt paraiméadair socraithe. Ina measc seo tá socruithe le haghaidh dathanna téacs, deilbhíní, páipéar balla cúlra, fuinneog saincheaptha, agus íomhánna eile. Chuir Sony bogearraí Ericsson Themes Creator ar fáil saor in aisce chun comhaid téama a chruthú. Is féidir é a úsáid freisin chun comhaid THM atá ann cheana a oscailt. Tá feidhmchlár cineál meáin/vnd.eri.thm ag comhaid THM.

## Formáid Chomhaid Sony Ericsson THM - Tuilleadh Eolais

Is éard atá i gcomhad THM iarbhír ná comhad coimeádán ina bhfuil roinnt comhaid eile. Déantar é a shábháil i bhformáid cartlainne [TAR](/compression/tar/) agus is féidir é a bhaint as aon bhogearraí caighdeánach dí-chomhbhrú a thacaíonn le formáid comhaid TAR.

A THM file contains an XML file, several image file sin [.png](/image/png/), [.pjg](/image/jpeg/), [.gif](/image/gif/) formats, or [.swf](/page-description-language/swf/) files.

## Conas Téamaí Sony Ericsson a chruthú?

Is féidir bogearraí Sony Themes Creator a úsáid chun téamaí nua a ghiniúint do fhóin phóca Sony Ericsson. Is féidir le húsáideoirí [Themes Creator Developer Guide](https://developer.sony.com/theme-creator/get-started) Sony a leanúint chun téamaí a chruthú.
