## Cad is comhad EALAÍN ann?

An ART file is an embroidery design file created with the [BERNINA](https://www.bernina.com/en-US/Home-United-States) ARTlink embroidery software version 1-4. Tá dearaí cruthaithe ag úsáideoirí ann ar féidir rudaí éagsúla a áireamh mar imlíne, leagan amach dearaidh, greamanna, snáitheanna daite, agus meiteashonraí eile. Tá an comhad EALAÍN luchtaithe isteach sa mheaisín fuála BERNINA agus is féidir é a fhuáil le cineálacha éagsúla fabraicí mar chuilt, léine, nó aon éadach eile. Is féidir comhaid ART a roinnt freisin le húsáideoirí eile ag baint úsáide as gléasanna stórála ar nós USB nó a aistriú tríd an idirlíon.

## Formáid Chomhaid ART

Tá an fhaisnéis seo a leanas i bhformáid comhaid ART:

 * Airíonna Oibiachta
 * Sonraí Stitch
 * Cineálacha Stitch
 * Comhordanáidí Stitch
 * Dathanna Snáithe
 * Tuairimí
 * Tarraing Socruithe Cúitimh
 * Socruithe Spásáil Uathoibríoch
