## Cad is comhad SSP ann?

Is comhad tionscadail é comhad SSP a chruthaigh bogearraí cruthaithe spárálaíscáileáin Windows ar a dtugtar Screensaver Producer le Axialis Software. Is féidir go bhfuil roinnt cineálacha éagsúla eilimintí spárálaíscáileáin ann, mar shampla sprites, scannáin Flash, seónna sleamhnán grianghraf agus físeáin. Féadfaidh téacs, íomhánna, cruthanna éagsúla, agus socruithe beochana a úsáidtear chun an bheochan a thaispeáint taobh istigh den chomhad SSP a bheith i gcomhad SSP. Is féidir comhaid SSP a oscailt le Léiritheoir spárálaíscáileáin Axilis.

## Formáid Chomhaid SSP - Tuilleadh Eolais

SSP files are stored to disc as binary files with no public information available about their internal file format structure. Axialis Screensaver producer professional screensaver authoring tool for Windows which lets you easily create hi-quality screensavers for all versions of Windows up to Windows 10. Ligeann sé pacáistí spárálaíscáileáin inshuiteáilte a chruthú ar féidir iad a dháileadh go héasca ar líne ar úsáideoirí.

Is féidir comhaid SSP a easpórtáil go formáid comhaid [SCR](/system/scr/).
