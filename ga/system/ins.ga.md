## Cad is comhad INS ann?

Úsáideann Microsoft Windows an comhad le síneadh .ins go ginearálta chun naisc Idirlín diailithe agus leathanbhanda a bhunú. I ndáiríre, tá faisnéis socraithe naisc ann a ligeann do Windows rochtain Idirlín ar ISP a shocrú. Úsáidtear comhad INS go coitianta chun socruithe nasc LAN Internet Explorer (IE) a shocrú. Na comhaid socruithe nasc IE uaireanta a sholáthraíonn na ISPanna agus riarthóirí córais do na húsáideoirí a bhfuil URL acu chuig an gcomhad INS.

## Formáid comhaid INS
Is féidir leat na comhaid Socruithe Idirlín (INS) a úsáid mar aon le Trealamh Riaracháin Internet Explorer 11 (IEAK 11) chun do bhrabhsálaí saincheaptha agus a chomhpháirteanna a chumrú. Is féidir leat leaganacha éagsúla de do phacáiste saincheaptha a scríobh trí chóipeanna den chomhad INS a shaincheapadh.

### Socruithe INS féideartha
Tá na socruithe INS féideartha tugtha sa tábla seo a leanas:

| Socrú | Cur Síos |
-----|---------|
| URL | Cibé an n-úsáidfear seachfhreastalaí uathchumraithe. |
| Brandáil | Saincheap an t-eolas brandála agus socraithe i do phacáiste brabhsálaí. |
| Meáin | Cineálacha meán ina bhfuil do phacáiste suiteála saincheaptha ar fáil. |
| Barraí Uirlisí Brabhsálaí | Saincheap cuma an bharra uirlisí IE. |
| CábSíniú | Eolas síniú digiteach do do chláir. |
| HideCustom | Cibé ar cheart an t-aitheantóir uathúil domhanda (GUID) a cheilt do gach comhpháirt saincheaptha. |
| Socruithe Ceangail | Eolas faoi na socruithe naisc líonraithe a úsáideadh chun do phacáiste saincheaptha a shuiteáil. |
| CustomBrandáil | Suíomh URL chuig do chomhad brandála (. cab). |
| ExtRegInf | Ainmneacha do chomhaid faisnéise Socraithe (.inf) agus an modh suiteála do chomhpháirteanna. |
| CeanáinEx | Cuir cosán le do chomhad deilbhíní do Cheanáin, socraigh an bhfuil Ceanáin ar fáil as líne, agus cuir URLanna le suíomh gach Ceanáin. |
| ISP_Slándáil | An teastas fréimhe atá á chur le do phacáiste saincheaptha. |
| Seachfhreastalaí | Cibé an n-úsáidfear seachfhreastalaí. |
| Allmhairí Slándála | Cibé ar cheart faisnéis slándála a iompórtáil do do phacáiste saincheaptha. |
