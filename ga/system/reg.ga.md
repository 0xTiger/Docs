## Cad is comhad REG ann?

Is formáid comhaid é comhad REG a úsáidtear chun sonraí Chlárlann Windows a allmhairiú nó a onnmhairiú. Is bunachar sonraí ordlathach é Clárlann Windows a stórálann socruithe cumraíochta agus roghanna do chóras oibriúcháin Windows agus feidhmchláir suiteáilte. Tá faisnéis sa chlár mar shainroghanna úsáideora, socruithe feidhmchlár, sonraí cumraíochta crua-earraí agus bogearraí, agus go leor eile.

De ghnáth bíonn iarmhír comhaid .reg ag comhad REG agus is comhad gnáth-théacs é ina bhfuil sraith iontrálacha clárlainne agus luachanna i bhformáid ar leith. Tá an fhormáid comhdhéanta de chuid ceanntásca a shainaithníonn an comhad mar chomhad clárlainne, agus ina dhiaidh sin tá sraith eochairphéirí agus péirí luacha a léiríonn iontrálacha clárlainne.

## Formáid Chomhaid REG - Tuilleadh Eolais

Seo sampla de fhormáid comhaid reg:

```
[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Run]
"Example"="C:\\Example.exe"

[HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced]
"Hidden"=dword:00000001
```

Sonraíonn an chéad líne den chomhad leagan eagarthóir na clárlainne. Sna línte seo a leanas tá na hiontrálacha clárlainne i bhformáid conair eochrach agus ainm luacha agus sonraí luacha ina dhiaidh sin. Sa sampla seo, tá dhá iontráil sa chomhad reg: ceann a chuireann clár ar a dtugtar Example.exe le liosta tosaithe Windows, agus ceann eile a leagann an luach Hidden i socruithe chun cinn Windows Explorer go fíor.

Is féidir comhaid reg a chruthú agus a chur in eagar trí úsáid a bhaint as eagarthóir téacs ar nós Notepad. Is minic a úsáidtear iad chun críocha cúltaca agus athchóiriú, chomh maith le chun ríomhairí iolracha a chumrú leis na socruithe clárlainne céanna.

## Iompórtáil nó Easpórtáil Clárlann Windows

Is cineál comhaid é comhad REG a úsáidtear chun sonraí a allmhairiú nó a onnmhairiú ó Chlárlann Windows. Is bunachar sonraí ordlathach é Clárlann Windows a stórálann socruithe cumraíochta agus roghanna do chóras oibriúcháin Windows agus feidhmchláir suiteáilte. Tá faisnéis sa chlár mar shainroghanna úsáideora, socruithe feidhmchlár, sonraí cumraíochta crua-earraí agus bogearraí, agus go leor eile.

Is féidir comhaid reg a úsáid chun iontrálacha clárlainne a chruthú nó a mhodhnú, agus is minic a úsáidtear iad chun críocha cúltaca agus athchóirithe, chomh maith le ríomhairí iolracha a chumrú leis na socruithe clárlainne céanna. Seo conas comhad reg a úsáid chun iontráil nua clárlainne a chur le Clárlann Windows:

1. Cruthaigh comhad téacs nua ag baint úsáide as eagarthóir téacs ar nós Notepad.
2. Clóscríobh an iontráil clárlainne san fhormáid cheart. Tá an fhormáid comhdhéanta de chuid ceanntásca a shainaithníonn an comhad mar chomhad clárlainne, agus ina dhiaidh sin tá sraith de phéirí eochair agus luach a léiríonn iontrálacha clárlainne. Seo sampla:

```
Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\Software\Example]
"Setting1"="Value1"
```

Cruthaíonn sé seo eochair nua ar a dtugtar Sampla faoin eochair Bogearraí i hive clárlainne an úsáideora reatha, agus socraíonn sé an luach Setting1 go Luach1.

3. Sábháil an comhad le síneadh comhad .reg.
4. Cliceáil faoi dhó ar an gcomhad .reg chun an iontráil nua clárlainne a chur le Clárlann Windows.

Tabharfar leid duit a dhearbhú gur mian leat an iontráil a chur leis an gclárlann. Nuair a dheimhníonn tú, cuirfear an iontráil nua leis an gclár, agus is féidir leat é a fhíorú ag baint úsáide as uirlis Eagarthóir na Clárlainne.
