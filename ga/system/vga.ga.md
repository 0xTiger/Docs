## Cad is comhad VGA ann?

Is comhad tiomána Windows é comhad VGA chun tacú le monatóirí VGA. Úsáideadh é i leaganacha luatha de Microsoft Windows chun monatóirí VGA a choigeartú ag leibhéal an Chórais Oibriúcháin (OS). Comhad VGA a úsáidtear chun ligean d'úsáideoirí taifeach agus doimhneacht dath an taispeántais a choigeartú ag baint úsáide as painéal rialaithe Monitor.

## Formáid Comhaid VGA

Ní fios formáid comhaid inmheánach formáid comhaid VGA le haghaidh tagartha an fhorbróra. Déanann córas oibriúcháin Windows tagairt dó chun cuma amhairc an mhonatóra VGA a shocrú.
