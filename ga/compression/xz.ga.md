## Cad is comhad XZ ann?

Is formáid comhaid chomhbhrúite é XZ a úsáideann algartam comhbhrú LZMA2. Ceapadh é mar ionadú ar na formáidí gzip agus bzip2 a bhfuil tóir orthu, agus tá roinnt buntáistí ag baint leis thar na caighdeáin níos sine seo. Tugtar tacaíocht mhaith do chomhaid XZ thar go leor ardán, agus is féidir iad a dhí-chomhbhrú go tapa agus go héasca. Cé nach bhfuil siad chomh coitianta le comhaid [ZIP](/compression/zip/) nó [RAR](/compression/rar/), is féidir cartlanna XZ a úsáid chun méid mór sonraí a stóráil gan cáilíocht an chomhbhrú a íobairt. Más gá duit comhaid mhóra a chomhbhrú nó a dhí-chomhbhrú, is fiú smaoineamh ar an síneadh comhad XZ.

## Formáid Chomhaid XZ

XZ comhad a ghintear agus a shábháil mar chomhaid dhénártha ar diosca. Úsáideann sé algartam LZMA chun sonraí a chomhbhrú, agus féadann sé cóimheas comhbhrú suas le 50% a bhaint amach. Úsáidtear comhaid XZ de ghnáth le haghaidh dáiltí bogearraí agus cartlanna cúltaca. Is féidir iad a dhí-chomhbhrú ag baint úsáide as an bhfóntas XZ, atá san áireamh sa chuid is mó dáileacháin Linux.

## Unzip Comhaid XZ ar Linux / Unix

Chun comhaid XZ a dhízipeáil, suiteáil pacáiste `xz-utils` ar dtús:
```
$ sudo apt-get install xz-utils
```

Úsáid an t-ordú `unxz` chun comhaid .xz a bhaint:
```
$ unxz compressed_xz_file.xz
```

nó ag baint úsáide as rogha --decompress de XZ:
```
$ xz --decompress compressed_xz_file.xz
```
