## Cad is comhad LZH ann? ##

A file with .lzh and .lha extension usually relates to archive compression file format. This file format is the same as other file compression formats like [ZIP](/compression/zip/), [RAR](/compression/rar/), etc. The main purpose of these file formats is to reduce the size of the format for easily sending as well as to keep them together in compressed form. AS compare to other western companies LZH files are very popular in Japan and usually used for compressing video game data. The LZH add-on for Windows 7 is built into the Japanese version of Windows 7. D'eisigh Microsoft an Breiseán Fillteán Comhbhrúite Microsoft (LZH) den chéad uair don leagan Seapánach de Windows XP. Cuirtear an fhormáid comhaid seo i bhfeidhm le forálacha comhbhrú agus gnéithe a úsáideann an algartam Lempel-Ziv agus Haruyasu

## Sonraíocht LZH ##

Taispeántar an modh comhbhrúite de chartlann LZH mar theaghrán téacs cúig bheart, mar -lz1-. Is iad seo an tríú trí seachtú beart sa chomhad comhbhrúite.

|Sonraíocht|Cur síos|
---|---|
|Síneadh | .lzh, .lha|
|Cineál Meán| feidhmchlár/x-lzh-comhbhrúite|
|Cineál Cód| LHA_ (LHA-SPACE)|
|UTI| poiblí.cartlann.lha|
|Arna fhorbairt ag| Haruyasu Yoshizaki (Yoshi)|
|Cineál| Comhbhrú|
|Sínte ó| LArc|

## Fadhbanna le comhad LZH a oscailt ##

Seo a leanas na roinnt fadhbanna a d'fhéadfadh a bheith ina gcúis le droch-oibriú formáid comhaid LZH:
  
* Féadfaidh an comhad a bheith truaillithe. Chun an fhadhb seo a réiteach, íoslódáil an comhad arís nó iarr ar an seoltóir an comhad a sheoladh arís
* Is é an chúis dara comhad ionfhabhtaithe sa chás seo scanadh an comhad i gceart
* Naisc mhíchuí leis an gcomhad LZH
  *	 Removal of the description of the LZH 
* Níl go leor acmhainní crua-earraí
* Tiománaithe as dáta an trealaimh
