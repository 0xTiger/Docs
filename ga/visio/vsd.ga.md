## Cad is comhad VSD ann?

Líníochtaí is ea comhaid VSD a cruthaíodh le feidhmchlár Microsoft Visio chun éagsúlacht rudaí grafacha agus an t-idirnasc eatarthu a léiriú. Is féidir réadamharc a bheith i líníochtaí den sórt sin mar rudaí amhairc, sreabhchairteacha, léaráid UML, sreabhadh faisnéise, cairteacha eagraíochtúla, léaráidí bogearraí, leagan amach líonra, samhlacha bunachar sonraí, mapáil réad agus faisnéis eile dá samhail. Cuireann Microsoft Visio an cumas comhaid Visio a thiontú go formáidí éagsúla comhaid lena n-áirítear [PNG](/image/png/), [BMP](/image/bmp/), [PDF](/pdf/) agus cinn eile.

# Formáid Chomhaid VSD #

VSD file format is the initial file format introduced by Microsoft Visio for storing drawing elements. The internal structure of VSD files is based on binary file format unlike [.VSDX](/visio/vsdx/) that was introduced with Microsoft Visio 2013. Tá an fhormáid nua bunaithe ar fhormáid comhaid Office Open XML agus stórálann sí faisnéis faoin tionscadal i gcartlann [ZIP](/compression/zip/). Cuireann Microsoft feidhmchlár vviewer.exe ar fáil do chomhaid líníochta Visio ar eagla nach bhfuil Visio suiteáilte ar an gcóras.

## Stair Ghearr ##

Visio was initially produced as a standalone product by Shapeware Corporation in 1992 by its version 1.0. D'athraigh Shapeware Corporation an t-ainm go Visio Corporation i 1995 chun leas a bhaint as aitheantas an mhargaidh. Déanann Microsoft Vision a athbhrandáil sa bhliain 2000 mar fheidhmchlár Microsoft Office. Tugadh isteach Visio for Enterprise Architects le Visual Studio .NET i 2003 agus 2005. Eisíodh Visio 2007 ar 30 Samhain, 2006. Thug Microsoft Visio isteach ribíní i gcomhéadan úsáideora i 2010.
