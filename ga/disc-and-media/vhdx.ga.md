## Cad is comhad VHDX ann?

Is comhad íomhá diosca é comhad VHDX san fhormáid comhaid Diosca Crua Fíorúil v2. Tá córas oibriúcháin iomlán ann ar féidir a luchtú agus a úsáid mar ghnáth-inneall chun bogearraí a thástáil nó bogearraí a rith a dteastaíonn córas oibriúcháin ar leith uathu. In ainneoin gur íomhá diosca iomlán é VHDX, déantar é a stóráil in aon chomhad amháin. Is féidir le bogearraí meaisín fíorúil ar nós Parallels Desktop, Windows Virtual Machine agus Virtual Box íomhá an diosca a luchtú agus a oscailt.

Is féidir comhad VHDX a thiontú go [VHD](/disc-and-media/vhd/) le Hyper-V Manager, nó go [VDI](/disc-and-media/vdi/) le VirtualBox.

## Formáid Chomhaid VHDX - Tuilleadh Eolais

The VHDX file format details are completely documented and available online as [VHDX File Format Specifications](https://learn.microsoft.com/en-us/openspecs/windows_protocols/ms-vhdx/83e061f8-f6e2-4de1-91bd-5d518a43d477) on Microsoft Documentation. It is an extension to the VHD format and includes enhanced capabilities. VHDX file format provides additional features that are available at the virtual hard disk and virtual hard disk file layers. It is more optimized and gives better results for storage hardware configuration and capabilities. VHDX files can be opened

### Tacaíocht do Dhiosca Crua Fíorúla i VHDX

Tacaíonn sé le trí chineál dioscaí crua fíorúil.

 1. **Diosca Crua Fíorúil Seasta** - Diosca crua fíorúil le méid sonraí seasta leithdháilte. Ní athraíonn méid an diosca crua fíorúil trí shonraí a shuimiú nó a bhaint.

 1. ** Diosca Crua Fíorúil Dinimiciúla** - Diosca crua fíorúil le méid diosca dinimiciúil. Tá sé ag am ar bith méid chomh mór leis na sonraí iarbhír scríofa chuige agus meiteashonraí. Méadaíonn a mhéid comhaid go dinimiciúil de réir mar a chuirtear sonraí leis nó nuair a bhaintear as é.

 1. **Difríocht Diosca Crua Fíorúil** - Léiríonn sé sruth an diosca crua fíorúil mar thacar de bhlocanna mionathraithe i gcomparáid le comhad diosca crua tuismitheora.
