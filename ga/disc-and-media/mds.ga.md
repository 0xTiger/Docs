## Cad is comhad MDS ann?

Is comhad é an MDS (Comhad Taoibhcharr Tuairisceoir Meán) a bhaineann leis na bogearraí Alcól 120% agus Deamhan Uirlisí, a úsáidtear an dá cheann chun tiomántáin CD agus DVD fíorúil a chruthú agus a bhainistiú. Tá faisnéis sa chomhad MDS faoi leagan amach na sonraí ar an diosca, lena n-áirítear suíomh na gcomhad go léir agus struchtúr an diosca. Ligeann sé seo don tiomántán fíorúil iompar diosca fisiceach a aithris, ionas gur féidir leis na bogearraí na sonraí ar an diosca fíorúil a léamh amhail is gur diosca fíor é.

When you want to mount an image file to a virtual drive, the MDS file is used along with the corresponding image file in a format such as ISO, BIN or CUE. The MDS file contains a description of the layout of the data on the disc, and the virtual drive software uses this information to create a virtual disc. This allows you to use the software as if you were using a physical disc, without having to physically insert the disc into the drive.

## Conas comhad MDS a oscailt?

Chun comhad MDS a oscailt in Alcól 120%, bheadh ort na bogearraí a bheith suiteáilte ar do ríomhaire. Nuair a bheidh Alcól 120% suiteáilte agat, is féidir leat an comhad MDS a oscailt tríd an méid seo a leanas a dhéanamh:

1. Seoladh Alcól 120%.
2. Cliceáil ar an roghchlár Comhad agus roghnaigh Oscailte.
3. Déan nascleanúint go dtí suíomh an chomhaid MDS ar do ríomhaire agus roghnaigh é.
4. Osclófar an comhad MDS anois in Alcól 120%, agus iarrfar ort an comhad íomhá comhfhreagrach a roghnú (mar shampla ISO, ARAID nó CUE ).
5. Tar éis duit an comhad íomhá a roghnú, cuirfidh Alcól 120% an íomhá ar thiomáint fíorúil.

Nó, is féidir leat comhad MDS a oscailt trí chliceáil faoi dhó air, má tá Alcól 120% socraithe mar an réamhchlár chun comhaid MDS a oscailt. Is féidir leat bogearraí eile a úsáid freisin a thacaíonn le formáid MDS cosúil le Deamhan Uirlisí, CloneDrive Fíorúil, Bainisteoir Tiomántán Fíorúil agus MagicISO.
