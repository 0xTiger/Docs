## Cad is comhad WIM ann?
Chonacthas na comhaid le síntí .wim den chéad uair i Windows Vista. Baineann an WIM le formáid íomháithe bunaithe ar chomhaid a tugadh isteach de ghnáth i Microsoft Windows Vista. Cuireann sé ar chumas íomhá diosca amháin a imscaradh chuig ardáin éagsúla ríomhaire. Úsáidtear comhaid WIM chun comhaid cosúil le nuashonruithe, tiománaithe agus comhpháirteanna a bhainistiú gan íomhá an chórais oibriúcháin a ath-thosú. Tá sraith comhad agus meiteashonraí gaolmhara i gcomhad AQ WIM atá an-chosúil le formáidí comhaid diosca eile.

## Formáidí comhaid WIM
Níl formáid comhaid WIM cosúil le formáidí earnáil-bhunaithe mar VHD nó IS, go deimhin is comhad-bhunaithe é a chiallaíonn gur comhad é an bunaonad faisnéise i WIM. Is é an buntáiste bunúsach a bhaineann le bheith bunaithe ar chomhaid ná go laghdaítear an forchostais a bhaineann le comhaid éagsúla a oscailt agus a dhúnadh ina n-aonar, mar go stóráiltear na comhaid taobh istigh de WIM amháin trí stóráil aonchéime de chomhad a ndéantar tagairt dó go minic i gcrann an chórais comhad agus neamhspleáchas crua-earraí. comhad. Is féidir le comhaid WIM íomhánna diosca iolracha a stóráil, a ndéantar tagairt dóibh trína n-ainm uathúil nó trína n-innéacs uimhriúil.
### Tacaíocht do halgartaim chomhbhrú
Tacaíonn WIM le teaghlaigh na n-algartam comhbhrú seo a leanas i luas íslitheach agus cóimheas ardaitheach:
- XPRESS
- LZX
- LZMS
- Soladach-comhbhrú.
Tá na chéad trí theaghlach bunaithe ar LZ77 agus tugadh isteach comhbhrú soladach mar aon le LZMS i WIMGAPI Windows 8 agus DISM Windows 8.1.
### Uirlisí le haghaidh formáid comhaid WIM
Is gnách go dtacaíonn na huirlisí seo a leanas le formáid comhaid WIM:

- **ImageX**: Uirlis ordú-líne a úsáidtear chun íomhánna diosca Windows a chur in eagar, a chruthú agus a imscaradh i bhFormáid Íomháú Windows. Mar aon leis an WIMGAPI ábhartha, déantar é a dháileadh mar chuid den Kit Suiteáil Uathoibrithe Windows saor in aisce. Ag tosú le Windows Vista, úsáideann Windows Setup an API WAIK chun Windows a shuiteáil.
- **DISM**: Uirlis a tugadh isteach i Windows 7 agus Windows Server 2008 R2 a fhéadfaidh tascanna a bhaineann le seirbhís a dhéanamh ar íomhá suiteála Windows, bíodh sé ina íomhá ar líne nó ina íomhá as líne laistigh d’fhillteán nó i gcomhad WIM. bhí an uirlis seo in ann íomhánna a fheistiú agus a dhífheistiú, tiománaí gléis a chur le híomhá as líne agus tiománaithe gléasanna suiteáilte a cheistiú in íomhá as líne.
