## Cad is comhad MPP ann?

Is comhad sonraí Tionscadal Microsoft é comhad le síneadh .mpp a stórálann faisnéis a bhaineann le bainistíocht tionscadail ar bhealach comhtháite. Tá an fhormáid comhaid dílsithe seo forbartha ag Microsoft chun go mbeidh sé comhoiriúnach le Microsoft Project (MSP), a dtáirge bogearraí bainistíochta tionscadail. Seachas MPP, is féidir le BPA oibriú le scéimre XML tionscadail. Ligeann go leor feidhmchlár agus APInna MPPanna a thiontú go formáidí comhaid eile. Tá Freastalaí Tionscadail ar líne ag Microsoft anois inar féidir le húsáideoirí iolracha comhaid bhainistíochta tionscadail a uaslódáil le haghaidh comhoibriú.

## Formáid Chomhaid MPP ##

D'fhorbair agus d'fhoilsigh Microsoft MPP mar fhormáid dhénártha comhaid agus níl a shonraíochtaí foilsithe ag Microsoft go dtí seo. Dá bhrí sin, níl an struchtúr inmheánach comhad agus sonraí ar fáil d'fhorbróirí. Tá roinnt API ar fáil ar líne a thacaíonn le léamh/scríobh chuig comhaid MPP, ach ní bhaintear amach cruinneas 100% riamh. Tá roinnt gnéithe tugtha isteach ag Microsoft dá fheidhmchlár bainistíochta tionscadail le himeacht ama, agus mar sin tá gnéithe formáid comhaid MPP aige.

Úsáideann comhad MPP na cineálacha MIME seo a leanas:

*iarratas/vnd.ms-tionscadal

* feidhmchlár/msproj

* feidhmchlár/tionscadal

* feidhmchlár/tionscadal x-ms

* feidhmchlár/tionscadal x-ms

* feidhmchlár/x-dos_ms_project

* feidhmchlár/mpp

* zz-iarratas/zz-winassoc-mpp


## Fadhbanna le Comhad MPP a Oscailt? ##

Seo liosta de roinnt saincheisteanna coitianta a d’fhéadfadh teacht chun cinn agus a d’fhéadfadh a bheith ina gcúis le mífheidhmiú na formáide MPP:

 *   Neamhláithreacht bogearraí tacaíochta
 *   Comhad truaillithe
 *   Comhad ionfhabhtaithe de bharr víreas
 *   Níl aon cheart rochtana sa chóras chun na comhaid a oscailt
 *   Tiomántán as dáta i do chóras
 *   Athainmnítear síneadh an chomhaid
