## Cad is comhad SAMI ann?

Tagraíonn comhad leis an síneadh .sami de ghnáth do chomhad Fotheidil Agus Idirmhalartú Meiteashonraí (SAMI). Is formáid fotheideal é SAMI a úsáidtear chun fotheidil nó fotheidil dúnta a thaispeáint i bhfíseáin. D'fhorbair Microsoft é ar dtús dá Windows Media Player.

Tá faisnéis uainiúcháin agus ábhar téacs le haghaidh fotheidil nó fotheidil dúnta i gcomhaid SAMI, rud a ligeann dóibh a bheith sioncronaithe le hathsheinm físe. Tacaíonn an fhormáid le roghanna formáidithe bunúsacha cosúil le stíl cló, dath, agus suíomh na bhfotheideal ar an scáileán.

De ghnáth is comhaid gnáth-théacs iad comhaid SAMI agus is féidir iad a oscailt agus a chur in eagar trí úsáid a bhaint as eagarthóir téacs. Cuimsíonn a bhfuil i gcomhad SAMI de ghnáth roinn ceanntásca a sholáthraíonn faisnéis faoin gcomhad, agus ina dhiaidh sin iontrálacha aonair fotheideal lena n-am agus a dtéacs faoi seach.

Seo sampla den chuma a bheadh ar chomhad SAMI:

```
<SAMI>
<HEAD>
<TITLE>Example Subtitles</TITLE>
</HEAD>
<BODY>
<SYNC Start=100><P Class=ENCC>Subtitle 1</P></SYNC>
<SYNC Start=500><P Class=ENCC>Subtitle 2</P></SYNC>
<SYNC Start=1000><P Class=ENCC>Subtitle 3</P></SYNC>
</BODY>
</SAMI>
```

Úsáidtear comhaid SAMI go coitianta i gcomhar le himreoirí físeáin nó seinnteoirí meáin a thacaíonn le taispeáint fotheideal, mar shampla Windows Media Player nó VLC Media Player. Léann an t-imreoir an comhad SAMI agus sioncronaíonn sé na fotheidil leis an ábhar físeáin, rud a ligeann don lucht féachana na fotheidil a léamh agus iad ag breathnú ar an bhfíseán.

## Clibeanna HTML tacaithe

SAMI (Subtitles And Metadata Interchange) files support a limited set of HTML-like tags for formatting and styling the subtitles. Here are some of the commonly used tags supported by SAMI:

- ``<SAMI> :`` An eilimint fréimhe a chuimsíonn an comhad SAMI ar fad.
- ``<HEAD> :`` Tá eolas ceanntásca don chomhad SAMI ann.
- ``<TITLE>:`` Specifies the title of the SAMI file.
- ``<BODY>:`` Encloses the subtitle entries and their timing information.
- ``<SYNC>:`` Represents a synchronization point for a subtitle entry. It specifies the timing at which the subtitle should be displayed.
- ``<P>:`` Encloses the actual text content of a subtitle. It is typically used within a <SYNC> block.
- `` <FONT>:`` Sainmhínítear airíonna cló don téacs faoi iamh. Is féidir tréithe cosúil le Dath, Aghaidh, Méid agus Stíl a úsáid chun cuma an chló a mhodhnú.</font>
- ``<BR>:`` Inserts a line break within a subtitle.
- `` <B>:`` Rindreáil an téacs faoi iamh i gcló trom.</b>
- `` <I>:`` Rindreáil an téacs faoi iamh i gcló iodálach.</i>
- `` <U>:`` Rindreáil an téacs faoi iamh a bhfuil líne faoi.</u>
- ``<C>:`` Specifies the position or alignment of the subtitle text on the screen. It supports attributes like Center, Middle, Left, Right, Top, Bottom, and their combinations.
- ``<LANG>:`` Specifies the language code for the subtitle text. It helps in identifying the language of the subtitles.

Seo cuid de na clibeanna bunúsacha a fhaigheann tacaíocht ó chomhaid SAMI. Tá sé tábhachtach a thabhairt faoi deara nach dtacaíonn SAMI leis an raon iomlán clibeanna agus tréithe HTML. Tá na clibeanna tacaithe dírithe go príomha ar stíliú agus suíomh na bhfotheideal seachas ar struchtúrú doiciméad fairsing nó idirghníomhaíocht a sholáthar.
