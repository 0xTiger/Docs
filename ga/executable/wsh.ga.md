## Cad is comhad WSH ann?
Tá airíonna agus paraiméadair i gcomhad le síneadh .wsh do script teanga ríomhchlárúcháin áirithe ar nós VB nó VBS etc. Is é an riachtanas iarbhír atá le WSH iad a úsáid chun feidhmiú scripteanna áirithe a shaincheapadh. Tá gá le WScript nó CScript a rith agus tá an dá cheann seo san áireamh le córas oibriúcháin Windows. Soláthraíodh comhaid WSH ar dtús ar Windows 95 ar na dioscaí suiteála mar roghnach in-chumraithe agus insuiteáilte don Phainéal Rialaithe, agus ansin mar chomhpháirt réamhshocraithe de Windows 98.

## Formáid comhaid WSH
Is féidir WSH (Windows Script Host) a úsáid chun críocha éagsúla, lena n-áirítear riarachán, scripteanna logáil isteach agus uathoibriú ginearálta. Bunaíonn WSH timpeallacht chun scripteanna a rith. iarrann sé an t-inneall scripte oiriúnach agus leithdháileann sé sraith seirbhísí agus réada le go n-oibreoidh an script leo. Is féidir na scripteanna seo a rith i mód GUI, ó oibiacht COM nó ó mhodh ordaithe, ag soláthar solúbthachta don úsáideoir maidir le scripteanna idirghníomhacha nó neamh-idirghníomhacha.

### Samplaí
Seo sampla an-simplí a thaispeánann roinnt VBScript a úsáideann an fhréamh WSH COM oibiacht WScript chun teachtaireacht a thaispeáint le cnaipe 'OK'. Nuair a sheolfar an script seo thabharfaí an t-inneall CScript nó WScript agus socrófar an timpeallacht rite.

```
WScript.Echo "Hello world"
WScript.Quit
```
