## Cad is comhad RUN ann?

Úsáidtear an fhormáid comhaid .run go coitianta chun bogearraí nó suiteálaithe feidhmchláir a dháileadh i dtimpeallacht Linux. Chun na bogearraí a shuiteáil, beidh ort an comhad a dhéanamh inrite, is féidir a dhéanamh ag baint úsáide as an ordú seo a leanas:

```bash
chmod +x file_name.run 
```

Ansin, is féidir leat an comhad a rith ag baint úsáide as an ordú seo a leanas:

```bash
./file_name.run 
```

Féadfaidh an próiseas suiteála a bheith éagsúil ag brath ar an gcomhad agus an clár sonrach atá tú ag iarraidh a shuiteáil.

Is cineál script bhlaosc é an fhormáid comhaid .run a úsáidtear chun bogearraí nó suiteálaithe feidhmchláir a dháileadh i dtimpeallacht Linux. Is pacáiste féinchuimsitheach é a chuimsíonn gach rud is gá chun na bogearraí a shuiteáil, lena n-áirítear comhaid dhénártha, leabharlanna, agus comhaid cumraíochta.

Tá sé tábhachtach a thabhairt faoi deara gur féidir cód mailíseach a bheith i gcomhaid .run freisin, mar sin is smaoineamh maith i gcónaí foinse an chomhaid a fhíorú agus é a scanadh le haghaidh víreas sula ritheann sé é.

Ina theannta sin, d'fhéadfadh go mbeadh pribhléidí fréimhe ag teastáil ó roinnt comhaid .run chun iad a rith agus a shuiteáil, mar sin b'fhéidir go mbeadh ort an t-ordú sudo a úsáid chun an comhad a rith le ceadanna ardaithe:

```bash
sudo ./filename.run
```

## Conas comhad RUN a oscailt?

Chun comhad .run a oscailt, ní mór duit é a dhéanamh inrite, is féidir a dhéanamh leis an ordú chmod:

```bash
chmod +x filename.run 
```

Nuair a bheidh an comhad inrite, is féidir leat é a rith trí chlóscríobh:

```bash
./filename.run
```

Ní hionann comhad .run a rith agus é a oscailt in eagarthóir téacs. Ritheann comhad .run a threoracha, rud a d'fhéadfadh a bheith mar rud ar bith ó chlár a shuiteáil go script a rith. Chun féachaint ar a bhfuil i gcomhad .run, ní mór duit é a oscailt in eagarthóir téacs, mar nano nó vim:

```
nano filename.run
```
nó
```
vim filename.run
```
