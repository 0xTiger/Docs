## Cad is comhad KIT ann?

Is comhad HTML é comhad a bhfuil iarmhír .kit aige a chruthaítear leis an bhfeidhmchlár teanga ríomhchlárúcháin [CodeKit](https://codekitapp.com/). Tá `allmhairí` agus `athróga` ann chomh maith le comhaid [HTML](/web/html/) atá ann cheana féin, rud a fhágann go bhfuil sé oiriúnach do shuímh statacha. Tiomsaíonn CodeKit comhaid KIT go HTML ar féidir iad a úsáid go héasca mar chomhaid statacha láithreáin ghréasáin.

## Formáid Chomhaid KIT

Is comhaid HTML iad comhaid KIT a chuimsíonn allmhairí agus athróga freisin. Stóráiltear iad seo mar chomhaid gnáth-théacs agus is féidir iad a oscailt le haon eagarthóir téacs nó le haon eagarthóirí comhaid gréasáin.

### Iompórtálacha Comhad KIT

Is féidir beagnach aon chineál comhaid a allmhairiú isteach i gcomhad Kit. Seo a leanas an chomhréir iompórtála a úsáidtear chun comhaid a allmhairiú isteach i gcomhad .kit.

```
<!-- @import "someFile.kit" -->
<!-- @import "file.html" -->
```

Nuair a chuirtear iompórtáil le comhad KIT agus nuair a shábháiltear é, cuirtear téacs an chomhaid atá á iompórtáil ina ionad. Is féidir leat @include a úsáid freisin in ionad @import.

### Iompórtálacha Il i gcomhad KIT

Is féidir leat freisin níos mó ná comhad amháin a iompórtáil ag an am céanna trí úsáid a bhaint as liosta camóg-scartha:

```
<!-- @import someFile, otherFile.html, ../thirdFile.kit -->
```
