## Cad is comhad DO?

Is comhad servlet Java é comhad DO a úsáideann freastalaithe gréasáin chun feidhmchláir Java gréasán-bhunaithe a sheachadadh. Baineann sé go coitianta le rialtóir Struts na bhfeidhmchlár a phróiseálann comhaid. Ligeann sé seo do chomhaid DO leathanaigh ghréasáin dhinimiciúla a ghiniúint ina bhfuil cuma amhairc chomh maith le sonraí a fuarthas ó bhunachair shonraí. Seolann Java servlets na leathanaigh feidhmchlár gréasáin seo a ghintear go dinimiciúil chuig brabhsálaí á iarraidh lena thaispeáint.

## DO Formáid Comhaid

Sábháiltear comhaid DO mar chomhaid gréasáin a osclaítear sa bhrabhsálaí nuair a sheolann úsáideoir iarratas chuig an bhfreastalaí. Próiseálann an servlet an t-iarratas agus seolann sé an comhad DO mar servlet chuig an mbrabhsálaí iarrthach.
