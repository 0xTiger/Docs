## Cad is comhad XML ann?

XML stands for Extensible Markup Language that is similar to **[HTML](/web/html/)** but different in using tags for defining objects. The whole idea behind creation of XML file format was to store and transport data without being dependent on software or hardware tools. Its popularity is due to it being both human as well as machine readable. This enables it to create common data protocols in the form of objects to be stored and shared over network such as World Wide Web (WWW). The "X" in XML is for extensible which implies that the language can be extended to any number of symbols as per user requirements. It is for these features that many standard file formats make use of it such as Microsoft Open XML, LibreOffice OpenDocument, **[XHTML](/web/xhtml/)** and **[SVG](/page-description-language/svg/)**.

## Formáid Comhaid XML

Tá formáid an chomhaid XML bunaithe ar Shamhail Oibiachta Doiciméid XML (DOM) atá ina API ríomhchláraithe do dhoiciméid HTML agus XML. Sainmhíníonn an XML DOM modh caighdeánach chun eilimintí doiciméad XML a rochtain agus a ionramháil. Déanann sé radharc struchtúr crann de dhoiciméad XML is féidir a úsáid chun rochtain a fháil ar gach eilimint tríd an gcrann DOM. Is féidir eilimintí reatha a mhodhnú/scriosadh chomh maith le heilimintí nua a chruthú sa chrann XML. Tugtar nód ar gach eilimint de dhoiciméad XML. Tá an XML DOM mar a thaispeántar san íomhá seo a leanas.

### Cur chuige uilíoch XML

De bharr chumhacht XML is teanga uilíoch í le haghaidh cumarsáide sonraí thar an líonra trí iompar sonraí agus athruithe ardáin a shimpliú. Cinntíonn sé seo freisin gur féidir sonraí a mhalartú idir córais neamh-chomhoiriúnacha trí shonraí a stóráil i bhformáid gnáth-théacs. Is le haghaidh ionadaíochta sonraí ar an ngréasán atá HTML, ach is le haghaidh malartú sonraí atá XML. Sainmhíníonn na péirí clibeanna marcála a úsáidtear taobh istigh de XML príomhghnéithe an struchtúir atá le húsáid ag feidhmchláir léitheoireachta.
