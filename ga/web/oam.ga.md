## Cad is comhad OAM ann?

A .oam file a an animated widget file exported from Adobe Edge Animate Widget. Animations exported as OAM widget file format can be inserted into other Adobe applications such as InDesign Documents ([INDD file](/page-description-language/indd/), Dreamweaver, and Muse. OAM files are like deployment packages that can be readily inserted into other Adobe's projects to make use of them. OAM files contain information about the animation's assets, behaviors, and actionscript code. You can create an OAM file by publishing an Adobe Animate project [.AN file](/web/an/).
Is féidir le húsáideoirí comhaid OAM a chruthú trí thionscadal Edge Animate (comhad .AN) a fhoilsiú.

## Formáid Comhaid OAM

Déantar comhad OAM a shábháil ar diosca mar chartlann chomhbhrúite [ZIP](/compression/zip/). Tugann sé seo le tuiscint gur féidir leat an síneadh comhad a athainmniú go .zip agus a bhaint as le haon áirgiúlacht comhbhrú ar nós WinZIP nó WinRAR. Mar gheall ar an méid comhaid laghdaithe is fusa an beochan a aistriú agus a roinnt le húsáideoirí eile ar an idirlíon.

### Struchtúr Comhad OAM

Tá struchtúr inmheánach comhad comhaid .oam dílsithe agus níl sé doiciméadaithe go poiblí ag Adobe. Mar sin féin, tá a fhios go bhfuil bailiúchán de chomhaid agus acmhainní (cosúil le grafaicí, fuaime, agus cód ActionScript) i gcomhad .oam pacáistithe i gcomhad amháin. D’fhéadfadh comhaid {{ HYPERLINK1}}, comhaid SWF, agus comhaid acmhainne eile a bheith san áireamh i gcomhad .oam. Braitheann struchtúr cruinn an chomhaid .oam ar an leagan sonrach de Adobe Animate a úsáideadh chun é a chruthú, chomh maith leis an gcineál beochana agus sócmhainní atá san áireamh sa chomhad.

Tá an fhaisnéis seo a leanas i gcomhad OAM.

`Sócmhainní:` Eolas faoi na sócmhainní grafaic, fuaime agus físe a úsáidtear sa bheochan.

`Iompar:` Cur síos ar na gníomhartha agus na hidirghníomhaíochtaí a tharlaíonn sa bheochan.

`Cód ActionScript:` An cód a úsáidtear chun iompar agus idirghníomhaíocht an bheochana a rialú.

`Socruithe a fhoilsiú:` Eolas faoin ardán agus formáid ina bhfoilseofar an beochan, mar HTML5 nó AIR.

`Metadata:` Additional information such as the animation's author, creation date, and version number.
