## Cad is comhad SUÍOMH GRÉASÁIN ann?

A WEBSITE file is a website shortcut file created with Microsoft Internet Explorer (IE). It contains a URL and an icon (ICO) file that points to the website's favicon. The icon is used for displaying the shortcut. WEBSITE file format was introduced with the release of IE version 9. Úsáidtear leaganacha roimhe seo de IE chun aicearraí chuig leathanaigh ghréasáin a shábháil mar chomhaid [.url](/web/url/).

## WEBSITE Formáid Chomhaid

Stóráiltear comhad .suíomh Gréasáin ar diosca mar chomhaid gnáth-théacs ar féidir a bhfuil iontu a fheiceáil in aon eagarthóir téacs. Chomh maith le hábhar eile, tá an URL sprice ann a bhfuil sé ina aicearra dó. Más mian leat an spriocláithreán gréasáin a oscailt i mbrabhsálaí seachas IE, is féidir leat an URL seo a chóipeáil agus é a ghreamú i mbrabhsálaithe eile cosúil le Google Chrome, Safari, nó Firefox chun an suíomh Gréasáin a fheiceáil.

## Conas comhad .WEBSITE a chruthú?

Is féidir comhad SUÍOMH GRÉASÁIN a chruthú in IE trí chliceáil ar an URL sa bharra seoltaí agus é a tharraingt chuig an áit atá uait ar do ríomhaire, mar shampla deasc nó doiciméid. Cruthaíonn sé seo an t-aicearra chuig an suíomh idirlín sin agus is féidir é a oscailt trí chliceáil faoi dhó ar an aicearra. Más mian leat an t-aicearra a shábháil mar chomhad `URL`, coinnigh síos an eochair SHIFT roimh duit favicon an spriocshuímh a tharraingt chuig an deasc.
