## Cad is comhad HYPE ann?

A HYPE file is a project file saved using the Hype software that is used for creation of web content with HTML5. Tá gnéithe gréasáin ann mar ghrafaic, téacs, beochan, faisnéis radharc agus gnéithe eile a chomhdhéanann an t-ábhar le chéile. Is féidir ábhar gréasáin i gcomhad HYPE a onnmhairiú chuig comhaid gréasáin is féidir a fhoilsiú ar leathanach gréasáin.

Is féidir comhad HYPE a oscailt le bogearraí [Tumult Hype](https://tumult.com/hype/).

## Formáid Chomhaid HYPE

Stóráiltear comhaid HYPE i bhformáid comhaid dílseánaigh agus ní bhíonn na sonraí inmheánacha ar eolas go poiblí. Ní féidir an t-ábhar taobh istigh de chomhad HYPE a fheiceáil ach amháin nuair a osclaítear é i mbogearraí Tumult Hype.

### Inneachar Comhad HYPE

Cruthaítear comhad HYPE agus tú ag baint úsáide as an bhfeidhmchlár Hype chun ábhar gréasáin a chruthú chun críocha éagsúla. Stórálann sé faisnéis faoi bheochan atá bunaithe ar eochairfhráma chun inneachar a chruthú mar ríomhchártaí, ríomhleabhair, láithreoireachtaí, fógraí, grafaic ghréasán agus ábhar leathanach gréasáin. Tá a n-ábhar faoi thiomáint ag HTML5 a ligeann don ábhar a thaispeántar ar ghléasanna ó ríomhairí deisce go iPhones agus iPads. Aschuireann Tumult Hype na stíleanna HTML5, [CSS3](/web/css/) agus [JavaScript](/web/js/) den scoth.
