## Cad is comhad VDISCO ann?

Is comhad fionnachtana é comhad VDISCO a úsáideann Microsoft Visual Studio ina Sheirbhísí Gréasáin. Soláthraíonn sé faisnéis faoi na Seirbhísí Gréasáin atá ar fáil amhail a URLanna, spásanna ainmneacha agus modhanna. Tá comhad VDISCO cosúil leis an bhformáid comhaid [DISCO](/web/disco/) ach gintear é ag am rite. Stóráiltear é san fheidhmchlár seirbhíse Gréasáin agus úsáideann an Visual Studio é chun Seirbhísí Gréasáin a aimsiú agus a úsáid go dinimiciúil i dtionscadal. Úsáidtear comhaid VDISCO in éineacht leis an gcomhad [.asmx](/web/asmx/) ina bhfuil fíorchód na Seirbhíse Gréasáin.

Is féidir leat comhad VDISCO a oscailt in aon eagarthóir téacs mar Microsoft Notepad, Github Atom, agus Apple TextEdit. Is féidir é a oscailt freisin le Microsoft Visual Studio 2012 agus níos déanaí chun oibriú leis.

## Formáid Comhaid VDISCO

Déantar comhad VDISCO a shábháil agus a óstáil i bhformáid comhaid [XML](/web/xml/) atá ina bhformáid uilíoch chun sonraí a chomhdhéanamh agus a fhoilsiú. Tá URL na seirbhíse, spásanna ainm agus modhanna ann i gclibeanna caighdeánacha XML ina bhfuil luach a chuid faisnéise faoi seach i ngach clib.
