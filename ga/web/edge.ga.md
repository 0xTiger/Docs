## Cad is comhad EDGE ann?

Comhad le síneadh .edge is ea comhad Tionscadal Adobe Edge a chruthaítear nuair a chruthaítear agus a shábháiltear tionscadal le Adobe Edge. Ligeann Adobe Edge do dhearthóirí gréasáin beochantóirí idirghníomhacha [HTML](/web/html/) a chruthú le haghaidh gréasáin, foilsitheoireacht dhigiteach, agus fógraíocht shaibhir meán do bhrabhsálaithe deisce agus do bhrabhsálaithe soghluaiste araon. Baintear é seo amach trí HTML/HTML5, [JavaScript](/web/js/), agus [CSS3](/web/css/) a úsáid. Gintear comhaid Edge mar chomhaid gnáth-théacs agus is féidir iad a oscailt le roinnt feidhmeanna éagsúla cosúil le Microsoft Notepad, Apple TextEdit, agus Notepad++.

## Formáid Chomhaid EDGE

Sábháiltear comhaid EDGE i bhformáid gnáth-théacs agus úsáidtear jQuery chun an beochan a chumasú. Rogha eile de bheochan EDGE ná comhaid Flash nach dtacaítear leo a thuilleadh. Sábháiltear an comhad EDGE taobh leis na comhaid HTML, JavaScript agus CSS. Nuair a osclaítear an comhad EDGE, osclaítear an leathanach gréasáin .i. an comhad HTML.
