## Cad is comhad JHTML ann?

Is comhad {{ HYPERLINK1}} le cód {{ HYPERLINK2}} é comhad a bhfuil iarmhír .jhtml air a fheidhmítear ar an bhfreastalaí nuair a iarrann cliant an leathanach seo i mbrabhsálaí. Próiseálann an freastalaí na hiarratais, feidhmíonn sé na feidhmeanna Java atá san fheidhm, agus cuireann sé an leathanach ar ais chuig brabhsálaí an chliaint. Ritheann na réada Java atá leabaithe sna leathanaigh JHTML ar fhreastalaí chun iarratais ar leathanaigh den chineál seo a láimhseáil. Is féidir le comhaid JHTML faisnéis a rochtain ón mbunachar sonraí freisin trí úsáid a bhaint as nasc JDBC (Java [Database](/database/) Connectivity). Is féidir comhaid JHTML a oscailt in aon eagarthóir téacs agus féachaint orthu i mbrabhsálaithe gréasáin mar Google Chrome, Firefox, agus Safari.

## Formáid Comhaid JHTML

Is teicneolaíocht dílseánaigh de chuid ATG é JHTML agus féadfar é a chruthú trí úsáid a bhaint as Eagarthóir Doiciméid Dynamo ATG (Art Technology Group). Scríobhtar comhaid JHTML i bhformáid gnáth-théacs agus úsáid á baint as an gcód caighdeánach HTML agus Java. Tá clibeanna caighdeánacha HTML iontu seo chomh maith le clibeanna dílseánaigh a dhéanann tagairt do réada Java. Nuair a iarrann úsáideoir a leithéid de leathanach, cuireann an freastalaí HTTP an t-iarratas ar aghaidh chuig freastalaí feidhmchláir Java. Tiontaítear an leathanach JHTML go comhad .java ar dtús agus tiomsaítear é chun comhad [.class](/programming/class/) a ghiniúint a fheidhmítear mar servlet. Gineann sé seo sruth de shonraí caighdeánacha HTTP agus HTML a chuirtear ar ais chuig an mbrabhsálaí a iarrtar lena thaispeáint don úsáideoir. Tá sé seo ina chuidiú chun fiosrúcháin a bhaineann le bunachar sonraí a rith ar an bhfreastalaí agus an toradh carntha críochnaithe a chur ar ais chuig brabhsálaí an chliaint.
