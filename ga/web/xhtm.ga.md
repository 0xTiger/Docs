## Cad is comhad XHTM ann?

An file with .xhtm extension is a webpage file that is written in XHTML. It uses XML 1.0 as its standard implementation and is considered a reformulation of HTML 4. Cosúil le XHTML, tá clibeanna caighdeánacha HTML i gcomhad XHTM a leanann treoirlínte XML níos déine. Úsáidtear comhaid XHTM chun inneachar an tsuímh Ghréasáin a fhoilsiú agus is minic a úsáideann {{ HYPERLINK1}} eisínteacht.

Is féidir leat brabhsálaithe gréasáin a úsáid mar Google Chrome, Apple Safari, agus Mozilla Firefox chun comhaid XHTM a oscailt.

## Sampla XHTM

Sainmhíníonn an sampla seo a leanas de dhoiciméad XHTM na dearbhuithe XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
