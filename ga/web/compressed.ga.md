## Cad is comhad comhbhrúite ann?

Is éard atá i gcomhad comhbhrúite ná leathanach gréasáin comhbhrúite Wordpress ar bheagán a ghintear leis an breiseán, **WordCompress**. Tá líne singil de chód i gceist leis chun an leathanach gréasáin a léiriú a ghintear trí chód HTML, CSS agus JavaScript a chomhbhrú leis an mbreiseán. Tá na comhaid comhbhrúite seo beag i méid a chabhraíonn le luchtú agus aistriú níos tapúla ar an idirlíon.

### Formáid Chomhaid Chomhbhrúite - Tuilleadh Eolais

Sábhálfar comhaid chomhbhrúite mar chomhaid téacs is féidir a oscailt le haon eagarthóir téacs mar Microsoft Notepad, Notepad++, agus Apple TextEdit. Chomh maith le heagarthóirí téacs, is féidir comhaid COMPRESSED a oscailt le bogearraí tógála gréasáin cosúil le Adobe Dreamweaver atá ar fáil do Windows agus Mac OS araon.

## Maidir le Breiseán WordCompress

Baineadh úsáid as an mbreiseán WordCompress ar dtús chun leathanaigh ghréasáin WordPress a chomhbhrú go formáid comhaid chomhbhrúite. Ní thacaítear leis an mbreiseán a thuilleadh agus scoirtear é. Scríobhfadh sé cód foinse an leathanaigh ghréasáin chuig comhad inrite amháin trí gach spás bán a bhaint de chód foinse an leathanaigh. Stóráiltear comhaid chomhbhrúite taobh le bunchód foinse an leathanaigh ghréasáin chun na buntráitis, eangú agus spás bán eile a chuimsítear i mbunchód foinse an leathanaigh a chaomhnú.
