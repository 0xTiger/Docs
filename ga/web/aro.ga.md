## Cad is comhad ARO ann?

Is comhad leathanach gréasáin ar thaobh an fhreastalaí é comhad ARO a ghintear le Feidhmchlár Gréasáin SteelArrow. Tá clibeanna agus scripteanna [HTML](/web/html/) agus SteelArrow ann. Déantar iad seo a fhorghníomhú ar an bhfreastalaí chun leathanach iomlán freagartha a ghiniúint sula seolfar iad chuig brabhsálaí an úsáideora. Tá beagnach 95% d'ábhar HTML i gcomhaid ARO agus tá cód SteelArrow sa chuid eile. Chun go n-oibreoidh comhaid ARO duit, ba cheart freastalaí Feidhmchláir Ghréasáin SteelArrow a shuiteáil agus a reáchtáil ar an meaisín freastalaí gréasáin.

## Formáid Chomhaid ARO

ARO files are written in HTML markup language file with embedded JavaScript code and Java applets. [SteelArrow tags](http://www.steelarrow.com/docs/basics1.aro) are the basic building blocks for development of web pages. Though these don't change the display of the page, they are used to fill the page with data. In addition, they may also be used to control the output with conditional statements.

### Sampla Clib ARO

Tá an<SAOUTPUT> Ligeann clib ARO d'fhorbróirí luachanna sonraí a aschur in aon áit laistigh de script. Mar shampla, is féidir é a úsáid chun aschur tábla PARAM a fháil le<SAOUTPUT VALUE=PARAM[]> is féidir a thaispeáint laistigh den HTML<BODY> chlib.
