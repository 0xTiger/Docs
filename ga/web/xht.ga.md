## Cad is comhad XHT ann?

Is comhad gréasáin é comhad a bhfuil iarmhír .xht air a bhaineann leis an gcineál comhaid [XHTML](/web/xhtml/) (Teanga Mharcála Hipirtéacs Breisithe), ar comhad marcála bunaithe ar [XML](/web/xml/) é féin. Tá an dá chomhad seo cosúil le HTML ach tá comhréir níos déine cosúil le XML acu. Tá comhaid XHT deartha ar bhealach níos struchtúrtha, bíonn níos lú scripte i gceist leo, agus tá siad cineálach chomh maith le húsáid na n-áiseanna atá ann cheana féin de XML.

## Formáid Chomhaid XHT

Cruthaítear comhad XHT agus stóráiltear é mar chomhad gnáth-théacs le hionchódú UTF-8 de réir réamhshocraithe. Tá cód foinse iomlán doiciméad XHTML ann ar féidir a oscailt agus a chur in eagar le haon eagarthóir téacs a thacaíonn le hionchódú Unicode. Chomh maith le heagarthóirí téacs, tá formáid comhaid XHT intuigthe ag formhór na mbrabhsálaithe gréasáin nua-aimseartha agus is féidir iad a oscailt trí úsáid a bhaint as.

## XHT Sampla

Sainmhíníonn an sampla seo a leanas de dhoiciméad XHT na dearbhuithe XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
