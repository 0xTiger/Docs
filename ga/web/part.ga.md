## Cad is comhad PÁIRTEANNA ann? ##

Comhad a íoslódáltar go páirteach is ea páirtchomhad nó síneadh .part. Úsáidtear é nuair a bhíonn íoslódálacha ar siúl nó nuair a chuirtear isteach orthu mar gheall ar aon cheist, rud a thugann deis don bhainisteoir íoslódála an íoslódáil a atosú nuair is féidir.
Ciallaíonn sé seo go stórálann an brabhsálaí, ar nós Firefox nó cláir aistrithe comhad mar eMule, eMule plus, FlashGet, etc. cuid den chomhad, ar a dtugtar Comhad Cuid, fad is atá an íoslódáil ag tarlú ar do ghléas. Taispeánfaidh an páirtchomhad seo duit má tá an íoslódáil ar siúl nó má cuireadh isteach air roimh chríochnú. Ní hamháin seo ach stórálann an comhad CUID na sonraí go léir go dtí go bhfuil an íoslódáil críochnaithe agus sin an fáth gur féidir cuid acu a atosú níos déanaí más mian leat an íoslódáil a thosú arís.

## Formáid Chomhaid PÁIRT ##

There are few browsers that break large download files into smaller downloads and assign each portion a .part extension. These files can be seen in the “Download” folder. Once the download is finished, the browser will combine all the PART files to give a final downloaded file. For example, while downloading a file from Firefox, the download manager will store the data being downloaded in a PART file and replaces the name with certain characters, and add a .part extension at the end, such as **doodle.or** becomes **JKzMn.or.part**. After completion, Firefox will remove the .part extension and the file will be ready to launch.
