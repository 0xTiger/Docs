## Cad is comhad CS ann?

Is comhaid cód foinse iad comhaid le síneadh .cs do theanga ríomhchlárúcháin C#. Arna thabhairt isteach ag Microsoft le húsáid leis an gCreat .NET, soláthraíonn formáid an chomhaid an teanga ríomhchlárúcháin íseal-leibhéil chun cód a scríobh a chuirtear le chéile chun an comhad aschuir deiridh a ghiniúint i bhfoirm EXE nó DLL. Is féidir iad seo a chruthú agus a chur le chéile le Microsoft Visual Studio. Is féidir Microsoft Visual Studio Express a úsáid freisin chun comhaid den sórt sin a chruthú agus a nuashonrú ar IDE saor in aisce é. Úsáidtear comhaid CS chun feidhmchláir a fhorbairt a d’fhéadfadh a bheith éagsúil ó fheidhmchláir shimplí deisce go cláir níos casta. Féadfaidh comhad amháin nó níos mó a bheith i dtionscadal simplí Visual Studio [solution](/programming/sln/) a cruthaíodh le teanga C#. Tá na comhaid atá marcáilte le cur san áireamh sa tiomsú liostaithe sa chomhad [CSPROJ](/programming/csproj/) atá mar chuid den tionscadal agus a insíonn don tiomsaitheoir na comhaid marcáilte a úsáid.

## Formáid Chomhaid CS ##

Is formáidí comhaid téacsbhunaithe iad comhaid CS is féidir a oscailt in aon eagarthóir téacs le haghaidh eagarthóireachta. Mar sin féin, nuair a osclaítear é in IDE tacaithe le béim chuí ar chomhréir, is furasta an cód a léamh agus a shocrú. Cuimsíonn comhad CS simplí:

* Dearbhú ainmspásanna - Chun tagairt a dhéanamh d'fheidhmíocht ar leith arna sainiú ag an ainmspás sonrach sin

* Dearbhú Athróga - Athróga leibhéal ranga a dhearbhú don chur chun feidhme ar leith

* Dearbhú Modhanna - Dearbhú modhanna a dhearbhú don fheidhmiúlacht ar leith


### Comhréir ###

* Úsáidtear leathchúil chun deireadh ráitis a chur in iúl.

* Úsáidtear lúibíní curacha chun ráitis a ghrúpáil. De ghnáth déantar ráitis a ghrúpáil i modhanna (feidhmeanna), modhanna i ranganna, agus aicmí i spásanna ainmneacha.

* Sanntar athróga le comhartha comhionann, ach cuirtear dhá chomhartha comhionanna as a chéile i gcomparáid leo.

* Úsáidtear lúibíní cearnacha le eagair, chun iad a dhearbhú agus chun luach a fháil ag innéacs ar leith i gceann acu.


### Sampla ###

```
using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Hello, world!");
}
}
```
