## Cad is comhad Java ann? ##
Comhad Java a thugtar ar chomhad ina bhfuil cód foinse Java agus a shábháiltear le síneadh comhad .java. Tá an Java ar cheann de na teicneolaíochta is mó a úsáidtear chun cluichí, feidhmchláir soghluaiste, gréasáin agus deisce a fhorbairt. Ós rud é go bhfuil an Java neamhspleách ar ardán, oibríonn sé gan locht ar Windows, Mac, Linux, Raspberry Pi, srl. Tá an Java an-chosúil le C# agus C++ agus mar sin is fusa aistriú idir na teangacha seo.

## Stair Ghearr ##

Is iad James Gosling, Mike Sheridan, agus Patrick Naughton a thionscain an tionscadal Java i mí an Mheithimh 1991. Ainmníodh Java ar dtús Darach. Athainmníodh Glas ina dhiaidh sin agus ar deireadh go Java. Dhear James Gosling Java le comhréir cosúil le C/C++. D'eisigh Sun Microsystems an chéad leagan poiblí de Java i 1996. D'fhéadfadh sé rith ar na córais mhóréilimh go léir a d'fhág go raibh tóir ar Java go tapa. Le scaoileadh Java 2 i mí na Nollag 1998, tógadh cumraíochtaí iolracha le haghaidh cineálacha éagsúla ardán. Bhí na leaganacha mar seo a leanas

- J2EE (Java EE): Le haghaidh réitigh fiontair
- J2ME (Java ME): Le haghaidh feidhmchláir shoghluaiste
- J2SE (Java SE): Le haghaidh feidhmchláir deisce

Ar 19 Samhain, 2006, d'eisigh Sun Java Virtual Machine (JVM) mar bhogearraí saor in aisce agus foinse oscailte. Tar éis do Oracle Corporation Sun Microsystems a fháil in 2009–2010, d’éirigh James Gosling as Oracle ar 2 Aibreán, 2010.

## Conas cód Java a rith / a fhorghníomhú ##

Chun an cód Java a fhorghníomhú, ní mór é a thiomsú ar dtús. Chun sin, tá an SDK Java ag teastáil. Tiomsaíonn an Java SDK an cód Java go comhad ranga bytecode. Tá IDE cosúil le Eclipse agus IntelliJ Idea a fhágann go bhfuil sé níos éasca oibriú le comhaid Java trí chomhlánú cód agus comhéadan atá éasca le húsáid a sholáthar chun an cód Java a thiomsú agus a fhorghníomhú.

## Formáid Chomhaid Java ##

Bhí an-tionchar ag C agus C++ ar chomhréir Java ach murab ionann agus C++, tógadh Java beagnach go heisiach mar theanga atá dírithe ar oibiachtaí. I Java, scríobhtar an cód ar fad laistigh de na ranganna agus is réad é gach mír sonraí. I gcodarsnacht le C++, ní thacaíonn Java le ró-ualú oibreoirí nó le hoidhreacht iolrach.

### Cód samplach Java ###

Seo a leanas sampla de chomhréir Java.

```java
/*
The example code prints
Hello World from Java to the console.
*/
    public class ExampleApp {
        public static void main(String[] args) {
            System.out.println("Hello World from Java"); // Prints the string to the console.
        }
    }
```
Sa chód thuas, seasann an eochairfhocal **poiblí** an modhnóir rochtana. Sonraíonn sé gur féidir leis na ranganna lasmuigh den ordlathas ranga an aicme seo a rochtain. Is féidir an modhnóir rochtana a **chosaint** freisin (is féidir é a rochtain sa phacáiste céanna) nó **príobháideach** (ní féidir rochtain a fháil ar na modhanna ach amháin ag an aicme chéanna). Tugann an **statach** os comhair an mhodha le fios gur féidir an modh a úsáid gan sampla ar leith den rang. Tugann an **folamh** le fios nach dtabharfadh an modh aon rud ar ais. Chun an teaghrán a phriontáil chuig an consól. Úsáidtear an t-ordú system.out.println. Sa ordú seo, tá réimse statach *Amach* ag an rang *Córas* ar sampla é den rang *PrintStream* ina bhfuil an modh *println*.

Ba cheart go mbeadh ainm comhaid na gcomhad Java mar an gcéanna le hainm an ranga. Mar sin bheadh an comhad Java don chód samplach ainmnithe *ExampleApp.java*.
