## Cad is comhad SH ann?

Comhad orduithe teanga scriptithe is ea comhad le síneadh .sh ina bhfuil ríomhchlár a bheidh á rith ag Unix shell. Féadfaidh sraith orduithe a bheith ann a ritheann go seicheamhach chun oibríochtaí a dhéanamh ar nós próiseáil comhad, cur i gcrích clár agus tascanna eile dá leithéid. Déantar iad seo a fhorghníomhú ón gcomhéadan líne ordaithe ag úsáideoir nó i mbaisc chun oibríochtaí iolracha a dhéanamh ag an am céanna. Is féidir comhaid scripte a oscailt in eagarthóirí téacs mar Notepad, Notepad ++, Vim, Apple Terminal agus feidhmchláir eile dá samhail ar Windows, MacOS agus Linux OS.

## Formáid Chomhaid SH

Scríobhtar comhaid SH i ngnáth-théacs de réir na comhréire sainithe. Tacaíonn na comhaid scripte seo le:

 * `Tuairimí` - Tosaíonn tuairimí le # agus déanann an bhlaosc neamhaird orthu.
 * `Aicearraí` - Is féidir iad seo a úsáid chun ordú a athainmniú le haghaidh forghníomhaithe gearr agus éasca.
 * `Baisc Jobs` - Is féidir roinnt orduithe a fhorghníomhú go huathoibríoch ar ghá iad a chur isteach de láimh murach sin. Fágann sé seo nach gá fanacht le húsáideoir gach céim den seicheamh a spreagadh.
 * `Ginearálú` - Trí lúba simplí a úsáid, baintear i bhfad níos mó ginearálú amach maidir le hoibríochtaí cosúil le híomhánna a thiontú ó cheann go ceann eile.

## Sampla comhad SH

```
$ echo '#!/bin/sh' > my-script.sh
$ echo 'echo Hello World' >> my-script.sh
$ cat my-script.sh
#!/bin/sh
echo Hello World
$ chmod 755 my-script.sh
$ ./my-script.sh
Hello World
```
## Conas comhad SH a rith?
De ghnáth ritheann na comhaid SH ar Linux, fiú amháin i Windows ní mór duit nascadh le críochfort Linux ag baint úsáide as bogearraí mar Putty chun na comhaid sh a rith. Seo a leanas na céimeanna chun comhad SH a rith ar chríochfort Linux.

1. Oscail críochfort Linux agus téigh chuig an eolaire ina bhfuil an comhad SH suite.
2. Trí úsáid a bhaint as ordú `chmod`, socraigh cead forghníomhaithe ar do script (mura bhfuil sé socraithe cheana féin).
3. Rith an script ag baint úsáide as ceann amháin díobh seo a leanas
	1. `./ainm comhaid.sh`
	2.  `sh comhadainm.sh`
	3.  `bash script-ainm-seo.sh`
