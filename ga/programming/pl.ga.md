## Cad is comhad PL ann?

Comhad le síneadh .pl is ea comhad Perl Script ar teanga scriptithe é. Déantar iad seo a thiomsú agus a rith ag baint úsáide as bogearraí Perl Interpreter. Tá línte de chód, athróga, agus tuairimí i gcomhad script PL. Tá sé sách deacair teangacha scripte a dhéanamh
formáidí comhaid ríomhchlárúcháin eile ar nós [PHP](/programming/php/) a thuiscint. Is féidir comhaid PL a chruthú agus tá siad comhoiriúnach le Windows, macOS, agus Linux.

## Stair gairid ar Theanga Scriptithe Perl

Coinníodh an teanga seo in úsáid den chéad uair i 1987, agus mar sin fuair na comhaid seo a mbunús an bhliain sin. Sa bhliain 1989 eisíodh an Perl 2. Níos déanaí, tá sé nuashonraithe agus modhnaithe suas go dtí an leagan 5.35, agus tá an chéad leagan eile dírithe le scaoileadh an bhliain seo chugainn.

I ngach nuashonrú, tá uirlisí curtha leis maidir le feidhmiúlacht, feidhmíocht, agus cuma na teanga agus na gcomhad. Is iomaí leasú a rinneadh ar na leaganacha sna blianta seo. Bunteanga scriptithe a bhí inti ar dtús ach anois tá go leor modúl eile inti freisin. Teanga an-simplí a bhí inti ar dtús, ach bhí scripteanna na teanga seo deacair go leor a thuiscint mar go raibh siad dlúth.

## Sonraíochtaí Síneadh Formáid Comhaid Perl

Tá roinnt airíonna nó sonraíochtaí na gcomhad ríomhchlárúcháin seo, agus tá cuid acu mar seo a leanas:

* Tá an cód atá sna comhaid seo i bhformáid gnáth-théacs agus úsáidtear é chun script a fhorbairt
* Is iad scripteáil an fhreastalaí, parsáil téacs, agus riarachán freastalaí na príomhghnéithe a chlúdaíonn script na teanga seo
* Is iomaí clár móréilimh a bhaineann leis an teanga seo ná Active state Active Perl agus Bare Bones BBEdit (comhoiriúnach le Mac OS)
* Breathnaítear ar an teanga seo mar theanga ardleibhéil
* I gcás Win32, seans go mbeidh ar an úsáideoir dáileadh dúchais na teanga a shuiteáil
* Éilíonn sé go mbeidh roinnt uirlisí scriptithe inrite i bhFeisteáin Acmhainní Windows éagsúla
* Is creat cáiliúil é Visual Studio .NET chun teangacha ríomhchlárúcháin a fhorbairt. Úsáidtear an uirlis Staid Ghníomhach ar a dtugtar Visual Perl chun Perl a chur le Visual studio
* Sna comhaid, déanann an chéad líne den bhunchód cur síos ar fhaisnéis an ateangaire atá in úsáid. Tosaíonn na comhaid PL de ghnáth le líne **#!/usr/bin/perl** a insíonn do do ríomhaire an script seo a rith ag úsáid ateangaire Perl atá suiteáilte sa ríomhaire


## Samplaí Scripteanna PL

```
#!/usr/bin/perl
print "Hello, world\n";
```

Beidh sé seo a phriontáil

```
Hello, World
```

### Nóta tráchta aon líne ###

```
#!/usr/bin/perl
# This is a single line comment
print "Hello Perl\n";
```

### Trácht illíne ###

```
#!/usr/bin/perl
=begin comment
This is a multiline comment.
Line 1
Line 2
=cut
print "Hello Perl\n";
```

### Tasc inathraithe ###

```
#!/usr/bin/perl
$a = 10;
print "Variable a = $a\n";
```

### Tasc athróg scálach ###

```
#!/usr/bin/perl
$age = 35; # Assigning an integer
$name = "Tony Stark"; # Assigning a string
$pi = 3.14; # Assigning a floating point
```

### Oibríochtaí scálach simplí ###

```
#!/usr/bin/perl
$constr = "hi" . "perl";# Concatenates two or more strings.
$add = 40 + 10; # addition of two numbers.
$prod = 4 * 51;# multiplication of two numbers.
$connumstr = $constr . $add;# concatenation of string and number.
```
