## Cad is comhad DÉANAMH?

Comhad scripte XCode is ea comhad MAKE a eagraíonn tiomsú an chóid. Úsáidtear é chun cláir a thiomsú agus a nascadh ó chomhaid chód foinse iolracha. Cuidíonn sé seo le cinneadh a dhéanamh ar na codanna d'fheidhmchlár mór atá le cur le chéile nuair is gá an feidhmchlár a nuashonrú. Déan comhaid úsáid as sraith treoracha le rith ag brath ar na comhaid a d'athraigh.

## DÉANAMH Sampla Formáid Comhaid

Déantar an t-ábhar seo a leanas a fhorghníomhú ag baint úsáide as an áirgiúlacht Make agus tá na haschuir mar a leanas.

```
hello:
	echo "hello world"
```
**Déan Aschur**

```
$ make
echo "hello world"
hello world
```
