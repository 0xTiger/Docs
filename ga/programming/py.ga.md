## Cad is comhad PY ann? ##

Tá cód foinse Python sna Comhaid leis an síneadh .py. Tá an teanga Python éirithe an-cháiliúil teanga anois le lá. Is féidir é a úsáid le haghaidh scriptithe córais, forbairt gréasáin agus bogearraí agus matamaitic. Tacaíonn Python comhoiriúnacht tras-ardán; Ciallaíonn sé gur féidir leis na feidhmchláir fhorbartha i Python oibriú ar ardáin éagsúla cosúil le Windows, MAC, Linux, Raspberry Pi, etc. Soláthraíonn Python comhréir shimplí agus éasca le léamh atá cosúil leis an mBéarla. Is féidir le forbróirí feidhmchlár bogearraí réasúnta a scríobh trí chúpla líne de chód Python a scríobh. Ós rud é go ritheann an Python ar chóras ateangaire, mar sin is féidir an cód a fhorghníomhú chomh luath agus atá sé scríofa a fhágann go bhfuil sé an-mhaith le haghaidh fréamhshamhlú.

## Conas comhad py ## a rith

Chun an leagan de Python suiteáilte a sheiceáil, is féidir leat an t-ordú seo a leanas a úsáid:

```cmd
python --version
```

Beidh sé seo aschur an leagan ar an consól mar

```cmd
Python 3.7.4
```

Mura bhfuil Python suiteáilte ar do ríomhaire, is féidir leat dul go dtí [python.org](https://www.python.org/) agus Python a íoslódáil agus a shuiteáil do do chóras oibriúcháin ábhartha.

Chun script Python a rith, is féidir leat an t-ordú seo a leanas a úsáid:

```cmd
python helloworld.py
```

Is comhad scripte é *helloworld.py* ina bhfuil an cód seo a leanas

```py
print("Hello World from Python")
```

Déanfaidh sé seo an méid seo a leanas a phriontáil ar fhuinneog an chonsóil.

```cmd
Hello World from Python
```

Nóta: Má úsáideann tú IDEanna, soláthraíonn siad cnaipí ar an scáileán nó aicearraí méarchláir éagsúla chun Python a rith. Mar shampla, taispeántar cnaipe súgartha sa gutter leis an eagarthóir i PyCharm a ligeann duit an script Python a fhorghníomhú.

## Formáid Chomhaid PY ##

Dearadh Python don inléiteacht agus tá cosúlachtaí aige le Béarla agus Mata. Úsáideann Python línte nua chun an t-ordú iomlán a chur in iúl i gcomparáid le leathstadanna nó lúibíní a úsáidtear i dteangacha eile. Le haghaidh scóip, lúba, agus feidhmeanna, braitheann Python ar eangú agus spásanna bána i gcomparáid le braces curly a úsáidtear i dteangacha eile.

### Comhréir ###

Seo a leanas sampla de chomhréir Python.

```py
print("Hello World")

#Variables
name = "John"
age = 25
print(name)
print(age)

#While Loop
i = 1
while i < 3:
  print(i)
  i += 1
  
#For Loop
names = ["John", "Harry", "Tom"]
for name in names:
  print(name)
```
