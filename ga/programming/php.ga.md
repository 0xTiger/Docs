## Cad is comhad PHP ann?
Tagraíonn comhad le síneadh .php do theanga ríomhchláraithe foinse oscailte, a úsáidtear chun scripteanna freastalaí a scríobh, le cur i gcrích ar fhreastalaí gréasáin. Is í an teanga scriptithe gréasáin is forleithne a úsáidtear go hiondúil chun feidhmchláir ghréasáin ar mhórscála a fhorbairt. Tá an córas blagála is mó ar an ngréasán ie WordPress agus an líonra sóisialta is mó Facebook bunaithe ar theanga PHP. Tá sé ag luí le hardáin éagsúla cosúil le Windows, Linux, Unix, Mac OS X, srl. Chomh maith leis sin ritheann sé go rathúil ar beagnach gach freastalaithe a úsáidtear inniu ie Apache, IIS, etc Tacaíonn PHP le raon leathan bunachar sonraí lena n-áirítear MySQL.

## Formáid Chomhaid PHP ##

Féadfaidh téacs, CSS, [HTML](/web/html/), JavaScript, agus cód PHP a bheith i PHP amháin. Déantar cód PHP a fhorghníomhú ar an bhfreastalaí, agus cuirtear an toradh ar ais chuig an mbrabhsálaí mar HTML simplí. Is féidir comhaid PHP a oscailt le haon eagarthóir téacs agus iad a chur in eagar, cé go dtugann feidhmchláir cosúil le Adobe Dreamweaver, uirlisí Forbartha Eclipse PHP bealach áisiúil chun cód PHP a scríobh agus a mhodhnú.

### Comhréir ###

Déantar script PHP a fhorghníomhú ar an bhfreastalaí, agus seoltar an gnáththoradh HTML ar ais chuig an mbrabhsálaí. Tosaíonn script PHP le<?php and ends with ?>

### PHP Sampla ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
