## Cad is comhad APPXUPLOAD ann?

Is comhad pacáiste aonair é comhad APPXUPLOAD ina bhfuil il-phacáistí aip nó beart aipeanna le huaslódáil chuig Microsoft Store. Gintear go huathoibríoch é ag [Microsoft Visual Studio](https://visualstudio.microsoft.com/) nuair a dhéanann forbróirí a n-aip a phacáistiú le foilsiú chuig [Microsoft Store](https://apps.microsoft.com/store/apps). Chomh maith le comhad [APPX](/programming/appx/), tá comhad siombailí sa chomhad APPXUPLOAD a dhéanann anailís ar fheidhmíocht aipe tar éis é a fhoilsiú ar an Microsoft Store.
