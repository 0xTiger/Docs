## Cad is comhad INO ann?

Baineann **comhad INO** le **Arduino Sketch** ar clár é a scríobhadh do bhoird Arduino ag baint úsáide as an **teanga ríomhchlárúcháin Arduino**. Tá teanga ríomhchlárúcháin Arduino bunaithe ar Sreangú; agus tá sé cosúil le C/C++. Sábháltar Sceitsí Arduino le síneadh comhad .ino agus úsáidtear iad chun cláir chiorcaid Arduino a rialú.

## Formáid Chomhaid INO - Tuilleadh Eolais

Cuireann an t-ardán Arduino, atá deartha le haghaidh fréamhshamhaltú leictreonach, ar chumas forbróirí cláir, dá ngairtear sceitsí, a cheardú agus a imscaradh ar chláir chiorcaid speisialaithe; Cuimsíonn na sceitsí seo ó thascanna bunúsacha cosúil le stiúir a shoilsiú ar chlár ciorcad agus oibríochtaí casta ar nós sonraí a tharchur thar an idirlíon; tarlaíonn cruthú sceitsí Arduino laistigh de **Timpeallacht Forbartha Comhtháite Arduino (IDE)**, agus stóráiltear na sceitsí seo leis an síneadh comhad .ino.

Sa ré roimh Arduino 1.0, sábháladh sceitsí le síneadh comhad .PDE ag an Arduino IDE. Cé go dtacaíonn an IDE le hoscailt comhaid .PDE fós, is é an cleachtas molta ná gach sceitse a shábháil, lena n-áirítear iad siúd a stóráiltear mar chomhaid [.PDE](/programming/pde/) ar dtús, leis an síneadh comhad .ino. Léiríonn an t-aistriú seo éabhlóid an ardáin agus cinntíonn sé comhoiriúnacht leis na caighdeáin reatha.

## Maidir le Arduino

Is ardán leictreonaic foinse oscailte é Arduino a shimplíonn fréamhshamhlú agus forbairt tionscadal idirghníomhach; a chuimsíonn gnéithe crua-earraí agus bogearraí araon, freastalaíonn Arduino ar bhonn éagsúil úsáideoirí, ó dhaoine nua-aimseartha go daoine gairmiúla. Is é croí an ardáin na boird Arduino, atá feistithe le bioráin dhigiteacha agus analógacha, nascacht USB agus samhlacha éagsúla bunaithe ar phróiseálaithe cosúil le Atmel AVR nó ARM.

Feidhmíonn an Arduino IDE mar chomhéadan atá éasca le húsáid chun cláir (sceitsí) a chódú, a thiomsú agus a uaslódáil chuig na boird; tá an teanga ríomhchlárúcháin, cosúil le C/C++, deartha le haghaidh inrochtaineachta, ag baint le castachtaí do thosaitheoirí. Soláthraíonn leabharlanna Arduino gearrthóga cód réamhscríofa le haghaidh tascanna ó chomhtháthú braiteoirí go prótacail chumarsáide, ag éascú forbairt éifeachtach.

Leathnaíonn sciatha Arduino, cláir bhreise, feidhmiúlachtaí, ag tairiscint gnéithe cosúil le nascacht Ethernet nó rialú mótair. Cuireann pobal bríomhar Arduino le bonn leathan eolais trí fhóraim, ranganna teagaisc agus tionscadail chomhoibríocha; le feidhmchláir a chuimsíonn róbataic, uathoibriú tí, suiteálacha ealaíne agus córais braiteoir-bhunaithe, fágann solúbthacht agus simplíocht Arduino an rogha tosaíochta do thionscadail leictreonacha éagsúla.

## Conas comhad INO a oscailt?

Is féidir **Timpeallacht Chomhtháite Forbartha Arduino** a úsáid chun comhad INO a oscailt. Ach ós rud é gur comhaid gnáth-théacs iad, is féidir aon eagarthóir téacs a úsáid chun a ábhar a oscailt nó a chur in eagar.
