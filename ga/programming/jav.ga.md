## Cad is comhad Jav ann?

Is comhad cód foinse Java é comhad Jav a cruthaíodh i dteanga ríomhchlárúcháin Java. Tá sé cosúil le comhad foinse Java agus is féidir é a chur le chéile le ateangairí Java. Cosúil le comhaid [Java](/programming/java/), úsáidtear comhad Jav chun feidhmchláir Java a fhorbairt. Féadfaidh sé a bheith ar cheann de na comhaid cód foinse iomadúla atá ar fáil i dtionscadal Java.

## Formáid Comhaid Java

Stóráiltear comhaid Jav mar chomhaid téacs agus is féidir iad a oscailt le haon eagarthóir téacs. Tá an cineál comhaid Java curtha in ionad cineálacha comhaid Jav go coitianta agus úsáidtear é go coitianta chun feidhmchláir Java a scríobh. Is féidir cóid Jav a scríobh le IDE cosúil le Eclipse agus IntelliJ Idea a fhágann go mbeidh sé níos éasca oibriú le comhaid Java.

### Sampla Formáid Comhaid Jav

```java
/*
The example code prints
Hello World from Java to the console.
*/
    public class JavaApp {
        public static void main(String[] args) {
            System.out.println("Hello World."); // Prints the string to the console.
        }
    }
```
