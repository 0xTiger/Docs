## Cad is comhad RB ann?

Is gnách go mbíonn síneadh comhaid .rb bainteach le comhaid teanga ríomhchlárúcháin Ruby. Is teanga ríomhchlárúcháin dhinimiciúil, réad-dhírithe í Ruby a bhfuil aithne uirthi as a simplíocht agus a inléiteacht.

Tá cód foinse Ruby i gcomhad .rb, ar féidir le teangaire Ruby nó timpeallacht forbartha Ruby é a fhorghníomhú. Is minic go mbíonn sainmhínithe sna comhaid seo ar ranganna, modhanna, athróga, agus tógálacha cód Ruby eile.

## Conas script Ruby a rith i gcomhad RB?

Chun script Ruby a shábháil i gcomhad .rb a rith, beidh ateangaire Ruby uait suiteáilte ar do chóras. Seo sampla bunúsach de script Ruby a sábháladh i gcomhad darb ainm example.rb:

```
# example.rb

# Define a method to calculate the square of a number
def square(number)
  number * number
end

# Call the square method with an argument
result = square(5)

# Print the result
puts "The square of 5 is: #{result}"
```

Chun an script seo a rith, is féidir leat do líne ordaithe nó teirminéal a oscailt, dul chuig an eolaire ina bhfuil an comhad example.rb suite agus ansin ateangaire Ruby a úsáid:

```
ruby example.rb
```

Ritheann an t-ordú thuas an script agus taispeánfar an t-aschur sa chonsól:

```
The square of 5 is: 25
```

Is sampla simplí é seo ach is féidir cód níos casta a bheith i gcomhaid .rb, lena n-áirítear ranganna, modúil agus struchtúir rialaithe, rud a ligeann duit feidhmchláir Ruby iomlán a thógáil.

## Cad atá i gcomhad RB?

Féadfaidh inneachar sonrach an chomhaid .rb a bheith éagsúil ag brath ar a chuspóir agus ar an ríomhchláraitheoir a scríobh é. Mar sin féin, go ginearálta, tá cód foinse Ruby i gcomhad .rb, atá comhdhéanta de shraith treoracha ar féidir leis an ateangaire Ruby a thuiscint agus a fhorghníomhú.

Seo roinnt gnéithe coitianta a d'fhéadfá a fháil i gcomhad Ruby:

- **Tuairimí:** Tacaíonn Ruby le tuairimí aonlíne agus illíne araon. Úsáidtear tuairimí chun nótaí míniúcháin a chur leis nó chun línte sonracha cód a dhíchumasú ón bhforghníomhú. Tá siad sainithe ag an # charachtar.

```
# This is a single-line comment

=begin
This is a
multi-line comment
=end
```

- **Dearbhuithe Athróg:** Is teanga chlóscríofa dinimiciúil í Ruby, mar sin ní gá dearbhuithe sainráite cineáil a dhéanamh le hathróga. Is féidir leat luachanna a shannadh d'athróga trí úsáid a bhaint as oibreoir an taisc (=).

- **Modhanna:** Úsáideann Ruby an eochairfhocal `def` chun modhanna a shainiú, ar bloic cód ath-inúsáidte iad a dhéanann tascanna sonracha.

- **Ranganna agus Oibiachtaí:** Is teanga réad-dhírithe í Ruby, agus úsáidtear ranganna chun gormchlónna oibiachta a shainiú. Is samplaí de ranganna iad réada agus is féidir tréithe (athróga cásanna) agus iompraíochtaí (modhanna cásanna) a bheith acu.

- ** Struchtúir Rialaithe:** Soláthraíonn Ruby struchtúir rialaithe éagsúla cosúil le ráitis choinníollacha (más rud é, eile, elsif, mura rud é), lúba (agus, go dtí, do gach ceann acu), agus níos mó, chun an sreabhadh forghníomhaithe sa chlár a rialú.

```
if age >= 18
  puts "You are an adult."
else
  puts "You are a minor."
end

# Output: You are an adult.
```

## Cad é formáid an chomhaid RB?

Gnáth-théacs is ea formáid an chomhaid .rb”, agus ionchódaítear é de ghnáth trí úsáid a bhaint as ionchódú UTF-8 nó ASCII. Leanann sé comhréir agus struchtúr ar leith atá sainmhínithe ag teanga ríomhchláraithe Ruby.

## Cad é an cineál MIME den chomhad RB?

- `iarratas/x-ruby`
