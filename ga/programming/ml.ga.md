## Cad is comhad ML ann?

Is comhad téacs é comhad ML a cruthaíodh leis an teanga ríomhchláraithe feidhme ginearálta ML (ar a dtugtar Meta Language go minic). Tá cód foinse ML ann a úsáidtear níos minice i dtaighde agus i ndearadh teanga ríomhchlárúcháin agus úsáid á baint as séimeantaic atá sainmhínithe go hiomlán. Tá éagsúlachtaí iomadúla idir ML agus Caml mar an ceann a úsáidtear go forleathan. Áirítear le cinn eile Cuspóir Caml (OCaml), JoCaml, ML Leathnaithe, agus MacroML. Úsáidtear teanga ML go coitianta le haghaidh scríbhneoireacht tiomsaithe, cruthú teoirim uathoibrithe, agus fíoruithe foirmiúla.
