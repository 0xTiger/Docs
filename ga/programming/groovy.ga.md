## Cad is comhad GROOVY ann?

Is comhad cód foinse é comhad GROOVY atá scríofa i dteanga ríomhchlárúcháin Groovy. Tá an cód scríofa taobh istigh de chomhad GROOVY cosúil leis an teanga atá dírithe ar oibiachtaí [Java](/programming/java/), rud a fhágann go bhfuil sé éasca feidhmchláir a dhearadh agus a fhorbairt. Tiomsaítear comhad GROOVY do sheachchód meaisín fíorúil Java (JVM) agus tá sé comhoiriúnach le cód agus leabharlanna Java eile.

## Formáid Chomhaid GROOVY - Tuilleadh Eolais

Tá cód foinse scríofa i gcomhréir Groovy i gcomhaid GROOVY. Is féidir cód a bheith ann is féidir a úsáid mar theanga ríomhchlárúcháin chomh maith le teangacha scriptithe don Ardán Java. Tá na gnéithe cosúil le gnéithe Python, Ruby, agus Smalltalk ag teanga Groovy.
