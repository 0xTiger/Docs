## Cad is comhad F95 ann?

De ghnáth léiríonn comhad le síneadh .f95 gur comhad cód foinse Fortran 95 é. Is teanga ríomhchláraithe ilfheidhmeach ríthábhachtach í Fortran atá feiliúnach go háirithe don ríomhaireacht uimhriúil agus eolaíoch. Is leagan feabhsaithe de Fortran 90 é Fortran 95, a thugann roinnt gnéithe agus feabhsuithe nua isteach.

Go hiondúil bíonn treoracha cláir scríofa i dteanga ríomhchlárúcháin Fortran i gcomhaid cód foinse Fortran; má tá comhad agat le síneadh .f95, is dócha go gcuirfear le chéile é trí úsáid a bhaint as tiomsaitheoir Fortran; Is éard is tiomsú ann ná an próiseas ina n-aistrítear cód foinse inléite ag an duine go cód dénártha atá inléite ag meaisín agus is féidir le ríomhaire a chur i gcrích.

## Conas comhad F95 a oscailt?

Más mian leat oibriú le nó clár a rith i gcomhad .f95, beidh tiomsaitheoir Fortran uait; I measc na dtiomsaitheoirí Fortran coitianta tá GNU Fortran (gfortran), Intel Fortran Compiler (ifort), agus daoine eile. Is féidir leat cláir Fortran a thiomsú agus a rith ag baint úsáide as orduithe cuí a sholáthraíonn tiomsaitheoir.

Mar shampla, má tá gfortran suiteáilte agat, is féidir leat na horduithe seo a leanas a úsáid i teirminéal chun clár Fortran a thiomsú agus a rith:

```
# Compile the Fortran program
gfortran -o my_program my_program.f95

# Run the compiled program
./my_program
```

Mura dteastaíonn uait ach féachaint ar a bhfuil sa chomhad F95, is comhad gnáth-théacs é, ionas gur féidir leat é a oscailt le haon eagarthóir téacs.
