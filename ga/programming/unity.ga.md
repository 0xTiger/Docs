## Cad is comhad UNITY ann?

Is comhad radhairc 3D é comhad Unity a cruthaíodh leis an inneall Cearrbhachais 3D, **Unity**. Úsáidtear é i gcomhcheangal le radhairc eile chun domhan fíorúil a chruthú agus eagarthóireacht á déanamh. Tá gnéithe radharc 3D ann mar rudaí, timpeallachtaí, maisiúcháin, ceamaraí, faisnéis uillinneacha, cásanna soilsithe, agus constaicí.

Is féidir comhaid Unity a oscailt le Unity Technologies Unity. Is féidir le Unity comhaid Unity [.fbx](/3d/fbx/), [.dae](/3d/dae/) (Colada), [.3ds](/3d/3ds/), [.dxf](/cad/dxf/), agus [.obj](/3d/obj/) a léamh freisin.

## Formáid Chomhaid UNITY

Stóráiltear comhaid aontacht go diosca i bhformáid dhénártha comhaid. Déantar iad seo a ghiniúint agus a úsáid in éineacht le radhairc eile le linn eagarthóireachta. Úsáidtear Unity chun cluichí a fhorbairt do Windows chomh maith le Mac OS X, agus is féidir é a úsáid chun cluichí 3D a chruthú don eweb, Nintendo Wii, iPhones, nó feidhmchláir neamhspleácha.
