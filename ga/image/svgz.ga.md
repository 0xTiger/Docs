## Cad is comhad SVGZ ann?

Is leagan comhbhrúite de chomhad Grafaic Veicteoir Inscálaithe (.SVG) é comhad le síneadh .svgz. Tá sé comhbhrúite le comhbhrú gzip agus tá sonraí i bhformáid [XML](/web/xml/) ann. Tacaíonn comhaid SVGZ le trédhearcacht, grádáin, beochan agus scagairí. Tá comhaid SVGZ níos lú i méid i gcomparáid leis na comhaid SVG réamhshocraithe agus cuidíonn an méid comhaid laghdaithe seo na comhaid ghrafaic a aistriú ar líne. Cruthaíonn dearthóir grafaic comhaid SVGZ ag úsáid uirlisí mar Adobe Illustrator, Corel PaintShop Pro, agus cinn eile. Mar sin féin, is féidir comhaid SVGZ a ghiniúint trí [GZip compression](/compression/gz/) a chumasú i bhFreastalaí Apache agus na sonraí íomhá á seoladh amach.

## Formáid Chomhaid SVGZ

SVGZ, being compressed [SVG](/page-description-language/svg/), is based on open compressed gzip format that reduces the size of SVG documents. This compressed archive contains plain text SVG documents inside that are purely based on the XML file format (/web/xml/). SVGZ files can be opened by viewers/applications that conform to the SVG 1.1 specifications. SVGZ files 20-50% smaller in size than the original SVG files.
