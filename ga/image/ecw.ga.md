## Cad is comhad ECW ann? ##
Is éard atá i gcomhad ECW ná íomhá atá deartha le húsáid le sonraí geospásúla. Is íomhá comhbhrúite iarbhír í an íomhá ina bhfuil a cruthaíodh i bhformáid Wavelet Comhbhrú Feabhsaithe. Go ginearálta úsáidtear comhaid ECW chun teilgean léarscáileanna satailíte agus aeríomhánna a stóráil. Chun tacú le feidhmchláir gheospásúla, is féidir faisnéis teilgean léarscáile a leabú i gcomhaid ECW.

## Formáid comhaid ECW
Is formáid íomhá dílseánaigh é ECW atá bunaithe ar chomhbhrú tonnlet; optamaithe le haghaidh íomhánna satailíte agus aerghrianghrafadóireacht. Is le Intergraph cuid de Hexagon AB an fhormáid seo anois ach go bunúsach is é Earth Resource Mapping a d’fhorbair í. Is formáid chomhbhrú chaillte é ECW a chomhbhrúíonn íomhánna mórmhéide le codarsnacht mhín eile agus a gcáilíocht amhairc á gcoimeád ag an am céanna.
 
### Airíonna
Seo roinnt airíonna tipiciúla formáid ECW:
- Is féidir faisnéis teilgean léarscáile a chur isteach i bhformáid comhaid ECW chun tacú le feidhmchláir gheospásúla.
- Is féidir sonraí íomhá suas le 65,535 banna lena n-áirítear sraitheanna nó dathanna a chomhbhrú isteach i bhformáid comhaid ECW ag ráta os cionn 25 MB in aghaidh an tsoicind.
- Is prótacal sruthaithe éifeachtach é prótacal ECWP a úsáidtear chun íomhánna ECW agus JPEG2000 a tharchur thar líonraí.
- Tá SDK tapa ag tacú le ECW agus JPEG2000 ar fáil saor in aisce le haghaidh feidhmiú deisce le haghaidh Linux, Windows agus MacOSX
