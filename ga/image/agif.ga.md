## Cad is comhad AGIF ann?

Is comhad GIF é comhad AGIF a cruthaíodh le Gníomhach GIF Creator. Is comhad tionscadail é is féidir a oscailt agus a fheiceáil i dtionscadail Active GIF Creator. Nuair a osclaítear é in Gníomhach GIF Creator, taispeánann sé seicheamh na n-íomhánna a dhéanann suas ar deireadh an comhad GIF críochnaithe le síneadh .gif.

## Formáid Chomhaid AGIF

Is comhad tionscadail é AGIF a shábháiltear mar chomhaid íomhá raster. Is féidir an comhad [GIF](/image/gif/) críochnaithe a oscailt agus a thaispeáint mar bheochan. Tá API Python ar fáil ar líne ar féidir iad a úsáid chun comhaid GIF beoite a chruthú.
