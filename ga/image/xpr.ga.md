## Cad is comhad XPR ann?

Is comhad íomhá veicteora é comhad XPR a cruthaíodh leis na bogearraí Microsoft Expression Graphics (EGD) atá scortha anois. Tá faisnéis ghrafaice ann ar féidir é a úsáid mar mhúnla comhéadan úsáideora. Cuireadh comhaid dearaidh in Microsoft Expression Design in ionad na gcomhad XPR níos déanaí. D'fhéadfaí comhaid XPR a oscailt le Microsoft Expression Studio atá scortha le tamall maith anois.

## Leochaileacht Formáid Comhaid XPR

Aithníodh comhaid XPR chun leasa leochaileachta i Microsoft Expression Design, rud a fhágann gur féidir cianchód a fhorghníomhú. Sa tsaincheist a tuairiscíodh, dá n-osclódh úsáideoir comhad .xpr dlisteanach atá brataithe san eolaire líonra mar aon le comhad leabharlainne nasc dinimiciúil (DLL), d'fhéadfadh Microsoft Expression Design iarracht a dhéanamh an comhad DLL a luchtú agus an cód atá ann a fhorghníomhú. Shocraigh Microsoft é seo níos déanaí i nuashonrú slándála.
