You must be familiar with **PPTX** and **PPT** extension files, these are Presentation file formats that store collection of records to accommodate presentation data such as: slides, shapes, text, animations, video, audio and embedded objects. A presentation can be saved/converted into other file formats as well such as PDF, BMP, PNG, JPEG, and XPS.

I measc na n-eisínteachtaí comhaid Common Presentation agus na formáidí comhaid a bhaineann leo tá **[PPTX](/presentation/pptx/)**, **[PPT](/presentation/ppt/)** agus **[ODP](/presentation/odp/)**.

An bhfuil ceisteanna agat maidir le formáidí comhaid Léirithe? Téigh ar aghaidh chuig ár bpobal [forums](https://forum.fileformat.com/c/presentation/8) chun leas a bhaint as an eolas atá roinnte ag saineolaithe Formáid Chomhaid.

## API Foinse Oscailte do Chomhaid Léirithe

Seiceáil an liosta de [Open Source APIs for working with Presentation File Formats](https://products.fileformat.com/presentation/).

## Liosta Síntí Comhad Láithreacha agus Formáidí Comhaid Ghaolmhara

Seo a leanas liosta de na síntí comhaid PowerPoint a bhfuil tóir orthu agus síntí comhaid eile le Léiriú agus na formáidí comhaid a bhaineann leo.
