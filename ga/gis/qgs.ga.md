## Cad is comhad QGS ann?

A file with .qgs extension is a project file created with [QGIS](https://www.qgis.org/en/site/), which is an open-source cross-platform GIS application. All the project settings such as layer properties and auxiliary data for the project. It is created when a map project is saved to disc. It is actually a configuration file that retains information such as pointers to the GIS data, styling information about the layers, and project title. QGS files can be opened with QGIS software that is freely available to download.

## Formáid Chomhaid QGS

Tá comhaid QGS i bhformáid [XML](/web/xml/) agus stóráiltear sonraí tionscadal mar chlibeanna XML. Tá faisnéis i gcomhad QGS mar:

 * teideal an tionscadail
 * tionscadal CRS
 * an crann ciseal
 * socruithe snapping
 * caidreamh
 * fairsinge an chanbhás léarscáil
 * samhlacha tionscadail
 * finscéal
 * duganna mapview (2D agus 3D)
 * na sraitheanna a bhfuil naisc acu leis na tacair shonraí bhunúsacha (foinsí sonraí) agus airíonna ciseal eile lena n-áirítear méid, SRS, nascanna, stíleanna, rindreálaí, modh cumaisc, teimhneacht agus go leor eile.
 * airíonna tionscadail
