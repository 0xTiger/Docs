## Cad is comhad SDF ann?

A Spatial Data File (SDF) is a Geodatabase file format that was developed by Autodesk. It stores geospatial data and is used by Autodesk GIS program MapGuide and AutoCAD Map 3D. SDF file format is based on the single file database file format SQLite3. Breathnaítear air mar fhormáid comhaid optamaithe le haghaidh tacair shonraí móra faisnéise spáis.

Is féidir leat comhad SDF a oscailt ag baint úsáide as Autodesk AutoCAD Map 3D 2022 agus Autodesk AutoCAD Civil 3D 2023.

## Formáid Chomhaid SDF - Tuilleadh Eolais

Stóráiltear formáid comhaid SDF go diosca mar chomhaid dhénártha. Tá sé bunaithe ar bhunachar sonraí SQLite a úsáideann comhpháirteanna stórála ar leibhéal íseal le haghaidh sraithiú dénártha sonraí. I gcomhad SDF tá ilranganna gné in aghaidh an chomhaid agus airíonna iolracha céimseata do gach gné-aicme. Tá rochtain ar shonraí i gcomhad SDF ar ardluas mar gheall ar innéacsú gach airí céimseata ag baint úsáide as R-crann.
