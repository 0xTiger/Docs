## Cad is comhad DRC ann?

Is formáid comhaid 3D comhbhrúite é comhad le síneadh .drc a cruthaíodh le leabharlann Google Draco. Cuireann Google Draco ar fáil mar leabharlann foinse oscailte chun mogaill gheoiméadracha 3D agus scamaill phointe a chomhbhrú agus a dhí-chomhbhrú, agus feabhsaíonn sé stóráil agus tarchur grafaicí 3D. Ionchódaíonn sé na sonraí ionchuir agus sábhálann sé mar chomhad .drc é. Ag an deireadh glactha, léann an API comhaid .drc agus is féidir leis iad seo a aschur mar chomhaid [PLY](/3d/ply/) nó [OBJ](/3d/obj/). Cuireann an comhad aschuir comhbhrúite ar chumas úsáideoirí aipeanna a íoslódáil níos tapúla agus lódáil tapa ar ghrafaic 3D i mbrabhsálaithe.

## Formáid Chomhaid DRC

File format details for DRC are not available publicly. Google has hosted the Draco library on [GitHub](https://github.com/google/draco/tree/master/docs) for developer's reference. It provides encoders and decoders for creating and converting DRC files.
