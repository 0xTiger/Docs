## Cad is comhad PAK ann?

Is comhad pacáiste é comhad PAK a chruthaíonn físchluichí éagsúla chun sonraí cluiche a chartlannú. Go bunúsach, is formáid comhaid cluiche é. D’fhéadfadh go n-áireofaí leis seo gnéithe iolracha cluiche ar nós grafaicí, uigeachtaí, fuaimeanna, réada, chomh maith le sonraí eile cluiche. Déantar an chartlann a shábháil mar chomhad .zip den chuid is mó agus is féidir í a bhaint as bogearraí dí-chomhbhrúite a bhfuil tóir orthu mar WinZip agus WinRAR. I measc samplaí de chluichí físeáin a úsáideann comhaid PAK tá Quake, Hexen, Crysis, agus Half-Life.

## Formáid Chomhaid PAK - Tuilleadh Eolais

Go hiondúil, déantar comhaid PAK a shábháil in [ZIP file format](/compression/zip/). Ach féadfaidh feidhmchláir éagsúla formáid comhaid éagsúla a úsáid chun na comhaid seo a stóráil.


## Conas comhaid PAK a oscailt?

Is féidir leat comhaid PAK a oscailt le feidhmchláir ar nós [PakExplorer](https://www.quaketerminus.com/tools.shtml) agus [SpriteExplorer](http://www.slackiller.com/hlprograms.htm).

## Formáid Chomhaid PAK - Comhad Oibiachta Simutrans

Is formáid comhaid cluiche insamhalta iompair [Simutrans](https://www.simutrans.com/en/) é comhad a bhfuil síneadh .pak aige. Tá rudaí ann a úsáidtear san ionsamhlúchán ar nós grafaic a rinne an t-úsáideoir agus inneachar sonraí. Féadann go leor rudaí éagsúla a bheith aige, mar shampla feithiclí cluiche, foirgnimh, tír-raon, srl. Gintear comhaid PAK ag baint úsáide as MakeObject, áirgiúlacht a thiomsaíonn comhaid .dat agus pictiúir .png chun na rudaí insamhalta seo a chruthú. Ligeann Simutrans do na himreoirí córas iompair rathúil a reáchtáil trí chórais iompair a thógáil agus a bhainistiú do phaisinéirí, post agus earraí de thalamh

## Conas Comhaid PAK a Chruthú?

Tá samplaí liostaithe ag Simutrans chun comhaid PAK a chruthú ar Windows agus Linux OS.

### Windows OS

```
simutrans_src
   makeobj.exe
   haus_01
        haus_01.dat
        haus_01.png
        pak.bat
   auto_03
        auto_03.dat
        auto_03.png
        pak.bat
   triebzug_01
        triebzug_vorn.dat
        triebzug_mitte.dat
        triebzug_hinten.dat
        triebzug_01.png
        pak.bat
```
### Linux BE/OS

```
simutrans_src
   makeobj
   haus_01
        haus_01.dat
        haus_01.png
        pak
   auto_03
        auto_03.dat
        auto_03.png
        pak
   triebzug_01
        triebzug_v.dat
        triebzug_m.dat
        triebzug_h.dat
        triebzug_01.png
        pak
```
