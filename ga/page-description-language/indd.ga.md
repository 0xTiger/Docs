## Cad is comhad indd ann?

Is comhad tionscadail Adobe InDesign é comhad le síneadh .indd ina bhfuil faisnéis maidir le leagan amach leathanaigh an tionscadail. Seasann INDD do InDesign Document. Stórálann na comhaid seo faisnéis faoi fhormáidiú leathanaigh, leabhair, irisí, nuachtáin, fógráin, bróisiúir nó meáin chlóite eile ar nós mockups soghluaiste & leathanaigh ghréasáin. Is formáid comhaid dílseánaigh Adobe é cineál comhaid INDD agus is é an fhormáid réamhshocraithe comhaid le haghaidh InDesign. Úsáideann Dearthóirí Grafaice, iriseoirí, margóirí agus foilsitheoirí a oibríonn le InDesign an cineál comhaid seo.

## Tuilleadh faoi Formáid Chomhaid INDD

Is comhaid veicteora iad comhaid INDD ar féidir iad a scála gan réiteach a chailleadh. Tá siad seo bunaithe ar chasta graif ar féidir iad a leathnú gan teorainn. Pacáistí sé gach cló agus faisnéis faoi leagan amach le roinnt le foireann.

Tá comhaid INDD foirfe chun prionta tarraingteach a dhearadh chomh maith le hábhar ar líne. Is féidir le comhad INDD réise ar leathanaigh iolracha agus féadann sé feidhmiú mar theimpléid chun leathanaigh, caibidlí agus foilseacháin a chruthú. Nuair a bheidh do thionscadal críochnaithe, is féidir é a fhoilsiú do mheáin onilne gan gá le comhshó, zipping nó ríomhphost.

## An féidir comhad INDD a Thiontú go Formáidí Eile?

INDD files can be converted to several other file formats. The most common conversion is INDD to PDF. InDesign lets you save the INDD file to PDF for sharing with other users. It also lets you export INDD file to JPG, [EPS](/page-description-language/eps/), [EPUB](/ebook/epub/), [SWF](/page-description-language/swf/), FLA, [HTML](/web/html/), [XML](/web/xml/), and IDML.

## Conas a Cruthaítear Comhad INDD?

An Indd file is created when you create a new project with [Adobe InDesign](https://www.adobe.com/products/indesign.html) and save it to disc. You can open the Indd project file, make changes to it, and save the updated file for the changes.
