## Cad is comhad SVG ann? ##

An SVG file is a Scalar Vector Graphics file that uses XML based text format for describing the appearance of an image. The word Scalable refers to the fact that the SVG can be scaled to different sizes without losing any quality. Text-based description of such files makes them independent of resolution. It is one of the most used formats for building a website and print graphics in order to achieve scalability. The format can only be used for two-dimensional graphics though. SVG files can be viewed/opened in almost all modern browsers including Chrome, Internet Explorer, Firefox, and Safari.

## Stair Ghearr ##

SVG specifications are available as open standard by World Wide Web Consortium (W3C) since 1999. Roimhe seo, cuireadh sonraíochtaí comhchosúla formáid comhaid i sé fhormáid dhifriúla faoi bhráid W3C go dtí 1998. Cuireadh nuashonrú i bhfeidhm ar na sonraíochtaí in 2011 agus cuireadh leagan 1.1 air. In 2016, foilsíodh SVG 2 mar leagan níos nuaí lena n-áirítear gnéithe sa bhreis orthu siúd in SVG 1.1.

## Sonraíochtaí Formáid Chomhaid ##

Leagann Samhail Oibiachta Doiciméad SVG (DOM) na bunsraitheanna do na sonraíochtaí agus na comhéadain go léir a fhreagraíonn do na hailt ar leith de na sonraíochtaí. Ní mór do lucht féachana SVG na comhéadain SVG DOM a chur i bhfeidhm mar atá sainmhínithe ar fud na sonraíochtaí W3C. Nochtann a DOM roinnt comhéadain le haghaidh cineálacha agus eilimintí éagsúla sonraí.

### cruthanna SVG ###

Tá roinnt eilimintí crutha réamhshainithe ag SVG is féidir le forbróirí a úsáid:

* Dronuilleog<rect>

* Ciorcal<circle>

* Éilips<ellipse>

* Líne<line>

* Polalíne<polyline>

* Polagán<polygon>

* Conair<path>


Bunaithe ar na cruthanna agus na sonraíochtaí seo, tá réimsí feidhme SVG mar a leanas.

**Cosáin** - Úsáidtear cosáin chun imlíne cruthanna simplí chomh maith le cruthanna cumaisc a léiriú. Úsáidtear códuithe chun nádúr na hoibríochta a shainiú. Mar shampla, úsáidtear M le haghaidh Bog Go, úsáidtear L le haghaidh Líne Chu, úsáidtear Z chun cosán a dhúnadh agus mar sin de.

**Cruthanna Bunúsacha** - Is féidir cosáin agus cosáin dhíreacha atá comhdhéanta de shraith de mhíreanna líne dírí ceangailte (polailínte), chomh maith le polagáin dúnta, ciorcail agus éilipsí a tharraingt. Is gnéithe caighdeánacha iad dronuilleoga agus dronuilleoga cruinn-chúinne freisin.

**Téacs** - Cuirtear léiriú téacs in iúl mar shonraí carachtair XML áit ar féidir go leor éifeachtaí amhairc a chur i bhfeidhm ar an téacs. Ceadaíonn na sonraíochtaí téacs déthreo, téacs ingearach agus carachtair a láimhseáil feadh cosán cuartha.

**Péinteáil** - Is féidir cruthanna a líonadh agus/nó a imlíne le dath, grádán nó patrún, rud a fhágann gur féidir é a dhéanamh teimhneach nó trédhearcacht ar bith a bheith aige. Léiríonn Marcóirí gnéithe deireadh líne ar nós cinn saighde nó siombailí atá le feiceáil ar rinn polagán.

**Dath ** - Ceadaíonn sonraíochtaí SVG dathanna a chur i bhfeidhm ar gach eilimint SVG infheicthe, go díreach nó trí líonadh, stróc, agus airíonna eile. Is féidir dathchóduithe difriúla a úsáid chun sonrú cosúil le dubh nó gorm, ionadaíocht heicsidheachúlach, deachúlach nó mar chéatadáin den fhoirm RGB.

** Grádáin agus Patrúin** - Is féidir cruthanna i gcomhad SVG a chomhlánú nó a imlíne le dathanna soladacha, le grádáin nó le patrúin athfhillteacha.

**Éifeachtaí Scagaire** - Is sraith oibríochtaí grafaice é a chuirtear i bhfeidhm ar ghrafaic veicteoir foinse tugtha chun toradh modhnaithe a tháirgeadh.

**Idirghníomhaíocht** - Is féidir le húsáideoirí idirghníomhú le comhaid SVG trí fhócas a athrú, cliceanna luiche, scrollaigh nó súmáil isteach san íomhá. Ligeann an Idirghníomhaíocht íomhánna SVG idirghníomhú le húsáideoirí ar go leor bealaí éagsúla mar a dúradh.

**Nascadh** - Is féidir le híomhánna SVG hipearnasc a bheith acu le doiciméid eile. Baintear é seo amach trí Theanga Nasctha XML nó XLink. Ligeann sé seo staideanna amhairc ar leith a chruthú a úsáidtear chun zúmáil isteach/amach as limistéar ar leith nó chun an radharc a theorannú d’eilimint ar leith.

**Scriptiú** - Cosúil le HTML, tá gach gné de dhoiciméad SVG inrochtana le húsáid scripteanna. Soláthraíonn na hoibiachtaí SVG DOM an treoir chun é seo a bhaint amach trí úsáid a bhaint as eilimint agus tréith SVG. Tá scripteanna faoi iamh in eilimintí na gclibeanna `script` agus is féidir leo rith mar fhreagra ar imeachtaí pointeora, méarchláir nó doiciméad de réir mar is gá.

**Beochan** - Na heilimintí DOM<animate> ,<animateMotion> agus<animateColor> ligeann sé duit beochan le haghaidh ábhar SVG a áireamh. Ar ndóigh, ní féidir é seo a bhaint amach gan scripteanna agus amadóirí ionsuite a úsáid. Is féidir leis na beochana seo a bheith leanúnach agus is féidir iad a chur ar lúb chomh maith le hathchraoltaí agus ag an am céanna freagairt d’imeachtaí úsáideoirí.

**Clónna** - Is féidir le téacs i SVG tagairt a dhéanamh do chomhaid chló seachtracha ar nós clónna córais. In éagmais clónna dá leithéid, ní dhéanfar téacs i SVG a rindreáil don aschur. Is féidir é seo a shárú trí na glyphs riachtanacha a ionchorprú i gcomhad dá leithéid mar chló a rindreáiltear ansin leis an<text> eilimint.

## Samplaí ##
Léiríonn na línte seo a leanas conas a léirítear Ciorcal ag baint úsáide as an script SVG.

```
<html>
<body>

<h1>My first SVG</h1>

<svg width#"100" height#"100">
  <circle cx#"50" cy#"50" r#"40" stroke#"green" stroke-width#"4" fill#"yellow" />
</svg>

</body>
</html>
```

Léiríonn na línte cód seo a leanas conas an<text> bloc chun téacs a thabhairt do SVG.

```
<svg height#"30" width#"200">
  <text x#"0" y#"15" fill#"red">I love SVG!</text>
</svg>
```
