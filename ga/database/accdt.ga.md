## Cad is comhad ACCDT ann?

A file with .accdt extension is a Microsoft Access database template file that contains pre-defined database elements. These elements are a collection of structures that define a database applications such as database schemas for storing data, layout description for views of the data, and metadata describing the database. Being template files, ACCDT files can be used to create databases based on the template settings available in these. Resultant database files are saved as [ACCDB](/database/accdb/) files and populated with data in tables. Microsoft Access 2007 and onwards can open ACCDT files.

## Formáid Chomhaid ACCDT

Tá comhaid teimpléid ACCDT bunaithe ar shonraíochtaí Oifig Oscailte XML agus tá na sonraí go léir i bpacáiste ZIP. Tá faisnéis faoi struchtúr agus faoi ábhar an bhunachair shonraí sna comhaid [XML](/web/xml/) agus sna comhaid téacs agus nasctar iad lena chéile trí chaidrimh. Is féidir leat comhad ACCDT a athainmniú mar shíneadh [.zip](/compression/zip/) agus aon bhogearra comhbhrú a úsáid chun inneachar na cartlainne ZIP a bhaint as.

### Struchtúr comhaid

Is pacáistí iad comhaid ACCDT ina bhfuil bailiúchán de chodanna gaolmhara. Stórálann gach **cuid** faisnéis faoi ábhar feidhmchláir bhunachair sonraí ag baint úsáide as XML, formáidí téacs agus dénártha, lena n-áirítear:

 * Rudaí bunachar sonraí
 * meiteashonraí gaolmhara
 * Struchtúr an phacáiste

#### Pacáiste

Is cartlann [ZIP](/compression/zip/) é Pacáiste ina bhfuil codanna iolracha agus a chloíonn leis na Coinbhinsiúin um Phacáistiú Oscailte atá sonraithe in [ISO/IEC-29500-2](https://www.iso.org/standard/51459.html). Ní mór go mbeadh cuid meiteashonraí Teimpléad amháin ar a laghad i gcomhaid ACCDT ar cheart go mbeadh an caidreamh mar sprioc aici. Is é an Meiteashonraí Teimpléad seo an chuid tosaigh de chomhad ACCDT.

#### Cuid

Is éard is cuid ann ná sruth beart a bhfuil cineál gaolmhar aige chun nádúr agus cineál an ábhair atá stóráilte ann a shonrú. Sonraítear in áireamh na gcodanna páirteanna bailí, cineálacha bailí ábhair, agus na gaolmhaireachtaí riachtanacha idir na codanna go léir i bpacáiste.

#### Gaol

`Caidreamh Pacáiste` - áit a bhfuil an sprioc ina chuid agus gurb é an fhoinse an pacáiste ina iomláine.

`Caidreamh Pháirt-go-Páirt` - áit a bhfuil an sprioc mar chuid agus an fhoinse mar chuid sa phacáiste.

`Caidreamh Sonrach` - nuair a dhéantar tagairt d'acmhainn ó inneachar na coda foinseach trí thagairt a dhéanamh d'eilimint choibhneasta de réir luach a tréithe aitheantais.

`Caidreamh intuigthe` - gaol nach bhfuil follasach.

`Caidreamh inmheánach` - áit a bhfuil an sprioc mar chuid sa phacáiste.

`Caidreamh seachtrach` - nuair is acmhainn sheachtrach nach bhfuil sa phacáiste an sprioc.
