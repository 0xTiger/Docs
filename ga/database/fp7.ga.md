## Cad is comhad FP7 ann?

A file with .fp7 extension is a database file created with FileMaker Pro versions 7-11. Stórálann sé sonraí i dtáblaí le taifid ina bhfuil réimse amháin nó níos mó i ngach taifead. Tá an cumas ag comhaid FP7 sonraí móra a stóráil lena n-áirítear sonraí bunachair shonraí pearsanta agus taifid eagraíochtúla. Is é {{ HYPERLINK1}} leagan níos nuaí de FileMaker Pro a tugadh isteach nuair a seoladh FileMake Pro leagan 12. Chomh maith le sonraí téacsúla, féadfaidh an comhad FP7 foirmeacha, cairteacha agus oibiachtaí bunachar sonraí eile a choinneáil freisin. Is féidir comhaid FP7 a oscailt le FileMaker Pro Advanced.

## Formáid Chomhaid FP7

Déantar comhaid FP7 a shábháil ar diosca mar chomhaid dhénártha i bhformáid comhaid dílseánaigh FileMaker. Níl na sonraí maidir le formáid comhaid inmheánach formáid comhaid FP7 ar fáil go poiblí.
