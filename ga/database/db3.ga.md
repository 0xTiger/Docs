## Cad is Comhad DB3 ann?
Is comhad bunachar sonraí é an comhad DB3 a chruthaigh bogearraí SQLite ar clár éadrom bunachar sonraí féinchuimsitheach é a chruthaíonn bunachair shonraí ag baint úsáide as gnáthchomhaid; ina bhfuil anatamaíocht an bhunachair shonraí chomh maith le taifid sonraí; a úsáidtear go coitianta chun sonraí struchtúrtha a aisghabháil nó a stóráil trí úsáid a bhaint as SQL. Is féidir na comhaid seo a úsáid i bhfeistí cliste nó nuair is gá an coimeád taifead nó bainistíocht sonraí eile ach ar thimpeallacht spáis íseal.


## Formáid Chomhaid DB3
Tá formáid comhaid DB3 bainteach le RDBMS (Córas Bainistíochta Bunachar Sonraí Coibhneasta) SQLite, rogha coitianta do bhunachar sonraí leabaithe. Níl aon iarmhír chomhaid sainiúil sainithe i SQLite_3 ina shonraíocht, féadfaidh sé úsáid a bhaint as eisínteachtaí lena n-áirítear [.dbf](/database/dbf/) agus [.sqlite](/database/sqlite/).

### An Bunachar Sonraí a Shocrú
Commonly SQLite, version 3 related db file format can be considered as the DB3 file format used as the publicly documented native format for the SQLite database engine since June 2004. Is ardán tras-ardán é formáid comhaid DB3, atá inaistrithe idir ailtireachtaí móra-endian agus ceann beag nó córais 32-giotán agus 64-giotán. Mar gheall ar na gnéithe seo tá DB3 mar rogha coitianta mar fhormáid comhaid iarratais. Tá leathanach amháin nó níos mó i gcomhad bhunachar sonraí SQLite_3 (DB3). Tá gach méid de gach leathanach comhionann sa bhunachar sonraí céanna. Tá dhá mhéid idir 512 agus 65536 san áireamh i méid an leathanaigh i mbearta.
