## Cad is comhad SDF ann?
I gcomhad le síneadh .sdf tá bunachar sonraí Microsoft SQL Server Compact (SQL CE) ar a dtugtar bunachar sonraí coibhneasta dlúth freisin; arna dtáirgeadh ag Microsoft do na feidhmchláir a dhéantar le haghaidh gléasanna soghluaiste agus deasc. Tacaíonn sé le córas oibriúcháin 32 agus 64 giotán araon agus tá ábhar iomlán an bhunachair shonraí san áireamh i gcomhad SDF amháin agus is féidir leis níos mó ná 4GB spás diosca a áitiú. Chun críche na slándála, is féidir comhad .sdf a chriptiú le criptiú 128-giotán. Socraíonn am rite SQL CE rochtain chomhthreomhar il-úsáideoir ar an gcomhad .sdf. Is féidir an comhad SDF a chóipeáil trí **QuickOnce** nó go simplí is féidir é a chóipeáil chuig an gceann scríbe le haghaidh úsáid an chórais.

## Formáid Comhaid SDF
Tá bunachar sonraí i gcomhad SDF ar a dtugtar de ghnáth bunachar sonraí coibhneasta. Tá an t-eolas go léir a bhaineann leis an mbunachar sonraí i gcomhad SDF agus is inneall bunachar sonraí saor in aisce é an SQL Server Compact a úsáidtear chun na comhaid .sdf a bhainistiú. Níor cheart go sáródh méid comhaid .sdf an teorainn méide 4 GB. Ní stórálann comhaid SDF an fhaisnéis faoi nósanna imeachta, truicear nó radharcanna atá stóráilte. Ní gá d'fheidhmchláir a úsáideann bunachar sonraí SQL CE an cosán chuig an gcomhad SDF a shonrú sa teaghrán ceangail ADO.NET, ina ionad sin is féidir é a lua mar |DataDirectory|\database_name.sdf, ag sainiú an eolaire sonraí atá á shainiú sa léiriú tionóil don fheidhmchlár
Tá an coinbhinsiún ainmnithe .sdf roghnach, agus is féidir aon síneadh a úsáid chun an comhad a shábháil. Is céim roghnach é pasfhocal a shocrú don chomhad bunachar sonraí. Chun an bunachar sonraí a chomhbhrú nó a dheisiú ba chóir an comhad a shábháil leis an rogha **bunachar sonraí casta/deisithe**.
