## Cad is comhad FIC ann?

Is leagan tiomsaithe de Bhunachar Sonraí HyperFileSQL é an comhad FIC in WinDev. Seasann an t-acrainm FIC do Fichier de Code i bhFraincis, a aistríonn go Code File i mBéarla. In WINDEV, is comhad tiomsaithe é comhad FIC ina bhfuil cód inrite tionscadail. Tá an síneadh .fic ag an gcomhad agus cruthaítear é nuair a thiomsaíonn tú tionscadal WINDEV. Nuair a chuirtear tionscadal le chéile i gcomhad FIC, is féidir é a dháileadh ar úsáideoirí mar chlár inrite nó mar phacáiste suiteála.

Baineann comhad FIC le WINDEV, uirlis forbartha bogearraí cruthaithe ag PCSoft, cuideachta Fhrancach. Ligeann sé d'fhorbróirí feidhmchláir Windows, gréasáin agus móibíleacha a chruthú ag baint úsáide as bonn cód amháin. Áiríonn sé timpeallacht forbartha amhairc, teanga ríomhchláraithe (WLlangage), agus éagsúlacht rialuithe agus feidhmeanna réamhthógtha le haghaidh tascanna coitianta. Úsáidtear é go príomha le haghaidh feidhmeanna gnó agus tá bonn láidir úsáideoirí aige san Eoraip.

## Gaol le bunachar sonraí Soghluaiste HyperFileSQL

In WinDev, i gcomhad FIC a chruthaítear do bhunachar sonraí HyperFileSQL Mobile tá an cód inrite don bhunachar sonraí, chomh maith leis na sonraí atá sa bhunachar sonraí. Is córas bainistíochta bunachar sonraí gaolmhar é HyperFileSQL Mobile atá cuimsithe le WINDEV agus atá deartha go sonrach le haghaidh feidhmchláir mhóibíleacha agus as líne. Is féidir an comhad FIC a dháileadh mar chomhad inrite neamhspleách nó mar chuid de phacáiste suiteála, agus is féidir é a rith ar an ngléas sprice chun rochtain a fháil ar na sonraí sa bhunachar sonraí HyperFileSQL Mobile agus iad a bhainistiú. Nuair a reáchtálfar an comhad FIC, osclóidh sé an bunachar sonraí agus cuirfidh sé na sonraí ar fáil le húsáid san fheidhmchlár soghluaiste.
