## Cad is comhad ADE ann?

Comhad le síneadh .ade is ea comhad Tionscadal Microsoft Access ina bhfuil aschur tiomsaithe na faisnéise atá i gcomhad tionscadail ADP Microsoft Access. Tá faisnéis sa chomhad tionscadail Rochtana, seachas Visual Basic for Applications (VBA), ar fáil i bhfoirm tiomsaithe i gcomhaid ADE. Stóráiltear sonraí i bhformáid chomhbhrúite sna comhaid seo chun méid an chomhaid a laghdú agus chun feidhmíocht in Rochtain a fheabhsú. Ós rud é gurb é ADE an t-aschur tiomsaithe den chomhad tionscadail Rochtana, tá aon chód foinse VBA atá mar chuid den tionscadal fós faoi chosaint gan ligean dó a bheith mar chuid den aschur. Is féidir comhaid ADE a oscailt i Microsoft Access 365.

## Formáid Chomhaid ADE - Tuilleadh Eolais

Is comhaid Bunachar Sonraí Rochtana iad ADE a stóráiltear go diosca mar chomhaid dhénártha. Ní fios struchtúr inmheánach comhad na gcomhad seo.

The ADE files can create issues in opening based on the version of Microsoft Access that has been used to create these files. Access databases that are using the 64-bit version of Microsoft Access 2010 and that are compiled as MDE, [ACCDE](/database/accde/), or ADE files have to be recompiled in Microsoft Access 2021 Service Pack 1 (SP1) to work correctly with Access 2010 SP1. Tarlaíonn an cheist seo toisc go n-úsáideann Access 2010 SP1 leagan níos nuaí den chomhad VBE7.dll (leagan 7.00.1619).
