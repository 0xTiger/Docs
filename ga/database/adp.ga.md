## Cad is comhad ADP ann?

A file with .adp extension is a Microsoft Access project file that uses OLE DB component architecture to provide a direct and efficient connection to a Microsoft SQL Server database. Such files do not contain the actual tables, database diagrams or other database elements. ADP files can be created with Access 2007 and 2010. Is féidir comhaid a cruthaíodh leis na leaganacha níos luaithe de Access a oscailt freisin Access 2007 agus 2010. Is féidir comhaid ADP a oscailt le Microsoft Access 365 a úsáid.

## Formáid Chomhaid ADP - Tuilleadh Eolais

Ligeann tionscadal Rochtana a cruthaíodh mar ADP duit athruithe dearaidh a dhéanamh ar rudaí freastalaí SQL ar nós táblaí agus radhairc. Ligeann sé freisin gnéithe SQL a chruthú, a chur in eagar agus a úsáid mar léaráidí bunachar sonraí, nósanna imeachta stóráilte, agus feidhm atá sainithe ag an úsáideoir. Tugann sé seo buntáiste as nascadh le bunachar sonraí freastalaí SQL áit nach féidir leat athruithe a dhéanamh ar an dearadh agus nach féidir leat ach nasc a dhéanamh le táblaí agus radharcanna freastalaí SQL.

### Cineálacha Sonraí agus Tacaíocht Léaráidí

**Cineálacha Sonraí Dáta/Am**

Tacaítear leis na cineálacha sonraí dáta/ama nua seo a leanas in Access 2010.

 * AMACH
 * DÁTA
 * DÁTATIME2
 * DÁTATIMEOFFSET

**Cineálacha Fad Athraitheacha Sonraí**

Is féidir na cineálacha sonraí fad athraitheach seo a leanas a úsáid i dtionscadail Access 2010:

 * VARBIN(MAX)
 * VARCHAR(MAX)
 * NVARCHAR(MAX)
