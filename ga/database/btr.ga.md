## Cad is comhad BTR ann?

Is comhad bunachar sonraí é comhad a bhfuil iarmhír .btr air a chruthaigh feidhmchlár bunachair sonraí idirbheartaíochta [Btrieve](https://www.actian.com/). Murab ionann agus córais bhainistíochta bunachar sonraí coibhneasta (RBMS), tá Btrieve bunaithe ar Mhodh Innéacsaithe Rochtana Seicheamhaí (ISAM), ar bealach é chun sonraí a stóráil le haghaidh aisghabháil tapa. Stórálann an comhad BTR bailiúchán taifead agus úsáidtear é chun tuarascálacha a ghiniúint de réir riachtanas. Is féidir comhaid BTR a oscailt trí úsáid a bhaint as Bainisteoir Bunachar Sonraí Pervasive Btrieve, Utility Cothabhála PSQL Forleatach, agus Amharcóir Finscéalta BTRIEVE.

## Struchtúr Comhad BTR - Tuilleadh Eolais

Níl struchtúr inmheánach sonraí agus ailíniú gach beart i struchtúr comhaid BTR ar fáil go poiblí. Mar sin féin, Btrieve
provides the [Btrieve API](https://docs.actian.com/psql/PSQLv13/index.html#page/btrieveapi%2Fbtrintro.htm) that can be used to read the information from BTR files.  It is compatible with the following programming languages and development environments.

 * Embarcadero C/C++
 * Embarcadero Delphi
 * GNU C/C++
 * Micrimhilseogra COBOL
 * Microsoft visual basic
 * Microsoft Visual C++
 * Watcom C/C++

Braitheann sonraí a aisghabháil ó chomhad BTR ar an gcomhad DDF a bhaineann leis. Gan an DDF, ní bheidh sé éasca an fhaisnéis a aisghabháil ó chomhad dá leithéid.
