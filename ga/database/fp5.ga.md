## Cad is comhad FP5 ann?

An FP5 file is a database file created with FileMaker Pro version 5 or 6. Stórálann sé sonraí úsáideoirí chomh maith le sonraí eagraíochtúla i bhfoirm taifead taobh istigh de tháblaí. Is córas bainistíochta bunachar sonraí é FileMaker Pro a thacaíonn le córais oibriúcháin Macintosh agus Windows. Is formáid comhaid níos sine é FP5 a fuair tacaíocht ón leagan FileMaker Pro a bhí ar fáil ag an am. Is í an fhormáid bhunachar sonraí FileMaker Pro is déanaí ná [FMP12](/database/fmp12/) a tugadh isteach le seoladh FileMaker Pro v 12. Ba é [FP7](/database/fp7/) formáid comhaid bunachar sonraí eile a úsáideadh go maith agus cuireadh FMP12 ina ionad.

## Formáid Chomhaid FP5 - Tuilleadh Eolais

Níl na sonraí maidir le formáid comhaid inmheánach formáid comhaid FP5 ar fáil go poiblí le haghaidh tagartha an fhorbróra. Formáidí comhaid bhunachar sonraí níos sine FileMaker Pro ar nós [FP5 and FP7 can be upgraded to latest FMP12](https://fmhelp.filemaker.com/help/16/fmp/en/index.html#page/FMP_Help/converting-files.html) ag baint úsáide as na treoirlínte mionsonraithe ó FileMaker Pro.
