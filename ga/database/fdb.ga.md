## Cad is comhad FDB ann?

Is iarmhír comhaid é comhad .fdb a úsáidtear do na comhaid bhunachar sonraí i Microsoft Dynamics NAV. Seasann an síneadh comhad .fdb do Comhad Bunachar Sonraí Firebird” agus is é an fhormáid comhaid a úsáideann Córas Bainistíochta Bunachar Sonraí Firebird, arb é an t-inneall bunachar sonraí bunúsach a úsáideann Microsoft Dynamics NAV. Tá na sonraí, táblaí agus struchtúir uile an chórais NAV sa chomhad .fdb, lena n-áirítear sonraí airgeadais, sonraí fardail, sonraí custaiméirí, agus níos mó. Tá sé tábhachtach a choinneáil cúltaca den chomhad .fdb chun sláine sonraí a chinntiú agus a bheith in ann sonraí a aisghabháil i gcás aon saincheist. Is féidir an comhad .fdb a onnmhairiú, a allmhairiú agus é a úsáid freisin i macasamhlú.

## Gaol le Microsoft Dynamics NAV

Is comhaid bhunachar sonraí iad comhaid FDB i Microsoft Dynamics NAV. Is bogearraí pleanála acmhainní fiontair (ERP) é Microsoft Dynamics NAV atá forbartha ag Microsoft. Tá sé deartha do ghnólachtaí beaga agus meánmhéide agus soláthraíonn sé raon leathan feidhmiúlacht bainistíochta gnó, lena n-áirítear bainistíocht airgeadais, bainistíocht slabhra soláthair, déantúsaíocht, bainistíocht tionscadail, agus níos mó. Tá sé tógtha ar ardán Microsoft Dynamics agus tá sé comhtháite go hiomlán le táirgí Microsoft eile, mar shampla Office agus Outlook. Dinimic Is féidir NAV a rochtain trí chliant gréasáin, chomh maith le trí chliant Windows, agus is féidir é a shaincheapadh agus a chomhtháthú freisin le córais eile ag baint úsáide as a thimpeallacht forbartha féin ar a dtugtar C/AL. Tá sé in úsáid go coitianta ag cuideachtaí sna tionscail déantúsaíochta, dáileadh mórdhíola, agus seirbhíse.

## Conas comhad .fdb a oscailt?

De ghnáth, osclaítear agus bainistítear comhad BDB ag baint úsáide as timpeallacht forbartha Microsoft Dynamics NAV nó cliant Microsoft Dynamics NAV.

Seo na céimeanna chun comhad .fdb a oscailt i Microsoft Dynamics NAV:

1. Oscail timpeallacht forbartha Microsoft Dynamics NAV.
2. Cliceáil ar an roghchlár Comhad agus roghnaigh Open nó brúigh Ctrl+O
3. Déan nascleanúint chuig an suíomh ina bhfuil an comhad .fdb sábháilte.
4. Roghnaigh an comhad .fdb agus cliceáil ar an gcnaipe Oscailte.

Is féidir leat an comhad .fdb a oscailt freisin trí nascadh leis an mbunachar sonraí ag baint úsáide as an gcliant NAV.

1. Oscail an cliant NAV.
2. Cliceáil ar an roghchlár Comhad agus roghnaigh Ceangail
3. Sa réimse Freastalaí, cuir isteach ainm nó seoladh IP an fhreastalaí ina bhfuil an comhad .fdb suite.
4. Cuir isteach an Ainm Bunachar Sonraí agus Dintiúir cuí
5. Cliceáil ar an gcnaipe Ceangail.

Is féidir freisin comhaid .fdb a oscailt le bogearraí bainistíochta córas bainistíochta bunachar sonraí Firebird, toisc gurb é an t-inneall bunachar sonraí bunúsach a úsáideann Microsoft Dynamics NAV.
