## Cad is comhad ACCDE ann?

Is formáid comhaid Microsoft Access 2007 é comhad le síneadh .accde chun comhaid bhunachar sonraí Execute Only a chruthú chun cód saincheaptha bunachar sonraí a chosaint. Tiomsaíonn sé gach cód VBA ina inrite amháin ionas nach féidir é a fheiceáil ná a chur in eagar. Tá formáid comhaid ACCDE in ionad formáid comhaid MDE a d’úsáid leaganacha níos luaithe de Microsoft Access. Is comhaid [ACCDB](/database/accdb/) iad ACCDE sa mhód inrite amháin agus níos dlúithe toisc nach bhfuil aon chód foinse ann. Feabhsaíonn méid dlúth na gcomhad seo an úsáid chuimhne chomh maith le feabhsaítear feidhmíocht an fheidhmchláir. Is féidir comhaid ACCDE a oscailt le Microsoft Access 365.

## Formáid Chomhaid ACCDE

Úsáideann gach comhad Microsoft Access 2007+ an Scaird ACE (inneall bunachar sonraí Access) a úsáideann an teaghlach formáid comhaid ACCDB. Mar sin féin, ní sholáthraíonn Microsoft cur chun feidhme foinse oscailte ná sonraíocht theicniúil mhionsonraithe.
