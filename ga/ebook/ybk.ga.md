## Cad is comhad YBK ann? ##

Úsáideann formáid comhaid ríomhleabhair YanCEyWare agus Microsoft Encarta araon síntí comhad YBK. Bhunaigh YanCEyWare ríomhleabhar YanCEyWare. Tá na feidhmchláir bogearraí ar fad ar fáil le haghaidh gléasanna a ritheann Windows, tacaíocht comhaid YBK. Is féidir an Encarta Encyclopedia a nuashonrú le comhaid .ybk gan an clár iomlán a nuashonrú. Is clár cáiliúil é YanCEyDesktop atá in ann comhaid YBK a bhainistiú agus a láimhseáil go héasca. Úsáideann an YanCEyWare Reader an fhormáid comhaid YBK seo go forleathan, mar fheidhmchlár le haghaidh léamh ríomhleabhair don Windows Mobile nó Pocket PC. Forbraíodh an comhad seo le hacmhainní móra hipear-nasctha.

## Fadhbanna le comhad YBK a oscailt ##

Uaireanta nuair a bhíonn sé deacair an comhad YBK a oscailt agus a fheidhmiú, seo é an liosta de roinnt fadhbanna féideartha:

* Comhad truaillithe YBK a oscailt
* Naisc mhíchuí agus míchuí chuig an gcomhad YBK i rochtain chlárlainne
*	Unintentional removal of the description of the YBK from the Windows archive
* Suiteáil páirteach feidhmchlár a sholáthraíonn formáid YBK
*	Infected YBK file with an undesirable malware
