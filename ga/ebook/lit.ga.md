## Cad is comhad ITL ann?

Baineann na comhaid ITL leis an **Microsoft Reader** ársa a úsáideadh chun ríomhleabhair a léamh. Tá an síneadh **.lit** ar na ríomhleabhair a thaispeánann Microsoft Reader a ghiorrú go **Literature**. Tá formáid comhaid LIT comhdhéanta de chomhad Microsoft ITOL/ITLS HTML Help 2.0, a thugann gach cineál gnéithe iontacha cosúil le comhbhrú [LZX](/compression/lzx/), léiriú dénártha ábhar marcála, agus sonraí cúnta roghnacha treallach a bhaineann le gach sruth. Is ionann téacs leabhair agus líon treallach de shruthanna marcála OEBPS 1.0 agus fo-thacar de OEBPS 1.0 CSS. Is iad na híomhánna ná [GIF](/image/gif/), [JPEG](/image/jpeg/), agus [PNG](/image/png/).

## Cé mhéad ama a thaispeáin Microsoft Reader na comhaid LIT

The **Microsoft Reader** is an application for reading e-books that have been discontinued by Microsoft. The first version of Microsoft Reader was released in August 2000, which used its own LIT format. It was available for PDAs and Windows computers. The name of the app was also used later for an application (which was initially released with Windows 8) for reading [PDF](/pdf/) and [XPS](/page-description-language/xps/) files. The Microsoft Reader application was discontinued in 2018.
