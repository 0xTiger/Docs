## Cad is comhad WEBZ ann? ##

Is formáid comhaid an-choitianta é síneadh comhaid WEBZ agus tugtar Comhad Ríomhleabhair Chomhbhrúite WEBZ air go hiomlán. Úsáidtear iarmhír EBZ go ginearálta do chomhaid Formáid Ríomhleabhair Chomhbhrúite WEBZ agus úsáidtear é go bunúsach mar chomhad ZIP de leathanaigh Ghréasáin agus d’íomhánna. Is é Web Books Publishing an t-údar agus chríochnaigh sé cáilíocht fhormáid WEBZ. Tá an fhormáid comhaid seo inchurtha le bogearraí is féidir a nascadh le hardán córais Gréasáin. Téann comhad WEBZ go dtí an cineál comhaid téacs díreach mar a théann síntí ainm comhaid eile. Is é Web Books Viewer an clár is mó a úsáidtear chun oibriú le formáidí comhaid dá leithéid. Tá an síneadh comhad seo solúbtha agus is féidir é a athrú ó .webz go .zip.
