## Cad is comhad RDLC ann? ##

Seasann an RDLC do thaobh an Chliaint Teanga Tuairiscigh. I ndáiríre Is síneadh comhad tuairisce é a cruthaíodh trí úsáid a bhaint as teicneolaíocht tuairiscithe Microsoft. Úsáidtear leagan SQL Server 2005 de Report Designer chun na comhaid seo a chruthú. Is féidir leis an rialú **ReportViewer** ar thaobh an chliaint na tuarascálacha RDLC a fhorghníomhú go díreach.

## Comhaid RDL vs RDLC
|.rdl Comhaid |.comhaid rdlc|
---|---|
|Cruthaítear comhaid RDL ag leagan SQL Server 2005 de Report Designer|Cruthaítear comhaid RDLC ag leagan Visual Studio 2005 de Report Designer|
|Úsáidtear é i Seirbhísí Tuairiscithe Freastalaí SQL|Úsáidtear é in Visual Studio|
|Is cianthuairisc é|Is tuairisc áitiúil é|
|Tá cás Seirbhísí Tuairiscithe ag teastáil chun é a rith|Níl gá le cás Seirbhísí Tuairiscithe|

## Comhaid um Shainmhíniú Tuairisce Cliant (.rdlc) a Chruthú
Úsáidtear an rialú ReportViewer chun comhaid sainmhínithe tuairisce cliaint (.rdlc) a rith ag baint úsáide as cumas próiseála ionsuite an rialaithe. Tuairiscíonn an cliant go ritheann tú i mód próiseála áitiúil is féidir a chruthú go héasca i do thionscadal iarratais. Seo a leanas na cuir chuige chun an tuarascáil a chruthú:

- Úsáid an **Treoraí Tuairisce** chun sainmhíniú nua ar thuairisc an chliaint (.rdlc) a chruthú.

- Cruthaigh comhad sainmhínithe tuairisce cliaint nua (.rdlc) trí úsáid a bhaint as Visual Studio.

- Gin sainmhíniú ar thuarascáil de réir cláir.


Ní féidir leat réamhamharc a dhéanamh ar thuairisc ach é a rith i rialú **ReportViewer**. Tabhair faoi deara le do thoil gur féidir leat sainmhíniú na tuarascála a oscailt agus a chur in eagar am ar bith, agus ansin an feidhmchlár a thógáil nó a imscaradh chun na torthaí a sheiceáil.
