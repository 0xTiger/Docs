## VDISCO faylı nədir?

VDISCO faylı Microsoft Visual Studio tərəfindən öz Veb Xidmətlərində istifadə edilən kəşf faylıdır. O, onların URL-ləri, ad boşluqları və metodları kimi mövcud Veb Xidmətləri haqqında məlumat verir. VDISCO faylı [DISCO](/web/disco/) fayl formatına bənzəyir, lakin icra zamanı yaradılır. O, Veb xidməti proqramında saxlanılır və Visual Studio tərəfindən layihədə Veb Xidmətlərini dinamik şəkildə kəşf etmək və istifadə etmək üçün istifadə olunur. VDISCO faylları faktiki Veb Xidmət kodunu ehtiva edən [.asmx](/web/asmx/) faylı ilə birlikdə istifadə olunur.

VDISCO faylını Microsoft Notepad, Github Atom və Apple TextEdit kimi istənilən mətn redaktorunda aça bilərsiniz. O, həmçinin Microsoft Visual Studio 2012 və daha sonra onunla işləmək üçün açıla bilər.

## VDISCO fayl formatı

VDISCO faylı [XML](/web/xml/) fayl formatında saxlanılır və yerləşdirilir ki, bu da məlumatların tərkibi və dərci üçün universal formatdır. O, xidmət URL-sini, ad boşluqlarını və standart XML teqlərindəki metodları ehtiva edir, burada hər bir teq öz məlumatının müvafiq dəyərini ehtiva edir.
