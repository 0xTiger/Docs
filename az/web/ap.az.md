## AP faylı nədir?

AP faylı veb serverdə işləyən və dinamik veb səhifələr yaratmaq üçün istifadə olunan Aktiv Veb Səhifədir. Bu veb-səhifələr daha sonra sorğu edən brauzerə təqdim edilir və istifadəçiyə göstərilir. AP faylı serverdə icra edilən HTML və JavaScript kodundan ibarətdir. Daha ümumi olduğundan, AP faylları adi server tərəfi [.php](/programming/php/) və ya [.asp](/web/asp/) genişləndirmələri əvəzinə **.ap uzantısından** istifadə edir.

## AP Fayl Format - Ətraflı Məlumat

AP faylları dəstəkləyici proqram təminatının köməyi ilə Veb serverdə yerləşdirilir. Daha sonra o, serverdə icra olunur və cavab nümayiş etdirmək və ya sonrakı emal üçün veb brauzerə geri göndərilir.
