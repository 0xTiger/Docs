## ECE faylı nədir?

ECE faylı məzmun idarəetmə platforması, **Escenic Content Engine (ECE)** tərəfindən dinamik şəkildə yaradılmış veb səhifədir. O, veb-server tərəfindən emal edilən və istifadəçi brauzerinə [HTML](/web/html/) səhifəsi olaraq geri qaytarılan server tərəfi kodu ehtiva edir. ECE fayl formatı ASP və PHP kimi digər server tərəfi fayl formatları ilə müqayisədə çox populyar deyil. Bu fayllar daxili olaraq təqdim olunur və ekran məqsədi ilə son istifadəçilərə xidmət göstərir.

## ECE Fayl Formatı

ECE faylları diskdə saxlanmır, lakin sorğu əsasında birbaşa istifadəçiyə verilir. Bunlar HTML-yə çevrilir və nümayiş üçün istifadəçinin brauzerinə geri göndərilir. Escenic Content Engine xəbər məqalələrinin səhifələrini dərc etmək üçün fəal şəkildə istifadə olunur. JEE və SQL texnologiyasına əsaslanan ECE serveri bütün onlayn məzmun üçün etibarlı və miqyaslana bilən repozitoriya təmin edir.

Müştəri brauzerləri HTTP/REST API vasitəsilə ECE serverlərindəki məzmuna daxil ola bilər. [HTML](/web/html/) səhifələrə xidmət etməklə yanaşı, ECE video/audio məzmunun idarə edilməsi və dərc edilməsi üçün yüksək səmərəli iş axını təmin edir.
