## ADR faylı nədir?

ADR faylı (.adr uzantısı ilə saxlanılır) Opera Veb brauzeri tərəfindən yaradılmış əlfəcinlərin siyahısıdır. İstifadəçi veb URL-ləri qeyd etməyə davam etdikcə bu siyahı doldurulur. Bu əlfəcinlər ADR faylında saxlanılır və lazım olduqda ehtiyat nüsxəsi çıxarıla və bərpa oluna bilər. Chrome və Firefox kimi əksər brauzerlər əlfəcinləri [HTML](/web/html/) formatında saxladığından, bunlar Opera brauzerində idxal etmək üçün ADR fayl formatına çevrilə bilər. ADR faylları Opera brauzerində açıla bilər.

## ADR Fayl Format - Ətraflı Məlumat

ADR fayllarının daxili fayl formatı təfərrüatları məlum deyil. Bununla belə, Operanın müxtəlif versiyaları ilə ixrac edilən ADR fayl formatları arasında fərqlər var. Bu, Operanın müxtəlif versiyaları ilə [merge ADR files](https://superuser.com/questions/471959/how-do-i-merge-several-opera-adr-bookmark-files-to-one-single-file) yaratmağı çətinləşdirir. Bunları birləşdirmək üçün bəzi xarici vasitələrdən istifadə edilə bilər.
