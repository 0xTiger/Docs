## CSP faylı nədir?

CSP faylı RadGs Konsept Tətbiq Serverində icra edilmək üçün yazılmış məzmunu ehtiva edən dinamik veb səhifə faylıdır. O, server tərəfində tərtib edilir və icra edilir, bundan sonra HTTP protokolu vasitəsilə istifadəçinin brauzerinə müraciət edilməsinə xidmət edir. CON fayl formatından fərqli olaraq, CSP faylları bu fayllara baxmaq üçün Konsept müştərinin istifadəçiyə quraşdırılmasını tələb etmir. Siz [PHP](/programming/php/) və [ASP](/web/asp/) kimi digər server yan səhifələrinə bənzər CSP fayllarına baxa bilərsiniz.
