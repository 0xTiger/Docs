## FWP faylı nədir?

.fwp faylı olan fayl Microsoft Expression Web ilə yaradılmış arxiv faylıdır və dizayn üçün bütün faylları və qovluqları ehtiva edir. Microsoft Expression Web Microsoft tərəfindən ümumi veb dizayn proqram məhsuludur və [HTML](/web/html/) redaktoruna əsaslanır. O, 2012-ci ildən dayandırılıb və Microsoft tərəfindən pulsuz yükləmək mümkündür. FWP faylları Microsoft-un .cab formatı kimi ixrac edilə və saxlanıla bilər.

## FWP Fayl Format - Ətraflı məlumat

FWP fayllarının dəqiq fayl formatı spesifikasiyası tərtibatçının arayışı üçün mövcud deyil. Bununla belə, bunlar ikili fayllardır və uzun müddət əvvəl dayandırılmış Microsoft Expression Studio-da açıla bilər.
