## XHTM faylı nədir?

An file with .xhtm extension is a webpage file that is written in XHTML. It uses XML 1.0 as its standard implementation and is considered a reformulation of HTML 4. XHTML kimi, bir XHTM faylı daha sərt XML qaydalarına əməl edən standart HTML etiketlərini ehtiva edir. XHTM faylları vebsayt məzmununu dərc etmək üçün istifadə olunur və adətən [XHTML](/web/xhtml/) uzantısından istifadə edir.

XHTM fayllarını açmaq üçün Google Chrome, Apple Safari və Mozilla Firefox kimi veb brauzerlərdən istifadə edə bilərsiniz.

## XHTM nümunəsi

XHTM sənədinin aşağıdakı nümunəsi XML bəyannamələrini müəyyən edir.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
