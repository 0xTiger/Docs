## JHTML faylı nədir?

.jhtml uzantılı fayl, müştəri bu səhifəni brauzerdə tələb etdikdə serverdə icra edilən [Java](/programming/java/) kodu olan [HTML](/web/html/) fayldır. Server sorğuları emal edir, funksiyada olan Java funksiyalarını yerinə yetirir və səhifəni müştərinin brauzerinə qaytarır. JHTML səhifələrinə daxil edilmiş Java obyektləri bu tip səhifələr üçün sorğuları idarə etmək üçün serverdə işləyir. JHTML faylları həmçinin JDBC (Java [Database](/database/) Bağlantı) bağlantısından istifadə edərək verilənlər bazasından məlumat əldə edə bilər. JHTML faylları istənilən mətn redaktorunda açıla və Google Chrome, Firefox və Safari kimi veb brauzerlərdə baxıla bilər.

## JHTML Fayl Format

JHTML ATG-nin xüsusi texnologiyasıdır və ATG (Art Technology Group) Dynamo Document Redaktorundan istifadə etməklə yaradıla bilər. JHTML faylları standart HTML və Java kodundan istifadə edərək düz mətn faylı formatında yazılır. Bunlar Java obyektlərinə istinad edən mülkiyyət teqlərinə əlavə olaraq standart HTML teqlərini ehtiva edir. Belə bir səhifə istifadəçi tərəfindən tələb edildikdə, HTTP server sorğunu Java proqram serverinə yönləndirir. JHTML səhifəsi əvvəlcə .java faylına çevrilir və servlet kimi icra edilən [.class](/programming/class/) faylı yaratmaq üçün tərtib edilir. Bu, istifadəçiyə göstərilməsi üçün brauzer sorğusuna qaytarılan standart HTTP və HTML məlumat axını yaradır. Bu, verilənlər bazası ilə əlaqəli sorğuları serverdə yerinə yetirmək və yekun yığılmış nəticəni müştərinin brauzerinə qaytarmaq üçün faydalıdır.
