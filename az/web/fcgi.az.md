## FCGI faylı nədir?

FCGI faylı FastCGI (FCGI) üçün yazılmış mənbə kodu ehtiva edən mənbə kodu faylıdır. O, bir neçə müxtəlif proqramlaşdırma dillərində yazıla bilər və Veb server sorğularının daha etibarlı idarə olunması üçün istifadə olunan CGI standartının genişləndirilməsini ehtiva edir. FCGI üçün ən çox dəstəklənən dillərə həm Perl kodunu, həm də FastCGI API-nin güclü xarici interfeysini dəstəkləyən **Perl** daxildir.

## FCGI fayl formatı

FCGI faylını vacib edən nədir? FCGI interfeysi veb sorğularını idarə etmək və onları çoxsaylı serverlər arasında yaymaq üçün paylanmış hesablamaları dəstəkləyir. Bu, FCGI fayllarını əsasən birdən çox server arasında yük balansı üçün istifadə edir.
