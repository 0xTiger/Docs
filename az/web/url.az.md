## URL faylı nədir?

.url uzantısı olan fayl istənilən vebsayt ünvanı ilə kompüterdə yadda saxlanıla bilən vebsayt URL qısayoludur. O, iki dəfə klikləməklə açıldıqda, URL standart brauzerdə vebsaytı açacaq və səhifə istifadəçiyə göstəriləcək. URL faylları adətən veb-saytlara qısa yol faylları kimi istinad edilir və standart brauzerin simvolu ilə təmsil olunur. URL faylları Microsoft Notepad, Notepad++ və ya Apple TextEdit kimi istənilən mətn redaktorunda açıla və redaktə edilə bilər.

## URL fayl formatı - Əlavə məlumat

URL faylları düz mətn formatında saxlanılır və istənilən mətn redaktorunda açıla və baxıla bilər. Bunlar istifadəçinin digər axtarışları yerinə yetirdiyi standart veb brauzerdə açılır.
