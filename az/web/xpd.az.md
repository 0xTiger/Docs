## XPD faylı nədir?

XPD faylı XML sənədindən yaradılmış veb səhifə faylıdır. Daha tez-tez bu, XSLT çevrilməsi ilə yaradılır. [XPD file format specifications](https://www.w3.org/TR/xml-pipeline/) çox prosesli [XML](/web/xml/) tətbiqlərinin qarşılıqlı işləmə qabiliyyətini artırmaq üçün bir yol müəyyən edən W3-də mövcuddur. XML Pipeline Language XML resursları arasında emal əlaqələrini müəyyən edir.
