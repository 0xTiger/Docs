## DISCOMAP faylı nədir?

DISCOMAP faylı Microsoft Veb Xidmətləri Kəşf Aləti **Disco.exe** tərəfindən yaradılmış çıxış faylıdır. Bu, xüsusi URL-də aşkar edilmiş ASP.NET Veb Xidmətləri olan nəticələri ehtiva edir. Bu cür aşkar edilmiş xidmətlərin ümumi nümunələrinə .DISCO və ya .WSDL fayllarının yerləri daxildir. Microsoft Veb Kəşf aləti işə düşdükdə, DISCO faylında tapılan XML məzmununa əsaslanan kəşfiyyat məlumatlarını axtarır. Nəticələr Microsoft Visual Studio IDE ilə açıla bilən DISCOMAP faylında saxlanılır.
