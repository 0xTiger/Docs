## DO faylı nədir?

DO faylı veb-əsaslı Java proqramlarını çatdırmaq üçün veb serverlər tərəfindən istifadə olunan Java servlet faylıdır. O, adətən faylları emal edən proqramların Struts nəzarətçisi ilə əlaqələndirilir. Bu, DO fayllarına vizual görünüş və verilənlər bazalarından əldə edilən məlumatları ehtiva edən dinamik veb səhifələr yaratmağa imkan verir. Java servletləri dinamik şəkildə yaradılan bu veb proqram səhifələrini ekran üçün sorğu göndərən brauzerə göndərir.

## Fayl Formatını EDİN

DO faylları istifadəçi serverə sorğu göndərdikdə brauzerdə açılan veb faylları kimi saxlanılır. Servlet sorğunu emal edir və DO faylını servlet kimi sorğu göndərən brauzerə göndərir.
