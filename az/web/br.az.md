## BR faylı nədir?

A BR file is a compressed web file generated by applying the open source data compression algorithm, **Brotli**. It is used to store webpage assets such as stylesheets ([CSS](/web/css/)), images ([SVG](/page-description-language/svg/)), [XML](/web/xml/), and scripting files ([JS](/web/js/)). Modern day websites, such as Chrome and Firefox, use BR files to reduce the page loading time, resulting in better user experience.

## BR Fayl Format

BR faylları Brotli sıxılma alqoritmi ilə yaradılan sıxılmış veb fayllarıdır. Brotli sıxılma itkisiz məlumat sıxılma alqoritmidir və Google tərəfindən Zopfli sıxılma alqoritmi üçün hazırlanmışdır. O, LZ77 itkisiz sıxılma alqoritmi və Huffman kodlaşdırmasının birləşməsinə əsaslanır.

Kiçik ölçüsünə görə BR faylları veb serverlər və məzmun çatdırma şəbəkələri (CDN) tərəfindən istifadə olunur. Bu serverlərə edilən sorğular HTTP məzmununun sıxılması ilə nəticələnir və vebsaytların daha sürətli yüklənməsinə səbəb olur. Brotli daha populyarlaşdı və gzip-dən daha yaxşı sıxılma təmin edir.
