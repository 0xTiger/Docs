## SIKILMALI fayl nədir?

Sıxılmış fayl **WordCompress** plug-in ilə yaradılan kiçik ölçülü Wordpress sıxılmış veb səhifəsidir. O, plagin tərəfindən HTML, CSS və JavaScript kodunu sıxaraq yaradılan veb səhifəni təmsil etmək üçün bir kod sətrini ehtiva edir. Bu konvertasiya edilmiş SIKILMAZ faylların ölçüsü kiçikdir ki, bu da internet üzərindən daha sürətli yükləməyə və köçürməyə kömək edir.

### SıKILMIŞ Fayl Format - Ətraflı Məlumat

SıKILMIŞ fayllar mətn faylları kimi saxlanılır və onları Microsoft Notepad, Notepad++ və Apple TextEdit kimi istənilən mətn redaktoru ilə açmaq olar. Mətn redaktorlarına əlavə olaraq, SIKILMIŞ fayllar həm Windows, həm də Mac OS üçün mövcud olan Adobe Dreamweaver kimi veb qurma proqramı ilə açıla bilər.

## WordCompress Plugin haqqında

WordCompress plagini əvvəlcə WordPress veb səhifələrini sıxılmış fayl formatına sıxışdırmaq üçün istifadə edilmişdir. Plugin artıq dəstəklənmir və dayandırılıb. O, səhifənin mənbə kodundan bütün boş boşluqları silməklə veb səhifənin sorğu kodunu tək icra edilə bilən fayla yazacaq. Sıxılmış fayllar səhifənin orijinal mənbə koduna daxil edilmiş orijinal şərhləri, girintiləri və başqa boşluqları qorumaq üçün veb-səhifənin orijinal mənbə kodu ilə yanaşı saxlanılır.
