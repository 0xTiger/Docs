## GSP faylı nədir?

GSP faylı e-poçtun çatdırılmasında problem olduğunu göstərən səhv mesajı faylıdır. O, çoxsaylı kiçik və orta biznes tərəfindən e-poçt rabitəsi üçün istifadə olunan e-poçt server proqramı olan IMail Server tərəfindən göndərilir. Səhv çox sayda məlumat, [MSG](/email/msg/) faylında məlumat itkisi və ya yaddaş çatışmazlığı kimi bir sıra səbəblərlə bağlı ola bilər.

## GSP Fayl Format - Ətraflı Məlumat

GSP faylları veb fayl formatında saxlanılır və Microsoft Outlook kimi e-poçt müştəriləri ilə asanlıqla baxıla bilər. GSP faylı xəta ilə nəticələnən göndərilmiş e-poçta cavab olaraq yaradılır.
