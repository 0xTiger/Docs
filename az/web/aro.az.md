## ARO faylı nədir?

ARO faylı SteelArrow Veb Tətbiqi ilə yaradılan server tərəfindəki veb səhifə faylıdır. O, [HTML](/web/html/) və SteelArrow teqlərini və skriptlərini ehtiva edir. Bunlar istifadəçinin brauzerinə göndərilməzdən əvvəl tam cavab səhifəsi yaratmaq üçün serverdə yerinə yetirilir. ARO faylları HTML məzmununun demək olar ki, 95%-ni, qalanları isə SteelArrow kodundan ibarətdir. ARO fayllarının sizin üçün işləməsi üçün SteelArrow Veb Proqramları serveri veb server maşınında quraşdırılmalı və işləməlidir.

## ARO fayl formatı

ARO files are written in HTML markup language file with embedded JavaScript code and Java applets. [SteelArrow tags](http://www.steelarrow.com/docs/basics1.aro) are the basic building blocks for development of web pages. Though these don't change the display of the page, they are used to fill the page with data. In addition, they may also be used to control the output with conditional statements.

### ARO Tag Nümunəsi

The<SAOUTPUT> ARO etiketi tərtibatçılara skriptin istənilən yerində məlumat dəyərlərini çıxarmağa imkan verir. Məsələn, PARAM cədvəlinin çıxışını ilə əldə etmək üçün istifadə edilə bilər<SAOUTPUT VALUE=PARAM[]> HTML daxilində göstərilə bilər<BODY> etiket.
