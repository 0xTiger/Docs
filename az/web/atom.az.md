## ATOM faylı nədir?

ATOM faylı Atom lenti və giriş sənədlərinin strukturu üçün Sindikasiya formatıdır. .RSS fayllarına bənzər, ATOM fayl formatı məzmunun dərci üçün standart olan XML əsaslı sindikasiya formatıdır. ATOM faylı proqram təminatı proqramlarına veb-saytda dərc olunan yeniləmələri yoxlamağa imkan verən veb yayımları üçün istifadə olunur. O, başlıqlar, tam mətnli məqalələr, çıxarışlar və ya vebsaytdakı məzmuna keçidlər kimi müxtəlif növlərdə ola bilən girişləri ehtiva edir.

**ATOM fayllarını** aça bilən proqramlara **Apple Safari** daxildir.

## ATOM Fayl Format - Ətraflı Məlumat

ATOM faylları məlumat mübadiləsi üçün universal format olan məşhur XML fayl formatında saxlanılır və dərc olunur. O, RSS fayl formatının məhdudiyyətlərini və qüsurlarını nəzərə alaraq RSS-ə alternativ olaraq hazırlanmışdır.

### Atom Sənədlərinin Növləri

 1. `Atom Giriş Sənədləri` - Atom lenti üçün giriş kimi tanınan bir məlumat elementindən ibarət XML sənədidir. Bir sıra uşaq elementləri ehtiva edən atom giriş elementinə malikdir. Atom girişinin məzmunu düz mətn, HTML, XHTML və ya digər IANA (Internet Assigned Numbers Authority) media növü ola bilər.
 1. `Atom Yem Sənədləri` - Bu, Atom lenti və lent üçün bir və ya daha çox giriş haqqında metadata təmin edən XML sənədidir. Müştəri lentdən məlumat üçün sorğu göndərdikdə, sorğunu yerinə yetirmək üçün bir sıra Atom girişlərini özündə cəmləşdirən server tərəfindən lent sənədi yaradılır.
 1. `Atom Collection` - Bu, redaktə edilə bilən Atom qeydlərinin URL-lərini ehtiva edən xüsusi növ Atom lent sənədidir.
