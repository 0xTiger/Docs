## HYPE faylı nədir?

A HYPE file is a project file saved using the Hype software that is used for creation of web content with HTML5. O, qrafika, mətn, animasiya, səhnə məlumatı və ümumi məzmunu təşkil edən digər elementlər kimi veb elementləri ehtiva edir. HYPE faylındakı veb məzmunu veb-səhifədə dərc oluna bilən veb fayllarına ixrac edilə bilər.

HYPE faylı [Tumult Hype](https://tumult.com/hype/) proqramı ilə açıla bilər.

## HYPE Fayl Format

HYPE faylları xüsusi fayl formatında saxlanılır və onların daxili təfərrüatları ictimaiyyətə məlum deyil. HYPE faylının içindəki məzmuna yalnız Tumult Hype proqramında açıldıqda baxmaq olar.

### HYPE faylının məzmunu

Müxtəlif məqsədlər üçün veb məzmun yaratmaq üçün Hype tətbiqindən istifadə etdiyiniz zaman HYPE faylı yaradılır. O, e-Kartlar, e-kitablar, təqdimatlar, reklamlar, infoqrafika və veb-səhifə materialı kimi məzmun yaratmaq üçün əsas kadr əsaslı animasiyalar haqqında məlumatları saxlayır. Bunların məzmunu HTML5 ilə təchiz edilmişdir ki, bu da məzmunun masaüstü kompüterlərdən iPhone və iPad-lərə qədər cihazlarda göstərilməsinə imkan verir. Tumult Hype ən müasir HTML5, [CSS3](/web/css/) üslubları və [JavaScript](/web/js/) çıxışlarını təqdim edir.
