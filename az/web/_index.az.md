Veb səhifə qurarkən bir sıra fayl növləri tələb olunur. Veb fayl formatları veb səhifələrin inkişafı standartlarını müəyyən edir və onların qurulduğu platforma ilə əlaqədardır. Statik və dinamik veb-səhifələrdən ibarət tam vebsayt yaradıla bilər. Əksər müasir veb-saytlar veb serverdə yüklənən və işləyən Active Server Pages (ASP) kimi server tərəfi texnologiyasında qurulur. Bunlara həmçinin UI-nin ümumi görünüşünü və hissini tərtib etmək üçün istifadə olunan kaskad üslub vərəqləri (CSS) və skript faylları daxildir.

Ümumi veb fayl uzantıları və onların əlaqəli fayl formatlarına **[HTML](/web/html/)** (Hipermətn İşarələmə Dili), **[ASP](/web/asp/) (Aktiv Server Səhifələri)** və **CSS** (Cascading Style Sheets) daxildir. ).

Veb fayl formatları ilə bağlı sorğularınız var? Fayl Format mütəxəssislərinin paylaşdığı biliklərdən faydalanmaq üçün [forums](https://forum.fileformat.com/c/web/13) icmamıza daxil olun.
