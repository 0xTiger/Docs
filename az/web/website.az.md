## WEBSITE faylı nədir?

A WEBSITE file is a website shortcut file created with Microsoft Internet Explorer (IE). It contains a URL and an icon (ICO) file that points to the website's favicon. The icon is used for displaying the shortcut. WEBSITE file format was introduced with the release of IE version 9. IE-nin əvvəlki versiyaları qısa yolları veb səhifələrə [.url](/web/url/) faylları kimi saxlamaq üçün istifadə olunurdu.

## WEBSITE fayl formatı

.veb-sayt faylı məzmunu istənilən mətn redaktorunda baxıla bilən düz mətn faylları kimi diskdə saxlanılır. Digər məzmunla yanaşı, qısayol olduğu hədəf URL-i ehtiva edir. Hədəf veb-saytını IE-dən başqa brauzerdə açmaq istəyirsinizsə, bu URL-i kopyalayıb vebsayta baxmaq üçün onu Google Chrome, Safari və ya Firefox kimi digər brauzerlərə yapışdıra bilərsiniz.

## .WEBSITE faylını necə yaratmaq olar?

Ünvan çubuğundakı URL-i klikləyərək və onu kompüterinizdə iş masası və ya sənədlər kimi istədiyiniz yerə çəkərək IE-də WEBSITE faylı yaradıla bilər. Bu, həmin internet məkanına qısa yol yaradır və qısayolu iki dəfə klikləməklə aça bilər. Qısayolu `URL` faylı kimi saxlamaq istəyirsinizsə, hədəf saytın favikonunu masaüstünə çəkməzdən əvvəl SHIFT düyməsini basıb saxlayın.
