## EDGE faylı nədir?

.edge uzantısı olan fayl, layihə Adobe Edge ilə yaradılan və saxlandıqda yaradılan Adobe Edge Project faylıdır. Adobe Edge veb dizaynerlərinə həm masaüstü, həm də mobil brauzerlər üçün veb, rəqəmsal nəşrlər və zəngin media reklamları üçün interaktiv [HTML](/web/html/) animatorlar yaratmağa imkan verir. Buna HTML/HTML5, [JavaScript](/web/js/) və [CSS3](/web/css/) istifadə etməklə nail olunur. Edge faylları düz mətn faylları kimi yaradılır və Microsoft Notepad, Apple TextEdit və Notepad++ kimi bir neçə fərqli proqramla açıla bilər.

## EDGE fayl formatı

EDGE faylları düz mətn formatında saxlanılır və animasiyanı aktivləşdirmək üçün jQuery-dən istifadə edir. EDGE animasiyasının alternativi artıq dəstəklənməyən Flash faylları idi. EDGE faylı HTML, JavaScript və CSS faylları ilə birlikdə saxlanılır. EDGE faylı açıldıqdan sonra veb səhifə, yəni HTML faylı açılır.
