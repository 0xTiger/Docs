## CCBJS faylı nədir?

CCBJS faylı 3D proqram, oyun və vebsayt redaktoru, **CopperCube** tərəfindən yaradılmış Javascript faylıdır. O, .ccb faylından yaradılmış [.html file](/web/html/)-in bir hissəsi kimi yaradılan JavaScript kodunu ehtiva edir. CopperCube heç bir kod yazmadan oyun məntiqini və qarşılıqlı əlaqəni inkişaf etdirmək üçün istifadəçi dostu skript interfeysi təqdim edir. CopperCube tərəfindən dəstəklənən digər fayl formatlarına .ccscene, .ccmodel, .ccmaterial, .cctex, .ccjs, .ccp və .ccpak daxildir.

## CCBJS fayl formatı

CCBJS fayl formatının daxili fayl formatı təfərrüatları belə məlum deyil. Javascript faylları diskdə düz mətn faylları kimi saxlanılır və Microsoft Notepad, Notepad++ və Apple TextEdit kimi istənilən mətn redaktoru ilə açıla bilər.
