## BROWSER faylı nədir?

.Browser genişləndirilməsi olan fayl ASP.NET veb proqramları tərəfindən veb brauzerlərin imkanlarını və konfiqurasiyasını müəyyən etmək üçün istifadə olunur. O, brauzerin imkanları, məhdudiyyətləri və xüsusiyyətləri haqqında məlumatları ehtiva edir. Bu xüsusiyyətlər ASP.NET çərçivəsinə sorğu edən brauzeri tanımağa və müvafiq olaraq veb səhifəni istifadəçilərə təqdim etməyə kömək edir.

## BROWSER fayl formatı

BROWSER tərif faylları brauzerin imkanlarını müəyyən etmək üçün ASP.NET tətbiqi tərəfindən istifadə olunur. Bu, server tərəfindən sorğu edən brauzerə uyğun olan işarələmə və digər resurslar yaratmaq üçün istifadə edilən bu faylda olan məlumatdır.

BROWSER faylları düz mətn faylı formatında saxlanılır. Siz BROWSER faylını Notepad, Notepad++, Apple TextEdit və Visual Studio IDE kimi mətn redaktorlarında aça bilərsiniz.

### BROWSER Fayl Formatının Fayl Strukturu

BROWSER tərif faylları iyerarxik quruluşdan istifadə edir. Hər bir .brauzer faylı müəyyən bir brauzerin və ya bir qrup brauzerin imkanlarını və xüsusiyyətlərini özündə birləşdirən XML işarələməsini ehtiva edir. Bu təfərrüatlara agent sətri, versiya nömrələri və JavaScript, CSS və ya digər HTML elementləri kimi texnologiyalara dəstək kimi xüsusiyyətlər daxildir. Bu brauzer tərif fayllarından istifadə etməklə, ASP.NET tərtibatçıları müxtəlif brauzerlərin imkanlarına əsaslanaraq istifadəçi təcrübələrini fərdiləşdirirlər.
