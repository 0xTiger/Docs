## ASR faylı nədir?

ASR faylı Adobe Dreamweaver ilə yaradılmış və server tərəfində yüklənən və işləyən ActionScript faylıdır. O, həmçinin tez-tez çoxdan dayandırılmış Contribute [HTML](/web/html/) redaktoru tərəfindən yaradılır. ASR faylı standart skript dilində yazılmış skripti ehtiva edir və ColdFusion-da daxili server əsaslı xüsusiyyətlərə daxil olmaq imkanına malikdir.

## ASR fayl formatı

ASR faylı standart skript dilində hazırlanmış skripti ehtiva edir. Onu Adobe Dreamweaver, Contribute HTML Redaktoru və Notepad və Notepad++ kimi istənilən mətn redaktorları ilə açmaq olar.
