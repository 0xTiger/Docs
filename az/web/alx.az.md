## ALX faylı nədir?

ALX faylı [HTML](/web/html/) komponentləri şəklində Veb səhifələrə daxil edilmiş ActiveX faylıdır. ActiveX nəzarətləri rəsmi olaraq OLE nəzarətləri və ya OCX kimi tanınırdı. ALX faylları veb səhifəyə müxtəlif funksiyaları daxil etmək üçün istifadə edilə bilər, məsələn, səhifənin məzmununu vaxtaşırı yeniləmək üçün taymer nəzarəti əlavə etmək və ya istifadəçi fəaliyyətinə əsaslanan alət tipli yardımı göstərmək üçün Pop-up pəncərə nəzarəti əlavə etmək. ALX faylları istifadə edərək bağlı qayğı<Param> veb səhifənin HTML kodu daxilində etiket. Siz ALX fayllarını Microsoft Internet Explore, Microsoft ActiveX Control Pad və Notepad və Notepad++ kimi mətn redaktorlarında aça bilərsiniz.

## ALX Fayl Format - Ətraflı Məlumat

ALX faylları düz mətn faylı formatındadır və onları mətn redaktorunda açmaqla redaktə edilə bilər. ActiveX nəzarətləri veb səhifəyə endirməklə veb səhifəyə daxil edilə bilər. Bununla belə, dinamik veb səhifələr yaratmaq üçün idarəetmələr [JScript](/web/js/) köməyi ilə birləşdirilir. Veb səhifədəki nəzarət elementləri idarəetmə vasitələrinin hadisələri və dəyərləri ilə əlaqələndirilir, beləliklə, istifadəçi qarşılıqlı əlaqəsi üçün tam nəzarət təmin edilir.
