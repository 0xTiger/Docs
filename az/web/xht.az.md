## XHT faylı nədir?

.xht uzantılı fayl [XHTML](/web/xhtml/) (Genişləndirilmiş Hipermətn İşarələmə Dili) fayl növü ilə əlaqəli veb fayldır və özü də [XML](/web/xml/) əsaslı işarələmə faylıdır. Bu faylların hər ikisi HTML-yə bənzəyir, lakin daha sərt XML kimi sintaksisə malikdir. XHT faylları daha strukturlaşdırılmış şəkildə hazırlanmışdır, daha az skript tələb edir və XML-in mövcud imkanlarından istifadə etməklə yanaşı ümumidir.

## XHT fayl formatı

XHT faylı standart olaraq UTF-8 kodlaşdırması ilə düz mətn faylı kimi yaradılır və saxlanılır. O, Unicode kodlamasını dəstəkləyən istənilən mətn redaktoru ilə açıla və redaktə edilə bilən XHTML sənədinin tam mənbə kodunu ehtiva edir. Mətn redaktorlarına əlavə olaraq, XHT fayl formatı müasir veb-brauzerlərin əksəriyyəti tərəfindən başa düşüləndir və onlardan istifadə etməklə açıla bilər.

## XHT nümunəsi

XHT sənədinin aşağıdakı nümunəsi XML bəyannamələrini müəyyən edir.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
