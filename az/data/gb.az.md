## GB GenBank Data faylı nədir?

GenBank fayl formatı kimi də tanınan GB fayl formatı əlaqəli metadata ilə birlikdə DNT, RNT və zülal ardıcıllığı kimi bioloji ardıcıllıq məlumatlarını saxlamaq üçün istifadə edilən standart düz mətn formatıdır. Genetik məlumatların mübadiləsi və saxlanması üçün bioinformatika və molekulyar biologiyada geniş istifadə olunur.

## GB Fayl Format Məlumatı

GenBank fayl formatının əsas xüsusiyyətləri bunlardır:

1.  **Başlıq Məlumatı:** Fayl ardıcıllıq və onun mənbəyi haqqında məlumat verən başlıq bölməsi ilə başlayır; bura qoşulma nömrəsi, orqanizm və ardıcıllıq məlumatlarının dərc edildiyi ədəbiyyata istinadlar kimi təfərrüatlar daxildir.
    
2.  **Funksiyalar Bölməsi:** Başlığın ardınca genlər, kodlaşdırma bölgələri, tənzimləyici elementlər və digər mühüm yerlər kimi ardıcıllığın müxtəlif xüsusiyyətlərini təsvir edən xüsusiyyətlər bölməsi var; hər bir xüsusiyyət onun ardıcıllıqla yeri, xüsusiyyət növü və əlavə kvalifikatorlar kimi xüsusi məlumatlarla qeyd olunur.
    
3.  **Ardıcıllıq Məlumatı:** Faktiki ardıcıllıq məlumatları xüsusiyyətlər bölməsini izləyir; bu bölmə nukleotid və ya amin turşusu ardıcıllığı şəklində xam genetik məlumatı ehtiva edir. Ardıcıllıq məlumatları adətən oxunaqlılıq üçün sətir fasilələri ilə standart formatda təqdim olunur.
    
4.  **Format Tags:** GenBank files use specific tags and keywords to structure the information; these tags help define different sections of file and provide a standardized way for software programs to interpret and parse the data.
    
5.  **Annotasiya:** GenBank fayllarına ardıcıllıqla müxtəlif regionların bioloji əhəmiyyəti haqqında məlumat verən geniş annotasiyalar daxildir; bura kodlaşdırma bölgələri, protein məhsulları və funksional annotasiyalar haqqında təfərrüatlar daxil ola bilər.
    
6.  **Origin Line:** Ardıcıllıq məlumatları tez-tez ardıcıllığın başlanğıcını göstərən və faktiki nukleotid və ya amin turşusu ardıcıllığı ilə müşayiət olunan ORIGIN xətti ilə dayandırılır.
