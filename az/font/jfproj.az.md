## JFPROJ faylı nədir?

A file with .jfproj extension is a JSON based custom font created with Microsoft Font Maker application on Windows 10. Bunlar planşetlər kimi sensor ekran cihazları ilə elektron qələmdən istifadə etməklə yaradıla bilər, baxmayaraq ki, siçan kimi digər kompüter əlavələri də kompüterlər vəziyyətində istifadə edilə bilər. JFPROJ faylları simvollar və sözlər arasındakı məsafəni və şriftin ölçüsünü təyin edərək simvolları ekrana göstərmək üçün istifadə olunan qlifləri ehtiva edir. Microsoft Font Maker proqramını Windows mağazasında yükləmək mümkündür.
