## FODT faylı nədir?

.fodt uzantılı fayl OpenDocument flat [XML](/web/xml/) fayl formatında saxlanılan mətn sənədidir. [.zip](/compression/zip/) konteyner fayl formatında saxlanılan .odt fayllarından fərqli olaraq, FODT faylları düz XML sənədləri kimi baxmaq üçün istənilən mətn redaktorunda açıla bilən insanlar tərəfindən oxuna bilən fayllar kimi diskdə saxlanılır. OpenDocument ofis sənədi standartı OASIS sənaye konsorsiumu tərəfindən hazırlanmışdır və hər kəsin heç bir məhdudiyyət olmadan hər kəs tərəfindən həyata keçirilməsinə yol açır. Digər məşhur OpenDocument fayl formatlarına [.odt](/word-processing/odt/), [.ott](/word-processing/ott/), [.ods](/spreadsheet/ods/), [.odp](/presentation/odp/) və buna bənzər digərləri daxildir.

## FODT Fayl Format - Ətraflı Məlumat

FODT faylları düz XML fayl formatında (həmçinin Flat XML və ya Sıxılmamış XML kimi tanınır) saxlanılır, baxmayaraq ki, OpenDocument tərəfindən çox istifadə olunan sənəd formatı deyil. OpenDocument faylları adətən bir sıra fayl və kataloqlardan ibarət ZIP paketi kimi saxlanılır. XML əsaslı OpenDocument faylları bəzi proqram təminatında dəstəklənməyə bilər və buna görə də onların istifadəsi ZIP paket əsaslı fayllarla müqayisədə daha az yayılmışdır.
