## EX faylı nədir?

EX faylı Euphoria proqramlaşdırma dilindən istifadə edərək hazırlanmış proqramlar üçün mənbə kodu ehtiva edən mənbə kodu faylıdır. Euphoria, mürəkkəb proqramlaşdırma dillərindən çox proqramlaşdırmanın sadəliyinə inanan prosedur proqramlaşdırma dilidir. Euphoria-da hazırlanmış proqram çoxsaylı EX fayllarından ibarət ola bilər, lakin bir EX faylı proqramın tam mənbəyini də ehtiva edə bilər.

**EX fayllarını aça bilən proqramlara** openEuphoria Euphoria, Microsoft Notepad, Notepad++ və Atom daxildir.

## EX Fayl Formatının Qısa Tarixi

The development and existence of EX file format is linked with the development of Euphoria programming language which was created by Robert Craig. It was first released for MS-DOS. It became open-source software in 2006 with the release of version 3. Onun inkişafı layihəni idarə edən və inkişaf etdirən **openEuphoria qrupu** tərəfindən həyata keçirilib. Onun 4-cü versiyası 2010-cu ilin dekabrında buraxılıb və Windows, Linux, macOS və üç BSD çeşidi üçün əlçatandır.

Euphoria əvvəlcə **[C](/programming/c/)** proqramlaşdırma dilində yazılmışdır. Daha sonra Euphoria tərcüməçisi iki hissəyə bölündü:

 * `Front-end Parser` - Euphoria dilində yazılmışdır və Euphoria-to-C tərcüməçisi və Binder ilə istifadə olunur.
 * `Back-end` - C proqramlaşdırma dilində yazılmış işləmə vaxtı kitabxanası.

## EX Fayl Format - Ətraflı Məlumat

EX faylları düz mətn faylları kimi saxlanılır və Euphoria proqramlaşdırma dilində yazılmış proqramlar üçün mənbə kodunu ehtiva edir. Bu fayllar istənilən mətn redaktorunda açıla bilər.
