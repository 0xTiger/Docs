## CXX faylı nədir?

CXX faylı proqramların yazılması üçün C++ ilə oxşar mənbə kodu faylıdır. O, bütün proqramın tam icrasını ehtiva edə bilər və ya digər C++ faylları ilə layihəyə daxil edilə bilən proqramın qismən icrasına malik ola bilər. CXX faylları istənilən mətn redaktorunda və Notepad++, Turbo C++ və Visual Studio kimi məşhur proqramlaşdırma İDE-lərində açıla bilər.

## CXX Fayl Formatı

CXX faylları düz mətn faylı kimi saxlanılır və C++ sintaksisində yazılmış kodu ehtiva edir. CXX faylları adətən [.cpp](/programming/cpp/) uzantısı ilə də tapılır. CXX faylları bəzi kompilyatorlar tərəfindən istifadə edilən .cpp faylına alternativ hesab olunur.
