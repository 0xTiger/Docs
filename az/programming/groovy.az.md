## GROOVY faylı nədir?

GROOVY faylı Groovy proqramlaşdırma dilində yazılmış mənbə kodu faylıdır. GROOVY faylı daxilində yazılmış kod obyekt yönümlü dil [Java](/programming/java/) ilə oxşardır ki, bu da proqramların dizaynını və işlənməsini asanlaşdırır. GROOVY faylı Java virtual maşını (JVM) bayt koduna tərtib edilib və digər Java kodu və kitabxanaları ilə uyğun gəlir.

## GROOVY Fayl Format - Ətraflı Məlumat

GROOVY faylları Groovy sintaksisində yazılmış mənbə kodunu ehtiva edir. O, həm proqramlaşdırma dili, həm də Java Platforması üçün skript dilləri kimi istifadə edilə bilən kodu ehtiva edə bilər. Groovy dili Python, Ruby və Smalltalk dillərinə bənzər xüsusiyyətlərə malikdir.
