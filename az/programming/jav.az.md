## Jav faylı nədir?

Jav faylı Java proqramlaşdırma dilində yaradılmış Java mənbə kodu faylıdır. O, Java mənbə faylına bənzəyir və Java tərcüməçiləri ilə tərtib edilə bilər. [Java](/programming/java/) faylları kimi, Java faylı Java proqramlarının hazırlanmasında istifadə olunur. Java layihəsində mövcud olan çoxlu mənbə kodu fayllarından biri ola bilər.

## Java fayl formatı

Jav faylları mətn faylları kimi saxlanılır və istənilən mətn redaktoru ilə açıla bilər. Java fayl növləri adətən Java fayl növü ilə əvəz edilmişdir və adətən Java proqramlarının yazılması üçün istifadə olunur. Jav kodları Java faylları ilə işləməyi asanlaşdıran Eclipse və IntelliJ Idea kimi IDE-lərlə yazıla bilər.

### Java fayl formatı nümunəsi

```java
/*
The example code prints
Hello World from Java to the console.
*/
    public class JavaApp {
        public static void main(String[] args) {
            System.out.println("Hello World."); // Prints the string to the console.
        }
    }
```
