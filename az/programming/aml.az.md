# AML - Microsoft Yardım İşarələmə Dil Faylı

## AML faylı nədir?

AML (Yardım İşarələmə Dili) faylı Microsoft Assistance Markup Language (MAML) ilə yaradılan XML faylıdır. O, Microsoft Windows OS üçün onlayn yardım göstərmək üçün Microsoft tərəfindən hazırlanmışdır. Bundan əvvəl Windows yardımı tərtib edilmiş HTML [CHM](/web/chm/) fayl formatında mövcud idi. AML faylları proqrama mənbə kodu və dəstək səhifələri yaratmağa imkan verən strukturlaşdırılmış sənəd təqdim edir. Bu, sənədləri və onların tərkib elementlərini kontekstinə görə müəyyən etməyə imkan verir. AML yardım faylları onlayn yayımlanır və onlayn mövcud paket yeniləmələrindən yenilənmək üçün konfiqurasiya edilə bilər.

## MAML fayl strukturu

MAML istifadə edərək yaradılan AML faylları [XML](/web/xml/) faylları kimi saxlanılır ki, bunlardan geniş spektrli aktiv anlayışları ifadə etmək üçün istifadə edilə bilər. O, həmçinin addım-addım sehrbazla istifadəçi üçün yardım faylını interaktiv edən rəhbər yardım (aktiv məzmun sehrbazı) təmin edə bilər.

An AML file generated using the MAML follows its authoring structure that can be divided into segments like:

 * Konseptual
 * Tez-tez verilən suallar
 * Lüğət
 * Prosedur
 * İstinad
 * Yenidən istifadə edilə bilən məzmun
 * Tapşırıq
 * Problemlərin aradan qaldırılması və
 * Dərslik

## MAML faylları necə yaradılır?

MAML faylları aşağıdakı üsullardan biri ilə yaradıla bilər:

 * Sandcastle-dan istifadə - Yardım faylını yaratmaq üçün bir sıra sxemlərdən və proqram icraedicilərindən istifadə edir. Lakin bu vasitə dayandırılıb.
 * DocProject-dən istifadə - Yardım məzmunu yaratmaq üçün Microsoft Visual Studio daxilində icra oluna bilən Microsoft Visual Studio plagini.

MAML faylları Sandcastle, .XSL sxemləri və proqram icraediciləri paketindən istifadə etməklə yaradıla bilər. Onlar həmçinin DocProject, Microsoft Visual Studio yardım müəlliflik aləti əlavəsi ilə yaradıla bilər.

# AML - Arc Makro Dil Faylı

## ARC Macro AML faylı nədir?

An AML (ARC Macro Language) file is a scripting file generated with the ArcInfo Workstation GIS application. It is written in the ARC Macro Language, which is a proprietary high-level algorithmic language for creating GIS applications in ArcInfo. AML was designed by ESRI in 1986 and served the purpose of creating applications from their command line ARCINFO GIS system. Being scripting file, AML files can contain script commands to carry out a number of tasks such as creating user interface components and manipulate map data.

## AML Fayl Format - Ətraflı Məlumat

AML, skript faylları olmaqla, məlumatları düz mətn faylları kimi diskdə saxlayın. O, PRIMOS əməliyyat sisteminin CPL qabıq dilinə əsaslanan AML sintaksisini izləyir. AML dili ArcGIS dəstinin bir hissəsi olan və VBA və ya Python vasitəsilə proqramlaşdırma dəstəyi təmin etmək üçün ArcObjects istifadə edən ESRI-nin geoprosesinq çərçivəsi ilə əvəz edilmişdir.
