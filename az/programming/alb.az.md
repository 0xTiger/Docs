## ALB faylı nədir?

ALB faylı Veb proqramları hazırlamaq üçün istifadə edilən **Alpha Five** IDE tətbiqi tərəfindən istifadə edilən kitabxana faylıdır. Onun əsas məqsədi müvafiq Alpha Five [ADB file](/database/adb/) üçün məlumat saxlamaqdır. Buraya parametrlər, skript göstəriciləri, menyular, istifadəçi tərəfindən müəyyən edilmiş funksiyalar, bitmaplar, alətlər panelləri və idxal tərifləri kimi məlumat lüğəti məlumatları daxildir.

ALB faylları **Alpha Software Alpha Anywhere** ilə açıla bilər.

## ALB Fayl Format - Ətraflı Məlumat

ALB faylları Alpha File xüsusi fayl formatında saxlanılır və onların daxili fayl formatı strukturu ictimaiyyətə açıq deyil. Bunlar ümumi Alpha Five tərtibatçı kitabxanası verilənlər bazasına əlavə olaraq saxlanılır. Kitabxanalar olan ALB faylları digər Alpha Five proqramlarında təkrar istifadə edilə bilər.
