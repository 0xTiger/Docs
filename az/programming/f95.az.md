## F95 faylı nədir?

.f95 uzantısı olan fayl adətən onun Fortran 95 mənbə kodu faylı olduğunu göstərir. Fortran ümumi təyinatlı, imperativ proqramlaşdırma dilidir və rəqəmsal və elmi hesablamalar üçün xüsusilə uyğundur. Fortran 95 Fortran 90-ın təkmilləşdirilmiş versiyasıdır və bir neçə yeni funksiya və təkmilləşdirmələr təqdim edir.

Fortran mənbə kodu fayllarında adətən Fortran proqramlaşdırma dilində yazılmış proqram təlimatları var; .f95 uzantılı faylınız varsa, o, çox güman ki, Fortran kompilyatorundan istifadə etməklə tərtib edilməlidir; kompilyasiya insan tərəfindən oxuna bilən mənbə kodunun kompüter tərəfindən yerinə yetirilə bilən maşın tərəfindən oxuna bilən ikili koda çevrilməsi prosesidir.

## F95 faylını necə açmaq olar?

Əgər siz .f95” faylında proqramla işləmək və ya işlətmək istəyirsinizsə, sizə Fortran kompilyatoru lazımdır; məşhur Fortran kompilyatorlarına GNU Fortran (gfortran), Intel Fortran Compiler (ifort) və başqaları daxildir. Fortran proqramlarını kompilyator tərəfindən verilən müvafiq əmrlərdən istifadə edərək tərtib edə və işlədə bilərsiniz.

Məsələn, sizdə gfortran quraşdırılıbsa, Fortran proqramını tərtib etmək və işlətmək üçün terminalda aşağıdakı əmrlərdən istifadə edə bilərsiniz:

```
# Compile the Fortran program
gfortran -o my_program my_program.f95

# Run the compiled program
./my_program
```

Əgər siz yalnız F95 faylının məzmununa baxmaq istəyirsinizsə, o, düz mətn faylıdır, ona görə də onu istənilən mətn redaktoru ilə aça bilərsiniz.
