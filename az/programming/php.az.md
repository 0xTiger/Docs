## PHP faylı nədir?
.php uzantısı olan fayl veb serverdə icra edilmək üçün server tərəfi skriptləri yazmaq üçün istifadə edilən açıq mənbəli proqramlaşdırma dilinə aiddir. Bu, adətən geniş miqyaslı veb proqramların inkişafı üçün istifadə olunan ən çox istifadə edilən veb-skript dilidir. Vebdəki ən böyük bloq sistemi, yəni WordPress və ən böyük sosial şəbəkə Facebook PHP dilinə əsaslanır. O, Windows, Linux, Unix, Mac OS X və s. kimi müxtəlif platformalarla uyğun gəlir. Həmçinin o, hazırda istifadə olunan demək olar ki, bütün serverlərdə, yəni Apache, IIS və s.-də uğurla işləyir. PHP MySQL də daxil olmaqla geniş verilənlər bazasını dəstəkləyir.

## PHP fayl formatı ##

Tək PHP mətn, CSS, [HTML](/web/html/), JavaScript və PHP kodunu ehtiva edə bilər. PHP kodu serverdə icra olunur və nəticə adi HTML kimi brauzerə qaytarılır. PHP faylları istənilən mətn redaktoru ilə açıla və yerində redaktə edilə bilər, baxmayaraq ki, Adobe Dreamweaver, Eclipse PHP İnkişaf alətləri kimi proqramlar PHP kodunu yazmaq və dəyişdirmək üçün rahat üsul təklif edir.

### Sintaksis ###

Serverdə PHP skripti icra edilir və sadə HTML nəticəsi brauzerə geri göndərilir. PHP skripti ilə başlayır<?php and ends with ?>

### PHP nümunəsi ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
