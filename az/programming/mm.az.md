## MM faylı nədir?

MM faylı həm Objective-C, həm də Objective-C++ proqramlaşdırma kodunu ehtiva edə bilən mənbə kodu faylıdır. O, MacOS-da istifadə olunur və yalnız Objective-C kodunu ehtiva edən .M fayllarından fərqlənir. Kompilyator .mm genişlənməsini gördükdə faylda mövcud olan C++ siniflərini tərtib edə bilir. Tək mənbə kodu faylı həm [C](/programming/c/), həm də [C++](/programming/cpp/) dillərinin qarışığından ibarət ola bildiyinə görə, .m fayllarının adının .mm genişlənməsi ilə dəyişdirilməsi C++ funksiyalarını quraşdırmaq imkanı verir. .MM faylını aça bilən proqramlar daxildir. Apple TextEdit, Microsoft Notepad, Notepad++ və Microsoft Wordpad.
