## HTA faylı nədir?

HTMLA **Hypertext Markup Language Application** mənasını verir, Microsoft Windows ilə uyğun gələn proqramdır. Bu proqramın mənbə koduna [HTML](/web/html/) və [JavaScript](/web/js/) kimi birdən çox skript dili daxildir. İstifadəçi interfeysi üçün HTML Tətbiqinə üstünlük verilir, proqram məntiqinin tələbini yerinə yetirmək üçün hər hansı digər skript dili istifadə olunur.

HTML Tətbiqi internet brauzerinin təhlükəsizlik modelindən müstəqildir və tam etibarlı proqram kimi işləyir. Bu proqramlarla bağlı fayllar üçün istifadə edilən genişləndirmə HTA-dır. Bu proqramlara digər skript dillərinin xassələri ilə yanaşı HTML xüsusiyyətləri də daxildir.


## Qısa tarix ##

The HTA was first introduced in 1999 by Microsoft along with the release of Internet Explorer 5. O, Internet Explorer ilə uyğun gəlirdi və buna görə də yalnız Windows əməliyyat sistemində icra oluna bilərdi. Bu texnologiya 2003-cü ildə patentləşdirilmişdir. HTA faylları digər .exe fayllarına bənzər şəkildə icra olunur. HTA faylları Windows 11-in bugünkü yenilənmiş versiyası ilə də uyğun gəlir.


## Texniki Spesifikasiya ##

HTA-lar hər hansı digər HTML səhifəsi kimi eyni formata malikdir, bəzi atributlar isə proqramların sərhədlərinin və ya nişanlarının üslublarını idarə etmək üçün istifadə olunur. Bundan əlavə, HTA-nın işə salınması üçün arqumentlər verilir. Bu proqramlar mshta.exe adlı proqramdan istifadə etməklə icra oluna bilər. Fayla sadəcə iki dəfə klikləməklə daxil olmaq olar. Bu proqramlar avtomatik olaraq İnternet Explorer ilə birlikdə işləyir. Digər spesifikasiyalarla yanaşı, bunlar Trident mühərrik brauzerindən müstəqil deyil, Internet Explorer-dən müstəqildir. Bu o deməkdir ki, bunlar Internet Explorer istifadə etmədən icra edilə bilər.

Teqlər bu proqramların görünüşünü fərdiləşdirmək üçün istifadə olunur. Microsoft HTML proqramından HTA formatına çevrilmə daha asandır, yəni yalnız genişləndirməni dəyişmək lazımdır. Bildiyimiz kimi, bu proqramlar tam etibarlıdır, ona görə də sadə HTML faylları ilə müqayisədə bunlar daha çox xüsusiyyət və üstünlüklərdən ibarətdir. Mətn redaktorları HTA yaratmaq üçün istifadə edilə bilər. Bu redaktorları Microsoft və ya hər hansı digər etibarlı mənbə əldə edə bilər.


## HTA Fayl Format Nümunəsi ##

```
<HTML>
<HEAD>
<HTA:APPLICATION ID="HelloExample" 
   BORDER="bold" 
   BORDERSTYLE="complex"/>
<TITLE>HTA - Hello World</TITLE>
</HEAD>
<BODY>
<H2>HTA - Hello World</H2>
</BODY>
</HTML>

```
