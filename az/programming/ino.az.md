## INO faylı nədir?

**INO faylı** **Arduino proqramlaşdırma dilindən** istifadə edərək Arduino lövhələri üçün yazılmış proqram olan **Arduino Sketch** ilə bağlıdır. Arduino proqramlaşdırma dili Wiring-ə əsaslanır; və C/C++ ilə oxşardır. Arduino eskizləri .ino fayl uzantısı ilə saxlanılır və Arduino dövrə lövhələrini idarə etmək üçün istifadə olunur.

## INO Fayl Format - Ətraflı Məlumat

Elektron prototipləmə üçün nəzərdə tutulmuş Arduino platforması tərtibatçılara eskizlər adlanan proqramları xüsusi dövrə lövhələrində hazırlamaq və yerləşdirmək imkanı verir; bu eskizlər elektron lövhədə LED-in işıqlandırılması kimi fundamental vəzifələrdən tutmuş internet üzərindən məlumatların ötürülməsi kimi mürəkkəb əməliyyatlara qədər dəyişir; Arduino eskizlərinin yaradılması **Arduino Integrated Development Environment (IDE)** daxilində baş verir və bu eskizlər .ino fayl uzantısı ilə saxlanılır.

Arduino 1.0-dan əvvəlki dövrdə eskizlər Arduino IDE tərəfindən .PDE fayl uzantısı ilə saxlanılırdı. Baxmayaraq ki, IDE hələ də .PDE fayllarının açılmasını dəstəkləsə də, tövsiyə olunan təcrübə, ilkin olaraq [.PDE](/programming/pde/) faylları kimi saxlananlar da daxil olmaqla, bütün eskizləri .ino fayl uzantısı ilə saxlamaqdır. Bu dəyişiklik platformanın təkamülünü əks etdirir və mövcud standartlara uyğunluğu təmin edir.

## Arduino haqqında

Arduino interaktiv layihələrin prototipini və inkişafını asanlaşdıran açıq mənbəli elektronika platformasıdır; həm aparat, həm də proqram elementlərindən ibarət olan Arduino, təcrübəsizlərdən tutmuş peşəkarlara qədər müxtəlif istifadəçi bazasına xidmət edir. Platformanın əsasını rəqəmsal və analoq sancaqlar, USB bağlantısı və Atmel AVR və ya ARM kimi prosessorlara əsaslanan müxtəlif modellərlə təchiz olunmuş Arduino lövhələri təşkil edir.

Arduino IDE proqramların (eskizlərin) kodlaşdırılması, tərtib edilməsi və lövhələrə yüklənməsi üçün istifadəçi dostu interfeys kimi xidmət edir; C/C++ dilinə bənzər proqramlaşdırma dili yeni başlayanlar üçün mürəkkəblikləri aradan qaldıraraq əlçatanlıq üçün nəzərdə tutulmuşdur. Arduino kitabxanaları səmərəli inkişafı asanlaşdıran sensor inteqrasiyasından tutmuş rabitə protokollarına qədər müxtəlif tapşırıqlar üçün əvvəlcədən yazılmış kod parçaları təqdim edir.

Arduino qalxanları, əlavə lövhələr, Ethernet bağlantısı və ya motor nəzarəti kimi funksiyaları təklif edərək funksiyaları genişləndirir. Canlı Arduino icması forumlar, dərs vəsaitləri və birgə layihələr vasitəsilə geniş bilik bazasına töhfə verir; robototexnika, ev avtomatlaşdırılması, incəsənət qurğuları və sensor əsaslı sistemləri əhatə edən tətbiqlərlə, Arduino-nun çox yönlü və sadəliyi onu müxtəlif elektron layihələr üçün üstünlük təşkil edən seçim edir.

## INO faylını necə açmaq olar?

**Arduino Integrated Development Environment** INO faylını açmaq üçün istifadə edilə bilər. Lakin onlar düz mətn faylları olduğundan onun məzmununu açmaq və ya redaktə etmək üçün istənilən mətn redaktoru istifadə edilə bilər.
