## MDS faylı nədir?

MDS (Media Descriptor Sidecar File) Alcohol 120% və Daemon Tools proqram təminatı ilə əlaqəli fayldır, hər ikisi virtual CD və DVD disklərini yaratmaq və idarə etmək üçün istifadə olunur. MDS faylı bütün faylların yeri və diskin strukturu daxil olmaqla, diskdəki məlumatların düzülüşü haqqında məlumatları ehtiva edir. Bu, virtual diskə fiziki diskin davranışını təqlid etməyə imkan verir ki, proqram virtual diskdəki məlumatları real disk kimi oxuya bilsin.

When you want to mount an image file to a virtual drive, the MDS file is used along with the corresponding image file in a format such as ISO, BIN or CUE. The MDS file contains a description of the layout of the data on the disc, and the virtual drive software uses this information to create a virtual disc. This allows you to use the software as if you were using a physical disc, without having to physically insert the disc into the drive.
