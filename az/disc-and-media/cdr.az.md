## CDR faylı nədir?

macOS-da **.CDR** faylı adətən **Disk Utility** proqramından istifadə etməklə yaradılmış disk şəklidir. Bu fayllar tez-tez **Macintosh DVD/CD Master** faylları adlanır. Onlar fiziki diskə yazıla və ya virtual sürücü kimi quraşdırıla bilən CD və ya DVD-nin məzmununun şəklini ehtiva edir. Bu .cdr faylları çox vaxt optik disklərin ehtiyat nüsxələrinin yaradılması, disk şəkillərinin paylaşılması və ya proqram təminatının yayılması üçün istifadə olunur.

## CDR faylı ilə işləmək

Mac-da .cdr faylları ilə necə işləyə bilərsiniz:

1.  **Montaj**: Siz .cdr faylını virtual disk kimi quraşdırmaq üçün üzərinə iki dəfə klikləyə bilərsiniz. Bu, fiziki diskə ehtiyac duymadan onun məzmununa daxil olmağa imkan verir.
    
2.  **Burning**: You can use Disk Utility application to burn a .cdr file back to a CD or DVD. Open Disk Utility, select "File" menu, and choose "Open Disk Image." Then, select .cdr file you want to burn and click "Burn" to create a new disc.
    
3.  **Converting**: Siz həmçinin Disk Utility-də Convert seçimindən istifadə edərək .cdr faylını .iso kimi başqa disk təsvir formatına çevirə bilərsiniz. Qeyri-Mac sistemlərində təsvirdən istifadə etməlisinizsə, bu faydalı ola bilər.

## CDR haqqında ətraflı məlumat

.cdr faylları və Macintosh DVD/CD Masters ilə işləyərkən bəzi əlavə məlumatları və ən yaxşı təcrübələri bilmək vacibdir:

1.  **Fayl Yedekləmələri**: Həmişə .cdr fayllarınızın ehtiyat nüsxələrini saxlayın. Orijinal mənbə CD/DVD və ya .cdr faylını itirsəniz, məlumatlara girişi itirə bilərsiniz.
    
2.  **.ISO-ya çevirmə**: Çarpaz platforma uyğunluğunu təmin etmək üçün .cdr fayllarınızı Windows və ya Linux istifadəçiləri ilə paylaşmaq lazımdırsa, onları daha universal .iso (ISO 9660) formatına çevirməyi düşünün. Bunu macOS-un Disk Utility və ya üçüncü tərəf alətlərindən istifadə edərək edə bilərsiniz.
    
3.  **Fayl Ölçüsü**: .cdr faylları təmsil etdikləri CD/DVD-nin məzmunundan asılı olaraq böyük ola bilər. Bu fayllarla işləyərkən kifayət qədər yaddaş yeriniz olduğundan əmin olun.
    
4.  **Yoxlama məbləğləri**: .cdr fayllarını paylaşarkən, faylın bütövlüyünü yoxlamaq üçün MD5 və ya SHA-256 kimi yoxlama məbləğlərini daxil etməyi düşünün. Bu, köçürmə zamanı faylın zədələnməməsini təmin etmək üçün vacibdir.
    
5.  **Yanma sürəti**: .cdr fayllarını yenidən fiziki mediaya yazarkən, yanma sürətinə diqqət yetirin. Bəzən daha yavaş yazma sürəti xüsusilə köhnə və ya aşağı keyfiyyətli CD/DVD mediası üçün daha etibarlı nüsxələrə səbəb ola bilər.
    
6.  **Etiketləmə**: Əgər nüsxələri yaratsanız, fiziki disklərinizi düzgün şəkildə etiketləyin. Bu, məzmunu asanlıqla müəyyən etməyə kömək edir.
    
7.  **macOS-da Disk Utiliti**: macOS-da Disk Utility proqramı .cdr faylları ilə işləmək üçün əsas vasitədir. Siz bu yardım proqramından istifadə edərək bu şəkilləri yarada, aça, çevirə və yandıra bilərsiniz.
    
8.  **İcazələr**: .cdr fayllarına daxil olmaq və onları dəyişdirmək üçün lazımi icazələrə malik olduğunuzdan əmin olun. İcazə problemi ilə qarşılaşsanız, fayl icazələrini tənzimləməli ola bilərsiniz.
    
9.  **Virtual Maşınlar**: VMware və ya VirtualBox kimi virtual maşın proqram təminatı ilə .cdr fayllarını virtual mühitdə istifadə üçün virtual CD/DVD diskləri kimi quraşdırmaq üçün istifadə edə bilərsiniz.
    
10.  **Arxiv və Təşkilat**: .cdr fayllarınızı yaxşı təşkil edin. Onları düzgün adlandırmaq və kateqoriyalara ayırmaq, xüsusi disk şəkillərinə daxil olmaq və ya paylaşmaq lazım olanda vaxtınıza qənaət edə bilər.
    

**.CDR** faylları macOS-a xasdır və qeyri-Apple sistemləri ilə birbaşa uyğun olmaya bilər. Onları .iso kimi daha standart formata çevirmək çox vaxt müxtəlif platformalarda məlumat paylaşarkən və ya onlarla işləyərkən yaxşı təcrübədir.
