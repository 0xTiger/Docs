## LBR faylı nədir?

.lbr uzantılı fayl LU proqramı ilə yaradılmış və 1980-ci illərdə CP/M və DOS əməliyyat sistemlərində istifadə edilən arxiv faylıdır. Artıq istifadə edilmir və müasir arxiv fayl formatları ilə əvəz edilmişdir. Format üzv faylları sıxışdırmır və onlar üçün konteyner arxivi kimi çıxış edir. Arxivləşdirmə funksiyası əsasən arxivləşdirilmiş faylların internet üzərindən ötürülməsi üçün istifadə olunurdu. LBR sıxılma təklif etmədiyindən, SQ və ya CRUNCH kimi digər yardım proqramları, ölçüsünü azaltmaq üçün üzv faylları əvvəlcədən arxivləşdirmədən və ya nəticədə arxiv faylını arxivləşdirmədən sonra sıxışdırmaq üçün istifadə edilmişdir.

## LBR fayl formatı

LBR file format was designed by Gary P. Novosielski and has no technical details available publicly. LBR files start with a 0x00 byte, followed by 11 spaces (0x20), then two 0x00 bytes, then two bytes that are not both 0x00. Konteyner fayl formatı olmaqla, LU.COM və NULU.COM istifadə edərək çıxarıla bilər.
