## Linux PUP faylı nədir?

Linux PUP faylı Puppy Linux əməliyyat sistemi tərəfindən istifadə edilən quraşdırma paketidir. Quraşdırıcı kimi işləyir və paket fayllarını quraşdırmağa başlayır. Linux-da o, tez-tez DotPup faylı adlanır və Windows ƏS-də istifadə edilən [.exe](/executable/exe/) və [.msi](/executable/msi/) quraşdırma fayllarına bənzəyir. Puppy Linux-un daha yeni versiyaları PUP fayl formatı əvəzinə .pet fayl formatından istifadə edir.
