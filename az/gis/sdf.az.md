## SDF faylı nədir?

A Spatial Data File (SDF) is a Geodatabase file format that was developed by Autodesk. It stores geospatial data and is used by Autodesk GIS program MapGuide and AutoCAD Map 3D. SDF file format is based on the single file database file format SQLite3. Bu, məkan məlumatlarının böyük verilənlər bazası üçün optimallaşdırılmış fayl formatı hesab olunur.

Siz Autodesk AutoCAD Map 3D 2022 və Autodesk AutoCAD Civil 3D 2023 proqramlarından istifadə edərək SDF faylı aça bilərsiniz.

## SDF Fayl Format - Ətraflı Məlumat

SDF fayl formatı ikili fayllar kimi diskdə saxlanılır. O, verilənlərin ikili seriallaşdırılması üçün aşağı səviyyəli saxlama komponentlərindən istifadə edən SQLite verilənlər bazasına əsaslanır. SDF faylı hər bir xüsusiyyət sinfi üçün çoxlu xüsusiyyət sinifləri və hər bir xüsusiyyət sinfi üçün çoxlu həndəsə xüsusiyyətləri ehtiva edir. SDF faylındakı məlumatlar R-ağacından istifadə edərək hər bir həndəsə xüsusiyyətinin indeksləşdirilməsi səbəbindən yüksək sürətlə əldə edilə bilər.
