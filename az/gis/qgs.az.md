## QGS faylı nədir?

A file with .qgs extension is a project file created with [QGIS](https://www.qgis.org/en/site/), which is an open-source cross-platform GIS application. All the project settings such as layer properties and auxiliary data for the project. It is created when a map project is saved to disc. It is actually a configuration file that retains information such as pointers to the GIS data, styling information about the layers, and project title. QGS files can be opened with QGIS software that is freely available to download.

## QGS Fayl Formatı

QGS faylları [XML](/web/xml/) formatındadır və layihələr məlumatlarını XML teqləri kimi saxlayır. QGS faylı aşağıdakı kimi məlumatları ehtiva edir:

 * layihə adı
 * layihə CRS
 * təbəqə ağacı
 * snapping parametrləri
 * münasibətlər
 * xəritə kətan ölçüsü
 * layihə modelləri
 * əfsanə
 * mapview dokları (2D və 3D)
 * əsas verilənlər toplusuna (məlumat mənbələri) və miqyas, SRS, birləşmələr, üslublar, renderer, qarışıq rejimi, qeyri-şəffaflıq və s. daxil olmaqla digər təbəqə xüsusiyyətlərinə keçidləri olan təbəqələr.
 * layihə xassələri
