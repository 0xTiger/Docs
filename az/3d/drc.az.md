## DRC faylı nədir?

.drc uzantısı olan fayl Google Draco kitabxanası ilə yaradılmış sıxılmış 3D fayl formatıdır. Google Draco-nu 3D həndəsi şəbəkələri və nöqtə buludlarını sıxmaq və açmaq üçün açıq mənbə kitabxanası kimi təklif edir və 3D qrafikaların saxlanmasını və ötürülməsini yaxşılaşdırır. O, daxil edilmiş məlumatı kodlayır və onu .drc faylı kimi saxlayır. Qəbul edən tərəfdə API .drc fayllarını oxuyur və onları [PLY](/3d/ply/) və ya [OBJ](/3d/obj/) faylları kimi çıxara bilər. Sıxılmış çıxış faylı istifadəçilərə proqramları daha sürətli yükləməyə və brauzerlərdə 3D qrafikaların tez yüklənməsini təmin etməyə imkan verir.

## DRC fayl formatı

File format details for DRC are not available publicly. Google has hosted the Draco library on [GitHub](https://github.com/google/draco/tree/master/docs) for developer's reference. It provides encoders and decoders for creating and converting DRC files.
