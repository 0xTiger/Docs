## PPS faylı nədir?

PPS, PowerPoint Slide Show, files are created using Microsoft PowerPoint for Slide Show purpose. PPS file reading and creation is supported by Microsoft PowerPoint 97-2003. Bu fayl formatının ən son versiyası Office OpenXML standartlarına əsaslanan [PPSX ](/presentation/ppsx/) versiyasıdır. PPS faylları hələ də Microsoft PowerPoint-in ən son versiyaları tərəfindən oxuna bilər, lakin yeni yaradılmış fayllar yalnız PPSX fayl formatında saxlanıla bilər. PPS faylı başqa istifadəçi ilə paylaşıldıqda və açıldıqda, redaktə edilə bilən rejimdə açılan [PPT](/presentation/ppt/) faylından fərqli olaraq Powerpoint şousu kimi başlayır.

## Fayl Format ##

PPS, [MS-PPT](https://msdn.microsoft.com/en-us/library/office/cc313106(v#office.12).aspx) fayl formatı sənədində təfərrüatlı olan spesifikasiyalara uyğun ikili fayl formatıdır. O, ümumi sənəd strukturunu və məlumatları təmsil etmək üçün bir sıra axınlardan ibarətdir. Bunlara daxildir:

* Cari İstifadəçi axını

* PowerPoint sənəd axını

* Şəkillər axını

* Xülasə Məlumat və Sənəd Xülasə Məlumatı (İstəyə görə)


PPT fayl formatı ilə bağlı tam təfərrüatları PPS-ə də eyni dərəcədə tətbiq olunan MS-PPT spesifikasiyalar sənədində tapa bilərsiniz.
