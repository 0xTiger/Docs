## POT faylı nədir?

.POT uzantısı olan fayllar PowerPoint 97-2003 versiyaları tərəfindən yaradılmış Microsoft PowerPoint şablon fayllarını təmsil edir. Microsoft PowerPoint-in bu versiyaları ilə yaradılmış fayllar PowerPoint-in daha yüksək versiyalarından istifadə edərək Office OpenXML fayl formatlarında yaradılanlarla müqayisədə ikili formatdadır. Beləliklə, yaradılan fayllar eyni tərtibata və yeni fayllara tətbiq edilmək üçün tələb olunan digər parametrlərə malik təqdimatlar yaratmaq üçün istifadə edilə bilər. Bu parametrlərə üslublar, fonlar, rəng palitrası, şriftlər və standart parametrlər daxil ola bilər. Bu cür fayllar rəsmi istifadə üçün istifadəyə hazır şablon faylları yaratmaq məqsədilə yaradılır.

## Fayl Formatının Xüsusiyyətləri ##

File format specifications for POT file format are not available individually by Microsoft. However, the template format is the same as that of the binary [PPT](/presentation/ppt/) file format and can be opened in MS PowerPoint for editing purpose. A POT file is identified by HEX identifier as follow:

```
D0 CF 11 E0 A1 B1 1A E1 00 00 00 00
```

POT fayl formatı ilə əlaqəli MIME növləri bunlardır:

* application/vnd.ms-powerpoint [rəsmi]

* proqram/mspowerpoint

* proqram/x-mspowerpoint

* proqram/powerpoint

* proqram/x-powerpoint

* proqram/x-dos_ms_powerpnt

* proqram/pot

* proqram/x-soffic
