You must be familiar with **PPTX** and **PPT** extension files, these are Presentation file formats that store collection of records to accommodate presentation data such as: slides, shapes, text, animations, video, audio and embedded objects. A presentation can be saved/converted into other file formats as well such as PDF, BMP, PNG, JPEG, and XPS.

Ümumi Təqdimat fayl uzantıları və onlarla əlaqəli fayl formatlarına **[PPTX](/presentation/pptx/)**, **[PPT](/presentation/ppt/)** və **[ODP](/presentation/odp/)** daxildir.

Təqdimat fayl formatları ilə bağlı sorğularınız var? Fayl Format mütəxəssisləri tərəfindən paylaşılan biliklərdən faydalanmaq üçün [forums](https://forum.fileformat.com/c/presentation/8) icmamıza daxil olun.

## Təqdimat Faylları üçün Açıq Mənbə API

[Open Source APIs for working with Presentation File Formats](https://products.fileformat.com/presentation/) siyahısına baxın.
