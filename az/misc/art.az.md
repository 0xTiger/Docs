## ART faylı nədir?

An ART file is an embroidery design file created with the [BERNINA](https://www.bernina.com/en-US/Home-United-States) ARTlink embroidery software version 1-4. O, konturlar, dizayn tərtibatı, tikişlər, rəngli iplər və digər metadata kimi müxtəlif obyektləri özündə birləşdirə bilən istifadəçi tərəfindən yaradılmış dizaynları ehtiva edir. ART faylı BERNINA tikiş maşınına yüklənir və yorğan, köynək və ya hər hansı digər parça kimi müxtəlif növ parçalara tikilə bilər. ART faylları həmçinin USB kimi saxlama cihazlarından istifadə edərək digər istifadəçilərlə paylaşıla və ya internet vasitəsilə ötürülə bilər.

## ART fayl formatı

ART fayl formatı aşağıdakı məlumatları ehtiva edir:

 * Obyekt xassələri
 * Dikiş məlumatları
 * Dikiş növləri
 * Dikiş koordinatları
 * Mövzu Rəngləri
 * Şərhlər
 * Kompensasiya Parametrlərini çəkin
 * Avtomatik Aralıq Parametrləri
