## SSP faylı nədir?

SSP faylı Axialis Software tərəfindən Screensaver Producer kimi tanınan Windows ekran qoruyucularının yaradılması proqramı tərəfindən yaradılmış layihə faylıdır. O, spritlər, Flash filmlər, foto slayd şouları və videolar kimi bir neçə müxtəlif tipli ekran qoruyucu elementləri ehtiva edə bilər. SSP faylı mətn, şəkillər, müxtəlif formalar və SSP faylı daxilində animasiyanı göstərmək üçün istifadə edilən animasiya parametrlərindən ibarət ola bilər. SSP faylları Axialis Screensaver Producer ilə açıla bilər.

## SSP Fayl Format - Ətraflı Məlumat

SSP files are stored to disc as binary files with no public information available about their internal file format structure. Axialis Screensaver producer professional screensaver authoring tool for Windows which lets you easily create hi-quality screensavers for all versions of Windows up to Windows 10. Bu, istifadəçilərə onlayn olaraq asanlıqla paylana bilən quraşdırıla bilən ekran qoruyucu paketləri yaratmağa imkan verir.

SSP faylları [SCR](/system/scr/) fayl formatına ixrac edilə bilər.
