## OGG faylı nədir?

OGG faylı Origin proqramı ilə yaradılmış qrafik fayldır. Məlumatların təhlili üçün idxal edilmiş məlumatları göstərmək üçün müxtəlif növ qrafikləri saxlamaq imkanı verir. OGG fayl növü tərəfindən dəstəklənən qrafik növlərinə sütun, bar, pasta, şəlalə, 3D, qütb və simvol daxildir. Origin proqramı OriginLab tərəfindən hazırlanmışdır və istifadəçilərə məlumatların təhlilini həyata keçirməyə və qrafiklər kimi çıxışlar yaratmağa imkan verir.

## OGG fayl formatı

OGG fayllarında idxal edilmiş məlumatların təsviri üçün qrafiklər var. OGG faylı açıldıqda, Origin proqram təminatı ilə müqayisədə sərbəst mövcud olan Origin və ya Origin Viewer-də qrafiki ayrıca pəncərədə göstərir. Qrafikin başlığı və rəngləri kimi metadata məlumatı da OGG faylında saxlanılır.

## OriginLab Origin proqramı haqqında

Origin proqramı həmçinin Avtomatlaşdırma Serveri kimi istifadə etmək imkanı verir. Bu halda, müştəri proqramları, məsələn, LabVIEW, Microsoft Excel, Python və ya C#/VB/VC-də xüsusi hazırlanmış proqramlar Origin ilə məlumat mübadiləsi edə bilər. [Python_to_OriginLab](https://github.com/jsbangsund/python_to_originlab) OriginLab-da verilənlərin göndərilməsini və əsas plan qurma funksiyalarını avtomatlaşdırmaq üçün Python API-dir.
