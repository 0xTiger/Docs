## BPS faylı nədir?

.BPS faylı əsasən sənədləri üçün .WPS fayl uzantısından istifadə edən Microsoft Works ilə yaradılmış mətn emal sənədinin ehtiyat nüsxəsidir. Bu .BPS faylları tez-tez sənəd üzərində işləyərkən proqramın qəzaya uğraması halında ehtiyat tədbiri olaraq Microsoft Works tərəfindən avtomatik olaraq yaradılır. Bu, işinizi qorumaq və məlumat itkisinin qarşısını almaq üçün bir yoldur.

Növbəti bölmədə biz BPS faylları ilə əlaqəli proqram proqramlarını araşdıracağıq.

## Microsoft işləyir

Microsoft Works, Microsoft tərəfindən hazırlanmış məhsuldarlıq proqram dəsti idi. İlk dəfə 1987-ci ildə təqdim edildi və tam Microsoft Office paketinə daha sərfəli alternativ kimi populyarlıq qazandı. Microsoft Works-ə bir neçə əsas ofis proqramları daxildir, məsələn:

- **Word Processor:** Microsoft Works-də istifadəçilərə sənədlər yaratmağa və redaktə etməyə imkan verən mətn emal proqramı var idi. O, mətn emal sənədləri üçün [.WPS](/word-processing/wps/) fayl formatından istifadə edir.

- **Spreadsheet:** Buraya elektron cədvəllərin yaradılması və idarə edilməsi üçün elektron cədvəl proqramı daxildir. Cədvəllər üçün fayl formatı [.XLR](/spreadsheet/xlr/) idi.

- **Verilənlər bazası:** Microsoft Works-də məlumatların təşkili və idarə edilməsi üçün verilənlər bazası proqramı da var idi. Verilənlər bazası faylları üçün fayl formatı [.WDB](/database/wdb/) idi.

- **Təqvim və Ünvan Kitabı:** Əsas ofis proqramlarına əlavə olaraq, Microsoft Works tez-tez təqvimləri və ünvan kitablarını idarə etmək üçün alətləri ehtiva edir.

- **Works Word Processor Format (.WPS):** Microsoft Works-in mətn emal komponentinin yerli fayl formatı. .WPS fayllarında mətn, formatlama və sadə qrafiklər var idi.

- **Xatırlatma:** Microsoft 2007-ci ildə Microsoft Works-i rəsmi olaraq dayandırdı və istifadəçiləri Microsoft Office və ya digər alternativlərə keçməyə təşviq etdi.
