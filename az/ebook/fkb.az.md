## FKB faylı nədir? ##

FKB, Flipkart proqram tətbiqi üçün Flipkart.com tərəfindən hazırlanmış elektron kitab fayl uzantısıdır. Bu tip fayl uzantısı adətən Windows 10 istifadəçilərində tapılır. FBK faylında, kitabın uzunluğundan və daxil edilən şəkillərin sayından asılı olaraq ölçüsü bir neçə yüz kilobaytdan çoxsaylı meqabayta qədər dəyişə bilən bir spesifikasiya var. Faylı Flipkart eBooks Android Tətbiqindən yükləmək üçün əvvəlcə elektron kitabı satın almalı, sonra Kitabxanada onun üzərinə klikləyin və kitab endirməyə başlayacaq. Satın alınan sənədi görə bilmirsinizsə, daha yaxşı işləmək üçün sistemi yeniləyin.

## Dəstəklənən Əməliyyat Sistemləri ##

Aşağıdakı əməliyyat sistemləri FKB fayllarını dəstəkləyir:

* Windows 7
* Windows 8
* Windows 10
* Windows Server 2003/2008/2012/2016
* Mac OS X
* FreeBSD
* iOS
* Linux
* Android
* NetBSD
* OpenBSD

## FKB faylını açmaqla bağlı problemlər ##

* Dəstəkləyici proqramların düzgün qurulmaması. Onun yenilənmiş versiya ilə düzgün yükləndiyinə əmin olun
* Aşağı hardware sahəsi
* Korlanmış və ya yoluxmuş fayllar. Bu problem sisteminizdə quraşdırılmış antivirus vasitəsilə asanlıqla həll edilə bilər
