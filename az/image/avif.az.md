## AVIF faylı nədir? ##

AVIF (AV1 Image File Format) AV1 ilə sıxılmış şəkilləri HEIF fayl formatında saxlayan şəkil faylı formatıdır. AVIF faylları .avif genişlənməsi ilə saxlanılırn. Version 1 of the AVIF was finalized in February 2019. Yüksək dinamik diapazon (HDR), 8, 10 və 12 rəng dərinliyi dəstəyi, istənilən rəng məkanı üçün dəstək (ISO/IEC CICP və ICC profilləri, geniş rəng gamutu) və s. kimi xüsusiyyətlərə malikdir.

## AVIF dəstəyi ##

The support for the AVIF file format is growing. Netflix published the first AVIF images on December 14, 2018. Microsoft Windows 10 da 19H1 buraxılışı ilə ona dəstək əlavə etdi. Paint.net müvafiq olaraq 2019-cu ilin sentyabrında və 2020-ci ilin avqustunda AVIF fayllarının açılması və saxlanması üçün dəstək əlavə etdi. Brauzerlərə AVIF dəstəyi də əlavə edilir. Google Chrome 2020-ci ilin avqust ayında AVIF üçün dəstək əlavə etdi və Mozilla Firefox onu dəstəkləmək üçün çalışır. Cloudflare 3 dekabr 2020-ci ildə AVIF-ə dəstəyini açıqladı.

## AVIF vs JPEG ##

AVIF-in JPEG-dən daha böyük qazancı kiçik fayl ölçüsüdür. Bu o deməkdir ki, AVIF-dən istifadə daha az bant genişliyi və məlumat istifadəsi ilə nəticələnəcək. AVIF formatı HDR-ni dəstəkləyir, JPEG isə 8 bitlik rəng dərinliyi ilə məhdudlaşır.

Hazırda AVIF ilə bağlı ən böyük iki problem kodlaşdırma sürəti və mütərəqqi göstərmənin olmamasıdır. Beləliklə, JPEG şəkillərindən fərqli olaraq, AVIF şəkilləri tam endirilənə qədər ekranda heç nə göstərilmir.
