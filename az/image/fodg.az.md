## FODG faylı nədir?

.fodg uzantılı fayl rəsm elementlərini saxlamaq üçün Apache OpenOffice Drawing fayl formatıdır. O, Advancement of Structural Information Standards (OASIS) tərəfindən qeyd olunan fayl formatı spesifikasiyasına əsaslanır. OpenOffice qrafikası üçün digər oxşar fayl formatı rəsm elementlərini vektor şəkli kimi saxlayan [ODG](/image/odg/) formatıdır. FODG faylları OpenOffice və LibreOffice ilə açıla bilər. OpenOffice tərəfindən dəstəklənən digər fayl formatlarına, məsələn, [ODT](/word-processing/odt/), ODF, [ODP](/presentation/odp/) və [ODS](/spreadsheet/ods/) daxildir.

## FODG fayl formatı

FODG is based on the OpenDocument's XML file format that conforms to OASIS OpenDocument Format ISO/IEC 26300. Onun İnternet Media Növü proqramı/vnd.oasis.opendocument.graphics var və həmçinin org.oasis-open.opendocument public.composite-content ilə uyğun gəlir. XML strukturu elektron cədvəl, rəsm faylları və mətn sənədləri kimi bütün sənəd növləri üçün ümumidir. OpenOffice sənədləri məzmunu, üslubları, metadataları və tətbiq parametrlərini dörd ayrı XML faylına ayırmaqla narahatlıqların ayrılmasından istifadə edir.

`<office:document-content> ` - Sənəd məzmunu və məzmunda istifadə olunan avtomatik üslublar.
`<office:document-styles> ` - Sənədin məzmununda istifadə olunan üslublar və üslubların özlərində istifadə olunan avtomatik üslublar.
`<office:document-meta>` - Document meta information, such as the author or the time of the last save action.
`<office:document-settings> ` - Pəncərə ölçüsü və ya printer məlumatı kimi proqrama xas parametrlər.
