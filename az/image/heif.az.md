## HEIF faylı nədir?

An HEIF file is a High-Efficiency Container Image file format that is capable of storing a single image or a sequence of images in a single file. It, thus, is basically a container having multiple images stored. HEIF file format compresses the images using the High Efficiency Video Coding (HEVC) standard. It is is more efficient and light weight as compared other image file formats such as [JPEG](/image/jpeg/). A single HEIF file can store sequences of images such as [GIF](/image/gif/) animations or HDR sequence, along with metadata that describes each image. Apple Inc. adopted its variant HEIC file format for their iOS 11. Bu, digər şirkətlərin Adobe daxil olmaqla proqram təminatında onu dəstəkləməsinə səbəb olur.

## HEIF fayl formatı

The HEIF format was developed by Moving Picture Experts Group (MPEG) and was registered under the ISO/IEC 23008-12 standard in 2015. Apple, Microsoft və Canon kimi bir sıra istehsalçılar tərəfindən qəbul edilmişdir. HEIF faylları .heif uzantısı ilə (əgər o tək şəkil ehtiva edirsə) və ya .heifs (şəkillər ardıcıllığını ehtiva edirsə) kimi saxlanılır.

### HEIF Data Saxlama

HEIF formatı aşağıdakı şəkillər və məlumat növlərini saxlaya bilər.

 * Fərdi şəkillər və şəkil miniatürləri
 * Şəkillərin ardıcıllığı
 * Post-emal təlimatları
 * [EXIF](/image/exif/) datası kimi şəkil metadatası

HEIF şəkil formatı şəkil keyfiyyətinə heç bir güzəştə getmədən [JPEG](/image/jpeg/), [PNG](/image/png/) və [GIF](/image/gif/) daxil olmaqla bir neçə fərqli şəkil formatının imkanlarını təmin edir.
