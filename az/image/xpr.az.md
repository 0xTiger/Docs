## XPR faylı nədir?

XPR faylı hazırda fəaliyyəti dayandırılmış Microsoft Expression Graphics (EGD) proqramı ilə yaradılmış vektor təsvir faylıdır. O, istifadəçi interfeysi maketi kimi istifadə edilə bilən qrafik məlumatları ehtiva edir. XPR faylları daha sonra Microsoft Expression Design-da .design faylları ilə əvəz olundu. XPR faylları artıq bir müddətdir dayandırılmış Microsoft Expression Studio ilə açıla bilər.

## XPR Fayl Formatında Zəiflik

XPR faylları uzaqdan kodun icrasına imkan verən Microsoft Expression Design-da boşluqdan faydalanmaq üçün müəyyən edilib. Məlumat verilmiş məsələdə, əgər istifadəçi dinamik keçid kitabxanası (DLL) faylı ilə birlikdə şəbəkə kataloqunda yerləşdirilmiş qanuni .xpr faylını açsa, Microsoft Expression Design DLL faylını yükləməyə və onun ehtiva etdiyi kodu icra etməyə cəhd edə bilər. Bu, daha sonra Microsoft tərəfindən təhlükəsizlik yeniləməsində düzəldildi.
