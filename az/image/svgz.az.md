## SVGZ faylı nədir?

.svgz uzantısı olan fayl Scalable Vector Graphics (.SVG) faylının sıxılmış versiyasıdır. O, gzip sıxılması ilə sıxılır və [XML](/web/xml/) formatında data ehtiva edir. SVGZ faylları şəffaflığı, gradientləri, animasiyaları və filtrləri dəstəkləyir. SVGZ faylları standart SVG faylları ilə müqayisədə daha kiçik ölçülüdür və bu azaldılmış fayl ölçüsü qrafik faylları onlayn ötürməyə kömək edir. Qrafik dizayner Adobe Illustrator, Corel PaintShop Pro və digər vasitələrdən istifadə edərək SVGZ faylları yaradır. Bununla belə, SVGZ faylları şəkil məlumatlarını göndərərkən Apache Serverində [GZip compression](/compression/gz/) funksiyasını aktivləşdirməklə yaradıla bilər.

## SVGZ fayl formatı

SVGZ, being compressed [SVG](/page-description-language/svg/), is based on open compressed gzip format that reduces the size of SVG documents. This compressed archive contains plain text SVG documents inside that are purely based on the XML file format (/web/xml/). SVGZ files can be opened by viewers/applications that conform to the SVG 1.1 specifications. SVGZ files 20-50% smaller in size than the original SVG files.
