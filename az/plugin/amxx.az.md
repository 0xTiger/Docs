## AMXX faylı nədir?

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. Bu, server administratorlarına öz serverlərinin funksionallığını fərdiləşdirməyə və genişləndirməyə imkan verən skript platformasıdır. AMXX MetaMod plagin idarəetmə sistemi ilə birlikdə istifadə olunur ki, bu da birdən çox plaginlərin eyni vaxtda serverdə münaqişəsiz işləməsinə imkan verir. AMXX plagininin təklif etdiyi xüsusiyyətlərə admin əmrləri, oyun modifikasiyası və server idarəçiliyi daxildir.
