## SPARC faylı nədir?

Alternativ olaraq Skype Plugin Arxivi kimi tanınan .sparc faylı Skype tətbiqi üçün əlavə funksiyaları və ya təkmilləşdirmələri əhatə edən paket kimi xidmət edir. Bu plaginlər Zəng qeydi, Ekran paylaşımı, Tərcümə,
Oyunlar və Əyləncə, Fərdi ifadələr və mövzular.

## SPARC Fayl Format - Ətraflı Məlumat

Skype-ın müəyyən versiyalarında Əlavələr Meneceri Skype quraşdırılmasına inteqrasiya oluna bilən seçmə komponentdir. SPARC faylını quraşdırarkən çətinliklərlə qarşılaşsanız, ilkin quraşdırma zamanı Əlavələr Menecerini buraxmısınız. Bu problemi həll etmək üçün Skype-ı yenidən quraşdırın və Əlavələr Meneceri seçimini seçdiyinizə əmin olun.

## Skype plaginini necə quraşdırmaq olar?

Skype plaginlərini quraşdırmaq üçün addımlar:

1. İstədiyiniz plaqinə uyğun .sparc faylını əldə edin.
2. Skype hazırda işləyirsə, onu bağlayın.
3. Yüklənmiş .sparc faylını tapın və üzərinə iki dəfə klikləyin.
4. Plugin quraşdırılmasını başa çatdırmaq üçün ekrandakı təlimatları izləyin.
5. Quraşdırıldıqdan sonra plagini aktivləşdirmək üçün Skype-ı yenidən başladın.
