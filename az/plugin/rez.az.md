## REZ faylı nədir?

.rez fayl uzantısı Ambrosia Software-in Escape Velocity seriyası tərəfindən istifadə edilən plagin faylıdır. Bu fayllar gəmilər, missiyalar və ya qrafika kimi yeni məzmun əlavə etməklə oyunu dəyişdirmək və ya genişləndirmək üçün istifadə olunur. Plugin sistemi seriyanın populyarlığına töhfə verən əsas xüsusiyyətlərdən biridir, çünki bu, oyunçulara oyun təcrübəsini fərdiləşdirməyə və yaradıcılıqlarını başqaları ilə bölüşməyə imkan verirdi.

## REZ Fayl Format - Ətraflı Məlumat

Escape Velocity-də .rez fayllarının necə işlədiyinə dair qısa icmal:

**Plugin Strukturu:** .rez faylları əsasən oyun üçün müxtəlif resurslar və skriptləri saxlayan konteynerlərdir. Bu resurslara qrafika, səslər, gəmi məlumatları, missiya skriptləri və s. daxil ola bilər.

**Fərdiləşdirmə:** Oyunçular və üçüncü tərəf tərtibatçıları oyuna yeni elementlər təqdim etmək üçün öz plaginlərini yarada bilərlər. Bu, yeni növ gəmilər və silahlar əlavə etməkdən tamamilə yeni hekayə xətləri və missiyalar yaratmağa qədər dəyişə bilər.

**Uyğunluq:** Plugin sistemi birdən çox plaqinin bir yerdə mövcud olmasına imkan verir və oyunçular hansını aktivləşdirməyi seçə bilərlər. Bu o deməkdir ki, oyunçular öz seçimlərinə uyğun plaginləri seçərək oyun təcrübələrini uyğunlaşdıra bilərlər.

**İcma Töhfələri:** Escape Velocity seriyası bir çox oyunçunun öz plaginlərini yaradaraq paylaşdığı inkişaf edən modding icmasına malik idi. Bu, oyunların uzunömürlülüyünə və təkrar oynanmasına əhəmiyyətli dərəcədə kömək etdi.

**Installation:** To use a plugin in Escape Velocity, players typically place the **.rez file** in a designated folder within the game directory. The game's launcher or menu system then allows players to activate or deactivate specific plugins.

## Ambrosia Escape Velocity haqqında

Escape Velocity seriyası Ambrosia Software tərəfindən Macintosh kompüterləri üçün hazırlanmışdır. Oyunlar, oyunçuların kəşf edə, ticarət edə və kosmik döyüşlərdə iştirak edə biləcəyi geniş, açıq bir kainatda qurulur. Seriya qeyri-xətti oyunu ilə diqqət çəkir, oyunçulara öz yollarını və sadiqliklərini seçməyə imkan verir.

Orijinal Escape Velocity oyunu 1996-cı ildə buraxıldı və onun ardınca iki seriya çıxdı: Escape Velocity Override (1998) və Escape Velocity Nova (2002). Bu oyunlar immersiv oyunu, geniş modinq icması və oyunçunun qərarları əsasında hekayəni formalaşdırmaq bacarığı ilə populyarlıq qazandı.

**Qeyd:** Ambrosia Escape Velocity və EV Nova dayandırıldı.

## REZ faylını necə açmaq olar?
