## Windows Host Skripti haqqında

Windows Script Host (WSH) Microsoft tərəfindən hazırlanmış skript hostudur və istifadəçilərə VBScript (Visual Basic Scripting Edition) və JScript (Microsoft-un JavaScript versiyası) kimi skript dillərində yazılmış skriptləri işlətməyə imkan verir; o, müxtəlif tapşırıqların avtomatlaşdırılmasına və sistem idarəçiliyinə imkan verən Windows əməliyyat sistemində skriptlər üçün çərçivə təmin edir.

Windows Script Host ilə bağlı bəzi əsas məqamlar bunlardır:

1.  **Skript Dilləri:** WSH çoxlu skript dilini dəstəkləyir, VBScript və JScript ən çox istifadə olunur; skriptlər tapşırıqları avtomatlaşdırmaq, əməliyyat sistemi ilə qarşılıqlı əlaqə yaratmaq və ya məlumatları manipulyasiya etmək üçün bu dillərdən hər hansı birində yazıla bilər.
    
2.  **Skriptin icrası:** WSH skriptləri adətən VBScript üçün .vbs və ya JScript üçün .js kimi uzantıları olan fayllarda saxlanılır; skript işlətdiyiniz zaman WSH kodu şərh edir və icra edir.
    
3.  **Avtomatlaşdırma:** WSH tez-tez təkrarlanan tapşırıqların avtomatlaşdırılması, sistem parametrlərinin idarə edilməsi və inzibati funksiyaların yerinə yetirilməsi kimi avtomatlaşdırma məqsədləri üçün istifadə olunur; sistem administratorları və güc istifadəçiləri üçün xüsusilə faydalıdır.
    
4.  **Konsol və GUI Skriptləri:** WSH skriptləri ya konsol, ya da GUI əsaslı ola bilər; konsol skriptləri komanda xətti pəncərəsində işləyir, GUI skriptləri isə daha interaktiv proqramlar üçün qrafik istifadəçi interfeysləri yarada bilir.
    
5.  **Security:** WSH has security features to help prevent malicious scripts from causing harm; users are typically prompted before executing scripts and scripts are subject to certain restrictions to prevent unauthorized access and actions.
    
6.  **Windows Script Host Object Model:** WSH skriptlərin Windows əməliyyat sistemi ilə qarşılıqlı əlaqədə olmasına imkan verən obyekt modelini təmin edir; bura fayl sistemlərinə, reyestr parametrlərinə, şəbəkə resurslarına və digər sistem komponentlərinə giriş daxildir.
    
7.  **Komanda Xəttindən Skript İcrası:** Skripti konsolda və ya GUI mühitində işə salmaq istəməyinizdən asılı olaraq, `cscript.exe` və ya `wscript.exe` icra olunan fayllarından istifadə edərək WSH skriptlərini komanda xəttindən icra edə bilərsiniz. .
