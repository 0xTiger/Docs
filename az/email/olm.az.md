## OLM faylı nədir?

.olm uzantılı fayl Mac Əməliyyat Sistemi üçün Microsoft Outlook faylıdır. OLM faylı e-poçt mesajlarını, jurnalları, təqvim məlumatlarını və digər növ proqram məlumatlarını saxlayır. Bunlar Windows Əməliyyat Sistemində Outlook tərəfindən istifadə edilən [PST](/email/pst/) fayllarına bənzəyir. Bununla belə, Mac üçün Outlook tərəfindən yaradılmış OLM faylları Windows üçün Outlook-da açıla bilməz. OLM fayl formatı Outlook-un yerli məlumatlarını saxlamaq üçün Outlook 2011 ilə təqdim edilmişdir. Tək bir OLM faylı məlumatları qovluqlar və alt qovluqlar şəklində təşkil edən çoxlu poçt qutularından ibarət ola bilər. OLM faylları Microsoft Outlook 365 və Mac üçün Outlook-da açıla/yüklənə bilər.

## OLM fayl formatı

OLM Tətbiq/oktet axını MIME tipli ikili fayllardır. OLM fayl formatının texniki spesifikasiyası Microsoft-dan açıq şəkildə mövcud deyil və tərtibatçının arayışı üçün heç bir format təfərrüatı yoxdur. Bununla belə, OLM fayl formatını oxuya və Aspose.Email kimi Microsoft Windows Əməliyyat Sistemində istifadə etmək üçün PST faylına yaza bilən bəzi API-lər var.
