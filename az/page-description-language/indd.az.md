## INDD faylı nədir?

.indd uzantısı olan fayl, layihənin səhifə tərtibatı məlumatını ehtiva edən Adobe InDesign layihə faylıdır. INDD InDesign Sənədi deməkdir. Bu fayllar adətən səhifə formatı məlumatlarını, kitabları, jurnalları, qəzetləri, flayerləri, broşuraları və ya mobil və veb səhifə maketləri kimi digər çap mediasını saxlayır. INDD fayl növü Adobe şirkətinin xüsusi fayl formatıdır və InDesign üçün standart fayl formatıdır. Qrafik Dizaynerlər, jurnalistlər, marketoloqlar və InDesign ilə işləyən naşirlər bu fayl növündən istifadə edirlər.

## INDD Fayl Formatı haqqında ətraflı

INDD faylları həllini itirmədən ölçülə bilən vektor fayllarıdır. Bunlar sonsuz şəkildə genişləndirilə bilən mürəkkəb qrafiklərə əsaslanır. O, komanda ilə paylaşılacaq bütün şriftlər və tərtibat məlumatlarını paketləyir.

INDD faylları cəlbedici çap və onlayn məzmunun dizaynı üçün mükəmməldir. INDD faylı bir neçə səhifəyə yayıla bilər və səhifələr, fəsillər və nəşrlər yaratmaq üçün şablon kimi xidmət edə bilər. Layihəniz tamamlandıqda, onu çevirmə, sıxışdırma və ya e-poçt göndərməyə ehtiyac olmadan onilne mediada dərc edə bilərsiniz.

## INDD faylı digər formatlara çevrilə bilərmi?

INDD files can be converted to several other file formats. The most common conversion is INDD to PDF. InDesign lets you save the INDD file to PDF for sharing with other users. It also lets you export INDD file to JPG, [EPS](/page-description-language/eps/), [EPUB](/ebook/epub/), [SWF](/page-description-language/swf/), FLA, [HTML](/web/html/), [XML](/web/xml/), and IDML.

## INDD faylı necə yaradılır?

An Indd file is created when you create a new project with [Adobe InDesign](https://www.adobe.com/products/indesign.html) and save it to disc. You can open the Indd project file, make changes to it, and save the updated file for the changes.
