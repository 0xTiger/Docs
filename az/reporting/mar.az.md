## Access Report (MAR faylı) nədir? ##
Microsoft Access tərəfindən hesabatının qısa yolu kimi yaradılmış fayl .mar fayl uzantısı ilə saxlanılır. **Microsoft Access hesabatı** Microsoft Access verilənlər bazasında məlumatı formatlaşdırmaq, nəzərdən keçirmək və ümumiləşdirmək üçün bir yol təklif edir. Bu hesabatlar sizə məlumatlarınızı ekranda görmək və ya çap etmək üçün dəqiq və məlumatlandırıcı tərtibatda formatlaşdırmağa imkan verir. Hesabat yalnız statik məlumatları göstərir, yəni Access hesabatına baxarkən cədvəllərdəki məlumatları dəyişdirə bilmərik. Hesabat açıldıqda ən son məlumatları göstərir.

## Microsoft Access Report (MAR) fayl formatı

MAR fayl formatı Microsoft Access proqramı tərəfindən qısayol və ya Microsoft Access Hesabatına keçid yaratmaq üçün istifadə olunur ki, bu da verilənlər bazası obyekti olan və aşağıdakı istifadələrdən hər hansı biri üçün verilənlər bazanızda məlumat təqdim etməyiniz lazım olduqda faydalı olur:

- Məlumatların xülasəsini göstərin.
- Məlumatların arxiv şəkilləri.
- Fərdi qeydlər haqqında təfərrüatları göstərin.
- Etiketlər yaradın.

## Giriş Hesabatının hissələri
Giriş hesabatının dizaynı Dizayn görünüşündə baxıla bilən müxtəlif bölmələrə bölünür. Aşağıdakı cədvəl hər bölmənin necə işlədiyini və hesabat yaratmağınıza kömək edə biləcəyini izah edir.
| Bölmə | Bölmə çap edildikdə necə göstərilir | Bölmənin harada istifadə oluna biləcəyi |
---------|----|------|
| Hesabat Başlığı | Hesabatın əvvəlində. | Normalda loqo, başlıq və ya tarix kimi örtük səhifəsində görünə biləcək məlumat üçün hesabat başlığından istifadə edin. Hesablanmış tənzimləməni hesabatın başlığında cəmi toplama funksiyasından istifadə edən yerləşdirdiyiniz zaman hesablanmış məbləğ bütün hesabat üçün olur. Hesabatın başlığı səhifə başlığından əvvəl çap olunur. |
| Səhifə Başlığı | Hər səhifənin yuxarısında. | Hər səhifədə hesabat başlığını təkrarlamaq üçün səhifə başlığından istifadə edin. |
| Qrup Başlığı | Hər yeni qeydlər qrupunun başında. | Qrup adını çap etmək üçün qrup başlığından istifadə edin. Məsələn, məhsula görə qruplaşdırılmış hesabatda məhsulun adını çap etmək üçün qrup başlığından istifadə edin. Qrup başlığında Sum aqreqat funksiyasından istifadə edən hesablanmış nəzarəti yerləşdirdiyiniz zaman cəm cari qrup üçündür. Neçə qruplaşdırma səviyyəsi əlavə etdiyinizdən asılı olaraq hesabatda bir neçə qrup başlıq bölməsi ola bilər. Qrup başlıqları və altbilgiləri yaratmaq haqqında ətraflı məlumat üçün Qruplaşdırma, çeşidləmə və ya cəmi əlavə edin bölməsinə baxın. |
| Ətraflı | Qeyd mənbəyində hər sıra üçün bir dəfə görünür. | Hesabatın əsas hissəsini təşkil edən nəzarətləri burada yerləşdirirsiniz. |
| Qrup Altbilgisi | Qeydlərin hər qrupunun sonunda. | Qrup üçün xülasə məlumatını çap etmək üçün qrup altbilgisindən istifadə edin. Neçə qruplaşdırma səviyyəsi əlavə etdiyinizdən asılı olaraq hesabatda çoxlu qrup altbilgi bölmələri ola bilər. |
| Səhifənin altbilgisi | Hər səhifənin sonunda. | Səhifə nömrələrini və ya hər səhifə məlumatını çap etmək üçün səhifə altbilgisindən istifadə edin. |
| Hesabat Altbilgisi | Hesabatın sonunda. Qeyd: Dizayn görünüşündə hesabat altbilgisi səhifənin altbilgisinin altında görünür. Bununla belə, bütün digər görünüşlərdə (məsələn, tərtibat görünüşü və ya hesabat çap edildikdə və ya əvvəlcədən baxıldıqda) hesabat altbilgisi son səhifədəki son qrup altbilgisindən və ya təfərrüat xəttindən dərhal sonra səhifənin altbilgisinin üstündə görünür. | Bütün hesabat üçün hesabat yekunlarını və ya digər ümumi məlumatı çap etmək üçün hesabat altbilgisindən istifadə edin. |
