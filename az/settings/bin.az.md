## BIN faylı nədir?

**policy.bin** faylları kimi də tanınan BlackBerry İT Siyasəti (İnformasiya Texnologiyası Siyasəti) faylları BlackBerry cihazlarında təhlükəsizlik parametrlərini və konfiqurasiya siyasətlərini tətbiq etmək üçün istifadə olunur. Bu fayllar adətən cihazlar üzərində uyğunluğu və nəzarəti təmin etmək üçün müəssisə mühitində administratorlar tərəfindən yaradılır və yerləşdirilir.

İT Siyasəti faylları cihazın funksionallığı və davranışının müxtəlif aspektlərini tənzimləyən qaydalar və məhdudiyyətlər toplusundan ibarətdir. Bu siyasətlərə parol tələbləri, şifrələmə parametrləri, tətbiq icazələri, şəbəkə parametrləri, cihaz məhdudiyyətləri və s. daxil ola bilər. İT Siyasəti faylını BlackBerry cihazına tətbiq etməklə, administratorlar bu siyasətləri tətbiq edə və istifadəçilərin müəyyən parametrləri dəyişdirməsinə və ya xüsusi funksiyalara daxil olmasına mane ola bilərlər.

İT Siyasəti faylları **BlackBerry Enterprise Server (BES)** və ya **BlackBerry UEM (Unified Endpoint Management)** proqram təminatından istifadə etməklə yaradılmışdır. Administratorlar idarəetmə konsolu vasitəsilə arzu olunan siyasətləri müəyyən edə və sonra siyasətləri İT Siyasət faylı kimi, adətən .bin genişləndirilməsi ilə binar formatda ixrac edə bilərlər.

İT Siyasəti faylını BlackBerry cihazlarına yerləşdirmək üçün administratorlar BlackBerry Enterprise Server və ya UEM-dən istifadə edərək siyasətləri korporativ şəbəkəyə qoşulmuş cihazlara ötürə bilərlər. Cihazlar tərəfindən qəbul edildikdən sonra siyasətlər tətbiq edilir və göstərilən məhdudiyyətlər və parametrlər qüvvəyə minir.

## BIN faylını necə açmaq olar?

BlackBerry İT Siyasəti faylını (policy.bin) açmaq üçün siz **BlackBerry Enterprise Server (BES)** proqram təminatından və ya **BlackBerry UEM (Unified Endpoint Management)** proqram təminatından istifadə etməlisiniz. Bu proqram həlləri BlackBerry cihazlarını korporativ mühitdə idarə etmək üçün nəzərdə tutulub və İT siyasətlərinin tətbiqi və idarə edilməsi üçün lazımi alətləri təmin edir.

Başlamaq üçün BlackBerry Enterprise Server və ya BlackBerry UEM proqramını müəssisə şəbəkəniz daxilində kompüter və ya serverdə quraşdırın və quraşdırın. Proqram təminatı üçün idarəetmə konsolunu işə salın və müvafiq BlackBerry cihaz idarəetmə mühitinə qoşulun.

İdarəetmə konsolunda İT Siyasətləri və ya Siyasət Qaydaları kimi etiketlənə bilən İT siyasətlərini idarə etmək üçün bölmə və ya variantı tapın. İT Siyasəti faylını idxal etmək və ya yükləmək üçün seçim axtarın. Bu, kompüterinizdə siyasət faylını axtarmağa imkan verən düymə və ya menyu elementi ola bilər.

İstədiyiniz yerdən İT Siyasəti faylını (policy.bin) idxal etmək üçün müvafiq seçimi seçin. Proqram təminatı siyasət faylını təsdiq edəcək və onun məzmununu idarəetmə konsoluna yükləyəcək. Import edildikdən sonra siz konsolun istifadəçi dostu interfeysində siyasət parametrlərinə baxa və dəyişə bilərsiniz.

Siyasətdə lazımi dəyişiklikləri edin və onları yadda saxlayın. Tələb olunarsa, yenilənmiş siyasəti dərc edə və ya müəssisə şəbəkəsinə qoşulmuş BlackBerry cihazlarına köçürə bilərsiniz.
