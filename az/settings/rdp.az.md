## RDP faylı nədir?

.rdp fayl uzantısı Remote Desktop Protocol (RDP) faylları üçün istifadə olunur. RDP, istifadəçilərə şəbəkə bağlantısı vasitəsilə uzaq kompüterə qoşulmağa imkan verən Microsoft tərəfindən hazırlanmış xüsusi protokoldur. RDP faylları xüsusi kompüterlə uzaqdan iş masası seansını qurmaq üçün istifadə edilən konfiqurasiya parametrlərini ehtiva edir.

RDP faylına iki dəfə kliklədiyiniz zaman o, adətən kompüterinizdə Uzaqdan Masaüstü Bağlantı proqramını işə salır və faylda göstərilən uzaq kompüterə qoşulmağa başlayır. RDP faylları mətn redaktorundan istifadə etməklə əl ilə yaradıla və ya Uzaq Masaüstü Bağlantısı və ya digər üçüncü tərəf proqram təminatı ilə avtomatik yaradıla bilər.

RDP fayllarına uzaq kompüterin host adı və ya IP ünvanı, uzaq sessiya üçün istifadəçi adı və parol, displey parametrləri və s. kimi müxtəlif konfiqurasiya seçimləri daxil ola bilər. RDP faylından istifadə etməklə istifadəçilər hər dəfə qoşulmaq istədikləri zaman bütün bu parametrləri əl ilə konfiqurasiya etmədən uzaq kompüterlərə tez və asanlıqla qoşula bilərlər.

## RDP Fayl Format - Ətraflı Məlumat

RDP file (Remote Desktop Protocol file) contains configuration settings that are used to connect to remote computer using the Remote Desktop Connection program or third-party RDP client. The configuration settings in RDP file may include:

- Qoşulmaq istədiyiniz uzaq kompüterin host adı və ya IP ünvanı.
- Uzaq kompüterə daxil olmaq üçün istifadə edəcəyiniz istifadəçi adı və parol.
- Mümkünsə, uzaq kompüterin domeni və ya işçi qrupu.
- Uzaq masa üstü seansı üçün displey həlli və rəng dərinliyi parametrləri.
- Uzaq masa üstü seansı üçün audio və video oynatma parametrləri.
- Uzaq iş masası sessiyası üçün printer və mübadilə buferinin yönləndirmə parametrləri.
- Yerli kompüterin ekranındakı uzaq iş masasının ölçüsü və mövqeyi.
- Uzaq masa üstü seansı üçün vaxt zonası və dil parametrləri.
- Uzaq masa üstü sessiyası üçün hər hansı xüsusi konfiqurasiya seçimləri və ya parametrləri.

RDP faylını açdığınız zaman Remote Desktop Connection proqramı konfiqurasiya parametrlərini oxuyur və onlardan uzaq kompüterlə əlaqə yaratmaq üçün istifadə edir. Bu, yerli kompüterinizdən uzaq kompüterə sanki qarşısında oturmuş kimi daxil olmaq və idarə etmək imkanı verir.

## RDP faylının formatı nədir?

RDP faylı (Remote Desktop Protocol file) xüsusi kompüterlə uzaqdan iş masası seansı qurmaq üçün istifadə edilən konfiqurasiya parametrlərini ehtiva edən mətn faylıdır. Bu, Notepad və ya hər hansı digər mətn redaktoru proqramı kimi mətn redaktoru ilə yaradıla və ya redaktə edilə bilən sadə mətn faylıdır.

RDP fayl formatı açar-dəyər cütləri seriyasından ibarətdir, burada hər bir düymə xüsusi konfiqurasiya parametrlərini və müvafiq dəyər həmin parametr üçün dəyəri təyin edir. Format kvadrat mötərizə ilə ayrılmış bölmələri və bərabər işarəsi ilə ayrılmış fərdi açar-dəyər cütləri olan INI faylına bənzəyir.
