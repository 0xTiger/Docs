## VSSX faylı nədir?

.vssx genişləndirilməsi olan fayllar [Microsoft Visio](https://products.office.com/en/visio/flowchart-software) 2013 və yuxarı versiyaları ilə yaradılmış rəsm trafaretləridir. VSSX fayl formatı Visio 2013 və yuxarı versiyaları ilə açıla bilər. Visio faylları formalar, birləşdiricilər, axın sxemləri, şəbəkə tərtibatı, UML diaqramları, proqram diaqramları, verilənlər bazası modelləri, obyektlərin xəritələşdirilməsi və digər oxşar məlumatlar kimi müxtəlif rəsm elementlərinin təmsil olunması ilə tanınır. Microsoft Visio həmçinin Visio fayllarını [PNG](/image/png/), [BMP](/image/bmp/), [PDF](/pdf/) və başqaları kimi müxtəlif fayl formatlarına çevirmək imkanı verir. Həm Windows, həm də Mac OS üçün mövcuddur.

## VSSX fayl formatı

The VSSX file format is based on the OpenOffice format adopted by Microsoft since 2007. O, XML fayl formatının spesifikasiyasına əsaslanan ümumi fayl formatını təmsil edən [ZIP](/compression/zip/) arxivinə əsaslanır. VSSX-in ikili fayl formatı ekvivalenti Visio 2007-ci ilə qədər dəstəklənən VSS idi. Siz VSSX fayl formatının məzmununa onun genişləndirilməsini .ZIP ilə əvəz etməklə baxa və WinZIP kimi istənilən arxiv fayl formatında aça bilərsiniz.

VSDX faylları Açıq Qablaşdırma Konvensiyalarına və XML-ə əsaslanır və tərtibatçılar bu faylla proqramlı şəkildə işləməyi öyrənməklə bu formatdan faydalana bilərlər. Format Visio XML Drawing fayl formatından (.vdx) onun hissələri kimi eyni XML strukturlarının çoxunu miras alır. Üçüncü tərəf proqram təminatı Visio fayllarını fayl formatı səviyyəsində manipulyasiya edə bildiyi üçün Visio faylları ilə qarşılıqlı fəaliyyət xeyli artır.

Visio 2013 fayl formatını təşkil edən bəzi digər fayl növlərinə aşağıdakılar daxildir:

* .vsdm (Visio makro ilə aktivləşdirilmiş rəsm)

* .vssx (Visio trafareti)

* .vssm (Visio makro-aktiv trafaret)

* .vstx (Visio şablonu)

* .vstm (Visio makro ilə aktivləşdirilmiş şablon)


Hər bir Visio faylı digər faylları və ya hissələri saxlayan paket adlanır. Paket hissəsi XML faylı, şəkil və ya hətta VBA həlli ola bilər. Paket daxilindəki hissələr sənəd və əlaqə hissələrinə bölünə bilər.
