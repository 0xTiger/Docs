## VSS faylı nədir?

VSS are stencil files created with Microsoft Visio 2007 and earlier. A relatively new file format is [VSSX](/visio/vssx/) that was introduced with Microsoft Visio 2013. Stencil faylları .VSD Visio rəsminə daxil edilə bilən rəsm obyektlərini təmin edir. Microsoft Visio özü formalar, birləşdiricilər, axın diaqramları, şəbəkə tərtibatı, UML diaqramları, proqram diaqramları, verilənlər bazası modelləri, obyektlərin xəritələşdirilməsi və digər oxşar məlumatlar kimi rəsm elementlərinin yaradılması ilə tanınır. O, həmçinin Visio sənədlərinin PNG, BMP, PDF və başqaları kimi digər fayl formatlarına zəngin konvertasiya xüsusiyyətlərinə malikdir. Visio həm Windows, həm də Mac OS üçün mövcuddur.

## VSS fayl formatı

Office Open XML fayl formatına əsaslanan VSSX-dən fərqli olaraq, VSS faylları ikili fayl formatına əsaslanır.

## Qısa tarix ##

Initially Visio was produced as a standalone product by Shapeware Corporation in 1992 by its version 1.0. Shapeware Korporasiyası bazarın tanınmasından istifadə etmək üçün 1995-ci ildə adını Visio Corporation olaraq dəyişdirdi. Microsoft 2000-ci ildə Vision-u Microsoft Office tətbiqi kimi yenidən brendləşdirdi. Visio for Enterprise Architects 2003 və 2005-ci illərdə Visual Studio .NET ilə təqdim edilmişdir. Visio 2007 30 noyabr 2006-cı ildə buraxılmışdır. Lentlər istifadəçi interfeysində 2010-cu ildə Microsoft Visio tərəfindən təqdim edilmişdir.
