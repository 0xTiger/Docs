## VSD faylı nədir?

VSD faylları müxtəlif qrafik obyektləri və bunlar arasındakı qarşılıqlı əlaqəni təmsil etmək üçün Microsoft Visio proqramı ilə yaradılmış çertyojlardır. Belə təsvirlər vizual obyektlər, axın qrafikləri, UML diaqramı, məlumat axını, təşkilati diaqramlar, proqram diaqramları, şəbəkə tərtibatı, verilənlər bazası modelləri, obyektlərin xəritələşdirilməsi və digər oxşar məlumatları ehtiva edə bilər. Microsoft Visio Visio fayllarını [PNG](/image/png/), [BMP](/image/bmp/), [PDF](/pdf/) və başqaları daxil olmaqla bir sıra müxtəlif fayl formatlarına çevirmək imkanı təklif edir.

# VSD Fayl Format #

VSD file format is the initial file format introduced by Microsoft Visio for storing drawing elements. The internal structure of VSD files is based on binary file format unlike [.VSDX](/visio/vsdx/) that was introduced with Microsoft Visio 2013. Yeni format Office Open XML fayl formatına əsaslanır və layihə məlumatlarını [ZIP](/compression/zip/) arxivində saxlayır. Visio sistemdə quraşdırılmadığı halda Microsoft Visio rəsm fayllarına vviewer.exe proqramını təklif edir.

## Qısa tarix ##

Visio was initially produced as a standalone product by Shapeware Corporation in 1992 by its version 1.0. Shapeware Korporasiyası bazarın tanınmasından istifadə etmək üçün 1995-ci ildə adını Visio Corporation olaraq dəyişdirdi. Microsoft 2000-ci ildə Vision-u Microsoft Office tətbiqi kimi yenidən brendləşdirdi. Visio for Enterprise Architects 2003 və 2005-ci illərdə Visual Studio .NET ilə təqdim edilmişdir. Visio 2007 30 noyabr 2006-cı ildə buraxılmışdır. Lentlər istifadəçi interfeysində 2010-cu ildə Microsoft Visio tərəfindən təqdim edilmişdir.
