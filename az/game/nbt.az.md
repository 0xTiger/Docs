## NBT faylı nədir?

NBT faylı Minecraft-da istifadə olunan fayl formatıdır. NBT, Adlandırılmış Binary Tag deməkdir və strukturlaşdırılmış məlumatların saxlanması üçün sadə ikili formatdır (JSON və ya XML-ə bənzər). Minecraft-da NBT faylları müxtəlif növ oyun məlumatlarını, o cümlədən oyunçu inventarlarını, dünya məlumatlarını, qurum məlumatları və s. saxlamaq üçün istifadə olunur.

NBT faylları adətən `.dat` uzantısı ilə saxlanılır və Minecraft dünyasının kataloq strukturunda tapıla bilər. Məsələn, oyunçu inventarları `playerdata` kataloqunda `.dat` fayllarında, dünya məlumatı isə `region` kataloqunda `.dat` fayllarında saxlanılır.

## NBT Fayl Format - Ətraflı Məlumat

NBT, böyük miqdarda ikili məlumatı səmərəli saxlamaq üçün nəzərdə tutulmuş sadə ikili formatdır. Çevikliyi və istifadəsi asanlığı səbəbindən Minecraft tərəfindən məlumat faylları üçün istifadə olunur. Minecraft, oyunçuların virtual dünyalar qura və kəşf edə biləcəyi məşhur 3D qum qutusu oyunudur. Minecraft-ın Java Buraxılışı, faylları redaktə etməklə və ya oyunu fərdiləşdirmək üçün yeniləri əlavə etməklə oyunu dəyişdirən moderatorlar arasında xüsusilə populyardır. Minecraft-da NBT faylları tez-tez modderlər tərəfindən yoxlanılır və redaktə edilir. Bu fayllar adətən GZip istifadə edərək sıxılır və dünyanın qurulması və fərdiləşdirilməsi üçün vacib olan müxtəlif oyun məlumat elementlərini ehtiva edə bilər.
