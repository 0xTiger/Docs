## UASSET faylı nədir?

UASSET faylı Epic Games tərəfindən hazırlanmış məşhur oyun inkişaf motoru olan Unreal Engine tərəfindən istifadə edilən fayl formatıdır. O, Unreal Asset mənasını verir və Unreal Engine layihələrində istifadə olunan 3D modellər, teksturalar, materiallar, animasiyalar, audio fayllar və s. kimi müxtəlif növ məlumatlardan ibarətdir.

## UASSET Fayl Format - Ətraflı Məlumat

`.uasset` faylları ilə bağlı bəzi əsas məqamlar bunlardır:

1.  **İkili Format**: `.uasset` faylları adətən ikili formatda saxlanılır, yəni mətn faylları kimi onlar birbaşa insan tərəfindən oxuna bilməz. Onlar Unreal Engine daxilində səmərəli yükləmə və emal üçün optimallaşdırılmışdır.
    
2.  **Aktiv Tipləri**: `.uasset` faylının məzmunu təmsil etdiyi aktivin növündən asılı olaraq dəyişə bilər. Məsələn, `.uasset` faylı statik şəbəkə, material, səs siqnalı, hissəcik sistemi və s. məlumatları ehtiva edə bilər.
    
3.  **İstinad**: `.uasset` faylları tez-tez layihə daxilindəki digər aktivlərə istinad edir. Məsələn, maddi aktiv digər `.uasset` fayllarında müəyyən edilmiş faktura və parametrlərə istinad edə bilər.
    
4.  **Redaktor İnteqrasiyası**: Unreal Engine Unreal Redaktor daxilində `.uasset` fayllarını yaratmaq, dəyişdirmək və idarə etmək üçün alətlər və redaktorlar təqdim edir. Tərtibatçılar bu aktivlərlə qarşılıqlı əlaqədə olmaq və layihələrində dinamik davranışlar yaratmaq üçün Blueprint skriptindən və ya C++ proqramlaşdırmasından istifadə edə bilərlər.
    
5.  **Versiyaya Nəzarət**: Unreal Engine layihələri üzərində birgə işləyərkən, `.uasset` faylları adətən dəyişiklikləri izləmək və komanda əməkdaşlığını asanlaşdırmaq üçün Git və ya Perforce kimi versiyaya nəzarət sistemlərindən istifadə etməklə idarə olunur.
    
6.  **Qablaşdırma**: Yayılma üçün Unreal Engine ilə qurulmuş oyunu və ya proqramı qablaşdırarkən, `.uasset` faylları adətən son quruluşun bir hissəsi kimi daxil edilir. Bu fayllar hədəf platformada yerləşdirilməsi üçün optimallaşdırılıb və paketlənib.

## UASSET faylını necə açmaq olar?

`.uasset` faylını açmaq üçün adətən Unreal Mühərrik Redaktorundan istifadə etməlisiniz. Sadəcə

1.  Unreal Engine Editor proqramını işə salın.
2.  Layihə açın və ya yaradın.
3.  Məzmun Brauzerinə keçin.
4.  `.uasset` faylını tapın.
5.  Onu açmaq üçün iki dəfə klikləyin.
6.  Redaktor pəncərəsində aktivlə işləyin.
7.  Lazım gələrsə dəyişiklikləri yadda saxlayın.
