## GCF faylı nədir?

.gcf faylı Steam oyun platforması tərəfindən istifadə edilən oyun keş faylıdır. O, oyun tərəfindən istifadə olunan fakturalar, modellər və digər fayllar kimi oyun məlumatlarını ehtiva edir. Bu fayllar istifadəçinin kompüterində saxlanılır və oyunu yeniləyərkən və ya quraşdırarkən yüklənməli olan məlumatların miqdarını azaltmaq üçün istifadə olunur. .gcf faylları həmçinin oyunun quraşdırılmasının ehtiyat nüsxələrini yaratmaq və ya oyunu kompüterlər arasında köçürmək üçün istifadə edilə bilər.

GCF faylları Açıq Mənbəli C++ proqramlaşdırma Kitabxanası, **HLLib** tərəfindən oxuna və yaza bilər.

## GCF fayl formatı

GCF faylları ikili fayllar kimi diskdə saxlanılır. GCF ilkin olaraq Grid Cache File (Steam-in ilk kod adı Grid idi) üçün qısaltma kimi istifadə edildi, lakin sonradan Game Cache File kimi qəbul edildi. GCF faylı Steam oyunlarını saxlayan arxiv faylıdır. Bütün rəsmi məzmun GCF faylları kimi endirilir. GCF faylları da oyunlar arasında paylaşıla bilər.

QEYD: GCF [VPK file format](/game/vpk/) sələfidir.
