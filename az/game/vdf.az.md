## VDF faylı nədir?

The Valve Data File kimi də tanınan VDF faylı müxtəlif növ oyunla bağlı metadataları saxlamaq üçün Valve's Source oyun mühərriki tərəfindən istifadə edilən fayl formatıdır; VDF faylları resurslar, quraşdırma skriptləri, konfiqurasiya skriptləri və vizuallaşdırma elementləri üçün məlumatlar daxil olmaqla, Valve's Source oyun mühərriki ilə əlaqəli məlumatları saxlayır; Half-Life 2 və Counter-Strike kimi məşhur başlıqlar kimi Valve's Steam oyun müştərisi vasitəsilə paylanmış oyunları quraşdırarkən VDF fayllarına tez-tez rast gəlinir.

VDF faylları düz mətn formatı olan Valve's KeyValues formatından istifadə edir; Açar Dəyərlər formatı JSON (JavaScript Object Notation) və YAML (YAML İşarələmə Dili Deyil) kimi formatlara bir qədər oxşar olan açar-dəyər cütlərindən istifadə edərək məlumatları iyerarxik strukturda təşkil edir. VDF fayllarında açar-dəyər parametrləri ağac iyerarxiyasında təşkil edilmişdir; bu iyerarxik quruluş oyun məlumatları daxilində mürəkkəb münasibətləri və konfiqurasiyaları təmsil etməyə imkan verir.

VDF faylları düz mətndə saxlandıqları üçün insanlar tərəfindən oxuna bilər; bu o deməkdir ki, siz sadə mətn redaktorundan istifadə edərək VDF faylının məzmununu aça və baxa bilərsiniz; VDF fayllarında KeyValues formatının strukturu JSON və YAML ilə oxşarlıqları bölüşür. Bu formatlar kimi, məlumat əlaqələrini və atributlarını təmsil etmək üçün sadə sintaksisdən istifadə edir. VDF faylları Valve Source oyun mühərriki tərəfindən istifadə üçün nəzərdə tutulsa da, onlar mətn redaktoru vasitəsilə açıla və yoxlana bilər ki, bu da istifadəçilərin və ya tərtibatçıların müəyyən oyun konfiqurasiyalarını başa düşməsini və dəyişdirməsini asanlaşdırır.

## Valve's Source Game Engine haqqında

Valve's Source oyun mühərriki Valve Corporation tərəfindən yaradılmış oyun inkişaf etdirmə mühərrikidir. Bu, Valve, eləcə də digər üçüncü tərəf oyun tərtibatçıları tərəfindən hazırlanmış bir çox məşhur video oyunların arxasında duran mühərrikdir. Mənbə mühərriki birinci şəxs atıcılarından tutmuş tapmaca oyunlarına qədər müxtəlif oyun janrlarını inkişaf etdirmək üçün istifadə edilmişdir. Mənbə mühərriki ilə bağlı bəzi əsas məqamlar bunlardır:

1.  **Oyun Başlıqları:** Mənbə mühərriki bir neçə tanınmış oyunu təchiz etmişdir, o cümlədən:
    
- Half-Life 2 və onun epizodları
- Counter-Strike: Source və Counter-Strike: Global Offensive
- Komanda Qalası 2
- Portal və Portal 2
- Sol 4 Ölü və Sol 4 Ölü 2
2.  **Bacarıqlar:** Mənbə mühərriki çevikliyi və imkanları ilə tanınır. O, real fizika, qabaqcıl qrafika, dinamik işıqlandırma və mürəkkəb səviyyəli dizayn kimi xüsusiyyətləri dəstəkləyir. Mühərrik tərtibatçılara immersiv və vizual olaraq təsir edici oyun təcrübəsi yaratmağa imkan verir.
    
3.  **Modlaşdırma İcması:** Mənbə mühərrikində canlı modding icması var. Valve istifadəçi tərəfindən yaradılan məzmunu təşviq etdi və bir çox oyunçu və tərtibatçı Source mühərrik oyunları üçün xüsusi modifikasiyalar (modlar) və xəritələr yaratdı. Hammer Editor kimi alətlərin mövcudluğu modding icmasının böyüməsinə kömək etdi.
    
4.  **Continual Updates:** Valve has periodically updated Source engine to incorporate new technologies and improvements. These updates have included enhancements to graphics, physics and overall performance.
