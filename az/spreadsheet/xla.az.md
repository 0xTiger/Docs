## XLA faylı nədir?

.xla uzantılı fayl Microsoft Excel-ə modullar, əlavə funksiyalar və digər alətlər əlavə edən Microsoft Excel Add-Ins fayl formatıdır. Reklamlar Excel cədvəllərinə əlavə funksionallıq təmin edir və 2007 və ya sonrakı versiyalarda dəstəklənir. XLA fayllarında mime media tipli application/vnd.ms-excel var. Bunlar Windows və MacOS üçün mövcud olan Microsoft Excel istifadə edərək açıla bilər.

## XLA fayl formatı

XLA faylları ikili məlumat formatına əsaslanır. Bununla belə, onun fayl formatı spesifikasiyası haqqında əlavə məlumat mövcud deyil. Ümumi fərziyyə ondan ibarətdir ki, XLA [[MS-XLS] - Excel Binary File Format Structure](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx-ə əməl edir.
