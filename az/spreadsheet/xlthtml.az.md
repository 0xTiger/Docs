## XLTHTML faylı nədir?

.xlthtml uzantılı fayl [XLT](/spreadsheet/xlt/) şablon fayllarının [HTML](/web/html/) versiyasıdır və ilk günlərdə Microsoft Excel istifadə edərək yaradılıb. Bu format artıq Excel tərəfindən dəstəklənmir və verilənlərin [MHT](/web/mht/) və [MHTML](/web/mhtml/) fayl formatlarına ixracı ilə əvəz edilmişdir. XLTHTML faylının əsas ideyası istifadəçilərə sistemdə Excel-in quraşdırılması ehtiyacı əvəzinə XLT şablon fayllarına veb-brauzerdə baxmaq imkanı vermək idi. XLTHTML fayl formatında ixrac edilən fayllar brauzerlərdə açıla bilər.

## XLTHTML Fayl Format - Ətraflı Məlumat

XLTHTML faylları XLT şablonlarını veb brauzerdə açmaq üçün saxlamaq üçün HTML fayl formatından istifadə edirdi. XLT özü Excel 97-2003 versiyası ilə təqdim edilmiş Excel ikili fayl formatına əsaslanır. Bu fayllar elektron cədvəl məlumatlarının kütləsi üçün normal sənədlər yaratmaq üçün müəyyən edilmiş formatlı başlanğıc faylları kimi istifadə edilə bilər. Yalnız istinad üçün, XLT ikili [XLS file format specifications](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx-ə əsaslanır və Microsoft tərəfindən tərtibatçının arayışı üçün sənədləşdirilmişdir.
