## XLR faylı nədir?

A file with .xlr extension is a Microsoft Works Spreadsheet file format that stored data in rows and columns. Microsoft Works was developed for use as an initial productivity suite and remained functional from 1987 to 2009. Buraya mətn prosessoru, elektron cədvəl və verilənlər bazası idarəetmə sistemi daxildir. Daha zəngin xüsusiyyətlərə malik Microsoft Office təqdim edildikdən sonra o, dayandırıldı.

## XLR Fayl Format - Ətraflı Məlumat

XLR faylları ikili fayllar kimi saxlandı, onun fayl formatı spesifikasiyası haqqında heç bir təfərrüat yoxdur. Buna görə də tərtibatçıların onun fayl strukturunu təhlil etmək və bu faylları proqramlı şəkildə oxumaq/yazmaq üçün proqramlar yazmaq imkanı yoxdur. Works Spreadsheet verilənləri .xlr fayllarında saxlamaq üçün eyni Excel fayl formatından istifadə edir.
