## TMV faylı nədir?

A TMV file is a spreadsheet file created with [TimeMap](https://www.lexisnexis.com/en-us/products/casemap.page) software by LexisNexis. TimeMap allows you to create timelines with the specified date and time of facts. With entry of new facts, the software creates factboxes for these and positions these on the occurrence of timeline. The TMV project can comprise of fact boxes, text boxes, and text inside these can be formatted using different fonts. TMV files can be opened with TimeMap software. LexisNexis also provides an online collaboration tool now in cloud based environment.

## TMV Fayl Formatı

TMV faylları diskdə CaseMap xüsusi fayl formatında saxlanılır, onların fayl formatı spesifikasiyası onlayn mövcud deyil. CaseMap LixesNexis tərəfindən alınıb.
