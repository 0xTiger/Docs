## DEF faylı nədir?

DEF faylı Informix Software tərəfindən hazırlanmış SmartWare II verilənlər bazası proqramı tərəfindən [spreadsheet file format](/spreadsheet/)-də yaradılmış məlumat faylıdır. Proqram II dörd moduldan ibarət idi, o cümlədən qrafika ilə elektron cədvəl, mətn prosessoru, rabitə və verilənlər bazası. SmartWare II verilənlər bazası digər proqramlarla da inteqrasiya oluna bilən müstəqil verilənlər bazasıdır. Sorğular çıxarıla və DEF fayl formatında saxlanıla bilən məlumatlar üzərində icra edilə bilər. DEF elektron cədvəl faylları hazırda fəaliyyəti dayandırılmış SmartWare proqramı ilə açıla bilər.
