## XLAM faylı nədir? ##

XLAM, elektron cədvəllərə yeni funksiyalar əlavə etmək üçün istifadə olunan Makro-aktiv əlavə fayldır. Add-in əlavə kod işlədən və cədvəllər üçün əlavə funksionallıq təmin edən əlavə proqramdır. XLAM faylları .xlam uzantısı ilə saxlanılır. XLAM faylları [XLSM](/spreadsheet/xlsm/) və [XLSX](/spreadsheet/xlsx/) fayl formatlarına bənzər XML əsaslı fayllardır və ümumi fayl ölçüsünü azaltmaq üçün ZIP sıxılma ilə saxlanılır.

### Nümunə ###

```cmd
Public Function Add(num1 As Double, num2 As Double)
Add = num1 + num2
End Function
```

Bundan sonra faylı .xlam uzantısı ilə yadda saxlayın.
