## XLSMHTML faylı nədir?

.xlsmhtml uzantısı olan fayl, verilənlərə brauzerdə baxmaq üçün Microsoft Excel daxilində ixrac edilən orijinal XLS faylının [HTML](/web/html/) versiyasıdır. Format çoxdan Excel tərəfindən dayandırılıb və artıq dəstəklənmir. İxrac edilmiş XLSMHTML faylı XLS fayl formatına uyğundur və bunlar veb brauzerlərdə açılıb baxıla bilər. Excel elektron cədvəl məlumatlarının [MHT](/web/mht/) və [MHTML](/web/mhtml/) fayl formatlarına ixracını təmin edir və hazırda geniş istifadə olunur.

## XLSMHTML Fayl Format - Ətraflı Məlumat

XLSMHTML faylları elektron cədvəl məlumatlarını XLS-ə oxşar formatda təqdim etmək üçün HTML kodu ilə yaradılmışdır. XLSMHTML faylının içərisindəki HTML kodu elə yerləşdirilib ki, çıxış faylı veb brauzerdə açıldığında orijinal fayla bənzəyir. XLSMHTML faylının fayl formatının spesifikasiyası Excel tərəfindən açıqlanmır və tərtibatçılar üçün proqramları başa düşmək və inkişaf etdirmək üçün belə istinadlar mövcud deyil.
