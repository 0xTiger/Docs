## VGA faylı nədir?

VGA faylı VGA monitorlarını dəstəkləmək üçün Windows sürücüsü faylıdır. O, əməliyyat sistemi (OS) səviyyəsində VGA monitorlarının tənzimlənməsi üçün Microsoft Windows-un erkən versiyalarında istifadə edilmişdir. İstifadəçilərə Monitorun idarəetmə panelindən istifadə edərək ekranın təsvir ölçüsünü və rəng dərinliyini tənzimləməyə imkan verən VGA faylı.

## VGA fayl formatı

VGA fayl formatının daxili fayl formatı tərtibatçının istinadı üçün məlum deyil. VGA monitorunun vizual görünüşünü təyin etmək üçün Windows əməliyyat sistemi tərəfindən istinad edilir.
