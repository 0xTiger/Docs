## PAT faylı nədir?

PAT faylı adətən Synology Network-Attached Storage (NAS) cihazları tərəfindən istifadə edilən **DiskStation Manager (DSM)** əməliyyat sistemi ilə əlaqələndirilir. PAT faylı DSM-i Synology NAS-da yeniləmək və ya quraşdırmaq üçün istifadə edilən quraşdırma faylıdır.

## PAT faylının istifadəsi

Here is how you typically use a PAT file to install or update DSM on Synology NAS:

1.  PAT Faylını endirin: PAT faylını rəsmi Synology saytından və ya digər etibarlı mənbələrdən əldə edə bilərsiniz.
    
2.  NAS-a daxil olun: Veb brauzerdə onun IP ünvanını daxil etməklə Synology NAS-ın veb-əsaslı istifadəçi interfeysinə daxil olun. Bu əməliyyatı yerinə yetirmək üçün sizə administrator imtiyazları lazımdır.
    
3.  Paket Mərkəzinə keçin: Veb interfeysində Paket Mərkəzinə keçin. NAS-da tətbiqləri idarə etdiyiniz və quraşdırdığınız yer budur.
    
4.  Manual Installation: In Package Center, there should be an option for "Manual Installation" or "Install/Update" depending on version of DSM you are using. Choose this option.
    
5.  PAT Faylını axtarın: Sizdən yerli fayllarınıza baxmaq və yüklənmiş PAT faylını seçmək təklif olunacaq.
    
6.  Install or Update: After selecting PAT file, follow on-screen instructions to either install DSM (if it is a fresh installation) or update DSM (if you are upgrading to newer version).
    
7.  Wait for Completion: The installation or update process can take some time and your NAS may reboot as part of process. Be patient and wait for it to finish.
    
8.  Post-Installation Configuration: After installation or update, you may need to configure your NAS settings as per your preferences.

## DiskStation Meneceri

Tez-tez DSM kimi qısaldılmış DiskStation Manager, Synology tərəfindən Şəbəkəyə Bağlı Yaddaş (NAS) cihazları üçün hazırlanmış əməliyyat sistemidir. O, Synology NAS serverləri üçün idarəetmə və nəzarət interfeysi kimi xidmət edir. DiskStation Manager istifadəçilərə məlumatların saxlanması, fayl paylaşımı, ehtiyat nüsxə həlləri, multimedia xidmətləri və s. kimi NAS-larının müxtəlif aspektlərini konfiqurasiya etməyə və idarə etməyə imkan verən istifadəçi dostu veb-əsaslı interfeys təqdim edir.

DSM çox yönlülüyü və geniş paket ekosistemi ilə tanınır ki, bu da istifadəçilərə Synology NAS-da müxtəlif proqramlar və xidmətləri quraşdırmağa və işə salmağa imkan verir və onu həm evdə, həm də biznesdə istifadə üçün çoxməqsədli serverə çevirir. DiskStation Manager-in bəzi ümumi funksiyalarına fayl paylaşımı, məlumatların ehtiyat nüsxəsi və sinxronizasiyası, media axını, təhlükəsizlik xüsusiyyətləri və virtuallaşdırma dəstəyi daxildir.

Synology təhlükəsizlik, performans və funksiyaları artırmaq üçün müntəzəm olaraq yeniləmələri və DSM-in yeni versiyalarını buraxır. İstifadəçilər PAT fayllarından istifadə edərək DSM-i əl ilə yeniləyə və ya NAS-larının DiskStation Manager-in ən son və ən təhlükəsiz versiyasını işlətməsini təmin etmək üçün avtomatik yeniləmələri konfiqurasiya edə bilərlər.

## PAT faylını necə açmaq olar

To manually update DiskStation Manager, you can use a PAT file that you have downloaded from Synology Download Center using the following steps:

1. DiskStation Manager İdarəetmə Panelinə daxil olun.
2. Yeniləmə və Bərpa bölməsinə keçin və DSM Yeniləmə seçin.
3. Oradan Manual DSM Update seçin.
4. Gözdən keçir düyməsini klikləyin və sonra yüklədiyiniz PAT faylını tapıb seçin.
5. DiskStation Manager yeniləməsini başlamaq üçün Tətbiq et düyməsini basın.
