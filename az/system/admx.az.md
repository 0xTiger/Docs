## ADMX faylı nədir?

An ADMX file is a policy configuration file used by Microsoft Windows Group Policy software that manages group of computers in a group. It is an XML-based administrative template file and was introduced with Windows Vista Service Pack 1. ADMX faylları istifadəçi hesabları, əməliyyat sistemi konfiqurasiyaları və tətbiqlər üçün konfiqurasiya parametrlərini ehtiva edir. ADMX faylları bir çox kompüter sistemlərinin mərkəzləşdirilmiş idarə edilməsi üçün konfiqurasiyaları saxlamaq və yerləşdirmək üçün istifadə olunur.

Siz istənilən XML uyğunluğu, istənilən mətn redaktoru və ya Microsoft Qrup Siyasəti Obyekt Redaktorundan istifadə edərək ADMX faylları yarada və ya redaktə edə bilərsiniz.

## ADMX Fayl Format - Ətraflı Məlumat

ADMX faylları insan tərəfindən oxuna bilən universal [XML](/web/xml/) fayl formatında saxlanılır. Bu, çoxdilli fayl formatıdır və müxtəlif dillərdən olan sistem administratorlarına eyni ADMX faylları ilə işləməyə imkan verir. ADMX faylları unicode formatlı mətn faylı formatı olan köhnə [ADM](/system/adm/) fayl formatını əvəz etdi. ADMX faylları müəyyən Qrup Siyasəti parametrləri dəyişdirildikdə dəyişdirilən qeyd açarlarını təyin edir.

### ADMX faylı ilə nə edə bilərəm?

ADMX faylları qrupun bir hissəsi olan istifadəçi kompüterlərinə hansı hərəkətlərə icazə veriləcəyini müəyyənləşdirir. Qrup siyasəti Active Directory proqramı ilə birlikdə müəyyən edilmiş qaydaları tətbiq edir. ADMX faylları domendə mərkəzi yer olan Windows OS-də mərkəzi mağazadan idarə olunur.

İş mühitində yerləşdirilən ADMX faylı administratorlara aşağıdakı kimi siyasətləri həyata keçirməyə imkan verə bilər:

 * İnternetdən istifadəyə nəzarət edin
 * Müəyyən növ faylların yüklənməsi
 * Sistemlərdə çıxarıla bilən cihazların istifadəsi
