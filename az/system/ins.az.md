## INS faylı nədir?

.ins uzantısı olan fayl ümumiyyətlə Microsoft Windows tərəfindən dial-up və genişzolaqlı İnternet bağlantılarının qurulması üçün istifadə olunur. Əslində, o, Windows-un ISP-yə İnternetə çıxışını qurmağa imkan verən əlaqə quraşdırma məlumatlarını ehtiva edir. INS faylı adətən Internet Explorer (IE) LAN bağlantısı parametrlərini qurmaq üçün istifadə olunur. IE əlaqə parametrləri faylları bəzən ISP-lər və sistem administratorları tərəfindən INS faylının URL-i ilə istifadəçilərə təqdim edilir.

## INS fayl formatı
Siz fərdi brauzerinizi və onun komponentlərini konfiqurasiya etmək üçün Internet Explorer Administration Kit 11 (IEAK 11) ilə birlikdə İnternet Parametrləri fayllarından (INS) istifadə edə bilərsiniz. INS faylının nüsxələrini fərdiləşdirərək, fərdi paketinizin müxtəlif versiyalarını yaza bilərsiniz.

### Mümkün INS Parametrləri
Mümkün INS parametrləri aşağıdakı cədvəldə verilmişdir:

| Parametrlər | Təsvir |
-----|---------|
| URL | Avtomatik konfiqurasiya edilmiş proksi serverdən istifadə edib-etməmək. |
| Brendinq | Brauzer paketinizdə brendinq və quraşdırma məlumatlarını fərdiləşdirin. |
| Media | Fərdi quraşdırma paketinizin mövcud olduğu media növləri. |
| BrowserToolbars | IE alətlər panelinin görünüşünü fərdiləşdirin. |
| CabSigning | Proqramlarınız üçün rəqəmsal imza məlumatı. |
| HideCustom | Hər bir fərdi komponent üçün qlobal unikal identifikatorun (GUID) gizlədilməməsi. |
| ConnectionSettings | Fərdi paketinizi quraşdırmaq üçün istifadə edilən şəbəkə bağlantısı parametrləri haqqında məlumat. |
| CustomBranding | Brendinq kabinetinizin (.cab) faylının URL yeri. |
| ExtRegInf | Quraşdırma məlumatı (.inf) fayllarınızın adları və komponentlər üçün quraşdırma rejimi. |
| SevimlilərEx | Sevimlilər üçün ikona faylınıza yol əlavə edin, Sevimlilərin oflayn mövcud olub-olmamasına qərar verin və hər Sevimlilər saytına URL əlavə edin. |
| ISP_Təhlükəsizlik | Fərdi paketinizə əlavə etdiyiniz kök sertifikat. |
| Proksi | Proksi serverdən istifadə edib-etməmək. |
| Təhlükəsizlik İdxal | Fərdi paketiniz üçün təhlükəsizlik məlumatının idxal edilib-edilməməsi. |
