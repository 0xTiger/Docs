## MPP faylı nədir?

.mpp uzantılı fayl layihənin idarə edilməsi ilə bağlı məlumatları inteqrasiya olunmuş şəkildə saxlayan Microsoft Project məlumat faylıdır. Microsoft bu xüsusi fayl formatını layihə idarəetmə proqram məhsulu olan Microsoft Project (MSP) ilə uyğunlaşdırmaq üçün işləyib hazırlayıb. MPP-dən başqa, MSP layihə XML sxemi ilə işləyə bilər. Bir çox proqramlar və API-lər MPP-lərin digər fayl formatlarına çevrilməsinə imkan verir. Microsoft indi layihə idarəetmə fayllarının çoxsaylı istifadəçi tərəfindən əməkdaşlıq üçün yüklənə biləcəyi onlayn Layihə Serverinə malikdir.

## MPP Fayl Format ##

Microsoft MPP-ni ikili fayl formatı kimi hazırlayıb nəşr etdi və onun spesifikasiyası bu günə qədər Microsoft tərəfindən açıqlanmayıb. Daxili fayl strukturu və təfərrüatları tərtibatçılar üçün əlçatan deyil. MPP fayllarının oxunmasını/yazılmasını dəstəkləyən onlayn bir neçə API mövcuddur, lakin 100% dəqiqlik heç vaxt əldə edilmir. Microsoft zamanla layihənin idarə edilməsi proqramına bir neçə funksiya təqdim etdi və MPP fayl formatının xüsusiyyətləri də belədir.

MPP faylı aşağıdakı MIME növlərindən istifadə edir:

* application/vnd.ms-project

* proqram/msproj

* proqram/msproject

* proqram/x-msproject

* proqram/x-ms-layihə

* proqram/x-dos_ms_project

* proqram/mpp

* zz-application/zz-winassoc-mpp


## MPP faylını açmaqda problemlər var? ##

Aşağıda MPP formatının düzgün işləməməsinə səbəb ola biləcək bəzi ümumi problemlərin siyahısı verilmişdir:

 *   Dəstəkləyici proqram təminatının olmaması
 *   Zədələnmiş fayl
 *   Virus səbəbindən yoluxmuş fayl
 *   Sistemdə faylları açmaq hüququ yoxdur
 *   Sisteminizdə köhnəlmiş disk
 *   Faylın genişləndirilməsinin adı dəyişdirilir
