## XER faylı nədir?

XER fayl formatı Primavera P6 layihənin planlaşdırma və idarəetmə tətbiqi tərəfindən istifadə edilən xüsusi layihə fayl formatıdır. [MPP](/project-management/mpp/) fayl formatı kimi, bu fayl formatı da tapşırıqlar, resurslar, vaxt qrafikləri və materiallar kimi layihənin planlaşdırılması ilə bağlı məlumatları saxlayır. Primavera Oracle tərəfindən hazırlanmışdır və tam layihə planlamasını təmin edir və neft və qaz, tikinti, istehsalat və s. kimi bir neçə praktiki sahədə tətbiqini tapır. O, layihə menecerlərinə inteqrasiya etməklə irimiqyaslı proqramları və fərdi layihələri planlaşdırmağa, planlaşdırmağa və idarə etməyə kömək edir. maliyyə idarəetməsi və insan kapitalının idarə edilməsi sistemləri ilə. Bir çox proqram təminatı və API XER fayllarını oxuya və XML və MPP kimi digər məşhur layihə fayl formatlarına çevirə bilər.

## XER Fayl Format ##

XER faylı mətn faylıdır və mətn redaktoru ilə asanlıqla aça bilər. Layihə ilə bağlı məlumatların mətn redaktorunda açılması faydalı ola bilər, lakin bağlantıları əl ilə qurmaqla onları başa düşmək asan deyil. Bu format digər P6 verilənlər bazası ilə məlumat mübadiləsi üçün ən çox istifadə olunan formatdır. Başqa formatdan idxal edilmiş məlumatlara nəzarətin nisbətən aşağı səviyyəsinə malikdir. XER-in ekvivalent XML formatında üstünlükləri qlobal kodları və təqvimləri layihə səviyyəsinə köçürmək və xəritə kodları, UDF, resurslar və təqvimləri mövcud dəyərlərə köçürmək qabiliyyətini əhatə edir.


## XER faylını açmaqda problem var? ##

XER formatının düzgün işləməməsinə səbəb ola biləcək bəzi ümumi problemlər aşağıdakılardır:

 *   Dəstəkləyici proqram təminatının olmaması
 *   Zədələnmiş fayl
 *   Virus səbəbindən yoluxmuş fayl
 *   XER description accidentally deleted from the Windows registry
 *   Sistemdə faylları açmaq hüququ yoxdur
 *   Sisteminizdə köhnəlmiş disk
 *   Faylın genişləndirilməsinin adı dəyişdirilir
