## ACCDW faylı nədir?

.accdw uzantısına malik fayl Microsoft Access ilə yaradılmış keçid faylıdır və SharePoint serverindən Access verilənlər bazası faylını, yəni [ACCDB](/database/accdb/) yükləmək üçün keçid haqqında məlumatı ehtiva edir. Bu, istifadəçilərə uzaqdan yerləşdirilən verilənlər bazası fayllarını paylaşmağa imkan verir. ACCDW faylının açılması əlaqəli ACCDB faylını istifadəçinin kompüterində yerli surət kimi endirir. Yüklənmiş fayl defolt endirmə məkanında saxlanılır və ona naviqasiya etməklə daxil olmaq olar. Adətən, bu fayllar müştəri verilənlər bazasına istinad edən SiteServer.accdb adı ilə yüklənir və saxlanılır.

## ACCDW Fayl Format - Ətraflı Məlumat

ACCDW faylı Access Xidmətlərinin yerləşdiyi SharePoint saytına keçid təmin edən XML faylıdır. Bu, yalnız qısa yoldur və onları işə salmaq üçün internet bağlantısı tələb olunur. Onlayn halda, SharePoint onu daha sonra oflayn etmək seçimini təmin etmək üçün yerli olaraq keşlənir.
