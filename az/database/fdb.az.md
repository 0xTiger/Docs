## FDB faylı nədir?

.fdb faylı Microsoft Dynamics NAV-da verilənlər bazası faylları üçün istifadə olunan fayl uzantısıdır. .fdb fayl uzantısı Firebird Database File mənasını verir və bu, Microsoft Dynamics NAV tərəfindən istifadə edilən əsas verilənlər bazası mühərriki olan Firebird Database Management System tərəfindən istifadə edilən fayl formatıdır. .fdb faylı maliyyə məlumatları, inventar məlumatları, müştəri məlumatları və s. daxil olmaqla, NAV sisteminin bütün məlumatları, cədvəlləri və strukturlarını ehtiva edir. Məlumatların bütövlüyünü təmin etmək və hər hansı problem yarandıqda məlumatları bərpa etmək üçün .fdb faylının ehtiyat nüsxəsini saxlamaq vacibdir. .fdb faylı ixrac, idxal və həmçinin replikasiyada istifadə edilə bilər.

## Microsoft Dynamics NAV ilə əlaqə

FDB faylları Microsoft Dynamics NAV-da verilənlər bazası fayllarıdır. Microsoft Dynamics NAV Microsoft tərəfindən hazırlanmış müəssisə resurs planlaması (ERP) proqramıdır. O, kiçik və orta biznes üçün nəzərdə tutulmuşdur və maliyyə menecmenti, təchizat zəncirinin idarə edilməsi, istehsal, layihənin idarə edilməsi və s. daxil olmaqla, biznesin idarə edilməsi funksiyalarının geniş spektrini təmin edir. O, Microsoft Dynamics platformasında qurulub və Office və Outlook kimi digər Microsoft məhsulları ilə tam inteqrasiya olunub. Dynamics NAV-a həm veb müştəri, həm də Windows müştərisi vasitəsilə daxil olmaq olar və o, həmçinin C/AL adlı öz inkişaf mühitindən istifadə edərək fərdiləşdirilə və digər sistemlərlə inteqrasiya oluna bilər. İstehsal, topdan satış və xidmət sahələrində şirkətlər tərəfindən istifadə olunur.

## .fdb faylını necə açmaq olar?

FDB faylı adətən Microsoft Dynamics NAV inkişaf mühitindən və ya Microsoft Dynamics NAV müştərisindən istifadə etməklə açılır və idarə olunur.

Microsoft Dynamics NAV-da .fdb faylını açmaq üçün addımlar bunlardır:

1. Microsoft Dynamics NAV inkişaf mühitini açın.
2. Fayl menyusuna klikləyin və Açıq seçin və ya Ctrl + O düyməsini basın.
3. .fdb faylının saxlandığı yerə gedin.
4. .fdb faylını seçin və Açıq düyməsini basın.

Siz həmçinin NAV müştərisindən istifadə edərək verilənlər bazasına qoşulmaqla .fdb faylını aça bilərsiniz.

1. NAV müştərisini açın.
2. Fayl menyusuna klikləyin və Qoşul seçin
3. Server sahəsinə .fdb faylının yerləşdiyi serverin adını və ya IP ünvanını daxil edin.
4. Müvafiq Verilənlər Bazasının Adı və Etibarnamələri daxil edin
5. Bağla düyməsini basın.

.fdb fayllarını Firebird verilənlər bazası idarəetmə sisteminin idarəetmə proqramı ilə açmaq da mümkündür, çünki bu, Microsoft Dynamics NAV tərəfindən istifadə edilən əsas verilənlər bazası mühərrikidir.
