## DB3 faylı nədir?
DB3 faylı sadə fayllardan istifadə edərək verilənlər bazası yaradan, yüngül, müstəqil verilənlər bazası proqramı olan SQLite proqramı tərəfindən yaradılmış verilənlər bazası faylıdır; verilənlər bazası anatomiyasını və məlumat qeydlərini ehtiva edir; SQL istifadə edərək strukturlaşdırılmış məlumatların alınması və ya saxlanması üçün adətən istifadə olunur. Bu fayllar ağıllı cihazlarda və ya qeydlərin saxlanması və ya digər məlumatların idarə edilməsi tələb olunduğu yerlərdə, lakin yer az olan mühitdə istifadə edilə bilər.


## DB3 Fayl Formatı
DB3 fayl formatı daxili verilənlər bazası üçün məşhur seçim olan RDBMS (Relational Database Management System) SQLite ilə əlaqələndirilir. SQLite_3-də spesifikasiyasında müəyyən edilmiş hər hansı xüsusi fayl uzantısı yoxdur, o, [.dbf](/database/dbf/) və [.sqlite](/database/sqlite/) daxil olmaqla genişlənmələrdən istifadə edə bilər.

### Verilənlər Bazasının Secification
Commonly SQLite, version 3 related db file format can be considered as the DB3 file format used as the publicly documented native format for the SQLite database engine since June 2004. DB3 fayl formatı, böyük-endian və kiçik-endian arxitekturaları və ya 32-bit və 64-bit sistemlər arasında ötürülə bilən çarpaz platformadır. Bu xüsusiyyətlər DB3-ü proqram fayl formatı kimi məşhur seçimə çevirir. Əsas SQLite_3 verilənlər bazası (DB3) faylı bir və ya bir neçə səhifədən ibarətdir. Bütün səhifələrin bütün ölçüləri eyni verilənlər bazasında bərabərdir. Baytdakı səhifə ölçüsü 512 və 65536 daxil olmaqla ikinin gücüdür.
