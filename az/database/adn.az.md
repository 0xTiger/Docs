## ADN faylı nədir?

.adn uzantılı fayl əvvəlcədən saxlanmış parametrlərlə Access layihə faylları yaratmaq üçün istifadə edilən Microsoft Access boş şablon faylıdır. Microsoft Access layihə faylları ADP faylları kimi saxlanılır və verilənlər bazası bağlantısı və layihənin başlanğıc parametrləri kimi məlumatları ehtiva edir. ADN faylları Microsoft Access-in 2007 və 2010-cu versiyaları üçün mövcuddur. Microsoft Access bu ADN fayllarına əlavə olaraq [MDB](/database/mdb/), [ACCDB](/database/accdb/), [ACCDC](/database/accdc/), [ACCDE](/database/accde/) və bir çox başqa fayl formatlarını dəstəkləyir. ADN faylları Office 365 və Microsoft Access istifadə edərək açıla bilər.

## ADN Fayl Format - Ətraflı Məlumat

ADN ikili fayllar kimi diskdə saxlanılan verilənlər bazası şablon fayllarıdır. ADN fayllarının daxili fayl formatı təfərrüatları tərtibatçının istinadı üçün əlçatan deyil.
