## FP5 faylı nədir?

An FP5 file is a database file created with FileMaker Pro version 5 or 6. O, istifadəçi və təşkilati məlumatları cədvəllərin içərisində qeydlər şəklində saxlayır. FileMaker Pro Macintosh və Windows əməliyyat sistemlərini dəstəkləyən verilənlər bazası idarəetmə sistemidir. FP5 o vaxt mövcud FileMaker Pro versiyası tərəfindən dəstəklənən köhnə fayl formatıdır. Ən son FileMaker Pro verilənlər bazası formatı FileMaker Pro v 12-nin işə salınması ilə təqdim edilmiş [FMP12](/database/fmp12/)-dir. Yaxşı istifadə edilən digər verilənlər bazası fayl formatı [FP7](/database/fp7/) idi və FMP12 ilə əvəz edilmişdir.

## FP5 Fayl Format - Ətraflı Məlumat

FP5 fayl formatının daxili fayl formatı təfərrüatları tərtibatçının arayışı üçün ictimaiyyətə açıq deyil. FileMaker Pro-nun ətraflı təlimatlarından istifadə edərək [FP5 and FP7 can be upgraded to latest FMP12](https://fmhelp.filemaker.com/help/16/fmp/en/index.html#page/FMP_Help/converting-files.html) kimi köhnə FileMaker Pro verilənlər bazası fayl formatları.
