## ACCFT faylı nədir?

.accft uzantısı olan fayl verilənlər bazası cədvəlləri üçün sahə məlumat növlərinin tərifini saxlayan Microsoft Access şablon faylıdır. Microsoft Access cədvəllərə əsaslanan verilənlər bazası, formalar, sorğular və görünüşlər yaratmaq üçün istifadə edilən masa üstü Relational Database Management System (RDBMS) sistemidir. ACCFT faylı cədvəl sahələri üçün məlumat növlərini müəyyən etmək üçün istifadə edilə bilər və verilənlər bazası cədvəlləri bu şablon faylından başlanğıc nöqtəsi kimi yaradıla bilər. Yeni cədvəllər yaratmaq üçün Microsoft Access 2016 ilə açıla bilər. ACCFT faylları MIME növü **application/msaccess** tərəfindən tanınır.

## ACCFT fayl formatı

ACCFT faylları fayl formatı spesifikasiyası ictimaiyyətə açıq olmayan ikili fayllardır. Başlıq məlumatları, fayl strukturu, altbilgilər və s. kimi daxili detalları hazırda mövcud deyil.
