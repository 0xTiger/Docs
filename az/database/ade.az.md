## ADE faylı nədir?

.ade uzantısı olan fayl Microsoft Access ADP layihə faylında olan məlumatın tərtib edilmiş çıxışını ehtiva edən Microsoft Access Layihə faylıdır. Visual Basic for Applications (VBA) xaricində Access layihə faylındakı məlumat ADE fayllarında tərtib edilmiş formada mövcuddur. Faylın ölçüsünü azaltmaq və Access-də performansı yaxşılaşdırmaq üçün verilənlər bu fayllarda sıxılmış formatda saxlanılır. ADE Access layihəsi faylının tərtib edilmiş çıxışı olduğundan, layihənin bir hissəsi olan istənilən VBA mənbə kodu çıxışın bir hissəsi olmasına icazə verməməklə qorunur. ADE faylları Microsoft Access 365-də açıla bilər.

## ADE Fayl Format - Ətraflı Məlumat

ADE ikili fayllar kimi diskdə saxlanılan Access Database fayllarıdır. Bu faylların daxili fayl strukturu məlum deyil.

The ADE files can create issues in opening based on the version of Microsoft Access that has been used to create these files. Access databases that are using the 64-bit version of Microsoft Access 2010 and that are compiled as MDE, [ACCDE](/database/accde/), or ADE files have to be recompiled in Microsoft Access 2021 Service Pack 1 (SP1) to work correctly with Access 2010 SP1. Bu problem Access 2010 SP1-in VBE7.dll faylının daha yeni versiyasından (versiya 7.00.1619) istifadə etdiyinə görə baş verir.
