## FP7 faylı nədir?

A file with .fp7 extension is a database file created with FileMaker Pro versions 7-11. O, məlumatları hər bir qeydin bir və ya daha çox sahənin olduğu qeydləri olan cədvəllərdə saxlayır. FP7 faylları şəxsi verilənlər bazası məlumatları və təşkilati qeydlər daxil olmaqla böyük məlumatları saxlamaq imkanına malikdir. FileMaker Pro-nun daha yeni versiyası FileMake Pro versiyası 12-nin işə salınması ilə təqdim edilmiş [FMP12](/database/fmp12/)-dir. FP7 faylı mətn məlumatlarına əlavə olaraq, formalar, diaqramlar və digər verilənlər bazası obyektlərini də saxlaya bilər. FP7 faylları FileMaker Pro Advanced ilə açıla bilər.

## FP7 Fayl Format

FP7 faylları ikili fayllar kimi diskdə FileMaker-in xüsusi fayl formatında saxlanılır. FP7 fayl formatının daxili fayl formatı təfərrüatları ictimaiyyətə açıq deyil.
