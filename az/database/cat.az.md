## CAT faylı nədir?

CAT faylı müxtəlif kompüter təhlükəsizliyi və kriminal proqram alətləri hazırlayan Rusiya proqram şirkəti ElcomSoft Co. Ltd tərəfindən hazırlanmış Advanced Disk Catalog proqramına aiddir. Onun bəzi məhsullarına parolun bərpası, mobil cihazın əldə edilməsi və bulud çıxarılması üçün proqramlar daxildir. Şirkət 1990-cı ildə yaradılıb və Rusiya və ABŞ-da ofisləri var. Onlar məhkəmə-tibbi proqram təminatı ilə yaxşı tanınırlar və ümumilikdə onların proqram təminatı həm özəl şirkətlər, həm də dövlət qurumları tərəfindən məhkəmə-tibbi analiz və araşdırmalar aparmaq üçün istifadə olunur.

Advanced Disk Catalog is a cataloguing program for Windows XP/2003. ADC-nin çox yönlü olması ondan irəli gəlir ki, o, verilənlər bazası mühərrikinə etibar etmir və onu yüksək səmərəli və yığcam edir. Onun istifadəçi interfeysi Windows Explorer-ə bənzəyir, sabit disklər (şəbəkə ötürücüləri daxil olmaqla), disketlər, optik disklər, ZIP və JAZ diskləri və s. kimi müxtəlif növ medialarda saxlanılan məlumatların asan təşkilinə imkan verir.

## CAT fayllarını necə açmaq olar?

Elcomsoft Advanced Disk Catalog proqramını yükləyin. Windows-da, fayl tədqiqatçısının üzərinə sağ klikləyib kontekst menyusu ilə aç seçimini edə bilərsiniz, cat faylını açmaq üçün Elcomsoft Advanced Disk Catalog proqramının yolunu göstərin.
