## فایل GCF چیست؟

یک فایل .gcf یک فایل کش بازی است که توسط پلتفرم بازی Steam استفاده می شود. این شامل داده های بازی مانند بافت ها، مدل ها و فایل های دیگری است که توسط بازی استفاده می شود. این فایل‌ها در رایانه کاربر ذخیره می‌شوند و برای کاهش حجم داده‌هایی که باید هنگام به‌روزرسانی یا نصب بازی دانلود شوند، استفاده می‌شوند. از فایل‌های .gcf نیز می‌توان برای ایجاد پشتیبان‌گیری از نصب بازی یا انتقال بازی بین رایانه‌ها استفاده کرد.

فایل‌های GCF را می‌توان توسط کتابخانه برنامه‌نویسی C++ منبع باز، **HLLib** خواند و نوشت.

## فرمت فایل GCF

فایل های GCF به عنوان فایل های باینری روی دیسک ذخیره می شوند. GCF در ابتدا به عنوان مخفف Grid Cache File استفاده می شد (نام رمز اولیه Steam Grid بود)، اما بعداً به عنوان Game Cache File در نظر گرفته شد. یک فایل GCF یک فایل آرشیو است که بازی های Steam را ذخیره می کند. تمام محتوای رسمی به عنوان فایل های GCF دانلود می شود. فایل های GCF را می توان بین بازی ها نیز به اشتراک گذاشت.

توجه: GCF سلف [VPK file format](/game/vpk/) است.
