## فایل XS چیست؟

فایل XS یک فایل مرجع است که در Age of Empires III (AOE3) برای تولید هوش مصنوعی (AI) استفاده می شود. در طول فرآیند کامپایل زمانی که یک فایل نقشه XML سفارشی بارگذاری می شود استفاده می شود. اگر AOE3 بازی کرده اید، از قبل می دانید که نه تنها می توانید هر نقشه ای را برای بازی انتخاب کنید، بلکه می توانید نقشه های خود را با استفاده از برنامه Random Map Script ایجاد کنید.

## چگونه نقشه هوش مصنوعی را در Age of Empires 3 ایجاد کنیم؟

 * یک برنامه ویرایش متن مانند Microsoft Notepad، Notepad++ یا TextEdit را باز کنید. این برای برنامه ریزی نقشه شما استفاده خواهد شد.
 * همه ویژگی های نقشه خود را با استفاده از زبان Random Map Script برنامه ریزی کنید.
 * برای اطمینان از درستی برنامه نویسی می توانید به آموزش های برنامه نویسی مراجعه کنید
 * فایل خود را به عنوان MyAOE3.xs ذخیره کنید و آن را به پوشه های بازی عصر امپراتوری 3 خود منتقل کنید.

### فرمت فایل XS - اطلاعات بیشتر

فایل های XS به عنوان فایل های متنی ساده ذخیره می شوند که می توانند با هر ویرایشگر متنی باز و ویرایش شوند.
