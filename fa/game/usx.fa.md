## فایل USX چیست؟

یک فایل USX حاوی مش های Unreal Engine است که اشیا/مدل های سه بعدی از پیش طراحی شده برای تولید محیط هستند. مدل‌های فایل USX موقعیت، جهت و اندازه‌های متفاوتی دارند، اما اینها ثابت می‌مانند. مش های استاتیک را می توان برای عملکرد بهینه کرد و می توان از آنها برای ایجاد سطوح مختلف جزئیات (LOD) برای بهبود عملکرد بازی استفاده کرد. آنها همچنین می توانند برای ایجاد حجم های برخورد استفاده شوند و می توان از آنها به عنوان بخشی از شبیه سازی های فیزیک استفاده کرد.

فایل های USX را می توان با ویرایشگر سطح Unreal UnrealEd ایجاد کرد.

## فرمت فایل USX - اطلاعات بیشتر

فایل های USX به عنوان فایل های باینری روی دیسک ذخیره می شوند. جزئیات ساختار فایل داخلی فرمت فایل USX به صورت عمومی برای مرجع توسعه‌دهنده در دسترس نیست.
