## فایل NSP چیست؟

فرمت فایل NSP در درجه اول با کنسول Nintendo Switch مرتبط است. NSP مخفف Nintendo Submission Package است. این فرمت فایلی است که توسط نینتندو برای توزیع و نصب بازی ها، به روز رسانی ها و DLC (محتوای قابل دانلود) در سیستم نینتندو سوییچ استفاده می شود.

فایل‌های NSP اساساً محفظه‌هایی هستند که حاوی تمام داده‌ها و دارایی‌های لازم برای بازی یا محتوای خاص هستند. این شامل فایل های اجرایی بازی، گرافیک، صدا و هر فایل اضافی مورد نیاز برای اجرای بازی است. فایل‌های NSP را می‌توان از طریق راه‌های مختلفی از جمله فروشگاه اینترنتی نینتندو یا نرم‌افزار سفارشی homebrew روی نینتندو سوییچ نصب کرد.

فایل‌های NSP معمولاً رمزگذاری یا با امضای دیجیتال امضا می‌شوند تا از توزیع یا دستکاری غیرمجاز جلوگیری شود. این تضمین می‌کند که فقط کپی‌های قانونی بازی‌ها یا محتوا می‌توانند روی کنسول نینتندو سوییچ نصب و اجرا شوند.

## چگونه فایل NSP را باز کنیم؟

فایل‌های NSP برای نصب و اجرا بر روی کنسول نینتندو سوییچ طراحی شده‌اند، بنابراین نمی‌توان آنها را مستقیماً روی رایانه یا دستگاه‌های دیگر بدون شبیه‌سازی نرم‌افزار یا سخت‌افزار مناسب باز یا اجرا کرد.

با این حال، ابزارها و ابزارهای نرم افزاری کمی وجود دارند که می توانند فایل های NSP را برای اهداف مختلف، مانند استخراج یا دستکاری محتوای درون فایل ها، مدیریت کنند. در اینجا چند نمونه آورده شده است:

- **Hactool:** Hactool یک ابزار خط فرمان است که به شما امکان می دهد محتویات فایل های NSP را مشاهده کنید، فایل های فردی را استخراج کنید یا فایل ها را رمزگشایی/رمزگذاری کنید. این در درجه اول برای توسعه خانگی یا اهداف تحقیقاتی استفاده می شود.
- **NUT:** NUT is graphical user interface (GUI) tool built on top of Hactool. It provides more user-friendly interface for managing NSP files, including the ability to extract files, display metadata and manage updates and DLC.
- **Tinfoil:** Tinfoil is homebrew application for Nintendo Switch that can install NSP files from various sources, including USB, SD card or network. It also has features like title management, firmware updates and more.

## فایل NSP شامل چه چیزی است؟

فایل NSP (بسته ارسال نینتندو) معمولاً شامل اجزای زیر است:

- **قابل اجرا بازی:** فایل NSP حاوی فایل اجرایی اصلی بازی است که وظیفه اجرای بازی بر روی کنسول نینتندو سوییچ را بر عهده دارد.
- ** دارایی های بازی: ** این شامل فایل های مختلفی مانند گرافیک، صدا، فیلم و سایر دارایی های رسانه ای مورد نیاز برای جلوه های بصری و صوتی بازی است.
- **Metadata:** The NSP file contains metadata information about game such as its title, version number, publisher, release date, supported languages and other relevant details.
- **داده های بازی:** فایل های NSP همچنین داده های بازی را ذخیره می کنند، از جمله فایل های ذخیره شده بازی، تنظیمات پیکربندی و هر فایل اضافی مورد نیاز برای عملکرد صحیح بازی.
- **DLC (محتوای قابل دانلود):** اگر فایل NSP شامل DLC باشد، حاوی محتوای اضافی است که می تواند به بازی پایه اضافه شود. این می تواند شامل سطوح جدید، شخصیت ها، آیتم ها یا ویژگی های دیگری باشد که تجربه گیم پلی را گسترش می دهد.
- **Updates and Patches:** NSP files may include updates or patches to game, which can provide bug fixes, performance improvements, new features or other enhancements to original game.

## فرمت فایل NSP چیست؟

فرمت فایل NSP مورد استفاده نینتندو برای کنسول نینتندو سوییچ یک فرمت کانتینری است. این در اصل بسته ای است که چندین فایل و داده مربوط به بازی یا محتوا را در خود جای می دهد. فرمت فایل NSP از ساختار و سازماندهی خاصی پیروی می کند تا از سازگاری و نصب مناسب بر روی سیستم نینتندو سوییچ اطمینان حاصل کند.

فرمت فایل NSP به صورت عمومی توسط نینتندو مستند نشده است، زیرا اختصاصی است و برای استفاده با نرم افزار و سخت افزار رسمی آنها در نظر گرفته شده است. با این حال، از طریق مهندسی معکوس و تجزیه و تحلیل توسط جامعه homebrew، برخی جزئیات در مورد فرمت NSP کشف شده است.

ساختار فایل NSP معمولاً شامل موارد زیر است:

- **Header:** فایل NSP با یک بخش سرصفحه شروع می شود که حاوی اطلاعاتی در مورد فایل مانند نسخه فرمت فایل، اندازه و جزئیات رمزگذاری (در صورت وجود) است.
- **فراداده سیستم فایل:** این بخش ابرداده مربوط به ساختار فایل سیستم را در فایل NSP نگهداری می کند. ساختار دایرکتوری، نام فایل ها و ویژگی ها را تعریف می کند.
- **Content Files:** The main portion of NSP file contains actual content files, including game executable, assets, data files, DLC and updates. These files are typically compressed or encrypted to prevent unauthorized access or tampering.
- **Ticket:** The NSP file may include a ticket, which is a digital certificate that verifies legitimacy of content and authorizes its installation on Nintendo Switch console.
