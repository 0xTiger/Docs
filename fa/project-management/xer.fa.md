## فایل XER چیست؟

فرمت فایل XER یک فرمت فایل پروژه اختصاصی است که توسط برنامه برنامه ریزی و مدیریت پروژه Primavera P6 استفاده می شود. مانند فرمت فایل [MPP](/project-management/mpp/)، این فرمت فایل داده های مربوط به برنامه ریزی پروژه مانند وظایف، منابع، جدول زمانی و مواد را ذخیره می کند. Primavera توسط Oracle توسعه داده شده است و برنامه ریزی کامل پروژه را ارائه می دهد و کاربردهای خود را در چندین زمینه عملی مانند نفت و گاز، ساخت و ساز، تولید و غیره پیدا می کند. این برنامه به مدیران پروژه کمک می کند تا با یکپارچه سازی برنامه های بزرگ مقیاس و پروژه های فردی را برنامه ریزی، زمان بندی و کنترل کنند. با سیستم های مدیریت مالی و مدیریت سرمایه انسانی بسیاری از نرم افزارها و API ها می توانند فایل های XER را به سایر فرمت های فایل پروژه محبوب مانند XML و MPP بخوانند و تبدیل کنند.

## فرمت فایل XER ##

یک فایل XER یک فایل متنی است و به راحتی با یک ویرایشگر متن باز می شود. باز کردن داده های مربوط به پروژه در یک ویرایشگر متن می تواند مفید باشد، اما درک آن با ایجاد پیوندها به صورت دستی آسان نیست. این فرمت بیشترین استفاده را برای تبادل داده با سایر پایگاه های داده P6 دارد. کنترل نسبتاً پایینی روی داده هایی که از فرمت دیگری وارد شده اند، دارد. مزایای XER در فرمت XML معادل شامل توانایی انتقال کدها و تقویم‌های جهانی به سطح پروژه و کدهای نقشه، UDF، منابع و تقویم‌ها به مقادیر موجود است.


## برای باز کردن فایل XER مشکل دارید؟ ##

برخی از مشکلات رایجی که ممکن است ایجاد شود و باعث عملکرد نادرست فرمت XER شود به شرح زیر است:

 *   عدم وجود نرم افزار پشتیبانی
 *   فایل خراب
 *   فایل آلوده به ویروس
 *   XER description accidentally deleted from the Windows registry
 *   دسترسی مستقیم به سیستم برای باز کردن فایل ها وجود ندارد
 *   درایو قدیمی در سیستم شما
 *   پسوند فایل تغییر نام داد
