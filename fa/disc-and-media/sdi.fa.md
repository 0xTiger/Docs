## فایل SDI چیست؟
فایل با پسوند .sdi حاوی یک load blob، boot blob و part blob است. معمولاً توسط Windows Embedded Studio برای ایجاد دیسک های RAM یا دیسک های بوت برای بارگذاری و نصب ویندوز استفاده می شود. SDI مخفف System Deployment Image است. Windows Embedded Studio برنامه ای است که برای ایجاد تصاویر دیسک برای ویندوز استفاده می شود. در واقع این برنامه حاوی برنامه SDI File Manager و یک ابزار خط فرمان به نام **sdimgr.exe** است که می توان از هر دو برای ایجاد، ایجاد و ویرایش تصاویر SDI استفاده کرد.

## فرمت فایل SDI
فرمت فایل SDI به طور گسترده ای برای فعال کردن استفاده از یک دیسک مجازی برای بوت کردن یک سیستم استفاده می شود. برخی از نسخه‌های مایکروسافت ویندوز، **راه‌اندازی RAM** را فعال می‌کنند، که برای بارگذاری یک فایل SDI در حافظه و سپس بوت شدن از آن مهم است. فرمت فایل SDI همچنین با استفاده از PXE (محیط اجرای پیش راه‌اندازی) خود را برای راه‌اندازی شبکه فعال می‌کند. همچنین برای تصویربرداری هارد دیسک استفاده می شود.
### بخش های فایل SDI
خود فایل SDI را می توان به بخش های زیر تقسیم کرد:
- **Boot BLOB**: این شامل برنامه بوت واقعی، STARTROM.COM است. این مشابه بخش بوت دیسک سخت است.
- **Load BLOB**: این معمولاً حاوی NTLDR است و توسط BLOB راه‌اندازی راه‌اندازی می‌شود.
- ** قسمت BLOB **: این شامل زمان اجرا بوت واقعی است. همچنین شامل فایل‌های boot.ini و ntdetect.com است که باید در دایرکتوری ریشه زمان اجرا قرار گیرند.
- **Disk BLOB**: این تصویر صاف HDD است که با MBR شروع می شود. برای تصویربرداری هارد دیسک به جای بوت استفاده می شود. همچنین فقط Disk BLOB ها را می توان با ابزارهای مایکروسافت نصب کرد.
