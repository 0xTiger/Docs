## فایل XHTM چیست؟

An file with .xhtm extension is a webpage file that is written in XHTML. It uses XML 1.0 as its standard implementation and is considered a reformulation of HTML 4. مانند XHTML، یک فایل XHTM حاوی برچسب‌های استاندارد HTML است که از دستورالعمل‌های XML دقیق‌تری پیروی می‌کنند. فایل‌های XHTM برای انتشار محتوای وب‌سایت استفاده می‌شوند و معمولاً از پسوند [XHTML](/web/xhtml/) استفاده می‌کنند.

برای باز کردن فایل‌های XHTM می‌توانید از مرورگرهای وب مانند Google Chrome، Apple Safari و Mozilla Firefox استفاده کنید.

## مثال XHTM

مثال زیر از یک سند XHTM اعلان های XML را تعریف می کند.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
