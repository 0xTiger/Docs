## فایل VDISCO چیست؟

فایل VDISCO یک فایل اکتشافی است که توسط مایکروسافت ویژوال استودیو در سرویس های وب خود استفاده می شود. اطلاعاتی در مورد سرویس های وب موجود مانند URL ها، فضاهای نام و روش های آنها ارائه می دهد. فایل VDISCO شبیه فرمت فایل [DISCO](/web/disco/) است اما در زمان اجرا تولید می شود. در برنامه وب سرویس ذخیره می شود و توسط ویژوال استودیو برای کشف و استفاده پویا از خدمات وب در یک پروژه استفاده می شود. فایل های VDISCO در ترکیب با فایل [.asmx](/web/asmx/) که حاوی کد واقعی وب سرویس است استفاده می شود.

می توانید فایل VDISCO را در هر ویرایشگر متنی مانند Microsoft Notepad، Github Atom و Apple TextEdit باز کنید. همچنین می‌توان آن را با Microsoft Visual Studio 2012 و بعد از آن برای کار با آن باز کرد.

## فرمت فایل VDISCO

یک فایل VDISCO در قالب فایل [XML](/web/xml/) ذخیره و میزبانی می‌شود که یک فرمت جهانی برای ترکیب و انتشار داده‌ها است. این شامل URL سرویس، فضاهای نام و روش ها در تگ های XML استاندارد است که در آن هر تگ حاوی مقدار مربوط به اطلاعات خود است.
