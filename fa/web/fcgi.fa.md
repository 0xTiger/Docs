## فایل FCGI چیست؟

فایل FCGI یک فایل کد منبع است که حاوی کد منبع نوشته شده برای FastCGI (FCGI) است. این می تواند به چندین زبان برنامه نویسی مختلف نوشته شود و شامل پسوند استاندارد CGI است که برای رسیدگی قوی تر درخواست های سرور وب استفاده می شود. رایج ترین زبان های پشتیبانی کننده برای FCGI شامل **Perl** است که هم از کد Perl و هم از رابط خارجی قوی FastCGI API پشتیبانی می کند.

## فرمت فایل FCGI

چه چیزی یک فایل FCGI را مهم می کند؟ رابط FCGI از محاسبات توزیع شده برای رسیدگی به درخواست های وب و توزیع آنها در چندین سرور پشتیبانی می کند. این باعث می‌شود که فایل‌های FCGI بیشتر برای تعادل بار در چندین سرور استفاده شوند.
