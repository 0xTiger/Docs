## فایل FWP چیست؟

یک فایل با فایل fwp. یک فایل بایگانی است که با Microsoft Expression Web ایجاد می‌شود و شامل تمام فایل‌ها و پوشه‌های یک طرح است. Microsoft Expression Web یک محصول نرم افزار طراحی وب عمومی توسط مایکروسافت است و بر اساس ویرایشگر [HTML](/web/html/) است. از سال 2012 تولید آن متوقف شده است و توسط مایکروسافت برای دانلود رایگان در دسترس است. فایل های FWP را می توان به عنوان فرمت cab. مایکروسافت صادر و ذخیره کرد.

## فرمت فایل FWP - اطلاعات بیشتر

مشخصات فرمت فایل دقیق فایل‌های FWP برای مرجع توسعه‌دهنده در دسترس نیست. با این حال، این فایل‌های باینری هستند و می‌توانند در Microsoft Expression Studio که مدت‌ها پیش متوقف شده است، باز شوند.
