## فایل BR چیست؟

A BR file is a compressed web file generated by applying the open source data compression algorithm, **Brotli**. It is used to store webpage assets such as stylesheets ([CSS](/web/css/)), images ([SVG](/page-description-language/svg/)), [XML](/web/xml/), and scripting files ([JS](/web/js/)). Modern day websites, such as Chrome and Firefox, use BR files to reduce the page loading time, resulting in better user experience.

## فرمت فایل BR

فایل های BR فایل های وب فشرده شده ای هستند که با استفاده از الگوریتم فشرده سازی Brotli تولید می شوند. فشرده سازی Brotli یک الگوریتم فشرده سازی داده بدون تلفات است و توسط گوگل به الگوریتم فشرده سازی Zopfli توسعه داده شده است. این مبتنی بر ترکیبی از الگوریتم فشرده سازی بدون اتلاف LZ77 و کدگذاری هافمن است.

فایل های BR به دلیل اندازه کوچک آن توسط وب سرورها و شبکه های تحویل محتوا (CDN) استفاده می شوند. درخواست های ارسال شده به این سرورها منجر به فشرده سازی محتوای HTTP می شود که باعث می شود وب سایت ها سریعتر بارگذاری شوند. Brotli محبوب تر شده است و فشرده سازی بهتری نسبت به gzip ارائه می دهد.
