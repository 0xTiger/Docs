## فایل HYPE چیست؟

A HYPE file is a project file saved using the Hype software that is used for creation of web content with HTML5. این شامل عناصر وب مانند گرافیک، متن، انیمیشن، اطلاعات صحنه و سایر عناصری است که در مجموع محتوا را تشکیل می دهند. محتوای وب در یک فایل HYPE را می توان به فایل های وب صادر کرد که می توانند در یک صفحه وب منتشر شوند.

یک فایل HYPE را می توان با نرم افزار [Tumult Hype](https://tumult.com/hype/) باز کرد.

## فرمت فایل HYPE

فایل‌های HYPE در قالب فایل اختصاصی ذخیره می‌شوند و جزئیات داخلی آن‌ها به‌طور عمومی مشخص نیست. محتویات داخل یک فایل HYPE تنها زمانی قابل مشاهده است که در نرم افزار Tumult Hype باز شود.

### محتویات یک فایل HYPE

هنگامی که از برنامه Hype برای ایجاد محتوای وب برای اهداف مختلف استفاده می کنید، یک فایل HYPE ایجاد می شود. اطلاعات مربوط به انیمیشن های مبتنی بر فریم کلیدی را برای ایجاد محتویاتی مانند کارت های الکترونیکی، کتاب های الکترونیکی، ارائه ها، تبلیغات، اینفوگرافیک ها و مطالب صفحه وب ذخیره می کند. محتوای این ها توسط HTML5 ارائه می شود که به محتوای نمایش داده شده در دستگاه ها از رایانه های رومیزی گرفته تا آیفون و آیپد امکان می دهد. Tumult Hype آخرین سبک های HTML5، [CSS3](/web/css/) و [JavaScript](/web/js/) را خروجی می دهد.
