## فایل XPD چیست؟

یک فایل XPD یک فایل صفحه وب است که از یک سند XML ایجاد شده است. بیشتر اوقات، این از طریق تبدیل XSLT ایجاد می شود. [XPD file format specifications](https://www.w3.org/TR/xml-pipeline/) در W3 موجود است که راهی را برای افزایش قابلیت همکاری برنامه‌های چند فرآیندی [XML](/web/xml/) تعریف می‌کند. زبان خط لوله XML روابط پردازشی بین منابع XML را تعریف می کند.
