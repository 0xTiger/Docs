## فایل DISCOMAP چیست؟

یک فایل DISCOMAP یک فایل خروجی است که توسط Microsoft Web Services Discovery Tool، **Disco.exe** ایجاد می شود. این شامل نتایجی است که خدمات وب ASP.NET در URL خاص کشف شده است. نمونه‌های رایج چنین سرویس‌های کشف‌شده شامل مکان‌های فایل‌های .DISCO یا .WSDL است. هنگامی که ابزار Microsoft Web Discovery اجرا می شود، اطلاعاتی را بر اساس محتوای XML موجود در فایل DISCO جستجو می کند. نتایج در فایل DISCOMAP ذخیره می شود که می تواند با Microsoft Visual Studio IDE باز شود.
