## فایل KIT چیست؟

یک فایل با پسوند kit. یک فایل HTML است که با برنامه زبان برنامه نویسی [CodeKit](https://codekitapp.com/) ایجاد می شود. علاوه بر فایل‌های موجود [HTML](/web/html/) حاوی «واردات» و «متغیرها» است که آن را برای سایت‌های ثابت ایده‌آل می‌کند. CodeKit فایل‌های KIT را به HTML کامپایل می‌کند که به راحتی می‌توان از آنها به عنوان فایل‌های استاتیک وب‌سایت استفاده کرد.

## فرمت فایل KIT

فایل‌های KIT فایل‌های HTML هستند که علاوه بر این شامل واردات و متغیرها می‌شوند. این فایل‌ها به‌عنوان فایل‌های متنی ساده ذخیره می‌شوند و می‌توانند با هر ویرایشگر متن یا ویرایشگر فایل وب باز شوند.

### واردات فایل KIT

تقریباً هر نوع فایلی را می توان به یک فایل کیت وارد کرد. در زیر دستور import مورد استفاده برای وارد کردن فایل ها به فایل .kit آمده است.

```
<!-- @import "someFile.kit" -->
<!-- @import "file.html" -->
```

هنگامی که یک import به یک فایل KIT اضافه می شود و ذخیره می شود، با متن فایل در حال وارد کردن جایگزین می شود. همچنین می توانید از @include به جای @import استفاده کنید.

### واردات چندگانه در یک فایل KIT

همچنین می توانید با استفاده از یک لیست جدا شده با کاما، بیش از یک فایل را همزمان وارد کنید:

```
<!-- @import someFile, otherFile.html, ../thirdFile.kit -->
```
