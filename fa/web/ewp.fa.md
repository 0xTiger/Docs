## فایل EWP چیست؟

یک فایل EWP یک فایل پروژه است که با نرم افزار Website Builder [EWISoft](http://www.ewisoft.com/) ایجاد شده است. این به یک فایل پایگاه داده باینری که حاوی تمام اطلاعات پروژه است پیوند داده شده است. فایل EWP معمولاً برای ذخیره پیشرفت توسعه وب استفاده می شود. این یک نرم افزار طراحی وب قالب محور است که به کاربران امکان می دهد وب سایت هایی را از قالب های وب سایت ایجاد کنند.

## فرمت فایل EWP - اطلاعات بیشتر

فایل های EWP فایل های پروژه ای هستند که نیازی به کامپایل یا میزبانی روی هیچ سروری ندارند. این فایل‌ها به‌عنوان فایل‌های متنی ساده ذخیره می‌شوند و حاوی ارجاعاتی به پیشرفت توسعه وب هستند. پس از تکمیل توسعه، پروژه ها را می توان در یک وب سایت منتشر کرد.
