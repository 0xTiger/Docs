## فایل XHT چیست؟

یک فایل با پسوند xht. یک فایل وب مرتبط با نوع فایل [XHTML](/web/xhtml/) (زبان نشانه گذاری فرامتن توسعه یافته) است که خود یک فایل نشانه گذاری مبتنی بر [XML](/web/xml/) است. هر دوی این فایل ها شبیه به HTML هستند اما نحوی شبیه به XML دقیق تر دارند. فایل‌های XHT به گونه‌ای طراحی شده‌اند که ساختارمندتر باشند، شامل اسکریپت‌نویسی کمتری هستند و علاوه بر استفاده از امکانات موجود XML، عمومی هستند.

## فرمت فایل XHT

یک فایل XHT به صورت پیش فرض به عنوان یک فایل متنی ساده با رمزگذاری UTF-8 ایجاد و ذخیره می شود. این شامل کد منبع کامل یک سند XHTML است که می تواند با هر ویرایشگر متنی که از رمزگذاری یونیکد پشتیبانی می کند باز و ویرایش شود. علاوه بر ویرایشگرهای متن، فرمت فایل XHT برای اکثر مرورگرهای وب مدرن قابل درک است و با استفاده از آنها قابل باز شدن است.

## مثال XHT

مثال زیر از یک سند XHT اعلان های XML را تعریف می کند.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
