## فایل VIM چیست؟

فایل VIM یک فایل پیکربندی/تنظیمات است که توسط برنامه ویرایش متن معروف به Vim استفاده می‌شود. تنظیمات ویرایشگر مانند برجسته کردن نحو را ذخیره می کند و توسط توسعه دهندگان برنامه برای پیکربندی ظاهر ویرایشگر هنگام باز شدن استفاده می شود. ویرایشگر متن Vim بسیار قابل تنظیم است و برای ایجاد و تغییر کارآمد متن ساخته شده است. با اکثر سیستم های یونیکس به صورت vi همراه است.
