## فایل HTML0 چیست؟ ##

HTML0 فرمت کتاب الکترونیکی بومی است که توسط **Book Designer** دارای کد استاندارد HTML و CSS است که محتوا و طراحی هر صفحه را توصیف می کند. **Book Designer** رایج ترین برنامه ای است که برای کارکرد با فایل های HTML0 و تغییر اسناد متنی به فرمت های کتاب الکترونیکی متعدد است. فرمت فایل HTML0 از نرم افزارهایی پشتیبانی می کند که می توانند بر روی سیستم ویندوز نصب شوند زیرا در دسته فایل های متنی قرار دارند. فایل‌های HTML0 از HTML معمولی تشکیل شده‌اند و با استفاده از ویرایشگر متن قابل مشاهده و اصلاح هستند. فایل‌های HTML0 را نمی‌توان در مقایسه با فایل‌های معمولی HTML به فایل‌های CSS مجزا پیوند داد.

## مشکل باز کردن فایل HTML0؟ ##

 * 	زمانی که باز کردن آن برای شما مشکل است ممکن است فایل آلوده شده باشد 
 *  باید فایل را برای ویروس ها توسط اسکنر آنتی ویروس آنلاین اسکن کنید
* برخی از فایل ها برای باز کردن آنها به حقوق دسترسی نیاز دارند
* اگر سیستم‌ها منابع کافی برای باز کردن فایل‌های HTML0 ندارند، همه برنامه‌ها را ببندید و سپس دوباره باز کنید.
 *  اگر فایل را دانلود کرده باشید یا از هر ایمیلی دریافت کرده باشید، ممکن است خراب یا دارای خطا باشد
* همیشه از آخرین سیستم و درایورها استفاده کنید تا رایانه خود را ایمن تر کنید و تمام مشکلات مربوط به پسوند فایل ها را حل کنید
