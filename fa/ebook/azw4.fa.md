## فایل AZW4 چیست؟

AZW4 یک کتاب الکترونیکی چاپ آمازون است که آمازون برای نرم افزار دسکتاپ Kindle و دستگاه های Kindle eReader توسعه داده است. AZW4 شبیه [MOBI](/ebook/mobi/)، [AZW](/ebook/azw/) یا [AZW3](/ebook/azw3/) نیست. این قالب در واقع یک بسته بندی برای کتاب های الکترونیکی [PDF](/pdf/) یا MOBI با DRM است.

## فرمت فایل AZW4

The **AZW4** has been introduced as a new version of **Textbooks** software which is similar to **PDF**. An AZW4 eBook contains a complete PDF file inside, or it can be considered as a PDF file with a wrapper called AZW4. در درجه اول AZW4 توسط Kindle eReader استفاده می شود. با این حال، برنامه های دیگر نیز ممکن است از پسوند فایل .azw4 استفاده کنند. حتی برنامه های بدافزار می توانند فایل های .azw4 را ایجاد کنند. در مورد فایل‌های azw4. که از منبع مشکوک می‌آیند محتاط باشید.
