## فایل LIT چیست؟

فایل های LIT متعلق به **Microsoft Reader** باستانی است که برای خواندن کتاب های الکترونیکی استفاده می شد. کتاب‌های الکترونیکی که توسط Microsoft Reader نمایش داده می‌شوند دارای پسوند **.lit** هستند که به **Literature** کوتاه شده است. فرمت فایل LIT از یک فایل Microsoft ITOL/ITLS HTML Help 2.0 تشکیل شده است که انواع ویژگی‌های شگفت‌انگیز مانند فشرده‌سازی [LZX](/compression/lzx/)، نمایش باینری مواد نشانه‌گذاری، و داده‌های کمکی اختیاری دلخواه مرتبط با هر جریان را ارائه می‌دهد. متن کتاب تعداد دلخواه جریان های نشانه گذاری OEBPS 1.0 و زیر مجموعه ای از OEBPS 1.0 CSS است. تصاویر [GIF](/image/gif/)، [JPEG](/image/jpeg/) و [PNG](/image/png/) هستند.

## مایکروسافت ریدر چقدر فایل های LIT را نمایش می دهد

The **Microsoft Reader** is an application for reading e-books that have been discontinued by Microsoft. The first version of Microsoft Reader was released in August 2000, which used its own LIT format. It was available for PDAs and Windows computers. The name of the app was also used later for an application (which was initially released with Windows 8) for reading [PDF](/pdf/) and [XPS](/page-description-language/xps/) files. The Microsoft Reader application was discontinued in 2018.
