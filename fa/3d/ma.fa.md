## فایل MA چیست؟

یک فایل با پسوند .ma یک فایل پروژه سه بعدی است که با برنامه Autodesk Maya ایجاد شده است. این شامل فهرست بزرگی از دستورات متنی برای مشخص کردن اطلاعات مربوط به فایل است. فایل **.ma** را می توان در هر ویرایشگر متنی باز کرد و ویرایش کرد تا در صورت خراب شدن فایل، مشکلات مربوط به دستورات را برطرف کند. این فایل ها حاوی اطلاعاتی برای تعریف اطلاعات صحنه سه بعدی مانند هندسه، نورپردازی، انیمیشن و رندر هستند.

## فرمت فایل MA

فایل‌های MA بر خلاف فایل‌های مگابایتی که در قالب فایل باینری روی دیسک ذخیره می‌شوند، در قالب متن ASCII روی دیسک ذخیره می‌شوند. یک مرجع دقیق برای فرمت فایل MA در [Autodesk Maya Documentation](https://download.autodesk.com/us/maya/2010help/index.html?url=Glossary_M_ma_file_format.htm,topicNumber=d0e192001) موجود است و می توان برای مرجع توسعه دهنده به آن مراجعه کرد.

### سربرگ فایل MA

هدر فایل MA با بخشی از نظرات شروع می شود که اطلاعات ایجاد فایل و تاریخ اصلاح را می دهد. خوانندگان فایل مایا این بلوک را نادیده می گیرند زیرا فقط برای اهداف اطلاعاتی استفاده می شود. یک هدر باید با شش کاراکتر اول به عنوان //Maya شروع شود.

هدر فایل ASCII به شکل زیر است.

```
//Maya ASCII 1.0 scene
//Name: solstice.ma
//Last modified: Sun, Dec 21, 97 10:18:26 AM
```
### مراجع فایل

بخش مراجع فایل یک فایل .ma حاوی اطلاعاتی در مورد سایر فایل های مایا است که در این فایل به آنها ارجاع داده شده است. هر فایل ارجاعی را می توان با استفاده از یک دستور فایل واحد که در همان فایل موجود است، خواند. نحو دستور فایل زمانی که برای ارجاع استفاده می شود عبارت است از:

```
file -r -rpr prefixString fileName;
```
یا

```
file -r -ns nameSpace fileName
```
در اینجا یک رشته نمونه است.

```
file -r -rpr "solar" "/u/sally/work/solar.ma";
```
این مثال نشان می دهد که شیء خورشید موجود در فایل `solar` در این فایل با استفاده از نام solar_sun قابل دسترسی است.

### الزامات

بخش Requirements یک فایل .ma از یک سری دستورات مورد نیاز تشکیل شده است و اطلاعات می مانند نسخه و اطلاعات افزونه را که برای خواندن فایل ها لازم است، می گوید. نمونه ای از بخش الزامات به شرح زیر است.

```
requires maya "2.0";
requires specialPlugIn "1.2";
```


بخش بعدی الزامات را مشخص می کند. این شامل یک سری دستورات نیاز است. این بخش از فایل به مایا می گوید که چه نرم افزاری برای بارگذاری صحیح فایل مورد نیاز است. به طور خاص، چه نسخه ای از مایا، و چه پلاگین هایی.
