## فایل DRC چیست؟

یک فایل با پسوند .drc یک فرمت فایل سه بعدی فشرده است که با کتابخانه Google Draco ایجاد شده است. Google دراکو را به عنوان کتابخانه منبع باز برای فشرده سازی و از حالت فشرده سازی شبکه های هندسی سه بعدی و ابرهای نقطه ای ارائه می دهد و ذخیره و انتقال گرافیک های سه بعدی را بهبود می بخشد. داده های ورودی را رمزگذاری می کند و آن را به عنوان فایل .drc ذخیره می کند. در انتهای دریافت، API فایل‌های .drc را می‌خواند و می‌تواند آنها را به‌عنوان فایل‌های [PLY](/3d/ply/) یا [OBJ](/3d/obj/) خروجی دهد. فایل خروجی فشرده شده کاربران را قادر می سازد تا برنامه ها را سریعتر دانلود کنند و بارگذاری سریع گرافیک های سه بعدی را در مرورگرها فراهم کنند.

## فرمت فایل DRC

File format details for DRC are not available publicly. Google has hosted the Draco library on [GitHub](https://github.com/google/draco/tree/master/docs) for developer's reference. It provides encoders and decoders for creating and converting DRC files.
