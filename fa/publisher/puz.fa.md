## فایل PUZ چیست؟

یک فایل با پسوند puz. یک آرشیو فشرده است که حاوی یک سند استاندارد Microsoft Publisher [.pub](/publisher/pub/) است. در نتیجه فشرده سازی، اندازه آن کوچکتر است و به راحتی از طریق اینترنت یا دستگاه های دیگر منتقل می شود. فایل های PUZ را می توان با استفاده از ابزار Microsoft Unpack باز یا باز کرد.

## فرمت فایل PUZ

فایل PUZ یک آرشیو فشرده است که حاوی فایل استاندارد PUB به همراه فایل های دیگر است. این شامل تمام فایل های مورد نیاز برای باز کردن، نمایش و چاپ نشریه است.

یک فایل PUZ حاوی موارد زیر است.

  * Unpack.exe - یک ابزار نرم افزاری برای باز کردن بسته بندی فایل های فشرده
  * `ReadMe.txt` - a description files
  * فایل .puz فشرده - حاوی انتشار است و ممکن است شامل فونت های جاسازی شده و گرافیک های مرتبط باشد.

### باز کردن بسته بندی فایل PUZ

فایل های بسته بندی شده Publisher را می توان با استفاده از ابزار Unpack.exe باز کرد. این فایل PUZ بسته بندی شده را به فهرست انتخابی که توسط کاربر انتخاب شده است استخراج می کند.
