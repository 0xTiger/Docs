فرمت فایل تصویری یک روش استاندارد برای سازماندهی و ذخیره تصاویر در دستگاه هایی مانند رایانه، تبلت و گوشی های هوشمند است. تصاویر دیجیتال داده های تصویر را در شبکه دو بعدی پیکسل ذخیره می کنند که در آن هر پیکسل بر حسب تعداد بیت ها نمایش رنگ است. انواع فایل های تصویری به فرمت های تصویر برداری و فرمت های تصویر شطرنجی طبقه بندی می شوند. تصاویر سه بعدی نوع دیگری از فرمت فایل تصویر برداری است که برای مدیریت تصاویر سه بعدی استفاده می شود.

## فرمت های شطرنجی ##

**رستر گرافیک** تصاویر دیجیتالی هستند که از داده های پیکسلی برای نمایش رنگ ها تشکیل شده اند. اینها رایج ترین انواع تصویر برای تصاویر گرافیکی هستند که برای وب و همچنین عکس های دیجیتال استفاده می شوند. برخی از تصاویر شطرنجی را می توان برای کاهش اندازه فایل تصویر فشرده کرد. پسوندهای رایج فایل تصویر شطرنجی و فرمت های فایل آنها شامل **[BMP](/image/bmp/)** (فایل تصویر بیت مپ)، **[PNG](/image/png/)** (گرافیک شبکه قابل حمل) و **[GIF](/image/gif/)** ( فایل تبادل گرافیکی).

## فرمت های برداری ##

**Vector images** are defined by 2D points, instead of pixels, which are connected to give a geometric shape to the image. The points have properties that define the direction of paths, color, shape, curve, thickness, and fill. Common vector image file extensions and their file formats include **[SVG](/page-description-language/svg/)** (Scalable Vector Graphics), **[EPS](/page-description-language/eps/)** (Encapsulated PostScript language) and **[PDF](/pdf/)** (Portable Document Format).

آیا سوالات مربوط به فرمت های فایل تصویری دارید؟ برای بهره مندی از دانش به اشتراک گذاشته شده توسط کارشناسان فرمت فایل، به انجمن ما [forums](https://forum.fileformat.com/c/image/9) بروید.
