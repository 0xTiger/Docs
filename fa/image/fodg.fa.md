## فایل FODG چیست؟

یک فایل با پسوند .fodg یک فرمت فایل طراحی OpenOffice Apache برای ذخیره عناصر طراحی است. این بر اساس مشخصات فرمت فایل است که توسط استانداردهای پیشرفته اطلاعات ساختاری (OASIS) مشخص شده است. فرمت فایل مشابه دیگر برای گرافیک OpenOffice، [ODG](/image/odg/) است که عناصر طراحی را به صورت تصویر برداری ذخیره می کند. فایل های FODG را می توان با OpenOffice و LibreOffice باز کرد. برای مثال، سایر قالب‌های فایلی که توسط OpenOffice پشتیبانی می‌شوند، عبارتند از: [ODT](/word-processing/odt/)، ODF، [ODP](/presentation/odp/) و [ODS](/spreadsheet/ods/).

## فرمت فایل FODG

FODG is based on the OpenDocument's XML file format that conforms to OASIS OpenDocument Format ISO/IEC 26300. دارای Internet Media Type application/vnd.oasis.opendocument.graphics است و همچنین با org.oasis-open.opedocument public.composite-content مطابقت دارد. ساختار XML برای همه انواع سند مانند صفحه گسترده، فایل های طراحی و اسناد متنی رایج است. اسناد OpenOffice از جداسازی نگرانی‌ها با جداسازی محتوا، سبک‌ها، ابرداده‌ها و تنظیمات برنامه در چهار فایل XML مجزا بهره می‌برند.

`<office:document-content> ` - محتوای سند و سبک های خودکار مورد استفاده در محتوا.
`<office:document-styles> ` - سبک های مورد استفاده در محتوای سند و سبک های خودکار مورد استفاده در خود سبک ها.
`<office:document-meta>` - Document meta information, such as the author or the time of the last save action.
`<office:document-settings> ` - تنظیمات خاص برنامه، مانند اندازه پنجره یا اطلاعات چاپگر.
