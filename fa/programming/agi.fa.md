## فایل AGI چیست؟

فایل AGI یک فایل اسکریپت است که توسط سیستم منبع باز تلفنی، استریسک استفاده می شود. این شامل اطلاعاتی است که می تواند برای خودکارسازی فرآیندها و برنامه شماره گیری ستاره استفاده شود. با استفاده از فایل های اسکریپت AGI، می توان با پایگاه های داده رابطه ای مانند PostgreSQL یا MySQL ارتباط برقرار کرد. علاوه بر این، از اسکریپت های AGI می توان برای دسترسی به منابع خارجی دیگر نیز استفاده کرد. اسکریپت‌های AGI می‌توانند کنترل پلان شماره‌گیری را به یک اسکریپت AGI خارجی تغییر دهند و استریسک را قادر به انجام وظایف پیچیده کنند.

## فرمت فایل AGI - اطلاعات بیشتر

فایل های اسکریپت AGI به عنوان فایل های متنی ذخیره می شوند و می توانند در هر ویرایشگر متنی برای ایجاد تغییرات باز شوند.

### الگوی استاندارد ارتباطات AGI

اسکریپت AGI متغیرها و مقادیر آنها را که توسط استریسک به آن ارسال می شود بارگذاری می کند. ظاهر رایج این متغیرها به شرح زیر است.

```
agi_request: test.py
agi_channel: Zap/1-1
agi_language: en
agi_callerid:
agi_context: default
agi_extension: 123
agi_priority: 2
```

این متغیرها توسط یک خط خالی ارسال شده توسط استریسک دنبال می شوند که نشان می دهد اسکریپت AGI می تواند اکنون کنترل شماره گیری را در دست بگیرد.

### زبان برنامه نویسی برای فایل های AGI Script

اسکریپت های AGI معمولاً می توانند در Perl، [PHP](/programming/php/)، [C](/programming/c/)، Pascal یا Bourne Shell نوشته شوند.
