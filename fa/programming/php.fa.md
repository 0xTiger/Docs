## فایل PHP چیست؟
یک فایل با پسوند php به زبان برنامه نویسی منبع باز اشاره دارد که برای نوشتن اسکریپت های سمت سرور استفاده می شود تا روی سرور وب اجرا شود. این پرکاربردترین زبان برنامه نویسی وب است که معمولاً برای توسعه برنامه های کاربردی وب در مقیاس بزرگ استفاده می شود. بزرگترین سیستم وبلاگ نویسی در وب یعنی وردپرس و بزرگترین شبکه اجتماعی فیس بوک بر اساس زبان PHP هستند. این نرم افزار با پلتفرم های مختلف مانند ویندوز، لینوکس، یونیکس، مک او اس ایکس و غیره سازگار است. همچنین تقریباً بر روی تمام سرورهایی که امروزه استفاده می شوند مانند آپاچی، IIS و غیره با موفقیت اجرا می شود. PHP از طیف گسترده ای از پایگاه های داده از جمله MySQL پشتیبانی می کند.

## فرمت فایل PHP ##

یک PHP می تواند حاوی متن، CSS، [HTML](/web/html/)، جاوا اسکریپت و کد PHP باشد. کد پی اچ پی بر روی سرور اجرا می شود و نتیجه به صورت HTML ساده به مرورگر بازگردانده می شود. فایل‌های PHP را می‌توان با هر ویرایشگر متنی باز کرد و در جای خود ویرایش کرد، اگرچه برنامه‌هایی مانند Adobe Dreamweaver، ابزارهای توسعه PHP Eclipse راه مناسبی برای نوشتن و تغییر کد PHP ارائه می‌دهند.

### نحو ###

یک اسکریپت PHP روی سرور اجرا می شود و نتیجه ساده HTML به مرورگر ارسال می شود. یک اسکریپت PHP با شروع می شود<?php and ends with ?>

### مثال PHP ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
