## فایل UNITY چیست؟

فایل Unity یک فایل صحنه سه بعدی است که با موتور بازی سه بعدی **Unity** ایجاد شده است. در ترکیب با صحنه های دیگر برای ایجاد دنیای مجازی در حین ویرایش استفاده می شود. این شامل عناصر صحنه سه بعدی مانند اشیا، محیط ها، تزئینات، دوربین ها، اطلاعات زاویه ها، سناریوهای نورپردازی و موانع است.

فایل های Unity را می توان با Unity Technologies Unity باز کرد. Unity همچنین می‌تواند فایل‌های Unity [.fbx](/3d/fbx/)، [.dae](/3d/dae/) (Collada)، [.3ds](/3d/3ds/)، [.dxf](/cad/dxf/) و [.obj](/3d/obj/) را بخواند.

## فرمت فایل UNITY

فایل های Unity در قالب فایل باینری روی دیسک ذخیره می شوند. اینها در هنگام ویرایش در کنار صحنه های دیگر تولید و استفاده می شوند. Unity برای توسعه بازی‌ها برای ویندوز و همچنین Mac OS X استفاده می‌شود و می‌توان از آن برای ساخت بازی‌های سه بعدی برای th eweb، Nintendo Wii، iPhone یا برنامه‌های مستقل استفاده کرد.
