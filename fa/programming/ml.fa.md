## فایل ML چیست؟

فایل ML یک فایل متنی است که با زبان برنامه نویسی تابعی عمومی ML (که اغلب به عنوان زبان متا شناخته می شود) ایجاد می شود. این شامل کد منبع ML است که بیشتر در تحقیقات و طراحی زبان برنامه نویسی با استفاده از معنای کاملاً تعریف شده استفاده می شود. انواع مختلفی از ML با Caml به عنوان یکی از پرکاربردها وجود دارد. سایر موارد عبارتند از Objective Caml (OCaml)، JoCaml، Extended ML و MacroML. زبان ML معمولاً برای نوشتن کامپایلر، اثبات خودکار قضیه و تأیید رسمی استفاده می شود.
