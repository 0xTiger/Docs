## فایل APPXBUNDLE چیست؟

یک فایل APPXBUNDLE یک فایل بسته ایجاد شده با [Microsoft Visual Studio](https://visualstudio.microsoft.com/) است و برای توزیع برنامه‌های ویندوز (هم دسکتاپ و هم UWP) در [Microsoft Store](https://apps.microsoft.com/store/apps) استفاده می‌شود. این شامل یک یا چند نسخه از یک برنامه است که معماری پردازنده خاصی مانند x86، x64 یا ARM را هدف قرار می دهد. این به کاربر نهایی امکان می دهد تا نسخه مناسب برنامه را بر اساس معماری رایانه دریافت و اجرا کند. فایل‌های APPXBUNDLE با استفاده از قالب فایل استاندارد [ZIP](/compression/zip/) با هم گروه‌بندی می‌شوند. روش انتشار Microsoft Visual Studio برای انتشار برنامه ها به صورت یک بسته استفاده می شود. برنامه‌های بسته تکی به صورت فایل [APPX](/programming/appx/) توزیع می‌شوند.

## فرمت فایل APPXBUNDLE

فایل های APPXBUNDLE در قالب فایل ZIP منتشر می شوند. اگر می‌خواهید محتویات بسته برنامه را ببینید، می‌توانید با استفاده از ابزارهای رفع فشار مانند [WinZIP](https://www.winzip.com/en/) یا WinRAR محتوای آن را استخراج کنید.
