## فایل CMAKE چیست؟

یک فایل CMAKE یک فایل پروژه است که توسط ابزار متن باز و چند پلتفرمی **CMake** ایجاد شده است. این شامل اسکریپت منبع به زبان CMake برای کل برنامه است. ابزار CMake از اسکریپت های برنامه نویسی معروف به CMakeLists برای تولید فایل های ساخت برای یک محیط خاص مانند makefiles ماشین یونیکس استفاده می کند. اسکریپت ها و ماژول های مورد استفاده در پروژه CMake به صورت script.cmake و module.cmake سازماندهی می شوند که در پروژه به آنها ارجاع داده می شود.
