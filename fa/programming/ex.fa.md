## فایل EX چیست؟

فایل EX یک فایل کد منبع است که حاوی کد منبع برای برنامه های کاربردی است که با استفاده از زبان برنامه نویسی Euphoria توسعه یافته اند. Euphoria یک زبان برنامه نویسی رویه ای است که به سادگی برنامه نویسی به جای زبان های برنامه نویسی پیچیده اعتقاد دارد. یک برنامه توسعه یافته در Euphoria می تواند از چندین فایل EX تشکیل شده باشد، اما یک فایل EX نیز می تواند حاوی منبع کامل برنامه باشد.

برنامه هایی که می توانند **فایل های EX را باز کنند** عبارتند از openEuphoria Euphoria، Microsoft Notepad، Notepad++، و Atom.

## تاریخچه مختصر فرمت فایل EX

The development and existence of EX file format is linked with the development of Euphoria programming language which was created by Robert Craig. It was first released for MS-DOS. It became open-source software in 2006 with the release of version 3. توسعه آن توسط **گروه openEuphoria** که پروژه را مدیریت و توسعه می دهد، بر عهده گرفت. نسخه 4 آن در دسامبر 2010 منتشر شد و برای ویندوز، لینوکس، macOS و سه نوع BSD در دسترس است.

Euphoria در ابتدا به زبان برنامه نویسی **[C](/programming/c/)** نوشته شد. بعداً مترجم سرخوشی به دو بخش تقسیم شد:

 * «تجزیه‌کننده جلویی» - به زبان Euphoria نوشته شده و با مترجم Euphoria-to-C و Binder استفاده می‌شود.
 * Back-end - کتابخانه زمان اجرا که به زبان برنامه نویسی C نوشته شده است.

## فرمت فایل EX - اطلاعات بیشتر

فایل‌های EX به صورت فایل‌های متنی ساده ذخیره می‌شوند و حاوی کد منبع برنامه‌هایی هستند که به زبان برنامه‌نویسی Euphoria نوشته شده‌اند. این فایل ها را می توان در هر ویرایشگر متنی باز کرد.
