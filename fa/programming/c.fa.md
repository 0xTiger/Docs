## فایل C چیست؟

فایل ذخیره شده با پسوند فایل c یک فایل کد منبع است که به زبان برنامه نویسی C نوشته شده است. **فایل C** شامل تمامی عملکردهای برنامه کاربردی در قالب کد منبع می باشد. اعلان کد منبع در فایل‌های هدر که با پسوند [.h](/programming/h/) ذخیره می‌شوند، نوشته می‌شود. C++ شکل مدرن زبان C است و امروزه برای توسعه بیشتر نرم افزارها استفاده می شود.

## تاریخچه مختصر

زبان C در نتیجه ایجاد ابزارهای مختلف برای سیستم عامل یونیکس به وجود آمد. دنیس ریچی، مرد سازنده این زبان برنامه نویسی، این کار در ابتدا در دهه 1960 و اوایل دهه 1970 شروع شد.

## فرمت فایل C

فایل های C در قالب فایل متنی ساده و بر اساس دستور زبان برنامه نویسی نوشته می شوند. یک فایل C معمولی دارای موارد زیر خواهد بود:

 * عبارت import در بالای فایل برای وارد کردن هر فایل هدر
 * یک یا چند روش برای پیاده سازی عملکرد مورد نظر

### وارد کردن سرصفحه ها

فایل‌های هدر، با پسوند h، حاوی عبارات شامل لازم برای گنجاندن سایر فایل‌های عملکردی در پروژه هستند. علاوه بر این، اینها حاوی اعلان روش های تعریف شده در فایل پیاده سازی هستند. فایل های هدر با استفاده از عبارت include مطابق شکل زیر گنجانده شده اند.

```
#include <filename.h>
```

### پیاده سازی کد منبع

اجرای واقعی الزامات عملکردی به عنوان روش در فایل C کدگذاری شده است. هر متد در یک فایل C دارای یک نوع بازگشتی، نام متد است و ممکن است برخی پارامترهای ورودی داشته باشد. اگر نوع برگشتی باطل نباشد، روش ممکن است مقداری را برگرداند.

### مثال کد C
در اینجا نمونه برنامه ac است:

```
long some_function();
/* int */ other_function();

/* int */ calling_function()
{
    long test1;
    register /* int */ test2;

    test1 = some_function();
    if (test1 > 0)
          test2 = 0;
    else
          test2 = other_function();
    return test2;
}
```
