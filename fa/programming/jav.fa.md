## فایل جاو چیست؟

فایل جاو یک فایل کد منبع جاوا است که به زبان برنامه نویسی جاوا ایجاد شده است. این شبیه به فایل منبع جاوا است و می تواند با مفسرهای جاوا کامپایل شود. مانند فایل های [Java](/programming/java/)، فایل جاو در توسعه برنامه های جاوا استفاده می شود. این می تواند یکی از بسیاری از فایل های کد منبع موجود در یک پروژه جاوا باشد.

## فرمت فایل جاو

فایل های جاو به صورت فایل متنی ذخیره می شوند و با هر ویرایشگر متنی قابل باز شدن هستند. نوع فایل جاوا معمولا با نوع فایل جاوا جایگزین شده است و معمولا برای نوشتن برنامه های جاوا استفاده می شود. کدهای جاوا را می توان با IDE هایی مانند Eclipse و IntelliJ Idea نوشت که کار با فایل های جاوا را آسان تر می کند.

### نمونه فرمت فایل جاو

```java
/*
The example code prints
Hello World from Java to the console.
*/
    public class JavaApp {
        public static void main(String[] args) {
            System.out.println("Hello World."); // Prints the string to the console.
        }
    }
```
