## فایل PY چیست؟ ##

فایل‌های با پسوند py حاوی کد منبع پایتون هستند. زبان پایتون امروزه به زبان بسیار معروفی تبدیل شده است. می توان از آن برای برنامه نویسی سیستم، توسعه وب و نرم افزار و ریاضیات استفاده کرد. پایتون از سازگاری بین پلتفرم پشتیبانی می کند. به این معنی که برنامه های توسعه یافته در پایتون می توانند بر روی پلتفرم های مختلف مانند ویندوز، مک، لینوکس، رزبری پای و غیره کار کنند. توسعه دهندگان می توانند با نوشتن چند خط کد پایتون یک برنامه نرم افزاری معقول بنویسند. از آنجایی که پایتون بر روی یک سیستم مفسر اجرا می شود، بنابراین کد را می توان به محض نوشتن اجرا کرد که آن را برای نمونه سازی بسیار خوب می کند.

## تاریخچه مختصر ##

Python was conceived in the late 1980s by Guido van Rossum as a successor to ABC programming language. Its implementation began in December 1989 with Van Rossum as the sole lead developer. He worked on python until 12 July 2018. در ژانویه 2019، توسعه دهندگان هسته فعال پایتون یک شورای راهبری پنج نفره متشکل از نیک کوگلان، برت کانن، کارول ویلینگ، بری ورشو و ون روسوم را برای رهبری پروژه انتخاب کردند.

## نحوه اجرای فایل py ##

برای بررسی نسخه نصب شده پایتون، می توانید از دستور زیر استفاده کنید:

```cmd
python --version
```

با این کار نسخه مشابه روی کنسول خروجی می شود

```cmd
Python 3.7.4
```

اگر پایتون روی دستگاه شما نصب نیست، می توانید به [python.org](https://www.python.org/) بروید و پایتون را برای سیستم عامل مربوطه خود دانلود و نصب کنید.

برای اجرای اسکریپت پایتون می توانید از دستور زیر استفاده کنید:

```cmd
python helloworld.py
```

*helloworld.py* یک فایل اسکریپت است که حاوی کد زیر است

```py
print("Hello World from Python")
```

با این کار موارد زیر در پنجره کنسول چاپ می شود.

```cmd
Hello World from Python
```

توجه: اگر از IDE ها استفاده می کنید، دکمه هایی روی صفحه یا میانبرهای صفحه کلید مختلف برای اجرای پایتون ارائه می کنند. به عنوان مثال، یک دکمه پخش در ناودان با ویرایشگر در PyCharm نشان داده می شود که به شما امکان می دهد اسکریپت پایتون را اجرا کنید.

## فرمت فایل PY ##

پایتون برای خوانایی طراحی شده است و شباهت هایی به زبان انگلیسی و ریاضی دارد. پایتون از خطوط جدیدی برای نشان دادن دستور کامل استفاده می کند، برخلاف نقطه ویرگول یا پرانتز که در زبان های دیگر استفاده می شود. برای دامنه‌ها، حلقه‌ها و توابع، پایتون به جای پرانتزهای فرفری که در زبان‌های دیگر استفاده می‌شود، بر تورفتگی و فضاهای سفید متکی است.

### نحو ###

در زیر نمونه ای از سینتکس پایتون آورده شده است.

```py
print("Hello World")

#Variables
name = "John"
age = 25
print(name)
print(age)

#While Loop
i = 1
while i < 3:
  print(i)
  i += 1
  
#For Loop
names = ["John", "Harry", "Tom"]
for name in names:
  print(name)
```
