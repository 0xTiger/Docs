## فایل F95 چیست؟

یک فایل با پسوند .f95 معمولاً نشان می دهد که یک فایل کد منبع Fortran 95 است. فرترن یک زبان برنامه نویسی همه منظوره و ضروری است که مخصوصاً برای محاسبات عددی و علمی مناسب است. Fortran 95 یک نسخه پیشرفته از Fortran 90 است که چندین ویژگی و پیشرفت جدید را معرفی می کند.

فایل های کد منبع فرترن معمولاً حاوی دستورالعمل های برنامه ای هستند که به زبان برنامه نویسی فرترن نوشته شده اند. اگر فایلی با پسوند .f95 دارید، احتمالاً با استفاده از کامپایلر فرترن کامپایل شده است. کامپایل فرآیند ترجمه کد منبع قابل خواندن توسط انسان به کد باینری قابل خواندن توسط ماشین است که می تواند توسط رایانه اجرا شود.

## چگونه فایل F95 را باز کنیم؟

اگر می خواهید با فایل .f95 کار کنید یا برنامه را اجرا کنید، به یک کامپایلر فرترن نیاز دارید. کامپایلرهای محبوب فرترن عبارتند از GNU Fortran (gfortran)، Intel Fortran Compiler (ifort) و غیره. شما می توانید برنامه های فرترن را با استفاده از دستورات مناسب ارائه شده توسط کامپایلر کامپایل و اجرا کنید.

به عنوان مثال، اگر gfortran را نصب کرده اید، می توانید از دستورات زیر در ترمینال برای کامپایل و اجرای یک برنامه Fortran استفاده کنید:

```
# Compile the Fortran program
gfortran -o my_program my_program.f95

# Run the compiled program
./my_program
```

اگر فقط می خواهید محتویات فایل F95 را مشاهده کنید، یک فایل متنی ساده است، بنابراین می توانید آن را با هر ویرایشگر متنی باز کنید.
