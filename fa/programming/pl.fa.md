## فایل PL چیست؟

یک فایل با پسوند pl. یک فایل Perl Script است که یک زبان برنامه نویسی است. این ها با استفاده از نرم افزار Perl Interpreter کامپایل و اجرا می شوند. یک فایل اسکریپت PL شامل خطوط کد، متغیرها و نظرات است. زبان های برنامه نویسی نسبتاً دشوار هستند
فرمت های فایل برنامه نویسی دیگر مانند [PHP](/programming/php/) را درک کنید. فایل های PL را می توان ایجاد کرد و با ویندوز، macOS و لینوکس سازگار است.

## تاریخچه مختصر زبان اسکریپت پرل

این زبان برای اولین بار در سال 1987 مورد استفاده قرار گرفت، بنابراین این فایل ها منشاء خود را در آن سال پیدا کردند. در سال 1989 پرل 2 منتشر شد. بعداً آپدیت شده و تا نسخه 5.35 اصلاح شده و نسخه بعدی قرار است سال آینده منتشر شود.

در هر به روز رسانی، ابزارهایی در مورد عملکرد، عملکرد و ظاهر زبان و فایل ها اضافه شده است. در این سال ها بازنگری های زیادی در مورد نسخه ها صورت گرفته است. این در ابتدا یک زبان برنامه نویسی اولیه بود اما اکنون شامل بسیاری از ماژول های دیگر نیز می شود. در اصل، این زبان بسیار ساده بود، اما درک نوشتارهای این زبان به دلیل فشرده بودن بسیار دشوار بود.

## مشخصات پسوند فرمت فایل Perl

برخی از ویژگی ها یا مشخصات این فایل های برنامه نویسی وجود دارد که برخی از آنها به شرح زیر است:

* کد موجود در این فایل ها به صورت متن ساده است و برای توسعه اسکریپت استفاده می شود
* اسکریپت سرور، تجزیه متن و مدیریت سرور اصلی ترین جنبه هایی است که اسکریپت این زبان پوشش می دهد.
* بسیاری از برنامه های محبوب مرتبط با این زبان عبارتند از Active State Active Perl و Bare Bones BBEdit (سازگار با Mac OS)
* این زبان یک زبان سطح بالا محسوب می شود
* برای Win32، کاربر ممکن است مجبور باشد توزیع باینری بومی زبان را نصب کند
* برای قابل اجرا شدن در کیت های منابع مختلف ویندوز به برخی ابزارهای برنامه نویسی نیاز دارد
* Visual Studio.NET یک فریمورک معروف برای توسعه زبان های برنامه نویسی است. ابزار Active State معروف به Visual Perl برای اضافه کردن Perl به Visual Studio استفاده می شود
* در فایل ها، خط اول کد منبع، اطلاعات مفسر مورد استفاده را توضیح می دهد. فایل‌های PL معمولاً با خط **#!/usr/bin/perl** شروع می‌شوند که به رایانه شما می‌گوید این اسکریپت را با استفاده از مترجم Perl نصب شده در رایانه اجرا کند.


## نمونه های اسکریپت PL

```
#!/usr/bin/perl
print "Hello, world\n";
```

این چاپ خواهد شد

```
Hello, World
```

### نظر تک خطی ###

```
#!/usr/bin/perl
# This is a single line comment
print "Hello Perl\n";
```

### نظر چند خطی ###

```
#!/usr/bin/perl
=begin comment
This is a multiline comment.
Line 1
Line 2
=cut
print "Hello Perl\n";
```

### تخصیص متغیر ###

```
#!/usr/bin/perl
$a = 10;
print "Variable a = $a\n";
```

### تخصیص متغیر اسکالر ###

```
#!/usr/bin/perl
$age = 35; # Assigning an integer
$name = "Tony Stark"; # Assigning a string
$pi = 3.14; # Assigning a floating point
```

### عملیات اسکالر ساده ###

```
#!/usr/bin/perl
$constr = "hi" . "perl";# Concatenates two or more strings.
$add = 40 + 10; # addition of two numbers.
$prod = 4 * 51;# multiplication of two numbers.
$connumstr = $constr . $add;# concatenation of string and number.
```
