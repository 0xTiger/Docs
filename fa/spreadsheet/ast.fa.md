## فایل AST چیست؟

یک فایل AST یک فایل قالب صفحه گسترده است که با نرم افزار Ability Spreadsheet، جزء صفحه گسترده مجموعه Ability Office ایجاد شده است. این فایل‌های الگو برای ایجاد مجموعه اولیه از الگوهای صفحه‌گسترده استفاده می‌شوند که می‌توانند برای ایجاد فایل‌های [AWS](/spreadsheet/aws/) استفاده شوند. فایل های AST را می توان با نرم افزار صفحه گسترده Ability باز کرد.

## فرمت فایل AST - اطلاعات بیشتر

AST files are saved to disc as binary files and can be opened in Ability Spreadsheet software. Though fully compatible with Microsoft XLS and XLSX file formats, AST file format specifications are not available publicly for developer's reference. AST files can be saved to disc in [PDF](/pdf/) as well as [XPS](/page-description-language/xps/) file formats.
