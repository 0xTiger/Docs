## فایل DIS چیست؟

A DIS file is a workbook file created with [Oracle Discoverer](https://docs.oracle.com/cd/E28389_01/bi.1111/b40107/overview.htm) software that comprises of several tools for querying, reporting, and data analysis for the Oracle Database. It contains multiple worksheets when data is exported from Oracle Discoverer. Oracle markets Discoverer as a Business Intelligence (BI) product used for for reporting and analysis that works in part of the [Oracle Fusion Middleware](https://docs.oracle.com/cd/E28280_01/bi.1111/b40108/toc.htm) suite. Oracle Discoverer doesn't save the reporting data in DIS file format by default.

## چگونه کتاب های کار Discoverer را با فرمت DIS دانلود کنیم؟

کتاب‌های کار Oracle Discoverer در پایگاه داده در قالب باینری ذخیره می‌شوند و نمی‌توانند مستقیماً به صورت [DIS files](https://forums.oracle.com/ords/apexds/post/can-i-download-all-discoverer-workbooks-to-my-computer-4127) دانلود شوند. با این حال، داده ها را می توان به عنوان فایل های EEX صادر کرد، آن را در Discoverer Desktop باز کرد و از گزینه Save As برای ذخیره فایل های EEX به عنوان فایل های DIS استفاده کرد.

## فرمت فایل DIS - اطلاعات بیشتر

فایل‌های DIS به‌عنوان فایل‌های باینری در دیسک ذخیره می‌شوند، اما جزئیات فرمت فایل داخلی آنها برای مرجع توسعه‌دهنده در دسترس نیست.
