## فایل XLAM چیست؟ ##

XLAM یک فایل افزودنی با قابلیت Macro-Enabled است که برای افزودن عملکردهای جدید به صفحات گسترده استفاده می شود. Add-In یک برنامه تکمیلی است که کدهای اضافی را اجرا می کند و عملکردهای اضافی را برای صفحات گسترده فراهم می کند. فایل های XLAM با پسوند xlam. ذخیره می شوند. فایل‌های XLAM فایل‌های مبتنی بر XML مشابه فرمت‌های فایل [XLSM](/spreadsheet/xlsm/) و [XLSX](/spreadsheet/xlsx/) هستند و با فشرده‌سازی ZIP ذخیره می‌شوند تا اندازه کلی فایل کاهش یابد.

### مثال ###

```cmd
Public Function Add(num1 As Double, num2 As Double)
Add = num1 + num2
End Function
```

پس از این، فایل را با پسوند xlam. ذخیره کنید.
