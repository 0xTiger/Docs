## فایل XLR چیست؟

A file with .xlr extension is a Microsoft Works Spreadsheet file format that stored data in rows and columns. Microsoft Works was developed for use as an initial productivity suite and remained functional from 1987 to 2009. این شامل یک پردازشگر کلمه، یک صفحه گسترده و یک سیستم مدیریت پایگاه داده بود. بعداً پس از معرفی مایکروسافت آفیس با ویژگی های غنی تر، متوقف شد.
