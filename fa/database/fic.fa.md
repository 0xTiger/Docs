## فایل FIC چیست؟

فایل FIC در WinDev یک نسخه کامپایل شده از پایگاه داده HyperFileSQL است. مخفف FIC مخفف Fichier de Code در فرانسوی است که به انگلیسی به Code File ترجمه می شود. در WINDEV، یک فایل FIC یک فایل کامپایل شده است که حاوی کد اجرایی یک پروژه است. این فایل دارای پسوند .fic است و هنگام کامپایل یک پروژه WINDEV ایجاد می شود. هنگامی که یک پروژه در یک فایل FIC کامپایل می شود، می توان آن را به عنوان یک برنامه اجرایی یا یک بسته نصبی بین کاربران توزیع کرد.

فایل FIC مربوط به WINDEV، یک ابزار توسعه نرم افزار است که توسط PCSoft، یک شرکت فرانسوی ایجاد شده است. این به توسعه دهندگان اجازه می دهد تا با استفاده از یک پایه کد واحد، برنامه های ویندوز، وب و موبایل ایجاد کنند. این شامل یک محیط توسعه بصری، یک زبان برنامه نویسی (WLangage) و انواع کنترل ها و توابع از پیش ساخته شده برای کارهای رایج است. این در درجه اول برای برنامه های تجاری استفاده می شود و یک پایگاه کاربر قوی در اروپا دارد.

## ارتباط با پایگاه داده HyperFileSQL Mobile

در WinDev، یک فایل FIC که برای پایگاه داده HyperFileSQL Mobile ایجاد می شود، حاوی کدهای اجرایی برای پایگاه داده و همچنین داده های موجود در پایگاه داده است. HyperFileSQL Mobile یک سیستم مدیریت پایگاه داده رابطه ای است که با WINDEV گنجانده شده است و به طور خاص برای برنامه های موبایل و آفلاین طراحی شده است. فایل FIC را می توان به عنوان یک فایل اجرایی مستقل یا به عنوان بخشی از یک بسته نصبی توزیع کرد و می تواند بر روی دستگاه مورد نظر برای دسترسی و مدیریت داده ها در پایگاه داده HyperFileSQL Mobile اجرا شود. هنگامی که فایل FIC اجرا می شود، پایگاه داده را باز می کند و داده ها را برای استفاده در برنامه تلفن همراه در دسترس قرار می دهد.
