## فایل ADN چیست؟

یک فایل با پسوند adn. یک فایل قالب خالی مایکروسافت اکسس است که برای ایجاد فایل های پروژه Access با تنظیمات از پیش ذخیره شده استفاده می شود. فایل های پروژه Microsoft Access به عنوان فایل های ADP ذخیره می شوند و حاوی اطلاعاتی مانند اتصال پایگاه داده و تنظیمات راه اندازی پروژه هستند. فایل های ADN برای نسخه های 2007 و 2010 مایکروسافت اکسس وجود دارد. مایکروسافت اکسس از فرمت‌های فایل دیگری مانند [MDB](/database/mdb/)، [ACCDB](/database/accdb/)، [ACCDC](/database/accdc/)، [ACCDE](/database/accde/) و بسیاری دیگر علاوه بر این فایل‌های ADN پشتیبانی می‌کند. فایل های ADN را می توان با استفاده از Office 365 و Microsoft Access باز کرد.

## فرمت فایل ADN - اطلاعات بیشتر

ADN فایل های قالب پایگاه داده هستند که به صورت فایل های باینری روی دیسک ذخیره می شوند. جزئیات فرمت فایل داخلی فایل‌های ADN برای مرجع توسعه‌دهنده در دسترس نیست.
