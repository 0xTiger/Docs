## فایل DB3 چیست؟
فایل DB3 یک فایل پایگاه داده است که توسط نرم افزار SQLite ایجاد شده است که یک برنامه پایگاه داده سبک وزن است که با استفاده از فایل های ساده پایگاه داده ایجاد می کند. شامل آناتومی پایگاه داده و همچنین رکوردهای داده است. معمولاً برای بازیابی یا ذخیره داده های ساخت یافته با استفاده از SQL استفاده می شود. این فایل‌ها را می‌توان در دستگاه‌های هوشمند یا در جاهایی که نیاز به نگهداری سوابق یا سایر مدیریت داده‌ها است، اما در محیط کم استفاده کرد.


## فرمت فایل DB3
فرمت فایل DB3 با RDBMS (سیستم مدیریت پایگاه داده رابطه ای) SQLite، یک انتخاب محبوب برای پایگاه داده جاسازی شده، مرتبط است. هیچ پسوند فایل خاصی در SQLite_3 در مشخصات آن تعریف نشده است، ممکن است از پسوندهایی از جمله [.dbf](/database/dbf/) و [.sqlite](/database/sqlite/) استفاده کند.

### تقسیم بندی پایگاه داده
Commonly SQLite, version 3 related db file format can be considered as the DB3 file format used as the publicly documented native format for the SQLite database engine since June 2004. فرمت فایل DB3، یک کراس پلتفرم است که بین معماری‌های بزرگ و کوچک اندین یا سیستم‌های 32 بیتی و 64 بیتی قابل انتقال است. این ویژگی‌ها DB3 را به عنوان یک فرمت فایل کاربردی محبوب می‌کند. فایل اصلی پایگاه داده SQLite_3 (DB3) از یک یا چند صفحه تشکیل شده است. اندازه همه صفحات در پایگاه داده یکسان است. اندازه صفحه در بایت توان دو بین 512 و 65536 است.
