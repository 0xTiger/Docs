## فایل FP5 چیست؟

An FP5 file is a database file created with FileMaker Pro version 5 or 6. داده های کاربر و سازمانی را در قالب رکوردهای داخل جداول ذخیره می کند. FileMaker Pro یک سیستم مدیریت پایگاه داده است که از سیستم عامل های مکینتاش و ویندوز پشتیبانی می کند. FP5 یک فرمت فایل قدیمی است که توسط نسخه FileMaker Pro موجود در آن زمان پشتیبانی می شد. آخرین فرمت پایگاه داده FileMaker Pro [FMP12](/database/fmp12/) است که با راه اندازی FileMaker Pro v 12 معرفی شد. دیگر قالب فایل پایگاه داده به خوبی استفاده شده [FP7](/database/fp7/) بود که با FMP12 جایگزین شد.

## فرمت فایل FP5 - اطلاعات بیشتر

جزئیات فرمت فایل داخلی فرمت فایل FP5 به صورت عمومی برای مرجع توسعه دهنده در دسترس نیست. فرمت های فایل پایگاه داده قدیمی FileMaker Pro مانند [FP5 and FP7 can be upgraded to latest FMP12](https://fmhelp.filemaker.com/help/16/fmp/en/index.html#page/FMP_Help/converting-files.html) با استفاده از دستورالعمل های دقیق FileMaker Pro.
