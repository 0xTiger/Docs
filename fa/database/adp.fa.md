## فایل ADP چیست؟

A file with .adp extension is a Microsoft Access project file that uses OLE DB component architecture to provide a direct and efficient connection to a Microsoft SQL Server database. Such files do not contain the actual tables, database diagrams or other database elements. ADP files can be created with Access 2007 and 2010. فایل‌های ایجاد شده با نسخه‌های قبلی Access را می‌توان با Access 2007 و 2010 باز کرد. فایل‌های ADP را می‌توان با استفاده از Microsoft Access 365 باز کرد.

## فرمت فایل ADP - اطلاعات بیشتر

پروژه Access ایجاد شده به عنوان ADP به شما امکان می دهد تغییرات طراحی را در اشیاء سرور SQL مانند جداول و نماها ایجاد کنید. همچنین امکان ایجاد، ویرایش و استفاده از ویژگی های SQL مانند نمودارهای پایگاه داده، رویه های ذخیره شده و عملکرد تعریف شده توسط کاربر را فراهم می کند. این مزیت را نسبت به پیوند دادن به پایگاه داده سرور SQL می دهد که در آن نمی توانید تغییراتی در طراحی ایجاد کنید و فقط می توانید به جداول و نماهای سرور SQL پیوند دهید.

### انواع داده و پشتیبانی از نمودار

**تاریخ/زمان انواع داده**

انواع داده های تاریخ/زمان جدید زیر در Access 2010 پشتیبانی می شوند.

 * زمان
 * تاریخ
 * تاریخTIME2
 * تاریخTIMEOFFSET

**انواع داده با طول متغیر**

انواع داده های طول متغیر زیر را می توان در پروژه های Access 2010 استفاده کرد:

 * VARBIN (MAX)
 * VARCHAR (MAX)
 * NVARCHAR (MAX)
