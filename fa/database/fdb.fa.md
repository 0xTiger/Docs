## فایل FDB چیست؟

یک فایل .fdb یک پسوند فایل است که برای فایل های پایگاه داده در Microsoft Dynamics NAV استفاده می شود. پسوند فایل .fdb مخفف Firebird Database File است و فرمت فایلی است که توسط سیستم مدیریت پایگاه داده Firebird استفاده می شود، که موتور پایگاه داده زیربنایی است که توسط Microsoft Dynamics NAV استفاده می شود. فایل .fdb شامل تمام داده ها، جداول و ساختارهای سیستم NAV از جمله داده های مالی، داده های موجودی، داده های مشتری و غیره است. مهم است که یک نسخه پشتیبان از فایل .fdb نگه دارید تا از یکپارچگی داده ها اطمینان حاصل کنید و در صورت بروز هر مشکلی قادر به بازیابی اطلاعات باشید. فایل .fdb را می توان اکسپورت، وارد کرد و همچنین در تکثیر از آن استفاده کرد.

## ارتباط با Microsoft Dynamics NAV

فایل های FDB فایل های پایگاه داده در Microsoft Dynamics NAV هستند. Microsoft Dynamics NAV یک نرم افزار برنامه ریزی منابع سازمانی (ERP) است که توسط مایکروسافت توسعه یافته است. این برای مشاغل کوچک و متوسط طراحی شده است و طیف گسترده ای از عملکردهای مدیریت کسب و کار از جمله مدیریت مالی، مدیریت زنجیره تامین، تولید، مدیریت پروژه و غیره را ارائه می دهد. این بر روی پلتفرم Microsoft Dynamics ساخته شده است و به طور کامل با سایر محصولات مایکروسافت مانند Office و Outlook یکپارچه شده است. Dynamics NAV را می توان از طریق یک سرویس گیرنده وب و همچنین از طریق یک کلاینت ویندوز در دسترس قرار داد و همچنین می توان آن را با استفاده از محیط توسعه خود به نام C/AL با سیستم های دیگر سفارشی و ادغام کرد. معمولاً توسط شرکت هایی در صنایع تولیدی، عمده فروشی و خدماتی استفاده می شود.

## چگونه فایل .fdb را باز کنیم؟

فایل FDB معمولاً با استفاده از محیط توسعه Microsoft Dynamics NAV یا کلاینت Microsoft Dynamics NAV باز و مدیریت می شود.

در اینجا مراحل باز کردن یک فایل .fdb در Microsoft Dynamics NAV آمده است:

1. محیط توسعه Microsoft Dynamics NAV را باز کنید.
2. روی منوی File کلیک کنید و Open را انتخاب کنید یا Ctrl+O را فشار دهید.
3. به محلی که فایل .fdb ذخیره شده است بروید.
4. فایل .fdb را انتخاب کرده و روی دکمه Open کلیک کنید.

همچنین می توانید فایل .fdb را با اتصال به پایگاه داده با استفاده از سرویس گیرنده NAV باز کنید.

1. کلاینت NAV را باز کنید.
2. روی منوی File کلیک کنید و Connect را انتخاب کنید.
3. در قسمت Server نام یا آدرس IP سروری که فایل fdb. در آن قرار دارد را وارد کنید.
4. نام پایگاه داده و Credentials مناسب را وارد کنید
5. بر روی دکمه اتصال کلیک کنید.

همچنین باز کردن فایل‌های fdb. با نرم‌افزار مدیریت سیستم مدیریت پایگاه داده Firebird امکان‌پذیر است، زیرا این موتور پایگاه داده زیربنایی است که توسط Microsoft Dynamics NAV استفاده می‌شود.
