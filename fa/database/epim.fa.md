## فایل EPIM چیست؟

پسوند فایل .epim توسط EssentialPIM، یک نرم افزار مدیریت اطلاعات شخصی استفاده می شود که به کاربران اجازه می دهد مخاطبین، قرارها، وظایف و یادداشت های خود را سازماندهی و مدیریت کنند. فایل .epim یک فایل پایگاه داده است که تمام داده های وارد شده به برنامه از جمله اطلاعات شخصی، رویدادهای تقویم و لیست وظایف را ذخیره می کند. این فایل را می توان پشتیبان گیری کرد، صادر کرد و به یکی دیگر از نصب های EssentialPIM وارد کرد. این نرم افزار برای پلتفرم های ویندوز، اندروید و iOS در دسترس است.

فایل‌های epim فرمت اختصاصی مورد استفاده EssentialPIM هستند. توجه به این نکته ضروری است که فایل های .epim مختص نرم افزار EssentialPIM هستند و نمی توان آنها را با نرم افزارهای دیگر باز یا ویرایش کرد.

## چگونه فایل EPIM ایجاد کنیم؟

برای ایجاد یک فایل .epim در EssentialPIM، ابتدا باید نرم افزار را روی رایانه خود نصب کنید.

1. EssentialPIM را باز کنید.
2. بر روی منوی File کلیک کنید و New Database را انتخاب کنید.
3. یک کادر محاوره ای ظاهر می شود که از شما می خواهد مکان و نام فایل پایگاه داده جدید را انتخاب کنید.
4. نامی را وارد کنید و مکانی را برای فایل انتخاب کنید و سپس روی ذخیره کلیک کنید.
5. فایل پایگاه داده جدید با پسوند فایل epim ایجاد می شود و می توانید شروع به اضافه کردن داده هایی مانند مخاطبین، قرار ملاقات ها، وظایف و یادداشت ها کنید.

همچنین می توانید از File -> Import برای وارد کردن داده ها از فرمت های مختلف و ایجاد یک فایل .epim جدید استفاده کنید. علاوه بر این، اگر از قبل یک فایل پایگاه داده دارید و می خواهید یک فایل جدید از آن ایجاد کنید، می توانید از گزینه File -> Export برای صادرات تمام داده ها از فایل .epim فعلی و وارد کردن آن به فایل جدید استفاده کنید. یکی

## مکان فایل های EPIM

مکان فایل های .epim در EssentialPIM به سیستم عاملی که استفاده می کنید بستگی دارد.

- در ویندوز، مکان پیش‌فرض فایل‌های epim در پوشه «اسناد من» است، اما می‌توانید هنگام ایجاد یک فایل پایگاه داده جدید، مکان دیگری را انتخاب کنید.
- در اندروید، مکان پیش‌فرض فایل‌های epim در پوشه «دانلود» است، اما وقتی یک فایل پایگاه داده جدید ایجاد می‌کنید، می‌توانید مکان دیگری را انتخاب کنید.
- در iOS، مکان پیش‌فرض فایل‌های epim در پوشه «EssentialPIM» است، اما هنگام ایجاد یک فایل پایگاه داده جدید، می‌توانید مکان دیگری را انتخاب کنید.

همچنین می توانید از گزینه File -> Open Database برای باز کردن یک فایل .epim موجود از مکان دیگری در رایانه یا دستگاه خود استفاده کنید.
