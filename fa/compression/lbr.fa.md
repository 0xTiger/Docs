## فایل LBR چیست؟

یک فایل با پسوند .lbr یک فایل آرشیو است که با برنامه LU ایجاد شده و در سیستم عامل های CP/M و DOS در طول دهه 1980 استفاده می شود. دیگر مورد استفاده قرار نمی گیرد و با فرمت های فایل بایگانی مدرن جایگزین شده است. فرمت فایل های عضو را فشرده نمی کند و به عنوان بایگانی کانتینری برای این فایل ها عمل می کند. ویژگی آرشیو بیشتر برای انتقال فایل های آرشیو شده از طریق اینترنت استفاده می شد. از آنجایی که LBR فشرده‌سازی را ارائه نمی‌دهد، از ابزارهای دیگر مانند SQ یا CRUNCH برای فشرده‌سازی فایل‌های عضو پیش از بایگانی یا فایل آرشیو حاصل پس از بایگانی برای کاهش اندازه آن استفاده شد.

## فرمت فایل LBR

LBR file format was designed by Gary P. Novosielski and has no technical details available publicly. LBR files start with a 0x00 byte, followed by 11 spaces (0x20), then two 0x00 bytes, then two bytes that are not both 0x00. از آنجایی که فرمت فایل کانتینری است، می توان آن را با استفاده از LU.COM و NULU.COM استخراج کرد.
