## فایل XZ چیست؟

XZ یک فرمت فایل فشرده است که از الگوریتم فشرده سازی LZMA2 استفاده می کند. این به عنوان جایگزینی برای فرمت های محبوب gzip و bzip2 طراحی شده است و چندین مزیت را نسبت به این استانداردهای قدیمی ارائه می دهد. فایل‌های XZ در بسیاری از پلتفرم‌ها به خوبی پشتیبانی می‌شوند و می‌توانند به سرعت و به راحتی از حالت فشرده خارج شوند. در حالی که آنها به اندازه فایل‌های [ZIP](/compression/zip/) یا [RAR](/compression/rar/) رایج نیستند، بایگانی‌های XZ می‌توانند برای ذخیره مقادیر زیادی از داده‌ها بدون کاهش کیفیت فشرده‌سازی استفاده شوند. اگر نیاز به فشرده سازی یا از حالت فشرده خارج کردن فایل های بزرگ دارید، پسوند فایل XZ ارزش در نظر گرفتن دارد.

## فرمت فایل XZ

فایل XZ تولید و به عنوان فایل های باینری در دیسک ذخیره می شود. از الگوریتم LZMA برای فشرده سازی داده ها استفاده می کند و می تواند به نسبت فشرده سازی تا 50 درصد دست یابد. فایل‌های XZ معمولاً برای توزیع نرم‌افزار و آرشیوهای پشتیبان استفاده می‌شوند. آنها را می توان با استفاده از ابزار XZ که در اکثر توزیع های لینوکس موجود است، از حالت فشرده خارج کرد.

## فایل های XZ را در لینوکس/یونیکس از حالت فشرده خارج کنید

برای اینکه فایل های XZ را از حالت فشرده خارج کنید، ابتدا بسته «xz-utils» را نصب کنید:
```
$ sudo apt-get install xz-utils
```

از دستور unxz برای استخراج فایل های xz. استفاده کنید:
```
$ unxz compressed_xz_file.xz
```

یا با استفاده از گزینه --decompress XZ:
```
$ xz --decompress compressed_xz_file.xz
```
