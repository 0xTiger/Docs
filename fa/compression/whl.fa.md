## فایل WHL چیست؟

فایل WHL (Wheel) یک فایل بسته توزیعی است که در قالب چرخ پایتون ذخیره شده است. این یک نصب با فرمت استاندارد توزیع‌های پایتون است و شامل تمام فایل‌ها و ابرداده‌های مورد نیاز برای نصب است. فایل WHL همچنین حاوی اطلاعاتی در مورد نسخه های پایتون و پلتفرم های پشتیبانی شده توسط این فایل چرخ است. فرمت فایل WHL مشابه یک فایل راه‌اندازی [MSI](/executable/msi/) یک قالب آماده نصب است که امکان اجرای بسته نصب را بدون ایجاد توزیع منبع فراهم می‌کند.

## فرمت فایل WHL

فرمت فایل WHL یک آرشیو [ZIP](/compression/zip/) (.zip) است که شامل تمام فایل‌های نصب و ابرداده‌های مورد نیاز نصب‌کنندگان برای نصب یک بسته است. این فایل های WHL را می توان با استفاده از گزینه unzip یا نرم افزارهای نرم افزار رفع فشرده سازی استاندارد مانند WinZIP و WinRAR استخراج کرد.

### قرارداد نام فایل WHL

یک فایل WHL طبق قرارداد زیر نامگذاری شده است.

```
{dist}-{version}(-{build})?-{python}-{abi}-{platform}.whl
```

نمونه ای از نام فایل WHL به شرح زیر است.

```
cryptography-2.9.2-cp35-abi3-macosx_10_9_x86_64.whl
```

 * رمز نگاری نام بسته است.
 * «2.9.2» نسخه بسته رمزنگاری است. یک نسخه یک رشته مطابق با PEP 440 مانند 2.9.2، 3.4، یا 3.9.0.a3 است.
 * «cp35» تگ پایتون است و نشان‌دهنده اجرای پایتون و نسخه‌ای است که چرخ درخواست می‌کند.
 * abi3 برچسب ABI است. ABI مخفف رابط باینری برنامه است.
 * «macosx_10_9_x86_64» تگ پلتفرم است که اتفاقاً کاملاً لقمه‌ای است.
