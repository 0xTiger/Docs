## فایل INDD چیست؟

یک فایل با پسوند indd. یک فایل پروژه Adobe InDesign است که حاوی اطلاعات طرح بندی صفحه پروژه است. INDD مخفف InDesign Document است. این فایل‌ها معمولاً اطلاعات قالب‌بندی صفحه، کتاب‌ها، مجلات، روزنامه‌ها، آگهی‌ها، بروشورها یا سایر رسانه‌های چاپی مانند مدل‌های موبایل و صفحه وب را ذخیره می‌کنند. نوع فایل INDD فرمت فایل انحصاری Adobe است و فرمت فایل پیش فرض InDesign است. طراحان گرافیک، روزنامه نگاران، بازاریابان و ناشرانی که با InDesign کار می کنند از این نوع فایل استفاده می کنند.

## اطلاعات بیشتر در مورد فرمت فایل INDD

فایل‌های INDD فایل‌های برداری هستند که می‌توانند بدون از دست دادن وضوح، مقیاس شوند. اینها بر اساس نمودارهای پیچیده هستند که می توانند بی نهایت گسترش یابند. تمام فونت ها و اطلاعات طرح بندی را بسته بندی می کند تا با یک تیم به اشتراک گذاشته شود.

فایل های INDD برای طراحی محتوای چاپی جذاب و همچنین محتوای آنلاین عالی هستند. یک فایل INDD می تواند در چندین صفحه باشد و می تواند به عنوان الگویی برای ایجاد صفحات، فصل ها و انتشارات باشد. هنگامی که پروژه شما کامل شد، می توان آن را بدون نیاز به تبدیل، فشرده سازی یا ایمیل در رسانه های آنلاین منتشر کرد.

## آیا فایل INDD را می توان به فرمت های دیگر تبدیل کرد؟

INDD files can be converted to several other file formats. The most common conversion is INDD to PDF. InDesign lets you save the INDD file to PDF for sharing with other users. It also lets you export INDD file to JPG, [EPS](/page-description-language/eps/), [EPUB](/ebook/epub/), [SWF](/page-description-language/swf/), FLA, [HTML](/web/html/), [XML](/web/xml/), and IDML.

## فایل INDD چگونه ایجاد می شود؟

An Indd file is created when you create a new project with [Adobe InDesign](https://www.adobe.com/products/indesign.html) and save it to disc. You can open the Indd project file, make changes to it, and save the updated file for the changes.
