## فایل POT چیست؟

فایل‌های با پسوند POT فایل‌های قالب پاورپوینت مایکروسافت را نشان می‌دهند که توسط نسخه‌های PowerPoint 97-2003 ایجاد شده‌اند. فایل‌هایی که با این نسخه‌های مایکروسافت پاورپوینت ایجاد می‌شوند در مقایسه با فایل‌هایی که در فرمت‌های فایل Office OpenXML با استفاده از نسخه‌های بالاتر پاورپوینت ایجاد شده‌اند، در قالب باینری هستند. از این رو، فایل‌های تولید شده را می‌توان برای ایجاد ارائه‌هایی استفاده کرد که دارای طرح‌بندی یکسان و سایر تنظیمات لازم برای اعمال بر روی فایل‌های جدید هستند. این تنظیمات می تواند شامل سبک ها، پس زمینه ها، پالت رنگ ها، فونت ها و پیش فرض ها باشد. چنین فایل هایی به منظور ایجاد فایل های قالب آماده برای استفاده برای استفاده رسمی تولید می شوند.

## مشخصات فرمت فایل ##

File format specifications for POT file format are not available individually by Microsoft. However, the template format is the same as that of the binary [PPT](/presentation/ppt/) file format and can be opened in MS PowerPoint for editing purpose. A POT file is identified by HEX identifier as follow:

```
D0 CF 11 E0 A1 B1 1A E1 00 00 00 00
```

انواع MIME مرتبط با فرمت فایل POT عبارتند از:

* application/vnd.ms-powerpoint [رسمی]

* برنامه/mspowerpoint

* application/x-mspowerpoint

* برنامه/پاورپوینت

* اپلیکیشن/x-powerpoint

* application/x-dos_ms_powerpnt

* کاربرد/گلدان

* application/x-soffic
