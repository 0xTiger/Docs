## فایل AMXX چیست؟

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. این یک پلت فرم برنامه نویسی است که به مدیران سرور اجازه می دهد تا عملکرد سرورهای خود را سفارشی کرده و گسترش دهند. AMXX در ارتباط با سیستم مدیریت پلاگین MetaMod استفاده می شود که به چندین پلاگین اجازه می دهد تا به طور همزمان روی یک سرور بدون درگیری اجرا شوند. از ویژگی های ارائه شده توسط افزونه AMXX می توان به دستورات مدیریت، تغییرات گیم پلی و مدیریت سرور اشاره کرد.
