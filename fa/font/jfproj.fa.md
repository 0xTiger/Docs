## فایل JFPROJ چیست؟

A file with .jfproj extension is a JSON based custom font created with Microsoft Font Maker application on Windows 10. اینها را می توان با استفاده از قلم الکترونیکی با دستگاه های صفحه لمسی مانند تبلت ایجاد کرد، اگرچه سایر لوازم جانبی رایانه مانند ماوس نیز در مورد رایانه ها قابل استفاده هستند. فایل‌های JFPROJ حاوی حروف‌هایی هستند که با تعیین فاصله بین کاراکترها و کلمات و اندازه فونت برای نمایش کاراکترها به صفحه نمایش استفاده می‌شوند. برنامه Microsoft Font Maker برای دانلود در فروشگاه ویندوز موجود است.
