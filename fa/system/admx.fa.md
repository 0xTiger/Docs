## فایل ADMX چیست؟

An ADMX file is a policy configuration file used by Microsoft Windows Group Policy software that manages group of computers in a group. It is an XML-based administrative template file and was introduced with Windows Vista Service Pack 1. فایل های ADMX حاوی تنظیمات پیکربندی برای حساب های کاربری، پیکربندی های سیستم عامل و برنامه ها هستند. فایل های ADMX برای ذخیره و استقرار تنظیمات برای مدیریت متمرکز بسیاری از سیستم های کامپیوتری استفاده می شود.

شما می‌توانید فایل‌های ADMX را با استفاده از هر ویرایشگر متنی سازگار با XML، یا ویرایشگر شیء سیاست گروه مایکروسافت ایجاد یا ویرایش کنید.

## فرمت فایل ADMX - اطلاعات بیشتر

فایل‌های ADMX در قالب فایل جهانی [XML](/web/xml/) که برای انسان قابل خواندن است ذخیره می‌شوند. این فرمت فایل چند زبانه است و به مدیران سیستم از زبان های مختلف اجازه می دهد تا با فایل های ADMX یکسان کار کنند. فایل‌های ADMX جایگزین فرمت فایل قدیمی [ADM](/system/adm/) شدند که فرمت فایل متنی با قالب یونیکد است. فایل های ADMX کلیدهای رجیستری را مشخص می کنند که با تغییر تنظیمات Group Policy خاص تغییر می کنند.

### با فایل ADMX چه کاری می توانم انجام دهم؟

فایل های ADMX تعریف می کنند که چه اقداماتی باید به رایانه های کاربر که بخشی از یک گروه هستند مجاز باشد. خط مشی گروه قوانین تنظیم شده در ترکیب با نرم افزار Active Directory را تحمیل می کند. فایل های ADMX از فروشگاه مرکزی سیستم عامل ویندوز که یک مکان مرکزی در دامنه است مدیریت می شوند.

یک فایل ADMX مستقر در یک محیط کاری می تواند به مدیران اجازه دهد سیاست هایی مانند:

 * کنترل استفاده از اینترنت
 * دانلود انواع خاصی از فایل ها
 * استفاده از دستگاه های متحرک در سیستم ها
