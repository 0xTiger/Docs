## فایل REG چیست؟

فایل REG فرمت فایلی است که برای وارد کردن یا صادر کردن داده‌های رجیستری ویندوز استفاده می‌شود. رجیستری ویندوز یک پایگاه داده سلسله مراتبی است که تنظیمات و گزینه های پیکربندی را برای سیستم عامل ویندوز و برنامه های نصب شده ذخیره می کند. رجیستری حاوی اطلاعاتی مانند تنظیمات برگزیده کاربر، تنظیمات برنامه، داده های پیکربندی سخت افزار و نرم افزار و غیره است.

یک فایل REG به طور معمول دارای پسوند فایل .reg است و یک فایل متنی ساده است که شامل یک سری از ورودی ها و مقادیر رجیستری در یک قالب خاص است. فرمت شامل یک بخش هدر است که فایل را به عنوان یک فایل رجیستری شناسایی می کند و به دنبال آن یک سری جفت کلید و مقدار که ورودی های رجیستری را نشان می دهد.

## فرمت فایل REG - اطلاعات بیشتر

در اینجا نمونه ای از فرمت فایل reg آمده است:

```
[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Run]
"Example"="C:\\Example.exe"

[HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced]
"Hidden"=dword:00000001
```

خط اول فایل، نسخه ویرایشگر رجیستری را مشخص می کند. خطوط زیر حاوی ورودی های رجیستری در قالب یک مسیر کلیدی است که با نام مقدار و داده های مقدار دنبال می شود. در این مثال، فایل reg شامل دو ورودی است: یکی که برنامه ای به نام Example.exe را به لیست راه اندازی ویندوز اضافه می کند و دیگری که مقدار Hidden را در تنظیمات پیشرفته Windows Explorer روی true تنظیم می کند.

فایل های Reg را می توان با استفاده از یک ویرایشگر متن مانند Notepad ایجاد و ویرایش کرد. آنها اغلب برای اهداف پشتیبان گیری و بازیابی و همچنین برای پیکربندی چندین رایانه با تنظیمات رجیستری یکسان استفاده می شوند.

## رجیستری ویندوز را وارد یا صادر کنید

فایل REG نوعی فایل است که برای وارد کردن یا صادر کردن داده ها از رجیستری ویندوز استفاده می شود. رجیستری ویندوز یک پایگاه داده سلسله مراتبی است که تنظیمات و گزینه های پیکربندی را برای سیستم عامل ویندوز و برنامه های نصب شده ذخیره می کند. رجیستری حاوی اطلاعاتی مانند تنظیمات برگزیده کاربر، تنظیمات برنامه، داده های پیکربندی سخت افزار و نرم افزار و غیره است.

فایل‌های Reg را می‌توان برای ایجاد یا تغییر ورودی‌های رجیستری استفاده کرد، و اغلب برای اهداف پشتیبان‌گیری و بازیابی، و همچنین برای پیکربندی چندین رایانه با تنظیمات رجیستری یکسان استفاده می‌شوند. در اینجا نحوه استفاده از یک فایل reg برای افزودن یک ورودی جدید رجیستری به رجیستری ویندوز آمده است:

1. با استفاده از یک ویرایشگر متنی مانند Notepad یک فایل متنی جدید ایجاد کنید.
2. ورودی رجیستری را با فرمت صحیح وارد کنید. فرمت شامل یک بخش هدر است که فایل را به عنوان یک فایل رجیستری شناسایی می کند و به دنبال آن یک سری جفت کلید و مقدار که ورودی های رجیستری را نشان می دهد. در اینجا یک مثال است:

```
Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\Software\Example]
"Setting1"="Value1"
```

این یک کلید جدید به نام Example در زیر کلید Software در رجیستری کاربر فعلی ایجاد می کند و مقدار Setting1 را روی Value1 تنظیم می کند.

3. فایل را با پسوند فایل .reg ذخیره کنید.
4. روی فایل reg دوبار کلیک کنید تا ورودی رجیستری جدید به رجیستری ویندوز اضافه شود.

از شما خواسته می شود تا تأیید کنید که می خواهید ورودی را به رجیستری اضافه کنید. پس از تأیید، ورودی جدید به رجیستری اضافه می شود و می توانید آن را با استفاده از ابزار ویرایشگر رجیستری تأیید کنید.
