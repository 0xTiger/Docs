## فایل ETL چیست؟

فایل‌های ETL فایل‌های گزارشی هستند که حاوی گزارش‌های رویداد هستند که توسط هسته سیستم عامل مایکروسافت تولید شده‌اند. این فایل های گزارش شامل خطاهای برنامه و سطح سیستم، هشدارها و سایر داده های رویدادها هستند. فایل های ETL در تجزیه و تحلیل و عیب یابی مشکلات سطح سیستم مفید هستند. مشکلات رایج ثبت شده در فایل های ETL شامل مواردی است که توسط دسترسی ها به دیسک و خطاهای صفحه ایجاد می شود. مایکروسافت دو برنامه کاربردی Tracerpt و Event Viewer را برای خواندن و مشاهده محتویات فایل های ETL ارائه می کند. فایل های ETL را می توان با استفاده از Tracerpt به فرمت های فایل دیگری مانند [TXT](/word-processing/txt/) و [CSV](/spreadsheet/csv/) تبدیل کرد.

## فرمت فایل ETL

فایل‌های ETL برای کاهش فضای دیسک در فرمت باینری فشرده روی دیسک ذخیره می‌شوند.

## فایل های ETL را با استفاده از Windows Performance Analyzer باز کنید

داده های فایل های ETL را می توان با استفاده از برنامه Microsoft Windows Performance Analyzer (WPA) در قالب جدولی و گرافیکی خواند و تجسم کرد. راهنمای [Opening and Analyzing ETL files](https://learn.microsoft.com/en-us/windows-hardware/test/wpt/opening-and-analyzing-etl-files-in-wpa) اطلاعاتی در مورد کار با فایل های ETL ارائه می دهد.
