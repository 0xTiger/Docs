## فایل سه بعدی چیست؟

یک فایل با پسوند .3d یک فایل GIS است که حاوی داده هایی در مورد سیستم غار است که با نرم افزار Survex قابل مشاهده است. این فایل با تمام اطلاعات مربوط به سیستم های غار ساخته شده است که می توان از آن برای رندر کردن این فایل اطلاعات جغرافیایی با استفاده از نرم افزارهایی مانند Aven استفاده کرد. Survex یک نرم افزار متن باز رایگان است که برای نقشه برداری غارها استفاده می شود و می تواند بر روی پلتفرم های مختلفی مانند macOS، Linux و Microsoft Windows اجرا شود.

## فرمت فایل 3D GIS - اطلاعات بیشتر

The Survex 3D GIS files are in Survex 3D File Format whose [file format specifications](https://survex.com/docs/3dformat.htm) are available for developer's reference. These specifications have been used successfully to read the Survex 3D GIS file format.

### سربرگ فایل سه بعدی Survex

هدر فایل Survex 3D از فیلدهای زیر تشکیل شده است.

 * شناسه فایل: رشته Survex 3D Image File و به دنبال آن یک خط تغذیه (اعشار 10، هگز 0a) است.
 * نسخه فرمت فایل: این نسخه فرمت فایل را توصیف می کند. نسخه فعلی 8 است و به همین دلیل است که این فیلد دارای مقدار v8 است. هر نسخه بعدی از فرمت فایل v9، v10 و غیره خواهد بود.
 * MetaData رشته مرتبط: این باید به ترتیب مشخص شده در سند ظاهر شود و باید از ترتیب پیروی کند.
 * سیستم مختصات: رشته PROJ4 که سیستم مختصات در حال استفاده را توصیف می کند.
 * مهر زمان: رشته‌ای متشکل از یک «@» که از ابتدای سال 1970 در UTC («Unix time_t») به‌عنوان یک رشته (به‌عنوان مثال: «@1371300355»)، به‌دنبال آن یک فید خطی، تعداد ثانیه‌ها را دنبال می‌کند. در نظر گرفته شده است که این زمان تولید فایل باشد، نه زمانی که داده های نظرسنجی جمع آوری شده است.
 * پرچم های گسترده فایل: یک بایت. اگر بیت 7 تنظیم شده باشد، این یک ارتفاع گسترده است. همه بیت های دیگر رزرو شده اند - هنگام نوشتن آنها را روی 0 تنظیم کنید و هنگام خواندن آنها را نادیده بگیرید.
