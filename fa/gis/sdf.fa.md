## فایل SDF چیست؟

A Spatial Data File (SDF) is a Geodatabase file format that was developed by Autodesk. It stores geospatial data and is used by Autodesk GIS program MapGuide and AutoCAD Map 3D. SDF file format is based on the single file database file format SQLite3. این به عنوان یک فرمت فایل بهینه شده برای مجموعه داده های بزرگ اطلاعات مکانی در نظر گرفته می شود.

می‌توانید یک فایل SDF را با استفاده از Autodesk AutoCAD Map 3D 2022 و Autodesk AutoCAD Civil 3D 2023 باز کنید.

## فرمت فایل SDF - اطلاعات بیشتر

فرمت فایل SDF به عنوان فایل های باینری روی دیسک ذخیره می شود. این بر اساس پایگاه داده SQLite است که از اجزای ذخیره سازی سطح پایین برای سریال سازی باینری داده ها استفاده می کند. یک فایل SDF شامل چندین کلاس ویژگی در هر فایل و چندین ویژگی هندسی برای هر کلاس ویژگی است. داده های موجود در یک فایل SDF به دلیل نمایه سازی هر ویژگی هندسه با استفاده از درخت R با سرعت بالا قابل دسترسی است.
