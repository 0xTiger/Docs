## What is a TEACHER file?

A .TEACHER file is an educational settings file created by SMART Response software. It is used in educational setups to provide a complete environment of managing classroom information, class lists, schedules, holidays, student profiles, exams routine, and assessment results. SMART Responseâ„¢ assessment software is installed on the computer that will be used to deliver SMART Response assessments.

## TEACHER File Format

TEACHER files are stored to disc in binary file format. Its internal file format specifications are not available publicly for developer's reference. The SMART Response software can be used to generate student reports. These can then shared between school computers.
