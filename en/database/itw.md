## What is a ITW file?

An ITW file is a database file format used by the [InTouch](https://www.intouchsoftware.co.uk/InTouch) CRM application. It stores contents in InTouch's Proprietary file format and its internal file format details are not available. ITW files store customized information for storing names, addresses that are further used for printing envelopes and labels. This informaiton is also used for sending emails, text messages, attractive e-newsletters, creating reusable templates, attachments and links. Data from CRM can also be expored to ITW format.
