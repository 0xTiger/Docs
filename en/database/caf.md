## What is a CAF file?

A CAF file is a catalog file created with the media organizing software tool, **Cathy**. It contains a list of indexed catalog that points to all the available media on externa devices sucha s flash disks and CDs. These catalog files help you maintain a clean list of media that is easily searchable and accessible. CAF files offer to search media by name, size, and date.

## CAF File Format

CAF files are stored in text file format. This means you can open CAF files in any text editor such as Notepad, Notepad++ and Apple TextEdit. Cathy provides the feature of exporting the directory structure of each catalog to TXT format, making lists of folders and files.

### Open CAF File

You can open CAF with the official Cathy software. This will need you to place your CAF file in the same directory where Cathy.exe file lies. Cathy will automatically show the catalog when launched.

To do so, place your CAF file in the same directory as your Cathy.exe file. Then, open Cathy. Your CAF catalog will appear in Cathy.
