## What is a BC3 file?

A BC3 file is a database file for storing information related to construction database exchange format. It was developed based on the Spanish standard construction database exchange format. A BC3 file stores information that has data records including details construction materials, cost of materials, and price of labor. The data cna be exchanged between other applications without converting the data to other formats.

## BC3 File Format

BC3 files are stored as binary files with details specified as per the FIEBDC standards. The implemenation of BC3 files have went through different reviesions and the following versions of BC3 file format exist:

 * FIEBDC-3/2004
 * FIEBDC-3/2007
 * FIEBDC-3/2012
 * FIEBDC-3/2016

FIEBDC is short for Formato de Intercambio Estándar - Bases de Datos de Construcción in Spanish language.
