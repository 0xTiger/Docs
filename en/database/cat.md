## What is a CAT file?

CAT file belongs to Advanced Disk Catalog software, developed by ElcomSoft Co. Ltd, Russian software company that develops various computer security and forensic software tools. Some of its products include software for password recovery, mobile device acquisition, and cloud extraction. The company was founded in 1990 and has offices in Russia and the United States. They are well known for their forensic software and overall their software is used by both private companies and government agencies to perform forensic analysis and investigations.

Advanced Disk Catalog is a cataloguing program for Windows XP/2003. ADC's versatility is derived from the fact that it doesn't rely on a database engine, making it highly efficient and compact. Its user interface, similar to the Windows Explorer, allows for easy organization of data stored on various types of media such as hard drives (including networked drives), floppy disks, optical disks, ZIP and JAZ disks, etc.

## How to open CAT files?

Download the Elcomsoft Advanced Disk Catalog software. On Windows, you can right click on file explorer and then select open with context menu, specify the path of the Elcomsoft Advanced Disk Catalog software to open the cat file.
