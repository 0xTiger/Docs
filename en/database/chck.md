## What is a CHCK File?

A CHCK file is created as a result of health monitoring checks on Microsoft Exchange Server. It contains informaiton about the overall status and operationalization of the server's database. Microsoft Exchange Server is used primarily as emailing server and lets you maintain your business contacts and calendars. Exchange server administrators use its EseUtil to check it's database health.

## CHCK File Format

The internal file format of CHCK file is not known and there is no reference material available for reading/writing CHCK files.
