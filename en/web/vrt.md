## What is a VRT file?

A VRT file is a 3D file format that contains three-dimensional worlds that can be viewed in web browser with [Viscape](https://www.viscape360.com/) SVR plugin. It is used by the plugin to load and display the 3D world data to user for visualization and analysis. Information stored in a VRT file may comprise of data such as textures, sounds, dimensions, and behaviors. The most recent VRT files are stored in SVR file format which is a compressed version of the VRT file.

## VRT File Format

VRT files are stored in Viscape proprietary file format and their internal file format details are not available publicly.
