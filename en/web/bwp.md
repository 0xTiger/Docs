## What is a BWP file?

A BWP file is a web page script file created in the programming language, **BuRg3r**. It is specific to BuRg3r and may contain HTML code with sections of BuRg3r source delimited by <?b and ?> tags. BuRg3r is a general purpose programming language that is also a game engine that was developed by Fluffy Smoke Entertainment.
