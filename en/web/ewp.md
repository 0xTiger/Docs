## What is an EWP file?

An EWP file is a project file created with [EWISoft](http://www.ewisoft.com/) Website Builder software. It is linked to a binary database file that contains all the information for the project. EWP file is commonly used for saving web development progress. It is a template-driven web design software that lets users create websites from website templates.

## EWP File Format - More Information

EWP files are project files that don't need to be compiled or hosted on any server. These are saved as plain text files and contain references to web development progress. Projects can be published to a website once the development is complete.
