## What is a HYPE file?

A HYPE file is a project file saved using the Hype software that is used for creation of web content with HTML5. It contains web elements such as graphics, text, animation, scene information and other elements that collectively make up the content. Web content in a HYPE file can be exported to web files that can be published on a webpage.

A HYPE file can be opened with [Tumult Hype](https://tumult.com/hype/) software.

## HYPE File Format

HYPE files are stored in proprietary file format and the internal details of these are not known publicly. The contents inside a HYPE file can only be viewed when opened in the Tumult Hype software.

### Contents of a HYPE File

A HYPE file is created when you are using the Hype application to create web content for different purposes. It stores information about keyframe-based animations for creating contents such  as eCards, eBooks, presentations, advertisements, infographics and webpage material. The content of these is powered by HTML5 that lets the content displayed on devices from desktops to iPhones and iPads. Tumult Hype outputs state of the art HTML5, [CSS3](/web/css/) styles, and [JavaScript](/web/js/).
