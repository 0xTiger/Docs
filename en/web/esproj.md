## What is an ESPROJ file?

An ESPROJ file is a project file created by webpages creating software, Espresso. It is available on Mac OS and contains a list of the related files that are part of the website project. When you open the ESPROJ file in Espresso, it opens all the related files in the editor for editing or updating the source code. ESPROJ file also contains server publishing settings and other related user preferences.

## ESPROJ File Format

ESPROJ files are saved as binary files and its contents are not human-readable. The ESPROJ file directory is also the root directory for the website.
