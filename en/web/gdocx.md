## What is a GDOCX file?

A GDOCX file is a type of shortcut file created that points to an online word processing documents stored in Google Drive. It is created with Google Docs and is saved to the user's Google Drive by default. GDOCX files are similar to [GDOC](/web/gdoc/) that are most commonly used. The target file information is stored in GDOCX files in plain text that can be opened with any text editor.

## GDOCX File Format - More Information

GDOCX files are similar to GDOC and contains the target URL that is to be opened. These reside in the Google Drive client folder on user system and remains synchronized to Google Drive folder online. Google provided the desktop client version of Google Drive until 2018 when it was finally discontinued. However, users can still create and store their files to online Google Drive online storage.
