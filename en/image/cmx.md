Files with CMX extension are Corel Exchange image file format (also known as Corel Metafile Exchange) that is used as presentation by CorelSuite applications. It contains image data as vector graphicsÂ as well as metadata that describes the image. CMX files can be opened by CorelDraw, Corel Presentations, Paint Shop Pro and some versions of Adobe Illustrator. CMX files can be converted to other formats such as JPG and EPS.

# CMX File Format #
CMX file format is based on the Resource Interchange File Format (RIFF) and supports both Intel and Motorola binary formats. CMX file uses the following MIME types:
  * image/x-cmx
  * application/x-cmx
  * application/cmx
  * drawing/cmx

## File Format Header ##
CMX files start with "RIFF" or "RIFX", and have "CMX1" at offset 8.
