## What is a APNG file?

A file with .apng extension is a raster graphic format and is unofficial extension to the Animated Portable Network Graphic (APNG). It comprises of multiple frames (each of .PNG image) that represent an animation sequence. This gives similar visualization as a .gif file. APNG files support 24-bit images and 8-bit transparency.
