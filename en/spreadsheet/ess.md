## What is an ESS file?

An ESS file is a spreadsheet file created and supported by EasySpreadsheet spreadsheet application software that comes as part of [EasyOffice](https://en.wikipedia.org/wiki/EasyOffice). EasyOffice was launched as alternative to Microsoft Office but is no more available for use and has been replaced by another suite, called One. It came in two variants namely EasyOffice Freeware and EasyOffice Premium with the later having a PDF Filter and EasyAntiVirus. ESS file is compatible with Microsoft [XLS](/spreadsheet/xls/) files.
