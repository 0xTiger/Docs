## What is an EFU file?

An EFU file is a [CSV](/spreadsheet/csv/) spreadsheet file created with the Everything software by [voidtools](https://www.voidtools.com/). It contains list of files searched using the Everything search engine for windows. Results in an EFU file are CSV-delimited and can be opened with spreadsheet programs such as Microsoft Excel. EFU files are hunman-readable and can be opeend with any text editor such as Microsoft Notepad, Notepad++, and Apple TextEdit.

## EFU File Format - More Information

EFU files are saved to disc in comma separated values file that contains information about the results obtained by the Everything search engine application. These records contain information about the found files such as file name, size, creation and last modified date.
