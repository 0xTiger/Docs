Note-taking program files contain sections and pages for storing notes. A note document can be as simple as a text document as well as more detailed consisting of digital images, audio/video clips, and hand sketch drawings.

Common OneNote file extensions and their associated file formats include **[.ONE](/note-taking/one/)** and its table of contents file **[.ONETOC2](/note-taking/onetoc2/)**.

Got queries related to Notes file formats? Head on to our community [forums](https://forum.fileformat.com/c/note-taking/23) to benefit from the knowledge shared by File Format experts.
