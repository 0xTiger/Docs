## What is an BKK file?
The files with BKK extension are being used by **BookBuddi** software and these files can be read back using Microsoft Text to Speech Agent. These files may include text, images and may consist of multiple pages. When the text is being read the respective section is highlighted and pages are turned automatically.

## How a BKK file works
The BookBuddi software uses files with the BKK extension for the regular file format of projects created with this app. The BookBuddi software is an e-book development and publishing software developed by BookBuddi. This application was developed for Microsoft Windows-based platform. When the BookBuddi application is installed by the user in Microsoft Windows system, support for creating, opening and viewing the text and images stored in the BKK digital books will be implemented into the computer. The Microsoft Text to Speech Agent, an application belongs to the  systems, is helpful in aiding users in accessing digital content stored in these BKK ebooks, since this application supports a feature which read back the text content of an e-book into spoken audio. The content of these BKK files is referenced by the BookBuddi app when a user views or edit the associated BKK file. These reference details also include text and image formatting attributes among other page layout elements entered by the creator of the BKK ebook.
