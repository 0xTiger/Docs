## What is a PWF file?

A file with .pwf extension is a waypoints file that contains "Ping of Death" bots (PODBots) information for Counter-Strike game. PODBots know the goals of the game current level and use these waypoints information to move and achieve the goals. In general terminology, waypoints are the points to be followed to reach the destination or a specific point. PWF files contains these waypoints, or locations that guide the players to reach the destination. There are several plugins available that can add computer players (acting as bots) to the Counter-Strike game.

## PWF File Format

There is not much information available about the Counter-Strike PODBot Waypoint file format. Players can join Counter-Strike game on network over the network on or on a LAN without the necessity of installing a Bot.
