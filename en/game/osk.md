## What is an OSK file?

An OSK file  is a layout or skin file used by the rhythm game osu! where players use mouse, keyboard in sync with music. It contains the layout of the editor interface and can be customized by the players to suit their preferences. osu! stores beatmap data in [OSZ](/game/osz/) file format.

**MIME Type of OSK File Format:** x-osu-skin-archive

You can open OSK file with [osu!](https://osu.ppy.sh/home).

## OSK File Format

OSK files are saved to disc as [ZIP](/compression/zip/) compressed files. You can simply remain a .osk file to .zip and extract the archive to view its contents. The extracted contents need to be placed in \Skins\ directory to be loaded in the game.

## What can be customized with OSK file?

The following game elements can be customized with .osk file.

 * Hit circles
 * Score display
 * Slider type
 * Cursor
 * Hit bursts
