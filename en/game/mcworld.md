## What is a MCWORLD file?
A MCWORLD file consists of resource and behavior files to create a virtual world in Bedrock Edition of Minecraft game which is a famous block-building game. It compressed various files, such as .DAT and .TXT files with ZIP compression. the files uses .mcworld file extension and these are particularly used to backup or transfer a world to another Minecraft user. These files are usually created by a Minecraft software called Minecraft Launcher.

## MCWORLD file format
The MCWORLD files are created by **MinecraftSP.exe** which is also called Minecraft Launcher. This file can be created if we zipped or compressed the world files into a single file and change the extension to .mcworld. On the other side we can expand the .mcworld files using simple zip decompressor utility softwares, because .mcworld files are compressed using zip compression.
