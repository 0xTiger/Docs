## What is an NSV File?

An NSV (NullSoft Stream Video) file is a streaming video file format used by NullSoft for streaming videos on the internet. The format was introduced and developed by NullSoft who was also the maker of Winamp media player. NSV supports [MP3](/audio/mp3/) and VP3 as its default audio and video formats. Lateral enhancements lets NSV to use lateral version of VP codecs for video such as [VP6](/video/vp6/), [VP8](/video/vp8/), and [VP9](/video/vp9/). NSV file format is no more supported after NullSoft was acquired by Radionomy in 2014.

## NSV File Format

NSV is a steaming container video file format that was developed for streaming videos over the internet. It's availability via SHOUTcast's listings offers integrated audio and video streams. Individual audio and video streams can be extracted and converted to MP3 and [MP4](/video/mp4/) file formats using VLC media player.
