## What is an VIM file?

A VIM file is a configuration/settings file used by the text editing program known as Vim. It stores settings of the editor such as syntax highlighting and are used by application developers to configure the appearance of the editor when it is opened. Vim text editor is highly configurable and is built for creating and changing of text efficiently. It comes bundled as "vi" with most UNIX systems.
