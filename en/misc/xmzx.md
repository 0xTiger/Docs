## What is an XMZX file?

An XMZX file is an exam test file for conducting tests and is used by the ExamSoft SofTest software application. The application can be used on both Windows and macOS. It contains questions from an exam that the examiner can take from students on a computer with the SofTest sfotware. XMZX files are sent to examiners for conducting the test. It is loaded in the SofTest software and the candidate or applicant takes the test. Upon completion of the test, the finalized XMDX file is automatically uploaded to ExamSoft's cloud servers.

## XMZX File Format - More Information

XMZX files are used by the digital assessment platform, ExamSoft that is used by variety of organizations to conduct tests for certifications. The platform has its own online cloud servers where all the completed tests are uploaded automatically. The tests are conducted either at an examination center or examiners can also take the test form candidates on their own computers. ExamSoft is available both on Windows and macOS operating systems. XSMX is another exam test file format by ExamSoft.

## How to Open an XMZX file?

You can open an XMZX file in ExamSoft SofTest software only as it is a proprietary file format. SofTest software has the option to **Take an Exam** that is used to start the exam by selecting the XMZX file.
