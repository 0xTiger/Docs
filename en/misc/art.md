## What is an ART file?

An ART file is an embroidery design file created with the [BERNINA](https://www.bernina.com/en-US/Home-United-States) ARTlink embroidery software version 1-4. It contains user created designs that can include different objects such as outlines, design layout, stitches, colored threads, and other metadata. The ART file is loaded into BERNINA sewing machine and can be stitched to different types of fabrics such as quilt, shirt, or any other cloth. ART files can also be shared with other users using storage devices such as USB or transferred via internet.

## ART File Format

ART file format contains the following information:

 * Object Properties
 * Stitch Data
 * Stitch Types
 * Stitch Coordinates
 * Thread Colors
 * Comments
 * Pull Compensation Settings
 * Auto Spacing Settings
