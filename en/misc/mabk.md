## What is an MABK file?

An MABK file is a Motorolla Android Backup file created by the Lenovo Android Management utitlity, Lenovo Moto Smart Assistant. It is a backup file containing a copy of the applications, contacts, text messages, and media files stored on the phone. MABK file does not save ifnormation user's application settings.

## MABK File Format

MABK files are generated using the Lenovo Moto Smart Assistant (LMSA) application on MOtorola phone, Lenovo phone, and Lenovo tablets. It is a windows application and stores the backup data on user's devices. The applicaiton, when executed, connects to user's device and takes creates the MABK file when the **Backup & Restore** option is selected.

End users can save the MABK files on their PC for using later on if they require to restore the content from the backup file. Users, usually, back up all files available on the device such as contacts, text messages, images, audios, videos, and apps.

## How to Open MABK files?

You can utilize an MABK file for the restoration or transfer of data on your Motorola phone, Lenovo phone, or Lenovo tablet using Lenovo Moto Smart Assistant for Windows. To accomplish this:

1. Connect your mobile device to your PC.
2. Launch Lenovo Moto Smart Assistant.
3. Navigate to Backup & Restore, then choose Restore.
4. Import your MABK file.
5. Select Restore and follow the on-screen instructions to complete the data restoration or transfer process.

In addition, you can manually access the files stored within an MABK file. For instance, you can extract the pictures contained in the MABK file and view them on your PC. To extract files from an MABK file:

1. Open Lenovo Moto Smart Assistant.
2. Go to Backup & Restore.
3. Select Restore.
4. Import your MABK file.
5. Choose Extract and follow the on-screen prompts to extract files from your MABK file.
