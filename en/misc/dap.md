## What is an DAP file?

A DAP file is a download-in-progress file created by Download Accelerator Plus (DAP) file. It is created when users initiates a download from any online file repository using DAP software. DAP provides the capability to pause and resume a download that is in progress. For this purpose, it saves the partially downloaded file with .dap extension until it is completely downloaded. The DAP file can be previewed during the download process.

DAP files can be opened with SpeedBit Download Accelerator Plus (DAP) software.
