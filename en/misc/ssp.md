## What is an SSP file?

An SSP file is a project file created by the Windows screensavers creation software known as Screensaver Producer by Axialis Software. It can contain several different type of screensaver elements such as sprites, Flash movies, photo slideshows and videos. An SSP file may comprise of text, images, different shapes, and animation settings used to show the animation inside the SSP file. SSP files can be opened with Axialis Screensaver Producer.

## SSP File Format - More Information

SSP files are stored to disc as binary files with no public information available about their internal file format structure. Axialis Screensaver producer professional screensaver authoring tool for Windows which lets you easily create hi-quality screensavers for all versions of Windows up to Windows 10. It allows creating installable screensaver packages that can be easily distributed online to users.

SSP files can be exported to [SCR](/system/scr/) file format.
