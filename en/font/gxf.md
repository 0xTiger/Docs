## What is an GXF file?

A GXF file is a font file by the CAD design tool for Windows, General CADD Pro. The software uses the font to insert text and other symbols into a CAD drawing. GXF fonts should be available to General CADD Pro software in order for a CAD drawing to be displayed properly.

You can open a GXF file using [General CADD Pro](https://www.generalcadd.com/oldSite/101release.htm).

## About General CADD Pro

General CADD Pro offers two-letter command structure that is incorporated and used by many drafting professionals. It supports ACAD 2013 file format. It also provides support for surveyor bearing/distance commands and Field to Finish command.
