## What is RVM file?

**RVM** data files are related to AVEVA PDMS. **RVM** file is an AVEVA Plant Design Management System Model project file. AVEVA's Plant Design Management System (PDMS) is the most popular 3D design system using data-centric technology for managing projects. Numerous applications are available to open and convert RVM files to other formats such as [3DS](/3d/3ds/).

## RVM File Format

Not much details can be found about the RVM file format other than that it can be saved in both binary and ASCII format. The RVM file supports entities such as:

 * All geometry
 * Attributes stored on groups
 * Textures (via RVS file)
 * Cameras and camera tracks (via RVS file)
 * Clip planes (via RVS file)
 * Signs (via RVS file)
 * Tags (via RVS file)
 * Labels (via RVS file)
 * Translucency (via RVS file)
 * PDMS origin points
