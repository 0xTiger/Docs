## What is a AVERY file?

An AVERY file is a printable document that contains one or more labels for printing purpose. It is created with the Avery Design & Print or Design & Print Online software. The labels inside an AVERY file may comprise of different element such as text, graphics, and page layout information. This information is used by printing companies for printing variety of design elements such as business cards, name tags, binder inserts, address labels, disc labels, obituary templates, and greeting cards.

## AVERY File Format - More Information

AVERY files are stored in binary file format. These can be opened with Avery Design & Print or Design & Print Online software. The designs created with Design & Print can be used to create AVERY labels that can serve a number of purposes. These designs can be saved as [PDF](/pdf/) files which can be shared with Prints company for printing purpose.

### Printing an AVERY File

An AVERY file can either be saved as PDF or can be directly printed to any standard printer. To print the file, open the file in Design & Print software and use the Print option from File menu to send the file to printer for printing.
