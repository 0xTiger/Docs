## What is an INDD file?

An file with .indd extension is an Adobe InDesign project file that contains the page layout information of the project. INDD stands for InDesign Document. These files commonly store page formatting information, books, magazines, newspapers, flyers, brochures or other print media like mobile & webpage mockups. INDD file type is proprietary file format of Adobe and is the default file format for InDesign. Graphic Designers, journalists, marketers, and publishers working with InDesign use this file type.

## More about INDD File Format

INDD files are vector files that can be scaled without losing resolution. These are based on complex graph-like that can be expanded infinitely. It packages all fonts and layouts information to be shared with a team.

INDD files are perfect for designing engaging print as well as online content. An INDD file can span on multiple pages and can serve as templates for creating pages, chapters, and publications. When your project is complete, it can be published to onilne media without the need of conversion, zipping or emailing.

## Can INDD file be Converted to Other Formats?

INDD files can be converted to several other file formats. The most common conversion is INDD to PDF. InDesign lets you save the INDD file to PDF for sharing with other users. It also lets you export INDD file to JPG, [EPS](/page-description-language/eps/), [EPUB](/publisher/epub/), [SWF](/page-description-language/swf/), FLA, [HTML](/web/html/), [XML](/web/xml/), and IDML.

## How is INDD File Created?

An Indd file is created when you create a new project with InDesign and save it to disc. You can open the Indd project file, make changes to it, and save the updated file for the changes.
