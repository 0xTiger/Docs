## What is an RBZ file?

An RBZ file is a plugin file used by the modeling program, **Trimble SketchUp**, for computer-aided design (CAD). It is a compressed ZIP file that contains one or more Ruby scripts (.rb files) inside it. That is why the file type is RBZ (RB + Z). RBZ files are easy to distribute and isntall over the network due to their small file size as result of compression.
