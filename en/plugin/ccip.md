## What is a CCIP file?

A CCIP file is a a plugin file created by the Curse Client program which is used for managing and installing add-ons for popular games like World of Warcraft, Minecraft, and others. Add-ons are user-created modifications or enhancements to a game's interface or functionality, and they can enhance the gaming experience by providing additional features or customization options. CCIP file contians hte required informaiton for downloading and installing the addon for one of hte managed games.

## CCIP File Format

When you select the "Install via Curse Client" option from the program, it downloads the files as CCIP files. These CCIP files can be downloaded via the Curse Client. The Curse Client allowed users to easily discover, download, and manage their game add-ons. It provided a centralized location for users to find and install the latest versions of their favorite add-ons.
