## What is an INX file?

An INX file is a plugin file used by the open source vector graphics editor, **Inkscape**. It contains information about the program including the name and the location of the functionality in menu. INX file also specifies a script for the extension that includes informatino such as script type, interpreter, location, and variable arguments.

## INX File Format

INX files enhance Inkscape's functionality by incorporating features like file importers and graphics filters. Stored in an XML format, these files are typically found in the \​extensions\​ directory within an Inkscape installation.

## How to Open INX files?

You can open INX files using the **Inkscape** program.
