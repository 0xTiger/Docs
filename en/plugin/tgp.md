## What is an TGP file?

A TGP file serves as a plugin utilized in Terragen Classic, a program designed for generating photorealistic natural scenery. This file encapsulates data that enhances Terragen's functionalities upon installation. TGP files are employed to introduce diverse features to Terragen, including camera effects and additional landscape options.

## About Terragen Classic

Terragen is a landscape generation and rendering software developed by Planetside Software. The "Classic" version refers to an older iteration of the software, as there is also a more recent version called **Terragen 4**.

Terragen Classic was known for its capabilities in generating realistic landscapes and terrains. It allowed users to create detailed and believable natural environments, including mountains, valleys, lakes, and skies. The software included features for terrain sculpting, texture mapping, and rendering.
