## What is a Q1Q file?

A .q1q file is a Photoshop plugin file developed by Flaming Pear Software for creating 3D planets. It lets you download LIVE clouds data, air, texture, climate and other tons of detials. It also lets you export all texture maps to multi-layer files and generate maps for 3D animation and mapmaking.

## Q1Q File Format

Q1Q plugin files are saved in Flaming Pear proprietary file format and its internal file format details are not available publicly. The following image shows the User Interface of Q1Q plugin when loaded in Photoshop.

!["LunarCell PluginFile Format"](../plugin/q1q.png "LunarCell PluginFile Format")

## How to Open Q1Q File?

You can open Q1Q files using **Adobe Photoshop 2023** and **Corel PaintShop Pro 2023**.
