## What is an AVX file?

AVX is a software extension utilized by Avid products like Media Composer and Xpress. It enhances software functionality by incorporating custom video effects and supporting the import of new file formats. This extension empowers third-party developers to expand upon the core features of the application.

## How to Open AVX file?

You can open AVX file using **Avid Media Composer** or **Avid Xpress**.
