## What is an AFPHOTO file?

An AFPHOTO file is an image file created by the image creation software, Serif Affinity Photo. It contains the overall contents of the image in the form of multiple image layers, shapes, text, paint strokes, swatches, coloring effects and lighting. Serif Affinity Photo is similar to Adobe Photoshop, rather it is an alternative to Adobe Photoshop with almost the same features and capabilities. Adobe Photoshop saves the files as [.psd file](/image/psd/) while [Affinity Photo](https://affinity.serif.com/en-gb/photo/) files are saved with .afphoto extension.

Serif Affinity Photo can open AFPHOTO files.

### AFPHOTO File Format

AFPHOTO file is saved in Serif Affinity Photo proprietary binary file format. Its internal file format structure and detailed specifications are not available for public reference.

## How to convert an AFPHOTO file?

AFPHOTO files can be converted to a number of different formats using the Serif Affinity Photo software. The following export file formats are supported.

 * [PSD](/image/psd/) - Adobe Photoshop Document
 * [PDF](/pdf/) - Portable Document Format
 * [JPG](/image/jpeg/) - JPEG Image
 * [PNG](/image/png/) - Portable Network Graphic
 * [GIF](/image/gif/) - Graphical Interchange Format
 * [EPS](/page-description-language/eps/) - Encapsulated PostScript
 * [SVG](/page-description-language/svg/) - Scalable Vector Graphics
 * [TIFF](/image/tiff/) - Tagged Image File Format
 * TGA - Targa Graphic
 * EXR - OpenEXR Image
 * [HDR](/image/hdr/) - High Dynamic Range Image
