## What is an ACD file?

An ACD file is a project file that contains information for programming [Rockwell Automation](https://www.rockwellautomation.com) Logix platforms. It includes instructions for controllers and processors. It is used for Logix platforms such as ControlLogix, CompactLogix, FlexLogix, SoftLogix5800, and DriveLogix systems.

Applications that can open ACD files include Rockwell Automation RSLogix 5000.
