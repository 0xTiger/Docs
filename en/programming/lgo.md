## What is an LGO file?

An LGO file a program code written with Logo application for drawing pictures. It contains programmed instructions that guide the reading application to make drawings according to these instructions. For example, the "turtle" program that can open LGO files, use these instructions to draw lines as per programmed code. The drawing is not limited to lines only. LGO files can be used for drawing simple to complex designs. Many educational programs used LGO files for teaching how to make drawings programmatically.

## LGO File Format

LGO files are saved in proprietary file format and their internal file structure is not known publicly. Another well known file format for logos is [.ico](/image/ico/) files.
