## What is a GMD file?

A GMD file is a game development source file generated with Game Maker software development tool that is used to create 2D games. The GMD file is stored as text file and is project file for the game to be developed. Game Maker is available for Windows, Mac, Linux, Android, iOS, HTML5, Xbox, PlayStation, and Nintendo Switch. The tool has extensive used by indie developers, professional studios, and educators worldwide for creating games.

## GMD File Format

GMD files are saved in binary file format and its internal file format details are not available for developer's reference.
