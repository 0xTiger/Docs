## About DOCX to RTF Conversion

If you have been an early age user of Microsoft Windows, you must know the RTF (Rich Text Format) file format. Microsoft introduced [RTF file format](/word-processing/rtf/) with Windows 3.0. RTF files represent a method for encoding formatted text and graphics for use within applications. An RTF file can contain text and images in addition to retaining text formatting such as Bold, Italic, and Underline. What's most interesting is RTF has been adopted as interoperable file format and is cross-platform. For this reason, DOCX to RTF conversion is a popular choice while sharing documents across different OS for editing purpose.
