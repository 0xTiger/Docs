## About DOCX to WEBP Conversion

WEBP is the modern image file format that has reduced file size without compromising file size. It is the most appropriate image file format that is used for web due to its small size and helps load the images faster. If you are using Microsoft Word for creating documents, these are saved in DOCX file format by default (since the launch of Microsoft Office 2007). DOCX to WEBP comprises of each page of a WORD document converted to [WEBP](/image/webp/) image that can be used indepedendtly. The output of this process is a set of WEBP iamges where each image is the graphical representation of the corresponding page from the DOCX file.
