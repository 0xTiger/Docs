## About DOCX to DOT Conversion

A DOT file is a Microsoft Word template file used to have pre-formatted settings. It is used to generate similar DOC or DOCX files with the predefined settings in it. DOT files are useful for quickly generating documents that should be based on the same template settings. Microsoft Word allows you to save DOCX files to DOT file format by using the **Save As** option.
