## About DOC to BMP Conversion

If you are a Microsoft Word user for its pre 2007 version, you must be familiar with the DOC file type that was the default file format used by Microsoft Word to save your documents by then. It had been in use since the start of Microsoft Word and remained there for quite long time after it was replaced with the latest DOC file format. Content Management Systems may still have DOC files as backup those can be converted to BMP images. Online tools such as Aspose.Word app for conversion lets you convert [DOC](/word-processing/doc/) files to [BMP](/image/bmp/) images.
