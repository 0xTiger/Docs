## About DOCX to TIFF Conversion

TIFF (Tagged Image File Format) is a raster image file format that are used on a variety of devices that comply with the TIFF file format. Microsoft Word users save their documents in the default DOCX file format. DOCX to TIFF converts all the pages of a Word document to TIFF images for display on [TIFF](/image/tiff/) compliant devices. Online conversion tools and apps such as Aspose.Words app can convert DOCX files to TIFF images.
