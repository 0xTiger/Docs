## What is a Toast file?
The Roxio Toast software creates CD or DVD image and saves the file with .toast extension. Roxio Toast is a well known optical disc authoring and media conversion software application for Macintosh first introduced by Miles Software GmbH, then bought by Adaptec. The Toast file may contain the any kind of discs such as video, audio or data disc also It might be a copy of and another disc image.

## Toast file format
The Toast file format is used by Roxio Toast software to save its output in a file with .toast extension. Although, the discs of various types can be burned directly through Mac OS systems, but Toast provides additional control over the process, and also offers some extra features, such as cataloging, tracking of files burned to disc and file recovery for damaged discs.It also provides support for audio and video formats that does not supported by Quicktime, such as [FLAC](/audio/flac/) and [OGG](/audio/ogg/). Since it is a  proprietary file format, the information about the entire structure of the file format is not available.  
### Types of discs
Therefore, the Roxio Toast can create Toast files, containing the disc image data of the following types:
- **Making Video Discs**: Toast can create Toast files containing video discs by using following projects:
	- DVD-Video
	- Blu-ray Video
	- High Definition DVD
	- VIDEO_TS Folders
	- VIDEO_TS Compilation
	- BDMV Folder
	- AVCHD Archive
- **Making Data Discs**: Toast can create Toast files containing data discs by using following projects:
	- Mac Only
	- Mac & PC
	- DVD-ROM
	- Custom Hybrid
	- Mac Volume
	- Photo Disc
- **Making Video Discs**: Toast can create Toast files containing data discs by using following projects:
	- Audio CD
	- Music DVD
	- MP3 Disc
	- Enhanced Audio CD
	- Mixed Mode CD
- **Copying Disc**:  Toast supports copying in the following ways:
	- Disc Copy
	- Image Copy
	- Disc Image Merge
