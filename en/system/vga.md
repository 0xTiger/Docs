## What is a VGA file?

A VGA file is a Windows drive file for supporting VGA monitors. It was used in early versions of Microsoft Windows for adjustment of VGA monitors at Operating System (OS) level. A VGA file used to allow users to adjust the resolution and color depth of display using Monitor's control panel.

## VGA File Format

The internal file format of VGA file format is not known for developer's reference. It is referenced by Windows operating system for setting the visual appearance of the VGA monitor.
