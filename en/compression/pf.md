## What is a PF file?

A PF file is a compressed and encrypted file that is created with Stuffit Deluxe software. It is password-protected file and can only be decrypted and decompressed by applying the password. Stuffit Deluxe is a well known file compressor and expander for Mac OS and can handle more than 30 file formats including [RAR](/compression/rar/), [7zip](/compression/7z/), and other popular file formats. PF files can be opened with Stuffit Deluxe software.

## PF File Format

PF files are saved to disc as binary files and are saved in the Stuffit Deluxe proprietary file format.
