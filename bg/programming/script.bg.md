## Какво е SCRIPT файл?

SCRIPT файлът е изходен файл, който съдържа скрипт на всеки скриптов език. Това го прави обобщен скриптов файл, който съдържа променливи и функции. Тъй като всеки скриптов език може да е различен, скриптът, написан във SCRIPT файл, трябва да се изпълни с помощта на подходяща скриптова машина.

## SCRIPT файлов формат - повече информация

SCRIPT файловете се съхраняват на диск в обикновени текстови файлове. Те могат да се отварят и редактират с всеки текстов редактор като Microsoft Notepad, Notepad++ и Apple TextEdit. Тъй като скриптовият език може да бъде различен, той може да бъде изпълнен чрез преименуване на файловото разширение на разширението по подразбиране на съответния скриптов език. Например има няколко често срещани скриптови езика като JavaScript ([.JS](/bg/web/js/)), VBScript ([.VBS](/bg/programming/vbs/)), AppleScript (.SCPT), PHP ([ .PHP](/bg/programming/php/)) и ASP ([.ASP](/bg/web/asp/)).
