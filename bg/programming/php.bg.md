## Какво е PHP файл?
Файл с разширение .php се отнася до език за програмиране с отворен код, използван за писане на скриптове от страна на сървъра, които да се изпълняват на уеб сървър. Това е най-широко използваният език за уеб скриптове, който обикновено се използва за разработване на широкомащабни уеб приложения. Най-голямата система за блогове в мрежата, т.е. WordPress и най-голямата социална мрежа Facebook, са базирани на PHP език. Съвместим е с различни платформи като Windows, Linux, Unix, Mac OS X и т.н. Също така успешно работи на почти всички сървъри, използвани днес, т.е. Apache, IIS и т.н. PHP поддържа широка гама от бази данни, включително MySQL.

## PHP файлов формат ##

Един PHP може да съдържа текст, CSS, [HTML](/bg/web/html/), JavaScript и PHP код. PHP кодът се изпълнява на сървъра и резултатът се връща в браузъра като обикновен HTML. PHP файловете могат да се отварят с всеки текстов редактор и да се редактират на място, въпреки че приложения като Adobe Dreamweaver, Eclipse PHP инструменти за разработка предлагат удобен начин за писане и модифициране на PHP код.

### Синтаксис ###

На сървъра се изпълнява PHP скрипт и обикновеният HTML резултат се изпраща обратно към браузъра. PHP скриптът започва с<?php and ends with ?>

### Пример за PHP ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
