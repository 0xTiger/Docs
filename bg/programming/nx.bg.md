## Какво е NX файл?

NX файлът е файл с видео игра, създаден за използване с виртуалната конзола за игри, **LowRes NX**. Написан е на езика за програмиране BASIC и съдържа изходния код на играта, който определя как функционира играта. Можете да създадете свои собствени NX файлове за използване с приложението, които могат да бъдат заредени и потребителите да могат да играят играта.

Можете да отворите NX файл в текстов редактор, LowRes NX или всеки редактор на изходен код, като Microsoft Visual Studio Code.

## NX файлов формат

NX файл се записва на диск като обикновен текстов файл, който съдържа инструкциите на играта. Всеки NX файл има изходния код на играта, който се състои от множество записи в ROM касета. Тези ROM записи съдържат информация за компонентите на играта като графики, карти на нивата, аудио и други данни за играта, които не са изходен код. Записите в ROM на дадена програма могат да се добавят или редактират с помощта на NX програми за редактиране, които идват в пакет с LowRes NX.

## Как да създадете NX файлове?

### Създавайте NX файлове на компютър

Можете да създавате NX файлове на компютър, като използвате:

* Текстови редактори - като Microsoft Notepad, Notepad++ или Github Atom
* Редактори на изходния код - като Microsoft VIsual Studio Code

### Създавайте NX файлове на iOS

LowRes NX има версия за iOS, която може да се използва от разработчиците за създаване на NX файлове от това приложение.
