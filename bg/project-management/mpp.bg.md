## Какво е MPP файл?

Файл с разширение .mpp е файл с данни на Microsoft Project, който съхранява информация, свързана с управлението на проекти по интегриран начин. Microsoft разработи този патентован файлов формат, за да го направи съвместим с Microsoft Project (MSP), техния софтуерен продукт за управление на проекти. Освен MPP, MSP може да работи с XML схема на проекта. Много приложения и API позволяват MPP да бъдат конвертирани в други файлови формати. Microsoft вече разполага с онлайн Project Server, където могат да се качват файлове за управление на проекти за съвместна работа от множество потребители.

## MPP файлов формат ##

Microsoft разработи и публикува MPP като двоичен файлов формат и неговите спецификации не са публикувани от Microsoft до момента. Поради това вътрешната файлова структура и подробности не са достъпни за разработчиците. Има няколко API, достъпни онлайн, които поддържат четене/запис в MPP файлове, но 100% точност никога не се постига. Microsoft въведе няколко функции в своето приложение за управление на проекти с течение на времето, както и функциите на файловия формат MPP.

MPP файлът използва следните типове MIME:

* приложение/vnd.ms-проект
* приложение/msproj
* приложение/msproject
* приложение/x-msproject
* приложение/x-ms-проект
* приложение/x-dos_ms_project
* приложение/mpp
* zz-приложение/zz-winassoc-mpp

## Проблеми при отваряне на MPP файл? ##

Ето списъка с някои често срещани проблеми, които могат да възникнат и да причинят неправилно функциониране на MPP формата:

* Липса на поддържащ софтуер
* Повреден файл
* Заразен файл поради вирус
* Няма право на достъп в системата за отваряне на файловете
* Остаряло устройство във вашата система
* Разширението на файла е преименувано
