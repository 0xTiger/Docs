## Какво е RUN файл?

Файловият формат .run обикновено се използва за разпространение на софтуер или инсталатори на приложения в средата на Linux. За да инсталирате софтуера, ще трябва да направите файла изпълним, което може да стане с помощта на следната команда:

```bash
chmod +x file_name.run 
```

След това можете да стартирате файла, като използвате следната команда:

```bash
./file_name.run 
```

Процесът на инсталиране може да варира в зависимост от конкретния файл и програма, които се опитвате да инсталирате.

Файловият формат .run е тип shell скрипт, използван за разпространение на софтуер или инсталатори на приложения в средата на Linux. Това е самостоятелен пакет, който включва всичко необходимо за инсталиране на софтуера, включително двоични файлове, библиотеки и конфигурационни файлове.

Важно е да се отбележи, че файловете .run също могат да съдържат злонамерен код, така че винаги е добра идея да проверите източника на файла и да го сканирате за вируси, преди да го стартирате.

Освен това, някои .run файлове може да изискват root привилегии за изпълнение и инсталиране, така че може да се наложи да използвате командата "sudo", за да стартирате файла с повишени разрешения:

```bash
sudo ./filename.run
```

## Как да отворя RUN файл?

За да отворите .run файл, трябва да го направите изпълним, което може да стане с командата "chmod":

```bash
chmod +x filename.run 
```

След като файлът стане изпълним, можете да го стартирате, като напишете:

```bash
./filename.run
```

Стартирането на .run файл не е същото като отварянето му в текстов редактор. Изпълнението на .run файл ще изпълни инструкциите му, които могат да бъдат всякакви - от инсталиране на програма до изпълнение на скрипт. За да видите съдържанието на .run файл, трябва да го отворите в текстов редактор, като nano или vim:

```
nano filename.run
```
или
```
vim filename.run
```
