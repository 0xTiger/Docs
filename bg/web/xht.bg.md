## Какво е XHT файл?

Файл с разширение .xht е уеб файл, свързан с файловия тип [XHTML](/bg/web/xhtml/) (Extended Hypertext Markup Language), който сам по себе си е базиран на [XML](/bg/web/xml/) файл за маркиране. И двата файла са подобни на HTML, но имат по-строг синтаксис, подобен на XML. XHT файловете са проектирани по начин да бъдат по-структурирани, включват по-малко скриптове и са общи в допълнение към използването на съществуващите съоръжения на XML.

## XHT файлов формат

XHT файл се създава и съхранява като обикновен текстов файл с UTF-8 кодиране по подразбиране. Той съдържа пълния изходен код на XHTML документ, който може да се отваря и редактира с всеки текстов редактор, който поддържа Unicode кодиране. В допълнение към текстовите редактори, файловият формат XHT е разбираем от повечето съвременни уеб браузъри и може да се отваря с тях.

## Пример за XHT

Следващият пример на XHT документ дефинира XML декларациите.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
