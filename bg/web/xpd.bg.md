## Какво е XPD файл?

XPD файл е файл на уеб страница, създаден от XML документ. По-често това се генерира чрез XSLT трансформация. [Спецификациите на файловия формат XPD](https://www.w3.org/TR/xml-pipeline/) са достъпни на W3, които определят начин за увеличаване на оперативната съвместимост на многопроцесния [XML](/bg/web/xml/) приложения. XML Pipeline Language дефинира връзките за обработка между XML ресурсите.
