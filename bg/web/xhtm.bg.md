## Какво е XHTM файл?

Файл с разширение .xhtm е файл на уеб страница, който е написан на XHTML. Той използва XML 1.0 като стандартна реализация и се счита за преформулиране на HTML 4. Подобно на XHTML, XHTM файлът съдържа стандартни HTML тагове, които следват по-строги указания за XML. XHTM файловете се използват за публикуване на съдържание на уебсайтове и обикновено използват разширение [XHTML](/bg/web/xhtml/).

Можете да използвате уеб браузъри като Google Chrome, Apple Safari и Mozilla Firefox, за да отваряте XHTM файлове.

## Пример за XHTM

Следващият пример на XHTM документ дефинира XML декларациите.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
