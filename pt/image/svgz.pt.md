## O que é um arquivo SVGZ?

Um arquivo com extensão .svgz é uma versão compactada do arquivo Scalable Vector Graphics (.SVG). Ele é compactado com compactação gzip e contém dados no formato [XML](/pt/web/xml/). Os arquivos SVGZ suportam transparência, gradientes, animações e filtros. Os arquivos SVGZ são menores em tamanho em comparação com os arquivos SVG padrão e esse tamanho de arquivo reduzido ajuda a transferir os arquivos gráficos online. O designer gráfico cria arquivos SVGZ usando ferramentas como Adobe Illustrator, Corel PaintShop Pro e outras. No entanto, os arquivos SVGZ podem ser gerados ativando a [compactação GZip](/pt/compression/gz/) no servidor Apache ao enviar os dados da imagem.

## Formato de arquivo SVGZ

O SVGZ, sendo compactado [SVG](/pt/page-description-language/svg/), é baseado no formato gzip compactado aberto que reduz o tamanho dos documentos SVG. Este arquivo compactado contém documentos SVG de texto simples que são puramente baseados no formato de arquivo XML (/pt/web/xml/). Os arquivos SVGZ podem ser abertos por visualizadores/aplicativos que estejam em conformidade com as especificações SVG 1.1. Arquivos SVGZ 20-50% menores em tamanho do que os arquivos SVG originais.
