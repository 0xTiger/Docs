## O que é um arquivo .AGIF?

Um arquivo AGIF é um arquivo GIF criado com Active GIF Creator. É um arquivo de projeto que pode ser aberto e visualizado em projetos do Active GIF Creator. Quando aberto no Active GIF Creator, mostra sequência de imagens que finalmente compõem o arquivo GIF finalizado com extensão .gif.

## Formato de arquivo AGIF

AGIF é um arquivo de projeto salvo no formato de arquivo de imagem raster. O arquivo [GIF](/image/gif/) finalizado pode ser aberto e exibido como animação. Existem APIs Python disponíveis online que podem ser usadas para criar arquivos GIF animados.
