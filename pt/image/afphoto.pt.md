## O que é um arquivo .AFPHOTO?

Um arquivo AFPHOTO é um arquivo de imagem criado pelo software de criação de imagens, Serif Affinity Photo. Ele contém o conteúdo geral da imagem na forma de múltiplas camadas de imagem, formas, texto, pinceladas, amostras, efeitos de coloração e iluminação. Serif Affinity Photo é semelhante ao Adobe Photoshop, mas é uma alternativa ao Adobe Photoshop com quase os mesmos recursos e capacidades. Adobe Photoshop salva os arquivos como [.psd file](/image/psd/) enquanto [Affinity Photo](https://affinity.serif.com/en-gb/photo/) os arquivos são salvos com extensão .afphoto.

Serif Affinity Photo pode abrir arquivos AFPHOTO.

### Formato de arquivo AFPHOTO

O arquivo AFPHOTO é salvo no formato de arquivo binário proprietário do Serif Affinity Photo. Sua estrutura interna de formato de arquivo e especificações detalhadas não estão disponíveis para referência pública.

## Como converter um arquivo AFPHOTO?

Os arquivos AFPHOTO podem ser convertidos para vários formatos diferentes usando o software Serif Affinity Photo. Os seguintes formatos de arquivo de exportação são suportados.

 * [PSD](/image/psd/) - Documento do Adobe Photoshop
 * [PDF](/pdf/) - Formato de documento portátil
 * [JPG](/image/jpeg/) - Imagem JPEG
 * [PNG](/image/png/) - Gráfico de rede portátil
 * [GIF](/image/gif/) - Formato de intercâmbio gráfico
 * [EPS](/page-description-language/eps/) - Encapsulated PostScript
 * [SVG](/page-description-language/svg/) - Scalable Vector Graphics
 * [TIFF](/image/tiff/) - Formato de arquivo de imagem marcado
 * TGA - Gráfico Targa
 * EXR - Imagem OpenEXR
 * [HDR](/image/hdr/) - Imagem de alta faixa dinâmica
