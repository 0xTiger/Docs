## O que é um arquivo ECW? ##
Um arquivo ECW consiste em uma imagem projetada para uso com dados geoespaciais. A imagem que a contém é, na verdade, uma imagem compactada criada no formato Wavelet de compactação aprimorada. Os arquivos ECW são geralmente usados para armazenar projeção de mapas de satélite e imagens aéreas. Para oferecer suporte a aplicativos geoespaciais, as informações de projeção do mapa podem ser incorporadas aos arquivos ECW.

## Formato de arquivo ECW
ECW é um formato de imagem proprietário baseado em compressão wavelet; otimizado para imagens de satélite e fotografia aérea. Este formato é agora propriedade da Intergraph, parte da Hexagon AB, mas basicamente foi desenvolvido pela Earth Resource Mapping. ECW é um formato de compactação com perdas que compacta imagens de tamanho grande com contraste alternado fino, mantendo sua qualidade visual.
 

### Propriedades
Aqui estão algumas propriedades típicas do formato ECW:
- As informações de projeção do mapa podem ser adicionadas ao formato de arquivo ECW para suportar aplicativos geoespaciais.
- Dados de imagem de até 65.535 bandas, incluindo camadas ou cores, podem ser compactados no formato de arquivo ECW a uma taxa de mais de 25 MB por segundo.
- O protocolo ECWP é um protocolo de streaming eficiente usado para transmitir imagens ECW e JPEG2000 em redes.
- Um SDK rápido compatível com ECW e JPEG2000 está disponível gratuitamente para implementação de desktop para Linux, Windows e MacOSX
