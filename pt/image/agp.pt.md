## O que é um arquivo .AGP?

Um arquivo AGP é um arquivo de imagem de 32 bits com múltiplas camadas. Ele é criado com o software ArtGem que oferece recursos de trabalho com imagens e é semelhante ao Adobe Photoshop em recursos básicos. AGP é um arquivo de projeto que contém todas as configurações para a criação da imagem final.

Os arquivos AGP podem ser abertos com RL Vision ArtGem. No entanto, o ArtGem foi descontinuado há muito tempo após ser vendido.

## Formato de arquivo AGP

Os arquivos AGP foram salvos no formato de arquivo proprietário ArtGem. A estrutura interna de arquivos AGP não está disponível publicamente para referência do desenvolvedor.
