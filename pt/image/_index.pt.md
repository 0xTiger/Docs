Um formato de arquivo de imagem é um método padrão para organizar e armazenar imagens em dispositivos como computadores, tablets e smartphones. As imagens digitais armazenam dados de imagem em uma grade de pixels bidimensional onde cada pixel é uma representação de cor em termos de número de bits. Os tipos de arquivo de imagem são classificados em formatos de imagem vetorial e formatos de imagem raster. Imagens 3D são outro tipo de formato de arquivo de imagem vetorial usado para gerenciar imagens 3D.

## Formatos Raster ##

**Raster Graphics** são imagens digitais compostas por dados de pixels para representação de cores. Esses são os tipos de imagem mais comuns para gráficos usados para a Web e também para fotos digitais. Algumas das imagens rasterizadas podem ser compactadas para reduzir o tamanho do arquivo de imagem. As extensões comuns de arquivo de imagem raster e seus formatos de arquivo incluem **[BMP](/pt/image/bmp/)** (arquivo de imagem bitmap), **[PNG](/pt/image/png/)** (gráficos de rede portáteis) e **[GIF](/pt/image/gif/)** (Arquivo de intercâmbio de gráficos).

## Formatos vetoriais ##

**Imagens vetoriais** são definidas por pontos 2D, em vez de pixels, que são conectados para dar uma forma geométrica à imagem. Os pontos têm propriedades que definem a direção dos caminhos, cor, forma, curva, espessura e preenchimento. As extensões comuns de arquivo de imagem vetorial e seus formatos de arquivo incluem **[SVG](/pt/page-description-language/svg/)** (Scalable Vector Graphics), **[EPS](/pt/page-description-language/eps/) ** (linguagem PostScript encapsulada) e **[PDF](/pt/pdf/)** (formato de documento portátil).
