## O que é um arquivo .INK?

O formato de arquivo ".ink" está associado à **Pantone**, uma empresa conhecida na área de correspondência de cores e impressão. Um **Arquivo de Referência Pantone** no formato ".ink" normalmente contém informações sobre as cores da tinta Pantone; esses arquivos são usados na indústria de impressão para garantir reprodução e consistência precisas de cores.

A Pantone é conhecida por seu **Pantone Matching System (PMS)**, que é um sistema padronizado de reprodução de cores amplamente utilizado em vários setores, incluindo design gráfico, impressão e fabricação; o PMS fornece um conjunto de cores de tinta, cada uma identificada por um código específico, como "PMS 185" para um tom específico de vermelho.

## O que o arquivo INK contém?

O formato de arquivo ".ink" pode armazenar dados relacionados às cores da tinta Pantone, como:

1. **Códigos de cores:** As cores das tintas Pantone são identificadas por códigos exclusivos e esses códigos normalmente são incluídos no arquivo.
    



2. **Valores de cores:** O arquivo pode conter valores de cores como **RGB (vermelho, verde, azul)**, **CMYK (ciano, magenta, amarelo, chave/preto)** e **Lab (CIELAB)** valores para especificar como a cor deve ser reproduzida em diferentes espaços de cores.
    



3. **Amostras de cores:** As cores da tinta Pantone geralmente têm amostras correspondentes que designers e impressores podem usar como referência.
    



4. **Informações sobre cores:** Informações adicionais sobre a cor da tinta, como nome, descrição e notas de aplicação, podem ser incluídas no arquivo ".ink".

## Como abrir o arquivo INK?

Os programas que abrem arquivos INK incluem

- **CorelDRAW** (avaliação gratuita) para Windows

CorelDRAW é um software gráfico vetorial que oferece suporte a cores Pantone e fornece ferramentas para trabalhar com elas.
