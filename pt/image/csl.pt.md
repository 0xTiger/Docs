## O que é um arquivo CSL?

Um arquivo Corel Symbol Library (CSL) é uma biblioteca de símbolos usados no software CorelDRAW. Ele armazena símbolos que podem ser usados em outros arquivos. Os arquivos CSL podem ser armazenados localmente ou em uma rede para torná-los acessíveis para fácil implantação e gerenciamento. Eles são salvos com o arquivo CorelDRAW junto com todas as instâncias de símbolo usadas no documento.

## Formato de arquivo CSL

Os arquivos CSL são salvos no formato de arquivo proprietário do CorelDRAW. Sendo uma coleção de símbolos, os símbolos dentro de um arquivo CSL podem ser importados e adicionados aos desenhos em um projeto.
