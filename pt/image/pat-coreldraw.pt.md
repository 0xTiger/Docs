## O que é um arquivo .PAT?

Um arquivo ".PAT" normalmente é associado ao **CorelDRAW**, um popular software de edição de gráficos vetoriais. No CorelDRAW, o arquivo ".PAT" é um "Arquivo de padrão" que contém informações sobre padrões ou texturas que podem ser aplicados a objetos em seu design. Os padrões podem ser usados para preencher objetos com vários designs repetidos, como listras, pontos ou texturas personalizadas. Esses padrões são armazenados em arquivos ".PAT" para fácil reutilização.

## Como criar e usar arquivos PAT?

Veja como você pode criar e usar arquivos de padrão no CorelDRAW:

1. Crie um padrão:
    



- Abra o CorelDRAW.
- Padrão de design que você deseja salvar como arquivo ".pat".
- Selecione objeto ou área com padrão.
- Vá ao menu "Objeto" e escolha "Preencher" para abrir a ferramenta Preenchimento.
- Na ferramenta Preenchimento, selecione "Preenchimento de padrão".
- Personalize as propriedades do padrão, como tamanho, ângulo e espaçamento.
2. Salve o padrão como arquivo ".pat":
    



- Depois de criar seu padrão, você pode salvá-lo como arquivo “.pat” para uso futuro.
- Vá ao menu “Arquivo” e escolha “Exportar” ou “Salvar como”.
- Escolha o formato de arquivo "Arquivo de padrão CorelDRAW (*.pat)."
- Especifique o nome do arquivo e escolha o local para salvá-lo.
3. Aplique o padrão:
    



- Para usar o padrão salvo em um novo projeto ou em um objeto diferente, você pode importá-lo.
- Abra novo projeto no CorelDRAW.
- Vá ao menu "Objeto", escolha "Preencher" e selecione "Preenchimento de padrão".
- Clique no botão "Importar" e navegue até o arquivo ".pat" que você salvou.
- Selecione o padrão dos padrões importados e aplique-o ao seu objeto.

## Arquivos PAT para imagens vetoriais e layouts de página

Os arquivos .PAT são projetados especificamente para imagens vetoriais e documentos de layout de página. Eles armazenam padrões que podem ser repetidos perfeitamente em imagens baseadas em vetores ou layouts de páginas e são frequentemente empregados para aplicar fundos texturizados a designs.

Corel CONNECT é uma ferramenta útil de navegação de conteúdo gráfico que vem com o CorelDRAW Graphics Suite. Ele oferece pacotes de vários padrões vetoriais que podem ser perfeitamente integrados em suas imagens ou documentos. Esses padrões vetoriais estão prontos para uso, tornando conveniente aprimorar seus designs com padrões pré-fabricados, tudo dentro do ambiente do CorelDRAW Graphics Suite.

## Suíte de gráficos CorelDRAW

CorelDRAW Graphics Suite é um pacote de software abrangente projetado para design gráfico e ilustração. Inclui uma variedade de ferramentas poderosas para criação de gráficos vetoriais, design de layout, edição de fotos e tipografia. O CorelDRAW é a peça central do pacote conhecido por seus recursos de ilustração vetorial, enquanto outros aplicativos como o Corel PHOTO-PAINT fornecem recursos de edição de imagens. A suíte é amplamente utilizada por profissionais e profissionais criativos para tarefas como design de logotipo, layout de página, edição de fotos e muito mais. Ele oferece um conjunto versátil de recursos para usuários iniciantes e avançados, tornando-o uma escolha popular para uma ampla variedade de projetos de design gráfico.
