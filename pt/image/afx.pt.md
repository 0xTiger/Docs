## O que é um arquivo .AFX?

Um arquivo AFX é um arquivo de imagem usado pelo software Auto FX PhotoGraphic Edges que é usado para aprimoramento de imagens digitais. Ele contém a imagem raster de sobreposição que é usada sobre outra imagem para dar o efeito Fotográfico. Este processo aprimora a imagem de saída criando efeitos digitais nas imagens do cliente. Também dá uma aparência digital aprimorada às fotografias que são muito melhores para apresentar.

## Formato de arquivo AFX

Os arquivos AFX são salvos como arquivos binários que podem ser abertos usando o software Auto FX PhotoGraphic Edges.
