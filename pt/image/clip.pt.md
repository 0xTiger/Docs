## O que é um arquivo CLIP?

Um arquivo CLIP é um arquivo de imagem raster gerado pelo aplicativo de desenho 2D Clip Studio Paint. É semelhante ao arquivo [PSD](/pt/image/psd/) da Adobe e contém várias camadas para gerenciar diferentes elementos gráficos na forma de camadas, como texto, imagens, caneta, lápis, pincéis e formas. O formato de arquivo CLIP é usado para projetar vários tipos de gráficos diferentes, como ilustrações e animações. O aplicativo Clip Studio Paint foi desenvolvido pela empresa japonesa CELSYS, Inc., fundada em maio de 1991.

## Formato de arquivo CLIP - Mais informações

O formato de arquivo CLIP foi introduzido pelo CELSYS para substituir o formato de arquivo LIP original. Ele armazena elementos gráficos na forma de camadas que podem ser editadas para alterações. Também é o formato de arquivo padrão para o aplicativo [CLIP Studio Paint](https://www.celsys.co.jp/en/service/studio_paint/) que está disponível para plataformas Windows, MacOS, Android e iOS.

## Tipos de desenho suportados

O formato de arquivo CLIP pode armazenar animações e ilustrações que podem ser usadas em diversos tipos de trabalho.
