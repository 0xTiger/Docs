## O que é um arquivo PICT?

Um arquivo PICT é uma imagem gráfica salva no formato de arquivo de imagem Apple Macintosh. Ele pode armazenar dados de imagem como formatos de arquivo raster e vetorial. Era o formato de arquivo de imagem padrão para o programa Apple QuickDraw. Os arquivos PICT permitem a troca de dados raster e vetoriais entre aplicativos Mac. O formato foi substituído há muito tempo pelo formato de arquivo [PDF](/pt/pdf/) mais flexível. Os arquivos PICT às vezes são encontrados com a extensão **[.PCT](/pt/image/pct/)**.

## Formato de arquivo PICT - Mais informações

Os arquivos PICT são salvos como arquivos de imagem binária que podem ser abertos com aplicativos como XnView, GIMP, Corel PaintShop Pro e Apple Preview. Ele suporta imagem de canal único (com capacidade de armazenamento de 8 cores) ou imagens RGB de canal de cores que podem armazenar milhares de cores. Eles são conhecidos como formatos de arquivo **PICT 1** e **PICT 2**, respectivamente.
