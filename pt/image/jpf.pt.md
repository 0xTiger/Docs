## O que é um arquivo JPF? ##

Um arquivo com extensão .jpf é uma extensão do sistema de codificação de imagem JPEG 2000 ISO/IEC 15444 e é referido como sua Parte 2 ISO/IEC 15444-2. Ele define e especifica um conjunto de métodos de compactação sem perdas (preservação de bits) e com perdas para codificar imagens estáticas digitais em cores de tons contínuos, de dois níveis, em escala de cinza, ou de vários componentes. A primeira parte da ISO/IEC 15444-1 é referida ao [JP2](/pt/image/jp2/) que usa a tecnologia wavelet para codificar conteúdo sem perdas e é a base para os formatos de arquivo de imagem JPEG 2000. O formato de arquivo JPF não recebeu uma recepção calorosa devido ao uso extensivo do formato JPEG. Os arquivos JPF podem ser abertos com aplicativos de imagem populares, como Adobe Photoshop 2020, Adobe Illustrator 2020 e CorelDraw Graphics Suite 2020.

## Breve história

Em 2000, o comitê do Joint Photographic Experts Group projetou o JP2 com o objetivo de melhorar seu próprio padrão JPEG baseado em transformada de cosseno discreto com este novo método baseado em wavelet. O comitê JPEG teve como objetivo fornecer seus métodos básicos livres de taxas de licença. Na licença JP2 ganhando concorrência entre 20 empresas, elas venceram por um triz. O JPEG 2000 foi declarado como um padrão ISO, embora a maioria dos navegadores da Web não esteja pronta para dar uma mão ao JPEG 2000 desde 2017. Em 2004, o formato ISO/IEC 15444-2 foi aceito publicamente como extensão do formato de arquivo JP2.

## Formato de arquivo JPF

O formato de arquivo JPF define processos de decodificação estendidos para conversão de dados de imagem compactados para reconstrução. É um formato de arquivo estendido que especifica a sintaxe de fluxo de código estendido contendo informações para interpretar os dados de imagem compactados. Esse padrão estendido especifica um contêiner para armazenar metadados de imagem e fornece orientação sobre processos de codificação estendidos para converter dados de imagem de origem em dados de imagem compactados.

### Organização de arquivos

JPF é o formato de arquivo de armazenamento formal quando os arquivos JPX são armazenados em sistemas de arquivos de computador. Além disso, outras Recomendações/Padrões Internacionais podem definir outras caixas para uso em arquivos JPX. No entanto, todas as informações contidas em um arquivo JPX devem estar no formato de caixa; byte-streams que não estejam no formato de caixa não serão encontrados no arquivo. A estrutura binária de uma caixa em um arquivo JPX é idêntica àquela definida no formato de arquivo [JP2](/pt/image/jp2/).
