## O que é um arquivo CMX?

Os arquivos com extensão .cmx são do formato de arquivo de imagem do Corel Exchange (também conhecido como Corel Metafile Exchange) usado como apresentação pelos aplicativos do CorelSuite. Ele contém dados de imagem como gráficos vetoriais, bem como metadados que descrevem a imagem. Os arquivos CMX podem ser abertos pelo CorelDraw, Corel Presentations, Paint Shop Pro e algumas versões do Adobe Illustrator. Os arquivos CMX podem ser convertidos para outros formatos, como [JPG](/pt/image/jpeg/) e [EPS](/pt/page-description-language/eps/).

## Formato de arquivo CMX
O formato de arquivo CMX é baseado no Resource Interchange File Format (RIFF) e suporta os formatos binários Intel e Motorola. O arquivo CMX usa os seguintes tipos MIME:
* imagem/x-cmx
* aplicação/x-cmx
* aplicação/cmx
* desenho/cmx

### Cabeçalho do formato de arquivo
Os arquivos CMX começam com "RIFF" ou "RIFX" e têm "CMX1" no deslocamento 8.
