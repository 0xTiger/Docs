## O que é um arquivo 3GP?

3GP (3GPP File Format) é um formato de contêiner multimídia usado em telefones celulares 3G. Também pode ser jogado em alguns telefones 2G e 4G. O 3GP é definido pelo Projeto de Parceria de Terceira Geração (3GPP) para serviços multimídia 3G UMTS (Sistema Universal de Telecomunicações Móveis). Os arquivos 3GP são armazenados com a extensão .3gp. O formato de arquivo 3GP foi projetado para telefones baseados em GSM (Global System for Mobile Communications) e é o formato padrão para arquivos de mídia enviados via MMS (Multimedia Messaging Service). Ao contrário do [MP4](/pt/video/mp4/), que usa o codec H.263, os arquivos 3GP usam preferencialmente o codec H.263.

## Detalhes técnicos do formato de arquivo 3GP

O formato de vídeo 3GP é baseado no formato de arquivo de mídia de base ISO definido na ISO/IEC 14496-12 – MPEG-4 Parte 12. Ele foi desenvolvido para telefones celulares para reduzir armazenamento, largura de banda e uso de dados. O formato de arquivo 3GP suporta os seguintes codecs:

- Vídeo
  - MPEG-4 Part 2
  - H.263
  - MPEG-4 Part 10 (AVC/H.264)
- Áudio
  - AMR-WB
  - AMR-WB+
  - AMR-NB
  - AAC-LC
  - HE-AAC v1
  - HE-AAC v2 (Enhanced aacPlus)

O formato de arquivo 3GP permitiu o uso de codecs H.263 e AMR e definiu novas caixas às quais os codecs se referem. Esses pontos de código foram registrados no formato de arquivo de mídia de base ISO para o armazenamento de informações específicas de mídia MPEG-4.

## Dispositivos de suporte para arquivos de vídeo 3GP

- Telefones celulares com recursos 3G
- Áudio do PlayStation 3 importado de um CD
- O Nintendo 3DS reproduziu vídeos do YouTube usando a tecnologia 3GP
- Os iDevices da Apple suportavam a reprodução de arquivos apenas como arquivos de passagem
