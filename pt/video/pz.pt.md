## O que é um arquivo PZ?

Um arquivo PZ é um arquivo de projeto criado com o aplicativo de criação de videoclipes online **Panzoid**. Ele contém informações sobre o projeto de vídeo e seu conteúdo, como cena, objetos, efeitos, iluminação, FX, áudio e música. O Panzoid permite que você crie vídeos online usando o recurso de edição de vídeo **ClipMaker**.

Arquivos PZ podem ser abertos com Panzoid ClipMaker e Panzoid Backgrounder.

## Formato de arquivo PZ

Os arquivos PZ são salvos como arquivos binários no disco. Quando um usuário cria um novo projeto com [Panzoid](https://panzoid.com/), é criado um arquivo PZ que mantém os detalhes dos objetos definidos pelo usuário salvos. As mesmas informações são sincronizadas com outros usuários se o projeto for de outros contribuidores. Os arquivos PZ podem ser baixados para o computador.

Quando você deseja abrir o arquivo PZ, você pode carregá-lo novamente no **Panzoid** usando a opção Carregar na interface online. O arquivo é carregado online e seu conteúdo é carregado para modificações.

### Clipes de vídeo suportados por arquivo PZ

O Panzoid possui os programas ClipMaker e Backgrounder que são usados juntos para criação de vídeos. Vários tipos diferentes de vídeos podem ser criados com o Panzoid, incluindo planos de fundo personalizados, arte do canal do YouTube, vídeos de mídia social e miniaturas.
