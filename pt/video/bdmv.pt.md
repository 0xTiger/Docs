## O que é um arquivo .BDMV?

Um arquivo BDMV é um tipo de formato de arquivo usado em discos de vídeo Blu-ray Disc (BD). BDMV significa "Blu-ray Disc Movie" e é um formato de contêiner usado para armazenar vídeo, áudio e outros conteúdos relacionados em alta definição em discos Blu-ray. Os arquivos BDMV fazem parte da estrutura de diretórios do disco Blu-ray e contêm vários componentes de um filme Blu-ray, incluindo fluxos de vídeo, faixas de áudio, legendas e estruturas de menu.

## Componentes principais de um diretório BDMV:

1. **Pasta STREAM**: Esta pasta normalmente contém os fluxos de vídeo e áudio do filme. O vídeo geralmente é armazenado na forma de arquivos M2TS (MPEG-2 Transport Stream), enquanto o áudio pode estar em formatos como Dolby TrueHD ou DTS-HD Master Audio.

2. **Pasta CLIPINF**: Esta pasta contém informações sobre os videoclipes individuais (arquivos M2TS) no disco.

3. **Pasta PLAYLIST**: contém arquivos de lista de reprodução que especificam a ordem em que os videoclipes devem ser reproduzidos. Os discos Blu-ray geralmente têm várias listas de reprodução para diferentes opções de visualização ou versões do filme.

4. **Pasta AUXDATA**: Esta pasta armazena dados auxiliares e metadados associados ao filme.

5. **Pasta BDJO**: Contém estruturas de menu interativas baseadas em Java para discos Blu-ray. Esses menus permitem recursos como seleção de cena, recursos especiais e muito mais.

6. **Pasta JAR**: Arquivos Java usados para menus interativos e recursos especiais.

7. **Arquivos INDEX.BDM e MOVIEOBJ.BDM**: Esses arquivos contêm informações sobre a estrutura e navegação do disco.

## Como abrir o arquivo BDMV?

Os programas que abrem o arquivo .BDMV são

- **VLC Media Player**: o VLC é um reprodutor de mídia versátil que pode lidar com uma ampla variedade de formatos multimídia, incluindo arquivos BDMV. Basta abrir o VLC, ir em “Mídia” no menu, selecionar “Abrir pasta” e navegar até o diretório BDMV. O VLC deve detectar e reproduzir automaticamente o conteúdo.

- **PowerDVD**: PowerDVD é um software reprodutor de Blu-ray comercial projetado para reprodução de discos Blu-ray e arquivos BDMV de alta qualidade. Oferece uma experiência premium com suporte para vários recursos de Blu-ray, incluindo menus e recursos especiais.

- **CyberLink PowerDirector**: Se você deseja editar arquivos BDMV, o CyberLink PowerDirector é um software de edição de vídeo que pode importar e trabalhar com arquivos BDMV. É útil para edição de vídeo e criação de discos Blu-ray.

- **Leawo Blu-ray Player**: Leawo Blu-ray Player é um software reprodutor de Blu-ray gratuito que suporta arquivos BDMV. Ele fornece uma interface simples para reproduzir conteúdo Blu-ray em seu computador.

- **AnyDVD HD**: AnyDVD HD não é um reprodutor, mas sim uma ferramenta que remove a proteção contra cópia de discos Blu-ray, incluindo arquivos BDMV. Você pode então usar outro software de reprodução para abrir e reproduzir os arquivos BDMV descriptografados.

- **ArcSoft TotalMedia Theatre**: Este é outro software comercial de reprodutor de Blu-ray que pode lidar com arquivos BDMV e oferece uma experiência de reprodução premium.
