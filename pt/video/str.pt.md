## O que é um arquivo .STR?

Um arquivo STR é um arquivo de gravação criado pelo YouTube para uma transmissão ao vivo hospedada pelo evento de streaming do usuário. O arquivo é salvo automaticamente pelo Youtube em seu backend assim que a transmissão ao vivo termina. Um vídeo ao vivo pode ser transmitido para vários canais para que o público possa interagir com o editor por meio de comentários. Esses arquivos SRT podem ser acessados usando a API de transmissão ao vivo do YouTube, que fornece acesso aos servidores do YouTube.

## Formato de arquivo STR

O YouTube salva arquivos STR no servidor em formato de arquivo binário. Os detalhes da estrutura interna dos arquivos STR não estão disponíveis publicamente.
