## O que é um arquivo EXO?

Um arquivo EXO é um arquivo de fragmento de vídeo, baixado pela versão Android do YouTube no formato EXO. Quando um usuário deseja baixar um vídeo para reprodução offline, o vídeo é dividido em pequenos pedaços e cada pedaço é baixado e salvo no dispositivo do usuário como um arquivo EXO. Esses vários arquivos EXO são reunidos para reprodução off-line pelo aplicativo do YouTube.

## Formato de arquivo EXO

Os arquivos EXO são salvos no formato de arquivo proprietário do YouTube. Eles são salvos como arquivos criptografados compactados e só podem ser abertos com o aplicativo YouTube para Android. Todos os vários pedaços baixados são montados e descriptografados pelo aplicativo para serem reproduzidos offline. Sendo criptografados, os arquivos EXO não podem ser convertidos em nenhum outro formato de arquivo, como [MP4](/pt/video/mp4/).
