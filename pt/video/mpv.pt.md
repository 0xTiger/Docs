## O que é um arquivo MPV?

Um arquivo MPV é um arquivo de vídeo de fluxo elementar MPEG que consiste em dados de vídeo brutos e não compactados. O conteúdo de vídeo dentro de um arquivo MPV inclui especificações de compactação MPEG-2 integradas. Os arquivos MPV não contêm dados de áudio. O formato de arquivo MPV também é usado como entrada para aplicativos de processamento de vídeo, pois o fluxo elementar [MPEG](/pt/video/mpg/) usa principalmente os padrões de codificação MPEG-2. Os arquivos MPV podem ser abertos e reproduzidos com aplicativos de software que suportam os formatos de arquivo de vídeo MPEG-2. O software de player de mídia VideoLAN LVC é um desses softwares que pode abrir e reproduzir arquivos MPV.

## Formato de arquivo MPV

Os arquivos MPV são salvos em disco como arquivos binários simples, seguindo a estrutura do formato de arquivo MPEG. No entanto, seu formato de arquivo interno não está disponível publicamente para referência do desenvolvedor.
