## O que é um arquivo BWP?

Um arquivo BWP é um arquivo de script de página da Web criado na linguagem de programação, **BuRg3r**. É específico para BuRg3r e pode conter código HTML com seções de fonte BuRg3r delimitadas por<?b and ?> Tag. BuRg3r é uma linguagem de programação de uso geral que também é um mecanismo de jogo desenvolvido pela Fluffy Smoke Entertainment.
