## O que é um arquivo HYPE?

Um arquivo HYPE é um arquivo de projeto salvo usando o software Hype que é usado para a criação de conteúdo da web com HTML5. Ele contém elementos da web, como gráficos, texto, animação, informações de cena e outros elementos que compõem coletivamente o conteúdo. O conteúdo da Web em um arquivo HYPE pode ser exportado para arquivos da Web que podem ser publicados em uma página da Web.

Um arquivo HYPE pode ser aberto com o software [Tumult Hype](https://tumult.com/hype/).

## Formato de arquivo HYPE

Os arquivos HYPE são armazenados em formato de arquivo proprietário e os detalhes internos destes não são conhecidos publicamente. O conteúdo dentro de um arquivo HYPE só pode ser visualizado quando aberto no software Tumult Hype.

### Conteúdo de um arquivo HYPE

Um arquivo HYPE é criado quando você está usando o aplicativo Hype para criar conteúdo da Web para diferentes propósitos. Ele armazena informações sobre animações baseadas em quadros-chave para a criação de conteúdos como eCards, eBooks, apresentações, anúncios, infográficos e material de página da web. O conteúdo destes é alimentado por HTML5 que permite que o conteúdo seja exibido em dispositivos de desktops a iPhones e iPads. Tumult Hype gera HTML5 de última geração, estilos [CSS3](/pt/web/css/) e [JavaScript](/pt/web/js/).
