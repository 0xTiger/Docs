## O que é um arquivo GSP?

Um arquivo GSP é um arquivo de mensagem de erro que indica um problema com a entrega do e-mail. Ele é enviado pelo IMail Server, um programa de servidor de e-mail que é usado para comunicação por e-mail por várias pequenas e médias empresas. O erro pode ser devido a vários motivos, como grande quantidade de dados, perda de dados no arquivo [MSG](/pt/email/msg/) ou erros de memória insuficiente.

## Formato de arquivo GSP - Mais informações

Os arquivos GSP são armazenados em formato de arquivo da Web e podem ser facilmente visualizados com clientes de e-mail, como o Microsoft Outlook. Um arquivo GSP é gerado como resposta de um e-mail enviado que resulta em erro.
