## O que é um arquivo JHTML?

Um arquivo com extensão .jhtml é um arquivo [HTML](/pt/web/html/) com código [Java](/pt/programming/java/) que é executado no servidor quando um cliente solicita esta página em um navegador. O servidor processa as solicitações, executa as funções Java contidas na função e retorna a página ao navegador do cliente. Os objetos Java embutidos nas páginas JHTML são executados no servidor para manipular solicitações de páginas desse tipo. Os arquivos JHTML também podem acessar informações do banco de dados usando a conexão JDBC (Java [Database](/pt/database/) Connectivity). Os arquivos JHTML podem ser abertos em qualquer editor de texto e visualizados em navegadores da Web, como Google Chrome, Firefox e Safari.

## Formato de arquivo JHTML

JHTML é uma tecnologia proprietária do ATG e pode ser criada usando o Editor de Documentos Dynamo do ATG (Art Technology Group). Os arquivos JHTML são escritos em formato de arquivo de texto simples usando o código HTML e Java padrão. Eles contêm tags HTML padrão, além de tags proprietárias que fazem referência a objetos Java. Quando tal página é solicitada pelo usuário, o servidor HTTP encaminha a solicitação para um servidor de aplicativos Java. A página JHTML é convertida primeiro em arquivo .java e compilada para gerar um arquivo [.class](/pt/programming/class/) que é executado como um servlet. Isso gera um fluxo de dados HTTP e HTML padrão que são retornados ao navegador solicitante para exibição ao usuário. Isso é útil para executar consultas relacionadas ao banco de dados no servidor e retornar o resultado acumulado finalizado ao navegador do cliente.
