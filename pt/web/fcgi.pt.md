## O que é um arquivo FCGI?

Um arquivo FCGI é um arquivo de código-fonte que contém código-fonte escrito para FastCGI (FCGI). Ele pode ser escrito em várias linguagens de programação diferentes e contém a extensão do padrão CGI usado para tratamento mais robusto de solicitações do servidor Web. As linguagens de suporte mais comuns para FCGI incluem **Perl** que fornece suporte para código Perl e a forte interface externa da API FastCGI.

## Formato de arquivo FCGI

O que torna um arquivo FCGI importante? A interface FCGI suporta computação distribuída para lidar com solicitações da Web e distribuí-las em vários servidores. Isso faz com que os arquivos FCGI sejam usados principalmente para balanceamento de carga em vários servidores.
