## O que é um arquivo DOTHTML?

Um arquivo DOTHTML é um documento de modelo do Microsoft Word salvo na versão HTML. Ele pode ser aberto no navegador da web e o conteúdo é exibido como qualquer outra página da web. Os arquivos DOTHTML são gerados a partir de arquivos .DOT do Microsoft Word que contêm configurações pré-formatadas para gerar esses arquivos. Quando um arquivo [DOT](/pt/word-processing/dot/) é convertido em DOCHTML, todas essas configurações são mantidas e o arquivo DOTHTML convertido é exibido com formatação semelhante ao arquivo DOT original. Dessa forma, os modelos publicados podem ser visualizados no navegador da web.

## Formato de arquivo DOTHTML - Mais informações

Os arquivos DOTHTML são salvos como arquivos [HTML](/pt/web/html/) e contêm código-fonte HTML que pode ser aberto com qualquer editor de HTML. Nos primeiros dias, o Microsoft Word oferecia as versões HTML de todos os seus formatos de documentos, como [DOCHTML](/pt/web/dochtml/), [DOCMHTML](/pt/web/docmhtml/), mas posteriormente essas versões baseadas em HTML foram descontinuadas . Resultantly, DOTHMLT também não é mais suportado.
