## O que é um arquivo ATOM?

Um arquivo ATOM é um formato de distribuição para a estrutura do feed Atom e documentos de entrada. Semelhante aos arquivos .RSS, o formato de arquivo ATOM é um formato de distribuição baseado em XML que é um padrão para publicação de conteúdo. Um arquivo ATOM é usado para feeds da Web que permitem que programas de software verifiquem atualizações publicadas no site. Ele contém entradas que podem ser de diferentes tipos, como títulos, artigos de texto completo, trechos ou links para conteúdo em um site

Os aplicativos que podem **abrir arquivos ATOM** incluem o **Apple Safari**.

## Formato de arquivo ATOM - Mais informações

Os arquivos ATOM são armazenados e publicados no popular formato de arquivo XML, que é um formato universal para troca de informações. Foi desenvolvido como uma alternativa ao RSS tendo em vista as limitações e falhas do formato de arquivo RSS.

### Tipos de Documentos Atom

1. `Documentos de Entrada Atom` - É um documento XML que consiste em um único item de informação, conhecido como entrada, para o feed Atom. Tem um<atom:entry> elemento que contém vários elementos filho. O conteúdo de uma entrada Atom pode ser texto simples, HTML, XHTML ou outro tipo de mídia IANA (Internet Assigned Numbers Authority).
1. `Documentos de feed Atom` - É um documento XML que fornece metadados sobre um feed Atom e uma ou mais entradas para o feed. Quando um cliente faz uma solicitação de informações do feed, o documento do feed é gerado pelo servidor que inclui várias entradas Atom para atender à solicitação.
1. `Coleção Atom` - É um tipo especial de documento de alimentação Atom que contém os URLs das entradas Atom que estão disponíveis para serem editadas.
