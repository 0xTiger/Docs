## O que é um arquivo WBS?

Um arquivo WBS é um arquivo de projeto criado pelo WebBlender que é um programa de apresentação e criação de sites para crianças. O arquivo WBS é criado quando um novo projeto é iniciado no WebBlender. Todos os recursos do site adicionados ao projeto, como páginas, imagens, texto, animações e assim por diante, são referenciados no arquivo do projeto WBS. Quando o projeto estiver pronto, ele publica o conteúdo do site como arquivo [HTML](/pt/web/html/). O WebBlender é criado e mantido por [Tech4Learning](https://www.tech4learning.com/).
