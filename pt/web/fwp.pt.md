## O que é um arquivo FWP?

Um arquivo com arquivo .fwp é um arquivo criado com o Microsoft Expression Web e contém todos os arquivos e pastas de um design. O Microsoft Expression Web é um produto de software de web design geral da Microsoft e é baseado no editor [HTML](/pt/web/html/). Ele foi descontinuado desde 2012 e está disponível gratuitamente para download pela Microsoft. Os arquivos FWP podem ser exportados e salvos no formato .cab da Microsoft.

## Formato de arquivo FWP - Mais informações

As especificações exatas de formato de arquivo dos arquivos FWP não estão disponíveis para referência do desenvolvedor. No entanto, estes são arquivos binários e podem ser abertos no Microsoft Expression Studio que foi descontinuado há muito tempo.
