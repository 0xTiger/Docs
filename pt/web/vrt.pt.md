## O que é um arquivo VRT?

Um arquivo VRT é um formato de arquivo 3D que contém mundos tridimensionais que podem ser visualizados no navegador da Web com o plug-in SVR [Viscape](https://www.viscape360.com/). Ele é usado pelo plugin para carregar e exibir os dados do mundo 3D ao usuário para visualização e análise. As informações armazenadas em um arquivo VRT podem incluir dados como texturas, sons, dimensões e comportamentos. Os arquivos VRT mais recentes são armazenados no formato de arquivo SVR, que é uma versão compactada do arquivo VRT.

## Formato de arquivo VRT

Os arquivos VRT são armazenados no formato de arquivo proprietário do Viscape e seus detalhes de formato de arquivo interno não estão disponíveis publicamente.
