## O que é um arquivo .XPD?

Um arquivo XPD é um arquivo de página da web criado a partir de um documento XML. Mais frequentemente, isto é gerado através de uma transformação XSLT. As [especificações de formato de arquivo XPD](https://www.w3.org/TR/xml-pipeline/) estão disponíveis no W3 e definem uma maneira de aumentar a interoperabilidade de multiprocessos [XML](/pt/web/xml/) formulários. A XML Pipeline Language define os relacionamentos de processamento entre recursos XML.
