## O que é um arquivo AWM?

Um arquivo AWM é um arquivo de projeto criado pelo programa construtor de menus da Web, **AllWebMenus**. Ele contém dados relacionados ao projeto para criar menus suspensos baseados em [HTML](/pt/web/html/). Um arquivo AWM é compilado em um pacote da Web finalizado que se integra aos sites para exibir os menus. Ele contém todos os dados relacionados ao projeto, como configurações de aparência, entradas de menu, estilo (vertical ou horizontal), cores e outras informações relacionadas.

Arquivos AWM podem ser abertos com **Likno AllWebMenus**.

## Formato de arquivo AWM - Mais informações

Os arquivos AWM são salvos como arquivos binários compilados e só podem ser abertos com o software Likno AllWebMenus. Os detalhes do formato de arquivo interno dos arquivos AWM não estão disponíveis publicamente.

AllWebMenus permite que você crie menus [CSS](/pt/web/css/) ou menus baseados em [JavaScript](/pt/web/js/) visualmente sem usar nenhuma linguagem de programação. Os menus podem ser totalmente personalizados e projetados usando modelos de aparência predefinidos disponíveis com o software.
