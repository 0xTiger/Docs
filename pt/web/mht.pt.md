## O que é um arquivo MHT?

Um arquivo com extensão .mht é um formato de arquivo de arquivamento habilitado para MIME que contém diferentes tipos de dados em um único arquivo. Ele pode armazenar dados como texto, imagens, estilo de página na forma de arquivos [CSS](/pt/web/css/), JavaScript e outros recursos como recursos incorporados nele. Os arquivos MHT, tendo o tipo MIME message/rfc822, encapsulam todo o conteúdo de um arquivo HTML como um único arquivo de arquivamento para armazenamento em arquivamento em dispositivos de armazenamento. Aplicativos de software como o Microsoft Word permitem converter seus documentos WORD para MHT exportando como arquivo MHT. Os arquivos MHT podem ser abertos usando navegadores populares, como o Microsoft Internet Explorer e o Google Chrome.

## Formato de arquivo MHT

Assim como o [MHTML](/pt/web/mhtml/), o MHT também é um encapsulamento MIME de documentos HTML agregados. O conteúdo do documento dentro de um documento MHT, como imagens embutidas, folhas de estilo, applets, etc. são codificados em MIME, onde são vinculados a um recurso raiz por meio de URIs. Clientes de e-mail, como o Microsoft Outlook, armazenam e-mails (geralmente [EML](/pt/email/eml/)) no formato de arquivo MHT. As especificações completas do formato de arquivo MHT estão disponíveis no [RFC 822](https://tools.ietf.org/html/rfc822) e podem ser consultadas para o desenvolvimento de aplicativos de software que podem processar arquivos MHT.

## Diferença entre MHT e MHTML

Ao salvar uma página online, o usuário é solicitado a definir o tipo de formato de arquivo no qual a página online deve ser salva no sistema nativo. Normalmente, o usuário fica confuso entre a linguagem de marcação e os formatos de arquivo MHTML. Eles percebem que é problemático ver que o formato de arquivo é mais saudável entre eles.

Quando um usuário decide evitar o desperdício de página online como linguagem de marcação, então é formada uma pasta, que contém vários arquivos para vários conteúdos da página, ou seja, unidade de área de arquivos totalmente diferente criada para texto, gráficos, applets etc. a página online, pois o MHTML cria um arquivo para a página online completa. Todos os elementos da página juntamente com gráficos, links e arquivos alternativos são uma unidade incorporada dentro de um arquivo.

Naturalmente, é fácil lidar com arquivos MHT ou MHTML porque o arquivo, que pode ser protegido e compartilhado simplesmente por e-mail. No entanto, os arquivos de linguagem de marcação estão abaixo da chance de perda de informações como perda ou exclusão de até mesmo um arquivo pode tornar a pasta de linguagem de marcação completa inútil para o usuário.
