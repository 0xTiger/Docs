## O que é um arquivo XD?

Um arquivo com extensão .xd é uma maquete criada para sites ou aplicativos móveis usando o software Adobe XD. Designers, equipes criativas e organizações usam esses arquivos para projetar aplicativos, referenciando-os. Um único arquivo XD contém elementos de design, como imagens, formas e texto, que fornecem um conjunto completo de recursos para desenvolver protótipos de wireframe rápidos e intuitivos para que os desenvolvedores comecem a desenvolvê-los como aplicativos.

## Formato de arquivo XD

Os arquivos XD contêm elementos gráficos (formas de imagem e texto) em camadas como uma ou mais pranchetas. Estes, juntamente com guias e camadas, fornecem os blocos de construção para os wireframes.

* `Artboards` - Representam as telas em um aplicativo ou páginas de um site e agilizam os processos de design. Uma prancheta cria designs interativos para vários tamanhos de tela em um único documento.
* `Guias` - Estes são usados para alinhar objetos ou gerenciar o espaçamento entre elementos em uma prancheta. Os guias usam grades que servem como estrutura para organizar os elementos do design (imagens, glifos, parágrafos) e garantir a simetria do layout.
* `Layers` - designers de UX usam camadas no XD da mesma forma que usam no Adobe Illustrator ou Photoshop. Eles são usados para organizar o trabalho em pranchetas para evitar confusão e são controlados a partir do painel Camadas.
