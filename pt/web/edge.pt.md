## O que é um arquivo EDGE?

Um arquivo com extensão .edge é um arquivo Adobe Edge Project que é criado quando um projeto é criado e salvo com o Adobe Edge. O Adobe Edge permite que os web designers criem animadores [HTML](/pt/web/html/) interativos para web, publicação digital e publicidade de mídia avançada para navegadores de desktop e móveis. Isso é feito usando HTML/HTML5, [JavaScript](/pt/web/js/) e [CSS3](/pt/web/css/). Os arquivos Edge são gerados como arquivos de texto simples e podem ser abertos com vários aplicativos diferentes, como Microsoft Notepad, Apple TextEdit e Notepad++.

## Formato de arquivo EDGE

Os arquivos EDGE são salvos em um formato de texto simples e usam jQuery para ativar a animação. Uma alternativa de animação EDGE foram os arquivos Flash que não são mais suportados. O arquivo EDGE é salvo junto com os arquivos HTML, JavaScript e CSS. Ao abrir o arquivo EDGE, a página da web, ou seja, o arquivo HTML é aberta.
