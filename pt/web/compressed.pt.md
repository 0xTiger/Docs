## O que é um arquivo .COMPRESSED?

Um arquivo COMPRIMIDO é uma página da Web compactada do Wordpress de tamanho pequeno gerada com o plug-in **WordCompress**. Ele contém uma única linha de código para representar a página da web gerada pela compactação do código HTML, CSS e JavaScript pelo plugin. Esses arquivos COMPRIMIDOS convertidos são de tamanho pequeno, o que ajuda a agilizar o carregamento e a transferência pela Internet.

### Formato de arquivo COMPRIMIDO - Mais informações

Arquivos COMPRIMIDOS são salvos como arquivos de texto que podem ser abertos com qualquer editor de texto, como Microsoft Notepad, Notepad++ e Apple TextEdit. Além dos editores de texto, os arquivos COMPRIMIDOS podem ser abertos com software de criação de sites, como o Adobe Dreamweaver, disponível para Windows e Mac OS.

## Sobre o plug-in WordCompress

O plugin WordCompress foi usado inicialmente para compactar páginas da web WordPress em formato de arquivo compactado. O plugin não é mais suportado e foi descontinuado. Ele escreveria o código fonte da página da Web em um único arquivo executável, removendo todos os espaços em branco do código fonte da página. Os arquivos COMPRIMIDOS são armazenados junto com o código-fonte original da página da web para preservar os comentários originais, recuos e outros espaços em branco incluídos no código-fonte original da página.
