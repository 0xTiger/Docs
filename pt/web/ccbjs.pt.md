## O que é um arquivo .CCBJS?

Um arquivo CCBJS é um arquivo Javascript criado pelo aplicativo 3D, jogo e editor de site, **CopperCube**. Ele contém o código JavaScript gerado como parte do [arquivo .html](/pt/web/html/) criado a partir do arquivo .ccb. CopperCube fornece uma interface de script amigável para desenvolver lógica de jogo e interações sem escrever nenhum código. Outros formatos de arquivo suportados pelo CopperCube incluem .ccscene, .ccmodel, .ccmaterial, .cctex, .ccjs, .ccp e .ccpak.

## Formato de arquivo CCBJS

Os detalhes do formato de arquivo interno do formato de arquivo CCBJS não são conhecidos como tal. Os arquivos Javascript, no entanto, são salvos como arquivos de texto simples em disco e podem ser abertos com qualquer editor de texto, como Microsoft Notepad, Notepad++ e Apple TextEdit.
