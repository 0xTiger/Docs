## O que é um arquivo OBML16?

Um arquivo OBML16 é uma página da Web salva usando o navegador móvel Opera Mini. Ele é baseado na Linguagem de Marcação Binária Opera ([OBML](/pt/web/obml/)) e é um formato compacto para dar acesso rápido a páginas da web salvas. Ele substituiu o formato de arquivo [OBML15](/pt/web/obml15/) quando o Opera Mini versão 7 foi lançado. Todas essas variantes de páginas da Web do Opera Mini foram atualizadas a partir do formato de arquivo OBML básico e cada atualização do Opera Mini é compatível apenas com um formato OBML. Os arquivos OBML16 podem ser convertidos para [PDF](/pt/pdf/) e [HTML](/pt/web/html/) online.

## Formato de arquivo OBML16 - Mais informações

O OBML16 é baseado no formato de arquivo OBML, que é um formato compacto para salvar páginas da web. A vantagem de usar este formato compacto é que ele reduz o tamanho do conteúdo da web, resultando em um aumento na velocidade de download. Após o download, os usuários do Opera Mini podem visualizar essas páginas sem a necessidade de ter acesso a uma conexão com a Internet.
