## O que é um arquivo .SEAM?

Um arquivo SEAM é um arquivo de configuração usado pela estrutura JBOS Seam. Ele contém metadados/referência a um servlet do lado do servidor que processa solicitações para o aplicativo da web. Ele também contém metadados sobre os componentes do Seam, como nomes de classes e métodos que implementam os componentes. Você pode usar arquivos SEAM em conjunto com a estrutura JavaServer Faces (JSF) para construir aplicativos da web dinâmicos e interativos. Os arquivos SEAM geralmente são armazenados no diretório WEB-INF de um aplicativo Seam.

## Formato de arquivo SEAM

Um arquivo SEAM é armazenado como um arquivo de texto, normalmente no formato [XML](/pt/web/xml/) e contém informações de configuração para uma aplicação Seam. É uma parte fundamental da configuração de uma aplicação Seam. Você pode abrir um arquivo SEAM com qualquer editor de texto ou XML.
