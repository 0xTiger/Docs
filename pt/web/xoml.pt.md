## O que é um arquivo XOML?

XOML é um acrônimo para Extensible Object Markup Language e é um formato de serialização para objetos de fluxo de trabalho do Windows Workflow Foundation. Baseado em **[XAML](/pt/web/xaml/)**, é usado principalmente para criar interfaces de usuário em **[XML](/pt/web/xml/)** simples. Eles são usados para declarar o fluxo de trabalho para a interface do usuário e são compilados junto com o arquivo separado que contém a lógica de implementação. Ele contém uma estrutura baseada em árvore que define um nó de fluxo de trabalho raiz, subelementos aninhados e segmentos de código incorporados. Quando um novo fluxo de trabalho é criado com o Visual Studio para .NET, ele tem a opção de escolher se deve estar no formato de código ou no formato de código separado. Caso você selecione o último, o IDE criará dois arquivos para você; um no formato XOML (consistindo no layout e configurações do fluxo de trabalho) e outro no formato [.CS](/pt/programming/cs/)/[.VB](/pt/programming/vb/) onde reside o código de programação.
