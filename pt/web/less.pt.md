## O que é um arquivo LESS?

Um arquivo LESS é uma folha de estilo usada para páginas da Web e criada com o Leaner Style Sheets (LESS). Ele implementa adições à linguagem CSS padrão, o que facilita o aprendizado e a implementação. Um arquivo LESS estende o [CSS](/pt/web/css/) adicionando recursos como Variáveis, Mixins, Aninhamento, Operações, Funções, Namespaces e Accessors.

## Formato de arquivo de folhas de estilo mais enxutos

Os arquivos LESS são criados no formato CSS, estendendo seus recursos pelos recursos adicionais que ele oferece. Eles são salvos como arquivos de texto e podem ser abertos em linguagem de folha de estilo MENOS, CodeKit e Adobe Dreamweaver.
