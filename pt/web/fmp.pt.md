## O que é um arquivo FMP?

Um arquivo com arquivo .fmp é um arquivo de menu animado criado com o aplicativo 123 Flash Menu. Ele contém menus animados para o site, incluindo componentes como fontes, estilo do menu, cores, dimensões, sons e animações. O aplicativo 123 Flash Menu não requer nenhum conhecimento especial de informática, pois o próprio aplicativo incorpora todas as funcionalidades para desenvolver esses menus. Os arquivos do menu FMP podem ser exportados para os arquivos [.swf](/pt/page-description-language/swf/) do Flash, que podem ser incorporados em uma página da web.
