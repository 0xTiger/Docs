## O que é um arquivo DOCMHTML?

Um arquivo DOCMHTML é uma antiga variante de arquivo da web salva pelas versões anteriores do Microsoft Word. Não é mais suportado pela versão mais recente do Microsoft Word. Um arquivo DOCMHTML contém representação HTML do documento original do Word .doc. Ele pode ser aberto no Microsoft Word, bem como no navegador da web. Quando um DOCMHTML é publicado, ele pode ser hospedado em um servidor web acessível aos usuários via solicitação web. Outro formato semelhante é o [DOCHTML](/pt/web/dochtml/) que também foi descontinuado.

## Formato de arquivo DOCMHTML - Mais informações

Os arquivos DOCMHTML foram usados para serem salvos no formato HTML. Ele pode ser aberto em um editor de texto como o Notepad e o Notepad++. Após ser marcado como obsoleto, o Microsoft Word introduziu as opções de salvar documentos do Word como arquivos [MHTML](/pt/web/mhtml/) e [MHT](/pt/web/mht/). Esses novos formatos de arquivo podem ser abertos com navegadores da web.
