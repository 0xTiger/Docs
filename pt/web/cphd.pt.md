## O que é um arquivo .CPHD?

Um arquivo CPHD é um documento de hipertexto usado pelo mecanismo de pré-processador CloudChan. Ele contém HTML e scripts para criar páginas e documentos dinâmicos da web. Os scripts dentro de um arquivo CPHD se assemelham à sintaxe usada por Java. A página da web CPHD retorna um documento HTML quando executado em processos CloudChan. O formato CPHD pode funcionar com qualquer documento XML.

## Formato de arquivo CPHD

Os arquivos CPHD são armazenados em disco como arquivos de texto simples. Isso torna possível abrir esses arquivos em qualquer editor de texto, como o Notepad ou o Notepad++.
