## O que é um arquivo XOJ?

Um arquivo XOJ é um arquivo de notebook que contém anotações feitas usando o software de anotações manuais Xournal. O Xournal é um aplicativo gratuito e multiplataforma que permite criar notas com diferentes tipos de conteúdo, como texto, imagens e rabiscos escritos à mão. Os arquivos XOJ são criados quando um novo arquivo de notebook é criado com o Xournal. Esses arquivos XOJ podem ser compartilhados com outros usuários também para compartilhamento de dados. Uma versão mais recente do Xournal é o [Xournal++](https://github.com/xournalpp/xournalpp) que é uma variante multiplataforma do Xournal.

## Formato de arquivo XOJ

Os arquivos XOJ estão no formato de arquivo XML e são salvos em disco em formato compactado usando a compactação [.gz](/pt/compression/gz/). Devido ao uso do formato XML universal, os arquivos XOJ também podem ser abertos e editados com a variante multiplataforma Xournal++.

Cada arquivo XOJ pode conter mais uma página de um notebook Xournal.

## XOJ - Plataformas Suportadas

O Xournal é multiplataforma e seus notebooks podem ser salvos como arquivo XOJ nos seguintes sistemas operacionais.

*Linux
* Mac OS
* Janelas
*Android
* Construir iOS
