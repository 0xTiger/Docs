## O que é um arquivo A4P?

Um arquivo A4P é um arquivo de projeto criado pelo programa de software A4Desk Pro que é usado para criar sites em Flash. Ele é criado com os modelos incluídos no A4Desk Pro e não precisa ter um conhecimento profundo de Flash. Arquivos A4P criados com modelos podem incluir conteúdo do site no texto do formulário e animações em Flash. Caso você não conheça o Flash, ele foi usado para criar animações que podem ser exportadas para os formatos de arquivo [SWF](/pt/page-description-language/swf/) e [GIF](/pt/image/gif/) para inclusão em sites.

## Formato de arquivo A4P

Os arquivos A4P são criados a partir de arquivos de modelo e salvos como arquivos binários no disco. A estrutura de formato de arquivo interno dos arquivos A4P não é conhecida.

Os arquivos A4P podem ser abertos com o software **Avanquest A4Desk Pro**.
