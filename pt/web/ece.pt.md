## O que é um arquivo ECE?

Um arquivo ECE é uma página da Web gerada dinamicamente pela plataforma de gerenciamento de conteúdo, **Escenic Content Engine (ECE)**. Ele contém código do lado do servidor que é processado pelo servidor da web e retornado ao navegador do usuário como página [HTML](/pt/web/html/). O formato de arquivo ECE não é muito popular em comparação com outros formatos de arquivo do lado do servidor, como ASP e PHP. Esses arquivos são renderizados internamente e servidos aos usuários finais para fins de exibição.

## Formato de arquivo ECE

Os arquivos ECE não são salvos em disco, mas são fornecidos diretamente ao usuário mediante solicitação. Estes são convertidos em HTML e enviados de volta ao navegador do usuário para exibição. O Escenic Content Engine é usado ativamente para publicar páginas de artigos de notícias. Baseado na tecnologia JEE e SQL, o servidor ECE fornece um repositório confiável e escalável para todo o conteúdo online.

Os navegadores cliente podem acessar o conteúdo nos servidores ECE por meio da API HTTP/REST. Além de servir páginas [HTML](/pt/web/html/), o ECE oferece um fluxo de trabalho altamente eficiente para gerenciar e publicar conteúdo de vídeo/áudio.
