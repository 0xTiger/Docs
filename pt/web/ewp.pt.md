## O que é um arquivo EWP?

Um arquivo EWP é um arquivo de projeto criado com o software [EWISoft](http://www.ewisoft.com/) Website Builder. Ele está vinculado a um arquivo de banco de dados binário que contém todas as informações do projeto. O arquivo EWP é comumente usado para salvar o progresso do desenvolvimento da web. É um software de web design orientado a modelos que permite aos usuários criar sites a partir de modelos de sites.

## Formato de arquivo EWP - Mais informações

Os arquivos EWP são arquivos de projeto que não precisam ser compilados ou hospedados em nenhum servidor. Eles são salvos como arquivos de texto simples e contêm referências ao progresso do desenvolvimento da web. Os projetos podem ser publicados em um site assim que o desenvolvimento estiver concluído.
