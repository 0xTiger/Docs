## O que é um arquivo WHTT?

Um arquivo WHTT é uma versão offline de um site que é baixado usando o software [HTTrack Website Copier](https://www.httrack.com/). Ele contém todo o conteúdo de um site, incluindo páginas da Web [HTML](/pt/web/html/), imagens, folhas de estilo, links e diretórios. Todas as páginas de um site são baixadas recursivamente e salvas no disco como espelho offline. Quando um site é baixado pelo software HTTrack, ele pode ser visualizado offline sem a necessidade de estar conectado à internet. Para visualizar o site baixado, basta abrir a página do site espelhado no navegador e cada imagem/link será carregado corretamente.

## Formato de arquivo WHTT

Os arquivos WHTT são salvos como uma combinação de páginas da Web THML, imagens, folhas de estilo e links de referência.
