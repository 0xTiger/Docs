## O que é um arquivo HTC?

Um arquivo HTC é um arquivo de formato de arquivo da Web usado pelo Microsoft Internet Explorer. Ele contém definições para componentes HTML que executam pequenas funções para realizar diferentes tarefas dentro do navegador da web. A funcionalidade definida dentro dos arquivos HTC é reutilizável e acessível em várias páginas [HTML](/pt/web/html/).

## Formato de arquivo HTC - Mais informações

Os arquivos HTC são salvos em formato de arquivo de texto simples. Isso significa que você pode abri-los em qualquer editor de texto e fazer alterações na funcionalidade. Eles geralmente são abertos no Internet Explorer, mas dessa forma você só poderá visualizar esses arquivos e não poderá editá-los.

Um arquivo HTC é na verdade um arquivo HTML que é uma combinação de script e um conjunto de elementos específicos do HTC para definir os componentes.
