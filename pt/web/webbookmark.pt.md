## O que é um arquivo WEBBOOKMARK?

Um arquivo WEBBOOKMARK é um arquivo de favoritos salvo com o navegador Safari da Apple no macOS. Ele contém informações sobre a página da Web que o usuário salvou do navegador. As informações armazenadas em um arquivo WEBBOOKMARK incluem o título da página, a URL e o grupo de marcadores no qual esta página está armazenada. Quando uma página da Web é marcada, o navegador Safari solicita ao grupo para armazenar a página marcada. A maioria dos usuários salva os favoritos na pasta Favoritos, que é o local padrão oferecido pelos navegadores comuns.

## Formato de arquivo WEBBOOKMARK

Os arquivos WEBBOOKMARK são salvos em disco em formato de arquivo binário que não é legível por humanos. Os usuários podem marcar uma página da Web como favorita se ela for visitada posteriormente ou se estiver relacionada aos interesses do usuário. Quando necessário, o usuário pode navegar até a seção de favoritos no navegador Safari e visitar a página novamente.
