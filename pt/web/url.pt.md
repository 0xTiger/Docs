## O que é um arquivo de URL?

Um arquivo com extensão .url é um atalho de URL de site que pode ser salvo no computador com qualquer endereço de site. Ao abrir com um duplo clique, a URL abrirá o site no navegador padrão e a página será exibida ao usuário. Os arquivos de URL geralmente são chamados de arquivos de atalho para sites e são representados pelo ícone do navegador padrão. Os arquivos de URL podem ser abertos e editados em qualquer editor de texto, como Microsoft Notepad, Notepad++ ou Apple TextEdit.

## Formato de arquivo de URL - Mais informações

Os arquivos de URL são salvos em formato de texto simples e podem ser abertos e visualizados em qualquer editor de texto. Eles são abertos no navegador da Web padrão, onde o usuário realiza outra navegação.
