## O que é um arquivo DOCHTMML?

Um arquivo DOCHTML é o formato HTML de um documento gerado com o Microsoft Word. É a versão [HTML](/pt/web/html/) do documento criado que pode ser usada para publicação no servidor web. Os arquivos DOCHTMML podem ser abertos no navegador da Web para visualizar o conteúdo do documento do Word convertido. A perspectiva e a aparência do arquivo DOCHTML são as mesmas do documento original do Word.

## Formato de arquivo DOCHTMML

Os arquivos DOCHTML são salvos em disco no formato HTML que está pronto para ser publicado em servidores web. O Microsoft Word interrompeu o suporte para salvar documentos como arquivos DOCHTML. Isso foi substituído pelos formatos de arquivo [.mht](/pt/web/mht/) e [.mhtml](/pt/web/mhtml/) que são comumente usados agora.
