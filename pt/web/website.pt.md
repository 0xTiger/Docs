## O que é um arquivo de SITE?

Um arquivo WEBSITE é um arquivo de atalho de site criado com o Microsoft Internet Explorer (IE). Ele contém uma URL e um arquivo de ícone (ICO) que aponta para o favicon do site. O ícone é usado para exibir o atalho. O formato de arquivo WEBSITE foi introduzido com o lançamento do IE versão 9. As versões anteriores do IE eram usadas para salvar atalhos para páginas da Web como arquivos [.url](/pt/web/url/).

## Formato de arquivo do SITE

Um arquivo .website é armazenado em disco como arquivos de texto simples, cujo conteúdo pode ser visualizado em qualquer editor de texto. Além de outros conteúdos, contém o URL de destino para o qual é um atalho. Se você quiser abrir o site de destino em um navegador diferente do IE, poderá copiar esse URL e colá-lo em outros navegadores, como Google Chrome, Safari ou Firefox para visualizar o site.

## Como criar um arquivo .WEBSITE?

Um arquivo WEBSITE pode ser criado no IE clicando no URL na barra de endereços e arrastando-o para o local desejado em seu computador, como área de trabalho ou documentos. Isso cria o atalho para esse local da Internet e pode ser aberto clicando duas vezes no atalho. Se você quiser salvar o atalho como arquivo `URL`, mantenha pressionada a tecla SHIFT antes de arrastar o favicon do site de destino para a área de trabalho.
