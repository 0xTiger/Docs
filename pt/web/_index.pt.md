São necessários vários tipos de arquivo ao criar uma página da web. Os formatos de arquivo da Web definem os padrões para o desenvolvimento de páginas da Web e estão relacionados à plataforma na qual elas são construídas. Um site completo pode ser construído consistindo de páginas estáticas e dinâmicas. A maioria dos sites modernos são criados com tecnologia do lado do servidor, como Active Server Pages (ASP), que são carregadas e executadas no servidor web. Isso também inclui folhas de estilo em cascata (CSS) e arquivos de script usados para estilizar a aparência geral da interface do usuário.

As extensões comuns de arquivo da Web e seus formatos de arquivo associados incluem **[HTML](/pt/web/html/)** (Hypertext Markup Language), **[ASP](/pt/web/asp/) (Active Server Pages)**, e **CSS** (Folhas de Estilo em Cascata).
