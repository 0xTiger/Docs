## O que é um arquivo ESPROJ?

Um arquivo ESPROJ é um arquivo de projeto criado por um software de criação de páginas da Web, o Espresso. Está disponível no Mac OS e contém uma lista dos arquivos relacionados que fazem parte do projeto do site. Ao abrir o arquivo ESPROJ no Espresso, ele abre todos os arquivos relacionados no editor para edição ou atualização do código-fonte. O arquivo ESPROJ também contém configurações de publicação do servidor e outras preferências de usuário relacionadas.

## Formato de arquivo ESPROJ

Os arquivos ESPROJ são salvos como arquivos binários e seu conteúdo não é legível por humanos. O diretório de arquivos ESPROJ também é o diretório raiz do site.
