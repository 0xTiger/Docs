## O que é um arquivo TS?

Um arquivo TS é um arquivo de tradução gerado com o utilitário de software lupdate que vem junto com o Qt SDK da Digia. O SDK está disponível para os sistemas operacionais Windows, Mac e Linux. O arquivo TS contém frases em um idioma e suas traduções para outro idioma. As traduções armazenadas dentro de um arquivo TS são específicas para um determinado aplicativo. Os arquivos TS podem ser abertos com o Digia Qt SDK.

## Formato de arquivo TS

Os arquivos Digia Qt TS são armazenados em disco no formato de arquivo XML e podem ser abertos em qualquer editor de texto, como Notepad e Notepad++. Alguns outros formatos de arquivo suportados pelo Digia Qt SDK incluem:

* .pri
* .qm
* .qml
* .qph
* .qrc
* .qs
* .qss
* .ts
* Arquivo de interface do usuário .ui
