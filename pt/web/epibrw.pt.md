## O que é um arquivo EPIBRW?

Um arquivo EPIBRW é um arquivo genérico hospedado no servidor da Web baseado em Microsoft IIS. A finalidade deste arquivo não é específica para nenhuma operação específica ou qualquer tipo de armazenamento de dados, mas é geral. É usado com mais frequência por arquivos [.pdf](/pt/pdf/), mas também pode ser utilizado por vários outros tipos de arquivo.

Quando um arquivo EPIBRW é clicado duas vezes, ele pode ser aberto no navegador exibindo o conteúdo. No entanto, se o arquivo estiver vinculado para download, o processo de download do arquivo será iniciado como resultado desse evento.
