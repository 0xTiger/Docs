## O que é um arquivo CODASITE?

Um arquivo CODASITE é um arquivo de projeto de site criado com o software de criação e publicação de páginas da Web [Coda](https://coda.io/). Ele contém uma lista de todos os arquivos que fazem parte do projeto. Isso inclui páginas da Web como arquivos [HTML](/pt/web/html/), arquivos de folha de estilo em cascata [CSS](/pt/web/css/), arquivos Javascript [JS](/pt/web/js/) e quaisquer outros arquivos de suporte relacionados. Os arquivos de projeto CODASITE também contêm informações para publicação de sites no servidor web.

## Formato de arquivo CODASITE

O arquivo de projeto CODASITE é salvo em disco como arquivos binários dentro do diretório ~/​Library/​Application Support/​Coda 2/​Sites/​. A interface de usuário Coda fornece a interface para gerenciar sites Coda, pois abri-los manualmente pode levar a configurações inadequadas do projeto e criar problemas.
