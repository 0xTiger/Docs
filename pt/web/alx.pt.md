## O que é um arquivo ALX?

Um arquivo ALX é um arquivo ActiveX incorporado em páginas da Web na forma de componentes [HTML](/pt/web/html/). Os controles ActiveX eram formalmente conhecidos como controles OLE ou OCXs. Os arquivos ALX podem ser usados para inserir uma variedade de funcionalidades em uma página da Web, como adicionar um controle de timer para atualizar o conteúdo da página periodicamente ou adicionar um controle de janela pop-up para exibir ajuda no estilo de dica de ferramenta com base na ação do usuário. Os arquivos ALX são vinculados usando<Param> tag dentro do código HTML de uma página da web. Você pode abrir arquivos ALX no Microsoft Internet Explore, no Microsoft ActiveX Control Pad e em editores de texto como o Notepad e o Notepad++.

## Formato de arquivo ALX - Mais informações

Os arquivos ALX estão em formato de arquivo de texto simples e podem ser editados abrindo-os em um editor de texto. Os controles ActiveX podem ser incluídos em uma página da Web, soltando-os na página da Web. No entanto, para criar páginas da Web dinâmicas, os controles são integrados com a ajuda de [JScript](/pt/web/js/). Os controles em uma página da Web são associados aos eventos e valores dos controles, proporcionando controle completo para as interações do usuário.
