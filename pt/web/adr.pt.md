## O que é um arquivo ADR?

Um arquivo ADR (salvo com extensão .adr) é uma lista de marcadores criados pelo navegador Opera. Esta lista é preenchida à medida que o usuário continua marcando URLs da web. Esses marcadores são armazenados no arquivo ADR que pode ser copiado e restaurado quando necessário. Como a maioria dos navegadores, como Chrome e Firefox, salva marcadores no formato [HTML](/pt/web/html/), eles podem ser convertidos para o formato de arquivo ADR para importação no navegador Opera. Os arquivos ADR podem ser abertos no navegador Opera.

## Formato de arquivo ADR - Mais informações

Os detalhes do formato de arquivo interno dos arquivos ADR são desconhecidos. No entanto, existem diferenças entre os formatos de arquivo ADR exportados com diferentes versões do Opera. Isso dificulta [mesclar arquivos ADR](https://superuser.com/questions/471959/how-do-i-merge-several-opera-adr-bookmark-files-to-one-single-file) gerar com diferentes versões do Opera. Algumas ferramentas externas podem ser usadas para mesclar estes embora.
