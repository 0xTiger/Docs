## O que é WEBLOC?

Um arquivo WEBLOC é um arquivo de atalho no macOS que aponta para um URL da web na internet. Ele é gerado arrastando o URL da web da barra de endereços do navegador para a área de trabalho ou qualquer outra pasta onde ele deve ser salvo. A URL para o site de destino é armazenada dentro desse arquivo de atalho e pode ser visualizada abrindo o arquivo em um editor de texto como o Microsoft Notepad, Notepad++ ou Apple TextEdit. Os arquivos WEBLOC são semelhantes aos arquivos [URL](/pt/web/url/).

## Formato de arquivo WEBLOC - Mais informações

Os arquivos WEBLOC são armazenados em formato de arquivo de texto simples. Eles podem ser gerados no macOS arrastando URLs de navegadores da Web, como Safari, Google Chrome ou Firefox. Se vários arquivos WEBLOC forem abertos simultaneamente, seria necessária uma solução mais rápida e fácil de abri-los por meio de aplicativos ou APIs externos. Essas APIs podem ler e converter WEBLOC para PDF e JPEG.
