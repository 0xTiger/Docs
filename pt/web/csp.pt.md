## O que é um arquivo CSP?

Um arquivo CSP é um arquivo de página web dinâmica que contém conteúdo escrito para ser executado no RadGs Concept Application Server. Ele é compilado e executado no lado do servidor, após o qual é servido ao navegador do usuário solicitante via protocolo HTTP. Ao contrário do formato de arquivo CON, os arquivos CSP não exigem que o cliente Concept seja instalado no usuário para visualizar esses arquivos. Você pode visualizar arquivos CSP semelhantes a qualquer outra página do lado do servidor, como [PHP](/pt/programming/php/) e [ASP](/pt/web/asp/).
