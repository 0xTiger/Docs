## O que é um arquivo .VDISCO?

Um arquivo VDISCO é um arquivo de descoberta usado pelo Microsoft Visual Studio em seus serviços da Web. Ele fornece informações sobre os serviços da Web disponíveis, como URLs, namespaces e métodos. O arquivo VDISCO é semelhante ao formato de arquivo [DISCO](/pt/web/disco/), mas é gerado em tempo de execução. Ele é armazenado no aplicativo de serviço Web e usado pelo Visual Studio para descobrir e usar serviços Web dinamicamente em um projeto. Os arquivos VDISCO são usados em combinação com o arquivo [.asmx](/pt/web/asmx/) que contém o código real do serviço da Web.

Você pode abrir o arquivo VDISCO em qualquer editor de texto, como Microsoft Notepad, Github Atom e Apple TextEdit. Também pode ser aberto com o Microsoft Visual Studio 2012 e posterior para trabalhar com ele.

## Formato de arquivo VDISCO

Um arquivo VDISCO é salvo e hospedado no formato de arquivo [XML](/pt/web/xml/), que é um formato universal para composição e publicação de dados. Ele contém a URL do serviço, namespaces e métodos em tags XML padrão onde cada tag contém o respectivo valor de suas informações.
