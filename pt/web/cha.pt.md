## O que é um arquivo CHA?

Um arquivo CHA é um arquivo de configurações do usuário criado pelos clientes de IRC, como o mIRC. Ele contém parâmetros como servidor, porta, nome do canal e, opcionalmente, uma senha para abrir um canal de IRC. Os clientes de IRC são usados para se comunicar com outras pessoas nas redes de IRC ao redor do mundo. Eles são usados para discussões privadas individuais ou em conferências com vários grupos. Quando um arquivo CHA é clicado, o cliente IRC abre automaticamente com os detalhes mencionados no arquivo.

Os aplicativos que podem abrir arquivos CHA incluem mIRC e Visual IRC.

## Formato de arquivo CHA

Os arquivos CHA são armazenados junto com outros arquivos de instalação do cliente IRC. Quando um usuário deseja se conectar a um servidor e ingressar em um canal, ele insere as informações no cliente. Essas informações são armazenadas no arquivo CHA que pode ser recuperado a qualquer momento.
