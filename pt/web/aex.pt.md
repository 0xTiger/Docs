## O que é um arquivo AEX?

Um arquivo AEX (com extensão .aex) é um arquivo compilado [Xbasic](https://documentation.alphasoftware.com/documentation/pages/Ref/Xbasic/index.xml) que contém código de programa compilado para funções globais. Ele pode ser usado em aplicativos da Web chamando-os das páginas [.a5w](/pt/web/a5w/) do lado do servidor. Os arquivos AEX foram carregados e descarregados chamando as funções a5w_load_aex e a5w_unload_aex, respectivamente, de arquivos A5W. No entanto, na nova implementação, esses arquivos são carregados na memória incluindo no arquivo a5_application.5i. A linguagem de programação Xbasic é usada pelo software Alpha Anywhere para desenvolvimento de aplicativos móveis e da Web.

## Formato de arquivo AEX - Mais informações

Arquivos AEX são salvos em disco em formato de arquivo binário e contém código compilado de funções escritas em Xbasic. As instruções de comando de script Xbasic determinam a estrutura e o fluxo de execução, incluindo a execução ou interrupção de uma ação repetida ou a escolha entre duas ou mais etapas com base nas condições. [Xbasic Language Reference](https://documentation.alphasoftware.com/documentation/pages/Ref/Xbasic/index.xml) pode ser consultado para obter informações detalhadas sobre ele.
