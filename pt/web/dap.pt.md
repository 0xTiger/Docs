## O que é um arquivo DAP?

Um arquivo com extensão .dap é um arquivo de página de dados do Microsoft Access que é criado no Microsoft Access e tem acesso aos dados armazenados no banco de dados. Os dados armazenados em tal arquivo da web são acessíveis pela rede (intranet e internet) que é diferente dos arquivos [MDB](/pt/database/mdb/) do Access que são desenvolvidos especificamente para aplicativos de desktop. Esses bancos de dados podem ser criados usando o Access 2010 e o Access Services (um componente opcional do SharePoint). Esses bancos de dados podem ser usados por pessoas com contas do SharePoint usando o endereço da Web em um navegador da Web.
