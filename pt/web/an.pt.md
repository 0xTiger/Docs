## O que é um arquivo AN?

Um arquivo com extensão .an é um arquivo de projeto do [Adobe Edge Animate](https://www.adobe.com/products/animate.html) que contém informações sobre a animação. Um arquivo .html é criado quando o projeto é salvo, contendo as referências aos arquivos associados [.css](/pt/web/css/) e [.js](/pt/web/js/) gerados pelo Adobe Edge Animate. O arquivo AN é salvo junto com esses arquivos que podem ser usados para abrir o arquivo do projeto. O Adobe edge animate pode ser usado como uma alternativa à criação de páginas da Web animadas usando o Adobe Flash, que não é mais suportado pelos navegadores.

## Formato de arquivo AN

Os arquivos AN são armazenados em disco em formato de arquivo de texto simples e podem ser abertos em editores de texto para visualizar seu conteúdo.
