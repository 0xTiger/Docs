## O que é um arquivo .XHTM?

Um arquivo com extensão .xhtm é um arquivo de página da web escrito em XHTML. Ele usa XML 1.0 como implementação padrão e é considerado uma reformulação do HTML 4. Assim como o XHTML, um arquivo XHTM contém tags HTML padrão que seguem diretrizes XML mais rígidas. Os arquivos XHTM são usados para publicar conteúdo de sites e geralmente usam a extensão [XHTML](/pt/web/xhtml/).

Você pode usar navegadores da web como Google Chrome, Apple Safari e Mozilla Firefox para abrir arquivos XHTM.

## Exemplo XHTM

O exemplo a seguir de um documento XHTM define as declarações XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
