## O que é um arquivo .SVR?

Um arquivo SVR é um formato de arquivo 3D usado pelo SuperScape para criar mundos interativos 3D. Esses mundos interativos tridimensionais podem ser visualizados através de navegadores da web com o plugin SVR. Os arquivos SVR são semelhantes ao formato [arquivo VRT](/pt/web/vrt/), mas são compactados, o que resulta em um tamanho de arquivo menor. Assim, os arquivos SVR são carregados mais rapidamente e também podem ser compartilhados pela internet.

## Formato de arquivo SVR – Mais informações

Os arquivos SVR são salvos em disco em formatos de arquivo binário. É um formato de arquivo proprietário e os detalhes de sua estrutura interna de arquivos não estão disponíveis publicamente para referência do desenvolvedor.
