## O que é um arquivo CRDOWNLOAD?

Um arquivo CRDOWNLOAD é um arquivo parcialmente baixado que é criado quando você baixa um arquivo da Internet usando os navegadores Microsoft Edge, Google Chrome ou Chromium. O conteúdo do arquivo que está sendo baixado é armazenado neste arquivo até que o download seja concluído. Quando o download for concluído, a extensão .crdownload será removida e substituída pela extensão original do arquivo que está sendo baixado.

## Formato de arquivo CRDOWNLOAD

Um arquivo CRDOWNLOAD pode estar em formato de texto ou binário. Os navegadores criam o arquivo .crdownload por padrão assim que o download é iniciado. Se o download for interrompido no meio, o arquivo CRDOWNLOAD permanece salvo no disco. Isso às vezes permite retomar o download de onde foi interrompido.

### Exemplo de CRDOWNLOAD

Suporte você está baixando um arquivo fileformatslist.doc da internet. Quando o download é iniciado, seu navegador salva o arquivo na pasta Downloads como arquivo fileformatslist.crdownload. O tamanho do arquivo aumenta à medida que o arquivo é baixado. Quando o download for concluído, o arquivo será renomeado para fileformatslist.doc.
