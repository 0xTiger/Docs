## O que é um arquivo STC?

Um arquivo com extensão .stc é um arquivo de conexão usado pelo software Adobe Contribute para armazenar as informações de conexão. Ele contém informações de conexão, como nome de usuário, URL do host, protocolos como FTP, SFTP, etc. para reutilização de dentro do site existente. Para quem não sabe, o Adobe Contribute costumava ser um editor [HTML](/pt/web/html/) especializado que era usado para enviar atualizações para sites existentes, incluindo artigos baseados em conteúdo conhecidos como blogs. Ele incluía plug-ins para navegadores populares, como Internet Explorer e Firefox, que contribuíam perfeitamente com conteúdo para sites de dentro das páginas. Ele foi descontinuado desde janeiro de 2017 e o Adobe Dreamweaver é usado agora para criar sites do zero.

## Formato de arquivo STC

O formato de dados internos dos arquivos STC é desconhecido. No entanto, a chave de conexão é criptografada com uma senha e qualquer informação de conexão de login de rede ou FTP enviada no arquivo permanece segura e só pode ser acessada por meio do Contribute. O arquivo de conexão de chave pode ser enviado por e-mail aos usuários ou salvo no computador. Quando clicado, abre automaticamente o Contribute que importa a conexão.
