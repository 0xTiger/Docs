## O que é um arquivo .CPG?

Um arquivo CPG é um arquivo de projeto criado pelo software editor de páginas da Web, Cool Page. O software tornou-se obsoleto há muitos anos e não é mais usado. O arquivo CPG continha detalhes sobre os objetos usados na página web e as propriedades desses objetos. Os arquivos do projeto CPG podem ser abertos no software Cool Page e após a finalização, a página web pode ser exportada como página web [HTML](/pt/web/html/).

## Formato de arquivo CPG

Os detalhes do formato de arquivo interno do formato de arquivo CPG não são conhecidos. Eles foram salvos como arquivos binários e só serão abertos no software Cool Page.
