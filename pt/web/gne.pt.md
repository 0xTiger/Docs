## O que é um arquivo GNE?

Um arquivo GNE é uma página da Web que é gerada dinamicamente e usada pelo site de compartilhamento de fotos do Flickr. As páginas são hospedadas no servidor de onde são servidas ao navegador do usuário solicitante. Os arquivos GNE são semelhantes às páginas PHP e ASP, mas são desenvolvidos em uma linguagem de programação proprietária cujos detalhes não estão disponíveis para referência do desenvolvedor.

## Formato de arquivo GNE - Mais informações

Os arquivos GNE foram usados pelo jogo de computador Game Never Ending, desenvolvido pela [Ludicorp](https://en.wikipedia.org/wiki/Ludicorp). Flicker foi posteriormente adquirido pelo yahoo! em março de 2005.
