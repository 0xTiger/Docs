## O que é um arquivo CHAT?

Um arquivo CHAT é um arquivo de configuração usado pelo IRC (Internet Relay Chat) para se conectar a um canal IRC. Ele contém parâmetros de conexão como servidor, porta, nome do canal de IRC e, opcionalmente, uma senha para abrir um canal de IRC. As configurações são usadas pelo cliente para se conectar a outros canais. Os chats e discussões no IRC são individuais e privados.

Os aplicativos que podem **abrir** arquivos CHAT incluem mIRC e Visual IRC.

## Formato de arquivo CHAT

Os arquivos CHAT são armazenados como arquivos binários no disco. Quando um usuário deseja se conectar a um canal no IRC, ele insere as informações no cliente de IRC, ele insere as informações no software cliente. As informações são armazenadas no arquivo CAHT para recuperação lateral.
