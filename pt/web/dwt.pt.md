## O que é um arquivo DWT?

Um arquivo com extensão .dwt é um arquivo de modelo de página da Web criado com o software Dreamweaver. Ele é usado como ponto de partida para desenvolver várias páginas que possuem as configurações básicas já definidas no modelo. Novas páginas [HTML](/pt/web/html/) podem ser criadas usando cópias desses modelos e alterando o conteúdo quando necessário, mantendo as seções comuns inalteradas. As regiões não editáveis desses modelos, como cabeçalho, rodapé e menu, permanecem as mesmas em todas as páginas da web.

## Formato de arquivo DWT - Mais informações

Os arquivos DWT são desenvolvidos e salvos como arquivos de texto simples. Eles podem ser abertos no Dreamweaver ou em qualquer editor de texto e podem ser editados para as alterações necessárias. O software Dreamweaver atualiza a interface do usuário à medida que as alterações são aplicadas e o usuário pode observar o efeito das alterações visualmente na interface do usuário do aplicativo.

Sempre que você salva a página [HTML](/pt/web/html/) como um arquivo de modelo, o Dreamweaver cria uma pasta de modelo no nível raiz da pasta raiz local. O link é quebrado se o arquivo .dwt for movido da pasta Modelos ou for renomeado. Assim, é importante manter os arquivos .dwt na pasta Templates onde o Dreamweaver os cria. É melhor manter outros elementos do seu site, como arquivos de origem de imagem ou documentos HTML, em um local diferente da pasta Modelos.

## Sobre o Dreamweaver

O software Dreamweaver foi originalmente desenvolvido pela Macromedia. Outro software popular da Macromedia incluiu o Flash player que permaneceu disponível como um player de animação para navegadores por um longo tempo. Depois que a Adobe adquiriu o Dreamweaver em 2005, o Dreamweaver tornou-se parte integrante dos aplicativos de software da Adobe e agora é distribuído pela Adobe separadamente.
