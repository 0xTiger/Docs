## O que é um arquivo HXS?

Um arquivo HXS é um arquivo de ajuda compilado e usado para o arquivo Microsoft Help 2.x. Ele é compilado e gerado a partir de um conjunto de páginas de tópicos e escrito em um subconjunto de [HTML](/pt/web/html/) semelhante ao formato de arquivo de ajuda anterior [CHM](/pt/web/chm/). Os arquivos HXS são armazenados em formato compactado e são criados a partir de um arquivo de projeto principal .HxC, um arquivo de inclusão .HxF, um índice .HxT, um arquivo de definição de atributo .HxA e vários índices .HxK. Os metadados de um arquivo HXS são criados a partir de [XML](/pt/web/xml/).

Os arquivos HXS podem ser abertos com um navegador da web.

## Formato de arquivo HXS

Os arquivos HXS são criados e salvos como arquivos compactados de conteúdo HTML e metadados XML.
