## O que é um arquivo PRO? ##

Um arquivo PRO é uma maquete de um site ou aplicativo móvel criado pelo Adobe Proto, um aplicativo Adobe Touch extinto. Inclui um ou mais layouts de interface do usuário para uma página da Web ou aplicativo móvel, cada um dos quais pode incluir elementos interativos. As tecnologias CSS, WebKit e jQuery são suportadas em arquivos PRO.

### Breve histórico do formato de arquivo PRO

Os arquivos PRO foram criados pelo Adobe Proto que foi substituído pelo Adobe XD. Ele foi usado para criar aplicativos de prototipagem para tablets. Aplicativos comuns de web design, como o Adobe Dreamweaver, usaram esses arquivos PRO para criar aplicativos web e móveis.

## Formato de arquivo PRO - Mais informações

Um arquivo PRO é o primeiro passo no desenvolvimento de um site ou aplicativo móvel. Os desenvolvedores primeiro criam maquetes digitais que definem a aparência do aplicativo. Eles também definem o fluxo de trabalho geral do aplicativo e a navegação pelas telas do aplicativo. Esses mockups também incluem vários elementos de navegação, como botões, caixas de combinação, barras de rolagem, campos de entrada e vários outros.
