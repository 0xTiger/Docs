## O que é um arquivo XHT?

Um arquivo com extensão .xht é um arquivo da Web associado ao tipo de arquivo [XHTML](/pt/web/xhtml/) (Extended Hypertext Markup Language), que é um arquivo de marcação baseado em [XML](/pt/web/xml/). Ambos os arquivos são semelhantes ao HTML, mas têm uma sintaxe semelhante a XML mais rigorosa. Os arquivos XHT são projetados de forma a serem mais estruturados, envolvem menos scripts e são genéricos, além de usar os recursos existentes do XML.

## Formato de arquivo XHT

Um arquivo XHT é criado e armazenado como um arquivo de texto simples com codificação UTF-8 por padrão. Ele contém o código fonte completo de um documento XHTML que pode ser aberto e editado com qualquer editor de texto que suporte a codificação Unicode. Além dos editores de texto, o formato de arquivo XHT é compreensível pela maioria dos navegadores modernos e pode ser aberto usando-os.

## Exemplo de XHT

O exemplo a seguir de um documento XHT define as declarações XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
