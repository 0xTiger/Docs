## O que é um arquivo .OBI?

O arquivo OBI é um arquivo de assinatura RSS do Outlook que permite aos usuários assinar um feed RSS no Microsoft Outlook. RSS significa Really Simple Syndication, que é um formato de feed da web usado para publicar conteúdo atualizado com frequência, como postagens de blogs, manchetes de notícias ou podcasts.

Para assinar o feed RSS no Outlook, os usuários podem clicar no link do feed RSS em um site ou copiar o URL do feed RSS e colá-lo na caixa de diálogo "Adicionar novo feed RSS" no Outlook. O Outlook verificará periodicamente o feed RSS em busca de novo conteúdo e o exibirá na pasta RSS Feed do usuário.

Os arquivos de assinatura RSS do Outlook têm a extensão de arquivo ".rss". Esses arquivos contêm a URL do feed RSS e podem ser usados para compartilhar assinaturas de feeds RSS com outras pessoas ou para fazer backup e restaurar assinaturas de feeds RSS.

## Formato de arquivo OBI – Mais informações

Os arquivos OBI são conhecidos como arquivo OPML (Outline Processor Markup Language) e contêm uma lista de URLs de feed RSS que foram assinados no Microsoft Outlook.

Quando o usuário exporta seus feeds RSS para o arquivo OBI, ele contém uma lista de todos os feeds RSS que ele assinou, incluindo o título de cada feed, a URL do feed e qualquer outra informação relevante. Isso permite que os usuários transfiram facilmente suas assinaturas de feeds RSS para outro leitor RSS ou façam backup de suas assinaturas por segurança.

Os arquivos OBI também podem ser usados para importar assinaturas de feeds RSS para o Microsoft Outlook ou outro leitor RSS. Para importar o arquivo OBI para o Outlook, os usuários podem clicar em "Arquivo" > "Abrir e Exportar" > "Importar/Exportar" > "Importar de outro programa ou arquivo" > "Avançar" > "Importar feeds RSS de um arquivo OPML" e em seguida, navegue até o local do arquivo OBI em seu computador.

## Qual é o formato do arquivo OBI?

O arquivo de assinatura RSS do Outlook, também conhecido como arquivo OBI, usa XML (eXtensible Markup Language) para definir a estrutura e o conteúdo do arquivo.

Os arquivos OBI são compostos de séries de elementos aninhados, cada um com seu próprio conjunto de atributos e valores. O elemento principal no arquivo OBI é o elemento “outline” que contém informações sobre o feed RSS, incluindo o título do feed, a URL do feed e qualquer outra informação relevante.

Cada elemento "contorno" também pode conter elementos filhos que representam subpastas ou subcategorias na lista de feeds RSS. A estrutura do arquivo OBI permite fácil organização e gerenciamento de assinaturas de feeds RSS e pode ser usado para transferir assinaturas entre diferentes leitores RSS ou para fazer backup e restaurar assinaturas.
