## O que é um arquivo FDF?

Um arquivo FDF (**Forms Data Format**) é um documento de texto gerado pela exportação de dados dos campos de formulário de um arquivo [PDF](/pt/pdf/). Inclui apenas dados de campos de texto extraídos dos campos de formulário disponíveis em um arquivo PDF. Isso resulta em um arquivo de dados comparativamente pequeno, pois os dados exportados não contêm o próprio formulário. O Adobe Acrobat fornece o recurso de exportação de dados de campos de formulário usando a opção 'Exportar dados de formulários' no menu do aplicativo.

## Formato de arquivo FDF - Mais informações

FDF é um formato de texto simples e está incluído como parte do [padrão ISO 32000](https://www.iso.org/standard/51502.html) para Portable Document Format. Foi desenvolvido pela Adobe para permitir a importação e exportação de dados do Acrobat Forms, ou AcroForms.

Existem dois tipos de arquivos FDF:

• `Clássico FDF` – Fornece dados para preenchimento de um formulário estático existente.

• `Modelo FDF` – Constrói um novo PDF com base em modelos de dentro de arquivos PDF especificados. Os formulários dentro do novo documento são preenchidos com o fornecimento de dados.

## Criando FDF usando Adobe Acrobat

O [kit de ferramentas FDF](https://opensource.adobe.com/dc-acrobat-sdk-docs/) da Adobe permite criar arquivos FDF a partir de dados de texto.
