## O que é um arquivo ML?

Um arquivo ML é um arquivo de texto criado com a linguagem de programação funcional de propósito geral ML (geralmente conhecida como Meta Language). Ele contém código-fonte de ML que é mais frequentemente usado em pesquisa e design de linguagem de programação usando semântica completamente definida. Existem várias variações de ML com Caml como o amplamente utilizado. Outros incluem Objective Caml (OCaml), JoCaml, Extended ML e MacroML. A linguagem ML é comumente usada para escrita de compiladores, prova automatizada de teoremas e verificações formais.
