## O que é um arquivo AS2PROJ?

Um arquivo AS2PROJ é um arquivo de projeto criado pelo IDE de programação Flash de código aberto, FlashDevelop. Ele contém informações relacionadas ao projeto, como configurações de compilação, referências aos arquivos de origem do projeto e opções para gerar a saída [arquivo .swf](/pt/page-description-language/swf/). Cada novo projeto criado no FlashDevelop é salvo como um arquivo AS2PROJ. Você pode alterar as configurações do seu projeto e todas essas alterações são salvas no arquivo AS2PROJ.

Os arquivos AS2PROJ podem ser abertos com o FlashDevelop.

## Formato de arquivo AS2PROJ

Um arquivo AS2PROJ é salvo no formato de arquivo [XML](/pt/web/xml/) que é um formato de arquivo universal para troca de informações. Ele pode ser aberto em qualquer editor de texto como Notepad, Notepad++ e Atom para examinar o conteúdo.
