## O que é um arquivo AU3?

Um arquivo AU3 é um arquivo de script automatizado criado com o programa de script freeware, [AutoIt v3](https://www.autoitscript.com/site/). Ele contém a linguagem de script AutoIt3 para automatizar comandos no Windows. A linguagem de programação AutoIt3 permite simular vários eventos que podem ser usados para automatizar tarefas. Por exemplo, o script dentro do arquivo AU3 pode ser usado para simular pressionamentos de tecla, movimento do mouse, controlar a posição das janelas e outros propósitos de script.

Os aplicativos que podem abrir arquivos AU3 incluem [SciTE4AutoIt3](https://www.autoitscript.com/site/autoit-script-editor/downloads/) e AutoIt.

## Formato de arquivo AU3

Os arquivos AU3 são salvos como arquivos de texto e contêm a linguagem de script AutoIt3.
