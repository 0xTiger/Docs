## O que é um arquivo ASI? ##

Um arquivo ASI é um formato de arquivo variável usado pelo software Alpha Five que é usado para desenvolver aplicativos da Web. Ele armazena variáveis usadas pelo aplicativo da Web. Quando você cria um novo projeto de desenvolvedor no Alpha Five, o arquivo ASI também é criado e salvo junto com o projeto de desenvolvedor Alpha Five.

Arquivos ASI podem ser abertos com Alpha Software Alpha Anywhere.

## Formato de arquivo ASI

Os arquivos ASI são salvos no formato de arquivo proprietário Alpha Five. Eles são lidos e gravados pelo aplicativo como parte do projeto do desenvolvedor.
