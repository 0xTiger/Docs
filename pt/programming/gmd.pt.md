## O que é um arquivo .GMD?

Um arquivo GMD é um arquivo fonte de desenvolvimento de jogos gerado com a ferramenta de desenvolvimento de software Game Maker que é usada para criar jogos 2D. O arquivo GMD é armazenado como arquivo de texto e é um arquivo de projeto do jogo a ser desenvolvido. Game Maker está disponível para Windows, Mac, Linux, Android, iOS, HTML5, Xbox, PlayStation e Nintendo Switch. A ferramenta é amplamente utilizada por desenvolvedores independentes, estúdios profissionais e educadores em todo o mundo para a criação de jogos.

## Formato de arquivo GMD

Os arquivos GMD são salvos em formato de arquivo binário e os detalhes de seu formato de arquivo interno não estão disponíveis para referência do desenvolvedor.
