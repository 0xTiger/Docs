## O que é um arquivo SCRIPT?

Um arquivo SCRIPT é um arquivo de origem que contém script em qualquer linguagem de script. Isso o torna um arquivo de script generalizado que contém variáveis e funções. Como cada linguagem de script pode ser diferente, o script escrito dentro de um arquivo SCRIPT precisa ser executado usando o mecanismo de script apropriado.

## Formato de arquivo SCRIPT - Mais informações

Os arquivos SCRIPT são armazenados no disco em arquivos de texto simples. Eles podem ser abertos e editados com qualquer editor de texto, como Microsoft Notepad, Notepad++ e Apple TextEdit. Como a linguagem de script pode ser diferente, ela pode ser executada renomeando a extensão do arquivo para a extensão padrão da respectiva linguagem de script. Por exemplo, existem várias linguagens de script comuns, como JavaScript ([.JS](/pt/web/js/)), VBScript ([.VBS](/pt/programming/vbs/)), AppleScript (.SCPT), PHP ([ .PHP](/pt/programming/php/)) e ASP ([.ASP](/pt/web/asp/)).
