## O que é um arquivo GROOVY?

Um arquivo GROOVY é um arquivo de código-fonte escrito na linguagem de programação Groovy. O código escrito dentro de um arquivo GROOVY é semelhante à linguagem orientada a objetos [Java](/pt/programming/java/), que facilita o design e o desenvolvimento de aplicativos. Um arquivo GROOVY é compilado para o bytecode da Java virtual machine (JVM) e é compatível com outros códigos e bibliotecas Java.

## Formato de arquivo GROOVY - Mais informações

Os arquivos GROOVY contêm código-fonte escrito em sintaxe Groovy. Ele pode conter código que pode ser usado tanto como linguagem de programação quanto como linguagens de script para a Plataforma Java. A linguagem Groovy possui recursos semelhantes aos de Python, Ruby e Smalltalk.
