## O que é um arquivo RBXM?

Um arquivo RBXM é um formato de arquivo de modelo 3D usado no Roblox Studio IDE para armazenar objetos 3D do jogo Roblox multiplayer online. Ele contém objetos 3D criados para o jogo usando o IDE. Exemplos de objetos 3D em um arquivo RBXM incluem camisas, calças, chapéus, prédios, veículos, animais, aviões, carros ou plantas. Os arquivos RBMX podem ser convertidos para o [formato de arquivo OBJ 3D](/pt/3d/obj/) usando o Roblox IDE. O Roblox IDE pode ser usado para abrir arquivos RBMX.

Outro formato de arquivo Roblox popular é [RBXL](/pt/programming/rbxl/), que é um formato de arquivo de localização e contém a localização definida em palavra 3D dentro do jogo Roblox.

## Formato de arquivo RBMX - Mais informações

Os arquivos RBXM são criados e salvos em formato de arquivo binário e suas especificações de formato de arquivo interno não estão disponíveis publicamente. Isso é diferente do formato de arquivo RBXMX que é baseado no formato de arquivo [XML](/pt/web/xml/). Existem muitas ferramentas/repositórios disponíveis no Github que podem ser usados para [processar o Roblox RBXM](https://github.com/search?q=rbxm) arquivos.
