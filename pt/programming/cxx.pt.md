## O que é um arquivo CXX?

Um arquivo CXX é um arquivo de código-fonte, semelhante ao C++, para escrever aplicativos. Ele pode conter a implementação completa de um aplicativo inteiro ou pode ter uma implementação parcial do programa que pode ser incluída em um projeto com outros arquivos C++. Os arquivos CXX podem ser abertos em qualquer editor de texto e IDEs de programação populares, como Notepad++, Turbo C++ e Visual Studio,

## Formato de arquivo CXX

Os arquivos CXX são armazenados como arquivo de texto simples e contém código escrito em sintaxe C++. Os arquivos CXX também são comumente encontrados com a extensão [.cpp](/pt/programming/cpp/). Os arquivos CXX são considerados uma alternativa ao .cpp usado por alguns compiladores.
