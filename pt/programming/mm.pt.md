## O que é um arquivo MM?

Um arquivo MM é um arquivo de código-fonte que pode conter código de programação Objective-C e Objective-C++. Ele é usado no MacOS e se diferencia dos arquivos .M que contêm apenas código Objective-C. Quando o compilador vê a extensão .mm, ele consegue compilar as classes C++ presentes no arquivo. Como um único arquivo de código-fonte pode conter uma combinação das linguagens [C](/pt/programming/c/) e [C++](/pt/programming/cpp/), renomear os arquivos .m para a extensão .mm permite que os recursos C++ definam Os aplicativos que podem abrir arquivos .MM incluem Apple TextEdit, Microsoft Notepad, Notepad++ e Microsoft Wordpad.
