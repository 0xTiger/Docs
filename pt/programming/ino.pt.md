## O que é um arquivo .INO?

**Arquivo INO** está relacionado ao **Arduino Sketch** que é um programa escrito para placas Arduino usando a **linguagem de programação Arduino**. A linguagem de programação Arduino é baseada em Wiring; e é semelhante ao C/C++. Os esboços do Arduino são salvos com uma extensão de arquivo .ino e são usados para controlar as placas de circuito do Arduino.

## Formato de arquivo INO – Mais informações

A plataforma Arduino, projetada para prototipagem eletrônica, capacita os desenvolvedores a criar e implantar programas, chamados de esboços, em placas de circuito especializadas; esses esboços abrangem desde tarefas fundamentais, como acender um LED na placa de circuito, até operações complexas, como transmitir dados pela Internet; a criação de esboços do Arduino ocorre no **Arduino Integrated Development Environment (IDE)**, e esses esboços são armazenados com a extensão de arquivo .ino.

Na era anterior ao Arduino 1.0, os esboços eram salvos com uma extensão de arquivo .PDE pelo Arduino IDE. Embora o IDE ainda suporte a abertura de arquivos .PDE, a prática recomendada é salvar todos os esboços, incluindo aqueles originalmente armazenados como arquivos [.PDE](/programming/pde/), com a extensão de arquivo .ino. Esta mudança reflete a evolução da plataforma e garante a compatibilidade com os padrões atuais.

## Sobre o Arduino

Arduino é uma plataforma eletrônica de código aberto que simplifica a prototipagem e o desenvolvimento de projetos interativos; compreendendo elementos de hardware e software, o Arduino atende a uma base diversificada de usuários, de novatos a profissionais. O núcleo da plataforma são as placas Arduino, equipadas com pinos digitais e analógicos, conectividade USB e diversos modelos baseados em processadores como Atmel AVR ou ARM.

O Arduino IDE serve como uma interface amigável para codificação, compilação e upload de programas (esboços) para as placas; a linguagem de programação, semelhante ao C/C++, é projetada para acessibilidade, abstraindo complexidades para iniciantes. As bibliotecas Arduino fornecem trechos de código pré-escritos para tarefas que vão desde integração de sensores até protocolos de comunicação, facilitando o desenvolvimento eficiente.

Shields Arduino, placas adicionais, expandem funcionalidades, oferecendo recursos como conectividade Ethernet ou controle de motor. A vibrante comunidade Arduino contribui para uma extensa base de conhecimento através de fóruns, tutoriais e projetos colaborativos; com aplicações que abrangem robótica, automação residencial, instalações artísticas e sistemas baseados em sensores, a versatilidade e simplicidade do Arduino o tornam a escolha preferida para diversos projetos eletrônicos.

## Como abrir o arquivo INO?

**Arduino Integrated Development Environment** pode ser usado para abrir o arquivo INO. Mas como são arquivos de texto simples, qualquer editor de texto pode ser usado para abrir ou editar seu conteúdo.
