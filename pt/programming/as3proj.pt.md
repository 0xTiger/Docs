## O que é um arquivo AS3PROJ?

Um arquivo AS3PROJ é um arquivo de projeto criado pelo FlashDevelop, um IDE de programação de código aberto gratuito para o desenvolvimento de aplicativos Flash e Web. Ele salva todas as informações e configurações relacionadas ao projeto aplicadas pelo usuário usando o FlashDevelop IDE. As informações armazenadas em um arquivo AS3PROJ incluem configurações de compilação, referências a arquivos de origem do projeto e opções para gerar a saída [arquivo .swf](/pt/page-description-language/swf/). Quando um novo projeto é criado, ele é salvo como arquivo AS3PROJ no mesmo local especificado pelo usuário para salvar os dados relacionados ao projeto.

## Formato de arquivo AS3PROJ

Um arquivo AS3PROJ é salvo no formato de arquivo [XML](/pt/web/xml/) que é um formato de arquivo universal para troca de informações. Ele pode ser aberto em qualquer editor de texto como Notepad, Notepad++ e Atom para examinar o conteúdo.
