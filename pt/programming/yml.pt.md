## O que é um arquivo YML?

YML é um arquivo de texto em linguagem de serialização de dados, [YAML](/pt/programming/yaml/) (Yet Another Markup Language), que é frequentemente usado para criar arquivos de configuração. Ele pode funcionar em combinação com qualquer linguagem de programação, incluindo C/C++, Ruby, Python, Java, PHP, C# e outras. Sendo independente de qualquer linguagem de programação específica, é aberto, interoperável e legível por humanos. Os arquivos YML podem ser abertos e editados com editores de texto comuns, como Microsoft Notepad, Notepad++, Github Atom, Apple TextEdit e Visual Studio IDE.

## Formato de arquivo YML - Mais informações

Os arquivos YML armazenam as configurações do aplicativo como configuração. Sua estrutura básica é um mapa que também pode ser chamado de diretório ou hash. As entradas em um arquivo YML consistem em um par de chave e valor. Geralmente, os arquivos de configuração .yml são salvos como `configure.yml`, embora algumas linguagens de programação, como Ruby on Rails, salvem a configuração de conectividade do banco de dados em arquivos `database.yml`. [Especificações de formato de arquivo YML/YAML](https://yaml.org/spec/1.2.2/) explicam as estruturas completas da linguagem e as informações relacionadas à sintaxe.
