## O que é um arquivo LGO?

Um arquivo LGO um código de programa escrito com o aplicativo Logo para desenhar imagens. Ele contém instruções programadas que orientam o aplicativo de leitura a fazer desenhos de acordo com essas instruções. Por exemplo, o programa "tartaruga" que pode abrir arquivos LGO, use estas instruções para desenhar linhas de acordo com o código programado. O desenho não se limita apenas a linhas. Os arquivos LGO podem ser usados para desenhar projetos simples a complexos. Muitos programas educacionais usavam arquivos LGO para ensinar como fazer desenhos programaticamente.

## Formato de arquivo LGO

Os arquivos LGO são salvos em formato de arquivo proprietário e sua estrutura interna de arquivos não é conhecida publicamente. Outro formato de arquivo bem conhecido para logotipos são os arquivos [.ico](/pt/image/ico/).
