## O que é um arquivo IN?

Um arquivo IN é um arquivo de entrada que o GNU Autoconf usa ao construir um aplicativo. Quando o processo de configuração é executado, um script Autoconf (arquivo .AC) pode fazer referência a um ou mais arquivos ".in" ou ".h.in". Os arquivos IN definem variáveis que são referenciadas durante a instalação do programa, como o número da versão de um aplicativo e a data de lançamento. Os arquivos IN também podem ser usados para determinar a presença ou ausência de recursos em uma instalação específica. Aplicativos como Notepad e Notepad++ podem ser usados para abrir e editar arquivos IN.

## Formato de arquivo IN - Mais informações

O Autoconf é um componente do sistema de compilação GNU que gera compilações personalizadas de aplicativos para várias plataformas. É uma das várias ferramentas de construção GNU. Outras ferramentas incluem Automake, Gnulib e Libtool. Sem a necessidade de intervenção manual do usuário, esses scripts podem adaptar os pacotes a uma ampla variedade de sistemas do tipo UNIX. O Autoconf gera um script de configuração do pacote a partir de um arquivo de modelo que lista os recursos do sistema operacional que o pacote pode usar na forma de chamadas de macro M4
