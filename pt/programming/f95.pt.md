## O que é um arquivo .F95?

Um arquivo com extensão .f95 normalmente indica que é um arquivo de código-fonte Fortran 95. Fortran é uma linguagem de programação imperativa de uso geral que é particularmente adequada para computação numérica e científica. Fortran 95 é uma versão aprimorada do Fortran 90, introduzindo vários novos recursos e melhorias.

Os arquivos de código-fonte Fortran normalmente contêm instruções de programa escritas na linguagem de programação Fortran; se você tiver um arquivo com extensão .f95, provavelmente ele deve ser compilado usando um compilador Fortran; compilação é o processo de tradução de código-fonte legível por humanos em código binário legível por máquina que pode ser executado por um computador.

## Como abrir o arquivo F95?

Se quiser trabalhar ou executar programas em arquivo .f95, você precisará de um compilador Fortran; compiladores Fortran populares incluem GNU Fortran (gfortran), Intel Fortran Compiler (ifort) e outros. Você pode compilar e executar programas Fortran usando comandos apropriados fornecidos pelo compilador.

Por exemplo, se você tiver o gfortran instalado, poderá usar os seguintes comandos no terminal para compilar e executar um programa Fortran:

```
# Compile the Fortran program
gfortran -o my_program my_program.f95

# Run the compiled program
./my_program
```

Se você deseja apenas visualizar o conteúdo do arquivo F95, ele é um arquivo de texto simples, portanto você pode abri-lo com qualquer editor de texto.
