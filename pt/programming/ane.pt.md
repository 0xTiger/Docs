## O que é um arquivo ANE?

Um arquivo ANE é um arquivo Adobe Air Native Extension usado pelos desenvolvedores para escrever funções específicas que invocam o código do [Adobe ActionScript](https://en.wikipedia.org/wiki/ActionScript). Na maioria dos casos, os arquivos ANE contêm a biblioteca de código nativa usada pela extensão. No entanto, o próprio arquivo é instalado no dispositivo. Permite funções específicas do sistema, como vibração do dispositivo. Os aplicativos Adobe AIR devem ser compilados para usar vinculação dinâmica para utilizar o arquivo ANE.

## Formato de arquivo ANE - Mais informações

Os arquivos ANE são salvos em disco como arquivos binários cujas especificações de formato de arquivo não estão disponíveis publicamente. Um arquivo ANE pode conter código-fonte Java Android nativo para acessar a funcionalidade do dispositivo Android, como invocar funções de volume ou vibração. Tudo isso é feito envolvendo o sistema nativo na linguagem ActionScript da Adobe.
