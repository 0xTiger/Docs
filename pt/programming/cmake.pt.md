## O que é um arquivo CMAKE?

Um arquivo CMAKE é um arquivo de projeto criado pela ferramenta de plataforma cruzada de código aberto **CMake**. Ele contém script de origem na linguagem CMake para todo o programa. A ferramenta CMake usa scripts de programação, conhecidos como CMakeLists, para gerar arquivos de compilação para um ambiente específico, como makefiles da máquina Unix. Scripts e módulos usados pelo projeto CMake são organizados como script.cmake e module.cmake que são referenciados no projeto.
