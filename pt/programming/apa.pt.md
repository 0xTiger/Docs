## O que é um arquivo APA?

Um arquivo com extensão .apa é um arquivo de projeto de desenvolvimento de software criado pelo aplicativo RSView. Ele contém as informações necessárias para a criação de aplicativos de interface homem-máquina (IHM). Os arquivos APA são armazenados em formato de arquivo morto. Seu formato de arquivo de arquivo ajuda a transferi-los de um computador para outro facilmente devido ao tamanho menor do arquivo. Utilitários de descompressão padrão podem ser usados para extrair o conteúdo de arquivos APA. Os arquivos APA podem ser abertos usando o software Rockwell Automation RSView Studio ou RSView ME.

## Formato de arquivo APA - Mais informações

O gerenciador de aplicativos RSView salva os arquivos .apa por padrão no local C:\Documents and Settings\All Users\Documents\RSView Enterprise\ME\Archives no Windows XP. No entanto, você pode salvar o arquivo em qualquer diretório conforme o requisito.
