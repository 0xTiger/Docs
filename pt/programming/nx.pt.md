## O que é um arquivo .NX?

Um arquivo NX é um arquivo de videogame criado para ser usado com o console de jogo virtual, **LowRes NX**. Está escrito na linguagem de programação BASIC e contém o código fonte do jogo que define como o jogo funciona. Você pode criar seus próprios arquivos NX para usar com o aplicativo que pode ser carregado e os usuários podem jogar.

Você pode abrir um arquivo NX em um editor de texto, LowRes NX ou qualquer editor de código-fonte, como o Microsoft Visual Studio Code.

## Formato de arquivo NX

Um arquivo NX é salvo no disco como um arquivo de texto simples que contém as instruções do jogo. Cada arquivo NX possui o código-fonte do jogo que compreende várias entradas de ROM de cartucho. Essas entradas de ROM contêm informações dos componentes do jogo, como gráficos, mapas de níveis, áudio e outros dados que não são do código-fonte do jogo. As entradas ROM de um programa podem ser adicionadas ou editadas usando programas de edição NX que acompanham o LowRes NX.

## Como criar arquivos NX?

### Crie arquivos NX no PC

Você pode criar arquivos NX no PC usando:

* Editores de texto - como Microsoft Notepad, Notepad++ ou Github Atom
* Editores de código-fonte - como Microsoft VIsual Studio Code

### Crie arquivos NX no iOS

LowRes NX possui versão iOS que pode ser usada por desenvolvedores para criar arquivos NX a partir deste aplicativo.
