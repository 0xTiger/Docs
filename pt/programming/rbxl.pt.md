## O que é um arquivo RBXL?

Um arquivo RBXL é um arquivo de localização Roblox que contém a localização definida no mundo 3D dentro do [jogo Roblox](https://www.roblox.com/). Roblox em si é um jogo de construção multiplayer que permite aos jogadores construir jogos usando as ferramentas de desenvolvedor fornecidas pela plataforma. Os arquivos RBXL usam ferramentas, scripts, equipes e peças para descrever o campo de jogo como modelos 3D. O Roblox Studio permite que o jogador crie os itens do jogo e pode ser usado para abrir arquivos RBXL.

## Formato de arquivo RBXL

Os arquivos RBXL são desenvolvidos usando o Roblox Studio e salvos no formato de arquivo [XML](/pt/web/xml/). Devido a esse formato de arquivo universal, existem muitas ferramentas e APIs que permitem criar e abrir esses arquivos. Uma dessas APIs é a [MaximumADHD](https://github.com/MaximumADHD/Roblox-File-Format), uma biblioteca [C#](/pt/programming/cs/) para criar e manipular arquivos no formato de arquivo model/place do Roblox .
