## O que é um arquivo ASVF?

Um arquivo ASVF é um arquivo compactado criado pelo software Asphyre Sphinx, que é uma estrutura 2D/3D para aplicações industriais e empresariais. Ele armazena dados como fontes, imagens e outras mídias. Sphinx também pode ser usado para desenvolver videogames 2D/3D. Um arquivo ASVF compactado pode conter arquivos comuns, como arquivos .JPEG e .XML.

Os arquivos ASVF podem ser abertos com o software Asphyre Sphinx que foi descontinuado há muito tempo.

## Formato de arquivo ASVF

Os arquivos ASVF são salvos como arquivos compactados e podem ter vários arquivos dentro do arquivo.
