## O que é um arquivo PHP?
Um arquivo com extensão .php refere-se à linguagem de programação de código aberto, usada para escrever scripts do lado do servidor, a serem executados em um servidor web. É a linguagem de script da Web mais amplamente usada, normalmente usada para o desenvolvimento de aplicativos da Web em grande escala. O maior sistema de blogs na web, ou seja, WordPress e a maior rede social Facebook são baseados na linguagem PHP. É compatível com várias plataformas, como Windows, Linux, Unix, Mac OS X, etc. Também é executado com sucesso em quase todos os servidores usados hoje, ou seja, Apache, IIS, etc. PHP suporta uma ampla gama de bancos de dados, incluindo MySQL.

## Formato de arquivo PHP ##

Um único PHP pode conter texto, CSS, [HTML](/pt/web/html/), JavaScript e código PHP. O código PHP é executado no servidor e o resultado é retornado ao navegador como HTML simples. Arquivos PHP podem ser abertos com qualquer editor de texto e editados no local, embora aplicativos como Adobe Dreamweaver, ferramentas Eclipse PHP Development ofereçam uma maneira conveniente de escrever e modificar código PHP.

### Sintaxe ###

Um script PHP é executado no servidor e o resultado HTML simples é enviado de volta ao navegador. Um script PHP começa com<?php and ends with ?>

### Exemplo PHP ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
