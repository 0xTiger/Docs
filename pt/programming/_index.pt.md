Arquivos de programação são arquivos de código-fonte que são usados por linguagens de programação para escrever aplicativos de software para executar determinadas tarefas. Cada linguagem de programação tem sua própria sintaxe e carrega seu próprio conjunto de arquivos para escrever código na forma de bibliotecas, arquivos de cabeçalho, classes ou constantes. Com base na linguagem de programação utilizada, o formato do arquivo do código-fonte será diferente para obedecer à regra de sintaxe definida por aquela linguagem.

As extensões comuns de arquivo de programação ou desenvolvedor e seus formatos de arquivo associados incluem **[C#](/pt/programming/cs/)** (CSharp Source Code File), **[CPP](/pt/programming/cpp/)** (C++ Source Code File) File) e **[VB](/pt/programming/vb/)** (Arquivo de código-fonte do Visual Basic).
