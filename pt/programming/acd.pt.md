## O que é um arquivo ACD?

Um arquivo ACD é um arquivo de projeto que contém informações para programação das plataformas Logix [Rockwell Automation](https://www.rockwellautomation.com). Inclui instruções para controladores e processadores. Ele é usado para plataformas Logix como ControlLogix, CompactLogix, FlexLogix, SoftLogix5800 e sistemas DriveLogix.

Os aplicativos que podem abrir arquivos ACD incluem Rockwell Automation RSLogix 5000.
