## O que é um arquivo ALB?

Um arquivo ALB é um arquivo de biblioteca usado pelo aplicativo IDE **Alpha Five** usado para desenvolver aplicativos da Web. Seu objetivo principal é armazenar informações para o Alpha Five correspondente [arquivo ADB](/pt/database/adb/). Isso inclui informações do dicionário de dados, como configurações, ponteiros de script, menus, funções definidas pelo usuário, bitmaps, barras de ferramentas e definições de importação.

Os arquivos ALB podem ser abertos com o **Alpha Software Alpha Anywhere**.

## Formato de arquivo ALB - Mais informações

Os arquivos ALB são salvos no formato de arquivo proprietário do Alpha File e sua estrutura de formato de arquivo interno não está disponível publicamente. Eles são salvos além do banco de dados geral da biblioteca do desenvolvedor Alpha Five. Sendo bibliotecas, os arquivos ALB podem ser reutilizados em outros aplicativos Alpha Five.
