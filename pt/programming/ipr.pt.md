## O que é um arquivo IPR?

O arquivo IPR é um arquivo de projeto criado com o Java IDE **IntelliJ IDEA**. Todas as informações relacionadas às configurações do projeto, como estrutura de diretórios, arquivos de código-fonte, bibliotecas incluídas no projeto e configurações do compilador, são armazenadas nos arquivos IPR. Quando um projeto é aberto a partir do IntelliJ IDEA, o IDE lê as informações no arquivo IPR e carrega os arquivos e configurações correspondentes. Os arquivos IPR foram substituídos pelos arquivos .IDEA que contêm todas as informações como Diretório de configurações.

## Formato de arquivo IPR

Os arquivos IPR são salvos no formato de arquivo [XML](/pt/web/xml/) universalmente compreensível que é legível por humanos. Ele contém todas as informações relacionadas ao projeto em tags que são carregadas para exibir o conteúdo relacionado ao projeto.

Como os arquivos IPR são armazenados no formato de arquivo XML, eles podem ser abertos em qualquer editor de texto, como Notepad, Notepad++, Atom e Apple TextEdit.
