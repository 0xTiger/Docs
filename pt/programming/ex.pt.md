## O que é um arquivo EX?

Um arquivo EX é um arquivo de código-fonte que contém código-fonte para aplicativos desenvolvidos usando a linguagem de programação Euphoria. Euphoria é uma linguagem de programação procedural que acredita na simplicidade de programação em vez de linguagens de programação complexas. Um aplicativo desenvolvido no Euphoria pode conter vários arquivos EX, mas um único arquivo EX também pode conter o código-fonte completo do programa.

Os aplicativos que podem **abrir arquivos EX** incluem openEuphoria Euphoria, Microsoft Notepad, Notepad++ e Atom.

## Breve História do Formato de Arquivo EX

O desenvolvimento e a existência do formato de arquivo EX estão ligados ao desenvolvimento da linguagem de programação Euphoria que foi criada por Robert Craig. Foi lançado pela primeira vez para MS-DOS. Tornou-se software de código aberto em 2006 com o lançamento da versão 3. Seu desenvolvimento foi assumido pelo **grupo openEuphoria** que administra e desenvolve o projeto. Sua versão 4 foi lançada em dezembro de 2010 e está disponível para Windows, Linux, macOS e três versões de BSD.

A Euphoria foi inicialmente escrita na linguagem de programação **[C](/pt/programming/c/)**. Mais tarde, o intérprete Euphoria foi dividido em duas partes:

* `Front-end Parser` - escrito em Euphoria e é usado com o tradutor Euphoria-to-C e o Binder.
* `Back-end` - biblioteca de tempo de execução escrita em linguagem de programação C.

## Formato de arquivo EX - Mais informações

Os arquivos EX são armazenados como arquivos de texto simples e contêm o código-fonte para programas escritos na linguagem de programação Euphoria. Esses arquivos podem ser abertos em qualquer editor de texto.
