## O que é um arquivo PUZ?

Um arquivo com extensão .puz é um arquivo compactado que contém um documento padrão do Microsoft Publisher [.pub](/pt/publisher/pub/). Como resultado da compressão, é menor em tamanho e fácil de transferir via internet ou outros dispositivos. Os arquivos PUZ podem ser descompactados/abertos usando o utilitário Microsoft Unpack.

## Formato de arquivo PUZ

Um arquivo PUZ é um arquivo compactado que contém o arquivo PUB padrão junto com outros arquivos. Ele contém todos os arquivos necessários para descompactar, exibir e imprimir a publicação.

Um arquivo PUZ contém o seguinte.

* `Unpack.exe` - um utilitário de software para descompactar arquivos compactados
* `ReadMe.txt` - arquivos de descrição
* `Arquivo .puz compactado` - Contém a publicação e pode incluir fontes incorporadas e gráficos vinculados.

### Descompactando o arquivo PUZ

Os arquivos compactados do Publisher podem ser descompactados usando o utilitário Unpack.exe. Ele extrai o arquivo PUZ compactado para a escolha do diretório selecionado pelo usuário.
