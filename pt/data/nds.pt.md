## O que é um arquivo .NDS?

A extensão do arquivo .NDS é comumente associada ao **Arquivo de compilação Nero DiscSpan**. O Nero DiscSpan é um recurso do **Nero Burning ROM**, um popular programa de criação de discos ópticos; esse recurso permite aos usuários distribuir grandes quantidades de dados em vários discos.

Ao criar uma compilação DiscSpan no Nero Burning ROM, os dados são divididos em segmentos menores que cabem em discos individuais; o arquivo .nds é criado como um arquivo descritor que contém informações sobre layout e dados distribuídos entre discos.

Se você tiver um arquivo .nds e segmentos de disco correspondentes, poderá usar o Nero Burning ROM ou o Nero DiscSpan para restaurar os dados originais. O Nero DiscSpan solicitará que você insira cada disco em sequência para reconstruir a compilação completa.
