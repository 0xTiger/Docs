## O que é um arquivo FIG?

Um arquivo FIG é um arquivo especial que contém uma imagem ou gráfico feito usando o programa MATLAB, que é usado para fazer contas e analisar dados. Este arquivo contém informações sobre imagens ou gráficos que o MATLAB desenha para ajudar as pessoas a ver e compreender melhor os dados. É como um contêiner que guarda todos os detalhes dos gráficos.

.FIG é o formato de arquivo de figura padrão do MATLAB. Ele armazena a figura completa, incluindo todos os eixos, rótulos e configurações. Para salvar uma figura neste formato, você pode usar a função `savefig`:

## Como criar um arquivo FIG com MATLAB?

Para criar um arquivo FIG usando MATLAB, siga estas etapas:

1. **Selecione seus dados:** Escolha a variável para a qual deseja criar um gráfico no painel "Espaço de trabalho" no MATLAB.

2. **Escolha o tipo de plotagem:** Vá até a aba "PLOTS" e clique no tipo de plotagem desejada.

3. **Salve o gráfico como arquivo FIG:**

     - Vá ao menu "Arquivo".
     - Escolha "Salvar" ou "Salvar como".
     - Forneça um nome para o seu arquivo e decida onde deseja salvá-lo.
     - Clique em "Salvar" para criar seu arquivo FIG.

## Como abrir o arquivo FIG?

Para abrir o arquivo FIG no MATLAB, siga estas etapas:

1. **Abra o MATLAB:** Inicie o MATLAB no seu computador.

2. **Navegue até a aba "PARCELAS":** Clique na plotagem da aba "PARCELAS", selecionando o tipo de plotagem desejada.

3. **Abra o arquivo FIG:**

     - Vá ao menu "Arquivo".
     - Escolha "Abrir".
     - Navegue até o local onde seu arquivo FIG foi salvo, selecione-o e clique em “Abrir”.
