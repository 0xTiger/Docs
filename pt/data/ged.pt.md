## O que é um arquivo .GED?

Um **arquivo GEDCOM (Comunicação de Dados Genealógicos)**, geralmente com extensão .ged, é um formato de arquivo padrão usado para troca de dados genealógicos entre diferentes programas de software de genealogia; Os arquivos GEDCOM são arquivos de texto simples que contêm informações sobre indivíduos, famílias, eventos e outros dados genealógicos; o objetivo do GEDCOM é fornecer um formato comum para a troca de informações genealógicas.

Os arquivos GEDCOM geralmente têm extensão de arquivo .ged; esta extensão é usada para indicar que o arquivo segue o **padrão GEDCOM** para troca de dados genealógicos; portanto, quando você salva ou exporta seus dados genealógicos de um programa de software de genealogia, o arquivo resultante geralmente recebe um nome como `seunomedoarquivo.ged`.

Por exemplo:

-suaárvorefamiliar.ged
- john_doe_family.ged
-ancestry_data.ged

## Pontos principais sobre arquivos GEDCOM

Aqui estão alguns pontos importantes sobre os arquivos GEDCOM:

1.  **Formato de texto simples:** Os arquivos GEDCOM são arquivos de texto simples; o que significa que podem ser abertos e visualizados com qualquer editor de texto; este formato facilita o compartilhamento e a compatibilidade entre diferentes aplicativos de software genealógico.
    
2.  **Estrutura:** Os arquivos GEDCOM possuem estrutura hierárquica; os dados são organizados em registros; cada registro possui tag específica que indica o tipo de informação que contém, por exemplo, o nome de um indivíduo pode ser representado pela tag NOME.
    
3.  **Individuals and Families:** GEDCOM files typically include information about individuals such as names, dates of birth and places of birth; they also include details about family relationships, marriages and other events.
    
4.  **Interoperabilidade de software:** Como o GEDCOM é um formato padronizado; permite aos usuários mover seus dados genealógicos entre diferentes programas de genealogia; isso é particularmente útil para usuários que podem mudar para um aplicativo de software diferente ou compartilhar suas pesquisas com outras pessoas.
    
5.  **Versão:** O GEDCOM passou por várias versões, cada versão introduzindo melhorias e novos recursos; a versão mais usada foi GEDCOM 5.5. 

## Como abrir o arquivo GED?

Para abrir um arquivo GED, geralmente você precisa de um software de genealogia que suporte o formato GEDCOM. Algumas opções populares de software de genealogia incluem:

- **Criador de Árvore Genealógica**
- **Ancestry.com**
- **Minha herança**
- **Vovô** (gratuito e de código aberto)
- **Árvore genealógica legada**

Depois de escolher o software de genealogia, procure uma opção como Importar”, Abrir GEDCOM” ou Importar do arquivo” no menu ou na barra de ferramentas.

Como os arquivos GED são salvos em formato de texto, você também pode abrir os arquivos com um editor de texto simples, por exemplo

- **Bloco de anotações**
- **Bloco de notas++**
- **Apple TextEdit**
- Qualquer editor de texto
