## O que é um arquivo .ENC?

Um **arquivo ENC** está relacionado à **Carta de Navegação Eletrônica (ENC)**, que é uma carta vetorial digital usada em sistemas de cartas eletrônicas para navegação marítima; Os arquivos ENC contêm dados geoespaciais, como contornos de profundidade, auxílios à navegação e outras informações necessárias para uma navegação segura. ENC é um formato padronizado definido pela **Organização Hidrográfica Internacional (IHO)** no **padrão S-57**.

O padrão S-57 especifica a estrutura e o conteúdo dos arquivos ENC; esses arquivos normalmente têm extensão de arquivo .enc. Os arquivos ENC são usados por **Electronic Chart Display and Information Systems (ECDIS)** em navios para navegação; Os sistemas ECDIS utilizam dados ENC para exibir cartas eletrônicas, fornecendo aos navegadores informações em tempo real para auxiliar na navegação segura e eficiente.

## Formato de arquivo ENC – Mais informações

**Arquivos ENC** são **arquivos binários** que armazenam informações de maneira estruturada, permitindo armazenamento e recuperação eficiente de dados de navegação; o conteúdo dos arquivos ENC inclui informações sobre costas, contornos de profundidade, bóias, luzes e outros recursos de navegação; os dados são organizados em várias camadas de dados e cada recurso é representado como um objeto vetorial com atributos associados.

Os arquivos ENC são atualizados regularmente pelos serviços hidrográficos para refletir as mudanças nas informações de navegação, garantindo que os navegantes tenham acesso aos dados mais atuais e precisos para as suas rotas.

## Sobre o padrão S-57

O **padrão S-57** é uma especificação internacional para codificação, armazenamento e troca de cartas de navegação eletrônicas (ENCs); foi desenvolvido pela Organização Hidrográfica Internacional (IHO) para fornecer um formato padronizado para cartas digitais usadas em Sistemas de Informação e Exibição de Cartas Eletrônicas (ECDIS) para navegação marítima.

## Como abrir o arquivo ENC?

Para abrir arquivos de Carta de Navegação Eletrônica (ENC), geralmente você precisa de um software de navegação especializado que suporte o padrão S-57 e o formato de arquivo ENC.

Alguns softwares de navegação populares incluem:

- OpenCPN
- MaxSea
- Mar Claro
- Transas iSailor
- TEMPO ZERO
