## O que é um arquivo .BRD?

Um formato de arquivo BRD é usado em software de automação de projeto eletrônico (EDA) para projeto de placa de circuito impresso (PCB). Softwares de design de PCB como Eagle, KiCad, Altium Designer e outros utilizam este formato de arquivo para armazenar layout, conexões e outras informações relacionadas ao design de uma placa de circuito impresso.

Um arquivo BRD é um tipo de arquivo CAD que são arquivos digitais usados por engenheiros e designers para criar, modificar e analisar projetos. O arquivo BRD está associado ao Autodesk EAGLE, que é um aplicativo de software comumente usado na indústria eletrônica para projetar placas de circuito impresso (PCBs). O Autodesk EAGLE fornece ferramentas para captura esquemática (criação de representação visual do circuito) e layout de PCB (organização de componentes e conexões de roteamento).

O arquivo BRD é criado usando o EAGLE Layout Editor, que é um componente do conjunto de software Autodesk EAGLE. Este editor permite aos usuários projetar o layout do PCB, incluindo a colocação de componentes, roteamento de traços, definição de regras de projeto e criação de arquivos necessários para fabricação.

Os arquivos BRD servem como modelos ou projetos para design de circuitos eletrônicos em PCB. Os projetistas usam arquivos EAGLE e .brd” para traduzir seus diagramas esquemáticos em layouts físicos que podem ser fabricados.

Os arquivos .BRD podem ser salvos no formato de dados de perfuração Gerber. Os arquivos Gerber são formatos padrão usados na fabricação de PCB para descrever padrões, camadas e recursos de design de PCB. Salvar arquivos .brd neste formato permite que eles sejam utilizados por programas de manufatura auxiliada por computador (CAM), que geram instruções necessárias para a fabricação de PCB.

## Visualizador de arquivos BRD

Existem diversas ferramentas de software disponíveis para visualização de arquivos .brd, permitindo aos usuários visualizar e inspecionar layouts de PCB criados com softwares como o Autodesk EAGLE.

1.  **Autodesk EAGLE**: Se você tiver acesso ao Autodesk EAGLE, poderá simplesmente abrir arquivos .brd diretamente no software para fins de visualização e edição.
    
2.  **KiCad**: KiCad é um conjunto de software EDA de código aberto que inclui editor de layout de PCB. Possui capacidade de importar e visualizar arquivos .brd criados com Autodesk EAGLE.
    
3.  **ViewPlot**: ViewPlot é um visualizador Gerber independente que suporta vários formatos de arquivo PCB, incluindo .brd. Ele permite que os usuários visualizem projetos de PCB sem a necessidade de software de design original.
    
4.  **GC-Prevue**: GC-Prevue é outro visualizador Gerber popular que também pode lidar com arquivos .brd. Ele fornece recursos para visualizar layouts de PCB, medir distâncias e inspecionar detalhes do projeto.
    
5.  **Gerbv**: Gerbv é um visualizador Gerber de código aberto que suporta a visualização de arquivos Gerber, que podem incluir arquivos .brd salvos no formato Gerber.
    
6.  **Ferramentas de visualização on-line**: também existem ferramentas on-line disponíveis que permitem fazer upload e visualizar arquivos .brd diretamente em seu navegador. Um exemplo é o Online Gerber Viewer da EasyEDA, que suporta a visualização de vários formatos de arquivo PCB, incluindo Gerber e .brd.

## Como abrir o arquivo BRD?

Os arquivos BRD que são usados no design de PCB podem ser abertos em vários aplicativos de design de PCB.

- **Autodesk EAGLE**: Este é um software de design de PCB de plataforma cruzada desenvolvido pela Autodesk. Ele suporta a criação de diagramas esquemáticos, layouts de PCB e roteamento de conexões elétricas. Os usuários podem abrir arquivos .brd diretamente no Autodesk EAGLE para visualização e edição posterior.
    
- **Altium Designer**: Altium Designer é um software abrangente de design de PCB principalmente para sistemas operacionais Windows. Ele oferece ampla gama de recursos para captura esquemática, layout de PCB e análise de projeto. Altium Designer também suporta a abertura de arquivos .brd”, permitindo aos usuários importar e trabalhar com designs criados em outros softwares.
    
- **Open Board Viewer**: Open Board Viewer é um software de visualização de PCB projetado especificamente para sistemas operacionais Linux. É uma ferramenta de código aberto que permite aos usuários visualizar layouts de PCB, inspecionar componentes e revisar detalhes de roteamento. O Open Board Viewer suporta a abertura de arquivos .brd, permitindo que usuários em plataformas Linux visualizem e analisem projetos criados em outros softwares.

Aqui está a lista de programas que você pode usar para abrir arquivos BRD.

- **Autodesk EAGLE** (avaliação gratuita) para (Windows, Mac, Linux)
- **Altium Designer** (avaliação gratuita) para (Windows, Mac, Linux)
- **Visualizador de quadro aberto** (gratuito) para Linux
