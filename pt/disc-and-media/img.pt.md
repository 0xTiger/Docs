## O que é um arquivo IMG?

No campo da computação, os arquivos com extensões de arquivo .img tratam do armazenamento de imagens brutas de discos como disquetes, discos rígidos ou discos ópticos, etc. Essa imagem bruta consiste em uma imagem binária da fonte com especificações de setor por setor.

O conteúdo do formato de arquivo IMG depende do sistema de arquivos do disco especificado para a criação da imagem. Quando se trata de CD ROMs e DVDs as imagens desses arquivos contêm dados dos setores correspondentes juntamente com os cabeçalhos de controle e campos para correção de erros de acordo com cada setor.

Os arquivos IMG têm conteúdo especificado para o disco, portanto, são compatíveis com os programas específicos adequados para a detecção de seus sistemas de arquivos.

## Formato de arquivo IMG ##

Durante os anos em que um disquete era usado para fins de armazenamento e gerenciamento de armazenamento de dados, esses arquivos foram desenvolvidos para acessar as imagens brutas do disco. Muitos programas foram referidos pelos arquivos IMG. Estes foram criados por programas de imagem de disco. Nos programas do sistema operacional Windows, como o power, um gravador de software é usado para lidar com esses arquivos.

## Especificação técnica ##

Existem outras extensões de arquivo como IMA e IMZ semelhantes a este formato de arquivo. O tamanho dessas imagens está sempre na forma de vários tamanhos de setor diferentes. Este tamanho é de 512 bytes para disquetes e discos rígidos. O tamanho bruto do setor é de 2.352 para os CD Roms e DVDs. O tamanho da imagem de disco bruta é múltiplo desse tamanho.

Esse formato é usado principalmente com programas como RawWrite e WinImage. Esses programas usam esse formato para ler e gravar as imagens de disquetes. Além disso, esses arquivos podem ser convertidos em outros formatos de arquivo mais populares usando algum software como o PowerISO.

Como os arquivos IMG não têm dados adicionados, exceto o conteúdo dos discos, esses arquivos IMG podem ser manipulados pelos programas automaticamente e detectar os sistemas de arquivos. Por exemplo, uma imagem de disco comum de um disquete bruto começa com o setor de inicialização do FAT e pode ser utilizada para identificar o sistema de arquivos.

As imagens de disco da mídia óptica são geralmente acompanhadas pelo arquivo descritor. Ele explica o layout do disco e envolve informações como limites de faixas e estas não são apenas armazenadas em um arquivo de imagem de forma bruta.
