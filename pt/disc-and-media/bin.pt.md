## O que é um arquivo BIN?

Um arquivo com extensão .bin pode ser um arquivo de imagem de disco, geralmente contém os dados binários armazenados em um disco óptico, como CD ou DVD. Um arquivo BIN é um arquivo único; contendo o conteúdo completo e estrutura; representando um meio ou dispositivo de armazenamento de dados, como uma unidade de fita, discos rígidos, disquete, CD, DVD, BD e unidade de chave. Os arquivos BIN podem ser salvos com outros arquivos de metadados; descrevendo as informações relacionadas ao disco.

## Formato de arquivo BIN

Embora a imagem do disco óptico salva com uma extensão .bin possa ser chamada de imagem de disco, geralmente criada copiando completamente a cópia setor por setor da mídia de origem e, assim, replicando perfeitamente a estrutura e o conteúdo de um dispositivo de armazenamento. A estrutura real dos dados dentro da imagem geralmente é algum sistema de arquivos.
Alguns softwares de imagem de disco reduzem o espaço de arquivo não utilizado da mídia de origem ou compactam o disco que representam para reduzir os requisitos de armazenamento. Embora sejam particularmente chamados de arquivos compactados, eles não são considerados imagens de disco.

## BIN VS ISO

O formato BIN foi normalmente desenvolvido para resolver o problema de arquivamento do CD de áudio com [ISO](/pt/compression/iso/). Normalmente, esse formato pode ser usado como uma alternativa para o mesmo conteúdo que a ISO suporta e sem nenhum inconveniente. ISO é um formato de imagem de disco simples, enquanto BIN/CUE é uma cópia bruta de um disco, incluindo proteção contra cópia, setor por setor, lista de faixas, correção de erros, várias faixas e qualquer informação específica do sistema no disco
