## O que é um arquivo WBFS?

Um arquivo WBFS (Wii Backup File System) é uma imagem de disco de videogame criada para o Nintendo Wii. Ele contém um jogo Wii que pode ser carregado como uma imagem para jogar. Ele é gerado usando a ferramenta Wimms WBPS ou o Wii Backup Manager e pode ser jogado em um Nintendo Wii carregando a partir de um USB Loader GX ou emulador de Wii, como o Dolphin. O jogo é armazenado nesses dispositivos de armazenamento externo usando o formato de arquivo WBFS. O Wii é uma plataforma de jogos distribuída pela Nintendo. O sistema de arquivos WBFS foi desenvolvido pelo codificador homebrew do Wii Waninkoko.

## Formato de arquivo WBFS - Mais informações

WBFS são arquivos binários que possuem formato de arquivo proprietário. Suas especificações de formato de arquivo não estão disponíveis publicamente.
