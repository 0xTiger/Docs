## O que é um arquivo .CUE?

Um arquivo **.CUE**, também conhecido como **CDRWIN Cue Sheet**, é um arquivo de texto simples que contém informações sobre faixas e layout de um CD de áudio ou imagem de disco, normalmente em formato BIN ou ISO. Esses arquivos são frequentemente usados para descrever a estrutura e o conteúdo do disco, permitindo que software de gravação de CD ou software de unidade virtual reproduza com precisão o disco original.

## Exemplo de arquivo CUE

Aqui está um exemplo da aparência do arquivo ".cue":

```
PERFORMER "Artist Name"
TITLE "Album Title"
FILE "DiscImage.bin" BINARY
  TRACK 01 AUDIO
    TITLE "Track 1 Title"
    PERFORMER "Artist Name"
    INDEX 01 00:00:00
  TRACK 02 AUDIO
    TITLE "Track 2 Title"
    PERFORMER "Artist Name"
    INDEX 01 03:45:21
  TRACK 03 AUDIO
    TITLE "Track 3 Title"
    PERFORMER "Artist Name"
    INDEX 01 07:28:17
  TRACK 04 AUDIO
    TITLE "Track 4 Title"
    PERFORMER "Artist Name"
    INDEX 01 12:15:40
  (Additional tracks go here...)
```

## Folha de dicas do CDRWIN

CDRWIN Cue Sheet é uma variação específica do formato de arquivo ".cue" usado pelo software CDRWIN. CDRWIN é um software de gravação e criação de CD/DVD e suas folhas ".cue" foram projetadas para uso com este software. Essas folhas ".cue" contêm informações necessárias para que o CDRWIN crie ou grave CD ou DVD com precisão.

Aqui estão alguns detalhes importantes específicos das folhas de dicas do CDRWIN:

1. **Exclusivo para CDRWIN**: As folhas ".cue" do CDRWIN são formatos proprietários específicos do software CDRWIN. Embora compartilhem semelhanças com arquivos ".cue" padrão, eles são adaptados para funcionar com os recursos e configurações do CDRWIN.
    






2. **Interface amigável**: CDRWIN fornece uma interface fácil de usar para criar e editar suas planilhas ".cue". Os usuários podem adicionar ou modificar informações sobre trilhas, índices, lacunas e outros parâmetros de maneira amigável.
    






3. **Tipos de disco compatíveis**: As Cue Sheets CDRWIN são usadas principalmente para criar vários tipos de CDs e DVDs, incluindo discos de dados, CDs de áudio, discos de modo misto e muito mais. O formato permite aos usuários especificar o tipo e o conteúdo do disco que desejam criar.
    






4. **Controle sobre o layout do disco**: As Cue Sheets do CDRWIN fornecem controle detalhado sobre o layout e a estrutura do disco, incluindo ordem das faixas, configurações de pausa/intervalo e quaisquer outras preferências específicas que o usuário possa ter.
    






5. **Suporte ISO e BIN**: O CDRWIN pode funcionar com formatos de imagem de disco ISO e BIN. A folha ".cue" especifica qual arquivo de imagem é usado para o disco, garantindo a sincronização adequada entre os cues e a imagem.
    






6. **Gravação e Extração**: Essas folhas ".cue" são cruciais ao gravar discos ou extrair faixas de discos usando o CDRWIN. Eles ajudam a garantir que o produto final corresponda ao layout e ao conteúdo pretendidos.
    






7. **Backup e Restauração**: Os usuários do CDRWIN costumam criar planilhas ".cue" ao fazer backups de seus CDs ou DVDs. Estas folhas podem ser usadas posteriormente para restaurar o conteúdo do disco, incluindo o layout e o tempo das faixas.

## Como abrir o arquivo CUE?

As Cue Sheets do CDRWIN são projetadas especificamente para o software CDRWIN, portanto, normalmente devem ser abertas e usadas dentro do CDRWIN.

Programas que abrem ou fazem referência a arquivos CUE

-CDRWIN
- Projetos Inteligentes IsoBuster
- Sistemas EZB UltraISO
