## O que é um arquivo WUD?

Um arquivo com extensão .wud é uma imagem de disco criada a partir de um disco de jogo Wii U. O arquivo de imagem de disco resultante pode ser carregado por emuladores de videogame como [Cemu](https://cemu.info/) para emular o jogo. Os arquivos WUD são grandes em tamanho e, portanto, são criados como vários segmentos para serem transferidos facilmente. As informações do jogo são exportadas como segmentos de arquivo WUD de aproximadamente 2 GB de tamanho de arquivo. O aplicativo de abertura, como o Cemu, precisa que todos esses arquivos segmentados sejam mesclados em um único arquivo WUD antes de abri-los.

## Formato de arquivo WUD

Os arquivos WUD são salvos em disco em formato de arquivo proprietário e podem ser carregados no emulador Cemu para emular aplicativos Wii U no PC. Devido ao seu tamanho grande, os arquivos WUD são [compactados](/pt/compression/) para reduzir o tamanho do arquivo. O arquivo compactado resultante é conhecido como arquivo WUX.
