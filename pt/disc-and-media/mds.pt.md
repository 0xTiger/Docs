## O que é um arquivo .MDS?

O MDS (Media Descriptor Sidecar File) é um arquivo associado ao software Alcohol 120% e Daemon Tools, ambos usados para criar e gerenciar unidades virtuais de CD e DVD. O arquivo MDS contém informações sobre o layout dos dados no disco, incluindo a localização de todos os arquivos e a estrutura do disco. Isso permite que a unidade virtual emule o comportamento de um disco físico, para que o software possa ler os dados do disco virtual como se fosse um disco real.

When you want to mount an image file to a virtual drive, the MDS file is used along with the corresponding image file in a format such as ISO, BIN or CUE. The MDS file contains a description of the layout of the data on the disc, and the virtual drive software uses this information to create a virtual disc. This allows you to use the software as if you were using a physical disc, without having to physically insert the disc into the drive.

## Relação com Álcool 120%

Alcohol 120% é um software popular para criar e gerenciar unidades virtuais de CD e DVD e usa o formato de arquivo MDS para salvar e acessar imagens de disco. O formato de arquivo MDS é propriedade do Alcohol 120% e só pode ser aberto e usado pelo software Alcohol 120% ou outro software que o suporte.

## Como abrir o arquivo MDS?

Para abrir um arquivo MDS no Alcohol 120%, você precisa ter o software instalado em seu computador. Depois de instalar o Alcohol 120%, você pode abrir o arquivo MDS fazendo o seguinte:

1. Lançamento Álcool 120%.
2. Clique no menu Arquivo” e selecione Abrir”.
3. Navegue até o local do arquivo MDS em seu computador e selecione-o.
4. O arquivo MDS agora será aberto no Alcohol 120% e você será solicitado a selecionar o arquivo de imagem correspondente (como ISO, BIN ou CUE).
5. Depois de selecionar o arquivo de imagem, o Alcohol 120% montará a imagem em um drive virtual.

Alternativamente, você também pode abrir um arquivo MDS clicando duas vezes nele, se o Alcohol 120% estiver definido como o programa padrão para abrir arquivos MDS. Você também pode usar outro software que suporte o formato MDS, como Daemon Tools, Virtual CloneDrive, Virtual Drive Manager e MagicISO.
