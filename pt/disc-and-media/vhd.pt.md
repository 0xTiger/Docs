## O que é um arquivo VHD?

VHD é uma abreviação de disco rígido virtual. Esses são formatos de arquivo que representam uma unidade de disco rígido virtual. Os dados encontrados em uma unidade de disco rígido física podem estar envolvidos nela, assim como as partições do disco rígido e o sistema de arquivos que compreende as pastas.

Para uma máquina virtual, os arquivos VHD são considerados discos rígidos. Nas versões modernas do Windows, eles são integrados. Esse tipo específico de disco rígido virtual fornece a facilidade para que diferentes sistemas operacionais residam na máquina host comum. Ele oferece uma grande chance para os desenvolvedores testarem o software em vários sistemas operacionais sem fazer o processo de instalação de outro disco rígido. Há muitas vantagens de usar esse formato de arquivo de arquivos VHD, pois eles fornecem muitos recursos.


## Breve história ##

**Connectix** desenvolveu este formato de arquivo como um produto de PC virtual para uso. Em 2003, a Microsoft adquiriu o Connectix e lançou outras versões deste formato, como o VHDx, juntamente com os melhores recursos e funcionalidades que faltavam na primeira versão. Em 2012, o VHDx foi adicionado ao Windows. Ele foi adicionado ao Windows para adicionar mais capacidade de armazenamento, proteção contra corrupção de dados, otimização para evitar a degradação do desempenho. Esses recursos foram adicionados para discos físicos com setores grandes.

## Formato de arquivo VHD ##

Um sistema de arquivos host é acessado para residir os arquivos VHD nele. Estes possuem características da imagem fixa do disco de guarda que permite ao usuário acessar um arquivo que está alocado no tamanho de um disco virtual específico. Os arquivos podem modificar o disco rígido de uma máquina virtual.

Esses arquivos VHD podem ser trocados facilmente do disco rígido virtual para o sistema de arquivos do host. Este formato também tem a capacidade de backup e restaurações e segurança antivírus. Os arquivos VHD podem suportar formatos específicos como uma imagem de disco rígido fixa, imagem de disco rígido diferencial e imagem de disco rígido dinâmica.
