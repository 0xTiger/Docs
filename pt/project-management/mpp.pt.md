## O que é um arquivo MPP?

Um arquivo com extensão .mpp é um arquivo de dados do Microsoft Project que armazena informações relacionadas ao gerenciamento de projetos de forma integrada. A Microsoft desenvolveu esse formato de arquivo proprietário para torná-lo compatível com o Microsoft Project (MSP), seu produto de software de gerenciamento de projetos. Além do MPP, o MSP pode trabalhar com o esquema XML do projeto. Muitos aplicativos e APIs permitem que MPPs sejam convertidos em outros formatos de arquivo. A Microsoft agora tem um Project Server online onde os arquivos de gerenciamento de projetos podem ser carregados para colaboração por vários usuários.

## Formato de arquivo MPP ##

A Microsoft desenvolveu e publicou o MPP como formato de arquivo binário e suas especificações não foram divulgadas pela Microsoft até o momento. A estrutura e os detalhes internos do arquivo não estão, portanto, disponíveis para desenvolvedores. Existem várias APIs disponíveis online que suportam leitura/gravação em arquivos MPP, mas a precisão de 100% nunca é alcançada. A Microsoft introduziu vários recursos em seu aplicativo de gerenciamento de projetos ao longo do tempo, assim como os recursos do formato de arquivo MPP.

O arquivo MPP usa os seguintes tipos MIME:

* application/vnd.ms-project
* aplicativo/msproj
* aplicativo/msproject
* aplicativo/x-msproject
* aplicativo/x-ms-project
* application/x-dos_ms_project
* aplicação/mp
* zz-application/zz-winassoc-mpp

## Problemas para abrir um arquivo MPP? ##

Aqui está a lista de alguns problemas comuns que podem surgir e causar mau funcionamento do formato MPP:

* Ausência de software de apoio
* Arquivo corrompido
* Arquivo infectado devido a vírus
* Sem direito de acesso no sistema para abrir os arquivos
* Unidade desatualizada em seu sistema
* A extensão do arquivo é renomeada
