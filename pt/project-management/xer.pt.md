## O que é um arquivo XER?

O formato de arquivo XER é um formato de arquivo de projeto proprietário usado pelo aplicativo de planejamento e gerenciamento de projetos Primavera P6. Assim como o formato de arquivo [MPP](/pt/project-management/mpp/), esse formato de arquivo armazena dados relacionados ao planejamento do projeto, como tarefas, recursos, cronogramas e materiais. O Primavera é desenvolvido pela Oracle e fornece planejamento completo de projetos, e encontra suas aplicações em diversos campos práticos, como petróleo e gás, construção, manufatura, etc. Ajuda os gerentes de projeto a planejar, programar e controlar programas de grande escala e projetos individuais, com sistemas de gestão financeira e gestão de capital humano. Muitos softwares e APIs podem ler e converter arquivos XER para outros formatos de arquivo de projeto populares, como XML e MPP.

## Formato de arquivo XER ##

Um arquivo XER é um arquivo de texto e pode ser facilmente aberto com um editor de texto. Abrir dados relacionados ao projeto em um editor de texto pode ser útil, mas não é fácil compreendê-los estabelecendo links manualmente. Este formato é o mais utilizado para troca de dados com outros bancos de dados P6. Ele tem um nível relativamente baixo de controle sobre os dados que foram importados de outro formato. As vantagens do XER no formato XML equivalente incluem a capacidade de mover códigos e calendários globais para o nível do projeto e códigos de mapa, UDFs, recursos e calendários para valores existentes.


## Problemas para abrir um arquivo XER? ##

Alguns problemas comuns que podem surgir e causar mau funcionamento do formato XER são os seguintes:

* Ausência de software de apoio
* Arquivo corrompido
* Arquivo infectado devido a vírus
* Descrição XER excluída acidentalmente do registro do Windows
* Sem direito de acesso no sistema para abrir os arquivos
* Unidade desatualizada em seu sistema
* A extensão do arquivo é renomeada
