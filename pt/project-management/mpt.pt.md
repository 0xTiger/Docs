## O que é um arquivo MPT? ##

Um arquivo com extensão .mpt é um arquivo de modelo do [Microsoft Project](https://products.office.com/en-us/project/project-and-portfolio-management-software). Ele contém informações básicas e configurações de estrutura e documento para criar arquivos [MPP](/pt/project-management/mpp/). Esses modelos fornecem configurações padrão que são essenciais para um projeto específico, como cronogramas ou informações de orçamento. Embora não possa salvar dados específicos do projeto, como tarefas, recursos ou atribuições. Uma vez modificados, os arquivos de modelo podem ser salvos como arquivos MPP padrão para trabalhar com ele. Um arquivo de modelo mestre, intitulado "Global.mpt" está disponível no Microsoft Project, aberto para criar um modelo individual. Os arquivos MPT usam *application/vnd.ms-project* e *application/x-project* como tipos MIME padrão. Windows, Linux e macOS podem abrir arquivos MPT, juntamente com Moos Project Viewer, Microsoft Project e MPXJ.

## Problemas para abrir um arquivo MPT? ##

Aqui está a lista de alguns problemas comuns que podem surgir e causar mau funcionamento do formato MPT:

* Ausência de software de apoio
* Arquivo corrompido
* Arquivo infectado devido a vírus
* Sem direito de acesso no sistema para abrir os arquivos
* Unidade desatualizada em seu sistema
 *   Extension of the file is renamed
