## O que é um arquivo MABK?

Um arquivo MABK é um arquivo Motorolla Android Backup criado pelo Lenovo Android Management
utilitário, Lenovo Moto Smart Assistant. É um arquivo de backup contendo uma cópia dos aplicativos,
contatos, mensagens de texto e arquivos de mídia armazenados no telefone. O arquivo MABK não salva
configurações do aplicativo do usuário do ifnormation.

## Formato de arquivo MABK

Os arquivos MABK são gerados usando o aplicativo Lenovo Moto Smart Assistant (LMSA) em
Telefone MOtorola, telefone Lenovo e tablets Lenovo. É um aplicativo do Windows e armazena o
fazer backup de dados nos dispositivos do usuário. O aplicativo, quando executado, se conecta ao dispositivo do usuário e
take cria o arquivo MABK quando a opção **Backup & Restore** é selecionada.
Os usuários finais podem salvar os arquivos MABK em seus PCs para uso posterior, se precisarem restaurar o
conteúdo do arquivo de backup. Os usuários geralmente fazem backup de todos os arquivos disponíveis no dispositivo, como
contatos, mensagens de texto, imagens, áudios, vídeos e aplicativos.

## Como abrir arquivos MABK?

Você pode utilizar um arquivo MABK para restaurar ou transferir dados em seu telefone Motorola,
Telefone Lenovo ou tablet Lenovo usando Lenovo Moto Smart Assistant para Windows. Para
realizar isso:
1. Conecte seu dispositivo móvel ao PC.
2. Inicie o Lenovo Moto Smart Assistant.
3. Navegue até Backup e restauração e escolha Restaurar.
4. Importe seu arquivo MABK.
5. Selecione Restaurar e siga as instruções na tela para concluir a restauração dos dados ou
processo de transferência.
Além disso, você pode acessar manualmente os arquivos armazenados em um arquivo MABK. Por exemplo, você pode
extraia as imagens contidas no arquivo MABK e visualize-as no seu PC. Para extrair arquivos de
um arquivo MABK:
1. Abra o Lenovo Moto Smart Assistant.
2. Vá para Backup e restauração.
3. Selecione Restaurar.
4. Importe seu arquivo MABK.
5. Escolha Extrair e siga as instruções na tela para extrair arquivos do seu arquivo MABK.
