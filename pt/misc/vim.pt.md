## O que é um arquivo VIM?

Um arquivo VIM é um arquivo de configuração/configurações usado pelo programa de edição de texto conhecido como Vim. Ele armazena configurações do editor, como realce de sintaxe, e é usado por desenvolvedores de aplicativos para configurar a aparência do editor quando ele é aberto. O editor de texto Vim é altamente configurável e foi desenvolvido para criar e alterar texto com eficiência. Ele vem empacotado como "vi" com a maioria dos sistemas UNIX.
