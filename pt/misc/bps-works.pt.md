## O que é um arquivo .BPS?

Um arquivo .BPS é essencialmente um backup de um documento de processamento de texto criado com o Microsoft Works, que normalmente usa a extensão de arquivo .WPS para seus documentos. Esses arquivos .BPS geralmente são gerados automaticamente pelo Microsoft Works como precaução caso o programa trave enquanto você trabalha no documento. É uma forma de proteger seu trabalho e evitar a perda de dados.

Na seção a seguir, exploraremos aplicativos de software associados a arquivos BPS.

## Microsoft funciona

Microsoft Works foi um pacote de software de produtividade desenvolvido pela Microsoft. Foi introduzido pela primeira vez em 1987 e ganhou popularidade como uma alternativa mais acessível ao pacote completo do Microsoft Office. O Microsoft Works incluiu vários aplicativos básicos de escritório, como:

- **Processador de texto:** o Microsoft Works tinha um aplicativo de processamento de texto que permitia aos usuários criar e editar documentos. Ele usou o formato de arquivo [.WPS](/pt/word-processing/wps/) para documentos de processamento de texto.

- **Planilha:** Inclui aplicativo de planilha para criação e gerenciamento de planilhas. O formato de arquivo das planilhas era [.XLR](/pt/spreadsheet/xlr/).

- **Banco de dados:** O Microsoft Works também tinha um aplicativo de banco de dados para organizar e gerenciar dados. O formato de arquivo para arquivos de banco de dados era [.WDB](/pt/database/wdb/).

- **Calendário e catálogo de endereços:** além dos principais aplicativos de escritório, o Microsoft Works geralmente incluía ferramentas para gerenciar calendários e catálogos de endereços.

- **Formato de processador de texto do Works (.WPS):** O formato de arquivo nativo do componente de processamento de texto do Microsoft Works. Os arquivos .WPS continham texto, formatação e gráficos simples.

- **Descontinuação:** A Microsoft descontinuou oficialmente o Microsoft Works em 2007 e incentivou os usuários a mudar para o Microsoft Office ou outras alternativas.
