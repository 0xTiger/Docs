## O que é um arquivo BUP?

Um BUP é o backup de um arquivo IFO. Ele armazena informações semelhantes ao arquivo IFO para que os players de DVD possam usar esse arquivo se o arquivo IFO original não puder ser lido. Os arquivos BUP informam ao DVD player sobre o menu de inicialização, a localização das faixas de vídeo e áudio e o início dos capítulos no DVD. Os arquivos BUP podem ser abertos com o VLC Player.

## Formato de arquivo BUP - Mais informações

Os DVDs podem ser corrompidos devido a possíveis arranhões em suas superfícies. Isso torna difícil para uma unidade de DVD lê-los. Se o [IFO](/pt/video/ifo/) não puder ser lido devido a arranhões, os DVD players não poderão reproduzir esse disco. Nesses casos, os media players recorrem ao arquivo BUP que é um backup real do arquivo IFO.
