## O que é um arquivo DOP?

Pepakura Designer é um programa que permite aos usuários criar modelos artesanais de papel a partir de modelos 3D; Os arquivos Pepakura normalmente têm uma extensão .pdo; esses arquivos contêm instruções e informações de layout necessárias para imprimir e montar um modelo de papel.

Os arquivos Pepakura Designer estão em um formato proprietário específico do software Pepakura; o formato de arquivo foi projetado para armazenar informações sobre o modelo 3D, incluindo geometria, referências de textura e instruções de montagem.

Pepakura Designer trabalha com modelos 3D em formatos como [.obj](/3d/obj/), [.3ds](/3d/3ds/), [.dxf](/cad/dxf/) e outros; esses modelos são então desdobrados em padrões 2D que podem ser impressos e montados no modelo final em papel.

O arquivo .pdo inclui instruções sobre como cortar, dobrar e colar pedaços de papel impresso para recriar o modelo 3D; cada peça é etiquetada e numerada para orientar o processo de montagem.

## Como abrir o arquivo PDO?

Para abrir e visualizar arquivos do Pepakura Designer, você precisará do **Pepakura Viewer**, um programa complementar gratuito do **Pepakura Designer**. O Visualizador permite que outras pessoas visualizem e imprimam modelos sem a necessidade de software de design completo.

## Sobre Pepakura Designer

Pepakura Designer é um software especializado desenvolvido para criar modelos artesanais de papel a partir de modelos digitais 3D. O software permite aos usuários importar modelos 3D em vários formatos, como .obj ou .3ds, e depois desdobrar e nivelar esses modelos em padrões 2D que podem ser impressos em papel. Esses padrões, salvos com uma extensão de arquivo .pdo, contêm instruções e marcações para cortar, dobrar e montar os pedaços de papel em um modelo tridimensional. Pepakura Designer é amplamente utilizado por amadores e entusiastas interessados em criar modelos de papel complexos de vários objetos, personagens e designs.
