## O que é um arquivo XMZX?

Um arquivo XMZX é um arquivo de teste de exame para a realização de testes e é usado pelo aplicativo de software ExamSoft SofTest. O aplicativo pode ser usado tanto no Windows quanto no macOS. Ele contém questões de um exame que o examinador pode responder aos alunos em um computador com o software SofTest. Os arquivos XMZX são enviados aos examinadores para a realização do teste. Ele é carregado no software SofTest e o candidato ou solicitante realiza a prova. Após a conclusão do teste, o arquivo XMDX finalizado é automaticamente carregado nos servidores em nuvem da ExamSoft.

## Formato de arquivo XMZX - Mais informações

Os arquivos XMZX são usados pela plataforma de avaliação digital ExamSoft, que é usada por diversas organizações para realizar testes para certificações. A plataforma possui servidores próprios em nuvem online onde todos os testes concluídos são carregados automaticamente. Os testes são realizados em um centro de exames ou os examinadores também podem realizar o teste dos candidatos em seus próprios computadores. ExamSoft está disponível nos sistemas operacionais Windows e macOS. XSMX é outro formato de arquivo de teste da ExamSoft.

## Como abrir um arquivo XMZX?

Você pode abrir um arquivo XMZX no software ExamSoft SofTest apenas porque é um formato de arquivo proprietário. O software SofTest tem a opção **Fazer um exame** que é usada para iniciar o exame selecionando o arquivo XMZX.
