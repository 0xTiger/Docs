## O que é um arquivo DAP?

Um arquivo DAP é um arquivo de download em andamento criado pelo arquivo Download Accelerator Plus (DAP). Ele é criado quando os usuários iniciam um download de qualquer repositório de arquivos online usando o software DAP. O DAP fornece a capacidade de pausar e retomar um download que está em andamento. Para isso, ele salva o arquivo parcialmente baixado com extensão .dap até que seja completamente baixado. O arquivo DAP pode ser visualizado durante o processo de download.

Os arquivos DAP podem ser abertos com o software SpeedBit Download Accelerator Plus (DAP).
