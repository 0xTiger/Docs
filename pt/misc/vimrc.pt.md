## O que é um arquivo VIMRC?

Um arquivo VIMRC é um arquivo de configurações usado pelo editor de texto Vim. Ele armazena as configurações do editor que são carregadas quando o editor é aberto. Desenvolvedores de aplicativos e administradores de sistema definem configurações relacionadas à aparência do texto, como realce de sintaxe, que são salvas no arquivo VIMRC. Assim, o arquivo VIMRC permite que os usuários personalizem as configurações do editor.

## Formato de arquivo VIMRC

Os arquivos VIMRC são salvos como arquivos de texto simples e podem ser abertos/editados em qualquer editor de texto, como Notepad, Notepad++ e TextEdit. Eles contêm instruções que podem ser executadas no editor do Vim começando com um ":", seguido pelo comando. Os arquivos VIMRC contêm instruções para configurações como recuo de texto e cores de janelas e fontes. Um formato de arquivo relevante é [VIM](/pt/misc/vim/).
