## O que é um arquivo OGG?

Um arquivo OGG é um arquivo gráfico criado com o software Origin. Ele fornece a capacidade de armazenar diferentes tipos de gráficos para exibir dados importados para análise de dados. Os tipos de gráfico suportados pelo tipo de arquivo OGG incluem coluna, barra, torta, cascata, 3D, polar e símbolo. O software Origin foi desenvolvido pela OriginLab e permite que os usuários realizem análises de dados e gerem resultados como gráficos.

## Formato de arquivo OGG

Os arquivos OGG contêm gráficos para representação de dados importados. Quando um arquivo OGG é aberto, ele exibe o gráfico em uma janela separada no Origin ou no Origin Viewer, que está disponível gratuitamente em comparação com o software Origin. Informações de metadados, como título e cores de um gráfico, também são armazenadas dentro do arquivo OGG.

## Sobre o software OriginLab Origin

O software Origin também oferece a opção de ser usado como Servidor de Automação. Nesse caso, aplicativos clientes como LabVIEW, Microsoft Excel, Python ou aplicativos desenvolvidos sob medida em C#/VB/VC podem trocar dados com o Origin. [Python_to_OriginLab](https://github.com/jsbangsund/python_to_originlab) é uma API Python para automatizar o envio de dados e funções básicas de plotagem no OriginLab.
