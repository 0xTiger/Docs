## O que é um arquivo AML?

Um arquivo AML (Automation Markup Language) é um arquivo de dados usado para armazenar e trocar informações de engenharia da planta. Baseia-se no formato de arquivo XML, fazendo com que a troca de informações seja baseada em padrões abertos. O AutomationML oferece a possibilidade de interconectar ferramentas de engenharia modernas nas áreas de engenharia de plantas mecânicas, projeto elétrico, controladores lógicos programáveis e controle de robôs. Os arquivos AML podem ser visualizados e editados usando a ferramenta AutomationML Editor.

## Formato de arquivo AML

AML usa modelagem de dados orientada a objetos baseada em [XML](/pt/web/xml/) para armazenamento e troca de modelos de engenharia. Os componentes da planta de engenharia são representados como objetos com diferentes aspectos, com cada objeto tendo a capacidade de atuar como subobjetos. Assim, cada modelo menor ou maior pode ser descrito. O [AutomationML Specifications](https://www.automationml.org/about-automationml/specifications/) fornece um pequeno guia para começar com os detalhes dessa linguagem de modelagem.

## Primeiros passos com AML

O uso de AML para troca de informações de engenharia requer certo conhecimento para aplicação bem-sucedida. A implementação de novas aplicações necessita de várias etapas para este fim conforme abaixo.

* Conheça a arquitetura
* Adquirir conhecimentos básicos
* Examinando o caso de uso
* Mapeamento e modelagem específicos do aplicativo
* Implementação
