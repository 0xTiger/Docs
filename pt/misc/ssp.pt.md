## O que é um arquivo SSP?

Um arquivo SSP é um arquivo de projeto criado pelo software de criação de screensavers do Windows conhecido como Screensaver Producer pela Axialis Software. Ele pode conter vários tipos diferentes de elementos de proteção de tela, como sprites, filmes em Flash, apresentações de slides de fotos e vídeos. Um arquivo SSP pode incluir texto, imagens, formas diferentes e configurações de animação usadas para mostrar a animação dentro do arquivo SSP. Os arquivos SSP podem ser abertos com o Axialis Screensaver Producer.

## Formato de arquivo SSP - Mais informações

Os arquivos SSP são armazenados em disco como arquivos binários sem informações públicas disponíveis sobre sua estrutura de formato de arquivo interno. Axialis Screensaver produtor profissional ferramenta de criação de screensaver para Windows que permite criar facilmente screensavers de alta qualidade para todas as versões do Windows até o Windows 10. Permite criar pacotes de screensaver instaláveis que podem ser facilmente distribuídos online para os usuários.

Os arquivos SSP podem ser exportados para o formato de arquivo [SCR](/pt/system/scr/).
