## O que é um arquivo OLM?

Um arquivo com extensão .olm é um arquivo do Microsoft Outlook para sistema operacional Mac. Um arquivo OLM armazena mensagens de email, diários, dados de calendário e outros tipos de dados de aplicativos. Eles são semelhantes aos arquivos [PST](/pt/email/pst/) usados pelo Outlook no sistema operacional Windows. No entanto, os arquivos OLM criados pelo Outlook para Mac não podem ser abertos no Outlook para Windows. O formato de arquivo OLM foi introduzido no Outlook 2011 para armazenar dados locais do Outlook. Um único arquivo OLM pode conter várias caixas de correio, organizando os dados na forma de pastas e subpastas. Os arquivos OLM podem ser abertos/carregados no Microsoft Outlook 365 e no Outlook para Mac.

## Formato de arquivo OLM

OLM são arquivos binários com tipo MIME Application/octet-stream. As especificações técnicas do formato de arquivo OLM não estão disponíveis abertamente na Microsoft e não há detalhes de formato disponíveis para referência do desenvolvedor. Existem algumas APIs, no entanto, que podem ler o formato de arquivo OLM e gravá-los em um arquivo PST para uso no sistema operacional Microsoft Windows, como o Aspose.Email.
