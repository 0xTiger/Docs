## O que é um arquivo .USX?

Um arquivo USX contém malhas do Unreal Engine que são objetos/modelos 3D pré-projetados usados para gerar o ambiente. Os modelos do arquivo USX têm posições, orientações e tamanhos diferentes, mas permanecem estáticos. As malhas estáticas podem ser otimizadas para desempenho e usadas para criar vários níveis de detalhe (LODs) para melhorar o desempenho do jogo. Eles também podem ser usados para criar volumes de colisão e como parte de simulações físicas.

Arquivos USX podem ser criados com o editor de nível Unreal UnrealEd.

## Formato de arquivo USX – Mais informações

Os arquivos USX são salvos no disco como arquivos binários. Os detalhes da estrutura interna do arquivo do formato de arquivo USX não estão disponíveis publicamente para referência do desenvolvedor.
