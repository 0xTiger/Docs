## O que é um arquivo .GCF?

Um arquivo .gcf é um arquivo de cache de jogo usado pela plataforma de jogos Steam. Ele contém dados do jogo, como texturas, modelos e outros arquivos usados pelo jogo. Esses arquivos são armazenados no computador do usuário e são usados para reduzir a quantidade de dados que precisam ser baixados ao atualizar ou instalar um jogo. Os arquivos .gcf também podem ser usados para criar backups da instalação de um jogo ou para transferir um jogo entre computadores.

Os arquivos GCF podem ser lidos e gravados pela biblioteca de programação C++ de código aberto, **HLLib**.

## Formato de arquivo GCF

Os arquivos GCF são salvos em disco como arquivos binários. GCF foi originalmente usado como um acrônimo para Grid Cache File (o codinome inicial do Steam era Grid), mas mais tarde foi considerado Game Cache File. Um arquivo GCF é um arquivo que armazena jogos Steam. Todo o conteúdo oficial é baixado como arquivos GCF. Os arquivos GCF também podem ser compartilhados entre jogos.

NOTA: GCF é o antecessor de [VPK file format](/game/vpk/).
