## .UPS opção número

Um arquivo UPS, que significa **Universal Patching System**, é um tipo de arquivo de patch usado para modificar ou aplicar alterações aos dados de um videogame ou outro software. Os arquivos UPS são normalmente usados na comunidade de emulação para aplicar patches ou modificações em ROMs (memória somente leitura) de videogames antigos.

## Arquivo de patch UPS

Veja como funciona um arquivo de patch UPS:

1. **Dados originais:** Você começa com o arquivo ROM original de um videogame ou software que deseja modificar. Esta ROM contém código e dados do jogo.
    






2. **Arquivo de patch:** Um arquivo de patch do UPS é criado usando um software de patch especializado. Este arquivo de patch contém alterações que você deseja aplicar aos dados originais. Essas alterações podem incluir correções de bugs, patches de tradução ou outras modificações.
    






3. **Processo de patch:** Para aplicar alterações, você usa uma ferramenta de patch do UPS que usa a ROM original e o arquivo de patch do UPS como entrada. A ferramenta de patch aplica então as alterações especificadas no arquivo de patch do UPS à ROM original, criando uma versão modificada do jogo ou software.
    






4. **Resultado:** A saída é uma ROM corrigida ou modificada que incorpora alterações do arquivo de correção do UPS. Esta ROM corrigida pode então ser reproduzida em um emulador ou usada de outras maneiras, dependendo das modificações feitas.
    







Os arquivos de patch UPS são populares na comunidade de emulação porque permitem aos usuários fazer melhorias ou modificações em jogos antigos sem distribuir a ROM inteira do jogo, que pode estar sujeita a direitos autorais. Em vez disso, os usuários podem compartilhar apenas o arquivo de patch do UPS que contém alterações e podem aplicá-lo à ROM original de propriedade legal.

Os patches UPS oferecem diversas vantagens em relação ao [IPS](/pt/game/ips/):

1. **Sem limitação de tamanho:** Ao contrário dos patches IPS que são restritos a 16 MB, os patches UPS podem ser usados com arquivos de qualquer tamanho. Essa flexibilidade torna o UPS particularmente útil para jogos e softwares maiores.
    






2. **Patch bidirecional:** O UPS suporta patch bidirecional, o que significa que um único patch do UPS pode aplicar e remover alterações em um jogo ou software. Este recurso simplifica o processo de modificação ou reversão de alterações, melhorando a experiência do usuário.
    






3. **Somas de verificação CRC32:** A UPS emprega somas de verificação CRC32 como um mecanismo integrado para garantir que os patches sejam aplicados ao jogo ou software correto. Essa verificação de soma de verificação ajuda a manter a integridade dos arquivos corrigidos e evita a aplicação acidental ou incorreta de patches.

## Como aplicar um patch UPS?

Para aplicar um patch UPS, você precisará de uma ferramenta de patch UPS, como Lunar IPS (para Windows), NUPS (para Windows) ou MultiPatch (para múltiplas plataformas) e ROM original do jogo ou software que deseja corrigir.

## Como abrir o arquivo UPS?

Programas que abrem arquivos UPS incluem

- **IPS Lunar** (para Windows)
- **NUPS** (para Windows)
- **MultiPatch** (para múltiplas plataformas)
