## O que é um arquivo CT?

Um arquivo CT é um arquivo de tabela de truques criado com o software [Cheat Engine](https://www.cheatengine.org/) que é usado para criar modificações em jogos baseados no Windows para trapaça. O Cheat Engine, um mecanismo de trapaça de código aberto, examina os jogos em execução e registra seus locais de endereço. Essas informações e suas substituições são gravadas no arquivo CT que é carregado pelos jogadores para alterar as propriedades do jogo, como pontuação de saúde, pontuação mais alta e vidas restantes.

## Formato de arquivo CT do Cheat Engine

Os arquivos CT são salvos com locais de endereço e outras informações associadas para serem hackeadas dentro de um jogo. Na maioria dos casos, os arquivos são salvos como arquivos compactados [ZIP](/pt/compression/zip/) que podem ser facilmente extraídos usando qualquer utilitário de descompactação padrão, como WinZIP ou 7-Zip.

## Como usar as tabelas de trapaças

1. Baixe a tabela e copie-a para a pasta do Cheat Engine
1. Execute o Cheat Engine
1. Execute o jogo;
1. Use a combinação de ALT+TAB e escolha o jogo na lista de processos via Cheat Engine
1. Se a tabela estiver em uma pasta diferente, basta pressionar Control+O e guiar o Cheat Engine para essa pasta. Em seguida, selecione a tabela (geralmente process_name.ct);
1. Uma vez carregada a tabela, se houver um script, basta verificar.
1. ALT+TAB de volta ao jogo e divirta-se.
