## O que é um arquivo PUP?

Um arquivo PUP é um arquivo de patch de atualização do sistema usado pelo software do sistema PlayStation 4 (PS4) e PlayStation 5 (PS5). Ele atualiza o firmware do jogo para uma versão mais recente que pode conter aprimoramentos de software em termos de correções de bugs, novos recursos do sistema operacional e melhorias em termos de desempenho do software. O arquivo PUP também pode conter patches para corrigir bugs que podem resultar em problemas de segurança. Os arquivos PUP podem ser baixados para o sistema conectando-o à Internet ou baixando manualmente a atualização do firmware para um computador e depois transferindo para o dispositivo usando um pendrive ou outro dispositivo de armazenamento. A versão anterior do PlayStation ie PS3 também usava o arquivo PUP para atualização do firmware.

## Formato de arquivo PUP - Mais informações

Os arquivos PUP são salvos no disco no formato de arquivo binário [compactado](/pt/compression/). No entanto, os detalhes da compactação não estão disponíveis publicamente. O conteúdo de um arquivo PUP pode ser extraído usando o programa PS3 PUP Extractor no sistema operacional Windows. Para atualizar o sistema de software do PS3, o arquivo PUP é carregado e salvo na pasta /PS3/UPDATE na máquina. Isso permite que a opção de atualização do sistema encontre o arquivo de atualização do firmware e o execute.

### Convenção de nomenclatura de formato de arquivo PUP

* `PS3UPDAT.PUP` - O nome de arquivo padrão usado para atualizações do PlayStation 3
* `PS4UPDATE.PUP` - O nome de arquivo padrão usado para atualizações do PlayStation 4
* `PS5UPDATE.PUP` - O nome de arquivo padrão usado para atualizações do PlayStation 5

## Usando o arquivo PUP para redefinir as configurações de fábrica PlayStation

Os arquivos PUP podem ser usados para redefinir o seu dispositivo PlayStation de fábrica. Isso substituirá todos os dados do dispositivo, removerá o firmware existente e o definirá como um dispositivo totalmente novo. A redefinição de fábrica do seu PlayStation também removerá todos os dados do jogo salvos.
