## O que é um arquivo W3N?

Um arquivo com extensão .w3n é um arquivo de mapas de campanha usado pelo jogo [Warcraft III: The Frozen Throne](https://en.wikipedia.org/wiki/Warcraft_III:_The_Frozen_Throne) que é uma extensão do Warcraft III, um tempo famoso jogo de vídeo. O arquivo W3N contém mapas de campanha que são usados para criar campanhas personalizadas usando a série de dados de missões disponíveis nele. O formato é muitas vezes mal interpretado como arquivos de mapa e os usuários não têm idéia de por que seus arquivos W3N não estão sendo carregados no Frozen Throne. Arquivos W3N podem ser abertos com Blizzard Warcraft 3: The Frozen Throne.

## Formato de arquivo W3N - Mais informações

Os arquivos W3N são armazenados em disco como arquivos binários e seu formato de arquivo interno não está disponível publicamente. O componente Campaign Editor do Warcraft III World Editor permite criar arquivos W3N.

## Como instalar arquivos W3N?

Não se engane colocando os arquivos W3N na pasta "maps" do seu computador. Os arquivos W3N não são arquivos de mapa, mas campanhas que podem ser usadas para criar missões. Portanto, para carregar esses arquivos, eles devem ser colocados na pasta "campanhas" e não na pasta "mapas".

### Exemplos de mapas de campanha W3N

Alguns exemplos de mapas de campanha incluem:
* Liga de Kalimdor II v3.46 AI
* Azeroth Reinventado
* Dia do Dragão
* Senhor dos Clãs
