## O que é um arquivo KODU?

Um arquivo .kudo é um arquivo de jogo 3D criado e salvo com o Microsoft [software Kodu Game Lab](https://www.kodugamelab.com/), que é um ambiente educacional de desenvolvimento de jogos. Ele foi projetado para ensinar princípios básicos de programação às crianças e permite que os criadores programem os comportamentos dos elementos do jogo usando a linguagem de programação visual. Os arquivos Kodu podem ser salvos e compartilhados com outros usuários, mas só podem ser abertos e executados no software Kodu.

## Formato de arquivo KODU - Mais informações

Os arquivos Kodu são arquivos binários, salvos no formato de arquivo proprietário da Microsoft. Suas especificações de formato de arquivo não estão disponíveis publicamente para desenvolvedores e é por isso que os arquivos Kodu só podem ser executados dentro do software Kodu Game Lab.
