## O que é um arquivo MCWORLD?
Um arquivo MCWORLD consiste em arquivos de recursos e comportamentos para criar um mundo virtual no jogo Bedrock Edition of Minecraft, que é um famoso jogo de construção de blocos. Comprimiu vários arquivos, como arquivos .DAT e .TXT com compactação ZIP. os arquivos usam a extensão de arquivo .mcworld e são particularmente usados para fazer backup ou transferir um mundo para outro usuário do Minecraft. Esses arquivos geralmente são criados por um software Minecraft chamado Minecraft Launcher.

## Formato de arquivo MCWORLD
Os arquivos MCWORLD são criados por **MinecraftSP.exe**, que também é chamado de Minecraft Launcher. Este arquivo pode ser criado se compactamos ou compactamos os arquivos do mundo em um único arquivo e alteramos a extensão para .mcworld. Por outro lado, podemos expandir os arquivos .mcworld usando softwares utilitários simples de compactação zip, porque os arquivos .mcworld são compactados usando compactação zip.
