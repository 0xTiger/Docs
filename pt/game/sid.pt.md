## O que é um arquivo SID?

Um arquivo SID é um arquivo de backup de jogo criado pelo aplicativo Steam que é usado para baixar e gerenciar **jogos de vídeo da Valve** no sistema operacional Windows. Ele armazena dados de jogos em formato compactado e pode ser usado como backup de jogos para mídia de armazenamento como DVDs e dispositivos de armazenamento USB. O arquivo SID é criado com o utilitário de backup do Steam e é semelhante ao formato de arquivo [CSD](/pt/game/csd/) para armazenar o backup do jogo.

## Formato de arquivo SID - Mais informações

Os arquivos SID são armazenados no disco como arquivos binários. É sempre acompanhado por um arquivo .sis e um arquivo .sim que são usados para extrair informações de metadados para restaurar os dados do jogo de backup.
