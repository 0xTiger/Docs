## O que é um arquivo .IPS?

Um "arquivo IPS" normalmente se refere a um "arquivo de patch do sistema de patch interno". IPS é uma abreviatura de "International Patching System" e é um formato de arquivo usado para criar e aplicar patches em arquivos ROM (memória somente leitura), geralmente no contexto de emulação de videogame e hacking de ROM.

Veja como funciona:

- **ROM original:** Você começa com um arquivo ROM original, que é essencialmente uma cópia de um videogame ou software armazenado em formato ROM. Este arquivo normalmente é somente leitura, o que significa que você não pode modificá-lo diretamente.

- **Arquivo de patch IPS:** Um arquivo de patch IPS contém as alterações que você deseja aplicar à ROM original. Essas alterações podem ser correções de bugs, traduções ou outras modificações.

- **Aplicativo de patch:** Para aplicar as alterações, você precisa de um utilitário ou programa que possa ler o arquivo de patch IPS e aplicá-lo à ROM original. Este processo basicamente "corrige" o arquivo ROM original com as alterações especificadas no arquivo IPS, criando um arquivo ROM modificado.

- **ROM modificada:** O resultado é uma ROM modificada que incorpora as alterações do arquivo de patch IPS. Esta ROM modificada pode então ser usada em um emulador ou em hardware compatível para jogar o jogo com as modificações desejadas.

Esses patches são normalmente usados para fazer alterações ou melhorias em jogos e podem abranger várias alterações, incluindo modificações em gráficos, modelos e outros dados do jogo. Os arquivos IPS são particularmente adequados para patches relativamente pequenos, normalmente com tamanho inferior a 16 megabytes.

Na seção a seguir, exploraremos os aplicativos de software associados aos arquivos IPS.

## IPS Lunar

Lunar IPS é um utilitário popular e fácil de usar usado para criar e aplicar patches IPS (Internal Patching System) em arquivos ROM. Aqui estão alguns recursos e informações importantes sobre o Lunar IPS:

- **Criação de Patch:** Lunar IPS permite que os usuários criem patches IPS especificando as alterações que desejam fazer em um arquivo ROM original. Você pode selecionar a ROM original e a versão modificada, e o Lunar IPS irá gerar um arquivo de patch IPS com base nas diferenças entre os dois arquivos.

- **Aplicação de patch:** Depois de ter um arquivo de patch IPS, o Lunar IPS permite aplicá-lo a uma ROM original. Este processo essencialmente “corrige” a ROM com as alterações especificadas no arquivo IPS, criando uma ROM modificada.

- **Leve:** Lunar IPS é um programa pequeno e leve, o que significa que não consome muitos recursos do sistema e é rápido para baixar e executar.

## IPSWin

IPSWin é um utilitário projetado principalmente para usuários do Windows que facilita a criação e aplicação de patches IPS (Internal Patching System) em arquivos ROM. É uma ferramenta simples e fácil de usar, adequada para quem deseja corrigir ROMs facilmente. Aqui estão algumas informações sobre o IPSWin:

- **Criação de Patch:** IPSWin permite aos usuários criar patches IPS comparando dois arquivos ROM: a ROM original e a ROM modificada. Ele identifica as diferenças entre os dois arquivos e gera um arquivo de patch IPS contendo essas alterações.

- **Aplicativo de patch:** Este utilitário também permite que os usuários apliquem arquivos de patch IPS existentes a ROMs originais. Ao selecionar o patch IPS apropriado e a ROM original, o IPSWin mesclará as alterações em um novo arquivo ROM, criando uma versão corrigida.
