## O que é um arquivo .OSB?

Um arquivo OSB é um arquivo de storyboard criado pelo jogo de ritmo Osu! que permite aos jogadores tocar em elementos visuais na tela com as batidas da música. O storyboard dentro do arquivo OSB contém um fundo animado personalizado que é usado por trás do jogo durante o jogo. O plano de fundo do storyboard é normalmente um arquivo JPG ou PNG mostrado com o arquivo beatmap OSZ durante o jogo.

## Formato de arquivo OSB

Os arquivos OSB são armazenados como arquivos de texto simples no disco em pastas beatmap. Este esquema permite criar diferentes storyboards com diferentes dificuldades dentro de um beatmap.
