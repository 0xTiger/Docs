## O que é um arquivo .NBT?

Arquivo NBT é um formato de arquivo usado no Minecraft. NBT significa Named Binary Tag e é um formato binário simples para armazenar dados estruturados (semelhante a JSON ou XML). No Minecraft, os arquivos NBT são usados para armazenar vários tipos de dados de jogos, incluindo inventários de jogadores, informações mundiais, dados de entidades e muito mais.

Os arquivos NBT são normalmente armazenados com uma extensão `.dat` e podem ser encontrados na estrutura de diretórios do mundo Minecraft. Por exemplo, os inventários dos jogadores são armazenados em arquivos `.dat` dentro do diretório `playerdata`, enquanto as informações mundiais são armazenadas em arquivos `.dat` dentro do diretório `region`.

## Formato de arquivo NBT – Mais informações

NBT é um formato binário simples projetado para armazenar com eficiência grandes quantidades de dados binários. É usado pelo Minecraft para seus arquivos de dados devido à sua flexibilidade e facilidade de uso. Minecraft é um popular jogo sandbox 3D onde os jogadores podem construir e explorar mundos virtuais. A edição Java do Minecraft é particularmente popular entre os modders, que modificam o jogo editando seus arquivos ou adicionando novos para personalizar a jogabilidade. Os arquivos NBT no Minecraft são frequentemente examinados e editados por modders. Esses arquivos são normalmente compactados usando GZip e podem conter vários itens de dados de jogos cruciais para a construção e personalização do mundo.

## Como abrir um arquivo NBT

Várias ferramentas de software estão disponíveis para abrir e editar arquivos NBT. Aqui está a lista de programas que podem abrir ou fazer referência a arquivos NBT.

- **NBTExplorer** (gratuito) para (Windows, Mac, Linux)
- **NBTEdit** (grátis) para Windows
- **MCEdit** (grátis) para (Windows, Linux)
