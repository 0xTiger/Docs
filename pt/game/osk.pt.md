## O que é um arquivo .OSK?

Um arquivo OSK é um arquivo de layout ou skin usado pelo jogo de ritmo osu! onde os jogadores usam mouse e teclado em sincronia com a música. Ele contém o layout da interface do editor e pode ser customizado pelos jogadores de acordo com suas preferências. osu! armazena dados de beatmap em formato de arquivo [OSZ](/game/osz/).

**Tipo MIME de formato de arquivo OSK:** x-osu-skin-archive

Você pode abrir o arquivo OSK com [osu!](https://osu.ppy.sh/home).

## Formato de arquivo OSK

Os arquivos OSK são salvos no disco como arquivos compactados [ZIP](/compression/zip/). Você pode simplesmente manter um arquivo .osk em .zip e extrair o arquivo para visualizar seu conteúdo. O conteúdo extraído precisa ser colocado no diretório \Skins\ para ser carregado no jogo.

## O que pode ser personalizado com arquivo OSK?

Os seguintes elementos do jogo podem ser personalizados com o arquivo .osk.

 * Acertar círculos
 * Exibição de pontuação
 * Tipo de controle deslizante
 * Cursor
 * Rajadas de acertos
