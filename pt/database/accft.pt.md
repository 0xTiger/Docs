## O que é um arquivo ACCFT?

Um arquivo com extensão .accft é um arquivo de modelo do Microsoft Access que contém a definição de tipos de dados de campo para as tabelas de um banco de dados. O Microsoft Access é um sistema de gerenciamento de banco de dados relacional (RDBMS) de desktop que é usado para criar bancos de dados, formulários, consultas e exibições baseados em tabelas. O arquivo ACCFT pode ser usado para definir os tipos de dados para os campos da tabela e as tabelas de banco de dados podem ser criadas usando este arquivo de modelo como ponto de partida. Ele pode ser aberto com o Microsoft Access 2016 para criar novas tabelas. Os arquivos ACCFT são reconhecidos pelo tipo MIME **application/msaccess**.

## Formato de arquivo ACCFT

Os arquivos ACCFT são arquivos binários cujas especificações de formato de arquivo não estão disponíveis publicamente. Seus detalhes internos como informações de cabeçalho, estrutura de arquivos, rodapés, etc. não estão disponíveis no momento.
