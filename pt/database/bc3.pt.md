## O que é um arquivo .BC3?

Um arquivo BC3 é um arquivo de banco de dados para armazenar informações relacionadas ao formato de troca de banco de dados de construção. Foi desenvolvido com base no formato de troca de banco de dados de construção padrão espanhol. Um arquivo BC3 armazena informações que possuem registros de dados, incluindo detalhes de materiais de construção, custo de materiais e preço de mão de obra. Os dados podem ser trocados entre outros aplicativos sem convertê-los para outros formatos.

## Formato de arquivo BC3

Os arquivos BC3 são armazenados como arquivos binários com detalhes especificados de acordo com os padrões FIEBDC. A implementação dos arquivos BC3 passou por diferentes revisões e existem as seguintes versões do formato de arquivo BC3:

*FIEBDC-3/2004
*FIEBDC-3/2007
*FIEBDC-3/2012
*FIEBDC-3/2016

FIEBDC é a abreviatura de Formato de Intercambio Estándar - Bases de Datos de Construcción em espanhol.
