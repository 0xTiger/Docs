## O que é um arquivo ADP?

Um arquivo com extensão .adp é um arquivo de projeto do Microsoft Access que usa a arquitetura de componentes OLE DB para fornecer uma conexão direta e eficiente com um banco de dados Microsoft SQL Server. Esses arquivos não contêm as tabelas reais, diagramas de banco de dados ou outros elementos de banco de dados. Os arquivos ADP podem ser criados com o Access 2007 e 2010. Os arquivos criados com as versões anteriores do Access também podem ser abertos com o Access 2007 e 2010. Os arquivos ADP podem ser abertos usando o Microsoft Access 365.

## Formato de arquivo ADP - Mais informações

Um projeto do Access criado como ADP permite fazer alterações de design em objetos do servidor SQL, como tabelas e exibições. Ele também permite criar, editar e utilizar recursos SQL, como diagramas de banco de dados, procedimentos armazenados e funções definidas pelo usuário. Isso oferece vantagem em relação à vinculação a um banco de dados do SQL Server, no qual você não pode fazer alterações no design e só pode vincular a tabelas e exibições do SQL Server.

### Tipos de dados e suporte a diagramas

**Tipos de dados de data/hora**

Os novos tipos de dados de data/hora a seguir têm suporte no Access 2010.

* TEMPO
* ENCONTRO
* DATETIME2
* DATETIMEOFFSET

**Tipos de dados de comprimento variável**

Os seguintes tipos de dados de comprimento variável podem ser usados em projetos do Access 2010:

* VARBINA(MAX)
*VARCHAR(MAX)
* NVARCHAR(MAX)
