## O que é um arquivo ANB?

Um arquivo com extensão .anb é um gráfico do IBM i2 Analyst's Notebook que contém informações de relacionamento entre dados diferentes. Como o nome sugere, o software Analyst's Notebook ajuda a vincular os dados, resultando em uma forma mais concisa de conectividade entre as diferentes informações e fornecendo saída de inteligência para análise. Isso ajuda a identificar e interromper atividades maliciosas, como ameaças criminosas, cibernéticas e fraudulentas. Os arquivos ANB podem ser abertos com o software Notebook do IBM i2Analyst.

## Formato de arquivo ANB - Mais informações

Um arquivo ANB é composto pelos seguintes itens constituintes:

* Entidades - Representação de Informações
* Links - Mostra a conectividade entre Entidades
* Propriedades - Descrição de uma entidade ou link com mais detalhes

Os arquivos ANB armazenam todas essas informações com base no feed de informações de entrada no ANB Analyst's Notebook. Com base na entrada de dados, o software cria links entre cada informação que ajuda a mapear relacionamentos e rastrear evidências geoespacial e temporalmente.
