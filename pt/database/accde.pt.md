## O que é um arquivo ACCDE?

Um arquivo com extensão .accde é um formato de arquivo do Microsoft Access 2007 para criar arquivos de banco de dados Executar Somente para proteger o código de banco de dados personalizado. Ele compila todo o código VBA em executável apenas para que não possa ser visualizado ou editado. O formato de arquivo ACCDE é a substituição do formato de arquivo MDE usado por versões anteriores do Microsoft Access. ACCDE são arquivos [ACCDB](/pt/database/accdb/) em modo somente executável e mais compactos devido à ausência de qualquer código fonte. O tamanho compacto desses arquivos melhora o uso da memória e melhora o desempenho do aplicativo. Os arquivos ACCDE podem ser abertos com o Microsoft Access 365.

## Formato de arquivo ACCDE

Todos os arquivos do Microsoft Access 2007+ utilizam o Jet ACE (mecanismo de banco de dados Access) que usa a família de formatos de arquivo ACCDB. No entanto, a Microsoft não fornece uma implementação de código aberto ou uma especificação técnica detalhada.
