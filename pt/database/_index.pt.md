Um banco de dados é uma coleção de dados organizados em tabelas e mantidos em um dispositivo de armazenamento de computador e acessados eletronicamente. Os bancos de dados relacionais armazenam dados em tabelas de banco de dados vinculadas entre si por meio de técnicas formais de design e modelagem. Os dados em um banco de dados são analisados usando o Sistema de Gerenciamento de Banco de Dados (SGBD) que é a principal fonte de interação com usuários finais, aplicativos e o próprio banco de dados.

As extensões de arquivo de banco de dados populares e seus formatos de arquivo incluem **SQLite**, **DB**, **ACCDB** e **MDB**.
