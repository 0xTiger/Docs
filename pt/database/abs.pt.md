## O que é um arquivo .ABS?

Um arquivo com extensão .abs é um banco de dados gerado e usado pelo banco de dados Absolute, que é um mecanismo de banco de dados para o IDE de desenvolvimento de software Delphi. Ele substituiu o Borland Database Engine (DBE) e é mais compacto, de alta velocidade, robusto e fácil de usar. O banco de dados ABS é compilado como parte do arquivo EXE do aplicativo, tornando seu aplicativo mais rápido e menor. Os dados são armazenados em um arquivo ABS em um formato estruturado e relacional.

## Formato de arquivo ABS – Mais informações

Os arquivos ABS são armazenados como parte do [arquivo EXE](/pt/executable/exe/) que geralmente é armazenado como um arquivo binário. Os arquivos binários não podem ser abertos em nenhum editor de texto e fornecem acesso rápido ao conteúdo do arquivo de banco de dados.

### Principais recursos do formato de arquivo ABS

Os arquivos ABS são bastante simples e incorporados ao executável do aplicativo. Possui os seguintes recursos:

*Sem BDE; sem DLLs
* Banco de dados de arquivo único
* Suporte SQL'92 (DDL e DML)
* Compatível com controles de banco de dados padrão e de terceiros
* Modo monousuário e multiusuário (servidor de arquivos)
* Funciona muito bem em todas as versões do Windows - do 98 ao mais recente, não requer atualizações ou service packs
* Tabelas na memória ultrarrápidas
* Facilidade de uso incomparável
* Criptografia forte
* Compressão BLOB
* Gratuito para uso pessoal
* Código fonte completo disponível
* Distribuição isenta de royalties
