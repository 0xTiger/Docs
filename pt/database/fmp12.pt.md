## O que é um arquivo FMP12?

Um arquivo FMP12 é um arquivo de banco de dados criado com o programa de desenvolvimento de banco de dados FileMaker Pro 12. Ele contém a estrutura geral do banco de dados (também conhecido como esquema de banco de dados) junto com os dados dentro dele. A estrutura de um arquivo FMP12 é baseada em tabelas onde cada tabela consiste em vários campos de banco de dados com diferentes tipos de dados. Alguns tipos de dados suportados pelo formato de arquivo FMP12 incluem texto, formulários personalizados, relatórios, data e hora, blobs e outros objetos de banco de dados. Formatos de arquivo mais antigos do FileMaker Pro, como [FP7](/pt/database/fp7/) podem ser atualizados para FMP12.

## Como converter arquivos antigos do FileMaker Pro para FMP12?

Formatos de arquivo mais antigos do FileMaker Pro, como bancos de dados 11, 10, 9, 8 e 7 (.fp7) para formatos de arquivo FMP12. O FileMaker tem diretrizes abrangentes para [converter a versão anterior dos formatos de arquivo do FileMaker Pro para FMP12](https://fmhelp.filemaker.com/help/16/fmp/en/index.html#page/FMP_Help/converting-files.html) .
