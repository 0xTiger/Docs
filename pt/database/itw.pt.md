## O que é um arquivo ITW?

Um arquivo ITW é um formato de arquivo de banco de dados usado pelo aplicativo CRM [InTouch](https://www.intouchsoftware.co.uk/InTouch). Ele armazena o conteúdo no formato de arquivo proprietário do InTouch e os detalhes do formato de arquivo interno não estão disponíveis. Os arquivos ITW armazenam informações personalizadas para armazenar nomes, endereços que são usados para imprimir envelopes e etiquetas. Essas informações também são usadas para enviar e-mails, mensagens de texto, boletins eletrônicos atraentes, criar modelos reutilizáveis, anexos e links. Os dados do CRM também podem ser exportados para o formato ITW.
