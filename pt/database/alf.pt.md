## O que é um arquivo ALF?

Um arquivo com extensão .alf é um log de transações SQL gerado por [ACT! CRM](https://www.act.com/)(Customer Relationship Management). Ele mantém registro de atividades entre o ACT! e SQL, e é usado para restaurar o ACT! base de dados. AJA! O CRM armazena informações do cliente, como nome, endereço de e-mail, contato, informações comerciais e histórico de relacionamento. Ele armazena essas informações no arquivo [ACT! ADF](/pt/database/adf/) arquivo que armazena os dados em tabelas para vinculação com outros módulos, como recuperação automática de informações para uma chamada recebida.
Os arquivos ALF podem ser abertos com o software SwiftPage ACT no sistema operacional Windows.

## Formato de arquivo ALF - Mais informações

Os arquivos ALF são salvos como arquivos binários e seu formato de arquivo interno não está disponível.
