## O que é um arquivo AQ?

Um arquivo com extensão .aq é um arquivo de banco de dados criado com o aplicativo Ancestral Quest pela Incline Software, LC. Ele armazena informações da árvore genealógica usando os dados genealógicos inseridos por indivíduos usando a versão gratuita e premium do software. O banco de dados armazena informações como nomes, locais de nascimento e relacionamentos de membros da família. O Ancestral Quest usa essas informações para vincular e classificar informações como linhagem, descendentes, família e visualização de linha do tempo. Os arquivos AQ podem ser abertos usando o [Incline Software Ancestral Quest](https://www.ancquest.com/index.htm).

## Formato de arquivo AQ - Mais informações

Os arquivos AQ são salvos como arquivos binários e seu formato de arquivo interno não está disponível.
