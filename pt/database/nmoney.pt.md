## O que é um arquivo .NMONEY?

Um arquivo NMONEY é um arquivo de banco de dados de armazenamento de dados financeiros que contém registro de transações financeiras. Foi desenvolvido pela Nickvision e usado no software Nickvision Denaro, que é um aplicativo de software financeiro pessoal. Os dados armazenados dentro de um arquivo NOMONEY incluem informações de transações de crédito e débito em uma conta.

Arquivos NOMONEY podem ser abertos com o aplicativo [Github](https://github.com/NickvisionApps/Denaro).

## Sobre Denaro Software

Denaro foi inicialmente desenvolvido pela [Nickvision](https://nickvision.org/) como um produto que mais tarde foi convertido em um aplicativo de software de código aberto hospedado no [Github](https://github.com/NickvisionApps/Denaro) . Desde então, o aplicativo foi modificado e atualizado apenas no Github. O aplicativo é desenvolvido em C# na UI do Windows com Windows App SDK (WinUI 3 no momento).

### Recursos oferecidos por Denaro

* Gerencie várias contas simultaneamente com sua interface de guias mais popular e familiar
* Filtre transações por tipo, grupo ou data
* Repita transações como contas que ocorrem todos os meses
* Transferir dinheiro de uma conta para outra
* Exporte os dados de uma conta como um arquivo CSV e importe um arquivo CSV, OFX ou QIF para adicionar transações a uma conta

## Formato de arquivo Denaro – Mais informações

Os arquivos Denaro são salvos em disco em formato de arquivo binário. Os dados financeiros são armazenados como um arquivo de banco de dados no formato de arquivo **.SQLITE3**. SQLITE é um arquivo de banco de dados SQL leve criado com o software SQLite. Ele fornece um mecanismo de banco de dados independente que é capaz de fornecer um banco de dados completo e altamente confiável. Os arquivos de banco de dados SQLite podem ser usados para compartilhar conteúdos ricos entre sistemas, simplesmente trocando esses arquivos pela rede. Existem ligações SQLite para linguagens de programação como C, [C#](/pt/programming/cs/), [C++](/pt/programming/cpp/), [Java](/pt/programming/java/), [PHP](/pt/programming/php/) e muitos outros.
