## O que é um arquivo .CAF?

Um arquivo CAF é um arquivo de catálogo criado com a ferramenta de software de organização de mídia, **Cathy**. Ele contém uma lista de catálogos indexados que apontam para todas as mídias disponíveis em dispositivos externos, como discos flash e CDs. Esses arquivos de catálogo ajudam a manter uma lista limpa de mídia que é facilmente pesquisável e acessível. Os arquivos CAF oferecem pesquisa de mídia por nome, tamanho e data.

## Formato de arquivo CAF

Os arquivos CAF são armazenados em formato de arquivo de texto. Isso significa que você pode abrir arquivos CAF em qualquer editor de texto, como Notepad, Notepad++ e Apple TextEdit. Cathy disponibiliza o recurso de exportar a estrutura de diretórios de cada catálogo para o formato TXT, fazendo listas de pastas e arquivos.

### Abrir arquivo CAF

Você pode abrir o CAF com o software oficial Cathy. Para isso, você precisará colocar o arquivo CAF no mesmo diretório onde está o arquivo Cathy.exe. Cathy mostrará automaticamente o catálogo quando for lançado.

Para fazer isso, coloque o arquivo CAF no mesmo diretório do arquivo Cathy.exe. Então, abra Cathy. Seu catálogo CAF aparecerá em Cathy.
