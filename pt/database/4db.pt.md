## O que é um arquivo 4DB?

A estrutura de banco de dados construída usando 4th Dimension, um programa de design de software que compreende código 4D, interfaces e informações de recursos é conhecido como 4DB. Ele é usado na estrutura de design de software integrado de 4D somente durante o processo de desenvolvimento. Os arquivos do 4DB são encontrados tanto em plataformas desktop quanto mobile e o sistema operacional Windows também é uma opção. A classificação de popularidade dos principais arquivos 4DB é "Baixa", o que significa que os arquivos 4DB geralmente estão presentes em um dispositivo móvel ou desktop típico.

O arquivo de estrutura 4DB possui todas as especificações relacionadas à estrutura do banco de dados, como propriedades de campo, campos e tabelas, listas de opções, senhas, menus, métodos e formulários. O arquivo recebe o nome que você digita na caixa de diálogo de criação do banco de dados seguido do “.4db”.

Há duas maneiras de conectar um documento a um novo aplicativo (por exemplo, meu-arquivo.4DB). O primeiro e mais simples método é clicar com o botão direito do mouse no arquivo 4DB em questão. Selecione "Escolher programa padrão" no menu de queda e clique em "Procurar" para localizar o software desejado. Ao clicar em OK, você deve confirmar toda a operação. O próximo e mais complexo passo é usar o Registro do Windows para vincular o tipo de dados 4DB ao software apropriado.
