## O que é um arquivo FP5?

Um arquivo FP5 é um arquivo de banco de dados criado com o FileMaker Pro versão 5 ou 6. Ele armazena dados do usuário e da organização na forma de registros dentro de tabelas. O FileMaker Pro é um sistema de gerenciamento de banco de dados com suporte para os sistemas operacionais Macintosh e Windows. FP5 é um formato de arquivo mais antigo que era compatível com a versão do FileMaker Pro então disponível. O formato de banco de dados do FileMaker Pro mais recente é [FMP12](/pt/database/fmp12/) que foi introduzido com o lançamento do FileMaker Pro v 12. Outro formato de arquivo de banco de dados bem usado foi [FP7](/pt/database/fp7/) que foi substituído por FMP12.

## Formato de arquivo FP5 - Mais informações

Os detalhes do formato de arquivo interno do formato de arquivo FP5 não estão disponíveis publicamente para referência do desenvolvedor. Formatos de arquivo de banco de dados do FileMaker Pro mais antigos, como [FP5 e FP7, podem ser atualizados para o FMP12 mais recente](https://fmhelp.filemaker.com/help/16/fmp/en/index.html#page/FMP_Help/converting-files. html) usando as diretrizes detalhadas do FileMaker Pro.
