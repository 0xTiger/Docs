## O que é um arquivo .TEACHER?

Um arquivo .TEACHER é um arquivo de configurações educacionais criado pelo software SMART Response. É utilizado em ambientes educacionais para fornecer um ambiente completo de gerenciamento de informações de sala de aula, listas de turmas, horários, feriados, perfis de alunos, rotina de exames e resultados de avaliações. O software de avaliação SMART Response™ está instalado no computador que será usado para aplicar as avaliações SMART Response.

## Formato de arquivo PROFESSOR

Os arquivos TEACHER são armazenados em disco em formato de arquivo binário. Suas especificações internas de formato de arquivo não estão disponíveis publicamente para referência do desenvolvedor. O software SMART Response pode ser usado para gerar relatórios de alunos. Eles podem então ser compartilhados entre os computadores da escola.
