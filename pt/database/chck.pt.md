## O que é um arquivo CHCK?

Um arquivo CHCK é criado como resultado de verificações de monitoramento de integridade no Microsoft Exchange Server. Contém informações sobre o status geral e a operacionalização do banco de dados do servidor. O Microsoft Exchange Server é usado principalmente como servidor de e-mail e permite manter seus contatos comerciais e calendários. Os administradores do servidor Exchange usam seu EseUtil para verificar a integridade do banco de dados.

## Formato de arquivo CHCK

O formato de arquivo interno do arquivo CHCK não é conhecido e não há material de referência disponível para leitura/gravação de arquivos CHCK.
