## O que é um arquivo DAT?
Um arquivo com extensão .dat pertence a um formato de arquivo genérico. Normalmente, eles não são criptografados nem binários, mas podem ser quase qualquer coisa. Vários aplicativos estão usando os arquivos DAT como referências para determinadas operações em seus respectivos programas. Portanto, não podemos dizer que um software ou programa específico é responsável por criar arquivos DAT. Esses arquivos são particularmente acessados apenas pelo aplicativo que os criou.


## Formato de arquivo DAT
O formato de arquivo DAT é simplesmente um formato geral que contém qualquer tipo de dados em um arquivo, geralmente salvo com extensão .dat. Pode conter dados em formato binário ou texto, geralmente um dado genérico consiste nas informações específicas da aplicação a que se refere. Na maioria das vezes, você pode encontrá-los com outros arquivos de configuração, como arquivos DLL. Aqui podemos listar alguns tipos de arquivos famosos, com extensão de arquivos .dat:

### Arquivo de vídeo VCD
Os VCDs eram fontes de vídeo ou multimídia bem conhecidas na década de 1990 para distribuir filmes e vídeos caseiros antes dos DVDs. Normalmente, os arquivos de vídeo do disco VCD foram encontrados com extensão .dat. Assim, ele armazenava os dados de vídeo reais reproduzidos do VCD.
### Anexo de e-mail do Exchange
Os arquivos DAT relacionados ao anexo de email do Exchange contêm informações de formatação para mensagens Rich Text (RTF) enviadas do Microsoft Exchange Server. Normalmente, esses arquivos são adicionados ao final dos e-mails quando o cliente de e-mail de um destinatário não oferece suporte a mensagens RTF. Esses arquivos geralmente são encontrados como **Winmail.dat**.
### Arquivo Hive do Registro do Windows
NTUSER.DAT é um arquivo hive de registro comum. O registro do Microsoft Windows usa arquivos DAT para armazenar as configurações do usuário. Esses arquivos DAT são carregados quando um usuário efetua login no Windows. Pode incluir configurações de conta, senhas e associações de arquivos.
### Arquivo de dados do Minecraft
O jogo Minecraft está usando os arquivos DAT para salvar as informações sobre o inventário do jogador, a posição do jogador e as informações do mundo. Os arquivos DAT também são usados para armazenar níveis, que são pequenos pedaços de mapa que são carregados sob demanda conforme o jogador navega pelo mundo.
### Arquivo DAT Piriforme
Os aplicativos como CCleaner, Speccy, Defraggler e Recuva estão usando arquivos DAT para obter informações sobre suas operações típicas.
