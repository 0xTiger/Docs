## O que é um arquivo ADB?

Um arquivo com extensão .adb é um arquivo de banco de dados criado com [Alpha Five](https://www.alphasoftware.com/) (agora conhecido como Alpha Anywhere). Ele contém os nomes de arquivo de todas as tabelas e conjuntos que são membros da área de trabalho do projeto. Além disso, contém os nomes de todas as bibliotecas incorporadas na área de trabalho. Alpha Anywhere é um IDE usado para desenvolver aplicações web. Além dos arquivos ADB, a área de trabalho do Alpha Anywhere pode conter muitos arquivos que iniciam o nome da tabela ou do conjunto, mas possuem extensões de arquivo diferentes.
