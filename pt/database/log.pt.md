## O que é um arquivo LOG?
Um arquivo com extensão .log contém a lista de texto simples com carimbo de data/hora. Normalmente, certos detalhes de atividades são registrados pelos softwares ou sistemas operacionais para ajudar os desenvolvedores ou usuários a rastrear o que estava acontecendo em um determinado período de tempo. Os usuários podem editar esses arquivos facilmente usando qualquer editor de texto. Normalmente, os relatórios de erros ou atividades de login são registrados pelos sistemas operacionais, mas outros softwares ou servidores da Web também geram arquivos de log para rastrear visitantes e monitorar o uso da largura de banda.

## Formato do arquivo LOG
O formato de arquivo LOG registra os eventos típicos que ocorrem em um sistema operacional, mensagens entre diferentes usuários de um software de comunicação ou qualquer outro software executado. Simplesmente podemos dizer que certas mensagens em determinados timestamps são registradas em um arquivo LOG. Alguns termos comumente usados para registro são:
### Registros de eventos
Registra eventos ocorridos na execução de um sistema para rastrear e entender a atividade do sistema e diagnosticar problemas. Eles são importantes para identificar as atividades de sistemas complexos, normalmente no caso de aplicativos com muito pouca interação do usuário.
### Registros de transações
Quase todos os sistemas de banco de dados mantêm o log de transações, que geralmente não é uma trilha de auditoria para análise posterior e não é legível por humanos. Esses logs apenas mantêm o registro de alterações nos dados existentes para permitir que o banco de dados se recupere de falhas ou outros erros de dados e mantenha os dados em um estado consistente. Portanto, os sistemas de banco de dados geralmente têm logs de eventos gerais e logs transacionais.
### Registros de mensagens
A comunicação textual salva pelo Internet Relay Chat (IRC), programas de mensagens instantâneas (IM), clientes de compartilhamento de arquivos ponto a ponto com funções de bate-papo e jogos multiplayer (especialmente MMORPGs) são chamados de logs de mensagens. Esses logs geralmente são salvos em arquivos de texto simples, mas os clientes de IM e VoIP (por exemplo, Skype) podem salvá-los em arquivos HTML para facilitar a leitura ou habilitar a criptografia.
### Registros do servidor
O log do servidor é, na verdade, um arquivo de log contendo uma lista de atividades executadas e criadas ou mantidas automaticamente pelo próprio servidor. Normalmente, esses arquivos não são acessíveis a usuários gerais da Internet, apenas ao webmaster ou outra pessoa administrativa de um serviço de Internet.
