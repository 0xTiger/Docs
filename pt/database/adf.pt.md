## O que é um arquivo ADF?

Um arquivo com extensão .adf é um arquivo de banco de dados criado com o software ACT CRM (Customer Relationship Management) para armazenar dados de relacionamento sobre clientes. Ele armazena informações sobre o nome do cliente, e-mail, contato, informações comerciais e histórico de relacionamento. Os sistemas de CRM são mais frequentemente integrados aos call centers estabelecidos pelas empresas para recuperar automaticamente as informações do usuário para as chamadas recebidas. Os arquivos ADF podem ser abertos com o software SwiftPage ACT no sistema operacional Windows.

## Formato de arquivo ADF - Mais informações

Os arquivos ADF são salvos em disco como arquivos binários e suas informações de estrutura interna não estão disponíveis publicamente. Outro arquivo com extensão .alf é salvo junto com os arquivos ADF que contêm metadados necessários para navegar, fazer backup e restaurar o ACT! base de dados.
