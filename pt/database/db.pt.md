## O que é um arquivo DB?

Um arquivo com extensão db é um arquivo de banco de dados genérico para armazenar dados. Não há especificações oficiais a serem seguidas para esse arquivo de banco de dados. Os dados são organizados em um formato estruturado dentro do arquivo nas tabelas de formulário, campos, tipos de dados e valores de campo. Arquivos de banco de dados também podem ser criados lendo arquivos de folha de dados com dados em linhas e colunas. O conteúdo dos arquivos DB pode ser exportado para os formatos de arquivo [CSV](/pt/spreadsheet/csv/) e [PDF](/pt/pdf/) para fins de relatório. Arquivos de banco de dados podem ser abertos em aplicativos como SQLite e LibreOffice.
