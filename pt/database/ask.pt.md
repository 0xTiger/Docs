## O que é um arquivo ASK?

Um arquivo com extensão .ask é um arquivo de banco de dados criado por askSam Systems, um aplicativo gerenciador de informações pessoais (PIM). Esses tipos de arquivos são armazenados em pastas dinâmicas e contêm dados do usuário, como mensagens de e-mail, documentos, arquivos de texto, planilhas, endereços e páginas da web. Isso permite que o software askSam os pesquise facilmente e recupere os dados para fins de pesquisa. O software askSam, apesar de ser um aplicativo de desktop simples baseado em formulário que armazena dados em banco de dados, era popular para organizar e recuperar esses dados. Os dados do arquivo ASK também podem ser exportados para [planilhas](/pt/spreadsheet/).

## Formato de arquivo ASK - Mais informações

Os arquivos ASK são salvos em disco em formato de arquivo binário proprietário cujos detalhes de formato de arquivo interno não estão disponíveis para referência do desenvolvedor.
