## O que é um arquivo DB3?
O arquivo DB3 é um arquivo de banco de dados criado pelo software SQLite, que é um programa de banco de dados leve e independente que cria bancos de dados usando arquivos simples; contém a anatomia do banco de dados, bem como registros de dados; comumente usado para recuperar ou armazenar dados estruturados usando SQL. Esses arquivos podem ser usados em dispositivos inteligentes ou onde a manutenção de registros ou outro gerenciamento de dados é necessário, mas em um ambiente de pouco espaço.


## Formato de arquivo DB3
O formato de arquivo DB3 está associado ao RDBMS (Relational Database Management System) SQLite, uma escolha popular para banco de dados incorporado. Não há nenhuma extensão de arquivo específica definida em um SQLite_3 em sua especificação, ele pode usar extensões incluindo [.dbf](/pt/database/dbf/) e [.sqlite](/pt/database/sqlite/).

### A Secificação do Banco de Dados
Comumente, o formato de arquivo db relacionado ao SQLite, versão 3, pode ser considerado como o formato de arquivo DB3 usado como o formato nativo documentado publicamente para o mecanismo de banco de dados SQLite desde junho de 2004. O formato de arquivo DB3 é uma plataforma cruzada, transferível entre big-endian e arquiteturas little-endian ou sistemas de 32 e 64 bits. Esses recursos tornam o DB3 uma escolha popular como formato de arquivo de aplicativo. O arquivo principal do banco de dados SQLite_3 (DB3) consiste em uma ou mais páginas. Todos os tamanhos de todas as páginas são iguais no mesmo banco de dados. O tamanho da página em bytes é uma potência de dois entre 512 e 65536 inclusive.
