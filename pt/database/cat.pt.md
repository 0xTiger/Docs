## O que é um arquivo .CAT?

O arquivo CAT pertence ao software Advanced Disk Catalog, desenvolvido pela ElcomSoft Co. Ltd, empresa de software russa que desenvolve várias ferramentas de segurança de computadores e software forense. Alguns de seus produtos incluem software para recuperação de senha, aquisição de dispositivos móveis e extração em nuvem. A empresa foi fundada em 1990 e possui escritórios na Rússia e nos Estados Unidos. Eles são bem conhecidos por seu software forense e, em geral, seu software é usado por empresas privadas e agências governamentais para realizar análises e investigações forenses.

Advanced Disk Catalog is a cataloguing program for Windows XP/2003. A versatilidade do ADC deriva do fato de não depender de um mecanismo de banco de dados, o que o torna altamente eficiente e compacto. Sua interface de usuário, semelhante ao Windows Explorer, permite fácil organização de dados armazenados em vários tipos de mídia, como discos rígidos (incluindo unidades de rede), disquetes, discos ópticos, discos ZIP e JAZ, etc.

## Como abrir os arquivos CAT?

Baixe o software Elcomsoft Advanced Disk Catalog. No Windows, você pode clicar com o botão direito no explorador de arquivos e selecionar abrir com menu de contexto, especificar o caminho do software Elcomsoft Advanced Disk Catalog para abrir o arquivo cat.
