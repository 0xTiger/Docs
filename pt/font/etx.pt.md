## O que é um arquivo .ETX?

Um arquivo ETX é um arquivo de fonte usado pelos sistemas de composição tipográfica TeX e LaTeX. Ele é salvo como um arquivo de texto e define a aparência de uma fonte por meio de sua codificação. Os arquivos de fonte ETX são carregados em software que fornece a capacidade de trabalhar com documentos Tex. Uma vez carregados, os arquivos de fonte ETX podem ser usados para customização de documentos TeX.
