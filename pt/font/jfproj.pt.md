## O que é um arquivo JFPROJ?

Um arquivo com extensão .jfproj é uma fonte personalizada baseada em JSON criada com o aplicativo Microsoft Font Maker no Windows 10. Eles podem ser criados usando caneta eletrônica com dispositivos de tela sensível ao toque, como tablets, embora outros periféricos de computador, como mouse, também possam ser usados no caso de computadores. Os arquivos JFPROJ contêm os glifos usados para renderizar os caracteres na tela definindo o espaçamento entre caracteres e palavras e o tamanho da fonte. O aplicativo Microsoft Font Maker está disponível para download na loja do Windows.
