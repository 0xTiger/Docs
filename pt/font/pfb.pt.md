## O que é um arquivo PFB?

Um arquivo com extensão .pfb é uma versão Printer Font Binary da fonte Type 1 da Adobe. É semelhante aos arquivos .pfa(/pt/font/pfa/), mas são de natureza binária. Os arquivos PFB armazenam fontes Adobe Type 1 usadas para o sistema operacional Windows. Estes contêm os dados do glifo da fonte e armazena os dados de forma compacta em formato binário. Além de ter conteúdo codificado em binário, os arquivos PFB são criptografados. Os arquivos PFB também exigiam arquivos .pfm que incluem medidas de caracteres dentro da fonte. Para instalar as fontes PFB, os arquivos .pfb e .pfm precisam ser colocados no sistema.
