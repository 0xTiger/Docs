## O que é um arquivo .VLW?

Um arquivo VLW é um formato de fonte usado pelo software de criação de imagens e animações, **Processing**. Ele armazena fontes vetoriais de forma compacta que pode ser dimensionada para qualquer tamanho sem perder qualidade. Isso torna seu uso em design gráfico e visual. O processamento fornece a capacidade de criar imagens e animações, onde esses glifos de fonte VLW são usados.

## Formato de arquivo VLW - Mais informações

Os arquivos VLW são salvos em disco como arquivos binários. Em Processing, você pode usar a função **createFont()** para criar um objeto de fonte a partir de um arquivo VLW. Você pode então usar este objeto de fonte para desenhar texto em seus esboços usando a função **text()**. O processamento pode ler arquivos VLW usando o comando **loadFont()**.
