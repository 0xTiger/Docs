## O que é um arquivo .GXF?

Um arquivo GXF é um arquivo de fonte da ferramenta de design CAD para Windows, General CADD Pro. O software usa a fonte para inserir texto e outros símbolos em um desenho CAD. As fontes GXF devem estar disponíveis no software General CADD Pro para que um desenho CAD seja exibido corretamente.

Você pode abrir um arquivo GXF usando [General CADD Pro](https://www.generalcadd.com/oldSite/101release.htm).

## Sobre Geral CADD Pro

Geral CADD Pro oferece estrutura de comando de duas letras que é incorporada e usada por muitos profissionais de desenho. Suporta o formato de arquivo ACAD 2013. Ele também fornece suporte para comandos de rumo/distância do topógrafo e comando Campo para Concluir.
