## O que é um arquivo .SPARC?

Um arquivo .sparc, alternativamente reconhecido como Skype Plugin Archive, serve como um pacote que abrange recursos extras ou aprimoramentos para o aplicativo Skype. Esses plug-ins têm a capacidade de introduzir uma ampla gama de recursos, como gravação de chamadas, compartilhamento de tela, tradução,
Jogos e entretenimento, emoticons e temas personalizados.

## Formato de arquivo SPARC – Mais informações

Em certas versões do Skype, o Gerenciador de Extras é um componente opcional que pode ser integrado à instalação do Skype. Se você encontrar dificuldades para instalar um arquivo SPARC, é possível que você tenha omitido o Gerenciador de Extras durante a instalação inicial. Para resolver esse problema, reinstale o Skype e certifique-se de selecionar a opção Gerenciador de Extras.

## Como instalar o plug-in do Skype?

Etapas para instalar plug-ins do Skype:

1. Obtenha o arquivo .sparc correspondente ao plugin desejado.
2. Feche o Skype se ele estiver em execução.
3. Localize o arquivo .sparc baixado e clique duas vezes nele.
4. Siga as instruções na tela para concluir a instalação do plugin.
5. Após a instalação, reinicie o Skype para ativar o plugin.
