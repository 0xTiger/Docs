## O que é um arquivo .CLEO?

Um arquivo .cleo é um arquivo de plug-in usado como parte da biblioteca CLEO para a série de jogos GRand Theft Auto. É usado para executar modificações que alteram ou expandem a jogabilidade. O arquivo CLEO armazena uma biblioteca no formato de arquivo DLL, adicionando comandos de script úteis (opcodes) para trabalhar com arquivos externos para alterar a memória do jogo. Existem diferentes versões do CLEO feitas para GTA III, GTA Vice City e GTA San Andreas.

## Formato de arquivo CLEO

CLEO 4 permite criar novos opcodes usando a fonte do CLEO SDK. Três exemplos mais comuns de tais arquivos de plug-in incluem:

 * IniFiles.cleo - opcodes para arquivos INI;
 * FileSystemOperations.cleo - opcodes para arquivos e pastas.
 * IntOperations.cleo - opcodes para operações lógicas (AND, OR, XOR, etc).

Os plug-ins são instalados de forma semelhante aos scripts. Para instalar um plugin CLEO, você precisa copiar o arquivo .CLEO para a pasta CLEO.

## Sobre CLEO

CLEO é um plugin extensível muito popular para a série de jogos Grand Theft Auto da Rockstar Games, permitindo o uso de milhares de mods exclusivos que alteram ou expandem a jogabilidade.
