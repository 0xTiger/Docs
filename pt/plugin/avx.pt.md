## O que é um arquivo .AVX?

AVX é uma extensão de software utilizada por produtos Avid como Media Composer e Xpress. Ele aprimora a funcionalidade do software incorporando efeitos de vídeo personalizados e suportando a importação de novos formatos de arquivo. Esta extensão permite que desenvolvedores terceirizados expandam os principais recursos do aplicativo.

## Como abrir o arquivo AVX?

Você pode abrir o arquivo AVX usando **Avid Media Composer** ou **Avid Xpress**.
