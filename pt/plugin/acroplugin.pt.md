## O que é um arquivo .ACROPLUGIN?

Um arquivo com extensão .acroplugin é um arquivo de plugin usado pelo Adobe Acrobat. Ele é usado para adicionar funcionalidade aos recursos existentes do aplicativo. O plug-in vem com o Adobe Acrobat, um programa de layout de página e publicação [PDF file](/pdf/). Como outros plugins, o arquivo ACROPLUGIN estende a funcionalidade existente oferecida pelo aplicativo.

## Formato de arquivo ACROPLUGIN - Mais informações

O arquivo ACROPLUGIN é usado explicitamente pelo Adobe Acrobat para estender sua funcionalidade existente. Os detalhes internos do formato do arquivo não estão disponíveis publicamente para referência do desenvolvedor.

## Como abrir o arquivo ACROPLUGIN?

Você pode abrir o arquivo ACROPLUGIN usando Adobe Acrobat.
