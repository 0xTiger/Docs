## O que é um arquivo .AMXX?

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. É uma plataforma de script que permite aos administradores de servidores personalizar e ampliar a funcionalidade de seus servidores. AMXX é usado em conjunto com o sistema de gerenciamento de plugins MetaMod que permite que vários plugins sejam executados simultaneamente em um servidor sem conflitos. Os recursos oferecidos pelo plugin AMXX incluem comandos de administrador, modificações de jogo e gerenciamento de servidor.

## Formato de arquivo AMXX - Mais informações

Arquivos AMXX são plug-ins salvos em formatos de arquivo proprietários. A versão anterior do plugin AMXX é AMX que foi usada pelo AMX Mod, o antecessor do AMX Mod X. As versões Half-Life suportadas pelo AMX Mod X incluem Counter-Strike, Day of Defeat, Natural Selection, Team Fortress Classic, The Specialists, e Forças Especiais da Terra.
