## O que é um arquivo .RPLN?

Um arquivo RPLN é um arquivo de plug-in usado pelo programa de layout de página, **InDesign**, que cria design de página e também fornece publicação impressa. ele contém funções necessárias para execução e está localizado em um diretório Obrigatório onde o software InDesign está instalado.

## Formato de arquivo RPLN

Os arquivos RPLN são salvos no formato de arquivo proprietário do InDesign e não há mais detalhes sobre a estrutura interna dos arquivos RPLN.

## Sobre o Adobe InDesign

Adobe InDesign é um aplicativo de software profissional de editoração eletrônica amplamente utilizado para a criação de mídia impressa e digital. O InDesign oferece suporte ao uso de plug-ins, que são componentes de software adicionais que podem ser adicionados para ampliar a funcionalidade do programa.

## Como abrir o arquivo RPLN?

Você pode abrir arquivos RPLN com Adobe InDesign 2023.
