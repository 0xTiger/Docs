O que é um arquivo .MDE?

Um arquivo .MDE é uma versão compilada de um banco de dados Access (.mdb ou .accdb). Quando você compila um banco de dados do Access em um arquivo .MDE, ele não pode mais ser editado usando as ferramentas de design padrão do Access. O objetivo principal da criação de um arquivo .MDE é distribuir um aplicativo de banco de dados sem expor o código-fonte original.

## Formato de arquivo MDE

Um arquivo MDE é criado compilando o código VBA, formulários, relatórios e outros objetos. Isso resulta na criação do formato de arquivo binário MDE. O arquivo .MDE resultante pode ser distribuído para usuários que podem executar o aplicativo, mas não podem modificar o design ou visualizar o código original. O arquivo MDE é na verdade a versão compilada de um [arquivo .MDA](/database/mda/). A distribuição de arquivos .MDE pode ajudar a proteger a propriedade intelectual, impedindo que os usuários acessem ou modifiquem o código-fonte. Também pode melhorar o desempenho à medida que o código é compilado e otimizado.
