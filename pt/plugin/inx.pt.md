## O que é um arquivo .INX?

Um arquivo INX é um arquivo de plug-in usado pelo editor de gráficos vetoriais de código aberto, **Inkscape**. Ele contém informações sobre o programa, incluindo o nome e a localização da funcionalidade no menu. O arquivo INX também especifica um script para a extensão que inclui informações como tipo de script, intérprete, localização e argumentos variáveis.

## Formato de arquivo INX

Os arquivos INX melhoram a funcionalidade do Inkscape incorporando recursos como importadores de arquivos e filtros gráficos. Armazenados em formato XML, esses arquivos são normalmente encontrados no diretório \​extensions\​ dentro de uma instalação do Inkscape.

## Como abrir arquivos INX?

Você pode abrir arquivos INX usando o programa **Inkscape**.
