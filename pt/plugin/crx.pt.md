## O que é um arquivo CRX?

Um arquivo CRX é uma extensão usada para estender a funcionalidade dos navegadores de Internet Google Chrome, Microsoft Edge e Firefox. Adiciona funcionalidade específica para acesso direto do navegador para o qual o arquivo do plugin foi desenvolvido. As extensões são criadas para diferentes finalidades, como análise de tráfego de sites, bloqueador de anúncios, gerenciador de senhas, analisador de listagem de comércio eletrônico, tradutor de páginas da web e muito mais.

## Formato de arquivo CRX

Os arquivos CRX são armazenados como um arquivo binário compactado com seus arquivos constituintes compactados dentro do arquivo. Ele pode conter arquivos [.JS](/pt/web/js/), [.JSON](/pt/web/json/), imagens ou até mesmo arquivos executáveis. Navegadores como o Google Chrome também permitem carregar extensões no modo de desenvolvedor, o que permite uma depuração adicional da funcionalidade da extensão.

## Como instalar uma extensão da Web?

Navegadores da Web como o Chrome possuem uma opção de menu “Extensões” que permite carregar arquivos de extensão CRX. Quando uma extensão é carregada do armazenamento de extensões online, ele baixa o conteúdo do arquivo de extensão, descompacta-o e instala a extensão para carregar a funcionalidade desejada. Todo esse processo não requer nenhuma interação do usuário para instalar o arquivo CRX.

## Como carregar arquivos crx no Chrome?

Para abrir um arquivo CRX:

  1. Vá para o menu de extensões no navegador Google Chrome
  1. Arraste e solte o arquivo CRX na página de extensão do navegador
  1. Clique em `Adicionar extensão` quando solicitado
