## O que é um arquivo .APLG?

Um arquivo com extensão .aplg é um arquivo de plug-in usado pelo software de gravação de fluxo de Internet Audials. O plugin é carregado pelo software para adicionar novos recursos, como novas funções de pesquisa e gravação, para capturar músicas e vídeos da Internet. Audials oferece um reprodutor de rádio rico em recursos que permite aos usuários acessar e ouvir um grande número de estações de rádio na Internet.

## Formato de arquivo APLG

Os arquivos APLG vêm embalados nos pacotes de plug-ins .APLP, o que significa que o software Audials pode carregar vários plug-ins do pacote ao mesmo tempo. Muitas vezes, os usuários podem gravar suas músicas ou programas de rádio favoritos para reprodução posterior. Isto pode ser útil para criar listas de reprodução personalizadas. Audials normalmente mantém um grande banco de dados de estações de rádio, facilitando a descoberta de novos conteúdos pelos usuários.
