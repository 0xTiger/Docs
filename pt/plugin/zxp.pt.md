## O que é um arquivo .ZXP?

Um arquivo ZXP serve como uma extensão, aumentando as funcionalidades de softwares Adobe como Adobe Photoshop ou Adobe After Effects. No passado, o Adobe Extension Manager facilitava a instalação de arquivos ZXP. Atualmente, o processo de instalação requer a utilização da ferramenta de linha de comando Unified Plugin Installer Agent (UPIA), que vem junto com o aplicativo de desktop Adobe Creative Cloud.

## Formato de arquivo ZXP

O termo ZXP significa Pacote ZIP eXtension. Os arquivos ZXP são usados para empacotar e distribuir extensões para aplicativos Adobe Creative Cloud, como Adobe Photoshop, Illustrator, InDesign e outros. Um arquivo ZXP é compactado usando o formato [ZIP](/compression/zip/), o que facilita a distribuição e instalação de extensões.

Dentro de um arquivo ZXP, você normalmente encontrará os arquivos necessários para a extensão, incluindo [HTML](/web/html/), [CSS](/web/css/), JavaScript, imagens e quaisquer outros ativos necessários para aprimorar ou adicionar recursos ao software Adobe Creative Cloud.

### Como instalar o arquivo ZXP?

Os arquivos ZXP são instalados usando o Adobe Extension Manager ou o aplicativo Creative Cloud Desktop. Os usuários podem simplesmente clicar duas vezes em um arquivo ZXP e o gerenciador de extensões cuidará do processo de instalação.

Os arquivos ZXP são projetados para funcionar em diferentes sistemas operacionais, incluindo Windows e macOS, tornando conveniente para os desenvolvedores criarem extensões que podem ser usadas por um público amplo.

### Segurança do formato de arquivo ZXP

Os arquivos ZXP podem ser assinados digitalmente pelos desenvolvedores, garantindo que a extensão não foi adulterada e vem de uma fonte confiável. Isso ajuda a manter a segurança e a integridade dos aplicativos da Adobe Creative Cloud.

Se você estiver trabalhando com extensões da Adobe Creative Cloud ou desenvolvendo uma por conta própria, poderá encontrar arquivos ZXP como meio de empacotar e distribuir sua extensão. No entanto, as especificidades do trabalho com arquivos ZXP podem estar sujeitas a alterações ou atualizações pela Adobe, portanto, é uma boa ideia consultar a documentação mais recente da Adobe para obter informações mais precisas e atualizadas.
