## O que é um arquivo Q1Q?

Um arquivo .q1q é um arquivo plugin do Photoshop desenvolvido pela Flaming Pear Software para a criação de planetas 3D. Ele permite que você baixe dados de nuvens AO VIVO, ar, textura, clima e outras toneladas de detalhes. Ele também permite exportar todos os mapas de textura para arquivos multicamadas e gerar mapas para animação 3D e criação de mapas.

## Formato de arquivo Q1Q

Os arquivos do plug-in Q1Q são salvos no formato de arquivo proprietário Flaming Pear e seus detalhes internos de formato de arquivo não estão disponíveis publicamente. A imagem a seguir mostra a interface do usuário do plugin Q1Q quando carregado no Photoshop.

## Como abrir o arquivo Q1Q?

Você pode abrir arquivos Q1Q usando o **Adobe Photoshop 2023** e o **Corel PaintShop Pro 2023**.
