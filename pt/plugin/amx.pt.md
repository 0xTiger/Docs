## O que é um arquivo .AMX?

Um arquivo AMX é um arquivo de plug-in personalizado para o plug-in metamod AMX Mod para Valve Half-Life e jogos relacionados. Ele adiciona recursos e funcionalidades adicionais aos recursos existentes do jogo.

## Formato de arquivo AMX – Mais informações

O Mod AMX usa a extensão de arquivo AMX. No entanto, a versão estendida do plugin usa a extensão de arquivo .AMXX.

## Como abrir o arquivo AMX?

Você pode abrir arquivos AMX usando [AMX Mod X](https://www.amxmodx.org/) nos sistemas operacionais Windows e Linux.
