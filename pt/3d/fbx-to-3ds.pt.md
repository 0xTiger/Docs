## Sobre FBX para 3DS

FilmBox, comumente referido como FBX, é um formato de arquivo 3D amplamente adotado, originalmente desenvolvido por Kaydara para facilitar a compatibilidade com MotionBuilder. A Autodesk Inc. assumiu o controle deste formato em 2006, estabelecendo-o como um padrão chave para a troca contínua de dados 3D entre diversos aplicativos de software 3D.

Um arquivo com a extensão [.3ds](/pt/3d/3ds/) significa o formato de arquivo de malha 3D Studio (DOS) empregado pelo software 3D Studio da Autodesk. A presença da Autodesk na área de formatos de arquivo 3D remonta à década de 1990 e desde então evoluiu para o 3D Studio MAX, uma ferramenta abrangente para modelagem, animação e renderização 3D. O arquivo .3ds encapsula dados para representação tridimensional de cenas e imagens, tornando-o um dos formatos de arquivo preferidos para importação e exportação de dados 3D.

A conversão de [FBX](/pt/3d/fbx/) para 3DS é um requisito frequente dos usuários. Existem várias ferramentas gratuitas disponíveis online para conversão de FBX para 3DS GRATUITAMENTE.
