## Sobre FBX para GLTF

FilmBox, comumente conhecido como FBX, é um formato de arquivo 3D amplamente adotado, inicialmente criado por Kaydara para complementar o MotionBuilder. Em 2006, a Autodesk Inc. assumiu o comando, impulsionando o FBX para uma posição de destaque como formato primário de troca de dados 3D, adotado por uma infinidade de ferramentas de software 3D.

glTF, abreviação de GL Transmission Format, serve como um formato de arquivo 3D projetado para encapsular dados de modelos 3D na notação JSON. Esta implementação com JSON serve para reduzir eficientemente as dimensões dos ativos 3D, bem como diminuir a sobrecarga computacional necessária para sua extração e utilização durante o tempo de execução.

É um requisito comum converter [FBX](/pt/3d/fbx/) para o formato de arquivo GLTF e vários aplicativos estão disponíveis online que podem converter FBX para GLTF GRATUITAMENTE. Se você é um desenvolvedor de aplicativos interessado em incorporar a funcionalidade FBX ao GLTF em seu aplicativo .NET, Java ou Python, você pode considerar o uso das APIs Aspose.3D para essa finalidade.
