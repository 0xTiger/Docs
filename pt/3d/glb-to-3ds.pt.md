## Sobre GLB para 3DS

O uso de arquivos GLB em formatos de arquivo 3D não é algo novo. GLB é um formato de arquivo binário 3D comumente usado que representa modelos 3D no formato GL Transmission, ou seja, ([glTF](/pt/3d/gltf/)). [3DS](/pt/3d/3ds/), por outro lado, representa o formato de arquivo de malha 3D Sudio (DOS) usado pelo Autodesk 3D Studio. Vários aplicativos estão disponíveis online que podem ser usados para conversão de [GLB](/pt/3d/glb/) para [3DS](/pt/3d/3ds/) GRATUITAMENTE. Se você é um desenvolvedor de aplicativos que gostaria de adicionar a funcionalidade do GLB ao 3DS em seu .NET ou Java, você pode usar a API Aspose.3D para fazer isso.
