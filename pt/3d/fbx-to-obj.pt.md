## Sobre FBX para OBJ

FBX, FilmBox, é um formato de arquivo 3D popular desenvolvido por Kaydara para MotionBuilder. Foi adquirido pela Autodesk Inc em 2006 e hoje é um dos principais formatos de arquivo de troca de dados 3D usado por muitas ferramentas 3D. Arquivos [OBJ](/pt/3d/obj/) são usados pelo aplicativo Advanced Visualizer do Wavefront para definir e armazenar os objetos geométricos. É um requisito comum converter [FBX](/pt/3d/fbx/) para o formato de arquivo OBJ e vários aplicativos estão disponíveis online que podem converter FBX para OBJ GRATUITAMENTE. Se você é um desenvolvedor de aplicativos interessado em incorporar a funcionalidade FBX ao OBJ em seu aplicativo .NET, Java ou Python, você pode considerar o uso das APIs Aspose.3D para essa finalidade.
