## O que é arquivo RVM?

Os arquivos de dados **RVM** estão relacionados ao AVEVA PDMS. **RVM** é um arquivo de projeto do modelo do sistema de gerenciamento de design da planta AVEVA. O Plant Design Management System (PDMS) da AVEVA é o sistema de projeto 3D mais popular que usa tecnologia centrada em dados para gerenciar projetos. Vários aplicativos estão disponíveis para abrir e converter arquivos RVM para outros formatos, como [3DS](/pt/3d/3ds/).

## Formato de arquivo RVM

Não há muitos detalhes sobre o formato de arquivo RVM, exceto que ele pode ser salvo no formato binário e ASCII. O arquivo RVM suporta entidades como:

* Toda a geometria
* Atributos armazenados em grupos
* Texturas (via arquivo RVS)
* Câmeras e trilhas de câmera (via arquivo RVS)
* Planos de clipe (via arquivo RVS)
* Sinais (via arquivo RVS)
* Tags (via arquivo RVS)
* Etiquetas (via arquivo RVS)
* Translucidez (via arquivo RVS)
* Pontos de origem PDMS
