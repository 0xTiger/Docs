## Sobre FBX to STL

FBX, FilmBox, é um formato de arquivo 3D popular desenvolvido por Kaydara para MotionBuilder. Foi adquirido pela Autodesk Inc em 2006 e hoje é um dos principais formatos de arquivo de troca de dados 3D usado por muitas ferramentas 3D. [STL](/pt/cad/stl/), abreviatura de estereolitrografia, é um formato de arquivo intercambiável para representação de geometria de superfície tridimensional.

É um requisito comum converter [FBX](/pt/3d/fbx/) para o formato de arquivo STL e vários aplicativos estão disponíveis online que podem converter FBX para STL GRATUITAMENTE. Se você é um desenvolvedor de aplicativos interessado em incorporar a funcionalidade FBX para STL em seu aplicativo .NET, Java ou Python, você pode considerar o uso das APIs Aspose.3D para essa finalidade.
