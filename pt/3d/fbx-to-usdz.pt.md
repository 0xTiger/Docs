## Sobre FBX para USDZ

FBX, abreviação de FilmBox, é um formato de arquivo 3D amplamente adotado, inicialmente criado por Kaydara para MotionBuilder. A Autodesk Inc. assumiu a propriedade em 2006, consolidando seu status como principal formato de troca de dados 3D utilizado por uma infinidade de ferramentas de software 3D.

Um arquivo com a extensão .usdz representa um arquivo ZIP descompactado e não criptografado adaptado para o formato de arquivo [USD](/pt/3d/usd/) (Universal Scene Description). Ele abriga proxies para vários tipos de arquivos, incluindo texturas e animações, todos convenientemente encapsulados no arquivo.

Se você é um desenvolvedor de aplicativos que deseja incorporar a funcionalidade [FBX](/pt/3d/fbx/) ao USDZ em seu aplicativo .NET, Java ou Python, você pode querer explorar as APIs Aspose.3D como um recurso valioso para alcançar este objetivo.
