## O que é um arquivo DBM?

Um arquivo DBM (Daz Brick Material) é um arquivo de rede Shader Mixer criado com o aplicativo de modelagem de caracteres 3D, **DAZ Studio**. Descreve o material do modo 3D, ou seja, superfície/deslocamento/volume interior. O DAZ permite que você crie seu próprio shader personalizado conectando uma série de tijolos. Isso combina coletivamente como um arquivo DBM que forma uma rede de sombreadores. É o arquivo DBM que especifica como um material é sombreado.

Os aplicativos que podem **abrir arquivos DBM** incluem DAZ 3D DAZ Studio 4.

## Formato de arquivo DBM

Os arquivos DBM são salvos no disco no formato de arquivo [XML](/pt/web/xml/).
