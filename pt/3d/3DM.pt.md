## O que é um arquivo 3DM?

Um arquivo com extensão .3dm é um formato de arquivo de código aberto usado para softwares gráficos 3D. Ele contém modelos 3D junto com seus elementos dependentes, como informações de superfície, pontos e curvas. O formato de arquivo 3DM foi desenvolvido pela iniciativa [openNURBS](https://github.com/mcneel/opennurbs) para transferir com precisão a geometria 3D entre aplicativos. Os arquivos 3DM podem ser abertos e convertidos com aplicativos como Rhinoceros, SAP VEViewer, Moment of Inspiration e Right Hemisphere Deep View.

## Formato de arquivo 3DM

[openNURBS](https://github.com/mcneel/opennurbs), um kit de ferramentas de código aberto, contém as especificações de formato de arquivo 3DM, documentação, bibliotecas de código-fonte C++ e assemblies .NET 2.0 para ler e gravar o formato de arquivo.

### Arquivos de exemplo 3DM

Alguns exemplos de arquivos 3DM podem ser baixados da seção openNURBS [examples](https://github.com/mcneel/opennurbs/tree/7.x/example_files). Eles podem ser carregados e exibidos usando o kit de ferramentas openNURBS.

## Como ler ou gravar arquivos 3DM?

As bibliotecas [openNURBS](https://github.com/mcneel/opennurbs) permitem que qualquer pessoa leia e grave o formato de arquivo 3DM sem a necessidade do Rhino. Ele fornece código-fonte C++ para uma biblioteca que lerá e gravará modelos 3D usando o formato de arquivo openNURBS. Este kit de ferramentas também fornece ferramentas de avaliação NURBS e ferramentas elementares de manipulação de visualização geométrica e 3D, além de incluir código-fonte para vários programas de exemplo. Os fóruns de suporte do [Rhinoceros](https://discourse.mcneel.com/c/opennurbs/6) são um rico conteúdo e local de discussão para compartilhar os problemas enfrentados pela comunidade 3DM e obter as soluções.
