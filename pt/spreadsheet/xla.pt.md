## O que é um arquivo XLA?

Um arquivo com extensão .xla é um formato de arquivo do Microsoft Excel Add-Ins que adiciona módulos, funções extras e outras ferramentas ao Microsoft Excel. Os Ad-Ins fornecem funcionalidade adicional para planilhas do Excel e são suportados na versão 2007 ou posterior. Os arquivos XLA têm o tipo de mídia mime application/vnd.ms-excel. Eles podem ser abertos usando o Microsoft Excel que está disponível para Windows e MacOS.

## Formato de arquivo XLA

Os arquivos XLA são baseados no formato de dados binários. No entanto, mais informações sobre suas especificações de formato de arquivo não estão disponíveis. A suposição geral é que o XLA siga a [[MS-XLS] - Excel Binary File Format Structure](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx).
