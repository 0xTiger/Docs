## O que é um arquivo TMV?

Um arquivo TMV é um arquivo de planilha criado com o software [TimeMap](https://www.lexisnexis.com/en-us/products/casemap.page) da LexisNexis. O TimeMap permite criar linhas do tempo com a data e hora especificadas dos fatos. Com a entrada de novos fatos, o software cria factboxes para estes e os posiciona na linha do tempo de ocorrência. O projeto TMV pode incluir caixas de fatos, caixas de texto e o texto dentro delas pode ser formatado usando fontes diferentes. Os arquivos TMV podem ser abertos com o software TimeMap. LexisNexis também fornece uma ferramenta de colaboração online agora em ambiente baseado em nuvem.

## Formato de arquivo TMV

Os arquivos TMV são armazenados em disco no formato de arquivo proprietário CaseMap cujas especificações de formato de arquivo não estão disponíveis online. CaseMap foi adquirido pela LixesNexis.
