## O que é um arquivo \_XLS?

Um arquivo com extensão .\_XLS é na verdade um arquivo XLS que foi renomeado para \_XLS por alguns outros aplicativos. Ele é baseado na estrutura do formato de arquivo XLS e, se renomeado para .xls, pode ser aberto com todos os aplicativos que suportam o formato de arquivo XLS. Na maioria dos casos, os arquivos \_XLS são gerados com o arquivo XLS anexado a um e-mail que o converte em \_XLS no final do recebimento. Dentro deles, o uso das mesmas [Especificações de formação de arquivo XLS](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) para conter diferentes tipos de informações, como gráficos , gráficos e dados de planilhas.

## Formato de arquivo \_XLS - Mais informações

Os arquivos \_XLS são armazenados no disco em formato de arquivo binário e podem ser abertos no Microsoft Word alterando sua extensão para XLS. As [especificações de formato de arquivo do arquivo \_XLS](https://msdn.microsoft.com/en-us/library/dd922181(v#office.12).aspx) completas estão disponíveis publicamente pela Microsoft para referência do desenvolvedor.
