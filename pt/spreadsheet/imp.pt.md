## O que é um arquivo IMP? ##

Um arquivo IMP é um arquivo de planilha criado com o software [Lotus Improv](https://en.wikipedia.org/wiki/Lotus_Improv) para armazenar dados em linhas e colunas. O conteúdo de um arquivo IMP é separado em três segmentos diferentes; ou seja, dados, visualização dos dados e fórmulas. Embora o objetivo dessa abordagem fosse facilitar a edição, isso dificultava a transferência de dados de um arquivo IMP para outros tipos comuns de arquivos de aplicativos, como o Excel. Os arquivos IMP foram marcados como obsoletos e não são mais um formato de arquivo suportado. O Lotus Improv foi posteriormente adquirido pela IBM em 1996.

## Formato de arquivo IMP

Os arquivos IMP são armazenados em disco como arquivos binários e seu formato de arquivo interno não está disponível publicamente. O Lotus Improv foi substituído pelo IBM Lotus 1-2-3 mais tarde.
