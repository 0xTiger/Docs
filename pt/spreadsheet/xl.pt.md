## O que é um arquivo XL?

Um arquivo com extensão .xl é um arquivo de planilha do Excel, semelhante a [XLS](/pt/spreadsheet/xls/), que armazena dados em linhas e colunas. Era um formato de arquivo anterior que não é mais suportado pelo Excel agora. Os dados armazenados em arquivos XL podem ser exportados para outros formatos, como CSV. Eles também podem ser convertidos em PDF usando o método padrão Salvar como do Excel. Arquivos XL podem ser abertos apenas com o Microsoft Excel.

## Formato de arquivo XL

Os arquivos XL são salvos como arquivos binários e suas especificações de formato de arquivo interno não são documentadas em nenhum lugar. No entanto, o formato de arquivo binário geral para o formato de arquivo [XLS File Format Specifications](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) também pode ser considerado se alguém tenta escrever um pedaço de código para fazer engenharia reversa desses arquivos.
