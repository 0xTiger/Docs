## O que é um arquivo SXC? ##

O formato de arquivo SXC (Sun XML Calc) pertence a uma suíte de escritório chamada OpenOffice.org. Esse formato geralmente lida com as necessidades de planilhas dos usuários, pois é um formato de arquivo de planilha baseado em XML. O formato SXC suporta fórmulas, funções, macros e gráficos junto com o DataPilot, que é um recurso incrível porque individualiza automaticamente e fornece resumos de dados brutos importados. Os arquivos criados com este software são salvos com extensão .sxc.


## Formato de arquivo SXC ##

Arquivos, com extensões .sxc, podem ser abertos com o Microsoft Excel, que é um software de planilhas do pacote Microsoft Office. Existem outros softwares que podem abrir arquivos com extensão .sxc, como Libre Office, StarOffice etc.

O tipo de conteúdo para este formato é:

* application/vnd.sun.xml.calc
* application/vnd.sun.xml.calc.template

## Compatibilidade de arquivo ##

Os arquivos SXC são compatíveis com o Apache Open Office calc e os dados podem ser exportados para o arquivo Microsoft Excel e outros formatos também.

## Manipulação de arquivos ##

Os comandos ImportMatrix e ExportMatrix podem ler e gravar arquivos SXC. Quando a origem é um arquivo SXC e o destino também é um arquivo SXC, ele faz com que **f** seja considerado como um arquivo Sun XML Calc Spreadsheet. Tanto a Matriz de Importação quanto a de Exportação suportam esse formato, que é detectado automaticamente quando o formato não é especificado e o nome do arquivo termina em .sxc.

Para abrir o arquivo SXC, é necessário que um programa apropriado esteja associado ao arquivo. Sempre que um arquivo com extensão .sxc for aberto, certifique-se de que esteja livre de erros e que não esteja corrompido. Qualquer erro no arquivo pode ser facilmente tratado pelo usuário e isso não requer suporte técnico na maioria das vezes.
