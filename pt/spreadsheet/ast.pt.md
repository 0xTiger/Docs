## O que é um arquivo AST?

Um arquivo AST é um arquivo de modelo de planilha criado com o software Ability Spreadsheet, um componente de planilha do pacote Ability Office. Esses arquivos de modelo são usados para criar um conjunto inicial de modelos de planilha que podem ser usados para criar arquivos [AWS](/pt/spreadsheet/aws/). Os arquivos AST podem ser abertos com o software de planilha Ability.

## Formato de arquivo AST - Mais informações

Os arquivos AST são salvos em disco como arquivos binários e podem ser abertos no software Ability Spreadsheet. Embora totalmente compatível com os formatos de arquivo Microsoft XLS e XLSX, as especificações de formato de arquivo AST não estão disponíveis publicamente para referência do desenvolvedor. Os arquivos AST podem ser salvos em disco nos formatos de arquivo [PDF](/pt/pdf/), bem como [XPS](/pt/page-description-language/xps/).
