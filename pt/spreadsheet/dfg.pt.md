## O que é um arquivo DFG?

Um arquivo com extensão .dfg é um arquivo de planilha criado pela Interscape Inc. para o software Interscape Data Flask. Os dados são preenchidos em arquivos DFG em linhas e colunas. O Data Flask é um software de planilha, projetado para reter tipos de dados, fontes e informações importantes. Ele pode migrar tabelas e índices alterando apenas a cadeia de conexão. Ele oferece recursos como linha de comando VBScript, fórmulas VBScript e macros junto com scripts SQL para atualização e migração de dados. Os arquivos DFG são salvos como arquivos binários em disco e podem ser abertos com o software Interscape Data Flask.

## Formato de arquivo DFG - Mais informações

Os arquivos DFG salvam informações de dados da planilha como arquivos binários no disco. No entanto, a estrutura de dados interna e os detalhes de formato desses arquivos não são conhecidos.
