## O que é um arquivo PMVX?

Um arquivo com extensão .pmvx é um arquivo de modelo de planilha criado com o programa de planilha [PlanMaker](https://www.softmaker.de/softmaker-office-planmaker), que vem como parte do Softmaker Office. Os dados são armazenados em linhas e colunas em arquivos PMVX da mesma forma que é feito no Microsoft Excel. Os arquivos PMVX armazenam configurações predefinidas como um modelo que pode ser usado para gerar arquivos PMD e PMDX após preencher os dados nele. [XLSX](/pt/spreadsheet/xlsx/) é o formato de arquivo padrão do PlanMaker e pode ser aberto diretamente com o Microsoft Excel. Você também pode editar arquivos [XLS](/pt/spreadsheet/xls/) e criar arquivos [PDF](/pt/pdf/) usando o PlanMaker.

## Formato de arquivo PMVX

Os arquivos PMVX são armazenados em disco no formato de arquivo proprietário do PlanMaker e suas estruturas de arquivo internas são desconhecidas. No que diz respeito ao salvamento de arquivos PMVX, ele salva os dados no formato Open XML como arquivos XLSX.
