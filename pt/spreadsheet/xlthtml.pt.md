## O que é um arquivo XLTHTML?

Um arquivo com extensão .xlthtml é uma versão [HTML](/pt/web/html/) de arquivos de modelo [XLT](/pt/spreadsheet/xlt/) e foram gerados usando o Microsoft Excel nos primeiros dias. Este formato não é mais suportado pelo Excel e foi substituído pela exportação de dados para os formatos de arquivo [MHT](/pt/web/mht/) e [MHTML](/pt/web/mhtml/). A ideia básica por trás do arquivo XLTHTML era permitir que os usuários visualizassem os arquivos de modelo XLT em um navegador da Web, em vez da necessidade de ter o Excel instalado no sistema. Arquivos exportados em formato de arquivo XLTHTML podem ser abertos em navegadores.

## Formato de arquivo XLTHTML - Mais informações

Os arquivos XLTHTML usavam o formato de arquivo HTML para salvar os modelos XLT para abertura em um navegador da web. O próprio XLT é baseado no formato de arquivo binário do Excel que foi introduzido com a versão Excel 97-2003. Esses arquivos podem ser usados como arquivos iniciais com formatação definida para criar documentos normais para preenchimento de dados de planilhas. Apenas para referência, o XLT é baseado no binário [especificações de formato de arquivo XLS](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) documentado pela Microsoft para referência do desenvolvedor.
