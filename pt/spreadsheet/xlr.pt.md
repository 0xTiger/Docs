## O que é um arquivo XLR?

Um arquivo com extensão .xlr é um formato de arquivo de planilha do Microsoft Works que armazena dados em linhas e colunas. O Microsoft Works foi desenvolvido para ser usado como uma suíte de produtividade inicial e permaneceu funcional de 1987 a 2009. Incluía um processador de texto, uma planilha e um sistema de gerenciamento de banco de dados. Mais tarde, foi descontinuado após a introdução do Microsoft Office com recursos mais avançados incluídos.

## Formato de arquivo XLR - Mais informações

Os arquivos XLR foram salvos como arquivos binários sem detalhes sobre suas especificações de formato de arquivo disponíveis publicamente. Os desenvolvedores, portanto, não têm a opção de analisar sua estrutura de arquivos e escrever aplicativos para leitura/gravação nesses arquivos programaticamente. A planilha do Works usa o mesmo formato de arquivo do Excel para armazenar dados em arquivos .xlr.
