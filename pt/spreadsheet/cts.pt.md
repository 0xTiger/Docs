## O que é um arquivo CTS?

Um arquivo CTS é um arquivo de planilha criado com o programa de organização e entrada de dados multiplataforma de código aberto, **TreeSheets**. Ele contém dados hierárquicos inseridos em diferentes formatos dentro da planilha, como notas, listas, calendários, listas de tarefas, notas, mapas mentais e vários outros formatos. Os arquivos CTS só podem ser abertos com TreeSheets, que é leve, rápido e funciona no Windows, Linux e macOS.

## Formato de arquivo CTS - Mais informações

TreeSheets salva arquivos CTS em seu próprio formato de arquivo proprietário. As especificações de formato de arquivo do formato de arquivo CTS não estão disponíveis publicamente e é por isso que os arquivos CTS só podem ser abertos no software TreeSheets.

### Exportar arquivos CTS do TreeSheets para outros formatos

Os usuários do TreeSheets podem exportar as planilhas CTS para vários formatos de arquivo diferentes. Os arquivos CTS podem ser exportados para os formatos de arquivo [HTML](/pt/web/html/), [JSON](/pt/web/json/), [CSV](/pt/spreadsheet/csv/) e [Image](/pt/image/) .

## Sobre o software TreeSheets

TreeSheets é um programa de software de código aberto que está disponível para multiplataforma. Ele permite que você crie planilhas hierárquicas e oferece a liberdade de colocar todos os tipos de dados em um único arquivo de planilha. A vantagem de usar TreeSheets é que você pode colocar todas as suas discussões na forma de notas, mapas mentais, dados de texto, imagens e outros tipos de dados em um único arquivo de planilha.
