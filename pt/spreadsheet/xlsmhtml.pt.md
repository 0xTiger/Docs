## O que é um arquivo XLSMHTML?

Um arquivo com extensão .xlsmhtml é uma versão [HTML](/pt/web/html/) do arquivo XLS original que é exportado do Microsoft Excel para exibir os dados em um navegador. O formato foi descontinuado há muito tempo pelo Excel e não é mais suportado. O arquivo XLSMHTML exportado seguiu o formato de arquivo XLS e estes podem ser abertos e visualizados em navegadores da web. O Excel permite exportar os dados da planilha para os formatos de arquivo [MHT](/pt/web/mht/) e [MHTML](/pt/web/mhtml/) e são amplamente usados agora.

## Formato de arquivo XLSMHTML - Mais informações

Os arquivos XLSMHTML foram gerados com código HTML para representar os dados da planilha em um formato semelhante ao XLS. O código HTML dentro do arquivo XLSMHTML é organizado de forma que o arquivo de saída pareça semelhante ao arquivo original quando aberto em um navegador da web. As especificações de formato de arquivo do arquivo XLSMHTML não são divulgadas pelo Excel e nenhuma referência está disponível para os desenvolvedores entenderem e desenvolverem aplicativos para ele.
