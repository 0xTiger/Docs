## O que é um arquivo PMDX?

Um arquivo PMDX é um documento de planilha criado com a versão mais recente do [PlanMaker](https://www.softmaker.de/softmaker-office-planmaker), um software que vem como parte do SoftMaker Office. Ele armazena dados em células na forma de linhas e colunas. Cada célula pode conter string, números, fórmulas ou outros dados formatados. Os arquivos PMDX substituíram os arquivos [PMD](/pt/spreadsheet/pmd/) da versão mais antiga pela atualização do PlanMaker. XLSX é o formato de arquivo padrão suportado pelo PlanMaker e não precisa de nenhuma conversão ao abrir esses arquivos com o Microsoft Excel. O PlanMaker também suporta trabalhar com arquivos [XLS](/pt/spreadsheet/xls/) e convertê-los para outros formatos, como [PDF](/pt/pdf/).

## Formato de arquivo PMDX

Os arquivos PMDX são armazenados em disco como arquivos binários em formato de arquivo proprietário e suas especificações não estão disponíveis publicamente.
