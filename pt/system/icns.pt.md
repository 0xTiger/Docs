## O que é um arquivo .ICNS?

Um arquivo ICNS é um formato de arquivo usado no macOS para armazenar ícones de aplicativos, pastas e outros itens. É um formato de arquivo contêiner que pode conter vários tamanhos e resoluções de imagem do mesmo ícone.

Os arquivos ICNS normalmente contêm imagens em diversas resoluções, variando de 16x16 pixels a 1024x1024 pixels, para acomodar diferentes tamanhos e resoluções de exibição em dispositivos macOS. O formato de arquivo usa um algoritmo de compactação para reduzir o tamanho do arquivo das imagens.

Os arquivos ICNS podem ser criados usando várias ferramentas de software, como Icon Composer ou Sketch, e podem ser atribuídos a vários itens em um sistema macOS, como aplicativos, pastas e documentos. Quando um item é selecionado em um sistema macOS, o sistema operacional recupera o tamanho e a resolução apropriados do ícone do arquivo ICNS e o exibe na tela.

## Formato de arquivo ICNS – Mais informações

Arquivos ICNS são arquivos contêineres que podem conter vários tamanhos e resoluções de imagem do mesmo ícone usando uma técnica chamada "família de ícones". Uma família de ícones consiste em um grupo de recursos de imagem, cada um representando o mesmo ícone, mas em diferentes tamanhos ou resoluções. Cada recurso de imagem dentro de uma família de ícones possui um tipo e ID exclusivos, que identificam o tamanho e a resolução da imagem. Os diferentes tamanhos e resoluções de imagem são armazenados no mesmo arquivo ICNS usando diferentes tipos e IDs.

Quando um item é selecionado em um sistema macOS, o sistema operacional recupera o tamanho e a resolução apropriados do ícone do arquivo ICNS usando seu tipo e ID e o exibe na tela. Isso permite que o sistema macOS exiba ícones de alta qualidade em telas com diferentes resoluções, como em telas Retina, mantendo um tamanho de arquivo menor.

Os arquivos ICNS no macOS são semelhantes aos arquivos .ICO no Windows, pois ambos servem como formatos de arquivo para armazenar ícones. Assim como os arquivos ICNS, os arquivos ICO podem conter vários tamanhos e resoluções de imagem do mesmo ícone, que são usados para exibir o ícone em diferentes tamanhos e resoluções de tela em dispositivos Windows.

No entanto, existem algumas diferenças entre os arquivos ICNS e ICO. Por exemplo, os arquivos ICNS podem armazenar imagens de resolução mais alta do que os arquivos ICO, o que os torna mais adequados para uso em telas de alta resolução, como telas Retina em dispositivos macOS. Além disso, os arquivos ICNS podem conter imagens com canal alfa, permitindo fundos transparentes, o que não é compatível com arquivos ICO.
