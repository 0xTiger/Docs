Os arquivos do sistema são um tipo integral de arquivos encontrados nos sistemas operacionais Microsoft Windows, Mac e Linux. Os arquivos do sistema são um dos arquivos mais importantes armazenados no sistema operacional do dispositivo, pois contêm todas as informações relacionadas ao sistema do dispositivo e, sem elas, um dispositivo pode apresentar mau funcionamento ou não funcionar. Por esse motivo, os arquivos do sistema são altamente protegidos com mecanismos de permissões, devido aos quais não podem ser excluídos acidentalmente ou mesmo intencionalmente do computador com facilidade. Para facilitar a especificação e identificação, esses arquivos são sempre armazenados em pastas específicas do sistema.

Os arquivos do sistema são encontrados em vários tipos de extensão de arquivo, e cada sistema operacional tem seu tipo exclusivo de extensão de arquivo do sistema para diferentes propósitos. As extensões de arquivos de sistema mais comuns são; [CAB](/pt/system/cab/), SYS, [DLL](/pt/system/dll/) e DRV.
