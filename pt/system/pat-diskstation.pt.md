## O que é um arquivo .PAT?

Um arquivo PAT é comumente associado ao **DiskStation Manager (DSM)**, sistema operacional usado pelos dispositivos Synology Network-Attached Storage (NAS). O arquivo PAT é um arquivo de instalação usado para atualizar ou instalar o DSM em um Synology NAS.

## Uso do arquivo PAT

Veja como você normalmente usa um arquivo PAT para instalar ou atualizar o DSM no Synology NAS:

1. Baixe o arquivo PAT: Você pode obter o arquivo PAT no site oficial da Synology ou em outras fontes confiáveis.
    







2. Faça login no seu NAS: Acesse a interface do usuário baseada na web do seu Synology NAS inserindo seu endereço IP em um navegador da web. Você precisará de privilégios de administrador para realizar esta operação.
    







3. Vá para Package Center: Na interface da web, navegue até “Package Center”. É aqui que você gerencia e instala aplicativos em seu NAS.
    







4. Instalação manual: No Package Center, deve haver uma opção para "Instalação manual" ou "Instalar/Atualizar" dependendo da versão do DSM que você está usando. Escolha esta opção.
    







5. Procure o arquivo PAT: você será solicitado a navegar pelos arquivos locais e selecionar o arquivo PAT baixado.
    







6. Instalar ou atualizar: Após selecionar o arquivo PAT, siga as instruções na tela para instalar o DSM (se for uma instalação nova) ou atualizar o DSM (se você estiver atualizando para uma versão mais recente).
    







7. Aguarde a conclusão: O processo de instalação ou atualização pode levar algum tempo e seu NAS pode reiniciar como parte do processo. Seja paciente e espere terminar.
    







8. Configuração pós-instalação: Após a instalação ou atualização, pode ser necessário definir as configurações do NAS de acordo com suas preferências.

## Gerenciador de DiskStation

DiskStation Manager, muitas vezes abreviado como DSM, é um sistema operacional desenvolvido pela Synology para seus dispositivos Network-Attached Storage (NAS). Serve como interface de gerenciamento e controle para servidores Synology NAS. O DiskStation Manager fornece uma interface amigável baseada na Web que permite aos usuários configurar e gerenciar vários aspectos de seu NAS, como armazenamento de dados, compartilhamento de arquivos, soluções de backup, serviços multimídia e muito mais.

O DSM é conhecido por sua versatilidade e extenso ecossistema de pacotes que permite aos usuários instalar e executar vários aplicativos e serviços em seu Synology NAS, transformando-o em um servidor multifuncional para uso doméstico e comercial. Algumas das funcionalidades comuns do DiskStation Manager incluem compartilhamento de arquivos, backup e sincronização de dados, streaming de mídia, recursos de segurança e suporte de virtualização.

A Synology lança regularmente atualizações e novas versões do DSM para aprimorar a segurança, o desempenho e os recursos. Os usuários podem atualizar manualmente o DSM usando arquivos PAT ou configurar atualizações automáticas para garantir que seu NAS esteja executando a versão mais recente e segura do DiskStation Manager.

## Como abrir um arquivo PAT

Para atualizar manualmente o DiskStation Manager, você pode usar um arquivo PAT que você baixou do Synology Download Center usando as seguintes etapas:

1. Acesse o Painel de Controle do DiskStation Manager.
2. Navegue até a seção "Atualização e restauração" e escolha "Atualização do DSM".
3. A partir daí, selecione "Atualização manual do DSM".
4. Clique no botão "Procurar", localize e escolha o arquivo PAT que você baixou.
5. Para iniciar a atualização do DiskStation Manager, clique no botão "Aplicar".
