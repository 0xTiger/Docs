## O que é um arquivo .VGA?

Um arquivo VGA é um arquivo de unidade do Windows para suporte a monitores VGA. Foi usado nas primeiras versões do Microsoft Windows para ajuste de monitores VGA no nível do sistema operacional (SO). Um arquivo VGA usado para permitir aos usuários ajustar a resolução e a profundidade de cores da exibição usando o painel de controle do Monitor.

## Formato de arquivo VGA

O formato de arquivo interno do formato de arquivo VGA não é conhecido para referência do desenvolvedor. É referenciado pelo sistema operacional Windows para definir a aparência visual do monitor VGA.
