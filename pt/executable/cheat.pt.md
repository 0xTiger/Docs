## O que é um arquivo .CHEAT?

No contexto do Navi, cheat file é um arquivo de texto que contém uma lista de comandos de terminal ou scripts de shell que podem ser acessados através da interface do Navi. Quando você pesquisa um comando ou tópico específico no Navi, ele exibe arquivos de truques relevantes que você pode navegar e pesquisar para encontrar as informações necessárias.

Navi é uma ferramenta de interface de linha de comando que fornece uma maneira de pesquisar e acessar rapidamente a documentação de diversas linguagens de programação, ferramentas e estruturas.

O Navi vem com um conjunto padrão de arquivos de truques, mas você também pode criar seus próprios arquivos de truques personalizados para adicionar à coleção. Esses arquivos são normalmente escritos no formato Markdown ou YAML e podem ser facilmente compartilhados e distribuídos entre outros usuários do Navi.

## Formato de arquivo CHEAT – Mais informações

Navi é um aplicativo de terminal multiplataforma que pode ser instalado e executado nos sistemas operacionais Windows, macOS e Linux. Ele foi projetado para funcionar com uma variedade de ambientes shell, incluindo Bash, Zsh e Fish, e fornece uma interface consistente para acessar documentação e folhas de dicas para várias linguagens e ferramentas de programação. Navi é escrito na linguagem de programação Rust e pode ser instalado usando gerenciadores de pacotes como Homebrew no macOS, APT em sistemas baseados em Ubuntu/Debian ou a partir do código-fonte no GitHub.

## Qual é o formato do arquivo CHEAT?

O formato de um arquivo de cheats para Navi é YAML ou Markdown.

## O que o arquivo CHEAT contém?

Um arquivo de cheats para Navi normalmente contém uma lista de comandos de terminal, scripts de shell ou outras informações relacionadas à programação organizadas em grupos ou categorias. Os comandos ou informações em um arquivo de cheats são normalmente breves e fornecem uma referência rápida para usuários que precisam procurar um comando ou sintaxe.

Aqui estão alguns tipos comuns de informações que podem ser incluídas em um arquivo de cheats:

- **Comandos:** O tipo mais comum de informação em um arquivo de truques são comandos de terminal ou scripts de shell. Podem ser comandos básicos usados com frequência ou comandos mais avançados específicos de uma determinada ferramenta ou linguagem de programação.
- **Sintaxe:** Os arquivos de truques também podem conter informações de sintaxe, como uma lista de opções de linha de comando disponíveis ou argumentos para um comando específico.
- **Configuração:** Alguns arquivos de truques podem conter informações de configuração, como como configurar uma ferramenta específica ou como configurar um IDE.
- **Exemplos:** Os arquivos de truques também podem incluir exemplos de como usar determinados comandos ou ferramentas em cenários comuns.
- **Atalhos:** Os arquivos de truques podem conter atalhos de teclado ou outras dicas de produtividade para trabalhar com uma ferramenta ou ambiente específico.

## Como abrir o arquivo CHEAT?

Os arquivos CHEAT são arquivos de texto simples, portanto podem ser abertos usando qualquer editor de texto, como o Notepad ou o Notepad++.

Para usar um arquivo CHEAT com o Navi, um aplicativo de terminal multiplataforma, você pode importar o arquivo de um repositório GitHub ou colocá-lo no diretório do Navi. Depois de fazer isso, os comandos incluídos no arquivo CHEAT ficarão disponíveis no Navi, permitindo que você os consulte e use rapidamente.
