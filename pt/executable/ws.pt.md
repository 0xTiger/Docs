## O que é um arquivo .WS?

Um **.WS arquivo** no Windows normalmente contém script executável projetado para ser executado por meio do **Windows Script Host**. Este script pode abranger diversos elementos e rotinas, como rotinas JScript e VBScript, bem como elementos XML. Ao clicar duas vezes em um arquivo .ws”, o Windows Script Host iniciará a execução do script incorporado no arquivo.

## Sobre o script de host do Windows

Windows Script Host (WSH) é um host de script desenvolvido pela Microsoft que permite aos usuários executar scripts escritos em linguagens de script como VBScript (Visual Basic Scripting Edition) e JScript (versão de JavaScript da Microsoft); fornece estrutura para scripts no sistema operacional Windows, permitindo a automação de diversas tarefas e administração do sistema.

Aqui estão alguns pontos importantes sobre o Windows Script Host:

1.  **Linguagens de script:** WSH oferece suporte a diversas linguagens de script, sendo VBScript e JScript as mais comumente usadas; os scripts podem ser escritos em qualquer uma dessas linguagens para automatizar tarefas, interagir com o sistema operacional ou manipular dados.
    
2.  **Execução de script:** Os scripts WSH normalmente são armazenados em arquivos com extensões como .vbs para VBScript ou .js para JScript; quando você executa um script, o WSH interpreta e executa o código.
    
3.  **Automação:** o WSH é frequentemente usado para fins de automação, como automatizar tarefas repetitivas, gerenciar configurações do sistema e executar funções administrativas; é particularmente útil para administradores de sistema e usuários avançados.
    
4.  **Scripts de console e GUI:** Os scripts WSH podem ser baseados em console ou em GUI; os scripts do console são executados em uma janela de linha de comando, enquanto os scripts GUI podem criar interfaces gráficas de usuário para aplicativos mais interativos.
    
5.  **Security:** WSH has security features to help prevent malicious scripts from causing harm; users are typically prompted before executing scripts and scripts are subject to certain restrictions to prevent unauthorized access and actions.
    
6.  **Modelo de objeto host de script do Windows:** O WSH fornece um modelo de objeto que permite que scripts interajam com o sistema operacional Windows; isso inclui acesso a sistemas de arquivos, configurações de registro, recursos de rede e outros componentes do sistema.
    
7.  **Execução de script a partir da linha de comando:** Você pode executar scripts WSH a partir da linha de comando usando os executáveis `cscript.exe` ou `wscript.exe`, dependendo se deseja executar o script em um console ou ambiente GUI .

## Como abrir o arquivo WS?

Os arquivos WS são arquivos de texto simples, então se você quiser abri-los ou editá-los, você pode usar qualquer editor de texto, por exemplo

- **Bloco de anotações**
- **Bloco de notas++**
- **Apple TextEdit**
- **Código do Visual Studio**

Observe que se você deseja executar o script dentro do arquivo WS, você pode simplesmente clicar duas vezes nele ou usar o comando `wscript` na linha de comando.
