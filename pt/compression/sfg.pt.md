## O que é um arquivo .SFG?

Um arquivo SFG é o formato de arquivo usado pelo Synfig Studio para salvar arquivos de projeto. Synfig Studio é um software de animação 2D gratuito e de código aberto que permite aos usuários criar animações de alta qualidade usando bitmap e gráficos vetoriais. O arquivo SFG contém todas as informações necessárias para recriar uma animação, incluindo as camadas, o tempo e os quadros-chave de cada camada. Também inclui quaisquer imagens ou recursos usados no projeto, bem como quaisquer faixas de áudio ou outras mídias.

## Formato de arquivo SFG

Os arquivos SFG são salvos em disco em formato de arquivo ZIP compactado. Outros formatos de arquivo suportados pelo [Synfig Studio](https://www.synfig.org/) Projeto compactado incluem [SIFZ](/compression/sifz/) e SIF.
