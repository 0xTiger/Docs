## O que é um arquivo SIT?

Um arquivo SIT é um arquivo compactado criado com o software Micro StuffIt. Este é o formato de arquivo padrão original que foi introduzido pelo StuffIt. O formato de arquivo SIT é comumente usado em sistemas Macintosh, mas agora é suportado nas plataformas Windows e Macintosh. StuffIt estendeu o formato de arquivo SIT para [SITX](/pt/compression/sitx/), que oferece mais compactação em comparação com arquivos SIT. Os arquivos SIT podem ser abertos com o software Smith Micro StuffIt Deluxe Mac 16.

## Formato de arquivo SIT - Mais informações

Os arquivos SIT são salvos em formato de arquivo binário e seus detalhes de formato de arquivo interno não estão disponíveis publicamente. Foi introduzido pelos Sistemas Aladdin e foi introduzido para compactar arquivos em arquivos.
