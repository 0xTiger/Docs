## O que é um arquivo PUP Linux?

Um arquivo Linux PUP é um pacote de instalação usado pelo sistema operacional Puppy Linux. Ele é executado como um instalador e inicia a instalação dos arquivos do pacote. No Linux, ele é frequentemente chamado de arquivo DotPup e é semelhante aos arquivos do instalador [.exe](/pt/executable/exe/) e [.msi](/pt/executable/msi/) usados no sistema operacional Windows. As versões mais recentes do Puppy Linux usam o formato de arquivo .pet em vez do formato de arquivo PUP.
