## O que é um arquivo BNDL?

Um arquivo BNDL é um arquivo de pacote de dados do jogo usado pelo jogo **Need for Speed: Most Wanted** da [Electronic Arts](https://www.ea.com/). É um arquivo de pacote que armazena sons de jogos e fluxos de áudio para serem usados para efeitos sonoros no jogo. Esses efeitos sonoros são usados quando o jogador joga o jogo. BNDL é a abreviação de pacote.

## Formato de arquivo BNDL

BNDL é um formato de arquivo proprietário usado pelo jogo Need for Speed. É um formato de arquivo proprietário e seus detalhes de formato de arquivo interno não estão disponíveis para referência pública ou desenvolvedores. Portanto, os arquivos BNDL não podem ser abertos por nenhum outro software.
