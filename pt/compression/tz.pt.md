## O que é um arquivo TZ?

Um arquivo TZ é um arquivo TAR compactado com algoritmo de compactação Unix. É uma combinação de arquivos compactados [TAR](/pt/compression/tar/) e [Z](/pt/compression/z/), e é um formato de compactação nativo para Unix OS. Os arquivos TZ podem ser abertos usando utilitários de compactação padrão, como WinZIP e WinRAR.

## Formato de arquivo TZ

Os arquivos TZ são salvos como arquivos binários usando o algoritmo de compactação do UNIX. O uso de arquivos Z facilita a formatação, execução e abertura de arquivos. Os arquivos TZ são menores em tamanho, pois a compactação Z atinge alta compactação. Os arquivos resultantes ocupam menos espaço no disco e são facilmente transferíveis pela mídia de comunicação.
