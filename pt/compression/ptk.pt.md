## O que é um arquivo PTK? ##

Os arquivos PTK são considerados arquivos de dados usados pelo Intuit Quicken. É um aplicativo de gerenciamento de finanças pessoais que auxilia a extensão de arquivo .ptk para o bom funcionamento. Possui diferentes recursos, como importação de dados diretamente dos bancos participantes, relatórios sobre todos os dados financeiros e download diário de preços de ações e fundos. O formato de arquivo PTK é o tipo de arquivo anterior que o Quicken usa para salvar informações financeiras que agora são substituídas pelo formato GIF. O principal objetivo dos arquivos PTK era importar dados de recursos online, como bancos e instituições financeiras, para o banco de dados Quickens.
