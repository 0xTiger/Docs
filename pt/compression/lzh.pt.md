## O que é um arquivo LZH? ##

Um arquivo com extensão .lzh e .lha geralmente está relacionado ao formato de arquivo de compactação de arquivo. Este formato de arquivo é o mesmo que outros formatos de compactação de arquivo como [ZIP](/pt/compression/zip/), [RAR](/pt/compression/rar/), etc. O objetivo principal desses formatos de arquivo é reduzir o tamanho do arquivo formato para envio fácil, bem como para mantê-los juntos em formato compactado. Comparado com outras empresas ocidentais, os arquivos LZH são muito populares no Japão e geralmente usados para compactar dados de videogame. O complemento LZH para Windows 7 está embutido na versão japonesa do Windows 7. A Microsoft lançou pela primeira vez o complemento de pasta Microsoft Compressed (LZH) para a versão japonesa do Windows XP. Este formato de arquivo é implementado com provisões de compactação e recursos usados pelo algoritmo Lempel-Ziv e Haruyasu

## Especificação LZH ##

O método de compactação de um arquivo LZH é mostrado como uma string de texto de cinco bytes, como -lz1-. Esses são do terceiro ao sétimo bytes no arquivo compactado.

|Especificação|Descrição|
---|---|
|Extensão | .lzh, .lha|
|Tipo de mídia| application/x-lzh-compressed|
|Código do Tipo| "LHA_" (LHA-ESPAÇO)|
|UTI| public.archive.lha|
|Desenvolvido por| Haruyasu Yoshizaki (Yoshi)|
|Tipo| Compressão|
|Estendido de| LArc|

## Problemas para abrir um arquivo LZH ##

A seguir estão alguns problemas que podem causar mau funcionamento do formato de arquivo LZH:
  

* O arquivo pode estar corrompido. Para resolver esse problema, baixe o arquivo novamente ou peça ao remetente para reenviar o arquivo novamente
* O segundo motivo é o arquivo infectado, neste caso, verifique o arquivo corretamente
* Links impróprios para o arquivo LZH
* Remoção da descrição do LZH
* Não há recursos de hardware suficientes
* Drivers desatualizados do equipamento
