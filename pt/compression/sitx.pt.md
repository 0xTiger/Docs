## O que é um arquivo SITX?

Um arquivo com extensão .sitx é um arquivo compactado criado com o software StuffIt. Esse formato foi introduzido como uma extensão do formato de arquivo padrão original Stuffit (.sit) e oferece uma compactação comparativamente maior do que seu antecessor. Os arquivos SITX são mais compactos e ocupam menos espaço junto com o suporte para compactação de 512 bits voltada para segurança. Os arquivos SITX são reconhecidos por seus tipos de mídia da Internet como application/x-stuffitx e application/x-sitx com Uniform Type Identifier como com.stuffit.archive.sitx.

## Formato de arquivo SITX - Mais informações

STIX são salvos em formato de arquivo binário e evoluíram após o uso do formato de arquivo SIT. Foi introduzido pela Aladdin Systems em setembro de 2002 para StuffIt Deluxe 7.0 para Macintosh. Este novo formato de arquivo foi projetado tendo em mente os seguintes objetivos:

* Extensível
* Suporte para mais métodos de compressão
* Suporte para nomes de arquivos longos
* Suporte a atributos de arquivo Unix e Windows

O StuffIt X atualizado melhorou o formato StuffIt original, incorporando vários algoritmos de compactação, como PPM e BWT para compactação do tipo LZW. O novo formato de arquivo também foi equipado com uma opção de "modo de bloqueio" para proteção contra perda de dados por opções de "redundância" de correção de erros.
