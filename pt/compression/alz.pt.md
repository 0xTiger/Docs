## O que é um arquivo ALZ? ##

Um arquivo com extensão .alz é um arquivo compactado, relacionado principalmente ao ALZip pela ESTsoft, Corp. O formato de arquivo ALZ como um arquivo compactado também é definido como arquivos ALZip. Os desenvolvedores desenvolveram a ferramenta de compactação de arquivos chamada ALZip e a mesma tecnologia de codificação e compactação é usada pelo formato de arquivo ALZ. Este software é criado pela ESTsoft para distribuição privada em versões amplamente utilizadas do Microsoft Windows. Com o tempo, a ferramenta de compactação e descompactação de arquivos ALZip espalhou sua provisão para mais de 36 registros de módulos diferentes e programas de compactação e sustentação para o algoritmo ZIP.

## Breve história ##

O formato de arquivo ALZ foi lançado em 1999. O ALZip e o formato de arquivo ALZ geralmente não impõem nenhum limite no tamanho do arquivo e no limite de expansão. Este software permite que o sistema Windows dite as limitações de tamanho para seus arquivos de saída.
