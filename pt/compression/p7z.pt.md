## O que é um arquivo .P7Z?

Um arquivo P7Z é um arquivo compactado que contém uma mensagem de e-mail no padrão de criptografia S/MIME. Ele foi introduzido pela Microsoft Corporation e é categorizado como arquivo de dados relacionado a e-mail. Os arquivos P7Z possuem tipo de mídia application/pkcs7-mime. Ele contém uma ou mais partes de uma mensagem que é compactada com o algoritmo de compactação zlib.

## Formato de arquivo P7Z – Mais informações

Os detalhes do formato de arquivo interno do formato de arquivo P7Z não são conhecidos. Algumas APIs como [Aspose.Email](https://products.aspose.com/email/) permitem ler a parte S/MIME das mensagens. Se um cliente de e-mail não tiver a capacidade de lidar com os dados compactados, as mensagens S/MIME compactadas serão fornecidas como anexos de e-mail.
