## O que é um arquivo MINT?

Um arquivo MINT é um arquivo de pacote de instalação usado pela distribuição [Linux Mint](https://linuxmint.com/). Ele contém arquivos de código-fonte compactados que são extraídos e usados para iniciar a instalação do Linux Mint OS. Linux Mint OS tem GUI (Graphical User Interface) fácil de usar e foi o primeiro a ter um design elegante de GUI. Os arquivos MINT são descompactados e armazenados em um diretório temporário junto com o pacote do instalador. A instalação é executada a partir do mesmo diretório para instalação do Linux Mint OS.

## Formato de arquivo Linux MINT

Os detalhes do formato de arquivo interno do formato de arquivo MINT não estão disponíveis publicamente. No entanto, os arquivos de instalação compactados armazenados nos arquivos MINT estão em conformidade com os padrões de compactação integrados aos instaladores do Linux Mint OS. Isso possibilita que esses arquivos sejam descompactados e se tornem parte da instalação.
