## O que é um arquivo S00?

Um arquivo com extensão .s00 é a primeira parte de um arquivo compactado que é dividido usando o software ZipSplitter. A divisão dos arquivos em arquivos menores reduz o tamanho geral do arquivo compactado e facilita o transporte pela Internet. As partes restantes do arquivo dividido são nomeadas com extensões .S01, .S02 e assim por diante. Quando o arquivo deve ser extraído, o arquivo .EXE incluído com o arquivo dividido é clicado duas vezes. O EXE procura o arquivo .S00 e usa os outros arquivos acompanhados para extração de todo o arquivo.

## Formato de arquivo S00

Os arquivos S00 são salvos no disco como arquivo compactado binário. No entanto, os detalhes do formato de arquivo interno do formato de arquivo S00 não estão disponíveis publicamente. Arquivos grandes, como arquivos .[WMV](/pt/video/wmv/) ou [.MP4](/pt/video/mp4/) geralmente são divididos usando o software ZipSplitter para reduzir seu tamanho.
