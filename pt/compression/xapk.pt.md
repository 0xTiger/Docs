## O que é um arquivo XAPK?

Um arquivo com extensão .xapk é um arquivo de pacote compactado usado para instalar aplicativos Android em dispositivos móveis. É um formato de arquivo de contêiner que incorpora APK e arquivos associados adicionais necessários para a instalação. O outro arquivo associado é um arquivo OBB que armazena arquivos adicionais, como gráficos, arquivos de mídia e dados do aplicativo que são necessários para manter o aplicativo em execução. Os arquivos XAPK não são compatíveis com o Google Play e são usados apenas para distribuição em sites de download de aplicativos Android de terceiros. Eles podem ser instalados em um dispositivo Android usando o XAPK Installer.

## Formato de arquivo XAPK

Os arquivos XAPK são compactados usando o formato de arquivo padrão [ZIP](/pt/compression/zip/). Eles podem ser extraídos usando um software padrão de compactação/descompactação, como o WinZIP. Depois que o arquivo XAPK é extraído para o disco, ele contém os seguintes arquivos na pasta.

* APK - Arquivo de instalação padrão para instalação do aplicativo em dispositivos Android
* OBB - Arquivo adicional que contém arquivos de recursos relevantes
