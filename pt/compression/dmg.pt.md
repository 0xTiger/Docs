## O que é um arquivo DMG?

Um arquivo com extensão .dmg é um arquivo de imagem de disco da Apple e conhecido como arquivo de imagem de disco do Mac OS X. O formato de arquivo DMG é o arquivo de imagem de disco mais usado para alocar aplicativos e arquivos entre os PCs da Apple. É muito conveniente montar arquivos DMG em uma unidade virtual ou pode usar o software Apple Disk Utility para abrir e visualizar o conteúdo do formato de arquivo DMG. A versão antiga dos arquivos IMG usados no Mac OS Classic é substituída pelo formato de arquivo DMG. Os arquivos DMG são gerados para usuários de Mac e não são suportados pelo sistema operacional Windows, mas os usuários podem acessar o conteúdo DMG no sistema operacional Windows usando ferramentas de compactação e descompactação como 7-Zip, Pea Zip, etc. A extensão do arquivo DMG também é usada por a ferramenta Oracle Export and Ingress para arquivos scrapheap mantidos no formato binário do banco de dados Oracle.

## Breve história

A extensão de arquivo DMG foi desenvolvida pela Apple e é usada principalmente em computadores Macintosh. Principalmente, o arquivo DMG é usado para fornecer aos dispositivos Mac a capacidade de instalar diferentes tipos de arquivos e softwares iniciados na Internet. O arquivo DMG é fundamentalmente uma imagem de disco montável que é exibida na área de trabalho quando é aberta. O arquivo compreende dados brutos que geralmente são criptografados e compactados. Os sistemas Mac tratam o DMG da mesma forma que tratariam um disco intercalado e instantaneamente tentam abrir ou “rastrear” o arquivo.

## Especificações técnicas do formato de arquivo DMG

* Gerado especialmente para computadores Macintosh
* Desenvolvido pela Apple
* O formato do arquivo é binário
* Categoria de arquivo é arquivo de imagem de disco
* Programa suportado é MacOS
* Pode conter uma tabela de partição
* Pode conter qualquer tipo de sistema de arquivos como HFS+
* Funcionalidade para compactar as imagens
