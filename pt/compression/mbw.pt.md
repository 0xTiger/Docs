## O que é um arquivo MBW? ##

Um arquivo com extensão .mbw é associado principalmente ao MBRWizard da Firesage Solutions. O Master Boot Record Wizard (MBRWizard) faz backup do MBR do seu computador e o salva em um arquivo MBW que pode ser restaurado, se necessário. Além das cópias MBR e VBR (Volume Boot Record), esses arquivos podem conter GPT, setores definidos pelo usuário ou imagens de disco inteiras. O MBR é um registro vital de 512 bytes no disco rígido que permite a inicialização do sistema operacional. Ele também pode conter a tabela de partição e uma assinatura de 32 bits que identifica a mídia do disco.

## Sistemas Operacionais Suportados ##

O formato de arquivo MPW é suportado apenas pelo sistema operacional Windows.

## Possíveis problemas ao abrir o arquivo ##

Se você não conseguir abrir e executar o arquivo MBW, pode haver um dos seguintes problemas possíveis:

* Arquivo MBW corrompido
* Links incorretos para o arquivo MBW nas entradas do registro
* Descrição excluída do MBW do registro do Windows
* Um arquivo infectado com um malware indesejável
* O computador não possui recursos de hardware suficientes para operar o arquivo MBW
* Os drivers usados pelo computador para abrir um arquivo MBW estão desatualizados
