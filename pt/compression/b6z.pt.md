## O que é um arquivo B6Z?

Um arquivo com extensão .b6z é um arquivo compactado criado pelo aplicativo de software macOS [B6Zip](http://b6zip.com) que é usado para compactar e descompactar arquivos. É um formato de arquivo comparativamente novo que permite maior taxa de compactação. B6Z tem arquitetura aberta e suporta tamanhos de arquivo de até 900.000 PB. Ele suporta compactação de dados, recuperação de erros e extensão de arquivos. O software utilitário B6Zip está disponível gratuitamente no MacOS para abrir diferentes tipos de arquivos compactados, incluindo B6Z.

## Formato de arquivo B6Z

O formato de arquivo B6Z é baseado em arquitetura aberta e fornece compactação sólida com algoritmo de criptografia AES-256. Ele usa o LZMA2 como método de compactação padrão, mas também suporta outros métodos de compactação como a seguir.

|Método|Descrição|
---|---|
|LZMA |Versão otimizada do algoritmo LZ77|
|LZMA2| Versão melhorada do LZMA|
|Esvaziar| Algoritmo LZ77 regular|
|BZip2| Algoritmo BWT padrão|
|PPMD| PPMdH de Dmitry Shkarin |

O utilitário B6Zip suporta todos esses padrões de compactação e é altamente otimizado, resultando em alta velocidade. Ele suporta trabalhar com os formatos de arquivo [ZIP](/pt/compression/zip/), [TAR](/pt/compression/tar/) e [RAR](/pt/compression/rar/). Os arquivos B6Z têm o tipo MIME application/x-b6z-compressed.
