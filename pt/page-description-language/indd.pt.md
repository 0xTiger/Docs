## O que é um arquivo INDD?

Um arquivo com extensão .indd é um arquivo de projeto do Adobe InDesign que contém as informações de layout de página do projeto. INDD significa Documento do InDesign. Esses arquivos geralmente armazenam informações de formatação de páginas, livros, revistas, jornais, folhetos, brochuras ou outras mídias impressas, como maquetes para dispositivos móveis e páginas da Web. O tipo de arquivo INDD é um formato de arquivo proprietário da Adobe e é o formato de arquivo padrão para o InDesign. Designers gráficos, jornalistas, profissionais de marketing e editores que trabalham com o InDesign usam esse tipo de arquivo.

## Mais sobre o formato de arquivo INDD

Os arquivos INDD são arquivos vetoriais que podem ser dimensionados sem perder a resolução. Estes são baseados em gráficos complexos que podem ser expandidos infinitamente. Ele empacota todas as informações de fontes e layouts para serem compartilhadas com uma equipe.

Os arquivos INDD são perfeitos para criar impressão envolvente, bem como conteúdo online. Um arquivo INDD pode abranger várias páginas e pode servir como modelos para a criação de páginas, capítulos e publicações. Quando seu projeto estiver concluído, ele poderá ser publicado em mídia onilne sem a necessidade de conversão, compactação ou envio por e-mail.

## O arquivo INDD pode ser convertido para outros formatos?

Os arquivos INDD podem ser convertidos para vários outros formatos de arquivo. A conversão mais comum é INDD para PDF. O InDesign permite salvar o arquivo INDD em PDF para compartilhar com outros usuários. Também permite exportar o arquivo INDD para JPG, [EPS](/pt/page-description-language/eps/), [EPUB](/pt/ebook/epub/), [SWF](/pt/page-description-language/swf/) , FLA, [HTML](/pt/web/html/), [XML](/pt/web/xml/) e IDML.

## Como o arquivo INDD é criado?

Um arquivo Indd é criado quando você cria um novo projeto com o [Adobe InDesign](https://www.adobe.com/products/indesign.html) e o salva em disco. Você pode abrir o arquivo de projeto Indd, fazer alterações nele e salvar o arquivo atualizado para as alterações.
