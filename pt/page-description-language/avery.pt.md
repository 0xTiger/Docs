## O que é um arquivo AVERY?

Um arquivo AVERY é um documento imprimível que contém uma ou mais etiquetas para fins de impressão. É criado com o software Avery Design & Print ou Design & Print Online. Os rótulos dentro de um arquivo AVERY podem incluir elementos diferentes, como texto, gráficos e informações de layout de página. Essas informações são usadas por empresas de impressão para imprimir uma variedade de elementos de design, como cartões de visita, crachás, encartes de fichário, etiquetas de endereço, etiquetas de disco, modelos de obituário e cartões comemorativos.

## Formato de arquivo AVERY - Mais informações

Os arquivos AVERY são armazenados em formato de arquivo binário. Estes podem ser abertos com o software Avery Design & Print ou Design & Print Online. Os designs criados com Design & Print podem ser usados para criar etiquetas AVERY que podem servir a vários propósitos. Esses designs podem ser salvos como arquivos [PDF](/pt/pdf/) que podem ser compartilhados com a empresa Prints para fins de impressão.

### Imprimindo um arquivo AVERY

Um arquivo AVERY pode ser salvo como PDF ou pode ser impresso diretamente em qualquer impressora padrão. Para imprimir o arquivo, abra o arquivo no software Design & Print e use a opção Imprimir no menu Arquivo para enviar o arquivo à impressora para impressão.
