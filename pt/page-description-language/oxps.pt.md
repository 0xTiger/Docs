## O que é um arquivo OXPS? ##

O formato de arquivo OXPS é conhecido como Open XML Paper Specification. É uma linguagem de descrição de página e formato de documento. A Microsoft é o desenvolvedor deste formato. O formato de arquivo OXPS é muito familiar para esses arquivos PDF. A configuração do arquivo OXPS é a mesma do formato XPS. A única diferença é a descrição do OXPS como um formato aberto. A extensão de arquivo .oxps é a versão mais atual do formato de arquivo XPS originado pela Microsoft e Ecma International. Se falarmos sobre a especificação, os arquivos OXPS e XPS são usados para publicar, compartilhar e alocar documentos de layout fixo. Como substituto dos PDFs, o formato de arquivo OXPS pode armazenar dados de texto e multimídia, com recursos de organização e características de layout de impressão.

## Breve história ##

A equipe da Microsoft escolheu a Global Graphics para fornecer consultoria em XPS em 2003, e a equipe de desenvolvimento trabalhou em estreita colaboração com as equipes do Windows para desenvolver a especificação e a arquitetura de referência para o novo formato. Reconhecendo as especificações XPS, em junho de 2007 a Ecma International criou o Technical Committee 46 (TC46) para desenvolver um padrão baseado na especificação OpenXPS. A Ecma International aprovou a Open XML Paper Specification é um padrão Ecma (ECMA-388) na 97ª Assembléia Geral realizada em Budapeste em 16 de junho de 2009.
