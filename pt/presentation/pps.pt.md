## O que é um arquivo PPS?

PPS, PowerPoint Slide Show, os arquivos são criados usando o Microsoft PowerPoint para fins de apresentação de slides. A leitura e a criação de arquivos PPS são suportadas pelo Microsoft PowerPoint 97-2003. A versão mais recente desse formato de arquivo é [PPSX ](/pt/presentation/ppsx/) que é baseada nos padrões Office OpenXML. Os arquivos PPS ainda podem ser lidos pelas versões mais recentes do Microsoft PowerPoint, mas os arquivos recém-criados só podem ser salvos no formato de arquivo PPSX. Quando um arquivo PPS é compartilhado com outro usuário e aberto, ele inicia como Powerpoint, ao contrário do arquivo [PPT](/pt/presentation/ppt/) que abre no modo editável.

## Formato de arquivo ##

PPS é um formato de arquivo binário que segue as especificações detalhadas no formato de arquivo [MS-PPT](https://msdn.microsoft.com/en-us/library/office/cc313106(v#office.12).aspx) documento. Ele consiste em vários fluxos para representar a estrutura e os dados gerais do documento. Esses incluem:

* Fluxo do usuário atual
* Fluxo de documentos do PowerPoint
* Fluxo de fotos
* Informações resumidas e informações resumidas do documento (opcional)

Detalhes completos sobre o formato de arquivo PPT podem ser encontrados no documento de especificações MS-PPT que também se aplica ao PPS.
