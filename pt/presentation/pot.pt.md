## O que é um arquivo POT?

Arquivos com extensão .POT representam arquivos de modelo do Microsoft PowerPoint criados pelas versões do PowerPoint 97-2003. Os arquivos criados com essas versões do Microsoft PowerPoint estão em formato binário em comparação com aqueles criados nos formatos de arquivo Office OpenXML usando as versões superiores do PowerPoint. Os arquivos, portanto, gerados podem ser usados para criar apresentações que tenham o mesmo layout e outras configurações necessárias para serem aplicadas aos novos arquivos. Essas configurações podem incluir estilos, planos de fundo, paleta de cores, fontes e padrões. Esses arquivos são gerados para criar arquivos de modelo prontos para uso para uso oficial.

## Especificações de formato de arquivo ##

As especificações de formato de arquivo para o formato de arquivo POT não estão disponíveis individualmente pela Microsoft. No entanto, o formato do modelo é o mesmo do formato de arquivo binário [PPT](/pt/presentation/ppt/) e pode ser aberto no MS PowerPoint para fins de edição. Um arquivo POT é identificado pelo identificador HEX da seguinte forma:

```
D0 CF 11 E0 A1 B1 1A E1 00 00 00 00
```

Os tipos MIME associados ao formato de arquivo POT são:

* application/vnd.ms-powerpoint [oficial]
* aplicativo/mspowerpoint
* aplicativo/x-mspowerpoint
* aplicação/power point
* aplicativo/x-powerpoint
* application/x-dos_ms_powerpnt
* aplicação/pote
* aplicativo/x-soffic
