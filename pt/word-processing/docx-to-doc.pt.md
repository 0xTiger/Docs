## Sobre a conversão de DOCX para DOC

O tipo de arquivo DOC é um formato de arquivo de documento popular usado em 2007 e em versões anteriores do Microsoft Word. Com o lançamento do Office 2007, a Microsoft introduziu o formato de arquivo DOCX baseado nas especificações do Office Open XML. Os arquivos DOC foram salvos como arquivos binários em comparação com DOCX, que é salvo em formato de arquivo XML compactado. Existem muitas ferramentas disponíveis online para converter arquivos [DOCX](/pt/word-processing/docx/) em [DOC](/pt/word-processing/doc/).

## Conversão de DOCX para DOC - demonstração gratuita

Você pode converter o formato de arquivo DOCX para DOC usando o aplicativo online GRATUITO, Aspose.app. É um conjunto de aplicativos que pode carregar, visualizar e converter arquivos DOCX em formatos de arquivo como DOC, PDF, TXT, RTF e outros. Os aplicativos Aspose são alimentados e executados por APIs Aspose no backend que podem executar uma série de operações em diferentes tipos de arquivos. Experimente o aplicativo online Aspose.Words agora clicando no botão a seguir.

### Etapas para converter DOCX em DOC online

Você pode converter o formato de arquivo DOCX para DOC usando o [aplicativo Aspose.Words DOCX para DOC](https://products.aspose.app/words/conversion/docx-to-doc) como segue.

1. Vá para o aplicativo Aspose.Words para [conversor de DOCX para DOC](https://products.aspose.app/words/conversion/docx-to-doc)
1. Carregue seu arquivo DOCX clicando no botão de upload
1. Quando terminar, pressione o botão Converter. Isso iniciará o processo de conversão do arquivo DOCX em DOC em segundo plano
1. Você será solicitado a baixar o arquivo RTF convertido assim que o processo de conversão de DOCX em DOC for concluído

## APIs Aspose.Words

[Aspose.Words](https://products.aspose.com/words/) é um conjunto de APIs que permite a qualquer desenvolvedor converter [DOCX](/pt/word-processing/docx/) em [DOC](/pt/word-processing/doc/) usando linhas simples de código. Está disponível para muitas linguagens de programação, incluindo [C#](/pt/programming/cs/), [Java](/pt/programming/java/), [Python](/pt/programming/py/) e [C++](/pt/programming/cpp/). Além do DOC, as APIs Aspose.Words permitem converter arquivos DOCX em muitos outros formatos de arquivo populares.
