## O que é um arquivo ODM?

Um arquivo com extensão .odm é um arquivo de documento OpenDocument Master que contém um ou mais arquivos [.odt](/pt/word-processing/odt/) vinculados a ele. Esses subdocumentos são todos combinados em um único documento mestre. Isso facilita o gerenciamento de todos os arquivos vinculados a partir de um único documento mestre. Arquivos de documentos mestres podem ser criados com Apache OpenOffice Writer ou OpenDocument Word Processor. Os arquivos ODM podem ser exportados para um único arquivo ODT usando a opção de exportação do aplicativo OpenOffice Writer.

## Formato de arquivo ODM - Mais informações

Os arquivos ODM são salvos como arquivos de texto que podem ser abertos no OpenOffice Writer. Um único arquivo ODM consiste em várias seções vinculadas aos subdocumentos .odt. Isso permite que vários escritores editem seus documentos ODT que estão vinculados no arquivo ODM principal. Assim, o arquivo ODM mestre é atualizado à medida que cada escritor atualiza suas respectivas seções.

Os arquivos ODM também podem incluir um índice e um índice. Cada vez que um documento mestre é aberto, o escritor do documento mestre solicita a atualização dos links. É preciso dizer sim para que o conteúdo seja mostrado corretamente.
