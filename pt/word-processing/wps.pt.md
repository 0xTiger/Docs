## O que é um arquivo WPS?

Um arquivo com extensão .wps é um documento do Microsoft Works que foi usado antes de 2006 como um documento do Word. Era semelhante ao recém-introduzido formato de arquivo [DOC](/pt/word-processing/doc/) até então, mas não era tão rico em recursos quanto o novo formato de arquivo. Por exemplo, ele não suportava a funcionalidade de macro e opções avançadas de formatação. Os arquivos WPS ainda podem ser abertos com processadores de texto modernos, mas para o Windows 10, você precisa instalar um conversor para carregar esses arquivos. O Microsoft Works foi substituído pelo processador de documentos Microsoft Word em 2010.

## Especificações de formato de arquivo

As especificações de formato de arquivo do documento WPS não estão disponíveis publicamente, pois a Microsoft não abriu as especificações para usuários finais. WPS são arquivos binários e podem ser abertos com filtros de conversão de formato de arquivo criados pela Microsoft para processador Word. LibreOffice, NeoOffice e OxygenOffice incluíram uma biblioteca C++ geral, libwps, que pode extrair texto de muitas versões diferentes do Microsoft Works.
