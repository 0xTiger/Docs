## Sobre a conversão de DOC para JPG

Se você já trabalhou com versões mais antigas do Microsoft Word (ou seja, versões anteriores ao Microsoft Word 2007), deve estar familiarizado com o formato de arquivo DOC. Era o formato de arquivo padrão usado pelo Microsoft Word antes da introdução do formato de arquivo DOCX com o Microsoft Word 2007. A versão DOC para JPG é um requisito muito comum, especialmente quando a Microsoft não permite exportar documentos do Word para imagens JPG. Além disso, as versões mais recentes do Microsoft Word não são totalmente compatíveis com arquivos DOC mais antigos. Você pode usar aplicativos on-line, como o aplicativo Aspose.Words, para converter formatos de arquivo [DOC](/pt/word-processing/doc/) em [JPG](/pt/image/jpeg/).

## Conversão de DOC para JPG - demonstração gratuita

O aplicativo Aspose.Words é um conjunto de ferramentas online que permite converter seus arquivos DOC em vários formatos de arquivo diferentes. Você pode converter arquivos DOC em JPG com apenas alguns cliques no seu navegador. A vantagem de usar o aplicativo Aspose.Words é que ele não requer codificação, é rápido e preciso e, o mais importante, é totalmente gratuito.

Durante o processo de DOC para JPG, cada página do documento é convertida em imagem JPG e fica disponível para download. A saída é um conjunto de imagens JPG salvas em disco.

### Como converter DOC para JPG?

Então, como você converte DOC para JPG? Você pode converter o formato de arquivo DOC para JPG usando o [aplicativo Aspose.Words DOC para JPG](https://products.aspose.app/words/conversion/doc-to-jpg) conforme segue.

1. Acesse o aplicativo Aspose.Words para [conversor de DOC para JPG](https://products.aspose.app/words/conversion/doc-to-jpg)
1. Carregue seu arquivo DOC clicando no botão de upload
1. Quando terminar, pressione o botão Converter. Isso iniciará o processo de conversão do arquivo DOC para JPG em segundo plano
1. Você será solicitado a baixar o arquivo JPG convertido assim que o processo de conversão de DOC para JPG for concluído
