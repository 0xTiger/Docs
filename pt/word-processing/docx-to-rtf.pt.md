## Sobre a conversão de DOCX para RTF

Se você é um usuário jovem do Microsoft Windows, deve conhecer o formato de arquivo RTF (Rich Text Format). A Microsoft introduziu o [formato de arquivo RTF](/pt/word-processing/rtf/) com o Windows 3.0. Os arquivos RTF representam um método de codificação de texto e gráficos formatados para uso em aplicativos. Um arquivo RTF pode conter texto e imagens, além de manter a formatação do texto, como Negrito, Itálico e Sublinhado. O mais interessante é que o RTF foi adotado como formato de arquivo interoperável e é multiplataforma. Por esse motivo, a conversão de DOCX em RTF é uma escolha popular ao compartilhar documentos em diferentes sistemas operacionais para fins de edição.
