## Sobre a conversão de DOC para DOCM

DOCM é a versão habilitada para macro de documentos do Microsoft Word que são capazes de executar macros. A diferença entre um documento normal do Word e um documento habilitado para macro é que o DOCM fornece o recurso adicional para suporte a macros.

Os arquivos DOC podem ser convertidos em DOCM usando ferramentas e aplicativos online gratuitos. **[Aplicativo Aspose.Words](https://products.aspose.app/words/family)** é um conjunto de aplicativos on-line que permite realizar diferentes operações em documentos do Word diretamente do seu navegador. Além dos recursos de [visualização de documentos](/pt/word-processing/docx-viewer/), divisão, mesclagem, comparação, rotação e outros, permite converter documentos Word de um formato para outro. Você pode converter [DOC](/pt/word-processing/doc/) em [DOCM](/pt/word-processing/docm/) usando o aplicativo Aspose.Words.

### Como converter DOC para DOCM?

Você pode converter o formato de arquivo DOC para DOCM usando o [aplicativo Aspose.Words DOC para DOCM](https://products.aspose.app/words/conversion/doc-to-docm) como segue.

1. Acesse o aplicativo Aspose.Words para [conversor DOC para DOCM](https://products.aspose.app/words/conversion/doc-to-docm)
1. Carregue seu arquivo DOC clicando no botão de upload
1. Quando terminar, pressione o botão Converter. Isso iniciará o processo de conversão do arquivo DOC em DOCM em segundo plano
1. Você será solicitado a baixar o arquivo DOCM convertido assim que o processo de conversão de DOC em DOCM for concluído
