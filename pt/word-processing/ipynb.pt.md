## O que é um arquivo IPYNB?

Um arquivo com extensão .ipynb é um arquivo Jupyter Notebook que contém todo o conteúdo do Notebook criado com a sessão do aplicativo web Jupyter Notebook. [Jupyter Notebook](https://jupyter.org/) é um aplicativo da web interativo usado para criar e compartilhar documentos computacionais para análise usando Python.

Um arquivo IPYNB serve como um registro computacional completo de uma sessão e inclui informações como entradas e saídas computacionais, funções matemáticas e rica representação de objetos resultantes, como imagens e texto explicativo.

## Formato de arquivo IPYNB

IPYNB são salvos internamente no formato de arquivo [JSON](/pt/web/json/), que é um formato de arquivo padrão aberto para compartilhamento de dados. Por esse motivo, os arquivos IPYNB são legíveis e facilmente compreensíveis. O JSON também facilita o compartilhamento desses arquivos com outras pessoas.

## Como converter arquivos IPYNB?

Os arquivos IPYNB podem ser exportados para vários formatos de arquivo diferentes usando o comando nbconvert da interface Jupyter. Isso inclui a conversão de IPYNB para:

*PDF
* HTML
* texto reestruturado
* Látex

## Como visualizar o arquivo IPYNB?

Existem dois métodos para visualizar ou abrir arquivos IPYNB.

1. O usuário pode carregar o documento do notebook .ipynb para algum URL público e abri-lo no Jupyter Notebook Viewer<nbviewer> que renderiza o bloco de anotações como uma página da Web estática do URL.
1. Baixe os arquivos e abra-os em sua própria instância do Jupyter Notebook
