## O que é um arquivo WPD?

Um arquivo com uma extensão .wpd conhecido como WordPerfect Document Files (WPD). O Corel WordPerfect foi usado para produzir este documento de texto. Um programa de processamento de texto é chamado de **WordPerfect**. Ele é usado em uma variedade de sistemas de computador, incluindo Windows. O WordPerfect também é utilizado em ambientes empresariais, acadêmicos e domésticos para boletins informativos, currículos, convites e cartas, além de outros documentos.

O arquivo WPD contém dados estruturados e os documentos podem armazenar vários objetos incorporados, como gráficos e gráficos desenvolvidos em outros aplicativos. Os documentos WPD também incluem elementos de mala direta, que permitem que informações de uma entidade externa, como uma planilha ou banco de dados, sejam inseridas em um design. Você pode tocar duas vezes em um arquivo WPD e em todos os outros documentos em seu PC para abri-lo. Se suas associações de documentos estiverem corretas, o aplicativo projetado para abrir seu documento .wpd o fará.


## Breve história ##

O WordPerfect para Mac não foi desenvolvido em paralelo com edições para outros aplicativos de software e, portanto, usou números de versão que não estavam relacionados às versões atuais para sistemas operacionais e Windows. A versão 2 foi uma reformulação completa que aderiu mais de perto aos princípios de interface de usuário da Apple.

A versão 3 expandiu isso, fazendo uso substancial da tecnologia introduzida pela Apple nos Sistemas 7.0–7.5, mantendo-se rápida e capaz de trabalhar efetivamente em máquinas anteriores. A versão 3.5 foi lançada em 1996, seguida pela versão atualizada 3.5e em 1997. Ela nunca foi atualizada depois disso, e o produto foi finalmente descontinuado. A Corel disse que não tem intenção de desenvolver ainda mais o WordPerfect para Mac a partir de 2004.

## Especificações de formato de arquivo WPD ##

Os documentos WPD foram usados principalmente para processar o texto preparado. Como resultado, esses arquivos incluem pedaços de texto. As tags são usadas para conter informações de formatação, que o WordPerfect reconhece como código. Isso facilita muito a localização de problemas de formatação porque o aplicativo destaca áreas do texto e expõe a formatação herdada. Documentos WPD, como tipos de arquivos de processamento de texto mais populares, como [DOC](/pt/word-processing/doc/), podem incorporar dados de diferentes fontes, como arquivos, bancos de dados ou designs.
