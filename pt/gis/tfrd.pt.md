## O que é um arquivo .TFRD?

Um arquivo TFRD é um formato de dados de imagem usado pela Agência Nacional de Inteligência Geoespacial (NGA) para armazenar dados geoespaciais. A NGA fornece inteligência geoespacial que dá vantagem na tomada de decisões. Os arquivos TFRD podem armazenar dados de imagem e textuais e podem ser usados por aplicativos GIS para renderizar dados geoespaciais.

## Formato de arquivo TFRD

Os detalhes do formato de arquivo interno do formato de arquivo TFRD não são conhecidos. Eles são, no entanto, salvos como arquivos compactados em disco e a taxa de compactação é maior do que os arquivos de formato de arquivo NITF equivalentes.
