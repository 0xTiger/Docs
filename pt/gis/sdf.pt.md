## O que é um arquivo .SDF?

A Spatial Data File (SDF) is a Geodatabase file format that was developed by Autodesk. It stores geospatial data and is used by Autodesk GIS program MapGuide and AutoCAD Map 3D. SDF file format is based on the single file database file format SQLite3. É considerado um formato de arquivo otimizado para grandes conjuntos de dados de informações espaciais.

Você pode abrir um arquivo SDF usando o Autodesk AutoCAD Map 3D 2022 e o Autodesk AutoCAD Civil 3D 2023.

## Formato de arquivo SDF – Mais informações

O formato de arquivo SDF é armazenado no disco como arquivos binários. É baseado no banco de dados SQLite que utiliza componentes de armazenamento de baixo nível para serialização binária de dados. Um arquivo SDF contém diversas classes de recursos por arquivo e diversas propriedades de geometria para cada classe de recursos. Os dados em um arquivo SDF são acessíveis em alta velocidade devido à indexação de cada propriedade geométrica usando uma árvore R.
