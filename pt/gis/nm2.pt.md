## O que é um arquivo NM2?

Um arquivo com extensão .nm2 é um arquivo de mapa criado com o sistema de navegação Navitel. Ele armazena dados geoespaciais relacionados a diferentes entidades do mapa, como estradas, ruas, pistas rodoviárias, pontos no mapa, pontos de referência e outros dados de GPS. Os arquivos NM2 podem ser abertos usando o software Navitel em diferentes tipos de dispositivos, como telefones Android, dispositivos iOS, Windows Phone e dispositivos Symbian. Esses aplicativos estão disponíveis gratuitamente e podem ser baixados e instalados nos respectivos dispositivos.

## Formato de arquivo NM2 - Mais informações

Os arquivos NM2 são salvos como arquivos binários e seu formato de arquivo interno/estrutura de dados não está disponível publicamente. As versões recentes do Navitel salvam os dados como arquivos .nm7. Existem vários arquivos de mapas disponíveis pela Navitel em seu [site](https://www.navitel.cz/en/downloads/demo). A [documentação Navitel](https://www.navitel.cz/en/downloads/documentation) fornece guias de usuário completos sobre como usar seu sistema de navegação para diferentes tipos de dispositivos e veículos.
