## O que é um arquivo QML?

Um arquivo com extensão .qml é um arquivo XML que armazena informações de estilo de camada para camadas QGIS. O QGIS é um aplicativo GIS de plataforma cruzada de código aberto usado para exibir dados geoespaciais com a capacidade de organizar dados na forma de camadas. Os arquivos QML contêm informações que são usadas pelo QGIS para renderizar geometrias de feições, incluindo definições de símbolos, tamanhos e rotações, rotulagem, opacidade, modo de mesclagem e muito mais. Ao contrário dos arquivos QLR, os arquivos QML contêm todas as informações de estilo em si. Os arquivos QML podem ser abertos em qualquer editor de texto, como o Notepad++.
