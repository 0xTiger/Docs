## O que é um arquivo .AVL?

Um arquivo AVL é um arquivo de legenda que contém uma chave para um mapa gerado com o software ESRI ArcView GIS (Geographic Information System). Contém informações de simbologia de referência utilizadas no mapa correspondente para acesso. Um arquivo AVL pode conter mais informações, como simbologia, configurações de exibição, como transparência, propriedades de exibição dependentes de escala, informações de tabela unida/tabela relacionada, consulta de definição, propriedades de rotulagem para camadas de feição e propriedades de exibição de anotação para camadas de anotação dependendo do tipo tipo de camada.

Um arquivo AVL pode ser referenciado no arquivo do projeto ArcView [.APR](/gis/apr/).

## Formato de arquivo AVL – Mais informações

Os arquivos AVL são salvos em formato de arquivo de texto simples. Isso significa que você pode abrir arquivos AVL em qualquer editor de texto, como Notepad, Notepad++ e Apple TextEdit. Uma vez aberto, você pode não apenas visualizar o conteúdo do arquivo, mas também modificá-lo.

### Diferença entre arquivos .LYR e .AVL

 * **Diferença de formato de arquivo** - .lyr é um arquivo binário enquanto .avl é um arquivo de texto
 * **Diferença de conteúdo** - Um arquivo .lyr contém mais informações que um arquivo .avl. Um arquivo AVL armazena apenas informações de simbologia, enquanto um arquivo LYR armazena todas as informações que estão disponíveis na caixa de diálogo de propriedades da camada no ArcMap.
