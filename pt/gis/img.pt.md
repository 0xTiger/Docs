## O que é um arquivo IMG?

Um arquivo com extensão .img é um arquivo de mapa Garmin usado por dispositivos GPS Garmin. Ele contém dados de mapas/geoespaciais para representar entidades de mapas, como nomes de ruas, rotas de navegação e pontos de interesse. Os usuários podem criar dados de mapas personalizados, salvá-los como arquivo .img e carregá-los em dispositivos Garmin. Garmin também fornece dados de mapas pré-instalados que são armazenados em dispositivos Garmin no formato de arquivo IMG. Os arquivos IMG podem ser abertos usando aplicativos como Garmin Mobile PC, cGPSmapper, MapUpload e GPS Map Manager.

## Formato de arquivo IMG - Mais informações

Os arquivos IMG são salvos como arquivos binários no formato de arquivo proprietário da Garmin. As informações disponíveis em um arquivo IMG podem ser renderizadas em unidades GPS Garmin. Algumas unidades de GPS, que suportam autorouting, também podem usar esses arquivos de mapa para orientação durante a navegação, como em carros, bicicletas ou durante caminhadas.

Os arquivos GMAP são semelhantes a uma imagem de disco rígido e são baseados em um sistema de arquivos composto de subarquivos para fornecer funcionalidade para unidades GPS Garmin. Os subarquivos .GMP fornecem um tipo de estrutura de subdiretório, apesar do formato de arquivo .gmap ser plano.
