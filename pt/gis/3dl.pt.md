## O que é um arquivo .3DL?

Um arquivo 3DL é um arquivo iGO Landmark usado pelo software de navegação iGO. Ele armazena informações de localização para pontos de interesse (POIs) e pontos de referência. Essas informações podem incluir o nome, localização e outros detalhes sobre o POI ou ponto de referência. O formato de arquivo iGO Landmark é proprietário e usado apenas pelo [iGO navigation software](https://en.wikipedia.org/wiki/IGO_(software)). Os arquivos 3DL podem ser usados em conjunto com arquivos de dados de mapas para fornecer orientação de navegação aos usuários.

## Formato de arquivo 3DL

O formato de arquivo 3DL é proprietário e não está documentado publicamente. O software de navegação iGO, que utiliza o formato de arquivo 3DL, é desenvolvido e distribuído por uma empresa chamada NNG (anteriormente Nav N Go). Como tal, talvez você não consiga encontrar informações técnicas detalhadas sobre o formato no [NNG website](https://www.nng.com/) ou em outro lugar na Internet.
