## O que é um arquivo .EMBR?

Um arquivo EMBR é um arquivo GIS que foi desenvolvido e usado pela GEOREID Pty Ltd. Ele contém dados geoespaciais, como comprimento e volumes calculados em qualquer ponto específico. O formato de arquivo EMBR armazena esses dados estatísticos em formato de texto simples que pode ser aberto em um editor de texto para visualização e edição. Os sistemas GIS armazenam informações geológicas com referência a qualquer ponto lat-long que representa a posição de absolvição na Terra.

Outros formatos de arquivo GIS populares usados para armazenar dados geoespaciais incluem [gpx](/gis/gpx/), [geojson](/gis/geojson/), [kml](/gis/kml/) e [kmz](/gis/kmz/).

## Formato de arquivo EMBR – Mais informações

Os arquivos EMBR são arquivos de texto que armazenam dados estatísticos de um ponto definido no tempo. Os dados podem ser usados para realizar diversos cálculos, como calcular projeções e comparar dados. Se você tiver um arquivo EMBR, poderá abri-lo em um editor de texto como o Notepad, Notepad++ ou Apple TextEdit para visualizar seu conteúdo e atualizar as informações.
