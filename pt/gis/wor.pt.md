## O que é um arquivo .WOR?

Um arquivo WOR é um arquivo GIS usado pelo software de mapeamento e análise geográfica MapInfo. Ele contém uma área geográfica que contém um mapa visual com dados sobre características geoespaciais, como população e informações demográficas. Você pode carregar um arquivo WOR no software MapInfo para visualizá-lo e realizar diferentes operações nele, como zoom e panorâmica.

## Formato de arquivo WOR

Os arquivos WOR são compostos por múltiplas camadas que são renderizadas em uma ordem específica quando carregadas no MapInfo. Detalhes sobre o formato de arquivo WOR não estão disponíveis publicamente para referência do desenvolvedor. Arquivos WOR podem ser convertidos para QGIS com [QGIS 3.20 Odense](https://www.qgis.org/en/site/forusers/visualchangelog320/index.html#feature-new-embedded-styling-renderer-with-ogr-feature-styles-support).
