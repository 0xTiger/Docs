## O que é um arquivo .MXD?

Um arquivo MXD é um arquivo de documento de mapa usado pelo software ESRI ArcMap. É um formato de arquivo proprietário que armazena diversos tipos de informações, como o layout e desenho dos mapas, os dados utilizados para gerar os mapas e a simbologia aplicada aos dados. O arquivo MXD é, portanto, um formato de arquivo composto que contém vários tipos diferentes de elementos de mapa em um único arquivo. Os arquivos MXD são usados pelo arquivo ArcGIS Map Package ([.mpk](/gis/mpk/)).

## Formato de arquivo MXD

Os arquivos MXD são salvos no formato de arquivo proprietário do ArcGIS e seus detalhes internos não estão disponíveis para referência do desenvolvedor. Os elementos incorporados de um arquivo MXD são organizados em unidades, conhecidas como armazenamentos e fluxos.
