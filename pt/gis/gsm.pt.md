## O que é um arquivo .GSM?

Um arquivo GSM é um arquivo de projeto criado com o software MapViewer que é usado para análise espacial e criação de mapas geográficos. É um arquivo de configurações de mapa que armazena informações de configuração relacionadas às configurações de visualização de mapas, como rótulos de texto, cores, legenda e arquivos de dados referenciados.

## Formato de arquivo GSM

Os arquivos GSM são salvos em formato de arquivo binário e os detalhes da estrutura interna do arquivo não estão disponíveis publicamente para referência do desenvolvedor. O software MapViewer vem com arquivos GSM de amostra e alguns arquivos [DAT](/database/dat/) que são usados para criar outros arquivos GSM.

### Sobre o MapViewer

MapViewer permite trabalhar com arquivos de mapas sobrepondo seus dados em áreas geográficas. Isso ajuda na visualização de dados espaciais, como mapas de densidade populacional, mapas climáticos e vários outros tipos de dados.
