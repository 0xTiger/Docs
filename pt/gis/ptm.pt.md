## O que é um arquivo .PTM?

Um arquivo PTM é um arquivo de mapa usado pelo software de mapeamento MapPoint desenvolvido pela Microsoft, que já foi descontinuado. Ele contém dados de mapas extraídos do Terra Server da Microsoft. As informações dos mapas contêm dados relacionados a rotas e territórios de viagens de negócios. MapPoint foi posteriormente substituído por mapas do Bing. Outros arquivos usados em conjunto com arquivos PTM incluem MPP (MapPoint Map), WPT (MapPoint Waypoint) e RTE (MapPoint Route).

## Formato de arquivo PTM

O formato arquivado PTM é um formato de arquivo proprietário e a estrutura interna dos arquivos .ptm não é documentada publicamente.
