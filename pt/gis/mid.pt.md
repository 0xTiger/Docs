## O que é um arquivo .MID?

Um arquivo MID na categoria de arquivos GIS é um arquivo de dados GIS gerado com o software de análise e visualização de dados espaciais MapInfo. Geralmente é criado junto com o arquivo MapInfo Interchange Format ([MIF](/gis/mif/)) correspondente e tem o mesmo nome. Os arquivos MID armazenam grandes conjuntos de dados que contêm informações de atributos para os objetos no mapa e são importados nos atributos do objeto.

## Formato de arquivo MID - Mais informações

Os arquivos MID são salvos em disco em formato de arquivo binário. Os atributos nos arquivos MID são salvos na mesma ordem do arquivo MIF correspondente. Por exemplo, a terceira entidade geométrica no arquivo .mif possui os atributos armazenados na terceira linha do arquivo .mid.
