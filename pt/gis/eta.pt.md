## O que é um arquivo ETA?

Um arquivo com extensão .eta é um arquivo de marcador do Google Earth criado com o software Google Earth. Ele contém marcadores/informações de marcadores para um local no mundo usando parâmetros como latitude, longitude, rotação, zoom e título. Isso permite que os usuários tenham uma visão panorâmica do local. Com a introdução dos novos formatos de arquivo, ou seja, [.kml](/pt/gis/kml/) e [.kmz](/pt/gis/kmz/), o formato de arquivo ETA foi extinto e substituído por esses novos formatos de arquivo. Os arquivos ETA podem ser abertos no software [Google Earth](https://earth.google.com/web/) que está disponível para download gratuito.


## Formato de arquivo ETA - Mais informações

Os detalhes internos do formato do arquivo ETA não estão disponíveis publicamente e não há software de código aberto ou proprietário disponível para ler as informações dos arquivos ETA.
