## O que é um arquivo .DIX?

Um arquivo DIX é um formato de arquivo proprietário criado com o programa de análise de dados geográficos DIVA-GIS. Consiste em um arquivo de projeto DIVA-GIS (.div) e outras camadas de dados GIS referenciadas pelo projeto. Os arquivos DIX podem ser exportados do DIVA-GIS e compartilhados com outros usuários que podem importá-los para o DIVA. Você também pode usar arquivos DIX para armazenar todos os arquivos relevantes relacionados a um projeto em um único local.

## Formato de arquivo DIX

Um arquivo DIX é armazenado no disco como um arquivo binário compactado. Devido à compactação, os arquivos DIX não ocupam muito espaço em disco e podem ser enviados a outros usuários por e-mail. Um arquivo DIX pode conter dados raster e vetoriais e também pode armazenar tabelas de atributos, metadados e outros dados associados aos dados espaciais.
