## O que é um arquivo .MXT?

Um arquivo MXT é um modelo de mapa geográfico criado com o aplicativo de mapeamento GIS, ArcMap, que vem como parte do ESRI ArcGIS Desktop. Ele é usado para armazenar camadas de mapa base ou layouts de página que podem ser reutilizados para criar mais mapas. Usar modelos economiza seu tempo, pois você não precisa reproduzir as partes comuns dos mapas. Os arquivos MXT podem ser compartilhados com outros usuários/organizações para construir rapidamente outros mapas.

## Formato de arquivo MXT

Os arquivos MXT são salvos como arquivos binários, mas os detalhes da estrutura interna do arquivo não estão disponíveis publicamente para referência do desenvolvedor. Ele contém informações como propriedades básicas de um mapa, posições de layout e visibilidade das barras de ferramentas do aplicativo. Os arquivos MXT permitem aos usuários criar vários mapas com configurações predefinidas já disponíveis para eles.
