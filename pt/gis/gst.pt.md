## O que é um arquivo .GST?

Um arquivo GST é um formato de arquivo de mapa GIS que contém dados geográficos e um mapa visual de uma área mapeada. Ele é usado pelo software de mapeamento **MapX** (desenvolvido pelo PNUMA/GRID-Genebra e pelo PNUMA para gerenciar dados geoespaciais sobre recursos naturais) e **MapXtreme** (um kit de ferramentas de desenvolvimento de software usado para criar aplicativos GIS personalizados) que são plataformas online para gerenciamento de dados geoespaciais sobre recursos naturais. Um arquivo GST contém informações geográficas na forma numérica, bem como imagens raster que são sobrepostas no mapa.

## Formato de arquivo GST – Mais informações

Os arquivos GST são salvos em formato de arquivo proprietário e seus detalhes internos de formato de arquivo não estão disponíveis publicamente para referência do desenvolvedor.
