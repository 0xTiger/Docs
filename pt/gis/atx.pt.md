## O que é um arquivo ATX?

Um arquivo com extensão .atx é um arquivo ESRI ArcGIS que armazena índices de atributos do ArcGIS 8. Existe um arquivo ATX separado para cada índice de atributo [shapefile](/pt/gis/shp/) ou dBase criado no ArcCatalog. Deve-se notar que os índices de atributos do ArcView GIS 3.x para shapefiles e arquivos dBase não são usados pelo ArcMap ou ArcCatalog. No entanto, esses índices de atributos do ArcView GIS 3.x podem ser convertidos em índices de atributos do ArcGIS 8 usando uma ferramenta ArcObjects.

## Formato de arquivo ATX - Mais informações

Não há especificações de formato de arquivo disponíveis para o formato de arquivo ATX para facilitar os desenvolvedores.
