## O que é um arquivo .DT1?

Um arquivo DT1 é um arquivo Digital Terrain Elevation Data (DTED) que contém dados do terreno espaçados uniformemente. É usado principalmente por aplicativos de software GIS militares para obter dados de elevação, inclinação e contorno de superfície. O arquivo DT1 é semelhante a [DT0](/gis/dt0/) e [DT2](/gis/dt2/), mas com granularidade média de informações.

Os arquivos DT1 podem ser abertos com aplicativos GIS, como ESRI ArcGIS Pro, TatukGIS, Blue Marble Geographic Global Mapper e Pitney Bowes MapInfo.

### Formato de arquivo DT1

Os arquivos DT1 são salvos como arquivos binários em disco e podem ser lidos usando muitos aplicativos GIS. Eles podem ser mesclados em um Catálogo Raster de Arquivo Geodatabase que pode ser usado para criar um grande mosaico.
