## O que é um arquivo .WLX?

Um arquivo WLX é um arquivo GIS criado pelo software MapSource que vem com os produtos de mapas Garmin. Ele contém uma localização geográfica (na forma de latitude-longitude) em formato XML e uma localização mundial para um site de mapeamento como Google Maps ou Bing Maps. O arquivo WLX atua como um link de atalho e abre o local armazenado em sites de terceiros na posição de latitude-longitude registrada.

Outro arquivo semelhante (embora não relacionado ao GIS) é o formato de arquivo [URL](/web/url/) que armazena o endereço da web e direciona os usuários para este URL quando aberto.

## Formato de arquivo WLX - mais informações

Os arquivos WLX são armazenados em formato de arquivo de texto que pode ser aberto em qualquer editor de texto para visualizar e atualizar seu conteúdo. MapSource é um software baseado em GIS que permite visualizar mapas, pontos de referência, rotas e trilhas. Você também pode transferir esses dados espaciais de ou para um dispositivo GPS Garmin.

O MapSource não é mais suportado pela Garmin e foi substituído pelo BaseCamp, que está disponível para download gratuito.
