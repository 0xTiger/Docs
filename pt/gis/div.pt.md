## O que é um arquivo DIV?

Um arquivo com extensão .div é um arquivo de projeto criado com o software [DIVA-GIS](https://en.wikipedia.org/wiki/DIVA-GIS) quando um novo projeto de mapeamento é criado e salvo. DIVA-GIS é um software de mapeamento que é usado para criar mapas espaciais e aplicativos de análise de dados geográficos. O arquivo de projeto consiste em uma imagem de mapa e camadas que são usadas para exibir os objetos e propriedades no mapa. Além das informações espaciais, também armazena informações sobre a escala e o centro do mapa.

## Formato de arquivo DIV - Mais informações

Os arquivos DIV são armazenados em disco como formato de arquivo proprietário DIVA-GIS e seu formato de arquivo interno não está disponível para referência do desenvolvedor.
