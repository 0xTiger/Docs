## O que é um arquivo 3DD?

Um arquivo com extensão .3dd é um arquivo de dados GIS criado com o software aplicativo ESRI ArcGlobe. Ele armazena informações para caminhos onde os dados 3D estão localizados na forma de geodatabase. Quando um documento é aberto usando o arquivo .3dd, ele aparecerá na mesma forma em que foi salvo. ArcGlobe é um aplicativo de visualização 3D que permite visualizar grandes quantidades de dados GIS em uma superfície de globo. ArcGlobe é um dos vários aplicativos que vêm vinculados ao pacote de software ArcGIS.

## Formato de arquivo 3DD - Mais informações

Os arquivos 3DD são armazenados em disco em formato de arquivo binário e o formato de arquivo interno desses arquivos não está disponível publicamente.
