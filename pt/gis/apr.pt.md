## O que é um arquivo APR?

Um arquivo com extensão .apr é um arquivo de projeto ESRI ArcView que contém informações de projeto para um projeto ESRI. Estes foram usados pelo ArcView 3.xe não são suportados pelas novas versões (9.xe 10.x) do ArcGIS. O ArcView 10 pode, no entanto, importar arquivos APR. ArcView é um Sistema de Informação Geográfica (GIS) que é usado para criar, visualizar, analisar e gerenciar dados geográficos. Existem outros formatos de arquivo que são usados pela ESRI para armazenar dados, como o popular ESRI ShapeFile ([.shp](/pt/gis/shp/)). Os arquivos APR podem ser abertos com qualquer editor de texto, como o Microsoft Notepad e o Notepad++.

## Formato de arquivo APR - Mais informações

Os arquivos APR são armazenados em disco como arquivos de texto simples que podem ser abertos com qualquer editor de texto para exame e edição.
