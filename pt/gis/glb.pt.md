## O que é um arquivo GLB?

Um arquivo com extensão .glb é um arquivo globalmente compartilhável gerado e usado pelo software AGI Systems Tool Kit (STK). Ele contém informações do globo 3D que podem ser compartilhadas com outros usuários para serem usadas em seus projetos. O software STK da AGI Systems é usado para simular missões aeroespaciais, sistemas de defesa e sistemas de telecomunicações. GLB é o formato de arquivo padrão para STK e muitos modelos 3D diferentes, como [.gltf](/pt/3d/gltf/) podem ser convertidos para este formato. Os arquivos GLB podem ser abertos com o software AGI Simulation Toolkit, bem como qualquer editor de texto.

## Formato de arquivo GLB - Mais informações

Os arquivos GLB armazenam informações sobre texturas, imagens e outros arquivos usados em um globo para um projeto. As referências à localização dos arquivos vinculados são armazenadas nos arquivos GLB e esses arquivos adicionais também devem ser compartilhados com outros usuários durante o compartilhamento do arquivo GLB globo para carregamento adequado. Os arquivos GLB armazenam todas essas informações em arquivos de texto simples que podem ser abertos com qualquer editor de texto.
