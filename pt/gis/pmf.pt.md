## O que é um arquivo .PMF?

Um arquivo PMF é um arquivo de mapa somente leitura gerado com ArcMap. Você pode dizer que é uma versão mais curta do formato de arquivo MXD que possui informações limitadas para serem compartilhadas com usuários que não possuem uma versão paga do software ArcGIS. Isso permite o compartilhamento de informações geográficas com outras pessoas ao mesmo tempo. Os arquivos PMF podem ser compartilhados com outros usuários por e-mail devido ao seu tamanho pequeno devido às informações restritas que contêm.

Os arquivos PMF podem ser abertos com ESRI ArcReader e ESRI ArcGIS Pro.

## Formato de arquivo PMF

Os arquivos PMF limitam o nível de dados acessíveis aos usuários com base em sua assinatura. Se um usuário estiver usando uma versão gratuita do software ArcGIS, os arquivos PMF limitarão a exibição de informações, como visualizações de dados, layouts, navegação, impressão, consulta, panorâmica e recursos de zoom do mapa. Os dados podem não estar realmente contidos no arquivo de limite, mas sim acessados de um local remoto hospedado no servidor ArcMap ou disponível na Internet.
