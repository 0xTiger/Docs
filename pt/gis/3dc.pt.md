## O que é um arquivo .3DC?

Um arquivo 3DC é um arquivo de dados GPS usado pelo aplicativo de software de navegação GPS, [iGO](https://en.wikipedia.org/wiki/IGO_(software)). Ele contém informações de dados de construção 3D junto com metadados para exibição do aplicativo. Os arquivos 3DC são salvos no cartão de memória da unidade de navegação e, em sua maioria, armazenados no diretório \​iGO\​content\​building\​.

## Sobre a Navegação iGO

O software iGO é usado para fins de navegação. Obtém informações de localização do usuário por meio de dispositivos GPS e auxilia na navegação por meio de mapas e rotas 3D. Está disponível para Android, iOS e outros dispositivos portáteis populares. Além de seus recursos de navegação, o software iGO também fornece armazenamento offline de mapas caso os usuários tenham dificuldade para se conectar à Internet para obter dados de mapas em tempo real.
