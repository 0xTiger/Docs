## O que é um arquivo AEP? ##

O arquivo com extensão .aep (que significa **Activ E-Book Project**) é classificado como livro eletrônico (eBook). Os arquivos AEP são criados usando o mesmo mecanismo do Internet Explorer para garantir 100% de compatibilidade. Ele contém as páginas como uma pessoa pode ler um livro físico. As páginas dos arquivos AEP geralmente incluem texto, imagens, animações e hiperlinks. Além disso, ele suporta gráficos [HTML](/pt/web/html/), [DHTML](/pt/web/dhtml/), [JPEG](/pt/image/jpeg/) e [PNG](/pt/image/png/), animações , [JavaScript](/pt/web/js/), VBScript e muitos dos plug-ins do Internet Explorer. Um arquivo AEP pode ser salvo como um arquivo executável do Windows (EXE) para que possa ser aberto no Windows95 ou em versões posteriores do Microsoft Windows.
 


## Formato de arquivo AEP ##

Os arquivos de formato AEP são criados usando um software AEC (Active E-Book Compiler). Este software está usando um mecanismo semelhante ao IE para criar eBooks. O usuário pode criar cópias ilimitadas de eBooks sem pagar nenhum custo pelo uso do programa. O processo de distribuição também é fácil, pois cada eBook é composto por um programa Windows independente, que pode ser distribuído em qualquer mídia como e-mails, download da Internet, disquete, CD-ROMs, discos ZIP, etc.

## Possíveis problemas ao abrir o arquivo

Se você não estiver conseguindo abrir e executar o arquivo AEP; isso não significa que você não tenha um software adequado instalado no seu dispositivo. Pode haver alguns outros problemas que impedem que o arquivo funcione corretamente. Os possíveis problemas podem ser um dos seguintes:

- Corrupção de um arquivo AEP
- Links errados para o arquivo AEP nas entradas do registro
- Descrição excluída do AEP do registro do Windows
- Instalação corrompida de um aplicativo que suporta o formato AEP
- Um arquivo AEP infectado com um malware indesejável
- O computador não tem recursos de hardware suficientes para operar o arquivo AEP
- Os drivers usados pelo computador para abrir um arquivo AEP estão desatualizados
