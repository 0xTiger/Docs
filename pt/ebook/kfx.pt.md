## O que é um arquivo KFX?

Um arquivo KFX é um eBook criado no Amazon Kindle File Format 10 (KF10) para dispositivos Kindle. As versões mais antigas de dispositivos KIndle usam os formatos de arquivo [AZW](/pt/ebook/azw/) e [AZW3](/pt/ebook/azw3/) (KF8). KFX é um aprimoramento das versões anteriores com recursos adicionais especificamente em termos de apresentação de layout de conteúdo. Ele apresenta novos recursos, como suporte para hífens suaves, usando um novo mecanismo de composição aprimorado. Ele também apresenta um novo formato de arquivo de imagem chamado JXR, que possui uma taxa de compactação mais alta, e a nova fonte Bookerly, que é uma fonte exclusiva do Kindle, proporcionando melhor legibilidade em dispositivos digitais. O formato vem com recursos aprimorados, como miniaturas de várias páginas, ajuste automático de conteúdo em torno de imagens e vídeos em locais fixos, resultando em layout de página e DRM aprimorados. Para abrir e visualizar esses arquivos, você pode usar o aplicativo Amazon Kinde disponível para Windows, macOS, Android e iOS. O Calibre é outro aplicativo de software de leitura de eBooks que pode ser usado para ler vários formatos de arquivos de eBooks diferentes.

## Formato de arquivo KFX ##

O KFX é um formato complexo, pois combina vários recursos diferentes dos formatos de arquivo Kindle, mas os detalhes internos sobre sua estrutura de arquivo não são conhecidos. A Amazon adicionou seu esquema DRM para proteger os direitos digitais e salvar o formato KFX da antipirataria.
