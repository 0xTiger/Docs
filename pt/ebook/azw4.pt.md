## O que é um arquivo AZW4?

O AZW4 é um ebook Amazon Print Replica que a Amazon desenvolveu para o software de desktop Kindle e dispositivos Kindle eReader. O AZW4 não é semelhante ao [MOBI](/pt/ebook/mobi/), [AZW](/pt/ebook/azw/) ou [AZW3](/pt/ebook/azw3/). Esse formato é na verdade um wrapper para e-books [PDF](/pt/pdf/) ou MOBI com DRM adicionado.

## Formato de arquivo AZW4

O **AZW4** foi apresentado como uma nova versão do software **Textbooks** semelhante ao **PDF**. Um eBook AZW4 contém um arquivo PDF completo dentro, ou pode ser considerado como um arquivo PDF com um wrapper chamado AZW4. Principalmente o AZW4 está sendo usado pelo Kindle eReader. No entanto, outros aplicativos também podem usar a extensão de arquivo .azw4. Até mesmo programas de malware podem criar arquivos .azw4. Seja cauteloso com arquivos .azw4 vindos de uma fonte duvidosa.
