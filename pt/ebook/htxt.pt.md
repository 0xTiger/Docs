## O que é um arquivo HTXT? ##

O formato HTXT foi projetado pela primeira vez por Hanvon e categorizado no tipo de arquivo de texto. Geralmente é usado para arquivos Hanvon eBook. O formato Hanvon eBook é suportado apenas pelo Windows ou compatível com o software que pode ser facilmente instalado no Windows. O software mais comum que provisiona arquivos HTXT é Transferir para o dispositivo via conexão USB. O formato de eBook usado pelos dispositivos Hanvon WISEreader eReader geralmente fornece texto em chinês e é usado para livros em chinês, salva em um formato duplo e pode integrar a segurança de DRM, não pode ser lido por programas de software de desktop, mas pode ser transferido para o dispositivo via conexão USB.


## Problema para abrir um arquivo HTXT? ##

Aqui está a lista de alguns problemas relevantes que podem surgir e achar difícil abrir este formato de arquivo:

* Arquivo vazio ou incompleto. Isso propõe cortar o conteúdo do arquivo HTXT ao baixar ou copiar. A única possibilidade de restauração é baixar o arquivo novamente
* O arquivo pode estar corrompido ou conter vírus. A melhor opção é o uso de um programa antivírus atualizado para esclarecer quaisquer dúvidas
* Uma falha de ligação distinta. O programa não fornece a versão desatualizada do arquivo HTXT. A solução mais fácil para este erro é na forma de um programa de adaptação ou uma versão mais antiga do programa
