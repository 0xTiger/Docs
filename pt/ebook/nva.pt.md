## O que é um arquivo NVA?
Um arquivo NVA é um e-book como um arquivo [PDF](/pt/pdf/). Este formato de arquivo é específico para **NVA Reader**, software para leitura dos livros eletronicamente. Os arquivos NVA são comumente usados para armazenar livros relacionados à medicina. O livro contido no arquivo é organizado de forma semelhante à forma como o livro aparece na forma impressa. No entanto, esse formato de arquivo tornou-se quase obsoleto.

## Sobre o NVA Reader

O NVA Reader é um bom software e funciona bem. É um programa fácil de usar que oferece um método realmente simples e conveniente de abrir e ler arquivos de eBooks NVA. Na verdade, esta ferramenta é tão fácil de usar que até mesmo os iniciantes podem usá-la sem nenhum problema. As opções básicas que permitem aos usuários girar ou ampliar e reduzir as páginas do eBook; selecione, copie e destaque o texto; ou até mesmo para encontrar strings de texto específicas. Além disso, os usuários podem até adicionar notas e marcadores ou imprimir páginas diretamente deste aplicativo de software.

O NVA Reader é leve e funciona sem problemas e, embora seja otimizado para dispositivos móveis, também funciona perfeitamente em computadores Windows. Os usuários podem abrir e ler até mesmo eBooks de 1 Gb ou maiores sem nenhum problema.

Conclusivamente, o NVA Reader é uma excelente ferramenta que torna a leitura de eBooks NVA uma experiência fácil, simples e maravilhosa.
