## O que é um arquivo WEBZ? ##

Uma extensão de arquivo WEBZ é um formato de arquivo muito popular e é completamente conhecido como o arquivo de eBook compactado WEBZ. O sufixo EBZ é geralmente usado para arquivos de formato de eBook compactado WEBZ e é fundamentalmente usado como um arquivo ZIP de páginas da Web e imagens. A Web Books Publishing é autora e demarca a qualidade do formato WEBZ. Este formato de arquivo é compatível com software que pode ser conectado a uma plataforma de sistema da Web. O arquivo WEBZ vai para o tipo de arquivo de texto, assim como outras extensões de nome de arquivo. O Web Books Viewer é o programa mais usado para trabalhar com esses formatos de arquivo. Esta extensão de arquivo é flexível e pode ser alterada de ".webz" para ".zip".
