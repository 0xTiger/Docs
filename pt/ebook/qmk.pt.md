## O que é um arquivo QMK?

Um arquivo com a extensão .qmk é geralmente conhecido como o formato **YanCEyDesktop Quickmarks**. O YanCEyWare introduziu o formato de arquivo YanCEyDesktop Quickmarks. O arquivo QMK está relacionado ao arquivo [YBK](/pt/ebook/ybk/) e é suportado por um Programa que está disponível para as máquinas que executam o sistema operacional Windows. Um arquivo com uma extensão QMK é classificado como um arquivo do tipo texto que permite que os usuários naveguem facilmente para locais salvos em eBooks. YanCEyDesktop é o software recomendado para gerenciar arquivos QMK. O software **YanCEyDesktop** foi feito por **D Remington**.

## Problemas para abrir um arquivo QMK

Se você não conseguir abrir e executar o arquivo QMK; isso não significa que você não tenha o software adequado instalado no seu dispositivo. Pode haver alguns outros problemas que impedem que o arquivo funcione corretamente. Os possíveis problemas podem ser um dos seguintes:

- Corrupção de um arquivo QMK
- Links errados para o arquivo QMK nas entradas do registro
- Descrição excluída do QMK do registro do Windows
- Instalação corrompida de um aplicativo que suporta o formato QMK
- Um arquivo QMK infectado com malware indesejável
- O computador não possui recursos de hardware suficientes para operar o arquivo QMK
- Os drivers usados pelo computador para abrir um arquivo QMK estão desatualizados
