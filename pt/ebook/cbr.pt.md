## O que é um arquivo CBR?

Um arquivo com extensão .cbr é uma coleção de imagens armazenadas como arquivos binários compactados usando o formato de arquivo de compactação RAR. Ele contém imagens em imagens [JPEG](/pt/image/jpeg/), [PNG](/pt/image/png/) ou [GIF](/pt/image/gif/) que representam imagens em uma história em quadrinhos e são mostradas ao usuário quando o arquivo é usado em leitores de ebook como Calibre. O CBR é semelhante ao [CBZ](/pt/ebook/cbz/), exceto que o CBZ é compactado e armazenado no formato de arquivo [ZIP](/pt/compression/zip/). Esses arquivos podem estar contidos em arquivos de coleção de quadrinhos, como [CBC](/pt/ebook/cbc/).

## Formato de arquivo CBR

Um arquivo CBR é um arquivo [RAR](/pt/compression/rar/) que é armazenado como um arquivo binário no disco. As imagens dentro de um arquivo CBR são armazenadas e exibidas aos usuários de forma ordenada quando abertas por um leitor de ebook. Os arquivos CBR podem ser descompactados usando qualquer software de descompactação que suporte o formato de arquivo RAR. No entanto, as imagens descompactadas dessa maneira não estão na mesma ordem em que aparecem na tela visual quando abertas no software do leitor.
