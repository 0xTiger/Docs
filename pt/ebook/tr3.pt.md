## O que é um arquivo TR3? ##

O arquivo TR3 é um eBook TomeRaider 3 que fornece suporte para pesquisa e compactação rápidas. Isso pode funcionar corretamente através do programa relacionado ao TomeRaider. A Yadabyte é a desenvolvedora deste software, uma empresa britânica de software e desenvolvimento web com sede no Reino Unido. O aplicativo leitor de eBook TomeRaider pode ser usado para o bom funcionamento e gerenciamento do conteúdo desses arquivos TR3. Este leitor de eBook relacionado pode ser instalado em computadores que executam sistemas baseados no Microsoft Windows e em muitos gadgets transferíveis. O arquivo TR3 pertence ao grupo de arquivos eBook usados em sistemas operacionais como Windows 10, Windows 7, Windows 8 / 8.1, Windows Vista, Windows XP.

**Tags HTML** específicas de TR3 são as seguintes:

|Tag|Descrição|
---|---|
|\<new> |Para criar arquivos TomeRaider a partir de arquivos de texto, basta a tag New.<new> tag define o início das páginas.|
|\<CATDEF> ... \</CATDEF> |define o nome da categoria|
|\<META> ... \</META> |é uma tag usada para definir todos os metadados usados no formato de arquivo. Essa tag inclui vários parâmetros.|
|\<EXPMSG> |Esta tag controla a mensagem que aparece quando um arquivo expirou. Sempre que um usuário tenta exibir uma página após a expiração do arquivo, a mensagem de expiração aparece em vez do texto da página real.|
|\<DIEMSG> |Esta tag é semelhante a EXPMSG. Ele é usado para exibir uma mensagem quando o arquivo morre.|
|\<ENGMSG> |Esta tag é usada para criar a mensagem exibida para páginas criptografadas. A mensagem aparece em vez do texto da página se o usuário tentar abrir uma página criptografada antes de desbloqueá-la.|
|\<expmsg> ,\<diemsg> e \<encmsg> |Ignorado no texto da página. Eles devem ser colocados na seção de cabeçalho do arquivo antes do primeiro<new> etiqueta.|
|\<SELECTION> . \</ SELECTION> |Usado para dar suporte a Consultas. Esta tag deve estar antes da primeira<new> marcação. Ele compila os dados de seleção para a consulta do usuário.|
|\<catset> . \</catset> | Usado para atribuir nomes de categoria para cada assunto.|


## Problemas para abrir um arquivo TR3 ##

Aqui está a lista de alguns problemas comuns que podem surgir e causar mau funcionamento do formato de arquivo:

* Arquivo corrompido
* Arquivo infectado devido a vírus
* Links impróprios para o arquivo TR3 em acessos de arquivo
* Sem direito de acesso no sistema para abrir os arquivos
* Unidade desatualizada em seu sistema
* Remoção do relatório do TR3 do arquivo do Windows
* Entrar em contato com o desenvolvedor pode ser uma opção melhor para uma melhor saída
