## O que é um arquivo POBI?

Um arquivo POBI pertence ao tipo de arquivo eBook; é na verdade um **Dados Periódicos do Kindle Touch**. Este formato de arquivo também é conhecido como periódico [MOBI](/pt/ebook/mobi/); usado no dispositivo de leitura eletrônica Amazon Kindle Touch. Embora o tipo de arquivo dos arquivos POBI seja um eBook, não podemos dizer que o arquivo com extensão .pobi seja um eBook; porque não tem o conteúdo como um eBook. Ele apenas contém os dados como uma revista e funciona como um arquivo MOBI. No entanto, se a extensão .pobi for alterada para .mobi, o índice de seções e artigos da revista não aparecerá.

## Problemas para abrir um arquivo POBI ##

Se você não estiver conseguindo abrir e executar o arquivo POBI; isso não significa que você não tenha o software adequado instalado no seu dispositivo. Pode haver alguns outros problemas que impedem que o arquivo funcione corretamente. Os possíveis problemas podem ser um dos seguintes:

- Corrupção de um arquivo POBI.
- Links errados para o arquivo POBI nas entradas do registro.
- Descrição excluída do POBI do registro do Windows
- Instalação corrompida de um aplicativo que suporta o formato POBI
- Um arquivo POBI infectado com malware indesejável.
- O computador não tem recursos de hardware suficientes para operar o arquivo POBI.
- Os drivers usados pelo computador para abrir um arquivo POBI estão desatualizados.
