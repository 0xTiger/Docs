## O que é um arquivo YBK? ##

O formato de arquivo YanCEyWare eBook e o Microsoft Encarta usam extensões de arquivo YBK. O eBook YanCEyWare foi formado pela YanCEyWare. Todos os aplicativos de software estão disponíveis para dispositivos que executam o Windows, suportam arquivos YBK. A Enciclopédia Encarta pode ser atualizada por arquivos .ybk sem precisar atualizar todo o programa. YanCEyDesktop é um programa conhecido e popular que pode gerenciar e manipular facilmente arquivos YBK. O YanCEyWare Reader usa este formato de arquivo YBK extensivamente, como um aplicativo de leitor de eBook para o Windows Mobile ou Pocket PC. Este arquivo foi desenvolvido com grandes recursos com hiperlinks.

## Problemas para abrir um arquivo YBK ##

Às vezes, quando é difícil abrir e funcionar o arquivo YBK, aqui está a lista de alguns possíveis problemas:

* Abrindo um arquivo YBK corrompido
* Links impróprios e impróprios para o arquivo YBK em acessos de registro
* Remoção não intencional da descrição do YBK do arquivo do Windows
* Instalação parcial de um aplicativo que provisiona o formato YBK
*	Infected YBK file with an undesirable malware
