## O que é um arquivo EBK?
Uma extensão EBK é usada pelo software **eBook Pro Viewer** ao salvar um arquivo de eBook. O eBook Pro é um software que pode criar eBooks e proteger informações protegidas por direitos autorais. Um arquivo EBK contém a versão criptografada e compactada de um e-book que pode ser lido usando o **eBook Pro Viewer**; após fornecer as credenciais apropriadas para autenticação.
