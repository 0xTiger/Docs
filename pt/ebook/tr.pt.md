## O que é um arquivo TR? ##

Um arquivo com extensão .tr é um arquivo de eBook TomeRaider criado com o software TomeRaider eReader. Esses arquivos podem ser abertos em dispositivos Windows e Handheld, como Android, Palm OS e EPOC, e são categorizados como arquivos de texto. O software TomeRaider desenvolvido pelo site [Yadabyte](https://www.yottabyte.ltd/) e o site oficial fornecem mais informações sobre os arquivos TR. O formato de arquivo TR2 foi substituído pelo formato TR3 (TomeRaider 3). Os arquivos TR oferecem busca rápida e altos níveis de compactação.

## Breve História do Formato de Arquivo TR

O TomeRaider foi desenvolvido por uma empresa de software e desenvolvimento web sediada no Reino Unido Yadabyte em 1999. A empresa levou ainda mais longe para o TomeRaider versão 3.

## Formato de arquivo TR

A versão mais recente do formato de arquivo TR é a versão 3, que oferece suporte para pesquisa e compactação rápidas. Ele comprime arquivos de texto para 45-60% do tamanho original. Isso permite que os leitores de e-books do TomeRaider realizem pesquisa, indexação e compactação rápidas. O formato de arquivo TR é adequado para documentos de referência grandes e a maioria dos documentos requer uma chave DRM para ser criptografada.

## Problemas para abrir um arquivo TR

Pode haver alguns problemas comuns devido aos quais você não consegue abrir o arquivo:

* Ausência de software de apoio
* Versão de software desatualizada
* Arquivo TR corrompido
* Links impróprios para o arquivo TR em acessos ao arquivo
* Sem direito de acesso no sistema para abrir os arquivos
* Unidade desatualizada em seu sistema
* Remoção do relatório do TR do arquivo do Windows
* Instalação parcial de um aplicativo compatível com o formato TR
* Arquivo TR infectado
* Espaço insuficiente para recursos de hardware para auxiliar na abertura do arquivo TR
