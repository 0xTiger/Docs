## O que é um arquivo TCR?

No início dos anos 90, o formato de arquivo TCR foi introduzido para os antigos dispositivos palmtop **Psion Series 3**. A empresa usou sua compactação específica da plataforma e esse formato tem suporte muito limitado para formatação de texto. É um formato de arquivo baseado em ZVR e oferece uma taxa de compactação comparativamente melhor que o PalmDOC. Também funciona muito bem com FontMachine. Embora o formato de arquivo TCR pertença a um dispositivo descontinuado, ainda assim esse formato de arquivo pode ser lido usando alguns eReaders. Este formato de arquivo também pode ser visualizado em um desktop com Sumatra PDF e com o aplicativo Caliber no Windows e Linux.

## Detalhes técnicos

Como o formato de arquivo TCR é baseado em um compressor ZVR que é capaz de reduzir muitos arquivos em cerca de 50% do tamanho original. O esquema de compressão usado pelo over é um pouco antigo. O arquivo compactado começa com um dicionário dos 256 símbolos possíveis que podem aparecer no arquivo compactado. O dicionário contém uma string alternativa fixa para cada um desses símbolos. A descompressão tornou-se muito rápida, mesmo em OPL, e pode começar em qualquer ponto do arquivo compactado.

## Problemas para abrir um arquivo TCR ##

Se você não conseguir abrir e executar o arquivo TCR; isso não significa que você não tenha o software adequado instalado no seu dispositivo. Pode haver alguns outros problemas que impedem que o arquivo funcione corretamente. Os possíveis problemas podem ser um dos seguintes:

- Corrupção de um arquivo TCR.
- Ligações erradas para o arquivo TCR nas entradas do registro.
- Descrição excluída do TCR do registro do Windows
- Instalação corrompida de um aplicativo que suporta o formato TCR
- Um arquivo TCR infectado com malware indesejável.
- O computador não possui recursos de hardware suficientes para operar o arquivo TCR.
- Os drivers usados pelo computador para abrir um arquivo TCR estão desatualizados.
