## O que é um arquivo FKB? ##

FKB é uma extensão de arquivo de eBook que foi desenvolvida pela Flipkart.com para o aplicativo de software Flipkart. Esse tipo de extensão de arquivo geralmente é encontrado em usuários do Windows 10. Há uma especificação no arquivo FBK que pode variar em tamanho de algumas centenas de kilobytes a vários megabytes, dependendo do tamanho do livro e do número de imagens incluídas. Para baixar o arquivo do Aplicativo Flipkart eBooks para Android, primeiro, deve-se comprar o eBook, depois clicar nele na Biblioteca e o livro começará a ser baixado. Se você não conseguir ver o documento adquirido, atualize o sistema para um melhor funcionamento.

## Sistemas Operacionais Suportados ##

Os seguintes sistemas operacionais suportam arquivos FKB:

* Windows 7
* Windows 8
*Windows10
* Windows Server 2003/2008/2012/2016
* Mac OS X
* FreeBSD
*iOS
*Linux
*Android
* NetBSD
* OpenBSD

## Problemas para abrir um arquivo FKB ##

* Instalação inadequada de programas de suporte. Certifique-se de que foi baixado corretamente com uma versão atualizada
* Pouco espaço de hardware
* Arquivos corrompidos ou infectados. Este problema pode ser facilmente resolvido através de um antivírus instalado em seu sistema
