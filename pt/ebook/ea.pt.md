## O que é um arquivo EA? ##

A extensão .ea é usada para o arquivo **Kindle End Action**, que é um dos formatos de arquivo Kindle. Esses arquivos são criados pelo **dispositivo de leitura eletrônica Amazon Kindle** para mostrar algo que um autor deseja exibir no final de um e-book. Um arquivo EA contém as informações no formato [XML](/pt/web/xml/). Portanto, este arquivo pode ser aberto ou visualizado usando qualquer editor de texto. Este arquivo pode ser encontrado onde os dispositivos de leitura eletrônica Kindle salvam os e-books. O conteúdo nele pode ser tags XML vazias se o cliente não tiver definido as informações a serem exibidas no final de um eBook.

## Um pouco mais sobre o formato de arquivo EA ##

Um arquivo com extensão .ea é um arquivo XML que contém os dados dos clientes que compraram o livro e também compraram um recurso para mostrar mais sobre o autor após a última página do e-book; incluindo **ASIN** ou **ISBN** para que possa ir para a página do título na Kindle Store.

### O que é um ASIN/ISBN? ###

O ASIN significa **Amazon Standard Identification Number**, que na verdade é um código alfanumérico de dez dígitos para identificar exclusivamente os produtos na Amazon. O ASIN é um identificador exclusivo para cada produto e é atribuído quando você cria um produto Amazon. Todo e qualquer produto na Amazon tem um ASIN com exceção dos livros. A Amazon usa o ISBN (International Standard Book Number) no lugar de um ASIN. Esse identificador da Amazon para seus produtos é necessário antes de vendê-los na Amazon.

## Problema para abrir um arquivo EA?

Se você não conseguir abrir e executar o arquivo EA; isso não significa que você não tenha um software adequado instalado no seu dispositivo. Pode haver alguns outros problemas que impedem que o arquivo funcione corretamente. Os possíveis problemas podem ser um dos seguintes:

- Corrupção de um arquivo EA.
- Links errados para o arquivo EA nas entradas do registro.
- Descrição excluída do EA do registro do Windows
- Instalação corrompida de um aplicativo que suporta o formato EA
- Um arquivo EA infectado com um malware indesejável.
- O computador não possui recursos de hardware suficientes para operar o arquivo EA.
- Os drivers usados pelo computador para abrir o arquivo EA estão desatualizados.
