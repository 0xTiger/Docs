## O que é um arquivo EAL? ##

Os arquivos **Kindle End Action** geralmente são encontrados com a extensão .EAL. Esses arquivos consistem em um xml que é escrito para mostrar algo sobre o autor após a última página do eBook.

## Formato de arquivo EAL ##

O arquivo .EAL é um arquivo xml que consiste nos dados dos clientes que compraram o livro e também compraram um recurso para mostrar algo mais sobre o autor do e-book após a última página do livro; O ASIN também está incluído para ir para a página do título na Kindle Store.
