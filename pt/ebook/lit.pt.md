## O que é um arquivo LIT?

Os arquivos LIT pertencem ao antigo **Microsoft Reader** que era usado para ler e-books. Os eBooks exibidos pelo Microsoft Reader têm a extensão **.lit** que é abreviada para **Literature**. O formato de arquivo LIT consiste em um arquivo Microsoft ITOL/ITLS HTML Help 2.0, que oferece todos os tipos de recursos incríveis, como compressão [LZX](/pt/compression/lzx/), a representação binária de material de marcação e dados auxiliares opcionais arbitrários associados a cada fluxo. O texto do livro é um número arbitrário de fluxos de marcação OEBPS 1.0 e um subconjunto de CSS OEBPS 1.0. As imagens são [GIF](/pt/image/gif/), [JPEG](/pt/image/jpeg/) e [PNG](/pt/image/png/).

## Quanto tempo o Microsoft Reader exibiu os arquivos LIT

O **Microsoft Reader** é um aplicativo para leitura de e-books que foi descontinuado pela Microsoft. A primeira versão do Microsoft Reader foi lançada em agosto de 2000, que usava seu próprio formato LIT. Estava disponível para PDAs e computadores Windows. O nome do aplicativo também foi usado posteriormente para um aplicativo (que foi lançado inicialmente com o Windows 8) para ler arquivos [PDF](/pt/pdf/) e [XPS](/pt/page-description-language/xps/). O aplicativo Microsoft Reader foi descontinuado em 2018.
