## O que é um arquivo OPF? ##

O OPF significa Formato de Pacote Aberto. Este formato de arquivo é na verdade um arquivo [XML](/pt/web/xml/), mas com extensão **.opf**. Este formato de arquivo é um OEB Package Format (OPF) que pode conter uma revista digital ou outra publicação que pode ser lida com o **FlipViewer**. Também inclui metadados que descrevem o conteúdo e uma lista de páginas, imagens e texto referenciados pela publicação.

## Detalhes técnicos do formato de arquivo OPF ##

A extensão de arquivo .opf pode ser categorizada em vários tipos de arquivos. Pode ser um arquivo formatado com o padrão definido pelo Open Packaging. Os OPFs são mais do que arquivos de eBook, pois contêm principalmente informações sobre o eBook, desde o título até o escritor, bem como outras meta-informações sobre ele. É por isso que esses arquivos podem ser abertos usando o software Adobe Reading. Além disso, é usado para arquivos criados pelo programa FlipAlbum. Este programa é usado para criar álbuns de fotos digitais e esses arquivos OPF referem-se ao álbum de fotos criado usando o software. Esses arquivos contêm configurações do álbum; sua estrutura e referências de arquivo de imagem, e pode ser compartilhado com outras pessoas também. Os arquivos OPF também estão associados ao **FlipBook**. Um arquivo OPF pode ser editado com qualquer editor de texto (por exemplo, Notepad ou Notepad++) porque eles podem ser formatados usando o formato XML.
