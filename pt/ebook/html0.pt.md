## O que é um arquivo HTML0? ##

HTML0 é o formato de eBook nativo usado pelo **Book Designer** com código HTML e CSS padrão que descreve o conteúdo e o design de cada página. **Book Designer** é o programa mais comum para funcionar com arquivos HTML0 e para alterar documentos de texto em vários formatos de eBook. O formato de arquivo HTML0 é compatível com software que pode ser instalado no sistema Windows, pois pertence à categoria de arquivo de texto. Os arquivos HTML0 compreendem HTML normal e podem ser vistos e alterados usando um editor de texto. Arquivos HTML0 não podem ser vinculados a arquivos CSS discretos em comparação com arquivos HTML comuns.

## Problema para abrir um arquivo HTML0? ##

* O arquivo pode estar infectado quando você achar difícil abri-lo
* Deve verificar o arquivo em busca de vírus por um scanner antivírus online
* Alguns arquivos precisam de direitos de acesso para abri-los
* Se os sistemas tiverem recursos inadequados para abrir arquivos HTML0, feche todos os aplicativos e reabra-os
* O arquivo pode estar corrompido ou apresentar algum erro caso você o tenha baixado ou recebido de qualquer e-mail
* Sempre use o sistema e os drivers mais recentes para tornar seu computador mais seguro e resolver todos os problemas relacionados às extensões de arquivo
