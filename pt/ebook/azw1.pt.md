## O que é um arquivo AZW1? ##

O AZW1 é um formato de arquivo de eBook que também é conhecido como **Amazon Kindle Topaz eBook** que foi lançado via **Whispernet**. Esse formato de arquivo pode ser usado com o software Kindle em PCs e smartphones e é responsável por armazenar eBooks baixados da rede 3G Whispernet da Amazon.

## Formato de arquivo AZW1 ##

Os dispositivos de leitura de livros Amazon Kindle e o software usam arquivos no formato AZW1 com extensão .azw1. Esses arquivos também são conhecidos como arquivos Amazon Topaz eBook, esses arquivos AZW1 são usados para montar e gerenciar partes baixadas de arquivos Amazon Kindle eBook da rede **3G Whispernet** da Amazon. O software Amazon Kindle pode ser instalado em um sistema baseado no **Microsoft Windows** para executar o suporte para esses arquivos AZW1 e os recursos relevantes integrados ao aplicativo. As versões deste programa para ambientes Mac também estão disponíveis. Esse software geralmente é usado para iniciar o download e a visualização do conteúdo desses livros do Amazon Kindle. Como o aplicativo Amazon Kindle oferece suporte para arquivos AZW1 e também pode ser instalado nesses dois sistemas conhecidos, os criadores podem facilmente criar e compartilhar livros Kindle de maneira adequada.
