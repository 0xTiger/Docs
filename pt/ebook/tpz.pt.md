## O que é um arquivo TPZ? ##

A extensão de arquivo .tpz também é conhecida como Kindle Topaz eBook Format pela Amazon. Linux, Mac OS e Windows são algumas das plataformas nas quais os arquivos com extensão .tpz podem ser distribuídos. Como arquivo de eBook Kindle Topaz, o arquivo TPZ é formatado para se parecer com ebooks tradicionais e contém padrões de compilação de dados proprietários da Amazon para o formato TPZ. Esse arquivo possui suporte avançado a fontes incorporadas e detalhes adicionais, como anotações e marcadores, também podem ser incluídos nos arquivos do Kindle Topaz eBook, embora esses detalhes de metadados sejam armazenados separadamente. O arquivo TPZ também pode ser categorizado como um arquivo de dados que pode ser baixado do banco de dados online da Amazon de e-books Kindle.
