## O que é um arquivo ASM?

Um arquivo ASM é um arquivo de dados usado pelo [Solid Edge](https://solidedge.siemens.com/en/) software de modelagem, renderização e montagem CAD 3D. Ele contém objetos 3D que são compostos de uma montagem de peças modeladas. Um único arquivo ASM pode conter várias partes para constituir o arquivo de dados inteiro. Os arquivos ASM são totalmente diferentes dos arquivos [ASM de linguagem de montagem](/pt/programming/asm/) e não têm conexão com eles. O software Siemens Solid Edge é usado para abrir esses arquivos ASM.
