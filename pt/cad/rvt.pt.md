## O que é um arquivo RVT?

Um arquivo com extensão .rvt é o formato proprietário da Autodesk para arquivos do Revit. O Revit é um programa de modelagem de informações de construção (BIM) da Autodesk que é usado para criar projetos de construção 3D para plantas baixas, elevações e seções. Oferece ciclo de vida completo do projeto, desde o projeto conceitual até a fabricação e construção. Todos os elementos do projeto de construção são atualizados à medida que o modelo se desenvolve. Os arquivos do Revit podem ser abertos usando o software Autodesk Revit ou Autocad Architecture.

## Formato de arquivo RVT

Os arquivos RVT são baseados no formato de arquivo proprietário da Autodesk e suas informações de formato não estão disponíveis publicamente. No entanto, você pode extrair seu banco de dados como OLE Compound File que pode ser aberto usando aplicativos como 7Zip.
