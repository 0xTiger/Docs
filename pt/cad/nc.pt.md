## O que é um arquivo NC?

Um arquivo NC é um arquivo de Controle Numérico Computadorizado (CNC) criado com o [software Mastercam CNC](https://www.mastercam.com/). É composto por instruções de controle numérico para direcionar o movimento mecânico de ferramentas de usinagem CNC, como furadeiras, tornos, fresadoras e impressoras 3D. As máquinas CNC usam instruções dos arquivos NC na forma de um programa sequencial para execução. Estas instruções do arquivo NC destinam-se a orientar a máquina para a produção do componente.

## Formato de arquivo NC

Os arquivos NC são armazenados em disco como arquivos binários e suas especificações de formato de arquivo interno não estão disponíveis especificamente. O software Mastercam CNC permite que os usuários criem percursos de máquina 2D ou 3D. Esses percursos, quando salvos em disco, geram o arquivo .nc que é finalmente utilizado pela máquina-ferramenta CNC para criar o componente desejado.
