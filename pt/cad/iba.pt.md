## O que é um arquivo .IBA?

A extensão do arquivo IBA é criada pela Lectra. Este formato de arquivo também é conhecido como arquivo Lectra Clothing Design Pieces e se enquadra na categoria de arquivos CAD. No entanto, o formato de arquivo principal usado pela Lectra Modaris é o formato de arquivo MDL, que é um formato proprietário desenvolvido pela Lectra. Os arquivos MDL contêm informações sobre os padrões, marcadores e modelos 2D e 3D que são criados e editados no software Modaris.

## Relação com Lectra Modaris

IBA é um arquivo cad relacionado ao software Lectra Modaris. Lectra Modaris é um software de modelagem 2D e 3D desenvolvido pela Lectra, empresa de tecnologia especializada em software, hardware e serviços para a indústria da moda. A Lectra Modaris foi projetada para ajudar designers de moda, modelistas e fabricantes a criar e desenvolver padrões de forma mais eficiente e precisa, com recursos como classificação de padrões, criação de marcadores e prototipagem virtual. ele usa principalmente arquivos CAD (Computer-Aided Design) nos seguintes formatos:

1. Formato de arquivo proprietário da Lectra, que possui uma extensão .mdl. Este formato é utilizado para guardar padrões, marcadores e modelos criados no software Modaris.
2. Formato de arquivo DXF (Drawing Interchange Format), amplamente utilizado na indústria CAD para troca de dados entre diferentes aplicativos de software. O formato DXF é útil para exportar peças de padrões 2D do Modaris.
3. Formato de arquivo AAMA (American Apparel Manufacturers Association), que é um formato padrão para arquivos de padrões usados na indústria da moda. O Modaris pode exportar padrões no formato AAMA para utilização noutras aplicações de software.

É importante notar que o Modaris também suporta vários outros formatos de arquivo, incluindo Gerber (ZIP), Lectra (MOD), Lectra (CIT) e Lectra (LAY). No entanto, esses formatos de arquivo são menos usados na indústria da moda.
