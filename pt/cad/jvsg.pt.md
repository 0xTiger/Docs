## O que é um arquivo JVSG?

JVSG é um arquivo CAD que pode ser iniciado no Windows. É um arquivo em formato binário criado por JVSG. Os arquivos do projeto de vigilância por vídeo usam principalmente o sufixo de nome de arquivo JVSG. Este arquivo pode armazenar a largura de banda, localizações, ângulos de visão e cálculos de disco rígido de uma câmera com informações sobre seu projeto de CFTV. Os arquivos de projeto JVSG oferecem análise e documentação avançadas ao renderizar nas visualizações 3D.

As redes de câmeras de segurança de diferentes empresas que precisam de câmeras de vigilância também podem ser projetadas com os arquivos do projeto JVSG. O software “ferramenta de design CCTV” é adequado para gerenciar arquivos JVSG.
