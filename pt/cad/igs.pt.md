## O que é um arquivo IGS?

Um arquivo com extensão .igs (Initial Graphics Exchange) é um formato de arquivo de troca de design 2D-3D que é independente das especificações de formato de arquivo de origem ou destino usadas por aplicativos CAD. É usado para trocar informações de projeto sobre diagramas de circuito, wireframe, superfície de forma livre entre dois sistemas independentes. É semelhante ao [IGES](/pt/cad/iges/) e é o principal requisito dos fabricantes para definir preços e projetar moldes para seu produto. Mais recentemente, o IGS foi substituído pelo formato de arquivo STEP(.STP) mais recente. Os arquivos IGS podem ser abertos por aplicativos como Autodesk, FreeCAD, CADEX CAD Exchanger e outros aplicativos semelhantes.

## Formato de arquivo IGS

Os arquivos IGS estão no formato de arquivo de texto ASCII e podem ser abertos com qualquer editor de texto. Cada registro em um arquivo IGS tem 80 caracteres. Os aplicativos CAD salvam e exportam dados vetoriais no formato de arquivo IGS para transferir arquivos de modelo 3D entre diferentes pacotes de software. Embora os arquivos IGS possam ser abertos em qualquer editor de texto, o modelo ou design só pode ser visualizado por aplicativos de visualização que ofereçam suporte ao formato de arquivo IGS.
