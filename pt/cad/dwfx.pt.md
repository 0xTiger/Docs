## O que é um arquivo DWFX?

Um DWFx (Design Web Format XPS) é uma representação formatada de desenho 2D/3D como documento XPS. Ele contém gráficos e texto como parte dos dados de projeto. É a versão mais recente do formato de arquivo DWF e pode ser visualizada e impressa com o Microsoft XPS Viewer. A natureza XPS desses arquivos permite compartilhar os dados de projeto com revisores sem exigir que eles instalem o Autodesk Design Review.
Semelhante ao DWF, o DWFx é desenvolvido pela Autodesk em formato compactado para tornar a transmissão pela Internet adequada. Um único arquivo DWFx pode conter um ou vários desenhos e conjuntos de folhas.
