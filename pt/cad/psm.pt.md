## O que é um arquivo PSM?

Um arquivo com extensão .psm é um arquivo CAD que é criado e aberto com o software de modelagem CAD Solid Edge para projetar peças e montagens 3D. Ele contém o modelo 3D do projeto para a fabricação de peças a partir da chapa metálica. O pacote de software Solid Edge oferece todas as opções necessárias para criar um modelo 3D que também pode ser integrado a outros projetos mecânicos. Seu aninhamento 2D suporta os formatos de arquivo PSM e PAR, além de outros formatos de dados neutros, incluindo DXF e DWG, extraindo dados de padrão plano.

### Formato de arquivo PSM

O PSM é um formato de arquivo fechado e seus detalhes não estão disponíveis no momento. Estes têm application/octet-stream do tipo MIME. Arquivos de modelo de metal podem ser importados no Solid Edge e convertidos em peças utilizáveis para torná-los editáveis. A modelagem síncrona oferece maior usabilidade para trabalhar com arquivos PSM.
