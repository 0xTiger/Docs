## O que é um arquivo ASC?

Os arquivos ASC são arquivos de texto da Autodesk que armazenam informações sobre objetos em seu desenho, incluindo sua localização, tamanho e cor. Isso os torna uma escolha popular para trocar desenhos entre diferentes programas [CAD](/pt/cad/). Os arquivos ASC podem ser abertos com os produtos de software da Autodesk. A Autodesk oferece a opção de importar arquivos ASC de dentro do aplicativo. Os arquivos ASC podem ser abertos em qualquer editor de texto, como Microsoft Notepad, Notepad++ e Apple TextEdit.

## Formato de arquivo ASC

Os arquivos ASC são armazenados em disco no formato de arquivo de texto ASCII simples. Estes são legíveis por humanos.

### Exemplo de arquivo ASC

```
ncols 200
nrows 200
xllcorner 119000.00
yllcorner 158000.00
cellsize 5.00
nodata_value -999999
```
