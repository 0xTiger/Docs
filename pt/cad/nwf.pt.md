## O que é um arquivo NWF?

Um arquivo com extensão .nwf é um formato de arquivo CAD proprietário da Autodesk que contém índices de todos os arquivos de modelo usados pelo projeto. Ele contém ponteiros para os elementos adicionados aos arquivos de modelo para referência. Os arquivos NWF também contêm informações como pontos de vista e redlining. A vantagem de usar o formato de arquivo NWF é o reflexo imediato das alterações nos arquivos CAD originais [por exemplo, DWG](/pt/cad/dwg/) devido aos ponteiros associados. O uso de um NWF é altamente recomendado para testes de interferência. O arquivo NWF salva os resultados de interferência originais e, após a modificação dos arquivos CAD, os arquivos CAD revisados serão abertos ao reabrir o arquivo NWF.

## Formato de arquivo NWF

Os detalhes do formato de arquivo interno do NWF não estão disponíveis, pois é um formato de arquivo proprietário da Autodesk.
