## O que é um arquivo .M4R?

O formato de arquivo .m4r é um tipo de formato de arquivo de áudio usado para toques de iPhone. O formato é essencialmente uma versão renomeada do formato contêiner multimídia MPEG-4 Parte 14 (MP4), com a extensão de arquivo .m4r indicando que o arquivo é um toque. Os arquivos M4R são codificados no formato Advanced Audio Coding (AAC) e podem ser criados usando vários aplicativos de software que suportam a criação de toques. Depois de criados, eles podem ser transferidos para um iPhone usando o iTunes ou outros métodos e, em seguida, definidos como toque do dispositivo. Embora os arquivos M4R sejam projetados para toques de iPhone, o formato não é exclusivo de dispositivos Apple e pode ser reproduzido em outros dispositivos que suportam arquivos de áudio AAC.

## Formato de arquivo M4R – Mais informações

O formato .m4r foi introduzido pela Apple em 2007 com o lançamento do iPhone. Antes disso, os toques do iPhone estavam no formato .m4a, que também é um formato de áudio AAC, mas não foi projetado especificamente para toques. Os arquivos M4R têm duração máxima de 40 segundos, que é a duração máxima permitida para um toque em um iPhone. Arquivos M4R são o formato usado pelo iPhone para toques.

Os arquivos M4R podem conter conteúdo de áudio e vídeo, embora o conteúdo de vídeo seja normalmente ignorado pelo iPhone quando o arquivo é usado como toque. Os arquivos M4R são criados renomeando um arquivo .m4a existente para .m4r. O conteúdo do arquivo não muda, mas a extensão .m4r informa ao iTunes que o arquivo deve ser tratado como toque.

Os arquivos M4R podem ser criados usando vários aplicativos de software, incluindo o iTunes, que inclui um criador de toques integrado. Aplicativos de software de terceiros, como GarageBand e Ringtone Maker, também podem ser usados para criar arquivos M4R. Os arquivos M4R podem ser transferidos para um iPhone usando o iTunes ou sincronizando o arquivo com o iPhone usando serviços de armazenamento em nuvem, como iCloud ou Dropbox.
