## O que é um arquivo .STY?

O arquivo STY normalmente se refere a um formato de arquivo usado pelos teclados e sintetizadores eletrônicos Yamaha. Os arquivos ".sty" da Yamaha contêm dados de estilo que definem o ritmo, padrões de acompanhamento e outros elementos musicais usados pelo instrumento, são usados para aprimorar a reprodução de arquivos MIDI, fornecendo estilos e arranjos pré-programados.

Para usar um arquivo .sty com um teclado ou sintetizador Yamaha, normalmente você transfere o arquivo para o instrumento usando uma conexão USB ou um cartão de memória. Uma vez carregado, o instrumento pode então aplicar os dados de estilo para acompanhar a reprodução MIDI ou tocar ao vivo com o estilo selecionado.

## Arquivos Yamaha vs Korg STY

Yamaha e Korg salvam arquivos STY em formatos diferentes, o que significa que não são multiplataforma. Yamaha e Korg são dois fabricantes proeminentes de teclados eletrônicos e sintetizadores e usam seus próprios formatos de arquivo proprietários para arquivos de estilo. A Yamaha fornece software dedicado para edição e criação de estilos, como Style Fixer ou Yamaha Expansion Manager. A Korg fornece ferramentas de software como GNIStyle ou Style Works para editar e criar estilos para teclados Korg. Esses programas permitem aos usuários modificar estilos existentes ou criar novos de acordo com suas preferências.

## Como abrir o arquivo STY?

Você pode usar os seguintes softwares para abrir arquivos STY.

- Fixador de estilo (estilos Yamaha)
- GNIStyle (estilos Korg)
- WAVE Xtrator
