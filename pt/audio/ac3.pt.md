## O que é um arquivo AC3?

Um arquivo com extensão .ac3 é um arquivo Audio Codec 3, introduzido pela Dolby Laboratories. É um formato de áudio que pode conter até seis canais de saída de áudio. O formato foi originalmente usado para áudio, mas agora também é usado para outras aplicações, como transmissão HDTV, DVDs, discos Blue-ray e consoles de jogos. Os aplicativos que podem abrir arquivos AC3 incluem o Apple QuickTime player, o Microsoft Windows Media Player, o Winamp, o MPlayer e outros similares.

## Formato de arquivo AC3

Os arquivos AC3 são de natureza binária e baseados no Modified Discrete Cosine Transform (MDCT), que é um algoritmo de compactação com perdas. A Dolby Laboratories usou o algoritmo MDCT juntamente com princípios de codificação perceptiva para desenvolver o formato de áudio AC-3. Isso levou ao lançamento do formato AC-3 como padrão Dolby Digital em 1991. O formato suporta o fornecimento de cinco canais para alto-falantes de alcance normal (20 Hz - 20.000 Hz) e um canal (20 Hz - 120 Hz) para os efeitos de baixa frequência acionados pelo subwoofer. AC3 suporta taxas de amostragem de até 48 kHz.

### Detalhes técnicos do formato de arquivo AC3

A tecnologia Dolby Digital usa uma maneira muito eficiente de compactar o tamanho dos arquivos de áudio multicanal sem prejudicar a qualidade do som. Essa compactação leva a um tamanho de arquivo menor, o que facilita a distribuição dos arquivos de som. Usando Dolby Digital, é possível incluir uma mixagem completa de áudio de 5.1 canais em uma impressão de filme ou DVD.
