## O que é um arquivo ACT? ##

Um formato ACT ou Adobe Color Table (também conhecido como arquivo Color Lookup Table) é um tipo de banco de dados Adobe Photoshop que armazena um conjunto de cores predefinidas. Você pode adicionar ou remover cores ao salvar uma imagem para publicação online para favorecer uma imagem de maior qualidade ou um tamanho de arquivo menor. Você pode ter um formato de áudio criptografado ADPCM se não for utilizado com o Illustrator. Modulação de código de pulso diferencial adaptativo é uma técnica de compressão de áudio utilizada por vários dispositivos de música e dispositivos de escuta.

O formato de arquivo ACT também é usado pelo sistema Alma CAD/CAM File. Estas são as diretrizes usadas pelas máquinas-ferramentas 3D para descobrir como cortar qualquer coisa. Um Genesis3D Player, uma ação de desenvolvedor de jogos DS ou um FoxPro Detailing Wizard Action Schematic são todos exemplos de ACT.

## Especificação de formato de arquivo ACT

Muitos gravadores que usam o sistema ACT estão fazendo isso apenas para a menor opção de gravação; se a qualidade for aumentada, um formato separado, geralmente WAV, é usado, resultando em documentos muito maiores.

Existem várias gerações de ACT; a partir de junho de 2009, as faixas criadas por dispositivos subsequentes simplesmente não podem ser reconhecidas por todos os players de música padrão gratuitos ou aplicativos de conversão, deixando apenas as ferramentas de MP3 incluídas.

A otimização de imagem para distribuição na Web é feita com arquivos ACT. Os clientes podem testar a qualidade da imagem em relação aos bancos de dados de imagens gerados, aumentando ou diminuindo o número de cores na CLUT.

Ao tocar na opção Tabela de cores e selecionar Carregar dados de cores na caixa de diálogo Exportar para a Web, as tabelas de cores da Adobe podem ser carregadas. O diretório */Presets/en_US/Save* for Web Settings/Color Tables/ contém vários arquivos ACT pré-configurados que acompanham o sistema Photoshop.

Os scripts de atos podem ser gerados exclusivamente no Genesis 3D ou transformando outros modelos 3D para o padrão ACT usando o Genesis3D Actor Designer
