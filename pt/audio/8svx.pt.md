## O que é um arquivo 8SVX? ##

Um 8SVX é o formato de arquivo padrão de áudio desenvolvido pela Electronic Arts. É considerado como um subtipo de dados do formato de arquivo IFF. O arquivo é usado para armazenar dados de áudio de 8 bits em blocos. Em pedaços de descritores separados, os metadados estão contidos no fluxo de dados 8SVX. É um formato de arquivo de áudio digital e é usado por PCs Amiga. Os sistemas de computador do Amiga agora são raramente usados, então esses arquivos 8SVX geralmente são usados para preservar e ouvir a música antiga.

## Formato de arquivo 8SVX

O arquivo 8SVX pode armazenar gravações de áudio com qualidade de 8 bits derivada da especificação [IFF](/pt/audio/aiff/). Esses são arquivos que contêm gravações de áudio como outros arquivos de áudio. Também incluiu o formato de contêiner do IFF. Nesse formato, vários formatos de dados foram incorporados em um arquivo e usados para detalhar ainda mais os fluxos incluídos nele. Eles podem ser usados sozinhos nos contêineres de arquivos somente de áudio dos arquivos IFF. Por outro lado, a multiplexação com outros subtipos de IFF pode resultar em fluxos de vídeo animados.

## Breve História do Formato 8SVX

Foi lançado inicialmente em 1985 por Steve Hayes e Jerry Morrison. Outro formato de arquivo foi desenvolvido em 2009. Este formato de arquivo foi usado em sistemas de computador Amiga para música antiga. Também é conhecido como o arquivo de som Amiga de 8 bits. O software de áudio feito para PCs Amiga suportava este formato de arquivo. Foi usado por muitos programas de edição de som inicialmente. A maioria de seus fluxos de dados são codificados usando fluxos PCM. Uma versão modificada do Amiga forneceu uma estrutura de decodificador 8SVX. Os formatos RIFF e WAVE co-desenvolvidos pela Microsoft e IBM são feitos sob a influência do formato container 8SVX/IFF.

## Especificações de formato de arquivo 8SVX

Muitos programas estão associados a esses arquivos no Windows e Mac OS. Muitos programas podem ser usados para lidar com esses arquivos. É um formato de áudio codificado para o Commodore Amiga. Os recursos como ataque, repetição de seção e lançamento são suportados por esses arquivos. Esses recursos são usados para armazenar amostras de instrumentos musicais. A compactação de dados com perdas de Fibonacci-delta está disponível, o que resulta na taxa de compactação de 50% e diminui a fidelidade.
