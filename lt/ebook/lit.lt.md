## Kas yra LIT failas?

LIT failai priklauso senoviniam **Microsoft Reader**, kuris buvo naudojamas elektroninėms knygoms skaityti. Microsoft Reader rodomos el. knygos turi plėtinį **.lit**, kuris sutrumpintas iki **Literatūra**. LIT failo formatą sudaro Microsoft ITOL / ITLS HTML žinyno 2.0 failas, kuriame pateikiamos įvairios nuostabios funkcijos, pvz., [LZX](/compression/lzx/) glaudinimas, žymėjimo medžiagos dvejetainis atvaizdavimas ir su kiekvienu srautu susiję pasirinktiniai papildomi duomenys. Knygos tekstas yra savavališkas OEBPS 1.0 žymėjimo srautų skaičius ir OEBPS 1.0 CSS poaibis. Vaizdai yra [GIF](/image/gif/), [JPEG](/image/jpeg/) ir [PNG](/image/png/).

## Kiek laiko „Microsoft Reader“ rodė LIT failus

The **Microsoft Reader** is an application for reading e-books that have been discontinued by Microsoft. The first version of Microsoft Reader was released in August 2000, which used its own LIT format. It was available for PDAs and Windows computers. The name of the app was also used later for an application (which was initially released with Windows 8) for reading [PDF](/pdf/) and [XPS](/page-description-language/xps/) files. The Microsoft Reader application was discontinued in 2018.
