## Kas yra VSD failas?

VSD failai yra brėžiniai, sukurti naudojant Microsoft Visio programą, vaizduojantys įvairius grafinius objektus ir jų tarpusavio ryšį. Tokiuose brėžiniuose gali būti vaizdinių objektų, tokių kaip vaizdiniai objektai, srautų diagramos, UML diagramos, informacijos srautas, organizacinės diagramos, programinės įrangos diagramos, tinklo išdėstymas, duomenų bazių modeliai, objektų žemėlapiai ir kita panaši informacija. Microsoft Visio siūlo galimybę konvertuoti Visio failus į įvairius failų formatus, įskaitant [PNG](/image/png/), [BMP](/image/bmp/), [PDF](/pdf/) ir kitus.

# VSD failo formatas #

VSD file format is the initial file format introduced by Microsoft Visio for storing drawing elements. The internal structure of VSD files is based on binary file format unlike [.VSDX](/visio/vsdx/) that was introduced with Microsoft Visio 2013. Naujasis formatas yra pagrįstas Office Open XML failo formatu ir saugo projekto informaciją [ZIP](/compression/zip/) archyve. Microsoft siūlo vviewer.exe programą Visio piešimo failams, jei Visio nėra įdiegta sistemoje.

## Trumpa istorija ##

Visio was initially produced as a standalone product by Shapeware Corporation in 1992 by its version 1.0. Shapeware Corporation 1995 m. pakeitė pavadinimą į Visio Corporation, kad pasinaudotų rinkos pripažinimu. 2000 m. Microsoft pervadino Vision kaip Microsoft Office programą. Visio for Enterprise Architects buvo pristatyta kartu su Visual Studio .NET 2003 ir 2005 m. Visio 2007 buvo išleista 2006 m. lapkričio 30 d. Juosteles vartotojo sąsajoje pristatė Microsoft Visio 2010 m.
