## Kas yra MPP failas?

Failas su plėtiniu .mpp yra Microsoft Project duomenų failas, kuriame integruotai saugoma su projekto valdymu susijusi informacija. Microsoft sukūrė šį patentuotą failo formatą, kad jis būtų suderinamas su Microsoft Project (MSP), jų projektų valdymo programinės įrangos produktu. Išskyrus MPP, MSP gali dirbti su projekto XML schema. Daugelis programų ir API leidžia MPP konvertuoti į kitus failų formatus. Microsoft dabar turi internetinį projektų serverį, kuriame projektų valdymo failus gali įkelti, kad bendradarbiautų keli vartotojai.

## MPP failo formatas ##

Microsoft sukūrė ir paskelbė MPP kaip dvejetainį failo formatą, o jo specifikacijų Microsoft iki šiol nepaskelbė. Taigi kūrėjams nepasiekiama vidinė failo struktūra ir išsami informacija. Internete yra keletas API, kurios palaiko MPP failų skaitymą / rašymą, tačiau 100% tikslumas niekada nepasiekiamas. Laikui bėgant Microsoft savo projektų valdymo programoje įdiegė keletą funkcijų, taip pat ir MPP failo formato funkcijų.

MPP failas naudoja šiuos MIME tipus:

* programa/vnd.ms-projektas

* programa/msproj

* programa / msproject

* programa / x-msproject

* programa / x-ms-projektas

* Application/x-dos_ms_project

* programa/mpp

* zz-application/zz-winassoc-mpp


## Kyla problemų atidarant MPP failą? ##

Čia pateikiamas kai kurių įprastų problemų, kurios gali kilti ir sukelti netinkamą MPP formato veikimą, sąrašas:

 *   Pagalbinės programinės įrangos nebuvimas
 *   Sugadintas failas
 *   Užkrėstas failas dėl viruso
 *   Sistemoje nėra prieigos teisės atidaryti failus
 *   Pasenęs diskas jūsų sistemoje
 *   Failo plėtinys pervardytas
