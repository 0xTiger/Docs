## Kas yra HEIF failas?

An HEIF file is a High-Efficiency Container Image file format that is capable of storing a single image or a sequence of images in a single file. It, thus, is basically a container having multiple images stored. HEIF file format compresses the images using the High Efficiency Video Coding (HEVC) standard. It is is more efficient and light weight as compared other image file formats such as [JPEG](/image/jpeg/). A single HEIF file can store sequences of images such as [GIF](/image/gif/) animations or HDR sequence, along with metadata that describes each image. Apple Inc. adopted its variant HEIC file format for their iOS 11. Tai paskatino kitas įmones palaikyti ją savo programinėje įrangoje, įskaitant Adobe.

## HEIF failo formatas

The HEIF format was developed by Moving Picture Experts Group (MPEG) and was registered under the ISO/IEC 23008-12 standard in 2015. Jį priėmė daugelis gamintojų, tokių kaip Apple, Microsoft ir Canon. HEIF failai saugomi su plėtiniu .heif (jei jame yra vienas vaizdas) arba kaip .heifs (jei jame yra vaizdų seka).

### HEIF duomenų saugykla

HEIF formatas gali saugoti šių tipų vaizdus ir duomenis.

 * Atskiri vaizdai ir vaizdų miniatiūros
 * Vaizdų sekos
 * Tolesnio apdorojimo instrukcijos
 * Vaizdo metaduomenys, pvz., [EXIF](/image/exif/) duomenys

HEIF vaizdo formatas suteikia galimybę naudoti kelis skirtingus vaizdo formatus, įskaitant [JPEG](/image/jpeg/), [PNG](/image/png/) ir [GIF](/image/gif/), nepažeidžiant vaizdo kokybės.
