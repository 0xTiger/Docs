## Kas yra MDS failas?

MDS (Media Descriptor Sidecar File) yra failas, susietas su programine įranga Alcohol 120% ir Daemon Tools, kurios abi naudojamos virtualiems CD ir DVD diskams kurti ir valdyti. MDS faile yra informacijos apie duomenų išdėstymą diske, įskaitant visų failų vietą ir disko struktūrą. Tai leidžia virtualiam diskui imituoti fizinio disko veikimą, kad programinė įranga galėtų nuskaityti duomenis virtualiame diske taip, lyg tai būtų tikras diskas.

When you want to mount an image file to a virtual drive, the MDS file is used along with the corresponding image file in a format such as ISO, BIN or CUE. The MDS file contains a description of the layout of the data on the disc, and the virtual drive software uses this information to create a virtual disc. This allows you to use the software as if you were using a physical disc, without having to physically insert the disc into the drive.

## Ryšys su alkoholiu 120 proc.

Alcohol 120% yra populiari programinė įranga, skirta kurti ir valdyti virtualius CD ir DVD įrenginius. Ji naudoja MDS failo formatą, kad išsaugotų ir pasiektų disko vaizdus. MDS failo formatas priklauso Alcohol 120% ir gali būti atidarytas ir naudojamas tik Alcohol 120% programine įranga arba kita programine įranga, kuri ją palaiko.

## Kaip atidaryti MDS failą?

Norint atidaryti MDS failą Alcohol 120%, kompiuteryje turi būti įdiegta programinė įranga. Įdiegę Alcohol 120% galite atidaryti MDS failą atlikdami šiuos veiksmus:

1. Paleisti Alkoholis 120%.
2. Spustelėkite meniu Failas ir pasirinkite Atidaryti.
3. Kompiuteryje eikite į MDS failo vietą ir pasirinkite jį.
4. Dabar MDS failas bus atidarytas 120% alkoholio ir būsite paraginti pasirinkti atitinkamą vaizdo failą (pvz., ISO, BIN arba CUE).
5. Pasirinkus vaizdo failą, Alcohol 120% prijungs vaizdą prie virtualaus disko.

Arba galite atidaryti MDS failą dukart spustelėdami jį, jei Alcohol 120% nustatyta kaip numatytoji programa MDS failams atidaryti. Taip pat galite naudoti kitą programinę įrangą, kuri palaiko MDS formatą, pvz., Daemon Tools, Virtual CloneDrive, Virtual Drive Manager ir MagicISO.
