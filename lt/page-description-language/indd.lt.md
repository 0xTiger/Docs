## Kas yra INDD failas?

Failas su plėtiniu .indd yra Adobe InDesign projekto failas, kuriame yra projekto puslapio išdėstymo informacija. INDD reiškia InDesign Document. Šiuose failuose dažniausiai saugoma puslapių formatavimo informacija, knygos, žurnalai, laikraščiai, skrajutės, brošiūros ar kita spausdinta laikmena, pvz., mobiliųjų telefonų ir tinklalapių maketai. INDD failo tipas yra patentuotas Adobe failo formatas ir numatytasis InDesign failo formatas. Grafikos dizaineriai, žurnalistai, rinkodaros specialistai ir leidėjai, dirbantys su InDesign, naudoja šį failo tipą.

## Daugiau apie INDD failo formatą

INDD failai yra vektoriniai failai, kurių mastelį galima keisti neprarandant skiriamosios gebos. Jie yra pagrįsti sudėtingais grafikais, kuriuos galima be galo išplėsti. Jame yra visi šriftai ir išdėstymo informacija, kuria galima dalytis su komanda.

INDD failai puikiai tinka kuriant patrauklų spaudinį ir internetinį turinį. INDD failas gali apimti kelis puslapius ir gali būti naudojamas kaip puslapių, skyrių ir leidinių kūrimo šablonai. Kai jūsų projektas bus baigtas, jis gali būti paskelbtas internetinėje žiniasklaidoje nereikalaujant konvertavimo, suspaudimo ar siuntimo el. paštu.

## Ar INDD failą galima konvertuoti į kitus formatus?

INDD files can be converted to several other file formats. The most common conversion is INDD to PDF. InDesign lets you save the INDD file to PDF for sharing with other users. It also lets you export INDD file to JPG, [EPS](/page-description-language/eps/), [EPUB](/ebook/epub/), [SWF](/page-description-language/swf/), FLA, [HTML](/web/html/), [XML](/web/xml/), and IDML.

## Kaip sukuriamas INDD failas?

An Indd file is created when you create a new project with [Adobe InDesign](https://www.adobe.com/products/indesign.html) and save it to disc. You can open the Indd project file, make changes to it, and save the updated file for the changes.
