## Kas yra SVETAINĖS failas?

A WEBSITE file is a website shortcut file created with Microsoft Internet Explorer (IE). It contains a URL and an icon (ICO) file that points to the website's favicon. The icon is used for displaying the shortcut. WEBSITE file format was introduced with the release of IE version 9. Ankstesnėse IE versijose tinklalapių spartieji klavišai buvo išsaugoti kaip [.url](/web/url/) failai.

## SVETAINĖS failo formatas

Svetainės failas saugomas diske kaip paprasto teksto failai, kurių turinį galima peržiūrėti bet kuriame teksto rengyklėje. Be kito turinio, jame yra tikslinis URL, kurio nuoroda yra nuoroda. Jei norite atidaryti tikslinę svetainę kitoje nei IE naršyklėje, galite nukopijuoti šį URL ir įklijuoti į kitas naršykles, pvz., Google Chrome, Safari ar Firefox, kad peržiūrėtumėte svetainę.

## Kaip sukurti .WEBSITE failą?

SVETAINĖS failą galima sukurti naudojant IE, spustelėjus URL adreso juostoje ir nuvilkus jį į norimą vietą kompiuteryje, pvz., darbalaukyje ar dokumentuose. Tai sukuria nuorodą į tą interneto vietą ir gali būti atidaryta dukart spustelėjus nuorodą. Jei norite išsaugoti spartųjį klavišą kaip URL failą, laikykite nuspaudę klavišą SHIFT prieš vilkdami tikslinės svetainės piktogramą į darbalaukį.
