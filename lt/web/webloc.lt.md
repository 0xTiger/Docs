## Kas yra WEBLOC?

WEBLOC failas yra sparčiųjų klavišų failas MacOS, nurodantis žiniatinklio URL internete. Jis sugeneruojamas nuvilkant žiniatinklio URL iš naršyklės adreso juostos į darbalaukį ar bet kurį kitą aplanką, kuriame jis turi būti išsaugotas. Paskirties svetainės URL yra saugomas šiame sparčiųjų klavišų faile ir jį galima peržiūrėti atidarius failą teksto rengyklėje, pvz., Microsoft Notepad, Notepad++ arba Apple TextEdit. WEBLOC failai yra panašūs į [URL](/web/url/) failus.

## WEBLOC failo formatas – daugiau informacijos

WEBLOC failai saugomi paprasto teksto failo formatu. Juos galima sugeneruoti MacOS nuvilkant URL iš žiniatinklio naršyklių, pvz., Safari, Google Chrome ar Firefox. Jei vienu metu reikia atidaryti kelis WEBLOC failus, reikės greitesnio ir lengvesnio sprendimo juos atidaryti naudojant išorines programas arba API. Šios API gali skaityti ir konvertuoti WEBLOC į PDF ir JPEG.
