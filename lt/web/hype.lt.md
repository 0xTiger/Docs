## Kas yra HYPE failas?

A HYPE file is a project file saved using the Hype software that is used for creation of web content with HTML5. Jame yra žiniatinklio elementų, tokių kaip grafika, tekstas, animacija, scenos informacija ir kiti elementai, kurie kartu sudaro turinį. HYPE failo žiniatinklio turinį galima eksportuoti į žiniatinklio failus, kuriuos galima paskelbti tinklalapyje.

HYPE failą galima atidaryti naudojant [Tumult Hype](https://tumult.com/hype/) programinę įrangą.

## HYPE failo formatas

HYPE failai saugomi patentuotu failų formatu, o vidinė jų informacija nėra viešai žinoma. HYPE failo turinį galima peržiūrėti tik atidarius programinę įrangą Tumult Hype.

### HYPE failo turinys

HYPE failas sukuriamas, kai naudojate Hype programą, kad sukurtumėte žiniatinklio turinį įvairiems tikslams. Jame saugoma informacija apie pagrindinio kadro animaciją, skirtą turiniui, pvz., el. kortelėms, el. knygoms, pristatymams, reklamai, infografikai ir tinklalapių medžiagai, kurti. Jų turinį palaiko HTML5, leidžiantis turinį rodyti įrenginiuose nuo stalinių kompiuterių iki iPhone ir iPad. Tumult Hype pateikia naujausius HTML5, [CSS3](/web/css/) stilius ir [JavaScript](/web/js/).
