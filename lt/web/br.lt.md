## Kas yra BR failas?

A BR file is a compressed web file generated by applying the open source data compression algorithm, **Brotli**. It is used to store webpage assets such as stylesheets ([CSS](/web/css/)), images ([SVG](/page-description-language/svg/)), [XML](/web/xml/), and scripting files ([JS](/web/js/)). Modern day websites, such as Chrome and Firefox, use BR files to reduce the page loading time, resulting in better user experience.

## BR failo formatas

BR failai yra suspausti žiniatinklio failai, generuojami naudojant Brotli glaudinimo algoritmą. Brotli glaudinimas yra neprarandantis duomenų glaudinimo algoritmas, kurį Google sukūrė Zopfli glaudinimo algoritmui. Jis pagrįstas LZ77 be nuostolių suspaudimo algoritmo ir Huffman kodavimo deriniu.

Dėl mažo dydžio BR failus naudoja žiniatinklio serveriai ir turinio pristatymo tinklai (CDN). Šiems serveriams pateikiamos užklausos suglaudina HTTP turinį, todėl svetainės įkeliamos greičiau. Brotli tapo populiaresnis ir užtikrina geresnį suspaudimą nei gzip.
