## Kas yra XHTM failas?

An file with .xhtm extension is a webpage file that is written in XHTML. It uses XML 1.0 as its standard implementation and is considered a reformulation of HTML 4. Kaip ir XHTML, XHTM faile yra standartinės HTML žymos, kurios atitinka griežtesnes XML gaires. XHTM failai naudojami svetainės turiniui skelbti ir dažniausiai naudojamas [XHTML](/web/xhtml/) plėtinys.

XHTM failams atidaryti galite naudoti žiniatinklio naršykles, tokias kaip Google Chrome, Apple Safari ir Mozilla Firefox.

## XHTM pavyzdys

Šis XHTM dokumento pavyzdys apibrėžia XML deklaracijas.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
