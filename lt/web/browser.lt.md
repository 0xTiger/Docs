## Kas yra BROWSER failas?

Failą su plėtiniu .browser naudoja ASP.NET žiniatinklio programos, kad nustatytų žiniatinklio naršyklių galimybes ir konfigūraciją. Jame pateikiama informacija apie naršyklės galimybes, apribojimus ir charakteristikas. Šios charakteristikos padeda ASP.NET sistemai atpažinti užklausą pateikusią naršyklę ir atitinkamai pateikti tinklalapį vartotojams.

## BROWSER Failo formatas

BROWSER apibrėžimo failus naudoja ASP.NET programa, kad nustatytų naršyklės galimybes. Tai šiame faile esanti informacija, kurią serveris naudoja žymėjimui ir kitiems ištekliams, suderinamiems su užklausą teikiančia naršykle, generuoti.

BROWSER failai išsaugomi paprasto teksto failo formatu. BROWSER failą galite atidaryti naudodami teksto rengykles, tokias kaip Notepad, Notepad++, Apple TextEdit ir Visual Studio IDE.

### BROWSER failo formato failo struktūra

BROWSER apibrėžimo failai naudoja hierarchinę struktūrą. Kiekviename .browser faile yra XML žymėjimas, apimantis konkrečios naršyklės ar naršyklių grupės galimybes ir funkcijas. Ši informacija apima tokias charakteristikas kaip agento eilutė, versijų numeriai ir technologijų, pvz., JavaScript, CSS ar kitų HTML elementų, palaikymas. Naudodami šiuos naršyklės apibrėžimo failus, ASP.NET kūrėjai pritaiko vartotojo patirtį pagal skirtingų naršyklių galimybes.
