## Kas yra CHM failas?

CHM failo formatas yra Microsoft **[HTML](/web/html/)** žinyno failas, kurį sudaro HTML puslapių rinkinys. Jame pateikiama rodyklė, leidžianti greitai pasiekti temas ir naršyti įvairias pagalbos dokumento dalis. CHM failo turinio galima ieškoti naudojant pateiktą paieškos parinktį. CHM yra Microsoft patentuotas internetinio žinyno failo formatas, dažnai naudojamas programinės įrangos dokumentacijai. Be to, jis naudojamas keliose kitose programose, pvz., mokymo vadovuose, interaktyviose knygose ir elektroniniuose informaciniuose biuleteniuose. Dauguma šiuolaikinių Microsoft Development aplinkų palaiko CHM dokumentacijos generavimą iš programoje esančios informacijos.

Unikali CHM failo formato galimybė įdiegti kombinuotą turinį ir rodyklę išskiria jį iš kitų standartinių HTML puslapių. Sukurtas CHM failas yra palyginti mažo dydžio, todėl gali būti lengvai platinamas naudojant programinės įrangos paketus. Pagalbos kūrimo įrankis, HTML žinyno seminaras, suteikia lengvai naudojamą sistemą pagalbos projektams ir su jais susijusiems failams kurti ir tvarkyti. CHM failuose gali būti teksto, vaizdų ir hipersaitų; galima peržiūrėti interneto naršyklėje; Windows ir kitos programos naudoja kaip internetinį pagalbos sprendimą.

## CHM failo formatas

Galutinė sugeneruoto CHM failo forma priklauso nuo to, kaip sukurta pagalbos sistema ir ar ji skirta programai, ar svetainei. CHM failas palaiko duomenų glaudinimą naudojant LZX glaudinimą, kad būtų generuojami suspausti HTML failai. Jame yra integruotas paieškos variklis, skirtas greitai ieškoti turinio, taip pat galimybė sujungti kelis .CHM failus. CHM failą sudaro HTML failų rinkinys, susietas turinys ir rodyklės failas.

### HTML failai

Nesvarbu, ar kuriate žinyno temas, skirtas platinti su programa, ar žiniatinklyje, jūsų sukurti dokumentai kuriami naudojant specialią formatavimo kalbą, vadinamą hiperteksto žymėjimo kalba (HTML). HTML temų failai turi .htm arba .html failo pavadinimo plėtinį.

Nors kiekviena jūsų sukurta žinyno tema arba tinklalapis atrodo kaip dokumentas su tekstu, grafika ar animuotais vaizdais, .htm failai iš tikrųjų yra tekstiniai dokumentai, turintys specialius HTML formatavimo kodus. Šie kodai, vadinami žymomis, nurodo naršyklei, kaip rodyti kiekvieną puslapį. Tik temoje arba tinklalapyje rodomas tekstas iš tikrųjų yra .htm faile. Bet kokia rodoma grafika, garsai, animuoti vaizdai ar kiti elementai yra atskiri failai, į kuriuos nukreipia jūsų HTML failas. Naršyklė nukopijuoja arba atsisiunčia grafiką, garsus ar kitus elementus, kai pamato tai daryti nurodančius žymes.

### Turinys (TOC)
Pagalbos turinio (.hhc) failas yra HTML failas, kuriame yra jūsų turinio temų pavadinimai. Kai vartotojas atidaro turinį sudarytame žinyno faile (arba tinklalapyje) ir spusteli temos pavadinimą, bus atidarytas su tuo pavadinimu susietas HTML failas.

### Indekso failas
Indekso (.hhk) failas yra HTML failas, kuriame yra indekso įrašai (raktiniai žodžiai). Kai vartotojas atidaro indeksą sudarytame žinyno faile arba tinklalapyje ir spusteli raktinį žodį, atsidarys su raktiniu žodžiu susietas HTML failas.

## HTML žinyno komponentai

HTML žinyną sudaro keli komponentai. Tai apima:

* `HTML Help Workshop`: a help authoring tool with an easy-to-use graphical interface for creating project files, HTML topic files, contents files, index files, and everything else you need to put together an online help system or Web site.
* „HTML žinyno ActiveX valdiklis“: maža, modulinė programa, naudojama pagalbos naršymui ir antrinėms langų funkcijoms įterpti į HTML failą.

* „HTML žinyno peržiūros priemonė“: visiškai veikiantis ir tinkinamas trijų langelių langas, kuriame gali būti rodomos internetinės pagalbos temos.

* `Microsoft HTML Help Image Editor`: internetinis grafikos įrankis, skirtas kurti ekrano kopijas; ir vaizdo failams konvertuoti, redaguoti ir peržiūrėti.

* `HTML žinyno Java programėlė`: maža, Java pagrindu sukurta programa, kurią galima naudoti vietoj ActiveX valdiklio, norint įterpti pagalbos naršymą į HTML failą.

* „HTML žinyno vykdomoji programa“: programa, kuri rodo ir paleidžia žinyną, kai spustelite sukompiliuotą žinyno failą.

* „HTML žinyno kompiliatorius“: programa, kuri sukompiliuoja projektą, turinį, rodyklę, temą ir kitus failus į sukompiliuotą žinyno failą.

* `The HTML Help Authoring Guide`: an online guide designed to assist help authors in using HTML Help to design a help system. The guide also contains a complete HTML Help reference for developers and an HTML tag reference for authors.
