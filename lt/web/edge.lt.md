## Kas yra EDGE failas?

Failas su plėtiniu .edge yra Adobe Edge Project failas, kuris sukuriamas, kai projektas sukuriamas ir išsaugomas naudojant Adobe Edge. Adobe Edge leidžia žiniatinklio dizaineriams sukurti interaktyvius [HTML](/web/html/) animatorius žiniatinkliui, skaitmeninei publikacijai ir raiškiosios medijos reklamai tiek stalinių kompiuterių, tiek mobiliųjų įrenginių naršyklėms. Tai pasiekiama naudojant HTML/HTML5, [JavaScript](/web/js/) ir [CSS3](/web/css/). Edge failai generuojami kaip paprasto teksto failai ir gali būti atidaryti naudojant kelias skirtingas programas, tokias kaip Microsoft Notepad, Apple TextEdit ir Notepad++.

## EDGE failo formatas

EDGE failai išsaugomi paprasto teksto formatu ir naudoja jQuery animacijai įjungti. EDGE animacijos alternatyva buvo Flash failai, kurie nebepalaikomi. EDGE failas išsaugomas kartu su HTML, JavaScript ir CSS failais. Atidarius EDGE failą, atidaromas tinklalapis, ty HTML failas.
