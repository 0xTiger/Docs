## Kas yra ATOM failas?

ATOM failas yra sindikacijos formatas, skirtas Atom tiekimo ir įvesties dokumentų struktūrai. Panašiai kaip .RSS failai, ATOM failo formatas yra XML pagrindu sukurtas sindikavimo formatas, kuris yra turinio publikavimo standartas. ATOM failas naudojamas žiniatinklio informacijos santraukoms, kurios leidžia programinės įrangos programoms patikrinti, ar nėra svetainėje paskelbtų naujinimų. Jame yra įrašų, kurie gali būti įvairių tipų, pavyzdžiui, antraštės, viso teksto straipsniai, ištraukos arba nuorodos į svetainės turinį

Programos, kurios gali **atidaryti ATOM failus**, apima **Apple Safari**.

## ATOM failo formatas – daugiau informacijos

ATOM failai saugomi ir skelbiami populiariu XML formatu, kuris yra universalus informacijos mainų formatas. Jis buvo sukurtas kaip RSS alternatyva, atsižvelgiant į RSS failo formato apribojimus ir trūkumus.

### Atominių dokumentų tipai

 1. Atom Entry Documents – tai XML dokumentas, kurį sudaro vienas informacijos elementas, žinomas kaip įrašas, skirtas Atom sklaidos kanalui. Jame yra atomo įvedimo elementas, kuriame yra daug antrinių elementų. Atom įrašo turinys gali būti paprasto teksto, HTML, XHTML arba kito IANA (Internet Assigned Numbers Authority) laikmenos tipo.
 1. Atom Feed Documents – tai XML dokumentas, kuriame pateikiami metaduomenys apie Atom sklaidos kanalą ir vienas ar daugiau informacijos santraukos įrašų. Kai klientas pateikia informacijos iš sklaidos kanalo užklausą, serveris sugeneruoja informacijos santraukos dokumentą, kuriame yra daug Atom įrašų, kad būtų įvykdyta užklausa.
 1. Atom kolekcija – tai specialus Atom tiekimo dokumentas, kuriame yra Atom įrašų, kuriuos galima redaguoti, URL.
