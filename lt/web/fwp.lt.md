## Kas yra FWP failas?

Failas su .fwp failu yra archyvo failas, sukurtas naudojant Microsoft Expression Web, kuriame yra visi dizaino failai ir aplankai. Microsoft Expression Web yra Microsoft bendras interneto dizaino programinės įrangos produktas, pagrįstas [HTML](/web/html/) redaktoriumi. Jis buvo nutrauktas nuo 2012 m. ir Microsoft jį galima atsisiųsti nemokamai. FWP failus galima eksportuoti ir išsaugoti kaip Microsoft .cab formatą.

## FWP failo formatas – daugiau informacijos

Tikslios FWP failų formato specifikacijos kūrėjui nepasiekiamos. Tačiau tai yra dvejetainiai failai ir juos galima atidaryti naudojant Microsoft Expression Studio, kuri jau seniai buvo nutraukta.
