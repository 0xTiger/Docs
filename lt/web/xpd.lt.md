## Kas yra XPD failas?

XPD failas yra tinklalapio failas, sukurtas iš XML dokumento. Dažniau tai generuojama naudojant XSLT transformaciją. [XPD file format specifications](https://www.w3.org/TR/xml-pipeline/) yra prieinami W3, kurie apibrėžia būdą, kaip padidinti kelių procesų [XML](/web/xml/) programų sąveiką. XML vamzdyno kalba apibrėžia apdorojimo ryšius tarp XML išteklių.
