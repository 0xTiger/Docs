## Kas yra CC failas?

CC failas yra šaltinio kodo failo tipas, kuriame yra kodas, parašytas C++ programavimo kalba. Jame gali būti visas kodas, reikalingas vienai programai, arba dalis kodo, kuris yra didesnio programavimo projekto, kurį sudaro keli CC failai, dalis. Norėdami pakeisti arba peržiūrėti kodą, esantį CC faile, kūrėjai dažnai naudoja šaltinio kodo rengyklę. Po redagavimo jie sukompiliuoja kodą naudodami kompiliatorių, kad konvertuotų jį į mašininį kodą, kurį kompiuterio operacinė sistema gali vykdyti programai paleisti.

## CC failo formatas – daugiau informacijos

.cc failo plėtinys buvo naudojamas C++ šaltinio kodo failams keletą dešimtmečių. C++ kalbą aštuntojo dešimtmečio pabaigoje sukūrė Bjarne'as Stroustrup, tuo metu dirbęs Bell Labs. Stroustrup norėjo išplėsti C programavimo kalbą, palaikydamas objektinio programavimo koncepcijas, kurios, jo nuomone, buvo būtinos kuriant dideles programinės įrangos sistemas.

Pirmoji C++ versija buvo išleista 1983 m. ir greitai išpopuliarėjo tarp kūrėjų, kurie įvertino jos galingas funkcijas ir lankstumą. Kai C++ tapo plačiau naudojamas, iškilo būtinybė nustatyti kodavimo ir failų pavadinimų konvencijas, kurios padėtų organizuoti ir valdyti dideles kodų bazes.

Google's C++ Style Guide which recommends using .cc file extension for C++ source code files, was first published in 2003. Nuo tada vadovas buvo periodiškai atnaujinamas, kad atspindėtų C++ kalbos pokyčius ir platesnės programavimo bendruomenės kūrimo praktiką.

Šiandien .cc failo plėtinys išlieka populiarus C++ šaltinio kodo failų pasirinkimas ir jį atpažįsta daugelis populiarių kūrimo įrankių ir platformų.

## Kaip atidaryti CC failą?

Daugelis šaltinio kodo redaktorių, įskaitant kelių platformų įrankius, tokius kaip Microsoft Visual Studio Code ir Code Blocks, gali būti naudojami CC failams atidaryti ir redaguoti. Šie redaktoriai siūlo naudingų funkcijų, tokių kaip sintaksės paryškinimas ir automatinis užbaigimas, todėl jie yra puikus pasirinkimas dirbant su CC failais.

Taip pat galite atidaryti CC failus naudodami bet kurį teksto rengyklę, pvz., Notepad, Notepad++ ir kt.

## CC kompiliavimas su C++ kompiliatoriais

CC failo kompiliavimas naudojant C++ kompiliatorių apima kelis veiksmus. Pirmas žingsnis yra įdiegti suderinamą C++ kompiliatorių, skirtą operacinei sistemai ir kūrimo aplinkai. Yra daug įvairių kompiliatorių, įskaitant GCC, Clang ir Microsoft Visual C++.

Įdiegę C++ kompiliatorių, kitas veiksmas yra atidaryti terminalą arba komandų eilutę ir pereiti į katalogą, kuriame yra CC failas. Iš čia galite paleisti kompiliatorių vykdydami kompiliatoriaus pateiktą komandinės eilutės įrankį. Konkreti CC failo kompiliavimo komanda gali skirtis priklausomai nuo naudojamo kompiliatoriaus. Pavyzdžiui, norėdami sudaryti CC failą naudodami GCC, paleiskite komandą, panašią į:

```
g++ -o myprogram myfile.cc
```
