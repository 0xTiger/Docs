## Kas yra PHP failas?
Failas su plėtiniu .php reiškia atvirojo kodo programavimo kalbą, naudojamą serverio scenarijų rašymui, kurie turi būti vykdomi žiniatinklio serveryje. Tai plačiausiai naudojama žiniatinklio scenarijų kalba, kuri paprastai naudojama kuriant didelio masto žiniatinklio programas. Didžiausia tinklaraščių sistema internete, ty WordPress, ir didžiausias socialinis tinklas Facebook yra pagrįsti PHP kalba. Jis suderinamas su įvairiomis platformomis, tokiomis kaip Windows, Linux, Unix, Mac OS X ir kt. Taip pat sėkmingai veikia beveik visuose šiandien naudojamuose serveriuose, ty Apache, IIS ir kt. PHP palaiko daugybę duomenų bazių, įskaitant MySQL.

## PHP failo formatas ##

Viename PHP gali būti tekstas, CSS, [HTML](/web/html/), JavaScript ir PHP kodas. PHP kodas vykdomas serveryje, o rezultatas grąžinamas į naršyklę kaip paprastas HTML. PHP failus galima atidaryti naudojant bet kurį teksto rengyklę ir redaguoti vietoje, nors tokios programos kaip Adobe Dreamweaver, Eclipse PHP Development Tools siūlo patogų būdą rašyti ir keisti PHP kodą.

### Sintaksė ###

Serveryje vykdomas PHP scenarijus, o paprastas HTML rezultatas siunčiamas atgal į naršyklę. PHP scenarijus prasideda<?php and ends with ?>

### PHP pavyzdys ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
