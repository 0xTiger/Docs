## Kas yra NUPKG failas?

NUPKG failas yra paketo failas, kuriame yra šaltinio kodas, kurį naudos NuGet programinė įranga, kad sukurtų paketus, naudojamus .NET projektuose. NuGet Package Manager komponentas yra įdiegtas kaip Microsoft Visual Studio dalis, skirta paketams gauti iš internetinių paketų saugyklos. NUPKG failai padeda kūrėjams gauti naujausius paketus iš [Nuget.org](https://nuget.org) naudojant NuGet Package Manager, o ne rankiniu būdu atsisiunčiant ir diegiant kūrimo paketus. NUPKG failai yra sukurti iš NUSPEC failų ir, kai jie gaunami, įdiekite paketą vartotojo sistemoje.

## NUPKG failo formatas

NUPKG failai yra [ZIP](/compression/zip/) archyvai, kuriuose yra supakuotos bibliotekos. Atsisiuntus jį galima pervadinti į .zip ir išskleisti naudojant bet kokias standartines ZIP programas, tokias kaip WinZIP, 7-Zip ir Apple Archive Utility.
