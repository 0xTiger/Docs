## Kas yra NX failas?

NX failas yra vaizdo žaidimų failas, sukurtas naudoti su virtualia žaidimų konsole **LowRes NX**. Jis parašytas BASIC programavimo kalba ir jame yra žaidimo šaltinio kodas, nurodantis, kaip žaidimas veikia. Galite sukurti savo NX failus, skirtus naudoti su programa, kuriuos galima įkelti ir vartotojai gali žaisti žaidimą.

NX failą galite atidaryti naudodami teksto rengyklę, LowRes NX arba bet kurią šaltinio kodo rengyklę, pvz., Microsoft Visual Studio Code.

## NX failo formatas

NX failas išsaugomas diske kaip paprasto teksto failas, kuriame yra žaidimo instrukcijos. Kiekviename NX faile yra žaidimo šaltinio kodas, kurį sudaro keli kasetės ROM įrašai. Šiuose ROM įrašuose yra žaidimo komponentų informacija, pvz., grafika, lygių žemėlapiai, garsas ir kiti žaidimo ne šaltinio kodo duomenys. Programos ROM įrašus galima pridėti arba redaguoti naudojant NX redagavimo programas, kurios yra kartu su LowRes NX.

## Kaip sukurti NX failus?

### Sukurkite NX failus kompiuteryje

NX failus galite kurti kompiuteryje naudodami:

 * Teksto redaktoriai – tokie kaip Microsoft Notepad, Notepad++ arba Github Atom
 * Šaltinio kodo redaktoriai, pvz., Microsoft VIsual Studio kodas

### Kurkite NX failus iOS

LowRes NX turi iOS versiją, kurią kūrėjai gali naudoti kurdami NX failus iš šios programos.
