## Kas yra GROOVY failas?

GROOVY failas yra šaltinio kodo failas, parašytas Groovy programavimo kalba. GROOVY faile įrašytas kodas yra panašus į objektinę kalbą [Java](/programming/java/), todėl lengva kurti ir kurti programas. GROOVY failas yra sukompiliuotas į Java virtualios mašinos (JVM) baitinį kodą ir yra suderinamas su kitu Java kodu ir bibliotekomis.

## GROOVY failo formatas – daugiau informacijos

GROOVY failuose yra šaltinio kodas, parašytas Groovy sintaksė. Jame gali būti kodas, kuris gali būti naudojamas ir kaip programavimo kalba, ir kaip Java platformos skriptų kalba. Groovy kalba turi panašias funkcijas kaip Python, Ruby ir Smalltalk.
