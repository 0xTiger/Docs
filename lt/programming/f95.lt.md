## Kas yra F95 failas?

Failas su plėtiniu .f95 paprastai nurodo, kad tai yra Fortran 95 šaltinio kodo failas. Fortran yra bendrosios paskirties, būtinoji programavimo kalba, kuri ypač gerai tinka skaitiniams ir moksliniams skaičiavimams. Fortran 95 yra patobulinta Fortran 90 versija, pristatanti keletą naujų funkcijų ir patobulinimų.

Fortran šaltinio kodo failuose paprastai yra programos instrukcijos, parašytos Fortran programavimo kalba; jei turite failą su plėtiniu .f95, tikėtina, kad jis turi būti sudarytas naudojant Fortran kompiliatorių; Kompiliavimas – tai žmogaus skaitomo šaltinio kodo pavertimo mašininiu būdu skaitomu dvejetainiu kodu, kurį gali vykdyti kompiuteris, procesas.

## Kaip atidaryti F95 failą?

Jei norite dirbti ar paleisti programą .f95 faile, jums reikės Fortran kompiliatoriaus; populiarūs Fortran kompiliatoriai yra GNU Fortran (gfortran), Intel Fortran Compiler (ifort) ir kt. Galite kompiliuoti ir paleisti Fortran programas naudodami atitinkamas kompiliatoriaus pateiktas komandas.

Pavyzdžiui, jei įdiegėte gfortran, terminale galite naudoti šias komandas, kad sukompiliuotumėte ir paleistumėte Fortran programą:

```
# Compile the Fortran program
gfortran -o my_program my_program.f95

# Run the compiled program
./my_program
```

Jei norite peržiūrėti tik F95 failo turinį, tai yra paprasto teksto failas, todėl galite jį atidaryti naudodami bet kurį teksto rengyklę.
