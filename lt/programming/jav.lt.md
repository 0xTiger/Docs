## Kas yra „Jav“ failas?

Jav failas yra Java šaltinio kodo failas, sukurtas Java programavimo kalba. Jis panašus į Java šaltinio failą ir gali būti sudarytas naudojant Java interpretatorius. Kaip ir [Java](/programming/java/) failai, Java failas naudojamas kuriant Java programas. Tai gali būti vienas iš daugelio šaltinio kodo failų, pasiekiamų Java projekte.

## Jav failo formatas

Jav failai saugomi kaip tekstiniai failai ir gali būti atidaryti naudojant bet kurį teksto rengyklę. Jav failų tipai dažniausiai buvo pakeisti Java failo tipu ir dažniausiai naudojami Java programoms rašyti. Jav kodus galima rašyti naudojant IDE, pvz., Eclipse ir IntelliJ Idea, kurios palengvina darbą su Java failais.

### Java failo formato pavyzdys

```java
/*
The example code prints
Hello World from Java to the console.
*/
    public class JavaApp {
        public static void main(String[] args) {
            System.out.println("Hello World."); // Prints the string to the console.
        }
    }
```
