## Kas yra POT failas?

Failai su .POT plėtiniu yra Microsoft PowerPoint šablonų failai, sukurti PowerPoint 97-2003 versijomis. Failai, sukurti naudojant šias Microsoft PowerPoint versijas, yra dvejetainio formato, palyginti su failais, sukurtais Office OpenXML failų formatais naudojant aukštesnes PowerPoint versijas. Taigi sugeneruoti failai gali būti naudojami kuriant pristatymus, kurių išdėstymas yra toks pat, ir kiti parametrai, kuriuos reikia pritaikyti naujiems failams. Šie nustatymai gali apimti stilius, fonus, spalvų paletę, šriftus ir numatytuosius nustatymus. Tokie failai generuojami siekiant sukurti paruoštus naudoti šablonų failus oficialiam naudojimui.

## Failo formato specifikacijos ##

File format specifications for POT file format are not available individually by Microsoft. However, the template format is the same as that of the binary [PPT](/presentation/ppt/) file format and can be opened in MS PowerPoint for editing purpose. A POT file is identified by HEX identifier as follow:

```
D0 CF 11 E0 A1 B1 1A E1 00 00 00 00
```

Su POT failo formatu susiję MIME tipai yra šie:

* Application/vnd.ms-powerpoint [oficialus]

* programa / mspowerpoint

* Application/x-mspowerpoint

* programa / powerpoint

* programa / x-powerpoint

* Application/x-dos_ms_powerpnt

* aplikacija/puodas

* Application/x-soffic
