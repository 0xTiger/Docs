## Kas yra AST failas?

AST failas yra skaičiuoklės šablono failas, sukurtas naudojant Ability Spreadsheet programinę įrangą, Ability Office rinkinio skaičiuoklės komponentą. Šie šablonų failai naudojami kuriant pradinį skaičiuoklės šablonų rinkinį, kurį galima naudoti kuriant [AWS](/spreadsheet/aws/) failus. AST failus galima atidaryti naudojant Ability skaičiuoklės programinę įrangą.

## AST failo formatas – daugiau informacijos

AST files are saved to disc as binary files and can be opened in Ability Spreadsheet software. Though fully compatible with Microsoft XLS and XLSX file formats, AST file format specifications are not available publicly for developer's reference. AST files can be saved to disc in [PDF](/pdf/) as well as [XPS](/page-description-language/xps/) file formats.
