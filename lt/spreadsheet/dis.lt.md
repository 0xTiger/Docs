## Kas yra DIS failas?

A DIS file is a workbook file created with [Oracle Discoverer](https://docs.oracle.com/cd/E28389_01/bi.1111/b40107/overview.htm) software that comprises of several tools for querying, reporting, and data analysis for the Oracle Database. It contains multiple worksheets when data is exported from Oracle Discoverer. Oracle markets Discoverer as a Business Intelligence (BI) product used for for reporting and analysis that works in part of the [Oracle Fusion Middleware](https://docs.oracle.com/cd/E28280_01/bi.1111/b40108/toc.htm) suite. Oracle Discoverer doesn't save the reporting data in DIS file format by default.

## Kaip atsisiųsti „Discoverer“ darbaknyges DIS formatu?

Oracle Discoverer darbaknygės saugomos duomenų bazėje dvejetainiu formatu ir negali būti tiesiogiai atsisiunčiamos kaip [DIS files](https://forums.oracle.com/ords/apexds/post/can-i-download-all-discoverer-workbooks-to-my-computer-4127). Tačiau duomenis galima eksportuoti kaip EEX failus, atidaryti juos Discoverer Desktop ir naudoti parinktį Išsaugoti kaip, kad išsaugotumėte EEX failus kaip DIS failus.

## DIS failo formatas – daugiau informacijos

DIS failai išsaugomi diske kaip dvejetainiai failai, tačiau jų vidinio failo formato informacija kūrėjui nepasiekiama.
