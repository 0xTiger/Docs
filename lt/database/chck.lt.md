## Kas yra CHCK failas?

CHCK failas sukuriamas tikrinant Microsoft Exchange Server būklę. Jame pateikiama informacija apie bendrą serverio duomenų bazės būseną ir veikimą. Microsoft Exchange Server pirmiausia naudojamas kaip el. pašto serveris ir leidžia tvarkyti verslo kontaktus ir kalendorius. Exchange serverio administratoriai naudoja EseUtil, kad patikrintų duomenų bazės būklę.

## CHCK failo formatas

Vidinis CHCK failo formatas nežinomas ir nėra jokios informacinės medžiagos CHCK failams skaityti / rašyti.
