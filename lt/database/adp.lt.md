## Kas yra ADP failas?

A file with .adp extension is a Microsoft Access project file that uses OLE DB component architecture to provide a direct and efficient connection to a Microsoft SQL Server database. Such files do not contain the actual tables, database diagrams or other database elements. ADP files can be created with Access 2007 and 2010. Failus, sukurtus naudojant ankstesnes Access versijas, taip pat galima atidaryti Access 2007 ir 2010. ADP failus galima atidaryti naudojant Microsoft Access 365.

## ADP failo formatas – daugiau informacijos

Access projektas, sukurtas kaip ADP, leidžia atlikti SQL serverio objektų, pvz., lentelių ir rodinių, dizaino pakeitimus. Tai taip pat leidžia kurti, redaguoti ir naudoti SQL funkcijas, tokias kaip duomenų bazės diagramos, saugomos procedūros ir vartotojo apibrėžtos funkcijos. Tai suteikia pranašumo, palyginti su susiejimu su SQL serverio duomenų baze, kai negalite keisti dizaino ir galite susieti tik su SQL serverio lentelėmis ir rodiniais.

### Duomenų tipų ir diagramų palaikymas

**Datos / laiko duomenų tipai**

Programoje Access 2010 palaikomi šie nauji datos ir laiko duomenų tipai.

 * LAIKAS
 * DATA
 * DATATIME2
 * DATATIMEOFFSET

**Kintamo ilgio duomenų tipai**

Access 2010 projektuose galima naudoti šiuos kintamo ilgio duomenų tipus:

 * VARBIN (MAX)
 * VARCHAR (MAX)
 * NVARCHAR (MAX)
