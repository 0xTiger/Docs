## Kas yra FP7 failas?

A file with .fp7 extension is a database file created with FileMaker Pro versions 7-11. Jis saugo duomenis lentelėse su įrašais, kur kiekviename įraše yra vienas ar daugiau laukų. 7BP failai gali saugoti didelius duomenis, įskaitant asmeninių duomenų bazių duomenis ir organizacinius įrašus. Naujesnė FileMaker Pro versija yra [FMP12](/database/fmp12/), kuri buvo pristatyta paleidus FileMake Pro 12 versiją. Be tekstinių duomenų, FP7 faile taip pat gali būti formų, diagramų ir kitų duomenų bazės objektų. FP7 failus galima atidaryti naudojant FileMaker Pro Advanced.

## FP7 failo formatas

FP7 failai išsaugomi diske kaip dvejetainiai failai FileMaker patentuotu failo formatu. FP7 failo formato vidinio failo formato informacija nėra viešai prieinama.
