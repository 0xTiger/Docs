## Kas yra DB3 failas?
DB3 failas yra duomenų bazės failas, sukurtas SQLite programine įranga, kuri yra lengva, savarankiška duomenų bazių programa, kurianti duomenų bazes naudojant paprastus failus; yra duomenų bazės anatomija ir duomenų įrašai; dažniausiai naudojamas struktūriniams duomenims gauti arba saugoti naudojant SQL. Šiuos failus galima naudoti išmaniuosiuose įrenginiuose arba ten, kur reikalingas įrašų tvarkymas ar kitas duomenų tvarkymas, tačiau mažai vietos aplinkoje.


## DB3 failo formatas
DB3 failo formatas yra susietas su RDBMS (Relational Database Management System) SQLite, populiariu įterptųjų duomenų bazių pasirinkimu. Specifikacijose SQLite_3 nėra apibrėžto jokio konkretaus failo plėtinio, jis gali naudoti plėtinius, įskaitant [.dbf](/database/dbf/) ir [.sqlite](/database/sqlite/).

### Duomenų bazės skirstymas
Commonly SQLite, version 3 related db file format can be considered as the DB3 file format used as the publicly documented native format for the SQLite database engine since June 2004. DB3 failo formatas yra kelių platformų, kurią galima perkelti tarp didelės ir mažos architektūros arba 32 bitų ir 64 bitų sistemų. Dėl šių savybių DB3 yra populiarus kaip programos failo formato pasirinkimas. Pagrindinis SQLite_3 duomenų bazės (DB3) failas susideda iš vieno ar daugiau puslapių. Visų puslapių dydžiai yra vienodi toje pačioje duomenų bazėje. Puslapio dydis baitais yra dviejų laipsnis nuo 512 iki 65536 imtinai.
