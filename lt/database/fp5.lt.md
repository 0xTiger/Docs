## Kas yra FP5 failas?

An FP5 file is a database file created with FileMaker Pro version 5 or 6. Jis saugo vartotojų ir organizacijos duomenis įrašų pavidalu lentelėse. FileMaker Pro yra duomenų bazių valdymo sistema, palaikanti Macintosh ir Windows operacines sistemas. FP5 yra senesnis failo formatas, kurį palaikė tuo metu turima FileMaker Pro versija. Naujausias FileMaker Pro duomenų bazės formatas yra [FMP12](/database/fmp12/), kuris buvo pristatytas paleidus FileMaker Pro v 12. Kitas gerai naudojamas duomenų bazės failo formatas buvo [FP7](/database/fp7/), kuris buvo pakeistas FMP12.

## FP5 failo formatas – daugiau informacijos

FP5 failo formato vidinio failo formato informacija nėra viešai prieinama kūrėjui. Senesni FileMaker Pro duomenų bazės failų formatai, pvz., [FP5 and FP7 can be upgraded to latest FMP12](https://fmhelp.filemaker.com/help/16/fmp/en/index.html#page/FMP_Help/converting-files.html), naudojant išsamias FileMaker Pro gaires.
