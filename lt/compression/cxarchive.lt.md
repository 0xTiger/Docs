## Kas yra CXARCHIVE failas?

**CrossOver** kontekste CXARCHIVE failai yra susiję su programos funkcijomis, skirtomis Windows programoms paleisti Mac ir Linux. .CXARCHIVE failuose yra konkrečių butelių, kurios yra virtualios Windows aplinkos, kuriose galima įdiegti ir paleisti Windows programas, duomenys ir nustatymai.

## CXARCHIVE failo pagrindiniai taškai

Štai keletas pagrindinių dalykų apie .CXARCHIVE failus, sukurtus CrossOver:

1.  **Paskirtis**: .CXARCHIVE failai naudojami duomenims, konfigūracijos nustatymams ir įdiegtai Windows programinei įrangai saugoti konkrečiame butelyje. Tai leidžia CrossOver vartotojams atskirai valdyti ir tvarkyti skirtingas Windows programas.
    
2.  **Netrinkite**: Svarbu neištrinti .CXARCHIVE failų, susijusių su CrossOver, nebent ketinate pašalinti atitinkamą buteliuką ir jo turinį. Ištrynus šiuos failus galite prarasti įdiegtas Windows programas ir jų duomenis.
    
3.  **Atsarginė kopija**: gera praktika reguliariai kurti atsargines .CXARCHIVE failų kopijas, ypač jei per CrossOver įdiegėte svarbias Windows programas. Tokiu būdu galite juos atkurti atsitiktinio ištrynimo ar sugadinimo atveju.
    
4.  **Atkūrimas**: jei reikia atkurti buteliuką iš .CXARCHIVE failo, paprastai tai galite padaryti per CrossOver sąsają. 

## CodeWeavers CrossOver

CrossOver yra komercinė programinė įranga, sukurta CodeWeavers. Jis sukurtas tam, kad Mac ir Linux operacinių sistemų naudotojai galėtų paleisti Windows programinę įrangą ir programas savo kompiuteriuose, nereikalaujant Windows operacinės sistemos licencijos. CrossOver yra pagrįsta atvirojo kodo Wine projektu, kuris suteikia suderinamumo sluoksnį Windows programoms paleisti Unix tipo sistemose.

Štai keletas pagrindinių CrossOver funkcijų ir aspektų:

1.  **Suderinamumas**: CrossOver siekia užtikrinti įvairių Windows programinės įrangos, įskaitant populiarias programas ir žaidimus, suderinamumą. Vartotojai gali patikrinti CrossOver svetainėje suderinamumo duomenų bazę, kurioje pateikiama konkrečių programų suderinamumo būsena.
    
2.  **Buteliai**: CrossOver naudoja butelių sąvoką, kad sukurtų izoliuotą aplinką, skirtą Windows programinei įrangai paleisti. Kiekvienas butelis gali turėti savo konfigūracijos nustatymus, bibliotekas ir įdiegtas programas, leidžiančias vartotojams atskirti skirtingas Windows programas.
    
3.  **Nereikalaujama Windows licencijos**: skirtingai nei naudojant virtualią mašiną su Windows, CrossOver nereikalauja, kad vartotojai pirktų Windows licenciją, kuri gali būti ekonomiška tiems, kuriems reikia paleisti kelias Windows programas savo Mac ar Linux sistemose.
    
4.  **Kelių platformų palaikymas**: CrossOver galima naudoti tiek MacOS, tiek įvairiems Linux paskirstymams, todėl tai yra universalus sprendimas skirtingų platformų naudotojams.
    
5.  **Suderinamumas su vynu**: CrossOver naudoja Wine suderinamumo sluoksnį, kurį nuolat kuria Wine projektas ir CrossOver komanda, kad pagerintų Windows programų palaikymą.
