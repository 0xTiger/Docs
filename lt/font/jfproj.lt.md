## Kas yra JFPROJ failas?

A file with .jfproj extension is a JSON based custom font created with Microsoft Font Maker application on Windows 10. Juos galima sukurti naudojant elektroninį rašiklį su jutiklinio ekrano įrenginiais, pvz., planšetiniais kompiuteriais, tačiau kompiuteriuose taip pat gali būti naudojami kiti kompiuterių išoriniai įrenginiai, pvz., pelė. JFPROJ failuose yra glifai, naudojami simboliams pateikti ekrane, nustatant tarpus tarp simbolių ir žodžių bei šrifto dydį. Microsoft Font Maker programą galima atsisiųsti iš Windows parduotuvės.
