## Kas yra INS failas?

Failas su plėtiniu .ins paprastai naudojamas Microsoft Windows, kad nustatytų telefono ryšį ir plačiajuostį interneto ryšį. Tiesą sakant, jame yra ryšio sąrankos informacija, leidžianti Windows nustatyti interneto prieigą prie IPT. INS failas dažniausiai naudojamas nustatant Internet Explorer (IE) LAN ryšio parametrus. IE ryšio nustatymų failai, kuriuos IPT ir sistemos administratoriai kartais pateikia vartotojams su INS failo URL.

## INS failo formatas
Galite naudoti interneto nustatymų failus (INS) kartu su Internet Explorer administravimo rinkiniu 11 (IEAK 11), kad sukonfigūruotumėte pasirinktinę naršyklę ir jos komponentus. Galite rašyti įvairias pasirinktinio paketo versijas tinkindami INS failo kopijas.

### Galimi INS nustatymai
Galimi INS nustatymai pateikti šioje lentelėje:

| Nustatymas | Aprašymas |
-----|---------|
| URL | Ar naudoti automatiškai sukonfigūruotą tarpinį serverį. |
| Prekės ženklas | Tinkinkite prekės ženklo ir sąrankos informaciją naršyklės pakete. |
| Žiniasklaida | Medijos tipai, kuriuose galimas pasirinktinis diegimo paketas. |
| Naršyklės įrankių juostos | Tinkinkite IE įrankių juostos išvaizdą. |
| Kabinos pasirašymas | Skaitmeninio parašo informacija jūsų programoms. |
| HideCustom | Ar slėpti kiekvieno tinkinto komponento visuotinį unikalų identifikatorių (GUID). |
| Ryšio nustatymai | Informacija apie tinklo ryšio nustatymus, naudojamus diegiant pasirinktinį paketą. |
| CustomBranding | URL vieta į jūsų prekės ženklo kabineto (.cab) failą. |
| ExtRegInf | Sąrankos informacijos (.inf) failų pavadinimai ir komponentų diegimo režimas. |
| MėgstamiausiEx | Įtraukite kelią į mėgstamiausių piktogramų failą, nuspręskite, ar pamėgtos vietos pasiekiamos neprisijungus, ir pridėkite URL prie kiekvienos Pamėgtos svetainės. |
| ISP_Security | Šakninis sertifikatas, kurį pridedate prie pasirinkto paketo. |
| Tarpinis serveris | Ar naudoti tarpinį serverį. |
| Saugumas Importas | Ar importuoti tinkinto paketo saugos informaciją. |
