## Co je soubor LIT?

Soubory LIT patří ke starému **Microsoft Readeru**, který se používal ke čtení elektronických knih. Elektronické knihy, které zobrazuje Microsoft Reader, mají příponu **.lit**, která je zkrácena na **Literatura**. Formát souboru LIT se skládá ze souboru Microsoft ITOL/ITLS HTML Help 2.0, který poskytuje všechny druhy úžasných funkcí, jako je komprese [LZX](/cs/compression/lzx/), binární reprezentace značkovacího materiálu a libovolná volitelná pomocná data spojená s každý proud. Text knihy je libovolný počet značkovacích streamů OEBPS 1.0 a podmnožina CSS OEBPS 1.0. Obrázky jsou [GIF](/cs/image/gif/), [JPEG](/cs/image/jpeg/) a [PNG](/cs/image/png/).

## Jak dlouho Microsoft Reader zobrazoval soubory LIT

**Microsoft Reader** je aplikace pro čtení elektronických knih, které společnost Microsoft přestala vyrábět. První verze Microsoft Readeru byla vydána v srpnu 2000, která používala svůj vlastní formát LIT. Byl k dispozici pro PDA a počítače se systémem Windows. Název aplikace byl později také použit pro aplikaci (která byla původně vydána s Windows 8) pro čtení souborů [PDF](/cs/pdf/) a [XPS](/cs/page-description-language/xps/). Aplikace Microsoft Reader byla ukončena v roce 2018.
