## Co je soubor VBK? ##

Soubor s příponou .vbk je e-kniha a nejčastěji patří do knihovny VitalSource Bookshelf. Obvykle se jedná o soubory chráněné DRM, které obsahují výukové materiály, jako jsou učebnice. Je k dispozici pro počítače se systémem Windows a Mac a také pro mobilní zařízení se systémem Android nebo iOS. Soubor VBK obvykle obsahuje digitální učebnici. Pomocí VitalSource Bookshelf máte možnost získat digitální verzi učebnic, kterou vlastníte, jako fyzickou kopii.

Soubory VBK, známé také jako Veeam Backup File, jsou přístupné a čteny pomocí softwaru pro zálohování a obnovu virtuálních strojů (VM) Veeam. Obvykle má úplnou zálohu virtuálního obrazu a je známý jako jeden z největších souborů vytvořených pro jakoukoli zálohu. Veeam Backup poskytuje zálohování i obnovu k ochraně před poškozením dat virtuálních strojů pro virtualizační software VMware a Microsoft, jako je vSphere, Hyper-V atd. Tyto soubory také souvisejí se souborem VisualCADD Backup File a jsou velmi běžné u uživatelů ve Spojených státech, kteří mají Windows 10. většinou v jejich systému.

## Problém s otevřením souboru VBK? ##

Zde jsou některé ze seznamu chyb, kterým můžete čelit při otevírání přípony VBK:

* Soubor může být infikován nebo poškozen
* Pokud byl soubor VBK odeslán z jiných zdrojů. Pravděpodobně může nastat problém při kopírování souboru na datové úložiště. Kvůli nedokončenému zpracování může být obtížné jej otevřít
* Nemáte správná přístupová práva k otevření souboru
 *	Inadequate system resources to open VBK files
