## Co je soubor YBK? ##

Formát souborů eBook YanCEyWare a Microsoft Encarta používají přípony souborů YBK. Elektronická kniha YanCEyWare byla vytvořena společností YanCEyWare. Všechny softwarové aplikace jsou dostupné pro zařízení se systémem Windows, podporují soubory YBK. Encyklopedie Encarta lze aktualizovat pomocí souborů .ybk, aniž by bylo nutné aktualizovat celý program. YanCEyDesktop je známý a oblíbený program, který dokáže snadno spravovat a zpracovávat soubory YBK. YanCEyWare Reader používá tento formát souboru YBK značně jako aplikaci pro čtení elektronických knih pro Windows Mobile nebo Pocket PC. Tento soubor byl vyvinut s velkými hypertextovými odkazy.

## Problémy s otevřením souboru YBK ##

Někdy, když je obtížné otevřít a zprovoznit soubor YBK, zde je seznam některých možných problémů:

* Otevření poškozeného souboru YBK
* Nevhodné a nevhodné odkazy na soubor YBK v přístupech do registru
* Neúmyslné odstranění popisu YBK z archivu Windows
* Částečná instalace aplikace, která poskytuje formát YBK
*	Infected YBK file with an undesirable malware
