## Co je soubor TR3? ##

TR3 file je elektronická kniha TomeRaider 3, která poskytuje podporu pro rychlé vyhledávání a kompresi. To může správně fungovat prostřednictvím programu souvisejícího s TomeRaider. Yadabyte je vývojář tohoto softwaru, britská společnost pro vývoj softwaru a webu se sídlem ve Spojeném království. Pro správné fungování a správu obsahu těchto TR3 souborů lze použít aplikaci TomeRaider eBook reader. Tuto související čtečku elektronických knih lze nainstalovat do počítačů běžících na systémech Microsoft Windows a mnoha přenosných miniaplikacích. TR3 file patří do skupiny eBook Files používaných v operačních systémech jako Windows 10, Windows 7, Windows 8 / 8.1, Windows Vista, Windows XP.

**HTML tagy** specifické pro TR3 jsou následující:

|Značka|Popis|
---|---|
|\<new> |K vytvoření souborů TomeRaider z textových souborů stačí tag New.<new> tag definuje začátek stránek.|
|\<CATDEF> ... \</CATDEF> |definuje název kategorie|
|\<META> ... \</META> |je značka používaná k definování všech metadat používaných ve formátu souboru. Tato značka obsahuje řadu parametrů.|
|\<EXPMSG> |Tato značka řídí zprávu, která se zobrazí po vypršení platnosti souboru. Kdykoli se uživatel pokusí zobrazit stránku po vypršení platnosti souboru, zobrazí se místo skutečného textu stránky zpráva o vypršení platnosti.|
|\<DIEMSG> |Tato značka je podobná EXPMSG. Používá se k zobrazení zprávy, když soubor zemře.|
|\<ENGMSG> |Tato značka se používá k vytvoření zprávy, která se zobrazí pro zašifrované stránky. Zpráva se zobrazí místo textu stránky, pokud se uživatel pokusí otevřít zašifrovanou stránku před jejím odemknutím.|
|\<expmsg> ,\<diemsg> a \<encmsg> |Ignorováno v textu stránky. Musí být umístěny v sekci záhlaví souboru před první<new> tag.|
|\<SELECTION> . \</ SELECTION> |Slouží k podpoře dotazů. Tato značka musí být před první<new> štítek. Sestavuje data výběru pro dotaz uživatele.|
|\<catset> . \</catset> | Používá se k přiřazení názvů kategorií pro každý předmět.|


## Problémy s otevřením souboru TR3 ##

Zde je seznam některých běžných problémů, které mohou nastat a způsobit nesprávné fungování formátu souboru:

* Poškozený soubor
* Infikovaný soubor kvůli viru
* Nesprávné odkazy na soubor TR3 v archivních přístupech
* Žádné přístupové právo v systému k otevření souborů
* Zastaralá jednotka ve vašem systému
* Odstranění zprávy TR3 z archivu Windows
* Kontaktování vývojáře může být lepší volbou pro lepší výstup
