## Co je soubor MINT?

Soubor MINT je soubor instalačního balíčku používaný distribucí [Linux Mint](https://linuxmint.com/). Obsahuje komprimované soubory zdrojového kódu, které jsou extrahovány a použity ke spuštění instalace operačního systému Linux Mint. Linux Mint OS má snadno použitelné GUI (Graphical User Interface) a jako první měl elegantní design GUI. Soubory MINT jsou dekomprimovány a uloženy v dočasném adresáři spolu s instalačním balíčkem. Instalace se provádí ze stejného adresáře pro instalaci OS Linux Mint.

## Formát souborů Linux MINT

Podrobnosti o interním formátu souboru ve formátu MINT nejsou veřejně dostupné. Komprimované instalační soubory uložené v souborech MINT však vyhovují kompresním standardům, které jsou integrovány s instalačními programy operačního systému Linux Mint. Díky tomu je možné tyto soubory dekomprimovat a stát se součástí instalace.
