## Co je soubor CXARCHIVE?

V kontextu **CrossOver** se soubory CXARCHIVE týkají funkcí programu pro spouštění programů Windows na Mac a Linux. Soubory .CXARCHIVE obsahují data a nastavení pro konkrétní "lahve", což jsou virtuální prostředí Windows, kde lze instalovat a spouštět aplikace Windows.

## Klíčové body souboru CXARCHIVE

Zde je několik klíčových bodů o souborech .CXARCHIVE vytvořených CrossOverem:

1. **Účel**: Soubory .CXARCHIVE se používají k ukládání dat, konfiguračních nastavení a nainstalovaného softwaru Windows v konkrétní lahvičce. To umožňuje uživatelům CrossOver spravovat a organizovat různé aplikace Windows samostatně.
    







2. **Do Not Delete**: Je důležité nemazat soubory .CXARCHIVE spojené s CrossOver, pokud nemáte v úmyslu odstranit odpovídající láhev a její obsah. Odstranění těchto souborů může vést ke ztrátě nainstalovaných aplikací Windows a jejich dat.
    







3. **Záloha**: Je dobrým zvykem pravidelně zálohovat soubory .CXARCHIVE, zvláště pokud máte důležité aplikace pro Windows nainstalované přes CrossOver. Tímto způsobem je můžete obnovit v případě náhodného smazání nebo poškození.
    







4. **Obnova**: Pokud potřebujete obnovit láhev ze souboru .CXARCHIVE, můžete to obvykle udělat prostřednictvím rozhraní CrossOver.

## CrossOver CodeWeavers

CrossOver je komerční softwarová aplikace vyvinutá společností CodeWeavers. Je navržen tak, aby umožnil uživatelům operačních systémů Mac a Linux spouštět na svých počítačích software a aplikace Windows bez nutnosti licence na operační systém Windows. CrossOver je založen na open-source projektu Wine, který poskytuje vrstvu kompatibility pro spouštění aplikací Windows na systémech podobných Unixu.

Zde jsou některé klíčové funkce a aspekty CrossOveru:

1. **Kompatibilita**: Cílem CrossOver je poskytnout kompatibilitu pro širokou škálu softwaru Windows včetně oblíbených aplikací a her. Uživatelé mohou na webu CrossOver vyhledat databázi kompatibility, která uvádí stav kompatibility konkrétních programů.
    







2. **Bottles**: CrossOver využívá koncept „lahví“ k vytvoření izolovaného prostředí pro běh softwaru Windows. Každá láhev může mít vlastní konfigurační nastavení, knihovny a nainstalované aplikace, což uživatelům umožňuje udržovat různé programy Windows odděleně.
    







3. **Nevyžaduje se žádná licence Windows**: Na rozdíl od provozování virtuálního stroje s Windows nevyžaduje CrossOver od uživatelů zakoupení licence Windows, což může být nákladově efektivní pro ty, kteří potřebují na svých systémech Mac nebo Linux spouštět několik aplikací pro Windows.
    







4. **Podpora napříč platformami**: CrossOver je k dispozici pro macOS i různé distribuce Linuxu, což z něj činí všestranné řešení pro uživatele na různých platformách.
    







5. **Kompatibilita Wine**: CrossOver využívá vrstvu kompatibility Wine, kterou neustále vyvíjí projekt Wine a tým CrossOver ke zlepšení podpory aplikací Windows.
