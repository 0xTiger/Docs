## Co je soubor PART? ##

Soubor součásti nebo přípona .part je částečně stažený soubor. Používá se, když stahování probíhá nebo bylo přerušeno kvůli nějakému problému, což dává správci stahování příležitost obnovit stahování, kdykoli je to možné.
To znamená, že prohlížeč, jako je Firefox nebo programy pro přenos souborů jako eMule, eMule plus, FlashGet atd., ukládá část souboru nazvanou Part file, zatímco stahování probíhá na vašem zařízení. Tento soubor součásti vám ukáže, zda stahování probíhá nebo bylo před dokončením přerušeno. Nejen toto, ale soubor PART ukládá všechna data až do dokončení stahování, což je důvod, proč lze některá z nich později obnovit, pokud chcete stahování spustit znovu.

## Formát souboru PART ##

There are few browsers that break large download files into smaller downloads and assign each portion a .part extension. These files can be seen in the “Download” folder. Once the download is finished, the browser will combine all the PART files to give a final downloaded file. For example, while downloading a file from Firefox, the download manager will store the data being downloaded in a PART file and replaces the name with certain characters, and add a .part extension at the end, such as **doodle.or** becomes **JKzMn.or.part**. After completion, Firefox will remove the .part extension and the file will be ready to launch.
