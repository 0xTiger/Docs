## Co je soubor FWP?

Soubor se souborem .fwp je archivní soubor vytvořený pomocí Microsoft Expression Web a obsahuje všechny soubory a složky pro návrh. Microsoft Expression Web je obecný softwarový produkt společnosti Microsoft pro návrh webových stránek a je založen na editoru [HTML](/cs/web/html/). Od roku 2012 byla ukončena a je k dispozici zdarma ke stažení společností Microsoft. Soubory FWP lze exportovat a uložit ve formátu .cab společnosti Microsoft.

## Formát souboru FWP – Další informace

Přesné specifikace formátu souborů souborů FWP nejsou k dispozici pro vývojáře. Jedná se však o binární soubory a lze je otevřít v Microsoft Expression Studio, které bylo již dávno ukončeno.
