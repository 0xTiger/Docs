## Co je soubor XHTM?

Soubor s příponou .xhtm je soubor webové stránky napsaný v XHTML. Používá XML 1.0 jako svou standardní implementaci a je považován za přeformulování HTML 4. Stejně jako XHTML obsahuje soubor XHTM standardní HTML značky, které se řídí přísnějšími pokyny pro XML. Soubory XHTM se používají pro publikování obsahu webových stránek a běžně používají rozšíření [XHTML](/cs/web/xhtml/).

K otevření souborů XHTM můžete použít webové prohlížeče, jako je Google Chrome, Apple Safari a Mozilla Firefox.

## Příklad XHTM

Následující příklad dokumentu XHTM definuje deklarace XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
