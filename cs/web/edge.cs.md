## Co je soubor EDGE?

Soubor s příponou .edge je soubor projektu Adobe Edge, který se vytvoří při vytvoření projektu a uložení pomocí aplikace Adobe Edge. Adobe Edge umožňuje webovým designérům vytvářet interaktivní [HTML](/cs/web/html/) animátory pro web, digitální publikování a multimediální reklamu pro stolní i mobilní prohlížeče. Toho je dosaženo pomocí HTML/HTML5, [JavaScript](/cs/web/js/) a [CSS3](/cs/web/css/). Soubory Edge jsou generovány jako soubory ve formátu prostého textu a lze je otevřít pomocí několika různých aplikací, jako je Microsoft Notepad, Apple TextEdit a Notepad++.

## Formát souboru EDGE

Soubory EDGE se ukládají ve formátu prostého textu a pro aktivaci animace používá jQuery. Alternativou animace EDGE byly soubory Flash, které již nejsou podporovány. Soubor EDGE se uloží spolu se soubory HTML, JavaScript a CSS. Po otevření souboru EDGE se otevře webová stránka, tj. soubor HTML.
