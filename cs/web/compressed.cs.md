## Co je KOMPRESOVANÝ soubor?

COMPRESSED soubor je malá webová stránka komprimovaná Wordpressem vygenerovaná pomocí zásuvného modulu **WordCompress**. Obsahuje jeden řádek kódu pro reprezentaci webové stránky, která je vygenerována komprimací kódu HTML, CSS a JavaScript pomocí pluginu. Tyto převedené KOMPRESOVANÉ soubory mají malou velikost, což pomáhá rychlejšímu načítání a přenosu přes internet.

### KOMPRESOVANÝ Formát souboru – Další informace

KOMPRESOVANÉ soubory se ukládají jako textové soubory, které lze otevřít pomocí libovolného textového editoru, jako je Microsoft Notepad, Notepad++ a Apple TextEdit. Kromě textových editorů lze soubory COMPRESSED otevřít pomocí softwaru pro tvorbu webu, jako je Adobe Dreamweaver, který je k dispozici pro Windows i Mac OS.

## O pluginu WordCompress

Plugin WordCompress byl původně používán ke kompresi webových stránek WordPress do komprimovaného formátu souboru. Plugin již není podporován a je ukončen. Zapsal by zdrojový kód webové stránky do jediného spustitelného souboru odstraněním všech prázdných míst ze zdrojového kódu stránky. KOMPRESOVANÉ soubory jsou uloženy spolu s původním zdrojovým kódem webové stránky, aby byly zachovány původní komentáře, odsazení a jiná prázdná místa obsažená v původním zdrojovém kódu stránky.
