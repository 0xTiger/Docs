Při vytváření webové stránky je zapotřebí několik typů souborů. Webové formáty souborů definují standardy pro vývoj webových stránek a vztahují se k platformě, na které jsou zabudovány. Lze vytvořit kompletní web složený ze statických i dynamických webových stránek. Většina moderních webových stránek je postavena na technologii na straně serveru, jako jsou Active Server Pages (ASP), které se načítají a spouštějí na webovém serveru. Patří sem také kaskádové styly (CSS) a skriptovací soubory používané pro stylizaci celkového vzhledu a dojmu uživatelského rozhraní.

Mezi běžné webové přípony souborů a jejich přidružené formáty souborů patří **[HTML](/cs/web/html/)** (Hypertext Markup Language), **[ASP](/cs/web/asp/) (Active Server Pages)**, a **CSS** (Cascading Style Sheets).

Máte dotazy týkající se formátů webových souborů? Navštivte naše komunitní [fóra](https://forum.fileformat.com/c/web/13), kde můžete těžit ze znalostí sdílených odborníky na formát souborů.
