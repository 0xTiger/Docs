## Co je soubor BROWSER?

Soubor s příponou .browser používají webové aplikace ASP.NET k určení schopností a konfigurace webových prohlížečů. Obsahuje informace o možnostech, omezeních a vlastnostech prohlížeče. Tyto vlastnosti pomáhají frameworku ASP.NET rozpoznat žádající prohlížeč a podle toho prezentovat webovou stránku uživatelům.

## Formát souboru PROHLÍŽEČ

Definiční soubory BROWSER používá aplikace ASP.NET k určení schopností prohlížeče. Jsou to informace obsažené v tomto souboru, které server používá ke generování značek a dalších zdrojů, které jsou kompatibilní s žádajícím prohlížečem.

Soubory BROWSER se ukládají ve formátu prostého textu. Soubor BROWSER můžete otevřít v textových editorech, jako je Notepad, Notepad++, Apple TextEdit a Visual Studio IDE.

### Struktura souborů formátu souborů PROHLÍŽEČE

Definiční soubory BROWSER používají hierarchickou strukturu. Každý soubor .browser obsahuje označení XML, které zahrnuje možnosti a funkce konkrétního prohlížeče nebo skupiny prohlížečů. Tyto podrobnosti zahrnují vlastnosti, jako je řetězec agenta, čísla verzí a podpora technologií, jako je JavaScript, CSS nebo jiné prvky HTML. Pomocí těchto definičních souborů prohlížeče přizpůsobují vývojáři ASP.NET uživatelské prostředí na základě schopností různých prohlížečů.
