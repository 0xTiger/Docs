## Co je soubor DFG?

Soubor s příponou .dfg je tabulkový soubor vytvořený společností Interscape Inc. pro software Interscape Data Flask. Data jsou v souborech DFG vyplněna v řádcích a sloupcích. Data Flask je tabulkový software určený k uchování datových typů, zdrojů a klíčových informací. Může migrovat tabulky a indexy změnou pouze připojovacího řetězce. Nabízí funkce, jako je příkazový řádek VBScript, vzorce VBScript a makra spolu se skripty SQL pro aktualizaci a migraci dat. Soubory DFG se ukládají jako binární soubory na disk a lze je otevřít pomocí softwaru Interscape Data Flask.

## Formát souboru DFG – Další informace

Soubory DFG ukládají informace o tabulkových datech jako binární soubory na disk. Vnitřní struktura dat a podrobnosti formátu těchto souborů však nejsou známy.
