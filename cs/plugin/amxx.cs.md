## Co je soubor AMXX?

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. Jedná se o skriptovací platformu, která umožňuje správcům serverů přizpůsobit a rozšířit funkčnost jejich serverů. AMXX se používá ve spojení se systémem správy zásuvných modulů MetaMod, který umožňuje spouštění více zásuvných modulů současně na serveru bez konfliktů. Funkce nabízené zásuvným modulem AMXX zahrnují příkazy správce, úpravy hry a správu serveru.
