## Co je soubor WPS?

Soubor s příponou .wps je dokument Microsoft Works, který se před rokem 2006 používal jako dokument aplikace Word. Podobal se tehdy nově zavedenému formátu souboru [DOC](/cs/word-processing/doc/), ale nebyl tolik rysem jako nový formát souboru. Například nepodporoval funkci maker a pokročilé možnosti formátování. Soubory WPS lze stále otevírat pomocí moderních textových procesorů, ale pro Windows 10 je k načtení těchto souborů nutné nainstalovat konvertor. Microsoft Works byl nahrazen dokumentovým procesorem Microsoft Word v roce 2010.

## Specifikace formátu souboru

Specifikace formátu souboru dokumentu WPS nejsou veřejně dostupné, protože Microsoft neotevřel specifikace koncovým uživatelům. WPS jsou binární soubory a lze je otevřít pomocí filtrů pro převod formátu souborů vytvořených společností Microsoft pro textový procesor. LibreOffice, NeoOffice a OxygenOffice obsahují obecnou knihovnu C++, libwps, která dokáže extrahovat text z mnoha různých verzí Microsoft Works.
