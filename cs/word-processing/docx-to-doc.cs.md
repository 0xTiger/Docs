## O konverzi DOCX na DOC

Typ souboru DOC byl populární formát souboru dokumentu používaný v roce 2007 a předchozích verzích aplikace Microsoft Word. S vydáním sady Office 2007 společnost Microsoft představila formát souborů DOCX založený na specifikacích Office Open XML. Soubory DOC byly uloženy jako binární soubory ve srovnání s DOCX, který je uložen ve formátu komprimovaného souboru XML. Existuje mnoho online dostupných nástrojů pro převod souborů [DOCX](/cs/word-processing/docx/) na soubory [DOC](/cs/word-processing/doc/).

## Konverze DOCX na DOC – bezplatná ukázka

DOCX můžete převést do formátu DOC pomocí bezplatné online aplikace Aspose.app. Jedná se o sadu aplikací, které mohou načítat, prohlížet a převádět soubory DOCX do formátů souborů, jako jsou DOC, PDF, TXT, RTF a další. Aplikace Aspose jsou poháněny a spouštěny pomocí rozhraní Aspose API na backendu, která mohou provádět řadu operací s různými typy souborů. Vyzkoušejte online aplikaci Aspose.Words nyní kliknutím na následující tlačítko.

### Kroky pro převod DOCX na DOC online

DOCX můžete převést na formát souboru DOC pomocí [aplikace DOCX Aspose.Words na DOC](https://products.aspose.app/words/conversion/docx-to-doc) následovně.

1. Přejděte do aplikace Aspose.Words pro [konvertor DOCX na DOC](https://products.aspose.app/words/conversion/docx-to-doc)
1. Nahrajte svůj soubor DOCX kliknutím na tlačítko nahrát
1. Po dokončení stiskněte tlačítko Převést. Tím se spustí proces převodu souboru DOCX na DOC na pozadí
1. Po dokončení převodu DOCX na DOC budete požádáni o stažení převedeného souboru RTF

## Aspose.Words API

[Aspose.Words](https://products.aspose.com/words/) je sada rozhraní API, která každému vývojáři umožňuje převést [DOCX](/cs/word-processing/docx/) na [DOC](/cs/word-processing/doc/) pomocí jednoduchých řádků kódu. Je k dispozici pro mnoho programovacích jazyků včetně [C#](/cs/programming/cs/), [Java](/cs/programming/java/), [Python](/cs/programming/py/) a [C++](/cs/programming/cpp/). Kromě DOC vám rozhraní API Aspose.Words umožňují převádět soubory DOCX do mnoha dalších oblíbených formátů souborů.
