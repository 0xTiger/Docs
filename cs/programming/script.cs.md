## Co je soubor SCRIPT?

Soubor SCRIPT je zdrojový soubor, který obsahuje skript v libovolném skriptovacím jazyce. To z něj dělá zobecněný skriptovací soubor, který obsahuje proměnné a funkce. Vzhledem k tomu, že každý skriptovací jazyk se může lišit, skript napsaný v souboru SCRIPT musí být spuštěn pomocí vhodného skriptovacího stroje.

## Formát souboru SCRIPT – Další informace

Soubory SCRIPT se ukládají na disk jako soubory ve formátu prostého textu. Ty lze otevřít a upravit pomocí libovolného textového editoru, jako je Microsoft Notepad, Notepad++ a Apple TextEdit. Protože se skriptovací jazyk může lišit, lze jej spustit přejmenováním přípony souboru na výchozí příponu příslušného skriptovacího jazyka. Například existuje několik běžných skriptovacích jazyků, jako je JavaScript ([.JS](/cs/web/js/)), VBScript ([.VBS](/cs/programming/vbs/)), AppleScript (.SCPT), PHP ([ .PHP](/cs/programming/php/)) a ASP ([.ASP](/cs/web/asp/)).
