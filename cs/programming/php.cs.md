## Co je to soubor PHP?
Soubor s příponou .php označuje programovací jazyk s otevřeným zdrojovým kódem, který se používá k psaní skriptů na straně serveru, které se mají spouštět na webovém serveru. Je to nejrozšířenější webový skriptovací jazyk, který se obvykle používá pro vývoj rozsáhlých webových aplikací. Největší blogovací systém na webu tj. WordPress a největší sociální síť Facebook jsou založeny na jazyce PHP. Je kompatibilní s různými platformami jako Windows, Linux, Unix, Mac OS X atd. Úspěšně také běží na téměř všech dnes používaných serverech, tj. Apache, IIS atd. PHP podporuje širokou škálu databází včetně MySQL.

## Formát souboru PHP ##

Jedno PHP může obsahovat text, CSS, [HTML](/cs/web/html/), JavaScript a PHP kód. PHP kód se spustí na serveru a výsledek se vrátí prohlížeči jako prostý HTML. Soubory PHP lze otevřít pomocí libovolného textového editoru a upravovat na místě, ačkoli aplikace jako Adobe Dreamweaver nebo vývojové nástroje Eclipse PHP nabízejí pohodlný způsob psaní a úpravy kódu PHP.

### Syntaxe ###

Na serveru je spuštěn PHP skript a výsledek v prostém HTML je odeslán zpět do prohlížeče. PHP skript začíná na<?php and ends with ?>

### Příklad PHP ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
