## Co je soubor GROOVY?

Soubor GROOVY je soubor zdrojového kódu napsaný v programovacím jazyce Groovy. Kód napsaný uvnitř souboru GROOVY je podobný objektově orientovanému jazyku [Java](/cs/programming/java/), což usnadňuje návrh a vývoj aplikací. Soubor GROOVY je zkompilován do bajtového kódu Java virtual machine (JVM) a je kompatibilní s dalším kódem Java a knihovnami.

## Formát souboru GROOVY – Další informace

Soubory GROOVY obsahují zdrojový kód napsaný v syntaxi Groovy. Může obsahovat kód, který lze použít jako programovací jazyk i jako skriptovací jazyky pro platformu Java. Jazyk Groovy má vlastnosti podobné těm z Pythonu, Ruby a Smalltalku.
