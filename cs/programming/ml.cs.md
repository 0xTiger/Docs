## Co je soubor ML?

Soubor ML je textový soubor vytvořený pomocí funkčního programovacího jazyka ML (často známého jako Meta Language). Obsahuje zdrojový kód ML, který se častěji používá při výzkumu a návrhu programovacích jazyků pomocí zcela definované sémantiky. Existuje několik variant ML s Caml jako široce používaným. Mezi další patří Objective Caml (OCaml), JoCaml, Extended ML a MacroML. Jazyk ML se běžně používá pro psaní kompilátoru, automatizované dokazování teorémů a formální ověřování.
