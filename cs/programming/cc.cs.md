## Co je soubor CC?

Soubor CC je typ souboru zdrojového kódu, který obsahuje kód napsaný v programovacím jazyce C++. Může obsahovat buď veškerý kód nezbytný pro jeden program, nebo část kódu, který je součástí většího programovacího projektu, který se skládá z více souborů CC. K úpravě nebo zobrazení kódu obsaženého v souboru CC vývojáři často používají editor zdrojového kódu. Po úpravě zkompilují kód pomocí kompilátoru, aby jej převedli na strojový kód, který může operační systém počítače spustit pro spuštění programu.

## Formát souboru CC – Další informace

Přípona souboru .cc se používá pro soubory zdrojového kódu C++ již několik desetiletí. Jazyk C++ vyvinul koncem 70. let Bjarne Stroustrup, který v té době pracoval v Bell Labs. Stroustrup chtěl rozšířit programovací jazyk C o podporu objektově orientovaných programovacích konceptů, které považoval za nezbytné pro budování velkých softwarových systémů.

První verze C++ byla vydána v roce 1983 a rychle si získala oblibu mezi vývojáři, kteří ocenili její výkonné funkce a flexibilitu. Jak se C++ začalo více používat, bylo nutné zavést konvence kódování a konvence pojmenovávání souborů, které by pomohly organizovat a spravovat rozsáhlé základny kódu.

Průvodce stylem C++ společnosti Google, který doporučuje používat pro soubory zdrojového kódu C++ příponu .cc, byl poprvé publikován v roce 2003. Od té doby byl průvodce pravidelně aktualizován, aby odrážel změny v jazyce C++ a vývojové postupy širší programátorské komunity.

Dnes zůstává přípona souboru .cc oblíbenou volbou pro soubory zdrojového kódu C++ a je uznávána mnoha populárními vývojovými nástroji a platformami.

## Jak otevřít soubor CC?

K otevírání a úpravě souborů CC lze použít mnoho editorů zdrojového kódu, včetně multiplatformních nástrojů, jako je Microsoft Visual Studio Code a Code Blocks. Tyto editory nabízejí užitečné funkce, jako je zvýrazňování syntaxe a automatické dokončování, díky čemuž jsou vynikající volbou pro práci se soubory CC.

Soubory CC můžete také otevřít pomocí libovolného textového editoru, např. Notepad, Notepad++ atd.

## Kompilace CC pomocí kompilátorů C++

Kompilace souboru CC pomocí kompilátoru C++ zahrnuje několik kroků. Prvním krokem je instalace kompatibilního kompilátoru C++ pro váš operační systém a vývojové prostředí. K dispozici je mnoho různých kompilátorů včetně GCC, Clang a Microsoft Visual C++.

Jakmile máte nainstalovaný kompilátor C++, dalším krokem je otevřít terminál nebo příkazový řádek a přejít do adresáře, kde je umístěn soubor CC. Odtud můžete spustit kompilátor spuštěním nástroje příkazového řádku poskytovaného kompilátorem. Konkrétní příkaz pro kompilaci souboru CC se může lišit v závislosti na kompilátoru, který používáte. Například pro kompilaci souboru CC pomocí GCC byste spustili příkaz podobný:

```
g++ -o myprogram myfile.cc
```
