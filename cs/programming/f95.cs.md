## Co je soubor F95?

Soubor s příponou .f95 obvykle označuje, že se jedná o soubor zdrojového kódu Fortran 95. Fortran je univerzální, imperativní programovací jazyk, který je zvláště vhodný pro numerické a vědecké výpočty. Fortran 95 je vylepšená verze Fortran 90, která přináší několik nových funkcí a vylepšení.

Soubory zdrojového kódu Fortran obvykle obsahují instrukce programu napsané v programovacím jazyce Fortran; pokud máte soubor s příponou .f95, je pravděpodobně určen ke kompilaci pomocí kompilátoru Fortran; kompilace je proces překladu lidsky čitelného zdrojového kódu do strojově čitelného binárního kódu, který může být spuštěn počítačem.

## Jak otevřít soubor F95?

Pokud chcete pracovat nebo spouštět program v souboru .f95, budete potřebovat kompilátor Fortran; Mezi oblíbené kompilátory Fortran patří GNU Fortran (gfortran), Intel Fortran Compiler (ifort) a další. Programy Fortran můžete kompilovat a spouštět pomocí příslušných příkazů poskytnutých kompilátorem.

Například, pokud máte nainstalovaný gfortran, můžete použít následující příkazy v terminálu ke kompilaci a spuštění programu Fortran:

```
# Compile the Fortran program
gfortran -o my_program my_program.f95

# Run the compiled program
./my_program
```

Pokud chcete pouze zobrazit obsah souboru F95, jedná se o prostý textový soubor, takže jej můžete otevřít v libovolném textovém editoru.
