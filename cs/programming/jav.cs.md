## Co je to soubor Java?

Soubor Java je soubor zdrojového kódu Java, který byl vytvořen v programovacím jazyce Java. Je podobný zdrojovému souboru Java a lze jej zkompilovat pomocí interpretů Java. Stejně jako soubory [Java](/cs/programming/java/) se soubor Java používá při vývoji aplikací Java. Může to být jeden z mnoha souborů zdrojového kódu dostupných v projektu Java.

## Formát souboru Java

Soubory Java jsou uloženy jako textové soubory a lze je otevřít v libovolném textovém editoru. Typy souborů Java byly běžně nahrazeny typem souborů Java a běžně se používají pro psaní aplikací Java. Kódy Java lze psát pomocí IDE, jako je Eclipse a IntelliJ Idea, což usnadňuje práci se soubory Java.

### Příklad formátu souboru Java

```java
/*
The example code prints
Hello World from Java to the console.
*/
    public class JavaApp {
        public static void main(String[] args) {
            System.out.println("Hello World."); // Prints the string to the console.
        }
    }
```
