## Co je soubor NX?

Soubor NX je soubor videohry vytvořený pro použití s virtuální herní konzolí **LowRes NX**. Je napsán v programovacím jazyce BASIC a obsahuje zdrojový kód hry, který definuje, jak hra funguje. Můžete si vytvořit své vlastní soubory NX pro použití s aplikací, kterou lze načíst a uživatelé mohou hrát hru.

Soubor NX můžete otevřít v textovém editoru, LowRes NX nebo libovolném editoru zdrojového kódu, jako je Microsoft Visual Studio Code.

## Formát souboru NX

Soubor NX se uloží na disk jako prostý textový soubor, který obsahuje pokyny ke hře. Každý soubor NX má zdrojový kód hry, který se skládá z několika položek ROM kazety. Tyto položky ROM obsahují informace o herních komponentách, jako je grafika, mapy úrovní, zvuk a další data, která nejsou zdrojovým kódem hry. Záznamy ROM programu lze přidávat nebo upravovat pomocí editačních programů NX, které se dodávají s LowRes NX.

## Jak vytvořit soubory NX?

### Vytvářejte soubory NX na PC

Soubory NX můžete vytvořit na PC pomocí:

* Textové editory – jako je Microsoft Notepad, Notepad++ nebo Github Atom
* Editory zdrojového kódu – jako je Microsoft VIsual Studio Code

### Vytvářejte soubory NX v systému iOS

LowRes NX má verzi pro iOS, kterou mohou vývojáři použít k vytváření souborů NX z této aplikace.
