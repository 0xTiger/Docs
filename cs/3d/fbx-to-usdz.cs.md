## O FBX na USDZ

FBX, zkratka pro FilmBox, představuje široce rozšířený 3D formát souborů původně vytvořený společností Kaydara pro MotionBuilder. Společnost Autodesk Inc. převzala vlastnictví v roce 2006, čímž upevnila svůj status primárního formátu pro výměnu 3D dat využívaného množstvím 3D softwarových nástrojů.

Soubor s příponou .usdz představuje nekomprimovaný a nešifrovaný archiv ZIP přizpůsobený formátu souboru [USD](/cs/3d/usd/) (Universal Scene Description). Obsahuje proxy pro různé typy souborů, včetně textur a animací, vše pohodlně zapouzdřené v archivu.

Pokud jste vývojář aplikací a chcete do své aplikace .NET, Java nebo Python začlenit funkce [FBX](/cs/3d/fbx/) do USDZ, možná budete chtít prozkoumat rozhraní API Aspose.3D jako cenný zdroj pro dosažení tento cíl.
