## Asi 3MF na STL

3MF, 3D Manufacturing Format, je používán aplikacemi k vykreslování 3D modelů objektů pro řadu dalších aplikací, platforem, služeb a tiskáren. [STL](/cs/cad/stl/), zkratka pro stereolitografii, slouží jako všestranný formát souboru pro zobrazení trojrozměrné geometrie povrchu.

Převod souborů [3MF](/cs/3d/3mf/) do formátu souboru STL je častou úlohou a četné online aplikace nabízejí služby převodu zdarma. Pokud jste vývojář softwaru, který chce integrovat možnosti převodu 3MF na STL do vaší aplikace .NET, Java nebo Python, možná zjistíte, že API Aspose.3D jsou cenným řešením pro dosažení této funkce.
