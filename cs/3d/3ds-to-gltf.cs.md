## O 3DS na GLTF

Přípona souboru .3ds označuje formát síťového souboru 3D Studio (DOS), který využívá software 3D Studio společnosti Autodesk. Autodesk 3D Studio zaujímá významnou pozici v oblasti 3D formátů souborů od 90. let a následně se vyvinulo v 3D Studio MAX.

glTF nebo GL Transmission Format je 3D formát souboru, který používá zápis JSON k zapouzdření dat 3D modelu. Tento přístup založený na JSON efektivně snižuje velikost 3D aktiv a minimalizuje výpočetní zdroje potřebné pro extrakci a použití v reálném čase.

Převod [3DS](/cs/3d/gltf/) na [formát souboru GLTF](/cs/3d/gltf/) je častou potřebou a četné online nástroje nabízejí převod z 3DS na GLTF zdarma. Pokud jste vývojář aplikací a chcete integrovat funkce 3DS do GLTF do vaší aplikace .NET, Java nebo Python, možná budete chtít prozkoumat API Aspose.3D pro tento úkol.
