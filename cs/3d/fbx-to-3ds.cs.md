## O FBX na 3DS

FilmBox, běžně označovaný jako FBX, je široce rozšířený formát 3D souborů původně vyvinutý společností Kaydara pro usnadnění kompatibility s MotionBuilder. Autodesk Inc. převzal kontrolu nad tímto formátem v roce 2006 a stanovil jej jako klíčový standard pro bezproblémovou výměnu 3D dat mezi různými 3D softwarovými aplikacemi.

Soubor s příponou [.3ds](/cs/3d/3ds/) označuje formát síťového souboru 3D Studio (DOS), který používá software 3D Studio společnosti Autodesk. Přítomnost Autodesku v aréně 3D formátů souborů sahá až do 90. let 20. století a od té doby se vyvinula v 3D Studio MAX, komplexní nástroj pro 3D modelování, animace a renderování. Soubor .3ds zapouzdřuje data pro trojrozměrné zobrazení scén a obrázků, což z něj činí jeden z oblíbených formátů souborů pro import a export 3D dat.

Konverze [FBX](/cs/3d/fbx/) na 3DS je častým požadavkem uživatelů. Existuje několik bezplatných online nástrojů pro konverzi FBX na 3DS ZDARMA. Pokud jste vývojář softwaru, který chce integrovat funkce převodu [FBX](/cs/3d/fbx/) do 3DS ve své aplikaci .NET, Java nebo Python, možná budete chtít prozkoumat rozhraní API Aspose.3D jako životaschopné řešení pro dosažení tuto funkci.
