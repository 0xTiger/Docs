## Co je soubor ACCDE?

Soubor s příponou .accde je formát souboru aplikace Microsoft Access 2007 pro vytváření databázových souborů typu Execute Only za účelem ochrany kódu vlastní databáze. Kompiluje veškerý kód VBA pouze do spustitelného souboru, takže jej nelze prohlížet ani upravovat. Formát souboru ACCDE je náhradou za formát souboru MDE, který byl používán dřívějšími verzemi aplikace Microsoft Access. ACCDE jsou soubory [ACCDB](/cs/database/accdb/) pouze ve spustitelném režimu a kompaktnější díky absenci jakéhokoli zdrojového kódu. Kompaktní velikost těchto souborů zlepšuje využití paměti a také zvyšuje výkon aplikace. Soubory ACCDE lze otevřít pomocí Microsoft Access 365.

## Formát souboru ACCDE

Všechny soubory Microsoft Access 2007+ využívají Jet ACE (databázový stroj Access), který používá rodinu formátů souborů ACCDB. Microsoft však neposkytuje implementaci open source ani podrobnou technickou specifikaci.
