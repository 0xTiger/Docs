## Co je soubor FDB?

Soubor .fdb je přípona souboru používaná pro databázové soubory v Microsoft Dynamics NAV. Přípona souboru .fdb znamená Firebird Database File a je to formát souboru používaný Firebird Database Management System, což je základní databázový stroj používaný Microsoft Dynamics NAV. Soubor .fdb obsahuje všechna data, tabulky a struktury systému NAV, včetně finančních dat, skladových dat, zákaznických dat a dalších. Je důležité ponechat si zálohu souboru .fdb, aby byla zajištěna integrita dat a aby bylo možné data v případě jakéhokoli problému obnovit. Soubor .fdb lze exportovat, importovat a také použít při replikaci.

## Vztah s Microsoft Dynamics NAV

Soubory FDB jsou databázové soubory v Microsoft Dynamics NAV. Microsoft Dynamics NAV je software pro plánování podnikových zdrojů (ERP) vyvinutý společností Microsoft. Je určen pro malé a střední podniky a poskytuje širokou škálu funkcí pro řízení podniku, včetně finančního řízení, řízení dodavatelského řetězce, výroby, projektového řízení a dalších. Je postaven na platformě Microsoft Dynamics a je plně integrován s dalšími produkty společnosti Microsoft, jako jsou Office a Outlook. K Dynamics NAV lze přistupovat prostřednictvím webového klienta i klienta Windows a lze jej také přizpůsobit a integrovat s jinými systémy pomocí vlastního vývojového prostředí C/AL. Běžně jej používají společnosti ve výrobě, velkoobchodní distribuci a službách.

## Jak otevřít soubor .fdb?

Soubor FDB se obvykle otevírá a spravuje pomocí vývojového prostředí Microsoft Dynamics NAV nebo klienta Microsoft Dynamics NAV.

Zde jsou kroky k otevření souboru .fdb v Microsoft Dynamics NAV:

1. Otevřete vývojové prostředí Microsoft Dynamics NAV.
2. Klikněte na nabídku Soubor a vyberte Otevřít nebo stiskněte Ctrl+O
3. Přejděte do umístění, kde je uložen soubor .fdb.
4. Vyberte soubor .fdb a klikněte na tlačítko Otevřít.

Soubor .fdb můžete také otevřít připojením k databázi pomocí klienta NAV.

1. Otevřete klienta NAV.
2. Klikněte na nabídku Soubor a vyberte Připojit
3. Do pole Server zadejte název nebo IP adresu serveru, na kterém je umístěn soubor .fdb.
4. Zadejte příslušné Název databáze a Přihlašovací údaje
5. Klikněte na tlačítko Připojit.

Soubory .fdb je také možné otevřít pomocí softwaru pro správu databázového systému Firebird, protože jde o základní databázový stroj používaný Microsoft Dynamics NAV.
