## Co je soubor CAT?

CAT file patří do Advanced Disk Catalog software, vyvinutý ElcomSoft Co. Ltd, ruská softwarová společnost, která vyvíjí různé softwarové nástroje pro počítačovou bezpečnost a forenzní. Některé z jejích produktů zahrnují software pro obnovu hesla, získávání mobilních zařízení a cloudovou extrakci. Společnost byla založena v roce 1990 a má pobočky v Rusku a Spojených státech. Jsou dobře známé pro svůj forenzní software a celkově jejich software používají soukromé společnosti i vládní agentury k provádění forenzních analýz a vyšetřování.

Advanced Disk Catalog is a cataloguing program for Windows XP/2003. Všestrannost ADC je odvozena ze skutečnosti, že se nespoléhá na databázový stroj, takže je vysoce efektivní a kompaktní. Jeho uživatelské rozhraní, podobné Windows Exploreru, umožňuje snadnou organizaci dat uložených na různých typech médií, jako jsou pevné disky (včetně síťových), diskety, optické disky, ZIP a JAZ disky atd.

## Jak otevřít soubory CAT?

Stáhněte si software Elcomsoft Advanced Disk Catalog. Ve Windows můžete kliknout pravým tlačítkem na průzkumník souborů a poté vybrat otevřít pomocí kontextové nabídky, zadat cestu k softwaru Elcomsoft Advanced Disk Catalog pro otevření souboru cat.
