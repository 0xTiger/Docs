## Co je soubor CHCK?

Soubor CHCK je vytvořen jako výsledek kontrol sledování stavu na serveru Microsoft Exchange Server. Obsahuje informace o celkovém stavu a provozu databáze serveru. Microsoft Exchange Server se používá především jako e-mailový server a umožňuje vám spravovat vaše obchodní kontakty a kalendáře. Správci serveru Exchange používají jeho EseUtil ke kontrole stavu databáze.
