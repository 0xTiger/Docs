## Co je soubor JET?

Soubor databáze JET, také známý jako soubor databáze Microsoft Access, je formát souboru používaný aplikací Microsoft Access, populárním systémem správy databází (DBMS). JET je zkratka pro Joint Engine Technology, což byl původní název databázového stroje, který Microsoft Access používal. Pozdější verze Accessu používaly různé databázové stroje, ale termín "JET databáze" se stále běžně používá k označení databází Accessu.

Zde jsou některé klíčové body o databázových souborech JET:

- **Přípona souboru:** Soubory databáze JET mají obvykle příponu ".mdb" pro starší verze Accessu (Access 2003 a starší) a ".accdb" pro novější verze (Access 2007 a novější). Soubory ".mdb" používají starší databázový stroj JET, zatímco soubory ".accdb" používají novější technologii ACE (Access Database Engine).

- **Relační databáze:** Databáze JET jsou relační databáze, což znamená, že ukládají data do tabulek s definovanými vztahy mezi nimi. Uživatelé mohou vytvářet, upravovat a dotazovat se na data pomocí SQL (Structured Query Language) nebo prostřednictvím grafického rozhraní Accessu.

- **Ukládání dat:** Databáze JET mohou ukládat širokou škálu typů dat, včetně textu, čísel, dat, obrázků a dalších. Zvládnou také složité datové struktury a vztahy mezi tabulkami.

- **Přístup pro více uživatelů:** Databáze JET podporují přístup pro více uživatelů, což umožňuje více uživatelům pracovat s databází současně, i když mohou existovat omezení z hlediska škálovatelnosti a výkonu pro rozsáhlé aplikace.

- **Integrace:** Databáze Accessu lze integrovat s dalšími aplikacemi Microsoft Office, jako jsou Excel a Outlook, což uživatelům usnadňuje výměnu dat mezi různými programy.

## Informace o databázích JET

**Historie JET Engine:**

Databázový stroj JET (Joint Engine Technology) byl původně vyvinut společností Microsoft na počátku 90. let pro použití v aplikaci Microsoft Access. Engine byl navržen jako odlehčené, desktopově orientované databázové řešení, které lze integrovat do různých produktů společnosti Microsoft.

**Součásti databáze JET**

- **Tabulky:** Databáze JET ukládají data do tabulek, což jsou kolekce záznamů uspořádaných do řádků a sloupců.
- **Dotazy:** Uživatelé mohou vytvářet dotazy pro extrakci, filtrování a manipulaci s daty z tabulek pomocí SQL (Structured Query Language).
- **Formuláře:** Access poskytuje návrhář formulářů, který uživatelům umožňuje vytvářet vlastní zadávání dat a zobrazovat formuláře.
- **Zprávy:** Uživatelé mohou z dat uložených v databázi vytvářet profesionálně vyhlížející zprávy.
- **Makra a VBA:** Uživatelé mohou automatizovat úlohy a přidávat vlastní funkce pomocí maker nebo psaním kódu VBA.

**Typy dat**

Databáze JET podporují různé typy dat, včetně:

- **Text:** Pro ukládání alfanumerických znaků.
- **Číslo:** Pro číselné hodnoty.
- **Datum/čas:** Pro hodnoty data a času.
- **Poznámka:** Pro delší text nebo poznámky.
- **Hypertextový odkaz:** Pro webové odkazy a e-mailové adresy.
- **Příloha:** Pro ukládání souborů a příloh.
