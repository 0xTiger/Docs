## Co je soubor MPT? ##

Soubory s příponou .mpt jsou soubory šablon [Microsoft Project](https://products.office.com/en-us/project/project-and-portfolio-management-software). Obsahuje základní informace a strukturu a nastavení dokumentu pro vytváření souborů [MPP](/cs/project-management/mpp/). Tyto šablony poskytují výchozí nastavení, která jsou nezbytná pro konkrétní projekt, jako jsou plány nebo informace o rozpočtu. Nemůže však ukládat data specifická pro projekt, jako jsou úkoly, zdroje nebo přiřazení. Po úpravě lze soubory šablon uložit jako standardní soubory MPP pro další práci s nimi. Soubor hlavní šablony s názvem "Global.mpt" je k dispozici v aplikaci Microsoft Project a lze jej otevřít za účelem vytvoření individuální šablony. Soubory MPT používají *application/vnd.ms-project* a *application/x-project* jako standardní typy MIME. Windows, Linux a macOS mohou všechny otevírat soubory MPT spolu s Moos Project Viewer, Microsoft Project a MPXJ.

## Problémy s otevřením souboru MPT? ##

Zde je seznam některých běžných problémů, které mohou nastat a způsobit nesprávné fungování formátu MPT:

* Absence podpůrného softwaru
* Poškozený soubor
* Soubor infikovaný virem
* Žádné přístupové právo v systému k otevření souborů
* Zastaralá jednotka ve vašem systému
 *   Extension of the file is renamed
