## Co je soubor MPP?

Soubor s příponou .mpp je datový soubor aplikace Microsoft Project, který integrovaným způsobem ukládá informace související se správou projektu. Společnost Microsoft vyvinula tento proprietární formát souborů, aby byl kompatibilní s Microsoft Project (MSP), jejich softwarovým produktem pro správu projektů. Kromě MPP může MSP pracovat se schématem XML projektu. Mnoho aplikací a rozhraní API umožňuje převod MPP do jiných formátů souborů. Microsoft má nyní online Project Server, kam lze nahrávat soubory pro správu projektů pro spolupráci více uživatelů.

## Formát souboru MPP ##

Společnost Microsoft vyvinula a zveřejnila MPP jako binární formát souboru a jeho specifikace dosud nebyly společností Microsoft zveřejněny. Vnitřní struktura souborů a podrobnosti proto nejsou pro vývojáře dostupné. Existuje několik rozhraní API dostupných online, která podporují čtení/zápis do souborů MPP, ale nikdy není dosaženo 100% přesnosti. Společnost Microsoft v průběhu času zavedla do své aplikace pro správu projektů několik funkcí, stejně jako funkce formátu souboru MPP.

Soubor MPP používá následující typy MIME:

* aplikace/projekt vnd.ms
* aplikace/msproj
* aplikace/msprojekt
* aplikace/x-msproject
* aplikace/x-ms-projekt
* aplikace/projekt x-dos_ms
* aplikace/mpp
* zz-application/zz-winassoc-mpp

## Problémy s otevřením souboru MPP? ##

Zde je seznam některých běžných problémů, které mohou nastat a způsobit nesprávné fungování formátu MPP:

* Absence podpůrného softwaru
* Poškozený soubor
* Infikovaný soubor kvůli viru
* Žádné přístupové právo v systému k otevření souborů
* Zastaralá jednotka ve vašem systému
* Přípona souboru je přejmenována
