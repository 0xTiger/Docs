## Co je soubor CDR?

Soubor **.CDR** s **"Raw CD Audio Data"** obvykle označuje soubor obsahující zvukové informace extrahované přímo ze zvukového CD, jako jsou hudební stopy. "Raw CD Audio Data" označuje nezpracovaná digitální zvuková data z CD, která jsou často uložena v binárním formátu. Tato nezpracovaná data obsahují zvukové vzorky přesně tak, jak jsou na CD, včetně informací o zvuku a podkódu.

Tyto soubory nejsou obvykle určeny k přímému přehrávání, ale místo toho se používají pro různé účely, jako je mastering CD, úprava zvuku a vypalování CD. Softwarové aplikace pro úpravu zvuku a tvorbu CD často používají tato nezpracovaná data k vytváření zvukových disků CD nebo je převádějí do jiných formátů zvukových souborů, jako je WAV nebo MP3.

Pokud potřebujete pracovat se soubory .cdr obsahujícími „Raw CD Audio Data“, obvykle byste k jejich zpracování nebo konverzi použili specializovaný zvukový software, protože nejde o standardní zvukové soubory, které lze přehrát ve většině přehrávačů médií.

## Jak převést soubor CDR?

Chcete-li převést soubor .cdr obsahující „Raw CD Audio Data“ na běžnější zvukový formát, jako je WAV nebo MP3, budete potřebovat specializovaný software. Pokud jste schopni přehrát soubor .cdr v přehrávači médií VideoLAN VLC, můžete jej také pomocí VLC převést do různých zvukových formátů, např.

- [.MP3](/cs/audio/mp3/) - Zvuk ve formátu MP3
- [.FLAC](/cs/audio/flac/) - Bezeztrátový zvukový kodek
- [.WAV](/cs/audio/wav/) - Zvuk WAVE
- [.OGG](/cs/audio/ogg/) - Ogg Vorbis Audio

## Jak otevřít soubor CDR?

Otevření souboru .cdr, zejména souboru obsahujícího „Raw CD Audio Data“, může být trochu složitější, protože to není standardní formát zvukového souboru. K otevření a práci s takovými soubory můžete použít specializovaný software, který zvládne nezpracovaná zvuková data CD. Zde je návod, jak to udělat, a některé možnosti softwaru:

**Software pro otevírání a práci se soubory .cdr:**

1. **Přesná kopie zvuku (EAC)**:
    





- Stáhněte a nainstalujte Exact Audio Copy, což je populární a výkonný nástroj pro práci se zvukovými disky CD a extrahování „Raw CD Audio Data“.
- Vložte CD se souborem .cdr do jednotky CD/DVD počítače.
- Otevřete EAC a nakonfigurujte jej podle svých preferencí, včetně výstupního formátu (např. WAV nebo MP3).
- Pomocí EAC extrahujte zvukové stopy z CD a uložte je v požadovaném formátu.
2. **Audacity**:
    





- Audacity je bezplatný software pro úpravu zvuku s otevřeným zdrojovým kódem, který může pracovat také s „Raw CD Audio Data“.
- Vložte CD.
- Otevřete Audacity a nakonfigurujte jej pro nahrávání zvuku z jednotky CD.
- Nahrajte zvuk z disku CD a poté jej uložte v běžnějším zvukovém formátu.
3. **VLC Media Player**:
    





- VLC umí přehrávat zvuk přímo z CD, včetně těch s "Raw CD Audio Data."
- Vložte CD do počítače.
- Otevřete VLC, přejděte do nabídky "Média" a vyberte "Otevřít disk."
- Vyberte jednotku CD a přehrávejte přímo zvukové stopy.
4. **Software CD Ripper**:
    





- K dispozici jsou různé možnosti softwaru pro ripper CD, jako je Windows Media Player (pro Windows), iTunes (pro Mac) a další.
- Vložte CD do počítače.
- Otevřete svůj preferovaný software pro ripování disků CD a použijte jej k extrahování zvukových stop a jejich uložení v běžném zvukovém formátu.
5. **Nástroje DAEMON** (pro soubory obrázků .cdr):
    





- Pokud je soubor .cdr obrazový soubor vytvořený softwarem, jako je DAEMON Tools, můžete obraz připojit pomocí DAEMON Tools nebo podobného softwaru.
- Jakmile je obraz připojen, můžete přistupovat k jeho obsahu, jako by to bylo fyzické CD, a přehrávat nebo extrahovat zvuk podle potřeby.
