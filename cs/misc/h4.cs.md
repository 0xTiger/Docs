## Co je soubor H4?

H4 nebo HDF4 je jeden z Hierarchických datových formátů (HDF) a používá se k ukládání velkého množství dat. Jedná se o souborový formát s otevřeným zdrojovým kódem, který podporuje ukládání složitých heterogenních dat. HDF4 je starší datový formát ve srovnání s [HDF5](/cs/misc/h5/). Formáty souborů HDF spravuje a vydává nezisková organizace HDF group. Soubory HDF také podporují ukládání metadat, která plně reprezentují uložená data.

## Stručná historie formátu souborů HDF

Formulace hierarchických datových formátů byla zahájena v roce 1987 pracovní skupinou Graphics Foundations Task Force (GFTF). Během této doby bylo navrženo několik různých formátů souborů pro standardizaci jednoho formátu. NASA zvolila HDF jako standardní metodu pro ukládání a poskytování vědeckých dat.

## Formát souboru HDF4

Soubory H4 se ukládají do struktury „souborového adresáře“ podle specifikace formátu souboru HDF. Tyto specifikace definují celkovou strukturu souborů HDF pro uložení na disk, aniž by se uživatelé museli starat o základní strukturu souborů.

Oblasti použití souborů H4 zahrnují letectví, fyziku, strojírenství, finance, akademický výzkum, genomiku, astronomii, elektronické přístroje a lékařské obory.

### Datový model HDF4

Soubor HDF4 nebo H4 je jako kontejner, který se skládá z heterogenních datových objektů. Typy datových sad podporované soubory H4 zahrnují:

* Snímky
* Tabulky
* Textové prvky
* Grafy
* Dokumenty jako [PDF](/cs/pdf/) a [dokumenty Excel](/cs/tabulkový procesor/)

## Prohlížeč souborů H4

Skupina HDF poskytuje [software HDFView](https://www.hdfgroup.org/solutions/hdf5/), který se skládá z utility HDFView a Java HDF Object Package.
