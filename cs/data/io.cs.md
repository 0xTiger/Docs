## Co je to IO soubor?

IO soubor je spojen s BrickLink Studio, softwarovým nástrojem používaným pro navrhování virtuálních modelů LEGO. Když uložíte model LEGO v BrickLink Studio, uloží se s příponou souboru .io.

BrickLink Studio používá příponu souboru IO k ukládání všech informací o modelu LEGO, který jste vytvořili, včetně umístění a konfigurace kostek, barev a dalších prvků; tento formát souboru vám umožňuje ukládat a sdílet vaše návrhy s ostatními, kteří používají BrickLink Studio nebo kompatibilní software.

## O BrickLink Studio

BrickLink Studio je softwarový nástroj určený pro LEGO nadšence, kteří milují digitální stavění a navrhování vlastních výtvorů; poskytuje uživatelům virtuální prostředí, kde mohou sestavovat modely LEGO pomocí široké škály digitálních prvků LEGO, včetně kostek, destiček, minifigurek a dalších.

Mezi klíčové funkce BrickLink Studio patří:

1.  **Stavební nástroje**: BrickLink Studio nabízí řadu nástrojů a funkcí, které uživatelům pomohou s přesností a kreativitou sestavovat modely LEGO; tyto nástroje umožňují snadnou manipulaci, otáčení a umísťování prvků LEGO a umožňují uživatelům přivést své nápady k životu v digitální sféře.
    
2.  **Vykreslování**: Software obsahuje možnosti vykreslování, které uživatelům umožňuje vizualizovat své výtvory LEGO ve vysoce kvalitní 3D grafice; tato funkce umožňuje uživatelům vidět, jak budou jejich modely vypadat z různých úhlů a perspektiv, což zlepšuje proces navrhování.
    
3.  **Generování instrukcí**: BrickLink Studio umožňuje uživatelům vytvářet podrobné návody na stavění jejich modelů LEGO; tato funkce je neocenitelná pro sdílení návrhů s ostatními nebo pro osobní použití při pozdějším budování fyzického modelu.
    
4.  **Knihovna součástí**: Software poskytuje přístup k rozsáhlé knihovně digitálních dílů LEGO, včetně dílů z různých témat a kolekcí LEGO; tento obrovský výběr prvků dává uživatelům svobodu vytvářet prakticky jakýkoli model LEGO, který si dokážou představit.

## Stud IO Files

Stud.io je další populární software používaný LEGO nadšenci pro digitální LEGO design, podobně jako BrickLink Studio. Stejně jako souborový formát .io BrickLink Studio, Stud.io také používá svůj vlastní formát souborů k ukládání modelů LEGO; nicméně, Stud.io ukládá soubory s příponou .io, což může někdy způsobit zmatek.

Soubory Stud.io, často označované jako soubory .io, obsahují všechny informace nezbytné k digitální reprezentaci modelu LEGO; to zahrnuje podrobnosti o umístění, orientaci a barvě každého prvku LEGO v modelu.

Stud.io nabízí mnoho stejných funkcí jako BrickLink Studio, včetně širokého výběru digitálních LEGO dílů, nástrojů pro stavbu a úpravu modelů, renderovacích možností pro vytváření vysoce kvalitních obrázků a schopnosti generovat podrobné návody na stavění.

Když uložíte model LEGO do Stud.io, vytvoří se soubor .io, ve kterém jsou uloženy všechny informace o modelu; tyto soubory lze znovu otevřít ve Stud.io pro další úpravy nebo sdílení s ostatními. Stud.io navíc umožňuje uživatelům exportovat modely do různých formátů souborů pro použití v jiném softwaru nebo pro sdílení s širší komunitou LEGO.

## Jak otevřít soubor IO

BrickLink Studio umí otevírat IO soubory na Windows a Mac. Jednoduše spusťte BrickLink Studio a přejděte do nabídky Soubor a z rozbalovací nabídky vyberte Otevřít.

Zde je seznam programů, které mohou otevřít soubory IO.

- **LEGO BrickLink Studio** (zdarma) pro (Windows, Mac)
- **Stud.io**
