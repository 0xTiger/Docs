## .3GP 파일이란?

3GP(3GPP File Format)는 3G 휴대폰에서 사용되는 멀티미디어 컨테이너 형식입니다. 일부 2G 및 4G 휴대폰에서도 재생할 수 있습니다. 3GP는 3G UMTS(Universal Mobile Telecommunications System) 멀티미디어 서비스를 위한 3GPP(3세대 파트너십 프로젝트)에 의해 정의됩니다. 3GP 파일은 .3gp 확장자로 저장됩니다. 3GP 파일 형식은 GSM 기반(Global System for Mobile Communications) 전화기용으로 설계되었으며 MMS(멀티미디어 메시징 서비스)를 통해 전송되는 미디어 파일의 표준 형식입니다. H.263 코덱을 사용하는 [MP4](/ko/video/mp4/)와 달리 3GP 파일은 H.263 코덱을 사용하는 것이 좋습니다.

## Technical Details of 3GP File Format

3GP 비디오 형식은 ISO/IEC 14496-12 – MPEG-4 Part 12에 정의된 ISO 기본 미디어 파일 형식을 기반으로 합니다. 저장 공간, 대역폭 및 데이터 사용량을 줄이기 위해 휴대폰용으로 개발되었습니다. 3GP 파일 형식은 다음 코덱을 지원합니다.

- 동영상
  - MPEG-4 Part 2
  - H.263
  - MPEG-4 Part 10 (AVC/H.264)
- 오디오
  - AMR-WB
  - AMR-WB+
  - AMR-NB
  - AAC-LC
  - HE-AAC v1
  - HE-AAC v2 (Enhanced aacPlus)

3GP 파일 형식은 H.263 및 AMR 코덱의 사용을 허용하고 코덱이 참조하는 새로운 상자를 정의했습니다. 이러한 코드 포인트는 MPEG-4 미디어 관련 정보의 저장을 위해 ISO 기본 미디어 파일 형식으로 등록되었습니다.

## 3GP 비디오 파일 지원 장치

- 3G 기능이 있는 휴대폰
- CD에서 가져온 PlayStation 3 오디오
- Nintendo 3DS는 3GP 기술을 사용하여 YouTube 동영상을 재생했습니다.
- Apple의 iDevices는 통과 파일로만 파일 재생을 지원했습니다.
