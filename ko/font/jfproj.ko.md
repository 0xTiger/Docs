## .JFPROJ 파일이란?

확장자가 .jfproj인 파일은 Windows 10에서 Microsoft Font Maker 응용 프로그램으로 만든 JSON 기반 사용자 지정 글꼴입니다. 태블릿과 같은 터치 스크린 장치와 함께 전자 펜을 사용하여 만들 수 있지만 경우에 따라 마우스와 같은 다른 컴퓨터 주변 장치를 사용할 수도 있습니다. 컴퓨터의. JFPROJ 파일에는 문자와 단어 사이의 간격 및 글꼴 크기를 정의하여 문자를 화면에 렌더링하는 데 사용되는 글리프가 포함되어 있습니다. Microsoft Font Maker 앱은 Windows 스토어에서 다운로드할 수 있습니다.
