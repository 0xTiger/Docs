## .KODU 파일이란?

.kudo 파일은 교육용 게임 개발 환경인 마이크로소프트[Kodu Game Lab 소프트웨어](https://www.kodugamelab.com/)로 생성하여 저장한 3D 게임 파일입니다. 아이들에게 기본적인 프로그래밍 원리를 가르치고 제작자가 시각적 프로그래밍 언어를 사용하여 게임 요소의 동작을 프로그래밍할 수 있도록 설계되었습니다. Kodu 파일은 저장하고 다른 사용자와 공유할 수 있지만 Kodu 소프트웨어에서만 열고 실행할 수 있습니다.

## KODU 파일 형식 - 추가 정보

Kodu 파일은 Microsoft 독점 파일 형식으로 저장된 이진 파일입니다. 파일 형식 사양은 개발자가 공개적으로 사용할 수 없기 때문에 Kodu 파일은 Kodu Game Lab 소프트웨어 내에서만 실행할 수 있습니다.
