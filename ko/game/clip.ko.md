## .CLIP 파일이란?

확장자가 .clip인 파일은 컨트롤러 입력 데이터가 포함된 GTA V(Grand Theft Auto 5) 재생 파일입니다. Rockstar 편집기는 게임 플레이에서 시퀀스를 재생하기 위해 이 데이터를 참조합니다. 실제 비디오 재생이 아니라 밀리초 단위의 컨트롤러 입력 기록입니다. 이렇게 하면 편집기를 통해 카메라 각도 설정을 조정할 수 있습니다. 그러나 편집기를 사용하여 [MP4](/ko/video/mp4/)와 같은 인기 있는 비디오 파일 형식으로 실제 비디오를 만들 수 있습니다. GTA V는 내보낸 비디오를 C:\Users\YOUR USER NAME\AppData\Local\Rockstar Games\GTA V\videos\rendered에 저장합니다.

## CLIP 파일 형식

CLIP 파일은 실제 비디오 파일이 아닙니다. 대신 여기에는 위치, 개체의 회전 값, 세계 상태 및 조건, 오디오 트리거, 게임 카메라 움직임, 플레이어 동작 등과 같은 게임 데이터가 포함됩니다. 게임 밖에서는 볼 수 없습니다.

## GTAV에서 클립을 녹화하고 Rockstar 편집기에 액세스하는 방법은 무엇입니까?

클립 녹화가 완료되면 다음 단계를 사용하여 클립을 녹화할 수 있습니다.

* 게임 일시 중지
* 맨 오른쪽 탭인 "Rockstar Editor"까지 스크롤하십시오.
* 이 탭에서 편집기 실행
* 여기에서 "클립 관리" 화면을 선택하고 GTA 온라인 또는 스토리 모드에서 녹화한 클립을 변경할 수 있습니다.
