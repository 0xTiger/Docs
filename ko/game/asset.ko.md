## ASSET 파일이란 무엇입니까?

ASSET 파일은 비디오 게임 제작 프로그램인 Unity에서 사용되는 특수 파일입니다. 게임 개체나 설정과 같은 중요한 구성을 보유합니다. 예를 들어 하나의 ASSET 파일에는 개체가 서로 튕겨 나가거나 떨어지는 방식과 같이 게임에서 개체가 움직이는 방식을 저장할 수 있습니다. 이러한 파일은 게임 개발자가 프로젝트를 쉽게 수행할 수 있도록 모든 중요한 정보를 체계적으로 정리하는 컨테이너와 같습니다.

## ASSET 파일을 여는 방법
  
Unity에서는 일반적으로 텍스트 편집기나 유사한 프로그램에서 문서나 파일을 여는 것과 같은 방식으로 자산 파일을 열지 않습니다. 대신 Unity는 Unity 에디터 자체 내에서 자산 파일을 관리합니다. Unity에서 에셋 파일에 액세스하고 작업하는 방법은 다음과 같습니다.

1.  **Import Assets**: To use an asset file in your Unity project, you need to import it into project. You can do this by either dragging and dropping asset file directly into Unity Editor or by using "Assets" menu and selecting "Import New Asset."
    
2.  **에셋 보기**: 가져온 후에는 Unity 에디터의 프로젝트 창에서 에셋 파일을 보고 관리할 수 있습니다. 이 창에는 프로젝트의 모든 자산이 표시되며 이를 폴더로 구성하고, 특정 자산을 검색하고, 해당 내용을 미리 볼 수 있습니다.
    
3.  **자산 사용**: 프로젝트 창에서 장면 계층 구조나 장면의 GameObject로 자산 파일을 드래그하여 Unity 프로젝트에서 자산 파일을 사용할 수 있습니다. 예를 들어 텍스처 자산을 재질로 드래그하여 3D 객체에 적용하거나 프리팹 자산을 장면으로 드래그하여 사전 구성된 GameObject를 인스턴스화할 수 있습니다.
    
4.  **에셋 편집**: 스크립트, 머티리얼, 애니메이션과 같은 일부 에셋 파일은 Unity Editor 내에서 직접 편집할 수 있습니다. 프로젝트 창에서 이러한 자산을 두 번 클릭하여 해당 편집기에서 열고 필요에 따라 변경할 수 있습니다.
    
5.  **에셋 저장**: Unity 에디터 내에서 에셋 파일에 대한 모든 변경 사항은 Unity 프로젝트 내에 자동으로 저장됩니다. 기존 소프트웨어 프로그램에서 문서를 저장하는 것처럼 개별 자산 파일을 명시적으로 저장할 필요가 없습니다.

ASSET 파일은 다음 프로그램을 사용하여 열거나 참조할 수 있습니다.

- (Windows, Mac, Linux)용 **Unity Technologies Unity**(무료)
