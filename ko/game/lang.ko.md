## .LANG 파일이란?

Minecraft 언어 현지화 파일은 Minecraft 게임에서 게임 내 텍스트에 대한 언어 번역을 제공하는 데 사용되는 파일 형식입니다. 이러한 파일은 .lang 확장자를 가지며 메뉴, 업적 및 항목과 같은 게임 요소를 특정 언어로 번역된 이름에 매핑하는 키-값 쌍을 포함합니다.

Minecraft는 여러 언어를 지원하며 게임은 플레이어의 언어 설정에 따라 적절한 언어 파일을 자동으로 사용합니다. 언어 파일은 게임 디렉터리 내의 "lang" 폴더에 저장되며 플레이어가 수정하여 자신의 번역을 제공하거나 기존 번역을 수정할 수 있습니다.

언어 현지화는 전 세계 플레이어가 모국어로 게임을 즐길 수 있도록 하여 더 많은 사람들이 더 쉽게 접근하고 즐길 수 있도록 하는 Minecraft의 필수 기능입니다.

## LANG 파일 여는 방법?

Minecraft .lang 파일은 텍스트 파일이며 UTF-8 인코딩을 지원하는 텍스트 편집기를 사용하여 열 수 있습니다. .lang 파일을 여는 단계는 다음과 같습니다.

1. .lang 파일이 있는 폴더로 이동합니다. Minecraft 언어 파일의 기본 위치는 `minecraft/resourcepacks/<resource-pack> /assets/minecraft/lang/`, 여기서 `<resource-pack> `는 사용 중인 리소스 팩의 이름입니다.
2. .lang 파일을 마우스 오른쪽 버튼으로 클릭하고 상황에 맞는 메뉴에서 "연결 프로그램"을 선택합니다.
3. 프로그램 목록에서 텍스트 편집기를 선택하거나 "다른 앱 선택"을 선택하여 다른 텍스트 편집기를 찾아보세요.
4. 사용하려는 텍스트 편집기가 목록에 없으면 "앱 더 보기"를 클릭하고 "이 PC에서 다른 앱 찾기"를 선택하여 사용하려는 프로그램을 찾아보세요.
5. 텍스트 편집기를 선택한 후 "확인"을 클릭하여 파일을 엽니다.

또는 .lang 파일을 텍스트 편집기 아이콘이나 열려 있는 텍스트 편집기 창에 끌어서 놓아 열 수도 있습니다.
