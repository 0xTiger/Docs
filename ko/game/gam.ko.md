## .GAM 파일이란?
확장자가 .gam인 파일은 다양한 비디오 게임에서 게이머의 참여를 유도하여 다음에 프로그램을 열 때 중단한 부분부터 계속하도록 유도하는 데 사용됩니다. 따라서 GAM 파일은 사용자가 어떤 시간에 무엇을 했는지에 대한 정보를 저장한다는 의미입니다. 이 파일은 게임 소프트웨어에 의해 자동으로 저장되거나 게이머가 게임을 떠나는 상태를 계속하려면 저장하라는 메시지가 표시될 수 있습니다.
## GAM 파일 형식
GAM 파일 형식은 기본적으로 세이브 게임에서 게임 데이터를 저장하는 데 사용됩니다. GAM 파일은 크리쳐, 아이템 정보, 지역을 저장하지 않고 파티원과 파티원에게 영향을 미치는 일반 변수 또는 전역 변수에 대한 데이터를 저장합니다.
### GAM 파일의 구조
GAM 파일의 구조는 다음과 같습니다.
- 헤더
- 파티원
- 파티원 CRE 파일 데이터
- NPC(파티에 참여하지 않은 사람)
- NPC 킬 통계(NPC 구조체에 포함됨)
- NPC CRE 파일 데이터
- 변수(GLOBAL 네임스페이스 내)
- 저널 항목
- 친숙한 정보
- 저장 위치
- 포켓 플레인 위치
- 친숙한 엑스트라
