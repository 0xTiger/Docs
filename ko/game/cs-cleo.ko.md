## .CS 파일이란?

CLEO(CLEO Library의 약자)와 관련된 .CS 파일은 일반적으로 Grand Theft Auto(GTA) 비디오 게임 시리즈에 사용되는 사용자 정의 스크립트 파일을 나타냅니다. CLEO는 플레이어가 사용자 정의 스크립트를 생성하고 GTA 게임에 추가하여 게임 플레이를 수정하고, 새로운 기능을 추가하고, 전반적인 게임 경험을 향상시킬 수 있는 인기 있는 모딩 프레임워크입니다.

## CS 파일 개요

CLEO의 .cs 파일에 포함될 수 있는 내용에 대한 기본 개요는 다음과 같습니다.

1. **스크립트 코드**: .cs 파일에는 CLEO 스크립트 언어로 작성된 스크립트 코드가 포함되어 있습니다. 이 스크립팅 언어는 CLEO에만 적용되며 게임 내에서 사용자 정의 스크립트의 동작을 정의하는 데 사용됩니다. 코드는 텍스트 편집기를 사용하여 작성할 수 있으며 일반적으로 특정 구문을 따릅니다.
    









2. **수정**: CLEO 스크립트는 게임 내 개체의 동작 변경, 맞춤형 미션 생성, 새로운 차량, 무기 등 추가 등 게임을 다양하게 수정할 수 있습니다. 가능성은 광범위하며 스크립트 작성자의 창의성과 프로그래밍 기술에 따라 달라집니다.
    









3. **트리거**: CLEO 스크립트에는 사용자 정의 스크립트를 실행해야 하는 시기와 방법을 결정하는 트리거가 포함되는 경우가 많습니다. 이러한 트리거는 게임 내 이벤트, 플레이어 작업 또는 특정 조건을 기반으로 할 수 있습니다.
    









4. **변수 및 함수**: CLEO 스크립트는 변수를 사용하여 데이터를 저장하고 조작할 수 있을 뿐만 아니라 함수를 사용하여 코드를 캡슐화하고 재사용할 수 있습니다. 이러한 변수와 함수는 스크립트의 동작을 제어하는 데 사용됩니다.

## CS 파일의 예

다음은 게임의 날씨를 변경하는 CLEO .cs 스크립트의 간단한 예입니다.

```
{$CLEO .cs}

03A4: name_thread 'WEATHER'

:WEATHER_LOOP
    // Change the weather to sunny
    0085: 0@ = 11 // Weather ID for sunny weather
    00D8: mission_cleanup
    0051: return 0@ // Exit the script

// Add more code and conditions as needed
```

## 클레오 도서관

간단히 "CLEO"라고도 불리는 **CLEO 라이브러리**는 Grand Theft Auto(GTA) 비디오 게임 시리즈를 위한 인기 있고 강력한 모딩 프레임워크입니다. 이를 통해 플레이어와 모더는 사용자 정의 스크립트를 생성하고 GTA 게임에 추가하여 게임 플레이를 수정하고 새로운 기능을 추가하며 전반적인 게임 경험을 향상시킬 수 있습니다. CLEO는 특히 GTA 모딩 커뮤니티에서 유연성과 사용 용이성으로 잘 알려져 있습니다.

CLEO Library의 주요 기능과 측면은 다음과 같습니다.

1. **스크립팅 언어**: CLEO는 모딩 프레임워크에 특화된 스크립팅 언어를 도입합니다. 스크립팅 언어는 상대적으로 이해하기 쉽고 초보자와 숙련된 모더 모두가 접근할 수 있도록 설계되었습니다.
    









2. **사용자 정의 스크립트**: CLEO를 사용하면 게임 세계 내에서 다양한 기능을 수행할 수 있는 사용자 정의 스크립트를 만들 수 있습니다. 이러한 스크립트는 게임 내 동작을 변경하고, 새로운 임무를 추가하고, 새로운 차량이나 무기를 도입하고, 게임 물리학을 변경하는 등의 작업을 수행할 수 있습니다.
    









3. **트리거 및 이벤트**: CLEO 스크립트는 다양한 게임 내 이벤트, 플레이어 동작 또는 특정 조건에 의해 트리거될 수 있습니다. 이를 통해 모더는 게임 내에서 동적이고 대화형 콘텐츠를 만들 수 있습니다.
    









4. **여러 GTA 버전 지원**: CLEO는 GTA III, GTA Vice City, GTA San Andreas, GTA IV 등을 포함한 다양한 GTA 게임에 맞춰진 버전을 제공합니다. 이는 모더가 다양한 GTA 타이틀에 대한 사용자 정의 스크립트를 만들고 공유할 수 있음을 의미합니다.

## CLEO 라이브러리에서 사용하는 파일 형식

Grand Theft Auto(GTA) 게임용 CLEO 모딩에서는 모드를 생성하고 설치하는 데 일반적으로 여러 파일 형식이 사용됩니다. 이러한 파일 형식은 실제 스크립트를 포함하는 것부터 텍스처, 모델 또는 오디오와 같은 추가 리소스를 저장하는 것까지 다양한 용도로 사용됩니다. CLEO 모딩에 사용되는 주요 파일 형식은 다음과 같습니다.

1. **.cs (Custom Script)**: CLEO .cs 파일은 CLEO 스크립팅 언어로 작성된 사용자 정의 스크립트 파일입니다. 이 파일에는 모드의 동작과 기능을 정의하는 코드가 포함되어 있습니다. .cs 파일은 CLEO 모딩의 핵심이며 게임에서 실행되어 사용자 정의 기능을 구현합니다.
    









2. **.csa(사용자 정의 스크립트 아카이브)**: .csa 파일은 여러 .cs 스크립트 파일을 저장할 수 있는 아카이브입니다. 보다 쉬운 설치 및 관리를 위해 관련 스크립트를 함께 패키지하는 데 자주 사용됩니다.
    









3. **.fxt(텍스트 파일)**: .fxt 파일은 CLEO 모드에 대한 텍스트 번역을 현지화하거나 제공하는 데 사용할 수 있는 텍스트 파일입니다. 여기에는 게임에 표시할 수 있는 텍스트 문자열이 포함되어 있어 플레이어가 다양한 언어로 모드에 액세스할 수 있습니다.
    









4. **[.bmp](/ko/image/bmp/), [.png](/ko/image/png/), [.jpg](/ko/image/jpeg/)(이미지 형식)**: 이러한 이미지 형식은 모드용 텍스처와 이미지를 저장하는 데 사용됩니다. 이는 맞춤형 스킨, 차량 텍스처, HUD 요소 등에 사용할 수 있습니다. 게임에 따라 선호되는 이미지 형식이 다를 수 있습니다.

## CS 파일을 여는 방법은 무엇입니까?

CLEO .cs(사용자 정의 스크립트) 파일의 내용을 열고 보려면 원하는 텍스트 편집기나 코드 편집기를 사용할 수 있습니다. 일반적인 선택 사항은 다음과 같습니다.

- **메모장**: Windows에 사전 설치되어 제공되는 간단한 텍스트 편집기입니다.
- **메모장++**: 코딩 및 스크립팅을 위해 설계된 더욱 풍부한 기능을 갖춘 텍스트 편집기입니다.
- **Visual Studio Code**: 인기 있고 확장성이 뛰어난 무료 코드 편집기입니다.
- **Sublime Text**: 속도와 다양성으로 유명한 또 다른 코드 편집기입니다.
- **Atom**: GitHub에서 개발한 오픈 소스 코드 편집기입니다.
