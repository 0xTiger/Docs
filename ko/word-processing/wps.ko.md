## .WPS 파일이란?

확장자가 .wps인 파일은 2006년 이전에 Word 문서로 사용되었던 Microsoft Works 문서입니다. 당시 새로 도입된 [DOC](/ko/word-processing/doc/) 파일 형식과 비슷했지만 새 파일 형식만큼 기능이 많지는 않았습니다. 예를 들어 매크로 기능과 고급 서식 옵션을 지원하지 않았습니다. WPS 파일은 최신 워드 프로세서로 열 수 있지만 Windows 10의 경우 이러한 파일을 로드하려면 변환기를 설치해야 합니다. Microsoft Works는 2010년에 Microsoft Word 문서 프로세서로 대체되었습니다.

## 파일 형식 사양

Microsoft가 최종 사용자에게 사양을 공개하지 않았기 때문에 WPS 문서의 파일 형식 사양은 공개적으로 사용할 수 없습니다. WPS는 이진 파일이며 Microsoft에서 워드 프로세서용으로 만든 파일 형식 변환 필터로 열 수 있습니다. LibreOffice, NeoOffice 및 OxygenOffice에는 다양한 Microsoft Works 버전에서 텍스트를 추출할 수 있는 일반 C++ 라이브러리인 libwps가 포함되어 있습니다.
