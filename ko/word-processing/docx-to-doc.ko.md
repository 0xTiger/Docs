## DOCX에서 DOC로의 변환 정보

DOC 파일 형식은 2007 및 이전 버전의 Microsoft Word에서 사용되는 인기 있는 문서 파일 형식이었습니다. Microsoft는 Office 2007 출시와 함께 Office Open XML 사양을 기반으로 하는 DOCX 파일 형식을 도입했습니다. 압축된 XML 파일 형식으로 저장되는 DOCX와 비교하여 DOC 파일은 바이너리 파일로 저장되었습니다. [DOCX](/ko/word-processing/docx/)를 [DOC](/ko/word-processing/doc/) 파일로 변환하는 데 사용할 수 있는 도구가 온라인에 많이 있습니다.

## DOCX에서 DOC로 변환 - 무료 데모

무료 온라인 앱인 Aspose.app을 사용하여 DOCX를 DOC 파일 형식으로 변환할 수 있습니다. DOCX 파일을 DOC, PDF, TXT, RTF 등과 같은 파일 형식으로 로드하고 보고 변환할 수 있는 앱 세트입니다. Aspose 앱은 다양한 파일 형식에 대해 다양한 작업을 수행할 수 있는 백엔드의 Aspose API에 의해 구동되고 실행됩니다. 다음 버튼을 클릭하여 지금 Aspose.Words 온라인 앱을 사용해 보세요.

[![DOCX에서 DOC로](../docx-to-doc.png?width=120px&height=60px)](https://products.aspose.app/words/conversion/docx-to-doc)

### 온라인에서 DOCX를 DOC로 변환하는 단계

[Aspose.Words DOCX to DOC 앱](https://products.aspose.app/words/conversion/docx-to-doc)을 사용하여 다음과 같이 DOCX를 DOC 파일 형식으로 변환할 수 있습니다.

1. [DOCX to DOC 변환기](https://products.aspose.app/words/conversion/docx-to-doc)를 위한 Aspose.Words 앱으로 이동합니다.
1. 업로드 버튼을 클릭하여 DOCX 파일을 업로드하세요.
1. 완료되면 변환 버튼을 누릅니다. 그러면 백그라운드에서 DOCX 파일을 DOC로 변환하는 프로세스가 시작됩니다.
1. DOCX에서 DOC로의 변환 프로세스가 완료되면 변환된 RTF 파일을 다운로드하라는 메시지가 표시됩니다.

## Aspose.Words API

[Aspose.Words](https://products.aspose.com/words/)는 모든 개발자가 [DOCX](/ko/word-processing/docx/)를 [DOC](/ko/word-processing/doc/)로 변환할 수 있는 API 세트입니다. 간단한 코드 라인을 사용하여 형식을 지정합니다. [C#](/ko/programming/cs/), [Java](/ko/programming/java/), [Python](/ko/programming/py/) 및 [C++](/ko/programming/cpp/). 를 포함한 다양한 프로그래밍 언어에서 사용할 수 있습니다. DOC 외에도 Aspose.Words API를 사용하면 DOCX 파일을 다른 널리 사용되는 파일 형식으로 변환할 수 있습니다.
