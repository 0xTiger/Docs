## .FODT 파일이란?

확장자가 .fodt인 파일은 OpenDocument 플랫 [XML](/ko/web/xml/) 파일 형식으로 저장된 텍스트 문서입니다. [.zip](/ko/compression/zip/) 컨테이너 파일 형식으로 저장되는 .odt 파일과 달리 FODT 파일은 일반 XML 문서로 보기 위해 모든 텍스트 편집기에서 열 수 있는 사람이 읽을 수 있는 파일로 디스크에 저장됩니다. . OpenDocument 오피스 문서 표준은 OASIS 산업 컨소시엄에서 개발하여 누구나 제한 없이 누구나 구현할 수 있는 방법을 제공합니다. 다른 인기 있는 OpenDocument 파일 형식에는 [.odt](/ko/word-processing/odt/), [.ott](/ko/word-processing/ott/), [.ods](/ko/spreadsheet/ods/), [.odp]가 있습니다. (/ko/presentation/odp/) 및 유사하게 기타.

## FODT 파일 형식 - 추가 정보

FODT 파일은 일반 XML 파일 형식(Flat XML 또는 Uncompressed XML이라고도 함)으로 저장되지만 OpenDocument에서 일반적으로 사용되는 문서 형식은 아닙니다. OpenDocument 파일은 일반적으로 여러 파일과 디렉터리로 구성된 ZIP 패키지로 저장됩니다. XML 기반 OpenDocument 파일은 일부 소프트웨어에서 지원되지 않을 수 있으므로 ZIP 패키지 기반 파일에 비해 사용 빈도가 낮습니다.
