## DOC에서 BMP로의 변환 정보

2007 이전 버전의 Microsoft Word 사용자라면 그때까지 Microsoft Word에서 문서를 저장하기 위해 사용하는 기본 파일 형식인 DOC 파일 형식을 잘 알고 있어야 합니다. Microsoft Word 시작부터 사용되어 왔으며 최신 DOC 파일 형식으로 교체된 후에도 꽤 오랫동안 그대로 유지되었습니다. 콘텐츠 관리 시스템에는 여전히 BMP 이미지로 변환할 수 있는 백업용 DOC 파일이 있을 수 있습니다. 변환용 Aspose.Word 앱과 같은 온라인 도구를 사용하면 [DOC](/ko/word-processing/doc/) 파일을 [BMP](/ko/image/bmp/) 이미지로 변환할 수 있습니다.

## DOC에서 BMP로 변환 - 무료 데모

Aspose.Words 앱은 Aspose의 인기 있는 프로그래밍 API를 기반으로 개발된 온라인 도구 세트입니다. 문서 보기, 변환, 압축, 회전, 분할, 병합 등과 같은 기능을 제공합니다. 웹 브라우저에서 바로 Aspose.Words 앱을 사용하여 [DOC](/ko/word-processing/doc/)를 [BMP](/ko/image/bmp/)로 변환할 수 있습니다. 변환을 수행하기 위해 소프트웨어를 설치할 필요가 없습니다.

### 온라인에서 DOC를 BMP로 변환하는 단계

다음과 같이 [Aspose.Words DOC to BMP 앱](https://products.aspose.app/words/conversion/doc-to-bmp)을 사용하여 DOC를 BMP 파일 형식으로 변환할 수 있습니다.

1. [DOC-BMP 변환기](https://products.aspose.app/words/conversion/doc-to-bmp)를 위한 Aspose.Words 앱으로 이동합니다.
1. 업로드 버튼을 클릭하여 DOC 파일을 업로드하세요.
1. 완료되면 변환 버튼을 누릅니다. 그러면 백그라운드에서 DOC 파일을 BMP로 변환하는 프로세스가 시작됩니다.
1. DOC에서 BMP로의 변환 프로세스가 완료되면 변환된 BMP 파일을 다운로드하라는 메시지가 표시됩니다.
