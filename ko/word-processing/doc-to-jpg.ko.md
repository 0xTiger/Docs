## DOC에서 JPG로 변환 정보

이전 버전의 Microsoft Word(예: Microsoft Word 2007 이전 버전)로 작업한 적이 있다면 DOC 파일 형식에 익숙해야 합니다. 이는 Microsoft Word 2007에 DOCX 파일 형식이 도입되기 전에 Microsoft Word에서 사용된 기본 파일 형식이었습니다. DOC에서 JPG로의 버전은 매우 일반적인 요구 사항이며, 특히 Microsoft에서 Word 문서를 JPG 이미지로 내보내는 것을 허용하지 않는 경우 더욱 그렇습니다. 또한 최신 버전의 Microsoft Word는 이전 DOC 파일과 완벽하게 호환되지 않습니다. Aspose.Words 앱과 같은 온라인 앱을 사용하여 [DOC](/ko/word-processing/doc/)를 [JPG](/ko/image/jpeg/) 파일 형식으로 변환할 수 있습니다.

## DOC를 JPG로 변환 - 무료 데모

Aspose.Words 앱은 DOC 파일을 여러 다른 파일 형식으로 변환할 수 있는 온라인 도구 세트입니다. 웹 브라우저에서 몇 번의 클릭만으로 DOC 파일을 JPG로 변환할 수 있습니다. Aspose.Words 앱을 사용하면 코딩이 필요 없고 빠르고 정확하며 가장 중요한 것은 완전 무료라는 장점이 있습니다.

DOC를 JPG로 변환하는 과정에서 문서의 각 페이지가 JPG 이미지로 변환되어 다운로드가 가능합니다. 출력은 디스크에 저장되는 JPG 이미지 세트입니다.

### DOC를 JPG로 변환하는 방법은 무엇입니까?

그렇다면 DOC를 JPG로 어떻게 변환합니까? 다음과 같이 [Aspose.Words DOC to JPG 앱](https://products.aspose.app/words/conversion/doc-to-jpg)을 사용하여 DOC를 JPG 파일 형식으로 변환할 수 있습니다.

1. [DOC-JPG 변환기](https://products.aspose.app/words/conversion/doc-to-jpg)를 위한 Aspose.Words 앱으로 이동합니다.
1. 업로드 버튼을 클릭하여 DOC 파일을 업로드하세요.
1. 완료되면 변환 버튼을 누릅니다. 그러면 백그라운드에서 DOC 파일을 JPG로 변환하는 프로세스가 시작됩니다.
1. DOC에서 JPG로의 변환 프로세스가 완료되면 변환된 JPG 파일을 다운로드하라는 메시지가 표시됩니다.
