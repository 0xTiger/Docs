## DOCX에서 RTF로의 변환 정보

Microsoft Windows의 초기 사용자라면 RTF(Rich Text Format) 파일 형식을 알아야 합니다. Microsoft는 Windows 3.0에서 [RTF 파일 형식](/ko/word-processing/rtf/)을 도입했습니다. RTF 파일은 응용 프로그램 내에서 사용할 서식이 지정된 텍스트와 그래픽을 인코딩하는 방법을 나타냅니다. RTF 파일에는 굵게, 기울임꼴, 밑줄과 같은 텍스트 서식을 유지하는 것 외에도 텍스트와 이미지가 포함될 수 있습니다. 가장 흥미로운 점은 RTF가 상호 운용 가능한 파일 형식으로 채택되었으며 크로스 플랫폼이라는 것입니다. 이러한 이유로 DOCX에서 RTF로의 변환은 편집 목적으로 다양한 OS에서 문서를 공유하는 동안 널리 사용되는 선택입니다.
