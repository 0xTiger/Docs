## DOC에서 DOCM으로의 변환 정보

DOCM은 매크로를 실행할 수 있는 Microsoft Word 문서의 매크로 지원 버전입니다. 일반 Word 문서와 매크로 지원 문서의 차이점은 DOCM이 매크로 지원을 위한 추가 기능을 제공한다는 것입니다.

무료 온라인 도구와 앱을 사용하여 DOC 파일을 DOCM으로 변환할 수 있습니다. **[Aspose.Words 앱](https://products.aspose.app/words/family)**은 웹 브라우저에서 바로 Word 문서에 대한 다양한 작업을 수행할 수 있는 온라인 앱 세트입니다. [문서 보기](/ko/word-processing/docx-viewer/), 분할, 병합, 비교, 회전 등의 기능 외에도 Word 문서를 한 형식에서 다른 형식으로 변환할 수 있습니다. Aspose.Words 앱을 사용하여 [DOC](/ko/word-processing/doc/)를 [DOCM](/ko/word-processing/docm/)으로 변환할 수 있습니다.

## DOC를 DOCM으로 변환 - 무료 데모

Aspose.Words 앱을 사용하면 온라인에서 무료로 DOC를 DOCM으로 변환할 수 있습니다. 웹 브라우저 내에서 DOC를 DOCM으로 변환하는 방법에 액세스할 수 있습니다. 빠르고 정확하며 코딩이 필요하지 않으며 완전 무료입니다. 게다가 DOC를 DOCM으로 변환하기 위해 PC에 추가 소프트웨어나 도구를 설치할 필요가 없습니다.

### DOC를 DOCM으로 변환하는 방법은 무엇입니까?

다음과 같이 [Aspose.Words DOC to DOCM 앱](https://products.aspose.app/words/conversion/doc-to-docm)을 사용하여 DOC를 DOCM 파일 형식으로 변환할 수 있습니다.

1. [DOC-DOCM 변환기](https://products.aspose.app/words/conversion/doc-to-docm)를 위한 Aspose.Words 앱으로 이동합니다.
1. 업로드 버튼을 클릭하여 DOC 파일을 업로드하세요.
1. 완료되면 변환 버튼을 누릅니다. 그러면 백그라운드에서 DOC 파일을 DOCM으로 변환하는 프로세스가 시작됩니다.
1. DOC에서 DOCM으로의 변환 프로세스가 완료되면 변환된 DOCM 파일을 다운로드하라는 메시지가 표시됩니다.

## Aspose.Words API

[![Aspose.Words](../try-aspose-words.png)](https://products.aspose.com/words/)

Aspose.Words API 제품군을 사용하면 개발자는 단 몇 줄의 코드만으로 [DOC](/ko/word-processing/doc/)를 [DOCM](/ko/word-processing/docm/) 형식으로 변환할 수 있습니다. C#, Java, Python 및 C++를 포함한 다양한 프로그래밍 언어에서 사용할 수 있습니다. DOCM 외에도 Aspose.Words API를 사용하면 DOC 파일을 다른 널리 사용되는 파일 형식으로 변환할 수 있습니다.

## DOC 뷰어

DOC 파일을 열고 볼 수 있는 Microsoft Word 또는 기타 호환 소프트웨어가 PC에 설치되어 있지 않은 경우 무료로 제공되는 앱 [Aspose.Words Viewer](/ko/word-processing/docx-viewer/)를 사용하여 열고 볼 수 있습니다. DOC 파일을 온라인으로 볼 수 있습니다.
