## .OF 파일이란?

확장자가 .oft인 파일은 Microsoft Outlook을 사용하여 만든 템플릿 파일입니다. 메시지 템플릿에 대해 미리 형식이 지정된 레이아웃 세트는 시간을 절약하기 위해 공통 정보가 포함된 이메일을 보내는 데 사용됩니다. 이러한 파일은 새 전자 메일을 만들고 필요한 정보를 추가한 다음 Microsoft Outlook에서 다른 이름으로 Office 템플릿(.oft)으로 저장 드롭다운을 사용하여 생성할 수 있습니다. 사용자는 OFT 파일을 두 번 클릭하여 열 수 있으며 해당 특정 시스템의 관련 응용 프로그램에서 열립니다.

## OFT 파일 구조 ##

.OFT 파일 형식은 기본으로 [MSG](/ko/email/msg/) 파일 형식을 사용합니다. 유일한 차이점은 OFT 파일은 CLSID_TemplateMessage({0006F046-0000-0000-C000-000000000046})를 스토리지 클래스(WriteClassStg)로 갖는 반면 MSG 파일은 CLSID_MailMessage({00020D0B-00000-0)를 사용한다는 점입니다(WriteClassStg).0000C00

CFB_3 형식은 MSG 파일의 기본입니다. 패러다임은 디렉토리 및 파일에 매우 가까운 저장소 및 스트림 개념을 기반으로 합니다. 따라서 전자의 주요 차이점은 복합 파일이라고 하는 별개의 파일로 패키징된 전체 계층 구조입니다. 개체는 메시지 파일을 구성하고 단일 속성 또는 해당 컬렉션으로 구성됩니다. 이 기능을 통해 응용 프로그램은 복잡하고 구조화된 데이터를 단일 파일에 저장할 수 있습니다. 이 형식은 또한 여러 저장소를 지정하며 각 저장소는 주요 구성 요소로 Message 개체를 나타냅니다. 이러한 저장소에는 해당 구성 요소의 속성을 나타내는 여러 스트림이 포함됩니다. 스토리지의 중첩도 가능합니다.

### OFT 속성

.msg 파일의 최상위 수준에서 저장소에는 속성을 저장하는 스트림이 포함됩니다. 속성은 다음과 같이 분류할 수 있습니다.

* 고정 길이 속성
* 가변 길이 속성
* 다중 값 속성

카테고리에 관계없이 속성은 태그가 지정되거나 이름이 지정됩니다. 그러나 매핑 저장소에 지정된 대로 명명된 속성에 대해 적절한 매핑 정보가 필요합니다.

### 최상위 구조

메시지 개체는 Msg 파일 형식의 전체 최상위 수준을 나타냅니다. 유형, 속성, 수신자 및 첨부 파일 개체 수에 따라 메시지 개체는 해당 .MSG 파일에 다른 스트림 저장소를 가질 수 있습니다.

#### 다른 구조와의 관계

MSG/OFT 파일 형식은 다른 구조와 다음과 같은 관계를 갖습니다.

* .msg의 기본은 복합 파일 바이너리 파일 형식입니다.
* Message 및 Attachment Object Protocol에서 사용하는 속성은 이 형식에서 사용됩니다.
