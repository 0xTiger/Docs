## .ACCDE 파일이란?

확장자가 .accde인 파일은 사용자 지정 데이터베이스 코드를 보호하기 위해 실행 전용 데이터베이스 파일을 만들기 위한 Microsoft Access 2007 파일 형식입니다. 모든 VBA 코드를 실행 파일로만 컴파일하여 보거나 편집할 수 없습니다. ACCDE 파일 형식은 이전 버전의 Microsoft Access에서 사용되었던 MDE 파일 형식을 대체합니다. ACCDE는 실행 전용 모드의 [ACCDB](/ko/database/accdb/) 파일이며 소스 코드가 없기 때문에 더 간결합니다. 이러한 파일의 크기가 작으면 메모리 사용량이 향상되고 응용 프로그램 성능이 향상됩니다. ACCDE 파일은 Microsoft Access 365로 열 수 있습니다.

## ACCDE 파일 형식

모든 Microsoft Access 2007+ 파일은 ACCDB 파일 형식 제품군을 사용하는 Jet ACE(Access 데이터베이스 엔진)를 사용합니다. 그러나 Microsoft는 오픈 소스 구현이나 자세한 기술 사양을 제공하지 않습니다.
