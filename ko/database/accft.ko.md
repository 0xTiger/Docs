## .ACCFT 파일이란?

확장자가 .accft인 파일은 데이터베이스 테이블에 대한 필드 데이터 형식의 정의를 포함하는 Microsoft Access 템플릿 파일입니다. Microsoft Access는 테이블 기반 데이터베이스, 양식, 쿼리 및 보기를 만드는 데 사용되는 데스크톱 관계형 데이터베이스 관리 시스템(RDBMS)입니다. ACCFT 파일을 사용하여 테이블 필드의 데이터 유형을 정의할 수 있으며 이 템플릿 파일을 시작점으로 사용하여 데이터베이스 테이블을 생성할 수 있습니다. 새 테이블을 만들기 위해 Microsoft Access 2016에서 열 수 있습니다. ACCFT 파일은 MIME 유형 **application/msaccess**로 인식됩니다.

## ACCFT 파일 형식

ACCFT 파일은 파일 형식 사양이 공개적으로 제공되지 않는 이진 파일입니다. 헤더 정보, 파일 구조, 바닥글 등과 같은 내부 세부 정보는 현재 사용할 수 없습니다.
