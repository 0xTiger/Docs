## .ABS 파일이란?

.abs 확장자를 가진 파일은 Delphi 소프트웨어 개발 IDE용 데이터베이스 엔진인 Absolute 데이터베이스에서 생성하고 사용하는 데이터베이스입니다. 이는 Borland 데이터베이스 엔진(DBE)을 대체했으며 더 작고, 빠르며, 강력하고 사용하기 쉽습니다. ABS 데이터베이스는 애플리케이션 EXE 파일의 일부로 컴파일되어 애플리케이션을 더 빠르고 작게 만듭니다. 데이터는 구조화된 관계형 형식으로 ABS 파일에 저장됩니다.

## ABS 파일 형식 - 추가 정보

ABS 파일은 일반적으로 바이너리 파일로 저장되는 응용 프로그램의 [EXE 파일](/ko/executable/exe/)의 일부로 저장됩니다. 바이너리 파일은 텍스트 편집기에서 열 수 없으며 데이터베이스 파일의 내용에 대한 빠른 액세스를 제공합니다.

### ABS 파일 형식의 주요 특징

ABS 파일은 매우 간단하며 애플리케이션의 실행 파일에 포함됩니다. 다음과 같은 기능이 있습니다.

* BDE 없음; DLL 없음
* 단일 파일 데이터베이스
* SQL'92(DDL 및 DML) 지원
* 표준 및 타사 데이터베이스 컨트롤과 호환 가능
* 단일 사용자 및 다중 사용자 모드(파일 서버)
* 98부터 최신 버전까지 모든 Windows 버전에서 훌륭하게 작동하며 업데이트나 서비스 팩이 필요하지 않습니다.
* 초고속 인메모리 테이블
* 비교할 수 없는 사용 편의성
* 강력한 암호화
* BLOB 압축
* 개인적인 용도로는 무료
* 전체 소스 코드 사용 가능
* 로열티 프리 배포
