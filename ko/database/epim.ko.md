## .EPIM 파일이란?

.epim 파일 확장자는 사용자가 연락처, 약속, 작업 및 메모를 구성하고 관리할 수 있는 개인 정보 관리자 소프트웨어인 EssentialPIM에서 사용됩니다. .epim 파일은 개인 정보, 일정 이벤트 및 작업 목록을 포함하여 프로그램에 입력된 모든 데이터를 저장하는 데이터베이스 파일입니다. 이 파일은 다른 EssentialPIM 설치로 백업, 내보내기 및 가져올 수 있습니다. 이 소프트웨어는 Windows, Android 및 iOS 플랫폼에서 사용할 수 있습니다.

.epim 파일은 EssentialPIM에서 사용하는 독점 형식입니다. .epim 파일은 EssentialPIM 소프트웨어에만 해당되며 다른 소프트웨어로 열거나 편집할 수 없다는 점에 유의해야 합니다.

## EPIM 파일을 만드는 방법?

EssentialPIM에서 .epim 파일을 생성하려면 먼저 컴퓨터에 소프트웨어를 설치해야 합니다.

1. EssentialPIM을 엽니다.
2. "파일" 메뉴를 클릭하고 "새 데이터베이스"를 선택합니다.
3. 새 데이터베이스 파일의 위치와 이름을 선택하라는 대화 상자가 나타납니다.
4. 이름을 입력하고 파일 위치를 선택한 다음 "저장"을 클릭합니다.
5. .epim 파일 확장자로 새 데이터베이스 파일이 생성되고 연락처, 약속, 작업 및 메모와 같은 데이터 추가를 시작할 수 있습니다.

또는 "파일" -> "가져오기"를 사용하여 다양한 형식에서 데이터를 가져오고 새 .epim 파일을 만들 수도 있습니다. 게다가 이미 데이터베이스 파일이 있고 이 파일에서 새 파일을 생성하려는 경우 "파일" -> "내보내기" 옵션을 사용하여 현재 .epim 파일에서 모든 데이터를 내보내고 새 파일로 가져올 수 있습니다. 하나.

## EPIM 파일의 위치

EssentialPIM에서 .epim 파일의 위치는 사용 중인 운영 체제에 따라 다릅니다.

- Windows에서 .epim 파일의 기본 위치는 "내 문서" 폴더에 있지만 새 데이터베이스 파일을 만들 때 다른 위치를 선택할 수 있습니다.
- Android에서 .epim 파일의 기본 위치는 "다운로드" 폴더에 있지만 새 데이터베이스 파일을 만들 때 다른 위치를 선택할 수 있습니다.
- iOS에서 .epim 파일의 기본 위치는 "EssentialPIM" 폴더이지만 새 데이터베이스 파일을 생성할 때 다른 위치를 선택할 수 있습니다.

"파일" -> "데이터베이스 열기" 옵션을 사용하여 컴퓨터나 장치의 다른 위치에 있는 기존 .epim 파일을 열 수도 있습니다.
