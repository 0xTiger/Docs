## .ADB 파일이란?

확장자가 .adb인 파일은 [Alpha Five](https://www.alphasoftware.com/)(현재 Alpha Anywhere로 알려짐)로 생성된 데이터베이스 파일입니다. 여기에는 프로젝트 작업 공간의 구성원인 모든 테이블 및 세트의 파일 이름이 포함됩니다. 또한 작업 공간에 통합된 모든 라이브러리의 이름이 포함됩니다. Alpha Anywhere는 웹 애플리케이션을 개발하는 데 사용되는 IDE입니다. ADB 파일 외에도 Alpha Anywhere의 작업 공간에는 테이블 또는 세트 이름을 시작하지만 파일 확장자가 다른 많은 파일이 포함될 수 있습니다.

## ADB 파일 형식 - 추가 정보

.adb 파일의 내부 파일 형식 정보는 개발자 참조용으로 제공되지 않습니다.
