## .ADP 파일이란?

확장명이 .adp인 파일은 OLE DB 구성 요소 아키텍처를 사용하여 Microsoft SQL Server 데이터베이스에 대한 직접적이고 효율적인 연결을 제공하는 Microsoft Access 프로젝트 파일입니다. 이러한 파일에는 실제 테이블, 데이터베이스 다이어그램 또는 기타 데이터베이스 요소가 포함되어 있지 않습니다. ADP 파일은 Access 2007 및 2010으로 만들 수 있습니다. 이전 버전의 Access로 만든 파일은 Access 2007 및 2010에서도 열 수 있습니다. ADP 파일은 Microsoft Access 365를 사용하여 열 수 있습니다.

## ADP 파일 형식 - 추가 정보

ADP로 만든 Access 프로젝트를 사용하면 테이블 및 보기와 같은 SQL Server 개체의 디자인을 변경할 수 있습니다. 또한 데이터베이스 다이어그램, 저장 프로시저 및 사용자 정의 함수와 같은 SQL 기능을 생성, 편집 및 활용할 수 있습니다. 이렇게 하면 디자인을 변경할 수 없고 SQL Server 테이블 및 보기에만 연결할 수 있는 SQL Server 데이터베이스에 연결하는 것보다 이점이 있습니다.

### 데이터 유형 및 다이어그램 지원

**날짜/시간 데이터 유형**

Access 2010에서는 다음과 같은 새로운 날짜/시간 데이터 형식이 지원됩니다.

* 시각
* 데이트
* DATETIME2
* DATETIMEOFFSET

**가변 길이 데이터 유형**

다음 가변 길이 데이터 형식은 Access 2010 프로젝트에서 사용할 수 있습니다.

* 바빈(MAX)
* VARCHAR(최대)
* NVARCHAR(최대)
