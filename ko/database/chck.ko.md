## .CHCK 파일이란?

CHCK 파일은 Microsoft Exchange Server의 상태 모니터링 검사 결과로 생성됩니다. 서버 데이터베이스의 전반적인 상태 및 운영에 대한 정보가 포함되어 있습니다. Microsoft Exchange Server는 주로 이메일 서버로 사용되며 비즈니스 연락처와 일정을 유지 관리할 수 있습니다. Exchange 서버 관리자는 EseUtil을 사용하여 데이터베이스 상태를 확인합니다.

## CHCK 파일 형식

CHCK 파일의 내부 파일 형식은 알려져 있지 않으며 CHCK 파일 읽기/쓰기에 사용할 수 있는 참조 자료도 없습니다.
