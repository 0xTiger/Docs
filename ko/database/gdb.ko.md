## .GDB 파일이란?

ESRI 파일 지리 데이터베이스(FileGDB)는 피쳐 데이터 세트, 피쳐 클래스 및 관련 테이블과 같은 관련 지리 공간 데이터를 보유하는 디스크의 폴더에 있는 파일 모음입니다. 작동하려면 동일한 디렉토리에 .gdb 파일과 함께 특정 다른 파일을 보관해야 합니다. 공간 및 비공간 데이터를 관리하기 위해 .gdb 파일에서 쿼리를 실행할 수 있습니다.

## GDB 파일 형식 - 추가 정보

파일 지오데이터베이스는 7개의 시스템 테이블과 사용자 데이터로 구성됩니다. 사용자 데이터는 다음 유형의 데이터세트에 저장할 수 있습니다.

* 피쳐 클래스
* 기능 데이터 세트
* 모자이크 데이터 세트
* 래스터 카탈로그
* 래스터 데이터세트
* 도식 데이터 세트
* 테이블(비공간)
* 도구 상자

피처 데이터세트에는 피처 클래스와 다음 유형의 데이터세트가 포함될 수 있습니다.

* 첨부 파일
* 기능 연결 주석
* 기하학적 네트워크
* 네트워크 데이터 세트
* 소포 직물
* 관계 수업
* 지형
* 토폴로지

파일 지오데이터베이스의 기본 최대 데이터세트 크기는 1TB입니다. 대용량 데이터 세트(일반적으로 래스터)의 경우 최대 크기를 256TB로 늘릴 수 있습니다. 이는 구성 키워드로 제어됩니다. 자세한 내용은 파일 지오데이터베이스에 대한 구성 키워드를 참조하세요.

파일 지오데이터베이스는 하위 유형과 도메인을 포함할 수 있으며 체크아웃/체크인 복제 및 단방향 복제에 참여할 수 있습니다.

파일 지오데이터베이스는 여러 사용자가 동시에 액세스할 수 있습니다. 사용자가 편집 중인 경우 다른 데이터 세트를 편집해야 합니다.

## GDB 파일 형식 사양 ##

파일 GDB는 ESRI 독점 형식이며 해당 사양은 공개적으로 사용할 수 없습니다. 이러한 이유로 FIle GDB에 대한 파일 형식 세부 정보는 리버스 엔지니어링에 의해 [일부](https://github.com/rouault/dump_gdbtable/wiki/FGDB-Spec) 수행한 일부 소스 이외의 다른 곳에서는 문서화할 수 없습니다. .
