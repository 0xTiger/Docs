## 웹록이란?

WEBLOC 파일은 인터넷의 웹 URL을 가리키는 macOS의 바로 가기 파일입니다. 브라우저의 주소 표시줄에서 웹 URL을 데스크탑이나 저장해야 하는 다른 폴더로 드래그하여 생성됩니다. 대상 사이트의 URL은 이 바로 가기 파일에 저장되며 Microsoft 메모장, 메모장++ 또는 Apple TextEdit과 같은 텍스트 편집기에서 파일을 열어 볼 수 있습니다. WEBLOC 파일은 [URL](/ko/web/url/) 파일과 유사합니다.

## WEBLOC 파일 형식 - 추가 정보

WEBLOC 파일은 일반 텍스트 파일 형식으로 저장됩니다. Safari, Google Chrome 또는 Firefox와 같은 웹 브라우저에서 URL을 끌어 macOS에서 생성할 수 있습니다. 여러 개의 WEBLOC 파일을 동시에 열려면 외부 응용 프로그램이나 API를 통해 파일을 여는 보다 빠르고 쉬운 솔루션이 필요합니다. 이러한 API는 WEBLOC을 읽고 PDF 및 JPEG로 변환할 수 있습니다.
