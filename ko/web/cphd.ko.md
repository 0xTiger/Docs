## .CPHD 파일이란?

CPHD 파일은 CloudChan 전처리기 엔진에서 사용되는 하이퍼텍스트 문서입니다. 여기에는 동적 웹 페이지와 문서를 생성하는 HTML과 스크립트가 포함되어 있습니다. CPHD 파일 내부의 스크립트는 Java에서 사용되는 구문과 유사합니다. CPHD 웹 페이지는 CloudChan 프로세스에서 실행될 때 HTML 문서를 반환합니다. CPHD 형식은 모든 XML 문서에서 작동할 수 있습니다.

## CPHD 파일 형식

CPHD 파일은 일반 텍스트 파일로 디스크에 저장됩니다. 이렇게 하면 메모장이나 Notepad++와 같은 텍스트 편집기에서 이러한 파일을 열 수 있습니다.
