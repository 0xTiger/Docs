## .JSPA 파일이란?

JSPA 파일은 Java 서버(.jsp) 파일을 가리키는 참조 바로 가기 파일입니다. JSPA 파일이 가리키는 URL은 논리적 이름이며 백엔드에서 Servelet, JSP 또는 기타 항목에 매핑될 수 있습니다. JSPA 파일은 호출될 때 여러 페이지를 실행할 수 있습니다.
