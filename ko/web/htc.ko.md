## .HTC 파일이란?

HTC 파일은 Microsoft Internet Explorer에서 사용하는 웹 파일 형식 파일입니다. 여기에는 웹 브라우저 내에서 다양한 작업을 수행하기 위한 작은 기능을 수행하는 HTML 구성 요소에 대한 정의가 포함되어 있습니다. HTC 파일 내부에 정의된 기능은 재사용이 가능하며 여러 [HTML](/ko/web/html/) 페이지에서 액세스할 수 있습니다.

## HTC 파일 형식 - 추가 정보

HTC 파일은 일반 텍스트 파일 형식으로 저장됩니다. 즉, 모든 텍스트 편집기에서 이를 열고 기능을 변경할 수 있습니다. 이러한 파일은 일반적으로 Internet Explorer에서 열리지만 그렇게 하면 이러한 파일을 볼 수만 있고 편집할 수 없습니다.

HTC 파일은 실제로 스크립트와 구성 요소를 정의하기 위한 HTC 고유 요소 집합의 조합인 HTML 파일입니다.
