## CRDOWNLOAD 파일이란?

CRDOWNLOAD 파일은 Microsoft Edge, Google Chrome 또는 Chromium 브라우저를 사용하여 인터넷에서 파일을 다운로드할 때 생성되는 부분적으로 다운로드된 파일입니다. 다운로드 중인 파일의 내용은 다운로드가 완료될 때까지 이 파일에 저장됩니다. 다운로드가 완료되면 .crdownload 확장자가 제거되고 다운로드 중인 파일의 원래 파일 확장자로 대체됩니다.

## CRDOWNLOAD 파일 형식

CRDOWNLOAD 파일은 텍스트 및 이진 형식일 수 있습니다. 브라우저는 다운로드가 시작되면 기본적으로 .crdownload 파일을 생성합니다. 다운로드가 중간에 중단되면 CRDOWNLOAD 파일은 디스크에 저장된 상태로 유지됩니다. 이것은 때때로 중단된 곳에서 다운로드를 재개하도록 허용합니다.

### CRDOWNLOAD 예

인터넷에서 fileformatslist.doc 파일을 다운로드 중임을 지원합니다. 다운로드가 시작되면 브라우저는 파일을 다운로드 폴더에 fileformatslist.crdownload 파일로 저장합니다. 파일이 다운로드되면 파일 크기가 커집니다. 다운로드가 완료되면 파일 이름이 fileformatslist.doc로 바뀝니다.
