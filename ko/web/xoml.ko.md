## .XOML 파일이란?

XOML은 Extensible Object Markup Language의 약자이며 Windows Workflow Foundation의 워크플로 개체에 대한 직렬화 형식입니다. **[XAML](/ko/web/xaml/)**을 기반으로 일반 **[XML](/ko/web/xml/)**에서 사용자 인터페이스를 만드는 데 주로 사용됩니다. 이들은 사용자 인터페이스에 대한 워크플로를 선언하는 데 사용되며 구현 논리가 포함된 별도의 파일과 함께 컴파일됩니다. 여기에는 루트 워크플로 노드, 중첩된 하위 요소 및 포함된 코드 세그먼트를 정의하는 트리 기반 구조가 포함됩니다. .NET용 Visual Studio를 사용하여 새 워크플로를 만들 때 코드 형식 또는 코드 구분 형식을 선택할 수 있는 옵션이 있습니다. 나중 파일을 선택하는 경우 IDE에서 두 개의 파일을 생성합니다. 하나는 XOML 형식(워크플로 레이아웃 및 설정으로 구성)이고 다른 하나는 프로그래밍 코드가 있는 [.CS](/ko/programming/cs/)/[.VB](/ko/programming/vb/) 형식입니다.
