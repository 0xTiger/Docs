## XHTM 파일이란 무엇입니까?

.xhtm 확장자를 가진 파일은 XHTML로 작성된 웹페이지 파일입니다. 이는 표준 구현으로 XML 1.0을 사용하며 HTML 4의 재구성으로 간주됩니다. XHTML과 마찬가지로 XHTM 파일에는 보다 엄격한 XML 지침을 따르는 표준 HTML 태그가 포함되어 있습니다. XHTM 파일은 웹 사이트 콘텐츠를 게시하는 데 사용되며 일반적으로 [XHTML](/ko/web/xhtml/) 확장자를 사용합니다.

Google Chrome, Apple Safari 및 Mozilla Firefox와 같은 웹 브라우저를 사용하여 XHTM 파일을 열 수 있습니다.

## XHTM 예

다음 XHTM 문서 예는 XML 선언을 정의합니다.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
