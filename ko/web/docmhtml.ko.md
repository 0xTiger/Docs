## DOCMHTML 파일이란?

DOCMHTML 파일은 이전 버전의 Microsoft Word에서 저장한 웹 아카이브의 오래된 변형입니다. 최신 버전의 Microsoft Word에서는 더 이상 지원하지 않습니다. DOCMHTML 파일에는 원본 .doc 워드 문서의 HTML 표현이 포함되어 있습니다. 웹 브라우저뿐만 아니라 Microsoft Word에서도 열 수 있습니다. DOCMHTML이 게시되면 웹 요청을 통해 사용자가 액세스할 수 있는 웹 서버에서 호스팅될 수 있습니다. 또 다른 유사한 형식은 중단된 [DOCHTML](/ko/web/dochtml/)입니다.

## DOCMHTML 파일 형식 - 추가 정보

DOCMHTML 파일은 HTML 형식으로 저장하는 데 사용되었습니다. 메모장 및 메모장++과 같은 텍스트 편집기에서 열 수 있습니다. 사용되지 않는 것으로 표시된 후 Microsoft Word는 Word 문서를 [MHTML](/ko/web/mhtml/) 및 [MHT](/ko/web/mht/) 파일로 저장하는 옵션을 도입했습니다. 이러한 새로운 파일 형식은 웹 브라우저에서 열 수 있습니다.
