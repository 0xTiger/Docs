## .ADR 파일이란?

ADR 파일(확장자가 .adr로 저장됨)은 Opera 웹 브라우저에서 생성된 책갈피 목록입니다. 이 목록은 사용자가 웹 URL을 계속 책갈피함에 따라 채워집니다. 이러한 북마크는 필요할 때 백업 및 복원할 수 있는 ADR 파일에 저장됩니다. 크롬, 파이어폭스 등 대부분의 브라우저는 북마크를 [HTML](/ko/web/html/) 형식으로 저장하기 때문에 ADR 파일 형식으로 변환하여 Opera 브라우저에서 가져올 수 있습니다. ADR 파일은 Opera 브라우저에서 열 수 있습니다.

## ADR 파일 형식 - 추가 정보

ADR 파일의 내부 파일 형식 세부 정보는 알 수 없습니다. 그러나 다른 버전의 Opera로 내보낸 ADR 파일 형식 간에는 차이가 있습니다. 이것은 [ADR 파일 병합](https://superuser.com/questions/471959/how-do-i-merge-several-opera-adr-bookmark-files-to-one-single-file) 생성을 어렵게 만듭니다. 다른 버전의 Opera. 일부 외부 도구를 사용하여 이들을 병합할 수 있습니다.
