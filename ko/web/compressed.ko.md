## 압축 파일이란 무엇입니까?

압축 파일은 **WordCompress** 플러그인으로 생성된 작은 크기의 Wordpress 압축 웹페이지입니다. 플러그인에 의해 HTML, CSS, JavaScript 코드를 압축하여 생성된 웹페이지를 표현하기 위한 한 줄의 코드가 포함되어 있습니다. 이렇게 변환된 압축 파일은 크기가 작기 때문에 인터넷을 통해 더 빠르게 로드하고 전송할 수 있습니다.

### 압축 파일 형식 - 추가 정보

압축된 파일은 Microsoft Notepad, Notepad++ 및 Apple TextEdit과 같은 텍스트 편집기로 열 수 있는 텍스트 파일로 저장됩니다. 텍스트 편집기 외에도 Windows 및 Mac OS에서 모두 사용할 수 있는 Adobe Dreamweaver와 같은 웹 구축 소프트웨어를 사용하여 압축된 파일을 열 수 있습니다.

## WordCompress 플러그인 정보

WordCompress 플러그인은 처음에 WordPress 웹페이지를 압축 파일 형식으로 압축하는 데 사용되었습니다. 플러그인은 더 이상 지원되지 않으며 중단됩니다. 페이지의 소스 코드에서 모든 공백을 제거하여 웹 페이지의 소스 코드를 단일 실행 파일에 작성합니다. 압축된 파일은 웹페이지의 원본 소스 코드와 함께 저장되어 페이지의 원본 소스 코드에 포함된 원본 주석, 들여쓰기 및 기타 공백을 보존합니다.
