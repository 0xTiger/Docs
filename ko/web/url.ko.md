## URL 파일이란?

확장자가 .url인 파일은 모든 웹사이트 주소로 컴퓨터에 저장할 수 있는 웹사이트 URL 바로 가기입니다. 더블 클릭하여 열면 URL은 기본 브라우저에서 웹 사이트를 열고 페이지는 사용자에게 표시됩니다. URL 파일은 일반적으로 웹 사이트에 대한 바로 가기 파일이라고 하며 기본 브라우저의 아이콘으로 표시됩니다. URL 파일은 Microsoft 메모장, 메모장++ 또는 Apple TextEdit와 같은 모든 텍스트 편집기에서 열고 편집할 수 있습니다.

## URL 파일 형식 - 추가 정보

URL 파일은 일반 텍스트 형식으로 저장되며 모든 텍스트 편집기에서 열고 볼 수 있습니다. 사용자가 다른 탐색을 수행하는 기본 웹 브라우저에서 열립니다.
