## DOTHTML 파일이란?

DOTHTML 파일은 HTML 버전으로 저장된 Microsoft Word 템플릿 문서입니다. 웹 브라우저에서 열 수 있으며 콘텐츠가 다른 웹 페이지로 표시됩니다. DOTHTML 파일은 이러한 파일을 생성하기 위한 미리 형식이 지정된 설정이 포함된 Microsoft Word .DOT 파일에서 생성됩니다. [DOT](/ko/word-processing/dot/) 파일이 DOCHTML로 변환되면 이러한 모든 설정이 유지되고 변환된 DOTHTML 파일은 원본 DOT 파일과 유사한 형식으로 표시됩니다. 이렇게 하면 게시된 템플릿을 웹 브라우저에서 볼 수 있습니다.

## DOTHTML 파일 형식 - 추가 정보

DOTHTML 파일은 [HTML](/ko/web/html/) 파일로 저장되며 모든 HTML 편집기에서 열 수 있는 HTML 소스 코드를 포함합니다. 초기에 Microsoft Word는 [DOCHTML](/ko/web/dochtml/), [DOCMHTML](/ko/web/docmhtml/)과 같은 모든 문서 형식의 HTML 버전을 제공했지만 나중에 이러한 HTML 기반 버전은 중단되었습니다. . 결과적으로 DOTHMLT도 더 이상 지원되지 않습니다.
