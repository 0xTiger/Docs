## DOCHTML 파일이란?

DOCHTML 파일은 Microsoft Word로 생성된 문서의 HTML 형식입니다. 생성된 문서의 [HTML](/ko/web/html/) 버전으로 웹서버에 퍼블리싱할 수 있습니다. DOCHTML 파일은 변환된 워드 문서의 내용을 보기 위해 웹 브라우저에서 열 수 있습니다. DOCHTML 파일의 모양과 모양은 원본 Word 문서와 동일합니다.

## DOCHTML 파일 형식

DOCHTML 파일은 웹 서버에 게시할 준비가 된 HTML 형식으로 디스크에 저장됩니다. Microsoft Word는 문서를 DOCHTML 파일로 저장하는 지원을 중단했습니다. 이것은 현재 일반적으로 사용되는 [.mht](/ko/web/mht/) 및 [.mhtml](/ko/web/mhtml/) 파일 형식으로 대체되었습니다.
