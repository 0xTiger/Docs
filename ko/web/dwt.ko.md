## .DWT 파일이란?

확장자가 .dwt인 파일은 Dreamweaver 소프트웨어로 만든 웹 페이지 템플릿 파일입니다. 템플릿에 이미 정의된 기본 설정이 있는 여러 페이지를 개발하기 위한 시작점으로 사용됩니다. 새 [HTML](/ko/web/html/) 페이지는 이러한 템플릿의 복사본을 사용하고 필요한 경우 내용을 변경하면서 구축할 수 있으며 공통 섹션은 변경되지 않은 상태로 유지합니다. 머리글, 바닥글 및 메뉴와 같은 이러한 템플릿의 편집 불가능한 영역은 모든 웹페이지에서 동일하게 유지됩니다.

## DWT 파일 형식 - 추가 정보

DWT 파일은 일반 텍스트 파일로 개발 및 저장됩니다. 이들은 Dreamweaver 또는 모든 텍스트 편집기에서 열 수 있으며 필요한 변경 사항에 대해 편집할 수 있습니다. Dreamweaver 소프트웨어는 변경 사항이 적용될 때 UI를 업데이트하고 사용자는 응용 프로그램의 사용자 인터페이스에서 변경 사항의 효과를 시각적으로 관찰할 수 있습니다.

[HTML] 페이지(/ko/ web / html /)를 템플릿 파일로 저장할 때마다 Dreamweaver는 로컬 루트 폴더의 루트 수준에 템플릿 폴더를 만듭니다. .dwt 파일을 Templates 폴더에서 이동하거나 이름을 바꾸면 링크가 끊어집니다. 따라서 Dreamweaver에서 파일을 생성하는 Templates 폴더에 .dwt 파일을 보관하는 것이 중요합니다. 이미지 소스 파일이나 HTML 문서와 같은 사이트의 다른 요소는 Templates 폴더가 아닌 다른 위치에 보관하는 것이 좋습니다.

## 드림위버 소개

Dreamweaver 소프트웨어는 원래 Macromedia에서 개발했습니다. Macromedia의 다른 인기 있는 소프트웨어에는 오랫동안 브라우저용 애니메이션 플레이어로 사용 가능한 Flash 플레이어가 포함되었습니다. Adobe가 2005년 Dreamweaver를 인수한 후 Dreamweaver는 Adobe 소프트웨어 응용 프로그램의 필수적인 부분이 되었으며 현재 Adobe에서 별도로 배포합니다.
