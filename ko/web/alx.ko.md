## .ALX 파일이란?

ALX 파일은 [HTML](/ko/web/html/) 구성 요소의 형태로 웹 페이지에 포함된 ActiveX 파일입니다. ActiveX 컨트롤은 공식적으로 OLE 컨트롤 또는 OCX로 알려져 있습니다. ALX 파일은 페이지의 콘텐츠를 주기적으로 업데이트하기 위한 타이머 컨트롤을 추가하거나 사용자 작업을 기반으로 툴팁 스타일의 도움말을 표시하기 위한 팝업 창 컨트롤을 추가하는 것과 같은 웹 페이지에 다양한 기능을 삽입하는 데 사용할 수 있습니다. ALX 파일 관리는 다음을 사용하여 연결되었습니다.<Param> 웹 페이지의 HTML 코드 내 태그. Microsoft Internet Explorer, Microsoft ActiveX Control Pad 및 메모장 및 메모장++과 같은 텍스트 편집기에서 ALX 파일을 열 수 있습니다.

## ALX 파일 형식 - 추가 정보

ALX 파일은 일반 텍스트 파일 형식이며 텍스트 편집기에서 열어 편집할 수 있습니다. ActiveX 컨트롤은 웹 페이지에 드롭하여 웹 페이지에 포함될 수 있습니다. 그러나 동적 웹 페이지를 구축하기 위해 컨트롤은 [JScript](/ko/web/js/)의 도움으로 함께 통합됩니다. 웹 페이지의 컨트롤은 컨트롤의 이벤트 및 값과 연결되므로 사용자 상호 작용을 완벽하게 제어할 수 있습니다.
