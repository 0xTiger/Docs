## .STC 파일이란?

확장자가 .stc인 파일은 Adobe Contribute 소프트웨어에서 연결 정보를 저장하는 데 사용하는 연결 파일입니다. 기존 웹사이트 내에서 재사용할 수 있도록 사용자 이름, 호스트 URL, FTP, SFTP 등의 프로토콜과 같은 연결 정보를 포함합니다. 모르는 사람들을 위해 Adobe Contribute는 블로그로 알려진 콘텐츠 기반 기사를 포함하여 기존 웹 사이트에 업데이트를 푸시하는 데 사용되는 특수 [HTML](/ko/web/html/) 편집기였습니다. 여기에는 페이지 내에서 웹사이트에 원활하게 콘텐츠를 제공하는 Internet Explorer 및 Firefox와 같은 인기 있는 웹 브라우저용 플러그인이 포함되었습니다. 2017년 1월부터 중단되었으며 이제 Adobe Dreamweaver를 사용하여 처음부터 웹 사이트를 만들 수 있습니다.

## STC 파일 형식

STC 파일의 내부 데이터 형식을 알 수 없습니다. 그러나 연결 키는 암호로 암호화되며 파일로 전송된 모든 네트워크 또는 FTP 로그인 연결 정보는 안전하게 유지되며 Contribute를 통해서만 액세스할 수 있습니다. 키 연결 파일은 사용자에게 이메일로 보내거나 컴퓨터에 저장할 수 있습니다. 클릭하면 연결을 가져오는 Contribute가 자동으로 열립니다.
