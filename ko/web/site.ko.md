## SITE 파일이란 무엇입니까?

SITE 파일은 웹사이트 디자인 소프트웨어 `Adobe GoLive`에서 사용하는 프로젝트 파일입니다. 파일, 폴더, 이미지, 사용자 정의 스타일 시트 및 스크립트와 같은 웹 사이트 프로젝트 정보를 저장합니다. 또한 사이트 웹 페이지에 사용되는 웹 사이트 구조와 링크도 포함되어 있습니다. 프로젝트 파일이므로 프로젝트 파일에 대한 참조가 포함되어 있지만 실제 파일 자체는 포함되어 있지 않습니다.

Adobe GoLive는 중단되었으며 더 이상 지원되지 않습니다.

## 사이트 파일 형식

SITE 파일은 Adobe 독점 파일 형식으로 디스크에 저장되며 해당 파일 형식의 내부 세부 정보는 개발자가 참조할 수 있도록 공개적으로 제공되지 않습니다.
