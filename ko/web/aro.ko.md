## .ARO 파일이란?

ARO 파일은 SteelArrow 웹 응용 프로그램으로 생성되는 서버 측 웹 페이지 파일입니다. [HTML](/ko/web/html/) 및 SteelArrow 태그 및 스크립트가 포함되어 있습니다. 이들은 사용자의 브라우저로 보내기 전에 완전한 응답 페이지를 생성하기 위해 서버에서 실행됩니다. ARO 파일은 HTML 콘텐츠의 거의 95%를 포함하고 나머지는 SteelArrow 코드로 구성됩니다. ARO 파일이 작동하려면 SteelArrow 웹 애플리케이션 서버가 웹 서버 시스템에 설치되어 실행 중이어야 합니다.

## ARO 파일 형식

ARO 파일은 JavaScript 코드와 Java 애플릿이 포함된 HTML 마크업 언어 파일로 작성됩니다. [SteelArrow 태그](http://www.steelarrow.com/docs/basics1.aro)는 웹 페이지 개발을 위한 기본 빌딩 블록입니다. 이것들은 페이지의 표시를 변경하지 않지만 페이지를 데이터로 채우는 데 사용됩니다. 또한 조건문으로 출력을 제어하는 데 사용할 수도 있습니다.

### ARO 태그 예

그만큼<SAOUTPUT> ARO 태그를 사용하면 개발자가 스크립트 내 임의의 위치에서 데이터 값을 출력할 수 있습니다. 예를 들어, 다음을 사용하여 PARAM 테이블의 출력을 얻는 데 사용할 수 있습니다.<SAOUTPUT VALUE=PARAM[]> HTML 내에 표시할 수 있는<BODY> 꼬리표.
