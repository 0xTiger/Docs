## .BWP 파일이란?

BWP 파일은 프로그래밍 언어 **BuRg3r**로 만든 웹페이지 스크립트 파일입니다. BuRg3r에만 해당되며 다음으로 구분된 BuRg3r 소스 섹션이 있는 HTML 코드를 포함할 수 있습니다.<?b and ?> 태그. BuRg3r은 Fluffy Smoke Entertainment에서 개발한 게임 엔진이기도 한 범용 프로그래밍 언어입니다.
