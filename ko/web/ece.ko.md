## .ECE 파일이란?

ECE 파일은 콘텐츠 관리 플랫폼인 **Escenic Content Engine(ECE)**에 의해 동적으로 생성된 웹 페이지입니다. 여기에는 웹 서버에서 처리되고 사용자 브라우저에 [HTML](/ko/web/html/) 페이지로 다시 제공되는 서버 측 코드가 포함되어 있습니다. ECE 파일 형식은 ASP 및 PHP와 같은 다른 서버 측 파일 형식에 비해 일반적으로 많이 사용되지 않습니다. 이러한 파일은 내부적으로 렌더링되어 표시 목적으로 최종 사용자에게 제공됩니다.

## ECE 파일 형식

ECE 파일은 디스크에 저장되지 않지만 요청 시 사용자에게 직접 제공됩니다. 이들은 HTML로 변환되어 표시를 위해 사용자의 브라우저로 다시 전송됩니다. Escenic Content Engine은 뉴스 기사 페이지 게시에 적극적으로 사용됩니다. JEE 및 SQL 기술을 기반으로 하는 ECE 서버는 모든 온라인 콘텐츠에 대해 안정적이고 확장 가능한 저장소를 제공합니다.

클라이언트 브라우저는 HTTP/REST API를 통해 ECE 서버의 콘텐츠에 액세스할 수 있습니다. [HTML](/ko/web/html/) 페이지를 제공하는 것 외에도 ECE는 비디오/오디오 콘텐츠를 관리하고 게시하기 위한 매우 효율적인 워크플로를 제공합니다.
