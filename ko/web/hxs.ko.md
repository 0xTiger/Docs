## .HXS 파일이란?

HXS 파일은 Microsoft 도움말 2.x 파일에 대해 컴파일되고 사용되는 도움말 파일입니다. 주제 페이지 세트에서 컴파일 및 생성되며 이전 [CHM](/ko/web/chm/) 도움말 파일 형식과 유사한 [HTML](/ko/web/html/)의 하위 집합으로 작성됩니다. HXS 파일은 압축된 형식으로 저장되며 .HxC 기본 프로젝트 파일, .HxF 포함 파일, .HxT 목차, .HxA 속성 정의 파일 및 여러 .HxK 인덱스에서 생성됩니다. HXS 파일의 메타데이터는 [XML](/ko/web/xml/)에서 생성됩니다.

HXS 파일은 웹 브라우저로 열 수 있습니다.

## HXS 파일 형식

HXS 파일은 HTML 콘텐츠 및 XML 메타데이터에서 압축 파일로 생성 및 저장됩니다.
