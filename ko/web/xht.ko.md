## .XHT 파일이란?

확장자가 .xht인 파일은 [XHTML](/ko/web/xhtml/)(Extended Hypertext Markup Language) 파일 형식과 연결된 웹 파일로, 그 자체가 [XML](/ko/web/xml/) 기반 마크업 파일입니다. 이 두 파일은 모두 HTML과 유사하지만 XML과 유사한 구문이 더 엄격합니다. XHT 파일은 보다 구조화되고 스크립팅이 적게 포함되도록 설계되었으며 기존 XML 기능을 사용하는 것 외에도 일반적입니다.

## XHT 파일 형식

XHT 파일은 기본적으로 UTF-8 인코딩을 사용하여 일반 텍스트 파일로 생성 및 저장됩니다. 여기에는 유니코드 인코딩을 지원하는 모든 텍스트 편집기로 열고 편집할 수 있는 XHTML 문서의 전체 소스 코드가 포함되어 있습니다. 텍스트 편집기 외에도 XHT 파일 형식은 대부분의 최신 웹 브라우저에서 이해할 수 있으며 이를 사용하여 열 수 있습니다.

## XHT 예

XHT 문서의 다음 예는 XML 선언을 정의합니다.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
