## .EPIBRW 파일이란?

EPIBRW 파일은 Microsoft IIS 기반 웹 서버에서 호스팅되는 일반 파일입니다. 이 파일의 목적은 특정 작업이나 모든 종류의 데이터 저장소에 국한되지 않고 일반적입니다. [.pdf](/ko/pdf/) 파일에서 가장 많이 사용되지만 다른 여러 파일 형식에서도 사용할 수 있습니다.

EPIBRW 파일을 더블 클릭하면 내용을 표시하는 브라우저에서 열릴 수 있습니다. 그러나 파일이 다운로드용으로 링크된 경우 이 이벤트의 결과로 파일 다운로드 프로세스가 시작됩니다.
