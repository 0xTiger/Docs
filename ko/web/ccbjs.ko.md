## .CCBJS 파일이란?

CCBJS 파일은 3D 앱, 게임 및 웹사이트 편집기 **CopperCube**에서 생성된 Javascript 파일입니다. 여기에는 .ccb 파일에서 생성된 [.html 파일](/ko/web/html/)의 일부로 생성된 JavaScript 코드가 포함되어 있습니다. CopperCube는 코드를 작성하지 않고도 게임 플레이 로직과 상호 작용을 개발할 수 있는 사용자 친화적인 스크립팅 인터페이스를 제공합니다. CopperCube에서 지원하는 다른 파일 형식으로는 .ccscene, .ccmodel, .ccmaterial, .cctex, .ccjs, .ccp 및 .ccpak가 있습니다.

## CCBJS 파일 형식

CCBJS 파일 형식의 내부 파일 형식 세부 사항은 알려져 있지 않습니다. 그러나 Javascript 파일은 일반 텍스트 파일로 디스크에 저장되며 Microsoft Notepad, Notepad++ 및 Apple TextEdit과 같은 텍스트 편집기로 열 수 있습니다.
