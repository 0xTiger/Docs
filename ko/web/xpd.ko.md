## .XPD 파일이란?

XPD 파일은 XML 문서에서 생성된 웹페이지 파일입니다. XSLT 변환을 통해 생성되는 경우가 더 많습니다. 다중 프로세스 [XML](/ko/web/xml/)의 상호 운용성을 높이는 방법을 정의하는 [XPD 파일 형식 사양](https://www.w3.org/TR/xml-pipeline/)은 W3에서 사용할 수 있습니다. 응용 프로그램. XML 파이프라인 언어는 XML 리소스 간의 처리 관계를 정의합니다.
