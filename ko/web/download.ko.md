## 다운로드 파일이란?

DOWNLOAD 파일은 인터넷에서 부분적으로 다운로드한 파일로 아직 다운로드 중입니다. Apple Safari 브라우저를 사용하여 인터넷에서 파일을 다운로드하는 경우 다운로드가 완료되는 데 시간이 걸릴 수 있습니다. 부분적으로 다운로드된 이 파일은 **Downloads** 폴더에 저장됩니다. 어떤 이유로 다운로드가 중단된 경우 파일을 두 번 클릭하여 다시 시작할 수 있습니다.

## 파일 형식 다운로드 - 추가 정보

DOWNLOAD 파일은 텍스트 및 바이너리 파일 형식으로 디스크에 저장됩니다. 여기에는 다운로드 중인 전체 파일이 포함되어 있지 않습니다. 다운로드가 완료되면 .download 파일 확장자가 다운로드 중인 파일의 원래 확장자로 바뀝니다. 예를 들어 인터넷에서 MP4 파일 형식으로 비디오를 다운로드하는 경우 파일이 다운로드되어 .download 확장자로 저장됩니다. 다운로드가 완료되면 파일 확장자가 .mp4로 변경됩니다.

DOWNLAOD 파일은 Google Chrome 및 Microsoft Edge에서 사용하는 [.CRDOWNLOAD](/ko/web/crdownload/) 파일과 유사합니다.
