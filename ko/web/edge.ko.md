## .EDGE 파일이란?

확장자가 .edge인 파일은 Adobe Edge로 프로젝트를 생성하고 저장할 때 생성되는 Adobe Edge 프로젝트 파일입니다. Adobe Edge를 사용하면 웹 디자이너가 데스크톱 및 모바일 브라우저용 웹, 디지털 출판 및 리치 미디어 광고를 위한 대화형 [HTML](/ko/web/html/) 애니메이터를 만들 수 있습니다. 이는 HTML/HTML5, [JavaScript](/ko/web/js/) 및 [CSS3](/ko/web/css/)를 사용하여 수행됩니다. Edge 파일은 일반 텍스트 파일로 생성되며 Microsoft 메모장, Apple TextEdit 및 메모장++과 같은 여러 응용 프로그램에서 열 수 있습니다.

## EDGE 파일 형식

EDGE 파일은 일반 텍스트 형식으로 저장되며 애니메이션을 활성화하기 위해 jQuery를 사용합니다. EDGE 애니메이션의 대안은 더 이상 지원되지 않는 Flash 파일이었습니다. EDGE 파일은 HTML, JavaScript 및 CSS 파일과 함께 저장됩니다. EDGE 파일을 열면 웹 페이지, 즉 HTML 파일이 열립니다.
