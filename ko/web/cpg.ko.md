## .CPG 파일이란?

CPG 파일은 웹 페이지 편집기 소프트웨어인 Cool Page에서 만든 프로젝트 파일입니다. 이 소프트웨어는 수년 전에 폐기되어 더 이상 사용되지 않습니다. CPG 파일에는 웹 페이지에서 사용되는 개체와 이러한 개체의 속성에 대한 세부 정보가 포함되어 있습니다. CPG 프로젝트 파일은 Cool Page 소프트웨어에서 열 수 있으며 마무리 후 웹 페이지를 [HTML](/ko/web/html/) 웹 페이지로 내보낼 수 있습니다.

## CPG 파일 형식

CPG 파일 형식의 내부 파일 형식 세부 정보는 알려져 있지 않습니다. 이 파일은 바이너리 파일로 저장되었으며 Cool Page 소프트웨어에서만 열립니다.
