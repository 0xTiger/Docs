## .DMG 파일이란?

확장자가 .dmg인 파일은 Apple 디스크 이미지 파일이며 Mac OS X 디스크 이미지 파일이라고 합니다. DMG 파일 형식은 Apple PC 간에 응용 프로그램과 파일을 할당하기 위해 가장 널리 사용되는 디스크 이미지 파일입니다. DMG 파일을 가상 드라이브에 마운트하는 것이 매우 편리하거나 Apple 디스크 유틸리티 소프트웨어를 사용하여 DMG 파일 형식의 내용을 열고 볼 수 있습니다. Mac OS Classic에서 사용되는 이전 버전의 IMG 파일은 DMG 파일 형식으로 대체됩니다. DMG 파일은 Mac 사용자용으로 생성되며 Windows OS에서는 지원되지 않지만 사용자는 7-Zip, Pea Zip 등과 같은 압축 및 압축 해제 도구를 사용하여 Windows 운영 체제에서 DMG 콘텐츠에 액세스할 수 있습니다. DMG 파일 확장자는 다음에서도 사용됩니다. Oracle 데이터베이스의 바이너리 형식으로 보관되는 스크랩 파일용 Oracle 내보내기 및 수신 도구.

## 약력

DMG 파일 확장자는 Apple에서 개발했으며 대부분 Macintosh 컴퓨터에서 사용됩니다. 대부분 DMG 파일은 Mac 장치에 인터넷에서 시작하는 다양한 유형의 파일 및 소프트웨어를 설치할 수 있는 기능을 제공하는 데 사용됩니다. DMG 파일은 기본적으로 데스크탑을 열 때 데스크탑에 표시되는 마운트 가능한 디스크 이미지입니다. 파일은 일반적으로 암호화되고 압축된 원시 데이터로 구성됩니다. Mac 시스템은 인터리브 처리된 디스크를 처리하는 것과 동일한 방식으로 DMG를 처리하며 즉시 파일을 열거나 "추적"하려고 합니다.

## DMG 파일 형식의 기술 사양

* Macintosh 컴퓨터용으로 특별히 생성됨
* 애플에서 개발
* 파일 형식은 바이너리입니다.
* 파일 카테고리는 디스크 이미지 파일입니다.
* 지원 프로그램은 MacOS입니다.
* 파티션 테이블을 포함할 수 있음
* HFS+와 같은 모든 종류의 파일 시스템을 포함할 수 있습니다.
* 이미지 압축 기능
