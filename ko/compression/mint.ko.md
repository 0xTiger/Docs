## .MINT 파일이란?

MINT 파일은 [Linux Mint](https://linuxmint.com/) 배포판에서 사용하는 설치 패키지 파일입니다. Linux Mint OS 설치를 시작하고 추출하는 데 사용되는 압축 소스 코드 파일이 포함되어 있습니다. Linux Mint OS는 GUI(Graphical User Interface)를 사용하기 쉽고 우아한 GUI 디자인을 가진 최초의 OS입니다. MINT 파일은 압축 해제되어 설치 프로그램 패키지와 함께 임시 디렉토리에 저장됩니다. 설치는 Linux Mint OS 설치와 동일한 디렉토리에서 실행됩니다.

## Linux MINT 파일 형식

MINT 파일 형식의 내부 파일 형식 세부 정보는 공개적으로 사용할 수 없습니다. 그러나 MINT 파일 내부에 저장된 압축 설치 파일은 Linux Mint OS 설치 프로그램과 통합된 압축 표준을 따릅니다. 이렇게 하면 이러한 파일의 압축을 풀고 설치의 일부가 될 수 있습니다.
