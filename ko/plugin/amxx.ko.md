## AMXX 파일이란 무엇입니까?

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. 서버 관리자가 서버 기능을 사용자 정의하고 확장할 수 있는 스크립팅 플랫폼입니다. AMXX는 여러 플러그인이 충돌 없이 서버에서 동시에 실행될 수 있도록 하는 MetaMod 플러그인 관리 시스템과 함께 사용됩니다. AMXX 플러그인이 제공하는 기능에는 관리 명령, 게임 플레이 수정 및 서버 관리가 포함됩니다.
