## SVGZ 파일이란?

확장자가 .svgz인 파일은 .SVG(Scalable Vector Graphics) 파일의 압축 버전입니다. gzip 압축으로 압축되며 [XML](/ko/web/xml/) 형식의 데이터를 포함합니다. SVGZ 파일은 투명도, 그라디언트, 애니메이션 및 필터를 지원합니다. SVGZ 파일은 기본 SVG 파일에 비해 크기가 더 작으며 이 축소된 파일 크기는 그래픽 파일을 온라인으로 전송하는 데 도움이 됩니다. 그래픽 디자이너는 Adobe Illustrator, Corel PaintShop Pro 등과 같은 도구를 사용하여 SVGZ 파일을 만듭니다. 단, 이미지 데이터를 전송하면서 Apache Server에서 [GZip 압축](/ko/compression/gz/)을 활성화하여 SVGZ 파일을 생성할 수 있습니다.

## SVGZ 파일 형식

[SVG](/ko/page-description-language/svg/)로 압축되는 SVGZ는 SVG 문서의 크기를 줄이는 개방형 압축 gzip 형식을 기반으로 합니다. 이 압축 아카이브에는 XML 파일 형식(/ko/web/xml/)을 기반으로 하는 일반 텍스트 SVG 문서가 포함되어 있습니다. SVGZ 파일은 SVG 1.1 사양을 준수하는 뷰어/응용 프로그램에서 열 수 있습니다. SVGZ 파일은 원본 SVG 파일보다 크기가 20-50% 작습니다.
