파일 형식은 응용 프로그램이 이러한 파일을 읽거나 만들기 위해 따르는 파일의 데이터 구조를 설명합니다.

파일 형식은 벡터 파일 형식, 스프레드시트 파일 형식, 비디오 파일 형식, 이미지 파일 형식, 3D 파일 형식 및 기타 여러 범주로 그룹화할 수 있습니다. 파일은 BMP에서 PNG와 같은 다른 파일 형식으로 변환할 수도 있습니다.
