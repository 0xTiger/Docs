## FBX에서 STL로 정보

FBX(FilmBox)는 Kaydara가 MotionBuilder용으로 개발한 널리 사용되는 3D 파일 형식입니다. 2006년 Autodesk Inc에 인수되었으며 현재 많은 3D 도구에서 사용되는 주요 3D 데이터 교환 파일 형식 중 하나입니다. [STL](/ko/cad/stl/)은 스테레오리소그래피(Stereolithrography)의 약어로, 3차원 표면 형상을 표현하기 위한 상호 교환 가능한 파일 형식입니다.

[FBX](/ko/3d/fbx/)를 STL 파일 형식으로 변환하는 것은 일반적인 요구 사항이며 FBX를 STL로 무료로 변환할 수 있는 여러 앱을 온라인에서 사용할 수 있습니다. .NET, Java 또는 Python 애플리케이션에 FBX를 STL 기능에 포함시키는 데 관심이 있는 애플리케이션 개발자라면 이 목적으로 Aspose.3D API 사용을 고려할 수 있습니다.
