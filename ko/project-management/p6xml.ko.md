## .P6XML 파일이란? ##

P6XML 파일 확장자는 가독성, 적응성 및 폭넓은 사용이 가능하여 이상적인 형식입니다. Primavera P6 Professional에는 [XER](/ko/project-management/xer/) 파일과 XML 파일을 모두 내보내고 가져오는 기능이 있습니다. XML은 **Extensible Markup Language** 를 나타냅니다. 이 일반 형식을 통해 사용자는 Primavera P6 데이터베이스 간에 프로젝트 정보를 공유할 수 있습니다. 프로젝트 데이터는 XML 파일을 통해 텍스트 파일에 쉽게 저장할 수 있습니다. 이것은 수많은 소프트웨어 프로그램이 데이터를 교환하는 것을 더 간단하게 만듭니다. 이러한 데이터는 Primavera P6 사용자가 쉽게 공유하고 관리할 수 있습니다.

## P6XML 파일 형식의 장점 ##

* Primavera 사용자는 XML 파일을 사용하여 내보내고 가져올 기준선을 쉽게 정의할 수 있습니다.
* XML 내보내기는 모든 프로젝트 세부 실행 계획을 포함할 수 있습니다.
* 활동 코드, 캘린더 및 리소스와 같은 특정 전역 데이터 항목을 편집, 추가 또는 삭제할 수 없는 Primavera 6을 사용하는 모든 사용자는 해당 항목을 가져올 수 없습니다.
* Primavera에서 사용하는 일반 및 프로젝트 보안 프로필 설정 외에도 데이터베이스의 데이터에 영향을 미치거나 변경하는 데이터 항목은 가져오지 않습니다.

## Primavera P6에서 XML 파일 내보내기 ##

P6 Professional 프로젝트 및 프로젝트 데이터를 P6XML 파일로 내보내려면 Oracle Primavera Cloud를 사용하십시오. 파일이 P6 Professional 데스크톱 애플리케이션에서 구성될 때 파일을 다운로드하고 Primavera Cloud로 가져올 수 있습니다. 내보내기 프로세스를 중단 없이 작동하기 전에 P6 Professional 데이터베이스 또는 P6 EPPM에 연결되어 있는지 규제해야 합니다. 내보내기 프로세스의 일부 기능은 데이터베이스에 따라 다를 수 있습니다. P6 Proficient 데이터베이스와 함께 P6 Specialized 설정이 있으면 내보내기 프로세스가 시스템에서 로컬로 작동합니다. P6 Professional 데이터베이스가 있으면 시스템 설정에서 P6 XML 파일을 쉽게 내보내고 열 수 있습니다.

다음은 내보내기 프로세스를 설명하는 몇 가지 단계입니다.

* 오픈욕구 프리마베라 P6 프로젝트
* 그런 다음 파일 탭에서 내보내기를 선택하십시오.
* "Primavera P6 – (XML)"을 선택합니다. 그 후 다음을 클릭하십시오.
* 내보낼 프로젝트를 두 번 클릭하십시오.
* 내보낸 파일을 넣을 경로의 잘림을 선택하고 "마침"을 선택합니다.
* 파일을 저장할 방법을 선택하십시오
* 내보내기 파일에 프로젝트 수준 레이아웃이 필요하지 않은 경우 "모든 프로젝트 수준 레이아웃 내보내기"에 대한 선택을 취소해야 합니다.
* 완료를 선택하면 XML 내보내기 확인을 받게 됩니다.

### 중요 팁 ###

* P6 EPPM 데이터베이스에 연결할 때마다 관리자는 콘텐츠 소스의 URL을 P6의 애플리케이션 설정 일반 페이지에 있는 P6 URL 필드에 전달해야 합니다.
* 여러 프로젝트를 하나의 XML 파일로 내보내는 것이 편리합니다.
* 프로젝트에 할당된 글로벌 데이터 내보내기
* 모든 리소스에 대한 승인 없이 XML 파일을 내보내는 것이 편리합니다.



## P6XML 파일을 여는 데 문제가 있습니까? ##

P6XML 형식의 오작동으로 인해 발생할 수 있는 몇 가지 일반적인 문제는 다음과 같습니다.

* 지원 소프트웨어 부재
* 손상된 파일
* 바이러스에 감염된 파일
* Windows 레지스트리에서 실수로 삭제된 P6XML 설명
* 시스템에 파일을 열 수 있는 액세스 권한이 없습니다.
* 시스템의 오래된 드라이브
* 파일 확장자의 이름이 변경되었습니다.
