## .MPP 파일이란?

확장자가 .mpp인 파일은 프로젝트 관리와 관련된 정보를 통합적으로 저장하는 Microsoft Project 데이터 파일입니다. Microsoft는 프로젝트 관리 소프트웨어 제품인 Microsoft Project(MSP)와 호환되도록 이 독점 파일 형식을 개발했습니다. MPP 외에 MSP는 프로젝트 XML 스키마로 작업할 수 있습니다. 많은 응용 프로그램과 API를 통해 MPP를 다른 파일 형식으로 변환할 수 있습니다. Microsoft는 이제 여러 사용자가 공동 작업을 위해 프로젝트 관리 파일을 업로드할 수 있는 온라인 Project Server를 보유하고 있습니다.

## MPP 파일 형식 ##

Microsoft는 MPP를 바이너리 파일 형식으로 개발 및 게시했으며 그 사양은 현재까지 Microsoft에서 공개하지 않았습니다. 따라서 내부 파일 구조 및 세부 정보는 개발자가 사용할 수 없습니다. MPP 파일 읽기/쓰기를 지원하는 여러 API가 온라인에서 제공되지만 100% 정확도는 달성되지 않습니다. Microsoft는 시간이 지남에 따라 프로젝트 관리 응용 프로그램에 여러 기능을 도입했으며 MPP 파일 형식의 기능도 도입했습니다.

MPP 파일은 다음 MIME 유형을 사용합니다.

* 애플리케이션/vnd.ms-project
* 응용 프로그램/msproj
* 응용 프로그램/msproject
* 응용 프로그램/x-msproject
* 응용 프로그램/x-ms-project
* 응용 프로그램/x-dos_ms_project
* 응용 프로그램/mpp
* zz-application/zz-winassoc-mpp

## MPP 파일을 여는 데 문제가 있습니까? ##

다음은 MPP 형식의 오작동을 일으킬 수 있는 몇 가지 일반적인 문제 목록입니다.

* 지원 소프트웨어의 부재
* 손상된 파일
* 바이러스에 감염된 파일
* 시스템에 파일을 열 수 있는 액세스 권한이 없습니다.
* 시스템의 오래된 드라이브
* 파일 확장자의 이름이 변경되었습니다.
