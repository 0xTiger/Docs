## .LNK 파일이란? ##

Mac 시스템의 ID와 유사한 LNK 파일은 원본 이미지 문서 폴더 또는 프로그램에 대한 연결 역할을 하는 Windows 대안 또는 "링크"입니다. 여기에는 바로 가기 대상의 유형, 위치 및 파일 이름과 대상 문서를 여는 응용 프로그램 및 추가 바로 가기 키가 포함됩니다.

Windows에서 파일, 폴더 또는 실행 프로그램을 똑바로 세우고 바로 가기 만들기를 선택합니다. 파일 형식의 위치와 "시작" 디렉토리는 LNK 파일의 두 가지 기본 기능입니다. LNK 파일의 파일 형식은 마스킹되어 있으며 곡선 화살표는 바로 가기임을 나타냅니다.

## LNK 파일 형식 ##

LNK 파일 형식은 일반적으로 대상 파일과 동일한 아이콘을 갖지만 파일이 다른 위치를 가리키고 있음을 나타내기 위해 약간 구부러진 화살표가 추가됩니다. 바로 가기를 더블 클릭하면 사용자가 실제 파일을 더블 클릭한 것처럼 동작합니다.

응용 프로그램 바로 가기가 포함된 LNK 파일에는 프로그램 실행 방식에 영향을 주는 속성이 있을 수 있습니다. 속성을 변경하려면 바로 가기 파일을 마우스 오른쪽 버튼으로 클릭하고 **속성**을 선택한 다음 대상 필드를 변경합니다.

LNK 파일은 파일 확장자가 아니라 Windows 탐색기 확장입니다. 터미널 확장자로서 .lnk 문서는 Windows 탐색기에서만 파일을 대체하는 데 사용할 수 있으며 Windows 탐색기에서 로컬 문서에 대한 바로 가기 역할을 하는 것 외에 다른 용도로 사용됩니다. 이 파일도 문자 "L"로 시작합니다.

바로 가기는 생성될 때 특정 파일 및 디렉터리에 연결되지만 대상이 다른 위치로 변경되면 작동하지 않을 수 있습니다. 탐색기는 열릴 때 죽은 대상을 가리키는 바로 가기 폴더를 복구하기 시작합니다.


## 기술 사양 ##

"인식된 파일 형식의 확장명 숨기기" 폴더 보기 설정이 선택 해제된 경우에만 Windows에서 문서 바로 가기에 대한 .lnk 문서 확장자를 표시하지 않습니다. 권장되지는 않지만 HKEY_CLASSES_ROOT\lnk 파일 Windows 레지스트리 항목에서 "NeverShowExt" 속성을 제거하여 파일 확장자가 표시되도록 설정할 수 있습니다.

이렇게 하려면 다음 단계를 수행하십시오.

* 작업 표시줄 검색 필드에 "Regedit"를 입력하고 프로그램을 선택하여 "등록 편집기"를 엽니다.
* 응용 프로그램에서 Computer\HKEY HKEY_CLASSES_ROOT\lnkfile 위치로 이동합니다.
* 항목을 마우스 오른쪽 버튼으로 클릭하고 내보내기를 선택하여 항목을 백업합니다.
* "NeverShowExt" 속성 선택 및 삭제
* Windows를 다시 시작해야 합니다.
