## ICNS 파일이란 무엇입니까?

ICNS 파일은 macOS에서 응용 프로그램, 폴더 및 기타 항목의 아이콘을 저장하는 데 사용되는 파일 형식입니다. 동일한 아이콘의 다양한 이미지 크기와 해상도를 저장할 수 있는 컨테이너 파일 형식입니다.

ICNS 파일에는 일반적으로 macOS 장치의 다양한 디스플레이 크기와 해상도를 수용할 수 있도록 16x16픽셀에서 1024x1024픽셀까지 다양한 해상도의 이미지가 포함되어 있습니다. 파일 형식은 압축 알고리즘을 사용하여 이미지의 파일 크기를 줄입니다.

ICNS 파일은 Icon Composer 또는 Sketch와 같은 다양한 소프트웨어 도구를 사용하여 생성할 수 있으며 응용 프로그램, 폴더 및 문서와 같은 macOS 시스템의 다양한 항목에 할당할 수 있습니다. macOS 시스템에서 항목을 선택하면 운영 체제는 ICNS 파일에서 아이콘의 적절한 크기와 해상도를 검색하여 화면에 표시합니다.

## ICNS 파일 형식 - 추가 정보

ICNS 파일은 "아이콘 계열"이라는 기술을 사용하여 동일한 아이콘의 다양한 이미지 크기와 해상도를 보유할 수 있는 컨테이너 파일입니다. 아이콘 계열은 각각 동일한 아이콘을 나타내지만 크기나 해상도가 다른 이미지 리소스 그룹으로 구성됩니다. 아이콘 계열 내의 각 이미지 리소스에는 이미지 크기와 해상도를 식별하는 고유한 유형과 ID가 있습니다. 다양한 이미지 크기와 해상도는 다양한 유형과 ID를 사용하여 동일한 ICNS 파일 내에 저장됩니다.

macOS 시스템에서 항목을 선택하면 운영 체제는 해당 유형과 ID를 사용하여 ICNS 파일에서 아이콘의 적절한 크기와 해상도를 검색하여 화면에 표시합니다. 이를 통해 macOS 시스템은 더 작은 파일 크기를 유지하면서 Retina 디스플레이와 같이 해상도가 다른 화면에 고품질 아이콘을 표시할 수 있습니다.

macOS의 ICNS 파일은 둘 다 아이콘 저장을 위한 파일 형식으로 사용된다는 점에서 Windows의 .ICO 파일과 유사합니다. ICNS 파일과 마찬가지로 ICO 파일에는 Windows 장치의 다양한 화면 크기와 해상도에 아이콘을 표시하는 데 사용되는 동일한 아이콘의 여러 이미지 크기와 해상도가 포함될 수 있습니다.

그러나 ICNS와 ICO 파일에는 몇 가지 차이점이 있습니다. 예를 들어 ICNS 파일은 ICO 파일보다 고해상도 이미지를 저장할 수 있으므로 macOS 장치의 Retina 디스플레이와 같은 고해상도 디스플레이에 사용하기에 더 적합합니다. 또한 ICNS 파일에는 알파 채널이 있는 이미지가 포함될 수 있으므로 ICO 파일에서는 지원되지 않는 투명한 배경이 가능합니다.
