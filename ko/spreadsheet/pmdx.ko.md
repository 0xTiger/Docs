## .PMDX 파일이란?

PMDX 파일은 SoftMaker Office의 일부로 제공되는 소프트웨어인 [PlanMaker](https://www.softmaker.de/softmaker-office-planmaker)의 최신 버전으로 만든 스프레드시트 문서입니다. 행과 열의 형태로 데이터를 셀에 저장합니다. 각 셀에는 문자열, 숫자, 수식 또는 기타 서식이 지정된 데이터가 포함될 수 있습니다. PMDX 파일은 이전 버전 [PMD](/ko/spreadsheet/pmd/) 파일을 PlanMaker 업그레이드로 대체했습니다. XLSX는 PlanMaker에서 지원하는 표준 파일 형식이며 Microsoft Excel에서 이러한 파일을 여는 동안 변환이 필요하지 않습니다. PlanMaker는 또한 [XLS](/ko/spreadsheet/xls/) 파일 작업을 지원하고 [PDF](/ko/pdf/)와 같은 다른 형식으로 변환합니다.

## PMDX 파일 형식

PMDX 파일은 독점 파일 형식의 이진 파일로 디스크에 저장되며 해당 사양은 공개적으로 사용할 수 없습니다.
