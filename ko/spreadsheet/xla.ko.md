## .XLA 파일이란?

확장자가 .xla인 파일은 Microsoft Excel에 모듈, 추가 기능 및 기타 도구를 추가하는 Microsoft Excel 추가 기능 파일 형식입니다. 애드인은 Excel 스프레드시트에 추가 기능을 제공하며 버전 2007 이상에서 지원됩니다. XLA 파일의 MIME 미디어 유형은 application/vnd.ms-excel입니다. Windows 및 MacOS에서 사용할 수 있는 Microsoft Excel을 사용하여 열 수 있습니다.

## XLA 파일 형식

XLA 파일은 바이너리 데이터 형식을 기반으로 합니다. 그러나 파일 형식 사양에 대한 추가 정보는 제공되지 않습니다. 일반적으로 XLA는 [[MS-XLS] - Excel 바이너리 파일 형식 구조](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx)를 따릅니다.
