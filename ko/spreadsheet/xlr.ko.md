## .XLR 파일이란?

확장자가 .xlr인 파일은 행과 열에 데이터를 저장한 Microsoft Works 스프레드시트 파일 형식입니다. Microsoft Works는 초기 생산성 제품군으로 사용하기 위해 개발되었으며 1987년부터 2009년까지 기능을 유지했습니다. 여기에는 워드 프로세서, 스프레드시트 및 데이터베이스 관리 시스템이 포함되었습니다. 더 풍부한 기능이 포함된 Microsoft Office가 도입된 후 나중에 중단되었습니다.

## XLR 파일 형식 - 추가 정보

XLR 파일은 공개적으로 사용할 수 있는 파일 형식 사양에 대한 세부 정보가 없는 이진 파일로 저장되었습니다. 따라서 개발자는 파일 구조를 분석하고 프로그래밍 방식으로 이러한 파일을 읽고 쓰기 위한 응용 프로그램을 작성할 수 없습니다. Works 스프레드시트는 동일한 Excel 파일 형식을 사용하여 데이터를 .xlr 파일에 저장합니다.
