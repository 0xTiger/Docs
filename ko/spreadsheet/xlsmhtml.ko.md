## XLSMHTML 파일이란?

확장자가 .xlsmhtml인 파일은 브라우저에서 데이터를 보기 위해 Microsoft Excel 내에서 내보낸 원본 XLS 파일의 [HTML](/ko/web/html/) 버전입니다. 이 형식은 Excel에서 오랫동안 중단되었으며 더 이상 지원되지 않습니다. 내보낸 XLSMHTML 파일은 XLS 파일 형식을 따랐으며 웹 브라우저에서 열어 볼 수 있습니다. Excel은 스프레드시트 데이터를 [MHT](/ko/web/mht/) 및 [MHTML](/ko/web/mhtml/) 파일 형식으로 내보내는 기능을 제공하며 현재 널리 사용됩니다.

## XLSMHTML 파일 형식 - 추가 정보

XLSMHTML 파일은 XLS와 유사한 형식으로 스프레드시트 데이터를 표현하기 위한 HTML 코드로 생성되었습니다. XLSMHTML 파일 내부의 HTML 코드는 웹 브라우저에서 열었을 때 출력 파일이 원본 파일과 비슷하게 보이도록 배열됩니다. XLSMHTML 파일의 파일 형식 사양은 Excel에 의해 공개되지 않으며 개발자가 해당 응용 프로그램을 이해하고 개발할 수 있는 참조가 없습니다.
