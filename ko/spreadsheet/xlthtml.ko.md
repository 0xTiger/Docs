## XLTHTML 파일이란?

확장자가 .xlthtml인 파일은 [XLT](/ko/spreadsheet/xlt/) 템플릿 파일의 [HTML](/ko/web/html/) 버전이며 초기에 Microsoft Excel을 사용하여 생성되었습니다. 이 형식은 Excel에서 더 이상 지원되지 않으며 데이터를 [MHT](/ko/web/mht/) 및 [MHTML](/ko/web/mhtml/) 파일 형식으로 내보내는 방식으로 대체되었습니다. XLTHTML 파일의 기본 아이디어는 사용자가 시스템에 Excel을 설치할 필요 없이 웹 브라우저에서 XLT 템플릿 파일을 볼 수 있도록 하는 것이었습니다. XLTHTML 파일 형식으로 내보낸 파일은 브라우저에서 열 수 있습니다.

## XLTHTML 파일 형식 - 추가 정보

XLTHTML 파일은 HTML 파일 형식을 사용하여 웹 브라우저에서 열 수 있도록 XLT 템플릿을 저장했습니다. XLT 자체는 Excel 97-2003 버전에서 도입된 Excel 바이너리 파일 형식을 기반으로 합니다. 이러한 파일은 스프레드시트 데이터 채우기에 대한 일반 문서를 생성하기 위해 정의된 형식을 가진 시작 파일로 사용할 수 있습니다. 참고로 XLT는 Microsoft에서 개발자 참조.
