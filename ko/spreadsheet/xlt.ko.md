## .XLT 파일이란?

확장자가 .xlt인 파일은 Microsoft Office 제품군의 일부로 제공되는 스프레드시트 응용 프로그램인 Microsoft Excel로 만든 템플릿 파일입니다. Microsoft Office 97-2003은 새 XLT 파일 생성 및 열기를 지원했습니다. 최신 버전의 Excel은 여전히 이 이전 형식 템플릿 파일을 열 수 있습니다. 이러한 템플릿 파일은 페이지 서식, 글꼴 크기, 여백, 차트 등과 같은 기본 데이터 및 설정을 사용하여 새 Excel 파일을 빠르게 생성하는 데 사용되며 새 [.xls](/ko/spreadsheet/xls/) 파일로 추가로 저장할 수 있습니다.

## XLT 파일 형식

XLT 파일 형식은 Excel 97-2003 버전에서 처음 지원되었던 Excel 바이너리 파일 형식을 기반으로 합니다. Office Open XML 형식을 기반으로 하는 Excel 2007이 도입되면서 [XLTX](/ko/spreadsheet/xltx/) 파일 형식으로 대체되었습니다. Microsoft Excel 외에도 Lotus Translation Table, PlanMaker, LibreOffice Calc 등과 같이 이 파일 형식으로 작업을 지원하는 다른 응용 프로그램이 있습니다. 여러 응용 프로그램 및 [API](https://products.aspose.com/cells)를 사용할 수 있습니다. XLT를 다음과 같은 다른 파일 형식으로 변환하려면:

* XLT에서 PDF로
* XLT에서 DOC로
* XLT를 텍스트로
* 로터스에 XLT
* XLT에서 ODT로
* XLT에서 ODS로
* HTML에서 XLT로
* XLT에서 CSV로

XLT는 개발자 참조용으로 Microsoft에서 문서화한 바이너리 [XLS 파일 형식 사양](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx)을 기반으로 합니다.
