## .PMD 파일이란?

PMD 파일은 SoftMaker Office와 함께 제공되는 [PlanMaker](https://www.softmaker.de/softmaker-office-planmaker) 소프트웨어로 만든 스프레드시트 파일입니다. Excel 파일과 유사한 행과 열에 스프레드시트 데이터를 저장합니다. [XLSX](/ko/spreadsheet/xlsx/)는 Microsoft Excel 대신 소프트웨어를 쉽게 사용할 수 있도록 하는 PlanMaker의 표준 파일 형식입니다. PMD 파일에는 셀에 저장된 데이터 외에 수식과 차트가 포함될 수 있습니다. PlanMaker로 생성된 파일은 추가 변환 없이 Microsoft Excel로 열 수 있습니다. 또한 [XLS](/ko/spreadsheet/xls/) 파일을 편집할 수 있으며 [PDF](/ko/pdf/)와 같은 추가 파일 형식을 지원합니다.

## PMD 파일 형식

PMD 파일은 독점 파일 형식의 이진 파일로 디스크에 저장되며 해당 사양은 공개적으로 사용할 수 없습니다.
