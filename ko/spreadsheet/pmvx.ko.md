## .PMVX 파일이란?

확장자가 .pmvx인 파일은 Softmaker Office의 일부로 제공되는 [PlanMaker](https://www.softmaker.de/softmaker-office-planmaker) 스프레드시트 프로그램으로 만든 스프레드시트 템플릿 파일입니다. 데이터는 Microsoft Excel에서와 같은 방식으로 PMVX 파일의 행과 열에 저장됩니다. PMVX 파일은 데이터를 채운 후 PMD 및 PMDX 파일을 생성하는 데 사용할 수 있는 템플릿으로 미리 정의된 설정을 저장합니다. [XLSX](/ko/spreadsheet/xlsx/)는 PlanMaker의 기본 파일 형식이며 Microsoft Excel에서 직접 열 수 있습니다. PlanMaker를 사용하여 [XLS](/ko/spreadsheet/xls/) 파일을 편집하고 [PDF](/ko/pdf/) 파일을 생성할 수도 있습니다.

## PMVX 파일 형식

PMVX 파일은 PlanMaker의 독점 파일 형식으로 디스크에 저장되며 내부 파일 구조는 알 수 없습니다. PMVX 파일 저장과 관련하여 데이터를 Open XML 형식으로 XLSX 파일로 저장합니다.
