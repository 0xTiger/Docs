## CJS 파일이란 무엇입니까?

CommonJS(CJS) 파일은 CommonJS 구문으로 작성된 JavaScript 코드가 포함된 파일입니다. CommonJS는 프런트엔드 웹 브라우저 이상의 환경에서 작동하도록 설계된 모듈 시스템이며 Node.js와 같은 서버 측 JavaScript 환경에서 자주 사용됩니다.

## CJS 파일 형식 - 추가 정보

CJS 파일은 CommonJS 구문으로 작성되었으며 Microsoft 메모장 또는 Apple TextEdit과 같은 텍스트 편집기에서 편집할 수 있습니다. CommonJS 모듈은 일반적으로 별도의 파일에 저장되며 더 나은 구성 및 유지 관리를 위해 코드를 캡슐화하고 모듈화하기 위한 것입니다. 이러한 모듈은 require 함수를 사용하여 종속성을 가져오고 module.exports 또는 내보내기 개체를 사용하여 코드의 다른 부분에서 사용할 수 있는 값과 함수를 노출합니다.

## CJS 코드 예

CommonJS 모듈에는 다른 모듈을 가져오기 위한 require 함수와 모듈에서 값, 함수 또는 개체를 내보내기 위한 module.exports 또는 내보내기 개체와 같은 기능을 포함하는 특정 구문 및 구조가 있습니다. 이러한 모듈은 코드 조각을 캡슐화하고 분리하는 데 사용되므로 대규모 JavaScript 코드베이스를 더 쉽게 관리하고 유지할 수 있습니다. CommonJS 모듈의 기본 예는 다음과 같습니다.
