## .PHP 파일이란?
확장자가 .php인 파일은 웹 서버에서 실행되는 서버 측 스크립트를 작성하는 데 사용되는 오픈 소스 프로그래밍 언어를 나타냅니다. 대규모 웹 응용 프로그램 개발에 일반적으로 사용되는 가장 널리 사용되는 웹 스크립팅 언어입니다. 웹에서 가장 큰 블로그 시스템인 WordPress와 가장 큰 소셜 네트워크인 Facebook은 PHP 언어를 기반으로 합니다. Windows, Linux, Unix, Mac OS X 등과 같은 다양한 플랫폼과 호환됩니다. 또한 Apache, IIS 등과 같이 오늘날 사용되는 거의 모든 서버에서 성공적으로 실행됩니다. PHP는 MySQL을 포함한 광범위한 데이터베이스를 지원합니다.

## PHP 파일 형식 ##

단일 PHP에는 텍스트, CSS, [HTML](/ko/web/html/), JavaScript 및 PHP 코드가 포함될 수 있습니다. PHP 코드는 서버에서 실행되고 결과는 일반 HTML로 브라우저에 반환됩니다. PHP 파일은 모든 텍스트 편집기로 열고 제자리에서 편집할 수 있지만 Adobe Dreamweaver, Eclipse PHP 개발 도구와 같은 응용 프로그램은 PHP 코드를 작성하고 수정하는 편리한 방법을 제공합니다.

### 구문 ###

PHP 스크립트가 서버에서 실행되고 일반 HTML 결과가 브라우저로 다시 전송됩니다. PHP 스크립트는<?php and ends with ?>

### PHP 예제 ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
