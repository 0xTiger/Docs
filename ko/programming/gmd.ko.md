## .GMD 파일이란?

GMD 파일은 2D 게임을 만드는 데 사용되는 Game Maker 소프트웨어 개발 도구로 생성된 게임 개발 소스 파일입니다. GMD 파일은 텍스트 파일로 저장되며, 개발할 게임용 프로젝트 파일입니다. Game Maker는 Windows, Mac, Linux, Android, iOS, HTML5, Xbox, PlayStation 및 Nintendo Switch에서 사용할 수 있습니다. 이 도구는 전 세계의 인디 개발자, 전문 스튜디오 및 교육자가 게임 제작에 광범위하게 사용하고 있습니다.

## GMD 파일 형식

GMD 파일은 바이너리 파일 형식으로 저장되며 내부 파일 형식 세부 정보는 개발자가 참조할 수 없습니다.
