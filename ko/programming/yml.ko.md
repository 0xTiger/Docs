## .YML 파일이란?

YML은 데이터 직렬화 언어인 [YAML](/ko/programming/yaml/)(Yet Another Markup Language)의 텍스트 파일로, 구성 파일을 만드는 데 자주 사용됩니다. C/C++, Ruby, Python, Java, PHP, C# 등을 포함한 모든 프로그래밍 언어와 함께 사용할 수 있습니다. 특정 프로그래밍 언어와 독립적이므로 개방적이고 상호 운용 가능하며 사람이 읽을 수 있습니다. YML 파일은 Microsoft 메모장, 메모장++, Github Atom, Apple TextEdit 및 Visual Studio IDE와 같은 일반적인 텍스트 편집기로 열고 편집할 수 있습니다.

## YML 파일 형식 - 추가 정보

YML 파일은 애플리케이션 설정을 구성으로 저장합니다. 기본 구조는 디렉토리 또는 해시라고도 하는 맵입니다. YML 파일의 항목은 키와 값 쌍으로 구성됩니다. Ruby on Rails와 같은 일부 프로그래밍 언어는 데이터베이스 연결 구성을 `database.yml` 파일에 저장하지만 대부분 구성 .yml 파일은 `configure.yml`로 저장됩니다. [YML/YAML 파일 형식 사양](https://yaml.org/spec/1.2.2/)은 전체 언어 구조 및 구문 관련 정보를 설명합니다.
