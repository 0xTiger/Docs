## .ASI 파일이란? ##

ASI 파일은 웹 응용 프로그램 개발에 사용되는 Alpha Five 소프트웨어에서 사용되는 가변 파일 형식입니다. 웹 응용 프로그램에서 사용하는 변수를 저장합니다. Alpha Five에서 새 개발자 프로젝트를 생성하면 ASI 파일도 생성되어 Alpha Five 개발자 프로젝트와 함께 저장됩니다.

ASI 파일은 Alpha Software Alpha Anywhere로 열 수 있습니다.

## ASI 파일 형식

ASI 파일은 Alpha Five 독점 파일 형식으로 저장됩니다. 개발자 프로젝트의 일부로 응용 프로그램에서 읽고 씁니다.
