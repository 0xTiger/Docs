## .SCRIPT 파일이란?

SCRIPT 파일은 모든 스크립팅 언어로 된 스크립트를 포함하는 소스 파일입니다. 이렇게 하면 변수와 함수를 포함하는 일반화된 스크립팅 파일이 됩니다. 각 스크립팅 언어가 다를 수 있으므로 SCRIPT 파일 내부에 작성된 스크립트는 적절한 스크립팅 엔진을 사용하여 실행해야 합니다.

## SCRIPT 파일 형식 - 추가 정보

SCRIPT 파일은 디스크에 일반 텍스트 파일로 저장됩니다. Microsoft 메모장, 메모장++ 및 Apple TextEdit와 같은 텍스트 편집기를 사용하여 열고 편집할 수 있습니다. 스크립팅 언어가 다를 수 있으므로 파일 확장자의 이름을 해당 스크립팅 언어의 기본 확장자로 변경하여 실행할 수 있습니다. 예를 들어 JavaScript([.JS](/ko/web/js/)), VBScript([.VBS](/ko/programming/vbs/)), AppleScript(.SCPT), PHP([ .PHP](/ko/programming/php/)) 및 ASP([.ASP](/ko/web/asp/)).
