## .ANE 파일이란?

ANE 파일은 개발자가 [Adobe ActionScript](https://en.wikipedia.org/wiki/ActionScript) 코드를 호출하는 특정 기능을 작성하는 데 사용하는 Adobe Air Native Extension 파일입니다. 대부분의 경우 ANE 파일에는 확장에서 사용하는 기본 코드 라이브러리가 포함되어 있습니다. 그러나 파일 자체는 장치에 설치됩니다. 장치 진동과 같은 시스템 특정 기능을 허용합니다. Adobe AIR 응용 프로그램은 ANE 파일을 활용하기 위해 동적 링크를 사용하도록 컴파일해야 합니다.

## ANE 파일 형식 - 추가 정보

ANE 파일은 파일 형식 사양이 공개적으로 제공되지 않는 바이너리 파일로 디스크에 저장됩니다. ANE 파일에는 볼륨 또는 진동 기능 호출과 같은 Android 장치 기능에 액세스하기 위한 기본 Android Java 소스 코드가 포함될 수 있습니다. 이 모든 작업은 기본 시스템을 Adobe의 ActionScript 언어로 래핑하여 수행됩니다.
