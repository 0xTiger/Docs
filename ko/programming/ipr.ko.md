## .IPR 파일이란?

IPR 파일은 Java IDE **IntelliJ IDEA**로 생성된 프로젝트 파일입니다. 디렉터리 구조, 소스 코드 파일, 프로젝트에 포함된 라이브러리, 컴파일러 설정 등 프로젝트 설정과 관련된 모든 정보는 IPR 파일에 저장됩니다. IntelliJ IDEA 내에서 프로젝트를 열면 IDE가 IPR 파일의 정보를 읽고 해당 파일과 설정을 로드합니다. IPR 파일은 모든 정보를 설정 디렉토리로 포함하는 .IDEA 파일로 대체되었습니다.

## IPR 파일 형식

IPR 파일은 사람이 읽을 수 있는 보편적인 [XML](/ko/web/xml/) 파일 형식으로 저장됩니다. 여기에는 프로젝트 관련 콘텐츠를 표시하기 위해 로드되는 태그에 모든 프로젝트 관련 정보가 포함됩니다.

IPR 파일은 XML 파일 형식으로 저장되므로 메모장, 메모장++, Atom 및 Apple TextEdit와 같은 모든 텍스트 편집기에서 열 수 있습니다.
