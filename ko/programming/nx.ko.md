## .NX 파일이란?

NX 파일은 가상 게임 콘솔인 **LowRes NX**와 함께 사용하기 위해 만들어진 비디오 게임 파일입니다. BASIC 프로그래밍 언어로 작성되었으며 게임 작동 방식을 정의하는 게임 소스 코드가 포함되어 있습니다. 로드할 수 있고 사용자가 게임을 플레이할 수 있는 앱과 함께 사용할 고유한 NX 파일을 만들 수 있습니다.

텍스트 편집기, LowRes NX 또는 Microsoft Visual Studio Code와 같은 소스 코드 편집기에서 NX 파일을 열 수 있습니다.

## NX 파일 형식

NX 파일은 게임 지침이 포함된 일반 텍스트 파일로 디스크에 저장됩니다. 각 NX 파일에는 여러 카트리지 ROM 항목으로 구성된 게임 소스 코드가 있습니다. 이러한 ROM 항목에는 그래픽, 레벨 맵, 오디오 및 기타 게임의 소스 코드가 아닌 데이터와 같은 게임 구성 요소 정보가 있습니다. 프로그램의 ROM 항목은 LowRes NX와 함께 제공되는 NX 편집 프로그램을 사용하여 추가하거나 편집할 수 있습니다.

## NX 파일을 만드는 방법?

### PC에서 NX 파일 만들기

다음을 사용하여 PC에서 NX 파일을 만들 수 있습니다.

* 텍스트 편집기 - Microsoft Notepad, Notepad++, Github Atom 등
* 소스 코드 편집기 - Microsoft VIsual Studio Code 등

### iOS에서 NX 파일 만들기

LowRes NX에는 개발자가 이 앱 내에서 NX 파일을 생성하는 데 사용할 수 있는 iOS 버전이 있습니다.
