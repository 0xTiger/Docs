## .CS 파일이란?

확장자가 .cs인 파일은 C# 프로그래밍 언어의 소스 코드 파일입니다. .NET Framework와 함께 사용하기 위해 Microsoft에서 도입한 이 파일 형식은 EXE 또는 DLL 형식의 최종 출력 파일을 생성하기 위해 컴파일되는 코드를 작성하기 위한 저수준 프로그래밍 언어를 제공합니다. Microsoft Visual Studio로 만들고 컴파일할 수 있습니다. Microsoft Visual Studio Express를 사용하여 무료 IDE인 이러한 파일을 만들고 업데이트할 수도 있습니다. CS 파일은 간단한 데스크톱 응용 프로그램에서 더 복잡한 프로그램에 이르는 응용 프로그램 개발에 사용됩니다. C# 언어로 생성된 간단한 Visual Studio 프로젝트[솔루션](/ko/programming/sln/)는 하나 이상의 이러한 파일로 구성될 수 있습니다. 컴파일에 포함하도록 표시된 파일은 프로젝트의 일부이며 컴파일러에 표시된 파일을 사용하도록 지시하는 [CSPROJ](/ko/programming/csproj/) 파일에 나열됩니다.

## CS 파일 형식 ##

CS 파일은 편집을 위해 모든 텍스트 편집기에서 열 수 있는 텍스트 기반 파일 형식입니다. 그러나 적절한 구문 강조 표시와 함께 지원되는 IDE에서 열면 코드를 읽고 정렬하기 쉽습니다. 간단한 CS 파일에는 다음이 포함됩니다.

* 네임스페이스 선언 - 특정 네임스페이스에 의해 정의된 특정 기능을 참조하기 위해
* 변수 선언 - 특정 구현을 위한 클래스 수준 변수 선언
* 메소드 선언 - 특정 기능에 대한 메소드 선언 선언

### 구문 ###

* 세미콜론은 문장의 끝을 표시하는 데 사용됩니다.
* 중괄호는 문장을 그룹화하는 데 사용됩니다. 명령문은 일반적으로 메서드(함수)로, 메서드는 클래스로, 클래스는 네임스페이스로 그룹화됩니다.
* 변수는 등호를 사용하여 할당하지만 두 개의 연속 등호를 사용하여 비교합니다.
* 대괄호는 배열을 선언하고 그 중 하나의 주어진 인덱스에서 값을 얻기 위해 배열과 함께 사용됩니다.

### 예시 ###

```
using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Hello, world!");
}
}
```
