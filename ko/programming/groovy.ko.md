## .GROVY 파일이란?

GROOVY 파일은 Groovy 프로그래밍 언어로 작성된 소스 코드 파일입니다. GROOVY 파일 내부에 작성된 코드는 객체지향 언어[Java](/ko/programming/java/)와 유사하여 애플리케이션의 설계 및 개발이 용이합니다. GROOVY 파일은 JVM(Java Virtual Machine) 바이트 코드로 컴파일되며 다른 Java 코드 및 라이브러리와 호환됩니다.

## GROOVY 파일 형식 - 추가 정보

GROOVY 파일에는 Groovy 구문으로 작성된 소스 코드가 포함되어 있습니다. 여기에는 프로그래밍 언어와 Java 플랫폼용 스크립팅 언어로 모두 사용할 수 있는 코드가 포함될 수 있습니다. Groovy 언어는 Python, Ruby, Smalltalk와 유사한 기능을 가지고 있습니다.
