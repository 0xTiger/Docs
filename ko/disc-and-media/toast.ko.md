## 토스트 파일이란?
Roxio Toast 소프트웨어는 CD 또는 DVD 이미지를 만들고 파일을 .toast 확장자로 저장합니다. Roxio Toast는 Miles Software GmbH에서 처음 소개한 후 Adaptec에서 구입한 Macintosh용으로 잘 알려진 광 디스크 제작 및 미디어 변환 소프트웨어 응용 프로그램입니다. Toast 파일에는 비디오, 오디오 또는 데이터 디스크와 같은 모든 종류의 디스크가 포함될 수 있습니다. 또한 다른 디스크 이미지의 복사본일 수도 있습니다.

## 토스트 파일 형식
Toast 파일 형식은 Roxio Toast 소프트웨어에서 출력을 확장자가 .toast인 파일에 저장하는 데 사용됩니다. 다양한 유형의 디스크를 Mac OS 시스템을 통해 직접 구울 수 있지만 Toast는 프로세스에 대한 추가 제어 기능을 제공하며 목록 작성, 디스크에 구운 파일 추적 및 손상된 디스크에 대한 파일 복구와 같은 몇 가지 추가 기능도 제공합니다. [FLAC](/ko/audio/flac/) 및 [OGG](/ko/audio/ogg/)와 같이 Quicktime에서 지원하지 않는 오디오 및 비디오 형식도 지원합니다. 독점 파일 형식이기 때문에 파일 형식의 전체 구조에 대한 정보는 사용할 수 없습니다.
### 디스크의 종류
따라서 Roxio Toast는 다음 유형의 디스크 이미지 데이터를 포함하는 Toast 파일을 생성할 수 있습니다.
- **비디오 디스크 만들기**: Toast는 다음 프로젝트를 사용하여 비디오 디스크가 포함된 Toast 파일을 만들 수 있습니다.
- DVD 비디오
- 블루레이 비디오
- 고화질 DVD
- VIDEO_TS 폴더
- VIDEO_TS 편집
- BDMV 폴더
- AVCHD 아카이브
- **데이터 디스크 만들기**: Toast는 다음 프로젝트를 사용하여 데이터 디스크가 포함된 Toast 파일을 만들 수 있습니다.
- 맥 전용
- 맥 & PC
- DVD-ROM
- 커스텀 하이브리드
- 맥 볼륨
- 포토 디스크
- **비디오 디스크 만들기**: Toast는 다음 프로젝트를 사용하여 데이터 디스크가 포함된 Toast 파일을 만들 수 있습니다.
- 오디오 CD
- 음악 DVD
- MP3 디스크
- 향상된 오디오 CD
- 혼합 모드 CD
- **디스크 복사**: Toast는 다음과 같은 방식으로 복사를 지원합니다.
- 디스크 복사
- 이미지 복사
- 디스크 이미지 병합
