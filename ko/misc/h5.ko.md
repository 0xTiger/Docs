## .H5 파일이란?

H5는 대용량 데이터를 저장하는 데 사용되는 HDF(계층적 데이터 형식) 중 하나입니다. 많은 양의 데이터를 다차원 배열 형태로 저장하는 데 사용됩니다. 형식은 주로 빠른 검색 및 분석을 위해 잘 구성된 과학 데이터를 저장하는 데 사용됩니다. H5는 H4에 보다 향상된 파일 형식으로 도입되었습니다. 원래 슈퍼컴퓨팅 애플리케이션을 위한 국립 센터(National Center for Supercomputing Applications)에서 개발했으며 현재 HDF 그룹에서 지원합니다.

## HDF 파일 형식의 간략한 역사

HDF 파일 형식은 NASA에서 과학 데이터를 저장하는 표준 방법으로 선택했습니다. 그 전에는 GFTF(Graphics Foundations Task Force)와 HDF가 1987년에 시작한 이러한 표준의 공식화가 15가지 다른 파일 형식에 대한 조사를 거쳐 공식적으로 승인되었습니다.

## HDF5 파일 형식

H5 파일은 HDF5 파일 형식 사양을 준수하는 계층적 데이터 형식입니다. 이러한 사양은 디스크에 저장하기 위한 HDF5 파일의 전체 구조를 나타내지만 최종 사용자는 이러한 기본 세부 정보가 필요하지 않습니다.

H5 파일은 항공 우주, 물리학, 공학, 금융, 학술 연구, 유전체학, 천문학, 전자 기기 및 의료 분야에서 일반적으로 사용됩니다.

### HDF5 데이터 모델

HDF5 파일은 이기종 데이터 객체로 구성된 컨테이너와 같습니다. 예를 들어, 이러한 데이터 세트는 이미지, 테이블, 텍스트 요소, 그래프 및 [PDF](/ko/pdf/) 및 [Excel 문서](/ko/spreadsheet/)와 같은 문서일 수도 있습니다.

## H5 파일 뷰어

HDF 그룹은 HDFView 유틸리티와 Java HDF Object Package로 구성된 [HDFView 소프트웨어](https://www.hdfgroup.org/solutions/hdf5/)를 제공합니다.
