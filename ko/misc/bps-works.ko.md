## .BPS 파일이란?

.BPS 파일은 본질적으로 Microsoft Works로 작성된 워드 프로세싱 문서의 백업이며 일반적으로 문서에 .WPS 파일 확장자를 사용합니다. 이러한 .BPS 파일은 문서 작업 중 프로그램 충돌에 대비하여 Microsoft Works에서 자동으로 생성되는 경우가 많습니다. 이는 작업을 보호하고 데이터 손실을 방지하는 방법입니다.

다음 섹션에서는 BPS 파일과 관련된 소프트웨어 응용 프로그램을 살펴보겠습니다.

## 마이크로소프트 웍스

Microsoft Works는 Microsoft에서 개발한 생산성 소프트웨어 제품군이었습니다. 1987년에 처음 출시되었으며 전체 Microsoft Office 제품군에 대한 보다 저렴한 대안으로 인기를 얻었습니다. Microsoft Works에는 다음과 같은 몇 가지 기본 사무용 응용 프로그램이 포함되어 있습니다.

- **워드 프로세서:** Microsoft Works에는 사용자가 문서를 만들고 편집할 수 있는 워드 프로세싱 응용 프로그램이 있었습니다. 워드 프로세싱 문서에는 [.WPS](/ko/word-processing/wps/) 파일 형식을 사용했습니다.

- **스프레드시트:** 스프레드시트 생성 및 관리를 위한 스프레드시트 애플리케이션이 포함되어 있습니다. 스프레드시트의 파일 형식은 [.XLR](/ko/spreadsheet/xlr/)이었습니다.

- **데이터베이스:** Microsoft Works에는 데이터 구성 및 관리를 위한 데이터베이스 애플리케이션도 있었습니다. 데이터베이스 파일의 파일 형식은 [.WDB](/ko/database/wdb/)였습니다.

- **캘린더 및 주소록:** Microsoft Works에는 핵심 오피스 애플리케이션 외에도 캘린더 및 주소록 관리 도구가 포함되어 있는 경우가 많습니다.

- **Works 워드 프로세서 형식(.WPS):** Microsoft Works 워드 프로세싱 구성 요소의 기본 파일 형식입니다. .WPS 파일에는 텍스트, 서식 및 간단한 그래픽이 포함되어 있습니다.

- **중단:** Microsoft는 2007년에 공식적으로 Microsoft Works를 중단하고 사용자에게 Microsoft Office 또는 다른 대안으로 전환하도록 권장했습니다.
