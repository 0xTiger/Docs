## .H4 파일이란?

H4 또는 HDF4는 HDF(계층적 데이터 형식) 중 하나로 대용량 데이터를 저장하는 데 사용됩니다. 복잡한 이기종 데이터 저장을 지원하는 오픈 소스 파일 형식입니다. HDF4는 [HDF5](/ko/misc/h5/)에 비해 오래된 데이터 형식입니다. HDF 파일 형식은 비영리 단체인 HDF 그룹에서 유지 관리하고 게시합니다. HDF 파일은 또한 저장된 데이터를 완전히 나타내기 위해 메타데이터 저장을 지원합니다.

## HDF 파일 형식의 간략한 역사

계층적 데이터 형식의 공식화는 GFTF(Graphics Foundations Task Force)에 의해 1987년에 시작되었습니다. 이 기간 동안 단일 형식의 표준화를 위해 여러 가지 다른 파일 형식이 제안되었습니다. NASA는 과학 데이터를 저장하고 제공하기 위한 표준 방법으로 HDF를 선택했습니다.

## HDF4 파일 형식

H4 파일은 HDF 파일 형식 사양에 따라 구조와 같은 "파일 디렉토리"에 저장됩니다. 이러한 사양은 디스크에 저장하기 위한 HDF 파일의 전체 구조를 정의하지만 사용자는 기본 파일 구조에 대해 걱정하지 않습니다.

H4 파일의 응용 분야에는 항공 우주, 물리학, 공학, 금융, 학술 연구, 유전체학, 천문학, 전자 기기 및 의료 분야가 포함됩니다.

### HDF4 데이터 모델

HDF4 또는 H4 파일은 이기종 데이터 객체로 구성된 컨테이너와 같습니다. H4 파일에서 지원하는 데이터 세트 유형은 다음과 같습니다.

* 이미지
* 테이블
* 텍스트 요소
* 그래프
* [PDF](/ko/pdf/) 및 [Excel 문서](/ko/spreadsheet/) 등의 문서

## H4 파일 뷰어

HDF 그룹은 HDFView 유틸리티와 Java HDF Object Package로 구성된 [HDFView 소프트웨어](https://www.hdfgroup.org/solutions/hdf5/)를 제공합니다.
