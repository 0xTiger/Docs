## .YBK 파일이란? ##

YanCEyWare eBook 파일 형식과 Microsoft Encarta는 모두 YBK 파일 확장자를 사용합니다. YanCEyWare eBook은 YanCEyWare에 의해 형성되었습니다. 모든 소프트웨어 응용 프로그램은 Windows를 실행하는 장치에서 사용할 수 있으며 YBK 파일을 지원합니다. Encarta 백과사전은 전체 프로그램을 업데이트하지 않고도 .ybk 파일로 업데이트할 수 있습니다. YanCEyDesktop은 YBK 파일을 쉽게 관리하고 처리할 수 있는 유명하고 인기 있는 프로그램입니다. YanCEyWare Reader는 Windows Mobile 또는 Pocket PC용 eBook 리더 응용 프로그램으로 이 YBK 파일 형식을 광범위하게 사용합니다. 이 파일은 대규모 하이퍼링크 리소스로 개발되었습니다.

## YBK 파일을 여는 데 문제 ##

때때로 YBK 파일을 열고 작동하기 어려울 때 가능한 문제 목록은 다음과 같습니다.

* 손상된 YBK 파일 열기
* 레지스트리 액세스에서 YBK 파일에 대한 부적절하고 부적절한 링크
* Windows 아카이브에서 의도하지 않은 YBK 설명 제거
* YBK 형식을 제공하는 응용 프로그램의 부분 설치
*	Infected YBK file with an undesirable malware
