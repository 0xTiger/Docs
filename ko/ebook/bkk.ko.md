## .BKK 파일이란?

확장자가 .bkk인 파일은 **BookBuddi** 소프트웨어에서 사용되고 있으며 이 파일은 Microsoft TTS(텍스트 음성 변환) 에이전트를 사용하여 다시 읽을 수 있습니다. 이러한 파일에는 텍스트, 이미지가 포함될 수 있으며 여러 페이지로 구성될 수 있습니다. 텍스트를 읽을 때 해당 섹션이 강조 표시되고 페이지가 자동으로 넘깁니다.

## BKK 파일 형식

BKK 파일 형식의 내용은 사용자가 연결된 BKK 파일을 보거나 편집할 때 BookBuddi 앱에서 참조합니다. 이러한 참조 세부 정보에는 BKK 전자책 작성자가 입력한 다른 페이지 레이아웃 요소 중 텍스트 및 이미지 서식 속성도 포함됩니다. 시스템에 속한 응용 프로그램인 Microsoft Text to Speech Agent는 이 응용 프로그램이 eBook의 텍스트 내용을 음성 오디오로 다시 읽는 기능을 지원하기 때문에 사용자가 이러한 BKK 전자책에 저장된 디지털 콘텐츠에 액세스하는 데 도움이 됩니다.
