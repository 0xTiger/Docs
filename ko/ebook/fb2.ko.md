## .FB2 파일이란?

확장자가 .fb2인 파일은 eBook의 구조를 포함하는 FictionBook 2.0 eBook 파일입니다. XML 형식을 기반으로 하며 책의 각 요소를 설명하기 위한 특수 태그를 포함합니다. 주로 허구의 글과 문학을 위해 개발되었지만 이들에만 국한되지는 않습니다. 이 형식은 모든 메타데이터와 콘텐츠 자체를 수용하며 자동 처리, 인덱싱 및 다른 형식으로의 변환과 같은 다양한 작업에 유연성을 제공합니다. 간단히 말해서 파일의 모양을 지정하는 대신 파일의 구조를 설명하는 데 중점을 둡니다. Windows, MacOS 및 Linux에서 여러 응용 프로그램과 API를 사용하여 FB2를 여러 다른 형식으로 변환할 수 있습니다.

## FB2 파일 형식

FictionBook 2.0 파일 형식은 러시아 FictionBook의 Dmitry Gribov가 개발했습니다. 여러 개의 XML 파일을 포함하는 [EPUB](/ko/ebook/epub/)와 같이 널리 사용되는 다른 전자책 파일 형식과 달리 콘텐츠에 대한 완전한 정보를 포함하는 단일 XML 파일을 기반으로 합니다. [PNG](/ko/image/png/) 또는 [JPG](/ko/image/jpeg/)와 같은 파일 내부에 이미지가 있는 경우 Base64로 변환되어 문서 내부에 포함됩니다. FB2에 대한 [파일 형식 사양](http://gribuser.ru/xml/fictionbook/index.html.en)에는 형식의 버전 2.0에 대한 정보가 포함되어 있습니다.
