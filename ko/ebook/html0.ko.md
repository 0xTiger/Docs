## HTML0 파일이란 무엇입니까? ##

HTML0은 **Book Designer**에서 사용하는 기본 eBook 형식으로, 각 페이지의 콘텐츠와 디자인을 설명하는 표준 HTML 및 CSS 코드가 있습니다. **Book Designer**는 HTML0 파일과 함께 작동하고 텍스트 문서를 다양한 eBook 형식으로 변경하는 가장 일반적인 프로그램입니다. HTML0 파일 형식은 텍스트 파일 범주에 속하므로 Windows 시스템에 설치할 수 있는 소프트웨어를 지원합니다. HTML0 파일은 일반 HTML로 구성되며 텍스트 편집기를 사용하여 보고 수정할 수 있습니다. HTML0 파일은 일반 HTML 파일과 비교하여 개별 CSS 파일에 연결할 수 없습니다.

## HTML0 파일을 여는 데 문제가 있습니까? ##

* 파일을 열기 어려울 때 감염될 수 있습니다.
* 온라인 안티바이러스 스캐너로 파일에 바이러스가 있는지 검사해야 함
* 일부 파일을 열려면 액세스 권한이 필요합니다.
* 시스템에 HTML0 파일을 열 수 있는 리소스가 충분하지 않은 경우 모든 응용 프로그램을 닫았다가 다시 엽니다.
* 파일을 다운로드했거나 이메일에서 받은 경우 파일이 손상되었거나 오류가 있을 수 있습니다.
* 항상 최신 시스템과 드라이버를 사용하여 컴퓨터를 더 안전하게 만들고 파일 확장자와 관련된 모든 문제를 해결하십시오.
