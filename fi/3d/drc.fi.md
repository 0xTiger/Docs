## Mikä on DRC-tiedosto?

Tiedosto, jonka laajennus on .drc, on pakattu 3D-tiedostomuoto, joka on luotu Google Draco -kirjastolla. Google tarjoaa Dracon avoimen lähdekoodin kirjastona 3D-geometristen verkkojen ja pistepilvien pakkaamiseen ja purkamiseen sekä parantaa 3D-grafiikan tallennusta ja siirtoa. Se koodaa syöttötiedot ja tallentaa sen .drc-tiedostona. Vastaanottopäässä API lukee .drc-tiedostoja ja voi tulostaa ne [PLY](/3d/ply/)- tai [OBJ](/3d/obj/)-tiedostoina. Pakatun tulostustiedoston avulla käyttäjät voivat ladata sovelluksia nopeammin ja ladata 3D-grafiikkaa nopeasti selaimissa.

## DRC tiedostomuoto

File format details for DRC are not available publicly. Google has hosted the Draco library on [GitHub](https://github.com/google/draco/tree/master/docs) for developer's reference. It provides encoders and decoders for creating and converting DRC files.
