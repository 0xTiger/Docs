## Mikä on LZH-tiedosto? ##

A file with .lzh and .lha extension usually relates to archive compression file format. This file format is the same as other file compression formats like [ZIP](/compression/zip/), [RAR](/compression/rar/), etc. The main purpose of these file formats is to reduce the size of the format for easily sending as well as to keep them together in compressed form. AS compare to other western companies LZH files are very popular in Japan and usually used for compressing video game data. The LZH add-on for Windows 7 is built into the Japanese version of Windows 7. Microsoft julkaisi ensimmäisen kerran Microsoft Compressed (LZH) Folder -lisäosan Windows XP:n japanilaiselle versiolle. Tämä tiedostomuoto on toteutettu Lempel-Ziv- ja Haruyasu-algoritmin käyttämillä pakkaussäännöksillä ja ominaisuuksilla

## LZH-määritys ##

LZH-arkiston pakkausmenetelmä esitetään viisitavuisena tekstijonona, kuten -lz1-. Nämä ovat pakatun tiedoston kolmannesta seitsemään tavua.

|Spesifikaation|Kuvaus|
---|---|
|Laajennus | .lzh, .lha|
|Mediatyyppi| application/x-lzh-compressed|
|Tyyppikoodi| LHA_ (LHA-VÄLILYÖNTI)|
|UTI| public.archive.lha|
|Kehittäjä| Haruyasu Yoshizaki (Yoshi)|
|Tyyppi| Kompressio|
|Jatkettu alkaen| LArc|

## Ongelmia LZH-tiedoston avaamisessa ##

Seuraavassa on muutamia ongelmia, jotka voivat aiheuttaa LZH-tiedostomuodon huonoa toimintaa:
  
* Tiedosto saattaa olla vioittunut. Voit ratkaista tämän ongelman lataamalla tiedoston uudelleen tai pyytämällä lähettäjää lähettämään tiedoston uudelleen
* Toinen syy on tartunnan saanut tiedosto, tässä tapauksessa skannaa tiedosto oikein
* Väärät linkit LZH-tiedostoon
  *	 Removal of the description of the LZH 
* Ei tarpeeksi laitteistoresursseja
* Laitteen vanhentuneet ajurit
