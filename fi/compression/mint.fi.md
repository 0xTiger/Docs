## Mikä on MINT-tiedosto?

MINT-tiedosto on [Linux Mint](https://linuxmint.com/)-jakelun käyttämä asennuspakettitiedosto. Se sisältää pakattuja lähdekooditiedostoja, jotka puretaan ja joita käytetään Linux Mint -käyttöjärjestelmän asennuksen käynnistämiseen. Linux Mint -käyttöjärjestelmässä on helppokäyttöinen GUI (Graphical User Interface) ja se oli ensimmäinen, jolla oli tyylikäs graafinen suunnittelu. MINT-tiedostot puretaan ja tallennetaan väliaikaiseen hakemistoon asennuspaketin mukana. Asennus suoritetaan samasta hakemistosta kuin Linux Mint OS asennetaan.

## Linux MINT -tiedostomuoto

MINT-tiedostomuodon sisäisen tiedostomuodon tiedot eivät ole saatavilla julkisesti. MINT-tiedostoihin tallennetut pakatut asennustiedostot ovat kuitenkin Linux Mint OS -asennusohjelmiin integroitujen pakkausstandardien mukaisia. Tämä mahdollistaa näiden tiedostojen purkamisen ja tulla osaksi asennusta.
