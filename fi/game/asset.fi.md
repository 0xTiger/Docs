## Mikä on ASSET-tiedosto?

ASSET-tiedosto on erityinen tiedosto, jota käytetään Unityssa, videopelien tekemiseen tarkoitetussa ohjelmassa. Se sisältää tärkeitä kokoonpanoja, kuten peliobjekteja tai asetuksia. Esimerkiksi yksi ASSET-tiedosto voi tallentaa kuinka esineet liikkuvat pelissä, kuten kuinka ne pomppivat toisistaan tai putoavat alas. Nämä tiedostot ovat kuin säilöjä, jotka pitävät kaikki tärkeät tiedot järjestyksessä, jotta pelien kehittäjät voivat helposti työstää projektejaan.

## Kuinka avata ASSET-tiedosto
  
Unityssa et tavallisesti avaa resurssitiedostoja samalla tavalla kuin avaisit asiakirjan tai tiedoston tekstieditorissa tai vastaavassa ohjelmassa. Sen sijaan Unity hallitsee omaisuustiedostoja itse Unity Editorissa. Näin pääset käsiksi Unityn sisältötiedostoihin ja työskentelet niiden kanssa:

1.  **Import Assets**: To use an asset file in your Unity project, you need to import it into project. You can do this by either dragging and dropping asset file directly into Unity Editor or by using "Assets" menu and selecting "Import New Asset."
    
2.  **Osauksien katseleminen**: Tuonnin jälkeen voit tarkastella ja hallita resurssitiedostoja Unity Editorin Projekti-ikkunassa. Tämä ikkuna näyttää kaikki projektisi resurssit, ja voit järjestää ne kansioihin, etsiä tiettyjä resursseja ja esikatsella niiden sisältöä.
    
3.  **Osauksien käyttäminen**: Voit käyttää resurssitiedostoja Unity-projektissasi vetämällä ne Projekti-ikkunasta kohtaushierarkiaan tai kohtauksen GameObjects-kohtaan. Voit esimerkiksi vetää pintakuvioresurssin materiaaliin soveltaaksesi sitä 3D-objektiin, tai voit vetää esivalmistetun resurssin kohtaukseen luodaksesi esikonfiguroidun GameObjectin.
    
4.  **Osauksien muokkaaminen**: Joitakin sisältötiedostoja, kuten skriptejä, materiaaleja ja animaatioita, voidaan muokata suoraan Unity Editorissa. Voit kaksoisnapsauttaa näitä resursseja Projekti-ikkunassa avataksesi ne vastaavissa muokkausohjelmissaan ja tehdäksesi muutoksia tarpeen mukaan.
    
5.  **Saving Assets**: Kaikki Unity Editorissa resurssitiedostoihin tekemäsi muutokset tallennetaan automaattisesti Unity-projektiisi. Yksittäisiä resurssitiedostoja ei tarvitse erikseen tallentaa samalla tavalla kuin dokumenttien kanssa perinteisessä ohjelmistossa.

ASSET-tiedostoja voidaan avata tai niihin voi viitata seuraavilla ohjelmilla.

- **Unity Technologies Unity** (ilmainen) (Windows, Mac, Linux)
