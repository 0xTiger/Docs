## Mikä on AMXX-tiedosto?

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. Se on komentosarjaalusta, jonka avulla palvelimen järjestelmänvalvojat voivat mukauttaa ja laajentaa palvelimiensa toimintoja. AMXX:ää käytetään yhdessä MetaMod-laajennusten hallintajärjestelmän kanssa, joka mahdollistaa useiden liitännäisten suorittamisen samanaikaisesti palvelimella ilman ristiriitoja. AMXX-laajennuksen tarjoamia ominaisuuksia ovat järjestelmänvalvojan komennot, pelin muutokset ja palvelimen hallinta.
