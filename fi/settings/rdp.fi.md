## Mikä on RDP-tiedosto?

.rdp-tiedostotunnistetta käytetään Remote Desktop Protocol (RDP) -tiedostoille. RDP on Microsoftin kehittämä protokolla, jonka avulla käyttäjät voivat muodostaa yhteyden etätietokoneeseen verkkoyhteyden kautta. RDP-tiedostot sisältävät määritysasetuksia, joita käytetään etätyöpöytäistunnon muodostamiseen tietyn tietokoneen kanssa.

Kun kaksoisnapsautat RDP-tiedostoa, se käynnistää tavallisesti Remote Desktop Connection -ohjelman tietokoneellasi ja muodostaa yhteyden tiedostossa määritettyyn etätietokoneeseen. RDP-tiedostoja voidaan luoda manuaalisesti tekstieditorilla tai luoda automaattisesti Remote Desktop Connectionilla tai muulla kolmannen osapuolen ohjelmistolla.

RDP-tiedostot voivat sisältää erilaisia määritysasetuksia, kuten etätietokoneen isäntänimen tai IP-osoitteen, etäistunnon käyttäjätunnuksen ja salasanan, näyttöasetukset ja paljon muuta. Käyttämällä RDP-tiedostoa käyttäjät voivat nopeasti ja helposti muodostaa yhteyden etätietokoneisiin ilman, että heidän tarvitsee manuaalisesti määrittää kaikkia näitä asetuksia aina, kun he haluavat muodostaa yhteyden.

## RDP-tiedostomuoto - lisätietoja

RDP file (Remote Desktop Protocol file) contains configuration settings that are used to connect to remote computer using the Remote Desktop Connection program or third-party RDP client. The configuration settings in RDP file may include:

- Sen etätietokoneen isäntänimi tai IP-osoite, johon haluat muodostaa yhteyden.
- Käyttäjätunnus ja salasana, joita käytät kirjautuessasi sisään etätietokoneeseen.
- Etätietokoneen toimialue tai työryhmä tarvittaessa.
- Näytön resoluutio ja värisyvyysasetukset etätyöpöytäistunnon aikana.
- Etätyöpöytäistunnon äänen ja videon toistoasetukset.
- Tulostimen ja leikepöydän uudelleenohjausasetukset etätyöpöytäistuntoa varten.
- Etätyöpöydän koko ja sijainti paikallisen tietokoneen näytöllä.
- Etätyöpöytäistunnon aikavyöhyke- ja kieliasetukset.
- Kaikki mukautetut määritysasetukset tai etätyöpöytäistunnon asetukset.

Kun avaat RDP-tiedoston, Remote Desktop Connection -ohjelma lukee kokoonpanoasetukset ja käyttää niitä yhteyden muodostamiseen etätietokoneeseen. Tämän avulla voit käyttää ja ohjata etätietokonetta paikalliselta tietokoneeltasi ikään kuin istuisit sen edessä.

## Mikä on RDP-tiedoston muoto?

RDP-tiedosto (Remote Desktop Protocol -tiedosto) on tekstitiedosto, joka sisältää kokoonpanoasetukset, joita käytetään etätyöpöytäistunnon muodostamiseen tietyn tietokoneen kanssa. Se on yksinkertainen tekstitiedosto, joka voidaan luoda tai muokata tekstieditorilla, kuten Notepadilla tai millä tahansa muulla tekstieditorisovelluksella.

RDP-tiedostomuoto koostuu sarjasta avain-arvopareja, joissa jokainen avain edustaa tiettyä konfigurointiasetusta ja vastaava arvo määrittää kyseisen asetuksen arvon. Muoto on samanlainen kuin INI-tiedostossa, jonka osat on erotettu hakasulkeilla ja yksittäiset avainarvoparit erotettuina yhtäläisyysmerkillä.
