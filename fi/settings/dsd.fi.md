## Mikä on DSD-tiedosto?
AutoCAD Drawing Set Description (DSD) -tiedosto on Autodeskin AutoCAD-ohjelmiston käyttämä tekstitiedostomuoto, joka sisältää tietoja piirustussarjasta. Nämä tiedot sisältävät yksittäisten piirustusten nimet ja polut sekä erilaisia asetuksia ja metatietoja.

DSD-tiedoston luomiseen käyttäjät voivat käyttää AutoCAD Batch Plot -apuohjelmaa, jonka avulla he voivat valita sarjaan sisällytettävät piirustukset ja määrittää erilaisia vaihtoehtoja ja asetuksia. Kun eräkaavio on valmis, DSD-tiedosto luodaan automaattisesti ja tallennetaan piirustusten rinnalle.

DSD-tiedoston avulla käyttäjät voivat kätevästi hallita ja järjestää useita piirustustiedostoja joukoksi, mikä yksinkertaistaa niiden tulostusta, julkaisua tai piirtämistä yhteen. Sen avulla käyttäjät voivat myös tallentaa ja hakea erilaisia asetuksia ja kokoonpanoja. Lisäksi sen avulla käyttäjät voivat luoda eräkaavion kaikista sarjan piirustuksista.

## Kuinka avata DSD-tiedosto?
Käyttäjät voivat avata DSD-tiedoston käyttämällä AutoCAD Batch Plot -apuohjelmaa tai käyttää Avaa-komentoa AutoCADissa ja valita DSD-tiedoston. Kun DSD-tiedosto on avattu, käyttäjät voivat tarkastella ja muokata sen sisältämiä tietoja, kuten piirustusten nimiä, polkuja ja asetuksia.

## Kuinka muokata DSD-tiedostoa AutoCADissa?
DSD-tiedoston muokkaaminen AutoCADissa on suhteellisen yksinkertainen prosessi. Aloita avaamalla AutoCAD ja siirtymällä nauhan Julkaise-välilehteen. Napsauta sieltä Sheet Set Manager avataksesi Sheet Set Manager -paletin.

Etsi muokattava DSD-tiedosto Sheet Set Manager -paneelista ja napsauta sitä hiiren kakkospainikkeella. Näyttöön tulevasta kontekstivalikosta voit valita Muokkaa taulukkojoukon ominaisuuksia.

Tämä käynnistää Sheet Set Properties -valintaikkunan, jossa on useita välilehtiä, joissa on erilaisia DSD-tiedostojen muokkausvaihtoehtoja. Voit lisätä tai poistaa tiettyjä piirustuksia joukosta esimerkiksi Sheet List -välilehden avulla ja voit mukauttaa erilaisia asetuksia tulostusta, julkaisua tai joukossa olevien piirustusten vientiä varten Julkaisuasetukset-välilehdellä.

Tee tarvittavat muutokset DSD-tiedostoon Sheet Set Properties -valintaikkunan eri välilehtien avulla ja napsauta OK tallentaaksesi muutokset. DSD-tiedostosi pitäisi nyt vastata määrittämiäsi päivitettyjä asetuksia ja määrityksiä.
