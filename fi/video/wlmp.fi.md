## Mikä on WLMP-tiedosto?

WLMP (Windows Live Movie Maker Project) on patentoitu tiedostomuoto, jota Windows Live Movie Maker -sovellus käyttää elokuvaprojektien tallentamiseen. Se sisältää tietoa videoleikkeiden, valokuvien ja ääniraitojen järjestelystä projektissa sekä niihin sovelletuista tehosteista ja siirtymistä. WLMP-tiedostoja ei voi toistaa mediasoittimilla, vaan ne on vietävä videotiedostomuotoon, kuten WMV tai MP4, jotta niitä voidaan toistaa.

WLMP-tiedoston todellinen sisältö on tallennettu XML-muodossa (Extensible Markup Language) ja se sisältää tietoja projektista, kuten videon koon ja pituuden, kunkin medialeikkeen sijainnin ja keston, käytetyt siirtymät ja tehosteet sekä muut projektiin liittyvät tiedot. WLMP-tiedoston XML-koodi kuvaa elokuvaprojektin tapahtumasarjaa ja toimii suunnitelmana lopulliselle videotuotannolle. Itse WLMP-tiedosto ei kuitenkaan sisällä varsinaisia mediatiedostoja, kuten videoleikkeitä ja ääniraitoja, vaan viittauksia niiden sijaintiin tietokoneen kiintolevyllä.

## Vie WLMP WMV- tai MP4-muotoon

To upload a WLMP file to YouTube, you need to first export it as a video file format that is compatible with YouTube, such as WMV or MP4. Voit tehdä tämän tallentamalla elokuvaprojektin videotiedostona Windows Live Movie Maker -sovelluksella.

Tässä on yleiset vaiheet WLMP-tiedoston viemiseksi WMV- tai MP4-muotoon:

1. Avaa Windows Live Movie Maker ja lataa WLMP-projekti.
2. Siirry Koti-välilehdelle ja napsauta Tallenna elokuva.
3. Valitse Tietokoneelle ja valitse sitten tiedostomuodoksi WMV tai MP4.
4. Tallenna elokuvaprojekti videotiedostona noudattamalla näytön ohjeita.

## Suhde Windows Live Movie Makeriin

WLMP (Windows Live Movie Maker Project) on tiedostomuoto, jota käyttää erityisesti Windows Live Movie Maker -sovellus, joka on Microsoftin Windows-käyttöjärjestelmille kehittämä ilmainen videonmuokkausohjelmisto. Se tarjoaa käyttäjäystävällisen käyttöliittymän elokuvien luomiseen, muokkaamiseen ja jakamiseen. Sovelluksen avulla käyttäjät voivat tuoda ja muokata videoleikkeitä, valokuvia ja ääniraitoja sekä lisätä erilaisia elementtejä, kuten tekstiä, tehosteita ja siirtymiä.

## Kuinka avata WLMP -tiedosto?

WLMP-tiedoston avaamiseksi sinun on käytettävä Windows Live Movie Maker -sovellusta. Voit avata WLMP-tiedoston seuraavasti:

1. Avaa Windows Live Movie Maker
2. Siirry Home-välilehdelle ja napsauta Avaa projekti.
3. Valitse WLMP-tiedosto, jonka haluat avata.
4. Projekti ladataan Windows Live Movie Makeriin, ja voit nyt muokata tai viedä elokuvaprojektia.

Huomaa, että WLMP-tiedosto ei ole videotiedosto, vaan projektitiedosto, joka sisältää tietoa videoleikkeiden, valokuvien ja ääniraitojen järjestelystä elokuvaprojektissa. Sinun on käytettävä Windows Live Movie Makeria projektin muokkaamiseen ja sen viemiseen videotiedostomuodossa, kuten WMV tai MP4, toistaaksesi mediasoittimissa.
