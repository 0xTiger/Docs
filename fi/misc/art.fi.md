## Mikä on ART-tiedosto?

An ART file is an embroidery design file created with the [BERNINA](https://www.bernina.com/en-US/Home-United-States) ARTlink embroidery software version 1-4. Se sisältää käyttäjien luomia malleja, jotka voivat sisältää erilaisia objekteja, kuten ääriviivoja, suunnittelun asettelua, ompeleita, värillisiä lankoja ja muita metatietoja. ART-tiedosto ladataan BERNINA-ompelukoneeseen ja voidaan ommella erilaisiin kankaisiin, kuten peittoon, paitaan tai mihin tahansa muuhun kankaaseen. ART-tiedostoja voidaan myös jakaa muiden käyttäjien kanssa käyttämällä tallennuslaitteita, kuten USB:tä tai siirtää Internetin kautta.

## ART-tiedostomuoto

ART-tiedostomuoto sisältää seuraavat tiedot:

 * Objektin ominaisuudet
 * Ommeltiedot
 * Ommeltyypit
 * Ompele koordinaatit
 * Langan värit
 * Kommentit
 * Vetokompensointiasetukset
 * Automaattinen väliasetukset
