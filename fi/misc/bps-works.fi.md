## Mikä on BPS-tiedosto?

.BPS-tiedosto on pohjimmiltaan varmuuskopio tekstinkäsittelydokumentista, joka on luotu Microsoft Worksilla, joka käyttää tyypillisesti .WPS-tiedostotunnistetta asiakirjoissaan. Microsoft Works luo usein nämä .BPS-tiedostot automaattisesti varotoimenpiteenä siltä varalta, että ohjelma kaatuu dokumentin käsittelyn aikana. Se on tapa suojata työsi ja estää tietojen häviäminen.

Seuraavassa osiossa tutkimme BPS-tiedostoihin liittyviä ohjelmistosovelluksia.

## Microsoft toimii

Microsoft Works oli Microsoftin kehittämä tuottavuusohjelmistopaketti. Se esiteltiin ensimmäisen kerran vuonna 1987 ja saavutti suosion edullisempana vaihtoehtona koko Microsoft Office -ohjelmistopaketille. Microsoft Works sisälsi useita toimiston perussovelluksia, kuten:

- **tekstinkäsittelyohjelma:** Microsoft Worksissa oli tekstinkäsittelysovellus, jonka avulla käyttäjät voivat luoda ja muokata asiakirjoja. Se käytti [.WPS](/word-processing/wps/)-tiedostomuotoa tekstinkäsittelyasiakirjoihin.

- **Laskentataulukko:** Se sisälsi laskentataulukkosovelluksen laskentataulukoiden luomiseen ja hallintaan. Laskentataulukoiden tiedostomuoto oli [.XLR](/spreadsheet/xlr/).

- **Tietokanta:** Microsoft Worksilla oli myös tietokantasovellus tietojen järjestämiseen ja hallintaan. Tietokantatiedostojen tiedostomuoto oli [.WDB](/database/wdb/).

- **Kalenteri ja osoitekirja:** Ydintoimistosovellusten lisäksi Microsoft Works sisälsi usein työkaluja kalentereiden ja osoitekirjojen hallintaan.

- **Works Word Processor Format (.WPS):** Microsoft Worksin tekstinkäsittelykomponentin alkuperäinen tiedostomuoto. .WPS-tiedostot sisälsivät tekstiä, muotoilua ja yksinkertaista grafiikkaa.

- **Lopettaminen:** Microsoft lopetti virallisesti Microsoft Worksin vuonna 2007 ja rohkaisi käyttäjiä siirtymään Microsoft Officeen tai muihin vaihtoehtoihin.
