## Mikä on SSP-tiedosto?

SSP-tiedosto on projektitiedosto, jonka on luonut Windowsin näytönsäästäjien luontiohjelmisto, joka tunnetaan nimellä Screensaver Producer by Axialis Software. Se voi sisältää useita erityyppisiä näytönsäästäjäelementtejä, kuten spritejä, Flash-elokuvia, valokuvadiaesityksiä ja videoita. SSP-tiedosto voi sisältää tekstiä, kuvia, erilaisia muotoja ja animaatioasetuksia, joita käytetään näyttämään animaatio SSP-tiedoston sisällä. SSP-tiedostoja voidaan avata Axialis Screensaver Producerilla.

## SSP-tiedostomuoto - lisätietoja

SSP files are stored to disc as binary files with no public information available about their internal file format structure. Axialis Screensaver producer professional screensaver authoring tool for Windows which lets you easily create hi-quality screensavers for all versions of Windows up to Windows 10. Sen avulla voidaan luoda asennettavia näytönsäästäjäpaketteja, jotka voidaan helposti jakaa verkossa käyttäjille.

SSP-tiedostot voidaan viedä [SCR](/system/scr/)-tiedostomuotoon.
