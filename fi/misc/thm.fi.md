## Mikä on THM-tiedosto?

THM-tiedosto on Sony Ericsson -matkapuhelimille luotu teematiedosto. Sitä käytetään loppukäyttäjän graafisen käyttöliittymän vaihtamiseen useilla asetusparametreilla. Näitä ovat tekstin värien, kuvakkeiden, taustakuvan, mukautetun ikkunan ja muiden kuvien asetukset. Sony on toimittanut ilmaisen Ericsson Themes Creator -ohjelmiston teematiedostojen luomiseen. Sitä voidaan käyttää myös olemassa olevien THM-tiedostojen avaamiseen. THM-tiedostoilla on mediatyyppi application/vnd.eri.thm.

## Sony Ericsson THM -tiedostomuoto - lisätietoja

THM-tiedosto on itse asiassa säilötiedosto, joka sisältää joukon muita tiedostoja. Se on tallennettu [TAR](/compression/tar/)-arkistotiedostomuotoon, ja se voidaan purkaa millä tahansa tavallisella purkuohjelmistolla, joka tukee TAR-tiedostomuotoa.

A THM file contains an XML file, several image file sin [.png](/image/png/), [.pjg](/image/jpeg/), [.gif](/image/gif/) formats, or [.swf](/page-description-language/swf/) files.

## Kuinka luoda Sony Ericsson -teemoja?

Sony Themes Creator -ohjelmistolla voidaan luoda uusia teemoja Sony Ericsson -puhelimiin. Käyttäjät voivat luoda teemoja seuraamalla Sonyn [Themes Creator Developer Guide](https://developer.sony.com/theme-creator/get-started).
