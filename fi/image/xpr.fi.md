## Mikä on XPR-tiedosto?

XPR-tiedosto on vektorikuvatiedosto, joka on luotu nyt lopetetulla Microsoft Expression Graphics (EGD) -ohjelmistolla. Se sisältää grafiikkaa, jota voidaan käyttää käyttöliittymämallina. XPR-tiedostot korvattiin myöhemmin .design-tiedostoilla Microsoft Expression Designissa. XPR-tiedostoja voitiin avata Microsoft Expression Studiolla, joka on ollut pois käytöstä jo jonkin aikaa.

## XPR-tiedostomuodon haavoittuvuus

XPR-tiedostojen todettiin hyödyttävän Microsoft Expression Designin haavoittuvuutta, mikä mahdollistaa etäkoodin suorittamisen. Raportoidussa ongelmassa, jos käyttäjä avasi laillisen .xpr-tiedoston, joka on päällystetty verkkohakemistossa DLL-tiedoston kanssa, Microsoft Expression Design saattaa yrittää ladata DLL-tiedoston ja suorittaa sen sisältämän koodin. Microsoft korjasi tämän myöhemmin tietoturvapäivityksessä.
