## Mikä on HEIF-tiedosto?

An HEIF file is a High-Efficiency Container Image file format that is capable of storing a single image or a sequence of images in a single file. It, thus, is basically a container having multiple images stored. HEIF file format compresses the images using the High Efficiency Video Coding (HEVC) standard. It is is more efficient and light weight as compared other image file formats such as [JPEG](/image/jpeg/). A single HEIF file can store sequences of images such as [GIF](/image/gif/) animations or HDR sequence, along with metadata that describes each image. Apple Inc. adopted its variant HEIC file format for their iOS 11. Tämä sai muut yritykset tukemaan sitä ohjelmistossaan, mukaan lukien Adobe.

## HEIF-tiedostomuoto

The HEIF format was developed by Moving Picture Experts Group (MPEG) and was registered under the ISO/IEC 23008-12 standard in 2015. Useat valmistajat, kuten Apple, Microsoft ja Canon, ottivat sen käyttöön. HEIF-tiedostot tallennetaan .heif-tunnisteella (jos se sisältää yhden kuvan) tai .heif-tiedostoina (jos se sisältää kuvasarjan).

### HEIF-tietojen tallennus

HEIF-muoto voi tallentaa seuraavan tyyppisiä kuvia ja tietoja.

 * Yksittäiset kuvat ja kuvien pikkukuvat
 * Kuvien jaksot
 * Jälkikäsittelyohjeet
 * Kuvan metatiedot, kuten [EXIF](/image/exif/)-tiedot

HEIF-kuvamuoto mahdollistaa useiden eri kuvamuotojen, mukaan lukien [JPEG](/image/jpeg/), [PNG](/image/png/) ja [GIF](/image/gif/), kuvanlaadusta tinkimättä.
