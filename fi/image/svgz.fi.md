## Mikä on SVGZ-tiedosto?

Tiedosto, jonka laajennus on .svgz, on pakattu versio Scalable Vector Graphics (.SVG) -tiedostosta. Se on pakattu gzip-pakkauksella ja sisältää tiedot muodossa [XML](/web/xml/). SVGZ-tiedostot tukevat läpinäkyvyyttä, liukuvärejä, animaatioita ja suodattimia. SVGZ-tiedostot ovat kooltaan pienempiä kuin oletusarvoiset SVG-tiedostot, ja tämä pienempi tiedostokoko auttaa siirtämään grafiikkatiedostoja verkossa. Graafinen suunnittelija luo SVGZ-tiedostoja käyttämällä työkaluja, kuten Adobe Illustrator, Corel PaintShop Pro ja muut. SVGZ-tiedostoja voidaan kuitenkin luoda ottamalla käyttöön [GZip compression](/compression/gz/) Apache-palvelimessa kuvatietojen lähettämisen aikana.

## SVGZ-tiedostomuoto

SVGZ, being compressed [SVG](/page-description-language/svg/), is based on open compressed gzip format that reduces the size of SVG documents. This compressed archive contains plain text SVG documents inside that are purely based on the XML file format (/web/xml/). SVGZ files can be opened by viewers/applications that conform to the SVG 1.1 specifications. SVGZ files 20-50% smaller in size than the original SVG files.
