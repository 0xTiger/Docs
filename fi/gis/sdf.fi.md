## Mikä on SDF-tiedosto?

A Spatial Data File (SDF) is a Geodatabase file format that was developed by Autodesk. It stores geospatial data and is used by Autodesk GIS program MapGuide and AutoCAD Map 3D. SDF file format is based on the single file database file format SQLite3. Sitä pidetään optimoituna tiedostomuotona suurille paikkatietoaineistoille.

Voit avata SDF-tiedoston Autodesk AutoCAD Map 3D 2022:lla ja Autodesk AutoCAD Civil 3D 2023:lla.

## SDF-tiedostomuoto – lisätietoja

SDF-tiedostomuodot tallennetaan levylle binääritiedostoina. Se perustuu SQLite-tietokantaan, joka hyödyntää matalan tason tallennuskomponentteja tietojen binaarista sarjoittamista varten. SDF-tiedosto sisältää useita ominaisuusluokkia tiedostoa kohden ja useita geometrian ominaisuuksia kullekin ominaisuusluokalle. SDF-tiedoston tiedot ovat käytettävissä suurella nopeudella, koska jokainen geometriaominaisuus indeksoidaan R-puun avulla.
