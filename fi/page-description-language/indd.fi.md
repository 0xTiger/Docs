## Mikä on INDD-tiedosto?

Tiedosto, jonka tunniste on .indd, on Adobe InDesign -projektitiedosto, joka sisältää projektin sivuasettelutiedot. INDD on lyhenne sanoista InDesign Document. Nämä tiedostot tallentavat yleensä sivun muotoilutietoja, kirjoja, aikakauslehtiä, sanomalehtiä, lentolehtisiä, esitteitä tai muuta painettua mediaa, kuten mobiili- ja verkkosivumalleja. INDD-tiedostotyyppi on Adoben oma tiedostomuoto ja InDesignin oletustiedostomuoto. InDesignin kanssa työskentelevät graafiset suunnittelijat, toimittajat, markkinoijat ja julkaisijat käyttävät tätä tiedostotyyppiä.

## Lisätietoja INDD-tiedostomuodosta

INDD-tiedostot ovat vektoritiedostoja, joita voidaan skaalata resoluutiota menettämättä. Nämä perustuvat monimutkaisiin kaaviomaisiin, joita voidaan laajentaa loputtomasti. Se pakkaa kaikki fontit ja asettelutiedot jaettavaksi tiimin kanssa.

INDD-tiedostot sopivat erinomaisesti kiinnostavan printti- ja verkkosisällön suunnitteluun. INDD-tiedosto voi ulottua useille sivuille ja se voi toimia malleina sivujen, lukujen ja julkaisujen luomiseen. Kun projektisi on valmis, se voidaan julkaista online-mediassa ilman muuntamista, pakkaamista tai sähköpostitse lähettämistä.

## Voidaanko INDD-tiedosto muuntaa muihin muotoihin?

INDD files can be converted to several other file formats. The most common conversion is INDD to PDF. InDesign lets you save the INDD file to PDF for sharing with other users. It also lets you export INDD file to JPG, [EPS](/page-description-language/eps/), [EPUB](/ebook/epub/), [SWF](/page-description-language/swf/), FLA, [HTML](/web/html/), [XML](/web/xml/), and IDML.

## Kuinka INDD-tiedosto luodaan?

An Indd file is created when you create a new project with [Adobe InDesign](https://www.adobe.com/products/indesign.html) and save it to disc. You can open the Indd project file, make changes to it, and save the updated file for the changes.
