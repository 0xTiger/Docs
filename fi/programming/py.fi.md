## Mikä on PY-tiedosto? ##

.py-laajennuksella varustetut tiedostot sisältävät Python-lähdekoodin. Python-kielestä on tullut nykyään hyvin kuuluisa kieli. Sitä voidaan käyttää järjestelmän komentosarjaan, web- ja ohjelmistokehitykseen sekä matematiikkaan. Python tukee alustojen välistä yhteensopivuutta; tarkoittaa, että Pythonissa kehitetyt sovellukset voivat toimia eri alustoilla, kuten Windows, MAC, Linux, Raspberry Pi jne. Python tarjoaa yksinkertaisen ja helposti luettavan syntaksin, joka muistuttaa englannin kieltä. Kehittäjät voivat kirjoittaa järkevän ohjelmistosovelluksen kirjoittamalla muutaman rivin Python-koodia. Koska Python toimii tulkkijärjestelmässä, koodi voidaan suorittaa heti, kun se on kirjoitettu, mikä tekee siitä erittäin hyvän prototyyppien tekemiseen.

## Lyhyt historia ##

Python was conceived in the late 1980s by Guido van Rossum as a successor to ABC programming language. Its implementation began in December 1989 with Van Rossum as the sole lead developer. He worked on python until 12 July 2018. Tammikuussa 2019 aktiiviset Python-ydinkehittäjät valitsivat projektia johtamaan viisijäsenisen ohjausneuvoston, johon kuuluivat Nick Coghlan, Brett Cannon, Carol Willing, Barry Warsaw ja Van Rossum.

## Kuinka ajaa py-tiedosto ##

Voit tarkistaa asennetun Python-version käyttämällä seuraavaa komentoa:

```cmd
python --version
```

Tämä tulostaa version konsolissa kuten

```cmd
Python 3.7.4
```

Jos koneellesi ei ole asennettu Pythonia, voit siirtyä osoitteeseen [python.org](https://www.python.org/) ja ladata ja asentaa Pythonin käyttöjärjestelmääsi varten.

Python-komentosarjan suorittamiseksi voit käyttää seuraavaa komentoa:

```cmd
python helloworld.py
```

*helloworld.py* on komentosarjatiedosto, joka sisältää seuraavan koodin

```py
print("Hello World from Python")
```

Tämä tulostaa seuraavat tiedot konsoli-ikkunaan.

```cmd
Hello World from Python
```

Huomautus: Jos käytät IDE:itä, ne tarjoavat näytöllä painikkeita tai erilaisia pikanäppäimiä Pythonin suorittamista varten. Esimerkiksi PyCharmin editorin kourussa näkyy toistopainike, jonka avulla voit suorittaa Python-komentosarjan.

## PY-tiedostomuoto ##

Python on suunniteltu luettavaksi, ja siinä on yhtäläisyyksiä englannin ja matematiikan kanssa. Python käyttää uusia rivejä ilmaisemaan koko komennon toisin kuin muissa kielissä käytettyjä puolipisteitä tai sulkeita. Python käyttää laajuuksia, silmukoita ja funktioita sisennyksiin ja välilyönteihin toisin kuin muissa kielissä käytettyjen kiharasulkujen kohdalla.

### Syntaksi ###

Seuraavassa on esimerkki Python-syntaksista.

```py
print("Hello World")

#Variables
name = "John"
age = 25
print(name)
print(age)

#While Loop
i = 1
while i < 3:
  print(i)
  i += 1
  
#For Loop
names = ["John", "Harry", "Tom"]
for name in names:
  print(name)
```
