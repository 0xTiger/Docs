## Mikä on EX-tiedosto?

EX-tiedosto on lähdekooditiedosto, joka sisältää lähdekoodia sovelluksille, jotka on kehitetty Euphoria-ohjelmointikielellä. Euphoria on proseduuriohjelmointikieli, joka uskoo ohjelmoinnin yksinkertaisuuteen monimutkaisten ohjelmointikielien sijaan. Euphoriassa kehitetty sovellus voi sisältää useita EX-tiedostoja, mutta yksi EX-tiedosto voi sisältää myös ohjelman täydellisen lähteen.

Sovelluksia, jotka voivat **avata EX-tiedostoja**, ovat openEuphoria Euphoria, Microsoft Notepad, Notepad++ ja Atom.

## EX-tiedostomuodon lyhyt historia

The development and existence of EX file format is linked with the development of Euphoria programming language which was created by Robert Craig. It was first released for MS-DOS. It became open-source software in 2006 with the release of version 3. Sen kehittämisen otti vastuulleen hanketta hallinnoiva ja kehittävä **openEuphoria-ryhmä**. Sen versio 4 julkaistiin joulukuussa 2010, ja se on saatavilla Windowsille, Linuxille, macOS:lle ja kolmelle BSD-versiolle.

Euphoria kirjoitettiin alun perin **[C](/programming/c/)**-ohjelmointikielellä. Myöhemmin Euphoria-tulkki jaettiin kahteen osaan:

 * Front-end Parser - kirjoitettu Euphoria-kielellä ja sitä käytetään Euphoria-to-C-kääntäjän ja Binderin kanssa.
 * `Back-end` - C-ohjelmointikielellä kirjoitettu ajonaikainen kirjasto.

## EX-tiedostomuoto – lisätietoja

EX-tiedostot tallennetaan tekstitiedostoina ja sisältävät lähdekoodin ohjelmille, jotka on kirjoitettu Euphoria-ohjelmointikielellä. Nämä tiedostot voidaan avata millä tahansa tekstieditorilla.
