## Mikä on F95-tiedosto?

Tiedosto, jonka pääte on .f95, osoittaa yleensä, että se on Fortran 95 -lähdekooditiedosto. Fortran on yleiskäyttöinen, pakollinen ohjelmointikieli, joka sopii erityisen hyvin numeeriseen ja tieteelliseen laskemiseen. Fortran 95 on paranneltu versio Fortran 90:stä, joka sisältää useita uusia ominaisuuksia ja parannuksia.

Fortran-lähdekooditiedostot sisältävät tyypillisesti Fortran-ohjelmointikielellä kirjoitetut ohjelmaohjeet; jos sinulla on tiedosto, jonka tunniste on .f95, se on todennäköisesti tarkoitettu käännettäväksi Fortran-kääntäjällä; käännös on prosessi, jossa ihmisen luettava lähdekoodi muunnetaan koneellisesti luettavaksi binäärikoodiksi, joka voidaan suorittaa tietokoneella.

## Kuinka avata F95-tiedosto?

Jos haluat työskennellä .f95-tiedoston kanssa tai ajaa sitä, tarvitset Fortran-kääntäjän. suosittuja Fortran-kääntäjiä ovat GNU Fortran (gfortran), Intel Fortran Compiler (ifort) ja muut. Voit kääntää ja ajaa Fortran-ohjelmia käyttämällä kääntäjän antamia sopivia komentoja.

Jos sinulla on esimerkiksi gfortran asennettuna, voit käyttää seuraavia komentoja terminaalissa kääntääksesi ja suorittaaksesi Fortran-ohjelman:

```
# Compile the Fortran program
gfortran -o my_program my_program.f95

# Run the compiled program
./my_program
```

Jos haluat nähdä vain F95-tiedoston sisällön, se on pelkkä tekstitiedosto, joten voit avata sen millä tahansa tekstieditorilla.
