## Mikä on PHP-tiedosto?
Tiedosto, jonka laajennus on .php, viittaa avoimen lähdekoodin ohjelmointikieleen, jota käytetään palvelinpuolen komentosarjojen kirjoittamiseen web-palvelimella suoritettaviksi. Se on laajimmin käytetty web-skriptikieli, jota tyypillisesti käytetään laajamittaisten verkkosovellusten kehittämiseen. Internetin suurin blogijärjestelmä eli WordPress ja suurin sosiaalinen verkosto Facebook perustuvat PHP-kieleen. Se on yhteensopiva eri alustojen, kuten Windows, Linux, Unix, Mac OS X, jne. kanssa. Se toimii myös menestyksekkäästi lähes kaikilla nykyisin käytetyillä palvelimilla, kuten Apache, IIS jne. PHP tukee monenlaisia tietokantoja, mukaan lukien MySQL.

## PHP-tiedostomuoto ##

Yksi PHP voi sisältää tekstiä, CSS:ää, [HTML](/web/html/), JavaScriptiä ja PHP-koodia. PHP-koodi suoritetaan palvelimella ja tulos palautetaan selaimeen tavallisena HTML-muodossa. PHP-tiedostoja voidaan avata millä tahansa tekstieditorilla ja muokata paikallaan, vaikka sovellukset, kuten Adobe Dreamweaver, Eclipse PHP -kehitystyökalut, tarjoavat kätevän tavan kirjoittaa ja muokata PHP-koodia.

### Syntaksi ###

PHP-skripti suoritetaan palvelimella ja pelkkä HTML-tulos lähetetään takaisin selaimeen. PHP-skripti alkaa<?php and ends with ?>

### PHP-esimerkki ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
