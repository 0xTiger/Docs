# AML – Microsoft Assistance Markup Language File

## Mikä on AML-tiedosto?

AML (Assistance Markup Language) -tiedosto on XML-tiedosto, joka on luotu Microsoft Assistance Markup Language (MAML) -kielellä. Microsoft on kehittänyt sen tarjoamaan online-apua Microsoft Windows -käyttöjärjestelmälle. Ennen tätä Windowsin ohje oli saatavilla käännetyssä HTML-tiedostomuodossa [CHM](/web/chm/). AML-tiedostot tarjoavat jäsennellyn asiakirjan, jonka avulla sovellus voi luoda lähdekoodia ja tukisivuja. Tämä mahdollistaa asiakirjojen ja niiden osatekijöiden määrittelyn kontekstin perusteella. AML-ohjetiedostot julkaistaan verkossa, ja ne voidaan määrittää päivittymään verkossa saatavilla olevista pakettipäivityksistä.

## MAML-tiedostorakenne

MAML:llä luodut AML-tiedostot tallennetaan [XML](/web/xml/)-tiedostoina, joita voidaan käyttää ilmaisemaan monenlaisia aktiivisia käsitteitä. Se voi myös tarjota ohjattua apua (aktiivinen sisällön ohjattu toiminto), joka tekee ohjetiedostosta vuorovaikutteisen käyttäjälle vaiheittaisen ohjatun toiminnon avulla.

An AML file generated using the MAML follows its authoring structure that can be divided into segments like:

 * Käsitteellinen
 * FAQ
 * Sanasto
 * Menettely
 * Viite
 * Uudelleenkäytettävä sisältö
 * Tehtävä
 * Vianetsintä ja
 * Opastus

## Kuinka luoda MAML-tiedostoja?

MAML-tiedostoja voidaan luoda jollakin seuraavista tavoista:

 * Sandcastlen käyttäminen - Käyttää skeemoja ja ohjelman suoritettavia tiedostoja ohjetiedoston luomiseen. Tämä työkalu on kuitenkin lopetettu.
 * DocProjectin käyttäminen – Microsoft Visual Studio -laajennus, joka voidaan suorittaa Microsoft Visual Studiosta ohjesisällön luomiseksi.

MAML-tiedostoja voidaan luoda käyttämällä Sandcastlea, joka on sarja .XSL-skeemoja ja ohjelmia. Ne voidaan myös luoda DocProjectilla, Microsoft Visual Studion ohjetyökalun apuohjelmalla.

# AML - Arc Macro Language File

## Mikä on ARC Macro AML -tiedosto?

AML (ARC Macro Language) -tiedosto on komentosarjatiedosto, joka on luotu ArcInfo Workstation GIS -sovelluksella. Se on kirjoitettu ARC-makrokielellä, joka on patentoitu korkean tason algoritmikieli GIS-sovellusten luomiseen ArcInfossa. ESRI suunnitteli AML:n vuonna 1986, ja sen tarkoituksena oli luoda sovelluksia komentorivi ARCINFO GIS -järjestelmästään. Koska AML-tiedostot ovat komentosarjatiedostoja, ne voivat sisältää komentosarjakomentoja useiden tehtävien suorittamiseksi, kuten käyttöliittymäkomponenttien luomiseksi ja karttatietojen käsittelemiseksi.

## AML-tiedostomuoto – lisätietoja

AML, koska se on komentosarjatiedostoja, tallentaa tiedot levylle tekstitiedostoina. Se noudattaa AML-syntaksia, joka perustui PRIMOS-käyttöjärjestelmän CPL-kuorikieleen. AML-kieli on korvattu ESRI:n geoprosessointikehyksellä, joka on osa ArcGIS-pakettia ja käyttää ArcObjectsia ohjelmointituen tarjoamiseen VBA:n tai Pythonin kautta.
