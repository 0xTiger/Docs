## Mikä on YML-tiedosto?

YML on tekstitiedosto tietojen serialisointikielellä [YAML](/programming/yaml/) (Yet Another Markup Language), jota käytetään usein asetustiedostojen luomiseen. Se voi toimia yhdessä minkä tahansa ohjelmointikielen kanssa, mukaan lukien C/C++, Ruby, Python, Java, PHP, C# ja muut. Koska se on riippumaton tietystä ohjelmointikielestä, se on avoin, yhteentoimiva ja ihmisen luettavissa. YML-tiedostoja voidaan avata ja muokata yleisillä tekstieditoreilla, kuten Microsoft Notepad, Notepad++, Github Atom, Apple TextEdit ja Visual Studio IDE.

## YML-tiedostomuoto - lisätietoja

YML-tiedostot tallentavat sovellusasetukset konfiguraatioina. Sen perusrakenne on kartta, jota voidaan kutsua myös hakemistoksi tai hashiksi. YML-tiedoston merkinnät koostuvat avain- ja arvoparista. Useimmiten määritysten .yml-tiedostot tallennetaan nimellä configure.yml, vaikka jotkin ohjelmointikielet, kuten Ruby on Rails, tallentaa tietokantayhteysmääritykset database.yml-tiedostoihin. {{HYPERLINKKI}} selittää täydelliset kielirakenteet ja syntaksiin liittyvät tiedot.
