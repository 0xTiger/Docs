## Mikä on ALB-tiedosto?

ALB-tiedosto on **Alpha Five** IDE -sovelluksen käyttämä kirjastotiedosto, jota käytetään verkkosovellusten kehittämiseen. Sen ensisijainen tarkoitus on tallentaa tietoja vastaavalle Alpha Fivelle {{HYPERLINKKI}}. Tämä sisältää tietosanakirjan tiedot, kuten asetukset, komentosarjaosoittimet, valikot, käyttäjän määrittämät funktiot, bittikartat, työkalurivit ja tuontimääritykset.

ALB-tiedostot voidaan avata **Alpha Software Alpha Anywhere** -sovelluksella.

## ALB-tiedostomuoto - lisätietoja

ALB-tiedostot tallennetaan Alpha File -tiedostomuodossa, eikä niiden sisäinen tiedostomuotorakenne ole saatavilla julkisesti. Nämä tallennetaan yleisen Alpha Five -kehittäjäkirjastotietokannan lisäksi. Kirjastoina ALB-tiedostoja voidaan käyttää uudelleen muissa Alpha Five -sovelluksissa.
