## Mikä on INS-tiedosto?

Microsoft Windows käyttää yleensä .ins-tunnisteella varustettua tiedostoa puhelinverkko- ja laajakaistayhteyksien määrittämiseen. Itse asiassa se sisältää yhteysasetustietoja, joiden avulla Windows voi määrittää Internet-yhteyden Internet-palveluntarjoajalle. INS-tiedostoa käytetään yleisesti Internet Explorerin (IE) LAN-yhteysasetusten määrittämiseen. Internet-palveluntarjoajien ja järjestelmänvalvojien joskus toimittamat IE-yhteysasetustiedostot käyttäjille, joilla on URL-osoite INS-tiedostoon.

## INS tiedostomuoto
Voit käyttää Internet-asetustiedostoja (INS) yhdessä Internet Explorer Administration Kit 11:n (IEAK 11) kanssa mukautetun selaimen ja sen komponenttien määrittämiseen. Voit kirjoittaa mukautetun paketin eri versioita mukauttamalla INS-tiedoston kopioita.

### Mahdolliset INS-asetukset
Mahdolliset INS-asetukset on esitetty seuraavassa taulukossa:

| Asetus | Kuvaus |
-----|---------|
| URL | Käytetäänkö automaattisesti määritettyä välityspalvelinta. |
| Brändäys | Mukauta selainpaketin brändäys- ja asetustietoja. |
| Media | Mediatyypit, joissa mukautettu asennuspakettisi on saatavilla. |
| Selaimen työkalupalkit | Mukauta IE-työkalupalkin ulkoasua. |
| CabSigning | Digitaaliset allekirjoitustiedot ohjelmillesi. |
| HideCustom | Piilottaako kunkin mukautetun komponentin globaalisti yksilöllinen tunniste (GUID). |
| Yhteysasetukset | Tietoja mukautetun paketin asentamiseen käytetyistä verkkoyhteysasetuksista. |
| CustomBranding | URL-osoite brändäyskaappitiedostoosi (.cab). |
| ExtRegInf | Asennustietotiedostojesi (.inf) nimet ja komponenttien asennustila. |
| SuosikitEx | Lisää Suosikkien kuvaketiedostoon polku, päätä, ovatko suosikit käytettävissä offline-tilassa, ja lisää URL-osoitteet jokaiseen Suosikkisivustoon. |
| ISP_Security | Juurivarmenne, jonka lisäät mukautettuun pakettiisi. |
| Välityspalvelin | Käytetäänkö välityspalvelinta. |
| Turvallisuus tuonti | Tuodaanko mukautetun paketin suojaustiedot. |
