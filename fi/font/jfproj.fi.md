## Mikä on JFPROJ-tiedosto?

A file with .jfproj extension is a JSON based custom font created with Microsoft Font Maker application on Windows 10. Näitä voidaan luoda käyttämällä elektronista kynää kosketusnäyttölaitteiden, kuten tablettien, kanssa, vaikka tietokoneissa voidaan käyttää myös muita tietokoneen oheislaitteita, kuten hiirtä. JFPROJ-tiedostot sisältävät kuvioita, joita käytetään hahmojen näyttämiseen näytölle määrittämällä merkkien ja sanojen väli sekä fontin koko. Microsoft Font Maker -sovellus on ladattavissa Windows Storesta.
