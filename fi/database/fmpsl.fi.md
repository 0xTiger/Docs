## Mikä on FMPSL-tiedosto?

An FMPSL file is a snapshot file of database created with FileMaker Pro 12. Se tallentaa kyselyn tulokset tietokannasta muiden tietojen, kuten visuaalisen asettelun ja näkyvän tiedon, ohella. FMPSL-tiedostoa voidaan käyttää kaikkien näiden tietojen palauttamiseen FileMaker Pro -tietokannan toisessa esiintymässä. Tämä tiedostomuoto otettiin käyttöön FileMaker Pro v 12:n julkaisun yhteydessä. Ennen tätä versiota tilannekuvalinkit esiteltiin FPSL-tiedostomuodossa FileMaker Pro v 11:ssä. FMPSL-tiedostoja voidaan avata FileMaker Pro Advanced -ohjelmistolla. Muita FileMaker Pron tukemia tiedostomuotoja ovat [FP5](/database/fp5/), [FP7](/database/fp7/) ja [FMP12](/database/fmp12/).

## FMPSL-tiedostomuoto

FMPSL-tiedostojen sisäisen tiedostomuodon tiedot eivät ole saatavilla julkisesti kehittäjien viitteeksi.

### Kuinka tallentaa tietueita FMPSL-tiedostona?

FileMaker Pro -tietokannan tiedot voidaan tallentaa snapshot-linkkitiedostona seuraavien vaiheiden avulla.

 1. Hae tietokannasta tietueita, jotka on tallennettava tilannekuvalinkkitiedostona.
 1. Tallenna tiedosto antamalla tiedostonimi käyttämällä valikon Lähetä tietue tilannekuvalinkkinä -vaihtoehtoa.
 1. Käytä selattavia tietueita tallentaaksesi kaikki löydetyt tietueet.

Luotu tilannekuvatiedosto tallennetaan levylle määritetyllä nimellä.
