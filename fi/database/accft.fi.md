## Mikä on ACCFT-tiedosto?

Tiedosto, jonka tarkenne on .accft, on Microsoft Access -mallitiedosto, joka sisältää tietokannan taulukoiden kenttätietotyyppien määritelmän. Microsoft Access on Desktop Relational Database Management System (RDBMS), jota käytetään luomaan taulukkopohjaisia tietokantoja, lomakkeita, kyselyitä ja näkymiä. ACCFT-tiedoston avulla voidaan määrittää tietotyypit taulukkokentille ja tietokantataulukoita voidaan luoda käyttämällä tätä mallitiedostoa lähtökohtana. Se voidaan avata Microsoft Access 2016:lla uusien taulukoiden luomista varten. ACCFT-tiedostot tunnistetaan MIME-tyypin **application/msaccess** perusteella.

## ACCFT-tiedostomuoto

ACCFT-tiedostot ovat binääritiedostoja, joiden tiedostomuotomääritykset eivät ole saatavilla julkisesti. Sen sisäiset tiedot, kuten otsikkotiedot, tiedostorakenne, alatunnisteet jne., eivät ole tällä hetkellä saatavilla.
