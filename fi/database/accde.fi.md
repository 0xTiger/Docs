## Mikä on ACCDE-tiedosto?

Tiedosto, jonka tarkenne on .accde, on Microsoft Access 2007 -tiedostomuoto, jolla luodaan Execute Only -tietokantatiedostoja mukautetun tietokantakoodin suojaamiseksi. Se kokoaa kaiken VBA-koodin vain suoritettavaksi, jotta sitä ei voi tarkastella tai muokata. ACCDE-tiedostomuoto korvaa MDE-tiedostomuodon, jota käytettiin Microsoft Accessin aiemmissa versioissa. ACCDE ovat [ACCDB](/database/accdb/) tiedostoja vain suoritettavassa tilassa ja kompaktimpia, koska niissä ei ole lähdekoodia. Näiden tiedostojen pieni koko parantaa muistin käyttöä ja parantaa sovelluksen suorituskykyä. ACCDE-tiedostot voidaan avata Microsoft Access 365:llä.

## ACCDE-tiedostomuoto

Kaikki Microsoft Access 2007+ -tiedostot käyttävät Jet ACE:tä (Access-tietokantamoottori), joka käyttää ACCDB-tiedostomuotoperhettä. Microsoft ei kuitenkaan tarjoa avoimen lähdekoodin toteutusta tai yksityiskohtaista teknistä eritelmää.
