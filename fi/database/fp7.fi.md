## Mikä on FP7-tiedosto?

A file with .fp7 extension is a database file created with FileMaker Pro versions 7-11. Se tallentaa tiedot tietueiden taulukoihin, joissa jokainen tietue sisältää yhden tai useamman kentän. FP7-tiedostoilla on mahdollisuus tallentaa suuria tietoja, mukaan lukien henkilökohtaisten tietokantojen tiedot ja organisaatiotietueet. FileMaker Pron uudempi versio on [FMP12](/database/fmp12/), joka otettiin käyttöön FileMake Pro -version 12 julkaisun yhteydessä. Tekstitiedon lisäksi FP7-tiedosto voi sisältää myös lomakkeita, kaavioita ja muita tietokantaobjekteja. FP7-tiedostot voidaan avata FileMaker Pro Advanced -sovelluksella.

## FP7 tiedostomuoto

FP7-tiedostot tallennetaan levylle binääritiedostoina FileMakerin omassa tiedostomuodossa. FP7-tiedostomuodon sisäisen tiedostomuodon tiedot eivät ole saatavilla julkisesti.
