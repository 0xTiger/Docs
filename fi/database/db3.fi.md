## Mikä on DB3-tiedosto?
DB3-tiedosto on SQLite-ohjelmiston luoma tietokantatiedosto, joka on kevyt, itsenäinen tietokantaohjelma, joka luo tietokantoja tavallisista tiedostoista. sisältää tietokannan anatomian sekä tietueita; käytetään yleisesti strukturoidun tiedon hakemiseen tai tallentamiseen SQL:n avulla. Näitä tiedostoja voidaan käyttää älylaitteissa tai missä kirjanpitoa tai muuta tiedonhallintaa vaatii, mutta vähän tilaa vaativassa ympäristössä.


## DB3-tiedostomuoto
DB3-tiedostomuoto liittyy RDBMS (Relational Database Management System) SQLiteen, joka on suosittu valinta sulautetuille tietokannoille. SQLite_3:n määrittelyssä ei ole määritetty mitään erityistä tiedostotunnistetta, se voi käyttää laajennuksia, kuten [.dbf](/database/dbf/) ja [.sqlite](/database/sqlite/).

### Tietokannan erittely
Commonly SQLite, version 3 related db file format can be considered as the DB3 file format used as the publicly documented native format for the SQLite database engine since June 2004. DB3-tiedostomuoto on monialustainen, jota voidaan siirtää suurten ja pienimuotoisten arkkitehtuurien tai 32- ja 64-bittisten järjestelmien välillä. Nämä ominaisuudet tekevät DB3:sta suositun valinnan sovellustiedostomuodoksi. SQLite_3-päätietokantatiedosto (DB3) koostuu yhdestä tai useammasta sivusta. Kaikkien sivujen koot ovat samat samassa tietokannassa. Sivun koko tavuina on kahden potenssi välillä 512–65536.
