## Mikä on PPS-tiedosto?

PPS, PowerPoint Slide Show, files are created using Microsoft PowerPoint for Slide Show purpose. PPS file reading and creation is supported by Microsoft PowerPoint 97-2003. Tämän tiedostomuodon uusin versio on [PPSX ](/presentation/ppsx/), joka perustuu Office OpenXML -standardeihin. Microsoft PowerPointin uusimmat versiot voivat edelleen lukea PPS-tiedostoja, mutta äskettäin luodut tiedostot voidaan tallentaa vain PPSX-tiedostomuodossa. Kun PPS-tiedosto jaetaan toisen käyttäjän kanssa ja avataan, se alkaa Powerpoint-showna toisin kuin [PPT](/presentation/ppt/)-tiedosto, joka avautuu muokattavassa tilassa.

## Tiedosto muoto ##

PPS on binääritiedostomuoto, joka noudattaa [MS-PPT](https://msdn.microsoft.com/en-us/library/office/cc313106(v#office.12).aspx) tiedostomuotoasiakirjassa esitettyjä määrityksiä. Se koostuu useista virroista, jotka edustavat asiakirjan yleistä rakennetta ja tietoja. Nämä sisältävät:

* Nykyinen käyttäjävirta

* PowerPoint Document Stream

* Kuvat Stream

* Yhteenvetotiedot ja asiakirjan yhteenvetotiedot (valinnainen)


Täydelliset tiedot PPT-tiedostomuodosta löytyvät MS-PPT-spesifikaatioista, jotka koskevat myös PPS:ää.
