## Mikä on AST-tiedosto?

AST-tiedosto on laskentataulukkomallitiedosto, joka on luotu Ability Spreadsheet -ohjelmistolla, joka on Ability Office -ohjelmistopaketin laskentataulukkokomponentti. Näitä mallitiedostoja käytetään luomaan alustava joukko laskentataulukkomalleja, joita voidaan käyttää [AWS](/spreadsheet/aws/)-tiedostojen luomiseen. AST-tiedostot voidaan avata Ability-laskentataulukko-ohjelmistolla.

## AST-tiedostomuoto – lisätietoja

AST files are saved to disc as binary files and can be opened in Ability Spreadsheet software. Though fully compatible with Microsoft XLS and XLSX file formats, AST file format specifications are not available publicly for developer's reference. AST files can be saved to disc in [PDF](/pdf/) as well as [XPS](/page-description-language/xps/) file formats.
