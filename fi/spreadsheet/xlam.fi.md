## Mikä on XLAM-tiedosto? ##

XLAM on makrokäyttöinen apuohjelma, jota käytetään lisäämään uusia toimintoja laskentataulukoihin. Apuohjelma on lisäohjelma, joka suorittaa lisäkoodia ja tarjoaa lisätoimintoja laskentataulukoille. XLAM-tiedostot tallennetaan .xlam-tunnisteella. XLAM-tiedostot ovat XML-pohjaisia tiedostoja, jotka muistuttavat tiedostomuotoja [XLSM](/spreadsheet/xlsm/) ja [XLSX](/spreadsheet/xlsx/), ja ne tallennetaan ZIP-pakkauksella tiedoston kokonaiskoon pienentämiseksi.

### Esimerkki ###

```cmd
Public Function Add(num1 As Double, num2 As Double)
Add = num1 + num2
End Function
```

Tallenna tämän jälkeen tiedosto .xlam-tunnisteella.
