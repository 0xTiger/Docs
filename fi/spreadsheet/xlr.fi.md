## Mikä on XLR-tiedosto?

A file with .xlr extension is a Microsoft Works Spreadsheet file format that stored data in rows and columns. Microsoft Works was developed for use as an initial productivity suite and remained functional from 1987 to 2009. Se sisälsi tekstinkäsittelyohjelman, laskentataulukon ja tietokannan hallintajärjestelmän. Se lopetettiin myöhemmin sen jälkeen, kun Microsoft Office esiteltiin sisältäen monipuolisemmat ominaisuudet.

## XLR-tiedostomuoto - lisätietoja

XLR-tiedostot tallennettiin binääritiedostoina ilman julkisia tietoja sen tiedostomuotomäärityksistä. Siksi kehittäjillä ei ole mahdollisuutta analysoida sen tiedostorakennetta ja kirjoittaa sovelluksia näiden tiedostojen lukemiseksi/kirjoittamiseksi ohjelmallisesti. Works Spreadsheet käyttää samaa Excel-tiedostomuotoa tietojen tallentamiseen .xlr-tiedostoihin.
