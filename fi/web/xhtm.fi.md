## Mikä on XHTM-tiedosto?

An file with .xhtm extension is a webpage file that is written in XHTML. It uses XML 1.0 as its standard implementation and is considered a reformulation of HTML 4. Kuten XHTML, XHTM-tiedosto sisältää standardinmukaisia HTML-tageja, jotka noudattavat tiukempia XML-ohjeita. XHTM-tiedostoja käytetään verkkosivuston sisällön julkaisemiseen, ja niissä käytetään yleisesti laajennusta [XHTML](/web/xhtml/).

Voit avata XHTM-tiedostoja web-selaimilla, kuten Google Chromella, Apple Safarilla ja Mozilla Firefoxilla.

## XHTM esimerkki

Seuraava esimerkki XHTM-asiakirjasta määrittelee XML-ilmoitukset.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
