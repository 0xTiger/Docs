## Mikä on DISCOMAP-tiedosto?

DISCOMAP-tiedosto on tulostetiedosto, jonka luo Microsoft Web Services Discovery Tool, **Disco.exe**. Se sisältää tulokset, jotka ovat tietystä URL-osoitteesta löydettyjä ASP.NET-verkkopalveluita. Yleisiä esimerkkejä tällaisista löydetyistä palveluista ovat .DISCO- tai .WSDL-tiedostojen sijainnit. Kun Microsoft Web Discovery -työkalu suoritetaan, se etsii etsintätietoja DISCO-tiedostosta löytyneen XML-sisällön perusteella. Tulokset tallennetaan DISCOMAP-tiedostoon, joka voidaan avata Microsoft Visual Studio IDE:llä.
