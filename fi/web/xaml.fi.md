## Mikä on XAML-tiedosto?

XAML, Extensible Application Markup Language, laajennustiedostot kuvaavat käyttöliittymäelementit Windows Presentation Foundation (WPF) -pohjaisiin ohjelmistoihin. Vaikka se on kieli, sitä ei tarvitse ohjelmoida, koska se perustuu **[XML](/web/xml/)** vakiomuotoon, jota on helppo käyttää ja ymmärtää. XAML (lausutaan zammel) on Microsoftin kehittämä erityistavoitteena käyttöliittymien luominen. Sen alkuperäinen lyhenne tarkoitti Extensible Avalon Markup Language, jossa Avalon oli WPF:n koodinimi. XAML-tiedostot tallennetaan joskus myös XOML-laajennuksella.

## XAML-sovellukset

XAML on valinta käytettäväksi .NET Framework 3.0- ja .NET Framework 4.0 -tekniikoissa, kuten WPF, Silverlight, Windows Workflow Foundation (WF) ja muutama muu. Käyttöliittymäelementit, datasidokset, tapahtumat ja muut ominaisuudet määritellään WPF:n XAML-lomakkeilla. Vastaavasti WF:n työnkulkuja voidaan määrittää XAML:n avulla. Se on helppo käsitellä työkaluilla, koska se perustuu XML:ään. Koska se on deklaratiivinen kieli eikä sitä tarvitse kääntää, on tulossa paljon tuotteita, jotka perustuvat XAML-pohjaisiin sovelluksiin. Kaikki XAML:ssä luotu tai toteutettu voidaan ilmaista perinteisemmällä .NET-kielellä, kuten C# tai Visual Basic .NET.

## XAML tiedostomuoto

XAML is totally based on the XML format. The initial specifications of [XAML Object Mapping](https://download.microsoft.com/download/0/A/6/0A6F7755-9AF5-448B-907D-13985ACCF53E/%5BMS-XAML%5D.pdf) were published in 2006, followed by another [version](https://download.microsoft.com/download/0/A/6/0A6F7755-9AF5-448B-907D-13985ACCF53E/%5BMS-XAML-2009%5D.pdf) published in 2009. Nämä eritelmät määrittelevät kaksi abstraktia tietomallia:

* XAML Schema Information Set Model

* XAML-tietosarjan malli


Xaml-tietojoukko (lyhyesti Xaml-infoset) määrittelee tiedon rakenteen, jota Xaml-ilmentymä voi edustaa. Xaml Schema Information Set mahdollistaa tiettyjen Xaml-sanastojen määrittämisen. Tämä määrittely määrittelee myös joukon sääntöjä XML-dokumentin muuntamiseksi Xaml-tietojoukoksi. XML on yleinen Xaml-muoto. (Termi Xaml-asiakirja viittaa XML-asiakirjaan, joka edustaa Xaml-tietojoukkoa.) Mutta vaikka tämä spesifikaatio ei määrittele muita esityksiä, mitä tahansa fyysistä esitystä voidaan käyttää, kunhan se voi edustaa Xaml-tietojoukon tietoja. .
