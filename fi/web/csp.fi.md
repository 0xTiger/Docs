## Mikä on CSP-tiedosto?

CSP-tiedosto on dynaaminen verkkosivu, joka sisältää RadGs Concept Application Server -palvelimella suoritettavaa sisältöä. Se käännetään ja suoritetaan palvelinpuolella, minkä jälkeen se toimitetaan pyytävän käyttäjän selaimelle HTTP-protokollan kautta. Toisin kuin CON-tiedostomuoto, CSP-tiedostot eivät edellytä Concept-asiakasohjelman asentamista käyttäjälle näiden tiedostojen katselua varten. Voit tarkastella CSP-tiedostoja, jotka ovat samankaltaisia kuin muut palvelinpuolen sivut, kuten [PHP](/programming/php/) ja [ASP](/web/asp/).
