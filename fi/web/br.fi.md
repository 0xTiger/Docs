## Mikä on BR-tiedosto?

A BR file is a compressed web file generated by applying the open source data compression algorithm, **Brotli**. It is used to store webpage assets such as stylesheets ([CSS](/web/css/)), images ([SVG](/page-description-language/svg/)), [XML](/web/xml/), and scripting files ([JS](/web/js/)). Modern day websites, such as Chrome and Firefox, use BR files to reduce the page loading time, resulting in better user experience.

## BR-tiedostomuoto

BR-tiedostot ovat pakattuja verkkotiedostoja, jotka luodaan Brotli-pakkausalgoritmilla. Brotli-pakkaus on häviötön tietojen pakkausalgoritmi, jonka Google on kehittänyt Zopfli-pakkausalgoritmiksi. Se perustuu häviöttömän LZ77-pakkausalgoritmin ja Huffman-koodauksen yhdistelmään.

Pienen kokonsa vuoksi BR-tiedostoja käyttävät verkkopalvelimet ja sisällönjakeluverkot (CDN). Näille palvelimille tehdyt pyynnöt johtavat HTTP-sisällön pakkaamiseen, mikä nopeuttaa verkkosivustojen lataamista. Brotli on tullut suositummaksi ja tarjoaa paremman pakkauksen kuin gzip.
