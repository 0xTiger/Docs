## Mikä on VDISCO-tiedosto?

VDISCO-tiedosto on Microsoft Visual Studion verkkopalveluissaan käyttämä hakutiedosto. Se tarjoaa tietoja käytettävissä olevista verkkopalveluista, kuten niiden URL-osoitteista, nimiavaruuksista ja menetelmistä. VDISCO-tiedosto on samanlainen kuin [DISCO](/web/disco/)-tiedostomuoto, mutta se luodaan suorituksen aikana. Se on tallennettu verkkopalvelusovellukseen, ja Visual Studio käyttää sitä Web-palveluiden dynaamiseen löytämiseen ja käyttämiseen projektissa. VDISCO-tiedostoja käytetään yhdessä todellisen verkkopalvelukoodin sisältävän [.asmx](/web/asmx/)-tiedoston kanssa.

Voit avata VDISCO-tiedoston missä tahansa tekstieditorissa, kuten Microsoft Notepad, Github Atom ja Apple TextEdit. Se voidaan myös avata Microsoft Visual Studio 2012:lla ja uudemmilla sen kanssa työskentelemistä varten.

## VDISCO tiedostomuoto

VDISCO-tiedosto tallennetaan ja sitä isännöidään [XML](/web/xml/)-tiedostomuodossa, joka on yleinen muoto tietojen kokoamista ja julkaisemista varten. Se sisältää palvelun URL-osoitteen, nimiavaruudet ja menetelmät tavallisissa XML-tunnisteissa, joissa jokainen tunniste sisältää tietonsa vastaavan arvon.
