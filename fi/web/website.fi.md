## Mikä on WEBSITE-tiedosto?

A WEBSITE file is a website shortcut file created with Microsoft Internet Explorer (IE). It contains a URL and an icon (ICO) file that points to the website's favicon. The icon is used for displaying the shortcut. WEBSITE file format was introduced with the release of IE version 9. Aiemmat IE-versiot tallensivat verkkosivuille pikakuvakkeita [.url](/web/url/)-tiedostoina.

## VERKKOSIVUSTO tiedostomuoto

.website-tiedosto tallennetaan levylle pelkkänä tekstitiedostona, jonka sisältöä voi tarkastella missä tahansa tekstieditorissa. Muun sisällön lisäksi se sisältää kohde-URL-osoitteen, jolle se on pikakuvake. Jos haluat avata kohdesivuston muulla selaimella kuin IE:llä, voit kopioida tämän URL-osoitteen ja liittää sen muihin selaimiin, kuten Google Chromeen, Safariin tai Firefoxiin, jotta voit tarkastella verkkosivustoa.

## Miten luodaan .WEBSITE-tiedosto?

WEBSITE-tiedosto voidaan luoda IE:ssä napsauttamalla URL-osoitetta osoitepalkissa ja vetämällä se haluamaasi paikkaan tietokoneellasi, kuten työpöydälle tai asiakirjoihin. Tämä luo pikakuvakkeen kyseiseen Internet-sijaintiin ja voidaan avata kaksoisnapsauttamalla pikakuvaketta. Jos haluat tallentaa pikakuvakkeen URL-tiedostona, pidä SHIFT-näppäintä painettuna ennen kuin vedät kohdesivuston suosikkikuvakkeen työpöydälle.
