## Mikä on XPD-tiedosto?

XPD-tiedosto on verkkosivun tiedosto, joka on luotu XML-dokumentista. Useimmiten tämä luodaan XSLT-muunnoksen avulla. W3:ssa on saatavilla [XPD file format specifications](https://www.w3.org/TR/xml-pipeline/), joka määrittelee tavan lisätä moniprosessisten [XML](/web/xml/)-sovellusten yhteentoimivuutta. XML Pipeline Language määrittää XML-resurssien väliset käsittelysuhteet.
