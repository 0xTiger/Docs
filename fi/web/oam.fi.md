## Mikä on OAM-tiedosto?

A .oam file a an animated widget file exported from Adobe Edge Animate Widget. Animations exported as OAM widget file format can be inserted into other Adobe applications such as InDesign Documents ([INDD file](/page-description-language/indd/), Dreamweaver, and Muse. OAM files are like deployment packages that can be readily inserted into other Adobe's projects to make use of them. OAM files contain information about the animation's assets, behaviors, and actionscript code. You can create an OAM file by publishing an Adobe Animate project [.AN file](/web/an/).
Käyttäjät voivat luoda OAM-tiedostoja julkaisemalla ne Edge Animate -projektista (.AN-tiedosto).

## OAM-tiedostomuoto

OAM-tiedosto tallennetaan levylle pakattuna [ZIP](/compression/zip/)-arkistona. Tämä tarkoittaa, että voit nimetä tiedostotunnisteen uudelleen muotoon .zip ja purkaa sen millä tahansa pakkausapuohjelmalla, kuten WinZIP tai WinRAR. Pienennetty tiedostokoko helpottaa animaation siirtämistä ja jakamista muiden käyttäjien kanssa Internetin kautta.

### OAM-tiedostorakenne

.oam-tiedoston sisäinen tiedostorakenne on omistusoikeus, eikä Adobe ole dokumentoinut sitä julkisesti. Tiedetään kuitenkin, että .oam-tiedostot sisältävät kokoelman tiedostoja ja resursseja (kuten grafiikkaa, ääntä ja ActionScript-koodia), jotka on pakattu yhteen tiedostoon. .oam-tiedoston sisältö voi sisältää [XML](/web/xml/)-tiedostoja, SWF-tiedostoja ja muita resurssitiedostoja. .oam-tiedoston tarkka rakenne riippuu sen luomiseen käytetystä Adobe Animaten versiosta sekä animaation tyypistä ja tiedostoon sisältyvästä sisällöstä.

OAM-tiedosto sisältää seuraavat tiedot.

Assets: tiedot animaatiossa käytetystä grafiikasta, ääni- ja videosisällöstä.

Käyttäytyminen: Kuvaukset animaatiossa esiintyvistä toimista ja vuorovaikutuksista.

`ActionScript-koodi:` Koodi, jota käytetään ohjaamaan animaation toimintaa ja interaktiivisuutta.

`Julkaise-asetukset:` Tietoja alustasta ja muodosta, jossa animaatio julkaistaan, kuten HTML5 tai AIR.

`Metadata:` Additional information such as the animation's author, creation date, and version number.

## Kuinka avata OAM-tiedosto?

Voit avata OAM-tiedostoja seuraavien sovellusten avulla.

 * Adobe Edge Animate CC (poistettu)
 * Adobe Animate 2023
 * Adobe InDesign 2023
 * Adobe Dreamweaver 2021
