## Mikä on HYPE-tiedosto?

A HYPE file is a project file saved using the Hype software that is used for creation of web content with HTML5. Se sisältää verkkoelementtejä, kuten grafiikkaa, tekstiä, animaatioita, kohtaustietoja ja muita elementtejä, jotka yhdessä muodostavat sisällön. HYPE-tiedoston verkkosisältö voidaan viedä web-tiedostoiksi, jotka voidaan julkaista verkkosivulla.

HYPE-tiedosto voidaan avata [Tumult Hype](https://tumult.com/hype/)-ohjelmistolla.

## HYPE tiedostomuoto

HYPE-tiedostot on tallennettu omaan tiedostomuotoon, eikä niiden sisäisiä tietoja tunneta julkisesti. HYPE-tiedoston sisältöä voi tarkastella vain, kun se avataan Tumult Hype -ohjelmistossa.

### HYPE-tiedoston sisältö

HYPE-tiedosto luodaan, kun käytät Hype-sovellusta verkkosisällön luomiseen eri tarkoituksiin. Se tallentaa tietoja avainkehyspohjaisista animaatioista sisällön, kuten e-korttien, e-kirjojen, esitysten, mainosten, infografioiden ja verkkosivumateriaalin, luomiseen. Näiden sisältö perustuu HTML5:ään, joka mahdollistaa sisällön näyttämisen laitteilla pöytäkoneista iPhoneen ja iPadiin. Tumult Hype tuottaa uusinta HTML5:tä, [CSS3](/web/css/)-tyylejä ja [JavaScript](/web/js/).
