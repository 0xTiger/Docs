## Mikä on LRX-tiedosto?

A file with .lrx extension is an associated file for Sony Broadband eBook (BBeB). It contains data such as images, text, and pagination information. The BBeB format is used by the Sony Portable Reader which is a mobile device allowing users to read eBooks. But as of July 2010, it has become obsolete and Sony now uses the [EPUB](/ebook/epub/) file format for ebooks. Applications that can open LRX files include Sony Reader which is available for both Windows and Mac OS.

## LRX tiedostomuoto

LRX-tiedostot ovat binääritiedostomuodossa ja kaikki tiedoston sisältö on salattu toisin kuin {{HYPERLINKKI}}, joka on salaamaton muoto. Toinen liitetty BBeB-tiedostomuoto on LRS, joka on XML-pohjainen ja on ihmisen luettavissa missä tahansa tekstieditorissa. LRX-tiedostojen otsikko sisältää tietoja, kuten:

* Versio

* Juuriobjektin tunnistus

* Pikkukuvan tyyppi

* Deflatoidut tietoarvot

* Hämäränäppäin

* Suuntalipun sidonta

* Tietojen koko deflaation jälkeen
