## Mikä on VSD-tiedosto?

VSD-tiedostot ovat piirustuksia, jotka on luotu Microsoft Visio -sovelluksella esittämään erilaisia graafisia objekteja ja niiden välisiä yhteyksiä. Tällaiset piirustukset voivat sisältää visuaalisia objekteja, kuten visuaalisia objekteja, vuokaavioita, UML-kaavioita, tietokulkuja, organisaatiokaavioita, ohjelmistokaavioita, verkkoasetteluja, tietokantamalleja, objektien kartoituksia ja muuta vastaavaa tietoa. Microsoft Visio tarjoaa mahdollisuuden muuntaa Visio-tiedostoja useisiin eri tiedostomuotoihin, mukaan lukien [PNG](/image/png/), [BMP](/image/bmp/), [PDF](/pdf/) ja muut.

# VSD-tiedostomuoto #

VSD file format is the initial file format introduced by Microsoft Visio for storing drawing elements. The internal structure of VSD files is based on binary file format unlike [.VSDX](/visio/vsdx/) that was introduced with Microsoft Visio 2013. Uusi muoto perustuu Office Open XML -tiedostomuotoon ja tallentaa projektitiedot [ZIP](/compression/zip/)-arkistoon. Microsoft tarjoaa vviewer.exe-sovelluksen Visio-piirustustiedostoille, jos Visioa ei ole asennettu järjestelmään.

## Lyhyt historia ##

Visio was initially produced as a standalone product by Shapeware Corporation in 1992 by its version 1.0. Shapeware Corporation muutti nimensä Visio Corporationiksi vuonna 1995 hyödyntääkseen markkinoiden tunnettuutta. Microsoft muutti Visionin vuonna 2000 Microsoft Office -sovellukseksi. Visio for Enterprise Architects esiteltiin Visual Studio .NET:n kanssa vuosina 2003 ja 2005. Visio 2007 julkaistiin 30. marraskuuta 2006. Microsoft Visio otti nauhat käyttöön käyttöliittymässä vuonna 2010.
