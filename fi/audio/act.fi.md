## Mikä on ACT-tiedosto? ##

ACT- tai Adobe Color Table -muoto (tunnetaan myös nimellä Color Lookup Table -tiedosto) on eräänlainen Adobe Photoshop -tietokanta, joka tallentaa joukon ennalta määritettyjä värejä. Voit lisätä tai poistaa värejä tallentaessasi kuvaa verkkojulkaisua varten paremman kuvanlaadun tai pienemmän tiedostokoon suosimiseksi. Sinulla saattaa olla ADPCM-salattu äänimuoto, jos sitä ei käytetä Illustratorin kanssa. Adaptive Differential Pulse Code Modulation on äänen pakkaustekniikka, jota käyttävät useat musiikkilaitteet ja kuuntelulaitteet.

ACT-tiedostomuotoa käyttää myös Alma CAD/CAM -tiedostojärjestelmä. Nämä ovat ohjeita, joita 3D-työstökoneet käyttävät selvittääkseen, kuinka leikata mitä tahansa. Genesis3D Player, DS Games Developer Action tai FoxPro Detailing Wizard Action Kaaviotiedostot ovat kaikki esimerkkejä ACT:stä.

## ACT-tiedostomuodon määritykset

Monet tallentimet, jotka käyttävät vain ACT-järjestelmää, tekevät sen pienimmällä tallennusvaihtoehdolla; jos laatua parannetaan, käytetään erillistä muotoa, yleensä WAV, jolloin saadaan valtavasti suurempia asiakirjoja.

ACT:tä on useita sukupolvia; kesäkuusta 2009 lähtien myöhemmillä laitteilla luotuja kappaleita ei vain voi tunnistaa kaikki ilmaiset tavalliset musiikkisoittimet tai muunninsovellukset, joten jäljelle jää vain mukana tulevat MP3-työkalut.

Kuvan optimointi verkkojakelua varten tehdään ACT-tiedostoilla. Asiakkaat voivat testata kuvan laatua luotujen kuvatietokantojen kanssa lisäämällä tai vähentämällä värien määrää CLUTissa.

Voit ladata Adobe-väritaulukoita napauttamalla Color Table -vaihtoehtoa ja valitsemalla Lataa väritiedot Export for Web -valintaikkunasta. */Presets/en_US/Save* for Web Settings/Color Tables/ -hakemisto sisältää useita valmiiksi määritettyjä ACT-tiedostoja, jotka tulivat Photoshop-järjestelmän mukana.

Act-skriptit voidaan luoda yksinomaan Genesis 3D:ssä tai muuntamalla muita 3D-malleja ACT-standardiin Genesis3D Actor Designerilla
