## একটি YML ফাইল কি?

YML হল ডেটা সিরিয়ালাইজেশন ল্যাঙ্গুয়েজ, [YAML](/programming/yaml/) (এখনও আরেকটি মার্কআপ ল্যাঙ্গুয়েজ), যা প্রায়শই কনফিগারেশন ফাইল তৈরি করতে ব্যবহৃত হয়। এটি সি/সি++, রুবি, পাইথন, জাভা, পিএইচপি, সি# এবং অন্যান্য সহ যেকোনো প্রোগ্রামিং ভাষার সাথে একত্রে কাজ করতে পারে। কোনো নির্দিষ্ট প্রোগ্রামিং ভাষা থেকে স্বাধীন হওয়ার কারণে, এটি উন্মুক্ত, আন্তঃচালনাযোগ্য এবং মানুষের পাঠযোগ্য। YML ফাইলগুলি সাধারণ পাঠ্য সম্পাদক যেমন Microsoft Notepad, Notepad++, Github Atom, Apple TextEdit, এবং Visual Studio IDE এর সাথে খোলা এবং সম্পাদনা করা যেতে পারে।

## YML ফাইল ফরম্যাট - আরও তথ্য

YML ফাইলগুলি কনফিগারেশন হিসাবে অ্যাপ্লিকেশন সেটিংস সংরক্ষণ করে। এর মৌলিক কাঠামো একটি মানচিত্র যাকে একটি ডিরেক্টরি বা হ্যাশও বলা যেতে পারে। একটি YML ফাইলের এন্ট্রিতে একটি কী এবং মান জোড়া থাকে। বেশিরভাগ ক্ষেত্রে, কনফিগারেশন .yml ফাইলগুলি `configure.yml` হিসাবে সংরক্ষণ করা হয়, যদিও কিছু প্রোগ্রামিং ভাষা যেমন রুবি অন রেল `database.yml` ফাইলে ডেটাবেস সংযোগ কনফিগারেশন সংরক্ষণ করে। [YML/YAML file format specifications](https://yaml.org/spec/1.2.2/) সম্পূর্ণ ভাষা কাঠামো এবং সিনট্যাক্স সম্পর্কিত তথ্য ব্যাখ্যা করুন।
