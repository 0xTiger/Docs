## একটি AWK ফাইল কি?

একটি AWK ফাইল হল একটি স্ক্রিপ্ট ফাইল যা টেক্সট প্রসেসিং অ্যাপ্লিকেশন **AWK** এর জন্য তৈরি করা হয় যা ইউনিক্স এবং লিনাক্স অপারেটিং সিস্টেমের সাথে একত্রিত হয়। এটিতে টেক্সট বা ফাইলের বিষয়বস্তু যেমন মেলানো, প্রতিস্থাপন এবং টেক্সট মুদ্রণের মতো ক্রিয়া সম্পাদনের জন্য যৌক্তিক নির্দেশাবলী রয়েছে। এটি ইনপুট ফাইল থেকে রিপোর্ট এবং ফরম্যাটেড টেক্সট তৈরি করতে সাহায্য করে। awk ইউটিলিটি সাধারণত বেশিরভাগ linux/unix ডিস্ট্রিবিউশনে /usr/bin/awk-এ অবস্থিত।

## কিভাবে AWK স্ক্রিপ্ট তৈরি করবেন?

লিনাক্স/ইউনিক্স ওএস-এর যেকোনো টেক্সট এডিটরে একটি AWK ফাইল তৈরি বা খোলা যায়। অনুসরণ হিসাবে একটি নতুন AWK স্ক্রিপ্ট ফাইল তৈরি করা যেতে পারে।

```
$ vi script.awk
```

এটি পাঠ্য সম্পাদকে AWK ফাইলটি খুলবে। নিচের মত টেক্সট এডিটরে কিছু স্টেটমেন্ট লিখুন।

```
#!/usr/bin/awk -f
BEGIN { printf "%s\n","Writing my first Awk executable script!" }
```
এই পাঠ্য বিষয়বস্তুর প্রথম লাইন অনুসরণ হিসাবে ব্যাখ্যা করা হয়.

 * \#! - 'শেবাং' হিসাবে উল্লেখ করা হয়, যা একটি স্ক্রিপ্টের নির্দেশাবলীর জন্য একটি দোভাষী নির্দিষ্ট করে
 * /usr/bin/awk - দোভাষী
 * -f - দোভাষী বিকল্প, একটি প্রোগ্রাম ফাইল পড়তে ব্যবহৃত

## কিভাবে AWK স্ক্রিপ্ট চালানো যায়?

ফাইলটি সংরক্ষণ করুন এবং নীচের কমান্ডটি জারি করে স্ক্রিপ্টটিকে কার্যকর করুন:
```
$ chmod +x script.awk
```

স্ক্রিপ্ট তারপর অনুসরণ হিসাবে চালানো যেতে পারে.

```
$ ./script.awk
```

যে ফলাফল নিম্নলিখিত আউটপুট.

```
Writing my first Awk executable script!
```
