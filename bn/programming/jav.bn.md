## জাভা ফাইল কি?

জাভা ফাইল হল একটি জাভা সোর্স কোড ফাইল যা জাভা প্রোগ্রামিং ভাষায় তৈরি করা হয়েছে। এটি জাভা সোর্স ফাইলের মতো এবং জাভা ইন্টারপ্রেটারের সাথে কম্পাইল করা যেতে পারে। [Java](/programming/java/) ফাইলগুলির মতো, Java ফাইলটি Java অ্যাপ্লিকেশনগুলির বিকাশে ব্যবহৃত হয়৷ এটি একটি জাভা প্রকল্পে উপলব্ধ অনেকগুলি সোর্স কোড ফাইলগুলির মধ্যে একটি হতে পারে৷

## জাভি ফাইল ফরম্যাট

Jav ফাইলগুলি টেক্সট ফাইল হিসাবে সংরক্ষণ করা হয় এবং যেকোনো টেক্সট এডিটর দিয়ে খোলা যায়। জাভা ফাইলের প্রকারগুলি সাধারণত জাভা ফাইলের প্রকার দ্বারা প্রতিস্থাপিত হয়েছে এবং সাধারণত জাভা অ্যাপ্লিকেশন লেখার জন্য ব্যবহৃত হয়। জাভা কোডগুলি Eclipse এবং IntelliJ Idea এর মত IDE দিয়ে লেখা যেতে পারে যা জাভা ফাইলগুলির সাথে কাজ করা সহজ করে তোলে।

### Jav ফাইল ফরম্যাটের উদাহরণ
