## একটি APPXUPLOAD ফাইল কি?

একটি APPXUPLOAD ফাইল হল একটি একক প্যাকেজ ফাইল যাতে Microsoft স্টোরে আপলোড করার জন্য একাধিক অ্যাপ প্যাকেজ বা অ্যাপ বান্ডেল থাকে। এটি স্বয়ংক্রিয়ভাবে [Microsoft Visual Studio](https://visualstudio.microsoft.com/) দ্বারা তৈরি হয় যখন বিকাশকারীরা তাদের অ্যাপকে [Microsoft Store](https://apps.microsoft.com/store/apps)-এ প্রকাশের জন্য প্যাকেজ করে। একটি [APPX](/programming/appx/) ফাইল ছাড়াও, APPXUPLOAD ফাইলটিতে একটি প্রতীক ফাইল রয়েছে যা Microsoft স্টোরে প্রকাশিত হওয়ার পর অ্যাপের কার্যক্ষমতা বিশ্লেষণ করে।

## AAPXUPLOAD ফাইল ফরম্যাট - আরও তথ্য

APPXBUNDLE ফাইলগুলি বাইনারি ফাইল বিন্যাসে ডিস্কে সংরক্ষণ করা হয়, তাদের ফাইল বিন্যাসের বৈশিষ্ট্যগুলি সর্বজনীনভাবে উপলব্ধ নয়।
