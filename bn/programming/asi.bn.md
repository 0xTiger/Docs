## একটি ASI ফাইল কি? ##

একটি ASI ফাইল আলফা ফাইভ সফ্টওয়্যার দ্বারা ব্যবহৃত একটি পরিবর্তনশীল ফাইল বিন্যাস যা ওয়েব অ্যাপ্লিকেশনগুলি বিকাশের জন্য ব্যবহৃত হয়। এটি ওয়েব অ্যাপ্লিকেশন দ্বারা ব্যবহৃত ভেরিয়েবল সঞ্চয় করে। আপনি যখন আলফা ফাইভ-এ একটি নতুন ডেভেলপার প্রজেক্ট তৈরি করেন, তখন ASI ফাইলটিও তৈরি হয় এবং আলফা ফাইভ ডেভেলপার প্রজেক্টের পাশে সংরক্ষিত হয়।

ASI ফাইলগুলি আলফা সফ্টওয়্যার আলফা যে কোনও জায়গায় খোলা যেতে পারে।

## ASI ফাইল ফরম্যাট

ASI ফাইলগুলি আলফা ফাইভ মালিকানাধীন ফাইল বিন্যাসে সংরক্ষিত হয়। এগুলি বিকাশকারী প্রকল্পের অংশ হিসাবে অ্যাপ্লিকেশন দ্বারা পড়া এবং লেখা হয়।
