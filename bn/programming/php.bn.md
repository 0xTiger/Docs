## পিএইচপি ফাইল কি?
.php এক্সটেনশন সহ একটি ফাইল ওপেন সোর্স প্রোগ্রামিং ভাষাকে বোঝায়, যা সার্ভার সাইড স্ক্রিপ্ট লিখতে ব্যবহৃত হয়, একটি ওয়েব সার্ভারে কার্যকর করা হয়। এটি সর্বাধিক ব্যবহৃত ওয়েব-স্ক্রিপ্টিং ভাষা যা সাধারণত বড় আকারের ওয়েব অ্যাপ্লিকেশনগুলির বিকাশের জন্য ব্যবহৃত হয়। ওয়েবে সবচেয়ে বড় ব্লগিং সিস্টেম অর্থাৎ ওয়ার্ডপ্রেস এবং বৃহত্তম সামাজিক নেটওয়ার্ক ফেসবুক পিএইচপি ভাষার উপর ভিত্তি করে। এটি উইন্ডোজ, লিনাক্স, ইউনিক্স, ম্যাক ওএস এক্স ইত্যাদির মতো বিভিন্ন প্ল্যাটফর্মের সাথে সামঞ্জস্যপূর্ণ। এছাড়াও এটি বর্তমানে ব্যবহৃত প্রায় সমস্ত সার্ভারে সফলভাবে চলে যেমন অ্যাপাচি, আইআইএস ইত্যাদি। পিএইচপি মাইএসকিউএল সহ বিস্তৃত ডেটাবেস সমর্থন করে।

## পিএইচপি ফাইল ফরম্যাট ##

একটি একক PHP-এ পাঠ্য, CSS, [HTML](/web/html/), JavaScript এবং PHP কোড থাকতে পারে৷ পিএইচপি কোড সার্ভারে কার্যকর করা হয়, এবং ফলাফলটি প্লেইন HTML হিসাবে ব্রাউজারে ফিরে আসে। পিএইচপি ফাইলগুলি যে কোনও টেক্সট এডিটর দিয়ে খোলা যায় এবং জায়গায় সম্পাদনা করা যেতে পারে, যদিও অ্যাডোব ড্রিমওয়েভার, ইক্লিপস পিএইচপি ডেভেলপমেন্ট টুলের মতো অ্যাপ্লিকেশনগুলি পিএইচপি কোড লিখতে এবং সংশোধন করার একটি সুবিধাজনক উপায় সরবরাহ করে।

### বাক্য গঠন ###

একটি PHP স্ক্রিপ্ট সার্ভারে নির্বাহ করা হয় এবং সাধারণ HTML ফলাফল ব্রাউজারে ফেরত পাঠানো হয়। একটি পিএইচপি স্ক্রিপ্ট দিয়ে শুরু হয়<?php and ends with ?>

### পিএইচপি উদাহরণ ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
