## একটি XPD ফাইল কি?

একটি XPD ফাইল একটি XML নথি থেকে তৈরি একটি ওয়েবপেজ ফাইল। প্রায়শই, এটি একটি XSLT রূপান্তরের মাধ্যমে তৈরি হয়। W3 তে [XPD file format specifications](https://www.w3.org/TR/xml-pipeline/) পাওয়া যায় যা মাল্টি-প্রসেস [XML](/web/xml/) অ্যাপ্লিকেশানগুলির আন্তঃক্রিয়াশীলতা বাড়ানোর একটি উপায় সংজ্ঞায়িত করে৷ XML পাইপলাইন ভাষা XML সম্পদের মধ্যে প্রক্রিয়াকরণ সম্পর্ক সংজ্ঞায়িত করে।
