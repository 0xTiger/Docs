## একটি এক্সএইচটিএম ফাইল কি?

An file with .xhtm extension is a webpage file that is written in এক্সএইচটিএমL. It uses XML 1.0 as its standard implementation and is considered a reformulation of HTML 4. এক্সএইচটিএমL এর মত, একটি এক্সএইচটিএম ফাইলে স্ট্যান্ডার্ড এইচটিএমএল ট্যাগ থাকে যা আরও কঠোর XML নির্দেশিকা অনুসরণ করে। এক্সএইচটিএম ফাইলগুলি ওয়েবসাইটের বিষয়বস্তু প্রকাশের জন্য ব্যবহৃত হয় এবং সাধারণত [এক্সএইচটিএমL](/web/xhtml/) এক্সটেনশন ব্যবহার করে৷

আপনি এক্সএইচটিএম ফাইল খুলতে Google Chrome, Apple Safari এবং Mozilla Firefox-এর মতো ওয়েব ব্রাউজার ব্যবহার করতে পারেন।

## এক্সএইচটিএম উদাহরণ

একটি এক্সএইচটিএম নথির নিম্নলিখিত উদাহরণটি XML ঘোষণাকে সংজ্ঞায়িত করে।

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD এক্সএইচটিএমL 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
