## একটি DISCOMAP ফাইল কি?

একটি DISCOMAP ফাইল হল একটি আউটপুট ফাইল যা Microsoft Web Services Discovery Tool, **Disco.exe** দ্বারা তৈরি করা হয়। এটি নির্দিষ্ট URL এ আবিষ্কৃত ASP.NET ওয়েব পরিষেবাগুলির ফলাফলগুলি ধারণ করে৷ এই ধরনের আবিষ্কৃত পরিষেবাগুলির সাধারণ উদাহরণগুলির মধ্যে একটি .DISCO বা .WSDL ফাইলগুলির অবস্থান অন্তর্ভুক্ত৷ যখন Microsoft Web Discovery টুলটি চলে, তখন এটি DISCO ফাইলে পাওয়া XML বিষয়বস্তুর উপর ভিত্তি করে তথ্য আবিষ্কারের জন্য অনুসন্ধান করে। ফলাফলগুলি DISCOMAP ফাইলে সংরক্ষিত থাকে যা Microsoft Visual Studio IDE দিয়ে খোলা যায়।
