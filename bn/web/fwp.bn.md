## একটি FWP ফাইল কি?

.fwp ফাইল সহ একটি ফাইল হল একটি আর্কাইভ ফাইল যা মাইক্রোসফ্ট এক্সপ্রেশন ওয়েব দিয়ে তৈরি করা হয় এবং এতে একটি ডিজাইনের জন্য সমস্ত ফাইল এবং ফোল্ডার থাকে। Microsoft Expression Web Microsoft-এর একটি সাধারণ ওয়েব ডিজাইন সফ্টওয়্যার পণ্য এবং এটি [HTML](/web/html/) সম্পাদকের উপর ভিত্তি করে। এটি 2012 সাল থেকে বন্ধ করা হয়েছে এবং Microsoft দ্বারা বিনামূল্যে ডাউনলোডের জন্য উপলব্ধ। FWP ফাইলগুলি মাইক্রোসফ্টের .cab ফর্ম্যাট হিসাবে রপ্তানি এবং সংরক্ষণ করা যেতে পারে।

## FWP ফাইল ফরম্যাট - আরও তথ্য

ডেভেলপারের রেফারেন্সের জন্য FWP ফাইলগুলির সঠিক ফাইল ফর্ম্যাট স্পেসিফিকেশন উপলব্ধ নয়। যাইহোক, এগুলি বাইনারি ফাইল এবং মাইক্রোসফ্ট এক্সপ্রেশন স্টুডিওতে খোলা যেতে পারে যা অনেক আগে বন্ধ হয়ে গেছে।
