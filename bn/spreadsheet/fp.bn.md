## একটি FP ফাইল কি?

.fp এক্সটেনশন সহ একটি ফাইল ক্লারিসের দ্বারা [FileMaker Pro](https://www.claris.com/filemaker/) সফ্টওয়্যার দিয়ে তৈরি একটি স্প্রেডশীট ফাইল৷ এটি মাইক্রোসফ্ট এক্সেলের মতো সারি এবং কলামে ডেটা সঞ্চয় করে। FileMaker Pro আপনাকে তাদের ডেটা আমদানি করতে CSV এবং XML এর মতো অনেক ফাইল প্রকার টেনে আনতে দেয়।

## FP ফাইল ফরম্যাট

FP ফাইলগুলি মালিকানাধীন ফাইল বিন্যাসে সংরক্ষিত হয় এবং তাদের ফাইল বিন্যাসের বিবরণ সর্বজনীন রেফারেন্সের জন্য উপলব্ধ নয়। যাইহোক, কিছু নির্দিষ্ট API পাওয়া যায় যেগুলি বিভিন্ন রূপান্তর করার জন্য FileMaker Pro প্ল্যাটফর্ম ব্যবহার করে যেমন [fmapi](https://github.com/stevenwhitespacesystems/fm-xml2json) যা [XML](/web/xml/) কে [JSON](/web/json/) তে রূপান্তর করে৷
