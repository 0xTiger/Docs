## একটি DEF ফাইল কি?

একটি DEF ফাইল হল একটি ডেটা ফাইল যা [spreadsheet file format](/spreadsheet/) এ স্মার্টওয়্যার II ডাটাবেস অ্যাপ্লিকেশন দ্বারা তৈরি করা হয়েছে, যা Informix সফ্টওয়্যার দ্বারা তৈরি করা হয়েছে৷ সফ্টওয়্যার II-তে গ্রাফিক্স, ওয়ার্ড প্রসেসর, যোগাযোগ এবং ডাটাবেস সহ স্প্রেডশীট সহ চারটি মডিউল ছিল। স্মার্টওয়্যার II ডাটাবেস একটি স্বতন্ত্র ডাটাবেস যা অন্যান্য অ্যাপ্লিকেশনের সাথেও একত্রিত করা যেতে পারে। DEF ফাইল ফরম্যাটে নিষ্কাশন এবং সংরক্ষণ করা যেতে পারে এমন ডেটার উপর প্রশ্নগুলি চালানো যেতে পারে। DEF স্প্রেডশীট ফাইল স্মার্টওয়্যার সফ্টওয়্যার দিয়ে খোলা যেতে পারে যা এখন বন্ধ করা হয়েছে।
