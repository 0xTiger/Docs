একটি **স্প্রেডশীট** ফাইলে সারি এবং কলাম আকারে ডেটা থাকে। একটি স্প্রেডশীট ফাইল বিভিন্ন ফাইল ফরম্যাটে সংরক্ষণ করা যেতে পারে, প্রতিটিতে অনন্য উপস্থাপনার জন্য আলাদা ফাইল এক্সটেনশন রয়েছে। ডেটা টেক্সট স্ট্রিং, সংখ্যা, তারিখ, মুদ্রা, ইত্যাদির মতো সরল আকারে বা সূত্র হিসাবে কোষে সংরক্ষিত হয় যা রেফারেন্স সেলের মান পরিবর্তন হলে একটি ঘরের মান পরিবর্তন করে।

সাধারণ স্প্রেডশীট ফাইল এক্সটেনশন এবং তাদের ফাইল ফরম্যাটের মধ্যে রয়েছে **[XLSX](/spreadsheet/xlsx/)** (Microsoft Excel Open XML Spreadsheet), **[ODS](/spreadsheet/ods/)** (OpenDocument Spreadsheet) এবং **[XLS](/spreadsheet/xls/)** ( মাইক্রোসফট এক্সেল বাইনারি ফাইল ফরম্যাট)।

স্প্রেডশীট ফাইল ফরম্যাট সম্পর্কিত প্রশ্ন পেয়েছেন? ফাইল ফর্ম্যাট বিশেষজ্ঞদের দ্বারা শেয়ার করা জ্ঞান থেকে উপকৃত হতে আমাদের সম্প্রদায় [forums](https://forum.fileformat.com/c/spreadsheet/6)-এ যান৷

## স্প্রেডশীট ফাইল সহ ওপেন সোর্স API

[Open Source APIs to work with Spreadsheet files](https://products.fileformat.com/spreadsheet/)-এর তালিকা দেখুন।

## স্প্রেডশীট ফাইল এক্সটেনশন এবং অ্যাসোসিয়েটেড ফাইল ফরম্যাটের তালিকা

নিম্নে তাদের ফাইল এক্সটেনশন সহ সাধারণ স্প্রেডশীট ফাইল ফরম্যাটের একটি তালিকা দেওয়া হল।
