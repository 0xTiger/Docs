## একটি DIS ফাইল কি?

A DIS file is a workbook file created with [Oracle Discoverer](https://docs.oracle.com/cd/E28389_01/bi.1111/b40107/overview.htm) software that comprises of several tools for querying, reporting, and data analysis for the Oracle Database. It contains multiple worksheets when data is exported from Oracle Discoverer. Oracle markets Discoverer as a Business Intelligence (BI) product used for for reporting and analysis that works in part of the [Oracle Fusion Middleware](https://docs.oracle.com/cd/E28280_01/bi.1111/b40108/toc.htm) suite. Oracle Discoverer doesn't save the reporting data in DIS file format by default.

## ডিসকভার ওয়ার্কবুকগুলি ডিআইএস ফরম্যাটে কীভাবে ডাউনলোড করবেন?

ওরাকল ডিসকভারার ওয়ার্কবুকগুলি ডাটাবেসে বাইনারি বিন্যাসে সংরক্ষণ করা হয় এবং সরাসরি [DIS files](https://forums.oracle.com/ords/apexds/post/can-i-download-all-discoverer-workbooks-to-my-computer-4127) হিসাবে ডাউনলোড করা যায় না। যাইহোক, ডেটা EEX ফাইল হিসাবে রপ্তানি করা যেতে পারে, এটি ডিসকভারার ডেস্কটপে খুলুন এবং EEX ফাইলগুলিকে DIS ফাইল হিসাবে সংরক্ষণ করতে Save As বিকল্পটি ব্যবহার করুন।

## ডিআইএস ফাইল ফরম্যাট - আরও তথ্য

ডিআইএস ফাইলগুলি ডিস্কে বাইনারি ফাইল হিসাবে সংরক্ষণ করা হয় তবে তাদের অভ্যন্তরীণ ফাইল বিন্যাসের বিবরণ বিকাশকারীর রেফারেন্সের জন্য উপলব্ধ নয়।
