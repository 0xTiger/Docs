## একটি AST ফাইল কি?

একটি AST ফাইল হল একটি স্প্রেডশীট টেমপ্লেট ফাইল যা অ্যাবিলিটি স্প্রেডশীট সফ্টওয়্যার দিয়ে তৈরি করা হয়েছে, অ্যাবিলিটি অফিস স্যুটের একটি স্প্রেডশীট উপাদান। এই টেমপ্লেট ফাইলগুলি স্প্রেডশীট টেমপ্লেটগুলির একটি প্রাথমিক সেট তৈরি করতে ব্যবহৃত হয় যা [AWS](/spreadsheet/aws/) ফাইল তৈরি করতে ব্যবহার করা যেতে পারে। এবিলিটি স্প্রেডশীট সফটওয়্যার দিয়ে AST ফাইল খোলা যায়।

## AST ফাইল ফরম্যাট - আরও তথ্য

AST files are saved to disc as binary files and can be opened in Ability Spreadsheet software. Though fully compatible with Microsoft XLS and XLSX file formats, AST file format specifications are not available publicly for developer's reference. AST files can be saved to disc in [PDF](/pdf/) as well as [XPS](/page-description-language/xps/) file formats.
