## একটি XLA ফাইল কি?

.xla এক্সটেনশন সহ একটি ফাইল হল একটি Microsoft Excel Add-Ins ফাইল ফরম্যাট যা Microsoft Excel এ মডিউল, অতিরিক্ত ফাংশন এবং অন্যান্য টুল যোগ করে। অ্যাড-ইন এক্সেল স্প্রেডশীটগুলিতে অতিরিক্ত কার্যকারিতা প্রদান করে এবং 2007 বা তার পরবর্তী সংস্করণে সমর্থিত। XLA ফাইলে মাইম মিডিয়া টাইপ অ্যাপ্লিকেশন/vnd.ms-excel আছে। এগুলি মাইক্রোসফ্ট এক্সেল ব্যবহার করে খোলা যেতে পারে যা উইন্ডোজের পাশাপাশি MacOS-এর জন্য উপলব্ধ।

## XLA ফাইল ফরম্যাট

XLA ফাইলগুলি বাইনারি ডেটা বিন্যাসের উপর ভিত্তি করে। যাইহোক, এর ফাইল ফরম্যাট স্পেসিফিকেশন সম্পর্কে আরও তথ্য পাওয়া যায় না। সাধারণ ধারণা হল XLA [[MS-XLS] - Excel Binary File Format Structure](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) অনুসরণ করে।
