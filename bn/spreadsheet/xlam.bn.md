## একটি XLAM ফাইল কি? ##

XLAM হল একটি ম্যাক্রো-সক্ষম অ্যাড-ইন ফাইল যা স্প্রেডশীটে নতুন ফাংশন যোগ করতে ব্যবহৃত হয়। একটি অ্যাড-ইন হল একটি সম্পূরক প্রোগ্রাম যা অতিরিক্ত কোড চালায় এবং স্প্রেডশীটের জন্য অতিরিক্ত কার্যকারিতা প্রদান করে। XLAM ফাইল .xlam এক্সটেনশনের সাথে সংরক্ষণ করা হয়। XLAM ফাইলগুলি হল XML-ভিত্তিক ফাইলগুলি [XLSM](/spreadsheet/xlsm/) এবং [XLSX](/spreadsheet/xlsx/) ফাইল ফর্ম্যাটের অনুরূপ এবং সামগ্রিক ফাইলের আকার কমাতে জিপ কম্প্রেশনের সাথে সংরক্ষিত হয়৷

### উদাহরণ ###

```cmd
Public Function Add(num1 As Double, num2 As Double)
Add = num1 + num2
End Function
```

এর পরে, .xlam এক্সটেনশন দিয়ে ফাইলটি সংরক্ষণ করুন।
