You must be familiar with **PPTX** and **PPT** extension files, these are Presentation file formats that store collection of records to accommodate presentation data such as: slides, shapes, text, animations, video, audio and embedded objects. A presentation can be saved/converted into other file formats as well such as PDF, BMP, PNG, JPEG, and XPS.

সাধারণ উপস্থাপনা ফাইল এক্সটেনশন এবং তাদের সম্পর্কিত ফাইল ফর্ম্যাটগুলির মধ্যে রয়েছে **[PPTX](/presentation/pptx/)**, **[PPT](/presentation/ppt/)** এবং **[ODP](/presentation/odp/)**৷

উপস্থাপনা ফাইল ফর্ম্যাট সম্পর্কিত প্রশ্ন পেয়েছেন? ফাইল ফর্ম্যাট বিশেষজ্ঞদের দ্বারা শেয়ার করা জ্ঞান থেকে উপকৃত হতে আমাদের সম্প্রদায় [forums](https://forum.fileformat.com/c/presentation/8)-এ যান৷

## উপস্থাপনা ফাইলের জন্য ওপেন সোর্স APIs

[Open Source APIs for working with Presentation File Formats](https://products.fileformat.com/presentation/)-এর তালিকা দেখুন।

## প্রেজেন্টেশন ফাইল এক্সটেনশন এবং অ্যাসোসিয়েটেড ফাইল ফরম্যাটের তালিকা

জনপ্রিয় পাওয়ারপয়েন্ট এবং অন্যান্য প্রেজেন্টেশন ফাইল এক্সটেনশন এবং তাদের সংশ্লিষ্ট ফাইল ফরম্যাটের একটি তালিকা নিচে দেওয়া হল।
