## একটি MBW ফাইল কি? ##

.mbw এক্সটেনশন সহ একটি ফাইল প্রাথমিকভাবে ফায়ারসেজ সলিউশন দ্বারা MBRWizard এর সাথে যুক্ত। মাস্টার বুট রেকর্ড উইজার্ড (MBRWizard) আপনার কম্পিউটারের MBR ব্যাক আপ করে এবং এটি একটি MBW ফাইলে সংরক্ষণ করে যা প্রয়োজনে পুনরুদ্ধার করা যেতে পারে। MBR এবং VBR (ভলিউম বুট রেকর্ড) কপি ছাড়াও, এই ফাইলগুলিতে GPT, ব্যবহারকারী-সংজ্ঞায়িত সেক্টর বা পুরো ডিস্কের ছবি থাকতে পারে। MBR আপনার হার্ড ড্রাইভে একটি অত্যাবশ্যক, 512-বাইট দীর্ঘ রেকর্ড যা আপনার অপারেটিং সিস্টেমকে বুট করার অনুমতি দেয়। এটি পার্টিশন টেবিল এবং ডিস্ক মিডিয়া সনাক্তকারী একটি 32-বিট স্বাক্ষরও ধরে রাখতে পারে।

## সমর্থিত অপারেটিং সিস্টেম ##

MPW ফাইল ফরম্যাট শুধুমাত্র Windows অপারেটিং সিস্টেম দ্বারা সমর্থিত।

## ফাইল খোলার সময় সম্ভাব্য সমস্যা ##

আপনি যদি MBW ফাইলটি খুলতে এবং চালাতে সক্ষম না হন তবে নিম্নলিখিত সম্ভাব্য সমস্যাগুলির মধ্যে একটি হতে পারে:

 *  দূষিত MBW ফাইল
 *  রেজিস্ট্রি এন্ট্রিতে MBW ফাইলের ভুল লিঙ্ক
 *  Deleted description of the MBW from the Windows registry
 *  একটি অবাঞ্ছিত ম্যালওয়্যার সহ একটি সংক্রামিত ফাইল৷
 *  MBW ফাইলটি পরিচালনা করার জন্য কম্পিউটারে পর্যাপ্ত হার্ডওয়্যার সংস্থান নেই
 *  একটি MBW ফাইল খুলতে কম্পিউটার দ্বারা ব্যবহৃত ড্রাইভারগুলি পুরানো৷
