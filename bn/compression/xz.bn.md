## একটি XZ ফাইল কি?

XZ হল একটি সংকুচিত ফাইল বিন্যাস যা LZMA2 কম্প্রেশন অ্যালগরিদম ব্যবহার করে। এটি জনপ্রিয় gzip এবং bzip2 ফর্ম্যাটের প্রতিস্থাপন হিসাবে ডিজাইন করা হয়েছিল এবং এই পুরানো মানগুলির তুলনায় অনেকগুলি সুবিধা প্রদান করে৷ XZ ফাইলগুলি অনেক প্ল্যাটফর্মে ভালভাবে সমর্থিত, এবং দ্রুত এবং সহজে ডিকম্প্রেস করা যায়। যদিও এগুলি [ZIP](/compression/zip/) বা [RAR](/compression/rar/) ফাইলগুলির মতো সাধারণ নয়, XZ সংরক্ষণাগারগুলি কম্প্রেশন গুণমানকে ত্যাগ না করেই প্রচুর পরিমাণে ডেটা সংরক্ষণ করতে ব্যবহার করা যেতে পারে৷ আপনি যদি বড় ফাইলগুলিকে সংকুচিত বা ডিকম্প্রেস করতে চান তবে XZ ফাইল এক্সটেনশনটি বিবেচনা করার মতো।

## XZ ফাইল ফরম্যাট

XZ ফাইল তৈরি করা হয় এবং ডিস্কে বাইনারি ফাইল হিসাবে সংরক্ষণ করা হয়। এটি ডেটা সংকুচিত করতে LZMA অ্যালগরিদম ব্যবহার করে এবং 50% পর্যন্ত কম্প্রেশন অনুপাত অর্জন করতে পারে। XZ ফাইলগুলি সাধারণত সফ্টওয়্যার বিতরণ এবং ব্যাকআপ সংরক্ষণাগারগুলির জন্য ব্যবহৃত হয়। XZ ইউটিলিটি ব্যবহার করে এগুলিকে ডিকম্প্রেস করা যেতে পারে, যা বেশিরভাগ লিনাক্স ডিস্ট্রিবিউশনে অন্তর্ভুক্ত।

## লিনাক্স/ইউনিক্সে XZ ফাইল আনজিপ করুন

XZ ফাইল আনজিপ করার জন্য, প্রথমে `xz-utils` প্যাকেজ ইনস্টল করুন:
```
$ sudo apt-get install xz-utils
```

.xz ফাইল বের করতে `unxz` কমান্ড ব্যবহার করুন:
```
$ unxz compressed_xz_file.xz
```

অথবা XZ এর --decompress বিকল্পের সাথে ব্যবহার করুন:
```
$ xz --decompress compressed_xz_file.xz
```
