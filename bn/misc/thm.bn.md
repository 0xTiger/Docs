## একটি THM ফাইল কি?

একটি THM ফাইল হল Sony Ericsson মোবাইল ফোনের জন্য তৈরি একটি থিম ফাইল৷ এটি বেশ কয়েকটি সেটিং পরামিতি সহ শেষ ব্যবহারকারীর জন্য গ্রাফিকাল ইন্টারফেস পরিবর্তন করতে ব্যবহৃত হয়। এর মধ্যে পাঠ্যের রঙ, আইকন, ব্যাকগ্রাউন্ড ওয়ালপেপার, কাস্টম উইন্ডো এবং অন্যান্য ছবির সেটিংস অন্তর্ভুক্ত রয়েছে। থিম ফাইল তৈরির জন্য Sony বিনামূল্যে এরিকসন থিম ক্রিয়েটর সফ্টওয়্যার প্রদান করেছে। এটি বিদ্যমান THM ফাইলগুলি খুলতেও ব্যবহার করা যেতে পারে। THM ফাইলগুলিতে মিডিয়া টাইপ অ্যাপ্লিকেশন/vnd.eri.thm আছে।

## Sony Ericsson THM ফাইল ফরম্যাট - আরও তথ্য

একটি THM ফাইল আসলে একটি ধারক ফাইল যা অন্যান্য ফাইলগুলির একটি গুচ্ছ ধারণ করে। এটি [TAR](/compression/tar/) আর্কাইভ ফাইল ফরম্যাটে সংরক্ষিত আছে এবং TAR ফাইল ফরম্যাট সমর্থন করে এমন যেকোনো স্ট্যান্ডার্ড ডিকম্প্রেশন সফ্টওয়্যার দিয়ে বের করা যেতে পারে।

A THM file contains an XML file, several image file sin [.png](/image/png/), [.pjg](/image/jpeg/), [.gif](/image/gif/) formats, or [.swf](/page-description-language/swf/) files.

## কিভাবে Sony Ericsson থিম তৈরি করবেন?

Sony থিম ক্রিয়েটর সফ্টওয়্যার Sony Ericsson মোবাইলের জন্য নতুন থিম তৈরি করতে ব্যবহার করা যেতে পারে। ব্যবহারকারীরা থিম তৈরি করতে Sony Themes Creator Developer Guide অনুসরণ করতে পারেন৷
