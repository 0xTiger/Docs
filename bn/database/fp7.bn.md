## একটি FP7 ফাইল কি?

A file with .fp7 extension is a database file created with FileMaker Pro versions 7-11. এটি রেকর্ড সহ টেবিলে ডেটা সঞ্চয় করে যেখানে প্রতিটি রেকর্ডে এক বা একাধিক ক্ষেত্র থাকে। FP7 ফাইলগুলিতে ব্যক্তিগত ডেটাবেস ডেটা এবং সাংগঠনিক রেকর্ড সহ বড় ডেটা সংরক্ষণ করার ক্ষমতা রয়েছে। FileMaker Pro এর একটি আরও নতুন সংস্করণ হল [FMP12](/database/fmp12/) যেটি FileMake Pro সংস্করণ 12 লঞ্চ করার সাথে সাথে প্রবর্তন করা হয়েছিল৷ পাঠ্য ডেটা ছাড়াও, FP7 ফাইলটি ফর্ম, চার্ট এবং অন্যান্য ডাটাবেস অবজেক্টও ধারণ করতে পারে৷ FP7 ফাইল ফাইলমেকার প্রো অ্যাডভান্সড দিয়ে খোলা যায়।

## FP7 ফাইল ফরম্যাট

FP7 ফাইলগুলি ফাইলমেকারের মালিকানাধীন ফাইল বিন্যাসে বাইনারি ফাইল হিসাবে ডিস্কে সংরক্ষণ করা হয়। FP7 ফাইল বিন্যাসের অভ্যন্তরীণ ফাইল বিন্যাসের বিবরণ সর্বজনীনভাবে উপলব্ধ নয়।
