## একটি FP5 ফাইল কি?

An FP5 file is a database file created with FileMaker Pro version 5 or 6. এটি টেবিলের ভিতরে রেকর্ড আকারে ব্যবহারকারীর পাশাপাশি সাংগঠনিক তথ্য সংরক্ষণ করে। FileMaker Pro হল একটি ডাটাবেস ম্যানেজমেন্ট সিস্টেম যা Macintosh এবং Windows অপারেটিং সিস্টেমের জন্য সমর্থন করে। FP5 হল একটি পুরানো ফাইল ফরম্যাট যা তৎকালীন উপলব্ধ FileMaker Pro সংস্করণ দ্বারা সমর্থিত ছিল। সর্বশেষ FileMaker Pro ডাটাবেস ফরম্যাট হল [FMP12](/database/fmp12/) যেটি FileMaker Pro v 12 লঞ্চ করার সাথে সাথে প্রবর্তন করা হয়েছিল। অন্যান্য ভালভাবে ব্যবহৃত ডাটাবেস ফাইল ফরম্যাটটি ছিল [FP7](/database/fp7/) যা FMP12 দ্বারা প্রতিস্থাপিত হয়েছিল।

## FP5 ফাইল ফরম্যাট - আরও তথ্য

FP5 ফাইল বিন্যাসের অভ্যন্তরীণ ফাইল বিন্যাস বিবরণ বিকাশকারীর রেফারেন্সের জন্য সর্বজনীনভাবে উপলব্ধ নয়। পুরানো FileMaker Pro ডাটাবেস ফাইল ফরম্যাট যেমন [FP5 and FP7 can be upgraded to latest FMP12](https://fmhelp.filemaker.com/help/16/fmp/en/index.html#page/FMP_Help/converting-files.html) FileMaker Pro থেকে বিস্তারিত নির্দেশিকা ব্যবহার করে।
