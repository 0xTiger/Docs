## একটি ADN ফাইল কি?

.adn এক্সটেনশন সহ একটি ফাইল হল একটি মাইক্রোসফ্ট অ্যাক্সেস ফাঁকা টেমপ্লেট ফাইল যা পূর্ব-সংরক্ষিত সেটিংস সহ অ্যাক্সেস প্রকল্প ফাইল তৈরি করার জন্য ব্যবহার করা হয়। মাইক্রোসফ্ট অ্যাক্সেস প্রকল্প ফাইলগুলি ADP ফাইল হিসাবে সংরক্ষণ করা হয় এবং এতে ডেটাবেস সংযোগ এবং প্রকল্প স্টার্টআপ সেটিংসের মতো তথ্য থাকে। ADN ফাইলগুলি Microsoft Access-এর 2007 এবং 2010 সংস্করণের জন্য বিদ্যমান। Microsoft Access অন্যান্য ফাইল ফরম্যাট সমর্থন করে যেমন [MDB](/database/mdb/), [ACCDB](/database/accdb/), [ACCDC](/database/accdc/), [ACCDE](/database/accde/), এবং এই ADN ফাইলগুলি ছাড়াও আরও অনেক কিছু। ADN ফাইলগুলি Office 365 এবং Microsoft Access ব্যবহার করে খোলা যেতে পারে।

## ADN ফাইল ফরম্যাট - আরও তথ্য

ADN হল ডাটাবেস টেমপ্লেট ফাইল যা ডিস্কে বাইনারি ফাইল হিসাবে সংরক্ষণ করা হয়। ADN ফাইলগুলির অভ্যন্তরীণ ফাইল বিন্যাসের বিবরণ বিকাশকারীর রেফারেন্সের জন্য উপলব্ধ নয়৷
