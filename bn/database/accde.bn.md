## একটি ACCDE ফাইল কি?

.accde এক্সটেনশন সহ একটি ফাইল হল একটি Microsoft Access 2007 ফাইল ফরম্যাট যাতে কাস্টম ডাটাবেস কোড রক্ষা করার জন্য শুধুমাত্র Execute ডাটাবেস ফাইল তৈরি করা যায়। এটি সমস্ত VBA কোডকে শুধুমাত্র এক্সিকিউটেবল কম্পাইল করে যাতে এটি দেখা বা সম্পাদনা করা যায় না। ACCDE ফাইল ফরম্যাট হল MDE ফাইল ফরম্যাটের প্রতিস্থাপন যা মাইক্রোসফ্ট অ্যাক্সেসের পূর্ববর্তী সংস্করণগুলি ব্যবহার করেছিল। ACCDE হল শুধুমাত্র এক্সিকিউটেবল মোডে থাকা [ACCDB](/database/accdb/) ফাইল এবং কোনো সোর্স কোডের অনুপস্থিতির কারণে আরও কমপ্যাক্ট। এই ফাইলগুলির কম্প্যাক্ট আকার মেমরি ব্যবহার উন্নত করার পাশাপাশি অ্যাপ্লিকেশন কর্মক্ষমতা বাড়ায়। ACCDE ফাইলগুলি Microsoft Access 365 দিয়ে খোলা যেতে পারে।

## ACCDE ফাইল ফরম্যাট

সমস্ত Microsoft Access 2007+ ফাইল জেট ACE (অ্যাক্সেস ডাটাবেস ইঞ্জিন) ব্যবহার করে যা ACCDB ফাইল ফরম্যাট পরিবার ব্যবহার করে। যাইহোক, মাইক্রোসফট একটি ওপেন সোর্স বাস্তবায়ন বা বিস্তারিত প্রযুক্তিগত স্পেসিফিকেশন প্রদান করে না।
