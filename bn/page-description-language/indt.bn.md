## একটি INDT ফাইল কি?

A file with .indt extension is an Adobe InDesign layout template file that contains predefined settings. The design and contents of these files include templates of newsletters, books, magazines, flyer, and print media. Once editing is finished, the INDT file can be saved to disc as [INDD](/page-description-language/indd/) file. User can also create new INDT file from within the InDesign and convert it to other formats such as [PDF](/pdf/), [EPUB](/ebook/epub/), or [JPEG](/image/jpeg/).

## INDT ফাইল ফরম্যাট - আরও তথ্য

Adobe InDesign সফ্টওয়্যারের পাশাপাশি অনলাইনে বেশ কিছু টেমপ্লেট প্রদান করে। এই টেমপ্লেটগুলি ব্যবহারকারীদের তাদের অনলাইন বা প্রিন্ট মিডিয়া ডিজাইন করার জন্য একটি সূচনা পয়েন্ট প্রদান করে। INDT ফাইলগুলি হালকা, প্রচুর ডিজাইনের বৈশিষ্ট্য রয়েছে (আকৃতি, পাঠ্য, বিভিন্ন রঙ এবং ফিল্টার), এবং অসীম রেজোলিউশন রয়েছে৷
