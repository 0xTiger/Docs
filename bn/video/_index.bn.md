টেলিভিশন, সিনেমার স্ক্রীন এবং ফেসবুক, টুইটার, ইউটিউব এবং আরও অনেকের মতো সোশ্যাল মিডিয়া চ্যানেলের মতো মিডিয়া ডিভাইসগুলিতে আমরা প্রতিদিন যে ভিডিওগুলি দেখি সেগুলি সম্পর্কে সবাই পরিচিত৷ কিন্তু আপনি কি কখনো ভেবে দেখেছেন কিভাবে এই ভিডিওগুলো সেভ করা হয় এবং কোন ফরম্যাটে এগুলো দেখার জন্য আপলোড করা যায়? আপনার ভিডিও বিভিন্ন স্টোরেজ মিডিয়াতে হোস্ট করার জন্য সেরা ফাইল বিন্যাস কি? ভিডিও ফাইল ফরম্যাটগুলি ভিডিও সংরক্ষণ করার জন্য এবং বিভিন্ন উদ্দেশ্যে ব্যবহার করার জন্য বিভিন্ন ধরণের ভিডিও নিয়ে গঠিত। আপনি যে ভিডিও ফাইল ফর্ম্যাটগুলি দেখেছেন তার মধ্যে রয়েছে [AVI](/video/avi/), [MOV](/video/mov/), [WMV](/video/wmv/), [M4V](/video/m4v/), এবং [MP4](/video/mp4/)৷

ভিডিও ফাইল ফরম্যাট সম্পর্কিত প্রশ্ন পেয়েছেন? ফাইল ফর্ম্যাট বিশেষজ্ঞদের দ্বারা শেয়ার করা জ্ঞান থেকে উপকৃত হতে আমাদের সম্প্রদায় [forums](https://forum.fileformat.com/c/video/27)-এ যান৷

## ভিডিও ফাইল ফরম্যাটের জন্য ওপেন সোর্স API

[Open Source APIs for working with Video File Formats](https://products.fileformat.com/video/) দেখুন।

## ভিডিও ফাইল এক্সটেনশন এবং অ্যাসোসিয়েটেড ফাইল ফরম্যাটের তালিকা

নিম্নলিখিত জনপ্রিয় ভিডিও ফাইল ফরম্যাটের একটি তালিকা রয়েছে যা ভিডিও সম্পর্কিত প্রায় সমস্ত দৈনন্দিন কাজে তাদের ব্যবহার খুঁজে পায়।
