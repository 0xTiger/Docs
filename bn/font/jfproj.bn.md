## একটি JFPROJ ফাইল কি?

A file with .jfproj extension is a JSON based custom font created with Microsoft Font Maker application on Windows 10. ট্যাবলেটের মতো টাচ স্ক্রিন ডিভাইসের সাথে ইলেকট্রনিক কলম ব্যবহার করে এগুলি তৈরি করা যেতে পারে যদিও কম্পিউটারের ক্ষেত্রে মাউসের মতো অন্যান্য কম্পিউটার পেরিফেরিয়ালগুলিও ব্যবহার করা যেতে পারে। JFPROJ ফাইলগুলিতে গ্লিফ রয়েছে যা অক্ষর এবং শব্দের মধ্যে ব্যবধান এবং ফন্টের আকার নির্ধারণ করে অক্ষরগুলিকে পর্দায় রেন্ডার করতে ব্যবহৃত হয়। Microsoft Font Maker অ্যাপ উইন্ডোজ স্টোরে ডাউনলোডের জন্য উপলব্ধ।
