## .CHCK Dosyası Nedir?

Microsoft Exchange Server üzerinde yapılan sağlık izleme kontrolleri sonucunda CHCK dosyası oluşturulur. Sunucu veritabanının genel durumu ve işleyişi hakkında bilgiler içerir. Microsoft Exchange Server öncelikle e-posta sunucusu olarak kullanılır ve iş bağlantılarınızı ve takvimlerinizi korumanıza olanak tanır. Exchange sunucusu yöneticileri, veritabanı sağlığını kontrol etmek için EseUtil'i kullanır.
