## SDF dosyası nedir?

A Spatial Data File (SDF) is a Geodatabase file format that was developed by Autodesk. It stores geospatial data and is used by Autodesk GIS program MapGuide and AutoCAD Map 3D. SDF file format is based on the single file database file format SQLite3. Büyük mekansal bilgi veri kümeleri için optimize edilmiş bir dosya formatı olarak kabul edilir.

Autodesk AutoCAD Map 3D 2022 ve Autodesk AutoCAD Civil 3D 2023'ü kullanarak bir SDF dosyasını açabilirsiniz.

## SDF Dosya Formatı - Daha Fazla Bilgi

SDF dosya formatı diskte ikili dosyalar olarak saklanır. Verilerin ikili serileştirilmesi için düşük seviyeli depolama bileşenlerini kullanan SQLite veritabanını temel alır. Bir SDF dosyası, dosya başına birden fazla özellik sınıfı ve her özellik sınıfı için birden fazla geometri özelliği içerir. Bir SDF dosyasındaki verilere, her geometri özelliğinin bir R-ağacı kullanılarak indekslenmesi nedeniyle yüksek hızda erişilebilir.
