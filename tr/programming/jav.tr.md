## Jav dosyası nedir?

Jav dosyası, Java programlama dilinde oluşturulmuş bir Java kaynak kodu dosyasıdır. Java kaynak dosyasına benzer ve Java yorumlayıcıları ile derlenebilir. [Java](/tr/programming/java/) dosyaları gibi, Jav dosyası da Java uygulamalarının geliştirilmesinde kullanılır. Bir Java projesinde bulunan birçok kaynak kod dosyasından biri olabilir.

## Jav Dosya Formatı

Jav dosyaları metin dosyaları olarak saklanır ve herhangi bir metin düzenleyiciyle açılabilir. Jav dosya türleri genellikle Java dosya türüyle değiştirilmiştir ve genellikle Java uygulamaları yazmak için kullanılır. Jav kodları, Java dosyalarıyla çalışmayı kolaylaştıran Eclipse ve IntelliJ Idea gibi IDE'lerle yazılabilir.

### Jav Dosya Formatı Örneği

```java
/*
The example code prints
Hello World from Java to the console.
*/
    public class JavaApp {
        public static void main(String[] args) {
            System.out.println("Hello World."); // Prints the string to the console.
        }
    }
```
