## NX dosyası nedir?

NX dosyası, **LowRes NX** sanal oyun konsoluyla kullanılmak üzere oluşturulmuş bir video oyunu dosyasıdır. BASIC programlama dilinde yazılmıştır ve oyunun nasıl çalıştığını tanımlayan oyunun kaynak kodunu içerir. Yüklenebilecek uygulama ile kullanmak üzere kendi NX dosyalarınızı oluşturabilirsiniz ve kullanıcılar oyunu oynayabilir.

Bir NX dosyasını bir metin düzenleyicide, LowRes NX'te veya Microsoft Visual Studio Code gibi herhangi bir kaynak kodu düzenleyicisinde açabilirsiniz.

## NX Dosya Formatı

Bir NX dosyası, oyun talimatlarını içeren düz metin dosyası olarak diske kaydedilir. Her NX dosyasında, birden fazla kartuş ROM girişinden oluşan oyunun kaynak kodu bulunur. Bu ROM girişleri, grafikler, seviye haritaları, ses ve oyuna ilişkin diğer kaynak kodu olmayan veriler gibi oyun bileşeni bilgilerine sahiptir. Bir programın ROM girişleri, LowRes NX ile birlikte gelen NX düzenleme programları kullanılarak eklenebilir veya düzenlenebilir.

## NX Dosyaları Nasıl Oluşturulur?

### PC'de NX Dosyaları Oluşturun

Aşağıdakileri kullanarak PC'de NX dosyaları oluşturabilirsiniz:

* Metin Düzenleyicileri - Microsoft Notepad, Notepad++ veya Github Atom gibi
* Kaynak Kodu Düzenleyicileri - Microsoft VIsual Studio Code gibi

### iOS'ta NX Dosyaları Oluşturun

LowRes NX, geliştiricilerin bu uygulama içinden NX dosyaları oluşturmak için kullanabileceği iOS sürümüne sahiptir.
