## PHP dosyası nedir?
.php uzantılı bir dosya, bir web sunucusunda yürütülmek üzere sunucu tarafı komut dosyaları yazmak için kullanılan açık kaynaklı programlama dilini ifade eder. Genellikle büyük ölçekli web uygulamalarının geliştirilmesi için kullanılan en yaygın kullanılan web komut dosyası dilidir. Web'deki en büyük blog sistemi, yani WordPress ve en büyük sosyal ağ Facebook, PHP dilini temel alır. Windows, Linux, Unix, Mac OS X, vb. gibi çeşitli platformlarla uyumludur. Ayrıca günümüzde kullanılan Apache, IIS, vb. gibi hemen hemen tüm sunucularda başarıyla çalışır. PHP, MySQL dahil olmak üzere çok çeşitli veritabanlarını destekler.

## PHP Dosya Biçimi ##

Tek bir PHP metin, CSS, [HTML](/tr/web/html/), JavaScript ve PHP kodu içerebilir. PHP kodu sunucuda yürütülür ve sonuç tarayıcıya düz HTML olarak döndürülür. PHP dosyaları herhangi bir metin düzenleyiciyle açılabilir ve yerinde düzenlenebilir, ancak Adobe Dreamweaver, Eclipse PHP Geliştirme araçları gibi uygulamalar PHP kodunu yazmak ve değiştirmek için uygun bir yol sunar.

### Sözdizimi ###

Sunucuda bir PHP betiği yürütülür ve düz HTML sonucu tarayıcıya geri gönderilir. Bir PHP betiği şununla başlar:<?php and ends with ?>

### PHP Örneği ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
