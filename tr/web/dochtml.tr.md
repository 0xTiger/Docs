## DOCHTML dosyası nedir?

DOCHTML dosyası, Microsoft Word ile oluşturulan bir belgenin HTML biçimidir. Oluşturulan belgenin web sunucusuna yayınlamak için kullanılabilecek [HTML](/tr/web/html/) sürümüdür. DOCHTML dosyaları, dönüştürülen kelime belgesinin içeriğini görüntülemek için web tarayıcısında açılabilir. DOCHTML dosyasının görünümü ve görünümü, orijinal Word belgesininkiyle aynıdır.

## DOCHTML Dosya Biçimi

DOCHTML dosyaları, web sunucularında yayınlanmaya hazır HTML biçiminde diske kaydedilir. Microsoft Word, belgeleri DOCHTML dosyaları olarak kaydetme desteğini durdurdu. Bu, şu anda yaygın olarak kullanılan [.mht](/tr/web/mht/) ve [.mhtml](/tr/web/mhtml/) dosya biçimleriyle değiştirildi.
