## DOTHTML dosyası nedir?

DOTHTML dosyası, HTML sürümünde kaydedilmiş bir Microsoft Word şablon belgesidir. Web tarayıcısında açılabilir ve içeriği diğer herhangi bir web sayfası gibi görüntülenir. DOTHTML dosyaları, bu dosyaları oluşturmak için önceden biçimlendirilmiş ayarlar içeren Microsoft Word .DOT dosyalarından oluşturulur. Bir [DOT](/tr/word-processing/dot/) dosyası DOCHTML'ye dönüştürüldüğünde, tüm bu ayarlar korunur ve dönüştürülen DOTHTML dosyası, orijinal DOT dosyasına benzer biçimlendirmeyle görüntülenir. Bu şekilde, yayınlanan şablonlar web tarayıcısında görüntülenebilir.

## DOTHTML Dosya Biçimi - Daha Fazla Bilgi

DOTHTML dosyaları [HTML](/tr/web/html/) dosyaları olarak kaydedilir ve herhangi bir HTML düzenleyiciyle açılabilen HTML kaynak kodunu içerir. İlk günlerde Microsoft Word, [DOCHTML](/tr/web/dochtml/), [DOCMHTML](/tr/web/docmhtml/) gibi tüm belge biçimlerinin HTML sürümlerini sunuyordu, ancak daha sonra bu HTML tabanlı sürümler kullanımdan kaldırıldı. . Sonuç olarak, DOTHMLT de artık desteklenmemektedir.
