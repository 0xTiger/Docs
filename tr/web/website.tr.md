## WEB SİTESİ dosyası nedir?

WEB SİTESİ dosyası, Microsoft Internet Explorer (IE) ile oluşturulan bir web sitesi kısayol dosyasıdır. Web sitesinin favicon'una işaret eden bir URL ve bir simge (ICO) dosyası içerir. Simge, kısayolu görüntülemek için kullanılır. WEB SİTESİ dosya formatı, IE sürüm 9'un piyasaya sürülmesiyle tanıtıldı. IE'nin önceki sürümleri, web sayfalarına kısayolları [.url](/tr/web/url/) dosyaları olarak kaydetmek için kullanılıyordu.

## WEB SİTESİ Dosya Formatı

Bir .web sitesi dosyası, içeriği herhangi bir metin düzenleyicide görüntülenebilen düz metin dosyaları olarak diske kaydedilir. Diğer içeriğin yanı sıra kısayolu olduğu hedef URL'yi içerir. Hedef web sitesini IE dışında bir tarayıcıda açmak isterseniz, web sitesini görüntülemek için bu URL'yi kopyalayıp Google Chrome, Safari veya Firefox gibi diğer tarayıcılara yapıştırabilirsiniz.

## .WEBSITE dosyası nasıl oluşturulur?

Bir WEB SİTESİ dosyası, adres çubuğundaki URL'ye tıklayıp bilgisayarınızda masaüstü veya belgeler gibi istediğiniz konuma sürükleyerek IE'de oluşturulabilir. Bu, o internet konumuna giden kısayolu oluşturur ve kısayola çift tıklanarak açılabilir. Kısayolu 'URL' dosyası olarak kaydetmek istiyorsanız, hedef sitenin favicon'unu masaüstüne sürüklemeden önce SHIFT tuşunu basılı tutun.
