## CODASITE dosyası nedir?

Bir CODASITE dosyası, [Coda](https://coda.io/) web sayfaları oluşturma ve yayınlama yazılımı ile oluşturulan bir web sitesi proje dosyasıdır. Projenin parçası olan tüm dosyaların bir listesini içerir. Bu, [HTML](/tr/web/html/) dosyaları, basamaklı stil sayfası [CSS](/tr/web/css/) dosyaları, Javascript [JS](/tr/web/js/) dosyaları ve diğer ilgili destekleyici dosyalar olarak web sayfalarını içerir. CODASITE proje dosyaları ayrıca web sunucusuna web sitesi yayınlama bilgileri içerir.

## CODASITE Dosya Biçimi

CODASITE proje dosyası, ~/​Library/​Application Support/​Coda 2/​Sites/​ dizini içindeki ikili dosyalar olarak diske kaydedilir. Coda kullanıcı arabirimi, Coda sitelerini yönetmek için arabirim sağlar, çünkü bunları manuel olarak açmak, yanlış proje ayarlarına yol açabilir ve sorunlar yaratabilir.
