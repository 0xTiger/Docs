## URL dosyası nedir?

.url uzantılı bir dosya, herhangi bir web sitesi adresiyle bilgisayara kaydedilebilen bir web sitesi URL kısayoludur. URL çift tıklanarak açıldığında web sitesi varsayılan tarayıcıda açılacak ve sayfa kullanıcıya gösterilecektir. URL dosyalarına genellikle web sitelerine giden kısayol dosyaları denir ve varsayılan tarayıcının simgesiyle temsil edilir. URL dosyaları, Microsoft Notepad, Notepad++ veya Apple TextEdit gibi herhangi bir metin düzenleyicide açılabilir ve düzenlenebilir.

## URL Dosya Biçimi - Daha Fazla Bilgi

URL dosyaları düz metin biçiminde kaydedilir ve herhangi bir metin düzenleyicide açılıp görüntülenebilir. Bunlar, kullanıcının diğer taramaları gerçekleştirdiği varsayılan web tarayıcısında açılır.
