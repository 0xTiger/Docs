## XHTM dosyası nedir?

.xhtm uzantılı bir dosya, XHTML ile yazılmış bir web sayfası dosyasıdır. Standart uygulaması olarak XML 1.0'ı kullanır ve HTML 4'ün yeniden formüle edilmiş hali olarak kabul edilir. XHTML gibi, bir XHTM dosyası da daha katı XML yönergelerini izleyen standart HTML etiketleri içerir. XHTM dosyaları web sitesi içeriğini yayınlamak için kullanılır ve genellikle [XHTML](/tr/web/xhtml/) uzantısını kullanır.

XHTM dosyalarını açmak için Google Chrome, Apple Safari ve Mozilla Firefox gibi web tarayıcılarını kullanabilirsiniz.

## XHTM Örneği

Aşağıdaki XHTM belgesi örneği, XML bildirimlerini tanımlar.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
