## WHTT dosyası nedir?

WHTT dosyası, bir web sitesinin [HTTrack Website Copier](https://www.httrack.com/) yazılımı kullanılarak indirilen çevrimdışı bir sürümüdür. [HTML](/tr/web/html/) web sayfaları, resimler, stil sayfaları, bağlantılar ve dizinler dahil bir web sitesinin tüm içeriğini içerir. Bir web sitesinin tüm sayfaları yinelemeli olarak indirilir ve çevrimdışı ayna olarak diske kaydedilir. Bir web sitesi HTTrack yazılımı tarafından indirildiğinde, internete bağlanmaya gerek kalmadan çevrimdışı olarak görüntülenebilir. İndirilen web sitesini görüntülemek için, yansıtılan web sitesinin sayfasını tarayıcıda açmanız yeterlidir; her resim/bağlantı düzgün bir şekilde yüklenecektir.

## WHTT Dosya Biçimi

WHTT dosyaları, THML web sayfalarının, resimlerin, stil sayfalarının ve referans bağlantılarının bir kombinasyonu olarak kaydedilir.
