## DOCMHTML dosyası nedir?

DOCMHTML dosyası, Microsoft Word'ün önceki sürümleri tarafından kaydedilen eski bir web arşivi çeşididir. Artık Microsoft Word'ün en son sürümü tarafından desteklenmemektedir. Bir DOCMHTML dosyası, orijinal .doc kelime belgesinin HTML temsilini içerir. Microsoft Word'de açılabileceği gibi web tarayıcısında da açılabilir. Bir DOCMHTML yayınlandığında, kullanıcıların web isteği yoluyla erişebileceği bir web sunucusunda barındırılabilir. Benzer bir başka biçim de artık üretilmeyen [DOCHTML](/tr/web/dochtml/) formatıdır.

## DOCMHTML Dosya Biçimi - Daha Fazla Bilgi

DOCMHTML dosyaları HTML biçiminde kaydedilmek için kullanıldı. Notepad ve Notepad++ gibi bir metin düzenleyicide açılabilir. Microsoft Word, eski olarak işaretlendikten sonra, kelime belgesini [MHTML](/tr/web/mhtml/) ve [MHT](/tr/web/mht/) dosyaları olarak kaydetme seçeneklerini tanıttı. Bu yeni dosya formatları web tarayıcıları ile açılabilir.
