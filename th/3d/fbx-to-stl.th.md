## เกี่ยวกับ FBX ถึง STL

FBX หรือ FilmBox เป็นรูปแบบไฟล์ 3 มิติยอดนิยมที่พัฒนาโดย Kaydara สำหรับ MotionBuilder Autodesk Inc ซื้อกิจการมาในปี 2549 และปัจจุบันเป็นหนึ่งในรูปแบบไฟล์แลกเปลี่ยนข้อมูล 3 มิติหลักที่ใช้โดยเครื่องมือ 3 มิติจำนวนมาก [STL](/th/cad/stl/) ซึ่งเป็นตัวย่อสำหรับสเตอริโอลิโธรกราฟี เป็นรูปแบบไฟล์ที่สามารถเปลี่ยนแทนกันได้สำหรับการแสดงเรขาคณิตพื้นผิว 3 มิติ

เป็นข้อกำหนดทั่วไปในการแปลง [FBX](/th/3d/fbx/) เป็นรูปแบบไฟล์ STL และมีแอปหลายตัวที่พร้อมใช้งานออนไลน์ซึ่งสามารถแปลง FBX เป็น STL ได้ฟรี หากคุณเป็นนักพัฒนาแอปพลิเคชันที่สนใจฝังฟังก์ชัน FBX เข้ากับ STL ในแอปพลิเคชัน .NET, Java หรือ Python ของคุณ คุณสามารถลองใช้ Aspose.3D API เพื่อจุดประสงค์นี้ได้

## การแปลง FBX เป็น STL - สาธิตฟรี

หากคุณกำลังมองหาโซลูชันออนไลน์ที่ไม่มีค่าใช้จ่ายในการแปลงไฟล์ FBX เป็นรูปแบบ STL คุณสามารถทำได้โดยใช้แอปพลิเคชันการแปลง Aspose.3D เพียงคลิกปุ่มด้านล่างเพื่อเริ่มต้น

[![FBX เป็น STL](../fbx-to-stl.png)](https://products.aspose.app/3d/conversion/fbx-to-stl)

### ขั้นตอนในการแปลง FBX เป็น STL

ต่อไปนี้เป็นขั้นตอนที่คุณต้องปฏิบัติตามเพื่อแปลงไฟล์ FBX ของคุณเป็น STL

1. ไปที่ [แอปแปลง Aspose.3D](https://products.aspose.app/3d/conversion/fbx-to-stl)
1. อัปโหลดไฟล์ FBX ของคุณไปยังเบราว์เซอร์
1. เลือก STL เป็นรูปแบบไฟล์เอาต์พุต
1. กดปุ่มแปลง

เมื่อกระบวนการแปลงเสร็จสิ้น คุณจะสามารถเลือกบันทึกไฟล์ STL ที่แปลงแล้วลงแผ่นดิสก์ได้

## Aspose.3D API

[![FBX เป็น STL](../try-aspose-3d.png)](https://products.aspose.com/3d/)

Aspose.3D API นำเสนอคุณสมบัติที่หลากหลายที่นักพัฒนาสามารถใช้เพื่อสร้างแอพสำหรับการทำงานกับไฟล์ FBX และ STL รองรับภาษาการเขียนโปรแกรมหลายภาษาเช่น:

* .สุทธิ
* ชวา
* หลาม

การสนับสนุนที่หลากหลายสำหรับภาษาการเขียนโปรแกรมที่แตกต่างกันนี้เปิดโอกาสให้คุณแปลง FBX เป็น STL ในภาษาเหล่านี้ได้
