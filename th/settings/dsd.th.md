## ไฟล์ DSD คืออะไร??
ไฟล์ AutoCAD Drawing Set Description (DSD) เป็นรูปแบบไฟล์ข้อความที่ใช้โดยซอฟต์แวร์ AutoCAD ของ Autodesk ซึ่งรวมถึงข้อมูลเกี่ยวกับชุดภาพวาด ข้อมูลนี้ประกอบด้วยชื่อและเส้นทางของภาพวาดแต่ละภาพ ตลอดจนการตั้งค่าและข้อมูลเมตาต่างๆ

ในการสร้างไฟล์ DSD ผู้ใช้สามารถใช้ยูทิลิตี้ AutoCAD Batch Plot ซึ่งอนุญาตให้พวกเขาเลือกภาพวาดที่ต้องการรวมไว้ในชุดและระบุตัวเลือกและการตั้งค่าต่างๆ เมื่อการลงจุดแบทช์เสร็จสมบูรณ์ ไฟล์ DSD จะถูกสร้างขึ้นและบันทึกควบคู่ไปกับภาพวาดโดยอัตโนมัติ

ด้วยไฟล์ DSD ผู้ใช้สามารถจัดการและจัดเรียงไฟล์ภาพวาดหลายไฟล์เป็นชุดได้อย่างสะดวก ลดความซับซ้อนของกระบวนการพิมพ์ เผยแพร่ หรือลงจุดร่วมกัน นอกจากนี้ยังช่วยให้ผู้ใช้สามารถจัดเก็บและเรียกดูการตั้งค่าและการกำหนดค่าต่างๆ สำหรับชุดได้ นอกจากนี้ยังช่วยให้ผู้ใช้สามารถสร้างพล็อตชุดของภาพวาดทั้งหมดภายในชุดได้

## เปิดไฟล์ .DSD ได้อย่างไร
หากต้องการเปิดไฟล์ DSD ผู้ใช้สามารถใช้ยูทิลิตี้ AutoCAD Batch Plot หรือใช้คำสั่ง "Open" ใน AutoCAD และเลือกไฟล์ DSD เมื่อเปิดไฟล์ DSD แล้ว ผู้ใช้จะดูและแก้ไขข้อมูลที่มีอยู่ได้ เช่น ชื่อรูปวาด เส้นทาง และการตั้งค่า

## จะแก้ไขไฟล์ DSD ใน AutoCAD ได้อย่างไร?
การแก้ไขไฟล์ DSD ใน AutoCAD เป็นกระบวนการที่ค่อนข้างง่าย ในการเริ่มต้น ให้เปิด AutoCAD แล้วไปที่แท็บ "เผยแพร่" ใน Ribbon จากนั้นคลิกที่ "Sheet Set Manager" เพื่อเปิดพาเล็ต Sheet Set Manager

ค้นหาไฟล์ DSD ที่คุณต้องการแก้ไขในแผง Sheet Set Manager จากนั้นคลิกขวาที่ไฟล์ เมนูบริบทที่แสดงให้คุณเลือก "แก้ไขคุณสมบัติชุดแผ่นงาน"

การดำเนินการนี้จะเปิดกล่องโต้ตอบ "คุณสมบัติชุดแผ่นงาน" ซึ่งมีแท็บจำนวนหนึ่งพร้อมตัวเลือกการแก้ไขไฟล์ DSD ต่างๆ คุณสามารถเพิ่มหรือลบภาพวาดบางส่วนออกจากชุดได้โดยใช้แท็บ "รายการแผ่นงาน" และคุณสามารถปรับแต่งตัวเลือกต่างๆ สำหรับการพิมพ์ การเผยแพร่ หรือการส่งออกภาพวาดในชุดโดยใช้แท็บ "ตัวเลือกการเผยแพร่"

ทำการเปลี่ยนแปลงไฟล์ DSD ที่จำเป็นโดยใช้แท็บต่างๆ ในกล่องโต้ตอบ "คุณสมบัติชุดแผ่นงาน" แล้วคลิก "ตกลง" เพื่อบันทึกการเปลี่ยนแปลงของคุณ ไฟล์ DSD ของคุณควรสะท้อนถึงการตั้งค่าและการกำหนดค่าที่อัปเดตที่คุณระบุไว้
