## ไฟล์ BPS คืออะไร??

ไฟล์ .BPS เป็นการสำรองเอกสารประมวลผลคำที่สร้างด้วย Microsoft Works ซึ่งโดยทั่วไปจะใช้นามสกุลไฟล์ .WPS สำหรับเอกสาร ไฟล์ .BPS เหล่านี้มักถูกสร้างขึ้นโดยอัตโนมัติโดย Microsoft Works เพื่อเป็นการป้องกันไว้ก่อนในกรณีที่โปรแกรมล่มในขณะที่คุณทำงานกับเอกสาร เป็นวิธีการปกป้องงานของคุณและป้องกันข้อมูลสูญหาย

ในส่วนต่อไปนี้ เราจะสำรวจแอปพลิเคชันซอฟต์แวร์ที่เกี่ยวข้องกับไฟล์ BPS

## ไมโครซอฟต์เวิร์ค

Microsoft Works คือชุดซอฟต์แวร์เพิ่มประสิทธิภาพการทำงานที่พัฒนาโดย Microsoft เปิดตัวครั้งแรกในปี 1987 และได้รับความนิยมในฐานะทางเลือกที่ประหยัดกว่าสำหรับชุดโปรแกรม Microsoft Office เต็มรูปแบบ Microsoft Works มีแอปพลิเคชันสำนักงานขั้นพื้นฐานหลายอย่าง เช่น:

- **โปรแกรมประมวลผลคำ:** Microsoft Works มีแอปพลิเคชันประมวลผลคำที่อนุญาตให้ผู้ใช้สร้างและแก้ไขเอกสาร ใช้รูปแบบไฟล์ [.WPS](/th/word-processing/wps/) สำหรับเอกสารประมวลผลคำ

- **สเปรดชีต:** รวมแอปพลิเคชันสเปรดชีตสำหรับการสร้างและจัดการสเปรดชีต รูปแบบไฟล์สำหรับสเปรดชีตคือ [.XLR](/th/spreadsheet/xlr/)

- **ฐานข้อมูล:** Microsoft Works ยังมีแอปพลิเคชันฐานข้อมูลสำหรับจัดระเบียบและจัดการข้อมูลอีกด้วย รูปแบบไฟล์สำหรับไฟล์ฐานข้อมูลคือ [.WDB](/th/database/wdb/)

- **ปฏิทินและสมุดที่อยู่:** นอกเหนือจากแอปพลิเคชันสำนักงานหลักแล้ว Microsoft Works ยังมีเครื่องมือสำหรับจัดการปฏิทินและสมุดที่อยู่อีกด้วย

- **รูปแบบไฟล์ประมวลผลคำที่ใช้งานได้ (.WPS):** รูปแบบไฟล์ดั้งเดิมของส่วนประกอบการประมวลผลคำของ Microsoft Works ไฟล์ .WPS มีข้อความ การจัดรูปแบบ และกราฟิกที่เรียบง่าย

- **การยุติการใช้งาน:** Microsoft ยกเลิก Microsoft Works อย่างเป็นทางการในปี 2550 และสนับสนุนให้ผู้ใช้เปลี่ยนไปใช้ Microsoft Office หรือทางเลือกอื่น

## เปิดไฟล์ BPS ได้อย่างไร

โปรแกรมที่เปิดไฟล์ BPS ได้แก่

- Microsoft Works - ยกเลิกแล้ว (ชำระเงินแล้ว)
- ไมโครซอฟต์เวิร์ด 365 (ทดลองใช้ฟรี)
