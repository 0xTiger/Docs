## ไฟล์ OGG คืออะไร??

ไฟล์ OGG เป็นไฟล์กราฟที่สร้างด้วยซอฟต์แวร์ Origin ให้ความสามารถในการจัดเก็บกราฟประเภทต่างๆ เพื่อแสดงข้อมูลที่นำเข้าสำหรับการวิเคราะห์ข้อมูล ประเภทกราฟที่รองรับโดยประเภทไฟล์ OGG ได้แก่ คอลัมน์ แท่ง พาย น้ำตก 3 มิติ โพลาร์ และสัญลักษณ์ ซอฟต์แวร์ Origin ได้รับการพัฒนาโดย OriginLab และให้ผู้ใช้ทำการวิเคราะห์ข้อมูลและสร้างผลลัพธ์เป็นกราฟ

## รูปแบบไฟล์ OGG

ไฟล์ OGG มีกราฟสำหรับแสดงข้อมูลที่นำเข้า เมื่อเปิดไฟล์ OGG กราฟจะแสดงกราฟในหน้าต่างแยกต่างหากใน Origin หรือ Origin Viewer ซึ่งมีให้ใช้งานฟรีเมื่อเทียบกับซอฟต์แวร์ Origin ข้อมูลเมตาดาต้า เช่น ชื่อเรื่องและสีของกราฟจะถูกจัดเก็บไว้ในไฟล์ OGG

## เกี่ยวกับซอฟต์แวร์ OriginLab Origin

ซอฟต์แวร์ Origin ยังมีตัวเลือกที่จะใช้เป็นเซิร์ฟเวอร์การทำงานอัตโนมัติ ในกรณีดังกล่าว แอปพลิเคชันไคลเอ็นต์ เช่น LabVIEW, Microsoft Excel, Python หรือแอปพลิเคชันที่พัฒนาขึ้นเองใน C#/VB/VC สามารถแลกเปลี่ยนข้อมูลกับ Origin ได้ [Python_to_OriginLab](https://github.com/jsbangsund/python_to_originlab) คือ Python API เพื่อทำให้การส่งข้อมูลและฟังก์ชันการลงจุดพื้นฐานเป็นแบบอัตโนมัติใน OriginLab
