## ไฟล์ BAK คืออะไร??

ไฟล์ .bak ในบริบทของเว็บเบราว์เซอร์ที่ใช้ Chromium เช่น Google Chrome และ Microsoft Edge ถือเป็นไฟล์สำรองของบุ๊กมาร์กของคุณ บุ๊กมาร์กเหล่านี้จะถูกบันทึกเป็นรายการข้อความธรรมดา และรูปแบบไฟล์ที่ใช้คือ JSON วัตถุประสงค์ของไฟล์ .bak เหล่านี้คือเพื่อปกป้องบุ๊กมาร์กของคุณโดยจัดให้มีการสำรองข้อมูลที่สามารถกู้คืนได้ในกรณีที่เกิดการลบหรือสูญหายโดยไม่ตั้งใจ

เบราว์เซอร์ที่ใช้ Chromium รวมถึง Google Chrome จะสร้างไฟล์สำรองเหล่านี้เป็นประจำและมักจะตั้งชื่อไฟล์ว่า "Bookmarks.bak" โดยพื้นฐานแล้วจะเป็นภาพรวมของบุ๊กมาร์กของคุณ ณ เวลาใดเวลาหนึ่ง

## วิธีใช้ไฟล์ .bak เพื่อกู้คืนบุ๊กมาร์กของคุณ

1. **ค้นหาไฟล์สำรอง:** โดยทั่วไปจะอยู่ในโฟลเดอร์เฉพาะที่เกี่ยวข้องกับโปรไฟล์ Chromium ของคุณ ตำแหน่งที่แน่นอนอาจแตกต่างกันไปขึ้นอยู่กับระบบปฏิบัติการของคุณ

บน Windows: ดูใน `C:\Users\<YourUsername> \AppData\Local\Chromium\ข้อมูลผู้ใช้\ค่าเริ่มต้น\Bookmarks.bak`

บน macOS: ค้นหาใน `~/Library/Application Support/Chromium/Default/Bookmarks.bak`

บน Linux: ทำเครื่องหมายที่ `~/.config/chromium/Default/Bookmarks.bak`

3. ตรวจสอบให้แน่ใจว่าเบราว์เซอร์ Chromium ของคุณปิดอยู่
4. เปลี่ยนชื่อไฟล์ "Bookmarks.bak" โดยลบนามสกุล ".bak" ออก จึงเรียกง่ายๆ ว่า "Bookmarks"
5. เริ่มโครเมียม

เมื่อเปลี่ยนชื่อไฟล์ .bak เป็น "บุ๊กมาร์ก" Chromium จะใช้ไฟล์ดังกล่าวเป็นไฟล์บุ๊กมาร์กปัจจุบัน และบุ๊กมาร์กก่อนหน้าของคุณควรได้รับการกู้คืน

## การสำรองข้อมูลบุ๊กมาร์กโครเมียม

การสำรองข้อมูลบุ๊กมาร์ก Chromium ของคุณเป็นมาตรการป้องกันที่ชาญฉลาดเพื่อให้แน่ใจว่าคุณจะไม่สูญเสียหน้าเว็บและ URL ที่สำคัญที่บันทึกไว้ Chromium เป็นเบราว์เซอร์โอเพ่นซอร์สที่ทำหน้าที่เป็นฐานสำหรับเบราว์เซอร์อื่นๆ เช่น Google Chrome และ Microsoft Edge หากต้องการสำรองข้อมูลบุ๊กมาร์ก Chromium ให้ทำตามขั้นตอนเหล่านี้:

## วิธีที่ 1: การสำรองข้อมูลด้วยตนเอง

1. **เปิด Chromium**: เปิดเบราว์เซอร์ Chromium บนคอมพิวเตอร์ของคุณ

2. **เข้าถึงบุ๊กมาร์ก**: คลิกที่จุดแนวตั้งสามจุด (ไอคอนเมนู) ที่มุมขวาบนของหน้าต่างเบราว์เซอร์ จากเมนูแบบเลื่อนลง วางเมาส์เหนือ "บุ๊กมาร์ก" เพื่อแสดงเมนูย่อยบุ๊กมาร์ก

3. **ส่งออกบุ๊กมาร์ก**: ในเมนูย่อยบุ๊กมาร์ก ให้คลิก "ตัวจัดการบุ๊กมาร์ก" นี่จะเป็นการเปิดแท็บใหม่ที่แสดงบุ๊กมาร์กของคุณ

4. **ส่งออกบุ๊กมาร์ก**: ในแท็บการจัดการบุ๊กมาร์ก ให้คลิกที่จุดแนวตั้งสามจุดอีกครั้ง (โดยปกติจะอยู่ที่มุมขวาบน) เพื่อเปิดเมนูการจัดการบุ๊กมาร์ก จากเมนูนี้ เลือก "ส่งออกบุ๊กมาร์ก"

5. **เลือกตำแหน่ง**: เลือกตำแหน่งที่คุณต้องการบันทึกไฟล์บุ๊กมาร์กที่ส่งออกบนคอมพิวเตอร์ของคุณ ชื่อไฟล์เริ่มต้นคือ "bookmarks.html" แต่คุณสามารถเปลี่ยนได้หากต้องการ บันทึกลงในตำแหน่งที่คุณจะจดจำ

6. **บันทึก**: คลิกปุ่ม "บันทึก" หรือ "ส่งออก" เพื่อบันทึกบุ๊กมาร์กของคุณเป็นไฟล์ HTML

บุ๊กมาร์กของคุณได้รับการสำรองข้อมูลเป็นไฟล์ HTML แล้ว คุณสามารถคัดลอกไฟล์นี้ไปยังไดรฟ์ภายนอกหรือที่เก็บข้อมูลบนคลาวด์เพื่อความปลอดภัย

## วิธีที่ 2: ซิงค์กับบัญชี Google (ถ้ามี)

หากคุณลงชื่อเข้าใช้บัญชี Google ใน Chromium คุณสามารถใช้คุณสมบัติการซิงค์ในตัวเพื่อรักษาบุ๊กมาร์กของคุณให้ปลอดภัยและซิงโครไนซ์กับอุปกรณ์ต่างๆ ด้วยวิธีนี้ บุ๊กมาร์กของคุณจะถูกสำรองข้อมูลไปยังบัญชี Google ของคุณโดยอัตโนมัติ

1. **เปิด Chromium**: เปิดเบราว์เซอร์ Chromium และให้แน่ใจว่าคุณได้ลงชื่อเข้าใช้บัญชี Google ของคุณ

2. **เปิดใช้งานการซิงค์**: คลิกที่จุดแนวตั้งสามจุด (ไอคอนเมนู) ที่มุมขวาบนของหน้าต่างเบราว์เซอร์ และไปที่ "การตั้งค่า"

3. **การซิงค์และบริการของ Google**: ในแท็บการตั้งค่า คลิกที่ "การซิงค์และบริการของ Google" ที่แถบด้านข้างซ้าย

4. **ซิงค์ข้อมูลของคุณ**: ใต้ "ซิงค์" ตรวจสอบให้แน่ใจว่าเปิด "บุ๊กมาร์ก" แล้ว นี่จะซิงค์บุ๊กมาร์กของคุณกับบัญชี Google ของคุณ

บุ๊กมาร์กของคุณจะได้รับการสำรองข้อมูลและซิงโครไนซ์กับบัญชี Google ของคุณโดยอัตโนมัติ คุณสามารถเข้าถึงได้โดยการลงชื่อเข้าใช้บัญชี Google ของคุณบนอุปกรณ์ใดๆ ที่มี Chromium หรือเบราว์เซอร์อื่นที่รองรับการซิงค์ของ Google

อย่าลืมสำรองข้อมูลบุ๊กมาร์กของคุณด้วยตนเองเป็นระยะๆ เช่นกัน โดยเฉพาะอย่างยิ่งหากคุณต้องการสำเนาในเครื่องที่ไม่ต้องใช้การซิงค์บัญชี Google ของคุณเพียงอย่างเดียว

## เปิดไฟล์ .BAK ได้อย่างไร

หากคุณต้องการสำรวจข้อมูลดิบของการสำรองข้อมูลบุ๊กมาร์ก คุณสามารถเปิดไฟล์ "Bookmarks.bak" ด้วยโปรแกรมแก้ไขข้อความต่างๆ เช่น Microsoft Visual Studio Code (พร้อมใช้งานบนหลายแพลตฟอร์ม), Microsoft Notepad (สำหรับผู้ใช้ Windows), Apple TextEdit (สำหรับผู้ใช้ Mac) หรือโปรแกรมแก้ไขข้อความอื่น ๆ ที่คุณเลือก การทำเช่นนี้จะทำให้คุณสามารถดูบุ๊กมาร์กและข้อมูลเมตาที่มีอยู่ในไฟล์ได้

คุณสามารถเปิดไฟล์ "Bookmarks.bak" และดูเนื้อหาได้โดยใช้ซอฟต์แวร์แก้ไขข้อความและโปรแกรมแก้ไขโค้ดต่างๆ นี่คือตัวเลือกบางส่วนที่ใช้กันทั่วไป:

1. รหัส Visual Studio (รหัส VS)
2. โน้ตแพด (วินโดวส์)
3. แก้ไขข้อความ (Mac)
4. ข้อความประเสริฐ
5. สมุดจด++
6. อะตอม
7. นาโนและเป็นกลุ่ม (Linux)
8. เวิร์ดแพด (วินโดวส์)
