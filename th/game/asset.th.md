## ไฟล์ ASSET คืออะไร??

ไฟล์ ASSET เป็นไฟล์พิเศษที่ใช้ใน Unity ซึ่งเป็นโปรแกรมสำหรับสร้างวิดีโอเกม มีการกำหนดค่าที่สำคัญ เช่น วัตถุในเกมหรือการตั้งค่า ตัวอย่างเช่น ไฟล์ ASSET หนึ่งไฟล์อาจจัดเก็บการเคลื่อนที่ของวัตถุในเกม เช่น การกระเด้งกันหรือล้มลง ไฟล์เหล่านี้เปรียบเสมือนคอนเทนเนอร์ที่เก็บข้อมูลสำคัญทั้งหมดไว้เพื่อให้นักพัฒนาเกมสามารถทำงานในโครงการของตนได้อย่างง่ายดาย

## วิธีการเปิดไฟล์ ASSET
  
ใน Unity โดยทั่วไปคุณจะไม่ เปิด ไฟล์เนื้อหาในลักษณะเดียวกับที่คุณเปิดเอกสารหรือไฟล์ในโปรแกรมแก้ไขข้อความหรือโปรแกรมที่คล้ายกัน Unity จะจัดการไฟล์เนื้อหาภายใน Unity Editor แทน ต่อไปนี้คือวิธีที่คุณสามารถเข้าถึงและทำงานกับไฟล์เนื้อหาใน Unity:

1.  **Import Assets**: To use an asset file in your Unity project, you need to import it into project. You can do this by either dragging and dropping asset file directly into Unity Editor or by using "Assets" menu and selecting "Import New Asset."
    
2.  **การดูเนื้อหา**: เมื่อนำเข้าแล้ว คุณจะสามารถดูและจัดการไฟล์เนื้อหาได้ในหน้าต่าง โครงการ ของ Unity Editor หน้าต่างนี้จะแสดงสินทรัพย์ทั้งหมดในโครงการของคุณและช่วยให้คุณสามารถจัดระเบียบเป็นโฟลเดอร์ ค้นหาสินทรัพย์เฉพาะ และดูตัวอย่างเนื้อหาได้
    
3.  **การใช้แอสเซท**: คุณสามารถใช้ไฟล์แอสเซทในโปรเจ็กต์ Unity ของคุณได้โดยลากจากหน้าต่าง โปรเจ็กต์ ไปยังลำดับชั้นฉากของคุณหรือไปยัง GameObjects ในฉาก ตัวอย่างเช่น คุณสามารถลากเนื้อหาพื้นผิวลงบนวัสดุเพื่อนำไปใช้กับวัตถุ 3 มิติ หรือคุณสามารถลากเนื้อหารูปแบบสำเร็จรูปเข้าไปในฉากเพื่อสร้างอินสแตนซ์ GameObject ที่กำหนดค่าไว้ล่วงหน้า
    
4.  **การแก้ไขเนื้อหา**: ไฟล์เนื้อหาบางไฟล์ เช่น สคริปต์ เนื้อหา และภาพเคลื่อนไหว สามารถแก้ไขได้โดยตรงภายใน Unity Editor คุณสามารถดับเบิลคลิกเนื้อหาเหล่านี้ในหน้าต่าง โครงการ เพื่อเปิดขึ้นมาในตัวแก้ไขที่เกี่ยวข้อง และทำการเปลี่ยนแปลงได้ตามต้องการ
    
5.  **การบันทึกเนื้อหา**: การเปลี่ยนแปลงใดๆ ที่คุณทำกับไฟล์เนื้อหาภายใน Unity Editor จะถูกบันทึกโดยอัตโนมัติภายในโปรเจ็กต์ Unity ของคุณ ไม่จำเป็นต้องบันทึกไฟล์เนื้อหาแต่ละไฟล์อย่างชัดเจนเหมือนกับที่คุณทำกับเอกสารในโปรแกรมซอฟต์แวร์แบบดั้งเดิม
