## ไฟล์ DXB คืออะไร??

DXB ย่อมาจาก Drawing Exchange Binary ซึ่งเป็นรูปแบบไฟล์ที่ใช้ในซอฟต์แวร์การออกแบบโดยใช้คอมพิวเตอร์ช่วย (CAD) ไฟล์ DXB มีข้อมูลไบนารี่ที่แสดงกราฟิกเวกเตอร์ 2D และใช้เพื่อแลกเปลี่ยนภาพวาดระหว่างโปรแกรม CAD ต่างๆ โดยทั่วไปจะสร้างขึ้นโดยการส่งออกภาพวาดจากโปรแกรม CAD ในรูปแบบ DXB จากนั้นนำเข้าไปยังโปรแกรม CAD อื่น

ปัจจุบันรูปแบบ DXB มีการใช้น้อยกว่ารูปแบบไฟล์ CAD อื่นๆ เช่น DXF หรือ DWG อย่างไรก็ตาม โปรแกรม CAD รุ่นเก่าบางโปรแกรมอาจยังคงใช้ DXB เป็นรูปแบบไฟล์สำหรับการส่งออกหรือนำเข้าภาพวาด

หากคุณมีไฟล์ DXB และต้องการดูหรือแก้ไข คุณจะต้องมีโปรแกรม CAD ที่รองรับรูปแบบ DXB ตัวอย่างของโปรแกรม CAD ที่รองรับ DXB ได้แก่ AutoCAD, DraftSight และ LibreCAD

## รูปแบบไฟล์ DXB - ข้อมูลเพิ่มเติม

DXB ( Drawing Exchange Binary ) เป็นรูปแบบไฟล์ DXF ( Drawing Exchange Format ) เวอร์ชันไบนารี่ ซึ่งเป็นรูปแบบมาตรฐานสำหรับการแลกเปลี่ยนภาพวาด CAD ระหว่างแอปพลิเคชันซอฟต์แวร์ต่างๆ

ไฟล์ DXB มีข้อมูลไบนารี่ที่แสดงกราฟิกเวกเตอร์ 2D เช่นเดียวกับไฟล์ DXF อย่างไรก็ตาม ไฟล์ DXB มีขนาดกะทัดรัดและโหลดได้เร็วกว่าไฟล์ DXF เนื่องจากไฟล์เหล่านั้นถูกจัดเก็บในรูปแบบไบนารีแทนที่จะเป็นข้อความธรรมดา

ในการสร้างไฟล์ DXB โปรแกรม CAD สามารถส่งออกภาพวาดในรูปแบบ DXB ซึ่งสร้างไฟล์ไบนารีที่โปรแกรม CAD อื่น ๆ ที่รองรับ DXB สามารถอ่านได้ ในทำนองเดียวกัน หากต้องการนำเข้าไฟล์ DXB โปรแกรม CAD สามารถอ่านข้อมูลไบนารี่และแปลงเป็นรูปแบบภายในของตัวเองเพื่อแก้ไขได้

## ไฟล์ DXB เป็นรูปแบบใด?

รูปแบบไฟล์ DXB ( Drawing Exchange Binary) เป็นรูปแบบไฟล์ไบนารี่ ซึ่งหมายความว่าประกอบด้วยข้อมูลไบนารี่ที่สามารถตีความได้โดยตรงด้วยคอมพิวเตอร์

รูปแบบไบนารี่ที่ DXB ใช้นั้นมีขนาดกะทัดรัดและโหลดได้เร็วกว่ารูปแบบไฟล์ DXF (รูปแบบการแลกเปลี่ยนรูปวาด) แบบข้อความ ซึ่งใช้สำหรับการแลกเปลี่ยนภาพวาด CAD ระหว่างแอปพลิเคชันซอฟต์แวร์ต่างๆ

โครงสร้างของไฟล์ DXB ประกอบด้วยชุดของบันทึกไบนารีซึ่งแต่ละชุดมีส่วนหัวและข้อมูลที่แสดงวัตถุหรือเอนทิตีเดี่ยวในรูปวาด ส่วนหัวประกอบด้วยข้อมูลเกี่ยวกับประเภทของออบเจ็กต์หรือเอนทิตีที่แสดงโดยข้อมูล รวมถึงขนาดและคุณสมบัติอื่นๆ

ข้อมูลในไฟล์ DXB ได้รับการจัดระเบียบตามลำดับชั้น โดยมีบันทึกระดับสูงกว่าที่มีการอ้างอิงไปยังบันทึกระดับล่างที่แสดงถึงวัตถุย่อยหรือส่วนประกอบของการวาด ตัวอย่างเช่น บันทึกที่แสดงถึงเส้นหลายเส้นอาจมีการอ้างอิงถึงบันทึกที่แสดงถึงส่วนของเส้นแต่ละเส้นที่ประกอบขึ้นเป็นเส้นหลายเส้น

## ไฟล์ DXB ประกอบด้วยอะไรบ้าง

ไฟล์ DXB ( Drawing Exchange Binary ) มีข้อมูลไบนารี่ที่แสดงกราฟิกเวกเตอร์ 2 มิติ เช่น เส้น ส่วนโค้ง วงกลม ข้อความ และรูปทรงเรขาคณิตอื่น ๆ ที่ประกอบเป็นภาพวาด CAD

เมื่อโปรแกรม CAD ส่งออกภาพวาดเป็นรูปแบบ DXB โปรแกรมจะสร้างไฟล์ไบนารีที่มีข้อมูลทั้งหมดที่จำเป็นในการสร้างภาพวาดขึ้นใหม่ในโปรแกรม CAD อื่นที่รองรับ DXB ซึ่งรวมถึงข้อมูลเกี่ยวกับตำแหน่ง ขนาด สี และลักษณะของแต่ละวัตถุในภาพวาด ตลอดจนข้อความหรือคำอธิบายประกอบที่อาจรวมอยู่ด้วย

เนื่องจากไฟล์ DXB ถูกจัดเก็บในรูปแบบไบนารี โดยทั่วไปแล้วจะมีขนาดเล็กกว่าและโหลดได้เร็วกว่าไฟล์ DXF (รูปแบบการแลกเปลี่ยนรูปวาด) ซึ่งจัดเก็บไว้ในข้อความธรรมดา อย่างไรก็ตาม ไฟล์ DXB สามารถอ่านได้โดยโปรแกรม CAD ที่รองรับรูปแบบ DXB เท่านั้น ในขณะที่ไฟล์ DXF สามารถอ่านได้โดยโปรแกรมที่หลากหลายกว่า
