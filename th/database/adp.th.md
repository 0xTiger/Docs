## ไฟล์ ADP คืออะไร??

ไฟล์ที่มีนามสกุล .adp คือไฟล์โครงการ Microsoft Access ที่ใช้สถาปัตยกรรมคอมโพเนนต์ OLE DB เพื่อให้การเชื่อมต่อโดยตรงและมีประสิทธิภาพกับฐานข้อมูล Microsoft SQL Server ไฟล์ดังกล่าวไม่มีตารางจริง ไดอะแกรมฐานข้อมูล หรือองค์ประกอบฐานข้อมูลอื่นๆ ไฟล์ ADP สามารถสร้างได้ด้วย Access 2007 และ 2010 ไฟล์ที่สร้างด้วย Access เวอร์ชันก่อนหน้าสามารถเปิด Access 2007 และ 2010 ได้เช่นกัน ไฟล์ ADP สามารถเปิดได้โดยใช้ Microsoft Access 365

## รูปแบบไฟล์ ADP - ข้อมูลเพิ่มเติม

โครงการ Access ที่สร้างขึ้นเป็น ADP ช่วยให้คุณสามารถเปลี่ยนแปลงการออกแบบวัตถุเซิร์ฟเวอร์ SQL เช่น ตารางและมุมมอง นอกจากนี้ยังอนุญาตให้สร้าง แก้ไข และใช้คุณสมบัติของ SQL เช่น ไดอะแกรมฐานข้อมูล กระบวนงานที่เก็บไว้ และฟังก์ชันที่ผู้ใช้กำหนด สิ่งนี้ให้ประโยชน์เหนือการเชื่อมโยงไปยังฐานข้อมูลเซิร์ฟเวอร์ SQL ซึ่งคุณไม่สามารถเปลี่ยนแปลงการออกแบบได้ และสามารถเชื่อมโยงไปยังตารางและมุมมองของเซิร์ฟเวอร์ SQL ได้เท่านั้น

### ประเภทข้อมูลและการสนับสนุนไดอะแกรม

**ประเภทข้อมูลวันที่/เวลา**

ชนิดข้อมูลวันที่/เวลาใหม่ต่อไปนี้ได้รับการสนับสนุนใน Access 2010

* เวลา
* วันที่
* DATETIME2
* DATETIMEOFFSET

**ประเภทข้อมูลที่มีความยาวผันแปรได้**

ชนิดข้อมูลที่มีความยาวผันแปรได้ต่อไปนี้สามารถใช้ได้ในโครงการ Access 2010:

* วาร์บิน (สูงสุด)
* VARCHAR (สูงสุด)
* NVARCHAR (สูงสุด)
