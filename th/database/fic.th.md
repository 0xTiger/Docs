## ไฟล์ FIC คืออะไร??

ไฟล์ FIC ใน WinDev เป็นเวอร์ชันที่คอมไพล์ของฐานข้อมูล HyperFileSQL FIC ตัวย่อย่อมาจาก Fichier de Code ในภาษาฝรั่งเศส ซึ่งแปลเป็น Code File ในภาษาอังกฤษ ใน WINDEV ไฟล์ FIC เป็นไฟล์ที่คอมไพล์ซึ่งมีรหัสปฏิบัติการของโครงการ ไฟล์นี้มีนามสกุล .fic และถูกสร้างขึ้นเมื่อคุณคอมไพล์โครงการ WINDEV เมื่อรวบรวมโปรเจ็กต์เป็นไฟล์ FIC แล้ว โปรเจ็กต์นั้นสามารถแจกจ่ายให้กับผู้ใช้เป็นโปรแกรมปฏิบัติการหรือแพ็คเกจการติดตั้งได้

ไฟล์ FIC เกี่ยวข้องกับ WINDEV ซึ่งเป็นเครื่องมือพัฒนาซอฟต์แวร์ที่สร้างโดย PCSoft บริษัท ฝรั่งเศส ช่วยให้นักพัฒนาสามารถสร้างแอปพลิเคชัน Windows เว็บ และมือถือได้โดยใช้โค้ดฐานเดียว ประกอบด้วยสภาพแวดล้อมการพัฒนาด้วยภาพ ภาษาการเขียนโปรแกรม (WLangage) และการควบคุมและฟังก์ชันที่สร้างไว้ล่วงหน้าที่หลากหลายสำหรับงานทั่วไป โดยส่วนใหญ่จะใช้สำหรับแอปพลิเคชันทางธุรกิจและมีฐานผู้ใช้ที่แข็งแกร่งในยุโรป

## ความสัมพันธ์กับฐานข้อมูล HyperFileSQL Mobile

ใน WinDev ไฟล์ FIC ที่สร้างขึ้นสำหรับฐานข้อมูล HyperFileSQL Mobile จะมีรหัสปฏิบัติการสำหรับฐานข้อมูลตลอดจนข้อมูลที่มีอยู่ในฐานข้อมูล HyperFileSQL Mobile คือระบบการจัดการฐานข้อมูลเชิงสัมพันธ์ที่มาพร้อมกับ WINDEV และได้รับการออกแบบมาโดยเฉพาะสำหรับแอปพลิเคชันมือถือและออฟไลน์ ไฟล์ FIC สามารถแจกจ่ายเป็นไฟล์ปฏิบัติการแบบสแตนด์อโลนหรือเป็นส่วนหนึ่งของแพ็คเกจการติดตั้ง และสามารถเรียกใช้บนอุปกรณ์เป้าหมายเพื่อเข้าถึงและจัดการข้อมูลในฐานข้อมูล HyperFileSQL Mobile เมื่อเรียกใช้ไฟล์ FIC ไฟล์จะเปิดฐานข้อมูลและทำให้ข้อมูลพร้อมใช้งานในแอปพลิเคชันมือถือ
