## ไฟล์ CHCK คืออะไร?

ไฟล์ CHCK ถูกสร้างขึ้นจากการตรวจสอบสุขภาพบน Microsoft Exchange Server ประกอบด้วยข้อมูลเกี่ยวกับสถานะโดยรวมและการดำเนินงานของฐานข้อมูลเซิร์ฟเวอร์ Microsoft Exchange Server ใช้เป็นเซิร์ฟเวอร์อีเมลเป็นหลัก และช่วยให้คุณสามารถรักษาข้อมูลติดต่อทางธุรกิจและปฏิทินของคุณได้ ผู้ดูแลระบบเซิร์ฟเวอร์ Exchange ใช้ EseUtil เพื่อตรวจสอบความสมบูรณ์ของฐานข้อมูล

## รูปแบบไฟล์ CHCK

ไม่ทราบรูปแบบไฟล์ภายในของไฟล์ CHCK และไม่มีข้อมูลอ้างอิงสำหรับการอ่าน/เขียนไฟล์ CHCK
