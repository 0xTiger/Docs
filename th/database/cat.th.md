## ไฟล์ CAT คืออะไร??

ไฟล์ CAT เป็นของซอฟต์แวร์ Advanced Disk Catalog ที่พัฒนาโดย ElcomSoft Co. Ltd บริษัท ซอฟต์แวร์รัสเซียที่พัฒนาความปลอดภัยคอมพิวเตอร์และเครื่องมือซอฟต์แวร์ทางนิติเวชต่างๆ ผลิตภัณฑ์บางอย่างของบริษัทมีซอฟต์แวร์สำหรับการกู้คืนรหัสผ่าน การได้มาซึ่งอุปกรณ์มือถือ และการแยกข้อมูลบนคลาวด์ บริษัทก่อตั้งขึ้นในปี 1990 และมีสำนักงานในรัสเซียและสหรัฐอเมริกา พวกเขาเป็นที่รู้จักกันดีในด้านซอฟต์แวร์ทางนิติวิทยาศาสตร์ และโดยรวมแล้วซอฟต์แวร์ของพวกเขาถูกใช้โดยทั้งบริษัทเอกชนและหน่วยงานของรัฐเพื่อทำการวิเคราะห์และการสอบสวนทางนิติวิทยาศาสตร์

Advanced Disk Catalog is a cataloguing program for Windows XP/2003. ความเก่งกาจของ ADC มาจากความจริงที่ว่า ADC ไม่ต้องพึ่งพากลไกฐานข้อมูล ทำให้มีประสิทธิภาพสูงและมีขนาดกะทัดรัด อินเทอร์เฟซผู้ใช้ซึ่งคล้ายกับ Windows Explorer ช่วยให้จัดระเบียบข้อมูลที่จัดเก็บไว้ในสื่อประเภทต่างๆ ได้อย่างง่ายดาย เช่น ฮาร์ดไดรฟ์ (รวมถึงไดรฟ์เครือข่าย) ฟล็อปปี้ดิสก์ ออปติคัลดิสก์ ดิสก์ ZIP และ JAZ เป็นต้น

## เปิดไฟล์ CAT ได้อย่างไร

ดาวน์โหลดซอฟต์แวร์ Elcomsoft Advanced Disk Catalog บน Windows คุณสามารถคลิกขวาที่ file explorer จากนั้นเลือกเปิดด้วยเมนูบริบท ระบุเส้นทางของซอฟต์แวร์ Elcomsoft Advanced Disk Catalog เพื่อเปิดไฟล์ cat
