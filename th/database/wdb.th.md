## ไฟล์ WDB คืออะไร??
ไฟล์ที่มีนามสกุล .wdb เป็นไฟล์ฐานข้อมูลที่สร้างโดย Microsoft Works ซึ่งใช้เพื่อทำหน้าที่ต่างๆ เช่น ระบบจัดการฐานข้อมูล ไฟล์ WDB คล้ายกับไฟล์ Access Database ([MDB](/th/database/mdb/)) แต่มีประสิทธิภาพน้อยกว่าและมีข้อจำกัดมากกว่า ไม่สามารถเปิดไฟล์ WDB โดยใช้ Microsoft Access อย่างไรก็ตาม คุณสามารถเปิดไฟล์ WDB ใน Microsoft Works แล้วส่งออกไปยังไฟล์ MDB ตามลำดับเพื่อเปิดไฟล์ WDB ใน MS Access

## รูปแบบไฟล์ WDB
Microsoft Works Database เป็นรูปแบบฐานข้อมูลดั้งเดิมของชุดโปรแกรม Microsoft Works office เนื่องจากลักษณะที่เป็นกรรมสิทธิ์ของรูปแบบและข้อจำกัดบางประการ ไม่สามารถเปิดไฟล์ WDB ในซอฟต์แวร์อื่นใดนอกจาก Microsoft Works ในรูปแบบไฟล์ที่เกิดซ้ำ สามารถพบส่วนหัว 10 ไบต์ก่อนแต่ละสตริงข้อความ ASCII ที่แสดงค่าของฟิลด์ซึ่งสิ้นสุดด้วยค่า NULL ส่วนหัวโดยทั่วไปเริ่มต้นด้วย **\x0f** ไบต์และ NULL ตามด้วยข้อมูล 4 ไบต์สลับกับ NULL

### โครงสร้างไฟล์

โครงสร้างของไฟล์ WDB แสดงไว้ด้านล่าง:
- **ส่วนหัวที่ 1**: จากจุดเริ่มต้นของไฟล์ ลงท้ายด้วย \x25\x00\xf2 และ 244 NULL ไบต์
- **ส่วนหัวที่ 2**: ขึ้นต้นด้วย \xffT - เช่น \xff\x54 และขยายเป็น 4096 ไบต์ ซึ่งมีชื่อคอลัมน์/ฟิลด์และสิ่งอื่นๆ และดูเหมือนว่าจะเริ่มต้นที่ตำแหน่งไบต์ 6144
- **ฟิลด์**: ค่ามีส่วนหัว 10 ไบต์ โดยมีรูปแบบดังนี้ {type byte} {type byte, part 2} {data byte 1} \x00 {db 2} \x00 {db 3} {db 3 , ตอนที่ 2} {db 4} \x00. ไบต์ข้อมูล 2 คือหมายเลขฟิลด์ ไบต์ข้อมูล 3 คือหมายเลขเร็กคอร์ด (เพิ่มไบต์ข้อมูล 3 ตอนที่ 2 เมื่อเกิน 256 เร็กคอร์ด)
