## ไฟล์ DB3 คืออะไร??
ไฟล์ DB3 เป็นไฟล์ฐานข้อมูลที่สร้างโดยซอฟต์แวร์ SQLite ซึ่งเป็นโปรแกรมฐานข้อมูลที่มีน้ำหนักเบาและมีอยู่ในตัวเองซึ่งสร้างฐานข้อมูลโดยใช้ไฟล์ธรรมดา มีกายวิภาคของฐานข้อมูลเช่นเดียวกับบันทึกข้อมูล ใช้กันทั่วไปสำหรับการดึงหรือจัดเก็บข้อมูลที่มีโครงสร้างโดยใช้ SQL ไฟล์เหล่านี้สามารถใช้ในอุปกรณ์อัจฉริยะหรือเมื่อต้องการเก็บบันทึกหรือจัดการข้อมูลอื่นๆ แต่ในสภาพแวดล้อมที่มีพื้นที่น้อย


## รูปแบบไฟล์ DB3
รูปแบบไฟล์ DB3 เชื่อมโยงกับ RDBMS (ระบบจัดการฐานข้อมูลเชิงสัมพันธ์) SQLite ซึ่งเป็นตัวเลือกยอดนิยมสำหรับฐานข้อมูลแบบฝังตัว ไม่มีนามสกุลไฟล์เฉพาะใด ๆ ที่กำหนดไว้ใน SQLite_3 ในข้อมูลจำเพาะ อาจใช้นามสกุลรวมถึง [.dbf](/th/database/dbf/) และ [.sqlite](/th/database/sqlite/)

### การจำแนกฐานข้อมูล
รูปแบบไฟล์ db ที่เกี่ยวข้องกับ SQLite เวอร์ชัน 3 โดยทั่วไปถือเป็นรูปแบบไฟล์ DB3 ที่ใช้เป็นรูปแบบเนทีฟที่เผยแพร่ต่อสาธารณะสำหรับเอ็นจิ้นฐานข้อมูล SQLite ตั้งแต่เดือนมิถุนายน 2547 รูปแบบไฟล์ DB3 เป็นรูปแบบข้ามแพลตฟอร์ม ถ่ายโอนระหว่าง big-endian ได้ และสถาปัตยกรรมแบบ little-endian หรือระบบ 32 บิตและ 64 บิต คุณลักษณะเหล่านี้ทำให้ DB3 เป็นตัวเลือกยอดนิยมในฐานะรูปแบบไฟล์แอ็พพลิเคชัน ไฟล์ฐานข้อมูล SQLite_3 หลัก (DB3) ประกอบด้วยหนึ่งเพจขึ้นไป ขนาดเท่ากันทุกหน้าในฐานข้อมูลเดียวกัน ขนาดเพจในหน่วยไบต์มีค่ากำลังสองระหว่าง 512 และ 65536
