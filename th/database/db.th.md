## ไฟล์ DB คืออะไร??

ไฟล์ที่มีนามสกุล db เป็นไฟล์ฐานข้อมูลทั่วไปเพื่อเก็บข้อมูล ไม่มีข้อกำหนดอย่างเป็นทางการสำหรับไฟล์ฐานข้อมูลดังกล่าว ข้อมูลถูกจัดระเบียบในรูปแบบที่มีโครงสร้างภายในไฟล์ในรูปแบบตาราง เขตข้อมูล ชนิดข้อมูล และค่าของเขตข้อมูล ไฟล์ DB ยังสามารถสร้างได้โดยการอ่านจากไฟล์แผ่นข้อมูลที่มีข้อมูลเป็นแถวและคอลัมน์ เนื้อหาของไฟล์ DB สามารถส่งออกเพิ่มเติมเป็นรูปแบบไฟล์ [CSV](/th/spreadsheet/csv/) และ [PDF](/th/pdf/) เพื่อการรายงาน ไฟล์ DB สามารถเปิดได้ในแอปพลิเคชันเช่น SQLite และ LibreOffice
