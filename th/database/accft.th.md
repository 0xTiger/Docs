## ไฟล์ ACCFT คืออะไร??

ไฟล์ที่มีนามสกุล .accft คือไฟล์เทมเพลต Microsoft Access ที่มีคำจำกัดความของชนิดข้อมูลฟิลด์สำหรับตารางของฐานข้อมูล Microsoft Access เป็นระบบจัดการฐานข้อมูลเชิงสัมพันธ์บนเดสก์ท็อป (RDBMS) ที่ใช้สร้างฐานข้อมูล ตาราง ฟอร์ม คิวรี และมุมมอง ไฟล์ ACCFT สามารถใช้กำหนดประเภทข้อมูลสำหรับฟิลด์ตาราง และสามารถสร้างตารางฐานข้อมูลได้โดยใช้ไฟล์เทมเพลตนี้เป็นจุดเริ่มต้น สามารถเปิดได้ด้วย Microsoft Access 2016 เพื่อสร้างตารางใหม่ ไฟล์ ACCFT รู้จักตามประเภท MIME **application/msaccess**

## รูปแบบไฟล์ ACCFT

ไฟล์ ACCFT เป็นไฟล์ไบนารีที่ไม่มีข้อกำหนดรูปแบบไฟล์แบบสาธารณะ รายละเอียดภายใน เช่น ข้อมูลส่วนหัว โครงสร้างไฟล์ ส่วนท้าย ฯลฯ ยังไม่พร้อมใช้งานในขณะนี้
