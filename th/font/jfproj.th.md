## ไฟล์ JFPROJ คืออะไร??

ไฟล์ที่มีนามสกุล .jfproj เป็นฟอนต์แบบกำหนดเองที่ใช้ JSON ซึ่งสร้างด้วยแอปพลิเคชัน Microsoft Font Maker บน Windows 10 สิ่งเหล่านี้สามารถสร้างได้โดยใช้ปากกาอิเล็กทรอนิกส์กับอุปกรณ์หน้าจอสัมผัส เช่น แท็บเล็ต แม้ว่าอุปกรณ์ต่อพ่วงคอมพิวเตอร์อื่นๆ เช่น เมาส์ก็สามารถใช้ได้ในกรณี ของคอมพิวเตอร์ ไฟล์ JFPROJ มีสัญลักษณ์ที่ใช้ในการแสดงอักขระบนหน้าจอโดยกำหนดระยะห่างระหว่างอักขระและคำ และขนาดของแบบอักษร แอป Microsoft Font Maker สามารถดาวน์โหลดได้ที่ Windows store
