## ไฟล์ VDI คืออะไร??
ไฟล์ที่มีนามสกุล .vdi เป็นภาพดิสก์เสมือน เฉพาะสำหรับโปรแกรมจำลองเสมือนเดสก์ท็อปโอเพ่นซอร์สของ Oracle ที่เรียกว่า VirtualBox ไฟล์ VDI ใช้เพื่อเริ่มเครื่องเสมือน VirtualBox VM อนุญาตให้ผู้ใช้เรียกใช้ระบบปฏิบัติการหรือโปรแกรมเพิ่มเติมบนคอมพิวเตอร์เครื่องเดียว ดังนั้น ประโยชน์ของไฟล์ VDI ก็คือผู้ใช้สามารถบันทึกไฟล์อิมเมจของดิสก์เหล่านี้บนฮาร์ดดิสก์และสามารถเรียกใช้ไฟล์เหล่านี้โดยใช้โปรแกรมจำลองได้ทุกเมื่อที่ต้องการ

## รูปแบบไฟล์ VDI
Virtual Disk Image (VDI) เป็นรูปแบบไฟล์ดิสก์หลักสำหรับ Oracle VM แบบโอเพ่นซอร์สที่พัฒนาอย่างแข็งขัน ซึ่งรู้จักกันในชื่อ VirtualBox ซึ่งเป็นผลิตภัณฑ์การจำลองเสมือนระดับองค์กร รูปแบบไฟล์ VDI ได้รับการออกแบบมาเพื่อสร้างอิมเมจของดิสก์แบบพกพาซึ่งสามารถเรียกใช้ได้อย่างง่ายดายโดยใช้โปรแกรมเวอร์ชวลไลเซชันอื่นๆ รองรับทั้งพื้นที่จัดเก็บที่จัดสรรแบบไดนามิกและขนาดคงที่ อนุญาตให้คุณขยายไฟล์รูปภาพหลังจากสร้างแล้ว แม้ว่าจะมีข้อมูลอยู่แล้วก็ตาม

### การจำลองเสมือนของดิสก์
ระบบจำลองฮาร์ดดิสก์ในรูปแบบภาพดิสก์ VDI VirtualBox VM สามารถใช้ดิสก์รุ่นก่อนหน้าที่สร้างใน Microsoft Virtual PC หรือ VMware รวมถึงรูปแบบดั้งเดิมของมันเอง VirtualBox ยังสามารถเชื่อมต่อกับเป้าหมาย iSCSI และการแบ่งพาร์ติชันดิบบนโฮสต์ โดยใช้เป็นฮาร์ดดิสก์เสมือน VirtualBox จำลอง IDE (ตัวควบคุม PIIX4 และ ICH6), SATA (ตัวควบคุม ICH8M), ตัวควบคุม SAS ที่สามารถแนบฮาร์ดไดรฟ์และ SCSI ได้

รองรับ Open Virtualization Format (OVF) ตั้งแต่เวอร์ชัน 2.2.0 ของ VirtualBox ทั้งอุปกรณ์ทางกายภาพที่เชื่อมต่อกับโฮสต์และอิมเมจ ISO สามารถติดตั้งเป็นไดรฟ์ซีดีหรือดีวีดีได้
ตามค่าเริ่มต้น VirtualBox รองรับกราฟิกผ่านการ์ดกราฟิกเสมือนแบบกำหนดเองที่เข้ากันได้กับ VESA

### รองรับการจำลองเสมือนสำหรับอีเธอร์เน็ต
VirtualBox จำลองเสมือนการ์ดอินเทอร์เฟซเครือข่ายต่อไปนี้สำหรับอะแดปเตอร์เครือข่ายอีเทอร์เน็ต:
- เอเอ็มดี PCnet PCI II (Am79C970A)
- เอเอ็มดี PCnet-Fast III (Am79C973)
- เดสก์ท็อป Intel Pro/1000 MT (82540EM)
- เซิร์ฟเวอร์ Intel Pro/1000 MT (82545EM)
- เซิร์ฟเวอร์ Intel Pro/1000 T (82543GC)
- อะแดปเตอร์เครือข่าย Paravirtualized (virtio-net)

การ์ดเครือข่ายจำลองช่วยให้ระบบปฏิบัติการเกสต์สามารถทำงานได้โดยไม่จำเป็นต้องค้นหาและติดตั้งไดรเวอร์สำหรับฮาร์ดแวร์เครือข่าย เนื่องจากมีให้โดยเป็นส่วนหนึ่งของระบบปฏิบัติการเกสต์ อะแดปเตอร์เครือข่ายเสมือนแบบพิเศษช่วยปรับปรุงประสิทธิภาพเครือข่ายโดยลดความจำเป็นในการจับคู่อินเทอร์เฟซฮาร์ดแวร์เฉพาะ ดังนั้นจึงต้องการการสนับสนุนคนขับเป็นพิเศษในแขก
