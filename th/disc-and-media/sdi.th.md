## ไฟล์ SDI คืออะไร??
ไฟล์ที่มีนามสกุล .sdi ประกอบด้วย load blob, boot blob และ part blob ใช้กันทั่วไปโดย Windows Embedded Studio เพื่อสร้างดิสก์ RAM หรือดิสก์สำหรับบูตสำหรับการโหลดและติดตั้ง Windows SDI ย่อมาจาก System Deployment Image Windows Embedded Studio เป็นโปรแกรมที่ใช้สำหรับสร้างดิสก์อิมเมจสำหรับ Windows จริงๆ แล้วโปรแกรมนี้ประกอบด้วยโปรแกรม SDI File Manager และเครื่องมือบรรทัดคำสั่งที่เรียกว่า **sdimgr.exe** ซึ่งทั้งสองอย่างนี้สามารถใช้เพื่อ deply สร้าง และแก้ไขอิมเมจ SDI

## รูปแบบไฟล์ SDI
รูปแบบไฟล์ SDI ถูกใช้อย่างกว้างขวางเพื่อเปิดใช้งานการใช้ดิสก์เสมือนสำหรับการบูทระบบ Microsoft Windows บางเวอร์ชันเปิดใช้งาน **การบูต RAM** ซึ่งเป็นสิ่งสำคัญในการโหลดไฟล์ SDI ลงในหน่วยความจำ จากนั้นจึงบูตจากไฟล์นั้น รูปแบบไฟล์ SDI ยังเปิดใช้งานตัวเองสำหรับการบูทเครือข่ายโดยใช้ PXE (Preboot Execution Environment) นอกจากนี้ยังใช้สำหรับการถ่ายภาพบนฮาร์ดดิสก์
### ส่วนไฟล์ SDI
ไฟล์ SDI สามารถแบ่งย่อยออกเป็นส่วนต่อไปนี้:
- **Boot BLOB**: ประกอบด้วยโปรแกรมบู๊ตจริง STARTROM.COM สิ่งนี้คล้ายกับบูตเซกเตอร์ของฮาร์ดดิสก์
- **โหลด BLOB**: โดยทั่วไปจะมี NTLDR และเปิดใช้งานโดย BLOB สำหรับบูต
- **ส่วน BLOB**: ประกอบด้วยรันไทม์สำหรับบูตจริง รวมถึงไฟล์ boot.ini และ ntdetect.com ซึ่งควรอยู่ในไดเรกทอรีรากของรันไทม์
- **Disk BLOB**: นี่คืออิมเมจ HDD แบบแฟลตที่ขึ้นต้นด้วย MBR ใช้สำหรับการถ่ายภาพฮาร์ดไดรฟ์แทนการบูต นอกจากนี้ เฉพาะ Disk BLOB เท่านั้นที่สามารถเมาต์ได้ด้วยยูทิลิตี้ของ Microsoft
