## ไฟล์ VHD คืออะไร??

VHD เป็นตัวย่อของ Virtual Hard Disk เหล่านี้เป็นรูปแบบไฟล์ที่แสดงถึงฮาร์ดดิสก์ไดรฟ์เสมือน ข้อมูลที่พบในฮาร์ดดิสก์ไดรฟ์อาจมีส่วนเกี่ยวข้องกับข้อมูลดังกล่าว เช่นเดียวกับพาร์ติชันของฮาร์ดดิสก์และระบบไฟล์ที่ประกอบด้วยโฟลเดอร์ต่างๆ

สำหรับเครื่องเสมือน ไฟล์ VHD จะถือเป็นฮาร์ดดิสก์ ในเวอร์ชัน Windows ที่ทันสมัย สิ่งเหล่านี้มีอยู่แล้วภายใน ฮาร์ดดิสก์เสมือนประเภทเฉพาะนี้จัดเตรียมสิ่งอำนวยความสะดวกสำหรับระบบปฏิบัติการต่างๆ ให้อยู่ในเครื่องโฮสต์ทั่วไป เป็นโอกาสที่ดีสำหรับนักพัฒนาซอฟต์แวร์ในการทดสอบซอฟต์แวร์บนระบบปฏิบัติการต่างๆ โดยไม่ต้องดำเนินการติดตั้งฮาร์ดดิสก์อื่น มีข้อดีหลายประการในการใช้รูปแบบไฟล์นี้ของไฟล์ VHD เนื่องจากมีคุณสมบัติมากมาย


## ประวัติย่อ ##

**Connectix** พัฒนารูปแบบไฟล์นี้เป็นผลิตภัณฑ์พีซีเสมือนสำหรับการใช้งาน ในปี 2546 Microsoft ได้ซื้อ Connectix และเปิดตัวเวอร์ชันอื่นของรูปแบบนี้ เช่น VHDx พร้อมกับฟีเจอร์และฟังก์ชันการทำงานที่ดีกว่าซึ่งขาดหายไปในเวอร์ชันแรก ในปี 2012 VHDx ถูกเพิ่มเข้าไปใน Windows มันถูกเพิ่มเข้ามาใน Windows เพื่อเพิ่มความจุของสตอเรจ การป้องกันความเสียหายของข้อมูล การปรับให้เหมาะสมเพื่อป้องกันการเสื่อมประสิทธิภาพ คุณสมบัติเหล่านี้ถูกเพิ่มเข้ามาสำหรับฟิสิคัลดิสก์ที่มีเซกเตอร์ขนาดใหญ่

## รูปแบบไฟล์ VHD ##

มีการเข้าถึงระบบไฟล์โฮสต์เพื่อเก็บไฟล์ VHD ไว้ สิ่งเหล่านี้มีคุณสมบัติของอิมเมจคงที่ของดิสก์การ์ดซึ่งอนุญาตให้ผู้ใช้เข้าถึงไฟล์ที่จัดสรรขนาดของดิสก์เสมือนเฉพาะ ไฟล์สามารถปรับเปลี่ยนฮาร์ดดิสก์ของเครื่องเสมือนได้

ไฟล์ VHD เหล่านี้สามารถแลกเปลี่ยนได้อย่างง่ายดายจากฮาร์ดดิสก์เสมือนไปยังระบบไฟล์ของโฮสต์ รูปแบบนี้ยังมีความสามารถในการสำรองและกู้คืนและป้องกันไวรัส ไฟล์ VHD สามารถรองรับรูปแบบเฉพาะ เช่น อิมเมจฮาร์ดดิสก์คงที่ อิมเมจฮาร์ดดิสก์ที่แตกต่างกัน และอิมเมจไดนามิกฮาร์ดดิสก์
