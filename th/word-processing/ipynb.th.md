## ไฟล์ IPYNB คืออะไร??

ไฟล์ที่มีนามสกุล .ipynb คือไฟล์ Jupyter Notebook ที่มีเนื้อหาทั้งหมดของ Notebook ที่สร้างด้วยเซสชันเว็บแอปพลิเคชัน Jupyter Notebook [Jupyter Notebook](https://jupyter.org/) เป็นเว็บแอปพลิเคชันเชิงโต้ตอบที่ใช้สำหรับสร้างและแชร์เอกสารการคำนวณสำหรับการวิเคราะห์โดยใช้ Python

ไฟล์ IPYNB ทำหน้าที่เป็นบันทึกการคำนวณที่สมบูรณ์ของเซสชันและรวมถึงข้อมูล เช่น อินพุตและเอาต์พุตการคำนวณ ฟังก์ชันทางคณิตศาสตร์ และการแสดงอ็อบเจกต์ผลลัพธ์ที่หลากหลาย เช่น รูปภาพและข้อความอธิบาย

## รูปแบบไฟล์ IPYNB

IPYNB ถูกบันทึกภายในในรูปแบบไฟล์ [JSON](/th/web/json/) ซึ่งเป็นรูปแบบไฟล์มาตรฐานแบบเปิดสำหรับการแบ่งปันข้อมูล ด้วยเหตุนี้ ไฟล์ IPYNB จึงสามารถอ่านได้โดยมนุษย์และเข้าใจได้ง่าย JSON ยังทำให้การแบ่งปันไฟล์เหล่านี้กับผู้อื่นเป็นเรื่องง่าย

## วิธีแปลงไฟล์ IPYNB

ไฟล์ IPYNB สามารถส่งออกไปยังรูปแบบไฟล์ต่างๆ ได้โดยใช้คำสั่ง nbconvert จากอินเทอร์เฟซ Jupyter ซึ่งรวมถึงการแปลง IPYNB เป็น:

* ไฟล์ PDF
* HTML
* ปรับโครงสร้างข้อความ
* ลาเท็กซ์

## วิธีดูไฟล์ IPYNB

มีสองวิธีในการดูหรือเปิดไฟล์ IPYNB

1. ผู้ใช้สามารถอัปโหลดเอกสารโน้ตบุ๊ก .ipynb ไปยัง URL สาธารณะและเปิดจาก Jupyter Notebook Viewer<nbviewer> ที่แสดงสมุดบันทึกเป็นหน้าเว็บแบบคงที่จาก URL
1. ดาวน์โหลดไฟล์และเปิดในอินสแตนซ์ของ Jupyter Notebook
