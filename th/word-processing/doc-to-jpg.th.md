## เกี่ยวกับการแปลง DOC เป็น JPG

หากคุณเคยทำงานกับ Microsoft Word เวอร์ชันเก่า (เช่น เวอร์ชันก่อน Microsoft Word 2007) คุณต้องคุ้นเคยกับรูปแบบไฟล์ DOC มันเป็นรูปแบบไฟล์เริ่มต้นที่ใช้โดย Microsoft Word ก่อนที่จะมีการเปิดตัวรูปแบบไฟล์ DOCX ใน Microsoft Word 2007 เวอร์ชัน DOC เป็น JPG เป็นข้อกำหนดทั่วไป โดยเฉพาะอย่างยิ่งเมื่อ Microsoft ไม่อนุญาตให้คุณส่งออกเอกสาร word เป็นรูปภาพ JPG นอกจากนี้ Microsoft Word เวอร์ชันล่าสุดยังไม่เข้ากันกับไฟล์ DOC รุ่นเก่าได้อย่างสมบูรณ์ คุณสามารถใช้แอปออนไลน์ เช่น แอป Aspose.Words เพื่อแปลงรูปแบบไฟล์ [DOC](/th/word-processing/doc/) เป็นรูปแบบไฟล์ [JPG](/th/image/jpeg/)

## การแปลง DOC เป็น JPG - สาธิตฟรี

แอป Aspose.Words เป็นชุดเครื่องมือออนไลน์ที่ให้คุณแปลงไฟล์ DOC ของคุณเป็นรูปแบบไฟล์ต่างๆ คุณสามารถแปลงไฟล์ DOC เป็น JPG ได้ด้วยการคลิกเพียงไม่กี่ครั้งจากเว็บเบราว์เซอร์ของคุณ ข้อดีของการใช้แอป Aspose.Words คือไม่ต้องเขียนโค้ด รวดเร็วและแม่นยำ และที่สำคัญที่สุดคือมันฟรีโดยสิ้นเชิง

ในระหว่างกระบวนการ DOC เป็น JPG แต่ละหน้าจากเอกสารจะถูกแปลงเป็นรูปภาพ JPG และพร้อมให้ดาวน์โหลด ผลลัพธ์คือชุดของภาพ JPG ที่ถูกบันทึกลงแผ่นดิสก์

### วิธีแปลง DOC เป็น JPG

ดังนั้นคุณจะแปลง DOC เป็น JPG ได้อย่างไร คุณสามารถแปลงรูปแบบไฟล์ DOC เป็น JPG ได้โดยใช้ [Aspose.Words DOC เป็น JPG](https://products.aspose.app/words/conversion/doc-to-jpg) ดังนี้

1. ไปที่แอป Aspose.Words สำหรับ [ตัวแปลง DOC เป็น JPG](https://products.aspose.app/words/conversion/doc-to-jpg)
1. อัปโหลดไฟล์ DOC ของคุณโดยคลิกปุ่มอัปโหลด
1. เมื่อเสร็จแล้วให้กดปุ่มแปลง นี่จะเริ่มกระบวนการแปลงไฟล์ DOC เป็น JPG ในเบื้องหลัง
1. คุณจะถูกขอให้ดาวน์โหลดไฟล์ JPG ที่แปลงแล้วเมื่อกระบวนการแปลง DOC เป็น JPG เสร็จสิ้น

## Aspose.Words API

ชุด API ของ Aspose.Words ช่วยให้นักพัฒนาสามารถแปลงรูปแบบ [DOC](/th/word-processing/doc/) เป็นรูปแบบ [JPG](/th/image/jpeg/) ได้โดยใช้โค้ดเพียงไม่กี่บรรทัด สามารถใช้ได้กับภาษาการเขียนโปรแกรมหลายภาษา รวมถึง C#, Java, Python และ C++ นอกจาก DOCX แล้ว Aspose.Words API ยังให้คุณแปลงไฟล์ DOC เป็นรูปแบบไฟล์ยอดนิยมอื่นๆ มากมาย
