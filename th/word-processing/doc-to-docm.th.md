## เกี่ยวกับการแปลง DOC เป็น DOCM

DOCM เป็นเอกสาร Microsoft Word เวอร์ชันที่เปิดใช้งานแมโครซึ่งสามารถเรียกใช้แมโครได้ ความแตกต่างระหว่างเอกสาร Word ปกติและเอกสารที่เปิดใช้งานแมโครคือ DOCM มีคุณสมบัติเพิ่มเติมสำหรับการรองรับแมโคร

ไฟล์ DOC สามารถแปลงเป็น DOCM โดยใช้เครื่องมือและแอปออนไลน์ฟรี **[Aspose.Words App](https://products.aspose.app/words/family)** คือชุดแอปออนไลน์ที่ให้คุณดำเนินการต่างๆ กับเอกสาร Word ได้โดยตรงจากเว็บเบราว์เซอร์ นอกจากฟีเจอร์ของ [การดูเอกสาร](/th/word-processing/docx-viewer/) การแยก การผสาน การเปรียบเทียบ การหมุน และอื่นๆ แล้ว ยังช่วยให้คุณแปลงเอกสาร Word จากรูปแบบหนึ่งเป็นรูปแบบอื่นได้ คุณสามารถแปลง [DOC](/th/word-processing/doc/) เป็น [DOCM](/th/word-processing/docm/) ได้โดยใช้แอป Aspose.Words

## แปลง DOC เป็น DOCM - สาธิตฟรี

แอป Aspose.Words ให้คุณแปลง DOC เป็น DOCM ได้ฟรีทางออนไลน์ คุณสามารถเข้าถึงการแปลง DOC เป็น DOCM ได้จากภายในเว็บเบราว์เซอร์ของคุณ มันรวดเร็วและแม่นยำ ไม่ต้องเขียนโค้ด และไม่มีค่าใช้จ่ายใดๆ ทั้งสิ้น ยิ่งไปกว่านั้น คุณไม่จำเป็นต้องติดตั้งซอฟต์แวร์หรือเครื่องมือเพิ่มเติมใด ๆ บนพีซีของคุณเพื่อทำการแปลง DOC เป็น DOCM

### วิธีแปลง DOC เป็น DOCM

คุณสามารถแปลงรูปแบบไฟล์ DOC เป็นรูปแบบ DOCM ได้โดยใช้ [Aspose.Words DOC เป็นแอป DOCM](https://products.aspose.app/words/conversion/doc-to-docm) ดังนี้

1. ไปที่แอป Aspose.Words สำหรับ [ตัวแปลง DOC เป็น DOCM](https://products.aspose.app/words/conversion/doc-to-docm)
1. อัปโหลดไฟล์ DOC ของคุณโดยคลิกปุ่มอัปโหลด
1. เมื่อเสร็จแล้วให้กดปุ่มแปลง นี่จะเริ่มกระบวนการแปลงไฟล์ DOC เป็น DOCM ในเบื้องหลัง
1. คุณจะถูกขอให้ดาวน์โหลดไฟล์ DOCM ที่แปลงแล้วเมื่อกระบวนการแปลง DOC เป็น DOCM เสร็จสิ้น

## Aspose.Words API

[![Aspose.Words](../try-aspose-words.png)](https://products.aspose.com/words/)

ชุด API ของ Aspose.Words ช่วยให้นักพัฒนาสามารถแปลงรูปแบบ [DOC](/th/word-processing/doc/) เป็นรูปแบบ [DOCM](/th/word-processing/docm/) ได้โดยใช้โค้ดเพียงไม่กี่บรรทัด สามารถใช้ได้กับภาษาการเขียนโปรแกรมหลายภาษา รวมถึง C#, Java, Python และ C++ นอกจาก DOCM แล้ว Aspose.Words API ยังให้คุณแปลงไฟล์ DOC เป็นรูปแบบไฟล์ยอดนิยมอื่นๆ มากมาย

## โปรแกรมดู DOC

หากคุณไม่มี Microsoft Word หรือซอฟต์แวร์อื่นๆ ที่เข้ากันได้ติดตั้งอยู่ในพีซีของคุณเพื่อเปิดและดูไฟล์ DOC คุณสามารถใช้แอปฟรี [Aspose.Words Viewer](/th/word-processing/docx-viewer/) เพื่อเปิดและ ดูไฟล์ DOC ออนไลน์
