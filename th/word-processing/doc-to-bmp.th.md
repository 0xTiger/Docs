## เกี่ยวกับการแปลง DOC เป็น BMP

หากคุณเป็นผู้ใช้ Microsoft Word ในเวอร์ชันก่อนปี 2007 คุณต้องคุ้นเคยกับประเภทไฟล์ DOC ซึ่งเป็นรูปแบบไฟล์เริ่มต้นที่ Microsoft Word ใช้เพื่อบันทึกเอกสารของคุณในขณะนั้น มีการใช้งานมาตั้งแต่เริ่มต้น Microsoft Word และยังคงอยู่ที่นั่นเป็นเวลานานหลังจากที่ถูกแทนที่ด้วยรูปแบบไฟล์ DOC ล่าสุด ระบบการจัดการเนื้อหาอาจยังคงมีไฟล์ DOC ไว้เป็นข้อมูลสำรองที่สามารถแปลงเป็นอิมเมจ BMP ได้ เครื่องมือออนไลน์ เช่น แอป Aspose.Word สำหรับการแปลงช่วยให้คุณสามารถแปลงไฟล์ [DOC](/th/word-processing/doc/) เป็นรูปภาพ [BMP](/th/image/bmp/) ได้

## การแปลง DOC เป็น BMP - สาธิตฟรี

แอป Aspose.Words คือชุดเครื่องมือออนไลน์ที่พัฒนาขึ้นจาก API การเขียนโปรแกรมยอดนิยมของ Aspose มันมีฟังก์ชันการทำงานต่างๆ เช่น การดูเอกสาร การแปลง การบีบอัด การหมุน การแยก การรวม และอื่นๆ อีกมากมาย คุณสามารถแปลง [DOC](/th/word-processing/doc/) เป็น [BMP](/th/image/bmp/) ได้โดยใช้แอป Aspose.Words จากเว็บเบราว์เซอร์ของคุณโดยตรง คุณไม่จำเป็นต้องติดตั้งซอฟต์แวร์ใด ๆ เพื่อทำการแปลง

### ขั้นตอนในการแปลง DOC เป็น BMP ออนไลน์

คุณสามารถแปลงรูปแบบไฟล์ DOC เป็นรูปแบบ BMP ได้โดยใช้ [Aspose.Words DOC เป็นแอป BMP](https://products.aspose.app/words/conversion/doc-to-bmp) ดังนี้

1. ไปที่แอป Aspose.Words สำหรับ [ตัวแปลง DOC เป็น BMP](https://products.aspose.app/words/conversion/doc-to-bmp)
1. อัปโหลดไฟล์ DOC ของคุณโดยคลิกปุ่มอัปโหลด
1. เมื่อเสร็จแล้วให้กดปุ่มแปลง นี่จะเริ่มกระบวนการแปลงไฟล์ DOC เป็น BMP ในเบื้องหลัง
1. คุณจะถูกขอให้ดาวน์โหลดไฟล์ BMP ที่แปลงแล้วเมื่อกระบวนการแปลง DOC เป็น BMP เสร็จสิ้น
