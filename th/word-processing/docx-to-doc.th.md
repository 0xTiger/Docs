## เกี่ยวกับการแปลง DOCX เป็น DOC

ประเภทไฟล์ DOC เป็นรูปแบบไฟล์เอกสารยอดนิยมที่ใช้ในปี 2550 และ Microsoft Word เวอร์ชันก่อนหน้า ด้วยการเปิดตัว Office 2007 Microsoft ได้เปิดตัวรูปแบบไฟล์ DOCX ตามข้อกำหนด Office Open XML ไฟล์ DOC ได้รับการบันทึกเป็นไฟล์ไบนารีเมื่อเปรียบเทียบกับ DOCX ซึ่งบันทึกในรูปแบบไฟล์ XML ที่บีบอัด มีเครื่องมือมากมายทางออนไลน์สำหรับการแปลงไฟล์ [DOCX](/th/word-processing/docx/) เป็น [DOC](/th/word-processing/doc/)

## การแปลง DOCX เป็น DOC - สาธิตฟรี

คุณสามารถแปลงรูปแบบไฟล์ DOCX เป็นรูปแบบ DOC ได้โดยใช้แอปออนไลน์ฟรี Aspose.app เป็นชุดแอปที่สามารถโหลด ดู และแปลงไฟล์ DOCX เป็นรูปแบบไฟล์ เช่น DOC, PDF, TXT, RTF และอื่นๆ แอป Aspose ได้รับการขับเคลื่อนและดำเนินการโดย Aspose API ที่แบ็กเอนด์ซึ่งสามารถดำเนินการหลายอย่างกับไฟล์ประเภทต่างๆ ลองใช้แอปออนไลน์ Aspose.Words ทันทีโดยคลิกปุ่มต่อไปนี้

[![DOCX เป็น DOC](../docx-to-doc.png?width=120px&height=60px)](https://products.aspose.app/words/conversion/docx-to-doc)

### ขั้นตอนในการแปลง DOCX เป็น DOC ออนไลน์

คุณสามารถแปลงรูปแบบไฟล์ DOCX เป็นรูปแบบ DOC ได้โดยใช้ [Aspose.Words DOCX เป็นแอป DOC](https://products.aspose.app/words/conversion/docx-to-doc) ดังนี้

1. ไปที่แอป Aspose.Words สำหรับ [ตัวแปลง DOCX เป็น DOC](https://products.aspose.app/words/conversion/docx-to-doc)
1. อัปโหลดไฟล์ DOCX ของคุณโดยคลิกปุ่มอัปโหลด
1. เมื่อเสร็จแล้วให้กดปุ่มแปลง นี่จะเริ่มกระบวนการแปลงไฟล์ DOCX เป็น DOC ในเบื้องหลัง
1. คุณจะถูกขอให้ดาวน์โหลดไฟล์ RTF ที่แปลงแล้วเมื่อกระบวนการแปลง DOCX เป็น DOC เสร็จสิ้น

## Aspose.Words API

[Aspose.Words](https://products.aspose.com/words/) คือชุดของ API ที่ช่วยให้นักพัฒนาซอฟต์แวร์สามารถแปลง [DOCX](/th/word-processing/docx/) เป็น [DOC](/th/word-processing /doc/) โดยใช้โค้ดบรรทัดง่ายๆ สามารถใช้ได้กับภาษาการเขียนโปรแกรมหลายภาษา รวมถึง [C#](/th/programming/cs/), [Java](/th/programming/java/), [Python](/th/programming/py/) และ [C++](/th/programming/cpp/ ). นอกจาก DOC แล้ว Aspose.Words API ยังช่วยให้คุณแปลงไฟล์ DOCX ไปเป็นรูปแบบไฟล์ยอดนิยมอื่นๆ ได้อีกมากมาย
