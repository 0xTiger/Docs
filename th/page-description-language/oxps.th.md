## ไฟล์ OXPS คืออะไร?? ##

รูปแบบไฟล์ OXPS เรียกว่า Open XML Paper Specification เป็นภาษาคำอธิบายหน้าและรูปแบบเอกสาร Microsoft เป็นผู้พัฒนารูปแบบนี้ รูปแบบไฟล์ OXPS คุ้นเคยกับไฟล์ PDF เหล่านี้เป็นอย่างดี การตั้งค่าไฟล์ OXPS จะเหมือนกับรูปแบบ XPS ข้อแตกต่างเพียงอย่างเดียวคือคำอธิบายของ OXPS เป็นรูปแบบเปิด นามสกุลไฟล์ .oxps เป็นรูปแบบไฟล์ XPS เวอร์ชันล่าสุดที่มาจาก Microsoft และ Ecma International หากเราพูดถึงข้อกำหนด ไฟล์ OXPS และ XPS จะใช้ในการเผยแพร่ แบ่งปัน และจัดสรรเอกสารที่มีเค้าโครงตายตัว แทนที่ไฟล์ PDF รูปแบบไฟล์ OXPS สามารถจัดเก็บข้อความและข้อมูลมัลติมีเดีย พร้อมคุณสมบัติการจัดเรียงและลักษณะเค้าโครงการพิมพ์

## ประวัติย่อ ##

ทีม Microsoft เลือก Global Graphics เพื่อให้คำปรึกษาเกี่ยวกับ XPS ในปี 2546 และทีมพัฒนาทำงานอย่างใกล้ชิดกับทีม Windows เพื่อพัฒนาข้อมูลจำเพาะและสถาปัตยกรรมอ้างอิงสำหรับรูปแบบใหม่ รับทราบข้อกำหนด XPS ในเดือนมิถุนายน พ.ศ. 2550 Ecma International ได้จัดตั้งคณะกรรมการด้านเทคนิค 46 (TC46) เพื่อพัฒนามาตรฐานตามข้อกำหนด OpenXPS Ecma International ได้รับการอนุมัติ Open XML Paper Specification เป็นมาตรฐาน Ecma (ECMA-388) ในการประชุมสมัชชาครั้งที่ 97 ที่กรุงบูดาเปสต์เมื่อวันที่ 16 มิถุนายน 2552
