## ไฟล์ SAMI คืออะไร??

โดยทั่วไปไฟล์ที่มีนามสกุล ".sami" จะหมายถึงไฟล์ Subtitles And Metadata Interchange (SAMI) SAMI เป็นรูปแบบคำบรรยายที่ใช้สำหรับแสดงคำบรรยายในวิดีโอ เดิมได้รับการพัฒนาโดย Microsoft สำหรับ Windows Media Player

ไฟล์ SAMI มีข้อมูลเวลาและเนื้อหาข้อความสำหรับคำบรรยาย ทำให้สามารถซิงโครไนซ์กับการเล่นวิดีโอได้ รูปแบบนี้รองรับตัวเลือกการจัดรูปแบบพื้นฐาน เช่น ลักษณะแบบอักษร สี และตำแหน่งของคำบรรยายบนหน้าจอ

ไฟล์ SAMI มักเป็นไฟล์ข้อความธรรมดาและสามารถเปิดและแก้ไขได้โดยใช้โปรแกรมแก้ไขข้อความ โดยทั่วไปเนื้อหาของไฟล์ SAMI จะมีส่วนหัวที่ให้ข้อมูลเกี่ยวกับไฟล์ ตามด้วยรายการคำบรรยายแต่ละรายการพร้อมช่วงเวลาและข้อความตามลำดับ

ต่อไปนี้คือตัวอย่างลักษณะของไฟล์ SAMI:

```
<SAMI>
<HEAD>
<TITLE>Example Subtitles</TITLE>
</HEAD>
<BODY>
<SYNC Start=100><P Class=ENCC>Subtitle 1</P></SYNC>
<SYNC Start=500><P Class=ENCC>Subtitle 2</P></SYNC>
<SYNC Start=1000><P Class=ENCC>Subtitle 3</P></SYNC>
</BODY>
</SAMI>
```

โดยทั่วไปไฟล์ SAMI จะใช้ร่วมกับเครื่องเล่นวิดีโอหรือเครื่องเล่นสื่อที่รองรับการแสดงคำบรรยาย เช่น Windows Media Player หรือ VLC Media Player โปรแกรมเล่นอ่านไฟล์ SAMI และซิงโครไนซ์คำบรรยายกับเนื้อหาวิดีโอ ทำให้ผู้ดูสามารถอ่านคำบรรยายในขณะที่ดูวิดีโอได้

## แท็ก HTML ที่รองรับ

ไฟล์ SAMI (คำบรรยายและการแลกเปลี่ยนข้อมูลเมตา) รองรับชุดแท็กคล้าย HTML ที่จำกัดสำหรับการจัดรูปแบบและจัดรูปแบบคำบรรยาย นี่คือแท็กบางส่วนที่ใช้กันทั่วไปซึ่งสนับสนุนโดย SAMI:

- ``<SAMI> :`` องค์ประกอบรูทที่ห่อหุ้มไฟล์ SAMI ทั้งหมด
- ``<HEAD> :`` มีข้อมูลส่วนหัวสำหรับไฟล์ SAMI
<html>- ``<TITLE> :`` ระบุชื่อของไฟล์ SAMI </html>
- ``<BODY> :`` แนบรายการคำบรรยายและข้อมูลจังหวะเวลา
- ``<SYNC> :`` แสดงถึงจุดประสานสำหรับรายการคำบรรยาย โดยระบุระยะเวลาที่ควรแสดงคำบรรยาย
- ``<P> :`` ล้อมรอบเนื้อหาข้อความจริงของคำบรรยาย โดยทั่วไปจะใช้ภายในก<SYNC> ปิดกั้น.
<html>- `` <FONT>:`` กำหนดคุณสมบัติแบบอักษรสำหรับข้อความที่แนบมา คุณสมบัติต่างๆ เช่น สี ใบหน้า ขนาด และลักษณะ สามารถใช้เพื่อแก้ไขลักษณะแบบอักษรได้</font>
- ``<BR> :`` แทรกตัวแบ่งบรรทัดภายในคำบรรยาย
<html>- `` <B>:`` แสดงข้อความที่แนบมาด้วยตัวหนา</b>
<html>- `` <I>:`` วาทกรรมข้อความที่แนบมาเป็นตัวเอียง</i>
<html>- `` <U>:`` วาทกรรมข้อความที่ขีดเส้นใต้</u>
- ``<C> :`` ระบุตำแหน่งหรือการจัดตำแหน่งของข้อความคำบรรยายบนหน้าจอ รองรับแอตทริบิวต์เช่น กลาง กลาง ซ้าย ขวา บน ล่าง และชุดค่าผสม
- ``<LANG> :`` ระบุรหัสภาษาสำหรับข้อความคำบรรยาย ช่วยในการระบุภาษาของคำบรรยาย

นี่คือแท็กพื้นฐานบางส่วนที่ไฟล์ SAMI รองรับ สิ่งสำคัญที่ควรทราบก็คือ SAMI ไม่รองรับแท็กและแอตทริบิวต์ HTML เต็มรูปแบบ แท็กที่รองรับมุ่งเน้นไปที่การจัดรูปแบบและการวางตำแหน่งคำบรรยายเป็นหลัก แทนที่จะจัดให้มีโครงสร้างเอกสารหรือการโต้ตอบที่ครอบคลุม
