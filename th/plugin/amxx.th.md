## ไฟล์ AMXX คืออะไร??

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. เป็นแพลตฟอร์มการเขียนสคริปต์ที่ช่วยให้ผู้ดูแลระบบเซิร์ฟเวอร์สามารถปรับแต่งและขยายฟังก์ชันการทำงานของเซิร์ฟเวอร์ของตนได้ AMXX ใช้ร่วมกับระบบการจัดการปลั๊กอิน MetaMod ที่อนุญาตให้ปลั๊กอินหลายตัวทำงานพร้อมกันบนเซิร์ฟเวอร์โดยไม่มีข้อขัดแย้ง คุณสมบัติที่นำเสนอโดยปลั๊กอิน AMXX ได้แก่ คำสั่งของผู้ดูแลระบบ การปรับเปลี่ยนการเล่นเกม และการจัดการเซิร์ฟเวอร์
