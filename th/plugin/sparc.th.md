## ไฟล์ SPARC คืออะไร??

ไฟล์ .sparc หรือที่รู้จักกันในชื่อ Skype Plugin Archive ทำหน้าที่เป็นชุดรวมคุณสมบัติพิเศษหรือการปรับปรุงสำหรับแอปพลิเคชัน Skype ปลั๊กอินเหล่านี้มีความสามารถในการแนะนำความสามารถที่หลากหลาย เช่น การบันทึกการโทร การแชร์หน้าจอ การแปล
เกมและความบันเทิง อีโมติคอนและธีมแบบกำหนดเอง

## รูปแบบไฟล์ SPARC - ข้อมูลเพิ่มเติม

ใน Skype บางเวอร์ชัน Extras Manager เป็นองค์ประกอบเลือกที่สามารถรวมเข้ากับการติดตั้ง Skype ได้ หากคุณประสบปัญหาในการติดตั้งไฟล์ SPARC อาจเป็นไปได้ว่าคุณไม่ได้ระบุ Extras Manager ในระหว่างการติดตั้งครั้งแรก เพื่อแก้ไขปัญหานี้ ให้ติดตั้ง Skype ใหม่และให้แน่ใจว่าคุณเลือกตัวเลือก Extras Manager

## จะติดตั้งปลั๊กอิน Skype ได้อย่างไร?

ขั้นตอนในการติดตั้งปลั๊กอิน Skype:

1. รับไฟล์ .sparc ที่สอดคล้องกับปลั๊กอินที่ต้องการ
2. ปิด Skype หากกำลังทำงานอยู่
3. ค้นหาไฟล์ .sparc ที่ดาวน์โหลดมา และดับเบิลคลิกที่ไฟล์
4. ปฏิบัติตามคำแนะนำบนหน้าจอเพื่อติดตั้งปลั๊กอินให้เสร็จสิ้น
5. หลังการติดตั้ง ให้รีสตาร์ท Skype เพื่อเปิดใช้งานปลั๊กอิน
