## ไฟล์ MPP คืออะไร??

ไฟล์ที่มีนามสกุล .mpp คือไฟล์ข้อมูล Microsoft Project ที่เก็บข้อมูลที่เกี่ยวข้องกับการจัดการโครงการในลักษณะที่บูรณาการ Microsoft ได้พัฒนารูปแบบไฟล์ที่เป็นกรรมสิทธิ์นี้เพื่อให้เข้ากันได้กับ Microsoft Project (MSP) ซึ่งเป็นผลิตภัณฑ์ซอฟต์แวร์การจัดการโครงการของพวกเขา นอกเหนือจาก MPP แล้ว MSP สามารถทำงานกับ XML schema ของโครงการได้ แอปพลิเคชั่นและ API จำนวนมากอนุญาตให้แปลง MPP เป็นรูปแบบไฟล์อื่น ขณะนี้ Microsoft มีเซิร์ฟเวอร์โครงการออนไลน์ที่สามารถอัปโหลดไฟล์การจัดการโครงการสำหรับการทำงานร่วมกันโดยผู้ใช้หลายคน

## รูปแบบไฟล์ MPP ##

Microsoft พัฒนาและเผยแพร่ MPP เป็นรูปแบบไฟล์ไบนารี และ Microsoft ยังไม่ได้เผยแพร่ข้อมูลจำเพาะต่อสาธารณะจนถึงปัจจุบัน โครงสร้างและรายละเอียดไฟล์ภายในจึงไม่พร้อมใช้งานสำหรับนักพัฒนา มี API ออนไลน์หลายตัวที่รองรับการอ่าน/เขียนไฟล์ MPP แต่ไม่เคยได้รับความแม่นยำ 100% Microsoft ได้แนะนำคุณสมบัติหลายอย่างให้กับแอปพลิเคชันการจัดการโครงการเมื่อเวลาผ่านไป และดังนั้นจึงมีคุณลักษณะของรูปแบบไฟล์ MPP

ไฟล์ MPP ใช้ประเภท MIME ต่อไปนี้:

* แอปพลิเคชัน/vnd.ms-project
* แอปพลิเคชัน/msproj
* แอปพลิเคชัน/msproject
* แอปพลิเคชัน/x-msproject
* แอปพลิเคชัน / x-ms-project
* แอปพลิเคชัน/x-dos_ms_project
* แอปพลิเคชัน / MPP
* zz-application/zz-winassoc-mpp

## ปัญหาในการเปิดไฟล์ MPP? ##

ต่อไปนี้คือรายการปัญหาทั่วไปที่อาจเกิดขึ้นและทำให้รูปแบบ MPP ทำงานผิดพลาด:

* ไม่มีซอฟต์แวร์สนับสนุน
* ไฟล์เสียหาย
* ไฟล์ติดไวรัสเนื่องจากไวรัส
* ไม่มีสิทธิ์ในระบบในการเปิดไฟล์
* ไดรฟ์ที่ล้าสมัยในระบบของคุณ
* นามสกุลของไฟล์ถูกเปลี่ยนชื่อ
