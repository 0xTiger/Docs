## ไฟล์ MPT คืออะไร?? ##

ไฟล์ที่มีนามสกุล .mpt คือไฟล์เทมเพลต [Microsoft Project](https://products.office.com/en-us/project/project-and-portfolio-management-software) ประกอบด้วยข้อมูลพื้นฐานและโครงสร้างและการตั้งค่าเอกสารสำหรับการสร้างไฟล์ [MPP](/th/project-management/mpp/) เทมเพลตเหล่านี้มีการตั้งค่าเริ่มต้นที่จำเป็นสำหรับโครงการเฉพาะ เช่น กำหนดการหรือข้อมูลงบประมาณ แม้ว่าจะไม่สามารถบันทึกข้อมูลเฉพาะโครงการ เช่น งาน ทรัพยากร หรือการมอบหมายงาน เมื่อแก้ไขแล้ว สามารถบันทึกไฟล์เทมเพลตเป็นไฟล์ MPP มาตรฐานเพื่อใช้งานต่อไปได้ ไฟล์เทมเพลตหลักชื่อ "Global.mpt" มีอยู่ใน Microsoft Project ซึ่งเปิดขึ้นเพื่อสร้างเทมเพลตแต่ละรายการ ไฟล์ MPT ใช้ *application/vnd.ms-project* และ *application/x-project* เป็นประเภท MIME มาตรฐาน Windows, Linux และ macOS สามารถเปิดไฟล์ MPT ได้ทั้งหมด รวมถึง Moos Project Viewer, Microsoft Project และ MPXJ

## ปัญหาในการเปิดไฟล์ MPT? ##

ต่อไปนี้เป็นรายการปัญหาทั่วไปที่อาจเกิดขึ้นและทำให้รูปแบบ MPT ทำงานผิดพลาด:

* ไม่มีซอฟต์แวร์สนับสนุน
* ไฟล์เสียหาย
* ไฟล์ติดไวรัสเนื่องจากไวรัส
* ไม่มีสิทธิ์ในระบบในการเปิดไฟล์
* ไดรฟ์ที่ล้าสมัยในระบบของคุณ
 *   Extension of the file is renamed
