## ไฟล์ CUR คืออะไร?? ##

ไฟล์ CUR เป็นรูปแบบไฟล์เคอร์เซอร์ Microsoft Windows แบบคงที่ โดยพื้นฐานแล้วจะเป็นภาพนิ่งเหมือนกับไฟล์ ICO (ไอคอน) ทุกประการ ยกเว้นส่วนขยาย ทั้ง CUR และ [ICO](/th/image/ico/) สร้างขึ้นจากข้อมูลจำเพาะบิตแมปที่ไม่ขึ้นกับอุปกรณ์ [DIB](/th/image/dib/) (บิตแมปที่ไม่ขึ้นกับอุปกรณ์) ค่าเริ่มต้นและเคอร์เซอร์แบบกำหนดเอง เช่น ตัวชี้เมาส์ Windows สำหรับระบบปฏิบัติการ Windows จะถูกจัดเก็บไว้ในไฟล์ CUR อาจเป็นลูกศรสำหรับการใช้งานปกติ นาฬิกาทรายหมุนเพื่อแสดงระยะเวลารอคอย หรือแถบ I สำหรับแก้ไขข้อความ ไฟล์ CUR มาพร้อมกับชุดการติดตั้งของธีมเดสก์ท็อปที่กำหนดเองเพื่อให้แน่ใจว่าเคอร์เซอร์ของ Windows ตรงกับธีมเดสก์ท็อปที่กำหนดเอง

## ข้อมูลทางเทคนิค ##

ไฟล์ CUR เป็นไฟล์ระบบไบนารีที่สามารถพบได้ในอุปกรณ์ที่ทำงานบนระบบปฏิบัติการ Microsoft Windows ภาพตัวชี้ CUR มีขนาดมาตรฐานต่างๆ เช่น 16×16, 32×32 เป็นต้น ไฟล์ CUR นั้นคล้ายกับไฟล์ ICO มาก; ทั้งสองประกอบด้วยภาพนิ่งขนาดเล็ก นามสกุลของไฟล์ CUR และ ICO เท่านั้นที่แตกต่างกัน นอกจากนี้ คำอธิบายของฮอตสปอตในส่วนหัวของไฟล์ CUR จะแตกต่างจากไฟล์ ICO ฮอตสปอตแปลค่าชดเชยพิกเซลจากมุมบนซ้ายไปยังตำแหน่งที่คุณชี้เมาส์ ระบบ Windows ยังคงใช้ไฟล์ CUR แต่ตอนนี้เคอร์เซอร์แบบเคลื่อนไหวมักจะมีนามสกุลไฟล์ ANI แทน CUR

## ประวัติย่อ ##

ไฟล์ CUR สร้างจากไฟล์ ICO รูปแบบไฟล์ CUR แรกรวมอยู่ในระบบปฏิบัติการ Windows 1.0 ของ Microsoft ในปี 1981 เพื่อให้การใช้งานเชิงพาณิชย์ราบรื่นขึ้น ในไม่ช้า เคอร์เซอร์แบบโต้ตอบก็ได้รับการแนะนำมากขึ้น ทำให้ผู้ใช้สามารถแก้ไขการตั้งค่าเคอร์เซอร์จากตัวเลือกที่ติดตั้งไว้ล่วงหน้าที่มีให้ อย่างไรก็ตาม การแก้ไขไฟล์ CUR ด้วยตัวเองนั้นเป็นเรื่องง่าย แม้ว่าคุณจะมีความรู้ทางเทคนิคไม่เพียงพอก็ตาม
