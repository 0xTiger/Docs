## ไฟล์ AGIF คืออะไร??

ไฟล์ AGIF เป็นไฟล์ GIF ที่สร้างด้วย Active GIF Creator เป็นไฟล์โครงการที่สามารถเปิดและดูได้ในโครงการ Active GIF Creator เมื่อเปิดใน Active GIF Creator มันจะแสดงลำดับของภาพที่ประกอบขึ้นเป็นไฟล์ GIF ที่สรุปผลแล้วด้วยนามสกุล .gif

## รูปแบบไฟล์ AGIF

AGIF เป็นไฟล์โครงการที่บันทึกเป็นรูปแบบไฟล์ภาพแรสเตอร์ ไฟล์ [GIF](/image/gif/) ที่สรุปผลแล้วสามารถเปิดและแสดงเป็นภาพเคลื่อนไหวได้ มี Python APIs ทางออนไลน์ที่สามารถใช้สร้างไฟล์ GIF แบบเคลื่อนไหวได้
