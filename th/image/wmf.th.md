## ไฟล์ WMF คืออะไร??

ไฟล์ที่มีนามสกุล WMF เป็นตัวแทนของ Microsoft Windows Metafile (WMF) สำหรับจัดเก็บข้อมูลภาพเวกเตอร์และรูปแบบบิตแมป เพื่อให้แม่นยำยิ่งขึ้น WMF อยู่ในหมวดหมู่รูปแบบไฟล์เวกเตอร์ของรูปแบบไฟล์กราฟิกที่ไม่ขึ้นกับอุปกรณ์ Windows Graphical Device Interface (GDI) ใช้ฟังก์ชันที่จัดเก็บไว้ในไฟล์ WMF เพื่อแสดงภาพบนหน้าจอ WMF เวอร์ชันที่ได้รับการปรับปรุงเพิ่มเติมหรือที่เรียกว่า Enhanced Meta Files (EMF) ได้รับการเผยแพร่ในภายหลังซึ่งทำให้รูปแบบมีคุณลักษณะที่หลากหลายมากขึ้น ในทางปฏิบัติ WMF คล้ายกับ SVG

## ข้อมูลจำเพาะรูปแบบไฟล์ WMF ##

ไฟล์ WMF อ้างถึงรูปแบบไฟล์ 16 บิต ณ เวลาที่เปิดตัวพร้อมกับ Windows 3.0 รูปแบบไฟล์ประกอบด้วยชุดของเร็กคอร์ดที่มีความยาวผันแปรได้ที่ประกอบด้วยคำสั่งการวาดกราฟิก คำจำกัดความของวัตถุ และคุณสมบัติ เนื่องจากไฟล์ WMF ขึ้นอยู่กับคำสั่งที่แสดงผลไปยัง GDI เพื่อวาดภาพ จึงเรียกอีกอย่างว่าการบันทึกภาพแบบดิจิทัลซึ่งสามารถเปิดเล่นเพื่อสร้างภาพนั้นซ้ำได้ ข้อกำหนดรูปแบบไฟล์ที่สมบูรณ์ของ WMF มีให้ทางออนไลน์ และควรอ้างอิงสำหรับการพัฒนาแอปพลิเคชันเพื่อทำงานกับไฟล์ WMF ไฟล์ WMF ถูกจัดระเบียบเป็น:

* บันทึกส่วนหัว WMF
* บันทึก WMF (s)
* WMF End-of-File บันทึก

### บันทึกส่วนหัว WMF ###

**META_HEADER Record** มีข้อมูลที่กำหนดคุณลักษณะของ metafile รวมถึง:

* ประเภทของเมตาไฟล์
* รุ่นของ metafile
* ขนาดของเมตาไฟล์
* จำนวนของวัตถุที่กำหนดไว้ใน metafile
* ขนาดของระเบียนเดียวที่ใหญ่ที่สุดใน metafile

### บันทึกส่วนหัวที่วางได้ของ WMF ###

**META_PLACEABLE Record** มีข้อมูลเพิ่มเติมเกี่ยวกับรูปภาพ รวมถึง:

* สี่เหลี่ยมผืนผ้าที่มีขอบเขต
* ขนาดหน่วยตรรกะ สำหรับการปรับขนาด
* การตรวจสอบความถูกต้อง

### บันทึก WMF ###

ระเบียน WMF มีรูปแบบทั่วไป ซึ่งระบุไว้ในเอกสารข้อกำหนด ทุกบันทึก WMF มีข้อมูลต่อไปนี้:

* ขนาดบันทึก
* ฟังก์ชั่นการบันทึก
* พารามิเตอร์ (ถ้ามี) สำหรับฟังก์ชันบันทึก
