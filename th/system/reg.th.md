## ไฟล์ REG คืออะไร??

ไฟล์ REG เป็นรูปแบบไฟล์ที่ใช้ในการนำเข้าหรือส่งออกข้อมูล Windows Registry Windows Registry เป็นฐานข้อมูลแบบลำดับชั้นที่เก็บการตั้งค่าการกำหนดค่าและตัวเลือกสำหรับระบบปฏิบัติการ Windows และแอปพลิเคชันที่ติดตั้ง รีจิสทรีประกอบด้วยข้อมูลต่างๆ เช่น การตั้งค่าผู้ใช้ การตั้งค่าแอปพลิเคชัน ข้อมูลการกำหนดค่าฮาร์ดแวร์และซอฟต์แวร์ และอื่นๆ

โดยทั่วไปไฟล์ REG จะมีนามสกุลไฟล์ ".reg" และเป็นไฟล์ข้อความธรรมดาที่ประกอบด้วยชุดของรายการรีจิสตรีและค่าในรูปแบบเฉพาะ รูปแบบประกอบด้วยส่วนหัวที่ระบุไฟล์เป็นไฟล์รีจิสตรี ตามด้วยชุดของคู่คีย์และค่าที่แสดงถึงรายการรีจิสตรี

## รูปแบบไฟล์ REG - ข้อมูลเพิ่มเติม

นี่คือตัวอย่างรูปแบบไฟล์ reg:

```
[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Run]
"Example"="C:\\Example.exe"

[HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced]
"Hidden"=dword:00000001
```

บรรทัดแรกของไฟล์ระบุเวอร์ชันของตัวแก้ไขรีจิสทรี บรรทัดต่อไปนี้ประกอบด้วยรายการรีจิสทรีในรูปแบบของเส้นทางคีย์ตามด้วยชื่อค่าและข้อมูลค่า ในตัวอย่างนี้ ไฟล์ reg มีสองรายการ: รายการหนึ่งเพิ่มโปรแกรมชื่อ "Example.exe" ลงในรายการเริ่มต้น Windows และอีกรายการหนึ่งที่ตั้งค่า "ซ่อน" ในการตั้งค่าขั้นสูงของ Windows Explorer เป็น "จริง"

ไฟล์ Reg สามารถสร้างและแก้ไขได้โดยใช้โปรแกรมแก้ไขข้อความ เช่น Notepad มักใช้เพื่อการสำรองและกู้คืนข้อมูล รวมถึงการกำหนดค่าคอมพิวเตอร์หลายเครื่องด้วยการตั้งค่ารีจิสทรีเดียวกัน

## นำเข้าหรือส่งออกรีจิสทรีของ Windows

ไฟล์ REG เป็นไฟล์ประเภทหนึ่งที่ใช้ในการนำเข้าหรือส่งออกข้อมูลจาก Windows Registry Windows Registry เป็นฐานข้อมูลแบบลำดับชั้นที่เก็บการตั้งค่าการกำหนดค่าและตัวเลือกสำหรับระบบปฏิบัติการ Windows และแอปพลิเคชันที่ติดตั้ง รีจิสทรีประกอบด้วยข้อมูลต่างๆ เช่น การตั้งค่าผู้ใช้ การตั้งค่าแอปพลิเคชัน ข้อมูลการกำหนดค่าฮาร์ดแวร์และซอฟต์แวร์ และอื่นๆ

ไฟล์ Reg สามารถใช้เพื่อสร้างหรือแก้ไขรายการรีจิสตรี และมักใช้เพื่อวัตถุประสงค์ในการสำรองและกู้คืน รวมถึงการกำหนดค่าคอมพิวเตอร์หลายเครื่องด้วยการตั้งค่ารีจิสทรีเดียวกัน ต่อไปนี้คือวิธีใช้ไฟล์ reg เพื่อเพิ่มรายการรีจิสทรีใหม่ลงใน Windows Registry:

1. สร้างไฟล์ข้อความใหม่โดยใช้โปรแกรมแก้ไขข้อความ เช่น Notepad
2. พิมพ์รายการรีจิสทรีในรูปแบบที่ถูกต้อง รูปแบบประกอบด้วยส่วนหัวที่ระบุไฟล์เป็นไฟล์รีจิสตรี ตามด้วยชุดของคู่คีย์และค่าที่แสดงถึงรายการรีจิสตรี นี่คือตัวอย่าง:

```
Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\Software\Example]
"Setting1"="Value1"
```

ซึ่งจะสร้างคีย์ใหม่ชื่อ "ตัวอย่าง" ใต้คีย์ "ซอฟต์แวร์" ในกลุ่มรีจิสทรีของผู้ใช้ปัจจุบัน และตั้งค่า "Setting1" เป็น "Value1"

3. บันทึกไฟล์ด้วยนามสกุลไฟล์ .reg
4. ดับเบิลคลิกที่ไฟล์ .reg เพื่อเพิ่มรายการรีจิสทรีใหม่ลงใน Windows Registry

คุณจะได้รับแจ้งให้ยืนยันว่าคุณต้องการเพิ่มรายการลงในรีจิสทรี เมื่อคุณยืนยันแล้ว รายการใหม่จะถูกเพิ่มลงในรีจิสทรี และคุณสามารถตรวจสอบได้โดยใช้เครื่องมือตัวแก้ไขรีจิสทรี
