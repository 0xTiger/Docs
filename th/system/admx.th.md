## ไฟล์ ADMX คืออะไร??

ไฟล์ ADMX เป็นไฟล์กำหนดค่านโยบายที่ใช้โดยซอฟต์แวร์ Microsoft Windows Group Policy ที่จัดการกลุ่มคอมพิวเตอร์ในกลุ่ม เป็นไฟล์เทมเพลตการดูแลระบบที่ใช้ XML และเปิดตัวกับ Windows Vista Service Pack 1 ไฟล์ ADMX มีการตั้งค่าการกำหนดค่าสำหรับบัญชีผู้ใช้ การกำหนดค่าระบบปฏิบัติการ และแอปพลิเคชัน ไฟล์ ADMX ใช้เพื่อจัดเก็บและปรับใช้การกำหนดค่าสำหรับการจัดการแบบรวมศูนย์ของระบบคอมพิวเตอร์จำนวนมาก

คุณสามารถสร้างหรือแก้ไขไฟล์ ADMX โดยใช้ตัวแก้ไขข้อความที่เข้ากันได้กับ XML หรือตัวแก้ไขวัตถุนโยบายกลุ่มของ Microsoft

## รูปแบบไฟล์ ADMX - ข้อมูลเพิ่มเติม

ไฟล์ ADMX จะถูกจัดเก็บในรูปแบบไฟล์สากล [XML](/th/web/xml/) ที่มนุษย์สามารถอ่านได้ เป็นรูปแบบไฟล์หลายภาษาและอนุญาตให้ผู้ดูแลระบบจากภาษาต่างๆ ทำงานกับไฟล์ ADMX เดียวกันได้ ไฟล์ ADMX จะแทนที่รูปแบบไฟล์เก่า [ADM](/th/system/adm/) ซึ่งเป็นรูปแบบไฟล์ข้อความในรูปแบบ Unicode ไฟล์ ADMX ระบุคีย์รีจิสทรีที่มีการเปลี่ยนแปลงเมื่อมีการเปลี่ยนแปลงการตั้งค่านโยบายกลุ่ม

### ฉันจะทำอะไรกับไฟล์ ADMX ได้บ้าง

ไฟล์ ADMX กำหนดการกระทำที่ได้รับอนุญาตให้ใช้กับคอมพิวเตอร์ของผู้ใช้ที่เป็นส่วนหนึ่งของกลุ่ม นโยบายกลุ่มกำหนดกฎที่ตั้งไว้ร่วมกับซอฟต์แวร์ Active Directory ไฟล์ ADMX ได้รับการจัดการจากร้านค้ากลางบน Windows OS ซึ่งเป็นตำแหน่งศูนย์กลางในโดเมน

ไฟล์ ADMX ที่ใช้งานในสภาพแวดล้อมการทำงานสามารถให้ผู้ดูแลระบบใช้นโยบายต่างๆ เช่น:

* ควบคุมการใช้อินเทอร์เน็ต
* การดาวน์โหลดไฟล์บางประเภท
* การใช้อุปกรณ์ที่ถอดออกได้บนระบบ
