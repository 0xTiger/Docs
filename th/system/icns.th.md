## ไฟล์ ICNS คืออะไร??

ไฟล์ ICNS เป็นรูปแบบไฟล์ที่ใช้ใน macOS เพื่อจัดเก็บไอคอนสำหรับแอปพลิเคชัน โฟลเดอร์ และรายการอื่นๆ เป็นรูปแบบไฟล์คอนเทนเนอร์ที่สามารถเก็บภาพได้หลายขนาดและความละเอียดของไอคอนเดียวกัน

โดยทั่วไปไฟล์ ICNS จะมีรูปภาพที่มีความละเอียดหลายระดับ ตั้งแต่ 16x16 พิกเซลถึง 1024x1024 พิกเซล เพื่อรองรับขนาดการแสดงผลและความละเอียดที่แตกต่างกันบนอุปกรณ์ macOS รูปแบบไฟล์ใช้อัลกอริธึมการบีบอัดเพื่อลดขนาดไฟล์ของรูปภาพ

ไฟล์ ICNS สามารถสร้างได้โดยใช้เครื่องมือซอฟต์แวร์ต่างๆ เช่น Icon Composer หรือ Sketch และสามารถกำหนดให้กับรายการต่างๆ บนระบบ macOS เช่น แอปพลิเคชัน โฟลเดอร์ และเอกสาร เมื่อเลือกรายการบนระบบ macOS ระบบปฏิบัติการจะดึงขนาดและความละเอียดที่เหมาะสมของไอคอนจากไฟล์ ICNS และแสดงบนหน้าจอ

## รูปแบบไฟล์ ICNS - ข้อมูลเพิ่มเติม

ไฟล์ ICNS เป็นไฟล์คอนเทนเนอร์ที่สามารถเก็บภาพได้หลายขนาดและความละเอียดของไอคอนเดียวกันโดยใช้เทคนิคที่เรียกว่า "ตระกูลไอคอน" ตระกูลไอคอนประกอบด้วยกลุ่มทรัพยากรรูปภาพ โดยแต่ละกลุ่มจะแสดงไอคอนเดียวกันแต่มีขนาดหรือความละเอียดต่างกัน ทรัพยากรรูปภาพแต่ละรายการภายในตระกูลไอคอนจะมีประเภทและรหัสที่ไม่ซ้ำกัน ซึ่งระบุขนาดและความละเอียดของรูปภาพ ขนาดรูปภาพและความละเอียดที่แตกต่างกันจะถูกจัดเก็บไว้ในไฟล์ ICNS เดียวกันโดยใช้ประเภทและ ID ที่แตกต่างกัน

เมื่อเลือกรายการบนระบบ macOS ระบบปฏิบัติการจะดึงขนาดและความละเอียดที่เหมาะสมของไอคอนจากไฟล์ ICNS โดยใช้ประเภทและ ID แล้วแสดงบนหน้าจอ ซึ่งช่วยให้ระบบ macOS สามารถแสดงไอคอนคุณภาพสูงบนหน้าจอที่มีความละเอียดต่างกัน เช่น บนจอแสดงผล Retina โดยที่ยังคงขนาดไฟล์ที่เล็กลงได้

ไฟล์ ICNS บน macOS นั้นคล้ายคลึงกับไฟล์ .ICO บน Windows โดยที่ทั้งคู่ใช้เป็นรูปแบบไฟล์สำหรับจัดเก็บไอคอน เช่นเดียวกับไฟล์ ICNS ไฟล์ ICO สามารถมีขนาดรูปภาพและความละเอียดได้หลายขนาดของไอคอนเดียวกัน ซึ่งใช้เพื่อแสดงไอคอนบนหน้าจอขนาดและความละเอียดที่แตกต่างกันบนอุปกรณ์ Windows

อย่างไรก็ตาม มีความแตกต่างบางประการระหว่างไฟล์ ICNS และ ICO ตัวอย่างเช่น ไฟล์ ICNS สามารถจัดเก็บภาพที่มีความละเอียดสูงกว่าไฟล์ ICO ได้ ซึ่งทำให้เหมาะสำหรับการใช้งานบนจอแสดงผลความละเอียดสูง เช่น จอแสดงผล Retina บนอุปกรณ์ macOS นอกจากนี้ ไฟล์ ICNS สามารถประกอบด้วยรูปภาพที่มีช่องอัลฟา ซึ่งทำให้มีพื้นหลังโปร่งใส ซึ่งไม่รองรับในไฟล์ ICO
