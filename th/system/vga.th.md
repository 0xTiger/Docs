## ไฟล์ VGA คืออะไร?

ไฟล์ VGA เป็นไฟล์ไดรฟ์ Windows เพื่อรองรับจอภาพ VGA มันถูกใช้ใน Microsoft Windows เวอร์ชันก่อนหน้าสำหรับการปรับจอภาพ VGA ที่ระดับระบบปฏิบัติการ (OS) ไฟล์ VGA ที่ใช้เพื่อให้ผู้ใช้สามารถปรับความละเอียดและความลึกของสีของจอแสดงผลโดยใช้แผงควบคุมของจอภาพ

## รูปแบบไฟล์ VGA

รูปแบบไฟล์ภายในของรูปแบบไฟล์ VGA ไม่เป็นที่รู้จักสำหรับการอ้างอิงของนักพัฒนา ระบบปฏิบัติการ Windows อ้างอิงถึงการตั้งค่าลักษณะที่ปรากฏของจอภาพ VGA
