## ไฟล์ CDR คืออะไร??

ไฟล์ **.CDR** ที่มี **"ข้อมูลเสียงซีดีดิบ"** โดยทั่วไปหมายถึงไฟล์ที่มีข้อมูลเสียงที่แยกจากซีดีเพลงโดยตรง เช่น แทร็กเพลง "ข้อมูลเสียงซีดีดิบ" หมายถึงข้อมูลเสียงดิจิทัลที่ยังไม่ได้ประมวลผลจากซีดี ซึ่งมักจะจัดเก็บในรูปแบบไบนารี ข้อมูลดิบนี้มีตัวอย่างเสียงเหมือนกับที่อยู่ในซีดีทุกประการ รวมทั้งข้อมูลเสียงและรหัสย่อย

โดยทั่วไปไฟล์เหล่านี้ไม่ได้มีไว้สำหรับเล่นโดยตรง แต่ใช้เพื่อวัตถุประสงค์ต่างๆ แทน เช่น การเรียนรู้ซีดี การแก้ไขเสียง และการเขียนซีดี แอปพลิเคชันซอฟต์แวร์สำหรับการตัดต่อเสียงและการเขียนซีดีมักใช้ข้อมูลดิบนี้เพื่อสร้างซีดีเพลงหรือแปลงเป็นรูปแบบไฟล์เสียงอื่นๆ เช่น WAV หรือ MP3

หากคุณต้องการทำงานกับไฟล์ .cdr ที่มี "ข้อมูลเสียงซีดีดิบ" โดยทั่วไปคุณจะใช้ซอฟต์แวร์เสียงพิเศษในการประมวลผลหรือแปลงไฟล์เหล่านั้น เนื่องจากไม่ใช่ไฟล์เสียงมาตรฐานที่สามารถเล่นได้ในเครื่องเล่นสื่อส่วนใหญ่

## วิธีแปลงไฟล์ CDR?

หากต้องการแปลงไฟล์ .cdr ที่มี "ข้อมูลเสียงซีดีดิบ" เป็นรูปแบบเสียงทั่วไป เช่น WAV หรือ MP3 คุณจะต้องมีซอฟต์แวร์พิเศษ หากคุณสามารถเล่นไฟล์ .cdr ในเครื่องเล่นสื่อ VideoLAN VLC ได้ คุณยังสามารถใช้ VLC เพื่อแปลงไฟล์เป็นรูปแบบเสียงต่างๆ ได้ เช่น

- [.MP3](/th/audio/mp3/) - เสียง MP3
- [.FLAC](/th/audio/flac/) - ฟรีตัวแปลงสัญญาณเสียงแบบไม่สูญเสีย
- [.WAV](/th/audio/wav/) - เสียง WAVE
- [.OGG](/th/audio/ogg/) - เสียง Ogg Vorbis

## เปิดไฟล์ CDR ได้อย่างไร

การเปิดไฟล์ .cdr โดยเฉพาะไฟล์ที่มี "Raw CD Audio Data" อาจยุ่งยากเล็กน้อยเนื่องจากไม่ใช่รูปแบบไฟล์เสียงมาตรฐาน หากต้องการเปิดและทำงานกับไฟล์ดังกล่าว คุณสามารถใช้ซอฟต์แวร์พิเศษที่สามารถจัดการข้อมูลเสียงดิบของซีดีได้ นี่คือวิธีการและตัวเลือกซอฟต์แวร์บางตัว:

**ซอฟต์แวร์สำหรับเปิดและทำงานกับไฟล์ .cdr:**

1. **สำเนาเสียงที่แน่นอน (EAC)**:
    





- ดาวน์โหลดและติดตั้ง Exact Audio Copy ซึ่งเป็นเครื่องมือยอดนิยมและมีประสิทธิภาพสำหรับการทำงานกับซีดีเพลงและแยก "Raw CD Audio Data"
- ใส่ซีดีที่มีไฟล์ .cdr ลงในไดรฟ์ซีดี/ดีวีดีของคอมพิวเตอร์ของคุณ
- เปิด EAC และกำหนดค่าตามความต้องการของคุณ รวมถึงรูปแบบเอาต์พุต (เช่น WAV หรือ MP3)
- ใช้ EAC เพื่อแยกแทร็กเสียงจากซีดีและบันทึกในรูปแบบที่ต้องการ
2. **ความกล้า**:
    





- Audacity เป็นซอฟต์แวร์แก้ไขเสียงโอเพ่นซอร์สฟรีที่สามารถทำงานร่วมกับ "Raw CD Audio Data" ได้
- ใส่แผ่นซีดี
- เปิด Audacity และกำหนดค่าให้บันทึกเสียงจากไดรฟ์ซีดีของคุณ
- บันทึกเสียงจากซีดีแล้วบันทึกในรูปแบบเสียงทั่วไป
3. **VLC มีเดียเพลเยอร์**:
    





- VLC สามารถเล่นเสียงได้โดยตรงจากซีดี รวมถึงแผ่นที่มี "Raw CD Audio Data"
- ใส่แผ่นซีดีลงในคอมพิวเตอร์ของคุณ
- เปิด VLC ไปที่เมนู "สื่อ" และเลือก "เปิดดิสก์"
- เลือกไดรฟ์ซีดีและเล่นแทร็กเสียงได้โดยตรง
4. **ซอฟต์แวร์ CD Ripper**:
    





- มีตัวเลือกซอฟต์แวร์ CD ripper ให้เลือกมากมาย เช่น Windows Media Player (สำหรับ Windows), iTunes (สำหรับ Mac) และอื่นๆ
- ใส่แผ่นซีดีลงในคอมพิวเตอร์ของคุณ
- เปิดซอฟต์แวร์ CD Ripper ที่คุณต้องการ และใช้เพื่อแยกแทร็กเสียงและบันทึกในรูปแบบเสียงทั่วไป
5. **เครื่องมือ DAEMON** (สำหรับไฟล์ภาพ .cdr):
    





- หากไฟล์ .cdr เป็นไฟล์รูปภาพที่สร้างโดยซอฟต์แวร์ เช่น DAEMON Tools คุณสามารถเมานต์รูปภาพโดยใช้ DAEMON Tools หรือซอฟต์แวร์ที่คล้ายกัน
- เมื่อติดตั้งรูปภาพแล้ว คุณสามารถเข้าถึงเนื้อหาได้เสมือนเป็นซีดีจริง และเล่นหรือแยกเสียงได้ตามต้องการ
