## ไฟล์ PY คืออะไร?? ##

ไฟล์ที่มีนามสกุล .py มีซอร์สโค้ดของ Python ภาษา Python กลายเป็นภาษาที่มีชื่อเสียงมากในปัจจุบัน สามารถใช้สำหรับการเขียนสคริปต์ระบบ การพัฒนาเว็บและซอฟต์แวร์ และคณิตศาสตร์ Python รองรับความเข้ากันได้ข้ามแพลตฟอร์ม หมายความว่าแอปพลิเคชันที่พัฒนาขึ้นใน Python สามารถทำงานได้บนแพลตฟอร์มต่างๆ เช่น Windows, MAC, Linux, Raspberry Pi เป็นต้น Python มีไวยากรณ์ที่เรียบง่ายและอ่านง่ายซึ่งคล้ายกับภาษาอังกฤษ นักพัฒนาสามารถเขียนแอปพลิเคชันซอฟต์แวร์ที่เหมาะสมโดยเขียนโค้ด Python สองสามบรรทัด เนื่องจาก Python ทำงานบนระบบล่าม ดังนั้นจึงสามารถดำเนินการโค้ดได้ทันทีที่เขียน ซึ่งช่วยให้สามารถสร้างต้นแบบได้ดีมาก

## ประวัติย่อ ##

Python กำเนิดขึ้นในช่วงปลายทศวรรษ 1980 โดย Guido van Rossum ในฐานะผู้สืบทอดภาษาโปรแกรม ABC เริ่มดำเนินการในเดือนธันวาคม พ.ศ. 2532 โดยมี Van Rossum เป็นผู้พัฒนาหลักแต่เพียงผู้เดียว เขาทำงานกับ Python จนถึงวันที่ 12 กรกฎาคม 2018 ในเดือนมกราคม 2019 นักพัฒนาหลักของ Python ที่ใช้งานอยู่ได้เลือกสมาชิก "Steering Council" ห้าคนซึ่งประกอบด้วย Nick Coghlan, Brett Cannon, Carol Willing, Barry Warsaw และ Van Rossum เพื่อเป็นผู้นำโครงการ

### รุ่น ###

- Python 2.0 เปิดตัวเมื่อวันที่ 16 ตุลาคม พ.ศ. 2543
- Python 3.0 เปิดตัวเมื่อวันที่ 3 ธันวาคม 2551

## วิธีเรียกใช้ไฟล์ py ##

หากต้องการตรวจสอบเวอร์ชันของ Python ที่ติดตั้ง คุณสามารถใช้คำสั่งต่อไปนี้:

```cmd
python --version
```

สิ่งนี้จะส่งออกเวอร์ชันบนคอนโซลเช่น

```cmd
Python 3.7.4
```

หากเครื่องของคุณไม่ได้ติดตั้ง Python ให้ไปที่ [python.org](https://www.python.org/) แล้วดาวน์โหลดและติดตั้ง Python สำหรับระบบปฏิบัติการที่เกี่ยวข้อง

ในการเรียกใช้สคริปต์ Python คุณสามารถใช้คำสั่งต่อไปนี้:

```cmd
python helloworld.py
```

*helloworld.py* เป็นไฟล์สคริปต์ที่มีรหัสต่อไปนี้

```py
print("Hello World from Python")
```

สิ่งนี้จะพิมพ์สิ่งต่อไปนี้บนหน้าต่างคอนโซล

```cmd
Hello World from Python
```

หมายเหตุ: หากคุณใช้ IDE จะมีปุ่มบนหน้าจอหรือแป้นพิมพ์ลัดต่างๆ เพื่อเรียกใช้ Python ตัวอย่างเช่น ปุ่มเล่นจะแสดงในรางน้ำพร้อมกับตัวแก้ไขใน PyCharm ที่ให้คุณเรียกใช้สคริปต์ Python

## รูปแบบไฟล์ PY ##

Python ได้รับการออกแบบมาให้อ่านง่ายและมีความคล้ายคลึงกับภาษาอังกฤษและคณิตศาสตร์ Python ใช้การขึ้นบรรทัดใหม่เพื่อระบุคำสั่งที่สมบูรณ์ซึ่งตรงข้ามกับเครื่องหมายอัฒภาคหรือวงเล็บที่ใช้ในภาษาอื่น สำหรับขอบเขต ลูป และฟังก์ชัน Python อาศัยการเยื้องและการเว้นวรรค ซึ่งต่างจากวงเล็บปีกกาที่ใช้ในภาษาอื่น

### ไวยากรณ์ ###

ต่อไปนี้เป็นตัวอย่างของไวยากรณ์ Python

```py
print("Hello World")

#Variables
name = "John"
age = 25
print(name)
print(age)

#While Loop
i = 1
while i < 3:
  print(i)
  i += 1
  
#For Loop
names = ["John", "Harry", "Tom"]
for name in names:
  print(name)
```
