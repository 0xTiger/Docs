## ไฟล์ RBXM คืออะไร??

ไฟล์ RBXM เป็นรูปแบบไฟล์โมเดล 3 มิติที่ใช้ใน Roblox Studio IDE สำหรับจัดเก็บวัตถุ 3 มิติของเกม Roblox แบบผู้เล่นหลายคนออนไลน์ ประกอบด้วยวัตถุ 3 มิติที่สร้างขึ้นสำหรับเกมโดยใช้ IDE ตัวอย่างของวัตถุ 3 มิติในไฟล์ RBXM ได้แก่ เสื้อ กางเกง หมวก อาคาร ยานพาหนะ สัตว์ เครื่องบิน รถยนต์ หรือพืช ไฟล์ RBMX สามารถแปลงเป็น [รูปแบบไฟล์ OBJ 3D](/th/3d/obj/) โดยใช้ Roblox IDE Roblox IDE สามารถใช้เปิดไฟล์ RBMX

รูปแบบไฟล์ Roblox ที่ได้รับความนิยมอีกรูปแบบหนึ่งคือ [RBXL](/th/programming/rbxl/) ซึ่งเป็นรูปแบบไฟล์ระบุตำแหน่งและมีตำแหน่งที่กำหนดด้วยคำ 3 มิติภายในเกม Roblox

## รูปแบบไฟล์ RBMX - ข้อมูลเพิ่มเติม

ไฟล์ RBXM ถูกสร้างขึ้นและบันทึกในรูปแบบไฟล์ไบนารีและข้อกำหนดรูปแบบไฟล์ภายในจะไม่เปิดเผยต่อสาธารณะ ซึ่งแตกต่างจากรูปแบบไฟล์ RBXMX ที่ยึดตามรูปแบบไฟล์ [XML](/th/web/xml/) มีเครื่องมือ/พื้นที่เก็บข้อมูลมากมายบน Github ที่สามารถใช้เพื่อ [ประมวลผล Roblox RBXM](https://github.com/search?q=rbxm) ไฟล์
