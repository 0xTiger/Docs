## ไฟล์ CMAKE คืออะไร??

ไฟล์ CMAKE เป็นไฟล์โครงการที่สร้างโดยเครื่องมือโอเพ่นซอร์สข้ามแพลตฟอร์ม **CMake** มีซอร์สสคริปต์ในภาษา CMake สำหรับทั้งโปรแกรม เครื่องมือ CMake ใช้สคริปต์การเขียนโปรแกรมที่เรียกว่า CMakeLists เพื่อสร้างไฟล์บิลด์สำหรับสภาพแวดล้อมเฉพาะ เช่น makefiles เครื่อง Unix สคริปต์และโมดูลที่ใช้โดยโครงการ CMake ถูกจัดระเบียบเป็น script.cmake และ module.cmake ที่อ้างอิงในโครงการ
