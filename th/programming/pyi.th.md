## ไฟล์ PYI คืออะไร??

ไฟล์ PYI เป็นไฟล์ Python Interface Definition ที่มีการอ้างอิงโค้ดสำหรับการดำเนินการของอินเทอร์เฟซ โมดูล Python แต่ละโมดูลจะแสดงเป็น .pyi stub ซึ่งเป็นไฟล์ Python ปกติแต่มีเมธอดว่างเปล่า ไวยากรณ์ของไฟล์ PYI เหมือนกับของโมดูล Python ปกติ การใช้งานเมธอดว่างนั้นถูกปล่อยให้ผู้ใช้ปลายทางบรรลุวัตถุประสงค์เฉพาะที่เขียนโมดูล นั่นคือจุดที่ไฟล์ PYI แตกต่างจากไฟล์ [PY](/th/programming/py/) ซึ่งมีการใช้งานโปรแกรมอย่างสมบูรณ์ ไฟล์ PYI สามารถเปิดได้ด้วยโปรแกรมแก้ไขข้อความ เช่น Notepad, Notepad++, Microsoft Visual Studio Code และ JetBrains PyCharm

## รูปแบบไฟล์ PYI

ไฟล์ PYI ถูกบันทึกเป็นไฟล์ข้อความล้วนที่สามารถเปิดได้ด้วยโปรแกรมแก้ไขข้อความ Python เป็นภาษาล่ามที่ทำการตรวจสอบประเภทเมื่อมีการเรียกใช้รหัส ในกรณีเช่นนี้ ไฟล์ PYI มีประโยชน์ในด้านที่นักพัฒนาสามารถกำหนดและตรวจสอบประเภทของโมดูลได้ด้วยตนเองในขณะที่เขียนแอปพลิเคชัน
