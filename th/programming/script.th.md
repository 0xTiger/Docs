## ไฟล์ SCRIPT คืออะไร??

ไฟล์ SCRIPT เป็นไฟล์ต้นฉบับที่มีสคริปต์ในภาษาสคริปต์ใดๆ ทำให้เป็นไฟล์สคริปต์ทั่วไปที่มีตัวแปรและฟังก์ชัน เนื่องจากภาษาสคริปต์แต่ละภาษาอาจแตกต่างกัน สคริปต์ที่เขียนภายในไฟล์ SCRIPT จึงจำเป็นต้องดำเนินการโดยใช้เครื่องมือสคริปต์ที่เหมาะสม

## รูปแบบไฟล์ SCRIPT - ข้อมูลเพิ่มเติม

ไฟล์ SCRIPT จะถูกจัดเก็บไว้ในดิสก์ในรูปแบบไฟล์ข้อความธรรมดา สามารถเปิดและแก้ไขได้ด้วยโปรแกรมแก้ไขข้อความใดๆ เช่น Microsoft Notepad, Notepad++ และ Apple TextEdit เนื่องจากภาษาสคริปต์อาจแตกต่างกัน จึงดำเนินการได้โดยเปลี่ยนชื่อนามสกุลไฟล์เป็นนามสกุลเริ่มต้นของภาษาสคริปต์ที่เกี่ยวข้อง ตัวอย่างเช่น มีภาษาสคริปต์ทั่วไปหลายภาษา เช่น JavaScript ([.JS](/th/web/js/)), VBScript ([.VBS](/th/programming/vbs/)), AppleScript (.SCPT), PHP ([ .PHP](/th/programming/php/)) และ ASP ([.ASP](/th/web/asp/))
