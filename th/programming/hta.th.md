## ไฟล์ HTA คืออะไร??

HTMLA ย่อมาจาก **Hypertext Markup Language Application** เป็นโปรแกรมที่เข้ากันได้กับ Microsoft Windows ซอร์สโค้ดของโปรแกรมนี้มีภาษาสคริปต์มากกว่าหนึ่งภาษา เช่น [HTML](/th/web/html/) และ [JavaScript](/th/web/js/) สำหรับส่วนต่อประสานกับผู้ใช้ แอปพลิเคชัน HTML เป็นที่ต้องการในขณะที่ตอบสนองความต้องการของตรรกะของโปรแกรมไม่ว่าจะใช้ภาษาสคริปต์อื่น ๆ

แอ็พพลิเคชัน HTML เป็นอิสระจากโมเดลการรักษาความปลอดภัยของอินเทอร์เน็ตเบราว์เซอร์ และรันเป็นแอ็พพลิเคชันที่เชื่อถือได้อย่างสมบูรณ์ นามสกุลที่ใช้สำหรับไฟล์เกี่ยวกับแอปพลิเคชันเหล่านี้คือ HTA แอปพลิเคชันเหล่านี้ประกอบด้วยคุณลักษณะของ HTML พร้อมกับคุณสมบัติของภาษาสคริปต์อื่นๆ


## ประวัติย่อ ##

HTA เปิดตัวครั้งแรกในปี 1999 โดย Microsoft พร้อมกับการเปิดตัว Internet Explorer 5 ซึ่งเข้ากันได้กับ Internet Explorer ดังนั้นจึงสามารถดำเนินการได้บนระบบปฏิบัติการ Windows เท่านั้น เทคโนโลยีนี้ได้รับการจดสิทธิบัตรในปี 2546 ไฟล์ HTA นั้นทำงานคล้ายกับไฟล์ .exe อื่นๆ ไฟล์ HTA เข้ากันได้กับ Windows 11 เวอร์ชันอัปเดตในปัจจุบันเช่นกัน


## ข้อมูลจำเพาะทางเทคนิค ##

HTA มีรูปแบบเดียวกับหน้า HTML อื่นๆ ในขณะที่แอตทริบิวต์บางอย่างใช้สำหรับควบคุมสไตล์ของเส้นขอบหรือไอคอนของโปรแกรม นอกจากนี้ ยังมีการให้ข้อโต้แย้งสำหรับการเปิดตัว HTA แอปพลิเคชันเหล่านี้สามารถดำเนินการได้โดยใช้โปรแกรมชื่อ mshta.exe สามารถเข้าถึงได้โดยดับเบิลคลิกที่ไฟล์ โปรแกรมเหล่านี้จะทำงานร่วมกับ Internet explorer โดยอัตโนมัติ นอกจากข้อกำหนดอื่นๆ แล้ว สิ่งเหล่านี้ไม่ได้ขึ้นกับเบราว์เซอร์เอนจิ้น Trident แต่ไม่ขึ้นกับ Internet Explorer หมายความว่าสิ่งเหล่านี้สามารถดำเนินการได้โดยไม่ต้องใช้ Internet Explorer

แท็กใช้เพื่อปรับแต่งลักษณะที่ปรากฏของแอปพลิเคชันเหล่านี้ การแปลงจากแอปพลิเคชัน Microsoft HTML เป็นรูปแบบ HTA นั้นง่ายกว่า โดยคุณจะต้องเปลี่ยนนามสกุลเท่านั้น ดังที่เราทราบดีว่าแอปพลิเคชันเหล่านี้ได้รับความไว้วางใจอย่างเต็มที่ ดังนั้นแอปพลิเคชันเหล่านี้จึงมีคุณลักษณะและข้อดีมากกว่าเมื่อเทียบกับไฟล์ HTML แบบธรรมดา สามารถใช้โปรแกรมแก้ไขข้อความเพื่อสร้าง HTA ตัวแก้ไขเหล่านี้สามารถได้รับจาก Microsoft หรือแหล่งที่เชื่อถือได้อื่นๆ


## ตัวอย่างรูปแบบไฟล์ HTA ##

```
<HTML>
<HEAD>
<HTA:APPLICATION ID="HelloExample" 
   BORDER="bold" 
   BORDERSTYLE="complex"/>
<TITLE>HTA - Hello World</TITLE>
</HEAD>
<BODY>
<H2>HTA - Hello World</H2>
</BODY>
</HTML>

```
