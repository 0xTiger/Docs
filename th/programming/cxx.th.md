## ไฟล์ CXX คืออะไร??

ไฟล์ CXX เป็นไฟล์ซอร์สโค้ด คล้ายกับ C++ สำหรับเขียนแอปพลิเคชัน อาจมีการใช้งานแอปพลิเคชันทั้งหมดอย่างสมบูรณ์หรืออาจมีการใช้งานโปรแกรมบางส่วนที่สามารถรวมอยู่ในโครงการด้วยไฟล์ C ++ อื่น ๆ ไฟล์ CXX สามารถเปิดได้ในโปรแกรมแก้ไขข้อความและ IDE โปรแกรมยอดนิยม เช่น Notepad++, Turbo C++ และ Visual Studio

## รูปแบบไฟล์ CXX

ไฟล์ CXX ถูกจัดเก็บเป็นไฟล์ข้อความธรรมดาและมีโค้ดที่เขียนด้วยไวยากรณ์ C++ โดยทั่วไปจะพบไฟล์ CXX ที่มีนามสกุล [.cpp](/th/programming/cpp/) เช่นกัน ไฟล์ CXX ถือเป็นทางเลือกแทน .cpp ที่ใช้โดยคอมไพเลอร์บางตัว
