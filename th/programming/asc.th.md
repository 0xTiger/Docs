## ไฟล์ ASC คืออะไร??

ไฟล์ ASC เป็นไฟล์ **script** ที่สร้างขึ้นด้วยภาษาโปรแกรม ActionScript ประกอบด้วยโค้ดที่ทำงานบนซอฟต์แวร์โฮสติ้ง Flash Media Server ฝั่งเซิร์ฟเวอร์ ไฟล์ ASC มีสคริปต์ที่ใช้ในการควบคุมเหตุการณ์ของแอปพลิเคชัน เช่น การโหลดแอปพลิเคชัน Flash การจัดการการกระทำของผู้ใช้จากเหตุการณ์เมาส์และแป้นพิมพ์ เล่นเนื้อหามัลติมีเดีย และแสดงเอาต์พุตของแอปพลิเคชัน Adobe เข้าซื้อกิจการ Macromedia Flash ซึ่งต่อมาได้กลายเป็นส่วนหนึ่งของ Adobe Creativity suit ในชื่อ Adobe Flash ใช้เพื่อบันทึกไฟล์ภาพเคลื่อนไหวใน [รูปแบบไฟล์ .swf](/th/page-description-language/swf/)

## รูปแบบไฟล์ ASC - ข้อมูลเพิ่มเติม

ไฟล์ ASC จะบันทึกเป็นไฟล์ข้อความที่สามารถเปิดและแก้ไขได้ด้วย [Adobe Media Encoder](https://www.adobe.com/products/media-encoder.html) ไฟล์เหล่านี้ถูกสร้างและบันทึกเป็นไฟล์ข้อความ และสามารถเปิดได้ด้วยโปรแกรมแก้ไขข้อความใดๆ เช่น Microsoft Notepad, Notepad++ และ Apple TextEdit ไฟล์ ASC จะถูกดำเนินการตามค่าเริ่มต้นเมื่อเรียกใช้ Flash Media Server และแอปพลิเคชัน
