## ไฟล์ NX คืออะไร??

ไฟล์ NX เป็นไฟล์วิดีโอเกมที่สร้างขึ้นเพื่อใช้กับคอนโซลเกมเสมือน **LowRes NX** เขียนด้วยภาษาโปรแกรม BASIC และมีซอร์สโค้ดของเกมที่กำหนดวิธีการทำงานของเกม คุณสามารถสร้างไฟล์ NX ของคุณเองเพื่อใช้กับแอปที่สามารถโหลดได้และผู้ใช้สามารถเล่นเกมได้

คุณสามารถเปิดไฟล์ NX ในโปรแกรมแก้ไขข้อความ, LowRes NX หรือโปรแกรมแก้ไขซอร์สโค้ดใดๆ เช่น Microsoft Visual Studio Code

## รูปแบบไฟล์ NX

ไฟล์ NX จะถูกบันทึกลงแผ่นดิสก์เป็นไฟล์ข้อความธรรมดาที่มีคำแนะนำเกม ไฟล์ NX แต่ละไฟล์มีซอร์สโค้ดของเกมที่ประกอบด้วยรายการ ROM คาร์ทริดจ์หลายรายการ รายการ ROM เหล่านี้มีข้อมูลส่วนประกอบของเกม เช่น กราฟิก แผนที่ระดับ เสียง และข้อมูลโค้ดที่ไม่ใช่ซอร์สโค้ดอื่นๆ สำหรับเกม รายการ ROM ของโปรแกรมสามารถเพิ่มหรือแก้ไขได้โดยใช้โปรแกรมแก้ไข NX ที่มาพร้อมกับ LowRes NX

## จะสร้างไฟล์ NX ได้อย่างไร

### สร้างไฟล์ NX บนพีซี

คุณสามารถสร้างไฟล์ NX บนพีซีได้โดยใช้:

* โปรแกรมแก้ไขข้อความ - เช่น Microsoft Notepad, Notepad++ หรือ Github Atom
* ตัวแก้ไขซอร์สโค้ด - เช่น Microsoft VIsual Studio Code

### สร้างไฟล์ NX บน iOS

LowRes NX มีเวอร์ชัน iOS ที่นักพัฒนาสามารถใช้เพื่อสร้างไฟล์ NX จากภายในแอปนี้ได้
