## ไฟล์ PHP คืออะไร?
ไฟล์ที่มีนามสกุล .php หมายถึงภาษาโปรแกรมแบบโอเพ่นซอร์ส ซึ่งใช้ในการเขียนสคริปต์ฝั่งเซิร์ฟเวอร์ เพื่อเรียกใช้งานบนเว็บเซิร์ฟเวอร์ เป็นภาษาเว็บสคริปต์ที่ใช้กันแพร่หลายมากที่สุด ซึ่งโดยทั่วไปจะใช้สำหรับการพัฒนาเว็บแอปพลิเคชันขนาดใหญ่ ระบบบล็อกที่ใหญ่ที่สุดบนเว็บ เช่น WordPress และโซเชียลเน็ตเวิร์กที่ใหญ่ที่สุดบน Facebook นั้นใช้ภาษา PHP มันเข้ากันได้กับแพลตฟอร์มต่าง ๆ เช่น Windows, Linux, Unix, Mac OS X และอื่น ๆ นอกจากนี้ยังสามารถทำงานบนเซิร์ฟเวอร์เกือบทั้งหมดที่ใช้ในปัจจุบันเช่น Apache, IIS และอื่น ๆ PHP รองรับฐานข้อมูลที่หลากหลายรวมถึง MySQL

## รูปแบบไฟล์ PHP ##

PHP เดียวสามารถประกอบด้วยข้อความ, CSS, [HTML](/th/web/html/), JavaScript และโค้ด PHP โค้ด PHP ถูกเรียกใช้บนเซิร์ฟเวอร์ และผลลัพธ์จะถูกส่งกลับไปยังเบราว์เซอร์เป็น HTML ธรรมดา ไฟล์ PHP สามารถเปิดได้ด้วยโปรแกรมแก้ไขข้อความและแก้ไขในสถานที่ แม้ว่าแอปพลิเคชันอย่าง Adobe Dreamweaver เครื่องมือพัฒนา PHP ของ Eclipse จะมอบวิธีที่สะดวกในการเขียนและแก้ไขโค้ด PHP

### ไวยากรณ์ ###

สคริปต์ PHP ดำเนินการบนเซิร์ฟเวอร์ และผลลัพธ์ HTML ธรรมดาจะถูกส่งกลับไปยังเบราว์เซอร์ สคริปต์ PHP เริ่มต้นด้วย<?php and ends with ?>

### ตัวอย่าง PHP ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
