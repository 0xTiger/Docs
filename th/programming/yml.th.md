## ไฟล์ YML คืออะไร??

YML คือไฟล์ข้อความในภาษาการทำให้เป็นอันดับของข้อมูล [YAML](/th/programming/yaml/) (Yet Another Markup Language) ซึ่งมักใช้ในการสร้างไฟล์การกำหนดค่า สามารถทำงานร่วมกับภาษาโปรแกรมต่างๆ รวมถึง C/C++, Ruby, Python, Java, PHP, C# และอื่นๆ เป็นอิสระจากภาษาโปรแกรมใด ๆ ภาษาเปิด ทำงานร่วมกันได้ และมนุษย์สามารถอ่านได้ ไฟล์ YML สามารถเปิดและแก้ไขได้ด้วยโปรแกรมแก้ไขข้อความทั่วไป เช่น Microsoft Notepad, Notepad++, Github Atom, Apple TextEdit และ Visual Studio IDE

## รูปแบบไฟล์ YML - ข้อมูลเพิ่มเติม

ไฟล์ YML เก็บการตั้งค่าแอปพลิเคชันเป็นการกำหนดค่า โครงสร้างพื้นฐานคือแผนที่ที่สามารถเรียกว่าไดเร็กทอรีหรือแฮช รายการในไฟล์ YML ประกอบด้วยคู่ของคีย์และค่า ส่วนใหญ่ ไฟล์การกำหนดค่า .yml จะบันทึกเป็น `configure.yml` แม้ว่าภาษาโปรแกรมบางภาษา เช่น Ruby on Rails จะบันทึกการกำหนดค่าการเชื่อมต่อฐานข้อมูลในไฟล์ `database.yml` [ข้อกำหนดรูปแบบไฟล์ YML/YAML](https://yaml.org/spec/1.2.2/) อธิบายโครงสร้างภาษาทั้งหมดและข้อมูลที่เกี่ยวข้องกับไวยากรณ์
