## ไฟล์ GROOVY คืออะไร??

ไฟล์ GROOVY เป็นไฟล์ซอร์สโค้ดที่เขียนด้วยภาษาโปรแกรม Groovy โค้ดที่เขียนภายในไฟล์ GROOVY คล้ายกับภาษาเชิงวัตถุ [Java](/th/programming/java/) ซึ่งทำให้การออกแบบและพัฒนาแอปพลิเคชันเป็นเรื่องง่าย ไฟล์ GROOVY ถูกคอมไพล์เป็น Java virtual machine (JVM) bytecode และเข้ากันได้กับโค้ด Java และไลบรารี่อื่นๆ

## รูปแบบไฟล์ GROOVY - ข้อมูลเพิ่มเติม

ไฟล์ GROOVY มีซอร์สโค้ดที่เขียนด้วยไวยากรณ์ของ Groovy สามารถบรรจุรหัสที่สามารถใช้เป็นทั้งภาษาโปรแกรมและภาษาสคริปต์สำหรับแพลตฟอร์ม Java ภาษา Groovy มีคุณสมบัติคล้ายกับ Python, Ruby และ Smalltalk
