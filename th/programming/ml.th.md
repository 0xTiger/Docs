## ไฟล์ ML คืออะไร??

ไฟล์ ML เป็นไฟล์ข้อความที่สร้างขึ้นด้วยภาษาการเขียนโปรแกรมเชิงฟังก์ชันเพื่อวัตถุประสงค์ทั่วไป ML (มักเรียกว่าภาษา Meta) ประกอบด้วยซอร์สโค้ด ML ซึ่งมักใช้ในการวิจัยและออกแบบภาษาโปรแกรมโดยใช้ความหมายที่กำหนดไว้อย่างสมบูรณ์ มี ML หลายรูปแบบที่มี Caml เป็นรูปแบบที่ใช้กันอย่างแพร่หลาย อื่นๆ ได้แก่ Objective Caml (OCaml), JoCaml, Extended ML และ MacroML ภาษา ML มักใช้สำหรับการเขียนคอมไพเลอร์ การพิสูจน์ทฤษฎีบทอัตโนมัติ และการตรวจสอบอย่างเป็นทางการ
