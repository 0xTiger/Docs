## ไฟล์ ARSC คืออะไร??

ไฟล์ ARSC เป็นไฟล์ตารางทรัพยากร Android ที่มีรายการทรัพยากรของแอปพลิเคชันในรูปแบบตาราง ข้อมูลนี้ประกอบด้วยข้อมูล เช่น ชื่อทรัพยากร คุณสมบัติ และรหัส ไฟล์ ARSC เป็นส่วนหนึ่งของแพ็คเกจ APK ที่ใช้ในการติดตั้งแอพ Android เหล่านี้ ทรัพยากรทั้งหมด เช่น รูปภาพ เลย์เอาต์ สไตล์ และสตริง ในแอปพลิเคชัน Android จะถูกแปลงเป็นไฟล์ไบนารี่เมื่อไฟล์ APK ถูกสร้างขึ้น ไฟล์ ARSC ยังถูกสร้างขึ้นพร้อมกัน ซึ่งมีรายการของทรัพยากรที่คอมไพล์แล้วของโปรแกรมทั้งหมดและ ID ของมัน

## รูปแบบไฟล์ ARSC

ไฟล์นามสกุล .arsc เป็นไฟล์ไบนารีที่สร้างขึ้นหลังจากคอมไพเลอร์ เช่น ANT (Eclipse) หรือ Gradle (AS) คอมไพล์โค้ดแอปพลิเคชัน Android เพื่อสร้างไฟล์ [APK](/th/compression/apk/) คุณสามารถแยกไฟล์ APK โดยใช้ยูทิลิตีการเก็บถาวรใดๆ เช่น WinZip เพื่อดูเนื้อหา ซึ่งเป็นไฟล์คลาส java ที่คอมไพล์แล้ว เช่น class.dex นอกจากนี้ ยังมีไฟล์ ARSC นี้ซึ่งมีข้อมูลเมตาเกี่ยวกับ:

* โหนด XML
* คุณสมบัติ
* รหัสทรัพยากร

ทรัพยากรในไฟล์ APK ถูกอ้างอิงโดย Resource ID ในขณะที่แอตทริบิวต์ได้รับการแก้ไขเป็นค่าที่รันไทม์ เครื่องมือ aapt ของ Android รวบรวมทรัพยากรทั้งหมดที่กำหนดไว้ในไฟล์ ณ เวลาที่สร้างและกำหนด ID ทรัพยากรให้กับพวกเขา หลังจากกำหนดตัวระบุให้กับรีซอร์สที่คอมไพล์แล้ว ไฟล์ R.java จะถูกสร้างขึ้นจากซอร์สโค้ดและรีซอร์ส.arsc
