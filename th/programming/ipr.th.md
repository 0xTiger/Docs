## ไฟล์ IPR คืออะไร??

ไฟล์ IPR เป็นไฟล์โครงการที่สร้างด้วย Java IDE **IntelliJ IDEA** ข้อมูลทั้งหมดที่เกี่ยวข้องกับการตั้งค่าโปรเจ็กต์ เช่น โครงสร้างไดเร็กทอรี ไฟล์ซอร์สโค้ด ไลบรารีที่รวมอยู่ในโปรเจ็กต์ และการตั้งค่าคอมไพเลอร์จะถูกจัดเก็บไว้ในไฟล์ IPR เมื่อเปิดโครงการจากภายใน IntelliJ IDEA IDE จะอ่านข้อมูลในไฟล์ IPR และโหลดไฟล์และการตั้งค่าที่เกี่ยวข้อง ไฟล์ IPR ถูกแทนที่ด้วยไฟล์ .IDEA ที่มีข้อมูลทั้งหมดเป็น Settings Directory

## รูปแบบไฟล์ IPR

ไฟล์ IPR จะถูกบันทึกในรูปแบบไฟล์ [XML](/th/web/xml/) ที่เข้าใจได้ในระดับสากล ซึ่งมนุษย์สามารถอ่านได้ ประกอบด้วยข้อมูลที่เกี่ยวข้องกับโครงการทั้งหมดในแท็กที่โหลดเพื่อแสดงเนื้อหาที่เกี่ยวข้องกับโครงการ

เนื่องจากไฟล์ IPR ถูกจัดเก็บในรูปแบบไฟล์ XML จึงสามารถเปิดไฟล์เหล่านี้ได้ในโปรแกรมแก้ไขข้อความใดๆ เช่น Notepad, Notepad++, Atom และ Apple TextEdit
