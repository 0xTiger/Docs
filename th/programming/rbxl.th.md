## ไฟล์ RBXL คืออะไร??

ไฟล์ RBXL เป็นไฟล์ตำแหน่ง Roblox ที่มีตำแหน่งที่กำหนดไว้ในโลก 3 มิติภายใน [เกม Roblox] (https://www.roblox.com/) Roblox เป็นเกมสร้างผู้เล่นหลายคนที่อนุญาตให้ผู้เล่นสร้างเกมโดยใช้เครื่องมือสำหรับนักพัฒนาที่แพลตฟอร์มมีให้ ไฟล์ RBXL ใช้เครื่องมือ สคริปต์ ทีม และส่วนต่างๆ เพื่ออธิบายสนามเด็กเล่นเป็นโมเดล 3 มิติ Roblox Studio ให้ผู้เล่นสร้างไอเท็มเกมและสามารถใช้เปิดไฟล์ RBXL ได้

## รูปแบบไฟล์ RBXL

ไฟล์ RBXL ได้รับการพัฒนาโดยใช้ Roblox Studio และบันทึกในรูปแบบไฟล์ [XML](/th/web/xml/) ด้วยรูปแบบไฟล์ที่เป็นสากลนี้ จึงมีเครื่องมือและ API มากมายที่ให้คุณสร้างและเปิดไฟล์เหล่านี้ได้ API หนึ่งดังกล่าวคือ [MaximumADHD](https://github.com/MaximumADHD/Roblox-File-Format) ซึ่งเป็นไลบรารี [C#](/th/programming/cs/) เพื่อสร้างและจัดการไฟล์ในรูปแบบไฟล์โมเดล/สถานที่ของ Roblox .
