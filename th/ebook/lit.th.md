## ไฟล์ LIT คืออะไร??

ไฟล์ LIT เป็นของ **Microsoft Reader** โบราณซึ่งใช้ในการอ่าน e-book eBooks ที่แสดงโดย Microsoft Reader มีนามสกุล **.lit** ซึ่งย่อเป็น **Literature** รูปแบบไฟล์ LIT ประกอบด้วยไฟล์ Microsoft ITOL/ITLS HTML Help 2.0 ซึ่งมีคุณสมบัติที่น่าทึ่งทุกประเภท เช่น การบีบอัด [LZX](/th/compression/lzx/) การแทนค่าแบบไบนารีของวัสดุมาร์กอัป และข้อมูลเสริมเพิ่มเติมโดยพลการที่เกี่ยวข้องกับ แต่ละสตรีม ข้อความในหนังสือเป็นจำนวนสตรีมมาร์กอัป OEBPS 1.0 และชุดย่อยของ OEBPS 1.0 CSS ตามอำเภอใจ รูปภาพคือ [GIF](/th/image/gif/), [JPEG](/th/image/jpeg/) และ [PNG](/th/image/png/)

## เวลาที่ Microsoft Reader แสดงไฟล์ LIT

**Microsoft Reader** เป็นแอปพลิเคชันสำหรับอ่าน e-book ที่ Microsoft ยุติการผลิตแล้ว Microsoft Reader เวอร์ชันแรกเปิดตัวในเดือนสิงหาคม พ.ศ. 2543 ซึ่งใช้รูปแบบ LIT ของตัวเอง มีให้สำหรับ PDA และคอมพิวเตอร์ Windows ชื่อของแอปนี้ยังใช้ในภายหลังสำหรับแอปพลิเคชัน (ซึ่งเปิดตัวครั้งแรกพร้อมกับ Windows 8) สำหรับอ่านไฟล์ [PDF](/th/pdf/) และ [XPS](/th/page-description-language/xps/) แอปพลิเคชัน Microsoft Reader ถูกยกเลิกในปี 2561
