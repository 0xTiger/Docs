## ไฟล์ DEF คืออะไร??

ไฟล์ DEF เป็นไฟล์ข้อมูลที่สร้างขึ้นใน [รูปแบบไฟล์สเปรดชีต](/th/spreadsheet/) โดยแอปพลิเคชันฐานข้อมูล SmartWare II ที่พัฒนาโดย Informix Software ซอฟต์แวร์ II มีสี่โมดูล ได้แก่ สเปรดชีตพร้อมกราฟิก โปรแกรมประมวลผลคำ การสื่อสาร และฐานข้อมูล ฐานข้อมูล SmartWare II เป็นฐานข้อมูลแบบสแตนด์อโลนที่สามารถรวมเข้ากับแอปพลิเคชันอื่นๆ ได้เช่นกัน สามารถเรียกใช้แบบสอบถามกับข้อมูลที่สามารถแยกและบันทึกเป็นรูปแบบไฟล์ DEF ไฟล์สเปรดชีต DEF สามารถเปิดได้ด้วยซอฟต์แวร์ SmartWare ซึ่งเลิกผลิตแล้ว
