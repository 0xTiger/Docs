## ไฟล์ XLSM คืออะไร??

ไฟล์ที่มีนามสกุล XLSM เป็นไฟล์สเปรดชีตประเภทหนึ่งที่รองรับมาโคร จากมุมมองของแอปพลิเคชัน มาโครคือชุดคำสั่งที่ใช้สำหรับกระบวนการอัตโนมัติ แมโครใช้เพื่อบันทึกขั้นตอนที่ดำเนินการซ้ำๆ และอำนวยความสะดวกในการดำเนินการโดยการเรียกใช้แมโครอีกครั้ง มาโครได้รับการตั้งโปรแกรมด้วย Visual Basic for Applications (VBA) ของ Microsoft จากภายในสมุดงาน Excel โดยใช้ Visual Basic Editor และสามารถเรียกใช้/ดีบักได้โดยตรงจากที่นั่น

ไฟล์ XLSM คล้ายกับรูปแบบไฟล์ XLM แต่อิงตามรูปแบบ Open XML ที่นำมาใช้ใน Microsoft Office 2007 กล่าวอีกนัยหนึ่ง XLSM คือไฟล์ [XLSX] (/th/spreadsheet/xlsx/) แต่รองรับมาโคร ตามค่าเริ่มต้น Excel เองมีมาโครหลายตัวสำหรับการใช้งานทั่วไป อย่างไรก็ตาม คุณยังสามารถบันทึกมาโครของคุณเองได้ด้วยฟังก์ชันที่จำเป็น

## XLSM - การบันทึกมาโคร ##

Excel มีขั้นตอนที่ใช้งานง่ายสำหรับการบันทึกมาโคร คุณจะต้องติดตั้งเครื่องมือสำหรับนักพัฒนาจึงจะทำงานกับมาโครได้ เมื่อการบันทึกแมโครอยู่ในระหว่างดำเนินการ ระบบจะบันทึกการกระทำของผู้ใช้แต่ละรายเพื่อเล่นในภายหลัง การบันทึกมาโครเกี่ยวข้องกับขั้นตอนทั้งหมดที่ผู้ใช้ดำเนินการหลังจากเริ่มการบันทึก ดังนั้น หากคุณทำให้เนื้อหาของเซลล์เป็นตัวหนา ตัวเอียง และตั้งค่าข้อความชิดขอบหลังจากเริ่มบันทึกแมโครแล้ว คำสั่งทั้งหมดเหล่านี้จะถูกบันทึก แต่ละมาโครที่บันทึกไว้สามารถกำหนดทางลัดสำหรับการเล่นอย่างรวดเร็วในภายหลัง การบันทึกมาโครสร้างโค้ด VBA ในรูปแบบของมาโครที่สามารถแก้ไขได้โดยใช้ Visual Basic Editor (VBE)
