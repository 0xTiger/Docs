## ไฟล์ GNUMERIC คืออะไร??

ไฟล์ที่มีนามสกุลไฟล์ GNUMERIC คือไฟล์สเปรดชีตที่สร้างด้วยซอฟต์แวร์ Gnumeric ซึ่งเป็นส่วนหนึ่งของชุดแอปพลิเคชันบนเดสก์ท็อป [GNOME](https://www.gnome.org/) สามารถเก็บข้อมูลแผ่นงานพร้อมกับการสนับสนุนแผนภูมิ ตัวเลือกการจัดรูปแบบ และข้อมูลอื่นๆ ข้อมูลในไฟล์ GNUMERIC จัดเก็บในรูปแบบไฟล์ [XML](/th/web/xml/) และบีบอัดโดยใช้รูปแบบการบีบอัด [GZIP](/th/compression/gz/) ไฟล์ GNUMERIC ยังบันทึกในรูปแบบไฟล์ [GNM](/th/spreadsheet/gnm/)

## รูปแบบไฟล์ GNUMERIC

ไฟล์ GNUMERIC ถูกบันทึกในรูปแบบไฟล์ XML บีบอัดโดยใช้การบีบอัด GZIP สิ่งเหล่านี้ไม่รองรับ Macros และ Pivot Table แต่เต็มไปด้วยคุณสมบัติอื่นๆ เช่น การแปลงเป็น [CSV](/th/spreadsheet/csv/), [XLSX](/th/spreadsheet/xlsx/), Microsoft Works (.wks), [ HTML](/th/web/html/), [LaTex](/th/word-processing/latex/) และ [Lotus 1-2-3](/th/spreadsheet/123/)

## Gnumeric บน Github

การพัฒนาแบบโอเพ่นซอร์สของ Gnumeric โฮสต์และดูแลบนที่เก็บ [Gnumeric official Github](https://github.com/GNOME/gnumeric)
