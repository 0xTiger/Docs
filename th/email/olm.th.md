## ไฟล์ OLM คืออะไร??

ไฟล์ที่มีนามสกุล .olm คือไฟล์ Microsoft Outlook สำหรับระบบปฏิบัติการ Mac ไฟล์ OLM เก็บข้อความอีเมล วารสาร ข้อมูลปฏิทิน และข้อมูลแอปพลิเคชันประเภทอื่นๆ ไฟล์เหล่านี้คล้ายกับไฟล์ [PST](/th/email/pst/) ที่ใช้โดย Outlook บนระบบปฏิบัติการ Windows อย่างไรก็ตาม ไม่สามารถเปิดไฟล์ OLM ที่สร้างโดย Outlook for Mac ใน Outlook สำหรับ Windows รูปแบบไฟล์ OLM ถูกนำมาใช้กับ Outlook 2011 เพื่อจัดเก็บข้อมูลในเครื่องของ Outlook ไฟล์ OLM ไฟล์เดียวสามารถมีกล่องจดหมายได้หลายกล่อง โดยจัดระเบียบข้อมูลในรูปแบบของโฟลเดอร์และโฟลเดอร์ย่อย ไฟล์ OLM สามารถเปิด/โหลดใน Microsoft Outlook 365 และ Outlook for Mac

## รูปแบบไฟล์ OLM

OLM เป็นไฟล์ไบนารีที่มีประเภท MIME ของ Application/octet-stream ข้อกำหนดทางเทคนิคของรูปแบบไฟล์ OLM ไม่พร้อมใช้งานอย่างเปิดเผยจาก Microsoft และไม่มีรายละเอียดรูปแบบสำหรับการอ้างอิงของนักพัฒนา อย่างไรก็ตาม มี API บางตัวที่สามารถอ่านรูปแบบไฟล์ OLM และเขียนลงในไฟล์ PST สำหรับใช้งานบนระบบปฏิบัติการ Microsoft Windows เช่น Aspose.Email
