## ไฟล์ ISO คืออะไร??

ไฟล์ที่มีนามสกุล .iso คือไฟล์อิมเมจของดิสก์เก็บถาวรที่ไม่มีการบีบอัด ซึ่งแสดงถึงเนื้อหาของข้อมูลทั้งหมดในออปติคอลดิสก์ เช่น ซีดีหรือดีวีดี ตามมาตรฐาน [ISO-9660](https://www.iso.org/standard/17505.html) รูปแบบไฟล์อิมเมจ ISO ประกอบด้วยข้อมูลดิสก์พร้อมกับข้อมูลระบบไฟล์ที่จัดเก็บไว้ในนั้น ความสามารถของไฟล์ ISO ในการมีเนื้อหาจำลองที่แน่นอนทำให้เป็นประเภทไฟล์ที่สมบูรณ์แบบสำหรับการสร้างสำเนาซีดี/ดีวีดี และส่วนใหญ่จะใช้เพื่อเก็บข้อมูลที่สามารถบู๊ตได้สำหรับการติดตั้ง ส่วนใหญ่แล้ว ไฟล์ ISO จะถูกเขียนลง USB/CD/DVD เป็นเนื้อหาที่สามารถบู๊ตได้สำหรับการบู๊ตเครื่องเพื่อการติดตั้ง ไฟล์ ISO มีแอปพลิเคชันประเภท MIME/x-iso9660-image

## รูปแบบไฟล์ ISO

รูปแบบไฟล์ ISO ไม่เหมือนกับรูปแบบไฟล์คอนเทนเนอร์อื่น ๆ แม้ว่าจะเก็บถาวรเนื้อหาข้อมูลที่ระบุ ไฟล์เก็บถาวรถูกสร้างขึ้นเป็นไฟล์ไบนารีที่มีโครงสร้างเนื้อหาและข้อมูลระบบไฟล์ที่แน่นอน รูปแบบไฟล์ ISO อธิบายโดย [ISO-9660](https://en.wikipedia.org/wiki/ISO_9660) ดังนี้

### โครงสร้างระดับบนสุดของไฟล์ ISO

โครงสร้างโดยรวมของไฟล์ประกอบด้วย:

* `System Area` - 32,768 ไบต์และไม่ถูกใช้งานโดย ISO_9660
* `Data Area` - ประกอบด้วย Volume descriptor set และ Path ตาราง ไดเร็กทอรี และไฟล์

### ชุดตัวอธิบายปริมาณ

พื้นที่ข้อมูลเริ่มต้นด้วยชุดตัวอธิบายปริมาณ ชุดของตัวอธิบายปริมาณหนึ่งตัวหรือมากกว่าที่สิ้นสุดด้วยชุดตัวอธิบายปริมาณ สิ่งเหล่านี้รวมกันทำหน้าที่เป็นส่วนหัวสำหรับพื้นที่ข้อมูล อธิบายเนื้อหา (คล้ายกับบล็อกพารามิเตอร์ BIOS ที่ใช้โดยดิสก์ที่ฟอร์แมตด้วย FAT, HPFS และ NTFS)

ชุดตัวอธิบายปริมาณแสดงไว้ด้านล่าง

|ชุดคำอธิบายปริมาณ|
---|
|ตัวอธิบายปริมาณ #1|
|...|
|ตัวบอกระดับเสียง #N|
|ตัวบอกปริมาณชุดตัวสิ้นสุด|

#### คำอธิบายปริมาณ

ตัวบอกปริมาณแต่ละตัวมีขนาด 2048 ไบต์และมีโครงสร้างดังต่อไปนี้:

|ส่วน |ประเภท |ตัวระบุ |เวอร์ชัน |ข้อมูล|
---|---|---|---|---|
|ขนาด |1 ไบต์ |5 ไบต์ ('CD001' เสมอ) |1 ไบต์ (0x01 เสมอ) |2,041 ไบต์|
