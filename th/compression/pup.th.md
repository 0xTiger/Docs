## ไฟล์ Linux PUP คืออะไร??

ไฟล์ Linux PUP เป็นแพ็คเกจการติดตั้งที่ใช้โดยระบบปฏิบัติการ Puppy Linux มันทำงานเป็นตัวติดตั้งและเริ่มติดตั้งไฟล์แพ็คเกจ บน Linux มักเรียกว่าไฟล์ DotPup และคล้ายกับไฟล์ตัวติดตั้ง [.exe](/th/executable/exe/) และ [.msi](/th/executable/msi/) ที่ใช้ในระบบปฏิบัติการ Windows Puppy Linux เวอร์ชันใหม่กว่าใช้รูปแบบไฟล์ .pet แทนรูปแบบไฟล์ PUP
