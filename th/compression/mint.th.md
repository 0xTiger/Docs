## ไฟล์ MINT คืออะไร??

ไฟล์ MINT เป็นไฟล์แพ็คเกจการติดตั้งที่ใช้โดยการกระจาย [Linux Mint](https://linuxmint.com/) ประกอบด้วยไฟล์ซอร์สโค้ดที่บีบอัดซึ่งแตกไฟล์และใช้เพื่อเริ่มการติดตั้ง Linux Mint OS Linux Mint OS มี GUI (Graphical User Interface) ที่ใช้งานง่ายและเป็นเจ้าแรกที่มีการออกแบบ GUI ที่สวยงาม ไฟล์ MINT ถูกขยายและจัดเก็บไว้ในไดเร็กทอรีชั่วคราวพร้อมกับแพ็คเกจตัวติดตั้ง การติดตั้งจะดำเนินการจากไดเร็กทอรีเดียวกันสำหรับการติดตั้ง Linux Mint OS

## รูปแบบไฟล์ Linux MINT

รายละเอียดรูปแบบไฟล์ภายในของรูปแบบไฟล์ MINT ไม่เปิดเผยต่อสาธารณะ อย่างไรก็ตาม ไฟล์การติดตั้งแบบบีบอัดที่จัดเก็บไว้ในไฟล์ MINT เป็นไปตามมาตรฐานการบีบอัดที่รวมเข้ากับตัวติดตั้ง Linux Mint OS ซึ่งจะทำให้ไฟล์เหล่านี้ขยายขนาดและกลายเป็นส่วนหนึ่งของการติดตั้งได้
