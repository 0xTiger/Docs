## ไฟล์ DMG คืออะไร??

ไฟล์ที่มีนามสกุล .dmg คือไฟล์ Apple Disk Image และรู้จักกันในชื่อไฟล์ Mac OS X Disk Image รูปแบบไฟล์ DMG เป็นไฟล์อิมเมจของดิสก์ที่ใช้กันแพร่หลายมากที่สุด โดยมีจุดประสงค์เพื่อจัดสรรแอปพลิเคชันและไฟล์ระหว่างพีซีของ Apple สะดวกมากในการต่อเชื่อมไฟล์ DMG ลงในไดรฟ์เสมือน หรือสามารถใช้ซอฟต์แวร์ Apple Disk Utility เพื่อเปิดและดูเนื้อหาในรูปแบบไฟล์ DMG ไฟล์ IMG เวอร์ชันเก่าที่ใช้ใน Mac OS Classic ถูกแทนที่ด้วยรูปแบบไฟล์ DMG ไฟล์ DMG ถูกสร้างขึ้นสำหรับผู้ใช้ Mac และไม่รองรับโดย Windows OS แต่ผู้ใช้สามารถเข้าถึงเนื้อหา DMG ในระบบปฏิบัติการ Windows โดยใช้เครื่องมือบีบอัดและคลายการบีบอัดเช่น 7-Zip, Pea Zip เป็นต้น นามสกุลไฟล์ DMG ยังใช้โดย เครื่องมือ Oracle Export และ Ingress สำหรับไฟล์สแครปฮีปที่เก็บในรูปแบบไบนารีของฐานข้อมูล Oracle

## ประวัติย่อ

นามสกุลไฟล์ DMG ได้รับการพัฒนาโดย Apple และส่วนใหญ่จะใช้กับคอมพิวเตอร์ Macintosh ไฟล์ DMG ส่วนใหญ่ใช้เพื่อให้อุปกรณ์ Mac สามารถติดตั้งไฟล์และซอฟต์แวร์ประเภทต่างๆ ที่เริ่มต้นจากอินเทอร์เน็ตได้ ไฟล์ DMG เป็นไฟล์อิมเมจของดิสก์ที่สามารถเมานต์ได้ โดยพื้นฐานแล้วจะแสดงบนเดสก์ท็อปของคุณเมื่อเปิดไฟล์ ไฟล์ประกอบด้วยข้อมูลดิบที่โดยทั่วไปมีการเข้ารหัสและบีบอัด ระบบ Mac จัดการกับ DMG ในลักษณะเดียวกับที่ใช้กับดิสก์ที่ถูกแทรกสลับ และพวกเขาพยายามเปิดหรือ "ติดตาม" ไฟล์ทันที

## ข้อมูลจำเพาะทางเทคนิคของรูปแบบไฟล์ DMG

* สร้างขึ้นเป็นพิเศษสำหรับคอมพิวเตอร์ Macintosh
* พัฒนาโดยแอปเปิ้ล
* รูปแบบไฟล์เป็นไบนารี
* ประเภทไฟล์คือไฟล์อิมเมจของดิสก์
* โปรแกรมที่รองรับคือ MacOS
* สามารถมีตารางพาร์ทิชัน
* อาจมีระบบไฟล์ประเภทใดก็ได้ เช่น HFS+
* ฟังก์ชั่นการบีบอัดภาพ
