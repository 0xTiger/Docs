## ไฟล์ BKF คืออะไร?? ##

ไฟล์ที่มีนามสกุล .bkf เกี่ยวข้องกับ WindowsXP โดย Microsoft Corporation ระบบปฏิบัติการทั้งหมดที่เข้าใจแคตตาล็อกไฟล์จะใช้ไฟล์ BKF ซึ่งได้รับการสนับสนุนจากยูทิลิตี้นี้ หากข้อมูลในคอมพิวเตอร์ของผู้ใช้เสียหาย ถูกเขียนทับ หรือลบออก ไฟล์ BKF จะมีประโยชน์มากในการสร้างข้อมูลในระบบของผู้ใช้ใหม่ นอกจากนี้ยังมีข้อมูลจำเพาะของไฟล์ BKF ซึ่งใช้โดยเกมคอมพิวเตอร์ Moto Racer ไฟล์เหล่านี้ประกอบด้วยไฟล์เก็บถาวรเกมที่สร้างโดยโปรแกรม

## ประวัติย่อ ##

นามสกุลไฟล์ BKF เป็นไฟล์ Windows Backup Utility ชนิดหนึ่งที่สร้างขึ้นสำหรับโปรแกรม Dragon UnPACKER โดย Open Source เป็นเรื่องปกติมากสำหรับผู้ใช้ในสหรัฐอเมริกาและผู้ที่ใช้ระบบปฏิบัติการ Windows 10

## ข้อกำหนดทางเทคนิค ##

* พัฒนาโดย Microsoft Corporation
* การเชื่อมโยงหลักกับ WindowsXP
* รองรับโดยแอพพลิเคชั่น Windows NT Backup
* หมวดหมู่แอปพลิเคชันคือไฟล์เก็บถาวร
* ประเภทรูปแบบคือไบนารี
* อยู่ในหมวดไฟล์สำรองข้อมูล
* เชื่อมโยงกับ BKF Viewer
