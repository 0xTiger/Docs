## ไฟล์ BZ2 คืออะไร??

BZ2 เป็นไฟล์บีบอัดที่สร้างขึ้นโดยใช้วิธีบีบอัดแบบโอเพ่นซอร์ส [BZIP2](http://www.bzip.org/) ส่วนใหญ่อยู่บนระบบ UNIX หรือ Linux ใช้สำหรับบีบอัดไฟล์เดียวและไม่ได้มีไว้สำหรับเก็บถาวรหลายไฟล์ สิ่งนี้ตรงกันข้ามกับรูปแบบไฟล์ TAR บนแพลตฟอร์มเดียวกันที่เก็บถาวรหลายไฟล์เป็นไฟล์เดียว แต่ไม่มีการบีบอัด ไฟล์ที่บีบอัดเป็น BZ2 สามารถแตกไฟล์ได้ด้วยแอปพลิเคชัน เช่น [WinZip](https://www.winzip.com/win/en/) BZIP2 ใช้การเข้ารหัสแบบ Run-Length Encoding (RLE) หรือ [Burrows-Wheeler](https://en.wikipedia.org/wiki/Burrows%E2%80%93Wheeler_transform) เพื่อให้ได้การบีบอัดระดับสูง

## รูปแบบไฟล์ BZ2

ไม่มีข้อกำหนดอย่างเป็นทางการสำหรับรูปแบบไฟล์นี้ อย่างไรก็ตาม [reverse engineered specifications](https://github.com/dsnet/compress/blob/master/doc/bzip2-format.pdf) อย่างไม่เป็นทางการแสดงว่าสตรีม .bz2 ประกอบด้วยส่วนหัว 4 ไบต์ซึ่งตามมาด้วย โดยบล็อกบีบอัดเป็นศูนย์หรือมากกว่า ตามด้วยเครื่องหมายสิ้นสุดสตรีมที่มี CRC 32 บิตสำหรับสตรีมข้อความล้วนที่ประมวลผล ไม่มีการเติมช่องว่างภายในระหว่างบล็อกที่ถูกบีบอัด และบล็อกทั้งหมดจะเรียงกันเป็นบิต

## เปิดเครื่องรูด / แยกไฟล์ BZ2

คุณสามารถแตกไฟล์ BZ2 บน Windows และ Mac OS โดยใช้ซอฟต์แวร์ เช่น WinZip บน linux คำสั่งต่อไปนี้ในเทอร์มินัล

```
bzip2 -d filename.bz2
```
