## ไฟล์ VDX คืออะไร??

ภาพวาดหรือแผนภูมิใด ๆ ที่สร้างใน Microsoft Visio แต่บันทึกในรูปแบบ XML จะมีนามสกุล .VDX ไฟล์ XML สำหรับการวาด Visio ถูกสร้างขึ้นในซอฟต์แวร์ Visio ซึ่งพัฒนาโดย Microsoft Microsoft Visio มีความสามารถในการสร้างเอกสารภาพที่สามารถใช้ในงานนำเสนอและเอกสารต่างๆ ไฟล์ XML รูปวาดของ Visio ประกอบด้วยวัตถุที่มองเห็นและรายละเอียดข้อมูลเมตาขององค์ประกอบที่มองเห็น นอกจากนี้ยังสามารถเพิ่มข้อความลงในไฟล์ XML การวาดภาพด้วยวิชันซิสเต็มเหล่านี้ได้อีกด้วย ไฟล์ XML การวาดภาพ Visio เหล่านี้รวมเข้ากับมาตรฐานการจัดรูปแบบ XML และข้อกำหนดการเข้ารหัสข้อมูลรูปภาพที่อนุญาตให้ซอฟต์แวร์ Microsoft Visio แสดงผลและจัดเก็บเนื้อหาในรูปแบบไฟล์ VDX ไฟล์ XML รูปวาด Visio เหล่านี้ถูกต่อท้ายด้วยนามสกุล .vdx ต่อไปนี้เป็นแอปพลิเคชันที่สามารถเปิดรูปแบบไฟล์ VDX

* ไมโครซอฟต์ Visio
* โครงการ CS Odessa ConceptDraw สำหรับ MAC

## ประวัติย่อ ##

เริ่มแรก Visio ผลิตเป็นผลิตภัณฑ์แบบสแตนด์อโลนโดย Shapeware Corporation ในปี 1992 โดยเป็นเวอร์ชัน 1.0 Shapeware Corporation เปลี่ยนชื่อเป็น Visio Corporation ในปี 1995 เพื่อใช้ประโยชน์จากการยอมรับของตลาด Microsoft รีแบรนด์ Vision ในปี 2000 เป็นแอปพลิเคชัน Microsoft Office Visio for Enterprise Architects เปิดตัวพร้อมกับ Visual Studio .NET ในปี 2546 และ 2548 Visio 2007 เปิดตัวเมื่อวันที่ 30 พฤศจิกายน 2549 Microsoft Visio เปิดตัว Ribbon ในส่วนติดต่อผู้ใช้ในปี 2553

## รูปแบบไฟล์ Visio ##

ตั้งแต่ปี 2013 visio ได้เปิดตัวรูปแบบไฟล์ใหม่ .vsdx ซึ่งแทนที่รูปแบบไฟล์อื่นอีกสองรูปแบบที่เรียกว่า .vsd (รูปแบบไฟล์ Visio binary) และ .vdx (รูปแบบไฟล์ Visio XML Drawing) .vsdx ได้เพิ่มความสามารถในการทำงานร่วมกันของไฟล์ Visio เพราะตอนนี้ซอฟต์แวร์ของบริษัทอื่นสามารถจัดการไฟล์ Visio ในระดับรูปแบบไฟล์ได้แล้ว รูปแบบไฟล์ใหม่ได้รับการสนับสนุนบน Visio Services ใน Microsoft SharePoint Server และไม่จำเป็นต้องมีรูปแบบไฟล์ "ตัวกลาง" สำหรับการเผยแพร่ไปยัง SharePoint Server มีไฟล์หลายประเภทที่ประกอบด้วย visio ที่มีนามสกุลต่อไปนี้:

* .vsdx (รูปวาด Visio)
* .vsdm (การวาดภาพที่เปิดใช้งานมาโคร Visio)
* .vssx (ลายฉลุ Visio)
* .vssm (สเตนซิลที่เปิดใช้งานมาโคร Visio)
* .vstx (เทมเพลต Visio)
* .vstm (เทมเพลตที่เปิดใช้งานมาโคร Visio)

## คุณสมบัติ ##

Visio มีเทมเพลตและรูปร่างที่หลากหลายซึ่งช่วยในการสร้างไดอะแกรมเครือข่าย ผังงาน แผนผังชั้น และการออกแบบทางวิศวกรรม เป็นต้น นอกจากนี้ เรายังสามารถรับความช่วยเหลือในการสร้างการสร้างแบบจำลองกระบวนการและการแสดงข้อมูล Microsoft Visio นั้นเรียบง่ายและเป็นมิตรกับผู้ใช้ในแง่ของการใช้งาน ช่วยในการสร้างไดอะแกรมภาพที่มีประสิทธิภาพซึ่งสามารถเพิ่มในแอปพลิเคชันอื่น ๆ เช่น PowerPoint, Word และ Publisher

Microsoft Visio ยังสนับสนุน Unified Modeling Language (UML) 2.4, Business Process Model และ Notation 2.0 นอกจากนี้ Visio ยังสนับสนุนการเผยแพร่ทั้งเวิร์กโฟลว์ SharePoint 2010 และ 2013 Visio ยังมีความสามารถในการสร้างไดอะแกรมแบบไดนามิกด้วยการเชื่อมโยงรูปร่างกับข้อมูลตามเวลาจริง ทีมงานมืออาชีพสามารถทำงานร่วมกันบนไดอะแกรมเดียวกันได้ในเวลาเดียวกัน และพวกเขายังสามารถดูความคิดเห็นของผู้ตรวจสอบได้ทั้งใน Visio และ Visio Services นอกจากนี้ ถ้าใครไม่ได้ติดตั้ง Visio ไว้ในระบบ พวกเขาก็ยังสามารถดูไดอะแกรมของคุณผ่าน Office 365 หรือ SharePoint

## รุ่น Microsoft Visio ##

1. **Microsoft Visio Standard**: รุ่น Standard ออกแบบมาสำหรับบุคคลที่ต้องการแพลตฟอร์มสร้างไดอะแกรมที่แข็งแกร่งซึ่งมีชุดสเตนซิลหรือเทมเพลตที่สร้างขึ้นจำนวนมาก แต่ละคนสามารถลดความซับซ้อนของข้อมูลที่ซับซ้อนให้เป็นไดอะแกรมที่ง่ายและไม่ซับซ้อน Microsoft Visio Standard ประกอบด้วยเทมเพลตสำหรับธุรกิจ ไดอะแกรมเครือข่าย แผนภูมิ ผังงาน และไดอะแกรมอเนกประสงค์อื่นๆ
1. **Microsoft Visio Professional**: รุ่น Professional สร้างขึ้นสำหรับบุคคลและทีมที่สามารถลดความซับซ้อนของข้อมูลที่ซับซ้อนผ่านไดอะแกรมระดับมืออาชีพและอเนกประสงค์ ฟังก์ชันพื้นฐานรวมถึง Visio Standard 2013 แต่มีรูปร่าง เทมเพลต และสไตล์เพิ่มเติมและอัปเดตพร้อมการสนับสนุนที่สูงขึ้นสำหรับการทำงานเป็นทีม ซึ่งรวมถึงการทำงานของบุคคลต่างๆ ที่ทำงานบนไดอะแกรมเดียวกันในเวลาเดียวกัน นอกจากนี้ Visio Professional 2016 ยังได้เพิ่มเทมเพลตเพิ่มเติมสำหรับไดอะแกรมธุรกิจและวิศวกรรม ไดอะแกรมกระบวนการ แผนที่ และแผนผังชั้น เป็นต้น
1. **Visio Pro สำหรับ Office 365**: เป็นรุ่นพิเศษของ Visio 2016 และสามารถเข้าถึงได้ผ่าน Office 365 มีฟังก์ชันเหมือนกับ MS Visio 2016 Professional

ต่อไปนี้คือคุณสมบัติ/ประโยชน์พื้นฐานบางประการของ MS Visio:

* ด้วยความช่วยเหลือของเครื่องมือสร้างไดอะแกรมระดับมืออาชีพ ความซับซ้อนจะง่ายขึ้น
* Jump-start การสร้างไดอะแกรมด้วยรูปทรงที่ทันสมัย ใช้งานง่าย และเรียบง่าย และเทมเพลตอัจฉริยะ
* ทุกสิ่งที่คุณต้องการมีอยู่ในที่เดียวผ่าน:
** แท็บริบบิ้น
** หน้าต่างรูปร่างที่ปรับปรุงแล้ว
** แถบสถานะใหม่
* คุณสมบัติและเครื่องมือการวาดภาพอัตโนมัติช่วยให้คุณสร้างและแก้ไขไดอะแกรมได้เร็วขึ้นผ่าน:
** แถบเครื่องมือขนาดเล็ก Quick Shapes
** ปรับปรุงไดนามิกกริด
** ปรับปรุงไดนามิกกริด
** ขนาดหน้าอัตโนมัติ
* รูปร่างที่เกี่ยวข้องจะถูกจัดกลุ่มเข้าด้วยกันผ่านคอนเทนเนอร์และกระบวนการย่อย
* ตรวจสอบความถูกต้องและความสอดคล้องในไดอะแกรมโดยใช้การตรวจสอบไดอะแกรม
* สร้างโมเดลเวิร์กโฟลว์ SharePoint ใน Visio แล้วเรียกใช้และตรวจสอบบน SharePoint Server 2010
* เชื่อมต่อข้อมูลกับไดอะแกรมและเชื่อมโยงข้อมูลกับรูปร่างได้อย่างง่ายดาย
* ใส่คำอธิบายแผนภูมิเพื่อชี้แจงกราฟิกข้อมูลของคุณ
* รีเฟรชข้อมูลในไดอะแกรมอย่างง่ายดายด้วยตนเองหรือโดยอัตโนมัติ
* เจาะลึกข้อมูลโดยละเอียดเมื่อคุณต้องการ
* แชร์ไดอะแกรมแบบอินเทอร์แอกทีฟ รีเฟรชได้ และเชื่อมโยงข้อมูลกับผู้อื่นผ่านเว็บเบราว์เซอร์
* สร้างแดชบอร์ดแบบโต้ตอบที่รวมไดอะแกรม Visio และบริการแอปพลิเคชันอื่นๆ
