## ไฟล์ GDOC คืออะไร??

ไฟล์ GDOC เป็นไฟล์ทางลัดที่ชี้ไปยังไฟล์ที่อยู่ใน Google Drive ซึ่งเป็นบริการจัดเก็บเอกสารบนคลาวด์ เมื่อติดตั้งไคลเอนต์ Google Drive บนคอมพิวเตอร์และสร้างเอกสารภายในนั้น ไดรฟ์จะสร้างเอกสารในพื้นที่เก็บข้อมูลออนไลน์บนคลาวด์และเขียน [URL](/th/web/url/) ของเอกสารนี้ในไฟล์ GDOC ใน Google ในระบบ โฟลเดอร์ไดรฟ์ เมื่อดับเบิลคลิกไฟล์ทางลัดนี้ เว็บเบราว์เซอร์เริ่มต้นจะเปิดเอกสารจากตำแหน่ง [URL](/th/web/url/) ถ้าไฟล์ทางลัดนี้ถูกย้ายออกจากโฟลเดอร์นี้ ลิงก์ไปยังเอกสารออนไลน์จะใช้งานไม่ได้

## รูปแบบไฟล์ GDOC - ข้อมูลเพิ่มเติม

ไฟล์ GDOC มีทางลัดไปยังเอกสารออนไลน์ที่เขียนในรูปแบบไฟล์ [JSON](/th/web/json/) เบราว์เซอร์ที่เปิดไฟล์ GDOC จะอ่านข้อมูล URL และเปิดเอกสารที่เชื่อมโยงเพื่อแสดงให้ผู้ใช้ลงชื่อเข้าใช้บัญชี Google นี้ซึ่งจัดเก็บเอกสารไว้ ไฟล์ GDOC ไม่มีเนื้อหาของเอกสาร

### ตัวอย่างไฟล์ GDOC

ไฟล์ GDOC สามารถเปิดได้ในโปรแกรมแก้ไขข้อความและเนื้อหามีลักษณะดังต่อไปนี้

```
{"url": "https://docs.google.com/a/test.com/spreadsheet/ccc?key=01234567898765432123456789&usp=docslist_api", "resource_id": "spreadsheet:0A12345B678HJK9TZPL9078767"}
```

อย่างที่คุณเห็น เนื้อหาถูกจัดรูปแบบเป็น JSON โดยมี URL ของเอกสารและ ID ของเอกสารที่เกี่ยวข้อง
