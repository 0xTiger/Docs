## ไฟล์ AXD คืออะไร??

ไฟล์ AXD เป็นไฟล์การตั้งค่าเว็บที่ใช้สำหรับจัดการและเรียกค้นคำขอทรัพยากรที่ฝังอยู่ใน ASP.NET ประกอบด้วยคำแนะนำในการดึงทรัพยากรที่ฝังไว้ เช่น รูปภาพ, ไฟล์ JavaScript ([.JS](/th/web/js/)) และไฟล์ [.CSS](/th/web/css/) ไฟล์ AXD ใช้สำหรับการฉีดทรัพยากรลงในเพจฝั่งไคลเอ็นต์ ซึ่งช่วยให้เพจไคลเอ็นต์สามารถเข้าถึงทรัพยากรเหล่านี้บนเซิร์ฟเวอร์ได้ด้วยวิธีมาตรฐาน

## รูปแบบไฟล์ AXD

ไฟล์ AXD จะถูกจัดเก็บเป็นไฟล์ไบนารี่ทางฝั่งเซิร์ฟเวอร์ มันอ้างถึงไฟล์ Web Handler ใน ASP.NET ซึ่งเป็นส่วนประกอบที่สร้างการตอบสนองต่อคำขอเฉพาะไปยังเว็บเซิร์ฟเวอร์ ไฟล์ AXD ทำการประมวลผลแบบกำหนดเองก่อนสร้างการตอบกลับตามคำขอเพจฝั่งไคลเอ็นต์ โดยทั่วไปสิ่งเหล่านี้จะถูกบันทึกเป็นไฟล์ **WebResource.axd**

### ไฟล์ WebResource.axd คืออะไร

โครงการ ASP.NET ส่วนใหญ่บันทึกไฟล์ AXD เป็นไฟล์ WebResource.axd ที่อนุญาตให้เพจฝั่งไคลเอ็นต์ดาวน์โหลดทรัพยากรที่ฝังอยู่ในแอสเซมบลี แอปพลิเคชันของคุณอาจเผชิญกับประสิทธิภาพการทำงานที่ช้าในกรณีที่คุณเรียกใช้ในโหมดแก้ไขจุดบกพร่อง เนื่องจากตัวจัดการ WebResources.axd ไม่ได้ถูกแคชไว้
