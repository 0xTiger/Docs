## ไฟล์ EDGE คืออะไร??

ไฟล์ที่มีนามสกุล .edge คือไฟล์ Adobe Edge Project ที่สร้างขึ้นเมื่อมีการสร้างและบันทึกโปรเจ็กต์ด้วย Adobe Edge Adobe Edge ช่วยให้นักออกแบบเว็บไซต์สร้างแอนิเมเตอร์ [HTML](/th/web/html/) แบบอินเทอร์แอกทีฟสำหรับเว็บ การเผยแพร่ดิจิทัล และการโฆษณาผ่านสื่อสมบูรณ์สำหรับเบราว์เซอร์เดสก์ท็อปและมือถือ ซึ่งทำได้โดยใช้ HTML/HTML5, [JavaScript](/th/web/js/) และ [CSS3](/th/web/css/) ไฟล์ Edge ถูกสร้างขึ้นเป็นไฟล์ข้อความธรรมดาและสามารถเปิดได้ด้วยแอพพลิเคชั่นต่างๆ เช่น Microsoft Notepad, Apple TextEdit และ Notepad++

## รูปแบบไฟล์ EDGE

ไฟล์ EDGE ถูกบันทึกในรูปแบบข้อความธรรมดาและใช้ jQuery เพื่อเปิดใช้งานภาพเคลื่อนไหว อีกทางเลือกหนึ่งของแอนิเมชั่น EDGE คือไฟล์ Flash ที่ไม่รองรับอีกต่อไป ไฟล์ EDGE ถูกบันทึกควบคู่ไปกับไฟล์ HTML, JavaScript และ CSS เมื่อเปิดไฟล์ EDGE หน้าเว็บเช่นไฟล์ HTML จะเปิดขึ้น
