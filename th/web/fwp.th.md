## ไฟล์ FWP คืออะไร??

ไฟล์ที่มีไฟล์ .fwp คือไฟล์เก็บถาวรที่สร้างด้วย Microsoft Expression Web และมีไฟล์และโฟลเดอร์ทั้งหมดสำหรับการออกแบบ Microsoft Expression Web เป็นผลิตภัณฑ์ซอฟต์แวร์การออกแบบเว็บไซต์ทั่วไปโดย Microsoft และใช้โปรแกรมแก้ไข [HTML](/th/web/html/) ถูกยกเลิกตั้งแต่ปี 2555 และไมโครซอฟต์สามารถดาวน์โหลดได้ฟรี ไฟล์ FWP สามารถส่งออกและบันทึกเป็นรูปแบบ .cab ของ Microsoft

## รูปแบบไฟล์ FWP - ข้อมูลเพิ่มเติม

ข้อกำหนดรูปแบบไฟล์ที่แน่นอนของไฟล์ FWP ไม่พร้อมใช้งานสำหรับการอ้างอิงของนักพัฒนา อย่างไรก็ตาม ไฟล์เหล่านี้เป็นไฟล์ไบนารีและสามารถเปิดได้ใน Microsoft Expression Studio ซึ่งเลิกผลิตไปนานแล้ว
