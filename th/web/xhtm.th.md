## ไฟล์ XHTM คืออะไร??

ไฟล์ที่มีนามสกุล .xhtm คือไฟล์เว็บเพจที่เขียนด้วย XHTML ใช้ XML 1.0 เป็นการใช้งานมาตรฐานและถือเป็นการปรับรูปแบบใหม่ของ HTML 4 เช่นเดียวกับ XHTML ไฟล์ XHTM มีแท็ก HTML มาตรฐานที่ปฏิบัติตามหลักเกณฑ์ XML ที่เข้มงวดมากขึ้น ไฟล์ XHTM ใช้สำหรับการเผยแพร่เนื้อหาเว็บไซต์และโดยทั่วไปจะใช้ส่วนขยาย [XHTML](/th/web/xhtml/)

คุณสามารถใช้เว็บเบราว์เซอร์ เช่น Google Chrome, Apple Safari และ Mozilla Firefox เพื่อเปิดไฟล์ XHTM

## ตัวอย่าง XHTM

ตัวอย่างต่อไปนี้ของเอกสาร XHTM กำหนดการประกาศ XML

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
