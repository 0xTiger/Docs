## เว็บบล็อคคืออะไร?

ไฟล์ WEBLOC เป็นไฟล์ทางลัดบน macOS ที่ชี้ไปยัง URL ของเว็บบนอินเทอร์เน็ต สร้างโดยการลาก URL ของเว็บจากแถบที่อยู่ของเบราว์เซอร์ไปยังเดสก์ท็อปหรือโฟลเดอร์อื่นๆ ที่ต้องการบันทึก URL ไปยังไซต์ปลายทางถูกจัดเก็บไว้ในไฟล์ทางลัดนี้ และสามารถดูได้โดยการเปิดไฟล์ในโปรแกรมแก้ไขข้อความ เช่น Microsoft Notepad, Notepad++ หรือ Apple TextEdit ไฟล์ WEBLOC คล้ายกับไฟล์ [URL](/th/web/url/)

## รูปแบบไฟล์ WEBLOC - ข้อมูลเพิ่มเติม

ไฟล์ WEBLOC ถูกจัดเก็บในรูปแบบไฟล์ข้อความธรรมดา สิ่งเหล่านี้สามารถสร้างขึ้นบน macOS โดยการลาก URL จากเว็บเบราว์เซอร์ เช่น Safari, Google Chrome หรือ Firefox หากต้องการเปิดไฟล์ WEBLOC หลายไฟล์พร้อมกัน จะต้องมีวิธีแก้ปัญหาที่รวดเร็วและง่ายที่สุดในการเปิดไฟล์เหล่านี้ผ่านแอปพลิเคชันภายนอกหรือ API API เหล่านี้สามารถอ่านและแปลง WEBLOC เป็น PDF และ JPEG
