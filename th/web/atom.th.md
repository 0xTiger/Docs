## ไฟล์ ATOM คืออะไร??

ไฟล์ ATOM เป็นรูปแบบ Syndication สำหรับโครงสร้างของฟีด Atom และเอกสารรายการ คล้ายกับไฟล์ .RSS รูปแบบไฟล์ ATOM เป็นรูปแบบการรวมที่ใช้ XML ซึ่งเป็นมาตรฐานสำหรับการเผยแพร่เนื้อหา ไฟล์ ATOM ใช้สำหรับเว็บฟีดที่อนุญาตให้โปรแกรมซอฟต์แวร์ตรวจสอบการอัปเดตที่เผยแพร่บนเว็บไซต์ ประกอบด้วยรายการที่สามารถเป็นประเภทต่างๆ ได้ เช่น หัวข้อข่าว บทความฉบับเต็ม ข้อความที่ตัดตอนมา หรือลิงก์ไปยังเนื้อหาบนเว็บไซต์

แอปพลิเคชันที่สามารถ **เปิดไฟล์ ATOM** ได้แก่ **Apple Safari**

## รูปแบบไฟล์ ATOM - ข้อมูลเพิ่มเติม

ไฟล์ ATOM ถูกจัดเก็บและเผยแพร่ในรูปแบบไฟล์ XML ยอดนิยมซึ่งเป็นรูปแบบสากลสำหรับการแลกเปลี่ยนข้อมูล ได้รับการพัฒนาเป็นทางเลือกแทน RSS โดยคำนึงถึงข้อจำกัดและข้อบกพร่องของรูปแบบไฟล์ RSS

### ประเภทของเอกสารอะตอม

1. `Atom Entry Documents` - เป็นเอกสาร XML ที่ประกอบด้วยข้อมูลรายการเดียวสำหรับฟีด Atom มันมี atom-entry องค์ประกอบที่มีองค์ประกอบย่อยจำนวนหนึ่ง เนื้อหาของรายการ Atom สามารถเป็นข้อความล้วน, HTML, XHTML หรือสื่อประเภท IANA (Internet Assigned Numbers Authority) อื่น
1. `Atom Feed Documents` - เป็นเอกสาร XML ที่ให้ข้อมูลเมตาเกี่ยวกับฟีด Atom และหนึ่งรายการขึ้นไปสำหรับฟีด เมื่อไคลเอนต์ทำการร้องขอข้อมูลจากฟีด เอกสารฟีดจะถูกสร้างขึ้นโดยเซิร์ฟเวอร์ ซึ่งรวมถึงรายการ Atom จำนวนหนึ่งเพื่อดำเนินการตามคำขอ
1. `Atom Collection` - เป็นเอกสารฟีด Atom ชนิดพิเศษที่มี URL ของรายการ Atom ที่สามารถแก้ไขได้
