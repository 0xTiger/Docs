## ไฟล์ CSP คืออะไร??

ไฟล์ CSP คือไฟล์หน้าเว็บไดนามิกที่มีเนื้อหาที่เขียนเพื่อดำเนินการบน RadGs Concept Application Server คอมไพล์และดำเนินการทางฝั่งเซิร์ฟเวอร์ หลังจากนั้นจะทำหน้าที่ร้องขอเบราว์เซอร์ของผู้ใช้ผ่านโปรโตคอล HTTP ซึ่งแตกต่างจากรูปแบบไฟล์ CON ไฟล์ CSP ไม่จำเป็นต้องติดตั้งไคลเอ็นต์แนวคิดที่ผู้ใช้เพื่อดูไฟล์เหล่านี้ คุณสามารถดูไฟล์ CSP คล้ายกับเพจฝั่งเซิร์ฟเวอร์อื่นๆ เช่น [PHP](/th/programming/php/) และ [ASP](/th/web/asp/)
