## ไฟล์ URL คืออะไร?

ไฟล์ที่มีนามสกุล .url เป็นทางลัด URL ของเว็บไซต์ที่สามารถบันทึกลงในคอมพิวเตอร์ด้วยที่อยู่เว็บไซต์ใดก็ได้ เมื่อเปิดโดยดับเบิลคลิก URL จะเปิดเว็บไซต์ในเบราว์เซอร์เริ่มต้นและหน้าจะแสดงต่อผู้ใช้ ไฟล์ URL มักเรียกว่าไฟล์ทางลัดไปยังเว็บไซต์และแสดงด้วยไอคอนของเบราว์เซอร์เริ่มต้น ไฟล์ URL สามารถเปิดและแก้ไขได้ในโปรแกรมแก้ไขข้อความใดๆ เช่น Microsoft Notepad, Notepad++ หรือ Apple TextEdit

## รูปแบบไฟล์ URL - ข้อมูลเพิ่มเติม

ไฟล์ URL จะถูกบันทึกในรูปแบบข้อความธรรมดาและสามารถเปิดและดูในโปรแกรมแก้ไขข้อความใดก็ได้ สิ่งเหล่านี้เปิดในเว็บเบราว์เซอร์เริ่มต้นที่ผู้ใช้ทำการเรียกดูอื่นๆ
