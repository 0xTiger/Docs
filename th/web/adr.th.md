## ไฟล์ ADR คืออะไร??

ไฟล์ ADR (บันทึกด้วยนามสกุล .adr) คือรายการบุ๊กมาร์กที่สร้างโดยเว็บเบราว์เซอร์ Opera รายการนี้ได้รับการเติมเมื่อผู้ใช้เก็บบุ๊กมาร์ก URL ของเว็บ บุ๊กมาร์กเหล่านี้จัดเก็บไว้ในไฟล์ ADR ที่สามารถสำรองและกู้คืนเมื่อจำเป็น เนื่องจากเบราว์เซอร์ส่วนใหญ่ เช่น Chrome และ Firefox จะบันทึกบุ๊กมาร์กในรูปแบบ [HTML](/th/web/html/) จึงแปลงเป็นรูปแบบไฟล์ ADR เพื่อนำเข้าในเบราว์เซอร์ Opera ได้ ไฟล์ ADR สามารถเปิดได้ในเบราว์เซอร์ Opera

## รูปแบบไฟล์ ADR - ข้อมูลเพิ่มเติม

ไม่ทราบรายละเอียดรูปแบบไฟล์ภายในของไฟล์ ADR อย่างไรก็ตาม มีความแตกต่างระหว่างรูปแบบไฟล์ ADR ที่ส่งออกด้วย Opera เวอร์ชันต่างๆ ซึ่งทำให้สร้าง [รวมไฟล์ ADR](https://superuser.com/questions/471959/how-do-i-merge-several-opera-adr-bookmark-files-to-one-single-file) ได้ยาก กับ Opera รุ่นต่างๆ สามารถใช้เครื่องมือภายนอกบางอย่างเพื่อรวมสิ่งเหล่านี้ได้
