## ไฟล์ CHAT คืออะไร??

ไฟล์ CHAT เป็นไฟล์กำหนดค่าที่ใช้โดย IRC (Internet Relay Chat) เพื่อเชื่อมต่อกับช่อง IRC ประกอบด้วยพารามิเตอร์การเชื่อมต่อ เช่น เซิร์ฟเวอร์ พอร์ต ชื่อของช่อง IRC และรหัสผ่านสำหรับเปิดช่อง IRC ไคลเอนต์ใช้การตั้งค่าเพื่อเชื่อมต่อกับช่องอื่น การแชทและการสนทนาบน IRC เป็นแบบตัวต่อตัวและเป็นส่วนตัว

แอปพลิเคชันที่สามารถ **เปิด** ไฟล์ CHAT ได้แก่ mIRC และ Visual IRC

## รูปแบบไฟล์ CHAT

ไฟล์ CHAT จะถูกจัดเก็บเป็นไฟล์ไบนารีลงในแผ่นดิสก์ เมื่อผู้ใช้ต้องการเชื่อมต่อกับช่องสัญญาณใน IRC เขาป้อนข้อมูลในไคลเอ็นต์ iRC และป้อนข้อมูลในซอฟต์แวร์ไคลเอ็นต์ ข้อมูลจะถูกจัดเก็บไว้ในไฟล์ CAHT สำหรับการดึงข้อมูลด้านข้าง
