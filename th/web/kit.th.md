## ไฟล์ KIT คืออะไร??

ไฟล์ที่มีนามสกุล .kit คือไฟล์ HTML ที่สร้างขึ้นด้วยแอปพลิเคชันภาษาโปรแกรม [CodeKit](https://codekitapp.com/) ประกอบด้วย `imports` และ `variables` นอกเหนือจากไฟล์ [HTML](/th/web/html/) ที่มีอยู่ ทำให้เหมาะสำหรับเว็บไซต์แบบคงที่ CodeKit รวบรวมไฟล์ KIT เป็น HTML ที่สามารถใช้เป็นไฟล์เว็บไซต์คงที่ได้อย่างง่ายดาย

## รูปแบบไฟล์ KIT

ไฟล์ KIT เป็นไฟล์ HTML ที่รวมการนำเข้าและตัวแปรเพิ่มเติม ไฟล์เหล่านี้จัดเก็บเป็นไฟล์ข้อความธรรมดาและสามารถเปิดได้ด้วยโปรแกรมแก้ไขข้อความหรือโปรแกรมแก้ไขไฟล์บนเว็บ

### การนำเข้าไฟล์ KIT

ไฟล์เกือบทุกชนิดสามารถนำเข้าไฟล์ Kit ได้ ต่อไปนี้คือไวยากรณ์การนำเข้าที่ใช้เพื่อนำเข้าไฟล์ไปยังไฟล์ .kit

```
<!-- @import "someFile.kit" -->
<!-- @import "file.html" -->
```

เมื่อมีการเพิ่มการนำเข้าไปยังไฟล์ KIT และบันทึก จะถูกแทนที่ด้วยข้อความของไฟล์ที่นำเข้า คุณยังสามารถใช้ @include แทน @import

### การนำเข้าหลายรายการในไฟล์ KIT

คุณยังสามารถนำเข้าไฟล์มากกว่าหนึ่งไฟล์พร้อมกันโดยใช้รายการที่คั่นด้วยเครื่องหมายจุลภาค:

```
<!-- @import someFile, otherFile.html, ../thirdFile.kit -->
```
