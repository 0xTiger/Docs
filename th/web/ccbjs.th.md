## ไฟล์ CCBJS คืออะไร??

ไฟล์ CCBJS เป็นไฟล์ Javascript ที่สร้างโดยแอป 3 มิติ เกม และโปรแกรมแก้ไขเว็บไซต์ **CopperCube** ประกอบด้วยโค้ด JavaScript ที่สร้างขึ้นโดยเป็นส่วนหนึ่งของ [ไฟล์ .html](/th/web/html/) ที่สร้างจากไฟล์ .ccb CopperCube มอบอินเทอร์เฟซการเขียนสคริปต์ที่เป็นมิตรต่อผู้ใช้เพื่อพัฒนาตรรกะการเล่นเกมและการโต้ตอบโดยไม่ต้องเขียนโค้ดใดๆ รูปแบบไฟล์อื่นๆ ที่ CopperCube รองรับ ได้แก่ .ccscene, .ccmodel, .ccmaterial, .cctex, .ccjs, .ccp และ .ccpak

## รูปแบบไฟล์ CCBJS

รายละเอียดรูปแบบไฟล์ภายในของรูปแบบไฟล์ CCBJS ไม่เป็นที่รู้จักเช่นนี้ อย่างไรก็ตาม ไฟล์ Javascript จะถูกบันทึกเป็นไฟล์ข้อความธรรมดาลงแผ่นดิสก์ และสามารถเปิดได้ด้วยโปรแกรมแก้ไขข้อความใดๆ เช่น Microsoft Notepad, Notepad++ และ Apple TextEdit
