## ไฟล์ APKG คืออะไร??

ไฟล์ที่มีนามสกุล .apkg คือชุดแฟลชการ์ดที่สร้างขึ้นเพื่อใช้ในแอปพลิเคชันซอฟต์แวร์ [Anki](https://ankiweb.net/about) ซึ่งเป็นโปรแกรมการเรียนรู้ที่ใช้แฟลชการ์ด มีข้อความ [HTML](/th/web/html/) ที่จะโหลดและแสดงในแอปพลิเคชัน Anki และสามารถมีรูปภาพและเสียงเพิ่มเติมสำหรับการเรียนรู้ด้วยภาพและเสียง Anki อนุญาตให้ผู้ใช้สร้างสำรับแฟลชการ์ด Anki ของตนเองรวมถึงนำเข้าสำรับแฟลชการ์ดของผู้ใช้รายอื่น

## รูปแบบไฟล์ APKG

สำรับไพ่ Anki สร้างขึ้นจากเทมเพลตที่เขียนด้วย HTML ซึ่งเป็นภาษายอดนิยมและเป็นภาษาทั่วไปสำหรับสร้างหน้าเว็บ การจัดรูปแบบสำรับไพ่ทำได้โดยใช้ CSS ซึ่งเป็นภาษาที่ใช้ในการจัดรูปแบบหน้าเว็บ สไตล์รวมถึงการดัดแปลงเพื่อ:

* font-family - ชื่อของแบบอักษรที่จะใช้ในการ์ด
* font-size - ขนาดของตัวอักษรเป็นพิกเซล
* text-align - กำหนดว่าข้อความควรจัดชิดกึ่งกลาง ซ้าย หรือขวา
* color - กำหนดสีของข้อความซึ่งอาจเป็นชื่อสีง่ายๆ เช่น "สีน้ำเงิน" "สีแดง" ฯลฯ หรือรหัสสี HTML
* สีพื้นหลัง - กำหนดสีพื้นหลังของการ์ด

ข้อมูลการจัดรูปแบบจะใช้ร่วมกันระหว่างการ์ดทั้งหมด ซึ่งมีผลกับการ์ดทั้งหมดเมื่อทำการเปลี่ยนแปลง ตัวอย่างต่อไปนี้จะใช้พื้นหลังสีเหลืองบนการ์ดทั้งหมด ยกเว้นการ์ดใบแรก:

```CSS
.card {
  background-color: yellow;
}
.card1 {
  background-color: blue;
}
```

การปรับขนาดรูปภาพในการ์ด Anki สามารถควบคุมได้ดังต่อไปนี้

```CS
img {
  max-width: none;
  max-height: none;
}
```

## การฝัง Javascript ในการ์ด Anki

เป็นไปได้ที่จะฝัง [Javascript](/th/web/js/) ในการ์ด Anki โดยใช้เทมเพลตการ์ด อย่างไรก็ตาม เนื่องจากระดับขั้นสูงของ Javascript จึงไม่มีการสนับสนุน นอกจากนี้ อุปกรณ์เรนเดอร์อาจแสดงผลการใช้งาน Javascript ในการ์ดแตกต่างกัน ทำให้ต้องมีการทดสอบการใช้งานบนอุปกรณ์ทั้งหมด คุณสมบัติบางอย่างของ Javascript เช่น window.alert ทำให้การแก้ไขข้อบกพร่องของโค้ด Javascript ที่คุณเขียนทำได้ยาก
