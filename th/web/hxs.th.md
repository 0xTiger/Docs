## ไฟล์ HXS คืออะไร??

ไฟล์ HXS เป็นไฟล์วิธีใช้ที่คอมไพล์และใช้สำหรับไฟล์ Microsoft Help 2.x ซึ่งรวบรวมและสร้างขึ้นจากชุดของหน้าหัวข้อและเขียนด้วยชุดย่อยของ [HTML](/th/web/html/) ซึ่งคล้ายกับรูปแบบไฟล์ช่วยเหลือของ [CHM](/th/web/chm/) ก่อนหน้า ไฟล์ HXS ถูกจัดเก็บในรูปแบบที่บีบอัดและสร้างขึ้นจากไฟล์โปรเจ็กต์หลัก .HxC, .HxF รวมถึงไฟล์, สารบัญ .HxT, ไฟล์นิยามแอตทริบิวต์ .HxA และดัชนี .HxK หลายรายการ ข้อมูลเมตาของไฟล์ HXS สร้างขึ้นจาก [XML](/th/web/xml/)

ไฟล์ HXS สามารถเปิดได้ด้วยเว็บเบราว์เซอร์

## รูปแบบไฟล์ HXS

ไฟล์ HXS ถูกสร้างขึ้นและบันทึกเป็นไฟล์บีบอัดจากเนื้อหา HTML และข้อมูลเมตา XML
