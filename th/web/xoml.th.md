## ไฟล์ XOML คืออะไร?

XOML เป็นตัวย่อสำหรับ Extensible Object Markup Language และเป็นรูปแบบการทำให้เป็นอนุกรมสำหรับวัตถุเวิร์กโฟลว์ของ Windows Workflow Foundation ขึ้นอยู่กับ **[XAML](/th/web/xaml/)** โดยส่วนใหญ่จะใช้สำหรับสร้างส่วนต่อประสานกับผู้ใช้ในแบบธรรมดา **[XML](/th/web/xml/)** สิ่งเหล่านี้ใช้เพื่อประกาศเวิร์กโฟลว์สำหรับอินเทอร์เฟซผู้ใช้ และรวบรวมพร้อมกับไฟล์แยกต่างหากที่มีตรรกะการใช้งาน ประกอบด้วยโครงสร้างแบบต้นไม้ที่กำหนดโหนดเวิร์กโฟลว์รูท องค์ประกอบย่อยที่ซ้อนกัน และเซ็กเมนต์ฝังโค้ด เมื่อเวิร์กโฟลว์ใหม่ถูกสร้างขึ้นด้วย Visual Studio สำหรับ .NET เวิร์กโฟลว์จะมีตัวเลือกให้คุณเลือกว่าควรอยู่ในรูปแบบโค้ดหรือรูปแบบโค้ดที่แยกจากกัน ในกรณีที่คุณเลือกไฟล์ในภายหลัง IDE จะสร้างไฟล์ให้คุณสองไฟล์ หนึ่งอยู่ในรูปแบบ XOML (ประกอบด้วยเค้าโครงเวิร์กโฟลว์และการตั้งค่า) และอีกอันอยู่ในรูปแบบ [.CS](/th/programming/cs/)/[.VB](/th/programming/vb/) ซึ่งมีรหัสโปรแกรมอยู่
