## ไฟล์ DAP คืออะไร??

ไฟล์ที่มีนามสกุล .dap คือไฟล์ Microsoft Access Data Page ที่สร้างขึ้นจากภายใน Microsoft Access และมีสิทธิ์เข้าถึงข้อมูลที่จัดเก็บไว้ในฐานข้อมูล ข้อมูลที่จัดเก็บไว้ในไฟล์เว็บดังกล่าวสามารถเข้าถึงได้ผ่านเครือข่าย (อินทราเน็ตและอินเทอร์เน็ต) ซึ่งแตกต่างจากไฟล์ [MDB](/th/database/mdb/) ของ Access ซึ่งพัฒนาขึ้นสำหรับแอปพลิเคชันเดสก์ท็อปโดยเฉพาะ ฐานข้อมูลดังกล่าวสามารถสร้างได้โดยใช้ Access 2010 และ Access Services (ส่วนประกอบเสริมของ SharePoint) ฐานข้อมูลเหล่านี้สามารถใช้ได้โดยผู้ที่มีบัญชี SharePoint โดยใช้ที่อยู่เว็บในเว็บเบราว์เซอร์

## รูปแบบไฟล์ DAP

ข้อมูลที่แน่นอนเกี่ยวกับรูปแบบไฟล์ DAP และข้อกำหนดไม่พร้อมใช้งานสำหรับการอ้างอิงของนักพัฒนา
