## ไฟล์ COMPRESSED คืออะไร?

ไฟล์บีบอัดคือหน้าเว็บบีบอัด Wordpress ขนาดเล็กที่สร้างด้วยปลั๊กอิน **WordCompress** ประกอบด้วยโค้ดบรรทัดเดียวสำหรับแสดงหน้าเว็บที่สร้างขึ้นโดยการบีบอัดโค้ด HTML, CSS และ JavaScript โดยปลั๊กอิน ไฟล์ COMPRESSED ที่แปลงแล้วเหล่านี้มีขนาดเล็กซึ่งช่วยให้โหลดและถ่ายโอนข้อมูลทางอินเทอร์เน็ตได้เร็วขึ้น

### รูปแบบไฟล์บีบอัด - ข้อมูลเพิ่มเติม

ไฟล์บีบอัดจะถูกบันทึกเป็นไฟล์ข้อความที่สามารถเปิดได้ด้วยโปรแกรมแก้ไขข้อความใดๆ เช่น Microsoft Notepad, Notepad++ และ Apple TextEdit นอกจากโปรแกรมแก้ไขข้อความแล้ว ไฟล์ COMPRESSED ยังเปิดได้ด้วยซอฟต์แวร์สร้างเว็บ เช่น Adobe Dreamweaver ที่สามารถใช้ได้ทั้ง Windows และ Mac OS

## เกี่ยวกับปลั๊กอิน WordCompress

ปลั๊กอิน WordCompress ถูกใช้ครั้งแรกเพื่อบีบอัดหน้าเว็บ WordPress เป็นรูปแบบไฟล์บีบอัด ปลั๊กอินไม่ได้รับการสนับสนุนอีกต่อไปและถูกยกเลิกแล้ว มันจะเขียนซอร์สโค้ดของเว็บเพจลงในไฟล์ปฏิบัติการไฟล์เดียวโดยการลบช่องว่างทั้งหมดออกจากซอร์สโค้ดของเพจ ไฟล์บีบอัดจะถูกจัดเก็บไว้ข้างๆ ซอร์สโค้ดต้นฉบับของหน้าเว็บ เพื่อรักษาความคิดเห็นต้นฉบับ การเยื้อง และพื้นที่สีขาวอื่นๆ ที่รวมอยู่ในซอร์สโค้ดต้นฉบับของเพจ
