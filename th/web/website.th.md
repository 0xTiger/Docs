## ไฟล์ WEBSITE คืออะไร?

ไฟล์ WEBSITE เป็นไฟล์ทางลัดเว็บไซต์ที่สร้างด้วย Microsoft Internet Explorer (IE) ประกอบด้วย URL และไฟล์ไอคอน (ICO) ที่ชี้ไปยัง favicon ของเว็บไซต์ ไอคอนใช้สำหรับแสดงทางลัด รูปแบบไฟล์ WEBSITE ถูกนำมาใช้พร้อมกับการเปิดตัว IE เวอร์ชัน 9 IE เวอร์ชันก่อนหน้าใช้เพื่อบันทึกทางลัดไปยังหน้าเว็บเป็นไฟล์ [.url](/th/web/url/)

## รูปแบบไฟล์เว็บไซต์

ไฟล์ .website จะถูกจัดเก็บไว้ในดิสก์เป็นไฟล์ข้อความล้วน ซึ่งเนื้อหาสามารถดูได้ในโปรแกรมแก้ไขข้อความใดๆ นอกจากเนื้อหาอื่นๆ แล้ว ยังมี URL เป้าหมายที่เป็นทางลัด หากคุณต้องการเปิดเว็บไซต์เป้าหมายในเบราว์เซอร์อื่นที่ไม่ใช่ IE คุณสามารถคัดลอก URL นี้และวางในเบราว์เซอร์อื่น เช่น Google Chrome, Safari หรือ Firefox เพื่อดูเว็บไซต์

## จะสร้างไฟล์ .WEBSITE ได้อย่างไร?

ไฟล์ WEBSITE สามารถสร้างขึ้นใน IE ได้โดยการคลิก URL ในแถบที่อยู่แล้วลากไปยังตำแหน่งที่ต้องการบนคอมพิวเตอร์ของคุณ เช่น เดสก์ท็อปหรือเอกสาร สิ่งนี้จะสร้างทางลัดไปยังตำแหน่งอินเทอร์เน็ตนั้นและสามารถเปิดได้โดยดับเบิลคลิกที่ทางลัด หากคุณต้องการบันทึกทางลัดเป็นไฟล์ `URL` ให้กดปุ่ม SHIFT ค้างไว้ก่อนที่จะลากไอคอน favicon ของไซต์เป้าหมายไปที่เดสก์ท็อป
