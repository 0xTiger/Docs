## ไฟล์ ECE คืออะไร??

ไฟล์ ECE คือหน้าเว็บที่สร้างขึ้นแบบไดนามิกโดยแพลตฟอร์มการจัดการเนื้อหา **Escenic Content Engine (ECE)** ประกอบด้วยโค้ดฝั่งเซิร์ฟเวอร์ที่ประมวลผลโดยเว็บเซิร์ฟเวอร์และส่งกลับไปยังเบราว์เซอร์ของผู้ใช้เป็นหน้า [HTML](/th/web/html/) รูปแบบไฟล์ ECE ไม่เป็นที่นิยมมากนักเมื่อเทียบกับรูปแบบไฟล์ฝั่งเซิร์ฟเวอร์อื่นๆ เช่น ASP และ PHP ไฟล์เหล่านี้แสดงผลภายในและให้บริการแก่ผู้ใช้ปลายทางเพื่อวัตถุประสงค์ในการแสดงผล

## รูปแบบไฟล์ ECE

ไฟล์ ECE ไม่ได้บันทึกลงดิสก์ แต่ส่งถึงผู้ใช้โดยตรงเมื่อมีการร้องขอ สิ่งเหล่านี้จะถูกแปลงเป็น HTML และส่งกลับไปยังเบราว์เซอร์ของผู้ใช้เพื่อแสดงผล Escenic Content Engine ใช้สำหรับเผยแพร่หน้าบทความข่าว เซิร์ฟเวอร์ ECE ใช้เทคโนโลยี JEE และ SQL มอบพื้นที่เก็บข้อมูลที่เชื่อถือได้และปรับขนาดได้สำหรับเนื้อหาออนไลน์ทั้งหมด

ไคลเอนต์เบราว์เซอร์สามารถเข้าถึงเนื้อหาบนเซิร์ฟเวอร์ ECE ผ่าน HTTP/REST API นอกเหนือจากการแสดงหน้า [HTML](/th/web/html/) แล้ว ECE ยังมีเวิร์กโฟลว์ที่มีประสิทธิภาพสูงในการจัดการและเผยแพร่เนื้อหาวิดีโอ/เสียง
