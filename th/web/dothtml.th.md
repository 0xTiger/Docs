## ไฟล์ DOTHTML คืออะไร??

ไฟล์ DOTHTML เป็นเอกสารเทมเพลต Microsoft Word ที่บันทึกในเวอร์ชัน HTML สามารถเปิดได้ในเว็บเบราว์เซอร์และเนื้อหาจะแสดงเหมือนหน้าเว็บอื่น ๆ ไฟล์ DOTHTML สร้างขึ้นจากไฟล์ Microsoft Word .DOT ที่มีการตั้งค่ารูปแบบล่วงหน้าสำหรับสร้างไฟล์เหล่านี้ เมื่อแปลงไฟล์ [DOT](/th/word-processing/dot/) เป็น DOCHTML การตั้งค่าเหล่านี้ทั้งหมดจะยังคงอยู่ และไฟล์ DOTHTML ที่แปลงแล้วจะแสดงในรูปแบบเดียวกันกับไฟล์ DOT ต้นฉบับ ด้วยวิธีนี้ เทมเพลตที่เผยแพร่สามารถดูได้ในเว็บเบราว์เซอร์

## รูปแบบไฟล์ DOTHTML - ข้อมูลเพิ่มเติม

ไฟล์ DOTHTML ถูกบันทึกเป็นไฟล์ [HTML](/th/web/html/) และมีซอร์สโค้ด HTML ที่สามารถเปิดได้ด้วยโปรแกรมแก้ไข HTML ใดๆ ในช่วงแรก Microsoft Word นำเสนอรูปแบบเอกสารทั้งหมดในรูปแบบ HTML เช่น [DOCHTML](/th/web/dochtml/), [DOCMHTML](/th/web/docmhtml/) แต่ภายหลังเวอร์ชันที่ใช้ HTML เหล่านี้ได้ยุติลง . ส่งผลให้ DOTHMLT ไม่รองรับอีกต่อไป
