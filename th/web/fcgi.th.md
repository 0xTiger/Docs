## ไฟล์ FCGI คืออะไร??

ไฟล์ FCGI เป็นไฟล์ซอร์สโค้ดที่มีซอร์สโค้ดที่เขียนขึ้นสำหรับ FastCGI (FCGI) สามารถเขียนด้วยภาษาโปรแกรมต่างๆ ได้หลายภาษา และมีส่วนขยายของมาตรฐาน CGI ที่ใช้สำหรับการจัดการคำขอเว็บเซิร์ฟเวอร์ที่มีประสิทธิภาพมากขึ้น ภาษาที่รองรับโดยทั่วไปสำหรับ FCGI รวมถึง **Perl** ที่ให้การสนับสนุนทั้งรหัส Perl และอินเทอร์เฟซภายนอกที่แข็งแกร่งของ FastCGI API

## รูปแบบไฟล์ FCGI

อะไรทำให้ไฟล์ FCGI มีความสำคัญ ส่วนต่อประสาน FCGI รองรับการประมวลผลแบบกระจายเพื่อจัดการคำขอเว็บและกระจายไปยังเซิร์ฟเวอร์หลายเครื่อง สิ่งนี้ทำให้ไฟล์ FCGI ส่วนใหญ่ใช้สำหรับโหลดบาลานซ์ข้ามเซิร์ฟเวอร์หลายเครื่อง
