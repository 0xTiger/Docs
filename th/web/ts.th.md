## ไฟล์ TS คืออะไร??

ไฟล์ TS เป็นไฟล์แปลที่สร้างขึ้นด้วยยูทิลิตี้ซอฟต์แวร์ lupdate ที่มาพร้อมกับ Qt SDK ของ Digia SDK พร้อมใช้งานสำหรับระบบปฏิบัติการ Windows, Mac และ Linux ไฟล์ TS มีวลีในภาษาหนึ่งและคำแปลเป็นภาษาอื่น คำแปลที่จัดเก็บไว้ในไฟล์ TS นั้นเฉพาะสำหรับแอปพลิเคชันเฉพาะ ไฟล์ TS สามารถเปิดได้ด้วย Digia Qt SDK

## รูปแบบไฟล์ TS

ไฟล์ Digia Qt TS ถูกจัดเก็บไว้ในดิสก์ในรูปแบบไฟล์ XML และสามารถเปิดได้ด้วยโปรแกรมแก้ไขข้อความใดๆ เช่น Notepad และ Notepad++ รูปแบบไฟล์อื่น ๆ ที่รองรับโดย Digia Qt SDK ได้แก่ :

* .pri
* .qm
* .qml
* .qph
* .qrc
* .qs
* .qss
* .ts
* ไฟล์ส่วนต่อประสานผู้ใช้ .ui
