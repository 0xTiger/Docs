## ไฟล์ ARO คืออะไร??

ไฟล์ ARO เป็นไฟล์เว็บเพจฝั่งเซิร์ฟเวอร์ที่สร้างด้วย SteelArrow Web Application ประกอบด้วยแท็กและสคริปต์ [HTML](/th/web/html/) และ SteelArrow สิ่งเหล่านี้ดำเนินการบนเซิร์ฟเวอร์เพื่อสร้างหน้าตอบกลับที่สมบูรณ์ก่อนที่จะส่งไปยังเบราว์เซอร์ของผู้ใช้ ไฟล์ ARO มีเนื้อหา HTML เกือบ 95% ในขณะที่ส่วนที่เหลือประกอบด้วยรหัส SteelArrow เพื่อให้ไฟล์ ARO ทำงานให้คุณได้ เซิร์ฟเวอร์ SteelArrow Web Applications ควรได้รับการติดตั้งและใช้งานบนเครื่องเว็บเซิร์ฟเวอร์

## รูปแบบไฟล์ ARO

ไฟล์ ARO ถูกเขียนด้วยไฟล์ภาษามาร์กอัป HTML พร้อมโค้ด JavaScript ในตัวและแอปเพล็ต Java [แท็ก SteelArrow](http://www.steelarrow.com/docs/basics1.aro) เป็นองค์ประกอบพื้นฐานสำหรับการพัฒนาหน้าเว็บ แม้ว่าสิ่งเหล่านี้จะไม่เปลี่ยนการแสดงหน้า แต่ใช้เพื่อเติมข้อมูลในหน้า นอกจากนี้ยังอาจใช้เพื่อควบคุมเอาต์พุตด้วยคำสั่งแบบมีเงื่อนไข

### ตัวอย่างแท็ก ARO

เดอะ<SAOUTPUT> แท็ก ARO ช่วยให้นักพัฒนาสามารถส่งออกค่าข้อมูลได้ทุกที่ภายในสคริปต์ ตัวอย่างเช่น สามารถใช้เพื่อรับเอาต์พุตของตาราง PARAM ด้วย<SAOUTPUT VALUE=PARAM[]> ที่สามารถแสดงได้ภายใน HTML<BODY> แท็ก
