## ไฟล์ CPHD คืออะไร??

ไฟล์ CPHD เป็นเอกสารไฮเปอร์เท็กซ์ที่ใช้โดยกลไกพรีโปรเซสเซอร์ CloudChan ประกอบด้วย HTML และสคริปต์เพื่อสร้างเว็บเพจและเอกสารแบบไดนามิก สคริปต์ภายในไฟล์ CPHD คล้ายกับไวยากรณ์ที่ใช้โดย Java หน้าเว็บ CPHD ส่งคืนเอกสาร HTML เมื่อดำเนินการบนกระบวนการ CloudChan รูปแบบ CPHD สามารถทำงานร่วมกับเอกสาร XML ใดก็ได้

## รูปแบบไฟล์ CPHD

ไฟล์ CPHD จะถูกจัดเก็บไว้ในแผ่นดิสก์เป็นไฟล์ข้อความธรรมดา ทำให้สามารถเปิดไฟล์เหล่านี้ในโปรแกรมแก้ไขข้อความใดก็ได้ เช่น Notepad หรือ Notepad++
