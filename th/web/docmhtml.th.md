## ไฟล์ DOCMHTML คืออะไร?

ไฟล์ DOCMHTML เป็นเวอร์ชันเก่าของไฟล์เก็บถาวรบนเว็บที่บันทึกโดย Microsoft Word เวอร์ชันก่อนหน้า ไม่รองรับ Microsoft Word เวอร์ชันล่าสุดอีกต่อไป ไฟล์ DOCMHTML มีการแสดง HTML ของเอกสารคำ .doc ต้นฉบับ สามารถเปิดได้ใน Microsoft Word และในเว็บเบราว์เซอร์ เมื่อเผยแพร่ DOCMHTML สามารถโฮสต์บนเว็บเซิร์ฟเวอร์ที่ผู้ใช้สามารถเข้าถึงได้ผ่านคำขอเว็บ อีกรูปแบบที่คล้ายกันคือ [DOCHTML](/th/web/dochtml/) ซึ่งถูกยกเลิกเช่นกัน

## รูปแบบไฟล์ DOCMHTML - ข้อมูลเพิ่มเติม

ไฟล์ DOCMHTML ใช้เพื่อบันทึกในรูปแบบ HTML สามารถเปิดได้ในโปรแกรมแก้ไขข้อความ เช่น Notepad และ Notepad++ หลังจากถูกทำเครื่องหมายว่าล้าสมัย Microsoft Word ได้เปิดตัวตัวเลือกในการบันทึกเอกสารคำเป็นไฟล์ [MHTML](/th/web/mhtml/) และ [MHT](/th/web/mht/) ไฟล์รูปแบบใหม่เหล่านี้สามารถเปิดได้ด้วยเว็บเบราว์เซอร์
