## ไฟล์ XPD คืออะไร??

ไฟล์ XPD เป็นไฟล์เว็บเพจที่สร้างจากเอกสาร XML บ่อยครั้งที่สิ่งนี้ถูกสร้างขึ้นผ่านการแปลง XSLT [ข้อกำหนดเฉพาะของรูปแบบไฟล์ XPD](https://www.w3.org/TR/xml-pipeline/) มีอยู่ใน W3 ซึ่งกำหนดวิธีเพิ่มการทำงานร่วมกันของหลายกระบวนการ [XML](/th/web/xml/) การใช้งาน ภาษาไปป์ไลน์ XML กำหนดความสัมพันธ์ในการประมวลผลระหว่างทรัพยากร XML
