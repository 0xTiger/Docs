## NX फ़ाइल क्या है?

NX फ़ाइल एक वीडियो गेम फ़ाइल है जिसे वर्चुअल गेम कंसोल, **LowRes NX** के साथ उपयोग करने के लिए बनाया गया है। यह बेसिक प्रोग्रामिंग भाषा में लिखा गया है और इसमें गेम का सोर्स कोड शामिल है जो परिभाषित करता है कि गेम कैसे काम करता है। आप ऐप के उपयोग के लिए अपनी स्वयं की एनएक्स फ़ाइलें बना सकते हैं जिन्हें लोड किया जा सकता है और उपयोगकर्ता गेम खेल सकते हैं।

आप एक NX फ़ाइल को टेक्स्ट एडिटर, LowRes NX, या किसी भी सोर्स कोड एडिटर जैसे Microsoft Visual Studio कोड में खोल सकते हैं।

## एनएक्स फ़ाइल स्वरूप

एक NX फ़ाइल को सादे पाठ फ़ाइल के रूप में डिस्क में सहेजा जाता है जिसमें गेम निर्देश होते हैं। प्रत्येक NX फ़ाइल में गेम का स्रोत कोड होता है जिसमें एकाधिक कार्ट्रिज ROM प्रविष्टियाँ शामिल होती हैं। इन ROM प्रविष्टियों में गेम घटकों की जानकारी होती है जैसे ग्राफिक्स, लेवल मैप, ऑडियो और गेम के लिए अन्य गैर-स्रोत कोड डेटा। किसी प्रोग्राम की ROM प्रविष्टियों को NX संपादन प्रोग्राम का उपयोग करके जोड़ा या संपादित किया जा सकता है जो LowRes NX के साथ बंडल में आते हैं।

## NX फ़ाइलें कैसे बनाएं?

### पीसी पर एनएक्स फ़ाइलें बनाएं

आप इसका उपयोग करके पीसी पर एनएक्स फ़ाइलें बना सकते हैं:

* टेक्स्ट संपादक - जैसे माइक्रोसॉफ्ट नोटपैड, नोटपैड++, या जीथब एटम
* सोर्स कोड संपादक - जैसे माइक्रोसॉफ्ट विज़ुअल स्टूडियो कोड

### iOS पर NX फ़ाइलें बनाएं

LowRes NX में iOS संस्करण है जिसका उपयोग डेवलपर्स इस ऐप के भीतर से NX फ़ाइलें बनाने के लिए कर सकते हैं।
