## AWK फाइल क्या है?

AWK फाइल टेक्स्ट प्रोसेसिंग एप्लिकेशन **AWK** के लिए बनाई गई एक स्क्रिप्ट फाइल है जो यूनिक्स और लिनक्स ऑपरेटिंग सिस्टम के साथ आती है। इसमें किसी फ़ाइल के टेक्स्ट या सामग्री पर कार्रवाई करने के लिए तार्किक निर्देश होते हैं जैसे कि टेक्स्ट का मिलान करना, बदलना और प्रिंट करना। यह इनपुट फ़ाइल से रिपोर्ट और स्वरूपित पाठ उत्पन्न करने में मदद करता है। अजीब उपयोगिता आमतौर पर अधिकांश लिनक्स/यूनिक्स वितरण पर/usr/bin/awk में स्थित होती है।

## AWK स्क्रिप्ट कैसे बनाते हैं?

Linux/Unix OS पर किसी भी टेक्स्ट एडिटर में AWK फाइल बनाई या खोली जा सकती है। एक नई AWK स्क्रिप्ट फ़ाइल निम्नानुसार बनाई जा सकती है।

```
$ vi script.awk
```

यह AWK फाइल को टेक्स्ट एडिटर में खोलेगा। पाठ संपादक में निम्नानुसार कुछ कथन दर्ज करें।

```
#!/usr/bin/awk -f
BEGIN { printf "%s\n","Writing my first Awk executable script!" }
```
इस पाठ सामग्री की पहली पंक्ति को निम्नानुसार समझाया गया है।

* \#! - `शेबांग` के रूप में संदर्भित, जो एक स्क्रिप्ट में निर्देशों के लिए एक दुभाषिया निर्दिष्ट करता है
* /usr/bin/awk - दुभाषिया है
* -f - दुभाषिया विकल्प, प्रोग्राम फ़ाइल को पढ़ने के लिए उपयोग किया जाता है

## AWK स्क्रिप्ट कैसे निष्पादित करें?

फ़ाइल को सहेजें और नीचे दिए गए आदेश को जारी करके स्क्रिप्ट को निष्पादन योग्य बनाएं:
```
$ chmod +x script.awk
```

स्क्रिप्ट को तब अनुसरण के रूप में चलाया जा सकता है।

```
$ ./script.awk
```

जिसके परिणामस्वरूप निम्न आउटपुट होता है।

```
Writing my first Awk executable script!
```
