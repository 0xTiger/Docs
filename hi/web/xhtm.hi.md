## XHTM फ़ाइल क्या है?

.xhtm एक्सटेंशन वाली फ़ाइल एक वेबपेज फ़ाइल है जो XHTML में लिखी जाती है। यह अपने मानक कार्यान्वयन के रूप में XML 1.0 का उपयोग करता है और इसे HTML 4 का सुधार माना जाता है। XHTML की तरह, एक XHTM फ़ाइल में मानक HTML टैग होते हैं जो अधिक कठोर XML दिशानिर्देशों का पालन करते हैं। XHTM फ़ाइलें वेबसाइट सामग्री प्रकाशित करने के लिए उपयोग की जाती हैं और आमतौर पर [XHTML](/hi/web/xhtml/) एक्सटेंशन का उपयोग करती हैं।

XHTM फ़ाइलें खोलने के लिए आप Google Chrome, Apple Safari और Mozilla Firefox जैसे वेब ब्राउज़र का उपयोग कर सकते हैं।

## एक्सएचटीएम उदाहरण

XHTM दस्तावेज़ का निम्नलिखित उदाहरण XML घोषणाओं को परिभाषित करता है।

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
