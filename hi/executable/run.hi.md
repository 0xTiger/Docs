## RUN फ़ाइल क्या है?

.रन फ़ाइल स्वरूप का उपयोग आमतौर पर लिनक्स वातावरण में सॉफ़्टवेयर या एप्लिकेशन इंस्टॉलर को वितरित करने के लिए किया जाता है। सॉफ़्टवेयर स्थापित करने के लिए, आपको फ़ाइल को निष्पादन योग्य बनाने की आवश्यकता होगी, जो निम्नलिखित कमांड का उपयोग करके किया जा सकता है:

```bash
chmod +x file_name.run 
```

फिर, आप निम्न आदेश का उपयोग करके फ़ाइल चला सकते हैं:

```bash
./file_name.run 
```

आप जिस विशिष्ट फ़ाइल और प्रोग्राम को इंस्टॉल करने का प्रयास कर रहे हैं उसके आधार पर इंस्टॉलेशन प्रक्रिया भिन्न हो सकती है।

.रन फ़ाइल स्वरूप एक प्रकार की शेल स्क्रिप्ट है जिसका उपयोग लिनक्स वातावरण में सॉफ़्टवेयर या एप्लिकेशन इंस्टॉलर को वितरित करने के लिए किया जाता है। यह एक स्व-निहित पैकेज है जिसमें सॉफ़्टवेयर स्थापित करने के लिए आवश्यक सभी चीज़ें शामिल हैं, जिसमें बाइनरी फ़ाइलें, लाइब्रेरी और कॉन्फ़िगरेशन फ़ाइलें शामिल हैं।

यह ध्यान रखना महत्वपूर्ण है कि .run फ़ाइलों में दुर्भावनापूर्ण कोड भी हो सकता है, इसलिए फ़ाइल के स्रोत को सत्यापित करना और इसे चलाने से पहले वायरस के लिए स्कैन करना हमेशा एक अच्छा विचार है।

इसके अतिरिक्त, कुछ .run फ़ाइलों को चलाने और स्थापित करने के लिए रूट विशेषाधिकारों की आवश्यकता हो सकती है, इसलिए आपको फ़ाइल को उन्नत अनुमतियों के साथ चलाने के लिए "sudo" कमांड का उपयोग करने की आवश्यकता हो सकती है:

```bash
sudo ./filename.run
```

## RUN फ़ाइल कैसे खोलें?

.run फ़ाइल खोलने के लिए, आपको इसे निष्पादन योग्य बनाना होगा, जो "chmod" कमांड के साथ किया जा सकता है:

```bash
chmod +x filename.run 
```

एक बार जब फ़ाइल निष्पादन योग्य बन जाए, तो आप इसे टाइप करके चला सकते हैं:

```bash
./filename.run
```

.run फ़ाइल को चलाना किसी टेक्स्ट एडिटर में खोलने के समान नहीं है। .run फ़ाइल चलाने से उसके निर्देश निष्पादित होंगे, जो प्रोग्राम इंस्टॉल करने से लेकर स्क्रिप्ट चलाने तक कुछ भी हो सकता है। किसी .run फ़ाइल की सामग्री को देखने के लिए, आपको इसे नैनो या विम जैसे टेक्स्ट एडिटर में खोलना होगा:

```
nano filename.run
```
या
```
vim filename.run
```
