## Mi az a DEF fájl?

A DEF fájl egy adatfájl, amelyet [táblázatfájl-formátum](/hu/spreadsheet/) formátumban hozott létre az Informix Software által kifejlesztett SmartWare II adatbázis-alkalmazás. A Software II-nek négy modulja volt, beleértve a táblázatkezelőt grafikával, szövegszerkesztőt, kommunikációt és adatbázist. A SmartWare II adatbázis egy önálló adatbázis, amely más alkalmazásokkal is integrálható. A lekérdezések végrehajthatók olyan adatokon, amelyek kivonhatók és DEF fájlformátumba menthetők. A DEF táblázatfájlok megnyithatók a SmartWare szoftverrel, amely már megszűnt.
