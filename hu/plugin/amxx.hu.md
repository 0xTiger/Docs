## Mi az AMXX fájl?

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. Ez egy parancsfájl-platform, amely lehetővé teszi a szerveradminisztrátorok számára, hogy testreszabják és kibővítsék szervereik funkcióit. Az AMXX a MetaMod beépülő modul-kezelő rendszerrel együtt használatos, amely lehetővé teszi több beépülő modul egyidejű futtatását a szerveren konfliktusok nélkül. Az AMXX beépülő modul által kínált funkciók közé tartoznak az adminisztrátori parancsok, a játékmenet módosításai és a szerverkezelés.
