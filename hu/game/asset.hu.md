## Mi az ASSET fájl?

Az ASSET fájl egy speciális fájl, amelyet a Unityben, egy videojátékok készítésére szolgáló programban használnak. Fontos konfigurációkat tartalmaz, például játékobjektumokat vagy beállításokat. Például egy ASSET fájl tárolhatja, hogy az objektumok hogyan mozognak a játékban, például hogyan pattannak le egymásról vagy zuhannak le. Ezek a fájlok olyan tárolók, amelyekben minden fontos információ rendszerezett, így a játékfejlesztők könnyedén dolgozhatnak projektjeiken.

## ASSET fájl megnyitása
  
A Unity rendszerben általában nem nyitja meg” az eszközfájlokat ugyanúgy, mint egy dokumentumot vagy fájlt egy szövegszerkesztőben vagy hasonló programban. Ehelyett a Unity magában a Unity Editorban kezeli az eszközfájlokat. A következőképpen érheti el az eszközfájlokat és dolgozhat velük a Unityben:

1.  **Import Assets**: To use an asset file in your Unity project, you need to import it into project. You can do this by either dragging and dropping asset file directly into Unity Editor or by using "Assets" menu and selecting "Import New Asset."
    
2.  **Eszközök megtekintése**: Az importálás után megtekintheti és kezelheti az eszközfájlokat a Unity Editor Project” ablakában. Ez az ablak megjeleníti a projektben lévő összes eszközt, és lehetővé teszi, hogy mappákba rendezze őket, keressen konkrét eszközöket, és megtekintse a tartalmukat.
    
3.  **Eszközök használata**: Az eszközfájlokat a Unity-projektben úgy használhatja, hogy áthúzza őket a Project” ablakból a jelenethierarchiába vagy a jelenetben lévő GameObjects-re. Például húzhat egy textúra elemet egy anyagra, hogy egy 3D objektumra alkalmazza, vagy egy előre gyártott elemet húzhat a jelenetbe, hogy egy előre konfigurált GameObject példányt készítsen.
    
4.  **Eszközök szerkesztése**: Egyes tartalomfájlok, például szkriptek, anyagok és animációk közvetlenül szerkeszthetők a Unity Editorban. Kattintson duplán ezekre az eszközökre a Projekt” ablakban, hogy megnyissa őket a megfelelő szerkesztőjükben, és szükség szerint módosítsa őket.
    
5.  **Eszközök mentése**: A Unity Editor eszközfájljain végzett minden módosítás automatikusan mentésre kerül a Unity projektben. Nincs szükség külön menteni az egyes eszközfájlokat, mint ahogyan azt egy hagyományos szoftverben tárolná.

Az ASSET fájlokat a következő programokkal lehet megnyitni vagy hivatkozni.
