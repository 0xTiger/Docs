## Mi az EDGE fájl?

Az .edge kiterjesztésű fájl az Adobe Edge Project fájl, amely akkor jön létre, amikor egy projektet létrehoznak és mentenek az Adobe Edge segítségével. Az Adobe Edge lehetővé teszi a webtervezők számára, hogy interaktív [HTML](/hu/web/html/) animátorokat hozzanak létre webes, digitális közzétételi és multimédiás hirdetésekhez asztali és mobil böngészőkhöz egyaránt. Ez HTML/HTML5, [JavaScript](/hu/web/js/) és [CSS3](/hu/web/css/) használatával érhető el. Az Edge fájlok egyszerű szöveges fájlokként jönnek létre, és számos különböző alkalmazással megnyithatók, például a Microsoft Notepad, az Apple TextEdit és a Notepad++ segítségével.

## EDGE fájlformátum

Az EDGE fájlok egyszerű szöveges formátumban kerülnek mentésre, és a jQuery segítségével engedélyezik az animációt. Az EDGE-animáció alternatívája a már nem támogatott Flash-fájlok voltak. Az EDGE fájl a HTML, JavaScript és CSS fájlokkal együtt kerül mentésre. Az EDGE fájl megnyitásakor megnyílik a weboldal, azaz a HTML fájl.
