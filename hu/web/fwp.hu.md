## Mi az FWP fájl?

Az .fwp fájlokat tartalmazó fájl a Microsoft Expression Web programmal létrehozott archív fájl, amely tartalmazza a terv összes fájlját és mappáját. A Microsoft Expression Web a Microsoft általános webdesign szoftverterméke, amely [HTML](/hu/web/html/) szerkesztőn alapul. 2012 óta megszűnt, és a Microsoft ingyenesen letölthető. Az FWP fájlok a Microsoft .cab formátumában exportálhatók és menthetők.

## FWP fájlformátum - További információ

Az FWP-fájlok pontos fájlformátum-specifikációi nem állnak rendelkezésre a fejlesztők számára. Ezek azonban bináris fájlok, és a Microsoft Expression Studio programban nyithatók meg, amely már régen megszűnt.
