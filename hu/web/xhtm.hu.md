## Mi az XHTM fájl?

Az .xhtm kiterjesztésű fájl egy XHTML nyelven írt weboldal. Az XML 1.0-t használja szabványos megvalósításaként, és a HTML 4 újrafogalmazásának tekintik. Az XHTML-hez hasonlóan az XHTM-fájlok is szabványos HTML-címkéket tartalmaznak, amelyek szigorúbb XML-irányelveket követnek. Az XHTM-fájlokat webhelytartalmak közzétételére használják, és gyakran [XHTML](/hu/web/xhtml/) kiterjesztést használnak.

Az XHTM-fájlok megnyitásához használhat webböngészőket, például a Google Chrome-ot, az Apple Safarit és a Mozilla Firefoxot.

## XHTM példa

Az XHTM dokumentum következő példája az XML deklarációkat határozza meg.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
