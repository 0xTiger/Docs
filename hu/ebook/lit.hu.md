## Mi az a LIT fájl?

A LIT fájlok az ősi **Microsoft Reader**-hez tartoznak, amelyet e-könyvek olvasására használtak. A Microsoft Reader által megjelenített e-könyvek kiterjesztése **.lit**, amely **Irodalom**-ra rövidül. A LIT fájlformátum egy Microsoft ITOL/ITLS HTML Help 2.0 fájlból áll, amely mindenféle elképesztő funkciót kínál, mint például az [LZX](/hu/compression/lzx/) tömörítést, a jelölőanyagok bináris megjelenítését és tetszőleges opcionális kiegészítő adatokat minden patak. A könyv szövege tetszőleges számú OEBPS 1.0 jelölőfolyam és az OEBPS 1.0 CSS egy részhalmaza. A képek a következők: [GIF](/hu/image/gif/), [JPEG](/hu/image/jpeg/) és [PNG](/hu/image/png/).

## Mennyi ideig jelenítette meg a Microsoft Reader a LIT fájlokat

A **Microsoft Reader** a Microsoft által leállított e-könyvek olvasására szolgáló alkalmazás. A Microsoft Reader első verziója 2000 augusztusában jelent meg, amely saját LIT formátumot használt. PDA-kra és Windows számítógépekre volt elérhető. Az alkalmazás nevét később egy alkalmazáshoz is használták (amelyet eredetileg a Windows 8 rendszerrel adtak ki) [PDF](/hu/pdf/) és [XPS](/hu/page-description-language/xps/) fájlok olvasására. A Microsoft Reader alkalmazás 2018-ban megszűnt.
