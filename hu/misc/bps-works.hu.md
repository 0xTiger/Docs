## Mi az a BPS fájl?

A .BPS fájl lényegében a Microsoft Works programmal készített szövegszerkesztő dokumentum biztonsági másolata, amely általában .WPS fájlkiterjesztést használ dokumentumaihoz. Ezeket a .BPS fájlokat gyakran a Microsoft Works automatikusan hozza létre elővigyázatosságból arra az esetre, ha a program összeomlana, miközben Ön a dokumentumon dolgozik. Ez egy módja annak, hogy megóvja munkáját és megelőzze az adatvesztést.

A következő részben a BPS fájlokhoz kapcsolódó szoftveralkalmazásokat vizsgáljuk meg.

## Microsoft működik

A Microsoft Works a Microsoft által kifejlesztett hatékonyságnövelő szoftvercsomag. Először 1987-ben mutatták be, és a teljes Microsoft Office programcsomag megfizethetőbb alternatívájaként vált népszerűvé. A Microsoft Works számos alapvető irodai alkalmazást tartalmazott, mint például:

- **Szövegszerkesztő:** A Microsoft Works rendelkezett szövegszerkesztő alkalmazással, amely lehetővé tette a felhasználók számára dokumentumok létrehozását és szerkesztését. [.WPS](/hu/word-processing/wps/) fájlformátumot használt a szövegszerkesztő dokumentumokhoz.

- **Táblázat:** Tartalmaz egy táblázatkezelő alkalmazást a táblázatok létrehozásához és kezeléséhez. A táblázatok fájlformátuma [.XLR](/hu/spreadsheet/xlr/).

- **Adatbázis:** A Microsoft Works adatbázis-alkalmazással is rendelkezett az adatok rendszerezésére és kezelésére. Az adatbázisfájlok fájlformátuma [.WDB](/hu/database/wdb/).

- **Naptár és címjegyzék:** Az alapvető irodai alkalmazásokon kívül a Microsoft Works gyakran tartalmazott naptárak és címjegyzékek kezelésére szolgáló eszközöket.

- **Works Word Processor Format (.WPS):** A Microsoft Works szövegszerkesztő összetevőjének natív fájlformátuma. A .WPS-fájlok szöveget, formázást és egyszerű grafikát tartalmaztak.

- **Leállítás:** A Microsoft 2007-ben hivatalosan leállította a Microsoft Works szolgáltatást, és arra ösztönözte a felhasználókat, hogy váltsanak Microsoft Office-ra vagy más alternatívákra.
