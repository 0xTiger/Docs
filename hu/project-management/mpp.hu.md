## Mi az MPP fájl?

Az .mpp kiterjesztésű fájl egy Microsoft Project adatfájl, amely integrált módon tárolja a projektmenedzsmenttel kapcsolatos információkat. A Microsoft azért fejlesztette ki ezt a szabadalmaztatott fájlformátumot, hogy kompatibilis legyen a Microsoft Projecttel (MSP), a projektmenedzsment szoftvertermékével. Az MPP-n kívül az MSP projekt XML sémával is tud dolgozni. Számos alkalmazás és API lehetővé teszi az MPP-k más fájlformátumokká való konvertálását. A Microsoft már rendelkezik egy online Project Serverrel, ahová projektmenedzsment fájlokat tölthet fel több felhasználó együttműködésére.

## MPP fájlformátum ##

A Microsoft bináris fájlformátumként fejlesztette ki és tette közzé az MPP-t, specifikációit pedig a Microsoft a mai napig nem tette közzé. A belső fájlstruktúra és a részletek ezért nem állnak rendelkezésre a fejlesztők számára. Számos API elérhető online, amelyek támogatják az MPP-fájlok olvasását/írását, de a 100%-os pontosság soha nem érhető el. A Microsoft az idők során számos szolgáltatást vezetett be projektmenedzsment alkalmazásába, és az MPP fájlformátum funkcióit is.

Az MPP fájl a következő MIME-típusokat használja:

* Application/vnd.ms-projekt
* Application/msproj
* alkalmazás/msproject
* Application/x-msproject
* Application/x-ms-projekt
* Application/x-dos_ms_project
* alkalmazás/mpp
* zz-application/zz-winassoc-mpp

## Problémái vannak az MPP fájl megnyitásakor? ##

Íme néhány gyakori probléma, amely előfordulhat, és az MPP formátum hibás működését okozhatja:

* A támogató szoftver hiánya
* Sérült fájl
* Fertőzött fájl vírus miatt
* Nincs hozzáférési jog a rendszerben a fájlok megnyitásához
* Elavult meghajtó a rendszerben
* A fájl kiterjesztése átnevezve
