## Mi az NX fájl?

Az NX-fájl egy videojáték-fájl, amelyet a **LowRes NX** virtuális játékkonzollal való használatra hoztak létre. A BASIC programozási nyelven íródott, és tartalmazza a játék forráskódját, amely meghatározza a játék működését. Saját NX-fájlokat hozhat létre az alkalmazással való használatra, amelyeket betölthet, és a felhasználók játszhatnak a játékkal.

Egy NX-fájlt megnyithat szövegszerkesztőben, LowRes NX-ben vagy bármilyen forráskód-szerkesztőben, például a Microsoft Visual Studio Code-ban.

## NX fájlformátum

Az NX fájl egyszerű szöveges fájlként kerül a lemezre, amely tartalmazza a játék utasításait. Minden NX fájl rendelkezik a játék forráskódjával, amely több patron ROM bejegyzést tartalmaz. Ezek a ROM-bejegyzések olyan játékkomponens-információkat tartalmaznak, mint a grafika, szinttérképek, hang és egyéb, nem forráskóddal kapcsolatos adatok a játékhoz. A programok ROM-bejegyzései hozzáadhatók vagy szerkeszthetők a LowRes NX-hez mellékelt NX szerkesztőprogramokkal.

## Hogyan készítsünk NX fájlokat?

### Hozzon létre NX fájlokat számítógépen

NX fájlokat hozhat létre PC-n a következőkkel:

* Szövegszerkesztők – például Microsoft Notepad, Notepad++ vagy Github Atom
* Forráskód-szerkesztők – például a Microsoft VIsual Studio Code

### NX-fájlok létrehozása iOS rendszeren

A LowRes NX iOS-verzióval rendelkezik, amelyet a fejlesztők használhatnak NX-fájlok létrehozására az alkalmazáson belül.
