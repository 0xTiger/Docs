## Mi az a GROOVY fájl?

A GROOVY fájl egy Groovy programozási nyelven írt forráskódfájl. A GROOVY fájlba írt kód hasonló az objektum-orientált nyelvhez [Java](/hu/programming/java/), ami megkönnyíti az alkalmazások tervezését és fejlesztését. A GROOVY fájl Java virtuális gép (JVM) bájtkódra van fordítva, és kompatibilis más Java kódokkal és könyvtárakkal.

## GROOVY fájlformátum - További információ

A GROOVY fájlok Groovy szintaxisban írt forráskódot tartalmaznak. Tartalmazhat olyan kódot, amely programozási nyelvként és a Java Platform szkriptnyelveként egyaránt használható. A groovy nyelv a Python, Ruby és Smalltalk jellemzőihez hasonló tulajdonságokkal rendelkezik.
