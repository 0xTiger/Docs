## Mi az a PHP fájl?
A .php kiterjesztésű fájl nyílt forráskódú programozási nyelvre utal, amelyet szerveroldali szkriptek írására használnak, amelyeket webszerveren kell végrehajtani. Ez a legszélesebb körben használt web-szkriptnyelv, amelyet általában nagyméretű webes alkalmazások fejlesztésére használnak. A web legnagyobb blogrendszere, azaz a WordPress és a legnagyobb közösségi hálózat, a Facebook PHP nyelven alapul. Különféle platformokkal kompatibilis, mint például Windows, Linux, Unix, Mac OS X stb. Szintén sikeresen fut szinte minden ma használt szerveren, pl. Apache, IIS stb. A PHP számos adatbázist támogat, beleértve a MySQL-t is.

## PHP fájlformátum ##

Egyetlen PHP tartalmazhat szöveget, CSS-t, [HTML](/hu/web/html/), JavaScriptet és PHP kódot. A PHP kód lefut a szerveren, és az eredmény sima HTML-ként kerül vissza a böngészőbe. A PHP-fájlok bármely szövegszerkesztővel megnyithatók és a helyükön szerkeszthetők, bár az olyan alkalmazások, mint az Adobe Dreamweaver, az Eclipse PHP-fejlesztőeszközök kényelmes módot kínálnak a PHP-kód írására és módosítására.

### Szintaxis ###

A szerveren egy PHP szkript fut le, és a sima HTML eredmény visszaküldésre kerül a böngészőnek. A PHP szkript ezzel kezdődik<?php and ends with ?>

### PHP példa ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
