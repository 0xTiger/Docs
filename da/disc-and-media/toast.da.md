## Hvad er en Toast-fil?
Roxio Toast-softwaren opretter cd- eller dvd-billede og gemmer filen med filtypenavnet .toast. Roxio Toast er et velkendt softwareprogram til oprettelse af optiske diske og mediekonvertering til Macintosh, som først blev introduceret af Miles Software GmbH, derefter købt af Adaptec. Toast-filen kan indeholde enhver form for diske, såsom video, lyd eller datadisk. Det kan også være en kopi af og et andet diskbillede.

## Toast filformat
The Toast file format is used by Roxio Toast software to save its output in a file with .toast extension. Although, the discs of various types can be burned directly through Mac OS systems, but Toast provides additional control over the process, and also offers some extra features, such as cataloging, tracking of files burned to disc and file recovery for damaged discs.It also provides support for audio and video formats that does not supported by Quicktime, such as [FLAC](/audio/flac/) and [OGG](/audio/ogg/). Since it is a  proprietary file format, the information about the entire structure of the file format is not available.
### Disktyper
Derfor kan Roxio Toast oprette Toast-filer, der indeholder diskbilleddata af følgende typer:
- **Lav videodiske**: Toast kan oprette Toast-filer indeholdende videodiske ved at bruge følgende projekter:
- DVD-video
- Blu-ray video
- High Definition DVD
- VIDEO_TS-mapper
- VIDEO_TS-kompilering
- BDMV-mappe
- AVCHD-arkiv
- **Lav datadiske**: Toast kan oprette Toast-filer indeholdende datadiske ved at bruge følgende projekter:
- Kun Mac
- Mac og pc
- DVD-ROM
- Brugerdefineret hybrid
- Mac-volumen
- Fotodisk
- **Lav videodiske**: Toast kan oprette Toast-filer, der indeholder datadiske ved at bruge følgende projekter:
- Lyd-cd
- Musik DVD
- MP3-disk
- Forbedret lyd-cd
- Mixed Mode CD
- **Kopiering af disk**: Toast understøtter kopiering på følgende måder:
- Diskkopi
- Billedkopi
- Diskbillede fletning
