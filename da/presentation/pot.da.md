## Hvad er en POT fil?

Filer med .POT-udvidelse repræsenterer Microsoft PowerPoint-skabelonfiler oprettet af PowerPoint 97-2003-versioner. Filer, der er oprettet med disse versioner af Microsoft PowerPoint, er i binært format sammenlignet med dem, der er oprettet i Office OpenXML-filformater, der bruger de højere versioner af PowerPoint. De genererede filer kan derfor bruges til at oprette præsentationer, der har samme layout og andre indstillinger, der skal anvendes på nye filer. Disse indstillinger kan omfatte stilarter, baggrunde, farvepalet, skrifttyper og standardindstillinger. Sådanne filer genereres for at skabe klar-til-brug skabelonfiler til officiel brug.

## Filformatspecifikationer ##

File format specifications for POT file format are not available individually by Microsoft. However, the template format is the same as that of the binary [PPT](/presentation/ppt/) file format and can be opened in MS PowerPoint for editing purpose. A POT file is identified by HEX identifier as follow:

```
D0 CF 11 E0 A1 B1 1A E1 00 00 00 00
```

De MIME-typer, der er knyttet til POT-filformatet, er:

* application/vnd.ms-powerpoint [officiel]

* applikation/mspowerpoint

* applikation/x-mspowerpoint

* applikation/powerpoint

* applikation/x-powerpoint

* application/x-dos_ms_powerpnt

* påføring/potte

* applikation/x-soffic
