## Hvad er en ADP fil?

A file with .adp extension is a Microsoft Access project file that uses OLE DB component architecture to provide a direct and efficient connection to a Microsoft SQL Server database. Such files do not contain the actual tables, database diagrams or other database elements. ADP files can be created with Access 2007 and 2010. Filer oprettet med de tidligere versioner af Access kan også åbnes Access 2007 og 2010. ADP-filer kan åbnes ved hjælp af Microsoft Access 365.

## ADP-filformat - flere oplysninger

Et Access-projekt oprettet som ADP giver dig mulighed for at foretage designændringer til SQL-serverobjekter såsom tabeller og visninger. Det giver også mulighed for at oprette, redigere og bruge SQL-funktioner såsom databasediagrammer, lagrede procedurer og brugerdefinerede funktioner. Dette giver en fordel i forhold til at linke til en SQL-serverdatabase, hvor du ikke kan foretage ændringer i designet og kun kan linke til SQL-servertabeller og -visninger.

### Datatyper og diagramunderstøttelse

**Datatyper for dato/klokkeslæt**

Følgende nye dato-/tidsdatatyper understøttes i Access 2010.

 * TID
 * DATO
 * DATOTIME2
 * DATOTIMEOFFSET

**Datatyper med variabel længde**

Følgende datatyper med variabel længde kan bruges i Access 2010-projekter:

 * VARBIN(MAX)
 * VARCHAR(MAX)
 * NVARCHAR(MAX)
