## Hvad er en FP7 fil?

A file with .fp7 extension is a database file created with FileMaker Pro versions 7-11. Den gemmer data i tabeller med poster, hvor hver post indeholder et eller flere felter. FP7-filer har mulighed for at gemme store data, herunder personlige databasedata og organisatoriske optegnelser. En mere nyere version af FileMaker Pro er [FMP12](/database/fmp12/), der blev introduceret med lanceringen af FileMake Pro version 12. Ud over tekstdata kan FP7-filen også indeholde formularer, diagrammer og andre databaseobjekter. FP7-filer kan åbnes med FileMaker Pro Advanced.

## FP7 filformat

FP7-filer gemmes på disken som binære filer i FileMakers proprietære filformat. De interne filformatdetaljer for FP7-filformatet er ikke offentligt tilgængelige.
