## Hvad er ADN fil?

En fil med filtypenavnet .adn er en tom Microsoft Access-skabelonfil, der bruges til at oprette Access-projektfiler med forudlagrede indstillinger. Microsoft Access-projektfiler gemmes som ADP-filer og indeholder oplysninger såsom databaseforbindelse og indstillinger for projektstart. ADN-filer findes for 2007- og 2010-versioner af Microsoft Access. Microsoft Access understøtter andre filformater såsom [MDB](/database/mdb/), [ACCDB](/database/accdb/), [ACCDC](/database/accdc/), [ACCDE](/database/accde/) og mange flere ud over disse ADN-filer. ADN-filer kan åbnes ved hjælp af Office 365 og Microsoft Access.

## ADN-filformat - flere oplysninger

ADN er databaseskabelonfiler, der gemmes på disken som binære filer. De interne filformatdetaljer for ADN-filer er ikke tilgængelige for udviklerens reference.
