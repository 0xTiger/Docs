## Hvad er SVGZ fil?

En fil med filtypenavnet .svgz er en komprimeret version af Scalable Vector Graphics-filen (.SVG). Det er komprimeret med gzip-komprimering og indeholder data i formatet [XML](/web/xml/). SVGZ-filer understøtter gennemsigtighed, gradienter, animationer og filtre. SVGZ-filer er mindre i størrelse sammenlignet med standard SVG-filer, og denne reducerede filstørrelse hjælper med at overføre grafikfilerne online. Grafikdesigner laver SVGZ-filer ved hjælp af værktøjer som Adobe Illustrator, Corel PaintShop Pro og andre. SVGZ-filer kan dog genereres ved at aktivere [GZip compression](/compression/gz/) i Apache-serveren, mens billeddataene udsendes.

## SVGZ filformat

SVGZ, being compressed [SVG](/page-description-language/svg/), is based on open compressed gzip format that reduces the size of SVG documents. This compressed archive contains plain text SVG documents inside that are purely based on the XML file format (/web/xml/). SVGZ files can be opened by viewers/applications that conform to the SVG 1.1 specifications. SVGZ files 20-50% smaller in size than the original SVG files.
