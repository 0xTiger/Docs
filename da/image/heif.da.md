## Hvad er en HEIF fil?

An HEIF file is a High-Efficiency Container Image file format that is capable of storing a single image or a sequence of images in a single file. It, thus, is basically a container having multiple images stored. HEIF file format compresses the images using the High Efficiency Video Coding (HEVC) standard. It is is more efficient and light weight as compared other image file formats such as [JPEG](/image/jpeg/). A single HEIF file can store sequences of images such as [GIF](/image/gif/) animations or HDR sequence, along with metadata that describes each image. Apple Inc. adopted its variant HEIC file format for their iOS 11. Dette fik andre virksomheder til at understøtte det i deres software, herunder Adobe.

## HEIF filformat

The HEIF format was developed by Moving Picture Experts Group (MPEG) and was registered under the ISO/IEC 23008-12 standard in 2015. Det blev vedtaget af en række producenter som Apple, Microsoft og Canon. HEIF-filer gemmes med filtypenavnet .heif (hvis det indeholder et enkelt billede) eller som .heifs (hvis det indeholder en sekvens af billeder).

### HEIF Datalagring

HEIF-formatet kan gemme følgende typer billeder og data.

 * Individuelle billeder og billedminiaturer
 * Sekvenser af billeder
 * Efterbehandlingsinstruktioner
 * Billedmetadata, såsom [EXIF](/image/exif/) data

HEIF billedformat giver mulighed for flere forskellige billedformater, herunder [JPEG](/image/jpeg/), [PNG](/image/png/) og [GIF](/image/gif/) uden at gå på kompromis med billedkvaliteten.
