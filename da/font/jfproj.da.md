## Hvad er JFPROJ fil?

A file with .jfproj extension is a JSON based custom font created with Microsoft Font Maker application on Windows 10. Disse kan oprettes ved at bruge elektronisk pen med berøringsskærmenheder såsom tablets, selvom andre computerudstyr såsom mus også kan bruges i tilfælde af computere. JFPROJ-filer indeholder de glyffer, der bruges til at gengive tegnene til skærmen ved at definere afstanden mellem tegn og ord og størrelsen af skrifttypen. Microsoft Font Maker-appen er tilgængelig til download i Windows Store.
