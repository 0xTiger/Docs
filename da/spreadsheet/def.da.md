## Hvad er DEF fil?

En DEF-fil er en datafil, der er oprettet i [spreadsheet file format](/spreadsheet/) af SmartWare II-databaseapplikationen, udviklet af Informix Software. Software II havde fire moduler inklusive regneark med grafik, tekstbehandling, kommunikation og database. SmartWare II-databasen er en selvstændig database, der også kan integreres med andre applikationer. Forespørgsler kan udføres på data, der kan udtrækkes og gemmes i DEF-filformatet. DEF-regnearksfiler kan åbnes med SmartWare-software, som er udgået nu.
