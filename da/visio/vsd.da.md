## Hvad er en VSD fil?

VSD-filer er tegninger oprettet med Microsoft Visio-applikationen for at repræsentere forskellige grafiske objekter og sammenkoblingen mellem disse. Sådanne tegninger kan indeholde visuelle objekter såsom visuelle objekter, flowdiagrammer, UML-diagram, informationsflow, organisationsdiagrammer, softwarediagrammer, netværkslayout, databasemodeller, objektkortlægning og anden lignende information. Microsoft Visio tilbyder muligheden for at konvertere Visio-filer til en række forskellige filformater, herunder [PNG](/image/png/), [BMP](/image/bmp/), [PDF](/pdf/) og andre.

# VSD-filformat #

VSD file format is the initial file format introduced by Microsoft Visio for storing drawing elements. The internal structure of VSD files is based on binary file format unlike [.VSDX](/visio/vsdx/) that was introduced with Microsoft Visio 2013. Det nye format er baseret på Office Open XML-filformatet og gemmer projektoplysninger i et [ZIP](/compression/zip/)-arkiv. Microsoft tilbyder vviewer.exe-applikationen til Visio-tegnefiler, hvis Visio ikke er installeret på systemet.

## Kort historie ##

Visio was initially produced as a standalone product by Shapeware Corporation in 1992 by its version 1.0. Shapeware Corporation ændrede navnet til Visio Corporation i 1995 for at drage fordel af markedets anerkendelse. Microsoft re-brander Vision i 2000 som Microsoft Office-applikation. Visio for Enterprise Architects blev introduceret med Visual Studio .NET i 2003 og 2005. Visio 2007 blev udgivet den 30. november 2006. Bånd blev introduceret i brugergrænsefladen af Microsoft Visio i 2010.
