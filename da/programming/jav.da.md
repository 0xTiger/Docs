## Hvad er en Java-fil?

En Java-fil er en Java-kildekodefil, der er blevet oprettet i Java-programmeringssproget. Det ligner Java-kildefilen og kan kompileres med Java-fortolkere. Ligesom [Java](/programming/java/) filer, bruges Java-fil til udvikling af Java-applikationer. Det kan være en af de mange kildekodefiler, der er tilgængelige i et Java-projekt.

## Java filformat

Java-filer gemmes som tekstfiler og kan åbnes med enhver teksteditor. Java-filtyper er almindeligvis blevet erstattet af Java-filtypen og bruges almindeligvis til at skrive Java-applikationer. Java-koder kan skrives med IDE'er som Eclipse og IntelliJ Idea, der gør det lettere at arbejde med Java-filer.

### Eksempel på Java-filformat

```java
/*
The example code prints
Hello World from Java to the console.
*/
    public class JavaApp {
        public static void main(String[] args) {
            System.out.println("Hello World."); // Prints the string to the console.
        }
    }
```
