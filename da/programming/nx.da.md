## Hvad er en NX fil?

En NX-fil er en videospilsfil, der er oprettet til brug med den virtuelle spillekonsol, **LowRes NX**. Det er skrevet i BASIC-programmeringssproget og indeholder kildekoden til spillet, der definerer, hvordan spillet fungerer. Du kan oprette dine egne NX-filer til brug med appen, som kan indlæses og brugere kan spille spillet.

Du kan åbne en NX-fil i en teksteditor, LowRes NX eller en hvilken som helst kildekodeeditor, såsom Microsoft Visual Studio Code.

## NX filformat

En NX-fil gemmes på disken som almindelig tekstfil, der indeholder spilinstruktionerne. Hver NX-fil har spillets kildekode, der består af flere patron ROM-indgange. Disse ROM-indgange har spilkomponentoplysninger såsom grafik, niveaukort, lyd og andre ikke-kildekodedata for spillet. Et programs ROM-indgange kan tilføjes eller redigeres ved hjælp af NX-redigeringsprogrammer, der følger med LowRes NX.

## Hvordan opretter man NX-filer?

### Opret NX-filer på pc

Du kan oprette NX-filer på pc ved hjælp af:

 * Teksteditorer - såsom Microsoft Notepad, Notepad++ eller Github Atom
 * Kildekodeeditorer - såsom Microsoft Visual Studio Code

### Opret NX-filer på iOS

LowRes NX har iOS-version, der kan bruges af udviklere til at oprette NX-filer fra denne app.
