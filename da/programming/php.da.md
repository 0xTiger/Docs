## Hvad er en PHP-fil?
En fil med filtypenavnet .php refererer til open source programmeringssprog, der bruges til at skrive scripts på serversiden, som skal udføres på en webserver. Det er det mest udbredte web-scripting sprog, der typisk bruges til udvikling af store webapplikationer. Det største bloggingsystem på nettet, dvs. WordPress og det største sociale netværk Facebook, er baseret på PHP-sprog. Det er kompatibelt med forskellige platforme såsom Windows, Linux, Unix, Mac OS X osv. Det kører også med succes på næsten alle servere, der bruges i dag, dvs. Apache, IIS osv. PHP understøtter en bred vifte af databaser inklusive MySQL.

## PHP-filformat ##

En enkelt PHP kan indeholde tekst, CSS, [HTML](/web/html/), JavaScript og PHP-kode. PHP-kode udføres på serveren, og resultatet returneres til browseren som almindelig HTML. PHP-filer kan åbnes med enhver teksteditor og redigeres på plads, selvom applikationer som Adobe Dreamweaver, Eclipse PHP Development-værktøjer tilbyder en bekvem måde at skrive og ændre PHP-kode på.

### Syntaks ###

Et PHP-script udføres på serveren, og det almindelige HTML-resultat sendes tilbage til browseren. Et PHP-script starter med<?php and ends with ?>

### PHP-eksempel ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
