## Hvad er INDD fil?

En fil med filtypen .indd er en Adobe InDesign-projektfil, der indeholder sidelayoutoplysningerne for projektet. INDD står for InDesign Document. Disse filer gemmer almindeligvis sideformateringsoplysninger, bøger, magasiner, aviser, flyers, brochurer eller andre trykte medier som mobil- og websidemodeller. INDD-filtypen er proprietært filformat af Adobe og er standardfilformatet for InDesign. Grafiske designere, journalister, marketingfolk og udgivere, der arbejder med InDesign, bruger denne filtype.

## Mere om INDD-filformat

INDD-filer er vektorfiler, der kan skaleres uden at miste opløsning. Disse er baseret på kompleks graflignende, der kan udvides uendeligt. Det pakker alle skrifttyper og layoutoplysninger, der skal deles med et team.

INDD-filer er perfekte til at designe engagerende print såvel som onlineindhold. En INDD-fil kan strække sig over flere sider og kan tjene som skabeloner til oprettelse af sider, kapitler og publikationer. Når dit projekt er færdigt, kan det publiceres på onilne medier uden behov for konvertering, zipning eller e-mail.

## Kan INDD-fil konverteres til andre formater?

INDD files can be converted to several other file formats. The most common conversion is INDD to PDF. InDesign lets you save the INDD file to PDF for sharing with other users. It also lets you export INDD file to JPG, [EPS](/page-description-language/eps/), [EPUB](/ebook/epub/), [SWF](/page-description-language/swf/), FLA, [HTML](/web/html/), [XML](/web/xml/), and IDML.

## Hvordan oprettes INDD-fil?

An Indd file is created when you create a new project with [Adobe InDesign](https://www.adobe.com/products/indesign.html) and save it to disc. You can open the Indd project file, make changes to it, and save the updated file for the changes.
