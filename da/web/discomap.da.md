## Hvad er en DISCOMAP fil?

En DISCOMAP-fil er en outputfil, der er oprettet af Microsoft Web Services Discovery Tool, **Disco.exe**. Den indeholder de resultater, som er ASP.NET Web Services opdaget på en bestemt URL. Almindelige eksempler på sådanne opdagede tjenester omfatter placeringer af .DISCO- eller .WSDL-filer. Når Microsoft Web Discovery-værktøjet kører, søger det efter at finde oplysninger baseret på XML-indholdet i DISCO-filen. Resultaterne gemmes i DISCOMAP-filen, der kan åbnes med Microsoft Visual Studio IDE.
