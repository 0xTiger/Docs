## Hvad er en XHTM fil?

An file with .xhtm extension is a webpage file that is written in XHTML. It uses XML 1.0 as its standard implementation and is considered a reformulation of HTML 4. Ligesom XHTML indeholder en XHTM-fil standard HTML-tags, der følger strengere XML-retningslinjer. XHTM-filer bruges til at udgive webstedsindhold og bruger almindeligvis udvidelsen [XHTML](/web/xhtml/).

Du kan bruge webbrowsere som Google Chrome, Apple Safari og Mozilla Firefox til at åbne XHTM-filer.

## XHTM eksempel

Følgende eksempel på et XHTM-dokument definerer XML-deklarationerne.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
