## Hvad er en WEBSTED fil?

A WEBSITE file is a website shortcut file created with Microsoft Internet Explorer (IE). It contains a URL and an icon (ICO) file that points to the website's favicon. The icon is used for displaying the shortcut. WEBSITE file format was introduced with the release of IE version 9. Tidligere versioner af IE bruges til at gemme genveje til websider som [.url](/web/url/) filer.

## WEBSTED Filformat

En .website-fil gemmes på disken som almindelige tekstfiler, hvis indhold kan ses i enhver teksteditor. Udover andet indhold indeholder den den mål-URL, som den er en genvej til. Hvis du ønsker at åbne målwebstedet i en anden browser end IE, kan du kopiere denne URL og indsætte den i andre browsere såsom Google Chrome, Safari eller Firefox for at se webstedet.

## Hvordan opretter man .WEBSITE-fil?

En WEBSTED-fil kan oprettes i IE ved at klikke på URL'en i adresselinjen og trække den til den ønskede placering på din computer, såsom skrivebordet eller dokumenter. Dette opretter genvejen til den pågældende internetplacering og kan åbnes ved at dobbeltklikke på genvejen. Hvis du vil gemme genvejen som `URL`-fil, skal du holde SHIFT-tasten nede, før du trækker destinationsstedets favicon til skrivebordet.
