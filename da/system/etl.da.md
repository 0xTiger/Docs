## Hvad er en ETL fil?

ETL-filer er logfiler, der indeholder hændelseslogfiler genereret af Microsoft Operating System Kernel. Disse logfiler inkluderer applikations- og systemfejl, advarsler og andre hændelsesdata. ETL-filer er nyttige til at analysere og fejlfinde problemer på systemniveau. Almindelige problemer registreret i ETL-filer inkluderer dem, der genereres af diskadgange og sidefejl. Microsoft leverer to applikationer, Tracerpt og Event Viewer til at læse og se indholdet af ETL-filer. ETL-filer kan konverteres til andre filformater såsom [TXT](/word-processing/txt/) og [CSV](/spreadsheet/csv/) ved hjælp af Tracerpt.

## ETL filformat

ETL-filer gemmes på disk i komprimeret binært format for at reducere diskplads.

## Åbn ETL-filer ved hjælp af Windows Performance Analyzer

ETL-fildata kan læses og visualiseres i tabel- såvel som grafisk format ved hjælp af Microsoft Windows Performance Analyzer (WPA)-applikationen. Guiden [Opening and Analyzing ETL files](https://learn.microsoft.com/en-us/windows-hardware/test/wpt/opening-and-analyzing-etl-files-in-wpa) giver oplysninger om at arbejde med ETL-filer.
