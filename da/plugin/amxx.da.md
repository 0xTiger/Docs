## Hvad er AMXX fil?

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. Det er en script-platform, der giver serveradministratorer mulighed for at tilpasse og udvide funktionaliteten af deres servere. AMXX bruges sammen med MetaMod plugin management system, der tillader flere plugins at køre samtidigt på en server uden konflikter. Funktioner, der tilbydes af AMXX-plugin, inkluderer admin-kommandoer, gameplay-modifikationer og serverstyring.
