## Hvad er en ART fil?

An ART file is an embroidery design file created with the [BERNINA](https://www.bernina.com/en-US/Home-United-States) ARTlink embroidery software version 1-4. Den indeholder brugerskabte designs, der kan omfatte forskellige objekter såsom konturer, designlayout, sømme, farvede tråde og andre metadata. ART-filen indlæses i BERNINA symaskine og kan sys til forskellige typer stoffer såsom quilt, skjorte eller enhver anden klud. ART-filer kan også deles med andre brugere ved hjælp af lagerenheder såsom USB eller overføres via internettet.

## ART filformat

ART-filformatet indeholder følgende oplysninger:

 * Objektegenskaber
 * Sømdata
 * Stingtyper
 * Stingkoordinater
 * Tråd farver
 * Kommentarer
 * Træk kompensationsindstillinger
 * Indstillinger for automatisk afstand
