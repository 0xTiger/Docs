## Ce este un fișier WBFS?

Un fișier WBFS (Wii Backup File System) este o imagine de disc de joc video creată pentru Nintendo Wii. Conține un joc Wii care poate fi încărcat ca imagine pentru joc. Este generat folosind instrumentul Wiimms WBPS sau Wii Backup Manager și poate fi jucat pe un Nintendo Wii prin încărcare de la un USB Loader GX sau un emulator Wii, cum ar fi Dolphin. Jocul este stocat pe aceste dispozitive de stocare externe folosind formatul de fișier WBFS. Wii este o platformă de jocuri distribuită de Nintendo. Sistemul de fișiere WBFS a fost dezvoltat de către Wii homebrew coder Waninkoko.

## Format de fișier WBFS - Mai multe informații

WBFS sunt fișiere binare care au format de fișier proprietar. Specificațiile sale de format de fișier nu sunt disponibile public.
