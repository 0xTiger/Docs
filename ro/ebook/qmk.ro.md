## Ce este un fișier QMK?

Un fișier cu extensia .qmk este, în general, cunoscut sub numele de formatul de fișier **YanCEyDesktop Quickmarks**. YanCEyWare a introdus formatul de fișier YanCEyDesktop Quickmarks. Fișierul QMK este legat de fișierul [YBK](/ro/ebook/ybk/) și este acceptat de un program care este disponibil pentru mașinile care rulează sistemul de operare Windows. Un fișier cu o extensie QMK este clasificat ca un fișier de tip text, care permite utilizatorilor să navigheze cu ușurință în locațiile salvate din cărți electronice. YanCEyDesktop este software-ul recomandat pentru gestionarea fișierelor QMK. Software-ul **YanCEyDesktop** a fost realizat de **D Remington**.

## Probleme la deschiderea unui fișier QMK

Dacă nu puteți deschide și rula fișierul QMK; nu înseamnă că nu aveți software adecvat instalat pe dispozitiv. Ar putea exista și alte probleme care împiedică fișierul să funcționeze corect. Posibilele probleme ar putea fi una dintre următoarele:

- Coruperea unui fișier QMK
- Legături greșite către fișierul QMK în intrările de registru
- Descrierea ștearsă a QMK din registrul Windows
- Instalarea coruptă a unei aplicații care acceptă formatul QMK
- Un fișier QMK infectat cu programe malware nedorite
- Computerul nu are resurse hardware suficiente pentru a funcționa cu fișierul QMK
- Driverele folosite de computer pentru a deschide un fișier QMK sunt învechite
