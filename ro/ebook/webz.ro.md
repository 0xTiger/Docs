## Ce este un fișier WEBZ? ##

O extensie de fișier WEBZ este un format de fișier foarte popular și este complet cunoscut sub numele de fișier electronic comprimat WEBZ. Sufixul EBZ este utilizat în general pentru fișierele în format eBook comprimat WEBZ și este folosit în mod fundamental ca fișier ZIP de pagini Web și imagini. Web Books Publishing este autorul și delimitează calitatea formatului WEBZ. Acest format de fișier este însoțit de software care poate fi conectat la o platformă de sistem Web. Fișierul WEBZ trece la tipul de fișiere text, la fel ca alte extensii de nume de fișier. Web Books Viewer este cel mai folosit program pentru lucrul cu astfel de formate de fișiere. Această extensie de fișier este flexibilă și poate fi schimbată din „.webz” în „.zip”.
