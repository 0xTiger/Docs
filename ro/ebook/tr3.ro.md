## Ce este un fișier TR3? ##

Fișierul TR3 este o carte electronică TomeRaider 3 care oferă suport pentru căutare și compresie rapidă. Acesta poate funcționa corect prin programul asociat TomeRaider. Yadabyte este dezvoltatorul acestui software, o companie britanică de software și dezvoltare web cu sediul în Marea Britanie. Aplicația TomeRaider eBook reader poate fi utilizată pentru funcționarea corectă și gestionarea conținutului acestor fișiere TR3. Acest cititor de cărți electronice asociat poate fi instalat pe computere care rulează pe sisteme bazate pe Microsoft Windows și pe multe gadget-uri transferabile. Fișierul TR3 aparține grupului de fișiere eBook utilizate în sisteme operaționale precum Windows 10, Windows 7, Windows 8 / 8.1, Windows Vista, Windows XP.

**Etichetele HTML** specifice TR3 sunt următoarele:

|Etichetă|Descriere|
---|---|
|\<new> |Pentru a crea fișiere TomeRaider din fișiere text, eticheta Nouă este tot ce este necesar.<new> tag definește începutul paginilor.|
|\<CATDEF> ... \</CATDEF> |definește numele categoriei|
|\<META> ... \</META> |este o etichetă folosită pentru a defini toate metadatele utilizate în formatul de fișier. Această etichetă include o serie de parametri.|
|\<EXPMSG> |Această etichetă controlează mesajul care apare când un fișier a expirat. Ori de câte ori un utilizator încearcă să vizualizeze o pagină după ce fișierul a expirat, apare mesajul de expirare în locul textului real al paginii.|
|\<DIEMSG> |Această etichetă este similară cu EXPMSG. Este folosit pentru a afișa un mesaj când fișierul moare.|
|\<ENGMSG> |Această etichetă este utilizată pentru a crea mesajul care se afișează pentru paginile criptate. Mesajul apare în locul textului paginii dacă utilizatorul încearcă să deschidă o pagină criptată înainte de a o debloca.|
|\<expmsg> ,\<diemsg> și \<encmsg> |Ignorat în textul paginii. Ele trebuie plasate în secțiunea antet fișierului înainte de prima<new> etichetă.|
|\<SELECTION> . \</ SELECTION> |Folosit pentru a susține interogările. Această etichetă trebuie să fie înaintea primei<new> etichetă. Compilează datele de selecție pentru interogarea utilizatorului.|
|\<catset> . \</catset> | Folosit pentru a atribui nume de categorii pentru fiecare subiect.|


## Probleme la deschiderea unui fișier TR3 ##

Iată lista unor probleme comune care pot apărea și pot cauza funcționarea defectuoasă a formatului de fișier:

* Fișier corupt
* Fișier infectat din cauza virusului
* Legături necorespunzătoare către fișierul TR3 în accesările arhivei
* Nu există drept de acces în sistem pentru a deschide fișierele
* Unitate învechită în sistemul dumneavoastră
* Eliminarea raportului TR3 din arhiva Windows
* Contactarea dezvoltatorului poate fi o opțiune mai bună pentru rezultate mai bune
