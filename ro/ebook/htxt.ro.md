## Ce este un fișier HTXT? ##

Formatul HTXT a fost conceput pentru prima dată de Hanvon și clasificat în tipul de fișier text. Este folosit în general pentru fișierele Hanvon eBook. Formatul Hanvon eBook este acceptat doar de Windows sau compatibil cu software-ul care poate fi instalat cu ușurință pe Windows. Cel mai comun software care furnizează fișiere HTXT este Transferul pe dispozitiv prin conexiune USB. Formatul de carte electronică folosit de dispozitivele Hanvon WISEreader eReader furnizează adesea text în chineză și este folosit pentru cărțile chineze, salvează într-un format dublu și poate integra siguranța DRM, nu poate fi citit de programele software Desktop, dar poate fi mutat pe dispozitiv prin conexiune USB.


## Problemă la deschiderea unui fișier HTXT? ##

Iată lista unor probleme relevante care pot apărea și care au fost dificil de deschis acest format de fișier:

* Fișier gol sau incomplet. Aceasta propune tăierea conținutului fișierului HTXT la descărcare sau copiere. Singura posibilitate de restaurare este să descărcați din nou fișierul
* Fișierul poate fi corupt sau viruși în el. Cea mai bună opțiune este utilizarea unui program antivirus actualizat pentru a clarifica orice incertitudine
* O defecțiune distinctivă a legăturii. Programul nu furnizează versiunea învechită a fișierului HTXT. Cea mai ușoară soluție la această eroare este sub forma unui program de adaptare sau a unei versiuni mai vechi a programului
