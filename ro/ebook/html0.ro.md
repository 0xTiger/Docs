## Ce este un fișier HTML0? ##

HTML0 este formatul nativ de carte electronică folosit de **Book Designer**, având cod HTML și CSS standard care descrie conținutul și designul fiecărei pagini. **Book Designer** este cel mai comun program care funcționează cu fișiere HTML0 și pentru a schimba documente text în numeroase formate de cărți electronice. Formatul de fișier HTML0 acceptă software-ul care poate fi instalat pe sistemul Windows, deoarece aparține categoriei de fișiere text. Fișierele HTML0 cuprind HTML normal și pot fi văzute și modificate folosind un editor de text. Fișierele HTML0 nu pot fi conectate la fișiere CSS discrete, în comparație cu fișierele HTML obișnuite.

## Problemă la deschiderea unui fișier HTML0? ##

* Fișierul poate fi infectat atunci când vă este dificil să îl deschideți
* Trebuie să scaneze fișierul pentru viruși printr-un scaner antivirus online
* Unele fișiere necesită drepturi de acces pentru a le deschide
* Dacă sistemele au resurse inadecvate pentru a deschide fișiere HTML0, atunci închideți toate aplicațiile și apoi redeschideți-le
* Fișierul poate fi corupt sau poate avea vreo eroare dacă l-ați descărcat sau l-ați primit de la orice e-mail
* Utilizați întotdeauna cel mai recent sistem și drivere pentru a vă face computerul mai sigur și pentru a rezolva toate problemele legate de extensiile de fișiere
