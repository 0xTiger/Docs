## Ce este un fișier AEP? ##

Fișierul cu extensia .aep (care înseamnă **Activ E-Book Project**) este clasificat ca o carte electronică (eBook). Fișierele AEP sunt create folosind același motor ca și cel al Internet Explorer pentru a asigura compatibilitate 100%. Conține paginile așa cum o persoană poate citi o carte fizică. Paginile fișierelor AEP includ de obicei textul, imaginile, animațiile și hyperlinkurile. De asemenea, acceptă [HTML](/ro/web/html/), [DHTML](/ro/web/dhtml/), [JPEG](/ro/image/jpeg/) și [PNG](/ro/image/png/) grafică, animații , [JavaScript](/ro/web/js/), VBScript și multe dintre pluginurile Internet Explorer. Un fișier AEP poate fi salvat ca fișier executabil Windows (EXE), astfel încât să poată fi deschis pe Windows95 sau versiuni ulterioare ale Microsoft Windows.
 


## Format de fișier AEP ##

Fișierele în format AEP sunt create utilizând un software AEC (Active E-Book Compiler). Acest software folosește un motor similar cu acel IE pentru a crea cărți electronice. Utilizatorul poate crea copii nelimitate ale cărților electronice fără a plăti niciun cost pentru utilizarea programului. Procesul de distribuție este, de asemenea, ușor, deoarece fiecare carte electronică este compusă dintr-un program Windows autonom, care poate fi distribuit pe orice suport, cum ar fi e-mailuri, descărcare de pe Internet, dischetă, CD-ROM-uri, discuri ZIP etc.

## Posibile probleme la deschiderea fișierului

Dacă nu puteți deschide și rula fișierul AEP; nu înseamnă că nu aveți un software adecvat instalat pe dispozitiv. Ar putea exista și alte probleme care împiedică fișierul să funcționeze corect. Posibilele probleme ar putea fi una dintre următoarele:

- Coruperea unui fișier AEP
- Legături greșite către fișierul AEP în intrările de registru
- Descrierea ștearsă a AEP din registrul Windows
- Instalarea coruptă a unei aplicații care acceptă formatul AEP
- Un fișier AEP infectat cu un malware nedorit
- Computerul nu are resurse hardware suficiente pentru a funcționa cu fișierul AEP
- Driverele folosite de computer pentru a deschide un fișier AEP sunt învechite
