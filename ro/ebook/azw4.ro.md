## Ce este un fișier AZW4?

AZW4 este o carte electronică Amazon Print Replica pe care Amazon a dezvoltat-o pentru software-ul desktop Kindle și dispozitivele Kindle eReader. AZW4 nu este similar cu [MOBI](/ro/ebook/mobi/), [AZW](/ro/ebook/azw/) sau [AZW3](/ro/ebook/azw3/). Acest format este de fapt un înveliș pentru cărți electronice [PDF](/ro/pdf/) sau MOBI cu DRM adăugat.

## Format de fișier AZW4

**AZW4** a fost introdus ca o nouă versiune a software-ului **Textbooks**, care este similară cu **PDF**. O carte electronică AZW4 conține un fișier PDF complet în interior sau poate fi considerată un fișier PDF cu un înveliș numit AZW4. În primul rând, AZW4 este folosit de Kindle eReader. Cu toate acestea, alte aplicații pot folosi și extensia de fișier .azw4. Chiar și programele malware pot crea fișiere .azw4. Fiți precaut cu fișierele .azw4 care provin dintr-o sursă îndoielnică.
