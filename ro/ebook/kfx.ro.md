## Ce este un fișier KFX?

Un fișier KFX este o carte electronică care este creată în Amazon Kindle File Format 10 (KF10) pentru dispozitivele Kindle. Versiunile mai vechi ale dispozitivelor KIndle folosesc formatele de fișier [AZW](/ro/ebook/azw/) și [AZW3](/ro/ebook/azw3/) (KF8). KFX este o îmbunătățire a versiunilor anterioare cu caracteristici suplimentare, în special în ceea ce privește prezentarea aspectului conținutului. Introduce noi funcții, cum ar fi suportul pentru cratimele soft, folosind un nou motor de compoziție îmbunătățit. De asemenea, introduce un nou format de fișier imagine numit JXR, care are un raport de compresie mai mare și noul font Bookerly, care este font exclusiv pentru Kindle, oferind o lizibilitate îmbunătățită pe dispozitivele digitale. Formatul vine cu capabilități îmbunătățite, cum ar fi miniaturi cu mai multe pagini, ajustare automată a conținutului în jurul imaginilor și videoclipurilor în locații fixe, ceea ce duce la îmbunătățirea aspectului paginii și DRM. Pentru a deschide și a vizualiza astfel de fișiere, puteți utiliza aplicația Amazon Kinde, care este disponibilă pentru Windows, macOS, Android și iOS. Caliber este o altă aplicație software de citire de cărți electronice care poate fi folosită pentru a citi mai multe formate diferite de fișiere de cărți electronice.

## Format de fișier KFX ##

KFX este un format complex, deoarece combină mai multe caracteristici diferite ale Formatelor de fișiere Kindle, dar detaliile interne despre structura fișierelor sale nu sunt cunoscute. Amazon și-a adăugat schema DRM pentru a proteja drepturile digitale și pentru a salva formatul KFX de anti-piraterie.
