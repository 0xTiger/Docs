## Ce este un fișier YBK? ##

Formatul de fișier YanCEyWare eBook și Microsoft Encarta folosesc ambele extensii de fișiere YBK. YanCEyWare eBook a fost creat de YanCEyWare. Toate aplicațiile software sunt disponibile pentru dispozitivele care rulează Windows, acceptă fișiere YBK. Enciclopedia Encarta poate fi actualizată prin fișiere .ybk fără a fi nevoie să actualizați întregul program. YanCEyDesktop este un program binecunoscut și popular care poate gestiona și gestiona cu ușurință fișierele YBK. YanCEyWare Reader utilizează pe scară largă acest format de fișier YBK, ca aplicație de citire de cărți electronice pentru Windows Mobile sau Pocket PC. Acest fișier a fost dezvoltat cu resurse mari hiper-linkate.

## Probleme la deschiderea unui fișier YBK ##

Uneori, când este dificil să deschideți și să funcționați fișierul YBK, iată o listă cu câteva probleme posibile:

* Deschiderea unui fișier YBK corupt
* Legături inadecvate și necorespunzătoare către fișierul YBK în accesările de registru
* Eliminarea neintenționată a descrierii YBK din arhiva Windows
* Instalarea parțială a unei aplicații care furnizează formatul YBK
*	Infected YBK file with an undesirable malware
