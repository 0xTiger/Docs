## Ce este un fișier EBK?
O extensie EBK este utilizată de software-ul **eBook Pro Viewer** atunci când salvează un fișier eBook. eBook Pro este un software care poate crea cărți electronice și poate proteja informațiile protejate prin drepturi de autor. Un fișier EBK conține versiunea criptată și comprimată a unei cărți electronice care poate fi citită folosind **eBook Pro Viewer**; după furnizarea acreditărilor corespunzătoare pentru autentificare.
