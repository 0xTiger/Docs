## Ce este un fișier FKB? ##

FKB este o extensie de fișier eBook care a fost dezvoltată de Flipkart.com pentru aplicația software Flipkart. Acest tip de extensie de fișiere se găsește de obicei la utilizatorii Windows 10. Există o specificație în fișierul FBK că dimensiunea poate varia de la câteva sute de kiloocteți la mulți megaocteți, în funcție de lungimea cărții și de numărul de imagini incluse. Pentru a descărca fișierul din aplicația Android Flipkart eBooks, trebuie mai întâi să cumpărați cartea electronică, apoi faceți clic pe ea în Bibliotecă și cartea va începe descărcarea. Dacă nu puteți vedea documentul achiziționat, reîmprospătați sistemul pentru o funcționare mai bună.

## Sisteme de operare acceptate ##

Următoarele sisteme de operare acceptă fișiere FKB:

* Windows 7
* Windows 8
* Windows 10
* Windows Server 2003/2008/2012/2016
* Mac OS X
* FreeBSD
* iOS
* Linux
* Android
* NetBSD
* OpenBSD

## Probleme la deschiderea unui fișier FKB ##

* Instalarea necorespunzătoare a programelor suport. Asigurați-vă că este descărcat corect cu o versiune actualizată
* Spațiu hardware redus
* Fișiere corupte sau infectate. Această problemă poate fi rezolvată cu ușurință printr-un antivirus instalat în sistemul dumneavoastră
