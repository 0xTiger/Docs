## Ce este un fișier TR? ##

Un fișier cu extensia .tr este fișierul TomeRaider eBook care este creat cu software-ul TomeRaider eReader. Aceste fișiere pot fi deschise atât pe dispozitive Windows, cât și pe dispozitive portabile, cum ar fi Android, Palm OS și EPOC, și sunt clasificate ca fișiere text. Software-ul TomeRaider dezvoltat de site-ul web [Yadabyte](https://www.yottabyte.ltd/) și site-ul oficial oferă mai multe informații despre fișierele TR. Formatul de fișier TR2 a fost înlocuit cu formatul TR3 (TomeRaider 3). Fișierele TR oferă căutare rapidă și niveluri ridicate de compresie.

## Scurt istoric al formatului de fișier TR

TomeRaider a fost dezvoltat de o companie de dezvoltare web și software Yadabyte din Marea Britanie în 1999. Compania a dus-o mai departe la TomeRaider versiunea 3.

## Format de fișier TR

Cea mai recentă versiune a formatului de fișier TR este versiunea 3, care oferă suport pentru căutare și comprimare rapidă. Comprimă fișierele text la 45-60% din dimensiunea originală. Acest lucru le permite cititorilor de cărți electronice TomeRaider să efectueze căutare, indexare și compresie rapidă. Formatul de fișier TR este potrivit pentru documente de referință mari și majoritatea documentelor necesită o cheie DRM pentru a fi criptate.

## Probleme la deschiderea unui fișier TR

Pot exista unele probleme comune din cauza cărora nu puteți deschide fișierul:

* Absența software-ului suport
* Versiune software învechită
* Fișier TR corupt
* Legături necorespunzătoare către fișierul TR în accesările arhivei
* Nu există drept de acces în sistem pentru a deschide fișierele
* Unitate învechită în sistemul dumneavoastră
* Eliminarea raportului TR din arhiva Windows
* Instalarea parțială a unei aplicații care acceptă formatul TR
* Fișier TR infectat
* Spațiu insuficient pentru resurse hardware pentru a ajuta la deschiderea fișierului TR
