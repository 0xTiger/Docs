## Ce este un fișier SVGZ?

Un fișier cu extensia .svgz este o versiune comprimată a fișierului Scalable Vector Graphics (.SVG). Este comprimat cu compresie gzip și conține date în format [XML](/ro/web/xml/). Fișierele SVGZ acceptă transparență, gradienți, animații și filtre. Fișierele SVGZ au dimensiuni mai mici în comparație cu fișierele SVG implicite și această dimensiune redusă a fișierului ajută la transferul fișierelor grafice online. Designerul grafic creează fișiere SVGZ folosind instrumente precum Adobe Illustrator, Corel PaintShop Pro și altele. Cu toate acestea, fișierele SVGZ pot fi generate activând [GZip compression](/ro/compression/gz/) în serverul Apache în timp ce se trimit datele imaginii.

## Format de fișier SVGZ

SVGZ, fiind comprimat [SVG](/ro/page-description-language/svg/), se bazează pe formatul deschis gzip comprimat care reduce dimensiunea documentelor SVG. Această arhivă comprimată conține documente SVG cu text simplu, care se bazează exclusiv pe formatul de fișier XML (/ro/web/xml/). Fișierele SVGZ pot fi deschise de către vizualizatori/aplicații care sunt conforme cu specificațiile SVG 1.1. Fișierele SVGZ cu 20-50% mai mici decât fișierele SVG originale.
