## Ce este un fișier CLIP?

Un fișier CLIP este un fișier imagine raster generat de aplicația de desen 2D Clip Studio Paint. Este similar cu fișierul [PSD](/ro/image/psd/) al Adobe și conține mai multe straturi pentru a gestiona diferite elemente grafice sub formă de straturi, cum ar fi text, imagini, creion, creion, pensule și forme. Formatul de fișier CLIP este utilizat pentru a proiecta un număr de tipuri diferite de grafice, cum ar fi ilustrații și animații. Aplicația Clip Studio Paint a fost dezvoltată de o companie japoneză CELSYS, Inc. care a fost înființată în mai 1991.

## Format fișier CLIP - Mai multe informații

Formatul de fișier CLIP a fost introdus de CELSYS pentru a înlocui formatul original de fișier LIP. Stochează elemente grafice sub formă de straturi care pot fi editate pentru modificări. Este, de asemenea, formatul de fișier implicit pentru aplicația [CLIP Studio Paint](https://www.celsys.co.jp/en/service/studio_paint/), care este disponibilă pentru platformele Windows, MacOS, Android și iOS.

## Tipuri de desen acceptate

Formatul de fișier CLIP poate stoca animații și ilustrații care pot fi utilizate într-o varietate de tipuri diferite de lucrări.
