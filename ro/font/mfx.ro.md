## Ce este un fișier MFX?

Un fișier MFX este un fișier de font utilizat de jocul de simulare Maxis SimCity 4. Este folosit pentru a reda textul jocului pe ecran dintr-unul dintre tipurile de font selectate, inclusiv Arial, Courier sau orice alt font disponibil în fișierul de font. Fișierele MFX sunt instalate împreună cu jocul SimCity 4 și nu trebuie create manual.

SimCity 4 este un joc de simulare pe computer pentru a crea simulări de construcție a orașului. Maxis, dezvoltatorul SimCity, este o subsidiară a Electronic Arts. Jocul a fost lansat inițial în ianuarie 2003.
