## Ce este un fișier JFPROJ?

Un fișier cu extensia .jfproj este un font personalizat bazat pe JSON, creat cu aplicația Microsoft Font Maker pe Windows 10. Acestea pot fi create folosind un stilou electronic cu dispozitive cu ecran tactil, cum ar fi tablete, deși alte periferice ale computerului, cum ar fi mouse-ul pot fi, de asemenea, utilizate în cazul în care a calculatoarelor. Fișierele JFPROJ conțin glifele care sunt folosite pentru a reda caracterele pe ecran prin definirea distanței dintre caractere și cuvinte și dimensiunea fontului. Aplicația Microsoft Font Maker este disponibilă pentru descărcare în magazinul Windows.
