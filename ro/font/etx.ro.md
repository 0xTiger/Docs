## Ce este un fișier ETX?

Un fișier ETX este un fișier cu fonturi utilizat de sistemele de tipărire, TeX și LaTeX. Este salvat ca fișier text și definește aspectul unui font prin codificarea acestuia. Fișierele cu fonturi ETX sunt încărcate în software care oferă capacitatea de a lucra cu documente Tex. Odată încărcate, fișierele de font ETX pot fi folosite pentru personalizarea documentelor TeX.
