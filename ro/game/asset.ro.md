## Ce este un fișier ASSET?

Un fișier ASSET este un fișier special folosit în Unity, un program pentru realizarea de jocuri video. Deține configurații importante, cum ar fi obiecte de joc sau setări. De exemplu, un fișier ASSET ar putea stoca modul în care obiectele se mișcă în joc, cum ar fi modul în care acestea sară unele de altele sau cad. Aceste fișiere sunt ca niște containere care păstrează toate informațiile importante organizate, astfel încât dezvoltatorii de jocuri să poată lucra cu ușurință la proiectele lor.

## Cum se deschide un fișier ASSET
  
În Unity, de obicei nu deschideți” fișierele de active în același mod în care ați deschide un document sau un fișier într-un editor de text sau un program similar. În schimb, Unity gestionează fișierele de active în cadrul Editorului Unity însuși. Iată cum puteți accesa și lucra cu fișiere de active în Unity:

1.  **Import Assets**: To use an asset file in your Unity project, you need to import it into project. You can do this by either dragging and dropping asset file directly into Unity Editor or by using "Assets" menu and selecting "Import New Asset."
    
2.  **Vizualizarea activelor**: Odată importate, puteți vizualiza și gestiona fișierele cu materiale în fereastra Proiect” a Editorului Unity. Această fereastră afișează toate activele din proiectul dvs. și vă permite să le organizați în dosare, să căutați anumite active și să previzualizați conținutul acestora.
    
3.  **Folosirea elementelor**: puteți utiliza fișiere cu materiale în proiectul dvs. Unity trăgându-le din fereastra Proiect” în ierarhia scenei sau pe GameObjects din scenă. De exemplu, puteți trage un material de textură pe un material pentru a-l aplica unui obiect 3D sau puteți trage un material prefabricat în scenă pentru a instanția un GameObject preconfigurat.
    
4.  **Editarea elementelor**: unele fișiere de materiale, cum ar fi scripturile, materialele și animațiile, pot fi editate direct în Editorul Unity. Puteți face dublu clic pe aceste active în fereastra Proiect” pentru a le deschide în editorii lor respectivi și pentru a face modificări după cum este necesar.
    
5.  **Salvarea elementelor**: orice modificări pe care le faci fișierelor cu materiale în Editorul Unity sunt salvate automat în proiectul tău Unity. Nu este nevoie să salvați în mod explicit fișierele de active individuale, așa cum ați face cu un document într-un program software tradițional.
