## Ce este un fișier KODU?

Un fișier .kudo este un fișier de joc 3D creat și salvat cu Microsoft [software Kodu Game Lab](https://www.kodugamelab.com/), care este un mediu de dezvoltare a jocurilor educaționale. Este conceput pentru a învăța copiii principiile de bază de programare și le permite creatorilor să programeze comportamentele elementelor de joc folosind limbajul de programare vizual. Fișierele Kodu pot fi salvate și partajate cu alți utilizatori, dar pot fi deschise și rulate numai în software-ul Kodu.

## Format de fișier KODU - Mai multe informații

Fișierele Kodu sunt fișiere binare, salvate în format de fișier proprietar Microsoft. Specificațiile sale de format de fișier nu sunt disponibile public pentru dezvoltatori și de aceea fișierele Kodu pot fi rulate numai în software-ul Kodu Game Lab.
