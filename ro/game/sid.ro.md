## Ce este un fișier SID?

Un fișier SID este un fișier de rezervă a jocului creat de aplicația Steam care este utilizat pentru descărcarea și gestionarea **jocuri video Valve** pe sistemul de operare Windows. Stochează datele jocului în format comprimat și pot fi folosite jocuri de rezervă pe medii de stocare precum DVD-uri și dispozitive de stocare USB. Fișierul SID este creat cu utilitarul de backup Steam și este similar cu formatul de fișier [CSD](/ro/game/csd/) pentru stocarea backup-ului jocului.

## Format de fișier SID - Mai multe informații

Fișierele SID sunt stocate pe disc ca fișiere binare. Este întotdeauna însoțit de un fișier .sis și un fișier .sim care sunt utilizate pentru a extrage informații despre metadate pentru restaurarea datelor de joc de rezervă.
