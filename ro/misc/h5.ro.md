## Ce este un fișier H5?

Un H5 este unul dintre formatele de date ierarhice (HDF) utilizate pentru a stoca cantități mari de date. Este folosit pentru a stoca cantități mari de date sub formă de tablouri multidimensionale. Formatul este folosit în primul rând pentru a stoca date științifice bine organizate pentru regăsire și analiză rapidă. H5 a fost introdus ca un format de fișier mai îmbunătățit pentru H4. A fost dezvoltat inițial de Centrul Național pentru Aplicații de Supercomputing, iar acum este susținut de The HDF Group.

## Scurt istoric al formatului de fișier HDF

Formatul de fișier HDF a fost selectat de NASA ca metodă standard pentru stocarea datelor științifice. Înainte de aceasta, formularea unui astfel de standard a început în 1987 de către Graphics Foundations Task Force (GFTF) și HDF a fost aprobată oficial după investigarea a 15 formate de fișiere diferite.

## Format de fișier HDF5

Fișierele H5 sunt în format ierarhic de date care aderă la specificațiile de format de fișier HDF5. Aceste specificații stabilesc structura generală a fișierului HDF5 pentru stocare pe disc, dar utilizatorii finali nu au nevoie de aceste detalii subiacente.

Fișierele H5 sunt utilizate în mod obișnuit în domeniul aerospațial, fizică, inginerie, finanțe, cercetare academică, genomică, astronomie, instrumente electronice și domenii medicale.

### Model de date HDF5

Fișierul HDF5 este ca un container care cuprinde obiecte de date eterogene. De exemplu, aceste seturi de date pot fi imagini, tabele, elemente de text, grafice și chiar documente precum [PDF](/ro/pdf/) și [documente Excel](/ro/spreadsheet/).

## H5 File Viewer

Grupul HDF oferă [software HDFView](https://www.hdfgroup.org/solutions/hdf5/) care constă din utilitarul HDFView și pachetul de obiecte Java HDF.
