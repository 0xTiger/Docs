## Ce este un fișier XMZX?

Un fișier XMZX este un fișier de test de examen pentru efectuarea testelor și este utilizat de aplicația software ExamSoft SofTest. Aplicația poate fi folosită atât pe Windows, cât și pe macOS. Conține întrebări dintr-un examen pe care examinatorul le poate lua de la studenți pe un computer cu software-ul SoftTest. Fișierele XMZX sunt trimise examinatorilor pentru efectuarea testului. Este încărcat în software-ul SoftTest și candidatul sau solicitantul susține testul. La finalizarea testului, fișierul XMDX finalizat este încărcat automat pe serverele cloud ale ExamSoft.

## Format de fișier XMZX - Mai multe informații

Fișierele XMZX sunt utilizate de platforma digitală de evaluare, ExamSoft, care este folosită de o varietate de organizații pentru a efectua teste pentru certificări. Platforma are propriile servere cloud online unde toate testele finalizate sunt încărcate automat. Testele se desfășoară fie la un centru de examinare, fie examinatorii pot susține și candidații formularul de testare pe propriile computere. ExamSoft este disponibil atât pe sistemele de operare Windows, cât și pe macOS. XSMX este un alt format de fișier de test de examen de la ExamSoft.

## Cum se deschide un fișier XMZX?

Puteți deschide un fișier XMZX în software-ul ExamSoft SofTest numai deoarece este un format de fișier proprietar. Software-ul SoftTest are opțiunea de a **Suspune un examen** care este utilizat pentru a începe examenul prin selectarea fișierului XMZX.
