## Ce este un fișier PDO?

Pepakura Designer este un program care permite utilizatorilor să creeze modele artizanale din hârtie din modele 3D; Fișierele Pepakura au de obicei o extensie .pdo; aceste fișiere conțin instrucțiuni și informații de aspect necesare pentru a tipări și a asambla un model de hârtie.

Fișierele Pepakura Designer sunt într-un format proprietar specific software-ului Pepakura; formatul de fișier este conceput pentru a stoca informații despre modelul 3D, inclusiv geometria acestuia, referințele de textură și instrucțiunile de asamblare.

Pepakura Designer lucrează cu modele 3D în formate precum [.obj](/3d/obj/), [.3ds](/3d/3ds/), [.dxf](/cad/dxf/) și altele; aceste modele sunt apoi desfășurate în modele 2D care pot fi imprimate și asamblate într-un model de hârtie final.

Fișierul .pdo include instrucțiuni despre cum să tăiați, să pliați și să lipiți bucăți de hârtie imprimată pentru a recrea modelul 3D; fiecare piesa este etichetata si numerotata pentru a ghida procesul de asamblare.

## Cum se deschide un fișier PDO?

Pentru a deschide și vizualiza fișierele Pepakura Designer, veți avea nevoie de **Pepakura Viewer**, un program însoțitor gratuit pentru **Pepakura Designer**. Vizualizatorul permite altora să vizualizeze și să imprime modele fără a avea nevoie de software de proiectare complet.

## Despre Pepakura Designer

Pepakura Designer este o aplicație software specializată, concepută pentru crearea de modele artizanale din hârtie din modele digitale 3D. Software-ul permite utilizatorilor să importe modele 3D în diferite formate, cum ar fi .obj sau .3ds, apoi desfășoară și aplatizează aceste modele în modele 2D care pot fi imprimate pe hârtie. Aceste modele, salvate cu o extensie de fișier .pdo, conțin instrucțiuni și marcaje pentru tăierea, plierea și asamblarea bucăților de hârtie într-un model tridimensional. Pepakura Designer este utilizat pe scară largă de pasionații și entuziaștii interesați să creeze modele complicate de hârtie ale diferitelor obiecte, personaje și modele.
