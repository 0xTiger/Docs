## Ce este un fișier OGG?

Un fișier OGG este un fișier grafic creat cu software-ul Origin. Oferă capacitatea de a stoca diferite tipuri de grafice pentru a afișa datele importate pentru analiza datelor. Tipurile de grafic acceptate de tipul de fișier OGG includ coloană, bară, plăcintă, cascadă, 3D, polar și simbol. Software-ul Origin a fost dezvoltat de OriginLab și permite utilizatorilor să efectueze analize de date și să genereze rezultate sub formă de grafice.

## Format de fișier OGG

Fișierele OGG conțin grafice pentru reprezentarea datelor importate. Când un fișier OGG este deschis, acesta afișează graficul într-o fereastră separată în Origin sau Origin Viewer, care este disponibil gratuit în comparație cu software-ul Origin. Informațiile despre metadate, cum ar fi titlul și culorile unui grafic, sunt, de asemenea, stocate în fișierul OGG.

## Despre software-ul OriginLab Origin

Software-ul Origin oferă, de asemenea, opțiunea de a fi utilizat ca server de automatizare. În acest caz, aplicațiile client, de exemplu, LabVIEW, Microsoft Excel, Python, sau aplicațiile personalizate dezvoltate în C#/VB/VC pot face schimb de date cu Origin. [Python_to_OriginLab](https://github.com/jsbangsund/python_to_originlab) este un API Python pentru automatizarea trimiterii datelor și a funcțiilor de bază de trasare în OriginLab.
