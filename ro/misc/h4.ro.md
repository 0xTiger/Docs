## Ce este un fișier H4?

Un H4 sau HDF4 este unul dintre formatele de date ierarhice (HDF) și este folosit pentru a stoca cantități mari de date. Este un format de fișier open source care acceptă stocarea de date complexe eterogene. HDF4 este un format de date mai vechi în comparație cu [HDF5](/ro/misc/h5/). Formatele de fișiere HDF sunt menținute și publicate de grupul HDF, o organizație non-profit. Fișierele HDF acceptă și stocarea metadatelor pentru a reprezenta complet datele stocate.

## Scurt istoric al formatului de fișier HDF

Formularea formatelor de date ierarhice a fost inițiată în 1987 de către Graphics Foundations Task Force (GFTF). În acest timp, au fost propuse mai multe formate de fișiere diferite pentru standardizarea unui singur format. NASA a selectat HDF ca metodă standard pentru stocarea și difuzarea datelor științifice.

## Format de fișier HDF4

Fișierele H4 sunt salvate în „Directorul de fișiere”, ca structură, conform specificațiilor de format de fișier HDF. Aceste specificații definesc structura generală a fișierelor HDF pentru stocare pe disc, dar fără ca utilizatorii să-și facă griji cu privire la structura fișierelor de bază.

Domeniile de aplicare ale fișierelor H4 includ aerospațial, fizică, inginerie, finanțe, cercetare academică, genomică, astronomie, instrumente electronice și domenii medicale.

### Model de date HDF4

Fișierul HDF4 sau H4 este ca un container care constă din obiecte de date eterogene. Tipurile de seturi de date acceptate de fișierele H4 includ:

* Imagini
* Mese
* Elemente de text
* Grafice
* Documente precum [PDF](/ro/pdf/) și [documente Excel](/ro/spreadsheet/)

## H4 File Viewer

Grupul HDF oferă [software HDFView](https://www.hdfgroup.org/solutions/hdf5/) care constă din utilitarul HDFView și pachetul de obiecte Java HDF.
