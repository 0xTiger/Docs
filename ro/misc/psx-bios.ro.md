## Ce este un PSX BIOS?

**PSX BIOS** este un sistem de intrare/ieșire de bază pentru **consola Sony PlayStation** originală. Este un set de programe de nivel scăzut care controlează hardware-ul PlayStation și oferă funcții esențiale pentru ca consola să funcționeze. BIOS-ul este responsabil pentru inițializarea și gestionarea componentelor sistemului, cum ar fi CPU, memoria, grafica și dispozitivele de intrare/ieșire.

## PSX BIOS - Mai multe informații

BIOS-ul PlayStation (PSX), cunoscut oficial ca **SCPH-1001**, este o componentă crucială a consolei originale Sony PlayStation. Iată câteva puncte cheie despre PSX BIOS:

1.  **Functionalitate**:

BIOS-ul PSX este un set de programe de nivel scăzut stocate pe un cip de memorie read-only (ROM) din hardware-ul PlayStation. Funcția sa principală este de a inițializa și controla componentele hardware esențiale ale consolei, cum ar fi CPU, memoria, unitatea de procesare grafică (GPU) și dispozitivele de intrare/ieșire.
    
2.  **Proces de pornire**:

Când porniți o consolă PlayStation, BIOS-ul este prima parte de software care este executată. Efectuează o serie de verificări și inițializează hardware-ul pentru a pregăti sistemul pentru încărcarea și rularea software-ului de joc.
    
3.  **Protecția drepturilor de autor**:

BIOS-ul PSX include mecanisme de protecție a drepturilor de autor care au fost concepute pentru a împiedica rularea copiilor neautorizate ale jocurilor pe consolă. Aceste protecții includ blocarea regiunii, ceea ce înseamnă că jocurile dintr-o regiune pot să nu funcționeze pe o consolă dintr-o regiune diferită.
    
4.  **Statut juridic**:

BIOS-ul PSX este un software proprietar deținut de Sony Computer Entertainment. Distribuirea sau utilizarea acestuia fără autorizația corespunzătoare este ilegală. De obicei, utilizatorilor li se cere să folosească BIOS-ul care vine cu propria lor consolă PlayStation și nu au voie să îl distribuie sau să îl descarce separat.
    
5.  **Emulare**:

În contextul emulatoarelor, care permit utilizatorilor să joace jocuri PlayStation pe un computer, obținerea și utilizarea PSX BIOS poate fi o zonă gri legală. Unii emulatori solicită utilizatorilor să furnizeze propriul lor BIOS, extras din consola lor personală PlayStation. Cu toate acestea, partajarea sau descărcarea BIOS-ului de pe internet fără permisiune este împotriva legii.
    
6.  **Fișiere BIOS**:

Fișierul PSX BIOS este de obicei numit **SCPH1001.BIN** sau o variantă similară. Este un fișier binar care conține codul de mașină necesar pentru funcțiile BIOS.
