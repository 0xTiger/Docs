## Ce este un fișier XSMX?

Un fișier XSMX este un fișier chestionar creat cu aplicația ExamSoft SofTest care este utilizat pentru a efectua examene. Este folosit pentru a trimite lucrarea de examen studenților pe care o susțin online și pentru a trimite rezultatul înapoi ca fișier XMDX. ExamSoft oferă soluții de evaluare bazate pe deciziile driverului de date prin tehnologia educațională, iar aplicația SofTest disponibilă atât pentru Windows, cât și pentru MacOS este una dintre acestea. Fișierul XSMX conține toate întrebările și este deschis în aplicația SofTest de pe computerul studentului.

## Format de fișier XSMX

Fișierele XSMX sunt salvate și trimise participanților la examen ca fișiere protejate cu parolă. Protecția prin parolă este menită să asigure securitatea și secretul întrebărilor de examen. Parola examenului este partajată cu utilizatorul final pentru deschiderea fișierului și susținerea testului.
