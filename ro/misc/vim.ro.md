## Ce este un fișier VIM?

Un fișier VIM este un fișier de configurare/setări utilizat de programul de editare de text cunoscut sub numele de Vim. Stochează setările editorului, cum ar fi evidențierea sintaxelor și sunt utilizate de dezvoltatorii de aplicații pentru a configura aspectul editorului atunci când este deschis. Editorul de text Vim este extrem de configurabil și este construit pentru crearea și schimbarea eficientă a textului. Vine la pachet ca „vi” cu majoritatea sistemelor UNIX.
