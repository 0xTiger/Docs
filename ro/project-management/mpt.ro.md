## Ce este un fișier MPT? ##

Un fișier cu extensia .mpt este fișierele șablon [Microsoft Project](https://products.office.com/en-us/project/project-and-portfolio-management-software). Acesta conține informații de bază și setări de structură și document pentru crearea fișierelor [MPP](/ro/project-management/mpp/). Aceste șabloane oferă setări implicite care sunt esențiale pentru un anumit proiect, cum ar fi programe sau informații despre buget. Deși, nu poate salva date specifice proiectului, cum ar fi sarcini, resurse sau sarcini. Odată modificate, fișierele șablon pot fi salvate ca fișiere MPP standard pentru a lucra în continuare cu ele. Un fișier șablon principal, intitulat „Global.mpt” este disponibil în Microsoft Project, deschis pentru a crea un șablon individual. Fișierele MPT folosesc *application/vnd.ms-project* și *application/x-project* ca tipuri standard MIME. Windows, Linux și macOS pot deschide toate fișierele MPT, împreună cu Moos Project Viewer, Microsoft Project și MPXJ.

## Probleme la deschiderea unui fișier MPT? ##

Iată lista unor probleme comune care pot apărea și pot cauza funcționarea defectuoasă a formatului MPT:

* Absența software-ului suport
* Fișier corupt
* Fișier infectat din cauza virusului
* Nu există drept de acces în sistem pentru a deschide fișierele
* Unitate învechită în sistemul dvs
 *   Extension of the file is renamed
