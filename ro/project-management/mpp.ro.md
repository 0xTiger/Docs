## Ce este un fișier MPP?

Un fișier cu extensia .mpp este un fișier de date Microsoft Project care stochează informații legate de managementul proiectelor într-o manieră integrată. Microsoft a dezvoltat acest format de fișier proprietar pentru a-l face compatibil cu Microsoft Project (MSP), produsul lor software de management de proiect. În afară de MPP, MSP poate lucra cu schema XML a proiectului. Multe aplicații și API-uri permit convertirea MPP-urilor în alte formate de fișiere. Microsoft are acum un server de proiect online unde fișierele de management de proiect pot fi încărcate pentru colaborare de mai mulți utilizatori.

## Format fișier MPP ##

Microsoft a dezvoltat și publicat MPP ca format de fișier binar, iar specificațiile sale nu au fost făcute publice de Microsoft până în prezent. Structura și detaliile interne ale fișierelor nu sunt, prin urmare, disponibile pentru dezvoltatori. Există mai multe API-uri disponibile online care acceptă citirea/scrierea în fișiere MPP, dar acuratețea de 100% nu este niciodată atinsă. Microsoft a introdus de-a lungul timpului mai multe caracteristici în aplicația sa de management de proiect, la fel și caracteristicile formatului de fișier MPP.

Fișierul MPP utilizează următoarele tipuri MIME:

* application/vnd.ms-project
* aplicație/msproj
* aplicație/msproject
* aplicație/x-msproject
* aplicație/x-ms-proiect
* application/x-dos_ms_project
* aplicație/mpp
* zz-application/zz-winassoc-mpp

## Probleme la deschiderea unui fișier MPP? ##

Iată lista unor probleme comune care pot apărea și pot cauza funcționarea defectuoasă a formatului MPP:

* Absența software-ului suport
* Fișier corupt
* Fișier infectat din cauza virusului
* Nu există drept de acces în sistem pentru a deschide fișierele
* Unitate învechită în sistemul dumneavoastră
* Extensia fișierului este redenumită
