## Ce este un fișier Linux PUP?

Un fișier Linux PUP este un pachet de instalare utilizat de sistemul de operare Puppy Linux. Se execută ca un program de instalare și începe să instaleze fișierele pachetului. Pe Linux, este adesea denumit fișier DotPup și este similar cu fișierele de instalare [.exe](/ro/executable/exe/) și [.msi](/ro/executable/msi/) utilizate pe sistemul de operare Windows. Versiunile mai noi de Puppy Linux folosesc formatul de fișier .pet în loc de formatul de fișier PUP.
