## Ce este un fișier MBW? ##

Un fișier cu extensia .mbw este asociat în primul rând cu MBRWizard de către Firesage Solutions. Master Boot Record Wizard (MBRWizard) face backup pentru MBR-ul computerului și îl salvează într-un fișier MBW care poate fi restaurat dacă este necesar. Pe lângă copiile MBR și VBR (Volume Boot Record), aceste fișiere pot conține GPT, sectoare definite de utilizator sau imagini de pe întregul disc. MBR-ul este o înregistrare vitală, lungă de 512 de octeți, pe hard disk, care permite sistemului de operare să pornească. De asemenea, poate deține tabelul de partiții și o semnătură pe 32 de biți care identifică suportul de disc.

## Sisteme de operare acceptate ##

Formatul de fișier MPW este acceptat numai de sistemul de operare Windows.

## Posibile probleme la deschiderea fișierului ##

Dacă nu puteți deschide și rula fișierul MBW, este posibil să existe una dintre următoarele probleme posibile:

* Fișier MBW corupt
* Legături greșite către fișierul MBW în intrările de registru
* Descrierea ștearsă a MBW din registrul Windows
* Un fișier infectat cu un malware nedorit
* Calculatorul nu are resurse hardware suficiente pentru a opera fișierul MBW
* Driverele folosite de computer pentru a deschide un fișier MBW sunt învechite
