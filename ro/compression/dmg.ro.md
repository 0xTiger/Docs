## Ce este un fișier DMG?

Un fișier cu o extensie .dmg este un fișier Apple Disk Image și cunoscut ca fișier Mac OS X Disk Image. Formatul de fișier DMG este cel mai utilizat fișier imagine de disc destinat să aloce aplicații și fișiere între computerele Apple. Este foarte convenabil să montați fișiere DMG într-o unitate virtuală sau puteți utiliza software-ul Apple Disk Utility pentru a deschide și a vizualiza conținutul formatului de fișier DMG. Vechea versiune a fișierelor IMG utilizate în Mac OS Classic este înlocuită de formatul de fișier DMG. Fișierele DMG sunt generate pentru utilizatorii Mac și nu sunt acceptate de sistemul de operare Windows, dar utilizatorii pot accesa conținutul DMG în sistemul de operare Windows utilizând instrumente de compresie și decompresie precum 7-Zip, Pea Zip etc. Extensia de fișier DMG este folosită și de instrumentul Oracle Export and Ingress pentru fișierele scrapheap păstrate în format binar al bazei de date Oracle.

## Scurt istoric

Extensia de fișier DMG a fost dezvoltată de Apple și este folosită mai ales pe computerele Macintosh. În cea mai mare parte, fișierul DMG este folosit pentru a oferi dispozitivelor Mac capacitatea de a instala diferite tipuri de fișiere și software care inițiază de pe internet. Fișierul DMG este în principiu o imagine de disc montabilă care se afișează pe desktop atunci când este deschis. Fișierul cuprinde date brute care sunt în general atât criptate, cât și comprimate. Sistemele Mac tratează DMG în același mod în care ar trata un disc care a fost intercalat și încearcă instantaneu să deschidă sau să „urmeze” fișierul.

## Specificațiile tehnice ale formatului de fișier DMG

* Special generat pentru computerele Macintosh
* Dezvoltat de Apple
* Formatul fișierului este binar
* Categoria de fișier este fișier imagine disc
* Programul acceptat este MacOS
* Poate conține un tabel de partiții
* Poate conține orice tip de sistem de fișiere, cum ar fi HFS+
* Funcționalitate de comprimare a imaginilor
