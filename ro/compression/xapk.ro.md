## Ce este un fișier XAPK?

Un fișier cu extensia .xapk este un fișier de pachet comprimat care este utilizat pentru a instala aplicații Android pe dispozitive mobile. Este un format de fișier container care încorporează APK și fișiere asociate suplimentare necesare pentru instalare. Celălalt fișier asociat este un fișier OBB care stochează fișiere suplimentare, cum ar fi grafice, fișiere media și date de aplicație, care sunt necesare pentru a menține aplicația în funcțiune. Fișierele XAPK nu sunt acceptate de Google Play și sunt utilizate numai pentru distribuirea pe site-uri web de descărcare a aplicațiilor Android terță parte. Acestea pot fi instalate pe un dispozitiv Android folosind XAPK Installer.

## Format de fișier XAPK

Fișierele XAPK sunt comprimate folosind formatul de fișier standard [ZIP](/ro/compression/zip/). Acestea pot fi extrase folosind un software standard de compresie/decompresie, cum ar fi WinZIP. Odată ce fișierul XAPK este extras pe disc, acesta conține următoarele fișiere în folder.

* APK - Fișier de instalare standard pentru instalarea aplicației pe dispozitive Android
* OBB - Fișier suplimentar care conține fișiere de resurse relevante
