## Ce este un fișier POT?

Fișierele cu extensia .POT reprezintă fișiere șablon Microsoft PowerPoint create de versiunile PowerPoint 97-2003. Fișierele create cu aceste versiuni de Microsoft PowerPoint sunt în format binar în comparație cu cele create în formate de fișiere Office OpenXML folosind versiunile superioare de PowerPoint. Prin urmare, fișierele generate pot fi folosite pentru a crea prezentări care au același aspect și alte setări necesare pentru a fi aplicate fișierelor noi. Aceste setări pot include stiluri, fundaluri, paletă de culori, fonturi și valori implicite. Astfel de fișiere sunt generate pentru a crea fișiere șablon gata de utilizare pentru uz oficial.

## Specificații de format de fișier ##

Specificațiile de format de fișier pentru formatul de fișier POT nu sunt disponibile individual de către Microsoft. Cu toate acestea, formatul șablonului este același cu cel al formatului de fișier binar [PPT](/ro/presentation/ppt/) și poate fi deschis în MS PowerPoint pentru editare. Un fișier POT este identificat prin identificatorul HEX după cum urmează:

```
D0 CF 11 E0 A1 B1 1A E1 00 00 00 00
```

Tipurile MIME asociate cu formatul de fișier POT sunt:

* application/vnd.ms-powerpoint [oficial]
* aplicație/mspowerpoint
* aplicație/x-mspowerpoint
* aplicație/powerpoint
* aplicație/x-powerpoint
* application/x-dos_ms_powerpnt
* aplicație/oală
* aplicație/x-sofic
