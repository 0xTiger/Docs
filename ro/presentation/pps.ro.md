## Ce este un fișier PPS?

PPS, PowerPoint Slide Show, fișierele sunt create folosind Microsoft PowerPoint pentru scopul Slide Show. Citirea și crearea fișierelor PPS este acceptată de Microsoft PowerPoint 97-2003. Cea mai recentă versiune a acestui format de fișier este [PPSX ](/ro/presentation/ppsx/), care se bazează pe standardele Office OpenXML. Fișierele PPS pot fi citite în continuare de cele mai recente versiuni ale Microsoft PowerPoint, dar fișierele nou create pot fi salvate numai în format de fișier PPSX. Când un fișier PPS este partajat cu un alt utilizator și este deschis, acesta începe ca afișare Powerpoint, spre deosebire de fișierul [PPT](/ro/presentation/ppt/), care se deschide în modul editabil.

## Tipul fisierului ##

PPS este un format de fișier binar care respectă specificațiile detaliate în formatul de fișier [MS-PPT](https://msdn.microsoft.com/en-us/library/office/cc313106(v#office.12).aspx) document. Este alcătuit dintr-un număr de fluxuri pentru a reprezenta structura generală a documentului și datele. Acestea includ:

* Flux de utilizator curent
* Flux de documente PowerPoint
* Flux de imagini
* Informații rezumate și informații rezumate document (opțional)

Detalii complete despre formatul de fișier PPT pot fi găsite în documentul cu specificații MS-PPT, care se aplică în mod egal și PPS.
