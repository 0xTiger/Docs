## Ce este un fișier SITE?

Un fișier SITE este un fișier de proiect utilizat de software-ul de proiectare a site-urilor web "Adobe GoLive". Stochează informații despre proiectul site-ului web, cum ar fi fișiere, foldere, imagini, foi de stil personalizate și scripturi. De asemenea, conține structura site-ului web și link-urile utilizate în paginile web ale site-ului. Deoarece este un fișier de proiect, acesta conține referințe la fișierele de proiect, dar nu conține fișierele reale în sine.

Adobe GoLive a fost întrerupt și nu mai este acceptat.

## Format de fișier SITE

Fișierele SITE sunt stocate pe disc în format de fișier proprietar Adobe, iar detaliile interne ale formatului său de fișier nu sunt disponibile public pentru referință de către dezvoltatori.
