## Ce este un fișier WBS?

Un fișier WBS este un fișier de proiect creat de WebBlender, care este un program de prezentare și creare de site-uri web pentru copii. Fișierul WBS este creat atunci când un nou proiect este inițiat în WebBlender. Toate caracteristicile site-ului adăugate proiectului, cum ar fi pagini, imagini, text, animații și așa mai departe, sunt menționate în fișierul de proiect WBS. Odată ce proiectul este gata, acesta publică conținutul site-ului ca fișier [HTML](/ro/web/html/). WebBlender este creat și întreținut de [Tech4Learning](https://www.tech4learning.com/).
