## Ce este un fișier DOWNLOAD?

Un fișier DOWNLOAD este un fișier descărcat parțial de pe internet care este încă în curs de descărcare. Când descărcați un fișier de pe internet utilizând browserul Apple Safari, poate dura ceva timp până când descărcarea este finalizată. Acest fișier descărcat parțial este salvat în dosarul **Descărcări**. Dacă descărcarea este întreruptă din orice motiv, se poate face dublu clic pe fișier pentru a încerca să-l reia.

## DESCARCĂ Formatul fișierului - Mai multe informații

Fișierele DOWNLOAD sunt salvate pe disc în format text, precum și în formate de fișiere binare. Nu conține întregul fișier care este în proces de descărcare. Odată ce descărcarea este completă, extensia de fișier .download este înlocuită cu extensia originală a fișierului care este descărcat. De exemplu, dacă descărcați un videoclip de pe internet în format de fișier MP4, fișierul va fi descărcat și salvat cu extensia .download. Odată ce descărcarea este completă, extensia fișierului va fi schimbată în .mp4.

Fișierele DOWNLAOD sunt similare cu fișierele [.CRDOWNLOAD](/ro/web/crdownload/) utilizate de Google Chrome și Microsoft Edge.
