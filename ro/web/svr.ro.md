## Ce este un fișier SVR?

Un fișier SVR este un format de fișier 3D folosit de SuperScape pentru a crea lumi interactive 3D. Aceste lumi interactive tridimensionale pot fi vizualizate prin intermediul browserelor web cu pluginul SVR. Fișierele SVR sunt similare cu formatul [Fișier VRT](/ro/web/vrt/), dar sunt comprimate, ceea ce duce la o dimensiune mai mică a fișierului. Astfel, fișierele SVR sunt încărcate mai repede și pot fi partajate și prin internet.

## Format de fișier SVR - Mai multe informații

Fișierele SVR sunt salvate pe disc în formate de fișiere binare. Este un format de fișier proprietar, iar detaliile sale interne ale structurii fișierelor nu sunt disponibile public pentru referință de către dezvoltatori.
