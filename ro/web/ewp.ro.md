## Ce este un fișier EWP?

Un fișier EWP este un fișier de proiect creat cu software-ul [EWISoft](http://www.ewisoft.com/) Website Builder. Este legat de un fișier de bază de date binar care conține toate informațiile pentru proiect. Fișierul EWP este utilizat în mod obișnuit pentru a salva progresul dezvoltării web. Este un software de design web bazat pe șabloane care permite utilizatorilor să creeze site-uri web din șabloane de site-uri web.

## Format de fișier EWP - Mai multe informații

Fișierele EWP sunt fișiere de proiect care nu trebuie să fie compilate sau găzduite pe niciun server. Acestea sunt salvate ca fișiere text simplu și conțin referințe la progresul dezvoltării web. Proiectele pot fi publicate pe un site web odată ce dezvoltarea este finalizată.
