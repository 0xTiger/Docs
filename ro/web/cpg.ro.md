## Ce este un fișier CPG?

Un fișier CPG este un fișier de proiect creat de software-ul editor de pagini Web, Cool Page. Software-ul a devenit învechit cu mulți ani în urmă și nu mai este folosit. Fișierul CPG conținea detalii despre obiectele utilizate în pagina web și proprietățile acestor obiecte. Fișierele de proiect CPG ar putea fi deschise în software-ul Cool Page și după finalizare, pagina web ar putea fi exportată ca pagină web [HTML](/ro/web/html/).

## Format de fișier CPG

Detaliile interne ale formatului fișierului CPG nu sunt cunoscute. Acestea au fost salvate ca fișiere binare și se vor deschide numai în software-ul Cool Page.
