## Ce este un fișier VRT?

Un fișier VRT este un format de fișier 3D care conține lumi tridimensionale care pot fi vizualizate în browser-ul web cu pluginul SVR [Viscape](https://www.viscape360.com/). Este folosit de plugin pentru a încărca și afișa utilizatorilor datele lumii 3D pentru vizualizare și analiză. Informațiile stocate într-un fișier VRT pot cuprinde date precum texturi, sunete, dimensiuni și comportamente. Cele mai recente fișiere VRT sunt stocate în format de fișier SVR, care este o versiune comprimată a fișierului VRT.

## Format de fișier VRT

Fișierele VRT sunt stocate în format de fișier proprietar Viscape, iar detaliile lor interne ale formatului de fișier nu sunt disponibile public.
