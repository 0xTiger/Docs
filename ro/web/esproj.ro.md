## Ce este un fișier ESPROJ?

Un fișier ESPROJ este un fișier de proiect creat de software-ul de creare a paginilor web, Espresso. Este disponibil pe Mac OS și conține o listă a fișierelor asociate care fac parte din proiectul site-ului web. Când deschideți fișierul ESPROJ în Espresso, acesta deschide toate fișierele aferente în editor pentru editarea sau actualizarea codului sursă. Fișierul ESPROJ conține, de asemenea, setări de publicare pe server și alte preferințe ale utilizatorului.

## Format de fișier ESPROJ

Fișierele ESPROJ sunt salvate ca fișiere binare, iar conținutul acestuia nu poate fi citit de om. Directorul de fișiere ESPROJ este, de asemenea, directorul rădăcină pentru site-ul web.
