Este nevoie de un număr de tipuri de fișiere atunci când construiți o pagină web. Formatele de fișiere web definesc standardele de dezvoltare a paginilor web și sunt legate de platforma în care acestea sunt construite. Un site web complet poate fi construit constând din pagini web statice și dinamice. Cele mai multe site-uri web moderne sunt construite pe tehnologia serverului, cum ar fi Active Server Pages (ASP), care sunt încărcate și rulate pe serverul web. Acestea includ, de asemenea, foi de stil în cascadă (CSS) și fișiere de scripting utilizate pentru stilarea aspectului general al interfeței de utilizare.

Extensiile obișnuite de fișiere web și formatele lor asociate includ **[HTML](/ro/web/html/)** (Hypertext Markup Language), **[ASP](/ro/web/asp/) (Active Server Pages)**, și **CSS** (Foi de stil în cascadă).
