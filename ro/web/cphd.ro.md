## Ce este un fișier CPHD?

Un fișier CPHD este un document hipertext utilizat de motorul pre-procesor CloudChan. Conține HTML și scripturi pentru a crea pagini web și documente dinamice. Scripturile din interiorul unui fișier CPHD seamănă cu sintaxa folosită de Java. Pagina web CPHD returnează un document HTML atunci când este executată pe un proces CloudChan. Formatul CPHD poate funcționa cu orice document XML.

## Format de fișier CPHD

Fișierele CPHD sunt stocate pe disc ca fișiere text simplu. Acest lucru face posibilă deschiderea acestor fișiere în orice editor de text, cum ar fi Notepad sau Notepad++.
