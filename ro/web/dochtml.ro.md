## Ce este un fișier DOCHTML?

Un fișier DOCHTML este formatul HTML al unui document generat cu Microsoft Word. Este versiunea [HTML](/ro/web/html/) a documentului creat care poate fi utilizată pentru publicare pe serverul web. Fișierele DOCHTML pot fi deschise în browserul web pentru a vizualiza conținutul documentului Word convertit. Perspectiva și aspectul fișierului DOCHTML sunt identice cu cele ale documentului Word original.

## Format de fișier DOCHTML

Fișierele DOCHTML sunt salvate pe disc în format HTML care este gata pentru a fi publicat pe serverele web. Microsoft Word a oprit suportul pentru salvarea documentelor ca fișiere DOCHTML. Acesta a fost înlocuit cu formatele de fișier [.mht](/ro/web/mht/) și [.mhtml](/ro/web/mhtml/) care sunt utilizate în mod obișnuit acum.
