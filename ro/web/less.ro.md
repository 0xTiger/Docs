## Ce este un fișier LESS?

Un fișier LESS este o foaie de stil folosită pentru pagini web și creată cu Leaner Style Sheets (LESS). Implementează completări la limbajul CSS standard, ceea ce îl face ușor de învățat și implementat. Un fișier LESS extinde [CSS](/ro/web/css/) prin adăugarea de caracteristici precum Variabile, Mixin, Imbricare, Operații, Funcții, Spații de nume și Accesori.

## Leaner Style Sheets File Format

Fișierele LESS sunt create în format CSS, extinzându-și capacitățile prin caracteristicile suplimentare pe care le oferă. Acestea sunt salvate ca fișiere text și pot fi deschise în limbajul pentru foile de stil LESS, CodeKit și Adobe Dreamweaver.
