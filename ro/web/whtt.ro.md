## Ce este un fișier WHTT?

Un fișier WHTT este o versiune offline a unui site web care este descărcată folosind software-ul [HTTrack Website Copier](https://www.httrack.com/). Conține tot conținutul unui site web, inclusiv [HTML](/ro/web/html/), pagini web, imagini, foi de stil, link-uri și directoare. Toate paginile unui site web sunt descărcate recursiv și salvate pe disc ca oglindă offline. Când un site web este descărcat de software-ul HTTrack, acesta poate fi vizualizat offline fără a fi nevoie să fie conectat la internet. Pentru a vizualiza site-ul descărcat, pur și simplu deschideți pagina site-ului în oglindă în browser și fiecare imagine/link va fi încărcat corect.

## Format de fișier WHTT

Fișierele WHTT sunt salvate ca o combinație de pagini web THML, imagini, foi de stil și link-uri de referință.
