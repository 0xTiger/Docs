## Ce este un fișier FWP?

Un fișier cu fișier .fwp este un fișier de arhivă creat cu Microsoft Expression Web și conține toate fișierele și folderele pentru un design. Microsoft Expression Web este un produs software general de proiectare web de la Microsoft și se bazează pe editorul [HTML](/ro/web/html/). A fost întrerupt din 2012 și disponibil gratuit pentru descărcare de către Microsoft. Fișierele FWP pot fi exportate și salvate în formatul Microsoft .cab.

## Format de fișier FWP - Mai multe informații

Specificațiile exacte ale formatului de fișiere ale fișierelor FWP nu sunt disponibile pentru referință de către dezvoltatori. Cu toate acestea, acestea sunt fișiere binare și pot fi deschise în Microsoft Expression Studio, care a fost întrerupt cu mult timp în urmă.
