## Ce este un fișier DCR?

Un fișier DCR este un fișier media care este creat și salvat în format Adobe Shockwave. Este folosit pentru crearea de jocuri web și jocuri interactive similare celor proiectate și salvate în fișierele [SWF](/ro/page-description-language/swf/). Fișierele DCR au fost create anterior cu Adobe Director, care a fost ulterior întrerupt după achiziționarea de către sistemele Adobe. Fișierele DCR sunt similare cu fișierele SWF create de Adobe Flash.
