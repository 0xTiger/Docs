## Ce este un fișier XHT?

Un fișier cu extensia .xht este un fișier web asociat cu tipul de fișier [XHTML](/ro/web/xhtml/) (Extended Hypertext Markup Language), care în sine este un fișier de marcare bazat pe [XML](/ro/web/xml/). Ambele fișiere sunt similare cu HTML, dar au o sintaxă mai strictă asemănătoare XML. Fișierele XHT sunt proiectate astfel încât să fie mai structurate, implică mai puține scripturi și sunt generice în plus față de utilizarea facilităților existente ale XML.

## Format de fișier XHT

Un fișier XHT este creat și stocat ca fișier text simplu cu codificare UTF-8 în mod implicit. Conține codul sursă complet al unui document XHTML care poate fi deschis și editat cu orice editor de text care acceptă codificarea Unicode. Pe lângă editorii de text, formatul de fișier XHT este de înțeles de majoritatea browserelor web moderne și poate fi deschis folosind acestea.

## Exemplu XHT

Următorul exemplu de document XHT definește declarațiile XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
