## Ce este un fișier XHTM?

Un fișier cu extensia .xhtm este un fișier de pagină web care este scris în XHTML. Utilizează XML 1.0 ca implementare standard și este considerată o reformulare a HTML 4. Ca și XHTML, un fișier XHTM conține etichete HTML standard care urmează un ghid XML mai sever. Fișierele XHTM sunt folosite pentru publicarea conținutului site-ului web și folosesc de obicei extensia [XHTML](/ro/web/xhtml/).

Puteți utiliza browsere web precum Google Chrome, Apple Safari și Mozilla Firefox pentru a deschide fișiere XHTM.

## Exemplu XHTM

Următorul exemplu de document XHTM definește declarațiile XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
