## Ce este un fișier CCBJS?

Un fișier CCBJS este un fișier Javascript creat de aplicația 3D, jocul și editorul de site-uri web, **CopperCube**. Conține codul JavaScript care este generat ca parte a [fișierului .html](/ro/web/html/) creat din fișierul .ccb. CopperCube oferă o interfață de scripting ușor de utilizat pentru a dezvolta logica de joc și interacțiunile fără a scrie niciun cod. Alte formate de fișiere acceptate de CopperCube includ .ccscene, .ccmodel, .ccmaterial, .cctex, .ccjs, .ccp și .ccpak.

## Format de fișier CCBJS

Detaliile interne ale formatului fișierului CCBJS nu sunt cunoscute ca atare. Cu toate acestea, fișierele Javascript sunt salvate ca fișiere text simplu pe disc și pot fi deschise cu orice editor de text, cum ar fi Microsoft Notepad, Notepad++ și Apple TextEdit.
