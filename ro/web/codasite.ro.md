## Ce este un fișier CODASITE?

Un fișier CODASITE este un fișier de proiect al site-ului web creat cu software-ul de creare și publicare a paginilor web [Coda](https://coda.io/). Conține o listă cu toate fișierele care fac parte din proiect. Acestea includ pagini web ca fișiere [HTML](/ro/web/html/), fișiere de foaie de stil în cascadă [CSS](/ro/web/css/), fișiere Javascript [JS](/ro/web/js/) și orice alte fișiere de suport asociate. Fișierele de proiect CODASITE conțin și informații pentru publicarea site-ului web pe serverul web.

## Format de fișier CODASITE

Fișierul de proiect CODASITE este salvat pe disc ca fișiere binare în directorul ~/​Library/​Application Support/​Coda 2/​Sites/​. Interfața de utilizator Coda oferă interfața pentru gestionarea site-urilor Coda, deoarece deschiderea manuală a acestora poate duce la setări necorespunzătoare ale proiectului și poate crea probleme.
