## Ce este un fișier FWTEMPLATEB?

Un fișier FWTEMPLATEB este un pachet de documente șablon de site web folosit de Freeway Pro, o aplicație software pentru crearea de site-uri web fără a scrie niciun cod. Conține mai multe șabloane de site-uri web care pot fi partajate și cu alți utilizatori Freeway Pro. Fiecare site web șablon din pachet conține informații despre aspectul și aspectul site-ului.

## Format de fișier FWTEMPLATEB

Fișierele FWTEMPLATEB sunt salvate ca o colecție de șabloane de site-uri web și conțin informații legate de aspectul site-ului. Acestea oferă cea mai ușoară modalitate de a crea un site web, deoarece toate setările de aspect și aspect sunt deja făcute în fișierul șablon. Un alt avantaj al fișierelor FWTEMPLATEB este că obțineți un pachet de șabloane pentru a crea cu ușurință site-uri web.
