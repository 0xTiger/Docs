## Ce este un fișier SEAM?

Un fișier SEAM este un fișier de configurare utilizat de cadrul JBOS Seam. Conține metadate/referință la un servlet de pe partea serverului care procesează cererile către aplicația web. De asemenea, conține metadate despre componentele Seam, cum ar fi numele claselor și metodele care implementează componentele. Puteți utiliza fișierele SEAM împreună cu cadrul JavaServer Faces (JSF) pentru a construi aplicații web dinamice și interactive. Fișierele SEAM sunt de obicei stocate în directorul WEB-INF al unei aplicații Seam.

## Format de fișier SEAM

Un fișier SEAM este stocat ca fișier text, de obicei în format [XML](/ro/web/xml/) și conține informații de configurare pentru o aplicație Seam. Este o parte cheie a configurației unei aplicații Seam. Puteți deschide un fișier SEAM cu orice editor de text sau XML.
