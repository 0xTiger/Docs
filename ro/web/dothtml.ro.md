## Ce este un fișier DOTHTML?

Un fișier DOTHTML este un document șablon Microsoft Word salvat în versiunea HTML. Poate fi deschis în browser web și conținutul este afișat ca orice altă pagină web. Fișierele DOTHTML sunt generate din fișierele Microsoft Word .DOT care conțin setări preformatate pentru generarea acestor fișiere. Când un fișier [DOT](/ro/word-processing/dot/) este convertit în DOCHTML, toate aceste setări sunt păstrate și fișierul DOTHTML convertit este afișat cu o formatare similară cu fișierul DOT original. În acest fel, șabloanele publicate pot fi vizualizate în browser web.

## Format de fișier DOTHTML - Mai multe informații

Fișierele DOTHTML sunt salvate ca fișiere [HTML](/ro/web/html/) și conțin cod sursă HTML care poate fi deschis cu orice editor HTML. În primele zile, Microsoft Word a oferit versiunile HTML ale tuturor formatelor sale de documente, cum ar fi [DOCHTML](/ro/web/dochtml/), [DOCMHTML](/ro/web/docmhtml/), dar mai târziu aceste versiuni bazate pe HTML au fost întrerupte. . În consecință, DOTHMLT nu mai este acceptat.
