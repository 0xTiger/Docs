## Ce este un fișier HXS?

Un fișier HXS este un fișier de ajutor compilat și utilizat pentru fișierul Microsoft Help 2.x. Este compilat și generat dintr-un set de pagini cu subiecte și scris într-un subset de [HTML](/ro/web/html/) care este similar cu formatul de fișier de ajutor [CHM](/ro/web/chm/) anterior. Fișierele HXS sunt stocate în format comprimat și sunt create dintr-un fișier de proiect principal .HxC, un fișier include .HxF, un cuprins .HxT, un fișier de definire a atributelor .HxA și mai mulți indecși .HxK. Metadatele unui fișier HXS sunt create din [XML](/ro/web/xml/).

Fișierele HXS pot fi deschise cu un browser web.

## Format de fișier HXS

Fișierele HXS sunt create și salvate ca fișiere comprimate din conținut HTML și metadate XML.
