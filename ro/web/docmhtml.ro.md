## Ce este un fișier DOCMHTML?

Un fișier DOCMHTML este o variantă veche a arhivei web salvată de versiunile anterioare ale Microsoft Word. Nu mai este acceptat de cea mai recentă versiune de Microsoft Word. Un fișier DOCMHTML conține o reprezentare HTML a documentului Word original .doc. Poate fi deschis atât în Microsoft Word, cât și în browser web. Când un DOCMHTML este publicat, acesta poate fi găzduit pe un server web care este accesibil utilizatorilor prin intermediul cererii web. Un alt format similar este [DOCHTML](/ro/web/dochtml/), care a fost, de asemenea, întrerupt.

## Format de fișier DOCMHTML - Mai multe informații

Fișierele DOCMHTML au fost folosite pentru a fi salvate în format HTML. Poate fi deschis într-un editor de text, cum ar fi Notepad și Notepad++. După ce a fost marcat ca învechit, Microsoft Word a introdus opțiunile de salvare a documentului Word ca fișiere [MHTML](/ro/web/mhtml/) și [MHT](/ro/web/mht/). Aceste noi formate de fișiere pot fi deschise cu browsere web.
