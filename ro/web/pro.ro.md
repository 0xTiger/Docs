## Ce este un fișier PRO? ##

Un fișier PRO este o machetă a unui site web sau a unei aplicații mobile create de Adobe Proto, o aplicație Adobe Touch defunctă. Include unul sau mai multe aspecte de UI pentru o pagină web sau o aplicație mobilă, fiecare dintre acestea putând include elemente interactive. Tehnologiile CSS, WebKit și jQuery sunt acceptate în fișierele PRO.

### Scurt istoric al formatului de fișiere PRO

Fișierele PRO au fost create de Adobe Proto, care a fost înlocuit cu Adobe XD. A fost folosit pentru a crea aplicații de prototipare pentru tablete. Aplicațiile obișnuite de web design precum Adobe Dreamweaver au folosit aceste fișiere PRO pentru a crea aplicații web și mobile.

## Format de fișier PRO - Mai multe informații

Un fișier PRO este primul pas în dezvoltarea unui site web sau a unei aplicații mobile. Dezvoltatorii creează mai întâi machete digitale care definesc aspectul aplicației. Acestea definesc, de asemenea, fluxul general de lucru al aplicației și navigarea prin ecranele aplicației. Aceste machete includ, de asemenea, o serie de elemente de navigare, cum ar fi butoane, casete combinate, bare de defilare, câmpuri de introducere și multe altele.
