## Ce este un fișier HTC?

Un fișier HTC este un fișier în format de fișier web utilizat de Microsoft Internet Explorer. Conține definiții pentru componentele HTML care îndeplinesc mici funcții pentru realizarea diferitelor sarcini în interiorul browserului web. Funcționalitatea definită în fișierele HTC este reutilizabilă și este accesibilă în mai multe pagini [HTML](/ro/web/html/).

## Format de fișier HTC - Mai multe informații

Fișierele HTC sunt salvate în format text simplu. Aceasta înseamnă că le puteți deschide în orice editor de text și puteți modifica funcționalitatea. Acestea sunt de obicei deschise în Internet Explorer, dar astfel veți putea vizualiza doar aceste fișiere și nu le veți putea edita.

Un fișier HTC este de fapt un fișier HTML care este o combinație de script și un set de elemente specifice HTC pentru definirea componentelor.
