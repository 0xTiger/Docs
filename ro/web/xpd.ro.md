## Ce este un fișier XPD?

Un fișier XPD este un fișier de pagină web creat dintr-un document XML. Mai des, aceasta este generată printr-o transformare XSLT. [Specificațiile formatului fișierului XPD](https://www.w3.org/TR/xml-pipeline/) sunt disponibile pe W3 care definesc o modalitate de a crește interoperabilitatea multi-procesului [XML](/ro/web/xml/) aplicatii. XML Pipeline Language definește relațiile de procesare dintre resursele XML.
