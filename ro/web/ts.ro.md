## Ce este un fișier TS?

Un fișier TS este un fișier de traducere generat cu utilitarul software lupdate care vine la pachet cu SDK-ul Digia Qt. SDK-ul este disponibil pentru sistemele de operare Windows, Mac și Linux. Fișierul TS conține expresii într-o limbă și traducerile acestora în altă limbă. Traducerile stocate în interiorul unui fișier TS sunt specifice unei anumite aplicații. Fișierele TS pot fi deschise cu Digia Qt SDK.

## Format de fișier TS

Fișierele Digia Qt TS sunt stocate pe disc în format de fișier XML și pot fi deschise în orice editor de text, cum ar fi Notepad și Notepad++. Alte formate de fișiere acceptate de Digia Qt SDK includ:

* .pri
* .qm
* .qml
* .qph
* .qrc
* .qs
* .qss
* .ts
* Fișier interfață utilizator .ui
