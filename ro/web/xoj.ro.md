## Ce este un fișier XOJ?

Un fișier XOJ este un fișier caiet care conține note luate folosind software-ul de luare de note manual Xournal. Xournal este o aplicație gratuită, multi-platformă, care vă permite să creați note cu diferite tipuri de conținut, cum ar fi text, imagini și doodle-uri scrise de mână. Fișierele XOJ sunt create atunci când se creează un nou fișier blocnotes cu Xournal. Aceste fișiere XOJ pot fi partajate cu alți utilizatori și pentru partajarea datelor. O versiune mai recentă a Xournal este [Xournal++](https://github.com/xournalpp/xournalpp) care este o variantă multiplatformă a Xournal.

## Format de fișier XOJ

Fișierele XOJ sunt în format de fișier XML și se salvează pe disc în formă comprimată utilizând compresia [.gz](/ro/compression/gz/). Datorită utilizării formatului XML universal, fișierele XOJ pot fi deschise și editate și cu varianta multiplatformă Xournal++.

Fiecare fișier XOJ poate conține încă o pagină dintr-un caiet Xournal.

## XOJ - Platforme acceptate

Xournal este multi-platformă, iar notebook-urile sale pot fi salvate ca fișier XOJ pe următoarele sisteme de operare.

* Linux
* MacOS
* Windows
* Android
* iOS Build
