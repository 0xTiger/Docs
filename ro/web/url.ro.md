## Ce este un fișier URL?

Un fișier cu extensia .url este o comandă rapidă la adresa URL a site-ului web care poate fi salvată pe computer cu orice adresă a site-ului web. Când este deschis făcând dublu clic, adresa URL va deschide site-ul web în browserul implicit și pagina va fi afișată utilizatorului. Fișierele URL sunt de obicei denumite fișiere de comandă rapidă către site-uri web și sunt reprezentate de pictograma browserului implicit. Fișierele URL pot fi deschise și editate în orice editor de text, cum ar fi Microsoft Notepad, Notepad++ sau Apple TextEdit.

## Format fișier URL - Mai multe informații

Fișierele URL sunt salvate în format text simplu și pot fi deschise și vizualizate în orice editor de text. Acestea se deschid în browserul web implicit, unde utilizatorul efectuează altă navigare.
