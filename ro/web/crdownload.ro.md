## Ce este un fișier CRDOWNLOAD?

Un fișier CRDOWNLOAD este un fișier descărcat parțial care este creat atunci când descărcați un fișier de pe internet utilizând browserele Microsoft Edge, Google Chrome sau Chromium. Conținutul fișierului descărcat este stocat în acest fișier până la finalizarea descărcării. Odată ce descărcarea se termină, extensia .crdownload este eliminată și este înlocuită cu extensia originală a fișierului descărcat.

## CRDOWNLOAD Format de fișier

Un fișier CRDOWNLOAD poate fi atât în format text, cât și în format binar. Browserele creează în mod implicit fișierul .crdownload odată ce începe descărcarea. Dacă descărcarea este întreruptă între ele, fișierul CRDOWNLOAD rămâne salvat pe disc. Acest lucru permite uneori să reluați descărcarea de unde a fost întreruptă.

### CRDOWNLOAD Exemplu

Asistență pentru că descărcați un fișier fileformatslist.doc de pe internet. Când începe descărcarea, browserul dvs. salvează fișierul în folderul Descărcări ca fișier fileformatslist.crdownload. Dimensiunea fișierului său crește pe măsură ce fișierul este descărcat. Odată ce descărcarea se termină, fișierul va fi redenumit în fileformatslist.doc.
