## Ce este un fișier WEBSITE?

Un fișier WEBSITE este un fișier de comandă rapidă a site-ului web creat cu Microsoft Internet Explorer (IE). Conține o adresă URL și un fișier pictogramă (ICO) care indică favicon-ul site-ului web. Pictograma este folosită pentru afișarea comenzii rapide. Formatul de fișier WEBSITE a fost introdus odată cu lansarea versiunii IE 9. Versiunile anterioare de IE erau folosite pentru a salva comenzile rapide către paginile web ca fișiere [.url](/ro/web/url/).

## Format fișier SITE WEB

Un fișier .website este stocat pe disc ca fișiere text simplu al căror conținut poate fi vizualizat în orice editor de text. Pe lângă alt conținut, conține adresa URL țintă pentru care este o comandă rapidă. Dacă doriți să deschideți site-ul web țintă într-un alt browser decât IE, puteți copia această adresă URL și o puteți lipi în alte browsere, cum ar fi Google Chrome, Safari sau Firefox, pentru a vizualiza site-ul web.

## Cum se creează fișierul .WEBSITE?

Un fișier WEBSITE poate fi creat în IE făcând clic pe adresa URL din bara de adrese și trăgându-l în locația dorită de pe computer, cum ar fi desktop sau documente. Aceasta creează o comandă rapidă către acea locație de internet și poate fi deschisă făcând dublu clic pe comandă rapidă. Dacă doriți să salvați comanda rapidă ca fișier „URL”, țineți apăsată tasta SHIFT înainte de a glisa favicon-ul site-ului țintă pe desktop.
