## Ce este un fișier CSP?

Un fișier CSP este un fișier dinamic de pagină web care conține conținut scris pentru a fi executat pe RadGs Concept Application Server. Este compilat și executat pe partea serverului, după care este servit browserului utilizatorului solicitant prin protocolul HTTP. Spre deosebire de formatul de fișier CON, fișierele CSP nu necesită ca clientul Concept să fie instalat la utilizator pentru a vizualiza aceste fișiere. Puteți vizualiza fișiere CSP similare cu orice alte pagini din partea serverului, cum ar fi [PHP](/ro/programming/php/) și [ASP](/ro/web/asp/).
