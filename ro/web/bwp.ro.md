## Ce este un fișier BWP?

Un fișier BWP este un fișier script de pagină web creat în limbajul de programare, **BuRg3r**. Este specific lui BuRg3r și poate conține cod HTML cu secțiuni ale sursei BuRg3r delimitate de<?b and ?> Etichete. BuRg3r este un limbaj de programare de uz general, care este și un motor de joc care a fost dezvoltat de Fluffy Smoke Entertainment.
