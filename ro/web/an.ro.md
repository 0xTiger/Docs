## Ce este un fișier AN?

Un fișier cu extensia .an este un fișier proiect [Adobe Edge Animate](https://www.adobe.com/products/animate.html) care conține informații despre animație. Când proiectul este salvat, este creat un fișier .html care conține referințele la fișierele asociate [.css](/ro/web/css/) și [.js](/ro/web/js/) generate de Adobe Edge Animate. Fișierul AN este salvat împreună cu aceste fișiere care pot fi folosite pentru a deschide fișierul proiect. Adobe edge animate poate fi folosit ca alternativă la crearea de pagini web animate folosind Adobe Flash, care nu mai este acceptat de browsere.

## Format de fișier AN

Fișierele AN sunt stocate pe disc în format text simplu și pot fi deschise în editorii de text pentru vizualizarea conținutului lor.
