## Ce este un fișier STC?

Un fișier cu extensia .stc este un fișier de conexiune utilizat de software-ul Adobe Contribute pentru a stoca informațiile de conectare. Conține informații de conectare, cum ar fi numele de utilizator, adresa URL a gazdei, protocoale precum FTP, SFTP etc. pentru reutilizare din site-ul web existent. Pentru cei care nu știu, Adobe Contribute era un editor specializat [HTML](/ro/web/html/) care a fost folosit pentru a trimite actualizări către site-urile web existente, inclusiv articole bazate pe conținut cunoscute sub numele de bloguri. Include pluginuri pentru browsere web populare, cum ar fi Internet Explorer și Firefox, care au contribuit fără probleme cu conținut la site-uri web din interiorul paginilor. A fost întrerupt din ianuarie 2017, iar Adobe Dreamweaver este folosit acum pentru a crea site-uri web de la zero.

## Format de fișier STC

Formatul intern de date al fișierelor STC este necunoscut. Cu toate acestea, cheia de conectare este criptată cu o parolă și orice informație de conectare la rețea sau FTP trimisă în fișier rămâne securizată și poate fi accesată numai prin Contribute. Fișierul de conexiune cheie poate fi trimis prin e-mail utilizatorilor sau salvat pe computer. Când se face clic pe acesta, se deschide automat Contribute, care importă conexiunea.
