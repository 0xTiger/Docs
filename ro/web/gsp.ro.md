## Ce este un fișier GSP?

Un fișier GSP este un fișier cu mesaj de eroare care indică o problemă cu livrarea e-mailului. Este trimis de IMail Server, un program de server de e-mail care este utilizat pentru comunicarea prin e-mail de numeroase întreprinderi mici și mijlocii. Eroarea se poate datora mai multor motive, cum ar fi cantitatea mare de date, pierderea datelor din fișierul [MSG](/ro/email/msg/) sau erori de memorie lipsită.

## Format de fișier GSP - Mai multe informații

Fișierele GSP sunt stocate în format de fișier web și sunt ușor de vizualizat cu clienții de e-mail, cum ar fi Microsoft Outlook. Un fișier GSP este generat ca răspuns la un e-mail trimis care are ca rezultat o eroare.
