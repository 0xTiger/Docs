## Ce este un fișier EDGE?

Un fișier cu extensia .edge este un fișier Adobe Edge Project care este creat atunci când un proiect este creat și salvat cu Adobe Edge. Adobe Edge le permite designerilor web să creeze animatori [HTML](/ro/web/html/) interactivi pentru web, publicații digitale și publicitate rich media atât pentru browsere desktop, cât și pentru cele mobile. Acest lucru se realizează folosind HTML/HTML5, [JavaScript](/ro/web/js/) și [CSS3](/ro/web/css/). Fișierele Edge sunt generate ca fișiere text simplu și pot fi deschise cu mai multe aplicații diferite, cum ar fi Microsoft Notepad, Apple TextEdit și Notepad++.

## Format de fișier EDGE

Fișierele EDGE sunt salvate într-un format de text simplu și utilizează jQuery pentru a activa animația. O alternativă a animației EDGE au fost fișierele Flash care nu mai sunt acceptate. Fișierul EDGE este salvat împreună cu fișierele HTML, JavaScript și CSS. La deschiderea fișierului EDGE, se deschide pagina web, adică fișierul HTML.
