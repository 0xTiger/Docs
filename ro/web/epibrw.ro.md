## Ce este un fișier EPIBRW?

Un fișier EPIBRW este un fișier generic găzduit pe un server web bazat pe Microsoft IIS. Scopul acestui fișier nu este specific nici unei operațiuni specifice sau oricărui fel de stocare a datelor, ci este general. Este cel mai des folosit de fișierele [.pdf](/ro/pdf/), dar poate fi folosit și de alte tipuri de fișiere.

Când se face dublu clic pe un fișier EPIBRW, acesta se poate deschide în browser afișând conținutul. Cu toate acestea, dacă fișierul este conectat pentru descărcare, procesul de descărcare a fișierului va începe ca urmare a acestui eveniment.
