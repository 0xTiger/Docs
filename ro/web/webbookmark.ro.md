## Ce este un fișier WEBBOOKMARK?

Un fișier WEBBOOKMARK este un fișier de marcaje salvat cu browserul web Safari al Apple pe macOS. Conține informații despre pagina web pe care utilizatorul a salvat-o din browser. Informațiile stocate într-un fișier WEBBOOKMARK includ titlul paginii, adresa URL și grupul de marcaje în care este stocată această pagină. Când o pagină web este marcată, browserul Safari solicită grupului să stocheze pagina marcată. Majoritatea utilizatorilor salvează marcajele în folderul Marcaje, care este locația implicită oferită de browserele obișnuite.

## Format de fișier WEBBOOKMARK

Fișierele WEBBOOKMARK sunt salvate pe disc în format binar care nu poate fi citit de om. Utilizatorii pot marca o pagină web marcată în cazul în care intenționează să fie vizitată mai târziu sau se referă la interesele utilizatorului. Când este necesar, utilizatorul poate naviga la secțiunea de marcaje din browserul Safari și poate vizita pagina din nou.
