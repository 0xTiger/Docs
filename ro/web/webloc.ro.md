## Ce este WEBLOC?

Un fișier WEBLOC este un fișier de comandă rapidă pe macOS care indică o adresă URL web pe internet. Este generat prin tragerea URL-ului web din bara de adrese a browserului pe desktop sau în orice alt folder în care este menit să fie salvat. URL-ul către site-ul de destinație este stocat în acest fișier de comandă rapidă și poate fi vizualizat prin deschiderea fișierului într-un editor de text, cum ar fi Microsoft Notepad, Notepad++ sau Apple TextEdit. Fișierele WEBLOC sunt similare cu fișierele [URL](/ro/web/url/).

## Format de fișier WEBLOC - Mai multe informații

Fișierele WEBLOC sunt stocate în format text simplu. Acestea pot fi generate pe macOS prin tragerea de adrese URL din browsere web, cum ar fi Safari, Google Chrome sau Firefox. Dacă mai multe fișiere WEBLOC vor fi deschise simultan, ar fi necesară o soluție mai rapidă și mai ușoară de deschidere a acestora prin aplicații externe sau API-uri. Aceste API-uri pot citi și converti WEBLOC în PDF și JPEG.
