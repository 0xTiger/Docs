## Ce este un fișier GNE?

Un fișier GNE este o pagină web care este generată și utilizată dinamic de site-ul Flickr de partajare a fotografiilor. Paginile sunt găzduite pe server de unde acestea sunt servite în browserul utilizatorului solicitant. Fișierele GNE sunt similare cu paginile PHP și ASP, dar sunt dezvoltate într-un limbaj de programare proprietar ale cărui detalii nu sunt disponibile pentru referința dezvoltatorului.

## Format de fișier GNE - Mai multe informații

Fișierele GNE au fost folosite de jocul de calculator, Game Never Ending, dezvoltat de [Ludicorp](https://en.wikipedia.org/wiki/Ludicorp). Flicker a fost achiziționat ulterior de Yahoo! în martie 2005.
