## Ce este un fișier DAP?

Un fișier cu extensia .dap este un fișier Microsoft Access Data Page care este creat din Microsoft Access și are acces la datele stocate în baza de date. Datele stocate într-un astfel de fișier web sunt accesibile prin intermediul rețelei (intranet și internet), care este diferită de fișierele [MDB](/ro/database/mdb/) ale Access, care sunt dezvoltate special pentru aplicații desktop. Astfel de baze de date pot fi create folosind Access 2010 și Access Services (o componentă opțională a SharePoint). Aceste baze de date pot fi folosite de persoanele care au conturi SharePoint folosind adresa web dintr-un browser web.
