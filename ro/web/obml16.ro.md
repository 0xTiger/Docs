## Ce este un fișier OBML16?

Un fișier OBML16 este o pagină web salvată folosind browserul web mobil Opera Mini. Se bazează pe Opera Binary Markup Language ([OBML](/ro/web/obml/)) și este un format compact pentru a oferi acces rapid la paginile web salvate. A înlocuit formatul de fișier [OBML15](/ro/web/obml15/) când a fost lansată Opera Mini versiunea 7. Toate aceste variante ale paginilor web Opera Mini au fost actualizate de la formatul de fișier OBML de bază și fiecare actualizare a Opera Mini este compatibilă cu un singur format OBML. Fișierele OBML16 pot fi convertite în [PDF](/ro/pdf/) și [HTML](/ro/web/html/) online.

## Format de fișier OBML16 - Mai multe informații

OBML16 se bazează pe formatul de fișier OBML, care este un format compact pentru salvarea paginilor web. Avantajul utilizării acestui format compact este că reduce dimensiunea conținutului web, rezultând o creștere a vitezei de descărcare. Odată descărcate, utilizatorii Opera Mini pot vizualiza aceste pagini fără a fi nevoie să aibă acces la o conexiune la Internet.
