## Ce este un fișier COMPRESSED?

Un fișier COMPRESSED este o pagină web comprimată Wordpress de dimensiuni mici, generată cu plug-in-ul **WordCompress**. Conține o singură linie de cod pentru reprezentarea paginii web, care este generată prin comprimarea codului HTML, CSS și JavaScript de către plugin. Aceste fișiere COMPRESSE convertite au dimensiuni mici, ceea ce ajută la încărcare și transfer mai rapid pe internet.

### Format fișier comprimat - Mai multe informații

Fișierele COMPRESSE sunt salvate ca fișiere text care pot fi deschise cu orice editor de text, cum ar fi Microsoft Notepad, Notepad++ și Apple TextEdit. Pe lângă editorii de text, fișierele COMPRESATE pot fi deschise cu un software de creare web, cum ar fi Adobe Dreamweaver, care este disponibil atât pentru Windows, cât și pentru Mac OS.

## Despre pluginul WordCompress

Pluginul WordCompress a fost folosit inițial pentru a comprima paginile web WordPress în format de fișier comprimat. Pluginul nu mai este acceptat și este întrerupt. Ar scrie codul sursă al paginii web într-un singur fișier executabil, eliminând toate spațiile albe din codul sursă al paginii. Fișierele COMPRESATE sunt stocate alături de codul sursă original al paginii web pentru a păstra comentariile originale, indentările și alte spații albe incluse în codul sursă original al paginii.
