## Ce este un fișier VDISCO?

Un fișier VDISCO este un fișier de descoperire utilizat de Microsoft Visual Studio în serviciile sale web. Acesta oferă informații despre serviciile web disponibile, cum ar fi adresele URL, spațiile de nume și metodele acestora. Fișierul VDISCO este similar cu formatul de fișier [DISCO](/ro/web/disco/), dar este generat în timpul execuției. Este stocat în aplicația de serviciu Web și este utilizat de Visual Studio pentru a descoperi și utiliza în mod dinamic serviciile Web într-un proiect. Fișierele VDISCO sunt utilizate în combinație cu fișierul [.asmx](/ro/web/asmx/) care conține codul serviciului web real.

Puteți deschide fișierul VDISCO în orice editor de text, cum ar fi Microsoft Notepad, Github Atom și Apple TextEdit. Poate fi deschis și cu Microsoft Visual Studio 2012 și mai târziu pentru a lucra cu el.

## Format de fișier VDISCO

Un fișier VDISCO este salvat și găzduit în format de fișier [XML](/ro/web/xml/), care este un format universal pentru compunerea și publicarea datelor. Conține adresa URL a serviciului, spațiile de nume și metodele în etichete XML standard în care fiecare etichetă conține valoarea respectivă a informațiilor sale.
