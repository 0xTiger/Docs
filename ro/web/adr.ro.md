## Ce este un fișier ADR?

Un fișier ADR (salvat cu extensia .adr) este o listă de marcaje creată de browserul web Opera. Această listă se completează pe măsură ce utilizatorul continuă să marcheze adrese URL web. Aceste marcaje sunt stocate în fișierul ADR, care poate fi salvat și restaurat atunci când este necesar. Deoarece majoritatea browserelor, cum ar fi Chrome și Firefox, salvează marcaje în format [HTML](/ro/web/html/), acestea pot fi convertite în format de fișier ADR pentru importare în browserul Opera. Fișierele ADR pot fi deschise în browserul Opera.

## Format de fișier ADR - Mai multe informații

Detaliile interne ale formatului fișierelor ADR sunt necunoscute. Cu toate acestea, există diferențe între formatele de fișiere ADR exportate cu diferite versiuni de Opera. Acest lucru face dificilă generarea de [merge fișiere ADR](https://superuser.com/questions/471959/how-do-i-merge-several-opera-adr-bookmark-files-to-one-single-file) cu diferite versiuni de Opera. Unele instrumente externe pot fi folosite pentru a le îmbina, deși.
