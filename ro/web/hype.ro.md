## Ce este un fișier HYPE?

Un fișier HYPE este un fișier de proiect salvat folosind software-ul Hype care este utilizat pentru crearea de conținut web cu HTML5. Conține elemente web, cum ar fi grafică, text, animație, informații despre scenă și alte elemente care alcătuiesc în mod colectiv conținutul. Conținutul web dintr-un fișier HYPE poate fi exportat în fișiere web care pot fi publicate pe o pagină web.

Un fișier HYPE poate fi deschis cu software-ul [Tumult Hype](https://tumult.com/hype/).

## Format de fișier HYPE

Fișierele HYPE sunt stocate în format de fișier proprietar, iar detaliile interne ale acestora nu sunt cunoscute public. Conținutul dintr-un fișier HYPE poate fi vizualizat numai atunci când este deschis în software-ul Tumult Hype.

### Conținutul unui fișier HYPE

Un fișier HYPE este creat atunci când utilizați aplicația Hype pentru a crea conținut web în diferite scopuri. Stochează informații despre animațiile bazate pe cadre cheie pentru crearea de conținut, cum ar fi carduri electronice, cărți electronice, prezentări, reclame, infografice și materiale pentru pagini web. Conținutul acestora este alimentat de HTML5, care permite afișarea conținutului pe dispozitive de la desktopuri la iPhone și iPad. Tumult Hype oferă stiluri HTML5 de ultimă generație, [CSS3](/ro/web/css/) și [JavaScript](/ro/web/js/).
