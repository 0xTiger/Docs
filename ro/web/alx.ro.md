## Ce este un fișier ALX?

Un fișier ALX este un fișier ActiveX care este încorporat în paginile Web sub formă de componente [HTML](/ro/web/html/). Controalele ActiveX erau cunoscute oficial ca controale OLE sau OCX. Fișierele ALX pot fi folosite pentru a insera o varietate de funcționalități într-o pagină web, cum ar fi adăugarea unui control temporizator pentru actualizarea periodică a conținutului paginii sau adăugarea unui control fereastră Popup pentru afișarea ajutorului în stilul sfaturii bazate pe acțiunea utilizatorului. Îngrijirea fișierelor ALX legată folosind<Param> etichetă în codul HTML al unei pagini web. Puteți deschide fișiere ALX în Microsoft Internet Explore, Microsoft ActiveX Control Pad și editori de text, cum ar fi Notepad și Notepad++.

## Format de fișier ALX - Mai multe informații

Fișierele ALX sunt în format text simplu și pot fi editate prin deschiderea acestora într-un editor de text. Controalele ActiveX pot fi incluse într-o pagină web prin plasarea pe pagina web. Cu toate acestea, pentru a construi pagini web dinamice, controalele sunt integrate împreună cu ajutorul [JScript](/ro/web/js/). Controalele de pe o pagină web sunt asociate cu evenimentele și valorile controalelor, oferind astfel control complet pentru interacțiunile utilizatorului.
