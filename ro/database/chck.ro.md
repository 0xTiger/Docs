## Ce este un fișier CHCK?

Un fișier CHCK este creat ca urmare a verificărilor de monitorizare a stării de sănătate pe Microsoft Exchange Server. Conține informații despre starea generală și operaționalizarea bazei de date a serverului. Microsoft Exchange Server este folosit în principal ca server de e-mail și vă permite să vă păstrați contactele și calendarele de afaceri. Administratorii serverului Exchange folosesc EseUtil pentru a verifica starea bazei de date.

## Format de fișier CHCK

Formatul intern al fișierului CHCK nu este cunoscut și nu există material de referință disponibil pentru citirea/scrierea fișierelor CHCK.
