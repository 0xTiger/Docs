## Ce este un fișier ACCDE?

Un fișier cu extensia .accde este un format de fișier Microsoft Access 2007 pentru crearea fișierelor de bază de date Execute Only pentru a proteja codul personalizat al bazei de date. Compilează tot codul VBA doar în executabil, astfel încât să nu poată fi vizualizat sau editat. Formatul de fișier ACCDE înlocuiește formatul de fișier MDE care a fost folosit de versiunile anterioare ale Microsoft Access. ACCDE sunt fișiere [ACCDB](/ro/database/accdb/) în modul doar executabil și mai compacte din cauza absenței oricărui cod sursă. Dimensiunea compactă a acestor fișiere îmbunătățește utilizarea memoriei, precum și performanța aplicației. Fișierele ACCDE pot fi deschise cu Microsoft Access 365.

## Format de fișier ACCDE

Toate fișierele Microsoft Access 2007+ utilizează Jet ACE (motor de bază de date Access) care utilizează familia de formate de fișiere ACCDB. Cu toate acestea, Microsoft nu oferă o implementare open source sau o specificație tehnică detaliată.
