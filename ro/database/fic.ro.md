## Ce este un fișier FIC?

Fișierul FIC din WinDev este o versiune compilată a unei baze de date HyperFileSQL. Acronimul FIC înseamnă Fichier de Code” în franceză, care se traduce prin Code File” în engleză. În WINDEV, un fișier FIC este un fișier compilat care conține codul executabil al unui proiect. Fișierul are extensia .fic și este creat atunci când compilați un proiect WINDEV. Odată ce un proiect este compilat într-un fișier FIC, acesta poate fi distribuit utilizatorilor ca program executabil sau pachet de instalare.

Fișierul FIC este legat de WINDEV, un instrument de dezvoltare software creat de PCSoft, o companie franceză. Permite dezvoltatorilor să creeze aplicații Windows, web și mobile folosind o singură bază de cod. Include un mediu de dezvoltare vizuală, un limbaj de programare (WLangage) și o varietate de controale și funcții pre-construite pentru sarcini comune. Este folosit în principal pentru aplicații de afaceri și are o bază puternică de utilizatori în Europa.

## Relația cu baza de date HyperFileSQL Mobile

În WinDev, un fișier FIC care este creat pentru o bază de date HyperFileSQL Mobile conține codul executabil pentru baza de date, precum și datele conținute în baza de date. HyperFileSQL Mobile este un sistem de gestionare a bazelor de date relaționale care este inclus cu WINDEV și este conceput special pentru aplicații mobile și offline. Fișierul FIC poate fi distribuit ca fișier executabil autonom sau ca parte a unui pachet de instalare și poate fi rulat pe dispozitivul țintă pentru a accesa și gestiona datele din baza de date HyperFileSQL Mobile. Când fișierul FIC este rulat, acesta va deschide baza de date și va face datele disponibile pentru utilizare în aplicația mobilă.
