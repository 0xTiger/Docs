## Ce este un fișier ADP?

Un fișier cu extensia .adp este un fișier de proiect Microsoft Access care utilizează arhitectura componentelor OLE DB pentru a oferi o conexiune directă și eficientă la o bază de date Microsoft SQL Server. Astfel de fișiere nu conțin tabelele reale, diagramele bazei de date sau alte elemente ale bazei de date. Fișierele ADP pot fi create cu Access 2007 și 2010. Fișierele create cu versiunile anterioare ale Access pot fi deschise și Access 2007 și 2010. Fișierele ADP pot fi deschise utilizând Microsoft Access 365.

## Format de fișier ADP - Mai multe informații

Un proiect Access creat ca ADP vă permite să faceți modificări de proiectare la obiectele serverului SQL, cum ar fi tabelele și vederile. De asemenea, permite crearea, editarea și utilizarea caracteristicilor SQL, cum ar fi diagrame de baze de date, proceduri stocate și funcții definite de utilizator. Acest lucru oferă un avantaj față de conectarea la o bază de date server SQL, unde nu puteți face modificări în proiectare și vă puteți conecta doar la tabele și vederi SQL server.

### Tipuri de date și suport pentru diagrame

**Tipuri de date data/ora**

Următoarele tipuri de date noi de date/ora sunt acceptate în Access 2010.

* TIMP
* DATA
* DATETIME2
* DATETIMEOFFSET

**Tipuri de date cu lungime variabilă**

Următoarele tipuri de date cu lungime variabilă pot fi utilizate în proiectele Access 2010:

* VARBIN(MAX)
* VARCHAR(MAX)
* NVARCHAR(MAX)
