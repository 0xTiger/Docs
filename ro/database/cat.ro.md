## Ce este un fișier CAT?

Fișierul CAT aparține software-ului Advanced Disk Catalog, dezvoltat de ElcomSoft Co. Ltd, companie rusă de software care dezvoltă diverse instrumente software de securitate și criminalistică. Unele dintre produsele sale includ software pentru recuperarea parolei, achiziția de dispozitive mobile și extragerea în cloud. Compania a fost fondată în 1990 și are birouri în Rusia și Statele Unite. Sunt bine cunoscuți pentru software-ul lor criminalistic și, în general, software-ul lor este folosit atât de companii private, cât și de agenții guvernamentale pentru a efectua analize și investigații criminalistice.

Advanced Disk Catalog is a cataloguing program for Windows XP/2003. Versatilitatea ADC este derivată din faptul că nu se bazează pe un motor de baze de date, ceea ce îl face extrem de eficient și compact. Interfața sa de utilizator, similară cu Windows Explorer, permite organizarea ușoară a datelor stocate pe diferite tipuri de suporturi, cum ar fi hard disk-uri (inclusiv unități în rețea), dischete, discuri optice, discuri ZIP și JAZ etc.

## Cum se deschide fișierele CAT?

Descărcați software-ul Elcomsoft Advanced Disk Catalog. Pe Windows, puteți face clic dreapta pe exploratorul de fișiere și apoi selectați Deschidere cu meniul contextual, specificați calea software-ului Elcomsoft Advanced Disk Catalog pentru a deschide fișierul cat.
