## Ce este un fișier ACCFT?

Un fișier cu extensia .accft este un fișier șablon Microsoft Access care conține definiția tipurilor de date de câmp pentru tabelele unei baze de date. Microsoft Access este un sistem de gestionare a bazelor de date relaționale (RDBMS) desktop care este utilizat pentru a crea baze de date bazate pe tabele, formulare, interogări și vizualizări. Fișierul ACCFT poate fi utilizat pentru a defini tipurile de date pentru câmpurile de tabel și tabelele bazei de date pot fi create folosind acest fișier șablon ca punct de plecare. Poate fi deschis cu Microsoft Access 2016 pentru a crea tabele noi. Fișierele ACCFT sunt recunoscute după tipul MIME **application/msaccess**.

## Format de fișier ACCFT

Fișierele ACCFT sunt fișiere binare ale căror specificații de format de fișier nu sunt disponibile public. Detaliile sale interne, cum ar fi informațiile din antet, structura fișierului, subsolurile etc., nu sunt disponibile momentan.
