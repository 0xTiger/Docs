## Ce este un fișier SXC? ##

Formatul de fișier SXC (Sun XML Calc) aparține unei suite de birou numită OpenOffice.org. Acest format se ocupă, în general, de nevoile utilizatorilor de foi de calcul, deoarece este un format de fișier de foi de calcul bazat pe XML. Formatul SXC acceptă formule, funcții, macrocomenzi și diagrame împreună cu DataPilot, care este o caracteristică incredibilă, deoarece individualizează și oferă rezumate ale datelor brute importate. Fișierele create cu acest software sunt salvate cu extensia .sxc.


## Format de fișier SXC ##

Fișierele, având extensia .sxc, pot fi deschise cu Microsoft Excel, care este un software pentru foi de calcul din suita Microsoft Office. Există și alte programe care pot deschide fișiere cu extensia .sxc, de exemplu Libre Office, StarOffice etc.

Tipurile de conținut pentru acest format sunt:

* application/vnd.sun.xml.calc
* application/vnd.sun.xml.calc.template

## Compatibilitate fișier ##

Fișierele SXC sunt compatibile cu Apache Open Office calc, iar datele pot fi exportate în fișiere Microsoft Excel și în alte formate.

## Gestionarea fișierelor ##

Comenzile ImportMatrix și ExportMatrix pot citi și scrie fișiere SXC. Când sursa este un fișier SXC și ținta este, de asemenea, un fișier SXC, aceasta face ca **f** să fie considerat fișier Sun XML Calc Spreadsheet. Atât Import și Export Matrix acceptă acest format, care este detectat automat când formatul nu este specificat și numele fișierului se termină în .sxc.

Pentru a deschide fișierul SXC, este necesar să fie asociat fișierului un program adecvat. Ori de câte ori urmează să fie deschis un fișier cu extensia .sxc, asigurați-vă că nu are erori și că nu este corupt. Orice eroare din fișier poate fi tratată cu ușurință de către utilizator și aceasta nu necesită suport tehnic de cele mai multe ori.
