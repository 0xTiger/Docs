## Ce este un fișier XLSMHTML?

Un fișier cu extensia .xlsmhtml este o versiune [HTML](/ro/web/html/) a fișierului XLS original care este exportat din Microsoft Excel pentru a vizualiza datele într-un browser. Formatul a fost întrerupt de mult de Excel și nu mai este acceptat. Fișierul XLSMHTML exportat a urmat formatul de fișier XLS și acestea puteau fi deschise și vizualizate în browserele web. Excel oferă exportarea datelor foilor de calcul în formatele de fișier [MHT](/ro/web/mht/) și [MHTML](/ro/web/mhtml/) și sunt utilizate pe scară largă acum.

## Format de fișier XLSMHTML - Mai multe informații

Fișierele XLSMHTML au fost generate cu cod HTML pentru a reprezenta datele foii de calcul într-un format similar cu XLS. Codul HTML din interiorul fișierului XLSMHTML este aranjat în așa fel încât fișierul de ieșire să arate similar cu fișierul original atunci când este deschis într-un browser web. Specificațiile de format de fișier ale fișierului XLSMHTML nu sunt făcute publice ca atare de Excel și nu sunt disponibile astfel de referințe pentru ca dezvoltatorii să înțeleagă și să dezvolte aplicații pentru acesta.
