## Ce este un fișier XLA?

Un fișier cu extensia .xla este un format de fișier Microsoft Excel Add-Ins care adaugă module, funcții suplimentare și alte instrumente la Microsoft Excel. Ad-In-urile oferă funcționalități suplimentare foilor de calcul Excel și sunt acceptate în versiunea 2007 sau mai recentă. Fișierele XLA au tipul media mime application/vnd.ms-excel. Acestea pot fi deschise folosind Microsoft Excel, care este disponibil atât pentru Windows, cât și pentru MacOS.

## Format de fișier XLA

Fișierele XLA se bazează pe format de date binare. Cu toate acestea, informații suplimentare despre specificațiile formatului său de fișier nu sunt disponibile. Presupunerea generală este că XLA urmează [[MS-XLS] - Structura formatului de fișier binar Excel](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx).
