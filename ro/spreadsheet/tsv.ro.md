## Ce este un fișier TSV?

Un format de fișier cu valori separate prin file (TSV) reprezintă date separate cu file în format text simplu. Formatul de fișier, similar cu [CSV](/ro/spreadsheet/csv/), este utilizat pentru organizarea datelor într-o manieră structurată pentru a importa și exporta între diferite aplicații. Formatul este utilizat în principal pentru importul/exportul și schimbul de date în aplicațiile și bazele de date pentru foi de calcul. Fiecare înregistrare dintr-un fișier TSV este conținută într-o singură linie de fișier text în care fiecare valoare de câmp este separată de un caracter de tabulație. Tipul media pentru formatul de fișier TSV este text/valori separate prin file.
