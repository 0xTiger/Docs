## Ce este un fișier VGA?

Un fișier VGA este un fișier de unitate Windows pentru suport pentru monitoare VGA. A fost folosit în versiunile timpurii ale Microsoft Windows pentru ajustarea monitoarelor VGA la nivel de sistem de operare (OS). Un fișier VGA folosit pentru a permite utilizatorilor să ajusteze rezoluția și adâncimea culorii afișajului folosind panoul de control al monitorului.

## Format de fișier VGA

Formatul de fișier intern al formatului de fișier VGA nu este cunoscut pentru referința dezvoltatorului. Este referit de sistemul de operare Windows pentru setarea aspectului vizual al monitorului VGA.
