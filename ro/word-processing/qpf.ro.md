## Ce este un fișier QPF?

Un fișier cu extensia .qpf este un fișier text creat cu software-ul QuickPad care este o aplicație gratuită oferită de Klumbu, ca versiune evoluată a Notepad pentru sistemul de operare Windows. Fișierele QPF pot fi deschise și vizualizate numai în QuickPad și oferă o securitate minimă a datelor sensibile în acest fel. Textul din interiorul documentului nu este criptat folosind nicio parolă. În schimb, este codificat de software pentru a-l face oarecum sigur.

## Format de fișier QPF

Fișierele QPF sunt fișiere text care sunt create și salvate într-o formă care poate fi citită de om. Singura diferență este că pentru a citi aceste fișiere, acestea trebuie să fie deschise în QuickPad. QuickPad oferă funcții precum Word Wrap, formatarea textului, salvare automată, modul de focalizare și multe altele.
În plus, textul poate fi importat din formatul de fișier INI și TXT.
