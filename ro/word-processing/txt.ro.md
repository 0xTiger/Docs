## Ce este un fișier TXT?

Un fișier cu extensia .TXT reprezintă un document text care conține text simplu sub formă de linii. Paragrafele dintr-un document text sunt recunoscute prin returnări de transport și sunt utilizate pentru o mai bună aranjare a conținutului fișierului. Un document text standard poate fi deschis în orice editor de text sau aplicație de procesare de text pe diferite sisteme de operare. Tot textul conținut într-un astfel de fișier este în format care poate fi citit de om și este reprezentat prin secvență de caractere.

Fișierele text pot stoca cantități mari de date, deoarece nu există nicio limitare în ceea ce privește dimensiunea conținutului. Cu toate acestea, editorii de text care deschid fișiere atât de mari trebuie să fie inteligenți pentru încărcarea și afișarea acestora. Aproape toate sistemele de operare vin cu editori de text care vă permit să creați și să editați fișiere text. De exemplu, sistemul de operare Windows vine cu Notepad și Wordpad în acest scop. În mod similar, MacOS vine cu TextEdit pentru crearea și editarea documentelor text. Există, totuși, și alte editoare de text gratuite disponibile pe internet, care vă oferă capacitatea de a lucra cu documente text precum Notepad++, care este mult mai avansat în ceea ce privește funcționalitatea.

## Specificații de format de fișier ##

Formatul de fișier text nu are specificații speciale de format de fișier. Fișierele text au tip MIME „text/plain” și au formatare redusă sau deloc. Acest lucru permite editorilor de text să deschidă astfel de fișiere fără alte cerințe. Setul de caractere implicit al fișierelor text este ASCII, care este utilizat pentru crearea și afișarea conținutului fișierului text. Caracterele sunt codificate folosind setul de caractere ASCII, dar acest lucru impune limitarea utilizării caracterelor precum semnul liră sterlină, semnul dolar și euro, care nu pot fi reprezentate folosind setul de caractere ASCII. Astfel, fișierele text pot fi salvate și în format Unicode, UTF-8 fiind cel mai utilizat.

### Format fișier text Windows ###

Fișierele text pe sistemul de operare Windows sunt formate din mai multe linii în care fiecare linie este alcătuită dintr-o secvență de caractere. Fiecare linie implicită de utilizator este definită prin combinarea a două caractere, adică returnarea căruciorului (CR) și Line Feed (LF). Fișierele text Windows pot fi în codificare ANSI, OEM, Unicode sau UTF-8. Codificarea UTF-16 ajută la salvarea informațiilor într-un fișier text care necesită doi octeți pentru reprezentare. Astfel de fișiere încep de obicei cu Byte Order Mark (BOM) care comunică caracterul final al conținutului fișierului. Trebuie remarcat faptul că alte aplicații de pe sistemul de operare Windows pot stoca informații în format de fișier text, dar cu extensii de fișiere diferite pentru a reprezenta textul specific aplicației. De exemplu, limbajele de programare salvează de obicei codul în fișierul text, dar cu propriile extensii.

### Format fișier text Unix ###

Toate aceste sisteme amendează un fișier text ca un fișier ale cărui caractere sunt organizate în zero sau mai multe linii. Fiecare linie este o secvență de zero sau mai multe caractere non-line și un caracter de sfârșit de linie nouă, în mod normal LF.
