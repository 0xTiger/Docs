## Ce este un fișier IPYNB?

Un fișier cu extensia .ipynb este un fișier Jupyter Notebook care conține întregul conținut al Notebook-ului creat cu sesiunea aplicației web Jupyter Notebook. [Jupyter Notebook](https://jupyter.org/) este o aplicație web interactivă utilizată pentru crearea și partajarea documentelor de calcul pentru analiză folosind Python.

Un fișier IPYNB servește ca înregistrare completă de calcul a unei sesiuni și include informații precum intrări și ieșiri de calcul, funcții matematice și reprezentare bogată a obiectelor rezultate, cum ar fi imagini și text explicativ.

## Format de fișier IPYNB

IPYNB sunt salvate intern în formatul de fișier [JSON](/ro/web/json/), care este un format de fișier standard deschis pentru partajarea datelor. Din acest motiv, fișierele IPYNB sunt citite de om și ușor de înțeles. JSON facilitează, de asemenea, partajarea acestor fișiere cu alții.

## Cum se convertesc fișierele IPYNB?

Fișierele IPYNB pot fi exportate în mai multe formate de fișiere diferite folosind comanda nbconvert din interfața Jupyter. Aceasta include conversia IPYNB în:

* PDF
* HTML
*Text reStructurat
* LaTex

## Cum să vizualizați fișierul IPYNB?

Există două metode de a vizualiza sau deschide fișierele IPYNB.

1. Utilizatorul poate încărca documentul blocnotes .ipynb la o adresă URL publică și îl poate deschide din Jupyter Notebook Viewer<nbviewer> care redă blocnotesul ca o pagină web statică din URL.
1. Descărcați fișierele și deschideți-le în propria instanță a Jupyter Notebook
