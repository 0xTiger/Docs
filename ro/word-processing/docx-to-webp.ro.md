## Despre conversia DOCX în WEBP

WEBP este formatul modern de fișier imagine care a redus dimensiunea fișierului fără a compromite dimensiunea fișierului. Este cel mai potrivit format de fișier imagine care este folosit pentru web datorită dimensiunilor sale mici și ajută la încărcarea mai rapidă a imaginilor. Dacă utilizați Microsoft Word pentru a crea documente, acestea sunt salvate în format de fișier DOCX în mod implicit (de la lansarea Microsoft Office 2007). DOCX la WEBP cuprinde fiecare pagină a unui document WORD convertită în [WEBP](/ro/image/webp/) imagine care poate fi utilizată independent. Ieșirea acestui proces este un set de imagini WEBP în care fiecare imagine este reprezentarea grafică a paginii corespunzătoare din fișierul DOCX.
