## Ce este un fișier WPS?

Un fișier cu extensia .wps este un document Microsoft Works care a fost folosit înainte de 2006 ca document Word. Era similar cu noul format de fișier [DOC](/ro/word-processing/doc/) introdus până atunci, dar nu era atât de complicat ca noul format de fișier. De exemplu, nu a acceptat funcționalitatea macro și opțiunile avansate de formatare. Fișierele WPS pot fi deschise în continuare cu procesoare de text moderne, dar pentru Windows 10, trebuie să instalați un convertor pentru a încărca aceste fișiere. Microsoft Works a fost înlocuit cu procesorul de documente Microsoft Word în 2010.

## Specificații de format de fișier

Specificațiile de format de fișier ale documentului WPS nu sunt disponibile public, deoarece Microsoft nu a deschis specificațiile pentru utilizatorii finali. WPS sunt fișiere binare și pot fi deschise cu filtre de conversie a formatului de fișier create de Microsoft pentru procesorul Word. LibreOffice, NeoOffice și OxygenOffice au inclus o bibliotecă generală C++, libwps, care poate extrage text din multe versiuni diferite de Microsoft Works.
