## Ce este un fișier WLX?

Un fișier WLX este un fișier GIS creat de software-ul MapSource care vine cu produsele pentru hărți Garmin. Conține o locație geografică (sub formă de latitudine-longitudine) în format XML și o locație mondială către un site web de cartografiere, cum ar fi Hărți Google sau Hărți Bing. Fișierul WLX acționează ca un link de comandă rapidă și deschide locația stocată pe site-uri web terță parte la poziția înregistrată latitudine-longitudine.

Un alt fișier similar (deși nu are legătură cu GIS) este formatul de fișier [URL](/web/url/) care stochează adresa web și direcționează utilizatorii către această adresă URL atunci când este deschis.

## Format de fișier WLX - Mai multe informații

Fișierele WLX sunt stocate în format de fișier text care poate fi deschis în orice editor de text pentru a vizualiza și actualiza conținutul acestuia. MapSource este un software bazat pe GIS care vă permite să vizualizați hărți, puncte de referință, rute și trasee. De asemenea, puteți transfera aceste date spațiale către sau de la un dispozitiv GPS Garmin.

MapSource nu mai este acceptat de Garmin și a fost înlocuit de BaseCamp, care este disponibil pentru descărcare gratuită.
