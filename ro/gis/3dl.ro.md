## Ce este un fișier 3DL?

Un fișier 3DL este un fișier iGO Landmark utilizat de software-ul de navigare iGO. Stochează informații despre locație pentru punctele de interes (POI) și repere. Aceste informații pot include numele, locația și alte detalii despre POI sau reper. Formatul de fișier iGO Landmark este proprietar și este utilizat numai de [iGO navigation software](https://en.wikipedia.org/wiki/IGO_(software)). Fișierele 3DL pot fi utilizate împreună cu fișierele de date ale hărților pentru a oferi utilizatorilor îndrumări de navigare.

## Format de fișier 3DL

Formatul de fișier 3DL este proprietar și nu este documentat public. Software-ul de navigare iGO, care folosește formatul de fișier 3DL, este dezvoltat și distribuit de o companie numită NNG (fostă Nav N Go). Ca atare, este posibil să nu puteți găsi informații tehnice detaliate despre format pe [NNG website](https://www.nng.com/) sau în altă parte pe internet.
