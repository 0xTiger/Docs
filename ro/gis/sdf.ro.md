## Ce este un fișier SDF?

A Spatial Data File (SDF) is a Geodatabase file format that was developed by Autodesk. It stores geospatial data and is used by Autodesk GIS program MapGuide and AutoCAD Map 3D. SDF file format is based on the single file database file format SQLite3. Este considerat un format de fișier optimizat pentru seturi mari de date de informații spațiale.

Puteți deschide un fișier SDF utilizând Autodesk AutoCAD Map 3D 2022 și Autodesk AutoCAD Civil 3D 2023.

## Format de fișier SDF - Mai multe informații

Formatul de fișier SDF este stocat pe disc ca fișiere binare. Se bazează pe baza de date SQLite care utilizează componente de stocare de nivel scăzut pentru serializarea binară a datelor. Un fișier SDF conține mai multe clase de caracteristici per fișier și mai multe proprietăți de geometrie pentru fiecare clasă de caracteristici. Datele dintr-un fișier SDF sunt accesibile la viteză mare datorită indexării fiecărei proprietăți de geometrie folosind un arbore R.
