## Ce este un fișier MXD?

Un fișier MXD este un fișier Map Document utilizat de software-ul ESRI ArcMap. Este un format de fișier proprietar care stochează diferite tipuri de informații, cum ar fi aspectul și designul hărților, datele utilizate pentru generarea hărților și simbolologia aplicată datelor. Fișierul MXD este, prin urmare, un format de fișier compus care conține mai multe tipuri diferite de elemente de hartă într-un singur fișier. Fișierele MXD sunt utilizate de fișierul ArcGIS Map Package ([.mpk](/gis/mpk/)).

## Format de fișier MXD

Fișierele MXD sunt salvate în format de fișier proprietar ArcGIS, iar detaliile sale interne nu sunt disponibile pentru referință de către dezvoltator. Elementele încorporate ale unui fișier MXD sunt organizate în unități, cunoscute sub numele de stocări și fluxuri.
