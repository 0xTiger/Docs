## Ce este un fișier ACT? ##

Un format ACT sau Adobe Color Table (cunoscut și ca fișier Color Lookup Table) este un tip de bază de date Adobe Photoshop care stochează un set de culori predefinite. Puteți adăuga sau elimina culori atunci când salvați o imagine pentru publicarea online, pentru a favoriza o imagine de calitate superioară sau o dimensiune mai mică a fișierului. Este posibil să aveți un format audio criptat ADPCM dacă nu este utilizat cu Illustrator. Modularea Adaptive Differential Pulse Code este o tehnică de compresie audio utilizată de diverse dispozitive muzicale și dispozitive de ascultare.

Formatul de fișier ACT este folosit și de sistemul de fișiere Alma CAD/CAM. Acestea sunt liniile directoare folosite de mașinile-unelte 3D pentru a descoperi cum să tăiați orice. Un jucător Genesis3D, o acțiune de dezvoltator de jocuri DS sau un fișier FoxPro Detailing Wizard Schematic sunt toate exemple de ACT.

## Specificația formatului fișierului ACT

Multe înregistratoare care folosesc sistemul ACT o fac doar pentru cea mai mică opțiune de înregistrare; dacă calitatea este crescută, se folosește un format separat, de obicei WAV, rezultând documente mult mai mari.

Există mai multe generații de ACT; din iunie 2009, melodiile create de dispozitivele ulterioare pur și simplu nu pot fi recunoscute de fiecare player muzical standard gratuit sau de aplicație de conversie, rămânând doar instrumentele MP3 incluse.

Optimizarea imaginilor pentru distribuția Web se face cu fișiere ACT. Clienții pot testa calitatea imaginii față de bazele de date de imagini generate prin creșterea sau scăderea numărului de culori din CLUT.

Atingând opțiunea Tabel de culori și selectând Încărcare date de culoare din caseta de dialog Export pentru Web, tabelele de culori Adobe pot fi încărcate. Directorul */Presets/en_US/Save* for Web Settings/Color Tables/ conține mai multe fișiere ACT preconfigurate care au venit împreună cu sistemul Photoshop.

Scripturile Act pot fi generate numai în Genesis 3D sau prin transformarea altor modele 3D la standardul ACT folosind Genesis3D Actor Designer
