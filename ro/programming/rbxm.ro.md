## Ce este un fișier RBXM?

Un fișier RBXM este un format de fișier model 3D utilizat în Roblox Studio IDE pentru stocarea obiectelor 3D ale jocului Roblox multiplayer online. Conține obiecte 3D create pentru joc folosind IDE. Exemplele de obiecte 3D dintr-un fișier RBXM includ cămăși, pantaloni, pălărie, clădiri, vehicule, animale, avioane, mașini sau plante. Fișierele RBMX pot fi convertite în [formatul fișierului OBJ 3D](/ro/3d/obj/) folosind IDE-ul Roblox. Roblox IDE poate fi folosit pentru a deschide fișiere RBMX.

Un alt format de fișier Roblox popular este [RBXL](/ro/programming/rbxl/), care este un format de fișier de locație și conține locația definită în cuvânt 3D în jocul Roblox.

## Format de fișier RBMX - Mai multe informații

Fișierele RBXM sunt create și salvate în format de fișier binar, iar specificațiile interne ale formatului de fișier nu sunt disponibile public. Acesta este diferit de formatul de fișier RBXMX care se bazează pe formatul de fișier [XML](/ro/web/xml/). Există o mulțime de instrumente/arhive disponibile pe Github care pot fi folosite pentru a [procesa fișierele Roblox RBXM](https://github.com/search?q=rbxm).
