## Ce este un fișier NX?

Un fișier NX este un fișier de joc video creat pentru a fi utilizat cu consola de jocuri virtuală, **LowRes NX**. Este scris în limbajul de programare BASIC și conține codul sursă al jocului care definește modul în care funcționează jocul. Puteți crea propriile fișiere NX pentru a fi utilizate cu aplicația care poate fi încărcată și utilizatorii pot juca jocul.

Puteți deschide un fișier NX într-un editor de text, LowRes NX sau orice editor de cod sursă, cum ar fi Microsoft Visual Studio Code.

## Format de fișier NX

Un fișier NX este salvat pe disc ca fișier text simplu care conține instrucțiunile jocului. Fiecare fișier NX are codul sursă al jocului, care constă din mai multe intrări ROM de cartuș. Aceste intrări ROM au informații despre componentele jocului, cum ar fi grafică, hărți de nivel, audio și alte date non-sursă ale jocului. Intrările ROM ale unui program pot fi adăugate sau editate folosind programele de editare NX care vin la pachet cu LowRes NX.

## Cum se creează fișiere NX?

### Creați fișiere NX pe computer

Puteți crea fișiere NX pe computer folosind:

* Editore de text - cum ar fi Microsoft Notepad, Notepad++ sau Github Atom
* Editore de cod sursă - cum ar fi codul Microsoft VIsual Studio

### Creați fișiere NX pe iOS

LowRes NX are versiunea iOS care poate fi folosită de dezvoltatori pentru a crea fișiere NX din această aplicație.
