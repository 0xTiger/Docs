## Ce este un fișier ACD?

Un fișier ACD este un fișier de proiect care conține informații pentru programarea platformelor [Rockwell Automation](https://www.rockwellautomation.com) Logix. Include instrucțiuni pentru controlori și procesatori. Este utilizat pentru platformele Logix precum ControlLogix, CompactLogix, FlexLogix, SoftLogix5800 și sistemele DriveLogix.

Aplicațiile care pot deschide fișiere ACD includ Rockwell Automation RSLogix 5000.
