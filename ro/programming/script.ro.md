## Ce este un fișier SCRIPT?

Un fișier SCRIPT este un fișier sursă care conține script în orice limbaj de scripting. Acest lucru îl face un fișier de scripting generalizat care conține variabile și funcții. Deoarece fiecare limbaj de scripting poate fi diferit, scriptul scris în interiorul unui fișier SCRIPT trebuie executat folosind un motor de scripting adecvat.

## Format de fișier SCRIPT - Mai multe informații

Fișierele SCRIPT sunt stocate pe disc în fișiere text simplu. Acestea pot fi deschise și editate cu orice editor de text, cum ar fi Microsoft Notepad, Notepad++ și Apple TextEdit. Deoarece limbajul de scripting poate fi diferit, acesta poate fi executat prin redenumirea extensiei de fișier la extensia implicită a limbajului de scripting respectiv. De exemplu, există mai multe limbaje de scripting comune, cum ar fi JavaScript ([.JS](/ro/web/js/)), VBScript ([.VBS](/ro/programming/vbs/)), AppleScript (.SCPT), PHP ([ .PHP](/ro/programming/php/)) și ASP ([.ASP](/ro/web/asp/)).
