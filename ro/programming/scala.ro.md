## Ce este un fișier SCALA?

Un fișier SCALA este un fișier de cod sursă creat cu limbajul de programare Scala (limbaj scalabil). Conține codul sursă scris în programarea orientată pe obiect și funcțional, care este tema de bază a limbajului Scala. Scale facilitează construirea de sisteme de înaltă performanță cu rulajele sale JVM și JavaScript. Acest lucru permite în continuare suportul funcțiilor de ordin superior și imbricarea definițiilor de funcții. Dezvoltarea Scala a fost începută în 2003 de Martin Odersky și grupul său de cercetare de la EPFL.

## Format de fișier SCALA

Un fișier SCALA este salvat ca fișier text simplu, care poate fi deschis cu orice editor de text, cum ar fi Scala IDE pentru Eclipse, Notepad, Notepad++ și TextEdit. Un singur fișier SCALA poate avea cod sursă care cuprinde clase, interfețe, șiruri brute și metode supraîncărcate de operator. Cea mai bună caracteristică a Scala este că codul poate fi compilat în octet Java care poate fi rulat pe JVM.

### Scala Language pe scurt

* **Seamless Java Interop** - Stivele Scala și Java pot fi amestecate pentru o integrare perfectă datorită faptului că Scala rulează pe JVM.
* **Type Interface** - Deci sistemul de tip nu se simte atât de static. Nu funcționează pentru sistemul de tip. Lăsați sistemul de tip să funcționeze pentru dvs.!
* **Concurență și distribuție** - Suportă operațiuni paralele de date pe colecții, utilizează actori pentru concurență și distribuție sau futures pentru programarea asincronă.
* **Potrivirea modelelor** - Suport pentru potrivirea cu ierarhii de clasă, secvențe, constante și multe altele.
* **Funcții de ordin superior** - Funcțiile sunt obiecte de primă clasă care pot fi compuse cu siguranță de tip garantată.
