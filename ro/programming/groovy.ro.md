## Ce este un fișier GROOVY?

Un fișier GROOVY este un fișier cod sursă scris în limbajul de programare Groovy. Codul scris în interiorul unui fișier GROOVY este similar cu limbajul orientat pe obiecte [Java](/ro/programming/java/), care facilitează proiectarea și dezvoltarea aplicațiilor. Un fișier GROOVY este compilat în bytecode Java virtual machine (JVM) și este compatibil cu alte coduri și biblioteci Java.

## Format de fișier GROOVY - Mai multe informații

Fișierele GROOVY conțin cod sursă scris în sintaxa Groovy. Poate conține cod care poate fi folosit atât ca limbaj de programare, cât și ca limbaj de scripting pentru Platforma Java. Limbajul groovy are caracteristici similare cu cele ale Python, Ruby și Smalltalk.
