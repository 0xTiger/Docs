Fișierele de programare sunt fișiere de cod sursă care sunt utilizate de limbajele de programare pentru a scrie aplicații software pentru a îndeplini anumite sarcini. Fiecare limbaj de programare are propria sa sintaxă și poartă propriul set de fișiere pentru scrierea codului sub formă de biblioteci, fișiere antet, clase sau constante. În funcție de limbajul de programare utilizat, formatul fișierului codului sursă va fi diferit pentru a respecta regula de sintaxă definită de acel limbaj.

Extensiile obișnuite ale fișierelor de programare sau dezvoltatori și formatele lor asociate includ **[C#](/ro/programming/cs/)** (CSharp Source Code File), **[CPP](/ro/programming/cpp/)** (C++ Source Code). Fișier) și **[VB](/ro/programming/vb/)** (Fișier cod sursă Visual Basic).
