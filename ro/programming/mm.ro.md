## Ce este un fișier MM?

Un fișier MM este un fișier de cod sursă care poate conține atât cod de programare Objective-C, cât și Objective-C++. Este folosit pe MacOS și se diferențiază de fișierele .M care conțin numai cod Objective-C. Când compilatorul vede extensia .mm, este capabil să compileze clasele C++ prezente în fișier. Deoarece un singur fișier de cod sursă poate conține o combinație de limbaje [C](/ro/programming/c/) și [C++](/ro/programming/cpp/), redenumirea fișierelor .m cu extensia .mm permite setarea caracteristicilor C++ in. Aplicațiile care pot deschide fișierul .MM includ Apple TextEdit, Microsoft Notepad, Notepad++ și Microsoft Wordpad.
