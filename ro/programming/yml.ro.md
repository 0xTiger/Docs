## Ce este un fișier YML?

YML este un fișier text în limbajul de serializare a datelor, [YAML](/ro/programming/yaml/) (Yet Another Markup Language), care este adesea folosit pentru a crea fișiere de configurare. Poate funcționa în combinație cu orice limbaj de programare, inclusiv C/C++, Ruby, Python, Java, PHP, C# și altele. Fiind independent de orice limbaj de programare specific, este deschis, interoperabil și ușor de citit de om. Fișierele YML pot fi deschise și editate cu editori de text obișnuiți, cum ar fi Microsoft Notepad, Notepad++, Github Atom, Apple TextEdit și Visual Studio IDE.

## Format de fișier YML - Mai multe informații

Fișierele YML stochează setările aplicației ca configurație. Structura sa de bază este o hartă care poate fi numită și director sau hash. Intrările dintr-un fișier YML constau dintr-o pereche cheie și valoare. În mare parte, fișierele de configurare .yml sunt salvate ca `configure.yml`, deși unele limbaje de programare, cum ar fi Ruby on Rails, salvează configurația conectivității bazei de date în fișierele `database.yml`. [Specificațiile formatului fișierului YML/YAML](https://yaml.org/spec/1.2.2/) explică structurile complete ale limbajului și informațiile legate de sintaxă.
