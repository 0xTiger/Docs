## Ce este un fișier SCM?

Un fișier SCM este un fișier de cod sursă creat cu limbajul de programare Scheme. Conține cod scris în sintaxa schemei care este apoi executată de un interpret Scheme. Limbajul Scheme a fost dezvoltat de Massachusetts Institute of Technology (MIT) în 1970 și acceptă programarea funcțională și tehnicile asociate, cum ar fi algoritmii recursivi. Fișierele SCM pot fi deschise și editate în orice editor de text, cum ar fi Microsoft Notepad, Notepad++ și Apple TextEdit.

## Format de fișier SCM

Fișierele SCM sunt create și salvate în format text simplu folosind orice editor de text. Scheme face parte din familia Lisp de limbaje de programare și este standardizată în standardul oficial IEEE. Unele caracteristici distinctive ale limbajului de programare Schema sunt următoarele.

* `Minimalism` - Schema este un limbaj simplu care este mult mai ușor decât multe alte limbaje cu putere comparabilă.
* `Lexical Scope` - Schema are un scop lexical care face posibile legături de variabile într-o unitate de program și poate fi analizată prin citirea textului programului.
