## Ce este un fișier ALB?

Un fișier ALB este un fișier de bibliotecă utilizat de aplicația **Alpha Five** IDE care este utilizată pentru dezvoltarea aplicațiilor Web. Scopul său principal este de a stoca informații pentru Alpha Five [fișier ADB](/ro/database/adb/). Acestea includ informații din dicționar de date, cum ar fi setări, indicatori de script, meniuri, funcții definite de utilizator, hărți de biți, bare de instrumente și definiții de import.

Fișierele ALB pot fi deschise cu **Alpha Software Alpha Anywhere**.

## Format de fișier ALB - Mai multe informații

Fișierele ALB sunt salvate în format de fișier proprietar Alpha File și structura lor internă a formatului de fișier nu este disponibilă public. Acestea sunt salvate în plus față de baza de date generală a bibliotecii pentru dezvoltatori Alpha Five. Fiind biblioteci, fișierele ALB pot fi reutilizate în alte aplicații Alpha Five.
