## Ce este un fișier F95?

Un fișier cu extensia .f95” indică de obicei că este un fișier cu cod sursă Fortran 95. Fortran este un limbaj de programare de uz general, imperativ, care este deosebit de potrivit pentru calculul numeric și științific. Fortran 95 este o versiune îmbunătățită a Fortran 90, care introduce câteva funcții și îmbunătățiri noi.

Fișierele de cod sursă Fortran conțin de obicei instrucțiuni de program scrise în limbajul de programare Fortran; dacă aveți un fișier cu extensia .f95”, probabil că este menit să fie compilat folosind un compilator Fortran; compilarea este procesul de traducere a codului sursă care poate fi citit de om în cod binar care poate fi citit de mașină, care poate fi executat de un computer.

## Cum se deschide fișierul F95?

Dacă doriți să lucrați cu sau să rulați un program în fișierul .f95”, veți avea nevoie de un compilator Fortran; compilatoarele populare Fortran includ GNU Fortran (gfortran), Intel Fortran Compiler (ifort) și altele. Puteți compila și rula programe Fortran folosind comenzile adecvate furnizate de compilator.

De exemplu, dacă aveți gfortran instalat, puteți utiliza următoarele comenzi în terminal pentru a compila și rula un program Fortran:

```
# Compile the Fortran program
gfortran -o my_program my_program.f95

# Run the compiled program
./my_program
```

Dacă doriți doar să vizualizați conținutul fișierului F95, acesta este un fișier text simplu, așa că îl puteți deschide cu orice editor de text.
