## Ce este un fișier PHP?
Un fișier cu extensia .php se referă la limbajul de programare open source, folosit pentru a scrie scripturi pe partea de server, care urmează să fie executate pe un server web. Este cel mai răspândit limbaj de scriptare web care este utilizat în mod obișnuit pentru dezvoltarea de aplicații web la scară largă. Cel mai mare sistem de blogging de pe web, adică WordPress și cea mai mare rețea socială Facebook sunt bazate pe limbajul PHP. Este compatibil cu diverse platforme precum Windows, Linux, Unix, Mac OS X, etc. De asemenea, rulează cu succes pe aproape toate serverele utilizate astăzi, adică Apache, IIS, etc. PHP acceptă o gamă largă de baze de date, inclusiv MySQL.

## Format fișier PHP ##

Un singur PHP poate conține text, CSS, [HTML](/ro/web/html/), JavaScript și cod PHP. Codul PHP este executat pe server, iar rezultatul este returnat browserului ca HTML simplu. Fișierele PHP pot fi deschise cu orice editor de text și editate în loc, deși aplicații precum Adobe Dreamweaver, instrumentele de dezvoltare PHP Eclipse oferă o modalitate convenabilă de a scrie și modifica codul PHP.

### Sintaxă ###

Un script PHP este executat pe server, iar rezultatul HTML simplu este trimis înapoi în browser. Un script PHP începe cu<?php and ends with ?>

### Exemplu PHP ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
