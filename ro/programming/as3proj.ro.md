## Ce este un fișier AS3PROJ?

Un fișier AS3PROJ este un fișier de proiect creat de FlashDevelop, un IDE de programare open source gratuit pentru dezvoltarea de aplicații Flash și Web. Salvează toate informațiile și setările legate de proiect aplicate de utilizator folosind FlashDevelop IDE. Informațiile stocate într-un fișier AS3PROJ includ setări de compilare, referințe la fișierele sursă ale proiectului și opțiuni pentru generarea [fișierului .swf](/ro/page-description-language/swf/). Când este creat un nou proiect, acesta este salvat ca fișier AS3PROJ în aceeași locație specificată de utilizator pentru salvarea datelor legate de proiect.

## Format de fișier AS3PROJ

Un fișier AS3PROJ este salvat în format de fișier [XML](/ro/web/xml/), care este un format de fișier universal pentru schimbul de informații. Poate fi deschis în orice editor de text, cum ar fi Notepad, Notepad++ și Atom pentru a examina conținutul.
