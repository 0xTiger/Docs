## Ce este un fișier GMD?

Un fișier GMD este un fișier sursă de dezvoltare a jocului generat cu instrumentul de dezvoltare software Game Maker care este utilizat pentru a crea jocuri 2D. Fișierul GMD este stocat ca fișier text și este fișier de proiect pentru jocul care urmează să fie dezvoltat. Game Maker este disponibil pentru Windows, Mac, Linux, Android, iOS, HTML5, Xbox, PlayStation și Nintendo Switch. Instrumentul a fost utilizat pe scară largă de dezvoltatorii indie, studiourile profesionale și educatorii din întreaga lume pentru a crea jocuri.

## Format de fișier GMD

Fișierele GMD sunt salvate în format de fișier binar, iar detaliile interne ale formatului de fișier nu sunt disponibile pentru referință de către dezvoltatori.
