## Ce este un fișier GED?

Un fișier **GEDCOM (Genealogical Data Communication)**, de obicei cu extensia .ged”, este un format de fișier standard utilizat pentru schimbul de date genealogice între diferite programe software de genealogie; Fișierele GEDCOM sunt fișiere text simplu care conțin informații despre indivizi, familii, evenimente și alte date genealogice; scopul GEDCOM este de a oferi un format comun pentru schimbul de informații genealogice.

Fișierele GEDCOM au de obicei extensia de fișier .ged”; această extensie este folosită pentru a indica faptul că fișierul urmează **standardul GEDCOM** pentru schimbul de date genealogice; deci, atunci când salvați sau exportați datele dvs. genealogice din programul software de genealogie, fișierul rezultat primește adesea un nume precum `yourfilename.ged`.

De exemplu:

- arborele dvs. de familie
- john_doe_family.ged
- ancestry_data.ged

## Puncte cheie despre fișierele GEDCOM

Iată câteva puncte cheie despre fișierele GEDCOM:

1.  **Format text simplu:** fișierele GEDCOM sunt fișiere text simplu; ceea ce înseamnă că pot fi deschise și vizualizate cu orice editor de text; acest format facilitează partajarea ușoară și compatibilitatea între diferite aplicații software de genealogie.
    
2.  **Structură:** Fișierele GEDCOM au structură ierarhică; datele sunt organizate în înregistrări; fiecare înregistrare are o etichetă specifică care indică tipul de informații pe care le conține, de exemplu, numele unei persoane poate fi reprezentat de eticheta NUME”.
    
3.  **Individuals and Families:** GEDCOM files typically include information about individuals such as names, dates of birth and places of birth; they also include details about family relationships, marriages and other events.
    
4.  **Interoperabilitate software:** Deoarece GEDCOM este format standard; permite utilizatorilor să-și mute datele genealogice între diferite programe software de genealogie; acest lucru este util în special pentru utilizatorii care pot trece la o aplicație software diferită sau pot împărtăși cercetările lor cu alții.
    
5.  **Versiune:** GEDCOM a trecut prin diferite versiuni, fiecare versiune introducând îmbunătățiri și funcții noi; cea mai des folosită versiune a fost GEDCOM 5.5. 

## Cum se deschide fișierul GED?

Pentru a deschide un fișier GED, în general aveți nevoie de un software de genealogie care acceptă formatul GEDCOM. Unele opțiuni de software de genealogie populare includ:

- **Family Tree Maker**
- **Ancestry.com**
- **Mostenirea mea**
- **Gramps** (gratuit și open-source)
- **Arborele genealogic moștenit**

După ce ați ales software-ul de genealogie, căutați o opțiune precum Import”, Deschide GEDCOM” sau Import din fișier” în meniu sau bara de instrumente.

Deoarece fișierele GED sunt salvate într-un format text, puteți deschide fișierele și cu un editor de text simplu, de ex

- **Notepad**
- **Notepad++**
- **Apple TextEdit**
- Orice editor de text
