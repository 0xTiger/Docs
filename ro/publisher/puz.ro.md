## Ce este un fișier PUZ?

Un fișier cu extensia .puz este o arhivă comprimată care conține un document standard Microsoft Publisher [.pub](/ro/publisher/pub/). Ca rezultat al compresiei, este mai mic ca dimensiune și ușor de transferat prin internet sau alte dispozitive. Fișierele PUZ pot fi despachetate/deschise folosind utilitarul Microsoft Unpack.

## Format fișier PUZ

Un fișier PUZ este o arhivă comprimată care conține fișierul PUB standard împreună cu alte fișiere. Conține toate fișierele necesare pentru a despacheta, afișa și tipări publicația.

Un fișier PUZ conține următoarele.

* `Unpack.exe` - un utilitar software pentru despachetarea fișierelor comprimate
* `ReadMe.txt` - un fișier de descriere
* `Fișier .puz comprimat` - Conține publicația și poate include fonturi încorporate și elemente grafice legate.

### Despachetarea fișierului PUZ

Fișierele Publisher împachetate pot fi dezambalate folosind utilitarul Unpack.exe. Extrage fișierul PUZ împachetat în directorul ales de utilizator.
