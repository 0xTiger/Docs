## Ce este un fișier PUB?

Un fișier PUB este un format de fișier de document Microsoft Publisher. Este folosit pentru a crea mai multe tipuri de documente de aspect, cum ar fi buletine informative, fluturași, broșuri, cărți poștale etc. Fișierele PUB pot conține text, imagini raster și vectoriale. Documentele create cu fișiere PUB sunt utilizate mai ales în site-uri web și materiale de marketing, cum ar fi e-mailurile. Fișierele PUB pot fi deschise cu aplicația Microsoft Publisher Desktop, [Microsoft Publisher 365](https://www.microsoft.com/en/microsoft-365), [LibreOffice Draw](https://www.libreoffice.org/discover) /draw/) și [Adobe InDesign](https://www.adobe.com/products/indesign.html).

## Format fișier PUB - Mai multe informații

Fișierele PUB sunt stocate ca fișiere binare ca pe disc. Conținutul fișierelor PUB poate fi, de asemenea, convertit în format de fișier [DOCX](/ro/word-processing/docx/) din aplicația Microsoft Publisher, folosind opțiunea Salvare ca.

## Cum se utilizează Microsoft Publisher?

Microsoft a stabilit mai multe linii directoare pentru a ajuta utilizatorii să înceapă cu Microsoft Publisher. Acest ghid [Activități de bază în Publisher](https://support.microsoft.com/en-us/office/basic-tasks-in-publisher-0e5ed249-1927-433f-a35c-63beb8216fcf) vă permite să începeți arătând cum pentru a crea o publicație, salvarea unei publicații, adăugarea de text și blocuri de construcție și tipărirea publicației.
