## Ce este un fișier OXPS? ##

Formatul de fișier OXPS este cunoscut ca Open XML Paper Specification. Este un limbaj de descriere a paginii și un format de document. Microsoft este dezvoltatorul acestui format. Formatul de fișier OXPS este foarte familiar acestor fișiere PDF. Configurarea fișierului OXPS este aceeași cu un format XPS. Singura diferență este descrierea OXPS ca format deschis. Extensia de fișier .oxps este cea mai recentă versiune a formatului de fișier XPS creat de Microsoft și Ecma International. Dacă vorbim despre specificație, fișierele OXPS și XPS sunt folosite pentru a publica, partaja și aloca documente cu aspect fix. Ca înlocuitor al fișierelor PDF, formatul de fișier OXPS poate stoca text și date multimedia, cu caracteristici de aranjare și caracteristici de aspect de imprimare.

## Scurt istoric ##

Echipa Microsoft a ales Global Graphics pentru a oferi consultanță pentru XPS în 2003, iar echipa de dezvoltare a lucrat îndeaproape cu echipele Windows pentru a dezvolta specificațiile și arhitectura de referință pentru noul format. Recunoscând specificațiile XPS, în iunie 2007, Ecma International a înființat Comitetul tehnic 46 (TC46) pentru a dezvolta un standard bazat pe specificația OpenXPS. Specificația de hârtie Open XML aprobată de Ecma International este un standard Ecma (ECMA-388) la cea de-a 97-a Adunare Generală care a avut loc la Budapesta pe 16 iunie 2009.
