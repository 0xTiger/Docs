## Ce este un fișier INX?

Un fișier INX este un fișier plugin utilizat de editorul de grafică vectorială open source, **Inkscape**. Conține informații despre program, inclusiv numele și locația funcționalității din meniu. Fișierul INX specifică, de asemenea, un script pentru extensie care include informații, cum ar fi tipul de script, interpretul, locația și argumentele variabile.

## Format de fișier INX

Fișierele INX îmbunătățesc funcționalitatea Inkscape prin încorporarea unor funcții precum importatorii de fișiere și filtrele grafice. Stocate într-un format XML, aceste fișiere se găsesc de obicei în directorul \​extensii\​ dintr-o instalare Inkscape.

## Cum se deschide fișierele INX?

Puteți deschide fișiere INX folosind programul **Inkscape**.
