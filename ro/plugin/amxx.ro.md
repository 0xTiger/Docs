## Ce este un fișier AMXX?

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. Este o platformă de scripting care permite administratorilor de server să personalizeze și să extindă funcționalitatea serverelor lor. AMXX este utilizat împreună cu sistemul de gestionare a pluginurilor MetaMod, care permite mai multor plugin-uri să ruleze simultan pe un server fără conflicte. Caracteristicile oferite de pluginul AMXX includ comenzi de administrare, modificări ale jocului și gestionarea serverului.
