## Ce este un fișier DST?

Un fișier DST este un fișier AutoCAD care conține asocieri și informații pentru definirea seturilor de foi. Acestea sunt stocate în folderul de stocare implicit al setului de foi, AutoCAD Sheet Sets. Fișierele DST nu conțin desenele reale, dar se referă la acestea din fișierele [DWG](/ro/cad/dwg/) și [DWT](/ro/cad/dwt/) selectate asociate cu aceste seturi de foi. În mediul de rețea, toți membrii echipei ar trebui să aibă acces la rețea la aceste fișiere asociate. Fișierele DST sunt salvate pe disc în format de fișier [XML](/ro/web/xml/).

## Format de fișier DST - Mai multe informații

Fișierele DST sunt create cu instrumentul AutoDesk Sheet Set Manager (SSM). Când un fișier este accesat de cineva din echipă, fișierul DST este deschis pentru scurt timp și apoi stocat înapoi cu informațiile actualizate. Accesul simultan la același fișier DST este gestionat de instrumentul SSM care arată o pictogramă de lacăt atunci când fișierul este în acces.

În orice moment, starea foii poate fi în oricare dintre următoarele stări.

* Foaia este disponibilă pentru editare.
* Foaia este blocată.
* Foaia lipsește sau s-a găsit într-o locație neașteptată a folderului.
