## Ce este un fișier DC3?

Un fișier cu extensia .DC3 este un desen 3D creat cu DesignCAD 3D MAX, care este un software CAD. Este folosit pentru a crea modele de înaltă calitate, randări simple și animații și acceptă importul/exportul [STL](/ro/cad/stl/). Un fișier DC3 este utilizat ca alternativă la formatul de fișier .DCD. Fișierele DC3 pot fi deschise cu IMSI DesignCAD 3D Max, care oferă o gamă completă de primitive puternice de modelare 3D, inclusiv cutie, sferă, emisferă, piramidă, cilindru, con și tor.

## Format de fișier DC3

Un fișier DC3 conține informații despre desen 3D în format text ASCII care poate fi deschis în orice editor de text. Cu toate acestea, sensul conținutului său poate fi înțeles doar de aplicațiile care înțeleg formatul de fișier de bază.
