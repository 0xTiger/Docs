## Ce este un fișier PSM?

Un fișier cu extensia .psm este un fișier CAD care este creat și deschis cu software-ul de modelare CAD Solid Edge pentru a proiecta piese și ansambluri 3D. Conține modelul 3D al designului pentru fabricarea pieselor din tablă. Setul de software Solid Edge oferă toate opțiunile necesare pentru a crea un model 3D care poate fi integrat și cu alte modele mecanice. Imbricarea sa 2D acceptă atât formatele de fișiere PSM, cât și PAR, pe lângă alte formate de date neutre, inclusiv DXF și DWG, prin extragerea datelor tip model plat.

### Format fișier PSM

PSM este un format de fișier apropiat și detaliile acestuia nu sunt disponibile deocamdată. Acestea au tip MIME application/octet-stream. Fișierele model metalice pot fi importate în Solid Edge și convertite în părți utilizabile pentru a le face editabile. Modelarea sincronă oferă o utilizare mai mare pentru lucrul cu fișiere PSM.
