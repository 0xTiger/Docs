## Ce este un fișier ASC?

Fișierele ASC sunt fișiere text Autodesk care stochează informații despre obiectele din desen, inclusiv locația, dimensiunea și culoarea acestora. Acest lucru le face o alegere populară pentru schimbul de desene între diferite programe [CAD](/ro/cad/). Fișierele ASC pot fi deschise cu produsele software Autodesk. Autodesk oferă opțiunea de a importa fișiere ASC din cadrul aplicației. Fișierele ASC pot fi deschise în orice editor de text, cum ar fi Microsoft Notepad, Notepad++ și Apple TextEdit.

## Format de fișier ASC

Fișierele ASC sunt stocate pe disc în format simplu de fișier text ASCII. Acestea sunt citite de om.

### Exemplu de fișier ASC

```
ncols 200
nrows 200
xllcorner 119000.00
yllcorner 158000.00
cellsize 5.00
nodata_value -999999
```
