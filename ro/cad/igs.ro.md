## Ce este un fișier IGS?

Un fișier cu extensia .igs (Initial Graphics Exchange) este un format de fișier de schimb de design 2D-3D care este independent de specificațiile de format de fișier sursă sau destinație utilizate de aplicațiile CAD. Este folosit pentru a face schimb de informații de proiectare despre diagrame de circuit, cadru electric, suprafață cu formă liberă între două sisteme independente. Este similar cu [IGES](/ro/cad/iges/) și este cerința principală a producătorilor pentru stabilirea prețurilor și proiectarea matrițelor pentru produsul dvs. IGS a fost înlocuit cel mai recent cu noul format de fișier STEP(.STP). Fișierele IGS pot fi deschise de aplicații precum Autodesk, FreeCAD, CADEX CAD Exchanger și alte aplicații similare.

## Format de fișier IGS

Fișierele IGS sunt în format de fișier text ASCII și pot fi deschise cu orice editor de text. Fiecare înregistrare dintr-un fișier IGS are o lungime de 80 de caractere. Aplicațiile CAD salvează și exportă date vectoriale în formatul de fișier IGS pentru transferul fișierelor model 3D între diferite pachete software. Deși fișierele IGS pot fi deschise în orice editor de text, modelul sau designul pot fi vizualizate numai de aplicațiile de vizualizare care oferă suport pentru formatul de fișier IGS.
