## Ce este un fișier RML?

Un fișier RML este un fișier de adnotare creat și utilizat de aplicația Volo View de către [Autodesk](https://autodesk.com). Conține informații despre adnotări și semne de text care trebuie suprapuse pe desen pentru a lua notițe. Aceste adnotări sunt afișate în text roșu peste desen și de aceea numele fișier Redline Markup Language. Volo View a fost întrerupt și nu mai este acceptat de Autodesk. Conținutul fișierelor RML este încorporat în desenele CAD sub formă de note.

## Format de fișier RML

Fișierele RML sunt create și salvate pe disc în format de fișier XML, care este un format de fișier universal pentru schimbul de date. Este într-un format care poate fi citit de om și poate fi deschis cu orice editor de text, cum ar fi Microsoft Notepad, Notepad++ și Apple TextEdit.
