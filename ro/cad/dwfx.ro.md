## Ce este un fișier DWFX?

Un DWFx (Design Web Format XPS) este o reprezentare formatată a desenului 2D/3D ca document XPS. Conține grafică și text ca parte a datelor de proiectare. Este cea mai nouă versiune a formatului de fișier DWF și poate fi vizualizată și tipărită cu Microsoft XPS Viewer. Natura XPS a acestor fișiere vă permite să partajați datele de proiectare cu examinatori fără a le solicita să instaleze Autodesk Design Review.
Similar cu DWF, DWFx este dezvoltat de Autodesk în format comprimat pentru a face transmisia pe internet adecvată. Un singur fișier DWFx poate conține unul sau mai multe desene și seturi de foi.
