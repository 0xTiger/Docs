## 什么是 .STR 文件？

STR 文件是 YouTube 为用户的流媒体活动托管的直播创建的录制文件。直播结束后，Youtube 后台会自动保存该文件。直播视频可以在多个频道播出，以便观众可以通过评论与发布者互动。这些 SRT 文件可使用 YouTube livestream API 访问，该 API 提供对 YouTube 服务器的访问。

## STR 文件格式

YouTube 以二进制文件格式将 STR 文件保存到服务器。但 STR 文件的内部文件结构详细信息并未公开。
