## 什么是 .TR3 文件？ ##

TR3 文件是 TomeRaider 3 电子书，提供对快速搜索和压缩的支持。这可以通过 TomeRaider 相关程序正常运行。雅大字节是该软件的开发者，一家位于英国的英国软件和网络开发公司。 TomeRaider 电子书阅读器应用程序可用于正常运行和管理这些 TR3 文件的内容。这种相关的电子书阅读器可以安装在运行基于 Microsoft Windows 的系统和许多可传输小工具的计算机中。 TR3 文件属于 Windows 10、Windows 7、Windows 8 / 8.1、Windows Vista、Windows XP 等操作系统中使用的电子书文件组。

TR3 特定的**HTML 标签**如下：

|标签|说明|
---|---|
|\<new> |要从文本文件创建 TomeRaider 文件，只需要 New 标签。<new>标签定义页面的开始。|
|\<CATDEF> ... \</CATDEF> |定义类别名称|
|\<META> ... \</META> |是一个标签，用于定义文件格式中使用的所有元数据。这个标签包括许多参数。|
|\<EXPMSG> |此标记控制文件过期时出现的消息。每当用户在文件过期后尝试查看页面时，就会出现过期消息而不是实际的页面文本。|
|\<DIEMSG> |此标签类似于 EXPMSG。它用于在文件终止时显示一条消息。|
|\<ENGMSG> |此标记用于创建为加密页面显示的消息。如果用户在解锁之前尝试打开加密页面，则会显示该消息而不是页面文本。
|\<expmsg> ,\<diemsg>和 \<encmsg> |在页面文本中被忽略。它们必须放在第一个之前的文件头部分中<new>标记。|
|\<SELECTION> . \</ SELECTION> |用于支持查询。这个标签必须在第一个之前<new>标签。它为用户的查询编译选择数据。|
|\<catset> . \</catset> |用于为每个主题分配类别名称。|


## 打开 TR3 文件的问题##

以下是一些可能出现并导致文件格式错误的常见问题的列表：

* 损坏的文件
* 由于病毒而感染的文件
* 存档访问中到 TR3 文件的链接不正确
* 在系统中没有打开文件的访问权限
* 系统中的过时驱动器
* 从 Windows 存档中删除 TR3 的报告
* 联系开发人员可能是获得更好输出的更好选择
