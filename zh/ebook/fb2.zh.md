## 什么是一 .FB2 文件？

扩展名为 .fb2 的文件是 FictionBook 2.0 电子书文件，其中包含电子书的结构。它基于 XML 格式并包含用于描述本书每个元素的特殊标签。它主要是为虚构作品和文学而开发的，但不仅限于这些。该格式本身包含所有元数据和内容，并允许灵活地进行许多操作，例如自动处理、索引和转换为其他格式。简而言之，它侧重于描述文件的结构，而不是指定其外观。一些应用程序和 API 可用于将 FB2 转换为 Windows、MacOS 和 Linux 上的几种其他格式。

## FB2 文件格式

FictionBook 2.0 文件格式由俄罗斯 FictionBook 的 Dmitry Gribov 开发。与包含多个 XML 文件的其他流行电子书文件格式（例如 [EPUB](/zh/ebook/epub/)）不同，它基于包含有关内容的完整信息的单个 XML 文件。如果文件中有图像，例如 [PNG](/zh/image/png/) 或 [JPG](/zh/image/jpeg/)，这些图像将转换为 Base64 并嵌入到文档中。 FB2 的 [文件格式规范](http://gribuser.ru/xml/fictionbook/index.html.en) 包含格式 2.0 版本的信息。
