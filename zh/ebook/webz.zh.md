## 什么是一 .WEBZ 文件？ ##

WEBZ 文件扩展名是一种非常流行的文件格式，完全称为 WEBZ 压缩电子书文件。 EBZ后缀一般用于WEBZ Compressed eBook Format文件，基本用作网页和图片的ZIP文件。 Web Books Publishing 是作者和标定WEBZ 格式的质量。这种文件格式与可以连接到 Web 系统平台的软件相匹配。 WEBZ 文件转到文本文件类型，就像其他文件扩展名一样。 Web Books Viewer 是处理此类文件格式最常用的程序。此文件扩展名很灵活，可以从“.webz"更改为“.zip"。
