## 什么是一 .koob 文件？ ##

.koob 文件扩展名最初由 Vijay Sridhara 开发，与电子书文件相关。它也被称为文本文件类别下的 Koob 电子书文件。关于 KOOB 文件的一个规范是它包含书籍部分、书籍样式、方向以及出版，以及有关电子书的其他信息。如果系统没有 Koober，那么 KoobReader 会帮助读取这个文件。 KOOB 文件帮助用户获取有关电子书的所有信息，这就是它的全名是 Koob eBook Format 的原因。

## 打开 KOOB 文件时出现问题？ ##

* 当有被感染或损坏的文件时，很难打开它。在这种情况下，请通过系统上安装的防病毒程序扫描 KOOB 文件
* 无权处理所需的文件扩展名可能会导致错误
* 系统上文件的快捷方式，而不是导致错误的正版文件
* 驱动器空间不足可能导致无法打开文件扩展名
 *	The process copies the file from an external source may be not completed. In this case, it's impossible to open an incomplete file
