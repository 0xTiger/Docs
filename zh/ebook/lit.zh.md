## 什么是一 .lit 文件？

LIT 文件属于古代用于阅读电子书的 **Microsoft Reader**。 Microsoft Reader 显示的电子书具有扩展名 **.lit**，缩写为 **Literature**。 LIT 文件格式由 Microsoft ITOL/ITLS HTML Help 2.0 文件组成，它提供了各种令人惊叹的功能，如 [LZX](/zh/compression/lzx/) 压缩、标记材料的二进制表示以及与相关联的任意可选辅助数据每个流。书籍文本是任意数量的 OEBPS 1.0 标记流和 OEBPS 1.0 CSS 的子集。图像是 [GIF](/zh/image/gif/)、[JPEG](/zh/image/jpeg/) 和 [PNG](/zh/image/png/)。

## Microsoft Reader 显示 LIT 文件的时间

**Microsoft Reader** 是用于阅读 Microsoft 已停产的电子书的应用程序。 Microsoft Reader 的第一个版本于 2000 年 8 月发布，它使用自己的 LIT 格式。它可用于 PDA 和 Windows 计算机。该应用程序的名称后来也用于读取 [PDF](/zh/pdf/) 和 [XPS](/zh/page-description-language/xps/) 文件的应用程序（最初随 Windows 8 发布）。 Microsoft Reader 应用程序已于 2018 年停止使用。
