## 什么是一 .aep 文件？ ##

扩展名为 .aep 的文件（代表 **Activ E-Book Project**）被归类为电子书 (eBook)。 AEP 文件使用与 Internet Explorer 相同的引擎创建，以确保 100% 的兼容性。它包含的页面就像一个人可以阅读实体书一样。 AEP 文件的页面通常包括文本、图像、动画和超链接。此外，它还支持[HTML](/zh/web/html/)、[DHTML](/zh/web/dhtml/)、[JPEG](/zh/image/jpeg/)和[PNG](/zh/image/png/)图形、动画、[JavaScript](/zh/web/js/)、VBScript 和许多 Internet Explorer 插件。 AEP 文件可以保存为 Windows 可执行文件 (EXE)，以便在 Windows95 或更高版本的 Microsoft Windows 上打开。
 


## AEP 文件格式##

AEP 格式文件是使用 AEC（Active E-Book Compiler）软件创建的。该软件使用与 IE 类似的引擎来创建电子书。用户可以创建无限的电子书副本，而无需为使用该程序支付任何费用。分发过程也很简单，因为每本电子书都由一个独立的 Windows 程序组成，可以在电子邮件、Internet 下载、软盘、CD-ROM、ZIP 磁盘等任何媒体上分发。

## 打开文件时可能出现的问题

如果您无法打开和运行 AEP 文件；这并不意味着您的设备上没有安装合适的软件。可能还有其他一些问题会阻止文件正常工作。可能的问题可能是以下之一：

- AEP 文件损坏
- 注册表项中指向 AEP 文件的错误链接
- 从 Windows 注册表中删除了 AEP 的描述
- 支持 AEP 格式的应用程序安装损坏
- 带有不良恶意软件的受感染 AEP 文件
- 电脑没有足够的硬件资源来操作AEP文件
- 计算机用于打开 AEP 文件的驱动程序已过时
