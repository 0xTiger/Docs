## 什么是一 .ybk 文件？ ##

YanCEyWare 电子书文件格式和 Microsoft Encarta 都使用 YBK 文件扩展名。 YanCEyWare 电子书由 YanCEyWare 创立。所有软件应用程序都可用于运行 Windows 的设备，支持 YBK 文件。 Encarta Encyclopedia 可以通过 .ybk 文件进行更新，而无需更新整个程序。 YanCEyDesktop 是一个著名且流行的程序，可以轻松管理和处理 YBK 文件。 YanCEyWare 阅读器广泛使用这种 YBK 文件格式，作为 Windows Mobile 或 Pocket PC 的电子书阅读器应用程序。该文件是使用大型超链接资源开发的。

## 打开 YBK 文件的问题##

有时当 YBK 文件难以打开和运行时，这里列出了一些可能的问题：

* 打开损坏的 YBK 文件
* 在注册表访问中指向 YBK 文件的不适当和不正确的链接
* 无意中从 Windows 存档中删除了 YBK 的描述
* 部分安装提供 YBK 格式的应用程序
*	Infected YBK file with an undesirable malware
