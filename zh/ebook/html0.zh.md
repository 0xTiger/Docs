## 什么是 HTML0 文件？ ##

HTML0 是 **Book Designer** 使用的本机电子书格式，具有描述每个页面的内容和设计的标准 HTML 和 CSS 代码。 **Book Designer** 是最常见的程序，用于处理 HTML0 文件并将文本文档更改为多种电子书格式。 HTML0 文件格式支持可安装在 Windows 系统上的软件，因为它属于文本文件类别。 HTML0 文件包含普通的 HTML，可以使用文本编辑器查看和修改。与通常的 HTML 文件相比，HTML0 文件不能链接到离散的 CSS 文件。
