## 什么是 .xhtm 文件？

扩展名为 .xhtm 的文件是用 XHTML 编写的网页文件。它使用 XML 1.0 作为其标准实现，并被视为 HTML 4 的重新表述。与 XHTML 一样，XHTM 文件包含遵循更严格的 XML 准则的标准 HTML 标记。 XHTM 文件用于发布网站内容，通常使用 [XHTML](/zh/web/xhtml/) 扩展名。

您可以使用 Google Chrome、Apple Safari 和 Mozilla Firefox 等 Web 浏览器打开 XHTM 文件。

## XHTM示例

以下 XHTM 文档示例定义了 XML 声明。

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
