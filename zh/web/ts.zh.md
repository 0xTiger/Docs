## 什么是一 .ts 文件？

TS 文件是使用 lupdate 软件实用程序生成的翻译文件，该实用程序与 Digia 的 Qt SDK 捆绑在一起。 SDK 适用于 Windows、Mac 和 Linux 操作系统。 TS 文件包含一种语言的短语及其翻译成另一种语言。存储在 TS 文件中的翻译是特定于特定应用程序的。 TS 文件可以使用 Digia Qt SDK 打开。

## TS 文件格式

Digia Qt TS 文件以 XML 文件格式存储到光盘中，并且可以在任何文本编辑器（例如 Notepad 和 Notepad++）中打开。 Digia Qt SDK 支持的其他一些文件格式包括：

* .pri
* .qm
* .qml
* .qph
* .qrc
* .qs
* .qss
* .ts
* .ui 用户界面文件
