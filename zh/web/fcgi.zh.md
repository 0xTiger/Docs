## 什么是一 .fcgi 文件？

FCGI 文件是包含为 FastCGI (FCGI) 编写的源代码的源代码文件。它可以用几种不同的编程语言编写，并包含 CGI 标准的扩展，用于更健壮地处理 Web 服务器请求。 FCGI 最常用的支持语言包括 **Perl**，它提供对 Perl 代码和 FastCGI API 的强大外部接口的支持。

## FCGI 文件格式

是什么让 FCGI 文件如此重要？ FCGI 接口支持分布式计算来处理 Web 请求并将它们分布在多个服务器上。这使得 FCGI 文件主要用于跨多个服务器的负载平衡。
