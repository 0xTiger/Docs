## 什么是 .hxs 文件？

HXS 文件是为 Microsoft Help 2.x 文件编译和使用的帮助文件。它由一组主题页面编译生成，并以类似于早期的 [CHM](/zh/web/chm/) 帮助文件格式的 [HTML](/zh/web/html/) 的子集编写。 HXS 文件以压缩格式存储，由一个 .HxC 主项目文件、一个 .HxF 包含文件、一个 .HxT 目录、一个 .HxA 属性定义文件和几个 .HxK 索引创建。 HXS 文件的元数据是从 [XML](/zh/web/xml/) 创建的。

HXS 文件可以用网络浏览器打开。

## HXS 文件格式

HXS 文件是根据 HTML 内容和 XML 元数据创建并保存为压缩文件的。
