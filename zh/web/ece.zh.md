## 什么是一 .ece 文件？

ECE 文件是由内容管理平台**Escenic Content Engine (ECE)** 动态生成的网页。它包含由 Web 服务器处理并作为 [HTML](/zh/web/html/) 页面返回给用户浏览器的服务器端代码。与其他服务器端文件格式（如 ASP 和 PHP）相比，ECE 文件格式不是很流行。这些文件在内部呈现并提供给最终用户用于显示目的。

## ECE 文件格式

ECE 文件不保存到光盘，而是根据要求直接提供给用户。这些被转换为 HTML 并发送回用户的浏览器进行显示。 Escenic Content Engine 被积极用于发布新闻文章页面。 ECE 服务器基于 JEE 和 SQL 技术，为所有在线内容提供可靠且可扩展的存储库。

客户端浏览器可以通过 HTTP/REST API 访问 ECE 服务器上的内容。除了提供 [HTML](/zh/web/html/) 页面之外，ECE 还为管理和发布视频/音频内容提供了高效的工作流程。
