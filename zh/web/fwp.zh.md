## 什么是一 .fwp 文件？

带有 .fwp 文件的文件是使用 Microsoft Expression Web 创建的存档文件，其中包含设计的所有文件和文件夹。 Microsoft Expression Web 是 Microsoft 的通用网页设计软件产品，基于 [HTML](/zh/web/html/) 编辑器。它自 2012 年起已停产，可供 Microsoft 免费下载。 FWP 文件可以导出并保存为 Microsoft 的 .cab 格式。

## FWP 文件格式 - 更多信息

FWP 文件的确切文件格式规范不供开发人员参考。但是，这些是二进制文件，可以在很久以前停止使用的 Microsoft Expression Studio 中打开。
