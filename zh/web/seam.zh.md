## 什么是SEAM文件？

SEAM 文件是 JBOS Seam 框架使用的配置文件。它包含对处理 Web 应用程序请求的服务器端 servlet 的元数据/引用。它还包含有关 Seam 组件的元数据，例如实现组件的类和方法的名称。您可以将 SEAM 文件与 JavaServer Faces (JSF) 框架结合使用来构建动态和交互式 Web 应用程序。 SEAM 文件通常存储在 Seam 应用程序的 WEB-INF 目录中。

## SEAM 文件格式

SEAM 文件存储为文本文件，通常采用 [XML](/zh/web/xml/) 格式，并且包含 Seam 应用程序的配置信息。它是Seam 应用程序配置的关键部分。您可以使用任何文本或 XML 编辑器打开 SEAM 文件。
