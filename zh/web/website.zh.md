## 什么是网站文件？

WEBSITE 文件是使用 Microsoft Internet Explorer (IE) 创建的网站快捷方式文件。它包含一个 URL 和一个指向网站图标的图标 (ICO) 文件。该图标用于显示快捷方式。 WEBSITE 文件格式是随着 IE 版本 9 的发布而引入的。以前版本的 IE 用于将网页的快捷方式保存为 [.url](/zh/web/url/) 文件。

## 网站文件格式

.website 文件作为纯文本文件存储到光盘中，其内容可以在任何文本编辑器中查看。除了其他内容之外，它还包含作为快捷方式的目标 URL。如果您想在 IE 以外的浏览器中打开目标网站，您可以复制此 URL 并将其粘贴到其他浏览器（例如 Google Chrome、Safari 或 Firefox）中以查看该网站。

## 如何创建 .WEBSITE 文件？

可以在 IE 中通过单击地址栏中的 URL 并将其拖动到计算机上所需的位置（例如桌面或文档）来创建 WEBSITE 文件。这将创建该 Internet 位置的快捷方式，并且可以通过双击该快捷方式打开。如果要将快捷方式保存为“URL"文件，请在将目标站点的图标拖到桌面之前按住 SHIFT 键。
