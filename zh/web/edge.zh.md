## 什么是 .edge 文件？

扩展名为 .edge 的文件是 Adobe Edge 项目文件，该文件是在使用 Adobe Edge 创建和保存项目时创建的。 Adobe Edge 允许网页设计师为桌面和移动浏览器的网页、数字出版和富媒体广告创建交互式 [HTML](/zh/web/html/) 动画师。这是使用 HTML/HTML5、[JavaScript](/zh/web/js/) 和 [CSS3](/zh/web/css/) 实现的。 Edge 文件以纯文本文件的形式生成，可以使用多种不同的应用程序打开，例如 Microsoft Notepad、Apple TextEdit 和 Notepad++。

## 边缘文件格式

EDGE 文件以纯文本格式保存，并使用 jQuery 启用动画。 EDGE 动画的替代方案是不再支持的 Flash 文件。 EDGE 文件与 HTML、JavaScript 和 CSS 文件一起保存。打开 EDGE 文件后，即打开网页，即 HTML 文件。
