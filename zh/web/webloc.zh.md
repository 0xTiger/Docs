## 什么是 WEBLOC？

WEBLOC 文件是 macOS 上指向 Internet 上的 Web URL 的快捷方式文件。它是通过将 Web URL 从浏览器的地址栏拖到桌面或任何其他要保存它的文件夹来生成的。目标站点的 URL 存储在此快捷方式文件中，可以通过在 Microsoft Notepad、Notepad++ 或 Apple TextEdit 等文本编辑器中打开文件来查看。 WEBLOC 文件类似于 [URL](/zh/web/url/) 文件。

## WEBLOC 文件格式 - 更多信息

WEBLOC 文件以纯文本文件格式存储。这些可以通过从 Safari、Google Chrome 或 Firefox 等 Web 浏览器中拖动 URL 在 macOS 上生成。如果要同时打开多个 WEBLOC 文件，则需要更快速、更简单的解决方案，通过外部应用程序或 API 打开这些文件。这些 API 可以读取 WEBLOC 并将其转换为 PDF 和 JPEG。
