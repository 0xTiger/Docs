## 什么是一 .awm 文件？

AWM 文件是由 Web 菜单构建程序 **AllWebMenus** 创建的项目文件。它包含用于创建基于 [HTML](/zh/web/html/) 的下拉菜单的项目相关数据。 AWM 文件被编译成最终的 Web 包，该包集成到网站中以显示菜单。它包含所有项目相关数据，例如外观设置、菜单条目、样式（垂直或水平）、颜色和其他相关信息。

可以使用 **Likno AllWebMenus** 打开 AWM 文件。

## AWM 文件格式 - 更多信息

AWM 文件保存为编译后的二进制文件，只能用力能 AllWebMenus 软件打开。 AWM 文件的内部文件格式详细信息不公开。

AllWebMenus 让您无需使用任何编程语言即可直观地设计 [CSS](/zh/web/css/) 菜单或基于 [JavaScript](/zh/web/js/) 的菜单。菜单可以完全定制，也可以使用软件提供的预定义外观模板进行设计。
