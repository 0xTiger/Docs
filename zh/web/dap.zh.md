## 什么是一 .dap 文件？

带有 .dap 扩展名的文件是一个 Microsoft Access 数据页文件，它是从 Microsoft Access 中创建的，并且可以访问存储在数据库中的数据。存储在此类 Web 文件中的数据可通过网络（Intranet 和 Internet）访问，这与专为桌面应用程序开发的 Access 的 [MDB](/zh/database/mdb/) 文件不同。可以使用 Access 2010 和 Access Services（SharePoint 的可选组件）创建此类数据库。拥有 SharePoint 帐户的人可以在 Web 浏览器中使用 Web 地址来使用这些数据库。
