## 什么是 .dothtml 文件？

DOTHTML 文件是以 HTML 版本保存的 Microsoft Word 模板文档。它可以在网络浏览器中打开，内容显示为任何其他网页。 DOTHTML 文件是从包含用于生成这些文件的预格式化设置的 Microsoft Word .DOT 文件生成的。当 [DOT](/zh/word-processing/dot/) 文件转换为 DOCHTML 时，所有这些设置都将保留，转换后的 DOTHTML 文件以与原始 DOT 文件类似的格式显示。这样，可以在 Web 浏览器中查看已发布的模板。

## DOTHTML 文件格式 - 更多信息

DOTHTML 文件保存为 [HTML](/zh/web/html/) 文件并包含可使用任何 HTML 编辑器打开的 HTML 源代码。最初，Microsoft Word 提供其所有文档格式的 HTML 版本，例如 [DOCHTML](/zh/web/dochtml/)、[DOCMHTML](/zh/web/docmhtml/)，但后来这些基于 HTML 的版本已停产.结果，也不再支持 DOTHMLT。
