## 什么是一 .chat 文件？

CHAT 文件是 IRC（Internet 中继聊天）用于连接到 IRC 频道的配置文件。它包含连接参数，例如服务器、端口、IRC 通道的名称，以及用于打开 IRC 通道的可选密码。客户端使用这些设置连接到其他通道。 IRC 上的聊天和讨论是一对一的并且是私密的。

可以**打开** CHAT 文件的应用程序包括 mIRC 和 Visual IRC。

## 聊天文件格式

CHAT 文件以二进制文件的形式存储到光盘中。当用户想要连接到 IRC 中的频道时，他在 iRC 客户端中输入信息，他在客户端软件中输入信息。信息存储在 CAHT 文件中，用于横向检索。
