## 什么是压缩文件？

压缩文件是使用插件 **WordCompress** 生成的小尺寸 Wordpress 压缩网页。它包含一行代码，用于表示插件通过压缩 HTML、CSS 和 JavaScript 代码生成的网页。这些转换后的压缩文件尺寸较小，有助于更快地加载和通过互联网传输。

### 压缩文件格式 - 更多信息

压缩文件保存为文本文件，可以使用任何文本编辑器（例如 Microsoft Notepad、Notepad++ 和 Apple TextEdit）打开。除了文本编辑器之外，压缩文件还可以使用 Web 构建软件（例如适用于 Windows 和 Mac OS 的 Adobe Dreamweaver）打开。

## 关于 WordCompress 插件

WordCompress 插件最初用于将 WordPress 网页压缩为压缩文件格式。该插件不再受支持并已停止使用。通过删除页面源代码中的所有空格，它将网页的源代码写入单个可执行文件。压缩文件与网页的原始源代码一起存储，以保留页面原始源代码中包含的原始注释、缩进和其他空白。
