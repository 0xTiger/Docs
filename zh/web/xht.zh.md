## 什么是一 .xht 文件？

具有 .xht 扩展名的文件是与 [XHTML](/zh/web/xhtml/)（扩展超文本标记语言）文件类型相关联的 Web 文件，它本身是基于 [XML](/zh/web/xml/) 的标记文件。这两个文件都类似于 HTML，但具有更严格的类似 XML 的语法。 XHT 文件的设计方式更加结构化，涉及的脚本更少，并且除了使用现有的 XML 工具外，还具有通用性。

## XHT 文件格式

默认情况下，XHT 文件被创建并存储为纯文本文件，采用 UTF-8 编码。它包含可以使用任何支持 Unicode 编码的文本编辑器打开和编辑的 XHTML 文档的完整源代码。除了文本编辑器之外，大多数现代 Web 浏览器都可以理解 XHT 文件格式，并且可以使用它们打开。

## XHT 示例

下面的 XHT 文档示例定义了 XML 声明。

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
