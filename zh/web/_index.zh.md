构建网页时需要多种文件类型。 Web 文件格式定义了网页开发的标准，并与构建这些标准的平台相关。一个完整的网站可以由静态网页和动态网页组成。大多数现代网站都是基于服务器端技术构建的，例如在 Web 服务器上加载和运行的 Active Server Pages (ASP)。这些还包括级联样式表 (CSS) 和用于设置 UI 整体外观样式的脚本文件。

常见的 Web 文件扩展名及其相关文件格式包括 **[HTML](/zh/web/html/)**（超文本标记语言）、**[ASP](/zh/web/asp/)（Active Server Pages）**、和 **CSS**（层叠样式表）。
