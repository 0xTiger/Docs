## 什么是 .ccbjs 文件？

CCBJS 文件是由 3D 应用程序、游戏和网站编辑器 **CopperCube** 创建的 Javascript 文件。它包含作为从 .ccb 文件创建的 [.html 文件](/zh/web/html/) 的一部分生成的 JavaScript 代码。 CopperCube 提供了一个用户友好的脚本接口，无需编写任何代码即可开发游戏逻辑和交互。 CopperCube 支持的其他文件格式包括 .ccscene、.ccmodel、.ccmaterial、.cctex、.ccjs、.ccp 和 .ccpak。

## CCBJS 文件格式

CCBJS 文件格式的内部文件格式细节尚不清楚。然而，Javascript 文件以纯文本文件形式保存到光盘上，并且可以使用任何文本编辑器（例如 Microsoft Notepad、Notepad++ 和 Apple TextEdit）打开。
