## 什么是一 EPIBRW 文件？

EPIBRW 文件是托管在基于 Microsoft IIS 的 Web 服务器上的通用文件。此文件的目的不是特定于任何特定操作或任何类型的数据存储，而是通用的。它最常被 [.pdf](/zh/pdf/) 文件使用，但也可以被其他几种文件类型使用。

双击 EPIBRW 文件时，它可能会在显示内容的浏览器中打开。但是，如果链接文件以供下载，则文件下载过程将作为此事件的结果启动。
