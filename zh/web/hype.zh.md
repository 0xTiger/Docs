## 什么是 .HYPE 文件？

HYPE 文件是使用 Hype 软件保存的项目文件，用于使用 HTML5 创建 Web 内容。它包含网络元素，例如图形、文本、动画、场景信息和其他共同构成内容的元素。 HYPE 文件中的 Web 内容可以导出为可以在网页上发布的 Web 文件。

HYPE文件可以用[Tumult Hype](https://tumult.com/hype/)软件打开。

## 炒作文件格式

HYPE 文件以专有文件格式存储，其内部细节不为公众所知。 HYPE 文件中的内容只有在 Tumult Hype 软件中打开时才能查看。

### HYPE 文件的内容

当您使用 Hype 应用程序为不同目的创建 Web 内容时，会创建一个 HYPE 文件。它存储有关基于关键帧的动画的信息，用于创建电子贺卡、电子书、演示文稿、广告、信息图表和网页材料等内容。这些内容由 HTML5 提供支持，允许内容显示在从台式机到 iPhone 和 iPad 的设备上。 Tumult Hype 输出最先进的 HTML5、[CSS3](/zh/web/css/) 样式和 [JavaScript](/zh/web/js/)。
