## 什么是 .htc 文件？

HTC 文件是 Microsoft Internet Explorer 使用的 Web 文件格式文件。它包含 HTML 组件的定义，这些组件执行在 Web 浏览器中执行不同任务的小功能。 HTC 文件中定义的功能是可重用的，并且可以跨多个 [HTML](/zh/web/html/) 页面访问。

## HTC 文件格式 - 更多信息

HTC 文件以纯文本文件格式保存。这意味着您可以在任何文本编辑器中打开它们并对功能进行更改。这些文件通常在 Internet Explorer 中打开，但这样您将只能查看这些文件而不能编辑它们。

HTC 文件实际上是一个 HTML 文件，它是脚本和一组用于定义组件的 HTC 特定元素的组合。
