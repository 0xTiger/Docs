## 什么是 .dochtml 文件？

DOCHTML 文件是使用 Microsoft Word 生成的文档的 HTML 格式。它是创建的文档的 [HTML](/zh/web/html/) 版本，可用于发布到 Web 服务器。 DOCHTML 文件可以在 Web 浏览器中打开，以查看转换后的 Word 文档的内容。 DOCHTML 文件的外观和外观与原始 Word 文档相同。

## DOCHTML 文件格式

DOCHTML 文件以 HTML 格式保存到光盘上，准备好在 Web 服务器上发布。 Microsoft Word 停止支持将文档另存为 DOCHTML 文件。这已替换为现在常用的 [.mht](/zh/web/mht/) 和 [.mhtml](/zh/web/mhtml/) 文件格式。
