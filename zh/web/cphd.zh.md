## 什么是 .CPHD 文件？

CPHD 文件是 CloudChan 预处理器引擎使用的超文本文档。它包含用于创建动态网页和文档的 HTML 和脚本。 CPHD 文件内的脚本类似于 Java 使用的语法。当在 CloudChan 进程上执行时，CPHD 网页会返回 HTML 文档。 CPHD 格式可以与任何 XML 文档一起使用。

## CPHD 文件格式

CPHD 文件以纯文本文件形式存储到光盘上。这使得可以在任何文本编辑器（例如记事本或记事本++）中打开这些文件。
