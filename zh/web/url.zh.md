## 什么是 URL 文件？

带有 .url 扩展名的文件是一个网站 URL 快捷方式，可以使用任何网站地址保存到计算机。双击打开时，URL 将在默认浏览器中打开网站，并将页面显示给用户。 URL 文件通常被称为网站的快捷方式文件，由默认浏览器的图标表示。 URL 文件可以在任何文本编辑器中打开和编辑，例如 Microsoft Notepad、Notepad++ 或 Apple TextEdit。

## URL 文件格式 - 更多信息

URL 文件以纯文本格式保存，可以在任何文本编辑器中打开和查看。这些在用户执行其他浏览的默认 Web 浏览器中打开。
