## 什么是一 .ewp 文件？

EWP 文件是使用 [EWISoft](http://www.ewisoft.com/) Website Builder 软件创建的项目文件。它链接到包含项目所有信息的二进制数据库文件。 EWP 文件通常用于保存 Web 开发进度。它是一个模板驱动的网页设计软件，允许用户从网站模板创建网站。

## EWP 文件格式 - 更多信息

EWP 文件是不需要在任何服务器上编译或托管的项目文件。这些文件保存为纯文本文件，并包含对 Web 开发进度的引用。开发完成后，可以将项目发布到网站上。
