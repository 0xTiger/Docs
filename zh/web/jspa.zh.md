## 什么是一 .jspa 文件？

JSPA 文件是指向 Java Server (.jsp) 文件的参考快捷方式文件。 JSPA 文件指向的 URL 是一个逻辑名称，可以映射到后端的 Servelet、JSP 或任何其他东西。一个 JSPA 文件在调用时可以运行多个页面。
