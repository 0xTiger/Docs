## 什么是 DOCMHTML 文件？

DOCMHTML 文件是早期版本的 Microsoft Word 保存的 Web 存档的旧变体。最新版本的 Microsoft Word 不再支持它。 DOCMHTML 文件包含原始 .doc word 文档的 HTML 表示形式。它可以在 Microsoft Word 和 Web 浏览器中打开。发布 DOCMHTML 后，它可以托管在用户可以通过 Web 请求访问的 Web 服务器上。另一种类似的格式是 [DOCHTML](/zh/web/dochtml/)，它也已停止使用。

## DOCMHTML 文件格式 - 更多信息

DOCMHTML 文件用于以 HTML 格式保存。它可以在 Notepad 和 Notepad++ 等文本编辑器中打开。在被标记为过时之后，Microsoft Word 引入了将 word 文档保存为 [MHTML](/zh/web/mhtml/) 和 [MHT](/zh/web/mht/) 文件的选项。这些新的文件格式可以用网络浏览器打开。
