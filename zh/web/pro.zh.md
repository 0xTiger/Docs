## 什么是 .PRO 文件？ ##

PRO 文件是由 Adobe Proto 创建的网站或移动应用程序的模型，Adobe Proto 是一个已失效的 Adobe Touch 应用程序。它包括一个或多个用于网页或移动应用程序的 UI 布局，每个布局都可能包含交互元素。 PRO 文件支持 CSS、WebKit 和 jQuery 技术。

### PRO 文件格式简史

PRO 文件由 Adobe Proto 创建，后来被 Adobe XD 取代。它用于为平板电脑创建原型应用程序。 Adobe Dreamweaver 等常见 Web 设计应用程序使用这些 PRO 文件来创建 Web 和移动应用程序。

## PRO 文件格式 - 更多信息

PRO 文件是开发网站或移动应用程序的第一步。开发人员首先创建定义应用程序外观的数字模型。这些还定义了整个应用程序工作流程和通过应用程序屏幕的导航。这些模型还包括许多导航元素，例如按钮、组合框、滚动条、输入字段和其他一些元素。
