## 什么是一 .CMAKE 文件？

CMAKE 文件是由开源跨平台工具 **CMake** 创建的项目文件。它包含整个程序的 CMake 语言源脚本。 CMake 工具使用称为 CMakeLists 的编程脚本来为特定环境（如 makefile Unix 机器）生成构建文件。 CMake 项目使用的脚本和模块被组织为项目中引用的 script.cmake 和 module.cmake。
