## 什么是一 .rbxm 文件？

RBXM 文件是 Roblox Studio IDE 中使用的 3D 模型文件格式，用于存储在线多人 Roblox 游戏的 3D 对象。它包含使用 IDE 为游戏创建的 3D 对象。 RBXM 文件中的 3D 对象示例包括衬衫、裤子、帽子、建筑物、车辆、动物、飞机、汽车或植物。 RBMX 文件可以使用 Roblox IDE 转换为 [OBJ 3D 文件格式](/zh/3d/obj/)。 Roblox IDE 可用于打开 RBMX 文件。

另一种流行的 Roblox 文件格式是 [RBXL](/zh/programming/rbxl/)，它是一种位置文件格式，包含在 Roblox 游戏中以 3D 字定义的位置。

## RBMX 文件格式 - 更多信息

RBXM 文件以二进制文件格式创建和保存，其内部文件格式规范不公开。这与基于 [XML](/zh/web/xml/) 文件格式的 RBXMX 文件格式不同。 Github 上有很多工具/存储库可用于[处理 Roblox RBXM](https://github.com/search?q=rbxm) 文件。
