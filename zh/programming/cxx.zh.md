## 什么是一 .cxx 文件？

CXX 文件是一个源代码文件，类似于 C++，用于编写应用程序。它可以包含整个应用程序的完整实现，也可以包含可以包含在具有其他 C++ 文件的项目中的程序的部分实现。 CXX 文件可以在任何文本编辑器和流行的编程 IDE（例如 Notepad++、Turbo C++ 和 Visual Studio）中打开，

## CXX 文件格式

CXX 文件存储为纯文本文件，并包含用 C++ 语法编写的代码。 CXX 文件通常也带有 [.cpp](/zh/programming/cpp/) 扩展名。 CXX 文件被认为是某些编译器使用的 .cpp 的替代文件。
