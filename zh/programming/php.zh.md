## 什么是一 .php 文件？
带有 .php 扩展名的文件是指开源编程语言，用于编写服务器端脚本，在 Web 服务器上执行。它是使用最广泛的 Web 脚本语言，通常用于开发大型 Web 应用程序。网络上最大的博客系统 WordPress 和最大的社交网络 Facebook 都是基于 PHP 语言的。它兼容各种平台，如 Windows、Linux、Unix、Mac OS X 等。它还成功地运行在当今使用的几乎所有服务器上，例如 Apache、IIS 等。PHP 支持包括 MySQL 在内的广泛数据库。

## PHP文件格式##

单个 PHP 可以包含文本、CSS、[HTML](/zh/web/html/)、JavaScript 和 PHP 代码。 PHP 代码在服务器上执行，结果以纯 HTML 形式返回给浏览器。 PHP 文件可以用任何文本编辑器打开并就地编辑，尽管 Adobe Dreamweaver、Eclipse PHP 开发工具等应用程序提供了一种方便的方式来编写和修改 PHP 代码。

＃＃＃ 句法 ＃＃＃

在服务器上执行 PHP 脚本，并将纯 HTML 结果发送回浏览器。 PHP 脚本以<?php and ends with ?>

### PHP 示例###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
