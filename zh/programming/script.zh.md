## 什么是脚本文件？

SCRIPT 文件是包含任何脚本语言的脚本的源文件。这使它成为一个包含变量和函数的通用脚本文件。由于每种脚本语言可能不同，因此在 SCRIPT 文件中编写的脚本需要使用适当的脚本引擎执行。

## 脚本文件格式 - 更多信息

SCRIPT 文件以纯文本文件的形式存储到光盘中。这些可以使用任何文本编辑器（例如 Microsoft Notepad、Notepad++ 和 Apple TextEdit）打开和编辑。由于脚本语言可以不同，因此可以通过将文件扩展名重命名为相应脚本语言的默认扩展名来执行。例如，有几种常见的脚本语言，如 JavaScript ([.JS](/zh/web/js/))、VBScript ([.VBS](/zh/programming/vbs/))、AppleScript (.SCPT)、PHP ([ .PHP](/zh/programming/php/)) 和 ASP ([.ASP](/zh/web/asp/))。
