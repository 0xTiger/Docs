## 什么是一 .asc 文件？

ASC 文件是用 ActionScript 编程语言创建的 **script** 文件。它包含在服务器端 Flash Media Server 托管软件上运行的代码。 ASC 文件包含用于控制应用程序事件的脚本，例如加载 Flash 应用程序、处理来自鼠标和键盘事件的用户操作、播放多媒体内容和呈现应用程序输出。 Adobe 收购了 Macromedia Flash，后者后来作为 Adobe Flash 成为 Adobe Creativity 套装的一部分。它用于将动画文件保存为[.swf 文件格式](page-description-language/swf/)。

## ASC 文件格式 - 更多信息

ASC 文件保存为文本文件，可以使用 [Adobe Media Encoder](https://www.adobe.com/products/media-encoder.html) 打开和编辑。这些文件被创建并保存为文本文件，可以使用任何文本编辑器打开，例如 Microsoft Notepad、Notepad++ 和 Apple TextEdit。 ASC 文件在 Flash Media Server 运行和应用程序时默认执行。
