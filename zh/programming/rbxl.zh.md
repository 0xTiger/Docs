## 什么是一 .rbxl 文件？

RBXL 文件是一个 Roblox 位置文件，其中包含在 [Roblox 游戏](https://www.roblox.com/) 内的 3D 世界中定义的位置。 Roblox 本身是一款多人构建游戏，允许玩家使用平台提供的开发者工具构建游戏。 RBXL 文件使用工具、脚本、团队和部件将比赛场地描述为 3D 模型。 Roblox Studio 允许玩家创建游戏项目，并可用于打开 RBXL 文件。

## RBXL 文件格式

RBXL 文件是使用 Roblox Studio 开发的，并以 [XML](/zh/web/xml/) 文件格式保存。由于这种通用文件格式，有很多工具和 API 可让您创建和打开这些文件。一个这样的 API 是 [MaximumADHD](https://github.com/MaximumADHD/Roblox-File-Format)，一个 [C#](/zh/programming/cs/) 库，用于以 Roblox 的模型/位置文件格式创建和操作文件.
