## 什么是 .asi 文件？ ##

ASI 文件是 Alpha Five 软件使用的一种可变文件格式，用于开发 Web 应用程序。它存储 Web 应用程序使用的变量。当您在 Alpha Five 中创建新的开发人员项目时，ASI 文件也会与 Alpha Five 开发人员项目一起创建和保存。

ASI 文件可以用 Alpha Software Alpha Anywhere 打开。

## ASI 文件格式

ASI 文件以 Alpha Five 专有文件格式保存。它们作为开发人员项目的一部分由应用程序读取和写入。
