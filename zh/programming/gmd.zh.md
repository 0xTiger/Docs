## 什么是一 .gmd 文件？

GMD 文件是使用 Game Maker 软件开发工具生成的游戏开发源文件，用于创建 2D 游戏。 GMD文件存储为文本文件，是要开发的游戏的项目文件。 Game Maker 适用于 Windows、Mac、Linux、Android、iOS、HTML5、Xbox、PlayStation 和 Nintendo Switch。该工具被世界各地的独立开发者、专业工作室和教育工作者广泛用于创建游戏。

## GMD 文件格式

GMD 文件以二进制文件格式保存，其内部文件格式详细信息不可供开发人员参考。
