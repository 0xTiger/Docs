## 什么是一 .yml 文件？

YML 是数据序列化语言 [YAML](/zh/programming/yaml/)（另一种标记语言）中的文本文件，通常用于创建配置文件。它可以与任何编程语言结合使用，包括 C/C++、Ruby、Python、Java、PHP、C# 等。它独立于任何特定的编程语言，是开放的、可互操作的和人类可读的。 YML 文件可以使用 Microsoft Notepad、Notepad++、Github Atom、Apple TextEdit 和 Visual Studio IDE 等常用文本编辑器打开和编辑。

## YML 文件格式 - 更多信息

YML 文件将应用程序设置存储为配置。它的基本结构是一个映射，也可以称为目录或哈希。 YML 文件中的条目由键和值对组成。大多数情况下，配置 .yml 文件保存为 `configure.yml`，尽管一些编程语言（如 Ruby on Rails）将数据库连接配置保存在 `database.yml` 文件中。 [YML/YAML 文件格式规范](https://yaml.org/spec/1.2.2/) 解释了完整的语言结构和语法相关信息。
