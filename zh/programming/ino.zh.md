## 什么是 INO 文件？

**INO 文件** 与 **Arduino Sketch** 相关，它是使用 **Arduino 编程语言** 为 Arduino 板编写的程序。 Arduino 编程语言基于 Wiring；它类似于C/C++。 Arduino 草图以 .ino 文件扩展名保存，用于控制 Arduino 电路板。

## INO 文件格式 - 更多信息

Arduino 平台专为电子原型设计而设计，使开发人员能够在专用电路板上制作和部署程序（称为草图）；这些草图的范围从基本任务（例如点亮电路板上的 LED）到复杂的操作（例如通过互联网传输数据）； Arduino 草图的创建在 **Arduino 集成开发环境 (IDE)** 中进行，这些草图以 .ino 文件扩展名存储。

在 Arduino 1.0 之前的时代，草图由 Arduino IDE 以 .PDE 文件扩展名保存。尽管 IDE 仍然支持打开 .PDE 文件，但建议的做法是使用 .ino 文件扩展名保存所有草图，包括最初存储为 [.PDE](/programming/pde/) 文件的草图。这种转变反映了平台的演变并确保与当前标准的兼容性。

## 关于阿杜伊诺

Arduino 是一个开源电子平台，可简化交互式项目的原型设计和开发； Arduino 由硬件和软件元素组成，可满足从新手到专业人士的不同用户群的需求。该平台的核心是 Arduino 板，配备数字和模拟引脚、USB 连接以及基于 Atmel AVR 或 ARM 等处理器的各种模型。

Arduino IDE 作为用户友好的界面，用于编码、编译程序（草图）并将其上传到电路板；这种编程语言类似于 C/C++，专为易于使用而设计，为初学者简化了复杂性。 Arduino 库为从传感器集成到通信协议等任务提供预先编写的代码片段，从而促进高效开发。

Arduino 扩展板、附加板、扩展功能，提供以太网连接或电机控制等功能。充满活力的 Arduino 社区通过论坛、教程和协作项目为广泛的知识库做出了贡献； Arduino 的应用涵盖机器人、家庭自动化、艺术装置和基于传感器的系统，其多功能性和简单性使其成为各种电子项目的首选。

## 如何打开 INO 文件？

**Arduino集成开发环境**可用于打开INO文件。但由于它们是纯文本文件，因此可以使用任何文本编辑器打开或编辑其内容。
