## 什么是一 .ml 文件？

ML 文件是使用通用函数式编程语言 ML（通常称为元语言）创建的文本文件。它包含 ML 源代码，这些源代码更常用于使用完全定义的语义进行编程语言研究和设计。 ML 有多种变体，其中 Caml 是广泛使用的一种。其他包括 Objective Caml (OCaml)、JoCaml、Extended ML 和 MacroML。 ML 语言通常用于编译器编写、自动定理证明和形式验证。
