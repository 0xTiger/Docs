## 什么是 .NX 文件？

NX 文件是创建用于虚拟游戏控制台 **LowRes NX** 的视频游戏文件。它是用 BASIC 编程语言编写的，包含定义游戏功能的游戏源代码。您可以创建自己的 NX 文件，以便与可以加载的应用程序一起使用，并且用户可以玩游戏。

您可以在文本编辑器、LowRes NX 或任何源代码编辑器（例如 Microsoft Visual Studio Code）中打开 NX 文件。

## NX 文件格式

NX 文件以包含游戏说明的纯文本文件保存到光盘上。每个 NX 文件都有游戏的源代码，其中包含多个卡带 ROM 条目。这些 ROM 条目具有游戏组件信息，例如图形、关卡地图、音频和游戏的其他非源代码数据。可以使用与 LowRes NX 捆绑在一起的 NX 编辑程序来添加或编辑程序的 ROM 条目。

## 如何创建 NX 文件？

### 在 PC 上创建 NX 文件

您可以使用以下方法在 PC 上创建 NX 文件：

* 文本编辑器 - 例如 Microsoft Notepad、Notepad++ 或 Github Atom
* 源代码编辑器 - 例如 Microsoft VIsual Studio Code

### 在 iOS 上创建 NX 文件

LowRes NX 具有 iOS 版本，开发人员可以使用该版本在此应用程序中创建 NX 文件。
