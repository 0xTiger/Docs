## 什么是 .groovy 文件？

GROOVY 文件是用 Groovy 编程语言编写的源代码文件。在 GROOVY 文件中编写的代码类似于面向对象的语言 [Java](/zh/programming/java/)，这使得应用程序的设计和开发变得容易。 GROOVY 文件被编译为 Java 虚拟机 (JVM) 字节码，并与其他 Java 代码和库兼容。

## GROOVY 文件格式 - 更多信息

GROOVY 文件包含用 Groovy 语法编写的源代码。它可以包含可用作 Java 平台的编程语言和脚本语言的代码。 Groovy 语言具有类似于 Python、Ruby 和 Smalltalk 的特性。
