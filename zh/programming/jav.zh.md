## 什么是 Jav 文件？

Jav 文件是用 Java 编程语言创建的 Java 源代码文件。它类似于Java源文件，可以用Java解释器编译。与 [Java](/zh/programming/java/) 文件一样，Jav 文件用于开发 Java 应用程序。它可以是 Java 项目中可用的众多源代码文件之一。

## JAV 文件格式

Jav 文件存储为文本文件，可以使用任何文本编辑器打开。 Jav 文件类型通常已被 Java 文件类型取代，并且通常用于编写 Java 应用程序。 Jav 代码可以使用 Eclipse 和 IntelliJ Idea 等 IDE 编写，这使得处理 Java 文件变得更加容易。

### Jav 文件格式示例

```java
/*
The example code prints
Hello World from Java to the console.
*/
    public class JavaApp {
        public static void main(String[] args) {
            System.out.println("Hello World."); // Prints the string to the console.
        }
    }
```
