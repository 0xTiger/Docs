## 关于 DOC 到 JPG 转换

如果您曾经使用过旧版本的 Microsoft Word（即 Microsoft Word 2007 之前的版本），您必须熟悉 DOC 文件格式。在 Microsoft Word 2007 引入 DOCX 文件格式之前，它是 Microsoft Word 使用的默认文件格式。DOC 到 JPG 版本是一个非常常见的要求，尤其是当 Microsoft 不允许您将 Word 文档导出为 JPG 图像时。此外，最新版本的 Microsoft Word 并不完全兼容较旧的 DOC 文件。您可以使用在线应用程序（例如 Aspose.Words 应用程序）将 [DOC](/zh/word-processing/doc/) 转换为 [JPG](/zh/image/jpeg/) 文件格式。

## DOC 到 JPG 转换 - 免费演示

Aspose.Words 应用程序是一组在线工具，可让您将 DOC 文件转换为多种不同的文件格式。只需在 Web 浏览器中单击几下，即可将 DOC 文件转换为 JPG。使用Aspose.Words应用程序的优点是它不需要编码，快速且精确，最重要的是它完全免费。

在 DOC 转 JPG 的过程中，文档中的每个页面都将转换为 JPG 图像并可供下载。输出是一组保存到光盘的 JPG 图像。

### 如何将 DOC 转换为 JPG？

那么，如何将 DOC 转换为 JPG 呢？您可以使用[Aspose.Words DOC to JPG app](https://products.aspose.app/words/conversion/doc-to-jpg)将DOC转换为JPG文件格式，如下所示。

1. 前往Aspose.Words应用程序进行[DOC转JPG转换器](https://products.aspose.app/words/conversion/doc-to-jpg)
1. 单击上传按钮上传 DOC 文件
1. 完成后，按转换按钮。这将在后台启动将 DOC 文件转换为 JPG 的过程
1. DOC 到 JPG 转换过程完成后，系统将要求您下载转换后的 JPG 文件
