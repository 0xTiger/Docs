## 关于 DOC 到 DOCM 的转换

DOCM 是 Microsoft Word 文档的启用宏的版本，能够运行宏。普通 Word 文档和启用宏的文档之间的区别在于 DOCM 提供了支持宏的附加功能。

可以使用免费的在线工具和应用程序将 DOC 文件转换为 DOCM。 **[Aspose.Words App](https://products.aspose.app/words/family)** 是一组在线应用程序，可让您直接从网络浏览器对 Word 文档执行不同的操作。除了[文档查看](/zh/word-processing/docx-viewer/)、拆分、合并、比较、旋转等功能外，它还可以让您将Word文档从一种格式转换为其他格式。您可以使用 Aspose.Words 应用程序将 [DOC](/zh/word-processing/doc/) 转换为 [DOCM](/zh/word-processing/docm/)。

## 将 DOC 转换为 DOCM - 免费演示

Aspose.Words 应用程序可让您免费在线将 DOC 转换为 DOCM。您可以从 Web 浏览器中访问 DOC 到 DOCM 的转换。它快速而精确，不需要编码，并且完全免费。最重要的是，您无需在 PC 上安装任何其他软件或工具即可将 DOC 转换为 DOCM。

### 如何将 DOC 转换为 DOCM？

您可以使用[Aspose.Words DOC to DOCM app](https://products.aspose.app/words/conversion/doc-to-docm)将DOC转换为DOCM文件格式，如下所示。

1. 前往 Aspose.Words 应用程序获取 [DOC 到 DOCM 转换器](https://products.aspose.app/words/conversion/doc-to-docm)
1. 单击上传按钮上传 DOC 文件
1. 完成后，按转换按钮。这将在后台启动将 DOC 文件转换为 DOCM 的过程
1. DOC 到 DOCM 转换过程完成后，系统将要求您下载转换后的 DOCM 文件
