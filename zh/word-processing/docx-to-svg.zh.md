## 关于 DOCX 到 SVG 转换

SVG（标量矢量图形）是一种图像文件，它使用基于 XML 的文本格式来表示图像的外观。这种文件格式的优点在于，保存为 SVG 文件的图像可以缩放到不同的尺寸，而不会损失质量。 DOCX（是保存 Word 文件最常用的文档文件格式之一）转换为 SVG 是 Microsoft Word 用户对 DOCX 文件执行的常用操作之一。在线有许多应用程序和工具可用于执行 DOCX 到 SVG 的转换。 Aspose.Words 应用程序就是这样一种在线工具，可让您免费将 [DOCX](/zh/word-processing/docx/) 转换为 [SVG](/zh/page-description-language/svg/)。
