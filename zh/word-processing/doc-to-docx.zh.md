## 关于 DOC 到 DOCX 转换

在 Microsoft 推出 Office 2007 之前，所有 Word 文档默认都以二进制 DOC 文件格式保存。随着 Microsoft Word 2007 的推出，Microsoft 从所有二进制文件格式转换为 Open Office XML 规范文件格式。在此移动之后，每个二进制扩展名都附加了一个“X”来表示新的文件格式。您可能仍然存档了旧的 DOC 文件，这些文件可能需要转换为新的 DOCX 文件格式。有多种在线工具可让您免费将 DOC 转换为 DOCX。 Aspose.Words 应用程序就是这样一种在线工具，可让您在浏览器中将 DOC 转换为 DOCX。

### 如何将 DOC 转换为 DOCX？

您可以使用 [Aspose.Words DOC to DOCX 应用程序](https://products.aspose.app/words/conversion/doc-to-docx) 将 DOC 文件格式转换为 BMP 文件格式，如下所示。

1. 前往 Aspose.Words 应用程序获取 [DOC 到 DOCX 转换器](https://products.aspose.app/words/conversion/doc-to-docx)
1. 单击上传按钮上传 DOC 文件
1. 完成后，按转换按钮。这将在后台启动将 DOC 文件转换为 DOCX 的过程
1. DOC 到 DOCX 转换过程完成后，系统将要求您下载转换后的 DOCX 文件
