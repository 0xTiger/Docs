## 关于 DOCX 到 RTF 转换

如果您是 Microsoft Windows 的早期用户，您一定知道 RTF（富文本格式）文件格式。 Microsoft 在 Windows 3.0 中引入了 [RTF 文件格式](/zh/word-processing/rtf/)。 RTF 文件代表一种对格式化文本和图形进行编码以供在应用程序中使用的方法。 RTF 文件除了保留粗体、斜体和下划线等文本格式外，还可以包含文本和图像。最有趣的是 RTF 已被采用为可互操作的文件格式并且是跨平台的。因此，在跨不同操作系统共享文档以进行编辑时，DOCX 到 RTF 转换是一种流行的选择。
