## 关于 DOC 到 BMP 转换

如果您是 Microsoft Word 2007 之前版本的用户，您必须熟悉 DOC 文件类型，这是当时 Microsoft Word 用于保存文档的默认文件格式。它自 Microsoft Word 诞生以来就一直在使用，并在被最新的 DOC 文件格式取代后保留了相当长的时间。内容管理系统可能仍然有 DOC 文件作为备份，这些文件可以转换为 BMP 图像。用于转换的在线工具（例如 Aspose.Word 应用程序）可让您将 [DOC](/zh/word-processing/doc/) 文件转换为 [BMP](/zh/image/bmp/) 图像。

## DOC 到 BMP 转换 - 免费演示

Aspose.Words应用程序是Aspose基于流行的编程API开发的一套在线工具。它提供文档查看、转换、压缩、旋转、分割、合并等功能。您可以直接从网络浏览器使用 Aspose.Words 应用程序将 [DOC](/zh/word-processing/doc/) 转换为 [BMP](/zh/image/bmp/)。它不需要您安装任何软件来进行转换。

### 在线将 DOC 转换为 BMP 的步骤

您可以使用[Aspose.Words DOC to BMP app](https://products.aspose.app/words/conversion/doc-to-bmp)将DOC文件格式转换为BMP文件格式，如下所示。

1. 前往 Aspose.Words 应用程序获取 [DOC 到 BMP 转换器](https://products.aspose.app/words/conversion/doc-to-bmp)
1. 单击上传按钮上传 DOC 文件
1. 完成后，按转换按钮。这将在后台启动将 DOC 文件转换为 BMP 的过程
1. DOC 到 BMP 转换过程完成后，系统将要求您下载转换后的 BMP 文件
