## 什么是一 .fodt 文件？

扩展名为 .fodt 的文件是以 OpenDocument 平面 [XML](/zh/web/xml/) 文件格式保存的文本文档。与以 [.zip](/zh/compression/zip/) 容器文件格式保存的 .odt 文件相反，FODT 文件作为人类可读文件存储在磁盘中，可以在任何文本编辑器中打开以作为纯 XML 文档查看. OpenDocument 办公文档标准由 OASIS 行业联盟开发，为任何人不受任何限制地实施铺平了道路。其他流行的 OpenDocument 文件格式包括 [.odt](/zh/word-processing/odt/)、[.ott](/zh/word-processing/ott/)、[.ods](/zh/spreadsheet/ods/)、[.odp](/zh/presentation/odp/)，以及其他类似的。

## FODT 文件格式 - 更多信息

FODT 文件以纯 XML 文件格式（也称为平面 XML 或未压缩 XML）存储，尽管它不是 OpenDocument 常用的文档格式。 OpenDocument 文件通常保存为 ZIP 包，由许多文件和目录组成。某些软件可能不支持基于 XML 的 OpenDocument 文件，因此与基于 ZIP 包的文件相比，它们的使用较少。
