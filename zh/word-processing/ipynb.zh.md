## 什么是一 .ipynb 文件？

扩展名为 .ipynb 的文件是一个 Jupyter Notebook 文件，其中包含使用 Jupyter Notebook Web 应用程序会话创建的 Notebook 的全部内容。 [Jupyter Notebook](https://jupyter.org/) 是一个交互式 Web 应用程序，用于创建和共享计算文档以使用 Python 进行分析。

IPYNB 文件用作会话的完整计算记录，包括计算输入和输出、数学函数以及结果对象（如图像和说明性文本）的丰富表示等信息。

## IPYNB 文件格式

IPYNB 在内部以 [JSON](/zh/web/json/) 文件格式保存，这是一种用于共享数据的开放标准文件格式。出于这个原因，IPYNB 文件是人类可读且易于理解的。 JSON 还使与他人共享这些文件变得容易。

## 如何转换 IPYNB 文件？

可以使用 Jupyter 界面中的 nbconvert 命令将 IPYNB 文件导出为多种不同的文件格式。这包括将 IPYNB 转换为：

* PDF
* HTML
* 重构文本
* 乳胶

## 如何查看 IPYNB 文件？

有两种方法可以查看或打开 IPYNB 文件。

1. 用户可以将 .ipynb notebook 文档上传到某个公共 URL 并从 Jupyter Notebook Viewer 打开它<nbviewer>它将笔记本呈现为来自 URL 的静态网页。
1. 下载文件并在自己的 Jupyter Notebook 实例中打开它们
