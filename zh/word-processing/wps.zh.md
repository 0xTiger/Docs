## 什么是一 .wps 文件？

扩展名为 .wps 的文件是 Microsoft Works 文档，在 2006 年之前用作 Word 文档。它类似于当时新引入的 [DOC](/zh/word-processing/doc/) 文件格式，但没有新文件格式那么多的功能。例如，它不支持宏功能和高级格式选项。 WPS 文件仍然可以使用现代文字处理器打开，但对于 Windows 10，您需要安装转换器来加载这些文件。 Microsoft Works 在 2010 年被 Microsoft Word 文档处理器取代。

## 文件格式规范

WPS 文档的文件格式规范不公开，因为微软没有向最终用户开放规范。 WPS 是二进制文件，可以使用 Microsoft 为 Word 处理器创建的文件格式转换过滤器打开。 LibreOffice、NeoOffice 和 OxygenOffice 包含一个通用 C++ 库 libwps，它可以从许多不同版本的 Microsoft Works 中提取文本。
