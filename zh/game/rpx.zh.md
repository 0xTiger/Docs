## 什么是一 .rpx 文件？

RPX 文件是可以由 Wii U 游戏机加载或播放的游戏文件。它也可以在为 Wii U 文件提供模拟器环境而流行的 Cemu 视频游戏模拟器中加载和播放。 Wii U 游戏文件很大，也可以保存为 [WUD](/zh/disc-and-media/wud/) 或 [WUX](/zh/compression/wux/) 文件供游戏模拟器加载。 RPX 文件包含整个可执行游戏，其中包含所有游戏组件，例如图形、纹理、序列和游戏 ROM。

## RPX 文件格式 - 更多信息

RPX 文件以专有文件格式保存到光盘。某些 API 可用于压缩或解压缩 RPX 文件，例如 [wiiurpxtool](https://github.com/0CBH0/wiiurpxtool)。此外，Cemu 视频游戏模拟器等应用程序可让您在 PC 上的 Windows 中玩 Wii U 游戏。
