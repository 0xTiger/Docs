## 什么是一 .kodu 文件？

.kudo 文件是使用 Microsoft [Kodu Game Lab 软件](https://www.kodugamelab.com/) 创建并保存的 3D 游戏文件，这是一个教育游戏开发环境。它旨在教孩子们基本的编程原理，并允许创作者使用可视化编程语言对游戏元素的行为进行编程。 Kodu 文件可以保存并与其他用户共享，但只能在 Kodu 软件中打开和运行。

## KODU 文件格式 - 更多信息

Kodu 文件是二进制文件，以 Microsoft 专有文件格式保存。它的文件格式规范不对开发者公开，这就是为什么 Kodu 文件只能在 Kodu Game Lab 软件中运行的原因。
