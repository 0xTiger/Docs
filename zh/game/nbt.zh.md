## 什么是 .NBT 文件？

NBT 文件是 Minecraft 中使用的文件格式。 NBT 代表命名二进制标签，它是一种用于存储结构化数据的简单二进制格式（类似于 JSON 或 XML）。在《我的世界》中，NBT 文件用于存储各种类型的游戏数据，包括玩家库存、世界信息、实体数据等。

NBT 文件通常以.dat”扩展名存储，可以在 Minecraft 世界的目录结构中找到。例如，玩家库存存储在playerdata”目录中的.dat”文件中，而世界信息存储在region”目录中的.dat”文件中。

## NBT 文件格式 - 更多信息

NBT 是一种简单的二进制格式，旨在高效存储大量二进制数据。由于其灵活性和易用性，Minecraft 将其用于其数据文件。 Minecraft 是一款流行的 3D 沙盒游戏，玩家可以在其中构建和探索虚拟世界。 Java 版《我的世界》在模组制作者中特别受欢迎，他们通过编辑文件或添加新文件来自定义游戏玩法来修改游戏。 Minecraft 中的 NBT 文件经常由模组作者检查和编辑。这些文件通常使用 GZip 进行压缩，并且可以包含对于世界构建和自定义至关重要的各种游戏数据项。

## 如何打开 .NBT 文件

有多种软件工具可用于打开和编辑 NBT 文件。以下是可以打开或引用 NBT 文件的程序列表。

- **NBTExplorer**（免费）适用于（Windows、Mac、Linux）
- **NBTEdit**（免费）适用于 Windows
- **MCEdit**（免费）适用于（Windows、Linux）
