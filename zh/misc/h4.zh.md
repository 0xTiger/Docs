## 什么是 .h4 文件？

H4 或 HDF4 是分层数据格式 (HDF) 之一，用于存储大量数据。它是一种开源文件格式，支持存储复杂的异构数据。与 [HDF5](/zh/misc/h5/) 相比，HDF4 是一种较旧的数据格式。 HDF 文件格式由非营利组织 HDF 组维护和发布。 HDF 文件还支持存储元数据以完整表示存储的数据。

## HDF 文件格式简史

分层数据格式的制定于 1987 年由图形基础工作组 (GFTF) 发起。在此期间，针对单一格式的标准化提出了几种不同的文件格式。 NASA 选择 HDF 作为存储和提供科学数据的标准方法。

## HDF4 文件格式

H4 文件按照 HDF 文件格式规范保存在类似结构的“文件目录"中。这些规范定义了用于存储在磁盘上的 HDF 文件的整体结构，但用户无需担心底层文件结构。

H4文件的应用领域包括航空航天、物理、工程、金融、学术研究、基因组学、天文学、电子仪器和医疗领域。

### HDF4 数据模型

HDF4 或 H4 文件就像一个由异构数据对象组成的容器。 H4文件支持的数据集类型包括：

* 图片
* 表
* 文本元素
* 图表
* [PDF](/zh/pdf/) 和 [Excel 文档](/zh/spreadsheet/) 等文档

## H4 文件查看器

HDF 组提供由 HDFView 实用程序和 Java HDF 对象包组成的 [HDFView 软件](https://www.hdfgroup.org/solutions/hdf5/)。
