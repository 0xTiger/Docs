## 什么是一 .bps 文件？

.BPS 文件本质上是使用 Microsoft Works 创建的文字处理文档的备份,该文档通常使用 .WPS 文件扩展名。这些 .BPS 文件通常由 Microsoft Works 自动生成,作为预防措施,以防您在处理文档时程序崩溃。这是保护您的工作并防止数据丢失的一种方法。

在下一节中,我们将探讨与 BPS 文件相关的软件应用程序。

## 微软作品

Microsoft Works 是微软开发的生产力软件套件。它于 1987 年首次推出,作为完整 Microsoft Office 套件的更实惠的替代品而广受欢迎。 Microsoft Works 包含几个基本的办公应用程序,例如：

- **文字处理器：** Microsoft Works 拥有文字处理应用程序,允许用户创建和编辑文档。它使用[.WPS](/zh/word-processing/wps/)文件格式进行文字处理文档。

- **电子表格：** 它包括用于创建和管理电子表格的电子表格应用程序。电子表格的文件格式为 [.XLR](/zh/spreadsheet/xlr/)。

- **数据库：** Microsoft Works 还具有用于组织和管理数据的数据库应用程序。数据库文件的文件格式为[.WDB](/zh/database/wdb/)。

- **日历和地址簿：** 除了核心办公应用程序之外,Microsoft Works 还经常包含用于管理日历和地址簿的工具。

- **Works 字处理器格式 (.WPS)：** Microsoft Works 字处理组件的本机文件格式。 .WPS 文件包含文本,格式和简单图形。

- **停产：** Microsoft 于 2007 年正式停产 Microsoft Works,并鼓励用户改用 Microsoft Office 或其他替代品。
