系统文件是 Microsoft Windows、Mac 和 Linux 操作系统中不可或缺的文件类型。系统文件是存储在设备操作系统中最重要的文件之一，因为它们包含设备的所有系统相关信息，没有它，设备必然会发生故障或根本无法运行。出于这个原因，系统文件受到权限机制的高度保护，因此它们不会被意外甚至故意从计算机中轻松删除。为了便于指定和识别，这些文件始终存储在特定的系统文件夹中。

系统文件存在于各种文件扩展名类型中，每个操作系统都有其独特的系统文件扩展名类型，用于不同的用途。最常见的系统文件扩展名是； [CAB](/zh/system/cab/)、SYS、[DLL](/zh/system/dll/) 和 DRV。
