## 什么是 .VGA 文件？

VGA 文件是用于支持 VGA 显示器的 Windows 驱动文件。它在 Microsoft Windows 的早期版本中用于在操作系统 (OS) 级别调整 VGA 显示器。 VGA 文件，用于允许用户使用监视器的控制面板调整显示的分辨率和颜色深度。

## VGA 文件格式

VGA文件格式的内部文件格式尚不清楚，供开发者参考。 Windows 操作系统引用它来设置 VGA 显示器的视觉外观。
