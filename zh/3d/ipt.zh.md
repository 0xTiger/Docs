## 什么是一 .ipt 文件？

扩展名为 .ipt 的文件是 Autodesk 的本机 Inventor 零件文件格式。
![IPT File](../ipt2.jpg "IPT File")
它与 Autodesk 装配体 (.iam) 文件结合使用。 IPT 文件可以在 3DS Max 中作为实体对象导入，其中 ACIS 实体格式的几何图形保持相同格式。 3DS Max 保留在 Autodesk Inventor 中指定的对象命名，并且可以像其他对象一样编辑组件模型，包括应用修改器、更改材质、添加照明和相机、创建动画等。您可以使用 Autodesk Inventor 和 Autodesk Fusion 等应用程序打开 IPT 文件。

## IPT 文件格式

Autodesk **IPT 文件扩展名** 用于保存由草图、特征和实体组成的文件，这些草图、特征和实体使零件能够在装配体中使用。草图是最常用的组件，大多数零件都是从草图开始的。它是特征的轮廓和创建特征所需的任何几何图形。多个特征组成一个“零件模型"，一个多体零件可以是共享特征。平行和垂直等几何关系由草图约束控制。修改的效果可以通过调整约束或尺寸参数来控制模型的大小和形状来观察。
![IPT File Format](../ipt.jpg "IPT File Format")

**注意：** IPT 文件格式规范不公开。
