## 关于 FBX 转 GLTF

FilmBox，通常称为 FBX，是一种广泛采用的 3D 文件格式，最初由 Kaydara 制作，用于补充 MotionBuilder。 2006 年，Autodesk Inc. 掌舵，将 FBX 推向了重要地位，成为众多 3D 软件工具所采用的主要 3D 数据交换格式。

glTF 是 GL Transmission Format 的缩写，是一种 3D 文件格式，旨在将 3D 模型数据封装在 JSON 表示法中。这种使用 JSON 的实现可以有效地减少 3D 资产的尺寸，并减少运行时提取和利用它们所需的计算开销。
