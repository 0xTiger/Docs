## 什么是一 .vdx 文件？

在 Microsoft Visio 中创建但以 XML 格式保存的任何绘图或图表都具有 .VDX 扩展名。 Visio 绘图 XML 文件是在 Microsoft 开发的 Visio 软件中创建的。 Microsoft Visio 能够生成可用于演示文稿和文档的可视文档。 Visio 绘图 XML 文件包含可视元素的可视对象和元数据详细信息。文本也可以添加到这些视觉元素视觉绘图 XML 文件中。这些 Visio 绘图 XML 文件与基于 XML 的格式标准和图像数据编码规范集成，允许 Microsoft Visio 软件以 VDX 文件格式呈现和存储其内容。这些 Visio 绘图 XML 文件附加了 .vdx 扩展名。以下是可以打开 VDX 文件格式的应用程序。

* 微软视觉
* 适用于 MAC 的 CS Odessa ConceptDraw 项目

## Visio 文件格式##

从 2013 年开始，visio 引入了一种新的文件格式 .vsdx，它取代了其他两种文件格式，即 .vsd（Visio 二进制文件格式）和 .vdx（Visio XML 绘图文件格式）。 .vsdx 增加了 Visio 文件的互操作性，因为现在第三方软件也可以在文件格式级别操作 Visio 文件。 Microsoft SharePoint Server 中的 Visio Services 支持新文件格式，并且不需要任何“中间"文件格式即可发布到 SharePoint Server。有几种文件类型构成了具有以下扩展名的 visio：

* .vsdx（Visio 绘图）
* .vsdm（支持 Visio 宏的绘图）
* .vssx（Visio 模具）
* .vssm（启用 Visio 宏的模具）
* .vstx（Visio 模板）
* .vstm（Visio 启用宏的模板）

## 特征 ＃＃

Visio 包含广泛的模板和形状，有助于创建网络图、流程图、平面图和工程设计等。此外，我们还可以在创建流程建模和数据可视化方面获得帮助。 Microsoft Visio 在使用方面简单且用户友好。它有助于创建有效的可视化图表，可以进一步添加到许多其他应用程序中，例如 PowerPoint、Word 和 Publisher。

Microsoft Visio 还支持统一建模语言 (UML) 2.4、业务流程模型和表示法 2.0。此外，Visio 还支持发布 SharePoint 2010 和 2013 工作流。 Visio 还具有通过将形状与实时数据链接来使图表动态化的能力。专业人员团队可以同时在同一个图表上一起工作，他们还可以在 Visio 和 Visio Services 中查看审阅者的评论。此外，如果有人没有在他们的系统上安装 Visio，他们仍然可以通过 Office 365 或 SharePoint 查看您的图表。

## Microsoft Visio 版本##

1. **Microsoft Visio Standard**：标准版专为需要强大图表平台的个人设计，该平台拥有大量内置模板或模板。个人可以将复杂的信息简化为简单明了的图表。 Microsoft Visio Standard 包含用于业务、网络图、图表、流程图和各种其他多用途图表的模板。
1. **Microsoft Visio Professional**：专业版专为可以通过专业和多功能图表简化复杂信息的个人和团队而构建。基本功能包括 Visio Standard 2013，但具有额外和更新的形状、模板和样式，对团队合作有更高的支持，包括同时处理同一图表的不同人员的工作。此外，Visio Professional 2016 还为业务和工程图、流程图、地图和平面图等添加了额外的模板。
1. **Visio Pro for Office 365**：这是 Visio 2016 的特别版，可以通过 Office 365 访问。它具有与 MS Visio 2016 Professional 相同的功能。
