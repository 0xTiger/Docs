## 什么是 .CHCK 文件？

CHCK 文件是 Microsoft Exchange Server 上运行状况监视检查的结果而创建的。它包含有关服务器数据库的整体状态和操作性的信息。 Microsoft Exchange Server 主要用作电子邮件服务器，可让您维护业务联系人和日历。 Exchange 服务器管理员使用其 EseUtil 来检查其数据库的运行状况。

## CHCK 文件格式

CHCK 文件的内部文件格式未知，并且没有可用于读取/写入 CHCK 文件的参考资料。
