## 什么是一 .dat 文件？
带有 .dat 扩展名的文件属于通用文件格式。通常这些既不是加密的也不是二进制的，但它们几乎可以是任何东西。各种应用程序使用 DAT 文件作为其各自程序中某些操作的参考。所以我们不能说某个特定的软件或程序负责创建 DAT 文件。这些文件仅由创建它们的应用程序特别访问。


## DAT 文件格式
DAT 文件格式只是一种通用格式，它在文件中包含任何类型的数据，通常以 .dat 扩展名保存。它可能包含二进制或文本格式的数据，通常通用数据由特定于它所引用的应用程序的信息组成。大多数情况下，您可能会在其他配置文件（如 DLL 文件）中找到它们。在这里，我们可以列出一些著名的文件类型，文件扩展名为 .dat：

### VCD 视频文件
在 1990 年代，VCD 是众所周知的视频或多媒体源，用于在 DVD 之前分发电影和家庭视频。通常从 VCD 光盘中找到的视频文件带有 .dat 扩展名。因此它存储了从 VCD 播放的实际视频数据。
### 交换电子邮件附件
与 Exchange 电子邮件附件相关的 DAT 文件包含从 Microsoft Exchange Server 发送的富文本 (RTF) 邮件的格式信息。通常，当收件人的电子邮件客户端不支持 RTF 消息时，这些文件会添加到电子邮件的末尾。这些文件通常类似于 **Winmail.dat**。
### Windows 注册表配置单元文件
NTUSER.DAT 是一个常见的注册表配置文件。 Microsoft Windows 注册表使用 DAT 文件来存储用户设置。当用户登录 Windows 时会加载这些 DAT 文件。它可能包括帐户设置、密码和文件关联。
### Minecraft 数据文件
Minecraft 游戏使用 DAT 文件来保存有关玩家库存、玩家位置和世界信息的信息。 DAT 文件还用于存储关卡，这是在玩家在世界中导航时按需加载的小地图片段。
###梨形DAT文件
CCleaner、Speccy、Defraggler 和 Recuva 等应用程序使用 DAT 文件获取有关其典型操作的信息。
