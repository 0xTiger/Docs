## 什么是 .CAT 文件？

CAT 文件属于 Advanced Disk Catalog 软件，由开发各种计算机安全和取证软件工具的俄罗斯软件公司 ElcomSoft Co. Ltd 开发。它的一些产品包括用于密码恢复、移动设备获取和云提取的软件。公司成立于1990年，在俄罗斯和美国设有办事处。他们以其取证软件而闻名，总体而言，私营公司和政府机构都使用他们的软件来执行取证分析和调查。

Advanced Disk Catalog 是适用于 Windows XP/2003 的编目程序。 ADC 的多功能性源于它不依赖于数据库引擎，因此非常高效和紧凑。它的用户界面类似于 Windows 资源管理器，可以轻松组织存储在各种类型媒体上的数据，例如硬盘驱动器（包括网络驱动器）、软盘、光盘、ZIP 和 JAZ 磁盘等。

## 如何打开 .CAT 文件？

下载 Elcomsoft Advanced Disk Catalog 软件。在 Windows 上，您可以右键单击文件资源管理器，然后选择使用上下文菜单打开，指定 Elcomsoft Advanced Disk Catalog 软件的路径以打开 cat 文件。
