## 什么是一 ACCDE 文件？

具有 .accde 扩展名的文件是 Microsoft Access 2007 文件格式，用于创建仅执行数据库文件以保护自定义数据库代码。它仅将所有 VBA 代码编译为可执行文件，因此无法查看或编辑。 ACCDE 文件格式是早期版本的 Microsoft Access 使用的 MDE 文件格式的替代品。 ACCDE 是仅可执行模式下的 [ACCDB](/zh/database/accdb/) 文件，由于没有任何源代码，因此更加紧凑。这些文件的紧凑大小提高了内存使用率并提高了应用程序性能。 ACCDE 文件可以用 Microsoft Access 365 打开。

## ACCDE 文件格式

所有 Microsoft Access 2007+ 文件都使用使用 ACCDB 文件格式系列的 Jet ACE（Access 数据库引擎）。但是，Microsoft 不提供开源实现或详细的技术规范。
