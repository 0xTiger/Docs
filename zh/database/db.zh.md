## 什么是一 .db 文件？

带有 db 扩展名的文件是用于存储数据的通用数据库文件。对于这样的数据库文件，没有要遵循的官方规范。数据以结构化格式在文件内以表格、字段、数据类型和字段值的形式组织。 DB 文件也可以通过从数据表文件中读取行和列中的数据来创建。 DB 文件的内容可以进一步导出为 [CSV](/zh/spreadsheet/csv/) 和 [PDF](/zh/pdf/) 文件格式以用于报告目的。 DB 文件可以在 SQLite 和 LibreOffice 等应用程序中打开。
