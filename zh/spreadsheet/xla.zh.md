## 什么是一 .xla 文件？

扩展名为 .xla 的文件是一种 Microsoft Excel 加载项文件格式，可将模块、额外功能和其他工具添加到 Microsoft Excel。插件为 Excel 电子表格提供附加功能，并在 2007 版或更高版本中受支持。 XLA 文件具有 mime 媒体类型 application/vnd.ms-excel。这些可以使用适用于 Windows 和 MacOS 的 Microsoft Excel 打开。

## XLA 文件格式

XLA 文件基于二进制数据格式。但是，没有关于其文件格式规范的更多信息。一般假设是 XLA 遵循 [[MS-XLS] - Excel 二进制文件格式结构](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx)。
