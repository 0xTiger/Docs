## 什么是一 .sxc 文件？ ##

文件格式 SXC(Sun XML Calc) 属于一个名为 OpenOffice.org 的办公套件。这种格式通常处理用户的电子表格需求，因为它是基于 XML 的电子表格文件格式。 SXC 格式支持公式、函数、宏和图表以及 DataPilot，这是一个令人难以置信的功能，因为它自动个性化并提供原始导入数据的摘要。使用此软件创建的文件以扩展名 .sxc 保存。


## SXC 文件格式 ##

扩展名为 .sxc 的文件可以用 Microsoft excel 打开，Microsoft Excel 是 Microsoft Office 套件中的电子表格软件。还有其他可以打开扩展名为 .sxc 的文件的软件，例如 Libre Office、StarOffice 等。

这种格式的内容类型是：

* 应用程序/vnd.sun.xml.calc
* 应用程序/vnd.sun.xml.calc.template

## 文件兼容性##

SXC 文件与 Apache Open Office calc 兼容，数据也可以导出为 Microsoft excel 文件和其他格式。

## 文件处理##

ImportMatrix 和 ExportMatrix 命令可以读写 SXC 文件。当源是 SXC 文件而目标也是 SXC 文件时，会导致 **f** 被视为 Sun XML Calc Spreadsheet 文件。 Import 和 Export Matrix 都支持这种格式，当格式未指定且文件名以 .sxc 结尾时会自动检测到这种格式。

为了打开 SXC 文件，需要将适当的程序关联到该文件。每当要打开扩展名为 .sxc 的文件时，请确保其没有错误且未损坏。用户可以轻松处理文件中的任何错误，并且大多数情况下不需要技术支持。
