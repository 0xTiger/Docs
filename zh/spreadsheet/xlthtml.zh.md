## 什么是 XLTHTML 文件？

扩展名为 .xlthtml 的文件是 [XLT](/zh/spreadsheet/xlt/) 模板文件的 [HTML](/zh/web/html/) 版本，早期使用 Microsoft Excel 生成。 Excel 不再支持这种格式，并且已被将数据导出为 [MHT](/zh/web/mht/) 和 [MHTML](/zh/web/mhtml/) 文件格式所取代。 XLTHTML 文件背后的基本思想是使用户能够在 Web 浏览器中查看 XLT 模板文件，而不需要在系统上安装 Excel。以 XLTHTML 文件格式导出的文件可以在浏览器中打开。

## XLTHTML 文件格式 - 更多信息

XLTHTML 文件使用 HTML 文件格式保存 XLT 模板，以便在 Web 浏览器中打开。 XLT 本身基于 Excel 97-2003 版本引入的 Excel 二进制文件格式。这些文件可以用作具有定义格式的起始文件，以创建用于填充电子表格数据的普通文档。仅供参考，XLT 基于 Microsoft 记录的二进制 [XLS 文件格式规范](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx)开发商参考。
