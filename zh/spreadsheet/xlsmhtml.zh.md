## 什么是 XLSMHTML 文件？

扩展名为 .xlsmhtml 的文件是从 Microsoft Excel 中导出的原始 XLS 文件的 [HTML](/zh/web/html/) 版本，以在浏览器中查看数据。 Excel 早已停止使用该格式，不再支持该格式。导出的 XLSMHTML 文件遵循 XLS 文件格式，可以在 Web 浏览器中打开和查看。 Excel 提供将电子表格数据导出为 [MHT](/zh/web/mht/) 和 [MHTML](/zh/web/mhtml/) 文件格式，现在被广泛使用。

## XLSMHTML 文件格式 - 更多信息

XLSMHTML 文件是使用 HTML 代码生成的，用于以与 XLS 类似的格式表示电子表格数据。 XLSMHTML 文件中的 HTML 代码的排列方式使得输出文件在 Web 浏览器中打开时看起来与原始文件相似。 XLSMHTML 文件的文件格式规范并未由 Excel 公开，并且没有此类参考可供开发人员理解和开发应用程序。
