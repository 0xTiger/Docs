## 什么是 .toast 文件？
Roxio Toast 软件创建 CD 或 DVD 映像并以 .toast 扩展名保存文件。 Roxio Toast 是著名的 Macintosh 光盘制作和媒体转换软件应用程序，最初由 Miles Software GmbH 推出，然后被 Adaptec 收购。 Toast 文件可能包含任何类型的光盘，例如视频、音频或数据光盘，也可能是另一个光盘映像的副本。

## 吐司文件格式
Roxio Toast 软件使用 Toast 文件格式将其输出保存在扩展名为 .toast 的文件中。虽然，各种类型的光盘都可以通过 Mac OS 系统直接刻录，但是 Toast 提供了对过程的额外控制，并且还提供了一些额外的功能，例如编目、跟踪刻录到光盘的文件以及对损坏的光盘进行文件恢复。还提供对 Quicktime 不支持的音频和视频格式的支持，例如 [FLAC](/zh/audio/flac/) 和 [OGG](/zh/audio/ogg/)。由于它是专有文件格式，因此无法获得有关文件格式整个结构的信息。
### 光盘类型
因此，Roxio Toast 可以创建 Toast 文件，包含以下类型的光盘映像数据：
- **制作视频光盘**：Toast 可以使用以下项目创建包含视频光盘的 Toast 文件：
- DVD 视频
- 蓝光视频
- 高清 DVD
- VIDEO_TS 文件夹
- VIDEO_TS 编译
- BDMV 文件夹
- AVCHD 档案
- **制作数据光盘**：Toast 可以使用以下项目创建包含数据光盘的 Toast 文件：
- 仅限 Mac
- Mac 和 PC
- DVD-ROM
- 自定义混合
- Mac 音量
- 照片光盘
- **制作视频光盘**：Toast 可以使用以下项目创建包含数据光盘的 Toast 文件：
- 音频 CD
- 音乐 DVD
- MP3 光盘
- 增强的音频 CD
- 混合模式 CD
- **复制光盘**：Toast 支持以下方式复制：
- 光盘复制
- 图像复制
- 光盘图像合并
