## 什么是一 .vcd 文件？
VCD 文件是 DVD 或 CD 的磁盘映像。 VCD 文件可以被典型的虚拟 CD ROM 软件识别，操作系统将这些文件理解为实际的 CD 或 DVD。而软件正在从硬盘中挑选数据。 VCD 文件是实际 CD 和 DVD 的副本，无需物理驱动器即可使用，有利于在网络计算机上安装光盘。

## VCD 文件格式
VCD 文件格式被归类为创建扩展名为 .vcd 的磁盘映像的磁盘映像文件格式；用于访问 CD 或 DVD 的内容，无需物理 CD-ROM 或 DVD-ROM 驱动器。 VCD文件集成了磁盘映像和格式规范的压缩，用于将各种类型的文件甚至光盘和硬盘驱动器的全部内容归档到一个虚拟CD文件中；可以通过使用虚拟驱动器软件来访问存储在VCD文件中的数据来挂载。因此，VCD 文件格式会创建您自己的光学媒体的图像，因此您可以随时使用它们，而无需手头有物理媒体。您的数据的可用性将是最大的，而丢失、损坏和被盗不是 VCD 文件的问题。此文件格式的整个结构或规范不公开。
