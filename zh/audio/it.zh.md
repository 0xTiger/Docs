## 什么是IT文件？

IT 文件格式与 Impulse Tracker 软件相关,该软件是 Jeffrey Lim 在 20 世纪 90 年代中期开发的流行音乐跟踪程序。 IT 文件是包含音乐模式,样本数据,乐器设置和其他排序信息的模块文件。

IT 文件格式是一种二进制格式,以其在表示复杂音乐作品方面的灵活性和效率而闻名。它支持多达 64 个通道,允许各种效果,如平移,音量滑块,颤音等,支持多种样本格式,包括数字样本和合成波形。 IT 文件在 demoscene 和 tracker 音乐社区中被广泛使用,特别是在 20 世纪 90 年代末和 2000 年代初。

## 如何打开IT文件？

Impulse Tracker (Windows) 可用于打开 IT 文件,但是,由于该软件不再积极开发,因此它可能与您的操作系统版本不兼容,并且可能无法按预期运行。

以下是一些可以打开 IT 文件的其他选项。

- 适用于 Windows 的 OpenMPT
- 适用于 Windows 的 Nullsoft Winamp
- 分裂追踪器
- Audacity 带有第三方插件

## 如何转换 .IT 文件?

OpenMPT 和 Schism Tracker 是能够将 IT 文件转换为不同文件格式的程序示例。例如

- [MP3](/zh/audio/mp3/) - MP3 音频文件
- [OPUS](/zh/audio/opus/) - Opus 音频文件
- [MID](/zh/音频/mid/) - MIDI 文件
- [WAV](/zh/audio/wav/) - WAVE 音频文件
- [OGG](/zh/audio/ogg/) - Ogg Vorbis 音频文件
- [FLAC](/zh/audio/flac/) - 免费无损音频编解码器文件
