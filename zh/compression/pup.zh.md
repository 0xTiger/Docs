## 什么是 Linux PUP 文件？

Linux PUP 文件是 Puppy Linux 操作系统使用的安装包。它作为安装程序执行并开始安装包文件。在 Linux 上，它通常被称为 DotPup 文件，类似于 Windows 操作系统上使用的 [.exe](/zh/executable/exe/) 和 [.msi](/zh/executable/msi/) 安装程序文件。较新版本的 Puppy Linux 使用 .pet 文件格式而不是 PUP 文件格式。
