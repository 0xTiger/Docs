## 什么是一 .mint 文件？

MINT 文件是 [Linux Mint](https://linuxmint.com/) 发行版使用的安装包文件。它包含被提取并用于启动 Linux Mint OS 安装的压缩源代码文件。 Linux Mint OS 具有易于使用的 GUI（图形用户界面），并且是第一个拥有优雅 GUI 设计的操作系统。 MINT 文件与安装程序包一起解压缩并存储在临时目录中。从安装 Linux Mint OS 的同一目录执行安装。

## Linux MINT 文件格式

MINT 文件格式的内部文件格式详细信息不公开。但是，存储在 MINT 文件中的压缩安装文件符合与 Linux Mint OS 安装程序集成的压缩标准。这使得这些文件可以解压缩并成为安装的一部分。
