## 什么是一 .jfproj 文件？

扩展名为 .jfproj 的文件是在 Windows 10 上使用 Microsoft Font Maker 应用程序创建的基于 JSON 的自定义字体。这些字体可以通过使用带有触摸屏设备（如平板电脑）的电子笔创建，但也可以使用其他计算机外围设备（如鼠标）以防万一的计算机。 JFPROJ 文件包含用于通过定义字符和单词之间的间距以及字体大小来将字符呈现到屏幕上的字形。 Microsoft Font Maker 应用程序可在 Windows 商店下载。
