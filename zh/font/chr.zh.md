## 什么是 .chr 文件？

CHR 文件还可以用作 Microsoft Windows 使用的字体配置文件。此类文件存储用于调整操作系统内字体大小和样式的基本数据。与系统字体交互的软件应用程序可以利用这些“.chr”文件来自定义这些字体在屏幕上的显示方式。

## Borland 字符集

“字符集”是指与特定数字代码相关联的字符、符号或字形的预定义集合。集合中的每个字符都被分配了唯一的代码或标识符，通常表示为数值。字符集是计算机和数字系统中编码和表示文本的基础。

以下是有关字符集的几个要点：

1. **ASCII字符集：** 最知名的字符集之一是ASCII（美国信息交换标准代码）字符集，它包括字母、数字、标点符号和控制字符等字符。 ASCII 使用 7 或 8 位来表示每个字符。
    





2. **Unicode：** Unicode 是广泛使用的字符集，旨在涵盖世界上所有书写系统的字符。它为每个字符使用唯一的代码点（数值），允许表示来自各种语言、脚本和符号的字符。
    





3. **字符编码：** 字符集常常与字符编码结合使用。字符编码是字符代码和可以以数字方式存储或传输的二进制表示形式（通常是字节）之间的映射。常见的字符编码包括 Unicode 的 UTF-8 和 UTF-16。
    





4. **语言支持：** 字符集在语言支持方面可能有所不同。某些字符集特定于某些语言或脚本，而其他字符集则更全面并支持多种语言。
    





5. **旧字符集：** 过去，不同的计算机系统和区域使用自己的字符集，这可能会导致系统之间共享数据时出现问题。 Unicode 在很大程度上解决了这些兼容性问题。
    





6. **HTML 和 Web：** 当使用 Web 开发和 HTML 时，在文档的元数据中指定字符集（例如，使用 `<meta> ` 带有“charset”属性的标签）对于确保正确的文本渲染非常重要。

## 如何打开 .CHR 文件

CHR 文件不适合用户手动打开或编辑。相反，它们充当 Windows 程序（特别是那些与系统字体交互的程序）在必要时可以访问的配置文件。这些程序加载 CHR 文件以检索有关如何显示特定字体的信息，包括有关其大小、样式和其他属性的详细信息。用户通常不需要直接与 CHR 文件交互，因为字体管理是由操作系统和软件应用程序处理的。
