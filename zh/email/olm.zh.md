## 什么是一 .olm 文件？

带有 .olm 扩展名的文件是适用于 Mac 操作系统的 Microsoft Outlook 文件。 OLM 文件存储电子邮件、日志、日历数据和其他类型的应用程序数据。这些类似于 Windows 操作系统上 Outlook 使用的 [PST](/zh/email/pst/) 文件。但是，由 Outlook for Mac 创建的 OLM 文件无法在 Outlook for Windows 中打开。 OLM 文件格式是在 Outlook 2011 中引入的，用于存储 Outlook 的本地数据。单个 OLM 文件可以包含多个邮箱，以文件夹和子文件夹的形式组织数据。 OLM 文件可以在 Microsoft Outlook 365 和 Outlook for Mac 中打开/加载。

## OLM 文件格式

OLM 是具有 Application/octet-stream MIME 类型的二进制文件。 Microsoft 未公开提供 OLM 文件格式的技术规范，也没有可供开发人员参考的格式详细信息。但是，有一些 API 可以读取 OLM 文件格式并将其写入 PST 文件，以便在 Microsoft Windows 操作系统（例如 Aspose.Email）上使用。
