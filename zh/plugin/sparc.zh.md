## 什么是SPARC文件？

.sparc 文件（也称为 Skype 插件存档）用作包含 Skype 应用程序的额外功能或增强功能的捆绑包。这些插件能够引入各种功能，例如通话录音、屏幕共享、翻译、
游戏和娱乐、自定义表情符号和主题。

## SPARC 文件格式 - 更多信息

在某些版本的 Skype 中，Extras Manager 是一个可选组件，可以集成到 Skype 安装中。如果您在安装 SPARC 文件时遇到困难，则可能是您在初始安装过程中省略了 Extras Manager。要解决此问题，请重新安装 Skype 并确保选择附加管理器”选项。

## 如何安装Skype插件？

安装 Skype 插件的步骤：

1. 获取所需插件对应的.sparc 文件。
2. 如果 Skype 当前正在运行，请将其关闭。
3. 找到下载的 .sparc 文件并双击它。
4. 按照屏幕上的说明完成插件安装。
5. 安装后，重新启动 Skype 以激活插件。
