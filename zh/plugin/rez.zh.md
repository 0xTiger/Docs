## 什么是一 .rez 文件？

.rez 文件扩展名是 Ambrosia Software 的 Escape Velocity 系列使用的插件文件。这些文件用于通过添加新内容（例如船舶、任务或图形）来修改或扩展游戏。插件系统是该系列受欢迎的关键功能之一，因为它允许玩家定制他们的游戏体验并与其他人分享他们的创作。

## REZ 文件格式 - 更多信息

以下是 .rez 文件在 Escape Velocity 中如何工作的简要概述：

**插件结构：** .rez 文件本质上是保存游戏各种资源和脚本的容器。这些资源可以包括图形、声音、船舶数据、任务脚本等。

**定制：** 玩家和第三方开发者可以创建自己的插件，将新元素引入游戏中。这可能包括添加新型船只和武器到创建全新的故事情节和任务。

**兼容性：** 插件系统允许多个插件共存，玩家可以选择激活哪些插件。这意味着玩家可以通过选择符合自己喜好的插件来定制自己的游戏体验。

**社区贡献：** Escape Velocity 系列拥有一个蓬勃发展的模组社区，许多玩家创建并分享自己的插件。这极大地提高了游戏的寿命和可重玩性。

**Installation:** To use a plugin in Escape Velocity, players typically place the **.rez file** in a designated folder within the game directory. The game's launcher or menu system then allows players to activate or deactivate specific plugins.

## 关于 Ambrosia 逃逸速度

Escape Velocity 系列由 Ambrosia Software 为 Macintosh 计算机开发。游戏以广阔的开放式宇宙为背景，玩家可以在其中探索、交易和参与太空战斗。该系列以其非线性游戏玩法而闻名，允许玩家选择自己的道路和效忠对象。

原版 Escape Velocity 游戏于 1996 年发布，随后推出了两部续集：Escape Velocity Override”（1998 年）和Escape Velocity Nova”（2002 年）。这些游戏因其身临其境的游戏玩法、广泛的模组社区以及根据玩家的决定塑造故事的能力而广受欢迎。

**注：** Ambrosia Escape Velocity 和 EV Nova 已停产。

## 如何打开 REZ 文件？

您可以在 Windows 和 Mac OS 上使用 Ambrosia Escape Velocity 和 Ambrosia EV Nova 软件打开 REZ 文件。
