## 什么是 .amxx 文件？

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. 它是一个脚本平台，允许服务器管理员自定义和扩展其服务器的功能。 AMXX 与 MetaMod 插件管理系统结合使用，允许多个插件在服务器上同时运行而不会发生冲突。 AMXX 插件提供的功能包括管理命令、游戏修改和服务器管理。
