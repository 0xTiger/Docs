## 什么是 .zxp 文件？

ZXP 文件用作扩展，增强了 Adobe 软件（如 Adobe Photoshop 或 Adobe After Effects）的功能。过去，Adobe Extension Manager 简化了 ZXP 文件的安装。目前，安装过程需要使用统一插件安装代理 (UPIA) 命令行工具，该工具与 Adobe Creative Cloud 桌面应用程序捆绑在一起。

## ZXP 文件格式

术语ZXP”代表ZIP 扩展包”。 ZXP 文件用于打包和分发 Adobe Creative Cloud 应用程序（例如 Adobe Photoshop、Illustrator、InDesign 等）的扩展。 ZXP 文件使用 [ZIP](/compression/zip/) 格式进行压缩，这使得分发和安装扩展变得更加容易。

在 ZXP 文件中，您通常会找到扩展所需的文件，包括 [HTML](/web/html/)、[CSS](/web/css/)、JavaScript、图像以及增强 Adobe Creative Cloud 软件或添加功能所需的任何其他资源。

### 如何安装ZXP文件？

ZXP 文件是使用 Adobe Extension Manager 或 Creative Cloud 桌面应用程序安装的。用户只需双击 ZXP 文件，扩展管理器就会处理安装过程。

ZXP 文件旨在跨不同操作系统（包括 Windows 和 macOS）工作，使开发人员可以方便地创建可供广泛受众使用的扩展。

### ZXP 文件格式的安全性

ZXP 文件可以由开发人员进行数字签名，确保扩展未被篡改并且来自可信来源。这有助于维护 Adobe Creative Cloud 应用程序的安全性和完整性。

如果您正在使用 Adobe Creative Cloud 扩展或自行开发扩展，您可能会遇到 ZXP 文件作为打包和分发扩展的一种方式。但是，使用 ZXP 文件的具体信息可能会受到 Adobe 的更改或更新，因此最好参考最新的 Adobe 文档以获取最准确和最新的信息。
