## 什么是一 .xer 文件？

XER 文件格式是 Primavera P6 项目规划和管理应用程序使用的专有项目文件格式。与 [MPP](/zh/project-management/mpp/) 文件格式一样，此文件格式存储与项目计划相关的数据，例如任务、资源、时间表和材料。 Primavera 由 Oracle 开发并提供完整的项目规划，并在石油和天然气、建筑、制造等多个实际领域得到应用。它通过集成帮助项目经理计划、调度和控制大型项目和单个项目具有财务管理和人力资本管理系统。许多软件和 API 可以读取 XER 文件并将其转换为其他流行的项目文件格式，例如 XML 和 MPP。

## XER 文件格式 ##

XER 文件是一个文本文件，可以使用文本编辑器轻松打开。在文本编辑器中打开项目相关数据可能会有所帮助，但通过手动建立链接并不容易理解。这种格式是与其他 P6 数据库交换数据最常用的格式。它对从另一种格式导入的数据的控制水平相对较低。 XER 在等效 XML 格式方面的优势包括将全局代码和日历移动到项目级别并将代码、UDF、资源和日历映射到现有值的能力。


## 打开 XER 文件的问题？ ##

可能出现并导致 XER 格式功能错误的一些常见问题如下：

* 缺少支持软件
* 损坏的文件
* 由于病毒而感染的文件
* XER 描述意外从 Windows 注册表中删除
* 在系统中没有打开文件的访问权限
* 系统中的过时驱动器
* 文件扩展名被重命名
