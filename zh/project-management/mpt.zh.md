## 什么是一 .mpt 文件？ ##

带有 .mpt 扩展名的文件是 [Microsoft Project](https://products.office.com/en-us/project/project-and-portfolio-management-software) 模板文件。这包含用于创建 [MPP](/zh/project-management/mpp/) 文件的基本信息、结构和文档设置。这些模板提供了对特定项目至关重要的默认设置，例如时间表或预算信息。但是，它不能保存特定于项目的数据，例如任务、资源或分配。修改后，模板文件可以保存为标准 MPP 文件以供进一步使用。 Microsoft Project 中提供了一个名为“Global.mpt"的主模板文件，打开它可以创建一个单独的模板。 MPT 文件使用 *application/vnd.ms-project* 和 *application/x-project* 作为标准 MIME 类型。 Windows、Linux 和 macOS 以及 Moos Project Viewer、Microsoft Project 和 MPXJ 都可以打开 MPT 文件。

## 打开 MPT 文件时出现问题？ ##

以下是可能出现并导致 MPT 格式功能错误的一些常见问题的列表：

* 缺少支持软件
* 损坏的文件
* 由于病毒而感染的文件
* 在系统中没有打开文件的访问权限
* 系统中的过时驱动器
 *   Extension of the file is renamed
