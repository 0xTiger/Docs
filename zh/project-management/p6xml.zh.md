## 什么是 P6XML 文件？ ##

P6XML 文件扩展名具有出色的可读性、适应性和广泛的用途，使其成为理想的格式。 Primavera P6 Professional 具有导出和导入 [XER](/zh/project-management/xer) 文件和 XML 文件的功能。 XML 代表**可扩展标记语言**。这种通用格式允许用户在 Primavera P6 数据库之间共享项目信息。项目数据可以通过 XML 文件轻松存储在文本文件中；这使得许多软件程序交换数据变得更简单。 Primavera P6 用户可以轻松共享和管理此类数据。

## P6XML 文件格式的优点##

* Primavera 用户可以轻松定义使用 XML 文件导出和导入的基线
* XML 导出可以包含所有项目详细的行动计划
* 任何使用 Primavera 6 的用户如果无法编辑、添加或删除某些全局数据项，例如活动代码、日历和资源，将无法导入它们
* 除了 Primavera 使用的 General 和 Project Security Profile 设置外，不会导入会影响或更改数据库中数据的数据项

## 从 Primavera P6 导出 XML 文件 ##

要将 P6 Professional 项目和项目数据导出到 P6XML 文件，请使用 Oracle Primavera Cloud。当文件从 P6 Professional 桌面应用程序形成时，您可以下载该文件并将其导入 Primavera Cloud。在不间断地操作导出过程之前，您应该规范您是否链接到 P6 Professional 数据库或 P6 EPPM。导出过程的某些功能可能因您的数据库而异。通过 P6 Proficient 数据库的 P6 Specialized 设置，导出过程在您的系统上本地运行。拥有 P6 Professional 数据库，您可以轻松地从系统上的设置中导出和打开 P6 XML 文件。

以下是描述导出过程的一些步骤：

* 打开愿望 Primavera P6 项目
* 之后，在“文件"选项卡上选择“导出"
* 选择“Primavera P6 – (XML)"。之后，点击下一步
* 双击要导出的项目
* 选择要放置导出文件的路径的截断，然后选择“完成"。
*选择您要如何保存文件的选项
* 如果导出文件不需要项目级布局，则必须清除选择以“导出所有项目级布局"。
* 选择完成，然后您将收到 XML 导出的确认

### 重要提示###

* 每当连接到 P6 EPPM 数据库时，管理员必须将内容源的 URL 传递到 P6 应用程序设置的常规页面上的 P6 URL 字段中
* 将多个项目导出到一个 XML 文件很方便
* 导出分配给项目的全局数据
* 方便导出XML文件，无需所有资源的准入
  


## 打开 P6XML 文件的问题？ ##

可能出现并导致 P6XML 格式功能错误的一些常见问题如下：

* 缺少支持软件
* 损坏的文件
* 由于病毒而感染的文件
* P6XML 描述意外从 Windows 注册表中删除
* 在系统中没有打开文件的访问权限
* 系统中的过时驱动器
* 文件扩展名被重命名
