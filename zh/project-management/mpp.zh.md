## 什么是一 .mpp 文件？

扩展名为 .mpp 的文件是 Microsoft Project 数据文件，它以集成方式存储与项目管理相关的信息。 Microsoft 开发了这种专有文件格式，以使其与他们的项目管理软件产品 Microsoft Project (MSP) 兼容。除了 MPP，MSP 还可以使用项目 XML 模式。许多应用程序和 API 允许将 MPP 转换为其他文件格式。微软现在有一个在线项目服务器，可以上传项目管理文件以供多个用户协作。

## MPP 文件格式##

Microsoft 将 MPP 开发并发布为二进制文件格式，其规范尚未由 Microsoft 公开。因此，开发人员无法获得内部文件结构和详细信息。网上有几个 API 支持读取/写入 MPP 文件，但从未达到 100% 的准确度。随着时间的推移，Microsoft 已经在其项目管理应用程序中引入了多项功能，MPP 文件格式的功能也是如此。

MPP 文件使用以下 MIME 类型：

* 应用程序/vnd.ms-项目
* 应用程序/msproj
* 应用程序/msproject
* 应用程序/x-msproject
* 应用程序/x-ms-项目
* 应用程序/x-dos_ms_project
* 应用程序/mpp
* zz-应用程序/zz-winassoc-mpp

## 打开 MPP 文件时出现问题？ ##

以下是可能出现并导致 MPP 格式功能错误的一些常见问题的列表：

* 缺少支持软件
* 损坏的文件
* 由于病毒而感染的文件
* 在系统中没有打开文件的访问权限
* 系统中的过时驱动器
* 文件扩展名被重命名
