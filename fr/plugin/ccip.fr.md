## Qu'est-ce qu'un fichier CCIP ?

Un fichier CCIP est un fichier plugin créé par le programme Curse Client qui est utilisé pour gérer et installer des modules complémentaires pour des jeux populaires comme World of Warcraft, Minecraft et autres. Les modules complémentaires sont des modifications ou des améliorations créées par l'utilisateur de l'interface ou des fonctionnalités d'un jeu, et ils peuvent améliorer l'expérience de jeu en fournissant des fonctionnalités supplémentaires ou des options de personnalisation. Le fichier CCIP contient les informations requises pour le téléchargement et l'installation de l'addon pour l'un des jeux gérés.

## Format de fichier CCIP

Lorsque vous sélectionnez l'option Installer via Curse Client dans le programme, il télécharge les fichiers sous forme de fichiers CCIP. Ces fichiers CCIP peuvent être téléchargés via le client Curse. Le client Curse a permis aux utilisateurs de découvrir, télécharger et gérer facilement leurs modules complémentaires de jeu. Il fournissait un emplacement centralisé permettant aux utilisateurs de rechercher et d'installer les dernières versions de leurs modules complémentaires préférés.
