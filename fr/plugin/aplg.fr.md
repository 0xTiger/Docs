## Qu'est-ce qu'un fichier APLG ?

Un fichier avec l'extension .aplg est un fichier plugin utilisé par le logiciel d'enregistrement de flux Internet Audials. Le plugin est chargé par le logiciel pour ajouter de nouvelles fonctionnalités telles que de nouvelles fonctions de recherche et d'enregistrement afin de capturer des flux de musique et de vidéo sur Internet. Audials propose un lecteur radio riche en fonctionnalités qui permet aux utilisateurs d'accéder et d'écouter un grand nombre de stations de radio Internet.

## Format de fichier APLG

Les fichiers APLG sont fournis dans les packages de plugins .APLP, ce qui signifie que le logiciel Audials peut charger plusieurs plugins du package en même temps. Les utilisateurs peuvent souvent enregistrer leurs chansons ou émissions de radio préférées pour une lecture ultérieure. Cela peut être utile pour créer des listes de lecture personnalisées. Audials gère généralement une vaste base de données de stations de radio, permettant aux utilisateurs de découvrir facilement de nouveaux contenus.
