Les fichiers plug-in améliorent et étendent les capacités des logiciels existants. Ils trouvent une application fréquente dans les outils d’édition d’images, de vidéos et d’audio, ainsi que dans les navigateurs Web. Ces ajouts sont souvent connus sous des termes alternatifs tels que modules complémentaires ou extensions.

Les extensions de fichier de plugin courantes incluent .PLUGIN, .8BI et .VST.
