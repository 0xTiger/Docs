## Qu'est-ce qu'un fichier INX ?

Un fichier INX est un fichier plugin utilisé par l'éditeur de graphiques vectoriels open source, **Inkscape**. Il contient des informations sur le programme, notamment le nom et l'emplacement de la fonctionnalité dans le menu. Le fichier INX spécifie également un script pour l'extension qui inclut des informations telles que le type de script, l'interpréteur, l'emplacement et les arguments de variable.

## Format de fichier INX

Les fichiers INX améliorent les fonctionnalités d'Inkscape en incorporant des fonctionnalités telles que des importateurs de fichiers et des filtres graphiques. Stockés au format XML, ces fichiers se trouvent généralement dans le répertoire \​extensions\​ d'une installation d'Inkscape.

## Comment ouvrir les fichiers INX?

Vous pouvez ouvrir les fichiers INX à l'aide du programme **Inkscape**.
