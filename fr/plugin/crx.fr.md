## Qu'est-ce qu'un fichier CRX ?

Un fichier CRX est une extension utilisée pour étendre les fonctionnalités des navigateurs Internet Google Chrome, Microsoft Edge et Firefox. Il ajoute des fonctionnalités spécifiques permettant d'accéder directement depuis le navigateur pour lequel le fichier du plugin a été développé. Les extensions sont conçues à différentes fins telles que l'analyse du trafic de sites Web, le bloqueur de publicités, le gestionnaire de mots de passe, l'analyseur de listes de commerce électronique, le traducteur de pages Web et bien d'autres.

## Format de fichier CRX

Les fichiers CRX sont stockés sous forme de fichier binaire compressé avec ses fichiers constitutifs emballés dans l'archive. Il peut contenir des fichiers [.JS](/fr/web/js/), [.JSON](/fr/web/json/), des images ou même des fichiers exécutables. Les navigateurs tels que Google Chrome permettent également de charger des extensions en mode développeur, ce qui permet un débogage plus approfondi des fonctionnalités de l'extension.

## Comment installer une extension Web ?

Les navigateurs Web tels que Chrome disposent d'une option de menu « Extensions » qui vous permet de charger des fichiers d'extension CRX. Lorsqu'une extension est chargée à partir du magasin d'extensions en ligne, elle télécharge le contenu du fichier d'extension, le décompresse et installe l'extension pour charger la fonctionnalité souhaitée. L'ensemble de ce processus ne nécessite aucune interaction de l'utilisateur pour installer le fichier CRX.

## Comment charger des fichiers crx dans Chrome ?

Pour ouvrir un fichier CRX :

  1. Accédez au menu des extensions dans le navigateur Google Chrome
  1. Faites glisser et déposez le fichier CRX sur la page d'extension du navigateur
  1. Cliquez sur « Ajouter une extension » lorsque vous y êtes invité
