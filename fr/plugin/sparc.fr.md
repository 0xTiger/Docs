## Qu'est-ce qu'un fichier SPARC ?

Un fichier .sparc, également reconnu comme une archive de plug-in Skype, sert d'ensemble englobant des fonctionnalités supplémentaires ou des améliorations pour l'application Skype. Ces plugins ont la capacité d'introduire une gamme diversifiée de fonctionnalités telles que l'enregistrement des appels, le partage d'écran, la traduction,
Jeux et divertissement, émoticônes et thèmes personnalisés.

## Format de fichier SPARC - Plus d'informations

Dans certaines versions de Skype, Extras Manager est un composant facultatif qui peut être intégré à l'installation de Skype. Si vous rencontrez des difficultés lors de l'installation d'un fichier SPARC, il est possible que vous ayez omis le gestionnaire d'extras lors de l'installation initiale. Pour résoudre ce problème, réinstallez Skype et assurez-vous de sélectionner l'option Extras Manager.

## Comment installer le plugin Skype ?

Étapes pour installer les plugins Skype :

1. Obtenez le fichier .sparc correspondant au plugin souhaité.
2. Fermez Skype s'il est actuellement en cours d'exécution.
3. Localisez le fichier .sparc téléchargé et double-cliquez dessus.
4. Suivez les instructions à l'écran pour terminer l'installation du plugin.
5. Après l'installation, redémarrez Skype pour activer le plugin.
