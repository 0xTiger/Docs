## Qu'est-ce qu'un fichier RBZ ?

Un fichier RBZ est un fichier plug-in utilisé par le programme de modélisation **Trimble SketchUp** pour la conception assistée par ordinateur (CAO). Il s'agit d'un fichier ZIP compressé contenant un ou plusieurs scripts Ruby (fichiers .rb). C'est pourquoi le type de fichier est RBZ (RB + Z). Les fichiers RBZ sont faciles à distribuer et ne sont pas répartis sur tout le réseau en raison de leur petite taille résultant de la compression.

## Format de fichier RBZ - Plus d'informations

Les fichiers RBZ sont des fichiers ZIP compressés contenant un fichier de script Ruby. Il est compressé avec la méthode de compression **DEFLATE** la plus couramment utilisée, décrite dans l'IETF **RFC 1951**.

## Comment ouvrir un fichier RBZ?

Vous pouvez ouvrir les fichiers RBZ avec Trimble SketchUp sous Windows ainsi que macOS.

En plus de cela, puisque les fichiers RBZ sont des fichiers ZIP compressés, vous pouvez les extraire avec les utilitaires d'extraction ZIP standard en renommant son extension de .rbz en .zip.
