## Qu'est-ce qu'un fichier AMXX ?

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. Il s'agit d'une plate-forme de script qui permet aux administrateurs de serveurs de personnaliser et d'étendre les fonctionnalités de leurs serveurs. AMXX est utilisé conjointement avec le système de gestion de plugins MetaMod qui permet à plusieurs plugins de s'exécuter simultanément sur un serveur sans conflits. Les fonctionnalités offertes par le plugin AMXX incluent les commandes d'administration, les modifications du gameplay et la gestion du serveur.
