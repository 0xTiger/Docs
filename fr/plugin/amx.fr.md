## Qu'est-ce qu'un fichier AMX ?

Un fichier AMX est un fichier de plug-in personnalisé pour le plug-in métamod AMX Mod pour Valve Half-Life et les jeux associés. Il ajoute des fonctionnalités et fonctionnalités supplémentaires aux fonctionnalités existantes du jeu.

## Format de fichier AMX - Plus d'informations

AMX Mod utilise l'extension de fichier AMX. Cependant, la version étendue du plugin utilise l'extension de fichier .AMXX.

## Comment ouvrir le fichier AMXX?

Vous pouvez ouvrir les fichiers AMX à l'aide de [AMX Mod X](https://www.amxmodx.org/) sur les systèmes d'exploitation Windows et Linux.
