## Qu'est-ce qu'un fichier Q1Q ?

Un fichier .q1q est un fichier plugin Photoshop développé par Flaming Pear Software pour créer des planètes 3D. Il vous permet de télécharger des données sur les nuages EN DIRECT, l'air, la texture, le climat et d'autres tonnes de détails. Il vous permet également d'exporter toutes les cartes de texture vers des fichiers multicouches et de générer des cartes pour l'animation 3D et la création de cartes.

## Format de fichier Q1Q

Les fichiers du plugin Q1Q sont enregistrés au format de fichier propriétaire Flaming Pear et les détails de son format de fichier interne ne sont pas disponibles publiquement. L'image suivante montre l'interface utilisateur du plugin Q1Q lorsqu'il est chargé dans Photoshop.

## Comment ouvrir le fichier Q1Q ?

Vous pouvez ouvrir les fichiers Q1Q à l'aide de **Adobe Photoshop 2023** et **Corel PaintShop Pro 2023**.
