## Qu'est-ce qu'un fichier CLEO ?

Un fichier .cleo est un fichier plugin utilisé dans le cadre de la bibliothèque CLEO pour la série de jeux GRand Theft Auto. Il est utilisé pour exécuter des modifications qui modifient ou élargissent le gameplay. Le fichier CLEO stocke une bibliothèque au format de fichier DLL, ajoutant des commandes de script (opcodes) utiles pour travailler avec des fichiers externes afin de modifier la mémoire du jeu. Il existe différentes versions de CLEO conçues pour GTA III, GTA Vice City et GTA San Andreas.

## Format de fichier CLEO

CLEO 4 permet de créer de nouveaux opcodes en utilisant la source du SDK CLEO. Trois exemples les plus courants de tels fichiers de plugin incluent :

 * IniFiles.cleo - opcodes pour les fichiers INI ;
 * FileSystemOperations.cleo - opcodes pour les fichiers et les dossiers.
 * IntOperations.cleo - opcodes pour les opérations logiques (AND, OR, XOR, etc.).

Les plugins sont installés de la même manière que les scripts. Pour installer un plugin CLEO, vous devez copier le fichier .CLEO dans le dossier CLEO.

## À propos de CLEO

CLEO est un plugin extensible extrêmement populaire pour la série de jeux Grand Theft Auto de Rockstar Games, permettant l'utilisation de milliers de mods uniques qui modifient ou étendent le gameplay.
