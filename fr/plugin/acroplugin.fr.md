## Qu'est-ce qu'un fichier ACROPLUGIN ?

Un fichier avec l'extension .acroplugin est un fichier plugin utilisé par Adobe Acrobat. Il est utilisé pour ajouter des fonctionnalités aux capacités existantes de l’application. Le plugin est fourni avec Adobe Acrobat, une mise en page et un programme de publication [PDF file](/pdf/). Comme d'autres plugins, le fichier ACROPLUGIN étend les fonctionnalités existantes offertes par l'application.

## Format de fichier ACROPLUGIN - Plus d'informations

Le fichier ACROPLUGIN est explicitement utilisé par Adobe Acrobat pour étendre ses fonctionnalités existantes. Les détails du format de fichier interne ne sont pas disponibles publiquement pour référence du développeur.

## Comment ouvrir le fichier ACROPLUGIN?

Vous pouvez ouvrir le fichier ACROPLUGIN à l'aide d'Adobe Acrobat.
