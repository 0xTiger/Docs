## Qu'est-ce qu'un fichier MFX ?

Un fichier MFX est un fichier de police utilisé par le jeu de simulation Maxis SimCity 4. Il est utilisé pour afficher le texte du jeu à l'écran à partir de l'un des types de police sélectionnés, notamment Arial, Courier ou toute autre police disponible dans le fichier de police. Les fichiers MFX sont installés avec le jeu SimCity 4 et n'ont pas besoin d'être créés manuellement.

SimCity 4 est un jeu informatique de simulation permettant de créer des simulations de construction de villes. Maxis, le développeur de SimCity, est une filiale d'Electronic Arts. Le jeu a été initialement lancé en janvier 2003.
