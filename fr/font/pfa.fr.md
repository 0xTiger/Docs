## Qu'est-ce qu'un fichier PFA ?

Un fichier avec l'extension .pfa est une version ASCII de la police d'imprimante de la police Type 1 d'Adobe et contient les données de glyphe de la police. Le fichier est dans un format lisible par l'homme et peut être ouvert dans n'importe quel éditeur de texte. L'interpréteur PostScript peut utiliser ce type de police dans son ensemble pour son fonctionnement. La version PFB (Printer Font Binary) de la police Type 1 d'Adobe contient les mêmes informations. Cependant, sans ligne PFA, il est au format de fichier binaire et n'est pas lisible par l'homme. Les fichiers PFA peuvent être intégrés au système en les copiant dans le dossier Fonts du système.
