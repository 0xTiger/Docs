## Qu'est-ce qu'un fichier PFB ?

Un fichier avec l'extension .pfb est une version Printer Font Binary de la police Type 1 d'Adobe. Il est similaire aux fichiers .pfa(/fr/font/pfa/) mais est de nature binaire. Les fichiers PFB stockent les polices Adobe Type 1 utilisées pour le système d'exploitation Windows. Ceux-ci contiennent les données de glyphe de la police et stockent les données sous une forme compacte au format binaire. En plus d'avoir un contenu codé en binaire, les fichiers PFB sont cryptés. Les fichiers PFB nécessitaient également des fichiers .pfm qui incluent des mesures de caractères dans la police. Pour installer les polices PFB, les fichiers .pfb et .pfm doivent être placés dans le système.
