## Qu'est-ce qu'un fichier PFM ?

Un fichier avec l'extension .pfm est un fichier Printer Font Metrics qui contient les mesures de caractères incluses dans le fichier de police. Il est indispensable d'avoir le fichier PFM à côté du fichier .pfb(/fr/font/pfb/) associé dans le même dossier lors de l'installation de ce dernier. Les fichiers PFM sont codés en binaire et ne sont pas lisibles par l'homme.
