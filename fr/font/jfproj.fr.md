## Qu'est-ce qu'un fichier JFPROJ ?

Un fichier avec l'extension .jfproj est une police personnalisée basée sur JSON créée avec l'application Microsoft Font Maker sur Windows 10. Celles-ci peuvent être créées à l'aide d'un stylo électronique avec des appareils à écran tactile tels que des tablettes, bien que d'autres périphériques informatiques tels que la souris puissent également être utilisés au cas où d'ordinateurs. Les fichiers JFPROJ contiennent les glyphes utilisés pour rendre les caractères à l'écran en définissant l'espacement entre les caractères et les mots, et la taille de la police. L'application Microsoft Font Maker est disponible en téléchargement sur le Windows Store.
