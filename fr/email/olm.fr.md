## Qu'est-ce qu'un fichier OLM ?

Un fichier avec l'extension .olm est un fichier Microsoft Outlook pour le système d'exploitation Mac. Un fichier OLM stocke les messages électroniques, les journaux, les données de calendrier et d'autres types de données d'application. Ceux-ci sont similaires aux fichiers [PST](/fr/email/pst/) utilisés par Outlook sur le système d'exploitation Windows. Cependant, les fichiers OLM créés par Outlook pour Mac ne peuvent pas être ouverts dans Outlook pour Windows. Le format de fichier OLM a été introduit avec Outlook 2011 pour stocker les données locales d'Outlook. Un seul fichier OLM peut contenir plusieurs boîtes aux lettres, organisant les données sous forme de dossiers et de sous-dossiers. Les fichiers OLM peuvent être ouverts/chargés dans Microsoft Outlook 365 et Outlook pour Mac.

## Format de fichier OLM

Les OLM sont des fichiers binaires de type Application/octet-stream MIME. Les spécifications techniques du format de fichier OLM ne sont pas disponibles ouvertement auprès de Microsoft et aucun détail de format n'est disponible pour la référence du développeur. Cependant, certaines API peuvent lire le format de fichier OLM et les écrire dans un fichier PST pour une utilisation sur le système d'exploitation Microsoft Windows, tel que Aspose.Email.
