## À propos de la conversion DOCX en DOC

Le type de fichier DOC est un format de fichier de document populaire utilisé par 2007 et les versions précédentes de Microsoft Word. Avec la sortie d'Office 2007, Microsoft a introduit le format de fichier DOCX basé sur les spécifications Office Open XML. Les fichiers DOC ont été enregistrés sous forme de fichiers binaires, contrairement au DOCX qui est enregistré au format de fichier XML compressé. Il existe de nombreux outils disponibles en ligne pour convertir des fichiers [DOCX](/fr/word-processing/docx/) en [DOC](/fr/word-processing/doc/).

### Étapes pour convertir DOCX en DOC en ligne

Vous pouvez convertir le format de fichier DOCX au format DOC à l'aide de l'[application Aspose.Words DOCX to DOC](https://products.aspose.app/words/conversion/docx-to-doc) comme suit.

1. Accédez à l'application Aspose.Words pour [convertisseur DOCX en DOC](https://products.aspose.app/words/conversion/docx-to-doc)
1. Téléchargez votre fichier DOCX en cliquant sur le bouton de téléchargement
1. Une fois terminé, appuyez sur le bouton Convertir. Cela lancera le processus de conversion du fichier DOCX en DOC en arrière-plan.
1. Il vous sera demandé de télécharger le fichier RTF converti une fois le processus de conversion DOCX en DOC terminé.
