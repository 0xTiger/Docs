## Qu'est-ce qu'un fichier IPYNB ?

Un fichier avec l'extension .ipynb est un fichier Jupyter Notebook qui contient l'intégralité du contenu de Notebook créé avec la session d'application Web Jupyter Notebook. [Jupyter Notebook](https://jupyter.org/) est une application Web interactive utilisée pour créer et partager des documents informatiques à des fins d'analyse à l'aide de Python.

Un fichier IPYNB sert d'enregistrement informatique complet d'une session et comprend des informations telles que des entrées et des sorties informatiques, des fonctions mathématiques et une représentation riche des objets résultants tels que des images et du texte explicatif.

## Format de fichier IPYNB

Les IPYNB sont enregistrés en interne au format de fichier [JSON](/fr/web/json/), qui est un format de fichier standard ouvert pour le partage de données. Pour cette raison, les fichiers IPYNB sont lisibles par l'homme et facilement compréhensibles. JSON facilite également le partage de ces fichiers avec d'autres.

## Comment convertir des fichiers IPYNB ?

Les fichiers IPYNB peuvent être exportés vers plusieurs formats de fichiers différents à l'aide de la commande nbconvert de l'interface Jupyter. Cela inclut la conversion d'IPYNB en :

*PDF
* HTML
* Texte restructuré
*Latex

## Comment afficher le fichier IPYNB ?

Il existe deux méthodes pour afficher ou ouvrir les fichiers IPYNB.

1. L'utilisateur peut télécharger le document .ipynb notebook sur une URL publique et l'ouvrir à partir de Jupyter Notebook Viewer<nbviewer> qui rend le bloc-notes sous la forme d'une page Web statique à partir d'une URL.
1. Téléchargez les fichiers et ouvrez-les dans leur propre instance de Jupyter Notebook
