## Qu'est-ce qu'un fichier WPS ?

Un fichier avec l'extension .wps est un document Microsoft Works utilisé avant 2006 en tant que document Word. Il ressemblait alors au nouveau format de fichier [DOC](/fr/word-processing/doc/), mais n'était pas aussi riche en fonctionnalités que le nouveau format de fichier. Par exemple, il ne prenait pas en charge la fonctionnalité de macro et les options de formatage avancées. Les fichiers WPS peuvent toujours être ouverts avec des processeurs de texte modernes, mais pour Windows 10, vous devez installer un convertisseur pour charger ces fichiers. Microsoft Works a été remplacé par le processeur de documents Microsoft Word en 2010.

## Spécifications du format de fichier

Les spécifications de format de fichier du document WPS ne sont pas disponibles publiquement car Microsoft n'a pas ouvert les spécifications aux utilisateurs finaux. WPS sont des fichiers binaires et peuvent être ouverts avec des filtres de conversion de format de fichier créés par Microsoft pour le traitement Word. LibreOffice, NeoOffice et OxygenOffice ont inclus une bibliothèque C++ générale, libwps, qui peut extraire du texte de nombreuses versions différentes de Microsoft Works.
