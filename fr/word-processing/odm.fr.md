## Qu'est-ce qu'un fichier ODM ?

Un fichier avec l'extension .odm est un fichier de document OpenDocument Master qui contient un ou plusieurs fichiers [.odt](/fr/word-processing/odt/) liés. Ces sous-documents sont tous combinés dans le document maître unique. Cela facilite la gestion de tous les fichiers liés à partir du document maître unique. Les fichiers de documents maîtres peuvent être créés avec Apache OpenOffice Writer ou OpenDocument Word Processor. Les fichiers ODM peuvent être exportés vers un seul fichier ODT à l'aide de l'option d'exportation depuis l'application OpenOffice Writer.

## Format de fichier ODM - Plus d'informations

Les fichiers ODM sont enregistrés sous forme de fichiers texte pouvant être ouverts dans OpenOffice Writer. Un seul fichier ODM se compose de plusieurs sections qui sont liées aux sous-documents .odt. Cela permet à plusieurs rédacteurs de modifier leurs documents ODT qui sont liés dans le fichier ODM maître. Ainsi, le fichier maître ODM est mis à jour au fur et à mesure que chaque rédacteur met à jour ses sections respectives.

Les fichiers ODM peuvent également inclure une table des matières et un index. Chaque fois qu'un document maître est ouvert, le rédacteur du document maître demande la mise à jour des liens. Il faut dire oui pour que le contenu s'affiche correctement.
