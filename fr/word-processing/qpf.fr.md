## Qu'est-ce qu'un fichier QPF ?

Un fichier avec l'extension .qpf est un fichier texte créé avec le logiciel QuickPad qui est une application gratuite proposée par Klumbu, en tant que version évoluée du Bloc-notes pour le système d'exploitation Windows. Les fichiers QPF ne peuvent être ouverts et visualisés que dans QuickPad et offrent ainsi une sécurité minimale des données sensibles. Le texte à l'intérieur du document n'est pas crypté à l'aide d'un mot de passe. Au lieu de cela, il est encodé par le logiciel pour le rendre quelque peu sécurisé.

## Format de fichier QPF

Les fichiers QPF sont des fichiers texte créés et enregistrés sous une forme lisible par l'homme. La seule différence est que pour lire ces fichiers, ceux-ci doivent être ouverts dans QuickPad. QuickPad offre des fonctionnalités telles que le retour à la ligne, la mise en forme du texte, l'enregistrement automatique, le mode de mise au point, etc.
De plus, le texte peut être importé à partir du format de fichier INI ainsi que TXT.
