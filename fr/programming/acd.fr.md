## Qu'est-ce qu'un fichier ACD ?

Un fichier ACD est un fichier de projet qui contient des informations pour la programmation des plates-formes Logix [Rockwell Automation](https://www.rockwellautomation.com). Il comprend des instructions pour les contrôleurs et les sous-traitants. Il est utilisé pour les plates-formes Logix telles que les systèmes ControlLogix, CompactLogix, FlexLogix, SoftLogix5800 et DriveLogix.

Les applications qui peuvent ouvrir les fichiers ACD incluent Rockwell Automation RSLogix 5000.
