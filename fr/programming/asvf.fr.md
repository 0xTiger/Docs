## Qu'est-ce qu'un fichier ASVF ?

Un fichier ASVF est un fichier compressé créé par le logiciel Asphyre Sphinx qui est un cadre 2D/3D pour les applications industrielles et d'entreprise. Il stocke des données telles que des polices, des images et d'autres médias. Sphinx peut également être utilisé pour développer des jeux vidéo 2D/3D. Un fichier ASVF compressé peut contenir des fichiers courants tels que des fichiers .JPEG et .XML.

Les fichiers ASVF peuvent être ouverts avec le logiciel Asphyre Sphinx qui a été abandonné depuis longtemps.

## Format de fichier ASVF

Les fichiers ASVF sont enregistrés sous forme de fichiers compressés et peuvent contenir plusieurs fichiers dans l'archive.
