## Qu'est-ce qu'un fichier FS ?

L'extension de fichier **.fs** est associée à **Visual F#**, langage de programmation développé par Microsoft dans le cadre du framework .NET. F# est un langage de programmation fonctionnel qui fait partie de l'écosystème Microsoft .NET et est conçu pour être utilisé pour une variété de types d'applications, notamment les applications Web, de bureau et cloud.

Les fichiers sources Visual F# (avec l'extension « .fs ») contiennent du code source écrit en **langage de programmation F#**. Ces fichiers peuvent inclure des fonctions, des modules, des types et d'autres constructions de langage F#. Visual Studio est un environnement de développement courant pour la programmation F#, fournissant des outils pour éditer, compiler et déboguer le code F#.

Les programmeurs utilisent des fichiers .fs pour écrire du code F# et ces fichiers sont ensuite compilés en fichiers exécutables ([.EXE](/executable/exe/)) ou bibliothèques, généralement avec l'extension [.DLL](/system/dll/). Le langage F# est connu pour sa syntaxe concise, son immuabilité par défaut et sa forte prise en charge des concepts de programmation fonctionnelle.
