## Qu'est-ce qu'un fichier GMD ?

Un fichier GMD est un fichier source de développement de jeux généré avec l'outil de développement logiciel Game Maker utilisé pour créer des jeux 2D. Le fichier GMD est stocké sous forme de fichier texte et constitue un fichier de projet pour le jeu à développer. Game Maker est disponible pour Windows, Mac, Linux, Android, iOS, HTML5, Xbox, PlayStation et Nintendo Switch. L'outil est largement utilisé par les développeurs indépendants, les studios professionnels et les éducateurs du monde entier pour créer des jeux.

## Format de fichier GMD

Les fichiers GMD sont enregistrés au format de fichier binaire et les détails de son format de fichier interne ne sont pas disponibles pour référence du développeur.
