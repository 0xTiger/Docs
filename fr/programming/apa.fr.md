## Qu'est-ce qu'un dossier APA ?

Un fichier avec l'extension .apa est un fichier de projet de développement logiciel créé par l'application RSView. Il contient les informations nécessaires à la création d'applications d'interface homme-machine (IHM). Les fichiers APA sont stockés au format de fichier d'archive. Son format de fichier d'archivage permet de les transférer facilement d'un ordinateur à un autre en raison de la taille réduite du fichier. Les utilitaires de décompression standard peuvent être utilisés pour extraire le contenu des fichiers APA. Les fichiers APA peuvent être ouverts à l'aide du logiciel Rockwell Automation RSView Studio ou RSView ME.

## Format de fichier APA - Plus d'informations

Le gestionnaire d'applications RSView enregistre les fichiers .apa par défaut dans l'emplacement C:\Documents and Settings\All Users\Documents\RSView Enterprise\ME\Archives sous Windows XP. Cependant, vous pouvez enregistrer le fichier dans n'importe quel répertoire selon les besoins.
