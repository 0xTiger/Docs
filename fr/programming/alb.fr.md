## Qu'est-ce qu'un fichier ALB ?

Un fichier ALB est un fichier de bibliothèque utilisé par l'application IDE **Alpha Five** utilisée pour développer des applications Web. Son objectif principal est de stocker des informations pour Alpha Five correspondant [fichier ADB](/fr/database/adb/). Cela inclut les informations du dictionnaire de données telles que les paramètres, les pointeurs de script, les menus, les fonctions définies par l'utilisateur, les bitmaps, les barres d'outils et les définitions d'importation.

Les fichiers ALB peuvent être ouverts avec **Alpha Software Alpha Anywhere**.

## Format de fichier ALB - Plus d'informations

Les fichiers ALB sont enregistrés au format de fichier propriétaire Alpha File et leur structure de format de fichier interne n'est pas disponible publiquement. Ceux-ci sont enregistrés en plus de la base de données globale de la bibliothèque de développeurs Alpha Five. En tant que bibliothèques, les fichiers ALB peuvent être réutilisés dans d'autres applications Alpha Five.
