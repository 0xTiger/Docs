## Qu'est-ce qu'un fichier AS3PROJ ?

Un fichier AS3PROJ est un fichier de projet créé par FlashDevelop, un IDE de programmation open source gratuit pour le développement d'applications Flash et Web. Il enregistre toutes les informations et paramètres liés au projet appliqués par l'utilisateur à l'aide de FlashDevelop IDE. Les informations stockées dans un fichier AS3PROJ incluent les paramètres de génération, les références aux fichiers source du projet et les options de génération de la sortie [fichier .swf](/fr/page-description-language/swf/). Lorsqu'un nouveau projet est créé, il est enregistré en tant que fichier AS3PROJ au même emplacement que celui spécifié par l'utilisateur pour enregistrer les données relatives au projet.

## Format de fichier AS3PROJ

Un fichier AS3PROJ est enregistré au format de fichier [XML](/fr/web/xml/) qui est un format de fichier universel pour l'échange d'informations. Il peut être ouvert dans n'importe quel éditeur de texte tel que Notepad, Notepad++ et Atom pour examiner le contenu.
