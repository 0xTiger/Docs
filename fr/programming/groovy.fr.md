## Qu'est-ce qu'un fichier GROOVY ?

Un fichier GROOVY est un fichier de code source écrit dans le langage de programmation Groovy. Le code écrit dans un fichier GROOVY est similaire au langage orienté objet [Java](/fr/programming/java/), ce qui facilite la conception et le développement d'applications. Un fichier GROOVY est compilé en bytecode de machine virtuelle Java (JVM) et est compatible avec d'autres codes et bibliothèques Java.

## Format de fichier GROOVY - Plus d'informations

Les fichiers GROOVY contiennent du code source écrit dans la syntaxe Groovy. Il peut contenir du code pouvant être utilisé à la fois comme langage de programmation et comme langage de script pour la plate-forme Java. Le langage Groovy a des fonctionnalités similaires à celles de Python, Ruby et Smalltalk.
