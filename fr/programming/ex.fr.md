## Qu'est-ce qu'un fichier EX ?

Un fichier EX est un fichier de code source qui contient le code source des applications développées à l'aide du langage de programmation Euphoria. Euphoria est un langage de programmation procédural qui croit en la simplicité de la programmation plutôt qu'en des langages de programmation complexes. Une application développée dans Euphoria peut comprendre plusieurs fichiers EX, mais un seul fichier EX peut également contenir la source complète du programme.

Les applications qui peuvent **ouvrir les fichiers EX** incluent openEuphoria Euphoria, Microsoft Notepad, Notepad++ et Atom.

## Bref historique du format de fichier EX

Le développement et l'existence du format de fichier EX sont liés au développement du langage de programmation Euphoria créé par Robert Craig. Il a d'abord été publié pour MS-DOS. Il est devenu un logiciel open-source en 2006 avec la sortie de la version 3. Son développement a été repris par le **groupe openEuphoria** qui administre et développe le projet. Sa version 4 est sortie en décembre 2010 et est disponible pour Windows, Linux, macOS et trois versions de BSD.

L'Euphoria a été initialement écrit en langage de programmation **[C](/fr/programming/c/)**. Plus tard, l'interprète d'Euphoria a été divisé en deux parties :

* `Front-end Parser` - écrit en Euphoria et utilisé avec le traducteur Euphoria-to-C et le Binder.
* `Back-end` - bibliothèque d'exécution écrite en langage de programmation C.

## Format de fichier EX - Plus d'informations

Les fichiers EX sont stockés sous forme de fichiers texte brut et contiennent le code source des programmes écrits en langage de programmation Euphoria. Ces fichiers peuvent être ouverts dans n'importe quel éditeur de texte.
