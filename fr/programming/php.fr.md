## Qu'est-ce qu'un fichier PHP ?
Un fichier avec l'extension .php fait référence à un langage de programmation open source, utilisé pour écrire des scripts côté serveur, à exécuter sur un serveur Web. Il s'agit du langage de script Web le plus largement utilisé, généralement utilisé pour le développement d'applications Web à grande échelle. Le plus grand système de blogging sur le web à savoir WordPress et le plus grand réseau social Facebook sont basés sur le langage PHP. Il est compatible avec diverses plates-formes telles que Windows, Linux, Unix, Mac OS X, etc. Il fonctionne également avec succès sur presque tous les serveurs utilisés aujourd'hui, c'est-à-dire Apache, IIS, etc. PHP prend en charge une large gamme de bases de données, y compris MySQL.

## Format de fichier PHP ##

Un seul PHP peut contenir du texte, du CSS, du [HTML](/fr/web/html/), du JavaScript et du code PHP. Le code PHP est exécuté sur le serveur et le résultat est renvoyé au navigateur sous forme de code HTML brut. Les fichiers PHP peuvent être ouverts avec n'importe quel éditeur de texte et modifiés sur place, bien que des applications comme Adobe Dreamweaver, les outils de développement PHP Eclipse offrent un moyen pratique d'écrire et de modifier du code PHP.

### Syntaxe ###

Un script PHP est exécuté sur le serveur et le résultat HTML brut est renvoyé au navigateur. Un script PHP commence par<?php and ends with ?>

### Exemple PHP ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
