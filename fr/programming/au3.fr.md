## Qu'est-ce qu'un fichier AU3 ?

Un fichier AU3 est un fichier de script automatisé créé avec le programme de script gratuit, [AutoIt v3](https://www.autoitscript.com/site/). Il contient le langage de script AutoIt3 pour automatiser les commandes dans Windows. Le langage de programmation AutoIt3 vous permet de simuler un certain nombre d'événements pouvant être utilisés pour automatiser des tâches. Par exemple, le script à l'intérieur du fichier AU3 peut être utilisé pour simuler les frappes au clavier, le mouvement de la souris, le contrôle de la position des fenêtres et à d'autres fins de script.

Les applications qui peuvent ouvrir les fichiers AU3 incluent [SciTE4AutoIt3](https://www.autoitscript.com/site/autoit-script-editor/downloads/) et AutoIt.

## Format de fichier AU3

Les fichiers AU3 sont enregistrés sous forme de fichiers texte et contiennent le langage de script AutoIt3.
