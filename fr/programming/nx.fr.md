## Qu'est-ce qu'un fichier NX ?

Un fichier NX est un fichier de jeu vidéo créé pour être utilisé avec la console de jeu virtuelle **LowRes NX**. Il est écrit dans le langage de programmation BASIC et contient le code source du jeu qui définit son fonctionnement. Vous pouvez créer vos propres fichiers NX à utiliser avec l'application qui peuvent être chargées et les utilisateurs peuvent jouer au jeu.

Vous pouvez ouvrir un fichier NX dans un éditeur de texte, LowRes NX ou n'importe quel éditeur de code source tel que Microsoft Visual Studio Code.

## Format de fichier NX

Un fichier NX est enregistré sur le disque sous forme de fichier texte brut contenant les instructions du jeu. Chaque fichier NX contient le code source du jeu qui comprend plusieurs entrées ROM de cartouche. Ces entrées ROM contiennent des informations sur les composants du jeu telles que des graphiques, des cartes de niveaux, de l'audio et d'autres données non sources du jeu. Les entrées ROM d'un programme peuvent être ajoutées ou modifiées à l'aide des programmes d'édition NX fournis avec le LowRes NX.

## Comment créer des fichiers NX ?

### Créer des fichiers NX sur PC

Vous pouvez créer des fichiers NX sur PC en utilisant :

* Éditeurs de texte - tels que Microsoft Notepad, Notepad++ ou Github Atom
* Éditeurs de code source - tels que Microsoft VIsual Studio Code

### Créer des fichiers NX sur iOS

LowRes NX a une version iOS qui peut être utilisée par les développeurs pour créer des fichiers NX à partir de cette application.
