## Qu'est-ce qu'un fichier RBXM ?

Un fichier RBXM est un format de fichier de modèle 3D utilisé dans Roblox Studio IDE pour stocker des objets 3D du jeu Roblox multijoueur en ligne. Il contient des objets 3D créés pour le jeu à l'aide de l'IDE. Des exemples d'objets 3D dans un fichier RBXM incluent des chemises, des pantalons, des chapeaux, des bâtiments, des véhicules, des animaux, des avions, des voitures ou des plantes. Les fichiers RBMX peuvent être convertis au [format de fichier OBJ 3D](/fr/3d/obj/) à l'aide de l'IDE Roblox. Roblox IDE peut être utilisé pour ouvrir des fichiers RBMX.

Un autre format de fichier Roblox populaire est [RBXL](/fr/programming/rbxl/) qui est un format de fichier de localisation et contient un emplacement défini en mot 3D dans le jeu Roblox.

## Format de fichier RBMX - Plus d'informations

Les fichiers RBXM sont créés et enregistrés au format de fichier binaire et leurs spécifications de format de fichier interne ne sont pas disponibles publiquement. Ceci est différent du format de fichier RBXMX qui est basé sur le format de fichier [XML](/fr/web/xml/). Il existe de nombreux outils/dépôts disponibles sur Github qui peuvent être utilisés pour [traiter les fichiers Roblox RBXM](https://github.com/search?q=rbxm).
