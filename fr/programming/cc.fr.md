## Qu'est-ce qu'un fichier CC ?

Un fichier CC est un type de fichier de code source qui contient du code écrit en langage de programmation C++. Il peut contenir soit tout le code nécessaire à un seul programme, soit une partie du code faisant partie d'un projet de programmation plus vaste composé de plusieurs fichiers CC. Pour modifier ou afficher le code contenu dans le fichier CC, les développeurs utilisent souvent un éditeur de code source. Après l'édition, ils compilent le code à l'aide d'un compilateur pour le convertir en code machine, que le système d'exploitation de l'ordinateur peut exécuter pour exécuter le programme.

## Format de fichier CC - Plus d'informations

L'extension de fichier .cc est utilisée pour les fichiers de code source C++ depuis plusieurs décennies. Le langage C++ a été développé à la fin des années 1970 par Bjarne Stroustrup qui travaillait alors aux Bell Labs. Stroustrup souhaitait étendre le langage de programmation C avec la prise en charge des concepts de programmation orientée objet qu'il considérait comme essentiels pour la construction de grands systèmes logiciels.

La première version de C++ a été publiée en 1983 et a rapidement gagné en popularité parmi les développeurs qui appréciaient ses fonctionnalités puissantes et sa flexibilité. À mesure que le C++ devenait plus largement utilisé, il est devenu nécessaire d'établir des conventions de codage et des conventions de dénomination de fichiers pour faciliter l'organisation et la gestion de bases de code volumineuses.

Le guide de style C++ de Google, qui recommande d'utiliser l'extension de fichier .cc pour les fichiers de code source C++, a été publié pour la première fois en 2003. Depuis lors, le guide a été mis à jour périodiquement pour refléter les changements intervenus dans le langage C++ et les pratiques de développement de la communauté de programmation au sens large.

Aujourd'hui, l'extension de fichier .cc reste un choix populaire pour les fichiers de code source C++ et elle est reconnue par de nombreux outils et plates-formes de développement populaires.

## Comment ouvrir le fichier CC?

De nombreux éditeurs de code source, y compris des outils multiplateformes tels que Microsoft Visual Studio Code et Code Blocks, peuvent être utilisés pour ouvrir et modifier des fichiers CC. Ces éditeurs offrent des fonctionnalités utiles telles que la coloration syntaxique et la saisie semi-automatique, ce qui en fait un excellent choix pour travailler avec des fichiers CC.

Vous pouvez également ouvrir les fichiers CC à l'aide de n'importe quel éditeur de texte, par exemple Notepad, Notepad++, etc.

## Compiler CC avec des compilateurs C++

La compilation du fichier CC avec le compilateur C++ implique plusieurs étapes. La première étape consiste à installer un compilateur C++ compatible pour votre système d'exploitation et votre environnement de développement. Il existe de nombreux compilateurs différents, notamment GCC, Clang et Microsoft Visual C++.

Une fois le compilateur C++ installé, l'étape suivante consiste à ouvrir le terminal ou l'invite de commande et à accéder au répertoire où se trouve le fichier CC. À partir de là, vous pouvez exécuter le compilateur en exécutant un outil de ligne de commande fourni par le compilateur. La commande spécifique pour compiler le fichier CC peut varier en fonction du compilateur que vous utilisez. À titre d'exemple, pour compiler un fichier CC à l'aide de GCC, vous exécuterez une commande similaire à :

```
g++ -o myprogram myfile.cc
```
