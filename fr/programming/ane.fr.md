## Qu'est-ce qu'un fichier ANE ?

Un fichier ANE est un fichier Adobe Air Native Extension utilisé par les développeurs pour écrire des fonctions spécifiques qui appellent du code [Adobe ActionScript](https://en.wikipedia.org/wiki/ActionScript). Dans la plupart des cas, les fichiers ANE contiennent la bibliothèque de code natif utilisée par l'extension. Cependant, le fichier lui-même est installé sur l'appareil. Il permet des fonctions spécifiques au système telles que les vibrations de l'appareil. Les applications Adobe AIR doivent être compilées pour utiliser la liaison dynamique pour utiliser le fichier ANE.

## Format de fichier ANE - Plus d'informations

Les fichiers ANE sont enregistrés sur le disque sous forme de fichiers binaires dont les spécifications de format de fichier ne sont pas disponibles publiquement. Un fichier ANE peut contenir du code source Android Java natif pour accéder aux fonctionnalités de l'appareil Android telles que l'appel des fonctions de volume ou de vibration. Tout cela est fait en enveloppant le système natif dans le langage ActionScript d'Adobe.
