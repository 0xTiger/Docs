Les fichiers de programmation sont des fichiers de code source utilisés par les langages de programmation pour écrire des applications logicielles afin d'effectuer certaines tâches. Chaque langage de programmation a sa propre syntaxe et porte son propre ensemble de fichiers pour écrire du code sous la forme de bibliothèques, de fichiers d'en-tête, de classes ou de constantes. En fonction du langage de programmation utilisé, le format du fichier de code source sera différent pour obéir à la règle de syntaxe définie par ce langage.

Les extensions de fichier de développement ou de programmation courantes et leurs formats de fichier associés incluent **[C#](/fr/programming/cs/)** (fichier de code source CSharp), **[CPP](/fr/programming/cpp/)** (fichier source C++ Fichier) et **[VB](/fr/programming/vb/)** (fichier de code source Visual Basic).
