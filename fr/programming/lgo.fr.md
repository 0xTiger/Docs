## Qu'est-ce qu'un fichier LGO ?

Un fichier LGO un code de programme écrit avec l'application Logo pour dessiner des images. Il contient des instructions programmées qui guident l'application de lecture pour réaliser des dessins selon ces instructions. Par exemple, le programme "tortue" qui peut ouvrir les fichiers LGO, utilisez ces instructions pour tracer des lignes selon le code programmé. Le dessin n'est pas limité aux seules lignes. Les fichiers LGO peuvent être utilisés pour dessiner des conceptions simples à complexes. De nombreux programmes éducatifs utilisaient des fichiers LGO pour apprendre à créer des dessins par programmation.

## Format de fichier LGO

Les fichiers LGO sont enregistrés dans un format de fichier propriétaire et leur structure de fichier interne n'est pas connue publiquement. Un autre format de fichier bien connu pour les logos est le fichier [.ico](/fr/image/ico/).
