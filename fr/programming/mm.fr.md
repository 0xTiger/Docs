## Qu'est-ce qu'un fichier MM ?

Un fichier MM est un fichier de code source qui peut contenir à la fois du code de programmation Objective-C et Objective-C++. Il est utilisé sur MacOS et se différencie des fichiers .M qui contiennent uniquement du code Objective-C. Lorsque le compilateur voit l'extension .mm, il est capable de compiler les classes C++ présentes dans le fichier. Étant donné qu'un seul fichier de code source peut contenir un mélange de langages [C](/fr/programming/c/) et [C++](/fr/programming/cpp/), renommer les fichiers .m en extension .mm permet aux fonctionnalités C++ de définir in. Les applications qui peuvent ouvrir le fichier .MM incluent Apple TextEdit, Microsoft Notepad, Notepad++ et Microsoft Wordpad.
