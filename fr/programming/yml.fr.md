## Qu'est-ce qu'un fichier YML ?

YML est un fichier texte en langage de sérialisation de données, [YAML](/fr/programming/yaml/) (Yet Another Markup Language), qui est souvent utilisé pour créer des fichiers de configuration. Il peut fonctionner en combinaison avec n'importe quel langage de programmation, y compris C/C++, Ruby, Python, Java, PHP, C# et autres. Indépendant de tout langage de programmation spécifique, il est ouvert, interopérable et lisible par l'homme. Les fichiers YML peuvent être ouverts et modifiés avec des éditeurs de texte courants tels que Microsoft Notepad, Notepad++, Github Atom, Apple TextEdit et Visual Studio IDE.

## Format de fichier YML - Plus d'informations

Les fichiers YML stockent les paramètres de l'application en tant que configuration. Sa structure de base est une carte qui peut également être appelée répertoire ou hachage. Les entrées d'un fichier YML consistent en une paire clé/valeur. Généralement, les fichiers de configuration .yml sont enregistrés sous `configure.yml`, bien que certains langages de programmation tels que Ruby on Rails enregistrent la configuration de la connectivité de la base de données dans les fichiers `database.yml`. [Spécifications du format de fichier YML/YAML](https://yaml.org/spec/1.2.2/) expliquent les structures complètes du langage et les informations relatives à la syntaxe.
