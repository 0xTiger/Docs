## Qu'est-ce qu'un fichier ML ?

Un fichier ML est un fichier texte créé avec le langage de programmation fonctionnel à usage général ML (souvent appelé Meta Language). Il contient du code source ML qui est plus souvent utilisé dans la recherche et la conception de langages de programmation en utilisant une sémantique complètement définie. Il existe plusieurs variantes de ML avec Caml comme celui largement utilisé. D'autres incluent Objective Caml (OCaml), JoCaml, Extended ML et MacroML. Le langage ML est couramment utilisé pour l'écriture de compilateurs, la démonstration automatisée de théorèmes et les vérifications formelles.
