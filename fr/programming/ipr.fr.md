## Qu'est-ce qu'un dossier IPR ?

Le fichier IPR est un fichier de projet créé avec Java IDE **IntelliJ IDEA**. Toutes les informations relatives aux paramètres du projet telles que la structure des répertoires, les fichiers de code source, les bibliothèques incluses dans le projet et les paramètres du compilateur sont stockées dans les fichiers IPR. Lorsqu'un projet est ouvert depuis IntelliJ IDEA, l'IDE lit les informations dans le fichier IPR et charge les fichiers et paramètres correspondants. Les fichiers IPR ont été remplacés par les fichiers .IDEA qui contiennent toutes les informations sous forme de répertoire de paramètres.

## Format de fichier DPI

Les fichiers IPR sont enregistrés dans le format de fichier [XML](/fr/web/xml/) universellement compréhensible et lisible par l'homme. Il contient toutes les informations liées au projet dans des balises qui sont chargées pour afficher le contenu lié au projet.

Étant donné que les fichiers IPR sont stockés au format de fichier XML, ils peuvent être ouverts dans n'importe quel éditeur de texte tel que Notepad, Notepad ++, Atom et Apple TextEdit.
