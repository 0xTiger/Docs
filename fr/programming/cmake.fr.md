## Qu'est-ce qu'un fichier CMAKE ?

Un fichier CMAKE est un fichier de projet créé par l'outil open source et multiplateforme **CMake**. Il contient le script source dans le langage CMake pour l'ensemble du programme. L'outil CMake utilise des scripts de programmation, connus sous le nom de CMakeLists, pour générer des fichiers de construction pour un environnement spécifique tel que les makefiles de la machine Unix. Les scripts et les modules utilisés par le projet CMake sont organisés en script.cmake et module.cmake qui sont référencés dans le projet.
