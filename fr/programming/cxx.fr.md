## Qu'est-ce qu'un fichier CXX ?

Un fichier CXX est un fichier de code source, similaire à C++, pour écrire des applications. Il peut contenir l'implémentation complète d'une application entière ou peut avoir une implémentation partielle du programme qui peut être incluse dans un projet avec d'autres fichiers C++. Les fichiers CXX peuvent être ouverts dans n'importe quel éditeur de texte et IDE de programmation populaires tels que Notepad ++, Turbo C ++ et Visual Studio,

## Format de fichier CXX

Les fichiers CXX sont stockés sous forme de fichier texte brut et contiennent du code écrit en syntaxe C++. Les fichiers CXX sont également généralement trouvés avec l'extension [.cpp](/fr/programming/cpp/). Les fichiers CXX sont considérés comme une alternative au .cpp utilisé par certains compilateurs.
