## Qu'est-ce qu'un fichier ASI ? ##

Un fichier ASI est un format de fichier variable utilisé par le logiciel Alpha Five qui est utilisé pour développer des applications Web. Il stocke les variables utilisées par l'application Web. Lorsque vous créez un nouveau projet de développeur dans Alpha Five, le fichier ASI est également créé et enregistré avec le projet de développeur Alpha Five.

Les fichiers ASI peuvent être ouverts avec Alpha Software Alpha Anywhere.

## Format de fichier ASI

Les fichiers ASI sont enregistrés au format de fichier propriétaire Alpha Five. Ils sont lus et écrits par l'application dans le cadre du projet du développeur.
