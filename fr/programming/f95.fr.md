## Qu'est-ce qu'un fichier F95 ?

Un fichier avec l'extension « .f95 » indique généralement qu'il s'agit d'un fichier de code source Fortran 95. Fortran est un langage de programmation impératif à usage général, particulièrement adapté au calcul numérique et scientifique. Fortran 95 est une version améliorée de Fortran 90, introduisant plusieurs nouvelles fonctionnalités et améliorations.

Les fichiers de code source Fortran contiennent généralement des instructions de programme écrites en langage de programmation Fortran ; si vous avez un fichier avec l'extension « .f95 », il est probablement destiné à être compilé à l'aide d'un compilateur Fortran ; la compilation est le processus de traduction d'un code source lisible par l'homme en code binaire lisible par machine qui peut être exécuté par un ordinateur.

## Comment ouvrir le fichier F95 ?

Si vous souhaitez travailler avec ou exécuter un programme dans un fichier « .f95 », vous aurez besoin d'un compilateur Fortran ; les compilateurs Fortran populaires incluent GNU Fortran (gfortran), Intel Fortran Compiler (ifort) et d'autres. Vous pouvez compiler et exécuter des programmes Fortran à l'aide des commandes appropriées fournies par le compilateur.

Par exemple, si gfortran est installé, vous pouvez utiliser les commandes suivantes dans le terminal pour compiler et exécuter un programme Fortran :

```
# Compile the Fortran program
gfortran -o my_program my_program.f95

# Run the compiled program
./my_program
```

Si vous souhaitez uniquement afficher le contenu du fichier F95, il s'agit d'un fichier texte brut, vous pouvez donc l'ouvrir avec n'importe quel éditeur de texte.
