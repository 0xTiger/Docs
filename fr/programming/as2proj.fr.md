## Qu'est-ce qu'un fichier AS2PROJ ?

Un fichier AS2PROJ est un fichier de projet créé par l'IDE de programmation Flash open source, FlashDevelop. Il contient des informations relatives au projet telles que les paramètres de construction, les références aux fichiers source du projet et les options de génération de la sortie [fichier .swf](/fr/page-description-language/swf/). Chaque nouveau projet créé dans FlashDevelop est enregistré en tant que fichier AS2PROJ. Vous pouvez modifier les paramètres de votre projet et toutes ces modifications sont enregistrées dans le fichier AS2PROJ.

Les fichiers AS2PROJ peuvent être ouverts avec FlashDevelop.

## Format de fichier AS2PROJ

Un fichier AS2PROJ est enregistré au format de fichier [XML](/fr/web/xml/) qui est un format de fichier universel pour l'échange d'informations. Il peut être ouvert dans n'importe quel éditeur de texte tel que Notepad, Notepad++ et Atom pour examiner le contenu.
