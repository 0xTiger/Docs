## Qu'est-ce qu'un fichier SCRIPT ?

Un fichier SCRIPT est un fichier source qui contient un script dans n'importe quel langage de script. Cela en fait un fichier de script généralisé qui contient des variables et des fonctions. Étant donné que chaque langage de script peut être différent, le script écrit dans un fichier SCRIPT doit être exécuté à l'aide du moteur de script approprié.

## Format de fichier SCRIPT - Plus d'informations

Les fichiers SCRIPT sont stockés sur disque dans des fichiers texte brut. Ceux-ci peuvent être ouverts et modifiés avec n'importe quel éditeur de texte tel que Microsoft Notepad, Notepad ++ et Apple TextEdit. Étant donné que le langage de script peut être différent, il peut être exécuté en renommant l'extension de fichier avec l'extension par défaut du langage de script respectif. Par exemple, il existe plusieurs langages de script courants tels que JavaScript ([.JS](/fr/web/js/)), VBScript ([.VBS](/fr/programming/vbs/)), AppleScript (.SCPT), PHP ([ .PHP](/fr/programming/php/)) et ASP ([.ASP](/fr/web/asp/)).
