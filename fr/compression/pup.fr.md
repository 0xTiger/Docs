## Qu'est-ce qu'un fichier PUP Linux ?

Un fichier Linux PUP est un package d'installation utilisé par le système d'exploitation Puppy Linux. Il s'exécute en tant qu'installateur et commence à installer les fichiers du package. Sous Linux, il est souvent appelé fichier DotPup et est similaire aux fichiers d'installation [.exe](/fr/executable/exe/) et [.msi](/fr/executable/msi/) utilisés sur le système d'exploitation Windows. Les nouvelles versions de Puppy Linux utilisent le format de fichier .pet au lieu du format de fichier PUP.
