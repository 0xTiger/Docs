## Qu'est-ce qu'un fichier WUX ?

Un fichier WUX est la version compressée d'un fichier Wii U Disk Image [WUD](/fr/disc-and-media/wud/). Il est généré à l'aide de l'outil de compression d'image Wii U pour réduire la taille de fichier des fichiers segmentés WUD qui, sinon, font environ 20 Go. Ces fichiers d'informations de jeu de grande taille sont réduits à environ 2 Go dans des fichiers WUX compressés, ce qui facilite le transfert de fichiers. Les fichiers WUX peuvent être ouverts et lus avec des émulateurs de jeux vidéo tels que [Cemu](https://cemu.info/).

## Format de fichier WUX

Les fichiers WUX sont enregistrés sous forme de fichiers compressés qui peuvent être décompressés pour être chargés dans des émulateurs à l'aide de l'outil de compression d'image Wii U. L'émulateur de jeu vidéo Cemu peut également charger les fichiers WUX et charger les informations complètes du jeu vidéo.
