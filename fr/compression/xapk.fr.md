## Qu'est-ce qu'un fichier XAPK ?

Un fichier avec l'extension .xapk est un fichier de package compressé utilisé pour installer des applications Android sur des appareils mobiles. Il s'agit d'un format de fichier conteneur qui intègre APK et des fichiers associés supplémentaires requis pour l'installation. L'autre fichier associé est un fichier OBB qui stocke des fichiers supplémentaires tels que des graphiques, des fichiers multimédias et des données d'application nécessaires au fonctionnement de l'application. Les fichiers XAPK ne sont pas pris en charge par Google Play et sont utilisés uniquement pour la distribution sur des sites Web tiers de téléchargement d'applications Android. Ceux-ci peuvent être installés sur un appareil Android à l'aide de XAPK Installer.

## Format de fichier XAPK

Les fichiers XAPK sont compressés à l'aide du format de fichier standard [ZIP](/fr/compression/zip/). Ceux-ci peuvent être extraits à l'aide d'un logiciel standard de compression/décompression tel que WinZIP. Une fois le fichier XAPK extrait sur le disque, il contient les fichiers suivants dans le dossier.

* APK - Fichier d'installation standard pour installer l'application sur les appareils Android
* OBB - Fichier supplémentaire contenant les fichiers de ressources pertinents
