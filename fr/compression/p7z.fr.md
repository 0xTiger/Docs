## Qu'est-ce qu'un fichier P7Z ?

Un fichier P7Z est un fichier compressé qui contient un message électronique dans la norme de cryptage S/MIME. Il a été introduit par Microsoft Corporation et est classé dans la catégorie des fichiers de données liés aux e-mails. Les fichiers P7Z ont le type de média application/pkcs7-mime. Il contient une ou plusieurs parties d'un message compressé avec l'algorithme de compression zlib.

## Format de fichier P7Z - Plus d'informations

Les détails du format de fichier interne du format de fichier P7Z ne sont pas connus. Certaines API telles que [Aspose.Email](https://products.aspose.com/email/) vous permettent de lire la partie S/MIME des messages. Si un client de messagerie n'a pas la capacité de gérer les données compressées, les messages S/MIME compressés sont envoyés en pièce jointe.
