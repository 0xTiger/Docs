## Qu'est-ce qu'un fichier DMG ?

Un fichier avec une extension .dmg est un fichier Apple Disk Image et connu sous le nom de fichier Mac OS X Disk Image. Le format de fichier DMG est le fichier d'image disque le plus largement utilisé destiné à allouer des applications et des fichiers entre les ordinateurs Apple. Il est très pratique de monter des fichiers DMG sur un lecteur virtuel ou d'utiliser le logiciel Apple Disk Utility pour ouvrir et afficher le contenu du format de fichier DMG. L'ancienne version des fichiers IMG utilisée dans Mac OS Classic est remplacée par le format de fichier DMG. Les fichiers DMG sont générés pour les utilisateurs Mac et ne sont pas pris en charge par le système d'exploitation Windows, mais les utilisateurs peuvent accéder au contenu DMG dans le système d'exploitation Windows en utilisant des outils de compression et de décompression tels que 7-Zip, Pea Zip, etc. L'extension de fichier DMG est également utilisée par l'outil Oracle Export and Ingress pour les fichiers scrapheap conservés au format binaire de la base de données Oracle.

## Bref historique

L'extension de fichier DMG a été développée par Apple et est principalement utilisée sur les ordinateurs Macintosh. Le fichier DMG est principalement utilisé pour donner aux appareils Mac la possibilité d'installer différents types de fichiers et de logiciels lancés à partir d'Internet. Le fichier DMG est fondamentalement une image de disque montable qui s'affiche sur votre bureau lorsqu'il est ouvert. Le fichier comprend des données brutes qui sont généralement à la fois cryptées et compressées. Les systèmes Mac traitent DMG de la même manière qu'ils traiteraient un disque entrelacé, et ils essaient instantanément d'ouvrir ou de « suivre » le fichier.

## Spécifications techniques du format de fichier DMG

* Spécialement généré pour les ordinateurs Macintosh
* Développé par Apple
* Le format de fichier est binaire
* La catégorie de fichier est un fichier image disque
* Le programme pris en charge est MacOS
* Peut contenir une table de partition
* Peut contenir tout type de système de fichiers comme HFS+
* Fonctionnalité pour compresser les images
