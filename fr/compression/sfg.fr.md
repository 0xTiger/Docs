## Qu'est-ce qu'un fichier SFG ?

Un fichier SFG est le format de fichier utilisé par Synfig Studio pour enregistrer les fichiers de projet. Synfig Studio est un logiciel d'animation 2D gratuit et open-source qui permet aux utilisateurs de créer des animations de haute qualité à l'aide de graphiques bitmap et vectoriels. Le fichier SFG contient toutes les informations nécessaires pour recréer une animation, y compris les calques, la synchronisation et les images clés de chaque calque. Il comprend également toutes les images ou ressources utilisées dans le projet, ainsi que toutes les pistes audio ou autres médias.

## Format de fichier SFG

Les fichiers SFG sont enregistrés sur le disque au format de fichier ZIP compressé. Les autres formats de fichiers pris en charge par [Synfig Studio](https://www.synfig.org/) Compressed Project incluent [SIFZ](/fr/compression/sifz/) et SIF.
