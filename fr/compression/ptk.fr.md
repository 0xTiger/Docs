## Qu'est-ce qu'un fichier PTK ? ##

Les fichiers PTK sont considérés comme des fichiers de données utilisés par Intuit Quicken. Il s'agit d'une application de gestion des finances personnelles qui aide l'extension de fichier .ptk à fonctionner correctement. Il a différentes fonctionnalités telles que l'importation de données directement depuis les banques participantes, des rapports sur toutes les données financières et le téléchargement quotidien des cours des actions et des fonds. Le format de fichier PTK est le type de fichier utilisé par Quicken pour enregistrer les informations financières, qui est maintenant remplacé par le format GIF. L'objectif principal des fichiers PTK était d'importer des données à partir de ressources en ligne, telles que des banques et des institutions financières, dans la base de données Quickens.
