## Qu'est-ce qu'un fichier MINT ?

Un fichier MINT est un fichier de package d'installation utilisé par la distribution [Linux Mint](https://linuxmint.com/). Il contient des fichiers de code source compressés qui sont extraits et utilisés pour démarrer l'installation du système d'exploitation Linux Mint. Le système d'exploitation Linux Mint est doté d'une interface utilisateur graphique (GUI) facile à utiliser et a été le premier à avoir une conception graphique élégante. Les fichiers MINT sont décompressés et stockés dans un répertoire temporaire avec le package d'installation. L'installation est exécutée à partir du même répertoire pour l'installation du système d'exploitation Linux Mint.

## Format de fichier Linux MINT

Les détails du format de fichier interne du format de fichier MINT ne sont pas disponibles publiquement. Cependant, les fichiers d'installation compressés stockés dans les fichiers MINT sont conformes aux normes de compression intégrées aux programmes d'installation du système d'exploitation Linux Mint. Cela permet à ces fichiers de se décompresser et de faire partie de l'installation.
