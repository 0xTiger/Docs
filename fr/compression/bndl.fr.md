## Qu'est-ce qu'un fichier BNDL ?

Un fichier BNDL est un fichier d'archive de données de jeu utilisé par le jeu **Need for Speed : Most Wanted** de [Electronic Arts](https://www.ea.com/). Il s'agit d'un fichier bundle qui stocke les sons du jeu et les flux audio à utiliser pour les effets sonores du jeu. Ces effets sonores sont utilisés lorsque le joueur joue au jeu. BNDL est l'abréviation de bundle.

## Format de fichier BNDL

BNDL est un format de fichier propriétaire utilisé par le jeu Need for Speed. Il s'agit d'un format de fichier propriétaire et les détails de son format de fichier interne ne sont pas disponibles pour référence publique ou développeurs. Par conséquent, les fichiers BNDL ne peuvent être ouverts par aucun autre logiciel.
