## Qu'est-ce qu'un fichier VGA ?

Un fichier VGA est un fichier de lecteur Windows destiné à prendre en charge les moniteurs VGA. Il était utilisé dans les premières versions de Microsoft Windows pour le réglage des moniteurs VGA au niveau du système d'exploitation (OS). Un fichier VGA utilisé pour permettre aux utilisateurs d'ajuster la résolution et la profondeur de couleur de l'affichage à l'aide du panneau de configuration du moniteur.

## Format de fichier VGA

Le format de fichier interne du format de fichier VGA n'est pas connu pour référence du développeur. Il est référencé par le système d'exploitation Windows pour définir l'apparence visuelle du moniteur VGA.
