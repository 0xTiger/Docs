## Qu'est-ce qu'un fichier MPP ?

Un fichier avec une extension .mpp est un fichier de données Microsoft Project qui stocke les informations relatives à la gestion de projet de manière intégrée. Microsoft a développé ce format de fichier propriétaire pour le rendre compatible avec Microsoft Project (MSP), leur produit logiciel de gestion de projet. Autre que MPP, MSP peut fonctionner avec le schéma XML du projet. De nombreuses applications et API permettent de convertir les MPP en d'autres formats de fichiers. Microsoft dispose désormais d'un serveur de projet en ligne sur lequel les fichiers de gestion de projet peuvent être téléchargés pour une collaboration entre plusieurs utilisateurs.

## Format de fichier MPP ##

Microsoft a développé et publié MPP en tant que format de fichier binaire et ses spécifications n'ont pas été rendues publiques par Microsoft à ce jour. La structure et les détails des fichiers internes ne sont donc pas disponibles pour les développeurs. Il existe plusieurs API disponibles en ligne qui prennent en charge la lecture/écriture dans les fichiers MPP, mais une précision de 100 % n'est jamais atteinte. Microsoft a introduit plusieurs fonctionnalités dans son application de gestion de projet au fil du temps, tout comme les fonctionnalités du format de fichier MPP.

Le fichier MPP utilise les types MIME suivants :

* application/vnd.ms-projet
* application/msproj
* application/msproject
* application/x-msproject
* application/x-ms-project
* application/x-dos_ms_project
* demande/mpp
* zz-application/zz-winassoc-mpp

## Des problèmes pour ouvrir un fichier MPP ? ##

Voici la liste de certains problèmes courants qui peuvent survenir et entraîner un dysfonctionnement du format MPP :

* Absence de logiciel de support
* Fichier corrompu
* Fichier infecté en raison d'un virus
* Aucun droit d'accès dans le système pour ouvrir les fichiers
* Lecteur obsolète dans votre système
* L'extension du fichier est renommée
