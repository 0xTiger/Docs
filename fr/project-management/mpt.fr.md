## Qu'est-ce qu'un fichier MPT ? ##

Un fichier avec l'extension .mpt est un fichier de modèle [Microsoft Project](https://products.office.com/en-us/project/project-and-portfolio-management-software). Il contient des informations de base et des paramètres de structure et de document pour la création de fichiers [MPP](/fr/project-management/mpp/). Ces modèles fournissent des paramètres par défaut qui sont essentiels pour un projet particulier, tels que les calendriers ou les informations budgétaires. Cependant, il ne peut pas enregistrer les données spécifiques au projet telles que les tâches, les ressources ou les affectations. Une fois modifiés, les fichiers de modèle peuvent être enregistrés en tant que fichiers MPP standard pour continuer à travailler avec. Un fichier de modèle principal, intitulé "Global.mpt" est disponible dans Microsoft Project, ouvert pour créer un modèle individuel. Les fichiers MPT utilisent *application/vnd.ms-project* et *application/x-project* comme types MIME standard. Windows, Linux et macOS peuvent tous ouvrir des fichiers MPT, ainsi que Moos Project Viewer, Microsoft Project et MPXJ.

## Des problèmes pour ouvrir un fichier MPT ? ##

Voici la liste de certains problèmes courants qui peuvent survenir et entraîner un dysfonctionnement du format MPT :

* Absence de logiciel de support
* Fichier corrompu
* Fichier infecté à cause d'un virus
* Aucun droit d'accès dans le système pour ouvrir les fichiers
* Lecteur obsolète dans votre système
 *   Extension of the file is renamed
