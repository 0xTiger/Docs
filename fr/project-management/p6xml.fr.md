## Qu'est-ce qu'un fichier P6XML ? ##

L'extension de fichier P6XML a une excellente lisibilité, une adaptabilité et une large utilisation qui en font le format idéal. Primavera P6 Professional a la fonctionnalité d'exporter et d'importer des fichiers [XER](/fr/project-management/xer/) et des fichiers XML. XML signifie **Extensible Markup Language**. Ce format général permet aux utilisateurs de partager des informations sur le projet entre les bases de données Primavera P6. Les données du projet peuvent facilement être stockées dans un fichier texte via le fichier XML ; cela simplifie les choses pour que de nombreux logiciels échangent les données. Ces données peuvent être facilement partagées et gérées par l'utilisateur Primavera P6.

## Avantages d'un format de fichier P6XML ##

* Il est facile pour les utilisateurs de Primavera de définir quelles lignes de base sont exportées et importées à l'aide du fichier XML
* L'exportation XML peut contenir tous les plans d'action détaillés du projet
* Tout utilisateur utilisant Primavera 6 qui ne peut pas modifier, ajouter ou supprimer certains éléments de données globales, tels que les codes d'activité, les calendriers et les ressources, ne pourra pas les importer.
* En plus des paramètres de profil de sécurité général et de projet utilisés par Primavera, les éléments de données qui affecteront ou modifieront les données de la base de données ne seront pas importés

## Exportation du fichier XML depuis Primavera P6 ##

Pour exporter des projets P6 Professional et des données de projet vers un fichier P6XML, utilisez Oracle Primavera Cloud. Vous pouvez télécharger le fichier lors de sa formation à partir de l'application de bureau P6 Professional et l'importer dans Primavera Cloud. Avant d'exécuter le processus d'exportation sans interruption, vous devez régler si vous êtes lié à une base de données P6 Professional ou à un P6 EPPM. Certaines fonctionnalités du processus d'exportation peuvent différer en fonction de votre base de données. Ayant des paramètres P6 Specialized avec une base de données P6 Proficient, le processus d'exportation fonctionne localement sur votre système. Avec une base de données P6 Professional, vous pouvez facilement exporter et ouvrir le fichier XML P6 à partir d'un paramètre de votre système.

Voici quelques étapes décrivant le processus d'exportation :

* Projet Open desires Primavera P6
* Après cela, sélectionnez Exporter dans l'onglet Fichier
* Sélectionnez "Primavera P6 – (XML)". Après cela, cliquez sur Suivant
* Double-cliquez sur le projet que vous allez exporter
* Sélectionnez les troncatures du chemin où vous souhaitez placer le fichier exporté et sélectionnez "Terminer".
* Sélectionnez l'option selon laquelle vous souhaitez enregistrer le fichier
* Si les mises en page au niveau du projet ne sont pas nécessaires avec le fichier d'exportation, il est essentiel de décocher la sélection sur "Exporter toutes les mises en page au niveau du projet".
* Sélectionnez terminer, puis vous recevrez une confirmation de l'exportation XML

### Conseils importants ###

* Chaque fois qu'il se connecte à une base de données P6 EPPM, l'administrateur doit transmettre l'URL de la source de contenu dans le champ URL P6 sur la page Général des paramètres de l'application dans P6
* Il est pratique d'exporter de nombreux projets vers un seul fichier XML
* Les données globales affectées au projet sont exportées
* Il est pratique d'exporter des fichiers XML sans avoir accès à toutes les ressources



## Des problèmes pour ouvrir un fichier P6XML ? ##

Certains problèmes courants pouvant survenir et entraîner un dysfonctionnement du format P6XML sont les suivants :

* Absence de logiciel de support
* Fichier corrompu
* Fichier infecté en raison d'un virus
* Description P6XML supprimée accidentellement du registre Windows
* Aucun droit d'accès dans le système pour ouvrir les fichiers
* Lecteur obsolète dans votre système
* L'extension du fichier est renommée
