## Qu'est-ce qu'un fichier BTAPP ?

Un fichier BTAPP est une application qui a été développée pour le programme de téléchargement TORRENT uTorrent. Il s'agit essentiellement d'une application pour uTorrent qui peut accéder à toutes les fonctionnalités du client uTorrent. Ces applications peuvent être ajoutées au client uTorrent depuis l'application. Cela installera l'application dans le client uTorrent et fournira toutes les fonctionnalités de téléchargement de fichiers [TORRENT](/fr/misc/torrent/).

## Format de fichier BTAPP - Plus d'informations

Un fichier d'application BTAPP est enregistré en tant que fichier binaire. Il contient du code [HTML](/fr/web/html/) et [JavaScript](/fr/web/js/) qui peut être ouvert dans un navigateur intégré.
