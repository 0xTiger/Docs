## Qu'est-ce qu'un fichier XEX ?

Un fichier XEX est un format de fichier exécutable utilisé pour les applications de la console de jeu vidéo XBox 360. Xbox, le prédécesseur de Xbox 360, utilise le format de fichier **XBE** pour les applications. Les fichiers XEX sont destinés à fonctionner uniquement sur Xbox 360 et ne s'exécutent pas sur PC. Les fichiers XEX par défaut sur la console XBox 360 sont nommés **default.xex** qui est chargé de contrôler et de télécharger les mises à jour logicielles pour l'appareil. Cela garantit la rétrocompatibilité avec la console XBox.

## Format de fichier XEX

Les fichiers XEX sont enregistrés sur le disque sous forme de fichiers binaires et peuvent être exécutés sur une console XBox 360. Ceux-ci peuvent également être ouverts et examinés à l'aide du logiciel XEXTool qui peut extraire des informations de ces fichiers. XEXTool peut également être utilisé pour modifier ces fichiers.
