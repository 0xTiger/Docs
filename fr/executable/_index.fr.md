Les fichiers exécutables sont utilisés pour demander à un ordinateur d'effectuer une ou plusieurs tâches ou opérations spécifiques en tant qu'ensemble d'instructions codées ; écrites dans ces fichiers. Le terme "Instructions" désigne généralement des instructions de **code machine** pour un **CPU** physique. Les fichiers exécutables peuvent être codés à la main en langage machine, bien qu'il soit très pratique de développer un logiciel sous forme de code source dans un langage de haut niveau facilement compréhensible par les humains. Parfois, le code source peut être écrit en langage assembleur à la place, reste étroitement associé à des instructions de code machine qui sont facilement lisibles par l'homme.

Certains formats de fichiers de disque populaires sont [BAT](/fr/executable/bat/), [CGI](/fr/executable/cgi) et [COM](/fr/executable/com/)
