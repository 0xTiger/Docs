## Qu'est-ce qu'un fichier SDF ?

Un fichier de données spatiales (SDF) est un format de fichier de géodatabase développé par Autodesk. Il stocke les données géospatiales et est utilisé par le programme Autodesk GIS MapGuide et AutoCAD Map 3D. Le format de fichier SDF est basé sur le format de fichier de base de données à fichier unique SQLite3. Il est considéré comme un format de fichier optimisé pour les grands ensembles de données d'informations spatiales.

Vous pouvez ouvrir un fichier SDF à l'aide d'Autodesk AutoCAD Map 3D 2022 et d'Autodesk AutoCAD Civil 3D 2023.

## Format de fichier SDF - Plus d'informations

Le format de fichier SDF est stocké sur le disque sous forme de fichiers binaires. Il est basé sur la base de données SQLite qui utilise des composants de stockage de bas niveau pour la sérialisation binaire des données. Un fichier SDF contient plusieurs classes d'entités par fichier et plusieurs propriétés géométriques pour chaque classe d'entités. Les données d'un fichier SDF sont accessibles à grande vitesse grâce à l'indexation de chaque propriété géométrique à l'aide d'un arbre R.
