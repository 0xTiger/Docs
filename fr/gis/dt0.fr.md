## Qu'est-ce qu'un fichier DT0 ?

Un fichier DT0 est un fichier de données numériques d'élévation du terrain (DTED) qui contient des informations de niveau 0 pour étudier les données d'élévation du terrain. Les données d'altitude sont uniformément espacées dans un fichier DT0 et peuvent être lues par des programmes SIG populaires tels que ESRI ArcGIS Pro, TatukGIS, Blue Marble Geographic Global Mapper et Pitney Bowes MapInfo. Les fichiers DT0 sont principalement utilisés par les communautés scientifiques et techniques pour étudier la pente, l'élévation et la rugosité de surface des terrains mondiaux.

Le format de fichier DT0 est similaire aux types de fichiers [DT1](/fr/gis/dt1/) et [DT2](/fr/gis/dt2/) mais avec moins de granularité.

### Format de fichier DT0

Les fichiers DT0 sont enregistrés sous forme de fichiers binaires sur le disque. Ceux-ci peuvent être lus à l'aide de nombreuses applications SIG. Les fichiers DT0 peuvent être fusionnés dans un catalogue d'images de géodatabase fichier qui peut être utilisé pour créer une grande mosaïque.

## Comment convertir des fichiers raster en DTED ?

Il existe plusieurs outils qui peuvent convertir des fichiers Raster en DTED. [ArgGIS Pro](https://pro.arcgis.com/en/pro-app/latest/tool-reference/data-management/raster-to-dted.htm) est l'un de ces outils qui peut convertir Raster en DTED .
