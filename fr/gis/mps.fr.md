## Qu'est-ce qu'un fichier MPS ?

Un fichier MPS est un fichier de carte utilisé sur les appareils portables Pocket PC. Il contient des informations géographiques sur une certaine zone telle qu'une ville ou un état. Il peut être chargé dans des appareils Pocket PC pour la navigation sans avoir besoin de récupérer des données LIVE à partir de n'importe quel serveur de données géospatiales. Le format de fichier MPS a été développé par Microsoft pour les appareils portables basés sur Windows et a été utilisé dans son application MapPoint GIS qui a été abandonnée il y a longtemps maintenant.

## Format de fichier MPS

La structure de fichier interne du format de fichier MPS n'est pas connue car elle a été interrompue depuis un certain temps maintenant avec MapPoint. Certains autres formats de fichiers cartographiques SIG populaires incluent [ESRI Shape Files](/fr/gis/shp/), [KML](/fr/gis/kml/) et [KMZ](/fr/gis/kmz/).
