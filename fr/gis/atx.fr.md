## Qu'est-ce qu'un fichier ATX ?

Un fichier avec l'extension .atx est un fichier ESRI ArcGIS qui stocke les index d'attributs ArcGIS 8. Il existe un fichier ATX distinct pour chaque [shapefile](/fr/gis/shp/) ou index d'attribut dBase créé dans ArcCatalog. Il convient de noter que les index d'attributs ArcView GIS 3.x pour les fichiers de formes et les fichiers dBase ne sont pas utilisés par ArcMap ou ArcCatalog. Cependant, ces index d'attributs ArcView GIS 3.x peuvent être convertis en index d'attributs ArcGIS 8 à l'aide d'un outil ArcObjects.

## Format de fichier ATX - Plus d'informations

Il n'y a pas de spécifications de format de fichier disponibles pour le format de fichier ATX pour faciliter les développeurs.
