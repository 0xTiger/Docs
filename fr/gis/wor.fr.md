## Qu'est-ce qu'un fichier WOR ?

Un fichier WOR est un fichier SIG utilisé par le logiciel de cartographie et d'analyse géographique MapInfo. Il contient une zone géographique qui contient une carte visuelle contenant des données sur les caractéristiques géospatiales telles que la population et les informations démographiques. Vous pouvez charger un fichier WOR dans le logiciel MapInfo pour le visualiser et y effectuer différentes opérations telles que le zoom et le panoramique.

## Format de fichier WOR

Les fichiers WOR comprennent plusieurs couches qui sont rendues dans un ordre spécifique lorsqu'elles sont chargées dans MapInfo. Les détails sur le format de fichier WOR ne sont pas disponibles publiquement pour la référence du développeur. Les fichiers WOR peuvent être convertis en QGIS avec [QGIS 3.20 Odense](https://www.qgis.org/en/site/forusers/visualchangelog320/index.html#feature-new-embedded-styling-renderer-with-ogr- fonctionnalité-styles-support).
