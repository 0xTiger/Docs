## Qu'est-ce qu'un fichier NM2 ?

Un fichier avec l'extension .nm2 est un fichier cartographique créé avec le système de navigation Navitel. Il stocke les données géospatiales liées à différentes entités cartographiques telles que les routes, les rues, les voies de circulation, les points sur la carte, les points de repère et d'autres données GPS. Les fichiers NM2 peuvent être ouverts à l'aide du logiciel Navitel sur différents types d'appareils tels que les téléphones Android, les appareils iOS, Windows Phone et les appareils Symbian. Ces applications sont disponibles gratuitement et peuvent être téléchargées et installées sur les appareils respectifs.

## Format de fichier NM2 - Plus d'informations

Les fichiers NM2 sont enregistrés en tant que fichiers binaires et leur format de fichier interne/structure de données n'est pas disponible publiquement. Les versions récentes de Navitel enregistrent les données sous forme de fichiers .nm7. Il existe plusieurs fichiers cartographiques disponibles par Navitel sur leur [site Web](https://www.navitel.cz/en/downloads/demo). La [documentation Navitel](https://www.navitel.cz/en/downloads/documentation) fournit des guides d'utilisation complets sur l'utilisation de son système de navigation pour différents types d'appareils et de véhicules.
