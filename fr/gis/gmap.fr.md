## Qu'est-ce qu'un fichier GMAP ?

Un fichier GMAP est un fichier SIG qui contient une combinaison de régions spécifiques du monde. Il a été utilisé par certains appareils Garmin. Les combinaisons de cartes peuvent comprendre différentes régions qui ne doivent pas nécessairement être contiguës. Appareils Garmin utilisés pour stocker ces fichiers cartographiques dans la mémoire intégrée. Cependant, les nouveaux appareils Garmin prennent en charge le mode de stockage de masse qui ressemble à un périphérique de stockage USB normal. Les fichiers GMAP peuvent être ouverts avec les logiciels Garmin BaseCamp et MapInstall sur les logiciels PC et Mac.

## Format de fichier GMAP - Plus d'informations

Les fichiers GMAP sont enregistrés sur le disque en tant que fichiers binaires et les détails de leur format de fichier interne ne sont pas disponibles publiquement.

## Types de fichiers courants utilisés sur les appareils Garmin

* Format d'échange GPS ([.gpx](/fr/gis/gpx/))
* Format de fichier FITness ([.fit](/fr/gis/fit/))
* Base de données GPS Garmin® (.gdb)
* Centre de formation Garmin ([.tcx](/fr/gis/tcx/))
* Localisation GPS ([.loc](/fr/gis/loc/))
* Carte personnalisée Garmin ([.kml](/fr/gis/kml/) ou [.kmz](/fr/gis/kmz/))
* Photo (.jpg ou [.jpeg](/fr/image/jpeg/))
