## Qu'est-ce qu'un fichier DT1 ?

Un fichier DT1 est un fichier de données numériques d'élévation du terrain (DTED) qui contient des données de terrain uniformément espacées. Il est principalement utilisé par les applications logicielles SIG militaires pour obtenir des données d'altitude, de pente et de contour de surface. Le fichier DT1 est similaire à [DT0](/fr/gis/dt0/) et [DT2](/fr/gis/dt2/) mais avec une granularité moyenne des informations.

Les fichiers DT1 peuvent être ouverts avec des applications SIG telles que ESRI ArcGIS Pro, TatukGIS, Blue Marble Geographic Global Mapper et Pitney Bowes MapInfo.

### Format de fichier DT1

Les fichiers DT1 sont enregistrés sous forme de fichiers binaires sur disque et peuvent être lus à l'aide de nombreuses applications SIG. Ceux-ci peuvent être fusionnés dans un catalogue d'images de géodatabase fichier qui peut être utilisé pour créer une grande mosaïque.
