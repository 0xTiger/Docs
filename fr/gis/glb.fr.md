## Qu'est-ce qu'un fichier GLB ?

Un fichier avec l'extension .glb est un fichier partageable à l'échelle mondiale généré et utilisé par le logiciel AGI Systems Tool Kit (STK). Il contient des informations de globe 3D qui peuvent être partagées avec d'autres utilisateurs pour être utilisées dans leurs projets. Le logiciel STK d'AGI systems est utilisé pour simuler des missions aérospatiales, des systèmes de défense et des systèmes de télécommunication. GLB est le format de fichier standard pour STK et de nombreux modèles 3D différents tels que [.gltf](/fr/3d/gltf/) peuvent être convertis dans ce format. Les fichiers GLB peuvent être ouverts avec le logiciel AGI Simulation Toolkit ainsi que n'importe quel éditeur de texte.

## Format de fichier GLB - Plus d'informations

Les fichiers GLB stockent des informations sur les textures, les images et d'autres fichiers utilisés dans un globe pour un projet. Les références à l'emplacement des fichiers liés sont stockées dans les fichiers GLB et ces fichiers supplémentaires doivent également être partagés avec d'autres utilisateurs lors du partage du fichier globe GLB pour un chargement correct. Les fichiers GLB stockent toutes ces informations dans des fichiers en texte brut qui peuvent être ouverts avec n'importe quel éditeur de texte.
