## Qu'est-ce qu'un fichier GST ?

Un fichier GST est un format de fichier de carte SIG qui contient des données géographiques et une carte visuelle d'une zone cartographiée. Il est utilisé par **MapX** (développé par le PNUE/GRID-Genève et le PNUE pour la gestion des données géospatiales sur les ressources naturelles) et le logiciel de cartographie **MapXtreme** (une boîte à outils de développement logiciel utilisée pour créer des applications SIG personnalisées) qui sont plateformes en ligne de gestion de données géospatiales sur les ressources naturelles. Un fichier GST contient des informations géographiques sous forme numérique ainsi que des images raster qui sont superposées sur la carte.

## Format de fichier GST - Plus d'informations

Les fichiers GST sont enregistrés dans un format de fichier propriétaire et les détails de son format de fichier interne ne sont pas disponibles publiquement pour la référence du développeur.
