## Qu'est-ce qu'un fichier ETA ?

Un fichier avec l'extension .eta est un fichier Google Earth Placemark créé avec le logiciel Google Earth. Il contient des informations sur les signets/marqueurs pour un emplacement dans le monde à l'aide de paramètres tels que la latitude, la longitude, la rotation, le zoom et le titre. Cela permet aux utilisateurs d'avoir une vue d'ensemble de l'emplacement. Avec l'introduction des nouveaux formats de fichier, c'est-à-dire [.kml](/fr/gis/kml/) et [.kmz](/fr/gis/kmz/), le format de fichier ETA a été supprimé et remplacé par ces nouveaux formats de fichier. Les fichiers ETA peuvent être ouverts dans le logiciel [Google Earth](https://earth.google.com/web/) disponible en téléchargement gratuit.


## Format de fichier ETA - Plus d'informations

Les détails internes du format de fichier ETA ne sont pas disponibles publiquement et il n'y a pas de logiciel open source ou propriétaire disponible pour lire les informations des fichiers ETA.
