## Qu'est-ce qu'un fichier MSD ?

Un fichier MSD est un fichier Map Service Distribution créé avec le logiciel de cartographie SIG ArcGIS Desktop. Il contient la carte publiée exportée à partir d'un fichier de carte [.mxd](/fr/gis/mxd/). Un fichier MSD contient des informations sur un service de carte hébergé sur ArcGIS Server. La carte hébergée est accessible sur Internet pour les autres. Le fichier MSD contient le document ArcMap, les informations sur les couches et les propriétés de la table.

## Format de fichier MSD

Les fichiers MSD sont enregistrés sur le disque au format de fichier binaire. Ceux-ci sont utilisés par l'application serveur ArcGIS pour créer et gérer le service de carte à des fins d'hébergement. Les fichiers MSD sont également utilisés par ArcMap pour se connecter au service de carte et afficher le contenu sur la carte.
