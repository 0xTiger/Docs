## Qu'est-ce qu'un fichier 3DD ?

Un fichier avec l'extension .3dd est un fichier de données SIG créé avec le logiciel d'application ESRI ArcGlobe. Il stocke les informations sur les chemins où se trouvent les données 3D sous forme de géodatabase. Lorsqu'un document est ouvert à l'aide du fichier .3dd, il apparaîtra sous la même forme qu'il a été enregistré. ArcGlobe est une application de visualisation 3D qui vous permet de visualiser de grandes quantités de données SIG sur une surface de globe. ArcGlobe est l'une des nombreuses applications qui sont liées au progiciel ArcGIS.

## Format de fichier 3DD - Plus d'informations

Les fichiers 3DD sont stockés sur disque au format de fichier binaire et le format de fichier interne de ces fichiers n'est pas disponible publiquement.
