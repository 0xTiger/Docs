## Qu'est-ce qu'un fichier IMG ?

Un fichier avec l'extension .img est un fichier de carte Garmin utilisé par les appareils GPS Garmin. Il contient des données cartographiques/géospatiales pour représenter des entités cartographiques telles que des noms de rues, des itinéraires de navigation et des points d'intérêt. Les utilisateurs peuvent créer des données cartographiques personnalisées, les enregistrer en tant que fichier .img et les charger sur les appareils Garmin. Garmin fournit également des données cartographiques préinstallées qui sont stockées sur les appareils Garmin au format de fichier IMG. Les fichiers IMG peuvent être ouverts à l'aide d'applications telles que Garmin Mobile PC, cGPSmapper, MapUpload et GPS Map Manager.

## Format de fichier IMG - Plus d'informations

Les fichiers IMG sont enregistrés sous forme de fichiers binaires au format de fichier propriétaire de Garmin. Les informations disponibles dans un fichier IMG peuvent être rendues sur les unités GPS Garmin. Certaines unités GPS, prenant en charge le routage automatique, peuvent également utiliser ces fichiers cartographiques pour se guider lors de la navigation, comme dans les voitures, à vélo ou lors de randonnées.

Les fichiers GMAP sont similaires à une image de disque dur et sont basés sur un système de fichiers composé de sous-fichiers pour fournir des fonctionnalités aux unités GPS Garmin. Les sous-fichiers .GMP fournissent une sorte de structure de sous-répertoires malgré le fait que le format de fichier .gmap soit plat.
