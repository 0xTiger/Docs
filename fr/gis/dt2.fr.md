## Qu'est-ce qu'un fichier DT2 ?

Un fichier DT2 est un fichier d'élévation qui représente le fichier de données numériques d'élévation du terrain (DTED). Il contient des données d'élévation dans des motifs espacés uniformément. Les données d'élévation peuvent être lues par des logiciels SIG et trouvent leurs applications dans différents domaines tels que les communautés militaires, scientifiques et autres.

Les applications SIG pouvant ouvrir les fichiers DT2 incluent ESRI ArcGIS Pro, TatukGIS, Blue Marble Geographic Global Mapper et Pitney Bowes MapInfo. Les autres variantes des types de fichiers DTED sont [DT0](/fr/gis/dt0/) et [DT1](/fr/gis/dt1/).

### Format de fichier DT2

Les fichiers DT2 sont enregistrés sous forme de fichiers binaires sur disque et peuvent être lus à l'aide de nombreuses applications SIG. Ceux-ci peuvent être fusionnés dans un catalogue d'images de géodatabase fichier qui peut être utilisé pour créer une grande mosaïque.
