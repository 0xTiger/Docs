## Qu'est-ce qu'un fichier FMP12 ?

Un fichier FMP12 est un fichier de base de données créé avec le programme de développement de base de données FileMaker Pro 12. Il contient la structure globale de la base de données (également appelée schéma de base de données) ainsi que les données qu'elle contient. La structure d'un fichier FMP12 est basée sur des tables où chaque table se compose d'un certain nombre de champs de base de données ayant différents types de données. Certains types de données pris en charge par le format de fichier FMP12 incluent le texte, les formulaires personnalisés, les rapports, la date et l'heure, les blobs et d'autres objets de base de données. Les anciens formats de fichiers FileMaker Pro tels que [FP7](/fr/database/fp7/) peuvent être mis à niveau vers FMP12.

## Comment convertir les anciens fichiers FileMaker Pro en FMP12 ?

Formats de fichier FileMaker Pro plus anciens tels que les bases de données 11, 10, 9, 8 et 7 (.fp7) vers les formats de fichier FMP12. FileMaker dispose de directives détaillées pour [convertir les versions antérieures des formats de fichiers FileMaker Pro en FMP12](https://fmhelp.filemaker.com/help/16/fmp/en/index.html#page/FMP_Help/converting-files.html) .
