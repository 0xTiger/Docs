## Qu'est-ce qu'un fichier 4DB ?

La structure de base de données construite à l'aide de 4e Dimension, un programme de conception de logiciel qui comprend du code 4D, des interfaces et des informations sur les ressources, est connue sous le nom de 4DB. Il est utilisé dans le framework de conception logicielle intégrative de 4D uniquement pendant le processus de développement. Les fichiers de 4DB se trouvent à la fois sur les plates-formes de bureau et mobiles et le système d'exploitation de Windows est également une option. La cote de popularité des principaux fichiers 4DB est "faible", ce qui signifie que les fichiers 4DB sont généralement présents sur un appareil mobile ou de bureau typique.

Le fichier de structure 4DB contient toutes les spécifications liées à la structure de la base de données telles que les propriétés des champs, les champs et les tables, les listes de choix, les mots de passe, les menus, les méthodes et les formulaires. Le fichier prend le nom que vous saisissez dans la boîte de dialogue de création de la base de données puis suivi du « .4db ».

Il existe deux manières de connecter un document à une nouvelle application (par exemple, mon-fichier.4DB). La première méthode, et la plus simple, consiste à cliquer avec le bouton droit sur le fichier 4DB en question. Sélectionnez "Choisir le programme par défaut" dans le menu automne, puis cliquez sur "Parcourir" pour localiser le logiciel que vous voulez. En cliquant sur OK, vous devez confirmer l'ensemble de l'opération. L'étape suivante, plus complexe, consiste à utiliser le registre Windows pour lier le type de données 4DB au logiciel approprié.
