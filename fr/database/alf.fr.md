## Qu'est-ce qu'un fichier ALF ?

Un fichier avec l'extension .alf est un journal de transactions SQL généré par [ACT! CRM](https://www.act.com/)(Gestion de la relation client). Il garde une trace des activités entre l'ACT! et SQL, et est utilisé pour restaurer l'ACT! base de données. LOI! CRM stocke les informations du client telles que le nom, l'adresse e-mail, le contact, les informations commerciales et l'historique des relations. Il stocke ces informations dans le [ACT! ADF](/fr/database/adf/) fichier qui stocke les données dans des tables pour la liaison avec d'autres modules tels que la récupération automatique d'informations pour un appel entrant.
Les fichiers ALF peuvent être ouverts avec le logiciel SwiftPage ACT sur le système d'exploitation Windows.

## Format de fichier ALF - Plus d'informations

Les fichiers ALF sont enregistrés en tant que fichiers binaires et leur format de fichier interne n'est pas disponible.
