## Qu'est-ce qu'un fichier ADP ?

Un fichier avec l'extension .adp est un fichier de projet Microsoft Access qui utilise l'architecture de composants OLE DB pour fournir une connexion directe et efficace à une base de données Microsoft SQL Server. Ces fichiers ne contiennent pas les tables, les diagrammes de base de données ou d'autres éléments de base de données réels. Les fichiers ADP peuvent être créés avec Access 2007 et 2010. Les fichiers créés avec les versions antérieures d'Access peuvent également être ouverts Access 2007 et 2010. Les fichiers ADP peuvent être ouverts à l'aide de Microsoft Access 365.

## Format de fichier ADP - Plus d'informations

Un projet Access créé en tant qu'ADP vous permet d'apporter des modifications de conception aux objets du serveur SQL, tels que les tables et les vues. Il permet également de créer, de modifier et d'utiliser des fonctionnalités SQL telles que des diagrammes de base de données, des procédures stockées et des fonctions définies par l'utilisateur. Cela offre un avantage par rapport à la liaison à une base de données SQL Server où vous ne pouvez pas apporter de modifications à la conception et ne pouvez que créer un lien vers des tables et des vues SQL Server.

### Prise en charge des types de données et des diagrammes

**Types de données de date/heure**

Les nouveaux types de données de date/heure suivants sont pris en charge dans Access 2010.

* TEMPS
* DATE
* DATEHEURE2
* DATETIMEOFFSET

**Types de données de longueur variable**

Les types de données de longueur variable suivants peuvent être utilisés dans les projets Access 2010 :

* VARBIN(MAX)
* VARCHAR(MAX)
* NVARCHAR(MAX)
