## Qu'est-ce qu'un fichier ABS ?

Un fichier avec l'extension .abs est une base de données générée et utilisée par Absolute Database, un moteur de base de données pour l'IDE de développement logiciel Delphi. Il a remplacé le Borland Database Engine (DBE) et est plus compact, plus rapide, plus robuste et plus facile à utiliser. La base de données ABS se compile dans le cadre du fichier EXE de l'application, ce qui rend votre application plus rapide et plus petite. Les données sont stockées dans un fichier ABS dans un format relationnel structuré.

## Format de fichier ABS - Plus d'informations

Les fichiers ABS sont stockés dans le cadre du [fichier EXE] (/fr/executable/exe/) de l'application qui est généralement stocké sous forme de fichier binaire. Les fichiers binaires ne peuvent être ouverts dans aucun éditeur de texte et fournissent un accès rapide au contenu du fichier de base de données.

### Principales caractéristiques du format de fichier ABS

Les fichiers ABS sont assez simples et intégrés dans l'exécutable de l'application. Il présente les caractéristiques suivantes :

* Pas de BDE ; pas de DLL
* Base de données à fichier unique
* Prise en charge de SQL'92 (DDL et DML)
* Compatible avec les contrôles de bases de données standards et tiers
* Mode mono-utilisateur et multi-utilisateur (serveur de fichiers)
* Fonctionne parfaitement sur toutes les versions de Windows - de 98 à la dernière, ne nécessite aucune mise à jour ni service pack
* Tables en mémoire ultra-rapides
* Facilité d'utilisation inégalée
* Cryptage fort
* Compression des BLOBs
* Gratuit pour un usage personnel
* Code source complet disponible
* Distribution libre de droits
