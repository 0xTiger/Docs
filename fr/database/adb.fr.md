## Qu'est-ce qu'un fichier ADB ?

Un fichier avec l'extension .adb est un fichier de base de données créé avec [Alpha Five](https://www.alphasoftware.com/) (maintenant connu sous le nom d'Alpha Anywhere). Il contient les noms de fichiers de toutes les tables et ensembles membres de l'espace de travail du projet. De plus, il contient les noms de toutes les bibliothèques qui sont incorporées dans l'espace de travail. Alpha Anywhere est un IDE utilisé pour développer des applications Web. En plus des fichiers ADB, l'espace de travail d'Alpha Anywhere peut contenir de nombreux fichiers commençant par le nom d'une table ou d'un ensemble, mais ayant des extensions de fichier différentes.

## Format de fichier ADB - Plus d'informations

Les informations de format de fichier interne des fichiers .adb ne sont pas disponibles pour la référence du développeur.
