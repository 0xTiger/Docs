Une base de données est une collection de données organisées en tableaux et conservées sur un périphérique de stockage informatique et accessibles par voie électronique. Les bases de données relationnelles stockent les données dans des tables de base de données qui sont liées les unes aux autres via des techniques de conception et de modélisation formelles. Les données d'une base de données sont analysées à l'aide du système de gestion de base de données (SGBD) qui est la principale source d'interaction avec les utilisateurs finaux, les applications et la base de données elle-même.

Les extensions de fichier de base de données populaires et leurs formats de fichier incluent **SQLite**, **DB**, **ACCDB** et **MDB**.
