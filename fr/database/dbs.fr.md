## Qu'est-ce qu'un fichier DBS ?

Un fichier .dbs, en particulier ceux avec l'extension SQLBase, fait référence à un fichier de base de données utilisé par le SGBDR SQLBase. SQLBase, est un SGBDR développé par Gupta Technologies, il permet de gérer et de stocker des bases de données relationnelles. Il stocke les données dans un format structuré, permettant une interrogation et une manipulation efficaces des données. SQLBase est un SGBDR populaire pour les petites et moyennes entreprises et est utilisé pour gérer et stocker de grandes quantités de données dans une variété de différents types d'applications.

## Format de fichier DBS

Le format de fichier .dbs de SQLBase est propriétaire, il n'est accessible ou modifiable qu'à l'aide d'outils spécifiquement associés à SQLBase. Les données stockées dans le fichier .dbs peuvent être exportées dans d'autres formats tels que .csv et .xls pour la sauvegarde ou pour la migration vers d'autres systèmes de gestion de base de données.
