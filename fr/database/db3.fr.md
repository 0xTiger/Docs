## Qu'est-ce qu'un fichier DB3 ?
Le fichier DB3 est un fichier de base de données créé par le logiciel SQLite qui est un programme de base de données léger et autonome qui crée des bases de données à l'aide de fichiers simples. contient l'anatomie de la base de données ainsi que les enregistrements de données ; couramment utilisé pour récupérer ou stocker des données structurées à l'aide de SQL. Ces fichiers peuvent être utilisés dans des appareils intelligents ou lorsque la tenue de registres ou toute autre gestion de données est requise, mais dans un environnement à faible espace.


## Format de fichier DB3
Le format de fichier DB3 est associé à RDBMS (Relational Database Management System) SQLite, un choix populaire pour les bases de données embarquées. Il n'y a pas d'extension de fichier spécifique définie dans un SQLite_3 dans sa spécification, il peut utiliser des extensions telles que [.dbf](/fr/database/dbf/) et [.sqlite](/fr/database/sqlite/).

### La sécurisation de la base de données
Généralement SQLite, le format de fichier db lié à la version 3 peut être considéré comme le format de fichier DB3 utilisé comme format natif documenté publiquement pour le moteur de base de données SQLite depuis juin 2004. Le format de fichier DB3 est une plateforme multiplateforme, transférable entre big-endian et les architectures little-endian ou les systèmes 32 bits et 64 bits. Ces fonctionnalités font de DB3 un choix populaire en tant que format de fichier d'application. Le fichier principal de la base de données SQLite_3 (DB3) se compose d'une ou plusieurs pages. Toutes les tailles de toutes les pages sont égales dans la même base de données. La taille de la page en octets est une puissance de deux entre 512 et 65536 inclus.
