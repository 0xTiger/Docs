## Qu'est-ce qu'un fichier ADF ?

Un fichier avec l'extension .adf est un fichier de base de données créé avec le logiciel ACT CRM (Customer Relationship Management) pour stocker des données de relation sur les clients. Il stocke des informations sur le nom, l'e-mail, le contact, les informations commerciales et l'historique des relations du client. Les systèmes CRM sont plus souvent intégrés aux centres d'appels établis par les entreprises pour récupérer automatiquement les informations des utilisateurs pour les appels entrants. Les fichiers ADF peuvent être ouverts avec le logiciel SwiftPage ACT sur le système d'exploitation Windows.

## Format de fichier ADF - Plus d'informations

Les fichiers ADF sont enregistrés sur le disque en tant que fichiers binaires et leurs informations de structure interne ne sont pas disponibles publiquement. Un autre fichier avec l'extension .alf est enregistré avec les fichiers ADF qui contiennent les métadonnées nécessaires pour naviguer, sauvegarder et restaurer l'ACT! base de données.
