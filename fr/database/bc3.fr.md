## Qu'est-ce qu'un fichier BC3 ?

Un fichier BC3 est un fichier de base de données permettant de stocker des informations liées au format d'échange de base de données de construction. Il a été développé sur la base du format d'échange de base de données de construction standard espagnol. Un fichier BC3 stocke des informations contenant des enregistrements de données comprenant des détails sur les matériaux de construction, le coût des matériaux et le prix de la main-d'œuvre. Les données peuvent être échangées entre d'autres applications sans convertir les données dans d'autres formats.

## Format de fichier BC3

Les fichiers BC3 sont stockés sous forme de fichiers binaires avec des détails spécifiés conformément aux normes FIEBDC. La mise en œuvre des fichiers BC3 a fait l'objet de différentes révisions et les versions suivantes du format de fichier BC3 existent :

*FIEBDC-3/2004
* FIEBDC-3/2007
*FIEBDC-3/2012
* FIEBDC-3/2016

FIEBDC est l'abréviation de Formato de Intercambio Estándar - Bases de Datos de Construcción en langue espagnole.
