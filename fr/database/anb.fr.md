## Qu'est-ce qu'un fichier ANB ?

Un fichier avec l'extension .anb est le graphique IBM i2 Analyst's Notebook qui contient des informations sur les relations entre différentes données. Comme son nom l'indique, le logiciel Analyst's Notebook aide à relier les données, ce qui se traduit par une forme plus concise de connectivité entre les différentes informations et fournit une sortie de renseignement pour l'analyse. Cela permet d'identifier et de perturber les activités malveillantes telles que les menaces criminelles, cybernétiques et frauduleuses. Les fichiers ANB peuvent être ouverts avec le logiciel Notebook d'IBM i2Analyst.

## Format de fichier ANB - Plus d'informations

Un fichier ANB comprend les éléments constitutifs suivants :

* Entités - Représentation des informations
* Liens - Afficher la connectivité entre les entités
* Propriétés - Description d'une entité ou d'un lien plus en détail

Les fichiers ANB stockent toutes ces informations sur la base des informations d'entrée qui alimentent le carnet de l'analyste ANB. Sur la base des données saisies, le logiciel crée des liens entre chaque élément d'information qui aide à cartographier les relations et à retracer les preuves géospatiales et temporelles.
