## Qu'est-ce qu'un fichier DB ?

Un fichier avec l'extension db est un fichier de base de données générique pour stocker des données. Il n'y a pas de spécifications officielles à suivre pour un tel fichier de base de données. Les données sont organisées dans un format structuré à l'intérieur du fichier dans les tables de formulaire, les champs, les types de données et les valeurs de champ. Les fichiers de base de données peuvent également être créés en lisant des fichiers de feuille de données contenant des données en lignes et en colonnes. Le contenu des fichiers DB peut en outre être exporté aux formats de fichier [CSV](/fr/spreadsheet/csv/) et [PDF](/fr/pdf/) à des fins de création de rapports. Les fichiers DB peuvent être ouverts dans des applications telles que SQLite et LibreOffice.
