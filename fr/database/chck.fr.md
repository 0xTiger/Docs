## Qu'est-ce qu'un fichier CHCK ?

Un fichier CHCK est créé à la suite de contrôles de santé sur Microsoft Exchange Server. Il contient des informations sur l'état général et l'opérationnalisation de la base de données du serveur. Microsoft Exchange Server est principalement utilisé comme serveur de messagerie et vous permet de gérer vos contacts professionnels et vos calendriers. Les administrateurs du serveur Exchange utilisent son EseUtil pour vérifier l'état de sa base de données.

## Format de fichier CHCK

Le format de fichier interne du fichier CHCK n'est pas connu et il n'existe aucun matériel de référence disponible pour lire/écrire des fichiers CHCK.
