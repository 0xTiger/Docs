## Qu'est-ce qu'un fichier ACCDE ?

Un fichier avec l'extension .accde est un format de fichier Microsoft Access 2007 permettant de créer des fichiers de base de données à exécuter uniquement pour protéger le code de base de données personnalisé. Il compile tout le code VBA en exécutable uniquement afin qu'il ne puisse pas être affiché ou modifié. Le format de fichier ACCDE remplace le format de fichier MDE utilisé par les versions antérieures de Microsoft Access. ACCDE sont des fichiers [ACCDB](/fr/database/accdb/) en mode exécutable uniquement et plus compacts en raison de l'absence de tout code source. La taille compacte de ces fichiers améliore l'utilisation de la mémoire ainsi que les performances de l'application. Les fichiers ACCDE peuvent être ouverts avec Microsoft Access 365.

## Format de fichier ACDE

Tous les fichiers Microsoft Access 2007+ utilisent le Jet ACE (moteur de base de données Access) qui utilise la famille de formats de fichiers ACCDB. Cependant, Microsoft ne fournit pas d'implémentation open source ni de spécification technique détaillée.
