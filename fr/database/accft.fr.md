## Qu'est-ce qu'un fichier ACCFT ?

Un fichier avec l'extension .accft est un fichier de modèle Microsoft Access qui contient la définition des types de données de champ pour les tables d'une base de données. Microsoft Access est un système de gestion de bases de données relationnelles (RDBMS) de bureau utilisé pour créer des bases de données, des formulaires, des requêtes et des vues basés sur des tables. Le fichier ACCFT peut être utilisé pour définir les types de données pour les champs de table et les tables de base de données peuvent être créées en utilisant ce fichier modèle comme point de départ. Il peut être ouvert avec Microsoft Access 2016 pour créer de nouvelles tables. Les fichiers ACCFT sont reconnus par le type MIME **application/msaccess**.

## Format de fichier ACCFT

Les fichiers ACCFT sont des fichiers binaires dont les spécifications de format de fichier ne sont pas disponibles publiquement. Ses détails internes tels que les informations d'en-tête, la structure des fichiers, les pieds de page, etc. ne sont pas disponibles pour le moment.
