## Qu'est-ce qu'un fichier CAF ?

Un fichier CAF est un fichier de catalogue créé avec l'outil logiciel d'organisation multimédia **Cathy**. Il contient une liste de catalogues indexés qui pointe vers tous les supports disponibles sur les périphériques externes tels que les disques flash et les CD. Ces fichiers de catalogue vous aident à maintenir une liste claire de médias, facilement consultable et accessible. Les fichiers CAF proposent de rechercher des médias par nom, taille et date.

## Format de fichier CAF

Les fichiers CAF sont stockés au format de fichier texte. Cela signifie que vous pouvez ouvrir les fichiers CAF dans n'importe quel éditeur de texte tel que Notepad, Notepad++ et Apple TextEdit. Cathy offre la fonctionnalité d'exporter la structure de répertoires de chaque catalogue au format TXT, en créant des listes de dossiers et de fichiers.

### Ouvrir le fichier CAF

Vous pouvez ouvrir CAF avec le logiciel officiel Cathy. Cela nécessitera que vous placiez votre fichier CAF dans le même répertoire où se trouve le fichier Cathy.exe. Cathy affichera automatiquement le catalogue au lancement.

Pour ce faire, placez votre fichier CAF dans le même répertoire que votre fichier Cathy.exe. Ensuite, ouvrez Cathy. Votre catalogue CAF apparaîtra dans Cathy.
