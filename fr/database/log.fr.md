## Qu'est-ce qu'un fichier LOG ?
Un fichier avec l'extension .log contient la liste du texte brut avec horodatage. Habituellement, certains détails d'activité sont enregistrés par les logiciels ou les systèmes d'exploitation pour aider les développeurs ou les utilisateurs à suivre ce qui se passait à une certaine période. Les utilisateurs peuvent modifier ces fichiers très facilement en utilisant n'importe quel éditeur de texte. Habituellement, les rapports d'erreurs ou les activités de connexion sont enregistrés par les systèmes d'exploitation, mais d'autres logiciels ou serveurs Web génèrent également des fichiers journaux pour suivre les visiteurs et surveiller l'utilisation de la bande passante.

## Format de fichier journal
Le format de fichier LOG enregistre les événements typiques qui se produisent dans un système d'exploitation, les messages entre différents utilisateurs d'un logiciel de communication ou tout autre logiciel exécuté. Nous pouvons simplement dire que certains messages à certains horodatages sont enregistrés dans un fichier LOG. Certains termes couramment utilisés pour la journalisation sont :
### Journaux des événements
Il enregistre les événements qui se produisent lors de l'exécution d'un système afin de suivre et de comprendre l'activité du système et de diagnostiquer les problèmes. Ils sont importants pour identifier les activités de systèmes complexes, généralement dans le cas d'applications avec très peu d'interaction avec l'utilisateur.
### Journaux des transactions
Presque tous les systèmes de base de données conservent le journal des transactions, qui ne sont généralement pas conçus comme une piste d'audit pour une analyse ultérieure, et ne sont pas lisibles par l'homme. Ces journaux conservent uniquement l'enregistrement des modifications apportées aux données existantes pour permettre à la base de données de se remettre des pannes ou d'autres erreurs de données et de conserver les données dans un état cohérent. Par conséquent, les systèmes de base de données ont généralement à la fois des journaux d'événements généraux et des journaux transactionnels.
### Journaux des messages
La communication textuelle enregistrée par Internet Relay Chat (IRC), les programmes de messagerie instantanée (IM), les clients de partage de fichiers peer-to-peer avec des fonctions de chat et les jeux multijoueurs (en particulier les MMORPG) sont appelés journaux de messages. Ces journaux sont généralement enregistrés dans des fichiers en texte brut, mais les clients de messagerie instantanée et VoIP (par exemple, Skype) peuvent les enregistrer dans des fichiers HTML pour faciliter la lecture ou activer le chiffrement.
### Journaux du serveur
Le journal du serveur est en fait un fichier journal contenant une liste des activités exécutées et créées ou maintenues automatiquement par le serveur lui-même. Habituellement, ces fichiers ne sont pas accessibles aux utilisateurs généraux d'Internet, uniquement au webmestre ou à une autre personne administrative d'un service Internet.
