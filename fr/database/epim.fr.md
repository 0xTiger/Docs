## Qu'est-ce qu'un fichier EPIM ?

L'extension de fichier .epim est utilisée par EssentialPIM, un logiciel de gestion d'informations personnelles qui permet aux utilisateurs d'organiser et de gérer leurs contacts, rendez-vous, tâches et notes. Le fichier .epim est un fichier de base de données qui stocke toutes les données saisies dans le programme, y compris les informations personnelles, les événements du calendrier et les listes de tâches. Ce fichier peut être sauvegardé, exporté et importé vers une autre installation EssentialPIM. Le logiciel est disponible pour les plateformes Windows, Android et iOS.

Les fichiers .epim sont le format propriétaire utilisé par EssentialPIM. Il est important de noter que les fichiers .epim sont spécifiques au logiciel EssentialPIM et ne peuvent pas être ouverts ou modifiés avec d'autres logiciels.

## Comment créer un fichier EPIM ?

Pour créer un fichier .epim dans EssentialPIM, vous devez d'abord installer le logiciel sur votre ordinateur.

1. Ouvrez EssentialPIM.
2. Cliquez sur le menu "Fichier" et sélectionnez "Nouvelle base de données".
3. Une boîte de dialogue apparaît, vous demandant de sélectionner l'emplacement et le nom du nouveau fichier de base de données.
4. Entrez un nom et choisissez un emplacement pour le fichier, puis cliquez sur "Enregistrer".
5. Le nouveau fichier de base de données sera créé avec l'extension de fichier .epim et vous pourrez commencer à ajouter des données telles que des contacts, des rendez-vous, des tâches et des notes.

Alternativement, vous pouvez également utiliser "Fichier" -> "Importer" pour importer des données de différents formats et créer un nouveau fichier .epim. De plus, si vous avez déjà un fichier de base de données et que vous souhaitez en créer un nouveau, vous pouvez utiliser l'option "Fichier" -> "Exporter" pour exporter toutes les données du fichier .epim actuel et les importer dans le nouveau un.

## Emplacement des fichiers EPIM

L'emplacement des fichiers .epim dans EssentialPIM dépend du système d'exploitation que vous utilisez.

- Sous Windows, l'emplacement par défaut des fichiers .epim se trouve dans le dossier "Mes documents", mais vous pouvez choisir un emplacement différent lorsque vous créez un nouveau fichier de base de données.
- Sur Android, l'emplacement par défaut des fichiers .epim se trouve dans le dossier "Télécharger", mais vous pouvez choisir un emplacement différent lorsque vous créez un nouveau fichier de base de données.
- Sur iOS, l'emplacement par défaut des fichiers .epim se trouve dans le dossier "EssentialPIM", mais vous pouvez choisir un emplacement différent lorsque vous créez un nouveau fichier de base de données.

Vous pouvez également utiliser l'option "Fichier" -> "Ouvrir la base de données" pour ouvrir un fichier .epim existant à partir d'un emplacement différent sur votre ordinateur ou appareil.
