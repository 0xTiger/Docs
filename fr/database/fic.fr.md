## Qu'est-ce qu'un fichier FIC ?

Le fichier FIC sous WinDev est une version compilée d'une Base HyperFileSQL. L'acronyme FIC signifie "Fichier de Code" en français, qui se traduit par "Code File" en anglais. En WINDEV, un fichier FIC est un fichier compilé contenant le code exécutable d'un projet. Le fichier porte l'extension .fic et il est créé lors de la compilation d'un projet WINDEV. Une fois qu'un projet est compilé dans un fichier FIC, il peut être distribué aux utilisateurs sous forme de programme exécutable ou de package d'installation.

Le fichier FIC est lié à WINDEV, un outil de développement logiciel créé par PCSoft, une société française. Il permet aux développeurs de créer des applications Windows, Web et mobiles à l'aide d'une seule base de code. Il comprend un environnement de développement visuel, un langage de programmation (WLangage) et une variété de commandes et de fonctions prédéfinies pour les tâches courantes. Il est principalement utilisé pour les applications professionnelles et dispose d'une solide base d'utilisateurs en Europe.

## Relation avec la base de données HyperFileSQL Mobile

Sous WinDev, un fichier FIC créé pour une base HyperFileSQL Mobile contient le code exécutable de la base ainsi que les données contenues dans la base. HyperFileSQL Mobile est un système de gestion de bases de données relationnelles inclus avec WINDEV et conçu spécifiquement pour les applications mobiles et hors ligne. Le fichier FIC peut être distribué en tant que fichier exécutable autonome ou dans le cadre d'un package d'installation, et il peut être exécuté sur le périphérique cible pour accéder et gérer les données de la base de données HyperFileSQL Mobile. Lorsque le fichier FIC est exécuté, il ouvre la base de données et rend les données disponibles pour une utilisation dans l'application mobile.
