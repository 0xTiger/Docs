## Qu'est-ce qu'un fichier FP5 ?

Un fichier FP5 est un fichier de base de données créé avec FileMaker Pro version 5 ou 6. Il stocke les données utilisateur et organisationnelles sous la forme d'enregistrements dans des tables. FileMaker Pro est un système de gestion de base de données prenant en charge les systèmes d'exploitation Macintosh et Windows. FP5 est un ancien format de fichier pris en charge par la version alors disponible de FileMaker Pro. Le dernier format de base de données FileMaker Pro est [FMP12](/fr/database/fmp12/) qui a été introduit avec le lancement de FileMaker Pro v 12. Un autre format de fichier de base de données bien utilisé était [FP7](/fr/database/fp7/) qui a été remplacé par FMP12.

## Format de fichier FP5 - Plus d'informations

Les détails du format de fichier interne du format de fichier FP5 ne sont pas disponibles publiquement pour la référence du développeur. Les anciens formats de fichiers de base de données FileMaker Pro tels que [FP5 et FP7 peuvent être mis à niveau vers le dernier FMP12](https://fmhelp.filemaker.com/help/16/fmp/en/index.html#page/FMP_Help/converting-files. html) en suivant les directives détaillées de FileMaker Pro.
