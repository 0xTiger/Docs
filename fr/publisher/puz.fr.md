## Qu'est-ce qu'un fichier PUZ ?

Un fichier avec l'extension .puz est une archive compressée qui contient un document standard Microsoft Publisher [.pub](/fr/publisher/pub/). En raison de la compression, il est plus petit et facile à transférer via Internet ou d'autres appareils. Les fichiers PUZ peuvent être décompressés/ouverts à l'aide de l'utilitaire Microsoft Unpack.

## Format de fichier PUZ

Un fichier PUZ est une archive compressée qui contient le fichier PUB standard avec d'autres fichiers. Il contient tous les fichiers requis pour décompresser, afficher et imprimer la publication.

Un fichier PUZ contient les éléments suivants.

* `Unpack.exe` - un utilitaire logiciel pour décompresser les fichiers compressés
* `ReadMe.txt` - un fichier de description
* `Fichier .puz compressé` - Contient la publication et peut inclure des polices intégrées et des graphiques liés.

### Déballage du fichier PUZ

Les fichiers Publisher compressés peuvent être décompressés à l'aide de l'utilitaire Unpack.exe. Il extrait le fichier PUZ compressé dans le choix du répertoire sélectionné par l'utilisateur.
