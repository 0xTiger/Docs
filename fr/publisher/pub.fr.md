## Qu'est-ce qu'un fichier PUB ?

Un fichier PUB est un format de fichier de document Microsoft Publisher. Il est utilisé pour créer plusieurs types de documents de mise en page de conception tels que des bulletins d'information, des dépliants, des brochures, des cartes postales, etc. Les fichiers PUB peuvent contenir du texte, des images raster et vectorielles. Les documents créés avec des fichiers PUB sont principalement utilisés dans les sites Web et les supports marketing tels que les e-mails. Les fichiers PUB peuvent être ouverts avec l'application Microsoft Publisher Desktop, [Microsoft Publisher 365](https://www.microsoft.com/en/microsoft-365), [LibreOffice Draw](https://www.libreoffice.org/discover /draw/) et [Adobe InDesign](https://www.adobe.com/products/indesign.html).

## Format de fichier PUB - Plus d'informations

Les fichiers PUB sont stockés sous forme de fichiers binaires sur le disque. Le contenu d'un fichier PUB peut également être converti au format de fichier [DOCX](/fr/word-processing/docx/) à partir de l'application Microsoft Publisher à l'aide de l'option Enregistrer sous.

## Comment utiliser Microsoft Publisher ?

Microsoft a établi plusieurs directives pour aider les utilisateurs à démarrer avec Microsoft Publisher. Ce guide [Tâches de base dans Publisher](https://support.microsoft.com/en-us/office/basic-tasks-in-publisher-0e5ed249-1927-433f-a35c-63beb8216fcf) vous permet de démarrer en montrant comment pour créer une publication, enregistrer une publication, ajouter du texte et des blocs de construction et imprimer une publication.
