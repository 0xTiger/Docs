## Qu'est-ce qu'un fichier ARF ?

Un fichier ARF (Advanced Recording File) est un fichier d'enregistrement basé sur le réseau (NBR) créé à partir d'une réunion en ligne menée via WebEx. Il contient une vidéo et un son de meilleure qualité et est le format de fichier largement utilisé dans les réunions WebEx. Les informations contenues dans un fichier ARF incluent les données vidéo, la table des matières, la liste des participants, l'enregistrement du bureau, le partage d'applications et d'autres contenus de réunion. Les fichiers ARF ne peuvent pas être modifiés directement, mais le début et la fin de la lecture peuvent être tronqués. ARF est le format de fichier par défaut utilisé par les réunions WebEx.

## Format de fichier ARF

Les fichiers ARF sont enregistrés sur le site Webex en tant que fichiers binaires dans la section **Mes enregistrements** du compte hôte. Celles-ci ne peuvent être effectuées que par un hôte de réunion à partir d'une réunion en ligne à l'aide de l'option Enregistrer sur le serveur. Les fichiers ARF peuvent être lus à l'aide du lecteur d'enregistrement réseau.

## Options de conversion de fichier ARF

Les fichiers ARF peuvent être convertis aux formats [WMV](/fr/video/wmv/), [SWF](/fr/page-description-language/swf/) ou [MP4](/fr/video/mp4/).
