## Qu'est-ce qu'un fichier MPV ?

Un fichier MPV est un fichier vidéo de flux élémentaire MPEG composé de données vidéo brutes non compressées. Le contenu vidéo à l'intérieur d'un fichier MPV comprend des spécifications de compression MPEG-2 intégrées. Les fichiers MPV ne contiennent aucune donnée audio. Le format de fichier MPV est également utilisé comme entrée pour les applications de traitement vidéo car le flux élémentaire [MPEG](/fr/video/mpg/) utilise principalement les normes de codage MPEG-2. Les fichiers MPV peuvent être ouverts et lus avec des applications logicielles prenant en charge les formats de fichiers vidéo MPEG-2. Le logiciel de lecteur multimédia VideoLAN LVC est l'un de ces logiciels qui peut ouvrir et lire des fichiers MPV.

## Format de fichier MPV

Les fichiers MPV sont enregistrés sur le disque sous forme de fichiers binaires simples, suivant la structure du format de fichier MPEG. Cependant, leur format de fichier interne n'est pas disponible publiquement pour la référence du développeur.
