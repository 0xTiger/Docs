## Qu'est-ce qu'un fichier BR ?

Un fichier BR est un fichier Web compressé généré en appliquant l'algorithme de compression de données open source, **Brotli**. Il est utilisé pour stocker des éléments de page Web tels que des feuilles de style ([CSS](/fr/web/css/)), des images ([SVG](/fr/page-description-language/svg/)), [XML](/fr/web/xml/) et des scripts fichiers ([JS](/fr/web/js/)). Les sites Web modernes, tels que Chrome et Firefox, utilisent des fichiers BR pour réduire le temps de chargement des pages, ce qui améliore l'expérience utilisateur.

## Format de fichier BR

Les fichiers BR sont des fichiers Web compressés générés à l'aide de l'algorithme de compression Brotli. La compression Brotli est un algorithme de compression de données sans perte et a été développé par Google pour l'algorithme de compression Zopfli. Il est basé sur une combinaison de l'algorithme de compression sans perte LZ77 et du codage Huffman.

En raison de leur petite taille, les fichiers BR sont utilisés par les serveurs Web et les réseaux de diffusion de contenu (CDN). Les requêtes adressées à ces serveurs entraînent une compression du contenu HTTP, ce qui accélère le chargement des sites Web. Brotli est devenu plus populaire et offre une meilleure compression que gzip.
