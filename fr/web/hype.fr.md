## Qu'est-ce qu'un fichier HYPE ?

Un fichier HYPE est un fichier de projet enregistré à l'aide du logiciel Hype utilisé pour la création de contenu Web avec HTML5. Il contient des éléments Web tels que des graphiques, du texte, des animations, des informations sur la scène et d'autres éléments qui constituent collectivement le contenu. Le contenu Web d'un fichier HYPE peut être exporté vers des fichiers Web pouvant être publiés sur une page Web.

Un fichier HYPE peut être ouvert avec le logiciel [Tumult Hype](https://tumult.com/hype/).

## Format de fichier HYPE

Les fichiers HYPE sont stockés dans un format de fichier propriétaire et les détails internes de ceux-ci ne sont pas connus publiquement. Le contenu d'un fichier HYPE ne peut être visualisé que lorsqu'il est ouvert dans le logiciel Tumult Hype.

### Contenu d'un fichier HYPE

Un fichier HYPE est créé lorsque vous utilisez l'application Hype pour créer du contenu Web à des fins différentes. Il stocke des informations sur les animations basées sur des images clés pour créer des contenus tels que des cartes électroniques, des livres électroniques, des présentations, des publicités, des infographies et du matériel de page Web. Le contenu de ceux-ci est alimenté par HTML5 qui permet d'afficher le contenu sur les appareils, des ordinateurs de bureau aux iPhones et iPads. Tumult Hype produit des styles HTML5, [CSS3](/fr/web/css/) et [JavaScript](/fr/web/js/) de pointe.
