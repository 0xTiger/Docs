## Qu'est-ce qu'un fichier ADR ?

Un fichier ADR (enregistré avec l'extension .adr) est une liste de signets créés par le navigateur Web Opera. Cette liste est remplie au fur et à mesure que l'utilisateur ajoute des URL Web aux signets. Ces signets sont stockés dans le fichier ADR qui peut être sauvegardé et restauré si nécessaire. Étant donné que la plupart des navigateurs tels que Chrome et Firefox enregistrent les signets au format [HTML](/fr/web/html/), ceux-ci peuvent être convertis au format de fichier ADR pour être importés dans le navigateur Opera. Les fichiers ADR peuvent être ouverts dans le navigateur Opera.

## Format de fichier ADR - Plus d'informations

Les détails du format de fichier interne des fichiers ADR sont inconnus. Cependant, il existe des différences entre les formats de fichiers ADR exportés avec différentes versions d'Opera. Il est donc difficile de [fusionner des fichiers ADR](https://superuser.com/questions/471959/how-do-i-merge-several-opera-adr-bookmark-files-to-one-single-file) générer avec différentes versions d'Opera. Certains outils externes peuvent cependant être utilisés pour les fusionner.
