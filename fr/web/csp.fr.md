## Qu'est-ce qu'un fichier CSP ?

Un fichier CSP est un fichier de page Web dynamique qui contient du contenu écrit pour être exécuté sur RadGs Concept Application Server. Il est compilé et exécuté côté serveur, après quoi il est servi au navigateur de l'utilisateur demandeur via le protocole HTTP. Contrairement au format de fichier CON, les fichiers CSP ne nécessitent pas l'installation du client Concept sur l'utilisateur pour afficher ces fichiers. Vous pouvez afficher des fichiers CSP similaires à toutes les autres pages côté serveur telles que [PHP](/fr/programming/php/) et [ASP](/fr/web/asp/).
