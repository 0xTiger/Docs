## Qu'est-ce qu'un fichier OBML16 ?

Un fichier OBML16 est une page Web enregistrée à l'aide du navigateur Web mobile Opera Mini. Il est basé sur le langage de balisage binaire Opera ([OBML](/fr/web/obml/)) et est un format compact permettant d'accéder rapidement aux pages Web enregistrées. Il a remplacé le format de fichier [OBML15](/fr/web/obml15/) lors du lancement de la version 7 d'Opera Mini. Toutes ces variantes de pages Web Opera Mini ont été mises à niveau à partir du format de fichier OBML de base et chaque mise à niveau d'Opera Mini est compatible avec un seul format OBML. Les fichiers OBML16 peuvent être convertis en [PDF](/fr/pdf/) et [HTML](/fr/web/html/) en ligne.

## Format de fichier OBML16 - Plus d'informations

OBML16 est basé sur le format de fichier OBML qui est un format compact pour enregistrer des pages Web. L'avantage d'utiliser ce format compact est qu'il réduit la taille du contenu Web, ce qui entraîne une augmentation de la vitesse de téléchargement. Une fois téléchargés, les utilisateurs d'Opera Mini peuvent visualiser ces pages sans avoir besoin d'avoir accès à une connexion Internet.
