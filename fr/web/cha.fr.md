## Qu'est-ce qu'un fichier CHA ?

Un fichier CHA est un fichier de paramètres utilisateur créé par les clients IRC tels que mIRC. Il contient des paramètres tels que le serveur, le port, le nom du canal et, éventuellement, un mot de passe pour ouvrir un canal IRC. Les clients IRC sont utilisés pour communiquer avec d'autres sur les réseaux IRC du monde entier. Ceux-ci sont utilisés pour des discussions privées en tête-à-tête ou dans des conférences multi-groupes. Lorsqu'un fichier CHA est cliqué, le client IRC s'ouvre automatiquement avec les détails mentionnés dans le fichier.

Les applications qui peuvent ouvrir les fichiers CHA incluent mIRC et Visual IRC.

## Format de fichier CHA

Les fichiers CHA sont stockés avec les autres fichiers d'installation du client IRC. Lorsqu'un utilisateur souhaite se connecter à un serveur et rejoindre un canal, il entre les informations dans le client. Ces informations sont stockées dans le fichier CHA qui peut être récupéré à tout moment.
