## Qu'est-ce qu'un fichier DOTHTML ?

Un fichier DOTHTML est un document modèle Microsoft Word enregistré en version HTML. Il peut être ouvert dans un navigateur Web et le contenu est affiché comme n'importe quelle autre page Web. Les fichiers DOTHTML sont générés à partir de fichiers Microsoft Word .DOT qui contiennent des paramètres pré-formatés pour générer ces fichiers. Lorsqu'un fichier [DOT](/fr/word-processing/dot/) est converti en DOCHTML, tous ces paramètres sont conservés et le fichier DOTHTML converti s'affiche avec un formatage similaire à celui du fichier DOT d'origine. De cette façon, les modèles publiés peuvent être visualisés dans un navigateur Web.

## Format de fichier DOTHTML - Plus d'informations

Les fichiers DOTHTML sont enregistrés en tant que fichiers [HTML](/fr/web/html/) et contiennent du code source HTML qui peut être ouvert avec n'importe quel éditeur HTML. Dans les premiers jours, Microsoft Word proposait les versions HTML de tous ses formats de document tels que [DOCHTML](/fr/web/dochtml/), [DOCMHTML](/fr/web/docmhtml/), mais plus tard, ces versions basées sur HTML ont été abandonnées. . Par conséquent, DOTHMLT n'est plus pris en charge.
