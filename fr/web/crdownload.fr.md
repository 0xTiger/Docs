## Qu'est-ce qu'un fichier CRDOWNLOAD ?

Un fichier CRDOWNLOAD est un fichier partiellement téléchargé qui est créé lorsque vous téléchargez un fichier depuis Internet à l'aide des navigateurs Microsoft Edge, Google Chrome ou Chromium. Le contenu du fichier en cours de téléchargement est stocké dans ce fichier jusqu'à la fin du téléchargement. Une fois le téléchargement terminé, l'extension .crdownload est supprimée et remplacée par l'extension de fichier d'origine du fichier en cours de téléchargement.

## Format de fichier CRDOWNLOAD

Un fichier CRDOWNLOAD peut être au format texte ou binaire. Les navigateurs créent le fichier .crdownload par défaut une fois le téléchargement démarré. Si le téléchargement est interrompu entre-temps, le fichier CRDOWNLOAD reste enregistré sur le disque. Cela permet parfois de reprendre le téléchargement là où il a été interrompu.

### CRDOWNLOAD Exemple

Vous êtes en train de télécharger un fichier fileformatslist.doc à partir d'Internet. Lorsque le téléchargement démarre, votre navigateur enregistre le fichier dans le dossier Téléchargements en tant que fichier fileformatslist.crdownload. Sa taille de fichier augmente à mesure que le fichier est téléchargé. Une fois le téléchargement terminé, le fichier sera renommé en fileformatslist.doc.
