## Qu'est-ce qu'un fichier BWP ?

Un fichier BWP est un fichier de script de page Web créé dans le langage de programmation **BuRg3r**. Il est spécifique à BuRg3r et peut contenir du code HTML avec des sections de source BuRg3r délimitées par<?b and ?> Mots clés. BuRg3r est un langage de programmation à usage général qui est également un moteur de jeu développé par Fluffy Smoke Entertainment.
