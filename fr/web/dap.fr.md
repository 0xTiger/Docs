## Qu'est-ce qu'un fichier DAP ?

Un fichier avec l'extension .dap est un fichier de page de données Microsoft Access qui est créé à partir de Microsoft Access et a accès aux données stockées dans la base de données. Les données stockées dans un tel fichier Web sont accessibles sur le réseau (intranet et Internet), ce qui est différent des fichiers [MDB](/fr/database/mdb/) d'Access qui sont spécifiquement développés pour les applications de bureau. Ces bases de données peuvent être créées à l'aide d'Access 2010 et d'Access Services (un composant facultatif de SharePoint). Ces bases de données peuvent être utilisées par des personnes ayant des comptes SharePoint en utilisant l'adresse Web dans un navigateur Web.
