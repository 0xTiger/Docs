## Qu'est-ce qu'un fichier PRO ? ##

Un fichier PRO est une maquette d'un site Web ou d'une application mobile créée par Adobe Proto, une ancienne application Adobe Touch. Il comprend une ou plusieurs dispositions d'interface utilisateur pour une page Web ou une application mobile, chacune pouvant inclure des éléments interactifs. Les technologies CSS, WebKit et jQuery sont prises en charge dans les fichiers PRO.

### Bref historique du format de fichier PRO

Les fichiers PRO ont été créés par Adobe Proto qui a été remplacé par Adobe XD. Il a été utilisé pour créer des applications de prototypage pour tablettes. Les applications de conception Web courantes telles qu'Adobe Dreamweaver ont utilisé ces fichiers PRO pour créer des applications Web et mobiles.

## Format de fichier PRO - Plus d'informations

Un dossier PRO est la première étape du développement d'un site internet ou d'une application mobile. Les développeurs créent d'abord des maquettes numériques qui définissent l'apparence de l'application. Ceux-ci définissent également le flux de travail global de l'application et la navigation dans les écrans de l'application. Ces maquettes incluent également un certain nombre d'éléments de navigation tels que des boutons, des listes déroulantes, des barres de défilement, des champs de saisie et plusieurs autres.
