## Qu'est-ce qu'un fichier FCGI ?

Un fichier FCGI est un fichier de code source qui contient le code source écrit pour FastCGI (FCGI). Il peut être écrit dans plusieurs langages de programmation différents et contient l'extension de la norme CGI utilisée pour une gestion plus robuste des requêtes du serveur Web. Les langages les plus couramment pris en charge pour FCGI incluent **Perl** qui prend en charge à la fois le code Perl et l'interface externe puissante de l'API FastCGI.

## Format de fichier FCGI

Qu'est-ce qui rend un fichier FCGI important ? L'interface FCGI prend en charge l'informatique distribuée pour gérer les requêtes Web et les distribuer sur plusieurs serveurs. Cela rend les fichiers FCGI principalement utilisés pour l'équilibrage de charge sur plusieurs serveurs.
