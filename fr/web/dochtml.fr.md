## Qu'est-ce qu'un fichier DOCHTML ?

Un fichier DOCHTML est le format HTML d'un document généré avec Microsoft Word. Il s'agit de la version [HTML](/fr/web/html/) du document créé qui peut être utilisée pour la publication sur le serveur Web. Les fichiers DOCHTML peuvent être ouverts dans un navigateur Web pour afficher le contenu du document Word converti. Les perspectives et l'apparence du fichier DOCHTML sont identiques à celles du document Word d'origine.

## Format de fichier DOCHTML

Les fichiers DOCHTML sont enregistrés sur un disque au format HTML prêt à être publié sur des serveurs Web. Microsoft Word a cessé de prendre en charge l'enregistrement de documents en tant que fichiers DOCHTML. Cela a été remplacé par les formats de fichier [.mht](/fr/web/mht/) et [.mhtml](/fr/web/mhtml/) qui sont couramment utilisés maintenant.
