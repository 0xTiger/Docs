## Qu'est-ce qu'un fichier EWP ?

Un fichier EWP est un fichier de projet créé avec le logiciel [EWISoft](http://www.ewisoft.com/) Website Builder. Il est lié à un fichier de base de données binaire qui contient toutes les informations du projet. Le fichier EWP est couramment utilisé pour enregistrer la progression du développement Web. Il s'agit d'un logiciel de conception Web basé sur des modèles qui permet aux utilisateurs de créer des sites Web à partir de modèles de sites Web.

## Format de fichier EWP - Plus d'informations

Les fichiers EWP sont des fichiers de projet qui n'ont pas besoin d'être compilés ou hébergés sur un serveur. Ceux-ci sont enregistrés sous forme de fichiers texte brut et contiennent des références à la progression du développement Web. Les projets peuvent être publiés sur un site Web une fois le développement terminé.
