## Qu'est-ce qu'un fichier TÉLÉCHARGEMENT ?

Un fichier TÉLÉCHARGEMENT est un fichier partiellement téléchargé sur Internet qui est toujours en cours de téléchargement. Lorsque vous téléchargez un fichier sur Internet à l'aide du navigateur Apple Safari, le téléchargement peut prendre un certain temps. Ce fichier partiellement téléchargé est enregistré dans le dossier **Téléchargements**. Si le téléchargement est interrompu pour une raison quelconque, le fichier peut être double-cliqué pour essayer de le reprendre.

## TÉLÉCHARGER le format de fichier - Plus d'informations

Les fichiers TÉLÉCHARGEMENT sont enregistrés sur le disque au format texte ainsi qu'aux formats de fichiers binaires. Il ne contient pas l'intégralité du fichier en cours de téléchargement. Une fois le téléchargement terminé, l'extension de fichier .download est remplacée par l'extension d'origine du fichier en cours de téléchargement. Par exemple, si vous téléchargez une vidéo sur Internet au format de fichier MP4, le fichier sera téléchargé et enregistré avec l'extension .download. Une fois le téléchargement terminé, l'extension du fichier sera changée en .mp4.

Les fichiers DOWNLAOD sont similaires aux fichiers [.CRDOWNLOAD](/fr/web/crdownload/) utilisés par Google Chrome et Microsoft Edge.
