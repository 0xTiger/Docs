## Qu'est-ce qu'un fichier VRT ?

Un fichier VRT est un format de fichier 3D qui contient des mondes en trois dimensions qui peuvent être visualisés dans un navigateur Web avec le plug-in SVR [Viscape](https://www.viscape360.com/). Il est utilisé par le plugin pour charger et afficher les données du monde 3D à l'utilisateur pour la visualisation et l'analyse. Les informations stockées dans un fichier VRT peuvent comprendre des données telles que des textures, des sons, des dimensions et des comportements. Les fichiers VRT les plus récents sont stockés au format de fichier SVR, qui est une version compressée du fichier VRT.

## Format de fichier VRT

Les fichiers VRT sont stockés dans le format de fichier propriétaire de Viscape et les détails de leur format de fichier interne ne sont pas disponibles publiquement.
