## Qu'est-ce qu'un fichier DWT ?

Un fichier avec une extension .dwt est un fichier de modèle de page Web créé avec le logiciel Dreamweaver. Il est utilisé comme point de départ pour développer plusieurs pages dont les paramètres de base sont déjà définis dans le modèle. De nouvelles pages [HTML](/fr/web/html/) peuvent être créées à l'aide de copies de ces modèles et en modifiant le contenu si nécessaire, tout en conservant les sections communes inchangées. Les régions non modifiables de ces modèles, telles que l'en-tête, le pied de page et le menu, restent les mêmes sur toutes les pages Web.

## Format de fichier DWT - Plus d'informations

Les fichiers DWT sont développés et enregistrés sous forme de fichiers texte brut. Ceux-ci peuvent être ouverts dans Dreamweaver ou n'importe quel éditeur de texte, et peuvent être modifiés pour les modifications requises. Le logiciel Dreamweaver met à jour l'interface utilisateur au fur et à mesure que les modifications sont appliquées et l'utilisateur peut observer visuellement l'effet des modifications dans l'interface utilisateur de l'application.

Chaque fois que vous enregistrez la page [HTML] (/fr/web/html/) en tant que fichier de modèle, Dreamweaver crée un dossier de modèle au niveau racine du dossier racine local. Le lien est rompu si le fichier .dwt est déplacé du dossier Modèles ou s'il est renommé. Ainsi, il est important de conserver les fichiers .dwt dans le dossier Templates où Dreamweaver les crée. Il est préférable de conserver les autres éléments de votre site, tels que les fichiers source d'images ou les documents HTML, à un emplacement autre que le dossier Modèles.

## À propos de Dreamweaver

Le logiciel Dreamweaver a été initialement développé par Macromedia. Parmi les autres logiciels populaires de Macromedia, citons le lecteur Flash qui est resté longtemps disponible en tant que lecteur d'animation pour les navigateurs. Après l'acquisition de Dreamweaver par Adobe en 2005, Dreamweaver est devenu une partie intégrante des applications logicielles Adobe et est désormais distribué séparément par Adobe.
