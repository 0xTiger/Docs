## Qu'est-ce que WEBLOC ?

Un fichier WEBLOC est un fichier de raccourci sur macOS qui pointe vers une URL Web sur Internet. Il est généré en faisant glisser l'URL Web de la barre d'adresse du navigateur vers le bureau ou tout autre dossier où il est censé être enregistré. L'URL du site de destination est stockée dans ce fichier de raccourci et peut être visualisée en ouvrant le fichier dans un éditeur de texte tel que Microsoft Notepad, Notepad++ ou Apple TextEdit. Les fichiers WEBLOC sont similaires aux fichiers [URL](/fr/web/url/).

## Format de fichier WEBLOC - Plus d'informations

Les fichiers WEBLOC sont stockés au format de fichier texte brut. Ceux-ci peuvent être générés sur macOS en faisant glisser des URL à partir de navigateurs Web tels que Safari, Google Chrome ou Firefox. Si plusieurs fichiers WEBLOC doivent être ouverts simultanément, une solution plus rapide et plus simple consistant à les ouvrir via des applications externes ou des API serait nécessaire. Ces API peuvent lire et convertir WEBLOC en PDF et JPEG.
