## Qu'est-ce qu'un fichier ALX ?

Un fichier ALX est un fichier ActiveX intégré dans des pages Web sous la forme de composants [HTML](/fr/web/html/). Les contrôles ActiveX étaient officiellement appelés contrôles OLE ou OCX. Les fichiers ALX peuvent être utilisés pour insérer diverses fonctionnalités dans une page Web, telles que l'ajout d'un contrôle de minuterie pour mettre à jour périodiquement le contenu de la page, ou ajouter un contrôle de fenêtre contextuelle pour afficher une aide de type info-bulle en fonction de l'action de l'utilisateur. Entretien des fichiers ALX lié à l'aide<Param> balise dans le code HTML d'une page Web. Vous pouvez ouvrir des fichiers ALX dans Microsoft Internet Explorer, Microsoft ActiveX Control Pad et des éditeurs de texte tels que Notepad et Notepad++.

## Format de fichier ALX - Plus d'informations

Les fichiers ALX sont au format de fichier texte brut et peuvent être modifiés en les ouvrant dans un éditeur de texte. Les contrôles ActiveX peuvent être inclus dans une page Web en les déposant sur la page Web. Cependant, afin de créer des pages Web dynamiques, les contrôles sont intégrés à l'aide de [JScript](/fr/web/js/). Les contrôles d'une page Web sont associés aux événements et aux valeurs des contrôles, offrant ainsi un contrôle complet des interactions de l'utilisateur.
