## Qu'est-ce qu'un fichier URL ?

Un fichier avec l'extension .url est un raccourci d'URL de site Web qui peut être enregistré sur un ordinateur avec n'importe quelle adresse de site Web. Lorsqu'il est ouvert en double-cliquant, l'URL ouvrira le site Web dans le navigateur par défaut et la page sera affichée à l'utilisateur. Les fichiers URL sont généralement appelés fichiers de raccourci vers des sites Web et sont représentés par l'icône du navigateur par défaut. Les fichiers URL peuvent être ouverts et modifiés dans n'importe quel éditeur de texte tel que Microsoft Notepad, Notepad++ ou Apple TextEdit.

## Format de fichier URL - Plus d'informations

Les fichiers URL sont enregistrés au format texte brut et peuvent être ouverts et affichés dans n'importe quel éditeur de texte. Ceux-ci s'ouvrent dans le navigateur Web par défaut où l'utilisateur effectue une autre navigation.
