## Qu'est-ce qu'un fichier XPD ?

Un fichier XPD est un fichier de page Web créé à partir d'un document XML. Le plus souvent, cela est généré via une transformation XSLT. Les [spécifications du format de fichier XPD](https://www.w3.org/TR/xml-pipeline/) sont disponibles sur W3 et définissent un moyen d'augmenter l'interopérabilité du [XML](/fr/web/xml/) multi-processus. applications. Le langage XML Pipeline définit les relations de traitement entre les ressources XML.
