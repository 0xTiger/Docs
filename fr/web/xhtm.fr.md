## Qu'est-ce qu'un fichier XHTM ?

Un fichier avec l'extension .xhtm est un fichier de page Web écrit en XHTML. Il utilise XML 1.0 comme implémentation standard et est considéré comme une reformulation de HTML 4. Comme XHTML, un fichier XHTM contient des balises HTML standard qui suivent des directives XML plus strictes. Les fichiers XHTM sont utilisés pour publier le contenu de sites Web et utilisent couramment l'extension [XHTML](/fr/web/xhtml/).

Vous pouvez utiliser des navigateurs Web tels que Google Chrome, Apple Safari et Mozilla Firefox pour ouvrir les fichiers XHTM.

## Exemple XHTM

L'exemple suivant d'un document XHTM définit les déclarations XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
