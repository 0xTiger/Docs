## Qu'est-ce qu'un fichier CCBJS ?

Un fichier CCBJS est un fichier Javascript créé par l'éditeur d'applications, de jeux et de sites Web 3D **CopperCube**. Il contient le code JavaScript généré dans le cadre du [fichier .html](/fr/web/html/) créé à partir du fichier .ccb. CopperCube fournit une interface de script conviviale pour développer la logique de jeu et les interactions sans écrire de code. Les autres formats de fichiers pris en charge par CopperCube incluent .ccscene, .ccmodel, .ccmaterial, .cctex, .ccjs, .ccp et .ccpak.

## Format de fichier CCBJS

Les détails du format de fichier interne du format de fichier CCBJS ne sont pas connus en tant que tels. Cependant, les fichiers Javascript sont enregistrés sous forme de fichiers texte brut sur le disque et peuvent être ouverts avec n'importe quel éditeur de texte tel que Microsoft Notepad, Notepad++ et Apple TextEdit.
