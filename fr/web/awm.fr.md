## Qu'est-ce qu'un fichier AWM ?

Un fichier AWM est un fichier de projet créé par le programme de création de menus Web, **AllWebMenus**. Il contient des données relatives au projet pour la création de menus déroulants basés sur [HTML](/fr/web/html/). Un fichier AWM est compilé dans un package Web finalisé qui s'intègre aux sites Web pour afficher les menus. Il contient toutes les données liées au projet telles que les paramètres d'apparence, les entrées de menu, le style (vertical ou horizontal), les couleurs et d'autres informations connexes.

Les fichiers AWM peuvent être ouverts avec **Likno AllWebMenus**.

## Format de fichier AWM - Plus d'informations

Les fichiers AWM sont enregistrés sous forme de fichiers binaires compilés et ne peuvent être ouverts qu'avec le logiciel Likno AllWebMenus. Les détails du format de fichier interne des fichiers AWM ne sont pas disponibles publiquement.

AllWebMenus vous permet de concevoir visuellement des menus [CSS](/fr/web/css/) ou des menus basés sur [JavaScript](/fr/web/js/) sans utiliser de langage de programmation. Les menus peuvent être entièrement personnalisés et conçus à l'aide de modèles d'apparence prédéfinis disponibles avec le logiciel.
