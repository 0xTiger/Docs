## Qu'est-ce qu'un fichier HXS ?

Un fichier HXS est un fichier d'aide compilé et utilisé pour le fichier Microsoft Help 2.x. Il est compilé et généré à partir d'un ensemble de pages de rubrique et écrit dans un sous-ensemble de [HTML](/fr/web/html/) similaire à l'ancien format de fichier d'aide [CHM](/fr/web/chm/). Les fichiers HXS sont stockés dans un format compressé et sont créés à partir d'un fichier de projet principal .HxC, d'un fichier d'inclusion .HxF, d'une table des matières .HxT, d'un fichier de définition d'attribut .HxA et de plusieurs index .HxK. Les métadonnées d'un fichier HXS sont créées à partir de [XML](/fr/web/xml/).

Les fichiers HXS peuvent être ouverts avec un navigateur Web.

## Format de fichier HXS

Les fichiers HXS sont créés et enregistrés sous forme de fichiers compressés à partir de contenu HTML et de métadonnées XML.
