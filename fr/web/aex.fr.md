## Qu'est-ce qu'un fichier AEX ?

Un fichier AEX (avec l'extension .aex) est un fichier [Xbasic](https://documentation.alphasoftware.com/documentation/pages/Ref/Xbasic/index.xml) compilé qui contient le code de programme compilé pour les fonctions globales. Il peut être utilisé dans les applications Web en les appelant depuis les pages côté serveur [.a5w](/fr/web/a5w/). Les fichiers AEX ont été chargés et déchargés en appelant respectivement les fonctions a5w_load_aex et a5w_unload_aex à partir des fichiers A5W. Cependant, dans la nouvelle implémentation, ces fichiers sont chargés en mémoire en les incluant dans le fichier a5_application.5i. Le langage de programmation Xbasic est utilisé par le logiciel Alpha Anywhere pour le développement d'applications mobiles et Web.

## Format de fichier AEX - Plus d'informations

Les fichiers AEX sont enregistrés sur le disque au format de fichier binaire et contiennent le code compilé des fonctions écrites en Xbasic. Les instructions de commande de script Xbasic déterminent la structure et le flux d'exécution, y compris l'exécution ou l'arrêt d'une action répétée, ou le choix entre deux étapes ou plus en fonction des conditions. [Xbasic Language Reference](https://documentation.alphasoftware.com/documentation/pages/Ref/Xbasic/index.xml) peut être consulté pour des informations détaillées à ce sujet.
