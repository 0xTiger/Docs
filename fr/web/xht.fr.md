## Qu'est-ce qu'un fichier XHT ?

Un fichier avec l'extension .xht est un fichier Web associé au type de fichier [XHTML](/fr/web/xhtml/) (Extended Hypertext Markup Language), qui est lui-même un fichier de balisage basé sur [XML](/fr/web/xml/). Ces deux fichiers sont similaires au HTML mais ont une syntaxe de type XML plus stricte. Les fichiers XHT sont conçus de manière à être plus structurés, impliquent moins de scripts et sont génériques en plus d'utiliser les fonctionnalités existantes de XML.

## Format de fichier XHT

Un fichier XHT est créé et stocké sous forme de fichier texte brut avec encodage UTF-8 par défaut. Il contient le code source complet d'un document XHTML qui peut être ouvert et modifié avec n'importe quel éditeur de texte prenant en charge le codage Unicode. En plus des éditeurs de texte, le format de fichier XHT est compréhensible par la plupart des navigateurs Web modernes et peut être ouvert à l'aide de ceux-ci.

## Exemple XHT

L'exemple suivant d'un document XHT définit les déclarations XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
