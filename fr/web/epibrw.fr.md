## Qu'est-ce qu'un fichier EPIBRW ?

Un fichier EPIBRW est un fichier générique hébergé sur un serveur Web basé sur Microsoft IIS. Le but de ce fichier n'est pas spécifique à des opérations spécifiques ou à tout type de stockage de données, mais est général. Il est le plus souvent utilisé par les fichiers [.pdf](/fr/pdf/), mais peut également être utilisé par plusieurs autres types de fichiers.

Lorsqu'un fichier EPIBRW est double-cliqué, il peut s'ouvrir dans un navigateur affichant le contenu. Cependant, si le fichier est lié pour le téléchargement, le processus de téléchargement du fichier sera lancé à la suite de cet événement.
