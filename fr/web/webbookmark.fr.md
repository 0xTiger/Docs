## Qu'est-ce qu'un fichier WEBBOOKMARK ?

Un fichier WEBBOOKMARK est un fichier de signets enregistré avec le navigateur Web Safari d'Apple sur macOS. Il contient des informations sur la page Web que l'utilisateur a enregistrée à partir du navigateur. Les informations stockées dans un fichier WEBBOOKMARK incluent le titre de la page, l'URL et le groupe de signets dans lequel cette page est stockée. Lorsqu'une page Web est mise en signet, le navigateur Safari demande au groupe de stocker la page en signet. La plupart des utilisateurs enregistrent les signets dans le dossier Signets qui est l'emplacement par défaut proposé par les navigateurs courants.

## Format de fichier WEBBOOKMARK

Les fichiers WEBBOOKMARK sont enregistrés sur le disque au format de fichier binaire qui ne sont pas lisibles par l'homme. Les utilisateurs peuvent marquer une page Web comme signet si elle est destinée à être visitée plus tard ou se rapporte aux intérêts de l'utilisateur. Si nécessaire, l'utilisateur peut accéder à la section des signets dans le navigateur Safari et visiter à nouveau la page.
