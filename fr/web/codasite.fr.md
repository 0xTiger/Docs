## Qu'est-ce qu'un fichier CODASITE ?

Un fichier CODASITE est un fichier de projet de site Web créé avec le logiciel de création et de publication de pages Web [Coda](https://coda.io/). Il contient une liste de tous les fichiers qui font partie du projet. Cela inclut les pages Web en tant que fichiers [HTML](/fr/web/html/), les fichiers de feuille de style en cascade [CSS](/fr/web/css/), les fichiers Javascript [JS](/fr/web/js/) et tout autre fichier de support associé. Les fichiers de projet CODASITE contiennent également des informations pour la publication du site Web sur le serveur Web.

## Format de fichier CODASITE

Le fichier de projet CODASITE est enregistré sur le disque sous forme de fichiers binaires dans le répertoire ~/​Library/​Application Support/​Coda 2/​Sites/​. L'interface utilisateur Coda fournit l'interface de gestion des sites Coda, car leur ouverture manuelle peut entraîner des paramètres de projet incorrects et créer des problèmes.
