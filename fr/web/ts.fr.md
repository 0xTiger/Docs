## Qu'est-ce qu'un fichier TS ?

Un fichier TS est un fichier de traduction généré avec l'utilitaire logiciel lupdate fourni avec le SDK Qt de Digia. Le SDK est disponible pour les systèmes d'exploitation Windows, Mac et Linux. Le fichier TS contient des phrases dans une langue et leurs traductions dans une autre langue. Les traductions stockées dans un fichier TS sont spécifiques à une application particulière. Les fichiers TS peuvent être ouverts avec le SDK Digia Qt.

## Format de fichier TS

Les fichiers Digia Qt TS sont stockés sur disque au format XML et peuvent être ouverts dans n'importe quel éditeur de texte tel que Notepad et Notepad ++. Certains autres formats de fichiers pris en charge par Digia Qt SDK incluent :

* .pri
* .qm
* .qml
* .qph
* .qrc
* .qs
* .qss
* .ts
* Fichier d'interface utilisateur .ui
