## Qu'est-ce qu'un fichier ESPROJ ?

Un fichier ESPROJ est un fichier de projet créé par un logiciel de création de pages Web, Espresso. Il est disponible sur Mac OS et contient une liste des fichiers associés qui font partie du projet de site Web. Lorsque vous ouvrez le fichier ESPROJ dans Espresso, tous les fichiers associés s'ouvrent dans l'éditeur pour modifier ou mettre à jour le code source. Le fichier ESPROJ contient également les paramètres de publication du serveur et d'autres préférences utilisateur associées.

## Format de fichier ESPROJ

Les fichiers ESPROJ sont enregistrés sous forme de fichiers binaires et leur contenu n'est pas lisible par l'homme. Le répertoire de fichiers ESPROJ est également le répertoire racine du site Web.
