## Qu'est-ce qu'un fichier FMP ?

Un fichier avec le fichier .fmp est un fichier de menu animé créé avec l'application 123 Flash Menu. Il contient des menus animés pour le site Web, y compris des composants tels que les polices, le style du menu, les couleurs, les dimensions, les sons et les animations. L'application 123 Flash Menu ne nécessite aucune compétence informatique particulière car l'application elle-même intègre toutes les fonctionnalités pour développer de tels menus. Les fichiers de menu FMP peuvent être exportés vers les fichiers Flash [.swf](/fr/page-description-language/swf/) qui peuvent être intégrés dans une page Web.
