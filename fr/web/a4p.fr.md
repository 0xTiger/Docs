## Qu'est-ce qu'un dossier A4P ?

Un fichier A4P est un fichier de projet créé par le logiciel A4Desk Pro qui est utilisé pour créer des sites Web Flash. Il est créé avec les modèles inclus avec A4Desk Pro et n'en a pas besoin pour avoir une connaissance approfondie de Flash. Les fichiers A4P créés avec des modèles peuvent inclure du contenu de site Web sous forme de texte et d'animations Flash. Si vous ne connaissez pas Flash, il a été utilisé pour créer des animations pouvant être exportées vers les formats de fichier [SWF](/fr/page-description-language/swf/) ainsi que [GIF](/fr/image/gif/) pour inclusion dans les sites Web.

## Format de fichier A4P

Les fichiers A4P sont créés à partir de fichiers modèles et enregistrés sous forme de fichiers binaires sur le disque. La structure interne du format de fichier des fichiers A4P n'est pas connue.

Les fichiers A4P peuvent être ouverts avec le logiciel **Avanquest A4Desk Pro**.
