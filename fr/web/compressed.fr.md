## Qu'est-ce qu'un fichier COMPRESSÉ ?

Un fichier COMPRESSÉ est une page Web compressée Wordpress de petite taille générée avec le plug-in **WordCompress**. Il contient une seule ligne de code pour représenter la page Web générée en compressant le code HTML, CSS et JavaScript par le plugin. Ces fichiers COMPRESSÉS convertis sont de petite taille, ce qui permet un chargement et un transfert plus rapides sur Internet.

### Format de fichier COMPRESSÉ - Plus d'informations

Les fichiers COMPRESSÉS sont enregistrés sous forme de fichiers texte pouvant être ouverts avec n'importe quel éditeur de texte tel que Microsoft Notepad, Notepad++ et Apple TextEdit. En plus des éditeurs de texte, les fichiers COMPRESSÉS peuvent être ouverts avec un logiciel de création de sites Web tel qu'Adobe Dreamweaver, disponible pour Windows et Mac OS.

## À propos du plugin WordCompress

Le plugin WordCompress a été initialement utilisé pour compresser des pages Web WordPress au format de fichier compressé. Le plugin n'est plus pris en charge et est abandonné. Il écrirait le code source de la page Web dans un seul fichier exécutable en supprimant tous les espaces blancs du code source de la page. Les fichiers COMPRESSÉS sont stockés avec le code source d'origine de la page Web afin de préserver les commentaires, les indentations et autres espaces blancs d'origine inclus dans le code source d'origine de la page.
