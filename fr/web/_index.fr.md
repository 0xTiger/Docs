Il faut un certain nombre de types de fichiers lors de la création d'une page Web. Les formats de fichiers Web définissent les normes de développement des pages Web et sont liés à la plate-forme sur laquelle elles sont construites. Un site Web complet peut être construit composé de pages Web statiques et dynamiques. La plupart des sites Web modernes sont construits sur une technologie côté serveur telle que les pages ASP (Active Server Pages) qui sont chargées et exécutées sur le serveur Web. Ceux-ci incluent également des feuilles de style en cascade (CSS) et des fichiers de script utilisés pour styliser l'apparence générale de l'interface utilisateur.

Les extensions de fichiers Web courantes et leurs formats de fichiers associés incluent **[HTML](/fr/web/html/)** (Hypertext Markup Language), **[ASP](/fr/web/asp/) (Active Server Pages)**, et **CSS** (feuilles de style en cascade).
