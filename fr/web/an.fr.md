## Qu'est-ce qu'un fichier AN ?

Un fichier avec l'extension .an est un fichier de projet [Adobe Edge Animate](https://www.adobe.com/products/animate.html) qui contient des informations sur l'animation. Un fichier .html est créé lors de l'enregistrement du projet qui contient les références aux fichiers associés [.css](/fr/web/css/) et [.js](/fr/web/js/) générés par Adobe Edge Animate. Le fichier AN est enregistré avec ces fichiers qui peuvent être utilisés pour ouvrir le fichier de projet. Adobe edge animate peut être utilisé comme alternative à la création de pages Web animées à l'aide d'Adobe Flash, qui n'est plus pris en charge par les navigateurs.

## Format de fichier AN

Les fichiers AN sont stockés sur disque au format de fichier texte brut et peuvent être ouverts dans des éditeurs de texte pour afficher leur contenu.
