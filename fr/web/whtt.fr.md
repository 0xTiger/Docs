## Qu'est-ce qu'un fichier WHTT ?

Un fichier WHTT est une version hors ligne d'un site Web qui est téléchargée à l'aide du logiciel [HTTrack Website Copier](https://www.httrack.com/). Il contient tout le contenu d'un site Web, y compris les pages Web [HTML](/fr/web/html/), les images, les feuilles de style, les liens et les répertoires. Toutes les pages d'un site Web sont téléchargées de manière récursive et enregistrées sur le disque en tant que miroir hors ligne. Lorsqu'un site Web est téléchargé par le logiciel HTTrack, il peut être consulté hors ligne sans avoir besoin d'être connecté à Internet. Pour afficher le site Web téléchargé, ouvrez simplement la page du site Web en miroir dans le navigateur et chaque image/lien sera chargé correctement.

## Format de fichier WHTT

Les fichiers WHTT sont enregistrés sous la forme d'une combinaison de pages Web THML, d'images, de feuilles de style et de liens de référence.
