## Qu'est-ce qu'un fichier MHT ?

Un fichier avec l'extension .mht est un format de fichier d'archivage compatible MIME qui contient différents types de données dans un seul fichier. Il peut stocker des données telles que du texte, des images, des styles de page sous la forme de fichiers [CSS](/fr/web/css/), JavaScript et d'autres ressources en tant que ressources intégrées. Les fichiers MHT, de type MIME message/rfc822, encapsulent tout le contenu d'un fichier HTML dans un seul fichier d'archive pour le stockage lors de l'archivage sur des périphériques de stockage. Les applications logicielles telles que Microsoft Word vous permettent de convertir vos documents WORD en MHT en les exportant sous forme de fichier MHT. Les fichiers MHT peuvent être ouverts à l'aide de navigateurs populaires tels que Microsoft Internet Explore et Google Chrome.

## Format de fichier MHT

Comme [MHTML](/fr/web/mhtml/), MHT est également une encapsulation MIME de documents HTML agrégés. Le contenu du document à l'intérieur d'un document MHT, tel que les images en ligne, les feuilles de style, les applets, etc., est codé MIME lorsqu'il est lié à une ressource racine via des URI. Les clients de messagerie tels que Microsoft Outlook stockent les e-mails (généralement [EML](/fr/email/eml/)) au format de fichier MHT. Les spécifications complètes du format de fichier MHT sont disponibles dans la [RFC 822](https://tools.ietf.org/html/rfc822) et peuvent être consultées pour le développement d'applications logicielles pouvant traiter des fichiers MHT.

## Différence entre MHT et MHTML

Lors de l'enregistrement d'une page en ligne, l'utilisateur est invité à choisir le type de format de fichier dans lequel la page en ligne doit être enregistrée sur le système natif. Le plus souvent, l'utilisateur est confus entre le langage de balisage et les formats de fichier MHTML. Ils remarquent qu'il est difficile de voir que le format de fichier est plus sain entre ceux-ci.

Lorsqu'un utilisateur décide d'éviter de gaspiller une page en ligne en tant que langage de balisage, un dossier est formé, qui contient plusieurs fichiers pour différents contenus de la page, c'est-à-dire une unité de zone de fichiers totalement différente créée pour le texte, les graphiques, les applets, etc. la page en ligne car MHTML crée un fichier pour la page en ligne complète. Tous les éléments de la page ainsi que les graphiques, les liens et les fichiers alternatifs sont intégrés dans un seul fichier.

Naturellement, il est simple de gérer les fichiers MHT ou MHTML car le fichier, qui peut être protégé et partagé simplement par e-mail. Cependant, la zone des fichiers de langage de balisage est sous le risque de perte d'informations, car la perte ou la suppression d'un seul fichier pourrait rendre le dossier complet du langage de balisage inutile pour l'utilisateur.
