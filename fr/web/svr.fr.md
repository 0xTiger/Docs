## Qu'est-ce qu'un fichier SVR ?

Un fichier SVR est un format de fichier 3D utilisé par SuperScape pour créer des mondes interactifs 3D. Ces mondes interactifs en trois dimensions peuvent être visualisés via des navigateurs Web avec le plugin SVR. Les fichiers SVR sont similaires au format [fichier VRT](/fr/web/vrt/) mais sont compressés, ce qui entraîne une taille de fichier plus petite. Ainsi, les fichiers SVR sont chargés plus rapidement et peuvent également être partagés sur Internet.

## Format de fichier SVR - Plus d'informations

Les fichiers SVR sont enregistrés sur disque au format de fichier binaire. Il s'agit d'un format de fichier propriétaire et les détails de sa structure de fichier interne ne sont pas disponibles publiquement pour référence par le développeur.
