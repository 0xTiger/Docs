## Qu'est-ce qu'un fichier CPHD ?

Un fichier CPHD est un document hypertexte utilisé par le moteur de préprocesseur CloudChan. Il contient du HTML et des scripts pour créer des pages Web et des documents dynamiques. Les scripts contenus dans un fichier CPHD ressemblent à la syntaxe utilisée par Java. La page Web CPHD renvoie un document HTML lorsqu'elle est exécutée sur un processus CloudChan. Le format CPHD peut fonctionner avec n'importe quel document XML.

## Format de fichier CPHD

Les fichiers CPHD sont stockés sur disque sous forme de fichiers texte brut. Cela permet d'ouvrir ces fichiers dans n'importe quel éditeur de texte tel que Notepad ou Notepad++.
