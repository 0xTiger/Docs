## Qu'est-ce qu'un fichier HTC ?

Un fichier HTC est un fichier au format de fichier Web utilisé par Microsoft Internet Explorer. Il contient des définitions pour les composants HTML qui exécutent de petites fonctions pour effectuer différentes tâches dans le navigateur Web. Les fonctionnalités définies dans les fichiers HTC sont réutilisables et accessibles sur plusieurs pages [HTML](/fr/web/html/).

## Format de fichier HTC - Plus d'informations

Les fichiers HTC sont enregistrés au format de fichier texte brut. Cela signifie que vous pouvez les ouvrir dans n'importe quel éditeur de texte et apporter des modifications à la fonctionnalité. Ceux-ci sont généralement ouverts dans Internet Explorer, mais de cette façon, vous ne pourrez voir que ces fichiers et ne pourrez pas les modifier.

Un fichier HTC est en fait un fichier HTML qui est une combinaison de script et d'un ensemble d'éléments spécifiques à HTC pour définir les composants.
