## Qu'est-ce qu'un fichier GSP ?

Un fichier GSP est un fichier de message d'erreur qui indique un problème avec la livraison de l'e-mail. Il est envoyé par IMail Server, un programme de serveur de messagerie utilisé pour la communication par courrier électronique par de nombreuses petites et moyennes entreprises. L'erreur peut être due à un certain nombre de raisons telles qu'une énorme quantité de données, la perte de données dans le fichier [MSG](/fr/email/msg/) ou des erreurs de mémoire insuffisante.

## Format de fichier GSP - Plus d'informations

Les fichiers GSP sont stockés au format de fichier Web et sont facilement consultables avec des clients de messagerie tels que Microsoft Outlook. Un fichier GSP est généré en réponse à un e-mail envoyé qui entraîne une erreur.
