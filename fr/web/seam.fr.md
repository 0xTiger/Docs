## Qu'est-ce qu'un fichier SEAM ?

Un fichier SEAM est un fichier de configuration utilisé par le framework JBOS Seam. Il contient des métadonnées/référence à un servlet côté serveur qui traite les requêtes adressées à l'application Web. Il contient également des métadonnées sur les composants Seam telles que les noms des classes et des méthodes qui implémentent les composants. Vous pouvez utiliser les fichiers SEAM conjointement avec le framework JavaServer Faces (JSF) pour créer des applications Web dynamiques et interactives. Les fichiers SEAM sont généralement stockés dans le répertoire WEB-INF d'une application Seam.

## Format de fichier SEAM

Un fichier SEAM est stocké sous forme de fichier texte, généralement au format [XML](/fr/web/xml/), et contient des informations de configuration pour une application Seam. C'est un élément clé de la configuration d'une application Seam. Vous pouvez ouvrir un fichier SEAM avec n'importe quel éditeur de texte ou XML.
