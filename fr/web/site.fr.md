## Qu'est-ce qu'un fichier SITE ?

Un fichier SITE est un fichier de projet utilisé par le logiciel de conception de sites Web « Adobe GoLive ». Il stocke les informations sur le projet de site Web telles que les fichiers, dossiers, images, feuilles de style personnalisées et scripts. Il contient également la structure du site Web et les liens utilisés dans les pages Web du site. Comme il s'agit d'un fichier de projet, il contient des références aux fichiers de projet mais ne contient pas les fichiers eux-mêmes.

Adobe GoLive a été abandonné et n'est plus pris en charge.

## Format de fichier SITE

Les fichiers SITE sont stockés sur disque au format de fichier propriétaire Adobe et les détails internes de son format de fichier ne sont pas disponibles publiquement pour référence du développeur.
