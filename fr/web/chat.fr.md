## Qu'est-ce qu'un fichier CHAT ?

Un fichier CHAT est un fichier de configuration utilisé par l'IRC (Internet Relay Chat) pour se connecter à un canal IRC. Il contient des paramètres de connexion tels que le serveur, le port, le nom du canal IRC et éventuellement un mot de passe pour ouvrir un canal IRC. Les paramètres sont utilisés par le client pour se connecter à d'autres canaux. Les chats et les discussions sur IRC sont en tête-à-tête et sont privés.

Les applications qui peuvent ** ouvrir ** les fichiers CHAT incluent mIRC et Visual IRC.

## Format de fichier CHAT

Les fichiers CHAT sont stockés sous forme de fichiers binaires sur le disque. Lorsqu'un utilisateur veut se connecter à un canal dans IRC, il entre les informations dans le client iRC, il entre les informations dans le logiciel client. Les informations sont stockées dans le fichier CAHT pour une récupération latérale.
