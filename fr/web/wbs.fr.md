## Qu'est-ce qu'un fichier WBS ?

Un fichier WBS est un fichier de projet créé par WebBlender qui est un programme de présentation et de création de site Web pour les enfants. Le fichier WBS est créé lorsqu'un nouveau projet est lancé dans WebBlender. Toutes les fonctionnalités du site ajoutées au projet telles que les pages, les images, le texte, les animations, etc., sont référencées dans le fichier de projet WBS. Une fois que le projet est prêt, il publie le contenu du site sous forme de fichier [HTML](/fr/web/html/). WebBlender est créé et maintenu par [Tech4Learning](https://www.tech4learning.com/).
