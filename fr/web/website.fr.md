## Qu'est-ce qu'un fichier SITE WEB ?

Un fichier WEBSITE est un fichier de raccourci de site Web créé avec Microsoft Internet Explorer (IE). Il contient une URL et un fichier d'icône (ICO) qui pointe vers le favicon du site Web. L'icône est utilisée pour afficher le raccourci. Le format de fichier WEBSITE a été introduit avec la version 9 d'IE. Les versions précédentes d'IE enregistraient les raccourcis vers les pages Web sous forme de fichiers [.url](/fr/web/url/).

## Format de fichier du SITE WEB

Un fichier .website est stocké sur le disque sous forme de fichiers texte brut dont le contenu peut être visualisé dans n'importe quel éditeur de texte. Outre d'autres contenus, il contient l'URL cible pour laquelle il s'agit d'un raccourci. Si vous souhaitez ouvrir le site Web cible dans un navigateur autre qu'IE, vous pouvez copier cette URL et la coller dans d'autres navigateurs tels que Google Chrome, Safari ou Firefox pour afficher le site Web.

## Comment créer un fichier .WEBSITE ?

Un fichier WEBSITE peut être créé dans IE en cliquant sur l'URL dans la barre d'adresse et en le faisant glisser vers l'emplacement souhaité sur votre ordinateur, tel que le bureau ou les documents. Cela crée le raccourci vers cet emplacement Internet et peut être ouvert en double-cliquant sur le raccourci. Si vous souhaitez enregistrer le raccourci en tant que fichier "URL", maintenez la touche MAJ enfoncée avant de faire glisser le favicon du site cible sur le bureau.
