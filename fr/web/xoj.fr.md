## Qu'est-ce qu'un fichier XOJ ?

Un fichier XOJ est un fichier bloc-notes qui contient des notes prises à l'aide du logiciel de prise de notes manuelles Xournal. Xournal est une application multiplateforme gratuite qui vous permet de créer des notes avec différents types de contenu tels que du texte, des images et des griffonnages manuscrits. Les fichiers XOJ sont créés lorsqu'un nouveau fichier de bloc-notes est créé avec Xournal. Ces fichiers XOJ peuvent également être partagés avec d'autres utilisateurs pour le partage de données. Une version plus récente de Xournal est [Xournal++](https://github.com/xournalpp/xournalpp) qui est une variante multiplateforme de Xournal.

## Format de fichier XOJ

Les fichiers XOJ sont au format de fichier XML et sont enregistrés sur le disque sous forme compressée à l'aide de la compression [.gz](/fr/compression/gz/). En raison de l'utilisation du format XML universel, les fichiers XOJ peuvent également être ouverts et modifiés avec la variante multiplateforme Xournal++.

Chaque fichier XOJ peut contenir une page supplémentaire d'un carnet Xournal.

## XOJ - Plates-formes prises en charge

Xournal est multi-plateforme et ses blocs-notes peuvent être enregistrés en tant que fichier XOJ sur les systèmes d'exploitation suivants.

*Linux
* Mac OS
* Les fenêtres
* Android
* Version iOS
