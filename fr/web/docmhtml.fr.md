## Qu'est-ce qu'un fichier DOCMHTML ?

Un fichier DOCMHTML est une ancienne variante d'archive Web enregistrée par les versions antérieures de Microsoft Word. Il n'est plus pris en charge par la dernière version de Microsoft Word. Un fichier DOCMHTML contient une représentation HTML du document Word .doc d'origine. Il peut être ouvert dans Microsoft Word ainsi que dans un navigateur Web. Lorsqu'un DOCMHTML est publié, il peut être hébergé sur un serveur Web accessible aux utilisateurs via une requête Web. Un autre format similaire est [DOCHTML](/fr/web/dochtml/) qui a également été abandonné.

## Format de fichier DOCMHTML - Plus d'informations

Les fichiers DOCMHTML ont été utilisés pour être enregistrés au format HTML. Il peut être ouvert dans un éditeur de texte tel que Notepad et Notepad++. Après avoir été marqué comme obsolète, Microsoft Word a introduit les options d'enregistrement du document Word en tant que fichiers [MHTML](/fr/web/mhtml/) et [MHT](/fr/web/mht/). Ces nouveaux formats de fichiers peuvent être ouverts avec des navigateurs Web.
