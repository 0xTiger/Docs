## Qu'est-ce qu'un fichier EDGE ?

Un fichier avec l'extension .edge est un fichier Adobe Edge Project qui est créé lorsqu'un projet est créé et enregistré avec Adobe Edge. Adobe Edge permet aux concepteurs Web de créer des animateurs [HTML](/fr/web/html/) interactifs pour le Web, la publication numérique et la publicité rich media pour les navigateurs de bureau et mobiles. Ceci est réalisé en utilisant HTML/HTML5, [JavaScript](/fr/web/js/) et [CSS3](/fr/web/css/). Les fichiers Edge sont générés sous forme de fichiers texte brut et peuvent être ouverts avec plusieurs applications différentes telles que Microsoft Notepad, Apple TextEdit et Notepad ++.

## Format de fichier BORD

Les fichiers EDGE sont enregistrés au format texte brut et utilisent jQuery pour activer l'animation. Une alternative à l'animation EDGE était les fichiers Flash qui ne sont plus pris en charge. Le fichier EDGE est enregistré avec les fichiers HTML, JavaScript et CSS. Lors de l'ouverture du fichier EDGE, la page Web, c'est-à-dire le fichier HTML, s'ouvre.
