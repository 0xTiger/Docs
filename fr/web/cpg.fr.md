## Qu'est-ce qu'un fichier CPG ?

Un fichier CPG est un fichier de projet créé par le logiciel d'édition de pages Web Cool Page. Le logiciel est devenu obsolète depuis de nombreuses années et n'est plus utilisé. Le fichier CPG contenait des détails sur les objets utilisés dans la page Web et les propriétés de ces objets. Les fichiers du projet CPG peuvent être ouverts dans le logiciel Cool Page et après finalisation, la page Web peut être exportée sous forme de page Web [HTML](/fr/web/html/).

## Format de fichier CPG

Les détails du format de fichier interne du format de fichier CPG ne sont pas connus. Ceux-ci ont été enregistrés sous forme de fichiers binaires et ne s'ouvriront que dans le logiciel Cool Page.
