## Qu'est-ce qu'un fichier FWP ?

Un fichier avec le fichier .fwp est un fichier d'archive créé avec Microsoft Expression Web et contient tous les fichiers et dossiers d'une conception. Microsoft Expression Web est un produit logiciel de conception Web général de Microsoft et est basé sur l'éditeur [HTML](/fr/web/html/). Il a été interrompu depuis 2012 et disponible gratuitement en téléchargement par Microsoft. Les fichiers FWP peuvent être exportés et enregistrés au format .cab de Microsoft.

## Format de fichier FWP - Plus d'informations

Les spécifications exactes du format de fichier des fichiers FWP ne sont pas disponibles pour la référence du développeur. Cependant, ce sont des fichiers binaires et peuvent être ouverts dans Microsoft Expression Studio qui a été abandonné il y a longtemps.
