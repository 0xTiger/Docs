## Qu'est-ce qu'un fichier STC ?

Un fichier avec l'extension .stc est un fichier de connexion utilisé par le logiciel Adobe Contribute pour stocker les informations de connexion. Il contient des informations de connexion telles que le nom d'utilisateur, l'URL de l'hôte, des protocoles tels que FTP, SFTP, etc. pour une réutilisation à partir du site Web existant. Pour ceux qui ne le savent pas, Adobe Contribute était auparavant un éditeur [HTML](/fr/web/html/) spécialisé qui était utilisé pour pousser les mises à jour des sites Web existants, y compris les articles basés sur le contenu appelés blogs. Il comprenait des plugins pour les navigateurs Web populaires tels qu'Internet Explorer et Firefox qui contribuaient de manière transparente au contenu des sites Web à partir des pages. Il a été abandonné depuis janvier 2017 et Adobe Dreamweaver est maintenant utilisé pour créer des sites Web à partir de zéro.

## Format de fichier STC

Le format de données interne des fichiers STC est inconnu. Cependant, la clé de connexion est chiffrée avec un mot de passe et toutes les informations de connexion réseau ou FTP envoyées dans le fichier restent sécurisées et ne sont accessibles que via Contribute. Le fichier de connexion clé peut être envoyé par e-mail aux utilisateurs ou enregistré sur l'ordinateur. Lorsqu'il est cliqué, il ouvre automatiquement Contribute qui importe la connexion.
