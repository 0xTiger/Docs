## Qu'est-ce qu'un fichier POT ?

Les fichiers avec l'extension .POT représentent les fichiers de modèle Microsoft PowerPoint créés par les versions PowerPoint 97-2003. Les fichiers créés avec ces versions de Microsoft PowerPoint sont au format binaire par rapport à ceux créés dans les formats de fichiers Office OpenXML à l'aide des versions supérieures de PowerPoint. Les fichiers ainsi générés peuvent être utilisés pour créer des présentations ayant la même mise en page et d'autres paramètres devant être appliqués aux nouveaux fichiers. Ces paramètres peuvent inclure des styles, des arrière-plans, une palette de couleurs, des polices et des valeurs par défaut. Ces fichiers sont générés afin de créer des fichiers modèles prêts à l'emploi pour une utilisation officielle.

## Spécifications du format de fichier ##

Les spécifications de format de fichier pour le format de fichier POT ne sont pas disponibles individuellement par Microsoft. Cependant, le format du modèle est le même que celui du format de fichier binaire [PPT](/fr/presentation/ppt/) et peut être ouvert dans MS PowerPoint à des fins d'édition. Un fichier POT est identifié par un identifiant HEX comme suit :

```
D0 CF 11 E0 A1 B1 1A E1 00 00 00 00
```

Les types MIME associés au format de fichier POT sont :

* application/vnd.ms-powerpoint [officiel]
* application/mspowerpoint
* application/x-mspowerpoint
* application/powerpoint
* application/x-powerpoint
* application/x-dos_ms_powerpnt
*application/pot
* application/x-soffic
