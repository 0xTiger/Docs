## Qu'est-ce qu'un fichier PPS ?

PPS, PowerPoint Slide Show, les fichiers sont créés à l'aide de Microsoft PowerPoint à des fins de diaporama. La lecture et la création de fichiers PPS sont prises en charge par Microsoft PowerPoint 97-2003. La version la plus récente de ce format de fichier est [PPSX ](/fr/presentation/ppsx/) qui est basée sur les normes Office OpenXML. Les fichiers PPS peuvent toujours être lus par les dernières versions de Microsoft PowerPoint, mais les fichiers nouvellement créés ne peuvent être enregistrés qu'au format de fichier PPSX. Lorsqu'un fichier PPS est partagé avec un autre utilisateur et ouvert, il démarre en tant que présentation PowerPoint contrairement au fichier [PPT](/fr/presentation/ppt/) qui s'ouvre en mode modifiable.

## Format de fichier ##

PPS est un format de fichier binaire qui respecte les spécifications détaillées dans le format de fichier [MS-PPT](https://msdn.microsoft.com/en-us/library/office/cc313106(v#office.12).aspx) document. Il se compose d'un certain nombre de flux pour représenter la structure et les données globales du document. Ceux-ci inclus:

* Flux d'utilisateurs actuel
* Flux de documents PowerPoint
* Flux d'images
* Informations récapitulatives et informations récapitulatives sur le document (facultatif)

Des détails complets sur le format de fichier PPT peuvent être trouvés dans le document de spécifications MS-PPT qui s'applique également à PPS.
