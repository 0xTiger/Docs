## À propos de FBX à USDZ

FBX, abréviation de FilmBox, est un format de fichier 3D largement adopté, initialement conçu par Kaydara pour MotionBuilder. Autodesk Inc. en est devenu propriétaire en 2006, consolidant ainsi son statut de principal format d'échange de données 3D utilisé par une multitude d'outils logiciels 3D.

Un fichier portant l'extension .usdz représente une archive ZIP non compressée et non cryptée adaptée au format de fichier [USD](/fr/3d/usd/) (Universal Scene Description). Il héberge des proxys pour différents types de fichiers, notamment des textures et des animations, tous commodément encapsulés dans l'archive.

Si vous êtes un développeur d'applications souhaitant intégrer la fonctionnalité [FBX](/fr/3d/fbx/) à USDZ dans votre application .NET, Java ou Python, vous souhaiterez peut-être explorer les API Aspose.3D comme ressource précieuse pour réaliser cet objectif.
