## Qu'est-ce qu'un fichier DBM ?

Un fichier DBM (Daz Brick Material) est un fichier réseau Shader Mixer créé avec l'application de modélisation de personnages 3D, **DAZ Studio**. Il décrit le matériau du mode 3D c'est-à-dire surface/déplacement/volume intérieur. DAZ vous permet de créer votre propre shader personnalisé en connectant une série de briques ensemble. Cela se combine collectivement en un fichier DBM qui forme un réseau de shaders. C'est le fichier DBM qui spécifie comment un matériau est ombré.

Les applications qui peuvent ** ouvrir les fichiers DBM ** incluent DAZ 3D DAZ Studio 4.

## Format de fichier DBM

Les fichiers DBM sont enregistrés sur le disque au format de fichier [XML](/fr/web/xml/).
