## Qu'est-ce qu'un fichier 3DM ?

Un fichier avec l'extension .3dm est un format de fichier open source utilisé pour les logiciels graphiques 3D. Il contient des modèles 3D ainsi que leurs éléments dépendants tels que des informations sur la surface, les points et les courbes. Le format de fichier 3DM a été développé par l'initiative [openNURBS](https://github.com/mcneel/opennurbs) pour transférer avec précision la géométrie 3D entre les applications. Les fichiers 3DM peuvent être ouverts et convertis avec des applications telles que Rhinoceros, SAP VEViewer, Moment of Inspiration et Right Hemisphere Deep View.

## Format de fichier 3DM

[openNURBS](https://github.com/mcneel/opennurbs), une boîte à outils open source, contient les spécifications du format de fichier 3DM, la documentation, les bibliothèques de code source C++ et les assemblages .NET 2.0 pour lire et écrire le format de fichier.

### Fichiers d'exemple 3DM

Certains exemples de fichiers 3DM peuvent être téléchargés à partir de la section openNURBS [examples](https://github.com/mcneel/opennurbs/tree/7.x/example_files). Ceux-ci peuvent être chargés et affichés à l'aide de la boîte à outils openNURBS.

## Comment lire ou écrire des fichiers 3DM ?

Les bibliothèques [openNURBS](https://github.com/mcneel/opennurbs) permettent à quiconque de lire et d'écrire le format de fichier 3DM sans avoir besoin de Rhino. Il fournit le code source C++ pour une bibliothèque qui lira et écrira des modèles 3D en utilisant le format de fichier openNURBS. Cette boîte à outils fournit également des outils d'évaluation NURBS et des outils élémentaires de manipulation de vues géométriques et 3D, ainsi que le code source de plusieurs exemples de programmes. Les forums d'assistance [Rhinoceros](https://discourse.mcneel.com/c/opennurbs/6) sont un riche contenu et un lieu de discussion pour partager les problèmes rencontrés par la communauté 3DM et obtenir des solutions.
