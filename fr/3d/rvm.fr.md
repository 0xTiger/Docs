## Qu'est-ce qu'un fichier RVM ?

Les fichiers de données **RVM** sont liés à AVEVA PDMS. Le fichier **RVM** est un fichier de projet AVEVA Plant Design Management System. Le système de gestion de conception d'usine (PDMS) d'AVEVA est le système de conception 3D le plus populaire utilisant une technologie centrée sur les données pour la gestion de projets. De nombreuses applications sont disponibles pour ouvrir et convertir des fichiers RVM vers d'autres formats tels que [3DS](/fr/3d/3ds/).

## Format de fichier RVM

Peu de détails peuvent être trouvés sur le format de fichier RVM autre que le fait qu'il peut être enregistré au format binaire et ASCII. Le fichier RVM prend en charge des entités telles que :

* Toute géométrie
* Attributs stockés sur les groupes
* Textures (via le fichier RVS)
* Caméras et pistes de caméra (via le fichier RVS)
* Plans de coupe (via le fichier RVS)
* Signes (via le fichier RVS)
* Balises (via fichier RVS)
* Étiquettes (via le fichier RVS)
* Translucidité (via le fichier RVS)
* Points d'origine PDMS
