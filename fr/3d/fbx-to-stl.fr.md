## À propos de FBX vers STL

FBX, FilmBox, est un format de fichier 3D populaire développé par Kaydara pour MotionBuilder. Il a été acquis par Autodesk Inc en 2006 et constitue désormais l'un des principaux formats de fichiers d'échange de données 3D utilisés par de nombreux outils 3D. [STL](/fr/cad/stl/), abréviation de stéréolithrographie, est un format de fichier interchangeable pour la représentation de la géométrie de surface tridimensionnelle.

Il est courant de convertir [FBX](/fr/3d/fbx/) au format de fichier STL et plusieurs applications sont disponibles en ligne qui peuvent convertir GRATUITEMENT FBX en STL. Si vous êtes un développeur d'applications souhaitant intégrer la fonctionnalité FBX vers STL dans votre application .NET, Java ou Python, vous pouvez envisager d'utiliser les API Aspose.3D à cette fin.
