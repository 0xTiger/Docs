## Qu'est-ce qu'un fichier CTG ?

Un fichier CTG est un fichier catalogue créé sur les appareils photo numériques Canon. Ceux-ci sont créés lorsque des photos sont prises par un appareil photo numérique Canon. Il stocke les informations sur le nombre de photographies/images dans chaque dossier sur le périphérique de stockage de l'appareil photo, c'est-à-dire la carte mémoire. Les fichiers CTG sont stockés dans le dossier CANONMSC sur la carte mémoire. Ces fichiers permettent à votre ordinateur d'indexer les fichiers après les avoir insérés dans l'ordinateur.

## Format de fichier CTG - Plus d'informations

Les fichiers CTG sont enregistrés au format de fichier propriétaire Canon. Ceux-ci sont générés automatiquement et ne doivent pas être modifiés manuellement. L'appareil photo supprime ces fichiers lorsque les photos correspondantes sont effacées par l'utilisateur.
