## Qu'est-ce qu'un fichier VIMRC ?

Un fichier VIMRC est un fichier de paramètres utilisé par l'éditeur de texte Vim. Il stocke les paramètres de l'éditeur qui sont chargés lorsque l'éditeur est ouvert. Les développeurs d'applications et les administrateurs système définissent les paramètres liés à l'apparence du texte, tels que la coloration syntaxique, qui sont enregistrés dans le fichier VIMRC. Ainsi, le fichier VIMRC permet aux utilisateurs de personnaliser les paramètres de l'éditeur.

## Format de fichier VIMRC

Les fichiers VIMRC sont enregistrés sous forme de fichiers texte brut et peuvent être ouverts/modifiés dans n'importe quel éditeur de texte tel que Notepad, Notepad++ et TextEdit. Ils contiennent des instructions qui peuvent être exécutées dans l'éditeur Vim en commençant par un ":", suivi de la commande. Les fichiers VIMRC contiennent des instructions pour les paramètres tels que l'indentation du texte et les couleurs des fenêtres et des polices. Un format de fichier pertinent est [VIM](/fr/misc/vim/).
