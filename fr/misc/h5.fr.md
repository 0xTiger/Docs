## Qu'est-ce qu'un fichier H5 ?

Un H5 est l'un des formats de données hiérarchiques (HDF) utilisé pour stocker une grande quantité de données. Il est utilisé pour stocker une grande quantité de données sous la forme de tableaux multidimensionnels. Le format est principalement utilisé pour stocker des données scientifiques bien organisées pour une récupération et une analyse rapides. H5 a été introduit en tant que format de fichier plus amélioré pour H4. Il a été développé à l'origine par le National Center for Supercomputing Applications et est maintenant pris en charge par le groupe HDF.

## Bref historique du format de fichier HDF

Le format de fichier HDF a été sélectionné par la NASA comme méthode standard de stockage des données scientifiques. Avant cela, la formulation d'une telle norme a commencé en 1987 par le Graphics Foundations Task Force (GFTF) et HDF a été officiellement approuvée après enquête sur 15 formats de fichiers différents.

## Format de fichier HDF5

Les fichiers H5 sont au format de données hiérarchique qui respecte les spécifications du format de fichier HDF5. Ces spécifications définissent la structure globale du fichier HDF5 pour le stockage sur disque, mais les utilisateurs finaux n'ont pas besoin de ces détails sous-jacents.

Les fichiers H5 sont couramment utilisés dans l'aérospatiale, la physique, l'ingénierie, la finance, la recherche universitaire, la génomique, l'astronomie, les instruments électroniques et les domaines médicaux.

### Modèle de données HDF5

Le fichier HDF5 est comme un conteneur composé d'objets de données hétérogènes. Par exemple, ces ensembles de données peuvent être des images, des tableaux, des éléments de texte, des graphiques et même des documents tels que [PDF](/fr/pdf/) et [documents Excel](/fr/spreadsheet/).

## Visionneuse de fichiers H5

Le groupe HDF fournit le [logiciel HDFView](https://www.hdfgroup.org/solutions/hdf5/) qui se compose de l'utilitaire HDFView et du package d'objets Java HDF.
