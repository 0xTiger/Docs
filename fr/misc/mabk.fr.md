## Qu'est-ce qu'un fichier MABK ?

Un fichier MABK est un fichier Motorola Android Backup créé par Lenovo Android Management.
utilitaire, Lenovo Moto Smart Assistant. Il s'agit d'un fichier de sauvegarde contenant une copie des applications,
contacts, messages texte et fichiers multimédias stockés sur le téléphone. Le fichier MABK ne s'enregistre pas
paramètres de l'application de l'utilisateur ifnormation.

## Format de fichier MABK

Les fichiers MABK sont générés à l'aide de l'application Lenovo Moto Smart Assistant (LMSA) sur
Téléphone MOtorola, téléphone Lenovo et tablettes Lenovo. C'est une application Windows et stocke le
sauvegarder les données sur les appareils des utilisateurs. L'application, une fois exécutée, se connecte à l'appareil de l'utilisateur et
prend crée le fichier MABK lorsque l'option **Sauvegarde et restauration** est sélectionnée.
Les utilisateurs finaux peuvent enregistrer les fichiers MABK sur leur PC pour les utiliser ultérieurement s'ils ont besoin de restaurer le
contenu du fichier de sauvegarde. Les utilisateurs sauvegardent généralement tous les fichiers disponibles sur l'appareil, tels que
contacts, messages texte, images, audios, vidéos et applications.

## Comment ouvrir les fichiers MABK ?

Vous pouvez utiliser un fichier MABK pour la restauration ou le transfert de données sur votre téléphone Motorola,
Téléphone Lenovo ou tablette Lenovo utilisant Lenovo Moto Smart Assistant pour Windows. À
accomplir ceci :
1. Connectez votre appareil mobile à votre PC.
2. Lancez Lenovo Moto Smart Assistant.
3. Accédez à Sauvegarde et restauration, puis choisissez Restaurer.
4. Importez votre fichier MABK.
5. Sélectionnez Restaurer et suivez les instructions à l'écran pour terminer la restauration des données ou
processus de transfert.
De plus, vous pouvez accéder manuellement aux fichiers stockés dans un fichier MABK. Par exemple, vous pouvez
extrayez les images contenues dans le fichier MABK et visualisez-les sur votre PC. Pour extraire des fichiers de
un fichier MABK :
1. Ouvrez l'Assistant intelligent Lenovo Moto.
2. Accédez à Sauvegarde et restauration.
3. Sélectionnez Restaurer.
4. Importez votre fichier MABK.
5. Choisissez Extraire et suivez les invites à l'écran pour extraire les fichiers de votre fichier MABK.
