## Qu'est-ce qu'un fichier XMZX?

Un fichier XMZX est un fichier de test d'examen permettant d'effectuer des tests et est utilisé par le logiciel ExamSoft SofTest. L'application peut être utilisée sur Windows et macOS. Il contient des questions d'un examen que l'examinateur peut poser aux étudiants sur un ordinateur équipé du logiciel SofTest. Les fichiers XMZX sont envoyés aux examinateurs pour effectuer le test. Il est chargé dans le logiciel SofTest et le candidat ou postulant passe le test. Une fois le test terminé, le fichier XMDX finalisé est automatiquement téléchargé sur les serveurs cloud d'ExamSoft.

## Format de fichier XMZX - Plus d'informations

Les fichiers XMZX sont utilisés par la plateforme d'évaluation numérique ExamSoft, utilisée par diverses organisations pour effectuer des tests de certification. La plateforme dispose de ses propres serveurs cloud en ligne sur lesquels tous les tests terminés sont téléchargés automatiquement. Les tests sont effectués soit dans un centre d'examen, soit les examinateurs peuvent également passer le test des candidats sur leur propre ordinateur. ExamSoft est disponible sur les systèmes d'exploitation Windows et macOS. [XSMX](/fr/misc/xsmx.md/) est un autre format de fichier de test d'examen d'ExamSoft.

## Comment ouvrir un fichier XMZX?

Vous pouvez ouvrir un fichier XMZX dans le logiciel ExamSoft SofTest uniquement car il s'agit d'un format de fichier propriétaire. Le logiciel SofTest a la possibilité de passer un examen qui est utilisé pour démarrer l'examen en sélectionnant le fichier XMZX.
