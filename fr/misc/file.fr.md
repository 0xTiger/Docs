## Qu'est-ce qu'un fichier?

Un fichier .File est un fichier générique et ne se lie à aucun format spécifique. Différents logiciels qui ne reconnaissent pas le format de fichier de l'extension particulière de fichier. Par exemple, si un navigateur rencontre un type de fichier qu'il ne reconnaît pas, il pourrait inciter l'utilisateur à choisir une application pour ouvrir le fichier ou l'enregistrer avec un nom générique. Dans certains cas, l'absence d'une extension reconnaissable peut entraîner une confusion, mais il n'est pas courant que les navigateurs ou les systèmes d'exploitation ajoutent automatiquement une extension ".

## Comment ouvrir le fichier.

Si vous connaissez le format du fichier .File, vous pouvez modifier son extension en format spécifique et l'ouvrir dans le logiciel connexe. Par exemple, si le format de fichier interne de .File est le format de fichier DOCX, la modification de l'extension .File en .docx vous permettra d'ouvrir le fichier dans Microsoft Word.

De même, si le format interne d'extension .File est le format PDF, la modification du fichier .pdf vous permettra d'ouvrir le fichier dans Adobe Reader.

Si vous ne connaissez pas le format de fichier interne ou son contenu interne, n'essayez pas de l'ouvrir car il peut contenir des virus. Identifiez d'abord le format à l'aide d'un outil d'identification du format de fichier, puis ouvrez-le.
