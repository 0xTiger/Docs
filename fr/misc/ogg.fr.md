## Qu'est-ce qu'un fichier OGG ?

Un fichier OGG est un fichier graphique créé avec le logiciel Origin. Il offre la possibilité de stocker différents types de graphiques pour afficher les données importées pour l'analyse des données. Les types de graphiques pris en charge par le type de fichier OGG incluent la colonne, la barre, le secteur, la cascade, la 3D, la polaire et le symbole. Le logiciel Origin a été développé par OriginLab et permet aux utilisateurs d'effectuer une analyse de données et de générer une sortie sous forme de graphiques.

## Format de fichier OGG

Les fichiers OGG contiennent des graphiques pour la représentation des données importées. Lorsqu'un fichier OGG est ouvert, il affiche le graphique dans une fenêtre séparée dans Origin ou Origin Viewer qui est disponible gratuitement par rapport au logiciel Origin. Les informations de métadonnées telles que le titre et les couleurs d'un graphique sont également stockées dans le fichier OGG.

## À propos du logiciel OriginLab Origin

Le logiciel Origin offre également la possibilité d'être utilisé comme serveur d'automatisation. Dans ce cas, des applications clientes telles que LabVIEW, Microsoft Excel, Python ou des applications développées sur mesure en C#/VB/VC peuvent échanger des données avec Origin. [Python_to_OriginLab](https://github.com/jsbangsund/python_to_originlab) est une API Python permettant d'automatiser l'envoi de données et les fonctions de traçage de base dans OriginLab.
