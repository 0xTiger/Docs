## Qu'est-ce qu'un fichier VIM ?

Un fichier VIM est un fichier de configuration/paramètres utilisé par le programme d'édition de texte connu sous le nom de Vim. Il stocke les paramètres de l'éditeur tels que la coloration syntaxique et est utilisé par les développeurs d'applications pour configurer l'apparence de l'éditeur lorsqu'il est ouvert. L'éditeur de texte Vim est hautement configurable et conçu pour créer et modifier efficacement du texte. Il est livré sous le nom de "vi" avec la plupart des systèmes UNIX.
