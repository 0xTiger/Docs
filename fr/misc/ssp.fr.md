## Qu'est-ce qu'un fichier SSP ?

Un fichier SSP est un fichier de projet créé par le logiciel de création d'économiseurs d'écran Windows connu sous le nom de Screensaver Producer par Axialis Software. Il peut contenir plusieurs types différents d'éléments d'économiseur d'écran tels que des sprites, des films Flash, des diaporamas de photos et des vidéos. Un fichier SSP peut comprendre du texte, des images, différentes formes et des paramètres d'animation utilisés pour afficher l'animation dans le fichier SSP. Les fichiers SSP peuvent être ouverts avec Axialis Screensaver Producer.

## Format de fichier SSP - Plus d'informations

Les fichiers SSP sont stockés sur disque sous forme de fichiers binaires sans aucune information publique disponible sur leur structure de format de fichier interne. Axialis Screensaver Producer Outil de création d'économiseurs d'écran professionnel pour Windows qui vous permet de créer facilement des économiseurs d'écran de haute qualité pour toutes les versions de Windows jusqu'à Windows 10. Il permet de créer des packages d'économiseurs d'écran installables qui peuvent être facilement distribués en ligne aux utilisateurs.

Les fichiers SSP peuvent être exportés au format de fichier [SCR](/fr/system/scr/).
