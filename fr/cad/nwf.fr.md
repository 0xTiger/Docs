## Qu'est-ce qu'un fichier NWF ?

Un fichier avec l'extension .nwf est un format de fichier CAO propriétaire d'Autodesk qui contient des index de tous les fichiers de modèle utilisés par le projet. Il contient des pointeurs vers les éléments ajoutés aux fichiers modèles pour référence. Les fichiers NWF contiennent également des informations telles que les points de vue et les lignes rouges. L'avantage d'utiliser le format de fichier NWF est le reflet immédiat des changements dans les fichiers CAO d'origine [par exemple DWG](/fr/cad/dwg/) en raison des pointeurs associés. L'utilisation d'un NWF est fortement recommandée pour les tests de collision. Le fichier NWF enregistre les résultats de conflit d'origine et après modification des fichiers CAO, les fichiers CAO révisés seront ouverts lors de la réouverture du fichier NWF.

## Format de fichier NWF

Les détails du format de fichier interne de NWF ne sont pas disponibles car il s'agit d'un format de fichier propriétaire d'Autodesk.
