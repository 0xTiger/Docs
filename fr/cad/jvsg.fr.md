## Qu'est-ce qu'un fichier JVSG ?

JVSG est un fichier CAO qui peut être lancé sous Windows. C'est un fichier au format binaire créé par JVSG. Les fichiers de projet de vidéosurveillance utilisent principalement le suffixe de nom de fichier JVSG. Ce fichier peut stocker la bande passante, les emplacements, les angles de vue et les calculs du disque dur d'une caméra avec des informations sur sa conception CCTV. Les fichiers de projet JVSG offrent une analyse et une documentation avancées grâce au rendu dans les vues 3D.

Les réseaux de caméras de sécurité de différentes entreprises qui ont besoin de caméras de surveillance peuvent également être conçus avec les fichiers de projet JVSG. Le logiciel « CCTV design tool » est adapté à la gestion des fichiers JVSG.
