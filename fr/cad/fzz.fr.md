## Qu'est-ce qu'un fichier FZZ ?

L'extension de fichier **.fzz** est associée à **Fritzing**, une initiative matérielle open source qui rend l'électronique accessible à tous en tant que matériel créatif. Fritzing permet aux utilisateurs de concevoir et de documenter leurs projets électroniques. Le **fichier fzz** est un **fichier de projet partageable Fritzing**, ce qui signifie qu'il contient des informations sur les composants, les connexions et la disposition d'un projet électronique particulier créé à l'aide de Fritzing.

## Fichier FZZ - Bref aperçu

Voici un bref aperçu de ce à quoi vous pouvez vous attendre dans le fichier « .fzz » :

1.  **Informations schématiques :** Le fichier stockera des informations sur les composants utilisés dans votre circuit, leurs connexions et leurs propriétés.
    
2.  **Disposition de la planche à pain :** Fritzing vous permet de créer une représentation physique de votre circuit sur une planche à pain virtuelle. Le fichier .fzz contiendra des données sur la manière dont les composants sont disposés sur la maquette.
    
3.  **Disposition du PCB (le cas échéant) :** Si vous avez créé une disposition de PCB (circuit imprimé) personnalisée pour votre projet dans Fritzing, le fichier .fzz stockera des informations sur le placement et les connexions des composants sur le PCB.
    
4.  **Metadata:** The file may also include metadata such as project name, author information and other details entered by user.

## Comment ouvrir un fichier FZZ

**Pour ouvrir un fichier .fzz, vous devez avoir installé Fritzing sur votre ordinateur.**

1. Ouvrez l'application Fritzing.
2. Dans Fritzing, allez dans le menu Fichier.
3. Choisissez l'option Ouvrir dans le menu.
4. Localisez le fichier .fzz sur votre ordinateur à l'aide du navigateur de fichiers.
5. Sélectionnez le fichier .fzz que vous souhaitez ouvrir et cliquez sur le bouton Ouvrir.
