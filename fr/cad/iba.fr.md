## Qu'est-ce qu'un fichier IBA ?

L'extension de fichier IBA est créée par Lectra. Ce format de fichier est également appelé fichier Lectra Clothing Design Pieces et appartient à la catégorie des fichiers CAO. Toutefois, le format de fichier principal utilisé par Lectra Modaris est le format de fichier MDL, qui est un format propriétaire développé par Lectra. Les fichiers MDL contiennent des informations sur les motifs, marqueurs et modèles 2D et 3D créés et modifiés dans le logiciel Modaris.

## Relation avec Lectra Modaris

IBA est un fichier CAO lié au logiciel Lectra Modaris. Lectra Modaris est un logiciel de modélisme 2D et 3D développé par Lectra, une entreprise technologique spécialisée dans les logiciels, matériels et services pour l'industrie de la mode. Lectra Modaris est conçu pour aider les créateurs de mode, les modélistes et les fabricants à créer et développer des patrons de manière plus efficace et plus précise, avec des fonctionnalités telles que l'étalonnage des patrons, la création de marqueurs et le prototypage virtuel. il utilise principalement des fichiers CAO (Conception Assistée par Ordinateur) aux formats suivants :

1. Format de fichier propriétaire de Lectra, portant l'extension .mdl. Ce format est utilisé pour enregistrer les motifs, marqueurs et modèles créés dans le logiciel Modaris.
2. Format de fichier DXF (Drawing Interchange Format), largement utilisé dans l'industrie de la CAO pour échanger des données entre différentes applications logicielles. Le format DXF est utile pour exporter des pièces de patron 2D depuis Modaris.
3. Format de fichier AAMA (American Apparel Manufacturers Association), qui est un format standard pour les fichiers de modèles utilisés dans l'industrie de la mode. Modaris peut exporter des modèles au format AAMA pour les utiliser dans d'autres applications logicielles.

Il convient de noter que Modaris prend également en charge plusieurs autres formats de fichiers, notamment Gerber (ZIP), Lectra (MOD), Lectra (CIT) et Lectra (LAY). Cependant, ces formats de fichiers sont moins couramment utilisés dans l’industrie de la mode.
