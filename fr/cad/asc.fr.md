## Qu'est-ce qu'un fichier ASC ?

Les fichiers ASC sont des fichiers texte Autodesk qui stockent des informations sur les objets de votre dessin, notamment leur emplacement, leur taille et leur couleur. Cela en fait un choix populaire pour échanger des dessins entre différents programmes [CAO](/fr/cad/). Les fichiers ASC peuvent être ouverts avec les produits logiciels d'Autodesk. Autodesk offre la possibilité d'importer des fichiers ASC depuis l'application. Les fichiers ASC peuvent être ouverts dans n'importe quel éditeur de texte tel que Microsoft Notepad, Notepad ++ et Apple TextEdit.

## Format de fichier ASC

Les fichiers ASC sont stockés sur disque au format de fichier texte ASCII brut. Ceux-ci sont lisibles par l'homme.

### Exemple de fichier ASC

```
ncols 200
nrows 200
xllcorner 119000.00
yllcorner 158000.00
cellsize 5.00
nodata_value -999999
```
