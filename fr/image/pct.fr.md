## Qu'est-ce qu'un fichier PCT ?

Un fichier PCT est un format de fichier Macintosh Picture Image qui était le format de fichier par défaut pour le programme Apple QuickDraw. Il a été développé par Apple mais après l'arrêt du programme QuickDraw, il n'est plus utilisé. **[PDF](/fr/pdf/)**, dans son ensemble, a remplacé le PCT et les formats de fichiers similaires. Les fichiers PCT peuvent également être parfois trouvés avec l'extension de fichier .pict, bien que PCT et PICT aient le même format de fichier.

## Format de fichier PCT - Plus d'informations

Les fichiers PCT sont enregistrés sur le disque en tant que fichiers binaires au ** format de fichier pict ** et les détails de leur format de fichier interne ne sont pas disponibles. Les applications pouvant ouvrir les fichiers PCT incluent XnView, GIMP, Corel PaintShop Pro et Apple Preview.

### Types de format de fichier PCT

Les données d'image d'un fichier Macintosh PCT peuvent être enregistrées au format original "PICT 1" ou au format "PICT 2".

**PICT 1** - Ce type de format de fichier PICT peut stocker huit couleurs
**PICT 2** - Ce type de format de fichier PICT prend en charge des milliers de couleurs avec des images 24 et 32 bits
