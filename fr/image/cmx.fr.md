## Qu'est-ce qu'un fichier CMX ?

Les fichiers avec l'extension .cmx sont au format de fichier image Corel Exchange (également connu sous le nom de Corel Metafile Exchange) qui est utilisé comme présentation par les applications CorelSuite. Il contient des données d'image sous forme de graphiques vectoriels ainsi que des métadonnées qui décrivent l'image. Les fichiers CMX peuvent être ouverts par CorelDraw, Corel Presentations, Paint Shop Pro et certaines versions d'Adobe Illustrator. Les fichiers CMX peuvent être convertis dans d'autres formats tels que [JPG](/fr/image/jpeg/) et [EPS](/fr/page-description-language/eps/).

## Format de fichier CMX
Le format de fichier CMX est basé sur le format RIFF (Resource Interchange File Format) et prend en charge les formats binaires Intel et Motorola. Le fichier CMX utilise les types MIME suivants :
* image/x-cmx
* application/x-cmx
* demande/cmx
* dessin/cmx

### En-tête de format de fichier
Les fichiers CMX commencent par "RIFF" ou "RIFX" et ont "CMX1" au décalage 8.
