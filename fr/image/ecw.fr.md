## Qu'est-ce qu'un fichier ECW ? ##
Un fichier ECW consiste en une image conçue pour être utilisée avec des données géospatiales. L'image contenante est en fait une image compressée créée au format Enhanced Compression Wavelet. Les fichiers ECW sont généralement utilisés pour stocker des projections de cartes satellites et des images aériennes. Pour prendre en charge les applications géospatiales, les informations de projection cartographique peuvent être intégrées dans les fichiers ECW.

## Format de fichier ECW
ECW est un format d'image propriétaire basé sur la compression par ondelettes ; optimisé pour l'imagerie satellitaire et la photographie aérienne. Ce format appartient maintenant à Intergraph, qui fait partie d'Hexagon AB, mais il a essentiellement été développé par Earth Resource Mapping. ECW est un format de compression avec perte qui comprime les images de grande taille avec un contraste fin en alternance tout en conservant leur qualité visuelle.
 

### Propriétés
Voici quelques propriétés typiques du format ECW :
- Les informations de projection cartographique peuvent être ajoutées au format de fichier ECW pour prendre en charge les applications géospatiales.
- Les données d'image jusqu'à 65 535 bandes, y compris les calques ou les couleurs, peuvent être compressées au format de fichier ECW à un débit supérieur à 25 Mo par seconde.
- Le protocole ECWP est un protocole de streaming efficace utilisé pour transmettre des images ECW et JPEG2000 sur des réseaux.
- Un SDK rapide prenant en charge ECW et JPEG2000 est disponible gratuitement pour l'implémentation de bureau pour Linux, Windows et MacOSX
