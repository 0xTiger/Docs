## Qu'est-ce qu'un fichier CDX ?

Les fichiers CDX sont créés par CorelDRAW, un programme d'édition de graphiques vectoriels. Ces fichiers sont une version compressée des fichiers de dessin .CDR, qui constituent le principal format de fichier utilisé par CorelDRAW. Le but de la création de fichiers CDX est de réduire la quantité d'espace de stockage requis pour un dessin.

## À propos de CorelDRAW

CorelDRAW est un programme d'édition de graphiques vectoriels populaire qui fait partie du progiciel CorelDRAW Graphics Suite. Lorsque vous créez un dessin dans CorelDRAW et que vous l'enregistrez, le format de fichier par défaut est généralement .CDR (CorelDRAW Drawing). Cependant, pour rendre ces fichiers plus faciles à gérer en termes de stockage et à partager, CorelDRAW permet aux utilisateurs d'enregistrer des dessins dans un format compressé appelé CDX (CorelDRAW Compressed File).

Le format de fichier CDX sert de version compressée du fichier .CDR original. La compression est une technique utilisée pour réduire la taille du fichier en codant les informations plus efficacement. Ceci est particulièrement avantageux pour les dessins comportant des détails et des graphiques complexes, car cela permet de minimiser la taille du fichier sans compromettre la qualité du contenu visuel.

Le principal avantage des fichiers CDX est leur petite taille, ce qui les rend idéaux pour le partage par courrier électronique ou l'archivage sur des supports de stockage externes où l'espace peut être limité. Lorsque vous devez distribuer des dessins CorelDRAW mais que vous souhaitez conserver une taille de fichier gérable, opter pour le format CDX peut être une solution pratique. Il permet aux utilisateurs de maintenir l'intégrité de leur travail graphique tout en facilitant le transfert de fichiers et la gestion du stockage.

## Comment ouvrir le fichier CDX ?

Les fichiers CDX peuvent être ouverts par les programmes suivants

- **CorelDRAW Graphics Suite 2023** (essai gratuit) pour Windows
