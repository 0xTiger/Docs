## Qu'est-ce qu'un fichier CLIP ?

Un fichier CLIP est un fichier image raster généré par l'application de dessin 2D Clip Studio Paint. Il est similaire au fichier [PSD](/fr/image/psd/) d'Adobe et contient plusieurs calques pour gérer différents éléments graphiques sous forme de calques tels que du texte, des images, un stylo, un crayon, des pinceaux et des formes. Le format de fichier CLIP est utilisé pour concevoir un certain nombre de types de graphiques différents tels que des illustrations et des animations. L'application Clip Studio Paint a été développée par une société japonaise CELSYS, Inc. qui a été créée en mai 1991.

## Format de fichier CLIP - Plus d'informations

Le format de fichier CLIP a été introduit par CELSYS pour remplacer le format de fichier LIP d'origine. Il stocke les éléments graphiques sous la forme de calques qui peuvent être modifiés pour les modifications. C'est également le format de fichier par défaut pour l'application [CLIP Studio Paint](https://www.celsys.co.jp/en/service/studio_paint/) qui est disponible pour les plateformes Windows, MacOS, Android et iOS.

## Types de dessin pris en charge

Le format de fichier CLIP peut stocker des animations et des illustrations qui peuvent être utilisées dans une variété de types de travaux différents.
