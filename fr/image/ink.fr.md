## Qu'est-ce qu'un fichier INK ?

Le format de fichier « .ink » est associé à **Pantone**, une entreprise bien connue dans le domaine de la correspondance des couleurs et de l'impression. Un **Fichier de référence Pantone** au format « .ink » contient généralement des informations sur les couleurs d'encre Pantone ; ces fichiers sont utilisés dans l’industrie de l’imprimerie pour garantir une reproduction précise et une cohérence des couleurs.

Pantone est connu pour son **Pantone Matching System (PMS)**, qui est un système de reproduction des couleurs standardisé largement utilisé dans diverses industries, notamment la conception graphique, l'impression et la fabrication ; le PMS fournit un ensemble de couleurs d'encre, chacune identifiée par un code spécifique tel que « PMS 185 » pour une nuance de rouge spécifique.

## Que contient le fichier INK ?

Le format de fichier « .ink » peut stocker des données liées aux couleurs d'encre Pantone telles que :

1. **Codes couleur :** Les couleurs de l'encre Pantone sont identifiées par des codes uniques et ces codes sont généralement inclus dans le fichier.
    



2. **Valeurs de couleur :** Le fichier peut contenir des valeurs de couleur telles que **RVB (rouge, vert, bleu)**, **CMYK (Cyan, Magenta, jaune, clé/noir)** et **Lab. (CIELAB)** valeurs pour spécifier comment la couleur doit être reproduite dans différents espaces colorimétriques.
    



3. **Échantillons de couleurs :** Les couleurs d'encre Pantone ont souvent des échantillons correspondants que les concepteurs et les imprimeurs peuvent utiliser à titre de référence.
    



4. **Informations sur la couleur :** Des informations supplémentaires sur la couleur de l'encre, telles que son nom, sa description et ses notes d'application, peuvent être incluses dans le fichier « .ink ».

## Comment ouvrir le fichier INK?

Les programmes qui ouvrent les fichiers INK incluent

- **CorelDRAW** (essai gratuit) pour Windows

CorelDRAW est un logiciel de graphiques vectoriels qui prend en charge les couleurs Pantone et fournit des outils pour travailler avec elles.

**Sous-type :** Fichiers d'images raster
