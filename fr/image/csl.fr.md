## Qu'est-ce qu'un fichier CSL ?

Un fichier Corel Symbol Library (CSL) est une bibliothèque de symboles utilisés dans le logiciel CorelDRAW. Il stocke des symboles qui peuvent être utilisés dans d'autres fichiers. Les fichiers CSL peuvent être stockés localement ou sur un réseau pour les rendre accessibles pour un déploiement et une gestion faciles. Ceux-ci sont enregistrés avec le fichier CorelDRAW avec toutes les instances de symboles utilisées dans le document.

## Format de fichier CSL

Les fichiers CSL sont enregistrés au format de fichier propriétaire CorelDRAW. Étant une collection de symboles, les symboles contenus dans un fichier CSL peuvent être importés et ajoutés aux dessins d'un projet.
