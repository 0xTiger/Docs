## Qu'est-ce qu'un fichier AGP ?

Un fichier AGP est un fichier image 32 bits avec plusieurs couches. Il est créé avec le logiciel ArtGem qui fournit les fonctionnalités de travail avec des images et est similaire à Adobe Photoshop dans les fonctionnalités de base. AGP est un fichier de projet qui contient tous les paramètres de création de l'image finale.

Les fichiers AGP peuvent être ouverts avec RL Vision ArtGem. Cependant, ArtGem a longtemps été abandonné après avoir été vendu.

## Format de fichier AGP

Les fichiers AGP ont été enregistrés au format de fichier propriétaire ArtGem. La structure de fichier interne des fichiers AGP n'est pas disponible publiquement pour la référence du développeur.
