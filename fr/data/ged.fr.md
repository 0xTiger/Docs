## Qu'est-ce qu'un fichier GED ?

Un **fichier GEDCOM (Genealogical Data Communication)**, généralement avec l'extension « .ged », est un format de fichier standard utilisé pour échanger des données généalogiques entre différents logiciels de généalogie ; Les fichiers GEDCOM sont des fichiers en texte brut contenant des informations sur des individus, des familles, des événements et d'autres données généalogiques ; le but de GEDCOM est de fournir un format commun pour l'échange d'informations généalogiques.

Les fichiers GEDCOM ont généralement l'extension de fichier « .ged » ; cette extension est utilisée pour indiquer que le fichier suit la **norme GEDCOM** pour l'échange de données généalogiques ; Ainsi, lorsque vous enregistrez ou exportez vos données généalogiques à partir d'un logiciel de généalogie, le fichier résultant reçoit souvent un nom tel que « votre nom de fichier.ged ».

Par exemple:

- votrearbrefamilial.ged
- john_doe_family.ged
- ancestry_data.ged

## Points clés sur les fichiers GEDCOM

Voici quelques points clés sur les fichiers GEDCOM :

1.  **Format de texte brut :** Les fichiers GEDCOM sont des fichiers de texte brut ; ce qui signifie qu'ils peuvent être ouverts et visualisés avec n'importe quel éditeur de texte ; ce format facilite le partage et la compatibilité entre différentes applications logicielles de généalogie.
    
2.  **Structure :** Les fichiers GEDCOM ont une structure hiérarchique ; les données sont organisées en enregistrements ; chaque enregistrement possède une balise spécifique qui indique le type d'informations qu'il contient. Par exemple, le nom d'un individu peut être représenté par la balise « NOM ».
    
3.  **Individuals and Families:** GEDCOM files typically include information about individuals such as names, dates of birth and places of birth; they also include details about family relationships, marriages and other events.
    
4.  **Interopérabilité logicielle :** Puisque GEDCOM est un format standardisé ; il permet aux utilisateurs de déplacer leurs données généalogiques entre différents logiciels de généalogie ; ceci est particulièrement utile pour les utilisateurs qui peuvent passer à une autre application logicielle ou partager leurs recherches avec d'autres.
    
5.  **Version :** GEDCOM a connu différentes versions, chaque version introduisant des améliorations et de nouvelles fonctionnalités ; la version la plus couramment utilisée était GEDCOM 5.5. 

## Comment ouvrir le fichier GED?

Pour ouvrir un fichier GED, vous avez généralement besoin d'un logiciel de généalogie prenant en charge le format GEDCOM. Certaines options de logiciels de généalogie populaires incluent :

- **Créateur d'arbre généalogique**
- **Ancestry.com**
- **Mon héritage**
- **Gramps** (gratuit et open source)
- **Arbre généalogique hérité**

Une fois que vous avez choisi votre logiciel de généalogie, recherchez une option comme « Importer », « Ouvrir GEDCOM » ou « Importer depuis un fichier » dans le menu ou la barre d'outils.

Étant donné que les fichiers GED sont enregistrés au format texte, vous pouvez également ouvrir les fichiers avec un éditeur de texte brut, par exemple

- **Bloc-notes**
- **Bloc-notes++**
- **Apple TextModifier**
- N'importe quel éditeur de texte
