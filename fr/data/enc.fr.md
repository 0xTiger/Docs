## Qu'est-ce qu'un fichier ENC ?

Un **fichier ENC** est lié à la **Carte électronique de navigation (ENC)** qui est une carte vectorielle numérique utilisée dans les systèmes de cartographie électronique pour la navigation maritime ; Les fichiers ENC contiennent des données géospatiales, telles que des courbes de profondeur, des aides à la navigation et d'autres informations nécessaires à une navigation sûre. ENC est un format normalisé défini par l'**Organisation hydrographique internationale (OHI)** dans la **norme S-57**.

La norme S-57 précise la structure et le contenu des fichiers ENC ; ces fichiers ont généralement l'extension de fichier « .enc ». Les fichiers ENC sont utilisés par les **systèmes d'affichage et d'information de cartes électroniques (ECDIS)** sur les navires pour la navigation ; Les systèmes ECDIS utilisent les données ENC pour afficher des cartes électroniques, fournissant aux navigateurs des informations en temps réel pour faciliter une navigation sûre et efficace.

## Format de fichier ENC - Plus d'informations

Les **fichiers ENC** sont des **fichiers binaires** qui stockent les informations de manière structurée, permettant un stockage et une récupération efficaces des données de navigation ; le contenu des fichiers ENC comprend des informations sur les côtes, les isobathes, les bouées, les feux et autres caractéristiques de navigation ; les données sont organisées en différentes couches de données et chaque entité est représentée sous la forme d'un objet vectoriel avec des attributs associés.

Les fichiers ENC sont régulièrement mis à jour par les services hydrographiques pour refléter les changements dans les informations de navigation, garantissant ainsi que les navigateurs ont accès aux données les plus récentes et les plus précises pour leurs itinéraires.

## À propos de la norme S-57

La **norme S-57** est une spécification internationale pour le codage, le stockage et l'échange de cartes électroniques de navigation (ENC) ; il a été développé par l'Organisation hydrographique internationale (OHI) pour fournir un format standardisé pour les cartes numériques utilisées dans les systèmes d'affichage et d'information des cartes électroniques (ECDIS) pour la navigation maritime.

## Comment ouvrir le fichier ENC?

Pour ouvrir les fichiers de cartes électroniques de navigation (ENC), vous avez généralement besoin d'un logiciel de navigation spécialisé prenant en charge la norme S-57 et le format de fichier ENC.

Certains logiciels de navigation populaires incluent :

-OpenCPN
-MaxSea
- SeaClear
- Transas iSailor
- TEMPS ZÉRO
