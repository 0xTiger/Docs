## Qu'est-ce qu'un fichier SID ?

Un fichier SID est un fichier de sauvegarde de jeu créé par l'application Steam qui est utilisé pour télécharger et gérer les **jeux vidéo Valve** sur le système d'exploitation Windows. Il stocke les données de jeu dans un format compressé et peut être utilisé pour sauvegarder des jeux sur des supports de stockage tels que des DVD et des périphériques de stockage USB. Le fichier SID est créé avec l'utilitaire de sauvegarde Steam et est similaire au format de fichier [CSD](/fr/game/csd/) pour stocker la sauvegarde du jeu.

## Format de fichier SID - Plus d'informations

Les fichiers SID sont stockés sur le disque sous forme de fichiers binaires. Il est toujours accompagné d'un fichier .sis et d'un fichier .sim qui sont utilisés pour extraire les informations de métadonnées pour restaurer les données de sauvegarde du jeu.
