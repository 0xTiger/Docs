## Qu'est-ce qu'un fichier USX ?

Un fichier USX contient les maillages d'Unreal Engine qui sont des objets/modèles 3D préconçus utilisés pour générer l'environnement. Les modèles du fichier USX ont une position, une orientation et des tailles différentes, mais celles-ci restent statiques. Les maillages statiques peuvent être optimisés pour les performances et peuvent être utilisés pour créer différents niveaux de détail (LOD) afin d'améliorer les performances du jeu. Ils peuvent également être utilisés pour créer des volumes de collision et peuvent être utilisés dans le cadre de simulations physiques.

Les fichiers USX peuvent être créés avec l'éditeur de niveau Unreal "UnrealEd".

## Format de fichier USX - Plus d'informations

Les fichiers USX sont enregistrés sur le disque sous forme de fichiers binaires. Les détails de la structure de fichier interne du format de fichier USX ne sont pas disponibles publiquement pour la référence du développeur.
