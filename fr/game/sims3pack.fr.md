## Qu'est-ce qu'un fichier SIMS3PACK ?

Un fichier SIMS3PACK est un package de jeu créé pour le jeu de simulation de vie ouvert, **Les Sims 3**. Juste au cas où vous n'auriez pas joué à ce jeu, Sims 3 représente les joueurs participants dans un environnement simulé en tant que personnes virtuelles appelées **Sims**. Le fichier SIMS3PACK contient en fait des informations représentant des parties de l'environnement virtuel telles que les personnes "Sim", les vêtements et les bâtiments.

## Format de fichier SIMS3PACK

Les fichiers SIMS3PACK sont enregistrés au format de fichier binaire, sans aucune information disponible sur la structure interne du fichier. Cependant, il existe des éditeurs de packages Sims 3 tiers disponibles qui peuvent être utilisés pour la génération de fichiers SIMS3PACK. Ces fichiers SIMS3PACK peuvent être chargés dans le jeu en double-cliquant simplement dessus, à condition que le jeu soit installé sur le système.

### Comment installer le fichier de package SIMS3PACK ?

Les Sims3Packs ont l'extension .sims3pack et peuvent être installés en suivant les étapes suivantes.

* Téléchargez et extrayez votre CC dans Mes Documents\Electronic Arts\Les Sims 3\Téléchargements\.
* Pour installer un seul Sims3Pack, double-cliquez dessus et le lanceur s'ouvrira et l'installera tout seul.

## À propos des Sims 3

Sims 3 est un jeu de simulation de vie d'Electronic Arts (EA) et est populaire parmi les gens pour générer des communautés en ligne dans le jeu. Le jeu est disponible pour jouer sur Mac OS, Windows, Android et iOS.
