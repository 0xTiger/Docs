## Qu'est-ce qu'un fichier W3N ?

Un fichier avec l'extension .w3n est un fichier de cartes de campagne utilisé par le jeu [Warcraft III : The Frozen Throne](https://en.wikipedia.org/wiki/Warcraft_III:_The_Frozen_Throne) qui est une extension de Warcraft III, un tout temps célèbre jeu vidéo. Le fichier W3N contient des cartes de campagne utilisées pour créer des campagnes personnalisées à l'aide de la série de données de missions disponibles. Le format est souvent mal compris car les fichiers de carte et les utilisateurs ne comprennent pas pourquoi leurs fichiers W3N ne sont pas chargés dans Frozen Throne. Les fichiers W3N peuvent être ouverts avec Blizzard Warcraft 3 : The Frozen Throne.

## Format de fichier W3N - Plus d'informations

Les fichiers W3N sont stockés sur disque sous forme de fichiers binaires et leur format de fichier interne n'est pas disponible publiquement. Le composant Campaign Editor de Warcraft III World Editor permet cependant de créer des fichiers W3N.

## Comment installer les fichiers W3N ?

Ne vous y trompez pas en plaçant les fichiers W3N dans le dossier "maps" de votre ordinateur. Les fichiers W3N ne sont pas des fichiers cartographiques, mais des campagnes qui peuvent être utilisées pour créer des missions. Par conséquent, pour charger ces fichiers, ceux-ci doivent être placés dans le dossier "campagnes" et non dans le dossier "cartes".
