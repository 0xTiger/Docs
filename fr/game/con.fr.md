## Qu'est-ce qu'un fichier CON ?

Un fichier CON est un fichier de configuration utilisé par le jeu vidéo de tir à la première personne **Battlefield 1942**. Il contient des informations de configuration chargées et utilisées par le jeu. Il existe plusieurs fichiers .con qui définissent différents types d'objets tels que les actions de bot, les dégâts d'armes, les animations, le son dans les niveaux ou d'autres caractéristiques.

## Format de fichier CON

Les fichiers CON sont enregistrés sur le disque sous forme de fichiers texte brut pouvant être ouverts dans n'importe quel éditeur de texte. Les paramètres sont enregistrés dans des lignes de texte qui peuvent être modifiées et sauvegardées pour être rechargées par le jeu.
