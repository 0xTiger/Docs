## Qu'est-ce qu'un fichier ASSET ?

Un fichier ASSET est un fichier spécial utilisé dans Unity, un programme de création de jeux vidéo. Il contient des configurations importantes telles que des objets ou des paramètres de jeu. Par exemple, un fichier ASSET peut stocker la façon dont les objets se déplacent dans le jeu, comme la façon dont ils rebondissent les uns sur les autres ou tombent. Ces fichiers sont comme des conteneurs qui conservent toutes les informations importantes organisées afin que les développeurs de jeux puissent facilement travailler sur leurs projets.

## Comment ouvrir un fichier ASSET
  
Dans Unity, vous n'ouvrez généralement pas les fichiers d'actifs de la même manière que vous ouvririez un document ou un fichier dans un éditeur de texte ou un programme similaire. Au lieu de cela, Unity gère les fichiers d'actifs dans l'éditeur Unity lui-même. Voici comment accéder aux fichiers d’actifs et travailler avec eux dans Unity :

1.  **Import Assets**: To use an asset file in your Unity project, you need to import it into project. You can do this by either dragging and dropping asset file directly into Unity Editor or by using "Assets" menu and selecting "Import New Asset."
    
2.  **Affichage des actifs** : une fois importés, vous pouvez afficher et gérer les fichiers d'actifs dans la fenêtre « Projet » de Unity Editor. Cette fenêtre affiche tous les actifs de votre projet et vous permet de les organiser dans des dossiers, de rechercher des actifs spécifiques et de prévisualiser leur contenu.
    
3.  **Utilisation d'actifs** : vous pouvez utiliser des fichiers d'actifs dans votre projet Unity en les faisant glisser de la fenêtre Projet vers votre hiérarchie de scène ou sur GameObjects dans la scène. Par exemple, vous pouvez faire glisser une ressource de texture sur un matériau pour l'appliquer à un objet 3D, ou vous pouvez faire glisser une ressource préfabriquée dans la scène pour instancier un GameObject préconfiguré.
    
4.  **Modification des ressources** : certains fichiers de ressources, tels que les scripts, les matériaux et les animations, peuvent être modifiés directement dans Unity Editor. Vous pouvez double-cliquer sur ces actifs dans la fenêtre « Projet » pour les ouvrir dans leurs éditeurs respectifs et apporter les modifications nécessaires.
    
5.  **Enregistrement des actifs** : toutes les modifications que vous apportez aux fichiers d'actifs dans Unity Editor sont automatiquement enregistrées dans votre projet Unity. Il n'est pas nécessaire d'enregistrer explicitement des fichiers d'actifs individuels comme vous le feriez avec un document dans un logiciel traditionnel.

Les fichiers ASSET peuvent être ouverts ou référencés à l'aide des programmes suivants.
