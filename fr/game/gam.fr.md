## Qu'est-ce qu'un fichier GAM ?
Un fichier avec l'extension .gam est utilisé par divers jeux vidéo pour inciter les joueurs à continuer là où ils se sont arrêtés la prochaine fois que le programme est ouvert. Par conséquent, cela signifie que le fichier GAM enregistre les informations sur ce qu'un utilisateur a fait à quel moment. Ces fichiers peuvent être enregistrés automatiquement par les logiciels de jeu ou les joueurs peuvent être invités à enregistrer s'ils veulent continuer l'état du jeu où ils quittent.
## Format de fichier GAM
Le format de fichier GAM est essentiellement utilisé pour conserver les données de jeu dans les sauvegardes de jeux. Le fichier GAM ne stocke pas les créatures, les informations sur les objets ou la zone, mais enregistre les données sur les membres du groupe et les variables générales ou globales qui affectent les membres du groupe.
### Structure du fichier GAM
Le fichier GAM a la structure suivante :
- Entête
- Membres du parti
- Données du fichier CRE membre du parti
- PNJ (qui ne sont pas dans le groupe)
- Statistiques sur les PNJ tués (intégrées dans la structure NPC)
- Données du fichier NPC CRE
- Variables (dans l'espace de noms GLOBAL)
- Entrées de journal
- Informations familières
- Emplacements stockés
- Emplacements des avions de poche
- Supplément familier
