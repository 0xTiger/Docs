## Qu'est-ce qu'un fichier OSK ?

Un fichier OSK est un fichier de mise en page ou de skin utilisé par le jeu de rythme osu ! où les joueurs utilisent la souris, le clavier en synchronisation avec la musique. Il contient la disposition de l'interface de l'éditeur et peut être personnalisé par les joueurs en fonction de leurs préférences. osu ! stocke les données de beatmap au format de fichier [OSZ](/fr/game/osz/).

**Type MIME du format de fichier OSK :** x-osu-skin-archive

Vous pouvez ouvrir le fichier OSK avec [osu!](https://osu.ppy.sh/home).

## Format de fichier OSK

Les fichiers OSK sont enregistrés sur le disque en tant que fichiers compressés [ZIP](/fr/compression/zip/). Vous pouvez simplement conserver un fichier .osk en .zip et extraire l'archive pour afficher son contenu. Le contenu extrait doit être placé dans le répertoire \Skins\ pour être chargé dans le jeu.

## Que peut-on personnaliser avec le fichier OSK ?

Les éléments de jeu suivants peuvent être personnalisés avec le fichier .osk.

* Frapper les cercles
* Affichage des scores
* Type de curseur
* Le curseur
* Hit rafales
