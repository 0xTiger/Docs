## Qu'est-ce qu'un fichier CT ?

Un fichier CT est un fichier de table de triche créé avec le logiciel [Cheat Engine](https://www.cheatengine.org/) qui est utilisé pour créer des modifications aux jeux Windows pour tricher. Le Cheat Engine, un moteur de triche open source, examine les jeux en cours d'exécution et enregistre ses emplacements d'adresse. Ces informations et leurs remplacements sont écrits dans le fichier CT qui est ensuite chargé par les joueurs pour modifier les propriétés du jeu telles que le score de santé, le score le plus élevé et les vies restantes.

## Format de fichier CT du moteur de triche

Les fichiers CT sont enregistrés avec les emplacements d'adresse et d'autres informations associées à pirater dans un jeu. Dans la plupart des cas, les fichiers sont enregistrés sous forme d'archives compressées [ZIP](/fr/compression/zip/) qui peuvent être facilement extraites à l'aide de n'importe quel utilitaire de décompression standard tel que WinZIP ou 7-Zip.

## Comment utiliser les tables de triche

1. Téléchargez le tableau et copiez-le dans le dossier de Cheat Engine
1. Exécutez le moteur de triche
1. Lancez le jeu ;
1. Utilisez la combinaison ALT+TAB et choisissez le jeu dans la liste des processus via Cheat Engine
1. Si le tableau se trouve dans un dossier différent, appuyez simplement sur Ctrl+O et guidez Cheat Engine vers ce dossier. Sélectionnez ensuite la table (généralement nom_processus.ct) ;
1. Une fois la table chargée, s'il y a un script, il suffit de le vérifier.
1. ALT+TAB pour revenir au jeu et amusez-vous.
