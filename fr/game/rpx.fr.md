## Qu'est-ce qu'un fichier RPX ?

Un fichier RPX est un fichier de jeu qui peut être chargé ou lu par la console de jeu Wii U. Il peut également être chargé et lu dans l'émulateur de jeu vidéo Cemu qui est populaire pour fournir un environnement d'émulation pour les fichiers Wii U. Les fichiers de jeu Wii U sont volumineux et peuvent également être enregistrés en tant que fichiers [WUD](/fr/disc-and-media/wud/) ou [WUX](/fr/compression/wux/) pour être chargés par des émulateurs de jeux. Un fichier RPX contient l'intégralité du jeu exécutable qui comprend tous les composants du jeu tels que les graphiques, les textures, les séquences et la ROM du jeu.

## Format de fichier RPX - Plus d'informations

Les fichiers RPX sont enregistrés sur disque au format de fichier propriétaire. Certaines API sont disponibles pour compresser ou décompresser les fichiers RPX tels que [wiiurpxtool](https://github.com/0CBH0/wiiurpxtool). De plus, des applications telles que l'émulateur de jeux vidéo Cemu vous permettent de jouer à des jeux Wii U sous Windows sur PC.
