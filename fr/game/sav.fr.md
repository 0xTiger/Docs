## Qu'est-ce qu'un fichier SAV ?

Un fichier avec l'extension .sav est un fichier de jeu enregistré sur Nintendo DS qui peut être chargé dans des émulateurs de jeux Nintendo DS tels que [DeSmuME](http://desmume.org/) et [NO$GBA](https://www. nogba.com/). Ces fichiers enregistrent l'état actuel du jeu qui peut ensuite être chargé dans les émulateurs Nintendo. Les informations stockées dans les fichiers SAV incluent le nom du personnage du jeu, les points d'expérience, les niveaux terminés et les ressources du jeu. Les fichiers SAV peuvent être chargés dans des émulateurs, mais la différence de versions du fichier SAV peut entraîner la génération d'erreurs au niveau des émulateurs. Les fichiers SAV peuvent être ouverts avec les émulateurs DeSmuME et NO$GBA.

## Format de fichier SAV - Plus d'informations

Les fichiers Nintendo DS SAVE sont enregistrés sur disque au format de fichier binaire. Bien qu'ils ne soient pas lisibles par l'homme, un éditeur HexaDecimal tel que [HxD](https://mh-nexus.de/en/hxd/) peut être utilisé pour les ouvrir à des fins d'analyse. Dans les scénarios où le fichier SAV ne peut pas être chargé dans une version mise à niveau d'un émulateur, il peut être ouvert dans l'éditeur hexadécimal pour supprimer toutes les données rembourrées indésirables.
