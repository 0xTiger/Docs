## Qu'est-ce qu'un fichier AZW1 ? ##

L'AZW1 est un format de fichier de livre électronique également connu sous le nom de ** Amazon Kindle Topaz eBook ** qui a été lancé via ** Whispernet **. Ce format de fichier peut être utilisé avec le logiciel Kindle sur les PC et les smartphones et il est responsable du stockage des livres électroniques téléchargés à partir du réseau 3G Whispernet d'Amazon.

## Format de fichier AZW1 ##

Les appareils de lecture de livres Amazon Kindle et le logiciel utilisent des fichiers au format AZW1 ayant une extension .azw1. Ces fichiers sont également connus sous le nom de fichiers Amazon Topaz eBook, ces fichiers AZW1 sont utilisés pour assembler et gérer les parties téléchargées des fichiers Amazon Kindle eBook à partir du réseau **3G Whispernet** d'Amazon. Le logiciel Amazon Kindle peut être installé sur un système basé sur **Microsoft Windows** pour exécuter la prise en charge de ces fichiers AZW1 et des fonctionnalités pertinentes intégrées à l'application. Les versions de ce programme pour les environnements Mac sont également disponibles. Ce logiciel est généralement utilisé pour commencer à télécharger et à visualiser le contenu de ces livres Amazon Kindle. Étant donné que l'application Amazon Kindle prend en charge les fichiers AZW1 et peut également être installée dans ces deux systèmes bien connus, les créateurs peuvent facilement créer et partager correctement des livres Kindle.
