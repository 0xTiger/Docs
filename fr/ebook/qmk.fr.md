## Qu'est-ce qu'un fichier QMK ?

Un fichier avec l'extension .qmk est généralement connu sous le nom de format **YanCEyDesktop Quickmarks file**. Le YanCEyWare a introduit le format de fichier YanCEyDesktop Quickmarks. Le fichier QMK est lié au fichier [YBK](/fr/ebook/ybk/) et pris en charge par un programme disponible pour les machines exécutant le système d'exploitation Windows. Un fichier avec une extension QMK est classé comme un fichier de type texte qui permet aux utilisateurs de naviguer facilement vers les emplacements enregistrés dans les livres électroniques. YanCEyDesktop est le logiciel recommandé pour gérer les fichiers QMK. Le logiciel **YanCEyDesktop** a été créé par **D Remington**.

## Problèmes pour ouvrir un fichier QMK

Si vous ne parvenez pas à ouvrir et à exécuter le fichier QMK ; cela ne signifie pas qu'aucun logiciel approprié n'est installé sur votre appareil. Il peut y avoir d'autres problèmes qui empêchent le fichier de fonctionner correctement. Les problèmes possibles peuvent être l'un des suivants :

- Corruption d'un fichier QMK
- Liens erronnés du fichier QMK dans les inscriptions du registre
- Suppression de la description du QMK du registre Windows
- Installation corrompue d'une application desservant le format QMK
- Un fichier QMK infecté par des logiciels malveillants indésirables
- L'ordinateur a trop peu de ressources d'équipement pour faire fonctionner le fichier QMK
- Les pilotes utilisés par l'ordinateur pour ouvrir un fichier QMK sont obsolètes
