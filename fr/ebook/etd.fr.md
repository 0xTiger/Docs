## Qu'est-ce qu'un fichier ETD ? ##

L'extension de fichier ETD est liée aux fichiers Adobe Reader EBX Transfer Data Format. Cette extension de fichier est généralement bien adaptée aux logiciels pouvant être téléchargés sur les plates-formes système Linux, Mac OS et Windows. C'est un format de fichier très connu et utilisé comme format de certains livres électroniques qui peuvent être téléchargés à partir du Net. Mais précisez que dans ce format, les fichiers eBook et leur contenu peuvent être enregistrés. Ceci n'est utilisé que pendant le transfert ou le processus de téléchargement. Le principal avantage de ce format est qu'il permet de télécharger des livres électroniques sur l'ordinateur et qu'il peut être lu ou consulté en ligne. Il est principalement associé à un livre électronique et fonctionne donc efficacement avec des applications telles qu'Adobe Reader et Acrobat.

## Un problème pour ouvrir un fichier ETD ? ##

Si vous rencontrez un problème pour ouvrir un fichier ETD, vérifiez les éventuelles erreurs liées à l'extension de fichier, mentionnées comme suit.

* Extension de fichier ETD corrompue. Pour résoudre ce problème, téléchargez un antivirus
* Liens rompus vers le fichier ETD dans l'archive. Vérifiez les liens avant d'ouvrir
* Suppression de la description de l'ETD de l'archive Windows
* Mauvaise installation d'une application qui supporte le format ETD
* Fichier infecté
* Version obsolète du logiciel. Installez la version mise à jour compatible avec le bon fonctionnement de l'extension de fichier ETD
