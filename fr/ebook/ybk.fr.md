## Qu'est-ce qu'un fichier YBK ? ##

Le format de fichier YanCEyWare eBook et Microsoft Encarta utilisent tous deux les extensions de fichier YBK. YanCEyWare eBook a été créé par YanCEyWare. Toutes les applications logicielles sont disponibles pour les appareils exécutant Windows, prennent en charge les fichiers YBK. L'Encyclopédie Encarta peut être mise à jour par des fichiers .ybk sans avoir à mettre à jour l'ensemble du programme. YanCEyDesktop est un programme bien connu et populaire qui peut facilement gérer et gérer les fichiers YBK. Le lecteur YanCEyWare utilise largement ce format de fichier YBK, en tant qu'application de lecture de livres électroniques pour Windows Mobile ou Pocket PC. Ce fichier a été développé avec de grandes ressources hyperliées.

## Problèmes pour ouvrir un fichier YBK ##

Parfois, lorsqu'il est difficile d'ouvrir et de faire fonctionner le fichier YBK, voici la liste des problèmes possibles :

* Ouverture d'un fichier YBK corrompu
* Liens inappropriés et inappropriés vers le fichier YBK dans les accès au registre
* Suppression involontaire de la description du YBK de l'archive Windows
* Installation partielle d'une application qui provisionne le format YBK
*	Infected YBK file with an undesirable malware
