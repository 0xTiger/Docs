## Qu'est-ce qu'un fichier TR ? ##

Un fichier avec une extension .tr est un fichier eBook TomeRaider créé avec le logiciel TomeRaider eReader. Ces fichiers peuvent être ouverts à la fois sur des appareils Windows et portables tels qu'Android, Palm OS et EPOC, et sont classés en tant que fichiers texte. Le logiciel TomeRaider développé par le site Web [Yadabyte](https://www.yottabyte.ltd/) et le site Web officiel fournissent plus d'informations sur les fichiers TR. Le format de fichier TR2 a été remplacé par le format TR3 (TomeRaider 3). Les fichiers TR offrent une recherche rapide et des niveaux de compression élevés.

## Bref historique du format de fichier TR

TomeRaider a été développé par une société de développement de logiciels et de sites Web basée au Royaume-Uni, Yadabyte, en 1999. La société l'a poussé plus loin vers la version 3 de TomeRaider.

## Format de fichier TR

La dernière version du format de fichier TR est la version 3 qui prend en charge la recherche et la compression rapides. Il compresse les fichiers texte à 45-60% de la taille d'origine. Cela permet aux lecteurs de livres électroniques TomeRaider d'effectuer une recherche, une indexation et une compression rapides. Le format de fichier TR convient aux documents de référence volumineux et la plupart des documents nécessitent une clé DRM pour être chiffrés.

## Problèmes pour ouvrir un fichier TR

Il peut y avoir des problèmes courants à cause desquels vous ne pouvez pas ouvrir le fichier :

* Absence de logiciel de support
* Version logicielle obsolète
* Fichier TR corrompu
* Liens incorrects vers le fichier TR dans les accès aux archives
* Aucun droit d'accès dans le système pour ouvrir les fichiers
* Lecteur obsolète dans votre système
* Suppression du rapport du TR de l'archive Windows
* Installation partielle d'une application supportant le format TR
* Fichier TR infecté
* Espace insuffisant pour les ressources matérielles pour aider à l'ouverture du fichier TR
