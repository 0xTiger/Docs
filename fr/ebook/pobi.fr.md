## Qu'est-ce qu'un fichier POBI ?

Un fichier POBI appartient au type de fichier eBook ; est en fait un **Kindle Touch Periodical Data**. Ce format de fichier est également appelé périodique [MOBI](/fr/ebook/mobi/); utilisé dans l'appareil de lecture électronique Amazon Kindle Touch. Bien que le type de fichier des fichiers POBI soit un eBook, nous ne pouvons pas dire que le fichier avec l'extension .pobi est un eBook ; parce qu'il n'a pas le contenu comme un eBook. Il contient juste les données comme un magazine et fonctionne comme un fichier MOBI. Cependant, si l'extension .pobi est remplacée par .mobi, l'index des sections et des articles du magazine n'apparaîtra pas.

## Problèmes pour ouvrir un fichier POBI ##

Si vous ne parvenez pas à ouvrir et à exécuter le fichier POBI ; cela ne signifie pas qu'aucun logiciel approprié n'est installé sur votre appareil. Il peut y avoir d'autres problèmes qui empêchent le fichier de fonctionner correctement. Les problèmes possibles peuvent être l'un des suivants :

- Corruption d'un fichier POBI.
- Liens erronnés du fichier POBI dans les inscriptions du registre.
- Suppression de la description de l'OPBI du registre Windows
- Installation corrompue d'une application desservant le format POBI
- Un fichier POBI infecté par des logiciels malveillants indésirables.
- L'ordinateur ne dispose pas de ressources d'équipement suffisantes pour faire fonctionner le fichier POB.
- Les pilotes utilisés par l'ordinateur pour ouvrir un fichier POBI sont obsolètes.
