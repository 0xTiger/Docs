## Qu'est-ce qu'un fichier EBK ?
Une extension EBK est utilisée par le logiciel **eBook Pro Viewer** lorsqu'il enregistre un fichier eBook. L'eBook Pro est un logiciel qui peut créer des livres électroniques et protéger les informations protégées par le droit d'auteur. Un fichier EBK contient la version cryptée et compressée d'un livre électronique qui peut être lu à l'aide de **eBook Pro Viewer** ; après avoir fourni les informations d'identification appropriées pour l'authentification.
