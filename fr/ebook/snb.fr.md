## Qu'est-ce qu'un fichier SNB ?

Le fichier avec l'extension .snb ; appartient aux appareils mobiles Samsung. Lorsque les téléphones Galaxy S Note créent une note à l'aide d'un logiciel typique (application de prise de notes avancée développée par Samsung), cette note est en fait enregistrée sous forme de fichier avec l'extension .snb dans la mémoire des téléphones intelligents (par exemple Galaxy S et Galaxy Note ). Un fichier de notes peut contenir des croquis, du texte, de la vidéo, de l'audio et des images. Les fichiers SNB ont été abandonnés par Samsung et ces fichiers ont été remplacés par des fichiers [SPD](/fr/ebook/snb/). Par conséquent, vous ne trouverez peut-être pas beaucoup d'informations sur ce type de fichier typique sur Internet.

## Problèmes pour ouvrir un fichier SNB

Si vous ne parvenez pas à ouvrir et à exécuter le fichier SNB ; cela ne signifie pas qu'aucun logiciel approprié n'est installé sur votre appareil. Il peut y avoir d'autres problèmes qui empêchent le fichier de fonctionner correctement. Les problèmes possibles peuvent être l'un des suivants :

- Corruption d'un fichier SNB.
- Connexions erronnées du fichier SNB dans les inscriptions du registre.
- Suppression de la description de SNB du registre Windows
- Installation corrompue d'une application desservant le format SNB
- Un fichier SNB infecté par un logiciel malveillant indésirable.
- L'ordinateur ne dispose pas de ressources d'équipement suffisantes pour faire fonctionner le fichier SNB.
- Les pilotes utilisés par l'ordinateur pour ouvrir un fichier SNB sont obsolètes.
