## Qu'est-ce qu'un fichier AZW4 ?

L'AZW4 est un ebook Amazon Print Replica développé par Amazon pour le logiciel de bureau Kindle et les appareils Kindle eReader. L'AZW4 n'est pas similaire à [MOBI](/fr/ebook/mobi/), [AZW](/fr/ebook/azw/) ou [AZW3](/fr/ebook/azw3/). Ce format est en fait un wrapper pour [PDF](/fr/pdf/) ou ebooks MOBI avec DRM ajouté.

## Format de fichier AZW4

Le **AZW4** a été présenté comme une nouvelle version du logiciel **Textbooks** qui est similaire au **PDF**. Un livre électronique AZW4 contient un fichier PDF complet à l'intérieur, ou il peut être considéré comme un fichier PDF avec un wrapper appelé AZW4. L'AZW4 est principalement utilisé par Kindle eReader. Cependant, d'autres applications peuvent également utiliser l'extension de fichier .azw4. Même les programmes malveillants peuvent créer des fichiers .azw4. Soyez prudent avec les fichiers .azw4 provenant d'une source douteuse.
