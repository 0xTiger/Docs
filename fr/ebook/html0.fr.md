## Qu'est-ce qu'un fichier HTML0 ? ##

HTML0 est le format de livre numérique natif utilisé par **Book Designer** avec un code HTML et CSS standard qui décrit le contenu et la conception de chaque page. **Book Designer** est le programme le plus courant utilisé pour fonctionner avec des fichiers HTML0 et pour transformer des documents texte en de nombreux formats de livres électroniques. Le format de fichier HTML0 prend en charge les logiciels pouvant être installés sur le système Windows car il appartient à la catégorie des fichiers texte. Les fichiers HTML0 comprennent du HTML normal et peuvent être visualisés et modifiés à l'aide d'un éditeur de texte. Les fichiers HTML0 ne peuvent pas être liés à des fichiers CSS discrets, contrairement aux fichiers HTML habituels.

## Problème pour ouvrir un fichier HTML0 ? ##

* Le fichier peut être infecté lorsque vous avez du mal à l'ouvrir
* Doit analyser le fichier à la recherche de virus à l'aide d'un antivirus en ligne
* Certains fichiers nécessitent des droits d'accès pour les ouvrir
* Si les systèmes ne disposent pas de ressources suffisantes pour ouvrir les fichiers HTML0, fermez toutes les applications, puis rouvrez-les
* Le fichier peut être corrompu ou avoir une erreur si vous l'avez téléchargé ou reçu de n'importe quel e-mail
* Utilisez toujours le système et les pilotes les plus récents pour rendre votre ordinateur plus sûr et résoudre tous les problèmes liés aux extensions de fichiers
