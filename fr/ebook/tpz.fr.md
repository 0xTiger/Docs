## Qu'est-ce qu'un fichier TPZ ? ##

L'extension de fichier .tpz est également connue sous le nom de format de livre électronique Kindle Topaz par Amazon. Linux, Mac OS et Windows sont quelques-unes des plates-formes sur lesquelles les fichiers avec l'extension de fichier .tpz peuvent être distribués. En tant que fichier de livre électronique Kindle Topaz, le fichier TPZ est formaté pour ressembler à des livres électroniques traditionnels et contient des normes de compilation de données propriétaires d'Amazon pour le format TPZ. Ce fichier a une prise en charge avancée des polices intégrées, et des détails supplémentaires tels que des annotations et des signets peuvent également être inclus dans les fichiers de l'eBook Kindle Topaz, bien que ces détails de métadonnées soient stockés séparément. Le fichier TPZ peut également être classé comme un fichier de données pouvant être téléchargé à partir de la base de données en ligne d'ebooks Kindle d'Amazon.

## Des problèmes pour ouvrir un fichier TPZ ? ##

Certains des problèmes courants mentionnés ci-dessous peuvent survenir :

* Fichier infecté. Cela peut être corrigé par un logiciel antivirus approprié pour le rendre propre
* Espace disque insuffisant. Pour cela, il faut fermer toutes les applications non pertinentes pour un meilleur fonctionnement
* Les liens brisés peuvent parfois causer des problèmes
* Parfois, le fichier n'a pas de droits d'accès, dans ce cas, assurez-vous que les droits d'accès ouvrent facilement le fichier
