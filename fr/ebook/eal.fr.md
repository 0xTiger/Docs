## Qu'est-ce qu'un fichier EAL ? ##

Les fichiers **Kindle End Action** se trouvent généralement avec l'extension .EAL. Ce fichier consiste en un xml qui est écrit pour montrer quelque chose sur l'auteur après la dernière page de l'eBook.

## Format de fichier EAL ##

Le fichier .EAL est un fichier xml qui comprend les données des clients qui ont acheté le livre et ont également acheté une fonctionnalité pour montrer quelque chose de plus sur l'auteur du livre électronique après la dernière page du livre ; L'ASIN est également inclus pour accéder à la page Kindle Store du titre.
