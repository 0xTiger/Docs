## Qu'est-ce qu'un fichier LIT ?

Les fichiers LIT appartiennent à l'ancien **Microsoft Reader** qui était utilisé pour lire des livres électroniques. Les livres électroniques affichés par Microsoft Reader portent l'extension **.lit** qui est abrégée en **Littérature**. Le format de fichier LIT consiste en un fichier Microsoft ITOL/ITLS HTML Help 2.0, qui offre toutes sortes de fonctionnalités étonnantes telles que la compression [LZX](/fr/compression/lzx/), la représentation binaire du matériel de balisage et des données auxiliaires facultatives arbitraires associées à chaque flux. Le texte du livre est un nombre arbitraire de flux de balisage OEBPS 1.0 et un sous-ensemble de CSS OEBPS 1.0. Les images sont [GIF](/fr/image/gif/), [JPEG](/fr/image/jpeg/) et [PNG](/fr/image/png/).

## Combien de temps Microsoft Reader a affiché les fichiers LIT

Le **Microsoft Reader** est une application permettant de lire des livres électroniques qui ont été abandonnés par Microsoft. La première version de Microsoft Reader est sortie en août 2000, qui utilisait son propre format LIT. Il était disponible pour les PDA et les ordinateurs Windows. Le nom de l'application a également été utilisé plus tard pour une application (qui a été initialement publiée avec Windows 8) pour lire les fichiers [PDF](/fr/pdf/) et [XPS](/fr/page-description-language/xps/). L'application Microsoft Reader a été abandonnée en 2018.
