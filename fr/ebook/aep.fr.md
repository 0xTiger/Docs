## Qu'est-ce qu'un dossier AEP ? ##

Le fichier avec l'extension .aep (qui signifie **Activ E-Book Project**) est classé comme un livre électronique (eBook). Les fichiers AEP sont créés en utilisant le même moteur que celui d'Internet Explorer pour assurer une compatibilité à 100%. Il contient les pages comme une personne peut lire un livre physique. Les pages des fichiers AEP incluent généralement le texte, les images, les animations et les hyperliens. En outre, il prend en charge [HTML](/fr/web/html/), [DHTML](/fr/web/dhtml/), [JPEG](/fr/image/jpeg/) et [PNG](/fr/image/png/) graphiques, animations , [JavaScript](/fr/web/js/), VBScript et de nombreux plug-ins Internet Explorer. Un fichier AEP peut être enregistré en tant que fichier exécutable Windows (EXE) afin qu'il puisse être ouvert sur Windows95 ou des versions ultérieures de Microsoft Windows.
 


## Format de fichier AEP ##

Les fichiers au format AEP sont créés à l'aide d'un logiciel AEC (Active E-Book Compiler). Ce logiciel utilise un moteur similaire à celui d'IE pour créer des livres électroniques. L'utilisateur peut créer des copies illimitées d'eBooks sans payer aucun coût pour l'utilisation du programme. Le processus de distribution est également simple car chaque eBook est composé d'un programme Windows autonome, qui peut être distribué sur n'importe quel support comme les e-mails, le téléchargement sur Internet, les disquettes, les CD-ROM, les disques ZIP, etc.

## Problèmes possibles lors de l'ouverture du fichier

Si vous ne parvenez pas à ouvrir et à exécuter le fichier AEP ; cela ne signifie pas qu'aucun logiciel approprié n'est installé sur votre appareil. Il peut y avoir d'autres problèmes qui empêchent le fichier de fonctionner correctement. Les problèmes possibles peuvent être l'un des suivants :

- Corruption d'un fichier AEP
- Liens erronnés du fichier AEP dans les inscriptions du registre
- Suppression de la description de l'AEP du registre Windows
- Installation corrompue d'une application desservant le format AEP
- Un fichier AEP infecté par un malware indésirable
- L'ordinateur a trop peu de ressources d'équipement pour faire fonctionner le fichier AEP
- Les pilotes utilisés par l'ordinateur pour ouvrir un fichier AEP ne sont pas à jour
