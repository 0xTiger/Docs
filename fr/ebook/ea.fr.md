## Qu'est-ce qu'un fichier EA ? ##

L'extension .ea est utilisée pour le fichier **Kindle End Action** qui est l'un des formats de fichier Kindle. Ces fichiers sont créés par l'**appareil de lecture électronique Amazon Kindle** pour afficher quelque chose qu'un auteur souhaite afficher à la fin d'un ebook. Un fichier EA contient les informations au format [XML](/fr/web/xml/). Ainsi, ce fichier peut être ouvert ou visualisé à l'aide de n'importe quel éditeur de texte. Ce fichier se trouve là où les appareils de lecture électronique Kindle enregistrent les livres électroniques. Le contenu peut être constitué de balises XML vides si le client n'a pas défini les informations à afficher à la fin d'un eBook.

## Un peu plus sur le format de fichier EA ##

Un fichier avec l'extension .ea est un fichier XML qui contient les données des clients qui ont acheté le livre et ont également acheté une fonctionnalité pour en montrer plus sur l'auteur après la dernière page du livre électronique ; y compris **ASIN** ou **ISBN** afin qu'il puisse accéder à la page Kindle Store du titre.

### Qu'est-ce qu'un ASIN/ISBN ? ###

L'ASIN signifie ** Amazon Standard Identification Number ** qui est en fait un code alphanumérique à dix chiffres permettant d'identifier de manière unique les produits sur Amazon. L'ASIN est un identifiant unique pour chaque produit et est attribué lorsque vous créez un produit Amazon. Chaque produit sur Amazon a un ASIN à l'exception des livres. Amazon utilise l'ISBN (International Standard Book Number) à la place d'un ASIN. Cet identifiant Amazon pour vos produits est nécessaire pour obtenir avant de les vendre sur Amazon.

## Problème pour ouvrir un fichier EA ?

Si vous ne parvenez pas à ouvrir et à exécuter le fichier EA ; cela ne signifie pas qu'aucun logiciel approprié n'est installé sur votre appareil. Il peut y avoir d'autres problèmes qui empêchent le fichier de fonctionner correctement. Les problèmes possibles peuvent être l'un des suivants :

- Corruption d'un fichier EA.
- Liens erronés du fichier EA dans les inscriptions du registre.
- Suppression de la description de l'EA du registre Windows
- Installation corrompue d'une application supportant le format EA
- Un fichier EA infecté par un logiciel malveillant indésirable.
- L'ordinateur ne dispose pas de ressources d'équipement suffisantes pour faire fonctionner le fichier EA.
- Les pilotes utilisés par l'ordinateur pour ouvrir un fichier EA sont obsolètes.
