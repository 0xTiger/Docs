## Qu'est-ce qu'un fichier TCR ?

Au début des années 90, le format de fichier TCR a été introduit pour les anciens appareils de poche **Psion Series 3**. La société a utilisé sa compression spécifique à la plate-forme et ce format a une prise en charge très limitée du formatage du texte. Il s'agit d'un format de fichier basé sur ZVR et offre un taux de compression comparativement meilleur que PalmDOC. Cela fonctionne aussi très bien avec FontMachine. Bien que le format de fichier TCR appartienne à un appareil abandonné, ce format de fichier peut toujours être lu à l'aide de certaines liseuses. Ce format de fichier peut également être visualisé sur un ordinateur avec Sumatra PDF et avec l'application Calibre sous Windows et Linux.

## Détails techniques

Étant donné que le format de fichier TCR est basé sur un compresseur ZVR qui est capable de réduire de nombreux fichiers d'environ 50 % de leur taille d'origine. Le schéma de compression utilisé par over est un peu ancien. Le fichier compressé commence par un dictionnaire des 256 symboles possibles qui pourraient apparaître dans le fichier compressé. Le dictionnaire contient une chaîne alternative fixe pour chacun de ces symboles. La décompression est devenue très rapide, même dans OPL, et elle peut commencer à n'importe quel endroit du fichier compressé.

## Problèmes pour ouvrir un fichier TCR ##

Si vous ne parvenez pas à ouvrir et à exécuter le fichier TCR ; cela ne signifie pas qu'aucun logiciel approprié n'est installé sur votre appareil. Il peut y avoir d'autres problèmes qui empêchent le fichier de fonctionner correctement. Les problèmes possibles peuvent être l'un des suivants :

- Corruption d'un fichier TCR.
- Liens erronés du fichier TCR dans les inscriptions du registre.
- Suppression de la description du TCR du registre Windows
- Installation corrompue d'une application desservant le format TCR
- Un fichier TCR infecté par des logiciels malveillants indésirables.
- L'ordinateur ne dispose pas de ressources d'équipement suffisantes pour faire fonctionner le fichier TRC.
- Les pilotes utilisés par l'ordinateur pour ouvrir un fichier TCR sont obsolètes.
