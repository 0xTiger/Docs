## Qu'est-ce qu'un fichier TR3 ? ##

Le fichier TR3 est un livre électronique TomeRaider 3 qui prend en charge la recherche et la compression rapides. Cela peut fonctionner correctement via le programme lié à TomeRaider. Yadabyte est le développeur de ce logiciel, une société britannique de développement de logiciels et de sites Web basée au Royaume-Uni. L'application TomeRaider eBook reader peut être utilisée pour le bon fonctionnement et la gestion du contenu de ces fichiers TR3. Ce lecteur de livre électronique associé peut être installé sur des ordinateurs fonctionnant sur des systèmes Microsoft Windows et sur de nombreux gadgets transférables. Le fichier TR3 appartient au groupe de fichiers eBook utilisés dans les systèmes opérationnels tels que Windows 10, Windows 7, Windows 8 / 8.1, Windows Vista, Windows XP.

Les **balises HTML** spécifiques à TR3 sont les suivantes :

|Étiquette|Description|
---|---|
|\<new> |Pour créer des fichiers TomeRaider à partir de fichiers texte, la balise Nouveau suffit.<new> balise définit le début des pages.|
|\<CATDEF> ... \</CATDEF> |définit le nom de la catégorie|
|\<META> ... \</META> |est une balise utilisée pour définir toutes les métadonnées utilisées dans le format de fichier. Cette balise inclut un certain nombre de paramètres.|
|\<EXPMSG> |Cette balise contrôle le message qui s'affiche lorsqu'un fichier a expiré. Chaque fois qu'un utilisateur tente d'afficher une page après l'expiration du fichier, le message d'expiration s'affiche à la place du texte réel de la page.|
|\<DIEMSG> |Cette balise est similaire à EXPMSG. Il est utilisé pour afficher un message lorsque le fichier meurt.|
|\<ENGMSG> |Cette balise est utilisée pour créer le message qui s'affiche pour les pages chiffrées. Le message s'affiche à la place du texte de la page si l'utilisateur tente d'ouvrir une page chiffrée avant de la déverrouiller.|
|\<expmsg> ,\<diemsg> et \<encmsg> |Ignoré dans le texte de la page. Ils doivent être placés dans la section d'en-tête du fichier avant le premier<new> balise.|
|\<SELECTION> . \</ SELECTION> |Utilisé pour prendre en charge les requêtes. Cette balise doit être avant la première<new> étiquette. Il compile les données de sélection pour la requête de l'utilisateur.|
|\<catset> . \</catset> | Utilisé pour attribuer des noms de catégorie pour chaque sujet.|


## Problèmes pour ouvrir un fichier TR3 ##

Voici la liste de certains problèmes courants qui peuvent survenir et entraîner un dysfonctionnement du format de fichier :

* Fichier corrompu
* Fichier infecté en raison d'un virus
* Liens incorrects vers le fichier TR3 dans les accès aux archives
* Aucun droit d'accès dans le système pour ouvrir les fichiers
* Lecteur obsolète dans votre système
* Suppression du rapport du TR3 de l'archive Windows
* Contacter le développeur peut être une meilleure option pour un meilleur résultat
