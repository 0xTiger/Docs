## Qu'est-ce qu'un fichier SBV ?

Un fichier avec l'extension de fichier .sbv est un fichier de sous-titres SubViewer. SubViewer est un format de sous-titres utilisé avec les fichiers vidéo né au début des années 2000. Le fichier .sbv contient le texte des sous-titres ainsi que des informations de synchronisation indiquant le moment où chaque sous-titre doit être affiché pendant la lecture vidéo.

Les fichiers SubViewer sont souvent utilisés avec les vidéos YouTube, car YouTube prend en charge le format SubViewer pour ses sous-titres vidéo. Les fichiers SubViewer peuvent également être utilisés avec d'autres lecteurs vidéo et logiciels prenant en charge le format.

Pour utiliser un fichier .sbv avec un fichier vidéo, le fichier vidéo et le fichier .sbv doivent se trouver dans le même dossier et avoir le même nom de fichier (à l'exception de l'extension de fichier). De nombreux lecteurs vidéo chargeront automatiquement le fichier .sbv lors de la lecture de la vidéo si les fichiers sont nommés de cette manière. Il existe diverses applications d'édition de sous-titres prenant en charge le format SubViewer si vous devez générer ou modifier un fichier .sbv. Aegisub, Subtitle Workshop et Subtitle Edit sont quelques-uns des outils souvent utilisés.

## Format de fichier SBV – Plus d’informations

Le format de fichier SBV est un format texte simple utilisé pour stocker les sous-titres des vidéos. SBV signifie « SubViewer » ou « SubRip Video », selon la source du fichier. Les fichiers SBV contiennent des sous-titres synchronisés avec le contenu vidéo. Chaque sous-titre est constitué d'un horodatage indiquant les heures de début et de fin du sous-titre, suivi du texte du sous-titre. L'horodatage est généralement au format « hh:mm:ss,mmm », où « hh » représente les heures, « mm » représente les minutes, « ss » représente les secondes et « mmm » représente les millisecondes. Le texte du sous-titre est généralement au format texte brut.

Voici un exemple de ce à quoi pourrait ressembler un fichier SBV :

```
0:00:01.000,0:00:03.000
Hello, and welcome to our video!

0:00:04.000,0:00:06.000
In this video, we will be discussing the SBV file format.

0:00:07.000,0:00:10.000
The SBV format is commonly used for storing subtitles for videos.
```

Les fichiers SBV peuvent être créés et édités avec une variété de logiciels d'édition de sous-titres, tels que Subtitle Workshop, Aegisub ou Subtitle Edit. Une fois les sous-titres terminés, le fichier SBV peut être importé dans un programme de montage vidéo pour créer une vidéo sous-titrée. Dans l'ensemble, le format de fichier SBV est un format simple et largement utilisé pour stocker des sous-titres, compatible avec de nombreux programmes et lecteurs de montage vidéo.

## Comment ouvrir le fichier SBV?

Les fichiers SBV sont des fichiers texte, vous pouvez donc les ouvrir à l'aide de n'importe quel éditeur de texte, par exemple Notepad ou Notepad++.
