## Qu'est-ce qu'un fichier CFG ?

Le fichier CFG est un fichier de configuration de clavier XML utilisé par les émulateurs de jeux vidéo d'arcade MAME. Il s'agit d'un élément crucial pour personnaliser les commandes du clavier et les raccourcis clavier en fonction des préférences du joueur. Ces fichiers stockent les mappages et les paramètres qui déterminent la façon dont le clavier interagit avec l'émulateur pendant le jeu. En modifiant ce fichier, les utilisateurs peuvent personnaliser leur expérience de jeu en attribuant des touches de clavier spécifiques à des actions au sein du jeu, telles que l'insertion de pièces, le démarrage, le mouvement et diverses autres fonctions.

## Fichier de configuration MAME

MAME, qui signifie **Multiple Arcade Machine Emulator**, est une application logicielle qui vous permet d'émuler et de jouer à des jeux d'arcade sur votre ordinateur. MAME utilise des fichiers de configuration pour personnaliser son comportement et ses paramètres. Ces fichiers de configuration se trouvent généralement dans le dossier « cfg » de votre répertoire MAME.

Voici les principaux fichiers de configuration que vous pouvez rencontrer lors de l'installation et de la configuration de MAME :

1. **mame.ini :** Il s'agit du fichier de configuration principal de MAME. Il contient des paramètres globaux qui s'appliquent à tous les jeux. Vous pouvez trouver ce fichier dans le répertoire racine de votre installation MAME.

1. **default.cfg :** Ce fichier stocke les paramètres par défaut de tous les jeux qui n'ont pas leurs propres fichiers de configuration. Il est utilisé comme solution de secours pour les paramètres spécifiques au jeu.

1. **game-special.cfg :** Ces fichiers sont utilisés pour stocker les paramètres de jeux individuels. Ils portent généralement le nom du fichier ROM du jeu auquel ils correspondent. Par exemple, si vous avez un jeu appelé "pacman.zip", le fichier de configuration correspondant sera "pacman.cfg".

Voici quelques paramètres courants que vous pourriez trouver dans le fichier de configuration MAME.

1. **rompath :** Spécifie le répertoire où se trouvent les ROM de vos jeux d'arcade.

1. **cfg_directory :** Spécifie le répertoire dans lequel les fichiers de configuration spécifiques au jeu sont stockés.

1. **nvram_directory :** Spécifie le répertoire dans lequel les fichiers RAM non volatile (NVRAM) sont stockés. La NVRAM stocke les meilleurs scores et d'autres données spécifiques au jeu.

1. **artwork_directory :** Spécifie le répertoire dans lequel les fichiers d'illustrations (tels que les cadres, les chapiteaux et les dépliants) sont stockés.

1. **samplepath :** Spécifie le répertoire où se trouvent les exemples de fichiers audio.

1. **cheatpath :** Spécifie le répertoire où se trouvent les fichiers de triche.

Vous pouvez également configurer divers autres paramètres tels que les options vidéo et audio, les commandes et les périphériques d'entrée. Pour modifier ces paramètres, vous pouvez ouvrir le fichier de configuration dans l'éditeur de texte et apporter les modifications nécessaires.

## MAME

MAME, qui signifie **"Multiple Arcade Machine Emulator"**, est une application logicielle conçue pour émuler et reproduire le matériel des machines d'arcade et des consoles de jeux d'arcade vintage. Il permet aux utilisateurs de jouer à une vaste bibliothèque de jeux d’arcade classiques sur des ordinateurs modernes et autres appareils compatibles. MAME est un projet open source et est devenu un émulateur incontournable pour préserver et profiter de la riche histoire des jeux d'arcade.

1. **Émulation :** L'objectif principal de MAME est d'émuler avec précision le matériel des machines d'arcade, y compris leurs unités centrales de traitement (CPU), leurs puces sonores, leurs puces graphiques et leurs périphériques d'entrée. Ce niveau de précision garantit que les jeux se comportent aussi près que possible de l’expérience d’arcade originale.

1. **Compatibilité :** MAME prend en charge une large gamme de ROM de jeux d'arcade, ce qui en fait l'un des émulateurs d'arcade les plus complets disponibles. Il peut exécuter des jeux à partir de diverses plates-formes d'arcade, y compris des jeux classiques des années 70, 80, 90 et même certains titres d'arcade plus récents.

1. **Préservation :** L'une des missions principales de MAME est de préserver l'histoire des jeux d'arcade. En émulant avec précision le matériel d'arcade, MAME aide à prévenir la perte des jeux classiques et garantit que les générations futures pourront les découvrir comme prévu à l'origine.

1. **Front-Ends :** De nombreux utilisateurs utilisent des applications frontales qui fournissent une interface graphique pour gérer et lancer des jeux via MAME. Ces frontaux facilitent la navigation dans la vaste bibliothèque de jeux de MAME.
