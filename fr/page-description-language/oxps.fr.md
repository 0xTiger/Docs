## Qu'est-ce qu'un fichier OXPS ? ##

Le format de fichier OXPS est connu sous le nom de Open XML Paper Specification. C'est un langage de description de page et un format de document. Microsoft est le développeur de ce format. Le format de fichier OXPS est très familier à ces fichiers PDF. La configuration du fichier OXPS est la même que celle d'un format XPS. La seule différence est la description d'OXPS en tant que format ouvert. L'extension de fichier .oxps est la version la plus récente du format de fichier XPS créé par Microsoft et Ecma International. Si nous parlons de la spécification, les fichiers OXPS et XPS sont utilisés pour publier, partager et allouer des documents à mise en page fixe. En remplacement des fichiers PDF, le format de fichier OXPS peut stocker du texte et des données multimédias, avec des fonctionnalités d'organisation et des caractéristiques de mise en page d'impression.

## Bref historique ##

L'équipe Microsoft a choisi Global Graphics pour fournir des conseils sur XPS en 2003, et l'équipe de développement a travaillé en étroite collaboration avec les équipes Windows pour développer la spécification et l'architecture de référence pour le nouveau format. Reconnaissant les spécifications XPS, en juin 2007, Ecma International a créé le comité technique 46 (TC46) pour développer une norme basée sur la spécification OpenXPS. La spécification papier Open XML approuvée par Ecma International est une norme Ecma (ECMA-388) lors de la 97e Assemblée générale qui s'est tenue à Budapest le 16 juin 2009.
