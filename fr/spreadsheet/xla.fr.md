## Qu'est-ce qu'un fichier XLA ?

Un fichier avec l'extension .xla est un format de fichier de compléments Microsoft Excel qui ajoute des modules, des fonctions supplémentaires et d'autres outils à Microsoft Excel. Les compléments fournissent des fonctionnalités supplémentaires aux feuilles de calcul Excel et sont pris en charge dans la version 2007 ou ultérieure. Les fichiers XLA ont le type de média mime application/vnd.ms-excel. Ceux-ci peuvent être ouverts à l'aide de Microsoft Excel disponible pour Windows et MacOS.

## Format de fichier XLA

Les fichiers XLA sont basés sur le format de données binaires. Cependant, plus d'informations sur ses spécifications de format de fichier ne sont pas disponibles. L'hypothèse générale est que XLA suit la [[MS-XLS] - Structure du format de fichier binaire Excel](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx).
