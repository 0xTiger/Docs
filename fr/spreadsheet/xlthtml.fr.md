## Qu'est-ce qu'un fichier XLTHTML ?

Un fichier avec l'extension .xlthtml est une version [HTML](/fr/web/html/) des fichiers modèles [XLT](/fr/spreadsheet/xlt/) et a été généré à l'aide de Microsoft Excel au début. Ce format n'est plus pris en charge par Excel et a été remplacé par l'exportation de données aux formats de fichier [MHT](/fr/web/mht/) et [MHTML](/fr/web/mhtml/). L'idée de base derrière le fichier XLTHTML était de permettre aux utilisateurs de visualiser les fichiers de modèle XLT dans un navigateur Web au lieu d'avoir Excel installé sur le système. Les fichiers exportés au format de fichier XLTHTML peuvent être ouverts dans les navigateurs.

## Format de fichier XLTHTML - Plus d'informations

Les fichiers XLTHTML utilisaient le format de fichier HTML pour enregistrer les modèles XLT à ouvrir dans un navigateur Web. XLT lui-même est basé sur le format de fichier binaire Excel qui a été introduit avec la version Excel 97-2003. Ces fichiers peuvent être utilisés comme fichiers de départ avec un formatage défini pour créer des documents normaux pour la population de données de feuille de calcul. Pour référence, XLT est basé sur le binaire [spécifications du format de fichier XLS](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) qui est documenté par Microsoft pour référence du développeur.
