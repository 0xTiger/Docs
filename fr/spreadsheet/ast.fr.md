## Qu'est-ce qu'un fichier AST ?

Un fichier AST est un fichier de modèle de feuille de calcul créé avec le logiciel Ability Spreadsheet, un composant de feuille de calcul de la suite Ability Office. Ces fichiers modèles sont utilisés pour créer un ensemble initial de modèles de feuilles de calcul pouvant être utilisés pour créer des fichiers [AWS](/fr/spreadsheet/aws/). Les fichiers AST peuvent être ouverts avec le logiciel de tableur Ability.

## Format de fichier AST - Plus d'informations

Les fichiers AST sont enregistrés sur le disque en tant que fichiers binaires et peuvent être ouverts dans le logiciel Ability Spreadsheet. Bien qu'entièrement compatibles avec les formats de fichiers Microsoft XLS et XLSX, les spécifications de format de fichier AST ne sont pas disponibles publiquement pour la référence du développeur. Les fichiers AST peuvent être enregistrés sur disque aux formats de fichier [PDF](/fr/pdf/) et [XPS](/fr/page-description-language/xps/).
