## Qu'est-ce qu'un fichier SXC ? ##

Le format de fichier SXC (Sun XML Calc) appartient à une suite bureautique appelée OpenOffice.org. Ce format répond généralement aux besoins des utilisateurs en matière de feuille de calcul, car il s'agit d'un format de fichier de feuille de calcul basé sur XML. Le format SXC prend en charge les formules, les fonctions, les macros et les graphiques ainsi que DataPilot, ce qui est une fonctionnalité incroyable car il individualise et fournit automatiquement des résumés des données brutes importées. Les fichiers créés avec ce logiciel sont enregistrés avec l'extension .sxc.


## Format de fichier SXC ##

Les fichiers, ayant des extensions .sxc, peuvent être ouverts avec Microsoft Excel qui est un tableur de la suite Microsoft Office. Il existe d'autres logiciels qui peuvent ouvrir des fichiers avec l'extension .sxc, c'est-à-dire Libre Office, StarOffice, etc.

Le type de contenu pour ce format est :

* application/vnd.sun.xml.calc
* application/vnd.sun.xml.calc.template

## Compatibilité des fichiers ##

Les fichiers SXC sont compatibles avec Apache Open Office calc et les données peuvent également être exportées vers un fichier Microsoft Excel et d'autres formats.

## La gestion des fichiers ##

Les commandes ImportMatrix et ExportMatrix peuvent lire et écrire le fichier SXC. Lorsque la source est un fichier SXC et que la cible est également un fichier SXC, **f** est considéré comme un fichier de feuille de calcul Sun XML Calc. Les matrices d'importation et d'exportation prennent en charge ce format, qui est automatiquement détecté lorsque le format n'est pas spécifié et que le nom de fichier se termine par .sxc.

Pour ouvrir le fichier SXC, un programme approprié doit être associé au fichier. Chaque fois qu'un fichier avec l'extension .sxc doit être ouvert, assurez-vous qu'il est sans erreur et qu'il n'est pas corrompu. Toute erreur dans le fichier peut être facilement gérée par l'utilisateur et cela ne nécessite pas de support technique la plupart du temps.
