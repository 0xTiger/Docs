## Qu'est-ce qu'un fichier CTS ?

Un fichier CTS est un fichier de feuille de calcul créé avec le programme open source de saisie et d'organisation de données multiplateforme, **TreeSheets**. Il contient des données hiérarchiques saisies dans différents formats dans la feuille de calcul, telles que des notes, des listes, des calendriers, des listes de tâches, des notes, des cartes mentales et un certain nombre d'autres formats. Les fichiers CTS ne peuvent être ouverts qu'avec TreeSheets qui est léger, rapide et fonctionne sous Windows, Linux et macOS.

## Format de fichier CTS - Plus d'informations

TreeSheets enregistre les fichiers CTS dans son propre format de fichier propriétaire. Les spécifications de format de fichier du format de fichier CTS ne sont pas disponibles publiquement et c'est pourquoi les fichiers CTS ne peuvent être ouverts que dans le logiciel TreeSheets.

### Exporter les fichiers TreeSheets CTS vers d'autres formats

Les utilisateurs de TreeSheets peuvent exporter les feuilles de calcul CTS vers plusieurs formats de fichiers différents. Les fichiers CTS peuvent être exportés aux formats de fichier [HTML](/fr/web/html/), [JSON](/fr/web/json/), [CSV](/fr/spreadsheet/csv/) et [Image](/fr/image/) .

## À propos du logiciel TreeSheets

TreeSheets est un logiciel open source disponible pour multiplateforme. Il vous permet de créer des feuilles de calcul hiérarchiques et vous donne la liberté de mettre tous les types de données dans un seul fichier de feuille de calcul. L'avantage d'utiliser TreeSheets est que vous pouvez mettre toutes vos discussions sous forme de notes, de cartes mentales, de données textuelles, d'images et d'autres types de données dans un seul fichier de feuille de calcul.
