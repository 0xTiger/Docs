## Qu'est-ce qu'un fichier TMVT ?

Un fichier TMVT est un fichier de modèle de feuille de calcul de chronologie TimeMap qui contient des paramètres prédéfinis pour la création de [fichiers TMV](/fr/spreadsheet/tmv/). Contrairement aux fichiers TMV, TMVT contient des paramètres tels que les styles d'indicateur de fait, les arrière-plans, les styles de titre et les styles d'échelle. Les fichiers TMVT peuvent être partagés avec les utilisateurs et réutilisés par eux pour conserver les mêmes paramètres et créer leurs propres fichiers de chronologie pour l'uniformité. Ceux-ci peuvent être ouverts avec le logiciel TimeMap par [LexisNexis](https://www.lexisnexis.com/en-us/products/casemap.page).

## Format de fichier TMVT

Les fichiers TMVT sont créés et enregistrés sur disque au format de fichier binaire.
