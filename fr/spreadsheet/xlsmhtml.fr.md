## Qu'est-ce qu'un fichier XLSMHTML ?

Un fichier avec l'extension .xlsmhtml est une version [HTML](/fr/web/html/) du fichier XLS d'origine qui est exporté depuis Microsoft Excel pour afficher les données dans un navigateur. Le format a longtemps été abandonné par Excel et n'est plus pris en charge. Le fichier XLSMHTML exporté suivait le format de fichier XLS et ceux-ci pouvaient être ouverts et affichés dans les navigateurs Web. Excel permet d'exporter les données de la feuille de calcul vers les formats de fichier [MHT](/fr/web/mht/) et [MHTML](/fr/web/mhtml/), et sont largement utilisés maintenant.

## Format de fichier XLSMHTML - Plus d'informations

Les fichiers XLSMHTML ont été générés avec du code HTML pour représenter les données de la feuille de calcul dans un format similaire à XLS. Le code HTML à l'intérieur du fichier XLSMHTML est organisé de manière à ce que le fichier de sortie ressemble au fichier d'origine lorsqu'il est ouvert dans un navigateur Web. Les spécifications de format de fichier du fichier XLSMHTML ne sont pas en tant que telles rendues publiques par Excel et aucune référence de ce type n'est disponible pour que les développeurs comprennent et développent des applications pour celui-ci.
