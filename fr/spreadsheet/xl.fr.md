## Qu'est-ce qu'un fichier XL ?

Un fichier avec l'extension .xl est un fichier de feuille de calcul Excel, similaire à [XLS](/fr/spreadsheet/xls/), qui stocke les données dans des lignes et des colonnes. C'était un format de fichier antérieur qui n'est plus pris en charge par Excel maintenant. Les données stockées dans des fichiers XL pourraient être exportées vers d'autres formats tels que CSV. Ceux-ci peuvent également être convertis en PDF à l'aide de la méthode Enregistrer sous par défaut d'Excel. Les fichiers XL ne pouvaient être ouverts qu'avec Microsoft Excel.

## Format de fichier XL

Les fichiers XL sont enregistrés en tant que fichiers binaires et leurs spécifications de format de fichier interne ne sont documentées nulle part. Cependant, le format de fichier binaire général pour le format de fichier [Spécifications du format de fichier XLS](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) peut également être pris en compte si quelqu'un essaie d'écrire un morceau de code pour désosser ces fichiers.
