## Qu'est-ce qu'un fichier DFG ?

Un fichier avec l'extension .dfg est un fichier de feuille de calcul créé par Interscape Inc. pour le logiciel Interscape Data Flask. Les données sont renseignées dans les fichiers DFG en lignes et en colonnes. Data Flask est un logiciel de feuille de calcul, conçu pour conserver les types de données, la source et les informations clés. Il peut migrer des tables et des index en modifiant uniquement la chaîne de connexion. Il offre des fonctionnalités telles que la ligne de commande VBScript, les formules VBScript et les macros ainsi que des scripts SQL pour la mise à jour et la migration des données. Les fichiers DFG sont enregistrés sous forme de fichiers binaires sur le disque et peuvent être ouverts avec le logiciel Interscape Data Flask.

## Format de fichier DFG - Plus d'informations

Les fichiers DFG enregistrent les informations sur les données de la feuille de calcul sous forme de fichiers binaires sur le disque. La structure de données interne et les détails de format de ces fichiers ne sont cependant pas connus.
