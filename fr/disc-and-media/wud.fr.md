## Qu'est-ce qu'un fichier WUD ?

Un fichier avec l'extension .wud est une image disque créée à partir d'un disque de jeu Wii U. Le fichier image disque résultant peut être chargé par des émulateurs de jeux vidéo tels que [Cemu](https://cemu.info/) pour émuler le jeu. Les fichiers WUD sont de grande taille et sont donc créés en plusieurs segments afin d'être facilement transférés. Les informations de jeu sont exportées sous forme de segments de fichier WUD d'une taille de fichier d'environ 2 Go. L'application d'ouverture, comme Cemu, a besoin que tous ces fichiers segmentés soient fusionnés en un seul fichier WUD avant de les ouvrir.

## Format de fichier WUD

Les fichiers WUD sont enregistrés sur disque dans un format de fichier propriétaire et peuvent être chargés dans l'émulateur Cemu pour émuler les applications Wii U sur PC. En raison de sa grande taille, les fichiers WUD sont [compressés](/fr/compression/) pour réduire leur taille de fichier. Le fichier compressé résultant est appelé fichier WUX.
