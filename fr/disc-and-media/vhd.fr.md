## Qu'est-ce qu'un fichier VHD ?

VHD est l'abréviation de Virtual Hard Disk. Ce sont des formats de fichiers qui représentent un disque dur virtuel. Les données présentes sur un disque dur physique peuvent y être impliquées tout comme les partitions du disque dur et du système de fichiers qui comprend des dossiers.

Pour une machine virtuelle, les fichiers VHD sont considérés comme des disques durs. Dans les versions modernes de Windows, ceux-ci sont intégrés. Ce type de disque dur virtuel spécifique permet à différents systèmes d'exploitation de résider sur la machine hôte commune. Il offre une grande chance aux développeurs de tester le logiciel sur plusieurs systèmes d'exploitation sans effectuer le processus d'installation d'un autre disque dur. L'utilisation de ce format de fichier de fichiers VHD présente de nombreux avantages, car ceux-ci offrent de nombreuses fonctionnalités.


## Bref historique ##

**Connectix** a développé ce format de fichier en tant que produit PC virtuel pour leur utilisation. En 2003, Microsoft a acquis Connectix et a lancé d'autres versions de ce format telles que VHDx ainsi que les meilleures caractéristiques et fonctionnalités qui manquaient dans la première version. En 2012, le VHDx a été ajouté à Windows. Il a été ajouté à Windows pour ajouter plus de capacité de stockage, une protection contre la corruption des données, une optimisation pour éviter la dégradation des performances. Ces fonctionnalités ont été ajoutées pour les disques physiques avec de grands secteurs.

## Format de fichier VHD ##

Un système de fichiers hôte est accessible pour y résider les fichiers VHD. Ceux-ci ont des caractéristiques de l'image fixe du disque de garde qui permet à l'utilisateur d'accéder à un fichier auquel est allouée la taille d'un disque virtuel spécifique. Les fichiers peuvent modifier le disque dur d'une machine virtuelle.

Ces fichiers VHD peuvent être échangés facilement du disque dur virtuel vers le système de fichiers de l'hôte. Ce format a également la capacité de sauvegarde et de restauration et de sécurité antivirus. Les fichiers VHD peuvent prendre en charge des formats spécifiques tels qu'une image de disque dur fixe, une image de disque dur de différenciation et une image de disque dur dynamique.
