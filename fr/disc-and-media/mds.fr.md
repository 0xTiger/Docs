## Qu'est-ce qu'un fichier MDS ?

Le MDS (Media Descriptor Sidecar File) est un fichier associé aux logiciels Alcohol 120% et Daemon Tools, tous deux utilisés pour créer et gérer des lecteurs de CD et DVD virtuels. Le fichier MDS contient des informations sur la disposition des données sur le disque, y compris l'emplacement de tous les fichiers et la structure du disque. Cela permet au lecteur virtuel d'émuler le comportement d'un disque physique, de sorte que le logiciel puisse lire les données sur le disque virtuel comme s'il s'agissait d'un vrai disque.

Lorsque vous souhaitez monter un fichier image sur un lecteur virtuel, le fichier MDS est utilisé avec le fichier image correspondant dans un format tel que ISO, BIN ou CUE. Le fichier MDS contient une description de la disposition des données sur le disque et le logiciel du lecteur virtuel utilise ces informations pour créer un disque virtuel. Cela vous permet d'utiliser le logiciel comme si vous utilisiez un disque physique, sans avoir à insérer physiquement le disque dans le lecteur.

## Relation avec l'alcool 120%

Alcohol 120% est un logiciel populaire pour créer et gérer des lecteurs de CD et de DVD virtuels, et il utilise le format de fichier MDS pour enregistrer et accéder aux images de disque. Le format de fichier MDS est la propriété de Alcohol 120% et ne peut être ouvert et utilisé que par le logiciel Alcohol 120% ou un autre logiciel qui le prend en charge.

## Comment ouvrir le fichier MDS ?

Pour ouvrir un fichier MDS dans Alcohol 120%, vous devez avoir installé le logiciel sur votre ordinateur. Une fois que vous avez installé Alcohol 120%, vous pouvez ouvrir le fichier MDS en procédant comme suit :

1. Lancez Alcool 120 %.
2. Cliquez sur le menu "Fichier" et sélectionnez "Ouvrir".
3. Accédez à l'emplacement du fichier MDS sur votre ordinateur et sélectionnez-le.
4. Le fichier MDS sera maintenant ouvert dans Alcohol 120% et vous serez invité à sélectionner le fichier image correspondant (tel que ISO, BIN ou CUE).
5. Après avoir sélectionné le fichier image, Alcohol 120% montera l'image sur un lecteur virtuel.

Alternativement, vous pouvez également ouvrir un fichier MDS en double-cliquant dessus, si Alcohol 120% est défini comme programme par défaut pour ouvrir les fichiers MDS. Vous pouvez également utiliser d'autres logiciels prenant en charge le format MDS, tels que Daemon Tools, Virtual CloneDrive, Virtual Drive Manager et MagicISO.
