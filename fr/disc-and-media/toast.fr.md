## Qu'est-ce qu'un fichier Toast ?
Le logiciel Roxio Toast crée une image de CD ou de DVD et enregistre le fichier avec l'extension .toast. Roxio Toast est une application logicielle bien connue de création de disques optiques et de conversion multimédia pour Macintosh, introduite pour la première fois par Miles Software GmbH, puis achetée par Adaptec. Le fichier Toast peut contenir n'importe quel type de disque tel qu'un disque vidéo, audio ou de données. Il peut également s'agir d'une copie et d'une autre image de disque.

## Format de fichier toast
Le format de fichier Toast est utilisé par le logiciel Roxio Toast pour enregistrer sa sortie dans un fichier avec l'extension .toast. Bien que les disques de différents types puissent être gravés directement via les systèmes Mac OS, Toast fournit un contrôle supplémentaire sur le processus et offre également des fonctionnalités supplémentaires, telles que le catalogage, le suivi des fichiers gravés sur disque et la récupération de fichiers pour les disques endommagés. prend également en charge les formats audio et vidéo non pris en charge par Quicktime, tels que [FLAC](/fr/audio/flac/) et [OGG](/fr/audio/ogg/). Comme il s'agit d'un format de fichier propriétaire, les informations sur la structure entière du format de fichier ne sont pas disponibles.
### Types de disques
Par conséquent, Roxio Toast peut créer des fichiers Toast contenant les données d'image disque des types suivants :
- **Création de disques vidéo** : Toast peut créer des fichiers Toast contenant des disques vidéo en utilisant les projets suivants :
- DVD-Vidéo
- Vidéo Blu-ray
- DVD haute définition
- Dossiers VIDEO_TS
- Compilation VIDEO_TS
- Dossier BDMV
- Archives AVCHD
- **Création de disques de données** : Toast peut créer des fichiers Toast contenant des disques de données en utilisant les projets suivants :
- Mac uniquement
- Mac et PC
- DVD ROM
- Hybride personnalisé
- Volume Mac
- Disque de photos
- **Création de disques vidéo** : Toast peut créer des fichiers Toast contenant des disques de données en utilisant les projets suivants :
- CD audio
- DVD de musique
- Disque MP3
- CD audio amélioré
- CD en mode mixte
- **Copie de disque** : Toast prend en charge la copie des manières suivantes :
- Copie de disque
- Copie d'images
- Fusion d'images de disque
