## Qu'est-ce qu'un fichier WBFS ?

Un fichier WBFS (Wii Backup File System) est une image de disque de jeu vidéo créée pour la Nintendo Wii. Il contient un jeu Wii qui peut être chargé en tant qu'image pour jouer. Il est généré à l'aide de l'outil Wiimms WBPS ou du Wii Backup Manager et peut être joué sur une Nintendo Wii en le chargeant à partir d'un USB Loader GX ou d'un émulateur Wii tel que Dolphin. Le jeu est stocké sur ces périphériques de stockage externes en utilisant le format de fichier WBFS. La Wii est une plateforme de jeu distribuée par Nintendo. Le système de fichiers WBFS a été développé par le codeur homebrew Wii Waninkoko.

## Format de fichier WBFS - Plus d'informations

WBFS sont des fichiers binaires qui ont un format de fichier propriétaire. Ses spécifications de format de fichier ne sont pas disponibles publiquement.
