## Qu'est-ce qu'un fichier MUI ?

MUI fait référence aux fichiers Myriad User Instrument. Myriad User Instrument (MUI) est un format de fichier propriétaire utilisé par le logiciel Harmony Assistant, qui est un logiciel de notation et de composition musicale développé par Myriad Software. Les fichiers MUI contiennent des sons d'instruments numériques et d'autres données musicales utilisées par le programme Harmony Assistant.

Les fichiers MUI ne sont généralement pas compatibles avec d'autres logiciels de musique, car ils sont conçus pour fonctionner spécifiquement avec le programme Harmony Assistant. Pour ouvrir un fichier MUI, vous devez avoir le logiciel Harmony Assistant installé sur votre ordinateur. Une fois Harmony Assistant installé, vous pouvez importer le fichier MUI dans le programme et utiliser les sons d'instruments inclus et d'autres données pour composer et arranger de la musique.

## Relation avec Harmony Assistant

Le format de fichier MUI est lié au logiciel Harmony Assistant, qui est un programme de composition musicale et de notation développé par Myriad Software. Le format de fichier MUI est utilisé par Harmony Assistant pour stocker les sons d'instruments personnalisés et d'autres données musicales créées par les utilisateurs.

Harmony Assistant permet aux utilisateurs de créer leurs propres instruments numériques à l'aide de l'éditeur de sons intégré ou en important des sons provenant d'autres sources. Ces instruments personnalisés peuvent être enregistrés sous forme de fichiers MUI et utilisés dans de futures compositions.

Les fichiers MUI peuvent également être partagés avec d'autres utilisateurs d'Harmony Assistant, permettant aux musiciens et compositeurs d'échanger leurs instruments personnalisés et leurs idées musicales. Cependant, les fichiers MUI ne peuvent pas être utilisés avec d'autres logiciels musicaux, car ils sont spécifiques au programme Harmony Assistant.

## Comment ouvrir le fichier MUI?

Pour ouvrir le fichier MUI, vous devez avoir installé le logiciel Harmony Assistant sur votre ordinateur. Voici les étapes générales pour ouvrir un fichier MUI à l'aide de Harmony Assistant :

1. Lancez le logiciel Harmony Assistant.
2. Cliquez sur « Fichier » dans la barre de menu et sélectionnez « Ouvrir » dans le menu déroulant.
3. Accédez à l'emplacement du fichier MUI que vous souhaitez ouvrir et sélectionnez-le.
4. Cliquez sur « Ouvrir » pour importer le fichier MUI dans Harmony Assistant.
5. Une fois le fichier MUI importé, vous devriez pouvoir accéder aux sons d'instruments inclus et à d'autres données dans le logiciel Harmony Assistant.
