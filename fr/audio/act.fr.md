## Qu'est-ce qu'un fichier ACT ? ##

Un format ACT ou Adobe Color Table (également connu sous le nom de fichier Color Lookup Table) est un type de base de données Adobe Photoshop qui stocke un ensemble de couleurs prédéfinies. Vous pouvez ajouter ou supprimer des couleurs lors de l'enregistrement d'une image pour la publication en ligne afin de favoriser une image de meilleure qualité ou une taille de fichier plus petite. Vous pourriez avoir un format ADPCM Encrypted Audio plutôt s'il n'est pas utilisé avec Illustrator. La modulation adaptative différentielle par impulsions codées est une technique de compression audio utilisée par divers appareils de musique et appareils d'écoute.

Le format de fichier ACT est également utilisé par le système de fichiers Alma CAD/CAM. Ce sont les directives utilisées par les machines-outils 3D pour comprendre comment couper n'importe quoi. Un lecteur Genesis3D, une action de développeur de jeux DS ou un schéma d'action de l'assistant de détail FoxPro sont tous des exemples d'ACT.

## Spécification du format de fichier ACT

De nombreux enregistreurs utilisant le système ACT ne le font que pour la plus petite option d'enregistrement ; si la qualité est améliorée, un format distinct, généralement WAV, est utilisé, ce qui donne des documents beaucoup plus volumineux.

Il existe plusieurs générations d'ACT; depuis juin 2009, les pistes créées par les appareils suivants ne peuvent tout simplement pas être reconnues par tous les lecteurs de musique ou applications de conversion standard gratuits, ne laissant que les outils MP3 fournis.

L'optimisation des images pour la distribution Web se fait avec des fichiers ACT. Les clients peuvent tester la qualité de l'image par rapport aux bases de données d'images générées en augmentant ou en diminuant le nombre de couleurs dans le CLUT.

En appuyant sur l'option Table des couleurs et en sélectionnant Charger les données de couleur dans la boîte de dialogue Exporter pour le Web, les tables de couleurs Adobe peuvent être téléchargées. Le répertoire */Presets/en_US/Save* for Web Settings/Color Tables/ contient plusieurs fichiers ACT préconfigurés fournis avec le système Photoshop.

Les scripts Act peuvent être générés uniquement dans Genesis 3D ou en transformant d'autres modèles 3D au standard ACT à l'aide de Genesis3D Actor Designer
