## Qu'est-ce qu'un fichier AA ?

Un fichier avec une extension .aa est spécifié pour stocker le format de fichier de données audio numériques incluant le fichier du livre audio Audible. Ces fichiers sont utilisés pour stocker les enregistrements audio parlés de l'Audible. Les fichiers AA sont également conçus pour stocker des livres audio. Ces fichiers AA sont conçus pour stocker le mot audio sous forme parlée et prendre en charge les signets, les marqueurs de chapitre, etc. Les données de ces fichiers incluent le brut mais le flux au format de codage audio. Outre le cryptage de la gestion des droits numériques, ces fichiers contiennent des clips audio.

## Format de fichier AA

Amazon détient l'audible et ces fichiers peuvent être utilisés sur divers appareils. Pour utiliser ces fichiers sur Windows Media Player, une conversion vers un format de fichier plus efficace comme [MP3](/fr/audio/mp3/) est nécessaire. Le fichier AA est une catégorie du fichier audio mais il y a une différence entre un simple fichier MP3 et un fichier audible. Ces fichiers peuvent être protégés par DRM afin qu'ils ne puissent pas être convertis dans d'autres formats audio même s'ils sont largement utilisés.

### Bref historique du format de fichier AA

La société Amazon a lancé son produit connu sous le nom de lecteur Audible en 1998. Audible est un service en ligne américain qui fournit la possibilité de livres audio et de podcasts. Ces fichiers avec l'extension .aa ont donc leur origine avec le lancement de ce service. En 2003, Audible a conclu un accord avec Apple. Il a la capacité d'être un fournisseur exclusif de livres iTunes. En 2008, Amazon prétendait acheter Audible. Au cours des années suivantes, il a été mis à jour et modifié en fonction des besoins modernes et des services de compatibilité. En 2020, il a été modifié pour la dernière fois lorsqu'Audible a apporté des modifications de paiement.

## Spécifications du format de fichier AA

Ces fichiers peuvent être compatibles avec divers appareils. Les lecteurs de ces fichiers sont disponibles sur les appareils Android et iOS. Certaines versions de ces fichiers ne sont pas compatibles avec Linux. Les programmes associés à ces fichiers sont disponibles en ligne pour Windows PC. Pour Mac OS, Apple iTunes est le programme associé à ces fichiers. Ces fichiers fournissent également une encapsulation du son encodé en MP3. Plusieurs qualités sont disponibles auprès d'Audible, mais les téléchargements ne sont actuellement disponibles que pour les formats "Format 4" et "Amélioré".
