## Qu'est-ce qu'un fichier M4R ?

Le format de fichier .m4r est un type de format de fichier audio utilisé pour les sonneries iPhone. Le format est essentiellement une version renommée du format de conteneur multimédia MPEG-4 Part 14 (MP4), avec l'extension de fichier .m4r indiquant que le fichier est une sonnerie. Les fichiers M4R sont codés au format Advanced Audio Coding (AAC) et peuvent être créés à l'aide de diverses applications logicielles prenant en charge la création de sonneries. Une fois créés, ils peuvent être transférés sur un iPhone à l'aide d'iTunes ou d'autres méthodes, puis définis comme sonnerie de l'appareil. Bien que les fichiers M4R soient conçus pour les sonneries iPhone, le format n'est pas exclusif aux appareils Apple et peut être lu sur d'autres appareils prenant en charge les fichiers audio AAC.

## Format de fichier M4R - Plus d'informations

Le format .m4r a été introduit par Apple en 2007 avec la sortie de l'iPhone. Avant cela, les sonneries iPhone étaient au format .m4a, qui est également un format audio AAC mais qui n'était pas spécifiquement conçu pour les sonneries. Les fichiers M4R ont une durée maximale de 40 secondes, ce qui correspond à la durée maximale autorisée pour une sonnerie sur un iPhone. Les fichiers M4R sont le format utilisé par l'iPhone pour les sonneries.

Les fichiers M4R peuvent contenir à la fois du contenu audio et vidéo, bien que le contenu vidéo soit généralement ignoré par l'iPhone lorsque le fichier est utilisé comme sonnerie. Les fichiers M4R sont créés en renommant un fichier .m4a existant en .m4r. Le contenu du fichier ne change pas, mais l'extension .m4r indique à iTunes que le fichier doit être traité comme une sonnerie.

Les fichiers M4R peuvent être créés à l'aide de diverses applications logicielles, notamment iTunes, qui inclut un créateur de sonnerie intégré. Des applications logicielles tierces, telles que GarageBand et Ringtone Maker, peuvent également être utilisées pour créer des fichiers M4R. Les fichiers M4R peuvent être transférés sur un iPhone à l'aide d'iTunes ou en synchronisant le fichier sur l'iPhone à l'aide de services de stockage cloud tels qu'iCloud ou Dropbox.
