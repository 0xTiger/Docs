## Qu'est-ce qu'un fichier 8SVX ? ##

Un 8SVX est le format de fichier audio standard développé par Electronic Arts. Il est considéré comme un sous-type de données du format de fichier IFF. Le fichier est utilisé pour stocker des données audio 8 bits dans des morceaux. Dans des blocs de descripteurs séparés, des métadonnées sont contenues sur le flux de données 8SVX. C'est un format de fichier audio numérique utilisé par les PC Amiga. Les systèmes informatiques d'Amiga sont désormais rarement utilisés, de sorte que ces fichiers 8SVX sont généralement utilisés pour conserver et écouter l'ancienne musique.

## Format de fichier 8SVX

Le fichier 8SVX peut stocker des enregistrements audio en qualité 8 bits dérivée de la spécification [IFF](/fr/audio/aiff/). Ce sont des fichiers qui contiennent des enregistrements audio comme d'autres fichiers contenant de l'audio. Il comprenait également le format de conteneur d'IFF. Dans ce format, plusieurs formats de données ont été intégrés dans un fichier et utilisés pour plus de détails sur les flux inclus. Ceux-ci peuvent être utilisés seuls dans les conteneurs de fichiers audio uniquement des fichiers IFF. D'autre part, le multiplexage avec d'autres sous-types d'IFF peut entraîner des flux vidéo animés.

## Bref historique du format 8SVX

Il a été initialement publié en 1985 par Steve Hayes et Jerry Morrison. Un autre format archivé a été développé en 2009. Ce format de fichier était utilisé sur les systèmes informatiques Amiga pour la musique ancienne. Il est également connu sous le nom de fichier son Amiga 8 bits. Les logiciels audio conçus pour les PC Amiga prenaient en charge ce format de fichier. Il a d'abord été utilisé par de nombreux programmes d'édition sonore. La plupart de ses flux de données sont codés à l'aide de flux PCM. Une version modifiée d'Amiga a fourni un cadre de décodeur 8SVX. Les formats RIFF et WAVE co-développés par Microsoft et IBM sont réalisés sous l'influence du format conteneur 8SVX/IFF.

## Spécifications du format de fichier 8SVX

De nombreux programmes sont associés à ces fichiers sous Windows et Mac OS. De nombreux programmes peuvent être utilisés pour traiter ces fichiers. C'est un format audio encodé pour Commodore Amiga. Les fonctionnalités telles que l'attaque, la répétition de section et la libération sont prises en charge par ces fichiers. Ces fonctionnalités sont utilisées pour stocker des échantillons d'instruments de musique. La compression de données avec perte de Fibonacci-delta est disponible, ce qui entraîne un taux de compression de 50 % et une diminution de la fidélité.
