## Що таке файл VPK?

Файл із розширенням .vpk — це стиснений архівний пакет, який використовується для встановлення програм сторонніх розробників на ігрову консоль Sony PlayStation Vita. Ці файли можна встановити лише на джейлбрейк Vita PlayStation від HENkaku, який дозволив PS Vita та PSTV використовувати налаштований користувачем вміст. Архівний файл VPK містить увесь вміст, з якого складається гра, включаючи зображення, такі як файли [PNG](/uk/image/png/), файли налаштувань, такі як [.bin](/uk/disc-and-media/bin/) та будь-які конфігурації у форматі файлу [XML](/uk/web/xml/).

## Формат файлу VPK

Файли VPK зберігаються на диску як стандартні стиснені [ZIP](/uk/compression/zip/) архіви. Вони можуть містити кілька папок та інших пов’язаних файлів для програм сторонніх розробників, які будуть встановлені на ігровій консолі Vita. Щоб переглянути вміст файлу пакета VPK, перейменуйте його розширення з .vpu на .zip і розпакуйте вміст за допомогою стандартних утиліт декомпресії, таких як WinZip або WinRAR.

У Valvesoftware є докладна інформація про [формат файлу VPK](https://developer.valvesoftware.com/wiki/VPK_File_Format), на яку можна посилатися з точки зору розробника.

## Як встановити файли VPK?

Файли VPK можна встановити з утиліти командного рядка VPK.exe. В ОС Windows файли можна перекинути у файл vpk.exe, який повертає \*.vpk, що містить усі запаковані файли. Крім того, можна використовувати інструмент командного рядка, як показано нижче.

### Створіть VPK і додайте файли

```
vpk <dirname>
```

### Розпакуйте файли VPK

```
vpk x <vpkfile> <filename1> <filename2> ...
```
