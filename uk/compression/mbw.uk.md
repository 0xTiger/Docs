## Що таке файл MBW? ##

Файл із розширенням .mbw пов’язаний переважно з MBRWizard від Firesage Solutions. Майстер головного завантажувального запису (MBRWizard) створює резервну копію MBR вашого комп’ютера та зберігає його у файл MBW, який можна відновити за потреби. На додаток до копій MBR і VBR (Volume Boot Record), ці файли можуть містити GPT, визначені користувачем сектори або цілі образи дисків. MBR — це життєво важливий запис довжиною 512 байт на вашому жорсткому диску, який дозволяє завантажувати вашу операційну систему. Він також може містити таблицю розділів і 32-розрядний підпис, що ідентифікує дисковий носій.

## Підтримувані операційні системи ##

Формат файлу MPW підтримується лише операційною системою Windows.

## Можливі проблеми під час відкриття файлу ##

Якщо ви не можете відкрити та запустити файл MBW, можливо, виникла одна з таких можливих проблем:

* Пошкоджений файл MBW
* Неправильні посилання на файл MBW у записах реєстру
* Опис MBW видалено з реєстру Windows
* Інфікований файл з небажаним шкідливим програмним забезпеченням
* Комп'ютер не має достатніх апаратних ресурсів для роботи з файлом MBW
* Драйвери, які комп’ютер використовує для відкриття файлу MBW, застаріли
