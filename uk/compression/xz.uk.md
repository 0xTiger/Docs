## Що таке файл XZ?

XZ — це формат стисненого файлу, який використовує алгоритм стиснення LZMA2. Він був розроблений як заміна популярним форматам gzip і bzip2 і пропонує низку переваг перед цими старими стандартами. Файли XZ добре підтримуються багатьма платформами, їх можна швидко й легко розпакувати. Хоча вони не такі поширені, як файли [ZIP](/uk/compression/zip/) або [RAR](/uk/compression/rar/), архіви XZ можна використовувати для зберігання великих обсягів даних без шкоди для якості стиснення. Якщо вам потрібно стиснути або розпакувати великі файли, варто розглянути розширення файлу XZ.

## Формат файлу XZ

Файли XZ генеруються та зберігаються на диску як двійкові файли. Він використовує алгоритм LZMA для стиснення даних і може досягати коефіцієнта стиснення до 50%. Файли XZ зазвичай використовуються для розповсюдження програмного забезпечення та архівування резервних копій. Їх можна розпакувати за допомогою утиліти XZ, яка входить до складу більшості дистрибутивів Linux.

## Розархівуйте файли XZ на Linux/Unix

Щоб розпакувати файли XZ, спочатку встановіть пакет `xz-utils`:
```
$ sudo apt-get install xz-utils
```

Використовуйте команду unxz, щоб видобути файли .xz:
```
$ unxz compressed_xz_file.xz
```

або використовуючи опцію --decompress XZ:
```
$ xz --decompress compressed_xz_file.xz
```
