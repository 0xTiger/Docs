## Що таке файл BKF? ##

Файл із розширенням .bkf пов’язаний із WindowsXP від Microsoft Corporation. Всі операційні системи, які сприймають каталог файлів, використовують файл BKF, який підтримується утилітою. Якщо дані на комп’ютері користувача пошкоджені, перезаписані або видалені, файл BKF дуже корисний для відновлення інформації в системі користувача. Також є специфікація файлу BKF, його використовує комп'ютерна гра Moto Racer. Ці файли містять архівні файли гри, створені програмою.

## Коротка історія ##

Розширення файлу BKF — це свого роду службовий файл Windows Backup Utility, створений для програми Dragon UnPACKER компанією Open Source. Це дуже часто зустрічається серед користувачів у Сполучених Штатах і тих, хто використовує операційну систему Windows 10.

## Технічні характеристики ##

* Розроблено корпорацією Microsoft
* Основна асоціація з WindowsXP
* Підтримується програмою Windows NT Backup
* Категорія програми – архівні файли
* Тип формату — двійковий
* Належить до категорії файлів резервного копіювання
* Пов’язано з BKF Viewer
