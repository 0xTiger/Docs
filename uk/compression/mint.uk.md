## Що таке файл MINT?

Файл MINT – це файл інсталяційного пакета, який використовується дистрибутивом [Linux Mint](https://linuxmint.com/). Він містить файли стисненого вихідного коду, які розпаковуються та використовуються для запуску інсталяції ОС Linux Mint. ОС Linux Mint має простий у використанні GUI (графічний інтерфейс користувача) і була першою, яка мала елегантний графічний інтерфейс. Файли MINT розпаковуються та зберігаються у тимчасовому каталозі разом із пакетом інсталятора. Інсталяція виконується з того ж каталогу, що й інсталяція ОС Linux Mint.

## Формат файлу Linux MINT

Інформація про внутрішній формат файлу MINT недоступна для всіх. Однак стиснуті інсталяційні файли, що зберігаються у файлах MINT, відповідають стандартам стиснення, інтегрованим у інсталятори ОС Linux Mint. Це дає змогу розпакувати ці файли та стати частиною встановлення.
