## Що таке файл XAPK?

Файл із розширенням .xapk — це стиснутий пакетний файл, який використовується для встановлення програм Android на мобільних пристроях. Це формат файлу-контейнера, який містить APK і додаткові пов’язані файли, необхідні для встановлення. Інший пов’язаний файл – це OBB-файл, який зберігає додаткові файли, такі як графіка, мультимедійні файли та дані програми, які необхідні для забезпечення роботи програми. Файли XAPK не підтримуються Google Play і використовуються лише для розповсюдження на сторонніх веб-сайтах для завантаження програм Android. Їх можна встановити на пристрій Android за допомогою інсталятора XAPK.

## Формат файлу XAPK

Файли XAPK стискаються за допомогою стандартного формату [ZIP](/uk/compression/zip/). Їх можна видобути за допомогою стандартного програмного забезпечення для стиснення/декомпресії, наприклад WinZIP. Коли файл XAPK буде розпаковано на диск, він містить такі файли в папці.

* APK - Стандартний інсталяційний файл для встановлення програми на пристрої Android
* OBB - додатковий файл, який містить відповідні файли ресурсів
