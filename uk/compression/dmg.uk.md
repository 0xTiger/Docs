## Що таке файл DMG?

Файл із розширенням .dmg є файлом образу диска Apple і відомий як файл образу диска Mac OS X. Формат файлу DMG є найпоширенішим файлом образу диска, призначеним для розподілу програм і файлів між ПК Apple. Дуже зручно монтувати файли DMG у віртуальний диск або використовувати програмне забезпечення Apple Disk Utility для відкриття та перегляду вмісту формату файлу DMG. Стару версію файлів IMG, що використовувалася в Mac OS Classic, замінено форматом DMG. Файли DMG створюються для користувачів Mac і не підтримуються ОС Windows, але користувачі можуть отримати доступ до вмісту DMG в операційній системі Windows за допомогою інструментів стиснення та розпакування, таких як 7-Zip, Pea Zip тощо. Розширення файлу DMG також використовується інструмент Oracle Export and Ingress для файлів scrapheap, які зберігаються у двійковому форматі бази даних Oracle.

## Коротка історія

Розширення файлу DMG розроблено Apple і в основному використовується на комп’ютерах Macintosh. Здебільшого файл DMG використовується для надання пристроям Mac можливості інсталювати різні типи файлів і програмного забезпечення, які ініціюються з Інтернету. Файл DMG — це, по суті, образ диска, який можна монтувати, який відображається на робочому столі під час його відкриття. Файл містить необроблені дані, які зазвичай зашифровані та стиснені. Системи Mac обробляють DMG так само, як і диск із перемежуванням, і вони миттєво намагаються відкрити або «відстежити» файл.

## Технічні характеристики формату файлу DMG

* Спеціально створено для комп’ютерів Macintosh
* Розроблено Apple
* Формат файлу — двійковий
* Категорія файлу — файл образу диска
* Підтримувана програма — MacOS
* Може містити таблицю розділів
* Може містити будь-яку файлову систему, наприклад HFS+
* Функціональність стиснення зображень
