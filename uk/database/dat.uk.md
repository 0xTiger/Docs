## Що таке файл DAT?
Файл із розширенням .dat належить до загального формату файлу. Зазвичай вони ні зашифровані, ні двійкові, але можуть бути майже будь-якими. Різноманітні програми використовують файли DAT як посилання на певні операції у відповідній програмі. Тому ми не можемо сказати, що певне програмне забезпечення або програма відповідає за створення файлів DAT. Зокрема, доступ до цих файлів має лише програма, яка їх створила.


## Формат файлу DAT
Формат файлу DAT — це просто загальний формат, який містить будь-які дані у файлі, зазвичай зберігається з розширенням .dat. Він може містити дані у двійковому або текстовому форматі, зазвичай загальні дані складаються з інформації, специфічної для програми, до якої вони посилаються. Найчастіше ви можете знайти їх з іншими конфігураційними файлами, наприклад файлами DLL. Тут ми можемо залучити деякі відомі типи файлів із розширенням .dat:

### Відеофайл VCD
VCD були добре відомим джерелом відео або мультимедіа в 1990-х роках для розповсюдження фільмів і домашнього відео до DVD. Зазвичай відеофайли з диска VCD були знайдені з розширенням .dat. Таким чином, він зберігав фактичні відеодані, відтворені з VCD.
### Вкладення електронної пошти обміну
Файли DAT, пов’язані з вкладенням електронної пошти Exchange, містять інформацію про форматування для повідомлень Rich Text (RTF), надісланих із сервера Microsoft Exchange Server. Зазвичай ці файли додаються в кінці електронних листів, якщо клієнт електронної пошти одержувача не підтримує повідомлення RTF. Ці файли зазвичай знаходяться як **Winmail.dat**.
### Файл куща реєстру Windows
NTUSER.DAT — це звичайний файл куща реєстру. Реєстр Microsoft Windows використовує файли DAT для зберігання налаштувань користувача. Ці файли DAT завантажуються, коли користувач входить у Windows. Він може містити налаштування облікового запису, паролі та асоціації файлів.
### Файл даних Minecraft
Гра Minecraft використовує файли DAT для збереження інформації про інвентар гравця, позицію гравця та інформацію про світ. Файли DAT також використовуються для зберігання рівнів, які є невеликими фрагментами карти, які завантажуються на вимогу, коли гравець переміщується світом.
### Файл Piriform DAT
Такі програми, як CCleaner, Speccy, Defraggler і Recuva, використовують файли DAT для інформації про свої типові операції.
