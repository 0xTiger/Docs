## Що таке файл NMONEY?

Файл NMONEY — це файл бази даних для зберігання фінансових даних, який містить історію фінансових операцій. Він був розроблений компанією Nickvision і використовувався в програмному забезпеченні Nickvision Denaro, яке є програмним забезпеченням для персональних фінансів. Дані, що зберігаються у файлі NOMONEY, містять інформацію про кредитні та дебетові операції на обліковий запис.

Файли NOMONEY можна відкрити за допомогою програми [Github](https://github.com/NickvisionApps/Denaro).

## Про Denaro Software

Denaro спочатку було розроблено [Nickvision](https://nickvision.org/) як продукт, який пізніше було перетворено на програмне забезпечення з відкритим кодом, розміщене на [Github](https://github.com/NickvisionApps/Denaro) . Відтоді програму було змінено та оновлено лише на Github. Додаток розроблено на C# в інтерфейсі користувача Windows із Windows App SDK (WinUI 3 на даний момент).

### Функції, які пропонує Denaro

* Керуйте кількома обліковими записами одночасно за допомогою найпопулярнішого та знайомого інтерфейсу вкладок
* Фільтруйте транзакції за типом, групою або датою
* Повторюйте транзакції, такі як рахунки, які виникають щомісяця
* Переказ грошей з одного рахунку на інший
* Експортуйте дані з облікового запису як файл CSV та імпортуйте файл CSV, OFX або QIF, щоб додати транзакції до облікового запису

## Формат файлу Denaro – додаткова інформація

Файли Denaro зберігаються на диск у двійковому форматі. Фінансові дані зберігаються як файл бази даних у форматі **.SQLITE3**. SQLITE — це легкий файл бази даних SQL, створений за допомогою програмного забезпечення SQLite. Він забезпечує самодостатній механізм бази даних, який здатний надавати повнофункціональну та високонадійну базу даних. Файли бази даних SQLite можна використовувати для обміну багатим вмістом між системами шляхом простого обміну цими файлами через мережу. Прив’язки SQLite існують для таких мов програмування, як C, [C#](/uk/programming/cs/), [C++](/uk/programming/cpp/), [Java](/uk/programming/java/), [PHP](/uk/programming/php/), та багато інших.
