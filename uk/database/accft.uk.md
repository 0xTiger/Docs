## Що таке файл ACCFT?

Файл із розширенням .accft — це файл шаблону Microsoft Access, який містить визначення типів даних полів для таблиць бази даних. Microsoft Access — це настільна система керування реляційними базами даних (RDBMS), яка використовується для створення баз даних на основі таблиць, форм, запитів і представлень. Файл ACCFT можна використовувати для визначення типів даних для полів таблиці, а таблиці бази даних можна створювати, використовуючи цей файл шаблону як початкову точку. Його можна відкрити за допомогою Microsoft Access 2016 для створення нових таблиць. Файли ACCFT розпізнаються за типом MIME **application/msaccess**.

## Формат файлу ACCFT

Файли ACCFT — це двійкові файли, специфікації формату файлів яких недоступні для всіх. Його внутрішні деталі, такі як інформація заголовка, структура файлу, нижні колонтитули тощо, наразі недоступні.
