## Що таке файл DB3?
Файл DB3 — це файл бази даних, створений програмним забезпеченням SQLite, яке є легкою автономною програмою для створення баз даних, використовуючи звичайні файли; містить анатомію бази даних, а також записи даних; зазвичай використовується для отримання або зберігання структурованих даних за допомогою SQL. Ці файли можна використовувати в інтелектуальних пристроях або там, де потрібне ведення записів чи інше керування даними, але в середовищі з обмеженим простором.


## Формат файлу DB3
Формат файлу DB3 пов’язаний із RDBMS (системою керування реляційною базою даних) SQLite, популярним вибором для вбудованої бази даних. У специфікації SQLite_3 немає певного розширення файлу, він може використовувати розширення, включаючи [.dbf](/uk/database/dbf/) і [.sqlite](/uk/database/sqlite/).

### Секретизація бази даних
Зазвичай SQLite, пов’язаний з версією 3 формат файлу db, можна розглядати як формат файлу DB3, який використовується як публічно задокументований рідний формат для механізму бази даних SQLite з червня 2004 року. Формат файлу DB3 є кросплатформним, його можна передавати між порядковим порядком байтів. архітектури з порядковим кінцем або 32-розрядні та 64-розрядні системи. Ці функції роблять DB3 популярним вибором як формат файлу програми. Основний файл бази даних SQLite_3 (DB3) складається з однієї або кількох сторінок. Усі розміри сторінок однакові в одній базі даних. Розмір сторінки в байтах є ступенем двійки від 512 до 65536 включно.
