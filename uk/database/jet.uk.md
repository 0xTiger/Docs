## Що таке файл JET?

Файл бази даних JET, також відомий як файл бази даних Microsoft Access, — це формат файлу, який використовується Microsoft Access, популярною системою керування базами даних (СУБД). JET означає Joint Engine Technology, що було оригінальною назвою механізму бази даних, який використовував Microsoft Access. Пізніші версії Access використовували інші механізми баз даних, але термін "база даних JET" все ще широко використовується для позначення баз даних Access.

Ось деякі ключові моменти щодо файлів бази даних JET:

- **Розширення файлу:** Файли бази даних JET зазвичай мають розширення файлу ".mdb" для старіших версій Access (Access 2003 і раніших) і ".accdb" для новіших версій (Access 2007 і пізніших). У файлах ".mdb" використовується старіша система баз даних JET, а у файлах ".accdb" — новіша технологія ACE (Access Database Engine).

- **Реляційна база даних:** бази даних JET є реляційними базами даних, що означає, що вони зберігають дані в таблицях із визначеними зв’язками між ними. Користувачі можуть створювати, змінювати та запитувати дані за допомогою SQL (мова структурованих запитів) або через графічний інтерфейс Access.

- **Зберігання даних:** бази даних JET можуть зберігати широкий діапазон типів даних, включаючи текст, числа, дати, зображення тощо. Вони також можуть обробляти складні структури даних і зв’язки між таблицями.

- **Багатокористувацький доступ:** Бази даних JET підтримують багатокористувацький доступ, що дозволяє кільком користувачам працювати з базою даних одночасно, хоча можуть бути обмеження щодо масштабованості та продуктивності для великомасштабних програм.

- **Інтеграція:** Бази даних Access можна інтегрувати з іншими програмами Microsoft Office, такими як Excel і Outlook, що робить користувачам зручним обмін даними між різними програмами.

## Інформація про бази даних JET

**Історія двигуна JET:**

Механізм баз даних JET (Joint Engine Technology) спочатку був розроблений Microsoft на початку 1990-х років для використання в Microsoft Access. Механізм був розроблений як легке, орієнтоване на робочий стіл рішення бази даних, яке можна інтегрувати в різні продукти Microsoft.

**Компоненти бази даних JET**

- **Таблиці:** бази даних JET зберігають дані в таблицях, які є наборами записів, організованих у рядки та стовпці.
- **Запити:** користувачі можуть створювати запити для отримання, фільтрації та обробки даних із таблиць за допомогою SQL (мова структурованих запитів).
- **Форми:** Access надає конструктор форм, який дозволяє користувачам створювати власні форми введення та відображення даних.
- **Звіти:** користувачі можуть створювати звіти професійного вигляду на основі даних, що зберігаються в базі даних.
- **Макроси та VBA:** користувачі можуть автоматизувати завдання та додавати спеціальні функції за допомогою макросів або написання коду VBA.

**Типи даних**

Бази даних JET підтримують різні типи даних, зокрема:

- **Текст:** Для зберігання буквено-цифрових символів.
- **Число:** для числових значень.
- **Дата/Час:** для значень дати та часу.
- **Пам'ятка:** для більшого тексту або приміток.
- **Гіперпосилання:** для веб-посилань і адрес електронної пошти.
- **Вкладення:** Для зберігання файлів і вкладень.
