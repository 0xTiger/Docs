## Що таке файл CHCK?

Файл CHCK створюється в результаті перевірки працездатності на сервері Microsoft Exchange Server. Він містить інформацію про загальний стан і роботу бази даних сервера. Microsoft Exchange Server використовується в основному як сервер електронної пошти та дозволяє підтримувати ваші ділові контакти та календарі. Адміністратори сервера Exchange використовують EseUtil, щоб перевірити справність бази даних.

## Формат файлу CHCK

Внутрішній формат файлу CHCK невідомий, і немає довідкового матеріалу для читання/запису файлів CHCK.
