## Що таке файл DDL?

Файл із розширенням .ddl — це файл мови визначення даних, який використовується для визначення схеми бази даних. Він містить оператори/команди для роботи зі структурами бази даних, такими як таблиці, стовпці, записи та інші поля. Команди у файлі DDL написані мовою [SQL](/uk/database/sql/) і можуть виконувати такі операції, як створення таблиці в базі даних, видалення та оновлення. Схема бази даних належить її створеному, і над нею можна виконувати всі операції CRUD. Популярними програмами, які можуть створювати та відкривати файли DDL, є текстовий редактор Windows, Jetbrains Intellij Idea та EclipseLink.

## Команди DDL

Один файл DDL може містити кілька команд, які, завдяки правильному синтаксису, виконуватимуться послідовно та вноситимуть зміни до схеми, наприклад створювати набори символів і таблиці, видаляти таблиці, перейменовувати та змінювати таблиці. Наступні команди DDL зазвичай використовуються під час роботи зі схемою бази даних.

`CREATE` – використовується для створення бази даних або її об’єктів (наприклад, таблиці, індексу, функції, представлень, процедури зберігання та тригерів).

`DROP` – використовується для видалення об’єктів із бази даних.

`ALTER` - використовується для зміни структури бази даних.

`TRUNCATE` – використовується для видалення всіх записів із таблиці, включно з усіма виділеними для записів місцями.

`COMMENT` – додає коментарі до словника даних.

`RENAME` – перейменовує існуючий об’єкт у базі даних.

## Приклад DDL

У наступному прикладі показано оператор DDL для команди CREATE, яка створює таблицю та визначає її поля.

```
CREATE TABLE employees (
    id            INTEGER       PRIMARY KEY,
    first_name    VARCHAR(50)   not null,
    last_name     VARCHAR(75)   not null,
    fname         VARCHAR(50)   not null,
    dateofbirth   DATE          not null
);
```
