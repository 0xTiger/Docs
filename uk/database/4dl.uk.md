## Що таке файл 4DL?

Файл журналу 4DL використовується для запису змін, внесених у файл бази даних 4D (.4DD). Цей файл зберігається поряд із файлом бази даних і має схоже ім’я. Його метою є точне відстеження та підтримка повного запису оновлень, реалізованих у базі даних. [Файл 4DD](/uk/database/4dd/), порівняно з файлом 4DL, в основному асоціюється з 4-м виміром від 4D Inc.

## Формат файлу 4DL - Додаткова інформація

Як правило, декілька файлів 4DL зберігаються разом із файлом 4DD. Таким чином, перша версія файлу журналу може бути названа як 0001.4dl, але бічні версії файлів журналу будуть збережені як 0002.4dl, 0003.4dl і так далі. Тепер, якщо сам файл журналу піддається 3 послідовним збереженням, він буде позначений як "db1[0005-0003].4dl."
