## Що таке файл TEACHER?

Файл .TEACHER – це файл освітніх налаштувань, створений програмним забезпеченням SMART Response. Він використовується в освітніх установах для забезпечення повного середовища керування інформацією в класі, списками класів, розкладами, канікулами, профілями студентів, розпорядком іспитів і результатами оцінювання. Програмне забезпечення для оцінювання SMART Response™ встановлено на комп’ютері, який використовуватиметься для проведення оцінювання SMART Response.

## ВЧИТЕЛЬ Формат файлу

Файли TEACHER зберігаються на диску у двійковому форматі. Специфікації внутрішнього формату файлу недоступні для розробників. Програмне забезпечення SMART Response можна використовувати для створення звітів учнів. Потім їх можна буде спільно використовувати між шкільними комп’ютерами.
