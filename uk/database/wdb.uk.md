## Що таке файл WDB?
Файл із розширенням .wdb — це фактично файл бази даних, створений Microsoft Works, який використовувався для виконання таких функцій, як система керування базами даних. Файл WDB схожий на файл бази даних Access ([MDB](/uk/database/mdb/)), але менш ефективний і має більше обмежень. Файли WDB не можна відкрити за допомогою Microsoft Access. Однак ви можете відкрити файл WDB у Microsoft Works, а потім експортувати його у файл MDB, щоб відкрити файл WDB у MS Access.

## Формат файлу WDB
База даних Microsoft Works — це рідний формат бази даних офісного пакету Microsoft Works. Через запатентований характер формату та деякі обмеження. Файли WDB не можна відкрити в жодному іншому програмному забезпеченні, крім Microsoft Works. У форматі файлу повторюваний 10-байтовий заголовок можна знайти перед кожним текстовим рядком ASCII, що представляє значення полів, які завершуються значенням NULL. Заголовок зазвичай починається з байта **\x0f** і NULL, після чого йдуть 4 байти даних, які чергуються з NULL.

### Структура файлу

Структура файлу WDB наведена нижче:
- **1-й заголовок**: від початку файлу, закінчуючись на \x25\x00\xf2 і 244 байти NULL
- **2-й заголовок**: починається з \xffT - тобто \xff\x54 і розширюється на 4096 байтів, містить назви стовпців/полів та інше, і, здається, починається з позиції байта 6144.
- **Поле**: значення має 10-байтовий заголовок у такому форматі: {тип байт} {тип байт, частина 2} {дані байт 1} \x00 {db 2} \x00 {db 3} {db 3 , частина 2} {db 4} \x00. байт даних 2 — це номер поля, байт даних 3 — це номер запису (додавання байта даних 3, частина 2, коли він перевищує 256 записів).
