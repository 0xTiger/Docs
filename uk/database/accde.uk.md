## Що таке файл ACCDE?

Файл із розширенням .accde — це формат файлу Microsoft Access 2007 для створення файлів бази даних із можливістю лише виконання для захисту спеціального коду бази даних. Він компілює весь код VBA лише у виконуваний файл, щоб його не можна було переглядати чи редагувати. Формат файлу ACCDE є заміною формату файлу MDE, який використовувався попередніми версіями Microsoft Access. ACCDE — це файли [ACCDB](/uk/database/accdb/) у режимі лише для виконання та більш компактні через відсутність будь-якого вихідного коду. Компактний розмір цих файлів покращує використання пам’яті, а також підвищує продуктивність програми. Файли ACCDE можна відкривати за допомогою Microsoft Access 365.

## Формат файлу ACCDE

Усі файли Microsoft Access 2007+ використовують Jet ACE (систему баз даних Access), яка використовує сімейство форматів ACCDB. Однак корпорація Майкрософт не надає реалізацію з відкритим кодом або докладну технічну специфікацію.
