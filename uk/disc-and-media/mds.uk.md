## Що таке файл MDS?

MDS (Media Descriptor Sidecar File) — це файл, пов’язаний із програмним забезпеченням Alcohol 120% і Daemon Tools, обидва з яких використовуються для створення та керування віртуальними дисководами CD і DVD. Файл MDS містить інформацію про структуру даних на диску, включаючи розташування всіх файлів і структуру диска. Це дозволяє віртуальному приводу емулювати поведінку фізичного диска, щоб програмне забезпечення могло зчитувати дані на віртуальному диску так, ніби це справжній диск.

When you want to mount an image file to a virtual drive, the MDS file is used along with the corresponding image file in a format such as ISO, BIN or CUE. The MDS file contains a description of the layout of the data on the disc, and the virtual drive software uses this information to create a virtual disc. This allows you to use the software as if you were using a physical disc, without having to physically insert the disc into the drive.

## Зв'язок з алкоголем 120%

Alcohol 120% — це популярне програмне забезпечення для створення та керування віртуальними приводами компакт-дисків і DVD, яке використовує формат файлу MDS для збереження та доступу до образів дисків. Формат файлу MDS є власністю Alcohol 120%, і його можна відкрити та використовувати лише програмним забезпеченням Alcohol 120% або іншим програмним забезпеченням, яке його підтримує.

## Як відкрити файл MDS?

Щоб відкрити файл MDS у Alcohol 120%, на вашому комп’ютері має бути встановлено програмне забезпечення. Після встановлення Alcohol 120% ви можете відкрити файл MDS, виконавши такі дії:

1. Launch Алкоголь 120%.
2. Натисніть на меню «Файл» і виберіть «Відкрити».
3. Перейдіть до розташування файлу MDS на вашому комп’ютері та виберіть його.
4. Файл MDS тепер буде відкрито в Alcohol 120%, і вам буде запропоновано вибрати відповідний файл зображення (наприклад, ISO, BIN або CUE).
5. Після того як ви виберете файл образу, Alcohol 120% змонтує образ на віртуальний диск.

Крім того, ви також можете відкрити файл MDS, двічі клацнувши його, якщо Alcohol 120% встановлено як програму за замовчуванням для відкриття файлів MDS. Ви також можете використовувати інше програмне забезпечення, яке підтримує формат MDS, наприклад Daemon Tools, Virtual CloneDrive, Virtual Drive Manager і MagicISO.
