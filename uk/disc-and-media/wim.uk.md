## Що таке файл WIM?
Файли з розширенням .wim вперше були помічені в Windows Vista. WIM належить до файлового формату зображень, який зазвичай був представлений у Microsoft Windows Vista. Це дозволяє розгортати один образ диска на різних комп’ютерних платформах. Файли WIM використовуються для керування такими файлами, як оновлення, драйвери та компоненти, без перезавантаження образу операційної системи. Файл AQ WIM містить набір файлів і пов’язаних метаданих, які дуже схожі на інші формати дискових файлів.

## Формати файлів WIM
Формат файлу WIM не схожий на секторні формати, такі як VHD або IS, насправді він заснований на файлі, що означає, що основною одиницею інформації в WIM є файл. Основна перевага використання файлів полягає в тому, що файл в одному екземплярі зберігається кілька разів у дереві файлової системи, а також апаратна незалежність. Це зменшує накладні витрати на відкриття та закриття різних файлів окремо, оскільки файли зберігаються в одному WIM файл. WIM-файли можуть зберігати кілька образів дисків, на які посилається або їхня унікальна назва, або їхній числовий індекс.
### Підтримка алгоритмів стиснення
WIM підтримує наступні сімейства алгоритмів стиснення у спадній швидкості та зростаючому співвідношенні:
- XPRESS
- LZX
- ЛЗМС
- Суцільно-компресійні.
Перші три сімейства базуються на LZ77, тоді як Solid-compression було представлено разом із LZMS у WIMGAPI Windows 8 і DISM Windows 8.1.
### Інструменти для формату файлу WIM
Наступні інструменти зазвичай підтримують формат файлу WIM:

- **ImageX**: інструмент командного рядка, який використовується для редагування, створення та розгортання образів дисків Windows у форматі Windows Imaging Format. Разом із відповідним WIMGAPI він розповсюджується як частина безкоштовного набору автоматичної інсталяції Windows. Починаючи з Windows Vista, програма встановлення Windows використовує API WAIK для встановлення Windows.
- **DISM**: інструмент, представлений у Windows 7 і Windows Server 2008 R2, який може виконувати завдання, пов’язані зі службою, на інсталяційному образі Windows, будь то онлайн-образ або офлайн-образ у папці чи файлі WIM. цей інструмент міг монтувати та демонтувати образи, додавати драйвер пристрою до офлайн-образу та запитувати встановлені драйвери пристрою в офлайн-образі.
