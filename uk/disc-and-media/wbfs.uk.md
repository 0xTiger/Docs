## Що таке файл WBFS?

Файл WBFS (Wii Backup File System) — це образ диска відеоігор, створений для Nintendo Wii. Він містить гру Wii, яку можна завантажити як зображення для гри. Він створюється за допомогою інструменту Wiimms WBPS або Wii Backup Manager, і його можна відтворювати на Nintendo Wii, завантаживши з USB Loader GX або емулятора Wii, наприклад Dolphin. Гра зберігається на цих зовнішніх накопичувачах у форматі WBFS. Wii — це ігрова платформа, яку розповсюджує Nintendo. Файлова система WBFS була розроблена програмістом Wii Homebrew Waninkoko.

## Формат файлу WBFS - Додаткова інформація

WBFS — це двійкові файли, які мають власний формат файлу. Специфікації його формату файлу недоступні для всіх.
