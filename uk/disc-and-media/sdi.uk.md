## Що таке файл SDI?
Файл із розширенням .sdi містить blob завантаження, blob завантаження та blob частини; зазвичай використовується Windows Embedded Studio для створення дисків RAM або завантажувальних дисків для завантаження та інсталяції Windows. SDI скорочено означає System Deployment Image. Windows Embedded Studio — це програма, яка використовується для створення образів дисків для Windows. Насправді ця програма містить програму диспетчера файлів SDI та інструмент командного рядка під назвою **sdimgr.exe**, обидва можна використовувати для розгортання, створення та редагування зображень SDI.

## Формат файлу SDI
Формат файлу SDI широко використовується для використання віртуального диска для завантаження системи. У деяких версіях Microsoft Windows увімкнено **завантаження з оперативної пам’яті**, що важливо для завантаження файлу SDI в пам’ять і завантаження з нього. Формат файлу SDI також активується для мережевого завантаження за допомогою PXE (середовище виконання перед завантаженням). Він також використовується для створення образу жорсткого диска.
### Розділи файлу SDI
Сам файл SDI можна розділити на такі розділи:
- **Boot BLOB**: містить фактичну програму завантаження STARTROM.COM. Це аналог завантажувального сектора жорсткого диска.
- **Load BLOB**: зазвичай містить NTLDR і запускається завантажувальним BLOB.
- **Part BLOB**: містить фактичне середовище виконання завантаження; також містить файли boot.ini та ntdetect.com, які мають бути розташовані в кореневому каталозі середовища виконання.
- **Disk BLOB**: це образ плоского жорсткого диска, який починається з MBR. Він використовується для створення образу жорсткого диска замість завантаження. Крім того, за допомогою утиліт Microsoft можна монтувати лише дискові BLOB-файли.
