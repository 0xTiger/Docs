## Що таке файл OSZ?

Файл OSZ — це формат стисненого файлу, який використовується в ритмічній грі osu! для зберігання даних тактових карт. Бітмапи — це, по суті, рівні для гри, і вони містять таку інформацію, як пісня, хронометраж тактів і розміщення влучних об’єктів на ігровому екрані. Файли OSZ дозволяють легко розповсюджувати біт-карти, їх зазвичай завантажують з Інтернету та імпортують у гру. ОСУ! також має формат файлу [OSR](/game/osr/), який є файлом відтворення гри, який можуть відтворювати гравці.

**Тип MIME формату файлу OSR:** x-osu-replay

## Формат файлу OSZ

Деталі внутрішнього формату файлів OSZ недоступні для всіх. Він містить [ZIP](/compression/zip/)-стиснуті дані, які містять інформацію для відтворення музики, показу графіки та відображення гравців про події, коли вони повинні взаємодіяти з грою.

## Як створити файл OSZ?

ОСУ! містить докладні інструкції для файлів [creating OSZ](https://osu.ppy.sh/wiki/en/Client/File_formats#creating-.osz-and-.osk-files) і OSK. Наступні кроки можна використати для створення файлу OSZ за допомогою OSU!.

1. Відкрийте біт-карту через редактор.
1. У верхньому меню виберіть Файл > Експортувати пакет....
1. Архів .osz буде розміщено в папці Exports у файлі osu! папку.
