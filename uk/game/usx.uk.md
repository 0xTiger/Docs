## Що таке файл USX?

Файл USX містить сітки Unreal Engine, які є попередньо розробленими 3D-об’єктами/моделями, які використовуються для створення середовища. Моделі з файлу USX мають різне положення, орієнтацію та розміри, але вони залишаються статичними. Статичні сітки можна оптимізувати для продуктивності та використовувати для створення різних рівнів деталізації (LOD) для покращення продуктивності гри. Їх також можна використовувати для створення об’ємів зіткнень і як частину фізичного моделювання.

Файли USX можна створювати за допомогою редактора рівнів Unreal UnrealEd.

## Формат файлу USX – Додаткова інформація

Файли USX зберігаються на диску як двійкові файли. Деталі внутрішньої файлової структури формату файлу USX недоступні для розробників.
