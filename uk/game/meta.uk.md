## Що таке файл META?

META-файл містить дані гри різними іграми, які використовують Exient XGS Engine, наприклад Angry Birds Go! або Angry Birds Transformers. META-файл фіксує та зберігає інформацію про певні аспекти гри, наприклад журнали збоїв або журнали елементів.

META-файли автоматично генеруються грою і не призначені для модифікації користувачем або прямого доступу. Якщо META-файл призначений для реєстрації збоїв гри, він зазвичай називатиметься "crashes.meta". Подібним чином, якщо META-файл використовується для реєстрації інформації, пов’язаної з елементом, він відповідатиме умовам іменування "XXX.xml.meta", де "XXX" представляє описовий ідентифікатор для певного типу елементів, що реєструються.

## Exient XGS Engine

Компанія Exient Entertainment створила ігровий движок Exient XGS Engine. Це програмна основа, створена для полегшення процесу створення та розробки відеоігор. Він пропонує програмістам різноманітні інструменти, бібліотеки та функції для прискорення процесу створення гри.

Різноманітні функції та можливості, які надає Exient XGS Engine, спрощують створення ігор на різних платформах, включаючи ПК, консолі та мобільні пристрої. Ці функції часто включають створення візуальних зображень, імітацію фізики, керування аудіо, обробку вхідних даних, мережеві можливості та інші речі.
