## Що таке файл OSU?

Файл OSU — це сценарій гри, написаний для osu! ритмічна гра. Він містить інформацію про біт-карту, яка використовується в грі, як-от рівень складності, пісню, яку потрібно відтворити, і час попадання об’єктів, які гравець має синхронізувати з музикою. Це дозволяє гравцям у ритмічні ігри розробляти власні завдання та ділитися ними з ігровою спільнотою.

**Тип MIME формату файлу OSR:** x-osu-beatmap

## Формат файлу OSU

Файли OSU зберігаються як звичайні текстові файли на диск, і їх структура дуже чітко визначена в [OSU file format](https://osu.ppy.sh/wiki/en/Client/File_formats/Osu_%28file_format%29) osu!.

### Розділи у файлі OSU

Типовий файл OSU має такі розділи.

|Розділ |Опис |Тип вмісту|
---|---|---|
|[Загальні]| Загальна інформація про бітмап| ключ: пари значень|
|[Редактор]| Збережені налаштування для редактора тактових карт| ключ: пари значень|
|[Метадані] |Інформація, яка використовується для ідентифікації тактової карти| ключ:пари значення|
|[Складність] |Параметри складності| ключ:пари значення|
|[Події]| Карти ритмів і графічні події розкадровки| Списки, розділені комами|
|[Часові точки]| Хронометраж і контрольні точки| Списки, розділені комами|
|[Кольори]| Комбо та кольори шкіри| ключ : пари значень|
|[HitObjects]| Удар об'єктів| Списки, розділені комами|
