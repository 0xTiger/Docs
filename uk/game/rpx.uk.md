## Що таке файл RPX?

Файл RPX – це файл гри, який можна завантажити або відтворити на ігровій консолі Wii U. Його також можна завантажити та відтворити в емуляторі відеоігор Cemu, який популярний для створення середовища емуляції для файлів Wii U. Файли ігор Wii U мають великий розмір і їх також можна зберегти як файли [WUD](/uk/disc-and-media/wud/) або [WUX](/uk/compression/wux/) для завантаження за допомогою ігрових емуляторів. Файл RPX містить усю виконувану гру, яка складається з усіх компонентів гри, таких як графіка, текстури, послідовності та ПЗУ гри.

## Формат файлу RPX - Додаткова інформація

Файли RPX зберігаються на диск у власному форматі. Певні API доступні для стиснення або розпакування файлів RPX, наприклад [wiiurpxtool](https://github.com/0CBH0/wiiurpxtool). Крім того, такі програми, як емулятор відеоігор Cemu, дозволяють грати в ігри Wii U в Windows на ПК.
