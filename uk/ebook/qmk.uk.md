## Що таке файл QMK?

Файл із розширенням .qmk зазвичай відомий як формат **файлу YanCEyDesktop Quickmarks**. YanCEyWare представив формат файлу YanCEyDesktop Quickmarks. Файл QMK пов’язаний із файлом [YBK](/uk/ebook/ybk/) і підтримується програмою, яка доступна для машин під керуванням операційної системи Windows. Файл із розширенням QMK класифікується як файл текстового типу, який дозволяє користувачам легко переходити до збережених місць у електронних книгах. YanCEyDesktop — рекомендоване програмне забезпечення для керування файлами QMK. Програмне забезпечення **YanCEyDesktop** створено **D Remington**.

## Проблеми з відкриттям файлу QMK

Якщо ви не можете відкрити та запустити файл QMK; це не означає, що на вашому пристрої не встановлено відповідне програмне забезпечення. Можуть виникнути деякі інші проблеми, які перешкоджають належній роботі файлу. Можливі проблеми можуть бути одними з наступного:

— Пошкодження файлу QMK
- Неправильні посилання на файл QMK у записах реєстру
— Видалено опис QMK із реєстру Windows
- Пошкоджена інсталяція програми, яка підтримує формат QMK
- Інфікований файл QMK з небажаним шкідливим програмним забезпеченням
- На комп'ютері недостатньо апаратних ресурсів для роботи з файлом QMK
- Драйвери, які комп'ютер використовує для відкриття файлу QMK, застаріли
