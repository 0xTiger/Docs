## Що таке файл TR3? ##

Файл TR3 — це електронна книга TomeRaider 3, яка підтримує швидкий пошук і стиснення. Це може належним чином працювати за допомогою пов’язаної з TomeRaider програми. Розробником цього програмного забезпечення є Yadabyte, британська компанія з програмного забезпечення та веб-розробки, що базується у Великобританії. Програму для читання електронних книг TomeRaider можна використовувати для належного функціонування та керування вмістом цих файлів TR3. Цей пов’язаний пристрій для читання електронних книг можна встановити на комп’ютерах, що працюють під керуванням систем на базі Microsoft Windows, і багатьох гаджетів, які можна переносити. Файл TR3 належить до групи файлів електронних книг, які використовуються в таких операційних системах, як Windows 10, Windows 7, Windows 8 / 8.1, Windows Vista, Windows XP.

**HTML-теги** TR3 є такими:

|Тег|Опис|
---|---|
|\<new> |Щоб створити файли TomeRaider із текстових файлів, усе, що потрібно, — новий тег.<new> тег визначає початок сторінок.|
|\<CATDEF> ... \</CATDEF> |визначає назву категорії|
|\<META> ... \</META> |це тег, який використовується для визначення всіх метаданих, що використовуються у форматі файлу. Цей тег містить низку параметрів.|
|\<EXPMSG> |Цей тег керує повідомленням, яке з’являється, коли термін дії файлу закінчився. Щоразу, коли користувач намагається переглянути сторінку після закінчення терміну дії файлу, замість тексту сторінки з’являється повідомлення про закінчення терміну дії.|
|\<DIEMSG> |Цей тег схожий на EXPMSG. Використовується для відображення повідомлення, коли файл закривається.|
|\<ENGMSG> |Цей тег використовується для створення повідомлення, яке відображається для зашифрованих сторінок. Повідомлення з’являється замість тексту сторінки, якщо користувач намагається відкрити зашифровану сторінку, перш ніж розблокувати її.|
|\<expmsg> ,\<diemsg> і \<encmsg> |Ігнорується в тексті сторінки. Вони повинні бути розміщені в розділі заголовка файлу перед першим<new> тег.|
|\<SELECTION> . \</ SELECTION> |Використовується для підтримки запитів. Цей тег має бути перед першим<new> тег. Він компілює дані вибору для запиту користувача.|
|\<catset> . \</catset> | Використовується для призначення назв категорій кожному предмету.|


## Проблеми з відкриттям файлу TR3 ##

Ось список деяких поширених проблем, які можуть виникнути та спричинити неправильну роботу формату файлу:

* Пошкоджений файл
* Файл, інфікований вірусом
* Некоректні посилання на файл TR3 у доступі до архіву
* Немає права доступу в системі для відкриття файлів
* Застарілий диск у вашій системі
* Видалення звіту TR3 з архіву Windows
* Щоб отримати кращий результат, краще зв’язатися з розробником
