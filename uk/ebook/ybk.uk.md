## Що таке файл YBK? ##

Формат файлів електронної книги YanCEyWare та Microsoft Encarta використовують розширення файлів YBK. YanCEyWare eBook була створена YanCEyWare. Усі програми доступні для пристроїв під керуванням Windows, підтримують файли YBK. Енциклопедію Encarta можна оновлювати за допомогою файлів .ybk без оновлення всієї програми. YanCEyDesktop — це добре відома та популярна програма, яка може легко керувати та обробляти файли YBK. YanCEyWare Reader широко використовує цей формат файлу YBK як програму для читання електронних книг для Windows Mobile або Pocket PC. Цей файл створено за допомогою великих ресурсів із гіперпосиланнями.

## Проблеми з відкриттям файлу YBK ##

Іноді, коли важко відкрити та працювати з файлом YBK, ось список деяких можливих проблем:

* Відкриття пошкодженого файлу YBK
* Невідповідні та неправильні посилання на файл YBK у доступі до реєстру
* Ненавмисне видалення опису YBK з архіву Windows
* Часткова інсталяція програми, яка підтримує формат YBK
*	Infected YBK file with an undesirable malware
