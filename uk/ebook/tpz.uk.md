## Що таке файл TPZ? ##

Розширення файлу .tpz також відоме як Kindle Topaz eBook Format від Amazon. Linux, Mac OS і Windows – це деякі платформи, на яких можна розповсюджувати файли з розширенням .tpz. Як файл електронної книги Kindle Topaz, файл TPZ відформатовано так, щоб виглядати як традиційні електронні книги, і містить власні стандарти компіляції даних від Amazon для формату TPZ. Цей файл має розширену підтримку вбудованих шрифтів, і додаткові деталі, такі як анотації та закладки, також можна включити у файли електронної книги Kindle Topaz, хоча ці деталі метаданих зберігаються окремо. Файл TPZ також можна класифікувати як файл даних, який можна завантажити з онлайн-бази даних електронних книг Kindle Amazon.

## Проблеми з відкриттям файлу TPZ? ##

Нижче наведено деякі поширені проблеми, які можуть виникнути:

* Інфікований файл. Це можна виправити за допомогою належного антивірусного програмного забезпечення, щоб зробити його чистим
* Недостатньо місця на диску. Для цього потрібно закрити всі непотрібні програми для кращої роботи
* Непрацюючі посилання іноді можуть бути причиною проблем
* Іноді файл не має прав доступу, тому в цьому випадку переконайтеся, що права доступу легко відкривають файл
