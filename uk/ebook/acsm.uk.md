## Що таке файл ACSM?

Файли з розширенням .acsm називаються файлами повідомлень Adobe Content Server. Ці файли використовуються **Adobe Digital Editions** для активації та завантаження вмісту, захищеного Adobe DRM. Насправді файл ACSM не є файлом електронної книги; її не можна відкрити та прочитати, як інші електронні книги (наприклад, PDF); Він містить лише дані для активації та завантаження електронної книги. Це означає, що файл ACSM складається лише з інформації, яка обмінюється даними із серверами Adobe. Користувачі Digital Editions можуть бачити файл ACSM, якщо вони завантажуватимуть електронну книгу, але завантаження буде зупинено з будь-якої причини. Користувачі можуть відновити завантаження, двічі клацнувши файл .acsm.

## Як працює файл ACSM?

Оскільки Adobe Content Server відповідає за захист електронних книг та інших цифрових матеріалів. Після покупки вміст автоматично прив’язується до Adobe ID покупця. Ідентифікатор є приватним і не може бути переданий іншим. Користувач повинен налаштувати його перед придбанням будь-якого документа із захистом від копіювання Adobe. Клієнти не можуть отримати доступ до своїх захищених від копіювання документів, не передавши свій ідентифікатор Adobe у серверну програму Adobe, що працює у фоновому режимі.

Коли клієнти роблять покупку, ACSM є єдиним файлом, який вони можуть спочатку завантажити. Щоб відкрити файл ACSM, клієнт має встановити програмне забезпечення Adobe Digital Editions у своїй системі та пов’язати його зі своїм Adobe ID. Adobe Digital Editions перевірить ідентифікатор для авторизації для перетворення файлу ACSM. Після автентифікації користувач може завантажити свою електронну книгу у форматі [EPUB](/uk/ebook/epub/) або [PDF](/uk/pdf/). Adobe Digital Editions також використовує файл ACSM для розпізнавання каталогу завантажень.
