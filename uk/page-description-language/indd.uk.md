## Що таке файл INDD?

Файл із розширенням .indd — це файл проекту Adobe InDesign, який містить інформацію про макет сторінки проекту. INDD означає InDesign Document. У цих файлах зазвичай зберігається інформація про форматування сторінок, книги, журнали, газети, листівки, брошури чи інші друковані носії, наприклад макети мобільних і веб-сторінок. Тип файлу INDD є власним форматом файлу Adobe і стандартним форматом файлу для InDesign. Графічні дизайнери, журналісти, маркетологи та видавці, які працюють з InDesign, використовують цей тип файлу.

## Докладніше про формат файлу INDD

Файли INDD — це векторні файли, які можна масштабувати без втрати роздільної здатності. Вони засновані на складних графах, які можна нескінченно розширювати. Він містить усю інформацію про шрифти та макети, якою можна поділитися з командою.

Файли INDD ідеально підходять для створення привабливого друкованого та онлайн-вмісту. Файл INDD може охоплювати кілька сторінок і служити шаблонами для створення сторінок, розділів і публікацій. Коли ваш проект буде завершено, його можна опублікувати в Інтернеті без необхідності конвертувати, архівувати або надсилати електронною поштою.

## Чи можна конвертувати файл INDD в інші формати?

Файли INDD можна конвертувати в кілька інших форматів. Найпоширенішим перетворенням є INDD у PDF. InDesign дозволяє зберегти файл INDD у форматі PDF для спільного використання з іншими користувачами. Він також дозволяє експортувати файл INDD у JPG, [EPS](/uk/page-description-language/eps/), [EPUB](/uk/ebook/epub/), [SWF](/uk/page-description-language/swf/) , FLA, [HTML](/uk/web/html/), [XML](/uk/web/xml/) і IDML.

## Як створюється файл INDD?

Файл Indd створюється, коли ви створюєте новий проект за допомогою [Adobe InDesign](https://www.adobe.com/products/indesign.html) і зберігаєте його на диск. Ви можете відкрити файл проекту Indd, внести в нього зміни та зберегти оновлений файл для змін.
