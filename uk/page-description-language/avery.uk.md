## Що таке файл AVERY?

Файл AVERY — це документ для друку, який містить одну або кілька міток для друку. Він створюється за допомогою програмного забезпечення Avery Design & Print або Design & Print Online. Мітки у файлі AVERY можуть складатися з різних елементів, таких як текст, графіка та інформація про макет сторінки. Ця інформація використовується поліграфічними компаніями для друку різноманітних елементів дизайну, таких як візитні картки, ярлики з іменами, вкладиші для палітурок, адресні етикетки, етикетки для дисків, шаблони некрологів і вітальних листівок.

## Формат файлу AVERY – Додаткова інформація

Файли AVERY зберігаються у двійковому форматі. Їх можна відкрити за допомогою програми Avery Design & Print або Design & Print Online. Дизайни, створені за допомогою Design & Print, можна використовувати для створення етикеток AVERY, які можуть використовуватися для багатьох цілей. Ці дизайни можна зберегти як [PDF](/uk/pdf/) файли, які можна надати компанії Prints для друку.

### Друк файлу AVERY

Файл AVERY можна зберегти як PDF або роздрукувати безпосередньо на будь-якому стандартному принтері. Щоб надрукувати файл, відкрийте файл у програмі Design & Print і скористайтеся опцією «Друк» у меню «Файл», щоб надіслати файл на принтер для друку.
