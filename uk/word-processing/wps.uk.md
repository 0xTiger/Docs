## Що таке файл WPS?

Файл із розширенням .wps — це документ Microsoft Works, який використовувався до 2006 року як документ Word. На той час він був схожий на нещодавно представлений формат файлу [DOC](/uk/word-processing/doc/), але не мав такої кількості функцій, як новий формат файлу. Наприклад, він не підтримував функції макросів і розширені параметри форматування. Файли WPS все ще можна відкривати за допомогою сучасних текстових процесорів, але для Windows 10 вам потрібно встановити конвертер, щоб завантажити ці файли. У 2010 році на зміну Microsoft Works прийшов процесор документів Microsoft Word.

## Специфікації формату файлу

Специфікації формату файлу документа WPS недоступні для всіх, оскільки Microsoft не відкривала специфікації для кінцевих користувачів. WPS — це двійкові файли, які можна відкрити за допомогою фільтрів перетворення форматів файлів, створених Microsoft для процесора Word. LibreOffice, NeoOffice та OxygenOffice включили загальну бібліотеку C++, libwps, яка може отримувати текст із багатьох різних версій Microsoft Works.
