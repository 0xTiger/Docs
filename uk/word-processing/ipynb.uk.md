## Що таке файл IPYNB?

Файл із розширенням .ipynb — це файл Jupyter Notebook, який містить увесь вміст Notebook, створений за допомогою сеансу веб-програми Jupyter Notebook. [Jupyter Notebook](https://jupyter.org/) – це інтерактивна веб-програма, яка використовується для створення та обміну обчислювальними документами для аналізу за допомогою Python.

Файл IPYNB служить повним обчислювальним записом сеансу та містить таку інформацію, як обчислювальні входи та виходи, математичні функції та багате представлення результуючих об’єктів, таких як зображення та пояснювальний текст.

## Формат файлу IPYNB

IPYNB зберігаються у внутрішньому форматі [JSON](/uk/web/json/), який є відкритим стандартним форматом файлу для обміну даними. З цієї причини файли IPYNB легко читаються та зрозумілі. JSON також полегшує обмін цими файлами з іншими.

## Як конвертувати файли IPYNB?

Файли IPYNB можна експортувати в декілька різних форматів за допомогою команди nbconvert з інтерфейсу Jupyter. Це включає перетворення IPYNB на:

* PDF
* HTML
* ReStructuredText
* LaTex

## Як переглянути файл IPYNB?

Існує два способи перегляду або відкриття файлів IPYNB.

1. Користувач може завантажити документ блокнота .ipynb за деякою загальнодоступною URL-адресою та відкрити його в Jupyter Notebook Viewer<nbviewer> який відображає блокнот як статичну веб-сторінку з URL.
1. Завантажте файли та відкрийте їх у власному екземплярі Jupyter Notebook
