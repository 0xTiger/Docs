## Про перетворення DOCX у DOC

Тип файлу DOC був популярним форматом файлу документа, який використовувався у 2007 році та попередніх версіях Microsoft Word. З випуском Office 2007 Microsoft представила формат файлу DOCX на основі специфікацій Office Open XML. Файли DOC зберігалися як двійкові файли порівняно з DOCX, який зберігається у форматі стисненого файлу XML. У мережі є багато інструментів для перетворення файлів [DOCX](/uk/word-processing/docx/) у [DOC](/uk/word-processing/doc/).

## Перетворення DOCX на DOC – безкоштовна демонстрація

Ви можете конвертувати файл DOCX у формат DOC за допомогою БЕЗКОШТОВНОЇ онлайн-програми Aspose.app. Це набір програм, які можуть завантажувати, переглядати та конвертувати файли DOCX у такі формати, як DOC, PDF, TXT, RTF та інші. Програми Aspose створені та працюють за допомогою API Aspose на сервері, які можуть виконувати низку операцій з різними типами файлів. Спробуйте онлайн-додаток Aspose.Words, натиснувши наступну кнопку.

### Кроки для конвертації DOCX у DOC онлайн

Ви можете конвертувати файл DOCX у формат DOC за допомогою програми [Aspose.Words DOCX to DOC](https://products.aspose.app/words/conversion/docx-to-doc), як описано нижче.

1. Перейдіть у програму Aspose.Words для [конвертера DOCX у DOC](https://products.aspose.app/words/conversion/docx-to-doc)
1. Завантажте свій файл DOCX, натиснувши кнопку завантаження
1. Після завершення натисніть кнопку "Перетворити". Це почне процес перетворення файлу DOCX у формат DOC у фоновому режимі
1. Після завершення процесу перетворення DOCX у DOC вам буде запропоновано завантажити перетворений файл RTF

Окрім DOC, API Aspose.Words дозволяють конвертувати файли DOCX у багато інших популярних форматів файлів.
