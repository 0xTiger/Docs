## Що таке файл CUR? ##

Файл CUR — це статичний формат файлу курсора Microsoft Windows. По суті, це стаціонарні зображення, ідентичні файлам ICO (значок) у всіх відношеннях, крім розширення. І CUR, і [ICO](/uk/image/ico/) створено на основі специфікації Device-Independent Bitmap [DIB](/uk/image/dib/) (Device-Independent Bitmap). За замовчуванням, а також настроюваний курсор, наприклад вказівник миші Windows для операційної системи Windows, зберігаються у файлах CUR. Це може бути стрілка для звичайного використання, обертовий пісочний годинник для демонстрації періодів очікування або I-бар для редагування тексту. Файли CUR постачаються разом із інсталяційним пакетом користувацьких тем робочого столу, щоб переконатися, що курсор Windows відповідає користувацькій темі робочого столу.

## Технічна специфікація ##

Файли CUR - це двійкові системні файли, які можна знайти на пристроях, що працюють під керуванням операційної системи Microsoft Windows. Зображення покажчиків CUR мають різні стандартні розміри, наприклад 16 × 16, 32 × 32 тощо. Файли CUR дуже схожі на файли ICO; обидва вони складаються з невеликих нерухомих зображень. Відрізняються лише розширення файлів CUR і ICO. Крім того, пояснення точки доступу в заголовку файлу CUR відрізняється від пояснення у файлі ICO. Точка доступу інтерпретує зміщення пікселів від верхнього лівого кута до місця, куди ви вказуєте мишею. Системи Windows все ще використовують файли CUR, але тепер анімовані курсори зазвичай мають розширення файлу ANI замість CUR.

## Коротка історія ##

Файл CUR створюється з файлу ICO. Перший формат файлу CUR було включено в операційну систему Microsoft Windows 1.0 у 1981 році для полегшення комерційного використання. Незабаром було представлено більше інтерактивних курсорів, які дозволяли користувачам змінювати свої параметри курсору за допомогою попередньо встановлених параметрів. Однак файл CUR легко редагувати самостійно, навіть якщо у вас недостатньо технічних знань.
