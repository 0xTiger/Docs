## Що таке файл AGP?

Файл AGP — це 32-розрядний файл зображення з кількома шарами. Він створений за допомогою програмного забезпечення ArtGem, яке забезпечує можливості роботи з зображеннями та за основними функціями схоже на Adobe Photoshop. AGP — це файл проекту, який містить усі налаштування для створення остаточного образу.

Файли AGP можна відкрити за допомогою RL Vision ArtGem. Однак ArtGem вже давно припинено після продажу.

## Формат файлу AGP

Файли AGP були збережені у власному форматі ArtGem. Внутрішня файлова структура файлів AGP недоступна для розробників.
