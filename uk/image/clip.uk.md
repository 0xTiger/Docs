## Що таке файл CLIP?

Файл CLIP — це файл растрового зображення, створений програмою для двовимірного малювання Clip Studio Paint. Він схожий на файл [PSD](/uk/image/psd/) від Adobe і містить кілька шарів для керування різними графічними елементами у формі шарів, таких як текст, зображення, ручка, олівець, пензлі та фігури. Формат файлу CLIP використовується для розробки різних типів графіки, таких як ілюстрації та анімація. Додаток Clip Studio Paint розроблено японською компанією CELSYS, Inc., заснованою в травні 1991 року.

## Формат файлу CLIP - Додаткова інформація

Формат файлу CLIP був представлений компанією CELSYS для заміни оригінального формату файлу LIP. Він зберігає графічні елементи у вигляді шарів, які можна редагувати для змін. Це також формат файлу за замовчуванням для програми [CLIP Studio Paint](https://www.celsys.co.jp/en/service/studio_paint/), яка доступна для платформ Windows, MacOS, Android та iOS.

## Підтримувані типи малюнків

Формат файлу CLIP може зберігати анімацію та ілюстрації, які можна використовувати в різних типах робіт.
