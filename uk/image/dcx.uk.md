## Що таке файл DCX?

Файл DCX — це файл зображення, який складається з кількох сторінок для створення файлів цифрового факсу. Кожен файл DCX починається з 4-байтового заголовка, який визначає файл DCX як підпис, за яким ідуть окремі зображення, які зберігаються як вбудовані файли [PCX](/uk/image/pcx/). 4-байтовий підпис ідентифікує файл DCX і супроводжується списком до 1023 4-байтових зміщень файлу. Формат файлу DCX розроблений Zsoft. Кілька програм можуть відкривати файли DCX, наприклад Adobe Photoshop, Corel PaintShop Pro та IrfanView.
