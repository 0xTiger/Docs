## Що таке файл GBR?

Файл із розширенням .gbr — це формат файлу зображення Gerber для обміну даними про проектування друкованої плати (PCB). Його розробила Ucamco. Дані про проектування друкованих плат є основним компонентом, необхідним для обробки промисловістю. Файл GRB містить дані друкованої плати, такі як мідні шари, паяльну маску, легенду, а також дані про свердління та маршрути. Його можна використовувати для передачі таких даних, як характеристики виготовлення друкованої плати, включаючи товщину або обробку, у стандартизованому форматі. Усі системи проектування друкованих плат виводять файли Gerber, які можуть оброблятися системами виготовлення друкованих плат. Файли GBR тепер стали стандартом де-факто для передачі даних проектування друкованих плат (PCB). Ucamco надала [безкоштовний онлайн-переглядач](https://gerber-viewer.ucamco.com/), щоб відкривати та переглядати формати файлів GBR.

## Формат файлу GBR

GBR — це зрозумілий формат UTF-8, який складається лише з 27 команд. Завдяки цьому короткому списку команд і його компактності GBR легко налагодити. За своєю суттю Gerber є відкритим векторним форматом для двовимірних бінарних зображень. Метаінформація передається разом із зображеннями через атрибути. Один файл GRB визначає одне зображення та не потребує жодних супровідних файлів або зовнішніх параметрів для інтерпретації. Для одного зображення потрібен лише один файл. Він використовує 7-розрядні символи ASCII для всіх команд і імен, визначених у специфікації, які можна роздрукувати. Це повністю охоплює створення повного зображення.

### Приклад файлу GBR

Нижче наведено приклад файлу Gerber, який створює коло діаметром 1,5 мм із центром у початку координат. Є одна команда на рядок.

```
%FSLAX26Y26*%
%MOMM*%
%ADD   100C,1.5*%
D100* X0Y0D03*
M02*
```
