## Що таке файл GCODE?

**Файл GCODE** — це простий текстовий файл, який містить інструкції з керування комп’ютеризованими верстатами та 3D-принтерами; G-код, або «геометричний код», — це мова, яка використовується для керування рухами та діями верстатів **CNC (комп’ютерне числове керування)**; Верстати з ЧПК включають фрезерні верстати, токарні верстати, маршрутизатори та 3D-принтери.

Команди G-коду записуються в спеціальному синтаксисі, який зазвичай складається з літер і цифр; кожна команда наказує машині виконати певну дію, наприклад перемістити інструмент у певне положення, змінити інструмент або налаштувати швидкість.

G-код часто генерується програмним забезпеченням **CAM (Computer-Aided Manufacturing)**; Програмне забезпечення CAM використовує 3D-модель або 2D-дизайн і генерує відповідні траєкторії та інструкції G-коду; файл G-коду потім завантажується в машину з ЧПК або 3D-принтер для виконання.

Файли G-code зазвичай мають розширення файлу .nc або .gcode, наприклад program.nc або print.gcode.

## Структура файлу GCODE:

Файли GCODE є простими текстовими файлами, кожен рядок яких містить певну команду; ці команди варіюються від керування рухом машини до налаштування температури, швидкості та інших параметрів, важливих для виготовлення об’єкта.

Синтаксис GCODE передбачає поєднання літер і цифр; кожен представляє окрему дію або параметр; загальні команди включають G0 і G1 для руху, M3 і M5 для керування шпинделем і S і F для регулювання швидкості та швидкості подачі відповідно.

## Генерація GCODE:

Програмне забезпечення для нарізки, таке як **Simplify3D** і **Slic3r**, переводить креслення автоматизованого проектування (CAD) у GCODE; Програмне забезпечення CAD використовується для створення 3D-моделей, які потім експортуються у такі формати, як STL; програмне забезпечення для нарізки використовує ці моделі та генерує файл GCODE із зазначенням таких деталей, як висота шару, швидкість друку та налаштування температури.

## Приклад GCODE

Ось простий приклад G-коду для переміщення верстата з ЧПК:

```
G0 X10 Y5      ; Rapid move to position X=10, Y=5
G1 Z2 F500     ; Linear move to Z=2 at feed rate of 500 units/minute
M3 S1000       ; Start spindle at 1000 RPM
G2 X20 Y10 I2 J0   ; Clockwise circular interpolation
G0 Z5          ; Rapid move to Z=5
M5             ; Stop spindle
```

## Як відкрити файл GCODE?

Щоб відкрити файл G-code, ви можете використовувати різні типи програм залежно від ваших потреб.

Якщо ви створили G-код для 3D-принтера; ви можете відкрити його за допомогою програмного забезпечення, яке постачається з вашим 3D-принтером, або спеціального програмного забезпечення для нарізки; приклади включають **PrusaSlicer**, **Cura**, **Simplify3D**, **MatterControl** або **Repetier-Host**; ці програми часто мають зручний інтерфейс, який дозволяє завантажувати та візуалізувати G-код.

Файли GCODE є звичайним текстом, тому ви можете відкрити їх у будь-якому текстовому редакторі; поширені текстові редактори включають **Блокнот (у Windows)**, **TextEdit (у macOS)** або **Gedit (у Linux)**; просто **клацніть правою кнопкою миші** на файлі G-коду, виберіть **Відкрити за допомогою** і виберіть текстовий редактор.
