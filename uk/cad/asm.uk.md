## Що таке файл ASM?

Файл ASM — це файл даних, який використовується програмним забезпеченням [Solid Edge](https://solidedge.siemens.com/en/) для 3D-моделювання, візуалізації та складання. Він містить тривимірні об’єкти, які складаються зі змодельованих частин. Один файл ASM може містити кілька частин, щоб скласти весь файл даних. Файли ASM повністю відрізняються від файлів ASM [мови складання](/uk/programming/asm/) і не мають до них зв’язку. Для відкриття цих файлів ASM використовується програмне забезпечення Siemens Solid Edge.
