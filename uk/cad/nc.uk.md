## Що таке файл NC?

NC-файл – це файл комп’ютерного числового керування (CNC), створений за допомогою [програмного забезпечення Mastercam CNC](https://www.mastercam.com/). Він містить інструкції з числовим керуванням для керування механічним рухом обробних інструментів з ЧПК, таких як свердла, токарні верстати, фрези та 3D-принтери. Верстати з ЧПК використовують інструкції з файлів ЧПУ у формі послідовної програми для виконання. Ці інструкції з файлу NC призначені для керівництва машиною для виробництва компонента.

## Формат файлу NC

NC-файли зберігаються на диску як двійкові файли, і їхні внутрішні специфікації формату файлу недоступні. Програмне забезпечення Mastercam CNC дозволяє користувачам створювати 2D або 3D траєкторії верстатів. Ці траєкторії, збережені на диску, створюють файл .nc, який нарешті використовується верстатом з ЧПК для створення потрібного компонента.
