Що таке файл MDE?

Файл .MDE — це скомпільована версія бази даних Access (.mdb або .accdb). Коли ви компілюєте базу даних Access у файл .MDE, її більше не можна редагувати за допомогою стандартних інструментів розробки Access. Основною метою створення файлу .MDE є розповсюдження програми бази даних без розкриття оригінального вихідного коду.

## Формат файлу MDE

Файл MDE створюється шляхом компіляції коду VBA, форм, звітів та інших об’єктів. Це призводить до створення двійкового формату файлу MDE. Отриманий файл .MDE можна розповсюдити серед користувачів, які можуть запускати програму, але не можуть змінювати дизайн або переглядати вихідний код. Файл MDE насправді є скомпільованою версією [файлу .MDA](/database/mda/). Розповсюдження файлів .MDE може допомогти захистити інтелектуальну власність, не дозволяючи користувачам отримати доступ або змінити вихідний код. Це також може підвищити продуктивність під час компіляції та оптимізації коду.
