## Що таке файл AMXX?

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. Це платформа сценаріїв, яка дозволяє адміністраторам серверів налаштовувати та розширювати функціональні можливості своїх серверів. AMXX використовується разом із системою керування плагінами MetaMod, яка дозволяє одночасно запускати декілька плагінів на сервері без конфліктів. Функції плагіна AMXX включають команди адміністратора, модифікації ігрового процесу та керування сервером.
