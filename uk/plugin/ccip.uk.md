## Що таке файл CCIP?

Файл CCIP — це файл плагіна, створений програмою Curse Client, який використовується для керування та встановлення доповнень для популярних ігор, таких як World of Warcraft, Minecraft та інших. Додатки — це створені користувачем модифікації або вдосконалення інтерфейсу або функцій гри, які можуть покращити ігровий досвід, надаючи додаткові функції або параметри налаштування. Файл CCIP містить необхідну інформацію для завантаження та встановлення аддона для однієї з керованих ігор.

## Формат файлу CCIP

Коли ви вибираєте в програмі опцію «Встановити через Curse Client», вона завантажує файли як файли CCIP. Ці файли CCIP можна завантажити через клієнт Curse. Клієнт Curse дозволяв користувачам легко знаходити, завантажувати та керувати своїми ігровими додатками. Він забезпечив централізоване розташування для користувачів, щоб знайти та встановити останні версії своїх улюблених додатків.
