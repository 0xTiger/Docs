## Що таке файл CLEO?

Файл .cleo — це файл плагіна, який використовується як частина бібліотеки CLEO для серії ігор GRAND Theft Auto. Він використовується для виконання модифікацій, які змінюють або розширюють ігровий процес. Файл CLEO зберігає бібліотеку у форматі файлу DLL, додаючи корисні команди сценаріїв (коди операцій) для роботи із зовнішніми файлами для зміни ігрової пам'яті. Існують різні версії CLEO для GTA III, GTA Vice City і GTA San Andreas.

## Формат файлу CLEO

CLEO 4 дозволяє створювати нові коди операцій за допомогою джерела CLEO SDK. Три найпоширеніші приклади таких файлів плагінів включають:

 * IniFiles.cleo - коди операцій для файлів INI;
 * FileSystemOperations.cleo - коди операцій для файлів і папок.
 * IntOperations.cleo - коди операцій для логічних операцій (AND, OR, XOR тощо).

Плагіни встановлюються подібно до скриптів. Щоб встановити плагін CLEO, вам потрібно скопіювати файл .CLEO в папку CLEO.

## Про CLEO

CLEO — це надзвичайно популярний розширюваний плагін для серії ігор Grand Theft Auto від Rockstar Games, який дозволяє використовувати тисячі унікальних модів, які змінюють або розширюють ігровий процес.
