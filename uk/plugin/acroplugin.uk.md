## Що таке файл ACROPLUGIN?

Файл із розширенням .acroplugin — це файл плагіна, який використовується Adobe Acrobat. Він використовується для додавання функціональності до існуючих можливостей програми. Плагін постачається в комплекті з Adobe Acrobat, макетом сторінки та програмою для публікації [PDF file](/pdf/). Як і інші плагіни, файл ACROPLUGIN розширює існуючу функціональність, яку пропонує програма.

## Формат файлу ACROPLUGIN - Додаткова інформація

Файл ACROPLUGIN явно використовується Adobe Acrobat для розширення наявних функціональних можливостей. Деталі внутрішнього формату файлу недоступні для розробників.

## Як відкрити файл ACROPLUGIN?

Ви можете відкрити файл ACROPLUGIN за допомогою Adobe Acrobat.
