## Формати файлів PRC
Розширення «.prc» використовуються для формату файлу Product Representation Compact 3D і формату файлу електронної книги від MobiPocket.

## Що таке компакт-файл представлення продукту (PRC)?

PRC (Product Representation Compact) — це формат 3D-файлу, оптимізований для зберігання, завантаження та відображення 3D-даних, особливо надходять із систем CAD (Computer-Aided Design). Це послідовний бінарний файл, записаний переносним способом. PRC можна використовувати для вбудовування 3D-даних у файл PDF. PRC підтримує більшість основних конструкцій САПР, таких як вузли та деталі, дерева тривимірних об’єктів, точне представлення геометрії, трикутне представлення та розмітку. PRC використовує розширення .prc, а використовуваний тип інтернет-носія — *model/prc*.

PRC — це багатоцільовий формат файлу, який, залежно від його використання, можна зберігати з використанням звичайного стиснення для безпосереднього представлення даних CAD або зберігати з використанням високого стиснення для зменшення розміру файлу. Тривимірні дані, що зберігаються у форматі PDF у форматі PRC, сумісні з додатками CAM і CAE.

### Специфікація формату файлу Product Representation Compact (PRC).

Файл PRC має один головний розділ заголовка, за яким слідує одна або кілька файлових структур, за якими в кінці йде один файл моделі. Структура файлу має один заголовок, за яким йдуть такі розділи даних:

- **Globals**: містить посилання на файлові структури та кольори, стилі ліній і системи координат для кожної деревоподібної сутності файлової структури.
- **Дерево**: містить опис дерева елементів, таких як входження продукту, визначення частин, елементи представлення та розмітка.
- **Тесселяція**: містить усі тесселяційні (триангульовані) дані в листкових сутностях дерева (представницькі елементи та розмітки).
- **Геометрія**: містить усі точні дані про геометрію та топологію листкових об’єктів дерева (представницькі елементи).
- **Додаткова геометрія**: містить підсумкові дані геометрії. Це дозволяє частково завантажити файл без завантаження всієї геометрії.

Нижче наведено структуру типового файлу PRC.

```console
PRC Header

File Structure #1
- Header for File Structure #1
- Globals for File Structure #1
- Tree for File Structure #1
- Tessellation for File Structure #1
- Geometry for File Structure #1
- Extra Geometry for File Structure #1

File Structure #2
...

File Structure #n
- Header for File Structure #n
- Globals for File Structure #n
- Tree for File Structure #n
- Tessellation for File Structure #n
- Geometry for File Structure #n
- Extra Geometry for File Structure #n

PRC model file data
```
## Що таке файл електронної книги MobiPocket із розширенням PRC
Формат файлу електронної книги, який представила французька компанія під назвою **Mobipocket**, також використовує розширення «.prc». Спочатку Mobipocket запустив безкоштовну програму для багатьох розумних пристроїв, таких як планшети, КПК і смартфони. Розширення ".prc" фактично ідентичне .mobi, але використовується особливо для КПК, які підтримують лише розширення **.prc** або **.pdb**.

### Характеристики формату файлу Mobipocket PRC
Формат файлу MobiPocket PRC базується на стандарті Open eBook із використанням XHTML, а також може містити JavaScript і фрейми. Також доступна підтримка власних запитів SQL для використання з вбудованими базами даних.

Mobipocket Reader має бібліотеку домашньої сторінки. Читачі можуть вставляти порожні сторінки в будь-яку частину книги та додавати різні малюнки. Такими об’єктами, як анотації, закладки, виправлення, примітки та малюнки, можна керувати з одного місця. Зображення перетворюються у формат GIF і мають максимальний розмір 64 КБ, що достатньо для мобільних телефонів із маленьким екраном. Mobipocket Reader має електронні закладки та вбудований словник.

Рідер має повноекранний режим для читання та підтримку багатьох КПК, комунікаторів і смартфонів. Продукти Mobipocket підтримують більшість операційних систем Palm, Windows, Symbian, BlackBerry, але не Android. Зчитувач працює під Linux або Mac OS X за допомогою WINE.
