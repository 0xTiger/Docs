## Про FBX в USDZ

FBX, скорочення від FilmBox, означає широко поширений формат 3D-файлу, спочатку створений Kaydara для MotionBuilder. Компанія Autodesk Inc. стала власником у 2006 році, закріпивши свій статус основного формату обміну 3D-даними, який використовується багатьма програмними інструментами 3D.

Файл із розширенням .usdz представляє нестиснутий і незашифрований ZIP-архів, призначений для формату файлу [USD](/uk/3d/usd/) (Universal Scene Description). Він містить проксі-файли для різних типів файлів, включаючи текстури та анімацію, усі зручно інкапсульовані в архіві.

Якщо ви розробник програми, який має на меті включити функції [FBX](/uk/3d/fbx/) до USDZ у свою програму .NET, Java або Python, ви можете дослідити API Aspose.3D як цінний ресурс для досягнення ця мета.
