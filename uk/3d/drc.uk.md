## Що таке файл DRC?

Файл із розширенням .drc — це формат стисненого 3D-файлу, створений за допомогою бібліотеки Google Draco. Google пропонує Draco як бібліотеку з відкритим кодом для стиснення та розпакування 3D-геометричних сіток і хмар точок, а також покращує зберігання та передачу 3D-графіки. Він кодує вхідні дані та зберігає їх як файл .drc. На одержувачі API зчитує файли .drc і може виводити їх як файли [PLY](/uk/3d/ply/) або [OBJ](/uk/3d/obj/). Стислий вихідний файл дозволяє користувачам швидше завантажувати програми та забезпечує швидке завантаження 3D-графіки в браузерах.

## Формат файлу DRC

Інформація про формат файлу для DRC недоступна для всіх. Google розмістив бібліотеку Draco на [GitHub](https://github.com/google/draco/tree/master/docs) для довідки розробників. Він надає кодери та декодери для створення та перетворення файлів DRC.
