## Що таке файл ASE?

Файл із розширенням .ase — це формат файлу Autodesk ASCII Scene Export, який представляє сцену в форматі ASCII, що містить 2D або 3D інформацію під час експорту даних сцени за допомогою Autodesk. Autodesk надає параметри для включення компонентів сцени під час експорту даних сцени. Ви можете включити визначення сітки разом із інформацією про вершини та грані, описом матеріалу, даними анімації перетворення для об’єктів, повним визначенням сітки для кожних n кадрів, даними анімації для камер і освітлення та налаштуваннями IK-з’єднань.

## Формат файлу ASE – Додаткова інформація

Файли ASE — це текстові файли, що зберігаються у форматі ASCII, які можна відкрити в будь-якому текстовому редакторі. Файл ASE може містити такі типи інформації, надані Autodesk.

### Параметри виведення

* `Визначення сітки` - експортує визначення кожної сітки, включаючи інформацію про вершини та грані для геометричних об’єктів. Крім того, якщо ввімкнути це, увімкнуть елементи в груповому полі «Параметри сітки», описані нижче.
* `Матеріали` - містить опис матеріалу. Якщо матеріал не присвоєно об’єкту, експортується колір його каркаса. Включено всі рівні дерева матеріалів, тому це може створити багато тексту.
* `Ключі анімації перетворення` - включає дані анімації перетворення для об’єктів. Якщо об’єкт є цільовою камерою або прожектором, це включатиме дані цільової анімації.
* `Анімована сітка` - експортує повне визначення сітки для кожних n кадрів. Частота визначається регулятором виходу контролера, описаним нижче. Кожен блок містить однакову інформацію, указану в груповому полі «Параметри сітки», описаному нижче. Увімкнення цієї функції може призвести до величезного файлу навіть для невеликих сцен.
* `Налаштування анімованої камери/світла` - експорт даних анімації для камер і світла, таких як колір, інтенсивність, спад, зміщення карти тощо. Виводить блок кожні n кадрів, як указано регулятором виводу контролера.
* `Inverse Kinematic Joints` - експортує налаштування IK-з’єднань у гілці «Ієрархія».

### Типи об'єктів

Елементи тут дозволяють вказати, яку категорію об’єктів ви бажаєте включити у вивід. Ви можете включити геометричні об’єкти, фігури, камери, світло та допоміжні об’єкти.

* Геометричний
* Фігури
* Камери
* Світло
* Помічники

### Параметри сітки

* `Mesh Normals` - експортує грані та нормалі вершин. Нормаль грані вказана спочатку, а потім нормалі трьох вершин, що підтримують грань. Якщо ввімкнути це, файл буде значно більшим.
* `Координати відображення` - експортує список вершин і граней відображення відповідно до структур TVert і TVFace, описаних у комплекті розробки програмного забезпечення 3ds Max. Якщо об’єкт використовує відображення обличчя, експортується список карт обличчя, що містить координати UVW для кожного обличчя.
* `Кольори вершин` - експортує кольори вершин.

### Вихід контролера

* `Use Keys` - експортує ключові значення. Якщо контролер не використовує ключі, то використовується метод Force Sample. У випадку контролерів трансформації параметр «Використовувати ключі» працює, лише якщо всі контролери трансформації є лінійними/TCB або Без’є. Якщо одна з доріжок перетворення використовує інший тип контролера, тоді метод Force Sample використовується для всіх доріжок перетворення.
* `Force Samples` - вибірка значень контролера на основі частоти, зазначеної в кадрах на контролер вибірки.
