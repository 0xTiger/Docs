## Що таке файл RVM?

Файли даних **RVM** пов’язані з AVEVA PDMS. Файл **RVM** – це файл проекту моделі системи управління проектуванням заводу AVEVA. Система управління проектуванням заводу (PDMS) AVEVA є найпопулярнішою системою 3D-проектування, яка використовує технологію, орієнтовану на дані, для управління проектами. Численні програми доступні для відкриття та перетворення файлів RVM в інші формати, такі як [3DS](/uk/3d/3ds/).

## Формат файлу RVM

Про формат файлу RVM можна знайти небагато деталей, крім того, що його можна зберегти як у двійковому форматі, так і у форматі ASCII. Файл RVM підтримує такі сутності, як:

* Вся геометрія
* Атрибути, що зберігаються в групах
* Текстури (через файл RVS)
* Камери та треки камер (через файл RVS)
* Кліп площини (через файл RVS)
* Знаки (через файл RVS)
* Теги (через файл RVS)
* Мітки (через файл RVS)
* Прозорість (через файл RVS)
* Початкові точки PDMS
