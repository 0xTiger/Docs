## Що таке файл DBM?

Файл DBM (Daz Brick Material) — це мережевий файл Shader Mixer, створений за допомогою програми 3D-моделювання персонажів **DAZ Studio**. Він описує матеріал режиму 3D, тобто поверхню/переміщення/внутрішній об’єм. DAZ дозволяє створювати власний власний шейдер, з’єднуючи серію цеглинок. Це разом об’єднується як файл DBM, який утворює мережу шейдерів. Саме файл DBM вказує, як затінюється матеріал.

Програми, які можуть **відкривати файли DBM**, включають DAZ 3D DAZ Studio 4.

## Формат файлу DBM

Файли DBM зберігаються на диск у форматі [XML](/uk/web/xml/).
