## Про FBX до 3DS

FilmBox, який зазвичай називають FBX, є широко поширеним форматом 3D-файлів, спочатку розробленим Kaydara для забезпечення сумісності з MotionBuilder. Autodesk Inc. взяла на себе контроль над цим форматом у 2006 році, встановивши його як ключовий стандарт для безперебійного обміну 3D-даними між різноманітними програмними програмами для 3D.

Файл із розширенням [.3ds](/uk/3d/3ds/) означає формат файлу сітки 3D Studio (DOS), який використовується програмним забезпеченням Autodesk 3D Studio. Присутність Autodesk на арені формату 3D-файлів сягає 1990-х років і з тих пір перетворилася на 3D Studio MAX, комплексний інструмент для 3D-моделювання, анімації та візуалізації. Файл .3ds інкапсулює дані для тривимірного зображення сцен і зображень, що робить його одним із улюблених форматів файлів для імпорту та експорту 3D-даних.

Перетворення [FBX](/uk/3d/fbx/) у 3DS є частою вимогою користувачів. В Інтернеті є кілька безкоштовних інструментів для БЕЗКОШТОВНОГО перетворення FBX у 3DS.
