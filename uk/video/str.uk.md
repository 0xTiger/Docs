## Що таке файл STR?

Файл STR – це файл запису, створений YouTube для прямого ефіру, який організовував користувач. Файл автоматично зберігається на сервері Youtube після завершення прямої трансляції. Пряме потокове відео можна транслювати на кілька каналів, щоб аудиторія могла взаємодіяти з видавцем за допомогою коментарів. Ці файли SRT доступні за допомогою API прямої трансляції YouTube, який забезпечує доступ до серверів YouTube.

## Формат файлу STR

YouTube зберігає файли STR на сервері у двійковому форматі. Деталі внутрішньої файлової структури файлів STR недоступні для всіх.
