## Що таке файл SAMI?

Файл із розширенням ".sami" зазвичай відноситься до файлу субтитрів і обміну метаданими (SAMI). SAMI — це формат субтитрів, який використовується для відображення субтитрів або субтитрів у відео. Спочатку він був розроблений компанією Microsoft для медіапрогравача Windows.

Файли SAMI містять інформацію про час і текстовий вміст для субтитрів або субтитрів, що дозволяє їх синхронізувати з відтворенням відео. Формат підтримує основні параметри форматування, такі як стиль шрифту, колір і розташування субтитрів на екрані.

Файли SAMI зазвичай є простими текстовими файлами, їх можна відкривати та редагувати за допомогою текстового редактора. Вміст файлу SAMI зазвичай включає розділ заголовка, який містить інформацію про файл, а потім окремі записи субтитрів із відповідним часом і текстом.

Ось приклад того, як може виглядати файл SAMI:

```
<SAMI>
<HEAD>
<TITLE>Example Subtitles</TITLE>
</HEAD>
<BODY>
<SYNC Start=100><P Class=ENCC>Subtitle 1</P></SYNC>
<SYNC Start=500><P Class=ENCC>Subtitle 2</P></SYNC>
<SYNC Start=1000><P Class=ENCC>Subtitle 3</P></SYNC>
</BODY>
</SAMI>
```

Файли SAMI зазвичай використовуються в поєднанні з відеопрогравачами або медіапрогравачами, які підтримують відображення субтитрів, наприклад Windows Media Player або VLC Media Player. Програвач зчитує файл SAMI та синхронізує субтитри з вмістом відео, дозволяючи глядачам читати субтитри під час перегляду відео.

## Підтримувані теги HTML

Файли SAMI (Subtitles And Metadata Interchange) підтримують обмежений набір HTML-тегів для форматування та стилізації субтитрів. Ось деякі з часто використовуваних тегів, які підтримує SAMI:

- ``<SAMI> :`` Кореневий елемент, який інкапсулює весь файл SAMI.
- ``<HEAD> :`` Містить інформацію заголовка для файлу SAMI.
<html>- ``<TITLE> :`` Вказує назву файлу SAMI. </html>
- ``<BODY> :`` Містить записи субтитрів та їх інформацію про час.
- ``<SYNC> :`` Представляє точку синхронізації для запису субтитрів. Він визначає час, коли мають відображатися субтитри.
- ``<P> :`` Охоплює фактичний текстовий вміст субтитрів. Зазвичай він використовується в межах a<SYNC> блокувати.
<html>- `` <FONT>:`` Визначає властивості шрифту для вкладеного тексту. Такі атрибути, як колір, обличчя, розмір і стиль, можна використовувати для зміни вигляду шрифту.</font>
- ``<BR> :`` Вставляє розрив рядка в субтитр.
<html>- `` <B>:`` Виводить текст у вкладенні жирним шрифтом.</b>
<html>- `` <I>:`` Відтворює текст, укладений курсивом.</i>
<html>- `` <U>:`` Підкреслює текст у вкладенні.</u>
- ``<C> :`` Визначає положення або вирівнювання тексту субтитрів на екрані. Він підтримує такі атрибути, як Center, Middle, Left, Right, Top, Bottom та їхні комбінації.
- ``<LANG> :`` Вказує код мови для тексту субтитрів. Це допомагає визначити мову субтитрів.

Це деякі з основних тегів, які підтримуються файлами SAMI. Важливо зазначити, що SAMI не підтримує повний спектр тегів і атрибутів HTML. Підтримувані теги в основному зосереджені на стилізації та позиціонуванні субтитрів, а не на забезпеченні розширеної структури документа чи інтерактивності.
