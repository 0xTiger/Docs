## Що таке файл ABA?

Файл із розширенням .aba — це формат файлу Австралійської банківської асоціації (ABA) або [Cemtext](https://www.cemtexaba.com/), який використовується банками для пакетних операцій. Він використовується більшістю банків для ведення фінансового обліку. Також відомий як файл прямого входу, формат файлу ABA був прийнятий більшістю австралійських банків як формат за замовчуванням для пакетних операцій. Його досі не визнано стандартним форматом, незважаючи на те, що він використовувався Bank of Queensland, NAB і Westpac. Файли ABA можна відкрити будь-яким текстовим редактором, наприклад Notepad++.


## Формат файлу ABA

Файл ABA використовує формат ASCII для зберігання даних для пересилання або завантаження в банківські системи. Формат був збережений простим, щоб полегшити інтеграцію у власну фінансову систему компанії для обробки пакетів транзакцій. Наприклад, у системі розрахунку заробітної плати пакет транзакцій може бути завантажений для обробки одним зверненням. Специфікації формату файлу ABA доступні як повна розшифровка на веб-сайті [Cemtext ABA](https://www.cemtexaba.com/aba-format/cemtex-aba-file-format-details), і на них можна посилатися для довідки розробників .

Файл ABA складається з кількох рядків, кожен рядок відомий як "запис". У файлі ABA є три основні записи:

* Описовий запис
* Детальний запис
* Загальний запис файлу

### Описовий запис

`Описовий запис` містить таку інформацію, як порядковий номер барабана, назва фінансової установи користувача, назва використання файлу, що надає, та іншу інформацію.

### Детальний запис

Тип «Докладний запис» включає таку інформацію, як номер банку/держави/філії, номер рахунку, який буде зараховано/дебетовано, індикатор, код транзакції, суму та іншу інформацію.

### Загальний запис файлу

Тип «Загальний запис файлу» включає тип запису 7, заповнювач формату BSB, загальну чисту суму файлу (користувача), загальну суму кредиту файлу (користувача), загальну суму дебету файлу (користувача) та іншу інформацію.

### Коди транзакцій

Нижче наведено список дійсних кодів транзакцій. У файлах ABA найчастіше використовується код "53 - Pay". Ці коди транзакцій охоплюють дебетування, кредити та податки, що утримуються.

|Код|Опис операції|
---|---|
|13|Зовнішні ініційовані дебетові позиції|
|50|Зовнішньоініційовані кредитні позиції, за винятком тих, що містять коди транзакцій|
|51|Охоронний інтерес уряду Австралії|
|52|Сімейна допомога|
|53|Оплата|
|54|Пенсійний|
|55|Наділ|
|56|Дивіденд|
|57|Відсотки за зобов'язаннями/нотами|
