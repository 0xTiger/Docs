## Що таке файл EPIBRW?

Файл EPIBRW — це загальний файл, розміщений на веб-сервері Microsoft IIS. Призначення цього файлу не є специфічним для будь-яких конкретних операцій чи будь-якого типу зберігання даних, а є загальним. Найчастіше він використовується для файлів [.pdf](/uk/pdf/), але також може використовуватися для кількох інших типів файлів.

Якщо файл EPIBRW двічі клацнути, він може відкритися в браузері з відображенням вмісту. Однак, якщо на файл є посилання для завантаження, процес завантаження файлу розпочнеться в результаті цієї події.
