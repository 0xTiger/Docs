## Що таке файл ARO?

Файл ARO — це файл веб-сторінки на стороні сервера, створений за допомогою веб-програми SteelArrow. Він містить теги та сценарії [HTML](/uk/web/html/) і SteelArrow. Вони виконуються на сервері для створення повної сторінки відповіді перед надсиланням у браузер користувача. Файли ARO містять майже 95% вмісту HTML, а решта складається з коду SteelArrow. Щоб файли ARO працювали для вас, сервер веб-додатків SteelArrow має бути встановлено та запущено на машині веб-сервера.

## Формат файлу ARO

Файли ARO написані у файлі мови розмітки HTML із вбудованим кодом JavaScript і аплетами Java. [Теги SteelArrow](http://www.steelarrow.com/docs/basics1.aro) є основними будівельними блоками для розробки веб-сторінок. Хоча вони не змінюють відображення сторінки, вони використовуються для заповнення сторінки даними. Крім того, вони також можуть використовуватися для керування виведенням за допомогою умовних операторів.

### Приклад тегу ARO

The<SAOUTPUT> Тег ARO дозволяє розробникам виводити значення даних у будь-якому місці сценарію. Наприклад, його можна використовувати для отримання результату таблиці PARAM<SAOUTPUT VALUE=PARAM[]> які можна відобразити в HTML<BODY> тег.
