## Що таке файл AN?

Файл із розширенням .an — це файл проекту [Adobe Edge Animate](https://www.adobe.com/products/animate.html), який містить інформацію про анімацію. Під час збереження проекту створюється файл .html, який містить посилання на пов’язані файли [.css](/uk/web/css/) і [.js](/uk/web/js/), згенеровані Adobe Edge Animate. Файл AN зберігається разом із цими файлами, які можна використовувати для відкриття файлу проекту. Adobe edge animate можна використовувати як альтернативу створенню анімованих веб-сторінок за допомогою Adobe Flash, який більше не підтримується браузерами.

## Формат файлу AN

Файли AN зберігаються на диску у форматі звичайного текстового файлу та можуть бути відкриті в текстових редакторах для перегляду їх вмісту.
