## Що таке файл DOCMHTML?

Файл DOCMHTML — це старий варіант веб-архіву, збережений попередніми версіями Microsoft Word. Він більше не підтримується останньою версією Microsoft Word. Файл DOCMHTML містить HTML-представлення оригінального документа .doc Word. Його можна відкрити в Microsoft Word, а також у веб-браузері. Коли DOCMHTML публікується, його можна розмістити на веб-сервері, доступному для користувачів через веб-запит. Ще один подібний формат — [DOCHTML](/uk/web/dochtml/), який також припинено.

## Формат файлу DOCMHTML – Додаткова інформація

Файли DOCMHTML були використані для збереження у форматі HTML. Його можна відкрити в текстовому редакторі, наприклад Notepad і Notepad++. Після того, як Microsoft Word був позначений як застарілий, у Microsoft Word були представлені параметри збереження документа Word як файли [MHTML](/uk/web/mhtml/) і [MHT](/uk/web/mht/). Ці нові формати файлів можна відкривати за допомогою веб-браузерів.
