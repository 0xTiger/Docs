## Що таке файл TS?

Файл TS — це файл перекладу, створений за допомогою утиліти lupdate, яка постачається разом із Digia Qt SDK. SDK доступний для операційних систем Windows, Mac і Linux. Файл TS містить фрази однією мовою та їх переклади іншою мовою. Переклади, що зберігаються у файлі TS, є специфічними для конкретної програми. Файли TS можна відкрити за допомогою Digia Qt SDK.

## Формат файлу TS

Файли Digia Qt TS зберігаються на диску у форматі файлу XML і можуть бути відкриті в будь-якому текстовому редакторі, наприклад Notepad і Notepad++. Деякі інші формати файлів, які підтримує Digia Qt SDK, включають:

* .при
* .qm
* .qml
* .qph
* .qrc
* .qs
* .qss
* .ts
* Файл інтерфейсу користувача .ui
