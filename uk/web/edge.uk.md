## Що таке файл EDGE?

Файл із розширенням .edge — це файл проекту Adobe Edge, який створюється під час створення та збереження проекту в Adobe Edge. Adobe Edge дозволяє веб-дизайнерам створювати інтерактивні [HTML](/uk/web/html/) аніматори для Інтернету, цифрової публікації та мультимедійної реклами як для комп’ютерних, так і для мобільних браузерів. Це досягається за допомогою HTML/HTML5, [JavaScript](/uk/web/js/) і [CSS3](/uk/web/css/). Файли Edge створюються як звичайні текстові файли, і їх можна відкривати за допомогою кількох різних програм, таких як Microsoft Notepad, Apple TextEdit і Notepad++.

## Формат файлу EDGE

Файли EDGE зберігаються у форматі звичайного тексту та використовують jQuery для активації анімації. Альтернативою анімації EDGE були файли Flash, які більше не підтримуються. Файл EDGE зберігається разом із файлами HTML, JavaScript і CSS. Після відкриття файлу EDGE відкривається веб-сторінка, тобто файл HTML.
