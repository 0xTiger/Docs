## Що таке файл APKG?

Файл із розширенням .apkg — це колода карток, згенерованих для використання в програмному додатку [Anki](https://ankiweb.net/about), який є навчальною програмою на основі карток. Він містить [HTML](/uk/web/html/) текст для завантаження та відображення в програмі Anki, а також може додатково містити зображення та звуки для візуального та аудіального навчання. Anki дозволяє користувачам створювати власні колоди карток Anki, а також імпортувати колоди карток інших користувачів.

## Формат файлу APKG

Колоди карток Anki створюються на основі шаблонів, написаних на HTML, відомій і поширеній мові для створення веб-сторінок. Стилізація колод карт виконується за допомогою мови CSS, яка використовується для оформлення веб-сторінок. Стиль включає модифікації:

* font-family - назва шрифту, який буде використовуватися на картці.
* розмір шрифту - розмір шрифту в пікселях.
* text-align - Визначає, чи слід вирівнювати текст по центру, ліворуч чи праворуч.
* колір – визначає колір тексту, який може бути простими назвами кольорів, наприклад «синій», «червоний» тощо, або кодами кольорів HTML.
* колір фону - визначає колір фону картки

Інформація про стиль спільно використовується для всіх карток, що впливає на всі картки, коли вноситься зміна. У наступному прикладі буде використано жовтий фон на всіх картках, крім першої:

```CSS
.card {
  background-color: yellow;
}
.card1 {
  background-color: blue;
}
```

Зміною розміру зображень у картці Anki можна керувати наступним чином.

```CS
img {
  max-width: none;
  max-height: none;
}
```

## Вбудовування Javascript у картки Anki

Можна вставити [Javascript](/uk/web/js/) у картку Anki за допомогою шаблонів карток. Однак через просунутий рівень Javascript його підтримка не передбачена. Крім того, пристрої рендерингу можуть по-різному відображати вплив впровадження Javascript на картку, що призводить до необхідності тестування впровадження на всіх пристроях. Певні функції Javascript, такі як window.alert, ускладнюють налагодження коду Javascript, який ви написали.
