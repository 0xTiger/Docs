## Що таке файл ATOM?

Файл ATOM — це формат синдикації для структури каналу Atom і документів введення. Подібно до файлів .RSS, формат файлу ATOM є форматом синдикації на основі XML, який є стандартом для публікації вмісту. Файл ATOM використовується для веб-каналів, які дозволяють програмам перевіряти наявність оновлень, опублікованих на веб-сайті. Він містить записи різних типів, наприклад заголовки, повні тексти статей, уривки або посилання на вміст веб-сайту

Програми, які можуть **відкривати файли ATOM**, включають **Apple Safari**.

## Формат файлу ATOM – Додаткова інформація

Файли ATOM зберігаються та публікуються в популярному форматі файлів XML, який є універсальним форматом для обміну інформацією. Він був розроблений як альтернатива RSS з урахуванням обмежень і недоліків формату файлу RSS.

### Типи документів Atom

1. `Документи Atom Entry` – це XML-документ, який складається з одного елемента інформації, відомого як запис, для каналу Atom. Він має<atom:entry> елемент, який містить кілька дочірніх елементів. Вмістом запису Atom може бути звичайний текст, HTML, XHTML або інший тип медіа IANA (Internet Assigned Numbers Authority).
1. `Документи каналу Atom` - це XML-документ, який надає метадані про канал Atom і один або кілька записів для каналу. Коли клієнт робить запит на отримання інформації з каналу, сервер створює документ каналу, який містить кілька записів Atom для виконання запиту.
1. `Колекція Atom` - це особливий вид документа каналу Atom, який містить URL-адреси записів Atom, доступних для редагування.
