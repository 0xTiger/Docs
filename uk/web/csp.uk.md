## Що таке файл CSP?

Файл CSP — це динамічний файл веб-сторінки, який містить вміст, написаний для виконання на RadGs Concept Application Server. Він компілюється та виконується на стороні сервера, після чого подається запитуючому браузеру користувача через протокол HTTP. На відміну від формату файлів CON, файли CSP не вимагають встановлення клієнта Concept у користувача для перегляду цих файлів. Ви можете переглядати файли CSP подібно до сторінок будь-якого іншого сервера, таких як [PHP](/uk/programming/php/) і [ASP](/uk/web/asp/).
