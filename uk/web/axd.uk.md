## Що таке файл AXD?

Файл AXD — це файл веб-налаштувань, який використовується для обробки та отримання запитів на вбудовані ресурси в ASP.NET. Він містить інструкції щодо отримання вбудованих ресурсів, таких як зображення, файли JavaScript ([.JS](/uk/web/js/)) і файли [.CSS](/uk/web/css/). Файли AXD використовуються для введення ресурсів у сторінки клієнта. Це дозволяє клієнтським сторінкам отримувати доступ до цих ресурсів на сервері стандартним способом.

## Формат файлу AXD

Файли AXD зберігаються як двійкові файли на стороні сервера. Це стосується файлу веб-обробника в ASP.NET, який є компонентами, які генерують відповіді на певні запити до веб-сервера. Файли AXD виконують спеціальну обробку перед тим, як генерувати відповідь на основі запитів сторінок клієнта. Зазвичай вони зберігаються як файли **WebResource.axd**.

### Що таке файл WebResource.axd?

Більшість проектів ASP.NET зберігають файли AXD як файли WebResource.axd, які дозволяють клієнтським сторінкам завантажувати ресурси, вбудовані в збірку. Ваша програма може мати низьку продуктивність, якщо ви запускаєте її в режимі налагодження, оскільки обробник WebResources.axd не кешується.
