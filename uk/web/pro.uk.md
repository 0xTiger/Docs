## Що таке файл PRO? ##

Файл PRO — це макет веб-сайту або мобільної програми, створений Adobe Proto, неіснуючою програмою Adobe Touch. Він містить один або кілька макетів інтерфейсу користувача для веб-сторінки або мобільного додатка, кожен з яких може містити інтерактивні елементи. Технології CSS, WebKit і jQuery підтримуються у файлах PRO.

### Коротка історія формату файлів PRO

Файли PRO були створені Adobe Proto, який був замінений Adobe XD. Його використовували для створення прототипів програм для планшетів. Поширені програми веб-дизайну, такі як Adobe Dreamweaver, використовували ці PRO-файли для створення веб- та мобільних програм.

## Формат файлу PRO - більше інформації

Файл PRO – це перший крок у розробці веб-сайту чи мобільного додатку. Розробники спочатку створюють цифрові макети, які визначають зовнішній вигляд програми. Вони також визначають загальний робочий процес програми та навігацію екранами програми. Ці макети також включають ряд навігаційних елементів, таких як кнопки, поля зі списком, смуги прокрутки, поля введення та деякі інші.
