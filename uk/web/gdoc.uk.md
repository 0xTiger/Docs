## Що таке файл GDOC?

Файл GDOC – це файл ярлика, який вказує на файл, розташований на Диску Google, хмарній службі зберігання документів. Коли на комп’ютері встановлено клієнт Google Drive і в ньому створюється документ, диск створює документ у хмарному онлайн-сховищі та записує [URL](/uk/web/url/) цього документа у файл GDOC у локальному Google Папка диска. Коли цей файл ярлика двічі клацнути, веб-браузер за замовчуванням відкриває документ із розташування [URL](/uk/web/url/). Якщо цей файл ярлика переміщено з цієї папки, посилання на онлайн-документ не працює.

## Формат файлу GDOC – Додаткова інформація

Файл GDOC містить ярлик для онлайн-документа, написаного у форматі [JSON](/uk/web/json/). Браузер, який відкриває файли GDOC, зчитує інформацію про URL-адресу та відкриває пов’язаний документ для відображення, якщо користувач увійшов до цього облікового запису Google, де зберігається документ. Файли GDOC не містять вмісту документа.

### Приклад файлу GDOC

Файл GDOC можна відкрити в текстовому редакторі, і його вміст виглядає наступним чином.

```
{"url": "https://docs.google.com/a/test.com/spreadsheet/ccc?key=01234567898765432123456789&usp=docslist_api", "resource_id": "spreadsheet:0A12345B678HJK9TZPL9078767"}
```

Як видно, вміст відформатовано в JSON із URL-адресою документа та пов’язаним ідентифікатором документа.
