## Що таке файл HTC?

Файл HTC — це файл формату веб-файлу, який використовується Microsoft Internet Explorer. Він містить визначення компонентів HTML, які виконують невеликі функції для виконання різних завдань у веб-браузері. Функціональні можливості, визначені у файлах HTC, можна використовувати повторно та доступні на кількох [HTML](/uk/web/html/) сторінках.

## Формат файлу HTC - Додаткова інформація

Файли HTC зберігаються у форматі простого тексту. Це означає, що ви можете відкрити їх у будь-якому текстовому редакторі та внести зміни до функціональності. Зазвичай вони відкриваються в Internet Explorer, але таким чином ви зможете лише переглядати ці файли, але не зможете їх редагувати.

Файл HTC насправді є файлом HTML, який є комбінацією сценарію та набору специфічних для HTC елементів для визначення компонентів.
