## Що таке файл STC?

Файл із розширенням .stc — це файл підключення, який використовується програмним забезпеченням Adobe Contribute для зберігання інформації про підключення. Він містить інформацію про з’єднання, таку як ім’я користувача, URL-адресу хосту, протоколи, такі як FTP, SFTP тощо, для повторного використання на існуючому веб-сайті. Для тих, хто не знає, Adobe Contribute раніше був спеціалізованим [HTML](/uk/web/html/) редактором, який використовувався для надсилання оновлень на існуючі веб-сайти, включаючи статті на основі вмісту, відомі як блоги. Він містив плагіни для популярних веб-браузерів, таких як Internet Explorer і Firefox, які безперешкодно додавали вміст на веб-сайти зі сторінок. Його було припинено з січня 2017 року, і зараз Adobe Dreamweaver використовується для створення веб-сайтів з нуля.

## Формат файлу STC

Внутрішній формат даних файлів STC невідомий. Однак ключ з’єднання зашифровано паролем, і будь-яка інформація про підключення до мережі або FTP, надіслана у файлі, залишається захищеною та доступна лише через Contribute. Файл підключення ключа можна надіслати користувачам електронною поштою або зберегти на комп’ютері. Коли його натиснути, автоматично відкривається Contribute, який імпортує з’єднання.
