## Що таке файл CRDOWNLOAD?

Файл CRDOWNLOAD — це частково завантажений файл, який створюється, коли ви завантажуєте файл з Інтернету за допомогою браузерів Microsoft Edge, Google Chrome або Chromium. Вміст файлу, який завантажується, зберігається в цьому файлі до завершення завантаження. Після завершення завантаження розширення .crdownload видаляється та замінюється оригінальним розширенням файлу, який завантажується.

## Формат файлу CRDOWNLOAD

Файл CRDOWNLOAD може бути як у текстовому, так і у двійковому форматі. Браузери створюють файл .crdownload за замовчуванням після початку завантаження. Якщо завантаження між ними переривається, файл CRDOWNLOAD залишається збереженим на диску. Іноді це дозволяє відновити завантаження з того місця, де воно було перервано.

### Приклад CRDOWNLOAD

Підтримка ви завантажуєте файл fileformatslist.doc з Інтернету. Коли завантаження починається, ваш браузер зберігає файл у папці завантажень як файл fileformatslist.crdownload. Його розмір файлу збільшується в міру завантаження файлу. Після завершення завантаження файл буде перейменовано на fileformatslist.doc.
