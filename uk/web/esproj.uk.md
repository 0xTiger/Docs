## Що таке файл ESPROJ?

Файл ESPROJ — це файл проекту, створений програмним забезпеченням для створення веб-сторінок Espresso. Він доступний у Mac OS і містить список пов’язаних файлів, які є частиною проекту веб-сайту. Коли ви відкриваєте файл ESPROJ в Espresso, він відкриває всі пов’язані файли в редакторі для редагування або оновлення вихідного коду. Файл ESPROJ також містить параметри публікації сервера та інші пов’язані параметри користувача.

## Формат файлу ESPROJ

Файли ESPROJ зберігаються як двійкові файли, і їх вміст не читається людиною. Каталог файлів ESPROJ також є кореневим каталогом веб-сайту.
