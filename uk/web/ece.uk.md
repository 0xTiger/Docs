## Що таке файл ECE?

Файл ECE – це динамічно створена веб-сторінка платформою керування вмістом **Escenic Content Engine (ECE)**. Він містить серверний код, який обробляється веб-сервером і подається назад у браузер користувача як сторінка [HTML](/uk/web/html/). Формат файлу ECE не дуже популярний порівняно з іншими форматами файлів на стороні сервера, такими як ASP і PHP. Ці файли відтворюються всередині та надаються кінцевим користувачам для відображення.

## Формат файлу ECE

Файли ECE не зберігаються на диску, а надаються безпосередньо користувачеві за запитом. Вони перетворюються на HTML і надсилаються назад у браузер користувача для відображення. Escenic Content Engine активно використовується для публікації сторінок новинних статей. Заснований на технології JEE та SQL, сервер ECE забезпечує надійне та масштабоване сховище для всього онлайн-контенту.

Клієнтські браузери можуть отримати доступ до вмісту на серверах ECE через HTTP/REST API. На додаток до обслуговування сторінок [HTML](/uk/web/html/), ECE забезпечує високоефективний робочий процес для керування та публікації відео/аудіовмісту.
