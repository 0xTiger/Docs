## Що таке файл MHT?

Файл із розширенням .mht — це формат архівованого файлу з підтримкою MIME, який містить різні типи даних в одному файлі. Він може зберігати такі дані, як текст, зображення, стилі сторінок у формі файлів [CSS](/uk/web/css/), JavaScript та інші ресурси як вбудовані ресурси. Файли MHT, що мають тип MIME message/rfc822, інкапсулюють увесь вміст файлу HTML як єдиний архівний файл для зберігання в архіві на запам’ятовуючих пристроях. Програмні програми, такі як Microsoft Word, дозволяють конвертувати ваші документи WORD у MHT шляхом експорту як файл MHT. Файли MHT можна відкривати за допомогою популярних браузерів, таких як Microsoft Internet Explore і Google Chrome.

## Формат файлу MHT

Як і [MHTML](/uk/web/mhtml/), MHT також є інкапсуляцією MIME сукупних документів HTML. Вміст документа всередині документа MHT, як-от вбудовані зображення, таблиці стилів, аплети тощо, закодовано за допомогою MIME, де вони пов’язані з кореневим ресурсом через URI. Поштові клієнти, такі як Microsoft Outlook, зберігають електронні листи (зазвичай [EML](/uk/email/eml/)) у форматі MHT. Повні специфікації формату файлів MHT доступні в [RFC 822](https://tools.ietf.org/html/rfc822), і їх можна використовувати для розробки програмного забезпечення, яке може обробляти файли MHT.

## Різниця між MHT і MHTML

Під час збереження онлайн-сторінки користувачеві пропонується вибрати формат файлу, у якому онлайн-сторінка має бути збережена у рідній системі. Найчастіше користувач плутається між мовою розмітки та форматами файлів MHTML. Вони помічають, що важко бачити, що формат файлу є здоровішим між цими.

Коли користувач вирішує не витрачати онлайн-сторінку як мову розмітки, тоді формується папка, яка містить кілька файлів для різного вмісту сторінки, тобто абсолютно різні одиниці площі файлів, створені для тексту, графіки, аплетів тощо. З іншого боку, збереження онлайн-сторінки, оскільки MHTML створює один файл для всієї онлайн-сторінки. Усі елементи сторінки разом із графікою, посиланнями та альтернативними файлами розташовані в одному файлі.

Природно, працювати з файлами MHT або MHTML просто, оскільки файл, який можна захищати та надавати спільний доступ просто через електронну пошту. Однак одиниця області файлів мови розмітки менша ймовірність втрати інформації, оскільки втрата або видалення навіть одного файлу може призвести до того, що вся папка мови розмітки стане марною для користувача.
