## Що таке файл FCGI?

Файл FCGI — це файл вихідного коду, який містить вихідний код, написаний для FastCGI (FCGI). Він може бути написаний кількома різними мовами програмування та містить розширення стандарту CGI, що використовується для більш надійної обробки запитів веб-сервера. Мови, які найчастіше підтримують FCGI, включають **Perl**, який підтримує як код Perl, так і потужний зовнішній інтерфейс API FastCGI.

## Формат файлу FCGI

Що робить файл FCGI важливим? Інтерфейс FCGI підтримує розподілене обчислення для обробки веб-запитів і розподілу їх між кількома серверами. Це робить файли FCGI переважно використовуваними для балансування навантаження на кількох серверах.
