## Що таке файл XHT?

Файл із розширенням .xht — це веб-файл, пов’язаний із типом файлу [XHTML](/uk/web/xhtml/) (розширена мова розмітки гіпертексту), який сам є файлом розмітки на основі [XML](/uk/web/xml/). Обидва ці файли схожі на HTML, але мають суворіший синтаксис, схожий на XML. Файли XHT розроблено таким чином, щоб бути більш структурованими, вимагати менше сценаріїв і є загальними на додаток до використання існуючих можливостей XML.

## Формат файлу XHT

Файл XHT створюється та зберігається як звичайний текстовий файл із кодуванням UTF-8 за замовчуванням. Він містить повний вихідний код документа XHTML, який можна відкривати та редагувати за допомогою будь-якого текстового редактора, який підтримує кодування Unicode. Крім текстових редакторів, формат файлу XHT зрозумілий більшістю сучасних веб-браузерів і може бути відкритий за допомогою них.

## Приклад XHT

У наступному прикладі документа XHT визначаються оголошення XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
