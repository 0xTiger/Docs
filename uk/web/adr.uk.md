## Що таке файл ADR?

Файл ADR (збережений із розширенням .adr) — це список закладок, створених веб-браузером Opera. Цей список заповнюється, коли користувач додає веб-URL-адреси в закладки. Ці закладки зберігаються у файлі ADR, який можна створити резервну копію та відновити за потреби. Оскільки більшість браузерів, як-от Chrome і Firefox, зберігають закладки у форматі [HTML](/uk/web/html/), їх можна конвертувати у формат файлу ADR для імпорту в браузер Opera. Файли ADR можна відкрити в браузері Opera.

## Формат файлу ADR – додаткова інформація

Деталі внутрішнього формату файлів ADR невідомі. Однак існують відмінності між форматами файлів ADR, експортованих різними версіями Opera. Це ускладнює [об’єднання файлів ADR](https://superuser.com/questions/471959/how-do-i-merge-several-opera-adr-bookmark-files-to-one-single-file) створення з різними версіями Opera. Деякі зовнішні інструменти можуть бути використані для їх об’єднання.
