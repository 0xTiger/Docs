## Що таке файл WBS?

Файл WBS — це файл проекту, створений WebBlender, який є програмою для створення презентацій і веб-сайтів для дітей. Файл WBS створюється під час запуску нового проекту в WebBlender. Усі функції сайту, додані до проекту, такі як сторінки, зображення, текст, анімація тощо, згадуються у файлі проекту WBS. Коли проект буде готовий, він публікує вміст сайту як файл [HTML](/uk/web/html/). WebBlender створено та підтримується [Tech4Learning](https://www.tech4learning.com/).
