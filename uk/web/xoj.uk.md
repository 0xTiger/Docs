## Що таке файл XOJ?

Файл XOJ — це файл блокнота, який містить нотатки, зроблені за допомогою програмного забезпечення для нотаток Xournal. Xournal — це безкоштовна мультиплатформена програма, яка дозволяє створювати нотатки з різними типами вмісту, як-от текст, зображення та рукописні дудли. Файли XOJ створюються, коли створюється новий файл блокнота за допомогою Xournal. Ці файли XOJ також можна надавати іншим користувачам для обміну даними. Новіша версія Xournal — [Xournal++](https://github.com/xournalpp/xournalpp), що є кросплатформним варіантом Xournal.

## Формат файлу XOJ

Файли XOJ мають формат XML і зберігаються на диск у стислому вигляді за допомогою стиснення [.gz](/uk/compression/gz/). Завдяки використанню універсального формату XML файли XOJ також можна відкривати та редагувати за допомогою кросплатформного варіанту Xournal++.

Кожен файл XOJ може містити ще одну сторінку блокнота Xournal.

## XOJ – підтримувані платформи

Xournal є мультиплатформенним, і його блокноти можна зберегти як файл XOJ у таких операційних системах.

* Linux
* MacOS
* Windows
* Android
* Збірка iOS
