## Що таке файл WEBSITE?

Файл WEBSITE – це файл ярлика веб-сайту, створений за допомогою Microsoft Internet Explorer (IE). Він містить URL-адресу та файл значка (ICO), який вказує на значок веб-сайту. Піктограма використовується для відображення ярлика. Формат файлу WEBSITE було представлено з випуском IE версії 9. Попередні версії IE використовувалися для збереження ярликів веб-сторінок у вигляді файлів [.url](/uk/web/url/).

## Формат файлу ВЕБ-САЙТУ

Файл .website зберігається на диску як звичайний текстовий файл, вміст якого можна переглянути в будь-якому текстовому редакторі. Окрім іншого вмісту, він містить цільову URL-адресу, для якої він є ярликом. Якщо ви хочете відкрити цільовий веб-сайт у веб-переглядачі, відмінному від IE, ви можете скопіювати цю URL-адресу та вставити її в інших браузерах, таких як Google Chrome, Safari або Firefox, щоб переглянути веб-сайт.

## Як створити файл .WEBSITE?

Файл WEBSITE можна створити в IE, клацнувши URL-адресу в адресному рядку та перетягнувши його в потрібне місце на комп’ютері, наприклад на робочий стіл або документи. Це створює ярлик до цього Інтернет-місцезнаходження, і його можна відкрити, двічі клацнувши ярлик. Якщо ви хочете зберегти ярлик як файл URL-адреси, утримуйте клавішу SHIFT, перш ніж перетягувати значок цільового сайту на робочий стіл.
