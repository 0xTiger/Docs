## Що таке файл SITE?

Файл SITE – це файл проекту, який використовується програмним забезпеченням для розробки веб-сайтів `Adobe GoLive`. Він зберігає інформацію про проект веб-сайту, таку як файли, папки, зображення, власні таблиці стилів і сценарії. Він також містить структуру веб-сайту та посилання, які використовуються на веб-сторінках сайту. Оскільки це файл проекту, він містить посилання на файли проекту, але не містить самих файлів.

Adobe GoLive припинено та більше не підтримується.

## Формат файлу SITE

Файли SITE зберігаються на диску у власному форматі файлу Adobe, а внутрішні відомості про його формат недоступні для довідки розробників.
