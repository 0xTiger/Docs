## Що таке СТИСНИЙ файл?

СТИСНИЙ файл – це стиснута веб-сторінка Wordpress невеликого розміру, створена за допомогою плагіна **WordCompress**. Він містить один рядок коду для представлення веб-сторінки, який створюється шляхом стиснення коду HTML, CSS і JavaScript плагіном. Ці конвертовані СТИСНЕНІ файли мають невеликий розмір, що сприяє швидшому завантаженню та передачі через Інтернет.

### СТИСНУТИЙ формат файлу - більше інформації

СТИСНЕНІ файли зберігаються як текстові файли, які можна відкрити в будь-якому текстовому редакторі, наприклад Microsoft Notepad, Notepad++ і Apple TextEdit. Окрім текстових редакторів, СТИСНУТИ файли можна відкривати за допомогою програмного забезпечення для створення веб-сторінок, наприклад Adobe Dreamweaver, доступного як для Windows, так і для Mac OS.

## Про плагін WordCompress

Плагін WordCompress спочатку використовувався для стиснення веб-сторінок WordPress у формат стисненого файлу. Плагін більше не підтримується та припинено. Він запише вихідний код веб-сторінки в один виконуваний файл, видаливши всі пробіли з вихідного коду сторінки. СТИСНЕНІ файли зберігаються разом із оригінальним вихідним кодом веб-сторінки, щоб зберегти оригінальні коментарі, відступи та інші пробіли, включені в оригінальний вихідний код сторінки.
