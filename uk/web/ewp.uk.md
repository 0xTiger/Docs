## Що таке файл EWP?

Файл EWP – це файл проекту, створений за допомогою програмного забезпечення [EWISoft](http://www.ewisoft.com/) Website Builder. Він пов’язаний із бінарним файлом бази даних, який містить усю інформацію про проект. Файл EWP зазвичай використовується для збереження прогресу веб-розробки. Це програмне забезпечення для веб-дизайну на основі шаблонів, яке дозволяє користувачам створювати веб-сайти з шаблонів веб-сайтів.

## Формат файлу EWP – додаткова інформація

Файли EWP – це файли проекту, які не потрібно компілювати чи розміщувати на будь-якому сервері. Вони зберігаються як звичайні текстові файли та містять посилання на прогрес веб-розробки. Після завершення розробки проекти можна опублікувати на веб-сайті.
