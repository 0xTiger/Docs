## Що таке файл GNE?

Файл GNE – це веб-сторінка, яка динамічно створюється та використовується веб-сайтом для обміну фотографіями Flickr. Сторінки розміщуються на сервері, звідки вони подаються до браузера користувача, який запитує. Файли GNE схожі на сторінки PHP і ASP, але розроблені на власній мові програмування, деталі якої недоступні для довідки розробника.

## Формат файлу GNE - Додаткова інформація

Файли GNE використовувалися комп’ютерною грою Game Never Ending, розробленою [Ludicorp](https://en.wikipedia.org/wiki/Ludicorp). Пізніше Flicker був придбаний yahoo! у березні 2005 року.
