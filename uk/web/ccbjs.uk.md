## Що таке файл CCBJS?

Файл CCBJS – це файл Javascript, створений редактором 3D-додатків, ігор і веб-сайтів **CopperCube**. Він містить код JavaScript, який створюється як частина [файлу .html](/uk/web/html/), створеного з файлу .ccb. CopperCube надає зручний інтерфейс сценаріїв для розробки логіки ігрового процесу та взаємодії без написання коду. Інші формати файлів, які підтримує CopperCube, включають .ccscene, .ccmodel, .ccmaterial, .cctex, .ccjs, .ccp і .ccpak.

## Формат файлу CCBJS

Деталі внутрішнього формату файлу CCBJS невідомі як такі. Однак файли Javascript зберігаються як звичайні текстові файли на диск і можуть бути відкриті за допомогою будь-якого текстового редактора, такого як Microsoft Notepad, Notepad++ і Apple TextEdit.
