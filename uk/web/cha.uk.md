## Що таке файл CHA?

Файл CHA – це файл налаштувань користувача, створений клієнтами IRC, такими як mIRC. Він містить такі параметри, як сервер, порт, назва каналу та, за бажанням, пароль для відкриття каналу IRC. Клієнти IRC використовуються для спілкування з іншими в мережах IRC по всьому світу. Вони використовуються для приватних обговорень один на один або під час багатогрупових конференцій. Коли клацнути файл CHA, клієнт IRC автоматично відкривається з деталями, згаданими у файлі.

Програми, які можуть відкривати файли CHA, включають mIRC і Visual IRC.

## Формат файлу CHA

Файли CHA зберігаються разом з іншими інсталяційними файлами клієнта IRC. Коли користувач хоче підключитися до сервера та приєднатися до каналу, він вводить інформацію в клієнт. Ця інформація зберігається у файлі CHA, який можна отримати в будь-який час.
