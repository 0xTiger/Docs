## Що таке файл FWP?

Файл із розширенням .fwp — це архівний файл, створений за допомогою Microsoft Expression Web і містить усі файли та папки для дизайну. Microsoft Expression Web — це загальний програмний продукт для веб-дизайну від Microsoft, заснований на редакторі [HTML](/uk/web/html/). Він був припинений з 2012 року та доступний для безкоштовного завантаження Microsoft. Файли FWP можна експортувати та зберігати у форматі Microsoft .cab.

## Формат файлу FWP – Додаткова інформація

Точні характеристики формату файлів FWP недоступні для довідки розробників. Однак це двійкові файли, і їх можна відкрити в Microsoft Expression Studio, яка давно припинена.
