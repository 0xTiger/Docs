## Що таке файл XHTM?

Файл із розширенням .xhtm — це файл веб-сторінки, написаний мовою XHTML. Він використовує XML 1.0 як стандартну реалізацію та вважається переформулюванням HTML 4. Подібно до XHTML, файл XHTM містить стандартні теги HTML, які відповідають суворішим інструкціям XML. Файли XHTM використовуються для публікації вмісту веб-сайтів і зазвичай використовують розширення [XHTML](/uk/web/xhtml/).

Ви можете використовувати такі веб-браузери, як Google Chrome, Apple Safari та Mozilla Firefox, щоб відкривати файли XHTM.

## Приклад XHTM

У наступному прикладі документа XHTM визначаються оголошення XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
