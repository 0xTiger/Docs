## Що таке файл H5?

H5 — це один із ієрархічних форматів даних (HDF), який використовується для зберігання великої кількості даних. Використовується для зберігання великої кількості даних у вигляді багатовимірних масивів. Цей формат в основному використовується для зберігання наукових даних, добре організованих для швидкого пошуку та аналізу. H5 був представлений як покращений формат файлу порівняно з H4. Спочатку він був розроблений Національним центром суперкомп’ютерних застосувань, а зараз підтримується The HDF Group.

## Коротка історія формату файлів HDF

Формат файлу HDF був обраний NASA як стандартний метод для зберігання наукових даних. До цього формулювання такого стандарту, розпочате в 1987 році робочою групою Graphics Foundations (GFTF) і HDF, було офіційно схвалено після дослідження 15 різних форматів файлів.

## Формат файлу HDF5

Файли H5 мають ієрархічний формат даних, який відповідає специфікаціям формату файлів HDF5. Ці специфікації визначають загальну структуру файлу HDF5 для зберігання на диску, але кінцевим користувачам ці базові деталі не потрібні.

Файли H5 зазвичай використовуються в аерокосмічній галузі, фізиці, інженерії, фінансах, академічних дослідженнях, геноміці, астрономії, електронних приладах і медицині.

### Модель даних HDF5

Файл HDF5 схожий на контейнер, який складається з різнорідних об’єктів даних. Наприклад, ці набори даних можуть бути зображеннями, таблицями, текстовими елементами, графіками та навіть такими документами, як [PDF](/uk/pdf/) і [документи Excel](/uk/spreadsheet/).

## Переглядач файлів H5

Група HDF надає [програмне забезпечення HDFView](https://www.hdfgroup.org/solutions/hdf5/), яке складається з утиліти HDFView і Java HDF Object Package.
