## Що таке файл XSMX?

Файл XSMX — це файл анкети, створений за допомогою програми ExamSoft SofTest, який використовується для проведення іспитів. Він використовується для надсилання екзаменаційних робіт студентам, які вони складають онлайн, і надсилання результатів у вигляді файлу XMDX. ExamSoft надає рішення для оцінювання на основі рішень драйверів даних за допомогою освітніх технологій, а програма SofTest, доступна як для Windows, так і для MacOS, є однією з них. Файл XSMX містить усі запитання та відкривається в програмі SofTest на комп’ютері студента.

## Формат файлу XSMX

Файли XSMX зберігаються та надсилаються учасникам іспиту як файли, захищені паролем. Захист паролем призначений для забезпечення безпеки та секретності екзаменаційних питань. Пароль іспиту надається кінцевому користувачеві для відкриття файлу та проходження тесту.
