## Що таке файл GPG?

Файл GPG — це файл ключа шифрування/дешифрування, який використовується програмою шифрування GNU Privacy Guard (GnuPG). Сама програма GnuPC базується на стандарті OpenPGP, визначеному RFC4880, і також відома як PGP. Запорукою успішного використання GPG у сучасній операційній системі є її універсальна система керування ключами. Утиліта командного рядка GPG дозволяє легко інтегрувати його з іншими програмами.

## Формат файлу GPG

Файли GPG зберігаються як зашифровані двійкові файли, і, звичайно, вони не читаються людиною. Щоб розшифрувати зашифрований файл GPG, потрібно використовувати той самий безпечний ключ. І тому внутрішній формат цих файлів невідомий.

## Шифруйте та розшифровуйте файли за допомогою GPG у Linux

Утиліту командного рядка GPG можна використовувати для шифрування та дешифрування файлів у Linux.

### Шифрування файлу

Файл можна зашифрувати за допомогою команди gpg із параметром -c (створити), як показано нижче.

```
gpg -c file1.txt
```
Виконання цієї команди запитує ключову фразу, за допомогою якої можна зашифрувати вміст вихідного файлу `file1.txt`. Це призводить до створення зашифрованого файлу file1.txt.gpg.

### Розшифровка та вилучення файлу

Щоб розшифрувати та видобути зашифрований файл, можна використати таку команду.

```
gpg cfile.txt.gpg
```
