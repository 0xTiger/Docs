## Що таке файл AML?

Файл AML (мова розмітки автоматизації) — це файл даних, який використовується для зберігання та обміну інформацією про техніку установки. Він заснований на форматі файлу XML, завдяки чому обмін інформацією базується на відкритих стандартах. AutomationML надає можливість поєднувати сучасні інженерні інструменти в галузі машинобудування, електричного проектування, програмованих логічних контролерів і керування роботами. Файли AML можна переглядати та редагувати за допомогою інструмента AutomationML Editor.

## Формат файлу AML

AML використовує [XML](/uk/web/xml/) об’єктно-орієнтоване моделювання даних для зберігання та обміну інженерними моделями. Компоненти машинобудівної установки представлені як об’єкти, що мають різні аспекти, причому кожен об’єкт має здатність діяти як підоб’єкти. Таким чином можна описати кожну меншу чи більшу модель. [Специфікації AutomationML](https://www.automationml.org/about-automationml/specifications/) містять короткий посібник для початку роботи з деталями цієї мови моделювання.

## Початок роботи з AML

Використання AML для обміну інженерною інформацією вимагає певних знань для успішного застосування. Для цього впровадження нових програм потребує кількох кроків, наведених нижче.

* Знати архітектуру
* Отримання базових знань
* Вивчення варіанту використання
* Відображення та моделювання для конкретної програми
* Реалізація
