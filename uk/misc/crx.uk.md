## Що таке файл CRX?

Формат файлу CRX пов’язаний із розширеннями браузера Google Chrome. Файл CRX — це, по суті, стиснений пакет, що містить необхідні файли та метадані для встановлення та запуску розширення в Google Chrome. Він покращує функціональність або зовнішній вигляд веб-браузера, надаючи додаткові функції або теми.

Коли файл CRX завантажується та встановлюється в Google Chrome, браузер перевіряє цілісність розширення за допомогою відкритого ключа та підпису. Якщо перевірка пройшла успішно, Chrome витягує вміст файлу CRX і встановлює розширення, роблячи його доступним для використання. Користувачі можуть керувати своїми розширеннями на сторінці розширень Chrome, яка дозволяє вмикати, вимикати або видаляти встановлені розширення.

## Як встановити файл CRX в Google Chrome?

Щоб установити файл CRX у Google Chrome, виконайте такі дії:

1. Відкрийте браузер Chrome.
2. Введіть `chrome://extensions` в адресний рядок і натисніть Enter.
3. Увімкніть перемикач "Режим розробника", розташований у верхньому правому куті сторінки розширень.
4. Натисніть на кнопку "Завантажити розпаковане".
5. Знайдіть і виберіть папку, що містить розпакований вміст файлу CRX (або просто виберіть сам файл CRX).
6. Натисніть "Відкрити", щоб інсталювати розширення.

## Що містить файл CRX?

Файл CRX містить необхідні файли та метадані, необхідні для розширення Google Chrome. Ось розбивка типового вмісту файлу CRX:

- **Файл маніфесту (manifest.json):** Цей файл є файлом у форматі JSON, який містить інформацію про розширення, наприклад назву, версію, опис, дозволи та фонові сценарії. Він визначає структуру та поведінку розширення.
- **Файли JavaScript:** Ці файли містять код, який визначає функціональність розширення. Вони можуть містити сценарії для обробки подій, зміни веб-сторінок або взаємодії з API Chrome.
- **HTML, CSS і файли зображень:** розширення часто включають елементи інтерфейсу користувача, такі як спливаючі вікна або сторінки параметрів. Файли HTML визначають структуру цих інтерфейсів, а файли CSS контролюють їх зовнішній вигляд. Файли зображень використовуються для піктограм або інших графічних ресурсів.
- **Додаткові файли ресурсів:** розширення можуть містити додаткові ресурси, такі як файли локалізації для підтримки кількох мов. Ці файли містять переклади тексту, який використовується в інтерфейсі користувача розширення.
- **Фонові сценарії:** якщо розширення має фонові процеси або сценарії, які виконуються незалежно від активної веб-сторінки, ці сценарії буде включено до файлу CRX.
- **Сценарії вмісту:** Сценарії вмісту – це сценарії, які можна вставляти на веб-сторінки, щоб змінити їх поведінку або взаємодіяти з їхнім вмістом. Якщо розширення використовує сценарії вмісту, необхідні файли для цих сценаріїв будуть присутні у файлі CRX.
- **Інші ресурси:** Залежно від конкретних вимог до розширення можуть бути включені додаткові файли, такі як аудіо- чи відеофайли, шрифти або файли даних.

Формат файлу CRX — це, по суті, стислий пакет, який містить усі ці файли та папки структурованим чином. Коли файл CRX інсталюється в Google Chrome, браузер витягує вміст і розміщує його у відповідних місцях, дозволяючи завантажувати та запускати розширення в браузері.

## Який формат файлу CRX?

Формат файлу CRX — це спеціальний формат для упаковки та розповсюдження розширень Google Chrome. По суті, це стислий ZIP-архів з іншим розширенням файлу. Основна структура файлу CRX така:

- **Підпис файлу:** Перші 4 байти файлу містять магічне число "Cr24" (шістнадцяткове: 43 72 32 34), яке служить підписом для ідентифікації файлу як файлу CRX.
- **Номер версії:** Наступні 4 байти представляють номер версії формату CRX.
- **Довжина відкритого ключа:** Наступні 4 байти вказують на довжину закодованого відкритого ключа, який використовується для перевірки підпису розширення.
- **Довжина підпису:** Наступні 4 байти вказують довжину підпису розширення.
- **Відкритий ключ:** Цей розділ містить закодований відкритий ключ, який використовується для перевірки цілісності розширення.
- **Підпис:** цей розділ містить підпис розширення, який створюється шляхом підписання вмісту розширення за допомогою закритого ключа, що відповідає відкритому ключу, згаданому вище.
- **ZIP-архів:** решта байтів CRX-файлу складається з стисненого ZIP-архіву. Цей архів містить усі необхідні файли та папки розширення, включаючи файл маніфесту, файли JavaScript, файли HTML, файли CSS, зображення та будь-які інші ресурси.
