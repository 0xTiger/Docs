## Що таке файл SSP?

Файл SSP — це файл проекту, створений за допомогою програмного забезпечення для створення заставок Windows, відомого як Screensaver Producer від Axialis Software. Він може містити кілька різних типів елементів заставки, таких як спрайти, флеш-фільми, слайд-шоу фотографій і відео. Файл SSP може містити текст, зображення, різні фігури та параметри анімації, які використовуються для відображення анімації у файлі SSP. Файли SSP можна відкрити за допомогою Axialis Screensaver Producer.

## Формат файлу SSP – Додаткова інформація

Файли SSP зберігаються на диску як двійкові файли без загальнодоступної інформації про структуру внутрішнього формату файлів. Професійний інструмент створення заставки Axialis Screensaver для Windows, який дає змогу легко створювати високоякісні заставки для всіх версій Windows до Windows 10. Він дозволяє створювати пакети заставок для встановлення, які можна легко розповсюджувати користувачам онлайн.

Файли SSP можна експортувати у формат [SCR](/uk/system/scr/).
