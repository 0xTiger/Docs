## Що таке файл OGG?

Файл OGG — це графічний файл, створений за допомогою програмного забезпечення Origin. Він надає можливість зберігати різні типи графіків для відображення імпортованих даних для аналізу даних. Типи графіків, які підтримуються типом файлу OGG, включають стовпець, смужку, кругову діаграму, водоспад, 3D, полярну та символьну. Програмне забезпечення Origin було розроблено OriginLab і дозволяє користувачам виконувати аналіз даних і генерувати вихідні дані у вигляді графіків.

## Формат файлу OGG

Файли OGG містять графіки для представлення імпортованих даних. Коли файл OGG відкривається, він відображає графік в окремому вікні в Origin або Origin Viewer, який є у вільному доступі порівняно з програмним забезпеченням Origin. Такі метадані, як заголовок і кольори графіка, також зберігаються у файлі OGG.

## Про програмне забезпечення OriginLab Origin

Програмне забезпечення Origin також надає можливість використовувати його як сервер автоматизації. У такому випадку клієнтські програми, наприклад LabVIEW, Microsoft Excel, Python, або спеціально розроблені програми на C#/VB/VC можуть обмінюватися даними з Origin. [Python_to_OriginLab](https://github.com/jsbangsund/python_to_originlab) — це API Python для автоматизації надсилання даних і основних функцій побудови в OriginLab.
