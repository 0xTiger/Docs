## Що таке файл XMZX?

Файл XMZX — це файл іспитового тесту для проведення тестів, який використовується програмним додатком ExamSoft SofTest. Додаток можна використовувати як на Windows, так і на macOS. Він містить запитання з іспиту, які екзаменатор може прийняти у студентів на комп’ютері з програмним забезпеченням SofTest. Файли XMZX надсилаються екзаменаторам для проведення тесту. Він завантажується в програмне забезпечення SofTest, і кандидат або абітурієнт проходить тест. Після завершення тесту остаточний файл XMDX автоматично завантажується на хмарні сервери ExamSoft.

## Формат файлу XMZX – додаткова інформація

Файли XMZX використовуються платформою цифрового оцінювання ExamSoft, яка використовується різними організаціями для проведення тестів для отримання сертифікатів. Платформа має власні хмарні онлайн-сервери, куди автоматично завантажуються всі виконані тести. Тести проводяться або в екзаменаційному центрі, або екзаменатори також можуть складати кандидати на власних комп’ютерах. ExamSoft доступний як для операційних систем Windows, так і для macOS. XSMX — ще один формат файлів екзаменаційних тестів від ExamSoft.

## Як відкрити файл XMZX?

Ви можете відкрити файл XMZX лише в програмному забезпеченні ExamSoft SofTest, оскільки це запатентований формат файлу. Програмне забезпечення SofTest має опцію **Скласти іспит**, яка використовується для початку іспиту шляхом вибору файлу XMZX.
