## Що таке файл H4?

H4 або HDF4 є одним із ієрархічних форматів даних (HDF) і використовується для зберігання великої кількості даних. Це формат файлу з відкритим кодом, який підтримує зберігання складних різнорідних даних. HDF4 є старішим форматом даних порівняно з [HDF5](/uk/misc/h5/). Формати файлів HDF підтримуються та публікуються некомерційною організацією HDF group. Файли HDF також підтримують зберігання метаданих для повного представлення збережених даних.

## Коротка історія формату файлів HDF

Формулювання ієрархічних форматів даних було розпочато в 1987 році Робочою групою графічних основ (GFTF). За цей час було запропоновано кілька різних форматів файлів для стандартизації єдиного формату. NASA вибрало HDF як стандартний метод зберігання та обслуговування наукових даних.

## Формат файлу HDF4

Файли H4 зберігаються у структурі, подібній до "каталогу файлів", відповідно до специфікацій формату файлу HDF. Ці специфікації визначають загальну структуру файлів HDF для зберігання на диску, але користувачі не хвилюються про базову файлову структуру.

Сфери застосування файлів H4 включають аерокосмічну галузь, фізику, інженерію, фінанси, академічні дослідження, геноміку, астрономію, електроніку та галузі медицини.

### Модель даних HDF4

Файл HDF4 або H4 схожий на контейнер, який складається з різнорідних об’єктів даних. Файли H4 підтримують такі набори даних:

* Зображення
* Таблиці
* Текстові елементи
* Графіки
* Такі документи, як [PDF](/uk/pdf/) і [документи Excel](/uk/spreadsheet/)

## Переглядач файлів H4

Група HDF надає [програмне забезпечення HDFView](https://www.hdfgroup.org/solutions/hdf5/), яке складається з утиліти HDFView і Java HDF Object Package.
