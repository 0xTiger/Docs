## Що таке файл BPS?

Файл .BPS – це, по суті, резервна копія документа текстового процесора, створеного за допомогою Microsoft Works, який зазвичай використовує розширення файлу .WPS для своїх документів. Ці файли .BPS часто автоматично генеруються Microsoft Works як запобіжний захід на випадок збою програми під час роботи над документом. Це спосіб захистити вашу роботу та запобігти втраті даних.

У наступному розділі ми вивчимо програмні додатки, пов’язані з файлами BPS.

## Microsoft Works

Microsoft Works — це пакет програмного забезпечення для підвищення продуктивності, розроблений Microsoft. Вперше він був представлений у 1987 році та набув популярності як більш доступна альтернатива повному пакету Microsoft Office. Microsoft Works включає кілька основних офісних програм, таких як:

- **Текстовий процесор:** Microsoft Works мала програму обробки текстів, яка дозволяла користувачам створювати та редагувати документи. Він використовував формат файлу [.WPS](/uk/word-processing/wps/) для текстових документів.

- **Електронна таблиця:** містить програму для створення електронних таблиць і керування ними. Формат файлу для електронних таблиць був [.XLR](/uk/spreadsheet/xlr/).

- **База даних:** Microsoft Works також мала програму бази даних для організації та керування даними. Формат файлів для файлів бази даних був [.WDB](/uk/database/wdb/).

- **Календар і адресна книга:** Окрім основних офісних програм, Microsoft Works часто включав інструменти для керування календарями та адресними книгами.

- **Формат текстового процесора Works (.WPS):** рідний формат файлу компонента текстового процесора Microsoft Works. Файли .WPS містили текст, форматування та просту графіку.

- **Припинення:** Microsoft офіційно припинила Microsoft Works у 2007 році та закликала користувачів перейти на Microsoft Office або інші альтернативи.
