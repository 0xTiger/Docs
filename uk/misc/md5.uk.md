## Що таке файл MD5?

Файл MD5 — це файл контрольної суми, який використовується для перевірки цілісності файлу. Він схожий на відбиток пов’язаного файлу та унікально генерується за допомогою алгоритму, який використовує кількість бітів у файлі. Він використовується для перевірки файлу за допомогою такого програмного забезпечення, як [md5sum](http://www.gnu.org/software/coreutils/manual/html_node/md5sum-invocation.html). Однією з переваг використання файлів MD5 є перевірка того, що завантажені файли не пошкоджені.

## Формат файлу MD5 – Додаткова інформація

Зазвичай файл MD5 зберігається з тим самим іменем, що й вихідний файл, але з розширенням .md5. Наприклад, якщо пов’язане ім’я файлу `abc_987_123456.grb`, відповідне ім’я файлу MD5 буде `abc_987_123456.grb.md5`. Файли MD5 допомагають визначити, що отриманий або завантажений файл не пошкоджений і не вражений шкідливим вмістом. Коротше кажучи, файли MD5 гарантують повноту файлу.


## Як перевірити контрольну суму MD5?

Один файл можна перевірити/підтвердити на наявність MD5 за допомогою інструмента [md5sum](https://en.wikipedia.org/wiki/Md5sum), як описано нижче.

```
md5sum -c abc_987_123456.grb.md5
```
