## Що таке файл NTF?
Файли з розширенням .ntf називаються файлами **National Transfer Format (NTF)**; здебільшого використовується UK Ordnance Survey (OS); спеціально для передачі геопросторових даних. Ним керує Британський інститут стандартів. Це стало стандартним форматом передачі цифрових даних Ordnance Survey. Проекція національної сітки Великобританії, яка є різновидом поперечної системи Меркатора, зберігає всю географічну інформацію для файлів NTF.

## Формат файлу NTF
Формат файлу NTF належить Ordnance Survey у Великобританії; підтримується бібліотекою GDB для імпорту. Поточна версія формату файлу NTF – 2.0. Цей формат файлу було введено для усунення обмежень векторного сегмента **PCIDSK**, який має лише одне поле атрибутів на структуру, але існує безліч атрибутів, які можна витягти з векторних даних. Щоб обійти формат файлу NTF, він має два сегменти. Кожен сегмент складається з однакових векторів, але з різними атрибутами. Перший сегмент векторного файлу NTF містить вектори з номером коду функції як атрибут. Другий сегмент містить вектори зі значенням як атрибут.

### Система координат
Бібліотека GDB представляє растрові та векторні дані з відповідними проекційними характеристиками TM:

- Довідкова довгота: 49,0
- Базова широта: 2,0
- False Easting: 400 000
- Помилковий північний: -100000
- Масштаб: 0,9996012717
- Еліпсоїд: Airy 1830 (E009)
Немає підтримки для оновлення чи запису файлів NTF, а також не можна зв’язати файли DTM.

### Приклад Ogrinfo
У наступному прикладі **ogrinfo** у файлі NTF використовується для отримання номерів шарів:
```
> ogrinfo llcontours.ntf
    INFO: Open of 'llcontours.ntf'
    using driver 'UK .NTF' successful.
    1: LANDLINE_POINT (Point)
    2: LANDLINE_LINE (Line String)
    3: LANDLINE_NAME (Point)
    4: FEATURE_CLASSES (None)
```
