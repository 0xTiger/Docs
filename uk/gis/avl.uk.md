## Що таке файл AVL?

Файл AVL — це файл легенди, який містить ключ для карти, створеної за допомогою програмного забезпечення ESRI ArcView GIS (Географічна інформаційна система). Він містить довідкову символічну інформацію, яка використовується у відповідній карті для доступу. Файл AVL може містити більше інформації, такої як символіка, налаштування відображення, наприклад прозорість, властивості відображення, залежні від масштабу, інформація про об’єднану таблицю/пов’язану таблицю, запит на визначення, властивості маркування для шарів ознак і властивості відображення анотацій для шарів анотацій залежно від типу. вид шару.

На файл AVL можна посилатися з файлу проекту ArcView [.APR](/gis/apr/).

## Формат файлу AVL – Додаткова інформація

Файли AVL зберігаються у форматі простого тексту. Це означає, що ви можете відкривати файли AVL у будь-якому текстовому редакторі, наприклад Notepad, Notepad++ і Apple TextEdit. Після відкриття ви можете не лише переглядати вміст файлу, але й змінювати його.

### Різниця між файлами .LYR і .AVL

 * **Відмінність формату файлу** - .lyr є двійковим файлом, тоді як .avl є текстовим файлом
 * **Відмінність вмісту** – файл .lyr містить більше інформації, ніж файл .avl. Файл AVL зберігає лише інформацію про символіку, тоді як файл LYR зберігає всю інформацію, доступну в діалоговому вікні властивостей шару в ArcMap.
