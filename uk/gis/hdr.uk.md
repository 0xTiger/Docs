## Що таке файл HDR?

Файл HDR — це файл заголовка ГІС, який містить інформацію заголовка для файлу Band interleaved by line (.BIL). Він описує дані зображення та має таке ж ім’я, як і файл зображення. Файл HDR складається з набору записів, кожна з яких описує певний атрибут зображення. Файл HDR описує макет і форматування файлу BIL для перекладу в координати реального світу в поєднанні з окремим файлом геоприв’язки.

## Формат файлу HDR

Файли HDR зберігаються у текстовому форматі ASCII. Він містить набір записів, де кожен запис описує певний атрибут зображення. Кожен файл HDR має такий формат:

```
<keyword> <value>
```

`<keyword> ` – вказує на конкретний атрибут, який встановлюється
`<value> ` – це значення, яке встановлюється атрибуту

Немає обмежень щодо порядку записів у заголовку. Однак кожен запис має бути в окремому рядку рядка, щоб відповідати методу аналізу, який використовується зчитувачами HDR.

## Підсумок ключових слів, використаних у файлі HDR

|Ключове слово |Прийнятне значення |За замовчуванням|
---|---|---|
|nrows |будь-яке ціле > 0| немає
|ncols |будь-яке ціле число > 0| немає
|nbands |будь-яке ціле число > 0| 1
|nbits |1, 4, 8, 16, 32| 8
|порядок байтів| I = Intel; M = Motorola |те саме, що хост-машина|
|макет| bil, bip, bsq| біл|
|скіпбайт| будь-яке ціле число ≥ 0| 0
|ulxmap |будь-яке дійсне число| 0|
|ulymap |будь-яке дійсне число |nrows - 1|
|xdim |будь-яке дійсне число| 1
|ydim |будь-яке дійсне число| 1
|bandrowbytes |будь-яке ціле число > 0| найменше ціле число ≥ (ncols x nbits) / 8|
|totalrowbytes |будь-яке ціле число > 0| для bil: nbands x bandrowbytes; для bip: найменше ціле число ≥ (ncols x nbands x nbits) / 8|
|bandgapbytes |будь-яке ціле число ≥ 0| 0|
