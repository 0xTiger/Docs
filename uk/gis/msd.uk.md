## Що таке файл MSD?

Файл MSD — це файл розповсюдження картографічного сервісу, створений за допомогою програмного забезпечення для картографування ГІС ArcGIS Desktop. Він містить опубліковану карту, експортовану з файлу карти [.mxd](/uk/gis/mxd/). Файл MSD містить інформацію про картографічний сервіс, розміщений на сервері ArcGIS. Розміщена карта доступна через Інтернет для інших. Файл MSD містить документ карти, інформацію про шар і властивості таблиці.

## Формат файлу MSD

Файли MSD зберігаються на диску у двійковому форматі. Вони використовуються серверною програмою ArcGIS для створення та керування картографічним сервісом для розміщення. Файли MSD також використовуються ArcMap для підключення до картографічного сервісу та відображення вмісту на карті.
