## Що таке файл GST?

Файл GST — це формат файлу карти ГІС, який містить географічні дані та візуальну карту нанесеної на карту території. Його використовують **MapX** (розроблений UNEP/GRID-Geneva та UNEP для керування геопросторовими даними про природні ресурси) та програмне забезпечення для картографування **MapXtreme** (набір інструментів для розробки програмного забезпечення, який використовується для створення спеціальних ГІС-додатків), які онлайн-платформи для управління геопросторовими даними про природні ресурси. Файл GST містить географічну інформацію у числовому вигляді, а також растрові зображення, які накладаються на карту.

## Формат файлу GST – Додаткова інформація

Файли GST зберігаються у власному форматі файлів, а деталі внутрішнього формату файлу недоступні для розробників.
