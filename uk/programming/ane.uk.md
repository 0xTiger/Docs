## Що таке файл ANE?

Файл ANE – це файл Adobe Air Native Extension, який використовується розробниками для написання спеціальних функцій, які викликають код [Adobe ActionScript](https://en.wikipedia.org/wiki/ActionScript). У більшості випадків файли ANE містять бібліотеку рідного коду, яку використовує розширення. Однак сам файл встановлений на пристрої. Він дозволяє виконувати певні системні функції, наприклад вібрацію пристрою. Програми Adobe AIR мають бути скомпільовані з використанням динамічного зв’язування для використання файлу ANE.

## Формат файлу ANE - Додаткова інформація

Файли ANE зберігаються на диску як двійкові файли, специфікації формату файлів яких недоступні для всіх. Файл ANE може містити власний вихідний код Android Java для доступу до функцій пристрою Android, таких як виклик функцій гучності або вібрації. Все це робиться шляхом обгортання рідної системи мовою ActionScript від Adobe.
