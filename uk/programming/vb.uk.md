## Що таке файл VB?

Файл VB — це файл вихідного коду, створений мовою Visual Basic, створений компанією Microsoft для розробки додатків .NET. Ще одна схожа мова з іншим синтаксисом – C#, файли якої зберігаються з розширенням [.CS](/uk/programming/cs/). Формат файлу надає мову програмування низького рівня для написання коду, який компілюється для створення кінцевого вихідного файлу у формі EXE або DLL. Їх можна створити та скомпілювати за допомогою Microsoft Visual Studio. Microsoft Visual Studio Express також можна використовувати для створення та оновлення таких файлів, що є безкоштовною IDE. Простий проект Visual Studio [рішення](/uk/programming/sln/), створений мовою VB, може складатися з одного або кількох таких файлів. Файли, позначені для включення в компіляцію, перераховані у файлі [CSPROJ](/uk/programming/csproj/), який є частиною проекту та вказує компілятору використовувати позначені файли.

## Формат файлу VB - Додаткова інформація

Файли VB — це текстові формати файлів, які можна відкрити в будь-якому текстовому редакторі для редагування. Однак, якщо відкрити в підтримуваній IDE з належним підсвічуванням синтаксису, код легко читати та впорядковувати. Простий файл VB містить:

* Декларація простору імен – для посилання на певну функціональність, визначену цим конкретним простором імен
* Оголошення змінних - для оголошення змінних рівня класу для конкретної реалізації
* Оголошення методів - для оголошення декларації методів для конкретної функції

## Приклад формату файлу VB

```
Imports System
Module Module1
   'This program will display Hello World
   Sub Main()
      Console.WriteLine("Hello World")
      Console.ReadKey()
   End Sub
End Module
```
