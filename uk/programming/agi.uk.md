## Що таке файл AGI?

Файл AGI — це файл сценарію, який використовується системою телефонії з відкритим кодом Asterisk. Він містить інформацію, яку можна використовувати для автоматизації процесів і плану номеру Asterisk. За допомогою файлів сценаріїв AGI можна встановити з’єднання з реляційними базами даних, такими як PostgreSQL або MySQL. Крім того, сценарії AGI також можна використовувати для доступу до інших зовнішніх ресурсів. Сценарії AGI можуть передавати контроль над планом телефонного зв’язку зовнішньому сценарію AGI, дозволяючи Asterisk виконувати складні завдання.

## Формат файлу AGI - Додаткова інформація

Файли сценаріїв AGI зберігаються як текстові файли та можуть бути відкриті в будь-якому текстовому редакторі для внесення змін.

### Стандартний шаблон зв'язку AGI

Сценарій AGI завантажує змінні та їхні значення, надіслані йому Asterisk. Загальний вигляд цих змінних виглядає наступним чином.

```
agi_request: test.py
agi_channel: Zap/1-1
agi_language: en
agi_callerid:
agi_context: default
agi_extension: 123
agi_priority: 2
```

Ці змінні супроводжуються порожнім рядком, надісланим Asterisk, який показує, що сценарій AGI тепер може керувати планом телефонного зв’язку.

### Мова програмування для файлів сценаріїв AGI

Сценарії AGI зазвичай можна писати на Perl, [PHP](/uk/programming/php/), [C](/uk/programming/c/), Pascal або Bourne Shell.
