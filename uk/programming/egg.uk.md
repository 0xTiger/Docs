## Що таке файл EGG?

Файл EGG, також відомий як яйця Python, є старішим форматом дистрибутивів Python. Це [ZIP](/uk/compression/zip/) стиснутий архів із розширенням .egg і містить вихідні файли програми Python разом із метаінформацією про дистрибутив. Файли EGG є альтернативою виконуваному файлу Windows [EXE](/uk/executable/exe/), але є кросплатформним. Цей старий формат для дистрибутивів Python було замінено новим форматом Wheel (WHL) на початку 2010 року.

## Формат файлу EGG

Файли EGG зберігаються як стиснені ZIP-архіви. Це означає, що якщо ви заміните розширення .egg на .zip, ви зможете відкрити його за допомогою стандартних утиліт декомпресії, таких як Corel WinZIP, Microsoft Explorer або RARLAB WinRAR.

Файли EGG можна створити за допомогою пакета **distutils**, доступного Python. Іншим інструментом, який може створювати та відкривати файли EGG, є **SetupTools**. Файли EGG можна встановити як пакет за допомогою **easy_install**.

`ПРИМІТКА. Формат файлу EGG застарів на користь нового формату файлу колеса WHL.`
