## Що таке файл CXX?

Файл CXX — це файл вихідного коду, подібний до C++, для написання програм. Він може містити повну реалізацію всієї програми або може містити часткову реалізацію програми, яку можна включити в проект разом з іншими файлами C++. Файли CXX можна відкрити в будь-якому текстовому редакторі та популярних IDE програмування, таких як Notepad++, Turbo C++ і Visual Studio,

## Формат файлу CXX

Файли CXX зберігаються як звичайний текстовий файл і містять код, написаний синтаксисом C++. Файли CXX також зазвичай зустрічаються з розширенням [.cpp](/uk/programming/cpp/). Деякі компілятори розглядають файли CXX як альтернативу .cpp.
