## Що таке файл INO?

**INO-файл** пов’язаний з **Arduino Sketch**, яка є програмою, написаною для плат Arduino з використанням **мови програмування Arduino**. Мова програмування Arduino заснована на Wiring; і він схожий на C/C++. Ескізи Arduino зберігаються з розширенням .ino та використовуються для керування платами Arduino.

## Формат файлу INO - Додаткова інформація

Платформа Arduino, призначена для створення електронних прототипів, дає розробникам можливість створювати та розгортати програми, які називаються ескізами, на спеціалізованих платах; ці ескізи варіюються від основних завдань, таких як підсвічування світлодіода на платі, до складних операцій, таких як передача даних через Інтернет; Створення ескізів Arduino відбувається в **інтегрованому середовищі розробки Arduino (IDE)**, і ці ескізи зберігаються з розширенням файлу .ino.

В епоху, що передувала Arduino 1.0, ескізи зберігалися з розширенням .PDE за допомогою Arduino IDE. Хоча IDE все ще підтримує відкриття файлів .PDE, рекомендовано зберігати всі ескізи, включно з тими, що спочатку зберігаються як файли [.PDE](/programming/pde/), із розширенням .ino. Ця зміна відображає еволюцію платформи та забезпечує сумісність із поточними стандартами.

## Про Arduino

Arduino — це електронна платформа з відкритим кодом, яка спрощує створення прототипів і розробку інтерактивних проектів; Arduino включає як апаратні, так і програмні елементи, що обслуговує різноманітну базу користувачів, від новачків до професіоналів. Ядром платформи є плати Arduino, оснащені цифровими та аналоговими контактами, USB-підключенням і різними моделями на основі процесорів, таких як Atmel AVR або ARM.

Arduino IDE служить зручним інтерфейсом для кодування, компіляції та завантаження програм (скетчів) на плати; мова програмування, схожа на C/C++, розроблена для доступності, абстрагування складності для початківців. Бібліотеки Arduino надають попередньо написані фрагменти коду для завдань, починаючи від інтеграції датчиків і закінчуючи протоколами зв’язку, сприяючи ефективній розробці.

Екрани Arduino, додаткові плати розширюють функціональні можливості, пропонуючи такі функції, як підключення до Ethernet або керування двигуном. Енергійне співтовариство Arduino робить внесок у розширену базу знань через форуми, навчальні посібники та спільні проекти; завдяки застосуванню, що охоплює робототехніку, домашню автоматизацію, художні інсталяції та сенсорні системи, універсальність і простота Arduino робить її кращим вибором для різноманітних електронних проектів.

## Як відкрити файл INO?

**Інтегроване середовище розробки Arduino** можна використовувати для відкриття файлу INO. Але оскільки це звичайні текстові файли, будь-який текстовий редактор можна використовувати для відкриття або редагування його вмісту.
