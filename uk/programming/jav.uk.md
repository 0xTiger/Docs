## Що таке файл Java?

Файл Java – це файл вихідного коду Java, створений мовою програмування Java. Він схожий на вихідний файл Java і може бути скомпільований за допомогою інтерпретаторів Java. Подібно до файлів [Java](/uk/programming/java/), файл Jav використовується для розробки програм Java. Це може бути один із багатьох файлів вихідного коду, доступних у проекті Java.

## Формат файлу Java

Файли Java зберігаються як текстові файли, і їх можна відкрити будь-яким текстовим редактором. Типи файлів Java були зазвичай замінені типом файлів Java і зазвичай використовуються для написання програм Java. Коди Java можна писати за допомогою IDE, як-от Eclipse та IntelliJ Idea, що полегшує роботу з файлами Java.

### Приклад формату файлу Java

```java
/*
The example code prints
Hello World from Java to the console.
*/
    public class JavaApp {
        public static void main(String[] args) {
            System.out.println("Hello World."); // Prints the string to the console.
        }
    }
```
