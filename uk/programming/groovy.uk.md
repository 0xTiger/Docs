## Що таке файл GROOVY?

Файл GROOVY — це файл вихідного коду, написаний мовою програмування Groovy. Код, написаний у файлі GROOVY, схожий на об’єктно-орієнтовану мову [Java](/uk/programming/java/), що полегшує проектування та розробку програм. Файл GROOVY скомпільований у байт-код віртуальної машини Java (JVM) і сумісний з іншим кодом і бібліотеками Java.

## Формат файлу GROOVY – більше інформації

Файли GROOVY містять вихідний код, написаний синтаксисом Groovy. Він може містити код, який можна використовувати як мову програмування, так і мову сценаріїв для платформи Java. Мова Groovy має функції, подібні до функцій Python, Ruby та Smalltalk.
