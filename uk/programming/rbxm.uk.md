## Що таке файл RBXM?

Файл RBXM — це формат файлу 3D-моделі, який використовується в Roblox Studio IDE для зберігання 3D-об’єктів багатокористувацької онлайн-гри Roblox. Він містить 3D-об’єкти, створені для гри за допомогою IDE. Приклади 3D-об’єктів у файлі RBXM включають сорочки, штани, капелюхи, будівлі, транспортні засоби, тварин, літаки, автомобілі або рослини. Файли RBMX можна конвертувати у [3D-формат OBJ](/uk/3d/obj/) за допомогою Roblox IDE. Roblox IDE можна використовувати для відкриття файлів RBMX.

Іншим популярним форматом файлу Roblox є [RBXL](/uk/programming/rbxl/), який є форматом файлу розташування та містить розташування, визначене у 3D-слові в грі Roblox.

## Формат файлу RBMX – Додаткова інформація

Файли RBXM створюються та зберігаються у двійковому форматі, і специфікації їхніх внутрішніх форматів файлів недоступні для всіх. Це відрізняється від формату файлу RBXMX, який базується на форматі файлу [XML](/uk/web/xml/). На Github є багато інструментів/репозиторіїв, які можна використовувати для [обробки файлів Roblox RBXM](https://github.com/search?q=rbxm).
