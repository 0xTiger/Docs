## Що таке 4-й файл?

Файл із розширенням .4th — це файл програмування, створений для **Четвертої мови програмування**. Він містить вихідний код для програм, написаних мовою програмування Forth, і генерує вихідні дані під час виконання програми. Це просто ще один файл мови програмування, схожий на вихідний файл [C](/uk/programming/c/) і [C++](/uk/programming/cpp/).

## 4-й формат файлу - більше інформації


### Приклад коду 4-ї мови програмування

Ось приклад простої програми, написаної мовою Forth, яка обчислює факториал заданого числа:

```
: factorial ( n -- n! )
   dup 0=
   if
      drop 1
      exit
   then
   1
   swap
   begin
      dup 1-
      dup 0=
      if
         drop
         exit
      then
      swap
      *
   repeat
;

```

У цьому прикладі факторіал приймає один аргумент n і повертає його факториал. Слово dup дублює значення на вершині стека, drop скидає значення на вершині стека, а * множить два значення на вершині стека. Конструкції if і begin/until керують потоком програми.

Щоб скористатися цією програмою, ви повинні ввести визначення в інтерпретатор Forth, а потім викликати факторіальне слово з числом, факторіал якого ви хочете знайти:

```
10 factorial .
```
Це призведе до результату 3628800, що є факторіалом 10.
