## Що таке файл CMAKE?

Файл CMAKE — це файл проекту, створений міжплатформним інструментом із відкритим кодом **CMake**. Він містить вихідний сценарій на мові CMake для всієї програми. Інструмент CMake використовує сценарії програмування, відомі як CMakeLists, для генерації файлів збірки для певного середовища, наприклад makefiles машини Unix. Сценарії та модулі, які використовуються проектом CMake, організовані як script.cmake і module.cmake, на які є посилання в проекті.
