## Що таке файл ASC?

Файл ASC — це файл **скрипту**, створений мовою програмування ActionScript. Він містить код, який виконується на сервері програмного забезпечення Flash Media Server. Файл ASC містить сценарії, які використовуються для керування такими подіями програми, як завантаження програм Flash, обробка дій користувача від подій миші та клавіатури, відтворення мультимедійного вмісту та візуалізація вихідних даних програми. Adobe придбала Macromedia Flash, який пізніше став частиною Adobe Creativity як Adobe Flash. Раніше він зберігав файли анімації у [форматі файлу .swf](мова опису сторінки/swf/).

## Формат файлу ASC - Додаткова інформація

Файли ASC зберігаються як текстові файли, які можна відкривати та редагувати за допомогою [Adobe Media Encoder](https://www.adobe.com/products/media-encoder.html). Вони створюються та зберігаються як текстові файли та можуть бути відкриті в будь-якому текстовому редакторі, наприклад Microsoft Notepad, Notepad++ і Apple TextEdit. Файл ASC виконується за замовчуванням під час запуску Flash Media Server і програми.
