## Що таке файл ALB?

Файл ALB — це файл бібліотеки, який використовується програмою IDE **Alpha Five** для розробки веб-програм. Його основна мета — зберігати інформацію для відповідного Alpha Five [файлу ADB](/uk/database/adb/). Це включає інформацію зі словника даних, таку як налаштування, покажчики сценаріїв, меню, визначені користувачем функції, растрові зображення, панелі інструментів та визначення імпорту.

Файли ALB можна відкривати за допомогою **Alpha Software Alpha Anywhere**.

## Формат файлу ALB - Додаткова інформація

Файли ALB зберігаються у власному форматі файлу Alpha File, і їх внутрішня структура файлового формату недоступна для всіх. Вони зберігаються на додаток до загальної бази даних бібліотеки розробників Alpha Five. Будучи бібліотеками, файли ALB можна повторно використовувати в інших програмах Alpha Five.
