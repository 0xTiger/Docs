## Що таке файл FS?

Розширення файлу **.fs ** пов’язане з **Visual F#**, мовою програмування, розробленою Microsoft як частину .NET framework. F# — це перша функціональна мова програмування, яка є частиною екосистеми Microsoft .NET і призначена для використання в різних типах додатків, включаючи веб-програми, програми для настільних комп’ютерів і хмарні програми.

Вихідні файли Visual F# (з розширенням «.fs») містять вихідний код, написаний **мовою програмування F#**. Ці файли можуть містити функції, модулі, типи та інші конструкції мови F#. Visual Studio — це звичайне середовище розробки для програмування F#, яке надає інструменти для редагування, компіляції та налагодження коду F#.

Програмісти використовують файли «.fs» для написання коду F#, а потім ці файли компілюються у виконувані файли ([.EXE](/executable/exe/)) або бібліотеки, зазвичай із розширенням [.DLL](/system/dll/). Мова F# відома своїм лаконічним синтаксисом, незмінністю за замовчуванням і сильною підтримкою концепцій функціонального програмування.
