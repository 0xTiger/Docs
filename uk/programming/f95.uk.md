## Що таке файл F95?

Файл із розширенням .f95 зазвичай означає, що це файл вихідного коду Fortran 95. Fortran — це універсальна імперативна мова програмування, яка особливо добре підходить для числових і наукових обчислень. Fortran 95 — це розширена версія Fortran 90, яка містить кілька нових функцій і вдосконалень.

Файли вихідного коду Fortran зазвичай містять програмні інструкції, написані мовою програмування Fortran; якщо у вас є файл із розширенням .f95, він, ймовірно, призначений для компіляції за допомогою компілятора Fortran; компіляція - це процес перекладу зрозумілого людині вихідного коду в зчитуваний машиною двійковий код, який може бути виконаний комп'ютером.

## Як відкрити файл F95?

Якщо ви хочете працювати або запускати програму у файлі .f95, вам знадобиться компілятор Fortran; популярні компілятори Fortran включають GNU Fortran (gfortran), Intel Fortran Compiler (ifort) та інші. Ви можете компілювати та запускати програми Fortran за допомогою відповідних команд, наданих компілятором.

Наприклад, якщо у вас встановлено gfortran, ви можете використовувати наступні команди в терміналі для компіляції та запуску програми Fortran:

```
# Compile the Fortran program
gfortran -o my_program my_program.f95

# Run the compiled program
./my_program
```

Якщо ви хочете лише переглянути вміст файлу F95, це звичайний текстовий файл, тому ви можете відкрити його за допомогою будь-якого текстового редактора.
