## Що таке файл ML?

Файл ML — це текстовий файл, створений за допомогою функціональної мови програмування загального призначення ML (часто відомої як мова мета). Він містить вихідний код ML, який частіше використовується в дослідженнях і проектуванні мов програмування з використанням повністю визначеної семантики. Існує кілька варіантів ML із широко використовуваним Caml. Інші включають Objective Caml (OCaml), JoCaml, Extended ML і MacroML. Мова ML зазвичай використовується для написання компіляторів, автоматизованого доведення теорем і формальних перевірок.
