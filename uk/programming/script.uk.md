## Що таке файл SCRIPT?

Файл SCRIPT — це вихідний файл, який містить сценарій на будь-якій мові сценаріїв. Це робить його узагальненим файлом сценаріїв, який містить змінні та функції. Оскільки кожна мова сценаріїв може відрізнятися, сценарій, написаний у файлі SCRIPT, потрібно виконувати за допомогою відповідного механізму створення сценаріїв.

## Формат файлу SCRIPT - Додаткова інформація

Файли SCRIPT зберігаються на диску у вигляді простих текстових файлів. Їх можна відкривати та редагувати за допомогою будь-якого текстового редактора, наприклад Microsoft Notepad, Notepad++ і Apple TextEdit. Оскільки мова сценаріїв може бути іншою, її можна виконати, перейменувавши розширення файлу на розширення за замовчуванням відповідної мови сценаріїв. Наприклад, існує кілька поширених мов сценаріїв, таких як JavaScript ([.JS](/uk/web/js/)), VBScript ([.VBS](/uk/programming/vbs/)), AppleScript (.SCPT), PHP ([ .PHP](/uk/programming/php/)) і ASP ([.ASP](/uk/web/asp/)).
