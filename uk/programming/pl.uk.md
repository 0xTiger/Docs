## Що таке файл PL?

Файл із розширенням .pl — це файл сценарію Perl, який є мовою сценаріїв. Вони скомпільовані та запускаються за допомогою програмного забезпечення Perl Interpreter. Файл сценарію PL містить рядки коду, змінні та коментарі. Мови сценаріїв порівняно складні
розуміти інші формати програмних файлів, наприклад [PHP](/uk/programming/php/). Файли PL можна створювати та сумісні з Windows, macOS та Linux.

## Коротка історія мови сценаріїв Perl

Ця мова вперше була використана в 1987 році, тому ці файли виникли в цьому році. У 1989 році був випущений Perl 2. Пізніше його було оновлено та модифіковано до версії 5.35, а наступну версію планується випустити наступного року.

У кожному оновленні було додано інструменти щодо функціональності, продуктивності та вигляду мови та файлів. За ці роки було багато переглядів версій. Спочатку це була базова мова сценаріїв, але тепер вона також містить багато інших модулів. Спочатку це була дуже проста мова, але скрипти цієї мови було досить важко зрозуміти, оскільки вони були компактними.

## Специфікації розширення формату файлу Perl

Є деякі властивості або специфікації цих файлів програмування, деякі з них такі:

* Код, включений у ці файли, має формат звичайного тексту та використовується для розробки сценаріїв
* Сценарії сервера, аналіз тексту та адміністрування сервера є основними аспектами, які охоплює сценарій цієї мови
* Багато популярних програм, пов’язаних із цією мовою, включають активний стан Active Perl і Bare Bones BBEdit (сумісні з Mac OS)
* Ця мова вважається мовою високого рівня
* Для Win32 користувачеві, можливо, доведеться встановити власний двійковий дистрибутив мови
* Потрібні деякі інструменти сценаріїв, щоб стати виконуваним у різних наборах ресурсів Windows
* Visual Studio .NET — відома структура для розробки мов програмування. Інструмент Active State, відомий як Visual Perl, використовується для додавання Perl у Visual Studio
* У файлах перший рядок вихідного коду описує інформацію про використовуваний інтерпретатор. Файли PL зазвичай починаються з рядка **#!/usr/bin/perl**, який повідомляє вашому комп’ютеру запустити цей сценарій за допомогою інтерпретатора Perl, встановленого на комп’ютері.


## Приклади сценаріїв PL

```
#!/usr/bin/perl
print "Hello, world\n";
```

Це буде надруковано

```
Hello, World
```

### Однорядковий коментар ###

```
#!/usr/bin/perl
# This is a single line comment
print "Hello Perl\n";
```

### Багаторядковий коментар ###

```
#!/usr/bin/perl
=begin comment
This is a multiline comment.
Line 1
Line 2
=cut
print "Hello Perl\n";
```

### Присвоєння змінної ###

```
#!/usr/bin/perl
$a = 10;
print "Variable a = $a\n";
```

### Присвоєння скалярної змінної ###

```
#!/usr/bin/perl
$age = 35; # Assigning an integer
$name = "Tony Stark"; # Assigning a string
$pi = 3.14; # Assigning a floating point
```

### Прості скалярні операції ###

```
#!/usr/bin/perl
$constr = "hi" . "perl";# Concatenates two or more strings.
$add = 40 + 10; # addition of two numbers.
$prod = 4 * 51;# multiplication of two numbers.
$connumstr = $constr . $add;# concatenation of string and number.
```
