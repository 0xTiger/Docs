## Що таке файл GMD?

Файл GMD – це вихідний файл розробки ігор, згенерований інструментом розробки програмного забезпечення Game Maker, який використовується для створення 2D-ігор. Файл GMD зберігається як текстовий файл і є файлом проекту для гри, яка буде розроблена. Game Maker доступний для Windows, Mac, Linux, Android, iOS, HTML5, Xbox, PlayStation і Nintendo Switch. Цей інструмент широко використовується незалежними розробниками, професійними студіями та викладачами по всьому світу для створення ігор.

## Формат файлу GMD

Файли GMD зберігаються у двійковому форматі файлу, а деталі внутрішнього формату файлу недоступні для довідки розробника.
