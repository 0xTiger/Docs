## Що таке файл ACD?

Файл ACD – це файл проекту, який містить інформацію для програмування платформ Logix [Rockwell Automation](https://www.rockwellautomation.com). Він містить інструкції для контролерів і процесорів. Він використовується для платформ Logix, таких як системи ControlLogix, CompactLogix, FlexLogix, SoftLogix5800 і DriveLogix.

Програми, які можуть відкривати файли ACD, включають Rockwell Automation RSLogix 5000.
