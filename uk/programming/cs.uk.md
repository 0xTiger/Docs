## Що таке файл CS?

Файли з розширенням .cs є файлами вихідного коду мови програмування C#. Представлений корпорацією Майкрософт для використання з .NET Framework, цей формат файлу надає мову програмування низького рівня для написання коду, який компілюється для створення кінцевого вихідного файлу у формі EXE або DLL. Їх можна створити та скомпілювати за допомогою Microsoft Visual Studio. Для створення та оновлення таких файлів також можна використовувати Microsoft Visual Studio Express, яка є безкоштовною IDE. Файли CS використовуються для розробки програм, які можуть варіюватися від простих настільних програм до більш складних програм. Простий проект Visual Studio [рішення](/uk/programming/sln/), створений мовою C#, може складатися з одного або кількох таких файлів. Файли, позначені для включення в компіляцію, перераховані у файлі [CSPROJ](/uk/programming/csproj/), який є частиною проекту та вказує компілятору використовувати позначені файли.

## Формат файлу CS ##

Файли CS — це текстові формати файлів, які можна відкрити в будь-якому текстовому редакторі для редагування. Однак, якщо відкрити в підтримуваній IDE з належним підсвічуванням синтаксису, код легко читати та впорядковувати. Простий файл CS містить:

* Декларація простору імен – для посилання на певну функціональність, визначену цим конкретним простором імен
* Оголошення змінних - для оголошення змінних рівня класу для конкретної реалізації
* Оголошення методів - для оголошення декларації методів для конкретної функції

### Синтаксис ###

* Крапка з комою використовується для позначення кінця висловлювання.
* Фігурні дужки використовуються для групування тверджень. Інструкції зазвичай групуються в методи (функції), методи в класи, а класи в простори імен.
* Змінні призначаються за допомогою знака рівності, але порівнюються за допомогою двох послідовних знаків рівності.
* Квадратні дужки використовуються з масивами як для їх оголошення, так і для отримання значення за заданим індексом в одному з них.
