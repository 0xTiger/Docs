## Що таке файл AS2PROJ?

Файл AS2PROJ — це файл проекту, створений у середовищі розробки Flash з відкритим кодом FlashDevelop. Він містить пов’язану з проектом інформацію, як-от параметри збірки, посилання на вихідні файли проекту та параметри для генерації вихідного [.swf-файлу](/uk/page-description-language/swf/). Кожен новий проект, створений у FlashDevelop, зберігається як файл AS2PROJ. Ви можете змінити параметри свого проекту, і всі ці зміни будуть збережені у файлі AS2PROJ.

Файли AS2PROJ можна відкрити за допомогою FlashDevelop.

## Формат файлу AS2PROJ

Файл AS2PROJ зберігається у форматі файлу [XML](/uk/web/xml/), який є універсальним форматом файлу для обміну інформацією. Його можна відкрити в будь-якому текстовому редакторі, наприклад Notepad, Notepad++ і Atom, щоб перевірити вміст.
