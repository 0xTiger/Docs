## Що таке файл NX?

Файл NX – це файл відеоігри, створений для використання з віртуальною ігровою консоллю **LowRes NX**. Він написаний на мові програмування BASIC і містить вихідний код гри, який визначає, як гра працює. Ви можете створювати власні файли NX для використання з програмою, які можна завантажити, і користувачі зможуть грати в гру.

Ви можете відкрити файл NX у текстовому редакторі LowRes NX або будь-якому редакторі вихідного коду, наприклад Microsoft Visual Studio Code.

## Формат файлу NX

Файл NX зберігається на диску як простий текстовий файл, який містить інструкції до гри. Кожен файл NX містить вихідний код гри, який складається з кількох записів ПЗУ картриджа. Ці записи ПЗУ містять інформацію про компоненти гри, наприклад графіку, карти рівнів, аудіо та інші дані, не пов’язані з вихідним кодом гри. Записи ПЗП програми можна додавати або редагувати за допомогою програм редагування NX, які постачаються в комплекті з LowRes NX.

## Як створити файли NX?

### Створення файлів NX на ПК

Ви можете створювати файли NX на ПК за допомогою:

* Текстові редактори, такі як Microsoft Notepad, Notepad++ або Github Atom
* Редактори вихідного коду, такі як Microsoft VIsual Studio Code

### Створення файлів NX на iOS

LowRes NX має версію для iOS, яку розробники можуть використовувати для створення файлів NX із цієї програми.
