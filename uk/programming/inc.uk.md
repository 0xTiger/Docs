## Що таке файл INC?

INC-файл — це включений файл, який використовується вихідним кодом програмного забезпечення для посилання на інформацію заголовків. Він схожий на [формат файлу .h](/uk/programming/h/) і містить оголошення, функції, заголовки або макроси, які можна повторно використовувати у файлах вихідного коду, таких як C++. Файли INC використовуються різними мовами програмування, такими як C, [C++](/uk/programming/cpp/), Pascal, [PHP](/uk/programming/php/) і [Java](/uk/programming/java/). Для відкриття файлів INC можна використовувати такі текстові редактори, як Microsoft Notepad, Notepad++ і TextEdit.

## Формат файлу INC - Додаткова інформація

Файл INC зберігається як звичайний текстовий файл із усіма оголошеннями, включеними відповідно до синтаксису оголошення. Один файл INC також може посилатися на інші файли INC. Після створення файл INC стає частиною проекту, і на нього можна посилатися з вихідних файлів, таких як C++. На нього можна посилатися в кількох вихідних файлах, щоб уникнути дублювання.

## Вміст файлу INC

Файли INC можуть містити таку інформацію.

**`Змінні`** – у випадку об’єктно-орієнтованого програмування (ООП) файл заголовка класу містить визначення всіх змінних рівня класу, які доступні у файлах вихідного коду реалізації.

**`Оголошення методів`** – усі оголошення методів включено у файли заголовків .h, щоб бути доступними для кількох файлів реалізації.

**`Означення невбудованих функцій`** - файли заголовків також можуть містити визначення невбудованих методів.

## Недолік використання файлу INC у PHP

PHP — це серверні сценарії, які виконуються на сервері та повертають результати на веб-сторінки, що викликають. Якщо файл PHP посилається на файл INC, а сервер не налаштовано для аналізу файлів .inc (що є дуже поширеним), користувач може переглянути вихідний код php у файлі .inc, відвідавши каталог URL. Отже, потрібно бути обережним, використовуючи файли INC як файли включення в PHP.
