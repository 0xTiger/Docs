## Що таке файл AS3PROJ?

Файл AS3PROJ — це файл проекту, створений FlashDevelop, безкоштовним IDE програмування з відкритим кодом для розробки Flash і веб-додатків. Він зберігає всю пов’язану з проектом інформацію та налаштування, застосовані користувачем за допомогою FlashDevelop IDE. Інформація, що зберігається у файлі AS3PROJ, включає параметри збірки, посилання на вихідні файли проекту та параметри для генерації вихідного [.swf-файлу](/uk/page-description-language/swf/). Коли створюється новий проект, він зберігається як файл AS3PROJ у тому самому місці, яке вказав користувач для збереження пов’язаних із проектом даних.

## Формат файлу AS3PROJ

Файл AS3PROJ зберігається у форматі файлу [XML](/uk/web/xml/), який є універсальним форматом файлу для обміну інформацією. Його можна відкрити в будь-якому текстовому редакторі, наприклад Notepad, Notepad++ і Atom, щоб перевірити вміст.
