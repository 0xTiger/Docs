## Що таке файл PHP?
Файл із розширенням .php відноситься до мови програмування з відкритим вихідним кодом, яка використовується для написання серверних сценаріїв, які будуть виконуватися на веб-сервері. Це найпоширеніша мова веб-сценаріїв, яка зазвичай використовується для розробки великомасштабних веб-додатків. Найбільша система ведення блогів в Інтернеті, тобто WordPress, і найбільша соціальна мережа Facebook засновані на мові PHP. Він сумісний з різними платформами, такими як Windows, Linux, Unix, Mac OS X тощо. Крім того, він успішно працює майже на всіх серверах, які використовуються сьогодні, наприклад Apache, IIS тощо. PHP підтримує широкий спектр баз даних, включаючи MySQL.

## Формат файлу PHP ##

Один PHP може містити текст, CSS, [HTML](/uk/web/html/), JavaScript і PHP-код. PHP-код виконується на сервері, а результат повертається в браузер як звичайний HTML. Файли PHP можна відкривати за допомогою будь-якого текстового редактора та редагувати на місці, хоча такі програми, як Adobe Dreamweaver, Eclipse, інструменти розробки PHP пропонують зручний спосіб писати та змінювати код PHP.

### Синтаксис ###

Сценарій PHP виконується на сервері, а звичайний результат HTML надсилається назад у браузер. Скрипт PHP починається з<?php and ends with ?>

### Приклад PHP ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
