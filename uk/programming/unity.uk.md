## Що таке файл UNITY?

Файл Unity – це файл 3D-сцени, створений за допомогою механізму 3D Gaming, **Unity**. Він використовується в поєднанні з іншими сценами для створення віртуального світу під час редагування. Він містить елементи тривимірної сцени, такі як об’єкти, середовища, декорації, камери, інформацію про ракурси, сценарії освітлення та перешкоди.

Файли Unity можна відкривати за допомогою Unity Technologies Unity. Unity також може читати Unity [.fbx](/uk/3d/fbx/), [.dae](/uk/3d/dae/) (Collada), [.3ds](/uk/3d/3ds/), [.dxf](/uk/cad/dxf/) і файли [.obj](/uk/3d/obj/).

## Формат файлу UNITY

Файли Unity зберігаються на диску у двійковому форматі. Вони генеруються та використовуються разом з іншими сценами під час редагування. Unity використовується для розробки ігор для Windows, а також для Mac OS X, і може використовуватися для створення 3D-ігор для Інтернету, Nintendo Wii, iPhone або автономних програм.
