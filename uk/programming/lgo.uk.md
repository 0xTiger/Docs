## Що таке файл LGO?

Файл LGO — програмний код, написаний за допомогою програми Logo для малювання зображень. Він містить запрограмовані інструкції, які керують програмою читання для створення креслень відповідно до цих інструкцій. Наприклад, програма «черепаха», яка може відкривати файли LGO, скористайтеся цими інструкціями, щоб намалювати лінії відповідно до запрограмованого коду. Малюнок не обмежується лише лініями. Файли LGO можна використовувати для малювання простих і складних конструкцій. Багато освітніх програм використовували файли LGO для навчання програмному створенню креслень.

## Формат файлу LGO

Файли LGO зберігаються у власному форматі файлів, і їх внутрішня файлова структура невідома. Іншим добре відомим форматом файлів для логотипів є файли [.ico](/uk/image/ico/).
