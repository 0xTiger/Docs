## Що таке файл PY? ##

Файли з розширенням .py містять вихідний код Python. Мова Python сьогодні стала дуже відомою мовою. Його можна використовувати для створення системних сценаріїв, веб-розробки та розробки програмного забезпечення та математики. Python підтримує кросплатформну сумісність; означає, що розроблені програми на Python можуть працювати на різних платформах, таких як Windows, MAC, Linux, Raspberry Pi тощо. Python забезпечує простий і легкий для читання синтаксис, подібний до англійської мови. Розробники можуть написати прийнятне програмне забезпечення, написавши кілька рядків коду Python. Оскільки Python працює на системі інтерпретатора, код можна виконати відразу після написання, що робить його дуже хорошим для прототипування.

## Коротка історія ##

Python був задуманий наприкінці 1980-х років Гвідо ван Россумом як наступник мови програмування ABC. Його реалізація почалася в грудні 1989 року з Ван Россумом як єдиним провідним розробником. Він працював над python до 12 липня 2018 року. У січні 2019 року активні розробники ядра Python обрали керівну раду з п’яти членів у складі Ніка Коглана, Бретта Кеннона, Керол Віллінг, Баррі Варшау та Ван Россума для керівництва проектом.

## Як запустити файл py ##

Щоб перевірити встановлену версію Python, ви можете скористатися такою командою:

```cmd
python --version
```

Це виведе версію на консоль як

```cmd
Python 3.7.4
```

Якщо Python не встановлено на вашій машині, ви можете перейти на [python.org](https://www.python.org/) і завантажити та встановити Python для вашої відповідної операційної системи.

Щоб виконати сценарій Python, ви можете використати таку команду:

```cmd
python helloworld.py
```

*helloworld.py* — це файл сценарію, який містить наведений нижче код

```py
print("Hello World from Python")
```

У вікні консолі буде надруковано наступне.

```cmd
Hello World from Python
```

Примітка. Якщо ви використовуєте IDE, вони надають кнопки на екрані або різні комбінації клавіш для запуску Python. Наприклад, кнопка відтворення показана в канаві з редактором у PyCharm, який дозволяє виконувати сценарій Python.

## Формат файлу PY ##

Python був розроблений для читабельності та має схожість з англійською та математикою. Python використовує нові рядки для позначення повної команди на відміну від крапки з комою чи круглих дужок, які використовуються в інших мовах. Для областей, циклів і функцій Python покладається на відступи та пробіли на відміну від фігурних дужок, які використовуються в інших мовах.

### Синтаксис ###

Нижче наведено приклад синтаксису Python.

```py
print("Hello World")

#Variables
name = "John"
age = 25
print(name)
print(age)

#While Loop
i = 1
while i < 3:
  print(i)
  i += 1
  
#For Loop
names = ["John", "Harry", "Tom"]
for name in names:
  print(name)
```
