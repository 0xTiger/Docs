## Що таке файл OLM?

Файл із розширенням .olm — це файл Microsoft Outlook для операційної системи Mac. Файл OLM зберігає повідомлення електронної пошти, журнали, дані календаря та інші типи даних програми. Вони схожі на файли [PST](/uk/email/pst/), які використовуються Outlook в операційній системі Windows. Однак файли OLM, створені в Outlook для Mac, не можна відкрити в Outlook для Windows. Формат файлу OLM було введено в Outlook 2011 для зберігання локальних даних Outlook. Один файл OLM може містити кілька поштових скриньок, упорядковуючи дані у формі папок і вкладених папок. Файли OLM можна відкривати/завантажувати в Microsoft Outlook 365 і Outlook для Mac.

## Формат файлу OLM

OLM — це двійкові файли з типом MIME Application/octet-stream. Технічні специфікації формату файлу OLM не доступні відкрито від корпорації Майкрософт, і немає жодних деталей формату, доступних для довідки розробників. Однак існують деякі API, які можуть читати формат файлу OLM і записувати його у файл PST для використання в операційній системі Microsoft Windows, наприклад Aspose.Email.
