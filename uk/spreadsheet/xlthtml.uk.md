## Що таке файл XLTHTML?

Файл із розширенням .xlthtml — це [HTML](/uk/web/html/) версія файлів шаблонів [XLT](/uk/spreadsheet/xlt/), які спочатку створювалися за допомогою Microsoft Excel. Цей формат більше не підтримується Excel і його замінено експортом даних у формати файлів [MHT](/uk/web/mht/) і [MHTML](/uk/web/mhtml/). Основна ідея файлу XLTHTML полягала в тому, щоб дозволити користувачам переглядати файли шаблонів XLT у веб-браузері замість того, щоб мати Excel, встановлений у системі. Файли, експортовані у форматі XLTHTML, можна відкривати в браузерах.

## Формат файлу XLTHTML – додаткова інформація

Файли XLTHTML використовували формат файлу HTML для збереження шаблонів XLT для відкриття у веб-браузері. Сам XLT базується на двійковому форматі файлу Excel, який був представлений у версії Excel 97-2003. Ці файли можна використовувати як початкові файли з визначеним форматуванням для створення звичайних документів для заповнення даних електронних таблиць. Просто для довідки, XLT базується на двійковому [специфікації формату файлу XLS](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx), який задокументовано Microsoft для довідка розробника.
