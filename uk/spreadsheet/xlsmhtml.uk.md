## Що таке файл XLSMHTML?

Файл із розширенням .xlsmhtml — це [HTML](/uk/web/html/) версія оригінального файлу XLS, який експортується з Microsoft Excel для перегляду даних у браузері. Формат уже давно припинений Excel і більше не підтримується. Експортований файл XLSMHTML відповідав формату файлу XLS, і його можна було відкривати та переглядати у веб-браузерах. Excel забезпечує експорт даних електронної таблиці у формати файлів [MHT](/uk/web/mht/) і [MHTML](/uk/web/mhtml/), які зараз широко використовуються.

## Формат файлу XLSMHTML – більше інформації

Файли XLSMHTML були створені з кодом HTML для представлення даних електронної таблиці у форматі, подібному до XLS. HTML-код у файлі XLSMHTML організовано таким чином, що вихідний файл виглядає схожим на вихідний файл, коли його відкривають у веб-браузері. Специфікації формату файлу XLSMHTML як такі не оприлюднюються Excel, і розробникам не доступні такі посилання для розуміння та розробки додатків для нього.
