## Що таке файл XLA?

Файл із розширенням .xla — це формат файлу надбудов Microsoft Excel, який додає модулі, додаткові функції та інші інструменти до Microsoft Excel. Ad-Ins надають додаткові функції для електронних таблиць Excel і підтримуються у версії 2007 або новішій. Файли XLA мають тип носія mime application/vnd.ms-excel. Їх можна відкрити за допомогою програми Microsoft Excel, яка доступна як для Windows, так і для MacOS.

## Формат файлу XLA

Файли XLA базуються на двійковому форматі даних. Однак додаткова інформація про специфікації його формату файлу недоступна. Загальне припущення полягає в тому, що XLA відповідає [[MS-XLS] – структура формату двійкового файлу Excel](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx).
