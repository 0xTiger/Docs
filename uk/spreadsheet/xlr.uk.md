## Що таке файл XLR?

Файл із розширенням .xlr — це формат файлу електронної таблиці Microsoft Works, який зберігає дані в рядках і стовпцях. Microsoft Works був розроблений для використання як початковий пакет продуктивності та залишався функціональним з 1987 по 2009 рік. Він включав текстовий процесор, електронну таблицю та систему керування базами даних. Пізніше його було припинено після представлення Microsoft Office із багатшими функціями.

## Формат файлу XLR - більше інформації

Файли XLR зберігалися як двійкові файли без загальнодоступних відомостей про специфікації формату файлів. Таким чином, розробники не мають можливості аналізувати структуру файлів і програмно писати програми для читання/запису в ці файли. Електронна таблиця Works використовує той самий формат файлу Excel для зберігання даних у файлах .xlr.
