## Що таке файл MPP?

Файл із розширенням .mpp — це файл даних Microsoft Project, який інтегровано зберігає інформацію, пов’язану з керуванням проектом. Корпорація Майкрософт розробила цей власний формат файлу, щоб зробити його сумісним із програмним продуктом для керування проектами Microsoft Project (MSP). Крім MPP, MSP може працювати з XML-схемою проекту. Багато програм і API дозволяють конвертувати MPP в інші формати файлів. У Microsoft тепер є онлайн-сервер Project Server, на який кілька користувачів можуть завантажувати файли керування проектами для спільної роботи.

## Формат файлу MPP ##

Корпорація Майкрософт розробила та опублікувала MPP як двійковий формат файлу, і на сьогодні його специфікації не були оприлюднені корпорацією Майкрософт. Таким чином, внутрішня файлова структура та деталі недоступні для розробників. В Інтернеті є кілька API, які підтримують читання/запис у файли MPP, але 100% точність ніколи не досягається. Корпорація Майкрософт з часом представила кілька функцій у своїй програмі для керування проектами, а також функції формату файлу MPP.

Файл MPP використовує такі типи MIME:

* додаток/vnd.ms-проект
* додаток/msproj
* додаток/msproject
* додаток/x-msproject
* додаток/x-ms-проект
* додаток/x-dos_ms_project
* додаток/mpp
* zz-додаток/zz-winassoc-mpp

## Проблеми з відкриттям файлу MPP? ##

Ось список деяких поширених проблем, які можуть виникнути та спричинити неправильну роботу формату MPP:

* Відсутність супровідного програмного забезпечення
* Пошкоджений файл
* Файл, інфікований вірусом
* Немає права доступу в системі для відкриття файлів
* Застарілий диск у вашій системі
* Розширення файлу перейменовано
