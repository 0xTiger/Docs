## Що таке файл MPT? ##

Файли з розширенням .mpt — це файли шаблонів [Microsoft Project](https://products.office.com/en-us/project/project-and-portfolio-management-software). Тут міститься основна інформація, структура та налаштування документа для створення файлів [MPP](/uk/project-management/mpp/). Ці шаблони надають параметри за замовчуванням, які є важливими для конкретного проекту, наприклад графіки або інформацію про бюджет. Хоча він не може зберігати дані про проект, такі як завдання, ресурси чи призначення. Після зміни файли шаблону можна зберегти як стандартні MPP-файли для подальшої роботи з ними. Файл головного шаблону під назвою "Global.mpt" доступний у Microsoft Project, відкритий для створення окремого шаблону. Файли MPT використовують *application/vnd.ms-project* і *application/x-project* як стандартні типи MIME. Windows, Linux і macOS можуть відкривати файли MPT разом із Moos Project Viewer, Microsoft Project і MPXJ.

## Проблеми з відкриттям файлу MPT? ##

Ось список деяких поширених проблем, які можуть виникнути та спричинити неправильну роботу формату MPT:

* Відсутність допоміжного програмного забезпечення
* Пошкоджений файл
* Файл, заражений вірусом
* Немає права доступу в системі для відкриття файлів
* Застарілий диск у вашій системі
 *   Extension of the file is renamed
