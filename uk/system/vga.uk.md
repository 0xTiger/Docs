## Що таке файл VGA?

Файл VGA — це файл диска Windows для підтримки моніторів VGA. Він використовувався в ранніх версіях Microsoft Windows для налаштування моніторів VGA на рівні операційної системи (ОС). Файл VGA, який дозволяє користувачам регулювати роздільну здатність і глибину кольору дисплея за допомогою панелі керування монітора.

## Формат файлу VGA

Внутрішній формат файлу VGA невідомий для довідки розробника. На нього посилається операційна система Windows для налаштування візуального вигляду монітора VGA.
