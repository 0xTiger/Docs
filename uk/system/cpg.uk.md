## Що таке файл CPG?

Файл CPG – це необов’язковий файл, який потребує шейп-файлу ESRI, який використовується для визначення кодової сторінки для визначення набору символів, який буде використовуватися. Вони зберігаються у форматі звичайного текстового файлу та містять інформацію про кодування, застосоване для створення шейп-файлу. Якщо файл CPG недоступний, шейп-файл використовує системне кодування за замовчуванням. Файл CPG, якщо він є, повинен мати той самий префікс, що й файл [SHP](/uk/gis/shp/), наприклад, roads.shp, roads.cpg.

Файли CPG можна відкрити за допомогою ESRI ArcGIS Pro.

### Формат файлу CPG - Додаткова інформація

Під час перегляду шейп-файлу в ArcCatalog або будь-якій іншій програмі ArcGIS ви бачите лише шейп-файл. Але насправді шейп-файл використовує інші пов’язані файли, які зчитуються разом із основним шейп-файлом. Файл CPG також є одним із них, якщо він присутній поряд з основним файлом SHP.
