## Що таке файл PUZ?

Файл із розширенням .puz — це стиснутий архів, який містить стандартний документ Microsoft Publisher [.pub](/uk/publisher/pub/). У результаті стиснення він менший за розміром і його легко передавати через Інтернет або інші пристрої. Файли PUZ можна розпакувати/відкрити за допомогою утиліти Microsoft Unpack.

## Формат файлу PUZ

Файл PUZ — це стиснутий архів, який містить стандартний файл PUB разом з іншими файлами. Він містить усі необхідні файли для розпакування, відображення та друку публікації.

Файл PUZ містить наступне.

* `Unpack.exe` - утиліта для розпакування стиснених файлів
* `ReadMe.txt` - опис файлів
* `Стиснутий файл .puz` – містить публікацію та може містити вбудовані шрифти та пов’язану графіку.

### Розпакування файлу PUZ

Упаковані файли Publisher можна розпакувати за допомогою утиліти Unpack.exe. Він розпаковує запакований файл PUZ у каталог, вибраний користувачем.
