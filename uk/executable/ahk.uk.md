## Що таке файл AHK?

Файл AHK — це файл сценарію, створений за допомогою програми AutoHotkey, яка використовується для автоматизації завдань у Microsoft Windows. Він містить інструкції щодо автоматизації завдань за допомогою комбінацій клавіш, які можна використовувати для виконання миттєвих дій. Файл виконується за допомогою AutoHotkey і всі дії виконуються послідовно. Файли AHK достатньо потужні, щоб виконувати складні завдання за допомогою гарячих клавіш, визначених за допомогою гарячих клавіш. Файли AHK можна відкривати за допомогою текстових редакторів, таких як Microsoft Notepad і Notepad++.

## Формат файлу AHK

Файли AHK зберігаються на диску як звичайні текстові файли та містять рядки коду, які можна виконати за допомогою AutoHotkey. AutoHotkey — це безкоштовна мова сценаріїв із відкритим вихідним кодом, яка дозволяє користувачам створювати від простих до складних сценаріїв для виконання таких дій, як заповнення форм, автоматичне клацання, виконання макросів тощо. Файл AHK може виконувати як мінімум одну дію, а потім вийти .

Існують інструменти, які можна використовувати для перетворення AHK у файли [Exe](/uk/executable/exe/).

### Приклад AHK

У наступному прикладі використовується клавіша Win+Z, щоб запустити Microsoft Notepad або вивести його на передній план, якщо він уже запущено.

```
#z::Run https://www.autohotkey.com  ; Win+Z

^!n::  ; Ctrl+Alt+N
if WinExist("Untitled - Notepad")
    WinActivate
else
    Run Notepad
return
```

## Як створити файл AHK?

Наступні кроки можна використати для створення файлу AHK. Вони припускають, що AutoHotkey уже встановлено на машині.

* Клацніть правою кнопкою миші на робочому столі.
* Знайдіть у меню «Новий».
* Натисніть «Сценарій AutoHotkey» у меню «Новий».
* Дайте сценарію нову назву.
* Знайдіть щойно створений файл на робочому столі та клацніть його правою кнопкою миші.
* Натисніть «Редагувати сценарій».
* Мало з’явитися вікно, ймовірно, Блокнот.
* Збережіть файл.
