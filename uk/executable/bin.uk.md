## Що таке файл BIN?

Файл BIN — це виконуваний файл в операційних системах на базі Unix, таких як Linux або FreeBSD. Він містить скомпільовану програму, що складається з двійкового коду, отриманого з вихідного коду, що робить її сумісною з основною архітектурою системи. Виконувані BIN-файли Unix можна порівняти з файлами Windows .EXE і файлами macOS .APP з точки зору їх ролі як виконуваних форматів.

У сфері розробки програмного забезпечення для систем Unix файли BIN зазвичай використовуються розробниками для упаковки та розповсюдження програм. Вони пропонують зручні засоби доставки програмного забезпечення користувачам, що дозволяє легко встановити та виконати. Окрім файлів BIN, існують інші типи виконуваних файлів Unix, зокрема [.ELF](/uk/executable/elf/), .X86, [.RUN](/uk/executable/run/) і .X86_64, кожен з яких адаптований до певного обладнання або Системні вимоги.

Що стосується внутрішньої структури файлу BIN, то він складається із закодованих даних, які операційна система Unix використовує для ідентифікації, читання та виконання програми, що міститься в ньому. Система покладається на конкретні підписи файлів або заголовки, щоб розпізнати BIN-файл як виконуваний файл, з подальшою інтерпретацією та виконанням бінарних інструкцій, що містяться в ньому.

Файли BIN часто постачаються разом із супровідним файлом **INSTALL.TXT**, який містить докладні інструкції щодо правильного встановлення та налаштування програми. Ця документація гарантує, що користувачі зможуть успішно пройти процес встановлення та почати використовувати програмне забезпечення без ускладнень.

## Як відкрити файл BIN?

Файли BIN не призначені для безпосереднього відкриття та перегляду. Однак ви можете виконувати програми або видобувати дані, які вони містять. Щоб отримати доступ до вмісту BIN-файлу, виконайте такі дії в командному рядку, припускаючи, що ім’я BIN-файлу "sample.bin":

1. **Забезпечте дозволи для виконуваних файлів:**

Перш ніж запускати файл BIN, переконайтеся, що він має необхідні дозволи для запуску як виконуваного файлу. Використовуйте команду:

```
$ chmod +x sample.bin
```

Ця команда надає файлу права на виконання.

2. **Виконайте файл BIN:**

Після встановлення дозволів для виконання ви можете запустити файл BIN, ввівши таку команду:

```
$ ./sample.bin
```

**УВАГА**

_Будьте обережні, коли маєте справу із завантаженими або надісланими електронною поштою файлами BIN, оскільки вони потенційно можуть бути використані кіберзлочинцями для розповсюдження зловмисного програмного забезпечення. Запускайте файли BIN лише з надійних джерел, щоб знизити ризик атак зловмисних виконуваних файлів._
