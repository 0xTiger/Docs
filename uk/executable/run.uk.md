## Що таке файл RUN?

Формат файлу .run зазвичай використовується для розповсюдження інсталяторів програмного забезпечення або програм у середовищі Linux. Щоб інсталювати програмне забезпечення, вам потрібно буде зробити файл виконуваним, що можна зробити за допомогою такої команди:

```bash
chmod +x file_name.run 
```

Потім ви можете запустити файл за допомогою такої команди:

```bash
./file_name.run 
```

Процес інсталяції може відрізнятися залежно від конкретного файлу та програми, яку ви намагаєтесь інсталювати.

Формат файлу .run — це тип сценарію оболонки, який використовується для розповсюдження інсталяторів програмного забезпечення або програм у середовищі Linux. Це самодостатній пакет, який містить усе необхідне для встановлення програмного забезпечення, включаючи двійкові файли, бібліотеки та файли конфігурації.

Важливо зауважити, що файли .run також можуть містити зловмисний код, тому перед запуском завжди варто перевірити джерело файлу та перевірити його на наявність вірусів.

Крім того, для запуску та встановлення деяких файлів .run можуть знадобитися права root, тому вам може знадобитися використати команду "sudo", щоб запустити файл із підвищеними правами:

```bash
sudo ./filename.run
```

## Як відкрити файл RUN?

Щоб відкрити файл .run, вам потрібно зробити його виконуваним, що можна зробити за допомогою команди "chmod":

```bash
chmod +x filename.run 
```

Коли файл стане виконуваним, ви можете запустити його, ввівши:

```bash
./filename.run
```

Запуск файлу .run — це не те саме, що відкриття його в текстовому редакторі. Запуск файлу .run виконає його інструкції, які можуть бути будь-якими: від встановлення програми до запуску сценарію. Щоб переглянути вміст файлу .run, його потрібно відкрити в текстовому редакторі, наприклад nano або vim:

```
nano filename.run
```
або
```
vim filename.run
```
