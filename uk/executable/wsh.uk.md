## Що таке файл WSH?
Файл із розширенням .wsh містить властивості та параметри для сценарію певної мови програмування, наприклад VB або VBS тощо. Справжня потреба WSH — використовувати їх для налаштування виконання певних сценаріїв. Для роботи потрібен WScript або CScript, і обидва вони входять до складу операційної системи Windows. Файли WSH спочатку надавалися в Windows 95 на інсталяційних дисках як додатковий настроюваний і інсталюваний для панелі керування, а потім як компонент за замовчуванням у Windows 98.

## Формат файлу WSH
WSH (хост сценаріїв Windows) можна використовувати для різних цілей, включаючи адміністрування, сценарії входу та загальну автоматизацію. WSH створює середовище для виконання сценаріїв. він викликає відповідний движок сценаріїв і виділяє набір служб і об’єктів для роботи сценарію. Ці сценарії можна запускати в режимі GUI, з об’єкта COM або в режимі командного рядка, забезпечуючи користувачеві гнучкість для інтерактивних і неінтерактивних сценаріїв.

### Приклади
Ось дуже простий приклад, який демонструє деякий VBScript, який використовує кореневий COM-об’єкт WSH «WScript» для відображення повідомлення з кнопкою «OK». Коли цей сценарій буде запущено, буде викликано механізм CScript або WScript і буде встановлено середовище виконання.

```
WScript.Echo "Hello world"
WScript.Quit
```
