## Що таке файл PFM?

Файл із розширенням .pfm — це файл метрики шрифту принтера, який містить розміри символів, включені у файл шрифту. Необхідно мати файл PFM разом із пов’язаним файлом .pfb(/uk/font/pfb/) у тій же папці під час інсталяції пізніше. Файли PFM закодовані у двійковому коді та не читаються людиною.
