## Що таке файл JFPROJ?

Файл із розширенням .jfproj — це спеціальний шрифт на основі JSON, створений за допомогою програми Microsoft Font Maker у Windows 10. Його можна створити за допомогою електронної ручки з пристроями з сенсорним екраном, наприклад планшетами, хоча також можна використовувати інші комп’ютерні периферійні пристрої, наприклад мишу, комп'ютерів. Файли JFPROJ містять гліфи, які використовуються для відтворення символів на екрані шляхом визначення інтервалів між символами та словами та розміру шрифту. Програма Microsoft Font Maker доступна для завантаження в магазині Windows.
