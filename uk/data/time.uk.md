## Що таке файл TIME?

Формат файлу TIME використовувався веб-системою під назвою LIGHT, яка допомагала користувачам записувати, упорядковувати та ділитися своїм життям. По суті, він зберігав колекцію дописів і являв собою папку на сторінці життя користувача в системі.

## Як дізнатися, коли файл було створено в Linux

У Linux ви можете використовувати команду `stat`, щоб дізнатися, коли було створено файл. Ось як ви можете це зробити:

Відкрийте термінал і введіть таку команду:

```
stat <file_path>
```

Замінити `<file_path> ` із шляхом до файлу, який вас цікавить. Наприклад:

```
stat /path/to/your/file.txt
```

Ця команда покаже різну інформацію про файл, включаючи час його створення. Знайдіть рядок, який починається з «Народження» або «Файл:». Час народження вказує, коли файл було створено у файловій системі.

Ось приклад того, як може виглядати результат:

```
  File: /path/to/your/file.txt
  Size: 1024       	Blocks: 8          IO Block: 4096   regular file
Device: 801h/2049d	Inode: 1643318     Links: 1
Access: (0644/-rw-r--r--)  Uid: ( 1000/   user)   Gid: ( 1000/   user)
Access: 2024-01-31 12:34:56.789012345 +0000
Modify: 2024-01-31 12:34:56.789012345 +0000
Change: 2024-01-31 12:34:56.789012345 +0000
 Birth: 2024-01-31 12:34:56.789012345 +0000
```

У цьому прикладі час народження вказує, коли було створено файл.
