## מהו קובץ CFG?

קובץ תצורה של Celestia הוא קובץ טקסט רגיל המשמש את Celestia, תוכנית הדמיית היקום התלת-ממדית. קבצים אלה חיוניים להתאמה אישית של האופן שבו Celestia מתנהג, אילו אובייקטים שמימיים מוצגים ואיך התוכנית מופיעה. עם זאת, עריכת קבצים אלה דורשת תשומת לב זהירה מכיוון ששינויים לא מתאימים עלולים לשבש את תהליך הטעינה של Celestia, ולמנוע ממנו לפעול כהלכה.

## סלסטיה

Celestia היא תוכנת הדמיית אסטרונומיה תלת מימדית חינמית בקוד פתוח, המאפשרת למשתמשים לחקור ולהמחיש את היקום בצורה מציאותית ואינטראקטיבית ביותר. הוא פותח על ידי כריס לורל והוצא לראשונה בתחילת שנות ה-2000. Celestia זמין עבור מערכות ההפעלה Windows, macOS ו-Linux.

תכונות והיבטים מרכזיים של Celestia כוללים:

- **הדמיה תלת מימדית מציאותית:** Celestia מספקת ייצוג מפורט ומדויק של מערכת השמש שלנו, כמו גם כוכבים, גלקסיות וחפצים שמימיים אחרים. הוא משתמש במודלים ובמרקמים תלת מימדיים באיכות גבוהה כדי ליצור חוויה סוחפת ויזואלית.

- **מסד נתונים שמימיים נרחב:** התוכנה מגיעה עם מסד נתונים מובנה עצום של עצמים שמימיים ידועים, כולל כוכבים, כוכבי לכת, ירחים, אסטרואידים, שביטים וחלליות. משתמשים יכולים לאתר ולחקור בקלות אובייקטים אלה.

- **דיוק מדעי:** בעוד ש-Celestia הוא כלי הדמיה, הוא שואף לייצג אובייקטים ותופעות שמימיים בצורה מדויקת ככל האפשר, בהתבסס על נתונים מדעיים זמינים.

## פורמטי קבצים בשימוש על ידי Celestia

Celestia משתמשת בפורמטים שונים של קבצים להיבטים שונים של הדמיית האסטרונומיה התלת-ממדית שלה. להלן כמה מפורמטי הקבצים העיקריים המופעלים על ידי Celestia:

1. **קובצי תצורה (.cel)**
- *תיאור*: קובצי טקסט פשוטים המאפשרים למשתמשים להתאים אישית את ההתנהגות, המראה והתוכן של Celestia.
- *מטרה*: התאמה אישית של הגדרות, הגדרת מיקומים וציון עצמים שמימיים.

2. **דגמים ורשתות תלת מימד**
- *פורמטים*: [.3ds](/he/3d/3ds/), [.obj](/he/3d/obj/), [.dae](/he/3d/dae/), .ac
- *תיאור*: פורמטים נתמכים של מודלים תלת-ממדיים המשמשים לעיבוד גרמי שמיים וחלליות.
- *מטרה*: הצגת אובייקטים תלת מימדיים בתוך Celestia.

3. **קבצי מרקם**
- *פורמטים*: [.jpg](/he/image/jpeg/), [.png](/he/image/png/), [.dds](/he/image/dds/)
- *תיאור*: קבצים אלה מספקים טקסטורות פני השטח לגרמי שמיים.
- *מטרה*: מיפוי טקסטורות על גבי מודלים תלת מימדיים למראה מציאותי.

4. **כוכבי קטלוגים וקבצי נתונים**
- *פורמטים*: פורמטים מותאמים אישית, [.csv](/he/spreadsheet/csv/), [.tsv](/he/spreadsheet/tsv/)
- *תיאור*: קבצי נתונים המשמשים לייצוג כוכבים וחפצים שמימיים אחרים, המבטיחים הדמיה מדויקת.
- *מטרה*: ייצוג מדויק של עצמים שמימיים.

5. **מפות קוביית מרקם**
- *תיאור*: מפות קוביות משמשות כדי לדמות מראה של עצמים שמימיים מרוחקים כמו גלקסיות.
- *מטרה*: עיבוד אובייקטים מרוחקים עם טקסטורות מציאותיות.

6. **קבצי סקריפט**
- *תיאור*: קבצים אלה מכילים סקריפטים מותאמים אישית שנכתבו בשפת הסקריפט של Celestia.
- *מטרה*: לאפשר למשתמשים ליצור אירועים ואנימציות דינמיות בתוך היקום של Celestia.

## קובץ תצורה של Celestia

הנה סקירה כללית בסיסית של מה שאתה יכול לעשות עם קובץ התצורה של Celestia:

1. **הגדרת המיקום שלך**: אתה יכול לציין את מיקומך על פני כדור הארץ באמצעות פרמטרים של קווי רוחב, קו אורך וגובה. זה מאפשר ל- Celestia להציג במדויק את שמי הלילה מהמיקום שלך.

```
Location "My Location"
{
    Latitude 40.7128
    Longitude -74.0060
    Altitude 0
}
```

2. **התאמה אישית של אפשרויות הצפייה שלך:** תוכל להתאים אפשרויות צפייה שונות כגון שדה ראייה, הגדרות זמן והעדפות רינדור.

```
Viewpoint
{
    Location "My Location"
    Follow "Earth"
    Eye [0.0 0.0 0.0]
    Center [0.0 0.0 0.0]
    Up [0.0 1.0 0.0]
    Fov 45
}

Time
{
    Date "2023-09-25T12:00:00.000Z"
    Clock "Now"
}

Rendering
{
    Atmosphere false
    Stars 7
    Planetshine 0.25
}

```

3. **טעינת אובייקטים שמימיים:** ניתן להוסיף לסימולציה אובייקטים שמימיים כגון כוכבים, כוכבי לכת, אסטרואידים, חלליות ועוד. כל אובייקט מוגדר בקובץ תצורה עם המאפיינים שלו.

```
Star "Sun"
{
    RA 0
    Dec 0
    Distance 0
    AppMag -26.74
    SpectralType "G2V"
}

Planet "Earth"
{
    Parent "Sol"
    Texture "earth.jpg"
    Radius 6371
    EllipticalOrbit
    {
        Period 365.25
        SemiMajorAxis 149.6e6
        Eccentricity 0.017
        Inclination 0
        AscendingNode 0
        ArgOfPericenter 102.94
        MeanAnomaly 100.464
}
}
```

4. **הגדרת חלליות:** אתה יכול ליצור חלליות בדיוניות משלך או להשתמש בחלליות אמיתיות על ידי ציון הפרמטרים שלהן כמו מיקום, כיוון ומודלים תלת מימדיים.

```
Spacecraft "Voyager 1"
{
    Parent "Sol"
    Class "spacecraft"
    Mesh "voyager.3ds"
    Radius 0.01
    EllipticalOrbit
    {
        Period 30700
        SemiMajorAxis 1.08e11
        Eccentricity 0.044
        Inclination 3.4
        AscendingNode 49.0
        ArgOfPericenter 44.0
        MeanAnomaly 35.0
}
}
```

5. **יצירת סקריפטים:** ניתן לכתוב סקריפטים בשפת הסקריפטים המותאמת אישית של Celestia כדי ליצור אירועים ואנימציות דינמיות.
