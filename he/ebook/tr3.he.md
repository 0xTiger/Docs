## מהו קובץ TR3? ##

קובץ TR3 הוא ספר אלקטרוני של TomeRaider 3 המספק תמיכה בחיפוש ודחיסה מהירים. זה יכול לתפקד כראוי דרך תוכנית הקשורה ל-TomeRaider. Yadabyte היא המפתחת של תוכנה זו, חברת תוכנה ופיתוח אתרים בריטית שבסיסה בבריטניה. ניתן להשתמש באפליקציית קורא הספרים האלקטרוניים של TomeRaider לתפקוד תקין ולניהול התוכן של קבצי TR3 אלה. קורא ספרים אלקטרוני קשור זה יכול להיות מותקן במחשבים הפועלים על מערכות מבוססות Microsoft Windows וגאדג'טים רבים הניתנים להעברה. קובץ TR3 שייך לקבוצת קבצי eBook המשמשים במערכות תפעוליות כגון Windows 10, Windows 7, Windows 8/8.1, Windows Vista, Windows XP.
