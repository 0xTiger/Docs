## מהו קובץ REG?

קובץ REG הוא פורמט קובץ המשמש לייבוא או ייצוא של נתוני הרישום של Windows. רישום Windows הוא מסד נתונים היררכי המאחסן הגדרות תצורה ואפשרויות עבור מערכת ההפעלה Windows ויישומים מותקנים. הרישום מכיל מידע כגון העדפות משתמש, הגדרות אפליקציה, נתוני תצורת חומרה ותוכנה ועוד.

לקובץ REG יש בדרך כלל סיומת קובץ ".reg" והוא קובץ טקסט רגיל המכיל סדרה של ערכי רישום וערכי בפורמט מסוים. הפורמט מורכב מקטע כותרת שמזהה את הקובץ כקובץ רישום, ואחריו סדרה של צמדי מפתח וערכים המייצגים ערכי רישום.

## פורמט קובץ REG - מידע נוסף

הנה דוגמה לפורמט קובץ reg:

```
[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Run]
"Example"="C:\\Example.exe"

[HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced]
"Hidden"=dword:00000001
```

השורה הראשונה של הקובץ מציינת את הגרסה של עורך הרישום. השורות הבאות מכילות את ערכי הרישום בפורמט של נתיב מפתח ואחריו שם ערך ונתוני ערך. בדוגמה זו, קובץ reg מכיל שני ערכים: אחד שמוסיף תוכנית בשם "Example.exe" לרשימת האתחול של Windows, ואחר שמגדיר את הערך "Hidden" בהגדרות המתקדמות של סייר Windows ל"true".

ניתן ליצור ולערוך קבצי Reg באמצעות עורך טקסט כגון פנקס רשימות. הם משמשים לעתים קרובות למטרות גיבוי ושחזור, כמו גם להגדרת מספר מחשבים עם אותן הגדרות רישום.

## ייבוא או ייצוא של רישום Windows

קובץ REG הוא סוג של קובץ המשמש לייבוא או ייצוא של נתונים מהרישום של Windows. רישום Windows הוא מסד נתונים היררכי המאחסן הגדרות תצורה ואפשרויות עבור מערכת ההפעלה Windows ויישומים מותקנים. הרישום מכיל מידע כגון העדפות משתמש, הגדרות אפליקציה, נתוני תצורת חומרה ותוכנה ועוד.

ניתן להשתמש בקובצי Reg ליצירה או שינוי של ערכי רישום, ולעתים קרובות הם משמשים למטרות גיבוי ושחזור, כמו גם להגדרת מספר מחשבים עם אותן הגדרות רישום. הנה איך להשתמש בקובץ reg כדי להוסיף ערך רישום חדש לרישום של Windows:

1. צור קובץ טקסט חדש באמצעות עורך טקסט כגון פנקס רשימות.
2. הקלד את ערך הרישום בפורמט הנכון. הפורמט מורכב מקטע כותרת שמזהה את הקובץ כקובץ רישום, ואחריו סדרה של צמדי מפתח וערכים המייצגים ערכי רישום. הנה דוגמה:

```
Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\Software\Example]
"Setting1"="Value1"
```

פעולה זו יוצרת מפתח חדש בשם "דוגמה" מתחת למפתח "תוכנה" בכוורת הרישום של המשתמש הנוכחי, ומגדירה את הערך "Setting1" ל-"Value1".

3. שמור את הקובץ עם סיומת קובץ .reg.
4. לחץ פעמיים על קובץ ה-.reg כדי להוסיף את ערך הרישום החדש לרישום של Windows.

תתבקש לאשר שברצונך להוסיף את הערך לרישום. לאחר שתאשר, הערך החדש יתווסף לרישום, ותוכל לאמת אותו באמצעות כלי עורך הרישום.
