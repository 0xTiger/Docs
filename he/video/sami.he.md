## מהו קובץ SAMI?

קובץ עם הסיומת ".sami" מתייחס בדרך כלל לקובץ Subtitles And Metadata Interchange (SAMI). SAMI הוא פורמט כתוביות המשמש להצגת כתוביות או כתוביות בסרטונים. זה פותח במקור על ידי מיקרוסופט עבור Windows Media Player שלהם.

קובצי SAMI מכילים מידע תזמון ותוכן טקסט עבור כתוביות או כיתובים, המאפשרים לסנכרן אותם עם הפעלת וידאו. הפורמט תומך באפשרויות עיצוב בסיסיות כגון סגנון גופן, צבע ומיקום הכתוביות על המסך.

קבצי SAMI הם בדרך כלל קבצי טקסט רגיל וניתן לפתוח ולערוך אותם באמצעות עורך טקסט. התוכן של קובץ SAMI כולל בדרך כלל קטע כותרת המספק מידע על הקובץ, ואחריו רשומות כתוביות בודדות עם התזמון והטקסט שלהם.

להלן דוגמה כיצד עשוי להיראות קובץ SAMI:

```
<SAMI>
<HEAD>
<TITLE>Example Subtitles</TITLE>
</HEAD>
<BODY>
<SYNC Start=100><P Class=ENCC>Subtitle 1</P></SYNC>
<SYNC Start=500><P Class=ENCC>Subtitle 2</P></SYNC>
<SYNC Start=1000><P Class=ENCC>Subtitle 3</P></SYNC>
</BODY>
</SAMI>
```

קבצי SAMI משמשים בדרך כלל בשילוב עם נגני וידאו או נגני מדיה התומכים בהצגת כתוביות, כגון Windows Media Player או VLC Media Player. הנגן קורא את קובץ ה-SAMI ומסנכרן את הכתוביות עם תוכן הווידאו, ומאפשר לצופים לקרוא את הכתוביות תוך כדי צפייה בסרטון.

## תגי HTML נתמכים

קבצי SAMI (Subtitles And Metadata Interchange) תומכים בקבוצה מוגבלת של תגים דמויי HTML לעיצוב ועיצוב הכתוביות. להלן כמה מהתגים הנפוצים הנתמכים על ידי SAMI:

- ``<SAMI> :`` אלמנט השורש שמכיל את כל קובץ ה-SAMI.
- ``<HEAD> :`` מכיל מידע כותרת עבור קובץ SAMI.
<html>- ``<TITLE> :`` מציין את הכותרת של קובץ SAMI. </html>
- ``<BODY style=";text-align:right;direction:rtl"> :`` מצרף את ערכי הכתוביות ופרטי התזמון שלהם.
- ``<SYNC> :`` מייצג נקודת סנכרון עבור ערך כתוביות. הוא מציין את התזמון שבו אמורה להציג את הכתובית.
- ``<P style=";text-align:right;direction:rtl"> :`` מקיף את תוכן הטקסט בפועל של כתובית. הוא משמש בדרך כלל בתוך a<SYNC> לַחסוֹם.
<html>- `` <FONT>:`` מגדיר מאפייני גופן עבור הטקסט המצורף. ניתן להשתמש בתכונות כמו צבע, פנים, גודל וסגנון כדי לשנות את מראה הגופן.</font>
- ``<BR> :`` הוספת מעבר שורה בתוך כתובית.
<html>- `` <B>:`` מציג את הטקסט המצורף בהדגשה.</b>
<html>- `` <I>:`` מציג את הטקסט המצורף באותיות נטוי.</i>
<html>- `` <U>:`` מציג את הטקסט המצורף עם קו תחתון.</u>
- ``<C> :`` מציין את המיקום או היישור של טקסט הכתוביות על המסך. הוא תומך בתכונות כמו מרכז, אמצע, שמאל, ימין, למעלה, תחתון והשילובים שלהם.
- ``<LANG> :`` מציין את קוד השפה עבור טקסט הכתוביות. זה עוזר בזיהוי שפת הכתוביות.

אלו הם חלק מהתגים הבסיסיים הנתמכים על ידי קבצי SAMI. חשוב לציין ש-SAMI אינו תומך במגוון המלא של תגיות ותכונות HTML. התגים הנתמכים מתמקדים בעיקר בעיצוב ובמיקום הכתוביות במקום לספק מבנה נרחב של מסמכים או אינטראקטיביות.
