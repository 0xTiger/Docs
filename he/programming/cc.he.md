## מהו קובץ CC?

קובץ CC הוא סוג של קובץ קוד מקור המכיל קוד שנכתב בשפת תכנות C++. זה יכול להכיל את כל הקוד הדרוש לתוכנית בודדת או חלק מהקוד שהוא חלק מפרויקט תכנות גדול יותר המורכב ממספר קבצי CC. כדי לשנות או להציג את הקוד הכלול בקובץ CC, מפתחים משתמשים לעתים קרובות בעורך קוד מקור. לאחר העריכה, הם מרכיבים את הקוד באמצעות מהדר כדי להמיר אותו לקוד מכונה, שמערכת ההפעלה של המחשב יכולה להפעיל כדי להפעיל את התוכנית.

## פורמט קובץ CC - מידע נוסף

סיומת הקובץ .cc שימשה עבור קובצי קוד מקור C++ במשך כמה עשורים. שפת C++ פותחה בסוף שנות ה-70 על ידי Bjarne Stroustrup שעבד ב-Bell Labs באותה תקופה. Stroustrup רצה להרחיב את שפת התכנות C עם תמיכה במושגי תכנות מונחה עצמים שלדעתו חיוניים לבניית מערכות תוכנה גדולות.

הגרסה הראשונה של C++ שוחררה בשנת 1983 והיא צברה במהירות פופולריות בקרב מפתחים שהעריכו את התכונות העוצמתיות והגמישות שלה. ככל ש-C++ נעשה שימוש נרחב יותר, היה צורך לקבוע מוסכמות קידוד ומוסכמות שמות קבצים כדי לסייע בארגון ובניהול בסיסי קוד גדולים.

מדריך הסגנון C++ של Google הממליץ להשתמש בסיומת קובץ .cc עבור קובצי קוד מקור של C++, פורסם לראשונה בשנת 2003. מאז המדריך עודכן מעת לעת כדי לשקף שינויים בשפת C++ ובנוהלי הפיתוח של קהילת התכנות הרחבה יותר.

כיום, סיומת הקובץ .cc נותרה בחירה פופולרית עבור קובצי קוד מקור C++ והיא מזוהה על ידי כלי פיתוח ופלטפורמות פופולריות רבות.

## איך פותחים קובץ CC?

עורכי קוד מקור רבים, כולל כלים מרובי פלטפורמות כמו Microsoft Visual Studio Code ו-Code Blocks, יכולים לשמש כדי לפתוח ולערוך קבצי CC. עורכים אלה מציעים תכונות שימושיות כגון הדגשת תחביר והשלמה אוטומטית, מה שהופך אותם לבחירה מצוינת לעבודה עם קבצי CC.

אתה יכול גם לפתוח קבצי CC באמצעות כל עורך טקסט כגון Notepad, Notepad++ וכו'.

## קומפילציה של CC עם מהדרים של C++

קומפילציה של קובץ CC עם מהדר C++ כרוכה במספר שלבים. הצעד הראשון הוא התקנת מהדר C++ תואם עבור מערכת ההפעלה וסביבת הפיתוח שלך. ישנם מהדרים רבים ושונים, כולל GCC, Clang ו- Microsoft Visual C++.

לאחר התקנת מהדר C++, השלב הבא הוא לפתוח מסוף או שורת פקודה ולנווט לספרייה שבה נמצא קובץ ה-CC. מכאן, אתה יכול להפעיל את המהדר על ידי הפעלת כלי שורת פקודה שסופק על ידי המהדר. הפקודה הספציפית להידור קובץ CC עשויה להשתנות בהתאם למהדר שבו אתה משתמש. כדוגמה, כדי להרכיב קובץ CC באמצעות GCC, תפעיל פקודה הדומה ל:

```
g++ -o myprogram myfile.cc
```
