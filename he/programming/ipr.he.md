## מהו קובץ IPR?

קובץ IPR הוא קובץ פרויקט שנוצר עם Java IDE **IntelliJ IDEA**. כל המידע הקשור להגדרות הפרויקט כגון מבנה ספריות, קובצי קוד מקור, ספריות הכלולות בפרויקט והגדרות מהדר מאוחסן בקבצי ה-IPR. כאשר פרויקט נפתח מתוך IntelliJ IDEA, ה-IDE קורא את המידע בקובץ ה-IPR וטוען את הקבצים וההגדרות המתאימים. קובצי IPR הוחלפו בקבצי .IDEA המכילים את כל המידע כ-Settings Directory.

## פורמט קובץ IPR

קובצי IPR נשמרים בפורמט הקובץ [XML](/he/web/xml/) המובן באופן אוניברסלי, הניתן לקריאה אנושית. הוא מכיל את כל המידע הקשור לפרויקט בתגים שנטענים להצגת תוכן הקשור לפרויקט.

מכיוון שקובצי IPR מאוחסנים בפורמט קובץ XML, ניתן לפתוח אותם בכל עורך טקסט כגון Notepad, Notepad++, Atom ו- Apple TextEdit.
