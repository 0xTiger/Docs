## מהו קובץ F95?

קובץ עם סיומת .f95 מציין בדרך כלל שזהו קובץ קוד מקור Fortran 95. Fortran היא שפת תכנות למטרות כלליות והכרחיות שמתאימה במיוחד למחשוב נומרי ומדעי. Fortran 95 היא גרסה משופרת של Fortran 90, המציגה מספר תכונות ושיפורים חדשים.

קובצי קוד מקור של Fortran מכילים בדרך כלל הוראות תוכנית שנכתבו בשפת התכנות Fortran; אם יש לך קובץ עם סיומת .f95, סביר להניח שהוא מיועד להידור באמצעות מהדר Fortran; קומפילציה היא תהליך של תרגום קוד מקור הניתן לקריאה על ידי אדם לקוד בינארי קריא במכונה, הניתן לביצוע על ידי מחשב.

## איך פותחים קובץ F95?

אם ברצונך לעבוד או להפעיל תוכנית בקובץ .f95, תזדקק למהדר של Fortran; מהדרים פופולריים של Fortran כוללים את GNU Fortran (gfortran), Intel Fortran Compiler (ifort) ואחרים. אתה יכול לקמפל ולהריץ תוכניות Fortran באמצעות פקודות מתאימות שסופק על ידי המהדר.

לדוגמה, אם התקנת את gfortran, אתה יכול להשתמש בפקודות הבאות בטרמינל כדי להדר ולהפעיל תוכנית Fortran:

```
# Compile the Fortran program
gfortran -o my_program my_program.f95

# Run the compiled program
./my_program
```

אם ברצונך להציג רק את התוכן של קובץ F95, זהו קובץ טקסט רגיל, כך שתוכל לפתוח אותו בכל עורך טקסט.
