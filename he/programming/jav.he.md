## מהו קובץ Java?

קובץ Java הוא קובץ קוד מקור של Java שנוצר בשפת התכנות Java. זה דומה לקובץ המקור של Java וניתן להרכיב אותו עם מתורגמני Java. כמו קבצי [Java](/he/programming/java/), קובץ Java משמש בפיתוח יישומי Java. זה יכול להיות אחד מקובצי קוד המקור הרבים הזמינים בפרויקט Java.

## פורמט קובץ Java

קובצי Java מאוחסנים כקבצי טקסט וניתנים לפתיחה בכל עורך טקסט. סוגי קבצי Java הוחלפו בדרך כלל בסוג קבצי Java ומשמשים בדרך כלל לכתיבת יישומי Java. ניתן לכתוב קודי Java עם IDE כמו Eclipse ו- IntelliJ Idea שמקל על העבודה עם קבצי Java.

### דוגמה לפורמט קובץ Java

```java
/*
The example code prints
Hello World from Java to the console.
*/
    public class JavaApp {
        public static void main(String[] args) {
            System.out.println("Hello World."); // Prints the string to the console.
        }
    }
```
