## מהו קובץ PHP?
קובץ עם סיומת php מתייחס לשפת תכנות בקוד פתוח, המשמשת לכתיבת סקריפטים בצד השרת, לביצוע בשרת אינטרנט. זוהי שפת ה-scripting האינטרנט הנפוצה ביותר המשמשת בדרך כלל לפיתוח של יישומי אינטרנט בקנה מידה גדול. מערכת הבלוגים הגדולה ביותר ברשת כלומר וורדפרס והרשת החברתית הגדולה ביותר פייסבוק מבוססות על שפת PHP. הוא תואם לפלטפורמות שונות כמו Windows, Linux, Unix, Mac OS X וכו'. כמו כן הוא פועל בהצלחה כמעט בכל השרתים המשמשים כיום, כלומר Apache, IIS וכו'. PHP תומך במגוון רחב של מסדי נתונים כולל MySQL.

## פורמט קובץ PHP ##

PHP בודד יכול להכיל טקסט, CSS, [HTML](/he/web/html/), JavaScript וקוד PHP. קוד PHP מופעל בשרת, והתוצאה מוחזרת לדפדפן כ-HTML רגיל. ניתן לפתוח קבצי PHP עם כל עורך טקסט ולערוך במקום, אם כי יישומים כמו Adobe Dreamweaver, כלי פיתוח PHP של Eclipse מציעים דרך נוחה לכתוב ולשנות קוד PHP.

### תחביר ###

סקריפט PHP מבוצע בשרת, ותוצאת ה-HTML הפשוטה נשלחת חזרה לדפדפן. סקריפט PHP מתחיל עם<?php and ends with ?>

### PHP דוגמה ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
