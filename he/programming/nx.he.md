## מהו קובץ NX?

קובץ NX הוא קובץ משחק וידאו שנוצר לשימוש עם קונסולת המשחקים הווירטואלית, **LowRes NX**. הוא כתוב בשפת התכנות BASIC ומכיל את קוד המקור של המשחק המגדיר את אופן פעולת המשחק. אתה יכול ליצור קבצי NX משלך לשימוש עם האפליקציה שניתן לטעון ומשתמשים יכולים לשחק במשחק.

אתה יכול לפתוח קובץ NX בעורך טקסט, LowRes NX, או כל עורך קוד מקור כגון Microsoft Visual Studio Code.

## פורמט קובץ NX

קובץ NX נשמר בתקליטור כקובץ טקסט רגיל המכיל את הוראות המשחק. לכל קובץ NX יש את קוד המקור של המשחק המורכב ממספר ערכי ROM של מחסניות. לערכי ROM אלה יש מידע על רכיבי משחק כגון גרפיקה, מפות רמות, אודיו ונתוני קוד מקור אחרים עבור המשחק. ניתן להוסיף או לערוך את ערכי ה-ROM של תוכנית באמצעות תוכניות עריכה של NX המגיעות יחד עם ה-LowRes NX.

## כיצד ליצור קבצי NX?

### צור קבצי NX במחשב

אתה יכול ליצור קבצי NX במחשב באמצעות:

* עורכי טקסט - כגון Microsoft Notepad, Notepad++ או Github Atom
* עורכי קוד מקור - כגון Microsoft VIsual Studio Code

### צור קבצי NX ב-iOS

ל- LowRes NX יש גרסת iOS שיכולה לשמש מפתחים ליצירת קבצי NX מתוך האפליקציה הזו.
