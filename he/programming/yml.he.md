## מהו קובץ YML?

YML הוא קובץ טקסט בשפת הסדרת נתונים, [YAML](/he/programming/yaml/) (Yet Another Markup Language), המשמש לעתים קרובות ליצירת קובצי תצורה. זה יכול לעבוד בשילוב עם כל שפת תכנות כולל C/C++, Ruby, Python, Java, PHP, C# ואחרים. בהיותה בלתי תלויה בכל שפת תכנות מסוימת, היא פתוחה, ניתנת להפעלה הדדית וניתנת לקריאה אנושית. ניתן לפתוח ולערוך קובצי YML עם עורכי טקסט נפוצים כגון Microsoft Notepad, Notepad++, Github Atom, Apple TextEdit ו-Visual Studio IDE.

## פורמט קובץ YML - מידע נוסף

קבצי YML מאחסנים את הגדרות האפליקציה כתצורה. המבנה הבסיסי שלו הוא מפה שאפשר לקרוא לה גם ספרייה או hash. ערכים בקובץ YML מורכבים מצמד מפתח וערכים. לרוב, קבצי התצורה .yml נשמרים בתור `configure.yml`, אם כי שפות תכנות מסוימות כגון Ruby on Rails שומרות את תצורת קישוריות מסד הנתונים בקבצי `database.yml`. [מפרטי פורמט קובץ YML/YAML](https://yaml.org/spec/1.2.2/) מסבירים את מבני השפה המלאים ומידע הקשור לתחביר.
