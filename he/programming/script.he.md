## מהו קובץ SCRIPT?

קובץ SCRIPT הוא קובץ מקור המכיל סקריפט בכל שפת סקריפט. זה הופך אותו לקובץ סקריפטים כללי המכיל משתנים ופונקציות. מכיוון שכל שפת סקריפטים עשויה להיות שונה, הסקריפט שנכתב בתוך קובץ SCRIPT צריך להתבצע באמצעות מנוע סקריפטים מתאים.

## פורמט קובץ SCRIPT - מידע נוסף

קבצי SCRIPT מאוחסנים על דיסק בקובצי טקסט רגיל. ניתן לפתוח ולערוך אותם בכל עורך טקסט כגון Microsoft Notepad, Notepad++ ו-Apple TextEdit. מכיוון ששפת הסקריפט יכולה להיות שונה, ניתן להפעיל אותה על ידי שינוי שם סיומת הקובץ לסיומת ברירת המחדל של שפת הסקריפט המתאימה. לדוגמה, ישנן מספר שפות סקריפטים נפוצות כגון JavaScript ([.JS](/he/web/js/)), VBScript ([.VBS](/he/programming/vbs/)), AppleScript (.SCPT), PHP ([ .PHP](/he/programming/php/)), ו-ASP ([.ASP](/he/web/asp/)).
