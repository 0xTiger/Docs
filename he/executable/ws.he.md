## מהו קובץ WS?

קובץ **.WS** ב-Windows מכיל בדרך כלל סקריפט בר הפעלה המיועד לביצוע באמצעות **Windows Script Host**. סקריפט זה יכול להקיף אלמנטים ושגרות מגוונים כגון שגרות JScript ו-VBScript וכן רכיבי XML. לאחר לחיצה כפולה על קובץ .ws, מארח הסקריפט של Windows יתחיל את ביצוע הסקריפט המוטבע בקובץ.

## אודות Windows Host Script

Windows Script Host (WSH) הוא מארח סקריפטים שפותח על ידי מיקרוסופט המאפשר למשתמשים להריץ סקריפטים שנכתבו בשפות סקריפטים כמו VBScript (מהדורת סקריפטים Visual Basic) ו-JScript (גרסת JavaScript של מיקרוסופט); הוא מספק מסגרת לסקריפטים במערכת ההפעלה Windows, המאפשרת אוטומציה של משימות שונות וניהול מערכת.

להלן כמה נקודות מפתח לגבי מארח סקריפט של Windows:

1.  **שפות סקריפטים:** WSH תומך במספר שפות סקריפטים, כאשר VBScript ו-JScript הם הנפוצים ביותר; ניתן לכתוב סקריפטים בכל אחת מהשפות הללו כדי להפוך משימות לאוטומטיות, לקיים אינטראקציה עם מערכת ההפעלה או לתפעל נתונים.
    
2.  **ביצוע סקריפט:** סקריפטים של WSH מאוחסנים בדרך כלל בקבצים עם סיומות כמו .vbs עבור VBScript או .js עבור JScript; כאשר אתה מפעיל סקריפט, WSH מפרש ומפעיל את הקוד.
    
3.  **אוטומציה:** WSH משמש לעתים קרובות למטרות אוטומציה כגון אוטומציה של משימות שחוזרות על עצמן, ניהול הגדרות מערכת וביצוע פונקציות ניהוליות; שימושי במיוחד עבור מנהלי מערכת ומשתמשים מתקדמים.
    
4.  **סקריפטים של מסוף ו-GUI:** סקריפטים של WSH יכולים להיות מבוססי קונסולות או מבוססי GUI; סקריפטים של מסוף פועלים בחלון שורת פקודה בעוד שסקריפטים של ממשק משתמש יכולים ליצור ממשקי משתמש גרפיים ליישומים אינטראקטיביים יותר.
    
5.  **Security:** WSH has security features to help prevent malicious scripts from causing harm; users are typically prompted before executing scripts and scripts are subject to certain restrictions to prevent unauthorized access and actions.
    
6.  **מודל אובייקט מארח של Windows Script:** WSH מספק מודל אובייקט המאפשר לסקריפטים ליצור אינטראקציה עם מערכת ההפעלה Windows; זה כולל גישה למערכות קבצים, הגדרות רישום, משאבי רשת ורכיבי מערכת אחרים.
    
7.  **ביצוע סקריפט משורת הפקודה:** ניתן להפעיל סקריפטים של WSH משורת הפקודה באמצעות קובצי ההפעלה `cscript.exe` או `wscript.exe`, תלוי אם ברצונך להפעיל את הסקריפט בסביבת מסוף או GUI .

## איך פותחים קובץ WS?

קבצי WS הם קבצי טקסט רגיל, כך שאם אתה רוצה לפתוח או לערוך אותם, אתה יכול להשתמש בכל עורך טקסט, למשל

- **פנקס רשימות**
- **פנקס רשימות++**
- **Apple TextEdit**
- **קוד סטודיו ויזואלי**

שים לב שאם אתה רוצה להפעיל את הסקריפט בתוך קובץ WS, אתה יכול פשוט ללחוץ עליו פעמיים או להשתמש בפקודה 'wscript' בשורת הפקודה.
