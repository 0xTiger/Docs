## מהו קובץ MA?

קובץ עם סיומת .ma הוא קובץ פרויקט תלת מימד שנוצר עם יישום Autodesk Maya. הוא מכיל רשימה גדולה של פקודות טקסטואליות לציון מידע על הקובץ. ניתן לפתוח ולערוך קובץ **.ma** בכל עורך טקסט כדי לתקן בעיות בפקודות למקרה שקובץ מושחת. קבצים אלה מכילים מידע להגדרת מידע סצינת תלת-ממד כגון גיאומטריה, תאורה, אנימציה ורינדור.

## פורמט קובץ MA

קבצי MA נשמרים בתקליטור בפורמט טקסט ASCII בניגוד לקבצי MB שנשמרים בפורמט קובץ בינארי לדיסק. הפניה מפורטת לפורמט קובץ MA זמינה ב-[תיעוד Autodesk Maya](https://download.autodesk.com/us/maya/2010help/index.html?url=Glossary_M_ma_file_format.htm,topicNumber=d0e192001) וניתן להפנותה לעיון המפתחים.

### כותרת קובץ MA

כותרת הקובץ MA מתחילה בקטע של הערות המספק את מידע היצירה של הקובץ ואת תאריך השינוי. קוראי קבצי Maya מתעלמים מהחסימה הזו מכיוון שהוא משמש רק למטרות מידע בלבד. כותרת חייבת להתחיל בששת התווים הראשונים בתור "// מאיה".

כותרת קובץ ASCII נראית כך.

```
//Maya ASCII 1.0 scene
//Name: solstice.ma
//Last modified: Sun, Dec 21, 97 10:18:26 AM
```
### הפניות לקבצים

קטע הפניות לקבצים של קובץ .ma מכיל מידע על כל שאר קבצי Maya שאליהם יש הפניה בקובץ זה. ניתן לקרוא כל קובץ עם הפניה באמצעות פקודת קובץ בודדת הכלולה באותו קובץ. התחביר של פקודת הקובץ בעת שימוש להפניה הוא:

```
file -r -rpr prefixString fileName;
```
אוֹ

```
file -r -ns nameSpace fileName
```
הנה מחרוזת לדוגמה.

```
file -r -rpr "solar" "/u/sally/work/solar.ma";
```
דוגמה זו מראה שאובייקט השמש הכלול בקובץ `solar` יהיה נגיש בקובץ זה באמצעות השם "solar_sun".

### דרישות

סעיף הדרישות של קובץ .ma מורכב מסדרה של פקודות נדרשות ומספר מידע למאי כגון מידע על גרסה ותוספים הנדרשים לקריאת הקבצים. דוגמה לסעיף הדרישות הוא כדלקמן.

```
requires maya "2.0";
requires specialPlugIn "1.2";
```


הסעיף הבא מפרט את הדרישות. זה מורכב מסדרה של פקודות דרישות. חלק זה של הקובץ אומר למאיה איזו תוכנה נדרשת כדי לטעון את הקובץ כראוי. ספציפית, איזו גרסה של Maya, ואיזה פלאגינים.
