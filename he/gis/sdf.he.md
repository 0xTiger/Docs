## מהו קובץ SDF?

A Spatial Data File (SDF) is a Geodatabase file format that was developed by Autodesk. It stores geospatial data and is used by Autodesk GIS program MapGuide and AutoCAD Map 3D. SDF file format is based on the single file database file format SQLite3. זה נחשב כפורמט קובץ מותאם עבור מערכי נתונים גדולים של מידע מרחבי.

אתה יכול לפתוח קובץ SDF באמצעות Autodesk AutoCAD Map 3D 2022 ו-Autodesk AutoCAD Civil 3D 2023.

## פורמט קובץ SDF - מידע נוסף

פורמט קובץ SDF מאוחסן בדיסק כקבצים בינאריים. הוא מבוסס על מסד הנתונים של SQLite שמשתמש ברכיבי אחסון ברמה נמוכה להסדרה בינארית של נתונים. קובץ SDF מכיל מחלקות תכונה מרובות לכל קובץ ומאפייני גיאומטריה מרובים עבור כל מחלקות תכונה. נתונים בקובץ SDF נגישים במהירות גבוהה עקב אינדקס של כל מאפיין גיאומטריה באמצעות עץ R.
