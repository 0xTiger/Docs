## מהו קובץ AMXX?

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. זוהי פלטפורמת סקריפטים המאפשרת למנהלי שרתים להתאים אישית ולהרחיב את הפונקציונליות של השרתים שלהם. AMXX משמש בשילוב עם מערכת ניהול תוספים MetaMod המאפשרת למספר תוספים לפעול בו זמנית על שרת ללא התנגשויות. התכונות המוצעות על ידי התוסף AMXX כוללות פקודות אדמין, שינויי משחק וניהול שרתים.
