## מהו קובץ MPP?

קובץ עם סיומת .mpp הוא קובץ נתונים של Microsoft Project המאחסן מידע הקשור לניהול פרויקטים בצורה משולבת. מיקרוסופט פיתחה את פורמט הקובץ הקנייני הזה כדי להפוך אותו לתואם ל-Microsoft Project (MSP), מוצר התוכנה שלהם לניהול פרויקטים. מלבד MPP, MSP יכול לעבוד עם סכימת XML של פרויקט. יישומים וממשקי API רבים מאפשרים להמיר MPPs לפורמטים אחרים של קבצים. למיקרוסופט יש כעת שרת פרויקטים מקוון שבו ניתן להעלות קבצי ניהול פרויקטים לשיתוף פעולה על ידי משתמשים מרובים.

## פורמט קובץ MPP ##

מיקרוסופט פיתחה ופרסמה את MPP כפורמט קובץ בינארי והמפרטים שלו לא פורסמו על ידי מיקרוסופט עד היום. מבנה הקבצים הפנימי והפרטים אינם זמינים, לפיכך, למפתחים. ישנם מספר ממשקי API זמינים באינטרנט התומכים בקריאה/כתיבה לקבצי MPP, אך 100% דיוק לעולם אינו מושג. מיקרוסופט הציגה מספר תכונות לאפליקציית ניהול הפרויקטים שלה לאורך זמן, וכך גם התכונות של פורמט הקובץ MPP.

קובץ MPP משתמש בסוגי MIME הבאים:

* application/vnd.ms-project
* application/msproj
* יישום/msproject
* יישום/x-msproject
* יישום/פרויקט x-ms
* application/x-dos_ms_project
* אפליקציה/mpp
* zz-application/zz-winassoc-mpp

## בעיות בפתיחת קובץ MPP? ##

להלן רשימה של כמה בעיות נפוצות שעלולות להתעורר ולגרום לתפקוד שגוי של פורמט MPP:

* היעדר תוכנה תומכת
* קובץ פגום
* קובץ נגוע עקב וירוס
* אין זכות גישה במערכת לפתיחת הקבצים
* כונן מיושן במערכת שלך
* שם ההרחבה של הקובץ שונה
