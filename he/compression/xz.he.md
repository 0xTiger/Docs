## מהו קובץ XZ?

XZ הוא פורמט קובץ דחוס המשתמש באלגוריתם הדחיסה LZMA2. הוא תוכנן כתחליף לפורמטים הפופולריים gzip ו-bzip2, ומציע מספר יתרונות על פני תקנים ישנים אלה. קבצי XZ נתמכים היטב בפלטפורמות רבות, וניתן לפרק אותם במהירות ובקלות. למרות שהם אינם נפוצים כמו קבצי [ZIP](/he/compression/zip/) או [RAR](/he/compression/rar/), ניתן להשתמש בארכיוני XZ לאחסון כמויות גדולות של נתונים מבלי להקריב את איכות הדחיסה. אם אתה צריך לדחוס או לשחרר קבצים גדולים, כדאי לשקול את סיומת הקובץ XZ.

## פורמט קובץ XZ

קובץ XZ נוצר ונשמר כקבצים בינאריים לדיסק. הוא משתמש באלגוריתם LZMA כדי לדחוס נתונים, ויכול להשיג יחס דחיסה של עד 50%. קבצי XZ משמשים בדרך כלל להפצות תוכנה וארכיוני גיבוי. ניתן לפרק אותם באמצעות כלי השירות XZ, הכלול ברוב ההפצות של לינוקס.

## פתח את קבצי XZ ב-Linux/Unix

על מנת לפתוח קבצי XZ, התקן תחילה את חבילת `xz-utils`:
```
$ sudo apt-get install xz-utils
```

השתמש בפקודה 'unxz' כדי לחלץ קבצי ‎.xz:
```
$ unxz compressed_xz_file.xz
```

או שימוש עם אפשרות --decompress של XZ:
```
$ xz --decompress compressed_xz_file.xz
```
