## מהו קובץ TIME?

פורמט הקובץ TIME שימש על ידי מערכת אינטרנט בשם LIGHT, שעזרה למשתמשים להקליט, לארגן ולשתף את חייהם. בעיקרו של דבר, הוא אחסן אוסף של פוסטים וייצג תיקיה בדף החיים של המשתמש במערכת.

## כיצד לגלות מתי נוצר קובץ בלינוקס

בלינוקס, אתה יכול להשתמש בפקודה `stat` כדי לגלות מתי נוצר קובץ. הנה איך אתה יכול לעשות את זה:

פתח מסוף והקלד את הפקודה הבאה:

```
stat <file_path>
```

החלף `<file_path> ` עם הנתיב לקובץ שבו אתה מעוניין. לדוגמה:

```
stat /path/to/your/file.txt
```

פקודה זו תציג מידע שונה על הקובץ, כולל זמן היצירה שלו. חפש את השורה שמתחילה ב-Birth או File:. שעת הלידה מציינת מתי הקובץ נוצר במערכת הקבצים.

להלן דוגמה כיצד עשוי להיראות הפלט:

```
  File: /path/to/your/file.txt
  Size: 1024       	Blocks: 8          IO Block: 4096   regular file
Device: 801h/2049d	Inode: 1643318     Links: 1
Access: (0644/-rw-r--r--)  Uid: ( 1000/   user)   Gid: ( 1000/   user)
Access: 2024-01-31 12:34:56.789012345 +0000
Modify: 2024-01-31 12:34:56.789012345 +0000
Change: 2024-01-31 12:34:56.789012345 +0000
 Birth: 2024-01-31 12:34:56.789012345 +0000
```

בדוגמה זו, שעת הלידה מציינת מתי הקובץ נוצר.
