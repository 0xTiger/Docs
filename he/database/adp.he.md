## מהו קובץ ADP?

קובץ עם סיומת .adp הוא קובץ פרויקט של Microsoft Access המשתמש בארכיטקטורת רכיבי OLE DB כדי לספק חיבור ישיר ויעיל למסד נתונים של Microsoft SQL Server. קבצים כאלה אינם מכילים את הטבלאות בפועל, דיאגרמות מסד נתונים או רכיבי מסד נתונים אחרים. ניתן ליצור קבצי ADP עם Access 2007 ו-2010. ניתן לפתוח קבצים שנוצרו עם הגירסאות הקודמות של Access גם ב-Access 2007 ו-2010. ניתן לפתוח קבצי ADP באמצעות Microsoft Access 365.

## פורמט קובץ ADP - מידע נוסף

פרויקט Access שנוצר כ-ADP מאפשר לך לבצע שינויים בעיצוב באובייקטי שרת SQL כגון טבלאות ותצוגות. זה גם מאפשר ליצור, לערוך ולהשתמש בתכונות SQL כגון דיאגרמות מסד נתונים, נהלים מאוחסנים ופונקציות מוגדרות על ידי משתמש. זה נותן יתרון על פני קישור למסד נתונים של שרת SQL שבו אתה לא יכול לבצע שינויים בעיצוב ויכול לקשר רק לטבלאות ותצוגות של שרת SQL.

### סוגי נתונים ותמיכה בתרשים

**סוגי נתונים של תאריך/שעה**

סוגי הנתונים החדשים הבאים של תאריך/שעה נתמכים ב-Access 2010.

*זמן
*תאריך
* DATETIME2
* DATETIME OFFSET

**סוגי נתונים באורך משתנה**

ניתן להשתמש בסוגי הנתונים הבאים באורך משתנה בפרויקטים של Access 2010:

* VARBIN(MAX)
* VARCHAR(MAX)
* NVARCHAR(MAX)
