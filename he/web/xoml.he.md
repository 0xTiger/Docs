## מהו קובץ XOML?

XOML הוא ראשי תיבות של Extensible Object Markup Language והוא פורמט סידורי עבור אובייקטי זרימת העבודה של Windows Workflow Foundation. בהתבסס על **[XAML](/he/web/xaml/)**, הוא משמש בעיקר ליצירת ממשקי משתמש בצורה פשוטה **[XML](/he/web/xml/)**. אלה משמשים להכרזה על זרימת העבודה עבור ממשק המשתמש והם מורכבים יחד עם הקובץ הנפרד המכיל את היגיון היישום. הוא מכיל מבנה מבוסס עץ המגדיר צומת זרימת עבודה שורש, תת-אלמנטים מקוננים וקטעי קוד משובצים. כאשר זרימת עבודה חדשה נוצרת עם Visual Studio עבור .NET, יש לה אפשרות לבחור אם היא צריכה להיות בפורמט Code או בפורמט Code Separated. במקרה שתבחר במאוחר יותר, ה-IDE יצור עבורך שני קבצים; אחד בפורמט XOML (המורכב מפריסת זרימת עבודה והגדרות), והשני הוא בפורמט [.CS](/he/programming/cs/)/[.VB](/he/programming/vb/) שבו נמצא קוד התכנות.
