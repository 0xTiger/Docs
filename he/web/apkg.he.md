## מהו קובץ APKG?

קובץ עם סיומת .apkg הוא חפיסת כרטיסי פלאש שנוצרה לשימוש ביישום התוכנה [Anki](https://ankiweb.net/about) שהיא תוכנית למידה מבוססת כרטיסי פלאש. הוא מכיל טקסט [HTML](/he/web/html/) לטעינה ולהצגה באפליקציית Anki ויכול להכיל בנוסף תמונות וצלילים ללמידה חזותית ונשמעת. Anki מאפשרת למשתמשים ליצור חפיסות כרטיסי פלאש של Anki משלהם וכן לייבא חפיסות כרטיסי פלאש של משתמשים אחרים.

## פורמט קובץ APKG

חפיסות קלפים של אנקי נוצרות מתבניות שכתובות ב-HTML, שפה מפורסמת ונפוצה ליצירת דפי אינטרנט. הסגנון של קלפי החפיסה נעשה באמצעות CSS שהיא השפה המשמשת לעיצוב דפי אינטרנט. הסטיילינג כולל שינויים ב:

* font-family - שם הגופן שישמש בכרטיס.
* font-size - גודל הגופן בפיקסלים.
* יישור טקסט - מגדיר אם יש ליישר את הטקסט במרכז, לשמאל או לימין.
* צבע - מגדיר את צבע הטקסט שיכול להיות שמות צבע פשוטים כגון "כחול", "אדום" וכו' או קודי צבע HTML.
* רקע-צבע - מגדיר את צבע הרקע של הכרטיס

מידע הסגנון משותף בין כל הכרטיסים, ומשפיע על כל הכרטיסים כאשר נעשה שינוי. הדוגמה הבאה תשתמש ברקע צהוב בכל הקלפים מלבד הראשון:

```CSS
.card {
  background-color: yellow;
}
.card1 {
  background-color: blue;
}
```

ניתן לשלוט בשינוי גודל התמונות בכרטיס Anki באופן הבא.

```CS
img {
  max-width: none;
  max-height: none;
}
```

## הטמעת Javascript בכרטיסי Anki

אפשר להטמיע [Javascript](/he/web/js/) בכרטיס Anki באמצעות תבניות הכרטיסים. עם זאת, בשל הרמה המתקדמת של Javascript, התמיכה בו אינה ניתנת. יתר על כן, מכשירי רינדור עשויים להראות את השפעות היישום של Javascript בכרטיס באופן שונה, וכתוצאה מכך הצורך בבדיקת היישום בכל המכשירים. תכונות Javascript מסוימות כגון window.alert, המקשות על ניפוי באגים בקוד ה-Javascript שכתבת.
