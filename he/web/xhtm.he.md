## מהו קובץ XHTM?

קובץ עם סיומת .xhtm הוא קובץ דף אינטרנט שנכתב ב-XHTML. הוא משתמש ב-XML 1.0 כיישום הסטנדרטי שלו ונחשב לניסוח מחדש של HTML 4. כמו XHTML, קובץ XHTM מכיל תגי HTML סטנדרטיים העוקבים אחר הנחיות XML מחמירות יותר. קובצי XHTM משמשים לפרסום תוכן אתר ומשתמשים בדרך כלל בסיומת [XHTML](/he/web/xhtml/).

אתה יכול להשתמש בדפדפני אינטרנט כגון Google Chrome, Apple Safari ו-Mozilla Firefox כדי לפתוח קבצי XHTM.

## דוגמה XHTM

הדוגמה הבאה של מסמך XHTM מגדירה את הצהרות ה-XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
