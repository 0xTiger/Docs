## מהו קובץ CCBJS?

קובץ CCBJS הוא קובץ Javascript שנוצר על ידי האפליקציה, המשחק והאתר התלת-ממדית, **CopperCube**. הוא מכיל את קוד JavaScript שנוצר כחלק מקובץ [.html](/he/web/html/) שנוצר מקובץ .ccb. CopperCube מספקת ממשק סקריפטים ידידותי למשתמש לפיתוח היגיון משחק ואינטראקציות מבלי לכתוב קוד כלשהו. פורמטים אחרים של קבצים הנתמכים על ידי CopperCube כוללים .ccscene, .ccmodel, .ccmaterial, .cctex, .ccjs, .ccp ו-.ccpak.

## פורמט קובץ CCBJS

פרטי פורמט הקובץ הפנימי של פורמט הקובץ CCBJS אינם ידועים ככאלה. עם זאת, קבצי Javascript נשמרים כקובצי טקסט רגיל בדיסק וניתן לפתוח אותם עם כל עורך טקסט כגון Microsoft Notepad, Notepad++ ו- Apple TextEdit.
