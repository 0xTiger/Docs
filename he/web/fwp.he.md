## מהו קובץ FWP?

קובץ עם קובץ .fwp הוא קובץ ארכיון שנוצר באמצעות Microsoft Expression Web ומכיל את כל הקבצים והתיקיות לעיצוב. Microsoft Expression Web הוא מוצר כללי לעיצוב אתרים מאת Microsoft ומבוסס על עורך [HTML](/he/web/html/). הוא הופסק מאז 2012 וזמין להורדה בחינם על ידי מיקרוסופט. ניתן לייצא ולשמור קובצי FWP כפורמט .cab של Microsoft.

## פורמט קובץ FWP - מידע נוסף

מפרטי פורמט קובץ מדויקים של קובצי FWP אינם זמינים לעיונם של המפתח. עם זאת, מדובר בקבצים בינאריים וניתן לפתוח אותם ב-Microsoft Expression Studio שהופסק מזמן.
