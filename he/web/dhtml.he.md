## מהו קובץ DHTML?

קובץ עם סיומת .dhtml הוא קובץ [HTML](/he/web/html/) דינמי המשמש ליצירת תוכן דינמי של דף אינטרנט. רכיב אינטרנט שנוצר ב-DHTML מונחה אירועים ואינו מצריך לטעון מחדש את הדף. ברוב המקרים, קובץ DHTML משמש ליצירת התוכן הדינמי של דף אינטרנט כגון תפריטים נפתחים, שכבות צפות, לחצני גלגול ותוכן דינמי אחר. אתה נתקל ברכיבי HTML דינמיים כמעט מדי יום בחייך כאשר אתה מרחף עם העכבר על פריט בתפריט והוא פותח אפשרויות נוספות של תפריט משנה. DHTML עושה שימוש בטכנולוגיות אינטרנט כגון [HTML](/he/web/html/), [Javascript](/he/web/js/), HTML DOM, HTML Events ו-[CSS](/he/web/css/) כדי להשיג את הדינמיקה התנהגות של אלמנטים.

## פורמט קובץ DHTML

קובצי DHTML הם קבצי טקסט רגיל המכילים קוד DHTML כדי ליישם את ההתנהגות הדינמית של רכיבי האינטרנט.


### DHTML DOM

מודל DHTML Document Object Model (DOM) מבוסס על HTML DOM שהוא מבנה עץ עם אלמנטים, תכונות וטקסט כפי שמוצג בתמונה הבאה.

{{< figure src="../dhtml.png" alt="HTML DOM" >}}

ניתן להשתמש בצומת 'מסמך' כדי לקרוא למספר פונקציות כדי ליישם פונקציונליות שונה. הדוגמה הבאה פשוט משתמשת בשיטת document.write() של JavaScript ב-DHTML.

```
<HTML>  
<head>  
<title>  
Method of a JavaScript  
</title>  
</head>  
<body>  
<script type="text/javascript">  
document.write("Hello World");  
</script>  
</body>  
</html>  
```

קוד זה כותב את הטקסט "Hello World" לפלט בדפדפן.

### אירועי DHTML

|S.No.|אירוע|התרחשות|
---|---|---|
|1|onabort|זה מתרחש כאשר המשתמש מבטל את טעינת הדף או קובץ המדיה.|
|2|onblur|זה מתרחש כאשר המשתמש עוזב אובייקט HTML.|
|3|onchange|זה מתרחש כאשר המשתמש משנה או מעדכן את הערך של אובייקט.|
|4|onclick|זה מתרחש או מופעל כאשר משתמש כלשהו לוחץ על אלמנט HTML.|
|5|ondblclick|זה מתרחש כאשר המשתמש לוחץ על רכיב HTML פעמיים יחד.|
|6|onfocus|זה מתרחש כאשר המשתמש מתמקד באלמנט HTML. מטפל באירועים זה פועל בניגוד ל-onblur.|
|7|onkeydown|זה מופעל כאשר משתמש לוחץ על מקש במכשיר מקלדת. מטפל באירועים זה עובד עבור כל המפתחות.|
|8|onkeypress|זה מופעל כאשר המשתמשים לוחצים על מקש במקלדת. מטפל באירועים זה אינו מופעל עבור כל המפתחות.|
|9|onkeyup|זה מתרחש כאשר משתמש משחרר מקש ממקלדת לאחר לחיצה על אובייקט או אלמנט.|
|10|onload|זה מתרחש כאשר אובייקט נטען לחלוטין.|
|11|onmousedown|זה מתרחש כאשר משתמש לוחץ על כפתור העכבר מעל אלמנט HTML.|
|12|onmousemove|זה מתרחש כאשר משתמש מזיז את הסמן על אובייקט HTML.|
|13|onmouseover|זה מתרחש כאשר משתמש מעביר את הסמן מעל אובייקט HTML.|
|14|onmouseout|זה מתרחש או מופעל כאשר מצביע העכבר מוזז מתוך אלמנט HTML.|
|15|onmouseup|זה מתרחש או מופעל כאשר כפתור העכבר משוחרר מעל אלמנט HTML.|
|16|onreset|הוא משמש את המשתמש לאיפוס הטופס.|
|17|onselect|זה מתרחש לאחר בחירת התוכן או הטקסט בדף אינטרנט.|
|18|onsubmit|זה מופעל כאשר המשתמש לוחץ על כפתור לאחר שליחת טופס.|
|19|onunload|זה מופעל כאשר המשתמש סוגר דף אינטרנט.|
