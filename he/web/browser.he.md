## מהו קובץ BROWSER?

קובץ עם סיומת .browser משמש יישומי אינטרנט של ASP.NET כדי לקבוע את היכולת והתצורה של דפדפני אינטרנט. הוא מכיל מידע על היכולות, המגבלות והמאפיינים של הדפדפן. מאפיינים אלו עוזרים למסגרת ASP.NET לזהות את הדפדפן המבקש ולהציג את דף האינטרנט למשתמשים בהתאם.

## פורמט קובץ דפדפן

קבצי הגדרות דפדפן משמשים את יישום ASP.NET כדי לקבוע את היכולות של הדפדפן. זהו מידע הכלול בקובץ זה המשמש את השרת ליצירת סימון ומשאבים אחרים התואמים לדפדפן המבקש.

קבצי דפדפן נשמרים בפורמט קובץ טקסט רגיל. אתה יכול לפתוח קובץ דפדפן בעורכי טקסט כגון Notepad, Notepad++, Apple TextEdit ו-Visual Studio IDE.

### מבנה הקובץ של פורמט הקובץ BROWSER

קבצי ההגדרות של BROWSER משתמשים במבנה היררכי. כל קובץ .browser מכיל סימון XML הכולל את היכולות והתכונות של דפדפן מסוים או קבוצת דפדפנים. פרטים אלה כוללים מאפיינים כגון מחרוזת סוכן, מספרי גרסאות ותמיכה בטכנולוגיות כגון JavaScript, CSS או רכיבי HTML אחרים. על ידי שימוש בקבצי הגדרות הדפדפן הללו, מפתחי ASP.NET מתאימים אישית את חוויות המשתמש בהתבסס על היכולות של דפדפנים שונים.
