## מהו קובץ VDISCO?

קובץ VDISCO הוא קובץ גילוי המשמש את Microsoft Visual Studio בשירותי האינטרנט שלה. הוא מספק מידע על שירותי האינטרנט הזמינים כגון כתובות האתרים, מרחבי השמות והשיטות שלהם. קובץ VDISCO דומה לפורמט הקובץ [DISCO](/he/web/disco/) אך נוצר בזמן ריצה. הוא מאוחסן באפליקציית שירות האינטרנט ומשמש את Visual Studio כדי לגלות ולהשתמש בשירותי אינטרנט באופן דינמי בפרויקט. נעשה שימוש בקובצי VDISCO בשילוב עם הקובץ [.asmx](/he/web/asmx/) המכיל את קוד שירות האינטרנט בפועל.

אתה יכול לפתוח קובץ VDISCO בכל עורך טקסט כגון Microsoft Notepad, Github Atom ו- Apple TextEdit. ניתן לפתוח אותו גם עם Microsoft Visual Studio 2012 ואילך לעבודה איתו.

## פורמט קובץ VDISCO

קובץ VDISCO נשמר ומתארח בפורמט קובץ [XML](/he/web/xml/) שהוא פורמט אוניברסלי לחיבור ופרסום נתונים. הוא מכיל את כתובת האתר של השירות, מרחבי השמות והשיטות בתגי XML סטנדרטיים כאשר כל תג מכיל את הערך המתאים של המידע שלו.
