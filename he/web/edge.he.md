## מהו קובץ EDGE?

קובץ עם סיומת .edge הוא קובץ Adobe Edge Project שנוצר כאשר פרויקט נוצר ונשמר עם Adobe Edge. Adobe Edge מאפשרת למעצבי אינטרנט ליצור אנימטורים אינטראקטיביים [HTML](/he/web/html/) עבור אינטרנט, פרסום דיגיטלי ופרסום במדיה עשירה עבור דפדפנים שולחניים וניידים כאחד. זה מושג באמצעות HTML/HTML5, [JavaScript](/he/web/js/) ו-[CSS3](/he/web/css/). קובצי Edge נוצרים כקובצי טקסט רגיל וניתן לפתוח אותם עם מספר יישומים שונים כגון Microsoft Notepad, Apple TextEdit ו-Notepad++.

## פורמט קובץ EDGE

קובצי EDGE נשמרים בפורמט טקסט רגיל ומשתמשים ב-jQuery להפעלת האנימציה. אלטרנטיבה לאנימציית EDGE הייתה קבצי פלאש שאינם נתמכים יותר. קובץ EDGE נשמר לצד קובצי HTML, JavaScript ו-CSS. עם פתיחת קובץ ה-EDGE, דף האינטרנט כלומר קובץ ה-HTML נפתח.
