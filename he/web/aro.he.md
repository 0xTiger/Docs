## מהו קובץ ARO?

קובץ ARO הוא קובץ עמוד אינטרנט בצד השרת שנוצר באמצעות SteelArrow Web Application. הוא מכיל תגים וסקריפטים של [HTML](/he/web/html/) ו-SteelArrow. אלה מבוצעים בשרת כדי ליצור דף תגובה מלא לפני שליחתם לדפדפן של המשתמש. קובצי ARO מכילים כמעט 95% מתוכן HTML בעוד שהשאר מורכב מקוד SteelArrow. כדי שקובצי ARO יעבדו עבורך, שרת SteelArrow Web Applications צריך להיות מותקן ופועל על מחשב שרת האינטרנט.

## פורמט קובץ ARO

קובצי ARO נכתבים בקובץ שפת סימון HTML עם קוד JavaScript מוטבע ויישומי Java. [תגיות SteelArrow](http://www.steelarrow.com/docs/basics1.aro) הם אבני הבניין הבסיסיות לפיתוח דפי אינטרנט. למרות שאלו אינם משנים את תצוגת הדף, הם משמשים למילוי הדף בנתונים. בנוסף, הם עשויים לשמש גם כדי לשלוט בפלט עם הצהרות מותנות.

### דוגמה לתג ARO

ה<SAOUTPUT> תג ARO מאפשר למפתחים להוציא ערכי נתונים בכל מקום בתוך סקריפט. לדוגמה, ניתן להשתמש בו כדי לקבל את הפלט של טבלת PARAM עם<SAOUTPUT VALUE=PARAM[]> שניתן להציג בתוך ה-HTML<BODY style=";text-align:right;direction:rtl"> תָג.
