## Kas ir AZW4 fails?

AZW4 ir Amazon Print Replica e-grāmata, ko Amazon izstrādāja Kindle darbvirsmas programmatūrai un Kindle eReader ierīcēm. AZW4 nav līdzīgs [MOBI](/ebook/mobi/), [AZW](/ebook/azw/) vai [AZW3](/ebook/azw3/). Šis formāts faktiski ir iesaiņojums [PDF](/pdf/) vai MOBI e-grāmatām ar pievienotu DRM.

## AZW4 faila formāts

The **AZW4** has been introduced as a new version of **Textbooks** software which is similar to **PDF**. An AZW4 eBook contains a complete PDF file inside, or it can be considered as a PDF file with a wrapper called AZW4. Galvenokārt AZW4 izmanto Kindle eReader. Tomēr citas lietojumprogrammas var izmantot arī .azw4 faila paplašinājumu. Pat ļaunprātīgas programmatūras programmas var izveidot .azw4 failus. Esiet piesardzīgs ar .azw4 failiem, kas nāk no apšaubāma avota.
