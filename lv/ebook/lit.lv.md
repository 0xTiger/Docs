## Kas ir LIT fails?

LIT faili pieder senajam **Microsoft Reader**, kas tika izmantots e-grāmatu lasīšanai. Programmā Microsoft Reader rādītajām e-grāmatām ir paplašinājums **.lit**, kas ir saīsināts līdz **Literatūra**. LIT faila formāts sastāv no Microsoft ITOL/ITLS HTML palīdzības 2.0 faila, kas nodrošina visu veidu pārsteidzošas funkcijas, piemēram, [LZX](/compression/lzx/) saspiešanu, iezīmēšanas materiāla bināro attēlojumu un patvaļīgus izvēles papildu datus, kas saistīti ar katru straumi. Grāmatas teksts ir patvaļīgs OEBPS 1.0 iezīmēšanas straumju skaits un OEBPS 1.0 CSS apakškopa. Attēli ir [GIF](/image/gif/), [JPEG](/image/jpeg/) un [PNG](/image/png/).

## Cik ilgi Microsoft Reader rādīja LIT failus

The **Microsoft Reader** is an application for reading e-books that have been discontinued by Microsoft. The first version of Microsoft Reader was released in August 2000, which used its own LIT format. It was available for PDAs and Windows computers. The name of the app was also used later for an application (which was initially released with Windows 8) for reading [PDF](/pdf/) and [XPS](/page-description-language/xps/) files. The Microsoft Reader application was discontinued in 2018.
