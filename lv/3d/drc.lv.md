## Kas ir DRC fails?

Fails ar paplašinājumu .drc ir saspiests 3D faila formāts, kas izveidots, izmantojot Google Draco bibliotēku. Google piedāvā Draco kā atvērtā pirmkoda bibliotēku 3D ģeometrisko tīklu un punktu mākoņu saspiešanai un atspiešanai, kā arī uzlabo 3D grafikas uzglabāšanu un pārraidi. Tas kodē ievades datus un saglabā tos kā .drc failu. Saņemšanas galā API nolasa .drc failus un var tos izvadīt kā [PLY](/3d/ply/) vai [OBJ](/3d/obj/) failus. Saspiestais izvades fails ļauj lietotājiem ātrāk lejupielādēt lietotnes un nodrošināt ātru 3D grafikas ielādi pārlūkprogrammās.

## KDR faila formāts

File format details for DRC are not available publicly. Google has hosted the Draco library on [GitHub](https://github.com/google/draco/tree/master/docs) for developer's reference. It provides encoders and decoders for creating and converting DRC files.
