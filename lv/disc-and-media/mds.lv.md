## Kas ir MDS fails?

MDS (Media Descriptor Sidecar File) ir fails, kas saistīts ar programmatūru Alcohol 120% un Daemon Tools, kuras abas tiek izmantotas virtuālo CD un DVD disku izveidei un pārvaldībai. MDS fails satur informāciju par datu izkārtojumu diskā, tostarp visu failu atrašanās vietu un diska struktūru. Tas ļauj virtuālajam diskam līdzināties fiziska diska darbībai, lai programmatūra varētu nolasīt datus virtuālajā diskā tā, it kā tas būtu īsts disks.

When you want to mount an image file to a virtual drive, the MDS file is used along with the corresponding image file in a format such as ISO, BIN or CUE. The MDS file contains a description of the layout of the data on the disc, and the virtual drive software uses this information to create a virtual disc. This allows you to use the software as if you were using a physical disc, without having to physically insert the disc into the drive.

## Saistība ar alkoholu 120%

Alcohol 120% ir populāra programmatūra virtuālo CD un DVD disku izveidei un pārvaldībai, un tā izmanto MDS faila formātu, lai saglabātu un piekļūtu diska attēliem. MDS faila formāts pieder Alcohol 120%, un to var atvērt un izmantot tikai Alcohol 120% programmatūra vai cita programmatūra, kas to atbalsta.

## Kā atvērt MDS failu?

Lai atvērtu MDS failu programmā Alcohol 120%, datorā ir jābūt instalētai programmatūrai. Kad esat instalējis Alcohol 120%, varat atvērt MDS failu, rīkojoties šādi:

1. Palaist Alkohols 120%.
2. Noklikšķiniet uz izvēlnes Fails un atlasiet Atvērt.
3. Datorā dodieties uz MDS faila atrašanās vietu un atlasiet to.
4. MDS fails tagad tiks atvērts programmā Alcohol 120%, un jums tiks piedāvāts atlasīt atbilstošo attēla failu (piemēram, ISO, BIN vai CUE).
5. Pēc attēla faila atlasīšanas Alcohol 120% pievienos attēlu virtuālajam diskam.

Varat arī atvērt MDS failu, veicot dubultklikšķi uz tā, ja Alcohol 120% ir iestatīts kā noklusējuma programma MDS failu atvēršanai. Varat arī izmantot citu programmatūru, kas atbalsta MDS formātu, piemēram, Daemon Tools, Virtual CloneDrive, Virtual Drive Manager un MagicISO.
