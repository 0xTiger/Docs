## Kas ir GADGET fails?

GADGET fails sastāv no nelielas programmatūras programmas, kas tiek izpildīta Windows Vista vai Windows 7 sānjoslā. Tas saspiež vairākus tīmekļa failus, un faili var ietvert [.html](/web/html/), [.css](/web/css/) vai [.js](/web/js/) failus, kā arī citus tīmekļa failus. GADGET faili tiek izmantoti maziem komponentiem, piemēram, meklēšanas rīkiem, ziņu plūsmām, sistēmas utilītprogrammām un mazām spēlēm. Lai gan Sīkrīki tiek mitināti sānjoslā, tie nav specifiski sānjoslas apgabalam; tos var atvienot un pēc vajadzības pārvietot uz darbvirsmu.

## GADGET faila formāts

SĪKĀKLAS fails ir pārdēvēts [ZIP](/compression/zip/) arhīvs, kurā ir HTML, XML, JScript un CSS (kaskādes stila lapas) failu kolekcija. Instalācija ietver .gadget faila lejupielādi un lejupielādes procesa iespējošanu komponenta instalēšanai vai .gadget faila saglabāšanu vietējā sistēmā un dubultklikšķi, lai sāktu instalēšanas procesu.

Būtībā SĪKrīkā ir divi faili:

1. **Gadget.xml**: manifests, XML fails, kas sastāv no sīkrīka vispārīgas prezentācijas un konfigurācijas informācijas.
2. **name.html**: HTML fails, kura nosaukums ir norādīts<name> saistītā sīkrīka manifesta tags, kas nodrošina sīkrīka lietotāja saskarnes iesaiņojumu un ietver sīkrīka pamatfunkcionalitāti.

Vienlaicīgi var palaist vairākus SĪKĀKLA faila gadījumus. Piemēram, ja kāds vēlas uzzināt laiku dažādās laika joslās, viņš var palaist vairākus pulksteņa sīkrīka gadījumus, iestatot katru pulksteni uz noteiktu laika joslu.

### Sīkrīka darbības pārtraukšana

Tā kā Windows sānjoslas platformai operētājsistēmā Windows 7 ir nopietnas ievainojamības, sīkrīki vairs nav pieejami Microsoft oficiālajā vietnē. Vēlāk Microsoft pārtrauca šīs funkcijas darbību jaunākajos Windows laidienos. GADGET var izmantot, lai jebkurā laikā piekļūtu datora failiem, kaitētu datoram, atklātu nevēlamu saturu vai mainītu to uzvedību.
