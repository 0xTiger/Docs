## Kas ir IP fails?

IP faila formāts ir īpašs faila veids, ko izveido programma **IconPackager**. Šis fails ir kā ikonu kopums vai pakotne, ko varat izmantot, lai mainītu datora darbvirsmas izskatu. Lietojot IP failu, tas aizvieto daudzas noklusējuma ikonas jūsu Windows datorā ar pielāgotām ikonām no pakotnes. Šīs pakotnes var ietvert tikai dažas ikonas vai pat vairāk nekā simts, piešķirot datoram personalizētu un unikālu izskatu.

## Par IconPackager

IconPackager ir Stardock izstrādāta lietojumprogramma, kas lietotājiem ļauj pielāgot un mainīt ikonu izskatu savā Windows datorā; tā vietā, lai izmantotu operētājsistēmas nodrošinātās noklusējuma ikonas, IconPackager ļauj lietotājiem lietot ikonu pakotnes vai motīvus.

Lūk, kā IconPackager parasti darbojas:

1.  **Ikonu pakotnes:** IconPackager ir aprīkots ar dažādām ikonu pakotnēm vai motīviem; katrā pakotnē ir iekļauta pielāgotu ikonu kopa dažādiem sistēmas elementiem, piemēram, mapēm, failiem un īsceļiem.
    
2.  **Lietojumprogramma:** lietotāji var izmantot IconPackager, lai lietotu šīs ikonu pakotnes savā Windows darbvirsmā; kad tiek lietota pakotne, tā aizstāj standarta ikonas darbvirsmā, uzdevumjoslā un failu pārlūkā ar pielāgotām ikonām no izvēlētā motīva.
    
3.  **Pielāgošana:** IconPackager nodrošina lietotājam draudzīgu saskarni, kas ļauj lietotājiem sajaukt un saskaņot ikonas no dažādām pakotnēm vai izveidot savus pielāgotos komplektus; šādā veidā lietotāji var personalizēt savas darbvirsmas izskatu atbilstoši savām vēlmēm.
    
4.  **Elastīgums:** lietotāji var viegli pārslēgties starp dažādām ikonu pakotnēm, dodot viņiem iespēju jebkurā laikā mainīt darbvirsmas vizuālo stilu.

## Kā atvērt IP failu?

IP failus var atvērt vai atsaukties uz tiem, izmantojot šādas programmas

- **Stardock IconPackager** (bezmaksas izmēģinājuma versija) operētājsistēmai Windows
