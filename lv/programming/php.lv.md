## Kas ir PHP fails?
Fails ar paplašinājumu .php attiecas uz atvērtā pirmkoda programmēšanas valodu, ko izmanto servera puses skriptu rakstīšanai, kas jāizpilda tīmekļa serverī. Tā ir visplašāk izmantotā tīmekļa skriptu valoda, ko parasti izmanto liela mēroga tīmekļa lietojumprogrammu izstrādei. Lielākā emuāru veidošanas sistēma tīmeklī, ti, WordPress un lielākais sociālais tīkls Facebook, ir balstīti uz PHP valodu. Tas ir savietojams ar dažādām platformām, piemēram, Windows, Linux, Unix, Mac OS X uc Tāpat tas veiksmīgi darbojas gandrīz visos mūsdienās izmantotajos serveros, piemēram, Apache, IIS uc PHP atbalsta plašu datu bāzu klāstu, ieskaitot MySQL.

## PHP faila formāts ##

Viens PHP var saturēt tekstu, CSS, [HTML](/web/html/), JavaScript un PHP kodu. PHP kods tiek izpildīts serverī, un rezultāts tiek atgriezts pārlūkprogrammā kā vienkāršs HTML. PHP failus var atvērt ar jebkuru teksta redaktoru un rediģēt vietā, lai gan tādas lietojumprogrammas kā Adobe Dreamweaver, Eclipse PHP izstrādes rīki piedāvā ērtu veidu, kā rakstīt un modificēt PHP kodu.

### Sintakse ###

Serverī tiek izpildīts PHP skripts, un vienkāršs HTML rezultāts tiek nosūtīts atpakaļ uz pārlūkprogrammu. PHP skripts sākas ar<?php and ends with ?>

### PHP piemērs ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
