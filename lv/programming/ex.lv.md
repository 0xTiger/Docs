## Kas ir EX fails?

EX fails ir pirmkoda fails, kas satur pirmkodu lietojumprogrammām, kas izstrādātas, izmantojot Euphoria programmēšanas valodu. Euphoria ir procesuāla programmēšanas valoda, kas tic programmēšanas vienkāršībai, nevis sarežģītām programmēšanas valodām. Programmā Euphoria izstrādāta lietojumprogramma var ietvert vairākus EX failus, bet vienā EX failā var būt arī pilns programmas avots.

Lietojumprogrammas, kas var **atvērt EX failus**, ietver openEuphoria Euphoria, Microsoft Notepad, Notepad++ un Atom.

## Īsa EX faila formāta vēsture

The development and existence of EX file format is linked with the development of Euphoria programming language which was created by Robert Craig. It was first released for MS-DOS. It became open-source software in 2006 with the release of version 3. Tā izstrādi pārņēma **openEuphoria grupa**, kas administrē un attīsta projektu. Tās 4. versija tika izlaista 2010. gada decembrī, un tā ir pieejama operētājsistēmām Windows, Linux, macOS un trim BSD versijām.

Euphoria sākotnēji tika uzrakstīta **[C](/programming/c/)** programmēšanas valodā. Vēlāk Euphoria tulks tika sadalīts divās daļās:

 * Front-end Parser — rakstīts Euphoria valodā un tiek izmantots ar Euphoria-to-C tulkotāju un Binder.
 * `Back-end` - izpildes laika bibliotēka, kas rakstīta C programmēšanas valodā.

## EX faila formāts — vairāk informācijas

EX faili tiek glabāti kā vienkārša teksta faili, un tajos ir Euphoria programmēšanas valodā rakstīto programmu pirmkods. Šos failus var atvērt jebkurā teksta redaktorā.
