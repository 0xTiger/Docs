## Kas ir Java fails?

Java fails ir Java pirmkoda fails, kas izveidots Java programmēšanas valodā. Tas ir līdzīgs Java avota failam, un to var apkopot ar Java tulkiem. Tāpat kā [Java](/programming/java/) faili, arī Java fails tiek izmantots Java lietojumprogrammu izstrādē. Tas var būt viens no daudzajiem avota koda failiem, kas pieejami Java projektā.

## Java faila formāts

Java faili tiek glabāti kā teksta faili, un tos var atvērt ar jebkuru teksta redaktoru. Java failu tipi parasti ir aizstāti ar Java faila tipu, un tos parasti izmanto Java programmu rakstīšanai. Java kodus var rakstīt ar IDE, piemēram, Eclipse un IntelliJ Idea, kas atvieglo darbu ar Java failiem.

### Java faila formāta piemērs
