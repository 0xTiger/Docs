## Kas ir CC fails?

CC fails ir avota koda faila veids, kas satur C++ programmēšanas valodā rakstītu kodu. Tajā var būt vai nu viss kods, kas nepieciešams vienai programmai, vai daļa no koda, kas ir daļa no lielāka programmēšanas projekta, kas sastāv no vairākiem CC failiem. Lai modificētu vai skatītu CC failā ietverto kodu, izstrādātāji bieži izmanto pirmkoda redaktoru. Pēc rediģēšanas viņi apkopo kodu, izmantojot kompilatoru, lai pārvērstu to mašīnkodā, ko datora operētājsistēma var izpildīt, lai palaistu programmu.

## CC faila formāts — vairāk informācijas

.cc faila paplašinājums ir izmantots C++ pirmkoda failiem vairākus gadu desmitus. C++ valodu 1970. gadu beigās izstrādāja Bjarne Stroustrup, kurš tajā laikā strādāja Bell Labs. Stroustrup vēlējās paplašināt C programmēšanas valodu ar atbalstu objektorientētām programmēšanas koncepcijām, kuras, viņaprāt, ir būtiskas lielu programmatūras sistēmu veidošanā.

Pirmā C++ versija tika izlaista 1983. gadā, un tā ātri ieguva popularitāti izstrādātāju vidū, kuri novērtēja tās jaudīgās funkcijas un elastību. Tā kā C++ kļuva arvien plašāk izmantots, kļuva nepieciešams izveidot kodēšanas konvencijas un failu nosaukšanas konvencijas, lai palīdzētu organizēt un pārvaldīt lielas kodu bāzes.

Google's C++ Style Guide which recommends using .cc file extension for C++ source code files, was first published in 2003. Kopš tā laika rokasgrāmata ir periodiski atjaunināta, lai atspoguļotu izmaiņas C++ valodā un plašākas programmēšanas kopienas attīstības praksi.

Mūsdienās .cc faila paplašinājums joprojām ir populāra izvēle C++ pirmkoda failiem, un to atpazīst daudzi populāri izstrādes rīki un platformas.

## Kā atvērt CC failu?

CC failu atvēršanai un rediģēšanai var izmantot daudzus pirmkoda redaktorus, tostarp vairāku platformu rīkus, piemēram, Microsoft Visual Studio Code un Code Blocks. Šie redaktori piedāvā noderīgas funkcijas, piemēram, sintakses izcelšanu un automātisko pabeigšanu, padarot tos par lielisku izvēli darbam ar CC failiem.

Varat arī atvērt CC failus, izmantojot jebkuru teksta redaktoru, piemēram, Notepad, Notepad++ utt.

## CC kompilēšana ar C++ kompilatoriem

CC faila kompilēšana ar C++ kompilatoru ietver vairākas darbības. Pirmais solis ir instalēt saderīgu C++ kompilatoru savai operētājsistēmai un izstrādes videi. Ir pieejami daudzi dažādi kompilatori, tostarp GCC, Clang un Microsoft Visual C++.

Kad esat instalējis C++ kompilatoru, nākamais solis ir atvērt termināli vai komandu uzvedni un pāriet uz direktoriju, kurā atrodas CC fails. Šeit varat palaist kompilatoru, izpildot kompilatora nodrošināto komandrindas rīku. Īpašā komanda CC faila kompilēšanai var atšķirties atkarībā no izmantotā kompilatora. Piemēram, lai apkopotu CC failu, izmantojot GCC, jums vajadzētu palaist komandu, kas līdzīga:

```
g++ -o myprogram myfile.cc
```
