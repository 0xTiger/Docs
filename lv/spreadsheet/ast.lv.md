## Kas ir AST fails?

AST fails ir izklājlapas veidnes fails, kas izveidots, izmantojot programmatūru Ability Spreadsheet, kas ir Ability Office komplekta izklājlapu komponents. Šie veidņu faili tiek izmantoti, lai izveidotu sākotnējo izklājlapu veidņu kopu, ko var izmantot, lai izveidotu [AWS](/spreadsheet/aws/) failus. AST failus var atvērt, izmantojot Ability izklājlapu programmatūru.

## AST faila formāts — vairāk informācijas

AST files are saved to disc as binary files and can be opened in Ability Spreadsheet software. Though fully compatible with Microsoft XLS and XLSX file formats, AST file format specifications are not available publicly for developer's reference. AST files can be saved to disc in [PDF](/pdf/) as well as [XPS](/page-description-language/xps/) file formats.
