## Kas ir XLR fails?

A file with .xlr extension is a Microsoft Works Spreadsheet file format that stored data in rows and columns. Microsoft Works was developed for use as an initial productivity suite and remained functional from 1987 to 2009. Tas ietvēra tekstapstrādes programmu, izklājlapu un datu bāzes pārvaldības sistēmu. Vēlāk tas tika pārtraukts pēc tam, kad tika ieviests Microsoft Office ar bagātīgākām funkcijām.

## XLR faila formāts — vairāk informācijas

XLR faili tika saglabāti kā bināri faili, un informācija par faila formāta specifikācijām nebija publiski pieejama. Tāpēc izstrādātājiem nav iespējas analizēt tā failu struktūru un rakstīt lietojumprogrammas šo failu lasīšanai/rakstīšanai programmatiski. Works Spreadsheet izmanto to pašu Excel faila formātu, lai saglabātu datus .xlr failos.
