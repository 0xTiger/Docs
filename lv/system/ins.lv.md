## Kas ir INS fails?

Failu ar paplašinājumu .ins parasti izmanto sistēma Microsoft Windows, lai iestatītu iezvanes un platjoslas interneta savienojumus. Faktiski tajā ir informācija par savienojuma iestatīšanu, kas ļauj sistēmai Windows iestatīt interneta piekļuvi ISP. INS fails parasti tiek izmantots, lai iestatītu Internet Explorer (IE) LAN savienojuma iestatījumus. IE savienojuma iestatījumu faili, ko ISP un sistēmas administratori dažreiz nodrošina lietotājiem ar INS faila URL.

## INS faila formāts
Varat izmantot interneta iestatījumu failus (INS) kopā ar Internet Explorer administrēšanas komplektu 11 (IEAK 11), lai konfigurētu savu pielāgoto pārlūkprogrammu un tās komponentus. Varat rakstīt dažādas pielāgotās pakotnes versijas, pielāgojot INS faila kopijas.

### Iespējamie INS iestatījumi
Iespējamie INS iestatījumi ir norādīti šajā tabulā:

| Iestatījums | Apraksts |
-----|---------|
| URL | Vai izmantot automātiski konfigurētu starpniekserveri. |
| Zīmols | Pielāgojiet zīmola un iestatīšanas informāciju pārlūkprogrammas pakotnē. |
| Plašsaziņas līdzekļi | Multivides veidi, kuros ir pieejama jūsu pielāgotā instalācijas pakotne. |
| Pārlūka rīkjoslas | Pielāgojiet IE rīkjoslas izskatu. |
| Kabīnes parakstīšana | Ciparparaksta informācija jūsu programmām. |
| HideCustom | Vai slēpt globāli unikālo identifikatoru (GUID) katram pielāgotajam komponentam. |
| Savienojuma iestatījumi | Informācija par tīkla savienojuma iestatījumiem, kas izmantoti pielāgotās pakotnes instalēšanai. |
| CustomBranding | URL atrašanās vieta jūsu zīmola kabineta (.cab) failam. |
| ExtRegInf | Iestatīšanas informācijas (.inf) failu nosaukumi un komponentu instalēšanas režīms. |
| IzlaseEx | Pievienojiet ceļu savam izlases ikonas failam, izlemiet, vai izlase ir pieejama bezsaistē, un pievienojiet vietrāžus URL katrai izlases vietnei. |
| ISP_Drošība | Saknes sertifikāts, ko pievienojat savai pielāgotajai pakotnei. |
| Starpniekserveris | Vai izmantot starpniekserveri. |
| Drošības imports | Vai importēt pielāgotās pakotnes drošības informāciju. |
