## Kas ir HTT fails?

Hiperteksta veidne (HTT) ir faila paplašinājums, ko Microsoft Internet Explorer izmantoja, lai izveidotu dinamiskas tīmekļa veidnes. HTT fails satur HTML un skripta koda kopu, ko var izmantot, lai dinamiski ģenerētu tīmekļa lapas, pamatojoties uz lietotāja ievadi vai citiem parametriem.

HTT faili būtībā bija teksta faili ar HTML un skripta koda kombināciju, kas parasti tika rakstīti VBScript. HTT faila formāts tika izstrādāts, lai ļautu tīmekļa izstrādātājiem izveidot tīmekļa lapu veidnes, kuras var viegli pielāgot un pārveidot, neprasot plašas zināšanas par HTML vai skriptu valodām.

Tomēr tagad HTT faila formāts tiek uzskatīts par novecojušu, un mūsdienu tīmekļa pārlūkprogrammas to vairs neatbalsta. Tā vietā izstrādātāji tagad izmanto progresīvākus tīmekļa izstrādes rīkus un tehnoloģijas, piemēram, servera puses skriptu valodas, piemēram, PHP, ASP.NET vai JavaScript ietvarus, piemēram, React vai Vue.js, lai izveidotu dinamiskas tīmekļa lapas.

## HTT faila formāts — vairāk informācijas

2. Skripta kods: HTT faila formāts atbalsta VBScript — skriptu valodu, ko var izmantot, lai pievienotu dinamisku funkcionalitāti veidnes HTML elementiem. Skripta kods var ietvert funkcijas lietotāja ievades apstrādei, datu apstrādei vai mijiedarbībai ar citām tīmekļa tehnoloģijām.

To var izmantot, lai izveidotu dinamiskas tīmekļa lapas, kas pielāgo to saturu, pamatojoties uz kontekstu vai lietotāja vēlmēm.
