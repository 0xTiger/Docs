## Kas ir ADMX fails?

An ADMX file is a policy configuration file used by Microsoft Windows Group Policy software that manages group of computers in a group. It is an XML-based administrative template file and was introduced with Windows Vista Service Pack 1. ADMX faili satur konfigurācijas iestatījumus lietotāju kontiem, operētājsistēmas konfigurācijām un lietojumprogrammām. ADMX faili tiek izmantoti, lai saglabātu un izvietotu konfigurācijas daudzu datorsistēmu centralizētai pārvaldībai.

Varat izveidot vai rediģēt ADMX failus, izmantojot jebkuru ar XML saderīgu, jebkuru teksta redaktoru vai Microsoft grupas politikas objektu redaktoru.

## ADMX faila formāts — vairāk informācijas

ADMX faili tiek glabāti universālā [XML](/web/xml/) faila formātā, kas ir cilvēkiem lasāms. Tas ir daudzvalodu faila formāts un ļauj dažādu valodu sistēmas administratoriem strādāt ar tiem pašiem ADMX failiem. ADMX faili aizstāja veco [ADM](/system/adm/) faila formātu, kas ir unikoda formāta teksta faila formāts. ADMX faili norāda reģistra atslēgas, kas tiek mainītas, mainot noteiktu grupas politikas iestatījumu.

### Ko es varu darīt ar ADMX failu?

ADMX faili nosaka, kādas darbības ir jāatļauj lietotāju datoriem, kas ir daļa no grupas. Grupas politika nosaka noteikumus, kas iestatīti kopā ar Active Directory programmatūru. ADMX faili tiek pārvaldīti no centrālā veikala operētājsistēmā Windows OS, kas ir galvenā domēna atrašanās vieta.

ADMX fails, kas izvietots darba vidē, var ļaut administratoriem īstenot tādas politikas kā:

 * Kontrolējiet interneta lietošanu
 * Noteiktu veidu failu lejupielāde
 * Noņemamo ierīču izmantošana sistēmās
