## Kas ir MPP fails?

Fails ar paplašinājumu .mpp ir Microsoft Project datu fails, kurā integrētā veidā tiek glabāta ar projektu pārvaldību saistīta informācija. Microsoft ir izstrādājusi šo patentēto faila formātu, lai padarītu to saderīgu ar Microsoft Project (MSP), viņu projektu pārvaldības programmatūras produktu. Izņemot MPP, MSP var strādāt ar projekta XML shēmu. Daudzas lietojumprogrammas un API ļauj MPP pārveidot citos failu formātos. Microsoft tagad ir tiešsaistes Project Server, kurā projektu pārvaldības failus var augšupielādēt vairāku lietotāju sadarbībai.

## MPP faila formāts ##

Microsoft izstrādāja un publicēja MPP kā bināro failu formātu, un tā specifikācijas Microsoft līdz šim nav publiskojis. Tādējādi iekšējā faila struktūra un informācija izstrādātājiem nav pieejama. Tiešsaistē ir pieejamas vairākas API, kas atbalsta MPP failu lasīšanu/rakstīšanu, taču 100% precizitāte nekad netiek sasniegta. Laika gaitā korporācija Microsoft savā projektu pārvaldības lietojumprogrammā ir ieviesusi vairākus līdzekļus, kā arī MPP faila formāta līdzekļus.

MPP fails izmanto šādus MIME veidus:

* lietojumprogramma/vnd.ms-projekts

* lietojumprogramma/msproj

* lietojumprogramma/msproject

* lietojumprogramma/x-msproject

* lietojumprogramma/x-ms-projekts

* lietojumprogramma/x-dos_ms_project

* lietojumprogramma/mpp

* zz-application/zz-winassoc-mpp


## Vai ir problēmas, atverot MPP failu? ##

Šeit ir saraksts ar dažām izplatītām problēmām, kas var rasties un izraisīt MPP formāta nepareizu darbību:

 *   Atbalsta programmatūras trūkums
 *   Bojāts fails
 *   Inficēts fails vīrusa dēļ
 *   Sistēmā nav piekļuves tiesību, lai atvērtu failus
 *   Novecojis disks jūsu sistēmā
 *   Faila paplašinājums tiek pārdēvēts
