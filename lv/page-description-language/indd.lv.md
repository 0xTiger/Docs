## Kas ir INDD fails?

Fails ar paplašinājumu .indd ir Adobe InDesign projekta fails, kas satur projekta lapas izkārtojuma informāciju. INDD apzīmē InDesign Document. Šajos failos parasti tiek glabāta lappušu formatēšanas informācija, grāmatas, žurnāli, laikraksti, skrejlapas, brošūras vai citi drukātie mediji, piemēram, mobilo ierīču un tīmekļa lapu maketi. INDD faila tips ir Adobe patentēts faila formāts, un tas ir noklusējuma faila formāts programmai InDesign. Grafiskie dizaineri, žurnālisti, mārketinga speciālisti un izdevēji, kas strādā ar InDesign, izmanto šo faila tipu.

## Vairāk par INDD faila formātu

INDD faili ir vektora faili, kurus var mērogot, nezaudējot izšķirtspēju. Tie ir balstīti uz sarežģītiem grafiem līdzīgiem, kurus var bezgalīgi paplašināt. Tajā ir apkopota visa informācija par fontiem un izkārtojumiem, lai tos kopīgotu ar komandu.

INDD faili ir lieliski piemēroti, lai izveidotu saistošu drukas, kā arī tiešsaistes saturu. INDD fails var aptvert vairākas lapas un var kalpot kā veidnes lapu, nodaļu un publikāciju izveidei. Kad jūsu projekts ir pabeigts, to var publicēt tiešsaistes plašsaziņas līdzekļos bez pārveidošanas, zipkopēšanas vai nosūtīšanas pa e-pastu.

## Vai INDD failu var konvertēt citos formātos?

INDD files can be converted to several other file formats. The most common conversion is INDD to PDF. InDesign lets you save the INDD file to PDF for sharing with other users. It also lets you export INDD file to JPG, [EPS](/page-description-language/eps/), [EPUB](/ebook/epub/), [SWF](/page-description-language/swf/), FLA, [HTML](/web/html/), [XML](/web/xml/), and IDML.

## Kā tiek izveidots INDD fails?

An Indd file is created when you create a new project with [Adobe InDesign](https://www.adobe.com/products/indesign.html) and save it to disc. You can open the Indd project file, make changes to it, and save the updated file for the changes.
