## Kas ir CHCK fails?

CHCK fails tiek izveidots Microsoft Exchange Server veselības uzraudzības pārbaužu rezultātā. Tajā ir informācija par servera datu bāzes vispārējo statusu un darbību. Microsoft Exchange Server galvenokārt tiek izmantots kā e-pasta serveris, un tas ļauj uzturēt biznesa kontaktus un kalendārus. Exchange servera administratori izmanto tā EseUtil, lai pārbaudītu datu bāzes stāvokli.

## CHCK faila formāts

CHCK faila iekšējais faila formāts nav zināms, un nav pieejams atsauces materiāls CHCK failu lasīšanai/rakstīšanai.
