## Kas ir FP7 fails?

A file with .fp7 extension is a database file created with FileMaker Pro versions 7-11. Tas saglabā datus tabulās ar ierakstiem, kur katrs ieraksts satur vienu vai vairākus laukus. FP7 failiem ir iespēja uzglabāt lielus datus, tostarp personas datu bāzu datus un organizatoriskos ierakstus. Jaunāka FileMaker Pro versija ir [FMP12](/database/fmp12/), kas tika ieviesta, izlaižot FileMake Pro versiju 12. Papildus teksta datiem FP7 failā var būt arī veidlapas, diagrammas un citi datu bāzes objekti. FP7 failus var atvērt, izmantojot programmu FileMaker Pro Advanced.

## FP7 faila formāts

FP7 faili tiek saglabāti diskā kā bināri faili FileMaker patentētā faila formātā. FP7 faila formāta iekšējā faila formāta informācija nav publiski pieejama.
