## Kas ir ADP fails?

A file with .adp extension is a Microsoft Access project file that uses OLE DB component architecture to provide a direct and efficient connection to a Microsoft SQL Server database. Such files do not contain the actual tables, database diagrams or other database elements. ADP files can be created with Access 2007 and 2010. Failus, kas izveidoti ar vecākām Access versijām, var atvērt arī programmā Access 2007 un 2010. ADP failus var atvērt, izmantojot Microsoft Access 365.

## ADP faila formāts — vairāk informācijas

Access projekts, kas izveidots kā ADP, ļauj veikt dizaina izmaiņas SQL servera objektos, piemēram, tabulās un skatos. Tas arī ļauj izveidot, rediģēt un izmantot SQL līdzekļus, piemēram, datu bāzes diagrammas, saglabātās procedūras un lietotāja definētas funkcijas. Tas dod priekšrocības salīdzinājumā ar saiti uz SQL servera datu bāzi, kur nevar veikt izmaiņas dizainā un var izveidot saiti tikai uz SQL servera tabulām un skatiem.

### Datu tipu un diagrammu atbalsts

**Datuma/laika datu veidi**

Programmā Access 2010 tiek atbalstīti šādi jaunie datuma/laika datu veidi.

 * LAIKS
 * DATUMS
 * DATUMSTIME2
 * DATUMSTIMEOFFSET

**Mainīga garuma datu veidi**

Programmas Access 2010 projektos var izmantot šādus mainīga garuma datu tipus:

 * VARBIN (MAX)
 * VARČARS (MAX)
 * NVARCHAR(MAX)
