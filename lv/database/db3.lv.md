## Kas ir DB3 fails?
DB3 fails ir datu bāzes fails, ko izveido programmatūra SQLite, kas ir viegla, autonoma datu bāzes programma, kas izveido datu bāzes, izmantojot vienkāršus failus; satur datu bāzes anatomiju, kā arī datu ierakstus; parasti izmanto strukturētu datu izgūšanai vai glabāšanai, izmantojot SQL. Šos failus var izmantot viedierīcēs vai vietās, kur nepieciešama lietvedība vai cita datu pārvaldība, bet maz vietas vidē.


## DB3 faila formāts
DB3 faila formāts ir saistīts ar RDBMS (relāciju datu bāzes pārvaldības sistēma) SQLite, kas ir populāra iegultās datu bāzes izvēle. SQLite_3 specifikācijā nav definēts neviens īpašs faila paplašinājums, tas var izmantot paplašinājumus, tostarp [.dbf](/database/dbf/) un [.sqlite](/database/sqlite/).

### Datu bāzes klasifikācija
Commonly SQLite, version 3 related db file format can be considered as the DB3 file format used as the publicly documented native format for the SQLite database engine since June 2004. DB3 faila formāts ir vairāku platformu, ko var pārnest starp lielām un mazām arhitektūrām vai 32 bitu un 64 bitu sistēmām. Šīs funkcijas padara DB3 par populāru lietojumprogrammas faila formāta izvēli. Galvenais SQLite_3 datu bāzes (DB3) fails sastāv no vienas vai vairākām lapām. Visi visu lapu izmēri vienā datu bāzē ir vienādi. Lapas lielums baitos ir divi no 512 līdz 65536 ieskaitot.
