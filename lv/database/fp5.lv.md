## Kas ir FP5 fails?

An FP5 file is a database file created with FileMaker Pro version 5 or 6. Tas saglabā lietotāju, kā arī organizācijas datus ierakstu veidā tabulās. FileMaker Pro ir datu bāzes pārvaldības sistēma, kas atbalsta Macintosh un Windows operētājsistēmas. FP5 ir vecāks faila formāts, ko atbalstīja tobrīd pieejamā FileMaker Pro versija. Jaunākais FileMaker Pro datu bāzes formāts ir [FMP12](/database/fmp12/), kas tika ieviests, uzsākot FileMaker Pro v12. Cits labi izmantots datu bāzes faila formāts bija [FP7](/database/fp7/), kas tika aizstāts ar FMP12.

## FP5 faila formāts — vairāk informācijas

FP5 faila formāta iekšējā faila formāta informācija nav publiski pieejama izstrādātāja uzziņai. Vecāki FileMaker Pro datu bāzes failu formāti, piemēram, [FP5 and FP7 can be upgraded to latest FMP12](https://fmhelp.filemaker.com/help/16/fmp/en/index.html#page/FMP_Help/converting-files.html), izmantojot detalizētās FileMaker Pro vadlīnijas.
