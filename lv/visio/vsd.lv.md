## Kas ir VSD fails?

VSD faili ir rasējumi, kas izveidoti ar Microsoft Visio lietojumprogrammu, lai attēlotu dažādus grafiskos objektus un to savstarpējo savienojumu. Šādos rasējumos var būt vizuāli objekti, piemēram, vizuālie objekti, blokshēmas, UML diagramma, informācijas plūsma, organizatoriskās diagrammas, programmatūras diagrammas, tīkla izkārtojums, datu bāzes modeļi, objektu kartēšana un cita līdzīga informācija. Microsoft Visio piedāvā iespēju pārveidot Visio failus dažādos failu formātos, tostarp [PNG](/image/png/), [BMP](/image/bmp/), [PDF](/pdf/) un citos.

# VSD faila formāts #

VSD file format is the initial file format introduced by Microsoft Visio for storing drawing elements. The internal structure of VSD files is based on binary file format unlike [.VSDX](/visio/vsdx/) that was introduced with Microsoft Visio 2013. Jaunais formāts ir balstīts uz Office Open XML faila formātu un glabā projekta informāciju [ZIP](/compression/zip/) arhīvā. Microsoft piedāvā lietojumprogrammu vviewer.exe Visio zīmējumu failiem gadījumā, ja Visio nav instalēts sistēmā.
