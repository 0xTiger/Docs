## Kas ir JFPROJ fails?

A file with .jfproj extension is a JSON based custom font created with Microsoft Font Maker application on Windows 10. Tos var izveidot, izmantojot elektronisku pildspalvu ar skārienekrāna ierīcēm, piemēram, planšetdatoriem, lai gan datoru gadījumā var izmantot arī citas datoru perifērijas ierīces, piemēram, peli. JFPROJ faili satur glifus, kas tiek izmantoti, lai attēlotu rakstzīmes ekrānā, nosakot atstarpi starp rakstzīmēm un vārdiem, kā arī fonta lielumu. Programma Microsoft Font Maker ir pieejama lejupielādei no Windows veikala.
