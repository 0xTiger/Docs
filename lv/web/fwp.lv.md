## Kas ir FWP fails?

Fails ar .fwp failu ir arhīva fails, kas izveidots ar Microsoft Expression Web un satur visus dizaina failus un mapes. Microsoft Expression Web ir Microsoft izstrādāts vispārīgs tīmekļa dizaina programmatūras produkts, kura pamatā ir [HTML](/web/html/) redaktors. Tas ir pārtraukts kopš 2012. gada, un Microsoft to var bez maksas lejupielādēt. FWP failus var eksportēt un saglabāt kā Microsoft .cab formātu.
