## Kas ir BROWSER fails?

Failu ar paplašinājumu .browser izmanto ASP.NET tīmekļa lietojumprogrammas, lai noteiktu tīmekļa pārlūkprogrammu iespējas un konfigurāciju. Tajā ir informācija par pārlūkprogrammas iespējām, ierobežojumiem un īpašībām. Šīs īpašības palīdz ASP.NET sistēmai atpazīt pieprasīto pārlūkprogrammu un attiecīgi parādīt tīmekļa lapu lietotājiem.

## BROWSER Faila formāts

BROWSER definīcijas failus izmanto ASP.NET lietojumprogramma, lai noteiktu pārlūkprogrammas iespējas. Tā ir šajā failā ietvertā informācija, ko serveris izmanto, lai ģenerētu marķējumu un citus resursus, kas ir saderīgi ar pieprasījuma iesniedzēju pārlūkprogrammu.

BROWSER faili tiek saglabāti vienkārša teksta faila formātā. BROWSER failu var atvērt teksta redaktoros, piemēram, Notepad, Notepad++, Apple TextEdit un Visual Studio IDE.

### BROWSER faila formāta faila struktūra

BROWSER definīcijas faili izmanto hierarhisku struktūru. Katrs .browser fails satur XML marķējumu, kas ietver konkrētas pārlūkprogrammas vai pārlūkprogrammu grupas iespējas un līdzekļus. Šī informācija ietver tādas īpašības kā aģenta virkne, versiju numuri un atbalsts tādām tehnoloģijām kā JavaScript, CSS vai citi HTML elementi. Izmantojot šos pārlūkprogrammas definīcijas failus, ASP.NET izstrādātāji pielāgo lietotāju pieredzi, pamatojoties uz dažādu pārlūkprogrammu iespējām.
