## Kas ir EDGE fails?

Fails ar paplašinājumu .edge ir Adobe Edge Project fails, kas tiek izveidots, kad projekts tiek izveidots un saglabāts ar Adobe Edge. Adobe Edge ļauj tīmekļa dizaineriem izveidot interaktīvus [HTML](/web/html/) animatorus tīmeklī, digitālajai publicēšanai un bagātinātas multivides reklāmām gan galddatoru, gan mobilajām pārlūkprogrammām. Tas tiek panākts, izmantojot HTML/HTML5, [JavaScript](/web/js/) un [CSS3](/web/css/). Edge faili tiek ģenerēti kā vienkārša teksta faili, un tos var atvērt ar vairākām dažādām lietojumprogrammām, piemēram, Microsoft Notepad, Apple TextEdit un Notepad++.

## EDGE faila formāts

EDGE faili tiek saglabāti vienkārša teksta formātā un izmanto jQuery, lai iespējotu animāciju. EDGE animācijas alternatīva bija Flash faili, kas vairs netiek atbalstīti. EDGE fails tiek saglabāts kopā ar HTML, JavaScript un CSS failiem. Atverot EDGE failu, tiek atvērta tīmekļa lapa, ti, HTML fails.
