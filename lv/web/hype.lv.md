## Kas ir HYPE fails?

A HYPE file is a project file saved using the Hype software that is used for creation of web content with HTML5. Tajā ir ietverti tīmekļa elementi, piemēram, grafika, teksts, animācija, ainas informācija un citi elementi, kas kopā veido saturu. Tīmekļa saturu HYPE failā var eksportēt uz tīmekļa failiem, kurus var publicēt tīmekļa lapā.

HYPE failu var atvērt, izmantojot [Tumult Hype](https://tumult.com/hype/) programmatūru.

## HYPE faila formāts

HYPE faili tiek glabāti patentētā failu formātā, un to iekšējā informācija nav publiski zināma. HYPE faila saturu var skatīt tikai tad, kad tas ir atvērts programmatūrā Tumult Hype.

### HYPE faila saturs

HYPE fails tiek izveidots, kad izmantojat Hype lietojumprogrammu, lai izveidotu tīmekļa saturu dažādiem mērķiem. Tajā tiek glabāta informācija par atslēgkadru animācijām, lai izveidotu tādu saturu kā e-kartes, e-grāmatas, prezentācijas, reklāmas, infografikas un tīmekļa lapu materiāli. To saturu nodrošina HTML5, kas ļauj rādīt saturu ierīcēs, sākot no galddatoriem līdz iPhone un iPad. Tumult Hype izvada jaunāko HTML5, [CSS3](/web/css/) stilus un [JavaScript](/web/js/).
