## Kas ir WEBLOC?

WEBLOC fails ir saīsnes fails operētājsistēmā macOS, kas norāda uz tīmekļa vietrādi URL internetā. Tas tiek ģenerēts, velkot tīmekļa URL no pārlūkprogrammas adreses joslas uz darbvirsmu vai jebkuru citu mapi, kurā tas ir paredzēts saglabāšanai. URL uz galamērķa vietni tiek saglabāts šajā saīsnes failā, un to var skatīt, atverot failu teksta redaktorā, piemēram, Microsoft Notepad, Notepad++ vai Apple TextEdit. WEBLOC faili ir līdzīgi [URL](/web/url/) failiem.

## WEBLOC faila formāts — vairāk informācijas

WEBLOC faili tiek glabāti vienkārša teksta faila formātā. Tos var ģenerēt operētājsistēmā macOS, velkot vietrāžus URL no tīmekļa pārlūkprogrammām, piemēram, Safari, Google Chrome vai Firefox. Ja vienlaikus jāatver vairāki WEBLOC faili, būs nepieciešams ātrāks un vienkāršāks risinājums to atvēršanai, izmantojot ārējās lietojumprogrammas vai API. Šīs API var lasīt un pārveidot WEBLOC uz PDF un JPEG.
