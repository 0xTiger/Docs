## Kas ir DISCOMAP fails?

DISCOMAP fails ir izvades fails, ko izveido Microsoft Web Services Discovery Tool **Disco.exe**. Tajā ir ietverti rezultāti, kas ir ASP.NET Web Services, kas atklāti noteiktā URL. Parasti šādu atklāto pakalpojumu piemēri ir .DISCO vai .WSDL failu atrašanās vietas. Kad darbojas Microsoft Web Discovery rīks, tas meklē informāciju, pamatojoties uz DISCO failā atrasto XML saturu. Rezultāti tiek saglabāti DISCOMAP failā, ko var atvērt, izmantojot Microsoft Visual Studio IDE.
