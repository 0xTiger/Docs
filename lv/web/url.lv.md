## Kas ir URL fails?

Fails ar paplašinājumu .url ir vietnes URL saīsne, ko var saglabāt datorā ar jebkuru vietnes adresi. Kad tas tiek atvērts, veicot dubultklikšķi, vietrādis URL atvērs vietni noklusējuma pārlūkprogrammā un lapa tiks parādīta lietotājam. URL faili parasti tiek saukti par saīsņu failiem uz vietnēm, un tos attēlo noklusējuma pārlūkprogrammas ikona. URL failus var atvērt un rediģēt jebkurā teksta redaktorā, piemēram, Microsoft Notepad, Notepad++ vai Apple TextEdit.

## URL faila formāts — vairāk informācijas

URL faili tiek saglabāti vienkārša teksta formātā, un tos var atvērt un skatīt jebkurā teksta redaktorā. Tie tiek atvērti noklusējuma tīmekļa pārlūkprogrammā, kurā lietotājs veic citu pārlūkošanu.
