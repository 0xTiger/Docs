## Kas ir WEBSITE fails?

A WEBSITE file is a website shortcut file created with Microsoft Internet Explorer (IE). It contains a URL and an icon (ICO) file that points to the website's favicon. The icon is used for displaying the shortcut. WEBSITE file format was introduced with the release of IE version 9. Iepriekšējās IE versijas tika izmantotas, lai saglabātu saīsnes uz tīmekļa lapām kā [.url](/web/url/) failus.

## Tīmekļa vietne Faila formāts

Vietnes fails tiek saglabāts diskā kā vienkārša teksta faili, kuru saturu var skatīt jebkurā teksta redaktorā. Papildus citam saturam tajā ir ietverts mērķa URL, kuram tas ir saīsne. Ja vēlaties atvērt mērķa vietni citā pārlūkprogrammā, nevis IE, varat kopēt šo URL un ielīmēt to citās pārlūkprogrammās, piemēram, Google Chrome, Safari vai Firefox, lai skatītu vietni.

## Kā izveidot .WEBSITE failu?

Vietnes failu var izveidot pārlūkprogrammā IE, adreses joslā noklikšķinot uz URL un velkot to uz vēlamo vietu datorā, piemēram, uz darbvirsmu vai dokumentiem. Tas izveido saīsni uz šo interneta atrašanās vietu, un to var atvērt, veicot dubultklikšķi uz saīsnes. Ja vēlaties saglabāt saīsni kā URL failu, turiet nospiestu taustiņu SHIFT, pirms velciet mērķa vietnes izlases ikonu uz darbvirsmu.
