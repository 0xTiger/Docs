## Kas ir CPHD fails?

CPHD fails ir hiperteksta dokuments, ko izmanto CloudChan pirmsprocesora programma. Tas satur HTML un skriptus, lai izveidotu dinamiskas tīmekļa lapas un dokumentus. CPHD failā esošie skripti atgādina Java izmantoto sintakse. CPHD tīmekļa lapa atgriež HTML dokumentu, kad tas tiek izpildīts CloudChan procesos. CPHD formāts var darboties ar jebkuru XML dokumentu.

## CPHD faila formāts

CPHD faili tiek saglabāti diskā kā teksta faili. Tas ļauj atvērt šos failus jebkurā teksta redaktorā, piemēram, Notepad vai Notepad++.
