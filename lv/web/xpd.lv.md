## Kas ir XPD fails?

XPD fails ir tīmekļa lapas fails, kas izveidots no XML dokumenta. Biežāk tas tiek ģenerēts, izmantojot XSLT transformāciju. [XPD file format specifications](https://www.w3.org/TR/xml-pipeline/) ir pieejami W3, kas nosaka veidu, kā palielināt vairāku procesu [XML](/web/xml/) lietojumprogrammu savietojamību. XML konveijera valoda nosaka apstrādes attiecības starp XML resursiem.
