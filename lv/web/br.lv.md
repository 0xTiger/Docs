## Kas ir BR fails?

A BR file is a compressed web file generated by applying the open source data compression algorithm, **Brotli**. It is used to store webpage assets such as stylesheets ([CSS](/web/css/)), images ([SVG](/page-description-language/svg/)), [XML](/web/xml/), and scripting files ([JS](/web/js/)). Modern day websites, such as Chrome and Firefox, use BR files to reduce the page loading time, resulting in better user experience.

## BR faila formāts

BR faili ir saspiesti tīmekļa faili, kas tiek ģenerēti, izmantojot Brotli saspiešanas algoritmu. Brotli saspiešana ir bezzudumu datu saspiešanas algoritms, un Google to izstrādāja Zopfli saspiešanas algoritmam. Tas ir balstīts uz LZ77 bezzudumu saspiešanas algoritma un Hafmena kodēšanas kombināciju.

Tā mazā izmēra dēļ BR failus izmanto tīmekļa serveri un satura piegādes tīkli (CDN). Šiem serveriem nosūtītie pieprasījumi izraisa HTTP satura saspiešanu, padarot vietnes ātrāk ielādētas. Brotli ir kļuvis populārāks un nodrošina labāku saspiešanu nekā gzip.
