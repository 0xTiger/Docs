## Kas ir CHM fails?

CHM faila formāts ir Microsoft **[HTML](/web/html/)** palīdzības fails, kas sastāv no HTML lapu kolekcijas. Tas nodrošina rādītāju, lai ātri piekļūtu tēmām un pārvietotos uz dažādām palīdzības dokumenta daļām. CHM faila saturu var meklēt, izmantojot piedāvāto meklēšanas opciju. CHM ir Microsoft patentēts tiešsaistes palīdzības faila formāts, ko bieži izmanto programmatūras dokumentācijai. Turklāt to izmanto vairākās citās lietojumprogrammās, piemēram, apmācību ceļvežos, interaktīvās grāmatās un elektroniskajos biļetenos. Lielākā daļa mūsdienu Microsoft izstrādes vidi atbalsta CHM dokumentācijas ģenerēšanu no lietojumprogrammā pieejamās informācijas.

CHM faila formāta unikālā spēja ieviest kombinētu satura rādītāju un indeksu padara to atšķirīgu no citām standarta HTML lapām. Ģenerētais CHM fails ir salīdzinoši mazs, un tāpēc to var viegli izplatīt ar programmatūras pakotnēm. Palīdzības autorēšanas rīks HTML palīdzības darbnīca nodrošina viegli lietojamu sistēmu palīdzības projektu un ar tiem saistīto failu izveidei un pārvaldībai. CHM failos var būt teksts, attēli un hipersaites; skatāms tīmekļa pārlūkprogrammā; izmanto Windows un citas programmas kā tiešsaistes palīdzības risinājumu.

## CHM faila formāts

Ģenerētā CHM faila galīgā forma ir atkarīga no palīdzības sistēmas izveides un tā, vai tā ir paredzēta lietojumprogrammai vai tīmekļa vietnei. CHM fails atbalsta datu saspiešanu ar LZX saspiešanu, lai ģenerētu saspiestos HTML failus. Tajā ir iebūvēta meklētājprogramma ātrai satura meklēšanai, kā arī iespēja apvienot vairākus .CHM failus. CHM fails sastāv no HTML failu kopas, saistītā satura rādītāja un indeksa faila.

### HTML faili

Neatkarīgi no tā, vai veidojat palīdzības tēmas izplatīšanai ar programmu vai tīmeklī, jūsu izveidotie dokumenti tiek izveidoti, izmantojot īpašu formatēšanas valodu, kas pazīstama kā hiperteksta iezīmēšanas valoda (HTML). HTML tēmu failiem ir .htm vai .html faila nosaukuma paplašinājums.

Lai gan šķiet, ka katra jūsu izveidotā palīdzības tēma vai Web lapa ir dokuments ar tekstu, grafiku vai animētiem attēliem, .htm faili patiesībā ir teksta dokumenti, kuriem ir īpaši HTML formatēšanas kodi. Šie kodi, ko sauc par tagiem, norāda pārlūkprogrammai, kā parādīt katru lapu. Tikai teksts, kas parādās tēmā vai Web lapā, faktiski ir .htm failā. Jebkura grafika, skaņas, animēti attēli vai citi elementi, kas parādās, ir atsevišķi faili, uz kuriem norāda jūsu HTML fails. Pārlūkprogramma kopē vai lejupielādē grafiku, skaņas vai citus elementus, kad tā redz atzīmes, kas tai liek to darīt.

### Satura rādītājs (TOC)
Palīdzības satura rādītāja (.hhc) fails ir HTML fails, kurā ir satura rādītāja tēmu nosaukumi. Kad lietotājs atver satura rādītāju apkopotā palīdzības failā (vai tīmekļa lapā) un noklikšķina uz tēmas nosaukuma, tiks atvērts ar šo nosaukumu saistītais HTML fails.

### Indeksa fails
Indeksa (.hhk) fails ir HTML fails, kurā ir jūsu rādītāja indeksa ieraksti (atslēgvārdi). Kad lietotājs atver indeksu apkopotā palīdzības failā vai Web lapā un noklikšķina uz atslēgvārda, tiks atvērts ar atslēgvārdu saistītais HTML fails.

## HTML palīdzības komponenti

HTML palīdzība sastāv no vairākiem komponentiem. Tie ietver:

* `HTML Help Workshop`: a help authoring tool with an easy-to-use graphical interface for creating project files, HTML topic files, contents files, index files, and everything else you need to put together an online help system or Web site.
* `HTML palīdzības ActiveX vadīkla`: neliela, modulāra programma, ko izmanto, lai HTML failā ievietotu palīdzības navigācijas un sekundāro logu funkcionalitāti.

* `HTML palīdzības skatītājs`: pilnībā funkcionējošs un pielāgojams trīs paneļu logs, kurā var parādīties tiešsaistes palīdzības tēmas.

* `Microsoft HTML palīdzības attēlu redaktors`: tiešsaistes grafikas rīks ekrānuzņēmumu izveidei; kā arī attēlu failu konvertēšanai, rediģēšanai un apskatei.

* `HTML palīdzības Java sīklietotne`: maza, uz Java balstīta programma, ko var izmantot ActiveX vadīklas vietā, lai HTML failā ievietotu palīdzības navigāciju.

* `HTML palīdzības izpildāmā programma`: programma, kas parāda un palaiž palīdzību, kad noklikšķināt uz kompilētā palīdzības faila.

* `HTML palīdzības kompilators`: programma, kas apkopo projektu, saturu, indeksu, tēmu un citus failus kompilētā palīdzības failā.

* `The HTML Help Authoring Guide`: an online guide designed to assist help authors in using HTML Help to design a help system. The guide also contains a complete HTML Help reference for developers and an HTML tag reference for authors.
