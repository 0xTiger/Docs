## Kas ir VDISCO fails?

VDISCO fails ir atklāšanas fails, ko Microsoft Visual Studio izmanto savos tīmekļa pakalpojumos. Tas sniedz informāciju par pieejamajiem tīmekļa pakalpojumiem, piemēram, to vietrāžiem URL, nosaukumvietām un metodēm. VDISCO fails ir līdzīgs faila formātam [DISCO](/web/disco/), taču tiek ģenerēts izpildlaikā. Tas tiek saglabāts tīmekļa pakalpojuma lietojumprogrammā, un Visual Studio to izmanto, lai dinamiski atklātu un izmantotu tīmekļa pakalpojumus projektā. VDISCO faili tiek izmantoti kopā ar failu [.asmx](/web/asmx/), kurā ir faktiskais tīmekļa pakalpojuma kods.

Varat atvērt VDISCO failu jebkurā teksta redaktorā, piemēram, Microsoft Notepad, Github Atom un Apple TextEdit. To var arī atvērt, izmantojot Microsoft Visual Studio 2012 un jaunāku versiju, lai strādātu ar to.

## VDISCO faila formāts

VDISCO fails tiek saglabāts un mitināts [XML](/web/xml/) faila formātā, kas ir universāls formāts datu apkopošanai un publicēšanai. Tas satur pakalpojuma URL, nosaukumvietas un metodes standarta XML tagos, kur katrs tags satur attiecīgās informācijas vērtību.
