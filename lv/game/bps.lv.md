## Kas ir BPS fails?

A ".bps" file extension typically refers to a Binary Patch File. These files are used to apply patches or updates to existing files or software. Here is some more information about BPS files:

1. **Patch Files:** BPS files contain binary data that represents the changes needed to update or modify an existing file or program. Instead of replacing the entire file, patch files only include the differences between the original file and the updated version.

2. **ROM uzlaušana:** BPS faili parasti tiek izmantoti ROM uzlaušanas kontekstā. ROM hakeri izmanto BPS failus, lai izplatītu ielāpus, kas modificē spēles kodu vai saturu klasiskajos videospēļu ROM. Spēlētāji var lietot šos ielāpus saviem spēļu ROM, lai izjustu fanu veiktās modifikācijas un tulkojumus.

3. **Ielāpu lietojumprogramma:** Lai lietotu BPS ielāpu, ir nepieciešams ielāpu rīks. Populāras ielāpu utilītas, piemēram, Floating IPS vai Beat, ļauj piemērot BPS ielāpus attiecīgajam ROM failam, izveidojot modificētu spēles vai programmatūras versiju.

Nākamajās sadaļās mēs izpētīsim programmatūras lietojumprogrammas, kas saistītas ar BPS failiem.

## Peldošais IPS

Peldošais IPS (Flips) ir populāra un lietotājam draudzīga utilīta ielāpu uzlikšanai ROM failiem ROM uzlaušanas kontekstā. To parasti izmanto emulācijas un fanu tulkošanas kopienās. Tālāk ir sniegta papildu informācija par to.

- **Ielāpu lietojumprogramma:** peldošā IPS ir izstrādāta, lai ROM failiem lietotu ielāpu failus, parasti BPS formātā. Šie ielāpi satur bināros datus, kas atspoguļo izmaiņas sākotnējā ROM, piemēram, tulkojumus, kļūdu labojumus vai spēles izmaiņas.

- **Automātiskā ielāpēšana:** peldošā IPS automātiski nosaka mērķa ROM galveni un attiecīgi lieto ielāpu, vienkāršojot ielāpu procesu.

- **Kontrolsummu verifikācija:** rīks bieži ietver kontrolsummu pārbaudes līdzekli, lai nodrošinātu labotā ROM integritāti, samazinot kļūdu vai bojājumu risku.

- **Ielāpu izveide:** lai gan peldošo IPS galvenokārt izmanto ielāpu uzlikšanai, to var izmantot arī, lai izveidotu ielāpu failus (BPS formātā) no atšķirībām starp diviem ROM, padarot to noderīgu ROM uzlaušanas un tulkošanas projektos.

## MultiPatch

MultiPatch ir vēl viena utilīta, ko parasti izmanto ROM uzlaušanas kopienā, īpaši, lai lietotu ielāpus ROM failiem. Pēc funkcijas tas ir līdzīgs peldošajam IPS, taču tam ir savas funkcijas un saskarne. Tālāk ir sniegta papildu informācija par to.

- **Ielāpu lietojumprogramma:** MultiPatch ir paredzēts, lai lietotu ielāpus ROM failiem, parasti tādos formātos kā IPS, UPS vai BPS. Šie ielāpi satur bināros datus, kas modificē sākotnējo ROM, bieži vien tādiem nolūkiem kā tulkojumi, kļūdu labojumi vai pielāgojumi.

- **Atbalsts dažādiem ielāpu formātiem:** atbalsta dažādus ielāpu failu formātus, tostarp IPS (starptautisko ielāpu sistēmu), UPS (universālo ielāpu formātu) un BPS (bināro ielāpu sistēmu), padarot to daudzpusīgu dažādu veidu ielāpu failiem. ielāpus.

- **Pakešu ielāps:** MultiPatch var izmantot, lai lietotu vairākus ielāpus vienam ROM failam vai vairākiem ROM failiem vienlaikus, kas ir noderīgi sarežģītiem ROM uzlaušanas gadījumiem vai projektiem, kas ietver vairākus ielāpus.

- **Ielāpu izveide:** līdzīgi kā peldošais IPS, arī MultiPatch var izmantot, lai izveidotu ielāpu failus no atšķirībām starp diviem ROM, kas ir noderīgi ROM uzlaušanai un tulkošanas projektiem.
