Wenn Sie in der digitalen Marketingbranche tätig sind, E-Mail-Marketing betreiben oder eine Verlagsagentur haben, müssen Sie Microsoft Publisher kennen. Sie können damit Design-Layouts erstellen, die Informationen wie Text, Raster- und Vektorgrafiken enthalten. Diese können verwendet werden, um Newsletter, Flyer, Broschüren, Postkarten und E-Mail-Inhalte zu erstellen.

Zu den Publisher-Dateiformaten gehören Dateitypen wie das PUB-Dateiformat, das auf der Disc gespeichert wird, wenn der Benutzer ein Projekt in Microsoft Publisher erstellt und auf der Disc speichert. Nachfolgend finden Sie eine Liste der Publisher-Dateiformate.
