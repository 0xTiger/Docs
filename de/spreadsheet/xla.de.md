## Was ist eine XLA-Datei?

Eine Datei mit der Erweiterung .xla ist ein Microsoft Excel-Add-Ins-Dateiformat, das Module, zusätzliche Funktionen und andere Tools zu Microsoft Excel hinzufügt. Die Ad-Ins bieten zusätzliche Funktionalität für Excel-Tabellen und werden ab Version 2007 unterstützt. XLA-Dateien haben den Mime-Medientyp application/vnd.ms-excel. Diese können mit Microsoft Excel geöffnet werden, das sowohl für Windows als auch für MacOS verfügbar ist.

## XLA-Dateiformat

XLA-Dateien basieren auf dem binären Datenformat. Weitere Informationen zu den Dateiformatspezifikationen sind jedoch nicht verfügbar. Allgemein wird davon ausgegangen, dass XLA der [[MS-XLS] - Excel Binary File Format Structure](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) folgt.
