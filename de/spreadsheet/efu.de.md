## Was ist eine EFU-Datei?

Eine EFU-Datei ist eine [CSV](/de/spreadsheet/csv/)-Tabellenkalkulationsdatei, die mit der Everything-Software von [voidtools](https://www.voidtools.com/) erstellt wurde. Es enthält eine Liste von Dateien, die mit der Everything-Suchmaschine für Windows durchsucht wurden. Ergebnisse in einer EFU-Datei sind CSV-getrennt und können mit Tabellenkalkulationsprogrammen wie Microsoft Excel geöffnet werden. EFU-Dateien sind menschenlesbar und können mit jedem Texteditor wie Microsoft Notepad, Notepad++ und Apple TextEdit geöffnet werden.

## EFU-Dateiformat - Weitere Informationen

EFU-Dateien werden in einer Datei mit kommagetrennten Werten auf der Disc gespeichert, die Informationen über die Ergebnisse enthält, die von der Everything-Suchmaschinenanwendung erhalten wurden. Diese Datensätze enthalten Informationen über die gefundenen Dateien wie Dateiname, Größe, Erstellungs- und Datum der letzten Änderung.
