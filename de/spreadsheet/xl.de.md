## Was ist eine XL-Datei?

Eine Datei mit der Erweiterung .xl ist eine Excel-Tabellendatei, ähnlich wie [XLS](/de/spreadsheet/xls/), die Daten in Zeilen und Spalten speichert. Es war ein früheres Dateiformat, das jetzt nicht mehr von Excel unterstützt wird. In XL-Dateien gespeicherte Daten können in andere Formate wie CSV exportiert werden. Diese können auch mit der Excel-Standardmethode Speichern unter in PDF konvertiert werden. XL-Dateien konnten nur mit Microsoft Excel geöffnet werden.

## XL-Dateiformat

XL-Dateien werden als Binärdateien gespeichert und ihre internen Dateiformatspezifikationen sind nirgendwo dokumentiert. Das allgemeine Binärdateiformat für [XLS-Dateiformatspezifikationen](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) kann jedoch auch in Betracht gezogen werden, wenn jemand versucht, einen Code zu schreiben, um diese Dateien zurückzuentwickeln.
