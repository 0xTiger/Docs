## Was ist eine DEX-Datei?

Eine DEX-Datei ist eine mit Microsoft Excel erstellte Tabellenkalkulationsdatei. Sie ähnelt den [XLSX](/de/spreadsheet/xlsx/)-Dateien, liegt jedoch im binären Dateiformat vor. Daten werden in Zeilen und Spalten in einer DEX-Datei gespeichert. Obwohl DEX-Dateien auch Microsoft Excel sind, sind diese weniger beliebt. Im Vergleich dazu ist das XLSX-Dateiformat die Standardauswahl zum Speichern von Excel-Arbeitsmappen. DEX-Dateien wurden als veraltet markiert und werden von Microsoft Excel nicht mehr unterstützt.
