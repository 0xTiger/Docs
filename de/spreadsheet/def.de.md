## Was ist eine DEF-Datei?

Eine DEF-Datei ist eine Datendatei, die im [Spreadsheet-Dateiformat](/de/spreadsheet/) von der SmartWare II-Datenbankanwendung erstellt wurde, die von Informix Software entwickelt wurde. Software II hatte vier Module, darunter Tabellenkalkulation mit Grafiken, Textverarbeitung, Kommunikation und Datenbank. Die SmartWare II-Datenbank ist eine eigenständige Datenbank, die auch in andere Anwendungen integriert werden kann. Abfragen können auf Daten ausgeführt werden, die extrahiert und im DEF-Dateiformat gespeichert werden können. DEF-Tabellendateien können mit SmartWare-Software geöffnet werden, die jetzt eingestellt wurde.
