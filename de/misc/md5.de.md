## Was ist eine MD5-Datei?

Eine MD5-Datei ist eine Prüfsummendatei, die zur Überprüfung der Integrität einer Datei verwendet wird. Es ähnelt einem Fingerabdruck für die zugehörige Datei und wird eindeutig mithilfe eines Algorithmus generiert, der die Anzahl der Bits in der Datei verwendet. Es wird verwendet, um die Datei mit Software wie [md5sum](http://www.gnu.org/software/coreutils/manual/html_node/md5sum-invocation.html) zu überprüfen. Ein Vorteil der Verwendung von MD5-Dateien besteht darin, zu überprüfen, ob die heruntergeladenen Dateien nicht beschädigt sind.

## MD5-Dateiformat – Weitere Informationen

Normalerweise wird eine MD5-Datei unter demselben Namen wie der ursprüngliche Dateiname, jedoch mit der Erweiterung .md5 gespeichert. Wenn der zugehörige Dateiname beispielsweise „abc_987_123456.grb“ lautet, lautet der entsprechende MD5-Dateiname „abc_987_123456.grb.md5“. MD5-Dateien helfen festzustellen, dass eine empfangene oder heruntergeladene Datei nicht beschädigt oder von schädlichen Inhalten betroffen ist. Kurz gesagt, MD5-Dateien garantieren die Vollständigkeit einer Datei.


## Wie überprüfe ich die MD5-Prüfsumme?

Eine einzelne Datei kann mit dem Tool [md5sum](https://en.wikipedia.org/wiki/Md5sum) wie folgt auf MD5 geprüft/verifiziert werden.

```
md5sum -c abc_987_123456.grb.md5
```
