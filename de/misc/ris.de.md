## Was ist eine RIS-Datei?

Eine RIS-Datei ist ein Tag-Dateiformat für den Austausch bibliografischer Daten durch Zitierprogramme. Es enthält mehrere Zeilen mit Zitationsdaten, bestehend aus einem zweistelligen Code und dem entsprechenden Wert, der durch einen Bindestrich getrennt ist. In einer RIS-Datei enthaltene Zitierinformationen umfassen Daten wie Autor, Titel, Veröffentlichungsdatum, Schlüsselwörter, Herausgeber, Zusammenfassung, Adresse des Autors usw. RIS-Dateien sind für Menschen lesbar und können mit jedem Texteditor wie Microsoft Notepad, Notepad++, und AppleTextEdit.

## RIS-Dateiformat – Weitere Informationen

RIS-Dateien werden als reine Textdatei gespeichert. Jede Zeile in einer RIS-Datei besteht aus einem aus zwei Buchstaben bestehenden Code (auch als Tags bezeichnet), gefolgt von zwei Leerzeichen und einem Bindestrich sowie dem entsprechenden Wert. Aufgrund von Tags sind RIS-Dateien als getaggtes Format zur Darstellung von bibliografischen Zitaten bekannt. Beispieldatensatz in einer RIS-Datei ist wie unten gezeigt.

## Beispiel für eine RIS-Datei

Das folgende Beispiel aus einem Wikipedia-Artikel zeigt den Inhalt der RIS-Datei für einen Artikel: Claude E. Shannon. Eine mathematische Theorie der Kommunikation, Bell System Technical Journal, 27:379–423, Juli 1948.


|RIS-Beispiel|
---|
|TY - JOUR|
|AU - Shannon, Claude E.|
|PY - 1948|
|DA - Juli|
|TI - Eine mathematische Theorie der Kommunikation|
|T2 - Technische Zeitschrift für Bell-Systeme|
|SP - 379|
|EP-423|
|VL - 27|
|ER - |

In diesem Beispiel ist „TY“ der Typ der Referenz und „ER“ das Ende der Referenz.
