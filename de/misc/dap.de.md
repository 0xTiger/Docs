## Was ist eine DAP-Datei?

Eine DAP-Datei ist eine Download-in-Progress-Datei, die von der Download Accelerator Plus (DAP)-Datei erstellt wurde. Es wird erstellt, wenn Benutzer einen Download von einem beliebigen Online-Datei-Repository mit DAP-Software starten. DAP bietet die Möglichkeit, einen laufenden Download anzuhalten und fortzusetzen. Zu diesem Zweck speichert es die teilweise heruntergeladene Datei mit der Erweiterung .dap, bis sie vollständig heruntergeladen ist. Die DAP-Datei kann während des Download-Vorgangs in der Vorschau angezeigt werden.

DAP-Dateien können mit der Software SpeedBit Download Accelerator Plus (DAP) geöffnet werden.
