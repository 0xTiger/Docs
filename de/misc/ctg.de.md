## Was ist eine CTG-Datei?

Eine CTG-Datei ist eine Katalogdatei, die auf Digitalkameras von Canon erstellt wird. Diese werden erstellt, wenn Fotos mit einer Canon-Digitalkamera aufgenommen werden. Es speichert die Informationen über die Anzahl der Fotos/Bilder in jedem Ordner auf dem Speichergerät der Kamera, dh der Speicherkarte. CTG-Dateien werden im CANONMSC-Ordner auf der Speicherkarte gespeichert. Diese Dateien lassen Ihren Computer die Dateien nach dem Einfügen in den Computer indizieren.

## CTG-Dateiformat – Weitere Informationen

CTG-Dateien werden im Canon-eigenen Dateiformat gespeichert. Diese werden automatisch generiert und sollten nicht manuell bearbeitet werden. Die Kamera löscht diese Dateien, wenn die entsprechenden Fotos vom Benutzer gelöscht werden.
