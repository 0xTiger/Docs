## Was ist eine SSP-Datei?

Eine SSP-Datei ist eine Projektdatei, die von der Windows-Bildschirmschoner-Erstellungssoftware erstellt wurde, die als Screensaver Producer von Axialis Software bekannt ist. Es kann verschiedene Arten von Bildschirmschonerelementen wie Sprites, Flash-Filme, Foto-Diashows und Videos enthalten. Eine SSP-Datei kann aus Text, Bildern, verschiedenen Formen und Animationseinstellungen bestehen, die zum Anzeigen der Animation in der SSP-Datei verwendet werden. SSP-Dateien können mit Axialis Screensaver Producer geöffnet werden.

## SSP-Dateiformat – Weitere Informationen

SSP-Dateien werden als Binärdateien auf Disc gespeichert, ohne dass öffentliche Informationen über ihre interne Dateiformatstruktur verfügbar sind. Axialis Screensaver Producer ist ein professionelles Bildschirmschoner-Authoring-Tool für Windows, mit dem Sie auf einfache Weise hochwertige Bildschirmschoner für alle Windows-Versionen bis zu Windows 10 erstellen können. Es ermöglicht die Erstellung installierbarer Bildschirmschonerpakete, die einfach online an Benutzer verteilt werden können.

SSP-Dateien können in das Dateiformat [SCR](/de/system/scr/) exportiert werden.
