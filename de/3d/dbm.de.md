## Was ist eine DBM-Datei?

Eine DBM-Datei (Daz Brick Material) ist eine Shader-Mixer-Netzwerkdatei, die mit der 3D-Charaktermodellierungsanwendung **DAZ Studio** erstellt wurde. Es beschreibt das Material des 3D-Modus, dh Oberfläche/Verschiebung/Innenvolumen. Mit DAZ können Sie Ihren eigenen benutzerdefinierten Shader erstellen, indem Sie eine Reihe von Bausteinen miteinander verbinden. Dies wird zusammen als DBM-Datei kombiniert, die ein Shader-Netzwerk bildet. Es ist die DBM-Datei, die angibt, wie ein Material schattiert wird.

Zu den Anwendungen, die DBM-Dateien öffnen können, gehört DAZ 3D DAZ Studio 4.

## DBM-Dateiformat

DBM-Dateien werden im Dateiformat [XML](/de/web/xml/) auf der Disc gespeichert.
