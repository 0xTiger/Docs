## Was ist eine DAP-Datei?

Eine Datei mit der Erweiterung .dap ist eine Microsoft Access-Datenseitendatei, die in Microsoft Access erstellt wird und Zugriff auf die in der Datenbank gespeicherten Daten hat. Daten, die in einer solchen Webdatei gespeichert sind, sind über das Netzwerk (Intranet und Internet) zugänglich, was sich von den [MDB](/de/database/mdb/)-Dateien von Access unterscheidet, die speziell für Desktop-Anwendungen entwickelt wurden. Solche Datenbanken können mit Access 2010 und Access Services (einer optionalen Komponente von SharePoint) erstellt werden. Diese Datenbanken können von Personen mit SharePoint-Konten unter Verwendung der Webadresse in einem Webbrowser verwendet werden.
