## Was ist eine SEAM-Datei?

Eine SEAM-Datei ist eine Konfigurationsdatei, die vom JBOS-Seam-Framework verwendet wird. Es enthält Metadaten/Referenzen zu einem serverseitigen Servlet, das Anfragen an die Webanwendung verarbeitet. Es enthält auch Metadaten zu Seam-Komponenten, wie z. B. die Namen von Klassen und Methoden, die die Komponenten implementieren. Sie können SEAM-Dateien in Verbindung mit dem JavaServer Faces (JSF)-Framework zum Erstellen dynamischer und interaktiver Webanwendungen verwenden. SEAM-Dateien werden normalerweise im WEB-INF-Verzeichnis einer Seam-Anwendung gespeichert.

## SEAM-Dateiformat

Eine SEAM-Datei wird als Textdatei gespeichert, normalerweise im Format [XML](/web/xml/) und enthält Konfigurationsinformationen für eine Seam-Anwendung. Es ist ein wichtiger Bestandteil der Konfiguration einer Seam-Anwendung. Sie können eine SEAM-Datei mit jedem Text- oder XML-Editor öffnen.
