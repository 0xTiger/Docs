## Was ist eine ECE-Datei?

Eine ECE-Datei ist eine dynamisch generierte Webseite von der Content-Management-Plattform **Escenic Content Engine (ECE)**. Sie enthält serverseitigen Code, der vom Webserver verarbeitet und als [HTML](/de/web/html/)-Seite an den Benutzerbrowser zurückgesendet wird. Das ECE-Dateiformat ist im Vergleich zu anderen serverseitigen Dateiformaten wie ASP und PHP nicht sehr beliebt. Diese Dateien werden intern gerendert und Endbenutzern zu Anzeigezwecken bereitgestellt.

## ECE-Dateiformat

ECE-Dateien werden nicht auf Disc gespeichert, sondern dem Benutzer auf Anfrage direkt bereitgestellt. Diese werden in HTML konvertiert und zur Anzeige an den Browser des Benutzers zurückgesendet. Escenic Content Engine wird aktiv zum Veröffentlichen von Nachrichtenartikelseiten verwendet. Basierend auf JEE- und SQL-Technologie bietet der ECE-Server ein zuverlässiges und skalierbares Repository für alle Online-Inhalte.

Client-Browser können über die HTTP/REST-API auf den Inhalt auf ECE-Servern zugreifen. Neben der Bereitstellung von [HTML](/de/web/html/)-Seiten bietet ECE einen hocheffizienten Workflow für die Verwaltung und Veröffentlichung von Video-/Audioinhalten.
