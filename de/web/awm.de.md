## Was ist eine AWM-Datei?

Eine AWM-Datei ist eine Projektdatei, die vom Webmenüerstellungsprogramm **AllWebMenus** erstellt wurde. Es enthält projektbezogene Daten zum Erstellen von [HTML](/de/web/html/)-basierten Dropdown-Menüs. Eine AWM-Datei wird zu einem fertigen Webpaket kompiliert, das in die Websites integriert wird, um die Menüs anzuzeigen. Es enthält alle projektbezogenen Daten wie Darstellungseinstellungen, Menüeinträge, Stil (vertikal oder horizontal), Farben und andere zugehörige Informationen.

AWM-Dateien können mit **Likno AllWebMenus** geöffnet werden.

## AWM-Dateiformat – Weitere Informationen

AWM-Dateien werden als kompilierte Binärdateien gespeichert und können nur mit der Likno AllWebMenus-Software geöffnet werden. Die internen Dateiformatdetails von AWM-Dateien sind nicht öffentlich verfügbar.

Mit AllWebMenus können Sie [CSS](/de/web/css/)-Menüs oder [JavaScript](/de/web/js/)-basierte Menüs visuell entwerfen, ohne eine Programmiersprache zu verwenden. Die Menüs können vollständig angepasst und mit vordefinierten Darstellungsvorlagen gestaltet werden, die mit der Software verfügbar sind.
