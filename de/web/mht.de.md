## Was ist eine MHT-Datei?

Eine Datei mit der Erweiterung .mht ist ein MIME-fähiges Archivierungsdateiformat, das verschiedene Datentypen in einer einzigen Datei enthält. Es kann Daten wie Text, Bilder, Seitengestaltung in Form von [CSS](/de/web/css/)-Dateien, JavaScript und andere Ressourcen als darin eingebettete Ressourcen speichern. MHT-Dateien mit dem MIME-Typ message/rfc822 kapseln den gesamten Inhalt einer HTML-Datei als eine einzelne Archivdatei zum Speichern und Archivieren auf Speichergeräten. Mit Softwareanwendungen wie Microsoft Word können Sie Ihre WORD-Dokumente in MHT konvertieren, indem Sie sie als MHT-Datei exportieren. MHT-Dateien können mit gängigen Browsern wie Microsoft Internet Explorer und Google Chrome geöffnet werden.

## MHT-Dateiformat

Wie [MHTML](/de/web/mhtml/) ist auch MHT eine MIME-Kapselung von zusammengefassten HTML-Dokumenten. Dokumentinhalte innerhalb eines MHT-Dokuments wie z. B. Inline-Bilder, Stylesheets, Applets usw. sind MIME-kodiert, wobei diese über URIs mit einer Root-Ressource verknüpft sind. E-Mail-Clients wie Microsoft Outlook speichern E-Mails (normalerweise [EML](/de/email/eml/)) im MHT-Dateiformat. Vollständige Spezifikationen des MHT-Dateiformats sind in [RFC 822](https://tools.ietf.org/html/rfc822) verfügbar und können für die Entwicklung von Softwareanwendungen herangezogen werden, die MHT-Dateien verarbeiten können.

## Unterschied zwischen MHT und MHTML

Beim Speichern einer Online-Seite wird der Benutzer aufgefordert, sich für ein Dateiformat zu entscheiden, in dem die Online-Seite auf dem nativen System gespeichert werden muss. Am häufigsten wird der Benutzer zwischen Markup-Sprache und MHTML-Dateiformaten verwirrt. Sie bemerken, dass es schwierig ist zu sehen, dass das Dateiformat zwischen diesen gesünder ist.

Wenn ein Benutzer beschließt, eine Online-Seite nicht als Auszeichnungssprache zu verschwenden, wird ein Ordner erstellt, der mehrere Dateien für verschiedene Inhalte der Seite enthält, dh völlig unterschiedliche Dateibereiche, die für Text, Grafiken, Applets usw. erstellt wurden. Auf der anderen Seite das Speichern die Online-Seite als MHTML erstellt eine Datei für die komplette Online-Seite. Alle Elemente der Seite zusammen mit Grafiken, Links und alternativen Dateien sind in einer Datei eingebettet.

Natürlich ist es einfach, MHT- oder MHTML-Dateien zu handhaben, da die Datei geschützt und einfach per E-Mail geteilt werden kann. Allerdings besteht im Bereich der Markup-Sprachdateien die Möglichkeit eines Informationsverlusts, da der Verlust oder das Löschen auch nur einer Datei den gesamten Markup-Sprachordner für den Benutzer unbrauchbar machen könnte.
