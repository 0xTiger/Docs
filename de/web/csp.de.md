## Was ist eine CSP-Datei?

Eine CSP-Datei ist eine dynamische Webseitendatei, die Inhalte enthält, die zur Ausführung auf RadGs Concept Application Server geschrieben wurden. Es wird auf der Serverseite kompiliert und ausgeführt, wonach es über das HTTP-Protokoll an den Browser des Benutzers gesendet wird. Im Gegensatz zum CON-Dateiformat muss für CSP-Dateien der Concept-Client nicht beim Benutzer installiert werden, um diese Dateien anzeigen zu können. Sie können CSP-Dateien ähnlich wie andere serverseitige Seiten wie [PHP](/de/programming/php/) und [ASP](/de/web/asp/) anzeigen.
