## Was ist eine DOTHTML-Datei?

Eine DOTHTML-Datei ist ein Microsoft Word-Vorlagendokument, das in der HTML-Version gespeichert wird. Sie kann im Webbrowser geöffnet werden und der Inhalt wird wie jede andere Webseite angezeigt. DOTHTML-Dateien werden aus Microsoft Word .DOT-Dateien generiert, die vorformatierte Einstellungen zum Generieren dieser Dateien enthalten. Wenn eine [DOT](/de/word-processing/dot/)-Datei in DOCHTML konvertiert wird, werden alle diese Einstellungen beibehalten und die konvertierte DOTHTML-Datei wird mit einer ähnlichen Formatierung wie die ursprüngliche DOT-Datei angezeigt. Auf diese Weise können veröffentlichte Vorlagen im Webbrowser angezeigt werden.

## DOTHTML-Dateiformat - Weitere Informationen

DOTHTML-Dateien werden als [HTML](/de/web/html/)-Dateien gespeichert und enthalten HTML-Quellcode, der mit jedem HTML-Editor geöffnet werden kann. In den ersten Tagen bot Microsoft Word die HTML-Versionen aller seiner Dokumentformate wie [DOCHTML](/de/web/dochtml/), [DOCMHTML](/de/web/docmhtml/) an, aber später wurden diese HTML-basierten Versionen eingestellt . Folglich wird auch DOTHMLT nicht mehr unterstützt.
