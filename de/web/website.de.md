## Was ist eine WEBSITE-Datei?

Eine WEBSITE-Datei ist eine Website-Verknüpfungsdatei, die mit Microsoft Internet Explorer (IE) erstellt wurde. Es enthält eine URL und eine Symboldatei (ICO), die auf das Favicon der Website verweist. Das Symbol wird zum Anzeigen der Verknüpfung verwendet. Das WEBSITE-Dateiformat wurde mit der Veröffentlichung von IE Version 9 eingeführt. Frühere Versionen von IE verwendeten zum Speichern von Verknüpfungen zu Webseiten als [.url](/de/web/url/)-Dateien.

## WEBSITE-Dateiformat

Eine .website-Datei wird als einfache Textdatei auf der Disc gespeichert, deren Inhalt in einem beliebigen Texteditor angezeigt werden kann. Neben anderen Inhalten enthält es die Ziel-URL, für die es eine Verknüpfung ist. Wenn Sie die Zielwebsite in einem anderen Browser als IE öffnen möchten, können Sie diese URL kopieren und in andere Browser wie Google Chrome, Safari oder Firefox einfügen, um die Website anzuzeigen.

## Wie erstelle ich eine .WEBSITE-Datei?

Eine WEBSITE-Datei kann im IE erstellt werden, indem Sie auf die URL in der Adressleiste klicken und sie an die gewünschte Stelle auf Ihrem Computer ziehen, z. B. auf den Desktop oder in Dokumente. Dadurch wird die Verknüpfung zu diesem Internetspeicherort erstellt und kann durch Doppelklicken auf die Verknüpfung geöffnet werden. Wenn Sie die Verknüpfung als „URL“-Datei speichern möchten, halten Sie die UMSCHALTTASTE gedrückt, bevor Sie das Favicon der Zielseite auf den Desktop ziehen.
