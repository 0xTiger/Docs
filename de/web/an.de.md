## Was ist eine AN-Datei?

Eine Datei mit der Erweiterung „.an“ ist eine [Adobe Edge Animate](https://www.adobe.com/products/animate.html)-Projektdatei, die Informationen über die Animation enthält. Beim Speichern des Projekts wird eine HTML-Datei erstellt, die die Verweise auf die zugehörigen [.css](/de/web/css/)- und [.js](/de/web/js/)-Dateien enthält, die von Adobe Edge Animate generiert wurden. Die AN-Datei wird zusammen mit diesen Dateien gespeichert, die zum Öffnen der Projektdatei verwendet werden können. Adobe Edge Animate kann als Alternative zum Erstellen animierter Webseiten mit Adobe Flash verwendet werden, das von Browsern nicht mehr unterstützt wird.

## AN-Dateiformat

AN-Dateien werden im Nur-Text-Dateiformat auf der Disc gespeichert und können in Texteditoren geöffnet werden, um ihren Inhalt anzuzeigen.
