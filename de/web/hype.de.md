## Was ist eine HYPE-Datei?

Eine HYPE-Datei ist eine mit der Hype-Software gespeicherte Projektdatei, die zur Erstellung von Webinhalten mit HTML5 verwendet wird. Es enthält Webelemente wie Grafiken, Text, Animationen, Szeneninformationen und andere Elemente, die zusammen den Inhalt ausmachen. Webinhalte in einer HYPE-Datei können in Webdateien exportiert werden, die auf einer Webseite veröffentlicht werden können.

Eine HYPE-Datei kann mit der Software [Tumult Hype](https://tumult.com/hype/) geöffnet werden.

## HYPE-Dateiformat

HYPE-Dateien werden in einem proprietären Dateiformat gespeichert und die internen Details davon sind nicht öffentlich bekannt. Der Inhalt einer HYPE-Datei kann nur angezeigt werden, wenn sie in der Tumult Hype-Software geöffnet wird.

### Inhalt einer HYPE-Datei

Eine HYPE-Datei wird erstellt, wenn Sie die Hype-Anwendung verwenden, um Webinhalte für verschiedene Zwecke zu erstellen. Es speichert Informationen über Keyframe-basierte Animationen zum Erstellen von Inhalten wie eCards, eBooks, Präsentationen, Anzeigen, Infografiken und Webseitenmaterial. Der Inhalt dieser wird von HTML5 unterstützt, mit dem der Inhalt auf Geräten von Desktops bis hin zu iPhones und iPads angezeigt werden kann. Tumult Hype gibt modernste HTML5-, [CSS3](/de/web/css/)-Stile und [JavaScript](/de/web/js/) aus.
