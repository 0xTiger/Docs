## Was ist eine WHTT-Datei?

Eine WHTT-Datei ist eine Offline-Version einer Website, die mit der Software [HTTrack Website Copier](https://www.httrack.com/) heruntergeladen wird. Es enthält alle Inhalte einer Website, einschließlich [HTML](/de/web/html/)-Webseiten, Bilder, Stylesheets, Links und Verzeichnisse. Alle Seiten einer Website werden rekursiv heruntergeladen und als Offline-Mirror auf Platte gespeichert. Wenn eine Website von der HTTrack-Software heruntergeladen wird, kann sie offline angezeigt werden, ohne dass eine Verbindung zum Internet erforderlich ist. Um die heruntergeladene Website anzuzeigen, öffnen Sie einfach die Seite der gespiegelten Website im Browser und jedes Bild/jeder Link wird ordnungsgemäß geladen.

## WHTT-Dateiformat

WHTT-Dateien werden als Kombination aus THML-Webseiten, Bildern, Stylesheets und Referenzlinks gespeichert.
