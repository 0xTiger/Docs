## Was ist eine OBML16-Datei?

Eine OBML16-Datei ist eine Webseite, die mit dem mobilen Webbrowser Opera Mini gespeichert wird. Es basiert auf der Opera Binary Markup Language ([OBML](/de/web/obml/)) und ist ein kompaktes Format für den schnellen Zugriff auf gespeicherte Webseiten. Es ersetzte das Dateiformat [OBML15](/de/web/obml15/), als Opera Mini Version 7 gestartet wurde. Alle diese Varianten von Opera Mini-Webseiten wurden vom Basis-OBML-Dateiformat aktualisiert, und jedes Upgrade von Opera Mini ist nur mit einem OBML-Format kompatibel. OBML16-Dateien können online in [PDF](/de/pdf/) und [HTML](/de/web/html/) konvertiert werden.

## OBML16-Dateiformat – Weitere Informationen

OBML16 basiert auf dem OBML-Dateiformat, einem kompakten Format zum Speichern von Webseiten. Der Vorteil dieses kompakten Formats besteht darin, dass es die Größe von Webinhalten reduziert, was zu einer Erhöhung der Downloadgeschwindigkeit führt. Einmal heruntergeladen, können Benutzer von Opera Mini diese Seiten anzeigen, ohne auf eine Internetverbindung zugreifen zu müssen.
