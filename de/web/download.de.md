## Was ist eine DOWNLOAD-Datei?

Eine DOWNLOAD-Datei ist eine teilweise aus dem Internet heruntergeladene Datei, die noch heruntergeladen wird. Wenn Sie eine Datei mit dem Apple Safari-Browser aus dem Internet herunterladen, kann es einige Zeit dauern, bis der Download abgeschlossen ist. Diese teilweise heruntergeladene Datei wird im Ordner **Downloads** gespeichert. Wenn der Download aus irgendeinem Grund unterbrochen wird, können Sie auf die Datei doppelklicken, um zu versuchen, sie fortzusetzen.

## DOWNLOAD-Dateiformat - Weitere Informationen

DOWNLOAD-Dateien werden sowohl im Text- als auch im Binärdateiformat auf der Disc gespeichert. Es enthält nicht die gesamte Datei, die gerade heruntergeladen wird. Sobald der Download abgeschlossen ist, wird die Dateierweiterung .download durch die ursprüngliche Erweiterung der heruntergeladenen Datei ersetzt. Wenn Sie beispielsweise ein Video aus dem Internet im MP4-Dateiformat herunterladen, wird die Datei heruntergeladen und mit der Erweiterung .download gespeichert. Sobald der Download abgeschlossen ist, wird die Dateierweiterung in .mp4 geändert.

DOWNLAOD-Dateien ähneln [.CRDOWNLOAD](/de/web/crdownload/)-Dateien, die von Google Chrome und Microsoft Edge verwendet werden.
