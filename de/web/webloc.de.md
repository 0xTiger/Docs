## Was ist WEBLOC?

Eine WEBLOC-Datei ist eine Verknüpfungsdatei auf macOS, die auf eine Web-URL im Internet verweist. Es wird generiert, indem die Web-URL von der Adressleiste des Browsers auf den Desktop oder einen anderen Ordner gezogen wird, in dem sie gespeichert werden soll. Die URL zur Zielsite wird in dieser Verknüpfungsdatei gespeichert und kann durch Öffnen der Datei in einem Texteditor wie Microsoft Notepad, Notepad++ oder Apple TextEdit angezeigt werden. WEBLOC-Dateien ähneln den [URL](/de/web/url/)-Dateien.

## WEBLOC-Dateiformat - Weitere Informationen

WEBLOC-Dateien werden im Nur-Text-Dateiformat gespeichert. Diese können auf macOS durch Ziehen von URLs aus Webbrowsern wie Safari, Google Chrome oder Firefox generiert werden. Wenn mehrere WEBLOC-Dateien gleichzeitig geöffnet werden sollen, wäre eine schnellere und einfachere Lösung erforderlich, um diese über externe Anwendungen oder APIs zu öffnen. Diese APIs können WEBLOC lesen und in PDF und JPEG konvertieren.
