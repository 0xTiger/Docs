## Was ist eine ADR-Datei?

Eine ADR-Datei (gespeichert mit der Erweiterung .adr) ist eine Liste mit Lesezeichen, die vom Opera-Webbrowser erstellt wird. Diese Liste wird gefüllt, wenn Benutzer Web-URLs mit Lesezeichen versehen. Diese Lesezeichen werden in der ADR-Datei gespeichert, die bei Bedarf gesichert und wiederhergestellt werden kann. Da die meisten Browser wie Chrome und Firefox Lesezeichen im [HTML](/de/web/html/)-Format speichern, können diese zum Importieren in den Opera-Browser in das ADR-Dateiformat konvertiert werden. ADR-Dateien können im Opera-Browser geöffnet werden.

## ADR-Dateiformat – Weitere Informationen

Die internen Dateiformatdetails von ADR-Dateien sind unbekannt. Es gibt jedoch Unterschiede zwischen den ADR-Dateiformaten, die mit verschiedenen Opera-Versionen exportiert werden. Dies macht es schwierig, [ADR-Dateien zusammenzuführen](https://superuser.com/questions/471959/how-do-i-merge-several-opera-adr-bookmark-files-to-one-single-file) zu generieren mit verschiedenen Versionen von Opera. Einige externe Tools können jedoch verwendet werden, um diese zusammenzuführen.
