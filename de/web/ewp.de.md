## Was ist eine EWP-Datei?

Eine EWP-Datei ist eine Projektdatei, die mit [EWISoft](http://www.ewisoft.com/) Website Builder-Software erstellt wurde. Es ist mit einer binären Datenbankdatei verknüpft, die alle Informationen für das Projekt enthält. Die EWP-Datei wird häufig zum Speichern des Fortschritts der Webentwicklung verwendet. Es ist eine vorlagengesteuerte Webdesign-Software, mit der Benutzer Websites aus Website-Vorlagen erstellen können.

## EWP-Dateiformat – Weitere Informationen

EWP-Dateien sind Projektdateien, die auf keinem Server kompiliert oder gehostet werden müssen. Diese werden als einfache Textdateien gespeichert und enthalten Verweise auf den Fortschritt der Webentwicklung. Projekte können nach Abschluss der Entwicklung auf einer Website veröffentlicht werden.
