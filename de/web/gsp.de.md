## Was ist eine GSP-Datei?

Eine GSP-Datei ist eine Fehlermeldungsdatei, die auf ein Problem bei der Zustellung der E-Mail hinweist. Es wird von IMail Server gesendet, einem E-Mail-Serverprogramm, das von zahlreichen kleinen und mittleren Unternehmen für die E-Mail-Kommunikation verwendet wird. Der Fehler kann verschiedene Gründe haben, wie z. B. eine große Datenmenge, Datenverlust in der Datei [MSG](/de/email/msg/) oder Speichermangel.

## GSP-Dateiformat – Weitere Informationen

GSP-Dateien werden im Webdateiformat gespeichert und können problemlos mit E-Mail-Clients wie Microsoft Outlook angezeigt werden. Als Antwort auf eine gesendete E-Mail, die zu einem Fehler führt, wird eine GSP-Datei generiert.
