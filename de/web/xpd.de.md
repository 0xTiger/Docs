## Was ist eine XPD-Datei?

Eine XPD-Datei ist eine Webseitendatei, die aus einem XML-Dokument erstellt wird. Häufiger wird dies durch eine XSLT-Transformation generiert. Die [XPD-Dateiformatspezifikationen](https://www.w3.org/TR/xml-pipeline/) sind auf W3 verfügbar und definieren eine Möglichkeit zur Verbesserung der Interoperabilität von Multiprozess-[XML](/web/xml/). Anwendungen. Die XML Pipeline Language definiert die Verarbeitungsbeziehungen zwischen XML-Ressourcen.
