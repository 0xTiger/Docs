## Was ist eine FWP-Datei?

Eine Datei mit .fwp-Datei ist eine Archivdatei, die mit Microsoft Expression Web erstellt wurde und alle Dateien und Ordner für ein Design enthält. Microsoft Expression Web ist ein allgemeines Webdesign-Softwareprodukt von Microsoft und basiert auf dem Editor [HTML](/de/web/html/). Es wurde seit 2012 eingestellt und steht von Microsoft kostenlos zum Download zur Verfügung. FWP-Dateien können im .cab-Format von Microsoft exportiert und gespeichert werden.

## FWP-Dateiformat – Weitere Informationen

Genaue Dateiformatspezifikationen von FWP-Dateien sind nicht als Referenz für Entwickler verfügbar. Allerdings handelt es sich hierbei um Binärdateien, die im längst eingestellten Microsoft Expression Studio geöffnet werden können.
