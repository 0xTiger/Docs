## Was ist eine TS-Datei?

Eine TS-Datei ist eine Übersetzungsdatei, die mit dem Softwaredienstprogramm lupdate generiert wird, das im Lieferumfang des Qt-SDK von Digia enthalten ist. Das SDK ist für Windows-, Mac- und Linux-Betriebssysteme verfügbar. Die TS-Datei enthält Sätze in einer Sprache und deren Übersetzungen in eine andere Sprache. Die in einer TS-Datei gespeicherten Übersetzungen sind spezifisch für eine bestimmte Anwendung. TS-Dateien können mit dem Digia Qt SDK geöffnet werden.

## TS-Dateiformat

Digia Qt TS-Dateien werden im XML-Dateiformat auf Disc gespeichert und können in jedem Texteditor wie Notepad und Notepad++ geöffnet werden. Einige andere vom Digia Qt SDK unterstützte Dateiformate sind:

* .pri
* .qm
* .qml
*.qph
* .qrc
*.qs
* .qss
*.ts
* .ui-Benutzeroberflächendatei
