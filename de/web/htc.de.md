## Was ist eine HTC-Datei?

Eine HTC-Datei ist eine Datei im Webdateiformat, die von Microsoft Internet Explorer verwendet wird. Es enthält Definitionen für HTML-Komponenten, die kleine Funktionen ausführen, um verschiedene Aufgaben innerhalb des Webbrowsers auszuführen. Die in HTC-Dateien definierte Funktionalität ist wiederverwendbar und über mehrere [HTML](/de/web/html/)-Seiten zugänglich.

## HTC-Dateiformat – Weitere Informationen

HTC-Dateien werden im Nur-Text-Dateiformat gespeichert. Das heißt, Sie können diese in jedem Texteditor öffnen und Änderungen an der Funktionalität vornehmen. Diese werden normalerweise im Internet Explorer geöffnet, aber auf diese Weise können Sie diese Dateien nur anzeigen und nicht bearbeiten.

Eine HTC-Datei ist eigentlich eine HTML-Datei, die eine Kombination aus Skript und einem Satz HTC-spezifischer Elemente zum Definieren der Komponenten ist.
