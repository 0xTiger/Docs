## Was ist eine XHT-Datei?

Eine Datei mit der Erweiterung .xht ist eine Webdatei, die dem Dateityp [XHTML](/de/web/xhtml/) (Extended Hypertext Markup Language) zugeordnet ist, der selbst eine auf [XML](/de/web/xml/) basierende Markup-Datei ist. Diese beiden Dateien ähneln HTML, haben jedoch eine strengere XML-ähnliche Syntax. XHT-Dateien sind so konzipiert, dass sie strukturierter sind, weniger Skripting erfordern und zusätzlich zur Verwendung der vorhandenen Möglichkeiten von XML generisch sind.

## XHT-Dateiformat

Eine XHT-Datei wird standardmäßig als einfache Textdatei mit UTF-8-Codierung erstellt und gespeichert. Es enthält den vollständigen Quellcode eines XHTML-Dokuments, das mit jedem Texteditor geöffnet und bearbeitet werden kann, der Unicode-Kodierung unterstützt. Neben Texteditoren ist das XHT-Dateiformat für die meisten modernen Webbrowser verständlich und kann mit diesen geöffnet werden.

## XHT-Beispiel

Das folgende Beispiel eines XHT-Dokuments definiert die XML-Deklarationen.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
