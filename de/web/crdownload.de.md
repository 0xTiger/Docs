## Was ist eine CRDOWNLOAD-Datei?

Eine CRDOWNLOAD-Datei ist eine teilweise heruntergeladene Datei, die erstellt wird, wenn Sie eine Datei aus dem Internet mit den Browsern Microsoft Edge, Google Chrome oder Chromium herunterladen. Der Inhalt der heruntergeladenen Datei wird in dieser Datei gespeichert, bis der Download abgeschlossen ist. Sobald der Download abgeschlossen ist, wird die Erweiterung .crdownload entfernt und durch die ursprüngliche Dateierweiterung der heruntergeladenen Datei ersetzt.

## CRDOWNLOAD-Dateiformat

Eine CRDOWNLOAD-Datei kann sowohl im Text- als auch im Binärformat vorliegen. Browser erstellen standardmäßig eine .crdownload-Datei, sobald der Download beginnt. Wird der Download zwischendurch unterbrochen, bleibt die CRDOWNLOAD-Datei auf der Disk gespeichert. Dadurch kann der Download manchmal dort fortgesetzt werden, wo er unterbrochen wurde.

### CRDOWNLOAD-Beispiel

Support Sie laden eine Datei fileformatslist.doc aus dem Internet herunter. Wenn der Download beginnt, speichert Ihr Browser die Datei im Download-Ordner als fileformatslist.crdownload-Datei. Die Dateigröße nimmt zu, wenn die Datei heruntergeladen wird. Sobald der Download abgeschlossen ist, wird die Datei in fileformatslist.doc umbenannt.
