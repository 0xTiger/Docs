## Was ist eine XHTM-Datei?

Eine Datei mit der Erweiterung .xhtm ist eine Webseitendatei, die in XHTML geschrieben ist. Es verwendet XML 1.0 als Standardimplementierung und gilt als Neuformulierung von HTML 4. Wie XHTML enthält eine XHTM-Datei Standard-HTML-Tags, die strengeren XML-Richtlinien folgen. XHTM-Dateien werden zum Veröffentlichen von Website-Inhalten verwendet und verwenden üblicherweise die Erweiterung [XHTML](/web/xhtml/).

Sie können Webbrowser wie Google Chrome, Apple Safari und Mozilla Firefox verwenden, um XHTM-Dateien zu öffnen.

## XHTM-Beispiel

Das folgende Beispiel eines XHTM-Dokuments definiert die XML-Deklarationen.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
