## Was ist eine URL-Datei?

Eine Datei mit der Erweiterung .url ist eine Website-URL-Verknüpfung, die mit einer beliebigen Website-Adresse auf dem Computer gespeichert werden kann. Wenn es durch Doppelklicken geöffnet wird, öffnet die URL die Website im Standardbrowser und die Seite wird dem Benutzer angezeigt. URL-Dateien werden normalerweise als Verknüpfungsdateien zu Websites bezeichnet und durch das Symbol des Standardbrowsers dargestellt. URL-Dateien können in jedem Texteditor wie Microsoft Notepad, Notepad++ oder Apple TextEdit geöffnet und bearbeitet werden.

## URL-Dateiformat – Weitere Informationen

URL-Dateien werden im Nur-Text-Format gespeichert und können in jedem Texteditor geöffnet und angezeigt werden. Diese werden im Standard-Webbrowser geöffnet, in dem der Benutzer andere Browsing-Vorgänge durchführt.
