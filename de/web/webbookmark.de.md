## Was ist eine WEBBOOKMARK-Datei?

Eine WEBBOOKMARK-Datei ist eine Lesezeichendatei, die mit dem Safari-Webbrowser von Apple unter macOS gespeichert wird. Es enthält Informationen über die Webseite, die der Benutzer vom Browser gespeichert hat. Zu den in einer WEBBOOKMARK-Datei gespeicherten Informationen gehören der Seitentitel, die URL und die Lesezeichengruppe, in der diese Seite gespeichert ist. Wenn eine Webseite mit einem Lesezeichen versehen ist, fordert der Safari-Browser die Gruppe auf, die mit einem Lesezeichen versehene Seite zu speichern. Die meisten Benutzer speichern die Lesezeichen im Lesezeichenordner, der der Standardspeicherort ist, der von gängigen Browsern angeboten wird.

## WEBBOOKMARK-Dateiformat

WEBBOOKMARK-Dateien werden im binären Dateiformat auf der Disc gespeichert, das nicht für Menschen lesbar ist. Benutzer können eine Webseite als Lesezeichen markieren, wenn sie später besucht werden soll oder sich auf die Interessen des Benutzers bezieht. Bei Bedarf kann der Benutzer im Safari-Browser zum Abschnitt „Lesezeichen“ navigieren und die Seite erneut besuchen.
