## Was ist eine WBS-Datei?

Eine PSP-Datei ist eine von WebBlender erstellte Projektdatei, ein Präsentations- und Website-Erstellungsprogramm für Kinder. Die PSP-Datei wird erstellt, wenn ein neues Projekt in WebBlender initiiert wird. Alle dem Projekt hinzugefügten Site-Features wie Seiten, Bilder, Text, Animationen usw. werden in der PSP-Projektdatei referenziert. Sobald das Projekt fertig ist, veröffentlicht es die Seiteninhalte als [HTML](/de/web/html/)-Datei. WebBlender wird von [Tech4Learning](https://www.tech4learning.com/) erstellt und gepflegt.
