## Was ist eine XOJ-Datei?

Eine XOJ-Datei ist eine Notizbuchdatei, die Notizen enthält, die mit der Handnotizsoftware Xournal erstellt wurden. Xournal ist eine kostenlose, plattformübergreifende Anwendung, mit der Sie Notizen mit verschiedenen Inhaltstypen wie Text, Bildern und handgeschriebenen Kritzeleien erstellen können. XOJ-Dateien werden erstellt, wenn eine neue Notebook-Datei mit Xournal erstellt wird. Diese XOJ-Dateien können auch mit anderen Benutzern zum Datenaustausch geteilt werden. Eine neuere Version von Xournal ist [Xournal++](https://github.com/xournalpp/xournalpp), eine plattformübergreifende Variante von Xournal.

## XOJ-Dateiformat

XOJ-Dateien sind im XML-Dateiformat und werden in komprimierter Form unter Verwendung der Komprimierung [.gz](/de/compression/gz/) auf der Disc gespeichert. Durch die Verwendung des universellen XML-Formats können XOJ-Dateien auch mit der plattformübergreifenden Variante Xournal++ geöffnet und bearbeitet werden.

Jede XOJ-Datei kann eine weitere Seite eines Xournal-Notizbuchs enthalten.

## XOJ - Unterstützte Plattformen

Xournal ist plattformübergreifend und seine Notizbücher können als XOJ-Datei auf den folgenden Betriebssystemen gespeichert werden.

* Linux
* Mac OS
* Fenster
* Android
* iOS-Build
