## Was ist eine FCGI-Datei?

Eine FCGI-Datei ist eine Quellcodedatei, die Quellcode enthält, der für FastCGI (FCGI) geschrieben wurde. Es kann in mehreren verschiedenen Programmiersprachen geschrieben werden und enthält die Erweiterung des CGI-Standards, der für eine robustere Handhabung von Webserver-Anfragen verwendet wird. Zu den am häufigsten unterstützten Sprachen für FCGI gehört **Perl**, das sowohl Perl-Code als auch die starke externe Schnittstelle der FastCGI-API unterstützt.

## FCGI-Dateiformat

Was macht eine FCGI-Datei wichtig? Die FCGI-Schnittstelle unterstützt verteiltes Computing, um Webanforderungen zu verarbeiten und sie auf mehrere Server zu verteilen. Dadurch werden FCGI-Dateien hauptsächlich für den Lastausgleich auf mehreren Servern verwendet.
