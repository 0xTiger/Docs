## Was ist eine HXS-Datei?

Eine HXS-Datei ist eine Hilfedatei, die für die Microsoft Help 2.x-Datei kompiliert und verwendet wird. Es wird aus einer Reihe von Themenseiten kompiliert und generiert und in einer Teilmenge von [HTML](/de/web/html/) geschrieben, die dem früheren Hilfedateiformat [CHM](/de/web/chm/) ähnelt. HXS-Dateien werden in komprimiertem Format gespeichert und aus einer .HxC-Hauptprojektdatei, einer .HxF-Include-Datei, einem .HxT-Inhaltsverzeichnis, einer .HxA-Attributdefinitionsdatei und mehreren .HxK-Indizes erstellt. Die Metadaten einer HXS-Datei werden aus [XML](/de/web/xml/) erstellt.

HXS-Dateien können mit einem Webbrowser geöffnet werden.

## HXS-Dateiformat

HXS-Dateien werden als komprimierte Dateien aus HTML-Inhalten und XML-Metadaten erstellt und gespeichert.
