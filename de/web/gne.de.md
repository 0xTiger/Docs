## Was ist eine GNE-Datei?

Eine GNE-Datei ist eine Webseite, die dynamisch generiert und von der Fotofreigabe-Website Flickr verwendet wird. Die Seiten werden auf dem Server gehostet, von wo aus sie an den Browser des anfragenden Benutzers geliefert werden. GNE-Dateien ähneln PHP- und ASP-Seiten, werden jedoch in einer proprietären Programmiersprache entwickelt, deren Details nicht als Referenz für Entwickler verfügbar sind.

## GNE-Dateiformat – Weitere Informationen

GNE-Dateien wurden vom Computerspiel Game Never Ending verwendet, das von [Ludicorp](https://en.wikipedia.org/wiki/Ludicorp) entwickelt wurde. Flicker wurde später von Yahoo übernommen! im März 2005.
