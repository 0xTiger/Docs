## Was ist eine EPIBRW-Datei?

Eine EPIBRW-Datei ist eine generische Datei, die auf einem Microsoft IIS-basierten Webserver gehostet wird. Der Zweck dieser Datei ist nicht spezifisch für bestimmte Vorgänge oder irgendeine Art der Datenspeicherung, sondern allgemein. Es wird am häufigsten von [.pdf](/de/pdf/)-Dateien verwendet, kann aber auch von mehreren anderen Dateitypen verwendet werden.

Wenn auf eine EPIBRW-Datei doppelgeklickt wird, wird sie möglicherweise im Browser geöffnet und zeigt den Inhalt an. Wenn die Datei jedoch zum Herunterladen verlinkt ist, wird der Datei-Download-Prozess als Ergebnis dieses Ereignisses initiiert.
