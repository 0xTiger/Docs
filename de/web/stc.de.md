## Was ist eine STC-Datei?

Eine Datei mit der Erweiterung .stc ist eine Verbindungsdatei, die von der Adobe Contribute-Software zum Speichern der Verbindungsinformationen verwendet wird. Es enthält Verbindungsinformationen wie Benutzername, Host-URL, Protokolle wie FTP, SFTP usw. zur Wiederverwendung innerhalb der bestehenden Website. Für diejenigen, die es nicht wissen, Adobe Contribute war früher ein spezialisierter [HTML](/de/web/html/)-Editor, der verwendet wurde, um Aktualisierungen auf bestehende Websites zu übertragen, einschließlich inhaltsbasierter Artikel, die als Blogs bekannt sind. Es enthielt Plugins für beliebte Webbrowser wie Internet Explorer und Firefox, die nahtlos Inhalte von innerhalb der Seiten zu Websites beisteuerten. Es wurde seit Januar 2017 eingestellt und Adobe Dreamweaver wird jetzt zum Erstellen von Websites von Grund auf verwendet.

## STC-Dateiformat

Das interne Datenformat von STC-Dateien ist unbekannt. Der Verbindungsschlüssel wird jedoch mit einem Kennwort verschlüsselt, und alle in der Datei gesendeten Netzwerk- oder FTP-Verbindungsinformationen bleiben sicher und können nur über Contribute aufgerufen werden. Die Schlüsselverbindungsdatei kann per E-Mail an Benutzer gesendet oder auf dem Computer gespeichert werden. Wenn darauf geklickt wird, wird automatisch Contribute geöffnet, das die Verbindung importiert.
