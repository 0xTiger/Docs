## Was ist eine CPHD-Datei?

Eine CPHD-Datei ist ein Hypertextdokument, das von der CloudChan-Präprozessor-Engine verwendet wird. Es enthält HTML und Skripte zum Erstellen dynamischer Webseiten und Dokumente. Skripte in einer CPHD-Datei ähneln der von Java verwendeten Syntax. Die CPHD-Webseite gibt ein HTML-Dokument zurück, wenn sie auf einem CloudChan-Prozess ausgeführt wird. Das CPHD-Format kann mit jedem XML-Dokument verwendet werden.

## CPHD-Dateiformat

CPHD-Dateien werden als reine Textdateien auf der Disc gespeichert. Dadurch ist es möglich, diese Dateien in jedem Texteditor wie Notepad oder Notepad++ zu öffnen.
