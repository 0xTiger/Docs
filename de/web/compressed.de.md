## Was ist eine KOMPRIMIERTE Datei?

Eine KOMPRIMIERTE Datei ist eine kleine komprimierte Wordpress-Webseite, die mit dem Plug-in **WordCompress** erstellt wurde. Es besteht aus einer einzigen Codezeile zur Darstellung der Webseite, die durch Komprimieren von HTML-, CSS- und JavaScript-Code durch das Plugin generiert wird. Diese konvertierten KOMPRIMIERTEN Dateien sind klein und ermöglichen so ein schnelleres Laden und Übertragen über das Internet.

### KOMPRIMIERTES Dateiformat – Weitere Informationen

KOMPRIMIERTE Dateien werden als Textdateien gespeichert, die mit jedem Texteditor wie Microsoft Notepad, Notepad++ und Apple TextEdit geöffnet werden können. Zusätzlich zu Texteditoren können KOMPRIMIERTE Dateien mit Web-Building-Software wie Adobe Dreamweaver geöffnet werden, das sowohl für Windows als auch für Mac OS verfügbar ist.

## Über das WordCompress-Plugin

Das WordCompress-Plugin wurde ursprünglich zum Komprimieren von WordPress-Webseiten in ein komprimiertes Dateiformat verwendet. Das Plugin wird nicht mehr unterstützt und eingestellt. Es würde den Quellcode der Webseite in eine einzelne ausführbare Datei schreiben, indem alle Leerzeichen aus dem Quellcode der Seite entfernt würden. KOMPRIMIERTE Dateien werden zusammen mit dem ursprünglichen Quellcode der Webseite gespeichert, um die ursprünglichen Kommentare, Einrückungen und anderen Leerzeichen, die im ursprünglichen Quellcode der Seite enthalten sind, beizubehalten.
