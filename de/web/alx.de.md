## Was ist eine ALX-Datei?

Eine ALX-Datei ist eine ActiveX-Datei, die in Form von [HTML](/de/web/html/)-Komponenten in Webseiten eingebettet ist. ActiveX-Steuerelemente wurden früher als OLE-Steuerelemente oder OCXs bezeichnet. ALX-Dateien können verwendet werden, um eine Vielzahl von Funktionalitäten in eine Webseite einzufügen, z. B. das Hinzufügen eines Timer-Steuerelements zum regelmäßigen Aktualisieren des Seiteninhalts oder das Hinzufügen eines Popup-Fenstersteuerelements zum Anzeigen von Hilfe im Tooltip-Stil basierend auf Benutzeraktionen. ALX-Dateien werden mit verknüpft<Param> -Tag im HTML-Code einer Webseite. Sie können ALX-Dateien in Microsoft Internet Explorer, Microsoft ActiveX Control Pad und Texteditoren wie Notepad und Notepad++ öffnen.

## ALX-Dateiformat - Weitere Informationen

ALX-Dateien sind im Nur-Text-Dateiformat und können bearbeitet werden, indem sie in einem Texteditor geöffnet werden. ActiveX-Steuerelemente können in eine Webseite eingefügt werden, indem sie auf die Webseite gezogen werden. Um jedoch dynamische Webseiten zu erstellen, werden die Steuerelemente zusammen mit Hilfe von [JScript](/de/web/js/) integriert. Steuerelemente auf einer Webseite sind den Ereignissen und Werten der Steuerelemente zugeordnet, wodurch die vollständige Kontrolle über Benutzerinteraktionen gegeben wird.
