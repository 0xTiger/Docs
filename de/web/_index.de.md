Beim Erstellen einer Webseite werden eine Reihe von Dateitypen benötigt. Webdateiformate definieren die Standards für die Entwicklung von Webseiten und beziehen sich auf die Plattform, auf der diese erstellt werden. Es kann eine komplette Website aufgebaut werden, die sowohl aus statischen als auch aus dynamischen Webseiten besteht. Die meisten modernen Websites basieren auf serverseitiger Technologie wie Active Server Pages (ASP), die auf dem Webserver geladen und ausgeführt werden. Dazu gehören auch Cascading Styling Sheets (CSS) und Skriptdateien, die für die Gestaltung des allgemeinen Erscheinungsbilds der Benutzeroberfläche verwendet werden.

Gängige Webdateierweiterungen und die zugehörigen Dateiformate sind **[HTML](/de/web/html/)** (Hypertext Markup Language), **[ASP](/de/web/asp/) (Active Server Pages)**, und **CSS** (Cascading Style Sheets).

Haben Sie Fragen zu Webdateiformaten? Besuchen Sie unsere Community [Foren](https://forum.fileformat.com/c/web), um vom Wissen der Dateiformatexperten zu profitieren.
