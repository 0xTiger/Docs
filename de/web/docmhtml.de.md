## Was ist eine DOCMHTML-Datei?

Eine DOCMHTML-Datei ist eine alte Variante des Webarchivs, die von früheren Versionen von Microsoft Word gespeichert wurde. Es wird von der neuesten Version von Microsoft Word nicht mehr unterstützt. Eine DOCMHTML-Datei enthält eine HTML-Darstellung des ursprünglichen .doc-Word-Dokuments. Es kann sowohl in Microsoft Word als auch im Webbrowser geöffnet werden. Wenn ein DOCMHTML veröffentlicht wird, kann es auf einem Webserver gehostet werden, auf den Benutzer über eine Webanfrage zugreifen können. Ein weiteres ähnliches Format ist [DOCHTML](/de/web/dochtml/), das ebenfalls eingestellt wurde.

## DOCMHTML-Dateiformat - Weitere Informationen

DOCMHTML-Dateien wurden verwendet, um im HTML-Format gespeichert zu werden. Es könnte in einem Texteditor wie Notepad und Notepad++ geöffnet werden. Nachdem es als veraltet markiert wurde, führte Microsoft Word die Optionen zum Speichern von Word-Dokumenten als [MHTML](/de/web/mhtml/)- und [MHT](/de/web/mht/)-Dateien ein. Diese neuen Dateiformate können mit Webbrowsern geöffnet werden.
