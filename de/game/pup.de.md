## Was ist eine PUP-Datei?

Eine PUP-Datei ist eine Patchdatei für Systemaktualisierungen, die von der Systemsoftware für PlayStation 4 (PS4) und PlayStation 5 (PS5) verwendet wird. Es aktualisiert die Firmware des Spiels auf eine neuere Version, die Softwareverbesserungen in Bezug auf Fehlerbehebungen, neue Betriebssystemfunktionen und Verbesserungen in Bezug auf die Softwareleistung enthalten kann. Die PUP-Datei kann auch Patches enthalten, um Fehler zu beheben, die ansonsten zu Sicherheitsproblemen führen können. PUP-Dateien können auf das System heruntergeladen werden, indem Sie es mit dem Internet verbinden oder das Firmware-Update manuell auf einen Computer herunterladen und dann mit einem USB-Stick oder einem anderen Speichergerät auf das Gerät übertragen. Die frühere Version von PlayStation, dh PS3, verwendete auch die PUP-Datei zum Aktualisieren der Firmware.

## PUP-Dateiformat – Weitere Informationen

PUP-Dateien werden im binären Dateiformat [komprimiert] (/de/compression/) auf der Disc gespeichert. Die Details der Komprimierung sind jedoch nicht öffentlich verfügbar. Der Inhalt einer PUP-Datei kann mit dem Programm PS3 PUP Extractor unter Windows OS extrahiert werden. Um das Softwaresystem der PS3 zu aktualisieren, wird die PUP-Datei hochgeladen und im Ordner /PS3/UPDATE auf dem Gerät gespeichert. Dadurch kann die Systemaktualisierungsoption die Firmware-Aktualisierungsdatei finden und ausführen.

### PUP-Dateiformat-Namenskonvention

* „PS3UPDAT.PUP“ – Der Standard-Dateiname, der für PlayStation 3-Updates verwendet wird
* „PS4UPDATE.PUP“ – Der Standard-Dateiname, der für PlayStation 4-Updates verwendet wird
* „PS5UPDATE.PUP“ – Der Standard-Dateiname, der für PlayStation 5-Updates verwendet wird

## Verwenden der PUP-Datei zum Zurücksetzen der PlayStation auf die Werkseinstellungen

PUP-Dateien können verwendet werden, um Ihr PlayStation-Gerät auf die Werkseinstellungen zurückzusetzen. Dadurch werden alle Daten vom Gerät überschrieben, die vorhandene Firmware entfernt und es als brandneues Gerät eingestellt. Durch das Zurücksetzen Ihrer PlayStation auf die Werkseinstellungen werden auch alle gespeicherten Spieldaten entfernt.
