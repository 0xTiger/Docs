## Was ist eine OSK-Datei?

Eine OSK-Datei ist eine Layout- oder Skin-Datei, die vom Rhythmusspiel osu! wo Spieler Maus und Tastatur synchron zur Musik verwenden. Es enthält das Layout der Editoroberfläche und kann von den Spielern an ihre Vorlieben angepasst werden. oh! speichert Beatmap-Daten im Dateiformat [OSZ](/de/game/osz/).

**MIME-Typ des OSK-Dateiformats:** x-osu-skin-archive

Sie können die OSK-Datei mit [osu!](https://osu.ppy.sh/home) öffnen.

## OSK-Dateiformat

OSK-Dateien werden als [ZIP](/de/compression/zip/) komprimierte Dateien auf der Disc gespeichert. Sie können einfach eine .osk-Datei in .zip speichern und das Archiv extrahieren, um seinen Inhalt anzuzeigen. Die extrahierten Inhalte müssen im Verzeichnis \Skins\ abgelegt werden, um in das Spiel geladen zu werden.

## Was kann mit der OSK-Datei angepasst werden?

Die folgenden Spielelemente können mit der .osk-Datei angepasst werden.

* Kreise treffen
* Ergebnisanzeige
* Slider-Typ
* Mauszeiger
* Trefferstöße
