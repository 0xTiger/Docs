## Was ist eine FOMOD-Datei?

Eine FOMOD-Datei ist eine Spieldatei, die Modifikationen enthält, die für ein Spiel installiert werden sollen. Diese Dateien werden von Mod-Programmen wie Fallout Mod Manager (FOMM) und Nexus Mod Manager (NMM) installiert. Die Änderungen in .fomod-Dateien können sich auf jeden Aspekt des Spiels beziehen und werden von Spielern oder Community-Fans des Spiels veröffentlicht. Mods in Form von FOMOD können die Lebensdauer eines Spiels verlängern, indem sie kleine Änderungen oder eine vollständige Überarbeitung des Spiels vornehmen.

## FOMOD-Dateiformat - Weitere Informationen

FOMOD-Dateien werden normalerweise als komprimiertes Archiv mit den Komprimierungsformaten [.7z](/de/compression/7z/), [.zip](/de/compression/zip/) oder [.rar](/de/compression/rar/) gespeichert. In FOMM wird diese Datei geladen und auf dem System installiert.

## Wie installiere ich eine .fomod-Datei?

Eine FOMOD-Datei kann durch Doppelklicken installiert werden. Dadurch wird es mit dem FOOM mit dem Modusnamen in der Liste geöffnet, der durch Auswählen und Drücken der Aktivierungsoption aktiviert werden kann.
