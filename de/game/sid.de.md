## Was ist eine SID-Datei?

Eine SID-Datei ist eine Spielsicherungsdatei, die von der Steam-Anwendung erstellt wurde und zum Herunterladen und Verwalten von **Valve-Videospielen** unter Windows OS verwendet wird. Es speichert Spieldaten in komprimiertem Format und kann als Sicherungskopie von Spielen auf Speichermedien wie DVDs und USB-Speichergeräten verwendet werden. Die SID-Datei wird mit dem Steam-Backup-Dienstprogramm erstellt und ähnelt dem Dateiformat [CSD](/de/game/csd/) zum Speichern von Spiel-Backups.

## SID-Dateiformat – Weitere Informationen

SID-Dateien werden als Binärdateien auf der Disc gespeichert. Es wird immer von einer .sis- und einer .sim-Datei begleitet, die zum Extrahieren von Metadateninformationen zum Wiederherstellen der Backup-Spieldaten verwendet werden.
