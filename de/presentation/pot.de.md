## Was ist eine POT-Datei?

Dateien mit der Erweiterung .POT stellen Microsoft PowerPoint-Vorlagendateien dar, die von PowerPoint 97-2003-Versionen erstellt wurden. Dateien, die mit diesen Versionen von Microsoft PowerPoint erstellt wurden, sind im Binärformat im Vergleich zu Dateien, die in Office OpenXML-Dateiformaten mit höheren Versionen von PowerPoint erstellt wurden. Die so generierten Dateien können verwendet werden, um Präsentationen zu erstellen, die das gleiche Layout und andere Einstellungen haben, die für die Anwendung auf neue Dateien erforderlich sind. Diese Einstellungen können Stile, Hintergründe, Farbpaletten, Schriftarten und Standardeinstellungen umfassen. Solche Dateien werden generiert, um gebrauchsfertige Vorlagendateien für den amtlichen Gebrauch zu erstellen.

## Dateiformatspezifikationen ##

Dateiformatspezifikationen für das POT-Dateiformat sind nicht einzeln von Microsoft erhältlich. Das Vorlagenformat ist jedoch dasselbe wie das des binären Dateiformats [PPT](/de/presentation/ppt/) und kann zu Bearbeitungszwecken in MS PowerPoint geöffnet werden. Eine POT-Datei wird wie folgt durch die HEX-Kennung identifiziert:

```
D0 CF 11 E0 A1 B1 1A E1 00 00 00 00
```

Die mit dem POT-Dateiformat verbundenen MIME-Typen sind:

* application/vnd.ms-powerpoint [offiziell]
* Anwendung/mspowerpoint
* Anwendung/x-mspowerpoint
* Anwendung/Powerpoint
* Anwendung/X-Powerpoint
* application/x-dos_ms_powerpnt
* Anwendung/Topf
* Anwendung/x-soffic
