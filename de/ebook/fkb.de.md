## Was ist eine FKB-Datei? ##

FKB ist eine eBook-Dateierweiterung, die von Flipkart.com für die Flipkart-Softwareanwendung entwickelt wurde. Diese Art von Dateierweiterung ist normalerweise bei Windows 10-Benutzern zu finden. Es gibt eine Spezifikation in der FBK-Datei, dass die Größe je nach Länge des Buches und der Anzahl der enthaltenen Bilder zwischen einigen hundert Kilobyte und mehreren Megabyte liegen kann. Um die Datei von der Flipkart eBooks Android-Anwendung herunterzuladen, müssen Sie zuerst das eBook kaufen, dann in der Bibliothek darauf klicken und das Buch wird heruntergeladen. Wenn Sie das gekaufte Dokument nicht sehen können, aktualisieren Sie das System, damit es besser funktioniert.

## Unterstützte Betriebssysteme ##

Die folgenden Betriebssysteme unterstützen FKB-Dateien:

* Windows 7
* Windows 8
*Windows10
* Windows-Server 2003/2008/2012/2016
* Mac OS X
* FreeBSD
* iOS
* Linux
* Android
* NetBSD
* OpenBSD

## Probleme beim Öffnen einer FKB-Datei ##

* Unsachgemäße Installation von unterstützenden Programmen. Stellen Sie sicher, dass es ordnungsgemäß mit einer aktualisierten Version heruntergeladen wird
* Geringer Hardware-Platz
* Beschädigte oder infizierte Dateien. Dieses Problem kann leicht durch ein in Ihrem System installiertes Antivirenprogramm gelöst werden
  *	 Broken Links
