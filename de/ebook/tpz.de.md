## Was ist eine TPZ-Datei? ##

Die Dateierweiterung .tpz ist bei Amazon auch als Kindle Topaz eBook-Format bekannt. Linux, Mac OS und Windows sind einige der Plattformen, auf denen Dateien mit der Dateierweiterung .tpz verteilt werden können. Als Kindle Topaz eBook-Datei ist die TPZ-Datei so formatiert, dass sie wie herkömmliche eBooks aussieht, und enthält proprietäre Datenkompilierungsstandards von Amazon für das TPZ-Format. Diese Datei bietet erweiterte Unterstützung für eingebettete Schriftarten, und zusätzliche Details wie Anmerkungen und Lesezeichen können ebenfalls in die Kindle Topaz eBook-Dateien aufgenommen werden, obwohl diese Metadatendetails separat gespeichert werden. Die TPZ-Datei kann auch als Datendatei kategorisiert werden, die von Amazons Online-Datenbank für Kindle eBooks heruntergeladen werden kann.

## Probleme beim Öffnen einer TPZ-Datei? ##

Im Folgenden werden einige der häufigsten Probleme aufgeführt, die auftreten können:

* Infizierte Datei. Dies kann durch eine geeignete Antivirensoftware behoben werden, um es sauber zu machen
* Unzureichender Speicherplatz. Dazu müssen alle irrelevanten Anwendungen für eine bessere Funktion geschlossen werden
* Defekte Links können manchmal eine Ursache für Probleme sein
* Manchmal hat die Datei keine Zugriffsrechte. Stellen Sie in diesem Fall sicher, dass die Zugriffsrechte die Datei problemlos öffnen
