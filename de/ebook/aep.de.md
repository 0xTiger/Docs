## Was ist eine AEP-Datei? ##

Die Datei mit der Erweiterung .aep (was für **Activ E-Book Project** steht) wird als elektronisches Buch (eBook) klassifiziert. Die AEP-Dateien werden mit derselben Engine wie Internet Explorer erstellt, um 100 % Kompatibilität zu gewährleisten. Es enthält die Seiten, wie eine Person ein physisches Buch lesen kann. Die Seiten von AEP-Dateien enthalten normalerweise Text, Bilder, Animationen und Hyperlinks. Außerdem unterstützt es [HTML](/de/web/html/), [DHTML](/de/web/dhtml/), [JPEG](/de/image/jpeg/) und [PNG](/de/image/png/) Grafiken, Animationen , [JavaScript](/de/web/js/), VBScript und viele der Internet Explorer-Plug-Ins. Eine AEP-Datei kann als ausführbare Windows-Datei (EXE) gespeichert werden, sodass sie unter Windows95 oder späteren Versionen von Microsoft Windows geöffnet werden kann.
 


## AEP-Dateiformat ##

Die Dateien im AEP-Format werden mit einer AEC-Software (Active E-Book Compiler) erstellt. Diese Software verwendet eine ähnliche Engine wie der IE, um eBooks zu erstellen. Der Benutzer kann unbegrenzt Kopien von eBooks erstellen, ohne Kosten für die Nutzung des Programms zu zahlen. Der Verteilungsprozess ist ebenfalls einfach, da jedes eBook aus einem eigenständigen Windows-Programm besteht, das auf beliebigen Medien wie E-Mails, Internet-Downloads, Disketten, CD-ROMs, ZIP-Disketten usw. verteilt werden kann.

## Mögliche Probleme beim Öffnen der Datei

Wenn Sie die AEP-Datei nicht öffnen und ausführen können; Das bedeutet nicht, dass Sie keine geeignete Software auf Ihrem Gerät installiert haben. Möglicherweise gibt es einige andere Probleme, die verhindern, dass die Datei ordnungsgemäß funktioniert. Die möglichen Probleme könnten eines der folgenden sein:

- Beschädigung einer AEP-Datei
- Falsche Verknüpfung der AEP-Datei in Registrierungseinträgen
- Beschreibung des AEP aus der Windows Registry gelöscht
- Beschädigte Installation einer Anwendung, die das AEP-Format unterstützt
- Eine infizierte AEP-Datei mit einer unerwünschten Malware
- Der Computer hat zu wenig Gebrauchsbestand, um die AEP-Datei zu betreiben
- Die vom Computer zum Öffnen einer AEP-Datei verwendeten Treiber sind veraltet
