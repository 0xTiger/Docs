## Was ist eine KFX-Datei?

Eine KFX-Datei ist ein eBook, das im Amazon Kindle File Format 10 (KF10) für Kindle-Geräte erstellt wurde. Ältere Versionen von KIndle-Geräten verwenden die Dateiformate [AZW](/de/ebook/azw/) und [AZW3](/de/ebook/azw3/) (KF8). KFX ist eine Erweiterung früherer Versionen mit zusätzlichen Funktionen speziell in Bezug auf die Präsentation des Inhaltslayouts. Es führt neue Funktionen wie die Unterstützung für weiche Bindestriche mit einer neuen verbesserten Schriftsatz-Engine ein. Es führt auch ein neues Bilddateiformat namens JXR ein, das eine höhere Komprimierungsrate hat, und die neue Bookerly-Schriftart, eine exklusive Kindle-Schriftart, die eine verbesserte Lesbarkeit auf digitalen Geräten bietet. Das Format verfügt über erweiterte Funktionen wie mehrseitige Miniaturansichten, automatische Inhaltsanpassung um Bilder und Videos an festen Positionen, was zu einem verbesserten Seitenlayout und DRM führt. Um solche Dateien zu öffnen und anzuzeigen, können Sie die Amazon Kinde-App verwenden, die für Windows, macOS, Android und iOS verfügbar ist. Calibre ist eine weitere eBook-Reader-Softwareanwendung, die zum Lesen verschiedener eBook-Dateiformate verwendet werden kann.

## KFX-Dateiformat ##

KFX ist ein komplexes Format, da es mehrere verschiedene Funktionen von Kindle-Dateiformaten kombiniert, aber die internen Details über seine Dateistruktur sind nicht bekannt. Amazon hat sein DRM-Schema hinzugefügt, um digitale Rechte zu schützen und das KFX-Format vor Piraterie zu schützen.
