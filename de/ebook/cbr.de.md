## Was ist eine CBR-Datei?

Eine Datei mit der Erweiterung .cbr ist eine Sammlung von Bildern, die als komprimierte Binärdateien im RAR-Komprimierungsdateiformat gespeichert sind. Es enthält Bilder in [JPEG](/de/image/jpeg/), [PNG](/de/image/png/) oder [GIF](/de/image/gif/) Bildern, die Bilder in einem Comic darstellen und dem Benutzer angezeigt werden, wenn die Datei wird in E-Book-Readern wie Calibre verwendet. CBR ähnelt [CBZ](/de/ebook/cbz/), abgesehen davon, dass CBZ komprimiert und im Dateiformat [ZIP](/de/compression/zip/) gespeichert wird. Diese Dateien können in Comicsammlungsdateien wie [CBC](/de/ebook/cbc/) enthalten sein.

## CBR-Dateiformat

Eine CBR-Datei ist ein [RAR](/de/compression/rar/)-Archiv, das als Binärdatei auf der Disc gespeichert wird. Die Bilder in einer CBR-Datei werden gespeichert und Benutzern in geordneter Weise angezeigt, wenn sie von einem E-Book-Reader geöffnet werden. CBR-Dateien können mit jeder Dekomprimierungssoftware dekomprimiert werden, die das RAR-Dateiformat unterstützt. Auf diese Weise dekomprimierte Bilder befinden sich jedoch nicht in der gleichen Reihenfolge, in der sie in der visuellen Anzeige erscheinen, wenn sie in der Lesesoftware geöffnet werden.
