## Was ist eine EDN-Datei? ##

Der EDN-Dateityp bezieht sich hauptsächlich auf Acrobat von Adobe Systems Incorporated. Alle Spezifikationen von EDN werden ausgeführt, einschließlich konventionsgetaggter Rudimente. Nachfolgend finden Sie einige grundlegende Informationen zu EDN.

* EDIF-Implementierung Netzliste Dateityp Beschreibung
* Dateientwickler von Xilinx, INC
* Xilinx ISE Webpack-bezogene Software
* Softwareentwickler von Xilinx, INC
 

## Dateikompatibilität ##

* Acrobat von Adobe Systems Incorporated. Dies ist eine Kategorie von Computerprogrammen, die von Adobe Systems entwickelt wurde. Der Hauptzweck dahinter war das Anzeigen, Generieren, Bedienen und Verwalten von Dateien im Portable Document Format (PDF) von Adobe. Einige der Software in dieser Kategorie sind kostenlos und einige sind kommerziell
* Die von Microsoft Developer entwickelte EDIF-Netzlistendatei ist auch als Electronic Design Interchange Format bekannt und wird hauptsächlich als neutrales Format verwendet, in dem elektronische Netzlisten und Schaltpläne gespeichert werden müssen
