## Was ist eine TR3-Datei? ##

Die TR3-Datei ist ein TomeRaider 3 eBook, das Unterstützung für schnelles Suchen und Komprimieren bietet. Das kann über das TomeRaider-bezogene Programm richtig funktionieren. Yadabyte ist der Entwickler dieser Software, ein britisches Software- und Webentwicklungsunternehmen mit Sitz in Großbritannien. Die TomeRaider eBook-Reader-Anwendung kann für das ordnungsgemäße Funktionieren und Verwalten des Inhalts dieser TR3-Dateien verwendet werden. Dieser verwandte eBook-Reader kann auf Computern installiert werden, die auf Microsoft Windows-basierten Systemen und vielen übertragbaren Gadgets laufen. Die TR3-Datei gehört zur Gruppe der eBook-Dateien, die in Betriebssystemen wie Windows 10, Windows 7, Windows 8 / 8.1, Windows Vista, Windows XP verwendet werden.

TR3-spezifische **HTML-Tags** lauten wie folgt:

|Tag|Beschreibung|
---|---|
|\<new> |Um TomeRaider-Dateien aus Textdateien zu erstellen, ist das New-Tag alles, was benötigt wird.<new> Tag definiert den Beginn der Seiten.|
|\<CATDEF> ... \</CATDEF> |definiert den Kategorienamen|
|\<META> ... \</META> |ist ein Tag, mit dem alle im Dateiformat verwendeten Metadaten definiert werden. Dieses Tag enthält eine Reihe von Parametern.|
|\<EXPMSG> |Dieses Tag steuert die Nachricht, die angezeigt wird, wenn eine Datei abgelaufen ist. Immer wenn ein Benutzer versucht, eine Seite anzuzeigen, nachdem die Datei abgelaufen ist, wird anstelle des eigentlichen Seitentexts die Ablaufmeldung angezeigt.|
|\<DIEMSG> |Dieses Tag ähnelt EXPMSG. Es wird verwendet, um eine Nachricht anzuzeigen, wenn die Datei stirbt.|
|\<ENGMSG> |Mit diesem Tag wird die Nachricht erstellt, die für verschlüsselte Seiten angezeigt wird. Die Meldung wird anstelle des Seitentexts angezeigt, wenn der Benutzer versucht, eine verschlüsselte Seite zu öffnen, bevor er sie entsperrt.|
|\<expmsg> ,\<diemsg> und \<encmsg> |Im Seitentext ignoriert. Sie müssen im Datei-Header-Abschnitt vor dem ersten platziert werden<new> Tag.|
|\<SELECTION> . \</ SELECTION> |Wird zur Unterstützung von Abfragen verwendet. Dieses Tag muss vor dem ersten stehen<new> Schild. Es stellt die Selektionsdaten für die Abfrage des Benutzers zusammen.|
|\<catset> . \</catset> | Wird verwendet, um jedem Thema Kategorienamen zuzuweisen.|


## Probleme beim Öffnen einer TR3-Datei ##

Hier ist die Liste einiger häufiger Probleme, die auftreten und Fehlfunktionen des Dateiformats verursachen können:

* Beschädigte Datei
* Infizierte Datei aufgrund eines Virus
* Falsche Links zur TR3-Datei bei Archivzugriffen
* Kein Zugriffsrecht im System zum Öffnen der Dateien
* Veraltetes Laufwerk in Ihrem System
* Entfernung des Berichts des TR3 aus dem Windows-Archiv
* Die Kontaktaufnahme mit dem Entwickler ist möglicherweise eine bessere Option für eine bessere Ausgabe
