## Was ist eine EBK-Datei?
Eine EBK-Erweiterung wird von der Software **eBook Pro Viewer** verwendet, wenn sie eine eBook-Datei speichert. eBook Pro ist eine Software, die eBooks erstellen und urheberrechtlich geschützte Informationen schützen kann. Eine EBK-Datei enthält die verschlüsselte und komprimierte Version eines E-Books, das mit **eBook Pro Viewer** gelesen werden kann; nach Bereitstellung der entsprechenden Anmeldeinformationen für die Authentifizierung.
