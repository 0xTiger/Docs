## Was ist eine TCR-Datei?

In den frühen 90er Jahren wurde das TCR-Dateiformat für die alten **Psion Series 3**-Palmtop-Geräte eingeführt. Das Unternehmen verwendete seine plattformspezifische Komprimierung und dieses Format unterstützt die Textformatierung nur sehr eingeschränkt. Es ist ein ZVR-basiertes Dateiformat und bietet eine vergleichsweise bessere Komprimierungsrate als PalmDOC. Es funktioniert auch sehr gut mit FontMachine. Obwohl das TCR-Dateiformat zu einem abgekündigten Gerät gehört, kann dieses Dateiformat dennoch von einigen eReadern gelesen werden. Dieses Dateiformat kann auch auf einem Desktop mit Sumatra PDF und mit der Calibre-Anwendung unter Windows und Linux angezeigt werden.

## Technische Details

Da das TCR-Dateiformat auf einem ZVR-Kompressor basiert, der in der Lage ist, viele Dateien um etwa 50 % ihrer ursprünglichen Größe zu reduzieren. Das von over verwendete Komprimierungsschema ist etwas alt. Die komprimierte Datei beginnt mit einem Wörterbuch der 256 möglichen Symbole, die in der komprimierten Datei vorkommen könnten. Das Wörterbuch enthält für jedes dieser Symbole eine feste Alternativzeichenfolge. Die Dekomprimierung ist sogar in OPL sehr schnell geworden und kann an jedem Punkt in der komprimierten Datei beginnen.

## Probleme beim Öffnen einer TCR-Datei ##

Wenn Sie die TCR-Datei nicht öffnen und ausführen können; Dies bedeutet nicht, dass Sie keine geeignete Software auf Ihrem Gerät installiert haben. Möglicherweise gibt es einige andere Probleme, die verhindern, dass die Datei ordnungsgemäß funktioniert. Die möglichen Probleme könnten eines der folgenden sein:

- Beschädigung einer TCR-Datei.
- Falsche Verbindung zur TCR-Datei in Registrierungseinträgen.
- Beschreibung des TCR aus der Windows Registry gelöscht
- Beschädigte Installation einer Anwendung, die das TCR-Format unterstützt
- Eine infizierte TCR-Datei mit unerwünschter Malware.
- Der Computer hat zu wenig Gebrauchsbestand, um die TCR-Datei zu betreiben.
- Die vom Computer zum Öffnen einer TCR-Datei verwendeten Treiber sind veraltet.
