## Was ist eine QMK-Datei?

Eine Datei mit der Erweiterung .qmk ist allgemein als **YanCEyDesktop Quickmarks-Datei**-Format bekannt. Die YanCEyWare führte das Dateiformat YanCEyDesktop Quickmarks ein. Die QMK-Datei ist mit der [YBK](/de/ebook/ybk/)-Datei verwandt und wird von einem Programm unterstützt, das für Computer verfügbar ist, auf denen das Windows-Betriebssystem ausgeführt wird. Eine Datei mit einer QMK-Erweiterung wird als Textdatei klassifiziert, die es Benutzern ermöglicht, einfach zu gespeicherten Orten in eBooks zu navigieren. YanCEyDesktop ist die empfohlene Software zum Verwalten von QMK-Dateien. Die Software **YanCEyDesktop** wurde von **D Remington** erstellt.

## Probleme beim Öffnen einer QMK-Datei

Wenn Sie die QMK-Datei nicht öffnen und ausführen können; Dies bedeutet nicht, dass Sie keine geeignete Software auf Ihrem Gerät installiert haben. Möglicherweise gibt es einige andere Probleme, die verhindern, dass die Datei ordnungsgemäß funktioniert. Die möglichen Probleme könnten eines der folgenden sein:

- Beschädigung einer QMK-Datei
- Falsche Verbindung zur QMK-Datei in Registrierungseinträgen
- Beschreibung des QMK aus der Windows-Registrierung gelöscht
- Beschädigte Installation einer Anwendung, die das QMK-Format unterstützt
- Eine infizierte QMK-Datei mit unerwünschter Malware
- Der Computer hat zu wenig Gebrauchsbestand, um mit der QMK-Datei zu arbeiten
- Die vom Computer zum Öffnen einer QMK-Datei verwendeten Treiber sind veraltet
