## Was ist eine LIT-Datei?

Die LIT-Dateien gehören zum alten **Microsoft Reader**, der zum Lesen von E-Books verwendet wurde. Die von Microsoft Reader angezeigten eBooks haben die Endung **.lit**, die zu **Literatur** abgekürzt wird. Das LIT-Dateiformat besteht aus einer Microsoft ITOL/ITLS HTML Help 2.0-Datei, die alle möglichen erstaunlichen Funktionen wie [LZX](/de/compression/lzx/)-Komprimierung, die binäre Darstellung von Markup-Material und beliebige optionale Hilfsdaten bietet jeder Strom. Buchtext ist eine beliebige Anzahl von OEBPS 1.0-Markup-Streams und eine Teilmenge von OEBPS 1.0-CSS. Bilder sind [GIF](/de/image/gif/), [JPEG](/de/image/jpeg/) und [PNG](/de/image/png/).

## Wie lange Microsoft Reader die LIT-Dateien angezeigt hat

Der **Microsoft Reader** ist eine Anwendung zum Lesen von E-Books, die von Microsoft eingestellt wurden. Die erste Version von Microsoft Reader wurde im August 2000 veröffentlicht, die ein eigenes LIT-Format verwendete. Es war für PDAs und Windows-Computer verfügbar. Der Name der App wurde später auch für eine Anwendung (die ursprünglich mit Windows 8 veröffentlicht wurde) zum Lesen von [PDF](/de/pdf/)- und [XPS](/de/page-description-language/xps/)-Dateien verwendet. Die Anwendung Microsoft Reader wurde 2018 eingestellt.
