## Was ist eine PIF-Datei?
Eine PIF-Datei besteht aus Informationen, die verwendet werden, um zu bestimmen, wie ein MS-DOS-basiertes Programm ausgeführt werden soll. Es kann auch als Verknüpfung einer ausführbaren Datei verwendet werden und wird normalerweise erstellt, wenn der Benutzer eine Verknüpfung zu einem DOS-Programm erstellt oder die Eigenschaften eines Programms aktualisiert. Microsoft Windows erkennt die PIF-Dateien als ausführbare Programme und kann wie andere DOS-basierte Programme ausgeführt werden. Die PIF-Dateien werden heute aufgrund des Fehlens von DOS-Anwendungen nicht häufig in Software verwendet.

## PIF-Dateiformat
Die PIF-Datei besteht im Grunde aus nur einem Datenblock, in dem die Parameter gespeichert sind, die für die Ausführung unter TopView erforderlich sind. Für Schalter, ob das Fenster beim Beenden des Programms ausgeschaltet werden soll oder nicht, enthalten die Felder für den Fenstertitel die maximale und minimale Menge an RAM und Bitmaps. Bei der Anpassung des Systems für den Einsatz unter Windows standen die Entwickler vor dem Problem, dass es zusätzliche Schalter gab, die für TopView nicht galten. Anstatt die neuen Schalter am Ende der Datei einzubauen, erstellen sie die Datei als Datenbankdatei, die beliebig viele Einträge enthält. Auf diese Weise bestand die Datei aus einer Reihe von Header-Bereichen, die beschreiben, welches Betriebssystem den Abschnitt lesen sollte, und einem Offset zum nächsten Abschnitt. Die Systeme lesen die Liste durch, bis sie die am besten geeignete gefunden haben. Daher hinterließ diese Theorie ein Problem der Abwärtskompatibilität. Die Datei wurde schließlich neu angeordnet, wobei der erste Header nach den Startdaten erschien, wodurch die ersten 253 Bytes der Datei im gleichen Format wie zuvor blieben.
