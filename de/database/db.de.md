## Was ist eine DB-Datei?

Eine Datei mit der Erweiterung db ist eine generische Datenbankdatei zum Speichern von Daten. Für eine solche Datenbankdatei sind keine offiziellen Vorgaben zu beachten. Daten werden in einem strukturierten Format innerhalb der Datei in Form von Tabellen, Feldern, Datentypen und Feldwerten organisiert. DB-Dateien können auch durch Lesen von Datenblattdateien mit Daten in Zeilen und Spalten erstellt werden. Der Inhalt von DB-Dateien kann außerdem zu Berichtszwecken in die Dateiformate [CSV](/de/spreadsheet/csv/) und [PDF](/de/pdf/) exportiert werden. DB-Dateien können in Anwendungen wie SQLite und LibreOffice geöffnet werden.
