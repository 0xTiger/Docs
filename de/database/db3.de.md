## Was ist eine DB3-Datei?
Die DB3-Datei ist eine Datenbankdatei, die von der SQLite-Software erstellt wurde, einem leichten, eigenständigen Datenbankprogramm, das Datenbanken mit einfachen Dateien erstellt. enthält die Datenbank Anatomie sowie Datensätze; Wird häufig zum Abrufen oder Speichern strukturierter Daten mit SQL verwendet. Diese Dateien können in intelligenten Geräten oder dort verwendet werden, wo die Aufzeichnung oder andere Datenverwaltung erforderlich ist, jedoch in einer Umgebung mit wenig Platz.


## DB3-Dateiformat
Das DB3-Dateiformat ist mit RDBMS (Relational Database Management System) SQLite verbunden, einer beliebten Wahl für eingebettete Datenbanken. In einer SQLite_3-Spezifikation ist keine bestimmte Dateierweiterung definiert, sie kann Erweiterungen wie [.dbf](/de/database/dbf/) und [.sqlite](/de/database/sqlite/) verwenden.

### Die Datenbankspezifikation
Üblicherweise kann das DB-Dateiformat von SQLite, Version 3, als das DB3-Dateiformat angesehen werden, das seit Juni 2004 als öffentlich dokumentiertes natives Format für die SQLite-Datenbank-Engine verwendet wird. Das DB3-Dateiformat ist plattformübergreifend und zwischen Big-Endian übertragbar und Little-Endian-Architekturen oder 32-Bit- und 64-Bit-Systeme. Diese Eigenschaften machen DB3 zu einer beliebten Wahl als Dateiformat für Anwendungen. Die Hauptdatei der SQLite_3-Datenbank (DB3) besteht aus einer oder mehreren Seiten. Alle Größen aller Seiten sind in derselben Datenbank gleich. Die Seitengröße in Byte ist eine Zweierpotenz zwischen 512 und 65536 einschließlich.
