## Was ist eine ITW-Datei?

Eine ITW-Datei ist ein Datenbankdateiformat, das von der CRM-Anwendung [InTouch](https://www.intouchsoftware.co.uk/InTouch) verwendet wird. Es speichert Inhalte im proprietären Dateiformat von InTouch und seine internen Dateiformatdetails sind nicht verfügbar. ITW-Dateien speichern benutzerdefinierte Informationen zum Speichern von Namen und Adressen, die weiter zum Drucken von Umschlägen und Etiketten verwendet werden. Diese Informationen werden auch zum Versenden von E-Mails, Textnachrichten, attraktiven E-Newslettern, Erstellen von wiederverwendbaren Vorlagen, Anhängen und Links verwendet. Daten aus CRM können auch in das ITW-Format exportiert werden.
