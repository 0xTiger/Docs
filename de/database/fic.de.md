## Was ist eine FIC-Datei?

Die FIC-Datei in WinDev ist eine kompilierte Version einer HyperFileSQL-Datenbank. Das Akronym FIC steht auf Französisch für „Fichier de Code“, was auf Englisch „Code-Datei“ bedeutet. In WINDEV ist eine FIC-Datei eine kompilierte Datei, die den ausführbaren Code eines Projekts enthält. Die Datei hat die Erweiterung .fic und wird erstellt, wenn Sie ein WINDEV-Projekt kompilieren. Sobald ein Projekt in eine FIC-Datei kompiliert ist, kann es als ausführbares Programm oder Installationspaket an Benutzer verteilt werden.

Die FIC-Datei ist mit WINDEV verwandt, einem Softwareentwicklungstool, das von PCSoft, einem französischen Unternehmen, entwickelt wurde. Es ermöglicht Entwicklern, Windows-, Web- und mobile Anwendungen mit einer einzigen Codebasis zu erstellen. Es enthält eine visuelle Entwicklungsumgebung, eine Programmiersprache (WLangage) und eine Vielzahl vorgefertigter Steuerelemente und Funktionen für allgemeine Aufgaben. Es wird hauptsächlich für Geschäftsanwendungen verwendet und hat eine starke Benutzerbasis in Europa.

## Beziehung zur HyperFileSQL Mobile-Datenbank

In WinDev enthält eine für eine HyperFileSQL Mobile-Datenbank erstellte FIC-Datei den ausführbaren Code für die Datenbank sowie die in der Datenbank enthaltenen Daten. HyperFileSQL Mobile ist ein Verwaltungssystem für relationale Datenbanken, das in WINDEV enthalten ist und speziell für mobile und Offline-Anwendungen entwickelt wurde. Die FIC-Datei kann als eigenständige ausführbare Datei oder als Teil eines Installationspakets verteilt und auf dem Zielgerät ausgeführt werden, um auf die Daten in der HyperFileSQL Mobile-Datenbank zuzugreifen und diese zu verwalten. Wenn die FIC-Datei ausgeführt wird, öffnet sie die Datenbank und macht die Daten für die Verwendung in der mobilen Anwendung verfügbar.
