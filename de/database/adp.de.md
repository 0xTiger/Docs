## Was ist eine ADP-Datei?

Eine Datei mit der Erweiterung .adp ist eine Microsoft Access-Projektdatei, die die OLE DB-Komponentenarchitektur verwendet, um eine direkte und effiziente Verbindung zu einer Microsoft SQL Server-Datenbank bereitzustellen. Solche Dateien enthalten nicht die eigentlichen Tabellen, Datenbankdiagramme oder andere Datenbankelemente. ADP-Dateien können mit Access 2007 und 2010 erstellt werden. Dateien, die mit früheren Versionen von Access erstellt wurden, können auch mit Access 2007 und 2010 geöffnet werden. ADP-Dateien können mit Microsoft Access 365 geöffnet werden.

## ADP-Dateiformat – Weitere Informationen

Mit einem als ADP erstellten Access-Projekt können Sie Designänderungen an SQL Server-Objekten wie Tabellen und Ansichten vornehmen. Es ermöglicht auch das Erstellen, Bearbeiten und Verwenden von SQL-Funktionen wie Datenbankdiagrammen, gespeicherten Prozeduren und benutzerdefinierten Funktionen. Dies bietet Vorteile gegenüber der Verknüpfung mit einer SQL Server-Datenbank, bei der Sie keine Änderungen am Design vornehmen können und nur Verknüpfungen zu SQL Server-Tabellen und -Ansichten herstellen können.

### Datentypen und Diagrammunterstützung

**Datum/Uhrzeit-Datentypen**

Die folgenden neuen Datentypen für Datum/Uhrzeit werden in Access 2010 unterstützt.

* ZEIT
* DATUM
* DATUMZEIT2
* DATETIMEOFFSET

**Datentypen variabler Länge**

Die folgenden Datentypen variabler Länge können in Access 2010-Projekten verwendet werden:

* VARBIN(MAX)
* VARCHAR(MAX)
* NVARCHAR(MAX)
