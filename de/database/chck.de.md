## Was ist eine CHCK-Datei?

Als Ergebnis der Integritätsüberwachungsprüfungen auf Microsoft Exchange Server wird eine CHCK-Datei erstellt. Es enthält Informationen über den Gesamtstatus und die Operationalisierung der Datenbank des Servers. Microsoft Exchange Server wird hauptsächlich als E-Mail-Server verwendet und ermöglicht Ihnen die Pflege Ihrer Geschäftskontakte und Kalender. Exchange-Serveradministratoren verwenden EseUtil, um den Datenbankzustand zu überprüfen.

## CHCK-Dateiformat

Das interne Dateiformat der CHCK-Datei ist nicht bekannt und es ist kein Referenzmaterial zum Lesen/Schreiben von CHCK-Dateien verfügbar.
