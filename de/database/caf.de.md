## Was ist eine CAF-Datei?

Eine CAF-Datei ist eine Katalogdatei, die mit dem Medienverwaltungssoftware-Tool **Cathy** erstellt wurde. Es enthält eine Liste des indizierten Katalogs, der auf alle verfügbaren Medien auf externen Geräten wie Flash-Disks und CDs verweist. Mithilfe dieser Katalogdateien können Sie eine saubere Liste von Medien verwalten, die leicht durchsuchbar und zugänglich ist. CAF-Dateien bieten die Möglichkeit, Medien nach Name, Größe und Datum zu durchsuchen.

## CAF-Dateiformat

CAF-Dateien werden im Textdateiformat gespeichert. Das bedeutet, dass Sie CAF-Dateien in jedem Texteditor wie Notepad, Notepad++ und Apple TextEdit öffnen können. Cathy bietet die Möglichkeit, die Verzeichnisstruktur jedes Katalogs in das TXT-Format zu exportieren und so Listen mit Ordnern und Dateien zu erstellen.

### CAF-Datei öffnen

Sie können CAF mit der offiziellen Cathy-Software öffnen. Dazu müssen Sie Ihre CAF-Datei im selben Verzeichnis ablegen, in dem sich die Datei Cathy.exe befindet. Cathy zeigt den Katalog beim Start automatisch an.

Platzieren Sie dazu Ihre CAF-Datei im selben Verzeichnis wie Ihre Cathy.exe-Datei. Dann öffnen Sie Cathy. Ihr CAF-Katalog wird in Cathy angezeigt.
