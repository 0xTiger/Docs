## Was ist eine ACCDE-Datei?

Eine Datei mit der Erweiterung .accde ist ein Microsoft Access 2007-Dateiformat zum Erstellen von Nur-Ausführen-Datenbankdateien zum Schutz von benutzerdefiniertem Datenbankcode. Es kompiliert den gesamten VBA-Code nur in ausführbare Dateien, sodass er nicht angezeigt oder bearbeitet werden kann. Das ACCDE-Dateiformat ersetzt das MDE-Dateiformat, das von früheren Versionen von Microsoft Access verwendet wurde. ACCDE sind [ACCDB](/de/database/accdb/)-Dateien im Nur-ausführbar-Modus und kompakter, da kein Quellcode vorhanden ist. Die kompakte Größe dieser Dateien verbessert die Speichernutzung sowie die Anwendungsleistung. ACCDE-Dateien können mit Microsoft Access 365 geöffnet werden.

## ACCDE-Dateiformat

Alle Microsoft Access 2007+-Dateien verwenden Jet ACE (Access-Datenbankmodul), das die ACCDB-Dateiformatfamilie verwendet. Microsoft stellt jedoch weder eine Open-Source-Implementierung noch eine detaillierte technische Spezifikation bereit.
