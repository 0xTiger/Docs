## Was ist eine 4DL-Datei?

Eine 4DL-Protokolldatei wird verwendet, um Änderungen an einer 4D-Datenbankdatei (.4DD) aufzuzeichnen. Diese Datei wird zusammen mit der Datenbankdatei gespeichert und hat einen ähnlichen Dateinamen. Sein Zweck besteht darin, die in der Datenbank durchgeführten Aktualisierungen genau zu verfolgen und umfassend aufzuzeichnen. Eine [4DD-Datei](/database/4dd/) wird im Vergleich zur 4DL-Datei hauptsächlich von 4D Inc. mit den 4. Dimensionen in Verbindung gebracht.

## 4DL-Dateiformat – Weitere Informationen

Normalerweise werden mehrere 4DL-Dateien zusammen mit einer 4DD-Datei gespeichert. Daher kann die erste Version der Protokolldatei als 0001.4dl benannt werden, die späteren Versionen der Protokolldateien werden jedoch als 0002.4dl, 0003.4dl usw. gespeichert. Wenn nun die Protokolldatei selbst dreimal hintereinander gespeichert wird, wird sie als "db1[0005-0003].4dl" gekennzeichnet.
