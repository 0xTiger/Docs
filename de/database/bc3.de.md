## Was ist eine BC3-Datei?

Eine BC3-Datei ist eine Datenbankdatei zum Speichern von Informationen im Zusammenhang mit dem Baudatenbank-Austauschformat. Es wurde auf der Grundlage des spanischen Standardformats für den Austausch von Baudatenbanken entwickelt. In einer BC3-Datei werden Informationen gespeichert, die Datensätze enthalten, einschließlich Details zu Baumaterialien, Materialkosten und Arbeitspreisen. Die Daten können zwischen anderen Anwendungen ausgetauscht werden, ohne dass die Daten in andere Formate konvertiert werden müssen.

## BC3-Dateiformat

BC3-Dateien werden als Binärdateien gespeichert, wobei die Details gemäß den FIEBDC-Standards angegeben werden. Die Implementierung von BC3-Dateien hat verschiedene Revisionen durchlaufen und es gibt die folgenden Versionen des BC3-Dateiformats:

* FIEBDC-3/2004
* FIEBDC-3/2007
* FIEBDC-3/2012
* FIEBDC-3/2016

FIEBDC ist die Abkürzung für Formato de Intercambio Estándar – Bases de Datos de Construcción in spanischer Sprache.
