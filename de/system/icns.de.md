## Was ist eine ICNS-Datei?

Eine ICNS-Datei ist ein Dateiformat, das unter macOS zum Speichern von Symbolen für Anwendungen, Ordner und andere Elemente verwendet wird. Es handelt sich um ein Containerdateiformat, das mehrere Bildgrößen und Auflösungen desselben Symbols enthalten kann.

ICNS-Dateien enthalten normalerweise Bilder in mehreren Auflösungen, die von 16 x 16 Pixel bis 1024 x 1024 Pixel reichen, um unterschiedliche Anzeigegrößen und Auflösungen auf macOS-Geräten zu berücksichtigen. Das Dateiformat verwendet einen Komprimierungsalgorithmus, um die Dateigröße der Bilder zu reduzieren.

ICNS-Dateien können mit verschiedenen Softwaretools wie Icon Composer oder Sketch erstellt und verschiedenen Elementen auf einem macOS-System zugewiesen werden, beispielsweise Anwendungen, Ordnern und Dokumenten. Wenn auf einem macOS-System ein Element ausgewählt wird, ruft das Betriebssystem die entsprechende Größe und Auflösung des Symbols aus der ICNS-Datei ab und zeigt es auf dem Bildschirm an.

## ICNS-Dateiformat – Weitere Informationen

ICNS-Dateien sind Containerdateien, die mithilfe einer Technik namens "Symbolfamilie" mehrere Bildgrößen und Auflösungen desselben Symbols enthalten können. Eine Symbolfamilie besteht aus einer Gruppe von Bildressourcen, die jeweils dasselbe Symbol darstellen, jedoch in unterschiedlichen Größen oder Auflösungen. Jede Bildressource innerhalb einer Symbolfamilie verfügt über einen eindeutigen Typ und eine eindeutige ID, die die Bildgröße und -auflösung identifiziert. Die unterschiedlichen Bildgrößen und Auflösungen werden in derselben ICNS-Datei unter Verwendung unterschiedlicher Typen und IDs gespeichert.

Wenn auf einem macOS-System ein Element ausgewählt wird, ruft das Betriebssystem anhand seines Typs und seiner ID die entsprechende Größe und Auflösung des Symbols aus der ICNS-Datei ab und zeigt es auf dem Bildschirm an. Dadurch kann das macOS-System hochwertige Symbole auf Bildschirmen mit unterschiedlichen Auflösungen, beispielsweise auf Retina-Displays, anzeigen und gleichzeitig eine kleinere Dateigröße beibehalten.

ICNS-Dateien unter macOS ähneln .ICO-Dateien unter Windows darin, dass sie beide als Dateiformate zum Speichern von Symbolen dienen. Genau wie ICNS-Dateien können ICO-Dateien mehrere Bildgrößen und Auflösungen desselben Symbols enthalten, die zur Anzeige des Symbols auf unterschiedlichen Bildschirmgrößen und Auflösungen auf Windows-Geräten verwendet werden.

Es gibt jedoch einige Unterschiede zwischen ICNS- und ICO-Dateien. ICNS-Dateien können beispielsweise Bilder mit höherer Auflösung speichern als ICO-Dateien, wodurch sie sich besser für die Verwendung auf hochauflösenden Displays wie Retina-Displays auf macOS-Geräten eignen. Darüber hinaus können ICNS-Dateien Bilder mit einem Alphakanal enthalten, was transparente Hintergründe ermöglicht, was in ICO-Dateien nicht unterstützt wird.
