## Was ist eine FDF-Datei?

Eine FDF-Datei (**Forms Data Format**) ist ein Textdokument, das durch Exportieren von Daten aus den Formularfeldern einer [PDF](/de/pdf/)-Datei generiert wird. Es enthält nur Textfelddaten, die aus den in einer PDF-Datei verfügbaren Formularfeldern extrahiert werden. Dies führt zu einer vergleichsweise kleinen Datendatei, da die exportierten Daten das Formular selbst nicht enthalten. Adobe Acrobat bietet die Funktion zum Exportieren von Formularfelddaten mithilfe der Option „Formulardaten exportieren“ im Anwendungsmenü.

## FDF-Dateiformat – Weitere Informationen

FDF ist ein Nur-Text-Format und ist Teil des [ISO 32000-Standards](https://www.iso.org/standard/51502.html) für Portable Document Format. Es wurde von Adobe entwickelt, um den Import und Export von Daten aus Acrobat Forms oder AcroForms zu ermöglichen.

Es gibt zwei Arten von FDF-Dateien:

• „Classic FDF“ – Liefert Daten zum Ausfüllen eines bestehenden statischen Formulars.

• „Template FDF“ – Erstellt ein neues PDF basierend auf Vorlagen aus angegebenen PDF-Dateien. Formulare innerhalb des neuen Dokuments werden durch die Bereitstellung von Daten ausgefüllt.

## Erstellen von FDF mit Adobe Acrobat

Mit dem [FDF-Toolkit](https://opensource.adobe.com/dc-acrobat-sdk-docs/) von Adobe können Sie FDF-Dateien aus Textdaten erstellen.
