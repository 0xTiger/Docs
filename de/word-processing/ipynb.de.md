## Was ist eine IPYNB-Datei?

Eine Datei mit der Erweiterung .ipynb ist eine Jupyter Notebook-Datei, die den gesamten Inhalt von Notebook enthält, das mit der Jupyter Notebook-Webanwendungssitzung erstellt wurde. [Jupyter Notebook](https://jupyter.org/) ist eine interaktive Webanwendung zum Erstellen und Freigeben von Computerdokumenten zur Analyse mit Python.

Eine IPYNB-Datei dient als vollständige Rechenaufzeichnung einer Sitzung und enthält Informationen wie Recheneingaben und -ausgaben, mathematische Funktionen und eine umfassende Darstellung der resultierenden Objekte wie Bilder und erklärenden Text.

## IPYNB-Dateiformat

IPYNB werden intern im Dateiformat [JSON](/de/web/json/) gespeichert, das ein offenes Standarddateiformat für die gemeinsame Nutzung von Daten ist. Aus diesem Grund sind IPYNB-Dateien für Menschen lesbar und leicht verständlich. JSON macht es auch einfach, diese Dateien mit anderen zu teilen.

## Wie konvertiert man IPYNB-Dateien?

IPYNB-Dateien können mit dem Befehl nbconvert von der Jupyter-Schnittstelle in verschiedene Dateiformate exportiert werden. Dies beinhaltet die Konvertierung von IPYNB in:

* Pdf
*HTML
* Umstrukturierter Text
* Latex

## Wie kann ich die IPYNB-Datei anzeigen?

Es gibt zwei Methoden zum Anzeigen oder Öffnen von IPYNB-Dateien.

1. Der Benutzer kann das .ipynb-Notebook-Dokument auf eine öffentliche URL hochladen und es über Jupyter Notebook Viewer öffnen<nbviewer> die das Notizbuch als statische Webseite von der URL rendert.
1. Laden Sie die Dateien herunter und öffnen Sie sie in ihrer eigenen Instanz von Jupyter Notebook
