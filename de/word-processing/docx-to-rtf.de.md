## Informationen zur Konvertierung von DOCX in RTF

Wenn Sie bereits in jungen Jahren Microsoft Windows verwendet haben, müssen Sie das RTF-Dateiformat (Rich Text Format) kennen. Microsoft hat mit Windows 3.0 das [RTF-Dateiformat](/word-processing/rtf/) eingeführt. RTF-Dateien stellen eine Methode zum Kodieren formatierter Texte und Grafiken zur Verwendung in Anwendungen dar. Eine RTF-Datei kann neben Textformatierungen wie Fett, Kursiv und Unterstrichen auch Text und Bilder enthalten. Am interessantesten ist, dass RTF als interoperables Dateiformat übernommen wurde und plattformübergreifend ist. Aus diesem Grund ist die Konvertierung von DOCX in RTF eine beliebte Wahl bei der gemeinsamen Nutzung von Dokumenten über verschiedene Betriebssysteme hinweg zu Bearbeitungszwecken.
