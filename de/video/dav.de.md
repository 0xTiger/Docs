## Was ist eine DAV-Datei?

Eine DAV-Datei ist eine komprimierte und verschlüsselte Videodatei, die von DVR365 oder dem digitalen Überwachungssystem von Dahua Technology erstellt wurde. Diese Systeme sind mit einem CCTV-Kameraüberwachungssystem verbunden. Die von diesen Kameras empfangenen Videoaufnahmen werden als DAV-Datei im System gespeichert. Da DAV-Dateien verschlüsselt sind, können sie nur von dem CCTV-System, das sie erstellt hat, oder einigen spezialisierten Mediaplayern geöffnet werden. DAV-Dateien können auch in das Dateiformat [AVI](/de/video/avi/) konvertiert werden.

## DAV-Dateiformat - Weitere Informationen

DAV-Dateien werden im verschlüsselten Binärdateiformat auf der Disc gespeichert. Die Verschlüsselung verhindert die unbefugte Nutzung dieser Dateien. Daher sind diese Eigentum der CCTV-Systeme von [Dahua Technology] (https://www.dahuasecurity.com/). Einige andere Kameras von Dahua Technologies können das Überwachungsvideo auch als DAV-Dateien speichern.

Der Dahua Smart Player kann zum Öffnen und Abspielen von DAV-Dateien verwendet werden.
