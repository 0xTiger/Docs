## Was ist eine PZ-Datei?

Eine PZ-Datei ist eine Projektdatei, die mit der Anwendung **Panzoid** zum Erstellen von Online-Videoclips erstellt wurde. Es enthält Informationen über das Videoprojekt und seine Inhalte wie Szene, Objekte, Effekte, Beleuchtung, FX, Audio und Musik. Mit Panzoid können Sie mit der Videobearbeitungsfunktion **ClipMaker** online Videos erstellen.

PZ-Dateien können mit Panzoid ClipMaker und Panzoid Backgrounder geöffnet werden.

## PZ-Dateiformat

PZ-Dateien werden als Binärdateien auf der Disc gespeichert. Wenn ein Benutzer ein neues Projekt mit [Panzoid](https://panzoid.com/) erstellt, wird eine PZ-Datei erstellt, die die Details benutzerdefinierter Objekte speichert. Dieselben Informationen werden mit anderen Benutzern synchronisiert, wenn das Projekt andere Mitwirkende hat. PZ-Dateien können dann auf den Computer heruntergeladen werden.

Wenn Sie eine PZ-Datei öffnen möchten, können Sie sie erneut in **Panzoid** laden, indem Sie die Option Laden von der Online-Schnittstelle verwenden. Die Datei wird online geladen und ihr Inhalt wird für Änderungen geladen.

### Unterstützte Videoclips nach PZ-Datei

Panzoid verfügt über die Programme ClipMaker und Backgrounder, die zusammen zum Erstellen von Videos verwendet werden. Mit Panzoid können verschiedene Arten von Videos erstellt werden, darunter benutzerdefinierte Hintergründe, YouTube-Kanalbilder, Social-Media-Videos und Miniaturansichten.
