## Was ist eine STR-Datei?

Eine STR-Datei ist eine von YouTube erstellte Aufzeichnungsdatei für einen Livestream, der vom Streaming-Event des Benutzers gehostet wurde. Die Datei wird automatisch von YouTube im Backend gespeichert, sobald das Live-Streaming beendet ist. Ein Live-Streaming-Video kann auf mehrere Kanäle übertragen werden, sodass das Publikum über Kommentare mit dem Herausgeber interagieren kann. Auf diese SRT-Dateien kann über die YouTube-Livestream-API zugegriffen werden, die den Zugriff auf die YouTube-Server ermöglicht.

## STR-Dateiformat

YouTube speichert STR-Dateien im Binärdateiformat auf dem Server. Die internen Dateistrukturdetails von STR-Dateien sind jedoch nicht öffentlich verfügbar.
