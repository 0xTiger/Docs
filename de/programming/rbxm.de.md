## Was ist eine RBXM-Datei?

Eine RBXM-Datei ist ein 3D-Modelldateiformat, das in Roblox Studio IDE zum Speichern von 3D-Objekten des Online-Multiplayer-Roblox-Spiels verwendet wird. Es enthält 3D-Objekte, die mit der IDE für das Spiel erstellt wurden. Beispiele für 3D-Objekte in einer RBXM-Datei sind Hemden, Hosen, Hüte, Gebäude, Fahrzeuge, Tiere, Flugzeuge, Autos oder Pflanzen. RBMX-Dateien können mit der Roblox-IDE in das [OBJ-3D-Dateiformat](/de/3d/obj/) konvertiert werden. Roblox IDE kann zum Öffnen von RBMX-Dateien verwendet werden.

Ein weiteres beliebtes Roblox-Dateiformat ist [RBXL](/de/programming/rbxl/), das ein Standortdateiformat ist und einen Standort enthält, der im 3D-Wort innerhalb des Roblox-Spiels definiert ist.

## RBMX-Dateiformat – Weitere Informationen

RBXM-Dateien werden im Binärdateiformat erstellt und gespeichert, und ihre internen Dateiformatspezifikationen sind nicht öffentlich verfügbar. Dies unterscheidet sich vom RBXMX-Dateiformat, das auf dem Dateiformat [XML](/de/web/xml/) basiert. Auf Github sind zahlreiche Tools/Repositories verfügbar, die zum [Verarbeiten von Roblox RBXM](https://github.com/search?q=rbxm)-Dateien verwendet werden können.
