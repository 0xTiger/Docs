Programmierdateien sind Quellcodedateien, die von Programmiersprachen zum Schreiben von Softwareanwendungen zum Ausführen bestimmter Aufgaben verwendet werden. Jede Programmiersprache hat ihre eigene Syntax und trägt ihren eigenen Satz von Dateien zum Schreiben von Code in Form von Bibliotheken, Header-Dateien, Klassen oder Konstanten. Je nach verwendeter Programmiersprache unterscheidet sich das Dateiformat des Quellcodes, um die von dieser Sprache definierte Syntaxregel einzuhalten.

Gängige Dateierweiterungen für Entwickler oder Programmierer und die zugehörigen Dateiformate sind **[C#](/de/programming/cs/)** (CSharp Source Code File), **[CPP](/de/programming/cpp/)** (C++ Source File) und **[VB](/de/programming/vb/)** (Visual Basic Source Code File).
