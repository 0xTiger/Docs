## Was ist eine CXX-Datei?

Eine CXX-Datei ist eine Quellcodedatei, ähnlich wie C++, zum Schreiben von Anwendungen. Es kann eine vollständige Implementierung einer gesamten Anwendung oder eine teilweise Implementierung des Programms enthalten, das in ein Projekt mit anderen C++-Dateien eingeschlossen werden kann. CXX-Dateien können in jedem Texteditor und gängigen Programmier-IDEs wie Notepad++, Turbo C++ und Visual Studio geöffnet werden.

## CXX-Dateiformat

CXX-Dateien werden als reine Textdatei gespeichert und enthalten Code, der in C++-Syntax geschrieben ist. CXX-Dateien werden häufig auch mit der Erweiterung [.cpp](/de/programming/cpp/) gefunden. CXX-Dateien werden als Alternative zu .cpp angesehen, die von einigen Compilern verwendet werden.
