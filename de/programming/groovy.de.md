## Was ist eine GROOVY-Datei?

Eine GROOVY-Datei ist eine Quellcodedatei, die in der Programmiersprache Groovy geschrieben ist. Der in einer GROOVY-Datei geschriebene Code ähnelt der objektorientierten Sprache [Java](/de/programming/java/), was das Design und die Entwicklung von Anwendungen erleichtert. Eine GROOVY-Datei wird in JVM-Bytecode (Java Virtual Machine) kompiliert und ist mit anderem Java-Code und Bibliotheken kompatibel.

## GROOVY-Dateiformat - Weitere Informationen

GROOVY-Dateien enthalten Quellcode, der in Groovy-Syntax geschrieben ist. Es kann Code enthalten, der sowohl als Programmiersprache als auch als Skriptsprache für die Java-Plattform verwendet werden kann. Die Groovy-Sprache hat ähnliche Funktionen wie Python, Ruby und Smalltalk.
