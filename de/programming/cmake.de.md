## Was ist eine CMAKE-Datei?

Eine CMAKE-Datei ist eine Projektdatei, die mit dem plattformübergreifenden Open-Source-Tool **CMake** erstellt wurde. Es enthält Quellskript in der CMake-Sprache für das gesamte Programm. Das CMake-Tool verwendet Programmierskripts, die als CMakeLists bekannt sind, um Build-Dateien für eine bestimmte Umgebung zu generieren, z. B. Makefiles-Unix-Maschinen. Vom CMake-Projekt verwendete Skripts und Module sind als script.cmake und module.cmake organisiert, auf die im Projekt verwiesen wird.
