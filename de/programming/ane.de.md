## Was ist eine ANE-Datei?

Eine ANE-Datei ist eine Adobe Air Native Extension-Datei, die von Entwicklern verwendet wird, um bestimmte Funktionen zu schreiben, die [Adobe ActionScript](https://en.wikipedia.org/wiki/ActionScript)-Code aufrufen. In den meisten Fällen enthalten die ANE-Dateien die native Codebibliothek, die von der Erweiterung verwendet wird. Die Datei selbst wird jedoch auf dem Gerät installiert. Es ermöglicht systemspezifische Funktionen wie Gerätevibration. Adobe AIR-Anwendungen sollten kompiliert werden, um dynamische Verknüpfungen zur Verwendung der ANE-Datei zu verwenden.

## ANE-Dateiformat – Weitere Informationen

ANE-Dateien werden als Binärdateien auf Disc gespeichert, deren Dateiformatspezifikationen nicht öffentlich verfügbar sind. Eine ANE-Datei kann nativen Android-Java-Quellcode enthalten, um auf Android-Gerätefunktionen wie das Aufrufen von Lautstärke- oder Vibrationsfunktionen zuzugreifen. All dies wird durch Umhüllen des nativen Systems in die ActionScript-Sprache von Adobe erreicht.
