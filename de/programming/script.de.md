## Was ist eine SCRIPT-Datei?

Eine SCRIPT-Datei ist eine Quelldatei, die Skript in einer beliebigen Skriptsprache enthält. Dies macht es zu einer verallgemeinerten Skriptdatei, die Variablen und Funktionen enthält. Da jede Skriptsprache unterschiedlich sein kann, muss das in einer SCRIPT-Datei geschriebene Skript mit einer geeigneten Skript-Engine ausgeführt werden.

## SCRIPT-Dateiformat - Weitere Informationen

SCRIPT-Dateien werden in reinen Textdateien auf der Disc gespeichert. Diese können mit jedem Texteditor wie Microsoft Notepad, Notepad++ und Apple TextEdit geöffnet und bearbeitet werden. Da die Skriptsprache unterschiedlich sein kann, kann sie ausgeführt werden, indem die Dateierweiterung in die Standarderweiterung der jeweiligen Skriptsprache umbenannt wird. Beispielsweise gibt es mehrere gängige Skriptsprachen wie JavaScript ([.JS](/de/web/js/)), VBScript ([.VBS](/de/programming/vbs/)), AppleScript (.SCPT), PHP ([ .PHP](/de/programming/php/)) und ASP ([.ASP](/de/web/asp/)).
