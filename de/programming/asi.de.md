## Was ist eine ASI-Datei? ##

Eine ASI-Datei ist ein variables Dateiformat, das von der Alpha Five-Software verwendet wird, die zum Entwickeln von Webanwendungen verwendet wird. Es speichert Variablen, die von der Webanwendung verwendet werden. Wenn Sie ein neues Entwicklerprojekt in Alpha Five erstellen, wird die ASI-Datei ebenfalls erstellt und zusammen mit dem Alpha Five-Entwicklerprojekt gespeichert.

ASI-Dateien können mit Alpha Software Alpha Anywhere geöffnet werden.

## ASI-Dateiformat

ASI-Dateien werden im proprietären Alpha Five-Dateiformat gespeichert. Sie werden von der Anwendung als Teil des Entwicklerprojekts gelesen und geschrieben.
