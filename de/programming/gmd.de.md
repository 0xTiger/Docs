## Was ist eine GMD-Datei?

Eine GMD-Datei ist eine Quelldatei für die Spieleentwicklung, die mit dem Game Maker-Softwareentwicklungstool erstellt wurde und zum Erstellen von 2D-Spielen verwendet wird. Die GMD-Datei wird als Textdatei gespeichert und ist eine Projektdatei für das zu entwickelnde Spiel. Game Maker ist für Windows, Mac, Linux, Android, iOS, HTML5, Xbox, PlayStation und Nintendo Switch verfügbar. Das Tool wird weltweit von Indie-Entwicklern, professionellen Studios und Pädagogen häufig zum Erstellen von Spielen verwendet.

## GMD-Dateiformat

GMD-Dateien werden im Binärdateiformat gespeichert und die Details zum internen Dateiformat stehen dem Entwickler nicht als Referenz zur Verfügung.
