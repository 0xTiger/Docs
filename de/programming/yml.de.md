## Was ist eine YML-Datei?

YML ist eine Textdatei in der Datenserialisierungssprache [YAML](/de/programming/yaml/) (Yet Another Markup Language), die häufig zum Erstellen von Konfigurationsdateien verwendet wird. Es kann in Kombination mit jeder Programmiersprache arbeiten, einschließlich C/C++, Ruby, Python, Java, PHP, C# und anderen. Da es unabhängig von einer bestimmten Programmiersprache ist, ist es offen, interoperabel und für Menschen lesbar. YML-Dateien können mit gängigen Texteditoren wie Microsoft Notepad, Notepad++, Github Atom, Apple TextEdit und Visual Studio IDE geöffnet und bearbeitet werden.

## YML-Dateiformat – Weitere Informationen

YML-Dateien speichern Anwendungseinstellungen als Konfiguration. Seine Grundstruktur ist eine Map, die auch als Verzeichnis oder Hash bezeichnet werden kann. Einträge in einer YML-Datei bestehen aus einem Schlüssel-Wert-Paar. Meistens werden die .yml-Konfigurationsdateien als „configure.yml“ gespeichert, obwohl einige Programmiersprachen wie Ruby on Rails die Konfiguration der Datenbankkonnektivität in „database.yml“-Dateien speichern. [YML/YAML-Dateiformatspezifikationen](https://yaml.org/spec/1.2.2/) erläutern die vollständigen Sprachstrukturen und syntaxbezogenen Informationen.
