## Was ist eine F95-Datei?

Eine Datei mit der Erweiterung .f95 weist normalerweise darauf hin, dass es sich um eine Fortran 95-Quellcodedatei handelt. Fortran ist eine universelle, imperative Programmiersprache, die sich besonders gut für numerisches und wissenschaftliches Rechnen eignet. Fortran 95 ist eine erweiterte Version von Fortran 90, die mehrere neue Funktionen und Verbesserungen einführt.

Fortran-Quellcodedateien enthalten normalerweise Programmanweisungen, die in der Programmiersprache Fortran geschrieben sind. Wenn Sie eine Datei mit der Erweiterung .f95 haben, ist diese wahrscheinlich dazu gedacht, mit einem Fortran-Compiler kompiliert zu werden. Kompilierung ist der Prozess der Übersetzung von menschenlesbarem Quellcode in maschinenlesbaren Binärcode, der von einem Computer ausgeführt werden kann.

## Wie öffne ich eine F95-Datei?

Wenn Sie mit der Datei .f95 arbeiten oder ein Programm darin ausführen möchten, benötigen Sie einen Fortran-Compiler. Zu den beliebten Fortran-Compilern gehören GNU Fortran (gfortran), Intel Fortran Compiler (ifort) und andere. Sie können Fortran-Programme mit den entsprechenden vom Compiler bereitgestellten Befehlen kompilieren und ausführen.

Wenn Sie beispielsweise gfortran installiert haben, können Sie die folgenden Befehle im Terminal verwenden, um ein Fortran-Programm zu kompilieren und auszuführen:

```
# Compile the Fortran program
gfortran -o my_program my_program.f95

# Run the compiled program
./my_program
```

Wenn Sie nur den Inhalt der F95-Datei anzeigen möchten, handelt es sich um eine reine Textdatei, sodass Sie sie mit jedem Texteditor öffnen können.
