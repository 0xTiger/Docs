## Was ist eine PHP-Datei?
Eine Datei mit der Erweiterung .php bezieht sich auf eine Open-Source-Programmiersprache, die zum Schreiben serverseitiger Skripts verwendet wird, die auf einem Webserver ausgeführt werden. Es ist die am weitesten verbreitete Web-Skriptsprache, die typischerweise für die Entwicklung umfangreicher Webanwendungen verwendet wird. Das größte Blogging-System im Internet, dh WordPress, und das größte soziale Netzwerk Facebook basieren auf der PHP-Sprache. Es ist mit verschiedenen Plattformen wie Windows, Linux, Unix, Mac OS X usw. kompatibel. Außerdem läuft es erfolgreich auf fast allen heute verwendeten Servern, dh Apache, IIS usw. PHP unterstützt eine breite Palette von Datenbanken, einschließlich MySQL.

## PHP-Dateiformat ##

Ein einzelnes PHP kann Text, CSS, [HTML](/de/web/html/), JavaScript und PHP-Code enthalten. PHP-Code wird auf dem Server ausgeführt und das Ergebnis als reines HTML an den Browser zurückgegeben. PHP-Dateien können mit jedem Texteditor geöffnet und an Ort und Stelle bearbeitet werden, obwohl Anwendungen wie Adobe Dreamweaver und Eclipse-PHP-Entwicklungstools eine bequeme Möglichkeit bieten, PHP-Code zu schreiben und zu ändern.

### Syntax ###

Ein PHP-Skript wird auf dem Server ausgeführt und das einfache HTML-Ergebnis wird an den Browser zurückgesendet. Ein PHP-Skript beginnt mit<?php and ends with ?>

### PHP-Beispiel ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
