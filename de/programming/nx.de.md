## Was ist eine NX-Datei?

Eine NX-Datei ist eine Videospieldatei, die für die Verwendung mit der virtuellen Spielekonsole **LowRes NX** erstellt wurde. Es ist in der Programmiersprache BASIC geschrieben und enthält den Quellcode des Spiels, der definiert, wie das Spiel funktioniert. Sie können Ihre eigenen NX-Dateien zur Verwendung mit der App erstellen, die geladen werden kann, und Benutzer können das Spiel spielen.

Sie können eine NX-Datei in einem Texteditor, LowRes NX oder einem beliebigen Quellcode-Editor wie Microsoft Visual Studio Code öffnen.

## NX-Dateiformat

Eine NX-Datei wird als reine Textdatei auf der Disc gespeichert, die die Spielanweisungen enthält. Jede NX-Datei enthält den Quellcode des Spiels, der aus mehreren ROM-Einträgen der Kassette besteht. Diese ROM-Einträge enthalten Informationen zu Spielkomponenten wie Grafiken, Levelkarten, Audio und andere Nicht-Quellcode-Daten für das Spiel. Die ROM-Einträge eines Programms können mit NX-Bearbeitungsprogrammen, die im Lieferumfang von LowRes NX enthalten sind, hinzugefügt oder bearbeitet werden.

## Wie erstelle ich NX-Dateien?

### NX-Dateien auf dem PC erstellen

Sie können NX-Dateien auf dem PC erstellen mit:

* Texteditoren – wie Microsoft Notepad, Notepad++ oder Github Atom
* Quellcode-Editoren – wie Microsoft VIsual Studio Code

### Erstellen Sie NX-Dateien auf iOS

LowRes NX verfügt über eine iOS-Version, mit der Entwickler NX-Dateien aus dieser App erstellen können.
