## Was ist eine ASVF-Datei?

Eine ASVF-Datei ist eine komprimierte Datei, die von der Asphyre Sphinx-Software erstellt wurde, einem 2D/3D-Framework für Industrie- und Unternehmensanwendungen. Es speichert Daten wie Schriftarten, Bilder und andere Medien. Sphinx kann auch zur Entwicklung von 2D/3D-Videospielen verwendet werden. Eine komprimierte ASVF-Datei kann allgemeine Dateien wie .JPEG- und .XML-Dateien enthalten.

ASVF-Dateien können mit der seit langem eingestellten Asphyre Sphinx-Software geöffnet werden.

## ASVF-Dateiformat

ASVF-Dateien werden als komprimierte Dateien gespeichert und können mehrere Dateien im Archiv enthalten.
