## Was ist eine AS3PROJ-Datei?

Eine AS3PROJ-Datei ist eine Projektdatei, die von FlashDevelop erstellt wurde, einer kostenlosen Open-Source-Programmier-IDE zum Entwickeln von Flash- und Webanwendungen. Es speichert alle projektbezogenen Informationen und Einstellungen, die vom Benutzer mithilfe der FlashDevelop IDE angewendet wurden. Zu den in einer AS3PROJ-Datei gespeicherten Informationen gehören Build-Einstellungen, Verweise auf Projektquelldateien und Optionen zum Generieren der Ausgabe [.swf-Datei](/de/page-description-language/swf/). Wenn ein neues Projekt erstellt wird, wird es als AS3PROJ-Datei an demselben Ort gespeichert, der vom Benutzer zum Speichern der projektbezogenen Daten angegeben wurde.

## AS3PROJ-Dateiformat

Eine AS3PROJ-Datei wird im Dateiformat [XML](/de/web/xml/) gespeichert, das ein universelles Dateiformat für den Informationsaustausch ist. Es kann in jedem Texteditor wie Notepad, Notepad++ und Atom geöffnet werden, um den Inhalt zu untersuchen.
