## Was ist eine AS2PROJ-Datei?

Eine AS2PROJ-Datei ist eine Projektdatei, die von der Open-Source-Flash-Programmier-IDE FlashDevelop erstellt wurde. Sie enthält projektbezogene Informationen wie Build-Einstellungen, Verweise auf Projektquelldateien und Optionen zum Generieren der Ausgabe [.swf-Datei](/de/page-description-language/swf/). Jedes neue Projekt, das in FlashDevelop erstellt wird, wird als AS2PROJ-Datei gespeichert. Sie können die Einstellungen für Ihr Projekt ändern und alle diese Änderungen werden in der AS2PROJ-Datei gespeichert.

AS2PROJ-Dateien können mit FlashDevelop geöffnet werden.

## AS2PROJ-Dateiformat

Eine AS2PROJ-Datei wird im Dateiformat [XML](/de/web/xml/) gespeichert, das ein universelles Dateiformat für den Informationsaustausch ist. Es kann in jedem Texteditor wie Notepad, Notepad++ und Atom geöffnet werden, um den Inhalt zu untersuchen.
