## Was ist eine APA-Datei?

Eine Datei mit der Erweiterung .apa ist eine Softwareentwicklungsprojektdatei, die von der RSView-Anwendung erstellt wurde. Es enthält Informationen, die zum Erstellen von Human Machine Interface (HMI)-Anwendungen erforderlich sind. APA-Dateien werden im Archivdateiformat gespeichert. Das Archivdateiformat hilft dabei, diese aufgrund der geringeren Dateigröße problemlos von einem Computer auf einen anderen zu übertragen. Zum Extrahieren des Inhalts von APA-Dateien können standardmäßige Dekomprimierungsprogramme verwendet werden. APA-Dateien können mit der Software RSView Studio oder RSView ME von Rockwell Automation geöffnet werden.

## APA-Dateiformat – Weitere Informationen

Der RSView-Anwendungsmanager speichert die .apa-Dateien unter Windows XP standardmäßig unter C:\Dokumente und Einstellungen\All Users\Dokumente\RSView Enterprise\ME\Archives. Sie können die Datei jedoch je nach Bedarf in einem beliebigen Verzeichnis speichern.
