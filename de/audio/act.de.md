## Was ist eine ACT-Datei? ##

Ein ACT- oder Adobe-Farbtabellenformat (auch bekannt als Color Lookup Table-Datei) ist eine Art von Adobe Photoshop-Datenbank, die einen Satz vordefinierter Farben speichert. Sie können Farben hinzufügen oder entfernen, wenn Sie ein Bild für die Online-Veröffentlichung speichern, um eine höhere Bildqualität oder eine kleinere Dateigröße zu erzielen. Möglicherweise haben Sie eher ein ADPCM-verschlüsseltes Audioformat, wenn es nicht mit Illustrator verwendet wird. Adaptive Differential Pulse Code Modulation ist eine Audiokomprimierungstechnik, die von verschiedenen Musikgeräten und Hörgeräten verwendet wird.

Das ACT-Dateiformat wird auch vom Alma CAD/CAM-Dateisystem verwendet. Dies sind die Richtlinien, die von 3D-Werkzeugmaschinen verwendet werden, um herauszufinden, wie man etwas schneidet. Beispiele für ACT sind ein Genesis3D-Player, eine DS Games Developer Action oder eine FoxPro Detailing Wizard Action Schematic-Datei.


## Spezifikation des ACT-Dateiformats##

Viele Rekorder, die das ACT-System verwenden, tun dies nur für die kleinste Aufnahmeoption; Wenn die Qualität erhöht wird, wird ein separates Format, normalerweise WAV, verwendet, was zu massiv größeren Dokumenten führt.

Es gibt mehrere Generationen von ACT; seit Juni 2009 können Titel, die von nachfolgenden Geräten erstellt wurden, einfach nicht mehr von jedem kostenlosen Standard-Musikplayer oder jeder Konvertierungsanwendung erkannt werden, sodass nur die gebündelten MP3-Tools übrig bleiben.

Die Bildoptimierung für die Webverteilung erfolgt mit ACT-Dateien. Kunden können die Bildqualität anhand der generierten Bilddatenbanken testen, indem sie die Anzahl der Farben in der CLUT erhöhen oder verringern.

Durch Tippen auf die Option „Farbtabelle“ und Auswählen von „Farbdaten laden“ im Dialogfeld „Für Web exportieren“ können Adobe-Farbtabellen hochgeladen werden. Das Verzeichnis */Presets/en_US/Save* for Web Settings/Color Tables/ enthält mehrere vorkonfigurierte ACT-Dateien, die mit dem Photoshop-System geliefert wurden.

Act-Skripte können ausschließlich in Genesis 3D oder durch Transformation anderer 3D-Modelle in den ACT-Standard mit Genesis3D Actor Designer generiert werden
