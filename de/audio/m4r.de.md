## Was ist eine M4R-Datei?

Das .m4r-Dateiformat ist eine Art Audiodateiformat, das für iPhone-Klingeltöne verwendet wird. Das Format ist im Wesentlichen eine umbenannte Version des Multimedia-Containerformats MPEG-4 Part 14 (MP4), wobei die Dateierweiterung .m4r darauf hinweist, dass es sich bei der Datei um einen Klingelton handelt. M4R-Dateien sind im Advanced Audio Coding (AAC)-Format kodiert und können mit verschiedenen Softwareanwendungen erstellt werden, die die Erstellung von Klingeltönen unterstützen. Sobald sie erstellt sind, können sie mit iTunes oder anderen Methoden auf ein iPhone übertragen und dann als Klingelton des Geräts festgelegt werden. Obwohl M4R-Dateien für iPhone-Klingeltöne konzipiert sind, ist das Format nicht nur für Apple-Geräte verfügbar und kann auf anderen Geräten abgespielt werden, die AAC-Audiodateien unterstützen.

## M4R-Dateiformat – Weitere Informationen

Das .m4r-Format wurde 2007 von Apple mit der Veröffentlichung des iPhone eingeführt. Zuvor lagen iPhone-Klingeltöne im .m4a-Format vor, das ebenfalls ein AAC-Audioformat ist, aber nicht speziell für Klingeltöne entwickelt wurde. M4R-Dateien haben eine maximale Länge von 40 Sekunden, was der maximal zulässigen Länge für einen Klingelton auf einem iPhone entspricht. M4R-Dateien sind das Format, das das iPhone für Klingeltöne verwendet.

M4R-Dateien können sowohl Audio- als auch Videoinhalte enthalten, obwohl der Videoinhalt normalerweise vom iPhone ignoriert wird, wenn die Datei als Klingelton verwendet wird. M4R-Dateien werden durch Umbenennen einer vorhandenen .m4a-Datei in .m4r erstellt. Der Inhalt der Datei ändert sich nicht, aber die Erweiterung .m4r teilt iTunes mit, dass die Datei als Klingelton behandelt werden soll.

M4R-Dateien können mit verschiedenen Softwareanwendungen erstellt werden, einschließlich iTunes, das über einen integrierten Klingelton-Ersteller verfügt. Auch Softwareanwendungen von Drittanbietern wie GarageBand und Ringtone Maker können zum Erstellen von M4R-Dateien verwendet werden. M4R-Dateien können mit iTunes auf ein iPhone übertragen werden oder indem die Datei über Cloud-Speicherdienste wie iCloud oder Dropbox mit dem iPhone synchronisiert wird.
