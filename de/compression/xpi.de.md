## Was ist eine XPI-Datei?

Eine XPI-Datei ist ein Installationsarchiv, das komprimiert wird, um die Größe der Datei zu reduzieren. Es wird von Mozilla-Anwendungen für die Installation von Plugins und Add-on-Dateien verwendet. Es enthält ein Installationsskript oder ein Manifest im Stammverzeichnis der Datei zusammen mit einer Reihe von Datendateien. Eine XPI-Datei kann Designs, Toolkits oder Firefox-Plugins enthalten, die Benutzer installieren können, um Teil des Firefox-Browsers, Thunderbird oder SeaMonkey zu werden.

## XPI-Dateiformat – Weitere Informationen

XPI-Dateien werden als [ZIP](/de/compression/zip/)-Archive auf der Disc gespeichert, die mehrere Dateien in einer einzigen komprimierten Datei kombinieren. Die Dateien in einer XPI-Datei können Installationsskriptdateien wie JS und Webdateien wie [CSS](/de/web/css/), [HTML](/de/web/html/) und [JSON](/de/web/json/) enthalten. ). Manchmal kann es auch [PNG](/de/image/png/) Bilddateien enthalten, die vom Add-On als Symbole verwendet werden.

### Wie kann ich den Inhalt der XPI-Datei anzeigen?

Eine XPI-Datei ist praktisch ein ZIP-Archiv, dessen Inhalt mit den folgenden Schritten angezeigt werden kann.

* Ändern Sie die Dateierweiterung von XPI in ZIP
* Extrahieren Sie den Inhalt der Datei mit einem beliebigen standardmäßigen Dekomprimierungsprogramm wie WinZip (Windows, Mac), 7-Zip (Windows) oder Apple Archive Utility (Mac).

### Installieren Sie die XPI-Datei auf Firefox Android

Die meisten Leute sind neugierig, ob XPI-Dateien in Firefox auf Android-Geräten installiert werden können. Unter Android können Sie das Add-On aus einer XPI-Datei installieren, indem Sie die Datei suchen und mit Firefox öffnen.
