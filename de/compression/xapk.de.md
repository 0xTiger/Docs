## Was ist eine XAPK-Datei?

Eine Datei mit der Erweiterung .xapk ist eine komprimierte Paketdatei, die zum Installieren von Android-Apps auf Mobilgeräten verwendet wird. Es ist ein Containerdateiformat, das APK und zusätzliche zugehörige Dateien enthält, die für die Installation erforderlich sind. Die andere zugehörige Datei ist eine OBB-Datei, die zusätzliche Dateien wie Grafiken, Mediendateien und App-Daten speichert, die erforderlich sind, um die Anwendung am Laufen zu halten. XAPK-Dateien werden von Google Play nicht unterstützt und nur für die Verteilung auf Download-Websites für Android-Apps von Drittanbietern verwendet. Diese können mit dem XAPK Installer auf einem Android-Gerät installiert werden.

## XAPK-Dateiformat

XAPK-Dateien werden mit dem Standarddateiformat [ZIP](/de/compression/zip/) komprimiert. Diese können mit einer Standard-Komprimierungs-/Dekomprimierungssoftware wie WinZIP extrahiert werden. Sobald die XAPK-Datei auf die Disc extrahiert wurde, enthält sie die folgenden Dateien im Ordner.

* APK - Standardinstallationsdatei für die Installation der Anwendung auf Android-Geräten
* OBB - Zusätzliche Datei, die relevante Ressourcendateien enthält
