## Was ist eine Linux PUP-Datei?

Eine Linux-PUP-Datei ist ein Installationspaket, das vom Betriebssystem Puppy Linux verwendet wird. Es wird als Installationsprogramm ausgeführt und beginnt mit der Installation der Paketdateien. Unter Linux wird sie oft als DotPup-Datei bezeichnet und ähnelt den Installationsdateien [.exe](/de/executable/exe/) und [.msi](/de/executable/msi/), die unter Windows verwendet werden. Die neueren Versionen von Puppy Linux verwenden das .pet-Dateiformat anstelle des PUP-Dateiformats.
