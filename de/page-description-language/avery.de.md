## Was ist eine AVERY-Datei?

Eine AVERY-Datei ist ein druckbares Dokument, das ein oder mehrere Etiketten zum Drucken enthält. Es wird mit der Software Avery Design & Print oder Design & Print Online erstellt. Die Etiketten in einer AVERY-Datei können verschiedene Elemente wie Text, Grafiken und Informationen zum Seitenlayout umfassen. Diese Informationen werden von Druckereien zum Drucken einer Vielzahl von Designelementen wie Visitenkarten, Namensschildern, Ordnereinlagen, Adressetiketten, Datenträgeretiketten, Nachrufvorlagen und Grußkarten verwendet.

## AVERY-Dateiformat - Weitere Informationen

AVERY-Dateien werden im binären Dateiformat gespeichert. Diese können mit der Software Avery Design & Print oder Design & Print Online geöffnet werden. Die mit Design & Print erstellten Designs können verwendet werden, um AVERY-Etiketten zu erstellen, die einer Reihe von Zwecken dienen können. Diese Entwürfe können als [PDF](/de/pdf/)-Dateien gespeichert werden, die zu Druckzwecken mit Prints Company geteilt werden können.

### Drucken einer AVERY-Datei

Eine AVERY-Datei kann entweder als PDF gespeichert oder direkt auf jedem Standarddrucker ausgedruckt werden. Um die Datei zu drucken, öffnen Sie die Datei in der Design & Print-Software und verwenden Sie die Option „Drucken“ im Menü „Datei“, um die Datei zum Drucken an den Drucker zu senden.
