## Was ist eine MPT-Datei? ##

Eine Datei mit der Erweiterung .mpt ist eine Vorlagendatei von [Microsoft Project](https://products.office.com/en-us/project/project-and-portfolio-management-software). Diese enthält grundlegende Informationen sowie Struktur- und Dokumenteinstellungen zum Erstellen von [MPP](/de/project-management/mpp/)-Dateien. Diese Vorlagen bieten Standardeinstellungen, die für ein bestimmtes Projekt unerlässlich sind, z. B. Zeitpläne oder Budgetinformationen. Es kann jedoch keine projektspezifischen Daten wie Aufgaben, Ressourcen oder Zuordnungen speichern. Nach der Änderung können die Vorlagendateien als Standard-MPP-Dateien gespeichert werden, um weiter damit zu arbeiten. Eine Master-Vorlagendatei mit dem Titel „Global.mpt“ ist in Microsoft Project verfügbar und kann geöffnet werden, um eine individuelle Vorlage zu erstellen. MPT-Dateien verwenden *application/vnd.ms-project* und *application/x-project* als Standard-MIME-Typen. Windows, Linux und macOS können alle MPT-Dateien öffnen, zusammen mit Moos Project Viewer, Microsoft Project und MPXJ.

## Probleme beim Öffnen einer MPT-Datei? ##

Hier ist die Liste einiger häufiger Probleme, die auftreten und zu Fehlfunktionen des MPT-Formats führen können:

* Keine unterstützende Software
* Beschädigte Datei
* Infizierte Datei aufgrund eines Virus
* Kein Zugriffsrecht im System zum Öffnen der Dateien
* Veraltetes Laufwerk in Ihrem System
 *   Extension of the file is renamed
