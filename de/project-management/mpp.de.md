## Was ist eine MPP-Datei?

Eine Datei mit der Erweiterung .mpp ist eine Microsoft Project-Datendatei, in der Informationen zum Projektmanagement integriert gespeichert werden. Microsoft hat dieses proprietäre Dateiformat entwickelt, um es mit Microsoft Project (MSP), ihrem Projektmanagement-Softwareprodukt, kompatibel zu machen. Anders als MPP kann MSP mit Projekt-XML-Schema arbeiten. Viele Anwendungen und APIs ermöglichen die Konvertierung von MPPs in andere Dateiformate. Microsoft verfügt jetzt über einen Online-Projektserver, auf dem Projektmanagementdateien für die Zusammenarbeit mehrerer Benutzer hochgeladen werden können.

## MPP-Dateiformat ##

Microsoft hat MPP als binäres Dateiformat entwickelt und veröffentlicht, und seine Spezifikationen wurden von Microsoft bis heute nicht veröffentlicht. Die interne Dateistruktur und Details sind daher für Entwickler nicht verfügbar. Es sind mehrere APIs online verfügbar, die das Lesen/Schreiben in MPP-Dateien unterstützen, aber eine Genauigkeit von 100 % wird nie erreicht. Microsoft hat im Laufe der Zeit mehrere Funktionen in seine Projektmanagementanwendung eingeführt, ebenso wie die Funktionen des MPP-Dateiformats.

Die MPP-Datei verwendet die folgenden MIME-Typen:

* application/vnd.ms-Projekt
* Anwendung/msproj
* Anwendung/msproject
* Anwendung/x-msproject
* Anwendung/x-ms-Projekt
* application/x-dos_ms_project
* Anwendung/mpp
* zz-application/zz-winassoc-mpp

## Probleme beim Öffnen einer MPP-Datei? ##

Hier ist die Liste einiger häufiger Probleme, die auftreten und Fehlfunktionen des MPP-Formats verursachen können:

* Keine unterstützende Software
* Beschädigte Datei
* Infizierte Datei aufgrund eines Virus
* Kein Zugriffsrecht im System zum Öffnen der Dateien
* Veraltetes Laufwerk in Ihrem System
* Erweiterung der Datei wird umbenannt
