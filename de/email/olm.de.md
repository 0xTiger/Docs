## Was ist eine OLM-Datei?

Eine Datei mit der Erweiterung .olm ist eine Microsoft Outlook-Datei für das Mac-Betriebssystem. Eine OLM-Datei speichert E-Mail-Nachrichten, Journale, Kalenderdaten und andere Arten von Anwendungsdaten. Diese ähneln [PST](/de/email/pst/)-Dateien, die von Outlook auf dem Windows-Betriebssystem verwendet werden. OLM-Dateien, die von Outlook für Mac erstellt wurden, können jedoch nicht in Outlook für Windows geöffnet werden. Das OLM-Dateiformat wurde mit Outlook 2011 eingeführt, um lokale Daten von Outlook zu speichern. Eine einzelne OLM-Datei kann mehrere Postfächer enthalten und die Daten in Form von Ordnern und Unterordnern organisieren. OLM-Dateien können in Microsoft Outlook 365 und Outlook für Mac geöffnet/geladen werden.

## OLM-Dateiformat

OLM sind Binärdateien mit dem MIME-Typ Application/octet-stream. Die technischen Spezifikationen des OLM-Dateiformats sind nicht offen von Microsoft erhältlich, und es sind keine Formatdetails als Referenz für Entwickler verfügbar. Es gibt jedoch einige APIs, die das OLM-Dateiformat lesen und diese in eine PST-Datei schreiben können, um sie auf einem Microsoft Windows-Betriebssystem wie Aspose.Email zu verwenden.
