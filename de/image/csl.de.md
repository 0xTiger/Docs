## Was ist eine CSL-Datei?

Eine Corel Symbol Library (CSL)-Datei ist eine Symbolbibliothek, die in der CorelDRAW-Software verwendet wird. Es speichert Symbole, die in anderen Dateien verwendet werden können. CSL-Dateien können lokal oder in einem Netzwerk gespeichert werden, um sie für eine einfache Bereitstellung und Verwaltung zugänglich zu machen. Diese werden mit der CorelDRAW-Datei zusammen mit allen im Dokument verwendeten Symbolinstanzen gespeichert.

## CSL-Dateiformat

CSL-Dateien werden im proprietären CorelDRAW-Dateiformat gespeichert. Da es sich um eine Sammlung von Symbolen handelt, können die Symbole in einer CSL-Datei importiert und Zeichnungen in einem Projekt hinzugefügt werden.
