## Was ist eine SKP-Datei?

Eine SKP-Datei ist eine proprietäre dreidimensionale Modelldatei, die mit dem 3D-Designprogramm SketchUp erstellt wurde. Es kann aus mehreren verschiedenen 3D-Elementen oder -Komponenten wie Schattierungen, Drahtmodellen und Texturen bestehen. Eine SKP-Datei kann die gesamte Architektur enthalten oder aus kleineren Entwürfen (z. B. einer Tür oder einem Stuhl) eines größeren Entwurfs bestehen. Dadurch können die in solchen Dateien definierten Modelle wiederverwendet werden, indem sie in andere SketchUp-Dokumente eingefügt werden.

SKP-Dateien können anhand ihrer Erweiterung identifiziert werden, z. B. .skp, die auf einem System mithilfe der Suchfunktion des Betriebssystems leicht gesucht und gefunden werden können.

## SKP-Dateiformat - Weitere Informationen

SKP-Dateien werden im proprietären binären Dateiformat von SketchUp auf der Disc gespeichert. Das SKP-Dateiformat kann neben 3D-Bildern auch 2D-Bilder speichern. Diese 2D-Dokumente können auch Pläne, Gebäudedetails und Ansichten zeigen.
