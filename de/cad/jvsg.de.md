## Was ist eine JVSG-Datei?

JVSG ist eine CAD-Datei, die unter Windows gestartet werden kann. Es ist eine Datei im Binärformat, die von JVSG erstellt wurde. Die Videoüberwachungsprojektdateien verwenden meistens das JVSG-Dateinamensuffix. Diese Datei kann die Bandbreite, Standorte, Blickwinkel und Festplattenberechnungen einer Kamera mit Informationen über ihr CCTV-Design speichern. JVSG-Projektdateien bieten eine erweiterte Analyse und Dokumentation durch Rendern in 3D-Ansichten.

Die Überwachungskameranetzwerke verschiedener Unternehmen, die Überwachungskameras benötigen, können ebenfalls mit den JVSG-Projektdateien entworfen werden. Die Software „CCTV Design Tool“ eignet sich zur Verwaltung von JVSG-Dateien.
