## Was ist eine DWFX-Datei?

Ein DWFx (Design Web Format XPS) ist eine formatierte Darstellung einer 2D/3D-Zeichnung als XPS-Dokument. Es enthält Grafiken und Text als Teil der Konstruktionsdaten. Es ist die neueste Version des DWF-Dateiformats und kann mit dem Microsoft XPS Viewer angezeigt und gedruckt werden. Aufgrund der XPS-Eigenschaften dieser Dateien können Sie die Konstruktionsdaten mit Prüfern teilen, ohne dass sie Autodesk Design Review installieren müssen.
Ähnlich wie DWF wird DWFx von Autodesk in einem komprimierten Format entwickelt, um die Übertragung über das Internet zu ermöglichen. Eine einzelne DWFx-Datei kann eine oder mehrere Zeichnungen und Plansätze enthalten.
