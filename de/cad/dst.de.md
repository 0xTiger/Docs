## Was ist eine DST-Datei?

Eine DST-Datei ist eine AutoCAD-Datei, die Zuordnungen und Informationen zum Definieren von Plansätzen enthält. Diese werden im Standardspeicherordner für Plansätze, AutoCAD Sheet Sets, gespeichert. DST-Dateien enthalten nicht die eigentlichen Zeichnungslayouts, sondern verweisen auf diese aus ausgewählten [DWG](/de/cad/dwg/)- und [DWT](/de/cad/dwt/)-Dateien, die diesen Plansätzen zugeordnet sind. In einer Netzwerkumgebung sollten alle Teammitglieder Netzwerkzugriff auf diese zugehörigen Dateien haben. DST-Dateien werden im Dateiformat [XML](/de/web/xml/) auf der Disc gespeichert.

## DST-Dateiformat – Weitere Informationen

DST-Dateien werden mit dem Tool AutoDesk Sheet Set Manager (SSM) erstellt. Wenn jemand im Team auf eine Datei zugreift, wird die DST-Datei kurz geöffnet und dann mit den aktualisierten Informationen zurückgespeichert. Der gleichzeitige Zugriff auf dieselbe DST-Datei wird vom SSM-Tool verwaltet, das ein Schlosssymbol anzeigt, wenn auf die Datei zugegriffen wird.

Der Blattstatus kann sich zu jedem beliebigen Zeitpunkt in einem der folgenden Zustände befinden.

* Das Blatt steht zur Bearbeitung zur Verfügung.
* Das Blatt ist gesperrt.
* Das Blatt fehlt oder wurde an einem unerwarteten Speicherort gefunden.
