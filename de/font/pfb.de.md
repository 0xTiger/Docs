## Was ist eine PFB-Datei?

Eine Datei mit der Erweiterung .pfb ist eine Printer Font Binary-Version der Type 1-Schriftart von Adobe. Sie ähnelt .pfa(/de/font/pfa/)-Dateien, ist aber binärer Natur. PFB-Dateien speichern Adobe Type 1-Schriftarten, die für das Windows-Betriebssystem verwendet werden. Diese enthalten Glyphendaten der Schriftart und speichern Daten in kompakter Form im Binärformat. PFB-Dateien haben nicht nur binär codierten Inhalt, sondern sind auch verschlüsselt. PFB-Dateien erforderten außerdem .pfm-Dateien, die Messungen von Zeichen innerhalb der Schriftart enthalten. Um PFB-Schriftarten zu installieren, müssen sowohl die .pfb- als auch die .pfm-Datei im System abgelegt werden.
