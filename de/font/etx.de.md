## Was ist eine ETX-Datei?

Eine ETX-Datei ist eine Schriftdatei, die von den Satzsystemen TeX und LaTeX verwendet wird. Sie wird als Textdatei gespeichert und definiert das Aussehen einer Schriftart durch ihre Kodierung. ETX-Schriftartendateien werden in Software geladen, die die Möglichkeit bietet, mit Tex-Dokumenten zu arbeiten. Nach dem Laden können die ETX-Schriftartendateien zur Anpassung von TeX-Dokumenten verwendet werden.
