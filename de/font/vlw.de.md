## Was ist eine VLW-Datei?

Eine VLW-Datei ist ein Schriftformat, das von der Bild- und Animationserstellungssoftware **Verarbeitung** verwendet wird. Es speichert Vektorschriftarten auf kompakte Weise, die ohne Qualitätsverlust auf jede beliebige Größe skaliert werden können. Dies macht seine Verwendung in Grafiken und visuellem Design. Die Verarbeitung bietet die Möglichkeit, Bilder und Animationen zu erstellen, bei denen diese VLW-Schriftart-Glyphen verwendet werden.

## VLW-Dateiformat - Weitere Informationen

VLW-Dateien werden als Binärdateien auf der Disc gespeichert. In Processing können Sie die Funktion **createFont()** verwenden, um ein Schriftartobjekt aus einer VLW-Datei zu erstellen. Sie können dieses Schriftobjekt dann verwenden, um Text in Ihren Skizzen zu zeichnen, indem Sie die Funktion **text()** verwenden. Die Verarbeitung kann VLW-Dateien mit dem Befehl **loadFont()** lesen.
