## Was ist eine JFPROJ-Datei?

Eine Datei mit der Erweiterung .jfproj ist eine JSON-basierte benutzerdefinierte Schriftart, die mit der Microsoft Font Maker-Anwendung unter Windows 10 erstellt wurde. Diese können mithilfe eines elektronischen Stifts mit Touchscreen-Geräten wie Tablets erstellt werden, obwohl auch andere Computerperipheriegeräte wie eine Maus verwendet werden können von Computern. JFPROJ-Dateien enthalten die Glyphen, die zum Rendern der Zeichen auf dem Bildschirm verwendet werden, indem sie den Abstand zwischen Zeichen und Wörtern sowie die Größe der Schriftart definieren. Die Microsoft Font Maker-App ist im Windows Store zum Download verfügbar.
