## Was ist eine PFA-Datei?

Eine Datei mit der Erweiterung .pfa ist eine ASCII-Version der Druckerschriftart der Type 1-Schriftart von Adobe und enthält die Glyphendaten der Schriftart. Die Datei ist in menschenlesbarem Format und kann in jedem Texteditor geöffnet werden. Der PostScript-Interpreter kann diese Schriftart als Ganzes für seine Arbeit verwenden. Die Printer Font Binary (PFB)-Version der Type 1-Schriftart von Adobe enthält dieselben Informationen. Wenn Sie jedoch PFA entlinen, liegt es im Binärdateiformat vor und ist nicht für Menschen lesbar. PFA-Dateien können in das System integriert werden, indem sie in den Fonts-Ordner des Systems kopiert werden.
