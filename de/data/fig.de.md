## Was ist eine FIG-Datei?

Eine FIG-Datei ist eine spezielle Datei, die Bilder oder Grafiken enthält, die mit dem MATLAB-Programm erstellt wurden und für Berechnungen und Datenanalysen verwendet werden. Diese Datei enthält Informationen zu Bildern oder Diagrammen, die MATLAB zeichnet, um Menschen dabei zu helfen, Daten besser zu sehen und zu verstehen. Es ist wie ein Container, der alle Details zu Diagrammen speichert.

.FIG ist das Standardformat für MATLAB-Figurendateien. Es speichert die vollständige Figur, einschließlich aller Achsen, Beschriftungen und Einstellungen. Um eine Figur in diesem Format zu speichern, können Sie die Funktion „savefig“ verwenden:

## Wie erstelle ich eine FIG-Datei mit MATLAB?

Um eine FIG-Datei mit MATLAB zu erstellen, gehen Sie folgendermaßen vor:

1. **Wählen Sie Ihre Daten aus:** Wählen Sie im Bereich „Arbeitsbereich“ in MATLAB die Variable aus, für die Sie ein Diagramm erstellen möchten.

2. **Wählen Sie den Grundstückstyp:** Gehen Sie zur Registerkarte „GRUNDSTÜCKE“ und klicken Sie auf den gewünschten Grundstückstyp.

3. **Speichern Sie den Plot als FIG-Datei:**

     - Gehen Sie zum Menü „Datei“.
     - Wählen Sie entweder „Speichern“ oder „Speichern unter“.
     - Geben Sie einen Namen für Ihre Datei ein und entscheiden Sie, wo Sie sie speichern möchten.
     - Klicken Sie auf „Speichern“, um Ihre FIG-Datei zu erstellen.

## Wie öffne ich eine FIG-Datei?

Um die FIG-Datei in MATLAB zu öffnen, gehen Sie folgendermaßen vor:

1. **Matlab öffnen:** Starten Sie MATLAB auf Ihrem Computer.

2. **Navigieren Sie zur Registerkarte „GRUNDSTÜCKE“:** Klicken Sie auf der Registerkarte „GRUNDSTÜCKE“ auf die Parzelle und wählen Sie den gewünschten Grundstückstyp aus.

3. **FIG-Datei öffnen:**

     - Gehen Sie zum Menü „Datei“.
     - Wählen Sie „Öffnen“.
     - Navigieren Sie zum Speicherort Ihrer FIG-Datei, wählen Sie sie aus und klicken Sie auf „Öffnen“.
