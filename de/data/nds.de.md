## Was ist eine NDS-Datei?

Die Dateierweiterung .NDS wird häufig mit **Nero DiscSpan Compilation File** in Verbindung gebracht. Nero DiscSpan ist eine Funktion in **Nero Burning ROM**, einem beliebten Programm zum Erstellen optischer Datenträger. Mit dieser Funktion können Benutzer große Datenmengen auf mehrere Discs verteilen.

Wenn Sie in Nero Burning ROM eine DiscSpan-Zusammenstellung erstellen, werden die Daten in kleinere Segmente unterteilt, die auf einzelne Discs passen. Die .nds-Datei wird als Deskriptordatei erstellt, die Informationen zum Layout und zu den datenträgerübergreifenden Daten enthält.

Wenn Sie über eine .nds-Datei und entsprechende Disc-Segmente verfügen, können Sie Nero Burning ROM oder Nero DiscSpan verwenden, um Originaldaten wiederherzustellen. Nero DiscSpan fordert Sie auf, die einzelnen Discs der Reihe nach einzulegen, um die vollständige Zusammenstellung wiederherzustellen.

## Wie öffne ich eine NDS-Datei?

Zu den Programmen, die NDS-Dateien öffnen, gehören:

- **Nero 2022** (kostenpflichtig) für Windows
