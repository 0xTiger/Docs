## Was ist eine WBFS-Datei?

Eine WBFS-Datei (Wii Backup File System) ist ein Videospiel-Disk-Image, das für Nintendo Wii erstellt wurde. Es enthält ein Wii-Spiel, das als Bild zum Spielen geladen werden kann. Es wird mit dem Wiimms WBPS-Tool oder dem Wii Backup Manager generiert und kann auf einer Nintendo Wii abgespielt werden, indem es von einem USB Loader GX oder einem Wii-Emulator wie Dolphin geladen wird. Das Spiel wird auf diesen externen Speichergeräten im WBFS-Dateiformat gespeichert. Die Wii ist eine Spieleplattform, die von Nintendo vertrieben wird. Das WBFS-Dateisystem wurde vom Wii-Homebrew-Programmierer Waninkoko entwickelt.

## WBFS-Dateiformat – Weitere Informationen

WBFS sind Binärdateien, die ein proprietäres Dateiformat haben. Seine Dateiformatspezifikationen sind nicht öffentlich verfügbar.
