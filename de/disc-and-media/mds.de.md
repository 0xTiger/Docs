## Was ist eine MDS-Datei?

Das MDS (Media Descriptor Sidecar File) ist eine Datei, die mit der Software Alcohol 120% und Daemon Tools verknüpft ist, die beide zum Erstellen und Verwalten virtueller CD- und DVD-Laufwerke verwendet werden. Die MDS-Datei enthält Informationen über das Layout der Daten auf der Disc, einschließlich des Speicherorts aller Dateien und der Struktur der Disc. Dadurch kann das virtuelle Laufwerk das Verhalten einer physischen Disc emulieren, sodass die Software die Daten auf der virtuellen Disc so lesen kann, als wäre es eine echte Disc.

Wenn Sie eine Image-Datei auf einem virtuellen Laufwerk bereitstellen möchten, wird die MDS-Datei zusammen mit der entsprechenden Image-Datei in einem Format wie ISO, BIN oder CUE verwendet. Die MDS-Datei enthält eine Beschreibung des Layouts der Daten auf der Disc, und die Software für virtuelle Laufwerke verwendet diese Informationen, um eine virtuelle Disc zu erstellen. Auf diese Weise können Sie die Software so verwenden, als ob Sie eine physische Disc verwenden würden, ohne die Disc physisch in das Laufwerk einlegen zu müssen.

## Beziehung zu Alkohol 120%

Alcohol 120% ist eine beliebte Software zum Erstellen und Verwalten virtueller CD- und DVD-Laufwerke und verwendet das MDS-Dateiformat zum Speichern und Zugreifen auf Disc-Images. Das MDS-Dateiformat ist Eigentum von Alcohol 120% und kann nur von der Alcohol 120%-Software oder anderer Software, die es unterstützt, geöffnet und verwendet werden.

## Wie öffne ich eine MDS-Datei?

Um eine MDS-Datei in Alcohol 120% zu öffnen, muss die Software auf Ihrem Computer installiert sein. Sobald Sie Alcohol 120% installiert haben, können Sie die MDS-Datei wie folgt öffnen:

1. Starten Sie Alkohol 120%.
2. Klicken Sie auf das Menü „Datei“ und wählen Sie „Öffnen“.
3. Navigieren Sie zum Speicherort der MDS-Datei auf Ihrem Computer und wählen Sie sie aus.
4. Die MDS-Datei wird nun in Alcohol 120% geöffnet und Sie werden aufgefordert, die entsprechende Image-Datei auszuwählen (z. B. ISO, BIN oder CUE).
5. Nachdem Sie die Image-Datei ausgewählt haben, stellt Alcohol 120% das Image auf einem virtuellen Laufwerk bereit.

Alternativ können Sie eine MDS-Datei auch per Doppelklick öffnen, wenn Alcohol 120% als Standardprogramm zum Öffnen von MDS-Dateien eingestellt ist. Sie können auch andere Software verwenden, die das MDS-Format unterstützt, wie Daemon Tools, Virtual CloneDrive, Virtual Drive Manager und MagicISO.
