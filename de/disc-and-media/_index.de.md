Eine Disc-Image-Datei ist eine Computerdatei, die den Inhalt und die Struktur eines Disc-Volumes enthält. Alternativ können Sie sagen, dass die Disc-Image-Datei ein ganzes Datenspeichergerät enthält, wie z. B. ein USB-Flash-Laufwerk, eine Diskette, ein Bandlaufwerk, eine optische Disc oder ein Festplattenlaufwerk. Normalerweise wird ein Disc-Image erstellt, indem eine Sektor-für-Sektor-Kopie des Quellmediums erstellt wird, wodurch die Struktur und der Inhalt eines Speichergeräts unabhängig vom Dateisystem genau kopiert werden. Ein Disc-Image kann eine oder mehrere Computerdateien umfassen.

Einige beliebte Disc-Dateiformate sind [Toast](/de/disc-and-media/toast/), [BIN](/de/disc-and-media/bin/) und [VCD](/de/disc-and-media/vcd/)
