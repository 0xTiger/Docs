## Was ist eine EMBR-Datei?

Eine EMBR-Datei ist eine GIS-Datei, die von GEOREID Pty Ltd entwickelt und verwendet wurde. Sie enthält Geodaten wie berechnete Länge und Volumen an einem bestimmten Punkt. Das EMBR-Dateiformat speichert diese statistischen Daten im Nur-Text-Format, das in einem Texteditor zum Anzeigen und Bearbeiten geöffnet werden kann. GIS-Systeme speichern geologische Informationen in Bezug auf jeden Lat-Long-Punkt, der die Absolutposition auf der Erde darstellt.

Andere beliebte GIS-Dateiformate zum Speichern von Geodaten sind [gpx](/de/gis/gpx/), [geojson](/de/gis/geojson/), [kml](/de/gis/kml/) und [kmz](/de/gis/kmz/).

## EMBR-Dateiformat – Weitere Informationen

EMBR-Dateien sind Textdateien, die statistische Daten zu einem bestimmten Zeitpunkt speichern. Die Daten können verwendet werden, um verschiedene Berechnungen durchzuführen, wie z. B. die Berechnung von Hochrechnungen und den Vergleich von Daten. Wenn Sie eine EMBR-Datei haben, können Sie sie in einem Texteditor wie Notepad, Notepad++ oder Apple TextEdit öffnen, um ihren Inhalt anzuzeigen und die Informationen zu aktualisieren.
