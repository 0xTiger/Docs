## Was ist eine PMF-Datei?

Eine PMF-Datei ist eine schreibgeschützte Kartendatei, die mit ArcMap generiert wird. Man kann sagen, dass es sich um eine kürzere Version des MXD-Dateiformats handelt, das begrenzte Informationen enthält, die mit Benutzern geteilt werden können, die keine kostenpflichtige Version der ArcGIS-Software haben. Auf diese Weise können Sie gleichzeitig geografische Informationen mit anderen teilen. PMF-Dateien können aufgrund ihrer geringen Größe aufgrund der eingeschränkten Informationen, die sie enthalten, mit anderen Benutzern per E-Mail geteilt werden.

PMF-Dateien können mit ESRI ArcReader und ESRI ArcGIS Pro geöffnet werden.

## PMF-Dateiformat

PMF-Dateien begrenzen die Datenmenge, auf die Benutzer basierend auf ihrem Abonnement zugreifen können. Wenn ein Benutzer eine kostenlose Version der ArcGIS-Software verwendet, schränken PMF-Dateien die Informationsanzeige wie Datenansichten, Layouts, Navigation, Drucken, Abfragen, Schwenken und Zoomen der Karte ein. Die Daten sind möglicherweise nicht tatsächlich in der Begrenzungsdatei enthalten, sondern werden von einem Remote-Standort abgerufen, der auf dem ArcMap-Server gehostet wird oder über das Internet verfügbar ist.
