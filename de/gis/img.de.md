## Was ist eine IMG-Datei?

Eine Datei mit der Erweiterung .img ist eine Garmin-Kartendatei, die von Garmin-GPS-Geräten verwendet wird. Es enthält Karten-/Geodaten zur Darstellung von Kartenelementen wie Straßennamen, Navigationsrouten und Points of Interest. Benutzer können benutzerdefinierte Kartendaten erstellen, als .img-Datei speichern und auf Garmin-Geräte laden. Garmin bietet auch vorinstallierte Kartendaten, die auf Garmin-Geräten im IMG-Dateiformat gespeichert sind. IMG-Dateien können mit Anwendungen wie Garmin Mobile PC, cGPSmapper, MapUpload und GPS Map Manager geöffnet werden.

## IMG-Dateiformat - Weitere Informationen

IMG-Dateien werden als Binärdateien im Garmin-eigenen Dateiformat gespeichert. Informationen, die in einer IMG-Datei verfügbar sind, können auf Garmin-GPS-Geräten gerendert werden. Einige GPS-Geräte, die Autorouting unterstützen, können diese Kartendateien auch zur Orientierung während der Navigation verwenden, z. B. im Auto, auf dem Fahrrad oder beim Wandern.

GMAP-Dateien ähneln einem Festplatten-Image und basieren auf einem Dateisystem, das aus Unterdateien besteht, um Funktionen für Garmin-GPS-Geräte bereitzustellen. Die .GMP-Unterdateien bieten eine Art Unterverzeichnisstruktur, obwohl das .gmap-Dateiformat flach ist.
