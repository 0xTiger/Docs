## Was ist eine SDF-Datei?

Eine Spatial Data File (SDF) ist ein Geodatabase-Dateiformat, das von Autodesk entwickelt wurde. Es speichert Geodaten und wird vom Autodesk GIS-Programm MapGuide und AutoCAD Map 3D verwendet. Das SDF-Dateiformat basiert auf dem Einzeldatei-Datenbankdateiformat SQLite3. Es gilt als optimiertes Dateiformat für große Datensätze mit räumlichen Informationen.

Sie können eine SDF-Datei mit Autodesk AutoCAD Map 3D 2022 und Autodesk AutoCAD Civil 3D 2023 öffnen.

## SDF-Dateiformat – Weitere Informationen

SDF-Dateiformat werden als Binärdateien auf der Disc gespeichert. Es basiert auf der SQLite-Datenbank, die Low-Level-Speicherkomponenten für die binäre Serialisierung von Daten verwendet. Eine SDF-Datei enthält mehrere Objektklassen pro Datei und mehrere Geometrieeigenschaften für jede Objektklasse. Auf Daten in einer SDF-Datei kann mit hoher Geschwindigkeit zugegriffen werden, da jede Geometrieeigenschaft mithilfe eines R-Baums indexiert wird.
