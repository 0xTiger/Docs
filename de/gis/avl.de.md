## Was ist eine AVL-Datei?

Eine AVL-Datei ist eine Legendendatei, die einen Schlüssel für eine Karte enthält, die mit ESRI ArcView GIS (Geographic Information System) Software erstellt wurde. Sie enthält Informationen zur Referenzsymbologie, die in der entsprechenden Karte für den Zugriff verwendet werden. Eine AVL-Datei kann je nach Typ weitere Informationen wie Symbologie, Anzeigeeinstellungen wie Transparenz, maßstabsabhängige Anzeigeeigenschaften, Informationen zu verknüpften Tabellen/zugehörigen Tabellen, Definitionsabfragen, Beschriftungseigenschaften für Feature-Layer und Annotation-Anzeigeeigenschaften für Annotation-Layer enthalten Art Schicht.

Eine AVL-Datei kann von der ArcView-Projektdatei [.APR](/de/gis/apr/) referenziert werden.

## AVL-Dateiformat – Weitere Informationen

AVL-Dateien werden im Nur-Text-Dateiformat gespeichert. Das bedeutet, dass Sie AVL-Dateien in jedem Texteditor wie Notepad, Notepad++ und Apple TextEdit öffnen können. Nach dem Öffnen können Sie den Inhalt der Datei nicht nur ansehen, sondern auch verändern.

### Unterschied zwischen .LYR- und .AVL-Dateien

* **Unterschied im Dateiformat** - .lyr ist eine Binärdatei, während .avl eine Textdatei ist
* **Inhaltsunterschied** - Eine .lyr-Datei enthält mehr Informationen als eine .avl-Datei. Eine AVL-Datei speichert nur Symbolisierungsinformationen, während eine LYR-Datei alle Informationen speichert, die im Dialogfeld "Layer-Eigenschaften" in ArcMap verfügbar sind.
