## Was ist eine NMC-Datei?

Eine NMC-Datei ist eine Kartenpaket-Inhaltsdatei, die mit ESRI ArcGIS Explorer-Software erstellt wurde. Es kann verschiedene Elemente einer Karte wie Bildüberlagerungen, GIS-Service-Layer, [KML](/de/gis/kml/)-Layer, Raster-Layer, Features und andere Pakete speichern. NCM-Dateien werden zum Exportieren von Kartendaten verwendet, die in anderen GIS-Karten verwendet werden können, die in ArcGIS Explorer geöffnet sind. Dadurch können mehrere Karten übereinander gelegt und angezeigt werden.

## NMC-Dateiformat

NMC-Dateien werden im Binärdateiformat gespeichert, wobei deren interne Struktur nicht verfügbar ist. Diese können mit anderen geteilt werden, die diese NMC-Dateien an ihrem Ende in ArcGIS Explorer laden können. Dadurch können NMC-Dateien auf den Computern anderer Benutzer verwendet werden, indem sie per E-Mail an andere gesendet werden.
