## Was ist eine MXD-Datei?

Eine MXD-Datei ist eine Kartendokumentdatei, die von der ESRI ArcMap-Software verwendet wird. Es handelt sich um ein proprietäres Dateiformat, das verschiedene Arten von Informationen speichert, z. B. das Layout und Design von Karten, Daten, die zum Erstellen von Karten verwendet werden, und die auf die Daten angewendete Symbologie. Die MXD-Datei ist daher ein zusammengesetztes Dateiformat, das mehrere verschiedene Arten von Kartenelementen in einer einzigen Datei enthält. MXD-Dateien werden von der ArcGIS Map Package-Datei ([.mpk](/de/gis/mpk/)) verwendet.

## MXD-Dateiformat

MXD-Dateien werden im proprietären ArcGIS-Dateiformat gespeichert und ihre internen Details sind nicht als Referenz für Entwickler verfügbar. Die eingebetteten Elemente einer MXD-Datei sind in Einheiten organisiert, die als Speicher und Streams bekannt sind.
