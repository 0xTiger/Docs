## Was ist eine TGP-Datei?

Eine TGP-Datei dient als Plugin für Terragen Classic, ein Programm zur Erstellung fotorealistischer Naturlandschaften. Diese Datei kapselt Daten, die die Funktionalitäten von Terragen bei der Installation verbessern. TGP-Dateien werden verwendet, um Terragen verschiedene Funktionen hinzuzufügen, darunter Kameraeffekte und zusätzliche Landschaftsoptionen.

## Über Terragen Classic

Terragen ist eine von Planetside Software entwickelte Software zur Landschaftsgenerierung und -darstellung. Die Classic-Version bezieht sich auf eine ältere Version der Software, da es auch eine neuere Version namens **Terragen 4** gibt.

Terragen Classic war für seine Fähigkeiten bei der Erstellung realistischer Landschaften und Gelände bekannt. Es ermöglichte Benutzern, detaillierte und glaubwürdige natürliche Umgebungen zu erstellen, darunter Berge, Täler, Seen und Himmel. Die Software umfasste Funktionen zur Geländeformung, Texturkartierung und Rendering.
