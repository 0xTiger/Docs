## Was ist eine Q1Q-Datei?

Eine .q1q-Datei ist eine von Flaming Pear Software entwickelte Photoshop-Plugin-Datei zum Erstellen von 3D-Planeten. Damit können Sie LIVE-Wolkendaten, Luft, Textur, Klima und viele andere Details herunterladen. Außerdem können Sie alle Texturkarten in mehrschichtige Dateien exportieren und Karten für 3D-Animationen und Kartenerstellung erstellen.

## Q1Q-Dateiformat

Q1Q-Plugin-Dateien werden im proprietären Flaming Pear-Dateiformat gespeichert und die internen Dateiformatdetails sind nicht öffentlich verfügbar. Das folgende Bild zeigt die Benutzeroberfläche des Q1Q-Plugins beim Laden in Photoshop.

## Wie öffne ich eine Q1Q-Datei?

Sie können Q1Q-Dateien mit **Adobe Photoshop 2023** und **Corel PaintShop Pro 2023** öffnen.
