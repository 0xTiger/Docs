## Was ist eine AMXX-Datei?

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. Dabei handelt es sich um eine Skriptplattform, die es Serveradministratoren ermöglicht, die Funktionalität ihrer Server anzupassen und zu erweitern. AMXX wird in Verbindung mit dem Plugin-Verwaltungssystem MetaMod verwendet, das die gleichzeitige Ausführung mehrerer Plugins auf einem Server ohne Konflikte ermöglicht. Zu den vom AMXX-Plugin angebotenen Funktionen gehören Admin-Befehle, Gameplay-Änderungen und Serververwaltung.

Sie können AMXX-Dateien mit [AMX Mod X](https://www.amxmodx.org/) auf Windows- und Linux-Betriebssystemen öffnen.
