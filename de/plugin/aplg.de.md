## Was ist eine APLG-Datei?

Eine Datei mit der Erweiterung .aplg ist eine Plugin-Datei, die von der Internet-Stream-Aufzeichnungssoftware Audials verwendet wird. Das Plugin wird von der Software geladen, um neue Funktionen wie neue Such- und Aufnahmefunktionen hinzuzufügen, um Internet-Musik- und Videostreams aufzunehmen. Audials bietet einen funktionsreichen Radioplayer, der es Nutzern ermöglicht, auf eine Vielzahl von Internetradiosendern zuzugreifen und diese anzuhören.

## APLG-Dateiformat

APLG-Dateien sind in den .APLP-Plugin-Paketen verpackt, was bedeutet, dass die Audials-Software mehrere Plugins gleichzeitig aus dem Paket laden kann. Benutzer können häufig ihre Lieblingslieder oder Radiosendungen für die spätere Wiedergabe aufzeichnen. Dies kann nützlich sein, um personalisierte Wiedergabelisten zu erstellen. Audials verfügt in der Regel über eine große Datenbank mit Radiosendern, sodass Benutzer leicht neue Inhalte entdecken können.
