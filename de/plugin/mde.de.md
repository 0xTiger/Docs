## Was ist eine MDE-Datei?

Eine .MDE-Datei ist eine kompilierte Version einer Access-Datenbank (.mdb oder .accdb). Wenn Sie eine Access-Datenbank in eine .MDE-Datei kompilieren, kann sie nicht mehr mit den standardmäßigen Access-Designtools bearbeitet werden. Der Hauptzweck der Erstellung einer .MDE-Datei besteht darin, eine Datenbankanwendung zu verteilen, ohne den ursprünglichen Quellcode offenzulegen.

## MDE-Dateiformat

Eine MDE-Datei wird durch Kompilieren des VBA-Codes, Formularen, Berichten und anderen Objekten erstellt. Dies führt zur Erstellung des MDE-Binärdateiformats. Die resultierende .MDE-Datei kann an Benutzer verteilt werden, die die Anwendung ausführen, aber das Design nicht ändern oder den Originalcode anzeigen können. Die MDE-Datei ist eigentlich die kompilierte Version einer [.MDA-Datei](/database/mda/). Das Verteilen von .MDE-Dateien kann zum Schutz des geistigen Eigentums beitragen, indem Benutzer daran gehindert werden, auf den Quellcode zuzugreifen oder ihn zu ändern. Es kann auch die Leistung verbessern, wenn der Code kompiliert und optimiert wird.
