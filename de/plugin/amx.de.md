## Was ist eine AMX-Datei?

Eine AMX-Datei ist eine angepasste Plugin-Datei für das AMX-Mod-Metamod-Plugin für Valve Half-Life und verwandte Spiele. Es fügt den bestehenden Features des Spiels zusätzliche Features und Funktionen hinzu.

## AMX-Dateiformat – Weitere Informationen

AMX Mod verwendet die AMX-Dateierweiterung. Die erweiterte Version des Plugins verwendet jedoch die Dateierweiterung .AMXX.

## Wie öffne ich eine AMXX-Datei?

Sie können AMX-Dateien mit [AMX Mod X](https://www.amxmodx.org/) auf Windows- und Linux-Betriebssystemen öffnen.
