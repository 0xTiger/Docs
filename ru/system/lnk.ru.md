## .LNK вариант № ##

Файл LNK, аналогичный удостоверению в системе Mac, является альтернативой Windows или «ссылкой», которая служит для подключения к исходной папке документа изображения или программе. Он содержит тип, положение и имя файла назначения ярлыка, а также приложение, открывающее целевой документ, и дополнительную клавишу быстрого доступа.

В Windows выберите файл, папку или исполняемую программу и выберите «Создать ярлык». Расположение формата файла и каталог «Начало» — две основные особенности файлов LNK. Формат файлов LNK замаскирован, а изогнутая стрелка указывает на то, что они являются ярлыками.

## Формат файла LNK ##

Форматы файлов LNK обычно имеют тот же значок, что и файлы назначения, но с добавлением небольшой изогнутой стрелки, показывающей, что файл указывает на другое место. Когда ярлык дважды щелкнут, он ведет себя так, как если бы пользователь дважды щелкнул фактический файл.

Файлы LNK, содержащие ярлыки приложений, могут иметь свойства, влияющие на работу программы. Чтобы изменить атрибуты, щелкните правой кнопкой мыши файл ярлыка и выберите **Свойства**, затем измените поле «Цель».

Вместо того, чтобы быть расширениями файлов, файлы LNK являются расширениями Windows Explorer. В качестве терминального расширения документы .lnk можно использовать только в проводнике Windows для замены файла, и они имеют другие цели в проводнике Windows, помимо использования в качестве ярлыка для локального документа. Эти файлы также начинаются с буквы «L».

Хотя ярлыки ссылаются на определенные файлы и каталоги при их создании, они могут стать неработоспособными, если цель будет изменена на другое место. Проводник начнет восстанавливать папку ярлыка, которая указывает на мертвую цель при ее открытии.


## Техническая спецификация ##

Только когда параметр просмотра папки «Скрыть расширения для распознаваемых типов файлов» не отмечен, Windows не отображает расширение документа .lnk для ярлыков документов. Хотя это не рекомендуется, вы можете включить отображение расширения файла, удалив свойство «NeverShowExt» из элемента реестра Windows HKEY_CLASSES_ROOT\lnk file.

Для этого выполните следующие действия:

* Откройте «Редактор регистрации», введя «Regedit» в поле поиска на панели задач и выбрав программу
* В приложении перейдите в папку Computer\HKEY HKEY_CLASSES_ROOT\lnkfile.
* Сделайте резервную копию элемента, щелкнув его правой кнопкой мыши и выбрав Экспорт.
* Выберите и удалите атрибут "NeverShowExt"
* Windows следует перезагрузить
