## .VGA вариант №

Файл VGA — это файл на диске Windows для поддержки мониторов VGA. Он использовался в ранних версиях Microsoft Windows для настройки мониторов VGA на уровне операционной системы (ОС). Файл VGA, используемый для того, чтобы пользователи могли регулировать разрешение и глубину цвета дисплея с помощью панели управления монитора.

## Формат файла VGA

Внутренний формат файла VGA неизвестен для справки разработчика. На него ссылается операционная система Windows для настройки внешнего вида монитора VGA.
