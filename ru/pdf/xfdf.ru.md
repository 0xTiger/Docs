## .XFDF вариант №

Файл с расширением .xfdf представляет собой формат данных XML Forms, созданный с помощью программного обеспечения Adobe Acrobat. Как и [FDF](/ru/pdf/fdf/), XFDF содержит описание элементов формы и их значения в формате XML. Это может включать имена и значения текстовых полей в форме PDF. XFDF сохраняются в удобочитаемом формате и могут быть прочитаны программно с использованием таких языков программирования, как C#.

## Формат файла XFDF — дополнительная информация

Файлы XFDF сохраняются в формате файла XML, который является универсальным форматом, используемым для импорта и экспорта данных. Структура файла XFDF состоит из заголовка, за которым следуют значения полей и данные элементов, как показано ниже.

|Элемент|Атрибут|Содержание|
---|---|---|
|\<XFDF> ||Самый верхний элемент|
|\<fields> ||Все значения полей для этого шаблона|
|<field (T)> |имя |имя поля|
|<value (V)> |значение |Значение поля|
