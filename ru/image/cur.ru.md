## .CUR вариант № ##

Файл CUR представляет собой статический формат файла курсора Microsoft Windows. По сути, это стационарные изображения, идентичные ICO-файлам (значкам) во всем, кроме расширения. И CUR, и [ICO](/ru/image/ico/) устанавливаются на основе спецификации Device-Independent Bitmap [DIB](/ru/image/dib/) (Device-Independent Bitmap). Курсор по умолчанию, а также пользовательский курсор, такой как указатель мыши Windows для операционной системы Windows, хранятся в файлах CUR. Это может быть стрелка для обычного использования, вращающиеся песочные часы для демонстрации периодов ожидания или I-bar для редактирования текста. Файлы CUR поставляются вместе с установочным пакетом пользовательских тем рабочего стола, чтобы гарантировать, что курсор Windows соответствует пользовательской теме рабочего стола.

## Техническая спецификация ##

Файлы CUR — это двоичные системные файлы, которые можно найти на устройствах, работающих в операционной системе Microsoft Windows. Изображения указателей CUR бывают разных стандартных размеров, таких как 16 × 16, 32 × 32 и т. Д. Файлы CUR очень похожи на файлы ICO; оба они состоят из небольших стационарных изображений. Различаются только расширения файлов CUR и ICO. Кроме того, объяснение точки доступа в заголовке файла CUR отличается от файла ICO. Активная точка интерпретирует смещение пикселя от верхнего левого угла до места, куда вы указываете мышью. Системы Windows по-прежнему используют файлы CUR, но теперь анимированные курсоры обычно имеют расширение файла ANI вместо CUR.

## Краткая история ##

Файл CUR создан из файла ICO. Первый формат файла CUR был включен в операционную систему Microsoft Windows 1.0 в 1981 году для упрощения коммерческого использования. Вскоре было представлено больше интерактивных курсоров, позволяющих пользователям изменять свои настройки курсора из доступных предустановленных опций. Однако файл CUR легко отредактировать самостоятельно, даже если у вас недостаточно технических знаний.
