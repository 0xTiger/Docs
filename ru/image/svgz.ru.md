## .SVGZ вариант №

Файл с расширением .svgz представляет собой сжатую версию файла масштабируемой векторной графики (.SVG). Он сжат сжатием gzip и содержит данные в формате [XML](/ru/web/xml/). Файлы SVGZ поддерживают прозрачность, градиенты, анимацию и фильтры. Файлы SVGZ меньше по размеру по сравнению с файлами SVG по умолчанию, и этот уменьшенный размер файла помогает передавать графические файлы онлайн. Графический дизайнер создает файлы SVGZ с помощью таких инструментов, как Adobe Illustrator, Corel PaintShop Pro и других. Однако файлы SVGZ можно создавать, включив [сжатие GZip](/ru/compression/gz/) на сервере Apache при отправке данных изображения.

## Формат файла SVGZ

SVGZ, будучи сжатым [SVG](/ru/page-description-language/svg/), основан на открытом сжатом формате gzip, который уменьшает размер документов SVG. Этот сжатый архив содержит простые текстовые документы SVG внутри, которые полностью основаны на формате файлов XML (/ru/web/xml/). Файлы SVGZ могут открываться средствами просмотра/приложениями, соответствующими спецификациям SVG 1.1. Файлы SVGZ на 20-50% меньше по размеру, чем исходные файлы SVG.
