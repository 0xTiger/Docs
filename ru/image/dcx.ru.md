## .DCX вариант №

Файл DCX представляет собой файл изображения, состоящий из нескольких страниц для создания файлов цифровых факсов. Каждый файл DCX начинается с 4-байтового заголовка, идентифицирующего файл DCX как сигнатуру, за которым следуют отдельные изображения, хранящиеся в виде встроенных файлов [PCX](/ru/image/pcx/). 4-байтовая подпись идентифицирует файл DCX, за ней следует список до 1023 4-байтовых смещений файла. Формат файла DCX был разработан Zsoft. Некоторые приложения могут открывать файлы DCX, такие как Adobe Photoshop, Corel PaintShop Pro и IrfanView.
