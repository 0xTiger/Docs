## .CR3 вариант №

Файл CR3 — это формат файла изображения Canon RAW, который создается некоторыми цифровыми камерами Canon. Он был представлен Canon для замены формата файла CR2 и используется некоторыми цифровыми устройствами Canon. Файлы CR3 хранят исходные необработанные данные изображения без потерь, снятые камерой. Использование этих необработанных изображений обеспечивает высокое качество изображений и дает фотографам достаточно места для редактирования. В дополнение к основным данным изображения файлы CR3 также хранят метаданные об изображении.

## Формат файла CR3

Файлы CR3 хранятся на диске в виде двоичного файла в базовом формате медиафайлов ISO (ISO/IEC 14496-12), а также включают пользовательские [теги Canon](https://exiftool.org/TagNames/Canon.html#uuid). Он также включает [кодек Canon 'crx'](https://github.com/LibRaw/LibRaw/blob/master/src/decoders/crx.cpp), представляющий собой смесь JPEG-LS (Rice-Golomb + RLE). кодирование) и JPEG-2000 (LeGall 5/3 DWT + квантификация).

## Пользовательские теги CR3

Файлы CR3 содержат настраиваемые теги для различных целей. Ниже перечислены некоторые из этих тегов.

|Идентификатор тега| Имя тега| Доступно для записи |Значения / Примечания|
---|---|---|---|
|'CCTP' |CanonCCTP| - |--> Теги Canon CCTP|
|'CMT1' |IFD0| - |--> EXIF-теги|
|'CMT2' |ExifIFD| - |--> EXIF-теги|
|'CMT3' |MakerNoteCanon| - |--> Теги Canon|
|'CMT4' |Информация GPS| - |--> GPS-метки|
|'CNCV' |Версия компрессора| нет| |
|'CNOP' |CanonCNOP| - |--> Теги Canon CNOP|
|'CNTH' |CanonCNTH| - |--> Теги Canon CNTH|
|'THMB' |Эскиз изображения| нет| |
