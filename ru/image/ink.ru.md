## .INK вариант №

Формат файла «.ink» связан с **Pantone**, известной компанией в области подбора цветов и печати. **Справочный файл Pantone** в формате «.ink» обычно содержит информацию о цветах чернил Pantone; эти файлы используются в полиграфической промышленности для обеспечения точной цветопередачи и единообразия.

Компания Pantone известна своей **Системой соответствия Pantone (PMS)**, которая представляет собой стандартизированную систему воспроизведения цвета, широко используемую в различных отраслях, включая графический дизайн, печать и производство; PMS предоставляет набор цветов чернил, каждый из которых идентифицируется определенным кодом, например «PMS 185» для определенного оттенка красного.

## Что содержит файл INK?

В формате файла «.ink» могут храниться данные, относящиеся к цветам чернил Pantone, например:

1. **Цветовые коды:** цвета чернил Pantone идентифицируются уникальными кодами, которые обычно включаются в файл.
    



2. **Значения цвета:** Файл может содержать такие значения цвета, как **RGB (красный, зеленый, синий)**, **CMYK (голубой, пурпурный, желтый, ключевой/черный)** и **Lab. (CIELAB)** значения, указывающие, как цвет должен воспроизводиться в различных цветовых пространствах.
    



3. **Образцы цветов.** Цвета чернил Pantone часто имеют соответствующие образцы, которые дизайнеры и полиграфисты могут использовать для справки.
    



4. **Информация о цвете.** Дополнительная информация о цвете чернил, такая как его название, описание и указания по применению, может быть включена в файл «.ink».

## Как открыть файл INK?

Программы, открывающие файлы INK, включают в себя

- **CorelDRAW** (бесплатная пробная версия) для Windows

CorelDRAW — это программа векторной графики, поддерживающая цвета Pantone и предоставляющая инструменты для работы с ними.
