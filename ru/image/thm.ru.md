## .THM вариант №

Файл THM представляет собой эскиз изображения, автоматически создаваемый из первого кадра видеофайла на цифровых видеокамерах, камерах и смартфонах. Он генерируется при сохранении видеофайла камерой устройства. Миниатюры изображений для предварительного просмотра из всех видео отображаются пользователю для выбора интересующего видео из коллекции видео. Файлы THM сохраняются в памяти в сжатом двоичном формате для уменьшения занимаемого места. Файлы THM можно открывать с помощью таких приложений, как Apple Preview и [Adobe Photoshop 2022](https://www.adobe.com/products/photoshop.html).

## Формат файла THM — дополнительная информация

Файлы THM представляют собой растровые изображения, сохраняемые на диск в двоичном формате. Основное назначение файлов THM — идентификация видеофайла по миниатюре. В большинстве случаев файлы THM имеют то же имя, что и соответствующее имя видеофайла. Например, видеофайл с именем MOV001.mov будет иметь MOV001.THM в качестве имени файла эскиза. Расширение файлов THM можно переименовать в [JPEG](/ru/image/jpeg/), чтобы открывать их, если они не открываются напрямую двойным щелчком.

## Можно ли удалять файлы THM?

Файлы THM представляют собой эскизы видеозаписей, записанных видеокамерами. Их удаление не будет иметь никакого эффекта, поскольку исходное видео останется сохраненным.
