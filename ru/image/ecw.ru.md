## .ECW вариант № ##
Файл ECW состоит из изображения, предназначенного для использования с геопространственными данными. Содержащее изображение на самом деле является сжатым изображением, созданным в формате Enhanced Compression Wavelet. Файлы ECW обычно используются для хранения проекции спутниковой карты и аэрофотоснимков. Для поддержки геопространственных приложений информацию о картографической проекции можно встроить в файлы ECW.

## Формат файла ECW
ECW — это собственный формат изображения, основанный на вейвлет-сжатии; оптимизирован для спутниковых снимков и аэрофотосъемки. Этот формат в настоящее время принадлежит Intergraph, входящей в Hexagon AB, но в основном он был разработан Earth Resource Mapping. ECW — это формат сжатия с потерями, который сжимает изображения большого размера с хорошей переменной контрастностью, сохраняя при этом их визуальное качество.
 

### Характеристики
Вот некоторые типичные свойства формата ECW:
- Информация о проекции карты может быть добавлена в формат файла ECW для поддержки геопространственных приложений.
- Данные изображения, содержащие до 65 535 каналов, включая слои или цвета, могут быть сжаты в файл формата ECW со скоростью более 25 МБ в секунду.
- Протокол ECWP — это эффективный протокол потоковой передачи, используемый для передачи изображений ECW и JPEG2000 по сети.
- Быстрое SDK, поддерживающее ECW и JPEG2000, доступно бесплатно для настольной реализации для Linux, Windows и MacOSX.
