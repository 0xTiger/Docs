## .CLIP вариант №

Файл CLIP представляет собой файл растрового изображения, созданный приложением для 2D-рисования Clip Studio Paint. Он похож на файл Adobe [PSD](/ru/image/psd/) и содержит несколько слоев для управления различными графическими элементами в виде слоев, таких как текст, изображения, перо, карандаш, кисти и фигуры. Формат файла CLIP используется для разработки ряда различных типов графики, таких как иллюстрации и анимация. Приложение Clip Studio Paint разработано японской компанией CELSYS, Inc., основанной в мае 1991 года.

## Формат файла CLIP — дополнительная информация

Формат файла CLIP был представлен CELSYS для замены исходного формата файла LIP. Он хранит графические элементы в виде слоев, которые можно редактировать для внесения изменений. Это также формат файла по умолчанию для приложения [CLIP Studio Paint] (https://www.celsys.co.jp/en/service/studio_paint/), которое доступно для платформ Windows, MacOS, Android и iOS.

## Поддерживаемые типы чертежей

Формат файла CLIP может хранить анимацию и иллюстрации, которые можно использовать в различных типах работ.
