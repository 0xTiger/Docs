## .OLM вариант №

Файл с расширением .olm представляет собой файл Microsoft Outlook для операционной системы Mac. В файле OLM хранятся сообщения электронной почты, журналы, данные календаря и другие типы данных приложений. Они аналогичны файлам [PST](/ru/email/pst/), используемым Outlook в операционной системе Windows. Однако файлы OLM, созданные Outlook для Mac, нельзя открыть в Outlook для Windows. Формат файла OLM был представлен в Outlook 2011 для хранения локальных данных Outlook. Один файл OLM может содержать несколько почтовых ящиков, в которых данные организованы в виде папок и подпапок. Файлы OLM можно открывать/загружать в Microsoft Outlook 365 и Outlook для Mac.

## Формат файла OLM

OLM — это двоичные файлы с типом MIME Application/octet-stream. Технические характеристики формата файла OLM не доступны в Microsoft в открытом доступе, и сведения о формате отсутствуют для справки разработчика. Однако есть некоторые API, которые могут читать формат файла OLM и записывать его в файл PST для использования в операционной системе Microsoft Windows, такой как Aspose.Email.
