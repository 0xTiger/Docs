## .SDF вариант №

Файл пространственных данных (SDF) — это формат файла базы геоданных, разработанный Autodesk. Он хранит геопространственные данные и используется программой Autodesk GIS MapGuide и AutoCAD Map 3D. Формат файла SDF основан на формате файла базы данных с одним файлом SQLite3. Он считается оптимизированным форматом файла для больших наборов данных пространственной информации.

Вы можете открыть файл SDF с помощью Autodesk AutoCAD Map 3D 2022 и Autodesk AutoCAD Civil 3D 2023.

## Формат файла SDF — дополнительная информация

Формат файлов SDF сохраняется на диск в виде двоичных файлов. Он основан на базе данных SQLite, в которой используются низкоуровневые компоненты хранилища для двоичной сериализации данных. Файл SDF содержит несколько классов пространственных объектов на файл и несколько свойств геометрии для каждого класса пространственных объектов. Данные в файле SDF доступны с высокой скоростью благодаря индексации каждого свойства геометрии с использованием R-дерева.
