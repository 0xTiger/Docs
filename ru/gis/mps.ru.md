## .MPS вариант №

Файл MPS представляет собой файл карты, который используется на портативных карманных устройствах Pocket PC. Он содержит географическую информацию об определенной области, такой как город или штат. Его можно загрузить в устройства Pocket PC для навигации без необходимости получения данных в режиме реального времени с какого-либо сервера геопространственных данных. Формат файла MPS был разработан Microsoft для портативных устройств на базе Windows и использовался в ее ГИС-приложении MapPoint, поддержка которого давно прекращена.

## Формат файла MPS

Внутренняя файловая структура формата файлов MPS неизвестна, поскольку она уже давно прекращена с помощью MapPoint. Некоторые другие популярные форматы файлов карт ГИС включают [файлы форм ESRI](/ru/gis/shp/), [KML](/ru/gis/kml/) и [KMZ](/ru/gis/kmz/).
