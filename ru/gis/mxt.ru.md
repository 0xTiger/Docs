## .MXT вариант №

Файл MXT представляет собой шаблон географической карты, созданный с помощью картографического приложения ГИС ArcMap, которое входит в состав ESRI ArcGIS Desktop. Он используется для хранения слоев базовой карты или макетов страниц, которые можно повторно использовать для создания дополнительных карт. Использование шаблонов экономит ваше время, поскольку вам не нужно воспроизводить общие части карт. Файлами MXT можно делиться с другими пользователями/организациями, чтобы быстро создавать другие карты.

## Формат файла MXT

Файлы MXT сохраняются как двоичные файлы, но сведения об их внутренней файловой структуре недоступны для разработчиков. Он содержит такую информацию, как базовые свойства карты, положения компоновки и видимость панелей инструментов приложения. Файлы MXT позволяют пользователям создавать несколько карт с уже доступными предопределенными настройками.
