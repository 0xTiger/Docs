## .DT1 вариант №

Файл DT1 представляет собой файл цифровых данных о высоте местности (DTED), который содержит равномерно распределенные данные о местности. Он в основном используется военными программными приложениями ГИС для получения данных о высоте, уклоне и контурах поверхности. Файл DT1 аналогичен файлам [DT0](/ru/gis/dt0/) и [DT2](/ru/gis/dt2/), но со средней степенью детализации информации.

Файлы DT1 можно открывать с помощью таких ГИС-приложений, как ESRI ArcGIS Pro, TatukGIS, Blue Marble Geographic Global Mapper и Pitney Bowes MapInfo.

### Формат файла DT1

Файлы DT1 сохраняются как двоичные файлы на диск и могут быть прочитаны с помощью многих приложений ГИС. Их можно объединить в каталог растров файловой базы геоданных, который можно использовать для создания одной большой мозаики.
