## .IMG вариант №

Файл с расширением .img представляет собой файл карты Garmin, который используется GPS-устройствами Garmin. Он содержит картографические/геопространственные данные для представления объектов карты, таких как названия улиц, навигационные маршруты и достопримечательности. Пользователи могут создавать собственные картографические данные, сохранять их в виде файла .img и загружать на устройства Garmin. Кроме того, Garmin предоставляет предустановленные картографические данные, которые хранятся на устройствах Garmin в формате файла IMG. Файлы IMG можно открывать с помощью таких приложений, как Garmin Mobile PC, cGPSmapper, MapUpload и GPS Map Manager.

## Формат файла IMG — дополнительная информация

Файлы IMG сохраняются в виде двоичных файлов в собственном формате файлов Garmin. Информация, доступная в файле IMG, может отображаться на устройствах Garmin GPS. Некоторые устройства GPS, поддерживающие автомаршрутизацию, также могут использовать эти файлы карт для навигации во время навигации, например, в автомобиле, на велосипеде или во время пешего туризма.

Файлы GMAP аналогичны образам жесткого диска и основаны на файловой системе, которая состоит из подфайлов для обеспечения функциональности устройств GPS Garmin. Подфайлы .GMP представляют собой своего рода структуру подкаталогов, несмотря на то, что формат файла .gmap является плоским.
