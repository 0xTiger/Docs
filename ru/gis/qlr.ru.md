## .QLR вариант №

Файл с расширением .qlr — это файл определения слоя, который содержит указатель на источник данных слоя. Это в дополнение к информации стиля [QGIS](https://www.qgis.org/en/site/) для слоя. Имея ссылки на все связанные стили, этот файл используется как единый источник данных для получения информации о стилях. С помощью файлов QLR информация об источниках данных может быть помещена в этот единственный файл, который может быть прочитан другими приложениями для загрузки информации о стилях. Файлы QLR можно открыть в любом текстовом редакторе, например Notepad++.

## Формат файла QLR

QLR, как и [QGS](/ru/gis/qgs/), представляет собой файл XML, который содержит указатели на источник данных слоя в виде тегов XML. Он похож на файл ArcGIS .lyr. Теги верхнего уровня файла QML показаны на изображении ниже.
