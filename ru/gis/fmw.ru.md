## .FMW вариант №

Файл FMW представляет собой файл проекта, созданный с помощью программного обеспечения FME Workbench (входит в состав пакета FME Desktop), который используется для преобразования пространственных данных. Он содержит пользовательские настройки, используемые для манипулирования пространственными данными, такие как наборы входных данных, свойства перевода, проекции и настройки вывода. Настройки обработки пространственных данных сохраняются в виде визуального макета, и их легко редактировать и сохранять снова.

Вы можете открывать файлы FMW с помощью [Safe Software FME Desktop](https://www.safe.com/fme/fme-desktop/).

## Формат файла FMW — дополнительная информация

Файлы FMW хранятся на диске в виде двоичных файлов и могут быть прочитаны только с помощью программного обеспечения FME Desktop. Программное обеспечение также может считывать данные из ряда форматов реляционных баз данных, а также поддерживает ряд форматов данных.
