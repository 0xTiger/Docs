## .QGS вариант №

Файл с расширением .qgs представляет собой файл проекта, созданный с помощью [QGIS] (https://www.qgis.org/en/site/), кроссплатформенного ГИС-приложения с открытым исходным кодом. Все настройки проекта, такие как свойства слоя и вспомогательные данные для проекта. Он создается при сохранении проекта карты на диск. На самом деле это файл конфигурации, в котором хранится такая информация, как указатели на данные ГИС, информация о стилях слоев и название проекта. Файлы QGS можно открыть с помощью бесплатного программного обеспечения QGIS.

## Формат файла QGS

Файлы QGS имеют формат [XML](/ru/web/xml/) и хранят данные проектов в виде тегов XML. Файл QGS содержит такую информацию, как:

* Название Проекта
* проект АСБ
* дерево слоев
* настройки привязки
* связи
* экстент холста карты
* модели проектов
* легенда
* доки карт (2D и 3D)
* Слои со ссылками на базовые наборы данных (источники данных) и другие свойства слоя, включая экстент, SRS, соединения, стили, средство визуализации, режим наложения, непрозрачность и многое другое.
* свойства проекта
