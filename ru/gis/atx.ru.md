## .ATX вариант №

Файл с расширением .atx представляет собой файл ESRI ArcGIS, в котором хранятся индексы атрибутов ArcGIS 8. Для каждого [shapefile](/ru/gis/shp/) или индекса атрибутов dBase, созданного в ArcCatalog, существует отдельный файл ATX. Следует отметить, что индексы атрибутов ArcView GIS 3.x для шейп-файлов и файлов dBase не используются в ArcMap или ArcCatalog. Однако эти индексы атрибутов ArcView GIS 3.x можно преобразовать в индексы атрибутов ArcGIS 8 с помощью инструмента ArcObjects.

Спецификации формата файлов для формата файлов ATX отсутствуют для облегчения работы разработчиков.
