## .OSC вариант №

Файл OSC представляет собой файл изменения формата, который содержит измененные данные карты улиц для существующей карты улиц .osm. Он содержит информацию об изменениях, внесенных в [файл OSM](/ru/gis/osm/). Файл OSC может иметь три возможных типа операций модификации данных OSM, т. е. «вставка», «модификация» или «удаление». Эти файлы изменений обычно используются для экспорта данных из редактора OSM и импорта в другой редактор.

Вы можете открывать файлы OSC с помощью программного обеспечения Merkaartar и osmchange.

## Формат файла OSC

Файлы OSC обычно сохраняются в формате файлов JOSM, где изменения представлены тегами. Он начинается с тега osmChange, указывающего на начало файла. Вложенные теги указывают, является ли это операцией вставки, изменения или удаления, которая должна выполняться на соответствующем узле в файле OSM. Содержимое узла фактически содержит содержимое тега osm.

### Пример формата файла OSC

Ниже приведен пример операции изменения узла. Каждый элемент должен иметь идентификатор набора изменений и номер версии.

```
<osmChange version="0.6" generator="acme osm editor">
    <modify>
        <node id="1234" changeset="42" version="2" lat="12.1234567" lon="-8.7654321">
            <tag k="amenity" v="school"/>
        </node>
    </modify>
</osmChange>
```
