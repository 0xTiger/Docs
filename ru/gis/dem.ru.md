## .DEM вариант №

Файл с расширением .dem представляет собой файл сопоставления, который представляет данные о высоте (высоте) в виде ландшафта в трехмерной компьютерной графике. Распространенным примером таких данных является рельеф земли, включающий горы, равнину и морские глубины. Файлы DEM обычно используются географическими информационными системами (ГИС) для создания цифровых карт высот. Информация, хранящаяся в файлах DEM, также включает крупномасштабную информацию о топографическом рельефе, которая используется в науках о Земле, управлении ресурсами, планировании земель, геодезии и инженерных проектах. Информация о земле XYZ размещается через равные промежутки сетки для представления рельефа местности.

## Формат файла DEM — дополнительная информация

Внутренние сведения о файлах DEM недоступны для разработчиков. Однако файлы DEM могут иметь разные форматы файлов, например:

* Батиметрическая атрибутивная сетка (BAG)
* ДТЭД
* База данных ESA DIMAP Sentinel 1
* СДТС ЦМР
* USGS ЦМР

## Где найти файлы DEM?

Файлы DEM обычно доступны в Интернете, и образцы можно загрузить для использования. [Геологическая служба США](https://www.usgs.gov/) (USGS) предоставляет множество файлов DEM на своем веб-сайте.
