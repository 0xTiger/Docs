## .MSD вариант №

Файл MSD — это файл распространения картографического сервиса, созданный с помощью картографического программного обеспечения ГИС ArcGIS Desktop. Он содержит опубликованную карту, экспортированную из файла карты [.mxd](/ru/gis/mxd/). Файл MSD содержит информацию о картографическом сервисе, размещенном на ArcGIS Server. Размещенная карта доступна через Интернет для других. Файл MSD содержит документ карты, информацию о слое и свойства таблицы.

## Формат файла MSD

Файлы MSD сохраняются на диск в двоичном формате. Они используются серверным приложением ArcGIS для создания картографического сервиса и управления им в целях хостинга. Файлы MSD также используются ArcMap для подключения к картографическому сервису и отображения содержимого на карте.
