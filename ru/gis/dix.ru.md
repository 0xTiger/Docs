## .DIX вариант №

Файл DIX — это собственный формат файла, созданный с помощью программы анализа географических данных DIVA-GIS. Он состоит из файла проекта DIVA-GIS (.div) и других слоев данных ГИС, на которые ссылается проект. Файлы DIX можно экспортировать из DIVA-GIS и поделиться ими с другими пользователями, которые смогут импортировать их в DIVA. Вы также можете использовать файлы DIX для хранения всех соответствующих файлов, связанных с проектом, в одном месте.

## Формат файла DIX

Файл DIX сохраняется на диске в виде сжатого двоичного файла. Благодаря сжатию файлы DIX не занимают много места на диске и могут быть отправлены другим пользователям по электронной почте. Файл DIX может содержать как растровые, так и векторные данные, а также хранить таблицы атрибутов, метаданные и другие данные, связанные с пространственными данными.
