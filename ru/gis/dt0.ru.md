## .DT0 вариант №

Файл DT0 представляет собой файл цифровых данных о высоте местности (DTED), который содержит информацию уровня 0 для изучения данных о высоте местности. Данные о высоте равномерно распределены в файле DT0 и могут быть прочитаны популярными ГИС-программами, такими как ESRI ArcGIS Pro, TatukGIS, Blue Marble Geographic Global Mapper и Pitney Bowes MapInfo. Файлы DT0 в основном используются научными и техническими сообществами для изучения наклона, высоты и шероховатости поверхности глобальных ландшафтов.

Формат файла DT0 аналогичен типам файлов [DT1](/ru/gis/dt1/) и [DT2](/ru/gis/dt2/), но с меньшей степенью детализации.

### Формат файла DT0

Файлы DT0 сохраняются как двоичные файлы на диск. Их можно прочитать с помощью многих приложений ГИС. Файлы DT0 можно объединить в каталог растров файловой базы геоданных, который можно использовать для создания одной большой мозаики.

## Как преобразовать растр в файлы DTED?

Существует несколько инструментов, которые могут конвертировать растровые файлы в файлы DTED. [ArgGIS Pro](https://pro.arcgis.com/en/pro-app/latest/tool-reference/data-management/raster-to-dted.htm) — один из таких инструментов, с помощью которого можно преобразовать растр в DTED. .
