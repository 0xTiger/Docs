## .AVL вариант №

Файл AVL представляет собой файл легенды, который содержит ключ для карты, созданной с помощью программного обеспечения ESRI ArcView GIS (географическая информационная система). Он содержит справочную информацию о символах, используемую в соответствующей карте для доступа. Файл AVL может содержать дополнительную информацию, такую как символы, настройки отображения, такие как прозрачность, свойства отображения, зависящие от масштаба, информацию о соединенной таблице/связанной таблице, запрос определения, свойства маркировки для векторных слоев и свойства отображения аннотаций для слоев аннотаций в зависимости от типа. вид слоя.

На файл AVL можно ссылаться из файла проекта ArcView [.APR](/ru/gis/apr/).

## Формат файла AVL — дополнительная информация

Файлы AVL сохраняются в формате обычного текстового файла. Это означает, что вы можете открывать файлы AVL в любом текстовом редакторе, таком как Notepad, Notepad++ и Apple TextEdit. После открытия вы можете не только просматривать содержимое файла, но и изменять его.

### Разница между файлами .LYR и .AVL

* **Разница в форматах файлов** — .lyr — это двоичный файл, а .avl — текстовый файл.
* **Разница в содержании** — файл .lyr содержит больше информации, чем файл .avl. В файле AVL хранится только информация о символах, а в файле LYR хранится вся информация, доступная в диалоговом окне свойств слоя в ArcMap.
