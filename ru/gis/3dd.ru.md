## 3DD вариант №

Файл с расширением .3dd представляет собой файл данных ГИС, созданный с помощью прикладного программного обеспечения ESRI ArcGlobe. Он хранит информацию о путях, где находятся 3D-данные в виде базы геоданных. Когда документ открывается с помощью файла .3dd, он отображается в том же виде, в каком был сохранен. ArcGlobe — это приложение для 3D-визуализации, позволяющее просматривать большие объемы данных ГИС на поверхности земного шара. ArcGlobe — одно из нескольких приложений, связанных вместе с программным пакетом ArcGIS.

## Формат файла 3DD — дополнительная информация

Файлы 3DD хранятся на диске в двоичном формате, и внутренний формат этих файлов общедоступен.
