## .NMC вариант №

Файл NMC представляет собой файл содержимого пакета карт, созданный с помощью программного обеспечения ESRI ArcGIS Explorer. Он может хранить различные элементы карты, такие как наложения изображений, сервисные слои ГИС, слои [KML](/ru/gis/kml/), растровые слои, объекты и другие пакеты. Файлы NCM используются для экспорта картографических данных, которые можно использовать в других картах ГИС, открытых в ArcGIS Explorer. Это позволяет нескольким картам накладываться друг на друга и отображаться.

## Формат файла NMC

Файлы NMC сохраняются в двоичном формате, внутренняя структура которых недоступна. Им можно поделиться с другими пользователями, которые могут загрузить эти файлы NMC в ArcGIS Explorer на своем конце. Это позволяет использовать файлы NMC на компьютерах других пользователей, отправляя их другим по электронной почте.
