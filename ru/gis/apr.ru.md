## .APR вариант №

Файл с расширением .apr представляет собой файл проекта ESRI ArcView, который содержит информацию о проекте ESRI. Они использовались ArcView 3.x и не поддерживаются новыми (9.x и 10.x) версиями ArcGIS. Однако ArcView 10 может импортировать файлы APR. ArcView — это географическая информационная система (ГИС), которая используется для создания, визуализации, анализа и управления географическими данными. Существуют и другие форматы файлов, используемые ESRI для хранения данных, такие как популярный ESRI ShapeFile ([.shp](/ru/gis/shp/)). Файлы APR можно открыть с помощью любого текстового редактора, например Microsoft Notepad и Notepad++.

## Формат файла APR — дополнительная информация

Файлы APR хранятся на диске в виде простых текстовых файлов, которые можно открывать в любом текстовом редакторе для изучения и редактирования.
