## .ETA вариант №

Файл с расширением .eta представляет собой файл метки Google Планета Земля, созданный с помощью программного обеспечения Google Планета Земля. Он содержит информацию о закладках/маркерах для местоположения в мире с использованием таких параметров, как широта, долгота, вращение, масштабирование и название. Это позволяет пользователям видеть местоположение с высоты птичьего полета. С введением новых форматов файлов, т.е. [.kml](/ru/gis/kml/) и [.kmz](/ru/gis/kmz/), формат файлов ETA был постепенно упразднен и заменен этими новыми форматами файлов. Файлы ETA можно открыть в программном обеспечении [Google Earth](https://earth.google.com/web/), которое можно загрузить бесплатно.


## Формат файла ETA — дополнительная информация

Внутренние сведения о формате файла ETA не доступны публично, и нет открытого или проприетарного программного обеспечения, доступного для чтения информационных файлов ETA.
