## .GLB вариант №

Файл с расширением .glb представляет собой файл с общим доступом, созданный и используемый программным обеспечением AGI Systems Tool Kit (STK). Он содержит информацию о трехмерном глобусе, которой можно поделиться с другими пользователями для использования в своих проектах. Программное обеспечение STK от AGI systems используется для моделирования аэрокосмических миссий, систем защиты и телекоммуникационных систем. GLB является стандартным файловым форматом для STK, и многие различные 3D-модели, такие как [.gltf](/ru/3d/gltf/), могут быть преобразованы в этот формат. Файлы GLB можно открывать с помощью программного обеспечения AGI Simulation Toolkit, а также любого текстового редактора.

## Формат файла GLB — дополнительная информация

Файлы GLB хранят информацию о текстурах, изображениях и других файлах, которые используются в глобусе для проекта. Ссылки на расположение связанных файлов хранятся в файлах GLB, и эти дополнительные файлы также должны быть доступны другим пользователям при совместном использовании файла глобуса GLB для правильной загрузки. Файлы GLB хранят всю эту информацию в виде простых текстовых файлов, которые можно открыть в любом текстовом редакторе.
