## .DT2 вариант №

Файл DT2 представляет собой файл высот, представляющий файл цифровых данных высот местности (DTED). Он содержит данные о высоте в равномерно распределенных шаблонах. Данные о высоте могут считываться программным обеспечением ГИС и находят свое применение в различных областях, таких как военные, научные и другие сообщества.

ГИС-приложения, которые могут открывать файлы DT2, включают ESRI ArcGIS Pro, TatukGIS, Blue Marble Geographic Global Mapper и Pitney Bowes MapInfo. Другими вариантами типов файлов DTED являются [DT0](/ru/gis/dt0/) и [DT1](/ru/gis/dt1/).

### Формат файла DT2

Файлы DT2 сохраняются как двоичные файлы на диск и могут быть прочитаны с помощью многих приложений ГИС. Их можно объединить в каталог растров файловой базы геоданных, который можно использовать для создания одной большой мозаики.
