## .TCX вариант №

Файл TCX (Training Center XML) — это формат обмена данными, используемый для обмена данными между фитнес-устройствами. Он был представлен в 2008 году вместе с продуктом Garmin Training Center. Данные о тренировке, такие как частота сердечных сокращений, темп бега, частота вращения велосипеда, калории и время круга, хранятся в формате [XML](/ru/web/xml/) внутри файла TCX. Кроме того, сводные данные о треке тренировки также включаются в файл TCX. Файлы TCX аналогичны файлам [FIT](/ru/gis/fit/), которые создаются носимыми спортивными устройствами Garmin.

## Формат файла TCX — дополнительная информация

Файлы TCX сохраняются на диск в виде файлов XML, при этом каждая запись сохраняется как действие. Активность включает в себя все данные тренировки, такие как время, время круга, идентификатор, частота сердечных сокращений, интенсивность, частота вращения педалей и информация о треке, которая содержит пары позиций вместе с отметкой времени для этой широты и долготы, аналогично [GPX](/ru/gis/gpx/).

### Версии формата файла TCX

Существует две версии этого формата с собственными XML-схемами, размещенными в Garmin. Ниже приведены некоторые из них:

* https://www8.garmin.com/xmlschemas/TrainingCenterDatabasev2.xsd
* https://www8.garmin.com/xmlschemas/UserProfileExtensionv1.xsd
* https://www8.garmin.com/xmlschemas/ActivityExtensionv2.xsd


## Протокол данных TCX

Быстрая версия формата TCX XML доступна на Github как [TcxDataProtocol](https://github.com/FitnessKit/TcxDataProtocol).
