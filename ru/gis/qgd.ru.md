## .QGD вариант №

Файл с расширением .QGD на самом деле является связанной базой данных sqlite проекта **QGIS**, которая содержит вспомогательные данные для проекта. Файл QGD останется пустым, если не будет вспомогательных данных для проекта.

## Подробная информация о формате файла QGD

В классическом режиме вспомогательная база данных сохраняется в виде файла с расширением .qgd вместе с файлом xml. Система **Вспомогательная база данных** позволяет читать и записывать данные в систему альтернативным способом. При создании QGZ, который представляет собой заархивированный контейнер, файл .qgd включается в .qgz.

## Что такое вспомогательная база данных?
Вспомогательная база данных фактически является резервной системой базы данных, созданной в ответ на дублирование целевой базы данных. Вспомогательная база данных на самом деле является случаем дублирующей базы данных, которая будет базой данных, в которую вы дублируете. Например, если вы настраиваете резервную базу данных, используя дублирующую базу данных, исходная база данных будет целевой, а резервная база данных будет известна как вспомогательная.
