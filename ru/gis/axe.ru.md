## .AX вариант №

Файл AX — это формат файла ГИС, созданный с помощью приложения маршрутизации Microsoft AutoRoute. Он содержит информацию о местоположении точек интереса (POI) на карте и другие навигационные данные. Эти POI просты в навигации и могут быть добавлены в путешествие как часть маршрута. AutoRoute был разработан для предоставления информации о маршрутах для навигации по дорогам и автомагистралям Европы.

Файлы AX можно открыть с помощью Microsoft AutoRoute, но его поддержка уже давно прекращена, и файлы AX больше не поддерживаются.

## Формат файла AX – дополнительная информация

Файлы AX сохраняются на диск в двоичном формате. Он сохраняется в собственном формате файла, и сведения о его внутренней файловой структуре недоступны публично для справки разработчика.
