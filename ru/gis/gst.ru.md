## .GST вариант №

Файл GST представляет собой формат файла карты ГИС, который содержит географические данные и визуальную карту нанесенной на карту области. Он используется программным обеспечением **MapX** (разработанным UNEP/GRID-Geneva и UNEP для управления геопространственными данными о природных ресурсах) и картографическим программным обеспечением **MapXtreme** (набор инструментов разработки программного обеспечения, используемый для создания пользовательских ГИС-приложений), которые онлайн-платформы для управления геопространственными данными о природных ресурсах. Файл GST содержит географическую информацию в виде числовых и растровых изображений, которые накладываются на карту.

## Формат файла GST — дополнительная информация

Файлы GST сохраняются в проприетарном формате файлов, и сведения о его внутреннем формате файлов недоступны для разработчиков.

## Использованная литература ##

* [MAPX](https://www.mapx.org/)
* [MapXtreme](https://mappingsolutions.com/mapxtreme/)
