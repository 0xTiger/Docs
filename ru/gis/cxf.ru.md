## .CXF вариант №

Файл CXF — это файл ГИС, сохраненный в формате Coordinates eXport Format (CXF), разработанный [HeightMap.org](http://heightmap.org/). Он содержит данные векторного изображения в упорядоченном виде и может состоять из данных ГИС или векторной графики, например рисунков.

## Формат файла CXF — дополнительная информация

Файл CXF сохраняется как обычный текстовый файл и содержит координаты векторной карты (VMAP) для представления географических местоположений. Некоторые данные векторной карты, которые файл CXF может включать в себя данные о местности, топографическую информацию, ориентиры и здания. Файлы CXF можно экспортировать в несколько различных форматов файлов ГИС, таких как .AI, [.XAML](/ru/web/xaml/), [.SVG](/ru/page-description-language/svg/), .EMF, [.KML ](/ru/gis/kml/) и [HTML5](/ru/web/html/).
