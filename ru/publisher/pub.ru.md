## .PUB вариант №

Файл PUB — это формат файла документа Microsoft Publisher. Он используется для создания нескольких типов макетов документов, таких как информационные бюллетени, листовки, брошюры, открытки и т. д. Файлы PUB могут содержать текст, растровые и векторные изображения. Документы, созданные с помощью файлов PUB, в основном используются на веб-сайтах и в маркетинговых материалах, таких как электронные письма. Файлы PUB можно открыть с помощью приложения Microsoft Publisher Desktop, [Microsoft Publisher 365] (https://www.microsoft.com/en/microsoft-365), [LibreOffice Draw] (https://www.libreoffice.org/discover /draw/) и [Adobe InDesign](https://www.adobe.com/products/indesign.html).

## Формат файла PUB — дополнительная информация

Файлы PUB хранятся как двоичные файлы на диске. Содержимое файлов PUB также можно преобразовать в формат файла [DOCX](/ru/word-processing/docx/) из приложения Microsoft Publisher с помощью параметра «Сохранить как».
