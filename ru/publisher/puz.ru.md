## .PUZ вариант №

Файл с расширением .puz представляет собой сжатый архив, содержащий стандартный документ Microsoft Publisher [.pub](/ru/publisher/pub/). В результате сжатия он меньше по размеру и его легко передавать через Интернет или другие устройства. Файлы PUZ можно распаковать/открыть с помощью утилиты Microsoft Unpack.

## Формат файла PUZ

Файл PUZ представляет собой сжатый архив, содержащий стандартный файл PUB вместе с другими файлами. Он содержит все необходимые файлы для распаковки, отображения и печати публикации.

Файл PUZ содержит следующее.

* `Unpack.exe` - программная утилита для распаковки сжатых файлов
* `ReadMe.txt` - описание файлов
* «Сжатый файл .puz» — содержит публикацию и может включать встроенные шрифты и связанную графику.

### Распаковка файла PUZ

Упакованные файлы Publisher можно распаковать с помощью утилиты Unpack.exe. Он извлекает упакованный файл PUZ в каталог, выбранный пользователем.
