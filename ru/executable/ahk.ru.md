## .AHK вариант №

Файл AHK представляет собой файл сценария, созданный с помощью программного приложения AutoHotkey, которое используется для автоматизации задач в Microsoft Windows. Он содержит инструкции по автоматизации задач с использованием сочетаний клавиш, которые можно использовать для выполнения мгновенных действий. Файл запускается AutoHotkey и все действия выполняются последовательно. Файлы AHK достаточно мощны, чтобы выполнять сложные задачи с помощью горячих клавиш, определенных с помощью горячих клавиш. Файлы AHK можно открывать с помощью текстовых редакторов, таких как Microsoft Notepad и Notepad++.

## Формат файла AHK

Файлы AHK сохраняются на диск в виде простых текстовых файлов и содержат строки кода, которые могут быть выполнены с помощью AutoHotkey. AutoHotkey — это бесплатный язык сценариев с открытым исходным кодом, который позволяет пользователям создавать простые и сложные сценарии для выполнения таких действий, как заполнение форм, автоматический щелчок, выполнение макросов и т. д. Как минимум файл AHK может выполнять одно действие, а затем выходить. .

Существуют инструменты, которые можно использовать для преобразования AHK в файлы [Exe](/ru/executable/exe/).

### Пример АХК

В следующем примере используется клавиша Win+Z для запуска Блокнота Microsoft или его перемещения на передний план, если он уже запущен.

```
#z::Run https://www.autohotkey.com  ; Win+Z

^!n::  ; Ctrl+Alt+N
if WinExist("Untitled - Notepad")
    WinActivate
else
    Run Notepad
return
```

## Как создать файл AHK?

Следующие шаги можно использовать для создания файла AHK. Предполагается, что AutoHotkey уже установлен на машине.

* Щелкните правой кнопкой мыши на рабочем столе.
* Найдите «Новый» в меню.
* Нажмите «Сценарий AutoHotkey» в меню «Создать».
* Дайте скрипту новое имя.
* Найдите вновь созданный файл на рабочем столе и щелкните его правой кнопкой мыши.
* Нажмите «Редактировать сценарий».
* Должно было появиться окно, возможно Блокнот.
* Сохраните файл.
