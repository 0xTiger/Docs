## .RUN вариант №

Формат файла .run обычно используется для распространения установщиков программного обеспечения или приложений в среде Linux. Чтобы установить программное обеспечение, вам необходимо сделать файл исполняемым, что можно сделать с помощью следующей команды:

```bash
chmod +x file_name.run 
```

Затем вы можете запустить файл с помощью следующей команды:

```bash
./file_name.run 
```

Процесс установки может различаться в зависимости от конкретного файла и программы, которую вы пытаетесь установить.

Формат файла .run — это тип сценария оболочки, используемый для распространения установщиков программного обеспечения или приложений в среде Linux. Это автономный пакет, включающий все необходимое для установки программного обеспечения, включая двоичные файлы, библиотеки и файлы конфигурации.

Важно отметить, что файлы .run также могут содержать вредоносный код, поэтому всегда полезно проверить источник файла и проверить его на наличие вирусов перед его запуском.

Кроме того, для запуска и установки некоторых файлов .run могут потребоваться права root, поэтому вам может потребоваться использовать команду «sudo» для запуска файла с повышенными разрешениями:

```bash
sudo ./filename.run
```

## Как открыть файл RUN?

Чтобы открыть файл .run, вам необходимо сделать его исполняемым, что можно сделать с помощью команды «chmod»:

```bash
chmod +x filename.run 
```

Как только файл станет исполняемым, вы можете запустить его, набрав:

```bash
./filename.run
```

Запуск файла .run — это не то же самое, что открытие его в текстовом редакторе. Запуск файла .run приведет к выполнению его инструкций, которые могут быть чем угодно: от установки программы до запуска сценария. Чтобы просмотреть содержимое файла .run, вам необходимо открыть его в текстовом редакторе, например nano или vim:

```
nano filename.run
```
или
```
vim filename.run
```
