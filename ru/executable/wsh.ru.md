## .WSH вариант №
Файл с расширением .wsh содержит свойства и параметры для определенного сценария языка программирования, такого как VB или VBS и т. д. Фактическая потребность WSH состоит в том, чтобы использовать их для настройки выполнения определенных сценариев. Для работы требуются WScript или CScript, и оба они включены в операционную систему Windows. Файлы WSH изначально поставлялись в Windows 95 на установочных дисках в качестве дополнительных настраиваемых и устанавливаемых для панели управления, а затем в качестве компонента по умолчанию в Windows 98.

## WSH формат файла
WSH (Windows Script Host) может использоваться для различных целей, включая администрирование, сценарии входа в систему и общую автоматизацию. WSH устанавливает среду для запуска сценариев. он вызывает подходящий обработчик сценариев и выделяет набор служб и объектов для работы сценария. Эти сценарии можно запускать в режиме графического интерфейса, из COM-объекта или в режиме командной строки, предоставляя пользователю гибкость как для интерактивных, так и для неинтерактивных сценариев.

### Примеры
Вот очень простой пример, который показывает некоторый VBScript, который использует корневой COM-объект WSH «WScript» для отображения сообщения с кнопкой «ОК». Когда этот сценарий будет запущен, будет вызван механизм CScript или WScript, и будет установлена среда выполнения.

```
WScript.Echo "Hello world"
WScript.Quit
```
