## .BTAPP вариант №

Файл BTAPP — это приложение, разработанное для программы загрузки TORRENT uTorrent. По сути, это приложение для uTorrent, которое может получить доступ ко всем функциям клиента uTorrent. Эти приложения можно добавить в клиент uTorrent из самого приложения. Это установит приложение в клиенте uTorrent и обеспечит все функции загрузки файлов [TORRENT](/ru/misc/torrent/).

## Формат файла BTAPP — дополнительная информация

Файл приложения BTAPP сохраняется как двоичный файл. Он содержит код [HTML](/ru/web/html/) и [JavaScript](/ru/web/js/), который можно открыть во встроенном браузере.
