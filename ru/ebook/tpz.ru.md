## .TPZ вариант № ##

Расширение файла .tpz также известно как формат электронных книг Kindle Topaz от Amazon. Linux, Mac OS и Windows — это некоторые из платформ, на которых могут распространяться файлы с расширением .tpz. Как и файл электронной книги Kindle Topaz, файл TPZ отформатирован так, чтобы выглядеть как традиционные электронные книги, и содержит проприетарные стандарты компиляции данных от Amazon для формата TPZ. Этот файл имеет расширенную поддержку встроенных шрифтов, а дополнительные сведения, такие как аннотации и закладки, также могут быть включены в файлы электронных книг Kindle Topaz, хотя эти сведения о метаданных хранятся отдельно. Файл TPZ также можно отнести к категории файлов данных, которые можно загрузить из онлайн-базы данных Amazon электронных книг Kindle.

## Проблемы с открытием файла TPZ? ##

Вот некоторые из общих проблем, упомянутых ниже, которые могут возникнуть:

* Зараженный файл. Это можно исправить с помощью соответствующего антивирусного программного обеспечения, чтобы сделать его чистым.
* Недостаточно места на диске. Для этого необходимо закрыть все неактуальные приложения для лучшей работы
* Неработающие ссылки иногда могут быть причиной проблем
* Иногда файл не имеет прав доступа, поэтому в этом случае убедитесь, что права доступа легко открывают файл
