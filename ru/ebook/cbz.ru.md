## .CBZ вариант №

Файл с расширением .cbz представляет собой архивный файл комиксов [ZIP](/ru/compression/zip/), который представляет собой набор изображений, представляющих страницы комиксов. CBZ отличается от других файлов электронных книг тем, что они не сжаты. Большинство программ для чтения электронных книг и комиксов поддерживают просмотр этих файлов. [CBR](/ru/ebook/cbr/) — это еще один формат файла архива комиксов, который использует сжатие [RAR](/ru/compression/rar/) вместо ZIP. Файлы CBZ и CBR могут содержаться в файлах коллекции комиксов, таких как [CBC](/ru/ebook/cbc/). Приложения, которые могут открывать и читать файлы CBZ, включают программные приложения, такие как Caliber и Comic Book Reader.

## Формат файла CBZ

Файл CBZ представляет собой ZIP-архив, содержащий страницы электронных книг в виде изображений в форматах файлов [PNG](/ru/image/png/), [JPEG](/ru/image/jpeg/) или [GIF](/ru/image/gif/). Его можно распаковать, а его содержимое можно извлечь с помощью любого программного обеспечения для распаковки, которое поддерживает формат файла ZIP. Изображения, извлеченные с помощью программного обеспечения для декомпрессии, расположены не в том порядке, в котором они появляются в файле CBZ, поскольку информация о порядке изображений находится внутри файла CBZ.
