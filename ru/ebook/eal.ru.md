## .EAL вариант № ##

Файлы **Kindle End Action** обычно имеют расширение .EAL. Этот файл состоит из xml, который написан, чтобы показать что-то об авторе после последней страницы электронной книги.

## Формат файла EAL ##

Файл .EAL представляет собой XML-файл, который состоит из данных о покупателях, купивших книгу, а также купивших функцию, чтобы показать что-то еще об авторе электронной книги после последней страницы книги; ASIN также включен для перехода на страницу Kindle Store.
