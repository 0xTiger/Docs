## .TCR вариант №

В начале 90-х файловый формат TCR был введен для древних карманных устройств **Psion Series 3**. Компания использовала сжатие для конкретной платформы, и этот формат имеет очень ограниченную поддержку форматирования текста. Это формат файла на основе ZVR, который обеспечивает сравнительно лучшую степень сжатия, чем PalmDOC. Он также очень хорошо работает с FontMachine. Хотя формат файла TCR принадлежит устройству, выпуск которого прекращен, этот формат файла все же можно прочитать с помощью некоторых электронных книг. Этот формат файла также можно просматривать на рабочем столе с помощью Sumatra PDF и с помощью приложения Calibre в Windows и Linux.

## Технические подробности

Поскольку формат файла TCR основан на компрессоре ZVR, который способен уменьшить многие файлы примерно на 50% от их исходного размера. Схема сжатия, используемая over, немного устарела. Сжатый файл начинается со словаря из 256 возможных символов, которые могут отображаться в сжатом файле. Словарь содержит фиксированную альтернативную строку для каждого из этих символов. Распаковка стала очень быстрой даже в OPL, и она может начаться в любой точке сжатого файла.

## Проблемы с открытием файла TCR ##

Если вы не можете открыть и запустить файл TCR; это не означает, что на вашем устройстве не установлено подходящее программное обеспечение. Могут быть некоторые другие проблемы, которые мешают правильной работе файла. Возможные проблемы могут быть одной из следующих:

- Повреждение файла TCR.
- Ошибочные связи файла TCR в записях реестра.
- Удалено описание TCR из реестра Windows.
- Поврежденная установка приложения, поддерживающего формат TCR
- Зараженный файл TCR с нежелательным вредоносным ПО.
- На компьютере недостаточно аппаратных ресурсов для работы с файлом TCR.
- Драйверы, используемые компьютером для открытия файла TCR, устарели.
