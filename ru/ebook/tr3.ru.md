## .TR3 вариант № ##

Файл TR3 представляет собой электронную книгу TomeRaider 3, которая поддерживает быстрый поиск и сжатие. Это может работать должным образом через программу, связанную с TomeRaider. Yadabyte является разработчиком этого программного обеспечения, британской компанией по разработке программного обеспечения и веб-сайтов, базирующейся в Великобритании. Приложение для чтения электронных книг TomeRaider можно использовать для правильного функционирования и управления содержимым этих файлов TR3. Это родственное устройство для чтения электронных книг можно установить на компьютеры, работающие под управлением систем Microsoft Windows, и на множество переносимых гаджетов. Файл TR3 принадлежит к группе файлов электронных книг, используемых в таких операционных системах, как Windows 10, Windows 7, Windows 8/8.1, Windows Vista, Windows XP.

Специальные **теги HTML** для TR3:

|Тег|Описание|
---|---|
|\<new> |Для создания файлов TomeRaider из текстовых файлов достаточно тега New.<new> тег определяет начало страниц.|
|\<CATDEF> ... \</CATDEF> |определяет имя категории|
|\<META> ... \</META> |это тег, используемый для определения всех метаданных, используемых в формате файла. Этот тег включает ряд параметров.|
|\<EXPMSG> |Этот тег управляет сообщением, которое появляется, когда срок действия файла истек. Всякий раз, когда пользователь пытается просмотреть страницу после истечения срока действия файла, вместо фактического текста страницы появляется сообщение об истечении срока действия.|
|\<DIEMSG> |Этот тег аналогичен EXPMSG. Он используется для отображения сообщения, когда файл умирает.|
|\<ENGMSG> |Этот тег используется для создания сообщения, отображаемого для зашифрованных страниц. Сообщение появляется вместо текста страницы, если пользователь пытается открыть зашифрованную страницу до ее разблокировки.|
|\<expmsg> ,\<diemsg> а также \<encmsg> |Игнорируется в тексте страницы. Они должны быть помещены в раздел заголовка файла перед первым<new> тег.|
|\<SELECTION> . \</ SELECTION> |Используется для поддержки запросов. Этот тег должен стоять перед первым<new> ярлык. Он компилирует данные выбора для пользовательского запроса.|
|\<catset> . \</catset> | Используется для назначения имен категорий для каждого предмета.|


## Проблемы с открытием файла TR3 ##

Вот список некоторых распространенных проблем, которые могут возникнуть и привести к неправильной работе формата файла:

* Поврежденный файл
* Зараженный файл из-за вируса
* Неправильные ссылки на файл TR3 в доступе к архиву
* Нет прав доступа в системе для открытия файлов
* Устаревший диск в вашей системе
* Удаление отчета TR3 из архива Windows
* Обращение к разработчику может быть лучшим вариантом для лучшего вывода
