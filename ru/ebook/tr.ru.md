## .TR вариант № ##

Файл с расширением .tr представляет собой файл электронной книги TomeRaider, созданный с помощью программного обеспечения TomeRaider eReader. Эти файлы можно открывать как на устройствах Windows, так и на портативных устройствах, таких как Android, Palm OS и EPOC, и они классифицируются как текстовые файлы. Программное обеспечение TomeRaider, разработанное на веб-сайте [Yadabyte](https://www.yottabyte.ltd/) и на официальном веб-сайте, предоставляет дополнительную информацию о файлах TR. Формат файла TR2 был заменен форматом TR3 (TomeRaider 3). Файлы TR предлагают быстрый поиск и высокий уровень сжатия.

## Краткая история формата файла TR

TomeRaider был разработан британской компанией Yadabyte, занимающейся разработкой программного обеспечения и веб-сайтов, в 1999 году. Компания усовершенствовала его до версии 3 TomeRaider.

## Формат файла TR

Последней версией формата файла TR является версия 3, обеспечивающая поддержку быстрого поиска и сжатия. Он сжимает текстовые файлы до 45-60% от исходного размера. Это позволяет читателям электронных книг TomeRaider выполнять быстрый поиск, индексирование и сжатие. Формат файла TR подходит для больших справочных документов, и для шифрования большинства документов требуется ключ DRM.

## Проблемы с открытием файла TR

Могут быть некоторые распространенные проблемы, из-за которых вы не сможете открыть файл:

* Отсутствие сопутствующего ПО
* Устаревшая версия ПО
* Поврежденный файл TR
* Некорректные ссылки на файл TR в обращениях к архиву
* Нет прав доступа в системе для открытия файлов
* Устаревший диск в вашей системе
* Удаление отчета ТР из архива Windows
* Частичная установка приложения, поддерживающего формат TR
* Зараженный файл TR
* Недостаточно места для аппаратных ресурсов, чтобы помочь открыть файл TR
