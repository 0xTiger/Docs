## .LIT вариант №

Файлы LIT принадлежат древней программе **Microsoft Reader**, которая использовалась для чтения электронных книг. Электронные книги, отображаемые программой Microsoft Reader, имеют расширение **.lit**, сокращенное до **Literature**. Формат файла LIT состоит из файла Microsoft ITOL/ITLS HTML Help 2.0, который предоставляет всевозможные удивительные функции, такие как сжатие [LZX](/ru/compression/lzx/), двоичное представление материала разметки и произвольные необязательные вспомогательные данные, связанные с каждый поток. Текст книги представляет собой произвольное количество потоков разметки OEBPS 1.0 и подмножество CSS OEBPS 1.0. Изображения бывают [GIF](/ru/image/gif/), [JPEG](/ru/image/jpeg/) и [PNG](/ru/image/png/).

## Сколько времени Microsoft Reader отображал файлы LIT

**Microsoft Reader** — это приложение для чтения электронных книг, выпуск которых прекращен корпорацией Майкрософт. Первая версия Microsoft Reader была выпущена в августе 2000 года и использовала собственный формат LIT. Он был доступен для КПК и компьютеров с Windows. Название приложения также позже использовалось для приложения (которое изначально было выпущено с Windows 8) для чтения файлов [PDF](/ru/pdf/) и [XPS](/ru/page-description-language/xps/). Приложение Microsoft Reader было прекращено в 2018 году.
