## .EA вариант № ##

Расширение .ea используется для файла **Kindle End Action**, который является одним из форматов файлов Kindle. Эти файлы создаются **устройством для чтения электронных книг Amazon Kindle**, чтобы показать то, что автор хочет отобразить в конце электронной книги. Файл EA содержит информацию в формате [XML](/ru/web/xml/). Таким образом, этот файл можно открыть или просмотреть с помощью любого текстового редактора. Этот файл можно найти там, где устройства для чтения электронных книг Kindle сохраняют электронные книги. Содержимое в нем может быть пустыми тегами XML, если клиент не настроил информацию для отображения в конце электронной книги.

## Еще немного о формате файлов EA ##

Файл с расширением .ea представляет собой XML-файл, содержащий данные о покупателях, купивших книгу, а также купивших функцию, позволяющую показывать больше информации об авторе после последней страницы электронной книги; включая **ASIN** или **ISBN**, чтобы он мог перейти на страницу Kindle Store с заголовком.

### Что такое ASIN/ISBN? ###

ASIN означает **Стандартный идентификационный номер Amazon**, который на самом деле представляет собой десятизначный буквенно-цифровой код для уникальной идентификации продуктов на Amazon. ASIN — это уникальный идентификатор для каждого продукта, который присваивается при создании продукта Amazon. Каждый продукт на Amazon имеет ASIN, за исключением книг. Amazon использует ISBN (международный стандартный номер книги) вместо ASIN. Этот идентификатор Amazon для ваших продуктов необходимо получить, прежде чем продавать их на Amazon.

## Проблема с открытием файла EA?

Если вы не можете открыть и запустить файл EA; это не означает, что на вашем устройстве не установлено подходящее программное обеспечение. Могут быть некоторые другие проблемы, которые мешают правильной работе файла. Возможные проблемы могут быть одной из следующих:

- Повреждение файла EA.
- Неправильные ссылки на файл EA в записях реестра.
- Удалено описание советника из реестра Windows
- Поврежденная установка приложения, поддерживающего формат EA
- Зараженный файл EA с нежелательным вредоносным ПО.
- На компьютере недостаточно аппаратных ресурсов для работы с файлом EA.
- Драйверы, используемые компьютером для открытия файла EA, устарели.
