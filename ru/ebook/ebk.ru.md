## .EBK вариант №
Расширение EBK используется программой **eBook Pro Viewer** при сохранении файла электронной книги. eBook Pro — это программное обеспечение, которое может создавать электронные книги и защищать информацию, защищенную авторским правом. Файл EBK содержит зашифрованную и сжатую версию электронной книги, которую можно прочитать с помощью **eBook Pro Viewer**; после предоставления соответствующих учетных данных для аутентификации.
