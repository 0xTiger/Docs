## .ASC вариант №

Файлы ASC — это текстовые файлы Autodesk, в которых хранится информация об объектах на чертеже, включая их расположение, размер и цвет. Это делает их популярным выбором для обмена чертежами между различными программами [CAD](/ru/cad/). Файлы ASC можно открывать с помощью программных продуктов Autodesk. Autodesk предоставляет возможность импортировать файлы ASC из приложения. Файлы ASC можно открывать в любом текстовом редакторе, таком как Блокнот Microsoft, Notepad++ и Apple TextEdit.

## Формат файла ASC

Файлы ASC сохраняются на диске в формате обычного текстового файла ASCII. Они удобочитаемы для человека.

### Пример файла ASC

```
ncols 200
nrows 200
xllcorner 119000.00
yllcorner 158000.00
cellsize 5.00
nodata_value -999999
```
