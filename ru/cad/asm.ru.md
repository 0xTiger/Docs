## .ASM вариант №

Файл ASM — это файл данных, используемый [Solid Edge](https://solidedge.siemens.com/en/) для трехмерного моделирования, визуализации и сборки в САПР. Он содержит 3D-объекты, состоящие из сборки смоделированных деталей. Один файл ASM может содержать несколько частей, составляющих полный файл данных. Файлы ASM полностью отличаются от файлов [ASM языка ассемблера](/ru/programming/asm/) и не имеют к ним никакого отношения. Для открытия этих файлов ASM используется программное обеспечение Siemens Solid Edge.

## использованная литература

* [Solid Edge ASM](https://www.wikidata.org/wiki/Q644575)
* [Siemens Solid Edge](https://solidedge.siemens.com/en/)
