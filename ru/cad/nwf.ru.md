## .NWF вариант №

Файл с расширением .nwf представляет собой собственный формат файлов САПР Autodesk, который содержит индексы всех файлов моделей, используемых в проекте. Он содержит указатели на элементы, добавленные в файлы модели для ссылки. Файлы NWF также содержат такую информацию, как точки обзора и пометки. Преимущество использования формата файла NWF заключается в немедленном отражении изменений в исходных файлах САПР [например, DWG](/ru/cad/dwg/) благодаря связанным указателям. Для тестирования на коллизии настоятельно рекомендуется использовать NWF. Файл NWF сохраняет исходные результаты коллизий, и после изменения файлов САПР измененные файлы САПР будут открываться при повторном открытии файла NWF.

## Формат файла NWF

Сведения о внутреннем формате файла NWF недоступны, поскольку это собственный формат файла Autodesk.
