## .JVSG вариант №

JVSG — это файл САПР, который можно запустить в Windows. Это файл в двоичном формате, созданный JVSG. Файлы проекта видеонаблюдения в основном используют суффикс имени файла JVSG. Этот файл может хранить пропускную способность, местоположение, углы обзора и расчеты жесткого диска камеры с информацией о ее конструкции CCTV. Файлы проекта JVSG предлагают расширенный анализ и документирование путем рендеринга в трехмерные виды.

Сети камер безопасности различных предприятий, которым нужны камеры наблюдения, также могут быть разработаны с помощью файлов проекта JVSG. Программное обеспечение «Инструмент проектирования видеонаблюдения» подходит для управления файлами JVSG.
