## .RML вариант №

Файл RML — это файл аннотации, созданный и используемый приложением Volo View компанией [Autodesk](https://autodesk.com). Он содержит аннотационную информацию и текстовые метки, которые следует накладывать на чертеж для создания заметок. Эти аннотации отображаются красным текстом поверх чертежа, поэтому файл называется Redline Markup Language. Volo View больше не поддерживается Autodesk. Содержимое файлов RML встраивается в чертежи САПР в виде примечаний.

## Формат RML-файла

Файлы RML создаются и сохраняются на диск в формате XML, который является универсальным форматом файлов для обмена данными. Он имеет удобочитаемый формат и может быть открыт в любом текстовом редакторе, таком как Microsoft Notepad, Notepad++ и Apple TextEdit.
