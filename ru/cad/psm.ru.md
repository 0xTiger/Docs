## .PSM вариант №

Файл с расширением .psm — это файл САПР, который создается и открывается в программе моделирования САПР Solid Edge для проектирования 3D-деталей и сборок. Содержит 3D модель конструкции для изготовления деталей из листового металла. Программный пакет Solid Edge предоставляет все необходимые опции для создания 3D-модели, которую можно интегрировать с другими механическими конструкциями. Его 2D Nesting поддерживает форматы файлов PSM и PAR в дополнение к другим нейтральным форматам данных, включая DXF и DWG, путем извлечения данных развертки.

### Формат файла PSM

PSM — это близкий формат файла, и его подробности на данный момент недоступны. Они имеют тип MIME application/octet-stream. Файлы металлических моделей можно импортировать в Solid Edge и преобразовать в пригодные для использования детали, чтобы сделать их редактируемыми. Синхронное моделирование обеспечивает большее удобство работы с файлами PSM.
