## .PAT вариант №

Файл с расширением .pat представляет собой файл САПР, который используется программным обеспечением AutoCAD. Приложения, которые могут открывать файлы PAT, используют образец штриховки, хранящийся в этих файлах, и получают информацию о текстуре/заливке области. Содержащиеся узоры дают информацию о внешнем виде материала для нарисованных объектов. Файлы PAT можно открывать в таких приложениях, как Autodesk AutoCAD, CorelDRAW Graphics Suite и Ketron Software. Файлы PAT можно конвертировать в различные форматы изображений, такие как JPG, PNG, BMP и т. д.

## Формат файла PAT

Файлы PAT записываются в текстовом формате и могут быть открыты в любом текстовом редакторе. Образцы штриховки в этих файлах основаны на векторах и следуют синтаксису, указанному в документации AutoCAD.

Все шаблоны начинаются с точки 0,0, шаблон рассчитывается так, чтобы покрыть границу штриховки.

### Определение шаблона

Все определения шаблонов состоят из следующих полей:
* Ведущий '\*'
* Имя. Имя не может содержать пробелов, знаков препинания, круглых скобок или косых черт.
* Описание

Стандартный формат шаблонов штриховки: `<\*><name> <, описание>`. Имя и описание разделяются запятой, длина строки описания не может превышать восьмидесяти символов. Образцы штриховки определяются после этой информации.

### Примеры образцов штриховки
Ниже приведен пример квадратного узора.
```
* square, a square pattern
0,     0,0,   0,1,   1,-1
90,   0,1,   0,1,   1,-1
```
Другой пример, показывающий шаблон параллелограмма, выглядит следующим образом.

```
*pgram, parallelogram pattern
0,     0,0,    1,1,                     1,-1
45,   0,0,    0,1.4142,   1.4142,-1.4142
45,   0,1,    0,1.4142,   1.4142,-1.4142
```
