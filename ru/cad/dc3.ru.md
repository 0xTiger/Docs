## .DC3 вариант №

Файл с расширением .DC3 представляет собой 3D-чертеж, созданный с помощью DesignCAD 3D MAX, программного обеспечения САПР. Он используется для создания высококачественных дизайнов, простых рендеров и анимации и поддерживает импорт/экспорт [STL](/ru/cad/stl/). Файл DC3 используется в качестве альтернативы формату файла .DCD. Файлы DC3 можно открыть с помощью IMSI DesignCAD 3D Max, который предоставляет полный спектр мощных примитивов 3D-моделирования, включая коробку, сферу, полусферу, пирамиду, цилиндр, конус и тор.

## Формат файла DC3

Файл DC3 содержит информацию о 3D-чертеже в текстовом формате ASCII, который можно открыть в любом текстовом редакторе. Однако смысл его содержимого может быть понят только приложениями, которые понимают базовый формат файла.
