## .NWD вариант №

Файл с расширением .nwd представляет собой файл САПР от Autodesk для файлов Navisworks, который представляет собой программное обеспечение для просмотра 3D-моделей для архитектуры, проектирования и строительства. Файлы NWD объединяют данные проектирования и строительства в единую модель. Это позволяет выявлять и устранять различия и проблемы помех до начала строительства с использованием программного обеспечения. В файлах NWD хранится информация о модели, среде, текущем виде и сохраненных точках обзора (включая пометки, комментарии и т. д.). Они также могут содержать пользовательские метаданные в опубликованной форме. Файлы NWD можно открывать в любом продукте Navisworks и средстве просмотра Navisworkds Freedom.

## Формат файла NWD

NWD — это собственный формат файлов Autodesk, поэтому подробности о нем недоступны. Файл NWD можно рассматривать как текущее состояние модели. Он содержит всю информацию о геометрии, а также обзорные пометки для уточнения точек обзора. Файлы NWD на 80% меньше по размеру из-за сжатия данных по сравнению с исходным размером.
