## .NKIT вариант №

Файл NKIT представляет собой образ диска с данными, извлеченными из игр NintendoWii и GameCube. NKIT предназначен для формата Nintendo Toolkit. В полученном сжатом файле [ISO](/ru/compression/iso/) используются основные данные этих игр для запуска с помощью таких программ-эмуляторов, как Dolphin, Swiss и Nintendont. NKit поставляется в необработанных (iso) и сжатых (gcz) форматах файлов, которые были разработаны с учетом удобства воспроизведения и размера.

## Формат файла NKIT

Формат файла NKit — это формат без потерь, который используется для сжатия и восстановления изображений Nintendo Wii и GameCube. Форматы доступны в форматах файлов ISO и GCZ для каждой из игр GameCube и Wii.

|Система |Формат |Поддерживаемое оборудование |Поддерживается Dolphin |Восстановление 1:1 |Примечания|
---|---|---|---|---|---|
|Игровой куб| nkit.iso| Да |Да| Да |То же, что и сжатый GameCube iso|
|Игровой куб| nkit.gcz| Нет| Да| Да |GCZ — это собственный формат сжатия Dolphin с возможностью поиска по блокам|
|Вии| nkit.iso| Нет Да| Да| Формат RVT-H воспроизводится только в Dolphin |
|Вии| nkit.gcz| Нет Да| Да| RVT-H в GCZ доступен только в Dolphin |

### Заголовок НКИТ

Заголовок формата файла NKIT выглядит следующим образом.

|Смещение |Длина |Имя|
---|---|---|
|0x200 |0x4 |Заголовок NKit 'NKIT'|
|0x204 |0x4 |Версия NKit 'v01'|
|0x208 |0x4 |Исходный образ исходный CRC32|
|0x20C |0x4 |NKit CRC — делает CRC32 файла NKit равным исходному CRC по адресу 0x208 (по адресу 0x4 в GCZ)|
|0x210 |0x4 |Длина исходного изображения|
|0x214 |0x4 |Forced Junk ID (когда ID диска отличается - редко - только для GameCube)|
|0x218 |0x4 |Wii Обновление CRC32 раздела, если оно было удалено при преобразовании|
