## .NRG вариант №

Формат файла изображения, созданный с использованием оптического диска, считается форматом файла NRG. Этот формат был специально создан Nero для утилиты Nero Burning Rom. Считается, что для хранения образов дисков используется этот формат, поскольку он подходит для этих конкретных файлов. Эти файлы могут быть в виде точной копии CD или DVD или могут иметь несколько файлов, которые можно смонтировать как диск. Другие более популярные форматы файлов, такие как форматы файлов [ISO](/ru/compression/iso/), — это те, в которые эти файлы можно преобразовать с помощью некоторых основных утилит. В основном файлы NRG используются для создания резервных копий или копий некоторых важных данных или дисков.

## Формат файла NRG ##

Этот формат файла был разработан Nero AG как формат образа диска. Он обладал особым свойством утилиты Nero Burning Rom, поскольку был разработан для хранения образов дисков. Его первая версия была предназначена для хранения значений в виде 32-битных целых чисел. Однако была запущена его вторая версия с поддержкой 64-битных целых чисел.

## Техническая спецификация ##

В начале файла этот формат не хранит свои данные в виде заголовка. Как и нижний колонтитул, он прикрепляется в конце файла. В виде фрагментов IFF хранится информация об изображении. Смещение первого фрагмента можно получить, прочитав нижний колонтитул NRG в последних 8-12 байтах файла. Во всех версиях формата файла NRG есть фрагменты, DAOI, текст компакт-диска, тип носителя с информацией о сеансе, информация о диске, Relo и конец цепочки, связанный с ним. Порядок байтов имеет обратный порядок байтов, и все целые значения не имеют знака во время хранения.

### Основные блоки ###

#### Cue Sheet ####

Этот Cue Sheet легко доступен во всех версиях формата файлов NRG. Кусок **CUEX** означает блоки конкатенации фиксированного размера, и каждый из них представляет ключевую точку.

#### Информация о DAO ####

Эта информация также присутствует во всех версиях формата NRG. Фрагменты **DAOI** хранят соответствующую конкретную информацию в двух частях. Его первая часть состоит только из данных, относящихся к сеансу. Его вторая часть просто повторяет серую информацию, связанную с отслеживанием, и это только один раз для каждого трека.

#### CD-текст ####

Это доступно во второй версии NRG. Фрагмент **CDTX** состоит из необработанной конкатенации пакета CD-текста, каждый из которых имеет 18 байтов.

#### Расширенная информация о треке ####

Это доступно в каждой версии формата файла NRG. Эти фрагменты используются для хранения информации об отслеживании для трека одновременных сеансов. Эти данные повторяются только один раз для каждой дорожки.

#### Информация о сеансе ####

Это также доступно в каждой версии формата файла NRG. Информацию о фрагментах сеанса необходимо использовать для быстрого сканирования изображения сеанса, а затем отслеживать подсчет.

#### Конец цепочки ####

Фрагмент конца означает, что больше нет фрагментов, которые нужно прочитать, и это также доступно во всех версиях NRG.
