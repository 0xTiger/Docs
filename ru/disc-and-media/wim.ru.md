## .WIM вариант №
Файлы с расширением .wim впервые появились в Windows Vista. WIM относится к файловому формату образов, который обычно был представлен в Microsoft Windows Vista. Он позволяет развертывать один образ диска на различных компьютерных платформах. Файлы WIM используются для управления такими файлами, как обновления, драйверы и компоненты, без перезагрузки образа операционной системы. Файл AQ WIM содержит набор файлов и связанных с ними метаданных, которые очень похожи на другие форматы дисковых файлов.

## WIM-форматы файлов
Формат файла WIM не похож на форматы на основе секторов, такие как VHD или IS, на самом деле он основан на файлах, что означает, что основной единицей информации в WIM является файл. Основным преимуществом работы с файлами является то, что хранилище одного экземпляра файла, на который несколько раз ссылаются в дереве файловой системы, и независимость от оборудования. Это снижает накладные расходы на открытие и закрытие различных файлов по отдельности, поскольку файлы хранятся в одном WIM. файл. Файлы WIM могут хранить несколько образов дисков, на которые можно ссылаться либо по их уникальному имени, либо по их числовому индексу.
### Поддержка алгоритмов сжатия
WIM поддерживает следующие семейства алгоритмов сжатия по убыванию скорости и по возрастанию:
- ЭКСПРЕСС
- ЛЗХ
- ЛЗМС
- Твердое сжатие.
Первые три семейства основаны на LZ77, а сжатие Solid было представлено вместе с LZMS в WIMGAPI Windows 8 и DISM Windows 8.1.
### Инструменты для формата файла WIM
Следующие инструменты обычно поддерживают формат файла WIM:

- **ImageX**: инструмент командной строки, используемый для редактирования, создания и развертывания образов дисков Windows в формате Windows Imaging. Вместе с соответствующим WIMGAPI он распространяется как часть бесплатного комплекта автоматической установки Windows. Начиная с Windows Vista, программа установки Windows использует API WAIK для установки Windows.
- **DISM**: инструмент, представленный в Windows 7 и Windows Server 2008 R2, который может выполнять задачи, связанные со службой, в установочном образе Windows, будь то онлайн-образ или автономный образ в папке или WIM-файле. этот инструмент мог монтировать и размонтировать образы, добавлять драйвер устройства в автономный образ и запрашивать установленные драйверы устройств в автономном образе.
