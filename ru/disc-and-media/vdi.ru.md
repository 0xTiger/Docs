## .VDI вариант №
Файл с расширением .vdi представляет собой образ виртуального диска; относится к программе виртуализации настольных компьютеров с открытым исходным кодом Oracle под названием VirtualBox. Файлы VDI используются для запуска виртуальной машины VirtualBox. Виртуальные машины позволяют пользователям запускать дополнительные операционные системы или программы на одном компьютере. Следовательно, преимущество файлов VDI заключается в том, что пользователи могут сохранять эти файлы образов дисков на своем жестком диске и запускать их с помощью эмуляторов в любое время.

## Формат VDI-файла
Virtual Disk Image (VDI) — это основной формат файла диска для активно разрабатываемой виртуальной машины Oracle с открытым исходным кодом, известной как VirtualBox, которая является продуктом виртуализации корпоративного класса. Формат файла VDI предназначен для создания переносимого образа диска, который можно легко запустить с помощью других программ виртуализации. Он поддерживает как динамически выделяемое хранилище, так и хранилище фиксированного размера. Он позволяет расширить файл изображения после его создания, даже если он уже содержит данные.

### Виртуализация дисков
Система эмулирует жесткие диски в формате образа диска VDI. Виртуальная машина VirtualBox может использовать более ранние диски, созданные в Microsoft Virtual PC или VMware, а также собственный собственный формат. VirtualBox также может подключаться к целям iSCSI и создавать необработанные разделы на хосте, используя либо виртуальные жесткие диски. VirtualBox эмулирует IDE (контроллеры PIIX4 и ICH6), SATA (контроллер ICH8M), контроллеры SAS, к которым можно подключать жесткие диски, и SCSI.

Поддержка доступна для формата Open Virtualization Format (OVF), начиная с версии 2.2.0 VirtualBox. Как физические устройства, подключенные к хосту, так и образы ISO могут быть смонтированы как приводы CD или DVD.
По умолчанию VirtualBox поддерживает графику через пользовательскую виртуальную видеокарту, совместимую со стандартом VESA.

### Поддержка виртуализации для Ethernet
VirtualBox виртуализирует следующие карты сетевого интерфейса для сетевого адаптера Ethernet:
- AMD PCnet PCI II (Am79C970A)
- AMD PCnet-Fast III (Am79C973)
- Intel Pro/1000 MT для настольных ПК (82540EM)
- Сервер Intel Pro/1000 MT (82545EM)
- Сервер Intel Pro/1000 T (82543GC)
- Паравиртуализированный сетевой адаптер (virtio-net)

Эмулированные сетевые карты позволяют запускать гостевые ОС без необходимости поиска и установки драйверов для сетевого оборудования, поскольку они доступны как часть гостевой ОС. Специальный паравиртуализированный сетевой адаптер повышает производительность сети, сокращая необходимость согласования с конкретным аппаратным интерфейсом. Следовательно, требуется специальная поддержка драйверов в гостевой системе.
