## .TIME вариант №

Формат файла TIME использовался веб-системой LIGHT, которая помогала пользователям записывать, систематизировать и делиться своей жизнью. По сути, он хранил коллекцию сообщений и представлял собой папку на странице жизни пользователя в системе.

## Как узнать, когда файл был создан в Linux

В Linux вы можете использовать команду «stat», чтобы узнать, когда был создан файл. Вот как вы можете это сделать:

Откройте терминал и введите следующую команду:

```
stat <file_path>
```

Заменить `<file_path> ` с путем к интересующему вас файлу. Например:

```
stat /path/to/your/file.txt
```

Эта команда отобразит различную информацию о файле, включая время его создания. Найдите строку, которая начинается с «Рождение» или «Файл:». Время рождения указывает, когда файл был создан в файловой системе.

Вот пример того, как может выглядеть результат:

```
  File: /path/to/your/file.txt
  Size: 1024       	Blocks: 8          IO Block: 4096   regular file
Device: 801h/2049d	Inode: 1643318     Links: 1
Access: (0644/-rw-r--r--)  Uid: ( 1000/   user)   Gid: ( 1000/   user)
Access: 2024-01-31 12:34:56.789012345 +0000
Modify: 2024-01-31 12:34:56.789012345 +0000
Change: 2024-01-31 12:34:56.789012345 +0000
 Birth: 2024-01-31 12:34:56.789012345 +0000
```

В этом примере время «Рождения» указывает, когда был создан файл.
