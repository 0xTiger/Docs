## .GED вариант №

Файл **GEDCOM (передача генеалогических данных)**, обычно с расширением «.ged», представляет собой стандартный формат файла, используемый для обмена генеалогическими данными между различными программами генеалогического программного обеспечения; Файлы GEDCOM представляют собой обычные текстовые файлы, содержащие информацию об отдельных лицах, семьях, событиях и другие генеалогические данные; Целью GEDCOM является обеспечение общего формата для обмена генеалогической информацией.

Файлы GEDCOM обычно имеют расширение «.ged». это расширение используется для обозначения того, что файл соответствует **стандарту GEDCOM** для обмена генеалогическими данными; поэтому, когда вы сохраняете или экспортируете свои генеалогические данные из программы генеалогического программного обеспечения, результирующему файлу часто присваивается имя типа «вашеимя_файла.ged».

Например:

- yourfamilytree.ged
- john_doe_family.ged
- ancestry_data.ged

## Ключевые моменты о файлах GEDCOM

Вот некоторые ключевые моменты о файлах GEDCOM:

1.  **Формат обычного текста:** файлы GEDCOM представляют собой обычные текстовые файлы; а это значит, что их можно открыть и просмотреть в любом текстовом редакторе; этот формат облегчает совместное использование и совместимость между различными программными приложениями по генеалогии.
    
2.  **Структура.** Файлы GEDCOM имеют иерархическую структуру; данные организованы в записи; каждая запись имеет специальный тег, который указывает тип содержащейся в ней информации, например, имя человека может быть представлено тегом «ИМЯ».
    
3.  **Individuals and Families:** GEDCOM files typically include information about individuals such as names, dates of birth and places of birth; they also include details about family relationships, marriages and other events.
    
4.  **Взаимодействие программного обеспечения:** поскольку GEDCOM является стандартизированным форматом; это позволяет пользователям перемещать свои генеалогические данные между различными программами по генеалогии; это особенно полезно для пользователей, которые могут переключиться на другое программное обеспечение или поделиться своими исследованиями с другими.
    
5.  **Версия:** GEDCOM выдержал несколько версий, каждая из которых содержала улучшения и новые функции; наиболее часто используемой версией была GEDCOM 5.5. 

## Как открыть файл GED?

Чтобы открыть файл GED, вам обычно требуется программное обеспечение для генеалогии, поддерживающее формат GEDCOM. Некоторые популярные варианты программного обеспечения для генеалогии включают в себя:

- **Создатель семейного древа**
- **Ancestry.com**
- **Мое наследие**
- **Gramps** (бесплатно и с открытым исходным кодом)
- **Семейное древо прошлого**

После того, как вы выбрали программное обеспечение для генеалогии, найдите в меню или панели инструментов такие опции, как «Импорт», «Открыть GEDCOM» или «Импорт из файла».

Поскольку файлы GED сохраняются в текстовом формате, вы также можете открыть файлы с помощью обычного текстового редактора, например

- **Блокнот**
- **Блокнот++**
- **Apple TextEdit**
- Любой текстовый редактор
