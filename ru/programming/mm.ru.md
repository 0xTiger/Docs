## .MM вариант №

Файл MM — это файл исходного кода, который может содержать код программирования Objective-C и Objective-C++. Он используется в MacOS и отличается от файлов .M, содержащих только код Objective-C. Когда компилятор видит расширение .mm, он может скомпилировать классы C++, присутствующие в файле. Поскольку один файл исходного кода может содержать сочетание языков [C](/ru/programming/c/) и [C++](/ru/programming/cpp/), переименование файлов .m в расширение .mm позволяет функциям C++ устанавливать in. Приложения, которые могут открывать файлы .MM, включают Apple TextEdit, Microsoft Notepad, Notepad++ и Microsoft Wordpad.
