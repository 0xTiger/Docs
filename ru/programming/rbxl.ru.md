## .RBXL вариант №

Файл RBXL — это файл местоположения Roblox, который содержит местоположение, определенное в трехмерном мире внутри [игры Roblox](https://www.roblox.com/). Сама Roblox — это многопользовательская строительная игра, которая позволяет игрокам создавать игры, используя инструменты разработчика, предоставляемые платформой. Файлы RBXL используют инструменты, сценарии, команды и части для описания игрового поля в виде 3D-моделей. Roblox Studio позволяет игроку создавать игровые предметы и может использоваться для открытия файлов RBXL.

## Формат файла RBXL

Файлы RBXL разрабатываются с использованием Roblox Studio и сохраняются в формате файла [XML](/ru/web/xml/). Благодаря этому универсальному формату файлов существует множество инструментов и API, позволяющих создавать и открывать эти файлы. Одним из таких API является [MaximumADHD](https://github.com/MaximumADHD/Roblox-File-Format), библиотека [C#](/ru/programming/cs/) для создания и управления файлами в формате Roblox model/place. .
