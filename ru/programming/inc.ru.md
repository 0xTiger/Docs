## .INC вариант №

Файл INC — это включаемый файл, который используется исходным кодом программы для ссылки на информацию заголовков. Он похож на [формат файла .h](/ru/programming/h/) и содержит объявления, функции, заголовки или макросы, которые могут повторно использоваться файлами исходного кода, такими как C++. Файлы INC используются различными языками программирования, такими как C, [C++](/ru/programming/cpp/), Pascal, [PHP](/ru/programming/php/) и [Java](/ru/programming/java/). Текстовые редакторы, такие как Microsoft Notepad, Notepad++ и TextEdit, можно использовать для открытия файлов INC.

## Формат файла INC — дополнительная информация

Файл INC сохраняется как обычный текстовый файл со всеми объявлениями, включенными в соответствии с синтаксисом объявлений. Один файл INC может также ссылаться на другие файлы INC. После создания файл INC становится частью проекта, и на него можно ссылаться из исходных файлов, таких как C++. На него могут ссылаться несколько исходных файлов, чтобы избежать дублирования.

## Содержимое файла INC

Файлы INC могут содержать следующую информацию.

**`Переменные`** — в случае объектно-ориентированного программирования (ООП) файл заголовка класса содержит определения всех переменных уровня класса, которые доступны в файлах исходного кода реализации.

**`Объявление методов`** — все объявления методов включены в файлы заголовков .h, чтобы быть доступными для нескольких файлов реализации.

**`Определения невстроенных функций`** — заголовочные файлы также могут содержать определения невстроенных методов.

## Недостаток использования файла INC в PHP

PHP — это скрипты на стороне сервера, которые запускаются на сервере и возвращают результаты на вызывающие веб-страницы. Если файл PHP ссылается на файл INC, а сервер не настроен для анализа файлов .inc (что очень часто), пользователь может просмотреть исходный код php в файле .inc, посетив каталог URL. Таким образом, нужно быть осторожным при использовании INC-файлов в качестве включаемых файлов в PHP.
