## .RBXM вариант №

Файл RBXM — это формат файла 3D-модели, используемый в Roblox Studio IDE для хранения 3D-объектов многопользовательской онлайн-игры Roblox. Он содержит 3D-объекты, созданные для игры с помощью IDE. Примеры 3D-объектов в файле RBXM включают рубашки, брюки, шляпу, здания, транспортные средства, животных, самолеты, автомобили или растения. Файлы RBMX можно преобразовать в [формат файла OBJ 3D](/ru/3d/obj/) с помощью Roblox IDE. Roblox IDE можно использовать для открытия файлов RBMX.

Другим популярным форматом файла Roblox является [RBXL](/ru/programming/rbxl/), который представляет собой формат файла местоположения и содержит местоположение, определенное в 3D-слове внутри игры Roblox.

## Формат файла RBMX — дополнительная информация

Файлы RBXM создаются и сохраняются в двоичном формате, и их внутренние спецификации формата файлов недоступны для общественности. Это отличается от формата файла RBXMX, основанного на формате файла [XML](/ru/web/xml/). На Github доступно множество инструментов/репозиториев, которые можно использовать для [обработки файлов Roblox RBXM](https://github.com/search?q=rbxm).
