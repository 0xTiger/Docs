## .C вариант №

Файл, сохраненный с расширением c, представляет собой файл исходного кода, написанный на языке программирования C. **Файл C** включает в себя всю реализацию функциональности приложения в виде исходного кода. Объявление исходного кода написано в заголовочных файлах, которые сохраняются с расширением [.h](/ru/programming/h/). C++ — это современная форма языка C, которая в настоящее время используется для разработки большей части программного обеспечения.

## Формат файла С

Файлы C записываются в формате обычного текстового файла в соответствии с синтаксисом языка программирования. Типичный файл C будет иметь:

* оператор импорта в верхней части файла для импорта любых заголовочных файлов
* один или несколько методов для реализации желаемой функциональности

### Импорт заголовков

Файлы заголовков с расширением .h содержат необходимые операторы включения для включения других функциональных файлов в проект. Кроме того, они содержат объявления методов, определенных в файле реализации. Файлы заголовков включаются с помощью оператора include, как показано ниже.

```
#include <filename.h>
```

### Реализация исходного кода

Фактическая реализация функциональных требований закодирована как методы в файле C. Каждый метод в файле C имеет тип возвращаемого значения, имя метода и может иметь некоторые входные параметры. Если возвращаемый тип не пуст, метод может возвращать какое-то значение.

### Пример кода C
Вот пример программы переменного тока:

```
long some_function();
/* int */ other_function();

/* int */ calling_function()
{
    long test1;
    register /* int */ test2;

    test1 = some_function();
    if (test1 > 0)
          test2 = 0;
    else
          test2 = other_function();
    return test2;
}
```
