## .AWK вариант №

Файл AWK — это файл сценария, созданный для приложения обработки текста **AWK**, которое поставляется в комплекте с операционными системами Unix и Linux. Он содержит логические инструкции для выполнения действий с текстом или содержимым файла, таких как сопоставление, замена и печать текста. Это помогает создавать отчеты и форматированный текст из входного файла. Утилита awk обычно находится в /usr/bin/awk в большинстве дистрибутивов Linux/Unix.

## Как создать AWK-скрипт?

Файл AWK можно создать или открыть в любом текстовом редакторе в ОС Linux/Unix. Новый файл сценария AWK можно создать следующим образом.

```
$ vi script.awk
```

Это откроет файл AWK в текстовом редакторе. Введите несколько утверждений в текстовом редакторе, как показано ниже.

```
#!/usr/bin/awk -f
BEGIN { printf "%s\n","Writing my first Awk executable script!" }
```
Первая строка этого текстового содержимого поясняется следующим образом.

* \#! – называется «Шебанг», который указывает интерпретатор инструкций в сценарии.
* /usr/bin/awk — интерпретатор
* -f – опция интерпретатора, используемая для чтения файла программы

## Как выполнить скрипт AWK?

Сохраните файл и сделайте скрипт исполняемым, введя команду ниже:
```
$ chmod +x script.awk
```

Затем сценарий можно запустить следующим образом.

```
$ ./script.awk
```

это приводит к следующему результату.

```
Writing my first Awk executable script!
```
