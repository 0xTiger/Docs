## .IPR вариант №

Файл IPR — это файл проекта, созданный с помощью Java IDE **IntelliJ IDEA**. Вся информация, относящаяся к настройкам проекта, такая как структура каталогов, файлы исходного кода, библиотеки, включенные в проект, и настройки компилятора, хранится в файлах IPR. Когда проект открывается из IntelliJ IDEA, среда IDE считывает информацию из файла IPR и загружает соответствующие файлы и настройки. Файлы IPR были заменены файлами .IDEA, которые содержат всю информацию в виде каталога настроек.

## Формат файла ПИС

Файлы IPR сохраняются в универсально понятном формате [XML](/ru/web/xml/), который удобочитаем. Он содержит всю информацию, связанную с проектом, в тегах, которые загружаются для отображения содержимого, связанного с проектом.

Поскольку файлы IPR хранятся в формате XML, их можно открыть в любом текстовом редакторе, таком как Notepad, Notepad++, Atom и Apple TextEdit.
