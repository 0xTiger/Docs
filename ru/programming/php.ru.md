## .php вариант №
Файл с расширением .php относится к языку программирования с открытым исходным кодом, используемому для написания сценариев на стороне сервера, которые должны выполняться на веб-сервере. Это наиболее широко используемый язык веб-скриптов, который обычно используется для разработки крупномасштабных веб-приложений. Крупнейшая система ведения блогов в Интернете, то есть WordPress, и крупнейшая социальная сеть Facebook основаны на языке PHP. Он совместим с различными платформами, такими как Windows, Linux, Unix, Mac OS X и т. д. Также он успешно работает практически на всех используемых сегодня серверах, т. е. Apache, IIS и т. д. PHP поддерживает широкий спектр баз данных, включая MySQL.

## Формат файла PHP ##

Один PHP может содержать текст, CSS, [HTML](/ru/web/html/), JavaScript и код PHP. Код PHP выполняется на сервере, а результат возвращается в браузер в виде простого HTML. Файлы PHP можно открывать в любом текстовом редакторе и редактировать на месте, хотя такие приложения, как Adobe Dreamweaver, Eclipse Инструменты разработки PHP предлагают удобный способ написания и изменения кода PHP.

### Синтаксис ###

Сценарий PHP выполняется на сервере, а результат в формате HTML отправляется обратно в браузер. PHP-скрипт начинается с<?php and ends with ?>

### Пример PHP ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
