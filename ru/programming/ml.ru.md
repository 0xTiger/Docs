## .ML вариант №

Файл ML представляет собой текстовый файл, созданный с помощью функционального языка программирования общего назначения ML (часто известного как метаязык). Он содержит исходный код ML, который чаще используется при исследовании и разработке языков программирования с использованием полностью определенной семантики. Существует несколько вариантов ML, наиболее широко используемым является Caml. Другие включают Objective Caml (OCaml), JoCaml, Extended ML и MacroML. Язык ML обычно используется для написания компиляторов, автоматического доказательства теорем и формальных проверок.
