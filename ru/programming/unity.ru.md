## .UNITY вариант №

Файл Unity — это файл 3D-сцены, созданный с помощью движка 3D Gaming **Unity**. Он используется в сочетании с другими сценами для создания виртуального мира при редактировании. Он содержит элементы 3D-сцены, такие как объекты, окружение, декорации, камеры, информацию об углах, сценарии освещения и препятствия.

Файлы Unity можно открыть с помощью Unity Technologies Unity. Unity также может читать Unity [.fbx](/ru/3d/fbx/), [.dae](/ru/3d/dae/) (Collada), [.3ds](/ru/3d/3ds/), [.dxf](/ru/cad/dxf/) и [.obj](/ru/3d/obj/).

## Формат файла UNITY

Файлы Unity хранятся на диске в двоичном формате. Они генерируются и используются вместе с другими сценами во время редактирования. Unity используется для разработки игр как для Windows, так и для Mac OS X, и может использоваться для создания 3D-игр для Интернета, Nintendo Wii, iPhone или отдельных приложений.
