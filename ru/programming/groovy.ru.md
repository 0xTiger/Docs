## .GROOVY вариант №

Файл GROOVY — это файл исходного кода, написанный на языке программирования Groovy. Код, написанный внутри файла GROOVY, подобен объектно-ориентированному языку [Java](/ru/programming/java/), что упрощает проектирование и разработку приложений. Файл GROOVY компилируется в байт-код виртуальной машины Java (JVM) и совместим с другим кодом и библиотеками Java.

## Формат файла GROOVY — дополнительная информация

Файлы GROOVY содержат исходный код, написанный с использованием синтаксиса Groovy. Он может содержать код, который можно использовать как в качестве языка программирования, так и в качестве языка сценариев для платформы Java. Язык Groovy имеет те же функции, что и Python, Ruby и Smalltalk.
