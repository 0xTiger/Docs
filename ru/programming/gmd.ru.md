## .GMD вариант №

Файл GMD — это исходный файл разработки игр, созданный с помощью инструмента разработки программного обеспечения Game Maker, который используется для создания 2D-игр. Файл GMD хранится в виде текстового файла и представляет собой файл проекта разрабатываемой игры. Game Maker доступен для Windows, Mac, Linux, Android, iOS, HTML5, Xbox, PlayStation и Nintendo Switch. Этот инструмент широко используется инди-разработчиками, профессиональными студиями и преподавателями по всему миру для создания игр.

## Формат файла GMD

Файлы GMD сохраняются в двоичном формате, и сведения о его внутреннем формате файла недоступны для справки разработчика.
