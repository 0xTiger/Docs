# M - Файлы исходного кода Matlab

## .M (Matlab) вариант №

Файл с расширением .m представляет собой файл исходного кода, используемый Matlab, платформой программирования и числовых вычислений, используемой для анализа, разработки алгоритмов и имитационного моделирования. Как и другие форматы файлов программирования, файл M содержит исходный код, который выполняет команды Matlab для построения графиков, запуска симуляций и других математических операций. Одна симуляция Matlab может охватывать несколько таких файлов .m, которые могут классифицировать приложение в сценариях, классах, функциях или объявлениях. Файлы Matlab M можно открыть в любом текстовом редакторе.

## Формат файла Matlab M — дополнительная информация

Файлы Matlab .m — это текстовые файлы, содержащие программный код на языке программирования Matlab. Их можно открывать и редактировать в любом текстовом редакторе, а также сохранять обратно для выполнения в программном обеспечении Matlab. Сам Matlab содержит Live Editor, который используется для создания сценариев, представляющих собой комбинацию кода, вывода и форматированного текста.

### Функциональные файлы Matlab

Как и в других языках программирования, вы можете создать файл .m, содержащий только определение функции, которая выполняет только определенную задачу. Такие файлы также сохраняются с расширением .m и реализуют функциональность, относящуюся только к этой функции.

### .M Пример файла

Ниже приведен пример файла функции Matlab, который вычисляет время, необходимое для падения объекта с высоты h.

```C++
function t= TimeToGround(h)  
  t=sqrt(h/4.9);
end
```

Чтобы вызвать эту функцию из редактора Matlab или из другого файла .m, можно использовать следующий код.
```C++
TimeToGround(100)
```

# M - файл реализации Objective-C

## Что такое файл M (Objective-C)?

Файл M также называется файлом реализации, который содержит исходный код класса, написанный на языке Objective-C, языке программирования, используемом для написания программных приложений для OS X и iOS. Objective-C — это основной язык программирования, используемый основными API-интерфейсами Apple, Cocoa и Cocoa Touch, для этих платформ. Одно программное приложение, разработанное на этом языке, может содержать несколько файлов .m, содержащих реализацию классов программы. Их можно открыть с помощью Apple XCode, jEdit и других распространенных текстовых редакторов.

## Формат файла Objective-C M — дополнительная информация

Файлы M записываются в текстовом формате с использованием синтаксиса программирования Objective-C. Каждый метод класса должен быть определен со всем необходимым кодом в этих файлах реализации. Эти M-файлы реализации могут импортировать один или несколько файлов заголовков .h в соответствии с требованиями. Оператор import сообщает компилятору, где найти заголовочный файл, принадлежащий этому файлу реализации. Оператор импорта записывается следующим образом.

```C++
#import "network.h"
```

Затем каждая реализация M-файла начинается с директивы `@implementation`, за которой следует имя файла класса реализации. Затем следует реализация всех методов, объявленных в заголовочном файле.

### Пример формата файла M

```C++
UrlConnection.m

#import "UrlConnection.h"

@implementation UrlConnection

(void)connect {
    // In here would be code to attempt a connection to the
    // specified URL, while possibly handling connection errors.
    //
}

 + (BOOL)canHandleRequest:(NSString \*)type
                   forUrl:(NSString \*)url {
    //And in here would be code to see if the given URL passed
    // in is capable of handling the HTTP request type specified
    // by the "type" parameter. It will return YES or NO.
 }

 @end
```
