## .YML вариант №

YML — это текстовый файл на языке сериализации данных [YAML](/ru/programming/yaml/) (еще один язык разметки), который часто используется для создания файлов конфигурации. Он может работать в сочетании с любым языком программирования, включая C/C++, Ruby, Python, Java, PHP, C# и другие. Будучи независимым от какого-либо конкретного языка программирования, он открыт, совместим и удобочитаем. Файлы YML можно открывать и редактировать с помощью обычных текстовых редакторов, таких как Microsoft Notepad, Notepad++, Github Atom, Apple TextEdit и Visual Studio IDE.

## Формат файла YML — дополнительная информация

Файлы YML хранят настройки приложения в качестве конфигурации. Его основная структура представляет собой карту, которую также можно назвать каталогом или хешем. Записи в файле YML состоят из пары ключ-значение. В основном файлы конфигурации .yml сохраняются как `configure.yml`, хотя некоторые языки программирования, такие как Ruby on Rails, сохраняют конфигурацию подключения к базе данных в файлах `database.yml`. [Спецификации формата файлов YML/YAML](https://yaml.org/spec/1.2.2/) объясняют полные языковые структуры и информацию, связанную с синтаксисом.
