## .ACD вариант №

Файл ACD — это файл проекта, который содержит информацию для программирования [Rockwell Automation](https://www.rockwellautomation.com) платформ Logix. Он включает инструкции для контроллеров и процессоров. Он используется для платформ Logix, таких как системы ControlLogix, CompactLogix, FlexLogix, SoftLogix5800 и DriveLogix.

Приложения, которые могут открывать файлы ACD, включают Rockwell Automation RSLogix 5000.

## использованная литература

* [Rockwell Automation](https://www.rockwellautomation.com/en-us.html)
