## .ASVF вариант №

Файл ASVF представляет собой сжатый файл, созданный программным обеспечением Asphyre Sphinx, которое представляет собой 2D / 3D-инфраструктуру для промышленных и корпоративных приложений. Он хранит такие данные, как шрифты, изображения и другие медиафайлы. Sphinx также можно использовать для разработки 2D/3D видеоигр. Сжатый файл ASVF может содержать общие файлы, такие как файлы .JPEG и .XML.

Файлы ASVF можно открыть с помощью программного обеспечения Asphyre Sphinx, которое уже давно прекращено.

## Формат файла ASVF

Файлы ASVF сохраняются в виде сжатых файлов и могут содержать несколько файлов внутри архива.
