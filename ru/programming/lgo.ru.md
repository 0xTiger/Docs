## .LGO вариант №

Файл LGO — программный код, написанный с помощью приложения Logo для рисования изображений. Он содержит запрограммированные инструкции, которые направляют приложение для чтения для создания рисунков в соответствии с этими инструкциями. Например, программа «черепаха», которая может открывать файлы LGO, использует эти инструкции для рисования линий в соответствии с запрограммированным кодом. Рисунок не ограничивается только линиями. Файлы LGO можно использовать для рисования простых и сложных конструкций. Многие образовательные программы использовали файлы LGO для обучения программному созданию рисунков.

## Формат файла LGO

Файлы LGO сохраняются в проприетарном формате файлов, и их внутренняя файловая структура публично не известна. Другой известный формат файлов для логотипов — файлы [.ico](/ru/image/ico/).
