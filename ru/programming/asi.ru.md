## .ASI вариант № ##

Файл ASI — это переменный формат файла, используемый программным обеспечением Alpha Five, которое используется для разработки веб-приложений. В нем хранятся переменные, используемые веб-приложением. Когда вы создаете новый проект разработчика в Alpha Five, файл ASI также создается и сохраняется вместе с проектом разработчика Alpha Five.

Файлы ASI можно открывать с помощью Alpha Software Alpha Anywhere.

## Формат файла ASI

Файлы ASI сохраняются в собственном формате файлов Alpha Five. Они читаются и записываются приложением как часть проекта разработчика.
