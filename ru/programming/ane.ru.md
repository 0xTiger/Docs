## .ANE вариант №

Файл ANE — это файл собственного расширения Adobe Air, который используется разработчиками для написания определенных функций, вызывающих код [Adobe ActionScript](https://en.wikipedia.org/wiki/ActionScript). В большинстве случаев файлы ANE содержат собственную библиотеку кода, используемую расширением. Однако сам файл устанавливается на устройство. Это позволяет использовать специальные функции системы, такие как вибрация устройства. Приложения Adobe AIR должны быть скомпилированы для использования динамической компоновки для использования файла ANE.

## Формат файла ANE — дополнительная информация

Файлы ANE сохраняются на диск как двоичные файлы, спецификации формата файлов которых недоступны для общественности. Файл ANE может содержать собственный исходный код Android Java для доступа к функциям устройства Android, таким как вызов функций громкости или вибрации. Все это делается путем обертывания родной системы языком Adobe ActionScript.
