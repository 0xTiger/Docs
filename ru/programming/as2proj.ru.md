## .AS2PROJ вариант №

Файл AS2PROJ — это файл проекта, созданный в среде программирования Flash с открытым исходным кодом, FlashDevelop. Он содержит связанную с проектом информацию, такую как параметры сборки, ссылки на исходные файлы проекта и параметры для создания выходного [файла .swf](/ru/page-description-language/swf/). Каждый новый проект, созданный в FlashDevelop, сохраняется в виде файла AS2PROJ. Вы можете изменить настройки своего проекта, и все эти изменения будут сохранены в файле AS2PROJ.

Файлы AS2PROJ можно открыть с помощью FlashDevelop.

## Формат файла AS2PROJ

Файл AS2PROJ сохраняется в формате файла [XML](/ru/web/xml/), который является универсальным форматом файла для обмена информацией. Его можно открыть в любом текстовом редакторе, таком как Notepad, Notepad++ и Atom, для изучения содержимого.
