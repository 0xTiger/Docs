## .ASC вариант №

Файл ASC представляет собой файл **скрипта**, созданный на языке программирования ActionScript. Он содержит код, который запускается на серверной стороне программного обеспечения хостинга Flash Media Server. Файл ASC содержит сценарии, которые используются для управления событиями приложений, такими как загрузка приложений Flash, обработка действий пользователя от событий мыши и клавиатуры, воспроизведение мультимедийного содержимого и рендеринг вывода приложения. Adobe приобрела Macromedia Flash, которая позже стала частью пакета Adobe Creativity как Adobe Flash. Он использовался для сохранения файлов анимации в [формат файла .swf](/ru/page-description-language/swf/).

## Формат файла ASC — дополнительная информация

Файлы ASC сохраняются в виде текстовых файлов, которые можно открывать и редактировать с помощью [Adobe Media Encoder](https://www.adobe.com/products/media-encoder.html). Они создаются и сохраняются в виде текстовых файлов и могут быть открыты в любом текстовом редакторе, таком как Microsoft Notepad, Notepad++ и Apple TextEdit. Файл ASC выполняется по умолчанию при запуске Flash Media Server и приложении.
