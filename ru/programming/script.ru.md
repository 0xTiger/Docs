## .SCRIPT вариант №

Файл SCRIPT — это исходный файл, содержащий сценарий на любом языке сценариев. Это делает его обобщенным файлом сценария, который содержит переменные и функции. Поскольку каждый язык сценариев может быть разным, сценарий, написанный внутри файла SCRIPT, необходимо выполнять с использованием соответствующего механизма сценариев.

## Формат файла SCRIPT — дополнительная информация

Файлы SCRIPT хранятся на диске в виде простых текстовых файлов. Их можно открывать и редактировать в любом текстовом редакторе, таком как Microsoft Notepad, Notepad++ и Apple TextEdit. Поскольку язык сценариев может быть другим, его можно выполнить, переименовав расширение файла в расширение по умолчанию для соответствующего языка сценариев. Например, существует несколько распространенных языков сценариев, таких как JavaScript ([.JS](/ru/web/js/)), VBScript ([.VBS](/ru/programming/vbs/)), AppleScript (.SCPT), PHP ([ .PHP](/ru/programming/php/)) и ASP ([.ASP](/ru/web/asp/)).
