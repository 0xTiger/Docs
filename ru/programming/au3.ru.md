## .AU3 вариант №

Файл AU3 представляет собой файл автоматизированного сценария, созданный с помощью бесплатной программы создания сценариев [AutoIt v3](https://www.autoitscript.com/site/). Он содержит язык сценариев AutoIt3 для автоматизации команд в Windows. Язык программирования AutoIt3 позволяет моделировать ряд событий, которые можно использовать для автоматизации задач. Например, сценарий внутри файла AU3 можно использовать для имитации нажатия клавиш, движения мыши, управления положением окон и других целей сценариев.

Приложения, которые могут открывать файлы AU3, включают [SciTE4AutoIt3](https://www.autoitscript.com/site/autoit-script-editor/downloads/) и AutoIt.

## Формат файла AU3

Файлы AU3 сохраняются как текстовые файлы и содержат язык сценариев AutoIt3.
