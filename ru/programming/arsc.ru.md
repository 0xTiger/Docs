## .ARSC вариант №

Файл ARSC — это файл таблицы ресурсов Android, который содержит список ресурсов приложения в табличном формате. Он содержит такую информацию, как имена ресурсов, свойства и идентификаторы. Файлы ARSC являются частью пакетов APK, которые используются для установки этих приложений Android. Все ресурсы, такие как изображения, макеты, стили и строки, в приложении для Android преобразуются в двоичные файлы при создании файла APK. Одновременно генерируется файл ARSC, содержащий список всех скомпилированных ресурсов программы и их идентификаторы.

## Формат файла ARSC

Файлы с расширением .arsc — это двоичные файлы, созданные после того, как компилятор, например ANT (Eclipse) или Gradle (AS), компилирует код приложения Android для создания файла [APK](/ru/compression/apk/). Вы можете извлечь APK-файл с помощью любой утилиты для архивации, такой как WinZip, чтобы просмотреть его содержимое, которое представляет собой скомпилированные файлы классов Java, например, class.dex. В дополнение к этому, он также содержит этот файл ARSC, который содержит метаинформацию о:

* XML-узлы
* Атрибуты
* Идентификаторы ресурсов

Ресурсы в файле APK ссылаются на идентификаторы ресурсов, а атрибуты преобразуются в значения во время выполнения. Инструмент Android aapt собирает все ресурсы, определенные в файлах во время сборки, и назначает им идентификаторы ресурсов. После присвоения идентификаторов скомпилированным ресурсам из исходного кода и файла resources.arsc создается файл R.java.
