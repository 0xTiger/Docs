## .AS3PROJ вариант №

Файл AS3PROJ — это файл проекта, созданный FlashDevelop, бесплатной IDE с открытым исходным кодом для разработки Flash и веб-приложений. Он сохраняет всю связанную с проектом информацию и настройки, примененные пользователем с помощью FlashDevelop IDE. Информация, хранящаяся в файле AS3PROJ, включает параметры сборки, ссылки на исходные файлы проекта и параметры для создания вывода [файл .swf](/ru/page-description-language/swf/). Когда создается новый проект, он сохраняется как файл AS3PROJ в том же месте, которое указано пользователем для сохранения данных, связанных с проектом.

## Формат файла AS3PROJ

Файл AS3PROJ сохраняется в формате файла [XML](/ru/web/xml/), который является универсальным форматом файлов для обмена информацией. Его можно открыть в любом текстовом редакторе, таком как Notepad, Notepad++ и Atom, для изучения содержимого.
