## .CMAKE вариант №

Файл CMAKE — это файл проекта, созданный кроссплатформенным инструментом с открытым исходным кодом **CMake**. Он содержит исходный скрипт на языке CMake для всей программы. Инструмент CMake использует сценарии программирования, известные как CMakeLists, для создания файлов сборки для конкретной среды, такой как make-файлы Unix-машины. Сценарии и модули, используемые проектом CMake, организованы как script.cmake и module.cmake, на которые есть ссылки в проекте.
