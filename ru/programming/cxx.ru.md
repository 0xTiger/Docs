## .CXX вариант №

Файл CXX представляет собой файл исходного кода, аналогичный C++, для написания приложений. Он может содержать полную реализацию всего приложения или частичную реализацию программы, которую можно включить в проект с другими файлами C++. Файлы CXX можно открывать в любом текстовом редакторе и популярных средах программирования, таких как Notepad++, Turbo C++ и Visual Studio.

## Формат файла CXX

Файлы CXX хранятся в виде обычного текстового файла и содержат код, написанный с синтаксисом C++. Файлы CXX также обычно имеют расширение [.cpp](/ru/programming/cpp/). Файлы CXX считаются альтернативой .cpp, используемой некоторыми компиляторами.
