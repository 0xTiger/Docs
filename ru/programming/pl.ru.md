## .PL вариант №

Файл с расширением .pl представляет собой файл сценария Perl, который является языком сценариев. Они компилируются и запускаются с использованием программного обеспечения Perl Interpreter. Файл сценария PL содержит строки кода, переменные и комментарии. Скриптовые языки относительно сложны в
понимают другие форматы файлов программирования, такие как [PHP](/ru/programming/php/). Файлы PL могут быть созданы и совместимы с Windows, macOS и Linux.

## Краткая история языка сценариев Perl

Этот язык был впервые использован в 1987 году, поэтому эти файлы появились в том же году. В 1989 году был выпущен Perl 2. Позже он был обновлен и модифицирован до версии 5.35, а следующая версия должна быть выпущена в следующем году.

В каждом обновлении добавлялись инструменты о функциональности, производительности и внешнем виде языка и файлов. За эти годы было внесено много изменений в версии. Первоначально это был базовый язык сценариев, но теперь он также включает в себя множество других модулей. Изначально это был очень простой язык, но скрипты этого языка было довольно сложно понять, так как они были компактными.

## Спецификации расширения формата файла Perl

Есть некоторые свойства или спецификации этих программных файлов, некоторые из них следующие:

* Код, включенный в эти файлы, представлен в текстовом формате и используется для разработки скриптов.
* Скрипты сервера, парсинг текста и администрирование сервера - основные аспекты, которые охватывает скрипт этого языка.
* Многие популярные программы, связанные с этим языком, находятся в активном состоянии Active Perl и Bare Bones BBEdit (совместимы с Mac OS)
* Этот язык считается языком высокого уровня
* Для Win32 пользователю может потребоваться установить родной бинарный дистрибутив языка.
* Требуется, чтобы некоторые инструменты сценариев стали исполняемыми в различных наборах ресурсов Windows.
* Visual Studio .NET — известная платформа для разработки языков программирования. Инструмент Active State, известный как Visual Perl, используется для добавления Perl в Visual Studio.
* В файлах первая строка исходного кода описывает информацию об используемом интерпретаторе. Файлы PL обычно начинаются со строки **#!/usr/bin/perl**, которая указывает вашему компьютеру запустить этот скрипт с помощью интерпретатора Perl, установленного на компьютере.


## Примеры PL-скриптов

```
#!/usr/bin/perl
print "Hello, world\n";
```

Это напечатает

```
Hello, World
```

### Однострочный комментарий ###

```
#!/usr/bin/perl
# This is a single line comment
print "Hello Perl\n";
```

### Многострочный комментарий ###

```
#!/usr/bin/perl
=begin comment
This is a multiline comment.
Line 1
Line 2
=cut
print "Hello Perl\n";
```

### Присвоение переменной ###

```
#!/usr/bin/perl
$a = 10;
print "Variable a = $a\n";
```

### Присвоение скалярной переменной ###

```
#!/usr/bin/perl
$age = 35; # Assigning an integer
$name = "Tony Stark"; # Assigning a string
$pi = 3.14; # Assigning a floating point
```

### Простые скалярные операции ###

```
#!/usr/bin/perl
$constr = "hi" . "perl";# Concatenates two or more strings.
$add = 40 + 10; # addition of two numbers.
$prod = 4 * 51;# multiplication of two numbers.
$connumstr = $constr . $add;# concatenation of string and number.
```
