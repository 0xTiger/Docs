## .NX вариант №

Файл NX — это файл видеоигры, созданный для использования с виртуальной игровой консолью **LowRes NX**. Он написан на языке программирования BASIC и содержит исходный код игры, определяющий ее функционирование. Вы можете создавать свои собственные файлы NX для использования с приложением, которое можно загрузить, и пользователи смогут играть в игру.

Вы можете открыть файл NX в текстовом редакторе LowRes NX или в любом редакторе исходного кода, например Microsoft Visual Studio Code.

## Формат файла NX

Файл NX сохраняется на диске в виде обычного текстового файла, содержащего инструкции к игре. Каждый файл NX содержит исходный код игры, состоящий из нескольких записей ПЗУ картриджа. Эти записи ПЗУ содержат информацию о компонентах игры, такую как графика, карты уровней, аудио и другие данные, не относящиеся к исходному коду игры. Записи ПЗУ программы можно добавлять или редактировать с помощью программ редактирования NX, поставляемых в комплекте с LowRes NX.

## Как создать файлы NX?

### Создание файлов NX на ПК

Вы можете создавать файлы NX на ПК, используя:

* Текстовые редакторы, такие как Microsoft Notepad, Notepad++ или Github Atom.
* Редакторы исходного кода, например Microsoft VIsual Studio Code.

### Создание файлов NX на iOS

У LowRes NX есть версия для iOS, которую разработчики могут использовать для создания файлов NX из этого приложения.
