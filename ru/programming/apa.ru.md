## .APA вариант №

Файл с расширением .apa представляет собой файл проекта разработки программного обеспечения, созданный приложением RSView. Он содержит информацию, необходимую для создания приложений человеко-машинного интерфейса (HMI). Файлы APA хранятся в формате архивного файла. Его формат архивного файла помогает легко переносить их с одного компьютера на другой из-за меньшего размера файла. Для извлечения содержимого файлов APA можно использовать стандартные утилиты декомпрессии. Файлы APA можно открыть с помощью программного обеспечения Rockwell Automation RSView Studio или RSView ME.

## Формат файла APA — дополнительная информация

Диспетчер приложений RSView сохраняет файлы .apa по умолчанию в папке C:\Documents and Settings\All Users\Documents\RSView Enterprise\ME\Archives в Windows XP. Однако вы можете сохранить файл в любой каталог в соответствии с требованиями.
