## .YPR вариант №

Файл YPR — это файл проекта, созданный средой визуального программирования BYOB (Build Your Own Blocks). Он содержит информацию о визуальном расположении блоков и рабочих процессах программирования между этими блоками. Вместе эти блоки определяют общие данные и логику прикладной программы. Файл проекта YPR содержит всю другую связанную информацию для сохранения и загрузки проектов. BYOB изначально был расширением программного обеспечения MIT Scratch (проект Lifelong Kindergarten Group в MIT Media Lab) и теперь разработан как [Snap](https://snap.berkeley.edu/about).

## Формат файла YPR — дополнительная информация

Файлы YPR сохраняются на диск как двоичные файлы, и подробности их внутреннего формата файлов неизвестны. Проекты BYOB — это веселое строительство, в котором блоки перетаскиваются для разработки рабочего процесса программирования.

### Типы данных YPR

Snap расширил первый класс списка Scratch, позволив создавать свои структуры данных (деревья, кучи, хеш-таблицы, словари и т. д.). Тип данных первого класса в языке программирования может быть:

* значение переменной
* член агрегата (массива, списка и т. д.)
* аргумент (ввод) процедуры
* значение, возвращаемое процедурой
* используется без имени (является значением переменной)
