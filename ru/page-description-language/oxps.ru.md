## .OXPS вариант № ##

Формат файла OXPS известен как спецификация Open XML Paper. Это язык описания страницы и формат документа. Microsoft является разработчиком этого формата. Формат файла OXPS очень хорошо знаком этим файлам PDF. Настройка файла OXPS такая же, как и формата XPS. Единственным отличием является описание OXPS как открытого формата. Расширение файла .oxps — это самая последняя версия формата файла XPS, разработанная Microsoft и Ecma International. Если говорить о спецификации, то файлы OXPS и XPS используются для публикации, совместного использования и размещения документов с фиксированным макетом. В качестве замены PDF-файлам формат файла OXPS может хранить текстовые и мультимедийные данные с функциями упорядочивания и характеристиками макета печати.

## Краткая история ##

Команда Microsoft выбрала Global Graphics для предоставления консультаций по XPS в 2003 году, и группа разработчиков тесно сотрудничала с командами Windows для разработки спецификации и эталонной архитектуры для нового формата. Принимая во внимание спецификации XPS, в июне 2007 года Ecma International создала Технический комитет 46 (TC46) для разработки стандарта на основе спецификации OpenXPS. Спецификация Open XML Paper, одобренная Ecma International, является стандартом Ecma (ECMA-388) на 97-й Генеральной Ассамблее, состоявшейся в Будапеште 16 июня 2009 г.
