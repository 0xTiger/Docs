## .SWF вариант №

SWF-файл — это файл анимации, созданный с помощью Adobe Flash. Он может содержать различные типы элементов, таких как текст, векторные изображения, растровые изображения, сценарии действий, объекты, такие как круги, линии, квадраты и прямоугольники, для создания анимации. Файлы SWF упорядочивают эти мультимедийные элементы в кадрах, которые могут воспроизводиться с разной частотой кадров в секунду (fps). SWF означает короткий веб-файл, но также известно, что он имеет формат Shockwave.

Приложения, которые могли *открывать SWF-файлы**, включали Adobe Flash Player (теперь снято с производства) и Eltima Elmedia Player.

## Формат файла SWF — дополнительная информация

Файлы SWF использовались для хранения на диске в виде двоичных файлов. Формат файла SWF использовался для разработки анимаций и игр, которые можно было встраивать в веб-сайты и воспроизводить независимо. Он также поддерживал видео и звуки, что давало разработчикам множество возможностей для создания интерактивных мультимедийных приложений. Файлы SWF можно воспроизводить в веб-браузерах с установленным Adobe Shockwave. Adobe Flash был прекращен в декабре 2020 года из-за недостатков и проблем с безопасностью.

## Краткая история формата файла SWF

Формат файла SWF был первоначально разработан FutureWave Software для отображения анимации с целью запуска в программном обеспечении проигрывателя в любой системе с более медленными сетевыми подключениями при сохранении небольшого размера файла. В декабре 1996 года компания Macromedia приобрела FutureWave и перешла на Macromedia Flash 1.0.

В 2005 году Macromedia была приобретена Adobe, которая объявила SWF как часть своего проекта с открытым исходным кодом в 2008 году. В том же году Adobe выпустила код для популярных в мире веб-движков, позволяющий им сканировать и индексировать файлы SWF. Однако, поскольку SWF-файлы стали стандартным форматом для публикации Flash-контента в Интернете, SWF был изменен и теперь означает малый веб-формат.

## Структура SWF-файла

Путь — это основной графический элемент в SWF, который представляет собой последовательность сегментов базовых элементов, начиная от простых линий и заканчивая кривыми Безье. Эти простые элементы также помогают создавать другие дополнительные примитивы, такие как кубы, эллипсы и даже тексты. Графические примитивы в SWF имеют сходство с графическими элементами других форматов, таких как SVG и MPEG-4 BIFS.

Формат также позволяет отображать списки и повторно использовать/переименовывать уже определенные элементы. Формат двоичного потока SWF можно сравнить с атомами QuickTime, которые аналогичны тегам, размеру и полезной нагрузке. Формат двоичного потока позволяет старым проигрывателям пропускать неподдерживаемый контент. Хотя исходные версии SWF были ограничены предложением векторной графики и изображений, поэтому новые версии также позволяют использовать аудио- и видеоконтент.

В версии 11 был представлен новый низкоуровневый 3D API проигрывателя Flash Player под названием Stage3D. Предполагалось, что этот API будет аналогом OpenGL или Direct3D. Stage3D определяет цвета на языке низкого уровня, который называется Adobe Graphics Assembly Language (AGAL). Ниже приведены несколько основных типов данных формата файла SWF.

### Координаты

Координаты XY в формате файла SWF хранятся как целые числа и измеряются в единицах, называемых твипами. Твип состоит из 1/20 логического пикселя. Логический пиксель и пиксель экрана совпадают при воспроизведении файла без масштабирования на 100%.

### Целочисленные типы и порядок байтов

Целые числа со знаком и без знака размером 8, 16, 32 и 64 бита разрешены в формате файла SWF. Порядок байтов с прямым порядком байтов используется для хранения целочисленных значений. Хотя внутри байтов, порядок битов хранится в формате big-endian. Все целочисленные значения должны быть выровнены по байтам. Целые числа со знаком представляются с использованием традиционных битовых шаблонов с дополнением до 2.

### Числа с фиксированной точкой

Формат файла SWF поддерживает два типа чисел с фиксированной точкой, т. е. 32- и 16-битные.

### Числа с плавающей запятой

SWF 8 и более поздние версии используют три типа чисел с плавающей запятой (FLOAT, FLOAT 16, DOUBLE), которые совместимы со стандартом IEEE 754 для типов с плавающей запятой.

### Закодированные целые числа

Один тип закодированного целого числа поддерживается в SWF 9 и более поздних версиях с переменным числом байтов.
