## .AVERY вариант №

Файл AVERY — это документ для печати, который содержит одну или несколько этикеток для печати. Он создается с помощью программного обеспечения Avery Design & Print или Design & Print Online. Метки внутри файла AVERY могут состоять из различных элементов, таких как текст, графика и информация о макете страницы. Эта информация используется полиграфическими компаниями для печати различных элементов дизайна, таких как визитные карточки, бирки с именами, вкладыши для папок, адресные этикетки, этикетки для дисков, шаблоны некрологов и поздравительные открытки.

## Формат файла AVERY — дополнительная информация

Файлы AVERY хранятся в двоичном формате. Их можно открыть с помощью программного обеспечения Avery Design & Print или Design & Print Online. Дизайны, созданные с помощью Design & Print, можно использовать для создания этикеток AVERY, которые могут служить ряду целей. Эти дизайны можно сохранить в виде файлов [PDF](/ru/pdf/), которыми можно поделиться с компанией Prints для печати.

### Печать файла AVERY

Файл AVERY можно сохранить в формате PDF или распечатать на любом стандартном принтере. Чтобы распечатать файл, откройте файл в программе Design & Print и используйте параметр «Печать» в меню «Файл», чтобы отправить файл на принтер для печати.
