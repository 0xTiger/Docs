## .INDD вариант №

Файл с расширением .indd — это файл проекта Adobe InDesign, который содержит информацию о макете страницы проекта. INDD означает документ InDesign. Эти файлы обычно хранят информацию о форматировании страниц, книги, журналы, газеты, листовки, брошюры или другие печатные материалы, такие как макеты мобильных и веб-страниц. Тип файла INDD является собственным форматом файла Adobe и форматом файла по умолчанию для InDesign. Этот тип файлов используют графические дизайнеры, журналисты, маркетологи и издатели, работающие с InDesign.

## Подробнее о формате файла INDD

Файлы INDD — это векторные файлы, которые можно масштабировать без потери разрешения. Они основаны на сложных графах, которые можно расширять бесконечно. Он упаковывает всю информацию о шрифтах и макетах для совместного использования с командой.

Файлы INDD идеально подходят для разработки привлекательного печатного и онлайн-контента. Файл INDD может занимать несколько страниц и может служить шаблоном для создания страниц, глав и публикаций. Когда ваш проект завершен, его можно опубликовать на онлайн-носителях без необходимости преобразования, архивирования или отправки по электронной почте.

## Можно ли преобразовать файл INDD в другие форматы?

Файлы INDD можно преобразовать в несколько других форматов файлов. Наиболее распространенным преобразованием является INDD в PDF. InDesign позволяет сохранить файл INDD в формате PDF для совместного использования с другими пользователями. Он также позволяет экспортировать файл INDD в JPG, [EPS](/ru/page-description-language/eps/), [EPUB](/ru/ebook/epub/), [SWF](/ru/page-description-language/swf/) , FLA, [HTML](/ru/web/html/), [XML](/ru/web/xml/) и IDML.

## Как создается файл INDD?

Файл Indd создается при создании нового проекта в [Adobe InDesign](https://www.adobe.com/products/indesign.html) и сохранении его на диск. Вы можете открыть файл проекта Indd, внести в него изменения и сохранить обновленный файл для внесения изменений.
