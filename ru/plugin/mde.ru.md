.MDE вариант №

Файл .MDE — это скомпилированная версия базы данных Access (.mdb или .accdb). Когда вы компилируете базу данных Access в файл .MDE, ее больше нельзя редактировать с помощью стандартных инструментов проектирования Access. Основная цель создания файла .MDE — распространение приложения базы данных без раскрытия исходного исходного кода.

## Формат файла MDE

Файл MDE создается путем компиляции кода VBA, форм, отчетов и других объектов. В результате создается двоичный формат файла MDE. Полученный файл .MDE можно распространить среди пользователей, которые могут запускать приложение, но не могут изменять дизайн или просматривать исходный код. Файл MDE на самом деле представляет собой скомпилированную версию [файла .MDA](/database/mda/). Распространение файлов .MDE может помочь защитить интеллектуальную собственность, предотвращая доступ пользователей к исходному коду или его изменение. Это также может повысить производительность по мере компиляции и оптимизации кода.
