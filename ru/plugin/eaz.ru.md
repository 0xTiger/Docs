## .EAZ вариант №

Файл EAZ представляет собой файл надстройки, используемый ArcGIS Explorer, бесплатным приложением, разработанным ESRI для исследования, визуализации и совместного использования карт. Он содержит сжатый файловый архив, включающий [XML file](/web/xml/), скомпилированный код и вспомогательные файлы, необходимые для надстройки. Эти файлы используются для расширения базовой функциональности программного обеспечения за счет включения новых кнопок, закрепляемых окон и других расширений.

## Формат файла EAZ — дополнительная информация

Файлы EAZ создаются с помощью ArcGIS Explorer SDK, комплекта разработки, предназначенного для создания пользовательских функций в ArcGIS Explorer. Эти файлы используют сжатие [ZIP](/compression/zip/) для эффективной упаковки своего содержимого. В основе архива находится файл **Addins.xml**, расположенный в корневом каталоге и служащий жизненно важным компонентом, в котором описываются и детализируются различные настройки, встроенные в файл EAZ.

Файл Addins.xml по существу действует как дорожная карта, предлагая информацию о конкретных улучшениях, модификациях и расширениях, которые файл EAZ вводит в ArcGIS Explorer. Благодаря этой комплексной структуре разработчики могут инкапсулировать и легко интегрировать новые функции, кнопки и закрепляемые окна, улучшая общую функциональность и удобство использования приложения ArcGIS Explorer.
