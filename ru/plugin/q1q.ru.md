## Что такое файл Q1Q?

Файл .q1q — это файл плагина Photoshop, разработанный Flaming Pear Software для создания 3D-планет. Он позволяет загружать в режиме реального времени данные об облаках, воздухе, текстуре, климате и множество других деталей. Он также позволяет экспортировать все карты текстур в многослойные файлы и создавать карты для 3D-анимации и создания карт.

## Формат файла Q1Q

Файлы плагина Q1Q сохраняются в собственном формате Flaming Pear, и сведения о его внутреннем формате файла недоступны публично. На следующем изображении показан пользовательский интерфейс плагина Q1Q при загрузке в Photoshop.

## Как открыть файл Q1Q?

Файлы Q1Q можно открывать с помощью **Adobe Photoshop 2023** и **Corel PaintShop Pro 2023**.
