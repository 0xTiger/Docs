## .RPLN вариант №

Файл RPLN — это файл плагина, используемый программой макетирования страниц **InDesign**, которая создает дизайн страницы, а также обеспечивает публикацию в печати. он содержит функции, необходимые для запуска, и находится в обязательном каталоге, где установлено программное обеспечение InDesign.

## Формат файла RPLN

Файлы RPLN сохраняются в собственном формате файлов InDesign, и дополнительная информация о внутренней файловой структуре файлов RPLN отсутствует.

## О Adobe InDesign

Adobe InDesign — это профессиональное программное обеспечение для настольных издательских систем, которое широко используется для создания печатных и цифровых носителей. InDesign поддерживает использование плагинов, представляющих собой дополнительные программные компоненты, которые можно добавлять для расширения функциональности программы.

## Как открыть файл RPLN?

Вы можете открывать файлы RPLN с помощью Adobe InDesign 2023.
