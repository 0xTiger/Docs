## .ACROPLUGIN вариант №

Файл с расширением .acroplugin — это файл плагина, используемый Adobe Acrobat. Он используется для добавления функциональности к существующим возможностям приложения. Плагин поставляется в комплекте с Adobe Acrobat, макетом страницы и программой публикации [PDF file](/pdf/). Как и другие плагины, файл ACROPLUGIN расширяет существующий функционал, предлагаемый приложением.

## Формат файла ACROPLUGIN — дополнительная информация

Файл ACROPLUGIN явно используется Adobe Acrobat для расширения существующих функций. Подробности внутреннего формата файла недоступны для справки разработчикам.

## Как открыть файл ACROPLUGIN?

Вы можете открыть файл ACROPLUGIN с помощью Adobe Acrobat.
