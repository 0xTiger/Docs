## .INX вариант №

Файл INX — это файл плагина, используемый редактором векторной графики с открытым исходным кодом **Inkscape**. Он содержит информацию о программе, включая название и расположение функций в меню. Файл INX также определяет сценарий для расширения, который включает такую информацию, как тип сценария, интерпретатор, местоположение и переменные аргументы.

## INX-формат файла

Файлы INX расширяют функциональность Inkscape за счет включения таких функций, как импорт файлов и графические фильтры. Эти файлы, хранящиеся в формате XML, обычно находятся в каталоге \​extensions\​ установки Inkscape.

## Как открыть файлы INX?

Вы можете открыть файлы INX с помощью программы **Inkscape**.
