## .RBZ вариант №

Файл RBZ — это файл плагина, используемый программой моделирования **Trimble SketchUp** для автоматизированного проектирования (САПР). Это сжатый ZIP-файл, содержащий внутри один или несколько сценариев Ruby (файлы .rb). Именно поэтому тип файла — RBZ (RB+Z). Файлы RBZ легко распространять и не передаются по сети из-за их небольшого размера в результате сжатия.

## Формат файла RBZ — дополнительная информация

Файлы RBZ представляют собой сжатые ZIP-файлы, содержащие файл сценария Ruby. Он сжимается с помощью наиболее часто используемого метода сжатия **DEFLATE**, описанного в IETF **RFC 1951**.

## Как открыть файл RBZ?

Вы можете открывать файлы RBZ с помощью Trimble SketchUp как в Windows, так и в macOS.

В дополнение к этому, поскольку файлы RBZ представляют собой сжатые ZIP-файлы, вы можете извлечь их с помощью стандартных утилит извлечения ZIP, переименовав его расширение с .rbz на .zip.
