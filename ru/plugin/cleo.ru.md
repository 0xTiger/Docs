## .CLEO вариант №

Файл .cleo — это файл плагина, используемый как часть библиотеки CLEO для серии игр GRand Theft Auto. Он используется для выполнения модификаций, которые меняют или расширяют игровой процесс. Файл CLEO хранит библиотеку в формате DLL, добавляя полезные команды сценариев (коды операций) для работы с внешними файлами для изменения игровой памяти. Существуют разные версии CLEO для GTA III, GTA Vice City и GTA San Andreas.

## Формат файла CLEO

CLEO 4 позволяет создавать новые коды операций, используя исходный код CLEO SDK. Три наиболее распространенных примера таких файлов плагинов включают:

 * IniFiles.cleo — коды операций для INI-файлов;
 * FileSystemOperations.cleo — коды операций для файлов и папок.
 * IntOperations.cleo — коды операций для логических операций (И, ИЛИ, XOR и т. д.).

Плагины устанавливаются аналогично скриптам. Чтобы установить плагин CLEO, вам необходимо скопировать файл .CLEO в папку CLEO.

## О КЛЕО

CLEO — чрезвычайно популярный расширяемый плагин для серии игр Grand Theft Auto от Rockstar Games, позволяющий использовать тысячи уникальных модов, изменяющих или расширяющих игровой процесс.
