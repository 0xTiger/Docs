## .AMXX вариант №

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. Это платформа сценариев, которая позволяет администраторам серверов настраивать и расширять функциональность своих серверов. AMXX используется вместе с системой управления плагинами MetaMod, которая позволяет нескольким плагинам одновременно работать на сервере без конфликтов. Функции, предлагаемые плагином AMXX, включают команды администратора, изменения игрового процесса и управление сервером.
