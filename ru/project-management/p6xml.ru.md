## .P6XML вариант № ##

Расширение файла P6XML имеет отличную читабельность, адаптивность и широкое использование, что делает его идеальным форматом. Primavera P6 Professional позволяет экспортировать и импортировать как файлы [XER](/ru/project-management/xer), так и файлы XML. XML означает **Расширяемый язык разметки**. Этот общий формат позволяет пользователям обмениваться информацией о проектах между базами данных Primavera P6. Данные проекта можно легко сохранить в текстовом файле через файл XML; это упрощает обмен данными для многочисленных программ. Пользователь Primavera P6 может легко обмениваться такими данными и управлять ими.

## Преимущества формата файла P6XML ##

* Пользователи Primavera могут легко определить, какие базовые показатели экспортируются и импортируются с помощью XML-файла.
* Экспорт XML может содержать все подробные планы действий проекта
* Любой пользователь, использующий Primavera 6, который не может редактировать, добавлять или удалять определенные элементы глобальных данных, такие как коды действий, календари и ресурсы, не сможет их импортировать.
* В дополнение к общим настройкам и настройкам профиля безопасности проекта, используемым Primavera, элементы данных, которые будут влиять или изменять данные в базе данных, не будут импортированы.

## Экспорт файла XML из Primavera P6 ##

Для экспорта проектов P6 Professional и данных проекта в файл P6XML используйте Oracle Primavera Cloud. Вы можете загрузить файл во время его формирования из настольного приложения P6 Professional и импортировать его в Primavera Cloud. Перед бесперебойной работой процесса экспорта вам следует определить, подключены ли вы к базе данных P6 Professional или P6 EPPM. Некоторые функции процесса экспорта могут отличаться в зависимости от вашей базы данных. Имея специализированные настройки P6 с базой данных P6 Proficient, процесс экспорта выполняется локально в вашей системе. Имея базу данных P6 Professional, вы можете легко экспортировать и открывать XML-файл P6 из настроек вашей системы.

Вот несколько шагов, описывающих процесс экспорта:

* Открытые желания проекта Primavera P6
* После этого выберите Экспорт на вкладке Файл
* Выберите «Primavera P6 – (XML)». После этого нажмите Далее
* Дважды щелкните проект, который вы хотите экспортировать
* Выберите усечения для пути, по которому вы хотите поместить экспортированный файл, и нажмите «Готово».
* Выберите вариант, как вы хотите сохранить файл
* Если макеты уровня проекта не нужны для файла экспорта, необходимо снять флажок «Экспортировать все макеты уровня проекта».
* Выберите «Готово», после чего вы получите подтверждение экспорта XML

### Важные советы ###

* При каждом подключении к базе данных P6 EPPM администратор должен передать URL-адрес источника контента в поле URL-адрес P6 на странице «Общие» параметров приложения в P6.
* Удобно экспортировать несколько проектов в один файл XML
* Глобальные данные, выделенные для проекта, экспортируются
* Удобно экспортировать XML-файлы, не имея доступа ко всем ресурсам
  


## Проблемы с открытием файла P6XML? ##

Вот некоторые распространенные проблемы, которые могут возникнуть и привести к неправильной работе формата P6XML:

* Отсутствие сопутствующего ПО
* Поврежденный файл
* Зараженный файл из-за вируса
* Описание P6XML случайно удалено из реестра Windows
* Нет прав доступа в системе для открытия файлов
* Устаревший диск в вашей системе
* Расширение файла переименовано
