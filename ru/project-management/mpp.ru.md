## .MPP вариант №

Файл с расширением .mpp представляет собой файл данных Microsoft Project, в котором интегрированно хранится информация, связанная с управлением проектами. Microsoft разработала этот собственный формат файла, чтобы сделать его совместимым с Microsoft Project (MSP), их программным продуктом для управления проектами. Помимо MPP, MSP может работать с XML-схемой проекта. Многие приложения и API позволяют преобразовывать MPP в другие форматы файлов. Теперь у Microsoft есть онлайн-сервер Project Server, на который можно загружать файлы управления проектами для совместной работы нескольких пользователей.

## Формат файла MPP ##

Microsoft разработала и опубликовала MPP в виде двоичного формата файла, и до настоящего времени Microsoft не обнародовала его спецификации. Следовательно, внутренняя файловая структура и детали недоступны разработчикам. В Интернете доступно несколько API, которые поддерживают чтение/запись файлов MPP, но 100% точность никогда не достигается. Со временем Microsoft представила несколько функций в своем приложении для управления проектами, а также функции формата файла MPP.

Файл MPP использует следующие типы MIME:

* приложение/vnd.ms-проект
* приложение/msproj
* приложение/msproject
* приложение/x-msproject
* приложение/x-ms-проект
* приложение/x-dos_ms_project
* приложение/mpp
* zz-приложение/zz-winassoc-mpp

## Проблемы с открытием файла MPP? ##

Вот список некоторых распространенных проблем, которые могут возникнуть и привести к неправильной работе формата MPP:

* Отсутствие сопутствующего ПО
* Поврежденный файл
* Зараженный файл из-за вируса
* Нет прав доступа в системе для открытия файлов
* Устаревший диск в вашей системе
* Расширение файла переименовано
