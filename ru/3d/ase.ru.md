## .ASE вариант №

Файл с расширением .ase представляет собой формат файла Autodesk ASCII Scene Export, представляющий собой ASCII-представление сцены, содержащее 2D- или 3D-информацию при экспорте данных сцены с помощью Autodesk. Autodesk предоставляет опции для включения компонентов сцены при экспорте данных сцены. Вы можете включить определение сетки вместе с информацией о вершинах и гранях, описанием материала, данными анимации преобразования для объектов, полным определением сетки для каждых n кадров, данными анимации для камер и источников света и настройками соединения IK.

## Формат файла ASE — дополнительная информация

Файлы ASE представляют собой текстовые файлы, хранящиеся в формате ASCII, которые можно открыть в любом текстовом редакторе. Файл ASE может содержать следующие типы информации, предоставленной Autodesk.

### Параметры вывода

* «Mesh Definition» — экспортирует определение каждой сетки, включая информацию о вершинах и гранях для геометрических объектов. Кроме того, включение этого параметра включает элементы в групповом окне Mesh Options, описанные ниже.
* «Материалы» — включает описание материала. Если материал не назначен объекту, экспортируется цвет его каркаса. Включены все уровни дерева материалов, поэтому может быть много текста.
* «Ключи анимации преобразования» — включает данные анимации преобразования для объектов. Если объектом является целевая камера или прожектор, это будет включать данные целевой анимации.
* «Анимированная сетка» — экспортирует полное определение сетки для каждых n кадров. Частота определяется счетчиком вывода контроллера, описанным ниже. Каждый блок содержит одну и ту же информацию, указанную в групповом окне Mesh Options, описанном ниже. Включение этого параметра может привести к созданию огромного файла даже для небольших сцен.
* «Анимированные настройки камеры/освещения» — экспортирует данные анимации для камер и источников света, такие как цвет, интенсивность, затухание, смещение карты и т. д. Выводит блок каждые n кадров, как указано счетчиком вывода контроллера.
* «Обратные кинематические соединения» — экспортирует настройки соединения IK в ветви «Иерархия».

### Типы объектов

Элементы здесь позволяют указать, какую категорию объектов вы хотите включить в вывод. Вы можете включать геометрические объекты, фигуры, камеры, источники света и вспомогательные объекты.

* Геометрический
* Формы
* Камеры
* Огни
* Помощники

### Параметры сетки

* `Mesh Normals` - Экспорт нормалей граней и вершин. Сначала указана нормаль грани, за ней следуют нормали трех вершин, поддерживающих грань. Включение этого приводит к гораздо большему файлу.
* «Mapping Coordinates» — экспортирует список сопоставленных вершин и граней в соответствии со структурами TVert и TVFace, описанными в комплекте для разработки программного обеспечения 3ds Max. Если объект использует сопоставление лиц, экспортируется список карт лиц, содержащий координаты UVW для каждого лица.
* «Цвета вершин» — экспортирует цвета вершин.

### Выход контроллера

* «Использовать ключи» — экспортирует значения ключей. Если контроллер не использует ключи, используется метод Force Sample. В случае контроллеров преобразования параметр «Использовать ключи» работает только в том случае, если все контроллеры преобразования являются линейными/TCB или Безье. Если одна из дорожек преобразования использует другой тип контроллера, то метод Force Sample используется для всех дорожек преобразования.
* `Force Samples` - выборки значений контроллера на основе частоты, указанной в Frames per Sample Controller.
