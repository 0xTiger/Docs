## .DRC вариант №

Файл с расширением .drc представляет собой сжатый формат 3D-файла, созданный с помощью библиотеки Google Draco. Google предлагает Draco в качестве библиотеки с открытым исходным кодом для сжатия и распаковки трехмерных геометрических сеток и облаков точек, а также для улучшения хранения и передачи трехмерной графики. Он кодирует входные данные и сохраняет их как файл .drc. На принимающей стороне API считывает файлы .drc и может выводить их в виде файлов [PLY](/ru/3d/ply/) или [OBJ](/ru/3d/obj/). Сжатый выходной файл позволяет пользователям быстрее загружать приложения и обеспечивает быструю загрузку 3D-графики в браузерах.

## Формат файла DRC

Информация о формате файла для DRC недоступна публично. Google разместил библиотеку Draco на [GitHub](https://github.com/google/draco/tree/master/docs) для справки разработчика. Он предоставляет кодировщики и декодеры для создания и преобразования файлов DRC.
