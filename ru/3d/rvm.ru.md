## Что такое файл RVM?

Файлы данных **RVM** относятся к AVEVA PDMS. Файл **RVM** представляет собой файл проекта модели AVEVA Plant Design Management System. Система управления проектированием предприятий AVEVA (PDMS) — самая популярная система 3D-проектирования, использующая технологию, ориентированную на данные, для управления проектами. Доступно множество приложений для открытия и преобразования файлов RVM в другие форматы, такие как [3DS](/ru/3d/3ds/).

## Формат файла RVM

Не так много подробностей можно найти о формате файла RVM, кроме того, что он может быть сохранен как в двоичном формате, так и в формате ASCII. Файл RVM поддерживает такие объекты, как:

* Вся геометрия
* Атрибуты, хранящиеся в группах
* Текстуры (через файл RVS)
* Камеры и треки камер (через файл RVS)
* Обрезать плоскости (через файл RVS)
* Знаки (через файл RVS)
* Теги (через файл RVS)
* Метки (через файл RVS)
* Прозрачность (через файл RVS)
* Точки происхождения PDMS
