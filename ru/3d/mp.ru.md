## .MP вариант №

Файл MP — это файл проекта, созданный с помощью Personal Learning Edition (PLE) программы 3D-моделирования **Maya**. Он поддерживается только PLE-версией Maya и предназначен исключительно для некоммерческого использования. Для этой цели версия Maya PLE включает водяной знак в экспортируемый файл. Maya PLE предоставляет ограниченную функциональность и может быть установлена только на персональных компьютерах. Файл MP содержит всю информацию о проекте, созданном с помощью PLE, и его можно открыть с помощью Autodesk Maya PLE.
