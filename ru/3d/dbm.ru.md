## .DBM вариант №

Файл DBM (Daz Brick Material) представляет собой сетевой файл Shader Mixer, созданный с помощью приложения для моделирования трехмерных персонажей, **DAZ Studio**. Он описывает материал режима 3D, т.е. поверхность/смещение/внутренний объем. DAZ позволяет создавать собственные шейдеры, соединяя ряд блоков вместе. Это объединяется как файл DBM, который формирует шейдерную сеть. Это файл DBM, который определяет, как затеняется материал.

Приложения, которые могут **открывать файлы DBM**, включают DAZ 3D DAZ Studio 4.

## Формат файла DBM

Файлы DBM сохраняются на диск в формате [XML](/ru/web/xml/).
