## Об обмене GLB на USDZ

GLB — популярный формат двоичных 3D-файлов, который представляет 3D-модели в формате передачи GL, т. е. ([glTF](/ru/3d/gltf/)). Файл .usdz по сути представляет собой несжатый и незашифрованный ZIP-архив, разработанный для формата файла [USD](/ru/3d/usd/) (универсальное описание сцены). Этот архив удобно инкапсулирует и служит прокси-сервером для различных файлов в альтернативных форматах (например, текстур и анимаций), которые не входят в него. Эти файлы могут быть легко использованы средой выполнения USD, что устраняет необходимость в каком-либо процессе распаковки.

В Интернете доступно множество приложений, которые можно БЕСПЛАТНО использовать для конвертации [GLB](/ru/3d/glb/) в [USDZ](/ru/3d/usdz/). Если вы разработчик приложений и хотите добавить функциональность GLB в USDZ в своей .NET или Java, вы можете использовать API Aspose.3D для достижения этой цели.
