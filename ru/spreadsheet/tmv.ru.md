## .TMV вариант №

Файл TMV представляет собой файл электронной таблицы, созданный с помощью программного обеспечения [TimeMap] (https://www.lexisnexis.com/en-us/products/casemap.page) от LexisNexis. TimeMap позволяет создавать временные шкалы с указанной датой и временем фактов. При вводе новых фактов программа создает для них информационные поля и размещает их в соответствии с временной шкалой. Проект TMV может состоять из полей фактов, текстовых полей, и текст внутри них может быть отформатирован с использованием различных шрифтов. Файлы TMV можно открыть с помощью программного обеспечения TimeMap. LexisNexis также предоставляет онлайн-инструмент для совместной работы в облачной среде.

## Формат файла TMV

Файлы TMV хранятся на диске в собственном формате файлов CaseMap, спецификации которого недоступны в Интернете. CaseMap была приобретена LixesNexis.

## Использованная литература ##

* [LexisNexis CaseMap] (https://www.lexisnexis.com/en-us/products/casemap.page)
