## .NB вариант №

Файл с расширением .nb представляет собой формат файла записной книжки Wolfram, в котором инструкции для математических инструкций сохраняются в текстовом файле. Он может содержать множество различных типов данных, таких как живые вычисления, произвольные динамические интерфейсы, ввод с полным набором текста, ввод изображений, автоматические аннотации кода, полный высокоуровневый программный интерфейс и тысячи тщательно организованных функций и опций. Текстовые инструкции представляют собой ввод и вывод Mathematica, которые генерируются и обновляются по мере того, как операторы ввода помещаются в файл.

## Формат файла Wolfram Notebook NB — дополнительная информация

Файлы Wolfram Notebook NB сохраняются в текстовом формате, удобном для чтения человеком. Содержимое записной книжки упорядочено по разделам в виде обычного текста, каждый из которых представлен группами ячеек, как в электронной таблице. Диапазон этих групп определяется скобкой в конце каждой ячейки. Стиль, назначенный ячейке, определяет ее роль в записной книжке, как подробно описано ниже.

### Блокноты как Wolfram Language

Когда предполагается сохранить записную книжку, состоящую из математических инструкций, для выполнения ядром Wolfram Language Kernal, ячейкам в документе автоматически назначается стиль текста «Ввод». Это говорит ядру рассматривать их как инструкции, которые выполняются, когда пользователь нажимает комбинацию клавиш Shift+Return. Блокнот Mathematica показан в следующем примере.

### Блокноты как документы

Блокноты Mathematica могут быть в формате документа, чтобы показать, что вы видите, что вы получаете (WYSIWYG). Эти документы аналогичны документам, просматриваемым на экране или напечатанным на бумаге, и являются интерактивными. Для этого ячейке назначается «Стиль текста» для отображения содержимого без выполнения операторов ядром.

## Экспорт блокнотов Mathematica

Блокноты Mathematica можно экспортировать во множество различных форматов файлов, таких как PDF, графика, ГИС, сжатые файлы и электронные таблицы.
