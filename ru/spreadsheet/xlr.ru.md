## .XLR вариант №

Файл с расширением .xlr представляет собой формат файла электронной таблицы Microsoft Works, в котором данные хранятся в строках и столбцах. Microsoft Works был разработан для использования в качестве первоначального пакета для повышения производительности и оставался функциональным с 1987 по 2009 год. Он включал текстовый процессор, электронные таблицы и систему управления базами данных. Позже он был прекращен после того, как Microsoft Office был представлен с более богатыми функциями.

## Формат файла XLR — дополнительная информация

Файлы XLR были сохранены в виде двоичных файлов без каких-либо общедоступных подробностей о спецификациях формата файла. Поэтому у разработчиков нет возможности анализировать его файловую структуру и программно писать приложения для чтения/записи в эти файлы. Электронная таблица Works использует тот же формат файлов Excel для хранения данных в файлах .xlr.
