## .XLHTML вариант №

Файл с расширением .xlthtml представляет собой [HTML](/ru/web/html/) версию файлов шаблонов [XLT](/ru/spreadsheet/xlt/) и был создан с использованием Microsoft Excel в первые дни. Этот формат больше не поддерживается Excel и был заменен экспортом данных в форматы файлов [MHT](/ru/web/mht/) и [MHTML](/ru/web/mhtml/). Основная идея файла XLTHTML заключалась в том, чтобы позволить пользователям просматривать файлы шаблонов XLT в веб-браузере вместо необходимости установки Excel в системе. Файлы, экспортированные в формате XLTHTML, можно открывать в браузерах.

## Формат файла XLTHTML — дополнительная информация

Файлы XLTHTML использовали формат файла HTML для сохранения шаблонов XLT для открытия в веб-браузере. Сам XLT основан на формате двоичных файлов Excel, который был представлен в версии Excel 97-2003. Эти файлы можно использовать в качестве начальных файлов с определенным форматированием для создания обычных документов для заполнения данных электронных таблиц. Просто для справки: XLT основан на двоичных [спецификациях формата файлов XLS](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx), которые задокументированы Microsoft для справка разработчика.
