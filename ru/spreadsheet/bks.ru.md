## .BKS вариант № ##

Файл BKS — это файл резервной копии [Microsoft Works](https://en.wikipedia.org/wiki/Microsoft_Works), в котором хранится последнее рабочее состояние файла электронной таблицы. Это сохраняет последнее допустимое состояние электронной таблицы для повторной загрузки в случае, если исходный документ поврежден или не может быть открыт по какой-либо причине. BKS похож на формат файла [WPS](/ru/word-processing/wps/), который содержит содержимое документа, похожее на документ Microsoft Word [DOC](/ru/word-processing/doc/). Поддержка Microsoft Works прекращена, и файлы BKS больше не поддерживаются.

## Формат файла BKS

Файлы BKS хранятся на диске в виде двоичных файлов, и спецификации их формата файлов недоступны для общественности.
