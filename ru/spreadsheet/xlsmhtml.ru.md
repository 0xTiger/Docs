## .XLSMHTML вариант №

Файл с расширением .xlsmhtml представляет собой [HTML](/ru/web/html/) версию исходного файла XLS, которая экспортируется из Microsoft Excel для просмотра данных в браузере. Excel уже давно не поддерживает этот формат и больше не поддерживает его. Экспортированный файл XLSMHTML соответствовал формату файла XLS, и его можно было открывать и просматривать в веб-браузерах. Excel обеспечивает экспорт данных электронной таблицы в форматы файлов [MHT](/ru/web/mht/) и [MHTML](/ru/web/mhtml/), которые сейчас широко используются.

## Формат файла XLSMHTML — дополнительная информация

Файлы XLSMHTML были сгенерированы с помощью HTML-кода для представления данных электронной таблицы в формате, аналогичном XLS. Код HTML внутри файла XLSMHTML устроен таким образом, что выходной файл выглядит аналогично исходному файлу при открытии в веб-браузере. Спецификации формата файла XLSMHTML как таковые не публикуются Excel, и такие ссылки не доступны разработчикам для понимания и разработки приложений для него.
