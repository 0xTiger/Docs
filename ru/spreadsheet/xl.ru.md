## .XL вариант №

Файл с расширением .xl представляет собой файл электронной таблицы Excel, аналогичный [XLS](/ru/spreadsheet/xls/), в котором данные хранятся в строках и столбцах. Это был более ранний формат файла, который больше не поддерживается Excel. Данные, хранящиеся в файлах XL, можно экспортировать в другие форматы, например CSV. Их также можно преобразовать в PDF с помощью стандартного метода Excel «Сохранить как». Файлы XL можно было открыть только с помощью Microsoft Excel.

## Формат файла XL

Файлы XL сохраняются как двоичные файлы, и их внутренние спецификации формата файлов нигде не документированы. Однако общий формат двоичного файла для [Спецификаций формата файлов XLS] (https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) также может быть рассмотрен, если кто-то пытается написать фрагмент кода для обратного проектирования этих файлов.
