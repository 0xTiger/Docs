## .SXC вариант № ##

Формат файла SXC (Sun XML Calc) принадлежит офисному пакету OpenOffice.org. Этот формат обычно отвечает потребностям пользователей в электронных таблицах, поскольку он представляет собой формат файла электронной таблицы на основе XML. Формат SXC поддерживает формулы, функции, макросы и диаграммы, а также DataPilot, что является невероятной функцией, поскольку она автоматически индивидуализирует и предоставляет сводки необработанных импортированных данных. Файлы, созданные с помощью этого программного обеспечения, сохраняются с расширением .sxc.


## Формат файла SXC ##

Файлы с расширением .sxc можно открыть с помощью Microsoft Excel, программы для работы с электронными таблицами из пакета Microsoft Office. Есть и другое программное обеспечение, которое может открывать файлы с расширением .sxc, например, Libre Office, StarOffice и т. д.

Тип содержимого для этого формата:

* приложение/vnd.sun.xml.calc
* приложение/vnd.sun.xml.calc.template

## Совместимость файлов ##

Файлы SXC совместимы с калькулятором Apache Open Office, и данные можно экспортировать в файл Microsoft Excel и другие форматы.

## Обработка файлов ##

Команды ImportMatrix и ExportMatrix могут читать и записывать файл SXC. Если источником является файл SXC, а целью также является файл SXC, это приводит к тому, что **f** рассматривается как файл электронной таблицы Sun XML Calc. И Import, и Export Matrix поддерживают этот формат, который определяется автоматически, если формат не указан и имя файла оканчивается на .sxc.

Чтобы открыть файл SXC, необходима соответствующая программа, связанная с файлом. Всякий раз, когда необходимо открыть файл с расширением .sxc, убедитесь, что он не содержит ошибок и не поврежден. Любая ошибка в файле может быть легко устранена пользователем, и в большинстве случаев это не требует технической поддержки.
