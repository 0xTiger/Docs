## .DEF вариант №

Файл DEF — это файл данных, созданный в [формате электронной таблицы] (/ru/spreadsheet/) приложением базы данных SmartWare II, разработанным Informix Software. Программное обеспечение II имело четыре модуля, включая электронные таблицы с графикой, текстовый процессор, средства связи и базу данных. База данных SmartWare II — это автономная база данных, которую также можно интегрировать с другими приложениями. Запросы можно выполнять с данными, которые можно извлечь и сохранить в формате файла DEF. Файлы электронных таблиц DEF можно открывать с помощью программного обеспечения SmartWare, поддержка которого в настоящее время прекращена.
