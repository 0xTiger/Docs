## .PMVX вариант №

Файл с расширением .pmvx представляет собой файл шаблона электронной таблицы, созданный с помощью программы электронных таблиц [PlanMaker](https://www.softmaker.de/softmaker-office-planmaker), которая входит в состав Softmaker Office. Данные хранятся в строках и столбцах в файлах PMVX так же, как это делается в Microsoft Excel. Файлы PMVX хранят предопределенные настройки в качестве шаблона, который можно использовать для создания файлов PMD и PMDX после заполнения в нем данных. [XLSX](/ru/spreadsheet/xlsx/) — формат файлов по умолчанию для PlanMaker, и их можно открывать непосредственно в Microsoft Excel. Вы также можете редактировать файлы [XLS](/ru/spreadsheet/xls/) и создавать файлы [PDF](/ru/pdf/) с помощью PlanMaker.

## Формат файла PMVX

Файлы PMVX хранятся на диске в собственном файловом формате PlanMaker, и их внутренние файловые структуры неизвестны. Что касается сохранения файлов PMVX, он сохраняет данные в формате Open XML в виде файлов XLSX.
