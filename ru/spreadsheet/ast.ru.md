## .AST вариант №

Файл AST представляет собой файл шаблона электронной таблицы, созданный с помощью программного обеспечения Ability Spreadsheet, компонента электронных таблиц пакета Ability Office. Эти файлы шаблонов используются для создания начального набора шаблонов электронных таблиц, которые можно использовать для создания файлов [AWS](/ru/spreadsheet/aws/). Файлы AST можно открыть с помощью программного обеспечения для работы с электронными таблицами Ability.

## Формат файла AST — дополнительная информация

Файлы AST сохраняются на диск в виде двоичных файлов и могут быть открыты в программе Ability Spreadsheet. Хотя спецификации формата файлов AST полностью совместимы с форматами файлов Microsoft XLS и XLSX, они недоступны для разработчиков. Файлы AST можно сохранять на диск в форматах файлов [PDF](/ru/pdf/), а также [XPS](/ru/page-description-language/xps/).
