## .GNM вариант №

Файл GNM представляет собой файл электронной таблицы, созданный с помощью [Gnumeric](https://en.wikipedia.org/wiki/Gnumeric), которая является программой для работы с электронными таблицами и входит в состав [GNOME](https://www.gnome.org/) Desktop. Программного обеспечения. Он хранит данные электронной таблицы в строках и столбцах и обеспечивает поддержку диаграмм и других функций электронных таблиц. Файлы GNM сохраняются на диск в формате [XML](/ru/web/xml/) (аналогично XLSX) и сжимаются с использованием [GZIP](/ru/compression/gz/). Цель Gnumeric состояла в том, чтобы предоставить приложение для работы с электронными таблицами с открытым исходным кодом, которое в конечном итоге станет альтернативой Microsoft Excel.

## Формат файла GNM

Файлы GNM представляют собой сжатые XML-файлы, которые можно извлечь с помощью стандартных утилит распаковки, таких как WinZIP. Несмотря на множество функций, Gnumeric не поддерживает макросы и сводные таблицы.

## Функции, поддерживаемые в файлах GNM

Поскольку Gnumeric поддерживает несколько различных форматов файлов, он может конвертировать исходные файлы GNM в [CSV](/ru/spreadsheet/csv/), [XLSX](/ru/spreadsheet/xlsx/), Microsoft Works (.wks), [HTML](/ru/ web/html/), [LaTex](/ru/word-processing/latex/) и [Lotus 1-2-3](/ru/spreadsheet/123/).

## Gnumeric на Github

Разработка Gnumeric с открытым исходным кодом размещается и поддерживается в [официальном репозитории Gnumeric Github](https://github.com/GNOME/gnumeric).
