## .DEX вариант №

Файл DEX — это файл электронной таблицы, созданный с помощью Microsoft Excel. Он похож на файлы [XLSX](/ru/spreadsheet/xlsx/), но имеет двоичный формат. Данные хранятся в строках и столбцах в файле DEX. Хотя файлы DEX также представляют собой файлы Microsoft Excel, они менее популярны. Для сравнения, формат файла XLSX является выбором по умолчанию для сохранения книг Excel. Файлы DEX были помечены как устаревшие и больше не поддерживаются Microsoft Excel.
