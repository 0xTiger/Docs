Файл **электронной таблицы** содержит данные в виде строк и столбцов. Файл электронной таблицы можно сохранить в нескольких различных форматах, каждый из которых имеет свое расширение файла для уникального представления. Данные хранятся в ячейках либо в простой форме, такой как текстовая строка, числа, дата, валюта и т. д., либо в виде формул, которые изменяют значение ячейки при изменении значений, на которые ссылаются ячейки.

Общие расширения файлов электронных таблиц и их форматы файлов включают **[XLSX](/ru/spreadsheet/xlsx/)** (электронная таблица Microsoft Excel Open XML), **[ODS](/ru/spreadsheet/ods/)** (электронная таблица OpenDocument) и **[XLS](/ru/spreadsheet/xls/)** (формат двоичного файла Microsoft Excel).

Есть вопросы, связанные с форматами файлов электронных таблиц? Присоединяйтесь к нашему сообществу [форумы](https://forum.fileformat.com/c/spreadsheet/6), чтобы воспользоваться знаниями, которыми поделились эксперты по форматам файлов.
