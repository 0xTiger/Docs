## .DFG вариант №

Файл с расширением .dfg представляет собой файл электронной таблицы, созданный Interscape Inc. для программного обеспечения Interscape Data Flask. Данные заполняются в файлах DFG в строках и столбцах. Data Flask — это программа для работы с электронными таблицами, предназначенная для хранения типов данных, источника и ключевой информации. Он может переносить таблицы и индексы, изменяя только строку подключения. Он предлагает такие функции, как командная строка VBScript, формулы VBScript и макросы, а также сценарии SQL для обновления и переноса данных. Файлы DFG сохраняются на диск как двоичные файлы и могут быть открыты с помощью программного обеспечения Interscape Data Flask.

## Формат файла DFG — дополнительная информация

Файлы DFG сохраняют данные электронной таблицы в виде двоичных файлов на диск. Однако внутренняя структура данных и детали формата этих файлов неизвестны.
