## .CTS вариант №

Файл CTS представляет собой файл электронной таблицы, созданный с помощью мультиплатформенной программы ввода и организации данных с открытым исходным кодом **TreeSheets**. Он содержит иерархические данные, введенные в различные форматы внутри электронной таблицы, такие как заметки, списки, календари, списки дел, заметки, интеллект-карты и ряд других форматов. Файлы CTS можно открывать только с помощью TreeSheets, который является легким, быстрым и работает в Windows, Linux и macOS.

## Формат файла CTS — дополнительная информация

TreeSheets сохраняет файлы CTS в собственном проприетарном формате. Спецификации формата файла CTS не доступны публично, поэтому файлы CTS можно открывать только в программном обеспечении TreeSheets.

### Экспорт файлов CTS TreeSheets в другие форматы

Пользователи TreeSheets могут экспортировать электронные таблицы CTS в несколько различных форматов файлов. Файлы CTS можно экспортировать в форматы файлов [HTML](/ru/web/html/), [JSON](/ru/web/json/), [CSV](/ru/spreadsheet/csv/) и [Image](/ru/image/). .

## О программном обеспечении TreeSheets

TreeSheets — это программа с открытым исходным кодом, доступная для нескольких платформ. Он позволяет создавать иерархические электронные таблицы и дает вам свободу помещать все типы данных в один файл электронной таблицы. Преимущество использования TreeSheets заключается в том, что вы можете поместить все свои обсуждения в виде заметок, ментальных карт, текстовых данных, изображений и других типов данных в один файл электронной таблицы.
