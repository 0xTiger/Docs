## .EFU вариант №

Файл EFU представляет собой файл электронной таблицы [CSV](/ru/spreadsheet/csv/), созданный с помощью программного обеспечения Everything компанией [voidtools](https://www.voidtools.com/). Он содержит список файлов, найденных с помощью поисковой системы Everything для Windows. Результаты в файле EFU имеют разделители CSV и могут быть открыты с помощью программ для работы с электронными таблицами, таких как Microsoft Excel. Файлы EFU удобочитаемы для человека и могут быть открыты в любом текстовом редакторе, таком как Microsoft Notepad, Notepad++ и Apple TextEdit.

## Формат файла EFU — дополнительная информация

Файлы EFU сохраняются на диск в файле значений, разделенных запятыми, который содержит информацию о результатах, полученных приложением поисковой системы Everything. Эти записи содержат информацию о найденных файлах, такую как имя файла, размер, дата создания и дата последнего изменения.
