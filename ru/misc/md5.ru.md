## Что такое файл MD5?

Файл MD5 — это файл контрольной суммы, используемый для проверки целостности файла. Он подобен отпечатку пальца для связанного файла и уникально генерируется с использованием алгоритма, использующего количество битов в файле. Он используется для проверки файла с помощью программного обеспечения, такого как [md5sum](http://www.gnu.org/software/coreutils/manual/html_node/md5sum-invocation.html). Одним из преимуществ использования файлов MD5 является проверка того, что загруженные файлы не повреждены.

## Формат файла MD5 — дополнительная информация

Обычно файл MD5 сохраняется с тем же именем, что и исходное имя файла, но с расширением .md5. Например, если связанное имя файла — «abc_987_123456.grb», соответствующее имя файла MD5 будет «abc_987_123456.grb.md5». Файлы MD5 помогают определить, что полученный или загруженный файл не поврежден и не затронут вредоносным содержимым. Короче говоря, файлы MD5 гарантируют полноту файла.


## Как проверить контрольную сумму MD5?

Один файл можно проверить на наличие MD5 с помощью инструмента [md5sum](https://en.wikipedia.org/wiki/Md5sum) следующим образом.

```
md5sum -c abc_987_123456.grb.md5
```
