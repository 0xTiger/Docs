## .ODF вариант №

Файл с расширением .odf представляет собой документ, содержащий математические уравнения, макросы и формулы. Он создается с помощью программы [Apache OpenOffice Math] (https://wiki.openoffice.org/wiki/Documentation/OOo3_User_Guides/Math_Guide), которая используется для создания математических формул, которые затем можно сохранить на диск или включить в другие документы. Эти файлы создаются и сохраняются на основе стандарта OASIS OpenDocument, открытого стандартного формата файлов для электронных таблиц, диаграмм, презентаций и документов обработки текста с использованием [ZIP](/ru/compression/zip/)-сжатого [XML](/ru/web/ xml/) файлы. Файлы ODF можно открывать с помощью таких приложений, как OpenOffice Math и LibreOffice Draw.

## Формат файла ODF — дополнительная информация

Файлы ODF представляют собой XML-файлы, сжатые zip, что означает, что все сведения о содержимом файла ODF организованы в файлы XML, которые затем объединяются в ZIP-архив. Любой может переименовать файл в .zip и извлечь его содержимое с помощью утилиты распаковки, например WinZIP.

## Включить файл ODF в другие приложения

Файлы ODF можно открывать в приложении OpenOffice Math. Однако эти файлы также можно встраивать в другие документы, такие как Calc или Impress. Подобно приложениям Microsoft Office, когда файл ODF вставляется в другие документы, он импортируется как объект, который можно дважды щелкнуть для редактирования в родительском приложении.
