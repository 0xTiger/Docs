## .SSP вариант №

Файл SSP — это файл проекта, созданный программой для создания экранных заставок Windows, известной как Screensaver Producer от Axialis Software. Он может содержать несколько различных типов элементов заставки, таких как спрайты, флэш-ролики, слайд-шоу фотографий и видео. Файл SSP может содержать текст, изображения, различные формы и настройки анимации, используемые для отображения анимации внутри файла SSP. Файлы SSP можно открывать с помощью Axialis Screensaver Producer.

## Формат файла SSP — дополнительная информация

Файлы SSP хранятся на диске в виде двоичных файлов без общедоступной информации об их внутренней структуре формата файлов. Производитель Axialis Screensaver — профессиональный инструмент для создания заставок для Windows, который позволяет легко создавать высококачественные заставки для всех версий Windows вплоть до Windows 10. Он позволяет создавать устанавливаемые пакеты заставок, которые можно легко распространять среди пользователей через Интернет.

Файлы SSP можно экспортировать в формат файлов [SCR](/ru/system/scr/).
