## .RIS вариант №

Файл RIS представляет собой формат файла тегов для обмена библиографическими данными программами цитирования. Он содержит несколько строк данных цитирования, состоящих из двухсимвольного кода и соответствующего значения, разделенного дефисом. Информация о цитировании, включенная в файл RIS, включает такие данные, как автор, название, дата публикации, ключевые слова, издатель, аннотация, адрес автора и т. д. Файлы RIS удобочитаемы и могут быть открыты в любом текстовом редакторе, таком как Microsoft Notepad, Notepad++, и Apple TextEdit.

## Формат файла RIS — дополнительная информация

Файлы RIS хранятся в виде обычного текстового файла. Каждая строка в RIS-файле состоит из двухбуквенного кода (известного также как теги), за которым следуют два пробела, дефис и соответствующее значение. Из-за тегов файлы RIS известны как тегированный формат для представления библиографических ссылок. Пример записи в файле RIS показан ниже.

## Пример RIS-файла

В этом примере «TY» — это тип ссылки, а «ER» — конец ссылки.
