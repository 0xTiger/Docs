## .DAP вариант №

Файл DAP — это файл в процессе загрузки, созданный файлом Download Accelerator Plus (DAP). Он создается, когда пользователи инициируют загрузку из любого онлайн-репозитория файлов с помощью программного обеспечения DAP. DAP позволяет приостанавливать и возобновлять текущую загрузку. Для этого он сохраняет частично загруженный файл с расширением .dap до его полной загрузки. Файл DAP можно просмотреть в процессе загрузки.

Файлы DAP можно открыть с помощью программного обеспечения SpeedBit Download Accelerator Plus (DAP).
