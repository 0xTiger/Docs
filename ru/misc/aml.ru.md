## .AML вариант №

Файл AML (Automation Markup Language) — это файл данных, который используется для хранения и обмена инженерной информацией предприятия. Он основан на формате файлов XML, благодаря чему обмен информацией осуществляется на основе открытых стандартов. AutomationML обеспечивает возможность объединения современных инженерных инструментов в области машиностроения, проектирования электрических систем, программируемых логических контроллеров и управления роботами. Файлы AML можно просматривать и редактировать с помощью инструмента AutomationML Editor.

## Формат файла AML

AML использует объектно-ориентированное моделирование данных на основе [XML](/ru/web/xml/) для хранения и обмена инженерными моделями. Компоненты инженерной установки представлены в виде объектов, имеющих разные аспекты, причем каждый объект может действовать как подобъект. Таким образом, можно описать каждую меньшую или большую модель. [Спецификации AutomationML] (https://www.automationml.org/about-automationml/specifications/) содержат краткое руководство по началу работы с деталями этого языка моделирования.
