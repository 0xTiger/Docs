## .H5 вариант №

H5 — это один из иерархических форматов данных (HDF), используемый для хранения больших объемов данных. Он используется для хранения больших объемов данных в виде многомерных массивов. Формат в основном используется для хранения научных данных, которые хорошо организованы для быстрого поиска и анализа. H5 был представлен как более усовершенствованный формат файла по сравнению с H4. Первоначально он был разработан Национальным центром суперкомпьютерных приложений, а теперь поддерживается The HDF Group.

## Краткая история формата файлов HDF

Формат файла HDF был выбран НАСА в качестве стандартного метода хранения научных данных. До этого формулировка такого стандарта была начата в 1987 году Целевой группой по основам графики (GFTF), и HDF был официально одобрен после исследования 15 различных форматов файлов.

## Формат файла HDF5

Файлы H5 имеют иерархический формат данных, соответствующий спецификациям формата файлов HDF5. Эти спецификации определяют общую структуру файла HDF5 для хранения на диске, но конечным пользователям не нужны эти базовые детали.

Файлы H5 обычно используются в аэрокосмической, физике, машиностроении, финансах, академических исследованиях, геномике, астрономии, электронных приборах и медицине.

### Модель данных HDF5

Файл HDF5 похож на контейнер, состоящий из разнородных объектов данных. Например, эти наборы данных могут быть изображениями, таблицами, текстовыми элементами, графиками и даже такими документами, как [PDF](/ru/pdf/) и [документы Excel](/ru/spreadsheet/).

## Просмотрщик файлов H5

Группа HDF предоставляет [программное обеспечение HDFView](https://www.hdfgroup.org/solutions/hdf5/), которое состоит из утилиты HDFView и пакета Java HDF Object Package.
