## .H4 вариант №

H4 или HDF4 — это один из форматов иерархических данных (HDF), который используется для хранения больших объемов данных. Это формат файла с открытым исходным кодом, который поддерживает хранение сложных разнородных данных. HDF4 — более старый формат данных по сравнению с [HDF5](/ru/misc/h5/). Форматы файлов HDF поддерживаются и публикуются некоммерческой организацией HDF group. Файлы HDF также поддерживают хранение метаданных для полного представления сохраненных данных.

## Краткая история формата файлов HDF

Разработка иерархических форматов данных была инициирована в 1987 году Целевой группой по основам графики (GFTF). За это время было предложено несколько различных форматов файлов для стандартизации единого формата. НАСА выбрало HDF в качестве стандартного метода хранения и обслуживания научных данных.

## Формат файла HDF4

Файлы H4 сохраняются в «каталоге файлов», подобно структуре, в соответствии со спецификациями формата файлов HDF. Эти спецификации определяют общую структуру файлов HDF для хранения на диске, но пользователям не нужно беспокоиться о базовой файловой структуре.

Области применения файлов H4 включают аэрокосмическую промышленность, физику, машиностроение, финансы, академические исследования, геномику, астрономию, электронные инструменты и медицинские области.

### Модель данных HDF4

Файл HDF4 или H4 похож на контейнер, состоящий из разнородных объектов данных. Типы наборов данных, поддерживаемые файлами H4, включают:

* Картинки
* Столы
* Текстовые элементы
* Графики
* Такие документы, как [PDF](/ru/pdf/) и [документы Excel](/ru/spreadsheet/)

## Просмотрщик файлов H4

Группа HDF предоставляет [программное обеспечение HDFView](https://www.hdfgroup.org/solutions/hdf5/), которое состоит из утилиты HDFView и пакета Java HDF Object Package.
