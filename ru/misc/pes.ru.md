## Что такое файл вышивки PES?

Файл вышивки PES — это файл дизайна, который содержит инструкции для вышивальных/швейных машин. Он был разработан Brother Industries для своих вышивальных машин, но позже был формализован как общий формат файла. Файлы PES используются швейными машинами для чтения инструкций по вышиванию рисунков на ткани. Эти файлы служат двум целям; во-первых, предоставление информации о дизайне для приложения PE-Design, разработанного Brother Industries, а во-вторых, предоставление названия дизайна, цветов и кодов вышивальной машины, таких как «стоп», «прыжок» и «обрезка».

## Формат файла PES — дополнительная информация

Файл PES сохраняется на диск в двоичном формате. Он содержит несколько разделов, в которых хранится информация о шитье с использованием предопределенного метода. Формат файла PES следующий.

* Данные версии — предоставляет информацию о версии и может быть любым значением `#PES0001`, `#PES0020`, `#PES0030`, `#PES0040`, `#PES0050`, `#PES0055`, `#PES0060`
* PEC Seek Value — 4-байтовое целое число с обратным порядком байтов, следующее сразу за данными версии и представляющее значение поиска для раздела PEC, который содержит сведения о конструкции.
* Раздел PSE — содержит информацию о дизайне, имеющую отношение к Brother PE-Design, и может быть другим швейным приложением.
* Раздел PCE — может находиться где угодно в файле PSE, но на него ссылается значение поиска PEC.

## Тип швейных машин с использованием файла PES

Файлы PES в основном создаются программным обеспечением PE-Design, используемым со швейными машинами Brother. Некоторые другие машины, которые могут поддерживать файлы PES, включают домашние вышивальные машины Babylock и Bernina.

## Как конвертировать файлы PSE?

Программное приложение PE-Design может [преобразовывать файл PSE](https://support.brother.com/g/s/hf/htmldoc/ped/im/ped11/en/PED11_EN/index.html#!/11_1088392?hl =pes+file) в другие форматы, такие как .pes, .dst, .exp, .pcs, .hus, .vip, .shv, .jef, .sew, .csd или .xxx. Это можно сделать с помощью приложения PE-Design, открыв файл и выбрав формат преобразования.
