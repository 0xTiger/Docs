## .RML вариант №

Файл RML представляет собой файл шаблона отчетов с механизмом отчетов [Elixir Repertoire](https://elixirtech.com/repertoire-2/). Файл шаблона используется для создания отчетов с источником данных, подключенным к файловой системе Elixir. Данные считываются и заполняются в файле шаблона RML и могут быть экспортированы в файлы различных форматов, таких как [PDF](/ru/pdf/), [RTF](/ru/word-processing/rtf/) и электронные таблицы [XLS]. (/ru/электронная таблица/xls/). Система отчетов Elixir Repertoire может быть подключена к большому количеству источников данных JDBC.

## Формат RML-файла

Подробная информация о внутренней файловой структуре формата файла RML публично недоступна. Файлы создаются и сохраняются внутри приложения Elixir Repertoire для создания отчетов с данными, заполненными из подключенных источников данных. Файл шаблона RML содержит общий макет и информацию о заполнителях окончательного отчета, который будет создан на основе данных.

## Как создать файл шаблона RML?

Чтобы сгенерировать шаблон RML в Elixir Repertoire, можно выполнить следующие шаги.

1. Подключите источник данных JDBC к файловой системе.
1. Запустите мастер шаблонов отчетов.
1. С помощью программного обеспечения найдите файл .ds источника данных.
1. Выберите табличный отчет в качестве варианта экспорта
1. Выберите поля, которые необходимо включить в шаблон отчета.
1. Наконец, после нажатия кнопки «Готово» файл First report.rml добавляется в репозиторий и открывается для отображения вкладки «Макет».
