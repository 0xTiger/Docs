## .BPS вариант №

Файл .BPS по существу представляет собой резервную копию документа текстового процессора, созданного с помощью Microsoft Works, который обычно использует расширение файла .WPS для своих документов. Эти файлы .BPS часто генерируются Microsoft Works автоматически в качестве меры предосторожности на случай сбоя программы во время работы над документом. Это способ защитить вашу работу и предотвратить потерю данных.

В следующем разделе мы рассмотрим программные приложения, связанные с файлами BPS.

## Microsoft работает

Microsoft Works — пакет программного обеспечения для повышения производительности, разработанный Microsoft. Впервые он был представлен в 1987 году и завоевал популярность как более доступная альтернатива полному пакету Microsoft Office. Microsoft Works включал в себя несколько основных офисных приложений, таких как:

- **Текстовый процессор.** В Microsoft Works было приложение для обработки текста, которое позволяло пользователям создавать и редактировать документы. Он использовал формат файла [.WPS](/ru/word-processing/wps/) для текстовых документов.

- **Таблица:** включает приложение для создания электронных таблиц и управления ими. Формат файла электронных таблиц был [.XLR](/ru/spreadsheet/xlr/).

- **База данных:** В Microsoft Works также имеется приложение базы данных для организации данных и управления ими. Формат файлов базы данных был [.WDB](/ru/database/wdb/).

- **Календарь и адресная книга.** Помимо основных офисных приложений, в состав Microsoft Works часто входят инструменты для управления календарями и адресными книгами.

- **Формат текстового процессора Works (.WPS):** Собственный формат файла компонента текстового процессора Microsoft Works. Файлы .WPS содержали текст, форматирование и простую графику.

- **Прекращение поддержки:** Microsoft официально прекратила поддержку Microsoft Works в 2007 году и призвала пользователей перейти на Microsoft Office или другие альтернативы.
