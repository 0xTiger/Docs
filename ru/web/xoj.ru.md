## .XOJ вариант №

Файл XOJ представляет собой файл записной книжки, содержащий заметки, сделанные с помощью программного обеспечения для ручных заметок Xournal. Xournal — это бесплатное многоплатформенное приложение, которое позволяет создавать заметки с различными типами контента, такими как текст, изображения и рукописные рисунки. Файлы XOJ создаются при создании нового файла записной книжки с помощью Xournal. Этими файлами XOJ можно делиться с другими пользователями, а также для обмена данными. Более поздняя версия Xournal — [Xournal++](https://github.com/xournalpp/xournalpp), которая представляет собой кроссплатформенный вариант Xournal.

## XOJ Формат файла

Файлы XOJ имеют формат файла XML и сохраняются на диск в сжатом виде с использованием сжатия [.gz](/ru/compression/gz/). Благодаря использованию универсального формата XML файлы XOJ также можно открывать и редактировать с помощью кроссплатформенного варианта Xournal++.

Каждый файл XOJ может содержать еще одну страницу блокнота Xournal.

## XOJ — Поддерживаемые платформы

Xournal является мультиплатформенным, и его записные книжки можно сохранять в виде файла XOJ в следующих операционных системах.

* Линукс
* МакОС
* Окна
* Андроид
* iOS-сборка
