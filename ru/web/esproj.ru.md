## .ESPROJ вариант №

Файл ESPROJ представляет собой файл проекта, созданный программным обеспечением для создания веб-страниц Espresso. Он доступен в Mac OS и содержит список связанных файлов, являющихся частью проекта веб-сайта. Когда вы открываете файл ESPROJ в Espresso, он открывает все связанные файлы в редакторе для редактирования или обновления исходного кода. Файл ESPROJ также содержит настройки публикации сервера и другие связанные пользовательские настройки.

## Формат файла ESPROJ

Файлы ESPROJ сохраняются как двоичные файлы, и их содержимое не может быть прочитано человеком. Каталог файлов ESPROJ также является корневым каталогом веб-сайта.
