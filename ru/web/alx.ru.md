## .ALX вариант №

Файл ALX — это файл ActiveX, встроенный в веб-страницы в виде компонентов [HTML](/ru/web/html/). Элементы управления ActiveX были официально известны как элементы управления OLE или OCX. Файлы ALX можно использовать для вставки различных функций на веб-страницу, таких как добавление элемента управления таймером для периодического обновления содержимого страницы или добавления элемента управления всплывающим окном для отображения справки в виде всплывающей подсказки на основе действий пользователя. Файлы ALX связаны с использованием<Param> тег в HTML-коде веб-страницы. Вы можете открывать файлы ALX в Microsoft Internet Explorer, Microsoft ActiveX Control Pad и текстовых редакторах, таких как Notepad и Notepad++.

## Формат файла ALX — дополнительная информация

Файлы ALX имеют формат обычного текстового файла и могут быть отредактированы путем открытия их в текстовом редакторе. Элементы управления ActiveX могут быть включены в веб-страницу путем перетаскивания на веб-страницу. Однако для создания динамических веб-страниц элементы управления интегрируются вместе с помощью [JScript](/ru/web/js/). Элементы управления на веб-странице связаны с событиями и значениями элементов управления, что дает полный контроль над взаимодействием с пользователем.
