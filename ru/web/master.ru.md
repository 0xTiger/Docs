## .MASTER вариант №

MASTER-файл — это файл шаблона главной веб-страницы, созданный с помощью ASP.NET. Он используется в качестве отправной точки для создания нескольких страниц с тем же макетом и настройками, что и MASTER-файл. MASTER-файл шаблона включает настройки заголовка, меню навигации, нижнего колонтитула, шрифта и информацию о стиле. Использование файла MASTER помогает быстро создавать несколько веб-страниц.

Вы можете открыть файл MASTER с помощью Microsoft Visual Studio 2022 и более поздних версий.

## Формат файла MASTER — дополнительная информация

MASTER-файл создается и сохраняется в формате HTML и аналогичен любому другому файлу веб-страницы. Он разделен на редактируемые и нередактируемые разделы. Редактируемые разделы — это те разделы, которые можно изменить в соответствии с требованиями пользователя. Нередактируемые разделы становятся серыми, когда файл MASTER открывается в Microsoft Visual Studio.

Главные страницы состоят из двух частей: самой главной страницы и одной или нескольких страниц контента.

### МАСТЕР-страница

Главная страница имеет расширение .master и создана в ASP.NET. Он имеет предопределенный макет, состоящий из статического текста, HTML-тегов и элементов управления на стороне сервера. В обычных страницах .aspx используется директива @ Page. В случае файлов .master это заменяется директивой @Master.

### Страницы контента

Страница содержимого представляет содержимое элементов управления-заполнителя главной страницы. Это страницы .aspx, которые на самом деле являются кодом главной страницы. Страницы контента привязываются с помощью директивы @ Page путем включения атрибута MasterPageFile, указывающего на главную страницу, которая будет использоваться, как показано ниже.

```
<%@ Page Language="VB" MasterPageFile="~/MasterPages/Master2.master" Title="Content Page of Master File" %>
```
