## .CCBJS вариант №

Файл CCBJS — это файл Javascript, созданный редактором 3D-приложений, игр и веб-сайтов **CopperCube**. Он содержит код JavaScript, который генерируется как часть [файла .html](/ru/web/html/), созданного из файла .ccb. CopperCube предоставляет удобный интерфейс сценариев для разработки логики игрового процесса и взаимодействия без написания кода. Другие форматы файлов, поддерживаемые CopperCube, включают .ccscene, .ccmodel, .ccmaterial, .cctex, .ccjs, .ccp и .ccpak.

## Формат файла CCBJS

Детали внутреннего формата файла формата CCBJS как таковые неизвестны. Однако файлы Javascript сохраняются в виде обычных текстовых файлов на диске и могут быть открыты в любом текстовом редакторе, например Microsoft Notepad, Notepad++ и Apple TextEdit.
