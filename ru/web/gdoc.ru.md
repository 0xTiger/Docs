## .GDOC вариант №

Файл GDOC — это файл ярлыка, указывающий на файл, расположенный на Google Диске, облачной службе хранения документов. Когда клиент Google Диска установлен на компьютере и внутри него создается документ, диск создает документ в облачном онлайн-хранилище и записывает [URL](/ru/web/url/) этого документа в файл GDOC в локальном Google. Папка диска. При двойном щелчке по этому ярлыку веб-браузер по умолчанию открывает документ из местоположения [URL](/ru/web/url/). Если этот файл ярлыка перемещается из этой папки, ссылка на онлайн-документ не работает.

## Формат файла GDOC — дополнительная информация

Файл GDOC содержит ярлык для онлайн-документа, написанного в формате файла [JSON](/ru/web/json/). Браузер, открывающий файлы GDOC, считывает информацию об URL-адресе и открывает связанный документ для отображения при условии, что пользователь вошел в эту учетную запись Google, в которой хранится документ. Файлы GDOC не содержат содержимого документа.

### Пример файла GDOC

Файл GDOC можно открыть в текстовом редакторе, и его содержимое выглядит следующим образом.

```
{"url": "https://docs.google.com/a/test.com/spreadsheet/ccc?key=01234567898765432123456789&usp=docslist_api", "resource_id": "spreadsheet:0A12345B678HJK9TZPL9078767"}
```

Как видно, содержимое отформатировано в формате JSON с URL-адресом документа и соответствующим идентификатором документа.
