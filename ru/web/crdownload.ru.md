## .CRDOWNLOAD вариант №

Файл CRDOWNLOAD — это частично загруженный файл, который создается при загрузке файла из Интернета с помощью браузеров Microsoft Edge, Google Chrome или Chromium. Содержимое загружаемого файла хранится в этом файле до завершения загрузки. После завершения загрузки расширение .crdownload удаляется и заменяется исходным расширением загружаемого файла.

## CRDOWNLOAD Формат файла

Файл CRDOWNLOAD может быть как в текстовом, так и в двоичном формате. Браузеры создают файл .crdownload по умолчанию после начала загрузки. Если загрузка прерывается, файл CRDOWNLOAD остается сохраненным на диске. Иногда это позволяет возобновить загрузку с того места, где она была прервана.

### CRDOWNLOAD Пример

Поддержка вы загружаете файл fileformatslist.doc из Интернета. Когда начинается загрузка, ваш браузер сохраняет файл в папку «Загрузки» как файл fileformatslist.crdownload. Размер файла увеличивается по мере загрузки файла. После завершения загрузки файл будет переименован в fileformatslist.doc.
