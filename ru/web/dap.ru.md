## .DAP вариант №

Файл с расширением .dap — это файл страницы данных Microsoft Access, который создается в Microsoft Access и имеет доступ к данным, хранящимся в базе данных. Данные, хранящиеся в таком веб-файле, доступны по сети (интранет и Интернет), что отличается от файлов Access [MDB](/ru/database/mdb/), которые разработаны специально для настольных приложений. Такие базы данных можно создавать с помощью Access 2010 и служб Access (дополнительный компонент SharePoint). Эти базы данных могут использоваться людьми, имеющими учетные записи SharePoint, с использованием веб-адреса в веб-браузере.
