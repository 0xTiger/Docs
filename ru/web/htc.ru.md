## .HTC вариант №

Файл HTC — это файл формата веб-файла, используемый Microsoft Internet Explorer. Он содержит определения для компонентов HTML, которые выполняют небольшие функции для выполнения различных задач внутри веб-браузера. Функциональность, определенная в файлах HTC, может использоваться повторно и доступна на нескольких страницах [HTML](/ru/web/html/).

## Формат файла HTC — дополнительная информация

Файлы HTC сохраняются в формате обычного текстового файла. Это означает, что вы можете открыть их в любом текстовом редакторе и внести изменения в функциональность. Обычно они открываются в Internet Explorer, но таким образом вы сможете только просматривать эти файлы и не сможете их редактировать.

Файл HTC на самом деле представляет собой HTML-файл, представляющий собой комбинацию сценария и набора элементов, специфичных для HTC, для определения компонентов.
