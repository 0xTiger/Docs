## .CSP вариант №

Файл CSP представляет собой файл динамической веб-страницы, содержащий содержимое, написанное для выполнения на сервере приложений RadGs Concept Application Server. Он компилируется и выполняется на стороне сервера, после чего передается запрашивающему браузеру пользователя по протоколу HTTP. В отличие от формата файла CON, файлы CSP не требуют установки клиента Concept у пользователя для просмотра этих файлов. Вы можете просматривать файлы CSP аналогично любым другим боковым страницам сервера, таким как [PHP](/ru/programming/php/) и [ASP](/ru/web/asp/).
