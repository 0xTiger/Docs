## .EDGE вариант №

Файл с расширением .edge — это файл проекта Adobe Edge, который создается при создании и сохранении проекта с помощью Adobe Edge. Adobe Edge позволяет веб-дизайнерам создавать интерактивные [HTML](/ru/web/html/) аниматоры для Интернета, цифровых публикаций и мультимедийной рекламы как для настольных, так и для мобильных браузеров. Это достигается с помощью HTML/HTML5, [JavaScript](/ru/web/js/) и [CSS3](/ru/web/css/). Файлы Edge создаются как обычные текстовые файлы и могут быть открыты с помощью нескольких различных приложений, таких как Microsoft Notepad, Apple TextEdit и Notepad++.

## Формат файла EDGE

Файлы EDGE сохраняются в текстовом формате и используют jQuery для включения анимации. Альтернативой анимации EDGE были файлы Flash, которые больше не поддерживаются. Файл EDGE сохраняется вместе с файлами HTML, JavaScript и CSS. При открытии файла EDGE открывается веб-страница, т.е. файл HTML.
