## .CHAT вариант №

Файл CHAT — это файл конфигурации, используемый IRC (Internet Relay Chat) для подключения к каналу IRC. Он содержит параметры подключения, такие как сервер, порт, имя IRC-канала и, возможно, пароль для открытия IRC-канала. Настройки используются клиентом для подключения к другим каналам. Чаты и обсуждения в IRC ведутся один на один и являются частными.

Приложения, которые могут **открывать** файлы ЧАТ, включают mIRC и Visual IRC.

## Формат файла чата

Файлы CHAT хранятся на диске в виде двоичных файлов. Когда пользователь хочет подключиться к каналу в IRC, он вводит информацию в iRC-клиент, он вводит информацию в клиентское программное обеспечение. Информация хранится в файле CAHT для бокового поиска.
