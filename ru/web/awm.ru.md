## .AWM вариант №

Файл AWM представляет собой файл проекта, созданный программой построения веб-меню **AllWebMenus**. Он содержит связанные с проектом данные для создания раскрывающихся меню на основе [HTML](/ru/web/html/). Файл AWM компилируется в окончательный веб-пакет, который интегрируется в веб-сайты для отображения меню. Он содержит все данные, связанные с проектом, такие как настройки внешнего вида, пункты меню, стиль (вертикальный или горизонтальный), цвета и другую связанную информацию.

Файлы AWM можно открывать с помощью **Likno AllWebMenus**.

## Формат файла AWM — дополнительная информация

Файлы AWM сохраняются как скомпилированные двоичные файлы и могут быть открыты только с помощью программного обеспечения Likno AllWebMenus. Сведения о внутреннем формате файлов AWM не доступны публично.

AllWebMenus позволяет создавать меню на основе [CSS](/ru/web/css/) или меню на основе [JavaScript](/ru/web/js/) визуально, без использования какого-либо языка программирования. Меню могут быть полностью настроены, а также разработаны с использованием предопределенных шаблонов внешнего вида, доступных в программном обеспечении.
