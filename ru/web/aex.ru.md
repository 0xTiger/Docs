## .AEX вариант №

Файл AEX (с расширением .aex) представляет собой скомпилированный файл [Xbasic] (https://documentation.alphasoftware.com/documentation/pages/Ref/Xbasic/index.xml), который содержит скомпилированный программный код для глобальных функций. Его можно использовать в веб-приложениях, вызывая их со страниц [.a5w](/ru/web/a5w/) на стороне сервера. Файлы AEX загружались и выгружались путем вызова функций a5w_load_aex и a5w_unload_aex соответственно из файлов A5W. Однако в новой реализации эти файлы загружаются в память путем включения в файл a5_application.5i. Язык программирования Xbasic используется программным обеспечением Alpha Anywhere для разработки мобильных и веб-приложений.

## Формат файла AEX — дополнительная информация

Файлы AEX сохраняются на диск в двоичном формате и содержат скомпилированный код функций, написанный на Xbasic. Операторы команды сценария Xbasic определяют структуру и поток выполнения, включая выполнение или остановку повторяющегося действия или выбор между двумя или более шагами в зависимости от условий. [Справочник по языку Xbasic] (https://documentation.alphasoftware.com/documentation/pages/Ref/Xbasic/index.xml) можно найти для получения подробной информации об этом.
