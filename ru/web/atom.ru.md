## .ATOM вариант №

Файл ATOM представляет собой формат синдикации для структуры каналов Atom и входных документов. Подобно файлам .RSS, формат файла ATOM представляет собой формат синдикации на основе XML, который является стандартом для публикации контента. Файл ATOM используется для веб-каналов, которые позволяют программам проверять наличие обновлений, публикуемых на веб-сайте. Он содержит записи, которые могут быть разных типов, таких как заголовки, полнотекстовые статьи, выдержки или ссылки на контент на веб-сайте.

Приложения, которые могут **открывать файлы ATOM**, включают **Apple Safari**.

## Формат файла ATOM — дополнительная информация

Файлы ATOM хранятся и публикуются в популярном формате файлов XML, который является универсальным форматом для обмена информацией. Он был разработан как альтернатива RSS с учетом ограничений и недостатков формата файла RSS.

### Типы документов Atom

1. «Документы входа Atom» — это XML-документ, состоящий из одного элемента информации, известного как запись, для фида Atom. Он имеет<atom:entry> элемент, содержащий несколько дочерних элементов. Содержимое записи Atom может быть простым текстом, HTML, XHTML или другим типом носителя IANA (Internet Assigned Numbers Authority).
1. «Документы канала Atom» — это XML-документ, содержащий метаданные о канале Atom и одной или нескольких записях для канала. Когда клиент делает запрос информации из фида, сервер генерирует документ фида, который включает ряд записей Atom для выполнения запроса.
1. «Коллекция Atom» — это особый вид документа канала Atom, который содержит URL-адреса записей Atom, доступных для редактирования.
