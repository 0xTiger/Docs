## .SITE вариант №

Файл SITE — это файл проекта, используемый программным обеспечением для дизайна веб-сайтов Adobe GoLive. Он хранит информацию о проекте веб-сайта, такую как файлы, папки, изображения, пользовательские таблицы стилей и сценарии. Он также содержит структуру веб-сайта и ссылки, используемые на веб-страницах сайта. Поскольку это файл проекта, он содержит ссылки на файлы проекта, но не содержит самих файлов.

Adobe GoLive прекращен и больше не поддерживается.

## Формат файла САЙТ

Файлы SITE хранятся на диске в собственном формате файлов Adobe, и внутренние детали этого формата файлов не доступны публично для справки разработчика.
