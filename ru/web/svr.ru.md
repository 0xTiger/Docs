## .SVR вариант №

Файл SVR — это формат 3D-файла, используемый SuperScape для создания интерактивных 3D-миров. Эти трехмерные интерактивные миры можно просматривать через веб-браузеры с помощью плагина SVR. Файлы SVR аналогичны формату [VRT file](/ru/web/vrt/), но сжимаются, что приводит к уменьшению размера файла. Таким образом, файлы SVR загружаются быстрее, и ими также можно делиться через Интернет.

## Формат файла SVR — дополнительная информация

Файлы SVR сохраняются на диск в двоичном формате. Это запатентованный формат файла, и сведения о его внутренней файловой структуре недоступны публично для справки разработчика.
