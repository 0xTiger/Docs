## .APKG вариант №

Файл с расширением .apkg представляет собой набор карточек, сгенерированных для использования в программном приложении [Anki](https://ankiweb.net/about), которое представляет собой обучающую программу на основе карточек. Он содержит [HTML](/ru/web/html/) текст для загрузки и отображения в приложении Anki и может дополнительно содержать изображения и звуки для визуального и звукового обучения. Anki позволяет пользователям создавать свои собственные колоды карточек Anki, а также импортировать колоды карточек других пользователей.

## Формат файла APKG

Колоды карт Anki создаются из шаблонов, написанных на HTML, известном и распространенном языке для создания веб-страниц. Стилизация карт колоды выполняется с помощью CSS, который является языком, используемым для стилизации веб-страниц. Стиль включает в себя модификации:

* font-family — название шрифта, который будет использоваться на карте.
* font-size — Размер шрифта в пикселях.
* text-align — Определяет, должен ли текст быть выровнен по центру, по левому или правому краю.
* цвет — определяет цвет текста, который может быть простым названием цвета, таким как «синий», «красный» и т. д., или цветовыми кодами HTML.
* background-color — определяет цвет фона карты

Информация о стилях распределяется между всеми картами и влияет на все карты при внесении изменений. В следующем примере будет использоваться желтый фон на всех карточках, кроме первой:

```CSS
.card {
  background-color: yellow;
}
.card1 {
  background-color: blue;
}
```

Изменением размера изображений на карте Anki можно управлять следующим образом.

```CS
img {
  max-width: none;
  max-height: none;
}
```

## Встраивание Javascript в карты Anki

Можно встроить [Javascript](/ru/web/js/) в карту Anki, используя шаблоны карт. Однако из-за продвинутого уровня Javascript его поддержка не предусмотрена. Кроме того, устройства рендеринга могут по-разному отображать влияние Javascript на карту, что приводит к необходимости тестирования реализации на всех устройствах. Некоторые функции Javascript, такие как window.alert, затрудняют отладку написанного вами кода Javascript.
