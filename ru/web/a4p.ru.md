## .A4P вариант №

Файл A4P представляет собой файл проекта, созданный программой A4Desk Pro, которая используется для создания веб-сайтов Flash. Он создается с помощью шаблонов, включенных в A4Desk Pro, и не требует глубоких знаний Flash. Файлы A4P, созданные с помощью шаблонов, могут включать содержимое веб-сайта в виде текста и Flash-анимации. Если вы не знаете о Flash, с его помощью можно было создавать анимации, которые можно было экспортировать в форматы файлов [SWF](/ru/page-description-language/swf/), а также [GIF](/ru/image/gif/). для размещения на веб-сайтах.

## Формат файла A4P

Файлы A4P создаются из файлов шаблонов и сохраняются на диск как двоичные файлы. Внутренняя структура формата файлов A4P неизвестна.

Файлы A4P можно открывать с помощью программного обеспечения **Avanquest A4Desk Pro**.
