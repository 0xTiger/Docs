## .BWP вариант №

Файл BWP представляет собой файл сценария веб-страницы, созданный на языке программирования **BuRg3r**. Он специфичен для BuRg3r и может содержать HTML-код с разделами исходного кода BuRg3r, разделенными символом<?b and ?> теги. BuRg3r — это язык программирования общего назначения, который также является игровым движком, разработанным Fluffy Smoke Entertainment.
