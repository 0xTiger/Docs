## .PRO вариант № ##

Файл PRO — это макет веб-сайта или мобильного приложения, созданный Adobe Proto, несуществующим приложением Adobe Touch. Он включает в себя один или несколько макетов пользовательского интерфейса для веб-страницы или мобильного приложения, каждый из которых может включать интерактивные элементы. Технологии CSS, WebKit и jQuery поддерживаются в файлах PRO.

### Краткая история формата файлов PRO

Файлы PRO были созданы Adobe Proto, который был заменен Adobe XD. Он использовался для создания прототипов приложений для планшетов. Обычные приложения для веб-дизайна, такие как Adobe Dreamweaver, использовали эти PRO-файлы для создания веб-приложений и мобильных приложений.

## Формат файла PRO — дополнительная информация

Файл PRO — это первый шаг в разработке веб-сайта или мобильного приложения. Разработчики сначала создают цифровые макеты, определяющие внешний вид приложения. Они также определяют общий рабочий процесс приложения и навигацию по экранам приложения. Эти макеты также включают ряд элементов навигации, таких как кнопки, поля со списком, полосы прокрутки, поля ввода и некоторые другие.
