## Что такое СЖАТЫЙ файл?

СЖАТЫЙ файл — это сжатая веб-страница Wordpress небольшого размера, созданная с помощью плагина **WordCompress**. Он представляет собой одну строку кода для представления веб-страницы, созданной плагином путем сжатия кода HTML, CSS и JavaScript. Эти конвертированные СЖАТЫЕ файлы имеют небольшой размер, что способствует более быстрой загрузке и передаче через Интернет.

### Формат файла СЖАТЫЙ – дополнительная информация

СЖАТЫЕ файлы сохраняются как текстовые файлы, которые можно открыть в любом текстовом редакторе, например Microsoft Notepad, Notepad++ и Apple TextEdit. Помимо текстовых редакторов, СЖАТЫЕ файлы можно открыть с помощью программного обеспечения для веб-создания, такого как Adobe Dreamweaver, которое доступно как для Windows, так и для Mac OS.

## О плагине WordCompress

Плагин WordCompress изначально использовался для сжатия веб-страниц WordPress в сжатый формат файла. Плагин больше не поддерживается и прекращен. Он запишет исходный код веб-страницы в один исполняемый файл, удалив все пробелы из исходного кода страницы. СЖАТЫЕ файлы хранятся вместе с исходным исходным кодом веб-страницы, чтобы сохранить исходные комментарии, отступы и другие пробелы, включенные в исходный исходный код страницы.
