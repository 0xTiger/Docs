## .XHTM вариант №

Файл с расширением .xhtm представляет собой файл веб-страницы, написанный на XHTML. В качестве стандартной реализации он использует XML 1.0 и считается переформулировкой HTML 4. Как и XHTML, файл XHTM содержит стандартные теги HTML, соответствующие более строгим правилам XML. Файлы XHTM используются для публикации содержимого веб-сайта и обычно используют расширение [XHTML](/ru/web/xhtml/).

Вы можете использовать веб-браузеры, такие как Google Chrome, Apple Safari и Mozilla Firefox, для открытия файлов XHTM.

## Пример XHTM

Следующий пример документа XHTM определяет объявления XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
