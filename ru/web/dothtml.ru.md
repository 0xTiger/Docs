## .DOTHTML вариант №

Файл DOTHTML представляет собой шаблонный документ Microsoft Word, сохраненный в HTML-версии. Его можно открыть в веб-браузере, и содержимое отображается как любая другая веб-страница. Файлы DOTHTML создаются из файлов Microsoft Word .DOT, которые содержат предварительно отформатированные параметры для создания этих файлов. Когда файл [DOT](/ru/word-processing/dot/) преобразуется в DOCHTML, все эти настройки сохраняются, и преобразованный файл DOTHTML отображается с тем же форматированием, что и исходный файл DOT. Таким образом, опубликованные шаблоны можно просматривать в веб-браузере.

## Формат файла DOTHTML — дополнительная информация

Файлы DOTHTML сохраняются как файлы [HTML](/ru/web/html/) и содержат исходный код HTML, который можно открыть в любом редакторе HTML. В первые дни Microsoft Word предлагал HTML-версии всех своих форматов документов, таких как [DOCHTML](/ru/web/dochtml/), [DOCMHTML](/ru/web/docmhtml/), но позже эти версии на основе HTML были прекращены. . В результате DOTHMLT также больше не поддерживается.
