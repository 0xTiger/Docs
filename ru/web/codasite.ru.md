## .CODASITE вариант №

Файл CODASITE представляет собой файл проекта веб-сайта, созданный с помощью программного обеспечения для создания и публикации веб-страниц [Coda](https://coda.io/). Он содержит список всех файлов, которые являются частью проекта. Сюда входят веб-страницы в виде файлов [HTML](/ru/web/html/), файлов каскадных таблиц стилей [CSS](/ru/web/css/), файлов Javascript [JS](/ru/web/js/) и любых других связанных вспомогательных файлов. Файлы проекта CODASITE также содержат информацию для публикации веб-сайта на веб-сервере.

## Формат файла CODASITE

Файл проекта CODASITE сохраняется на диск в виде двоичных файлов в каталоге ~/​Library/​Application Support/​Coda 2/​Sites/​. Пользовательский интерфейс Coda предоставляет интерфейс для управления сайтами Coda, поскольку их открытие вручную может привести к неправильным настройкам проекта и возникновению проблем.
