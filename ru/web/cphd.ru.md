## .CPHD вариант №

Файл CPHD — это гипертекстовый документ, используемый механизмом препроцессора CloudChan. Он содержит HTML и сценарии для создания динамических веб-страниц и документов. Скрипты внутри файла CPHD напоминают синтаксис, используемый Java. Веб-страница CPHD возвращает HTML-документ при выполнении в процессах CloudChan. Формат CPHD может работать с любым документом XML.

## Формат файла CPHD

Файлы CPHD сохраняются на диске в виде обычных текстовых файлов. Это позволяет открывать эти файлы в любом текстовом редакторе, например Notepad или Notepad++.
