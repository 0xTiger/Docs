## .DOCHTML вариант №

Файл DOCHTML представляет собой HTML-формат документа, созданного с помощью Microsoft Word. Это [HTML](/ru/web/html/) версия созданного документа, которую можно использовать для публикации на веб-сервере. Файлы DOCHTML можно открывать в веб-браузере для просмотра содержимого преобразованного документа Word. Внешний вид и внешний вид файла DOCHTML такие же, как у исходного документа Word.

## Формат файла DOCHTML

Файлы DOCHTML сохраняются на диск в формате HTML, который готов к публикации на веб-серверах. Microsoft Word прекратил поддержку сохранения документов в виде файлов DOCHTML. Он был заменен форматами файлов [.mht](/ru/web/mht/) и [.mhtml](/ru/web/mhtml/), которые сейчас широко используются.
