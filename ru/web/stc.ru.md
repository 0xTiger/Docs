## .STC вариант №

Файл с расширением .stc — это файл подключения, используемый программным обеспечением Adobe Contribute для хранения информации о подключении. Он содержит информацию о подключении, такую как имя пользователя, URL-адрес хоста, протоколы, такие как FTP, SFTP и т. д., для повторного использования на существующем веб-сайте. Для тех, кто не знает, Adobe Contribute раньше был специализированным редактором [HTML](/ru/web/html/), который использовался для отправки обновлений на существующие веб-сайты, включая статьи на основе контента, известные как блоги. Он включал плагины для популярных веб-браузеров, таких как Internet Explorer и Firefox, которые беспрепятственно добавляли контент на веб-сайты со страниц. Он был прекращен с января 2017 года, и теперь Adobe Dreamweaver используется для создания веб-сайтов с нуля.

## Формат файла STC

Внутренний формат данных файлов STC неизвестен. Однако ключ подключения зашифрован паролем, и любая информация о подключении к сети или FTP, отправленная в файле, остается защищенной, и доступ к ней возможен только через Contribute. Файл подключения ключа можно отправить пользователям по электронной почте или сохранить на компьютер. При нажатии автоматически открывается Contribute, который импортирует соединение.
