## .AN вариант №

Файл с расширением .an представляет собой файл проекта [Adobe Edge Animate](https://www.adobe.com/products/animate.html), который содержит информацию об анимации. При сохранении проекта создается файл .html, который содержит ссылки на связанные файлы [.css](/ru/web/css/) и [.js](/ru/web/js/), созданные Adobe Edge Animate. Файл AN сохраняется вместе с этими файлами, которые можно использовать для открытия файла проекта. Adobe Edge Animate можно использовать в качестве альтернативы созданию анимированных веб-страниц с помощью Adobe Flash, который больше не поддерживается браузерами.

## Формат файла AN

Файлы AN сохраняются на диск в текстовом формате и могут быть открыты в текстовых редакторах для просмотра их содержимого.

## использованная литература

* [Adobe Edge Animate] (https://www.adobe.com/products/animate.html)
