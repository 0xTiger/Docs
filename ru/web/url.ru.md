## Что такое URL-файл?

Файл с расширением .url — это ярлык URL-адреса веб-сайта, который можно сохранить на компьютер с любым адресом веб-сайта. Когда он открывается двойным щелчком, URL-адрес открывает веб-сайт в браузере по умолчанию, и страница будет отображаться для пользователя. Файлы URL обычно называются файлами ярлыков для веб-сайтов и представлены значком браузера по умолчанию. Файлы URL можно открывать и редактировать в любом текстовом редакторе, таком как Блокнот Microsoft, Notepad++ или Apple TextEdit.

## Формат файла URL — дополнительная информация

Файлы URL сохраняются в текстовом формате и могут быть открыты и просмотрены в любом текстовом редакторе. Они открываются в веб-браузере по умолчанию, где пользователь выполняет другой просмотр.
