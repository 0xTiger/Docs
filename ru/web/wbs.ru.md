## .WBS вариант №

Файл WBS — это файл проекта, созданный WebBlender, который представляет собой программу создания презентаций и веб-сайтов для детей. Файл WBS создается при запуске нового проекта в WebBlender. Все функции сайта, добавленные в проект, такие как страницы, изображения, текст, анимация и т. д., упоминаются в файле проекта WBS. Когда проект готов, он публикует содержимое сайта в виде файла [HTML](/ru/web/html/). WebBlender создан и поддерживается [Tech4Learning] (https://www.tech4learning.com/).
