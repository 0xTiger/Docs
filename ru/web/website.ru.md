## .WEBSITE вариант №

Файл WEBSITE — это файл ярлыка веб-сайта, созданный с помощью Microsoft Internet Explorer (IE). Он содержит URL-адрес и файл значка (ICO), который указывает на фавикон веб-сайта. Значок используется для отображения ярлыка. Формат файла WEBSITE был введен с выпуском IE версии 9. В предыдущих версиях IE ярлыки веб-страниц сохранялись в виде файлов [.url](/ru/web/url/).

## Формат файла веб-сайта

Файл .website хранится на диске в виде простых текстовых файлов, содержимое которых можно просмотреть в любом текстовом редакторе. Помимо другого содержимого, он содержит целевой URL-адрес, для которого он является ярлыком. Если вы хотите открыть целевой веб-сайт в браузере, отличном от IE, вы можете скопировать этот URL-адрес и вставить его в другие браузеры, такие как Google Chrome, Safari или Firefox, для просмотра веб-сайта.

## Как создать файл .WEBSITE?

Файл WEBSITE можно создать в IE, щелкнув URL-адрес в адресной строке и перетащив его в нужное место на вашем компьютере, например на рабочий стол или в документы. Это создает ярлык для этого места в Интернете, и его можно открыть, дважды щелкнув ярлык. Если вы хотите сохранить ярлык как файл URL, удерживайте нажатой клавишу SHIFT, прежде чем перетаскивать значок целевого сайта на рабочий стол.
