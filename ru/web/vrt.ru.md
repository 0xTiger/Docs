## .VRT вариант №

Файл VRT представляет собой формат 3D-файла, который содержит трехмерные миры, которые можно просматривать в веб-браузере с помощью плагина [Viscape](https://www.viscape360.com/) SVR. Он используется плагином для загрузки и отображения данных трехмерного мира пользователю для визуализации и анализа. Информация, хранящаяся в файле VRT, может содержать такие данные, как текстуры, звуки, размеры и поведение. Самые последние файлы VRT хранятся в формате файла SVR, который представляет собой сжатую версию файла VRT.

## Формат файла VRT

Файлы VRT хранятся в проприетарном формате файлов Viscape, и сведения об их внутреннем формате файлов недоступны для общественности.

## использованная литература

* [Viscape 360](https://www.viscape360.com/)
