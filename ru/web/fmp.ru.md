## .FMP вариант №

Файл с расширением .fmp представляет собой анимированный файл меню, созданный с помощью приложения 123 Flash Menu. Он содержит анимированные меню для веб-сайта, включая такие компоненты, как шрифты, стиль меню, цвета, размеры, звуки и анимацию. Приложение 123 Flash Menu не требует специальных навыков работы с компьютером, так как само приложение включает в себя все функции для разработки таких меню. Файлы меню FMP можно экспортировать в известные во все времена файлы Flash [.swf](/ru/page-description-language/swf/), которые можно встроить в веб-страницу.

## Формат файла FMP

Как таковой информации о формате файла FMP нет.
