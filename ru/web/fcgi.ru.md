## .FCGI вариант №

Файл FCGI — это файл исходного кода, содержащий исходный код, написанный для FastCGI (FCGI). Он может быть написан на нескольких разных языках программирования и содержит расширение стандарта CGI, используемое для более надежной обработки запросов веб-сервера. Наиболее часто поддерживаемые языки для FCGI включают **Perl**, который обеспечивает поддержку как кода Perl, так и надежного внешнего интерфейса API FastCGI.

## Формат файла FCGI

Что делает файл FCGI важным? Интерфейс FCGI поддерживает распределенные вычисления для обработки веб-запросов и распределения их по нескольким серверам. Это делает файлы FCGI в основном используемыми для балансировки нагрузки между несколькими серверами.
