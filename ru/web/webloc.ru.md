## Что такое ВЕБЛОК?

Файл WEBLOC — это файл ярлыка в macOS, который указывает на URL-адрес в Интернете. Он создается путем перетаскивания URL-адреса из адресной строки браузера на рабочий стол или в любую другую папку, где он должен быть сохранен. URL-адрес целевого сайта хранится в этом файле ярлыка, и его можно просмотреть, открыв файл в текстовом редакторе, таком как Блокнот Microsoft, Notepad++ или Apple TextEdit. Файлы WEBLOC аналогичны файлам [URL](/ru/web/url/).

## Формат файла WEBLOC — дополнительная информация

Файлы WEBLOC хранятся в формате обычного текстового файла. Их можно создать в macOS путем перетаскивания URL-адресов из веб-браузеров, таких как Safari, Google Chrome или Firefox. Если несколько файлов WEBLOC должны быть открыты одновременно, потребуется более быстрое и простое решение для их открытия через внешние приложения или API. Эти API могут читать и конвертировать WEBLOC в PDF и JPEG.
