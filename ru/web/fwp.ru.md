## .FWP вариант №

Файл с расширением .fwp представляет собой архивный файл, созданный с помощью Microsoft Expression Web и содержащий все файлы и папки проекта. Microsoft Expression Web — это общий программный продукт для веб-дизайна от Microsoft, основанный на редакторе [HTML](/ru/web/html/). Он был прекращен с 2012 года и доступен для бесплатной загрузки Microsoft. Файлы FWP можно экспортировать и сохранять в формате Microsoft .cab.

## Формат файла FWP — дополнительная информация

Точные спецификации формата файлов FWP недоступны для справки разработчика. Однако это двоичные файлы, и их можно открыть в Microsoft Expression Studio, поддержка которой давно прекращена.
