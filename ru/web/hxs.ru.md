## .HXS вариант №

Файл HXS представляет собой файл справки, скомпилированный и используемый для файла Microsoft Help 2.x. Он скомпилирован и сгенерирован из набора тематических страниц и написан в подмножестве [HTML](/ru/web/html/), аналогичном более раннему формату файла справки [CHM](/ru/web/chm/). Файлы HXS хранятся в сжатом формате и создаются из основного файла проекта .HxC, включаемого файла .HxF, оглавления .HxT, файла определения атрибутов .HxA и нескольких индексов .HxK. Метаданные файла HXS создаются из [XML](/ru/web/xml/).

Файлы HXS можно открывать в веб-браузере.

## Формат файла HXS

Файлы HXS создаются и сохраняются как сжатые файлы из содержимого HTML и метаданных XML.

## использованная литература

* [Файл справки Microsoft 2.0] (https://en.wikipedia.org/wiki/Microsoft_Help_2)
