## .XPD вариант №

Файл XPD — это файл веб-страницы, созданный из документа XML. Чаще всего это создается посредством преобразования XSLT. [Спецификации формата файла XPD](https://www.w3.org/TR/xml-pipeline/) доступны на W3 и определяют способ повышения совместимости многопроцессного [XML](/ru/web/xml/) Приложения. Язык конвейера XML определяет отношения обработки между ресурсами XML.
