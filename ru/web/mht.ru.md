## .MHT вариант №

Файл с расширением .mht представляет собой формат архивного файла с поддержкой MIME, который содержит различные типы данных в одном файле. Он может хранить такие данные, как текст, изображения, стили страниц в виде файлов [CSS](/ru/web/css/), JavaScript и другие ресурсы в виде встроенных в него ресурсов. Файлы MHT, имеющие тип MIME message/rfc822, инкапсулируют все содержимое файла HTML в виде единого архивного файла для хранения при архивировании на устройствах хранения. Программные приложения, такие как Microsoft Word, позволяют преобразовывать документы WORD в формат MHT путем экспорта в виде файла MHT. Файлы MHT можно открывать с помощью популярных браузеров, таких как Microsoft Internet Explorer и Google Chrome.

## Формат файла MHT

Как и [MHTML](/ru/web/mhtml/), MHT также представляет собой MIME-инкапсуляцию совокупных документов HTML. Содержимое документа внутри документа MHT, такое как встроенные изображения, таблицы стилей, апплеты и т. д., закодировано с помощью MIME, если оно связано с корневым ресурсом через URI. Почтовые клиенты, такие как Microsoft Outlook, хранят электронные письма (обычно [EML](/ru/email/eml/)) в формате файла MHT. Полные спецификации формата файлов MHT доступны в [RFC 822](https://tools.ietf.org/html/rfc822), и их можно использовать для разработки программного обеспечения, которое может обрабатывать файлы MHT.

## Разница между MHT и MHTML

При сохранении онлайн-страницы пользователю предлагается выбрать формат файла, в котором онлайн-страница должна быть сохранена в собственной системе. Чаще всего пользователь путается между языком разметки и форматами файлов MHTML. Они замечают, что трудно видеть, что формат файла более здоров между ними.

Когда пользователь решает не использовать онлайн-страницу в качестве языка разметки, создается папка, содержащая несколько файлов для различного содержимого страницы, т.е. совершенно разные единицы области файлов, созданные для текста, графики, апплетов и т. д. С другой стороны, сохранение онлайн-страницы, так как MHTML создает один файл для полной онлайн-страницы. Все элементы страницы вместе с графикой, ссылками и альтернативными файлами объединены в один файл.

Естественно, работать с файлами MHT или MHTML несложно, потому что файл можно защитить и отправить по электронной почте. Тем не менее, размер файлов языка разметки ниже вероятности потери информации, так как потеря или удаление даже одного файла может привести к тому, что вся папка языка разметки станет бесполезной для пользователя.
