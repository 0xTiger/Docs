## Что такое СКАЧАТЬ файл?

Файл DOWNLOAD — это частично загруженный файл из Интернета, который все еще находится в процессе загрузки. Когда вы загружаете файл из Интернета с помощью браузера Apple Safari, загрузка может занять некоторое время. Этот частично загруженный файл сохраняется в папке **Загрузки**. Если загрузка прервана по какой-либо причине, можно дважды щелкнуть файл, чтобы попытаться возобновить его.

## СКАЧАТЬ Формат файла — дополнительная информация

Файлы DOWNLOAD сохраняются на диск как в текстовом, так и в двоичном формате. Он не содержит всего файла, который находится в процессе загрузки. После завершения загрузки расширение файла .download заменяется исходным расширением загружаемого файла. Например, если вы загружаете видео из Интернета в формате файла MP4, файл будет загружен и сохранен с расширением .download. После завершения загрузки расширение файла будет изменено на .mp4.

Файлы DOWNLAOD аналогичны файлам [.CRDOWNLOAD](/ru/web/crdownload/), используемым Google Chrome и Microsoft Edge.
