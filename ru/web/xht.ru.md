## .XHT вариант №

Файл с расширением .xht представляет собой веб-файл, связанный с типом файлов [XHTML](/ru/web/xhtml/) (расширенный язык разметки гипертекста), который сам является файлом разметки на основе [XML](/ru/web/xml/). Оба эти файла похожи на HTML, но имеют более строгий синтаксис, подобный XML. Файлы XHT разработаны таким образом, чтобы быть более структурированными, включать меньше сценариев и являются универсальными в дополнение к использованию существующих возможностей XML.

## Формат файла XHT

Файл XHT создается и сохраняется как обычный текстовый файл с кодировкой UTF-8 по умолчанию. Он содержит полный исходный код документа XHTML, который можно открывать и редактировать в любом текстовом редакторе, поддерживающем кодировку Unicode. Помимо текстовых редакторов, формат файла XHT понятен большинству современных веб-браузеров и может быть открыт с их помощью.

## Пример XHT

Следующий пример документа XHT определяет объявления XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
