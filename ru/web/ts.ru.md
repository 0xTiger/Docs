## .TS вариант №

Файл TS — это файл перевода, созданный с помощью утилиты lupdate, которая поставляется вместе с Qt SDK Digia. SDK доступен для операционных систем Windows, Mac и Linux. Файл TS содержит фразы на одном языке и их переводы на другой язык. Переводы, хранящиеся в файле TS, относятся к конкретному приложению. Файлы TS можно открыть с помощью Digia Qt SDK.

## Формат файла TS

Файлы Digia Qt TS хранятся на диске в формате файлов XML и могут быть открыты в любом текстовом редакторе, таком как Notepad и Notepad++. Некоторые другие форматы файлов, поддерживаемые Digia Qt SDK, включают:

* .при
* .кв.м
* .qml
* .qph
* .qrc
* .qs
* .qss
* .тс
* Файл пользовательского интерфейса .ui
