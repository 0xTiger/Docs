## .DOCMHTML вариант №

Файл DOCMHTML — это старый вариант веб-архива, сохраненный более ранними версиями Microsoft Word. Он больше не поддерживается последней версией Microsoft Word. Файл DOCMHTML содержит HTML-представление исходного документа Word в формате .doc. Его можно открыть как в Microsoft Word, так и в веб-браузере. После публикации DOCMHTML его можно разместить на веб-сервере, доступном для пользователей через веб-запрос. Другой аналогичный формат — [DOCHTML](/ru/web/dochtml/), поддержка которого также прекращена.

## Формат файла DOCMHTML — дополнительная информация

Файлы DOCMHTML использовались для сохранения в формате HTML. Его можно открыть в текстовом редакторе, таком как Notepad и Notepad++. После того, как Microsoft Word был помечен как устаревший, в него были введены параметры сохранения документа Word в виде файлов [MHTML](/ru/web/mhtml/) и [MHT](/ru/web/mht/). Эти новые форматы файлов можно открывать в веб-браузерах.
