## .HYPE вариант №

Файл HYPE — это файл проекта, сохраненный с помощью программного обеспечения Hype, которое используется для создания веб-контента с помощью HTML5. Он содержит веб-элементы, такие как графика, текст, анимация, информация о сцене и другие элементы, которые в совокупности составляют контент. Веб-контент в файле HYPE можно экспортировать в веб-файлы, которые можно опубликовать на веб-странице.

Файл HYPE можно открыть с помощью программного обеспечения [Tumult Hype](https://tumult.com/hype/).

## Формат файла HYPE

Файлы HYPE хранятся в проприетарном формате файлов, и их внутренние детали не известны публично. Содержимое внутри файла HYPE можно просмотреть только при открытии в программном обеспечении Tumult Hype.

### Содержимое файла HYPE

Файл HYPE создается, когда вы используете приложение Hype для создания веб-контента для различных целей. Он хранит информацию об анимации на основе ключевых кадров для создания содержимого, такого как электронные открытки, электронные книги, презентации, рекламные объявления, инфографика и материалы веб-страниц. Их содержимое основано на HTML5, что позволяет отображать содержимое на устройствах от настольных компьютеров до iPhone и iPad. Tumult Hype выводит современные стили HTML5, [CSS3](/ru/web/css/) и [JavaScript](/ru/web/js/).
