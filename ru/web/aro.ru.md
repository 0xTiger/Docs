## .ARO вариант №

Файл ARO — это файл веб-страницы на стороне сервера, созданный с помощью веб-приложения SteelArrow. Он содержит теги и скрипты [HTML](/ru/web/html/) и SteelArrow. Они выполняются на сервере для создания полной страницы ответа перед отправкой их в браузер пользователя. Файлы ARO содержат почти 95% содержимого HTML, а остальная часть состоит из кода SteelArrow. Чтобы файлы ARO работали на вас, сервер веб-приложений SteelArrow должен быть установлен и запущен на компьютере с веб-сервером.

## Формат файла ARO

Файлы ARO записываются в файл языка разметки HTML со встроенным кодом JavaScript и апплетами Java. [Теги SteelArrow](http://www.steelarrow.com/docs/basics1.aro) являются основными строительными блоками для разработки веб-страниц. Хотя они не изменяют отображение страницы, они используются для заполнения страницы данными. Кроме того, их также можно использовать для управления выводом с помощью условных операторов.

### Пример тега ARO

<SAOUTPUT>Тег ARO позволяет разработчикам выводить значения данных в любом месте скрипта. Например, его можно использовать для получения вывода таблицы PARAM с помощью<SAOUTPUT VALUE=PARAM[]> которые могут отображаться в HTML<BODY> ярлык.
