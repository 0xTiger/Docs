## .MFX вариант №

Файл MFX — это файл шрифта, используемый игрой-симулятором Maxis SimCity 4. Он используется для отображения игрового текста на экране одним из выбранных типов шрифта, включая Arial, Courier или любой другой шрифт, доступный в файле шрифта. Файлы MFX устанавливаются вместе с игрой SimCity 4, и их не нужно создавать вручную.

SimCity 4 — компьютерная игра-симулятор для создания симуляторов городского строительства. Maxis, разработчик SimCity, является дочерней компанией Electronic Arts. Первоначально игра была запущена в январе 2003 года.
