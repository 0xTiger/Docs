## .PFA вариант №

Файл с расширением .pfa представляет собой версию Printer Font ASCII шрифта Adobe Type 1 и содержит данные глифов шрифта. Файл имеет удобочитаемый формат и может быть открыт в любом текстовом редакторе. Интерпретатор PostScript может использовать этот тип шрифта в целом для своей работы. Версия Printer Font Binary (PFB) шрифта Adobe Type 1 содержит ту же информацию. Однако нелинейный PFA находится в двоичном формате и не читается человеком. Файлы PFA можно включить в систему, скопировав их в системную папку Fonts.
