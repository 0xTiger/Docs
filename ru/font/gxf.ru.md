## .GXF вариант №

Файл GXF представляет собой файл шрифта с помощью инструмента проектирования САПР для Windows, General CADD Pro. Программное обеспечение использует шрифт для вставки текста и других символов в чертеж САПР. Шрифты GXF должны быть доступны для программного обеспечения General CADD Pro, чтобы чертеж САПР отображался правильно.

Вы можете открыть файл GXF с помощью [General CADD Pro] (https://www.generalcadd.com/oldSite/101release.htm).

## Об общем CADD Pro

General CADD Pro предлагает двухбуквенную структуру команд, которая включена и используется многими профессионалами в области черчения. Он поддерживает формат файла ACAD 2013. Он также обеспечивает поддержку геодезических команд азимута/расстояния и команды Field to Finish.
