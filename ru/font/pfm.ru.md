## .PFM вариант №

Файл с расширением .pfm представляет собой файл Printer Font Metrics, который содержит размеры символов, включенные в файл шрифта. Необходимо иметь файл PFM вместе с соответствующим файлом .pfb(/ru/font/pfb/) в той же папке при последующей установке. Файлы PFM имеют двоичную кодировку и не могут быть прочитаны человеком.
