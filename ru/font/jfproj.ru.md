## .JFPROJ вариант №

Файл с расширением .jfproj представляет собой пользовательский шрифт на основе JSON, созданный с помощью приложения Microsoft Font Maker в Windows 10. Их можно создать с помощью электронной ручки с устройствами с сенсорным экраном, такими как планшеты, хотя другие компьютерные периферийные устройства, такие как мышь, также могут использоваться в случае компьютеров. Файлы JFPROJ содержат глифы, которые используются для отображения символов на экране путем определения интервала между символами и словами и размера шрифта. Приложение Microsoft Font Maker доступно для загрузки в магазине Windows.
