## .PFB вариант №

Файл с расширением .pfb представляет собой двоичную версию шрифта Printer Font шрифта Adobe Type 1. Он похож на файлы .pfa(/ru/font/pfa/), но по своей природе является двоичным. В файлах PFB хранятся шрифты Adobe Type 1, используемые для операционной системы Windows. Они содержат данные глифа шрифта и хранят данные в компактной форме в двоичном формате. В дополнение к двоичному закодированному содержимому, файлы PFB зашифрованы. Для файлов PFB также требуются файлы .pfm, которые включают размеры символов в шрифте. Для установки шрифтов PFB в систему необходимо поместить файлы .pfb и .pfm.
