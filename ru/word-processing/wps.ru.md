## .WPS вариант №

Файл с расширением .wps — это документ Microsoft Works, который до 2006 года использовался как текстовый документ. К тому времени он был похож на недавно представленный файловый формат [DOC](/ru/word-processing/doc/), но не обладал такими функциональными возможностями, как новый формат файлов. Например, он не поддерживал функции макросов и расширенные параметры форматирования. Файлы WPS по-прежнему можно открывать с помощью современных текстовых процессоров, но для Windows 10 вам необходимо установить конвертер для загрузки этих файлов. Microsoft Works был заменен процессором документов Microsoft Word в 2010 году.

## Спецификации формата файла

Спецификации формата файла документа WPS недоступны для общественности, поскольку Microsoft не открывала спецификации для конечных пользователей. WPS являются двоичными файлами и могут быть открыты с помощью фильтров преобразования формата файлов, созданных Microsoft для процессора Word. LibreOffice, NeoOffice и OxygenOffice включают общую библиотеку C++, libwps, которая может извлекать текст из различных версий Microsoft Works.
