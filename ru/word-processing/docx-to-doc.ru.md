## О преобразовании DOCX в DOC

Тип файла DOC — популярный формат файлов документов, используемый в 2007 году и предыдущих версиях Microsoft Word. С выпуском Office 2007 компания Microsoft представила формат файлов DOCX, основанный на спецификациях Office Open XML. Файлы DOC сохранялись как двоичные файлы по сравнению с DOCX, который сохраняется в сжатом формате XML. В Интернете доступно множество инструментов для преобразования файлов [DOCX](/ru/word-processing/docx/) в файлы [DOC](/ru/word-processing/doc/).

## Преобразование DOCX в DOC — бесплатная демо-версия

Вы можете конвертировать файлы DOCX в формат DOC с помощью БЕСПЛАТНОГО онлайн-приложения Aspose.app. Это набор приложений, которые могут загружать, просматривать и конвертировать файлы DOCX в такие форматы, как DOC, PDF, TXT, RTF и другие. Приложения Aspose работают и запускаются с помощью API-интерфейсов Aspose на серверной стороне, которые могут выполнять ряд операций с файлами разных типов. Попробуйте онлайн-приложение Aspose.Words прямо сейчас, нажав следующую кнопку.

### Шаги по конвертации DOCX в DOC онлайн

Вы можете преобразовать формат файла DOCX в DOC с помощью [приложения Aspose.Words DOCX to DOC](https://products.aspose.app/words/conversion/docx-to-doc), как показано ниже.

1. Перейдите в приложение Aspose.Words для [конвертера DOCX в DOC](https://products.aspose.app/words/conversion/docx-to-doc)
1. Загрузите файл DOCX, нажав кнопку загрузки.
1. После этого нажмите кнопку «Преобразовать». Это запустит процесс преобразования файла DOCX в DOC в фоновом режиме.
1. Вам будет предложено загрузить преобразованный файл RTF после завершения процесса преобразования DOCX в DOC.
