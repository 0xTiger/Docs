## .FODT вариант №

Файл с расширением .fodt представляет собой текстовый документ, сохраненный в формате OpenDocument flat [XML](/ru/web/xml/). В отличие от файлов .odt, которые сохраняются в формате файла-контейнера [.zip](/ru/compression/zip/), файлы FODT хранятся на диске как удобочитаемые файлы, которые можно открыть в любом текстовом редакторе для просмотра в виде простых XML-документов. . Стандарт офисных документов OpenDocument был разработан отраслевым консорциумом OASIS, что открывает путь для внедрения кем угодно без каких-либо ограничений. Другие популярные форматы файлов OpenDocument включают [.odt](/ru/word-processing/odt/), [.ott](/ru/word-processing/ott/), [.ods](/ru/spreadsheet/ods/), [.odp] (/ru/presentation/odp/) и другие подобные.

## Формат файла FODT — дополнительная информация

Файлы FODT хранятся в формате простого XML-файла (также известного как Flat XML или Uncompressed XML), хотя OpenDocument обычно не использует этот формат документа. Файлы OpenDocument обычно сохраняются в виде ZIP-пакета, состоящего из нескольких файлов и каталогов. Файлы OpenDocument на основе XML могут не поддерживаться некоторым программным обеспечением, поэтому их использование менее распространено по сравнению с файлами на основе пакетов ZIP.
