## Преобразование DOCX в XPS – БЕСПЛАТНАЯ ДЕМО

Вы можете БЕСПЛАТНО использовать приложение для конвертации Aspose.Words, чтобы [конвертировать DOCX в XPS] (https://products.aspose.app/words/conversion/docx-to-xps). Вот шаги, которые необходимо выполнить, чтобы преобразовать файлы DOCX в XPS.

1. Перейдите в [Приложение для преобразования Aspose.Words](https://products.aspose.app/words/conversion/docx-to-xps)
1. Загрузите файл DOCX в браузер.
1. Выберите XPS в качестве формата выходного файла.
1. Нажмите кнопку Конвертировать.

Это запустит процесс преобразования вашего файла DOCX в XPS, и появится опция загрузки для сохранения файла на ваш компьютер.

## О преобразовании DOCX в XPS

Преобразование страниц DOCX в формат файла XPS (спецификации бумаги XML) является одной из часто используемых операций. Когда файл DOCX преобразуется в XPS, все его страницы преобразуются в один выходной файл XPS. Microsoft Windows предоставляет программу просмотра XPS по умолчанию, которую можно использовать для открытия и просмотра файлов XPS. DOCX можно конвертировать в XPS с помощью нескольких различных онлайн-инструментов. Приложение Aspose.Words — одно из таких онлайн-приложений, которое может бесплатно конвертировать [DOCX](/ru/word-processing/docx/) в [XPS](/ru/page-description-language/xps/) онлайн.
