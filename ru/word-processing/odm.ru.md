## .ODM вариант №

Файл с расширением .odm — это файл документа OpenDocument Master, который содержит один или несколько связанных файлов [.odt](/ru/word-processing/odt/). Все эти вложенные документы объединены в один главный документ. Это упрощает управление всеми связанными файлами из одного мастер-документа. Файлы основных документов можно создавать с помощью Apache OpenOffice Writer или текстового процессора OpenDocument. Файлы ODM можно экспортировать в один файл ODT, используя параметр экспорта в приложении OpenOffice Writer.

## Формат файла ODM — дополнительная информация

Файлы ODM сохраняются как текстовые файлы, которые можно открыть в OpenOffice Writer. Один файл ODM состоит из нескольких разделов, связанных с поддокументами .odt. Это позволяет нескольким авторам редактировать свои документы ODT, связанные в основном файле ODM. Таким образом, основной файл ODM обновляется по мере того, как каждый писатель обновляет свои соответствующие разделы.

Файлы ODM также могут включать оглавление и указатель. Каждый раз, когда мастер-документ открывается, автор мастер-документа запрашивает обновление ссылок. Необходимо сказать «да», чтобы контент отображался правильно.
