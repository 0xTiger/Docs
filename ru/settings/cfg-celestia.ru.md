## .CFG вариант №

Файл конфигурации Celestia представляет собой обычный текстовый файл, используемый Celestia, программой моделирования 3D-вселенной. Эти файлы жизненно важны для настройки поведения Celestia, отображения небесных объектов и внешнего вида программы. Однако редактирование этих файлов требует особого внимания, поскольку неправильные изменения могут нарушить процесс загрузки Celestia и помешать его правильной работе.

## Селестия

Celestia — это бесплатное программное обеспечение для 3D-астрономического моделирования с открытым исходным кодом, которое позволяет пользователям исследовать и визуализировать Вселенную в очень реалистичной и интерактивной форме. Он был разработан Крисом Лорелом и впервые выпущен в начале 2000-х годов. Celestia доступна для операционных систем Windows, macOS и Linux.

Ключевые особенности и аспекты Celestia включают в себя:

- **Реалистичная 3D-визуализация:** Celestia обеспечивает детальное и точное представление нашей Солнечной системы, а также звезд, галактик и других небесных объектов. Он использует высококачественные 3D-модели и текстуры для создания визуально захватывающего опыта.

- **Обширная база данных небесных объектов.** Программное обеспечение поставляется с обширной встроенной базой данных известных небесных объектов, включая звезды, планеты, луны, астероиды, кометы и космические корабли. Пользователи могут легко находить и исследовать эти объекты.

- **Научная точность.** Хотя Celestia является инструментом визуализации, ее целью является максимально точное представление небесных объектов и явлений на основе доступных научных данных.

## Форматы файлов, используемые Celestia

Celestia использует различные форматы файлов для разных аспектов своего трехмерного астрономического моделирования. Вот некоторые из ключевых форматов файлов, используемых Celestia:

1. **Файлы конфигурации (.cel)**
- *Описание*: обычные текстовые файлы, которые позволяют пользователям настраивать поведение, внешний вид и содержимое Селестии.
- *Цель*: настройка параметров, определение местоположения и указание небесных объектов.

2. **3D-модели и сетки**
- *Форматы*: [.3ds](/ru/3d/3ds/), [.obj](/ru/3d/obj/), [.dae](/ru/3d/dae/), .ac
- *Описание*: поддерживаемые форматы файлов 3D-моделей, используемые для рендеринга небесных тел и космических кораблей.
- *Цель*: Отображение 3D-объектов в Celestia.

3. **Файлы текстур**
- *Форматы*: [.jpg](/ru/image/jpeg/), [.png](/ru/image/png/), [.dds](/ru/image/dds/)
- *Описание*: Эти файлы содержат текстуры поверхности небесных тел.
- *Цель*: наложение текстур на 3D-модели для придания им реалистичного вида.

4. **Звездные каталоги и файлы данных**
- *Форматы*: пользовательские форматы, [.csv](/ru/spreadsheet/csv/), [.tsv](/ru/spreadsheet/tsv/)
- *Описание*: файлы данных, используемые для представления звезд и других небесных объектов, обеспечивающие точное моделирование.
- *Цель*: Точное представление небесных объектов.

5. **Карты текстурных кубов**
- *Описание*: Кубические карты используются для имитации внешнего вида далеких небесных объектов, таких как галактики.
- *Цель*: визуализация удаленных объектов с реалистичными текстурами.

6. **Файлы сценариев**
- *Описание*: Эти файлы содержат пользовательские сценарии, написанные на языке сценариев Селестии.
- *Цель*: Предоставление пользователям возможности создавать динамические события и анимацию во вселенной Селестии.

## Файл конфигурации Celestia

Вот базовый обзор того, что вы можете сделать с файлом конфигурации Celestia:

1. **Установка вашего местоположения**: вы можете указать свое местоположение на Земле, используя параметры широты, долготы и высоты. Это позволяет Селестии точно отображать ночное небо с вашей позиции.

```
Location "My Location"
{
    Latitude 40.7128
    Longitude -74.0060
    Altitude 0
}
```

2. **Настройка параметров просмотра.** Вы можете настроить различные параметры просмотра, такие как поле обзора, настройки времени и параметры рендеринга.

```
Viewpoint
{
    Location "My Location"
    Follow "Earth"
    Eye [0.0 0.0 0.0]
    Center [0.0 0.0 0.0]
    Up [0.0 1.0 0.0]
    Fov 45
}

Time
{
    Date "2023-09-25T12:00:00.000Z"
    Clock "Now"
}

Rendering
{
    Atmosphere false
    Stars 7
    Planetshine 0.25
}

```

3. **Загрузка небесных объектов.** Вы можете добавлять в симуляцию небесные объекты, такие как звезды, планеты, астероиды, космические корабли и многое другое. Каждый объект определен в файле конфигурации со своими свойствами.

```
Star "Sun"
{
    RA 0
    Dec 0
    Distance 0
    AppMag -26.74
    SpectralType "G2V"
}

Planet "Earth"
{
    Parent "Sol"
    Texture "earth.jpg"
    Radius 6371
    EllipticalOrbit
    {
        Period 365.25
        SemiMajorAxis 149.6e6
        Eccentricity 0.017
        Inclination 0
        AscendingNode 0
        ArgOfPericenter 102.94
        MeanAnomaly 100.464
}
}
```

4. **Определение космических кораблей.** Вы можете создать свой собственный вымышленный космический корабль или использовать реальные, указав их параметры, такие как положение, ориентация и 3D-модели.

```
Spacecraft "Voyager 1"
{
    Parent "Sol"
    Class "spacecraft"
    Mesh "voyager.3ds"
    Radius 0.01
    EllipticalOrbit
    {
        Period 30700
        SemiMajorAxis 1.08e11
        Eccentricity 0.044
        Inclination 3.4
        AscendingNode 49.0
        ArgOfPericenter 44.0
        MeanAnomaly 35.0
}
}
```

5. **Создание сценариев.** Вы можете писать сценарии на специальном языке сценариев Celestia для создания динамических событий и анимации.
