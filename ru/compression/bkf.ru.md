## .BKF вариант № ##

Файл с расширением .bkf относится к WindowsXP корпорации Microsoft. Все операционные системы, поддерживающие каталог файлов, используют файл BKF, поддерживаемый утилитой. Если данные на компьютере пользователя повреждены, перезаписаны или удалены, файл BKF очень полезен для восстановления информации в системе пользователя. Также есть спецификация файла BKF, он используется компьютерной игрой Moto Racer. Эти файлы содержат архивные файлы игр, созданные программой.

## Краткая история ##

Расширение файла BKF — это своего рода файл утилиты резервного копирования Windows, установленный для программы Dragon UnPACKER компанией Open Source. Это очень распространено среди пользователей в Соединенных Штатах и тех, кто работает под управлением операционной системы Windows 10.

## Технические характеристики ##

* Разработано корпорацией Майкрософт
* Первичная ассоциация с WindowsXP
* Поддерживается приложением резервного копирования Windows NT
* Категория приложения - Архивные файлы
* Тип формата — двоичный.
* Относится к категории файлов резервных копий
* Связан с BKF Viewer
