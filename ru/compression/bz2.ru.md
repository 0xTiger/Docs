## .BZ2 вариант №

BZ2 — это сжатые файлы, созданные с использованием метода сжатия с открытым исходным кодом [BZIP2](http://www.bzip.org/), в основном в системах UNIX или Linux. Он используется для сжатия одного файла и не предназначен для архивирования нескольких файлов. Это отличается от формата файла TAR на тех же платформах, который архивирует несколько файлов в один файл, но без сжатия. Файлы, сжатые как BZ2, можно распаковать с помощью таких приложений, как [WinZip] (https://www.winzip.com/win/en/). BZIP2 использует алгоритм сжатия Run-Length Encoding (RLE) или [Burrows-Wheeler](https://en.wikipedia.org/wiki/Burrows%E2%80%93Wheeler_transform) для достижения высокого уровня сжатия.

## Формат файла BZ2

Официальных спецификаций для этого формата файла нет. Однако неофициальные [обратно спроектированные спецификации] (https://github.com/dsnet/compress/blob/master/doc/bzip2-format.pdf) показывают, что поток .bz2 состоит из 4-байтового заголовка, за которым следует нулем или более сжатыми блоками. За ним сразу же следует маркер конца потока, содержащий 32-битный CRC для всего обработанного потока открытого текста. Между сжатыми блоками нет заполнения, и все блоки выровнены по битам.

## Разархивируйте / извлеките файл BZ2

Вы можете разархивировать файл BZ2 в Windows и Mac OS с помощью программного обеспечения, такого как WinZip. В Linux следующая команда в терминале.

```
bzip2 -d filename.bz2
```
