## .PKG вариант №

Файл с расширением .pkg — это файл пакета установщика, который используется для установки программного обеспечения в macOS. Помимо компьютеров Macintosh, он также используется на iPhone. Встроенное приложение-установщик Apple можно использовать для установки содержимого файла PKG. Файл PKG похож на файл MSI, используемый в операционной системе Windows для установки программного обеспечения. Во время установки эти файлы пакетов могут регистрировать сообщения, которые дают вам представление о том, какие дополнительные компоненты устанавливаются этим пакетом.

## Формат файла PKG

PKR — это двоичные файлы, сжатые для уменьшения общего размера файла. Начиная с OSX 10.5 Apple представила плоские пакеты с одним файлом установки. Они отличаются от более ранних форматов упаковки, которые поставлялись в виде пакетов, а не отдельных файлов. Эти объединенные пакеты содержали структурированную иерархию каталогов, в которой файлы хранились таким образом, чтобы облегчить их поиск через какой-либо индексный файл. Плоский формат файла PKG на самом деле представляет собой архив [XAR](/ru/compression/xar/), который имеет:

* Заголовок — определяет размер, контрольную сумму и информацию о версии.
* Оглавление — XML-документ, который (и должен) быть закодирован как UTF-8 и хранится в начале файла, что упрощает сканирование архива для извлечения отдельного файла.
* Heap - неструктурированная куча данных, на которую ссылается оглавление
