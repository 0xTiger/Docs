## .SIFZ вариант №

Файл SIFZ представляет собой сжатый gzip файл SIF, созданный приложением для создания 2D-анимации с открытым исходным кодом **Synfig Studio**. Он содержит несколько графических элементов, которые составляют анимацию. Общее содержание анимации строится с использованием комбинации холста, заполненного фигурами, мазками кисти или карандаша и текстом. Все они располагаются по своим позициям, радиусу, касательной, углу, вершине и ручке ширины. Файлы SIFZ можно открыть с помощью [Synfig Studio] (https://www.synfig.org/).

## Формат файла SIFZ

Файлы SIFZ представляют собой сжатые файлы [ZIP](/ru/compression/zip/), упакованные с использованием метода сжатия gzip. Synfig используется для создания анимации кинематографического качества с использованием векторных и растровых изображений. В отличие от старого метода создания покадровой анимации, он позволяет создавать 2D-анимации более высокого качества с меньшими ресурсами.

Сжатые файлы SIFZ меньше несжатых файлов SIF. Это также упрощает передачу через интернет-соединения с низкой пропускной способностью.
