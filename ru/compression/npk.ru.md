## .NPK вариант №

Файл NPK представляет собой файл пакета обновления программного обеспечения, используемый маршрутизаторами **MikroTik** для обновления операционной системы маршрутизатора. Он поставляется в виде сжатого двоичного архива, который загружается в маршрутизатор для установки обновлений программного обеспечения. Информация, содержащаяся в файлах NPK, включает сетевую информацию, такую как IP-адрес и IP-служба, информацию о соединителях (интерфейсы Ethernet и последовательный порт), настройку электронной почты, конфигурацию моста и локальное управление пользователями.

## Формат файла NPK

Файлы NPK хранятся в виде сжатого двоичного архива. Это закрытый формат файла, который используется только самими MikroTik. Он не предназначен для создания надстроек RouterOS через третьих лиц. Файлы NPK поддерживаются самим MikroTik, и их обновленные версии можно загрузить из разделов загрузки на веб-сайте [MikroTik](https://mikrotik.com/download).

Когда файл NPK загружается на маршрутизатор, ОС маршрутизатора не вступает в силу, пока не будет перезагружена. Поэтому для обновления ОС маршрутизатора требуется перезагрузка.
