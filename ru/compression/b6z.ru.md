## .B6Z вариант №

Файл с расширением .b6z представляет собой сжатый архивный файл, созданный программным приложением macOS [B6Zip](http://b6zip.com), который используется для сжатия и распаковки файлов. Это сравнительно новый формат архива, обеспечивающий более высокую степень сжатия. B6Z имеет открытую архитектуру и поддерживает файлы размером до 900 000 ПБ. Он поддерживает сжатие данных, восстановление после ошибок и объединение файлов. Утилита B6Zip доступна бесплатно в MacOS для открытия различных типов сжатых файлов, включая B6Z.

## Формат файла B6Z

Формат файла B6Z основан на открытой архитектуре и обеспечивает надежное сжатие с помощью алгоритма шифрования AES-256. Он использует LZMA2 в качестве метода сжатия по умолчанию, но также поддерживает другие методы сжатия, указанные ниже.

|Метод|Описание|
---|---|
|LZMA |Оптимизированная версия алгоритма LZ77|
|ЛЗМА2| Улучшенная версия LZMA|
|Сдуть| Обычный алгоритм LZ77|
|BZip2| Стандартный алгоритм BWT|
|ППМД| PPMdH Дмитрия Шкарина|

Утилита B6Zip поддерживает все эти стандарты сжатия и оптимизирована, что обеспечивает высокую скорость. Он поддерживает работу с форматами файлов [ZIP](/ru/compression/zip/), [TAR](/ru/compression/tar/) и [RAR](/ru/compression/rar/). Файлы B6Z имеют тип MIME application/x-b6z-compressed.
