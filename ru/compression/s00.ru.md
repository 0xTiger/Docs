## .S00 вариант №

Файл с расширением .s00 — это первая часть сжатого файла, который разделяется с помощью программного обеспечения ZipSplitter. Разделение файлов на более мелкие файлы уменьшает общий размер сжатого архива и упрощает его передачу через Интернет. Остальные части разделенного архива именуются с расширениями .S01, .S02 и т.д. Когда файл должен быть извлечен, дважды щелкните файл .EXE, включенный в разделенный файл. EXE ищет файл .S00 и использует другие сопутствующие файлы для извлечения всего архива.

## Формат файла S00

Файлы S00 сохраняются на диск в виде бинарного сжатого архива. Однако сведения о внутреннем формате файла формата S00 не являются общедоступными. Большие файлы, такие как файлы .[WMV](/ru/video/wmv/) или [.MP4](/ru/video/mp4/) чаще всего разбиваются с помощью программного обеспечения ZipSplitter для уменьшения их размера.
