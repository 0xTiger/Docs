## .RPM вариант №

Файл с расширением .rpm представляет собой пакет операционной системы Red Hat Linux для установки программ в системах Linux. Диспетчер пакетов Red Hat использует формат файла RPM и является бесплатной системой управления пакетами с открытым исходным кодом. Хотя файлы RPM можно использовать как есть для установки программ, их можно преобразовать в другие форматы пакетов, такие как [DEB](/ru/compression/deb/), используя программу Debian под названием Alien.

## Формат файла RPM

Файл RPM представляет собой двоичный файл, который может содержать набор файлов. В большинстве случаев файлы RPM представляют собой «двоичные RPM», которые представляют собой скомпилированные исполняемые файлы программного обеспечения. Файл RPM может содержать исходные пакеты RPM (SRPM), которые можно использовать для сборки пакета из исходного кода. Формат файла RPM состоит из четырех разделов.

* Ведущий — идентифицирует файл как RPM-файл.
* Подпись — ее можно использовать для обеспечения целостности и/или подлинности
* Заголовок — содержит метаданные, включая имя пакета, версию, архитектуру, список файлов и т. д.
* Файловый архив — также известный как полезная нагрузка, обычно в формате cpio, сжатый с помощью [GZIP](/ru/compression/gz/)
