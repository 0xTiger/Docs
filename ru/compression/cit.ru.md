## .CIT вариант №

Файл CIT — это файл проекта, созданный приложением In8 Cite для написания отчетов о расследованиях на основе цитирования. Он содержит детали инцидента в виде отчета, а также имеет каталог, содержащий вспомогательную документацию и доказательства, приведенные в отчете. Следователи по уголовным делам используют файлы CIT для ссылок на уголовные дела во время своих расследований.

## Формат файла CIT

Файлы CIT сохраняются на диск в сжатом формате. Он использует стандартный алгоритм сжатия ZIP для сжатия файлов. Файл CIT и сопутствующие документы могут быть преобразованы и экспортированы в файл [PDF](/ru/pdf/), который может быть предоставлен другим пользователям для просмотра.

### О In8

In8 — компания-разработчик программного обеспечения, которая имеет опыт разработки приложений, используемых для документирования и составления отчетов об уголовных расследованиях.
