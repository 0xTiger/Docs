## .XAPK вариант №

Файл с расширением .xapk представляет собой сжатый файл пакета, который используется для установки приложений Android на мобильные устройства. Это формат файла-контейнера, который включает в себя APK и дополнительные связанные файлы, необходимые для установки. Другой связанный файл — это файл OBB, в котором хранятся дополнительные файлы, такие как графика, мультимедийные файлы и данные приложения, которые необходимы для поддержания работы приложения. Файлы XAPK не поддерживаются Google Play и используются для распространения только на сторонних веб-сайтах загрузки приложений для Android. Их можно установить на устройство Android с помощью установщика XAPK.

## XAPK-формат файла

Файлы XAPK сжимаются в стандартном формате [ZIP](/ru/compression/zip/). Их можно извлечь с помощью стандартного программного обеспечения для сжатия/распаковки, такого как WinZIP. После извлечения файла XAPK на диск он содержит следующие файлы в папке.

* APK - Стандартный установочный файл для установки приложения на устройства Android
* OBB — Дополнительный файл, содержащий соответствующие файлы ресурсов
