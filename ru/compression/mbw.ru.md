## .MBW вариант № ##

Файл с расширением .mbw в первую очередь связан с MBRWizard от Firesage Solutions. Мастер основной загрузочной записи (MBRWizard) создает резервную копию MBR вашего компьютера и сохраняет ее в файл MBW, который при необходимости можно восстановить. Помимо копий MBR и VBR (Volume Boot Record), эти файлы могут содержать GPT, определяемые пользователем сектора или целые образы дисков. MBR — это важная запись длиной 512 байт на жестком диске, которая позволяет вашей операционной системе загружаться. Он также может содержать таблицу разделов и 32-битную подпись, идентифицирующую дисковый носитель.

## Поддерживаемые операционные системы ##

Формат файла MPW поддерживается только операционной системой Windows.

## Возможные проблемы при открытии файла ##

Если вы не можете открыть и запустить файл MBW, может возникнуть одна из следующих возможных проблем:

* Поврежденный файл MBW
* Ошибочные ссылки на файл MBW в записях реестра
* Удалено описание MBW из реестра Windows
* Зараженный файл с нежелательным вредоносным ПО
* На компьютере недостаточно аппаратных ресурсов для работы с файлом MBW.
* Драйверы, используемые компьютером для открытия файла MBW, устарели.
