## .DMG вариант №

Файл с расширением .dmg является файлом образа диска Apple и известен как файл образа диска Mac OS X. Формат файла DMG является наиболее широко используемым файлом образа диска, предназначенным для размещения приложений и файлов между компьютерами Apple. Очень удобно монтировать файлы DMG на виртуальный диск или использовать программное обеспечение Apple Disk Utility для открытия и просмотра содержимого формата файла DMG. Старая версия файлов IMG, используемая в Mac OS Classic, заменена форматом файлов DMG. Файлы DMG создаются для пользователей Mac и не поддерживаются ОС Windows, но пользователи могут получить доступ к содержимому DMG в операционной системе Windows с помощью инструментов сжатия и распаковки, таких как 7-Zip, Pea Zip и т. д. Расширение файла DMG также используется инструмент Oracle Export and Ingress для файлов свалки, хранящихся в двоичном формате базы данных Oracle.

## Краткая история

Расширение файла DMG было разработано Apple и в основном используется на компьютерах Macintosh. В основном файл DMG используется, чтобы дать устройствам Mac возможность устанавливать различные типы файлов и программного обеспечения, которые запускаются из Интернета. Файл DMG представляет собой монтируемый образ диска, который отображается на рабочем столе при его открытии. Файл содержит необработанные данные, которые обычно зашифрованы и сжаты. Системы Mac обрабатывают DMG так же, как и диск с чередованием, и они мгновенно пытаются открыть или «отследить» файл.

## Технические характеристики формата файла DMG

* Специально создано для компьютеров Macintosh
* Разработано Apple
* Формат файла двоичный
* Категория файла — файл образа диска
* Поддерживаемая программа: MacOS
* Может содержать таблицу разделов
* Может содержать любую файловую систему, такую как HFS+
* Функциональность для сжатия изображений
