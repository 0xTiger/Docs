## .RTE вариант №

Файл с расширением .rte представляет собой закодированный файл, используемый для обмена файлами, защищенными авторским правом. Это файлы данных, сжатые и закодированные для обеспечения защиты таких файлов. Стандарт сжатия этих файлов с кодировкой RTE может варьироваться и соответственно уменьшать размер файла. Механизм сжатия и кодирования обеспечивает безопасность контента, защищенного авторскими правами. Формат файла RTE надежно используется для обмена защищенным авторским правом контентом через Интернет или с помощью таких носителей, как DVD, CD, флэш-накопители USB и внешние жесткие диски.

## Формат файла RTE

RTE — это сжатые файлы, формат которых не является общедоступным.
