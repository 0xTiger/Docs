## .SITX вариант №

Файл с расширением .sitx представляет собой сжатый архивный файл, созданный с помощью программного обеспечения StuffIt. Этот формат был представлен как расширение исходного стандартного формата файлов Stuffit (.sit) и предлагает сравнительно большее сжатие по сравнению с его предшественником. Файлы SITX более компактны и занимают меньше места, а также поддерживают 512-битное сжатие, направленное на безопасность. Файлы SITX распознаются по типам носителей в Интернете как application/x-stuffitx и application/x-sitx с универсальным идентификатором типа com.stuffit.archive.sitx.

## Формат файла SITX — дополнительная информация

STIX сохраняются в двоичном формате файла и были разработаны после использования формата файла SIT. Он был представлен компанией Aladdin Systems в сентябре 2002 года для StuffIt Deluxe 7.0 для Macintosh. Этот новый формат файла был разработан с учетом следующих целей:

* Расширяемый
* Поддержка дополнительных методов сжатия
* Поддержка длинных имен файлов
* Поддержка атрибутов файлов Unix и Windows

Обновленный StuffIt X улучшил исходный формат StuffIt, включив несколько алгоритмов сжатия, таких как PPM и сжатие типа BWT в LZW. Новый формат файла также был оснащен опцией «блочного режима» для защиты от потери данных путем исправления ошибок «избыточными» опциями.
