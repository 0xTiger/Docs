## .PTK вариант № ##

Файлы PTK считаются файлами данных, используемыми Intuit Quicken. Это приложение для управления личными финансами, которое поддерживает расширение файла .ptk для правильной работы. Он имеет различные функции, такие как импорт данных прямо из участвующих банков, отчеты обо всех финансовых данных и ежедневная загрузка цен на акции и фонды. Формат файла PTK — это более ранний тип файла, который Quicken использует для сохранения финансовой информации, который теперь заменен форматом GIF. Основной целью файлов PTK был импорт данных из онлайн-ресурсов, таких как банки и финансовые учреждения, в базу данных Quickens.
