## .XPI вариант №

Файл XPI представляет собой установочный архив, сжатый для уменьшения размера файла. Он используется приложениями Mozilla для установки плагинов и дополнительных файлов. Он содержит сценарий установки или манифест в корне файла вместе с рядом файлов данных. Файл XPI может содержать темы, набор инструментов или плагины Firefox, которые пользователь может установить, чтобы стать частью браузера Firefox, Thunderbird или SeaMonkey.

## Формат файла XPI — дополнительная информация

Файлы XPI сохраняются на диск в виде архивов [ZIP](/ru/compression/zip/), которые объединяют несколько файлов в один сжатый файл. Файлы внутри файла XPI могут включать файл сценария установки, такой как JS, и веб-файлы, такие как [CSS](/ru/web/css/), [HTML](/ru/web/html/) и [JSON](/ru/web/json/ ). Иногда он также может содержать файлы изображений [PNG](/ru/image/png/), которые надстройка использует в качестве значков.

### Как просмотреть содержимое файла XPI?

Файл XPI практически представляет собой ZIP-архив, содержимое которого можно просмотреть, выполнив следующие действия.

* Измените расширение файла с XPI на ZIP
* Извлеките содержимое файла с помощью любой стандартной утилиты распаковки, такой как WinZip (Windows, Mac), 7-Zip (Windows) или Apple Archive Utility (Mac).

### Установите файл XPI на Firefox Android

В основном людям интересно узнать, можно ли установить файлы XPI в Firefox на устройствах Android. На Android вы можете установить надстройку из файла XPI, найдя файл и открыв его в Firefox.
