## .EXO вариант №

Файл EXO представляет собой файл фрагмента видео, загруженный версией YouTube для Android в формате EXO. Когда пользователь хочет загрузить видео для воспроизведения в автономном режиме, видео делится на небольшие фрагменты, и каждый фрагмент загружается и сохраняется на устройстве пользователя в виде файла EXO. Эти несколько файлов EXO объединяются для автономного воспроизведения с помощью приложения YouTube.

## EXO Формат файла

Файлы EXO сохраняются в собственном формате файлов YouTube. Они сохраняются в виде сжатых зашифрованных файлов и могут быть открыты только с помощью приложения YouTube для Android. Все несколько загруженных фрагментов собираются и расшифровываются приложением для воспроизведения в автономном режиме. Будучи зашифрованными, файлы EXO не могут быть преобразованы в какие-либо другие форматы файлов, такие как [MP4](/ru/video/mp4/).
