## .PZ вариант №

Файл PZ — это файл проекта, созданный с помощью онлайн-приложения для создания видеоклипов **Panzoid**. Он содержит информацию о видеопроекте и его содержимом, таком как сцена, объекты, эффекты, освещение, спецэффекты, звук и музыка. Panzoid позволяет создавать видео в Интернете с помощью средства редактирования видео **ClipMaker**.

Файлы PZ можно открывать с помощью Panzoid ClipMaker и Panzoid Backgrounder.

## PZ-формат файла

Файлы PZ сохраняются как двоичные файлы на диск. Когда пользователь создает новый проект с помощью [Panzoid](https://panzoid.com/), создается файл PZ, в котором сохраняются детали определенных пользователем объектов. Эта же информация синхронизируется с другими пользователями, если в проекте есть другие участники. Затем файлы PZ можно загрузить на компьютер.

Если вы хотите открыть файл PZ, вы можете снова загрузить его в **Panzoid**, используя опцию «Загрузить» в онлайн-интерфейсе. Файл загружается онлайн, и его содержимое загружается для модификации.

### Видеоклипы, поддерживаемые файлом PZ

У Panzoid есть программы ClipMaker и Backgrounder, которые используются вместе для создания видеороликов. С Panzoid можно создавать несколько различных типов видео, включая настраиваемые фоны, обложку канала YouTube, видео в социальных сетях и миниатюры.
