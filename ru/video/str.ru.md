## .STR вариант №

Файл STR — это файл записи, созданный YouTube для прямой трансляции, организованной пользователем. Файл автоматически сохраняется на YouTube после завершения прямой трансляции. Потоковое видео в реальном времени может транслироваться на несколько каналов, чтобы аудитория могла взаимодействовать с издателем посредством комментариев. Эти файлы SRT доступны с помощью API прямой трансляции YouTube, который обеспечивает доступ к серверам YouTube.

## Формат файла STR

YouTube сохраняет файлы STR на сервере в двоичном формате. Однако сведения о внутренней файловой структуре файлов STR недоступны публично.
