## .MPV вариант №

Файл MPV представляет собой видеофайл элементарного потока MPEG, который состоит из необработанных несжатых видеоданных. Видеосодержание внутри файла MPV включает в себя встроенные спецификации сжатия MPEG-2. Файлы MPV не содержат аудиоданных. Формат файла MPV также используется в качестве входных данных для приложений обработки видео, поскольку элементарный поток [MPEG](/ru/video/mpg/) в основном использует стандарты кодирования MPEG-2. Файлы MPV можно открывать и воспроизводить с помощью программных приложений, поддерживающих форматы видеофайлов MPEG-2. Программное обеспечение медиаплеера VideoLAN LVC является одним из таких программ, которое может открывать и воспроизводить файлы MPV.

## Формат файла MPV

Файлы MPV сохраняются на диск как простые двоичные файлы в соответствии со структурой формата файлов MPEG. Однако их внутренний формат файла недоступен для разработчиков.

## Использованная литература ##

- [MPEG-1 - Википедия](https://en.wikipedia.org/wiki/MPEG-1)
