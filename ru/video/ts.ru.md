## .TS вариант №

Файл с расширением .ts представляет собой видеопоток, хранящий данные на DVD-дисках. Он использует алгоритм сжатия MPEG-2 ([.mpeg](/ru/video/mpg/)) для достижения максимальной эффективности и совместимости с различными типами мультимедиа, такими как потоковое интернет-вещание и вещание. Формат файла TS был создан для того, чтобы его можно было воспроизводить на устройствах с плохим подключением к Интернету, таких как Smart TV.

## Формат файла TS — дополнительная информация

Данные в файле TS аналогичны файлу [MP4](/ru/video/mp4/), но разделены на крошечные фрагменты. Каждый фрагмент состоит из крошечного фрагмента видео, за которым следует небольшой фрагмент аудио и необязательная подпись. Один файл TS состоит из множества таких фрагментов. В дополнение к видео, аудио и подписи, каждый фрагмент также содержит некоторые дополнительные данные для обнаружения ошибок в фрагменте. Из-за этого файлы TS имеют несколько больший размер.

### Зачем использовать формат файла TS?

Итак, если файлы TS несколько велики по размеру, какие преимущества дает их использование вместо файлов других форматов? Ну, в вещании крошечные фрагменты видео и аудио могут быть отправлены по средствам связи (проводным или радио) в режиме реального времени. Дополнительные данные в блоках используются получателем для пропуска подверженных ошибкам блоков.

Кроме того, используются файлы TS, поскольку системе вещания не требуется весь поток для его воспроизведения. Передачу можно принимать и использовать в режиме реального времени путем сборки аудио и видео.

## Как воспроизводить файлы TS?

Что ж, файлы TS можно открывать и воспроизводить в популярном [медиаплеере VideoLAN VLC] (https://www.videolan.org/vlc/features.html), который доступен для бесплатной загрузки.
