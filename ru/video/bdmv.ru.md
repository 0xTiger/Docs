## .BDMV вариант №

Файл BDMV — это тип формата файла, используемый на видеодисках Blu-ray Disc (BD). BDMV означает «Blu-ray Disc Movie» и представляет собой формат контейнера, используемый для хранения видео, аудио и другого сопутствующего контента высокой четкости на дисках Blu-ray. Файлы BDMV являются частью структуры каталогов диска Blu-ray и содержат различные компоненты фильма Blu-ray, включая видеопотоки, звуковые дорожки, субтитры и структуры меню.

## Ключевые компоненты каталога BDMV:

1. **Папка STREAM**: эта папка обычно содержит видео- и аудиопотоки фильма. Видео обычно хранится в виде файлов M2TS (транспортный поток MPEG-2), а аудио может быть в таких форматах, как Dolby TrueHD или DTS-HD Master Audio.

2. **Папка CLIPINF**: эта папка содержит информацию об отдельных видеоклипах (файлах M2TS) на диске.

3. **Папка PLAYLIST**: содержит файлы списков воспроизведения, определяющие порядок воспроизведения видеоклипов. Диски Blu-ray часто содержат несколько списков воспроизведения для разных вариантов просмотра или версий фильма.

4. **Папка AUXDATA**: в этой папке хранятся вспомогательные данные и метаданные, связанные с фильмом.

5. **Папка BDJO**: содержит интерактивные структуры меню на основе Java для дисков Blu-ray. Эти меню позволяют использовать такие функции, как выбор сцены, специальные функции и многое другое.

6. **Папка JAR**: файлы Java Archive, используемые для интерактивных меню и специальных функций.

7. **Файлы INDEX.BDM и MOVIEOBJ.BDM**: эти файлы содержат информацию о структуре диска и навигации.

## Как открыть файл BDMV?

Программы, открывающие файлы BDMV,

- **VLC Media Player**: VLC — это универсальный медиаплеер, который поддерживает широкий спектр мультимедийных форматов, включая файлы BDMV. Просто откройте VLC, перейдите в меню «Медиа», выберите «Открыть папку» и перейдите в каталог BDMV. VLC должен автоматически обнаружить и воспроизвести контент.

- **PowerDVD**: PowerDVD — это коммерческое программное обеспечение проигрывателя Blu-ray, предназначенное для высококачественного воспроизведения дисков Blu-ray и файлов BDMV. Он предлагает первоклассные возможности с поддержкой различных функций Blu-ray, включая меню и специальные функции.

- **CyberLink PowerDirector**: если вы хотите редактировать файлы BDMV, CyberLink PowerDirector — это программное обеспечение для редактирования видео, которое может импортировать файлы BDMV и работать с ними. Это полезно для редактирования видео и создания дисков Blu-ray.

- **Leawo Blu-ray Player**: Leawo Blu-ray Player — это бесплатное программное обеспечение проигрывателя Blu-ray, поддерживающее файлы BDMV. Он обеспечивает простой интерфейс для воспроизведения контента Blu-ray на вашем компьютере.

- **AnyDVD HD**: AnyDVD HD — это не проигрыватель, а инструмент, снимающий защиту от копирования с дисков Blu-ray, включая файлы BDMV. Затем вы можете использовать другое программное обеспечение проигрывателя для открытия и воспроизведения расшифрованных файлов BDMV.

- **ArcSoft TotalMedia Theatre**: это еще один коммерческий проигрыватель Blu-ray, который может обрабатывать файлы BDMV и обеспечивает превосходное качество воспроизведения.
