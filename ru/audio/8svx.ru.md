## 8SVX вариант № ##

8SVX — это стандартный формат аудиофайлов, разработанный Electronic Arts. Он считается подтипом данных формата файла IFF. Файл используется для хранения 8-битных аудиоданных в фрагментах. В отдельных фрагментах дескриптора содержатся метаданные о потоке данных 8SVX. Это формат цифрового аудиофайла, который используется компьютерами Amiga. Компьютерные системы Amiga в настоящее время используются редко, поэтому эти файлы 8SVX обычно используются для сохранения и прослушивания старой музыки.

## Формат файла 8SVX

8SVX может хранить аудиозаписи с 8-битным качеством, полученным из спецификации [IFF](/ru/audio/aiff/). Это файлы, содержащие аудиозаписи, как и другие аудиофайлы. Он также включал контейнерный формат IFF. В этом формате несколько форматов данных были встроены в один файл и использовались для дальнейшей детализации включенных в него потоков. Их можно использовать отдельно в файловых контейнерах только для аудио файлов IFF. С другой стороны, мультиплексирование с другими подтипами IFF может привести к анимированным видеопотокам.

## Краткая история формата 8SVX

Первоначально он был выпущен в 1985 году Стивом Хейсом и Джерри Моррисоном. Еще один архивный формат был разработан в 2009 году. Этот формат файлов использовался в компьютерных системах Amiga для хранения старой музыки. Он также известен как 8-битный звуковой файл Amiga. Звуковое программное обеспечение, созданное для компьютеров Amiga, поддерживает этот формат файла. Первоначально он использовался многими программами для редактирования звука. Большинство его потоков данных закодированы с использованием потоков PCM. Модифицированная версия Amiga предоставила структуру декодера 8SVX. Форматы RIFF и WAVE, совместно разработанные Microsoft и IBM, созданы под влиянием формата контейнера 8SVX/IFF.

## Спецификации формата файла 8SVX

Многие программы связаны с этими файлами в Windows и Mac OS. Многие программы могут быть использованы для обработки этих файлов. Это закодированный аудиоформат для Commodore Amiga. Эти файлы поддерживают такие функции, как атака, повтор раздела и выпуск. Эти функции используются для хранения сэмплов музыкальных инструментов. Доступно сжатие данных с потерями Фибоначчи-дельта, что приводит к коэффициенту сжатия 50% и снижению точности.
