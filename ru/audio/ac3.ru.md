## .AC3 вариант №

Файл с расширением .ac3 представляет собой файл Audio Codec 3, представленный Dolby Laboratories. Это аудиоформат, который может содержать до шести каналов аудиовыхода. Первоначально этот формат использовался для аудио, но теперь он также используется для других приложений, таких как вещание HDTV, DVD, диски Blue-ray и игровые приставки. Приложения, которые могут открывать файлы AC3, включают проигрыватель Apple QuickTime, проигрыватель Microsoft Windows Media, Winamp, MPlayer и другие подобные.

## Формат файла AC3

Файлы AC3 являются двоичными по своей природе и основаны на модифицированном дискретном косинусном преобразовании (MDCT), который представляет собой алгоритм сжатия с потерями. Dolby Laboratories использовали алгоритм MDCT вместе с принципами перцептивного кодирования для разработки аудиоформата AC-3. Это привело к выпуску формата AC-3 в качестве стандарта Dolby Digital в 1991 году. Формат поддерживает предоставление пяти каналов для динамиков нормального диапазона (20–20 000 Гц) и одного канала (20–120 Гц) для низкочастотные эффекты, управляемые сабвуфером. AC3 поддерживает частоту дискретизации до 48 кГц.

### Технические детали формата файла AC3

Технология Dolby Digital использует очень эффективный способ сжатия многоканальных аудиофайлов без ухудшения качества звука. Это сжатие приводит к уменьшению размера файла, что упрощает распространение звуковых файлов. Используя Dolby Digital, можно включить полный 5.1-канальный аудиомикс на кинопленку или DVD.
