## .ABS вариант №

Файл с расширением .abs представляет собой базу данных, созданную и используемую базой данных Absolute, которая является ядром базы данных для среды разработки программного обеспечения Delphi. Он заменил Borland Database Engine (DBE) и стал более компактным, высокоскоростным, надежным и простым в использовании. База данных ABS компилируется как часть EXE-файла приложения, что делает ваше приложение быстрее и меньше. Данные хранятся в файле ABS в структурированном реляционном формате.

## Формат файла ABS — дополнительная информация

Файлы ABS хранятся как часть [EXE-файла] приложения (/ru/executable/exe/), который обычно хранится в виде двоичного файла. Бинарные файлы невозможно открыть ни в одном текстовом редакторе и обеспечить быстрый доступ к содержимому файла базы данных.

### Ключевые особенности формата файла ABS

Файлы ABS довольно просты и встраиваются в исполняемый файл приложения. Он имеет следующие особенности:

* Нет БДЭ; нет DLL
* Однофайловая база данных
* Поддержка SQL'92 (DDL и DML).
* Совместимость со стандартными и сторонними элементами управления базами данных.
* Однопользовательский и многопользовательский режим (файл-сервер)
* Прекрасно работает на всех версиях Windows — от 98 до последней, не требует никаких обновлений или пакетов обновлений.
* Сверхбыстрые таблицы в памяти
* Непревзойденная простота использования
* Надежное шифрование
* Сжатие BLOB-объектов
* Бесплатно для личного использования.
* Доступен полный исходный код
* Распространение без лицензионных отчислений
