## .ADF вариант №

Файл с расширением .adf представляет собой файл базы данных, созданный с помощью программного обеспечения ACT CRM (Customer Relationship Management) для хранения данных о взаимоотношениях с клиентами. Он хранит информацию об имени клиента, адресе электронной почты, контактной информации, бизнес-информации и истории отношений. Системы CRM чаще интегрируются с колл-центрами, созданными предприятиями, для автоматического получения информации о пользователях для входящих звонков. Файлы ADF можно открывать с помощью программного обеспечения SwiftPage ACT в ОС Windows.

## Формат файла ADF — дополнительная информация

Файлы ADF сохраняются на диск как двоичные файлы, и информация об их внутренней структуре недоступна для общественности. Другой файл с расширением .alf сохраняется вместе с файлами ADF, которые содержат метаданные, необходимые для навигации, резервного копирования и восстановления ACT! база данных.
