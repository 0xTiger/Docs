## .BTR вариант №

Файл с расширением .btr представляет собой файл базы данных, созданный приложением транзакционной базы данных [Btrieve](https://www.actian.com/). В отличие от систем управления реляционными базами данных (RBMS), Btrieve основан на методе индексированного последовательного доступа (ISAM), который представляет собой способ хранения данных для быстрого поиска. Файл BTR хранит набор записей и используется для создания отчетов в соответствии с требованиями. Файлы BTR можно открыть с помощью Pervasive Btrieve Database Manager, Pervasive PSQL Maintenance Utility и Legend Software BTRIEVE Viewer.

## Структура файла BTR — дополнительная информация

Внутренняя структура данных и выравнивание каждого байта в структуре файла BTR не являются общедоступными. Тем не менее,
предоставляет [Btrieve API](https://docs.actian.com/psql/PSQLv13/index.html#page/btrieveapi%2Fbtrintro.htm), который можно использовать для чтения информации из файлов BTR. Он совместим со следующими языками программирования и средами разработки.

* Эмбаркадеро С/С++
* Эмбаркадеро Дельфи
* ГНУ С/С++
* Микрофокус КОБОЛ
* Microsoft Visual Basic
*Майкрософт Визуал С++
* Уотком С/С++

Извлечение данных из файла BTR зависит от связанного с ним файла DDF. Без DDF получить информацию из такого файла будет непросто.
