База данных представляет собой набор данных, которые организованы в виде таблиц и хранятся на запоминающем устройстве компьютера и доступны в электронном виде. Реляционные базы данных хранят данные в таблицах базы данных, которые связаны друг с другом с помощью методов формального проектирования и моделирования. Данные в базе данных анализируются с помощью системы управления базами данных (СУБД), которая является основным источником взаимодействия с конечными пользователями, приложениями и самой базой данных.

Популярные расширения файлов баз данных и их форматы файлов включают **SQLite**, **DB**, **ACCDB** и **MDB**.
