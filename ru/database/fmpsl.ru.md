## .FMPSL вариант №

Файл FMPSL — это файл моментального снимка базы данных, созданный с помощью FileMaker Pro 12. В нем хранятся запрошенные результаты из базы данных вместе с другой информацией, такой как визуальный макет и видимая информация. Файл FMPSL можно использовать для восстановления всех этих данных в другом экземпляре базы данных FileMaker Pro. Этот формат файла был введен с запуском FileMaker Pro v 12. До этой версии ссылки на моментальные снимки были представлены как формат файлов FPSL в FileMaker Pro v 11. Файлы FMPSL можно открывать с помощью программного обеспечения FileMaker Pro Advanced. Другие форматы файлов, поддерживаемые FileMaker Pro, включают [FP5](/ru/database/fp5/), [FP7](/ru/database/fp7/) и [FMP12](/ru/database/fmp12/).

### Как сохранить записи в виде файла FMPSL?

Данные из базы данных FileMaker Pro можно сохранить в виде файла ссылки на моментальный снимок, выполнив следующие действия.

1. Найдите в базе данных записи, которые необходимо сохранить в виде файла ссылки на моментальный снимок.
1. Используя пункт меню «Отправить запись как ссылку на снимок», сохраните файл, введя имя файла.
1. Используйте просматриваемые записи, чтобы сохранить весь найденный набор записей.

Сгенерированный файл снимка будет сохранен на диск с указанным именем.
