## .FMP12 вариант №

Файл FMP12 — это файл базы данных, созданный с помощью программы разработки баз данных FileMaker Pro 12. Он содержит общую структуру базы данных (также известную как схема базы данных) вместе с данными внутри нее. Структура файла FMP12 основана на таблицах, каждая из которых состоит из нескольких полей базы данных, имеющих разные типы данных. Некоторые типы данных, поддерживаемые форматом файла FMP12, включают текст, настраиваемые формы, отчеты, дату и время, большие двоичные объекты и другие объекты базы данных. Старые форматы файлов FileMaker Pro, такие как [FP7](/ru/database/fp7/), можно обновить до FMP12.

## Как конвертировать старые файлы FileMaker Pro в FMP12?

Старые форматы файлов FileMaker Pro, такие как базы данных 11, 10, 9, 8 и 7 (.fp7), в форматы файлов FMP12. В FileMaker есть подробные рекомендации по [преобразованию форматов файлов предыдущей версии FileMaker Pro в FMP12](https://fmhelp.filemaker.com/help/16/fmp/en/index.html#page/FMP_Help/converting-files.html) .
