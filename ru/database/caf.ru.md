## .CAF вариант №

Файл CAF представляет собой файл каталога, созданный с помощью программного обеспечения для организации мультимедиа **Cathy**. Он содержит список индексированного каталога, указывающий на все доступные носители на внешних устройствах, таких как флэш-диски и компакт-диски. Эти файлы каталога помогут вам поддерживать чистый список носителей, который легко доступен для поиска и доступа. Файлы CAF предлагают поиск мультимедиа по имени, размеру и дате.

## Формат файла CAF

Файлы CAF хранятся в текстовом формате. Это означает, что вы можете открывать файлы CAF в любом текстовом редакторе, например Notepad, Notepad++ и Apple TextEdit. Кэти предоставляет возможность экспорта структуры каталогов каждого каталога в формат TXT, создания списков папок и файлов.

### Открыть файл CAF

Вы можете открыть CAF с помощью официального программного обеспечения Cathy. Для этого вам потребуется поместить файл CAF в тот же каталог, где находится файл Cathy.exe. Кэти автоматически покажет каталог при запуске.

Для этого поместите файл CAF в тот же каталог, что и файл Cathy.exe. Затем откройте Кэти. Ваш каталог CAF появится в Cathy.
