## .BC3 вариант №

Файл BC3 представляет собой файл базы данных для хранения информации, связанной с форматом обмена строительной базой данных. Он был разработан на основе стандартного испанского формата обмена базами данных по строительству. В файле BC3 хранится информация, содержащая записи данных, включая подробные сведения о строительных материалах, стоимости материалов и цене рабочей силы. Данные могут обмениваться между другими приложениями без преобразования данных в другие форматы.

## Формат файла BC3

Файлы BC3 хранятся в виде двоичных файлов с деталями, указанными в соответствии со стандартами FIEBDC. Реализация файлов BC3 претерпела различные версии, и существуют следующие версии формата файлов BC3:

* ФИЭБДК-3/2004
* ФИЭБДК-3/2007
* ФИЭБДК-3/2012
* ФИЭБДК-3/2016

FIEBDC — это сокращение от Formato de Intercambio Estándar — Bases de Datos de Construcción на испанском языке.
