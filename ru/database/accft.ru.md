## .ACCFT вариант №

Файл с расширением .accft представляет собой файл шаблона Microsoft Access, который содержит определения типов данных полей для таблиц базы данных. Microsoft Access — это настольная система управления реляционными базами данных (RDBMS), которая используется для создания баз данных, форм, запросов и представлений на основе таблиц. Файл ACCFT можно использовать для определения типов данных для полей таблицы, а таблицы базы данных можно создавать с использованием этого файла шаблона в качестве отправной точки. Его можно открыть с помощью Microsoft Access 2016 для создания новых таблиц. Файлы ACCFT распознаются по типу MIME **application/msaccess**.

## Формат файла ACCFT

Файлы ACCFT — это двоичные файлы, спецификации формата файлов которых не доступны публично. Его внутренние детали, такие как информация заголовка, файловая структура, нижние колонтитулы и т. д., в настоящее время недоступны.
