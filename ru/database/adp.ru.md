## .ADP вариант №

Файл с расширением .adp представляет собой файл проекта Microsoft Access, в котором используется архитектура компонентов OLE DB для обеспечения прямого и эффективного подключения к базе данных Microsoft SQL Server. Такие файлы не содержат реальных таблиц, диаграмм базы данных или других элементов базы данных. Файлы ADP можно создавать с помощью Access 2007 и 2010. Файлы, созданные с помощью более ранних версий Access, также можно открывать в Access 2007 и 2010. Файлы ADP можно открывать с помощью Microsoft Access 365.

## Формат файла ADP — дополнительная информация

Проект Access, созданный как ADP, позволяет вносить изменения в структуру объектов SQL Server, таких как таблицы и представления. Он также позволяет создавать, редактировать и использовать функции SQL, такие как диаграммы базы данных, хранимые процедуры и пользовательские функции. Это дает преимущество перед связыванием с базой данных SQL-сервера, где вы не можете вносить изменения в дизайн и можете связываться только с таблицами и представлениями SQL-сервера.

### Типы данных и поддержка диаграмм

**Типы данных даты/времени**

В Access 2010 поддерживаются следующие новые типы данных даты и времени.

* ВРЕМЯ
* СВИДАНИЕ
* ДАТАВРЕМЯ2
* ДАТАВРЕМЯСМЕЩЕНИЕ

**Типы данных переменной длины**

В проектах Access 2010 можно использовать следующие типы данных переменной длины:

* ВАРБИН(МАКС)
* ВАРЧАР(МАКС)
* NVARCHAR(МАКС)
