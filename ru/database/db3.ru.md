## Что такое файл DB3?
Файл DB3 — это файл базы данных, созданный программным обеспечением SQLite, которое представляет собой облегченную автономную программу базы данных, которая создает базы данных с использованием простых файлов; содержит анатомию базы данных, а также записи данных; обычно используется для извлечения или хранения структурированных данных с помощью SQL. Эти файлы можно использовать в интеллектуальных устройствах или там, где требуется ведение записей или другое управление данными, но в среде с ограниченным пространством.


## Формат файла DB3
Формат файла DB3 связан с RDBMS (системой управления реляционными базами данных) SQLite, популярным выбором для встроенных баз данных. В спецификации SQLite_3 не определено какое-либо конкретное расширение файла, он может использовать расширения, включая [.dbf](/ru/database/dbf/) и [.sqlite](/ru/database/sqlite/).

### Секретация базы данных
Обычно формат файла db, связанный с SQLite версии 3, можно рассматривать как формат файла DB3, используемый в качестве общедоступного собственного формата для ядра базы данных SQLite с июня 2004 года. и архитектуры с прямым порядком байтов или 32-битные и 64-битные системы. Эти функции делают DB3 популярным форматом файлов приложений. Основной файл базы данных SQLite_3 (DB3) состоит из одной или нескольких страниц. Все размеры всех страниц одинаковы в одной базе данных. Размер страницы в байтах равен степени двойки от 512 до 65536 включительно.
