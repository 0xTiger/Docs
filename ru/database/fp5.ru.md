## .FP5 вариант №

Файл FP5 — это файл базы данных, созданный с помощью FileMaker Pro версии 5 или 6. В нем хранятся как пользовательские, так и организационные данные в виде записей внутри таблиц. FileMaker Pro — это система управления базами данных, поддерживающая операционные системы Macintosh и Windows. FP5 — это более старый формат файлов, который поддерживался доступной на тот момент версией FileMaker Pro. Последним форматом базы данных FileMaker Pro является [FMP12](/ru/database/fmp12/), который был представлен с запуском FileMaker Pro версии 12. Другой широко используемый формат файла базы данных — [FP7](/ru/database/fp7/), который был заменен на ФМП12.

## Формат файла FP5 — дополнительная информация

Сведения о внутреннем формате файла FP5 недоступны для разработчиков. Старые форматы файлов базы данных FileMaker Pro, такие как [FP5 и FP7, можно обновить до последней версии FMP12](https://fmhelp.filemaker.com/help/16/fmp/en/index.html#page/FMP_Help/converting-files. html), используя подробные рекомендации FileMaker Pro.
