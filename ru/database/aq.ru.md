## .AQ вариант №

Файл с расширением .aq представляет собой файл базы данных, созданный с помощью приложения Ancestral Quest компании Incline Software, LC. Он хранит информацию о генеалогическом древе с использованием генеалогических данных, введенных людьми, использующими бесплатную и расширенную версии программного обеспечения. В базе данных хранится такая информация, как имена, места рождения и отношения членов семьи. Ancestral Quest использует эту информацию для связывания и классификации такой информации, как родословная, потомки, семья и временная шкала. Файлы AQ можно открыть с помощью [Incline Software Ancestral Quest](https://www.ancquest.com/index.htm).

## Формат файла AQ — дополнительная информация

Файлы AQ сохраняются как двоичные файлы, и их внутренний формат файла недоступен.
