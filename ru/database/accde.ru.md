## .ACCDE вариант №

Файл с расширением .accde — это формат файла Microsoft Access 2007 для создания файлов базы данных Execute Only для защиты пользовательского кода базы данных. Он компилирует весь код VBA только в исполняемый файл, чтобы его нельзя было просмотреть или отредактировать. Формат файла ACCDE заменяет формат файла MDE, который использовался в более ранних версиях Microsoft Access. ACCDE - это файлы [ACCDB](/ru/database/accdb/) в режиме только исполняемого файла и более компактные из-за отсутствия какого-либо исходного кода. Компактный размер этих файлов улучшает использование памяти, а также повышает производительность приложений. Файлы ACCDE можно открывать с помощью Microsoft Access 365.

## Формат файла ACCDE

Все файлы Microsoft Access 2007+ используют Jet ACE (движок базы данных Access), использующий семейство форматов файлов ACCDB. Однако Microsoft не предоставляет реализацию с открытым исходным кодом или подробную техническую спецификацию.
