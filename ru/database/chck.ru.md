## .CHCK вариант №

Файл CHCK создается в результате проверок работоспособности на сервере Microsoft Exchange. Он содержит информацию об общем состоянии и работоспособности базы данных сервера. Microsoft Exchange Server используется в основном как сервер электронной почты и позволяет вам хранить деловые контакты и календари. Администраторы сервера Exchange используют EseUtil для проверки работоспособности базы данных.

## CHCK — формат файла

Внутренний формат файла CHCK неизвестен, и нет справочных материалов для чтения/записи файлов CHCK.
