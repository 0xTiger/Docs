## .ASK вариант №

Файл с расширением .ask представляет собой файл базы данных, созданный [askSam Systems](https://asksam.en.softonic.com/), приложением для управления личной информацией (PIM). Файлы этого типа хранятся в динамических папках и содержат пользовательские данные, такие как сообщения электронной почты, документы, текстовые файлы, электронные таблицы, адреса и веб-страницы. Это позволяет программному обеспечению askSam легко их искать и извлекать данные для исследовательских целей. Программное обеспечение askSam, хотя и представляло собой простое настольное приложение на основе форм, хранящее данные в базе данных, было популярно для организации и извлечения таких данных. Данные из файла ASK также можно экспортировать в [электронные таблицы](/ru/spreadsheet/).

## Формат файла ASK — дополнительная информация

Файлы ASK сохраняются на диске в проприетарном двоичном формате, детали внутреннего формата которого недоступны для разработчиков.
