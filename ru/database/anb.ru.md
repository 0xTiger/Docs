## .ANB вариант №

Файл с расширением .anb представляет собой диаграмму IBM i2 Analyst's Notebook, содержащую информацию о взаимосвязях между различными данными. Как следует из названия, программное обеспечение Analyst's Notebook помогает связать данные, в результате чего получается более краткая форма связи между разнородными фрагментами информации и предоставляется аналитический вывод для анализа. Это помогает выявлять и пресекать вредоносные действия, такие как криминальные, кибернетические и мошеннические угрозы. Файлы ANB можно открывать с помощью программного обеспечения IBM i2Analyst Notebook.

## Формат файла ANB — дополнительная информация

Файл ANB состоит из следующих составных элементов:

* Сущности - Представление фрагментов информации
* Ссылки - Показать связь между сущностями
* Свойства - Описание сущности или ссылки более подробно

Файлы ANB хранят всю эту информацию на основе входной информации, поступающей в Блокнот аналитика ANB. На основе введенных данных программное обеспечение создает связи между каждой частью информации, что помогает наметить взаимосвязи и отслеживать доказательства в геопространственном и временном масштабе.
