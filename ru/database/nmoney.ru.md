## .NMONEY вариант №

Файл NMONEY представляет собой файл базы данных хранилища финансовых данных, который содержит записи о финансовых транзакциях. Он был разработан Nickvision и использовался в программном обеспечении Nickvision Denaro, которое представляет собой программное обеспечение для личных финансов. Данные, хранящиеся в файле NOMONEY, включают информацию о кредитных и дебетовых транзакциях по счету.

Файлы NOMONEY можно открыть с помощью приложения [Github](https://github.com/NickvisionApps/Denaro).

## О программном обеспечении Денаро

Denaro изначально был разработан [Nickvision](https://nickvision.org/) как продукт, который позже был преобразован в программное приложение с открытым исходным кодом, размещенное на [Github](https://github.com/NickvisionApps/Denaro). . С тех пор приложение модифицировалось и обновлялось только на Github. Приложение разработано на C# в пользовательском интерфейсе Windows с использованием Windows App SDK (на данный момент WinUI 3).

### Возможности, предлагаемые Denaro

* Управляйте несколькими учетными записями одновременно с помощью самого популярного и знакомого интерфейса вкладок.
* Фильтрация транзакций по типу, группе или дате
* Повторяющиеся транзакции, такие как счета, которые происходят каждый месяц.
* Перевести деньги с одного счета на другой
* Экспортируйте данные из учетной записи в виде файла CSV и импортируйте файл CSV, OFX или QIF для добавления транзакций в учетную запись.

## Формат файла Denaro — дополнительная информация

Файлы Denaro сохраняются на диск в двоичном формате. Финансовые данные хранятся в виде файла базы данных в формате **.SQLITE3**. SQLITE — это облегченный файл базы данных SQL, созданный с помощью программного обеспечения SQLite. Он предоставляет автономный механизм базы данных, способный предоставить полнофункциональную и высоконадежную базу данных. Файлы базы данных SQLite можно использовать для обмена богатым содержимым между системами путем простого обмена этими файлами по сети. Привязки SQLite существуют для таких языков программирования, как C, [C#](/ru/programming/cs/), [C++](/ru/programming/cpp/), [Java](/ru/programming/java/), [PHP](/ru/programming/ php/) и многие другие.
