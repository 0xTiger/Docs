## .4DL вариант №

Файл журнала 4DL используется для записи изменений, внесенных в файл базы данных 4D (.4DD). Этот файл хранится вместе с файлом базы данных и имеет аналогичное имя. Его цель — точно отслеживать и вести полный учет обновлений, реализованных в базе данных. [Файл 4DD](/ru/database/4dd/), по сравнению с файлом 4DL, в основном связан с 4-м измерением компанией 4D Inc.

## Формат файла 4DL — дополнительная информация

Обычно несколько файлов 4DL хранятся вместе с файлом 4DD. Таким образом, первая версия файла журнала может называться 0001.4dl, но последующие версии файлов журнала будут сохранены как 0002.4dl, 0003.4dl и т. д. Теперь, если сам файл журнала будет сохранен 3 раза подряд, он будет помечен как «db1[0005-0003].4dl».
