## .ABX вариант №

Файл .abx — это адресная книга, экспортированная из программного обеспечения WordPerfect Office, разработанного **Corel**. WordPerfect — это программа для обработки текста, разработанная корпорацией Corel и используемая уже довольно давно. Файлы ABX содержали информацию о контактах, извлеченную из почтового клиента WordPerfect. Экспортированную информацию о контактах можно перенести на другие устройства и использовать соответствующим образом.

## Формат файла ABX — дополнительная информация

Файлы ABX хранятся в собственном собственном формате файлов, и сведения о внутреннем формате файлов ABX неизвестны. Позже почта WordPerfect была интегрирована в пакет WordPerfect Office.
