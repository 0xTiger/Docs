## .DAT вариант №
Файл с расширением .dat относится к общему формату файлов. Обычно они не зашифрованы и не двоичны, но могут быть практически любыми. Различные приложения используют файлы DAT в качестве ссылок на определенные операции в соответствующих программах. Поэтому мы не можем сказать, что конкретное программное обеспечение или программа отвечает за создание файлов DAT. К этим файлам, в частности, обращается только приложение, которое их создало.


## Формат DAT-файла
Формат файла DAT — это просто общий формат, который содержит любые данные в файле, обычно сохраняемом с расширением .dat. Он может содержать данные в двоичном или текстовом формате, обычно общие данные состоят из информации, специфичной для приложения, к которому они относятся. Чаще всего вы можете найти их с другими файлами конфигурации, такими как файлы DLL. Здесь мы можем перечислить некоторые известные типы файлов, имеющие расширение .dat:

### Видеофайл VCD
VCD были хорошо известным источником видео или мультимедиа в 1990-х годах для распространения фильмов и домашнего видео до DVD. Обычно файлы видео с диска VCD обнаруживались с расширением .dat. Таким образом, он сохранял фактические видеоданные, воспроизводимые с диска VCD.
### Вложение электронной почты Exchange
Файлы DAT, связанные с вложением электронной почты Exchange, содержат информацию о форматировании для сообщений в формате RTF, отправляемых с сервера Microsoft Exchange. Обычно эти файлы добавляются в конец электронных писем, когда почтовый клиент получателя не поддерживает RTF-сообщения. Эти файлы обычно находятся как **Winmail.dat**.
### Файл куста реестра Windows
NTUSER.DAT — это общий файл куста реестра. Реестр Microsoft Windows использует файлы DAT для хранения пользовательских настроек. Эти файлы DAT загружаются, когда пользователь входит в Windows. Он может включать настройки учетной записи, пароли и ассоциации файлов.
### Файл данных Minecraft
Игра Minecraft использует файлы DAT для сохранения информации об инвентаре игрока, позиции игрока и информации о мире. Файлы DAT также используются для хранения уровней, которые представляют собой небольшие фрагменты карты, которые загружаются по запросу, когда игрок перемещается по миру.
### Файл Piriform DAT
Такие приложения, как CCleaner, Speccy, Defraggler и Recuva, используют файлы DAT для получения информации о своих типичных операциях.
