## .DB вариант №

Файл с расширением db представляет собой универсальный файл базы данных для хранения данных. Для такого файла базы данных нет официальных спецификаций. Данные организованы в структурированном формате внутри файла в виде таблиц, полей, типов данных и значений полей. Файлы БД также можно создавать путем чтения из файлов таблицы данных, содержащих данные в строках и столбцах. Содержимое файлов БД можно дополнительно экспортировать в форматы файлов [CSV](/ru/spreadsheet/csv/) и [PDF](/ru/pdf/) для целей отчетности. Файлы DB можно открывать в таких приложениях, как SQLite и LibreOffice.
