## .TEACHER вариант №

Файл .TEACHER — это файл образовательных настроек, созданный программным обеспечением SMART Response. Он используется в образовательных учреждениях для обеспечения комплексной среды управления информацией о классе, списками занятий, расписаниями, праздниками, профилями учащихся, расписанием экзаменов и результатами оценок. Программное обеспечение для оценивания SMART Response™ устанавливается на компьютер, который будет использоваться для проведения оценивания SMART Response.

## УЧИТЕЛЬ Формат файла

Файлы TEACHER сохраняются на диске в двоичном формате. Его внутренние спецификации формата файлов недоступны для публичного использования для справки разработчика. Программное обеспечение SMART Response можно использовать для создания отчетов учащихся. Затем их можно будет использовать между школьными компьютерами.
