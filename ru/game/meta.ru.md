## .META вариант №

Файл META содержит данные различных игр, использующих Exient XGS Engine, таких как Angry Birds Go! или Angry Birds Transformers. Файл META собирает и сохраняет информацию об определенных аспектах игры, например журналы сбоев или журналы предметов.

МЕТА-файлы генерируются игрой автоматически и не предназначены для изменения пользователем или прямого доступа. Если файл META предназначен для регистрации сбоев в игре, он обычно будет называться «crashes.meta». Аналогично, если файл META используется для регистрации информации, связанной с элементами, он будет следовать соглашению об именах «XXX.xml.meta», где «XXX» представляет собой описательный идентификатор для определенного типа регистрируемых элементов.

## Внешний движок XGS

Компания Exient Entertainment создала игровой движок Exient XGS Engine. Это программная платформа, созданная для упрощения процесса создания и разработки видеоигр. Он предлагает программистам разнообразные инструменты, библиотеки и функции для ускорения процесса создания игр.

Разнообразие функций и возможностей, предоставляемых Exient XGS Engine, упрощают создание игр на различных платформах, включая ПК, консоли и мобильные устройства. Эти функции часто включают создание визуальных эффектов, моделирование физики, управление звуком, обработку ввода, наличие сетевых возможностей и другие вещи.
