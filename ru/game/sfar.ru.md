## .SFAR вариант №

Файл с расширением .sfar представляет собой файл данных игры, используемый Mass Effect 3 для хранения DLC (загружаемого контента) в сжатом проприетарном архиве. Mass Effect 3 — игра, созданная Electronic Arts, ведущей компанией-разработчиком игр. Контент DLC, хранящийся в файле SFAR, используется для расширения игры дополнительным контентом и функциями.

## Формат файла SFAR — дополнительная информация

Файлы SFAR хранятся/сохраняются как сжатые архивные файлы в двоичном формате. Они используют алгоритмы сжатия LZX и LZMA для сжатия содержимого. Файлы SFAR можно открыть с помощью редактора DLC, который поставляется вместе с ME3 Explorer. Помимо SFAR, DLC Editor может извлекать другие игровые файлы, такие как [PCC](/ru/game/pcc/).

## Спецификации формата файла SFAR

Файл SFAR разделен на следующие четыре основных фрагмента.

### Заголовок SFAR

Заголовок SFF содержит информацию о различных фрагментах, составляющих файл.

### Таблица файлов SFAR

Таблица файлов SFAR содержит список файловых записей, каждая из которых имеет длину 30 байт.

### Таблица размеров блоков

Размер блока содержит несколько записей, каждая из которых имеет длину 2 байта. Это значение указывает размер блока, соответствующий записи в таблице файлов.

### Блоки

Фрагменты блоков в файле SFAR содержат данные внутри файла SFAR.
