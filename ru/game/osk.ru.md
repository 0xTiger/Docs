## .OSK вариант №

Файл OSK представляет собой файл макета или скина, используемый ритм-игрой osu! где игроки используют мышь, клавиатуру синхронно с музыкой. Он содержит макет интерфейса редактора и может быть настроен игроками в соответствии со своими предпочтениями. осу! хранит данные карты битов в формате файла [OSZ](/ru/game/osz/).

**MIME Тип файла OSK Формат:** x-osu-skin-archive

Вы можете открыть файл OSK с помощью [osu!](https://osu.ppy.sh/home).

## Формат файла OSK

Файлы OSK сохраняются на диск в виде сжатых файлов [ZIP](/ru/compression/zip/). Вы можете просто сохранить файл .osk в .zip и извлечь архив, чтобы просмотреть его содержимое. Извлеченное содержимое необходимо поместить в каталог \Skins\ для загрузки в игру.

## Что можно настроить с помощью файла OSK?

Следующие игровые элементы можно настроить с помощью файла .osk.

* Хит круги
* Отображение очков
* Тип слайдера
* Курсор
* Хит всплески
