## .BMZ вариант №

Файл BMZ — это файл бонусной карты, используемый 3D-игрой-головоломкой Portal, разработанной Valve. Он содержит бонусную карту, которую также могут создавать сторонние разработчики сообщества и импортировать в игру с помощью встроенного пользовательского интерфейса. BMZ представляет собой архивный файл [ZIP](/ru/compression/zip/), внутри которого находятся файлы BSP, TGA и [BNS](/ru/game/bns/). Файлы BMZ можно открыть с помощью Valve Portal и Valve Portal 2.

## Формат файла БМЗ

BMZ сохраняются в виде файлов ZIP и могут быть извлечены с помощью стандартных утилит для извлечения ZIP, таких как WinZIP или WinRAR. При извлечении файла BMZ можно найти следующие файлы:

* BSP - файл карты
* TGA - файл логотипа
* BNS - Файл скрипта бонусной карты портала

## Как создать и установить файлы бонусной карты

Сообщество разработчиков Valve Software разработало рекомендации для людей, которые хотят [создать и установить Bonus Map](https://developer.valvesoftware.com/wiki/Bonus_Maps) файлы. Он включает в себя следующие шаги:

1. Добавление бонусных карт BSP в мод
1. Добавление миниатюры изображения
1. Создание файлов описания бонусов (.BNS)
