## .KODU вариант №

Файл .kudo — это файл 3D-игры, созданный и сохраненный с помощью Microsoft [программное обеспечение Kodu Game Lab](https://www.kodugamelab.com/), которая представляет собой образовательную среду разработки игр. Он предназначен для обучения детей основным принципам программирования и позволяет создателям программировать поведение игровых элементов с помощью визуального языка программирования. Файлы Kodu можно сохранять и делиться ими с другими пользователями, но открывать и запускать их можно только в программном обеспечении Kodu.

## Формат файла KODU — дополнительная информация

Файлы Kodu — это двоичные файлы, сохраненные в собственном формате файлов Microsoft. Его спецификации формата файлов недоступны для разработчиков, поэтому файлы Kodu можно запускать только в программном обеспечении Kodu Game Lab.
