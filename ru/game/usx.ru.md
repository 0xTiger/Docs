## .USX вариант №

Файл USX содержит сетки Unreal Engine, которые представляют собой предварительно разработанные 3D-объекты/модели, используемые для создания среды. Модели из файла USX имеют разное положение, ориентацию и размеры, но они остаются статичными. Статические сетки можно оптимизировать для повышения производительности и использовать для создания различных уровней детализации (LOD) для повышения производительности игры. Их также можно использовать для создания объемов столкновений и как часть физического моделирования.

Файлы USX можно создавать с помощью редактора уровней Unreal «UnrealEd».

## Формат файла USX — дополнительная информация

Файлы USX сохраняются на диск как двоичные файлы. Подробная информация о внутренней файловой структуре формата файла USX недоступна для разработчиков.
