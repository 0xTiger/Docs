## .SAV вариант №

Файл с расширением .sav представляет собой сохраненный файл игры Nintendo DS, который можно загрузить в эмуляторы игр Nintendo DS, такие как [DeSmuME] (http://desmume.org/) и [NO$GBA] (https://www. nogba.com/). Эти файлы сохраняют текущее состояние игры, которое затем можно загрузить в эмуляторах Nintendo. Информация, хранящаяся в файлах SAV, включает имя игрового персонажа, очки опыта, пройденные уровни и внутриигровые ресурсы. Файлы SAV можно загружать в эмуляторы, но разница в версиях файла SAV может привести к возникновению ошибок в эмуляторах. Файлы SAV можно открывать с помощью эмуляторов DeSmuME и NO$GBA.

## Формат файла SAV — дополнительная информация

Файлы Nintendo DS SAVE сохраняются на диск в двоичном формате. Хотя они не читаются человеком, для их открытия для анализа можно использовать редактор HexaDecimal, такой как [HxD](https://mh-nexus.de/en/hxd/). В случаях, когда файл SAV не может быть загружен в обновленную версию эмулятора, его можно открыть в шестнадцатеричном редакторе для удаления любых нежелательных дополненных данных.

## использованная литература

* [DeSmuME] (http://desmume.org/)
* [NO$GBA](https://www.nogba.com/)
