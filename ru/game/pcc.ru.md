## Формат файла PCC — дополнительная информация

Файлы PCC содержат сжатые и/или несжатые структуры, которые способствуют общей организации файлов.

### Сжатая структура PCC

Сжатый файл PCC состоит из таблиц и данных, разделенных на фрагменты. Каждый фрагмент содержит переменное количество сжатых блоков.

* `Chunks Header` — 16-байтовый общий заголовок, содержащий информацию о чанках.
* «Chunk Header» — 16-байтовый заголовок, содержащий информацию о необработанных сжатых данных, содержащихся в форме блока.

### Несжатая структура PCC

Несжатые файлы PCC разделены на следующие пять частей.

* `Заголовок` - содержит основную информацию о структуре файла PCC.
* «Таблица имен» — содержит имя, найденное внутри пакета, включая классы импорта, экспорты и свойства экспорта.
* «Таблица импорта» — содержит все классы и объекты, импортированные PCC.
* `Таблица экспорта` - содержит все объекты, хранящиеся в пакете. Каждый экспорт может различаться по размеру.
