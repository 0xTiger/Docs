## .SID вариант №

Файл SID — это файл резервной копии игры, созданный приложением Steam, который используется для загрузки и управления **видеоиграми Valve** в ОС Windows. Он хранит игровые данные в сжатом формате и может использоваться для резервного копирования игр на носители, такие как DVD и USB-накопители. Файл SID создается с помощью утилиты резервного копирования Steam и аналогичен формату файла [CSD](/ru/game/csd/) для хранения резервной копии игры.

## Формат файла SID — дополнительная информация

Файлы SID хранятся на диске в виде двоичных файлов. Он всегда сопровождается файлами .sis и .sim, которые используются для извлечения информации метаданных для восстановления данных игры из резервной копии.
