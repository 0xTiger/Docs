## .CT вариант №

Файл CT представляет собой файл чит-таблицы, созданный с помощью программного обеспечения [Cheat Engine](https://www.cheatengine.org/), который используется для создания модификаций игр на базе Windows для мошенничества. Cheat Engine, читерский движок с открытым исходным кодом, проверяет запущенные игры и записывает их адреса. Эта информация и ее переопределения записываются в файл CT, который затем загружается игроками для изменения свойств игры, таких как оценка здоровья, наивысший балл и оставшиеся жизни.

## Формат файла Cheat Engine CT

Файлы CT сохраняются с адресами и другой связанной информацией, которую можно взломать внутри игры. В большинстве случаев файлы сохраняются в виде сжатых архивов [ZIP](/ru/compression/zip/), которые можно легко извлечь с помощью любой стандартной утилиты распаковки, такой как WinZIP или 7-Zip.

## Как использовать чит-таблицы

1. Скачайте таблицу и скопируйте ее в папку Cheat Engine.
1. Запустите чит-движок
1. Запустить игру;
1. Используйте комбинацию ALT+TAB и выберите игру в списке процессов через Cheat Engine.
1. Если таблица находится в другой папке, просто нажмите Control+O и направьте Cheat Engine в эту папку. Затем выберите таблицу (обычно это имя_процесса.ct);
1. После загрузки таблицы, если есть скрипт, просто проверяем его.
1. ALT+TAB вернитесь в игру и получайте удовольствие.

## использованная литература

* [Чит-движок](https://www.cheatengine.org/)
