## .CLIP вариант №

Файл с расширением .clip представляет собой файл воспроизведения Grand Theft Auto 5 (GTA V), который содержит входные данные контроллера. Редактор Rockstar использует эти данные для воспроизведения последовательности игрового процесса. Это не настоящий видеоповтор, а запись действий контроллера с точностью до миллисекунды. Таким образом, настройки ракурсов камеры можно настроить через редактор. Однако редактор можно использовать для создания реального видео в популярных форматах видеофайлов, таких как [MP4](/ru/video/mp4/). GTA V сохраняет экспортированное видео в C:\Users\ВАШЕ ИМЯ ПОЛЬЗОВАТЕЛЯ\AppData\Local\Rockstar Games\GTA V\videos\rendered.

## Формат файла CLIP

Файлы CLIP не являются настоящими видеофайлами. Вместо этого они содержат игровые данные, такие как положение, значения вращения для объектов, состояния и условия мира, звуковые триггеры, движения игровой камеры, действия игрока и тому подобное. Их нельзя просматривать вне игры.

## Как записывать клипы и получать доступ к редактору Rockstar в GTA V?

После завершения записи клипа можно использовать следующие шаги для записи клипов.

* Пауза в игре
* Прокрутите до самой правой вкладки «Редактор Rockstar».
* Запустить редактор с этой вкладки
* Здесь вы сможете выбрать экран «Управление клипами» и внести изменения в клипы, которые вы записали в GTA Online или в сюжетном режиме.
