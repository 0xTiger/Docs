## .PUP вариант №

Файл PUP представляет собой файл исправления системного обновления, используемый системным программным обеспечением PlayStation 4 (PS4) и PlayStation 5 (PS5). Он обновляет прошивку игры до более новой версии, которая может содержать улучшения программного обеспечения с точки зрения исправления ошибок, новых функций операционной системы и улучшений производительности программного обеспечения. Файл PUP также может содержать исправления для исправления ошибок, которые в противном случае могут привести к проблемам с безопасностью. Файлы PUP можно загрузить в систему, подключив ее к Интернету или вручную загрузив обновление прошивки на компьютер, а затем перенеся на устройство с помощью USB-накопителя или другого запоминающего устройства. Более ранняя версия PlayStation, т.е. PS3, также использовала файл PUP для обновления прошивки.

## Формат файла PUP — дополнительная информация

Файлы PUP сохраняются на диск в [сжатом](/ru/compression/) двоичном формате файла. Однако подробности сжатия не доступны публично. Содержимое файла PUP можно извлечь с помощью программы PS3 PUP Extractor в ОС Windows. Чтобы обновить систему программного обеспечения PS3, файл PUP загружается и сохраняется в папке /PS3/UPDATE на машине. Это позволяет опции обновления системы найти файл обновления прошивки и запустить его.

### Соглашение об именах форматов файлов PUP

* PS3UPDAT.PUP — стандартное имя файла, используемое для обновлений PlayStation 3.
* PS4UPDATE.PUP — стандартное имя файла, используемое для обновлений PlayStation 4.
* PS5UPDATE.PUP — стандартное имя файла, используемое для обновлений PlayStation 5.

## Использование файла PUP для сброса настроек PlayStation

Файлы PUP можно использовать для сброса настроек устройства PlayStation. Это перезапишет все данные с устройства, удалит существующую прошивку и установит его как совершенно новое устройство. Сброс настроек PlayStation к заводским настройкам также удалит все сохраненные игровые данные.
