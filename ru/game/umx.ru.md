## .UMX вариант №

Файл UMX представляет собой файл музыкального пакета, используемый игрой Unreal Tournament First-Person Shooter, разработанной Epic Games, и содержит внутри файлы трекера фоновой музыки. Вот почему его часто считают просто еще одним расширением файла пакета, содержащим файлы отслеживания, такие как .s3m и .it. Музыкальный трекер — это разновидность музыкального секвенсора для создания коротких музыкальных файлов с использованием дискретных музыкальных нот. Различные варианты файлов UMX содержат звуковые эффекты, такие как взрывы или шаги. Файлы UMX работают с версиями, предшествующими выпуску Unreal Engline 4.

## Формат файла UMX — дополнительная информация

Файлы UMX сохраняются на диск в двоичном формате с использованием собственного формата файлов Epic, и сведения о его внутреннем формате файлов не доступны публично. Файлы UMX можно открывать с помощью приложений для отслеживания музыки, таких как XimpleMOD, и утилит Unreal, таких как Epic Games Unreal Tournament.
