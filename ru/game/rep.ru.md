## .REP вариант №

Файл с расширением .rep представляет собой файл воспроизведения Grand Theft Auto (GTA), который содержит захваченные данные кадра, которые можно воспроизвести в GTA. GTA — это гоночная и кампания с открытым миром, в которую играют игроки по сети. Файлы REP предоставляют механизм для обмена и анализа игрового процесса после завершения игры. Повторы в таком файле — это последовательность сцен и действий, которые хранятся в файле REP. Файлы REP хранятся в каталоге User Files и могут быть открыты с помощью GTA Replays Manager.

## Формат файла REP — дополнительная информация

Файлы GTA REP сохраняются на диск как двоичные файлы и не читаются человеком. Каждый файл .rep имеет строку заголовка и содержит переменное количество сцен (также называемых кадрами). Любая конкретная сцена содержит информацию об актерах, положении транспортных средств, погоде, вышивке и т. д. в любой момент времени. Поскольку экраны записываются один за другим, они создают такой же эффект, как и при воспроизведении фильма.

Спецификации формата файла REP доступны на [gta fandom](https://gta.fandom.com/wiki/Replay/Format_details), и на них можно ссылаться для получения дополнительной информации о его заголовках и блоках данных.
