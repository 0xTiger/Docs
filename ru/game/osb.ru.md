## .OSB вариант №

Файл OSB представляет собой файл раскадровки, созданный ритм-игрой Osu! который позволяет игрокам касаться визуальных элементов на экране в такт музыке. Раскадровка внутри файла OSB содержит настраиваемый анимированный фон, который используется во время игры. Фон раскадровки обычно представляет собой файл JPG или PNG, который отображается вместе с файлом битовой карты OSZ во время игры.

## Формат файла ОСБ

Файлы OSB хранятся в виде простых текстовых файлов на диске в папках битмапов. Эта схема позволяет создавать разные раскадровки с разной сложностью в битовой карте.
