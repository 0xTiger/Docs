## Apa itu file WFP?

File WFP adalah file tambalan yang digunakan oleh kartu synthesizer Turtle Beach WaveFront. Itu tidak lagi digunakan karena Turtle Beach telah berhenti memproduksi kartu suara dan synthesizer. File WFP berisi satu program atau tambalan yang dapat diterapkan ke penyintesis. File WFP juga dapat dikonversi ke format file [MP4](/id/video/mp4/).

## Format File WFP

Detail format file internal file WF tidak tersedia. Sudah lama pengerjaan file WFP dihentikan karena tidak ada lagi produksi synthesizer oleh Turtle Beach.
