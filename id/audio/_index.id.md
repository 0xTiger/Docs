Format file audio adalah kategori format file digital untuk representasi informasi audio beserta meta-datanya. Sejumlah format file Audio ada berdasarkan sifat data yang terkandung dalam file audio. File tersebut dapat disimpan dalam format file audio terkompresi maupun tidak terkompresi. Format file audio populer termasuk [MP3](/id/audio/mp3/), [WAV](/id/audio/wav/), PCM dan WMA.
