## Apa itu file ENC?

**File ENC** juga dapat mewakili komposisi musik yang dibuat menggunakan **Encore dari Passport Music Software**, sebuah aplikasi penulisan skor yang kompatibel dengan platform Windows dan macOS; dalam file ini, Anda akan menemukan permadani notasi musik yang kaya, mencakup elemen seperti not, birama, tanda birama, dan tanda kunci. Selain itu, file tersebut merangkum detail penting seperti judul musik dan informasi tentang komposer yang menghidupkan karya musik.

## Tentang Perangkat Lunak Encore

Encore, awalnya dikembangkan oleh Don Williams untuk Passport Designs Inc. pada tahun 1984, pada awalnya dirancang untuk Apple Macintosh. Pada tahun 1998, Lyrrus Inc., yang beroperasi sebagai GVOX, mengakuisisi kekayaan intelektual Passport, yang menyebabkan peluncuran Encore 5 satu dekade setelah Encore 4.

Aspek perintis Encore terletak pada perbedaannya sebagai salah satu program penulis skor paling awal yang memberdayakan pengguna untuk menambahkan dan mengedit elemen dalam skor musik dengan lancar melalui penggunaan mouse; fitur ini menandai kemajuan signifikan dalam antarmuka pengguna perangkat lunak notasi musik.

## Bagaimana cara membuka file ENC?

Anda dapat membuka file ENC yang terkait dengan komposisi musik menggunakan perangkat lunak ini

- **Encore Perangkat Lunak Musik Paspor** (Berbayar) untuk (Windows, Mac)
- **Myriad Harmony Assistant** (Uji Coba Gratis) untuk (Windows, Mac)

Dengan menggunakan perangkat lunak di atas, Anda juga dapat mengonversi file ENC ke format [.MXL](/audio/mxl/) (Compressed MusicXML).
