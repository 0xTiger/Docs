## Apa itu file ACT? ##

Format ACT atau Adobe Color Table (juga dikenal sebagai file Color Lookup Table) adalah jenis database Adobe Photoshop yang menyimpan sekumpulan warna yang telah ditentukan sebelumnya. Anda dapat menambahkan atau menghapus warna saat menyimpan gambar untuk penerbitan online guna memilih gambar dengan kualitas lebih tinggi atau ukuran file yang lebih kecil. Anda mungkin memiliki format Audio Terenkripsi ADPCM jika tidak digunakan dengan Illustrator. Modulasi Kode Pulsa Diferensial Adaptif adalah teknik kompresi audio yang digunakan oleh berbagai perangkat Musik dan perangkat pendengar.

Format file ACT juga digunakan oleh sistem File Alma CAD/CAM. Ini adalah pedoman yang digunakan oleh peralatan mesin 3D untuk mengetahui cara memotong apa pun. File Genesis3D Player, DS Games Developer Action, atau FoxPro Detailing Wizard Action Schematic adalah contoh ACT.

## Spesifikasi Format File ACT

Banyak perekam yang menggunakan sistem ACT hanya melakukannya untuk opsi perekaman terkecil; jika kualitas ditingkatkan, format terpisah, biasanya WAV, digunakan, menghasilkan dokumen yang jauh lebih besar.

Ada beberapa generasi ACT; per Juni 2009, trek yang dibuat oleh perangkat berikutnya tidak dapat dikenali oleh setiap aplikasi pemutar musik atau konverter standar gratis, hanya menyisakan alat MP3 yang dibundel.

Optimalisasi gambar untuk distribusi Web dilakukan dengan file ACT. Pelanggan dapat menguji kualitas gambar terhadap basis data gambar yang dihasilkan dengan menambah atau mengurangi jumlah warna di CLUT.

Dengan mengetuk opsi Tabel Warna dan memilih Muat Data Warna dari kotak dialog Ekspor untuk Web, tabel warna Adobe dapat diunggah. Direktori */Presets/en_US/Save* for Web Settings/Color Tables/ berisi beberapa file ACT yang telah dikonfigurasi sebelumnya yang disertakan dengan sistem Photoshop.

Skrip tindakan dapat dibuat hanya dalam Genesis 3D atau dengan mengubah model 3D lainnya ke standar ACT menggunakan Genesis3D Actor Designer
