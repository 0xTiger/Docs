## Apa itu berkas MP3?

File dengan ekstensi .mp3 adalah format file yang disandikan secara digital untuk file audio yang secara formal didasarkan pada MPEG-1 Audio Layer III atau MPEG-2 Audio Layer III. Ini dikembangkan oleh Moving Picture Experts Group (MPEG) yang menggunakan kompresi audio Layer 3. Kompresi yang dicapai oleh format file MP3 adalah 1/10 ukuran file .WAV atau .AIF. Format tersebut memberikan keuntungan streaming file audio semacam itu melalui internet untuk mendengarkan secara online yang sebelumnya tidak mungkin dilakukan karena ukuran file audio yang besar. Kualitas suara file audio MP3 dapat dikontrol dengan pengaturan parameter seperti bit rate, sample rate, joint atau normal stereo.

## Sejarah Singkat Format File MP3

Format MP3 ditemukan dan dikembangkan oleh Perusahaan Jerman, Fraunhofer-Gesellshart. Algoritme memiliki lisensi paten untuk teknologi kompresi yang digunakannya. Inilah garis waktu praktis MP3:

• **1987** - The Fraunhofer Institute di Jerman mulai meneliti pengkodean audio kecepatan bit rendah berkualitas tinggi. Itu disebut proyek EUREKA EU147, Digital Audio Broadcasting.

• **Januari 1988** - The Moving Picture Experts Group, atau MPEG, didirikan.

• **April 1989** - Fraunhofer menerima paten di Jerman untuk MP3.

• **1992** - Dieter Seitzer, yang membantu penelitian Fraunhofer, mengintegrasikan kode audionya dengan MPEG-1.

• **1993** - Standar MPEG-1 diterbitkan.

• **1994** - Standar MPEG-2 dikembangkan dan kemudian diterbitkan setahun kemudian.

• **Nov. 26 Agustus 1996** - Paten AS untuk MP3 dikeluarkan.

• **September 1998** - Fraunhofer mulai memberlakukan hak paten. Siapa pun yang menggunakan kode audio MP3 membayar biaya lisensi kepada Fraunhofer.

• **Februari 1999** - SubPop, sebuah perusahaan rekaman, mendistribusikan musik dalam format MP3, perusahaan pertama yang melakukannya.

• **1999** - Pemutar MP3 portabel pertama kali muncul.

## Berformat File MP3

File MP3 terdiri dari frame MP3 dimana setiap frame terdiri dari header dan blok data. Frame tidak independen dan biasanya tidak dapat diekstraksi pada batas frame yang sewenang-wenang. Blok data file berisi informasi tentang audio dalam hal frekuensi dan amplitudo. Kata sinkronisasi di header mengidentifikasi awal dari frame yang valid. Ini diikuti oleh 3 bit dimana bit pertama menunjukkan bahwa itu adalah standar MPEG dan sisa 2 bit menunjukkan bahwa lapisan 3 digunakan; karenanya MPEG-1 Audio Layer 3 atau MP3. Setelah itu, nilainya akan berbeda, tergantung pada file MP3.

[ISO](https://en.wikipedia.org/wiki/International_Organization_for_Standardization)/[IEC](https://en.wikipedia.org/wiki/International_Electrotechnical_Commission) 11172-3 menentukan rentang nilai untuk setiap bagian dari header beserta spesifikasi header. Sebagian besar file MP3 saat ini berisi [ID3](https://en.wikipedia.org/wiki/ID3) [metadata](https://en.wikipedia.org/wiki/Metadata), yang mendahului atau mengikuti frame MP3, seperti yang dicatat dalam diagram. Aliran data dapat berisi checksum opsional.
