## Apa itu file SDT?

File SDT digunakan sebagai sarana penyimpanan suara game, termasuk musik latar dan efek suara. Beberapa program, seperti Dragon UnPACKer dan MAStudio 2002, dapat membuka dan menyalin konten audio dari file SDT. File SDT hanya dibuka oleh perangkat lunak khusus. Ada 3 jenis file dan masing-masing dapat dibuka oleh perangkat lunak yang berbeda. Game lama yang dirilis oleh Electronic Arts (misalnya Dungeon Keeper 2 (DK2), Theme Park World, video game seri Formula One (F1)) file paket audio dengan kumpulan file audio terkompresi yang dapat disimpan menggunakan MPEG-2 format. Format dan ekstensi file serupa termasuk .sdsk,.sds, .sdr, .sdq, .sdv, .sdw, .sdx, dan .sdz.

File SDT dalam format file berpemilik dan detail format filenya tidak tersedia.
