## Apa itu file WV? ##

File WV adalah file audio yang dikompres menggunakan kompresi WavPack Hybrid. Kompresi lossless dan lossy didukung oleh WavPack. Format audio ini menawarkan fungsionalitas yang tak tertandingi. File WV berisi audio digital yang dikompresi oleh WavPack Audio.

Mode lossless sempurna untuk pengarsipan data audio dan kompresi data tanpa degradasi dan kehilangan sedikit pun adalah kemampuan Wavpack. Ini memiliki antarmuka perpustakaan sumber terbuka bersama dengan lisensi tidak terbatas. File-file ini sebagian besar terkait dengan kompresor Sonarc. File audio terkompresi WavPack ini memiliki 30 hingga 70 persen dari ukuran file aslinya.

WV mendukung kompresi audio hybrid. File terkompresi ini berukuran relatif kecil, kualitas lebih tinggi, dan dapat digunakan dalam format yang sama (yaitu tidak diperlukan konversi untuk membuka dan menggunakan ). File WV asli adalah format klip audio yang sangat terkompresi dan lossy. Penambahan data yang benar dari file WVC dapat digunakan untuk mengembalikan audio asli seluruhnya.

## Sejarah Singkat File WV

Pada pertengahan 1998 versi 1.0 dari WavPack dikembangkan oleh David Bryant dan tampil sebagai yang terbaik untuk efisiensi dan juga kecepatan. File WV berasal dari versi pertama. Pada tahun yang sama setelah versi pertama, versi 2.0 modifikasi lainnya dirilis. Pada tahun 1999 versi 3.0 dikembangkan dan dirilis. Itu berisi mode cepat yang diperbarui dan kemampuan untuk mengompres file audio mentah.

## Spesifikasi Format File WV

Kode WavPack portabel dan kompatibel dengan Unix, Mac OS, dan Windows. Plugin untuk berbagai aplikasi tersedia untuk file WV. Ini dapat digunakan pada banyak arsitektur seperti x86 dan Motorola 68k. Jenis kompresi ini kompatibel dengan sebagian besar format audio PCM.

Beberapa di antaranya adalah rekaman 8,16 dan 32-bit. Terkadang file WVC disertakan sebagai pendamping file WV. File audio ini kompatibel untuk digunakan di hampir semua pemutar media yang saat ini digunakan bersama dengan persyaratan codec eksternal WavPack.

Mode hibrid di WV memecahkan masalah harus memilih antara file kecil dengan kualitas rendah dan file terkompresi dengan kualitas tinggi. Daripada membuat satu file tunggal, mode hibrid membuat file berkualitas tinggi dengan ukuran kecil yang dapat dengan mudah digunakan sendiri dan kombinasi ini dengan file lossy, memberikan file "koreksi". Kombinasi ini memberikan peluang untuk melakukan pemulihan penuh tanpa kehilangan. Mode hybrid ini bekerja pada sekitar 196 kbps untuk audio CD atau 2,25 bit per sampel.
