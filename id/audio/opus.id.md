## Apa itu file OPUS?

Opus adalah format file audio yang dibuat oleh yayasan Xiph.Org dan kemudian distandarisasi oleh IETF (Internet Engineering Task Force). Ini telah dikembangkan terutama untuk streaming internet, Voice over IP (VOIP), konferensi video dan obrolan dalam game, oleh karena itu dirancang untuk mempertahankan latensi rendah sambil mempertahankan komunikasi interaktif waktu nyata. Menjaga kualitas suara file Opus, banyak tes mendengarkan buta telah menandai Opus sebagai format audio berkualitas tinggi daripada format audio lainnya pada bitrate tertentu.

## Spesifikasi Format File OPUS

Format Opus menggunakan codec SILK (digunakan oleh Skype) dan Xiph.Org dan mendukung laju bit variabel dari 6 kb/dtk hingga 510 kb/dtk. Ekstensi Opus menggunakan algoritme SILK berbasis LPC yang berorientasi pada ucapan dan algoritme CELT berbasis MDCT latensi rendah, terkadang beralih di antara keduanya atau terkadang dengan menggabungkan kedua algoritme untuk memberikan efisiensi maksimum. File dalam format Opus memiliki ekstensi .opus. Tidak seperti format file Vorbis, buku kode besar tidak diperlukan oleh Opus untuk setiap file, itulah alasan mengapa format ini relatif efisien.
