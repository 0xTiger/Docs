## Apa itu file MID?

File yang memiliki ekstensi file .MID atau .MIDI adalah file Antarmuka Alat Musik Digital (MID). Ini adalah perbedaan dari file audio biasa seperti [WAV](/id/audio/wav/) atau [MP3](/id/audio/mp3/) karena tidak membawa konten audio yang sebenarnya; karenanya, ukurannya cukup kecil. Konten tersebut mencakup not yang dimainkan, waktunya, durasinya, dan kenyaringan yang diinginkan untuk setiap not.

File MID pada dasarnya terdiri dari informasi yang menjelaskan bagaimana seharusnya suara terdengar saat dihubungkan dengan perangkat pemutaran atau dimuat ke perangkat lunak khusus yang menginterpretasikan data audio. Ini menjadikannya alat yang ideal untuk berbagi data musik antara aplikasi yang kompatibel dan melalui koneksi bandwidth rendah. Ukuran yang lebih kecil memungkinkan penyimpanan yang mudah bahkan floppy disk dapat digunakan.

### Sejarah Singkat Format File MID

Instrumen pertama, Roland Jupiter-6 dan Nabi 600, diperkenalkan dengan MIDI pada tahun 1982. Mesin drum MIDI pertama, Roland TR-909 keluar pada tahun 1983. Setelah itu, sequencer MIDI pertama, Roland MSQ-700 adalah dilepaskan. NEC PC-88 dan PC-98, dirilis pada tahun 1982, adalah komputer pertama yang mendukung MIDI.

## Spesifikasi Format File MID

* Berukuran kecil – seringkali sekecil 10 KB untuk pemutaran satu menit
* Dapat menyesuaikan not, tempo, dan pitch untuk performa yang diinginkan
* Ini terdiri dari tiga bagian – konektor fisik, format pesan, dan format penyimpanan
* Dapat dikonversi menjadi MP3, WAV, WMA, FLAC, OGG, [AAC](/id/audio/aac/), MPC di Windows melalui Total Audio Converter.
