## Apa itu file AC3?

File dengan ekstensi .ac3 adalah file Audio Codec 3, yang diperkenalkan oleh Dolby Laboratories. Ini adalah format audio yang dapat berisi hingga enam saluran keluaran audio. Format tersebut awalnya digunakan untuk audio tetapi sekarang juga digunakan untuk aplikasi lain seperti siaran HDTV, DVD, cakram Blue-ray, dan konsol game. Aplikasi yang bisa membuka file AC3 antara lain Apple QuickTime player, Microsoft Windows Media Player, Winamp, MPlayer dan sejenisnya.

## Format File AC3

File AC3 bersifat biner dan berdasarkan Modified Discrete Cosine Transform (MDCT) yang merupakan algoritma kompresi lossy. Dolby Laboratories menggunakan algoritma MDCT bersama dengan prinsip pengkodean perseptual untuk mengembangkan format audio AC-3. Hal ini menyebabkan dirilisnya format AC-3 sebagai standar Dolby Digital pada tahun 1991. Format tersebut mendukung penyediaan lima saluran untuk speaker rentang normal (20 Hz - 20.000 Hz) dan satu saluran (20 Hz - 120 Hz) untuk efek frekuensi rendah yang digerakkan oleh subwoofer. AC3 mendukung laju sampel hingga 48 kHz.

### Detail Teknis Format File AC3

Teknologi Dolby Digital menggunakan cara yang sangat efisien untuk mengompres ukuran file audio multisaluran tanpa merusak kualitas suaranya. Kompresi ini menghasilkan ukuran file yang lebih kecil yang membuatnya mudah untuk mendistribusikan file suara. Dengan menggunakan Dolby Digital, dimungkinkan untuk menyertakan campuran audio 5.1 saluran lengkap pada cetakan film atau DVD.
