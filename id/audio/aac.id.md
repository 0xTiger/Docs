## Apa itu file AAC?

AAC (Advanced Audio Coding) mengacu pada standar pengkodean audio digital yang mewakili file audio berdasarkan kompresi audio lossy. Diluncurkan sebagai penerus format file **[MP3](/id/audio/mp3/)** dengan mempertimbangkan bahwa masalah yang dihadapi lateral untuk implementasi ide baru dalam proses pengkodean berdasarkan pengembangan metode untuk kompresi data. AAC mencapai kualitas suara yang lebih baik dibandingkan dengan MP3 pada kecepatan bit yang sama. Itu didefinisikan dalam MPEG-2 Bagian 7 (ISO/IEC 13818-7), dan dalam bentuk yang diperbarui dalam MPEG-4 Bagian 3 (ISO/IEC 14496-3).

Format AAC diadopsi sebagai format media default oleh YouTube, iPhone, iPod, iPad, Apple iTunes dan beberapa platform lainnya. Beberapa aplikasi dan API tersedia untuk konversi format file AAC ke format lain seperti MP3, WMA, M4A, **[WAV](/id/audio/wav/)** dan lainnya.

### Sejarah Singkat File AAC

Format file AAC adalah versi MP3 yang disempurnakan dengan beberapa perbaikan. Disumbangkan oleh beberapa perusahaan termasuk Fraunhofer Institute (Fraunhofer IIS - pengembang MP3), Nokia, Dolby, AT&T dan Sony, format tersebut dinyatakan sebagai standar internasional oleh Moving Picture Experts Group (MPEG) pada April 1997. Kemudian pada tahun 1999, MPEG-2 Bagian 7 telah diperbarui dan termasuk dalam keluarga standar MPEG-4. Itu dikenal sebagai MPEG-4 Bagian 3, diidentifikasi sebagai ISO/IEC 14496-3: 1999 atau Audio MPEG-4.

## Spesifikasi Format File AAC

Spesifikasi format file AAC memungkinkan lebih banyak fleksibilitas untuk mendesain codec daripada MP3, menghasilkan strategi pengkodean yang lebih bersamaan dan kompresi yang efisien. Format tersebut telah menjadi pilihan pilihan oleh sejumlah platform perangkat keras untuk peningkatannya dibandingkan MP3 dalam hal memberikan dukungan untuk lebih banyak opsi bahkan pada bitrate yang lebih rendah. Spesifikasi format file AAC tersedia sebagai [MPEG-2 Bagian 7](https://www.iso.org/standard/43345.html) dan [MPEG-4 Bagian 3](https://www.iso.org /standard/53943.html) (tidak gratis untuk diunduh). Formatnya menggunakan jenis media berikut:

* suara/aac
* audio/aacp
* suara/3gpp
* audio/3gpp2
* suara/mp4
* audio/mp4a-latm
* audio/mpeg4-generik

## AAC vs MP3 - Peningkatan ##

Perbedaan utama antara AAC dan MP3 dalam hal peningkatan adalah sebagai berikut:

* AAC mendukung rentang saluran yang lebih luas (hingga 48) dan tingkat pengambilan sampel (dari 8 kHz hingga 96 kHz)
* AAC memiliki frekuensi pengkodean yang lebih baik di atas 16 kHz
* AAC memiliki batas variasi yang lebih luas dalam resolusi frekuensi-waktu dari bank filter yang mengarah pada peningkatan pengkodean transien dan bagian stasioner dari sinyal audio
* Bank filter yang lebih efisien dan sederhana: bank filter hibrida telah digantikan oleh MDCT standar (modified discrete cosine transform)
* Mendukung peningkatan efektivitas kompresi menggunakan Temporal Noise Shaping (TNS), koefisien prediksi waktu MDCT (prediksi jangka panjang), stereo parametrik, substitusi noise perseptual, replikasi pita spektral (SBR).
* stereo bersama yang lebih fleksibel (metode yang berbeda dapat digunakan dalam rentang frekuensi yang berbeda);
