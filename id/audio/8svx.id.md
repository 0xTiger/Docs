## Apa itu file 8SVX? ##

8SVX adalah format file standar audio yang dikembangkan oleh Electronic Arts. Itu dianggap sebagai subtipe data dari format file IFF. File ini digunakan untuk menyimpan data audio 8-bit dalam potongan. Dalam metadata potongan deskriptor terpisah terdapat tentang aliran data 8SVX. Ini adalah format file audio digital dan digunakan oleh PC Amiga. Sistem komputer Amiga sekarang jarang digunakan sehingga file 8SVX ini biasanya digunakan untuk menyimpan dan mendengarkan musik lama.

## Format File 8SVX

File 8SVX dapat menyimpan rekaman audio dalam kualitas 8-bit yang berasal dari spesifikasi [IFF](/id/audio/aiff/). Ini adalah file yang berisi rekaman audio seperti file berisi audio lainnya. Itu juga termasuk format wadah IFF. Dalam format ini, beberapa format data disematkan dalam satu file dan digunakan untuk merinci lebih lanjut aliran yang disertakan di dalamnya. Ini dapat digunakan sendiri dalam wadah file audio saja dari file IFF. Di sisi lain, multiplexing dengan subtipe IFF lainnya dapat menghasilkan aliran video animasi.

## Sejarah Singkat Format 8SVX

Awalnya dirilis pada tahun 1985 oleh Steve Hayes dan Jerry Morrison. Format arsip lainnya dikembangkan pada tahun 2009. Format file ini digunakan pada sistem komputer Amiga untuk musik lama. Ini juga dikenal sebagai file suara Amiga 8-bit. Perangkat lunak audio yang dibuat untuk PC Amiga mendukung format file ini. Itu digunakan oleh banyak program pengeditan suara pada awalnya. Sebagian besar aliran datanya dikodekan menggunakan aliran PCM. Versi Amiga yang dimodifikasi menyediakan kerangka dekoder 8SVX. Format RIFF dan WAVE yang dikembangkan bersama oleh Microsoft dan IBM dibuat di bawah pengaruh format wadah 8SVX/IFF.

## Spesifikasi Format File 8SVX

Banyak program yang dikaitkan dengan file-file ini di Windows dan Mac OS. Banyak program dapat digunakan untuk menangani file-file ini. Ini adalah format audio yang disandikan untuk Commodore Amiga. Fitur-fitur seperti attack, section repeat, dan release didukung oleh file-file ini. Fitur ini digunakan untuk menyimpan sampel alat musik. Kompresi data lossy Fibonacci-delta tersedia yang menghasilkan rasio kompresi 50% dan menurunkan fidelitas.
