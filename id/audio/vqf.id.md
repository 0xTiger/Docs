## Apa itu file VQF?
Ekstensi file .vqf digunakan untuk file TwinVQ. TwinVQ adalah format audio yang dikembangkan oleh Nippon Telegraph and Telephone Corporation dan dipasarkan oleh Yamaha dengan menggunakan nama SoundVQ. Diumumkan oleh pengembang bahwa file TwinVQ berukuran sekitar 30 hingga 35% lebih kecil dari file MP3 dengan kualitas tertentu. Kualitas yang lebih tinggi dicapai dengan biaya penggunaan prosesor yang lebih tinggi.

## format file VQF

File dengan ekstensi .vqf sebenarnya termasuk dalam format audio TwinVQ yang juga dikenal sebagai teknik kompresi audio. Teknik ini telah digunakan dalam desain berpemilik dan standar. Kita dapat mengatakan bahwa TwinVQ adalah codec audio yang ditingkatkan untuk pengkodean audio pada bitrate ultra rendah sekitar 8 kbit/s. TwinVQ menggunakan kuantisasi vektor kembar. Codec ini mendukung pengkodean laju bit konstan pada 80, 96, 112, 128, 160, dan 192 kbit/dtk.

Yamaha memasarkan codec audio kompresi ini sebagai alternatif untuk MP3, tetapi formatnya tidak pernah menjadi sangat terkenal, karena sifat kepemilikan format TwinVQ, pengkodeannya sangat lambat dan tidak banyak musik yang tersedia dalam format TwinVQ.
