## Apa itu file RAW?

Format file mentah mengacu pada format file yang menyimpan data yang belum diproses dan dikompresi langsung dari sensor gambar digital atau sumber lain. Ini biasanya digunakan dalam fotografi dan pencitraan digital.

Format lain menerapkan algoritma kompresi dan pemrosesan untuk mengurangi ukuran file dan mengoptimalkan gambar untuk ditampilkan atau dicetak, file mentah berisi semua data yang ditangkap oleh sensor tanpa perubahan apa pun. Ini mencakup informasi seperti warna, kecerahan, dan data spesifik sensor lainnya.

File mentah sering kali dikhususkan untuk produsen dan model kamera tertentu karena file tersebut menyimpan data dalam format yang unik sesuai karakteristik sensor dan perangkat keras kamera tersebut. Format file mentah yang umum mencakup CR2 (Canon), NEF (Nikon), ARW (Sony) dan DNG (Adobe Digital Negative).

Karena file mentah belum diproses, file tersebut memberikan fleksibilitas lebih besar untuk penyesuaian pasca-pemrosesan dan menawarkan kontrol lebih besar terhadap gambar akhir. Fotografer dan profesional lebih suka bekerja dengan file mentah karena mereka dapat melakukan penyesuaian yang tepat pada aspek seperti eksposur, white balance, gradasi warna, dan pengurangan noise selama proses pengeditan.

Namun, file mentah biasanya berukuran lebih besar dibandingkan dengan format terkompresi seperti JPEG. Selain itu, file mentah memerlukan perangkat lunak khusus atau editor gambar khusus yang mendukung format mentah tertentu untuk membuka, melihat, dan memprosesnya.

## Bagaimana cara membuka file RAW?

Anda dapat membuka file RAW menggunakan berbagai aplikasi perangkat lunak pengedit gambar, misalnya

- Adobe Photoshop
- Elemen Adobe Photoshop
- Corel PaintShop Pro
- Sistem ACD ACDSee
- Terapi Mentah

## Bagaimana cara mengonversi file RAW?

Beberapa perangkat lunak pengedit gambar dapat mengonversi file RAW ke format berbeda. Adobe Photoshop dapat mengkonversi file RAW Anda ke dalam format gambar seperti

- [PNG](/id/image/png/) - Grafik Jaringan Portabel
- [JPEG](/id/gambar/jpeg/) - Gambar JPEG
- [GIF](/id/image/gif/) - Format Pertukaran Grafis
- [PSD](/id/image/psd/) - Dokumen Adobe Photoshop
- [BMP](/id/image/bmp/) - Gambar Bitmap

## Apa format file RAWnya?

Format file RAW adalah format biner.
