## Apa itu file THM?

File THM adalah gambar mini yang dibuat secara otomatis dari bingkai pertama file video pada camcorder digital, kamera, dan telepon pintar. Itu dihasilkan ketika file video disimpan oleh kamera perangkat. Gambar pratinjau thumbnail dari semua video ditampilkan kepada pengguna untuk memilih video yang menarik dari koleksi video. File THM disimpan dalam format biner terkompresi di memori untuk mengurangi pemanfaatan ruang. File THM dapat dibuka dengan aplikasi seperti Apple Preview dan [Adobe Photoshop 2022](https://www.adobe.com/products/photoshop.html).

## Format File THM - Informasi Lebih Lanjut

File THM adalah gambar raster yang disimpan ke disk dalam format file biner. Tujuan utama file THM adalah untuk mengidentifikasi file video dengan gambar kecilnya. Dalam kebanyakan kasus, file THM memiliki nama yang sama dengan nama file video yang sesuai. Misalnya, file video bernama MOV001.mov akan memiliki MOV001.THM sebagai nama file thumbnail-nya. Ekstensi file THM dapat diubah namanya menjadi [JPEG](/id/image/jpeg/) untuk membukanya jika tidak langsung dibuka dengan mengklik dua kali.

## Apakah saya boleh menghapus file THM?

File THM adalah gambar mini untuk video yang direkam oleh camcorder. Menghapus ini tidak akan berpengaruh apa pun karena video asli tetap dipertahankan.
