## Apa itu file SKP?

File SKP adalah file model tiga dimensi berpemilik yang dibuat dengan program desain 3D SketchUp. Itu dapat terdiri dari beberapa elemen atau komponen 3D yang berbeda seperti bayangan, gambar rangka, dan tekstur. File SKP dapat berisi seluruh arsitektur atau dapat terdiri dari desain yang lebih kecil (seperti pintu atau kursi) dari desain yang lebih besar. Ini memungkinkan model yang ditentukan dalam file tersebut dapat digunakan kembali dengan memasukkan ke dalam dokumen SketchUp lainnya.

File SKP dapat dikenali dari ekstensinya yaitu .skp yang dapat dengan mudah dicari dan ditemukan di sistem dengan menggunakan fitur pencarian Sistem Operasi.

## Format File SKP - Informasi Lebih Lanjut

File SKP disimpan ke disk dalam format file biner milik SketchUp. Format file SKP dapat menyimpan gambar 2D selain gambar 3D. Dokumen 2D ini dapat menunjukkan denah, detail bangunan, dan ketinggian juga.
