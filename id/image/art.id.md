## Apa itu file ART?

File ART adalah file gambar bitmap terkompresi yang dibuat dan digunakan oleh layanan America Online (AOL) dan perangkat lunak klien. Itu dibangun untuk mengurangi ukuran gambar untuk pengunduhan cepat melalui internet. Algoritme kompresi yang diterapkan pada gambar untuk menghasilkan file .art mirip dengan format [JPEG](/id/image/jpeg/), tetapi kadang-kadang mengurangi kualitas gambar sekaligus mendukung pengurangan ukuran file. File ART dapat dibuka menggunakan AOL File Viewer.

## Format File ART

Format file ART menggunakan kompresi untuk memudahkan transfer di internet dan berbagi di antara pengguna AOL. Ini telah digunakan untuk presentasi gambar di layanan online menggunakan browser web klien. Ukuran file ART yang diperkecil memfasilitasi pengunduhan yang lebih cepat bahkan pada koneksi internet yang lebih lambat di masa-masa awal.
