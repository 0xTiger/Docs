## Apa itu file JPF? ##

File dengan ekstensi .jpf adalah ekstensi untuk sistem pengkodean gambar JPEG 2000 ISO/IEC 15444 dan disebut sebagai Bagian 2 ISO/IEC 15444-2. Ini mendefinisikan dan menentukan satu set metode kompresi lossless (bit-preserving) dan lossy untuk pengkodean nada kontinu, bi-level, skala abu-abu, gambar diam digital berwarna, atau gambar multi-komponen. Bagian pertama dari ISO/IEC 15444-1 dirujuk ke [JP2](/id/image/jp2/) yang menggunakan teknologi wavelet untuk mengkode konten lossless dan merupakan dasar untuk format file gambar JPEG 2000. Format file JPF tidak menerima sambutan hangat karena penggunaan ekstensif format JPEG. File JPF dapat dibuka dengan aplikasi pencitraan populer seperti Adobe Photoshop 2020, Adobe Illustrator 2020, dan CorelDraw Graphics Suite 2020.

## Sejarah Singkat

Pada tahun 2000, komite Joint Photographic Experts Group merancang JP2 dengan tujuan untuk meningkatkan standar JPEG berbasis transformasi kosinus diskrit mereka sendiri dengan metode berbasis gelombang kecil ini. Komite JPEG bertujuan untuk menyediakan metode dasar mereka tanpa biaya lisensi. Dalam persaingan lisensi JP2 di antara 20 perusahaan, mereka menang tipis. JPEG 2000 telah dinyatakan sebagai standar ISO, meskipun sebagian besar browser web belum siap untuk membantu JPEG 2000 sejak 2017. Pada tahun 2004, format ISO/IEC 15444-2 diterima secara publik sebagai ekstensi ke format file JP2.

## Format File JPF

Format file JPF mendefinisikan proses decoding yang diperluas untuk konversi data gambar terkompresi untuk rekonstruksi. Ini adalah format file yang diperluas yang menentukan sintaks codestream yang diperluas yang berisi informasi untuk menginterpretasikan data gambar terkompresi. Standar yang diperluas ini menentukan wadah untuk menyimpan metadata gambar dan memberikan panduan tentang proses pengkodean yang diperluas untuk mengonversi data gambar sumber menjadi data gambar terkompresi.

### Organisasi Berkas

JPF adalah format file penyimpanan formal ketika file JPX disimpan dalam sistem file komputer. Selain itu, Rekomendasi/Standar Internasional lainnya dapat menentukan kotak lain untuk digunakan dalam file JPX. Namun, semua informasi yang terkandung dalam file JPX harus dalam format kotak; byte-stream tidak dalam format kotak tidak akan ditemukan dalam file. Struktur biner kotak dalam file JPX identik dengan yang ditentukan dalam format file [JP2](/id/image/jp2/).
