## Apa itu file PICT?

File PICT adalah gambar grafik yang disimpan dalam format file gambar Apple Macintosh. Itu dapat menyimpan data gambar sebagai format file raster dan vektor. Itu adalah format file gambar default untuk program Apple QuickDraw. File PICT memungkinkan pertukaran raster serta data vektor antara aplikasi Mac. Formatnya telah lama digantikan oleh format file [PDF](/id/pdf/) yang lebih fleksibel. File PICT terkadang ditemukan dengan ekstensi **[.PCT](/id/image/pct/)**.

## Format File PICT - Informasi Lebih Lanjut

File PICT disimpan sebagai file gambar biner yang dapat dibuka dengan aplikasi seperti XnView, GIMP, Corel PaintShop Pro, dan Apple Preview. Ini mendukung gambar saluran tunggal (dengan kemampuan penyimpanan 8 warna) atau gambar RGB saluran warna yang dapat menyimpan ribuan warna. Ini dikenal sebagai format file **PICT 1** dan **PICT 2**.
