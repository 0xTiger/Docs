## Apa itu file ECW? ##
File ECW terdiri dari gambar yang dirancang untuk digunakan dengan data geospasial. Gambar yang memuat sebenarnya adalah gambar terkompresi yang dibuat dalam format Enhanced Compression Wavelet. File ECW umumnya digunakan untuk menyimpan proyeksi peta satelit dan gambar udara. Untuk mendukung aplikasi geospasial, informasi proyeksi peta dapat disematkan ke dalam file ECW.

## format file ECW
ECW adalah format gambar berpemilik berdasarkan kompresi wavelet; dioptimalkan untuk citra satelit dan foto udara. Format ini sekarang dimiliki oleh Intergraph bagian dari Hexagon AB tetapi pada dasarnya dikembangkan oleh Earth Resource Mapping. ECW adalah format kompresi lossy yang memampatkan gambar berukuran besar dengan kontras bolak-balik halus sambil mempertahankan kualitas visualnya.
 

### Properti
Berikut adalah beberapa sifat khas format ECW:
- Informasi proyeksi peta dapat ditambahkan ke dalam format file ECW untuk mendukung aplikasi geospasial.
- Data gambar hingga 65.535 pita termasuk lapisan atau warna dapat dikompresi ke dalam format file ECW dengan kecepatan lebih dari 25 MB per detik.
- Protokol ECWP adalah protokol streaming efisien yang digunakan untuk mengirimkan gambar ECW dan JPEG2000 melalui jaringan.
- SDK cepat yang mendukung ECW dan JPEG2000 tersedia gratis untuk implementasi desktop untuk Linux, Windows, dan MacOSX
