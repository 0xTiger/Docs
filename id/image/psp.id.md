## Apa itu file PSP?

File PSP adalah gambar bitmap berwarna yang dibuat dengan PaintShop Pro. Ini adalah format file asli PaintShop Pro dan mendukung file terkompresi yang tidak terkompresi, terkompresi RLE, terkompresi LZ77, dan terkompresi [JPEG](/id/image/jpeg/) (termasuk Lossless JPEG). Ini juga mendukung lapisan yang mirip dengan file [PSD](/id/image/psd/) dan dapat memiliki informasi lain seperti panduan, metadata, dan objek yang relevan di dalam file gambar. Setiap lapisan dapat diedit dan diintegrasikan dengan gambar utama sendiri. File PSP dapat dibuka dengan perangkat lunak PaintShop Pro.

## Format File PSP

File PSP disimpan ke disk sebagai file biner dengan dukungan algoritme kompresi yang berbeda. Informasi bit per piksel dapat disimpan dalam file PSP dengan opsi berikut.

* 1 (abu-abu)
* 4 (abu-abu)
* 8 (abu-abu atau warna)
* 24 (warna)

File PSP dapat disimpan di sistem operasi x32 dan x64. [PSPIMAGE](/id/image/pspimage/) adalah format file lain yang digunakan oleh PaintShop Pro.
