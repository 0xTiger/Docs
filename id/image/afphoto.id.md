## Apa itu file AFPHOTO?

File AFPHOTO adalah file gambar yang dibuat oleh perangkat lunak pembuatan gambar, Serif Affinity Photo. Berisi keseluruhan isi gambar berupa beberapa lapisan gambar, bentuk, teks, goresan cat, contoh, efek pewarnaan dan pencahayaan. Serif Affinity Photo mirip dengan Adobe Photoshop, melainkan merupakan alternatif dari Adobe Photoshop dengan fitur dan kemampuan yang hampir sama. Adobe Photoshop menyimpan file sebagai [.psd file](/image/psd/) sedangkan file [Affinity Photo](https://affinity.serif.com/en-gb/photo/) disimpan dengan ekstensi .afphoto.

Serif Affinity Photo dapat membuka file AFPHOTO.

### Format Berkas AFPHOTO

File AFPHOTO disimpan dalam format file biner milik Serif Affinity Photo. Struktur format file internal dan spesifikasi rincinya tidak tersedia untuk referensi publik.

## Bagaimana cara mengonversi file AFPHOTO?

File AFPHOTO dapat dikonversi ke sejumlah format berbeda menggunakan perangkat lunak Serif Affinity Photo. Format file ekspor berikut didukung.

 * [PSD](/image/psd/) - Dokumen Adobe Photoshop
 * [PDF](/pdf/) - Format Dokumen Portabel
 * [JPG](/image/jpeg/) - Gambar JPEG
 * [PNG](/image/png/) - Grafik Jaringan Portabel
 * [GIF](/image/gif/) - Format Pertukaran Grafis
 * [EPS](/page-description-language/eps/) - Encapsulated PostScript
 * [SVG](/page-description-language/svg/) - Scalable Vector Graphics
 * [TIFF](/image/tiff/) - Format File Gambar yang Ditandai
 * TGA - Grafik Targa
 * EXR - Gambar OpenEXR
 * [HDR](/image/hdr/) - Gambar Rentang Dinamis Tinggi
