## Apa itu file CT?

File CT (Continuous Tone) adalah file gambar raster yang digunakan oleh peralatan pemrosesan grafis Scitex. Beberapa suite grafis seperti Adobe Photoshop, Adobe InDesign, dan QuarkXPress mendukung penggunaannya untuk rendering. Gambar data disimpan di dalam file CT dalam format gambar CMYK (Cyan, Magenta, Yellow Key), RGB, dan skala abu-abu, tetapi tidak mendukung saluran alfa. File CT dapat dibuka dengan aplikasi seperti Adobe Photoshop 2022, Adobe InDesign 2022, dan XnViewMP.

## Format File CT

File CT Scitex terdiri dari blok kontrol, blok parameter, dan data gambar sesuai [artikel CT Wikipedia](https://en.wikipedia.org/wiki/Scitex_CT) ini. Spesifikasi format file format file CT tidak tersedia tetapi beberapa penelitian telah menunjukkan bahwa gambar CT mewakili 4 warna, dengan setiap piksel berwarna berukuran hingga 128 bit. File CT biasanya berukuran besar dan memungkinkan gambar berkualitas tinggi.
