## Apa itu file CMX?

File dengan ekstensi .cmx adalah format file gambar Corel Exchange (juga dikenal sebagai Corel Metafile Exchange) yang digunakan sebagai presentasi oleh aplikasi CorelSuite. Ini berisi data gambar sebagai grafik vektor serta metadata yang menggambarkan gambar tersebut. File CMX dapat dibuka dengan CorelDraw, Corel Presentations, Paint Shop Pro dan beberapa versi Adobe Illustrator. File CMX dapat dikonversi ke format lain seperti [JPG](/id/image/jpeg/) dan [EPS](/id/page-description-language/eps/).

## Format File CMX
Format file CMX didasarkan pada Resource Interchange File Format (RIFF) dan mendukung format biner Intel dan Motorola. File CMX menggunakan tipe MIME berikut:
* gambar/x-cmx
* aplikasi / x-cmx
*aplikasi/cmx
* gambar/cmx

### Judul Format File
File CMX dimulai dengan "RIFF" atau "RIFX", dan memiliki "CMX1" pada offset 8.
