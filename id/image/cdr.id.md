## Apa itu file CDR?

File CDR adalah file gambar vektor yang dibuat dengan [CorelDRAW](https://www.coreldraw.com/en/pages/open-cdr-file/) untuk menyimpan gambar digital yang disandikan dan dikompresi. File gambar seperti itu berisi teks, garis, bentuk, gambar, warna, dan efek untuk representasi vektor dari konten gambar. File CDR dapat dibuka dengan CorelDRAW sebagai aplikasi utama dan juga dapat dikonversi ke format lain seperti [PDF](/id/pdf/), [JPG](/id/image/jpeg/), [PNG](/id/image/png/ ), [BMP](/id/image/bmp/) dan AI. Ini dapat digunakan untuk representasi berbagai data grafik seperti brosur, tabloid, amplop, dan kartu pos. Selain CorelDRAW, produk Corel lainnya seperti Corel Paintshop Pro dan CorelDRAW Graphics suite juga dapat membuka format file CDR.

## Format File CDR

CDR, sebagai format milik Corel, tidak memiliki spesifikasi teknis yang tersedia untuk umum. Format ini terus berkembang setelah diluncurkan pertama kali pada tahun 1989 dan mendukung fitur dasar seperti teks, manajemen warna, dan warna titik.
