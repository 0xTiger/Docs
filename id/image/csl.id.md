## Apa itu file CSL?

File Corel Symbol Library (CSL) adalah pustaka simbol yang digunakan dalam perangkat lunak CorelDRAW. Ini menyimpan simbol yang dapat digunakan di file lain. File CSL dapat disimpan secara lokal atau di jaringan agar dapat diakses untuk penerapan dan pengelolaan yang mudah. Ini disimpan dengan file CorelDRAW bersama dengan semua contoh simbol yang digunakan dalam dokumen.

## Format File CSL

File CSL disimpan dalam format file milik CorelDRAW. Menjadi kumpulan simbol, simbol di dalam file CSL dapat diimpor dan ditambahkan ke gambar dalam sebuah proyek.
