## Apa itu file DCX?

File DCX adalah file gambar yang terdiri dari beberapa halaman untuk menghasilkan file faks digital. Setiap file DCX dimulai dengan header 4-byte yang mengidentifikasi file DCX sebagai tanda tangan, diikuti dengan gambar individual, disimpan sebagai file [PCX](/id/image/pcx/) tersemat. Tanda tangan 4-byte mengidentifikasi file DCX dan diikuti dengan daftar offset file 4-byte hingga 1023. Format file DCX dikembangkan oleh Zsoft. Beberapa aplikasi dapat membuka file DCX seperti Adobe Photoshop, Corel PaintShop Pro, dan IrfanView.
