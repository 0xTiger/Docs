## Apa itu file PAT?

File ".PAT" biasanya dikaitkan dengan **CorelDRAW**, perangkat lunak pengedit grafik vektor yang populer. Di CorelDRAW, file ".PAT" adalah "File Pola" yang berisi informasi tentang pola atau tekstur yang dapat diterapkan ke objek di desain. Pola dapat digunakan untuk mengisi objek dengan berbagai desain berulang, seperti garis, titik, atau tekstur khusus. Pola ini disimpan dalam file ".PAT" agar mudah digunakan kembali.

## Bagaimana cara membuat dan menggunakan file PAT?

Berikut ini cara membuat dan menggunakan file pola di CorelDRAW:

1. Buat Pola:
    



- Buka CorelDRAW.
- Pola desain yang ingin Anda simpan sebagai file ".pat".
- Pilih objek atau area dengan pola.
- Buka menu "Objek" dan pilih "Isi" untuk membuka alat Isi.
- Di alat Isi, pilih "Isi pola".
- Sesuaikan properti pola, seperti ukuran, sudut, dan jarak.
2. Simpan Pola sebagai File ".pat":
    



- Setelah membuat pola, Anda dapat menyimpannya sebagai file ".pat" untuk digunakan di masa mendatang.
- Buka menu "File" dan pilih "Ekspor" atau "Simpan Sebagai."
- Pilih format file "File Pola CorelDRAW (*.pat)."
- Tentukan nama file dan pilih lokasi untuk menyimpannya.
3. Terapkan Pola:
    



- Untuk menggunakan pola tersimpan di proyek baru atau pada objek berbeda, Anda dapat mengimpornya.
- Buka proyek baru di CorelDRAW.
- Buka menu "Objek", pilih "Isi", dan pilih "Isi pola".
- Klik tombol "Impor" dan telusuri file ".pat" yang Anda simpan.
- Pilih pola dari pola yang diimpor dan terapkan ke objek Anda.
