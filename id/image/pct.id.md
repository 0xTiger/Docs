## Apa itu file PCT?

File PCT adalah format file Macintosh Picture Image yang merupakan format file default untuk program Apple QuickDraw. Ini dikembangkan oleh Apple tetapi setelah penghentian program QuickDraw, itu tidak lagi digunakan. **[PDF](/id/pdf/)**, pada umumnya, telah menggantikan PCT dan format file serupa. File PCT terkadang juga dapat ditemukan dengan ekstensi file .pict, meskipun PCT dan PICT memiliki format file yang sama.

## Format File PCT - Informasi Lebih Lanjut

File PCT disimpan ke disk sebagai file biner dalam **format file pict** dan detail format file internalnya tidak tersedia. Aplikasi yang bisa membuka file PCT antara lain XnView, GIMP, Corel PaintShop Pro, dan Apple Preview.

### Jenis Format File PCT

Data gambar dalam file PCT Macintosh dapat disimpan dalam format `PICT 1` asli atau sebagai format `PICT 2`.

**PICT 1** - Jenis format file PICT ini dapat menyimpan delapan warna
**PICT 2** - Jenis format file PICT ini mendukung ribuan warna dengan gambar 24 dan 32-bit
