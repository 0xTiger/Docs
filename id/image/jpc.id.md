## Apa itu file JPC?

File JPC adalah format file gambar yang dikodekan menggunakan aliran kode [JPEG2000](/id/image/jp2/). Ini juga terkadang disebut sebagai file Gambar Jepang. File JPC menggunakan kompresi wavelet canggih JPEG 2000 yang juga digunakan oleh file yang disandikan JPEG2000. Ini mendukung kedalaman warna 8, 24, dan 32 bit per piksel dalam skala abu-abu, RGB, YCbCr, XYZ, dan ruang warna Lab.

File JPC dapat dibuka dengan Adobe Photoshop 2022, Adobe Photoshop Elements, dan Adobe Illustrator 2021.
