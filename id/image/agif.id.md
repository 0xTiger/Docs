## Apa itu file AGIF?

File AGIF adalah file GIF yang dibuat dengan Active GIF Creator. Ini adalah file proyek yang dapat dibuka dan dilihat di proyek Active GIF Creator. Saat dibuka di Active GIF Creator, ini menunjukkan urutan gambar yang akhirnya menggantikan file GIF akhir dengan ekstensi .gif.

## Format File AGIF

AGIF adalah file proyek yang disimpan sebagai format file gambar raster. File [GIF](/image/gif/) yang telah diselesaikan dapat dibuka dan ditampilkan sebagai animasi. Ada API Python yang tersedia online yang dapat digunakan untuk membuat file GIF animasi.
