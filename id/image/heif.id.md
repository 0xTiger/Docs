## Apa itu file HEIF?

File HEIF adalah format file Gambar Kontainer Efisiensi Tinggi yang mampu menyimpan satu gambar atau urutan gambar dalam satu file. Dengan demikian, pada dasarnya adalah sebuah wadah yang menyimpan banyak gambar. Format file HEIF memampatkan gambar menggunakan standar High Efficiency Video Coding (HEVC). Ini lebih efisien dan ringan dibandingkan format file gambar lainnya seperti [JPEG](/id/image/jpeg/). Satu file HEIF dapat menyimpan urutan gambar seperti animasi [GIF](/id/image/gif/) atau urutan HDR, beserta metadata yang menjelaskan setiap gambar. mengadopsi format file HEIC variannya untuk iOS 11 mereka. Hal ini menyebabkan perusahaan lain mendukungnya dalam perangkat lunak mereka termasuk Adobe.

## Format File HEIF

Format HEIF dikembangkan oleh Moving Picture Experts Group (MPEG) dan terdaftar di bawah standar ISO/IEC 23008-12 pada tahun 2015. Format ini diadopsi oleh sejumlah produsen seperti Apple, Microsoft, dan Canon. File HEIF disimpan dengan ekstensi .heif (jika berisi satu gambar) atau sebagai .heifs (jika berisi urutan gambar).

### Penyimpanan Data HEIF

Format HEIF dapat menyimpan jenis gambar dan data berikut.

* Gambar individual dan thumbnail gambar
* Urutan gambar
* Instruksi pasca-pemrosesan
* Metadata gambar, seperti data [EXIF](/id/image/exif/).

Format gambar HEIF menyediakan kemampuan beberapa format gambar yang berbeda termasuk [JPEG](/id/image/jpeg/), [PNG](/id/image/png/), dan [GIF](/id/image/gif/) tanpa mengorbankan gambar kualitas.
