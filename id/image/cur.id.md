## Apa itu file CUR? ##

File CUR adalah format file kursor Microsoft Windows statis. Pada dasarnya, ini adalah gambar stasioner yang identik dengan file ICO (ikon) dalam segala hal kecuali ekstensi. Baik CUR dan [ICO](/id/image/ico/) dibuat berdasarkan spesifikasi Device-Independent Bitmap [DIB](/id/image/dib/) (Device-Independent Bitmap). Default, serta kursor khusus seperti penunjuk mouse Windows untuk sistem operasi Windows, disimpan dalam file CUR. Itu bisa berupa panah untuk penggunaan normal, jam pasir yang berputar untuk mendemonstrasikan waktu tunggu, atau I-bar untuk mengedit teks. File CUR disertakan dengan paket instalasi tema desktop khusus untuk memastikan bahwa kursor Windows cocok dengan tema desktop khusus.

## Spesifikasi teknis ##

File CUR adalah file sistem biner yang dapat ditemukan pada perangkat yang berfungsi pada sistem operasi Microsoft Windows. Gambar penunjuk CUR tersedia dalam berbagai ukuran standar seperti 16×16, 32×32, dll. File CUR sangat mirip dengan file ICO; keduanya terdiri dari gambar stasioner kecil. Hanya ekstensi file CUR dan ICO yang berbeda. Selanjutnya, penjelasan hotspot di header file CUR berbeda dengan file ICO. Hotspot mengartikan offset piksel dari sudut kiri atas ke tempat Anda mengarahkan mouse. Sistem Windows masih menggunakan file CUR, tetapi sekarang kursor animasi biasanya memiliki ekstensi file ANI, bukan CUR.

## Sejarah Singkat ##

File CUR dibuat dari file ICO. Format file CUR pertama dimasukkan ke dalam sistem operasi Microsoft Windows 1.0 pada tahun 1981 untuk memperlancar penggunaan komersial. Segera lebih banyak kursor interaktif diperkenalkan yang memungkinkan pengguna untuk mengubah preferensi kursor mereka dari opsi pra-instal yang tersedia. Namun, mudah untuk mengedit file CUR sendiri meskipun Anda tidak memiliki pengetahuan teknis yang memadai.
