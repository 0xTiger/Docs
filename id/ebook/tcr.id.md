## Apa itu file TCR?

Di awal tahun 90-an, format file TCR diperkenalkan untuk perangkat palmtop **Psion Series 3** kuno. Perusahaan menggunakan kompresi khusus platformnya dan format ini memiliki dukungan yang sangat terbatas untuk pemformatan teks. Ini adalah format file berbasis ZVR dan memberikan tingkat kompresi yang relatif lebih baik daripada PalmDOC. Ini juga bekerja sangat baik dengan FontMachine. Meskipun format file TCR milik perangkat yang dihentikan, format file ini tetap dapat dibaca dengan menggunakan beberapa eReader. Format file ini juga dapat dilihat di desktop dengan Sumatra PDF dan dengan aplikasi Calibre di Windows dan Linux.

## Detail Teknis

Karena format file TCR didasarkan pada kompresor ZVR yang mampu mengurangi banyak file sekitar 50% dari ukuran aslinya. Skema kompresi yang digunakan oleh over agak tua. File terkompresi dimulai dengan kamus dari 256 kemungkinan simbol yang mungkin ditampilkan dalam file terkompresi. Kamus berisi string alternatif tetap untuk masing-masing simbol ini. Dekompresi menjadi sangat cepat bahkan di OPL, dan dapat dimulai kapan saja di file terkompresi.

## Masalah untuk membuka file TCR ##

Jika Anda tidak dapat membuka dan menjalankan file TCR; itu tidak berarti bahwa Anda tidak menginstal perangkat lunak yang sesuai di perangkat Anda. Mungkin ada beberapa masalah lain yang mencegah file berfungsi dengan benar. Masalah yang mungkin terjadi adalah salah satu dari berikut ini:

- Korupsi file TCR.
- Tautan yang salah ke file TCR di entri registri.
- Deskripsi TCR yang dihapus dari registri Windows
- Instalasi aplikasi yang rusak yang mendukung format TCR
- File TCR yang terinfeksi dengan malware yang tidak diinginkan.
- Komputer tidak memiliki sumber daya perangkat keras yang memadai untuk mengoperasikan file TCR.
- Driver yang digunakan oleh komputer untuk membuka file TCR sudah usang.
