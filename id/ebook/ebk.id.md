## Apa itu file EBK?
Ekstensi EBK digunakan oleh perangkat lunak **eBook Pro Viewer** saat menyimpan file eBook. eBook Pro adalah perangkat lunak yang dapat membuat eBook dan melindungi informasi berhak cipta. File EBK berisi versi e-book terenkripsi dan terkompresi yang dapat dibaca menggunakan **eBook Pro Viewer**; setelah memberikan kredensial yang sesuai untuk otentikasi.
