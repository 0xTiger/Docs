## Apa itu file TR3? ##

File TR3 adalah eBuku TomeRaider 3 yang menyediakan dukungan untuk pencarian dan kompresi cepat. Itu dapat berfungsi dengan baik melalui program terkait TomeRaider. Yadabyte adalah pengembang perangkat lunak ini, perangkat lunak Inggris, dan perusahaan pengembangan web yang berbasis di Inggris. Aplikasi pembaca eBook TomeRaider dapat digunakan untuk memfungsikan dan mengelola konten file TR3 ini dengan benar. Pembaca e-book terkait ini dapat diinstal di komputer yang menjalankan sistem berbasis Microsoft Windows dan banyak gadget yang dapat ditransfer. File TR3 milik grup File eBook yang digunakan dalam sistem operasional seperti Windows 10, Windows 7, Windows 8 / 8.1, Windows Vista, Windows XP.

**Tag HTML** khusus TR3 adalah sebagai berikut:

|Tag|Deskripsi|
---|---|
|\<new> |Untuk membuat file TomeRaider dari file teks, hanya tag Baru yang diperlukan.<new> tag mendefinisikan awal halaman.|
|\<CATDEF> ... \</CATDEF> |mendefinisikan nama kategori|
|\<META> ... \</META> |adalah tag yang digunakan untuk menentukan semua metadata yang digunakan dalam format file. Tag ini menyertakan sejumlah parameter.|
|\<EXPMSG> |Tag ini mengontrol pesan yang muncul saat file telah kedaluwarsa. Setiap kali pengguna mencoba melihat halaman setelah file kedaluwarsa, pesan kedaluwarsa muncul alih-alih teks halaman yang sebenarnya.|
|\<DIEMSG> |Tag ini mirip dengan EXPMSG. Digunakan untuk menampilkan pesan ketika file mati.|
|\<ENGMSG> |Tag ini digunakan untuk membuat pesan yang ditampilkan untuk halaman terenkripsi. Pesan muncul alih-alih teks halaman jika pengguna mencoba membuka halaman terenkripsi sebelum membukanya.|
|\<expmsg> ,\<diemsg> dan \<encmsg> |Diabaikan dalam teks halaman. Mereka harus ditempatkan di bagian header file sebelum yang pertama<new> tag.|
|\<SELECTION> . \</ SELECTION> |Digunakan untuk mendukung Query. Tag ini harus sebelum yang pertama<new> menandai. Ini mengkompilasi data pemilihan untuk kueri pengguna.|
|\<catset> . \</catset> | Digunakan untuk menetapkan nama kategori untuk setiap mata pelajaran.|


## Masalah untuk membuka file TR3 ##

Berikut adalah daftar beberapa masalah umum yang mungkin timbul dan menyebabkan kesalahan fungsi format file:

* File rusak
* File Terinfeksi karena virus
* Tautan yang salah ke file TR3 dalam akses arsip
* Tidak ada hak akses di sistem untuk membuka file
* Drive usang di sistem Anda
* Penghapusan laporan TR3 dari arsip Windows
* Menghubungi pengembang mungkin merupakan opsi yang lebih baik untuk hasil yang lebih baik
