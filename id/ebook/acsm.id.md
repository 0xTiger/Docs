## Apa itu file ACSM?

File dengan ekstensi .acsm disebut file Pesan Server Konten Adobe. File ini digunakan oleh **Adobe Digital Editions** untuk mengaktifkan dan mengunduh konten yang dilindungi Adobe DRM. Sebenarnya, file ACSM bukanlah file eBuku; tidak bisa dibuka dan dibaca seperti eBook lainnya (misalnya PDF); Itu hanya berisi data untuk mengaktifkan dan mengunduh ebook. Artinya, file ACSM hanya berisi informasi yang berkomunikasi dengan server Adobe. Pengguna Edisi Digital dapat melihat file ACSM jika mereka akan mengunduh eBuku, tetapi pengunduhan akan berhenti karena alasan apa pun. Pengguna dapat melanjutkan pengunduhan dengan mengklik dua kali pada file .acsm.

## Bagaimana cara kerja file ACSM?

Karena Adobe Content Server bertanggung jawab untuk melindungi eBook dan barang digital lainnya. Setelah pembelian, konten secara otomatis ditautkan ke ID Adobe pembeli. ID bersifat pribadi dan tidak dapat ditransfer ke orang lain. Pengguna harus mengaturnya sebelum membeli dokumen apa pun dengan perlindungan salinan Adobe. Pelanggan tidak bisa mendapatkan akses ke dokumen yang dilindungi dari penyalinan tanpa meneruskan ID Adobe mereka ke aplikasi server Adobe yang berjalan di latar belakang.

Saat pelanggan melakukan pembelian, ACSM adalah satu-satunya file yang dapat mereka unduh terlebih dahulu. Pelanggan harus menginstal perangkat lunak Adobe Digital Editions di sistem mereka dan menautkannya ke ID Adobe mereka untuk membuka file ACSM. Adobe Digital Editions akan memverifikasi ID untuk otorisasi untuk mengonversi file ACSM. Jika diautentikasi, pengguna dapat mendownload eBook mereka dalam format [EPUB](/id/ebook/epub/) atau [PDF](/id/pdf/). Adobe Digital Editions juga menggunakan file ACSM untuk mengenali direktori unduhan.
