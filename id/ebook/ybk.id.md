## Apa itu file YBK? ##

Format file eBook YanCEyWare dan Microsoft Encarta keduanya menggunakan ekstensi file YBK. eBuku YanCEyWare dibentuk oleh YanCEyWare. Semua aplikasi perangkat lunak tersedia untuk perangkat yang menjalankan Windows, mendukung file YBK. Ensiklopedia Encarta dapat diperbarui dengan file .ybk tanpa harus memperbarui keseluruhan program. YanCEyDesktop adalah program terkenal dan populer yang dapat dengan mudah mengelola dan menangani file YBK. YanCEyWare Reader menggunakan format file YBK ini secara ekstensif, sebagai aplikasi pembaca eBook untuk Windows Mobile atau Pocket PC. File ini dikembangkan dengan sumber daya hyper-linked yang besar.

## Masalah untuk membuka file YBK ##

Terkadang ketika sulit untuk membuka dan memfungsikan file YBK, berikut adalah daftar beberapa kemungkinan masalah:

* Membuka file YBK yang rusak
* Tautan yang tidak pantas dan tidak pantas ke file YBK dalam akses registri
* Penghapusan deskripsi YBK secara tidak sengaja dari arsip Windows
* Instalasi sebagian aplikasi yang menyediakan format YBK
*	Infected YBK file with an undesirable malware
