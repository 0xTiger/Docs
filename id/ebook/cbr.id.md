## Apa itu berkas CBR?

File dengan ekstensi .cbr adalah kumpulan gambar yang disimpan sebagai file biner terkompresi menggunakan format file kompresi RAR. Ini berisi gambar dalam gambar [JPEG](/id/image/jpeg/), [PNG](/id/image/png/) atau [GIF](/id/image/gif/) yang mewakili gambar dalam komik dan ditampilkan kepada pengguna saat file digunakan dalam pembaca ebook seperti Calibre. CBR mirip dengan [CBZ](/id/ebook/cbz/) selain itu CBZ dikompresi dan disimpan dalam format file [ZIP](/id/compression/zip/). File ini dapat disimpan di dalam file koleksi buku komik seperti [CBC](/id/ebook/cbc/).

## Format File CBR

File CBR adalah arsip [RAR](/id/compression/rar/) yang disimpan sebagai file biner ke disk. Gambar di dalam file CBR disimpan dan ditampilkan kepada pengguna secara berurutan saat dibuka oleh pembaca ebook. File CBR dapat didekompresi menggunakan perangkat lunak dekompresi apa pun yang mendukung format file RAR. Namun, gambar yang didekompresi dengan cara ini tidak dalam urutan yang sama seperti yang muncul di tampilan visual saat dibuka di perangkat lunak pembaca.
