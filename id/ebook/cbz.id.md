## Apa itu file CBZ?

File dengan ekstensi .cbz adalah file arsip buku komik [ZIP](/id/compression/zip/) yang merupakan kumpulan gambar, yang mewakili halaman buku komik. CBZ berbeda dengan file ebook lain yang tidak dikompresi. Sebagian besar pembaca ebook dan buku komik mendukung melihat file-file ini. [CBR](/id/ebook/cbr/) adalah format file arsip buku komik lain yang menggunakan kompresi [RAR](/id/compression/rar/), bukan ZIP. File CBZ dan CBR dapat disimpan di dalam file koleksi buku komik seperti [CBC](/id/ebook/cbc/). Aplikasi yang dapat membuka dan membaca file CBZ termasuk aplikasi perangkat lunak seperti Calibre dan Comic Book Reader.

## Format File CBZ

File CBZ adalah arsip ZIP yang berisi halaman eBook sebagai gambar dalam format file [PNG](/id/image/png/), [JPEG](/id/image/jpeg/), atau [GIF](/id/image/gif/). Itu dapat didekompresi dan isinya dapat diekstraksi menggunakan perangkat lunak dekompresi apa pun yang mendukung format file ZIP. Gambar yang diekstraksi menggunakan perangkat lunak dekompresi tidak dalam urutan yang sama seperti yang muncul di file CBZ karena informasi tentang urutan gambar ada di dalam file CBZ.
