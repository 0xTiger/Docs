## Apa itu file LIT?

File LIT milik **Microsoft Reader** kuno yang digunakan untuk membaca e-book. EBook yang ditampilkan oleh Microsoft Reader memiliki ekstensi **.lit** yang disingkat menjadi **Literature**. Format file LIT terdiri dari file Microsoft ITOL/ITLS HTML Help 2.0, yang memberikan semua jenis fitur luar biasa seperti kompresi [LZX](/id/compression/lzx/), representasi biner dari materi markup, dan data tambahan opsional arbitrer yang terkait dengan setiap aliran. Teks buku adalah sejumlah aliran markup OEBPS 1.0 dan subset dari OEBPS 1.0 CSS. Gambar adalah [GIF](/id/image/gif/), [JPEG](/id/image/jpeg/), dan [PNG](/id/image/png/).

## Berapa lama Microsoft Reader menampilkan file LIT

**Microsoft Reader** adalah aplikasi untuk membaca e-book yang telah dihentikan oleh Microsoft. Versi pertama Microsoft Reader dirilis pada Agustus 2000, yang menggunakan format LIT-nya sendiri. Itu tersedia untuk PDA dan komputer Windows. Nama aplikasi juga digunakan kemudian untuk aplikasi (yang awalnya dirilis dengan Windows 8) untuk membaca file [PDF](/id/pdf/) dan [XPS](/id/page-description-language/xps/). Aplikasi Microsoft Reader dihentikan pada tahun 2018.
