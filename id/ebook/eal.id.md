## Apa itu File EAL? ##

File **Kindle End Action** biasanya ditemukan dengan ekstensi .EAL. File ini terdiri dari xml yang ditulis untuk menunjukkan sesuatu tentang penulis setelah halaman terakhir eBuku.

## Format File EAL ##

File .EAL adalah file xml yang terdiri dari data pelanggan yang membeli buku dan juga fitur untuk menunjukkan lebih banyak tentang penulis e-book setelah halaman terakhir buku; ASIN juga disertakan untuk melompat ke halaman judul Kindle Store.
