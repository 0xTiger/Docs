## Apa itu file FKB? ##

FKB adalah ekstensi file eBook yang dikembangkan oleh Flipkart.com untuk aplikasi perangkat lunak Flipkart. Jenis ekstensi file ini biasanya ditemukan pada pengguna Windows 10. Ada spesifikasi dalam file FBK yang ukurannya dapat berkisar dari beberapa ratus kilobyte hingga beberapa megabyte tergantung pada panjang buku dan jumlah gambar yang disertakan. Untuk mengunduh file dari Aplikasi Android Flipkart eBooks, pertama-tama harus membeli eBook, lalu klik di Perpustakaan dan buku akan mulai diunduh. Jika Anda tidak dapat melihat dokumen yang dibeli, segarkan sistem agar berfungsi lebih baik.

## Sistem Operasi yang Didukung ##

Sistem operasi berikut mendukung file FKB:

* Windows 7
* Windows 8
* Windows 10
* Windows Server 2003/2008/2012/2016
* Mac OS X
* FreeBSD
* iOS
* Linux
* Android
* NetBSD
* OpenBSD

## Masalah untuk membuka file FKB ##

* Instalasi program pendukung yang tidak benar. Pastikan itu diunduh dengan benar dengan versi yang diperbarui
* Ruang perangkat keras rendah
* File rusak atau terinfeksi. Masalah ini dapat dengan mudah diselesaikan melalui antivirus yang terpasang di sistem Anda
