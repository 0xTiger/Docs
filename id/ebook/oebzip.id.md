## Apa itu file OEBZIP? ##

File OEBZIP diperkenalkan oleh International Digital Publishing Forum (**IDPF**). Demikian pula, [OEB](/id/ebook/oeb/), file ini adalah spesifikasi berbasis XML untuk struktur, konten, dan presentasi buku elektronik tetapi dalam bentuk terkompresi atau zip.

## Detail Teknis Format File OEBZIP ##

OEBZIP menyediakan fungsionalitas yang lebih baru di area kontrol rendering, termasuk, antara lain, peningkatan kosa kata markup (sekarang menjadi bagian dasar dari XHTML 1.1), dan dukungan CSS yang diperluas secara luas. Faktor keberlanjutan tidak berubah dari versi sebelumnya.

Format file OEBZIP terdiri dari satu set file, termasuk file paket wajib, yang menyertakan daftar manifes semua file komponen lainnya, dan tulang belakang, yang menunjukkan urutan pembacaan logis. Selain itu, untuk mengirim teks dalam format [XML](/id/web/xml/), pembaca file OEBZIP harus dapat merender gambar dalam format [JPEG](/id/image/jpeg/) dan [PNG](/id/image/png/) . Komponen konten dalam format lain dapat digabungkan, tetapi konten fallback harus disediakan dalam salah satu format dasar OEB.
  

Bundel file OEBZIP digunakan terutama sebagai format mid-state, dengan publikasi ke pengguna akhir yang dikelola melalui penerbit atau agregator yang menyediakan eBuku dalam bentuk yang sesuai untuk pemirsa eBuku yang berbeda. Versi penggantinya, [EPUB](/id/ebook/epub/) kemungkinan besar akan menjangkau pengguna akhir dalam format status akhir, serta berfungsi sebagai format status sedang.
