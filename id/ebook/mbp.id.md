## Apa itu file MBP?

File MBP dikenal sebagai File **Mobipocket Notes** tetapi tidak berisi eBuku itu sendiri; berisi catatan e-book yang dibuat oleh pengguna dalam sebuah e-book seperti koreksi, anotasi, gambar, serta beberapa tanda penting lainnya. File MBP hanya mengacu pada catatan yang ditambahkan pengguna saat membaca eBuku tertentu. Dengan kata lain, kita dapat mengatakan eBuku memungkinkan pengguna untuk menambahkan beberapa catatan di tempat yang mereka inginkan saat membaca dan catatan yang ditandai biasanya disimpan menggunakan ekstensi **.mbp**.

## Penjelasan singkat tentang file Mobipocket Notes

Paragraf di atas menunjukkan bahwa Mobipocket Notes memfasilitasi pengguna untuk menambahkan beberapa catatan jika diperlukan dan catatan ini disimpan dalam file MBP. Karena file-file ini sepenuhnya terkait dengan file e-book karena terdapat di dalam e-book, mereka biasanya juga disimpan dengan e-book dan terletak di lokasi yang sama di mana file e-book disimpan. File-file ini biasanya dihasilkan oleh program **MBP Reader** atau **Mobipocket Reader Desktop**, yaitu aplikasi yang berguna untuk membaca buku secara elektronik. File MBP disimpan dalam format biner dan dapat dibuka menggunakan utilitas pembaca MBP.
