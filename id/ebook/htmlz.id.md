## Apa itu berkas HTMLZ? ##

Ini adalah file arsip yang terdiri dari banyak file yang berisi situs web, seperti file HTML, file gambar, kode CSS, dll. Ini juga dapat mencakup file metadata OPF. Pembaca eBook dapat dengan mudah membuka catatan HTMLZ tanpa perlu membongkarnya. File dengan ekstensi file HTMLZ umumnya dapat dimulai sebagai dokumen dan publikasi dalam format file HTML zip khusus. Ekstensi file HTMLZ menentukan sistem aplikasi mana yang dapat membuka file tersebut. Padahal, aplikasi yang berbeda mungkin menggunakan ekstensi file yang serupa untuk jenis informasi yang berbeda. Ini berarti pembuka HTMLZ mungkin tidak dapat membuka semua jenis file HTMLZ. Pengembang utama file HTMLZ atau file eBuku lainnya adalah sumber terbuka untuk aplikasi perangkat lunak Calibre.

Masalah umum yang muncul ketika file HTMLZ sulit dibuka dan berfungsi adalah tidak adanya perangkat lunak yang sesuai. Mungkin ada beberapa masalah lain yang juga membatasi kemampuan untuk mengoperasikan file Format HTMLZ. Berikut beberapa masalah tersebut adalah:

* Pengemudi peralatan yang ketinggalan jaman
* Penghapusan penggambaran HTMLZ yang tidak disengaja dari arsip Windows
* File mungkin rusak jika sulit dibuka alih-alih perangkat lunak yang sesuai
* Tautan yang salah ke file HTMLZ dalam akses arsip
* Pengunduhan aplikasi yang mendukung format HTMLZ tidak sempurna
* File mungkin terinfeksi. Untuk ini, harus menginstal perangkat lunak antivirus yang tepat dan asli
