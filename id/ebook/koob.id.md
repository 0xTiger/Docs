## Apa itu file KOOB? ##

Ekstensi file .koob pertama kali dikembangkan oleh Vijay Sridhara yang terkait dengan file eBook. Ia juga dikenal sebagai file Koob eBook di bawah kategori file teks. Ada satu spesifikasi tentang file KOOB yaitu berisi bagian buku, gaya buku, arahan serta publikasi, dan informasi lain terkait eBook. Jika sistem tidak memiliki Koober maka KoobReader membantu membaca file ini. File KOOB membantu pengguna untuk mendapatkan semua informasi tentang eBook, itulah sebabnya nama lengkapnya adalah Koob eBook Format.

## Masalah untuk membuka file KOOB? ##

* Ketika ada file yang terinfeksi atau rusak, sulit untuk membukanya. Dalam situasi ini, pindai file KOOB dengan program antivirus yang terinstal di sistem
* Tidak memiliki akses untuk mengerjakan ekstensi file yang diperlukan dapat menyebabkan kesalahan
* Shortcut file pada sistem bukan file asli yang menyebabkan error
* Ruang drive yang tidak mencukupi dapat menyebabkan masalah untuk membuka ekstensi file
 *	The process copies the file from an external source may be not completed. In this case, it's impossible to open an incomplete file
