## Apa itu file OPF? ##

OPF adalah singkatan dari Open Package Format. Format file ini sebenarnya adalah file [XML](/id/web/xml/) tetapi dengan ekstensi **.opf**. Format file ini adalah OEB Package Format (OPF) yang mungkin berisi majalah digital atau publikasi lain yang dapat dibaca dengan **FlipViewer**. Ini juga mencakup metadata yang mendeskripsikan konten dan daftar halaman, gambar, dan teks yang direferensikan oleh publikasi.

## Detail Teknis Format File OPF ##

Ekstensi file .opf dapat dikategorikan ke dalam berbagai jenis file. Itu bisa berupa file yang diformat dengan standar yang ditetapkan oleh Open Packaging. OPF lebih dari sekadar file eBuku karena sebagian besar berisi informasi tentang eBuku dari judul hingga penulis serta informasi meta lainnya tentangnya. Inilah mengapa file-file ini dapat dibuka menggunakan perangkat lunak Adobe Reading. Juga, ini digunakan untuk file yang dibuat oleh program FlipAlbum. Program ini digunakan untuk membuat album foto digital dan file OPF ini mengacu pada album foto yang dibuat menggunakan perangkat lunak. File-file ini berisi konfigurasi album; struktur dan referensi file gambarnya, dan dapat dibagikan dengan orang lain juga. File OPF juga dikaitkan dengan **FlipBook**. File OPF dapat diedit dengan editor teks apa saja (misalnya Notepad atau Notepad++) karena dapat diformat menggunakan format XML.
