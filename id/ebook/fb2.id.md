## Apa itu berkas FB2?

File dengan ekstensi .fb2 adalah file eBook FictionBook 2.0 yang berisi struktur eBook. Ini didasarkan pada format XML dan berisi tag khusus untuk mendeskripsikan setiap elemen buku. Ini dikembangkan terutama untuk tulisan dan sastra fiksi, tetapi tidak terbatas pada ini saja. Formatnya mengakomodasi semua metadata serta konten itu sendiri dan memungkinkan fleksibilitas untuk sejumlah operasi seperti pemrosesan otomatis, pengindeksan, dan konversi ke format lain. Singkatnya, ini berfokus pada mendeskripsikan struktur file alih-alih menentukan penampilannya. Beberapa aplikasi serta API tersedia untuk mengonversi FB2 ke beberapa format lain di Windows, MacOS, dan Linux.

## Format File FB2

Format file FictionBook 2.0 dikembangkan oleh Dmitry Gribov dari FictionBook di Rusia. Tidak seperti format file Ebook populer lainnya seperti [EPUB](/id/ebook/epub/) yang berisi banyak file XML, ini didasarkan pada satu file XML yang berisi informasi lengkap tentang kontennya. Jika ada gambar di dalam file seperti [PNG](/id/image/png/) atau [JPG](/id/image/jpeg/), ini akan dikonversi ke Base64 dan disematkan di dalam dokumen. [Spesifikasi format file](http://gribuser.ru/xml/fictionbook/index.html.en) untuk FB2 berisi informasi untuk format versi 2.0.
