## Apa itu file DNL?

File DNL diklasifikasikan sebagai file e-book yang diperkenalkan oleh **DNAML** yang merupakan perusahaan pengembangan perangkat lunak yang berbasis di Australia. File DNL pada dasarnya digunakan untuk eBook, mereka juga dapat digunakan untuk ePresentations, eBrochures, eCatalogs, eCards, dan album foto. Selama dekade terakhir, DNAML telah menciptakan produk canggih di bidang penerbitan elektronik, berdasarkan sistem penulisan dokumen, sistem pengecer, solusi distribusinya, dan alat konversi penerbitan. Format eBuku DNL adalah salah satu produk terbaik dari pengembang DNAML. Faktanya, DNL terdiri dari teks mentah terkompresi, gambar, dan data navigasi; dilindungi menggunakan manajemen hak digital (DRM).
