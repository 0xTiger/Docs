## Apa itu file CEB?

File dengan ekstensi .ceb (ebook Cina) dikenal sebagai **file eBook Apabi**. File CEB menyertakan teks berformat bahasa Mandarin dan informasi tata letak lainnya, oleh karena itu, format file ini telah dihentikan. File format CEB dikembangkan oleh penerbit Apabi Pendiri Electronics. File dengan format ini tersedia dari perpustakaan APABI D-Lib yang mencakup koleksi eBuku berbahasa Mandarin. File-file ini sebenarnya adalah file biner yang dikodekan yang dapat dibaca menggunakan perangkat lunak **Apabi Reader 4.5**. Perangkat [Iliad eBook](https://en.wikipedia.org/wiki/ILiad) menyertakan penampil Apabi. Perangkat eBook Iliad keluar dari pasar pada tahun 2010.
