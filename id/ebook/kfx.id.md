## Apa itu file KFX?

File KFX adalah eBuku yang dibuat dalam Amazon Kindle File Format 10 (KF10) untuk perangkat Kindle. Perangkat KIndle versi lama menggunakan format file [AZW](/id/ebook/azw/) dan [AZW3](/id/ebook/azw3/) (KF8). KFX merupakan penyempurnaan dari versi sebelumnya dengan penambahan fitur khusus dalam hal penyajian content layout. Ini memperkenalkan fitur-fitur baru seperti dukungan untuk tanda hubung lunak menggunakan mesin pengaturan huruf baru yang disempurnakan. Itu juga memperkenalkan format file gambar baru yang disebut JXR, yang memiliki rasio kompresi lebih tinggi, dan font Bookerly baru yang merupakan font eksklusif Kindle yang memberikan peningkatan keterbacaan pada perangkat digital. Format ini hadir dengan kemampuan yang disempurnakan seperti gambar mini multi-halaman, penyesuaian otomatis konten di sekitar gambar dan video di lokasi tetap yang menghasilkan tata letak halaman dan DRM yang lebih baik. Untuk membuka dan melihat file semacam itu, Anda dapat menggunakan aplikasi Amazon Kinde yang tersedia untuk Windows, macOS, Android, dan iOS. Calibre adalah aplikasi perangkat lunak pembaca e-book lain yang dapat digunakan untuk membaca beberapa format file e-book yang berbeda.

## Format File KFX ##

KFX adalah format yang rumit karena menggabungkan beberapa fitur Format File Kindle yang berbeda, namun detail internal tentang struktur filenya tidak diketahui. Amazon telah menambahkan skema DRM untuk melindungi hak digital dan menyimpan format KFX dari anti-pembajakan.
