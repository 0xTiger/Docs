## Apa itu berkas AZW1? ##

AZW1 adalah format file eBook yang juga dikenal sebagai **eBook Amazon Kindle Topaz** yang telah diluncurkan melalui **Whispernet**. Format file ini dapat digunakan dengan perangkat lunak Kindle di PC dan smartphone dan bertanggung jawab untuk menyimpan eBuku yang diunduh dari jaringan 3G Whispernet Amazon.

## Format File AZW1 ##

Perangkat pembaca buku Amazon Kindle dan perangkat lunaknya menggunakan file format AZW1 yang memiliki ekstensi .azw1. File ini juga dikenal sebagai file eBook Amazon Topaz, file AZW1 ini digunakan untuk menyusun dan mengelola bagian yang diunduh dari file eBook Amazon Kindle dari jaringan **3G Whispernet** Amazon. Perangkat lunak Amazon Kindle dapat diinstal pada sistem berbasis **Microsoft Windows** untuk menjalankan dukungan untuk file AZW1 ini dan fitur relevan yang terintegrasi ke dalam aplikasi. Versi program ini untuk lingkungan Mac juga tersedia. Perangkat lunak ini biasanya digunakan untuk mulai mengunduh dan melihat konten buku-buku Amazon Kindle ini. Karena aplikasi Amazon Kindle menyediakan dukungan untuk file AZW1 dan juga dapat diinstal di dua sistem terkenal ini, pencipta dapat dengan mudah membuat dan membagikan buku Kindle dengan benar.
