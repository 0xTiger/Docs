## Apa itu berkas AZW4?

AZW4 adalah ebook Amazon Print Replica yang dikembangkan Amazon untuk perangkat lunak desktop Kindle dan perangkat eReader Kindle. AZW4 tidak mirip dengan [MOBI](/id/ebook/mobi/), [AZW](/id/ebook/azw/) atau [AZW3](/id/ebook/azw3/). Format ini sebenarnya adalah pembungkus untuk eBook [PDF](/id/pdf/) atau MOBI dengan tambahan DRM.

## Format file AZW4

**AZW4** telah diperkenalkan sebagai versi baru software **Textbooks** yang mirip dengan **PDF**. Sebuah eBuku AZW4 berisi file PDF lengkap di dalamnya, atau dapat dianggap sebagai file PDF dengan pembungkus yang disebut AZW4. Terutama AZW4 digunakan oleh Kindle eReader. Namun, aplikasi lain juga dapat menggunakan ekstensi file .azw4. Bahkan program malware dapat membuat file .azw4. Hati-hati dengan file .azw4 yang berasal dari sumber yang meragukan.
