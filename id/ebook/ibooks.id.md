## Apa itu file IBOOKS? ##

Sebagai format file eBook, format file iBooks dibuat pada perangkat yang berbeda seperti komputer, ponsel laptop, dan tablet yang menjalankan sistem operasi Apple. File iBooks tidak hanya berisi teks tetapi juga video, presentasi, gambar, objek 3D serta media lain yang terdapat dalam digital. Satu hal penting adalah bebas biaya untuk merancang dan menerbitkan eBuku. File iBooks hanya dapat ditransfer ke format iBooks, file PDF, atau file teks biasa.

## Spesifikasi Teknis ##

* Apple mengembangkan iBooks berdasarkan format "EPUB 3" dengan beberapa ketentuan tambahan dan eksklusif
* Program iBooks Author memungkinkan pengguna mentransfer buku mereka baik dalam format iBooks, sebagai file teks biasa, atau sebagai dokumen PDF
* Buku yang disimpan sebagai file iBooks kemudian dapat tersedia di iTunes untuk diunduh dan dibaca oleh semua jenis perangkat Apple oleh pengguna
* Terutama, format iBooks menggunakan format "iBooks 2" dan hanya digunakan untuk membuat buku teks
* Namun, saat ini digunakan untuk menghasilkan semua jenis buku digital, tidak hanya buku pendidikan
* Ini mendukung format [EPUB](/id/ebook/epub/) dan [PDF](/id/pdf/), dan aplikasi iBooks di perangkat iOS atau Mac yang menjalankan OS 10.9 atau lebih lama dapat digunakan untuk membacanya
* Apple menawarkan platform iBooks Author untuk membuat iBooks. iBooks Author mengizinkan pengguna untuk membuat iBooks menggunakan standar EPUB multi-sentuh khusus Apple yang unik

## Sejarah Singkat ##

Pada 19 Januari 2012, Apple mengumumkan iBooks Author di acara khusus terkait pendidikan di New York City. Kemudian pada saat yang sama, Apple juga merilis iBooks 2 dan grup iBooks Bookstore baru untuk buku teks. Meskipun Apple membuatnya dapat diakses untuk diunduh gratis di Mac App Store, itu eksklusif dan hanya tersedia untuk macOS sebelum ditahan pada tahun 2020. Umumnya, dokumen dibuat di Pages dan kemudian dibawa ke iBooks Author untuk menyelesaikan tata letaknya. Semua proses ini dapat dengan mudah dilakukan di laptop atau desktop.
