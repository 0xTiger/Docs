## Apa itu file TS?

File dengan ekstensi .ts adalah aliran video yang menyimpan data pada DVD. Ia menggunakan algoritme kompresi MPEG-2 ([.mpeg](/id/video/mpg/)) untuk mencapai efisiensi dan kompatibilitas maksimum di berbagai jenis media seperti streaming internet dan penyiaran. Format file TS dibuat agar dapat diputar di perangkat dengan koneksi internet yang buruk, seperti smart TV.

## Format File TS - Informasi Lebih Lanjut

Data dalam file TS mirip dengan file [MP4](/id/video/mp4/), tetapi dibagi menjadi potongan-potongan kecil. Setiap potongan terdiri dari potongan kecil video, diikuti dengan sedikit audio, dan keterangan opsional. Satu file TS terdiri dari banyak potongan seperti itu. Selain video, audio, dan keterangan, setiap potongan juga terdiri dari beberapa data tambahan untuk mendeteksi kesalahan dalam potongan tersebut. Karena itu, ukuran file TS agak lebih besar.

### Mengapa menggunakan Format File TS?

Jadi, jika file TS berukuran agak besar, apa keuntungan yang ditawarkan untuk menggunakannya daripada format file lain? Nah, dalam penyiaran, potongan kecil video dan audio dapat dikirim melalui media komunikasi (kabel atau radio) secara real-time. Data ekstra dalam potongan digunakan oleh penerima untuk melewati potongan rawan kesalahan.

Selain itu, file TS digunakan karena sistem penyiaran tidak memerlukan seluruh aliran untuk memutarnya. Transmisi dapat diambil dan dapat digunakan secara real time dengan merakit audio dan video.

## Bagaimana Cara Memutar File TS?

Nah, file TS dapat dibuka dan diputar di [pemutar media VLC VideoLAN](https://www.videolan.org/vlc/features.html) populer yang tersedia secara gratis untuk diunduh.
