## Apa itu file PZ?

File PZ adalah file proyek yang dibuat dengan aplikasi pembuat klip video online **Panzoid**. Ini berisi informasi tentang proyek video dan kontennya seperti adegan, objek, efek, pencahayaan, FX, audio, dan musik. Panzoid memungkinkan Anda membuat video online menggunakan fasilitas pengeditan video **ClipMaker**.

File PZ dapat dibuka dengan Panzoid ClipMaker dan Panzoid Backgrounder.

## Format File PZ

File PZ disimpan sebagai file biner ke disk. Saat pengguna membuat project baru dengan [Panzoid](https://panzoid.com/), file PZ akan dibuat untuk menyimpan detail objek yang ditentukan pengguna. Informasi yang sama disinkronkan dengan pengguna lain jika proyek tersebut adalah kontributor lain. File PZ kemudian dapat diunduh ke komputer.

Saat Anda ingin membuka file PZ, Anda dapat memuatnya lagi di **Panzoid** dengan menggunakan opsi Muat dari antarmuka online. File dimuat secara online dan isinya dimuat untuk modifikasi.

### Klip Video yang Didukung oleh File PZ

Panzoid memiliki program ClipMaker dan Backgrounder yang digunakan bersama untuk pembuatan video. Beberapa jenis video berbeda dapat dibuat dengan Panzoid termasuk latar belakang khusus, seni saluran YouTube, video media sosial, dan gambar mini.
