## Apa itu file MJPEG? ##

File dengan ekstensi .mjpeg (Motion JPEG) adalah file video yang dibuat dengan mengompresi bingkai video individual sebagai gambar JPEG. Ini berbeda dari format file MPEG-1 dan MPEG-2 standar di mana kompresinya adalah interframes dan ditentukan berdasarkan prediksi konten dalam frame berikutnya. MJPEG didukung oleh berbagai aplikasi yang mencakup browser web, pemutar media, kamera digital, webcam, server streaming. Plug-in tersedia untuk aplikasi yang ingin menggunakan format file MJPEG.

## Format File MJPEG ##

MJPEG didasarkan pada standar kompresi matang yaitu [JPEG](/id/image/jpeg/) yang memiliki pustaka yang terdefinisi dengan baik. Itu tidak dapat distandarisasi mirip dengan standar seperti MPEG-2 dan tidak ada dokumentasi yang tersedia yang dapat disebut sebagai spesifikasi lengkap untuk "Motion JPEG". Dengan tidak adanya standar seperti itu, keluaran dari produsen yang berbeda dapat menimbulkan masalah kompatibilitas. Namun, perusahaan seperti Microsoft dan Apple telah mendokumentasikan bagaimana file M-JPEG disimpan dalam format aslinya seperti [AVI](/id/video/avi/) dan [QT](/id/video/qt/). [rfc2435](https://tools.ietf.org/html/rfc2435) menjelaskan Format Muatan RTP untuk Video terkompresi JPEG dan dapat dikonsultasikan sebagai materi referensi.
