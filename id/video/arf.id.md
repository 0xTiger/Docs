## Apa itu file ARF?

File ARF (Advanced Recording File) adalah file Perekaman Berbasis Jaringan (NBR) yang dibuat dari rapat online yang dilakukan melalui WebEx. Ini berisi video dan audio berkualitas lebih tinggi, dan merupakan format file yang banyak digunakan dalam rapat WebEx. Informasi yang terkandung dalam file ARF mencakup data video, daftar isi, daftar peserta, rekaman desktop, berbagi aplikasi, dan konten rapat lainnya. File ARF tidak dapat diarahkan untuk diedit, tetapi awal dan akhir pemutaran dapat dipotong. ARF adalah format file default yang digunakan oleh rapat WebEx.

## Format File ARF

File ARF disimpan ke situs Webex sebagai file biner di bagian **Rekaman Saya** pada akun host. Ini hanya dapat dilakukan oleh penyelenggara rapat dari dalam rapat online menggunakan opsi Rekam di Server. File ARF dapat diputar ulang menggunakan Network-Recording Player.

## Opsi Konversi File ARF

File ARF dapat dikonversi ke format [WMV](/id/video/wmv/), [SWF](/id/page-description-language/swf/), atau [MP4](/id/video/mp4/).
