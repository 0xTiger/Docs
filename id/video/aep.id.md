## Apa itu file AEP?

File AEP adalah file proyek yang dibuat oleh Adobe After Effects saat Anda menyimpan proyek After Effect. Ini adalah jenis file utama yang digunakan oleh After Effects untuk menyimpan semua pengaturan dan data yang terkait dengan proyek seperti cuplikan video, gambar diam, file audio, dan efek lainnya. File AEP berisi informasi tentang komposisi, lapisan, bingkai utama, efek, dan elemen lain yang merupakan bagian dari proyek After Effects.

Anda dapat membuka file AEP dengan perangkat lunak Adobe After Effects.

## Format File AEP - Informasi Lebih Lanjut

Adobe After Effect menyimpan file AEP dalam format file biner. Struktur file internal file AEP tidak tersedia untuk umum untuk referensi pengembang dan, itulah sebabnya, ini hanya dapat dibuka di Adobe After Effect. File AEP menyimpan referensi ke elemen yang merupakan bagian dari proyek tetapi sebenarnya tidak berisi koleksi dan elemen tersebut. Jika Anda memindahkan file referensi setelah proyek dibuat, After Effects tidak akan dapat menemukan file tersebut dan Anda tidak akan dapat menggunakan file media tersebut.

### Jenis File Proyek After Effects

After Effects menyimpan file proyek di:

* `File AEP` - Ini disimpan dalam format biner dan berukuran kecil. File biner dimuat lebih cepat dan dapat dibagikan dengan pengguna lain dengan mudah.
* `File AEPX` - Ini adalah format file proyek After Effects berbasis XML yang menyimpan data dalam format teks. File AEPX biasanya lebih besar dan lebih lambat untuk dimuat dibandingkan format file AEP.

## Cara Mengonversi File AEP

File AEP dapat dikonversi ke format file populer lainnya seperti:

* [.MOV](/id/video/mov/) - Film Apple QuickTime
* [.AVI](/id/video/avi/) - Interleave Audio Video
* [.JPG](/id/image/jpeg/) - Gambar JPEG
* [.PNG](/id/image/png/) - Grafik Jaringan Portabel
* .DPX - Pertukaran Gambar Digital
* .EXR - Gambar OpenEXR
* .M4A - Audio MPEG-4
* [.MP3](/id/audio/mp3/) - Audio MP3
* [.WAV](/id/audio/wav/) - Audio GELOMBANG
* .AEPX - Proyek XML After Effects
* .AET - Templat After Effects
* .PRPROJ - Adobe Premiere Pro
