## Apa itu file STR?

File STR adalah file rekaman yang dibuat oleh YouTube untuk streaming langsung yang diselenggarakan oleh acara streaming pengguna. File disimpan secara otomatis oleh Youtube di backendnya setelah streaming langsung selesai. Video streaming langsung dapat disiarkan ke berbagai saluran sehingga penonton dapat berinteraksi dengan penerbit melalui komentar. File SRT ini dapat diakses menggunakan API streaming langsung YouTube yang menyediakan akses ke server YouTube.

## Format File STR

YouTube menyimpan file STR ke server dalam format file biner. Detail struktur file internal file STR tidak tersedia untuk umum.
