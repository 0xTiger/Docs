## Apa itu file AVCHD? ##

File AVCHD adalah format file rekaman yang diperkenalkan untuk menyimpan video definisi tinggi (HD) ke DVD, hard disk drive, dan kartu memori. Video AVCHD, yang direkam ke DVD dan Blu-ray, dapat diputar dengan mudah di sebagian besar pemutar Disk Blu-ray. Juga, barang-barang AVCHD juga dapat direkam pada kartu SD dan dimainkan oleh banyak pemutar Disk Blu-ray, perangkat televisi, dan konsol media. **Percobaan ffdshow** dan **Libavcodec** adalah kodek sumber terbuka yang dapat mendekode file AVCHD.


## format file AVCHD

Format file AVCHD memungkinkan kamera video digital definisi tinggi untuk merekam sinyal HD; menggunakan teknologi pengkodean kompresi efisiensi tinggi. Untuk mengompres video; format [MPEG-4](/id/video/mp4/) AVC/[H.264](/id/video/h264/) telah diadopsi, dan Dolby Digital® digunakan untuk mengompresi data audio. Sedangkan format MPEG-4 AVC/H.264 lebih efisien daripada format kompresi citra konvensional. Selain itu, format video 3D (format MVC) dan 1080/60p(1080/50p) telah ditambahkan sebagai ekstensi ke format file AVCHD untuk format versi 2.0 (AVCHD 3D, AVCHD Progressive).
Harap perhatikan bahwa perangkat harus kompatibel dengan AVCHD V.2.0 untuk menangani video terkait AVCHD V.2.0.

## Perbandingan antara format file AVCHD dan MP4 ##

Format file Advanced Video Codec High Definition (AVCHD) sangat cocok untuk membuat rekaman Blu-ray Disc® dan menonton HDTV. Sebagai perbandingan, MP4 jauh lebih mudah diputar di perangkat portabel dan juga mudah dipindahkan, disalin, dan disimpan di web.
