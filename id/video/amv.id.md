## Apa itu file AMV? ##

Akronim AMV sebagai Actions Media Video diklasifikasikan sebagai format file video yang diproduksi untuk pemutar media [MP4](/id/video/mp4/) portabel Cina dan pemutar MP3 S1 dengan pemutaran video. Video AMV pada dasarnya memiliki dua format berbeda: Yang pertama untuk chip Actions dan yang terbaru untuk chip M5661 ALi; format untuk ALi one adalah ALIAVI. Format video adalah varian dari Motion JPEG, dengan tabel pembagian tetap, bukan variabel. Format audio adalah varian dari IMA ADPCM. 8 byte pertama dari setiap frame adalah asal, indeks, dan beberapa sampel 16-bit yang disandikan; biasanya, file AMV memutar suara pada 22050 sampel/detik. Video AMV adalah resolusi terkompresi tetapi rendah (96x96 hingga 320x240 piksel), yang dapat diputar di desktop menggunakan perangkat lunak pemutar media apa pun.

## Format file AMV ##

Sebagai varian dari file [AVI](/id/video/avi/), beberapa pemutar media China biasanya menggunakan file AMV. Hebatnya, file AMV berisi teks AMV dan amvh yang khas di headernya; sedangkan file AVI berisi teks AMI dan avih.

Format AMV dirancang untuk memungkinkan pemutar media Cina yang murah dan tersedia dengan mudah untuk menyimpan dan memutar banyak video beresolusi rendah. Video AMV biasanya tidak terkompresi tinggi, tetapi ukuran filenya kecil karena resolusi dan frekuensi gambarnya yang rendah. Hal ini membantu pemutar media murah untuk menjalankan format file AMV tanpa kesulitan apapun.
