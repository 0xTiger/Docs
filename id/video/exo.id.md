## Apa itu file EXO?

File EXO adalah file potongan video, diunduh oleh YouTube versi Android dalam format EXO. Saat pengguna ingin mengunduh video untuk pemutaran offline, video tersebut dibagi menjadi potongan-potongan kecil dan setiap potongan diunduh dan disimpan ke perangkat pengguna sebagai file EXO. Beberapa file EXO ini digabungkan untuk diputar secara offline oleh aplikasi YouTube.

## Format File EXO

File EXO disimpan dalam format file milik YouTube. Mereka disimpan sebagai file terenkripsi terkompresi dan hanya dapat dibuka dengan aplikasi YouTube Android. Semua potongan yang diunduh dikumpulkan dan didekripsi oleh aplikasi untuk dimainkan secara offline. Dienkripsi, file EXO tidak dapat dikonversi ke format file lain seperti [MP4](/id/video/mp4/).
