## Apa itu File NSV?

File NSV (NullSoft Stream Video) adalah format file video streaming yang digunakan oleh NullSoft untuk streaming video di internet. Format tersebut diperkenalkan dan dikembangkan oleh NullSoft yang juga pembuat media player Winamp. NSV mendukung [MP3](/id/audio/mp3/) dan VP3 sebagai format audio dan video default. Penyempurnaan lateral memungkinkan NSV menggunakan codec VP versi lateral untuk video seperti [VP6](/id/video/vp6/), [VP8](/id/video/vp8/), dan [VP9](/id/video/vp9/). Format file NSV tidak lagi didukung setelah NullSoft diakuisisi oleh Radionomy pada tahun 2014.

## Format File NSV

NSV adalah format file video wadah uap yang dikembangkan untuk streaming video melalui internet. Ketersediaannya melalui daftar SHOUTcast menawarkan streaming audio dan video terintegrasi. Streaming audio dan video individual dapat diekstraksi dan dikonversi ke format file MP3 dan [MP4](/id/video/mp4/) menggunakan pemutar media VLC.
