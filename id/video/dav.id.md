## Apa itu file DAV?

File DAV adalah file video terkompresi dan terenkripsi yang dibuat oleh DVR365 atau sistem pengawasan digital Dahua Technology. Sistem ini terhubung ke sistem pengawasan kamera closed-circuit television (CCTV). Rekaman video yang diterima dari kamera ini disimpan sebagai file DAV di sistem. Karena file DAV dienkripsi, hanya sistem CCTV yang membuatnya atau beberapa pemutar media khusus yang dapat membukanya. File DAV juga dapat dikonversi ke format file [AVI](/id/video/avi/).

## Format File DAV - Informasi Lebih Lanjut

File DAV disimpan ke disk dalam format file biner terenkripsi. Enkripsi mencegah penggunaan yang tidak sah dari file-file ini. Oleh karena itu, ini adalah hak milik sistem CCTV oleh [Teknologi Dahua](https://www.dahuasecurity.com/). Beberapa kamera lain dari Dahua Technologies juga dapat menyimpan video pengawasan sebagai file DAV.

Dahua Smart Player dapat digunakan untuk membuka dan memutar file DAV.
