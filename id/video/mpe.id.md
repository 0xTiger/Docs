## Apa itu file MPE?

File MPE adalah file video yang disandikan dan disimpan dalam format file [MPEG](/id/video/mpg/). Itu diperpanjang dengan bit-rate tambahan untuk didukung sebagai versi berikutnya dari kompresi MPEG-2. MPEG-2, bagaimanapun, adalah jenis kompresi MPEG yang paling umum digunakan. Ekstensi .mpe awalnya digunakan sebagai kependekan dari format file .mpeg.
