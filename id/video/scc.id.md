## Apa itu file SCC?

File SCC adalah file video yang dibuat oleh ScreenFlow, aplikasi perekaman layar desktop Mac OS X. Ini menyimpan video yang diambil dalam format eksklusif dan mungkin juga menyertakan audio yang diambil dari mikrofon komputer atau dari kartu suara komputer.

File SCC digunakan oleh ScreenFlow sebagai file sementara untuk menyimpan data video yang belum dirender sebelum diekspor ke format video akhir, selama proses pengeditan di ScreenFlow, saat Anda membuat perubahan pada video, menerapkan efek, atau melakukan pengeditan lainnya, perangkat lunak menyimpan kemajuan proyek di File SCC yang berisi data video mentah dan instruksi pengeditan.

Setelah Anda menyelesaikan proses pengeditan dan siap mengekspor video Anda, ScreenFlow merender file SCC ke dalam format keluaran yang diinginkan, seperti MP4 atau MOV. Proses rendering melibatkan penerapan semua pengeditan dan efek untuk menghasilkan file video akhir.

Setelah proses rendering selesai, file SCC sementara biasanya dihapus dan file video akhir disimpan dengan format keluaran pilihan yang dapat dibagikan, diunggah, atau digunakan sesuai kebutuhan.

## Apa itu ScreenFlow?

ScreenFlow dari Telestream, Inc. adalah perangkat lunak screencasting dan pengeditan video untuk sistem operasi macOS. Itu dapat menangkap audio dan video dari komputer, mengedit video yang diambil, menambahkan sorotan atau anotasi, dan menghasilkan sejumlah jenis file berbeda seperti AIFF, GIF, M4V, MOV, dan MP4.

Setelah menangkap layar Anda, ScreenFlow menyediakan antarmuka pengeditan video yang komprehensif, Anda dapat memotong dan membagi klip video, menambahkan transisi dan animasi, menyisipkan teks dan keterangan, dan menggabungkan file media tambahan seperti gambar, musik, atau video lainnya, juga menawarkan built-in alat pengeditan audio untuk menyesuaikan tingkat volume, menghilangkan kebisingan latar belakang, dan menerapkan efek.
