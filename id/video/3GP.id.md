## Apa itu file 3GP?

3GP (Format File 3GPP) adalah format wadah multimedia yang digunakan pada ponsel 3G. Itu juga dapat dimainkan di beberapa ponsel 2G dan 4G. 3GP didefinisikan oleh Third Generation Partnership Project (3GPP) untuk layanan multimedia 3G UMTS (Universal Mobile Telecommunications System). File 3GP disimpan dengan ekstensi .3gp. Format file 3GP dirancang untuk telepon berbasis GSM (Sistem Global untuk Komunikasi Seluler) dan merupakan format standar untuk file media yang dikirim melalui MMS (Layanan Pesan Multimedia). Tidak seperti [MP4](/id/video/mp4/), yang menggunakan codec H.263, file 3GP sebaiknya menggunakan codec H.263.

## Detail Teknis Format File 3GP

Format video 3GP didasarkan pada format file media berbasis ISO yang ditentukan dalam ISO/IEC 14496-12 â€“ MPEG-4 Bagian 12. Ini dikembangkan untuk ponsel guna mengurangi penyimpanan, bandwidth, dan penggunaan data. Format file 3GP mendukung codec berikut:

- Video
  - MPEG-4 Part 2
  - H.263
  - MPEG-4 Part 10 (AVC/H.264)
- Audio
  - AMR-WB
  - AMR-WB+
  - AMR-NB
  - AAC-LC
  - HE-AAC v1
  - HE-AAC v2 (Enhanced aacPlus)

Format file 3GP memungkinkan penggunaan codec H.263 dan AMR dan menentukan kotak baru yang dirujuk oleh codec tersebut. Poin kode ini didaftarkan dalam format file media berbasis ISO untuk penyimpanan informasi khusus media MPEG-4.
