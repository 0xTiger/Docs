## Apa itu file MPV?

File MPV adalah file video aliran dasar MPEG yang terdiri dari data video mentah dan tidak terkompresi. Konten video di dalam file MPV terdiri dari spesifikasi kompresi MPEG-2 terintegrasi. File MPV tidak berisi data audio apa pun. Format file MPV juga digunakan sebagai input untuk aplikasi pemrosesan video karena aliran dasar [MPEG](/id/video/mpg/) menggunakan standar pengkodean MPEG-2 terutama. File MPV dapat dibuka dan diputar dengan aplikasi perangkat lunak yang mendukung format file video MPEG-2. Perangkat lunak pemutar media VideoLAN LVC adalah salah satu perangkat lunak yang dapat membuka dan memutar file MPV.

## Format File MPV

File MPV disimpan ke disk sebagai file biner biasa, mengikuti struktur format file MPEG. Namun, format file internal mereka tidak tersedia secara publik untuk referensi pengembang.
