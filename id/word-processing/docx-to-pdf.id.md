## DOCX vs PDF

DOCX dan PDF adalah format file dokumen populer untuk menulis konten dan berbagi. Kami memberikan perbandingan kedua format file tersebut dengan membandingkan fitur-fiturnya seperti di bawah ini.

|Fitur|DOCX|PDF|
|---|---|---|
|Pengeditan dan pemformatan konten |+|–|
|Anotasi dan komentar |+ |+|
|Pengeditan bersama |+ |–|
|Pembuatan dan pengisian formulir |+ |+|
|Alat perlindungan konten |+ |+|
|Tanda tangan digital |+| +|
|Tampilan yang dijamin sama (WYSIWIG) |– |+|
