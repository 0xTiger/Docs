## Tentang Konversi DOC ke JPG

Jika Anda pernah bekerja dengan Microsoft Word versi lama (yaitu versi sebelum Microsoft Word 2007), Anda harus terbiasa dengan format file DOC. Itu adalah format file default yang digunakan oleh Microsoft Word sebelum diperkenalkannya format file DOCX dengan Microsoft Word 2007. Versi DOC ke JPG adalah persyaratan yang sangat umum, terutama ketika Microsoft tidak mengizinkan Anda mengekspor dokumen word ke gambar JPG. Selain itu, versi terbaru Microsoft Word tidak sepenuhnya kompatibel dengan file DOC lama. Anda dapat menggunakan aplikasi online seperti aplikasi Aspose.Words untuk mengonversi format file [DOC](/id/word-processing/doc/) ke [JPG](/id/image/jpeg/).

## Konversi DOC ke JPG - Demo Gratis

Aplikasi Aspose.Words adalah seperangkat alat online yang memungkinkan Anda mengonversi file DOC Anda ke beberapa format file berbeda. Anda dapat mengonversi file DOC ke JPG hanya dengan beberapa klik dari browser web Anda. Keuntungan menggunakan aplikasi Aspose.Words adalah tidak memerlukan pengkodean, cepat dan tepat, dan yang terpenting, sepenuhnya gratis.

Selama proses DOC ke JPG, setiap halaman dokumen diubah menjadi gambar JPG dan tersedia untuk diunduh. Outputnya adalah sekumpulan gambar JPG yang disimpan ke disk.

### Bagaimana Cara Mengonversi DOC ke JPG?

Jadi, bagaimana cara mengubah DOC ke JPG? Anda dapat mengonversi format file DOC ke JPG menggunakan [aplikasi Aspose.Words DOC ke JPG](https://products.aspose.app/words/conversion/doc-to-jpg) sebagai berikut.

1. Buka aplikasi Aspose.Words untuk [Konverter DOC ke JPG](https://products.aspose.app/words/conversion/doc-to-jpg)
1. Unggah file DOC Anda dengan mengklik tombol unggah
1. Setelah selesai, tekan tombol Konversi. Ini akan memulai proses konversi file DOC ke JPG di latar belakang
1. Anda akan diminta untuk mengunduh file JPG yang telah dikonversi setelah proses konversi DOC ke JPG selesai
