## Tentang Konversi DOCX ke DOCM

DOCM adalah format file dokumen Microsoft Word 2007 yang memiliki kemampuan menjalankan makro. Format file internal DOCM mirip dengan format file DOCX tetapi dengan tambahan fitur dukungan makro yang tidak tersedia di DOCX.

Anda dapat mengonversi file DOCX ke DOCM menggunakan layanan online gratis. **[Aspose.Words App](https://products.aspose.app/words/family)** adalah salah satu rangkaian alat yang memungkinkan Anda melakukan pengoperasian pada berbagai jenis dokumen Word secara online dari browser web Anda. Ia menawarkan fitur-fitur seperti memuat dokumen WORD, melihat isinya, mengonversi ke berbagai jenis format file keluaran seperti PDF, PNG, BMP, dan JPEG, pemisahan, rotasi, dan banyak lainnya. Anda dapat mengonversi [DOCX](/id/word-processing/docx/) menjadi [DOCM](/id/word-processing/docm/) menggunakan aplikasi Aspose.Words.

Anda dapat memuat dokumen Word dan mengonversinya ke format file lain seperti DOC, DOCX, PDF, BMP, PNG, JPEG, dan format file lainnya.

Coba aplikasi online Aspose.Words sekarang untuk **[Mengonversi DOCX ke DOCM](https://products.aspose.app/words/conversion/docx-to-docm)** tanpa repot.

### Langkah-langkah Mengonversi DOCX ke DOCM online

Anda dapat mengonversi format file DOCX ke DOC menggunakan [aplikasi Aspose.Words DOCX ke DOCM](https://products.aspose.app/words/conversion/docx-to-docm) sebagai berikut.

1. Buka aplikasi Aspose.Words untuk [Konverter DOCX ke DOCM](https://products.aspose.app/words/conversion/docx-to-docm)
1. Unggah file DOCX Anda dengan mengklik tombol unggah
1. Setelah selesai, tekan tombol Konversi. Ini akan memulai proses konversi file DOCX ke DOC di latar belakang
1. Anda akan diminta untuk mengunduh file RTF yang telah dikonversi setelah proses konversi DOCX ke DOC selesai
