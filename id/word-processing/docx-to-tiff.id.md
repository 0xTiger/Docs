## Tentang Konversi DOCX ke TIFF

TIFF (Tagged Image File Format) adalah format file gambar raster yang digunakan pada berbagai perangkat yang mematuhi format file TIFF. Pengguna Microsoft Word menyimpan dokumen mereka dalam format file DOCX default. DOCX ke TIFF mengonversi semua halaman dokumen Word menjadi gambar TIFF untuk ditampilkan pada perangkat yang mendukung [TIFF](/id/image/tiff/). Alat dan aplikasi konversi online seperti aplikasi Aspose.Words dapat mengonversi file DOCX menjadi gambar TIFF.

### Langkah-Langkah Mengonversi DOCX ke TIFF online - DEMO GRATIS

Mengonversi DOCX ke TIFF menggunakan aplikasi online Aspose.Words sederhana dan mencakup langkah-langkah berikut.

1. Buka aplikasi Aspose.Words untuk [Konverter DOCX ke TIFF](https://products.aspose.app/words/conversion/docx-to-tiff)
1. Unggah file DOCX Anda dengan mengklik tombol unggah
1. Setelah selesai, tekan tombol Konversi. Ini akan memulai proses konversi file DOCX ke TIFF di latar belakang
1. Anda akan diminta untuk mengunduh file TIFF yang telah dikonversi setelah proses konversi DOCX ke TIFF selesai
