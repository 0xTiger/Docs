## Apa itu file WPS?

File dengan ekstensi .wps adalah dokumen Microsoft Works yang digunakan sebelum tahun 2006 sebagai dokumen kata. Itu mirip dengan format file [DOC](/id/word-processing/doc/) yang baru diperkenalkan saat itu, tetapi fiturnya tidak sebanyak format file baru. Misalnya, itu tidak mendukung fungsionalitas makro dan opsi pemformatan lanjutan. File WPS masih dapat dibuka dengan pengolah kata modern, tetapi untuk Windows 10, Anda perlu menginstal konverter untuk memuat file-file ini. Microsoft Works digantikan oleh pengolah dokumen Microsoft Word pada tahun 2010.

## Spesifikasi Format File

Spesifikasi format file dokumen WPS tidak tersedia untuk umum karena Microsoft tidak membuka spesifikasinya untuk pengguna akhir. WPS adalah file biner dan dapat dibuka dengan filter konversi format file yang dibuat oleh Microsoft untuk prosesor Word. LibreOffice, NeoOffice, dan OxygenOffice telah menyertakan pustaka C++ umum, libwps, yang dapat mengekstraksi teks dari berbagai versi Microsoft Works.
