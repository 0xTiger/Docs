## Tentang Konversi DOCX ke DOC

Jenis file DOC telah menjadi format file dokumen populer yang digunakan pada tahun 2007 dan versi Microsoft Word sebelumnya. Dengan dirilisnya Office 2007, Microsoft memperkenalkan format file DOCX berdasarkan spesifikasi Office Open XML. File DOC disimpan sebagai file biner dibandingkan dengan DOCX yang disimpan dalam format file XML terkompresi. Ada banyak alat yang tersedia online untuk mengonversi file [DOCX](/id/word-processing/docx/) menjadi [DOC](/id/word-processing/doc/).

## Konversi DOCX ke DOC - Demo Gratis

Anda dapat mengonversi format file DOCX ke DOC menggunakan aplikasi online GRATIS, Aspose.app. Ini adalah sekumpulan aplikasi yang dapat memuat, melihat, dan mengonversi file DOCX ke format file seperti DOC, PDF, TXT, RTF, dan lainnya. Aplikasi Aspose didukung dan dijalankan oleh Aspose API di backend yang dapat melakukan sejumlah operasi pada jenis file berbeda. Coba aplikasi online Aspose.Words sekarang dengan mengklik tombol berikut.

[![DOCX ke DOC](../docx-to-doc.png?width=120px&height=60px)](https://products.aspose.app/words/conversion/docx-to-doc)

### Langkah-Langkah Mengonversi DOCX ke DOC online

Anda dapat mengonversi format file DOCX ke DOC menggunakan [aplikasi Aspose.Words DOCX ke DOC](https://products.aspose.app/words/conversion/docx-to-doc) sebagai berikut.

1. Buka aplikasi Aspose.Words untuk [Konverter DOCX ke DOC](https://products.aspose.app/words/conversion/docx-to-doc)
1. Unggah file DOCX Anda dengan mengklik tombol unggah
1. Setelah selesai, tekan tombol Konversi. Ini akan memulai proses konversi file DOCX ke DOC di latar belakang
1. Anda akan diminta untuk mengunduh file RTF yang telah dikonversi setelah proses konversi DOCX ke DOC selesai

## Aspose.Words API

[Aspose.Words](https://products.aspose.com/words/) adalah sekumpulan API yang memungkinkan pengembang mana pun mengonversi [DOCX](/id/word-processing/docx/) menjadi [DOC](/id/word-processing /doc/) format menggunakan baris kode sederhana. Ini tersedia untuk banyak bahasa pemrograman termasuk [C#](/id/programming/cs/), [Java](/id/programming/java/), [Python](/id/programming/py/) dan [C++](/id/programming/cpp/ ). Selain DOC, Aspose.Words API memungkinkan Anda mengonversi file DOCX ke banyak format file populer lainnya.
