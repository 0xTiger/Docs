## Tentang DOCX hingga TAR

Saat bekerja dengan Microsoft Word, ukuran dokumen mungkin menjadi besar jika isinya terlalu panjang dan menyertakan item seperti gambar dan media lainnya. DOCX to TAR adalah proses mengompresi dokumen word menjadi arsip yang berukuran kecil. Ini membantu dalam berbagi file DOCX berukuran besar dengan orang lain di internet maupun melalui media penyimpanan online seperti Google Drive atau Dropbox. Ada beberapa alat dan aplikasi online yang dapat mengonversi DOCX ke TAR. Aspose.Words untuk konversi adalah salah satu aplikasi yang memungkinkan Anda mengonversi [DOCX](/id/word-processing/docx/) menjadi [TAR](/id/compression/tar/).

## Konversi DOCX ke TAR - DEMO GRATIS

Anda dapat menggunakan aplikasi konversi Aspose.Words GRATIS untuk [mengonversi DOCX ke TAR](https://products.aspose.app/words/conversion/docx-to-tar). Berikut adalah langkah-langkah yang perlu Anda ikuti untuk mengonversi file DOCX Anda ke TAR.

1. Buka [aplikasi konversi Aspose.Words](https://products.aspose.app/words/conversion/docx-to-tar)
1. Unggah file DOCX Anda ke browser
1. Pilih TAR sebagai format file keluaran
1. Tekan tombol Konversi

Ini akan memulai proses konversi file DOCX Anda ke TAR dan opsi unduh akan muncul untuk menyimpan file ke PC Anda.
