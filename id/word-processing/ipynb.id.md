## Apa itu file IPYNB?

File dengan ekstensi .ipynb adalah file Jupyter Notebook yang berisi seluruh konten Notebook yang dibuat dengan sesi aplikasi web Jupyter Notebook. [Jupyter Notebook](https://jupyter.org/) adalah aplikasi web interaktif yang digunakan untuk membuat dan membagikan dokumen komputasi untuk analisis menggunakan Python.

File IPYNB berfungsi sebagai catatan komputasi lengkap dari suatu sesi dan mencakup informasi seperti input dan output komputasi, fungsi matematika, dan representasi yang kaya dari objek yang dihasilkan seperti gambar dan teks penjelasan.

## Format File IPYNB

IPYNB disimpan secara internal dalam format file [JSON](/id/web/json/) yang merupakan format file standar terbuka untuk berbagi data. Untuk alasan ini, file IPYNB dapat dibaca manusia dan mudah dimengerti. JSON juga memudahkan untuk berbagi file ini dengan orang lain.

## Bagaimana cara Mengonversi file IPYNB?

File IPYNB dapat diekspor ke beberapa format file berbeda menggunakan perintah nbconvert dari antarmuka Jupyter. Ini termasuk konversi IPYNB ke:

* PDF
* HTML
* reStructuredText
* Lateks

## Bagaimana cara melihat file IPYNB?

Ada dua metode untuk melihat atau membuka file IPYNB.

1. Pengguna dapat mengunggah dokumen .ipynb notebook ke beberapa URL publik dan membukanya dari Jupyter Notebook Viewer<nbviewer> yang merender notebook sebagai halaman web statis dari URL.
1. Unduh file dan buka di instance Jupyter Notebook mereka sendiri
