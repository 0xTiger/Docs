## Apa itu file ODM?

File dengan ekstensi .odm adalah file dokumen OpenDocument Master yang berisi satu atau lebih file [.odt](/id/word-processing/odt/) yang ditautkan di dalamnya. Semua sub dokumen ini digabungkan dalam dokumen master tunggal. Ini memudahkan untuk mengelola semua file yang ditautkan dari dokumen master tunggal. File dokumen master dapat dibuat dengan Apache OpenOffice Writer atau OpenDocument Word Processor. File ODM dapat diekspor ke satu file ODT menggunakan opsi ekspor dari dalam aplikasi OpenOffice Writer.

## Format File ODM - Informasi Lebih Lanjut

File ODM disimpan sebagai file teks yang dapat dibuka di OpenOffice Writer. File ODM tunggal terdiri dari beberapa bagian yang ditautkan ke subdokumen .odt. Ini memungkinkan banyak penulis untuk mengedit dokumen ODT mereka yang ditautkan dalam file master ODM. Dengan demikian, file master ODM diperbarui karena setiap penulis memperbarui bagiannya masing-masing.

File ODM juga dapat menyertakan daftar isi dan indeks. Setiap kali dokumen Master dibuka, penulis dokumen master meminta untuk memperbarui tautan. Harus mengatakan ya agar konten ditampilkan dengan benar.
