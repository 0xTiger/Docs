## Apa itu file WPD?

File dengan ekstensi .wpd disebut sebagai WordPerfect Document Files (WPD). Corel WordPerfect digunakan untuk menghasilkan dokumen teks ini. Program pengolah kata disebut sebagai **WordPerfect**. Ini digunakan di berbagai sistem komputer, termasuk Windows. WordPerfect juga digunakan dalam lingkungan bisnis, akademik, dan rumah untuk buletin, resume, undangan, dan surat bersama dengan dokumen lainnya.

File WPD berisi data terstruktur, dan dokumen dapat menyimpan berbagai objek yang disematkan seperti grafik dan grafik yang dikembangkan di aplikasi lain. Dokumen WPD juga menyertakan elemen gabungan surat, yang memungkinkan informasi dari entitas eksternal, seperti spreadsheet atau database, untuk dimasukkan ke dalam desain. Anda dapat mengetuk dua kali file WPD dan setiap dokumen lain di PC Anda untuk membukanya. Jika pengaitan dokumen Anda akurat, aplikasi yang dirancang untuk membuka dokumen .wpd Anda akan melakukannya.

## Spesifikasi Format File WPD ##

Dokumen WPD terutama digunakan untuk memproses teks yang disiapkan. Akibatnya, file tersebut menyertakan potongan teks. Tag digunakan untuk memuat informasi pemformatan, yang dikenali WordPerfect sebagai kode. Ini membuat pencarian masalah pemformatan jauh lebih mudah karena aplikasi menyoroti area teks dan memperlihatkan pemformatan yang diwariskan. Dokumen WPD, seperti jenis file pengolah kata yang lebih populer seperti [DOC](/id/word-processing/doc/), dapat menggabungkan data dari berbagai sumber seperti file, database, atau desain.
