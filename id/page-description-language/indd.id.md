## Apa itu file INDD?

File dengan ekstensi .indd adalah file proyek Adobe InDesign yang berisi informasi tata letak halaman proyek. INDD adalah singkatan dari InDesign Document. File-file ini biasanya menyimpan informasi pemformatan halaman, buku, majalah, surat kabar, selebaran, brosur, atau media cetak lainnya seperti maket seluler & halaman web. Jenis file INDD adalah format file milik Adobe dan merupakan format file default untuk InDesign. Desainer Grafis, jurnalis, pemasar, dan penerbit yang bekerja dengan InDesign menggunakan jenis file ini.

## Lebih lanjut tentang Format File INDD

File INDD adalah file vektor yang dapat diskalakan tanpa kehilangan resolusi. Ini didasarkan pada seperti grafik kompleks yang dapat diperluas tanpa batas. Ini mengemas semua informasi font dan tata letak untuk dibagikan dengan tim.

File INDD sempurna untuk merancang cetakan yang menarik serta konten online. File INDD dapat menjangkau beberapa halaman dan dapat berfungsi sebagai templat untuk membuat halaman, bab, dan publikasi. Ketika proyek Anda selesai, itu dapat dipublikasikan ke media online tanpa perlu konversi, zip atau email.

## Bisakah file INDD Dikonversi ke Format Lain?

File INDD dapat dikonversi ke beberapa format file lainnya. Konversi yang paling umum adalah INDD ke PDF. InDesign memungkinkan Anda menyimpan file INDD ke PDF untuk dibagikan dengan pengguna lain. Ini juga memungkinkan Anda mengekspor file INDD ke JPG, [EPS](/id/page-description-language/eps/), [EPUB](/id/ebook/epub/), [SWF](/id/page-description-language/swf/) , FLA, [HTML](/id/web/html/), [XML](/id/web/xml/), dan IDML.

## Bagaimana File INDD Dibuat?

File Indd dibuat saat Anda membuat proyek baru dengan [Adobe InDesign](https://www.adobe.com/products/indesign.html) dan menyimpannya ke disk. Anda dapat membuka file proyek Indd, mengubahnya, dan menyimpan file yang diperbarui untuk perubahan tersebut.
