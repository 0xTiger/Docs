## Apa itu file AVERY?

File AVERY adalah dokumen yang dapat dicetak yang berisi satu atau lebih label untuk tujuan pencetakan. Itu dibuat dengan perangkat lunak Avery Design & Print atau Design & Print Online. Label di dalam file AVERY dapat terdiri dari elemen yang berbeda seperti teks, grafik, dan informasi tata letak halaman. Informasi ini digunakan oleh perusahaan percetakan untuk mencetak berbagai elemen desain seperti kartu nama, tag nama, sisipan pengikat, label alamat, label diska, template obituari, dan kartu ucapan.

## Format File AVERY - Informasi Lebih Lanjut

File AVERY disimpan dalam format file biner. Ini dapat dibuka dengan perangkat lunak Avery Design & Print atau Design & Print Online. Desain yang dibuat dengan Design & Print dapat digunakan untuk membuat label AVERY yang dapat digunakan untuk beberapa tujuan. Desain ini dapat disimpan sebagai file [PDF](/id/pdf/) yang dapat dibagikan dengan perusahaan Percetakan untuk tujuan pencetakan.

### Mencetak File AVERY

File AVERY dapat disimpan sebagai PDF atau dapat langsung dicetak ke printer standar apa pun. Untuk mencetak file, buka file di perangkat lunak Desain & Cetak dan gunakan opsi Cetak dari menu File untuk mengirim file ke printer untuk dicetak.
