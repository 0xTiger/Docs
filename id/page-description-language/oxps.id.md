## Apa itu file OXPS? ##

Format file OXPS dikenal sebagai Spesifikasi Kertas Open XML. Ini adalah bahasa deskripsi halaman dan format dokumen. Microsoft adalah pengembang format ini. Format file OXPS sangat akrab dengan file PDF ini. Pengaturan file OXPS sama dengan format XPS. Satu-satunya perbedaan adalah deskripsi OXPS sebagai format terbuka. Ekstensi file .oxps adalah versi terbaru dari format file XPS yang berasal dari Microsoft dan Ecma International. Jika kita berbicara tentang spesifikasinya, file OXPS dan XPS digunakan untuk menerbitkan, membagikan, dan mengalokasikan dokumen dengan tata letak tetap. Sebagai pengganti PDF, format file OXPS dapat menyimpan teks dan data multimedia, dengan fitur pengaturan dan karakteristik tata letak pencetakan.

## Sejarah Singkat ##

Tim Microsoft memilih Grafik Global untuk memberikan konsultasi pada XPS pada tahun 2003, dan tim pengembangan bekerja sama dengan tim Windows untuk mengembangkan spesifikasi dan referensi arsitektur untuk format baru. Mengakui spesifikasi XPS, pada Juni 2007 Ecma International mendirikan Technical Committee 46 (TC46) untuk mengembangkan standar berdasarkan spesifikasi OpenXPS. Spesifikasi Kertas Open XML yang disetujui Ecma International adalah standar Ecma (ECMA-388) pada Sidang Umum ke-97 yang diadakan di Budapest pada 16 Juni 2009.
