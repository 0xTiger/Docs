## Apa itu file INDT?

File dengan ekstensi .indt adalah file template tata letak Adobe InDesign yang berisi pengaturan yang telah ditentukan sebelumnya. Desain dan isi dari file-file ini meliputi template buletin, buku, majalah, selebaran, dan media cetak. Setelah pengeditan selesai, file INDT dapat disimpan ke disk sebagai file [INDD](/id/page-description-language/indd/). Pengguna juga dapat membuat file INDT baru dari dalam InDesign dan mengonversinya ke format lain seperti [PDF](/id/pdf/), [EPUB](/id/ebook/epub/), atau [JPEG](/id/image/jpeg/) .

## Format File INDT - Informasi Lebih Lanjut

Adobe InDesign menyediakan beberapa template dengan perangkat lunak maupun online. Template ini memberi pengguna titik awal untuk mendesain media online atau cetak mereka. File INDT ringan, memiliki banyak fitur desain (bentuk, teks, warna berbeda, dan filter), dan memiliki resolusi tak terbatas.
