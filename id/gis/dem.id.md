## Apa itu file DEM?

File dengan ekstensi .dem adalah file pemetaan yang merepresentasikan data elevasi (ketinggian) sebagai medan dalam grafik komputer 3D. Contoh umum dari data tersebut adalah medan bumi yang meliputi pegunungan, daratan datar, dan kedalaman laut. File DEM umumnya digunakan oleh Sistem Informasi Geografis (SIG) untuk menghasilkan peta elevasi digital. Informasi yang disimpan dalam file DEM juga mencakup informasi relief topografi berskala besar yang digunakan dalam ilmu bumi, pengelolaan sumber daya, perencanaan lahan, survei, dan proyek teknik. Informasi tentang tanah XYZ ditempatkan pada interval spasi reguler untuk representasi relief tanah.

## Format File DEM - Informasi Lebih Lanjut

Detail internal file DEM tidak tersedia secara publik untuk referensi pengembang. Namun, file DEM dapat memiliki format file yang berbeda seperti:

* Kisi Atribut Batimetrik (TAS)
* DTED
* DIMAP Sentinel 1 basis data ESA
* DEM SDTS
* DEM USG

## Di mana menemukan file DEM?

File DEM umumnya tersedia di internet dan sampel dapat diunduh untuk digunakan. [Survei Geologi Amerika Serikat](https://www.usgs.gov/) (USGS) menyediakan banyak file DEM di situs web mereka.
