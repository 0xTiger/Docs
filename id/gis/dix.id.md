## Apa itu file DIX?

File DIX adalah format file berpemilik yang dibuat dengan program analisis data geografis DIVA-GIS. Ini terdiri dari file proyek DIVA-GIS (.div) dan lapisan data GIS lainnya yang direferensikan oleh proyek tersebut. File DIX dapat diekspor dari dalam DIVA-GIS dan dapat dibagikan dengan pengguna lain yang dapat mengimpornya ke DIVA. Anda juga dapat menggunakan file DIX untuk menyimpan semua file relevan yang terkait dengan suatu proyek di satu lokasi.

## Format File DIX

File DIX disimpan ke disk sebagai file biner terkompresi. Karena kompresi, file DIX tidak memakan banyak ruang disk dan dapat dikirim ke pengguna lain melalui email. File DIX dapat berisi data raster dan vektor, dan juga dapat menyimpan tabel atribut, metadata, dan data lain yang terkait dengan data spasial.
