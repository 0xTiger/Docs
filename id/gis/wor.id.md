## Apa itu file WOR?

File WOR adalah file GIS yang digunakan oleh perangkat lunak pemetaan MapInfo dan analisis geografis. Berisi wilayah geografis yang berisi peta visual yang memiliki data tentang fitur geospasial seperti informasi populasi dan demografi. Anda dapat memuat file WOR di perangkat lunak MapInfo untuk melihatnya dan melakukan berbagai operasi seperti memperbesar dan menggeser.

## Format File WOR

File WOR terdiri dari beberapa lapisan yang dirender dalam urutan tertentu saat dimuat di MapInfo. Detail tentang format file WOR tidak tersedia untuk umum untuk referensi pengembang. File WOR dapat dikonversi ke QGIS dengan [QGIS 3.20 Odense](https://www.qgis.org/en/site/forusers/visualchangelog320/index.html#feature-new-embedded-styling-renderer-with-ogr-feature-styles-support).
