## Apa itu berkas SP3?

File dengan ekstensi .sp3 adalah format orbit National Geodetic Survey (NGS) untuk menyimpan informasi orbital. Ini adalah perluasan dari format SP1 dan SP2 NGS, dan menyertakan informasi koreksi jam satelit yang dihitung secara bersamaan dengan orbit. Ini terutama didasarkan pada posisi dan jam, dan tidak termasuk kecepatan. Formatnya diusulkan dalam Remondi, 1989 dan diadopsi pada tahun 1991. Selain informasi koreksi jam satelit, ini mencakup informasi seperti eksponen akurasi orbit, baris komentar, minggu GPS, dan detik dalam seminggu yang terkait dengan zaman pertama. File SP3 dapat dibuka dengan Wolfram Research Mathematica.

## Format File SP3 - Informasi Lebih Lanjut

File SP3 disimpan ke disk dalam format file ASCII dan isinya dapat dibuka di editor teks apa pun untuk dilihat. Struktur file SP3 dapat dilihat pada gambar berikut.
