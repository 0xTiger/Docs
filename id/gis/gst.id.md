## Apa itu file GST?

File GST adalah format file peta GIS yang berisi data geografis dan peta visual suatu wilayah yang dipetakan. Ini digunakan oleh **MapX** (dikembangkan oleh UNEP/GRID-Geneva dan UNEP untuk mengelola data geospasial sumber daya alam) dan perangkat lunak pemetaan **MapXtreme** (perangkat pengembangan perangkat lunak yang digunakan untuk membangun aplikasi GIS khusus) yang platform online untuk mengelola data geospasial sumber daya alam. File GST berisi informasi geografis dalam bentuk gambar numerik dan raster yang dihamparkan pada peta.

## Format File GST - Informasi Lebih Lanjut

File GST disimpan dalam format file kepemilikan dan detail format file internalnya tidak tersedia untuk umum untuk referensi pengembang.
