## Apa itu file WLX?

File WLX adalah file GIS yang dibuat oleh perangkat lunak MapSource yang disertakan dengan produk peta Garmin. Berisi lokasi Geografis (dalam bentuk lintang-bujur) dalam format XML dan lokasi dunia ke situs pemetaan seperti Google Maps atau Bing Maps. File WLX bertindak sebagai tautan pintasan dan membuka lokasi yang disimpan di situs web pihak ketiga pada posisi lintang-bujur yang tercatat.

File serupa lainnya (walaupun tidak terkait dengan GIS) adalah format file [URL](/web/url/) yang menyimpan alamat web dan mengarahkan pengguna ke URL ini saat dibuka.

## Format File WLX - Informasi Lebih Lanjut

File WLX disimpan dalam format file teks yang dapat dibuka di editor teks apa pun untuk melihat dan memperbarui isinya. MapSource adalah perangkat lunak berbasis GIS yang memungkinkan Anda melihat peta, titik arah, rute, dan trek. Anda juga dapat mentransfer data spasial ini ke atau dari perangkat GPS Garmin.

MapSource tidak lagi didukung oleh Garmin dan telah digantikan oleh BaseCamp yang dapat diunduh secara gratis.
