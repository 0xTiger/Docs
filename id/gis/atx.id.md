## Apa itu file ATX?

File dengan ekstensi .atx adalah file ESRI ArcGIS yang menyimpan indeks atribut ArcGIS 8. Ada file ATX terpisah untuk setiap [shapefile](/id/gis/shp/) atau indeks atribut dBase yang dibuat di ArcCatalog. Perlu dicatat bahwa indeks atribut ArcView GIS 3.x untuk file shapefile dan dBase tidak digunakan oleh ArcMap atau ArcCatalog. Namun, indeks atribut ArcView GIS 3.x ini dapat dikonversi menjadi indeks atribut ArcGIS 8 menggunakan alat ArcObjects.

## Format File ATX - Informasi Lebih Lanjut

Tidak ada spesifikasi format file yang tersedia untuk format file ATX untuk memudahkan pengembang.
