## Apa itu file MXT?

File MXT adalah templat peta geografis yang dibuat dengan aplikasi pemetaan GIS, ArcMap yang hadir sebagai bagian dari ESRI ArcGIS Desktop. Ini digunakan untuk menyimpan lapisan peta dasar atau tata letak halaman yang dapat digunakan kembali untuk membuat lebih banyak peta. Menggunakan templat menghemat waktu Anda karena Anda tidak perlu mereproduksi bagian-bagian umum peta. File MXT dapat dibagikan dengan pengguna/organisasi lain untuk membuat peta lain dengan cepat.

## Format Berkas MXT

File MXT disimpan sebagai file biner tetapi detail struktur file internalnya tidak tersedia untuk umum untuk referensi pengembang. Ini berisi informasi seperti properti dasar peta, posisi tata letak, dan visibilitas toolbar aplikasi. File MXT memungkinkan pengguna membuat banyak peta dengan pengaturan yang telah ditentukan sebelumnya sudah tersedia bagi mereka.
