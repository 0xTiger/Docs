## Apa itu file DT2?

File DT2 adalah file ketinggian yang mewakili file Digital Terrain Elevation Data (DTED). Ini berisi data ketinggian dalam pola spasi seragam. Data ketinggian dapat dibaca oleh program perangkat lunak GIS dan dapat diterapkan di berbagai bidang seperti militer, ilmu pengetahuan, dan komunitas lainnya.

Aplikasi GIS yang dapat membuka file DT2 antara lain ESRI ArcGIS Pro, TatukGIS, Blue Marble Geographic Global Mapper, dan Pitney Bowes MapInfo. Varian lain dari jenis file DTED adalah [DT0](/gis/dt0/) dan [DT1](/gis/dt1/).

### Format Berkas DT2

File DT2 disimpan sebagai file biner ke disk dan dapat dibaca menggunakan banyak aplikasi GIS. Ini dapat digabungkan ke dalam File Geodatabase Raster Catalog yang dapat digunakan untuk membuat satu mosaik besar.
