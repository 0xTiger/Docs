## Apa itu file GLB?

File dengan ekstensi .glb adalah file yang dapat dibagikan secara global yang dibuat dan digunakan oleh perangkat lunak AGI Systems Tool Kit (STK). Ini berisi informasi globe 3D yang dapat dibagikan dengan pengguna lain untuk digunakan dalam proyek mereka. Perangkat lunak STK sistem AGI digunakan untuk mensimulasikan misi kedirgantaraan, sistem pertahanan, dan sistem telekomunikasi. GLB adalah format file standar untuk STK dan berbagai model 3D seperti [.gltf](/id/3d/gltf/) dapat dikonversi ke format ini. File GLB dapat dibuka dengan perangkat lunak AGI Simulation Toolkit serta editor teks apa pun.

## Format File GLB - Informasi Lebih Lanjut

File GLB menyimpan informasi tentang tekstur, gambar, dan file lain yang digunakan di globe untuk suatu proyek. Referensi ke lokasi file tertaut disimpan dalam file GLB dan file tambahan ini juga harus dibagikan dengan pengguna lain saat berbagi file globe GLB untuk memuat dengan benar. File GLB menyimpan semua informasi ini dalam file teks biasa yang dapat dibuka dengan editor teks apa pun.
