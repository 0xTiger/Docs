## Apa itu file DT0?

File DT0 adalah file Digital Terrain Elevation Data (DTED) yang berisi informasi Level 0 untuk mempelajari data ketinggian medan. Data ketinggian ditempatkan secara seragam dalam file DT0 dan dapat dibaca oleh program GIS populer seperti ESRI ArcGIS Pro, TatukGIS, Blue Marble Geographic Global Mapper, dan Pitney Bowes MapInfo. File DT0 sebagian besar digunakan oleh komunitas ilmiah dan teknis untuk mempelajari kemiringan, ketinggian, dan kekasaran permukaan medan global.

Format file DT0 mirip dengan jenis file [DT1](/gis/dt1/) dan [DT2](/gis/dt2/) tetapi dengan rincian yang lebih sedikit.

### Format Berkas DT0

File DT0 disimpan sebagai file biner ke disk. Ini dapat dibaca menggunakan banyak aplikasi GIS. File DT0 dapat digabungkan menjadi Katalog Raster File Geodatabase yang dapat digunakan untuk membuat satu mosaik besar.

## Bagaimana Mengonversi file Raster ke DTED?

Ada beberapa alat yang dapat mengonversi file Raster ke DTED. [ArgGIS Pro](https://pro.arcgis.com/en/pro-app/latest/tool-reference/data-management/raster-to-dted.htm) adalah salah satu alat yang dapat mengonversi Raster ke DTED.
