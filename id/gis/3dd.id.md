## Apa itu file 3DD?

File dengan ekstensi .3dd adalah file data GIS yang dibuat dengan perangkat lunak aplikasi ESRI ArcGlobe. Ini menyimpan informasi ke jalur di mana data 3D berada dalam bentuk geodatabase. Saat dokumen dibuka menggunakan file .3dd, dokumen akan muncul dalam bentuk yang sama seperti saat disimpan. ArcGlobe adalah aplikasi visualisasi 3D yang memungkinkan Anda melihat data GIS dalam jumlah besar di permukaan globe. ArcGlobe adalah salah satu dari beberapa aplikasi yang digabungkan dengan paket perangkat lunak ArcGIS.

## Format File 3DD - Informasi Lebih Lanjut

File 3DD disimpan ke disk dalam format file biner dan format file internal dari file ini tidak tersedia untuk umum.
