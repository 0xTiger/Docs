## Apa itu file APR?

File dengan ekstensi .apr adalah file proyek ESRI ArcView yang berisi informasi proyek untuk proyek ESRI. Ini digunakan oleh ArcView 3.x dan tidak didukung oleh ArcGIS versi baru (9.x dan 10.x). Namun, ArcView 10 dapat mengimpor file APR. ArcView adalah Sistem Informasi Geografis (SIG) yang digunakan untuk membuat, memvisualisasikan, menganalisis, dan mengelola data geografis. Ada format file lain yang digunakan oleh ESRI untuk menyimpan data seperti ESRI ShapeFile yang populer ([.shp](/id/gis/shp/)). File APR dapat dibuka dengan editor teks apa pun seperti Microsoft Notepad dan Notepad ++.

## Format File APR - Informasi Lebih Lanjut

File APR disimpan ke disk sebagai file teks biasa yang dapat dibuka dengan editor teks apa pun untuk diperiksa dan diedit.
