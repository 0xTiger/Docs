## Apa itu file LAN?

File dengan ekstensi .lan adalah file raster GIS yang digunakan oleh aplikasi perangkat lunak ERDAS versi lama. Ini juga digunakan oleh aplikasi GIS lainnya dan berfungsi untuk menyimpan dan menampilkan citra satelit dan udara. Selain itu juga dapat menyimpan jenis data raster lainnya. File LAN ERDAS juga dapat berisi informasi tambahan khusus untuk ERDAS seperti referensi geografis dan informasi lapisan.

Anda dapat membuka file LAN dengan ERDAS IMAGINE dan ArcGIS Desktop.

## Format Berkas LAN

File LAN disimpan ke disk dalam format file biner dan tidak dapat dibaca manusia. Format file LAN banyak digunakan dalam aplikasi penginderaan jauh dan GIS. Namun, format file tersebut digantikan oleh format file ERDAS IMAGINE ([.IMG](/id/gis/img/)).
