## Apa itu file GSM?

File GSM adalah file proyek yang dibuat dengan perangkat lunak MapViewer yang digunakan untuk analisis spasial dan membuat peta geografis. Ini adalah file pengaturan peta yang menyimpan informasi konfigurasi terkait pengaturan visualisasi peta seperti label teks, warna, legenda, dan file data referensi.

## Format Berkas GSM

File GSM disimpan dalam format file biner dan rincian struktur file internalnya tidak tersedia untuk umum untuk referensi pengembang. Perangkat lunak MapViewer dilengkapi dengan contoh file GSM dan beberapa file [DAT](/database/dat/) yang digunakan untuk membuat file GSM lebih lanjut.

### Tentang Penampil Peta

MapViewer memungkinkan Anda bekerja dengan file peta dengan melapisi data Anda pada wilayah geografis. Hal ini membantu dalam memvisualisasikan data spasial seperti peta kepadatan penduduk, peta iklim, dan beberapa jenis data lainnya.
