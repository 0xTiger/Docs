## Apa itu file IMG?

File dengan ekstensi .img adalah File Peta Garmin yang digunakan oleh perangkat GPS Garmin. Ini berisi data peta/geospasial untuk mewakili entitas peta seperti nama jalan, rute navigasi, dan tempat menarik. Pengguna dapat membuat data peta yang disesuaikan, menyimpannya sebagai file .img dan memuatnya ke perangkat Garmin. Garmin juga menyediakan data peta pra-instal yang disimpan di perangkat Garmin dalam format file IMG. File IMG dapat dibuka menggunakan aplikasi seperti Garmin Mobile PC, cGPSmapper, MapUpload dan GPS Map Manager.

## Format File IMG - Informasi Lebih Lanjut

File IMG disimpan sebagai file biner dalam format file milik Garmin. Informasi yang tersedia dalam file IMG dapat ditampilkan di unit GPS Garmin. Beberapa unit GPS yang mendukung perutean otomatis juga dapat menggunakan file peta ini untuk panduan selama navigasi seperti di mobil, di sepeda, atau saat mendaki.

File GMAP serupa dengan gambar hard disk dan didasarkan pada sistem file yang terdiri dari subfile untuk menyediakan fungsionalitas unit GPS Garmin. Subfile .GMP menyediakan semacam struktur subdirektori meskipun format file .gmap datar.
