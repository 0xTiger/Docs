## Apa itu file MXD?

File MXD adalah file Dokumen Peta yang digunakan oleh perangkat lunak ESRI ArcMap. Ini adalah format file berpemilik yang menyimpan berbagai jenis informasi seperti tata letak dan desain peta, data yang digunakan untuk menghasilkan peta, dan simbologi yang diterapkan pada data. Dengan demikian, file MXD adalah format file gabungan yang berisi beberapa jenis elemen peta berbeda dalam satu file. File MXD digunakan oleh file ArcGIS Map Package ([.mpk](/gis/mpk/)).

## Format Berkas MXD

File MXD disimpan dalam format file milik ArcGIS dan detail internalnya tidak tersedia untuk referensi pengembang. Elemen yang disematkan pada file MXD disusun menjadi beberapa unit, yang dikenal sebagai penyimpanan dan aliran.
