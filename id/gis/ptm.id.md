## Apa itu file PTM?

File PTM adalah file peta yang digunakan oleh perangkat lunak pemetaan MapPoint yang dikembangkan oleh Microsoft yang sekarang sudah dihentikan. Ini berisi data peta yang diekstraksi dari Server Terra Microsoft. Informasi peta berisi data terkait rute dan wilayah perjalanan bisnis. MapPoint kemudian digantikan oleh peta Bing. File lain yang digunakan bersama dengan file PTM termasuk MPP (MapPoint Map), WPT (MapPoint Waypoint), dan RTE (MapPoint Route).

## Format Berkas PTM

Format file PTM adalah format file berpemilik dan struktur file internal file .ptm tidak didokumentasikan secara publik.
