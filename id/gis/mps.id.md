## Apa itu file MPS?

File MPS adalah file peta yang digunakan pada perangkat genggam Pocket PC portabel. Ini berisi informasi Geografis tentang area tertentu seperti kota atau negara bagian. Itu dapat dimuat di perangkat Pocket PC untuk navigasi tanpa perlu mengambil data LANGSUNG dari server data geospasial mana pun. Format file MPS dikembangkan oleh Microsoft untuk perangkat genggam berbasis Windows dan digunakan dalam aplikasi GIS MapPoint yang telah dihentikan sejak lama.

## Format File MPS

Struktur file internal format file MPS tidak diketahui karena telah dihentikan untuk beberapa waktu sekarang dengan MapPoint. Beberapa format file peta GIS populer lainnya mencakup [ESRI Shape Files](/id/gis/shp/), [KML](/id/gis/kml/), dan [KMZ](/id/gis/kmz/).
