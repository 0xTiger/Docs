## Apa itu file 3DC?

File 3DC adalah file data GPS yang digunakan oleh aplikasi perangkat lunak navigasi GPS, [iGO](https://en.wikipedia.org/wiki/IGO_(software)). Berisi informasi data bangunan 3D beserta metadata untuk tampilan aplikasi. File 3DC disimpan ke kartu memori unit navigasi dan sebagian besar disimpan di direktori \​iGO\​content\​building\​.

## Tentang Navigasi iGO

Perangkat lunak iGO digunakan untuk tujuan navigasi. Ia mendapat informasi lokasi pengguna menggunakan perangkat GPS dan membantu navigasi menggunakan peta dan rute 3D. Ini tersedia untuk Android, iOS, dan perangkat genggam populer lainnya. Selain fitur navigasinya, perangkat lunak iGO juga menyediakan penyimpanan peta offline jika pengguna kesulitan terhubung ke internet untuk data peta real-time.
