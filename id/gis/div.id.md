## Apa itu file DIV?

File dengan ekstensi .div adalah file proyek yang dibuat dengan perangkat lunak [DIVA-GIS](https://en.wikipedia.org/wiki/DIVA-GIS) saat proyek pemetaan baru dibuat dan disimpan. DIVA-GIS adalah perangkat lunak pemetaan yang digunakan untuk membuat peta spasial dan aplikasi analisis data geografis. File proyek terdiri dari gambar peta dan lapisan yang digunakan untuk menampilkan objek dan properti di peta. Selain informasi spasial, ia juga menyimpan informasi tentang skala peta dan pusatnya.

## Format File DIV - Informasi Lebih Lanjut

File DIV disimpan pada disk sebagai format file milik DIVA-GIS dan format file internalnya tidak tersedia untuk referensi pengembang.
