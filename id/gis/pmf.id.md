## Apa itu file PMF?

File PMF adalah file peta read-only yang dihasilkan dengan ArcMap. Bisa dibilang ini adalah versi pendek dari format file MXD yang memiliki informasi terbatas untuk dibagikan kepada pengguna yang tidak memiliki perangkat lunak ArcGIS versi berbayar. Hal ini memungkinkan berbagi informasi geografis dengan orang lain pada saat yang bersamaan. File PMF dapat dibagikan dengan pengguna lain melalui email karena ukurannya yang kecil akibat terbatasnya informasi yang dikandungnya.

File PMF dapat dibuka dengan ESRI ArcReader dan ESRI ArcGIS Pro.

## Format Berkas PMF

File PMF membatasi tingkat data yang dapat diakses oleh pengguna berdasarkan langganan mereka. Jika pengguna menggunakan perangkat lunak ArcGIS versi gratis, file PMF akan membatasi tampilan informasi seperti tampilan data, tata letak, navigasi, pencetakan, kueri, panning, dan kemampuan zoom peta. Data mungkin sebenarnya tidak terkandung dalam file batas, namun diakses dari lokasi jarak jauh yang dihosting di server ArcMap atau tersedia melalui internet.
