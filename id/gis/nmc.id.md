## Apa itu file NMC?

File NMC adalah file konten paket peta yang dibuat dengan perangkat lunak ESRI ArcGIS Explorer. Ia dapat menyimpan berbagai elemen peta seperti hamparan gambar, lapisan layanan GIS, lapisan [KML](/gis/kml/), lapisan raster, fitur, dan paket lainnya. File NCM digunakan untuk mengekspor data peta yang dapat digunakan di peta GIS lain yang dibuka di ArcGIS Explorer. Hal ini memungkinkan beberapa peta ditumpangkan satu sama lain dan ditampilkan.

## Format File NMC

File NMC disimpan dalam format file biner, dengan struktur internalnya tidak tersedia. Ini dapat dibagikan dengan orang lain yang dapat memuat file NMC ini di ArcGIS Explorer pada akhirnya. Hal ini membuat file NMC dapat digunakan di komputer pengguna lain dengan mengirimkannya melalui email ke orang lain.
