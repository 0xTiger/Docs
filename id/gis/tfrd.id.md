## Apa itu file TFRD?

File TFRD adalah format data pencitraan yang digunakan oleh Badan Intelijen Geospasial Nasional (NGA) untuk menyimpan data geospasial. NGA menyediakan kecerdasan geospasial yang memberikan keuntungan dalam pengambilan keputusan. File TFRD dapat menyimpan data gambar dan tekstual dan dapat digunakan oleh aplikasi GIS untuk merender data geospasial.

## Format File TFRD

Detail format file internal dari format file TFRD tidak diketahui. Namun, file tersebut disimpan sebagai file terkompresi ke disk dan rasio kompresinya lebih besar daripada file format file NITF yang setara.
