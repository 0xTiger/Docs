## Apa itu file AX?

File AX adalah format file GIS yang dibuat dengan aplikasi perutean Microsoft AutoRoute. Ini berisi informasi lokasi Tempat Menarik (POI) di peta dan data navigasi lainnya. POI ini mudah dinavigasi dan dapat ditambahkan ke perjalanan sebagai bagian dari rute. AutoRoute dikembangkan untuk memberikan informasi rute untuk menavigasi jalan raya, jalan raya melalui Eropa.

File AX dapat dibuka dengan Microsoft AutoRoute tetapi sudah lama dihentikan dan file AX tidak lagi didukung.

## Format File AX - Informasi Lebih Lanjut

File AX disimpan ke disk dalam format file biner. Itu disimpan dalam format file kepemilikan dan rincian struktur file internalnya tidak tersedia untuk umum untuk referensi pengembang.
