## Apa itu file SDF?

A Spatial Data File (SDF) is a Geodatabase file format that was developed by Autodesk. It stores geospatial data and is used by Autodesk GIS program MapGuide and AutoCAD Map 3D. SDF file format is based on the single file database file format SQLite3. Ini dianggap sebagai format file yang dioptimalkan untuk kumpulan data informasi spasial yang besar.

Anda dapat membuka file SDF menggunakan Autodesk AutoCAD Map 3D 2022 dan Autodesk AutoCAD Civil 3D 2023.

## Format File SDF - Informasi Lebih Lanjut

Format file SDF disimpan ke disk sebagai file biner. Hal ini didasarkan pada database SQLite yang menggunakan komponen penyimpanan tingkat rendah untuk serialisasi data biner. File SDF berisi beberapa kelas fitur per file dan beberapa properti geometri untuk setiap kelas fitur. Data dalam file SDF dapat diakses dengan kecepatan tinggi karena pengindeksan setiap properti geometri menggunakan R-tree.
