## Apa itu file MSD?

File MSD adalah file Distribusi Layanan Peta yang dibuat dengan perangkat lunak pemetaan GIS ArcGIS Desktop. Ini berisi peta terbitan yang diekspor dari file peta [.mxd](/id/gis/mxd/). File MSD berisi informasi tentang layanan peta yang dihosting di Server ArcGIS. Peta yang dihosting dapat diakses melalui internet oleh orang lain. File MSD memiliki dokumen peta, informasi lapisan dan properti tabel.

## Format Berkas MSD

File MSD disimpan ke disk dalam format file biner. Ini digunakan oleh aplikasi server ArcGIS untuk membuat dan mengelola layanan peta untuk tujuan hosting. File MSD juga digunakan oleh ArcMap untuk terhubung ke layanan peta dan menampilkan konten di peta.
