## Apa itu file DT1?

File DT1 adalah file Digital Terrain Elevation Data (DTED) yang berisi data medan dengan jarak yang seragam. Hal ini terutama digunakan oleh aplikasi perangkat lunak GIS militer untuk mendapatkan data ketinggian, kemiringan, dan kontur permukaan. File DT1 mirip dengan [DT0](/gis/dt0/) dan [DT2](/gis/dt2/) tetapi dengan rincian informasi sedang.

File DT1 dapat dibuka dengan aplikasi GIS seperti ESRI ArcGIS Pro, TatukGIS, Blue Marble Geographic Global Mapper, dan Pitney Bowes MapInfo.

### Format Berkas DT1

File DT1 disimpan sebagai file biner ke disk dan dapat dibaca menggunakan banyak aplikasi GIS. Ini dapat digabungkan menjadi Katalog Raster File Geodatabase yang dapat digunakan untuk membuat satu mosaik besar.
