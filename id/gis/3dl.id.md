## Apa itu file 3DL?

File 3DL adalah file iGO Landmark yang digunakan oleh perangkat lunak navigasi iGO. Ini menyimpan informasi lokasi untuk tempat menarik (POI) dan landmark. Informasi ini dapat mencakup nama, lokasi, dan detail lainnya tentang POI atau landmark. Format file iGO Landmark adalah hak milik dan hanya digunakan oleh [iGO navigation software](https://en.wikipedia.org/wiki/IGO_(software)). File 3DL dapat digunakan bersama dengan file data peta untuk memberikan panduan navigasi kepada pengguna.

## Format Berkas 3DL

Format file 3DL adalah hak milik dan tidak didokumentasikan secara publik. Perangkat lunak navigasi iGO yang menggunakan format file 3DL dikembangkan dan didistribusikan oleh perusahaan bernama NNG (sebelumnya Nav N Go). Oleh karena itu, Anda mungkin tidak dapat menemukan informasi teknis mendetail tentang format tersebut di [NNG website](https://www.nng.com/) atau di tempat lain di internet.
