## Apa itu file QML?

File dengan ekstensi .qml adalah file XML yang menyimpan informasi gaya layer untuk layer QGIS. QGIS adalah aplikasi GIS lintas platform sumber terbuka yang digunakan untuk menampilkan data geospasial dengan kemampuan mengatur data dalam bentuk lapisan. File QML berisi informasi yang digunakan oleh QGIS untuk membuat geometri fitur termasuk definisi simbol, ukuran dan rotasi, pelabelan, opasitas, mode campuran, dan banyak lagi. Berbeda dengan file QLR, file QML berisi semua informasi gaya itu sendiri. File QML dapat dibuka di editor teks apa pun seperti Notepad ++.

## Format File QML

QLR, mirip dengan [QGS](/id/gis/qgs/) dan [QLR](/id/gis/qlr/), adalah file XML yang berisi informasi gaya untuk layer.

Gambar di bawah menunjukkan tag tingkat atas dari file QML (dengan hanya renderer_v2 dan tag simbolnya diperluas).
