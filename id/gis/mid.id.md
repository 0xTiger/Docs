## Apa itu file MID?

File MID dalam kategori file GIS adalah file data GIS yang dihasilkan dengan perangkat lunak analisis dan visualisasi data spasial MapInfo. Biasanya dibuat bersama dengan file MapInfo Interchange Format ([MIF](/gis/mif/)) yang sesuai dan memiliki nama yang sama. File MID menyimpan kumpulan data besar yang berisi informasi atribut untuk objek di peta dan diimpor ke atribut objek.

## Format File MID - Informasi Lebih Lanjut

File MID disimpan ke disk dalam format file biner. Atribut dalam file MID disimpan dalam urutan yang sama dengan file MIF terkait. Misalnya, entitas geometris ketiga dalam file .mif memiliki atribut yang disimpan di baris ketiga file .mid.
