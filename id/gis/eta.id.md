## Apa itu berkas ETA?

File dengan ekstensi .eta adalah file Tanda Letak Google Earth yang dibuat dengan perangkat lunak Google Earth. Ini berisi informasi bookmark/penanda untuk lokasi di dunia menggunakan parameter seperti lintang, bujur, rotasi, zoom, dan judul. Hal ini memungkinkan pengguna untuk memiliki pandangan mata burung dari lokasi. Dengan diperkenalkannya format file baru yaitu [.kml](/id/gis/kml/) dan [.kmz](/id/gis/kmz/), format file ETA dihapus dan diganti dengan format file baru ini. File ETA dapat dibuka di perangkat lunak [Google Earth](https://earth.google.com/web/) yang tersedia untuk diunduh secara gratis.


## Format File ETA - Informasi Lebih Lanjut

Detail internal format file ETA tidak tersedia untuk umum dan tidak ada perangkat lunak sumber terbuka atau berpemilik yang tersedia untuk membaca informasi file ETA.
