## Apa itu file 4DB?

Struktur basis data dibangun dengan menggunakan 4th Dimension, program desain perangkat lunak yang terdiri dari kode 4D, antarmuka, dan informasi sumber daya yang dikenal sebagai 4DB. Ini digunakan dalam kerangka desain perangkat lunak integratif 4D hanya selama proses pengembangan. File 4DB ditemukan di platform desktop dan seluler dan sistem operasi Windows juga merupakan opsi. Peringkat popularitas file 4DB utama adalah "Rendah" yang berarti file 4DB biasanya ada di perangkat seluler atau desktop biasa.

File struktur 4DB memiliki semua spesifikasi yang terkait dengan struktur database seperti properti bidang, bidang, dan tabel, daftar pilihan, kata sandi, menu, metode, dan formulir. File mendapatkan nama yang Anda masukkan di kotak dialog pembuatan database lalu diikuti dengan ".4db".

Ada dua cara untuk menghubungkan dokumen dengan aplikasi baru (misalnya, my-file.4DB). Metode pertama dan paling mudah adalah mengklik kanan file 4DB yang dimaksud. Pilih "Pilih program default" dari menu musim gugur, lalu klik "Jelajahi" untuk mencari perangkat lunak yang Anda inginkan. Dengan mengklik OK, Anda harus mengonfirmasi seluruh operasi. Langkah selanjutnya dan lebih kompleks adalah menggunakan Windows Registry untuk menautkan tipe data 4DB ke perangkat lunak yang sesuai.

## Sejarah Singkat ##

Apple mendukung perusahaan baru dan memanfaatkan 4D secara luas di seluruh perusahaan untuk proyek-proyek seperti administrasi fasilitas kebugaran dan CIM (Computer Integrated Manufacturing). Lance McAndrew di kantor Apple di California dan Andrew O'Donoghue di kantor pusat produksi Apple di Eropa Skotlandia, tempat program Dimensi ke-4 mengawasi seluruh manajemen Departemen Layanan Eropa, termasuk di antara karyawan Apple yang menjadi pakar 4D.
