## Apa itu file ADF?

File dengan ekstensi .adf adalah file database yang dibuat dengan perangkat lunak ACT CRM (Customer Relationship Management) untuk menyimpan data hubungan tentang pelanggan. Ini menyimpan informasi tentang nama pelanggan, email, kontak, informasi bisnis, dan riwayat hubungan. Sistem CRM lebih sering diintegrasikan dengan call center yang didirikan oleh bisnis untuk secara otomatis mengambil informasi pengguna untuk panggilan masuk. File ADF dapat dibuka dengan perangkat lunak SwiftPage ACT di OS Windows.

## Format File ADF - Informasi Lebih Lanjut

File ADF disimpan ke disk sebagai file biner dan informasi struktur internalnya tidak tersedia untuk umum. File lain dengan ekstensi .alf disimpan bersama dengan file ADF yang berisi metadata yang diperlukan untuk menavigasi, mencadangkan, dan memulihkan ACT! basis data.
