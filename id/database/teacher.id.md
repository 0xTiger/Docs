## Apa itu file GURU?

File .TEACHER adalah file pengaturan pendidikan yang dibuat oleh perangkat lunak SMART Response. Ini digunakan dalam pengaturan pendidikan untuk menyediakan lingkungan lengkap dalam mengelola informasi kelas, daftar kelas, jadwal, hari libur, profil siswa, rutinitas ujian, dan hasil penilaian. Perangkat lunak penilaian SMART Responseâ„¢ diinstal pada komputer yang akan digunakan untuk melaksanakan penilaian SMART Response.

## Format File GURU

File GURU disimpan ke disk dalam format file biner. Spesifikasi format file internalnya tidak tersedia untuk umum untuk referensi pengembang. Perangkat lunak SMART Response dapat digunakan untuk menghasilkan laporan siswa. Ini kemudian dapat dibagikan antar komputer sekolah.
