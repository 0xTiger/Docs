Basis data adalah kumpulan data yang diatur dalam tabel dan dipelihara di perangkat penyimpanan komputer dan diakses secara elektronik. Database relasional menyimpan data dalam tabel database yang dihubungkan satu sama lain melalui desain formal dan teknik pemodelan. Data dalam basis data dianalisis menggunakan Sistem Manajemen Basis Data (DBMS) yang merupakan sumber utama interaksi dengan pengguna akhir, aplikasi, dan basis data itu sendiri.

Ekstensi file database populer dan format filenya mencakup **SQLite**, **DB**, **ACCDB**, dan **MDB**.
