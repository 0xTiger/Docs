## Apa itu file CAT?

File CAT milik perangkat lunak Katalog Disk Tingkat Lanjut, yang dikembangkan oleh ElcomSoft Co. Ltd, perusahaan perangkat lunak Rusia yang mengembangkan berbagai perangkat lunak keamanan komputer dan forensik. Beberapa produknya mencakup perangkat lunak untuk pemulihan kata sandi, akuisisi perangkat seluler, dan ekstraksi cloud. Perusahaan ini didirikan pada tahun 1990 dan memiliki kantor di Rusia dan Amerika Serikat. Mereka terkenal dengan perangkat lunak forensiknya dan secara keseluruhan perangkat lunak mereka digunakan oleh perusahaan swasta dan lembaga pemerintah untuk melakukan analisis dan investigasi forensik.

Advanced Disk Catalog is a cataloguing program for Windows XP/2003. Fleksibilitas ADC berasal dari fakta bahwa ADC tidak bergantung pada mesin database, sehingga sangat efisien dan kompak. Antarmuka penggunanya, mirip dengan Windows Explorer, memungkinkan pengorganisasian data yang disimpan dengan mudah di berbagai jenis media seperti hard drive (termasuk drive jaringan), floppy disk, disk optik, disk ZIP dan JAZ, dll.

## Bagaimana cara membuka file CAT?

Unduh perangkat lunak Katalog Disk Lanjutan Elcomsoft. Di Windows, Anda dapat mengklik kanan pada file explorer lalu pilih buka dengan menu konteks, tentukan jalur perangkat lunak Elcomsoft Advanced Disk Catalog untuk membuka file cat.
