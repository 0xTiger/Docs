## Apa itu file ABX?

File .abx adalah buku alamat yang diekspor dari perangkat lunak WordPerfect Office yang dikembangkan oleh **Corel**. WordPerfect adalah perangkat lunak pengolah kata yang dikembangkan oleh Corel Corporation dan telah digunakan cukup lama. File ABX berisi informasi tentang informasi kontak yang diekstraksi dari klien email WordPerfect. Informasi kontak yang diekspor dapat ditransfer ke perangkat lain dan digunakan sebagaimana mestinya.

## Format File ABX - Informasi Lebih Lanjut

File ABX disimpan dalam format file miliknya sendiri dan detail format file internal file ABX tidak diketahui. Email WordPerfect kemudian diintegrasikan ke dalam WordPerfect Office Suite.
