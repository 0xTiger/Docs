## Apa itu File DB3?
File DB3 adalah file database yang dibuat oleh perangkat lunak SQLite yang merupakan program database mandiri yang ringan yang membuat database menggunakan file biasa; berisi anatomi database serta catatan data; biasa digunakan untuk mengambil atau menyimpan data terstruktur menggunakan SQL. File-file ini dapat digunakan di perangkat pintar atau di mana pencatatan atau manajemen data lainnya diperlukan tetapi di lingkungan ruang yang rendah.


## Format File DB3
Format file DB3 dikaitkan dengan RDBMS (Sistem Manajemen Basis Data Relasional) SQLite, pilihan populer untuk basis data tersemat. Tidak ada ekstensi file khusus yang ditentukan dalam SQLite_3 dalam spesifikasinya, mungkin menggunakan ekstensi termasuk [.dbf](/id/database/dbf/) dan [.sqlite](/id/database/sqlite/).

### Sesifikasi Basis Data
Biasanya SQLite, format file db terkait versi 3 dapat dianggap sebagai format file DB3 yang digunakan sebagai format asli yang didokumentasikan secara publik untuk mesin database SQLite sejak Juni 2004. Format file DB3, adalah lintas platform, dapat ditransfer antara antara big-endian dan arsitektur little-endian atau sistem 32-bit dan 64-bit. Fitur-fitur ini menjadikan DB3 pilihan populer sebagai format file aplikasi. File database SQLite_3 utama (DB3) terdiri dari satu atau lebih halaman. Semua ukuran semua halaman sama dalam database yang sama. Ukuran halaman dalam byte adalah kekuatan dua antara 512 dan 65536 inklusif.
