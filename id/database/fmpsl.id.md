## Apa itu file FMPSL?

File FMPSL adalah file snapshot dari database yang dibuat dengan FileMaker Pro 12. Ini menyimpan hasil kueri dari database bersama dengan informasi lain seperti tata letak visual dan informasi yang terlihat. File FMPSL dapat digunakan untuk memulihkan semua data ini dalam contoh lain dari database FileMaker Pro. Format file ini diperkenalkan dengan peluncuran FileMaker Pro v 12. Sebelum versi ini, tautan snapshot diperkenalkan sebagai format file FPSL di FileMaker Pro v 11. File FMPSL dapat dibuka dengan perangkat lunak FileMaker Pro Advanced. Format file lain yang didukung oleh FileMaker Pro mencakup [FP5](/id/database/fp5/), [FP7](/id/database/fp7/), dan [FMP12](/id/database/fmp12/).

### Bagaimana Cara Menyimpan Catatan sebagai file FMPSL?

Data dari database FileMaker Pro dapat disimpan sebagai file tautan snapshot menggunakan langkah-langkah berikut.

1. Cari catatan dari database yang diperlukan untuk disimpan sebagai file link snapshot.
1. Menggunakan opsi Send Record As Snapshot Link dari menu, simpan file dengan memasukkan nama file.
1. Gunakan Catatan yang sedang dijelajahi untuk menyimpan seluruh kumpulan catatan yang ditemukan.

File snapshot yang dihasilkan akan disimpan ke disk dengan nama yang ditentukan.
