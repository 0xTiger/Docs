## Apa itu file ADP?

File dengan ekstensi .adp adalah file proyek Microsoft Access yang menggunakan arsitektur komponen OLE DB untuk menyediakan koneksi langsung dan efisien ke database Microsoft SQL Server. File tersebut tidak berisi tabel aktual, diagram basis data, atau elemen basis data lainnya. File ADP dapat dibuat dengan Access 2007 dan 2010. File yang dibuat dengan versi Access yang lebih lama juga dapat dibuka dengan Access 2007 dan 2010. File ADP dapat dibuka menggunakan Microsoft Access 365.

## Format File ADP - Informasi Lebih Lanjut

Proyek Access yang dibuat sebagai ADP memungkinkan Anda membuat perubahan desain pada objek server SQL seperti tabel dan tampilan. Itu juga memungkinkan untuk membuat, mengedit, dan memanfaatkan fitur SQL seperti diagram basis data, prosedur tersimpan, dan fungsi yang ditentukan pengguna. Ini memberikan keuntungan dibandingkan menautkan ke database server SQL di mana Anda tidak dapat membuat perubahan pada desain dan hanya dapat menautkan ke tabel dan tampilan server SQL.

### Tipe Data dan Dukungan Diagram

**Jenis Data Tanggal/Waktu**

Tipe data tanggal/waktu baru berikut ini didukung di Access 2010.

* WAKTU
* TANGGAL
* TANGGAL WAKTU2
* DATETIMEOFFSET

**Jenis Data Panjang Variabel**

Tipe data panjang variabel berikut dapat digunakan dalam proyek Access 2010:

* VARBIN(MAX)
* VARCHAR(MAX)
* NVARCHAR(MAX)
