## Apa itu file 4DL?

File log 4DL digunakan untuk mencatat modifikasi yang dilakukan pada file database 4D (.4DD). File ini disimpan bersama file database dan memiliki nama file yang sama. Tujuannya adalah untuk secara akurat melacak dan memelihara catatan komprehensif dari pembaruan yang diterapkan dalam database. [File 4DD](/id/database/4dd/), dibandingkan dengan file 4DL, terutama dikaitkan dengan Dimensi ke-4 oleh 4D Inc.

## Format File 4DL - Informasi Lebih Lanjut

Biasanya, beberapa file 4DL disimpan bersama dengan file 4DD. Jadi, versi pertama berkas log mungkin diberi nama 0001.4dl, namun versi lateral berkas log akan disimpan sebagai 0002.4dl, 0003.4dl, dan seterusnya. Sekarang, jika file log itu sendiri mengalami 3 kali penyimpanan berikutnya, maka akan diberi label sebagai "db1[0005-0003].4dl."
