## Apa itu file ABS?

File dengan ekstensi .abs adalah database yang dihasilkan dan digunakan oleh database Absolute yang merupakan mesin database untuk IDE pengembangan perangkat lunak Delphi. Ini menggantikan Borland Database Engine (DBE) dan lebih kompak, berkecepatan tinggi, kuat dan mudah digunakan. Basis data ABS dikompilasi sebagai bagian dari file EXE aplikasi, membuat aplikasi Anda lebih cepat dan lebih kecil. Data disimpan dalam file ABS dalam format relasional yang terstruktur.

## Format File ABS - Informasi Lebih Lanjut

File ABS disimpan sebagai bagian dari [file EXE](/id/executable/exe/) aplikasi yang biasanya disimpan sebagai file biner. File biner tidak dapat dibuka di editor teks apa pun dan menyediakan akses cepat ke konten file database.

### Fitur Utama Format File ABS

File ABS cukup sederhana dan tertanam dalam aplikasi yang dapat dieksekusi. Ini memiliki beberapa fitur berikut:

* Tidak ada BDE; tidak ada DLL
* Basis data file tunggal
* Dukungan SQL'92 (DDL & DML).
* Kompatibel dengan kontrol basis data standar dan pihak ketiga
* Mode pengguna tunggal dan multi-pengguna (server file)
* Berfungsi dengan baik di semua versi Windows - dari 98 hingga terbaru, tidak memerlukan pembaruan atau paket layanan apa pun
* Tabel dalam memori ultra-cepat
* Kemudahan penggunaan yang tak tertandingi
* Enkripsi yang kuat
* Kompresi gumpalan
* Gratis untuk penggunaan pribadi
* Kode sumber lengkap tersedia
* Distribusi bebas royalti
