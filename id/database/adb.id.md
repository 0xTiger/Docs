## Apa itu file ADB?

File dengan ekstensi .adb adalah file database yang dibuat dengan [Alpha Five](https://www.alphasoftware.com/) (sekarang dikenal sebagai Alpha Anywhere). Ini berisi nama file dari semua tabel dan set yang merupakan anggota ruang kerja proyek. Selain itu, berisi nama semua perpustakaan yang tergabung dalam ruang kerja. Alpha Anywhere adalah IDE yang digunakan untuk mengembangkan aplikasi web. Selain file ADB, ruang kerja Alpha Anywhere mungkin berisi banyak file yang memulai nama tabel atau set, tetapi memiliki ekstensi file yang berbeda.
