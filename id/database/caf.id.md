## Apa itu file CAF?

File CAF adalah file katalog yang dibuat dengan perangkat lunak pengorganisasian media, **Cathy**. Ini berisi daftar katalog terindeks yang menunjuk ke semua media yang tersedia pada perangkat eksternal seperti flash disk dan CD. File katalog ini membantu Anda menjaga daftar media yang bersih dan mudah dicari dan diakses. File CAF menawarkan untuk mencari media berdasarkan nama, ukuran, dan tanggal.

## Format File CAF

File CAF disimpan dalam format file teks. Ini berarti Anda dapat membuka file CAF di editor teks apa pun seperti Notepad, Notepad++, dan Apple TextEdit. Cathy menyediakan fitur mengekspor struktur direktori setiap katalog ke format TXT, membuat daftar folder dan file.

### Buka File CAF

Anda dapat membuka CAF dengan software resmi Cathy. Ini mengharuskan Anda untuk menempatkan file CAF Anda di direktori yang sama dengan tempat file Cathy.exe berada. Cathy akan otomatis menampilkan katalog saat diluncurkan.

Untuk melakukannya, letakkan file CAF Anda di direktori yang sama dengan file Cathy.exe Anda. Lalu, buka Cathy. Katalog CAF Anda akan muncul di Cathy.
