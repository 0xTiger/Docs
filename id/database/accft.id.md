## Apa itu file ACCFT?

File dengan ekstensi .accft adalah file template Microsoft Access yang menyimpan definisi tipe data bidang untuk tabel database. Microsoft Access adalah Sistem Manajemen Basis Data Relasional desktop (RDBMS) yang digunakan untuk membuat basis data, formulir, kueri, dan tampilan berbasis tabel. File ACCFT dapat digunakan untuk menentukan tipe data untuk bidang tabel dan tabel database dapat dibuat menggunakan file template ini sebagai titik awal. Itu dapat dibuka dengan Microsoft Access 2016 untuk membuat tabel baru. File ACCFT dikenali oleh tipe MIME **application/msaccess**.

## Format File ACCFT

File ACCFT adalah file biner yang spesifikasi format filenya tidak tersedia untuk umum. Detail internalnya seperti informasi header, struktur file, footer, dll. Tidak tersedia saat ini.
