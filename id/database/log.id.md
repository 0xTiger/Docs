## Apa itu file LOG?
File dengan ekstensi .log berisi daftar teks biasa dengan stempel waktu. Biasanya, detail aktivitas tertentu dicatat oleh perangkat lunak atau sistem operasi untuk membantu pengembang atau pengguna melacak apa yang terjadi pada periode waktu tertentu. Pengguna dapat mengedit file-file ini dengan sangat mudah menggunakan editor teks apa pun. Biasanya laporan kesalahan atau aktivitas login dicatat oleh sistem operasi, tetapi perangkat lunak lain atau server web juga menghasilkan file log untuk melacak pengunjung dan memantau penggunaan bandwidth.

## Format File LOG
Format file LOG mencatat peristiwa khas yang terjadi baik dalam sistem operasi, pesan antara pengguna yang berbeda dari perangkat lunak komunikasi atau perangkat lunak lain yang dijalankan. Secara sederhana kita dapat mengatakan bahwa pesan tertentu pada stempel waktu tertentu dicatat dalam file LOG. Beberapa istilah yang umum digunakan untuk logging adalah:
### Log Peristiwa
Itu Merekam peristiwa yang terjadi dalam pelaksanaan sistem untuk melacak dan memahami aktivitas sistem dan untuk mendiagnosis masalah. Mereka penting untuk mengidentifikasi aktivitas sistem yang kompleks, biasanya dalam kasus aplikasi dengan interaksi pengguna yang sangat sedikit.
### Log transaksi
Hampir semua sistem basis data memelihara log transaksi, yang biasanya tidak dimaksudkan sebagai jejak audit untuk analisis selanjutnya, dan tidak dapat dibaca oleh manusia. Log ini hanya menyimpan catatan perubahan pada data yang ada untuk memungkinkan database pulih dari crash atau kesalahan data lainnya dan menyimpan data dalam keadaan konsisten. Oleh karena itu, sistem basis data biasanya memiliki log peristiwa umum dan log transaksional.
### Log pesan
Komunikasi tekstual yang disimpan oleh Internet Relay Chat (IRC), program pesan instan (IM), klien berbagi file peer-to-peer dengan fungsi obrolan, dan game multipemain (terutama MMORPG) disebut log pesan. Log ini umumnya disimpan dalam file teks biasa, tetapi klien IM dan VoIP (misalnya Skype) mungkin menyimpannya dalam file HTML untuk memudahkan membaca atau mengaktifkan enkripsi.
### Log server
Log server sebenarnya adalah file log yang berisi daftar aktivitas yang dilakukan dan dibuat atau dipelihara secara otomatis oleh server itu sendiri. Biasanya file-file ini tidak dapat diakses oleh pengguna Internet umum, hanya untuk webmaster atau orang administratif lain dari layanan Internet.
