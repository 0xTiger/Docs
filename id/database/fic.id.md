## Apa itu file FIC?

File FIC di WinDev adalah versi kompilasi dari Database HyperFileSQL. Akronim FIC adalah singkatan dari Fichier de Code dalam bahasa Perancis, yang diterjemahkan menjadi Code File dalam bahasa Inggris. Di WINDEV, file FIC adalah file kompilasi yang berisi kode proyek yang dapat dieksekusi. File tersebut memiliki ekstensi .fic dan dibuat saat Anda mengkompilasi proyek WINDEV. Setelah sebuah proyek dikompilasi menjadi file FIC, proyek tersebut dapat didistribusikan ke pengguna sebagai program yang dapat dieksekusi atau paket instalasi.

File FIC terkait dengan WINDEV, alat pengembangan perangkat lunak yang dibuat oleh PCSoft, sebuah perusahaan Perancis. Hal ini memungkinkan pengembang untuk membuat aplikasi Windows, web, dan seluler menggunakan satu basis kode. Ini mencakup lingkungan pengembangan visual, bahasa pemrograman (WLangage), dan berbagai kontrol dan fungsi bawaan untuk tugas-tugas umum. Ini terutama digunakan untuk aplikasi bisnis dan memiliki basis pengguna yang kuat di Eropa.

## Hubungan dengan database HyperFileSQL Mobile

Di WinDev, file FIC yang dibuat untuk database HyperFileSQL Mobile berisi kode yang dapat dieksekusi untuk database, serta data yang terdapat dalam database. HyperFileSQL Mobile adalah sistem manajemen basis data relasional yang disertakan dengan WINDEV dan dirancang khusus untuk aplikasi seluler dan offline. File FIC dapat didistribusikan sebagai file executable mandiri atau sebagai bagian dari paket instalasi, dan dapat dijalankan pada perangkat target untuk mengakses dan mengelola data dalam database HyperFileSQL Mobile. Ketika file FIC dijalankan, itu akan membuka database dan membuat data tersedia untuk digunakan dalam aplikasi seluler.
