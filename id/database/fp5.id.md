## Apa itu file FP5?

File FP5 adalah file database yang dibuat dengan FileMaker Pro versi 5 atau 6. Ini menyimpan pengguna serta data organisasi dalam bentuk catatan di dalam tabel. FileMaker Pro adalah sistem manajemen basis data yang mendukung sistem operasi Macintosh dan Windows. FP5 adalah format file lama yang didukung oleh versi FileMaker Pro yang tersedia saat itu. Format database FileMaker Pro terbaru adalah [FMP12](/id/database/fmp12/) yang diperkenalkan dengan peluncuran FileMaker Pro v 12. Format file database lain yang juga digunakan adalah [FP7](/id/database/fp7/) yang digantikan oleh FMP12.

## Format File FP5 - Informasi Lebih Lanjut

Detail format file internal dari format file FP5 tidak tersedia secara publik untuk referensi developer. Format file database FileMaker Pro yang lebih lama seperti [FP5 dan FP7 dapat ditingkatkan ke FMP12 terbaru](https://fmhelp.filemaker.com/help/16/fmp/en/index.html#page/FMP_Help/converting-files. html) menggunakan panduan terperinci dari FileMaker Pro.
