## Apa itu file ANB?

File dengan ekstensi .anb adalah bagan Notebook IBM i2 Analyst yang berisi informasi hubungan antara data yang berbeda. Seperti namanya, perangkat lunak Notebook Analis membantu dalam menghubungkan data, menghasilkan bentuk konektivitas yang lebih ringkas antara potongan-potongan informasi yang berbeda dan memberikan output intelijen untuk analisis. Ini membantu mengidentifikasi dan menghentikan aktivitas jahat seperti ancaman kriminal, dunia maya, dan penipuan. File ANB dapat dibuka dengan perangkat lunak Notebook IBM i2Analyst.

## Format File ANB - Informasi Lebih Lanjut

File ANB terdiri dari item penyusun berikut:

* Entitas - Representasi potongan Informasi
* Tautan - Menampilkan konektivitas antar Entitas
* Properti - Deskripsi entitas atau tautan lebih terinci

File ANB menyimpan semua informasi ini berdasarkan umpan informasi masukan ke dalam Notebook Analis ANB. Berdasarkan input data, perangkat lunak membuat tautan antara setiap bagian informasi yang membantu memetakan hubungan dan melacak bukti secara geospasial dan temporal.
