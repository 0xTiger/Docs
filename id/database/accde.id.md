## Apa itu file ACCDE?

File dengan ekstensi .accde adalah format file Microsoft Access 2007 untuk membuat file database Eksekusi Saja untuk melindungi kode database kustom. Itu mengkompilasi semua kode VBA menjadi hanya dapat dieksekusi sehingga tidak dapat dilihat atau diedit. Format file ACCDE adalah pengganti format file MDE yang digunakan oleh versi Microsoft Access sebelumnya. ACCDE adalah file [ACCDB](/id/database/accdb/) dalam mode hanya dapat dieksekusi dan lebih ringkas karena tidak adanya kode sumber apa pun. Ukuran kompak dari file-file ini meningkatkan penggunaan memori serta meningkatkan kinerja aplikasi. File ACCDE dapat dibuka dengan Microsoft Access 365.

## Format File ACCDE

Semua file Microsoft Access 2007+ menggunakan Jet ACE (Access database engine) yang menggunakan keluarga format file ACCDB. Namun, Microsoft tidak menyediakan implementasi sumber terbuka atau spesifikasi teknis terperinci.
