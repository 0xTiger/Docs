## Apa itu file AQ?

File dengan ekstensi .aq adalah file database yang dibuat dengan aplikasi Ancestral Quest oleh Incline Software, LC. Ini menyimpan informasi silsilah keluarga menggunakan data silsilah yang dimasukkan oleh individu menggunakan versi gratis dan premium dari perangkat lunak. Database menyimpan informasi seperti nama, tempat lahir, dan hubungan anggota keluarga. Ancestral Quest menggunakan informasi ini untuk menautkan dan mengklasifikasikan informasi seperti tampilan Pedigree, Descendants, Family, dan Timeline. File AQ dapat dibuka menggunakan [Incline Software Ancestral Quest](https://www.ancquest.com/index.htm).

## Format File AQ - Informasi lebih lanjut

File AQ disimpan sebagai file biner dan format file internalnya tidak tersedia.
