## Apa itu File CHCK?

File CHCK dibuat sebagai hasil pemeriksaan pemantauan kesehatan di Microsoft Exchange Server. Ini berisi informasi tentang status keseluruhan dan operasionalisasi database server. Microsoft Exchange Server digunakan terutama sebagai server email dan memungkinkan Anda mengelola kontak bisnis dan kalender Anda. Administrator server Exchange menggunakan EseUtil untuk memeriksa kesehatan basis datanya.

## Format File CHCK

Format file internal file CHCK tidak diketahui dan tidak ada bahan referensi yang tersedia untuk membaca/menulis file CHCK.
