## Apa itu file BC3?

File BC3 adalah file database untuk menyimpan informasi terkait konstruksi format pertukaran database. Ini dikembangkan berdasarkan format pertukaran basis data konstruksi standar Spanyol. File BC3 menyimpan informasi yang memiliki catatan data termasuk rincian bahan konstruksi, biaya bahan, dan harga tenaga kerja. Data dapat dipertukarkan antar aplikasi lain tanpa mengonversi data ke format lain.

## Format Berkas BC3

File BC3 disimpan sebagai file biner dengan rincian yang ditentukan sesuai standar FIEBDC. Implementasi file BC3 telah melalui revisi yang berbeda dan ada versi format file BC3 berikut:

* FIEBDC-3/2004
* FIEBDC-3/2007
*FIEBDC-3/2012
*FIEBDC-3/2016

FIEBDC adalah kependekan dari Formato de Intercambio Estándar - Bases de Datos de Construcción dalam bahasa Spanyol.
