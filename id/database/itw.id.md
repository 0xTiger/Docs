## Apa itu file ITW?

File ITW adalah format file database yang digunakan oleh aplikasi CRM [InTouch](https://www.intouchsoftware.co.uk/InTouch). Ini menyimpan konten dalam format file Kepemilikan InTouch dan detail format file internalnya tidak tersedia. File ITW menyimpan informasi khusus untuk menyimpan nama, alamat yang selanjutnya digunakan untuk mencetak amplop dan label. Informasi ini juga digunakan untuk mengirim email, pesan teks, buletin elektronik yang menarik, membuat templat, lampiran, dan tautan yang dapat digunakan kembali. Data dari CRM juga dapat diekspor ke format ITW.
