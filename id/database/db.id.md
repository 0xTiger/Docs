## Apa itu file DB?

File dengan ekstensi db adalah file database umum untuk menyimpan data. Tidak ada spesifikasi resmi yang harus diikuti untuk file database semacam itu. Data diatur dalam format terstruktur di dalam file dalam bentuk tabel, bidang, tipe data, dan nilai bidang. File DB juga dapat dibuat dengan membaca dari file lembar data yang memiliki data dalam baris dan kolom. Konten file DB selanjutnya dapat diekspor ke format file [CSV](/id/spreadsheet/csv/) dan [PDF](/id/pdf/) untuk tujuan pelaporan. File DB dapat dibuka di aplikasi seperti SQLite dan LibreOffice.
