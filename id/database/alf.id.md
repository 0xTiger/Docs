## Apa itu file ALF?

File dengan ekstensi .alf adalah log Transaksi SQL yang dihasilkan oleh [ACT! Perangkat lunak CRM](https://www.act.com/)(Customer Relationship Management). Itu membuat catatan kegiatan antara ACT! dan SQL, dan digunakan untuk memulihkan ACT! basis data. BERTINDAK! CRM menyimpan informasi pelanggan seperti nama, alamat email, kontak, informasi bisnis, dan riwayat hubungan. Ini menyimpan informasi ini di [ACT! File ADF](/id/database/adf/) yang menyimpan data dalam tabel untuk ditautkan dengan modul lain seperti pengambilan informasi otomatis untuk panggilan masuk.
File ALF dapat dibuka dengan perangkat lunak SwiftPage ACT di OS Windows.

## Format File ALF - Informasi lebih lanjut

File ALF disimpan sebagai file biner dan format file internalnya tidak tersedia.
