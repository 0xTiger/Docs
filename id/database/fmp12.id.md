## Apa itu berkas FMP12?

File FMP12 adalah file database yang dibuat dengan program pengembangan database FileMaker Pro 12. Ini berisi keseluruhan struktur database (juga dikenal sebagai skema database) bersama dengan data di dalamnya. Struktur file FMP12 didasarkan pada tabel dimana setiap tabel terdiri dari sejumlah field database yang memiliki tipe data yang berbeda. Beberapa tipe data yang didukung oleh format file FMP12 termasuk teks, formulir kustom, laporan, waktu, blob, dan objek database lainnya. Format file FileMaker Pro yang lebih lama seperti [FP7](/id/database/fp7/) dapat ditingkatkan ke FMP12.

## Bagaimana cara Mengonversi File FileMaker Pro Lama ke FMP12?

Format file FileMaker Pro yang lebih lama seperti database 11, 10, 9, 8 dan 7 (.fp7) ke format file FMP12. FileMaker memiliki panduan ekstensif untuk [mengonversi format file FileMaker Pro versi sebelumnya ke FMP12](https://fmhelp.filemaker.com/help/16/fmp/en/index.html#page/FMP_Help/converting-files.html) .
