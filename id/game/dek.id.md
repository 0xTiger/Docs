## Apa itu file DEK?

File dengan ekstensi .dek adalah setumpuk file kartu yang digunakan oleh game mobil koleksi online, Magic: The Gathering Online (MTGO). Meskipun dalam banyak kasus, setumpuk kartu ini dapat dimainkan, dalam beberapa kasus, file DEK mencantumkan kumpulan kartu yang tidak dapat dimainkan. Ini tidak dapat dimainkan tetapi bertindak sebagai bagian dari koleksi. File DEK dapat dibuat dari file [CSV](/id/spreadsheet/csv/) dengan mengekspor menggunakan Delver Lens dan dapat digunakan di Tabletop Simulator.

## Format File DEK

File DEK disimpan sebagai file teks biasa multibaris dalam format [XML](/id/web/xml/) yang dapat dibaca manusia. Ini dapat dibuka dengan editor teks seperti Microsoft Notepad, Notepad++, dan Wizards of the Coast Magic: The Gathering Online. File DEK juga digunakan oleh banyak pemain Magic untuk mengimpor/mengekspor dari banyak pembuatan dek dan situs web seperti [CardHoarder](https://www.cardhoarder.com/). Situs web ini memungkinkan untuk mengunggah file MTGO DEK sebagai satu atau daftar file.
