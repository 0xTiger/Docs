## Apa itu file REP?

File dengan ekstensi .rep adalah file replay Grand Theft Auto (GTA) yang berisi data bingkai yang diambil yang dapat diputar ulang di GTA. GTA adalah game balap dan kampanye mobil dunia terbuka yang dimainkan di antara pemain di jaringan. File REP menyediakan mekanisme untuk berbagi dan menganalisis pengalaman bermain game setelah game selesai. Pemutaran ulang dalam file seperti itu adalah urutan adegan dan tindakan yang disimpan dalam file REP. File REP disimpan di direktori File Pengguna dan dapat dibuka dengan GTA Replays Manager.

## Format File REP - Informasi Lebih Lanjut

File GTA REP disimpan ke disk sebagai file biner dan tidak dapat dibaca manusia. Setiap file .rep memiliki string header dan berisi jumlah adegan yang bervariasi (disebut juga bingkai). Setiap adegan tertentu berisi informasi tentang aktor, posisi kendaraan, cuaca, sulaman, dll. pada waktu tertentu. Saat layar direkam satu demi satu, mereka menciptakan efek yang sama seperti saat film diputar.

Spesifikasi format file REP tersedia di [gta fandom](https://gta.fandom.com/wiki/Replay/Format_details) dan dapat dirujuk untuk informasi selengkapnya tentang header dan blok datanya.
