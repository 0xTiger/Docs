## Apa itu file MCWORLD?
File MCWORLD terdiri dari file sumber daya dan perilaku untuk membuat dunia virtual dalam game Bedrock Edition of Minecraft yang merupakan game pembangunan blok yang terkenal. Itu memampatkan berbagai file, seperti file .DAT dan .TXT dengan kompresi ZIP. file menggunakan ekstensi file .mcworld dan ini terutama digunakan untuk mencadangkan atau mentransfer dunia ke pengguna Minecraft lain. File-file ini biasanya dibuat oleh perangkat lunak Minecraft yang disebut Minecraft Launcher.

## format file MCWORLD
File MCWORLD dibuat oleh **MinecraftSP.exe** yang juga disebut Minecraft Launcher. File ini dapat dibuat jika kita meng-zip atau mengkompres file dunia menjadi satu file dan mengubah ekstensinya menjadi .mcworld. Di sisi lain kita dapat memperluas file .mcworld menggunakan software zip decompressor utilitas sederhana, karena file .mcworld dikompresi menggunakan kompresi zip.
