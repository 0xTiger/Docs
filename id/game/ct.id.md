## Apa itu file CT?

File CT adalah file tabel cheat yang dibuat dengan perangkat lunak [Cheat Engine](https://www.cheatengine.org/) yang digunakan untuk membuat modifikasi pada game berbasis Windows untuk melakukan cheat. Cheat Engine, mesin curang sumber terbuka, memeriksa game yang dijalankan dan mencatat lokasi alamatnya. Informasi ini dan penggantiannya ditulis dalam file CT yang kemudian dimuat oleh pemain game untuk mengubah properti game seperti skor kesehatan, skor tertinggi, dan sisa nyawa.

## Format File CT Cheat Engine

File CT disimpan dengan lokasi alamat dan informasi terkait lainnya untuk diretas di dalam game. Dalam kebanyakan kasus, file disimpan sebagai arsip [ZIP](/id/compression/zip/) terkompresi yang dapat dengan mudah diekstrak menggunakan utilitas dekompresi standar seperti WinZIP atau 7-Zip.

## Cara Menggunakan Tabel Curang

1. Unduh tabel dan salin ke folder Cheat Engine
1. Jalankan Cheat Engine
1. Jalankan game;
1. Gunakan kombinasi ALT+TAB dan pilih game pada daftar proses melalui Cheat Engine
1. Jika tabel berada di folder yang berbeda, cukup tekan Control+O dan arahkan Cheat Engine ke folder tersebut. Kemudian pilih tabel (biasanya process_name.ct);
1. Setelah tabel dimuat, jika ada skrip, centang saja.
1. ALT+TAB kembali ke permainan dan bersenang-senanglah.
