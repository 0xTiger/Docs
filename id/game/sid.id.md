## Apa itu file SID?

File SID adalah file cadangan game yang dibuat oleh aplikasi Steam yang digunakan untuk mengunduh dan mengelola **Valve video game** di OS Windows. Ini menyimpan data game dalam format terkompresi dan dapat digunakan untuk mencadangkan game ke media penyimpanan seperti DVD dan perangkat penyimpanan USB. File SID dibuat dengan utilitas cadangan Steam dan mirip dengan format file [CSD](/id/game/csd/) untuk menyimpan cadangan game.

## Format File SID - Informasi Lebih Lanjut

File SID disimpan ke disk sebagai file biner. Itu selalu disertai dengan file .sis dan .sim yang digunakan untuk mengekstrak informasi metadata untuk memulihkan data game cadangan.
