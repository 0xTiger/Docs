## Apa itu file ASSET?

File ASSET adalah file khusus yang digunakan di Unity, sebuah program untuk membuat video game. Ini menyimpan konfigurasi penting seperti objek atau pengaturan game. Misalnya, satu file ASSET mungkin menyimpan bagaimana objek bergerak dalam game, seperti bagaimana objek tersebut memantul atau jatuh. File-file ini seperti wadah yang menyimpan semua informasi penting agar tetap terorganisir sehingga pengembang game dapat dengan mudah mengerjakan proyek mereka.

## Cara membuka file ASSET
  
Di Unity, Anda biasanya tidak membuka file aset dengan cara yang sama seperti Anda membuka dokumen atau file di editor teks atau program serupa. Sebaliknya, Unity mengelola file aset di dalam Unity Editor itu sendiri. Inilah cara Anda dapat mengakses dan bekerja dengan file aset di Unity:

1.  **Import Assets**: To use an asset file in your Unity project, you need to import it into project. You can do this by either dragging and dropping asset file directly into Unity Editor or by using "Assets" menu and selecting "Import New Asset."
    
2.  **Melihat Aset**: Setelah diimpor, Anda dapat melihat dan mengelola file aset di jendela Proyek pada Unity Editor. Jendela ini menampilkan semua aset dalam proyek Anda dan memungkinkan Anda mengaturnya ke dalam folder, mencari aset tertentu, dan melihat pratinjau isinya.
    
3.  **Menggunakan Aset**: Anda dapat menggunakan file aset dalam proyek Unity Anda dengan menyeretnya dari jendela Proyek ke dalam hierarki adegan atau ke GameObjects dalam adegan. Misalnya, Anda dapat menyeret aset tekstur ke material untuk menerapkannya ke objek 3D, atau Anda dapat menyeret aset prefab ke dalam adegan untuk membuat instance GameObject yang telah dikonfigurasi sebelumnya.
    
4.  **Mengedit Aset**: Beberapa file aset, seperti skrip, materi, dan animasi, dapat diedit langsung dalam Unity Editor. Anda dapat mengklik dua kali aset ini di jendela Proyek untuk membukanya di editor masing-masing dan membuat perubahan sesuai kebutuhan.
    
5.  **Menyimpan Aset**: Setiap perubahan yang Anda buat pada file aset dalam Unity Editor secara otomatis disimpan dalam proyek Unity Anda. Tidak perlu menyimpan file aset individual secara eksplisit seperti yang Anda lakukan dengan dokumen di program perangkat lunak tradisional.

File ASSET dapat dibuka atau direferensikan menggunakan program berikut.
