## Apa itu file USX?

File USX berisi jerat Unreal Engine yang merupakan objek/model 3D pra-desain yang digunakan untuk menghasilkan lingkungan. Model dari file USX memiliki posisi, orientasi, dan ukuran berbeda, namun tetap statis. Static Meshes dapat dioptimalkan untuk performa dan dapat digunakan untuk membuat berbagai tingkat detail (LOD) guna meningkatkan performa game. Mereka juga dapat digunakan untuk membuat volume tumbukan dan dapat digunakan sebagai bagian dari simulasi fisika.

File USX dapat dibuat dengan editor level Unreal UnrealEd.

## Format File USX - Informasi Lebih Lanjut

File USX disimpan ke disk sebagai file biner. Detail struktur file internal format file USX tidak tersedia untuk umum untuk referensi pengembang.
