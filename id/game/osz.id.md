## Apa itu file OSZ?

File OSZ adalah format file terkompresi yang digunakan oleh game ritme osu! untuk menyimpan data beatmap. Beatmap pada dasarnya adalah level dalam game, dan mencakup informasi seperti lagu, waktu ketukan, dan penempatan objek pukulan di layar game. File OSZ memungkinkan distribusi beatmap dengan mudah dan biasanya diunduh dari internet dan diimpor ke dalam game. OSU! juga memiliki format file [OSR](/game/osr/) yang merupakan file replay game dan dapat diputar ulang oleh pemain game.

**Jenis MIME Format File OSR:** x-osu-replay

## Format File OSZ

Detail format file internal jenis file OSZ tidak tersedia untuk umum. Ini berisi data terkompresi [ZIP](/compression/zip/) yang memiliki informasi untuk memutar musik, menampilkan grafik, dan menampilkan pemain tentang peristiwa ketika mereka harus berinteraksi dengan game.

## Bagaimana Cara Membuat File OSZ?

OSU! memiliki petunjuk rinci untuk file [creating OSZ](https://osu.ppy.sh/wiki/en/Client/File_formats#creating-.osz-and-.osk-files) dan OSK. Langkah-langkah berikut dapat digunakan untuk membuat file OSZ menggunakan OSU!.

1. Buka beatmap melalui editor.
1. Dari menu atas, pilih File > Ekspor paket....
1. Arsip .osz akan ditempatkan di folder Ekspor di dalam osu! map.
