## Apa itu file KODU?

File .kudo adalah file Game 3D yang dibuat dan disimpan dengan Microsoft [perangkat lunak Kodu Game Lab](https://www.kodugamelab.com/) yang merupakan lingkungan pengembangan game edukasi. Ini dirancang untuk mengajarkan prinsip pemrograman dasar kepada anak-anak dan memungkinkan pencipta memprogram perilaku elemen permainan menggunakan bahasa pemrograman visual. File Kodu dapat disimpan dan dibagikan dengan pengguna lain tetapi hanya dapat dibuka dan dijalankan di perangkat lunak Kodu.

## Format File KODU - Informasi Lebih Lanjut

File Kodu adalah file biner, disimpan dalam format file milik Microsoft. Spesifikasi format filenya tidak tersedia untuk umum untuk pengembang dan itulah sebabnya file Kodu hanya dapat dijalankan di dalam perangkat lunak Kodu Game Lab.
