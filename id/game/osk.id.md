## Apa itu file OSK?

File OSK adalah file tata letak atau skin yang digunakan oleh game ritme osu! dimana pemain menggunakan mouse, keyboard yang sinkron dengan musik. Ini berisi tata letak antarmuka editor dan dapat disesuaikan oleh pemain agar sesuai dengan preferensi mereka. osu! menyimpan data beatmap dalam format file [OSZ](/game/osz/).

**Jenis MIME Format File OSK:** x-osu-skin-archive

Anda dapat membuka file OSK dengan [osu!](https://osu.ppy.sh/home).

## Format File OSK

File OSK disimpan ke disk sebagai file terkompresi [ZIP](/compression/zip/). Anda cukup menyimpan file .osk ke .zip dan mengekstrak arsip untuk melihat isinya. Konten yang diekstraksi harus ditempatkan di direktori \Skins\ untuk dimuat ke dalam game.

## Apa yang bisa dikustomisasi dengan file OSK?

Elemen permainan berikut dapat dikustomisasi dengan file .osk.

 * Pukul lingkaran
 * Tampilan skor
 * Tipe penggeser
 * Kursor
 * Pukulan meledak
