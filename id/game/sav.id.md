## Apa itu file SAV?

File dengan ekstensi .sav adalah file game tersimpan Nintendo DS yang dapat dimuat di Emulator Game Nintendo DS seperti [DeSmuME](http://desmume.org/) dan [NO$GBA](https://www. nogba.com/). File-file ini menyimpan status game saat ini yang kemudian dapat dimuat di emulator Nintendo. Informasi yang disimpan dalam file SAV mencakup nama karakter game, poin pengalaman, level yang diselesaikan, dan sumber daya dalam game. File SAV dapat dimuat di emulator tetapi perbedaan versi file SAV dapat menyebabkan kesalahan pada emulator. File SAV dapat dibuka dengan emulator DeSmuME dan NO$GBA.

## Format File SAV - Informasi Lebih Lanjut

File Nintendo DS SAVE disimpan ke disk dalam format file biner. Meskipun ini tidak dapat dibaca oleh manusia, editor HexaDecimal seperti [HxD](https://mh-nexus.de/en/hxd/) dapat digunakan untuk membukanya untuk analisis. Dalam skenario di mana file SAV tidak dapat dimuat ke versi emulator yang ditingkatkan, file SAV dapat dibuka di editor Hex untuk menghapus data empuk yang tidak diinginkan.
