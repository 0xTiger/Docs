## Apa itu file KLIP?

File dengan ekstensi .clip adalah file replay Grand Theft Auto 5 (GTA V) yang berisi data input pengontrol. Editor Rockstar mereferensikan data ini untuk memutar ulang urutan dari gameplay. Ini bukan pemutaran ulang video sungguhan, tetapi rekaman input pengontrol dengan waktu hingga milidetik. Dengan demikian, pengaturan sudut kamera bisa diatur melalui editor. Namun, editor dapat digunakan untuk membuat video sebenarnya dalam format file video populer seperti [MP4](/id/video/mp4/). GTA V menyimpan video yang diekspor ke C:\Users\YOUR USER NAME\AppData\Local\Rockstar Games\GTA V\videos\rendered.

## Format File KLIP

File CLIP bukan file video asli. Sebaliknya, ini berisi data game, seperti posisi, nilai rotasi untuk objek, status dan kondisi dunia, pemicu audio, pergerakan kamera game, tindakan pemain, hal-hal seperti itu. Ini tidak dapat dilihat di luar game.

## Bagaimana Cara Merekam Klip dan Mengakses Editor Rockstar di GTAV?

Setelah perekaman klip selesai, langkah-langkah berikut dapat digunakan untuk merekam klip.

* Jeda permainan
* Gulir ke tab paling kanan, "Rockstar Editor"
* Luncurkan editor dari tab ini
* Anda akan dapat memilih layar "Manajemen Klip" dari sini dan membuat perubahan pada klip yang telah Anda rekam di GTA Online atau Story Mode.
