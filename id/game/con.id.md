## Apa itu file CON?

File CON adalah file konfigurasi yang digunakan oleh video game first-person shooter **Battlefield 1942**. Ini berisi informasi konfigurasi yang dimuat dan digunakan oleh game. Ada beberapa file .con yang mendefinisikan berbagai jenis objek seperti tindakan bot, kerusakan senjata, animasi, suara dalam level, atau karakteristik lainnya.

## Format File CON

File CON disimpan ke disk sebagai file teks biasa yang dapat dibuka di editor teks apa pun. Pengaturan disimpan dalam baris teks yang dapat diedit dan disimpan kembali untuk dimuat ulang oleh game.
