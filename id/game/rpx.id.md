## Apa itu file RPX?

File RPX adalah file game yang dapat dimuat atau dimainkan oleh konsol game Wii U. Itu juga dapat dimuat dan dimainkan di emulator video game Cemu yang populer untuk menyediakan lingkungan emulator untuk file Wii U. File game Wii U berukuran besar dan juga dapat disimpan sebagai file [WUD](/id/disc-and-media/wud/) atau [WUX](/id/compression/wux/) untuk dimuat oleh emulator game. File RPX berisi seluruh game yang dapat dieksekusi yang terdiri dari semua komponen game seperti grafik, tekstur, urutan, dan ROM game.

## Format File RPX - Informasi Lebih Lanjut

File RPX disimpan ke disk dalam format file berpemilik. API tertentu tersedia untuk mengompres atau mendekompresi file RPX seperti [wiiurpxtool](https://github.com/0CBH0/wiiurpxtool). Selain itu, aplikasi seperti emulator video game Cemu memungkinkan Anda memainkan game Wii U di Windows di PC.
