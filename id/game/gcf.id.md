## Apa itu file GCF?

File .gcf adalah file cache game yang digunakan oleh platform game Steam. Ini berisi data game seperti tekstur, model, dan file lain yang digunakan oleh game. File-file ini disimpan di komputer pengguna dan digunakan untuk mengurangi jumlah data yang perlu diunduh saat memperbarui atau menginstal game. File .gcf juga dapat digunakan untuk membuat cadangan instalasi game atau untuk mentransfer game antar komputer.

File GCF dapat dibaca dan ditulis oleh Perpustakaan pemrograman C++ Sumber Terbuka, **HLLib**.

## Format File GCF

File GCF disimpan ke disk sebagai file biner. GCF awalnya digunakan sebagai singkatan dari Grid Cache File (nama kode awal Steam adalah Grid), tetapi kemudian diambil sebagai Game Cache File. File GCF adalah file arsip yang menyimpan game Steam. Semua konten resmi diunduh sebagai file GCF. File GCF juga dapat dibagikan antar game.

CATATAN: GCF adalah pendahulu dari [VPK file format](/game/vpk/).
