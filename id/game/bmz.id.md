## Apa itu file BMZ?

File BMZ adalah file peta Bonus yang digunakan oleh game pemecahan teka-teki 3D `Portal` yang dikembangkan oleh Valve. Ini berisi peta bonus yang juga dapat dibuat oleh pengembang komunitas pihak ketiga dan diimpor ke dalam game menggunakan UI bawaan. BMZ hadir sebagai file arsip [ZIP](/id/compression/zip/), yang memiliki file BSP, TGA, dan [BNS](/id/game/bns/) di dalamnya. File BMZ dapat dibuka menggunakan Valve Portal dan Valve Portal 2.

## Format File BMZ

BMZ disimpan sebagai file ZIP dan dapat diekstraksi dengan utilitas ekstraksi ZIP standar seperti WinZIP atau WinRAR. Saat file BMZ diekstraksi, file berikut dapat ditemukan:

* BSP - File peta
* TGA - File logo
* BNS - File Skrip Peta Bonus Portal

## Cara Membuat dan Memasang File Peta Bonus

Komunitas developer di Valve Software telah membuat panduan bagi orang yang ingin [membuat dan menginstal Bonus Map](https://developer.valvesoftware.com/wiki/Bonus_Maps) file. Ini mencakup langkah-langkah berikut:

1. Menambahkan peta bonus BSP ke mod
1. Menambahkan Gambar Thumbnail
1. Membuat file deskripsi bonus (.BNS)
