## Apa itu file XS?

File XS adalah file referensi yang digunakan di Age of Empires III (AOE3) untuk menghasilkan kecerdasan buatan (AI). Ini digunakan selama proses kompilasi ketika file peta XML khusus dimuat. Jika Anda pernah memainkan AOE3, Anda pasti tahu bahwa Anda tidak hanya bisa memilih peta apa saja untuk dimainkan, tapi juga bisa membuat peta sendiri menggunakan program Random Map Script.

## Bagaimana cara menghasilkan Peta AI di Age of Empires 3?

 * Buka program pengeditan teks, seperti Microsoft Notepad, Notepad++, atau TextEdit. Ini akan digunakan untuk memprogram peta Anda.
 * Program semua fitur peta Anda menggunakan bahasa Random Map Script.
 * Anda dapat merujuk ke tutorial pemrograman untuk memastikan Anda memprogramnya dengan benar
 * Simpan file Anda sebagai MyAOE3.xs dan pindahkan ke folder game Age of Empires 3.

### Format File XS - Informasi Lebih Lanjut

File XS disimpan sebagai file teks biasa yang dapat dibuka dan diedit dengan editor teks apa pun.
