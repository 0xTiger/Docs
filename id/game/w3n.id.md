## Apa itu file W3N?

File dengan ekstensi .w3n adalah file peta kampanye yang digunakan oleh game [Warcraft III: The Frozen Throne](https://en.wikipedia.org/wiki/Warcraft_III:_The_Frozen_Throne) yang merupakan ekstensi dari Warcraft III, semua video game terkenal waktu. File W3N berisi peta kampanye yang digunakan untuk membuat kampanye khusus menggunakan rangkaian data misi yang tersedia di dalamnya. Formatnya sering disalahpahami sebagai file peta dan pengguna tidak tahu mengapa file W3N mereka tidak dimuat di Frozen Throne. File W3N dapat dibuka dengan Blizzard Warcraft 3: The Frozen Throne.

## Format File W3N - Informasi Lebih Lanjut

File W3N disimpan ke disk sebagai file biner dan format file internalnya tidak tersedia untuk umum. Komponen Editor Kampanye dari Editor Dunia Warcraft III memungkinkan untuk membuat file W3N.

## Bagaimana cara menginstal file W3N?

Jangan salah menempatkan file W3N di folder "maps" di komputer Anda. File W3N bukanlah file peta, tetapi kampanye yang dapat digunakan untuk membuat misi. Oleh karena itu, untuk memuat file-file ini, ini harus ditempatkan di folder "campaigns" dan bukan folder "maps".

### Contoh Peta Kampanye W3N

Beberapa contoh peta kampanye meliputi:
* Liga Kalimdor II v3.46 AI
* Azeroth Diciptakan Kembali
* Hari Naga
* Tuan Klan
