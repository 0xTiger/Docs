## Apa itu file META?

File META berisi data game dari berbagai game yang menggunakan Exient XGS Engine seperti Angry Birds Go! atau Transformer Angry Birds. File META menangkap dan menyimpan informasi tentang aspek-aspek tertentu dari permainan seperti log kerusakan atau log item.

File META dibuat secara otomatis oleh game dan tidak dimaksudkan untuk modifikasi pengguna atau akses langsung. Jika file META didedikasikan untuk mencatat kerusakan game, biasanya akan diberi nama "crashes.meta." Demikian pula, jika file META digunakan untuk mencatat informasi terkait item, maka akan mengikuti konvensi penamaan "XXX.xml.meta," di mana "XXX" mewakili pengidentifikasi deskriptif untuk tipe item tertentu yang dicatat.

## Mesin XGS yang ada

Exient Entertainment menciptakan Exient XGS Engine, sebuah mesin game. Merupakan framework perangkat lunak yang dibuat untuk mempermudah proses pembuatan dan pengembangan video game. Ia menawarkan kepada programmer berbagai alat, perpustakaan, dan fungsi untuk mempercepat proses pembuatan game.

Beragam fitur dan kemampuan yang diberikan Exient XGS Engine memudahkan dalam membuat game di berbagai platform, termasuk PC, konsol, dan perangkat seluler. Fungsi-fungsi ini sering kali terdiri dari produksi visual, simulasi fisika, pengelolaan audio, penanganan input, kemampuan jaringan, dan lain-lain.
