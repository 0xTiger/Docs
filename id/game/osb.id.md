## Apa itu file OSB?

File OSB adalah file storyboard yang dibuat oleh game ritme Osu! yang memungkinkan pemain mengetuk elemen visual di layar dengan irama musik. Storyboard di dalam file OSB berisi latar belakang animasi khusus yang digunakan di seluruh bagian belakang game saat bermain. Latar belakang storyboard biasanya berupa file JPG atau PNG yang ditampilkan dengan file beatmap OSZ selama permainan.

## Format Berkas OSB

File OSB disimpan sebagai file teks biasa ke disk dalam folder beatmap. Skema ini memungkinkan untuk membuat storyboard berbeda yang memiliki tingkat kesulitan berbeda dalam sebuah beatmap.
