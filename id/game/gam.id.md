## Apa itu file GAM?
File dengan ekstensi .gam digunakan oleh berbagai video game untuk melibatkan para gamer untuk melanjutkan dari bagian terakhir yang mereka tinggalkan saat program dibuka lagi. Oleh karena itu, itu berarti file GAM menyimpan informasi tentang apa yang telah dilakukan pengguna pada waktu tertentu. File-file ini dapat disimpan secara otomatis oleh perangkat lunak game atau gamer mungkin diminta untuk menyimpan jika mereka ingin melanjutkan keadaan permainan di mana mereka pergi.
## format file gam
Format file GAM pada dasarnya digunakan untuk menyimpan data game di save game. File GAM tidak menyimpan makhluk, informasi item, atau area, melainkan menyimpan data anggota party dan variabel umum atau global yang memengaruhi anggota party.
### Struktur file GAM
File GAM memiliki struktur berikut:
- Tajuk
- Anggota partai
- Data file CRE anggota partai
- NPC (yang bukan anggota party)
- NPC membunuh statistik (tertanam dalam struktur NPC)
- Data file NPC CRE
- Variabel (dalam namespace GLOBAL)
- Entri jurnal
- Informasi yang familiar
- Lokasi Tersimpan
- Lokasi Saku Pesawat
- Ekstra Akrab
