## Apa itu file PSS?

File dengan ekstensi .pss adalah file Video Game Sony PlayStation 2 dan digunakan untuk menyimpan data audio dan video oleh game untuk konsol Playstation 2. Data video dan audio ini sebenarnya adalah cutscene yang dapat diekstraksi dari file PSS menggunakan utilitas gratis. Salah satu utilitas tersebut adalah [RipBot](http://forum.doom9.org/showthread.php?t=127611) yang telah dikembangkan di Delphi untuk mengekstrak konten dari file PSS. Data dari file PSS dapat dibagi menjadi komponen audio [MPEG2](/id/video/mpg/) video dan [WAV](/id/audio/wav/), dan dikonversi ke format file audio dan video yang berbeda. File PSS dapat dibuka dengan aplikasi seperti pemutar media VideoLAN VLC, PSS Plex, dan PSS Player.

## Format File PSS - Informasi Lebih Lanjut

PSS adalah file biner yang detail format file internalnya tidak tersedia untuk umum. Meskipun informasi struktur file PSS tidak tersedia, utilitas tertentu dapat membagi konten file PSS menjadi format file video MPEG2 dan audio WAV yang terpisah.
