## Apa itu file FOMOD?

File FOMOD adalah file game yang berisi modifikasi yang akan diinstal untuk sebuah game. File-file ini diinstal oleh program mod seperti Fallout mod manager (FOMM) dan Nexus Mod Manager (NMM). Modifikasi dalam file .fomod dapat dikaitkan dengan aspek apa pun dari game, dan diterbitkan oleh pemain atau penggemar komunitas game tersebut. Mod berupa FOMOD dapat memperpanjang umur sebuah game dengan melakukan perubahan termasuk perubahan kecil atau perombakan total pada game tersebut.

## Format File FOMOD - Informasi lebih lanjut

File FOMOD biasanya disimpan sebagai arsip terkompresi menggunakan format kompresi [.7z](/id/compression/7z/), [.zip](/id/compression/zip/), atau [.rar](/id/compression/rar/). Di FOMM, file ini dimuat dan diinstal pada sistem.

## Bagaimana cara menginstal file .fomod?

File FOMOD dapat diinstal dengan mengklik dua kali. Ini akan membukanya menggunakan FOOM dengan nama mode dalam daftar yang dapat diaktifkan dengan memilih dan menekan opsi aktifkan.
