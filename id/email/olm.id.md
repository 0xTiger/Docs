## Apa itu file OLM?

File dengan ekstensi .olm adalah file Microsoft Outlook untuk Sistem Operasi Mac. File OLM menyimpan pesan email, jurnal, data kalender, dan jenis data aplikasi lainnya. Ini mirip dengan file [PST](/id/email/pst/) yang digunakan oleh Outlook di Sistem Operasi Windows. Namun, file OLM yang dibuat oleh Outlook untuk Mac tidak dapat dibuka di Outlook untuk Windows. Format file OLM diperkenalkan dengan Outlook 2011 untuk menyimpan data lokal Outlook. Satu file OLM dapat berisi banyak kotak surat, mengatur data dalam bentuk folder dan subfolder. File OLM dapat dibuka/dimuat di Microsoft Outlook 365 dan Outlook untuk Mac.

## Format File OL

OLM adalah file biner dengan tipe MIME Application/octet-stream. Spesifikasi teknis format file OLM tidak tersedia secara terbuka dari Microsoft dan tidak ada detail format yang tersedia untuk referensi pengembang. Namun, ada beberapa API yang dapat membaca format file OLM dan menulisnya ke file PST untuk digunakan di Sistem Operasi Microsoft Windows seperti Aspose.Email.
