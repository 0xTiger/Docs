## Apa itu file RPMSG?

File RPMSG adalah pesan email terenkripsi yang dibuat dengan aplikasi klien email Microsoft Outlook, aplikasi email dan kalender Microsoft. Itu dikirim sebagai lampiran terpisah untuk pesan email standar dan hanya dapat dibuka oleh pengguna tertentu. Ini membatasi akses file-file ini ke pengguna lain yang tidak ditakdirkan untuk menerima informasi dalam file-file ini. Umumnya file-file ini dikirim dengan nama file message.rpmsg dan dapat diunduh juga dari email, tetapi tidak akan berfungsi dalam kasus ini.

## Format File RPMSG

File RPMSG disimpan ke disk dalam format file biner. Namun, spesifikasi format filenya tidak tersedia secara publik untuk referensi pengembang karena enkripsi.
