Format file menjelaskan struktur data dalam file yang diikuti aplikasi untuk membaca atau membuat file tersebut.

Jenis file dapat dikelompokkan ke dalam kategori seperti format file vektor, format file spreadsheet, format file video, format file gambar, format file 3D, dan beberapa lainnya. Sebuah file juga dapat diubah menjadi format file lain seperti BMP ke PNG.
