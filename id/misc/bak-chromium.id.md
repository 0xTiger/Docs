## Apa itu file BAK?

File .bak dalam konteks browser web berbasis Chromium, seperti Google Chrome dan Microsoft Edge, pada dasarnya adalah file cadangan bookmark Anda. Bookmark ini disimpan sebagai daftar teks biasa, dan format file yang digunakan adalah JSON. Tujuan dari file .bak ini adalah untuk melindungi bookmark Anda dengan menyediakan cadangan yang dapat dipulihkan jika terjadi penghapusan atau kehilangan yang tidak disengaja.

Browser berbasis Chromium, termasuk Google Chrome, secara rutin membuat file cadangan ini dan biasanya memberi nama "Bookmarks.bak". Mereka pada dasarnya adalah cuplikan dari bookmark Anda pada titik waktu tertentu.

## Cara menggunakan file .bak untuk memulihkan bookmark Anda

1. **Temukan file cadangan:** Biasanya terletak di folder tertentu yang terkait dengan profil Chromium Anda. Lokasi tepatnya dapat bervariasi tergantung pada sistem operasi Anda.

Di Windows: Lihat di `C:\Users\<YourUsername> \AppData\Local\Chromium\Data Pengguna\Default\Bookmarks.bak`.

Di macOS: Cari di `~/Library/Application Support/Chromium/Default/Bookmarks.bak`.

Di Linux: Centang `~/.config/chromium/Default/Bookmarks.bak`.

3. Pastikan browser Chromium Anda ditutup.
4. Ganti nama file "Bookmarks.bak" dengan menghapus ekstensi ".bak", sehingga cukup disebut "Bookmarks".
5. Mulai Chromium.

Dengan mengganti nama file .bak menjadi "Bookmark", Chromium akan menggunakannya sebagai file bookmark saat ini, dan bookmark Anda sebelumnya akan dipulihkan.

## Cadangan Bookmark Chromium

Mencadangkan bookmark Chromium adalah tindakan pencegahan yang bijaksana untuk memastikan Anda tidak kehilangan halaman web dan URL penting yang disimpan. Chromium adalah browser sumber terbuka yang berfungsi sebagai basis untuk browser lain seperti Google Chrome dan Microsoft Edge. Untuk mencadangkan bookmark Chromium Anda, ikuti langkah-langkah berikut:

## Metode 1: Pencadangan Manual

1. **Buka Chromium**: Luncurkan browser Chromium di komputer Anda.

2. **Akses Bookmark**: Klik pada tiga titik vertikal (ikon menu) di sudut kanan atas jendela browser. Dari menu tarik-turun, arahkan kursor ke "Bookmark" untuk membuka submenu bookmark.

3. **Ekspor Bookmark**: Di submenu bookmark, klik "Manajer bookmark". Ini akan membuka tab baru yang menampilkan bookmark Anda.

4. **Ekspor Bookmark**: Di tab pengelola bookmark, klik lagi tiga titik vertikal (biasanya di pojok kanan atas) untuk membuka menu pengelola bookmark. Dari menu ini, pilih "Ekspor bookmark".

5. **Pilih Lokasi**: Pilih tempat Anda ingin menyimpan file bookmark yang diekspor di komputer Anda. Nama file default adalah "bookmarks.html", namun Anda dapat mengubahnya jika diinginkan. Simpan ke lokasi yang akan Anda ingat.

6. **Simpan**: Klik tombol "Simpan" atau "Ekspor" untuk menyimpan bookmark Anda sebagai file HTML.

Bookmark Anda kini telah dicadangkan sebagai file HTML. Anda dapat menyalin file ini ke drive eksternal atau penyimpanan cloud untuk diamankan.

## Metode 2: Sinkronisasi dengan Akun Google (jika ada)

Jika Anda masuk ke akun Google di Chromium, Anda juga dapat menggunakan fitur sinkronisasi bawaan untuk menjaga bookmark Anda tetap aman dan tersinkronisasi di seluruh perangkat. Dengan cara ini, bookmark Anda akan secara otomatis dicadangkan ke akun Google Anda.

1. **Buka Chromium**: Luncurkan browser Chromium dan pastikan Anda masuk ke akun Google Anda.

2. **Aktifkan Sinkronisasi**: Klik pada tiga titik vertikal (ikon menu) di sudut kanan atas jendela browser dan buka "Pengaturan."

3. **Sinkronisasi dan layanan Google**: Di tab Setelan, klik "Sinkronisasi dan layanan Google" di sidebar kiri.

4. **Sinkronkan Data Anda**: Pada "Sinkronisasi", pastikan "Bookmark" diaktifkan. Ini akan menyinkronkan bookmark Anda dengan akun Google Anda.

Bookmark Anda sekarang akan secara otomatis dicadangkan dan disinkronkan dengan akun Google Anda. Anda dapat mengaksesnya dengan masuk ke akun Google Anda di perangkat apa pun dengan Chromium atau browser lain yang mendukung sinkronisasi Google.

Ingatlah untuk mencadangkan bookmark Anda secara manual secara berkala, terutama jika Anda menginginkan salinan lokal yang tidak hanya bergantung pada sinkronisasi akun Google Anda.

## Cara membuka file BAK

Jika Anda ingin menjelajahi data mentah cadangan bookmark Anda, Anda dapat membuka file "Bookmarks.bak" dengan berbagai editor teks seperti Microsoft Visual Studio Code (tersedia di berbagai platform), Microsoft Notepad (untuk pengguna Windows), Apple TextEdit (untuk pengguna Mac), atau editor teks lain pilihan Anda. Melakukannya akan memungkinkan Anda melihat bookmark dan metadata yang terdapat dalam file.

Anda dapat membuka file "Bookmarks.bak" dan melihat isinya menggunakan berbagai perangkat lunak pengedit teks dan editor kode. Berikut beberapa opsi yang umum digunakan:

1. Kode Visual Studio (Kode VS)
2. Buku Catatan (Windows)
3. Edit Teks (Mac)
4. Teks Luhur
5. Buku Catatan++
6. atom
7.nano dan Vim (Linux)
8. WordPad (Windows)
