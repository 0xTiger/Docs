## Apa itu file BUP?

BUP adalah cadangan file IFO. Ini menyimpan informasi yang mirip dengan file IFO sehingga pemutar DVD dapat menggunakan file ini jika file IFO asli tidak dapat dibaca. File BUP memberi tahu pemutar DVD tentang menu startup, lokasi trek video dan audio, dan awal bab pada DVD. File BUP dapat dibuka dengan VLC Player.

## Format File BUP - Informasi Lebih Lanjut

DVD bisa rusak karena kemungkinan goresan pada permukaannya. Ini menyulitkan drive DVD untuk membacanya. Jika [IFO](/id/video/ifo/) tidak dapat dibaca karena tergores, pemutar DVD tidak dapat memutar disk tersebut. Dalam kasus tersebut, pemutar media menggunakan file BUP yang merupakan cadangan sebenarnya dari file IFO.
