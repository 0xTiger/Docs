## Apa itu file MD5?

File MD5 adalah file checksum yang digunakan untuk verifikasi integritas file. Ini mirip dengan sidik jari untuk file terkait dan dihasilkan secara unik menggunakan algoritme yang menggunakan jumlah bit dalam file. Ini digunakan untuk memverifikasi file dengan perangkat lunak seperti [md5sum](http://www.gnu.org/software/coreutils/manual/html_node/md5sum-invocation.html). Salah satu keuntungan menggunakan file MD5 adalah memverifikasi bahwa file yang diunduh tidak rusak.

## Format File MD5 - Informasi Lebih Lanjut

Biasanya file MD5 disimpan dengan nama yang sama dengan nama file aslinya tetapi dengan ekstensi .md5. Misalnya, jika nama file terkait adalah `abc_987_123456.grb`, nama file MD5 yang sesuai adalah `abc_987_123456.grb.md5`. File MD5 membantu mengidentifikasi bahwa file yang diterima atau diunduh tidak rusak atau terpengaruh oleh konten berbahaya. Singkatnya, file MD5 menjamin kelengkapan file.


## Bagaimana cara memeriksa Checksum MD5?

Satu file dapat diperiksa/diverifikasi untuk MD5 menggunakan alat [md5sum](https://en.wikipedia.org/wiki/Md5sum) sebagai berikut.

```
md5sum -c abc_987_123456.grb.md5
```
