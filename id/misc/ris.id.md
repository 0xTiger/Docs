## Apa itu file RIS?

File RIS adalah format file tag untuk pertukaran data bibliografi dengan program kutipan. Ini berisi beberapa baris data kutipan, terdiri dari kode dua karakter dan nilai terkait yang dibatasi oleh tanda hubung. Informasi kutipan yang disertakan dalam file RIS mencakup data seperti penulis, judul, tanggal publikasi, kata kunci, penerbit, abstrak, alamat penulis, dll. File RIS dapat dibaca manusia dan dapat dibuka dengan editor teks apa pun seperti Microsoft Notepad, Notepad++, dan Apple TextEdit.

## Format File RIS - Informasi Lebih Lanjut

File RIS disimpan sebagai file teks biasa. Setiap baris dalam file RIS terdiri dari kode dua huruf (juga dikenal sebagai tag), diikuti oleh dua spasi dan tanda hubung, serta nilai yang sesuai. Karena tag, file RIS dikenal sebagai format tag untuk representasi kutipan bibliografi. Contoh record dalam file RIS adalah seperti gambar di bawah ini.

|DA - Juli|
|TI - Teori Komunikasi Matematika|

|EP - 423|

Dalam contoh ini, `TY` adalah Jenis Referensi dan `ER` adalah Akhir Referensi.
