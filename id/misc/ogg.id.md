## Apa itu file OGG?

File OGG adalah file grafik yang dibuat dengan perangkat lunak Origin. Ini memberikan kemampuan untuk menyimpan berbagai jenis grafik untuk menampilkan data yang diimpor untuk analisis data. Jenis grafik yang didukung oleh jenis file OGG meliputi kolom, batang, pai, air terjun, 3D, kutub, dan simbol. Perangkat lunak asal dikembangkan oleh OriginLab dan memungkinkan pengguna melakukan analisis data dan menghasilkan keluaran sebagai grafik.

## Format File OGG

File OGG berisi grafik untuk representasi data yang diimpor. Ketika file OGG dibuka, itu menampilkan grafik di jendela terpisah di Origin atau Origin Viewer yang tersedia secara bebas dibandingkan dengan perangkat lunak Origin. Informasi metadata seperti judul dan warna grafik juga disimpan di dalam file OGG.

## Tentang perangkat lunak OriginLab Origin

Perangkat lunak asal juga menyediakan opsi untuk digunakan sebagai Server Otomasi. Dalam kasus seperti itu, aplikasi klien misalnya LabVIEW, Microsoft Excel, Python, atau aplikasi yang dikembangkan khusus dalam C#/VB/VC dapat bertukar data dengan Origin. [Python_to_OriginLab](https://github.com/jsbangsund/python_to_originlab) adalah API Python untuk mengotomatiskan pengiriman data dan fungsi perencanaan dasar di OriginLab.
