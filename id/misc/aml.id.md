## Apa itu file AML?

File AML (Automation Markup Language) adalah file data yang digunakan untuk menyimpan dan bertukar informasi rekayasa pabrik. Itu didasarkan pada format file XML, membuat pertukaran informasi berdasarkan standar terbuka. AutomationML memberikan kemungkinan untuk menghubungkan alat teknik modern di bidang teknik mesin pabrik, desain listrik, pengontrol logika yang dapat diprogram, dan kontrol robot. File AML dapat dilihat dan diedit menggunakan alat AutomationML Editor.

## Format File AML

AML menggunakan pemodelan data berorientasi objek berbasis [XML](/id/web/xml/) untuk penyimpanan dan pertukaran model teknik. Komponen engineering plant direpresentasikan sebagai objek yang memiliki aspek yang berbeda, dengan masing-masing objek memiliki kemampuan untuk bertindak sebagai sub-objek. Dengan demikian, setiap model yang lebih kecil atau lebih besar dapat dijelaskan. [Spesifikasi AutomationML](https://www.automationml.org/about-automationml/specifications/) memberikan panduan singkat untuk memulai detail bahasa pemodelan ini.

## Memulai dengan AML

Menggunakan AML untuk pertukaran informasi teknik membutuhkan pengetahuan tertentu untuk aplikasi yang sukses. Implementasi aplikasi baru memerlukan beberapa langkah untuk tujuan ini seperti di bawah ini.

* Tahu arsitektur
* Mendapatkan pengetahuan dasar
* Memeriksa kasus penggunaan
* Pemetaan dan pemodelan khusus aplikasi
* Penerapan
