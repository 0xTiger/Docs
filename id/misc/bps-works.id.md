## Apa itu file BPS?

File .BPS pada dasarnya adalah cadangan dokumen pengolah kata yang dibuat dengan Microsoft Works, yang biasanya menggunakan ekstensi file .WPS untuk dokumennya. File .BPS ini sering kali dibuat secara otomatis oleh Microsoft Works sebagai tindakan pencegahan jika program mogok saat Anda sedang mengerjakan dokumen. Ini adalah cara untuk melindungi pekerjaan Anda dan mencegah kehilangan data.

Di bagian berikut, kita akan menjelajahi aplikasi perangkat lunak yang terkait dengan file BPS.

## Microsoft Bekerja

Microsoft Works adalah rangkaian perangkat lunak produktivitas yang dikembangkan oleh Microsoft. Ini pertama kali diperkenalkan pada tahun 1987 dan mendapatkan popularitas sebagai alternatif yang lebih terjangkau untuk paket lengkap Microsoft Office. Microsoft Works menyertakan beberapa aplikasi perkantoran dasar, seperti:

- **Pengolah Kata:** Microsoft Works memiliki aplikasi pengolah kata yang memungkinkan pengguna membuat dan mengedit dokumen. Ini menggunakan format file [.WPS](/id/word-processing/wps/) untuk dokumen pengolah kata.

- **Spreadsheet:** Ini termasuk aplikasi spreadsheet untuk membuat dan mengelola spreadsheet. Format file untuk spreadsheet adalah [.XLR](/id/spreadsheet/xlr/).

- **Database:** Microsoft Works juga memiliki aplikasi database untuk mengatur dan mengelola data. Format file untuk file database adalah [.WDB](/id/database/wdb/).

- **Kalender dan Buku Alamat:** Selain aplikasi perkantoran inti, Microsoft Works sering kali menyertakan alat untuk mengelola kalender dan buku alamat.

- **Works Word Processor Format (.WPS):** Format file asli komponen pengolah kata Microsoft Works. File .WPS berisi teks, pemformatan, dan grafik sederhana.

- **Penghentian:** Microsoft secara resmi menghentikan Microsoft Works pada tahun 2007 dan mendorong pengguna untuk beralih ke Microsoft Office atau alternatif lain.
