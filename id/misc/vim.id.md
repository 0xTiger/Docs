## Apa itu file VIM?

File VIM adalah file konfigurasi/pengaturan yang digunakan oleh program pengeditan teks yang dikenal sebagai Vim. Ini menyimpan pengaturan editor seperti penyorotan sintaks dan digunakan oleh pengembang aplikasi untuk mengonfigurasi tampilan editor saat dibuka. Editor teks Vim sangat dapat dikonfigurasi dan dibangun untuk membuat dan mengubah teks secara efisien. Muncul dibundel sebagai "vi" dengan sebagian besar sistem UNIX.
