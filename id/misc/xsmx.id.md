## Apa itu file XSMX?

File XSMX adalah file kuesioner yang dibuat dengan aplikasi ExamSoft SofTest yang digunakan untuk melakukan ujian. Ini digunakan untuk mengirimkan kertas ujian kepada siswa yang mereka ambil secara online dan mengirimkan kembali hasilnya sebagai file XMDX. ExamSoft memberikan solusi penilaian berdasarkan keputusan data-driver melalui teknologi pendidikan dan aplikasi SofTest yang tersedia untuk Windows dan MacOS adalah salah satunya. File XSMX berisi semua soal dan dibuka di aplikasi SoftTest di komputer siswa.

## Format File XSMX

File XSMX disimpan dan dikirim ke peserta ujian sebagai file yang dilindungi kata sandi. Perlindungan kata sandi dimaksudkan untuk memastikan keamanan dan kerahasiaan pertanyaan ujian. Kata sandi ujian dibagikan dengan pengguna akhir untuk membuka file dan mengikuti ujian.
