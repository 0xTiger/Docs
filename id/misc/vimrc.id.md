## Apa itu file VIMRC?

File VIMRC adalah file pengaturan yang digunakan oleh editor teks Vim. Ini menyimpan pengaturan untuk editor yang dimuat saat editor dibuka. Pengembang aplikasi dan administrator sistem menentukan pengaturan yang terkait dengan tampilan teks seperti penyorotan sintaks yang disimpan dalam file VIMRC. Dengan demikian, file VIMRC memungkinkan pengguna untuk menyesuaikan pengaturan editor.

## Format File VIMRC

File VIMRC disimpan sebagai file teks biasa dan dapat dibuka/diedit di editor teks apa pun seperti Notepad, Notepad ++, dan TextEdit. Mereka berisi instruksi yang dapat dijalankan di editor Vim dengan memulai dengan ":", diikuti dengan perintah. File VIMRC berisi instruksi untuk pengaturan seperti lekukan teks, dan warna jendela dan font. Format file yang relevan adalah [VIM](/id/misc/vim/).
