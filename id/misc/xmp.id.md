## Apa itu file XMP?

File XMP adalah file metadata yang berisi pengeditan yang dilakukan pada file mentah kamera, seperti [CR2](/id/image/cr2/) atau [NEF](/id/image/nef/). Mereka dibuat dan disimpan secara otomatis di direktori yang sama di samping gambar JPEG yang sesuai. XMP dibuat oleh Adobe Systems Inc. untuk pembuatan, pemrosesan, dan pertukaran metadata standar dan khusus untuk dokumen digital dan kumpulan data. Itu diterima sebagai **standar ISO 16684-1:2019 bagian 1 & ISO 16684-2:2014 bagian 2**. XMP adalah merek dagang terdaftar dari Adobe Systems Incorporated.

## Format File XMP - Informasi Lebih Lanjut

XMP adalah format serialisasi berdasarkan pendekatan model data. Ini menetapkan prinsip untuk definisi dan pemrosesan metadata. Format ini dapat digunakan untuk menyematkan informasi XMP ke berbagai jenis file seperti gambar, video, dan dokumen, misalnya [JPEG](/id/image/jpeg/) dan [PDF](/id/pdf/).

## Apa yang ada di dalam file XMP?

File XMP berisi informasi metadata tentang berbagai aspek gambar mentah. Ini mendukung data seperti:

* Informasi Penulis dan Hak Cipta
* Informasi properti tampilan gambar seperti kecerahan, pencahayaan, kontras, sudut pangkas, bayangan, saturasi, ketajaman, rona, dan suhu
