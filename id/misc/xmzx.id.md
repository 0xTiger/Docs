## Apa itu berkas XMZX?

File XMZX adalah file tes ujian untuk melakukan tes dan digunakan oleh aplikasi perangkat lunak ExamSoft SofTest. Aplikasi ini dapat digunakan di Windows dan MacOS. Ini berisi pertanyaan-pertanyaan dari ujian yang dapat diambil oleh penguji dari siswa di komputer dengan sfotware SoftTest. File XMZX dikirim ke penguji untuk melakukan tes. Itu dimuat dalam perangkat lunak SofTest dan kandidat atau pelamar mengikuti tes. Setelah pengujian selesai, file XMDX yang diselesaikan secara otomatis diunggah ke server cloud ExamSoft.

## Format File XMZX - Informasi Lebih Lanjut

File XMZX digunakan oleh platform penilaian digital, ExamSoft yang digunakan oleh berbagai organisasi untuk melakukan tes sertifikasi. Platform ini memiliki server cloud online sendiri tempat semua tes yang telah selesai diunggah secara otomatis. Tes dilakukan di pusat ujian atau penguji juga dapat mengambil formulir tes calon di komputer mereka sendiri. ExamSoft tersedia di sistem operasi Windows dan macOS. XSMX adalah format file tes ujian lainnya dari ExamSoft.

## Bagaimana Cara Membuka file XMZX?

Anda dapat membuka file XMZX di perangkat lunak ExamSoft SofTest hanya karena ini adalah format file berpemilik. Perangkat lunak SoftTest memiliki opsi untuk **Ikuti Ujian** yang digunakan untuk memulai ujian dengan memilih file XMZX.
