## Apa itu file CTG?

File CTG adalah file katalog yang dibuat pada kamera digital Canon. Ini dibuat saat foto diambil dengan kamera digital Canon. Ini menyimpan informasi tentang jumlah foto/gambar di setiap folder pada perangkat penyimpanan kamera yaitu kartu memori. File CTG disimpan dalam folder CANONMSC pada kartu memori. File-file ini membiarkan komputer Anda mengindeks file setelah dimasukkan ke dalam komputer.

## Format File CTG - Informasi lebih lanjut

File CTG disimpan dalam format file berpemilik kanon. Ini dihasilkan secara otomatis dan tidak boleh diedit secara manual. Kamera menghapus file ini saat foto terkait dihapus oleh pengguna.
