## Apa itu file H5?

H5 adalah salah satu Format Data Hierarkis (HDF) yang digunakan untuk menyimpan data dalam jumlah besar. Ini digunakan untuk menyimpan sejumlah besar data dalam bentuk array multidimensi. Format ini terutama digunakan untuk menyimpan data ilmiah yang tertata dengan baik untuk pengambilan dan analisis cepat. H5 diperkenalkan sebagai format file yang lebih disempurnakan ke H4. Awalnya dikembangkan oleh National Center for Supercomputing Applications, dan sekarang didukung oleh The HDF Group.

## Sejarah Singkat Format File HDF

Format file HDF dipilih oleh NASA sebagai metode standar untuk menyimpan data ilmiah. Sebelum ini, perumusan standar semacam itu dimulai pada tahun 1987 oleh Gugus Tugas Yayasan Grafik (GFTF) dan HDF secara resmi disetujui setelah penyelidikan terhadap 15 format file yang berbeda.

## Format File HDF5

File H5 dalam Format Data Hirarkis yang mematuhi spesifikasi format file HDF5. Spesifikasi ini meletakkan keseluruhan struktur file HDF5 untuk penyimpanan pada disk, tetapi pengguna akhir tidak memerlukan detail mendasar ini.

File H5 biasanya digunakan di ruang angkasa, fisika, teknik, keuangan, penelitian akademik, genomik, astronomi, instrumen elektronik, dan bidang medis.

### Model Data HDF5

File HDF5 seperti wadah yang terdiri dari objek data yang heterogen. Misalnya, kumpulan data ini dapat berupa gambar, tabel, elemen teks, grafik, dan bahkan dokumen seperti [PDF](/id/pdf/) dan [dokumen Excel](/id/spreadsheet/).

## Penampil File H5

Grup HDF menyediakan [perangkat lunak HDFView](https://www.hdfgroup.org/solutions/hdf5/) yang terdiri dari utilitas HDFView dan Java HDF Object Package.
