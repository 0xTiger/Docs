## Apa itu file ART?

File ART adalah file desain bordir yang dibuat dengan software bordir ARTlink [BERNINA](https://www.bernina.com/en-US/Home-United-States) versi 1-4. Ini berisi desain yang dibuat pengguna yang dapat menyertakan objek berbeda seperti garis besar, tata letak desain, jahitan, benang berwarna, dan metadata lainnya. File ART dimuat ke dalam mesin jahit BERNINA dan dapat dijahit ke berbagai jenis kain seperti selimut, kemeja, atau kain lainnya. File ART juga dapat dibagikan dengan pengguna lain menggunakan perangkat penyimpanan seperti USB atau ditransfer melalui internet.

## Format File ART

Format file ART berisi informasi berikut:

* Properti Objek
* Jahitan Data
* Jenis Jahitan
* Koordinat Jahitan
* Warna Benang
* Komentar
* Pengaturan Kompensasi Tarik
* Pengaturan Jarak Otomatis
