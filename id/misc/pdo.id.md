## Apa itu berkas PDO?

Pepakura Designer adalah program yang memungkinkan pengguna membuat model kerajinan kertas dari model 3D; File Pepakura biasanya memiliki ekstensi .pdo; file-file ini berisi instruksi dan informasi tata letak yang diperlukan untuk mencetak dan merakit model kertas.

File Pepakura Designer berada dalam format kepemilikan khusus untuk perangkat lunak Pepakura; format file dirancang untuk menyimpan informasi tentang model 3D, termasuk geometri, referensi tekstur, dan instruksi perakitan.

Pepakura Designer bekerja dengan model 3D dalam format seperti [.obj](/3d/obj/), [.3ds](/3d/3ds/), [.dxf](/cad/dxf/) dan lainnya; model-model ini kemudian dibuka menjadi pola 2D yang dapat dicetak dan dirangkai menjadi model kertas akhir.

File .pdo berisi instruksi tentang cara memotong, melipat, dan merekatkan potongan kertas yang dicetak untuk membuat ulang model 3D; setiap bagian diberi label dan nomor untuk memandu proses perakitan.

## Bagaimana cara membuka file PDO?

Untuk membuka dan melihat file Pepakura Designer, Anda memerlukan **Pepakura Viewer**, program pendamping gratis untuk **Pepakura Designer**. Viewer memungkinkan orang lain untuk melihat dan mencetak model tanpa memerlukan perangkat lunak desain lengkap.

## Tentang Desainer Pepakura

Pepakura Designer adalah aplikasi perangkat lunak khusus yang dirancang untuk membuat model kerajinan kertas dari model digital 3D. Perangkat lunak ini memungkinkan pengguna untuk mengimpor model 3D dalam berbagai format, seperti .obj atau .3ds, lalu membuka dan meratakan model tersebut menjadi pola 2D yang dapat dicetak di atas kertas. Pola-pola ini, disimpan dengan ekstensi file .pdo, berisi instruksi dan tanda untuk memotong, melipat, dan merakit potongan kertas menjadi model tiga dimensi. Pepakura Designer banyak digunakan oleh para penghobi dan peminat yang tertarik untuk membuat model kertas rumit dari berbagai objek, karakter, dan desain.
