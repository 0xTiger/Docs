## Apa itu file H4?

H4 atau HDF4 adalah salah satu dari Hierarchical Data Formats (HDF) dan digunakan untuk menyimpan data dalam jumlah besar. Ini adalah format file sumber terbuka yang mendukung penyimpanan data heterogen yang kompleks. HDF4 adalah format data yang lebih lama dibandingkan dengan [HDF5](/id/misc/h5/). Format file HDF dikelola dan diterbitkan oleh grup HDF, sebuah organisasi nirlaba. File HDF juga mendukung penyimpanan metadata untuk sepenuhnya mewakili data yang disimpan.

## Sejarah Singkat Format File HDF

Formulasi Format Data Hirarkis dimulai pada tahun 1987 oleh Gugus Tugas Yayasan Grafika (GFTF). Selama ini, beberapa format file berbeda diusulkan untuk standarisasi format tunggal. NASA memilih HDF sebagai metode standar untuk menyimpan dan menyajikan data ilmiah.

## Format File HDF4

File H4 disimpan dalam "Direktori file" seperti struktur sesuai spesifikasi format file HDF. Spesifikasi ini menentukan keseluruhan struktur file HDF untuk penyimpanan pada disk tetapi tanpa pengguna mengkhawatirkan struktur file yang mendasarinya.

Area aplikasi file H4 meliputi kedirgantaraan, fisika, teknik, keuangan, penelitian akademik, genomik, astronomi, instrumen elektronik, dan bidang medis.

### Model Data HDF4

File HDF4 atau H4 seperti wadah yang terdiri dari objek data yang heterogen. Jenis dataset yang didukung oleh file H4 meliputi:

* Gambar-gambar
* Tabel
* Elemen teks
* Grafik
* Dokumen seperti [PDF](/id/pdf/) dan [dokumen Excel](/id/spreadsheet/)

## Penampil File H4

Grup HDF menyediakan [perangkat lunak HDFView](https://www.hdfgroup.org/solutions/hdf5/) yang terdiri dari utilitas HDFView dan Java HDF Object Package.
