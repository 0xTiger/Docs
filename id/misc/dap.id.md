## Apa itu file DAP?

File DAP adalah file yang sedang diunduh yang dibuat oleh file Download Accelerator Plus (DAP). Itu dibuat ketika pengguna memulai pengunduhan dari repositori file online apa pun menggunakan perangkat lunak DAP. DAP menyediakan kemampuan untuk menjeda dan melanjutkan unduhan yang sedang berlangsung. Untuk tujuan ini, ini menyimpan file yang diunduh sebagian dengan ekstensi .dap hingga diunduh sepenuhnya. File DAP dapat dipratinjau selama proses pengunduhan.

File DAP dapat dibuka dengan perangkat lunak SpeedBit Download Accelerator Plus (DAP).
