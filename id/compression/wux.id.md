## Apa itu file WUX?

File WUX adalah versi terkompresi dari file Wii U Disk Image [WUD](/id/disc-and-media/wud/). Itu dihasilkan menggunakan alat kompresi gambar Wii U untuk mengurangi ukuran file dari file tersegmentasi WUD yang berukuran sekitar 20 GB. File informasi game berukuran besar ini dikurangi menjadi sekitar 2 GB dalam file WUX terkompresi yang membuat transfer file lebih mudah. File WUX dapat dibuka dan dimainkan dengan emulator video game seperti [Cemu](https://cemu.info/).

## Format File WUX

File WUX disimpan sebagai file terkompresi yang dapat didekompresi untuk dimuat di emulator menggunakan alat kompresi gambar Wii U. Emulator video game Cemu juga dapat memuat file WUX dan memuat informasi video game lengkap.
