## Apa itu file SFG?

File SFG adalah format file yang digunakan oleh Synfig Studio untuk menyimpan file proyek. Synfig Studio adalah perangkat lunak animasi 2D sumber terbuka dan gratis yang memungkinkan pengguna membuat animasi berkualitas tinggi menggunakan bitmap dan grafik vektor. File SFG berisi semua informasi yang diperlukan untuk membuat ulang animasi, termasuk lapisan, pengaturan waktu, dan bingkai utama untuk setiap lapisan. Ini juga mencakup gambar atau aset apa pun yang digunakan dalam proyek, serta trek audio atau media lainnya.

## Format Berkas SFG

File SFG disimpan ke disk dalam format file ZIP terkompresi. Format file lain yang didukung oleh Proyek Terkompresi [Synfig Studio](https://www.synfig.org/) meliputi [SIFZ](/compression/sifz/) dan SIF.
