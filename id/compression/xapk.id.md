## Apa itu file XAPK?

File dengan ekstensi .xapk adalah file paket terkompresi yang digunakan untuk menginstal aplikasi Android di perangkat seluler. Ini adalah format file kontainer yang menggabungkan APK dan file terkait tambahan yang diperlukan untuk penginstalan. File terkait lainnya adalah file OBB yang menyimpan file tambahan seperti grafik, file media, dan data aplikasi yang diperlukan agar aplikasi tetap berjalan. File XAPK tidak didukung oleh Google Play dan hanya digunakan untuk mendistribusikan di situs web unduhan aplikasi Android pihak ketiga. Ini dapat diinstal ke perangkat Android menggunakan Penginstal XAPK.

## Format File XAPK

File XAPK dikompres menggunakan format file [ZIP](/id/compression/zip/) standar. Ini dapat diekstraksi menggunakan perangkat lunak kompresi/dekompresi standar seperti WinZIP. Setelah file XAPK diekstraksi ke disk, itu berisi file-file berikut di folder.

* APK - File instalasi standar untuk menginstal aplikasi pada perangkat Android
* OBB - File tambahan yang berisi file sumber daya yang relevan
