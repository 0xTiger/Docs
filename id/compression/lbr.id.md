## Apa itu file LBR?

File dengan ekstensi .lbr adalah file arsip yang dibuat dengan program LU dan digunakan pada sistem operasi CP/M dan DOS selama tahun 1980-an. Itu tidak lagi digunakan dan telah digantikan oleh format file pengarsipan modern. Formatnya tidak memampatkan file anggota dan bertindak sebagai wadah arsip untuk ini. Fitur pengarsipan sebagian besar digunakan untuk transfer file arsip melalui internet. Karena LBR tidak menawarkan kompresi, utilitas lain seperti SQ atau CRUNCH digunakan untuk mengompres pra-pengarsipan file anggota atau file arsip yang dihasilkan pasca-pengarsipan untuk mengurangi ukurannya.

## Format File LBR

Format file LBR dirancang oleh Gary P. Novosielski dan detail teknisnya tidak tersedia untuk umum. File LBR dimulai dengan 0x00 byte, diikuti dengan 11 spasi (0x20), lalu dua 0x00 byte, lalu dua byte yang bukan keduanya 0x00. Menjadi format file kontainer, dapat diekstraksi menggunakan LU.COM dan NULU.COM.
