## Apa itu file PUP Linux?

File PUP Linux adalah paket instalasi yang digunakan oleh sistem operasi Puppy Linux. Itu dijalankan sebagai penginstal dan mulai menginstal file paket. Di Linux, file ini sering disebut sebagai file DotPup dan mirip dengan file penginstal [.exe](/id/executable/exe/) dan [.msi](/id/executable/msi/) yang digunakan di OS Windows. Versi Puppy Linux yang lebih baru menggunakan format file .pet daripada format file PUP.
