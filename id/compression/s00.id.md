## Apa itu file S00?

File dengan ekstensi .s00 adalah bagian pertama dari file terkompresi yang dipisahkan menggunakan perangkat lunak ZipSplitter. Memisahkan file menjadi file yang lebih kecil akan mengurangi ukuran keseluruhan arsip terkompresi dan memudahkan transportasi melalui internet. Bagian yang tersisa dari arsip terpisah diberi nama dengan ekstensi .S01, .S02 dan seterusnya. Saat file seharusnya diekstrak, file .EXE yang disertakan dengan file split diklik dua kali. EXE mencari file .S00 dan menggunakan file lain yang disertakan untuk mengekstraksi seluruh arsip.

## Format File S00

File S00 disimpan ke disk sebagai arsip terkompresi biner. Namun, detail format file internal dari format file S00 tidak tersedia untuk umum. File besar seperti file .[WMV](/id/video/wmv/) atau [.MP4](/id/video/mp4/) paling sering dipisahkan menggunakan perangkat lunak ZipSplitter untuk memperkecil ukurannya.
