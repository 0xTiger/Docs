## Apa itu file MPQ? ##

File dengan ekstensi .mpq terutama milik Blizzard Game oleh Blizzard Entertainment. Spesifikasi utama dari setiap file MPQ adalah grafik terkompresi, suara, data level, dll. Jika kita berbicara tentang arsip MPQ, ini pertama kali dibiasakan dengan game Diablo dan sekarang sangat populer di banyak game. File MPQ adalah wadah sumber biner besar untuk peta level, maket item, musik plus video, dll.

## Format File MPQ ##

Format file MPQ pertama kali dikembangkan oleh Mike O'Brien untuk menyimpan data game. Sejak Diablo, MPQ telah digunakan di banyak game yang diterbitkan oleh Blizzard Entertainment seperti Diablo II, StarCraft, Warcraft II dan III, World of WarCraft, StarCraft II.
