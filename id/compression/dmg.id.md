## Apa itu file DMG?

File dengan ekstensi .dmg adalah file Gambar Disk Apple dan dikenal sebagai file Gambar Disk Mac OS X. Format file DMG adalah file gambar disk yang paling banyak digunakan yang dimaksudkan untuk mengalokasikan aplikasi dan file di antara PC Apple. Sangat nyaman untuk memasang file DMG ke drive virtual atau dapat menggunakan perangkat lunak Apple Disk Utility untuk membuka dan melihat konten format file DMG. Versi lama file IMG yang digunakan di Mac OS Classic diganti dengan format file DMG. File DMG dibuat untuk pengguna Mac dan tidak didukung oleh OS Windows tetapi pengguna dapat mengakses konten DMG di sistem Operasi Windows dengan menggunakan alat kompresi dan dekompresi seperti 7-Zip, Pea Zip, dll. Ekstensi file DMG juga digunakan oleh alat Ekspor dan Masuknya Oracle untuk file scrapheap yang disimpan dalam format biner dari database Oracle.

## Sejarah Singkat

Ekstensi file DMG dikembangkan oleh Apple dan sebagian besar digunakan di komputer Macintosh. Sebagian besar file DMG digunakan untuk memberi perangkat Mac kemampuan untuk menginstal berbagai jenis file dan perangkat lunak yang dimulai dari internet. File DMG pada dasarnya adalah gambar disk yang dapat dipasang yang ditampilkan di desktop Anda saat dibuka. File tersebut terdiri dari data mentah yang umumnya dienkripsi dan dikompresi. Sistem Mac memperlakukan DMG dengan cara yang sama seperti mereka memperlakukan disk yang disisipkan, dan mereka secara instan mencoba membuka atau "melacak" file tersebut.

## Spesifikasi Teknis Format File DMG

* Khusus dibuat untuk komputer Macintosh
* Dikembangkan oleh Apple
* Format file adalah Biner
* Kategori File adalah File Gambar Disk
* Program yang Didukung adalah MacOS
* Dapat berisi tabel partisi
* Dapat berisi segala jenis sistem file seperti HFS+
* Fungsionalitas untuk mengompres gambar
