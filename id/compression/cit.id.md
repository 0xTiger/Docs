## Apa itu file CIT?

File CIT adalah file proyek yang dihasilkan oleh aplikasi penulisan laporan investigasi berbasis kutipan, In8 Cite. Berisi rincian kejadian dalam bentuk laporan dan juga memiliki katalog yang berisi dokumentasi pendukung dan bukti-bukti yang dikutip dalam laporan tersebut. Penyidik kriminal menggunakan file CIT untuk merujuk kasus kriminal selama penyelidikan mereka.

## Format File CIT

File CIT disimpan ke disk dalam format file terkompresi. Ia menggunakan algoritma kompresi ZIP standar untuk mengompres file. File CIT dan dokumen pendukungnya dapat dikonversi dan diekspor ke file [PDF](/pdf/) yang dapat dibagikan kepada orang lain untuk ditinjau.

### Tentang Dalam8

In8 adalah perusahaan perangkat lunak yang memiliki keahlian dalam mengembangkan aplikasi yang digunakan untuk mendokumentasikan dan melaporkan laporan investigasi kriminal.
