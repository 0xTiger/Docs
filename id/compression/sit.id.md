## Apa itu file SIT?

File SIT adalah file arsip terkompresi yang dibuat dengan perangkat lunak Micro StuffIt. Ini adalah format file standar asli yang diperkenalkan oleh StuffIt. Format file SIT umumnya digunakan pada sistem Macintosh, tetapi sekarang didukung pada platform Windows dan Macintosh. StuffIt memperluas format file SIT ke [SITX](/id/compression/sitx/) yang menawarkan lebih banyak kompresi dibandingkan dengan file SIT. File SIT dapat dibuka dengan perangkat lunak Smith Micro StuffIt Deluxe Mac 16.

## Format File SIT - Informasi Lebih Lanjut

File SIT disimpan dalam format file biner dan detail format file internalnya tidak tersedia untuk umum. Itu diperkenalkan oleh Sistem Aladdin dan diperkenalkan untuk mengompres file ke arsip.
