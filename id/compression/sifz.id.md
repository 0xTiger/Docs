## Apa itu file SIFZ?

File SIFZ adalah file SIF terkompresi gzip yang dibuat oleh aplikasi pembuat animasi 2d open source **Synfig Studio**. Ini berisi beberapa elemen grafis yang membentuk animasi. Keseluruhan isi animasi dibangun menggunakan kombinasi kanvas yang diisi dengan bentuk, sapuan kuas atau pensil, dan teks. Semua ini diatur pada posisinya sendiri, jari-jari, garis singgung, sudut, titik sudut, dan pegangan lebar. File SIFZ dapat dibuka dengan [Synfig Studio](https://www.synfig.org/).

## Format File SIFZ

File SIFZ adalah file [ZIP](/id/compression/zip/) terkompresi yang dikemas menggunakan metode kompresi gzip. Synfig digunakan untuk membuat animasi berkualitas film menggunakan karya seni vektor dan bitmap. Tidak seperti metode pembuatan animasi frame-by-frame yang lama, ini memungkinkan Anda menghasilkan animasi 2D dengan kualitas lebih tinggi dengan sumber daya yang lebih sedikit.

File SIFZ, yang dikompresi, lebih kecil dari file SIF yang tidak dikompresi. Ini juga memudahkan untuk mentransfer melalui koneksi internet bandwidth rendah.
