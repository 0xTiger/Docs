## Apa itu file HALAMAN?

File PAGES adalah dokumen kata yang dibuat dengan aplikasi halaman Apple yang merupakan pengolah kata yang dikembangkan oleh Apple Inc. Dokumen yang disimpan sebagai file PAGES dapat berisi berbagai jenis konten yang berbeda seperti laporan, poster, brosur, buletin, dan buku. Apple PAGES juga menyediakan sejumlah template berbeda untuk membuat jenis dokumen ini. Konten file PAGES dapat berisi teks, gambar, grafik, tabel, dan bagan. Apple PAGES berjalan di sistem operasi macOS, iPadOS, dan iOS.

## Format File HALAMAN

File PAGES disimpan sebagai file [ZIP](/id/compression/zip/) yang dapat diubah namanya menjadi ekstensi ZIP dan dibuka dengan utilitas pengarsipan untuk melihat kontennya. Ini termasuk file JPG yang dibundel dalam arsip yang dapat digunakan untuk melihat pratinjau halaman pertama dokumen. Ini juga berisi file PDF yang digunakan untuk melihat seluruh dokumen.

## Bagaimana Mengkonversi Dokumen PAGES ke WORD

Untuk mengonversi file Halaman Apple ke PDF, langkah-langkah berikut dapat digunakan.

1. Buka Aplikasi Halaman
1. Buka dokumen yang perlu diubah menjadi dokumen WORD
1. Gunakan opsi Ekspor dan pilih format Word
1. Simpan dokumen dalam format Word

## Cara Mengonversi File PAGES

Dokumen Apple PAGES dapat dikonversi ke beberapa format file. Ini dapat dilakukan dari dalam perangkat lunak Halaman Apple menggunakan opsi **Ekspor Ke**. File PAGES dapat dikonversi ke:

* Microsoft Word [DOC](/id/word-processing/doc/) dan [DOCX](/id/word-processing/docx/)
* [PDF](/id/pdf/)
* [EPUB](/id/ebook/epub/)
* [RTF](/id/word-processing/rtf/)
* [TXT](/id/word-processing/txt/)
