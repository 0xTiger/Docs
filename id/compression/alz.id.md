## Apa itu file ALZ? ##

File dengan ekstensi .alz adalah file terkompresi, terutama terkait dengan ALZip oleh ESTsoft, Corp. Format file ALZ sebagai file terkompresi juga didefinisikan sebagai arsip ALZip. Pengembang telah mengembangkan alat kompresi file yang disebut ALZip dan teknologi pengkodean dan kompresi yang sama digunakan oleh format file ALZ. Perangkat lunak ini dibuat oleh ESTsoft untuk distribusi pribadi di bawah versi Microsoft Windows yang banyak digunakan. Seiring waktu, alat kompresi dan dekompresi file ALZip menyebarkan ketentuannya ke lebih dari 36 catatan modul yang berbeda dan program kompresi serta dukungan untuk algoritme ZIP.

## Sejarah Singkat ##

Format file ALZ dirilis pada tahun 1999. ALZip dan format file ALZ umumnya tidak membatasi ukuran file dan batas ekspansi. Perangkat lunak ini mengizinkan Sistem Windows untuk menentukan batasan ukuran untuk file keluarannya.
