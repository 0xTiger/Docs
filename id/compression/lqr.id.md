## Apa itu file LQR? ##

Jenis file LQR terutama terkait dengan file arsip terkompresi yang mencakup satu atau lebih file yang telah diringkas dalam ukuran file agar mudah didistribusikan melalui Internet atau melalui media yang dapat ditransfer seperti disk optik seperti DVD dan CD di antara penyimpanan data eksternal lainnya. perangkat. File apa pun yang disimpan dalam format File LQR dapat digabungkan dengan ketentuan kompresi yang dijalankan oleh algoritme kompresi tertentu. File dalam format LQR disematkan dengan ekstensi .lqr. Sekarang, ekstensi LQR telah diganti dengan standar kompresi dan fitur enkode yang lebih banyak digunakan, termasuk format [RAR](/id/compression/rar/) dan [ZIP](/id/compression/zip/).

## Sejarah Singkat ##

Pada awal 1980-an, format file [LBR](/id/compression/lbr/) adalah format file arsip yang digunakan pada sistem operasi CP/M dan MS-DOS. File LBR secara khas membatasi file yang ditangani oleh SQ, atau arsip itu sendiri dikompres dengan SQ. File arsip LBR terkompresi diberi ekstensi LQR atau LZR.
