## Apa itu file TZ?

File TZ adalah arsip TAR yang dikompresi dengan algoritma kompresi Unix. Ini adalah kombinasi dari arsip [TAR](/id/compression/tar/) dan file terkompresi [Z](/id/compression/z/), dan merupakan format kompresi asli untuk Unix OS. File TZ dapat dibuka menggunakan utilitas kompresi standar seperti WinZIP dan WinRAR.

## Format File TZ

File TZ disimpan sebagai file biner menggunakan algoritma kompresi UNIX. Penggunaan file Z membuat pemformatan, menjalankan, dan membuka file menjadi lebih mudah. File TZ berukuran lebih kecil karena kompresi Z mencapai kompresi tinggi. File yang dihasilkan menempati lebih sedikit ruang pada disk dan mudah ditransfer melalui media komunikasi.
