## Apa itu file ES?

File dengan ekstensi .ice adalah arsip terkompresi yang dibuat dengan utilitas perangkat lunak [ICEOWS](http://www.iceows.com/index.html) (Interface de Compression Ergonomique pour windowsOWS) yang sebelumnya dikenal sebagai ArjFolder. Ini memberikan kompresi yang sangat baik untuk mengurangi ukuran file teks. ICEOWS awalnya dikembangkan dengan dukungan bahasa Polandia, tetapi sekarang tersedia dalam bahasa lain juga termasuk Inggris, Prancis, Jerman, Polandia, Portugis, Spanyol, Italia, Cina, Ibrani, Ukraina, dan Swedia. Ini juga memungkinkan Anda mengekstrak konten file arsip dan mendukung ICE, ARJ, [ZIP](/id/compression/zip/), [GunZip](/id/compression/gz/), [TAR](/id/compression/tar/), Microsoft CAB , [RAR](/id/compression/rar/), ACE, MIME, Mac HQX, UUEncode, XXEncode, Base64, [JAR](/id/programming/jar/), EAR, WAR, LHA, IMP, [BZ2](/id/compression /bz2/) format file.

## Format File ES

ICE adalah file biner tetapi tidak ada spesifikasi teknis yang tersedia tentang detail format file internalnya.
