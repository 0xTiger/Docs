## Apa itu berkas P7Z?

File P7Z adalah file terkompresi yang berisi pesan email dalam standar enkripsi S/MIME. Itu diperkenalkan oleh Microsoft Corporation dan dikategorikan sebagai file data terkait Email. File P7Z memiliki tipe media application/pkcs7-mime. Ini berisi satu atau lebih bagian pesan yang dikompresi dengan algoritma kompresi zlib.

## Format File P7Z - Informasi Lebih Lanjut

Detail format file internal format file P7Z tidak diketahui. Beberapa API seperti [Aspose.Email](https://products.aspose.com/email/) memungkinkan Anda membaca bagian S/MIME dari pesan. Jika klien email tidak memiliki kemampuan untuk menangani data kompresi, pesan S/MIME terkompresi akan dikirimkan sebagai lampiran email.
