## Apa itu file ASV?

File ASV adalah file pengaturan Warna Selektif Adobe Photoshop. Ini berisi pengaturan warna untuk preset nilai CMYK (Cyan, magenta, kuning, dan hitam) yang dapat diterapkan ke gambar raster seperti [PNG](/id/image/png/) dan [BMP](/id/image/bmp/). File ASV dapat diekspor dan disimpan untuk dibagikan dengan pengguna lain melalui internet. Ini dapat dimuat di Adobe Photoshop melalui dialog Selective Color dengan menggunakan pilihan Image->Adjustments->Selective Color. Setelah dimuat, preset ini juga dapat dimodifikasi menggunakan Adobe Photoshop. File ASV dapat dimuat menggunakan
metode relatif atau absolut.

## Format File ASV - Informasi Lebih Lanjut

File ASV disimpan dalam format file biner dan isinya tidak dalam bentuk yang dapat dibaca manusia.
