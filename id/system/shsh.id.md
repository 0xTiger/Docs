## Apa itu file SHSH?

File SHSH, juga dikenal sebagai Signature HaSH, adalah tanda tangan digital yang digunakan oleh Apple untuk mengautentikasi dan memverifikasi pembaruan firmware untuk perangkat iOS seperti iPhone, iPad, dan iPod.

## Perbedaan Format File SHSH dan SHSH2

Seperti file SHSH2, file SHSH berisi pengidentifikasi unik untuk perangkat yang disebut "ECID" (ID Chip Eksklusif), serta informasi tentang versi firmware yang diinstal pada perangkat. Namun, file SHSH digunakan di iOS versi lama (sebelum iOS 10) dan sejak itu digantikan oleh file SHSH2.

## Format File SHSH - Informasi Lebih Lanjut

File SHSH disimpan ke disk dalam format file biner. Detail struktur file internal format file ini tidak tersedia untuk umum.

File SHSH penting bagi pengguna yang ingin menurunkan versi firmware perangkatnya ke versi sebelumnya yang tidak lagi ditandatangani oleh Apple. Dengan menyimpan file SHSH untuk versi firmware tertentu saat masih ditandatangani oleh Apple, pengguna nantinya dapat menggunakannya untuk melewati verifikasi tanda tangan Apple dan menginstal versi firmware tersebut di perangkat mereka. Proses ini juga dikenal sebagai `jailbreaking`.
