File Sistem adalah jenis file integral yang ditemukan di sistem operasi Microsoft Windows, Mac, dan Linux. File sistem adalah salah satu file terpenting yang disimpan dalam sistem operasi perangkat karena berisi semua informasi terkait sistem perangkat, dan tanpanya, perangkat pasti tidak berfungsi atau tidak beroperasi sama sekali. Untuk alasan ini, file sistem sangat dilindungi dengan mekanisme izin sehingga file tersebut tidak dapat dihapus secara tidak sengaja atau bahkan sengaja dari komputer dengan mudah. Untuk spesifikasi dan identifikasi yang mudah, file-file ini selalu disimpan di folder sistem tertentu.

File sistem ditemukan dalam berbagai jenis ekstensi file, dan setiap sistem operasi memiliki jenis ekstensi file sistem yang unik untuk tujuan yang berbeda. Ekstensi file sistem yang paling umum adalah; [CAB](/id/system/cab/), SYS, [DLL](/id/system/dll/), dan DRV.
