## Apa itu file YPR?

File YPR adalah file proyek yang dibuat oleh lingkungan pemrograman visual BYOB (Build Your Own Blocks). Ini berisi informasi tentang pengaturan visual blok dan alur kerja pemrograman di antara blok-blok ini. Bersama-sama, blok-blok ini menentukan keseluruhan data dan logika program aplikasi. File proyek YPR berisi semua informasi terkait lainnya untuk menyimpan dan memuat proyek. BYOB awalnya merupakan perpanjangan dari perangkat lunak Scratch MIT (proyek Lifelong Kindergarten Group di MIT Media Lab) dan sekarang dikembangkan sebagai [Snap](https://snap.berkeley.edu/about).

## Format File YPR - Informasi Lebih Lanjut

File YPR disimpan ke disk sebagai file biner dan detail format file internalnya tidak diketahui. Proyek BYOB menyenangkan membangun di mana blok diseret dan dijatuhkan untuk mengembangkan alur kerja pemrograman.

### Tipe Data YPR

Snap memperluas kelas pertama daftar Scratch, memungkinkan untuk membuat struktur datanya (pepohonan, tumpukan, tabel hash, kamus, dan sebagainya). Tipe data kelas satu dalam bahasa pemrograman dapat berupa:

* nilai variabel
* anggota agregat (array, daftar, dll.)
* argumen (masukan) ke prosedur
* nilai yang dikembalikan oleh prosedur
* digunakan tanpa nama (menjadi nilai variabel)
