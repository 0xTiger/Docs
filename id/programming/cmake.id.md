## Apa itu file CMAKE?

File CMAKE adalah file proyek yang dibuat oleh alat lintas platform sumber terbuka **CMake**. Ini berisi skrip sumber dalam bahasa CMake untuk seluruh program. Alat CMake menggunakan skrip pemrograman, yang dikenal sebagai CMakeLists, untuk menghasilkan file build untuk lingkungan tertentu seperti mesin makefiles Unix. Skrip dan modul yang digunakan oleh proyek CMake diatur sebagai script.cmake dan module.cmake yang direferensikan dalam proyek.
