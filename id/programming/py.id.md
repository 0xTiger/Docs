## Apa itu file PY? ##

File dengan ekstensi .py berisi kode sumber Python. Bahasa Python telah menjadi bahasa yang sangat terkenal sekarang. Ini dapat digunakan untuk skrip sistem, pengembangan web dan perangkat lunak, dan matematika. Python mendukung kompatibilitas lintas platform; artinya aplikasi yang dikembangkan dengan Python dapat bekerja pada platform yang berbeda seperti Windows, MAC, Linux, Raspberry Pi, dll. Python menyediakan sintaks yang sederhana dan mudah dibaca yang mirip dengan bahasa Inggris. Pengembang dapat menulis aplikasi perangkat lunak yang masuk akal dengan menulis beberapa baris kode Python. Karena Python berjalan pada sistem juru bahasa, maka kode dapat dieksekusi segera setelah ditulis yang membuatnya sangat bagus untuk pembuatan prototipe.

## Sejarah Singkat ##

Python dikandung pada akhir 1980-an oleh Guido van Rossum sebagai penerus bahasa pemrograman ABC. Implementasinya dimulai pada Desember 1989 dengan Van Rossum sebagai satu-satunya pengembang utama. Dia mengerjakan python hingga 12 Juli 2018. Pada Januari 2019, pengembang inti Python yang aktif memilih "Dewan Pengarah" beranggotakan lima orang yang terdiri dari Nick Coghlan, Brett Cannon, Carol Willing, Barry Warsawa, dan Van Rossum untuk memimpin proyek tersebut.

## Cara menjalankan file py ##

Untuk memeriksa versi Python yang diinstal, Anda dapat menggunakan perintah berikut:

```cmd
python --version
```

Ini akan menampilkan versi di konsol seperti

```cmd
Python 3.7.4
```

Jika Python belum terinstal di mesin Anda, Anda dapat membuka [python.org](https://www.python.org/) dan mendownload serta menginstal Python untuk sistem operasi Anda yang relevan.

Untuk mengeksekusi skrip Python, Anda dapat menggunakan perintah berikut:

```cmd
python helloworld.py
```

*helloworld.py* adalah file skrip yang berisi kode berikut

```py
print("Hello World from Python")
```

Ini akan mencetak yang berikut di jendela konsol.

```cmd
Hello World from Python
```

Catatan: Jika Anda menggunakan IDE, mereka menyediakan tombol di layar atau pintasan keyboard yang berbeda untuk menjalankan Python. Misalnya, tombol putar ditampilkan di selokan dengan editor di PyCharm yang memungkinkan Anda menjalankan skrip Python.

## Format File PY ##

Python dirancang agar mudah dibaca dan memiliki kemiripan dengan Bahasa Inggris dan Matematika. Python menggunakan baris baru untuk menunjukkan perintah lengkap sebagai kebalikan dari titik koma atau tanda kurung yang digunakan dalam bahasa lain. Untuk cakupan, putaran, dan fungsi, Python bergantung pada lekukan dan spasi putih sebagai kebalikan dari kurung kurawal yang digunakan dalam bahasa lain.

### Sintaks ###

Berikut ini adalah contoh sintaks Python.

```py
print("Hello World")

#Variables
name = "John"
age = 25
print(name)
print(age)

#While Loop
i = 1
while i < 3:
  print(i)
  i += 1
  
#For Loop
names = ["John", "Harry", "Tom"]
for name in names:
  print(name)
```
