## Apa itu file ASI? ##

File ASI adalah format file variabel yang digunakan oleh perangkat lunak Alpha Five yang digunakan untuk mengembangkan aplikasi Web. Ini menyimpan variabel yang digunakan oleh aplikasi Web. Saat Anda membuat proyek pengembang baru di Alpha Five, file ASI juga dibuat dan disimpan bersama proyek pengembang Alpha Five.

File ASI dapat dibuka dengan Alpha Software Alpha Anywhere.

## Format File ASI

File ASI disimpan dalam format file berpemilik Alpha Five. Mereka dibaca dan ditulis oleh aplikasi sebagai bagian dari proyek pengembang.
