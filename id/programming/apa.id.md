## Apa itu file APA?

File dengan ekstensi .apa adalah file proyek pengembangan perangkat lunak yang dibuat oleh aplikasi RSView. Ini berisi informasi yang diperlukan untuk membuat aplikasi Human Machine Interface (HMI). File APA disimpan dalam format file arsip. Format file arsipnya membantu mentransfer ini dari satu komputer ke komputer lain dengan mudah karena ukuran file yang lebih kecil. Utilitas dekompresi standar dapat digunakan untuk mengekstrak konten file APA. File APA dapat dibuka menggunakan perangkat lunak Rockwell Automation RSView Studio atau RSView ME.

## Format File APA - Informasi Lebih Lanjut

Manajer aplikasi RSView menyimpan file .apa secara default ke lokasi C:\Documents and Settings\All Users\Documents\RSView Enterprise\ME\Archives di Windows XP. Namun, Anda dapat menyimpan file ke direktori mana pun sesuai kebutuhan.
