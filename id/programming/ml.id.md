## Apa itu file ML?

File ML adalah file teks yang dibuat dengan bahasa pemrograman fungsional tujuan umum ML (sering dikenal sebagai Meta Language). Ini berisi kode sumber ML yang lebih sering digunakan dalam penelitian dan desain bahasa pemrograman menggunakan semantik yang didefinisikan sepenuhnya. Ada beberapa variasi ML dengan Caml sebagai yang banyak digunakan. Lainnya termasuk Objective Caml (OCaml), JoCaml, Extended ML, dan MacroML. Bahasa ML umumnya digunakan untuk penulisan kompiler, pembuktian teorema otomatis, dan verifikasi formal.
