## Apa itu file C3?
File C3 berisi kode sumber yang ditulis dalam bahasa pemrograman C3. C3 adalah bahasa pemrograman eksperimental yang bertujuan untuk menggabungkan fitur terbaik C++ dan Python.

File kode sumber C3 dapat dibuat menggunakan editor teks misalnya Notepad atau Sublime Text. Mereka dapat diedit dan dikompilasi menggunakan kompiler C3 yang menerjemahkan kode sumber C3 menjadi kode yang dapat dieksekusi yang dapat dijalankan di komputer. Kompiler C3 menghasilkan kode mesin yang dapat dijalankan langsung di CPU sehingga memberikan kinerja yang baik.

File kode sumber C3 dapat berisi berbagai konstruksi pemrograman termasuk fungsi, kelas, variabel, loop, dan kondisional. C3 juga menyertakan sejumlah tipe data bawaan seperti bilangan bulat, angka floating-point, dan string.

Untuk mengkompilasi file kode sumber C3, Anda harus menginstal kompiler C3 di komputer Anda. Setelah kompiler diinstal, Anda dapat menjalankannya dari baris perintah dan menentukan lokasi file kode sumber yang ingin Anda kompilasi. Kompiler akan menghasilkan file yang dapat dieksekusi yang kemudian dapat Anda jalankan di komputer Anda.

## Apa format file C3?

File kode sumber C3 adalah file teks biasa, formatnya mengikuti aturan file teks biasa, dengan setiap baris berisi urutan karakter yang membentuk kode. Kode C3 mengikuti sintaks yang mirip dengan C++ dan Python. File kode sumber C3 biasanya berisi fungsi, kelas, variabel, loop, kondisional, dan konstruksi pemrograman lainnya. Kode dalam file C3 disusun menjadi serangkaian pernyataan dengan setiap pernyataan biasanya menempati satu baris file.

## Apa sajakah ciri-ciri bahasa pemrograman C3?

Beberapa fitur bahasa pemrograman C3 adalah

- Kesederhanaan
- Pengetikan yang kuat
- Pemrograman berorientasi objek
- Generik
- Pengumpulan sampah
- Pemrograman meta
- Interoperabilitas

Anda memerlukan editor teks atau lingkungan pengembangan terintegrasi (IDE) untuk membuka file C3. Beberapa opsi populer termasuk Visual Studio Code, Sublime Text, Atom dan IntelliJ IDEA.
