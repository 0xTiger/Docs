## Apa itu file HTA?

HTMLA singkatan dari **Hypertext Markup Language Application** adalah program yang kompatibel dengan Microsoft Windows. Kode sumber program ini menyertakan lebih dari satu bahasa skrip seperti [HTML](/id/web/html/) dan [JavaScript](/id/web/js/). Untuk antarmuka pengguna, Aplikasi HTML lebih disukai sementara untuk memenuhi persyaratan logika program, bahasa skrip lain digunakan.

Aplikasi HTML tidak bergantung pada model keamanan browser internet dan berjalan sebagai aplikasi yang sepenuhnya tepercaya. Ekstensi yang digunakan untuk file terkait aplikasi ini adalah HTA. Aplikasi ini menyertakan fitur HTML bersama dengan properti bahasa scripting lainnya.


## Sejarah Singkat ##

HTA pertama kali diperkenalkan pada tahun 1999 oleh Microsoft bersamaan dengan peluncuran Internet Explorer 5. HTA ini kompatibel dengan Internet Explorer dan hanya dapat dijalankan pada sistem operasi Windows. Teknologi ini dipatenkan pada tahun 2003. File HTA dijalankan sama seperti file .exe lainnya. File HTA juga kompatibel dengan versi Windows 11 yang diperbarui hari ini.


## Spesifikasi Teknis ##

HTA memiliki format yang sama dengan halaman HTML lainnya, sementara beberapa atribut digunakan untuk mengontrol gaya batas atau ikon program. Selain itu, argumen diberikan untuk peluncuran HTA. Aplikasi ini dapat dijalankan dengan menggunakan program bernama mshta.exe. Itu dapat diakses hanya dengan mengklik dua kali pada file. Program-program ini secara otomatis berjalan bersama dengan penjelajah Internet. Selain spesifikasi lainnya, ini tidak terpisah dari browser mesin Trident tetapi tidak tergantung pada Internet Explorer. Artinya, ini dapat dijalankan tanpa menggunakan Internet Explorer.

Tag digunakan untuk kepentingan penyesuaian tampilan aplikasi ini. Konversi dari aplikasi Microsoft HTML ke format HTA lebih mudah yaitu hanya perlu mengubah ekstensi. Seperti yang kita ketahui bahwa aplikasi ini sepenuhnya tepercaya sehingga ini memiliki lebih banyak fitur dan keunggulan dibandingkan dengan file HTML sederhana. Editor teks dapat digunakan untuk membuat HTA. Editor ini dapat diperoleh oleh Microsoft atau sumber tepercaya lainnya.


## Contoh Format File HTA ##

```
<HTML>
<HEAD>
<HTA:APPLICATION ID="HelloExample" 
   BORDER="bold" 
   BORDERSTYLE="complex"/>
<TITLE>HTA - Hello World</TITLE>
</HEAD>
<BODY>
<H2>HTA - Hello World</H2>
</BODY>
</HTML>

```
