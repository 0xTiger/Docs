## Apa itu file ACD?

File ACD adalah file proyek yang berisi informasi untuk pemrograman [Rockwell Automation](https://www.rockwellautomation.com) platform Logix. Ini termasuk instruksi untuk pengontrol dan prosesor. Ini digunakan untuk platform Logix seperti ControlLogix, CompactLogix, FlexLogix, SoftLogix5800, dan sistem DriveLogix.

Aplikasi yang bisa membuka file ACD antara lain Rockwell Automation RSLogix 5000.
