## Apa itu file EX?

File EX adalah file kode sumber yang berisi kode sumber untuk aplikasi yang dikembangkan menggunakan bahasa pemrograman Euphoria. Euphoria adalah bahasa pemrograman prosedural yang percaya pada kesederhanaan pemrograman daripada bahasa pemrograman yang kompleks. Aplikasi yang dikembangkan di Euphoria dapat terdiri dari beberapa file EX, tetapi satu file EX juga dapat berisi sumber program yang lengkap.

Aplikasi yang dapat **membuka file EX** termasuk openEuphoria Euphoria, Microsoft Notepad, Notepad++, dan Atom.

## Sejarah Singkat Format File EX

Perkembangan dan keberadaan format file EX tidak terlepas dari perkembangan bahasa pemrograman Euphoria yang diciptakan oleh Robert Craig. Ini pertama kali dirilis untuk MS-DOS. Itu menjadi perangkat lunak sumber terbuka pada tahun 2006 dengan dirilisnya versi 3. Pengembangannya diambil alih oleh **kelompok openEuphoria** yang mengelola dan mengembangkan proyek. Versi 4 dirilis pada Desember 2010 dan tersedia untuk Windows, Linux, macOS, dan tiga versi BSD.

Euphoria awalnya ditulis dalam bahasa pemrograman **[C](/id/programming/c/)**. Belakangan, juru bahasa Euphoria terpecah menjadi dua bagian:

* `Front-end Parser` - ditulis dalam Euphoria dan digunakan dengan penerjemah Euphoria-to-C dan Binder.
* `Back-end` - run time library yang ditulis dalam bahasa pemrograman C.

## Format File EX - Informasi Lebih Lanjut

File EX disimpan sebagai file teks biasa dan berisi kode sumber untuk program yang ditulis dalam bahasa pemrograman Euphoria. File-file ini dapat dibuka di editor teks apa pun.
