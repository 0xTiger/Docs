## Apa itu file LGO?

LGO mengajukan kode program yang ditulis dengan aplikasi Logo untuk menggambar. Ini berisi instruksi terprogram yang memandu aplikasi membaca untuk membuat gambar sesuai dengan instruksi ini. Misalnya, program "kura-kura" yang dapat membuka file LGO, gunakan petunjuk ini untuk menggambar garis sesuai kode yang diprogram. Gambar tidak terbatas pada garis saja. File LGO dapat digunakan untuk menggambar desain sederhana hingga rumit. Banyak program pendidikan menggunakan file LGO untuk mengajarkan cara membuat gambar secara terprogram.

## Format File LGO

File LGO disimpan dalam format file berpemilik dan struktur file internalnya tidak diketahui publik. Format file terkenal lainnya untuk logo adalah file [.ico](/id/image/ico/).
