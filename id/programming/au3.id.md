## Apa itu file AU3?

File AU3 adalah file skrip otomatis yang dibuat dengan program skrip freeware, [AutoIt v3](https://www.autoitscript.com/site/). Ini berisi bahasa skrip AutoIt3 untuk mengotomatiskan perintah dalam Windows. Bahasa pemrograman AutoIt3 memungkinkan Anda mensimulasikan sejumlah peristiwa yang dapat digunakan untuk mengotomatisasi tugas. Misalnya, skrip di dalam file AU3 dapat digunakan untuk mensimulasikan penekanan tombol, gerakan mouse, mengontrol posisi jendela, dan tujuan skrip lainnya.

Aplikasi yang dapat membuka file AU3 antara lain [SciTE4AutoIt3](https://www.autoitscript.com/site/autoit-script-editor/downloads/) dan AutoIt.

## Format File AU3

File AU3 disimpan sebagai file teks dan berisi bahasa skrip AutoIt3.
