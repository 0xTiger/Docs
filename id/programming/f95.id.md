## Apa itu berkas F95?

File dengan ekstensi .f95 biasanya menunjukkan bahwa itu adalah file kode sumber Fortran 95. Fortran adalah bahasa pemrograman penting dan bertujuan umum yang sangat cocok untuk komputasi numerik dan ilmiah. Fortran 95 adalah versi Fortran 90 yang disempurnakan, memperkenalkan beberapa fitur dan peningkatan baru.

File kode sumber Fortran biasanya berisi instruksi program yang ditulis dalam bahasa pemrograman Fortran; jika Anda memiliki file dengan ekstensi .f95, kemungkinan besar file tersebut dimaksudkan untuk dikompilasi menggunakan kompiler Fortran; kompilasi adalah proses menerjemahkan kode sumber yang dapat dibaca manusia menjadi kode biner yang dapat dibaca mesin dan dapat dijalankan oleh komputer.

## Bagaimana cara membuka file F95?

Jika Anda ingin bekerja atau menjalankan program dalam file .f95, Anda memerlukan kompiler Fortran; kompiler Fortran yang populer termasuk GNU Fortran (gfortran), Intel Fortran Compiler (ifort), dan lainnya. Anda dapat mengkompilasi dan menjalankan program Fortran menggunakan perintah yang sesuai yang disediakan oleh kompiler.

Misalnya, jika Anda telah menginstal gfortran, Anda dapat menggunakan perintah berikut di terminal untuk mengkompilasi dan menjalankan program Fortran:

```
# Compile the Fortran program
gfortran -o my_program my_program.f95

# Run the compiled program
./my_program
```

Jika Anda hanya ingin melihat isi file F95, itu adalah file teks biasa, sehingga Anda dapat membukanya dengan editor teks apa pun.
