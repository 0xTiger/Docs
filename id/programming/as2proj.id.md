## Apa itu file AS2PROJ?

File AS2PROJ adalah file proyek yang dibuat oleh IDE pemrograman Flash open source, FlashDevelop. Ini berisi informasi terkait proyek seperti pengaturan build, referensi ke file sumber proyek, dan opsi untuk menghasilkan output [file .swf](/id/page-description-language/swf/). Setiap proyek baru yang dibuat di FlashDevelop disimpan sebagai file AS2PROJ. Anda dapat mengubah pengaturan proyek Anda dan semua perubahan ini disimpan ke file AS2PROJ.

File AS2PROJ dapat dibuka dengan FlashDevelop.

## Format File AS2PROJ

File AS2PROJ disimpan dalam format file [XML](/id/web/xml/) yang merupakan format file universal untuk pertukaran informasi. Itu dapat dibuka di editor teks apa pun seperti Notepad, Notepad ++ dan Atom untuk memeriksa kontennya.
