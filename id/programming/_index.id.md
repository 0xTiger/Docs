File pemrograman adalah file kode sumber yang digunakan oleh bahasa pemrograman untuk menulis aplikasi perangkat lunak untuk melakukan tugas tertentu. Setiap bahasa pemrograman memiliki sintaksnya sendiri dan membawa kumpulan file sendiri untuk menulis kode dalam bentuk pustaka, file header, kelas, atau konstanta. Berdasarkan bahasa pemrograman yang digunakan, format file kode sumber akan berbeda untuk mematuhi aturan sintaksis yang ditentukan oleh bahasa tersebut.

Ekstensi file pengembang atau pemrograman umum dan format file terkait termasuk **[C#](/id/programming/cs/)** (File Kode Sumber CSharp), **[CPP](/id/programming/cpp/)** (Sumber C++ File) dan **[VB](/id/programming/vb/)** (File Kode Sumber Visual Basic).
