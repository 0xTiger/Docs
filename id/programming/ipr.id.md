## Apa itu file IPR?

File IPR adalah file proyek yang dibuat dengan Java IDE **IntelliJ IDEA**. Semua informasi yang terkait dengan pengaturan proyek seperti struktur direktori, file kode sumber, pustaka yang disertakan dalam proyek, dan pengaturan kompiler disimpan dalam file IPR. Saat proyek dibuka dari dalam IntelliJ IDEA, IDE membaca informasi dalam file IPR dan memuat file dan pengaturan terkait. File IPR telah digantikan oleh file .IDEA yang berisi semua informasi sebagai Direktori Pengaturan.

## Format File HKI

File IPR disimpan dalam format file [XML](/id/web/xml/) yang dapat dipahami secara universal dan dapat dibaca manusia. Ini berisi semua informasi terkait proyek dalam tag yang dimuat untuk menampilkan konten terkait proyek.

Karena file IPR disimpan dalam format file XML, ini dapat dibuka di editor teks apa pun seperti Notepad, Notepad ++, Atom, dan Apple TextEdit.
