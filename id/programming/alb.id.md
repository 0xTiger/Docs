## Apa itu file ALB?

File ALB adalah file pustaka yang digunakan oleh aplikasi IDE **Alpha Five** yang digunakan untuk mengembangkan aplikasi Web. Tujuan utamanya adalah untuk menyimpan informasi untuk [file ADB](/id/database/adb/) Alpha Five yang sesuai. Ini termasuk informasi kamus data seperti pengaturan, penunjuk skrip, menu, fungsi yang ditentukan pengguna, bitmap, bilah alat, dan definisi impor.

File ALB dapat dibuka dengan **Alpha Software Alpha Anywhere**.

## Format File ALB - Informasi Lebih Lanjut

File ALB disimpan dalam format file milik File Alpha dan struktur format file internalnya tidak tersedia untuk umum. Ini disimpan sebagai tambahan dari keseluruhan database perpustakaan pengembang Alpha Five. Menjadi pustaka, file ALB dapat digunakan kembali di aplikasi Alpha Five lainnya.
