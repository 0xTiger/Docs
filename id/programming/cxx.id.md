## Apa itu file CXX?

File CXX adalah file kode sumber, mirip dengan C++, untuk menulis aplikasi. Itu dapat berisi implementasi lengkap dari seluruh aplikasi atau dapat memiliki implementasi sebagian dari program yang dapat disertakan dalam proyek dengan file C++ lainnya. File CXX dapat dibuka di editor teks apa pun dan IDE pemrograman populer seperti Notepad ++, Turbo C ++ dan Visual Studio,

## Format File CXX

File CXX disimpan sebagai file teks biasa dan berisi kode yang ditulis dalam sintaks C++. File CXX umumnya ditemukan dengan ekstensi [.cpp](/id/programming/cpp/). File CXX dianggap sebagai alternatif dari .cpp yang digunakan oleh beberapa kompiler.
