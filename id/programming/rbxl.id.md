## Apa itu file RBXL?

File RBXL adalah file lokasi Roblox yang berisi lokasi yang ditentukan dalam dunia 3D di dalam [game Roblox](https://www.roblox.com/). Roblox sendiri adalah game membangun multipemain yang memungkinkan pemain membuat game menggunakan alat pengembang yang disediakan oleh platform. File RBXL menggunakan alat, skrip, tim, dan bagian untuk mendeskripsikan lapangan bermain sebagai model 3D. Roblox Studio memungkinkan pemain membuat item game dan dapat digunakan untuk membuka file RBXL.

## Format File RBXL

File RBXL dikembangkan menggunakan Roblox Studio dan disimpan dalam format file [XML](/id/web/xml/). Karena format file universal ini, ada banyak alat dan API yang memungkinkan Anda membuat dan membuka file ini. Salah satu API tersebut adalah [MaximumADHD](https://github.com/MaximumADHD/Roblox-File-Format), pustaka [C#](/id/programming/cs/) untuk membuat dan memanipulasi file dalam format file model/tempat Roblox .
