## Apa itu file TELUR?

File EGG, juga dikenal sebagai telur Python, adalah format distribusi distribusi Python yang lebih lama. Ini adalah arsip terkompresi [ZIP](/id/compression/zip/) dengan ekstensi .egg dan berisi file sumber aplikasi Python beserta informasi meta tentang distribusinya. File EGG adalah alternatif untuk file Windows Executable [EXE](/id/executable/exe/) tetapi bersifat lintas platform. Format lama untuk distribusi Python ini telah diganti dengan format file Wheel (WHL) yang lebih baru pada awal 2010.

## Format File TELUR

File EGG disimpan sebagai arsip ZIP terkompresi. Artinya jika Anda mengganti ekstensi .egg dengan .zip, Anda akan dapat membukanya dengan utilitas dekompresi standar seperti Corel WinZIP, Microsoft Explorer, atau RARLAB WinRAR.

File EGG dapat dibuat menggunakan paket **distutils** yang tersedia oleh Python. Alat lain yang dapat membuat dan membuka file EGG adalah **SetupTools**. File EGG dapat diinstal sebagai paket menggunakan **easy_install**.

`CATATAN - format file EGG telah usang dan diganti dengan format file WHL roda baru.`
