## Apa itu file MM?

File MM adalah file kode sumber yang dapat berisi kode pemrograman Objective-C dan Objective-C++. Ini digunakan pada MacOS dan berbeda dari file .M yang hanya berisi kode Objective-C. Saat kompiler melihat ekstensi .mm, ia dapat mengkompilasi kelas C++ yang ada di file. Karena satu file kode sumber dapat berisi campuran bahasa [C](/id/programming/c/) dan [C++](/id/programming/cpp/), mengganti nama file .m menjadi ekstensi .mm memungkinkan fitur C++ untuk mengatur di. Aplikasi yang bisa membuka file .MM antara lain Apple TextEdit, Microsoft Notepad, Notepad++, dan Microsoft Wordpad.
