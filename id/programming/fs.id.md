## Apa itu file FS?

Ekstensi file **.fs** dikaitkan dengan **Visual F#**, bahasa pemrograman yang dikembangkan oleh Microsoft sebagai bagian dari kerangka .NET. F# adalah bahasa pemrograman yang mengutamakan fungsi yang merupakan bagian dari ekosistem Microsoft .NET, dan dirancang untuk digunakan untuk berbagai jenis aplikasi, termasuk aplikasi web, desktop, dan cloud.

File Sumber Visual F# (dengan ekstensi .fs) berisi kode sumber yang ditulis dalam **bahasa pemrograman F#**. File-file ini dapat mencakup fungsi, modul, tipe, dan konstruksi bahasa F# lainnya. Visual Studio adalah lingkungan pengembangan umum untuk pemrograman F#, menyediakan alat untuk mengedit, mengkompilasi, dan men-debug kode F#.

Pemrogram menggunakan file .fs untuk menulis kode F# dan file ini kemudian dikompilasi menjadi file yang dapat dieksekusi ([.EXE](/executable/exe/)) atau pustaka, biasanya dengan ekstensi [.DLL](/system/dll/). Bahasa F# dikenal dengan sintaksisnya yang ringkas, kekekalan secara default, dan dukungan kuat untuk konsep pemrograman fungsional.
