## Apa itu file YML?

YML adalah file teks dalam bahasa serialisasi data, [YAML](/id/programming/yaml/) (Yet Another Markup Language), yang sering digunakan untuk membuat file konfigurasi. Itu dapat bekerja dalam kombinasi dengan bahasa pemrograman apa pun termasuk C/C++, Ruby, Python, Java, PHP, C#, dan lainnya. Menjadi independen dari bahasa pemrograman tertentu, itu terbuka, dapat dioperasikan, dan dapat dibaca manusia. File YML dapat dibuka dan diedit dengan editor teks umum seperti Microsoft Notepad, Notepad++, Github Atom, Apple TextEdit, dan Visual Studio IDE.

## Format File YML - Informasi Lebih Lanjut

File YML menyimpan pengaturan aplikasi sebagai konfigurasi. Struktur dasarnya adalah peta yang juga bisa disebut direktori atau hash. Entri dalam file YML terdiri dari pasangan kunci dan nilai. Sebagian besar, file .yml konfigurasi disimpan sebagai `configure.yml`, meskipun beberapa bahasa pemrograman seperti Ruby on Rails menyimpan konfigurasi konektivitas database dalam file `database.yml`. [Spesifikasi format file YML/YAML](https://yaml.org/spec/1.2.2/) menjelaskan struktur bahasa lengkap dan Informasi terkait sintaks.
