## Apa itu file ANE?

File ANE adalah file Adobe Air Native Extension yang digunakan oleh developer untuk menulis fungsi tertentu yang memanggil kode [Adobe ActionScript](https://en.wikipedia.org/wiki/ActionScript). Di sebagian besar kasus, file ANE berisi pustaka kode asli yang digunakan oleh ekstensi. Namun, file itu sendiri diinstal pada perangkat. Ini memungkinkan fungsi spesifik sistem seperti getaran perangkat. Aplikasi Adobe AIR harus dikompilasi untuk menggunakan tautan dinamis untuk menggunakan file ANE.

## Format File ANE - Informasi Lebih Lanjut

File ANE disimpan ke disk sebagai file biner yang spesifikasi format filenya tidak tersedia untuk umum. File ANE mungkin berisi kode sumber Java Android asli untuk mengakses fungsionalitas perangkat Android seperti menjalankan fungsi volume atau getar. Semua ini dilakukan dengan membungkus sistem asli dalam bahasa ActionScript Adobe.
