## Apa itu file SCM?

File SCM adalah file kode sumber yang dibuat dengan bahasa pemrograman Skema. Ini berisi kode yang ditulis dalam sintaks skema yang kemudian dieksekusi oleh juru bahasa Skema. Bahasa skema dikembangkan oleh Massachusetts Institute of Technology (MIT) pada tahun 1970 dan mendukung pemrograman fungsional dan teknik terkait seperti algoritma rekursif. File SCM dapat dibuka dan diedit di editor teks apa pun seperti Microsoft Notepad, Notepad ++, dan Apple TextEdit.
