## Apa itu file SCRIPT?

File SCRIPT adalah file sumber yang berisi skrip dalam bahasa skrip apa pun. Ini menjadikannya file skrip umum yang berisi variabel dan fungsi. Karena setiap bahasa skrip mungkin berbeda, skrip yang ditulis di dalam file SCRIPT perlu dijalankan menggunakan mesin skrip yang sesuai.

## Format File SCRIPT - Informasi Lebih Lanjut

File SCRIPT disimpan ke disk dalam file teks biasa. Ini dapat dibuka dan diedit dengan editor teks seperti Microsoft Notepad, Notepad++, dan Apple TextEdit. Karena bahasa scripting bisa berbeda, itu dapat dijalankan dengan mengganti nama ekstensi file ke ekstensi default masing-masing bahasa scripting. Misalnya, ada beberapa bahasa skrip umum seperti JavaScript ([.JS](/id/web/js/)), VBScript ([.VBS](/id/programming/vbs/)), AppleScript (.SCPT), PHP ([ .PHP](/id/programming/php/)), dan ASP ([.ASP](/id/web/asp/)).
