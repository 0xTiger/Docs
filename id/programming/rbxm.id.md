## Apa itu file RBXM?

File RBXM adalah format file Model 3D yang digunakan di Roblox Studio IDE untuk menyimpan objek 3D dari game Roblox multipemain daring. Ini berisi objek 3D yang dibuat untuk game menggunakan IDE. Contoh objek 3D dalam file RBXM antara lain baju, celana, topi, gedung, kendaraan, binatang, pesawat terbang, mobil, atau tanaman. File RBMX dapat dikonversi ke [format file OBJ 3D](/id/3d/obj/) menggunakan Roblox IDE. Roblox IDE dapat digunakan untuk membuka file RBMX.

Format file Roblox populer lainnya adalah [RBXL](/id/programming/rbxl/) yang merupakan format file lokasi dan berisi lokasi yang ditentukan dalam kata 3D di dalam game Roblox.

## Format File RBMX - Informasi Lebih Lanjut

File RBXM dibuat dan disimpan dalam format file biner dan spesifikasi format file internalnya tidak tersedia untuk umum. Ini berbeda dari format file RBXMX yang didasarkan pada format file [XML](/id/web/xml/). Ada banyak alat/repositori yang tersedia di Github yang dapat digunakan untuk [memproses Roblox RBXM](https://github.com/search?q=rbxm) file.
