## Apa itu file NX?

File NX adalah file video game yang dibuat untuk digunakan dengan konsol game virtual, **LowRes NX**. Itu ditulis dalam bahasa pemrograman BASIC dan berisi kode sumber game yang menentukan cara kerja game. Anda dapat membuat file NX Anda sendiri untuk digunakan dengan aplikasi yang dapat dimuat dan pengguna dapat memainkan game tersebut.

Anda dapat membuka file NX di editor teks, LowRes NX, atau editor kode sumber apa pun seperti Microsoft Visual Studio Code.

## Format Berkas NX

File NX disimpan ke disk sebagai file teks biasa yang berisi instruksi permainan. Setiap file NX memiliki kode sumber permainan yang terdiri dari beberapa entri ROM kartrid. Entri ROM ini memiliki informasi komponen game seperti grafik, peta level, audio, dan data kode non-sumber lainnya untuk game tersebut. Entri ROM suatu program dapat ditambahkan atau diedit menggunakan program pengeditan NX yang disertakan dengan LowRes NX.

## Bagaimana Cara Membuat File NX?

### Buat File NX Di PC

Anda dapat membuat file NX di PC menggunakan:

* Editor Teks - seperti Microsoft Notepad, Notepad++, atau Github Atom
* Editor Kode Sumber - seperti Microsoft VIsual Studio Code

### Buat File NX di iOS

LowRes NX memiliki versi iOS yang dapat digunakan oleh pengembang untuk membuat file NX dari dalam aplikasi ini.
