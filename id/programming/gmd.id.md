## Apa itu file GMD?

File GMD adalah file sumber pengembangan game yang dihasilkan dengan alat pengembangan perangkat lunak Game Maker yang digunakan untuk membuat game 2D. File GMD disimpan sebagai file teks dan merupakan file proyek untuk game yang akan dikembangkan. Game Maker tersedia untuk Windows, Mac, Linux, Android, iOS, HTML5, Xbox, PlayStation, dan Nintendo Switch. Alat ini telah banyak digunakan oleh pengembang indie, studio profesional, dan pendidik di seluruh dunia untuk membuat game.

## Format File GMD

File GMD disimpan dalam format file biner dan detail format file internalnya tidak tersedia untuk referensi pengembang.
