## Apa itu WEBLOC?

File WEBLOC adalah file pintasan di macOS yang mengarah ke URL web di internet. Itu dihasilkan dengan menyeret URL web dari bilah alamat browser ke desktop atau folder lain mana pun yang dimaksudkan untuk disimpan. URL ke situs tujuan disimpan di dalam file pintasan ini dan dapat dilihat dengan membuka file di editor teks seperti Microsoft Notepad, Notepad++, atau Apple TextEdit. File WEBLOC mirip dengan file [URL](/id/web/url/).

## Format File WEBLOC - Informasi Lebih Lanjut

File WEBLOC disimpan dalam format file teks biasa. Ini dapat dibuat di macOS dengan menyeret URL dari browser web seperti Safari, Google Chrome, atau Firefox. Jika beberapa file WEBLOC dibuka secara bersamaan, diperlukan solusi yang lebih cepat dan termudah untuk membukanya melalui aplikasi eksternal atau API. API ini dapat membaca dan mengonversi WEBLOC ke PDF dan JPEG.
