## Apa itu file CCBJS?

File CCBJS adalah file Javascript yang dibuat oleh aplikasi 3D, game, dan editor situs web, **CopperCube**. Ini berisi kode JavaScript yang dihasilkan sebagai bagian dari [file .html](/id/web/html/) yang dibuat dari file .ccb. CopperCube menyediakan antarmuka skrip yang ramah pengguna untuk mengembangkan logika dan interaksi gameplay tanpa menulis kode apa pun. Format file lain yang didukung oleh CopperCube termasuk .ccscene, .ccmodel, .ccmaterial, .cctex, .ccjs, .ccp, dan .ccpak.

## Format File CCBJS

Detail format file internal format file CCBJS tidak diketahui. Namun, file Javascript disimpan sebagai file teks biasa ke disk dan dapat dibuka dengan editor teks apa pun seperti Microsoft Notepad, Notepad++, dan Apple TextEdit.
