## Apa itu file CHA?

File CHA adalah file pengaturan pengguna yang dibuat oleh klien IRC seperti mIRC. Ini berisi parameter seperti server, port, nama saluran dan, secara opsional, kata sandi untuk membuka saluran IRC. Klien IRC digunakan untuk berkomunikasi dengan orang lain di jaringan IRC di seluruh dunia. Ini digunakan untuk diskusi pribadi satu-ke-satu atau dalam konferensi multi-grup. Ketika file CHA diklik, klien IRC secara otomatis terbuka dengan rincian yang disebutkan dalam file.

Aplikasi yang bisa membuka file CHA antara lain mIRC dan Visual IRC.

## Format File CH

File CHA disimpan bersama file instalasi lain dari klien IRC. Ketika seorang pengguna ingin terhubung ke server dan bergabung dengan saluran, dia memasukkan informasi di klien. Informasi ini disimpan dalam file CHA yang dapat diambil kapan saja.
