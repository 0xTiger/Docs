## Apa itu file CHAT?

File CHAT adalah file konfigurasi yang digunakan oleh IRC (Internet Relay Chat) untuk menghubungkan ke saluran IRC. Ini berisi parameter koneksi seperti server, port, nama saluran IRC, dan opsional kata sandi untuk membuka saluran IRC. Pengaturan digunakan oleh klien untuk terhubung ke saluran lain. Obrolan dan diskusi di IRC bersifat pribadi.

Aplikasi yang dapat **membuka** file CHAT termasuk mIRC dan Visual IRC.

## Format File CHAT

File CHAT disimpan sebagai file biner ke disk. Ketika seorang pengguna ingin terhubung ke saluran di IRC, dia memasukkan informasi di klien iRC, dia memasukkan informasi di perangkat lunak klien. Informasi disimpan dalam file CAHT untuk pengambilan lateral.
