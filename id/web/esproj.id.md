## Apa itu file ESPROJ?

File ESPROJ adalah file proyek yang dibuat oleh halaman web yang membuat perangkat lunak, Espresso. Ini tersedia di Mac OS dan berisi daftar file terkait yang merupakan bagian dari proyek situs web. Saat Anda membuka file ESPROJ di Espresso, itu membuka semua file terkait di editor untuk mengedit atau memperbarui kode sumber. File ESPROJ juga berisi pengaturan penerbitan server dan preferensi pengguna terkait lainnya.

## Format File ESPROJ

File ESPROJ disimpan sebagai file biner dan isinya tidak dapat dibaca manusia. Direktori file ESPROJ juga merupakan direktori root untuk situs web.
