## Apa itu file ARO?

File ARO adalah file halaman web sisi server yang dibuat dengan Aplikasi Web SteelArrow. Ini berisi tag dan skrip [HTML](/id/web/html/) dan SteelArrow. Ini dijalankan di server untuk menghasilkan halaman respons lengkap sebelum mengirimkannya ke browser pengguna. File ARO berisi hampir 95% konten HTML sedangkan sisanya terdiri dari kode SteelArrow. Agar file ARO berfungsi untuk Anda, server Aplikasi Web SteelArrow harus diinstal dan dijalankan di mesin server web.

## Format File ARO

File ARO ditulis dalam file bahasa markup HTML dengan kode JavaScript tersemat dan applet Java. [Tag SteelArrow](http://www.steelarrow.com/docs/basics1.aro) adalah blok bangunan dasar untuk pengembangan halaman web. Meskipun ini tidak mengubah tampilan halaman, mereka digunakan untuk mengisi halaman dengan data. Selain itu, mereka juga dapat digunakan untuk mengontrol keluaran dengan pernyataan bersyarat.

### Contoh Tag ARO

Itu<SAOUTPUT> Tag ARO memungkinkan pengembang untuk menampilkan nilai data di mana saja dalam skrip. Misalnya, dapat digunakan untuk mendapatkan output dari tabel PARAM<SAOUTPUT VALUE=PARAM[]> yang dapat ditampilkan dalam HTML<BODY> menandai.
