## Apa itu file SITUS WEB?

File SITUS WEB adalah file pintasan situs web yang dibuat dengan Microsoft Internet Explorer (IE). Ini berisi file URL dan ikon (ICO) yang mengarah ke favicon situs web. Ikon digunakan untuk menampilkan pintasan. Format file WEBSITE diperkenalkan dengan rilis IE versi 9. Versi IE sebelumnya digunakan untuk menyimpan pintasan ke halaman web sebagai file [.url](/id/web/url/).

## Format File WEBSITE

File .website disimpan ke disk sebagai file teks biasa yang isinya dapat dilihat di editor teks apa pun. Selain konten lain, itu berisi URL target yang merupakan jalan pintas. Jika Anda ingin membuka situs web target di browser selain IE, Anda dapat menyalin URL ini dan menempelkannya di browser lain seperti Google Chrome, Safari, atau Firefox untuk melihat situs web tersebut.

## Bagaimana cara membuat file .WEBSITE?

File SITUS WEB dapat dibuat di IE dengan mengklik URL di bilah alamat dan menyeretnya ke lokasi yang diinginkan di komputer Anda seperti desktop atau dokumen. Ini membuat pintasan ke lokasi internet itu dan dapat dibuka dengan mengklik dua kali pintasan tersebut. Jika Anda ingin menyimpan pintasan sebagai file `URL`, tahan tombol SHIFT sebelum menyeret favicon situs target ke desktop.
