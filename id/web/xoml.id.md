## Apa itu berkas XOML?

XOML adalah singkatan dari Extensible Object Markup Language dan merupakan format serialisasi untuk objek alur kerja Windows Workflow Foundation. Berdasarkan **[XAML](/id/web/xaml/)**, sebagian besar digunakan untuk membuat antarmuka pengguna di **[XML](/id/web/xml/)** biasa. Ini digunakan untuk mendeklarasikan alur kerja untuk antarmuka pengguna dan dikompilasi bersama dengan file terpisah yang berisi logika implementasi. Ini berisi struktur berbasis pohon yang mendefinisikan simpul alur kerja root, sub-elemen bersarang, dan segmen kode yang disematkan. Saat alur kerja baru dibuat dengan Visual Studio untuk .NET, Anda memiliki opsi untuk memilih apakah harus dalam format Kode atau format Terpisah Kode. Jika Anda memilih yang terakhir, IDE akan membuat dua file untuk Anda; satu dalam format XOML (terdiri dari tata letak dan pengaturan alur kerja), dan lainnya dalam format [.CS](/id/programming/cs/)/[.VB](/id/programming/vb/) tempat kode pemrograman berada.
