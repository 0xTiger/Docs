## Apa itu file CRDOWNLOAD?

File CRDOWNLOAD adalah file yang diunduh sebagian yang dibuat saat Anda mengunduh file dari internet menggunakan browser Microsoft Edge, Google Chrome, atau Chromium. Konten file yang sedang diunduh disimpan dalam file ini hingga pengunduhan selesai. Setelah pengunduhan selesai, ekstensi .crdownload dihapus dan diganti dengan ekstensi file asli dari file yang sedang diunduh.

## Format File CRDOWNLOAD

File CRDOWNLOAD dapat berupa teks dan juga format biner. Browser membuat file .crdownload secara default setelah unduhan dimulai. Jika pengunduhan terputus di antaranya, file CRDOWNLOAD tetap disimpan ke disk. Ini terkadang memungkinkan untuk melanjutkan unduhan dari tempat terputusnya.

### CRDOWNLOAD Contoh

Mendukung Anda mengunduh file fileformatslist.doc dari internet. Saat pengunduhan dimulai, browser Anda menyimpan file ke folder Unduhan sebagai file formatslist.crdownload file. Ukuran filenya bertambah saat file diunduh. Setelah pengunduhan selesai, file akan diubah namanya menjadi fileformatslist.doc.
