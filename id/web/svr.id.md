## Apa itu file SVR?

File SVR adalah format file 3D yang digunakan oleh SuperScape untuk membuat dunia interaktif 3D. Dunia interaktif tiga dimensi ini dapat dilihat melalui browser web dengan plugin SVR. File SVR mirip dengan format [file VRT](/id/web/vrt/) tetapi dikompresi sehingga menghasilkan ukuran file yang lebih kecil. Dengan demikian, file SVR dimuat lebih cepat dan juga dapat dibagikan melalui internet.

## Format File SVR - Informasi Lebih Lanjut

File SVR disimpan ke disk dalam format file biner. Ini adalah format file berpemilik dan rincian struktur file internalnya tidak tersedia untuk umum untuk referensi pengembang.
