## Apa itu file FWP?

File dengan file .fwp adalah file arsip yang dibuat dengan Microsoft Expression Web dan berisi semua file dan folder untuk sebuah desain. Microsoft Expression Web adalah produk perangkat lunak desain web umum oleh Microsoft dan didasarkan pada editor [HTML](/id/web/html/). Itu telah dihentikan sejak 2012 dan tersedia gratis untuk diunduh oleh Microsoft. File FWP dapat diekspor dan disimpan sebagai format .cab Microsoft.

## Format File FWP - Informasi selengkapnya

Spesifikasi format file yang tepat dari file FWP tidak tersedia untuk referensi pengembang. Namun, ini adalah file biner dan dapat dibuka di Microsoft Expression Studio yang sudah lama dihentikan.
