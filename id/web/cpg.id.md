## Apa itu file CPG?

File CPG adalah file proyek yang dibuat oleh perangkat lunak editor halaman Web, Cool Page. Perangkat lunak ini sudah usang beberapa tahun yang lalu dan tidak lagi digunakan. File CPG berisi detail tentang objek yang digunakan di halaman web dan properti objek tersebut. File proyek CPG dapat dibuka di perangkat lunak Cool Page dan setelah diselesaikan, halaman web dapat diekspor sebagai halaman web [HTML](/id/web/html/).

## Format File CPG

Detail format file internal dari format file CPG tidak diketahui. Ini disimpan sebagai file biner dan hanya akan dibuka di perangkat lunak Cool Page.
