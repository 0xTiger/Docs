## Apa itu file ADR?

File ADR (disimpan dengan ekstensi .adr) adalah daftar bookmark yang dibuat oleh browser Web Opera. Daftar ini diisi karena pengguna terus mem-bookmark URL web. Penanda ini disimpan dalam file ADR yang dapat dicadangkan dan dipulihkan bila diperlukan. Karena sebagian besar browser seperti Chrome dan Firefox menyimpan bookmark dalam format [HTML](/id/web/html/), ini dapat dikonversi ke format file ADR untuk diimpor di browser Opera. File ADR dapat dibuka di browser Opera.

## Format File ADR - Informasi Lebih Lanjut

Detail format file internal dari file ADR tidak diketahui. Namun, ada perbedaan antara format file ADR yang diekspor dengan versi Opera yang berbeda. Hal ini mempersulit pembuatan [penggabungan file ADR](https://superuser.com/questions/471959/how-do-i-merge-several-opera-adr-bookmark-files-to-one-single-file) dengan berbagai versi Opera. Beberapa alat eksternal dapat digunakan untuk menggabungkan ini.
