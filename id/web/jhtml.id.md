## Apa itu file JHTML?

File dengan ekstensi .jhtml adalah file [HTML](/id/web/html/) dengan kode [Java](/id/programming/java/) yang dijalankan di server saat klien meminta halaman ini di browser. Server memproses permintaan, mengeksekusi fungsi Java yang terkandung dalam fungsi, dan mengembalikan halaman ke browser klien. Objek Java yang disematkan di halaman JHTML berjalan di server untuk menangani permintaan halaman jenis ini. File JHTML juga dapat mengakses informasi dari database menggunakan koneksi JDBC (Java [Database](/id/database/) Connectivity). File JHTML dapat dibuka di editor teks apa pun dan dilihat di browser web seperti Google Chrome, Firefox, dan Safari.

## Format File JHTML

JHTML adalah teknologi milik ATG dan dapat dibuat menggunakan Editor Dokumen Dynamo ATG (Art Technology Group). File JHTML ditulis dalam format file teks biasa menggunakan kode HTML dan Java standar. Ini berisi tag HTML standar selain tag kepemilikan yang mereferensikan objek Java. Ketika halaman tersebut diminta oleh pengguna, server HTTP meneruskan permintaan tersebut ke server aplikasi Java. Halaman JHTML diubah terlebih dahulu menjadi file .java dan dikompilasi untuk menghasilkan file [.class](/id/programming/class/) yang dijalankan sebagai servlet. Ini menghasilkan aliran data HTTP dan HTML standar yang dikembalikan ke browser yang meminta untuk ditampilkan kepada pengguna. Ini berguna untuk menjalankan kueri terkait basis data di server dan mengembalikan hasil akumulasi akhir ke browser klien.
