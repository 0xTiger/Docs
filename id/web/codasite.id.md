## Apa itu file CODASIT?

File CODASITE adalah file proyek situs web yang dibuat dengan software pembuatan dan penerbitan halaman web [Coda](https://coda.io/). Ini berisi daftar semua file yang merupakan bagian dari proyek. Ini termasuk halaman web sebagai file [HTML](/id/web/html/), file stylesheet berjenjang [CSS](/id/web/css/), file Javascript [JS](/id/web/js/), dan file pendukung terkait lainnya. File proyek CODASITE juga berisi informasi untuk penerbitan situs web ke server web.

## Format File CODASIT

File proyek CODASITE disimpan ke disk sebagai file biner di dalam direktori ~/​Library/​Application Support/​Coda 2/​Sites/​. Antarmuka pengguna Coda menyediakan antarmuka untuk mengelola situs Coda karena membukanya secara manual dapat menyebabkan pengaturan proyek yang tidak tepat dan menimbulkan masalah.
