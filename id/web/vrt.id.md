## Apa itu file VRT?

File VRT adalah format file 3D yang berisi dunia tiga dimensi yang dapat dilihat di browser web dengan plugin SVR [Viscape](https://www.viscape360.com/). Ini digunakan oleh plugin untuk memuat dan menampilkan data dunia 3D kepada pengguna untuk visualisasi dan analisis. Informasi yang disimpan dalam file VRT dapat terdiri dari data seperti tekstur, suara, dimensi, dan perilaku. File VRT terbaru disimpan dalam format file SVR yang merupakan versi terkompresi dari file VRT.

## Format File VRT

File VRT disimpan dalam format file milik Viscape dan detail format file internalnya tidak tersedia untuk umum.
