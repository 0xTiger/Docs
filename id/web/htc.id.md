## Apa itu file HTC?

File HTC adalah file format file web yang digunakan oleh Microsoft Internet Explorer. Ini berisi definisi untuk komponen HTML yang menjalankan fungsi kecil untuk melakukan berbagai tugas di dalam browser web. Fungsionalitas yang ditentukan di dalam file HTC dapat digunakan kembali dan dapat diakses di beberapa halaman [HTML](/id/web/html/).

## Format File HTC - Informasi Lebih Lanjut

File HTC disimpan dalam format file teks biasa. Ini berarti Anda dapat membukanya di editor teks apa pun dan membuat perubahan pada fungsionalitasnya. Ini biasanya dibuka di Internet Explorer, tetapi dengan cara itu Anda hanya dapat melihat file-file ini dan tidak dapat mengeditnya.

File HTC sebenarnya adalah file HTML yang merupakan kombinasi skrip dan sekumpulan elemen khusus HTC untuk menentukan komponen.
