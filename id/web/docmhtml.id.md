## Apa itu file DOCMHTML?

File DOCMHTML adalah varian lama dari arsip web yang disimpan oleh versi Microsoft Word sebelumnya. Itu tidak lagi didukung oleh Microsoft Word versi terbaru. File DOCMHTML berisi representasi HTML dari dokumen kata .doc asli. Itu dapat dibuka di Microsoft Word dan juga di browser web. Saat DOCMHTML diterbitkan, DOCMHTML dapat dihosting di server web yang dapat diakses oleh pengguna melalui permintaan web. Format serupa lainnya adalah [DOCHTML](/id/web/dochtml/) yang juga telah dihentikan.

## Format File DOCMHTML - Informasi Lebih Lanjut

File DOCMHTML digunakan untuk disimpan dalam format HTML. Itu bisa dibuka di editor teks seperti Notepad dan Notepad ++. Setelah ditandai sebagai usang, Microsoft Word memperkenalkan opsi untuk menyimpan dokumen kata sebagai file [MHTML](/id/web/mhtml/) dan [MHT](/id/web/mht/). Format file baru ini dapat dibuka dengan browser web.
