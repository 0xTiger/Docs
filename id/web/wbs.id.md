## Apa itu file WBS?

File WBS adalah file proyek yang dibuat oleh WebBlender yang merupakan program presentasi dan pembuatan situs web untuk anak-anak. File WBS dibuat saat proyek baru dimulai di WebBlender. Semua fitur situs yang ditambahkan ke proyek seperti halaman, gambar, teks, animasi, dan sebagainya, direferensikan dalam file proyek WBS. Setelah proyek siap, proyek menerbitkan konten situs sebagai file [HTML](/id/web/html/). WebBlender dibuat dan dikelola oleh [Tech4Learning](https://www.tech4learning.com/).
