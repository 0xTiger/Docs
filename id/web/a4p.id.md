## Apa itu file A4P?

File A4P adalah file proyek yang dibuat oleh program perangkat lunak A4Desk Pro yang digunakan untuk membuat Situs Web Flash. Itu dibuat dengan template yang disertakan dengan A4Desk Pro dan tidak memerlukan seseorang untuk memiliki pengetahuan Flash yang mendalam. File A4P yang dibuat dengan template dapat menyertakan konten situs web dalam bentuk teks dan animasi Flash. Jika Anda tidak tahu tentang Flash, Flash digunakan untuk membuat animasi yang dapat diekspor ke format file [SWF](/id/page-description-language/swf/) serta [GIF](/id/image/gif/) untuk dimasukkan dalam situs web.

## Format File A4P

File A4P dibuat dari file template dan disimpan sebagai file biner ke disk. Struktur format file internal file A4P tidak diketahui.

File A4P dapat dibuka dengan perangkat lunak **Avanquest A4Desk Pro**.
