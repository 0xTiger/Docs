## Apa itu file XHT?

File dengan ekstensi .xht adalah file web yang terkait dengan jenis file [XHTML](/id/web/xhtml/) (Extended Hypertext Markup Language), yang merupakan file markup berbasis [XML](/id/web/xml/). Kedua file ini mirip dengan HTML tetapi memiliki sintaks seperti XML yang lebih ketat. File XHT dirancang sedemikian rupa agar lebih terstruktur, melibatkan lebih sedikit skrip, dan bersifat umum selain menggunakan fasilitas XML yang ada.

## Format File XHT

File XHT dibuat dan disimpan sebagai file teks biasa dengan pengkodean UTF-8 secara default. Ini berisi kode sumber lengkap dari dokumen XHTML yang dapat dibuka dan diedit dengan editor teks apa pun yang mendukung penyandian Unicode. Selain editor teks, format file XHT dapat dimengerti oleh sebagian besar browser web modern dan dapat dibuka menggunakan ini.

## Contoh XHT

Contoh dokumen XHT berikut mendefinisikan deklarasi XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
