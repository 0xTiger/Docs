## Apa itu file ECE?

File ECE adalah halaman web yang dibuat secara dinamis oleh platform pengelolaan konten, **Escenic Content Engine (ECE)**. Ini berisi kode sisi server yang diproses oleh server web dan disajikan kembali ke browser pengguna sebagai halaman [HTML](/id/web/html/). Format file ECE tidak terlalu populer dibandingkan dengan format file sisi server lainnya seperti ASP dan PHP. File-file ini dirender secara internal dan disajikan kepada pengguna akhir untuk tujuan tampilan.

## Format File EC

File ECE tidak disimpan ke disk tetapi disajikan langsung ke pengguna berdasarkan permintaan. Ini dikonversi ke HTML dan dikirim kembali ke browser pengguna untuk ditampilkan. Escenic Content Engine secara aktif digunakan untuk menerbitkan halaman artikel berita. Berdasarkan teknologi JEE dan SQL, server ECE menyediakan repositori yang andal dan dapat diskalakan untuk semua konten online.

Browser klien dapat mengakses konten di server ECE melalui HTTP/REST API. Selain melayani halaman [HTML](/id/web/html/), ECE menyediakan alur kerja yang sangat efisien untuk mengelola dan memublikasikan konten video/audio.
