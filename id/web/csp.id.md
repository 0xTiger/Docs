## Apa itu file CSP?

File CSP adalah file halaman web dinamis yang berisi konten yang ditulis untuk dieksekusi di Server Aplikasi Konsep RadGs. Itu dikompilasi dan dijalankan di sisi server, setelah itu disajikan untuk meminta browser pengguna melalui protokol HTTP. Tidak seperti format file CON, file CSP tidak memerlukan klien Konsep diinstal pada pengguna untuk melihat file-file ini. Anda dapat melihat file CSP yang mirip dengan halaman sisi server lainnya seperti [PHP](/id/programming/php/) dan [ASP](/id/web/asp/).
