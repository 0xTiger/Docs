## Apa itu file XOJ?

File XOJ adalah file notebook yang berisi catatan yang diambil menggunakan perangkat lunak pencatat tangan Xournal. Xournal adalah aplikasi multi-platform gratis yang memungkinkan Anda membuat catatan dengan berbagai jenis konten seperti teks, gambar, dan orat-oret tulisan tangan. File XOJ dibuat saat file notebook baru dibuat dengan Xournal. File XOJ ini juga dapat dibagikan dengan pengguna lain untuk berbagi data. Versi Xournal yang lebih baru adalah [Xournal++](https://github.com/xournalpp/xournalpp) yang merupakan varian lintas platform dari Xournal.

## Format File XOJ

File XOJ dalam format file XML dan disimpan ke disk dalam bentuk terkompresi menggunakan kompresi [.gz](/id/compression/gz/). Karena penggunaan format XML universal, file XOJ juga dapat dibuka dan diedit dengan varian lintas platform Xournal++.

Setiap file XOJ dapat berisi satu halaman lagi dari buku catatan Xournal.

## XOJ - Platform yang Didukung

Xournal adalah multi-platform dan notebook-nya dapat disimpan sebagai file XOJ pada sistem operasi berikut.

* Linux
* MacOS
* Jendela
* Android
* Bangun iOS
