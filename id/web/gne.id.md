## Apa itu file GNE?

File GNE adalah halaman web yang dibuat secara dinamis dan digunakan oleh situs web berbagi foto Flickr. Halaman-halaman di-host di server dari mana ini disajikan ke browser pengguna yang meminta. File GNE mirip dengan halaman PHP dan ASP, tetapi dikembangkan dalam bahasa pemrograman berpemilik yang detailnya tidak tersedia untuk referensi pengembang.

## Format File GNE - Informasi Lebih Lanjut

File GNE digunakan oleh game komputer, Game Never Ending, yang dikembangkan oleh [Ludicorp](https://en.wikipedia.org/wiki/Ludicorp). Flicker kemudian diakuisisi oleh yahoo! pada Maret 2005.
