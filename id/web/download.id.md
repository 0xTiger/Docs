## Apa itu file DOWNLOAD?

File UNDUH adalah file yang diunduh sebagian dari internet yang masih dalam proses pengunduhan. Saat Anda mengunduh file dari internet menggunakan browser Apple Safari, mungkin diperlukan beberapa waktu untuk menyelesaikan pengunduhan. File yang diunduh sebagian ini disimpan ke folder **Unduhan**. Jika pengunduhan terhenti karena alasan apa pun, file dapat diklik dua kali untuk mencoba melanjutkannya.

## UNDUH Format File - Informasi Lebih Lanjut

File DOWNLOAD disimpan ke disk dalam format teks dan file biner. Itu tidak berisi seluruh file yang sedang dalam proses pengunduhan. Setelah pengunduhan selesai, ekstensi file .download diganti dengan ekstensi asli dari file yang sedang diunduh. Misalnya, jika Anda mengunduh video dari internet dalam format file MP4, file tersebut akan diunduh dan disimpan dengan ekstensi .download. Setelah unduhan selesai, ekstensi file akan diubah menjadi .mp4.

File DOWNLAOD mirip dengan file [.CRDOWNLOAD](/id/web/crdownload/) yang digunakan oleh Google Chrome dan Microsoft Edge.
