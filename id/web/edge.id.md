## Apa itu file EDGE?

File dengan ekstensi .edge adalah file Proyek Adobe Edge yang dibuat saat proyek dibuat dan disimpan dengan Adobe Edge. Adobe Edge memungkinkan desainer web untuk membuat animator [HTML](/id/web/html/) interaktif untuk web, penerbitan digital, dan iklan multimedia untuk browser desktop dan seluler. Hal ini dicapai dengan menggunakan HTML/HTML5, [JavaScript](/id/web/js/), dan [CSS3](/id/web/css/). File Edge dibuat sebagai file teks biasa dan dapat dibuka dengan beberapa aplikasi berbeda seperti Microsoft Notepad, Apple TextEdit, dan Notepad++.

## Format File EDGE

File EDGE disimpan dalam format teks biasa dan menggunakan jQuery untuk mengaktifkan animasi. Alternatif animasi EDGE adalah file Flash yang tidak lagi didukung. File EDGE disimpan bersama file HTML, JavaScript, dan CSS. Setelah membuka file EDGE, halaman web yaitu file HTML dibuka.
