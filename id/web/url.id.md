## Apa itu berkas URL?

File dengan ekstensi .url adalah pintasan URL situs web yang dapat disimpan ke komputer dengan alamat situs web apa pun. Ketika dibuka dengan mengklik dua kali, URL akan membuka situs web di browser default dan halaman akan ditampilkan kepada pengguna. File URL biasanya disebut sebagai file pintasan ke situs web dan diwakili oleh ikon browser default. File URL dapat dibuka dan diedit di editor teks apa pun seperti Microsoft Notepad, Notepad++, atau Apple TextEdit.

## Format File URL - Informasi Lebih Lanjut

File URL disimpan dalam format teks biasa dan dapat dibuka dan dilihat di editor teks apa pun. Ini terbuka di browser web default tempat pengguna melakukan penelusuran lainnya.
