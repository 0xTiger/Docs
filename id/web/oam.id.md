## Apa itu file OAM?

File .oam adalah file widget animasi yang diekspor dari Adobe Edge Animate Widget. Animasi yang diekspor sebagai format file widget OAM dapat dimasukkan ke dalam aplikasi Adobe lainnya seperti Dokumen InDesign ([file INDD](/id/page-description-lingual/indd/), Dreamweaver, dan Muse. File OAM seperti paket penerapan yang dapat dengan mudah dimasukkan ke proyek Adobe lain untuk memanfaatkannya. File OAM berisi informasi tentang aset animasi, perilaku, dan kode skrip tindakan. Anda dapat membuat file OAM dengan menerbitkan proyek Adobe Animate [file .AN](/id/web/an/) .
Pengguna dapat membuat file OAM dengan menerbitkan dari proyek Edge Animate (file .AN).

## Format Berkas OAM

File OAM disimpan ke disk sebagai arsip [ZIP](/id/compression/zip/) terkompresi. Ini menyiratkan bahwa Anda dapat mengganti nama ekstensi file menjadi .zip dan mengekstraknya dengan utilitas kompresi apa pun seperti WinZIP atau WinRAR. Ukuran file yang diperkecil memudahkan transfer dan berbagi animasi dengan pengguna lain melalui internet.

### Struktur Berkas OAM

Struktur file internal file .oam adalah hak milik dan tidak didokumentasikan secara publik oleh Adobe. Namun, diketahui bahwa file .oam berisi kumpulan file dan sumber daya (seperti grafik, audio, dan kode ActionScript) yang dikemas ke dalam satu file. Konten file .oam dapat mencakup file [XML](/id/web/xml/), file SWF, dan file sumber daya lainnya. Struktur sebenarnya dari file .oam bergantung pada versi spesifik Adobe Animate yang digunakan untuk membuatnya, serta jenis animasi dan aset yang disertakan dalam file.

File OAM berisi informasi berikut.

`Aset:` Informasi tentang aset grafik, audio, dan video yang digunakan dalam animasi.

`Perilaku:` Deskripsi tindakan dan interaksi yang terjadi dalam animasi.

`Kode ActionScript:` Kode yang digunakan untuk mengontrol perilaku dan interaktivitas animasi.

`Pengaturan publikasi:` Informasi tentang platform dan format animasi yang akan dipublikasikan, seperti HTML5 atau AIR.

`Metadata:` Informasi tambahan seperti pembuat animasi, tanggal pembuatan, dan nomor versi.
