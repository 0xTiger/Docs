## Apa itu file VDISCO?

File VDISCO adalah file penemuan yang digunakan oleh Microsoft Visual Studio di Layanan Web-nya. Ini memberikan informasi tentang Layanan Web yang tersedia seperti URL, namespace, dan metodenya. File VDISCO mirip dengan format file [DISCO](/id/web/disco/) tetapi dibuat saat runtime. Itu disimpan dalam aplikasi layanan Web dan digunakan oleh Visual Studio untuk menemukan dan menggunakan Layanan Web secara dinamis dalam sebuah proyek. File VDISCO digunakan bersama dengan file [.asmx](/id/web/asmx/) yang berisi kode Layanan Web sebenarnya.

Anda dapat membuka file VDISCO di editor teks apa pun seperti Microsoft Notepad, Github Atom, dan Apple TextEdit. Itu juga dapat dibuka dengan Microsoft Visual Studio 2012 dan yang lebih baru untuk bekerja dengannya.

## Format File VDISCO

File VDISCO disimpan dan dihosting dalam format file [XML](/id/web/xml/) yang merupakan format universal untuk komposisi dan penerbitan data. Ini berisi URL layanan, ruang nama dan metode dalam tag XML standar di mana setiap tag berisi nilai informasi masing-masing.
