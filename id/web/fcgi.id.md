## Apa itu file FCGI?

File FCGI adalah file kode sumber yang berisi kode sumber yang ditulis untuk FastCGI (FCGI). Itu dapat ditulis dalam beberapa bahasa pemrograman yang berbeda dan berisi ekstensi standar CGI yang digunakan untuk penanganan permintaan server Web yang lebih kuat. Bahasa pendukung yang paling umum untuk FCGI termasuk **Perl** yang menyediakan dukungan untuk kode Perl dan antarmuka eksternal FastCGI API yang kuat.

## Format File FCGI

Apa yang membuat file FCGI penting? Antarmuka FCGI mendukung komputasi terdistribusi untuk menangani permintaan web dan mendistribusikannya ke beberapa server. Ini membuat file FCGI sebagian besar digunakan untuk penyeimbangan beban di beberapa server.
