## Apa itu file XD?

File dengan ekstensi .xd adalah maket yang dibuat untuk situs web atau aplikasi seluler menggunakan perangkat lunak Adobe XD. Desainer, tim kreatif, dan organisasi menggunakan file ini untuk merancang aplikasi dengan merujuknya. Satu file XD berisi elemen desain seperti gambar, bentuk, dan teks yang menyediakan rangkaian fitur lengkap untuk mengembangkan prototipe gambar rangka yang cepat dan intuitif bagi pengembang untuk mulai mengembangkannya sebagai aplikasi.

## Format File XD

File XD berisi elemen grafik (gambar, bentuk, dan teks) berlapis-lapis sebagai satu atau lebih artboards. Ini bersama dengan panduan dan lapisan menyediakan blok bangunan untuk wireframes.

* `Artboards` - Ini mewakili layar dalam aplikasi atau halaman situs web, dan merampingkan proses desain. Artboard membuat desain interaktif untuk berbagai ukuran layar dalam satu dokumen.
* `Guides` - Ini digunakan untuk menyejajarkan objek atau mengatur jarak antar elemen pada artboard. Panduan menggunakan kisi-kisi yang berfungsi sebagai kerangka kerja untuk mengatur elemen desain (gambar, mesin terbang, paragraf) dan memastikan simetri tata letak.
* `Layers` - Desainer UX menggunakan lapisan di XD dengan cara yang sama seperti menggunakannya di Adobe Illustrator atau Photoshop. Ini digunakan untuk mengatur pekerjaan pada artboards untuk menghindari kekacauan dan dikontrol dari panel Layers.
