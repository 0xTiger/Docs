## Apa itu file FMP?

File dengan file .fmp adalah file menu animasi yang dibuat dengan aplikasi 123 Flash Menu. Ini berisi menu animasi untuk situs web termasuk komponen seperti font, gaya menu, warna, dimensi, suara, dan animasi. Aplikasi 123 Menu Flash tidak memerlukan keterampilan komputer khusus karena aplikasi itu sendiri menggabungkan semua fungsi untuk mengembangkan menu tersebut. File menu FMP dapat diekspor ke file [.swf](/id/page-description-language/swf/) Flash yang terkenal sepanjang masa yang dapat disematkan di dalam halaman web.
