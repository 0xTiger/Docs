## Apa itu file GSP?

File GSP adalah file pesan kesalahan yang menunjukkan masalah pengiriman email. Itu dikirim oleh IMail Server, program server email yang digunakan untuk komunikasi email oleh banyak usaha kecil dan menengah. Kesalahan dapat terjadi karena sejumlah alasan seperti data dalam jumlah besar, kehilangan data dalam file [MSG](/id/email/msg/), atau kesalahan kehabisan memori.

## Format File GSP - Informasi Lebih Lanjut

File GSP disimpan dalam format file web dan mudah dilihat dengan klien email seperti Microsoft Outlook. File GSP dihasilkan sebagai respons dari email terkirim yang menghasilkan kesalahan.
