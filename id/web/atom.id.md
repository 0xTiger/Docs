## Apa itu berkas ATOM?

File ATOM adalah format Sindikasi untuk struktur feed Atom dan dokumen entri. Mirip dengan file .RSS, format file ATOM adalah format sindikasi berbasis XML yang merupakan standar untuk menerbitkan konten. File ATOM digunakan untuk umpan web yang memungkinkan program perangkat lunak memeriksa pembaruan yang dipublikasikan di situs web. Ini berisi entri yang dapat dari berbagai jenis seperti tajuk utama, artikel teks lengkap, kutipan, atau tautan ke konten di situs web

Aplikasi yang dapat **membuka file ATOM** termasuk **Apple Safari**.

## Format File ATOM - Informasi Lebih Lanjut

File ATOM disimpan dan diterbitkan dalam format file XML populer yang merupakan format universal untuk pertukaran informasi. Itu dikembangkan sebagai alternatif untuk RSS dengan tetap melihat keterbatasan dan kekurangan format file RSS.

### Jenis Dokumen Atom

1. `Dokumen Entri Atom` - Ini adalah dokumen XML yang terdiri dari satu item informasi, dikenal sebagai entri, untuk umpan Atom. Ini memiliki<atom:entry> elemen yang berisi sejumlah elemen anak. Konten entri Atom dapat berupa teks biasa, HTML, XHTML, atau jenis media IANA (Internet Assigned Numbers Authority) lainnya.
1. `Dokumen Umpan Atom` - Ini adalah dokumen XML yang menyediakan metadata tentang umpan Atom dan satu atau beberapa entri untuk umpan. Saat klien membuat permintaan informasi dari umpan, dokumen umpan dihasilkan oleh server yang mencakup sejumlah entri Atom untuk memenuhi permintaan tersebut.
1. `Koleksi Atom` - Ini adalah jenis dokumen umpan Atom khusus yang berisi URL entri Atom yang tersedia untuk diedit.
