## Apa itu file WEBBOOKMARK?

File WEBBOOKMARK adalah file bookmark yang disimpan dengan browser web Safari Apple di macOS. Ini berisi informasi tentang halaman web yang telah disimpan pengguna dari browser. Informasi yang disimpan dalam file WEBBOOKMARK mencakup judul halaman, URL, dan grup bookmark tempat halaman ini disimpan. Saat halaman web di-bookmark, browser Safari meminta grup untuk menyimpan halaman yang di-bookmark. Sebagian besar pengguna menyimpan bookmark di folder Bookmark yang merupakan lokasi default yang ditawarkan oleh browser umum.

## Format File WEBBOOKMARK

File WEBBOOKMARK disimpan ke disk dalam format file biner yang tidak dapat dibaca manusia. Pengguna dapat menandai halaman web sebagai bookmark jika dimaksudkan untuk dikunjungi nanti atau terkait dengan minat pengguna. Bila diperlukan, pengguna dapat menavigasi ke bagian bookmark di browser Safari dan mengunjungi halaman itu lagi.
