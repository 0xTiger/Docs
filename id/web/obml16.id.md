## Apa itu file OBML16?

File OBML16 adalah halaman web yang disimpan menggunakan browser web seluler Opera Mini. Ini didasarkan pada Opera Binary Markup Language ([OBML](/id/web/obml/)) dan merupakan format ringkas untuk memberikan akses cepat ke halaman web yang disimpan. Ini menggantikan format file [OBML15](/id/web/obml15/) ketika Opera Mini versi 7 diluncurkan. Semua varian halaman web Opera Mini ini ditingkatkan dari format file dasar OBML dan setiap peningkatan Opera Mini hanya kompatibel dengan satu format OBML. File OBML16 dapat dikonversi ke [PDF](/id/pdf/) dan [HTML](/id/web/html/) secara online.

## Format File OBML16 - Informasi Lebih Lanjut

OBML16 didasarkan pada format file OBML yang merupakan format kompak untuk menyimpan halaman web. Keuntungan menggunakan format ringkas ini adalah mengurangi ukuran konten web, sehingga meningkatkan kecepatan pengunduhan. Setelah diunduh, pengguna Opera Mini dapat melihat halaman ini tanpa perlu memiliki akses ke koneksi Internet.
