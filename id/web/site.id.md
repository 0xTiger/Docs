## Apa itu file SITUS?

File SITE adalah file proyek yang digunakan oleh perangkat lunak desain situs web `Adobe GoLive`. Ini menyimpan informasi proyek situs web seperti file, folder, gambar, lembar gaya khusus, dan skrip. Ini juga berisi struktur situs web dan tautan yang digunakan di halaman web situs. Karena ini adalah file proyek, ini berisi referensi ke file proyek tetapi tidak berisi file sebenarnya.

Adobe GoLive telah dihentikan dan tidak lagi didukung.

## Format File SITUS

File SITE disimpan ke disk dalam format file milik Adobe dan rincian internal format filenya tidak tersedia untuk umum untuk referensi pengembang.
