## Apa itu file CPHD?

File CPHD adalah dokumen Hypertext yang digunakan oleh mesin pra-prosesor CloudChan. Ini berisi HTML dan skrip untuk membuat halaman web dan dokumen dinamis. Skrip di dalam file CPHD menyerupai sintaks yang digunakan oleh Java. Halaman web CPHD mengembalikan dokumen HTML saat dijalankan pada proses CloudChan. Format CPHD dapat bekerja dengan dokumen XML apa pun.

## Format File CHD

File CPHD disimpan ke disk sebagai file teks biasa. Hal ini memungkinkan untuk membuka file-file ini di editor teks apa pun seperti Notepad atau Notepad++.
