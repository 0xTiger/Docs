## Apa itu file BAGIAN? ##

File bagian atau ekstensi .part adalah file yang diunduh sebagian. Ini digunakan saat pengunduhan sedang berlangsung atau terhenti karena masalah apa pun, memberikan pengelola unduhan kesempatan untuk melanjutkan pengunduhan jika memungkinkan.
Ini berarti bahwa browser, seperti Firefox atau program transfer file seperti eMule, eMule plus, FlashGet, dll. menyimpan sebagian file, yang disebut file Part, saat pengunduhan sedang berlangsung di perangkat Anda. File bagian ini akan menunjukkan kepada Anda jika pengunduhan sedang berlangsung atau terhenti sebelum selesai. Tidak hanya itu tetapi file PART menyimpan semua data sampai pengunduhan selesai, itulah sebabnya beberapa di antaranya nanti dapat dilanjutkan jika Anda ingin memulai pengunduhan lagi.

## Format File BAGIAN ##

There are few browsers that break large download files into smaller downloads and assign each portion a .part extension. These files can be seen in the “Download” folder. Once the download is finished, the browser will combine all the PART files to give a final downloaded file. For example, while downloading a file from Firefox, the download manager will store the data being downloaded in a PART file and replaces the name with certain characters, and add a .part extension at the end, such as **doodle.or** becomes **JKzMn.or.part**. After completion, Firefox will remove the .part extension and the file will be ready to launch.
