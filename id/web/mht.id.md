## Apa itu file MHT?

File dengan ekstensi .mht adalah format file pengarsipan berkemampuan MIME yang berisi berbagai jenis data ke dalam satu file. Itu dapat menyimpan data seperti teks, gambar, gaya halaman dalam bentuk file [CSS](/id/web/css/), JavaScript, dan sumber daya lainnya sebagai sumber daya tertanam di dalamnya. File MHT, memiliki pesan tipe MIME/rfc822, merangkum semua konten file HTML sebagai file arsip tunggal untuk disimpan pada pengarsipan di perangkat penyimpanan. Aplikasi perangkat lunak seperti Microsoft Word memungkinkan Anda mengonversi dokumen WORD Anda ke MHT dengan mengekspor sebagai file MHT. File MHT dapat dibuka menggunakan browser populer seperti Microsoft Internet Explore dan Google Chrome.

## Format File MHT

Seperti [MHTML](/id/web/mhtml/), MHT juga merupakan enkapsulasi MIME dari kumpulan dokumen HTML. Konten dokumen di dalam dokumen MHT seperti gambar sebaris, lembar gaya, applet, dll. dikodekan dengan MIME yang ditautkan ke sumber daya root melalui URI. Klien email seperti Microsoft Outlook menyimpan email (biasanya [EML](/id/email/eml/)) dalam format file MHT. Spesifikasi lengkap format file MHT tersedia di [RFC 822](https://tools.ietf.org/html/rfc822) dan dapat dirujuk untuk pengembangan aplikasi perangkat lunak yang dapat memproses file MHT.

## Perbedaan antara MHT dan MHTML

Saat menyimpan halaman online, pengguna diminta untuk memilih jenis format file di mana halaman online harus disimpan di sistem asli. Biasanya, pengguna bingung antara bahasa markup dan format file MHTML. Mereka merasa sulit untuk melihat bahwa format file lebih sehat di antara keduanya.

Ketika pengguna memutuskan untuk menghindari pemborosan halaman online sebagai bahasa markup, maka sebuah folder terbentuk, yang berisi banyak file untuk berbagai konten halaman yaitu file yang berbeda yang dibuat untuk teks, grafik, applet dll. Di sisi lain, menyimpan halaman online karena MHTML membuat satu file untuk halaman online lengkap. Semua elemen halaman termasuk grafik, tautan, dan file alternatif tertanam di dalam satu file.

Tentu saja, mudah untuk mengelola file MHT atau MHTML karena file tersebut, yang dapat dilindungi dan dibagikan dengan mudah melalui email. Namun, file bahasa markup berada di bawah kemungkinan kehilangan data karena kehilangan atau penghapusan bahkan satu file dapat membuat seluruh folder bahasa markup tidak berguna bagi pengguna.
