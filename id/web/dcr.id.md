## Apa itu file DCR?

File DCR adalah file media yang dibuat dan disimpan dalam format Adobe Shockwave. Ini digunakan untuk membuat game web dan game interaktif yang serupa dengan yang dirancang dan disimpan dalam file [SWF](/id/page-description-language/swf/). File DCR sebelumnya dibuat dengan Adobe Director yang kemudian dihentikan setelah diakuisisi oleh sistem Adobe. File DCR mirip dengan file SWF yang dibuat oleh Adobe Flash.
