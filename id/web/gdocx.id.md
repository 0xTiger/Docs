## Apa itu file GDOCX?

File GDOCX adalah jenis file pintasan yang dibuat yang mengarah ke dokumen pengolah kata online yang disimpan di Google Drive. Itu dibuat dengan Google Docs dan disimpan ke Google Drive pengguna secara default. File GDOCX mirip dengan [GDOC](/id/web/gdoc/) yang paling umum digunakan. Informasi file target disimpan dalam file GDOCX dalam teks biasa yang dapat dibuka dengan editor teks apa pun.

## Format File GDOCX - Informasi Lebih Lanjut

File GDOCX mirip dengan GDOC dan berisi URL target yang akan dibuka. Ini berada di folder klien Google Drive pada sistem pengguna dan tetap disinkronkan ke folder Google Drive secara online. Google menyediakan versi klien desktop Google Drive hingga 2018 ketika akhirnya dihentikan. Namun, pengguna tetap dapat membuat dan menyimpan file mereka ke penyimpanan online Google Drive online.
