## Apa itu file STC?

File dengan ekstensi .stc adalah file koneksi yang digunakan oleh perangkat lunak Adobe Contribute untuk menyimpan informasi koneksi. Ini berisi informasi koneksi seperti nama pengguna, url host, protokol seperti FTP, SFTP, dll. Untuk digunakan kembali dari dalam situs web yang ada. Bagi yang belum tahu, Adobe Contribute dulunya adalah editor [HTML](/id/web/html/) khusus yang digunakan untuk mendorong pembaruan ke situs web yang ada, termasuk artikel berbasis konten yang dikenal sebagai blog. Itu termasuk plugin untuk browser web populer seperti Internet Explorer dan Firefox yang dengan mulus menyumbangkan konten ke situs web dari dalam halaman. Itu telah dihentikan sejak Januari 2017 dan Adobe Dreamweaver sekarang digunakan untuk membuat situs web dari awal.

## Format File STC

Format data internal file STC tidak diketahui. Namun, kunci koneksi dienkripsi dengan kata sandi dan semua informasi koneksi masuk jaringan atau FTP yang dikirim dalam file tetap aman dan hanya dapat diakses melalui Kontribusi. File koneksi kunci dapat dikirim melalui email ke pengguna atau disimpan ke komputer. Ketika diklik, secara otomatis membuka Kontribusi yang mengimpor koneksi.
