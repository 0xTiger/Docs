## Apa itu file XHTM?

File dengan ekstensi .xhtm adalah file halaman web yang ditulis dalam XHTML. Ia menggunakan XML 1.0 sebagai implementasi standarnya dan dianggap sebagai reformulasi HTML 4. Seperti XHTML, file XHTM berisi tag HTML standar yang mengikuti pedoman XML yang lebih ketat. File XHTM digunakan untuk menerbitkan konten situs web dan biasanya menggunakan ekstensi [XHTML](/id/web/xhtml/).

Anda dapat menggunakan browser web seperti Google Chrome, Apple Safari, dan Mozilla Firefox untuk membuka file XHTM.

## Contoh XHTM

Contoh dokumen XHTM berikut mendefinisikan deklarasi XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
