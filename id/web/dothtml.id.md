## Apa itu file DOTHTML?

File DOTHTML adalah dokumen template Microsoft Word yang disimpan dalam versi HTML. Itu dapat dibuka di browser web dan konten ditampilkan sebagai halaman web lainnya. File DOTHTML dihasilkan dari file .DOT Microsoft Word yang berisi pengaturan yang telah diformat sebelumnya untuk membuat file ini. Ketika file [DOT](/id/word-processing/dot/) dikonversi ke DOCHTML, semua pengaturan ini dipertahankan dan file DOTHTML yang dikonversi ditampilkan dengan format yang mirip dengan file DOT asli. Dengan cara ini, template yang diterbitkan dapat dilihat di browser web.

## Format File DOTHTML - Informasi Lebih Lanjut

File DOTHTML disimpan sebagai file [HTML](/id/web/html/) dan berisi kode sumber HTML yang dapat dibuka dengan editor HTML apa pun. Pada awalnya, Microsoft Word menawarkan versi HTML dari semua format dokumennya seperti [DOCHTML](/id/web/dochtml/), [DOCMHTML](/id/web/docmhtml/), tetapi kemudian versi berbasis HTML ini dihentikan . Akibatnya, DOTHMLT juga tidak lagi didukung.
