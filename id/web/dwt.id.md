## Apa itu file DWT?

File dengan ekstensi .dwt adalah file template halaman web yang dibuat dengan perangkat lunak Dreamweaver. Ini digunakan sebagai titik awal untuk mengembangkan beberapa halaman yang memiliki pengaturan dasar yang sudah ditentukan dalam template. Halaman [HTML](/id/web/html/) baru dapat dibuat menggunakan salinan template ini dan mengubah konten bila perlu, dengan tetap mempertahankan bagian umum tidak berubah. Bagian yang tidak dapat diedit dari template ini seperti header, footer, dan menu tetap sama di semua halaman web.

## Format File DWT - Informasi Lebih Lanjut

File DWT dikembangkan dan disimpan sebagai file teks biasa. Ini dapat dibuka di Dreamweaver atau editor teks apa pun, dan dapat diedit untuk perubahan yang diperlukan. Perangkat lunak Dreamweaver memperbarui UI saat perubahan diterapkan dan pengguna dapat mengamati efek perubahan secara visual di antarmuka pengguna aplikasi.

Setiap kali Anda menyimpan halaman [HTML] (/id/ web / html /) sebagai file template, Dreamweaver membuat folder template di level root dari folder root lokal. Tautan terputus jika file .dwt dipindahkan dari folder Templat atau diubah namanya. Oleh karena itu, penting untuk menyimpan file .dwt di folder Template tempat Dreamweaver membuatnya. Sebaiknya simpan elemen lain situs Anda, seperti file sumber gambar atau dokumen HTML, di lokasi selain folder Template.

## Tentang Dreamweaver

Perangkat lunak Dreamweaver pada awalnya dikembangkan oleh Macromedia. Perangkat lunak populer lainnya oleh Macromedia termasuk Flash player yang tetap tersedia sebagai pemutar animasi untuk browser untuk waktu yang lama. Setelah Adobe mengakuisisi Dreamweaver pada tahun 2005, Dreamweaver menjadi bagian integral dari aplikasi perangkat lunak Adobe dan sekarang didistribusikan oleh Adobe secara terpisah.
