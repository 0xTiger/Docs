## Apa itu file DOCHTML?

File DOCHTML adalah format HTML dari dokumen yang dibuat dengan Microsoft Word. Ini adalah versi [HTML](/id/web/html/) dari dokumen yang dibuat yang dapat digunakan untuk memublikasikan ke server web. File DOCHTML dapat dibuka di browser web untuk melihat konten dokumen kata yang dikonversi. Tampilan dan tampilan file DOCHTML sama dengan dokumen Word asli.

## Format File DOC HTML

File DOCHTML disimpan ke disk dalam format HTML yang siap dipublikasikan di server web. Microsoft Word menghentikan dukungan untuk menyimpan dokumen sebagai file DOCHTML. Ini diganti dengan format file [.mht](/id/web/mht/) dan [.mhtml](/id/web/mhtml/) yang umum digunakan sekarang.
