## Apa itu file DAP?

File dengan ekstensi .dap adalah file Halaman Data Microsoft Access yang dibuat dari dalam Microsoft Access dan memiliki akses ke data yang disimpan dalam database. Data yang disimpan dalam file web tersebut dapat diakses melalui jaringan (intranet dan internet) yang berbeda dari file Access [MDB](/id/database/mdb/) yang dikembangkan khusus untuk aplikasi desktop. Database tersebut dapat dibuat menggunakan Access 2010 dan Access Services (komponen opsional SharePoint). Database ini dapat digunakan oleh orang yang memiliki akun SharePoint menggunakan alamat web di browser web.
