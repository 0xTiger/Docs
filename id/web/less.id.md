## Apa itu file KURANG?

File LESS adalah style sheet yang digunakan untuk halaman web dan dibuat dengan Leaner Style Sheets (LESS). Itu mengimplementasikan tambahan pada bahasa CSS standar, yang membuatnya mudah dipelajari dan diterapkan. File KURANG memperluas [CSS](/id/web/css/) dengan menambahkan fitur seperti Variabel, Mixin, Penyusunan, Operasi, Fungsi, Ruang Nama, dan Aksesor.

## Format File Lembar Gaya Lebih Ramping

File KURANG dibuat dalam format CSS, memperluas kemampuannya dengan fitur tambahan yang ditawarkannya. Ini disimpan sebagai file teks dan dapat dibuka dalam bahasa lembar gaya LESS, CodeKit, dan Adobe Dreamweaver.
