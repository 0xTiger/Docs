## Apa itu file EPIBRW?

File EPIBRW adalah file umum yang dihosting di server web berbasis Microsoft IIS. Tujuan file ini tidak khusus untuk operasi tertentu atau jenis penyimpanan data apa pun, tetapi bersifat umum. Ini paling sering digunakan oleh file [.pdf](/id/pdf/) tetapi juga dapat digunakan oleh beberapa jenis file lainnya.

Ketika file EPIBRW diklik dua kali, itu mungkin terbuka di browser yang menampilkan konten. Namun, jika file ditautkan untuk diunduh, proses pengunduhan file akan dimulai sebagai hasil dari peristiwa ini.
