## Apa itu file XPD?

File XPD adalah file halaman web yang dibuat dari dokumen XML. Lebih seringnya, hal ini dihasilkan melalui transformasi XSLT. [Spesifikasi format file XPD](https://www.w3.org/TR/xml-pipeline/) tersedia di W3 yang menentukan cara untuk meningkatkan interoperabilitas multi-proses [XML](/id/web/xml/) aplikasi. Bahasa Pipa XML mendefinisikan hubungan pemrosesan antara sumber daya XML.
