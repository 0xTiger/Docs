## Apa itu file AN?

File dengan ekstensi .an adalah file proyek [Adobe Edge Animate](https://www.adobe.com/products/animate.html) yang berisi informasi tentang animasi. File .html dibuat saat proyek disimpan yang berisi referensi ke file [.css](/id/web/css/) dan [.js](/id/web/js/) terkait yang dibuat oleh Adobe Edge Animate. File AN disimpan bersama dengan file-file ini yang dapat digunakan untuk membuka file proyek. Adobe edge animate dapat digunakan sebagai alternatif untuk membuat halaman web animasi menggunakan Adobe Flash, yang tidak lagi didukung oleh browser.

## Sebuah Format File

File AN disimpan ke disk dalam format file teks biasa dan dapat dibuka di editor teks untuk melihat isinya.
