## Apa itu file TS?

File TS adalah file terjemahan yang dihasilkan dengan utilitas perangkat lunak lupdate yang dibundel dengan Qt SDK Digia. SDK tersedia untuk Sistem Operasi Windows, Mac dan Linux. File TS berisi frasa dalam satu bahasa dan terjemahannya ke bahasa lain. Terjemahan yang disimpan di dalam file TS khusus untuk aplikasi tertentu. File TS dapat dibuka dengan Digia Qt SDK.

## Format File TS

File Digia Qt TS disimpan ke disk dalam format file XML dan dapat dibuka di editor teks apa pun seperti Notepad dan Notepad ++. Beberapa format file lain yang didukung oleh Digia Qt SDK meliputi:

* .pri
* .qm
* .qml
* .qph
* .qrc
* .qs
* .qss
* .ts
* File Antarmuka Pengguna .ui
