## Apa itu file CON?

File CON adalah file kode sumber untuk aplikasi yang dikembangkan untuk **Server Aplikasi Konsep**. Ini digunakan untuk menulis aplikasi untuk pertukaran data antara server dan pengguna. File CON yang dihosting di server dapat diakses dengan browser Web asalkan Klien Konsep diinstal pada sisi pengguna. Server Aplikasi Konsep adalah server web dengan bahasa pemrograman skala kecil yang mungkin memiliki subset database engine (TinDB) [SQL](/id/database/sql/).

File CON dapat dibuka/dijalankan dengan RadGs Concept Application Server.

## Format File CON - Informasi Lebih Lanjut

File CON dihosting di server dan diakses menggunakan browser web di mesin pengguna. Konsep [URL](/id/web/url/) berbeda dari url HTTP normal dan diawali dengan **concept://**.

### Contoh URL Berkas CON

File CON yang dihosting di Server Aplikasi Konsep dapat diakses melalui browser web menggunakan URL seperti:

```
concept://domain.server.com/web_application/main.con.
```
