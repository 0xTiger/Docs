## Apa itu file BWP?

File BWP adalah file skrip halaman web yang dibuat dalam bahasa pemrograman, **BuRg3r**. Ini khusus untuk BuRg3r dan mungkin berisi kode HTML dengan bagian sumber BuRg3r yang dibatasi oleh<?b and ?> tag. BuRg3r adalah bahasa pemrograman tujuan umum yang juga merupakan mesin game yang dikembangkan oleh Fluffy Smoke Entertainment.
