## Apa itu file HXS?

File HXS adalah file bantuan yang dikompilasi dan digunakan untuk file Microsoft Help 2.x. Itu dikompilasi dan dihasilkan dari serangkaian halaman topik dan ditulis dalam subset [HTML](/id/web/html/) yang mirip dengan Format File Bantuan [CHM](/id/web/chm/) sebelumnya. File HXS disimpan dalam format terkompresi dan dibuat dari file proyek utama .HxC, file termasuk .HxF, daftar isi .HxT, file definisi atribut .HxA, dan beberapa indeks .HxK. Metadata file HXS dibuat dari [XML](/id/web/xml/).

File HXS dapat dibuka dengan browser web.

## Format File HXS

File HXS dibuat dan disimpan sebagai file terkompresi dari konten HTML dan metadata XML.
