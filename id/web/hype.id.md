## Apa itu file HYPE?

File HYPE adalah file proyek yang disimpan menggunakan perangkat lunak Hype yang digunakan untuk pembuatan konten web dengan HTML5. Ini berisi elemen web seperti grafik, teks, animasi, informasi adegan, dan elemen lain yang secara kolektif membentuk konten. Konten web dalam file HYPE dapat diekspor ke file web yang dapat dipublikasikan di halaman web.

File HYPE dapat dibuka dengan software [Tumult Hype](https://tumult.com/hype/).

## Format File HYPE

File HYPE disimpan dalam format file berpemilik dan detail internalnya tidak diketahui publik. Konten di dalam file HYPE hanya dapat dilihat saat dibuka di perangkat lunak Tumult Hype.

### Isi File HYPE

File HYPE dibuat saat Anda menggunakan aplikasi Hype untuk membuat konten web untuk berbagai tujuan. Ini menyimpan informasi tentang animasi berbasis keyframe untuk membuat konten seperti eCard, eBook, presentasi, iklan, infografis, dan materi halaman web. Konten ini diberdayakan oleh HTML5 yang memungkinkan konten ditampilkan di perangkat dari desktop hingga iPhone dan iPad. Tumult Hype menampilkan HTML5 canggih, gaya [CSS3](/id/web/css/), dan [JavaScript](/id/web/js/).
