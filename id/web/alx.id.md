## Apa itu file ALX?

File ALX adalah file ActiveX yang disematkan di halaman Web dalam bentuk komponen [HTML](/id/web/html/). Kontrol ActiveX secara resmi dikenal sebagai kontrol OLE atau OCX. File ALX dapat digunakan untuk menyisipkan berbagai fungsi di halaman web seperti menambahkan kontrol pengatur waktu untuk memperbarui konten halaman secara berkala, atau menambahkan kontrol jendela Popup untuk menampilkan bantuan bergaya tooltip berdasarkan tindakan pengguna. Perawatan file ALX ditautkan menggunakan<Param> tag dalam kode HTML laman web. Anda dapat membuka file ALX di Microsoft Internet Explore, Microsoft ActiveX Control Pad, dan editor teks seperti Notepad dan Notepad++.

## Format File ALX - Informasi Lebih Lanjut

File ALX dalam format file teks biasa dan dapat diedit dengan membukanya di editor teks. Kontrol ActiveX dapat disertakan dalam halaman web dengan menjatuhkannya ke halaman web. Namun, untuk membangun halaman web yang dinamis, kontrol terintegrasi bersama dengan bantuan [JScript](/id/web/js/). Kontrol pada halaman web dikaitkan dengan peristiwa dan nilai kontrol, sehingga memberikan kontrol penuh untuk interaksi pengguna.
