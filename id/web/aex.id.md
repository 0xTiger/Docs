## Apa itu file AEX?

File AEX (dengan ekstensi .aex) adalah file [Xbasic](https://documentation.alphasoftware.com/documentation/pages/Ref/Xbasic/index.xml) terkompilasi yang berisi kode program terkompilasi untuk fungsi global. Ini dapat digunakan dalam aplikasi web dengan memanggilnya dari halaman sisi server [.a5w](/id/web/a5w/). File AEX dimuat dan dibongkar dengan memanggil fungsi a5w_load_aex dan a5w_unload_aex masing-masing dari file A5W. Namun, dalam implementasi baru, file-file ini dimuat ke dalam memori dengan memasukkannya ke dalam file a5_application.5i. Bahasa pemrograman Xbasic digunakan oleh perangkat lunak Alpha Anywhere untuk pengembangan aplikasi Seluler dan Web.

## Format File AEX - Informasi Lebih Lanjut

File AEX disimpan ke disk dalam format file biner dan berisi kode fungsi terkompilasi yang ditulis dalam Xbasic. Pernyataan perintah skrip dasar menentukan struktur dan aliran eksekusi termasuk melakukan atau menghentikan tindakan berulang, atau membuat pilihan antara dua langkah atau lebih berdasarkan kondisi. [Referensi Bahasa Xbasic](https://documentation.alphasoftware.com/documentation/pages/Ref/Xbasic/index.xml) dapat dikonsultasikan untuk informasi mendetail tentangnya.
