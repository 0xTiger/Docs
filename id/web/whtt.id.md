## Apa itu file WHTT?

File WHTT adalah versi offline situs web yang diunduh menggunakan perangkat lunak [HTTrack Website Copier](https://www.httrack.com/). Ini berisi semua konten situs web termasuk halaman web [HTML](/id/web/html/), gambar, stylesheet, tautan, dan direktori. Semua halaman situs web diunduh secara rekursif dan disimpan ke disk sebagai cermin luring. Ketika sebuah situs web diunduh oleh perangkat lunak HTTrack, itu dapat dilihat secara offline tanpa perlu terhubung ke internet. Untuk melihat situs web yang diunduh, cukup buka halaman situs web yang dicerminkan di browser dan setiap gambar/tautan akan dimuat dengan benar.

## Format File WHT

File WHTT disimpan sebagai kombinasi halaman web THML, gambar, stylesheet, dan tautan referensi.
