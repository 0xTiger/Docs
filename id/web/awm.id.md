## Apa itu file AWM?

File AWM adalah file proyek yang dibuat oleh program pembuat menu Web, **AllWebMenus**. Ini berisi data terkait proyek untuk membuat menu tarik-turun berbasis [HTML](/id/web/html/). File AWM dikompilasi menjadi paket web final yang diintegrasikan ke situs web untuk menampilkan menu. Ini berisi semua data terkait proyek seperti pengaturan tampilan, entri menu, gaya (vertikal atau horizontal), warna, dan informasi terkait lainnya.

File AWM dapat dibuka dengan **Likno AllWebMenus**.

## Format File AWM - Informasi Lebih Lanjut

File AWM disimpan sebagai file biner yang dikompilasi dan hanya dapat dibuka dengan perangkat lunak Likno AllWebMenus. Detail format file internal file AWM tidak tersedia untuk umum.

AllWebMenus memungkinkan Anda merancang menu [CSS](/id/web/css/) atau menu berbasis [JavaScript](/id/web/js/) secara visual tanpa menggunakan bahasa pemrograman apa pun. Menu dapat sepenuhnya dikustomisasi serta dirancang menggunakan template tampilan standar yang tersedia dengan perangkat lunak.
