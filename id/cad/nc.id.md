## Apa itu file NC?

File NC adalah file Computer Numerical Control (CNC) yang dibuat dengan [perangkat lunak Mastercam CNC](https://www.mastercam.com/). Ini terdiri dari instruksi kontrol numerik untuk mengarahkan gerakan mekanis alat permesinan CNC seperti bor, mesin bubut, pabrik, dan printer 3D. Mesin CNC menggunakan instruksi dari file NC dalam bentuk program berurutan untuk dieksekusi. Instruksi dari file NC ini dimaksudkan untuk memandu mesin untuk produksi komponen.

## Format File NC

File NC disimpan ke disk sebagai file biner dan spesifikasi format file internalnya tidak tersedia secara spesifik. Perangkat lunak Mastercam CNC memungkinkan pengguna membuat jalur alat mesin 2D atau 3D. Jalur alat ini, saat disimpan ke disk, menghasilkan file .nc yang akhirnya digunakan oleh alat mesin CNC untuk membuat komponen yang diinginkan.
