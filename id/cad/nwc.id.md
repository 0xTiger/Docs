## Apa itu file NWC?

File dengan ekstensi .nwc adalah file cache yang dihasilkan oleh perangkat lunak Navisworks Autodesk saat bekerja dengan file CAD. Ini dihasilkan secara otomatis ketika file CAD dibuka

## Format File NWC

NWC adalah format file milik Autodesk dan itulah sebabnya detailnya tidak tersedia. .
