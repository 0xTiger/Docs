## Apa itu file PRT?

File **.PRT** biasanya mengacu pada **CADKEY Part File** yang merupakan format file yang terkait dengan perangkat lunak desain berbantuan komputer (CAD) **CADKEY** yang banyak digunakan di masa lalu. CADKEY dikembangkan oleh CADKEY Corporation dan terutama digunakan untuk desain dan penyusunan 2D dan 3D.

Ekstensi file ".prt" digunakan untuk menyimpan desain bagian atau komponen individual yang dibuat menggunakan CADKEY. File bagian ini dapat berisi sketsa 2D, model 3D, dimensi, dan data desain lainnya yang spesifik untuk komponen tertentu dari perakitan yang lebih besar.

## KUNCI CAD

CADKEY adalah aplikasi perangkat lunak desain berbantuan komputer (CAD) populer yang digunakan untuk desain dan penyusunan 2D dan 3D. Ini awalnya dikembangkan oleh Micro Engineering Solutions, Inc. (Micro Engineering, Inc.), yang kemudian menjadi CADKEY Corporation. Berikut adalah beberapa poin penting tentang CADKEY:

1. **Sejarah:** CADKEY pertama kali dirilis pada tahun 1980-an dan mendapatkan popularitas sebagai perangkat lunak CAD yang hemat biaya dan mudah digunakan.
    





2. **Fitur:** CADKEY menawarkan berbagai fitur untuk membuat dan mengedit gambar 2D dan model 3D dan pengguna dapat mendesain komponen, rakitan, dan komponen mekanis, dan mencakup alat untuk membuat dimensi, rendering, dan dokumentasi.
    





3. **Kemudahan Penggunaan:** CADKEY dikenal dengan antarmuka pengguna yang relatif mudah, sehingga dapat diakses oleh berbagai pengguna, termasuk insinyur, arsitek, dan desainer.
    





4. **Format File:** CADKEY menggunakan format file miliknya sendiri termasuk file ".prt" untuk desain komponen dan file ".asm" untuk rakitan. File-file ini dapat dibagikan dan diedit dalam perangkat lunak CADKEY.
    





5. **Penggunaan Industri:** CADKEY digunakan di berbagai industri, termasuk otomotif, manufaktur, dan arsitektur.
