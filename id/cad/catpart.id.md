## Apa itu berkas CATPART?

File CATPART adalah format file asli perangkat lunak CATIA V5. Ini menyimpan komponen dan rakitan yang dibuat di CATIA V5, aplikasi manufaktur CAD 3D. File-file ini dapat dibuka dan diedit di CATIA V5 atau aplikasi CAD lainnya yang mendukung format file CATPART seperti AutoCAD. File CATPART berisi geometri 3D bersama dengan data struktur model. Itu disimpan ke disk dalam format file biner dan memiliki nomor ajaib `56 35 5F 43 46 56 32`.
