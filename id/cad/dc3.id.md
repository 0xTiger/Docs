## Apa itu berkas DC3?

File dengan ekstensi .DC3 adalah gambar 3D yang dibuat dengan DesignCAD 3D MAX yang merupakan perangkat lunak CAD. Ini digunakan untuk membuat desain berkualitas tinggi, render dan animasi sederhana, dan mendukung impor/ekspor [STL](/id/cad/stl/). File DC3 digunakan sebagai alternatif dari format file .DCD. File DC3 dapat dibuka dengan IMSI DesignCAD 3D Max yang menyediakan berbagai primitif pemodelan 3D yang kuat termasuk kotak, bola, belahan bumi, piramida, silinder, kerucut, dan torus.

## Format File DC3

File DC3 berisi informasi gambar 3D dalam format teks ASCII yang dapat dibuka di editor teks apa pun. Namun, arti isinya hanya dapat dipahami oleh aplikasi yang memahami format file yang mendasarinya.
