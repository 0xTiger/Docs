## Apa itu file RML?

File RML adalah file anotasi yang dibuat dan digunakan oleh aplikasi Volo View oleh [Autodesk](https://autodesk.com). Ini berisi informasi anotasi dan tanda teks yang akan dihamparkan pada gambar untuk membuat catatan. Anotasi ini ditampilkan dalam teks merah di atas gambar dan itulah sebabnya nama file Redline Markup Language. Volo View telah dihentikan dan tidak lagi didukung oleh Autodesk. Konten file RML disematkan dalam gambar CAD sebagai catatan.

## Format File RML

File RML dibuat dan disimpan ke disk dalam format file XML yang merupakan format file universal untuk pertukaran data. Itu dalam format yang dapat dibaca manusia dan dapat dibuka dengan editor teks apa pun seperti Microsoft Notepad, Notepad ++, dan Apple TextEdit.
