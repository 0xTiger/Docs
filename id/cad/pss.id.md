## Apa itu file PSS?

File dengan ekstensi .pss digunakan oleh AutoCAD dan berisi informasi stempel plot. Ini berisi informasi stempel plot seperti tanggal, waktu, skala plot, ukuran kertas, nama tata letak, dan informasi spesifik plotter. Perintah Plot Stamp menempatkan informasi cap ini pada sudut tertentu dari setiap gambar dan mencatatnya ke file. File PSS dapat dibuka dengan aplikasi perangkat lunak Autodesk AutoCAD.
