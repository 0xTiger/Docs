## Apa itu file IBA?

Ekstensi file IBA dibuat oleh Lectra. Format file ini juga disebut sebagai file Lectra Clothing Design Pieces dan termasuk dalam kategori file CAD. Namun, format file utama yang digunakan oleh Lectra Modaris adalah format file MDL, yang merupakan format kepemilikan yang dikembangkan oleh Lectra. File MDL berisi informasi tentang pola 2D dan 3D, penanda, dan model yang dibuat dan diedit dalam perangkat lunak Modaris.

## Hubungan dengan Lectra Modaris

IBA adalah file cad yang terkait dengan perangkat lunak Lectra Modaris. Lectra Modaris adalah perangkat lunak pembuat pola 2D dan 3D yang dikembangkan oleh Lectra, sebuah perusahaan teknologi yang mengkhususkan diri pada perangkat lunak, perangkat keras, dan layanan untuk industri fashion. Lectra Modaris dirancang untuk membantu perancang busana, pembuat pola, dan produsen membuat dan mengembangkan pola dengan lebih efisien dan akurat, dengan fitur seperti penilaian pola, pembuatan penanda, dan pembuatan prototipe virtual. ini terutama menggunakan file CAD (Computer-Aided Design) dalam format berikut:

1. Format file milik Lectra sendiri, yang memiliki ekstensi .mdl. Format ini digunakan untuk menyimpan pola, penanda, dan model yang dibuat dalam perangkat lunak Modaris.
2. Format file DXF (Drawing Interchange Format), yang banyak digunakan dalam industri CAD untuk pertukaran data antar aplikasi perangkat lunak yang berbeda. Format DXF berguna untuk mengekspor potongan pola 2D dari Modaris.
3. Format file AAMA (American Apparel Produsen Association), yaitu format standar untuk file pola yang digunakan dalam industri fashion. Modaris dapat mengekspor pola dalam format AAMA untuk digunakan dalam aplikasi perangkat lunak lain.

Perlu dicatat bahwa Modaris juga mendukung beberapa format file lainnya, termasuk Gerber (ZIP), Lectra (MOD), Lectra (CIT), dan Lectra (LAY). Namun, format file ini kurang umum digunakan di industri fashion.
