## Apa itu file IGS?

File dengan ekstensi .igs (Initial Graphics Exchange) adalah format file pertukaran desain 2D-3D yang tidak bergantung pada spesifikasi format file sumber atau tujuan yang digunakan oleh aplikasi CAD. Ini digunakan untuk bertukar informasi desain tentang diagram sirkuit, wireframe, permukaan bentuk bebas antara dua sistem independen. Ini mirip dengan [IGES](/id/cad/iges/) dan merupakan persyaratan utama produsen untuk penetapan harga dan desain cetakan untuk produk Anda. IGS baru-baru ini telah digantikan oleh format file STEP(.STP) yang lebih baru. File IGS dapat dibuka dengan aplikasi seperti Autodesk, FreeCAD, CADEX CAD Exchanger dan aplikasi serupa lainnya.

## Format File IGS

File IGS dalam format file teks ASCII dan dapat dibuka dengan editor teks apa pun. Setiap catatan dalam file IGS panjangnya 80 karakter. Aplikasi CAD menyimpan dan mengekspor data vektor dalam format file IGS untuk mentransfer file model 3D antara paket perangkat lunak yang berbeda. Meskipun file IGS dapat dibuka di editor teks apa pun, model atau desain hanya dapat dilihat oleh aplikasi visualisasi yang mendukung format file IGS.
