## Apa itu file ASC?

File ASC adalah file teks Autodesk yang menyimpan informasi tentang objek dalam gambar Anda, termasuk lokasi, ukuran, dan warnanya. Ini menjadikannya pilihan populer untuk bertukar gambar antara berbagai program [CAD](/id/cad/). File ASC dapat dibuka dengan produk perangkat lunak Autodesk. Autodesk menyediakan opsi untuk mengimpor file ASC dari dalam aplikasi. File ASC dapat dibuka di editor teks apa pun seperti Microsoft Notepad, Notepad ++, dan Apple TextEdit.

## Format File ASC

File ASC disimpan ke disk dalam format file teks ASCII biasa. Ini bisa dibaca manusia.

### Contoh Berkas ASC

```
ncols 200
nrows 200
xllcorner 119000.00
yllcorner 158000.00
cellsize 5.00
nodata_value -999999
```
