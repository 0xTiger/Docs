## Apa itu file ASM?

File ASM adalah file data yang digunakan oleh [Solid Edge](https://solidedge.siemens.com/en/) perangkat lunak pemodelan, rendering, dan perakitan CAD 3D. Ini berisi objek 3D yang terdiri dari perakitan bagian model. Satu file ASM dapat berisi banyak bagian untuk membentuk seluruh file data. File ASM sama sekali berbeda dari file [assembly language ASM](/id/programming/asm/) dan tidak memiliki koneksi ke file tersebut. Perangkat lunak Siemens Solid Edge digunakan untuk membuka file ASM ini.
