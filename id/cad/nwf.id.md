## Apa itu file NWF?

File dengan ekstensi .nwf adalah format file CAD milik Autodesk yang berisi indeks semua file model yang digunakan oleh proyek. Ini berisi pointer ke elemen yang ditambahkan ke file model untuk referensi. File NWF juga berisi informasi seperti sudut pandang dan redlining. Keuntungan menggunakan format file NWF adalah refleksi langsung dari perubahan pada file CAD asli [mis. DWG](/id/cad/dwg/) karena petunjuk terkait. Penggunaan NWF sangat dianjurkan untuk pengujian bentrokan. File NWF menyimpan hasil bentrokan asli dan setelah modifikasi file CAD, file CAD yang direvisi akan dibuka saat file NWF dibuka kembali.
