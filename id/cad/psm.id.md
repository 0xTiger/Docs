## Apa itu file PSM?

File dengan ekstensi .psm adalah file CAD yang dibuat dan dibuka dengan perangkat lunak pemodelan CAD Solid Edge untuk merancang komponen dan rakitan 3D. Ini berisi model desain 3D untuk membuat bagian-bagian dari lembaran logam. Setelan perangkat lunak Solid Edge menyediakan semua opsi yang diperlukan untuk membuat model 3D yang juga dapat diintegrasikan dengan desain mekanis lainnya. Nesting 2D-nya mendukung format file PSM dan PAR selain format data netral lainnya termasuk DXF dan DWG dengan mengekstraksi data pola datar.

### Format File PSM

PSM adalah format file tertutup dan detailnya tidak tersedia untuk saat ini. Ini memiliki aplikasi tipe MIME/octet-stream. File model logam dapat diimpor di Solid Edge dan diubah menjadi bagian yang dapat digunakan agar dapat diedit. Pemodelan sinkron memberikan kegunaan yang lebih tinggi untuk bekerja dengan file PSM.
