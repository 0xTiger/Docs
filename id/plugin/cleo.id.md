## Apa itu file CLEO?

File .cleo adalah file plugin yang digunakan sebagai bagian dari perpustakaan CLEO untuk seri game GRand Theft Auto. Ini digunakan untuk melakukan modifikasi yang mengubah atau memperluas gameplay. File CLEO menyimpan perpustakaan dalam format file DLL, menambahkan perintah skrip yang berguna (opcode) untuk bekerja dengan file eksternal guna mengubah memori game. Ada berbagai versi CLEO yang dibuat untuk GTA III, GTA Vice City, dan GTA San Andreas.

## Format File CLE

CLEO 4 memungkinkan untuk membuat opcode baru menggunakan sumber CLEO SDK. Tiga contoh paling umum dari file plugin tersebut meliputi:

 * IniFiles.cleo - opcode untuk file INI;
 * FileSystemOperations.cleo - opcode untuk file dan folder.
 * IntOperations.cleo - opcode untuk operasi logis (AND, OR, XOR, dll).

Plugin dipasang mirip dengan skrip. Untuk menginstal plugin CLEO, Anda perlu menyalin file .CLEO ke folder CLEO.

## Tentang CLEO

CLEO adalah plugin yang sangat populer dan dapat diperluas untuk seri game Grand Theft Auto oleh Rockstar Games, memungkinkan penggunaan ribuan mod unik yang mengubah atau memperluas gameplay.
