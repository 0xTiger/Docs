## Apa itu file ZXP?

File ZXP berfungsi sebagai ekstensi, menambah fungsionalitas perangkat lunak Adobe seperti Adobe Photoshop atau Adobe After Effects. Di masa lalu, Adobe Extension Manager memfasilitasi instalasi file ZXP. Saat ini, proses instalasi memerlukan penggunaan alat baris perintah Unified Plugin Installer Agent (UPIA), yang disertakan dengan aplikasi desktop Adobe Creative Cloud.

## Format File ZXP

Istilah ZXP adalah singkatan dari Paket ZIP eXtension. File ZXP digunakan untuk mengemas dan mendistribusikan ekstensi untuk aplikasi Adobe Creative Cloud seperti Adobe Photoshop, Illustrator, InDesign, dan lain-lain. File ZXP dikompresi menggunakan format [ZIP](/compression/zip/), yang memudahkan pendistribusian dan pemasangan ekstensi.

Di dalam file ZXP, Anda biasanya akan menemukan file yang diperlukan untuk ekstensi, termasuk [HTML](/web/html/), [CSS](/web/css/), JavaScript, gambar, dan aset lainnya yang diperlukan untuk menyempurnakan atau menambahkan fitur ke perangkat lunak Adobe Creative Cloud.

### Bagaimana cara menginstal berkas ZXP?

File ZXP diinstal menggunakan Adobe Extension Manager atau aplikasi Creative Cloud Desktop. Pengguna cukup mengklik dua kali pada file ZXP, dan manajer ekstensi akan menangani proses instalasi.

File ZXP dirancang untuk bekerja di berbagai sistem operasi, termasuk Windows dan macOS, sehingga memudahkan pengembang untuk membuat ekstensi yang dapat digunakan oleh khalayak luas.

### Keamanan Format File ZXP

File ZXP dapat ditandatangani secara digital oleh pengembang, memastikan bahwa ekstensinya tidak dirusak dan berasal dari sumber tepercaya. Hal ini membantu menjaga keamanan dan integritas aplikasi Adobe Creative Cloud.

Jika Anda bekerja dengan ekstensi Adobe Creative Cloud atau mengembangkannya sendiri, Anda mungkin menemukan file ZXP sebagai sarana untuk mengemas dan mendistribusikan ekstensi Anda. Namun, cara bekerja dengan file ZXP secara spesifik mungkin dapat diubah atau diperbarui oleh Adobe, jadi sebaiknya lihat dokumentasi Adobe terbaru untuk mendapatkan informasi paling akurat dan terkini.
