## Apa itu file AMXX?

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. Ini adalah platform skrip yang memungkinkan administrator server untuk menyesuaikan dan memperluas fungsionalitas server mereka. AMXX digunakan bersama dengan sistem manajemen plugin MetaMod yang memungkinkan beberapa plugin berjalan secara bersamaan di server tanpa konflik. Fitur yang ditawarkan plugin AMXX antara lain perintah admin, modifikasi gameplay, dan manajemen server.
