## Apa itu file AVX?

AVX adalah ekstensi perangkat lunak yang digunakan oleh produk Avid seperti Media Composer dan Xpress. Ini meningkatkan fungsionalitas perangkat lunak dengan menggabungkan efek video khusus dan mendukung impor format file baru. Ekstensi ini memberdayakan pengembang pihak ketiga untuk memperluas fitur inti aplikasi.

Anda dapat membuka file AVX menggunakan **Avid Media Composer** atau **Avid Xpress**.
