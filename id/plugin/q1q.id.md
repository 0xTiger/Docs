## Apa itu berkas Q1Q?

File .q1q adalah file plugin Photoshop yang dikembangkan oleh Flaming Pear Software untuk membuat planet 3D. Ini memungkinkan Anda mengunduh data awan LANGSUNG, udara, tekstur, iklim, dan banyak detail lainnya. Ini juga memungkinkan Anda mengekspor semua peta tekstur ke file multi-layer dan menghasilkan peta untuk animasi 3D dan pembuatan peta.

## Format Berkas Q1Q

File plugin Q1Q disimpan dalam format file milik Flaming Pear dan detail format file internalnya tidak tersedia untuk umum. Gambar berikut menunjukkan Antarmuka Pengguna plugin Q1Q ketika dimuat di Photoshop.

{{<figure src="../q1q.png" alt="Format File Plugin LunarCell" >}}

## Bagaimana Cara Membuka File Q1Q?

Anda dapat membuka file Q1Q menggunakan **Adobe Photoshop 2023** dan **Corel PaintShop Pro 2023**.
