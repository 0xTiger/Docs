## Apa itu berkas 8BI8?

File .8BI8 adalah file plugin yang menyempurnakan perangkat lunak Photoshop dengan memperkenalkan filter atau efek tambahan. Itu harus ditempatkan di Folder Plug-in di dalam direktori Photoshop agar berfungsi dengan baik.
