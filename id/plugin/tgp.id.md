## Apa itu file TGP?

File TGP berfungsi sebagai plugin yang digunakan dalam Terragen Classic, sebuah program yang dirancang untuk menghasilkan pemandangan alam fotorealistik. File ini merangkum data yang meningkatkan fungsionalitas Terragen saat instalasi. File TGP digunakan untuk memperkenalkan beragam fitur ke Terragen, termasuk efek kamera dan opsi lanskap tambahan.

## Tentang Terragen Klasik

Terragen adalah perangkat lunak pembuatan dan rendering lanskap yang dikembangkan oleh Planetside Software. Versi Klasik mengacu pada versi perangkat lunak yang lebih lama, karena ada juga versi yang lebih baru yang disebut **Terragen 4**.

Terragen Classic dikenal karena kemampuannya dalam menghasilkan lanskap dan medan yang realistis. Ini memungkinkan pengguna untuk membuat lingkungan alam yang detail dan dapat dipercaya, termasuk gunung, lembah, danau, dan langit. Perangkat lunak ini menyertakan fitur untuk pembentukan medan, pemetaan tekstur, dan rendering.
