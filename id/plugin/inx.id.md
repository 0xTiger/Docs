## Apa itu file INX?

File INX adalah file plugin yang digunakan oleh editor grafis vektor sumber terbuka, **Inkscape**. Ini berisi informasi tentang program termasuk nama dan lokasi fungsi dalam menu. File INX juga menentukan skrip untuk ekstensi yang mencakup informasi seperti jenis skrip, juru bahasa, lokasi, dan argumen variabel.

## Format File INX

File INX meningkatkan fungsionalitas Inkscape dengan menggabungkan fitur seperti pengimpor file dan filter grafis. Disimpan dalam format XML, file-file ini biasanya ditemukan di direktori \​extensions\​ dalam instalasi Inkscape.

## Bagaimana Cara Membuka file INX?

Anda dapat membuka file INX menggunakan program **Inkscape**.
