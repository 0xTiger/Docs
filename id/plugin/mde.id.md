Apa itu file MDE?

File .MDE adalah versi kompilasi database Access (.mdb atau .accdb). Saat Anda mengkompilasi database Access menjadi file .MDE, database tersebut tidak lagi dapat diedit menggunakan alat desain Access standar. Tujuan utama pembuatan file .MDE adalah untuk mendistribusikan aplikasi database tanpa memaparkan kode sumber aslinya.

## Format Berkas MDE

File MDE dibuat dengan mengkompilasi kode VBA, formulir, laporan, dan objek lainnya. Hal ini menghasilkan pembuatan format file biner MDE. File .MDE yang dihasilkan dapat didistribusikan kepada pengguna yang dapat menjalankan aplikasi tetapi tidak dapat mengubah desain atau melihat kode aslinya. File MDE sebenarnya adalah versi kompilasi dari [file .MDA](/database/mda/). Mendistribusikan file .MDE dapat membantu melindungi kekayaan intelektual dengan mencegah pengguna mengakses atau memodifikasi kode sumber. Itu juga dapat meningkatkan kinerja saat kode dikompilasi dan dioptimalkan.
