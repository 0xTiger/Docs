## Apa itu file NDS?

Ekstensi file .NDS umumnya dikaitkan dengan **File Kompilasi Nero DiscSpan**. Nero DiscSpan adalah fitur di **Nero Burning ROM**, program pembuat cakram optik yang populer; fitur ini memungkinkan pengguna untuk menyebarkan data dalam jumlah besar ke beberapa disk.

Saat Anda membuat kompilasi DiscSpan di Nero Burning ROM, ia membagi data menjadi segmen-segmen lebih kecil yang muat pada masing-masing disk; file .nds dibuat sebagai file deskriptor yang berisi informasi tentang tata letak dan data yang direntangkan di seluruh disk.

Jika Anda memiliki file .nds dan segmen disk yang sesuai, Anda dapat menggunakan Nero Burning ROM atau Nero DiscSpan untuk memulihkan data asli. Nero DiscSpan akan meminta Anda memasukkan setiap disk secara berurutan untuk merekonstruksi kompilasi lengkap.
