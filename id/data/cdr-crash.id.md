## Apa itu file CDR?

**File CDR** adalah singkatan dari **"File Data Pengambilan Data Crash"**. File-file ini dikaitkan dengan perekam data peristiwa (EDR) di kendaraan. EDR, sering disebut sebagai "kotak hitam" pada mobil, mencatat berbagai data terkait performa kendaraan dan kejadian yang terjadi sebelum dan selama kecelakaan. Data ini sangat penting untuk rekonstruksi dan analisis kecelakaan.

## Data CDR

Data spesifik yang terdapat dalam file .cdr dapat mencakup informasi seperti:

1. Kecepatan kendaraan
2. RPM Mesin
3. Penggunaan rem
4. Penyebaran kantung udara
5. Penggunaan sabuk pengaman
6. Masukan kemudi
7. Tingkat akselerasi dan deselerasi
8. Waktu dan tanggal acara
9. Informasi identifikasi kendaraan

Penegakan hukum, perusahaan asuransi, dan penyelidik kecelakaan dapat menggunakan file .cdr untuk lebih memahami keadaan seputar kecelakaan mobil. Perangkat lunak khusus biasanya diperlukan untuk menafsirkan dan menganalisis data dalam file-file ini.

## Perangkat Lunak Sistem Pengambilan Data Kerusakan Bosch

Sistem Pengambilan Data Kecelakaan (CDR) Bosch adalah solusi perangkat lunak dan perangkat keras khusus yang dirancang untuk pengambilan dan analisis data dari perekam data kejadian (EDR) di kendaraan. EDR biasanya disebut sebagai "kotak hitam" pada kendaraan dan mencatat informasi penting terkait kinerja kendaraan dan kejadian menjelang dan selama kecelakaan.

Bosch adalah salah satu perusahaan yang menyediakan alat dan perangkat lunak untuk mengakses dan menganalisis data dari EDR tersebut. Sistem CDR Bosch banyak digunakan dalam bidang rekonstruksi kecelakaan dan forensik otomotif. Hal ini memungkinkan para ahli untuk mengekstrak dan menafsirkan data yang disimpan dalam EDR untuk mendapatkan wawasan tentang keadaan kecelakaan kendaraan.

Beberapa fitur dan fungsi utama Sistem CDR Bosch meliputi:

1. **Pengambilan Data:** Perangkat lunak dan perangkat keras dapat berinteraksi dengan EDR kendaraan untuk mengekstrak data seperti kecepatan kendaraan, penggunaan rem, pengembangan kantung udara, RPM mesin, dan parameter penting lainnya.
    



2. **Analisis Data:** Perangkat lunak ini menyediakan alat untuk menganalisis dan menafsirkan data yang diambil dari EDR. Hal ini dapat membantu dalam memahami peristiwa-peristiwa yang menyebabkan terjadinya tabrakan dan dampak kekuatan selama tabrakan.
    



3. **Pembuatan Laporan:** Sistem CDR sering kali memungkinkan pengguna membuat laporan yang merangkum temuan berdasarkan analisis data. Laporan-laporan ini dapat bermanfaat bagi penegak hukum, perusahaan asuransi, dan penyelidik kecelakaan.
