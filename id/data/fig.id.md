## Apa itu file Gambar?

File FIG adalah file khusus yang berisi gambar atau grafik yang dibuat menggunakan program MATLAB, yang digunakan untuk menghitung dan menganalisis data. File ini menyimpan informasi tentang gambar atau grafik yang digambar MATLAB untuk membantu orang melihat dan memahami data dengan lebih baik. Ini seperti sebuah wadah yang menyimpan semua detail tentang grafik.

.FIG adalah format file gambar MATLAB default. Ini menyimpan gambar lengkap, termasuk semua sumbu, label, dan pengaturan. Untuk menyimpan gambar dalam format ini, Anda dapat menggunakan fungsi `savefig`:

## Bagaimana cara membuat file FIG dengan MATLAB?

Untuk membuat file FIG menggunakan MATLAB, ikuti langkah-langkah berikut:

1. **Pilih data Anda:** Pilih variabel yang ingin Anda buat plotnya dari panel "Ruang Kerja" di MATLAB.

2. **Pilih jenis plot:** Buka tab "PLOT" dan klik jenis plot yang Anda inginkan.

3. **Simpan plot sebagai file Gambar:**

     - Buka menu "Berkas".
     - Pilih "Simpan" atau "Simpan Sebagai".
     - Berikan nama untuk file Anda dan putuskan di mana Anda ingin menyimpannya.
     - Klik "Simpan" untuk membuat file FIG Anda.

## Bagaimana cara membuka file FIG?

Untuk membuka file FIG di MATLAB, ikuti langkah-langkah berikut:

1. **Buka MATLAB:** Mulai MATLAB di komputer Anda.

2. **Navigasi ke tab "PLOTS":** Klik plot di tab "PLOTS", pilih jenis plot yang Anda inginkan.

3. **Buka file Gambar:**

     - Buka menu "Berkas".
     - Pilih "Buka."
     - Arahkan ke lokasi penyimpanan file FIG Anda, pilih, dan klik "Buka."
