## Apa itu file GED?

**File GEDCOM (Komunikasi Data Silsilah)**, biasanya dengan ekstensi .ged, adalah format file standar yang digunakan untuk bertukar data silsilah antara program perangkat lunak silsilah yang berbeda; File GEDCOM adalah file teks biasa yang berisi informasi tentang individu, keluarga, peristiwa, dan data silsilah lainnya; tujuan GEDCOM adalah untuk menyediakan format umum untuk pertukaran informasi silsilah.

File GEDCOM biasanya memiliki ekstensi file .ged; ekstensi ini digunakan untuk menunjukkan bahwa file mengikuti **standar GEDCOM** untuk pertukaran data silsilah; jadi ketika Anda menyimpan atau mengekspor data silsilah Anda dari program perangkat lunak silsilah, file yang dihasilkan sering diberi nama seperti `namafileAnda.ged`.

Misalnya:

- pohon keluargamu.ged
- john_doe_family.ged
- data_keturunan.ged

## Poin Penting tentang File GEDCOM

Berikut adalah beberapa poin penting tentang file GEDCOM:

1.  **Format Teks Biasa:** File GEDCOM adalah file teks biasa; yang artinya dapat dibuka dan dilihat dengan editor teks apa pun; format ini memfasilitasi pembagian dan kompatibilitas yang mudah antara berbagai aplikasi perangkat lunak silsilah.
    
2.  **Struktur:** File GEDCOM memiliki struktur hierarki; data disusun menjadi catatan; setiap record mempunyai tag khusus yang menunjukkan jenis informasi yang dikandungnya, misalnya, nama seseorang mungkin diwakili oleh tag NAMA.
    
3.  **Individuals and Families:** GEDCOM files typically include information about individuals such as names, dates of birth and places of birth; they also include details about family relationships, marriages and other events.
    
4.  **Interoperabilitas Perangkat Lunak:** Karena GEDCOM adalah format standar; ini memungkinkan pengguna untuk memindahkan data silsilah mereka di antara program perangkat lunak silsilah yang berbeda; ini sangat berguna bagi pengguna yang mungkin beralih ke aplikasi perangkat lunak lain atau berbagi penelitian mereka dengan orang lain.
    
5.  **Versi:** GEDCOM telah melalui berbagai versi dengan setiap versi memperkenalkan peningkatan dan fitur baru; versi yang paling umum digunakan adalah GEDCOM 5.5. 

## Bagaimana cara membuka file GED?

Untuk membuka file GED, biasanya Anda memerlukan software silsilah yang mendukung format GEDCOM. Beberapa opsi perangkat lunak silsilah yang populer meliputi:

- **Pembuat Pohon Keluarga**
- **Leluhur.com**
- **Warisanku**
- **Kakek** (gratis dan sumber terbuka)
- **Pohon Keluarga Warisan**

Setelah Anda memilih perangkat lunak silsilah Anda, cari opsi seperti Impor, Buka GEDCOM atau Impor dari File di menu atau toolbar.

Karena file GED disimpan dalam format teks, Anda juga dapat membuka file dengan editor teks biasa misalnya

- **Buku Catatan**
- **Buku Catatan++**
- **Apple TextEdit**
- Editor teks apa pun
