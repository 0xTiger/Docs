## Apa itu file RVM?

File data **RVM** terkait dengan AVEVA PDMS. File **RVM** adalah file proyek Model Sistem Manajemen Desain Pabrik AVEVA. Plant Design Management System (PDMS) AVEVA adalah sistem desain 3D paling populer yang menggunakan teknologi data-centric untuk mengelola proyek. Banyak aplikasi tersedia untuk membuka dan mengonversi file RVM ke format lain seperti [3DS](/id/3d/3ds/).

## Format File RVM

Tidak banyak detail yang dapat ditemukan tentang format file RVM selain itu dapat disimpan dalam format biner dan ASCII. File RVM mendukung entitas seperti:

* Semua geometri
* Atribut disimpan di grup
* Tekstur (melalui file RVS)
* Kamera dan trek kamera (melalui file RVS)
* Klip pesawat (melalui file RVS)
* Tanda (melalui file RVS)
* Tag (melalui file RVS)
* Label (melalui file RVS)
* Transparansi (melalui file RVS)
* Titik asal PDMS
