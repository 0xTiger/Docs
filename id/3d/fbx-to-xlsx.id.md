## Tentang FBX ke XLSX

FilmBox, sering dikenal sebagai FBX, adalah format file 3D yang banyak digunakan dan awalnya dibuat oleh Kaydara untuk meningkatkan kompatibilitas dengan MotionBuilder. Pada tahun 2006, Autodesk Inc. mengambil kendali format ini, memperkuat statusnya sebagai standar untuk pertukaran data 3D dengan mudah di berbagai aplikasi perangkat lunak 3D.

**XLSX** adalah format yang dikenal luas untuk dokumen Microsoft Excel yang diperkenalkan oleh Microsoft bersamaan dengan peluncuran Microsoft Office 2007. Format ini mematuhi organisasi terstruktur berdasarkan Konvensi Pengemasan Terbuka, sebagaimana dijelaskan secara rinci di [Bagian 2](https:// www.ecma-international.org/publications/standards/Ecma-376.htm) dari standar OOXML ECMA-376. Format baru ini pada dasarnya adalah paket zip yang berisi banyak file XML.

Anda dapat menggunakan aplikasi konversi FBX ke XLSX yang tersedia online untuk mengonversi file FBX Anda ke format XLSX. Aplikasi online ini tidak mengharuskan Anda memasang perangkat lunak khusus apa pun untuk konversi tersebut.
