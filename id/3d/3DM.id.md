## Apa itu file 3DM?

File dengan ekstensi .3dm adalah format file sumber terbuka yang digunakan untuk perangkat lunak grafik 3D. Ini berisi model 3D bersama dengan elemen dependennya seperti informasi permukaan, titik, dan kurva. Format file 3DM dikembangkan oleh inisiatif [openNURBS](https://github.com/mcneel/opennurbs) untuk mentransfer geometri 3-D antar aplikasi secara akurat. File 3DM dapat dibuka dan dikonversi dengan aplikasi seperti Rhinoceros, SAP VEViewer, Moment of Inspiration, dan Right Hemisphere Deep View.

## Format File 3DM

[openNURBS](https://github.com/mcneel/opennurbs), toolkit sumber terbuka, berisi spesifikasi format file 3DM, dokumentasi, pustaka kode sumber C++, dan rakitan .NET 2.0 untuk membaca dan menulis format file.

### File Contoh 3DM

Beberapa contoh file 3DM dapat diunduh dari bagian openNURBS [contoh](https://github.com/mcneel/opennurbs/tree/7.x/example_files). Ini dapat dimuat dan ditampilkan menggunakan toolkit openNURBS.

## Bagaimana cara membaca atau menulis file 3DM?

Library [openNURBS](https://github.com/mcneel/opennurbs) memungkinkan siapa saja membaca dan menulis format file 3DM tanpa memerlukan Rhino. Ini menyediakan kode sumber C++ untuk perpustakaan yang akan membaca dan menulis model 3D menggunakan format file openNURBS. Toolkit ini juga menyediakan alat evaluasi NURBS dan alat manipulasi tampilan geometris dan 3D dasar serta menyertakan kode sumber untuk beberapa contoh program. Forum dukungan [Rhinoceros](https://discourse.mcneel.com/c/opennurbs/6) adalah tempat konten dan diskusi yang kaya untuk berbagi masalah yang dihadapi komunitas 3DM dan mendapatkan solusinya.
