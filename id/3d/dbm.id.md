## Apa itu file DBM?

File DBM (Daz Brick Material) adalah file jaringan Shader Mixer yang dibuat dengan aplikasi pemodelan karakter 3D, **DAZ Studio**. Ini menjelaskan materi mode 3D yaitu volume permukaan/perpindahan/interior. DAZ memungkinkan Anda membuat shader khusus Anda sendiri dengan menghubungkan serangkaian batu bata menjadi satu. Ini digabungkan secara kolektif sebagai file DBM yang membentuk jaringan shader. Ini adalah file DBM yang menentukan bagaimana suatu bahan diarsir.

Aplikasi yang dapat **membuka file DBM** termasuk DAZ 3D DAZ Studio 4.

## Format File DB

File DBM disimpan ke disk dalam format file [XML](/id/web/xml/).
