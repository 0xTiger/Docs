## Tentang FBX ke STL

FBX, FilmBox, adalah format file 3D populer yang dikembangkan oleh Kaydara untuk MotionBuilder. Itu diakuisisi oleh Autodesk Inc pada tahun 2006 dan sekarang menjadi salah satu format file pertukaran data 3D utama yang digunakan oleh banyak alat 3D. [STL](/id/cad/stl/), singkatan dari stereolithrography, adalah format file yang dapat dipertukarkan untuk representasi geometri permukaan 3 Dimensi.

Merupakan persyaratan umum untuk mengonversi [FBX](/id/3d/fbx/) ke format file STL dan beberapa aplikasi tersedia online yang dapat mengonversi FBX ke STL secara GRATIS. Jika Anda adalah pengembang aplikasi yang tertarik untuk menyematkan fungsionalitas FBX ke STL di aplikasi .NET, Java, atau Python, Anda dapat mempertimbangkan untuk menggunakan API Aspose.3D untuk tujuan ini.
