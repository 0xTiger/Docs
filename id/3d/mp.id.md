## Apa itu file MP?

File MP adalah file proyek yang dibuat dengan Personal Learning Edition (PLE) dari program pemodelan 3D, **Maya**. Ini hanya didukung oleh versi PLE dari Maya dan murni untuk penggunaan non-komersial. Untuk tujuan ini, edisi Maya PLE menyertakan tanda air pada file yang diekspor. Maya PLE menyediakan fungsionalitas terbatas dan hanya dapat diinstal di komputer pribadi. File MP berisi semua informasi tentang proyek yang dibuat dengan PLE dan dapat dibuka dengan Autodesk Maya PLE.
