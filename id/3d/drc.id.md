## Apa itu file DRC?

File dengan ekstensi .drc adalah format file 3D terkompresi yang dibuat dengan perpustakaan Google Draco. Google menawarkan Draco sebagai pustaka sumber terbuka untuk mengompresi dan mendekompresi mesh geometris 3D dan awan titik, serta meningkatkan penyimpanan dan transmisi grafik 3D. Itu mengkodekan data input dan menyimpannya sebagai file .drc. Pada sisi penerima, API membaca file .drc dan dapat menampilkannya sebagai file [PLY](/id/3d/ply/) atau [OBJ](/id/3d/obj/). File keluaran terkompresi memungkinkan pengguna mengunduh aplikasi lebih cepat dan menyediakan pemuatan cepat grafik 3D di browser.

## Format File DRC

Detail format file untuk DRC tidak tersedia untuk umum. Google telah menghosting perpustakaan Draco di [GitHub](https://github.com/google/draco/tree/master/docs) untuk referensi pengembang. Ini menyediakan pembuat enkode dan dekoder untuk membuat dan mengonversi file DRC.
