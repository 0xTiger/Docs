## Apa itu file MPT? ##

File dengan ekstensi .mpt adalah file template [Microsoft Project](https://products.office.com/en-us/project/project-and-portfolio-management-software). Ini berisi informasi dasar dan pengaturan struktur dan dokumen untuk membuat file [MPP](/id/project-management/mpp/). Templat ini menyediakan pengaturan default yang penting untuk proyek tertentu, seperti jadwal atau informasi anggaran. Meskipun, itu tidak dapat menyimpan data khusus proyek seperti tugas, sumber daya, atau tugas. Setelah dimodifikasi, file template dapat disimpan sebagai file MPP standar untuk dikerjakan lebih lanjut. File template master, berjudul "Global.mpt" tersedia di Microsoft Project, dibuka untuk membuat template individual. File MPT menggunakan *application/vnd.ms-project* dan *application/x-project* sebagai tipe MIME standar. Windows, Linux, dan macOS semuanya dapat membuka file MPT, bersama dengan Moos Project Viewer, Microsoft Project, dan MPXJ.

## Masalah untuk membuka file MPT? ##

Berikut adalah daftar beberapa masalah umum yang mungkin timbul dan menyebabkan kesalahan fungsi format MPT:

* Tidak adanya perangkat lunak pendukung
* File rusak
* File yang terinfeksi karena virus
* Tidak ada hak akses di sistem untuk membuka file
* Drive usang di sistem Anda
 *   Extension of the file is renamed
