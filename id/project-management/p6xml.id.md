## Apa itu file P6XML? ##

Ekstensi file P6XML memiliki keterbacaan yang sangat baik, kemampuan beradaptasi, dan penggunaan luas yang menjadikannya format yang ideal. Primavera P6 Professional memiliki fungsionalitas untuk mengekspor dan mengimpor file [XER](/id/project-management/xer) dan file XML. XML adalah singkatan dari **Extensible Markup Language**. Format umum ini memungkinkan pengguna untuk berbagi informasi proyek antara database Primavera P6. Data proyek dapat dengan mudah disimpan dalam file teks melalui file XML; ini mempermudah berbagai program perangkat lunak untuk bertukar data. Data tersebut dapat dengan mudah dibagikan dan dikelola oleh pengguna Primavera P6.

## Keuntungan Format File P6XML ##

* Sangat mudah bagi pengguna Primavera untuk menentukan Garis Dasar mana yang diekspor dan diimpor menggunakan file XML
* Ekspor XML dapat berisi semua Rencana Aksi rinci Proyek
* Setiap pengguna yang menggunakan Primavera 6 yang tidak dapat mengedit, menambah atau menghapus item data global tertentu, seperti Kode Aktivitas, Kalender, dan Sumber Daya tidak akan dapat mengimpornya
* Selain pengaturan General dan Project Security Profile yang digunakan oleh Primavera, item data yang akan mempengaruhi atau mengubah data di database tidak akan diimpor

## Mengekspor File XML dari Primavera P6 ##

Untuk Mengekspor proyek Profesional P6 dan data proyek ke file P6XML, gunakan Oracle Primavera Cloud. Anda dapat mengunduh file saat sedang dibentuk dari aplikasi desktop P6 Professional dan mengimpornya ke Primavera Cloud. Sebelum menjalankan proses ekspor tanpa gangguan, Anda harus mengatur apakah Anda terhubung ke database P6 Professional atau P6 EPPM. Beberapa fitur proses ekspor mungkin berbeda bergantung pada database Anda. Memiliki pengaturan P6 Specialized dengan database P6 Mahir, proses ekspor beroperasi secara lokal di sistem Anda. Memiliki database P6 Professional, Anda dapat dengan mudah mengekspor dan membuka file XML P6 dari pengaturan di sistem Anda.

Berikut adalah beberapa langkah yang menjelaskan proses ekspor:

* Buka keinginan proyek Primavera P6
* Setelah itu pilih Export pada tab File
* Pilih "Primavera P6 â€“ (XML)". Setelah itu, klik Berikutnya
* Klik dua kali proyek yang akan Anda ekspor
* Pilih pemotongan untuk jalur tempat Anda ingin meletakkan file yang diekspor dan pilih "Selesai".
* Pilih opsi bagaimana Anda ingin menyimpan file
* Jika tata letak tingkat proyek tidak diperlukan dengan file ekspor, penting untuk mengosongkan pilihan "Ekspor semua tata letak tingkat proyek".
* Pilih selesai, dan kemudian Anda akan menerima konfirmasi ekspor XML

### Tips Penting ###

* Setiap kali tersambung ke database EPPM P6, administrator harus meneruskan URL sumber konten ke bidang URL P6 di halaman Umum Pengaturan Aplikasi di P6
* Lebih mudah untuk mengekspor banyak proyek ke satu file XML
* Data global yang dialokasikan ke proyek diekspor
* Lebih mudah untuk mengekspor file XML tanpa akses ke semua sumber daya
  


## Masalah untuk Membuka file P6XML? ##

Beberapa masalah umum yang mungkin timbul dan menyebabkan kesalahan fungsi format P6XML adalah sebagai berikut:

* Tidak adanya perangkat lunak pendukung
* File rusak
* File Terinfeksi karena virus
* Deskripsi P6XML secara tidak sengaja terhapus dari registri Windows
* Tidak ada hak akses di sistem untuk membuka file
* Drive usang di sistem Anda
* Ekstensi file diubah namanya
