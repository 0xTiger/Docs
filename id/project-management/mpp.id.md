## Apa itu file MPP?

File dengan ekstensi .mpp adalah file data Microsoft Project yang menyimpan informasi terkait manajemen proyek secara terintegrasi. Microsoft telah mengembangkan format file berpemilik ini agar kompatibel dengan Microsoft Project (MSP), produk perangkat lunak manajemen proyek mereka. Selain MPP, MSP dapat bekerja dengan skema XML proyek. Banyak aplikasi dan API memungkinkan MPP dikonversi ke format file lain. Microsoft sekarang memiliki Server Proyek online tempat file manajemen proyek dapat diunggah untuk kolaborasi oleh banyak pengguna.

## Format File MPP ##

Microsoft mengembangkan dan menerbitkan MPP sebagai format file biner dan spesifikasinya belum dipublikasikan oleh Microsoft hingga saat ini. Oleh karena itu, struktur dan detail file internal tidak tersedia untuk pengembang. Ada beberapa API yang tersedia online yang mendukung pembacaan/penulisan ke file MPP, tetapi akurasi 100% tidak pernah tercapai. Microsoft telah memperkenalkan beberapa fitur ke aplikasi manajemen proyeknya dari waktu ke waktu, dan begitu pula fitur format file MPP.

File MPP menggunakan tipe MIME berikut:

* aplikasi/vnd.ms-proyek
* aplikasi/msproj
* aplikasi/proyek ms
* aplikasi/proyek x-ms
* aplikasi/x-ms-proyek
* aplikasi/x-dos_ms_project
* aplikasi/mpp
* zz-application/zz-winassoc-mpp

## Masalah untuk membuka file MPP? ##

Berikut adalah daftar beberapa masalah umum yang mungkin timbul dan menyebabkan kesalahan fungsi format MPP:

* Tidak adanya perangkat lunak pendukung
* File rusak
* File Terinfeksi karena virus
* Tidak ada hak akses di sistem untuk membuka file
* Drive usang di sistem Anda
* Ekstensi file diubah namanya
