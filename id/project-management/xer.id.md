## Apa itu file XER?

Format file XER adalah format file proyek eksklusif yang digunakan oleh aplikasi perencanaan dan manajemen proyek Primavera P6. Seperti format file [MPP](/id/project-management/mpp/), format file ini menyimpan data yang terkait dengan perencanaan proyek seperti tugas, sumber daya, jadwal, dan materi. Primavera dikembangkan oleh Oracle dan menyediakan perencanaan proyek yang lengkap, dan menemukan aplikasinya di beberapa bidang praktis seperti minyak dan gas, konstruksi, manufaktur, dll. Primavera membantu manajer proyek merencanakan, menjadwalkan, dan mengendalikan program skala besar dan proyek individu dengan mengintegrasikan dengan manajemen keuangan dan sistem manajemen sumber daya manusia. Banyak perangkat lunak dan API dapat membaca dan mengonversi file XER ke format file proyek populer lainnya seperti XML dan MPP.

## Format File XER ##

File XER adalah file teks dan dapat dengan mudah dibuka dengan editor teks. Membuka data terkait proyek dalam editor teks dapat membantu, tetapi tidak mudah untuk memahaminya dengan membuat tautan secara manual. Format ini adalah yang paling banyak digunakan untuk bertukar data dengan database P6 lainnya. Ini memiliki tingkat kontrol yang relatif rendah atas data yang telah diimpor dari format lain. Keunggulan XER pada format XML yang setara mencakup kemampuan untuk memindahkan kode dan kalender global ke tingkat proyek dan kode peta, UDF, sumber daya, dan kalender ke nilai yang ada.


## Masalah untuk Membuka file XER? ##

Beberapa masalah umum yang mungkin timbul dan menyebabkan kesalahan fungsi format XER adalah sebagai berikut:

* Tidak adanya perangkat lunak pendukung
* File rusak
* File Terinfeksi karena virus
* Deskripsi XER tidak sengaja terhapus dari registri Windows
* Tidak ada hak akses di sistem untuk membuka file
* Drive usang di sistem Anda
* Ekstensi file diubah namanya
