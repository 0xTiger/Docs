## Apa itu file WUD?

File dengan ekstensi .wud adalah gambar disk yang dibuat dari disk game Wii U. File image disk yang dihasilkan dapat dimuat oleh emulator video game seperti [Cemu](https://cemu.info/) untuk mengemulasi game. File WUD berukuran besar dan karenanya dibuat sebagai beberapa segmen agar dapat ditransfer dengan mudah. Informasi game diekspor sebagai segmen file WUD dengan ukuran file sekitar 2 GB. Aplikasi pembuka, seperti Cemu, membutuhkan semua file tersegmentasi ini untuk digabungkan menjadi satu file WUD sebelum membukanya.

## Format File WUD

File WUD disimpan ke disk dalam format file berpemilik dan dapat dimuat di emulator Cemu untuk meniru aplikasi Wii U di PC. Karena ukurannya yang besar, file WUD [dikompresi](/id/compression/) untuk mengurangi ukuran filenya. File terkompresi yang dihasilkan dikenal sebagai file WUX.
