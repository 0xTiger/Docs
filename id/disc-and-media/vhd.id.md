## Apa itu file VHD?

VHD adalah singkatan dari Virtual Hard Disk. Ini adalah format file yang mewakili drive hard disk virtual. Data yang ditemukan pada hard disk drive fisik mungkin terlibat di dalamnya seperti halnya partisi hard disk dan sistem file yang terdiri dari folder.

Untuk mesin virtual, file VHD dianggap sebagai hard disk. Dalam versi Windows modern, ini sudah ada di dalamnya. Jenis hard disk virtual khusus ini menyediakan fasilitas untuk sistem operasi yang berbeda untuk berada di mesin host yang sama. Ini memberikan peluang besar bagi para pengembang untuk menguji perangkat lunak pada beberapa OS tanpa melakukan proses instalasi hard disk lain. Ada banyak keuntungan menggunakan format file file VHD ini karena menyediakan banyak fitur.


## Sejarah Singkat ##

**Connectix** mengembangkan format file ini sebagai produk PC virtual untuk penggunaannya. Pada tahun 2003 Microsoft mengakuisisi Connectix dan meluncurkan versi lain dari format ini seperti VHDx bersama dengan fitur dan fungsionalitas yang lebih baik yang tidak ada di versi pertama. Pada 2012 VHDx ditambahkan ke Windows. Itu ditambahkan ke Windows untuk menambah lebih banyak kapasitas penyimpanan, perlindungan terhadap korupsi data, pengoptimalan untuk mencegah penurunan kinerja. Fitur-fitur ini ditambahkan untuk disk fisik dengan sektor besar.

## Format File VHD ##

Sistem file host diakses untuk menyimpan file VHD di dalamnya. Ini memiliki fitur gambar tetap dari disk penjaga yang memungkinkan pengguna untuk mengakses file yang dialokasikan ukuran disk virtual tertentu. File dapat memodifikasi hard disk mesin virtual.

File VHD ini dapat ditukar dengan mudah dari hard disk virtual ke sistem file host. Format ini juga memiliki kemampuan backup dan restore serta keamanan antivirus. File VHD dapat mendukung format tertentu seperti gambar hard disk Tetap, gambar hard disk Diferensiasi, dan gambar hard disk Dinamis.
