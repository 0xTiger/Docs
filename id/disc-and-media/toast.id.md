## Apa itu file roti panggang?
Perangkat lunak Roxio Toast membuat gambar CD atau DVD dan menyimpan file dengan ekstensi .toast. Roxio Toast adalah aplikasi perangkat lunak pembuat cakram optik dan konversi media terkenal untuk Macintosh yang pertama kali diperkenalkan oleh Miles Software GmbH, kemudian dibeli oleh Adaptec. File Toast mungkin berisi segala jenis disk seperti video, audio atau disk data juga Mungkin salinan dan gambar disk lainnya.

## Format file roti panggang
Format file Toast digunakan oleh perangkat lunak Roxio Toast untuk menyimpan hasilnya dalam file dengan ekstensi .toast. Meskipun, berbagai jenis cakram dapat dibakar langsung melalui sistem Mac OS, tetapi Toast memberikan kontrol tambahan atas proses tersebut, dan juga menawarkan beberapa fitur tambahan, seperti membuat katalog, melacak berkas yang dibakar ke cakram, dan pemulihan berkas untuk cakram yang rusak. juga menyediakan dukungan untuk format audio dan video yang tidak didukung oleh Quicktime, seperti [FLAC](/id/audio/flac/) dan [OGG](/id/audio/ogg/). Karena ini adalah format file berpemilik, informasi tentang seluruh struktur format file tidak tersedia.
### Jenis disk
Oleh karena itu, Roxio Toast dapat membuat file Toast, yang berisi data gambar disk dari jenis berikut:
- **Membuat Disk Video**: Toast dapat membuat file Toast yang berisi disk video dengan menggunakan proyek berikut:
- DVD-Video
- Video Blu-ray
- DVD Definisi Tinggi
- Folder VIDEO_TS
- Kompilasi VIDEO_TS
- Folder BDMV
- Arsip AVCHD
- **Membuat Disk Data**: Toast dapat membuat file Toast yang berisi disk data dengan menggunakan proyek berikut:
- Hanya Mac
- Mac & PC
- DVD ROM
- Hibrida Kustom
- Volume Mac
- Disk Foto
- **Membuat Disk Video**: Toast dapat membuat file Toast yang berisi disk data dengan menggunakan proyek berikut:
- CD audio
-DVD musik
- Disk MP3
- CD Audio yang Disempurnakan
- CD Mode Campuran
- **Menyalin Disk**: Toast mendukung penyalinan dengan cara berikut:
- Salinan Disk
- Salinan Gambar
- Penggabungan Gambar Disk
