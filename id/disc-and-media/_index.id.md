File gambar disk adalah file komputer yang berisi konten dan struktur volume disk. Alternatifnya, Anda dapat mengatakan bahwa file image disk berisi seluruh perangkat penyimpanan data, seperti USB flash drive, floppy disk, tape drive, optical disc, atau hard disk drive. Biasanya, gambar disk dibuat dengan membuat salinan sektor demi sektor dari media sumber, sehingga secara akurat menyalin struktur dan konten perangkat penyimpanan yang tidak bergantung pada sistem file. Gambar disk dapat menjangkau satu atau lebih file komputer.

Beberapa format file disk yang populer adalah [Toast](/id/disc-and-media/toast/), [BIN](/id/disc-and-media/bin/) dan [VCD](/id/disc-and-media/vcd/)
