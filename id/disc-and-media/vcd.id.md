## Apa itu file VCD?
File VCD adalah image disk dari DVD atau CD. File VCD dapat dikenali oleh perangkat lunak CD ROM virtual biasa dan sistem operasi memahami file ini sebagai CD atau DVD yang sebenarnya. Sedangkan perangkat lunak mengambil data dari hard disk. File VCD adalah salinan dari CD dan DVD aktual untuk digunakan tanpa memerlukan drive fisik dan bermanfaat untuk memasang disk pada komputer jaringan.

## format file VCD
Format file VCD dikategorikan sebagai format file image disk yang membuat image disk dengan ekstensi .vcd; digunakan untuk mengakses konten CD atau DVD tanpa memerlukan CD-ROM fisik atau drive DVD-ROM. File VCD terintegrasi dengan kompresi gambar disk dan spesifikasi format yang digunakan untuk pengarsipan berbagai jenis file dan bahkan seluruh konten disk optik dan hard drive ke dalam satu file CD virtual; dapat dipasang dengan menggunakan perangkat lunak drive virtual untuk mengakses data yang disimpan dalam file VCD. Oleh karena itu, format file VCD membuat gambar dari media optik Anda sendiri, sehingga Anda dapat menggunakannya kapan saja tanpa harus memiliki media fisik. Ketersediaan data Anda akan maksimal, sementara hilang, rusak dan dicuri bukan urusan file VCD. Seluruh struktur atau spesifikasi format file ini tidak tersedia untuk umum.
