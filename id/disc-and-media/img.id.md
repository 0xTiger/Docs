## Apa itu file IMG?

Di bidang komputasi, file dengan ekstensi file .img berurusan dengan penyimpanan gambar disk mentah dari disk seperti floppy disk, hard drive atau cakram optik, dll. Gambar mentah ini terdiri dari gambar biner sumber dengan spesifikasi sektor demi sektor.

Isi format file IMG bergantung pada sistem file disk yang ditentukan untuk pembuatan image. Ketika datang ke CD ROM dan DVD, gambar dari file-file ini berisi data dari sektor yang sesuai bersama dengan header kontrol dan bidang untuk koreksi kesalahan menurut masing-masing sektor.

File IMG telah menentukan konten untuk disk, jadi ini kompatibel dengan program khusus yang cocok untuk mendeteksi sistem file mereka.

## Format File IMG ##

Selama bertahun-tahun ketika floppy disk digunakan untuk tujuan penyimpanan dan pengelolaan penyimpanan data, file-file ini dikembangkan untuk mengakses gambar mentah dari disk. Banyak program dirujuk oleh file IMG. Ini dibuat oleh program pencitraan disk. Dalam program sistem operasi Windows seperti daya, pembakar perangkat lunak digunakan untuk menangani file-file ini.

## Spesifikasi teknis ##

Ada ekstensi file lain seperti IMA dan IMZ yang mirip dengan format file ini. Ukuran gambar-gambar ini selalu dalam bentuk beberapa ukuran sektor yang berbeda. Ukuran ini adalah 512 byte untuk floppy disk dan hard drive. Ukuran mentah sektor ini adalah 2.352 untuk CD Rom dan DVD. Ukuran gambar disk mentah adalah kelipatan dari ukuran ini.

Format ini banyak digunakan dengan program seperti RawWrite dan WinImage. Program ini menggunakan format ini untuk membaca dan menulis image floppy disk. Selain itu, file-file ini dapat dikonversi ke format file lain yang lebih populer dengan menggunakan beberapa perangkat lunak seperti PowerISO.

Karena file IMG tidak memiliki data tambahan kecuali isi disk, file IMG ini hanya dapat ditangani oleh program secara otomatis dan mendeteksi sistem file. Misalnya, image disk biasa dari floppy disk mentah dimulai dengan sektor boot FAT dan dapat digunakan untuk mengidentifikasi sistem file.

Gambar cakram media optik sebagian besar disertai dengan file deskriptor. Ini menjelaskan tata letak disk dan melibatkan informasi seperti batas trek dan ini tidak hanya disimpan dalam file gambar dalam bentuk mentah.
