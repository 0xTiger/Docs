## Apa itu file BIN?

File dengan ekstensi .bin dapat berupa file gambar disk, biasanya berisi data biner yang disimpan dari disk optik seperti CD atau DVD. File BIN adalah file tunggal; berisi isi dan struktur yang lengkap; mewakili media atau perangkat penyimpanan data, seperti tape drive, hard drive, floppy disk, CD, DVD, BD, dan key drive. File BIN dapat disimpan dengan file metadata lainnya; menjelaskan informasi terkait disk.

## format file BIN

Meskipun citra cakram optik yang disimpan dengan ekstensi .bin dapat dirujuk sebagai citra cakram, biasanya dibuat dengan menyalin salinan sektor per sektor media sumber sepenuhnya dan dengan demikian mereplikasi struktur dan konten perangkat penyimpanan dengan sempurna. Struktur sebenarnya dari data di dalam gambar biasanya berupa beberapa sistem file.
Beberapa perangkat lunak pencitraan disk mengurangi ruang file yang tidak terpakai dari media sumber, atau memampatkan disk yang diwakilinya untuk mengurangi kebutuhan penyimpanan. Meskipun ini secara khusus disebut sebagai file arsip, mereka tidak dianggap sebagai gambar disk.

## BIN VS ISO

Format BIN biasanya dikembangkan untuk mengatasi masalah pengarsipan CD Audio dengan [ISO](/id/compression/iso/). Biasanya format ini dapat digunakan sebagai alternatif untuk konten yang sama yang didukung ISO dan tanpa kekurangan apapun. ISO adalah format gambar disk sederhana, sedangkan BIN/CUE adalah salinan mentah disk, termasuk perlindungan penyalinan, sektor demi sektor, daftar lagu, koreksi kesalahan, multi-track, dan informasi khusus sistem apa pun pada disk
