## Apa itu file MDS?

MDS (Media Descriptor Sidecar File) adalah file yang terkait dengan perangkat lunak Alcohol 120% dan Daemon Tools, keduanya digunakan untuk membuat dan mengelola drive CD dan DVD virtual. File MDS berisi informasi tentang tata letak data pada disk, termasuk lokasi semua file dan struktur disk. Hal ini memungkinkan drive virtual untuk meniru perilaku disk fisik, sehingga perangkat lunak dapat membaca data pada disk virtual seolah-olah itu adalah disk sebenarnya.

When you want to mount an image file to a virtual drive, the MDS file is used along with the corresponding image file in a format such as ISO, BIN or CUE. The MDS file contains a description of the layout of the data on the disc, and the virtual drive software uses this information to create a virtual disc. This allows you to use the software as if you were using a physical disc, without having to physically insert the disc into the drive.

## Hubungan dengan Alkohol 120%

Alcohol 120% adalah perangkat lunak populer untuk membuat dan mengelola drive CD dan DVD virtual, dan menggunakan format file MDS untuk menyimpan dan mengakses image disk. Format file MDS adalah hak milik Alcohol 120% dan hanya dapat dibuka dan digunakan oleh software Alcohol 120% atau software lain yang mendukungnya.

## Bagaimana cara membuka file MDS?

Untuk membuka file MDS dalam Alkohol 120%, Anda perlu menginstal perangkat lunak di komputer Anda. Setelah Anda menginstal Alkohol 120%, Anda dapat membuka file MDS dengan melakukan hal berikut:

1. Luncurkan Alkohol 120%.
2. Klik pada menu File dan pilih Buka.
3. Arahkan ke lokasi file MDS di komputer Anda dan pilih.
4. File MDS sekarang akan dibuka dalam Alkohol 120%, dan Anda akan diminta untuk memilih file gambar yang sesuai (seperti ISO, BIN atau CUE).
5. Setelah Anda memilih file gambar, Alcohol 120% akan memasang gambar tersebut ke drive virtual.

Alternatifnya, Anda juga dapat membuka file MDS dengan mengklik dua kali file tersebut, jika Alkohol 120% disetel sebagai program default untuk membuka file MDS. Anda juga dapat menggunakan software lain yang mendukung format MDS seperti Daemon Tools, Virtual CloneDrive, Virtual Drive Manager dan MagicISO.
