## Apa itu file ROM?
File dengan ekstensi .rom sebenarnya adalah file data; salinan tepat dari chip ROM memori hanya-baca untuk suatu perangkat, yang diambil dari kartrid video game atau komputer. Tujuan pembuatan file ROM bisa bermacam-macam, seperti emulasi perangkat lunak, mencadangkan atau mengarsipkan data ROM, atau memperbarui firmware pada perangkat tertentu. Banyak perangkat tersedia untuk membuang ROM video game ke dalam file ROM, namun, diperlukan alat dan keahlian yang lebih khusus untuk membuang ROM dari perangkat lain seperti game arcade.

## format file ROM
Format file ROM dirancang untuk berisi data salinan persis dari kartrid video game, game arcade, firmware komputer. Istilah ROM biasanya digunakan ketika kita membahas tentang emulasi, Juga game atau firmware lama disalin ke file ROM di komputer modern dan ini dapat digunakan sebagai perangkat lunak yang dikenal sebagai emulator. File ROM biasanya menarik semua data dari chip memori read-only yang ditemukan di game berbasis kartrid atau mesin arcade menggunakan perangkat khusus dalam proses yang dikenal sebagai dumping.
### Perlindungan salinan
File ROM juga memfasilitasi penyalinan dan distribusi ulang game modern yang tidak sah. Banyak perusahaan pengembang game seperti Nintendo, Capcom, SNK dll menggunakan metode enkripsi mereka untuk melindungi barang-barang mereka untuk disalin tetapi perusahaan gagal melindungi game mereka untuk waktu yang lama. Peretas akhirnya berhasil mendekripsi metode enkripsi kompleks yang menyebabkan penurunan penjualan game yang sangat besar karena game tersebut tersedia secara bebas untuk dimainkan dengan menggunakan perangkat lunak emulasi.
### Penggunaan
- **Emulasi**: Emulator konsol video game biasanya mengambil gambar ROM sebagai file input.
- **ROM Perangkat Lunak**: Perangkat lunak yang sedang dikembangkan untuk komputer tertanam sering ditulis ke file ROM untuk pengujian pada komputer standar.
- **Pelestarian digital**: Ada kekhawatiran yang signifikan bahwa banyak game komputer dan video awal mungkin tidak dapat bertahan tanpa dipindahkan ke media baru. Jadi, mereka yang tertarik dengan pelestarian secara aktif mencari game arcade dan video lama dan mencoba membuangnya ke gambar ROM
- **Peretasan dan terjemahan penggemar**: Peretasan dapat berkisar dari tweak sederhana seperti perbaikan grafis dan cheat, hingga desain ulang game secara menyeluruh, yang pada dasarnya menciptakan game yang sama sekali baru menggunakan yang asli sebagai basis.
