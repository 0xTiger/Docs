## Apa itu file PFA?

File dengan ekstensi .pfa adalah versi Printer Font ASCII dari font Adobe Type 1 dan berisi data glyph font. File dalam format yang dapat dibaca manusia dan dapat dibuka di editor teks apa pun. Penerjemah PostScript dapat menggunakan jenis font ini secara keseluruhan untuk kerjanya. Printer Font Binary (PFB) versi font Adobe Type 1 berisi informasi yang sama. Namun, batalkan baris PFA, ini dalam format file biner dan tidak dapat dibaca manusia. File PFA dapat dimasukkan ke dalam sistem dengan menyalin ke folder Font sistem.
