## Apa itu file JFPROJ?

File dengan ekstensi .jfproj adalah font khusus berbasis JSON yang dibuat dengan aplikasi Microsoft Font Maker di Windows 10. Ini dapat dibuat dengan menggunakan pena elektronik dengan perangkat layar sentuh seperti tablet meskipun periferal komputer lain seperti mouse juga dapat digunakan untuk berjaga-jaga. komputer. File JFPROJ berisi mesin terbang yang digunakan untuk merender karakter ke layar dengan menentukan jarak antara karakter dan kata, dan ukuran font. Aplikasi Microsoft Font Maker tersedia untuk diunduh di toko Windows.
