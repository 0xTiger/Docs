## Apa itu file PFM?

File dengan ekstensi .pfm adalah file Metrik Font Printer yang berisi pengukuran karakter yang disertakan dalam file font. Itu harus memiliki file PFM di samping file .pfb(/id/font/pfb/) terkait di folder yang sama saat menginstal nanti. File PFM dikodekan biner dan tidak dapat dibaca manusia.
