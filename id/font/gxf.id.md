## Apa itu file GXF?

File GXF adalah file font oleh alat desain CAD untuk Windows, General CADD Pro. Perangkat lunak ini menggunakan font untuk memasukkan teks dan simbol lain ke dalam gambar CAD. Font GXF harus tersedia untuk perangkat lunak General CADD Pro agar gambar CAD dapat ditampilkan dengan benar.

Anda dapat membuka file GXF menggunakan [General CADD Pro](https://www.generalcadd.com/oldSite/101release.htm).

## Tentang Umum CADD Pro

General CADD Pro menawarkan struktur perintah dua huruf yang digabungkan dan digunakan oleh banyak profesional perancang. Ini mendukung format file ACAD 2013. Ini juga menyediakan dukungan untuk perintah arah/jarak surveyor dan perintah Field to Finish.
