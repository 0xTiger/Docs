## Apa itu file VLW?

File VLW adalah format font yang digunakan oleh perangkat lunak pembuat gambar dan animasi, **Pemrosesan**. Ini menyimpan font vektor dengan cara yang ringkas yang dapat diskalakan ke ukuran apa pun tanpa kehilangan kualitas. Ini memanfaatkannya dalam desain grafis dan visual. Pemrosesan memberikan kemampuan untuk membuat gambar dan animasi, di mana mesin terbang font VLW ini digunakan.

## Format File VLW - Informasi Lebih Lanjut

File VLW disimpan ke disk sebagai file biner. Dalam Pemrosesan, Anda dapat menggunakan fungsi **createFont()** untuk membuat objek font dari file VLW. Anda kemudian dapat menggunakan objek font ini untuk menggambar teks dalam sketsa Anda menggunakan fungsi **text()**. Pemrosesan dapat membaca file VLW menggunakan perintah **loadFont()**.
