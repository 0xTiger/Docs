## Apa itu Font Tipe 1?

Font tipe 1 adalah teknologi Adobe yang sudah tidak digunakan lagi yang banyak digunakan dalam perangkat lunak penerbitan berbasis desktop dan printer yang dapat menggunakan PostScript. Meskipun font Tipe 1 tidak didukung di banyak platform modern, browser web, dan sistem operasi seluler, tetapi ini masih didukung di beberapa sistem operasi. Ketidakcukupan dukungan informasi Unicode dalam font Tipe 1 juga membatasi kemampuannya untuk mendukung rangkaian karakter bahasa yang diperluas.

## Sejarah Singkat

Teknologi font Tipe 1 diluncurkan pada tahun 1984 untuk digunakan dengan bahasa deskripsi halaman PostScript dari Adobe. Pada pertengahan 90-an, Adobe memutuskan untuk berkonsentrasi pada penggunaan font OpenType yang lebih fleksibel daripada Tipe 1.

### Kapan dukungan Tipe 1 akan berakhir?
Pengguna tidak akan dapat lagi menulis konten menggunakan font Tipe 1 mulai Januari 2023. Hingga saat itu, tidak ada perubahan yang akan dilakukan.
Beberapa produk seperti, aplikasi Document Cloud, akan terus menggunakan font Tipe 1 seperti yang selama ini mereka lakukan.


## Keterbatasan Font Tipe 1

Beberapa batasan Tipe 1 yang diketahui adalah sebagai berikut:

- Itu tidak memungkinkan lebih dari 256 mesin terbang untuk dimasukkan dalam satu font. Font CID dapat melakukan ini, banyak perangkat keluaran mungkin tidak menangani font CID dengan benar.
- Font tipe 1 bukan lintas platform. Diperlukan konversi yang kompleks saat mengonversi satu font platform ke font lainnya.
- Menempelkan nama font ke batas 8 karakter di masa DOS lama, yang menyulitkan untuk menentukan jenis huruf mana yang disimpan dalam file.

## Ketik 1 Font pada mesin Windows
Data font Tipe 1 terdiri dari dua file terpisah di windows:

1. File dengan ekstensi “.PFB” terdiri dari data outline.
2. File dengan ekstensi “.PFM” berisi data metrik .
