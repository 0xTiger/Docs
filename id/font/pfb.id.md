## Apa itu file PFB?

File dengan ekstensi .pfb adalah versi Printer Font Binary dari font Adobe Type 1. Ini mirip dengan file .pfa(/id/font/pfa/) tetapi bersifat biner. File PFB menyimpan font Adobe Type 1 yang digunakan untuk sistem operasi windows. Ini berisi data font mesin terbang dan menyimpan data dalam bentuk kompak dalam format biner. Selain memiliki konten yang disandikan biner, file PFB juga dienkripsi. File PFB juga membutuhkan file .pfm di samping yang menyertakan ukuran karakter dalam font. Untuk menginstal font PFB, file .pfb dan .pfm harus ditempatkan di sistem.
