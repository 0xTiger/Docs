## Apa itu file MFX?

File MFX adalah file font yang digunakan oleh game simulasi Maxis SimCity 4. Ini digunakan untuk merender teks game di layar dari salah satu jenis font yang dipilih termasuk Arial, Courier, atau font lain yang tersedia di file font. File MFX terinstal dengan game SimCity 4 dan tidak perlu dibuat secara manual.

SimCity 4 adalah game komputer simulasi untuk membuat simulasi pembangunan kota. Maxis, pengembang SimCity, adalah anak perusahaan Electronic Arts. Permainan ini awalnya diluncurkan pada Januari 2003.
