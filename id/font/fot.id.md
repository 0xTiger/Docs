## Apa itu file FOT?

File FOT adalah file font yang digunakan untuk file data kerangka font. Ini berisi informasi tentang keseluruhan bentuk dan garis besar karakter dalam font. Informasi lain yang terkandung dalam file .fot mencakup nama font, gaya dan ukurannya. File FOT digunakan oleh aplikasi yang mendukung TrueType (file [.TTF ](/font/ttf/)). Ini lebih sering digunakan oleh penginstal aplikasi selama instalasi font.

## Lebih lanjut tentang Format File FOT

File FOT disimpan ke disk sebagai file biner dan struktur file internalnya tidak diketahui untuk referensi pengembang.
