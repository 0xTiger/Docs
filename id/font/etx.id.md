## Apa itu file ETX?

File ETX adalah file font yang digunakan oleh sistem penyusunan huruf, TeX dan LaTeX. Itu disimpan sebagai file teks dan menentukan tampilan font berdasarkan pengkodeannya. File font ETX dimuat dalam perangkat lunak yang menyediakan kemampuan untuk bekerja dengan dokumen Tex. Setelah dimuat, file font ETX dapat digunakan untuk penyesuaian dokumen TeX.
