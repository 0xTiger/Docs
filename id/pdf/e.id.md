# Apa itu PDF/E? #

"E" dalam PDF/E adalah singkatan dari Engineering. PDF/E diterbitkan sebagai ISO 24517 pada tahun 2008 sebagai standar untuk membuat dokumen Teknik berbasis PDF untuk digunakan di berbagai area aplikasi. Area utama yang menggunakan format file PDF/E meliputi alur kerja geospasial, konstruksi, dan manufaktur. Standar PDF/E menyediakan mekanisme untuk pertukaran dan pengarsipan dokumen teknik berdasarkan format PDF. PDF/E hadir dengan dukungan media interaktif, termasuk animasi dan data model rekayasa 3D.

## Sejarah Singkat ##

* Maret 2004 - Pengumuman Pertama
* 2005 - Pengajuan ke ISO
* 2006 - Voting untuk format PDF/E
* 2007 - Persetujuan untuk format PDF/E
* 2008 - Ketersediaan resmi dan pembaruan standar

## Karakteristik PDF/E ##

PDF/E telah digunakan untuk pembuatan dan pertukaran dokumen teknik dan teknis. Karakteristik utamanya untuk mendukung data model rekayasa 3D membuatnya menonjol karena file yang dihasilkan dalam format PDF/E tidak memerlukan aplikasi perangkat lunak khusus untuk melihatnya. Sebaliknya, ini dapat dilihat menggunakan perangkat lunak Adobe Reader yang gratis dan banyak tersedia. PDF/E didasarkan pada PDF v1.6 tetapi memiliki kemampuan tertanam yang terbatas yang tidak digunakan saat membuat dokumen teknik.

Standar PDF/E memiliki manfaat sebagai berikut;

* Mengurangi persyaratan untuk perangkat lunak mahal & berpemilik
* Biaya penyimpanan dan penukaran yang lebih rendah (vs. kertas)
* Pertukaran yang dapat dipercaya di berbagai aplikasi dan platform
* Mandiri
* Cara yang hemat biaya dan akurat untuk menangkap markup
* Dikembangkan dan dikelola oleh komite PDF/E ISO
