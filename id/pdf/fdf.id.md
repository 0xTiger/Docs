## Apa itu file FDF?

File FDF (**Forms Data Format**) adalah dokumen teks yang dibuat dengan mengekspor data dari bidang formulir file [PDF](/id/pdf/). Ini hanya mencakup data bidang teks yang diekstraksi dari bidang formulir yang tersedia dalam file PDF. Ini menghasilkan file data yang relatif kecil karena data yang diekspor tidak berisi formulir itu sendiri. Adobe Acrobat menyediakan fitur untuk mengekspor data form field dengan menggunakan opsi `Export Forms Data` dari menu aplikasi.

## Format File FDF - Informasi Lebih Lanjut

FDF adalah format teks biasa dan disertakan sebagai bagian dari [standar ISO 32000](https://www.iso.org/standard/51502.html) untuk Format Dokumen Portabel. Ini dikembangkan oleh Adobe untuk memungkinkan impor dan ekspor data dari Acrobat Forms, atau AcroForms.

Ada dua jenis file FDF:

• `Classic FDF` – Menyediakan data untuk mengisi formulir statis yang ada.

• `Template FDF` – Membuat PDF baru berdasarkan template dari dalam file PDF yang ditentukan. Formulir di dalam dokumen baru diisi dengan memberikan data.

## Membuat FDF menggunakan Adobe Acrobat

[Toolkit FDF](https://opensource.adobe.com/dc-acrobat-sdk-docs/) dari Adobe memungkinkan Anda membuat file FDF dari data teks.
