File yang dapat dieksekusi digunakan untuk menginstruksikan komputer untuk melakukan satu atau beberapa tugas atau operasi tertentu sebagai rangkaian instruksi yang disandikan; ditulis dalam file-file itu. Istilah "Instruksi" biasanya berarti instruksi **kode mesin** untuk **CPU** fisik. File yang dapat dieksekusi dapat dikodekan dengan tangan dalam bahasa mesin, meskipun sangat mudah untuk mengembangkan perangkat lunak sebagai kode sumber dalam bahasa tingkat tinggi yang mudah dipahami oleh manusia. Kadang-kadang, kode sumber mungkin ditulis dalam bahasa rakitan, tetap terkait erat dengan instruksi kode mesin yang mudah dibaca manusia.

Beberapa format file disk yang populer adalah [BAT](/id/executable/bat/), [CGI](/id/executable/cgi) dan [COM](/id/executable/com/)
