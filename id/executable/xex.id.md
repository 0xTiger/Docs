## Apa itu file XEX?

File XEX adalah format file yang dapat dieksekusi yang digunakan untuk aplikasi konsol video game XBox 360. Xbox, pendahulu Xbox 360, menggunakan format file **XBE** untuk aplikasi. File XEX dimaksudkan untuk dijalankan di Xbox 360 saja dan tidak dijalankan di PC. File XEX default di konsol XBox 360 dinamai **default.xex** yang bertanggung jawab untuk mengontrol dan mengunduh pembaruan perangkat lunak untuk perangkat. Ini memastikan kompatibilitas mundur dengan konsol XBox.

## Format File XEX

File XEX disimpan ke disk sebagai file biner dan dapat dijalankan di konsol XBox 360. Ini juga dapat dibuka dan diperiksa menggunakan perangkat lunak XEXTool yang dapat mengekstrak informasi dari file ini. XEXTool juga dapat digunakan untuk memodifikasi file-file ini.
