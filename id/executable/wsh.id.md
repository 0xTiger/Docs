## Apa itu file WSH?
File dengan ekstensi .wsh berisi properti dan parameter untuk skrip bahasa pemrograman tertentu seperti VB atau VBS dll. Kebutuhan sebenarnya dari WSH adalah menggunakannya untuk menyesuaikan eksekusi skrip tertentu. WScript atau CScript diperlukan untuk dijalankan dan keduanya disertakan dengan sistem operasi Windows. File WSH awalnya disediakan pada Windows 95 pada disk instalasi sebagai opsional yang dapat dikonfigurasi dan diinstal untuk Control Panel, dan kemudian menjadi komponen default Windows 98.

## format file WSH
WSH (Windows Script Host) dapat digunakan untuk berbagai keperluan, termasuk administrasi, skrip logon, dan otomatisasi umum. WSH menetapkan lingkungan untuk menjalankan skrip. itu memanggil mesin skrip yang sesuai dan mengalokasikan satu set layanan dan objek untuk skrip bekerja dengannya. Skrip ini dapat dijalankan dalam mode GUI, dari objek COM atau mode baris perintah, memberikan fleksibilitas kepada pengguna untuk skrip interaktif atau non-interaktif.

### Contoh
Berikut adalah contoh yang sangat sederhana yang menunjukkan beberapa VBScript yang menggunakan objek root WSH COM "WScript" untuk menampilkan pesan dengan tombol 'OK'. Saat skrip ini akan diluncurkan, mesin CScript atau WScript akan dipanggil dan lingkungan runtime akan dibuat.

```
WScript.Echo "Hello world"
WScript.Quit
```
