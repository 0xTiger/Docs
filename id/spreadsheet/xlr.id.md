## Apa itu file XLR?

File dengan ekstensi .xlr adalah format file Microsoft Works Spreadsheet yang menyimpan data dalam baris dan kolom. Microsoft Works dikembangkan untuk digunakan sebagai rangkaian produktivitas awal dan tetap berfungsi dari tahun 1987 hingga 2009. Ini termasuk pengolah kata, spreadsheet, dan sistem manajemen basis data. Itu kemudian dihentikan setelah Microsoft Office diperkenalkan dengan fitur yang lebih kaya disertakan.

## Format File XLR - Informasi Lebih Lanjut

File XLR disimpan sebagai file biner tanpa detail tentang spesifikasi format file yang tersedia untuk umum. Oleh karena itu, pengembang tidak memiliki opsi untuk menganalisis struktur file dan menulis aplikasi untuk membaca/menulis ke file ini secara terprogram. Spreadsheet Works menggunakan format file Excel yang sama untuk menyimpan data ke file .xlr.
