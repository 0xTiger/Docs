## Apa itu file XLA?

File dengan ekstensi .xla adalah format file Microsoft Excel Add-Ins yang menambahkan modul, fungsi ekstra, dan alat bantu lainnya ke Microsoft Excel. Ad-Ins memberikan fungsionalitas tambahan ke spreadsheet Excel dan didukung dalam versi 2007 atau lebih baru. File XLA memiliki mime media type application/vnd.ms-excel. Ini dapat dibuka menggunakan Microsoft Excel yang tersedia untuk Windows dan juga MacOS.

## Berformat XLA

File XLA didasarkan pada format data biner. Namun, informasi lebih lanjut tentang spesifikasi format filenya tidak tersedia. Asumsi umum adalah bahwa XLA mengikuti [[MS-XLS] - Struktur Format File Biner Excel](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx).
