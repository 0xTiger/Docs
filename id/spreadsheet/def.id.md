## Apa itu file DEF?

File DEF adalah file data yang dibuat dalam [format file spreadsheet](/id/spreadsheet/) oleh aplikasi database SmartWare II, yang dikembangkan oleh Informix Software. Software II memiliki empat modul termasuk spreadsheet dengan grafis, pengolah kata, komunikasi, dan database. Database SmartWare II adalah database mandiri yang dapat diintegrasikan dengan aplikasi lain juga. Kueri dapat dijalankan pada data yang dapat diekstraksi dan disimpan ke format file DEF. File spreadsheet DEF dapat dibuka dengan perangkat lunak SmartWare yang sekarang telah dihentikan.
