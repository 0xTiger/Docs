## Apa itu file XLT?

File dengan ekstensi .xlt adalah file template yang dibuat dengan Microsoft Excel yang merupakan aplikasi spreadsheet yang hadir sebagai bagian dari rangkaian Microsoft Office. Microsoft Office 97-2003 mendukung pembuatan file XLT baru serta membukanya. Excel versi terbaru masih mampu membuka file template format lama ini. File template semacam itu digunakan untuk membuat file Excel baru dengan cepat dengan data dan pengaturan default seperti pemformatan halaman, ukuran font, margin, bagan, dll. Yang selanjutnya dapat disimpan sebagai file [.xls](/id/spreadsheet/xls/) baru.

## Berformat XLT

Format file XLT didasarkan pada format file biner Excel yang awalnya didukung dengan versi Excel 97-2003. Itu diganti dengan format file [XLTX](/id/spreadsheet/xltx/) dengan pengenalan Excel 2007 yang didasarkan pada format Office Open XML. Selain Microsoft Excel, ada aplikasi lain yang mendukung bekerja dengan format file ini seperti Lotus Translation Table, PlanMaker, LibreOffice Calc, dll. Tersedia beberapa aplikasi dan [API](https://products.aspose.com/cells) untuk mengonversi XLT ke format file lain seperti:

* XLT ke PDF
* XLT ke DOC
* XLT ke Teks
* XLT ke Teratai
* XLT ke ODT
* XLT ke ODS
* XLT ke HTML
* XLT ke CSV

XLT didasarkan pada biner [spesifikasi format file XLS](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) yang didokumentasikan oleh Microsoft untuk referensi pengembang.
