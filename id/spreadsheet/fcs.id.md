## Apa itu file FCS?

File FCS adalah file spreadsheet yang dibuat dengan perangkat lunak PFS: First Choice yang merupakan office suite yang terdiri dari pengolah kata, spreadsheet, grafik, database, dan telekomunikasi. FCS singkatan dari First Choice Spreadsheet dan dapat dibuka dengan PFS: First Choice suite untuk DOS oleh [Software Publishing Corporation](https://en.wikipedia.org/wiki/Software_Publishing_Corporation).
