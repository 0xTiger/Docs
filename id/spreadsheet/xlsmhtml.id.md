## Apa itu file XLSMHTML?

File dengan ekstensi .xlsmhtml adalah versi [HTML](/id/web/html/) dari file XLS asli yang diekspor dari dalam Microsoft Excel untuk melihat data di browser. Format tersebut telah lama dihentikan oleh Excel dan tidak lagi didukung. File XLSMHTML yang diekspor mengikuti format file XLS dan ini dapat dibuka dan dilihat di browser web. Excel menyediakan ekspor data spreadsheet ke format file [MHT](/id/web/mht/) dan [MHTML](/id/web/mhtml/), dan sekarang banyak digunakan.

## Format File XLSMHTML - Informasi Lebih Lanjut

File XLSMHTML dibuat dengan kode HTML untuk merepresentasikan data spreadsheet dalam format yang mirip dengan XLS. Kode HTML di dalam file XLSMHTML diatur sedemikian rupa sehingga file output terlihat mirip dengan file aslinya saat dibuka di web browser. Spesifikasi format file dari file XLSMHTML tidak dipublikasikan oleh Excel dan tidak ada referensi yang tersedia bagi pengembang untuk memahami dan mengembangkan aplikasi untuknya.
