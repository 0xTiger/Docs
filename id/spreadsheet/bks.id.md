## Apa itu berkas BKS? ##

File BKS adalah file cadangan [Microsoft Works](https://en.wikipedia.org/wiki/Microsoft_Works) yang menyimpan status kerja terakhir dari file spreadsheet. Ini menyimpan status valid terakhir dari spreadsheet untuk dimuat ulang jika dokumen asli rusak atau tidak dapat dibuka karena alasan apa pun. BKS mirip dengan format file [WPS](/id/word-processing/wps/) yang berisi konten dokumen yang mirip dengan dokumen Microsoft Word [DOC](/id/word-processing/doc/). Microsoft Works telah dihentikan dan file BKS tidak lagi didukung.

## Format File BKS

File BKS disimpan ke disk sebagai file biner dan spesifikasi format filenya tidak tersedia untuk umum.
