## Apa itu file DEX?

File DEX adalah file spreadsheet yang dibuat dengan Microsoft Excel. Mirip dengan file [XLSX](/id/spreadsheet/xlsx/) tetapi dalam format file biner. Data disimpan dalam baris dan kolom dalam file DEX. Meskipun file DEX juga merupakan Microsoft Excel, ini kurang populer. Sebagai perbandingan, format file XLSX adalah pilihan default untuk menyimpan buku kerja Excel. File DEX telah ditandai sebagai usang dan tidak lagi didukung oleh Microsoft Excel.
