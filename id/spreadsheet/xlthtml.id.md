## Apa itu file XLTHTML?

File dengan ekstensi .xlthtml adalah versi [HTML](/id/web/html/) dari file template [XLT](/id/spreadsheet/xlt/) dan dibuat menggunakan Microsoft Excel pada awalnya. Format ini tidak lagi didukung oleh Excel dan telah digantikan dengan mengekspor data ke format file [MHT](/id/web/mht/) dan [MHTML](/id/web/mhtml/). Ide dasar di balik file XLTHTML adalah untuk memungkinkan pengguna melihat file template XLT di browser web alih-alih perlu menginstal Excel di sistem. File yang diekspor dalam format file XLTHTML dapat dibuka di browser.

## Format Berkas XLTHTML - Informasi Lebih Lanjut

File XLTHTML menggunakan format file HTML untuk menyimpan template XLT untuk dibuka di browser web. XLT sendiri didasarkan pada format file biner Excel yang diperkenalkan dengan versi Excel 97-2003. File-file ini dapat digunakan sebagai file awal dengan pemformatan yang ditentukan untuk membuat dokumen normal untuk populasi data spreadsheet. Sekadar referensi, XLT didasarkan pada biner [spesifikasi format file XLS](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) yang didokumentasikan oleh Microsoft untuk referensi pengembang.
