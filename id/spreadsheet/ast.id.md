## Apa itu file AST?

File AST adalah file template spreadsheet yang dibuat dengan perangkat lunak Ability Spreadsheet, komponen spreadsheet dari paket Ability Office. File template ini digunakan untuk membuat kumpulan awal template spreadsheet yang dapat digunakan untuk membuat file [AWS](/id/spreadsheet/aws/). File AST dapat dibuka dengan perangkat lunak spreadsheet Ability.

## Format File AST - Informasi Lebih Lanjut

File AST disimpan ke disk sebagai file biner dan dapat dibuka di perangkat lunak Ability Spreadsheet. Meskipun sepenuhnya kompatibel dengan format file Microsoft XLS dan XLSX, spesifikasi format file AST tidak tersedia secara publik untuk referensi pengembang. File AST dapat disimpan ke disk dalam format file [PDF](/id/pdf/) serta [XPS](/id/page-description-language/xps/).
