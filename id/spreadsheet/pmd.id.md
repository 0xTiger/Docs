## Apa itu file PMD?

File PMD adalah file spreadsheet yang dibuat dengan software [PlanMaker](https://www.softmaker.de/softmaker-office-planmaker) yang disertakan dengan SoftMaker Office. Ini menyimpan data spreadsheet dalam baris dan kolom yang mirip dengan file Excel. [XLSX](/id/spreadsheet/xlsx/) adalah format file standar PlanMaker yang memudahkan penggunaan perangkat lunak, bukan Microsoft Excel. File PMD dapat berisi rumus dan bagan selain data yang disimpan dalam sel. File yang dibuat dengan PlanMaker dapat dibuka dengan Microsoft Excel tanpa konversi tambahan. Itu juga dapat mengedit file [XLS](/id/spreadsheet/xls/) dan mendukung format file tambahan seperti [PDF](/id/pdf/).

## Format File PMD

File PMD disimpan ke disk sebagai file biner dalam format file berpemilik dan spesifikasinya tidak tersedia untuk umum.
