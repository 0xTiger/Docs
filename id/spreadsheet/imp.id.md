## Apa itu file IMP? ##

File IMP adalah file spreadsheet yang dibuat dengan software [Lotus Improv](https://en.wikipedia.org/wiki/Lotus_Improv) untuk menyimpan data dalam baris dan kolom. Konten dalam file IMP dipisahkan dalam tiga segmen berbeda; yaitu data, tampilan data, dan formula. Meskipun tujuan dari pendekatan ini adalah untuk mempermudah pengeditan, hal ini mempersulit transfer data dari file IMP ke jenis file aplikasi umum lainnya seperti Excel. File IMP telah ditandai sebagai usang dan tidak lagi menjadi format file yang didukung. Lotus Improv kemudian diakuisisi oleh IBM pada tahun 1996.

## Berformat IMP

File IMP disimpan pada disk sebagai file biner dan format file internalnya tidak tersedia untuk umum. Lotus Improv kemudian digantikan oleh IBM Lotus 1-2-3.
