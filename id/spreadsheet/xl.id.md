## Apa itu file XL?

File dengan ekstensi .xl adalah file Excel Spreadsheet, mirip dengan [XLS](/id/spreadsheet/xls/), yang menyimpan data dalam baris dan kolom. Itu adalah format file sebelumnya yang tidak lagi didukung oleh Excel sekarang. Data yang disimpan dalam file XL dapat diekspor ke format lain seperti CSV. Ini juga dapat dikonversi ke PDF menggunakan metode Save As default Excel. File XL hanya dapat dibuka dengan Microsoft Excel.

## Berformat XL

File XL disimpan sebagai file biner dan spesifikasi format file internalnya tidak didokumentasikan di mana pun. Namun, format file biner umum untuk [Spesifikasi Format File XLS](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) format file juga dapat dipertimbangkan jika seseorang mencoba menulis sepotong kode untuk merekayasa balik file-file ini.
