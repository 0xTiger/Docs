## Apakah berkas \_XLS itu?

File dengan ekstensi .\_XLS sebenarnya adalah file XLS yang telah diubah namanya menjadi \_XLS oleh beberapa aplikasi lain. Itu didasarkan pada struktur format file XLS, dan jika diubah namanya menjadi .xls, dapat dibuka dengan semua aplikasi yang mendukung format file XLS. Dalam kebanyakan kasus, file \_XLS dihasilkan dengan file XLS yang dilampirkan ke email yang mengubahnya menjadi \_XLS di pihak penerima. Di dalamnya, gunakan [Spesifikasi Formasi File XLS](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) yang sama untuk memuat berbagai jenis informasi seperti bagan , grafik, dan data spreadsheet.

## \_XLS File Format - Informasi lebih lanjut

File \_XLS disimpan ke disk dalam format file biner dan dapat dibuka di Microsoft Word dengan mengubah ekstensinya menjadi XLS. Format lengkap [format file dari \_XLS file](https://msdn.microsoft.com/en-us/library/dd922181(v#office.12).aspx) tersedia secara publik oleh Microsoft untuk referensi pengembang.
