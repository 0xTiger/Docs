## Apa itu file EFU?

File EFU adalah file spreadsheet [CSV](/id/spreadsheet/csv/) yang dibuat dengan perangkat lunak Semuanya oleh [voidtools](https://www.voidtools.com/). Ini berisi daftar file yang dicari menggunakan mesin pencari Semuanya untuk windows. Hasil dalam file EFU dibatasi CSV dan dapat dibuka dengan program spreadsheet seperti Microsoft Excel. File EFU dapat dibaca dengan sangat baik dan dapat dibuka dengan editor teks apa pun seperti Microsoft Notepad, Notepad ++, dan Apple TextEdit.

## Format File EFU - Informasi Lebih Lanjut

File EFU disimpan ke disk dalam file nilai yang dipisahkan koma yang berisi informasi tentang hasil yang diperoleh oleh aplikasi mesin pencari Semuanya. Catatan ini berisi informasi tentang file yang ditemukan seperti nama file, ukuran, pembuatan, dan tanggal modifikasi terakhir.
