## Apa itu file TMV?

File TMV adalah file spreadsheet yang dibuat dengan software [TimeMap](https://www.lexisnexis.com/en-us/products/casemap.page) oleh LexisNexis. TimeMap memungkinkan Anda membuat garis waktu dengan tanggal dan waktu fakta yang ditentukan. Dengan masuknya fakta baru, perangkat lunak membuat kotak fakta untuk ini dan memposisikannya pada kemunculan garis waktu. Proyek TMV dapat terdiri dari kotak fakta, kotak teks, dan teks di dalamnya dapat diformat menggunakan font yang berbeda. File TMV dapat dibuka dengan perangkat lunak TimeMap. LexisNexis juga menyediakan alat kolaborasi online sekarang di lingkungan berbasis cloud.

## Format File TMV

File TMV disimpan ke disk dalam format file berpemilik CaseMap yang spesifikasi format filenya tidak tersedia secara online. CaseMap diakuisisi oleh LixesNexis.
