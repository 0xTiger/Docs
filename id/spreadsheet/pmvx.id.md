## Apa itu file PMVX?

File dengan ekstensi .pmvx adalah file template spreadsheet yang dibuat dengan program spreadsheet [PlanMaker](https://www.softmaker.de/softmaker-office-planmaker), yang hadir sebagai bagian dari Softmaker Office. Data disimpan dalam baris dan kolom dalam file PMVX dengan cara yang sama seperti yang dilakukan di Microsoft Excel. File PMVX menyimpan pengaturan yang telah ditentukan sebelumnya sebagai template yang dapat digunakan untuk menghasilkan file PMD dan PMDX setelah mengisi data di dalamnya. [XLSX](/id/spreadsheet/xlsx/) adalah format file default untuk PlanMaker dan ini dapat dibuka langsung dengan Microsoft Excel. Anda juga dapat mengedit file [XLS](/id/spreadsheet/xls/) dan membuat file [PDF](/id/pdf/) menggunakan PlanMaker.

## Format File PMVX

File PMVX disimpan ke disk dalam format file milik PlanMaker dan struktur file internalnya tidak diketahui. Sejauh menyimpan file PMVX, ini menyimpan data ke format Open XML sebagai file XLSX.
