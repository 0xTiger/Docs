## Apa itu file SDC? ##

File SDC adalah file spreadsheet yang dibuat dengan program spreadsheet [Apache OpenOffice Calc](https://en.wikipedia.org/wiki/LibreOffice_Calc) yang juga merupakan bagian dari StarOffice. Seperti aplikasi spreadsheet lainnya, data disimpan dalam file SDC dalam kisi sel dalam baris dan kolom. File SDC memiliki MIME Type application/vnd.stardivision.calc dan rencananya akan digunakan untuk spreadsheet StarCalc 5.x. Calc adalah program spreadsheet gratis dan digunakan oleh pendatang baru maupun profesional. Itu dapat mengekspor data dari file SDC ke format file Microsoft Excel XLS dan PDF. Selain Calc, [LibreOffice](https://www.libreoffice.org/) dan [Planamesa NeoOffice](https://www.neooffice.org/neojava/en/index.php).

## Format File SDC

File SDC disimpan dalam format file berpemilik dan format file internalnya tidak diketahui.
