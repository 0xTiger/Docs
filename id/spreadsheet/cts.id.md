## Apa itu file CTS?

File CTS adalah file spreadsheet yang dibuat dengan entri data multiplatform sumber terbuka dan program organisasi, **TreeSheets**. Ini berisi data hierarkis yang dimasukkan dalam berbagai format di dalam spreadsheet seperti catatan, daftar, kalender, daftar tugas, catatan, peta pikiran, dan sejumlah format lainnya. File CTS hanya dapat dibuka dengan TreeSheets yang ringan, cepat, dan berfungsi di Windows, Linux, dan macOS.

## Format File CTS - Informasi Lebih Lanjut

TreeSheets menyimpan file CTS dalam format file miliknya sendiri. Spesifikasi format file format file CTS tidak tersedia untuk umum dan itulah sebabnya file CTS hanya dapat dibuka di perangkat lunak TreeSheets.

### Mengekspor File CTS TreeSheets ke Format Lain

Pengguna TreeSheets dapat mengekspor spreadsheet CTS ke beberapa format file yang berbeda. File CTS dapat diekspor ke format file [HTML](/id/web/html/), [JSON](/id/web/json/), [CSV](/id/spreadsheet/csv/), dan [Gambar](/id/image/) .

## Tentang Perangkat Lunak TreeSheets

TreeSheets adalah program perangkat lunak sumber terbuka yang tersedia untuk multiplatform. Ini memungkinkan Anda membuat spreadsheet hierarkis dan memberi Anda kebebasan untuk meletakkan semua jenis data dalam satu file spreadsheet. Keuntungan menggunakan TreeSheets adalah Anda dapat memasukkan semua diskusi Anda dalam bentuk catatan, peta pikiran, data teks, gambar, dan jenis data lainnya dalam satu file spreadsheet.
