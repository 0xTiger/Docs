## Apa itu file SXC? ##

Format file SXC (Sun XML Calc) milik sebuah office suite bernama OpenOffice.org. Format ini umumnya menangani kebutuhan spreadsheet pengguna karena merupakan format file spreadsheet berbasis XML. Format SXC mendukung rumus, fungsi, makro, dan bagan bersama dengan DataPilot, yang merupakan fitur luar biasa karena secara otomatis mengindividualisasikan dan menyediakan ringkasan data mentah yang diimpor. File yang dibuat dengan perangkat lunak ini disimpan dengan ekstensi .sxc.


## Format File SXC ##

File berekstensi .sxc dapat dibuka dengan Microsoft excel yang merupakan software spreadsheet dari Microsoft Office suite. Ada software lain yang bisa membuka file dengan ekstensi .sxc yaitu Libre Office, StarOffice dll.

Jenis konten untuk format ini adalah:

* aplikasi/vnd.sun.xml.calc
* aplikasi/vnd.sun.xml.calc.template

## Kompatibilitas File ##

File SXC kompatibel dengan Apache Open Office calc dan data dapat diekspor ke file Microsoft excel dan format lainnya juga.

## Penanganan File ##

Perintah ImportMatrix dan ExportMatrix dapat membaca dan menulis file SXC. Ketika sumber adalah file SXC dan target juga merupakan file SXC, ini menyebabkan **f** dianggap sebagai file Sun XML Calc Spreadsheet. Matriks Impor dan Ekspor mendukung format ini, yang terdeteksi secara otomatis ketika format tidak ditentukan dan nama file diakhiri dengan .sxc.

Untuk membuka file SXC, diperlukan program yang sesuai untuk dikaitkan ke file tersebut. Setiap kali file dengan ekstensi .sxc dibuka, pastikan file tersebut bebas dari kesalahan dan tidak rusak. Kesalahan apa pun dalam file dapat dengan mudah ditangani oleh pengguna dan ini sebagian besar tidak memerlukan dukungan teknis.
