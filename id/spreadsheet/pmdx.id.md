## Apa itu file PMDX?

File PMDX adalah dokumen spreadsheet yang dibuat dengan versi terbaru [PlanMaker](https://www.softmaker.de/softmaker-office-planmaker), perangkat lunak yang hadir sebagai bagian dari SoftMaker Office. Ini menyimpan data dalam sel dalam bentuk baris dan kolom. Setiap sel dapat berisi string, angka, rumus, atau data berformat lainnya. File PMDX menggantikan file [PMD](/id/spreadsheet/pmd/) versi lama dengan pemutakhiran PlanMaker. XLSX adalah format file standar yang didukung oleh PlanMaker dan tidak memerlukan konversi apa pun saat membuka file ini dengan Microsoft Excel. PlanMaker juga mendukung bekerja dengan file [XLS](/id/spreadsheet/xls/) dan mengonversinya ke format lain seperti [PDF](/id/pdf/).

## Format File PMDX

File PMDX disimpan ke disk sebagai file biner dalam format file berpemilik dan spesifikasinya tidak tersedia untuk umum.
