## Apa itu file TMVT?

File TMVT adalah file template spreadsheet garis waktu TimeMap yang berisi pengaturan standar untuk membuat [file TMV](/id/spreadsheet/tmv/). Berbeda dengan file TMV, TMVT berisi pengaturan seperti gaya bendera fakta, latar belakang, gaya judul, dan gaya skala. File TMVT dapat dibagikan dengan pengguna dan digunakan kembali oleh mereka untuk mempertahankan pengaturan yang sama dan membuat file timeline mereka sendiri untuk keseragaman. Ini dapat dibuka dengan perangkat lunak TimeMap oleh [LexisNexis](https://www.lexisnexis.com/en-us/products/casemap.page).

## Format File TMVT

File TMVT dibuat dan disimpan ke disk dalam format file biner.
