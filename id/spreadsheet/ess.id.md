## Apa itu file ESS?

File ESS adalah file spreadsheet yang dibuat dan didukung oleh perangkat lunak aplikasi spreadsheet EasySpreadsheet yang hadir sebagai bagian dari [EasyOffice](https://en.wikipedia.org/wiki/EasyOffice). EasyOffice diluncurkan sebagai alternatif untuk Microsoft Office tetapi tidak lagi tersedia untuk digunakan dan telah digantikan oleh rangkaian lain, yang disebut One. Itu datang dalam dua varian yaitu EasyOffice Freeware dan EasyOffice Premium dengan yang kemudian memiliki Filter PDF dan EasyAntiVirus. File ESS kompatibel dengan file Microsoft [XLS](/id/spreadsheet/xls/).
