## Apa itu file NUMBERS-TEF?
File NUMBERS-TEF dibuat saat perangkat lunak spreadsheet Apple iWork Numbers menyimpan spreadsheet ke iCloud. Sedangkan, spreadsheet Numbers yang disimpan di komputer lokal menggunakan ekstensi **.numbers**.


## Perbedaan antara ANGKA dan ANGKA-TEF
File NUMBERS-TEF adalah spreadsheet serupa seperti .numbers. Kedua file dapat dibuat dengan menggunakan perangkat lunak spreadsheet Apple iWork Numbers. Harap diperhatikan bahwa Apple iWork Numbers termasuk dalam iWork Productivity Suite. NUMBERS-TEF hanyalah sufiks **-TEF** yang ditambahkan ke dokumen Numbers saat menyimpannya ke iCloud untuk digunakan di perangkat iOS, atau Mac lain.
Numbers versi Mac OS atau iOS membuka **.numbers-tef** tanpa masalah. Spreadsheet akan dimuat dengan benar seperti file **.numbers**.
