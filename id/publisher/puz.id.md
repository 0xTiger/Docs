## Apa itu file PUZ?

File dengan ekstensi .puz adalah arsip terkompresi yang berisi dokumen Microsoft Publisher [.pub](/id/publisher/pub/) standar. Sebagai hasil dari kompresi, ukurannya lebih kecil dan mudah ditransfer melalui internet atau perangkat lain. File PUZ dapat dibongkar/dibuka menggunakan utilitas Microsoft Unpack.

## Format File PUZ

File PUZ adalah arsip terkompresi yang berisi file PUB standar bersama dengan file lainnya. Ini berisi semua file yang diperlukan untuk membongkar, menampilkan, dan mencetak publikasi.

File PUZ berisi yang berikut ini.

* `Unpack.exe` - utilitas perangkat lunak untuk membongkar file terkompresi
* `ReadMe.txt` - file deskripsi
* `File .puz terkompresi` - Berisi publikasi dan mungkin menyertakan font tersemat dan grafik tertaut.

### Membongkar File PUZ

File Publisher yang dikemas dapat dibongkar menggunakan utilitas Unpack.exe. Ini mengekstrak file PUZ yang dikemas ke direktori pilihan yang dipilih oleh pengguna.
