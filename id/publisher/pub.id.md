## Apa itu file PUB?

File PUB adalah format file dokumen Microsoft Publisher. Ini digunakan untuk membuat beberapa jenis dokumen tata letak desain seperti buletin, selebaran, brosur, kartu pos, dll. File PUB dapat berisi teks, raster, dan gambar vektor. Dokumen yang dibuat dengan file PUB sebagian besar digunakan di situs web dan materi pemasaran seperti Email. File PUB dapat dibuka dengan aplikasi Microsoft Publisher Desktop, [Microsoft Publisher 365](https://www.microsoft.com/en/microsoft-365), [LibreOffice Draw](https://www.libreoffice.org/discover /draw/), dan [Adobe InDesign](https://www.adobe.com/products/indesign.html).

## Format File PUB - Informasi Lebih Lanjut

File PUB disimpan sebagai file biner lakukan disk. Konten file PUB juga dapat dikonversi ke format file [DOCX](/id/word-processing/docx/) dari dalam aplikasi Microsoft Publisher dengan menggunakan opsi Simpan Sebagai.

## Bagaimana cara menggunakan Microsoft Publisher?

Microsoft telah menetapkan beberapa pedoman untuk membantu pengguna memulai dengan Microsoft Publisher. Panduan [Tugas dasar di Publisher](https://support.microsoft.com/en-us/office/basic-tasks-in-publisher-0e5ed249-1927-433f-a35c-63beb8216fcf) ini memungkinkan Anda memulai dengan menunjukkan caranya untuk membuat publikasi, menyimpan publikasi, menambahkan teks dan blok penyusun, dan mencetak publikasi.
