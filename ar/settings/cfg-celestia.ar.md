## ما هو ملف CFG؟

ملف تكوين Celestia هو ملف نصي عادي يستخدمه Celestia, برنامج محاكاة الكون ثلاثي الأبعاد. تعتبر هذه الملفات حيوية لتخصيص كيفية تصرف Celestia, والأجرام السماوية التي يتم عرضها, وكيفية ظهور البرنامج. ومع ذلك, يتطلب تحرير هذه الملفات اهتمامًا دقيقًا لأن التغييرات غير الصحيحة يمكن أن تعطل عملية تحميل Celestia, مما يمنعها من العمل بشكل صحيح.

## سيليستيا

Celestia هو برنامج محاكاة فلك ثلاثي الأبعاد مجاني ومفتوح المصدر يتيح للمستخدمين استكشاف الكون وتصوره بطريقة واقعية وتفاعلية للغاية. تم تطويره بواسطة كريس لوريل وتم إصداره لأول مرة في أوائل العقد الأول من القرن الحادي والعشرين. يتوفر Celestia لأنظمة التشغيل Windows وmacOS وLinux.

تشمل الميزات والجوانب الرئيسية لـ Celestia ما يلي:

- **تصور ثلاثي الأبعاد واقعي:** توفر سيليستيا تمثيلاً تفصيليًا ودقيقًا لنظامنا الشمسي, بالإضافة إلى النجوم والمجرات والأجرام السماوية الأخرى. يستخدم نماذج وأنسجة ثلاثية الأبعاد عالية الجودة لإنشاء تجربة غامرة بصريًا.

- **قاعدة بيانات سماوية واسعة النطاق:** يأتي البرنامج مع قاعدة بيانات مدمجة واسعة للأجرام السماوية المعروفة, بما في ذلك النجوم والكواكب والأقمار والكويكبات والمذنبات والمركبات الفضائية. يمكن للمستخدمين تحديد موقع هذه الكائنات واستكشافها بسهولة.

- **الدقة العلمية:** على الرغم من أن Celestia عبارة عن أداة تصور, إلا أنها تهدف إلى تمثيل الأجرام والظواهر السماوية بأكبر قدر ممكن من الدقة, استنادًا إلى البيانات العلمية المتاحة.

## تنسيقات الملفات المستخدمة من قبل سيليستيا

تستخدم Celestia تنسيقات ملفات مختلفة لجوانب مختلفة من محاكاة علم الفلك ثلاثي الأبعاد. فيما يلي بعض تنسيقات الملفات الرئيسية التي تستخدمها Celestia:

1. **ملفات التكوين (.cel)**
- *الوصف*: ملفات نصية عادية تسمح للمستخدمين بتخصيص سلوك Celestia ومظهرها ومحتواها.
- *الغرض*: تخصيص الإعدادات وتحديد المواقع وتحديد الأجرام السماوية.

2. **النماذج والشبكات ثلاثية الأبعاد**
- *التنسيقات*: [.3ds](/ar/3d/3ds/), [.obj](/ar/3d/obj/), [.dae](/ar/3d/dae/), .ac
- *الوصف*: تنسيقات ملفات النماذج ثلاثية الأبعاد المدعومة المستخدمة لعرض الأجرام السماوية والمركبات الفضائية.
- *الغرض*: عرض الكائنات ثلاثية الأبعاد داخل سيليستيا.

3. ** ملفات النسيج **
- *التنسيقات*: [.jpg](/ar/image/jpeg/), [.png](/ar/image/png/), [.dds](/ar/image/dds/)
- *الوصف*: توفر هذه الملفات أنسجة سطحية للأجرام السماوية.
- *الغرض*: رسم خرائط للأنسجة على نماذج ثلاثية الأبعاد للحصول على مظهر واقعي.

4. **كتالوجات النجوم وملفات البيانات**
- *التنسيقات*: التنسيقات المخصصة, [.csv](/ar/spreadsheet/csv/), [.tsv](/ar/spreadsheet/tsv/)
- *الوصف*: ملفات البيانات المستخدمة لتمثيل النجوم والأجرام السماوية الأخرى, مما يضمن محاكاة دقيقة.
- *الغرض*: تمثيل دقيق للأجرام السماوية.

5. **خرائط المكعبات**
- *الوصف*: تُستخدم الخرائط المكعبة لمحاكاة مظهر الأجرام السماوية البعيدة مثل المجرات.
- *الغرض*: عرض الأشياء البعيدة بأنسجة واقعية.

6. **ملفات البرنامج النصي**
- *الوصف*: تحتوي هذه الملفات على نصوص برمجية مخصصة مكتوبة بلغة البرمجة النصية الخاصة بـ Celestia.
- *الغرض*: تمكين المستخدمين من إنشاء أحداث ديناميكية ورسوم متحركة داخل عالم سيليستيا.

## ملف تكوين سيليستيا

فيما يلي نظرة عامة أساسية على ما يمكنك فعله باستخدام ملف تكوين Celestia:

1. **تحديد موقعك**: يمكنك تحديد موقعك على الأرض باستخدام معلمات خطوط الطول والعرض والارتفاع. يتيح ذلك لـ Celestia عرض السماء ليلاً بدقة من موقعك.

```
Location "My Location"
{
    Latitude 40.7128
    Longitude -74.0060
    Altitude 0
}
```

2. **تخصيص خيارات العرض الخاصة بك:** يمكنك ضبط خيارات العرض المتنوعة مثل مجال الرؤية, وإعدادات الوقت, وتفضيلات العرض.

```
Viewpoint
{
    Location "My Location"
    Follow "Earth"
    Eye [0.0 0.0 0.0]
    Center [0.0 0.0 0.0]
    Up [0.0 1.0 0.0]
    Fov 45
}

Time
{
    Date "2023-09-25T12:00:00.000Z"
    Clock "Now"
}

Rendering
{
    Atmosphere false
    Stars 7
    Planetshine 0.25
}

```

3. ** تحميل الأجرام السماوية: ** يمكنك إضافة الأجرام السماوية مثل النجوم والكواكب والكويكبات والمركبات الفضائية والمزيد إلى المحاكاة الخاصة بك. يتم تعريف كل كائن في ملف التكوين مع خصائصه.

```
Star "Sun"
{
    RA 0
    Dec 0
    Distance 0
    AppMag -26.74
    SpectralType "G2V"
}

Planet "Earth"
{
    Parent "Sol"
    Texture "earth.jpg"
    Radius 6371
    EllipticalOrbit
    {
        Period 365.25
        SemiMajorAxis 149.6e6
        Eccentricity 0.017
        Inclination 0
        AscendingNode 0
        ArgOfPericenter 102.94
        MeanAnomaly 100.464
}
}
```

4. **تعريف سفن الفضاء:** يمكنك إنشاء مركبة فضائية خيالية خاصة بك أو استخدام مركبات حقيقية من خلال تحديد معلماتها مثل الموقع والاتجاه والنماذج ثلاثية الأبعاد.

```
Spacecraft "Voyager 1"
{
    Parent "Sol"
    Class "spacecraft"
    Mesh "voyager.3ds"
    Radius 0.01
    EllipticalOrbit
    {
        Period 30700
        SemiMajorAxis 1.08e11
        Eccentricity 0.044
        Inclination 3.4
        AscendingNode 49.0
        ArgOfPericenter 44.0
        MeanAnomaly 35.0
}
}
```

5. **إنشاء البرامج النصية:** يمكنك كتابة البرامج النصية بلغة البرمجة النصية المخصصة لـ Celestia لإنشاء أحداث ورسوم متحركة ديناميكية.
