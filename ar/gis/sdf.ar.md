## ما هو ملف SDF؟

A Spatial Data File (SDF) is a Geodatabase file format that was developed by Autodesk. It stores geospatial data and is used by Autodesk GIS program MapGuide and AutoCAD Map 3D. SDF file format is based on the single file database file format SQLite3. ويعتبر بمثابة تنسيق ملف محسّن لمجموعات البيانات الكبيرة من المعلومات المكانية.

يمكنك فتح ملف SDF باستخدام Autodesk AutoCAD Map 3D 2022 وAutodesk AutoCAD Civil 3D 2023.

## تنسيق ملف SDF - مزيد من المعلومات

يتم تخزين تنسيق ملف SDF على القرص كملفات ثنائية. يعتمد على قاعدة بيانات SQLite التي تستخدم مكونات تخزين منخفضة المستوى للتسلسل الثنائي للبيانات. يحتوي ملف SDF على فئات ميزات متعددة لكل ملف وخصائص هندسية متعددة لكل فئة معالم. يمكن الوصول إلى البيانات الموجودة في ملف SDF بسرعة عالية بسبب فهرسة كل خاصية هندسية باستخدام شجرة R.
