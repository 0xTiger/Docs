## ما هو ملف GCF؟

ملف .gcf هو ملف ذاكرة تخزين مؤقت للعبة تستخدمه منصة ألعاب Steam. يحتوي على بيانات اللعبة مثل الأنسجة والنماذج والملفات الأخرى التي تستخدمها اللعبة. يتم تخزين هذه الملفات على كمبيوتر المستخدم وتستخدم لتقليل كمية البيانات التي يجب تنزيلها عند تحديث اللعبة أو تثبيتها. يمكن أيضًا استخدام ملفات .gcf لإنشاء نسخ احتياطية لتثبيت اللعبة أو لنقل اللعبة بين أجهزة الكمبيوتر.

يمكن قراءة ملفات GCF وكتابتها بواسطة مكتبة برمجة C++ مفتوحة المصدر، **HLLib**.

## تنسيق ملف جي سي إف

يتم حفظ ملفات GCF على القرص كملفات ثنائية. تم استخدام GCF في الأصل كاختصار لـ Grid Cache File (الاسم الرمزي المبكر لـ Steam كان Grid)، ولكن تم اعتباره لاحقًا كملف Game Cache File. ملف GCF هو ملف أرشيف يقوم بتخزين ألعاب Steam. يتم تنزيل كل المحتوى الرسمي كملفات GCF. يمكن أيضًا مشاركة ملفات GCF بين الألعاب.

ملاحظة: GCF هو الإصدار السابق لـ [VPK file format](/game/vpk/).
