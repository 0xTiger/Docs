## ما هو ملف WS؟

عادةً ما يحتوي ملف **.WS** الموجود على نظام التشغيل Windows على برنامج نصي قابل للتنفيذ مصمم ليتم تنفيذه من خلال **Windows Script Host**. يمكن أن يشمل هذا البرنامج النصي عناصر وإجراءات متنوعة مثل إجراءات JScript وVBScript بالإضافة إلى عناصر XML. عند النقر المزدوج فوق ملف .ws، سيبدأ Windows Script Host في تنفيذ البرنامج النصي المضمن داخل الملف.

## حول ويندوز هوست سكريبت

Windows Script Host (WSH) هو مضيف برمجة نصية تم تطويره بواسطة Microsoft والذي يسمح للمستخدمين بتشغيل البرامج النصية المكتوبة بلغات البرمجة النصية مثل VBScript (إصدار Visual Basic Scripting) وJScript (إصدار Microsoft من JavaScript)؛ فهو يوفر إطارًا للبرمجة النصية في نظام التشغيل Windows، مما يتيح أتمتة المهام المختلفة وإدارة النظام.

فيما يلي بعض النقاط الأساسية حول Windows Script Host:

1.  **لغات البرمجة النصية:** تدعم WSH العديد من لغات البرمجة النصية، مع كون VBScript وJScript الأكثر استخدامًا؛ يمكن كتابة البرامج النصية بأي من هذه اللغات لأتمتة المهام أو التفاعل مع نظام التشغيل أو معالجة البيانات.
    
2.  **تنفيذ البرنامج النصي:** عادةً ما يتم تخزين البرامج النصية WSH في ملفات ذات امتدادات مثل .vbs لـ VBScript أو .js لـ JScript؛ عند تشغيل برنامج نصي، يفسر WSH التعليمات البرمجية وينفذها.
    
3.  **الأتمتة:** غالبًا ما يتم استخدام WSH لأغراض التشغيل الآلي مثل أتمتة المهام المتكررة وإدارة إعدادات النظام وتنفيذ الوظائف الإدارية؛ مفيد بشكل خاص لمسؤولي النظام ومستخدمي الطاقة.
    
4.  ** البرامج النصية لوحدة التحكم وواجهة المستخدم الرسومية: ** يمكن أن تكون البرامج النصية لـ WSH إما قائمة على وحدة التحكم أو واجهة المستخدم الرسومية؛ تعمل البرامج النصية لوحدة التحكم في نافذة سطر الأوامر بينما يمكن للبرامج النصية لواجهة المستخدم الرسومية إنشاء واجهات مستخدم رسومية لمزيد من التطبيقات التفاعلية.
    
5.  **Security:** WSH has security features to help prevent malicious scripts from causing harm; users are typically prompted before executing scripts and scripts are subject to certain restrictions to prevent unauthorized access and actions.
    
6.  **نموذج كائن Windows Script Host:** يوفر WSH نموذج كائن يسمح للبرامج النصية بالتفاعل مع نظام التشغيل Windows؛ يتضمن ذلك الوصول إلى أنظمة الملفات وإعدادات التسجيل وموارد الشبكة ومكونات النظام الأخرى.
    
7.  **تنفيذ البرنامج النصي من سطر الأوامر:** يمكنك تنفيذ البرامج النصية WSH من سطر الأوامر باستخدام الملفات التنفيذية `cscript.exe` أو `wscript.exe`، اعتمادًا على ما إذا كنت تريد تشغيل البرنامج النصي في وحدة تحكم أو بيئة واجهة المستخدم الرسومية .

## كيفية فتح ملف WS؟

ملفات WS هي ملفات نصية عادية، لذلك إذا كنت تريد فتحها أو تحريرها، يمكنك استخدام أي محرر نصوص على سبيل المثال

- **المفكرة**
- **المفكرة++**
- **برنامج Apple TextEdit**
- **كود فيجوال ستوديو**

يرجى ملاحظة أنه إذا كنت تريد تشغيل البرنامج النصي داخل ملف WS، فيمكنك ببساطة النقر عليه نقرًا مزدوجًا أو استخدام الأمر `wscript` في سطر الأوامر.
