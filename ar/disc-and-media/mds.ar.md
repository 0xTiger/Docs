## ما هو ملف MDS؟

يعد MDS (ملف Media Descriptor Sidecar File) ملفًا مرتبطًا ببرنامج الكحول 120% وDaemon Tools، وكلاهما يستخدم لإنشاء وإدارة محركات الأقراص المضغوطة وأقراص DVD الافتراضية. يحتوي ملف MDS على معلومات حول تخطيط البيانات الموجودة على القرص، بما في ذلك موقع جميع الملفات وبنية القرص. يسمح هذا لمحرك الأقراص الظاهري بمحاكاة سلوك القرص الفعلي، بحيث يتمكن البرنامج من قراءة البيانات الموجودة على القرص الظاهري كما لو كان قرصًا حقيقيًا.

When you want to mount an image file to a virtual drive, the MDS file is used along with the corresponding image file in a format such as ISO, BIN or CUE. The MDS file contains a description of the layout of the data on the disc, and the virtual drive software uses this information to create a virtual disc. This allows you to use the software as if you were using a physical disc, without having to physically insert the disc into the drive.

## العلاقة مع الكحول 120%

يعد برنامج الكحول 120% برنامجًا شائعًا لإنشاء وإدارة محركات الأقراص المضغوطة وأقراص DVD الافتراضية، ويستخدم تنسيق ملف MDS لحفظ صور القرص والوصول إليها. يعد تنسيق ملف MDS ملكية خاصة بـ الكحول 120% ولا يمكن فتحه واستخدامه إلا بواسطة برنامج الكحول 120% أو البرامج الأخرى التي تدعمه.

## كيفية فتح ملف MDS؟

لفتح ملف MDS في الكحول 120%، سوف تحتاج إلى تثبيت البرنامج على جهاز الكمبيوتر الخاص بك. بمجرد تثبيت الكحول 120%، يمكنك فتح ملف MDS عن طريق القيام بما يلي:

1. إطلاق الكحول 120٪.
2. انقر على القائمة ملف وحدد فتح.
3. انتقل إلى موقع ملف MDS على جهاز الكمبيوتر الخاص بك وحدده.
4. سيتم الآن فتح ملف MDS في نسبة الكحول 120%، وسيُطلب منك تحديد ملف الصورة المقابل (مثل ISO أو BIN أو CUE).
5. بعد تحديد ملف الصورة، سيقوم الكحول 120% بتحميل الصورة على محرك أقراص افتراضي.

وبدلا من ذلك، يمكنك أيضا فتح ملف MDS بالنقر المزدوج عليه، إذا تم تعيين الكحول 120% كبرنامج افتراضي لفتح ملفات MDS. يمكنك أيضًا استخدام برامج أخرى تدعم تنسيق MDS مثل Daemon Tools وVirtual CloneDrive وVirtual Drive Manager وMagicISO.
