## ما هو الملف الرابع؟

الملف ذو ملحق الملف .4 هو ملف برمجة تم إنشاؤه لـ **لغة البرمجة الرابعة**. يحتوي على الكود المصدري للبرامج المكتوبة بلغة البرمجة الرابعة ويقوم بإنشاء المخرجات عند تنفيذ البرنامج. إنه مجرد ملف لغة برمجة آخر يشبه الملف المصدر [C](/ar/programming/c/) و[C++](/ar/programming/cpp/).

## تنسيق الملف الرابع - مزيد من المعلومات


### المثال الرابع لرمز لغة البرمجة

فيما يلي مثال لبرنامج بسيط مكتوب بلغة Forth يقوم بحساب مضروب رقم معين:

```
: factorial ( n -- n! )
   dup 0=
   if
      drop 1
      exit
   then
   1
   swap
   begin
      dup 1-
      dup 0=
      if
         drop
         exit
      then
      swap
      *
   repeat
;

```

في هذا المثال, تأخذ الكلمة المضروب وسيطة واحدة n وترجع مضروبها. تقوم الكلمة dup بتكرار القيمة الموجودة أعلى المكدس, ويتجاهل الإسقاط القيمة الموجودة أعلى المكدس, و* يضاعف القيمتين الموجودتين أعلى المكدس. تتحكم بنيات if و begin/ until في تدفق البرنامج.

لاستخدام هذا البرنامج, يجب عليك إدخال التعريف في المترجم الرابع, ثم استدعاء الكلمة المضروب بالرقم الذي تريد العثور على المضروب منه:

```
10 factorial .
```
سيؤدي هذا إلى الناتج 3628800, وهو مضروب 10.
