## ما هو ملف جافا؟

ملف Jav هو ملف تعليمات برمجية مصدر Java تم إنشاؤه بلغة برمجة Java. إنه مشابه لملف Java المصدر ويمكن تجميعه باستخدام مترجمي Java. مثل ملفات [Java](/ar/programming/java/), يُستخدم ملف Jav في تطوير تطبيقات Java. يمكن أن يكون أحد ملفات التعليمات البرمجية المصدر العديدة المتوفرة في مشروع Java.

## تنسيق ملف جافا

يتم تخزين ملفات Jav كملفات نصية ويمكن فتحها باستخدام أي محرر نصوص. تم استبدال أنواع ملفات Jav بشكل شائع بنوع ملف Java ويستخدم بشكل شائع لكتابة تطبيقات Java. يمكن كتابة أكواد Jav باستخدام بيئة تطوير متكاملة (IDE) مثل Eclipse وIntelliJ Idea مما يسهل العمل مع ملفات Java.

### مثال على تنسيق ملف Jav

```java
/*
The example code prints
Hello World from Java to the console.
*/
    public class JavaApp {
        public static void main(String[] args) {
            System.out.println("Hello World."); // Prints the string to the console.
        }
    }
```
