## ما هو ملف NX؟

ملف NX هو ملف لعبة فيديو تم إنشاؤه لاستخدامه مع وحدة التحكم في الألعاب الافتراضية, **LowRes NX**. إنه مكتوب بلغة البرمجة BASIC ويحتوي على الكود المصدري للعبة الذي يحدد كيفية عمل اللعبة. يمكنك إنشاء ملفات NX الخاصة بك لاستخدامها مع التطبيق الذي يمكن تحميله ويمكن للمستخدمين تشغيل اللعبة.

يمكنك فتح ملف NX في محرر نصوص أو LowRes NX أو أي محرر تعليمات برمجية مصدر مثل Microsoft Visual Studio Code.

## تنسيق ملف NX

يتم حفظ ملف NX على القرص كملف نصي عادي يحتوي على تعليمات اللعبة. يحتوي كل ملف NX على كود مصدر اللعبة الذي يتكون من إدخالات ROM متعددة للخرطوشة. تحتوي إدخالات ROM هذه على معلومات مكونات اللعبة مثل الرسومات وخرائط المستوى والصوت وغيرها من بيانات التعليمات البرمجية غير المصدرية للعبة. يمكن إضافة أو تحرير إدخالات ROM الخاصة بأحد البرامج باستخدام برامج التحرير NX التي تأتي مرفقة مع LowRes NX.

## كيفية إنشاء ملفات NX؟

### إنشاء ملفات NX على جهاز الكمبيوتر

يمكنك إنشاء ملفات NX على جهاز الكمبيوتر باستخدام:

* برامج تحرير النصوص - مثل Microsoft Notepad أو Notepad++ أو Github Atom
* برامج تحرير كود المصدر - مثل Microsoft VIsual Studio Code

### إنشاء ملفات NX على iOS

يحتوي LowRes NX على إصدار iOS يمكن للمطورين استخدامه لإنشاء ملفات NX من داخل هذا التطبيق.
