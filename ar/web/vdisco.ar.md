## ما هو ملف VDISCO؟

ملف VDISCO هو ملف اكتشاف يستخدمه Microsoft Visual Studio في خدمات الويب الخاصة به. فهو يوفر معلومات حول خدمات الويب المتاحة مثل عناوين URL ومساحات الأسماء والأساليب الخاصة بها. يشبه ملف VDISCO تنسيق الملف [DISCO](/ar/web/disco/) ولكن يتم إنشاؤه في وقت التشغيل. يتم تخزينه في تطبيق خدمة الويب ويستخدمه Visual Studio لاكتشاف خدمات الويب واستخدامها ديناميكيًا في المشروع. يتم استخدام ملفات VDISCO مع الملف [.asmx](/ar/web/asmx/) الذي يحتوي على رمز خدمة الويب الفعلي.

يمكنك فتح ملف VDISCO في أي محرر نصوص مثل Microsoft Notepad وGithub Atom وApple TextEdit. يمكن أيضًا فتحه باستخدام Microsoft Visual Studio 2012 والإصدارات الأحدث للعمل معه.

## تنسيق ملف فديسكو

يتم حفظ ملف VDISCO واستضافته بتنسيق ملف [XML](/ar/web/xml/) وهو تنسيق عالمي لتكوين البيانات ونشرها. يحتوي على عنوان URL للخدمة ومساحات الأسماء والأساليب في علامات XML القياسية حيث تحتوي كل علامة على القيمة الخاصة بمعلوماتها.
