## ما هو ملف XHTM؟

الملف ذو الامتداد .xhtm هو ملف صفحة ويب مكتوب بلغة XHTML. يستخدم XML 1.0 كتطبيق قياسي ويعتبر إعادة صياغة لـ HTML 4. مثل XHTML, يحتوي ملف XHTM على علامات HTML القياسية التي تتبع إرشادات XML الأكثر صرامة. تُستخدم ملفات XHTM لنشر محتوى موقع الويب وتستخدم بشكل شائع امتداد [XHTML](/ar/web/xhtml/).

يمكنك استخدام متصفحات الويب مثل Google Chrome وApple Safari وMozilla Firefox لفتح ملفات XHTM.

## مثال XHTM

يحدد المثال التالي لمستند XHTM تعريفات XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
