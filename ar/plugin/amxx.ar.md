## ما هو ملف AMXX؟

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. إنها منصة برمجة نصية تسمح لمسؤولي الخادم بتخصيص وتوسيع وظائف خوادمهم. يتم استخدام AMXX جنبًا إلى جنب مع نظام إدارة المكونات الإضافية MetaMod الذي يسمح بتشغيل المكونات الإضافية المتعددة في وقت واحد على الخادم دون حدوث تعارضات. تتضمن الميزات التي يقدمها البرنامج الإضافي AMXX أوامر المسؤول وتعديلات اللعب وإدارة الخادم.
