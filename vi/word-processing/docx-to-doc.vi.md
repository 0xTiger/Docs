## Giới thiệu về chuyển đổi DOCX sang DOC

Loại tệp DOC là định dạng tệp tài liệu phổ biến được sử dụng bởi năm 2007 và các phiên bản Microsoft Word trước đó. Với việc phát hành Office 2007, Microsoft đã giới thiệu định dạng tệp DOCX dựa trên thông số kỹ thuật của Office Open XML. Các tệp DOC được lưu dưới dạng tệp nhị phân so với DOCX được lưu ở định dạng tệp XML nén. Có rất nhiều công cụ trực tuyến có sẵn để chuyển đổi các tệp [DOCX](/vi/word-processing/docx/) thành [DOC](/vi/word-processing/doc/).

## Chuyển đổi DOCX sang DOC - Bản demo miễn phí

Bạn có thể chuyển đổi định dạng tệp DOCX sang DOC bằng ứng dụng trực tuyến MIỄN PHÍ, Aspose.app. Đây là một bộ ứng dụng có thể tải, xem và chuyển đổi các tệp DOCX sang các định dạng tệp như DOC, PDF, TXT, RTF và các định dạng khác. Các ứng dụng Aspose được cung cấp và chạy bởi API Aspose ở phần phụ trợ có thể thực hiện một số thao tác trên các loại tệp khác nhau. Hãy dùng thử ứng dụng trực tuyến Aspose.Words ngay bây giờ bằng cách nhấp vào nút sau.

### Các bước chuyển đổi DOCX sang DOC trực tuyến

Bạn có thể chuyển đổi định dạng tệp DOCX sang DOC bằng [ứng dụng Aspose.Words DOCX to DOC](https://products.aspose.app/words/conversion/docx-to-doc) như sau.

1. Đi tới ứng dụng Aspose.Words để [chuyển đổi DOCX sang DOC](https://products.aspose.app/words/conversion/docx-to-doc)
1. Tải lên tệp DOCX của bạn bằng cách nhấp vào nút tải lên
1. Sau khi hoàn tất, hãy nhấn nút Chuyển đổi. Thao tác này sẽ bắt đầu quá trình chuyển đổi tệp DOCX sang DOC ở chế độ nền
1. Bạn sẽ được yêu cầu tải xuống tệp RTF đã chuyển đổi sau khi quá trình chuyển đổi DOCX sang DOC hoàn tất

## API Aspose.Words

[Aspose.Words](https://products.aspose.com/words/) là một bộ API cho phép mọi nhà phát triển chuyển đổi [DOCX](/vi/word-processing/docx/) thành [DOC](/vi/word-processing /doc/) bằng các dòng mã đơn giản. Nó có sẵn cho nhiều ngôn ngữ lập trình bao gồm [C#](/vi/programming/cs/), [Java](/vi/programming/java/), [Python](/vi/programming/py/) và [C++](/vi/programming/cpp/ ). Ngoài DOC, API Aspose.Words cho phép bạn chuyển đổi tệp DOCX sang nhiều định dạng tệp phổ biến khác.
