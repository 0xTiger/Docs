## Tập tin SDF là gì?

A Spatial Data File (SDF) is a Geodatabase file format that was developed by Autodesk. It stores geospatial data and is used by Autodesk GIS program MapGuide and AutoCAD Map 3D. SDF file format is based on the single file database file format SQLite3. Nó được coi là một định dạng tệp được tối ưu hóa cho các bộ dữ liệu lớn về thông tin không gian.

Bạn có thể mở tệp SDF bằng Autodesk AutoCAD Map 3D 2022 và Autodesk AutoCAD Civil 3D 2023.

## Định dạng tệp SDF - Thông tin thêm

Định dạng tệp SDF được lưu vào đĩa dưới dạng tệp nhị phân. Nó dựa trên cơ sở dữ liệu SQLite sử dụng các thành phần lưu trữ cấp thấp để tuần tự hóa dữ liệu nhị phân. Tệp SDF chứa nhiều lớp đối tượng trên mỗi tệp và nhiều thuộc tính hình học cho từng lớp đối tượng. Dữ liệu trong tệp SDF có thể truy cập được ở tốc độ cao do lập chỉ mục cho từng thuộc tính hình học bằng cây R.
