## Tập tin XTM là gì?

Tệp có phần mở rộng .xhtm là tệp trang web được viết bằng XHTML. Nó sử dụng XML 1.0 làm triển khai tiêu chuẩn và được coi là bản cải tiến của HTML 4. Giống như XHTML, tệp XHTM chứa các thẻ HTML tiêu chuẩn tuân theo các nguyên tắc XML nghiêm ngặt hơn. Các tệp XTM được sử dụng để xuất bản nội dung trang web và thường sử dụng tiện ích mở rộng [XHTML](/vi/web/xhtml/).

Bạn có thể sử dụng các trình duyệt web như Google Chrome, Apple Safari và Mozilla Firefox để mở tệp XTM.

## Ví dụ XTM

Ví dụ sau đây về tài liệu XTM định nghĩa các khai báo XML.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
