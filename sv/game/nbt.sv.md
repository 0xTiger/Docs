## Vad är en NBT fil?

NBT-fil är ett filformat som används i Minecraft. NBT står för Named Binary Tag och det är ett enkelt binärt format för att lagra strukturerad data (liknande JSON eller XML). I Minecraft används NBT-filer för att lagra olika typer av speldata, inklusive spelarinventeringar, världsinformation, enhetsdata och mer.

NBT-filer lagras vanligtvis med filtillägget `.dat` och kan hittas i en Minecraft-världs katalogstruktur. Till exempel lagras spelarinventeringar i `.dat`-filer i `playerdata`-katalogen, medan världsinformation lagras i `.dat`-filer i `region`-katalogen.

## NBT-filformat - Mer information

NBT är ett enkelt binärt format utformat för att effektivt lagra stora mängder binär data. Det används av Minecraft för sina datafiler på grund av dess flexibilitet och användarvänlighet. Minecraft är ett populärt 3D-sandlådespel där spelare kan bygga och utforska virtuella världar. Java Edition av Minecraft är särskilt populär bland moddare, som modifierar spelet genom att redigera dess filer eller lägga till nya för att anpassa spelet. NBT-filer i Minecraft granskas och redigeras ofta av moddare. Dessa filer komprimeras vanligtvis med GZip och kan innehålla olika speldataobjekt som är avgörande för världsbyggande och anpassning.

## Hur man öppnar en NBT-fil

Flera mjukvaruverktyg finns tillgängliga för att öppna och redigera NBT-filer. Här är listan över program som kan öppna eller referera till NBT-filer.

- **NBTExplorer** (gratis) för (Windows, Mac, Linux)
- **NBTEdit** (gratis) för Windows
- **MCEdit** (gratis) för (Windows, Linux)
