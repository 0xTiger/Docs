## Vad är en USX fil?

En USX-fil innehåller Unreal Engines mesh som är fördesignade 3D-objekt/modeller som används för att generera miljön. Modeller från USX-fil har olika position, orientering och storlekar, men dessa förblir statiska. Statiska meshes kan optimeras för prestanda och kan användas för att skapa olika detaljnivåer (LODs) för att förbättra spelets prestanda. De kan också användas för att skapa kollisionsvolymer och kan användas som en del av fysiksimuleringar.

USX-filer kan skapas med Unreal-nivåredigeraren UnrealEd.

## USX-filformat - Mer information

USX-filer sparas på skiva som binära filer. De interna filstrukturdetaljerna för USX-filformatet är inte tillgängliga offentligt för utvecklarens referens.
