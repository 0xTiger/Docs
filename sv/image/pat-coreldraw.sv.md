## Vad är en PAT fil?

En ".PAT-fil är vanligtvis associerad med **CorelDRAW**, ett populärt redigeringsprogram för vektorgrafik. I CorelDRAW är ".PAT"-filen "Pattern File" som innehåller information om mönster eller texturer som kan appliceras på objekt i din design. Mönster kan användas för att fylla objekt med olika återkommande design, såsom ränder, prickar eller anpassade texturer. Dessa mönster lagras i ".PAT"-filer för enkel återanvändning.

## Hur skapar och använder man PAT-filer?

Så här kan du skapa och använda mönsterfiler i CorelDRAW:

1. Skapa ett mönster:
    



- Öppna CorelDRAW.
- Designmönster du vill spara som ".pat"-fil.
- Välj objekt eller område med mönster.
- Gå till menyn "Objekt" och välj "Fyll" för att öppna Fyllningsverktyget.
- Välj "Mönsterfyllning" i Fyllningsverktyget.
- Anpassa mönstrets egenskaper, såsom storlek, vinkel och avstånd.
2. Spara mönster som ".pat"-fil:
    



- När du har skapat ditt mönster kan du spara det som ".pat"-fil för framtida bruk.
- Gå till "Arkiv"-menyn och välj "Exportera" eller "Spara som."
- Välj filformat "CorelDRAW Pattern File (*.pat)."
- Ange namn på filen och välj plats för att spara den.
3. Applicera mönstret:
    



- För att använda sparat mönster i ett nytt projekt eller på ett annat objekt kan du importera det.
- Öppna nytt projekt i CorelDRAW.
- Gå till "Objekt"-menyn, välj "Fyll" och välj "Mönsterfyllning."
- Klicka på "Importera"-knappen och bläddra till ".pat"-filen du sparade.
- Välj mönster från importerade mönster och applicera det på ditt objekt.

## PAT-filer för vektorbilder och sidlayouter

.PAT-filer är speciellt utformade för vektorbilder och sidlayoutdokument. De lagrar mönster som sömlöst kan upprepas inom vektorbaserad bild- eller sidlayout och de används ofta för att applicera texturerade bakgrunder på design.

Corel CONNECT är ett praktiskt webbläsarverktyg för grafikinnehåll som levereras med CorelDRAW Graphics Suite. Det erbjuder ett paket med olika vektormönster som sömlöst kan integreras i dina bilder eller dokument. Dessa vektormönster är redo att användas, vilket gör det bekvämt att förbättra dina mönster med färdiga mönster, allt inom CorelDRAW Graphics Suite-miljön.

## CorelDRAW Graphics Suite

CorelDRAW Graphics Suite är ett omfattande programvarupaket designat för grafisk design och illustration. Den innehåller en rad kraftfulla verktyg för att skapa vektorgrafik, layoutdesign, fotoredigering och typografi. CorelDRAW är mittpunkten i sviten känd för sina vektorillustrationsmöjligheter, medan andra applikationer som Corel PHOTO-PAINT tillhandahåller bildredigeringsfunktioner. Sviten används i stor utsträckning av proffs och kreativa proffs för uppgifter som logotypdesign, sidlayout, fotoredigering med mera. Den erbjuder mångsidig uppsättning funktioner för både nybörjare och avancerade användare, vilket gör den till ett populärt val för många olika grafiska designprojekt.
