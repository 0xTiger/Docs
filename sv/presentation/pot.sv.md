## Vad är en POT fil?

Filer med filtillägget .POT representerar Microsoft PowerPoint-mallfiler skapade av PowerPoint 97-2003-versioner. Filer som skapats med dessa versioner av Microsoft PowerPoint är i binärt format jämfört med de som skapats i Office OpenXML-filformat med de högre versionerna av PowerPoint. Filerna som genereras kan därför användas för att skapa presentationer som har samma layout och andra inställningar som krävs för att tillämpas på nya filer. Dessa inställningar kan inkludera stilar, bakgrunder, färgpalett, typsnitt och standardinställningar. Sådana filer genereras för att skapa färdiga mallfiler för officiellt bruk.

## Filformatspecifikationer ##

Filformatspecifikationer för POT-filformat är inte tillgängliga individuellt av Microsoft. Mallformatet är dock detsamma som för det binära [PPT](/sv/presentation/ppt/) filformatet och kan öppnas i MS PowerPoint för redigering. En POT-fil identifieras med HEX-identifierare enligt följande:

```
D0 CF 11 E0 A1 B1 1A E1 00 00 00 00
```

MIME-typerna associerade med POT-filformat är:

* application/vnd.ms-powerpoint [officiell]
* applikation/mspowerpoint
* applikation/x-mspowerpoint
* applikation/powerpoint
* applikation/x-powerpoint
* application/x-dos_ms_powerpnt
* applicering/kruka
* applikation/x-soffic
