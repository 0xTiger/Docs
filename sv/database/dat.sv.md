## Vad är en DAT fil?
En fil med filtillägget .dat tillhör ett generiskt filformat. Vanligtvis är dessa varken krypterade eller binära, men de kan vara nästan vad som helst. En mängd olika applikationer använder DAT-filerna som referenser till vissa operationer i sina respektive program. Så vi kan inte säga att en specifik programvara eller program är ansvarig för att skapa DAT-filer. Dessa filer nås särskilt endast av det program som skapade dem.


## DAT filformat
DAT-filformatet är helt enkelt ett allmänt format som innehåller alla typer av data i en fil, vanligtvis sparad med filtillägget .dat. Den kan innehålla data i binärt eller textformat, vanligtvis består en generisk data av informationen som är specifik för applikationen den refererar till. Oftast kan du hitta dem med andra konfigurationsfiler som DLL-filer. Här kan vi anlita några kända filtyper, med filtillägget .dat:

### VCD-videofil
VCD-skivor var välkända video- eller multimediakällor på 1990-talet för distribution av filmer och hemmavideor före DVD-skivor. Vanligtvis hittades videofilerna från VCD-skivan med .dat-tillägget. Så det lagrade de faktiska videodata som spelades upp från VCD:n.
### Exchange e-postbilaga
DAT-filer relaterade till Exchange e-postbilaga innehåller formateringsinformation för RTF-meddelanden som skickas från Microsoft Exchange Server. Vanligtvis läggs dessa filer till i slutet av e-postmeddelanden när en mottagares e-postklient inte stöder RTF-meddelanden. Dessa filer finns vanligtvis som **Winmail.dat**.
### Windows Registry Hive-fil
NTUSER.DAT är en vanlig registerfil. Microsoft Windows-registret använder DAT-filer för att lagra användarinställningar. Dessa DAT-filer laddas när en användare loggar in på Windows. Det kan innehålla kontoinställningar, lösenord och filassociationer.
### Minecraft-datafil
Minecraft-spelet använder DAT-filerna för att spara information om spelarens inventering, spelarposition och världsinformation. DAT-filerna används också för att lagra nivåer, som är små kartbitar som laddas på begäran när en spelare navigerar genom världen.
### Piriform DAT-fil
Applikationerna som CCleaner, Speccy, Defraggler och Recuva använder DAT-filer för information om deras typiska operationer.
