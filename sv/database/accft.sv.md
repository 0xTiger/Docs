## Vad är en ACCFT fil?

En fil med filtillägget .accft är en Microsoft Access-mallfil som innehåller definitionen av fältdatatyper för tabellerna i en databas. Microsoft Access är ett RDBMS (Relational Database Management System) som används för att skapa tabellbaserade databaser, formulär, frågor och vyer. ACCFT-filen kan användas för att definiera datatyperna för tabellfälten och databastabeller kan skapas med denna mallfil som utgångspunkt. Den kan öppnas med Microsoft Access 2016 för att skapa nya tabeller. ACCFT-filer känns igen av MIME-typen **application/msaccess**.

## ACCFT filformat

ACCFT-filer är binära filer vars filformatspecifikationer inte är tillgängliga offentligt. Dess interna detaljer som sidhuvudsinformation, filstruktur, sidfötter etc. är inte tillgängliga för tillfället.
