## Vad är JET fil?

En JET-databasfil, även känd som en Microsoft Access-databasfil, är ett filformat som används av Microsoft Access, ett populärt databashanteringssystem (DBMS). JET står för Joint Engine Technology, vilket var det ursprungliga namnet på databasmotorn som Microsoft Access använde. Senare versioner av Access har använt olika databasmotorer, men termen "JET-databas" används fortfarande ofta för att referera till Access-databaser.

Här är några viktiga punkter om JET-databasfiler:

- **Filtillägg:** JET-databasfiler har vanligtvis filtillägget ".mdb" för äldre versioner av Access (Access 2003 och tidigare) och ".accdb" för nyare versioner (Access 2007 och senare). ".mdb"-filerna använder den äldre JET-databasmotorn, medan ".accdb"-filerna använder den nyare ACE-tekniken (Access Database Engine).

- **Relationsdatabas:** JET-databaser är relationsdatabaser, vilket innebär att de lagrar data i tabeller med definierade relationer mellan dem. Användare kan skapa, ändra och fråga data med SQL (Structured Query Language) eller via det grafiska gränssnittet Access.

- **Datalagring:** JET-databaser kan lagra ett brett utbud av datatyper, inklusive text, siffror, datum, bilder och mer. De kan också hantera komplexa datastrukturer och relationer mellan tabeller.

- **Multi-användaråtkomst:** JET-databaser stöder åtkomst för flera användare, vilket gör att flera användare kan arbeta med databasen samtidigt, även om det kan finnas begränsningar vad gäller skalbarhet och prestanda för storskaliga applikationer.

- **Integration:** Access-databaser kan integreras med andra Microsoft Office-program som Excel och Outlook, vilket gör det bekvämt för användare att utbyta data mellan olika program.

## Information om JET-databaser

**JET-motorns historia:**

Databasmotorn JET (Joint Engine Technology) utvecklades ursprungligen av Microsoft i början av 1990-talet för användning i Microsoft Access. Motorn designades för att vara en lätt, skrivbordsorienterad databaslösning som kunde integreras i olika Microsoft-produkter.

**Komponenter i en JET-databas**

- **Tabell:** JET-databaser lagrar data i tabeller, som är samlingar av poster organiserade i rader och kolumner.
- **Frågor:** Användare kan skapa frågor för att extrahera, filtrera och manipulera data från tabeller med SQL (Structured Query Language).
- **Formulär:** Access tillhandahåller en formulärdesigner som tillåter användare att skapa anpassade datainmatnings- och visningsformulär.
- **Rapporter:** Användare kan skapa professionella rapporter från data som lagras i databasen.
- **Makron och VBA:** Användare kan automatisera uppgifter och lägga till anpassade funktioner med hjälp av makron eller genom att skriva VBA-kod.

**Datatyper**

JET-databaser stöder en mängd olika datatyper, inklusive:

- **Text:** För lagring av alfanumeriska tecken.
- **Nummer:** För numeriska värden.
- **Datum/tid:** För datum- och tidsvärden.
- **Memo:** För längre text eller anteckningar.
- **Hyperlänk:** För webblänkar och e-postadresser.
- **Bilaga:** För lagring av filer och bilagor.
