## Vad är en CHCK-fil?

En CHCK-fil skapas som ett resultat av hälsokontroller på Microsoft Exchange Server. Den innehåller information om den övergripande statusen och operationaliseringen av serverns databas. Microsoft Exchange Server används främst som e-postserver och låter dig underhålla dina affärskontakter och kalendrar. Exchange-serveradministratörer använder dess EseUtil för att kontrollera dess databastillstånd.

## CHCK filformat

Det interna filformatet för CHCK-filen är inte känt och det finns inget referensmaterial tillgängligt för att läsa/skriva CHCK-filer.
