## Vad är FDB fil?

En .fdb-fil är ett filtillägg som används för databasfilerna i Microsoft Dynamics NAV. Filtillägget .fdb står för Firebird Database File och det är filformatet som används av Firebird Database Management System, som är den underliggande databasmotorn som används av Microsoft Dynamics NAV. .fdb-filen innehåller alla data, tabeller och strukturer i NAV-systemet, inklusive finansiella data, lagerdata, kunddata och mer. Det är viktigt att ha en säkerhetskopia av .fdb-filen för att säkerställa dataintegritet och för att kunna återställa data i händelse av problem. .fdb-filen kan exporteras, importeras och även användas i replikering.

## Relation med Microsoft Dynamics NAV

FDB-filer är databasfiler i Microsoft Dynamics NAV. Microsoft Dynamics NAV är en ERP-programvara (Enterprise Resource Planning) utvecklad av Microsoft. Den är designad för små och medelstora företag och tillhandahåller ett brett utbud av företagsledningsfunktioner, inklusive ekonomistyrning, supply chain management, tillverkning, projektledning och mer. Den är byggd på Microsoft Dynamics-plattformen och är helt integrerad med andra Microsoft-produkter, som Office och Outlook. Dynamics NAV kan nås via en webbklient, såväl som via en Windows-klient, och den kan även anpassas och integreras med andra system med hjälp av sin egen utvecklingsmiljö som kallas C/AL. Det används ofta av företag inom tillverkning, grossistdistribution och tjänstesektorn.

## Hur öppnar man en .fdb-fil?

FDB-filen öppnas och hanteras vanligtvis med hjälp av utvecklingsmiljön Microsoft Dynamics NAV eller Microsoft Dynamics NAV-klienten.

Så här öppnar du en .fdb-fil i Microsoft Dynamics NAV:

1. Öppna utvecklingsmiljön Microsoft Dynamics NAV.
2. Klicka på Arkiv-menyn och välj Öppna eller tryck på Ctrl+O
3. Navigera till platsen där .fdb-filen sparas.
4. Välj .fdb-filen och klicka på knappen Öppna.

Du kan också öppna .fdb-filen genom att ansluta till databasen med NAV-klienten.

1. Öppna NAV-klienten.
2. Klicka på Arkiv-menyn och välj Anslut
3. I fältet Server anger du namnet eller IP-adressen för servern där .fdb-filen finns.
4. Ange lämpligt Databasnamn och Inloggningsuppgifter
5. Klicka på knappen Anslut.

Det är också möjligt att öppna .fdb-filer med Firebirds programvara för databashanteringssystem, eftersom det är den underliggande databasmotorn som används av Microsoft Dynamics NAV.
