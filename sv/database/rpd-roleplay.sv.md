## Vad är RPD fil?

En "RPD"-fil är en specialiserad datafil som är associerad med **Roleplay Designer**, ett samarbetande utvecklingssystem för e-lärande som är skräddarsytt för att skapa beslutsbaserade utbildningsmoduler. Inom detta system lagrar dessa filer viktig information som beskriver strukturen för de träningsscenarier som har utformats noggrant. Varje "RPD"-fil innehåller en ritning av de interaktiva elementen, förgreningsvägar och olika beslutspunkter som utgör träningsupplevelsen.

När det gäller att använda träningsmodulerna fungerar dessa "RPD"-filer som lagringsplatser för information. De nås och laddas av Roleplay Designer-systemet under träningspass. Data som finns i filerna vägleder systemet i att presentera scenarierna, hantera användarinteraktioner och ge feedback baserat på de beslut som tagits av deltagarna.

Observera att Roleplay Designer har **upphört**, vilket betyder att det kan vara svårt att hitta officiell support eller dokumentation för filformatet "RPD". I sådana fall kan individer som arbetar med dessa filer behöva utforska metoder som att analysera filernas innehåll, söka hjälp från tidigare användare av programvaran, försöka omvända formatet eller överväga alternativa verktyg för att uppnå sina utvecklingsmål för e-lärande.

## Programvara för rollspelsdesigner

Roleplay Designer är ett samarbetande utvecklingssystem för e-lärande som används för att designa och bygga beslutsbaserade utbildningsmoduler.

**Associerade registernycklar**
```
HKEY_LOCAL_MACHINE\SOFTWARE\Interscape\RoleplayDesigner
HKEY_CURRENT_USER\SOFTWARE\RoleplayDesigner
```
