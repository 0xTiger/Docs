## Om FBX till XLSX

FilmBox, ofta känt som FBX, är ett flitigt använt 3D-filformat som ursprungligen skapades av Kaydara för att förbättra kompatibiliteten med MotionBuilder. 2006 tog Autodesk Inc. tyglarna i detta format och befäste dess status som en standard för att enkelt utbyta 3D-data mellan olika 3D-program.

**XLSX** är ett allmänt erkänt format för Microsoft Excel-dokument som introducerades av Microsoft tillsammans med lanseringen av Microsoft Office 2007. Det följer en strukturerad organisation baserad på Open Packaging Conventions, som beskrivs i [Del 2](https:// www.ecma-international.org/publications/standards/Ecma-376.htm) i ECMA-376 OOXML-standarden. Detta nya format är i huvudsak ett zip-paket som innehåller flera XML-filer.

Du kan använda FBX till XLSX-konverteringsapparna som finns tillgängliga online för att konvertera dina FBX-filer till XLSX-format. Dessa onlineappar behöver du inte installera någon specialiserad programvara för sådana konverteringar.
