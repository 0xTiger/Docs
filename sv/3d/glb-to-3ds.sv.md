## Om GLB till 3DS

Användningen av GLB-filer i 3D-filformat är inte något nytt. GLB är ett vanligt använt binärt 3D-filformat som representerar 3D-modeller i GL Transmission-formatet, dvs ([glTF](/sv/3d/gltf/)). [3DS](/sv/3d/3ds/), å andra sidan, representerar 3D Sudio (DOS) mesh-filformat som används av Autodesk 3D Studio. Flera appar finns tillgängliga online som kan användas för konvertering av [GLB](/sv/3d/glb/) till [3DS](/sv/3d/3ds/) GRATIS. Om du är en applikationsutvecklare som vill lägga till funktionaliteten hos GLB till 3DS i sitt .NET eller Java, kan du använda Aspose.3D API för att uppnå detta.
