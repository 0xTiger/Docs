## Om GLB till PDF

GLB-filer är binära filer som används för att spara och representera 3D-modeller i filformatet GL Transmission. [PDF-filformat](/sv/pdf/) introducerade en standard för representation av dokument och annat referensmaterial som är oberoende av applikationsprogramvara, hårdvara samt operativsystem. Flera appar finns tillgängliga online som kan användas för att konvertera [GLB](/sv/3d/glb/) till PDF online GRATIS. För applikationsutvecklare som skriver applikationer för att konvertera GLB-filer till andra filformat, kan du använda Aspose.3D API för att uppnå detta.
