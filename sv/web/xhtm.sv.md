## Vad är en XHTM fil?

En fil med filändelsen .xhtm är en webbsidesfil som är skriven i XHTML. Den använder XML 1.0 som standardimplementering och anses vara en omformulering av HTML 4. Liksom XHTML innehåller en XHTM-fil standard HTML-taggar som följer en mer sträng XML-riktlinjer. XHTM-filer används för att publicera webbplatsinnehåll och använder vanligen tillägget [XHTML](/sv/web/xhtml/).

Du kan använda webbläsare som Google Chrome, Apple Safari och Mozilla Firefox för att öppna XHTM-filer.

## XHTM exempel

Följande exempel på ett XHTM-dokument definierar XML-deklarationerna.

```
<!DOCTYPE html
     PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns#"http://www.w3.org/1999/xhtml" xml:lang#"en" lang#"en">
  <head>
    <title>Public Property</title>
  </head>
  <body>
    <p>changed to <a href#"http://sample.com/">sample.com</a>.</p>
  </body>
</html>
```
