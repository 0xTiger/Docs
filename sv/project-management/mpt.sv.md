## Vad är en MPT fil? ##

En fil med filtillägget .mpt är mallfilerna [Microsoft Project](https://products.office.com/en-us/project/project-and-portfolio-management-software). Den innehåller grundläggande information och struktur och dokumentinställningar för att skapa [MPP](/sv/project-management/mpp/)-filer. Dessa mallar tillhandahåller standardinställningar som är viktiga för ett visst projekt, såsom scheman eller budgetinformation. Även om det inte kan spara projektspecifika data som uppgifter, resurser eller uppdrag. När de har modifierats kan mallfilerna sparas som standard MPP-filer för vidare arbete med dem. En huvudmallfil, med titeln "Global.mpt" är tillgänglig i Microsoft Project, öppnad för att skapa en individuell mall. MPT-filer använder *application/vnd.ms-project* och *application/x-project* som standard MIME-typer. Windows, Linux och macOS kan alla öppna MPT-filer, tillsammans med Moos Project Viewer, Microsoft Project och MPXJ.

## Problem med att öppna en MPT-fil? ##

Här är listan över några vanliga problem som kan uppstå och orsaka att MPT-formatet inte fungerar:

* Avsaknad av stödprogramvara
* Korrupt fil
* Infekterad fil på grund av virus
* Ingen åtkomsträtt i systemet för att öppna filerna
* Föråldrad enhet i ditt system
 *   Extension of the file is renamed
