## Vad är en JFPROJ fil?

En fil med filtillägget .jfproj är ett JSON-baserat anpassat typsnitt skapat med Microsoft Font Maker-applikationen på Windows 10. Dessa kan skapas med hjälp av elektronisk penna med pekskärmsenheter som surfplattor även om annan kringutrustning som mus också kan användas i fall av datorer. JFPROJ-filer innehåller glyfer som används för att återge tecknen på skärmen genom att definiera avståndet mellan tecken och ord, och storleken på teckensnittet. Microsoft Font Maker-appen är tillgänglig för nedladdning i Windows Store.
