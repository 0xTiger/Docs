## Vad är en XL fil?

En fil med filtillägget .xl är en Excel-kalkylbladsfil, liknande [XLS](/sv/spreadsheet/xls/), som lagrar data i rader och kolumner. Det var ett tidigare filformat som inte längre stöds av Excel nu. Data som lagras i XL-filer kan exporteras till andra format som CSV. Dessa kan också konverteras till PDF med hjälp av Excels standardmetod Spara som. XL-filer kunde endast öppnas med Microsoft Excel.

## XL filformat

XL-filer sparas som binära filer och deras interna filformatsspecifikationer dokumenteras inte någonstans. Det allmänna binära filformatet för filformatet [XLS File Format Specifications](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx) kan dock också övervägas om någon försöker skriva en bit kod för att omvända dessa filer.
