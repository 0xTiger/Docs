## Vad är XLA fil?

En fil med filtillägget .xla är ett Microsoft Excel-tilläggsfilformat som lägger till moduler, extrafunktioner och andra verktyg till Microsoft Excel. Annonserna ger extra funktionalitet till Excel-kalkylblad och stöds i version 2007 eller senare. XLA-filer har mime-mediatyp application/vnd.ms-excel. Dessa kan öppnas med Microsoft Excel som är tillgängligt för Windows såväl som MacOS.

## XLA filformat

XLA-filer är baserade på binärt dataformat. Ytterligare information om dess filformatspecifikationer finns dock inte tillgänglig. Allmänt antagande är att XLA följer [[MS-XLS] - Excel Binary File Format Structure](https://msdn.microsoft.com/en-us/library/cc313154(v#office.12).aspx).
