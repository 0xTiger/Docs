## Vad är AMXX fil?

An AMXX file is an AMX Mod X plugin file used by first-person shooter video game such as Half-Life developed by the Valve Corporation in 1998. Det är en skriptplattform som tillåter serveradministratörer att anpassa och utöka funktionaliteten på sina servrar. AMXX används tillsammans med MetaMod plugin-hanteringssystem som tillåter flera plugins att köras samtidigt på en server utan konflikter. Funktioner som erbjuds av AMXX-plugin inkluderar administratörskommandon, spelmodifieringar och serverhantering.
