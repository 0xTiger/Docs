## Vad är SPARC fil?

En .sparc-fil, alternativt känd som ett Skype-pluginarkiv, fungerar som ett paket som innehåller extra funktioner eller förbättringar för Skype-applikationen. Dessa plugins har kapacitet att introducera en mängd olika funktioner som samtalsinspelning, skärmdelning, översättning,
Spel och underhållning, anpassade uttryckssymboler och teman.

## SPARC-filformat - Mer information

I vissa versioner av Skype är Extrahanteraren en valfri komponent som kan integreras i Skype-installationen. Om du stöter på problem med att installera en SPARC-fil, är det möjligt att du har utelämnat Extrahanteraren under den första installationen. För att lösa det här problemet, installera om Skype och se till att du väljer alternativet Extrahanteraren.

## Hur installerar jag Skype-plugin?

Steg för att installera Skype-plugins:

1. Skaffa .sparc-filen som motsvarar önskad plugin.
2. Stäng Skype om det körs för närvarande.
3. Leta reda på den nedladdade .sparc-filen och dubbelklicka på den.
4. Följ instruktionerna på skärmen för att slutföra plugin-installationen.
5. Efter installationen, starta om Skype för att aktivera insticksprogrammet.
