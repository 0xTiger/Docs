## Vad är REZ fil?

Filtillägget .rez är en plugin-fil som används av Ambrosia Softwares Escape Velocity-serie. Dessa filer används för att modifiera eller utöka spelet genom att lägga till nytt innehåll, såsom skepp, uppdrag eller grafik. Pluginsystemet är en av nyckelfunktionerna som bidrog till seriens popularitet, eftersom det gjorde det möjligt för spelare att anpassa sin spelupplevelse och dela sina skapelser med andra.

## REZ-filformat - Mer information

Här är en kort översikt över hur .rez-filerna fungerar i Escape Velocity:

**Pluginstruktur:** .rez-filerna är i huvudsak behållare som innehåller olika resurser och skript för spelet. Dessa resurser kan inkludera grafik, ljud, fartygsdata, uppdragsskript och mer.

**Anpassning:** Spelare och tredjepartsutvecklare kan skapa sina egna plugins för att introducera nya element i spelet. Detta kan sträcka sig från att lägga till nya typer av fartyg och vapen till att skapa helt nya berättelser och uppdrag.

**Kompatibilitet:** Pluginsystemet tillåter flera plugins att samexistera, och spelare kan välja vilka som ska aktiveras. Detta innebär att spelare kan skräddarsy sin spelupplevelse genom att välja de plugins som matchar deras preferenser.

**Communitybidrag:** Escape Velocity-serien hade en blomstrande moddinggemenskap, med många spelare som skapade och delade sina egna plugins. Detta bidrog avsevärt till spelens livslängd och omspelbarhet.

**Installation:** To use a plugin in Escape Velocity, players typically place the **.rez file** in a designated folder within the game directory. The game's launcher or menu system then allows players to activate or deactivate specific plugins.

## Om Ambrosia Escape Velocity

Escape Velocity-serien utvecklades av Ambrosia Software för Macintosh-datorer. Spelen utspelar sig i ett stort, öppet universum där spelare kan utforska, handla och delta i rymdstrider. Serien är anmärkningsvärd för sitt icke-linjära spel, vilket gör det möjligt för spelare att välja sin egen väg och lojalitet.

Det ursprungliga Escape Velocity-spelet släpptes 1996, och det följdes av två uppföljare: Escape Velocity Override (1998) och Escape Velocity Nova (2002). Dessa spel blev populära för sitt uppslukande spel, omfattande moddinggemenskap och förmågan att forma historien baserat på spelarens beslut.

**Obs:** Ambrosia Escape Velocity och EV Nova har utgått.

## Hur öppnar man filen REZ?

Du kan öppna REZ-filen med programvaran Ambrosia Escape Velocity och Ambrosia EV Nova på både Windows och Mac OS.
