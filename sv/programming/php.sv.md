## Vad är en PHP-fil?
En fil med filtillägget .php hänvisar till programmeringsspråk med öppen källkod, som används för att skriva skript på serversidan, som ska köras på en webbserver. Det är det mest använda webbskriptspråket som vanligtvis används för utveckling av storskaliga webbapplikationer. Det största bloggsystemet på webben, dvs WordPress och det största sociala nätverket Facebook är baserade på PHP-språk. Det är kompatibelt med olika plattformar som Windows, Linux, Unix, Mac OS X, etc. Det körs också framgångsrikt på nästan alla servrar som används idag dvs Apache, IIS, etc. PHP stöder ett brett utbud av databaser inklusive MySQL.

## PHP-filformat ##

En enda PHP kan innehålla text, CSS, [HTML](/sv/web/html/), JavaScript och PHP-kod. PHP-kod exekveras på servern och resultatet returneras till webbläsaren som vanlig HTML. PHP-filer kan öppnas med vilken textredigerare som helst och redigeras på plats, även om applikationer som Adobe Dreamweaver, Eclipse PHP Development-verktyg erbjuder ett bekvämt sätt att skriva och ändra PHP-kod.

### Syntax ###

Ett PHP-skript körs på servern och det vanliga HTML-resultatet skickas tillbaka till webbläsaren. Ett PHP-skript börjar med<?php and ends with ?>

### PHP-exempel ###

```
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```
