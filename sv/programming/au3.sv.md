## Vad är AU3 fil?

En AU3-fil är en automatiserad skriptfil som skapats med gratisprogramprogrammet [AutoIt v3](https://www.autoitscript.com/site/). Den innehåller skriptspråket AutoIt3 för att automatisera kommandon i Windows. Programmeringsspråket AutoIt3 låter dig simulera ett antal händelser som kan användas för att automatisera uppgifter. Till exempel kan skriptet inuti AU3-filen användas för att simulera tangenttryckningar, musrörelser, styra Windows-position och andra skriptändamål.

Applikationer som kan öppna AU3-filer inkluderar [SciTE4AutoIt3](https://www.autoitscript.com/site/autoit-script-editor/downloads/) och AutoIt.

## AU3 filformat

AU3-filer sparas som textfiler och innehåller AutoIt3-skriptspråk.
