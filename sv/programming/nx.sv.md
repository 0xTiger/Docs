## Vad är NX fil?

En NX-fil är en videospelsfil skapad för att användas med den virtuella spelkonsolen, **LowRes NX**. Det är skrivet i BASIC-programmeringsspråket och innehåller källkoden för spelet som definierar hur spelet fungerar. Du kan skapa dina egna NX-filer för användning med appen som kan laddas och användare kan spela spelet.

Du kan öppna en NX-fil i en textredigerare, LowRes NX eller någon annan källkodsredigerare som Microsoft Visual Studio Code.

## NX filformat

En NX-fil sparas på skivan som vanlig textfil som innehåller spelinstruktionerna. Varje NX-fil har spelets källkod som består av flera patron ROM-poster. Dessa ROM-poster har spelkomponentinformation som grafik, nivåkartor, ljud och andra icke-källkodsdata för spelet. ROM-posterna för ett program kan läggas till eller redigeras med hjälp av NX-redigeringsprogram som följer med LowRes NX.

## Hur skapar man NX-filer?

### Skapa NX-filer på PC

Du kan skapa NX-filer på PC med:

* Textredigerare - som Microsoft Notepad, Notepad++ eller Github Atom
* Källkodsredigerare - som Microsoft Visual Studio Code

### Skapa NX-filer på iOS

LowRes NX har iOS-version som kan användas av utvecklare för att skapa NX-filer från denna app.
