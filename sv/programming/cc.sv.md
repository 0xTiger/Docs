## Vad är CC fil?

En CC-fil är en typ av källkodsfil som innehåller kod skriven i programmeringsspråket C++. Den kan antingen innehålla all kod som behövs för ett enda program eller en del av koden som är en del av ett större programmeringsprojekt som består av flera CC-filer. För att ändra eller visa koden som finns i CC-filen använder utvecklare ofta en källkodsredigerare. Efter redigering kompilerar de koden med hjälp av kompilatorn för att konvertera den till maskinkod, som datorns operativsystem kan köra för att köra programmet.

## CC-filformat - Mer information

Filtillägget .cc har använts för C++-källkodsfiler i flera decennier. Språket C++ utvecklades i slutet av 1970-talet av Bjarne Stroustrup som vid den tiden arbetade på Bell Labs. Stroustrup ville utöka programmeringsspråket C med stöd för objektorienterade programmeringskoncept som han ansåg var avgörande för att bygga stora mjukvarusystem.

Den första versionen av C++ släpptes 1983 och den blev snabbt populär bland utvecklare som uppskattade dess kraftfulla funktioner och flexibilitet. När C++ blev mer allmänt använt blev det nödvändigt att upprätta kodningskonventioner och filnamnskonventioner för att hjälpa till att organisera och hantera stora kodbaser.

Googles C++ Style Guide, som rekommenderar att man använder .cc filtillägg för C++ källkodsfiler, publicerades först 2003. Sedan dess har guiden uppdaterats regelbundet för att återspegla förändringar i C++-språket och utvecklingspraxis i en bredare programmeringsgemenskap.

Idag är .cc-filtillägget fortfarande ett populärt val för C++-källkodsfiler och det känns igen av många populära utvecklingsverktyg och plattformar.

## Hur öppnar man filen CC?

Många källkodsredigerare, inklusive multiplattformsverktyg som Microsoft Visual Studio Code och Code Blocks, kan användas för att öppna och redigera CC-filer. Dessa redigerare erbjuder användbara funktioner som syntaxmarkering och autokomplettering, vilket gör dem till ett utmärkt val för att arbeta med CC-filer.

Du kan också öppna CC-filer med valfri textredigerare, t.ex. Notepad, Notepad++ etc.

## Kompilera CC med C++-kompilatorer

Att kompilera CC-fil med C++-kompilator innebär flera steg. Det första steget är att installera en kompatibel C++-kompilator för ditt operativsystem och din utvecklingsmiljö. Det finns många olika kompilatorer tillgängliga inklusive GCC, Clang och Microsoft Visual C++.

När du har installerat C++-kompilatorn är nästa steg att öppna terminalen eller kommandotolken och navigera till katalogen där CC-filen finns. Härifrån kan du köra kompilatorn genom att köra ett kommandoradsverktyg från kompilatorn. Det specifika kommandot för att kompilera CC-filen kan variera beroende på kompilatorn du använder. Som ett exempel, för att kompilera CC-fil med GCC, skulle du köra ett kommando som liknar:

```
g++ -o myprogram myfile.cc
```
