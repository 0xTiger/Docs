## Vad är en Java-fil?

En Java-fil är en Java-källkodsfil som har skapats i programmeringsspråket Java. Den liknar Java-källfilen och kan kompileras med Java-tolkar. Liksom [Java](/sv/programming/java/)-filer, används Java-filer i utvecklingen av Java-applikationer. Det kan vara en av de många källkodsfiler som finns tillgängliga i ett Java-projekt.

## Java filformat

Java-filer lagras som textfiler och kan öppnas med vilken textredigerare som helst. Java-filtyper har ofta ersatts av Java-filtyper och används ofta för att skriva Java-program. Java-koder kan skrivas med IDE:s som Eclipse och IntelliJ Idea som gör det lättare att arbeta med Java-filer.

### Exempel på Java-filformat

```java
/*
The example code prints
Hello World from Java to the console.
*/
    public class JavaApp {
        public static void main(String[] args) {
            System.out.println("Hello World."); // Prints the string to the console.
        }
    }
```
