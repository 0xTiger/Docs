## Vad är TPZ fil? ##

Filtillägget .tpz är även känt som Kindle Topaz eBook Format av Amazon. Linux, Mac OS och Windows är några av de plattformar där filer med filtillägget .tpz kan distribueras. Som Kindle Topaz eBook-fil är TPZ-filen formaterad för att se ut som traditionella e-böcker och innehåller proprietära datasammanställningsstandarder från Amazon för TPZ-formatet. Den här filen har avancerat stöd för inbäddade teckensnitt, och ytterligare detaljer som anteckningar och bokmärken kan också inkluderas i Kindle Topaz eBook-filer, även om dessa metadatadetaljer lagras separat. TPZ-fil kan också kategoriseras som en datafil som kan laddas ner från Amazons onlinedatabas med Kindle e-böcker.

## Problem med att öppna filen TPZ? ##

Det finns några av de vanliga problemen som nämns nedan som kan uppstå:

* Infekterad fil. Detta kan åtgärdas med ett lämpligt antivirusprogram för att göra det rent
* Otillräckligt diskutrymme. För detta måste stänga alla irrelevanta ansökningar för bättre funktion
* Trasiga länkar kan ibland orsaka problem
* Ibland har filen inte åtkomsträttigheter så i det här fallet se till att åtkomsträttigheterna enkelt öppnar filen
