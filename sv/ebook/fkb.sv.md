## Vad är FKB fil? ##

FKB är ett eBook-filtillägg som utvecklades av Flipkart.com för Flipkart-programvaran. Denna typ av filtillägg finns vanligtvis på Windows 10-användare. Det finns en specifikation i FBK-filen att den kan variera i storlek från ett par hundra kilobyte till många megabyte beroende på bokens längd och antalet bilder som ingår. För att ladda ner filen från Flipkart eBooks Android-applikation måste du först köpa e-boken, klicka sedan på den i biblioteket och boken börjar laddas ner. Om du inte kan se det köpta dokumentet, uppdatera sedan systemet för bättre funktion.

## Operativsystem som stöds ##

Följande operativsystem stöder FKB-filer:

* Windows 7
* Windows 8
* Windows 10
* Windows Server 2003/2008/2012/2016
* Mac OS X
* FreeBSD
* iOS
* Linux
* Android
* NetBSD
* OpenBSD

## Problem med att öppna en FKB-fil ##

* Felaktig installation av stödprogram. Se till att den laddas ner ordentligt med en uppdaterad version
* Lite hårdvaruutrymme
* Korrupta eller infekterade filer. Detta problem kan enkelt lösas genom ett antivirus installerat i ditt system
