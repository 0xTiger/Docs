## Vad är en CDR fil?

En **.CDR**-fil med **"Raw CD Audio Data"** hänvisar vanligtvis till en fil som innehåller ljudinformation extraherad direkt från en ljud-CD, till exempel musikspår. "Raw CD Audio Data" hänvisar till obearbetade digitala ljuddata från CD, som ofta lagras i binärt format. Dessa rådata innehåller ljudprover exakt som de är på CD, inklusive både ljud- och underkodsinformation.

Dessa filer är vanligtvis inte avsedda att spelas direkt utan används istället för olika ändamål som CD-mastering, ljudredigering och CD-bränning. Programvaruapplikationer för ljudredigering och CD-redigering använder ofta dessa rådata för att skapa ljud-CD-skivor eller konvertera dem till andra ljudfilformat som WAV eller MP3.

Om du behöver arbeta med .cdr-filer som innehåller "Raw CD Audio Data" skulle du vanligtvis använda specialiserad ljudmjukvara för att bearbeta eller konvertera dem, eftersom de inte är standardljudfiler som kan spelas i de flesta mediespelare.

## Hur konverterar man en CDR-fil?

För att konvertera en .cdr-fil som innehåller "Raw CD Audio Data" till vanligare ljudformat som WAV eller MP3, behöver du specialiserad programvara. Om du kan spela din .cdr-fil i VideoLAN VLC mediaspelare kan du också använda VLC för att konvertera den till olika ljudformat t.ex.

- [.MP3](/sv/audio/mp3/) - MP3-ljud
- [.FLAC](/sv/audio/flac/) - Gratis förlustfri ljudkodek
- [.WAV](/sv/audio/wav/) - WAVE Audio
- [.OGG](/sv/audio/ogg/) - Ogg Vorbis Audio

## Hur öppnar man filen CDR?

Att öppna .cdr-filen, särskilt en som innehåller "Raw CD Audio Data," kan vara lite knepigt eftersom det inte är standard ljudfilformat. För att öppna och arbeta med sådana filer kan du använda specialiserad programvara som kan hantera rå CD-ljuddata. Så här gör du det och några programvarualternativ:

**Programvara för att öppna och arbeta med .cdr-filer:**

1. **Exakt ljudkopia (EAC)**:
    





- Ladda ner och installera Exact Audio Copy, som är ett populärt och kraftfullt verktyg för att arbeta med ljud-CD-skivor och extrahera "Raw CD Audio Data".
- Sätt i CD med .cdr-fil i din dators CD/DVD-enhet.
- Öppna EAC och konfigurera den efter dina önskemål, inklusive utdataformat (t.ex. WAV eller MP3).
- Använd EAC för att extrahera ljudspår från CD och spara dem i önskat format.
2. **Adacity**:
    





- Audacity är gratis ljudredigeringsprogram med öppen källkod som också kan fungera med "Raw CD Audio Data."
- Sätt i CD.
- Öppna Audacity och konfigurera den för att spela in ljud från din CD-enhet.
- Spela in ljud från CD och spara det sedan i vanligare ljudformat.
3. **VLC Media Player**:
    





- VLC kan spela upp ljud direkt från CD-skivor, inklusive de med "Raw CD Audio Data."
- Sätt i CD-skivan i din dator.
- Öppna VLC, gå till "Media"-menyn och välj "Öppna skiva."
- Välj CD-enhet och spela upp ljudspår direkt.
4. **CD Ripper Software**:
    





- Det finns olika alternativ för CD-ripperprogram, som Windows Media Player (för Windows), iTunes (för Mac) och andra.
- Sätt i CD-skivan i din dator.
- Öppna din favoritprogramvara för CD-ripper och använd den för att extrahera ljudspår och spara dem i vanligt ljudformat.
5. **DAEMON-verktyg** (för .cdr-bildfiler):
    





- Om .cdr-filen är en bildfil skapad av programvara som DAEMON Tools, kan du montera en bild med hjälp av DAEMON Tools eller liknande programvara.
- När bilden väl är monterad kan du komma åt innehållet som om det vore en fysisk CD och spela upp eller extrahera ljud efter behov.
