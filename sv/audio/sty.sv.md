## Vad är STY fil?

STY-fil hänvisar vanligtvis till ett filformat som används av Yamahas elektroniska keyboards och synthesizers. Yamaha ".sty"-filer innehåller stildata som definierar rytmen, ackompanjemangsmönster och andra musikaliska element som används av instrument, används för att förbättra uppspelningen av MIDI-filer genom att tillhandahålla förprogrammerade stilar och arrangemang.

För att använda en .sty-fil med ett Yamaha-klaviatur eller synthesizer överför du vanligtvis filen till instrumentet med en USB-anslutning eller ett minneskort. När det väl har laddats kan instrumentet applicera stildata för att ackompanjera MIDI-uppspelning eller spela live med den valda stilen.

## Yamaha vs Korg STY-filer

Yamaha och Korg sparar STY-filer i olika format, vilket innebär att de inte är plattformsoberoende. Yamaha och Korg är två framstående tillverkare av elektroniska keyboards och synthesizers och de använder sina egna proprietära filformat för stilfiler. Yamaha tillhandahåller dedikerad programvara för att redigera och skapa stilar, som Style Fixer eller Yamaha Expansion Manager. Korg tillhandahåller mjukvaruverktyg som GNIStyle eller Style Works för att redigera och skapa stilar för Korg-tangentbord. Dessa program gör det möjligt för användare att ändra befintliga stilar eller skapa nya enligt deras preferenser.

## Hur öppnar man filen STY?

Du kan använda följande program för att öppna STY-filer.

- Style Fixer (Yamaha-stilar)
- GNIStyle (Korg-stilar)
- WAVE Xtractor
