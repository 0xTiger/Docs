Om du arbetar med digital marknadsföring, gör e-postmarknadsföring eller har någon förlagsbyrå måste du vara medveten om Microsoft Publisher. Den låter dig skapa designlayouter som innehåller information inklusive text, raster och vektorgrafik. Dessa kan användas för att skapa nyhetsbrev, flygblad, broschyrer, vykort och e-postinnehåll.
